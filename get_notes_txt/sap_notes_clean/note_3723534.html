Symptom

The image of meter's readings cannot display directly, only display the message "data:image/png:xxxxx" for example

Environment

SAP Service and Asset Manager 2505 version

Reproducing the Issue

  1. Navigate to the operation screen for the work order not started yet
  2. Choose the meter's readings history
  3. The image of meter's readings cannot display directly, only display the message "data:image/png:xxxxx" for example
  4. There is no change when switching between two different layout options

Cause

Product design

Resolution

Currently SAP Service and Asset Manager does not support the function about "Preview the image for meter's readings", we can follow the influence program in the sap note 11 for new feature request

Keywords
