  • Bank Communication Management (BCM)

  • SAP ERP
  • SAP ERP Central Component
  • SAP enhancement package for SAP ERP 605 and higher
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA all packages
  • SAP S/4HANA Finance

Reproducing the Issue

  1. Access transaction BNK_APP.
  2. Attempt to approve a batch while another user is approving the same batch.
  3. Observe update termination error in SM13.

Cause

Parallel attempts to update the batch status result in update termination errors, as the program's architecture prevents duplicate actions on the same batch.

Resolution

When one user successfully approves a batch, any concurrent attempt to approve the same batch results in an update termination error. This behavior is intentional and aligns with the program's architecture, ensuring duplicate actions on the same batch are not permitted.
