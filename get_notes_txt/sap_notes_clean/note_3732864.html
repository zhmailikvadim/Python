Symptom

This note generates the supplementary objects of SAP Note 3722688.

Version information

Version 1: Released for Customer
Version 2: Validity modified

Other Terms

SAP Note 3722688, RPUUDOH0_3732864

Reason and Prerequisites

Supplementary objects of SAP Note 3722688

Solution

Implementation:

1. Apply the Correction Instructions of the SAP Note 3722688 with the Note Assistant (transaction SNOTE; more information on service.sap.com/note-assistant)

2. Apply the Correction Instructions of the SAP Note 3732864 with the Note Assistant (transaction SNOTE; more information on service.sap.com/note-assistant)

  • Run the newly created RPUUDOH0_3732864 report in "Update & Activate" mode. 

The modifications will also be delivered with the HR-CEE/SAP_HR/S4HCM SP05/2026 package.
