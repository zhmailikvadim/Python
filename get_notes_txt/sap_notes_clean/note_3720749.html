Symptom

Requirement for an API to fetch all groups along with their respective members from the Team tab of the SAP Ariba Sourcing project.

Inquiry regarding relevant API details available from the SAP Ariba APIs Developer Portal.

Cause

Resolution

Utilize the TEAM endpoint of the Sourcing Project Management API to access team-related information for a specific project.

Execute the following sample curl command to retrieve the required data:

curl -X GET "https://openapi.ariba.com/api/sourcing-project-management/v2/prod/projects/WS4444444/teams?realm=mySite&user=myUser&sswordAdapter=passwordAdapter" -H "accept: application/json" -H "apiKey: APIKEY" -H "Authorization: Bearer Token"

Replace the placeholder values such as realm, user, passwordAdapter, APIKEY, and Bearer Token with the actual credentials and configuration details.
