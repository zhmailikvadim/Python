Symptom

  • Handling unit output does not archive.
  • Error message: Message no. VN048 - "No archive exists for output &" 
  • Error message: Message no. VN049 - "Archiving not possible for application &: Object type is missing" 

Environment

  • Logistics General - Handling Unit Management (LO-HU)
  • SAP S/4HANA
  • SAP R/3
  • SAP R/3 Enterprise
  • SAP ERP Central Component
  • SAP ERP
  • SAP Enhancement package for SAP ERP
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA Cloud Private Edition 

Reproducing the Issue

  1. Access transaction VL03N. 
  2. Click on the packing button. 
  3. Go to the handling unit. 
  4. Navigate to extra > output. 
  5. Select the HU output. 
  6. Click "Display originals". 

Cause

Archiving is not possible for application V6 (handling unit outputs) in the standard setup. No object type (OJ_NAME) is maintained in table T681Z. 

Resolution

  1. Note that archiving for handling unit outputs is not supported in the standard setup for application V6. 
  2. Request guidance from the SAP consultant on how to enable archiving for custom HU outputs, as modifications may be required. 

Keywords

archiving, handling unit, output determination, message VN048, table T681Z, application V6
