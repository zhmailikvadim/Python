Symptom

Users want to enable SAP Joule to automatically open the chat interface (pop‑up) when an Integrated Solution page is loaded and expect the chat window to launch without requiring any user interaction, such as clicking the Joule icon.

Environment

  • SAP BTP, Cloud Foundry runtime and environment
  • Joule

Resolution

  • SAP Joule does not provide functionality for automatic initialization of the chat interface during page load. The Joule application must be launched manually by the end user. The chat interface becomes available only after the user selects the Joule (diamond) icon in the upper‑right corner of the SAP SuccessFactors application.
  • There is currently no supported configuration, parameter, or feature within SAP SuccessFactors, SAP BTP, or the Joule integration framework that enables automatic activation, preloading, or forced display of the Joule chat interface upon page load. Manual user interaction is required for initiating the Joule application.
