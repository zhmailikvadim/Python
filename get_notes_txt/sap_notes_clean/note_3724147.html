Symptom

Regarding the IMG documentation of 'Specify Transaction Types for Unplanned Depreciation', it was explained the business case of unplanned depreciation posting in tax depreciation area. But for the business case of unplanned depreciation posting in book depreciation area, it's not explained.

IMG node: Financial Accounting -> Asset Accounting -> Depreciation -> Unplanned Depreciation -> Specify Transaction Types for Unplanned Depreciation.

Other Terms

Japan, Localization, Annex16, FI-AA, RAIDJP_ANNEX16, IMG documentation

Reason and Prerequisites

This SAP Note will update this IMG document to explain the business case of unplanned depreciation posting in book depreciation area.

Solution

Apply the corresponding support package or implement the correction instructions in this note.
