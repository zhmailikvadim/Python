Symptom

Runtime error BEHAVIOR_ILLEGAL_STATEMENT is raised in program SAPLEBNE when creating a third party sales order using app Import Sales Orders (F3893).

Environment

  • SAP S/4HANA 2021 and 2022

Reproducing the Issue

Open app import sales orders and try to create a third party sales order from an excel file.

Cause

Program error.

Resolution

Implement note 3269668.

Keywords

F3893, import sales, import order, BEHAVIOR_ILLEGAL_STATEMENT, SAPLEBNE, third party
