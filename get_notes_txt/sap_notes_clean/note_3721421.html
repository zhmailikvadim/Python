Symptom

The Dynamic Selections UI displays a set of fields, that are grouped and denoted by a label field. There may be different field groups and group labels on the screen. 
For accessibiliy requirements it is required that each group label, which is only a text label from a technical point of view, is tagged as heading, so that appropriate UI addon plugins (e.g. "headingsmap" for Chrome) are able to recognize and display it as heading.

Reason and Prerequisites

Issue in ABAP code. A correction is required.

Solution

Apply the provided correction or upgrade to the support package level, that is assigned to this note.

Please note that additional patches or versions of the SAP kernel are necessary to make theses changes become effective: note

3727167 DynproTextField with custom attribute heading

is required on kernel level.
