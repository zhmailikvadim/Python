Symptom

During the implementation of SAP Note 3702483, the system raises an error indicating that the field "GC_TDOBJECT_SB" is unknown.

Environment

  • SAP ERP
  • SAP S/4HANA
  • SAP Document and Reporting Compliance, Cloud Edition
  • SAP S/4HANA cloud, Private edition

Reproducing the Issue

  1. Launch Tx code Snote
  2. Try to implement SAP note 3702483 and the error occurs

Cause

Missing code correction

Resolution

Implement SAP Note 3618083 – Electronic Document Processing for Poland – KSeF Self-Billing to resolve the issue.

Note: Before implementing the note make sure to follow the sequence mentioned in Overview Guide - 3279206 - Electronic Document Processing for Poland - KSeF: Implementation Overview
