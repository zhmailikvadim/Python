Symptom

The statistics according to Sec. 163 Para. 2 SGB IX for severely challenged persons generate the error message "Leaving date & is before validity start date of certificate &".

Environment

The log of the report RPLEHAD3 lists the error message mentioned above under the section 'Consistency Checks' for a personnel number.
The affected personnel number has a leaving and reentry in the company.

Cause

The message occurs if the characteristics LEAVE and ENTRY are not coded correctly.

Solution

To ensure that reentries are processed correctly, an assignment with the report class CSEH to the return value REENT is required for both features ENTRY and LEAVE.
In this assignment, a query should be defined for the return value REENT, which has an empty return value. The return value must not be defined with 'X'.

Example:

ENTRY Rule for Determining Entry Date
      > MOLGA Country Grouping
          > 01 Germany
             > RCLAS Class of requesting report
                 > CSEH
                     > REENT
