Symptom

There are values maintained in the following key figures to consider Inter-Subnetwork Production Demand:

  • DEPCOMPDEMNONDELIVCOSTRATE

  • DEPCOMPDEMLATEDELIVCOSTRATE

  • DEPCOMPDEMLATEDELIVMAX

However, after running the optimizer, these key figure values are not being considered.

Environment

SAP Integrated Business Planning for Supply Chain (SCM-IBP)

Cause

The issue is caused because the additional parameter USE_PRIORITIZATION_OF_DEPENDENT_COMPONENT_DEMANDS is either not set to Yes or is maintained incorrectly (for example, with a typo or extra space). Due to this, the optimizer does not consider the related key figure values.

Resolution

To ensure the following key figures are considered:

  • DEPCOMPDEMNONDELIVCOSTRATE

  • DEPCOMPDEMLATEDELIVCOSTRATE

  • DEPCOMPDEMLATEDELIVMAX

the additional parameter USE_PRIORITIZATION_OF_DEPENDENT_COMPONENT_DEMANDS must be set to Yes in the relevant operator profile (in the S&OP Operator Profiles app).

This is well documented in SAP Help: Subnetworks and Production Sources - Production Receipts (Optimizer/Finite Heuristic)

Also, please make sure there are no typos in the additional parameter name. If there is any mistake (for example, an extra space in the parameter name), the optimizer will not consider these key figures.
