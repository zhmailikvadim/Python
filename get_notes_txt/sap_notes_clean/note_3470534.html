Symptom

Cannot edit Group currency filed of the consolidation version in configuration Define Versions.

Environment

S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Open SSCUI Define Versions. 
  2. Choose wanted version, for example Y10.
  3. Click Group Currency field and cannot change it.

Cause

Certain settings can only be set once when you create the version in the Create Version from Reference Version configuration activity. Afterwards, you can no longer change these settings in the Define Versions configuration activity. These settings are the following:

  • Version Type
  • Reference Version
  • Group Currency
  • Run Mode
  • Allow LC, TC, Qty Changes
  • Fiscal Year Variant

Resolution

This symptom is expected action based on system design.

If a different group currency is required, this must be handled by creating a new version or an extension version, rather than modifying an existing one. For additional background, please refer to the documentation on Consolidation Versions.

For more information, please check Consolidation Versions.
