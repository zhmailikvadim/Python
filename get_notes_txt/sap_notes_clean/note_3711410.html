Symptom

Header text maintained in the Stock Transfer Order is not automatically copied to the Outbound Delivery, even though the respective text IDs and their corresponding access sequences have been maintained in the following configuration activities:

  • Define Texts for Purchase Order (102270)
  • Define Text Types for Sales and Distribution Document Header (105684)
  • Set Up Text Determination for Delivery Header (101291)

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Configure the system to enable copying of Stock Transport Order purchase order header text to the outbound delivery header text.
  2. Maintain the Stock Transport Order header text. 
  3. Create a Stock Transport Order with the maintained header text.
  4. Observe that the header text from the Stock Transport Order is not automatically copied to the outbound delivery header text.

Cause

Text IDs other than F06 are maintained in Define Texts for Purchase Orders (102270).

Resolution

Review the configuration for Access Sequence for Text IDs and ensure that the correct text ID, F06, is maintained, then retest the scenario to confirm that the header text is now copied automatically.
