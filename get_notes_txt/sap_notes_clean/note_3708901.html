Symptom

  • When inserting a table into the newly created optimized story, story designers would automatically work in the new table build experience since SAC 2026 QRC1 update.

Environment

  • SAP Analytics Cloud 2026.2

Resolution

  • Since SAC 2026 QRC1 update, users would automatically work in the new table build experience when creating a new story and adding a table widget.
  • The same applies when converting a classic story with tables to the optimized story experience.
  • For existing stories, users can decide per story if it is willing to use the new table build experience or keep the current one. 
  • For detailed information, please refer to help document New Table Build Experience | SAP Help Portal.
