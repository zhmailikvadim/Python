Symptom

You are using ABAP Data Integration to replicate the data of S/4 HANA data products into SAP Business Data Cloud and the respective data product is based on SQL service.

In the application log you can find an incorrect error from the background job /1DH/PUSH_CDS_DELTA:

"CDS view XXX is not enabled for data extraction"

Other Terms

S/4HANA, ABAP Data Integration, SAP Business Data Cloud, APRS, CDC, Delta Reconstruction, DHADM, transfer job

Reason and Prerequisites

The ABAP-based source system is an S/4HANA 2025 FPS01 system.

Solution

Implement the corrections in the ABAP-based source system.
