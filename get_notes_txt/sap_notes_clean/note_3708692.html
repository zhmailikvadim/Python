Symptom

When using transaction code /US4G/EA_GEN to generate energy amounts, the system cannot generate energy amounts for a simple Malo if energy amounts have already been generated for another complex Malo within the same structure.

This issue is relevant for the market role: MOS.

Other Terms

/US4G/EA_GEN, complex structure, energy amount generation

Reason and Prerequisites

The error occurred because check for the existence of energy amounts was not restricted by installation or Malo.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the system can generate energy amounts for each Malo within a complex structure. You need to generate the energy amount again with the meter reading document. You can verify the fix by checking the application log, which indicates that energy amount documents were created with meter reading documents.
