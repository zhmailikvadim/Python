Symptom

Role descriptions in SAP Access Control are not getting updated or reflected in Role Search despite running the Repository Object Sync job and even after using a custom update program.

Environment

SAP Access Control 12.x

Reproducing the Issue

  1. Schedule a regular Repository Object Synch job
  2. Check in ROLE SEARCH description - no description updated

Cause

Resolution

After changing description, in the backend SAP system (e.g PFCG), you must re-import the role into SAP GRC. 

Also please have a look at this Note 2836605 - Is there an automated process to update composite roles in GRC?

See Also

Refer to: How to enable the Role Description field in the Risk Analysis Result tab.
