Symptom

Due to restricted authorization, user can run tasks only on specific consolidation units in Data Monitor app. Nevertheless, all consolidation units are disclosed in Data Monitor app, after Consolidation Unit is set to blank in Set Global Parameters app.

Environment

  • SAP S/4HANA Cloud Private Edition, SAP S/4HANA Finance for Group Reporting
    or
  • SAP S/4HANA Cloud Public Edition, SAP S/4HANA Cloud Finance for Group Reporting

Resolution

Designed system behavior: Despite restricted authorization, all consolidation units are disclosed in Data Monitor app, after Consolidation Unit is set to blank in Set Global Parameters app.
