Symptom

  • Using the standard template, it is not possible to delete contact relationships for customers using the mass process functionality.
  • Attempting to delete relationships using the "Manage Mass Process" feature does not yield the expected results.

Environment

  • Master Data Governance, Consolidation and Mass Processing
  • Master Data Governance on S/4 HANA on-premise 2022
  • Master Data Governance on S/4 HANA on-premise 2023

Reproducing the Issue

  1. Open Fiori App 'Manage Mass Processes - BP Relationships'.
  2. Use the standard template to create a mass deletion process for contact relationships.
  3. Attempt to execute the process to delete the relationships.

Cause

  • When using a file, start the process then continue to the edit step for the application to read the file

Resolution

  1. Use the app 'Export Master Data - BP Relationships'.
  2. Prepare the input file with the required columns: BUT050-PARTNER1, BUT050-PARTNER2, and BUT050-RELTYP.
  3. User the app "Manage Mass Processes - BP Relationships'.
  4. Click on Create > Mass Deletion
  5. Start the process and select the file to be used.
  6. Hit the continue button to get to the Edit step and then the application will read the file.
