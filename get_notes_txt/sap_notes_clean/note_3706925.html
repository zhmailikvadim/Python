Symptom

Navigation map elements linked to Signavio diagrams or external content, display irrelevant hints ('Image' or 'Link (deprecated)') when hovering with the mouse cursor.

Environment

Navigation Map

Resolution

The elements of background and link stencil have been replaced with the image stencil and are therefore marked as deprecated. While existing models that utilize these deprecated stencils will continue to function, new creations will no longer be possible as part of the improvements.

Keywords

SOE, Link (deprecated), Navigation map
