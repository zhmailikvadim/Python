Symptom

After value input performed in the 'Raw Material' field, this is immediately set as read-only, not allowing further modifications (ex. if the value used for initial input is incorrect)

Other Terms

IS-DRY, Manage Raw Material Deliveries, read-only input field, Raw Material, SAP Dairy Management by msg

Reason and Prerequisites

Program Error

Solution

History:

Version 1 of January 19, 2026: New creation
Install the code correction.

Description of Correction:

In the method EXECUTE of class /DRY/CL_D_I_DELIVERYMATERIALTP, the logic for setting the 'Raw Material' field as read-only has been enhanced to allow that only for active entities (now permitting the modification of the field value in case of incorrect/inexistent materials used as input data).

Functional Dependencies:

There are no functional dependencies.

Manual Changes:

There are no manual activities.
