Symptom

Once a marketing document is added and a journal entry is created, the line-level project assignment becomes locked and cannot be removed or changed anymore.

Reproducing the issue

Prerequisites:

Enable Project Management is selected in Company Details (follow path Administration → System Initialization → Company Details → Basic Initialization).

Scenario:

 

  1. Create Project FinnProj A with one stage under Project Management → Project.
  2. Create an A/R Invoice with one Item line, specify FinnProj A and Stage in Line Item.
  3. Go to Financials → Project Management → Project → FinnProj A Project.
  4. Mark the Stage and remove the marketing document from Document area. 
  5. Go back to A/R Invoice.

Expected behavior:

The FinnProjA is removed.

Actual behavior:

The FinnProjA is not removed in the A/R Invoice line.

Solution
