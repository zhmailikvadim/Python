Symptom

The user needs to activate the API debug function in API Portal but when the API Debug Viewer screen is appearing the following error is raised in a pop-up window:

Cause

  • API Management
  • SAP Integration Suite

Resolution

Please ensure that the user(s) performing the debugging have only the APIPortal.Tester role collection assigned. This error can occur if a user(s) has both APIPortal.Tester and APIPortal.Guest (or any other/further) role collections assigned.
 
For more information on the available role collections, see the SAP Help page: Comprehensive Breakdown of Role Collections in API Management
