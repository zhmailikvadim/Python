Symptom

  • CDS view I_INVENTORYPRICEBYKEYDATE shows unexpected inventory price behavior.
  • Even when a past key date is provided, the system is picking the current inventory value.
  • This issue is observed only for some materials, not all.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Access the SAP S/4HANA Cloud Public Edition system.
  2. Navigate to the I_INVENTORYPRICEBYKEYDATE CDS view.
  3. Specify a past key date for a material.
  4. Observe that the system displays the current inventory price instead of the expected inventory price for the specified key date.

Cause

Resolution

This should be standard behavior as explained in I_InventoryPriceByKeyDate 
This CDS view provides the inventory price determined for all materials at the end of a fiscal period related to the key date.
If the end of the fiscal period has not been reached yet, the CDS view specifies the price determined for the current date.
