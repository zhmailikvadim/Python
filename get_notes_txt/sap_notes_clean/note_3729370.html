Symptom

When using the SAP Build Apps Standard plan to deploy an application to SAP Work Zone, the Visualization screen is not available. As a result, you are unable to configure essential settings such as Title, Subtitle, Additional Information, Icon, Supported Devices, and Launch Behavior for your application.

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

SAP Build Apps

Reproducing the Issue

  1. Open SAP Build Apps
  2. Select the Project which should be deployed
  3. Select Publish
  4. Select Build and Deploy
  5. Choose Create Configuration
  6. Choose SAP Build Work Zone

Cause

SAP Build Apps Standard plan users get a different Work Zone configuration screen that doesn't include that section.

Resolution

Visualization settings like title, icon, etc., those settings are a feature that's only available on the build-default plan.
