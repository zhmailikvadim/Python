Symptom

1. If you call a loss group and select the assigned losses, a dump occurs.

2. If you call a loss group, go to "Loss Assignment", and choose "Assign Loss", a dump occurs and the loss cannot be assigned to the loss group.

Reason and Prerequisites

Cause: A new field has been added to the loss group.

Solution

History:

February 23, 2026: New

March 04, 2026: Clarification of the solution

10. March 2026: Release 809 added to SAP Note validity

Troubleshooting Description:

1. When you call a loss group and select an assigned loss, a dump does not occur.

2. If you call a loss group, choose "Loss Assignment", and choose "Assign Loss", the system does not issue a dump and you can assign the loss to the loss group.
