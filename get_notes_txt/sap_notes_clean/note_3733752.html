Symptom

  • A pop-up warns: "The SSL certificate for your career site is expiring in X days... go to Career Site Builder > Tools > SSL Certificates and start the certificate renewal process..."
  • A banner on the same page states: "You're using a standard site URL, so the system manages the SSL certificates for you. If you don't need a custom URL anymore, you can let the certificates expire."
  • In Settings > Site Configuration > Site URL, only Standard Site URL is in use.
  • Confusion arises due to the conflicting messages.

Environment

Successfactors Recruiting Marketing

Reproducing the Issue

  1. Navigate to Career Site Builder > Tools > SSL Certificates.
  2. Observe the pop-up about the SSL certificate expiration.
  3. Check that the SSL Certificates section is empty.
  4. Go to Settings > Site Configuration > Site URL and confirm that only Standard Site URL is in use.
  5. Compare the conflicting pop-up and banner messages.

Resolution

Customers can ignore the pop up since the SSL management is done by SAP and that this will be addressed in a future release.

Disable 'Enable SSL Notification' to stop sending the SSL renewal reminders EMAIL notifications to users 

If you do not wish to see the pop up, you can click 'Do not show this message again' option.
