Symptom

  • An application error occurs when attempting to update successors for a specific position in the Succession Org Chart.
  • The error message displayed is: "An application error occurred. Please try again later, or notify support if the issue keeps occurring."

Environment

SAP SuccessFactors Succession & Development

Reproducing the Issue

  1. Log in to the system.
  2. Navigate to the Succession Org Chart.
  3. Search for the position.
  4. Attempt to update the successors for the position.
  5. Observe the application error message displayed.

Cause

One or more users nominated for the affected position was already terminated (inactive in the system), which led to the error. 

Resolution

  1. Enable the following Provisioning setting:
    "Succession Management: Hide inactive users from the list of successors in the org chart and omit them from new form creation (auto-population of previous successors)."
    This ensures inactive users are hidden in the Succession Org Chart.

  2. Verify via the JDM Check Tool whether any error is present. If the tool indicates the error "Job codes were active and valid", review the job code mapping and resolve any underlying inconsistencies.

    2.1 Confirm whether the Job Code should remain active. If it should be active, associate (tie) the Job Code with an active user to reactivate it.

    2.2 For the remaining Job Codes that should be inactive, execute the Quick Fix to correct the inconsistency.
