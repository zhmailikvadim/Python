Symptom

After currency translation (CT) task there is only message in Task Logs application instead of expected calculated numbers:
The message is "No data records to be computed found".

Environment

SAP S/4HANA Finance for Group Reporting

Reproducing the Issue

Run CT task for period and unit combination with entries that have posting level higher than 10 in table ACDOCU.

Cause

Currency translation task run entries are equal and lower than posting level 10.

Resolution

Designed behaviour. 

Keywords

CT, Currency translation, posting level, posting levels, G0453, G0 453, 453
