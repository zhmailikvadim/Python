Symptom

You may come across an issue where when a settlement for a calculation run is executed in the background, the settlement log is not captured on the calculation run.

Other Terms

Calculation run, settlement, logs, final calculation mode, background mode, batch mode, /IRM/IPSIPCRCMP

Reason and Prerequisites

This is caused by program error.

SAP Note #  3073706 is available for 60E Releases.

Solution

Please implement the correction instructions attached. These corrections are only valid if you have applied the Support Packs.

Release V4 - 1610 Base Package, 1709 Base Package or Support Pack 1,1809 Base Package or Support Pack 1, 1909 Base Package, 1910 Base Package,2020 Base Package.

 Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object.
