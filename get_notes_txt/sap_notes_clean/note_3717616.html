Symptom

On the results list in the Manage Waste Transfer Requests (F6084) and My Waste Transfer Requests (F6083) apps, multiple entries with the same waste transfer request ID are displayed.

When you navigate to the object page of these entries, no data is shown.

This issue occurs after you create a new transportation document based on a transfer request that was previously assigned to another transportation document that has been cancelled.

Reason and Prerequisites

Reason: Program error.

Prerequisites: The validity of the correction is maintained in the correction instructions.

Solution

The support packages that contain this correction are listed in the "Support Packages" section.

Alternatively, you can implement the correction instructions of this SAP Note.
