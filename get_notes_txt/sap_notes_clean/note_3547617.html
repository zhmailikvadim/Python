Symptom

CheckMan error in transaction SXI_MONITOR

Other Terms

CheckMan error, CHK_TSTC, SXI_MONITOR_EXTERNAL, 0007.

Reason and Prerequisites

You use SAP Exchange Infrastructure as a central Integration Server or as a local Integration Engine in an application system. The following Check Manager error occurs for transaction SXI_MONITOR:

Check   Rule Message
CHK_TSTC 0007﻿ There are unmaintained default statuses

These CheckMan errors do not affect the runtime.

Solution

Import the specified support package.

After you implement the corrections, the specified Check Manager error is corrected.
