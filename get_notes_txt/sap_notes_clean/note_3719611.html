Solution

What’s New for SAP GRC Process Control for SAP HANA (Support Pack - 00)

Improved User Experience: You can now use the following Fiori Apps for enhanced usability and accessibility.

  • My Compliance Tasks  

Displays different OVP cards as follows:

    • Control Effectiveness Test - showing Test Control Effectiveness count by status done or not done
    • Organizations in Evaluations - showing evaluations of top 5 organizations
    • Due Dates - showing evaluation based on due dates
    • Due Date Status - showing evaluation based on status (over due or not due)
    • Evaluation Results - showing evaluation based on issue priority

Separated evaluation based on type, that is, Test Effectiveness, Design Assessment, Self Assessment, and Manual Control Performance Result

  • Manage Controls
    • Display list of Local control and Central Controls based on Timeframe and Year
    • Filter them based on name, regulation, and so on.
    • See all details related to control.
    • Display different sections such as General, Roles, Regulations, Control Performance, Control Requirements, Common Control Objectives, Evaluations, Related Objects, and Attachments and links
    • Change timeframe - allows to show details for selected timeframe
  • Manual Control Performance 

    • Display list of Manual Control Performance which are in process or review assigned to a user

    • Filter the Manual Control Performance based on different filters, such as Organization, Subprocess, Controls, Status, and so on
    • Edit the detailed information of Manual Control Performance.
    • Maintain the data related to step such as Evidence or comments
    • Display the all steps which are done or pending in the Manual Control Performance object page
    • Report Adhoc issue button at step level
    • Upload attachments and links
  • Manage Assessments

    • Display list of assessments (design assessments and self assessment), which are in draft or review assigned to user

    • Filter the assessment based on different filters such as Search(name), Organization, Subprocess, Controls, Regulation, and so on
    • Edit the detailed information of assessment
    • Maintain the data related to questions, overall rating, and evidence
    • Display the related underlying and parent objects like risks and account groups
    • Report issue based on rating
    • Upload attachments and links
  • Import Control Requirements

    • Import Confirmed Control Requirements from Regulatory Insights

    • Map Regulatory Insights Framework in Process Control Regulation and Timeframe
    • You can import Control Requirements many times depending on its acceptance in Regulatory Insights
  • Manage Control Requirements

    • Manage Control Requirements Imported from Regulatory Insights

    • Add Existing Central Controls, Local Controls and Policy documents
    • Quickly create Central or Local Control with Mandatory Attributes and Map it to Selected Control Requirement
    • Get Intelligent Control Proposals and quickly add them to Control Requirements
  • Intelligent Self-Diagnostic Cockpit

    Intelligent Self Diagnostic Cockpit analyses failed monitoring jobs intelligently. It identifies reasons behind the failure. The app analyses Collects and Processes Logs from SLG1, Background Job Logs SM37, Runtime Errors such as ST22 from GRC, and Connected Remote systems. It then classifies these errors into one of the following categories:

      • Self-Correctable – These are the jobs, which can be corrected by the Intelligent Self Diagnostic Cockpit without any manual efforts.
      • Manually-Correctable – Groups the jobs based on unique errors and already available Correction Notes, FAQ Documents, and KBAs, which could potentially resolve the errors.
      • Others: Remaining jobs are placed under this category. Users get visibility of impacted processes and can try to find the information themselves.
    • Manage Issues

      • The new issue analytics application utilizes predictive analysis and SAP HANA capabilities to manage and analyze the unstructured data available in the form of issues in your system. The application thereby gives you a visual representation of all the issues by various parameters, and most importantly, the most used words in these issues.

      • Issues can be filtered based on Year, Organization, Status, and so on
      • Display different sections such as Header, General Information, Additional Information, Remediation Plan
      • Assign Remediation Plan or void action can be executed for the issue

    Enhanced Integration: You can now leverage the SAP Integration Suite and Framework for seamless connectivity. New sub-scenarios for Integration Suite enhance connectivity with SAP Integration Suite.

    Flexible Workflow Management: You can now adapt to business needs with customizable workflows.

    Regulatory Alignment: Now, you can integrate with Regulatory Insights for compliance and oversight.
