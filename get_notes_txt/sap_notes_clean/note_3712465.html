Symptom

  • Improve performance for Related Documents in Solution Documentation.
  • Filter out finished (also with status withdraw) Change Documents.

Other Terms

Related Documents, Change Document, SBOM, DOS, CD, Change Control, SOLDOC

Reason and Prerequisites

Low performance because of program error and not using optimal functionality.
Functionality retrieves finished Change Documents (Status with FINI).

Solution

Implement the current corrections of the SAP Note with transaction SNOTE.
The performance in the SOLDOC is improved if you are using many Change Documents for this solution.
Also finished Change Documents are not more counted.
