Symptom

When you execute the background program "Carry Reserves Forward to Key Date", processing terminates and the system issues the error message "Error in function /MSG/R_A_CFWD_FIND_ALL_REC".

The error occurs both in the Basic System and in Risk Manager when reserves are carried forward to the key date.

The error does not occur when reserves are carried forward to the target year.

Reason and Prerequisites

The following data constellation is available in the system:

The 100% account to be carried forward, A1, and the SL adjustment posting to be carried forward, A2, have a different financial year:

A1 in accounting/financial year = X / X - 1

A2 in accounting/financial year X / X

The reserves are carried forward to the key date that is in the target year X + 1.

Solution

The background program "Carry Reserves Forward to Key Date" now ends without any error messages and generates the expected accounts.
