Symptom

'SELECT VAWK' and 'SELECT VARK' statements with a long runtime can be observed in the ST12 trace file.

Environment

  • SAP ERP Available-to-Promise
  • SAP R/3
  • SAP ERP Central Component
  • SAP ERP
  • SAP Enhancement package for SAP ERP
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA

Reproducing the Issue

Perform route scheduling functionality.

Cause

The selection of route schedules were not designed to handle a high amount of entries efficiently (such as multiple 100.000). Due to that, selection time can be longer.

Resolution

  1. Please try to reduce the number of entries in VAWK and VARK.

  2. SAP offers modifications which can shorter the selection. Please consider SAP Note 170183 - What to consider when applying modifications before applying any modification since they are not covered by SAP Standard Support.
    SAP Note 1235113 - Performance problems table VARK and invalid route schedule selected
    SAP Note 1099443 - Performance problems table VAWK
    SAP Note 948002 - Performance problem table VAWK

Keywords

Select VAWK, Select VARK, VAWK, VARK, route schedule, performance, modification, ST12, rescheduling
