Symptom

There should be a possibility to specify default Yard for newly created users.

Other Terms

/SAPYL/CL_HELPER_BASE GET_ESDUS_VALUE
/SAPYL/CL_HELPER_BASE GET_VBI_BIN_SUBSET
/SAPYL/CL_HELPER_BASE GET_YARD_NO
/SAPYL/CL_UI_UTILS GET_DEFAULT_VALUES

Solution

BAdI /SAPYL/ESDUS_VALUE provides the ability to set default values for new users. In the standard BAdI initialization, the Yard number is determined from user-parameter /SAPYL/DEFAULT_YARD.
