Symptom

The SAP S/4HANA migration cockpit is a tool designed for Customers who want to transfer their business data from SAP or non-SAP software systems.

This tool delivers preconfigured content with automated mapping between source and target - it means if Customers' need matches the migration objects available, a tool does not have to be built from scratch - it is all ready to use, reducing the time effort of data upload.

VSS Data Migration Cockpit is a project accelerator which is delivered by the means of standard SAP Transport files.
Please notice that SAP S/4HANA migration cockpit (which is the case here) should be only used for initial data import for new systems and not as everyday used tool for data creation/update.
