Symptom

In Rail Freight Order's Items tab, after chancing the hierarchy view to Load Plan, custom-made hierarchies are no longer available in the hierarchy dropdown list.

This correction includes a rail-specific copy of the Load Plan view's WDCC that can be added to the custom hierarchy with transportation mode "rail".

Other Terms

WDCC, UIBB

Reason and Prerequisites

Configuration Error. The Load Plan was developed for road scenario.

Solution

Implement the attached correction instructions or install the corresponding Support Package.
