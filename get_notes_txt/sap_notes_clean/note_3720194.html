Symptom

When comparing the completion date time between User Learning Plan Audit Report and the content completion details under View Content Object Details, a few seconds difference is perceived.

Environment

SAP SuccessFactors Learning

Reproducing the Issue

  1. Access SuccessFactors Learning >
  2. Launch and complete an item >
  3. Access Learning Administration >
  4. Navigate to People > Users and search for the user >
  5. Go to the Completed tab and search for the item >
  6. Click on the three dots and select View Object Details >
  7. Click on the content object >
  8. Take note of the 'Complete as of' date and time >
  9. Go to Reports and select User Learning Plan Audit report >
  10. Add the filters and run the report >
  11. Observe a few seconds difference in the completion date between the report and View Object Details.

Resolution

The completion date discrepancy occurs because the data is sourced from two different database tables. The completion date on the User Learning Plan Audit report is determined after evaluating the status of all content objects, while each content object’s completion date is recorded based on its individual content communication.
