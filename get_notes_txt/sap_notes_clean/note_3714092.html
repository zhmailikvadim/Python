Symptom

Customer believes there is a 15-user limit for WebIDE and requests to increase the number of users for their tenant.

Environment

  • SAP Incentive Management
  • Incentive Management on Legacy HANA 
  • Incentive Management on Hyperscalers

Cause

  • There is no hard-coded limit for the number of WebIDE users in HANA DB.
  • The perceived limitation is due to infrastructure constraints: as the number of users increases, especially with simultaneous connections, the system requires more CPU and memory resources.
  • Performance issues may arise if the infrastructure cannot handle the increased load.

Resolution

Confirmed that there is no specific user limit for WebIDE; more users can be added.

Keywords

WebIDE user limit, HANA DB, resource consumption, CPU memory
