Symptom

A dummy order is created in case request agent specific errors occur during CAMT.029 file import. However, this behavior can affect overall file import and might lead to manual efforts.

Other Terms

Dummy Order, Transaction not found, Request Agent.

Reason and Prerequisites

Program error.

Solution

Preliminary correction according to the correction instructions included in this SAP Note or via Support Package.

In detail, request agent processing errors are not transferred to file import and request agents with state indicator error will be created instead.
