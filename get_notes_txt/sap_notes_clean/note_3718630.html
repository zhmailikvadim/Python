Symptom

When you run or simulate a BOP variant, it doesn’t select any documents.

Environment

  • S/4HANA Retail & Fashion Management (LO-RFM) 
  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Open the ARUN application/transaction for BOP processing.
  2. Enter/select the identified BOP variant.
  3. The ARUN does not select any document.

Cause

The BOP variant or BOP segment isn’t maintained correctly.

Resolution

Verify that a BOP Segment is maintained for the document, and that a BOP variant is maintained for the same document type.

Keywords

BOP, variant, segment, document type, ARUN, STO, SO, fashion, allocation run
