Symptom

The following issues are fixed for processing graphical display:

  • In edit mode of process configuration, after you changed some steps and press SAVE button, the graphical display button sometimes disappeared.
  • You display any processing graphical display by pressing the graphical display button. Close the graph and switch it to change mode and change some steps or step precondition. Then save the changes. Press the graphical display again, at this time the graph still shows the old configuration.

Other Terms

IDXPF, graphical display, /IDXPF/CL_UI_PDOC_GRAPH, GET_PROCESS_CONFIG

Reason and Prerequisites

Program error.

Solution

Implement the SAP Note or install further Support Package.

Note:

This SAP Note does not directly address a customer incident and, therefore, does not have to be implemented.

Changes made to the associated code and DDIC may impact your existing implementations. You can defer installing this SAP Note if it is not required for your implementation, and instead wait until the next service pack is available that will implicitly include the associated corrections.

If you have installed an associated add-on in your system, such as IDEX-AT or IDEX-UK, and the add-on may also be impacted by this SAP Note, we strongly recommend that you contact your SAP IDEX add-on maintenance team to find out whether installing this SAP Note will cause any disruptions.
