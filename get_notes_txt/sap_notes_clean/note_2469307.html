Symptom

You are ordering the shopping cart with a catalog item, but during the follow on document creation you get the error: Shopping cart XXX: M3 351 Material XXX not maintained in plant XXX

Environment

  • SAP Supplier Relationship Management (SRM)
  • SAP enhancement package for SAP Supplier Relationship Management

Reproducing the Issue

  1. Create a shopping cart with catalog items referring SRM product ID
  2. Order the shopping cart

Cause

  • Product with local logsys will not be interpreted in the backend

  • In the SPRO catalog customizing, the field logical system should be empty or populated by the SRM system ID

Resolution

  1. Maintain the product according to the target system.
  2. Maintain the catalog customizing logical system field empty, so that the logic can choose the right system from the received data or from material master maintained in SRM.  
    This can be done in the following SPRO path: SRM Server >> Master Data >> Content Management >> Define External Web Services  >>  Double-click the relevant Catalog to open the Details tab >> set the Logical System field to empty.
