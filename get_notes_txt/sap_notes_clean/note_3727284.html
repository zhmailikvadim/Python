Symptom

  • I am updating the Work Order Owner from Actions > Edit Work Order Details
  • After selecting a new Owner, the Site and Location fields are getting cleared out
  • Errors are generated: Site is required. Location is required.

Reproducing the Issue

  1. Navigate to the work order
  2. Click Actions > Click Edit Work Order Details > Click Edit Details in the top section
  3. Select a new Work Order Owner from the drop down menu
  4. Observe that the Site and Location fields are cleared out and the following errors display: Site is required. Location is required.

Cause

  • The supplier has not defined an Account Manager for the site associated with the work order
  • If no Account Manager is assigned, the site will not display on this page

Resolution

Ask the supplier to follow the steps in KB0619017 - How can Supplier assign an Account Manager to a Site

KB0619017 - How can Supplier assign an Account Manager to a Site
