Symptom

The system generates an incorrect Block 51 for the entire month when an employee returns from suspension during the month. This results in incorrect Block 53 Unit 40 transfers that appear as duplicates in the monthly DSN.

Other Terms

DSN, Block 51, Block 53, suspension, inactive periods, ACTI table, WPBP

Reason and Prerequisites

This issue occurs when the system processes suspension through inactive periods and has entries in the ACTI table for Block 53 Type 02 "Durée d'absence partiellement ou pas du tout rémunérée" on the inactive WPBP split.

Solution

After implementing this SAP Note, the system correctly generates Block 51 and Block 53 Unit 40 transfers for employees returning from suspension during the month.

The correction in this SAP Note is part of the HR Support Packages referenced below.
If you need the correction before the HRSP is available, implement this SAP Note using the attached correction instructions. Correction instructions are a short-term solution and not a substitute for the Support Package.
