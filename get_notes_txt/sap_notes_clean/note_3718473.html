Symptom


--- Description ---

When starting transaction WRF_DIS_SEL or WRF_DIS_MON, the resulting list shows 3 columns with article status.

The 'Site-specific' article status, reporting field MMSTA from table MARC is never shown.

The report uses structure WRF_DISCONT_POS_ALV_STY, field MMSTA.

Reason and Prerequisites

Missing information on result screen

Solution

Implement the following correction instructions

Other Terms

SAF-T Bulgaria monthly file, Bulgaria, BG, SAF-T, Tax Reporting, Audit, Standard Audit File, Legal Change, SAFT, Monthly File

Reason and Prerequisites

Incorrect structure of the XML file.

This SAP Note is only relevant for Bulgaria.

You have implemented the prerequisites listed in SAP note 3702420 - SAF-T Bulgaria: Collective Note (SAP ERP).

Solution

As a general rule, SAP recommends that you install a solution by applying a Support Package. However, if you need to install a solution earlier, use the SAP Note Assistant to implement the correction instruction. You can find more information about the SAP Note Assistant in SAP Service Marketplace, under service.sap.com/note-assistant.
