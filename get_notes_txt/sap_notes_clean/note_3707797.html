Symptom

  • Error message 'Transaction /SAPSLL/SPL_CHSS1 cannot be performed (SAP Note 2476734)' appears after attempting the following transactions: 
    • SAPSLL/SPL_CHSS1
    • SAPSLL/SPL_CHSS2
    • SAPSLL/SPL_CHSS3

Environment

  • Global Trade Services (GTS)
  • GTS Edition for HANA

Reproducing the Issue

  1. Navigate to transaction SAPSLL/SPL_CHSS1.
  2. The following error message appears:  

    • Message No. 00977
      Transaction /SAPSLL/SPL_CHSS1 cannot be executed (SAP Note 2476734)
      The execution of the transaction was disabled by a system configuration (the blacklist monitor).
      Avoid using this transaction or contact your system administrator.
      For more information about the blacklist monitor, see SAP Note 2476734.

Cause

  • The simulation SPL screening transactions are blocklisted in the backend system following the migration to S4 in SAP GTS, as outlined in SAP Note 2476734.
  • They are not compatible with the requirements of the updated data models in S/4HANA and may generate incompatible or disruptive data.
  • These transactions are outdated. 

Resolution

  • Transactions for simulation of sanctioned party list screening are no longer supported. Refer to 2234168 - Blocklist Monitor not started or has to be restarted - message P013(TG) for additional information.  
  • Transaction SAPSLL/SPL_CHSS3 has been replaced by the app "Check General Addresses".
  • For Transactions SAPSLL/SPL_CHSS1 and SAPSLL/SPL_CHSS2 there is no replacement, as there is no longer a business requirement for them and no replacement is planned. 
  • These transactions will remain unsupported, even if they are removed from the blocklist monitor.

See Also
