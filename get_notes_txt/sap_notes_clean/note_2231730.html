Symptom

Improvement Request:

Scheduling in the network graphic screen.

Request Reason:

Scheduling option is inactive in the network graphic if more than a single network is available in the network graphic screen.

Other Terms

Scheduling, CJ20n, Network graphic, LNWGRF36, LNWGRF1H

Reason and Prerequisites

Advance development.

Solution

Improvement:

To enable the scheduling option if more than a single network is available in the network graphic screen, activate the BAdi PS_FUNCTION_SWITCH and return SWITCH_VALUES-GRPH_NETW_SCH_FLG as 'X'.

Benefit:

Network scheduling is possible when

* You group the order according to the WBS (not the WBS scheduling).

* You group the order according to the work center

Limitation:

Only one network can be scheduled at a time in the network graphic
