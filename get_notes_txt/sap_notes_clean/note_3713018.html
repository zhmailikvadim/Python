Symptom

The goal is to release only groups allocated at 100%. Accordingly, the grouping rule is assigned to the release check variant “OTC_MAYORISTAS,” with a corresponding 100% check at the Document Item level. However, the release check at the item group level is not working.

Environment

  • S/4HANA Retail & Fashion Management (LO-RFM) 
  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Cause

In the release check variant OTC_MAYORISTAS, the determination logic is set to Always use default. With this setting, the system always uses the Default release rule configuration.

If the order has fully allocated quantity, the minimum 100% threshold is met, and status F is set for items according to the release rule at the Document Item level.

Resolution

Please verify whether the ARUN status of the items is set to F in ARUNITA.
