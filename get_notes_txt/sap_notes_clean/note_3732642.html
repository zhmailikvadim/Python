Symptom

  • Users, Teams, Roles list would appear as blank or loading status in us10 SAC systems
  • "Sorry, we can't find this file" would appear when accessing model and public dimension tables in SAC systems
  • In har file, observe some error messages would appear on request 
    https://sapui5.hana.ondemand.com/1.120.2/ 

Environment

  • SAP Analytics Cloud 2026.2.10

Reproducing the Issue

  1. Log onto SAC
  2. Navigate to Security-> Uses or Security-> Teams or Security-> Roles
    => Observe that the page would be blank or would be always in a loading status
  3. Navigate to Modeler and try opening any models or public dimension table
    => Observe that the error message "Sorry, we can't find this file" would appear

Cause

The issue is caused by a defect in the product.

Resolution

This issue has been fixed.
