Symptom

  • Error: "You are not authorized to query the remote system. Please ask your administrator to grant you the InA role."
  • SSL error occurs when creating or testing the live data connection (tunnel).
  • Cloud Connector error in logs:
    • +0000#ERROR#com.sap.core.connectivity.tunnel.core.impl.processing.channel.AbstractChannel.ChannelWriteFutureListener#tunnel-client-22-7#0xbef60b49#Write operation FAILED for payload message packet with size 8,571 for [id: 0x3580e332, L:/xzy - R:xzy]. Cause: com.sap.core.connectivity.protocol.http.handlers.HttpProtocolException: Access denied to resource https://xzy/sap/bw/ina/GetServerInfo?sap-client=100 on system xzy. In case this was a valid request, ensure to expose the resource correctly in your cloud connector.. Tunnel id: account:///

Environment

  • SAP Datasphere

Reproducing the Issue

  1. In SAP Datasphere, open the Space’s Connections and validate the existing SAP S/4HANA On-Premise connection.
  2. Create an SAP S/4HANA live data connection of type Tunnel and test the connection.
  3. Observe that the test fails with an SSL error or "You are not authorized..." message; Cloud Connector logs show access denied for the INA service path.

Prepare Connectivity to SAP S/4HANA On-Premise | SAP Help Portal

Cause

The SAP Cloud Connector access control did not expose the required INA service path; requests to /sap/bw/ina/GetServerInfo were denied for the virtual host and port used by the tunnel.

Resolution

  1. In SAP Cloud Connector, open the mapping for the virtual host and port that the live data (tunnel) connection uses.
  2. In Access Control for this mapping, add the resource "/" and set the access policy to "Path and all Sub-Paths", or explicitly allow "/sap/bw/ina/*".
  3. Save the changes in Cloud Connector.
  4. In SAP Datasphere, ensure the live data (tunnel) connection uses exactly the same virtual host and port as defined in the Cloud Connector mapping.
  5. Recreate or retest the live data connection; it should connect without the SSL or "not authorized" error.
