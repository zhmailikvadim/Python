Symptom

When manual settling the production orders in the app "Schedule Product Costing Jobs", job template 'Manual Settlement: Event-Based Order and PCC'.
Encounter the performance error indicating "Dispatcher advised to cancel processing completely due to No more memory available to add rows...".  

Environment

SAP S/4HANA Cloud Public Edition

Cause

Resolution

This hotfix will be deployed to customer on Jan 17th weekend. Ideally customer will not face the performance issue from Jan 19th. 
If still encounter long time processing, it's suggested to adjust the 'Package Size' to a small number, such as 10 (default is 100). 
A big number has risk to cause memory issue.

Keywords

production order settlement, schedule product costing jobs, performance, job log error, S/4HANA Cloud Public Edition.
