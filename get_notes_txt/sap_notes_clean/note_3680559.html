Symptom

You are unable to start your local SAP Commerce environment, and the following error appears in the console:

*** LICENSE VERIFICATION HAS FAILED! ***

This occurs when the temporary unlimited DEV license has expired.

Environment

SAP Commerce Cloud

Cause

The temporary unlimited DEV license in SAP Commerce environments is designed for short-term usage and evaluation.

  • A 30-day temporary license is issued automatically during installation.

  • After expiration, the license can be extended for an additional 90 days using the command-line license tool.

  • The maximum validity of a temporary license is 120 days (30 days + 90 days).

Once this period ends, the license must be renewed or regenerated to restore system access.

Resolution

To extend or generate a new temporary DEV license, follow the steps below:

Open a terminal/command prompt in your SAP Commerce installation directory.

Run the command-line license tool to generate a new 90-day license key:

ant licenseext

or

./license.sh -temp CPS_MSS

Restart your SAP Commerce server after the license renewal.
