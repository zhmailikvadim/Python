Symptom

  • Space availability and performance constraints are being queried for restoring EXT and TCMP schema tables from Production to NON - Production.

Environment

SPM Titan Ref Architecture and Migration Incentive Mgmt.

Reproducing the Issue

  1. Query raised regarding the possibility of restoring EXT and TCMP schema tables 
  2. Inquiry includes validation of space availability and performance constraints in NON - PRD.
  3. Customer provided information indicating that space availability in Hana does not need to be checked as systems allocate space as needed.

Cause

Resolution

  1. Systems will allocate space as needed; no need to check for space availability in Hana.
  2. Create a service request for restore.
  3. After the restore is completed, monitor for pipeline performance issues.
  4. If pipeline performance issues are observed post-restore, create a new case to adjust workers' memory.

Keywords
