Symptom

The archive write report RFKKOPLIST01 defines the minimum residence time as 365 days. You want to shorten the minimum residence time to 90 days.

Other Terms

FI_MKKOPLS, RFKKOPLIST01, FKK_ARCH_BASE_WRITE, residence time

Reason and Prerequisites

The system does not use residence times shorter than 365 days that you define in FPAROPLST0.

Solution

After implementing this SAP Note, the system allows a minimum residence time of 90 days.

In transaction FPAROPLST0, teh system accepts any residence time of 90 days or longer.
