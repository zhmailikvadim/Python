Symptom

The start of consideration of the valuation in a contract is in February. A condition relevant for the valuation has a validity-from date set to the 30th. of a month preceding the valuation start date. The frequency start is defined as the condition start.

When executing the valuation, the system generates duplicate cashflow items for every calculation period following the initial period.

Other Terms

SOC, duplicated items, clearing, valuation, leasing

Reason and Prerequisites

This is caused by a program error.

The frequency start is the start of the condition on the 30th of a month, the start of consideration of the valuation is in February and the value date is the 28th or 29th of February

Solution

Implement the attached correction instruction or import the specified support package.
