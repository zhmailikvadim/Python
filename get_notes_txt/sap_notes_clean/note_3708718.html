Symptom

  • When creating a goal, as outlined in Automatic Target and Rating Scale Population, the first Metric Lookup Table from the dropdown list does not auto-populate instantly
  • Instead there is a text saying "Please select"

Environment

SAP SuccessFactors Goal Management

Reproducing the Issue

  1. Access a goal plan where you have configured multiple Metric Lookup Tables to be autopoulated.
  2. Click "create a goal"
  3. In the goal creation page, the Metric Lookup Table dropdown field says "Please select" instead of already auto-populating one of the Metric Lookup Tables

Cause

You did not set a default-value for the field definition of "target-baseline".

Resolution

Set a default-value for the field definition of "target-baseline".
