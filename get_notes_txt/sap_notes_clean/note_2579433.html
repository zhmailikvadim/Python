Symptom

You are using Resource view periodic in Product Planning Table (/SAPAPO/PPT1). You see that the informarmation row 'Available without fixing interval' gives wrong value in some cases.

Other Terms

/SAPAPO/PPT1

Resource view periodic

Available without fixing interval

Reason and Prerequisites

Due to a program error.

Solution

As a general rule, SAP recommends that you install Support Package as your solution. However, if you wish to install a solution earlier, use the Note Assistant to implement the correction instruction.
