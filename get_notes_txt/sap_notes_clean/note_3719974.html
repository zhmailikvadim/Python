Symptom

Prerequisite Objects for SAP Note 3703019.

Other Terms

RFUMSV50, S_AC0_52000644, Deferred Tax Transfer (New)

Reason and Prerequisites

The implementation of this SAP Note alone has no impact on your existing functionality until the solution SAP Note is applied.

SAP Note 3703019 requires this SAP Note to be implemented as a prerequisite.

Solution

To create the prerequisite objects of SAP Note 3703019:

Use transaction SNOTE to implement the attached correction instructions before you implement primary SAP Note 3703019.
Perform manual post-implementation steps.
After you implement this SAP Note, implement the primary SAP Note 3703019.
