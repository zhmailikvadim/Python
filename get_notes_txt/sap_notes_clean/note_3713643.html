Symptom

When creating a Validation Rule Type for a specific HCM localization in Configuration Workbench (Payroll Process) transaction (PYC_CONF_WB), a given infotype is not available in the dimension catalog.

Environment

  • Human Capital Management (HCM)
  • SAP enhancement package 8 for SAP ERP and above

Reproducing the Issue

  1. Execute transaction in PYC_CONF_WB.
  2. From the dropdown menu, select 'Validation Rule Type'.
  3. Select a standard Validation Rule Type (e.g. SBP_EE_PERSINFO_XX) and create a copy.
  4. In the 'Basic Information' tab change the Country/Region to your HCM localization.
  5. Click on the 'Dimensions' tab.
  6. You miss some infotype you need for your Validation Rule Type.

Cause

Configuration of tables T777D, T582A, T582L, T582V and T582W

Resolution

  1. If you are using the standard runtime class, the dimension picklist is created at CL_PYC_CWB_VT_SETUP=>GET_DIMENSION_PICKLIST( ).

  2. The list of infotypes to be included in the dimensions is done based on tables T777D, T582A, T582L, T582V and T582W, and retrieved at the following sentence:

        "1. get all infotypes

        SELECT DISTINCT a~infty,'INF' as category FROM t582a AS a
         INNER JOIN t777d AS d ON a~infty = d~infty
         WHERE d~infkn = 'I'
           AND ( EXISTS ( SELECT * FROM t582l AS l1 WHERE l1~infty = a~infty AND l1~molga IN @it_molga_so AND l1~statu <> 'E' )
            OR NOT EXISTS ( SELECT * FROM t582l AS l2 WHERE l2~infty = a~infty AND ( l2~molga IN @it_molga_so OR ( l2~molga NOT IN @it_molga_so AND l2~subty = '*' AND l2~statu = 'X' ) ) ) )
        UNION
        SELECT DISTINCT w~infty,'ITV' as category FROM t582v AS v
         INNER JOIN t582w AS w ON w~vinft = v~vinft
         WHERE w~seqnr = '2' AND v~molga IN @it_molga_so
          INTO TABLE @DATA(lt_infty). "#EC CI_BUFFSUBQ "#EC CI_BUFFJOIN

        SORT lt_infty BY infty.

        DELETE ADJACENT DUPLICATES FROM lt_infty COMPARING infty.

  3. Based on the SELECT above, adjust the content of tables T777D, T582A, T582L, T582V and T582W so that the required infotype is included in the dimension picklist.
