Symptom

You would like to have authorization check on pur. org. / sales org. when doing selection on MSR_TRC_I / MSR_TRC_C selection screen. But you experience that such way you can not control the selection screens.

Environment

Advance Return Management (LO-ARM)

  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA
  • SAP S/4HANA Private Cloud 

Cause

The ARM standard design performs authority checks at the transaction code level within transactions MSR_TRC_C and MSR_TRC_I. Each application is responsible for the application-specific authority checks, so this only happens when trying to access an application document, not during the initial selection.

Resolution

Use the ARM software logic in this above info's sense, or if further authorization checks needed, enhance the program.
