Symptom

When Product Compliance Event Processing (PCEP) processes product compliance events and product compliance tasks, memory leaks can occur during the creation of a large number of assessments in combination with  message logging which can lead to short dumps.

Other Terms

 PCEP, Migration Cockpit, Creation of Compliance Assessments

Reason and Prerequisites

 The Product Compliance Event Processing (PCEP) is not optimized for the creation of a large number of assessments.

Solution

After implementing this SAP Note, the system optimizes the creation of compliance assessments. This optimization prevents the out-of-memory error.
