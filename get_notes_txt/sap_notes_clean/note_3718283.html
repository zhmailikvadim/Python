Symptom

You have already followed SAP Note to add the certificate in SUM jvm keystore 2374620 - SYB: SUM AS JAVA Upgrade/SP Update with Database SSL Encryption enabled

During SUM with ASE with ssl enable cannot connect with error :

[Error ]: Could not connect.
[Error ]: com.sap.sql.log.OpenSQLException: Exception of type java.sql.SQLException caught: JZ006: Caught IOException: java.io.IOException: JZ0T3 use getCause() to see the error chain.
[Error ]: Could not establish open SQL connection. Exception com.sap.sql.log.OpenSQLException: Exception of type java.sql.SQLException caught: JZ006: Caught IOException: java.io.IOException: JZ0T3 use getCause() to see the error chain..

And in ASE error log at same time you see errors:

kernel  ssl_server_handshake: vsn 135 kpid 555548833 enp_sslAccept failed sslerr 0x20000305 state 2
kernel  ssl_server_handshake: vsn 135 kpid 555548833 FAILED sslerr 0x20000305

Environment

  • SAP SUM 1.1 SP05 and higher.
  • SAP Adaptive Server Enterprise 16 SP04 PL08 and higher.

Resolution

Set ASE Cipher with sapsso in isql :

sp_ssladmin 'setcipher', 'Strong'
go
