Symptom

The sum line of log in the transaction /SAPAPO/BOD_MAINT is not calculated by default.

Other Terms

Transaction /SAPAPO/BOD_MAINT

Reason and Prerequisites

The log messages will be displayed and calculated immediately if the component 'SHOW_ALL' is filled with 'X' in the display profile.

Function module 'BAL_DSP_PROFILE_DETLEVEL_GET' is used to get the display profile, while the component 'SHOW_ALL' is empty in this case.

Solution

Replace function module 'BAL_DSP_PROFILE_DETLEVEL_GET' with 'BAL_DSP_PROFILE_SINGLE_LOG_GET'.

The log messages will be displayed and calculated immediately.
