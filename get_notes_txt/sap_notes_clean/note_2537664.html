Symptom

You want to deploy an ASE instance in a GCP IaaS environment.

Other Terms

ASE on Google Cloud Platform, ASE on GCP, IaaS

Solution

ASE can be installed (ASE Implementation Documentation) and run on an operating system that conforms to the documented requirements for ASE and SAPNote 2489781 (ASE 16.0 SP03) or SAPNote 2987324 (ASE 16.0 SP04)

The operating system selected must be supported by GCP.

ASE Features that may not be supported by GCP

Following ASE 16 features may not be supported on GCP

  • Real Time Data Services
  • Web Services
  • Kerberos
  • IPv6
  • ASE High Availability Option w/ Native Cluster Managers
  • Raw disk as database devices
  • Kernel Process Mode
  • Tivoli Storage Management
