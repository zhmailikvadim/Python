Symptom

The  API calls on the Production Connector like Read Indicator, Write Indicator or Execute an Automation Sequence show a longer processing time.

Other Terms

Performance, SignalR, ProdConMainService

Reason and Prerequisites

The performance impact was caused by

  • avoidable connection checks that verified the status of the agent instance
  • an incorrect handling of parallel incoming service requests

Solution

Workaround

No workaround.

Problem Solution by Patch

The problem will be corrected with the following patches:

  • Patch 5 for Production Connector 3.0
  • Patch 4 for Production Connector 2.7

Installation of Patches

Download the current installation program for the patch from the SAP Software Center and install it in accordance with the instructions in the installation guide. For the current version of the installation guide, see the SAP Help Portal at https://help.sap.com/docs/PRODUCTION_CONNECTOR. You can use the installation program for the patch to update a version of your product that is already installed or to set up a new installation if the product is not yet installed on the target computer.

Before you start the installation, see also the relevant release installation note 3706769 for Production Connector 3.0 and note 3665529 for Production Connector 2.7.
