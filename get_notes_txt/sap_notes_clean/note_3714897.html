Symptom

When adding a custom scope in MDCIMG->Configure Scope IMG for BO Type - 194, the dump is raised as below :

Category               ABAP programming error                                                       
Runtime Errors         ITAB_DUPLICATE_KEY                                                           
Except.                CX_SY_ITAB_DUPLICATE_KEY                                                     
ABAP: Program          CL_MDC_SHLP_SCOPE=============CP                                             
Application Component  CA-MDG-CMP-AF    

Environment

  • SAP Master Data Governance-Consolidation and Mass Processing

Reproducing the Issue

  1. Launch transaction MDCIMG -> Configure Scope
  2. Create New entry for Scope by clicking “New Entries” button.
  3. Provide Config ID and BO Type - 194.
  4. Select just created Config ID line and navigate on the left side to “Scope Fields” section.
  5. Click “New Entries”. Enter manually Table and Field, for example: Table - MARA, in Field - MATKL. Click Save and you will get dump.

Resolution

  • The dump is raised due to no field defined for tables DRAD and DRAW at IMG : MDCIMG -> Master Data Governance, Consolidation and Mass Processing -> Configure Process Models and Field Properties -> Configure Process Models, and Specify Table and Field Properties 
  • To correct the issue, please either remove those tables or uncheck Process check box associated to them.
