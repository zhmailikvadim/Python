Symptom

How to submit the external documents to KSeF 2.0 created in an non-SAP system?

Environment

  • SAP ERP
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition
  • SAP Document and Reporting Compliance, Cloud Edition

Cause

How-to query.

Resolution

Refer the below help portal documentation:

  • SAP ERP: Submitting Electronic Customer Invoices Created in External Systems | SAP Help Portal 
  • SAP S/4HANA: Submitting Electronic Customer Invoices Created in External Systems | SAP Help Portal 

NOTE:

This functionality is released with note 3695841 - Electronic Document Processing for Poland - KSeF: External Documents for FA(3) Schema. If the SAP Note is implemented and the functionality is not present, raise a case with component CA-GTF-CSC-EDO-PL.

See Also

KBA 3346323 - eDocument Poland B2B(KSeF) Troubleshooting Guide
