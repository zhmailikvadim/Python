Symptom

When attempting to select an RFC connection during comparison or adjustment in SPRO, SIMG, SM30, SM34, SCU0 or SCMP, the status bar displays the message: No values found. 
Message No. DH801

Environment

S/4HANA Private Cloud and On Premise 

Cause

The data sources / destinations are not present in the table BCTOOLS_SOURCE

Resolution

  1. Make sure, that the user has authorization for authorization object S_CC_COMP  
  2. Maintain the data sources / destinations via the following path in the Implementation Activities App or via transaction SPRO
    ABAP Platform -> Application Server -> System Administration -> Customizing Tools -> Maintain Destinations for Comparison and Adjustment

Documentation for Scenario BC_CC_COMPARE

This switchable authorization check scenario activates the "S_CC_COMP" authorization object check for protecting customizing comparison of an object / IMG activity with remote sources. This allows control over the remote data sources / destinations with which an object / IMG activity comparison and adjust can be carried out.

Affected Business Processes and Roles

The new authorization object "S_CC_COMP" has been introduced in transactions SIMG_ACTIVITY_COMP, SPRO, SIMG, SM30, SM34, SCU0 and SCMP. In S/4HANA Cloud, this authorization object has been introduced during compare which can be launched via the Implementation Activities app and/or Central Business Configuration via Manage Your Solution App.

Following role is impacted by new authorization:

  • SAP_BC_CUS_CUSTOMIZER (Customizing Project Team Member)

Affected Technical Objects

  • Transactions SIMG_ACTIVITY_COMP, SPRO, SIMG, SM30, SM34, SCU0 and SCMP
    During launch of comparison via these transactions, the authorization check is performed against the remote data sources / destinations present in the table BCTOOLS_SOURCE and only the authorized entries are displayed for comparison selection.
  • Authorization Object S_CC_COMP
    This authorization object controls which remote data sources / destinations are displayed for comparison.

Prerequisites

The data sources / destinations are present in the table BCTOOLS_SOURCE.

  • S/4HANA Public Cloud
    The data sources / destinations are deployed in the table during tenant provisioning via task list.
  • S/4HANA Private Cloud and On Premise
    The data sources / destinations can be maintained via the following path in the Implementation Activities App or via transaction SIMG
    Application Server -> System Administration -> Customizing Tools -> Maintain Destinations for Comparison and Adjustment
    For OnPremise transactions SM30 and SM34, authorization variants SM30_COMPARE and SM34_COMPARE provide the restrictions.
