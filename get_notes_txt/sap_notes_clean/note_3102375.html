Symptom

Entries of the Security Audit Log show the transaction code 'S000' along with the program 'SAPMSSY1' for events of type AUK and AU5 (e.g. successfull RFC calls). However, note 2191612 defines that (quote), 'The "Transaction Code" and "Program" fields are filled in accordance with the trigger location of an event in ABAP NetWeaver. This means that the calling context is logged during a preliminary check of the accessibility of a resource.'

In cases of events of type AUK and AU5, the field "transaction code" is without meaning as an RFC call is handled by the modules of module-pool SAPMSSY1. Thus, the correction delivered with this note makes sure that the "transaction code" field is empty for log entries of type AU5/AUK written by the SAP kernel.

Other Terms

SAL

Reason and Prerequisites

Programming error in the kernel.

Solution

This correction is delivered with the kernel archives:

  • hotfix - update the disp+work package of the kernel of your system to at least the patch level given in this note.
  • SP Stack Kernel - files ﻿SAPEXE.SAR and SAPEXEDB.SAR  ﻿﻿

The correction is contained in all patch levels that are equal to or higher than the patch level listed in the "Support Package Patches" section of this Note for the desired kernel release.
If there is no entry, the patch is not yet delivered. The patch level will be added automatically when the patch is available for download. You can subscribe to this SAP Note to get informed when it is updated.

Follow the SAP recommendations on how to patch the SAP kernel:

  1. Apply the latest SP Stack Kernel if it already contains the correction. For the list of current SP Stack Kernels, see SAP Note ﻿2083594﻿.
  2. Apply the hotfix only if you are experiencing a serious error that is not yet corrected by the latest SP Stack Kernel. 
  3. Review the regression note for the required patch level before installing the kernel patch. For details, see SAP Note ﻿1802333﻿.
  4. For instructions on how to download and install kernel patches, see SAP Note ﻿19466﻿.

The paper ﻿Update Strategy for the Kernel of the Application Server ABAP in On Premise Landscapes﻿ provides detailed information on the SAP recommendations.
