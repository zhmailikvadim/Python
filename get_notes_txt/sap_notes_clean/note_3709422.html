Symptom

When using the Manage Process Orders Fiori app, the exported file does not include the material description.
Users observe that only the product number appears after downloading the list, and the description column remains empty.
The issue occurs consistently during export, although the description is visible in the app UI.

Other Terms

F4587

Reason and Prerequisites

The reason is a program error.
A missing field mapping in the export logic prevents the material description from being transferred to the output file.

Solution

Apply the attached Correction Instruction.
The correction adjusts the export logic so that material description is included in the generated file.
After implementation, clear the frontend/UI cache as described in SAP Note 2319491 to ensure the updated logic is effective.
