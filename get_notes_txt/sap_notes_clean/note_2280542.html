Symptom

  • The time to check in a PowerDesigner PDM that targets ORA12C is a lot longer compared to ORA11GR1 PDM.
  • It also happens when selecting the Preview tab in the model property sheet.

Environment

SAP PowerDesigner (PD)

Reproducing the Issue

  1. Use any pdm (simplest is easier)
  2. Open repro model .
  3. Select Database > Change Current DBMS to select ORACLE Version 11g.
  4. Check the model in to the repository( Check in takes up to 1 min).
  5. Select Database > Change Current DBMS to select ORACLE Version 12c, or generate a new ORA12C PDM.
  6. Check the model in to the repository. Check in may take up to 10 min.

Cause

The issue has been identified and logged under CR 795761.

Resolution

The issue is still being investigated under CR 795761. Resolution will be updated once issue is fixed.
