Symptom

What is EZ Rater and how do I enable it for our Performance template?

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

SAP SuccessFactors Performance Management

Resolution

What is EZ Rater?

There is a need to provide an innovative approach to rating objectives, competencies and behaviors that will allow the user to compare various ratings side-by-side in one view while the user is filling out the performance review form while at the same time having a real-time, on-form gap analysis.  The new user interface that SuccessFactors will provide is called EZ-Rater. EZ-Rater will aid in employee analysis by providing users with a simple, compact user interface for rating and comparing against other ratings as well as providing real-time, on-form gap analysis.  It will also provide an improved way of rating by using radio button functionality therefore increasing the ease of applying ratings.﻿

How to enable it?

EZ rater works only for objective section and competency section. In form template xml section configuration sect-mode="EZ-Rater".

Please Note: As a customer, you do not have access to Provisioning. To complete tasks in Provisioning, contact your Implementation Partner. For our list of supported configuration changes, please refer to this article 2250186 - SCR Form: XML Software Change Requests for PM / Mutli-Rater Templates.

To Partners, please refer to the PM v12A Implementation Guide. For Configuration or Implementation assistance, please direct your questions to Partner Delivery Community.

Other Configuration available

In form template xml
ez-rater-expand-all="true"|"false"
<ez-rater show-gap="true"/> by default gap will be shown (the gap analysis graphic is only supported when rating-opt="3" is configured for the section)

In form template setting page
Show Manager Rating first for EZ Rater mode
