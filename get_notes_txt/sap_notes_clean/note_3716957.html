Symptom

The replication report RP_HRSFEC_JOB_POS_REPL_V2 runs succesfully and without errors in log. However, after the report run there remain certain number of position entries which are not fully replicated, that is  - the mapping in the table HRSFEC_JOB_POS is present in system, but there remains inconsistencies between position tables HRP1000, T528B and T528T - entries might be present in one table, while missing in another.

Other Terms

RP_HRSFEC_JOB_POS_REPL, RP_HRSFEC_JOB_POS_REPL_WCS, RP_HRSFEC_JOB_POS_REPL_V2

Solution

Import the support package specified for your release or implement the attached automatic correction instructions.

As a general rule, SAP recommends that you implement the support package specified for your release. However, if you need to install a solution earlier, use the SAP Note Assistant to implement the automatic correction instructions. You can find more information about the SAP Note Assistant in SAP Support Portal, under SAP Note Assistant.
