Symptom

Version History

Version 1 (20.02.2026): Pilot Version

Version 2 (27.02.2026): General Availability (No changes from Pilot Version)

When Automatic Enrolment Payroll Notifications (AEPNs) are processed by the Automatic Enrolment: AEPN Upload and Processing report (HIECAEN0), the start date (BEGDA) of the Automatic Enrolment (Ireland) infotype (3448) can be left empty. This causes a rejection message being issued during the update of IT3448 by the report;

"Fill out all required entry fields"

This incorrect processing occurs when an employee has transferred from one payroll area to another, and both payroll areas have a Payroll Control Record status of 'EXIT' (Transaction PA03).

This SAP Note corrects this issue.

Other Terms

HIECAEN0, IT3448, Start Date

Solution

EITHER

Apply the relevant HRSP

OR

Apply the correction instructions attached to this SAP Note, via SNOTE.
