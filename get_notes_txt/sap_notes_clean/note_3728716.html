Symptom

You modify a flow, that an accumulating condition creates, on the Cash Flow tab in transaction FTR_EDIT. The system raises runtime error RAISE_EXCEPTION with exception MODIFICATION_FAILED in class CL_FTR_CF method FLOW_MODIFY.

Other Terms

swap, interest rate instrument, IRI

Reason and Prerequisites

This is a programming error.

Solution

After implementing this SAP Note, the system correctly processes cash flow modifications without raising the runtime error.

To implement this SAP Note, run transaction SNOTE and follow the correction instructions.
