Symptom

  • Following the update to version 2512, a click in the "Item Number/EAN Number" field is required before each item can be added to the shopping cart.
  • After clicking the field, the barcode can be scanned, and the item is added to the cart.
  • To add another item using its barcode, the field must be clicked again.
  • In the previous version of SAP Event Ticketing, barcodes could be scanned seamlessly without requiring manual field selection.

Environment

SAP Event Ticketing, Cloud Edition

Reproducing the Issue

  1. Access the SAP Event Ticketing platform.
  2. Search for an item, such as "xyz".
  3. Select the item.
  4. Locate the "Item Number/EAN Number" field on the upper right side of the user interface.
  5. Observe that the mouse cursor does not automatically focus on this field upon opening the interface.
  6. Manually click the field to enable barcode scanning for adding the item to the cart.
  7. Repeat the process for subsequent items, requiring manual field selection each time.

Cause

  • The update to version 2512 introduced changes to comply with the Barrier-Free Accessibility Act (BFSG).
  • The selection of the first element in backend dialogs is disabled, requiring users to manually select the input field.

Resolution

The system works as intended and is compliant to the BFSG regulations.
