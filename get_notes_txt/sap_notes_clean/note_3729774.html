Symptom

The form title in API/reporting is not updated after making changes in manage templates

Environment

SAP SuccessFactors Performance & Goals

Reproducing the Issue

  1. Launch a form
  2. Go to manage templates, change the form template name
  3. In Performance review UI, the form title is changed to new template name
  4. In API/reporting, the title is still showing old form template name

Cause

This is current system behavior.

Resolution

In performance review UI updates in real-time is that the 'form title' is dynamically generated based on the combination of `form template name (localized)`,

The API/reporting display information that aligns with the metadata. Once a form is launched, its ‘meta data’ remains static and does not change.
