Symptom

In the payroll run, the system previously issued an error message during the processing of the technical wage type /154 (cumulation wage type for special payments AT) that caused the payroll to terminate.
After you implement this correction, the system issues a warning message instead so that payroll continues and the wage type /154 is displayed in the payroll result.

Other Terms

/154, cumulation wage type, special payment AT, payroll termination, warning message instead of error

Solution

The correction is provided by HR Support Package (via CLC if necessary).
You can implement an advance correction by performing the manual activity. Note that a customer layer may exist for AJA.
