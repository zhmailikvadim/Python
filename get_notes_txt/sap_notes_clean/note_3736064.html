Symptom

  • Users plan to analyze customers' language using a statistic.
  • The language field is available on the customer account screen.
  • In Statistics, under Processing > Attributes, the language business object (BO) is not available for selection.
  • Users ask if the language attribute can be added to the business objects for statistics.

Environment

  • SAP Event Ticketing, cloud edition
  • SAP Event Ticketing

Reproducing the Issue

  1. Log on to the system.
  2. Navigate to Sales > Statistics.
  3. Click Find and open the desired statistic.
  4. In the left panel, select Processing.
  5. Open Attributes.
  6. Observe that the language business object is not available for selection.

Cause

The language business object/attribute is currently not available for selection in Statistics; this is an enhancement request.

Resolution

  1. Verify that the language attribute is not listed under Statistics > Processing > Attributes.
  2. Submit a feature request through the standard enhancement channel to have the language attribute added to the relevant business object for statistics.
  3. Track the enhancement request and implement the feature once it becomes available.
