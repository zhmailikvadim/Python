Symptom

This KBA contains the guidance of how to add additional fields in app Stock - Multiple Materials.

Environment

SAP S/4HANA Cloud Public Edition

Resolution

Currently app "Stock - Multiple Materials" does not support custom fields. It is only possible to extend the app by creating data source extensions in the Custom Fields app with following additional standard fields:

  • Master Data: Product General
  • Master Data: Product Plant
  • Master Data: Batch

You can add additional SAP fields to this data source in section Data Source Extensions of the Custom Fields and Logic app and publish your extension. After this extension is published, the additional fields will be available in filter and table view in app Stock - Multiple Materials.
