Symptom

  • The transaction codes /SAPSLL/EMB_BLRL_IMP and /SAPSLL/EMB_BLRL_EXP are obsolete in GTS E4H. You would like to know what is the successor of the transaction codes.

Environment

  • SAP Global Trade Services, edition for SAP HANA

Cause

Resolution

  1. In GTS E4H, the Fiori app 'Manage Blocked Documents' (F4714) is used as the standard solution to replace the transaction codes /SAPSLL/EMB_BLRL_IMP and /SAPSLL/EMB_BLRL_EXP.
  2. This application provides functionality to release blocked documents effectively.
  3. Refer to the help document for detailed guidance: Releasing Blocked Documents and Payments | SAP Help Portal.

See Also

  • Refer to: Releasing Blocked Documents and Payments | SAP Help Portal.
