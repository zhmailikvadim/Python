Symptom

Administrators previously lacked the ability to quickly identify, sort, or filter documents based on the actual owner (the person for whom the document was uploaded). This led to:

  • Difficulty in auditing documents uploaded on behalf of others.

  • Inefficiency in managing large document lists without owner-based sorting.

  • Lack of specific owner metadata (User ID and User Name) in the primary list view.

Environment

SAP SuccessFactors HCM Suite

Reproducing the Issue

To view or customize the new owner information columns, follow these steps:

  1. Log in as an Administrator.

  2. Navigate to the Manage Documents page.

  3. Select the Documents tab.

  4. Observe the document list; by default, the Owned By column (Last Name, First Name) is now visible.

  5. To see additional ID data, click on the View Settings dialog box.

  6. Navigate to the Columns tab to toggle Owned By User ID or Owned By User Name.

  7. Navigate to the Sort tab to select any of these new fields as your primary or secondary sorting criteria.

Resolution

The system has been updated to include three new metadata columns specifically for document ownership. This change provides full traceability for "upload on behalf of" scenarios.

  • Implementation: The Owned By column is now a standard part of the default view.

  • Administrative Control: Admins can use the View Settings menu to add Owned By User ID and Owned By User Name to their dashboard.

  • Functionality: All three new fields are fully supported in the Sort criteria, allowing for better organization and faster retrieval of user-specific documentation.

Frequently Asked Questions (FAQ)

Q: Why was this change implemented?
A: This update was designed to improve visibility and traceability. It is now much easier to identify the actual document owner, especially in scenarios where an administrator or manager has uploaded a file on behalf of another employee.

Q: Is the "Owned By" column visible to everyone by default?
A: Yes, the Owned By column (displaying the concatenated Last Name and First Name) is now included by default in the standard document list view for Administrators on the Manage Documents page.

Q: Can I see the specific User ID of the document owner?
A: Yes. While not shown by default to keep the interface clean, you can enable the Owned By User ID and Owned By User Name columns through the View Settings dialog box under the Columns tab.

Q: How do I organize my document list by owner name?
A: You can set your sorting preferences by opening View Settings, navigating to the Sort tab, and selecting your preferred "Owned By" field as the primary sort criteria.

Q: Can I remove these columns if I don't need them?
A: Absolutely. If you prefer a more minimalist view, you can use the View Settings dialog to uncheck any of the ownership columns, including the default Owned By column.

Q: Does this change affect how documents are uploaded?
A: No, this change only affects how document information is displayed and organized for Administrators. The actual upload process remains the same.
