Symptom

  • Performance Analysis Tool displays Widget ID instead of Widget Name in the story view.

Environment

SAP Analytics Cloud 2025.21.11  

Reproducing the Issue

  1. Navigate to Performance > Analysis Tool.
  2. Select an item in the list.
  3. Switch to the relevant tab.
  4. Observe that the table is showing Widget ID instead of Widget Name.

Cause

This is a Limitation for tenants on HanaCloud.

Resolution

How to know if your tenant is running on HANA Cloud:

  • By navigating to the “About” section under the SAC profile. Tenants running on SAP HANA Cloud will show the HANA Cloud version. 
    SAP Analytics Cloud infrastructure is moving to SAP HANA Cloud.

How to Identify which widget corresponds to the widget ID:

  • you can use F12 / Developer Tools > Elements in the story and select (first marker) the widget to find out the widget ID from source code.
