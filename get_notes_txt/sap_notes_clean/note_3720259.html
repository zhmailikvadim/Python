Symptom

Goal plan introduction section in the mobile app allows editing text, which is not an expected behavior. Users can insert random text, delete text, and delete images within the introduction section but cannot save the changes. 

Environment

  • SAP SuccessFactors Performance and Goals
  • SAP SuccessFactors Mobile

Reproducing the Issue

  1. Login to the mobile app.
  2. Navigate to the Objectives and Key Results (OKR) page.
  3. Open the Introduction section of the goal plan.
  4. Place the cursor anywhere within the Introduction section.
  5. Observe that the section allows editing text and images.
  6. Note that there is no option to save the changes made.

Resolution

This issue will fixed via a patch and is scheduled to be released on May 18, 2026.

For release timelines, please review SAP SuccessFactors Product Release & Road Map Information.
