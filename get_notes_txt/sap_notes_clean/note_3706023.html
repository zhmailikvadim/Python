Symptom

When re‑initiating assessment in the Redesigned Workbench, the “Initiate Assessment” button is greyed out, while it remains available in the Legacy Workbench.

Environment

SAP SuccessFactors Recruiting Management

Reproducing the Issue

  1. Initiate the assessment in the Redesigned Workbench 
  2. Observe that the "Initiate Assessment" button is greyed out
  3. Switch to the Legacy Workbench and repeat the steps
  4. Observe that the "Initiate Assessment" button is enabled

Cause

It is an expected behavior in Redesigned Workbench. 

Resolution

As per the Redesigned Workbench design, the "Initiate Assessment" button is enabled only when an error occurs during an already initiated assessment or if an initiated assessment fails.
