Symptom

You are replicating business partners into SAP Cloud for Customer but receive this error:

Save not allowed. Finalize skipped by esf (call_plugins_pre_exit)

Environment

SAP Cloud for Customer

Reproducing the Issue

1. Logon to the C4C tenant
2. Navigate to the Web Service Message Monitoring
3. Open the Error Log of the affected SOAP message
4. The Detailed Error Text shows the error

Cause

There is a workflow rule configured to use a custom field for an operation on an account. If the custom field is then deleted but the workflow rule continues to reference the deleted object, then the error occurs.

Resolution

Update or remove the workflow rule referencing the deleted field.

Keywords

SAP, SAP Cloud for Customer, C4C, BusinessPartnerERPReplicationIn, customer, account, workflow rule, custom field
