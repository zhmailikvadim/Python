Symptom

You run the Tax Reporter (PU19) transaction to generate a Social Unemployment Insurance (SUI) electronic file for New Jersey (HR_F_SUI_NJ_TAPE). However, the resulting file is including the record Code X: Transmitter Data, which should not be included in the file, as per legal requirement.

Other Terms

HR_F_SUI_NJ_TAPE

Reason and Prerequisites

Reason

Correction of legal function

Prerequisites

As a prerequisite, you have applied SAP Note 2360638 - Year End 2016 Phase I for U.S. Tax Reporter

Solution

As general rule, SAP recommends that you install a solution by applying a Support Package. However, if you need to install a solution earlier install the manual steps described in this SAP Note.

You can find more information about the Note Assistant in SAP Service Marketplace, under service.sap.com/note-assistant.
