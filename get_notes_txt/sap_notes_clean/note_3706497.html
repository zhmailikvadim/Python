Symptom

The Manage Assignments app and Manage Matching Documents app are experiencing prolonged loading time when unassigning a large number of matching items or deleting matching documents.

Other Terms

ICMR, unassign, assign

Reason and Prerequisites

Long runtime can occur due to too many database queries executed at the same time.

Solution

Apply the SAP Note to your backend system.

The Note improves system performance with the following enhancements:

  1. Activate table buffering for the customization tables: ICA_DS, ICA_METHOD_DS, ICA_RC, and ICA_RC_METHOD.
  2. Store configuration data and authorization check results in buffer, retrieving them directly from buffer for improved performance.
  3. Verify workflows only when a reason code is associated with a workflow scenario ID.
