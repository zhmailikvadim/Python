Symptom

When creating a billing document for company code 1020 that has not yet been posted, the "Test Copy" watermark does not appear in the PDF output.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Access the Create Billing Documents app in SAP S/4HANA Cloud Public Edition.
  2. Generate a billing document for company code 1020.
  3. Observe that the watermark "Test Copy" does not appear in the PDF output.

Cause

The master form template and its usage of the Watermark are defined by form processing. In transaction APOC_SUPPORT, generate PDF for company code 1020 using master form template used in other company code such as 1010.

Watermark displayed normally, so the issue is caused by the custom master form template associated with company code 1020.

Resolution

Since this issue stems from a customized activity rather than standard system functionality, you need request your local development team to review and adjust your customized master form template associated with company code 1020. For more details, refer to KBA 2984817-  Form design in Adobe LiveCycle Designer.

Keywords
