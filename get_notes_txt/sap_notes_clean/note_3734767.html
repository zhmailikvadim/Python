Symptom

Button "Create Fiori App" is missing in the service binding editor in in ABAP Development Tools (ADT).

Environment

  • SAP Netweaver Release lower than 816
  • ABAP Development Tools (ADT)

Resolution

Create Fiori App feature is supported from S4HANA 2025 release.

Keywords

ABAP Development Tools, ADT, service binding, Create Fiori App
