Symptom

1.On execution of the Single Touch Payroll Phase 2 report (TCODE - PC00_M13_STP_GEN) for Non-CE, an authorization error is coming after implementing the SAP Note 3660828. This occurred due to unwanted access of Infotype 0712.

2.SAP Note 3660828 had some Correction Instructions missed.
Please check the text of SAP Note 3660828 for more details.

Other Terms

STP2, Single Touch Payroll Phase 2, 0712

Transaction: PC00_M13_STP2_GEN / PC00_M13_STP2_GEN_CE

Class: CL_HRAU_STP2_TOOL_RECORD

SAP Note: 3660828
RPCHRRROT_FORM_CE, RPCHRRROT_MAIN_CE

Reason and Prerequisites

Reason: Performance fix introduced by SAP Note 3660828 was accessing the Infotype 0712 even for Non-CE reporting.

Reason for Versioning:

Version Reason
   01 General Release for Customer

Solution

Code correction required to fix the issue described in the 'Symptom' section has been provided in this Note.

To get the changes in your system, apply the HR Support Package(s) indicated for your release in the 'Reference to Support Packages' section of this SAP Note.

Alternatively to get the code corrections immediately, apply the correction instructions attached with this SAP Note.

As a general rule SAP recommends that you implement the Support Package specified for your release. However, if you need to install a solution earlier, use the SAP Note Assistant to implement the correction instruction. You can find more information about the SAP Note Assistant in SAP Support Portal, under SAP Note Assistant.
