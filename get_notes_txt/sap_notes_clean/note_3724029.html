Symptom

SLG1 error: HCM001 address type is missing ( message number /SHCM/EMPLOYEE_BL 070 ) occur when execution report /SHCM/RH_SYNC_BUPA_EMPL_SINGLE

Environment

S/4 HANA OP with HR integration active

Reproducing the Issue

  1. Execution report /SHCM/RH_SYNC_BUPA_EMPL_SINGLE of the specific employee
  2. In SLG1, the error: HCM001 address type is missing ( message number /SHCM/EMPLOYEE_BL 070 ) occur.

Cause

The address type HCM001 configuration is required in TB009.

Resolution

Please add the address type HCM001 in table TB009 via transaction SM30

Or

Please add the standard address type HCM001 via following IMG path

SAP Customizing Implementation Guide -> Cross-Application Components -> SAP Business Partner -> Business Partner -> Basic Settings -> Address Determination -> Define Address Types
 
It is mandatory during employee synchronization to BP as default.
