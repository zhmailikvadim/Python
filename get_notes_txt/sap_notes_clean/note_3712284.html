Symptom

  • BAS -> Explorer, cannot find "SAP HANA PROJECTS" like below:

Environment

SAP Business Application Studio (BAS)

Reproducing the Issue

  • BAS -> Explorer

Cause

Extension "SAP HANA Calculation View Editor" and "SAP HANA Tools" are missing. 

Resolution

Install Extension "SAP HANA Calculation View Editor" and "SAP HANA Tools"
like https://developers.sap.com/tutorials/hana-cloud-cap-create-project.html
Create dev space in SAP Business Application Studio step 5

See Also

https://developers.sap.com/tutorials/hana-cloud-cap-create-project.html

Keywords

SAP HANA Calculation View Editor, SAP HANA Tools, Explorer, BAS
