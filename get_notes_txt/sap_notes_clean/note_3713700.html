Symptom

When adjusting a saved filter, the following error can occur:

  •  You cannot hide or delete a default filter.

Environment

  • SAP Sales Cloud Version 2
  • SAP Service Cloud Version 2

Reproducing the Issue

  1. Navigate to an Object Worklist (OWL).
  2. Click on the User Icon.
  3. Enter the Adaptation Mode.
  4. Click on the Filters.
  5. Select Saved Filters.
  6. Attempt to delete the Default Filter.
  7. Observe the error message.

Cause

The error occurs because default filters cannot be deleted or hidden.

Resolution

You should set a new default query before hiding the existing default query. If you try to hide the set default query, the system will not respect those changes. Therefore, please set a new default query before hiding or deleting the existing default query.
