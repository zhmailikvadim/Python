Symptom

How to extract Purchase Order documents from SAP Ariba Business Network using API?

Environment

SAP Business Network REST APIs - Buyers

Resolution

Depending on your account/profile, to retrieve Purchase Order details from SAP Ariba Business Network, including both header and item information, you can:

  • as a Buyer, utilize the Ariba Network Purchase Orders API
  • as a Supplier, utilize the Purchase Orders Supplier API

Both APIs offer the same functionality:

  • Use the GET /orders endpoint to access purchase order header details.
  • Use the GET /items endpoint to retrieve line-item information from purchase orders.
