Symptom

In Roadmap systems, when a Venture is created via the Interface, Operator validation is done and error "Operator is not set as PRA Business Associate" is returned to interface.

Other Terms

Related Keywords:

  • General keywords: Add-On, IS-PRA, IS-PRA-XT, Industry Solution, PRA - Production and Revenue Accounting, Upstream Oil and Gas, Ownership, Roadmap
  • Problem-specific key words: IS-OIL-PRA-REV-OW, Interface monitor, validation of Operator, Operator is not set as PRA Business Associate, Venture Create

Reason and Prerequisites

Reason: Program Error. Operator Validation is not required in roadmap system.

Prerequisites: Enhancement point definition must exist in the system which is created under the note 2414877.

Solution

Apply the correction instructions attached with the note.
