Symptom

Posting key for the given accounting document number was missing after performing release to accounting with transaction WZR2.

Other Terms

SETI TXINTF

Reason and Prerequisites

In the TXINTF object, the required function module the populates the posting key was not performed due to the current condition. This condition must be satisfied in order for the posting key field to be populated.

Solution

A minor adjustment was implemented in the code to ensure that the required function module, which restores the value of the posting key, is performed.
