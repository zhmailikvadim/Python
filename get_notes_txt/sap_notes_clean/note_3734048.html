Symptom

With this SAP Note feature BMOSI - "Determine modifier for additional contributions" has been extended by SI Contribution Reduction (RSZWG) as a field for decisions.

The feature BMOSI, structure PMEBD as well as corresponding function module HR_BE_GET_BMOSI, are enhanced accordingly.

Other Terms

BMOSI, PMEBD, HR_BE_GET_BMOSI, RSZWG

Reason and Prerequisites

SAP Note 3734207 is prerequisite.

This SAP Note is only relevant for Payroll Belgium.

Solution

  • Generally, SAP recommends that you implement the HR Support Package specified for your release.
  • However, if you need to install a solution earlier, use the SAP Note Assistant to implement the correction instruction.
  • You can find more information about the SAP Note Assistant in SAP Support Portal, under SAP Note Assistant.
  • Note that documentation, transactions, and Customizing activities are delivered via Support Package only.
