Symptom

You use SAP Forms by Adobe and configured over the logical port Cloud ADS.

The print programs always use the deafult configured logical port for Cloud ADS instead of the RFC ADS connection, requested by parameter CONNECTION.

Other Terms

FPCONNECT, ADS RFC

Reason and Prerequisites

Program error.

Solution

Use the Note Assistant to implement the correction instructions or import the relevant Support Package.

The RFC connection with name "ADS" is the default RFC connection.

When the default logical port connection to Cloud ADS is configured, it will be preferd to default RFC connection "ADS" and use the corresponded rendering settings from the table FPCONNECT for the entry "ADS".
