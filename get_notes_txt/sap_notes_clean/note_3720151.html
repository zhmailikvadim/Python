Symptom

  • The tables FAGLFLEXA and FAGLFLEXT are empty, and do not contain any line items and totals.
  • New General Ledger functionality is active in the system (the field ACTIVE in the table FAGL_ACTIVEC is 'X').

Environment

  • SAP R/3
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA

Reproducing the Issue

  1. Go to the transaction code SE16/SE16N/SE16H.
  2. Enter the tables FAGLFLEXA or FAGLFLEXT.
  3. Execute.
  4. "No values found" (Message No. WUSL002) is displayed.

Cause

A different table is maintained for the New G/L line item and total table, instead of FAGLFLEXA and FAGLFLEXT.

Resolution

Go to the below path:

Transaction code SPRO 
    SAP Reference IMG
        Financial Accounting (New)
            Financial Accounting Global Settings (New)
                Ledgers
                    Ledger
                        Define Ledgers in General Ledger Accounting

The table maintained in the field 'Totals Table' stores the totals of documents in the G/L view, for the corresponding ledger.
To identify the line item table, change the ending 'T' in the table name to 'A'.
