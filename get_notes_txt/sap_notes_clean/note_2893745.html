Symptom

create database mydb on mydb_dev1 = 5 log on mydb_dev2 = 5 with durability="at_shutdown"

After reboot, databases are receiving:

﻿Error: 624, Severity: 21, State: 1
Adaptive Server failed to retrieve a row via its RID in database 'mydb' because the requested RID has a higher number than the last RID on the page. Rid pageid =XXXX; row num = XXX. Page pointer = 0x0xXXX, pageno = XX, status = 0x1, ptnid = 8, indexid = 0, level = 0, pnextrno = 8.

Error: 926, Severity: 14, State: 1
Database 'mydb' cannot be opened. An earlier attempt at recovery marked it 'suspect'. Check the ASE errorlog for information as to the cause.

Environment

SAP Adaptive Server Enterprise 16.0

Cause

An abnormal shutdown of a server on a database created with "at_shutdown" will cause databases to receive errors.

Resolution

The errors are expected behavior - See Durability Levels for details.

Work around:

  • Create the database with durability=full.
    or
  • Rebuild the databases when an abnormal shutdown is performed.
