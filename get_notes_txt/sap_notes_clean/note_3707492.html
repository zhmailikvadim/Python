Symptom

It takes a few minutes to open the builder panel in Data Analyzer.

Environment

SAP Analytics Cloud

Reproducing the Issue

Open the builder panel in Data Analyzer

Cause

There are a lot of members (e.g. 18,000) in the account dimension, the large number of accounts cause slow rendering in the Builder Panel in Data Analyzer. 

Resolution

As a workaround, please remove unnecessary accounts before your data analysis.

If analysing a large number of accounts is a common use case for you within Data Analyzer, we suggest that you raise a enhancement request.

Keywords

SAC, data analyzer, builder panel, builder
