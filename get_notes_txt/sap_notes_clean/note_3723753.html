Symptom

The line items data in the reporting run of report Croatia G/L Account line Item are empty. This occurs when you request large amounts of data and the system experiences performance issues.

Other Terms

Croatia, HR, Document and Reporting Compliance, DRC, HR_GL_LINE_ITEMS, HR GL Line Items, HR General Ledger Line Items

Reason and Prerequisites

Reason

This SAP Note is relevant only for Croatia.

Solution

As a general rule, SAP recommends that you install this solution by applying a Support Package. However, if you need to install it earlier, use Note Assistant to implement the correction instructions.
