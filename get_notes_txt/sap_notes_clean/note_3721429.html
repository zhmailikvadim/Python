Symptom

Incorrect app name and missing details on the review booklet:
Location:

SAP S/4HANA -> Production Planning and Detailed Scheduling (PP/DS) -> Fiori Apps in PP/DS -> Production Planning Review Booklet – Resources

Contrary to the information provided in the online documentation for SAP S/4HANA on SAP Help Portal, the name of the app is not valid and have been corrected in the attached .pdf file. The online documentation is also missing information about data scope and selection, as well as some app configuration details. You can find this information in the Data Scope and Selection and Additional Details sections in the attached .pdf file.

Other Terms

 Area of Responsibility, AOR, KPI, Resource Capacity

Solution

The name of the app has been edited and the Data Scope and Selection and Additional Details sections have been added to the attached .pdf file and will be published on the SAP Help Portal as of release SAP S/4HANA 2025 FPS02
