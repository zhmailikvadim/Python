Symptom

In the program FINCS_RULE_ENGINE_TASK behind the task for automated postings you enter a wrong parameter value. You get a short dump as the resulting error message is not processed properly.

In addition, the task check doesn't check the task category.

Other Terms

UNCAUGHT_EXCEPTION CX_FINCS_RUL_SEL_PARAMS_ERROR CL_FINCS_RT_SEL_PARAMS_CHECK==CP CHECK_TASK

Solution

Implement the related support package. In urgent cases, implement the attached correction instruction.
