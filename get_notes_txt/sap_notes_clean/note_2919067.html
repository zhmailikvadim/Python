Symptom

In Fiori app "Manage Billing Documents", you nagivate to the Process Flow of a billing document, click on a node such as Accounting Document and then choose "More Links". Now in the "Define Link List" screen, you select one or more links to add. When you press the OK button, however, the app becomes unresponsive.

There is an error of the form:

Uncaught Error: Generated ID attribute found - to offer flexibility a stable control ID is needed to assign the changes to, but for this control the ID was generated by SAPUI5 __component206---sapuicompnavpopoverNavigationPopover---BillingDocument

Other Terms

F0797, F1901, Manage Billing Documents, Process Flow, ID issue, SD_BILDOC_FSS1, Fact, Sheet, Factsheet

Reason and Prerequisites

The issue is caused by a missing ID in the process flow view of SD_BILDOC_FSS1 in OP1809

Solution

ID was added to the process flow view of SD_BILDOC_FSS1 for Billing Document Factsheet. Please also implement note 2918561 - "Billing Document Factsheet: The app hangs when you try to add more links (Part 1)" as well.
