Symptom

In the process "Receipt of Meter Readings as Part of Device Change or DSO Change Process," the system displays an error in step 0150-0050 when receiving a meter reading for an IM4G device change or removal: "Discrete MR for electricity IMS should only be sent from MOS View." As a result, the meter reading can't be booked.

This issue is relevant for the market role: SUP.

Other Terms

US4G, MET, IF_EXU_ISU_MR_FORMAL_CHECK, IM4G device change, IM4G device removal, meter reading, discrete meter reading

Reason and Prerequisites

The error occurred because of a program error.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the system books the meter reading successfully, and the process completes without errors. You need to retest the process. You can verify the fix by checking that the process completes successfully without displaying the error message.
