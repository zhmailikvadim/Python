Symptom

In transactions IW28 and IW29, the maintenance activity type field cannot be hidden as it does not appear in the variant selection fields list.

Environment

  • SAP ERP Plant Maintenance (PM)
  • SAP R/3
  • SAP R/3 Enterprise
  • SAP ERP Central Component
  • SAP ERP
  • SAP Enhancement package for SAP ERP
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA, on-premise
  • SAP S/4HANA, Cloud Private Edition

Reproducing the Issue

  1. Run transaction IW28.
  2. Create new variant 
  3. In the Create Variant screen 
  4. Search for "activity type" 
  5. No hits 

Cause

Maintenance Activity Type is displayed using it's technical ID SO-ILART


Resolution

Maintenance Activity Type is maintained using the technical ID SO-ILART 

  1. In the Variant Selection screen
  2. Search for SO-ILART
  3. Set to Hide
  4. Save variant 
  5. Run IW28 again and load newly created variant
