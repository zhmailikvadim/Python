Download of Build Artifacts

Prerequisites:

- (Windows 10/11): curl.exe
Test via a PowerShell terminal: curl.exe --version
- (MacOS/Linux): curl
Replace curl.exe with curl in the statements below.
- Create a Technical User in the [RBSC - Repository Management UI](https://ui.repositories.cloud.sap/www/webapp/users)

To download the build artifacts, use the following commands in a terminal (e.g., PowerShell for Windows, Terminal for MacOS/Linux).

Important: Replace <tech-user>:<basic-auth> with the actual credentials of your Technical User created in the RBSC.

> curl.exe -u "<tech-user>:<basic-auth>" https://73554900100200027046.mavensrv.base.repositories.cloud.sap/com/sap/hyperspace/ticketsystemAA/3.0.4/ticketsystemAA-3.0.4.mtar -o ticketsystemAA-3.0.4.mtar
Cross-Check SHA-256: 057e566af04e3324e205ad83ef3ebbf4194dd9c8046010efaa3aad6610adf0ae

After the downloads, follow the administration documentation for detailed deployment steps.
