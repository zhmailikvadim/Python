Symptom

A service type SAP Integration Suite(Advanced Event Mesh) is listed in Landscape Management.
In section "supported use-cases" it has Health Monitoring shown.
However in the Health Monitoring application there is no option to select that service type and so monitoring cannot be activated. 

Environment

  • SAP Integration Suite (Advanced Event Mesh)
  • SAP Cloud ALM - Health Monitoring

Cause

Resolution

It is a mistake that Health Monitoring is shown there.
The development team is working on it to support "Advanced Event Mesh" in Year 2026.

All the supported solutions can be found in online documents:
Help Portal of Supported Solutions
Expert Portal of Health Monitoring -Setup and Configuration
