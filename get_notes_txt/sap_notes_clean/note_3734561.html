Symptom

Function module EXIT_SAPLAPCF_006 is not available in the S/4HANA system.

Environment

  • SAP S/4HANA Finance
  • SAP S/4HANA

Reproducing the Issue

Use EXIT_SAPLAPCF_006 and observe that is it not available

Cause

In S/4HANA, as of release 1909,  the legacy user exit for changing asset document line items (EXIT_SAPLAPCF_006) from the classic system is not available; this functionality is replaced by BAdIs such as BADI_FIAA_DOCLINES and FAA_DOCLINES_CUSTOMER. The enhancement AINT0002 is also no longer usable and is replaced by FAA_DOCLINES_CUSTOMER.

For detailed information, refer to below SAP Note and portal links.
SAP Note 2270388 - S4TWL - Asset Accounting: Parallel valuation and journal entry
SAP Note 2960690 - Incomplete what’s new information in "BAdI: Customer-Specific Change to Line Items in Asset Document" (SAP S/4HANA 1909)

What’s New Viewer – S/4HANA 1909 (PDF)
BAdI: Customer-Specific Change to Line Items in Asset Document

Resolution

Consider to use relevant BADIs BADI_FIAA_DOCLINES and FAA_DOCLINES_CUSTOMER.
