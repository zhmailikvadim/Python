Symptom

This Knowledge Base Article (KBA) provides options for downloading files from STP2 reports in the .xlsx format rather than .xls format. "Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Reproducing the Issue

  • Human Capital Management (HCM)
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP SuccessFactors Employee Central Payroll
  • SAP S/4HANA

Resolution

Choose either of the below options.

  • Option 1: Select "Export" - "Spreadsheet" in STP2 report, choose the .xlsx format. Refer below screenshot.
  • Option 2: Set the parameter LIST_CONVERT_TO_DAT as .xlsx for the user, then download from "Export" - "Local File" - "Text with Tabs". See the below screenshot.

    Also see KBA 1622947 - Wrong file extension when saving data to a spreadsheet for detail on changing the setting for users.
