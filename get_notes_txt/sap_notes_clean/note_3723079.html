Symptom

SSAM currently does not support Screening process from mobile, it is currently a gap compared to EAM Fiori process where customer want the execution of Notification with Processing context “Minor Work with Screening” & "Emergency Work with Screening" as part of the Reactive Maintenance process. 
This note provides the additional authority checks required for processing Maintenance Request creation with these processing contexts in SSAM.

Other Terms

S4MOBADDON-8161, SSAM, EAM, /MERP/CL_PM_NOTIFICATION_OD, Processing context, Minor Work with screening, Emergency Work with screening, Maintenance Request, Authority check OBJECT 'I_EXCP_MN'.

Reason and Prerequisites

New functionality.

Solution

Please apply the attached correction instructions in SAP system.

Note: This note provides only backend objects. Additional configuration & MDK solution are required for this functionality to work for SAP Service and Asset Manager client.
Correction instructions include preparatory changes for release 107 with no functional impact. The functionality is only available from release 108 onwards.
