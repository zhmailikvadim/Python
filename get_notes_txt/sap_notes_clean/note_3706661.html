Symptom

When you report an incident in the Report Incident app with a past start date, the HR proxy call doesn't always return party basic data that's valid for that date. As a result, lr_s_party_basic_data isn't bound, but the code still tries to access lr_s_party_basic_data->group_desc. This causes a NULL data reference error.

Other Terms

Report Incident, bound, null, past date

Reason and Prerequisites

Reason:
Program error.

Prerequisites:
This is a fix for the code from SAP Note: 3443539.

 

Solution

The "Support Packages" section lists the Support Packages that contain these corrections.

Alternatively, you can implement the attached correction instructions.
