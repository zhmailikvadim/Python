Symptom

Due to the alignment of calculations and checks on the part of JMHZ, it is necessary to install SAP Note No. 3705364, which corrects the rounding method for employee social insurance contributions from considering hundredths to considering thousandths. In rare cases, specifically when the insurance contribution for the employee (EE) before rounding was in the range of x.xx1 – x.xx4, it was not rounded up to the nearest whole crown. This SAP Note must be installed before the closing of the 01/2026 month. The adjustment is valid from January 1, 2026.

Other Terms

HRCZ, RPCALCTM, RPCALCT0

Reason and Prerequisites

 This SAP Note is relevant only for the Czech Republic.

Solution

Important: Apply current version of SAP Note 875986 for NW 7.0x or 1668882 for NW 7.3x and higher. Otherwise an old version of SNOTE can lead to missing coding changes when applying SAP Notes.

  • As a general rule, SAP recommends you to install a solution by applying a Support Package. However, if you need to install the solution earlier, use the SAP Note Assistant to implement the correction instructions. You can find more information about the SAP Note Assistant in SAP Service Marketplace, under SAP Note Assistant.
  • Note that documentation, transactions, and Customizing activities are only delivered via Support Package.
  • To implement the solution, you must carry out manual activities (see attachment if it exists) and implement the correction instructions that are valid for your release.
