Symptom

Error message "Check table 159L: entry &  does not exist - Message No. M7001" or "Check table T159L: entry & does not exist - Message no. M7001" is sent when trying to post a goods movement or when trying to create a reservation.

Environment

  • Materials Management (MM)
  • Inventory Management (IM)
  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA

Cause

Although the Plant in the error message has been correctly created in the customizing of the Enterprise Structure:

SPRO
Enterprise Structure 
Definition 
Logistics - General 
Define Plant 

For some reason it is missing in the customizing for Inventory Management.

Resolution

Create the missing entry for the Plant under the IMG activity:

SPRO
Materials Management 
Inventory Management and Physical Inventory 
Plant Parameters 

Here click "New Entries", add the missing data for the Plant and save the record.

Keywords

Plant missing, 159L, T159L
