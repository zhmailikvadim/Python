Symptom

You execute a task for interunit reconciliation (IUR) or interunit elimination (IUE) with selection screen where you have switched on the option "Report". Task execution terminates in runtime error (shortdump GETWA_NOT_ASSIGNED).

Other Terms

GETWA_NOT_ASSIGNED, CL_UC_METHOD_REPORT_DATA======CP, CL_UC_METHOD_REPORT_DATA======CM002, ADD_DATA

Reason and Prerequisites

You are running IUR or IUE with task log as report.

Solution

Import of Support Package or advanced correction by implementation of correction instruction
