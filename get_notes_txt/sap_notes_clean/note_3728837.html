Symptom

In the process "Receipt of Settlement-Relevant Change with Dependencies from DSO to SUP as SUP" (44123, 55123), the load profile is not updated as expected in step 0150-200(Update Master Data). After the update step runs, the profile definition doesn't reflect the expected changes.

This issue is relevant for the market roles: SUP, DSO.

Other Terms

Profile Definition, Load Profile Allocation, Load Profile Factor, /US4G/CL_UBD_DATA_PROFILE_DEF, GET_DATA_FOR_UPDATE, PROFILE_DEFINITION, /US4G/VC_PROFDEF, /US4G/VC_PREFDTM

Reason and Prerequisites

The error occurred because the system was missing the functionality to update the load profile correctly.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the system updates the load profile as expected in steps 0150-200. You need to repeat the "Update Master Data" step or retrigger a new process. You can verify the fix by checking that the load profile is updated correctly after the update step runs.
