Symptom

The IDCNDOC report does not support exporting to Excel, although Excel output is expected.



Environment


  • IDCNDOC
  • China

Reproducing the Issue


  1. Start transaction IDCNDOC.
  2. Attempt to export the displayed result to Excel.

Cause


  • By standard design, IDCNDOC supports only ALV display, PDF preview, and Direct Printing.
  • Excel export is not provided because the report is intended for printing accounting vouchers for auditing purposes.

Resolution


  1. Excel export is not available for IDCNDOC by design.
  2. Use one of the supported output options: ALV display, PDF preview, or Direct Printing.

Keywords


IDCNDOC, export to Excel, Excel download, China localization, FI
