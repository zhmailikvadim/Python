Symptom

Entry not allowed error in KE4M as per below

Environment

Controlling (CO)
•SAP R/3
•SAP R/3 Enterprise 
•SAP ERP Central Component
•SAP ERP
•SAP enhancement package for SAP ERP
•SAP enhancement package for SAP ERP, version for SAP HANA
•SAP S/4HANA Finance
•SAP S/4HANA

Reproducing the Issue

  1. Go to transaction KE4M
  2. Try to assign a certain quantity field but it is not shown in the drop down menu.
  3. Manually enter ZMENG for example
  4. Error KE261 Displayed

Cause

It is not possible to take over the these quantity fields from SD:'BMENG', 'LMENG', 'WMENG', 'ZMENG', 'KWMENG_ORG', ‘/CWM/WMENG_BEW’. All other quantity fields from the sender structure 'S007' can be used. 

Resolution

To populate the quantity field ('BMENG', 'LMENG', 'WMENG', 'ZMENG', 'KWMENG_ORG', ‘/CWM/WMENG_BEW') in CO-PA, you can utilize the function module EXIT_SAPLKEAB_001 within the enhancement COPA0002. This function module allows you to populate the field effectively. Before proceeding, it is essential to add a user exit ID to your valuation strategy. You can do this by navigating to transaction KE4U.
