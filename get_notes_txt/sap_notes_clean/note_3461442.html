Symptom

  • There is binary data stored in database to store picture data. (E.g. BLOB, CLOB type data)
  • Is it possible to store picture binary data in the acquiring data model in SAP Analytics Cloud?

Environment

SAP Analytics Cloud 2024.2

Resolution

  • Currently it is not possible to store such kind of data in Acquiring data model.
  • There is also no such kind of properties available for such requirement now.
