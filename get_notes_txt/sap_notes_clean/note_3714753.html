Symptom

Open the Recipe Finder application and apply a dynamic filter. In the results table header, click “Analyze Result” and choose “By Search Item Quantity” in the popup.

In the Recipe Analysis window that opens, the results table includes an “Average Quantity (%)” column. The values in this column are not rounded and display up to 16 digits.

Other Terms

Ingredient, Material, Component Item,  Dynamic filter

Reason and Prerequisites

Program error

Solution

Implement the correction instruction or implement the support package of the SAP Note.
