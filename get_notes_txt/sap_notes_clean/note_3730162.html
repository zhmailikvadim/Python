Symptom

You need to enable the Additional Fiscal Data Framework DDIC objects so that the Additional Fiscal Data Framework table (GSLOG_LFD) can be used by Central Finance (CFIN). The structure CFIN_DATA_S, that contains the field LOGSYSTEM_SENDER, is added to structure GSLOG_LFD_TRANSACTIONAL_FIELDS, enabling the LOGSYSTEM_SENDER field usage within Additional Fiscal Data Framework by CFIN.

Other Terms

Additional Fiscal Data, DDIC, CFIN, LOGSYS, LFD

Reason and Prerequisites

You have implemented the SAP Notes:

  • 3586385: Additional Fiscal Data - DDIC PreReq Objects
  • 3425931: ODN Framework - Data Structure OND_DATA_S
  • 3551904: Additional Fiscal Data - DDIC
  • 3725342: ODN Integration: Structure adjustment

Solution

After implementing this SAP Note, the system enables the Additional Fiscal Data Framework data dictionary objects required for CFIN integration. The LOGSYSTEM_SENDER field is added to the Additional Fiscal Data Framework Transactional Fields table, and Central Finance can use the Additional Fiscal Framework Data table.

List of the affected objects:

Table Definition

  • GSLOG_LFD_TRANSACTIONAL_FIELDS

For more information, access the SAP Note 3562005 - Additional Fiscal Data - Delivery and Support Guidelines
