Symptom

Advanced Search on HANA for Shopping Cart BO: If EV_SKIP_RESULT is abap_true then HANA related method GET_BUFFERED_RESULT will be called instead of GET_RESULT in method of GET_RESULT_SC of class /SAPSRM/CL_CH_WD_SOSM_SC while performing Advanced Search. To fully exploit HANA performance method GET_RESULT should not be called for data enrichment because required data is already stored in buffer table.

Other Terms

/SAPSRM/WDC_UI_DS_SO_SC, V_SC_SYSTEM_RESULT, WDDOINIT, /SAPSRM/CL_CH_WD_SOSM_SC, /SAPSRM/IF_CLL_SOSM_SC~GET_RESULT_SC.

Reason and Prerequisites

program error.

Solution

Implement the Note or import relevant SP.
