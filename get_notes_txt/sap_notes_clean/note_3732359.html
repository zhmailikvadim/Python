Symptom

  • Statistics are not visible for OData v4 when using transaction /IWFND/STATS.
  • /IWFND/STATS displays usage data for OData v2 but no entries appear for OData v4
  • There is a requirement to monitor activity for OData V4 services in the SAP Gateway environment

Environment

  • SAP Gateway (SAP_GWFND)
  • SAP NetWeaver
  • SAP S/4HANA

Reproducing the Issue

  1. Start transaction /IWFND/STATS
  2. Review the statistics list and observe that no activity is recorded for OData v4 services

Cause

There is no transaction to monitor OData v4 services individually. /IWFND/STATS only displays OData v2 usage data.

Resolution

Monitoring for OData V4 services is not available via a dedicated transaction code.

As an alternative, use transaction STATS:

  1. Execute transaction /nSTATS
  2. Select Task Type T (HTTPS) or H (HTTP)
  3. Display the list of requests
  4. Select a relevant entry to view details
  5. Find the Gateway tab for OData related information
