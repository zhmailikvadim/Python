Symptom

Short dump "ITAB_LINE_NOT_FOUND" is raised in class CL_FINCS_BCF_CORE_LOGIC when running balance carry forward. Which looks like below:

Category: ABAP programming error
Runtime Errors: ITAB_LINE_NOT_FOUND
Exception: CX_SY_ITAB_LINE_NOT_FOUND
ABAP Program: CL_FINCS_BCF_CORE_LOGIC=======CP
Application Component: FIN-CS-COR

Environment

  • Group Reporting
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Navigate to Data Monitor.
  2. Set the Global Parameters.
  3. Right-click on the Balance Carry Forward (BCF) .
  4. Select "Test Mode" or "Update Mode".
  5. Observe the short dump "ITAB_LINE_NOT_FOUND".

Cause

Incorrect customizing in "SAP S/4HANA for Group Reporting -> Balance Carryforward -> Specify FS Items to be Carried Forward".

Resolution

  1. Navigate to transaction code CXS3.
  2. For every row maintained in "Specify FS Items to be Carried Forward", ensure the FS item/FS item role is maintained correctly.
