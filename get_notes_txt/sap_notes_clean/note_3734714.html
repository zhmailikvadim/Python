Symptom

  • A table added to a data product shows only the technical name in the Name column of the Product Artifacts tab.
  • The table’s business name/description is not displayed in the Product Artifacts tab.
  • No error message is shown.

Environment

SAP Datasphere Data Marketplace

Reproducing the Issue

  1. Open a data product.
  2. Navigate to the Product Artifacts tab.
  3. Add a local table that has a maintained business name/description.
  4. Observe that, after adding, the Name column displays the technical name while the business name/description is not shown.

Cause

A known product defect in affected versions prevents the business name from being set/displayed for tables added to data product artifacts. A fix is delivered with version 2026.08.

Resolution

  1. Update the system to version 2026.08 or later.
  2. Create new data products after the update; the business name will be set correctly for newly created data products.

For existing data products created before the update, request a manual metadata update from SAP Support and provide the ids or names of the impacted data products so they can be corrected.
