Resolution

Note: The field “Employee Status” is defined in Corporate Data Model (CDM) with id “emplStatus”. The translations are to be added in this data model.

Step1: Create Custom Employee Profile or userinfos

  1. Go To BCUI or SDM Data Model
  2. Under 
  3. Employee Profile
  4. Use custom field or Userinfo Element field Here 2363922 - How to create a HRIS Sync Mapping between an Employee Central field and an Employee Profile Using Userinfo Element field

Step 2: Grant RBP Permission 

  1. Go to Permission role and grant a Permission "View" under: Employee Data

Step 3: Add Field in Configure Latest People Profile 

    1. Go to Admin Tool Configure Latest People Profile 
    2. Select profile portlet 
    3. Live profile User information
    4. Add field 
    5. save

Step 3: Export SDM and add Hris sync Mapping only Via Data Model

  1. 3066117 - How to export Data Models?

      </hris-mapping>
