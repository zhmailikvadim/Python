Symptom

Encounter an issue where the source objects of the data source are not accessible or visible under the Database Privileges. This means that although the data source is defined, the underlying objects (such as tables, views, or procedures) that support it cannot be directly viewed or managed due to missing or insufficient privileges

Other Terms

View,Database Privileges, source objects of the data source, /VIZI/DB_PRIVILEGES_GRANT

Reason and Prerequisites

This is caused by program error

Solution

Please implement the correction instructions attached. These corrections are only valid if you have applied the following support packs.

Release V4 - 2023 V2 and 2023 V3

Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object.
