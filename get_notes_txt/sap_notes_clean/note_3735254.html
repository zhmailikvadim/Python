Symptom

After updating HANA Cockpit to version 2.18.6, the Database List on the Database Directory page displays incorrect values in some columns, such as Type/Version, Credentials, SAP Control Credentials, etc.
Additionally, the "Database Management" and "SQL Console" links are missing. 


In normal circumstances, you would expect to see the following screen:


"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Environment

SAP HANA Cockpit 2.18.6

Reproducing the Issue

  1. Update HANA Cockpit to version 2.18.6.
  2. Open the Database Directory page in HANA Cockpit.
  3. Observe that columns after “Groups” display incorrect values and that the “Database Management” and "SQL Console" links are missing.

Cause

This is a known issue.

Resolution

The fix will be delivered in the next patch of HANA Cockpit. For more details, refer to KBA 3152301 - Components version information in recent SAP HANA Cockpit releases/patches.

As a workaround, open the database resource and find the "Database Management" and "SQL Console" links in the upper-right corner of the Database Overview page.
