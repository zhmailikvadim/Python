Symptom

  • From the public cloud edition 2602 the filter options 'Consider Intraday Reconciliation Results' and 'Consider End-of-Day Reconciliation Results' are missing in the Cash Flow Analyzer app.
  • How to enable the filter options 'Consider Intraday Reconciliation Results' and 'Consider End-of-Day Reconciliation Results' in the Cash Flow Analyzer app.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Open the Cash Flow Analyzer app.
  2. Check for the filter options 'Consider Intraday Reconciliation Results' and 'Consider End-of-Day Reconciliation Results'.
  3. Observe that these filter options are not visible.

Cause

  • With the public cloud edition 2602, the filter options 'Consider Intraday Reconciliation Results' and 'Consider End-of-Day Reconciliation Results' require enabling the 'New Reconciliation Model' in the configuration activity 'Define Basic Settings'.

Resolution

  1. Navigate to the configuration activity 'Define Basic Settings' (SSCUI ID 101025).
  2. Select the checkbox 'Enable New Reconciliation Model'.
  3. Save the changes.
