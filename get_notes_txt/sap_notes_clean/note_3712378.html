Symptom

When creating a Task from a Playbook suggestion on an Opportunity, the Start Date/Due Date is in the past.

Environment

SAP Sales Cloud Version 2

Reproducing the Issue

  1. Open an Opportunity.
  2. Create the Task from the Playbook suggestion.
  3. Observe that the Start/Due Date is in the past.

Cause

The Playbook-generated Task Start Time is based on the time when the Opportunity reached the corresponding Sales Phase.

Resolution

This is the expected behaviour of the system.

As a workaround, either:

  • Change the suggested Start Date/Due Date while creating the task, or;
  • Change the duration of the Task under General Settings -> Task -> Start Date & Time
