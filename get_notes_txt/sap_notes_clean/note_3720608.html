Symptom

The text descriptions for wage types M840 and M841 are identical.

Other Terms

V_512W_T, HRCZ

Reason and Prerequisites

This SAP Note is relevant for the Czech Republic only.

Solution

The description for wage type M841 has been revised to provide clarity on its application. This wage type should be used when an employee meets the following criteria:

  • Has an existing voluntary retirement contribution (pension insurance) recorded in IT0011
  • The voluntary contribution was concluded after January 1, 2024
  • The employee is now also applying the mandatory contribution

In such cases, please use wage type M841 to record the mandatory contribution.
