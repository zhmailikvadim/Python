Symptom

Runtime error at execution of program  RHCHECKP.

Text: The ABAP program lines are wider than the internal table.

Runtime: Error READ_REPORT_LINE_TOO_LONG .

Exception: CX_SY_READ_SRC_LINE_TOO_LONG .

Other Terms

SAPLSKBH, K_KKB_FIELDCAT_MERGE

Reason and Prerequisites

Two lines in the program RHCHECKP exceed 72 characters due to added pragma.
This exist from release SAP_BASIS 756.

The occupied line length in the program text must not exceed the width of the internal table.
Internal table L_ABAP_SOURCE in the function K_KKB_FIELDCAT_MERGE is 72 characters.

Solution

Apply attached correction instruction or import corresponding support package.
