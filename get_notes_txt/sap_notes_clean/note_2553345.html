Symptom

Lumira Server BIP uses old SAPJVM which needs to be updated.

Other Terms

Lumira Server , sapjvm_7

Reason and Prerequisites

Lumira Server deployed on the BI platform uses an older version of SAPJVM (SAPJVM7) in folder "<Install Directory>\SAP BusinessObjects Enterprise XI 4.0\win64_x64\sapjvm_7" and this needs to be updated to newer version (SAPJVM8).

 

Solution

Runtime JVM has been updated to 8 and will be available in 2.0 SP1
