Symptom

I would like to confirm if I'm using Custom Risk Categories scoring or Legacy Scoring in SAP Ariba Supplier Risk

Resolution

Check if the parameter Application.SR.Risk.EnableCustomRiskCategories is enabled on your site.

  • If the parameter is enabled, you are using Custom Risk Categories scoring.
  • If the parameter is not enabled, you are using Legacy scoring.

You cannot revert back to Legacy scoring once Custom Risk Categories scoring is enabled.
