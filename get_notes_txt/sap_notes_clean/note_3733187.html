Symptom

When attempting to copy a freight agreement line item, a new rate table with maintained scales and rates is also generated (if source line item has rate table, scales and rates maintained). However, performing a 'Check' triggers the error e785(/scmtms/tc_rates): 'Rates already maintained; cannot add, update, or delete scales.

Other Terms

rate table, RT, scale item, line item, mv_chg_scale_item,  mv_chg_scaleref, rate scale reference

Reason and Prerequisites

Program error

Solution

 Implement the correction instructions or upgrade to the latest support package
