Symptom

 

Is it possible for all users to turn off all the automatic messages that are sent by SAP Signavio Process Governance?

Resolution

It is possible to restrict which types of e-mails are automatically sent by SAP Signavio Process Governance. Please note that it isn't possible for all notification types.

The following two links describe which types of messages can be restricted and how to set it up.
