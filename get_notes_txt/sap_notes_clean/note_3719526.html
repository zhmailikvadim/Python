Symptom

In the process "Check of EEG MaLo with Direct Selling Obligation for 100% Supplier Assignment (SUP View)" (DE_B_MPES_OBS_FOR_DS_OBLIGATION_SUP), the system is missing prerequisites required for the process to run, particularly when customer extensions are used. Additionally, the waiting time is incorrect in step 1100-0140.

This issue is relevant for the market role: SUP.

Other Terms

 MPES, EEG MaLo, direct selling obligation, 100% supplier assignment, DE_B_MPES_OBS_FOR_DS_OBLIGATION_SUP, waiting time, customer extensions, prerequisites

Reason and Prerequisites

The error occurred because the process was missing prerequisites necessary for execution, especially when customer extensions were implemented. Additionally, the waiting time required manual intervention.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the system includes the missing prerequisites in steps 1100-0050, 1100-0090, 1100-0140, and 1100-8950, and replaces the waiting time with the correct value. Import the BC set /US4G/4_0_SUP_SP08_NOTE_3719526 to apply these corrections. You can verify the fix by checking that the waiting time in step 1100-0140 is correct and that step 1100-8950 includes the necessary check.
