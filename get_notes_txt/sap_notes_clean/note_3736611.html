Symptom

  • Ticket Benefit Error : TBCAL --> Wage Type Missing for Ticket Beneficiary ID
  • Error message observed: "Wage type missing for ticket beneficiary ID".
  • After enabling 'When Paid' in Ticket benefit getting error "Wage Type Missing for Ticket Beneficiary ID"
  • SAP HCM - Payroll Run Error: Wage type missing for ticket beneficiary ID
  • HCM - Ticket benefits issue
  • UAE Payroll TBCAL function throwing error as Wage type missing for Ticket Beneficiary ID

Environment

  • Human Capital Management (HCM)
  • Payroll India
  • SAP R/3
  • SAP ERP Core Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA

Reproducing the Issue

Disclaimer: Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

  1. Execute transaction PC00_MAE_CALC (UAE payroll driver).
  2. Getting error in Payroll Calculation

Cause

Missing wage type mapping for the relevant ticket benefit beneficiary ID combination in table V_T77TB06, or an incorrect Ticket ID assigned by feature TBTID.

Resolution

Wage type missing for ticket beneficiary ID - This error is seen when the customizing for a particular set of values of Ticket Subtype and ID does not have a wage type mapped to it in the table V_T77TB06.  

If you have activated the 'when paid' method for ticket benefits. The feature TBMTH has been maintained for UAE with the date 01.01.20XX. If you activate the 'when paid' method then the system will expect you to maintain the wagetype for the when paid calculation method in the configuration table SM34 --> VC_T77TB02. 

Sometimes, there might be cases where the default Ticket ID assigned by the feature TBTID might be different from the one desired by the business users and thus, in that case, the problem would be incorrect entry for the Ticket ID in this feature.  However, in case the value maintained for the Ticket ID in the feature TBTID is correct, then please check the customizing in table V_T77TB06 ( SM34 --> VC_T77TB02).  

Alternatively, you can check the customizing by going to the following IMG node:  Payroll - <Country Name> -> Ticket Benefit -> Define Ticket Benefit Subtypes and Beneficiary IDs

Refer Help.sap.com Page link for details for what all WT to be maintained and customizing guidance
