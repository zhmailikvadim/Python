Symptom

When you execute the program "Copy Production Models", the system issues the message "... does not correspond to material type ...".

The message is issued if there is no valid record type in the target model.

SAP Dairy Management by msg --> Dairy Costing --> Tools --> Copy Production Models

Other Terms

Warning message

Reason and Prerequisites

Program error

Solution

History:

Version 1 from October 30, 2020: Initial data creation

Implement the source code corrections.

Solution:

The check has been refined and the message is now more informative.

The system issues the message if no suitable record type was found in the target.

In this case, the entry is not processed further.

There are no technical dependencies.
