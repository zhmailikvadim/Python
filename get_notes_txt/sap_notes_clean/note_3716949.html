Symptom

  • Depreciation Area 15 is identified as a group asset area.
  • Settlement with transaction AIBU did not transfer values in area assigned to group asset, the values in the receiving asset is 0.
  • The issue occurs with line item settlement for group assets.

Environment

  • FIN Asset Accounting
  • SAP ERP
  • SAP ERP Central Component
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA
  • SAP S/4HANA Finance
  • SAP S/4HANA Cloud Private Edition
  • SAP Fiori for SAP S/4HANA Finance,
  • SAP Fiori for SAP S/4HANA,
  • SAP Fiori for SAP S/4HANA Cloud Private Edition

Cause

The root cause lies in the assignment of the asset under construction with line item settlement to a group asset. Group assets do not manage depreciation, and assigning them to such assets is not expected in the settlement process, leading to the issue.

Resolution

  1. Refer to SAP Notes 3014869 and 3532580 for the official SAP strategy regarding group assets functionality.
  2. Plan to use one of the alternative solutions described in SAP Note 3532580.
  3. For incorrectly created group assets with such assignments, avoid using transactions AIAB/AIBU.
  4. Use the appropriate transaction to transfer values from the asset under construction to the final asset.
