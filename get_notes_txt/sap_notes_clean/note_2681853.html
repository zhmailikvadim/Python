Symptom

In the fine tuning activity Assignment of Activities to Sales Phases, you customized the descriptions for sales activities. When you add the related activities to an opportunity, the descriptions are not displayed as maintained in the fine tuning.

Environment

SAP Hybris Cloud for Customer

Reproducing the Issue

  1. Go to the Sales work center.
  2. Go to the Opportunities view.
  3. Create a new opportunity by selecting the New button.
  4. Save and open.
  5. Go the the Sales Activities facet.
  6. In the Activity Advisor, add a task for which you have maintained a customized description in fine tuning.
  7. Save.

Result: The task decription does not correspond to the fine tuned description.

Cause

The text displayed in the UI is taken from Detailed Description in the fine tuning activity for Sales Assistant.

Resolution

You need to change the detailed description instead of the description in the fine tuning activity Assignment of Activities to Sales Phases in Business Configuration.

The text of the detailed description maintained is displayed in the UI after the changes.

Keywords

Sales Assistant, task description, activity description, sales activity, sales activity fine tuning
