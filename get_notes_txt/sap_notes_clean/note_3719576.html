Symptom

You are working with the Last Mile Distribution solution. When you settle a route, the settlement run fails immediately and the system displays the following error message:

An error occurred. Contact the help desk to check transactions ST22/SM21.

In transaction ST22, the Error Analysis section contains the following information:

The exception is related to the previous exception "CX_SY_ITAB_LINE_NOT_FOUND" that occurred in  program "LMD_RTSL_CR_WP_HELPER=========CP", in line 5 of include "LMD_RTSL_CR_WP_HELPER=========CM00Q".The reason for this exception was:

The table does not contain a row with the specified values.

Other Terms

Last Mile Distribution, LMD, Settlement

Reason and Prerequisites

The issue is typically caused by missing authorizations for the user who runs the settlement. As a result, sales orders cannot be read from the database.

Solution

Please implement the related correction instruction to resolve the described issue.
