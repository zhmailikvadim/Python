Symptom

Version:

  • March 23, 2026: Added a note regarding the basic configuration from the corresponding SAP Note for the PDF form.

This SAP Note delivers the SAPscript templates for generating the payslip for leavings in 2026:

  • HR_AT_LZ_GLZ_16
  • HR_AT_LZ_GLZ_16K
  • HR_AT_LZ_GLZ_02K
  • Special adjustments due to SAPscript print process

It also delivers the following corrections/technical improvements:

  • Modularization of FABOP child assignment with more than four children (PDF and SAPscript)
  • Corrections to T5A1M:
    • Discrepancy between fields BMAPR and TMAPR
    • Inclusion of field FLABZ, internal: KFLAB 

Other Terms

PCALZ, SAPScript, Payslip Tax (L16)

Reason and Prerequisites

Legal change

Solution

Import the corresponding HR Support Package (or CLC) into your SAP system.

For an advance correction, proceed as follows:

Refer to the additional SAP Notes that are listed in SAP Note 3705010 (see also note analyzer of SAP Note 3705010).

The basic configuration for SAPScript is already contained in SAP Note 3720588. From that SAP Note, load at least the BC set as described there.

  • First, carry out the advance corrections described in the manual instructions.
  • Then upload the attached correction instruction into your SAP system using transaction SNOTE.
  • Then carry out the post-implementation steps described in the manual instructions.
    Refer to the attachment to this SAP Note (binary files of the forms).
