Symptom

In change request, an error is thrown when object is read after currency value is changed with negative sign.

Other Terms

CMS_WB, CR

Reason and Prerequisites

When the changed value gets retrieved from the change requests it gets converted from a character type to the target type. This conversion operation does not allow currency fields to have negative values.

Solution

The program logic is changed to handle signed currency values.

Please implement the attached correction instructions or import relevant support package.
