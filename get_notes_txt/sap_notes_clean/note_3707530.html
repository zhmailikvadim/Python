Symptom

  • After performing a conversion from ECC to SAP S/4HANA, certain environments display missing content(JA) in the info object level and header.
  • The table-item names are displayed instead of the expected content in the info object level and header.
  • In some environments, the original content of the table-item text information is retained.

Environment

  • SAP S/4HANA
  • SAP NetWeaver Application Server for ABAP

Reproducing the Issue

  1. Perform a conversion from ECC to SAP S/4HANA.
  2. Execute a query in the converted environment.
  3. Observe that the info object level content and header are missing in some environments, while table-item names are displayed instead.
