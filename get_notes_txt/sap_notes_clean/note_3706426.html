Symptom

  • I cannot approve work item because Approve button is missing on the work item.
  • I received notification for approval of the work item but Approve button is missing. 

Resolution

The notification is sent to the user depending on the Messaging settings from Admin Configuration menu.

If the user or the User Role of the user is selected as Selected Recipients for Message, notification will be sent to the user regardless of user's permission to approve item.

You will be able to check the Messaging notification settings by following steps below.

  1. Admin Configuration > Messaging
  2. Search for the notification settings depending on the work items module.
  3. Check the section Selected Recipients for Message

Next check the item's Approval/Audit Trail tab for the work item's current approval level. 

If the work item is not at the approval level of the user, Approve button will not be visible as the work item is still at lower level of approval process.
