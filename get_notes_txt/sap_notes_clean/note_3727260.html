Symptom

Freight Document: dump when copying after building customs group with deviating address e.g. for Ship-to Party.

Other Terms

/BOBF/CX_FRW_FATAL, RAISE_SHORTDUMP, copy control, customs activity, customs group

Reason and Prerequisites

Program Error.

Solution

As Customs Activity is rebuilt anyway after a document copy, copying this data is now deactivated (performance improvement).

Implement the attached correction instructions or install the corresponding Support Package.
