Symptom

Inventory Audit Report displays empty results, when user language is set as French and the Enable Quick Display checkbox is selected.

Reproducing the issue

Prerequisites:

You are running SAP Business One for SAP HANA.

Scenario:

  1. Add a goods receipt for any item available.

  2. Set the user language to "French".

  3. Run Inventory Audit Report with the following selection criteria:

  • Code: Item from the goods receipt created in step 1.
  • Display by Items.
  • Checkbox "Enable Quick Display" is selected.

Expected behavior:

Inventory Audit Report displays the transaction created in step 1.

Actual behavior:

Inventory Audit Report displays empty results.
