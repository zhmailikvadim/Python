Symptom

In work agreement, it only shows the active status, but since the "End Date" has already passed, the inactive status needs to be displayed.

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental. 

Environment

S/4HANA Cloud

Resolution

The status defines the status of the employment for the given start date and end date and not for the current date.

If it is requested to see the inactive status for the time period where the employment is not active, then kindly check it in Worker Overview application. There a show inactive button can be found in the work agreement section.

Justification for the above claim:

For example, an employee was hired on 01.01.2020 and terminated on 31.12.2022.

In the above case two records would be created in work agreement as below,

01.01.2020 to 31.12.2020 - Active

01.01.2023 to 31.12.9999 - Inactive

Now check when the employee was terminated, go to Worker Overview application and see the inactive period start date as 01.01.2023 and come to a conclusion that the employee was terminated on 01.01.2023.
