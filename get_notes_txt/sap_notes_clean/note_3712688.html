Symptom

When maintaining job description and posting, the text overflows the text box in the career site pgae.

Environment

SAP SuccessFactors Recruiting Management

Cause

The issue arises due to the wrong use of HTML code.

 

Resolution

Ensure the job description uses <p> </p> tag, not <pre> </pre> tag in source code.

Keywords

RCM, Recruiting Management, job requisition, job description, text, field, value, overflow, career site, source code, HTML code, tag, posting
