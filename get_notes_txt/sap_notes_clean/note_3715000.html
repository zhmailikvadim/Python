Symptom

  • New hires initiated manually (via Admin Center > Add New Hire to Onboarding) are not visible in the Onboarding Dashboard.
  • Multiple processes remain in an "Unfinished" status in Admin Center > BPE: Monitor Processes, with a Job ID associated.

Environment

SAP SuccessFactors Onboarding

Reproducing the Issue

  1. Go to Admin Center > Add New Hire to Onboarding
  2. Enter the new hire's Basic Information
  3. Initiate Onboarding
  4. Results:
    • The new hire is not visible in the Onboarding Dashboard.
    • In Admin Center > BPE: Monitor Processes, multiple processes remains "Unfinished", with a Job ID associated.

Cause

The issue occurs when a BPE Service User is not added to the OnboardingBPEServiceUser permission group or is inactive in the system.

@SAP Support: Please refer to the internal memo for sample back-end logs related to this issue.

Resolution

Verification:

  • Verify that a valid BPE Service User is added to the OnboardingBPEServiceUser permission group under Admin Center > Manage Permission Groups.

If no user is assigned, complete the following steps:

  1. Navigate to Admin Center > Manage Permission Groups
    1. Add a valid BPE Service User to the OnboardingBPEServiceUser group.
  2. Go to Admin Center > Update Process Flow for Onboarding.
    1. In the "Select Service User" field, select the BPE Service User added to the permission group.
    2. Click Update
  3. Navigate to Admin Center > BPE: Manage Process Tasks.
  4. Restart all unfinished tasks.
