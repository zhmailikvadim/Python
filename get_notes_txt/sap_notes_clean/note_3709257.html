Symptom

You create a node or variant name with lowercase letters using the integration SVasPPECMPPVS. The SV description, components group ID, or customizing fields have values with lowercase letters.

When you open the object using the TA PPE search and enter the full name with lowercase letters, the search does not find the object.

Other Terms

PLMSI, PLM System Integration, PPE, Access Node Name, Structure Node Name, Access Variant Name, Component Variant Name, lower case

Solution

Implement this SAP Note to enable auto conversion to upper case for node or variant names that include lowercase letters.
