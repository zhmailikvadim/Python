Symptom

When you run report RPLDSNF0, the system displays an empty frame with the title "Génération de conteneur ECATT" at the bottom of the selection screen.

Other Terms

RPLDSNF0_SEL, RPLDSNF0_SEL_FORMS

Reason and Prerequisites

The system does not hide an empty line parameter in the SAP internal selection block with the title "Génération de conteneur ECATT" as intended. As a result, the frame with this title appears at the bottom of the selection screen.

Solution

After implementing this SAP Note, the system no longer displays the empty frame with the title "Génération de conteneur ECATT" on the selection screen of report RPLDSNF0.

The correction included within this SAP Note is part of the HR Support Packages referenced below.
If you need the correction before the HRSP is available, implement the attached correction instructions. Note that correction instructions are a short-term solution and not a substitute for the Support Package.
