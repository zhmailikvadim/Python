Symptom

After opening a recipe and navigating to the Formula tab, you select the Input and Output subtab. In the table header you notice that the Copy and Paste buttons are unlabeled.

Other Terms

/PLMU/WDC_RFO, V_ITEM_TABLE, BTN_COPY_ROW, BTN_PASTE, $OTR, OTR

Reason and Prerequisites

Program error

Solution

Implement the correction instruction or implement the support package of the SAP Note.
