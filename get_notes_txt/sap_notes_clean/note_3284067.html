Symptom

  • The following error occurs while trying to publish existing query in Custom Analytic Queries app:

Query validation
Query could not be saved. Try again.

  • In HTTP network trace there is following error with OData batch all of service ANA_QUERY_DESIGNER

Item ID XXX is already in use. Choose a different one.

Environment

SAP S/4HANA

Cause

Query is in inactive state in ADT. This could have been caused by network or system problems.

Resolution

Please go to ADT to activate the query.

Keywords

ValidateCreateAQAndSuggestSqlName, artifact
