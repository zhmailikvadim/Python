Symptom

As of SaaS Release Version 12.1, according to Release Notes, it is possible to use your own graphics in diagrams. I would be happy to use this feature, but haven't found out how to do that.


Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Solution

Only users with an Enterprise or Enterprise Plus license can use their own images for BPMN 2.0 diagrams, customer journey maps and value chains. An upload is made under Setup-> Notations/Attributes. After selecting a modeling language and a diagram element type, you can upload your own graphics. Only SVG graphics with a maximum file size of 20KB can be used.


You can only use your own graphics for elements (diagram element type) that are not contained in the BPMN 2.0 standard.

For the value chain modeling language, this would be:

  • Process and Collapsed Process

For Modeling Language Process Diagram BPMN 2.0

  • IT System and Additional Participant

For the modeling language Customer journey map

  • Persona,Touchpoint,DecisionPoint,Customer and Decoration
