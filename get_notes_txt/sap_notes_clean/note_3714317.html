Symptom

  • Inquiry regarding SAP IQ support for AWS Snapshot backups for restore, specifically using AMI and AWS Snapshot as a backup strategy.

Environment

  • Sybase IQ

Resolution

  • EC2 snapshots of a running database are not supported by SAP and will likely result in problems.

  • The only valid ways to back up an SAP IQ database are:
    1. Shut down SAP IQ and copy all files to a new location.

    2. If SAP IQ is running, use the BACKUP command as the only valid method to take a backup.
      • Using the BACKUP command with the VIRTUAL clause allows the use of snapshots or other non-IQ methods to copy user data files. This requires an OS script or command mechanism to be provided to the SAP IQ BACKUP command.
