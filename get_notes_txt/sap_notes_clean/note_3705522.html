Symptom

The registered products in service order ABC are not visible in the data source "CRMSORIB" (ABC represents the ID of the service order)

Environment

SAP Business ByDesign

Reproducing the Issue

  1. Navigate to the Service Orders work center.
  2. Go to the Service Order Processing view.
  3. Search for the service order "ABC." (ABC represents the ID of the service order)
  4. Click "Edit," and navigate to the General tab.
  5. Under the "Accounts and Affected Product" sub-tab, the affected products are listed; for example, 8 registered products are maintained, with one designated as the main product.
  6. Navigate to the work center: Business Analytics.
  7. Select the view: Data Sources.
  8. Set the display to Show: All.
  9. Search for the data source "CRMSORIB" and click "Preview."
  10. In the selection criteria, enter the service order "ABC." (ABC represents the ID of the service order)
  11. Select the following fields for viewing:
    • Service Order
    • Affected Registered Product
    • Registered Product ID
    • Main

It will be observed that only the main registered product is displayed.

Cause

The data source "CRMSORIB" includes a query definition designed to retrieve the main registered product.

Resolution

The expect system behavior.

There are no other special reports available that display the other registered products associated with a service order.

Keywords

Data source, service order
