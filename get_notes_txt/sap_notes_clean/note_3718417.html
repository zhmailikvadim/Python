Symptom

When trying to create new Manage Pay Increase "Name of Pay Change already exists" error pops up

Environment

  • SAP SuccessFactors Employee Central - Job Information
  • SAP SuccessFactors Employee Central - Pay Scale Increase

Reproducing the Issue

  1. Navigate to Adjust Employees Compensation to Tariff Changes.
  2. click create new.
  3. Fill all the required details like name, Pay Scale Changes From, Employee group, Pay Scale Type, Pay Scale Area, Pay Scale Group.
  4. On clicking save error pops up "Name of Pay Change already exists".

Cause

Please navigate to Configure Object Definitions and search for the object PayScalePayIncreaseDefinition.

Ensure that the Secured field is set to Yes.

Resolution

Grant permission for the PayScalePayIncreaseDefinition object in the required Manage Permission Role, then proceed to create the pay increase.
