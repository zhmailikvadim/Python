Symptom

You are requested to upload a TXT file of P.P.30 Attachment (ใบแนบ ภ.พ.30) to the e-Filing system of the Revenue Department of Thailand.

When you generate additional filings using the VAT Return – PP30 report and the VAT Return – Extended – PP30 report in the Run Statutory Reports app, you realize that the P.P.30 Attachment file is in the PDF format.

This note provides information about the field mappings from the P.P.30 Attachment PDF file converted to the TXT format.

Environment

SAP S/4HANA Cloud Public Edition.

Cause

This SAP KBA is related to the legal change in Thailand.

This SAP KBA is relevant only for Thailand.

Resolution

DISCLAIMER: Customers are solely responsible for evaluating the applicability of the information provided and for implementing appropriate testing, validation, and change management procedures in their specific environment. This guide provides technical formatting instructions only and does not constitute tax advice. Please consult with your tax advisor for tax compliance matters.
