Symptom

During allocation table creation in the WA01 transaction, the Plan Quantity is not populated for Structured Articles, whereas Plan Quantity is populated for the Variants.

Environment

  • Retail
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA

Reproducing the Issue

  1. Attempt to create an allocation table in WA01.
  2. Observe that Plan Quantity does not populate for Structured Articles, while it populates for Variants.

Cause

When an article has variants, its quantity and value become statistical, and only the quantities and values from the variants are relevant.

Resolution

Standard behavior.
