Symptom

HANA database (DB) is hosted in cloud environment, and you need information on its operating system (OS) version for auditing purpose.

Environment

  • SAP HANA, platform edition 1.0
  • SAP HANA, platform edition 2.0

Cause

Resolution

  1. Connect directly to HANA DB with a user that has the INIFILE ADMIN privilege.
  2. Query the OS information (e.g. OS name and kernel version) from system view M_HOST_INFORMATION via SQL:

       select * from m_host_information where key like 'os%';
