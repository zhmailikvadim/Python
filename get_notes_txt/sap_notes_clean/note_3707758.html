Symptom

You want to use Third-Party Payment functionality and there is no option to import the Third-Party payment file via manage incoming payment file (MIPF) app

Other Terms

Third-Party Payments, manage incoming payment file, MIPF

Reason and Prerequisites

Reason: You want to use Third-Party Payment framework

Pre-requisites:

3698442 - Third-Party Payment framework-prerequisite objects

3685291 - Third-Party Payment framework

3700115 - Third-Party Payments - MIPF(Manage Incoming Payment Files) extension

Solution

As a general rule, SAP recommends that you install a solution by applying a Support Package. However, if you need to install a solution earlier, use the Note Assistant to implement the correction instruction. You can find more information about the Note Assistant in SAP Service Marketplace, under service.sap.com/note-assistant.

After installing the SAP Note, you will be able to import incoming payment file of type 'Third-party payment' in Manage Incoming Payment file(MIPF) fiori application. When you click Upload button, you will see a new option 'Third-Party Payment'. Upon selection, you can choose the right format and upload the file in the next popup screen to process the file further.
