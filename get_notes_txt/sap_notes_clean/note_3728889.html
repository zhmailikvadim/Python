Symptom

This Release Information Note (RIN) contains information and references to notes for implementing Support Package Stack 38 of SAP NetWeaver 7.31

Note: This note is subject to change.

Other Terms

SAP Netweaver 7.31 SP Stack 38 implementation.

Reason and Prerequisites

You are planning to implement Netweaver 7.31 SP38. Before initiating the update/upgrade scenario, SAP recommends you check the SP equivalence for this SP Stack - 1653343

Please refer 2309320 for PI CTC: ABAP Dumps WF-BATCH MESSAGE_TYPE_X after installation, update or upgrade.

Also refer to https://help.sap.com/viewer/62c8d2b1a71046a09b9c7ec745910ae4/Latest/en-US/d1facd2bef6a4179a728252eaa651556.html --> for 7.31 the UDI scenario (up to date installer) is not supported.

 

Solution

This SP stack can be downloaded via SAP Service Marketplace

It contains corrections for several known issues. For more details please see below:

Non-ABAP:

The Maintenance for the Java stack has ended and hence no new SPs would be released.

Please refer to SAP Note 3078654 for Netweaver releases with redelivered JAVA stacks.
