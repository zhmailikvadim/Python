Symptom

When trying to create a transfer order via transaction LT03 for a delivery which is relevant for Advanced Shipping and Receiving and is not released for warehouse execution error message VLA 079  'Delivery is not released for Warehouse Execution' is issued. Nevertheless it is possible to continue with the creation of the transfer order.

Other Terms

VLA079, LIKP-TM_WHSE_EXEC, TM_WHSE_EXEC

Reason and Prerequisites

Program error

Solution

Implement the correction instruction relevant for your release.
