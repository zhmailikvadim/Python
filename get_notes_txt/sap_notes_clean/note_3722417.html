Symptom

This SAP Note provides you with the relevant system changes to address the following issue:

You run the Balance Sheet / Income statement app for China with a large number of lines in the Universal Journal Entry Line Items (ACDOCA) table. You select Income Statement, and the system dumps with an out-of-memory error.

Other Terms

China, Balance Sheet / Income Statement - China, CN, F1698, BS/IS

Reason and Prerequisites

This is a performance improvement.

The SQL statement uses a lot of memory, causing the system to dump.

Solution

Apply the corresponding support package or implement the correction instructions in this note.
