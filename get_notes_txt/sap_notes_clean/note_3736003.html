Symptom

In CK91N, linking a production version to a procurement alternative for a material/plant is not possible.

Environment

  • Controlling (CO)
  • SAP ERP
  • SAP ERP Central Component
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA

Reproducing the Issue

  1. Start transaction CK91N.
  2. For a material/plant with process category “production” where a procurement alternative was previously created, attempt to create a new procurement alternative using “production version”.
  3. Observe that the choice dialog (BOM/routing vs production version) is not displayed and the production version cannot be linked.

Cause

When the creating the very first procurement alternative for a material and plant combination with process category ‘production’, a dialog box is be displayed in CK91/CK91N to allow system to select whether to use BOM/Routing or production version.  The chosen selection sets the corresponding setting in the controlling level and that controlling level setting is used when creating future procurement alternatives for that material with that process category. However, this dialog box is not shown for the material because the controlling level has already been set.

Therefore system behavior is correct.

Resolution

In order to change the controlling level, please use t-code CKMLMV_CA. 

The controlling level setting for a material can be changed directly via transaction CKMLMV_CA.  To be able to change the setting for that material and process category, there must be no existing procurement alternatives under the relevant process category for that material.
