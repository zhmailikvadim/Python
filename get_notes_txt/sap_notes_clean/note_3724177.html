Symptom

Unable to update Company Details or General Settings in the Web Client when the thousands separator is set to space.

Reproducing the issue

Prerequisites:

In General Settings → Display tab, Thousands Sep. is set to ' ' (space).

Scenario:

  1. In the Web Client, click the user profile button in the top right corner and select Settings.
  2. In the Settings dialog, choose General Settings and update anything in Content Security Policy.
  3. Click Save.

Expected behavior:

The settings save successfully.

Actual behavior:

An error blocks the save action: 'You cannot update the selected fields when other users are connected to the company. To change the setting "Single User Connection", go to the "Services" tab of "General Settings".'.

Solution

SAP intends to provide a feature package or patch to solve the symptom that is described in this SAP Note. The section References below lists the related feature package or patch once they become available. Be aware that References can only be confirmed at feature package or patch release date. SAP delivers feature packages or patches only for selected releases at its own discretion, based on the business impact and the complexity of implementation.

Workaround:

Do not use ' ' for Thousands Sep.
