Symptom

In SAP S/4HANA Cloud Public Edition, after activating Time-Dependent Taxes (TDT) in non-production tenants (such as customizing tenant), in SSCUI 101016 of test tenant/production tenant when trying to maintain tax codes for a certain country/region that supports TDT, the below error is shown:

BCP project not available, so app will be read-only
Message no. /FTI/WEBGUI_MSG127

Diagnosis
Error is raised because system setup is not complete (BCP project not available).

Procedure
Please wait till system is ready for usage.

Environment

SAP S/4HANA Cloud Public Edition

Cause

The Time-Dependent Taxes (TDT) activation/migration is not transported between the systems.

Resolution

It is needed to repeat the TDT activation/migration in other system tenants. Until TDT is in a consistent state, it is also not allowed to maintain any tax codes for that country/region.

Recommended approach: Activate TDT in the test system and perform regression tests (to check whether the critical processes still work as intended). After a successful test, carry out the activation in the production system. Afterwards, new tax codes can be maintained for further usage.
