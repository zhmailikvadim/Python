Symptom

You are using Change Maintenance Order Web Dynpro Application. If you change the technical object and enter a new one, error message EAMS_SP 102 ("Enter a value for "Planner Group for Customer Service and Plant Maint"") occurs, although planner group is already maintained.

Other Terms

W0017, planner group, EAMS_SP, message, error, EAMS_SP102, 102(EAMS_SP), INGPR, PLANGROUP.

Reason and Prerequisites

Side effect of Note 3484523.

Solution

Implement the attached correction instruction or the corresponding Support Package.
