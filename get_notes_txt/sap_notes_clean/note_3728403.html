Symptom

  • Error message: "You are not authorized to query the remote system. Please ask your administrator to grant you the InA role."
  • Unable to create Live Data Connection type tunnel in Datasphere with S/4 system.

Environment

  • SAP Datasphere
  • S/4 HANA On-Premise

Reproducing the Issue

  1. Attempt to create a Live Data Connection type tunnel in Datasphere with S/4 system.
  2. Encounter the error message: "You are not authorized to query the remote system. Please ask your administrator to grant you the InA role."

Cause

Cloud Connector shows the following error: Access denied to resource "https://<tenant_url/sap/bw/ina/GetServerInfo?sap-client=xyz" on system <virtual host>:<port>. In case this was a valid request, ensure to expose the resource correctly in your cloud connector.

Resolution

  1. Enable option "Allow live data to securely leave my network" from Live Data Sources section.
    1. Go to System -> Administration -> Data Source Configuration.
    2. Click on pencil icon at top right corner.
    3. Save the change.
  2. Follow the prerequisite in Support Content - Import Entities from SAP S/4HANA On-Premise via "CSN Exposure" Service (BASIS >= 756)
    1. Make sure that the Cloud Connector Resources has been configured completed as mentioned in section Cloud Connector
