Symptom

The error "An error occurred. Please create an incident on component CA-GTF-CSC-EDO" in the Schedule eDocument Jobs App is preventing the creation of eDocuments from Chile.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Go to the Schedule eDocument Jobs App
  2. Select the "Error" Icon in one of the Runs

The error "An error occurred. Please create an incident on component CA-GTF-CSC-EDO." is being displayed in the UI.

Cause

The Schedule Jobs are failing in this scenario due to the error message ""Posting Date" contains an invalid selection line". This is occuring due to some incompatibility with the Chile eDocuments in the Schedule eDocument Jobs (General) app.

Resolution

In this case, it is recommend to use the Schedule Jobs for eDocument Chile App to automatically trigger specific actions for the eDocument from Chile.

Help Portal: Schedule eDocument Jobs - Chile
