Symptom

This SAP Note introduces new flexible report for Bulgaria to produce the VAT Margin Protocol. The new report /CEECV/BGFI_VAT_MARGIN (transaction /CEECV/BGFI_MARG) is going to replace the former solution /BGLOCS/FI_VATPROTOCOL_MARGIN.

The main improvements of the new report:

  • A new PDF based form has been designed (PDF /CEECV/BGFI_VAT_MARGIN)
  • A new BAdI method in Enhancement Spot /CEECV/BGFI_VAT_MARGIN_ES is provided where the user can adjust the output
  • Data protection and Privacy is integrated (DPP)
  • Customizing is compatible with new generation of VAT Return solution (SAP note 2114571 - VAT Return for Bulgaria) and VAT Protocol (SAP Note 2314169 - VAT Protocol for Bulgaria)

Other Terms

VAT Protocol Margin for Bulgaria, VAT Return, Tax reporting, Bulgaria,  /CEECV/BGFI_VAT_MARGIN

Reason and Prerequisites

SAP notes 2114571 and 2314169 are already completely implemented.

Solution

As a general rule, SAP recommends that you install a solution by applying a Support Package.
