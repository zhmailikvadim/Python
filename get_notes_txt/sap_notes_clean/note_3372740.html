Symptom

When running a "Connection Test" for SM59 destinations type G or H, the following warning message is raised: 

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental. 

Environment

  • ABAP PLATFORM 2022 - Application Server ABAP (SAP_BASIS 7.57)
  • ABAP PLATFORM 2023 - Application Server ABAP (SAP_BASIS 7.58)

Reproducing the Issue

  1. Run SM59 transaction;
  2. Open a destination type G or H; 
  3. Connection Test. 

Cause

As of SAP_BASIS 7.57, a warning message was introduced in RSHTTPPIN program when the HTTP server response is not equal to 200. 

Resolution

The warning message is raised when the HTTP server response is not equal to HTTP 200 OK. 

This is the expected behavior.
