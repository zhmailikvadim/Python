Symptom

Certificates generated by managers for their subordinates' completed courses display null data.

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Environment

SAP SuccessFactors Learning

Reproducing the Issue

  1. Log into Learning Admin.
  2. Search for the manager 
  3. Proxy in as the affected manager.
  4. Click on Team View.
  5. Click ellipsis for subordinate.
  6. Click View History.
  7. Click Print History icon for any completed course.
  8. Certificate displayed with null results.

Cause

The issue is related to a customized SQL query for CertificateOfCompletionLearningHistoryUser, which does not handle preferences correctly.

Resolution

  1. Identify that the issue is caused by the customized SQL query for CertificateOfCompletionLearningHistoryUser.
  2. Fix the SQL query to handle preferences correctly.
  3. Verify that the issue is resolved by testing the certificate generation through both Admin's and User's preferences.
  4. Configure the system as per business needs to ensure proper functionality.
  5. For further assistance on configuration, contact a certified partner or SAP Professional Services team.
