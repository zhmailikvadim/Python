Symptom

You select a structure element in Project Cost Status (Hierarchical) report or in Financial Planning Application. It taken a long time to load or may result in timeout error.

Other Terms

Project Cost Status Hierarchical, PCS, HCS, Workspace, Performance Improvement, Timeout error, /CPD/AVR_MP09_Q0010, RSMD_VIEW_USE_DATA_BINDING, HANA View, Repeated Database Access, Financial Plan, QFP.

Reason and Prerequisites

Reason: Master data validation for multiple values for a single-value variable is executed in a loop triggering repeated database call to HANA calculation views.

Prerequisites: Commercial Projects linked to large execution structure.

Solution

Implement the below SAP Notes along with additional steps mentioned:

  1. 3662210 - Replacement path and SAP HANA view: Bind variables are not used
  2. 3693568 - Lange Laufzeit in der Methode IN_EX_CONVERT der Klasse CL_RSR_SINGLE_VARIABLE
  3. 3645161 - SAP HANA view and bind variable: Problems when reading texts for compound characteristic
  4. 3466376 - Error 1699 when you access an SAP HANA view

Additional Steps:

  • Go to transaction SA38/SE38 and execute the report 'SAP_RSADMIN_MAINTAIN'
  • Pass 'RSMD_VIEW_USE_DATA_BINDING' in Object input parameter and 'X' in Value input parameter
  • Select the Radio button Insert and execute
