Symptom

After completing the activation process for a JOULE service, the error message "Processing Error: The service is currently unavailable. Please reload or contact your administrator" is encountered.

Environment

SAP Business Technology Platform (BTP) Cloud Foundry runtime and environment

Reproducing the Issue

  1. Complete the activation process for JOULE Service.
  2. Attempt to access the JOULE, resulting in the error message "Processing Error: The service is currently unavailable".

Cause

The issue was identified as a network-related problem affecting connectivity and proper functioning of the service.

Resolution

  1. Refer to KBA 2418879 - Information about IP ranges / addresses on SAP Business Technology Platform - to ensure proper network configuration.
  2. Consult the SAP Help Portal documentation for additional guidance on network setup: SAP Help Portal | SAP Online Help.
  3. Confirm that IAS and the service use the same domain as outlined in KBA 3544482 - Getting Error / White / Blank / IAS login Screen When Trying to Access SuccessFactors.
  4. Collaborate with the network team to verify and resolve any misconfigurations or restrictions affecting connectivity.
