Symptom

This SAP Note announces that the Argentina Printing from Last Official Documents (J_1AFONR) report will be replaced by the Argentina Last Official Document Number (AR_ODN_RANGE_REP) report for SAP S/4HANA version 2023 and subsequent releases. You can run the AR_ODN_RANGE_REP report in the Run Statutory Reports app for SAP Document and Reporting Compliance.

The J_1AFONR report will remain available in the system. However, SAP will not provide maintenance to it after July 31st, 2026.

In case of legal changes, corrections, or new features after July 31st, 2026, SAP will not provide any update to the J_1AFONR report. All legal changes, corrections, or new features will only be provided to the Argentina Last Official Document Number (AR_ODN_RANGE_REP) report.

We recommend that, as of July 31st, 2026, you use only the AR_ODN_RANGE_REP report in the Run Statutory Reports app for SAP Document and Reporting Compliance.

Other Terms

Argentina, AR, DRC, fka ACR, fka SAP S/4HANA for Advanced Compliance Reporting, AR_ODN_RANGE_REP, Argentina Last Official Document Number, Impresión de los últimos documentos oficiales, Printing from last official documents, J_1AFONR, Replacement, Run Statutory Reports app, Sunset.

Reason and Prerequisites

This SAP Note is relevant only for Argentina.

You have applied the SAP Notes listed in 3103052 - DRC Argentina Statutory Reporting: Implementation Leading Note

The Argentina Last Official Document Number (AR_ODN_RANGE_REP) report was developed to replace the Printing from last official documents (J_1AFONR) report in the SAP S/4HANA version 2023 and subsequent releases.

Solution

As of July 31st, 2026, from SAP S/4HANA 2023 version onwards, use the Argentina Last Official Document Number (AR_ODN_RANGE_REP) report in the Run Statutory Reports app for SAP Document and Reporting Compliance.

As a general rule, SAP recommends that you install a solution by applying a Support Package. However, if you need to install a solution earlier, use the Note Assistant to implement the correction instruction. You can find more information about the SAP Note Assistant in the SAP Support Portal, under SAP Note Assistant.

Documentation for SAP S/4HANA for Argentina is available in the system and in the SAP Help Portal. System documentation (such as field help, customizing, messages) is delivered via Support Packages. SAP Help Portal documentation is available at help.sap.com/viewer/s4h_op_argentina.
