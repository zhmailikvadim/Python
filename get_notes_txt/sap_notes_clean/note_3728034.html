Symptom

How to create an activity item?

Cause

Resolution

An administrator or user with the Activity and Offboarding Activity - Manage role permission can create activity items by following the steps below:

  1. Go to Admin > Activity Item and click New.
  2. Select the type of activity item that you want to create.
  3. Fill up the details and click Add.

Once created, the activity item can be added to activity checklists if so desired.
