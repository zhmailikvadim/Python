Symptom

You need to delete entries in the eDocument Cockpit (EDOC_COCKPIT) transaction using the EDOC_BR_DELETE_EDOCUMENT report, then you receive the following error message:

EDOCUMENT_BR number 218 (An error occurred, and the document couldn't be deleted. See long text.).

Other Terms

Brazil, Inbound, NF-e, Nota Fiscal Eletronica, Nota Fiscal Eletrônica, Electronic Nota Fiscal, Brazil eInvoice, EDOC_BR_DELETE_EDOCUMENT, EDOC_GUID, SAP Document and Reporting Compliance, Cloud Edition, SAP Document and Reporting Compliance, inbound invoicing option for Brazil

Reason and Prerequisites

You have implemented the SAP Note 3674338 (eDocument Brazil Inbound NF-e/CT-e: EDOC_BR_DELETE_EDOCUMENT new parameter EDOC_GUID)

This SAP Note is relevant only for Brazil.

Solution

After you implement this SAP Note, when you run the EDOC_BR_DELETE_EDOCUMENT report, it is possible to use the EDOC_GUID as a parameter for deletion and remove all entries, even if the entries do not exist in the EDOBRINCOMING or EDOBRCTEINCOMING tables.

Use the SNOTE transaction to implement the attached correction instructions.
