Symptom

You are using GRC with Fiori menus. Upon customizing the Risk POWL in order to add a new category to it, the list of active queries doesn't show it.

Environment

SAP GRC Risk Management 10.1

SAP GRC Risk Management 12.0

Reproducing the Issue

1. Open Risk and Opportunity POWL(this is also reproducible in other POWLs)

2. Personalize button at the top of the POWL

3. Add category and provide description

4. Check that list of active queries is not displaying the newly created category.

Resolution

Apply the following POWL component note to fix it:

2484413 - POWL: Unable to add new category

Keywords

POWL, add, category, new, Risk, query.
