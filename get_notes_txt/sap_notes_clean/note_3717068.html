Symptom

  • Default locale translation are missing for Manage Banner Cards
  • The Manage Banner Cards is in English for all default locales

Environment

SAP SuccessFactors HCM Suite

Reproducing the Issue

  1. Access the instance and set the display language to German for example.
  2. Navigate to Startseite Verwalten (Manage Home Page) > Bannerkarten (Banner Cards) > Manage Banner Cards and observe the translation inconsistencies.

Resolution

  • This is a known issue. The Engineering team is currently working on a fix.

  • The issue is planned to be resolved in the upcoming 1H 2026 release with the following timeline:

    • Preview: April 14, 2026

    • Production: May 16, 2026
