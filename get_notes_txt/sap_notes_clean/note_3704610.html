Symptom

You download the DMR report in Fiori and expect the system to display the Source Field Description. However, the report shows the Source Field Value instead.

This note corrects the behavior so that the DMR report in Fiori displays the source field description when downloaded.

Other Terms

/VIZI/LOPERF0P, /VIZI/OPER, Excel, Exported

Reason and Prerequisites

This is caused by program error

Solution

Please implement the correction instructions attached. These corrections are only valid if you have applied the following support pack(s).

Release V4 -  2022 V2, 2023, and 2023 V2

Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object.
