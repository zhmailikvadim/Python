Symptom

In SAP Predictive Analysis, if a table or a view contains at least one column of type SECONDDATE, executing a model with an algorithm in the Predict room yields the following error message:

"Cause: Error details: SAP DBTech JDBC: [257]: sql syntax error: incorrect syntax near "(": line 1 col 92 (at pos 92)"

Other Terms

Environment:

  • Predictive Analysis 1.21
  • Windows 7 32/64 bit
  • HANA DB

Steps to Reproduce:

1. Open Predictive Analysis 1.21.
2. Create a new document that connects to HANA online. Enter the connection details and Connect.
3. Select objects (including one with SECONDDATE data type) and click Create.
4. In Predict room, add Preprocessors > Filter (for example).
5. Configure the settings are required for the selected algorithm.
6. Execute the model.
7. Error is returned: Filter: An error occurred while executing the query. Details. "Cause: Error details: SAP DBTech JDBC: [257]: sql syntax error: incorrect syntax near "(": line 1 col 92 (at pos 92)"

 

Reason and Prerequisites

This issue is a defect and bug reference BITDUB1-931 has been created.

Solution

A fix for this issue should be available in the next release. This is tentative and subject to change without prior notification.
