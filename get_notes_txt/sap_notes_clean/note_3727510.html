Symptom

The following error is reported in ACT Check:

Check title: Zero Downtime Options (ZDO): Identify compliance violations

Check Message: Table referenced in the maintenance view cannot be set read-only

Finding Details: Table referenced in the maintenance view cannot be set read-only
R3TR TOBJ <Z/Y object>: Table <Z/Y object> referenced by Maint. View <Z/Y object> cannot be read-only (145)

Environment

SAP ABAP System

Reproducing the Issue

Run ACT check for ZDO.

Cause

Customer objects(Z/Y objects) is not supported in ZDO by default.

Resolution

The customer objects should be excluded in ZDO scenario.

If the ZDO scenario is indeed required for the customer objects, please refer to the section "Registration Process for Customer-Specific Projects" in the note 2707731 and register the service.
