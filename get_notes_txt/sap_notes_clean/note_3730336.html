Symptom

Performance issue when processing BUBA camt.087 message via Request Agent UI.

Other Terms

/PE1/CL_RA_TYPE_MOD_REQ_PASS->GET_ITEM, BUBA camt.029, BUBA camt.087.

Reason and Prerequisites

Program error. Even if item key is initial in method GET_ITEM, it tries to read records from /pe1/db_item table according to package size. This leads to performance issue as no matching entry can be found if item key is initial.

Solution

Implement the correction instruction attached to this SAP Note. Alternatively, you can import the referenced Support Package.
