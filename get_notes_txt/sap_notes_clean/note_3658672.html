Symptom

Installation of SAP Business One Cloud System Landscape Directory (SLD) fails when the curl version on Linux Server is 8.12.1 or higher.

Reproducing the issue

Prerequisites:

  • You are using SAP Business One Cloud.
  • You have manually installed curl version 8.12.1 or higher on the Linux server.

Scenario:

Install the SAP Business One Cloud System Landscape Directory (SLD) using the Server Component Setup Wizard.

Expected behavior:

The installation is successful.

Actual behavior:

The installation fails. The installation log contains:

# curl --insecure -noproxy '*' --output /dev/null https://localhost:40000
curl: (3) URL rejected: Bad hostname
curl: (26) .netrc error: no such file
