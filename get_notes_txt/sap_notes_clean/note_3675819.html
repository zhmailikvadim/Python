Symptom

WDA Analytic Grid: While downloading the data/report from "Group Data Analysis" app, in spreadsheet the Parameters are missing.

This also affects performance of the background export. Especially when there are large amounts of header cells with lots of merging, this is really taking a long time to get exporting.

Environment

FPM Analytical GUIBB
SAP S/4HANA

SAP_BW 758

Resolution

Apply SAP note 0003462412 and SAP note 0003427386 to solve this issue and have the header info in your exported spreadsheet file

Keywords

BICS Grid, asynchronous export, excel, spreadsheet, download, Export header, export, FPM Analytical GUIBB, Background Export, header cells
