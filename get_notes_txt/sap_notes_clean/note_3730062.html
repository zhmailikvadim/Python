Symptom

  • Regression issue that the script execution onInitialization for specific Story which does not work anymore.
  • It worked previously , but after 9th March, some stories are getting error message : either in Edit mode or in View mode :
    TypeError: (this.getUnifiedStore(...).getState(...) || []).reduce is not a function.
  • In the View mode the page will load the error: Script execution failed for event "Page0.onInitialization" 
  • In Edit mode, when opening the Page1, switch on Page 2 and back to Page1 we get the same error
    TypeError: (this.getUnifiedStore(...).getState(...) || []).reduce is not a function.

Environment

  • SAP Analytics Cloud 2026.5.1 

Resolution

  • A fix will be delivered in the next SAC version 2026.6.x
