Symptom

Fiori App: Manage Purchase Contracts application (F1600A)

Tax code search help returns tax codes of all the procedures instead of the procedure used by the Contract.

Environment

  • Materials Management (MM)
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Access Fiori app Manage Purchase Contracts
  2. Create a new Contract or update an existing Contact
  3. Navigate to item details screen
  4. Go to "GR / IR Control" factsheet and click on the search help of Tax Code

Cause

Standard design.

Manage Purchase Contracts app uses value help 'I_Purcontracttaxcodevh' for Tax Code. 

The necessary fields and associations required for dynamically passing the Tax Procedure are not accessible to this value help. Due to this technical limitation, currently it is not possible to search for Tax Codes based on Tax Procedure dynamically in the app.

Resolution

As a workaround, fill in Tax Procedure manually in the value help of Tax Code to search for the required codes in app Manage Purchase Contracts.
