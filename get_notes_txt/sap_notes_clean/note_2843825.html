Symptom

This KBA describes how to produce an Archive or Export of a project that can help to troubleshoot problems with the operation of a SAP Intelligent Robotic Process Automation (IRPA) bot or a SAP Build Process Automation (SBPA) project.

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental. 

Environment

  • SAP Intelligent Robotic Process Automation.
  • SAP Build Process Automation.

Resolution

Archive a Desktop Project

This can be done through the IRPA Studio. It is possible to create a backup of the Project by using the 'File\Archive Project…' menu item in the Menu bar. This generates a new output archive file into the archive output folder.

The archive file name has the following format:

      archive_<project name>_<project version>_<date>_<time>.zip

This output file contains all files and folders of the Project Structure, except 'archive' and 'export' folders. You can find the Archive folder in the same location that your project is stored. A new folder called "Archive" will be created with the .zip inside.

Cloud Project Exports (IRPA (A) or SBPA (B))

A.) Export a Cloud Studio project via the SAP IRPA Factory

This can be done through the Cloud Studio. It is possible to create a backup of the Project by using the 'Export the project' button, available on the home screen of your project. This generates a new output archive file which is downloaded into your download folder.

  1. Open the project you wish to Export and open the Overview tab
  2. Click on the Export button as shown below:
  3. Choose the relevant format for your project (if in doubt choose SAP Intelligent RPA) and click Export:


  4. The Project will now be exported into a compressed ZIP file. This might take a few moments, and longer for more complicated projects. Your browser will start the download of this zip file automatically (depending on browser settings) once the export has successfully completed

The export file name has the following format:

      <project name>.zip

This output file contains all files and folders of the Project Structure, and can be imported to a tenant using the Import button available on the Projects tab.

B.) Export a Cloud Studio project via SAP Build Process Automation

  1. Access the SAP Process Automation Lobby page

  2. Locate your project you wish to export under All Projects and click the ellipsis button (...) beside the project you wish to export. Click on the Export button:



  3. A project will be Exported in the format <project_name>_<version>.mtar. Using a compression program e.g: WinRAR, 7-Zip, GZip etc. compress this mtar and attach it to the case
