Symptom

This SAP Note provides guidelines for implementing the GSTR1A report for India using SAP Document and Reporting Compliance, cloud edition. It helps every registered business file GSTR1A report which includes details of all outward supplies.

This SAP Note contains its prerequisites and a list of SAP Notes that you must implement.

Other Terms

Implementation Overview Note, Document Compliance, DC, Document and Reporting Compliance, electronic document processing, India GST, cloud edition, GST Reporting, GST, Goods and Service Tax, GSTR1A, GSTR-1A, gstr1a, GSTR-1a.

Reason and Prerequisites

 Reason:

  • You want to implement, configure, and use the GSTR1A report for India

Prerequisites:

  • You must subscribe to SAP Document and Reporting Compliance, cloud edition to file the return. Please get in touch with your SAP representative for further details. 
  • You have implemented the electronic document processing framework in your system. Refer to the implementation instructions provided in the Help Portal for  SAP S/4HANA 
  • You have implemented GSTR1 Note

List of relevant SAP Notes and Support Package Information

Apply the SAP Notes in ascending order. Notice that:

  • Some of the SAP Notes have pre and post implementation steps. You need to perform them in your system.
  • Depending on your Support Package, you might not need to implement certain SAP Notes.
    • Column "Delivered SP": If your system edition is equal or higher than delivered Support Package, you can skip the SAP Note.
    • Column "Supported SP": If your system edition is mentioned in column Supported SP, you need to implement the SAP Note.
  • Quickly access SAP Note via composited link: https://me.sap.com/notes/<SAP Note number>.
