Symptom

In Configuration Activity Prepare Automatic Clearing (101971) it is not possible to add clearing criteria which exceed character limits, such as SGTXT (Line item text).

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Environment

SAP S/4HANA Cloud Public Edition 2508

Reproducing the Issue

  1. Go to App Manage Your Solution
  2. Select Configuration Activity Prepare Automatic Clearing (101971)
  3. Select a Criterion Field
  4. Click the dropdown menu

5. Field list does not contain SGTXT - Line Item Text

Cause

The field cannot be added due to length limitation. The mass clearing program has 24-character slots for the clearing criteria and it truncates the content of the chosen attributes to that length.

Resolution

Only use the criteria that can be selected from the fields listed in the dropdown menu.
