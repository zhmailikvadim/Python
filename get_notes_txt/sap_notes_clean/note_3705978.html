Symptom

The context is during Advanced-CPE adoption, running data conversion for Advanced-CPE in data conversion workbench CPE_CONV_WB, an error "Exception in PFW parallelization, check SHDB_PFW_SUPPORT for more detail" is encountered in the application log for price fixation set migration.

Typically this error occurs when there is a shortdump in the parallel task. However in certain situation, there is no shortdump.

Other Terms

Advanced-CPE, data conversion, ACM, Price fixation set data conversion.

Reason and Prerequisites

missing error handling for corrupt data.

Solution

Implement the attached correction instructions to solve the issue.
