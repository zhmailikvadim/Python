Symptom

OP 2025 FPS0 and FPS1: in Fiori app "Manage Mass Maintenance - eSPP Data" (F7729), on the Data Type edit page's HotTable, given that you firstly double click a cell that has a F4 Value Help icon, later you click its F4 icon to open its Value Help Dialog, when you choose a new different value in the Value Help Dialog, then the cell display text is not updated until you double click the cell again.

Other Terms

ESPP_MASS_MNTS1 scm.espp.massmaint F7729

Reason and Prerequisites

Programming error in Software Component UIS4H 109 FPS0, FPS1.

Solution

We recommend to upgrade to UIS4H 109 FPS2.

Or upgrade to UIS4H 109 FPS1 then apply the attached corrections. Before testing the App, to get the latest UI changes, follow the steps of the note 2227577 to recalculate the Index.
