Symptom

  • The "Change Document" field is not visible under the "Related Documents" section in SAP Solution Manager.
  • The issue occurs when attempting to link ZMMJ types to the system.

Environment

SAP Solution Manager 7.2

Reproducing the Issue

  1. Access transaction SOLDOC
  2. Attempt to locate the "Change Document" field under the "Related Documents" section.

Cause

Missing configuration.

Resolution

  1. Run transaction code: solman_setup.
  2. Navigate to Process Management.
  3. Select "Configure Solution Documentation Model".
  4. Click "Edit".
  5. Define Related Documents Visible at the Attribute Pane.
  6. Activate the Enhancement Implementation: AGS_CRM_API_LCO_CD.
