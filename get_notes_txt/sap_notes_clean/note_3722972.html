Symptom

  • Recurring instance is auto deleted when it is expired 

Reason and Prerequisites

Environment:

  • SAP BusinessObjects Business Intelligence Platform 4.3

Steps to Reproduce:

  1. Right click on any report and choose schedule.
  2. Click on Recurrence
  3. Choose from Now to Hourly.
  4. Set Hour to 0 and Minute to 3
  5. Set the End time to be Start time + 15mins. Eg: the start time is 9:05, please set End time to be 9:20
  6. Check the history again after 9:20
  7. There is no expired instance

Cause:

  • The cause is currently investigated by Development team

Solution
