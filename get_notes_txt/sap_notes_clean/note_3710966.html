Symptom

Error in SAP Analytic Cloud Story build with a BW Live model when using the search engine for filtering on the dimension.

  • Filtering works when selecting an element from the displayed list but fails when using the search engine with any ID.
  • Error messages observed:

    "Unable to initialize query"
    "Query can't be executed without setting values for all mandatory variables"

  • It also generates an ABAP Dump on transaction ST22:
Category               ABAP programming error                                                       
Runtime Errors         CONVT_NO_NUMBER                                                              
ABAP Program           CL_RSMDS_DIMENSION============CP                                             
Application Component  BC-BW-SRV

Short Text: "*" cannot be interpreted as a number

What happened?: Error in the ABAP application program.
The current ABAP program "CL_RSMDS_DIMENSION============CP" had to be terminated because it found a statement that could not be executed.

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Environment

  • SAP Analytics Cloud
  • SAP BW4/HANA
  • SAP BW4/HANA Cloud
  • SAP NetWeaver

Reproducing the Issue

  1. Open a Data Analyzer Insight or a Story which contains a BW Live Model.
  2. Create a filter with the affected dimension.
  3. Use the search engine and filter for any ID.
  4. Observe the error message and failed filter application.

Cause

Missing correction.

Resolution

Implement the following SAP Note:

3657316 - Wrong or missing data in query with active hierarchy
