Symptom

RFC Server Interface log shows error message below when trying to start a SNC enabled RFC Connection. 

[ SEVERE ] An error occurred while connecting to the server. :Initialization of destination <destination URL> failed: Name or password is incorrect (repeat logon)
[ FINE ] Stopped
[ FINE ] Ready

Environment

  • SAP Data Services 4.3
  • SAP Data Services 2025
  • BW

Cause

Only a single SNC name can be assigned to a user account in the SAP system.

Resolution

A single SNC must not be used for more than one user. Make sure the user in the RFC connection to Data Services is the only one using the SNC name.
