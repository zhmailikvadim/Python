Symptom

For SAP Business One Cloud, the selection of a company database from a list in the logon window is not avaliable.

Reproducing the issue

Scenario

  1. In SAP Business One Cloud 1.1 SP4, assign multiple tenants to a service.
  2. Configure the integration framework for SAP Business One Cloud.
  3. Login to the first tenant using the mobile application 1.11.1 for iPad or iPhone.
  4. Logout from the tenant. 

Expected Result 
You can select the Choose Company option in the mobile application.

Actual Result 
You cannot use the Choose Company option in the mobile application, because it does not exist.

Cause

Software limitation for SAP Business One Cloud

Solution

Implementation of this function is not feasible in the currently supported versions of SAP Business One. The mobile solution can only connect to the SAP Business One database, if the company database is defined in the SAP Business One mobile application settings.
