Symptom

When trying to export the Matrix Grid report the report is not downloaded and no action is triggered by the export button.

Environment

SAP SuccessFactors Succession Management 

Reproducing the Issue

  1. Navigate to Succession.
  2. Select Matrix grid reports tab.
  3. Apply filters that generate results in the grid.
  4. Click on the export button.
  5. Observe that no action occurs.

Resolution

The fix for this issue is planned to be deployed on February 10th for Preview and February 12th for Production instances.
