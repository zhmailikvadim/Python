Symptom

Missing product information observed after performing an update through SUM with stack xml file.

Environment

Upgrade tools for ABAP

Cause

The missing product information is obsolete before running SUM.

Resolution

The behavior is normal. MP will calculate the correct information for the system when the system is planned for the stack xml file, after upgrade, the obsolete information will be removed.
