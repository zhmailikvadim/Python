Symptom

BBP_PD_VERSIONS_ARE_SUPPORTED is enhanced. And it is intended to centralize and unify the determination of BO Versions processing support.

E.G. It is not needed to perform expensive BBP_PDHGP DB selection to determine the VERSION_TYPE and ACTIVE_HEADER on BOs not supporting versions.

This note is a side-effect note of SAP note 3259846.

Other Terms

BBP_HDRPROC_GET_CHANGE_VERSION, /SAPSRM/IF_CLL_BO_MAPPER~CHANGE_TRANSACTION_MODE, BBP_PROCDOC_LOCK, BBP_PROCDOC_UNLOCK, BBP_PROCDOC_GETDETAIL_DIRECT, SRM_701_OPTIMISTIC_LOCK, Optimistic Lock.

Reason and Prerequisites

Due to program error

Pre requisite Note : 3259846

Solution

Implement the correction instructions from this SAP Note or upgrade to the corresponding Support Package.
