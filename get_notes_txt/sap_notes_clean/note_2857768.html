Symptom

Improvement request

Provision of hour based input with traditional minutes as alternative to industry minutes for part time absences.

Request reason

Usually hour based input are offered by using decimal format for entering minutes. E.g. 4h 30m will be entered as 4,5. This approach is known under the term "industry minutes". As an user friendly alternative the leave request should provide the option to enter the typical hours & minute format using 60 units for representing a full hour.

Other Terms

Customer Connection, HCM CC 2017, idea 45334, Leave Request V3, BSP HCMFAB_LEAV_MAN, hcm.fab.myleaverequest.

Reason and Prerequisites

This SAP Note relates to an improvement request that was submitted by a customer.

Solution

Improvement

You now have the option of using traditional minutes for input hours.

Please read the attached document for more description, including screenshots.

Benefit

Traditional minutes will be supported as alternative to industry minutes.
