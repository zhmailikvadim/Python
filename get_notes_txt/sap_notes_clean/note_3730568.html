Symptom

The Badi EDOC_MX_FILL_PAYMENT_IMPUESTOP restrict changes for the Parameter IT_TRANSLADO_DR.

Environment

SAP S/4HANA Cloud Public Edition

Cause

The BAdI EDOC_MX_FILL_PAYMENT_IMPUESTOP (Mexico: Modifying Tax Data (Impuesto_P) for Payments) has the responsibility and objective to solely modify the Impuesto_P tag data. The parameter IT_TRANSLADO_DR is used as an input to help the implementation with the translado data, so the changing of translado DR is not possible through this BAdI.

Resolution

In order to modify the translado DR, it is recommended to use the BAdI EDOC_MX_FILL_PMNT_TRASLADOS_DR, method FILL_TRASLADOS_DR.

Keywords

FILL_TRASLADOS_DR, EDOC_MX_FILL_PMNT_TRASLADOS_DR, EDOC_MX_FILL_PAYMENT_IMPUESTOP, IT_TRANSLADO_DR, Badi
