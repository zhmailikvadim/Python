Symptom

You use the Group Data Analysis app (App ID: W0135) and in your report you would like to see all the FS Items, even if there is no posted data on them.

Environment

Embedded Analytics for S/4HANA Finance for Group Reporting
Release SAP S/4HANA 2022 or higher

Resolution

In the Group Data Analysis app (App ID: W0135), in your report, if you would like to see all the FS Items, even if there is no posted data on them, then first, please click to the FS Item column, then use the "Display" --> "Master Data" option. 
To revert this setting, first, please click to the FS Item column, then use the "Display" --> "Posted Values" option. 

Note: This functionality is available starting from release SAP S/4HANA 2022.
