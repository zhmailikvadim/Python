Symptom

How do I update the status of Supplier Qualification Process Projects that are stuck in InProgress status which are pointing to a deactivated Process Type?

Resolution

To update the Process Types:

  1. Click Manage > Administration > Data Import/Export
  2. Click Export tab > Export next to Export Process Types
  3. In the SMProcessFlowTypeExport.csv , enter a row for the inactive Process Type with exactly the same UniqueName that was used earlier.
  4. If required update the Name column value with a slightly different name if there is already another active Process Type with exactly the same name in order to differentiate them and Save the file.
  5. Click Import tab > Load option > Select the updated SMProcessFlowTypeExport.csv file and Click Run

To update the Process Statuses:

  1. Click Manage > Administration > Data Import/Export
  2. Click Export tab > Export next to Export Process Statuses
  3. In the SMResourceStatusExport.csv , add entries for all statuses associated with the reactivated process type with exactly the same UniqueName that was used earlier for the ProcessType.UniqueName column
  4. Save the file.
  5. Click Import tab > Load option > Select the updated SMResourceStatusExport.csv file and Click Run

To update the impacted Qualification Process Project status: 

  1. Access Supplier 360 profile
  2. Click Processes > Advanced View > Overview tab
  3. Click Actions > Sync Status
  4. Click Actions > Sync Task Data
