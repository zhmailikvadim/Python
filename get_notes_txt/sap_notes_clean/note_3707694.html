Symptom

In some scenarios not all the activities are captured in the Firefighter logs. This is especially true for the last activities within the session.

Other Terms

GRC, 12.0, EAM, SPM, firefighter, FF, missing logs

Reason and Prerequisites

Issue may happen if the Firefighter session gets closed the exact moment when the FF log collection retrieves the logs for the same session.

Solution

Please implement the attached correction instructions or the equivalent SP.
