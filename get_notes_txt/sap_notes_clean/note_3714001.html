Symptom

The user maintained partner conditions for the price sheet, and after maintaining the rule data, the information is updated correctly in the Partners tab. However, when a customer is deleted and a new customer is added at the rule level, the updated data is not reflected in the Partners tab accordingly.

Other Terms

/IRM/IPARM, rule_delete, Primary Partner, Partner conditions, Partner Update from Rules

Reason and Prerequisites

This is caused by Program error.

Solution

Please implement the correction instructions. These corrections are only valid if you have applied the following Support Packs.

Release V/4 – 2022 V2, 2023, 2023 V2, 2023 V3 and 2023 V3 Support Package 1

Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object
