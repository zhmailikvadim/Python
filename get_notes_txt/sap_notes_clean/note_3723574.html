Symptom

  • Candidates in Onboarding 1.0 are unable to proceed past address panel due to USPS verification issues. 
  • The error message "Web Tools has been retired effective January 25, 2026. For migration support go to https://emailus.usps.com/s/usps-apis select onboarding radio button and API issue of API onboarding/upgrade or attend daily webinars from 12pm-2pm ET through March 26, 2026. Refer to the latest Industry Alert at https://postalpro.usps.com/node/14807. To request an extension please go to https://forms.office.com/g/2r7nu4x6lq" appears.

Environment

  • SAP SuccessFactors Onboarding 1.O

Reproducing the Issue

  1. Go to Onboarding 1.0.
  2. Filter for users in the New Employee Step.
  3. Move forward from the SSN step to the Physical Address Step.
  4. Enter the physical address and click next, and observe the error message.

Cause

Account option "XpressHR.AddressCheckingIsRequired" is set to true for this specific panel, which causes this error to appear on the physical address panel. 

Resolution

1. Navigate to SAP SuccessFactors onboarding Superadmin Account.
2. Account options.
3. Set the key "XpressHR.AddressCheckingIsRequired" is to "False".
