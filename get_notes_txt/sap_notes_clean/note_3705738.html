Symptom

  • An application error occurs when an internal candidate logs into their profile, navigates to the careers page, and attempts to access their profile by clicking on "My Candidate Profile."
  • Error message: errorId=null, timestamp=null, errorMessage=Cannot read properties of null (reading 'candidateId'), errorName=TypeError, server=9sc57bcf07, versionInfo=Release%3A%202411.20241123150754%0A%20%20%20%20Server%3A%209sc57bcf07%0A%20%20%20%20Timestamp%3A%202024-12-03T02%3A44%3A29.448%2B0000

Environment

  • SAP SuccessFactors Recruiting Management 

Reproducing the Issue

  1. Log into the SAP SuccessFactors instance.
  2. Proxy as the user with the issue.
  3. Navigate to the "Careers" section.
  4. Click on "My Candidate Profile."
  5. Observe the error message displayed.

Cause

  • The user was incorrectly mapped to a candidate record, which prevented the system from creating a new candidate account for the user.
  • This appears to have been caused by a previously resolved issue that resulted in incorrect data mapping.

Resolution

  1. Engage the operations team to execute necessary scripts for data correction.
  2. Verify the mapping of the user to ensure it aligns correctly with the candidate record.
  3. Confirm that the issue is resolved by testing the functionality after the scripts have been executed.
