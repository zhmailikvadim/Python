Symptom

When performing an account closure with Async payment advice mode, you want the ability to use different transaction types to post the remaining balance to the interim account.

Other Terms

Account Closure, Cash Settlement, Async PAM, Suspense Account, Process Result Category, Payment Directive Type, Transaction Type, BCA_OR_TOC.

Reason and Prerequisites

Special development.

Solution

This note provides an enhancement spot (ES_FSTB_OR_TOC_PROC_RSLT) to overwrite the standard process result category.

Implement the correction instructions provided with this SAP Note.
