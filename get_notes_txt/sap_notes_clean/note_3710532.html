Symptom

When an application document is applied using propose or manual selection of contract functionality, the pop-up that appears has blank value in counterparty reference column so this field is mapped to extenral identifier from contract header.

Other Terms

Agricultural Contract Management; ACM; Application workcenter; Manual selection of contract; Propose contract; Counterparty reference

Reason and Prerequisites

Curently counterparty refrence is not mapped to any field within the contract.

Solution

Counterparty reference field is mapped to external identifier field from contract header. Since the field lengths are different, only initial 20 characters will be considered.

Implement the Correction Instruction to fix the issue.
