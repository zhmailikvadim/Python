Symptom

There is a requirement to know where the SAP JVM used by the NetWeaver Application Server Java is installed.

Environment

SAP NetWeaver Application Server for Java

Resolution

The SAP JVM is located in the following paths:

  • /usr/sap/<System ID>/<Instance ID>/exe/sapjvm_8/
  • /usr/sap/<System ID>/SYS/exe/jvm/<Architecture>/sapjvm_8.1.XXX/sapjvm_8/
