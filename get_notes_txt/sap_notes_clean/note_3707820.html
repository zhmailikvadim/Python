Symptom

  • To repost the CO-PA document generated through SAP SD billing, T-code KE4S can be utilized.
  • Inquiry regarding availability of a transaction code for reversing and reposting CO-PA documents generated through agency billing documents created via the TMS module using T-codes WZR3/WLF3.

Environment

  • Contolling(CO) 
  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA Finance
  • SAP S/4HANA

Reproducing the Issue

Documents created in transaction codes WZR3/WLF3 need to be reposted in profitability analysis (component CO-PA).

Cause

Documents created in transaction codes  WZR3/WLF3 need to be reposted in profitability analysis due to errors in those documents

Resolution

  1. Execute the report RWLF2063, which functions similarly to KE4S.
