Symptom

During query execution, the system issues the error message "Specify a valid interval for variable ...".

Other Terms

 Interval

Reason and Prerequisites

Program error

Solution

  • SAP BW 7.50
    Import Support Package 35 for SAP BW 7.50 (SAPK-75035INSAPBW) into your BW system. The support package is available once SAP Note 3649452 ("SAPBWNews NW BW 7.50 ABAP SP35"), which describes this SP in more detail, has been released for customers.
  • SAP BW 7.56
    Import Support Package 10 for SAP BW 7.56 (SAPK-75610INSAPBW) into your BW system. The support package is available once SAP Note 3672673 ("SAPBWNews NW BW 7.56 ABAP SP10"), which describes this SP in more detail, has been released for customers.
  • SAP BW 7.57
    Import Support Package 08 for SAP BW 7.57 (SAPK-75708INSAPBW) into your BW system or AS ABAP system. The support package is available once SAP Note 3672675 ("SAPBWNews NW BW 7.57 ABAP SP08"), which describes this SP in more detail, has been released for customers.

  • SAP BW 7.58
    Import Support Package 06 for SAP BW 7.58 (SAPK-75806INSAPBW) into your BW or AS ABAP system. The support package is available once SAP Note 3538326 ("SAPBWNews NW BW 7.58 ABAP SP06"), which describes this SP in more detail, has been released for customers.
  • SAP BW 8.16
    Import Support Package 02 for SAP BW 8.16 (SAPK-81602INSAPBW) into your BW system or AS ABAP system. The support package is available once SAP Note 3692519 ("SAPBWNews NW BW 8.16 ABAP SP02"), which describes this SP in more detail, has been released for customers.
  • SAP BW/4HANA 2021
    Import Support Package 14 for SAP BW/4HANA 2021 (SAPK-30014INDW4CORE) into your BW system. The support package is available once SAP Note 3625395 ("SAPBWNews SAP BW/4HANA 2021 SP14"), which describes this SP in more detail, has been released for customers.

  • SAP BW/4HANA 2023
    Import Support Package 08 for SAP BW/4HANA 2023 (SAPK-40008INDW4CORE) into your BW system. The support package is available once SAP Note 3639252 ("SAPBWNews SAP BW/4HANA 2023 SP08"), which describes this SP in more detail, has been released for customers.

In urgent cases, you can implement the correction instruction as an advance correction.

You must first read SAP Notes 1668882 and 2248091, which provide information about transaction SNOTE.

To provide information in advance, the SAP Note(s) specified above might already be available before the support package is released. In this case, however, the short text of the SAP Note still contains the term "preliminary version".
