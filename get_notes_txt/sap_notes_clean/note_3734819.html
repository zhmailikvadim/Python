Symptom

In the process "Receipt of Termination of Supplier Assignment for Electricity (SUP View)" (DE_B_GPKE_TERM_OF_ASSGN_SUP), there is no customer extension step available before the supply scenario update.

This issue is relevant for the market role: SUP.

Other Terms

GPKE_SUP, DE_B_GPKE_TERM_OF_ASSGN_SUP, customer extension, step 450, supply scenario update, termination of supplier assignment

Reason and Prerequisites

The system did not provide a customer extension step before the supply scenario update, limiting customization options at this point in the process.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the system provides a customer extension step before the supply scenario update, allowing you to add a custom checkgroup. Import the BC set to activate this enhancement. If you need the extension step, add your custom logic to the checkgroup. You can verify the fix by retesting the process from the back end and confirming that your custom extension step runs successfully before the supply scenario update.
