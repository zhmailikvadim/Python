Symptom

  • This solution ensures a seamless update of the database password in the connection string, preventing workspace and repository connection issues. 
  • In adherence to company protocols, the database server password undergoes a reset every three months, necessitating the update of the password. This periodic change in the database server password leads to a disruption in the Checkinall process of newly created TestObjects in the workspace. Consequently, users encounter difficulties saving these TestObjects to the Tosca repository.



Environment

SAP Enterprise Continuous Testing by Tricentis

Resolution

Check the following knowledge article from Tricentis:

Resolving TestObjects Checkinall Issue After Database Password Reset of Workspace Connecting String

Keywords

TOSCA, workspace, testobjects, new, .tws
