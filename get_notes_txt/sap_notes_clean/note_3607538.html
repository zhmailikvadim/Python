Symptom

This article explains the option "Select from Goal Plan" when creating goals on form.

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental. 

Environment

  • SAP SuccessFactors Performance Management
  • SAP SuccessFactors Goal Management
  • SAP SuccessFactors Career Development Planning 

Reproducing the Issue

  1. Open form and create goal on form.
  2. On the goal creation page, notice that for some goal section, it navigates to "Goal Creation Options" page and display an option "Select from Goal Plan".
  3. Notice that for some other goal sections, it directly opens the goal creation window for user to fill goal fields.


Resolution

The "Select from Goal Plan" option appears when 

  • "Show Add Existing Objective Button" option is enabled for performance goal section for form template manage templates;
  • "Show Add Existing Development Objective Button" option is enabled for development goal section for form template manage templates.

The button is used to add existing goals in the linked goal plan to the form.

If you do not want to display the option "Select from Goal Plan", do not enable "Show Add Existing Objective Button" option for performance goal section and/or "Show Add Existing Development Objective Button" for development goal section respectively.
