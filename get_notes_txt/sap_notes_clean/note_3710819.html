Symptom

Alumni users can now authenticate to SAP SuccessFactors using Identity Authentication Service (IAS) as the identity provider.

Environment

  • SAP SucccessFactors HCM suite
  • Identity Authentication Services

Resolution

As of the 1H 2025 release, Alumni users can authenticate into SAP SuccessFactors using IAS to securely access company materials after leaving the company. Refer to the SAP Help Guide for detailed guidance on this functionality:

  • Enabling Alumni Access
  • Alumni Can Now Authenticate into SAP SuccessFactors Using Identity Authentication
