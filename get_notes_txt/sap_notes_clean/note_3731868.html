Symptom

  • After upgrading Tosca from version 2023.1.3 to 2025.1.3, users have encountered an error when printing execution reports stating that the List & Label version is no longer supported. This issue occurs because existing execution report definitions are incompatible with the newer List & Label version used in Tosca Commander. This typically occurs from version 2024.1 to 2025.1.
    The issue can be resolved by updating the report definitions either by importing the appropriate standard subset and upgrading the ExecutionEntries to the latest List & Label version, or by replacing the old ExecutionEntries with updated ones from the standard subset. Once updated, execution reports can be printed successfully without errors.

  • After upgrading Tosca from version 2023.1.3 to 2025.1.3, users have encountered the following error when attempting to print execution reports using Tosca Commander 2024.1 – 2025.1:

    Error: "This report uses a List & Label version that's no longer supported. Please upgrade your report to version 30, then try again."

    Error: "This report uses a List & Label version that's no longer supported. Please upgrade your report to version 29, then try again."



Environment

SAP Enterprise Continuous Testing by Tricentis

Cause

This occurs because the execution report definitions are using an outdated List & Label version that is incompatible with the new Tosca version.

Resolution

Check the following knowledge article from Tricentis:

Error Printing Execution Reports After Upgrading Tosca - List & Label version no longer supported
