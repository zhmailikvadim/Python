Symptom

  • Not able to create and save a new layout in the new version of the material documents overview app (new mb51).

Environment

Product: SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Open the Material Documents Overview app (new version of MB51).
  2. Attempt to create a new layout/view and save it.
  3. Observe that the system does not allow creating or saving the layout.

Cause

The new version of the app uses a different workflow for creating and saving views/layouts; no defect was found and the system works as designed.

Resolution

  1. Open the Material Documents Overview app (new MB51).
  2. Switch to the new version if prompted.
  3. Open the My Views menu (down arrow next to STANDARD).
  4. Select the view you want to change or use as a base (for example, the standard view).
  5. Set required filters (for example, plant, storage location) and select Go to run the report.
  6. Open Settings (gear icon) and select or deselect columns/parameters as needed.
  7. Choose OK to apply settings.
  8. Open the My Views menu again and select Save (appears only if changes are made to the columns/parameters) to update the current view or Save As to create a new view/layout.
