Symptom

When attempting to use the "Propose Items" function in the SAP Fiori app "Manage Sales Orders - Version 2", the "Propose Items" button is either not visible or is disabled (greyed out).

Other Terms

F3893, item proposal, customizing.

Reason and Prerequisites

Prerequites for the "Propose items" function are unclear.

Solution

In general the "Propose items" function is only available when creating sales orders in the app. It is not available when editing or displaying sales orders.

Furthermore, the Fiori app does not support the full scope of the dynamic product proposal as it is available in the classic transaction VA01. In particular, for use in the Fiori app both the "Customer Procedure for Product Proposal" and the "Document Procedure for Product Proposal" must be set to the value "A". Other values are not supported by this function in the Fiori app. For details on the dynamic product proposal in general and how to configure it, see the corresponding documentation.
