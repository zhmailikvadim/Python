Symptom

When you run report /US4G/RP_TRIGGER_MDC_BDR_SUP to trigger the process "Receipt of Business Data Request (DSO View)" (DE_B_MDC_RECV_BDR_DSO), a short dump occurs.

This issue is relevant for the market role: DSO.

Other Terms

Report /US4G/RP_TRIGGER_MDC_BDR_SUP, short dump, BDR, Receipt of Business Data Request

Reason and Prerequisites

The error occurred because of a program error.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the system prevents the short dump.

You need to rerun the report. You can verify the fix by checking that no short dump occurs when you run the report.
