Symptom

Unexpected or inconsistent behaviour of the SAP 3D Visual Enterprise applications

Other Terms

troubleshooting | best practices

Reason and Prerequisites

Upgrades, patches, or critical fixes

Solution

  • Clean install / re-install
    • VE Author: SAP Note 2560043
    • VE Generator: SAP Note 2657404
    • VE Viewer: SAP Note 2633067
    • VE Structure Manager: SAP Note 2801768
  • Don’t rename x3d’s / plugins
    • Whenever a critical fix is applied or a plugin is updated, it's important to note that .x3d files have to be removed (or replaced) from the plugins folder entirely.  Simply renaming the plugins will not work.  Best practice is to copy the plugin to a backup folder elsewhere.
  • Fixes are rolled into next major releases as well.
    • All maintenance patches, service/feature packs are cumulative, so fixes will be in next releases.
  • Don't upgrade directly in production without going through testing
    • This is very important for upgrading SAP 3D Visual Enterprise Generator.  Perform validation of updates in a test environment before upgrading a production environment.'
  • Unblock downloaded files
    • SAP Note 2616448
  • Estimated Release Dates
    • SAP Note 2197650
  • Backing up user data
    • SAP Note 2660858
