Symptom

The SAP Note Import Tax in Total Item Value of Nota Fiscal Match Total Value of Nota Fiscal (3707905) requires this SAP Note to be implemented as a prerequisite.

Other Terms

Brazil, SEFAZ, Nota Fiscal, Import, NF, Taxes, NF Totals, vItem, vNFTot, vNF, XML, Rejection 1094

Reason and Prerequisites

This SAP Note is relevant only for Brasil.

This SAP Note contains a report and automatic correction instructions to create/modify objects that are used in the solution delivered via SAP Note 3707905.

Solution

After you implement this SAP Note, the system includes the Data Dictionary (DDIC) object for the Total Value of all items in the Nota Fiscal (NF_ITEM_TOT) field in the Nota Fiscal Header Additional Segment (J_1BINDOCD) structure.

You can receive some object either via automatic correction instruction (aCI) or the new report NOTE_3707904. The delivery method depends on your SAP_BASIS level.

Use the Note Assistant (SNOTE) to implement the correction instructions. You can find more information about the Note Assistant in SAP Support Portal, under https://support.sap.com/en/my-support/knowledge-base/note-assistant.html.

IMPORTANT

  • This report serves only as a tool to generate the required objects automatically. Therefore, it has no positive or negative effect to transport the report NOTE_3707904 to your production system.
  • The objects created by the execution of this report can only be used after you apply the main SAP Note that delivers the solution.

List of Affected Objects:

  • Data Element
    • J_1BNF_ITEM_TOT 
  • Table Definition
    • J_1BINDOCD
