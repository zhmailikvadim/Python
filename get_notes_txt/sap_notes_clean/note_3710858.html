Symptom

You encounter the error "Stock segment &1 not valid for plant &2 and segmentation strategy &3" (Message no. SGT_01 643) when posting a movement type in MIGO for a specific plant.

Environment

  • S/4HANA Retail & Fashion Management (LO-RFM) 
  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Cause

The SGT_01643 error occurs because the components don’t have their batch and stock segment values determined. Note that a blank value is not a valid segment.

Resolution

Implement the BADI_SGT_DET_STRUC_ART to populate the stock segment value for the components. Based on this stock segment value, the batch will be determined, and the validation should pass successfully.

Keywords

BADI, segmentation, SGT_01643, error, stock, MIRO, movement type, plant, BADI_SGT_DET_STRUC_ART, segmentation strategy
