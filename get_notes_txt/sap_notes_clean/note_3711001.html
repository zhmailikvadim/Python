Symptom

Metadata extension is not supported for /PF1/CV_ORDER_MANAGE.
Customers are not able to add their own annotations.

Other Terms

Manage Payments app, metadata extension, /PF1/CV_ORDER_MANAGE.

Reason and Prerequisites

Missing annotation, which allows metadata extension in the CDS view.

Solution

Annotation, which allows metadata extension, is added to the CDS view.

Please apply SAP Note or assigned Support Package.
