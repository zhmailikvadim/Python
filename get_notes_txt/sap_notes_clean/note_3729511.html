Symptom

How do scheduled report execution notifications work in Report Center and Report Distributor?

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

SAP SuccessFactors HCM Suite

Resolution

Report Center

Schedule notification in Report Center can be setup to be sent to multiple users by adding their emails to the 'Send Notifications to' field and selecting which type stage of the execution you would like them to be notified on by selecting the desired options in the 'Send Notification On' section.

Report Distributor (Legacy)

For bundles with an SFTP destination, only the bundle owner/creator will be notified of the scheduled report(s) execution. Even if you add users to the 'Recipients' tab, they will not be notified as this section is exclusive for bundles scheduled to be sent via Email.

This means that if you want to notifiy multiple users, you ought to schedule reports to be sent via SFTP in Report Center rather than in Report Distributor.
