Symptom

When you have only "display access" granted to the EDOBR_REPT object, you can still create, delete, and activate activities in the EDOC_BR_MNG_DISTDFE (Activate Receiver Acknowledgment) transaction.

Other Terms

Inbound, DF-e, NF-e, NFS-e, EDOC_BR_MNG_DISTDFE, EDOC_BR_MNG_DIST_CTE, DRC Manager Distribution DF-e, Nota Fiscal Eletronica, Nota Fiscal Eletrônica, Electronic Nota Fiscal, Brazil, SAP Document and Reporting Compliance, inbound invoicing option for Brazil

Reason and Prerequisites

This SAP Note is relevant only for Brazil.

You use SAP Document and Reporting Compliance, inbound invoicing option for Brazil.

Solution

After you implement this SAP Note, the EDOC_BR_MNG_DISTDFE transaction will apply the display‑only permissions correctly.

Use transaction SNOTE to implement the attached correction instructions.
