Symptom

In the Fiori App “Find Technical Objects’”, when you select the advanced filtering conditions via the tab  “Define Conditions” in the search helps ,  it does not give the expected results. The issue occurs mainly for the D&S specific fields - Tail No/Identifier, Site and Operational Status.

Other Terms

Find Technical Objects, Fiori ID F4635, Defense, DFS

Reason and Prerequisites

The symptom occurs because of the limitation from the underlying framework.

Pre-requiste : Implement the note 3699340 - S/4HANA Defense and Security - Duplicate filter fields in Find Technical Objects application

Solution

Please implement the attached correction instruction.

 This note will remove the tab ‘Define Condition’ for the D&S fields as it is not supported from the underlying framework. However it should be possible to use the standard search.
