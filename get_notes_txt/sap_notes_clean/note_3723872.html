Symptom

You can now integrate your Employee Central system to SmartRecruiters for SAP SuccessFactors. The integration allows you to create a SmartRecruiters job directly from a position, reducing manual work and streamlining your hiring process.

Environment

  • SmartRecruiters for SAP SuccessFactors
  • SAP SuccessFactors Employee Central

Resolution

To avoid any confusion about terminology, a Job Requisition in the SAP SuccessFactors HCM suite is the same as a Job in SmartRecruiters for SAP SuccessFactors. 

Prerequisites

  • You have a license for SAP SuccessFactors Employee Central
  • You have a license for SmartRecruiters for SAP SuccessFactors.
  • Position Management is enabled and set up in your system. For more information, refer to Basic Setup details for Important Initial Settings for Position Management.
  • The Position Org Chart is enabled and set up in the system. For more information, refer to Setting Up The Position Organization Chart
  • You have included the Job Title field in the Position Org Chart configuration.

Permissions:

  • User → SmartRecruiters → Create Jobs in SmartRecruiters from Position
    • This permission enables a link in the context menu of the position in the position org chart to view and create a job (requisition) in SmartRecruiters.
  • User → SmartRecruiters → View SmartRecruiters Job from Position
    • This permission allows users to only view the job (requisition) created in SmartRecruiters from the position.

Important Notes:

  • Note that Users who have the Create Jobs in SmartRecruiters from Position permission must also have the corresponding role in SmartRecruiters.
  • The Position can't be linked to an existing SmartRecruiters Job (Requisition), or vice-versa.

Requisites to create a Job (Requisition):

  • The position must be active as of the current date.
  • Job Title must be maintained in the Position.
  • User creating the Job (Requisition) from Position must have "Create a job" permission on SmartRecruiters

Creating a Job (Requisition):

  1. Go to a Position on Position Org Chart
  2. Select Actions → Create Job in SmartRecruiters (The system redirects you to SmartRecruiters to complete the Job (Requisition) creation process)
  3. Once the Job (Requisition) is created in SmartRecruiters, an icon appears in the position in the Position. If needed, a User can select the icon to open the Job (Requisition) in SmartRecruiters in a new browser tab.
  4. In SmartRecruiters, the Job (Requisition) will be opened on edit mode and the following fields are defaulted in the new Job (Requisition) based on the values from the Position on SuccessFactors:
    • Job Title
    • External Code of the Position
  5. All the remaining required data must be manually added.

Common Issues:

  • "Invalid input parameters for creating job in SmartRecruiters" - Make sure that Job Title is being maintained on the Position.
