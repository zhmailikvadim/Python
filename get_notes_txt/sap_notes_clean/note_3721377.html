Symptom

When refreshing Web Intelligence reports, users experience one or more of the following issues:

  • Prompt text messages overlap with other screen elements.
  • Mouseover tooltips display incorrectly or overlap with other content.
  • Visual rendering issues appear in the prompt dialog interface

Environment

  • SAP BusinessObjects BI Platform 4.3 Support Pack 4 Patch 4.
  • Web Intelligence (Webi)

Reproducing the Issue

  1. Log in to BI Launchpad with valid credentials.
  2. Navigate to a Web Intelligence report containing prompts.
  3. Refresh the report data
    >> Observe the prompt dialog for overlapping text messages or mouseover display issues.

Cause

Cached or corrupted display properties stored within the Web Intelligence document.

Resolution

To resolve this issue, perform the following steps:

  1. Open the affected Web Intelligence report in Design Mode.
  2. Purge the report data along with all saved prompt values.
  3. Save the report to apply the changes.
  4. Close and reopen the Web Intelligence report.


    Note: If multiple reports are affected, repeat this process for each document.
