Symptom

  • Configured Hyperlink, in a table widget cell, to a different page of the story is not working as expected.
  • When clicking on a specific cell and clicking on the "Go to Link" option, the hyperlink is expected to navigate to a different page based on the selected calculated measure.
  • However, the selected calculated measure is not applied as a filter in the different page.

Environment

SAP Analytics Cloud.

Reproducing the Issue

  1. Login to SAC.
  2. Open a story with configured hyperlink to a different page of the story.
  3. Right-click on any data cell and Click on Go to Link > Select Navigate to 'page'.
  4. Notice that the page data is displayed accordingly to selected dimensions, however, it is not filtered according to the calculated measures.

Cause

By design behavior.

Resolution

  • No filter can be applied on values when using a hyperlink to a page or a story. These hyperlinks only support dimension filters (same with table and chart widgets).
  • On the other hand, calculated dimensions can be used as filters, like any other dimension.
