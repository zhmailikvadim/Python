Symptom

In BPC NW 10.1 SP05 Embedded Model, there are two issues when copy and paste Master Data.

  1. User clicks "Add" to append new Master Data in Dimension view. Then user copies from Excel a range of data which contains more than one rows, and pastes it to the new line in Dimension View. The second line becomes an empty line, and the last line is invisible.
  2. Once issue 1 occurs, if user clicks "Add" again to append more data, user will find the last line is shown but user is not able to paste more than one rows after that.

Other Terms

PAK

Reason and Prerequisites

This is a program error.

Solution

Apply BPC NW 10.1 SP05 Patch09 to fix this issue.
