Symptom

When your system is running with UPA (Universal Parallel Accounting) turned off, the CDS view P_FixedAsset does not support the IsMainAsset main‑asset indicator.

The same limitation applies to the fields AssetLifecycleStatus and AssetCompletenessStatus as well.

Other Terms

Universal Parallel Accounting, UPA, P_FixedAsset

Reason and Prerequisites

This is currently a missing feature. When UPA is turned off, these fields cannot be retrieved directly and must be derived from other fields instead.

Solution

Install the note to have those fields included in P_FixedAsset when UPA is turned off.
