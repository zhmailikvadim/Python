Symptom

  • You prepare a VAT return declaration for Hungary using the DRC report HU_VAT_DCL.
  • When you enable the option to show technically canceled documents on the user interface in Analyze Data, the system displays MM technically canceled correction invoices with an incorrect sign for the tax amount.

Other Terms

Hungary, VAT reporting, VAT Return, DRC, HU_VAT_DCL, HUF, M page, MM, Analyze Data, sign, tax amount

Reason and Prerequisites

  • The system does not recognize MM technically canceled invoices as correction invoices in the program.

Solution

  • After implementing this SAP Note, the system correctly identifies MM technically canceled correction invoices in the program and displays the correct sign for the tax amount on the user interface in Analyze Data.
