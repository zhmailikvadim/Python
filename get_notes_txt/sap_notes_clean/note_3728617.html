Symptom

When confirming the Mapping Task "Mapping of Responsible Planner Group or Department" for object PM - General maintenance task list, the error "Mapping task Mapping of Responsible Planner Group or Department contains invalid values XXX" occurs.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Upload the migration template on the migration object: PM - General maintenance task list.
  2. Open the Mapping task: Mapping of Responsible Planner Group or Department.
  3. Confirm the value configured on "Define Planner Groups" (SPRO > Plant Maintenance and Customer Service > Master Data in Plant Maintenance and Customer Service > Technical Objects > General Data > Define Planner Groups).
  4. Observe the error message: "Mapping task Mapping of Responsible Planner Group or Department contains invalid values XXX." 

Cause

The planner group configuration is missing in SSCUI 102761 - Configure Planner Group.

Resolution

  1. Access SSCUI 102761 - Configure Planner Group.
  2. Configure the planner group by adding the required values to the table T024A.
  3. Verify that the planner group is correctly configured.
  4. Retry the data migration process.
