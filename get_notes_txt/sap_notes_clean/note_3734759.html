Symptom

You use the BAdI BADI_RECE_EVALUATION_PROCESS to activate the "Term Reduction" function in a leasing contract. A probable end date prior to the contracts's end date is maintained, and the contract is valuated. The contract end date is then changed to the same probable end date, and a new valuation step is performed.

The system incorrectly identifies a term reduction.

Other Terms

CF_CONSIDER_TERM_REDUCTION, term reduction, absolute end, probable end, right-of-use, RoU, leasing, lease-in, IFRS16

Reason and Prerequisites

 This is caused by a program error.

Solution

Implement the attached correction instructions or import the specified support package.
