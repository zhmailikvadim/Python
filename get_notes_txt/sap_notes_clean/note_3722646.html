Symptom

Users were unable to access the Informatica repository.

Environment

  • SAP Sales Performance Management
  • Incentive Management

Reproducing the Issue

  1. Attempt to log in to the Informatica repository using the provided domain name, gateway host, and port details.
  2. Ensure VPN connectivity before attempting to log in.
  3. Observe the error message displayed during the login attempt.

Cause

To connect into some specifics servers and hosts need to add allow servers and host into VPN Global protect.

Resolution

  1. Ensure that the correct domain name, gateway host, and port are being used during login.
  2. Confirm that the VPN connection is established before attempting to log in to Informatica.
  3. Verify that the user has the necessary permissions to log in to the Informatica repository with support team.
  4. After the support team allows access to the server, validate the connectivity and confirm resolution.
  5. If the issue persists, schedule a meeting with the support team to troubleshoot the problem live.
