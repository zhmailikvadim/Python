Symptom

The following changes are delivered in this SAP Note:

  • Infotype 0164: the 'Occupation codes' help values were displayed in English even when users were logged in in Hungarian; this has been fixed.

  • Infotype 0193: on subtype '7' ('Detailed data of tax allowances'), tax allowances could previously be recorded only up to September;
    we have now extended this to cover the period through the end of the year.

Version information

Version 1: General availability
Version 2: Correction of implementation description

Other Terms

R/3 HR CEE, Hungary, SI allowance 2026

Reason and Prerequisites

The prerequisite for the correction:
HR-CEE/SAP_HR/S4HCM SP11/2025 - HR-CEE/SAP_HR/S4HCM SP03/2026

Solution

---------------------------------------------------------
Installation steps
---------------------------------------------------------

The modifications consist of the following parts:
1. Correction Instructions (attached to SAP Note 3722062)
2. Correction Instructions (attached to the current - 3722061 - SAP Note)

1. Apply the Correction Instructions of SAP Note 3722062 with the Note Assistant (transaction SNOTE; more information on service.sap.com/note-assistant) and execute the further implementation steps in the SAP Note.

IMPORTANT!
- Log on to the requested clients and run the newly created report in Update mode: RPUUDOH0_3722061
(Depending on the SAP_BASIS release and SP level, SNOTE may handle the objects which the report tries to update. If so, running the report will display such objects as "Covered by SNOTE" and skip the object. In this case, SNOTE updates the relevant object.)
- Make sure that the created or modified objects are all active.

2. Apply the Correction Instructions of the current SAP Note (3722061) with the Note Assistant (transaction SNOTE; more information on service.sap.com/note-assistant).

The modifications will also be delivered with the HR-CEE/SAP_HR/S4HCM SP04/2026 package (and the equivalent CLC package for Hungary).
