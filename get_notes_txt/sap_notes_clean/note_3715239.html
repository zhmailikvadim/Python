Symptom

  • Error encountered while generating stack XML: "SOFTWARE UPDATE MANAGER 2.0 SP025 HP-UX on IA64 64bit is not available for user Sxxxxxxxx."
  • Preselection of SUM version SP25 in Maintenance Planner  planning is not possible
  • SOFTWARE UPDATE MANAGER (SUM) version 2.0 SP25 or higher  is not listed on Download SUM - SUM 2.0 page

Environment

  • Maintenance Planning Tools
  • Systems running on Netwaver 7.*

Reproducing the Issue

  1. You are planning a system update on  Maintenance Planner for a system running on operating system version of HP-UX 
    1. In planning step  Select OS/DB version you select SOFTWARE UPDATE MANAGER  
    2. Encounter the error message: "SOFTWARE UPDATE MANAGER 2.0 SP025 HP-UX on IA64 64bit is not available for user S00xxxxxxx0."
  2. While accessing the  Download SUM- SUM 2.0 page, no SUM version is listed which was released after December 31 2025 

Cause

  • KBA 3629028 FAQ: HP-UX all versions reaching end of standard support December 31 2025 

Resolution

  • As described in the KBA 3629028 FAQ: HP-UX all versions reaching end of standard support December 31 2025 , there will be no new SAP Software version released for HP-UX operating system
  • As SUM version 2.0 SP25 was released in 2026, it was not released for HP-UX
  • We recommend to plan a migration to a supported operating system platform
  • In the meantime you can use the old SUM version 2.0 SP24 for system updates

See Also
