Symptom

  • Both of the options of stop indexing and start indexing are grayed out in platform search application. 

Environment

  • SAP BusinessObjects Business Intelligence Platform 4.3/2025
  • Any supported operating system

Reproducing the Issue

  1. Go to CMC(central management console)
  2. Go to application->Platform search application
  3. Stop indexing and Start indexing options are grayed out

Cause

  • The cause of the issue is unknown

Resolution

  1. Go to CMC
  2. Navigate to servers 
  3. Recreate the APS(adaptive processing server) which contains platform search service in it.
  4. Restart the SIA

Keywords

Grayed out, Platform search, Stop indexing,Start indexing, CMC
