Symptom

How to fill Podmiot3 in the FA(3) Schema while submitting the invoice to KSeF 2.0?

Environment

  • SAP ERP
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition
  • SAP Document and Reporting Compliance, Cloud Edition

Reproducing the Issue

  1. Execute EDOC_COCKPIT
  2. Navigate to Poland KSeF Invoice node
  3. Display the eDocument

Cause

How-To

Resolution

For FA(3) Schema, the configuration for the Podmiot3 should be done via EDOPLPODMIOTV2V. Refer

  • SAP ERP: Defining Additional Business Partner Functions for KSeF Version 2 | SAP Help Portal 
  • SAP S/4HANA: Defining Additional Business Partner Functions for KSeF Version 2 | SAP Help Portal 

It is important to know that the view EDOPLPODMIOTV2V is used only if IDWew status is enabled in EDO_PL_ADDNLPARV. If it is not filled, the view EDOPLPODMIOTV is picked for determining Podmiot3. Refer

  • SAP ERP: Defining Settings for IDWew Field for Poland e-Invocing Through KSeF | SAP Help Portal 
  • SAP S/4HANA: Defining Settings for IDWew Field for Poland e-Invocing Through KSeF | SAP Help Portal
