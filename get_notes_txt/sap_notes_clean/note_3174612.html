Question

How to configure Light Enablement feature?

Answer

To configure Light Enablement feature please follow these steps:

  1. Make sure that the Application.Procure.ASNOrderForPrivateSupplier parameter is set to true. This parameter is set to true in the default configuration.
  2. Set the supplier location's Preferred Ordering Method to one of the following values:
    1. AribaEmail: routes orders by email through the SAP Business Network (applicable for on premise)
    2. AribaFax: routes orders by fax through the SAP Business Network (applicable for on premise)
    3. Email or Fax ((applicable for on-demand solution)
  3. Make sure that entitlement Enable Intelligent PO is enabled in your SAP Business Network account.
Additional Information

To enable this, the supplier should not have any relationship with Buyer on Buyer AN.
