Symptom

Consumed budget shows negative sign.
Postings are allowed even if there is no budget, or budget is less than the consumed amount.
As a result, there is more available amount than expected, leading to permitted postings higher than the consumable budget.

Environment

Funds Management (PSM-FM) and BCS active

Reproducing the Issue

Run for instance FMAVCR01 for an expenditure Control object
Note the negative consumption sign
As a result, the available amount, that is a calculation of 'Consumable budget' - ('Consumed amount' - ), gives a positive result, even if there was no consumable budget maintained.

Cause

Customizing of tolerance profiles not controlling Incoming amounts.

When posting reversals on Expenditure commitment items,  they will be authorized by default even if there is no Consumable budget maintained

Resolution

Adapt the tolerance profiles including Incoming amounts as ceiling type in this IMG activity:
Public Sector Management -> Funds Management Government ->Budget Control System (BCS)  ->Availability Control 
Edit Tolerance Profiles 

More information about Ceiling Types in the Online help for Ceiling types in BCS

See that, by default, Incoming Amounts are authorized by AVC without any limit

Keywords
