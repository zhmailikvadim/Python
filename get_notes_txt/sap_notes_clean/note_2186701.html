Symptom

The infotype 0662 ("Semiretirement A Extras - Notification Supplements") can be switched over to the new infotype framework (screen 2100) using a configuration - see SAP Note 2130644 ("Austria-Specific Infotypes in New Infotype Framework").

Other Terms

Screen 2100 [Dec]
IT0662
Semiretirement A - Notif. Supplmnts

Reason and Prerequisites

Further development

Solution

Import the corresponding HR Support Package in the affected system and perform configuration in accordance with SAP Note 2130644.
