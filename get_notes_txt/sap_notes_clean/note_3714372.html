Symptom

When editing any position or create a position then click on save, error "The access validation is failed" pops up.

Environment

SuccessFactors Employee Central

  • Position Management

Reproducing the Issue

  1. Navigate to manage positions.
  2. Edit any position or create a position then click on save.
  3. Error pops up: errorId=null, timestamp=null, errorMessage=The access validation is failed! Permission check failed.

Cause

User does not have permission to Manage Position-> Access Position Organization Chart. This is a defect confirmed by the engineering team.

Resolution

Workaround:

Provide permission to Manage Position-> Access Position Organization Chart

The issue will be permanently fixed in 2026 1H release.
