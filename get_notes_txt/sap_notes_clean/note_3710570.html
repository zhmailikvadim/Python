Symptom

During the BP sync process in the new model one or more personnel numbers failed to sync with the error message "Employee &1 : There is no valid CP-P relationship maintained.". No entry is made in the table /SHCM/D_BP_SYNC.

Other Terms

 /SHCM/D_BP_SYNC, SHCM/EMPLOYEE_BL073

Reason and Prerequisites

The error is a major inconsistency which should be addressed directly. You can refer to Note "3427429 - RPU46CX_CENTRAL_PERSON_ONLINE: Generate Missing P-CP Relationship" in order to find out more about how to fix this issue.

Solution

Apply the corresponding support package. You may also apply the corresponding correction instructions.
