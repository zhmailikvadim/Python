Symptom

The "Go" button is not being translated to other languages in Growth Portfolio > Skills/ Competencies. When debug is used, it has no label key available.

Environment

SAP SuccessFactors Talent Intelligence Hub 2H 2025

Reproducing the Issue

  1. Login and change system language to a language other than English
  2. Go to Growth Portfolio
  3. Open any attribute types (e.g Skills)
  4. Check the Go button (see that in other languages it is not being translated, in debug it has no label key available)

Cause

This is a Known Issue.

Resolution

The fix for this issue is planned to be deployed on 1H 2026 release.

For release timelines, please review SAP SuccessFactors Product Release & Road Map Information.

Workaround

No workaround is available.
