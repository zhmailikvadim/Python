Symptom

  • Reports are displaying incorrect data after universe conversion and report repointing.
  • SQL scripts are being updated unexpectedly after the repointing of webi reports created from custom sql from unv to unx.
  • @prompts from custom sql are missing after repointing the report from unv to unx.

Other Terms

webi, universe, unv, unx, change source

Reason and Prerequisites

Environment:

  • SAP BusinessObjects Business Intelligence Platform 4.3 Support Package 05 Patch 0
  • Web Intelligence

Steps to Reproduce:

  1. Login into BI Launchpad.
  2. Opened the WebI report.
  3. Refreshed the report.
  4. Ran the query. The data was displayed, but the SQL script still shows custom SQL with the @prompt function.
  5. Changed the source from UNV to UNX using the change source option.
  6. Observed that the custom SQL script gets changed after change source and no @prompt function observed in the query.
  7. Different result observed on the report
