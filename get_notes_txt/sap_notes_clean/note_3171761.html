Question

How do I search for contract workspaces and contract requests using Internal ID?

Answer

To search specifically on the Internal ID field:

  1. Click Search (the magnifying glass) and choose your project type to search within.
  2. In the search form, change Project Type to All.
  3. Click Reset to fully refresh the screen.
  4. Click the + icon to add a search term.
  5. Add either ID or ID (Exact Match)

The ID field allows for partial matching. The ID (Exact Match) field will only return results that exactly match your search string.

Additional Information

The free text search bar will find the inputted search string in all searchable fields and documents, this includes Internal ID.  Keep in mind, the free text search bar searches all fields and documents. Therefore, your search results may contain more results than expected.

The free text search bar is located on the Contracts tab or by clicking Search (the magnifying glass) and selecting the project type you would like to search. It is the first field in the search form.
