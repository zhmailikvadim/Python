Symptom

This note addresses several errors that were discovered in the SP09 EIM in HANA ABAP API post release.

1) The Schema is determined incorrectly on system running IM in HANA in side-by-side mode and on an app server running on a non-HANA DB.

2) The API doesn't handle scenario where Task Framework task is not dropped and recreated with every run.

3) Error message does not return task framework error correctly.

 

Solution

Implement the manual pre-implementation correction instructions followed by the automatic correction instructions.
