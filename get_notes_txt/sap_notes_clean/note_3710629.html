Symptom

 In the process "Start of Basic or Backup Supply for Electricity (DSO View)" (DE_C_GPKE_DSO_SBS), an unnecessary refetch option appeared in step 150-060 when an APERAK message was received.

Other Terms

 GPKE, DE_C_GPKE_DSO_SBS, APERAK, refetch, Start of Basic or Backup Supply for Electricity, SBS, basic supply, backup supply

Reason and Prerequisites

The error occurred because the system didn't remove the refetch option when processing the APERAK message in the SBS process, even though this option isn't needed for this process.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the system deletes the refetch option when an APERAK message is received in step 150-060. You can verify the fix by retesting the scenario and checking that the refetch option no longer appears in the SBS process.
