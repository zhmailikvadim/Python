Symptom

During posting from Data Object (DO) to Claim, the system should complete all required updates successfully.

However, an update termination occurs during the update process, which interrupts the DO to Claim posting and prevents completion.
This note will fix the issue so that the update termination no longer occurs and the posting process completes successfully.

Other Terms

update termination,  claim posting issue, posting interruption, update failure, /IRM/GPELD, /IRM/GCR_UPDATE_DB, Runtime Error

Reason and Prerequisites

This is caused by program error.

Note : SAP Note#3707150 is available for V/4 release.

Solution

Please implement the correction instructions attached. These corrections are only valid if you have applied the Support Packs

Release 6.4 – Base Package & Support Pack 1
Release 6.3 – Base Package

Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object
