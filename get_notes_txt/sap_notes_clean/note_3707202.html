Symptom

  • Report views still show fields that are moved to the unused fields section
  • No way to delete these fields from the views section in reports

Environment

SAP LeanIX - Reports and meta model configuration

Resolution

  1. Navigate to the meta model configuration of the fact sheet type of the field.
  2. Move the attribute out of the "unused fields" section to any temporary section.
  3. Select the attribute and click on the gear icon.
  4. Disable the "include in views" option for the attribute.
  5. Move the attribute back to the "unused fields" section.
  6. Save the changes to the meta model configuration.

Keywords

leanix, meta model configuration, unused fields, report views, attribute settings, disable views, leanix reports, configuration settings
