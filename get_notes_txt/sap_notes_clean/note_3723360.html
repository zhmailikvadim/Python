Symptom

Repository Object Sync Error: HTTP Call Error, Connection Broken

Environment

 SAP Cloud Identity Access Governance 2.0

Reproducing the Issue

1. Go to SPRO >  Governance, Risks and Compliance > Synchronization Jobs and run the Repository Object Sync (t-code. You can also use t-code GRAC_REP_OBJ_SYNC.

There is a Proxy Host configured in the HTTP Proxy Options section for the IAG_SOD_AUTH and IAG_SOD Destinations (SM59).

Resolution

IAG_SOD destination (SM59) missing "/" (without quotes) for the Path Prefix.

Keywords

IAG, Bridge, Repository, Object, Sync, Error, Path, Prefix, Connection, Broken, Destination, IAG_SOD
