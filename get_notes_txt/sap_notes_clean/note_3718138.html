Symptom

  • A custom UI was created for a custom MDF object, and the URL field displays a different result in the general overview compared to the editing mode.
  • Clicking the URL in the general overview of the portlet does not redirect to the URL.

Environment

SAP SuccessFactors HCM Core

Reproducing the Issue

  1. Configure a UI for a custom MDF object and add a URL field.
  2. Save the UI and add this to the people profile.
  3. Observe the link in the general overview of the portlet and compare it with the link during editing. Only in editing mode will the correct URL show.
  4. Click on the link in the general overview and note that nothing happens.

Cause

It has been identified that if special characters are used within the URL this issue occurs.

Resolution

This fix for this issue is planned to be deployed on 1H 2026 release.
For release timelines, please review SAP SuccessFactors Product Release & Road Map Information.
