Symptom

When you open app F2101A: Monitor Material Coverage - Net and Individual Segments (Version 2), the options for the Shortage Evaluation Period filter do not correspond with the definition of the selected shortage profile, e.g. the end of the evaluation horizon is not what you have maintained in customizing.

Other Terms

F2101A, Monitor Material Coverage - Net and Individual Segments (Version 2), Shortage Evaluation Period, Shortage Profile, filter, evaluation horizon

Reason and Prerequisites

The filter options were not created based on the selected shortage profile on initial app load.

Solution

This note changes the behavior so that the filter options are always created based on the selected profile.
