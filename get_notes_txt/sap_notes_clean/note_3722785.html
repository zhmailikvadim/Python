Symptom

When executing "Mark for Testing" for AVAS object under the Deletions tab in SPAU, the error "Unknown assignment id" occurs.

Reproducing the Issue

1. Tcode SPAU.

2. Go to the Deletions tab.

3. Select AVAS type object.

4. Click on 'Mark for Testing' and execute.

Resolution

AVAS type objects do not support locking or modifying objects of this type that have been deleted.

For AVAS type objects, actions that attempt to modify deleted object (such as "Mark for Testing" or "Change Processing Status") will not succeed. This is expected behavior.

Keywords

SPAU, Deletions, AVAS, Mark for Testing, Change Processing Status, Unknown assignment id
