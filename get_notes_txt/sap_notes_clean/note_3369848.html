Symptom

You want to monitor cloud services of type SAP BTP, Cloud Foundry environment in application Health Monitoring or Real User Monitoring with SAP Focused Run 4.0 feature pack 01.

This requires some down ports and notes to be implemented first.

Other Terms

FRUN Proxy

Reason and Prerequisites

  • You need a SAP Cloud ALM tenant and the API service key for that.
  • Your cloud services must be connected to your SAP Cloud ALM tenant. 
  • Your SAP Focused Run system must be connected to this SAP Cloud ALM tenant.
  • A SAP Cloud Connector must be set up and configured in SAP CloudALM/Landscape Management (see Expert Portal).

The SAP Focused Run system requires the following notes:

Note Description
3324172 ICF-OAuth Support in SM59
3338898 Corrections for Real User Monitoring FRUN 4.0 FP01 and FRUN Proxy objects
3369171

Health Monitoring: Active or deactivate data collection for Cloud Foundry cloud services

Solution

Import the required notes.

Required Authorizations.

The user who wants to configure the connection to SAP Cloud ALM and activate/deactivate the data collection needs the following authorizations:
