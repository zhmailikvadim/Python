Symptom

There is a need to create a new Purchasing Organization in your Set Up Organizational Structure app within your SAP Central Business Configuration workspace.

Environment

SAP Central Business Configuration

Cause

Purchasing Organizations are not created directly under the Company Code node in the organizational structure in the Organizational Structure.

Resolution

Before proceeding with the Purchasing Organization, ensure that the workspace is in the “Scope and Organizational Structure” phase.

To create a Purchasing Organization, follow the steps below:

  1. Access the activity “Set Up Organizational Structure” and click on “Edit”.
  2. Select either Table View or Chart View to proceed with the creation.
  3. In Table View, use the search functionality to find the option “+ Create Purchasing Organization” and select it.
  4. In Chart View, scroll horizontally to the right until the tile “Create Purchasing Organization” becomes visible and select it.
  5. Maintain the required information, including the Purchasing Organization ID, Name, and assign the relevant Company Code.
  6. Confirm the entered data and save the changes to create the new org unit.

After completing these steps, the new Purchasing Organization will be available in the Organizational Structure.
