Symptom

This SAP Note enables you to calculate contribution amounts for the Child and Child-Rearing Support Fund in monthly payroll and Shoyo payroll calculations.

The system provides the following new wage types for contributions in payroll calculation:

  • /607, /608, and /609 for monthly payroll
  • /617, /618, and /619 for periodical Shoyo payroll
  • /627, /628, and /629 for non-periodical Shoyo payroll

The system uses the following new rate types for Child and Child-Rearing Support Fund calculation:

  • 13: contribution rate for monthly payroll
  • 14: contribution rate for Shoyo payroll

You can configure these rates in view V_T5JSR for each social insurance (SI) modifier.

Personnel calculation rules are also revised to calculate the contribution wage types during monthly payroll and Shoyo payroll runs. For more information, see the manual activities section of this SAP Note.

Other Terms

J14S, JPS1, JK11, JPSC, JK12, JK15, JPC0, JNC0, JPCR, J141, J14X, J14R

Reason and Prerequisites

This is a part of legal changes.

Solution

You can apply the relevant HR support package. Alternatively, you can implement the manual changes together with the correction instructions in this SAP Note.

After you implement the changes in this SAP Note, the system adds table MST to the input parameters in the payroll log for payroll function EXPRT.
