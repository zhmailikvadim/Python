Symptom

When changing the status of material BOMs, the full status description is not displayed.

Other Terms

PLM-42278, /DSCSAG/MAT_BOM_STATUS_GET_MUL, /DSCSAG/BOMSTAT

Reason and Prerequisites

Incorrect structure being used inside the function module /DSCSAG/MAT_BOM_STATUS_GET_MUL. The structure /DSCSAG/MATSTAT was designed to store material status information.

Solution

A dedicated structure /DSCSAG/BOMSTAT has been created for /DSCSAG/MAT_BOM_STATUS_GET_MUL to support the full 30‑character BOM status description.
