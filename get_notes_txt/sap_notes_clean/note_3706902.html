Symptom

Range of User Ids do not appear in the Maintain User Data App.

Environment

SAP Cloud Identity Access Governance 2.0

Reproducing the Issue

  1. Open the Maintain User Data App
  2. Search for a group of User Ids which belong to a particular range.

No results are returned.

Resolution

Check the Application > User Filter tab from the Applications App to ensure that you are not excluding users that belong to a certain range.

  1. Open the Applications App.
  2. Select the User Filter tab
