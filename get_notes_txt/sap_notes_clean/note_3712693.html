Symptom

Following 2 syntax issues occurs during implementation of S/4HANA Readiness Check notes:

  1. Program CL_SATC_CI_CHECK_MODULE_BASE==CP, Include CL_SATC_CI_CHECK_MODULE_BASE==CM00L: Syntax error i
    The field 'C_AT_EXEMPTN_EXISTS' is unknown, but there are the following fields with similar names: '
  2. Program CL_SATC_CI_ADAPTER============CP, Include CL_SATC_CI_ADAPTER============CCIMP: Syntax error in line 000201
    Method 'SET_CLIENT_VERSION' is unknown or PROTECTED or PRIVATE.

Environment

SAP S/4HANA

SAP NetWeaver Application Server for ABAP

ATC

Reproducing the Issue

  1. Prepare for S/4HANA conversion by implementing lots of SAP Notes in 2436688
  2. Encounter mentioned syntax errors during note implementation.

Cause

SAP Note 2786572 was not implemented in RD1, which is required as per the text part of SAP Note 2766934.

Resolution

  1. Verify whether SAP Note 2786572 has been implemented.
  2. Ensure that SAP Note 2786572 is implemented in both the central check system and the affected checked systems, as stated in the solution part of SAP Note 2766934.

See Also

  • Refer to: SAP Note 2786572 - Refine Classification of Exempted Code-Inspector Remote Check Results
  • Refer to: SAP Note 2766934 - Central Check System Implementation
