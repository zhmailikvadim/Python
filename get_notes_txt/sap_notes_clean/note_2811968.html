Symptom

You are looking to have a relative selection for an extension field, but when opening the value help from the report selection, there is no relative selection offered for this field.

Environment

SAP Cloud for Customer

Reproducing the Issue

  1. Go to the work center Business Analytics and the view Design Reports.
  2. Open your report.
  3. In the selection, open the value help of your extension field.

You can select absolute values, but no relative selection is possible

Cause

Relative selections are automatically added depending on the type of the field.

Resolution

Relative selections are available for the field type Date, for example. There are no relative selections for the field types Indicator, Text, List and Date/Time.

If you would like to suggest changed to the standard behavior, please make use of the Continuous Influence Session.

See Also

2340173 - No Relative Select Or Value Help Entries For Some Report Fields
