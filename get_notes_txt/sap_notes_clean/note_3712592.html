Symptom

You create a Production Operation List and use a Change Object as the Change Master with a specific validity date. After you release this Change Object as the Change Master, the PLM integration sets the processing status of the related Change Master to inactive. When you try to update the operations in the Production Operation List using a new Change Object as the Change Master with the same validity date as the released Change Object, you see the error message “Change master status 02 does not allow changes.”

Other Terms

PLMSI, PLM system integration, POL, Production Operation List, Change Object, Change Master, Status, Operation

Solution

Implement this SAP Note to resolve the symptom.
