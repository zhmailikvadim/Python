Symptom

You use the Run Statutory Reports application to create the Russia Concession Claim for Assets.

In reporting entity configuration you maintained concession claim versions with different time validity.

You create a new report run for the period. You expect the system to take the value of the concession claim version valid on the start of the current date, however, the system takes the value valid on the reporting period end date.

Other Terms

Russia, RU, SAP Document and Reporting Compliance – statutory reporting, formerly known as Advanced Compliance Reports, Run Statutory Reports, Run Advanced Compliance Reports, Russia Concession Claim for Assets, RU_ASSET_TAX_CONC_APPLICATION

Reason and Prerequisites

This SAP Note is relevant only for Russian Federation.

Solution

Import the Support Package specified for your release or implement the attached correction instruction.

As a general rule SAP recommends that you implement the Support Package specified for your release. However, if you need to install a solution earlier, use the SAP Note Assistant to implement the correction instruction. You can find more information about the SAP Note Assistant in SAP Support Portal, under SAP Note Assistant.
