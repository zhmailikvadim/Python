Symptom

After proxying to an employee with a different theme and returning to the original user (proxy), the default theme does not revert. Instead, the proxier employee's theme remains displayed on the main page.

Environment

SAP SuccessFactors HCM Suite

Reproducing the Issue

  1. Log in to the tenant in your own user.
  2. Check the default theme.
  3. Proxy as another user
  4. Check the user's theme.
  5. Return to the own user.
    1. The theme as the own user remains as proxier theme.

Cause

Our Engineering team has identified this behavior as a known error, that need fix at code level.

Resolution

This behavior is identified as a known error, and it's fix will be issued in the next release cycle (1H2026), by May 15th 2026 for production environments.
