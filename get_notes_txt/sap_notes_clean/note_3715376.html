Symptom

You try to update PRT(assignedProductVersionTools or assignedDocumentVersions) in the Production Operation List. When you do this, you see one of these error messages:

  • "Object to be processed could not be uniquely determined" (CPCC_DM 001)
  • "Customer object of type PRT counter *** exists. Clean up object first" (/PLMF/POL_CORE 052)

 You also try to update a Production Operation List by sending an empty Suboperations-Descriptions collection. In this case, you see the error message "No suitable object found" (CPCC_DM 002).

Other Terms

PLMSI, PLM system integration, POL, Production Operation List, PRT, Production Resource Tools, assignedProductVersionTools, assignedDocumentVersions, Description

Solution

 Implement this SAP Note to resolve the symptom.
