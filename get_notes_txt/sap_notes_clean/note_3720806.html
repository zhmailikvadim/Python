SAP S/4HANA FOUNDATION 2025, CL_SALV_FUNCTIONS_TREE->set_adapter_fc_handling

Program Error

Solution

Import the relevant Support Package for your release. You can use transaction SNOTE to implement the correction instructions relevant for your release in advance.
There are no manual activities.

This note will fix the issue so that claims with zero amount are processed correctly during posting.
