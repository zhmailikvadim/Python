Symptom

  • The Tricentis License Usage Statistics service is failing to start.
    This is due to the .NET 4.7.2 Framework not being installed on the Server/Machine running the License Server.
  • Service 'Tricentis License Usage Statistics' (TricentisLicenseUsageStatistics) failed to start. Verify that you have sufficient privileged to start system services.

Environment

SAP Enterprise Continuous Testing by Tricentis

Cause

The .NET Framework 4.7.2 shipped with Windows Server 2019.

If your targeted License server is older than Windows Server 2019, say 2016 or older, verify that the .NET Framework 4.7.2 is installed.

If the targeted server is a Windows Server 2016 and during the installation, you're prompted with the error.

Resolution

Check the following knowledge article from Tricentis:

License Server | License Usage Statistics Startup Failure
