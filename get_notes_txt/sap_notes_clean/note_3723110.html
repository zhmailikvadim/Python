Symptom

No results are returned when querying warehouse tasks using the web service, even though the task details are visible in the UI.

Environment

SAP Business ByDesign

Reproducing the Issue

  1. Go to Application and User Management work center.
  2. Choose Service Explorer view.
  3. Search for the service QuerySiteLogisticsTaskIn.
  4. Select the service and download the WSDL file.
  5. Create a new project in the SoapUi.
  6. Execute the query using the request payload in SoapUI.
  7. No Records returned in response payload.

Cause

  • The web service shows only the warehouse tasks details with status “Not Started” or “In Process”,
  • The webservice "QuerySiteLogisticsTaskIn" doesn't return the Finished warehouse task details.

Resolution

This is the expected system behavior.
