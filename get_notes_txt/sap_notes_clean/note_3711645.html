Symptom

  • Links are displayed in the MDF section of the latest people profile, even when no MDF records exist.
  • Instead of the “Add Data” button, configured links are shown.

Environment

  • SAP SuccessFactors HCM Suite

Reproducing the Issue

  1. Navigate to the user profile on Latest People Profile.
  2. Access the MDF section where links are configured.
  3. Observe that links configured via Manage Configuration UI are displayed even though no MDF records exist.

Resolution

  • This is a known issue, which was resolved by the engineering team.
  • The corrective patch was deployed as of February 21st, 2026.
  • In case issue persists, please log a ticket with component LOD-SF-MDF-PPX.
