Symptom

An exception to the IN_WRONG_TABLESPACE is required.

Environment

  • Oracle RDBMS
  • SAP Netweaver
  • Any OS platform

Cause

Usually tied to Unified Auditing objects that will throw warnings due IN_WRONG_TABLESPACE validation.

Resolution

Exceptions for the IN_WRONG_TABLESPACE should be added line by line, for each tablespace or object to be excluded, as below:

check_cond = ("DBA:IN_WRONG_TABLESPACE:N:W::::<OBJECT_NAME>")
check_cond = ("DBA:IN_WRONG_TABLESPACE:N:W::::<TABLESPACE_NAME>")

The remaining validation will continue normally.

See Also

SAP Note 892294 - Enhanced support for non-ABAP database in BRCONNECT 7.00

SAP KBA 2507100 - BR0970W Database administration alert for SYSAUD tablespace

Keywords
