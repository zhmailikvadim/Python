Symptom

When opening a production order in transaction code CO02 or CO03, the time zones displayed in the 'General' tab and the 'Administration' tab are different.

Environment

  • SAP ERP Production Planning (PP)
  • SAP R/3
  • SAP R/3 Enterprise
  • SAP ERP Central Component
  • SAP ERP
  • SAP Enhancement package for SAP ERP
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA, on-premise
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Access transaction code CO03.
  2. Input production order number.
  3. Check the time in 'General' tab and the 'Administration' tab.

Cause

In the production order, the dates on the "General" tab are displayed in the time zone of the plant, whereas the dates on the "Administration" tab are displayed in the system time zone. Therefore, if the system time zone is different from the plant time zone, the time zones displayed in the 'General' tab and the 'Administration' tab will be different.

Resolution

This is not an error, but as per standard system design.

Keywords

Basic Dates, Dates/Times, Creation/change data, Production order header.
