Symptom

Our Customer Influence Program allows for the requesting of a new feature, enhancement, or improvement that is not currently available in SAP Business Data Cloud. 

Environment

SAP Business Data Cloud

Resolution

  1. To create a new enhancement request, go to the following link and click "Submit Improvement":  BDC Enhancement Request in the Influence Portal  
  2. The following link can be followed to upvote existing ideas and enhancement requests: https://influence.sap.com/sap/ino/#/ideas
  3. Find the new features, as they are released, documented here: What's New in SAP Business Data Cloud
  4. View the roadmap for new features here: SAP Roadmap Explorer

See Also

  • For SAP Analytics Cloud, see KBA 2424376 How do you create an enhancement request and provide feedback for SAP Analytics Cloud?
  • For SAP Datasphere, see KBA 3332382 - How to create an enhancement request for SAP Datasphere?
