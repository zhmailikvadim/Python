Symptom

  • An error is encountered while performing cash concentration in the Manage Cash Concentration App (F3265)
  • The application log fails to display relevant information about the error.
  • This issue prevents log analysis when there is an error. 

“Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.”

Environment

SAP S/4HANA 1809 onwards

Reproducing the Issue

  1. Perform cash concentration in the Manage Cash Concentration App (F3265)
  2. Verify that the operation status is displayed as "Error".

  3. Access the application log and try to obtain details about the error.

     

  4. Confirm that the application log does not provide any error-related information. 

Cause

This is the expected standard behavior of the system.

Resolution

  1. Open the Manage Cash Concentration app (F3265).
  2. Navigate to the Application log section.
  3. Select the search button (magnifying glass icon) to review the log details.
