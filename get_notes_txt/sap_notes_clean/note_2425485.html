Symptom

 
Scope delivered on 09.02.2017:
  • Improvements to the TMRU cockpit functionality:
    • Improvements of Default Templates loading during Waybill Creation process;
 

Other Terms

 Transportation Management, TM Russian Localization, Waybill.

Reason and Prerequisites

  • The above mentioned symptoms appear in your system.
  • This SAP Note is relevant only for Russian Federation.
  • TMRU 100 SP03 must be installed.

Solution

  • As a general rule, SAP recommends that you install a solution by applying a Support Package. However, if you need to install a solution earlier, use the Note Assistant to implement the correction instruction. You can find more information about the Note Assistant in SAP Service Marketplace, under service.sap.com/note-assistant.
  • Note that documentation, transactions, and Customizing activities are delivered via Support Package only.
  • To implement the solution, you must carry out Manual Post-Implementation activities (see attachment) and implement the correction instructions that are valid for your release using transaction SNOTE.
