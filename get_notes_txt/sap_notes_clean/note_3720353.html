Symptom

You have setup eligibility rules for the Variable Pay template, however ineligible users are being added to the forms. 

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

SAP SuccessFactors Variable Pay

Cause

The XML tag "include-eligible-users-only" is set to FALSE

Resolution

When tag "include-eligible-users-only" is set to false the system will include all users available in Employee History, despite they being eligible or not. 

If you want the system includes only eligible users please change this XML tag to TRUE.
