Symptom

You have an Iflow which face the following error:
       org.apache.cxf.interceptor.Fault: Response was of unexpected text/html ContentType. Incoming portion of HTML stream

Environment

  • Cloud Integration
  • SAP Integration Suite

Cause

The error message shows, this is a response error.
This means that the called receiver system is responding back with error message.
As the error received in a different format then what the receiver adapter in the Iflow could understand, the unexpected error showed.

Resolution

The real problem is at receiver side.
Reach out the admins of the receiver system and request to check what the problem is.

Keywords

SAP Cloud Platform Integration, SAP HCI, SAP CPI, SCPI, CPI, CI, tenant, IFlow, Integration Flow, tracing, HTML, response, format, unexpected, interceptor, ContentType
