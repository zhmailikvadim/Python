Symptom

For Sales Orders that are relevant to SAP CGTO, when you create an outbound delivery using transaction VL01N - Create Outbound Dlv. with Order Ref. and post Goods Issue directly, the system does not trigger the outbound event.

Other Terms

  • SAP S/4HANA, add-on for Industry Solution for Life Sciences (ISLS)
  • SAP Cell and Gene Therapy Orchestration (SAP CGTO)
  • Outbound Event for Goods Issue

Reason and Prerequisites

Program error.

Solution

Option 1: Please implement the correction instructions provided with the SAP Note.

Option 2: This note depends on note 3692291. Note 3692291 provides advanced development to support integration of Subcontracting Purchase Orders with SAP CGTO. If you don't need this advanced development, SAP recommends an alternative approach:

  • Create an outbound delivery using transaction VL01N - Create Outbound Dlv. with Order Ref.
  • Use transaction VL02N - Change Outbound Delivery to post Goods Issue.
