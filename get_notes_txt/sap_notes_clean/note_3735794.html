Symptom

  • In IDT, when creating or editing List of values get error:

 Edit Query Error

 Problem editing query.
       Error:
       Query script generation failed. See your administrator. Session ID was not specified (IES 00013)

 Cause of Error
 Session ID was not specified

  • Connectionserver is crashing intermittently

Other Terms

Google Big Query, LOV, LOVs, LOV's, SQL expression field

Reason and Prerequisites

Environment:

  • SAP BusinessObjects Business Intelligence Platform 2025 patch 9
  • Information Design Tool (IDT)
  • Lists of Values (LOVs)

Steps To Reproduce:

Workflow 1:

  1. Open IDT
  2. Open session in Repository resource
  3. retrieve eFashion unx universe
  4. In the .blx, go to Parameters and Lists of Values
  5. Click on one of the existing LOVs.
  6. Click on "Edit Query" button
  7. Get error

Workflow 2:

  1. Try creating a new List of Values based on Custom SQL
  2. Click on Edit SQL... nothing happens and no error

Possible workaround:

  1. Close IDT
  2. Reopen IDT
  3. Open a new session/log in to BI platform
  4. Retrieve eFashion (.unx)
  5. Open the .blx from eFashion universe
  6. Click on Parameters and Lists of values
  7. Choose one of the LOVs (not static)
  8. Click Preview and the values display
  9. After this step, you can create and edit LOV's.
