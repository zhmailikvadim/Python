Symptom

My customer sent a new version of Purchase Order. However the status is not 'Confirmed'. 

Resolution

If the Change Order contains any changes in the Line Item, it is most likely that the rule 'Retain confirmation status for changed line items on change orders that match supplier confirmation' is not enabled.

When this rule is not enabled, the confirmation status from the previous PO version will not be retained on the new PO version for changed line items.

You can see if this transaction rule is enabled for your buyer by following these steps:

  1. Sign in to the SAP Business Network.
  2. In the upper right corner, click your initials and select Settings > Customer Relationships.
  3. In the Current Customers section, click on the company name in the Customer column.
  4. On the Customer Details page, go to the Order Routing Rules at the bottom.
  5. If 'Retain confirmation status for changed line items on change orders that match supplier confirmation' is not listed, then this rule is disabled.

See Also

This rule applies only if you have allowed suppliers to send order confirmations on the line-item level and when the order confirmation approval rule is not enabled. When this rule is enabled, the confirmation status of changed line items from the previous version of an order is retained on change orders that match supplier confirmation. If this rule is disabled, confirmation status is not retained on change orders. This rule does not apply to scheduling agreements and scheduling agreement releases.
