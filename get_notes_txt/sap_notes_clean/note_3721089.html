Symptom

Users are unable to access diagrams in the SAP Signavio Process Collaboration Hub.

"Access denied" error message appears when attempting to view a model.

Cause

User does not have a Hub license assigned.

Resolution

  1. Verify whether the affected user has a Hub license assigned.
  2. If the user does not have a Hub license, assign the Hub license to the user.
  3. Confirm that the user is now able to access the diagrams in the SAP Signavio Process Collaboration Hub.

See Also

Refer to: 3160476 - Restrict Access rights of a user
