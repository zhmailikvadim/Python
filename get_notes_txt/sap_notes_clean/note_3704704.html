Symptom

When trying to execute a portfolio transfer with transaction TPM82 the transaction is not processed and discarded instead. 

Environment

Treasury and Risk Management (TRM)

Cause

Configuration

Resolution

  1. A portfolio transfer with TPM82 can only be performed if the position uses portfolio as a differentiation criteria.
  2. Refer to the following documentation regarding this prerequisite: Portfolio Transfer
    Kindly create a new product type and within the below customizing path make sure to assign portfolio as differentiation criteria:
    Financial Supply Chain Management
    -Treasury and Risk Management
    --Transaction Manager
    ---General Settings
    ----Accounting
    -----Settings for Position Management
    ------Define and Assign Differentiations 
  3. Keep in mind that the existing customizing must not be changed for already existing positions as this can lead to inconsistencies.
  4. In addition kindly note that in transaction management (in FTR_EDIT) the portfolio can only be changed if no postings were performed.
