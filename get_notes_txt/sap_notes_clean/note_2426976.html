Symptom

Negative value appears for some questions in Item Evaluation by Instructor report

This KBA whill help you understand why for some questions mean score is negative value.

Environment

SuccessFactors Learning Management System (LMS) - All Supported Versions

Reproducing the Issue

  1. Login to LMS
  2. Naviagte to Reports and search for Item Evaluation by Instructor report
  3. Select Item and Survey ID from filter criteria
  4. Run report
  5. Mean score for few questions appear as negative value (for example, -3,-2 etc.)

Resolution

Negative value means that many users has selected the option "N/A" for that specific question.

This is expected report behaviour.
