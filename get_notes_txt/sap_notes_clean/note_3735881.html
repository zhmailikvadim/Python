Symptom

While posting goods receipt towards valuated sales order stock, error occurs "Valuation data for material xxxx for sales order item xxxxxx xxx does not exist". 

Environment

  • SAP ERP
  • SAP ERP Central Component
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA
  • SAP S/4HANA Finance
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Open transaction MIGO.
  2. Enter the material and other required information for the goods receipt.
  3. Perform the check.
  4. Observe that error MM124 is raised.

Cause

Check firstly whether cost component structure is changed recently. If it is changed, please follow this resolution.

Resolution

Implement the correction from Note 0003157727.

See Also

Note 0003157727
