Symptom

This SAP Note contains technical corrections to improve the performance of eSocial event S-1210 - Pagamentos de Rendimentos do Trabalho (Labor Income Payments).

The performance is improved when you try to generate an S-1210 event for an employee, but the S-1200 Remuneração de trabalhador vinculado ao Regime Geral de Previd. Social (Employee remuneration from Social Security General Regime) event has not been generated yet.

Other Terms

eSocial; S-1200; S-1210; 1200; 1210; remuneration; payroll receipt;

Reason and Prerequisites

The system was performing a complete search on T7BREFD_PYRCPT, even though there was no S-1200 saved on T7BREFD_EVENT for the employee.

Solution

The system now identifies that there is no saved S-1200 event for the employee and if doesn't perform a search on T7BREFD_PYRCPT .

The changes described in this SAP Note will be included in an HR Support Package, as indicated in section Support Packages.

The Support Package includes the following objects:

  • Changes in the following includes:
    • RP_PAYBR_INC_PAY_CDEF0
    • RP_PAYBR_INC_PAY_CIMP0
