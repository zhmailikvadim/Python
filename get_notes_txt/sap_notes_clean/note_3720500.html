Symptom

During Route Settlement, the exception BEHAVIOR_ILLEGAL_STMT_IN_CALL is raised.

Other Terms

LMD, Route Stock, Route, Settlement, Route Process Tracking

Reason and Prerequisites

The implementation of IF_EX_LE_SHP_DELIVERY_PROC=>DELIVERY_FINAL_CHECK in class CL_EX_LE_SHP_DELIVERY_PROC attempts to execute a “CALL FUNCTION IN BACKGROUND UNIT” operation, even though the method DELIVERY_FINAL_CHECK is transaction‑classified as “Read”.

Such an operation is not permitted in this transactional phase.

Solution

Implement the attached correction instruction
