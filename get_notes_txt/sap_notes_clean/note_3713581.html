Symptom

The system permits creation of production orders for quantities exceeding the open quantity from the production request.

Environment

SAP Business ByDesign

Reproducing the Issue

  1. Go to production control work center.
  2. Go to production requests view.
  3. Search and select the relevant production request.
  4. Choose Create Order button, select Partial Quantity.
  5. Enter a quantity more than the open quantity in the production request.
  6. A production order is created successfully.

Cause

  • The system provides flexibility to accommodate over production and other operational realities. 
  • This enables companies to produce or confirm quantities exceeding the originally requested amount. 
  • Currently, there is no restriction available to limit this behavior.

Resolution

This is standard system behavior.
