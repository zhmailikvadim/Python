Symptom

In the process "Template for Receiving MR Info Process" (DE_B_MET_TMP_MR_INFO), there is no extension step available for customers to implement their specific logic to send APERAK Z37 when receiving MSCONS with segment DTM+60.

Other Terms

APERAK, Z37, DTM, DE_B_MET_MR_INFO_DSO, MET, MSCONS, meter reading, extension step, segment DTM+60

Reason and Prerequisites

The error occurred because the system lacked an extension step in the process for customers to enhance their specific logic to send APERAK Z37.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

Then activate BC-Set /US4G/3_0_COM_SP06_NOTE_3709186 with T-Code SCPR20, make sure option [Deletion Functionality->Switch On] is ticked.

After implementing this SAP Note, the system provides an extension step 0115 in process DE_B_MET_TMP_MR_INFO.

This allows you to enhance your specific logic to send APERAK Z37 when receiving MSCONS with segment DTM+60.

You need to ask your developers to enhance the check logic in extension step 0115 (process DE_B_MET_TMP_MR_INFO).

There is sample code in Class /US4G/CL_CHK_MET_PROCESS, Method  CHECK_MET_APERAK_SPECIFIC, plesae read before starting your enhancement.

You can verify the fix by checking that extension step 0115 is available in the process and that your custom logic for sending APERAK Z37 runs successfully.
