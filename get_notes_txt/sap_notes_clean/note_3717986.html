Symptom

The application tile for the "Manage Incoming Documents" application is not accessible in the fiori launchpad. 

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Open fiori lauchpad.
  2. Search for the application "Manage Incoming Documents".

Resolution

  • The "Manage Incoming Documents" application is currently only available in the India GSTR3B and GSTR6 Reports.
  • Refer to Manage Incoming Documents | SAP Help Portal for more information.

Keywords

manage incoming documents, not, accessible, available, GSTR1, India, application, fiori launchpad, tile
