Symptom

Standard released matching methods cannot be edited or deleted.

Error message: "Cannot edit or delete predefined objects with IDs starting with 'S'."

Environment

S/4HANA Intercompany Matching and Reconciliation

Reproducing the Issue

  1. Open the application "Define Matching Methods".
  2. Attempt to edit or delete standard released matching methods.
  3. Observe the error message: "Cannot edit or delete predefined objects with IDs starting with 'S'."

Cause

Resolution

  1. Standard released matching methods cannot be edited or deleted due to system design.
  2. To adjust standard released matching methods, copy them to create custom matching methods and modify them according to specific business requirements.
  3. To restrict the display of standard matching methods for end users, use the authorization object F_ICA_MM.
  4. Refer to the SAP Help Portal for detailed information on the authorization object F_ICA_MM:

Authorization Objects | SAP Help Portal
