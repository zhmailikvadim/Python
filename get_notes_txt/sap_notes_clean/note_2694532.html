Symptom

  • You are using Change Request Management (ChaRM) to control your software distribution, you enabled Allowlist Object Check for your Standard Change process;
  • However after you performed a manual check for the Standard Change, expected Allowlist Object conflicts were not reported on the Transport-Related Checks assignment block. 

Other Terms

Allowlist Object Check, Transport-Related Checks, manual check, Standard Change, ChaRM.

Reason and Prerequisites

This is due to program error.

Solution

Implement the attached correction into your SAP Solution Manager system by using the SAP Note Assistant (transaction SNOTE), or import the relevant support package stack.
