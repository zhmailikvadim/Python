Symptom

Issue while opening the Cost Centers Review Booklet (F7763).
Error: “Metadata could not be loaded”

                                [OR]

While performing manual post activities as mentioned in SAP Note 3655718 – Cost Center Review Booklet: Query Changes, the message
“[DEV] SAP_FIN_CO_RB_COST_CENTER (Review Booklet) does not exist” is triggered.

Environment

Controlling (CO)
SAP S/4HANA
SAP S/4HANA Finance
SAP S/4HANA Cloud Private Edition
SAP Fiori for SAP S/4HANA

Reproducing the Issue

  1. Navigate to the Cost Centers Review Booklet (F7763).
  2. Observe the error message “Metadata could not be loaded”.

                                         [OR]

  1. Perform manual post activities as per SAP Note 3655718 – Cost Center Review Booklet: Query Changes.
  2. Observe the error message “[DEV] SAP_FIN_CO_RB_COST_CENTER (Review Booklet) does not exist”.

Cause

Booklet configuration is not active.

Resolution

Activate it using transaction ACTIVATE_REVIEW_BOOKLET (booklet: SAP_FIN_CO_RB_COST_CENTER) in the relevant system and client where the review booklets are intended to be used.

Keywords

Cost Centers Review Booklet, F7763, SAP_FIN_CO_RB_COST_CENTER, Review Booklet, Metadata,  ACTIVATE_REVIEW_BOOKLET
