Symptom

  • Users are unable to locate FI documents related to a clearing document in the 'Manage Journal Entries' Fiori app.
  • The SAP GUI transaction code FB03 enables users to find FI documents related to a clearing document under the Environment > Payment Usage option.

Environment

  • Financial Accounting (FI)
  • SAP S/4HANA Finance
  • SAP S/4HANA
  • SAP Fiori

Reproducing the Issue

  1. Open the 'Manage Journal Entries' Fiori app (F0717)
  2. Try to locate FI documents linked to a clearing document.
  3. Notice that the functionality available in SAP GUI (via transaction code FB03 under Environment > Payment Usage) is not present in the Fiori app.

Cause


The payment usage feature is not supported by the 'Manage Journal Entries' Fiori app.

Resolution

  1. Use the 'Display Line Items in General Ledger' app (F2217) to review cleared items. 
  2. Add the 'Clearing Entry' column in the 'Display Line Items in General' app to access the required information.
