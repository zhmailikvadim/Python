Symptom

Executing a Planning Function in trace mode in RSPLAN: Item "Spreadsheet..." is missing in context menu (right-mouse) of the trace display.

Other Terms

ALV, Excel, workbook, download

Reason and Prerequisites

SAP_BASIS 758 or higher.

Solution

With SAP_BASIS 758 or higher, the behaviour of an ALV table has changed: The display of a context menu item depends now on the display of the corresponding button in the ALVs tool bar. (Basis note 3468780 describes this.)
Before this BW note in RSPLAN (in trace display) the button "Export" was not displayed for the ALV tables. Therefore, with this new Basis Release, the export functionality "Spreadsheet..." was not displayed anymore.

This BW note adds the button "Export" to the tool bar of ALV tables of a trace displayed in RSPLAN. Consequently, the context menu item "Spreadsheet..." is also available again.

Important: 
This SAP Note does not change anything about the operations of BW Planning itself. It only affects the export options of a displayed trace.

To solve the issue please apply the Support Package as mentioned below. In urgent cases you can use the correction instructions.
