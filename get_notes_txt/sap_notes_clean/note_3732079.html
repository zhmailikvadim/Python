Symptom

  • Exporting a workspace from one tenant and importing it into another results in the workspace reflecting an outdated version instead of current content.
  • Importing the same exported workspace into another tenant results in an empty workspace.
  • Import status displays “Completed” while actual content becomes available only later, indicating delayed completion.
  • Images are missing in the imported workspace.
  • No specific error messages or error codes were observed.

Environment

  • SAP Build Work Zone Advanced Edition
  • SAP SuccessFactors Work Zone

Reproducing the Issue

  1. Sign in to the source tenant.
  2. Export the target workspace.
  3. Sign in to the destination tenant.
  4. Import the exported workspace.

Cause

This is a Known Issue.

Resolution

The fix for this issue is planned to be deployed on 1H 2026 release.

For release timelines, please review SAP SuccessFactors Product Release & Road Map Information.
