Symptom

In the process "Creation of a 100% Supplier Assignment to a Feed-In Installation (DSO View)" (DE_B_MPES_OBS_FOR_DS_OBLIGATION_DSO), the supply scenario is not updated in step 1100.

 
This issue is relevant for the market role: DSO.

Other Terms

MPES, DE_B_MPES_OBS_FOR_DS_OBLIGATION_DSO, End of Supply, EOS, Supply Scenario, Feed-In Installation, Supplier Assignment, Installation Type

Reason and Prerequisites

The error occurred because the installation type was missing in the case.

Solution

Implement the corrections of this SAP Note or import the relevant support package.
 
After implementing this SAP Note, the system updates the supply scenario correctly. You can verify the fix by retesting the process and checking that the supply scenario is updated successfully.
