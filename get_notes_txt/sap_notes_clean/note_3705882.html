Symptom

SAP Solution Tools Plug-In (ST-PI) contains a function module that does not perform the necessary authorization checks for authenticated users, allowing sensitive information to be disclosed. This vulnerability has a high impact on confidentiality and does not affect integrity or availability.

Other Terms

Access control, Authorization profile, CVE-2026-24322

Reason and Prerequisites

Missing authorization checks in the SAP Solution Tools Plug-In (ST-PI) allowed authenticated users to access certain functions that exposed sensitive information.

Solution

Following this fix, the affected Function Module no longer discloses sensitive information.
