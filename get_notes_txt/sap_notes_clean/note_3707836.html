Symptom

You are executing either a migration or conversion using Object-Based Transformation (OBT) with SAP Landscape Transformation (SAP LT). You started the preselection 1 scheduler and the tasks do not finish.

Other Terms

SAP Landscape Transformation, SAP LT, S4CORE, On-Premise, OP, Cloud Private Edition, Object-Based Transformation, OBT, Preselection, Preselection Monitor

Reason and Prerequisites

The user that is used to run the scheduler job lacks the right authorization in the remote system. The exception for that case was not handled so far.

Solution

Implement the attached correction using transaction SNOTE or import the relevant support package for OP / Cloud Private Edition into the control system.
