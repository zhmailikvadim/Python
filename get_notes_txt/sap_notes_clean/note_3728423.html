Symptom

Users are unable to change the Opportunity status directly between "Won" and "Lost", as the following error message is displayed: "Invalid lifecycle status {0}".

The system requires the opportunity to first be moved back to an "In Process/Open" phase before it can be changed to the other final status.

Environment

  • SAP Sales Cloud Version 2

Reproducing the Issue

  1. Go to Guided Selling.
  2. Open any Opportunity. 
  3. Set the sales phase to "Won". 
  4. Attempt to change the phase to "Lost".
  5. The error "Invalid lifecycle status {0}" appears.
  6. To successfully change the phase to "Lost", first move the opportunity back to "In Process", add a "Reason for Status" for "Lost", and then set the status to "Lost".

Cause

The behavior is due to the standard system logic that governs the lifecycle status transitions of opportunities. The system enforces specific allowed transitions between statuses to ensure consistent opportunity lifecycle management.

Resolution

Understand the standard lifecycle status transitions for opportunities:

  • From "Won": The opportunity can move to "Won" or "In Process".
  • From "Lost": The opportunity can move to "Lost" or "In Process".
  • From "Stopped": The opportunity can move to "Lost", "In Process", "Won", or "Stopped".
  • From "In Process": The opportunity can move to any lifecycle status.
  • From "Open": The opportunity can move to "Open", "In Process", "Won", or "Lost".
