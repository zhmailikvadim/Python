Symptom

When performing an online search within the description fields (Notifications / Work Orders / Materials), the system only returns results when the entered text matches the exact letter case stored in the database. If the same text is entered using a different uppercase/lowercase combination, Other or no results are returned.

Other Terms

SAP Service and Asset Manager, ONLINE SEARCH, S4MOBADDON-8282

Reason and Prerequisites

This issue occurs in systems running ABAP versions prior to 7.51, as these releases do not support the OpenSQL UPPER function required to perform case-insensitive value comparisons in queries.
