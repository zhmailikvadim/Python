Symptom

VEMP allows you to assign an MRS location to an MBOM node that is not part of the EBOM. The user must be notified of this. The new check has been implemented.

Other Terms

BAEP2, MRS Location, VEMP

Reason and Prerequisites

Program improvement. New functionality.

Solution

Implement the attached correction instructions and the modifications using the SAP modification assistant manually to the standard objects as attached in the documents.
