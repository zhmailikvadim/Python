Symptom

  • In the Variable Pay module, the "show bonus details" icon/button is not displayed in Executive Review.
  • The "show bonus details" icon/button is visible when accessing the same employee data via the worksheet/form view.
  • When changing the view from "Employees I can edit" to "Employees I can view," the "show bonus details" icon/button becomes visible.

Environment

SAP SuccessFactors Variable Pay

Reproducing the Issue

  1. Go to Compensation > Executive Review.
  2. Observe that the "show bonus details" icon/button is missing.
  3. Change the view from "Employees I can edit" to "Employees I can view" and observe that the icon/button becomes visible.

Cause

The Enable Executive Review Edit-in-place setting is not enabled in Actions for all plans > Company Settings > Variable Pay. When this setting is disabled, the details icon only becomes visible in the "Employees I can view" mode.

Resolution

  1. Navigate to Actions for all plans > Company Settings.
  2. Scroll to the Variable Pay section.
  3. Enable (check) "Executive Review Edit-in-place."
  4. Save the changes and refresh Executive Review.
  5. Verify that the "show bonus details" icon/button is now visible also when the view is set to "Employees I can edit."
