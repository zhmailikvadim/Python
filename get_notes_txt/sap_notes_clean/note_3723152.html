Symptom

Report /DBE/ORD_ARC_PRE prepares the archiving of VSS orders by setting the deletion flag. The corresponding action ORD_TO_ARCHIVE checks the status ORD_CLOSE - which is not set (cannot be set) when the order is rejected. It should be possible to archive completed as well as rejected orders for the sake of completeness.

Other Terms

VSS Order, Archiving, /DBE/ORD_ARC_PRE, ORD_TO_ARCHIVE

Reason and Prerequisites

This is due to wrong OE Action customizing

Solution

Process the steps below to adjust customizing settings for OE Action ORD_TO_ARCHIVE:

  1. Open Vehicle Sales and Service customizing by executing transaction /DBE/SPRO or opening SAP Reference IMG node: Logistic Execution-> Vehicle Sales and Service.
  2. Go to Order -> Order Control -> Define Action maintenance in change mode.
  3. Locate ORD_TO_ARCHIVE action and enter the action's details.
  4. Double-click on the Status subnode in Dialog Structure tree (in the left pane).
  5. In line with status ID: ORD_CLOSE, ensure that there is only one column marked: Full (completely processed).
  6. Set checkbox in colum N.Rel. (not relevant) on (marked), so there are now 2 checkboxes marked in that line.
  7. Save the changes. Choose transport request when asked.
