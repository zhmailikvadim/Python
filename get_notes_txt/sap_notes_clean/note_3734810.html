Symptom

In SAP S/4HANA Cloud Public Edition there is a need to update short text and long text of G/L account, but the fields can't be edited.

When trying to delete the G/L account, error "Not possible to delete account since transaction data exists" is raised.

Environment

SAP S/4HANA Cloud Public Edition

Cause

The G/L account’s implementation status was set to "I" instead of "P", which blocked text changes and led to the error.

Resolution

Change the implementation status of the G/L account to "P" and proceed.

Keywords
