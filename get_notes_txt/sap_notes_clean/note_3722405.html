Symptom

In the report result, the YearMonth field is displayed with the proper format(e.g 01.2026), but the F4 help of parameter built on I_YEARMONTH shows the value in the format "202601" instead of "01.2026".

Environment

  • SAP S/4 HANA 
  • SAP BW/4HANA

Reproducing the Issue

  1. Create one CDS query view with similar code below to achieve value help option to get the content from standard CDS view I_YEARMONTH:

    -----------------------------------------------------------
    .....
    define view ZSHEL_MONTH2

    with parameters

    @Consumption.filter: { selectionType: #SINGLE, multipleSelections: false, mandatory: true}

    @Consumption.valueHelpDefinition: [{entity:{ name: 'I_YEARMONTH', element:'YearMonth' }}]

    p_yearmonth: rscalmonth

    .....
    -----------------------------------------------------------
  2. Execute the query, click on F4 help for parameter p_yearmonth.
  3. Observe that the value is displayed in the internal format "202601" instead of external format "01.2026".

Cause

The standard CDS view I_YEARMONTH does not fulfill the formatting requirements. The data element used is VDM_YEARMONTH, and the domain is NUM6, with a data type of NUMC and a length of 6. There is no routine in place to convert the output to the desired format.

Resolution

  1. Create a new dimension CDS view ZI_YEARMONTH on top of I_YEARMONTH with Data Element RSCALMONTH with routine PERI6 as below:

    define view ZI_YEARMONTH as select from I_YearMonth 
    {key cast (YearMonth as rscalmonth) as YearMonth,
     CalendarMonth,
     CalendarYear}

  2. Replace the I_YEARMONTH with this new created CDS view ZI_YEARMONTH as below:

    -----------------------------------------------------------
    define view ZSHEL_MONTH2

    with parameters

    @Consumption.filter: { selectionType: #SINGLE, multipleSelections: false, mandatory: true}

    @Consumption.valueHelpDefinition: [{entity:{ name: 'ZI_YEARMONTH', element:'YearMonth' }}]

    p_yearmonth: rscalmonth
    -----------------------------------------------------------

  3. Execute the query, click on F4 help for parameter p_yearmonth, and get the external format value result.
