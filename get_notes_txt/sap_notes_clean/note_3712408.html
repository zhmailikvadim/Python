Symptom

The copy button is not available in Purchase Request Screen

Environment

SAP Business ByDesign

Reproducing the Issue

  1. Go to Purchase Requests and Order work center.
  2. Go to Purchase Requests View.
  3. Enable All Purchase Requests in the drop-down.
  4. The Copy button is not available.

Cause

System Limitation

Resolution

The function for copying purchase requisitions is currently not available in the system and cannot be added by the settings.

Turn to SAP BYD Influence Forum for new requirement proposals:

https://influence.sap.com/sap/ino/#/campaign/896
