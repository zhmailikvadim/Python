Symptom

  • VL_OPR is displayed as "0" for cfop 1604.
  • The icms value is not correctly populated in the sped-fiscal c190 register.

Environment

  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Execute the program `/tmf/efd_report` for statutory reporting
  2. Generate the sped-fiscal register c190 for cfop 1604
  3. Observe that vl_opr is displayed as "0" and the icms value is missing

Cause

Incorrect COD_SIT under Nota Fiscal data for CFOP 1604, it's required that COD_SIT has been set to "08".

Resolution

  1. Verify that the cod_sit for the nota fiscal is set to "08"
  2. Update the cod_sit field to "08" if it is not already set
  3. Re-run the program `/tmf/efd_report`
