Symptom

An error occurs when activating the ADSO request for cold store via the RSMNG:

Request declaration creation contains invalid source requests

Message no. RSPM116

Neither cleaning up the cold store via RSDA_CLEANUP_DTO_COLDSTORE nor activation via RSDG_ADSO_ACTIVATE fixes the problem.

Environment

SAP BW/4HANA

Reproducing the Issue

  1. Use transaction RSMNG
  2. Activate the ADSO request for cold store 

Cause

The requests are not yet propagated to all targets and only updated to some targets. 
This stops further cold activation. 

Resolution

Propagate the data to all targets and check the activation again.

Keywords

Request declaration creation contains invalid source requests , RSPM116
