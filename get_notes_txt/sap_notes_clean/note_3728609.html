Symptom

An error message occurs when checking connection in the Concur Integration Setup app.

Error message: "Re-execute the System Setup wizard to assign the access token.".

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. In the Concur Integration Setup app, Click System > Check Connection.
  2. Observe that the ping is successful (200 OK), but all API calls return 400 errors with the message: "Re-execute the System Setup wizard to assign the access token.".

Cause

The refresh/request token is missing from the current connection configuration.

Resolution

  1. Make sure the token is available. If you are unsure, you can generate a new one first.
  2. Enter the refresh/request token in the current connection configuration and check the connection again.
  3. If the issue persists after entering the token, delete the current connection and create a new one instead of modifying the existing connection.
