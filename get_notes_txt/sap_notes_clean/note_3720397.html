Symptom

  • Users who leave involuntarily are unable to access the alumni portal due to the inability to complete the Employee Data Review step.
  • The Employee Data Review step is a mandatory prerequisite for enabling alumni access.
  • There is no system-supported way for involuntary leavers to complete this step themselves.

Environment

SAP SuccessFactors Onboarding

Reproducing the Issue

  1. Configure Offboarding 2.0 with the Employee Data Review step as part of the process variant.
  2. Terminate an employee involuntarily.
  3. Attempt to provide alumni access to the terminated employee.
  4. Observe that the employee cannot complete the Employee Data Review step and alumni access cannot be granted.

Cause

The Employee Data Review step is a mandatory prerequisite for enabling alumni access. In cases of involuntary termination, the user loses system access and cannot complete this step, resulting in the inability to grant alumni access.

Resolution

  1. Complete the Employee Data Review task via BPE (User Tasks) on behalf of the involuntarily terminated user.
  2. After the Employee Data Review step is completed, the offboarding process can continue and Alumni Experience access can be enabled.
  3. Update the Offboarding configuration to align with the guidelines provided in the Enabling Alumni Access implementation guide.
  4. If this scenario is common in the organization, consider submitting an enhancement request to address this limitation in future releases.
