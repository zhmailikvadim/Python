Symptom

  • System monitoring for web dispatchers was configured with the “Web Dispatcher Certificates Expired” alert and SSL client/server metrics, but monitoring does not work.
  • Web dispatcher certificates metrics shows grey
  • System monitoring UI data collection error shows " No values reported: data provider GetAlertTree , data provider type PUSH_GENRIC "

Environment

  • SAP Solution Manager 7.2
  • System Monitoring

Reproducing the Issue

  1. Set up system monitoring for web dispatchers.
  2. Activate the “Web Dispatcher Certificates Expired” alert with SSL client and SSL server metrics.
  3. Assign the technical systems to the monitoring.
  4. Navigate to system monitoring UI in Workcenter
  5. Select the affected system and check the metrics

Resolution

  1. Allow GetAlertTree to be called by setting the instance/profile parameter
  2. Maintain the parameter in the Default Profile file service/protectedwebmethods = SDEFAULT -GetAlertTree
  3. There after giving the parameter restart the service as stated in the following KBA and check if the issue still persists or not.
    0003227051 - Gray or red Metric for SAP Web dispatcher SSL Client Certificate Expiry metric in System monitoring
