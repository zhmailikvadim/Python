Symptom

The HTTP Allowlist Check ignores the settings of the No-Log-List and still writes log entries to the database.

Other Terms

HTTP Allowlist, ucon_chw, No-Log-List

Solution

The behavior of the No-LOg_list entries in transaction UCON_CHW has been changed and the allowlist check no longer writes additional log entries for checks covered by the No-Log-List entries.

  • SAP S/4HANA 2025 / SAP ABAP Basis 816 SP02
  • SAP S/4HANA 2022 / SAP ABAP Basis 757 SP08
  • SAP S/4HANA 2021 / SAP ABAP Basis 756 SP10
  • SAP S/4HANA 2020 / SAP ABAP Basis 755 SP12 

for previous service packs implement the attached correction instructions.

For release SAP S/4HANA 2023 / SAP ABAP Basis 758 implement SAP Note 3642142.
