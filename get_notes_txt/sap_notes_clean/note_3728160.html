Symptom

User opens an object say "Specification" from PLM Web user interface (PLMWUI) and navigates to "Documents" section. If there are more than one document selected and "Add attachment" is used, then both the Document info Records (DIR) get the same file with same IDs. This causes an issue when user wants to delete the file from one of the document and throws an error "Deletion not possible, original is being used in other documents".

Other Terms

 DIR_DO_LINK, add attachment, LOIO, used in other documents, 26 293.

Reason and Prerequisites

 Program error.

Solution

Implement the instructions attached.

Note: Since the underlying framework (attachment service) integrated with PLMWUI can handle only one upload at a time, the "Add Attachment" click will indicate this when multiple documents are selected.
