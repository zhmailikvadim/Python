Symptom

  • Files fail to load or preview in SAP Build Work Zone, the system displays the following messages instead of a preview
  • Error seen : “This document contains a cross-site script (XSS).  A document preview cannot be provided.”  

Environment

SAP Build Work Zone

Reproducing the Issue

  1. Upload the XSS file into SAP Build Work Zone.
  2. Attempt to preview the file in Work Zone.
  3. Observe the error messages:
  • “This document contains a cross-site script (XSS). A document preview cannot be provided.”
  • “A preview is currently unavailable for this file. Please download it to view.”

Cause

This is caused by a general security restriction applied to both SAP Jam and SAP Build Work Zone to disable PDF preview containing XSS content.

Resolution

  • The correct behavior is to disable PDF preview containing XSS content to enhance overall product safety.
  • When you try to preview such document it is expected to see the message: "This document contains a cross-site script (XSS). A document preview cannot be provided.”
