Symptom

You are executing an OData V4 request with $search containing only whitespaces. The request will result in a dump

Other Terms

OData, OData V4, SAP Gateway, $search

Reason and Prerequisites

Error when parsing the $search expression. This happens because the parser/tokenizer cannot match the searchWord, thus creating an empty searchNode type W (word) without a search text. Later in the request the System will dump. Hindering the creation of empty searchWord tokens solves this issue.
