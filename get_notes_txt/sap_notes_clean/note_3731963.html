Symptom

In some cases the results row may not be required in the report.

Environment

 SAP Business ByDesign 

Reproducing the Issue

  1. Go to Business Analytics Work center.
  2. Go to Design reports sub view.
  3. Search and open required report.
  4. Enter the selection parameters and click on apply.
  5. The system will display result row for each attribute in the report.

Cause

This behavior is as per report setting.

Resolution

  1. Navigate to Business Analytics work center.
  2. Select the Design Reports sub view.
  3. Use the search field to locate the report you need and open it.
  4. Enter Selection Criteria and Click Apply to display the report results.
  5. Once the report results are displayed, go to the View tab.
  6. Modify the row field, Click on the three dots (⋮) next to field name and Select Characteristics Settings.
  7. In the popup window, go to the Result Row dropdown and select “Do Not Show”.
    Click OK to apply the change.
  8. You will now see that the result rows have been removed for the selected field.
