Symptom

It is extracted using a DataSource (RSDS) with SDA - the indicator "use for packaging and partitioning" is set for some fields. The DTP on top has defined some fields in the "grouped extraction" section.

As soon as the DTP uses "grouped extraction", all other fields defined in RSDS must be filled with the following: indicator to ensure that the packages are created correctly.

After you implement the corrections, the fields defined in RSDS with "use for packaging and partitioning" = X are ignored.
