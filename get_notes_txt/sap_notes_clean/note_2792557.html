Symptom

You execute the Calculo nómina y vacaciones  ( CO ) (HCOCALV0) report for an employee that has a ID personal ﻿(IT0185) record with subtype Permiso Espec. Permanencia (12).

The following warning generates in the payroll log, on payroll function COEVS:

"Tipo y Número de Identificación, no definido en Inf. 0185"

Other Terms

&TECHNICAL (further information see SAP Note 2189265)

PC00_M38_CALV; COEVS; IT0185

Reason and Prerequisites

The following Support Package level is the minimum prerequisite:

ReleaseSupport Package
600 HR-CEECCO SP F3
604 HR-CEECCO SP B9
608

SAP_HRCCO SP 47

(For more information, click "HR Support Package Schedule" in support.sap.com/maintenance-schedule.)

Solution

Version Date Comments
2 May 20, 2019 Released for Customers.

From now on, COEVS payroll function will use table T7CO85 to read document type IDs.

As general rule, SAP recommends that you install a solution by applying a Support Package. However, if you need to install the solution earlier, proceed as follows:

    1. Use Note Assistant (SNOTE) to implement the correction instructions.

You can find more information about the Note Assistant in SAP Support Portal, under support.sap.com/kb-incidents/notes/assistant.html.
