Symptom

When opening app "Account Payable Overview" or "Account Receivable Overview", not all the cards are displayed

Environment

Financial Accounting (FI)
SAP Fiori for SAP ERP
SAP S/4HANA
SAP Fiori for SAP S/4HANA Finance
SAP S/4HANA Cloud Private Edition

Reproducing the Issue

1. Open app "Account Payable Overview", or "Account Receivable Overview"
2. Not all the cards are displayed

Resolution

1. Open app "Account Payable Overview"
2. In user settings, click menu "Manage Cards"

3. Select the target cards to be displayed

Keywords

"Account Payable Overview","Account Receivable Overview",  Missing, Cards, "Manage Cards"
