Symptom

Framework update (2211‑jdk21.1 and later) upgrades the platform to JDK 21 and Spring 6, establishing SapMachine JDK 21 as the runtime baseline for all subsequent 2211‑jdk21.x releases.

Environment

SAP Commerce Cloud

Reproducing the Issue

SAP Commerce Cloud 2211-jdk21.x releases may include updated:

SapMachine (JDK 21) patch levels

Security patches

Resolution

Kindly refer to the official SAP documentation regarding SapMachine (JDK 21) and version compatibility for SAP Commerce Cloud 2211:

Install the Basic Prerequisites

Download the latest Data Hub ZIP file containing the DH web application. See the Download section of this guide for further information.

  1. Install SapMachine JDK 17 from https://sap.github.io/SapMachine/. Data Hub supports 64-bit Java. Based on the version of Data Hub that you’ve downloaded, install one of the following from https://sap.github.io/SapMachine/:

    • SapMachine JDK 17 (for Data Hub version 2211.x)

    • SapMachine JDK 21 (for Data Hub version 2211.jdk21.x)

  2. Install Apache Tomcat 9.0.x from https://tomcat.apache.org/download-90.cgi.

    Follow the instructions in the Apache Tomcat documentation to install.

  3. Configure Tomcat to add SSL authentication. Follow the instructions in the Tomcat SSL/TLS Configuration How-To guide.
    Tip

    The folders you create in the next steps are required for the Hello World tutorials and for using Data Hub.

  4. Create an /opt/datahub folder.
  5. In the datahub folder, create the following new folders:
    1. config, used for the files local.properties and datahub.encryption.key.txt
    2. extensions
  6. Create an XML file.
    Remember

    All of the Data Hub documentation is based on the datahub-webapp.xml file you create in the following steps.

  7. Open a new file in a text editor and copy the following into the file:
     
    <?xml version='1.0' encoding='utf-8'?>
<!--

<Context antiResourceLocking="true"
		docBase="{full path to your Data Hub installation}/hybris/bin/ext-integration/datahub/web-app/datahub-webapp-x.x.x.x-RCx.war"
		reloadable="true">

    <Resources className="org.apache.catalina.webresources.StandardRoot" >
      <PostResources className="org.apache.catalina.webresources.DirResourceSet"
                    base="/opt/datahub/config" internalPath="/" webAppMount="/WEB-INF/classes" />
      <PostResources className="org.apache.catalina.webresources.DirResourceSet"
                    base="/opt/datahub/extensions" webAppMount="/WEB-INF/lib" />
    </Resources>

</Context>
     
     
  8. Edit the <docBase> parameter to reflect the full path to your SAP Commerce Cloud installation.
     
  9. Save the file as datahub-webapp.xml into the <TOMCAT_HOME>/conf/Catalina/localhost folder. Note: the <TOMCAT_HOME>/conf/Catalina/localhost does not exist until Tomcat is run.
  10. Install the cURL command-line tool.

    From https://curl.haxx.se/, download and install the cURL software that is appropriate for your operating system.
