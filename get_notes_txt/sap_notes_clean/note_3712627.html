Symptom

Email notifications generated by the custom workflow display a link instead of the expected token values for users.

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

  • SAP SuccessFactors HCM Suite
  • SAP SuccessFactors Employee Central
    • Email notification - Place Holders

Cause

The Document Generation Template source code created automatically "http://[View Link]//", causing the issue.

Resolution

Remove the "http://[View Link]//" from the Document Generation Template configuration.

Steps:

  1. Access: Manage Document Template
  2. Search for: Document Generation Template
  3. Select the desired template
  4. Choose Actions -> Make Correction
  5. In the Template Content section, click on the three dots (⋯)
  6. Select Show Source Code
  7. Update the content as needed
