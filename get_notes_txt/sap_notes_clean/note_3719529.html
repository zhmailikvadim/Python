Symptom

Import status is displayed as in process for approximately three days.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Download the template from the manage tax rates app.
  2. Fill in the required data in the template.
  3. Import the completed file using the manage tax rates app with 5000+ records

Cause

Importing more than 5000 records results in extended processing time due to the large import size.

Resolution

To avoid prolonged background job processing, restrict the import to a maximum of 1,000 records per batch. This ensures efficient system memory utilization and prevents performance issues by limiting the number of records processed at a time.
