Symptom

It is possible to observe different data numbers in AI-assisted Report Builder report compared to Trends report in the same time period and observed metrics.

Environment

SAP Emarsys

Cause

The number difference between "AI-assisted Report Builder" and "Trends" report features results, happens due to the following:

Although the source of the data for both reports is the same, there are slight differences in the implementation (such as the time-zone and the freshness of data), that could lead to minor discrepancies between them. In which, the AI-assisted Report Builder feature uses UTC time and Trend Reporting displays the time zone of the administrator.

Therefore, both will provide you stable and reliable results, but as they are different features and the "AI-assisted Report Builder" feature is newly implemented, it is possible to notice a small variance.

Resolution

This is expected system behaviour.

SAP is constantly thriving to improve the product and hearing customer's suggestions is the best way forward. The product managers regularly check the Influence page to hear what changes users would like to see in place and check those for feasibility. Idea requests can also be voted for by other customers.

To suggest a change to this behaviour, please submit the improvement/feature idea at the SAP Emarsys Influence page.

Please note that it is not possible for SAP Support Team to provide additional information regarding the availability of the request in a future release.
