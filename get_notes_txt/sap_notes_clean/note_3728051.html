Symptom

You want to add own fields to the operation table of Manage Schedule App (under Resource Scheduling), when using the option 'Accelerate Data Access' (or 'Use Optimized Data Access').

Other Terms

RSH, Manage Schedule App, accelerate data access, F3326

Reason and Prerequisites

Due to the original enhancement category, this was not possible

Solution

Please apply the attached correction, which will enable the enhancement of the operation structure.

Make sure you have installed note 3356080 as pre-requisite.
