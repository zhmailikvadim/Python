Symptom

On the Member page of a BPC Embedded model, the default new row for adding a new member does not load. It happens when the dimension has fewer than 1,000 members.

Reason and Prerequisites

This is a program error.

Solution

For SAP Business Planning and Consolidation 2021, version for SAP BW/4HANA , upgrade to(component BPC4HANA300) Support Package 14.
