Symptom

You use a Unified Rendering-based application with fields in a table. For these fields a HelpButton will be visible once the field is either hovered or focused. The styles which visualize the hover state, like for example borders, may remain visible althrough the user is no longer interacting with the field.

Other Terms

Web Dynpro Java, Web Dynpro ABAP, WDJ, WDA, SAP GUI for HTML, UnifiedRendering, Unified Rendering, Lightspeed, hover, focus, styles

Reason and Prerequisites

Using WD ABAP starting from 7.02, WD JAVA starting from 7.30 and SAP GUI for HTML all kernel releases.

Solution

As part of regular intense quality- and patch creation tasks, all Unified Rendering fixes will be collected and shipped about every four weeks.

The fix will be shipped with the next "Cumulative patch for Unified Rendering 2205".

Once available it will be delivered for WD ABAP starting from 7.02, WD JAVA starting from 7.30 and SAP GUI for HTML all kernel releases.

Be aware this SAP note can’t be implemented.

Check SAP note 2500800 for the availability of the corresponding framework patch (delivery dates) and information about the patch process.

See also section "References".
