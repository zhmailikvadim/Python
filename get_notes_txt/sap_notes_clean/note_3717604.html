Symptom

You are using CIF integration to SAP SCM (APO) and the MRP DS scenario is turned on for your plants. In the meanwhile you also activate the Real-Time Integration (RTI) to the SAP IBP system for the same location products.

For some SCM systems the transactional data for MRP DS locations are not integrated after the integration to SAP IBP was set up.

Other Terms

GET_MRPDS_CUST_ENTRY, IMOD_DESTINATIONS, multi CIF

Reason and Prerequisites

The MRP DS customizing was read for the first destination, and if the first was the SAP IBP, then the customizing read was unsuccessful.

Solution

Please implement the attached source code corrections. It will introduce filtering for destinations, and only APO destinations will be taken into account when the MRP DS customizing is read.
