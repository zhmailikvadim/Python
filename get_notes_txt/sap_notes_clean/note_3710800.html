Symptom

This SAP Note gives a summary of legal changes, corrections and enhancements that are available with the country-specific legal change packages below for HCM Spain (components PY-ES and PA-PA-ES):

Release Support Package
S4HCM 101 SAPK-10131INS4HCMCES
S4HCM 100 SAPK-10044INS4HCMCES
SAP_HR 6.08 SAPK-608F5INSAPHRCES
SAP_HR 6.04 SAPK-604M7INSAPHRCES

Other Terms

CLC; HCM; Legal Change; SAP_HRCES; S4HCMCES; Support Package

Reason and Prerequisites

Prerequisite is the installation of the HR support package (or the corresponding available CLC) that is relevant for your release and listed below:

Release Support Package
101 HR SP 30
100 HR SP 43
6.08 HR SP F4
6.04 HR SP M6

This package includes all the content delivered for the components mentioned above since SAP Note 3702869 - CLC package for SAP_HRCES and S4HCMCES (HCM Spain) for January 2026, made available on January 22, 2025. This CLC will be available according to the planning published on the SAP Schedules for Maintenance Deliveries. The planned delivery date is set to February 13, 2026.

Solution

Please see the related SAP Notes for a detailed description of the main legal changes, corrections and enhancements that are listed below. Pay attention also to additional activities that might be necessary after the installation.
