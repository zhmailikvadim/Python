Symptom

The sum line of log in the transaction /SAPAPO/BOD_MAINT is not calculated by default.

Other Terms

Transaction /SAPAPO/BOD_MAINT

Reason and Prerequisites

The log messages will be displayed and calculated immediately if the component 'SHOW_ALL' is filled with 'X' in the display profile.

In note 3711082, function module 'BAL_DSP_PROFILE_DETLEVEL_GET' is replaced with 'BAL_DSP_PROFILE_SINGLE_LOG_GET'.

However, this leads to log messages only being grouped by message problem class in overview.

This note is to revert the changes in note 3711082 to get the log messages organized per BOD.

Solution

Replace function module 'BAL_DSP_PROFILE_SINGLE_LOG_GET' with 'BAL_DSP_PROFILE_DETLEVEL_GET'.

Then fill the component 'SHOW_ALL' with 'X'.
