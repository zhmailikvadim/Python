Symptom

The SAP Note does not correct any symptoms;

SAP Note allows create development package PAOC_PAY_RU_CALC which would be used for other development objects

Other Terms

DEVC PAOC_PAY_RU_CALC, PAOC_PAY_RU

Reason and Prerequisites

This SAP Note is relevant only for Russia

Solution

As a general rule, SAP recommends that you install a solution by applying a Support Package. However, if you need to install a solution earlier, perform following steps to implement the DEVC creation:

  1. Log on in the system on English language;
  2. Use the Note Assistant to implement the correction instruction of this SAP Note. You can find more information about the SAP Note Assistant in SAP Support Portal, under SAP Note Assistant.
  3. Call transaction SA38 and execute report RPUPAYRU_NOTE_2555992_EA_HR in a client of your development system with standard SAP objects changes allowed;
  4. Follow on-screen instructions on how to implement changes in your system. You might encounter various errors during the first update run. Simply run again in update mode. The second run should end without any errors.
  5. Make sure that all objects, which created by the program, are active. You can check the list of inactive objects with transaction WORKINGAREA.
