Symptom

The Period Closure Run is not functioning as expected. The last successful Period Closure Date remains unchanged, and subsequent runs are not updating the system.

Environment

SAP Business ByDesign

Reproducing the Issue

  1. Go to Time Administration Work Center.
  2. Go to Regular Tasks View.
  3. Under Time Valuations, select Administer Day Closure Runs.
  4. Check for the SAP_COMMON.
  5. Select View Jobs.
  6. Check If the Administer Day Closer Run was scheduled.

Cause

The daily Day Closure Run (SAP_COMMON) was not scheduled.

Resolution

  1. Go to Time Administration Work Center.
  2. Go to Regular Tasks View.
  3. Open Administer Day Closure Runs.
  4. Open the SAP_COMMON Day Closure Run configuration.
  5. Maintain and activate an appropriate daily schedule according to business requirements.
  6. Save the schedule and ensure it is active.
  7. Monitor subsequent Day Closure Runs and verify that Period Closure Dates update accordingly.
  8. Confirm that time account balances, overtime, and extra pay calculations resume as expected.
