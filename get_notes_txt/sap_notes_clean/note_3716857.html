Symptom

This SAP Note delivers Display and Edit options for Official Document Numbering ODN based on Additional Fiscal Data at header level, through transactions FB02 and FB03.

FI records with ODN Type U (End User Input) have been posted via the ODN existing processes, such as FI_SUPPLIER_INV, FI_CUSTOMER_INV, MM_SUPPLIER_INV and SD_BILLING_HEADER. It is possible to display and change ODN values via FB03/FB02 for these FI documents.

Other Terms

ADH, FI ODN, FIN ODN, FI-ODN , ODN, Official Document Numbering, EI_FI_ODN_FIN_AFD_ENH, CL_FI_ODN_IM_BADI_AFD_INIT, CL_FI_ODN_IM_BADI_AFD_OKCODE, Reference, Additional Fiscal Data

Reason and Prerequisites

New functionality.

Before implementing this SAP Note, implement SAP Note 3674547 and 3726691.
