Symptom

You use the electronic bank statement and use a posting rule with clearing. 
If the clearing was not successful, a payment advice note is generated.

You want to deactivate the creation of the payment advice note.

Other Terms

RFEBBU00, AKONTO, AVIS, FEBA, FEBAN, deactivate payment advice notes

Reason and Prerequisites

For SAP_APPL and SAP_FIN 617, the corrections from SAP Note 1910415 must be contained.

Solution

You can use the customizable message "FB211" to set the payment advice note creation:
To prevent payment advice note creation, you must perform the following steps:

  • Call transaction "SM30".
  • In the "Table/View" field, enter V_T100C and choose "Maintain".
  • Enter "FB" as the application area.
  • Create a new entry:

           MsgNo: 211
Message text: Suppress payment advice note creation in bank statement
User name: (can be restricted to users, if initial, the entry applies to all users)
Dialog: S
Batch. S

  • Click Save.
