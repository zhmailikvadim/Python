Symptom

You post accruals with the reset procedure. The exchange rate in the "Exchange Rate for Translation of Settlement Currency to LC" field (KURS2) is always the exchange rate from the key date of the accrual/deferral.

This is incorrect if

  • is accrued/deferred with the option "Use Exchange Rate from Flows to be Accrued/Deferred" (SAP Note 1715601)
  • The exchange rate was changed using the BAdI FVD_LOAN_ACCRUAL with the method CHANGE_LC_AMOUNTS (SAP Note 2407625).
