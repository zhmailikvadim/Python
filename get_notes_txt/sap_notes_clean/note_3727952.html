Symptom

This Note addresses 2 issues as following:

1.Control box change process(DE_B_IDEV_CTB_CHANGE) for control resource(CR) can't continue even relevent control box change process on Melo is finished

2.No seperate deadline step and period for control resource to wait for the closeure of control box change process on Melo

Other Terms

DE_B_IDEV_CTB_CHANGE, Control box, control resource,  /US4G/CL_PST_CTB_INST_MELO,  /US4G/CL_PST_IM_INFORM_CR_INST,  /US4G/CL_CHK_IM_PROCESS,  CHK_CTB_LEADING_POD,  CHK_IF_CTB_RM_TRIGGER

Reason and Prerequisites

Enhancement

Solution

Please implement the correction instruction in the SAP Note or import the higher version of Support Package.

The main changes in process are the following, please note you can redefine the deadline period in step 0550 by your own customer extension

  • New step 2100 is to inform CR that CTB installation completed on Melo.
  • New step 0620 is to receive information from Melo process.
  • Adjust deadline period for Waiting Control Box Install on Melo to 1 day.
  • New Deadline step 0550 for waiting Parameterzation Process Completed on Melo is 1 hour.
