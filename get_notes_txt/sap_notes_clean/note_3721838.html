Symptom

  • When accessing a PS Project via navigation from a Portfolio Item, only the project definition is displayed.
  • The WBS structure cannot be expanded (no WBS elements visible).
  • The issue may not affect all users.
  • The issue does not occur when the same users access the PS Project directly in Project Builder transaction CJ20N.

Environment

  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition 

Reproducing the Issue

  1. The affected user accesses a Portfolio Item in Portfolio Management.
  2. Click on the PS Project link.
  3. Project Builder transaction opens.
  4. Only the project definition is shown – WBS structure cannot be expanded.

Cause

  • Issue with navigation to transaction OLR3_CJ20N as described in SAP Note 3504657.

Resolution

  1. The affected user can manually display all objects by navigating to "Settings" → "Options" in Project Builder transaction and setting the flags to display the relevant project elements.
  2. Implement SAP Note 3504657 to resolve the issue, so the flags to display the project elements are set by default.
