Symptom

When trying to log into SAP Information Steward web application with a different CMS system or cluster, an error message "Information Steward web application is only configured for the system: <SYSTEM/CLUSTER> (UIM-10038)" is displayed.

Environment

  • SAP Information Steward 4.3 all versions
  • SAP Information Steward 2025 all versions

Reproducing the Issue

  1. Log into SAP Information Steward web application with "CMS1" as system. The user can successfully login to the Information Steward web application.
  2. Logout the current user associated with CMS1.
  3. On the logon form, use "CMS2" as system and try to login with the user account associated with CMS2.
  4. The error message "Information Steward web application is only configured for the system: CMS1 (UIM-10038)" is displayed on the page.

Cause

This is by design from how the architecture of SAP Information Steward web application work. One instance of SAP Information Steward web application can only be associated with one specific CMS system or cluster until the web application server is restarted.

Resolution

  • If user wants to permanently switch to a different CMS system/cluster, restart web application server and login with the new CMS system/cluster and associated user.
  • If user wants to use multiple CMS systems/clusters, user should deploy the web application to multiple instances of web application servers in order to support this use case.
