Symptom

When maintaining site URL in Career site builder, error message displayed: "The URL you specified is already in use. Choose a different, unique URL for the Site URL field and try again."

Environment

SAP SuccessFactors Recruiting Marketing

Reproducing the Issue

  1. Navigate to the Site Configuration Page in Career Site Builder.
  2. Attempt to save a Site URL.
  3. Observe the error message: "The URL you specified is already in use. Choose a different, unique URL for the Site URL field and try again."

Cause

The site address attempted to be registered is already taken in another career-site system.

Resolution

  1. Remove the URL from the existing site.
  2. Update the URL for the intended site.

Keywords

Recruiting Marketing, career site builder, CSB, site URL, error, save, update, site configuration error, conflict, error message, other RMK system, already in use
