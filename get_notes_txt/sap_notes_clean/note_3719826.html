Symptom

Error message informs that the field "IDNFBRJ_1BNFSE_DOCREF" is unknown.
This may happen after implementing SAP Notes from Brazil Localization.

Environment

SAP ERP
SAP S4CORE

Cause

Missing manual step activity proper execution.

Resolution

To resolve, perform manual step 3.1 ('Change NOTAFISCAL') from the 'Post-implementation steps for SAP Note 3666810.pdf' file found in SAP Note 3666810.

If this above was already done and the issue remains, then it is very likely that the previous generations of SCDO were done using Class instead of Function Module.  

So it now requires the re-generating the object "NOTAFISCAL" using Function Module. 

You can check that by accessing the Generation Information of this object:

  1.  Transaction SCDO
  2.  Object: NOTAFISCAL
  3.  Click "Information" Button

It may be possible to verify that the object CL_NOTAFISCAL_CHDO is listed within Generation Information. 

Please follow the steps below to fix the generated information of change object NOTAFISCAL:   

  1. Access transaction code SCDO.  
  2. Enter object "NOTAFISCAL" (without quotes) and select in the menu the option Change Document Object -> Delete.   
  3. On the popup, select "Generated So"

After that, to re-generate the object, please follow the steps from Section 3.1 Change NOTAFISCAL in "Post-implementation steps for SAP Note 3666810.pdf"
