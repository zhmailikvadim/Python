Symptom

  • After assigning the Controller to an FFID and setting 'Notification by' as Email, workflow is not triggering and no email is sent.
  • Additionally, on GRAC_FFSESSION tcode the message "Controllers delivery option is set to email or log display" is shown.

Environment

  • SAP Access Control 12.0
  • SAP Access Control 12.0 for SAP S/4HANA

Reproducing the Issue

  1. Open NWBC
  2. Click on Setup -> Controller under "Emergency Access Maintenance"
  3. Click on an FFID line
  4. Assign or Reassign
  5. Edit fields and set the Notification by = Email
  6. Save

Cause

Delivery option for the controller must be 'Workflow'. If Notification by field is set as Email' or 'Log display', workflow is not sent.

Resolution

Change the 'Notification by' to 'Workflow'
