Symptom

Normally there is one old gateway trace file named dev_rd.old in ST11. You may want to increase the number of old trace files, such as dev_rd.old.1, dev_rd.old.2, etc. 

Environment

  • Gateway

Resolution

This setting cannot be changed. The number of old files is hard-coded and is always set to 1. 

Keywords

gateway, dev_rd, trace generation, dev_rd.old, trace files, backup file, ST11
