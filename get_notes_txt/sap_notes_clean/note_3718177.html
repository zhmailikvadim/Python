Symptom

  • Stories using linked models show a freezing/loading screen when a dimension is added to the widgets.
  • The issue also appears when some rows are removed from tables.
  • The issue is observed in both new tables and charts.

Environment

SAP Analytics Cloud

Reproducing the Issue

  1. Access SAP Analytics Cloud.
  2. Navigate to the story.
  3. Open the right panel, click on the table, go to builder, and in columns, click "Add Dimension" > "xxx". The table keeps loading indefinitely.
  4. Alternatively, open the right panel, click on the table, go to builder, and remove a few rows (usually after 6 or 7 rows). The table keeps loading indefinitely.
  5. The same behavior is observed in the chart created below the main table when adding dimension > "xxx".

Resolution

The fix has been deployed with wave 2026.02.10.
