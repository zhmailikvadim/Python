Symptom

  • What customizations are available in the Greeting section in the new Home Page?
  • Can I configure the color and format of the new Greeting section?

Environment

SAP SuccessFactors HCM

Cause

Resolution

The Welcome Banner has been now replaced by a Greeting section in the Latest Home Page experience. As of now (2511 release), the customizations and limitations of this section are below:

  • Changing the name: Now, users are welcomed by their first name in the Greeting section. They can decide with they want to enable it or not, by following KBA 3683301 - Latest Homepage Experience - There is no name showing in the Greetings in the Homepage banner - SAP for Me.
  • Changing the Welcome text: This is still possible to be done, by changing the label via Manage Languages. See KBA 3212276 - How to customize or remove greetings on the Latest Homepage banner - SAP for Me.
  • Changing the color/adding image in the Greeting section: As of now, it is not possible to configure the color and/or add a background image in the banner. Our Product team is planning to deliver those enhancements in the upcoming release (1H26). 
  • Featured Cards: Now, cards are appearing in the Greeting section, for information that does not require action, but is a "Good to Know". As of now, it is possible to personalize only the image of the card, but not the text. Also, they appear in the home page according to the availability of the content, and it cannot be controlled.
