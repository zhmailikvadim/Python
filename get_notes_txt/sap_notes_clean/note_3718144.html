Symptom

Defer aborts calculation with error "No Cashflow found.."

Reason and Prerequisites

When cash flow versions in other cashflow categories are delivered (e.g. 251, 252) and the trigger service recognizes a change, defer does a time-effect and a EIR recalc. This is not correct, only the time-effect before change is necessary. The cashflow of other categories cannot be found and no operation on it is possible, thus the error message.

Solution

Implement the following changes
