Symptom

  • Understanding the process of transferring materials from ERP to EWM is essential for maintaining accurate data synchronization.
  • The ERP system serves as the primary system, and transferring the material master data to EWM ensures efficient warehouse management.

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Environment

  • SAP Extended Warehouse Management (SCM-EWM)
  • SAP Extended Warehouse Management Master Data (SCM-EWM-MD)

Reproducing the Issue

Attempt to transfer material master data from ERP to EWM using the integration model.

Resolution

Setup an integration model for a material data transfer at ERP (ECC) Side:

  1. Use transaction CFM1 to create the integration model. Enter the model name, logical system, and APO application. Select the flag for materials and input the material name.
  2. Click on Generate IM. A confirmation message will appear. 
  3. Use transaction CFM2 to activate the integration model. 
  4. Verify the details of the integration model. 
  5. Change the status of the model and click on start to initiate the data transfer. 
  6.  Confirm that the products are reflected in the EWM system's /N/SCWM/MAT1 transaction and the EWM Product Master tables.
