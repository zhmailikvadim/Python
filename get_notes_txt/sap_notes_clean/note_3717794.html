Symptom

In SAP Business Network that is integrated with SAP ERP or SAP S/4HANA using SAP Integration Suite, managed gateway for spend management and SAP Business Network, When an Advanced Ship Notice (ASN) with shipment type Planned is sent from SAP Business Network to SAP ERP or SAP S/4HANA, the ASN is processed successfully; however, the External Reference Number from the ASN is not updated in the Delivery Note in ERP causing failure in Component Acknowledgement.

Cause

The error occurred due to erroneous code.

Resolution

Now, the code has been enhanced to resolve the problem. For more information, please see SAP NOTE 3710536.
