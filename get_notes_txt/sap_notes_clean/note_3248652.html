Symptom

It is not possible to enter a release for a financing with two loans each linked to a building savings contract and distribution algorithm "Building Savings Contract". The system issues the error message "BSG [...] is not assigned".

Reason and Prerequisites

For the above The distribution algorithm must specify a loan number when you enter the release, otherwise the financing itself is interpreted as the reference object of the release by default. However, the error message issued does not indicate this.

Solution

The checks of the call are changed so that the system first checks for the existence of the constellation described above and for the specification of a loan number. If the constellation is given and no loan number has been entered, the system issues the error message "Call with reference object FIN not permitted during distribution via BSG".
