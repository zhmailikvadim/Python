Symptom

The IR56F XML file created by T-code PC00_M27_CTXF shows sheet number 000000, which will lead to error when uploading to tax portal. "Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Environment

  • Human Capital Management (HCM)
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for HANA
  • SAP S/4HANA
  • SAP SuccessFactors Employee Central Payroll

Reproducing the Issue

  1. Run T-code PC00_M27_CTXF or program HHKCTXF0 to generate the IR56F XML file.
  2. Find that the sheet number is 000000.

Cause

Missing configuration in view V_T5F99K2.

Resolution

  1. Open view V_T5F99K2 and input HCM Localization 27 via T-code SM30.
  2. Insert an entry as below.

Keywords
