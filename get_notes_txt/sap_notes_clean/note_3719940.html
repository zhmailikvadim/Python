Symptom

  • I would like to upload Statement of Work with clauses to be included in that upload
  • When using SOW XML Upload, getting error message: This upload cannot be used with SOW Bids

Reproducing the Issue

  1. Attempt to upload a file using the SOW XML upload connector in the test environment
  2. Include clauses in the SOW XML Upload file
  3. Execute the upload process
  4. Observe the error message: This upload cannot be used with SOW Bids

Cause

The SOW Type settings for the template used in the upload were configured with Use to create SOW Bid set to YES, which caused the error

Resolution

  1. Verify the SOW Type settings for the template used in the upload
  2. Update the Use to create SOW Bid field to NO in the SOW Type settings
  3. Retry the upload process with the corrected file

Keywords

sow xml upload, clauses upload, sow template settings
