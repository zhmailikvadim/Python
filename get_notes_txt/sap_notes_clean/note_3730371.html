Symptom

When publishing a theme in UI theme designer in S/4HANA (OnPrem) you are getting an error about processing CRMWebUI, although all theming and UI framework components are up to date.

Other Terms

WebUI, UI theme designer, silent error, CRMWebUI

Reason and Prerequisites

There is a (small) chance that the theming content contains outdated CRMWebUI content which “overrules” the current one.

Solution

  1. Start the UI theme tool via the transaction /UI5/THEME_TOOL.
  2. You will see a list of custom themes. Scroll down until you find an entry “baseTheme”.
  3. If you want to be safe download this entry to your local disk via the button [Download].
  4. Delete this entry via the button [Delete].

Now you should be able to rebuild the theme without an error.
