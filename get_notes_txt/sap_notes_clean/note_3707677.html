Symptom

You use organizational changes to assign new profit centers to orders. You also employ rules to select the orders to process.

This SAP Note makes the view FINOC_RULE_ORD_DEFAULT more robust and to avoid error FINOC 138.

Other Terms

Error message FINOC 138, "Order ...: New profit center same as old profit center (...).".

Reason and Prerequisites

Program error.

Solution

Apply the attached correction instruction.
