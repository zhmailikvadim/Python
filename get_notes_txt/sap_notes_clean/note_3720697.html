Symptom

  • With each login to BI Launchpad (BILP) a pop-up is observed: "We made updates since you were last here. Click here to find out more"
  • When users logout and login again at that time, the pop-up appears again.
  • The issue occurs even without clearing the browser cache.

Environment

SAP BusinessObjects Business Intelligence Platform 4.3 SP4 P7

Reproducing the Issue

  1. User logs in for first time since some changes >> Banner is displayed to user
  2. User logs off, back in again >> Banner is displayed again
  3. User doesn't clear browser cache >> Banner is displayed

Cause

This message is generated by Web Assistant function.

Resolution

  • The new by design behavior is included starting from SAP BO 4.3 SP5 P4 and BI 2025 base:
  1. User logs in for first time since some changes >> Banner is displayed to user
  2. User logs off, back in again >> Banner is not displayed since it was already displayed earlier
  3. Even if the session times-out >> Banner does not appear again
  4. User clears browser cache
  5. Logs back in again >> Banner is displayed
  • The banner can be removed completely from BILP by disabling the Web Assistant as per KBA: 2731980 - Disable automatic Web Assistant activation on login for all users in Fiori BI Launchpad

See Also

  • 2731980 - Disable automatic Web Assistant activation on login for all users in Fiori BI Launchpad
