Symptom

  • Unable to access the solution link from the alert.
  • Unable to retrieve job information by clicking the 'Additional Details' button.

Environment

  • Human Capital Management (HCM)
  • SAP enhancement package for SAP ERP
  • Payroll control center
  • S/4HANA 

Reproducing the Issue

  • Go to My alert > Not solved.
      Click the solution link that requires access to the ECP system.
      Result: You encounter a 403 Forbidden error.

  • Navigate to Create test payroll data > See details.
      Click on a job and then the Additional Details button.
      Result: You encounter a 403 Forbidden error.

Cause

The service /default_host/sap/bc/gui/sap/its/webgui has not been activated in transaction SICF.

Resolution

To resolve this issue, please activate the following service in your system using transaction SICF:  /default_host/sap/bc/gui/sap/its/webgui 

Activating this service will allow you to access the ECP system from alerts and retrieve job information via PCC.

Keywords

SICF, 403 Forbidden, Solution link, Alert, 'additional details', 'see details', spool.
