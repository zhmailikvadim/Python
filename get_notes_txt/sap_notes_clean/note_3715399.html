Symptom

Functionality to schedule/create treasury payment requests for PRA payment runs is not available in SAP S/4HANA 2023 release.

Other Terms

Related Keywords:

   •   General keywords: Add-on, IS-Oil, Industry Solution, Oil and Gas Upstream, PRA, Roadmap, Production and Revenue Accounting
   •   Problem-specific key words: IS-OIL-PRA, Payment Processing Treasury Integration, Schedule PP Treasury Payment Run, ILM, data destruction, /PRA/REMIT_LINES

Reason and Prerequisites

New enhancement

Solution

A new Fiori application “Schedule PP Treasury Payment Run” has been provided to enable creation of treasury payment requests for PRA payments.

This application enables you to:

  • Schedule a background job to create treasury payment requests

  • Schedule the same process in test mode

  • View log for both test and productive modes

Additionally, a new data destruction ILM object /PRA/REMIT_LINES has been provided to enable the destruction of remittance data from the database table /PRA/REMIT_LINES.

  • Data retention rules can be maintained in ILM policies using transaction IRMPOL

Apply correction instructions and manual steps attached in the note.
