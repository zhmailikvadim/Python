Symptom

The following programs of the type analysis/adjustment program cannot be executed by the user:

  • Correction report for adjusting incorrect superior objects
  • Analysis/Adjustment Program for Period Overlaps 

The programs are terminated with the following message: "The program (program name) must not be executed for the FS-RI release (release name)."

Other Terms

/MSG/X_ADJ_PROGRAM,

005

Reason and Prerequisites

Reason:

The programs check whether the programs can be executed in the relevant release. The programs perform this check with an incorrect release.

Solution

History:
February 23, 2026: New

Troubleshooting Description:
The programs do not perform a release check. The programs are valid for all releases. The user can execute the programs.
