Symptom

  • Incoming Messages for phased contracts(subscriptionContractS4ReplicationMessageIn) are marked obsolete during replication in SAP Sales Cloud Version 2.
  • The replication of non-phased contracts is successful, while phased contracts fail with "isphased = true" marked as obsolete.
  • The issue occurs even for the first replication message of subscription contracts.

Environment

Integration for SAP Sales/Service Cloud Version 2 with SAP S4HANA

Reproducing the Issue

  1. Navigate to System settings -> Integration -> Message monitoring.
  2. Filter message entity -> sap.crm.subscriptioncontractservice.entity.subscriptionContractS4ReplicationMessageIn.
  3. Locate the message ID: 87eebeca-d7f5-4da6-804e-67b0c8d8cac7.
  4. Observe that the message is marked obsolete.
  5. Compare with a successful replication message ID: 61e041c7-14c0-46db-8a4a-72c7e22e1674 where "isphased = false".

Cause

Phased contracts are not supported in SAP Sales Cloud Version 2, as per the product documentation.

Resolution

Refer to the product documentation for SAP Sales Cloud Version 2 at:  SAP Sales Cloud Version 2 Documentation.

  1. Phased contracts are not supported, which is why integration transformation did not occur, leading to the message being marked obsolete.
  2. For successful replication, ensure that the contracts being replicated are non-phased.
