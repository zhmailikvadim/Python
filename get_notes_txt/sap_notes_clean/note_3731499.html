Symptom

  • The Get Started step in the Off‑Neo Migration tab shows the message: "TARGET LOCATION NOT FOUND."
  • The "Continue" button in Get Started is disabled and migration cannot be initiated.

Environment

SAP Cloud Integration for data services

Cause

The tenant is not yet assigned to a migration wave.

Resolution

  1. Create an incident to component LOD-HCI-DS to report this.
  2. Provide CIDS Organization detail.
  3. Check Migration Waves. And mention which Wave is preferred in the description.

Support team will involve DEV team to assign customer Org to the preferred Wave.
