Symptom

  • Is storing backup locally possible?
  • What backup frequencies are supported or recommended for such replication/backup scenarios?
  • Are there any limitations or restrictions on backup frequency?

Environment

SAP BTP, Cloud Foundry runtime and environment

Resolution

  1. Storing backup locally is not possible.
  2. As mentioned in  Backups and Recovery from Backup | SAP Help Portal,Backups are stored encrypted in an object store independent of a specific availability zone.
  3. Backups are stored in a regional object store bucket provided by the IaaS provider. This ensures that backups are accessible from all availability zones within the same region, supporting high availability and rapid recovery even in the event of a zone failure.
  4. Taking system backups are supported but storing them at a location in Vietnam is not supported. Backups are stored encrypted in an object store.
