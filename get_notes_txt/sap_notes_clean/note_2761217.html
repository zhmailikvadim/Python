Symptom

You trigger Complete/Delta Cache Refresh via SXI_CACHE transaction or via activating the relevant object in the Integration Directory. However, the desired objects do not appear in the ABAP Cache (SXI_CACHE), the status remains green and states that the Cache is up-to-date.

Other Terms

SXI_CACHE, PICACHEUSER, Status, up-to-date

Reason and Prerequisites

You use the SAP Exchange Infrastructure and configure a system (Integration server or Application/Business system) to be able to use the ABAP Cache Refresh process. When you trigger the Cache Refresh, the status icon remains green and the desired objects do not appear in the SXI_CACHE.

Apparently, when there is a problem with the user configured to run the refresh process ( usually PICACHEUSER ), the refresh process is terminated, but there is no text message propagated to the user.

Solution

This correction improves the interaction with the user by providing more accurate text for the status of the ABAP Cache (SXI_CACHE)

Implement the correction instructions or the relevant Support Package.
