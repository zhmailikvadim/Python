Symptom

  • The Job Posting Upload and Work Order Upload connector templates downloaded from buyer do not include rate headers required for correct rates on Job Postings and Work Orders.
  • Ensured the rates were configured before the templates were downloaded.
  • The tenant is using legacy rates configuration (rates 1.0).

Reproducing the Issue

  1. Log in as the Fieldglass help desk user (buyer@FG_HELP).
  2. Navigate to the Admin Config page and click on Upload Data.
  3. In the Download File Format section, type Requisition Upload in the Name field and press Enter.
  4. Select the dial next to the found template and click Download to download the template.
  5. Repeat the same steps to download the any other upload.
  6. Open the downloaded templates and check for rate headers.

Cause

The templates downloaded from buyers do not include rate headers for certain countries and currencies despite the rates being configured in the tenant environment.

Resolution

  1. Verify that the rate headers for the required countries and currencies are configured in the tenant environment.
  2. If the rate headers are missing in the downloaded templates, manually add the required headers and values to the upload file.
  3. Refer to the connector guides for the Job Posting Upload (Requisition Upload) and Work Order Upload templates to ensure the correct format for rate headers.
