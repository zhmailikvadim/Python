Symptom

When opening any hierarchy nodes in Fiori app F0960 (Market Segments - Actual), the page automatically scrolls to the top. 

Environment

  • Controlling (CO)
  • SAP S/4HANA
  • SAP S/4HANA Finance
  • SAP S/4HANA Cloud Private Edition
  • SAP Fiori for SAP S/4HANA

Reproducing the Issue

  1. Open an analytical Fiori app, e.g., F0960.
  2. Expand the hierarchy node in the table output.
  3. Observe that the screen scrolls back to the top. 

Resolution

Update the UR version according to SAP Note 2090746. Also, upgrade to the newest SAP_BW and SAP_UI Sp (more information in SAP Note 375631).
