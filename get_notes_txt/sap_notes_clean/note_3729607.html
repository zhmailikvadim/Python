Symptom

This note is only relevant for IDXCH2 Customers!

When exporting C02 "Assignment lists" messages using transaction /IDXCH2/MET_TRIGGER, processes are not selected correctly. This is because the selection does not take into account the cycle selected on the selection screen.

Other Terms

IDXCH2

Reason and Prerequisites

IDXCH2 SP09 or lower

Solution

Implement this note or IDXCH2 SP10
