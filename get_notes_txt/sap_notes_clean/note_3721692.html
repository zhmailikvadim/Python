Symptom

After triggering reprocess in the DO Workspace, a duplicate correction code is being set, resulting in an update termination. The update termination occurs at the system code table level. This note corrects the reprocess logic to prevent duplicate correction code entries and avoid update termination errors.

Other Terms

DO Reprocess Issue, Duplicate Correction Code, Update Termination, /VTA/BDOM

Reason and Prerequisites

This is caused by program error

Solution

Please implement the correction instructions attached. These corrections are only valid if you have applied the Support Packs.

Release V/4 - 2023 V2, and 2023 V3 and 2023 V3 Support Pack 1

Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object.
