Symptom

This note provides a  Function Module (FM) to automate connectivity to SAP Business Technology Platform (SAP BTP) AI services via Intelligent Scenario Lifecycle Management (ISLM).

The delivered Function Module automates the required setup for ISLM connectivity. In particular, it performs the following steps:

  • OAuth configuration
  • Connection mapping in ISLM for the relevant ISLM scenario

Other Terms

ISLM, SAP BTP AI Services, Connectivity, OAUTH Configuration, RFC

Reason and Prerequisites

Are you ready with the prerequisites for Function Module (FM) mentioned below? If not, make sure that you fulfill them before applying this SAP Note.

1. For SAP_BASIS 816 & 758 - Implement the TCI note 3682148

2. For SAP_BASIS 757 & 756 - Implement the TCI note 3490653

Solution

To support the automatic connectivity to BTP AI services, please update to the relevant Support Package or implement the corresponding Correction Instruction.
