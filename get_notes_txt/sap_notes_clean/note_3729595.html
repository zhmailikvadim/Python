Environment

  • SAP Cloud Connector(SCC);
  • Business Technology Platform(BTP);
  • SAP AS ABAP Backend system.

Reproducing the Issue

Post setting up Principal Propagation, test the authentication using PP from BTP application.

Cause

CERTRULE is not properly maintained to consume the correct authentication logon type.

Resolution

Log onto T-code CERTRULE changing option "Logon With" from the current value to "Email", saving it.

Bear in mind that the value of CERTRULE "Logon With" must match same value set from SCC -> Configuration -> On-Premise -> Principal Propagation -> Subject Pattern Rules -> "Subject Pattern", as per image below.
