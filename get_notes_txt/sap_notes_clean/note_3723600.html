Symptom

When approved Employee Time Sheet is amended by adding new entry with attachment, system stores an orphaned duplicate attachment which cannot be purged via Purge MDF Attachment function.

Environment

  • SAP SuccessFactors Employee Central - Time Sheet
  • SAP SuccessFactors Time Tracking

Reproducing the Issue

  1. Navigate to an Approved Employee Time Sheet, add a new entry with attachment TEST.pdf.
  2. Navigate to Manage Document, note there're two files with the same name. 
  3. Use Purge MDF Attachment function to purge the time sheet attachment. 
  4. Only one file is purged and become TEST.txt file, the other one remained as TEST.pdf.

Cause

Duplicate documents are generated during timesheet amendments and one of them is orphaned.

Thus, Purge MDF Attachment cannot remove it from the system. 

Resolution

The development team is working on the fix to enhancement the time sheet amendment logic.

The patch release date would be updated in the KBA once confirmed.
