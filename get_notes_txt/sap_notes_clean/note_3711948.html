Symptom

During data conversion for Advanced-CPE adoption as described in note 3631220, in some cases, SDM status is shown as 'GREEN' even though the status of the SDM job is 'N-Not relevant'. The data conversion has been started in CPE_CONV_WB but the status indicated is incorrect.

The status Not relevant is correct before the data conversion is set to 'Started' in the data conversion workbench CPE_CONV_WB. Once it is started, the status should be shown as 'Red' which means SDM has not yet started or 'Yellow' - if the job is in progress.

Other Terms

CPE, Advanced-CPE adoption, Data conversion.

Reason and Prerequisites

This is a program error

Solution

Apply the correction instructions provided in this note.
