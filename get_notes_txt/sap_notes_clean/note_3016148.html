Symptom

The length of field SERIE_SAIDA in register C181 is incorrect, according to the Guia Prático do EFD.

Other Terms

EFD IMCS IPI, C181

Reason and Prerequisites

This SAP Note is relevant only for Brazil.

This issue is a side effect caused by the solution delivered by SAP Note 2976865

Solution

After implementing this solution, the length of field SERIE_SAIDA becomes 3 instead of 2, being in accordance with the Guia Prático do EFD.

Please carry out the steps below to implement the solution.
