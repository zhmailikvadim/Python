Symptom

During Revision where start date was changed in the past but no change occurred in current one, no change pointer for No Change was created.

Other Terms

/ASADEV/CL_ACI_PREQ_XML_UNKWN, _CHECK_PR_CHANGES

Reason and Prerequisites

PR start date is not possible to be updated through BAPI_PR_CHANGE. Check for start date changes should occur on PO validity start date.

Solution

Implement oss note to check for PO start date if its updated instead of PR.
