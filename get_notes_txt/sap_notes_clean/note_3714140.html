Symptom

  • The report SDMFSTRP does not display quantities posted through service entry in the SD/MM document flow for third-party processes.
  • The report only includes quantities from sales orders, purchase orders, and invoiced quantities.

Environment

  • Sales And Distribution (SD)
  • Logistics Execution (LE)
  • SAP R/3
  • SAP R/3 Enterprise 4.7
  • SAP ERP Central Component
  • SAP ERP
  • SAP Enhancement package for SAP ERP
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Execute transaction SE38.
  2. Run the report SDMFSTRP.
  3. Observe that the report does not display quantities related to service entry or goods receipt postings.

Cause

Report SDMFSTRP is designed to analyze and display quantity flow exclusively between Sales Orders (SD) and Purchase Orders (MM), including invoiced quantities.
Goods receipt quantities, including service entry quantities, are not considered by this report. This is the standard design.

Resolution

This behavior is by design; no standard change is available.
If additional functionality is required, custom modification needs to be applied. See SAP Note 170183 when applying custom modifications.
