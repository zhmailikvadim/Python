Symptom

A large BPMN file failed to import due to a system file size limit and would like the limit increased. 

Environment

import / export for SAP Signavio Process Manager

Reproducing the Issue

  1. Attempt to import a BPMN file larger than 10MB into SAP Signavio Process Manager.
  2. Observe that the file cannot be imported due to the file size limitation.

Cause

  • File size limit prevents importing BPMN files larger than 10MB into SAP Signavio Process Manager. 

Resolution

  1. It is not possible to increase the 10MB file size limit for BPMN file imports in SAP Signavio Process Manager.
  2. If you want to export all your diagrams but are faced with the 'file size limitation' error, try exporting a few diagrams at a time instead.
