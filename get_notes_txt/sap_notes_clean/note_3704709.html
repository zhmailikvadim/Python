Symptom

A new 'Terms of Payment' was created in S/4 HANA system. The initial load for the object DNL_CUST_PRICE was also executed in CRM after that, but the new 'Terms of Payment' is not transferred to CRM.

Environment

  • SAP Customer Relationship Management (CRM)
  • SAP S/4HANA

Reproducing the Issue

1) Create a new 'Terms of Payment' (e.g. Z100) in S/4 HANA system through SPRO path: SAP Customizing Implementation Guide > Sales and Distribution > Basic Functions > SEPA > Maintain Terms of Payment.
*T-code: OBB8 can be used to navigate to the configuration directly.

2) Run T-code R3AS in CRM system, execute the initial load for the object DNL_CUST_PRICE.

3) The newly created 'Terms of Payment' is not transferred to CRM. Check the entry through SPRO path in CRM system: SAP Customizing Implementation Guide > Customer Relationship Management > Master Data > Business Partner > Define Attributes > Define Terms Of Payment -> no such entry  (e.g. Z100) exists.

Resolution

1) Follow the details described in note 1420329 & 1357730 to check the PARVAL1 value used in table SMOFPARSFA in CRM system.

Example: CRM system > T-cd: SE16 > table: SMOFPARSFA 
---------------------------------------
PARSFAKEY     R3A_COMMON
PARNAME        CRMDNL_CUST_PRICE
PARNAME2      PAYMENT_TERMS
PARNAME3

UNAME
PARVAL1       TVZB                                 <<<<< TVZB is only an example here.
---------------------------------------

2) Check the configuration of the object DNL_CUST_PRICE through T-code R3AC3 in CRM, if the configuration is not set up as the following description, please make adjustment accordingly.

a) Run T-code: R3AC3.
b) Open the detail view for the object name: DNL_CUST_PRICE, and go the "Tables/Structures" tab page.

• If the table T052 is desired to be used, then the following entries are required to be configured:

 Table(Source Site)      Main Table/Structure
      T052
      T052                         T052U       
***********

• If the table TVZB is desired to be used, then the following entries are required to be configured:

 Table(Source Site)     Main Table/Structure
      TVZB              
      TVZBT                         TVZB
***********

3) Follow the instruction described by note 1498111 (point 2) to execute the report SMOF_FILL_CRMATAB for object DNL_CUST_PRICE, then execute the initial load for object DNL_CUST_PRICE (through T-code: R3AS) again.
