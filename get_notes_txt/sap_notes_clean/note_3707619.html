Symptom

You create a rule based derivation for entity types that do not contain attributes. Examples:

  • Alternative Payees
  • Alternative Payers
  • Roles without validity periods

Although the derivation step is processed, the data is not added to the change request.  

Other Terms

SAP Master Data Governance, Central Governance

MDGBP, MDG-BP, MDGC, MDG-C, MDGS, MDG-S

SAP MDG for Business Partners

SAP MDG for Customers

SAP MDG for Suppliers

Reason and Prerequisites

Software Error

Solution

Apply the attached correction instructions or the related support package to the affected system(s).
