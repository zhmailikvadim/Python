Symptom

The approval process is triggered twice when adding an approved document.

Reproducing the issue

Prerequisites:

  • You are working in a company in Portugal localization; refer to note 3484377 for the setting of  EDS and B1 Client. 
  • Set up the approval template for an A/R credit memo where user manager (the originator) will create the A/R credit memo and user Vicky (the authorizer) will approve the A/R credit memo.           

Scenario:

  1. Manager creates an A/R credit memo. It is saved as a draft.
  2. Vicky approves the A/R credit memo.
  3. Log in as the originator manager user and add the document.

Expected Behavior:

The draft should be added without the second approval.

Actual Behavior:

The same approval template triggers again.

 

Solution
