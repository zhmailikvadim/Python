Symptom

You use the Credit Management and want do calculate the DSO value for multiple customer account. The value calculated for a single account is displayed also for account where no balances exist in the corresponding periods.

Other Terms

DSO, FDK43, RFDKLI43, F.31, F.35, FD32, FD33, RFDKLI40, RFDKLI41

Reason and Prerequisites

This issue is caused by a program error.

Solution

Import the relevant Support Package or implement the attached corrections.
