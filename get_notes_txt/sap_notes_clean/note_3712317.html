Symptom

Requirement to change or add values in a Variant within Test Plan in the Test Your Processes.

Environment

SAP S/4HANA Cloud Public Edition

Cause

Resolution

  1. Open the required Test Plan in the Test Your Processes.
  2. Navigate to the Variants tab.
  3. Select the required Variant and open the detail page.
  4. Click the Edit button (available at the bottom-right corner).
  5. Locate the required Action ID.
  6. In the Value column, add or change the value for the specific Action ID.
  7. Save the changes.

Keywords
