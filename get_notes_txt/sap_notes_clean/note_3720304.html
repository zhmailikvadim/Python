Symptom

There is a "limitation' in the Resource Related Quotation (RRQ) functionality when used with Advance Execution Item processing.

## Technical Details

**Affected Component:** Dynamic Item Processor (DIP) Profile - Material Determination

**Unsupported Configuration:** "Transfer Cost Only" option

Other Terms

AEI, ODP1, RRQ

Reason and Prerequisites

When "Transfer Cost Only" is selected in the DIP profile's material determination:

1. System generates service transaction items with **costs but no quantity**
2. System sets Quantity as 0 for these items

Solution

Implement the attached corrections.
