Symptom

The DTP with "No further processing without Master data" is set.

When transferred values which doesn't exist in the master data, the DTP gets executed successfully instead of ending up with SID not found error and the values gets added to the master data.

Other Terms

No further processing without Master data, SID generation, SAP HANA execution, Data Transfer Process

Reason and Prerequisites

Program Error.

Solution

To solve the issue please apply the Support Package as mentioned below. In urgent cases you can use the correction instructions.
