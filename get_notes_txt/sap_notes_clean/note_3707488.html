Symptom

After completing the kernel upgrade, the system encountered a large number of "RFC_NO_AUTHORITY" dumps.

In SM58, there were numerous errors indicating "No RFC authorization for function module XXX."

Affected Kernel patch level is

722 >= 1512

753 >= 1529

754 >= 626

Environment

  • ABAP Platform
  • SAP NetWeaver
  • SAP Web Application Server for SAP S/4 HANA

Cause

Known kernel issue

Resolution

Permanent solution:

It is necessary to follow SAP Note 3694152.

  • For Kernel 722, make sure the version is upgraded to at least SP 1513 or higher.
  • For Kernel 753, make sure the version is upgraded to at least SP 1532 or higher.
  • For Kernel 754, make sure the version is upgraded to at least SP 629 or higher.

Workaround:

Refer to KBA 2569749 and add the missing S_RFC to the user mentioned in the RFC_NO_AUTHORITY dump.
