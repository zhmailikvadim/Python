Symptom

You are using the SAP S/4HANA Migration Object Modeler (transaction LTMOM) to modify a migration object.

You open the Migration Object Detail Screen and change, for example, the description.

Although the Save button was pressed, the changes were not persisted.

Other Terms

Migration Cockpit, staging table, staging approach, data migration, LTMOM, Migration Object Modeler

Reason and Prerequisites

You are using:

SAP S/4HANA

  • Migration Object Modeler 

Solution

Implement the attached correction using transaction SNOTE.
