Symptom


In a JavaScript task, we want to incorporate HTML code that is read and used by the process later on. Unfortunately, the system issues the following error message:

Is HTML code within a JavaScript task supported or not working?



Solution


Remove the line breaks in the HTML code. Currently, the check fails because the HTML code is split over several lines, which means that the system expects a variable name. Once your HTML snippet is in one line, you can use it for later.
