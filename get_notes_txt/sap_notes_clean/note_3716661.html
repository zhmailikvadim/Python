Symptom

  • The system currently allows email notifications to be sent to dynamic groups but does not support the automatic inclusion of individual approvers' first and last names within the email content.
  • The absence of a token in SAP SuccessFactors Onboarding 2.0 that can dynamically include the first and last names of multiple approvers from a dynamic group within email notifications.

Environment

SAP SuccessFactors Onboarding

Reproducing the Issue

  1. Attempt to configure an email template in SAP SuccessFactors Onboarding 2.0 to include dynamic tokens for individual approvers' names from a dynamic group.
  2. Observe that the system allows email notifications to be sent to the dynamic group but does not support the inclusion of individual approvers' names automatically.

Cause

The current functionality of SAP SuccessFactors Onboarding 2.0 does not support dynamic tokens for listing individual approvers' names within email notifications.

Resolution

  • Regarding the functionality of Onboarding 2.0. At this time, `there is no available token in Onboarding 2.0 that can list the names of all three approvers from a dynamic group within an email`. While the system does allow email notifications to be sent to the dynamic group, it does not currently support the automatic inclusion of each approver’s first and last name within the email content.
