Symptom

Access to applications CreateTechnical Object and Change Technical Object is unavailable due to missing business catalog SAP_EAM_BC_TO_MC in SAP S/4HANA Public Cloud system.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Navigate to the App: Business Catalogs.
  2. Search for Catalog ID SAP_EAM_BC_TO_MC.
  3. Observe that no results found for the specified catalog ID.

Cause

  • Business catalog SAP_EAM_BC_TO_MC is deprecated since version 2302 with old scope items BH1, BH2, and BJ2.
  • To access apps CreateTechnical Object and Change Technical Object, new scope items 4HH and 4HI must be activated as prerequisite.

Resolution

  1. Make sure scope items 4HH & 4HI are activated.
  2. Assign the business catalog SAP_EAM_BC_TO_MNG_PC to user.
