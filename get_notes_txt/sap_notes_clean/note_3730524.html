Symptom

In transaction code TPM44, the system does not calculate accruals for interest rate instruments that use conditions with the interest calculation type 'Average linear interest calculation'.

Other Terms

550, 580, risk free rates, SOFR, SONIA, TONA, ESTER

Reason and Prerequisites

This is a programming error.

Solution

After implementing this SAP Note, the system correctly calculates accruals and creates accrual flows for interest rate instruments that use conditions with the interest calculation type 'Average linear interest calculation' in transaction TPM44.

Run transaction SNOTE to implement the correction instructions.
