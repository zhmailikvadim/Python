Symptom

In an EAM order in EAM Simplicity, you add a non-stocked material or an externally processed operation with specification of a contract/outline agreement. The system does not redetermine the G/L account according to the newly transferred material group. This system behavior is not desired.
In return, if an external procurement item is changed in the EAM order and the same contract/outline agreement is added, a G/L account determination takes place. The newly determined G/L account is set and the warning message EAMS_BO 401 'Cost element changed from &1 to &2 by material group, check entries' is issued. displayed to the user.

Reason and Prerequisites

Reason: 
Program error

Solution

Implement the program corrections.

After you implement the program corrections, the system performs a G/L account determination after the transfer of the material group by the contract. The newly determined G/L account is set. If the user does not want this, he or she can adjust the G/L account.
