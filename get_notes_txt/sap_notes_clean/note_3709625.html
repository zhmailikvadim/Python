Symptom

While posting a document withholding tax amount is not shown in document when lower deduction certificate exemption is maintained.

Environment

  • SAP ERP
  • SAP R/3
  • SAP ERP Central Component
  • SAP S/4HANA
  • SAP S/4HANA Finance
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • This is relevant for India Localisation or Country India Version (CIN) component.

Reproducing the Issue

  1. Maintain exemption threshold in BP as 99.99% for Vendor.
  2. Post an invoice and observe that the tax amount is not calculated at all.

Cause

The issue is observed due to the commercial rounding rules configured in SPRO for the withholding tax type, which affects the calculation of minimal TDS amounts.

Resolution

  1. Review the withholding tax configuration settings in SPRO for the relevant tax type.
  2. Verify if the commercial rounding rule is selected and assess its impact on minimal TDS amounts.
  3. Consider modifying the rounding rule configuration if it is causing the issue, and then re-test the case with other rounding rules.
  4. Refer to F1 help documentation for details on rounding rules before making adjustments.
