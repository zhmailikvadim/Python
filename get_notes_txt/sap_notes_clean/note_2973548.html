Symptom

A fresh relevant product is changed to a non-fresh product (based on Forecast profile assignment). When FRP is executed for locations having these products the exceptions( occurred in previous runs related to Fresh Product ) are still getting generated in each FRP execution.

Other Terms

FFISCM, Fresh Product Forecast, Standard Forecast Horizon, Exception, FRP Run

Reason and Prerequisites

Code Error

Solution

Please implement automatic correction instructions. You can also implement SAPK-70205INFFISCM (SP05), if available
