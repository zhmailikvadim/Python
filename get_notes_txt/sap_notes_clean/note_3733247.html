Symptom

  • The Service Quotation number is not displayed in the transaction history of the Service Order.
  • Instead of the quotation number, the description of the quotation is shown in the Transaction column.

Environment

SAP S/4HAHA

Reproducing the Issue

  1. Create a Service Quotation.
  2. Create a Follow-up Service Order.
  3. Open the transaction history tab of the service order.

Cause

The transaction history in the Web UI is designed to display the business transaction’s description in the “Transaction” column, while the document number (OBJECT_ID) is shown in the “Transaction ID” column.

Resolution

The system behavior for displaying transaction history in business documents is as follows:  
- For service documents such as service orders, service confirmations, and service contracts, the "Transaction" field shows the description entered in the business transaction (DESCRIPTION field).  
- For maintenance documents like maintenance plans and maintenance orders, the "Transaction" field displays the transaction ID.  
- For all documents, the "Transaction ID" field shows the document number of the business transaction (field OBJECT_ID).
