Symptom

You wish to add a country specific field e.g. “Local Information” (under the Personal Information portlet) to the Employee Delta Export Add-In for Microsoft Excel.


This does not seem to be available.

Environment

  • BizX
  • SuccessFactors

Reproducing the Issue

  1. use the Standard Add Column wizard to find Local Information
  2. Local Information is not available 

Cause

Expected behaviour. Neither Local nor Global information are supported in the Employee Delta Export Add-In

Resolution

Please note there are no plans to further enhance the Employee Delta Export Add-In for Microsoft Excel
