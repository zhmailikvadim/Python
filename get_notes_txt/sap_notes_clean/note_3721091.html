Cause

Resolution

When using maximum stock level(lot size HB/H1) together with MRP type PD:

In case lot size = HB (indicator 'Max. stock level' set blank), the system calculates the lot size as below:

  •  When shortage qty is smaller than maximum stock level (seeing Case 1),
    (Formula 1) Lot size = maximum stock level - current warehouse stock - existing fixed receipt elements.  
  • When shortage qty is greater than maximum stock level (seeing Case 2),
    (Formula 2) Lot size = total requirements - current warehouse stock - existing fixed receipt elements.  

In case lot size = H1 (indicator 'Max. stock level' set 1), the system calculates the lot size as below (seeing Case 3):

  • Lot size = maximum stock level + total requirements - current warehouse stock - existing fixed receipt elements.

Maximum stock level variant/V439A-HBVAR

Controls how the system calculates lot sizes in the lot-sizing procedure, "Replenishment up to maximum stock level" in reorder point planning, demand-driven replenishment, and in MRP.

The maximum stock level is then considered the limit for the physical stock. The maximum stock level can be the capacity of a tank, for example. 
Even if more requirements exist, the system only creates an order proposal to bring the stock level up to the maximum stock level.

If indicator 1 is set, the system orders enough to bring stock up to the maximum stock level and to cover all requirements.

Use

If indicator 1 is set, the system calculates the lot size using the following formula:

Lot size = maximum stock level + total requirements - current warehouse stock - existing fixed receipt elements

 

If no indicator is set, the system calculates the lot size using the procedure previously used.  
The system calculates the lot size in two steps using two formulas. The system then creates the order proposal using the greater of the two lot sizes.

(Formula 1)

Lot size = maximum stock level - current warehouse stock - existing fixed receipt elements

(Formula 2)

Lot size = reorder point + total requirements - current warehouse stock - existing fixed receipt elements
