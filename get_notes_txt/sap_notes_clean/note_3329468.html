Symptom

When report RPUTMS9S (transaction PC00_M12_DMFA_MG) is executed to upload a file of the DmfA Update Notification (form type = DMUN) (Notificatie van wijzigingen/Notification de modification/Aenderungsanzeige) an ABAP runtime error can be raised, or the uploaded TemSe file might be incorrect.

This error can happen after the implementation of SAP Note 3304310.

The reason for this error is a wrong structure assignment for the new section ECDP 90600 Employer Compensation Declaration Part (EmpCompensationDclPart) for form DMUN DmfA Update Notification.

Other Terms

RPCDMFB0, DMFA, 20232, 20231, 2023Q1, T5F99FS, T5F99FD, T5F99FF, T5F99F0, DMUN, FIELD_NOT_FOUND
