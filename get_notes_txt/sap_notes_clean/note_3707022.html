Symptom

When editing a report template in transaction CG42, the report symbols for example <02GEMP_NAME1[D:NAME]> are not displayed.

Environment

  • SAP ERP Environment, Health and Safety (EHS)
  • SAP R/3
  • SAP R/3 Enterprise
  • SAP ERP Central Component
  • SAP ERP
  • SAP Enhancement package for SAP ERP
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Open transaction CG42 Edit Report Template screen.
  2. Select a report template and click Document button.
  3. Report symbols are not displayed in the report template.

Resolution

Click the "Switch Display (shift+F7)" button and the hidden text, i.e. the report symbols would appear.

Keywords

Report Template, CG42, report symbol
