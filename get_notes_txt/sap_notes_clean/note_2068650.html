Symptom

You search for business transactions via the Agent Inbox. One of the search criteria is "Status = Not Completed". The search returns also completed transactions.

Other Terms

CRMD_AUI_BTIDX, STAT_NOTCOMPL, BT Index table, business transaction table

Reason and Prerequisites

There is a program error. It is relevant only if business function "IC Inbox Extensions 2" (CRM_IC_INBOX_2) is active and if you are using the predefined inbox search table for business transactions in the non-accelerated search (as configured in your inbox profile).

Solution

Apply the attached correction instructions.
