Symptom

The Message Editor can show the message payload of PI messages. The payload window shows the message in the format that was processed by the system. Often, messages contain XML payloads without line breaks or any formatting because the XML was generated by a system. It should have an option to format XML payloads so that humans can read it more easily.

Other Terms

Message Monitor, pretty print

Reason and Prerequisites

Missing feature.

Solution

We provide a new option with release 7.5 SP10 to pretty print XML payloads and show them formatted for better reading. The user can decide to show the original payload or to show it formatted.
