Symptom

You use the function for automatically setting a retroactive accounting trigger when the tax year is opened (see subapplication STAR Open Tax Year: Automatic Retroactive Accounting Trigger). This SAP Note corrects the following errors:

  • The system sets a date for a retroactive accounting trigger in the infotype Payroll Status (0003) that is before the earliest personal retroactive accounting date.
  • The personnel number had a mid-year change of payroll area in the previous year. Payroll is no longer run for the previous payroll area. As a result, no retroactive accounting trigger to the previous year is set.  

Other Terms

 CL_HRPAYDE_ST_SET_RRDAT_TXYEAR, STAR

Reason and Prerequisites

 Program error

Solution

Implement the attached correction instructions or import the specified support package.

After you import the HR Support Package or implement the correction instructions, you do not have to perform any further follow-on activities.
