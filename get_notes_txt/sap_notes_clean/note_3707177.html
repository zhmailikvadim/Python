Symptom

This NOTE contains BC set for Defense Disconnected Operations - 2023 FPS2 & FPS3 - 2025 FPS0 & FPS1 Releases BC Set.

Other Terms

 BC Set, Defense Disconnected S/4.

Solution

Please implement the correction instructions as per the manual steps provided with this NOTE. During activation of the BC-Set be aware that this could lead to overwriting custom setup with standard delivery shipments, depending on your decision in the transaction SCPR3, since the BC-Set delivery is not incremental.

Please follow below steps to activate BC set for 2023 SP2 & SP3 - 2025 SP0 & SP1 Releases.

1.Unzip the file ZDFS_DIS_REMOTE_SERVER_2023FPS2V.zip in the attachments to get the file ZDFS_DIS_REMOTE_SERVER_2023FPS2V.bcs.
2.Launch the transaction SCPR3.
3.Click on the Upload BC Set button and upload the file ZDFS_DIS_REMOTE_SERVER_2023FPS2V.bcs
4.Save the BC set and activate using transaction SCPR20.
5.Alternatively use transaction SCPR20 to activate in other systems once BC set is available.
