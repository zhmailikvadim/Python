Symptom

1. For public sector payroll
The function of feature '25GRC' is not activated. The setting of '25GRC' does not take effect if you run payroll using public sector schema 'RPSG'.

2. For private sector payroll
The setting of '25GRC' should only affect non-fullrate employees, whose monthly earnings are under $750 (referring to CPF rate table since year2014 and onwards).
Currently, if return value of '25GRC' is set to '2', the setting will still affect employees whose monthly earnings are between $750 to $1200.

Other Terms

25GRC

Reason and Prerequisites

This is a program error.

Solution

Apply Support Package.
Alternatively, apply corr.instructions attached to the SAP Note.
