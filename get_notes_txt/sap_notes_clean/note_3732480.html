Symptom

When loading a component through a Webcuif UIF callback, if the component uses namespaces in its name, like \namespace\compName, and the component renders a Cellerator tag, the component does not load and there is an error in the console of the Developer Tools: Uncaught Error: Syntax Error, unrecognized expression.

Other Terms

AJAX, WEBCUIF, uif_callback, handle_request

Reason and Prerequisites

JQuery does not allow slash or dot characters in its selectors.

Solution

WARNING: This Note contains JavaScript code.

Please do not forget to erase your browser cache (temporary internet files) and server cache, and close all local browser instances before restarting the application again.

Please implement the attached Correction Instructions or the mentioned Support Package.
