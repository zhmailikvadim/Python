Symptom

When sending a meter reading complaint using transaction /US4G/MR_COMPLAINT, the system displays the error message "Error determining the configuration for the specified criteria" and fails to send the complaint.

Other Terms

US4G, MET, /US4G/MR_COMPLAINT, /APE/ENGINE_APPL, /US4G/CL_WIM_API, COMPLAINT_MR_WITHOUT_PERIOD, COMPLAINT_MR_WITH_PERIOD, NOSCENARIO, Not Relevant for Supply Scenario, DE_B_MET_MV_COMPLAINT_DSO

Reason and Prerequisites

Program error.

Solution

Please implement the correction instructions mentioned in this SAP Note or import higher support packages.
