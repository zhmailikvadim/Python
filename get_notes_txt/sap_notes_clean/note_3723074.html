Symptom

This note contains information about:

  • OMF nr. OMF nr. 2036/2025 privind principalele aspecte legate de întocmirea şi depunerea situaţiilor financiare anuale şi a raportărilor contabile anuale ale operatorilor economici la unităţile teritoriale ale Ministerului Finanţelor, precum şi pentru reglementarea unor aspecte contabile.

  • The Authority Software has updated with new structure of XML files needed to be sent.

  • This information is based on legislation and regulations which could be subject to change at any time by the respective authorities and are therefore not binding. The note will be updated if necessary.

Other Terms

Romania, Financial Statement 2025, RO_FS_GAAP, RO_FS_IFRS

Reason and Prerequisites

  • OMF nr. 2036/2025 privind principalele aspecte legate de întocmirea şi depunerea situaţiilor financiare anuale şi a raportărilor contabile anuale ale operatorilor economici la unităţile teritoriale ale Ministerului Finanţelor, precum şi pentru reglementarea unor aspecte contabile.

        The applications for Annual Reporting - XML files were published on the Tax Authority site:

        https://static.anaf.ro/static/10/Anaf/Declaratii_R/situatiifinanciare/2025/1002_5_2025.html  for RO GAAP

        https://static.anaf.ro/static/10/Anaf/Declaratii_R/situatiifinanciare/2025/1040_1041_2025.html for RO IFRS

Solution

This legal change affect RO GAAP Reporting and RO IFRS Reporting.

  • You will update your FSV accordingly with templates above or with new rows introduced by law. 
  • To implement this legal change, implement the relevant correction instructions or apply the relevant Support Package when available.

There are several sample Romania Financial Statment Versions available:

  • 2810   Annual Financial Statement for Romania - BL
  • 2811   Annual Financial Statement for Romania - BS
  • 2812   Annex 30 Romania
  • 2813   Semester reporting RO
  • 2814   Semester annex 30 Romania
  • 2815   Annual Financial Statement for Romania - IR
  • 2816   Annex 30 Romania -IR
  • 2817   Semester Financial Statetment for Romania - IT
  • 2818   Semester Annex 30 Romania - IT

These are delivered by BC Set /CEECV/ROFI_FSV and they are based on accounts from YBRO chart of accounts.

If you want to get the sample data, it's necessary to upload the BC Set:

  • Download attached file CEECV_ROFI_FSV.bcs.zip and extract it locally
  • Call transcation SCPR3 and selet from Menu: BC Set -> Upload
  • Select file CEECV_ROFI_FSV.bcs extracted in the first step
  • Save the BC Set. Use package /CEECV/ROFI when asked for.

Then it's necessary to activate the BC Set:

  • Make sure, yu are logged in your Development/Customizing or Test system. Never perform following steps in your Productive system!
  • Call transaction OB58 and check if any of FSVs 2810 - 2818 exists. If so, consider if you want to rewrite them. If not, skip further activation. If so, delete all 2810 - 2818 FSVs. Otherwise you youd face activation errors.
  • Call transaction SCPR20, put /CEECV/ROFI_FSV as BC Set and activate it by pressing F7 key.

After successful activation of the BC Set call transaction OB58 and adjust all necessary 2810 - 2818 FSVs manually.
