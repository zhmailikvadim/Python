Symptom

Manage Source Lists app (ID F1859), the source line item paste function will throw error when pasting line copied from this function through excel.

Environment

  • Materials Management (MM)
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA
  • SAP Fiori

Reproducing the Issue

  1. Manage Source Lists app
  2. click into an entry
  3. edit
  4. choose an entry
  5. click on the icon 'copy to clipboard'
  6. paste it to excel
  7. create a new line
  8. choose the new line and paste the data back, the error is throwed. 

Resolution

Implement Note 3715666 to resolve the issue with the pasting functionality.

Currently, when a row is copied and pasted into Excel, the associated description/text values are included along with IDs. 
When the same data is copied from excel into the application, the system interprets it as single value which results in an error.
This is not the right way to do it as Excel does some formatting, please remove texts/descriptions and paste, it will work.
Instead, Notepad can be used to copy and paste.
