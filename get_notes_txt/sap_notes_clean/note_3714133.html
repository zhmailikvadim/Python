Symptom

This SAP Note delivers the creation and/or modification of the documentation of objects that are relevant to the solution delivered via SAP Note 3695876.

Other Terms

&CALC (for more information, see SAP Note 2327917)

Reason and Prerequisites

The following Support Package (SP) level is the minimum prerequisite:

ReleaseSupport Package
S4HCM 102 HR SP 00
S4HCM 101 HR SP 06
S4HCM 100 HR SP 19
SAP_HR 6.08 HR SP D0
SAP_HR 6.04 (EhP6 and EhP7) HR SP K2

SAP Note 3695876 is required.

Solution

The new report NOTE_3714133 is delivered with this SAP Note. Use the Note Assistant (SNOTE) to implement the correction instructions.

Versioning

Version Date Reason
1 March 03, 2026 Released for customer

Important Information

  • The report serves only as a tool to generate the required objects automatically. Therefore, it has no positive or negative effect on transporting the report <NOTE_3714133> to your productive system.
  • The objects created by the execution of the report can only be used after you apply the main SAP Note that delivers the solution.

Good to Know

  • When you install an SAP Note, some prerequisites, which are SAP Notes that have already been released and that change the same context of the current one, might be required. Even if these prerequisites are not necessarily related to the symptom of this SAP Note, you must implement them due to technical reasons.
  • During the piloting phase, this SAP Note is available only for pilot testing. Even though SAP performs internal functional tests, it is possible that tests in a customer environment reveal issues not captured before, making it necessary to update the solution and version of this SAP Note. After this SAP Note has its status updated to "Released for Customer", it will only be changed if there is a very specific situation that demands an update, and the reason for it will be added to the log of changes.
