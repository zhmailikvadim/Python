Symptom

You are using the SAP Fiori App - Advanced Scheduling Board (ASB) in Production Planning and Detailed Scheduling (PP/DS) for SAP S/4HANA OP2023 release or OP2025 (FPS0/FPS1) release, and you notice that the operation activities such as tear down, setup or produce are drawn incorrectly in the operation chart. Or you are getting Server error when trying to open the operation chart having partially confirmed orders with message- 'Server error. Please contact your system administrator.'.

Solution

As general rule, SAP recommends that you install a solution by applying a Support Package. However, if you need to install a solution earlier, use the Note Assistant to implement the correction instruction.
