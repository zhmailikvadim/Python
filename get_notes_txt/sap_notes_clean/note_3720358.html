Symptom

This SAP Note provides you with adjustments in the Inflation Accounting for Asset Accounting solution for the Inflation Management of Fixed Assets (J_1AA001) report. It addresses the incorrect update of the last revaluation date.

When you run the J_1AA001 report using the batch input version, the system updates the last revaluation date of the asset even when the index is provisional.

Other Terms

Inflation Management, INMA, Inflation, Inflation Accounting, Inflation FI, Asset Accounting, J_1AA001, J1AI, J_1AAINFL_REVERSE, Last Revaluation Date, Batch input.

Reason and Prerequisites

You have implemented previous SAP Notes listed in 3502825 - Inflation Accounting for Asset Accounting and General Ledger Accounting: Overview Note.

Solution

After implementing this SAP Note, when you run the J_1AA001 report in batch input version, the system does not update the last revaluation date of the asset if the index is provisional.
