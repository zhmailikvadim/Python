Symptom

When submitting a report electronically from Run Statutory Reports Fiori application, you may encounter the following error: "*******  is not configured in Partner Directory."

Environment

  • SAP S/4HANA 
  • SAP Fiori for SAP S/4HANA

Reproducing the Issue

  1. Open the Run Statutory Report Fiori application.
  2. Open the reporting task for the DE_VAT_ADV_RTN report by entering the report name, entity name, and reporting year.
  3. Generate a report run.
  4. Open the generated report run and try to submit the report to the government.
  5. Report submission fails with the error message: *******  is not configured in Partner Directory

Cause

ELSTER Tax Identification Number was not maintained 

Resolution

Verify the configuration in the Manage Communication Settings app and confirm that the communication setting for the communication type Tax Reports ELSTER (DE) is maintained correctly.

Please ensure that:

  • The correct Identification Number is maintained, and
  • The appropriate ELSTER details are correctly configured on the authority portal.
