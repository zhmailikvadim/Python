Symptom

In Manage Usage Decision app, the Stock Posting Summary section is not editable. The content filled is cleared automatically.

Environment

SAP S/4HANA Cloud Public Edition

Cause

There might be a cache issue.

Resolution

Try the following methods to see whether the issue can be resolved:

  • Clear the browser cache or change to another browser.
  • Logon with another language.

Keywords

Manage Usage Decision app, Stock Posting Summary, not editable, clear, delete, cache, browser, logon language, delete
