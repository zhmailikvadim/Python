Symptom

Unable to find the Transfer Deferred Tax app and run deferred tax jobs for Switzerland in SAP S/4HANA Cloud Public Edition.

Environment

SAP S/4HANA Cloud Public Edition.

Cause

Switzerland is currently not available in the country supported for deferred tax processing, as mentioned in the Help Portal: Deferred Taxes

Resolution

The feature request ID: 367850 is raised for this feature. Vote this request if the same feature is required.

Each session has a minimum number of votes. Once an improvement request reaches this voting threshold, SAP will review and evaluate your submitted improvement request and will maintain a status on SAP Customer Influence.
