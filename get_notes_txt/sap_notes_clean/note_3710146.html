Symptom

You are using the API 'BankAccountPrenoteEntry' to change an existing prenote. While doing so, you receive the old value of field ChangeStateID in the response structure.

Other Terms

Class CL_BCA_IN_DAPI_BAPRE_DPC

Reason and Prerequisites

The field ChangeStateID was not updated to the new value from the database.

Solution

Import the relevant Support Package that is specified for your release. As a manual advance correction, you can implement the attached correction instructions using the transaction SNOTE. There are no manual activities.
