Symptom

When selecting the show benefit button on the IT0442 (Company Cars) renewal screen the display incorrectly derives the tax year from the current date.

Example:
Current date is 08.09.2021, the show benefit calculation displays the values for the tax year 2020/21 even though the current tax year is 2021/22.

Other Terms

IT0442, Company car,

Solution

After implementing this SAP Note the show benefit calculation works in a similar way to transaction PA20 / PA30.
The show benefit calculation button is now on the top of the screen and a new tax year selection input field is displayed next to the button.

To implement the changes in your system, apply the assigned correction instruction of this SAP Note. Alternatively, apply the HR Support Package indicated for your release under Reference to Support Packages.
