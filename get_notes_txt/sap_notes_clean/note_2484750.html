Symptom

Error message FAGL_SFWC001 is raised when you start FAGL_FCV. The error says that Business Function FIN_GL_ERR_CORR is not activated.

Environment

  • Financial Accounting (FI)
  • SAP S/4HANA FINANCE 1605
  • SAP S/4HANA, on-premise edition 1511
  • SAP S/4HANA 1610

Cause

As you can see in the application help for report FAGL_FCV, one of the prerequisites is that business functions FIN_GL_ERR_CORR and FIN_GL_ERR_CORR_SUSP are activated.

In this version of the software, ECS is used to process the postings of the foreign currency remeasurement. That is, the business function ECS (FIN_GL_ERR_CORR) has to be activated and configured to run this report. ECS is required, since it supports the new currency types available in the General Ledger. Batch input does not support the new currency types (only the FI currencies).

In previous versions, batch input was used in the case of errors. If there were no errors, the remeasurement report could make the postings directly.

Resolution

You can review these business functions in transaction code SFW5 and activate them if required.
Please refer to the following KBA in case you need help:

1641394 - SFW5: How-to activate Business Function
