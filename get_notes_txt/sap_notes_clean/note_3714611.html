Symptom

After 2602 upgrade, the Flexible Workflow Administration app displays text information in tables incorrectly, with text being hidden.

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental." 

Environment

SAP S/4HANA Cloud Public Edition 2602

Reproducing the Issue

  1. Open the Flexible Workflow Administration app.
  2. Select a workflow.
  3. Navigate to the workflow log.
  4. Observe that the text in the table is not displayed correctly.

Resolution

A fix for this issue is planned to be delivered with SAP S/4HANA Cloud Public Edition 2608

SAP S/4HANA Cloud Public Edition, 2-System Landscape – Upgrade & Maintenance Schedule

SAP S/4HANA Cloud Public Edition, 3-System Landscape – Upgrade & Maintenance Schedule
