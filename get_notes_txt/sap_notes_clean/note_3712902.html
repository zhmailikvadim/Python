Symptom

  • Log of BTCTRNS1 / BTCTRNS2 run details required.
  • Need to check when jobs were suspended and released by BTCTRNS1 / BTCTRNS2.

Environment

SAP S/4HANA

Resolution

  1. In transaction SJOBREPO use the menu option: Goto -> Deactivation History
  2. This provides a log of when the Job Repository was Deactivated / Activated, including by BTCTRNS1 / BTCTRNS2
  3. Date, time and user details are provided
