Symptom

The BAPI BAPI_OBJCL_CREATE doesn't create the classification to BOM components.

The error message CL763 - Object does not exist occurs

Environment

  • SAP ERP
  • SAP S/4HANA
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Execute transaction SE37.
  2. Test the BAPI BAPI_OBJCL_CREATE with setting parameters:
    • OBJECTNAME = object with existing BOM number and BOM item
    • OBJECTTABLE = STPO
    • CLASSTYPE = 023 or 050
    • CREATE_MISSING_KEY = X
  3. Observe that the object key is not created.
  4. The error message occurs

Cause

The field STPO-CLSZU (Classification number) is set internally by BOM programs and passed to classification programs where necessary. It cannot be created or changed with the BAPI. 

Resolution

There are no standard BOM APIs available to populate the field STPO-CLSZU in customer programs.
