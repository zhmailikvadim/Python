Symptom

In a Measurement Concept Instance, if a MaLo PoD contains an actor with ID 00000000000 and actor code CONSUMER, an unnecessary hyperlink is available on the actor ID that navigates to the Technical Resource transaction.

This issue is relevant for the market roles: SUP, DSO.

Other Terms

Master Data Objects, MaLo, Actors tab, CONSUMER, Actor ID, Technical Resource, transaction code

Reason and Prerequisites

The error occurred because the hyperlink was incorrectly configured.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the system no longer displays a hyperlink for Actor ID 00000000000 with the CONSUMER actor code, preventing unnecessary navigation to the Technical Resource transaction.

You can verify the fix by creating a new instance or retesting an existing one and checking that the hyperlink is no longer present for Actor ID 00000000000 with the CONSUMER actor code on the Actors tab.
