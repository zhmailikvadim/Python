Symptom

An error occurred while updating Employee Skills in the Data Workbench. Error: Unprocessed record; Re-import.

Environment

SAP Cloud for Customer

Reproducing the Issue

  1. Go to Data Workbench work center
  2. Click on Import view
  3. Click on Complete Business Object
  4. Choose Employee Skills
  5. Click on Next
  6. Upload file
  7. Click on Import
  8. Click on Monitor view
  9. Select the relevant Task Name
  10. Open the file
  11. See the error "unprocessed record; re-import"

Cause

The External Key mentioned in the Import file is already present in the system.

Resolution

Ensure that the External Key's in the Import file does not already exist in the system.

Keywords
