Environment

Disclosure Management 10.1

Cause

Incorrect numerical value

Resolution

The issue is caused by an Excel value having a numeric format, with a - (minus) as value. This cannot be converted to a valid iXBRL value and the system will error out on this.

The value needs to be changed (to 'empty' or 0 )

Note: from the message, you cannot determine which cell is causing this. Therefore, all data-cache chapters need to be verified
