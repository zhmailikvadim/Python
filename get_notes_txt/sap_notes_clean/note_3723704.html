Symptom

Transaction /PACG/ECM_VFSMPG is used to maintain the entries stored in table /PACG/ECM_FSMPG. A dump occurs when the user adds an entry to the transaction /PACG/ECM_VFSMPG, and that entry already exists in the table /PACG/ECM_FSMPG with the DELFLAG = ‘X’.

Other Terms

FSM, E4C, Connector, /PACG/ECM_VFSMPG, /PACG/ECM_FSMPG, TABLE_INVALID_INDEX

Reason and Prerequisites

Changes in logic /PACG/LECM_VFSMPGF01, FORM NEW_RECORD.

Solution

Please, implement this correction instruction.
