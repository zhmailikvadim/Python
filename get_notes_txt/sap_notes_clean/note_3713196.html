Symptom

Warning message appears during purchase order creation about characters in fields limitation:

The use of the last &1 characters in field &2 is restricted (&3 of &4)
Message no. AM228

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Open the Manage Purchase Orders app.
  2. Enter the relevant information in the required fields.
  3. Attempt to create a purchase order.

Observe the warning message, for example: "The use of the last 25 characters in field STREET is restricted (49 of 60) - Street".

Cause

The warning is raised to inform the user that address data may be truncated when transmitting this information. This is why some characters can be used only with restrictions.

Fields where this restrictions applies:

FieldLength   Processed Characters
Name fields          4035
Street6035
Location 4035
District4035
Search Term 12010

Resolution

Adjust the input in the mentioned fields to ensure that the last characters are not utilized to avoid truncation.
Our recommendation is to use secondary a line for the field that is being displayed in the warning message. If needed you can add additional text to other fields e.g. Name2, Name3.

See Also

  • Refer to: KBA 3288863 - Name truncated in Payment medium.
