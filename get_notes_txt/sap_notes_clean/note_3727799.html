Symptom

You archive billing documents with the archiving object SD_VBRK and document flow records that belong to IS-OIL documents are incorrectly archived and deleted.

Other Terms

SD_VBRK, S3VBRKPTS, S3VBRKWRS, S3VBRKDLS, VBFA, VBTYP_V, VBFA-VBTYP_V, r, document flow, TD Transpor, OIGS.

Reason and Prerequisites

Program error.

Prerequisite:
The number ranges of billing documents (number range object RV_BELEG) overlap with those of IS-OIL documents, for example, TD transports (number range object OIG_S).

Solution

Implement the attached correction or upgrade to the corresponding support packages.
