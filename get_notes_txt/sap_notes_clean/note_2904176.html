Symptom

You use report RM06ERVE to change document currency for Contract. In this process, system correctly updates the target value by determining exchange rate using input date for 'Exchange Rate Date'. However, Contract header data still contains old exchange rate determined during Contract creation.

Other Terms

RM06ERVE, EKKO-WKURS, MM06E_EKKO_WAERS_CHANGE

Reason and Prerequisites

Program Error

Solution

Implement attached correction instruction
