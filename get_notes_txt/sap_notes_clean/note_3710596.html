Symptom

CVE-2025-42940 Memory Corruption vulnerability in SAP CommonCryptoLib(CCL)

Environment

  • SAP Adaptive Server Enterprise (ASE) 16.0
  • SAP Adaptive Server Enterprise (ASE) 16.0 for Business Suite
  • SAP Replication Server (SRS)
  • SAP Adaptive Server Enterprise (ASE) 16.0 with Always-on Option (HADR)
  • SAP Adaptive Server Enterprise (ASE) 16.0 for Business Suite with High Availability and Disaster Recovery (HADR)
  • SAP Adaptive Server Enterprise (ASE) Software Developer Kit (SDK) 16.0

Resolution

CVE-2025-42940 in CCL impacts a feature that is not supported in ASE/SRS/SDK. In other words ASE/SRS/SDK SSL connection does not use the impacted feature of the CCL library.
So ASE/SRS/SDK are NOT impacted by this vulnerability.

If your compliance/audit policy recommends to not use the impacted CCL version at all, please consider to replace the vulnerable version of CCL in their current installation as per 3376440 - SYB: Replacement of a vulnerable version of CCL

Additionally, with regard to SAP ASE, SAP ASE 16.0 SP04 PL08 includes CCL 8.5.60, which addresses CVE‑2025‑42940.
（SAP ASE 16.0 SP04 PL07 HF1 includes CCL 8.5.52.）
