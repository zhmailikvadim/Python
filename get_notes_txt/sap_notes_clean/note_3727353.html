Symptom

The Title field in the Supplier Invoice Advance app allows a maximum length of 50 characters.

The system does not support exceeding the 50-character limit for the Title field.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Access the "Create Supplier Invoice - Advanced" app.
  2. Navigate to the header information section.
  3. Attempt to enter a title exceeding 50 characters in the Title field.
  4. Observe that the Title field does not allow more than 50 characters.

Cause

In S/4HANA Cloud Public Edition, the length and data type of standard fields are defined in the system data model and cannot be modified through configuration or system settings.

Resolution

The standard Title field's character limit cannot be increased due to system data model restrictions. If additional characters are required, use the "Custom Fields and Logic" app to create a custom field.

Configure the custom field to meet the character length requirements and make the custom field available in the Supplier Invoice user interface for use.
