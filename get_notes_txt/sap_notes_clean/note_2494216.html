Symptom

With ABAP development tools for SAP NetWeaver you can discover a modern ABAP development in the well-known Eclipse environment. You capitalize on the open Eclipse platform with its usability, speed and flexibility. At the same time, you still benefit from the proven ABAP Workbench features such as the tight integration with SAP's powerful application server and convenient lifecycle management. We let you take advantage of the best of two worlds to increase your developer productivity!

Other Terms

Business Benefits

  • Be ready for the future: ABAP Development Tools for SAP NetWeaver tie in perfectly with SAP HANA Studio and SAP's in-memory technology by allowing highly productive application development on top of SAP HANA.
  • Increase developer productivity with rich Eclipse user experience and flexibility, new capabilities for sophisticated source code implementation, task-oriented and test-driven business development.
  • Enable cross-platform application development by integrating ABAP and non-ABAP development tools into your open and powerful IDE
  • Benefit from the huge Eclipse ecosystem offering a wide range of development and productivity tools and even contribute to this vastly growing community.


Licensing of ABAP Development Tools for SAP NetWeaver is included in SAP NetWeaver developer license.

Solution

"ABAP development tools for SAP NetWeaver" are available with "SAP enhancement package 3 for SAP NetWeaver 7.0", i.e. from SAP NetWeaver 7.31 SP04, 7.40, 750 and 7.51

Download the Eclipse platform separately from Eclipse.org. For ABAP development the "Eclipse IDE for Java Developers" package is sufficient (other packages to your needs), versions 4.7.x (Oxygen) and 4.6.x (Neon) are supported.

The easiest way to install the ABAP Development Tools is to use SAPs update sites described/linked on https://tools.hana.ondemand.com/#abap. They host the latest available version of the tools.

Alternatively you can download ABAP Development Tools for SAP NetWeaver from SAP's software marketplace https://support.sap.com/swdc

Please visit also SCN and the developer community http://scn.sap.com/community/abap/eclipse for more information and to interact with other developers.
