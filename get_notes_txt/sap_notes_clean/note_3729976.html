Symptom

In My Timesheet (S/4HANA/Version 4) application:

  1. The default values set for attributes did not appear for a new record in Form Entry.
  2. The text of default values did not appear during the creation of an assignment.
  3. When a record is deleted and the delete action is subsequently canceled, the record is still removed upon submission of other changes.

Other Terms

F3074A, My Timesheet V4, Delete action, Cancel action, Horizontal view, Vertical view, Attributes, Form entry, Default values, Create Assignment

Reason and Prerequisites

  1. Default values for Form entry was not handled. 
  2. Text was not retrieved for the default values
  3. Deletion of records is captured in a separate list, which is not handled in case the Cancel activity is performed right after the records were deleted.

Solution

Please implement the automatic correction instruction attached to this SAP Note or alternatively you can implement the corresponding support pack.
