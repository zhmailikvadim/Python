Symptom

  • Noticing an additional tab 'External Schema Tables' in the Transaction Research Widget in My Incentive Dashboard setup
  • The dropdown list for External Schema Tables is empty.
  • The tab or option is not available when using the Transaction Research Widget from Incentive Management Portal.

Environment

  • SAP Sales Performance Management
  • Incentive Management on Hyperscalers

Reproducing the Issue

  1. From the Application dropdown select 'My Incentive Dashboard Setup'
  2. Create a new dashboard for title or plan. 
  3. Create a Configure Transaction Research widget. 
  4. On the tab 'External Schema Tables', observe that the dropdown list is not populated with any details.

Cause

  • The feature is incorrectly enabled by default in the My Incentive Dashboard setup.
  • Enabling the feature in Incentive Management Portal requires activating a FeatureFlag. 

Resolution

  1. As part of the 2605 release, the tab 'External Schema Tables' in the Transaction Research Widget in My Incentive Dashboard setup will be disabled by default.
  2. Refer to the release schedule -  SAP SPM 2026 Release Schedule.
  3. The feature will be available upon request after the 2605 release.
  4. For more details on the widget, refer to Using External Schema Tables in Transaction Research Widget | SAP Help Portal.
  5. To enable the feature, refer to KBA 3729025: How to enable external schema tables in the transaction research widget for SAP Incentive Management.

See Also

  • KBA 3729025: How to enable external schema tables in the transaction research widget for SAP Incentive Management.
  • Refer to Using External Schema Tables in Transaction Research Widget | SAP Help Portal.
