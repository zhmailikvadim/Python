Symptom

You use a Freight Order with Combination Resource. On the Term and Conditions tab the incoterms related fields are empty but all the assigned document has the same incoterms settings.

Other Terms

Incoterm Location, Incoterm Location (Free Text), INC_CLASS_CODE, INC_TRANSF_LOC_N, INC_LOC_KEY, INC_LOC_ID

Reason and Prerequisites

Program error

Solution

Implement the attached correction instructions or apply the corresponding Support Package.
