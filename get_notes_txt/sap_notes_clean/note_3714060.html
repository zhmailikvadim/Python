Symptom

You are running a WD ABAP application and open a popup. Either while opening the popup the focus is not set correctly or while working on the popup you are opening a second popup (e.g. as confirmation popup) and close it again and the resulting focus on the first popup is not set correctly.

Reason and Prerequisites

Program error

Solution

Please implement the attached correction instruction or the corresponding support package.

Caution:
It is recommended to implement the corresponding support package as a cascade of pre-requistes, i.e. several notes of different support packages, are to be implemented as well. To avoid problems while implementing all the needed pre-requistes it is recommended to implement them all together by implementing the support package containing the most recent note (see as well SAP note 3697097 - Collection note "FocusHandling").
