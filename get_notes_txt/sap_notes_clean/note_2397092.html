Symptom

Given that you have configured the use of payroll function UPAR1 with the new parameter introduced by SAP Note 2278797, in your custom payroll schema, or via the standard schema UTX0, the following error codes may be displayed without an explanatory text:

  • HRPAYUS_MESSAGES:105
    • This error message occurs when an invalid parameter value is informed for the new parameter in payroll function UPAR1.
  • HRPAYUS_MESSAGES:106
    • This error message occurs when an value has not been set at the UPAR1 parameters.

Other Terms

RPCMASUT_ADD_WH, PRART, prepare_previous_addwh, arrears, PC00_M10_CALC, PC00_M10_CALC_SIMU, RPCALCU0, ADWH, UPAR1, UTX0

Reason and Prerequisites

You are on release 600 of SAP_HRCUS, and you have installed either the SAP Note 2278797, or its respective support package, SP E1.

Solution

As a general rule, SAP recommends that you install a solution by applying a Support Package.
However, if you need to install a solution earlier, apply the manual instructions referenced by this SAP Note.

For release SAP_HRCUS 604, refer to SAP Note 2278797.
