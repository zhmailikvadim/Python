Symptom

When you run Mexico Auxiliary Report of Folios (MX_FOLIOS_AUX) for non-leading ledger, the result doesn't fetch any data.

Environment

  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Open Fiori Launchpad
  2. Execute Run Statutory Reports Apps (F1515)
  3. Select the Mexico Auxiliary Report of Folios (MX_FOLIOS_AUX)
  4. Execute a new run for non leading ledger
  5. No data fetched

Cause

Missing SAP Note 3690423 - DRC - Electronic Accounting: Adjustments to Reporting of Non-Leading Ledger Documents in Journal Entry Details Report and Auxiliary Report of Folios.

Resolution

Implement SAP Note 3690423 - DRC - Electronic Accounting: Adjustments to Reporting of Non-Leading Ledger Documents in Journal Entry Details Report and Auxiliary Report of Folios (with caution to check for prerequisites and manual steps that may be required). Execute a new run after.

Keywords

MEXICO DRC REPORT, NON-LEADING LEDGER, STATUTORY REPORTING, S/4HANA, ELECTRONIC ACCOUNTING REPORTS, AUXILIARY REPORT OF FOLIOS, MX_FOLIOS_AUX, SECONDARY LEDGER, ACDOCA, BSEG.
