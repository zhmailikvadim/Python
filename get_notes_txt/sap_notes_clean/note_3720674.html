Symptom

When trying to access SSCUI/Configuration Activity "Assign Warehouse Number to Plant and Storage Location" ID 101109, the following error message is raised: 

No authorization to start the UI for transaction S_ALR_87007419

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Navigate to the CBC workspace.
  2. Click on the "Configuration Activities" tab.
  3. Search for the Configuration Activity ID 101109.
  4. Click on the entry for the activity.
  5. Click the "Go to Activity" button.
  6. Observe the error message: "No Authorization to start the UI for transaction S_ALR_87007419".

Cause

Missing Business Catalog.

Resolution

    Add the business catalog SAP_CA_BC_IC_LND_LE_DWM0_PC to the BPC_EXPERT business role (or the similar role in use).

    Additionally, to have editing rights, please ensure to grant "Write Access" to "Unrestricted" for the BPC_EXPERT business role (or for the similar role in use).

    The business catalog SAP_CA_BC_IC_LND_LE_DWM0_PC is only available in S/4HANA Cloud if one of the below scope items is activated:

    • 1ZQ

    • 2RU

    • 3BR

    • 3BS

    See Also
