Symptom

  • You run the DRC report HU_VAT_DCL. 
  • You want to display documents in embedded analytics under Analyze Data.
  • The numbers in the XML output do not match the numbers in Embedded Analytics.

Other Terms

Hungary, VAT reporting, DRC, HU_VAT_DCL, Analyze Data, XML

Reason and Prerequisites

Both the XML output and the corresponding Embedded Analytics display the information that is in the system at the moment when you run the report.

Solution

After implementing this SAP Note, the system displays matching numbers in the XML output and in Embedded Analytics.
