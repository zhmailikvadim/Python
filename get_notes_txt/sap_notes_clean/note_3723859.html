Symptom

Error message: A System Error has Occurred. If the problem persists, please contact the administrator.

Environment

  • SAP Sales Performance Management
  • Advanced Workflow

Reproducing the Issue

This error message can happen in any action performed in Advanced Workflow.

Cause

To investigate the cause of this error message, a user with administrator privileges must check in the System Logs, Script execution logs or Debug Tool to get the true error message. "A system error has occurred" is a generic placeholder message that hides the true error error message to end users.

Resolution

  1. Reproduce the action that throws the error
  2. Search in the system logs for the timestamp of when the error occurred as described in:
    SAP For Me: 3725557 - How to Search the System Logs for Timestamps - SAP for Me 
  3. Click on the link of the "PID" column to see the related script execution log. 
  4. Now in the script execution logs tab, click on the "PID" column again to analyze the execution of the process. "Parent PID" may be clicked instead to show the parent scripts.
  5. Each node can be visualized individually for more information.
