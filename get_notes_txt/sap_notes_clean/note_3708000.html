Symptom

When using PurchasingDocumentDeletionCode via API_PURCHASEORDER_2 to set deletion indicator in PO item, the item cannot be deleted.

Environment

  • Materials Management (MM)
  • SAP S/4HANA 2023
  • SAP S/4HANA 2025

Reproducing the Issue

  1. Open /n/iwfnd/gw_client.
  2. Use HTTP Method PATCH and input below into URI:
    /sap/opu/odata4/sap/api_purchaseorder_2/srvd_a2x/sap/purchaseorder/0001/PurchaseOrderItem/{PurchaseOrder}/{PurchaseOrderItem}
  3. Execute and check the result.

Cause

PurchasingDocumentDeletionCode is Read-only for internal processing. It cannot be used to delete item. In SAP Business Accelerator Hub, PurchasingDocumentDeletionCode is not available in PATCH under PurchaseOrderItem. 

Resolution

To delete the item, use HTTP Method DELETE and URI /sap/opu/odata4/sap/api_purchaseorder_2/srvd_a2x/sap/purchaseorder/0001/PurchaseOrderItem/{PurchaseOrder}/{PurchaseOrderItem}. 

See Also

  1. Additional information about PurchasingDocumentDeletionCode:
    When using HTTP Method GET, system will return PO item without deletion indicator only, which means that PurchasingDocumentDeletionCode in the Response is always blank.
