Symptom

  • On the payslip (custom form Z*PAYSLIP_AU and standard SAP_PAYSLIP_AU), the employee address displays:

    • An additional country line, which is not expected for Australian payslips.

    • The country name appears in a foreign language (e.g., “AUSTRALIEN” instead of “Australia”).

  • Issue occurs for specific employees (e.g., PERNR X) but not for others (e.g., PERNR Y) using the same form and variant.

  • Program involved: H99_HRFORMS_CALL

Cause

  • Country line added due to sender vs. receiver country mismatch

    • The system determines:

      • Sender country from the company code (BUKRS)

      • Receiver country from the employee address

    • If both countries differ:

      • The system automatically adds a country line to the address.

    • Example:

      • PERNR X → Company code CH02 (Switzerland) → Sender country = CH

      • Employee address → Australia (AU)

      • Result: Country line is added

    • In contrast:

      • PERNR Y → Company code 1300 (Australia) → Sender = Receiver

      • Result: No country line added

  • Country name displayed in foreign language

    • Country text is retrieved from table T005T

    • Based on:

      • SPRAS = destination country

      • LAND1 = language of sender country

    • For PERNR X:

      • Sender language = D (German)

      • Country = AU

      • Result from T005T = “Australien”

Resolution

  • The behavior is standard system logic (working as designed).

  • To achieve the expected payslip format (no country line, English country name):

    • Adjust the configuration of the employee/company assignment, for example:

      • Align company code country with the employee’s country (if applicable), or

      • Review cross-country employee setup (e.g., global transfer scenarios)

    • Alternatively, review:

      • Address formatting logic in HRFORMS

      • Country/language settings impacting table T005T

  • As this is a configuration/design topic, it is recommended to:

    • Engage your SAP HCM/Payroll consulting team

    • Review business requirements for global transfers and payslip formatting
