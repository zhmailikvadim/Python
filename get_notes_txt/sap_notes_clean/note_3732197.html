Symptom

This SAP Note provides you with corrections on the issue that unexpected CPF paid-in-error wage types generated after you implemented the SAP Note 3674622 and created BAdI implementation for BAdI method CUSTOM_ADJUST_OW.

Other Terms

CPF Paid-in-Error, BAdI HRPAYSG_ADJUST_OW, SAP Note 3674622

Reason and Prerequisites

This is a program error.

Solution

You could apply HR Support Package or implement instructions in this SAP Note.
