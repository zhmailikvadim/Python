Symptom

When recording the results of an Inspection lot and after entering all the inspection characteristics, the proposal valuation will not show up.

Other Terms

  • SAP Asset Manager
  • SAP Service and Asset Manager
  • SAM
  • SSAM
  • Inspection Points
  • Record Results
  • Characteristics
  • Inspection lost
  • Valuation

Reason and Prerequisites

Reason:

Word order information from the operation was not correctly defined at the binding. This information was required to show the valuation page

Prerequisites:

  • BAS or Visual Studio Code with SAM MDK project
  • `patch` command (available on BAS and macOS by default) via terminal or other patching utility

Solution

The solution was to get the work order information from the current binding by default, but if not present, check the operation level read link.

Download the attached 3507838.patch file and save it in a known location, preferably adjacent to your SAM.mdkproject

  1. Open a terminal in the root of your SAM metadata (i.e. SAM.mdkproject/metadata or SAM.mdkproject/metadata/base.definitions)
  2. Run the command `patch --no-backup-if-mismatch -p4 < path/to/3507838.patch` in the terminal. 
  3. Verify the following files has been modified.
  • patching file 'Pages/WorkOrders/Operations/WorkOrderOperationDetails.page'
  • patching file 'Rules/InspectionCharacteristics/Update/InspectionCharacteristicsChangeSetOnSuccess.js'
  • patching file 'Rules/InspectionCharacteristics/Update/InspectionCharacteristicsChangeSetOnSuccessEDT.js'
  • patching file 'Rules/WorkOrders/Operations/InspectionPoints/InspectionPointCodeWithRead.js'
