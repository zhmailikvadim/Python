Symptom

  • In SAP Cloud for Customer, changing the opportunity status to "Won" automatically updates the probability to 100%, and setting it to "Lost" updates the probability to 0%.
  • In SAP Sales Cloud Version 2, this automatic update behaviour is not functioning as expected.

Environment

  • SAP Sales Cloud Version 2
 



Reproducing the Issue

  1. Go to Navigation Menu.
  2. Go to Guided Selling.
  3. Open opportunity ID 43.
  4. Change the status to "Won".
  5. Verify that the probability is not updated to 100%.

Cause

Some of the functional dependencies in SAP Cloud for Customer are not mapped directly to SAP Sales Cloud Version 2. The automatic update of probability based on opportunity status is not part of the standard functionality in SAP Sales Cloud Version 2. The probability field is controlled by Sales phase field.

Resolution

This is the current system behaviour.

See Also

  • Refer to: KBA 1925772 - Custom Status Dependency in SAP Sales Cloud Version 1.
