Symptom

When you processing classification of class type 017 in CL30N, there is dump SYSTEM_ABAP_ACCESS_DENIED:

Category                       ABAP programming error
Runtimes Errors             SYSTEM_ABAP_ACCESS_DENIED
ABAP: Program              SAPLCLSD
Application Component   CA-CL-CL

You found this is because of calling of T-Code CV03.

Environment

  • SAP ERP Logistics (LO)
  • SAP ERP Classification (CA)
  • SAP S/4HANA
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA Cloud Private Edition

Cause

Incorrect customizing in O1CL for class type 017.

Resolution

Following below steps:

  1. Go to T-Code O1CL
  2. Find table DRAW and double click 'Class types' in the tree
  3. Choose class type 017 and double click 'Functions/Filters for Finding Objects'
  4. Check if there is only one entry maintained with 'Transaction/report = CV03N'.

If there is other line with T-Code CV03 maintained, please remove the line.

Keywords

SAPLCLSD, CALL_FUNCTION, LCLSDF08, P_TCODE, CV03
