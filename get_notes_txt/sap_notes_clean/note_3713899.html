Symptom

XML tag PrefixPodatnika (Taxpayer's Prefix) not limited to EU only

ODN Integration for the Self Billing Invoice (only for S4HANA OnPrem)

Updates for the VAT Reporting Currency - When Plants Abroad is marked as not active.

Other Terms

Poland, eDocument, eDoc, eInvoicing, electronic invoice, electronic invoicing, Poland eInvoice, Document Compliance, Digital Compliance, B2B, KSeF, Electronic Document, Document and Reporting Compliance, DRC, FA(3), FA(3) Schema Version, PrefixPodatnika

Reason and Prerequisites

  • Implement the SAP Note 3711827 Poland KSEF FA( 3 ) : VAT rate ( P_12Z tag ) calculation fix
  • Implement the SAP Note 3713943 Electronic Document Processing for Poland - KSeF: FA (3) Correction of fixed values for EDO_PL_TAX_RATE_CODEV2
  • Implement the SAP Note 3712301 Allow U-type ODNs in Self-Billing Invoice via API
  • For a complete prerequisite list, refer the SAP Note 3279206 (Electronic Document Processing for Poland - KSeF: Implementation Overview).
