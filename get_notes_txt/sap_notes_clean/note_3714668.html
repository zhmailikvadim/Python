Symptom

  • The total amount is not displayed in the exported Excel and PDF files from the “Manage Customer Line Items” and “Manage Supplier Line Items” apps.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Export data from the “Manage Customer Line Items” or “Manage Supplier Line Items” apps using the Excel or PDF export functionality.
  2. Open the exported file and observe that the total amount is not included.

Cause

  • The Excel export functionality in Fiori is designed to export only the data visible in the list or table. The `Total Amount` is calculated through a separate data request, which is not included in the export process.

Resolution

  1. Manually add the total in the exported Excel file using the `Subtotal` function available under the `Data` tab in Excel.
  2. Submit a feature request via the SAP Customer Influence website to propose enhancements for including the total amount in exported files. Refer to KBA 0002963059 for detailed instructions on submitting feature requests for SAP S/4HANA Cloud Product.
