Symptom

If regressions are known for kernel 781 patch level 210, they are listed in this SAP Note in the "References" section.

This list is updated as soon as new regressions are discovered. Please subscribe to this note to receive update notifications.

Other Terms

Kernel, error list, disp+work, KRNL781PL210, regression

Reason and Prerequisites

Regressions have been discovered for kernel 781 patch level 210.

Regression is an error that was caused as an unwanted side effect of a previous kernel patch.

The known regressions are listed below in the "References" section.

Solution

Review this regression note before installing the kernel patch. Examine the individual SAP Notes listed in the "References" section, in order to determine:

  • What regressions are currently known for kernel 781 PL 210
  • Whether a workaround exists to avoid the specific regression
  • Whether a new kernel PL is available to fix the specific regression

There is a regression note for each released kernel PL (for further information see SAP Note 1802333).

The list of all changes (error corrections and improvements) is available in the "Content Info" file of the respective kernel archive in the SAP Support Portal.
