Symptom

When attempting to add rule data to Agreements through the mass processing functionality in the Agreement Workbench, the system does not allow the rule data to be added if partner data already exists in the Agreements. The system is throwing an duplicate error.

Other Terms

/IRM/IPAGRA, data_tables_prepare, Rules Mass Change, DUPREC:POS&/IRM/IPGSP&/IRM/S_IPAPRCD

Reason and Prerequisites

This is caused by program error.
Note : SAP Note#3724215 is available for ECC release.

Solution

Please implement the correction instructions. These corrections are only valid if you have applied the following Support Packs.

Release V/4 – 2023 V2, 2023 V3 and Support Pack 1

Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object
