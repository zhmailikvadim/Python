Symptom

The tooltip texts defined in the CUA Status of function group SALV_FUNCTIONS_STATUS have been adapted to remove the abbreviation, as following:

Function code   Current tooltip text    New tooltip text after implementing this note
&OUP Sortieren aufsteig. Sortieren aufsteigend
&SAL Alle Mark. löschen Alle Markierungen löschen

These adjustments are purely textual and do not affect functionality.

The updated tooltips now display complete and grammatically correct German phrases, providing a clearer user experience.

Note that the changes from this Note are made in the standard CUA Status: SALV_FUNCTIONS_STATUS only.
If any application copied this CUA Status before the changes from this note, it will be necessary that the application adapts its copy, as this change is not carried over to copied objects.

Other Terms

ALV; ABAP List Viewer; SAP List Viewer; Grid; List; SALV_FUNCTIONS_STATUS; PF-STATUS;

Reason and Prerequisites

Reason

This adjustment is made to improve readability and ensure that the tooltip texts are displayed in a complete and clearly understandable form.

When you install an SAP Note, some prerequisites might be required. The prerequisites are SAP Notes that have already been released and that change the same context of the current one. Then, even if these prerequisites are not necessarily related to the symptom of this SAP Note, you must implement them due to technical reasons.

Solution

From now on, the tooltip text is no longer abbreviated in German.
