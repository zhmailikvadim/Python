Symptom

When executing the Agreement Posting Report, the amounts displayed in the Periods view appear correctly during the first navigation. However, when the user exits and opens the Periods view again, the accrual amounts are displayed as doubled, which can mislead the user.

This SAP Note provides the necessary correction to ensure that the accrual amounts are displayed consistently and accurately in the Periods view.

Other Terms

Posting Report, Accrual Amount, /IRM/IPCRAGP, Agreement

Reason and Prerequisites

This is caused by program error

Solution

Please implement the correction instructions attached. These corrections are only valid if you have applied the Support Packs.

Release V/4 - 2023 V2

 Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object.
