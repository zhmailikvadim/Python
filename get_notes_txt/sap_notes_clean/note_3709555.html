Symptom

Due to ongoing specifications of new APD files from authority side, new changes are needed.

Other Terms

APD, APD 2025, HGRUAPD0, HGRUAPD4, IKA, EFKA

Reason and Prerequisites

This SAP Note is relevant only for Greece.

Solution

Multiple corrections and changes have been done to APD reports based on EFKA feedback.

Maintenance view for T7GR4APD has been corrected to save the changes of field APD_DPY_A39.

Please make sure to check customizing of main pension SI Branches ("κλάδος κύριας σύνταξης"). In table T7GR3SIB (Social Insurance Branches) field SIBID (Social Insurance Branch Indicator) must be 04 for main pension SI branches.

Implementation:

  • As a general rule, SAP recommends that you install a solution by applying a Support Package. However, if you need to install a solution earlier, use the Note Assistant to implement the correction instruction. You can find more information about the Note Assistant in SAP Service Marketplace, under service.sap.com/note-assistant.
  • Please, note that documentation, transactions, and Customizing activities are only delivered via Support Package.
