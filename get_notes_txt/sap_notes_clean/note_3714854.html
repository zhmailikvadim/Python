Symptom

  • Error AC760 encountered: "Closed fiscal years cannot be reset".
  • Issue arises during migration from ECC to S4 HANA when attempting to change fiscal year for company code.
  • Error occurs while running a program to populate values in a newly created depreciation area.

Environment

  • Contract and Lease Management
  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA Finance
  • SAP S/4HANA

Reproducing the Issue

  1. Create a new depreciation area;
  2. Run the program to populate values in the new deprecation area;
  3. Try to adjust the Last Closed Fiscal Year trough OAAQ;
  4. Encounter error AC760: "Closed fiscal years cannot be reset".

Cause

  • Closed fiscal years cannot be reset due to standard system process restrictions.

Resolution

Due to standard process it is not possible to change it manually;

  1. Create a small ABAP program to update the fiscal year to the same as other areas.
  2. Refer to SAP Note 3120761 for guidance on creating the ABAP program using z_t093b_update_abgja.txt as a reference.

See Also

Note 3120761 - FAA_CMP Initial Closed Fiscal Years Check
