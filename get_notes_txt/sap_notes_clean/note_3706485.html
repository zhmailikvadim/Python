Symptom

Custom conditional field is not displaying on time off workflow details page, but visible when checking via Time off Workbench.

Environment

SAP SuccessFactors Employee Central 2H 2025

Reproducing the Issue

  1. A custom conditional field is configured based on fractionQuantity field on Employee Time object, such as cust_AMPM.
  2. User applied a leave request and filled in value for this custom field.
  3. On workflow details page, this field is not visible (Respect Permission in wfconfig is set to Yes)
  4. When checking via Timeoff Workbench, the custom field is visible.

Note: When Respect Permission in wfconfig is set to No, the field is visible on workflow details page.

Resolution

The fix will be available via 1H2026 release.
