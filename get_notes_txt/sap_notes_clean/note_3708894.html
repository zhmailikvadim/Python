Symptom

   Flexeraaxf$aaa: Windows DLL failed to load
      at Flexeraaxf.af(Unknown Source)
      at Flexeraaxf.aa(Unknown Source)
      at com.zerog.ia.installer.LifeCycleManager.init(Unknown Source)
      at com.zerog.ia.installer.LifeCycleManager.executeApplication(Unknown Source)
      at com.zerog.ia.installer.Main.main(Unknown Source)
      at sun.reflect.NativeMethodAccessorImpl.invoke0(Native Method)
      at sun.reflect.NativeMethodAccessorImpl.invoke(NativeMethodAccessorImpl.java:62)
      at sun.reflect.DelegatingMethodAccessorImpl.invoke(DelegatingMethodAccessorImpl.java:43)
      at java.lang.reflect.Method.invoke(Method.java:498)
      at com.zerog.lax.LAX.launch(Unknown Source)
      at com.zerog.lax.LAX.main(Unknown Source)

Environment

  • Sybase Software Asset Management (SySAM) 2.5.0
  • Microsoft Windows Server 2025

Reproducing the Issue

Execute SySAM_Suite\setup.exe.

Cause

The installation process fails due to compatibility issues with the Java environment and the operating system version.

Resolution

  1. In the Command Prompt, navigate to the folder containing the setup file.
  2. Set the Java environment variable by typing the following command exactly:
       SET JAVA_TOOL_OPTIONS="-Dos.name=Windows 2022"
  3. Press Enter to execute the command.
  4. Execute the SYSAM 2.5.0 setup file.
