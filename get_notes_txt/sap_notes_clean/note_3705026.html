Symptom

In the process "Sending of Request for Standard Load Profile or Profile Sets (SUP View)" (DE_B_EQS_SUBSCR_SLP_SEND_SUP), the system displays an error when you try to end a subscription if two or more subscription entries exist with the same starting date and one of them is already ended.

This issue is relevant for the market role: SUP.

Other Terms

MaBiS, EQS, Settlement, SLP, Standard Load Profile, Profile Sets, End of Subscription, subscription entries, starting date

Reason and Prerequisites

The error occurred because of a program error in the subscription management logic.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the system allows you to send the end of subscription without error if there is an active subscription, regardless of other subscription entries.

You need to resend the end of subscription for SLP. You can verify the fix by checking that the subscription is ended without any issue.
