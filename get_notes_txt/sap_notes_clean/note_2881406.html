Symptom

Error MMPUR_SPROC022 - 'Invoice Receipt' for items with Product Type Group 'Service' - is displayed when flagging the Free-Of-Charge flag within non-physical / Service POs.

Environment

  • Materials Management (MM)
  • SAP S/4HANA

Reproducing the Issue

  1. Transaction ME21N: Create PO.
  2. Enter PR number or vendor, material and Org data.
  3. Check / flag the 'Free-Of-Charge' option.
  4. Try to Save, or use the 'Check' option.
  5. Error MMPUR_SPROC022 is returned. 

Cause

The term and logic of 'Free-of-Charge' relates only to shippable / physical items that can be delivered and GR'd, so this does not apply to Service Procurement (as there are no items 'issued' as such).  Due to the nature of the Service PO structure (and the Service Materials being non-physical in nature) , the option to write off / zero items can only be done in later steps (for example, Invoicing, or any process 'after' the PO process).

Resolution

The 'Free-of-Charge' flag / option is only for Material item POs - they are not supported in Service Procurement.
