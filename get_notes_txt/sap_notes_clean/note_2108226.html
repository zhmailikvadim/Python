Symptom

If user enable query cache from backend, refreshing report from Excel with changing some context member may not get right result.

Other Terms

Standard Model, Excel, EPM Addin

Reason and Prerequisites

Program error.

Solution

For SAP Business Planning and Consolidation 10.1, version for SAP Netweaver, install this note or upgrade to (Component CPMBPC 810) support package 06 and upgrade BW740(Component SAP_BW) 11.
