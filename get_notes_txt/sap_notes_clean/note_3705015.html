Symptom

  • The List status popup cannot be closed after it has been opened.
  • Clicking the "Close" or "Continue" buttons have no effect, and the dialog remains on the screen.

Environment

SAP S/4HANA 2023

Reproducing the Issue

  1. Open an ALV grid, which can be switched to List view (for example test report BCALV_TEST_GRID)
  2. On the toolbar click 'List output'.
  3. Then open the List status via 'Settings' -> 'List status'
  4. Observe that the dialog box opens.
  5. Attempt to close the dialog box by clicking the "Close" button.

Cause

Program error, side effect of SAP Note 3633491.

This issue does not happen if the Note above is not implemented.

Resolution

Implement SAP Note 3696507.

See Also

  • SAP Note 3696507 - ALV: List status popup shows wrong function codes in print preview
  • SAP Note 3633491 - ALV: Expand/Collapse buttons missing in print preview

Keywords
