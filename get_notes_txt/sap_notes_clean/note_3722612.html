Symptom

In the developer traces there are misleading messages like
OA2C: Error: Invalid PSE Key Algorithm ..
OA2C: Error: Invalid PSE Key Length ..
from exception class CX_OA2C_CLIENT_AUTH_JWT
raised e.g. from classes CL_OA2C_JWT_SETTINGS or CL_OA2C_JWT_PSE
about PSEs that could not be used for JWT client authentication.

Other Terms

OAuth 2.0

Reason and Prerequisites

The trace entries are erroneously written during a filter step in SOAUTH2_CLIENT.

Solution

Implement the support package or the correction instruction.
