Symptom

This note provides a correction for the 'Function Module Name' mentioned within the exception handling of the FM FSTB_API_RS_ITEM_PREP_CHK . The exception message contains incorrect function module name which is now corrected to accurately reflect the right function module name during error handling.

Other Terms

Restricted Savings, RS, FM, Exception Messages

Reason and Prerequisites

Program Error

Solution

 Import the relevant support package for your release. Use the transaction SNOTE to implement the attached correction instructions in advance.
