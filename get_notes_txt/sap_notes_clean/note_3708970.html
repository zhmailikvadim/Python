Symptom

  • Activation of the action fails with this error  "Group missing action areas".
  • Requests are discarded during activation.
  • The system behavior leads to data discrepancies

Environment

SAP Customer Data Platform

Reproducing the Issue

  1. Configure the action with a specific business area defined in your Businessunit
  2. Attempt to activate the action.
  3. Observe that requests are discarded with the error message "Group missing action areas" in the monitoring section for the action

Cause

The field defined as an area is not associated with the outbound processing purpose within the action, causing the activation to fail.

Resolution

  1. Retrieve the outbound processing purpose from the action application. 
  2. Verify that the field designated as the action area is included within that outbound processing purpose
  3. Try activating again after selecting the field defined as the area in the outbound processing purpose
