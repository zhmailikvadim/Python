Symptom

  • Some objects got "SAP" as value for the column SRCSYSTEM in the table TADIR. But the value was changed to a normal SID by SUM(Software Update Manager).
  • Those objects are belong to LOCAL or HOME component.

Environment

SAP ABAP system.

Reproducing the Issue

Run SAP system upgrade.

Cause

The original value "SAP" was incorrect in the system.

Resolution

This was not an issue but an expected behavior.

The program RSUPG_TADIR_COMPONENT_CHECK was called as a background job by SUM(Software Update Manager) during the upgrade to check the TADIR content.

It fixed the incorrect settings for LOCAL and HOME objects to prevent object loss.

Keywords

TADIR, SRCSYSTEM, SUM, change, fix, LOCAL, HOME, SUM, upgrade
