Symptom

This new feature is part of Next gen sourcing intergration(NEXUS) with Ariba, this feature is specific to Process purchase requisition app (F1048A). When the purchaser tries to create a Request for quotation using F1048A app and if the PR is NEXUS relevant, which is determined by the implementation of BADI ME_MEREQ_EXT_REQ in method is_ext_rfq_active, then the RFQ is sent for external creation via the BADI method create_ext_rfq.

Also When the user clicks on the "Create RFQ" button, if the PR is NEXUS relevant then a UI message "RFQ sent for external creation" is displayed
