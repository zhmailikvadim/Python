Symptom

  • Budget notifications from Availability Control (AVC) are sent to the SAP Business Workplace (transaction SBWP).
  • You want to receive these information-only budget notifications directly in an external email client (e.g., Microsoft Outlook) instead of SBWP.

Environment

  • Controlling (CO)
  • SAP R/3
  • SAP R/3 Enterprise 
  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA Finance
  • SAP S/4HANA

Cause

By default, the mail from Availability Control is designed to only be sent to the SAP Business Workplace (SBWP) of the assigned SAP user.

Resolution

To forward Availability Control notifications to an email address, choose one of the following methods:

Option 1: Set Automatic Forwarding in SBWP

  1. Navigate to the menu path: Settings > Office settings.

  2. Configure the automatic forwarding to route messages to the required external email address.

Option 2: Implement User-Exit

  1. Implement user-exit EXIT_SAPLBPFC_001 to forward the mail to transaction SOST For further details and instructions, refer to SAP Note 429793.

See Also

Refer to: SAP Note 429793 - User exit for mail recipient determination in availability control.
