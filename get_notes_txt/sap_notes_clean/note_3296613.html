Symptom

You cannot see the Actions button at all

Environment

SAP Business ByDesign

Cause

Currently system is using the classic OWL.

Resolution

New SADL setting has to be scoped in Business Configuration.

  • Go to Business configuration work center
  • Go to Implementation projects
  • Click on edit scoping
  • Go to step questions
  • Expand System Management element
  • Select User Inteface List Mode
  • Check mark the question related to SADL
  • Finish the scoping
