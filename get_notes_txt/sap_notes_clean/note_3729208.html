Symptom

This note fix a issue that "The SD billing can still be cancelled if the invoice got cleared before Central Payment activation (Central Payment historical documents) ". The fixing can block the billing cancellation if the SD billing invoice got cleared before Central Payment activation.

Other Terms

Central Finance, Cross-System Process Control, CPAY, CSPC, VF11, Cancel Billing

Reason and Prerequisites

This note deliver the fixing for CPAY part.

Please implement the note 3731404 first.

Solution

Apply the corresponding support package or implement this SAP Note both in your Source systems.
