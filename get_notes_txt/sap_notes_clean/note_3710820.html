Symptom

You need to know how many records are supported inside an Installation Points, under the Items Hierarchy list.

Environment

SAP Cloud for Customer

Reproducing the Issue

  1. Go to Installed Base work center.
  2. Click on Installation Points view.
  3. Select any Installation Point.
  4. Open Items Hierarchy tab.
  5. Number of records is not accurate or they UI is unable to load and shows an error. 

Cause

The Items Hierarchy view for Installation Points has a practical upper limit of approximately 2,000 records per hierarchy load. The exact threshold can vary with system conditions (network, CPU, memory, overall data volume).

Resolution

This is a product limitation from this view. It cannot be lifted or tuned via backend/system parameters.

Keywords
