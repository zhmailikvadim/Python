Symptom

Is it possible to hide specific Internal Candidates from the system?

Environment

SAP SuccessFactors Recruiting Management

Resolution

Currently, it is not possible to hide internal candidates. However, there are settings available to manage the searchability of internal candidates, where searching of internal candidates can be disabled or limit internal candidate search by a date range so only candidates meeting that criteria are searchable. 

Set up Internal Candidate Search

  • Disable searching of internal candidates: This will disable searching for candidates entirely. No internal candidates will return in the candidate search results

  • Find candidates where <select> is <x> days from today or earlier: This will limit the internal candidates search results to only return the candidates if the specified date on their Employee Profile matches the criteria specified in the setting. Note that if no userinfo-element date fields are set up on the succession-data-model, this option will be grayed out and unavailable.
