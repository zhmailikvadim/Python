Symptom

The "Duplicate" option is visible on the Planning Board within Planning and Dispatching in the Dispatching Board of the SAP Field Service Management web application, and there is a requirement to remove or disable this option.

Environment

SAP Field Service Management

Reproducing the Issue

NA

Cause

The availability of the “Duplicate” option is governed by configuration parameters at either:

  • User Policy Group level (permissions), or
  • Company Settings level

Resolution

The "Duplicate" option can be controlled through any one of the following configuration approaches:
  1. Adjust user policy group permissions:
  • Open the relevant User Policy Group.
  • Under Permissions > Activity, set the Create permission to None to remove access to the Duplicate action.

     2. Under Companies, update company settings:

  • In Company Settings, navigate to CoreSystems.CoresystemsFSM.ResourcePlanner.
  • Set the parameter allowOverlappingAssignments to false.
