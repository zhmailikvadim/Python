Environment

SAP NetWeaver 7.5

Reproducing the Issue

Send a new message and make sure that it will fail during the message processing.

Cause

  • The CBMA alerting is not activated, instead of that still the old Message Based Alerting is used.
  • The error mentioned in the Symptom section is also appearing in JAVA only PI installations where this property cannot exist.

Resolution

Setup the following 2 parameters to in NWA in order to use the CBMA alerting:

 - alerting.target    =  1    (1=CBMA, 0=Message Based alerting)
 - alerting.isActive  =  true

These parameters are located in NWA -> Configuration -> Infrastructure -> Java System properties -> Services (TAB) -> XPI Service: AF Core.
