Symptom

  • Trailing spaces in SAP ECC tables are being trimmed by BODS automatically.
  • Is there a setting to preserve trailing spaces when RFC streaming SAP Applications tables?

Environment

SAP Data Services

Cause

The JCO RFC libraries used consider trailing spaces as insignificant filler and are removed as standard behavior when transferring data to optimize message sizes.

Resolution

To preserve trailing spaces, instead of using an SAP Applications datastore to connect to the SAP data, connect directly to the backend database itself with a database datastore type.
