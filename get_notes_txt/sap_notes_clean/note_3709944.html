Symptom

  • An error occurs when creating a transaction using transaction code FTR_CREATE. The error message displayed is: "Partner & not authorized for selected transaction type" (Error ID: T0739).
  • The issue is related to missing data in table `VTBSTA3`.

Environment

  • Treasury and Risk Management (TRM)
  • EA-FINSERV
  • S/4CORE
  • SAP ERP
  • SAP ERP Central Component
  • SAP enhancement package for SAP ERP
  • SAP S/4HANA
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA Finance

Reproducing the Issue

  1. Use transaction code `ftr_create` to create a transaction.
  2. Select Company code, Product type, Transaction type and Partner and press Enter key.
  3. Observe the error message: "Partner not authorized for selected transaction type" (Error ID: T0739).

Cause

Missing authorization for the related product type and transaction type for the business partner in transaction code TBI6 "Authorization - maintain SI".

Resolution

  1. Check transaction code TBI6 "Authorization - maintain SI".
  2. Assign authorization to the related product type and transaction type for the related business partner.

Keywords

Error T0 739, partner not authorized, transaction type, ftr_create, vtbsta3 "SI for Business Partner: Transaction Authorizations", authorization, TBI6, Treasury Management, Function module TB_AUTH_ALLOCATION_DETERMIN.
