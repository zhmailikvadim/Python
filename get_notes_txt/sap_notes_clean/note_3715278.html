Symptom

Non-recurring payments subject to contributions, such as regular remuneration that is paid out as a non-recurring payment, are not indicated correctly in the wage type table of the Employee Wage data record (DSLA). In the field Indicator SI (KENNZSV), the value 5 is not entered for non-recurring remuneration subject to contributions.

Other Terms

Social insurance, euBP

Reason and Prerequisites

Program error

Solution

Implement the attached correction instructions or import the specified support package. For the software release SAP_HRCDE 604, refer to the manual post-implementation steps. Apart from this, you do not have to perform any further follow-on activities after you import the HR Support Package or implement the correction instructions.

Wage types that have one of the following statement wage types in the Wage Type Valuation table (T512W) are now flagged as non-recurring remuneration subject to contributions:

  • Gross Contrib. NrP Remaining (/3N4)
  • Gross Contr. NrP Inflow (/3NC)
  • Reg. EG as NRP Inflow Pr. (/3W4)
  • Reg. Remun. as NRP (/3WB)
  • Reg. EC as NRP Correction (/3WC)
  • Outflow: Val. Val. Inf. Pr. Corr. (/3WD)

In addition, the wage types must be coded in one of the cumulations 03, 05, or 75.
