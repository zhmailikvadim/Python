Symptom

Requirement: UX Improvements in Fiori Apps in S/4HANA.

This SAP Note contains the backend code related to "Manage Change Record"(F2097) application.

Other Terms

PLM-CR, UX 3.0, List Report Table, Object Page Table, Context Menu, Column Width header, UX Improvements in Fiori Apps in S/4HANA (2), 3728118.

Reason and Prerequisites

Standard FIORI Recommendations.

This SAP Note contains backend changes.

Continue with the implementation of the UI corrections from SAP Note 3728118.

Solution

Use transaction SNOTE to implement the corrections delivered with this SAP Note in the affected Change Record system or import the relevant Support Package.

Benefit:

After implementing the corrections from both the SAP Notes, the business users can get the below mentioned benefits:

  • Sort functionality for the attribute "Description" will be available for all columns that display both "ID" and "Description" together.
  • Table column header description will be expanded for tables that contain lesser than six columns.
  • Context Menu actions will be executed correctly based on the selected line item (row item).
