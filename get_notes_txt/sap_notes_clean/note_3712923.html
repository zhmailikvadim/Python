Symptom

A query using ATOV_ARS_CLOUD_CUSTOM_DEV_OBJ returns incorrect values for key user items in the elements ORIGIN and/or SOFTWARE_COMPONENT - even though the SAP notes 3706047 and 3593439 have been applied to the system.

Other Terms

Key User Items

Reason and Prerequisites

CDS view ATOV_ARS_CLOUD_CUSTOM_DEV_OBJ returns incorrect results due to incorrect or missing filter conditions.

Solution

Apply the correction instruction or install the respective support package.
