Symptom

  • Trying to release a transport 
  • Getting an error when trying to release:
    Request <TR ID> does not have mandatory attribute CALM_CDM_FEATURE_ID
  • Want to turn off the check that is causing the CALM_CDM_FEATURE_ID issue
  • Cannot release transports because of CALM_CDM_FEATURE_ID error

Environment

  • SAP Cloud ALM
  • SAP S/4 HANA Private Cloud 2023

Reproducing the Issue

  1. On managed system, launch se09
  2. Display TR <TR ID>
  3. Click the transport request ID to highlight it
  4. Click Truck icon (Release) and encounter error:
    Request <TR ID> does not have mandatory attribute CALM_CDM_FEATURE_ID

Cause

The development system for the Transport Request is configured to require the attribute CALM_CDM_FEATURE_ID.

Resolution

On the managed system:

  1. Launch transaction SE03
  2. Click Administration > Display/Change Request Attributes and click Execute:
  3. Ensure CALM_CDM_FEATURE_ID is not set as mandatory:

Keywords

CALM, CDM, Features, TR
