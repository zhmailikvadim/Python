Symptom

When accessing the Data Services Mangement Console, if manually change the value in some of the cookie, some possible XSS issue may occur.

Other Terms

The issue is tracked by CM: 002075125800000532802026.

Environment:

  • SAP Data Services 2025 earlier than SP 0 Patch 3.

Steps:

  1. Logon the Data Serivces Management Console. 
  2. In the browser developer tool, change the value of cookie items, some XSS related issue may occur.

Reason and Prerequisites

The root cause is the xss issue in the cookie value handling.

Solution

There is no workaround in this issue, please upgrade to the SAP Data Services 2025 SP0 Patch 3 for official update.
