Symptom

With the execution of transaction CK24, it's found that the default fiscal year and period are not expected.

Environment

  • Controlling (CO)
  • SAP ERP 
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA
  • SAP S/4HANA Finance

Reproducing the Issue

  1. Execute transaction CK24.

Cause

The system does not get the default fiscal year and period from the fiscal year variant. Instead, it will get it from the system date SY-DATLO.

Resolution

In case you don't want this default behavior, you can set the values in the user parameters GEJ ( for fiscal year) and BUP ( for period) in transaction SU3.
