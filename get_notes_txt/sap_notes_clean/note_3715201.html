Symptom

After executing the 'Employment Tax (IRPF) reports' program (RPC190E0) the warning message "Tax expenses without payments or deductions; correct with IT 0090" is displayed.

Environment

  • Human Capital Management (HCM)
  • SAP R/3
  • SAP ERP Core Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA

Reproducing the Issue

  1. Generate any tax model.
  2. The output log displays the warning message "Tax expenses without payments or deductions; correct with IT 0090".
  3. This situation usually occurs when there are deductible expenses arrears.

Cause

The Spanish Tax Agency does not admit records with only tax expenses and without payments or deductions. Therefore, the program implements a check and displays a warning message in the case this situation occurs:

FORM correct_gastos_sin_reten CHANGING
                                p_tab_main LIKE main_tab[].

  DATA wa_main LIKE main_tab.
  DATA wa_main2 LIKE main_tab.
  DATA lv_idx_tabmain TYPE syindex.

  FIELD-SYMBOLS <ls_tab_main> TYPE pesw0_190_0002_line.

  LOOP AT p_tab_main INTO wa_main
                     WHERE cotiz <> 0
                       AND per_int = 0
                       AND valor    = 0
                       AND ret_prac        = 0
                       AND benefits_amount = 0.
*  Existen gastos sin retenciones, se intenta sumar estos gastos a un
*  registro que admita gastos y tenga retenciones
    lv_idx_tabmain = sy-tabix.
    LOOP AT p_tab_main ASSIGNING <ls_tab_main>
                         WHERE ( cper1 CA 'ABCD'
                            OR ( cper1 = 'L'
                           AND ( cper2 = '05' OR cper2 = '10' ) ) )
                           AND ( per_int <> 0
                              OR valor    <> 0
                              OR ret_prac <> 0 )
                           AND ejdev = wa_main-ejdev.
      DELETE p_tab_main INDEX lv_idx_tabmain.
      <ls_tab_main>-cotiz = <ls_tab_main>-cotiz + wa_main-cotiz.
      EXIT.
    ENDLOOP.
    IF sy-subrc <> 0.

      PERFORM warning_message USING
                                pernr-pernr
                                text-e25.

    ENDIF.

  ENDLOOP.

ENDFORM.                    "correct_gastos_sin_reten

Resolution

Correct the tax expenses using infotype 'Additional Income E' (0090). 

Note: If you do not know the amount of the tax expenses to be corrected, you may set a breakpoint at FORM correct_gastos_sin_reten and check the content of the p_tab_main table looking for an entry with any amount in field COTIZ and no amount in fields PER_INT and RET_PRAC.
