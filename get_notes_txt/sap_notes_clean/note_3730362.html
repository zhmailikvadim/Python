Symptom

Procedure of enhancing BAdI AC_DOCUMENT for suppress updates on tables ACCT* according to SAP note 1281616 - Enhancing BAdI AC_DOCUMENT

Reproducing the Issue

Cause

The procedure is not known.

Resolution

  1. Execute transaction SE18
  2. Enter BAdi Name AC_DOCUMENT
  3. Press icon Display  (F7)
  4. Navigate over the menu Implementation → Create
  5. Enter any Implementation Name in customer name space Z* for example Z_SKIP_ACCT
  6. Double click IS_SUPPRESSED_ACCT
  7. Press icon Display <-> Change (Ctrl+F1)
  8. Enter result = abap_true. between method and endmethod as follows
    method IF_EX_AC_DOCUMENT~IS_SUPPRESSED_ACCT.

     result = abap_true.

    endmethod.
  9. Press icon Activate  (Ctrl+F3)

  • refer to: SAP Note 1281616 - Enhancing BAdI AC_DOCUMENT
