Symptom

The lines of regularizations dedicated to PAS – Prélèvement à la source / Withholding Tax in the pay slip, are not correctly printed because no changes are detected in RPCEDTF0. Those lines are deleted by RPCEDTF0 if, for a given wage type, the first regularization line has no difference (all fields are equal to 0.00).

Other Terms

RPCEDTF0, payslip, s56, regularization.

Reason and Prerequisites

This is program error.

Solution

If there are several regularizations, not only the first one is checked for differences.

This SAP Note will be available via HR Support Package, in more urgent cases install the available correction instructions.
