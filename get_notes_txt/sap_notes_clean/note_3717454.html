Symptom

You are looking for a way to show user name (password) in SAP Datasphere at connection setup, but this feature is not currently available.
You'd like to understand, how to access that suppressed information, but this functionality is not currently supported in SAP Datasphere. 

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Environment

SAP Datasphere

Reproducing the Issue

  1. Navigate to Connections from the left-side navigation menu
  2. Select your system and click Edit to access properties
  3. Scroll down to the "Credentials (User Name and Password)" section.
  4. The User and pw is not displayed.

Cause

User credentials is stored encrypted in the Database, as a result, it cannot be displayed as plain text in connection.

Resolution

This is a feature request and there is an existing feature: Idea ID 308938

Furthermore, stay tuned about upcoming features in SAP Datasphere Roadmap.

Keywords

SAP Datasphere, User not displayed, not showed, hidden.
