Symptom

  • Error encountered during the execution of PC00_M99_FPAYM: "Window MAIN is not defined for form & (TD426)".

Environment

  • Financial Accounting (FI)
  • SAP R/3
  • SAP R/3 Enterprise 
  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA
  • SAP S/4HANA Finance

Reproducing the Issue

  1. Execute transaction code PC00_M99_FPAYM - Create DME - Payment Medium Workbench.
  2. Observe the error message: "Window MAIN is not defined for form & (TD426)".

Cause

  • In the view V_T042E_Z, the form (Ex: F110_CA_DTA) is maintained in Paying company code for payment method as 'Next form'.

Resolution

  1. Access view V_T042E_Z using transaction SM30.
  2. Locate the entry where Paying company code is and payment method.
  3. Delete the form (Ex: F110_CA_DTA) from the 'Next form' field.
  4. Retest the process to confirm resolution.
