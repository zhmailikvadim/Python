Symptom

Any other user except windows administartor is not able to launch Webi Rich client.

Environment

  • SAP BusinessObjects Business Intelligence Platform 4.1
  • Windows OS

Reproducing the Issue

  • Install Webi Rich client with administrative privilege.
  • No other user (non-administator) will able to launch Webi Rich Client
  • In log file, we get error message:-

An error occured while trying to lock the file :  C:\Users\Administrator\.businessobjects\.instance.lock 

java.io.IOException: The system cannot find the path specified

Cause

User is unable to lock instance of webi rich client. 

Resolution

  1. Open run dialog box.
  2. Type regedit and hit enter.
  3. Go to this location:-
  4. HKEY_LOCAL_MACHINE\Software\Wow6432Node\SAP BusinessObjects\Suite XI 4.0\defaultWebIntelligence\Richclient\JVMOptions.
  5. Add a new string value with "Next number" and value data -Duser.home=%homedrive%

Keywords

Webi rich client crash, webi client lauch
