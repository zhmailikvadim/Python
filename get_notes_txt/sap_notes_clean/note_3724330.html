Symptom

SAP Note 3646906 provided the option to restore reversal data from previous ELDA notifications.

This leads to incorrect results for public sector employees (BVA) under certain constellations, and this occurs if the old fields ASVGR (SI group) and AUSVA (SI institution) are still filled in the corresponding SI records (IT0044). These were still transferred during the transition phase to the New Tariff System (NTS).

Other Terms

IT3670, IT0044, BVA, NTS

Reason and Prerequisites

Program gap

Solution

The correction is provided by HR Support Package (via CLC if necessary). You can implement an advance correction by implementing the correction instructions using SNOTE.
