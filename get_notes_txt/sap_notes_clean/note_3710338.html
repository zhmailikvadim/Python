Symptom

  • The "Book Entry" field in the 24Q report is marked as "Y" in the E-file section (PC00_M40_F24Q).
  • This causes FVU (File Validation Utility) errors during validation.

Environment

  • Human Capital Management (HCM)
  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • Payroll India

Reproducing the Issue

  1. Execute transaction PC00_M40_F24Q.
  2. Observe that the "Book Entry" field is marked as "Y" in the E-file section.
  3. Attempt to validate the file using the File Validation Utility (FVU).
  4. Validation errors occur due to the "Book Entry" field being marked as "Y".

Cause

  • During the creation of the mapping in transaction PC00_M40_CMAP, the checkbox for "Book Entry" was mistakenly selected, resulting in the "Book Entry" field being marked as "Y".

Resolution

  1. Execute transaction PC00_M40_CMAP in SAP System.
  2. Locate the mapping configuration where the "Book Entry" checkbox was selected.
  3. Deselect the checkbox for "Book Entry".
  4. After that perform rest of the step to ensure correct challan mapping..
