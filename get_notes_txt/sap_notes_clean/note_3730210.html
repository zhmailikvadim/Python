Symptom

The system does not provide version management for the following HCM repository objects: PMKS, PSCS, PFCS, POCS, POTS, PFTS, PFCC, POCC, POTC, PFTC.

Other Terms

PE03, PE04

Reason and Prerequisites

The system lacks the functionality to support version management for these HCM objects.

Solution

After implementing this SAP Note, the system provides version management for the HCM repository objects PMKS, PSCS, PFCS, POCS, POTS, PFTS, PFCC, POCC, POTC, and PFTC. To enable this functionality, implement the relevant SAP Support Package.
