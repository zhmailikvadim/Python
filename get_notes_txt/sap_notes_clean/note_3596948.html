Symptom

The field Material is displayed on the header level when creating a new certificate assignment in transaction QC15 instead of the expected item level.

"Image/data in this KBA is from SAP internal test systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."   

Environment

  • SAP ERP Quality Management (QM)
  • SAP ERP Central Component
  • SAP ERP
  • SAP Enhancement package for SAP ERP
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA, on-premise
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Call transaction QC15.
  2. Select Key combination Material/Plant.
  3. The material displayed on the header level.

Expected system response: the material is displayed on the item level.

Cause

Configuration.

Resolution

Set flag for 'Item field' in the condition table for profile determination in customizing.

  1. Call transaction QCC0.
  2. Quality certificate>Certificate profile>Profile determination>Condition table for profile determination>Edit condition tables for profile determination.
  3. Create/Change condition table for certificates.
  4. Select Material/Plant.
  5. Use button 'Technical view'.
  6. Set flag for 'Item field' for material.
