Symptom

Refer to the symptoms from: SAP Note 3711624:UX 3.0 - Improvements in Manage Change Records.

This SAP Note introduces the front end changes required to "Manage Change Record"(F2097) application.

Other Terms

PLM-FIO-CR, UX 3.0, List Report Table, Object Page Table, Context Menu, Column Width header, UX Improvements in Fiori Apps in S/4HANA (2), 3711624.

Reason and Prerequisites

Standard FIORI Recommendations.

Prerequisite SAP Note: 3711624. The correction from the current SAP Note should be implemented after implementing the backend changes from the SAP Note 3711624.

Solution

Use transaction SNOTE to implement the corrections delivered with this SAP Note in the affected Change Record system or import the relevant Support Package.
