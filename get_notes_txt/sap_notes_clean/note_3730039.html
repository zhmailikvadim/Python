Symptom

In the process "Decision Tree Diagram E_0608 (Registration of an Assignment) (DE_B_GPKE_DSO_NIN_DTD_E0608)" (DE_B_GPKE_NIN_DSO), an error occurs in step 0100-8900 because of an incorrect hierarchical assignment to the subprocess in step 0900.

This issue is relevant for the market role: DSO.

Other Terms

New Installation, change of supplier 24, DE_B_GPKE_NIN_DSO, DE_B_GPKE_DSO_NIN_DTD_E0608, GPKE_DSO, subprocess, hierarchical assignment, preconditions

Reason and Prerequisites

The error occurred because the preconditions for the hierarchical assignment to the subprocess were incorrect.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the system corrects the hierarchical assignment to the subprocess, and step 0100-8900 executes without errors. The BC set /US4G/4_0_DSO_SP08_NOTE_3730039 contains the necessary corrections. You can verify the fix by checking that the embedded process completes successfully with no error in step 0100-8900 and that the hierarchical assignment to the subprocess is correct.
