Symptom

When executing "Plan Raw Material Deliveries" and restricting the Plant authorization field of authorization object /DRY/RDYPN to a specific value, the weeks are displayed in a random sequence with multiple duplicate entries of the same week. When Plant authorization field is set to “*”, the weeks are displayed correctly in sequential order.

Other Terms

/DRY/RDYPN, Raw Material Delivery Planning, Plant filter, Week sequence, Duplication

Reason and Prerequisites

Program error

Solution

History:

Version 1 of 29 January, 2026: New creation

Version 2 of 13 February, 2026: Validity extension

Install the code correction.

Description of Correction:

When executing "Plan Raw Material Deliveries" and restricting the Plant authorization field of authorization object /DRY/RDYPN to a specific value, the weeks are displayed properly without any duplication.

Functional Dependencies:

There are no functional dependencies.

Manual Changes:

There are no manual activities.
