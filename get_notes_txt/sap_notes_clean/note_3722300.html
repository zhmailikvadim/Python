Symptom

Missing configuration in /US4G/V_LBSOBJRF caused 'Technical Resource refer to MeLo' data missing, this lead to 'RFF+Z34' missing in market message.

This issue is relevant for all market roles.

Other Terms

 MDC, Location Bundle, REF_TO_LOC_BUNDLE_POD, LOCATION_BUNDLE_LOCATION, RFF+Z34, SEQ+Z78, SEQ+Z58

Reason and Prerequisites

 The error occurred because the configuration was missing.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the system fills the 'TR refer to MeLo' data in the field ubd-location_bundle-location_bundle_location-ref_to_loc_bundle_pod. You can verify the fix by retesting a bundle-related process from the back end and checking that the 'TR refer to MeLo' data is filled in ubd-location_bundle.
