Symptom

When creating a new user using the RESTful SDK, the option "User can change password" is not available and is set to true by default.

Other Terms

Keywords

change password cannot default

Reason and Prerequisites

 Environment

  •  SAP BusinessObjects Business Intelligence Platform 4.2 SP5
  •  SAP BusinessObjects Business Intelligence Platform 4.2 SP6
  •  SAP BusinessObjects Business Intelligence Platform 4.2 SP7

Steps to Reproduce:

  • Create a new user through restful web service http://<ip>:<port>/biprws/v1/users/user.
  • Observe there is no option to set "allowchangepassword" option.

Solution

This issue is fixed in the patches listed in the "Support Packages & Patches" section below.

The "Support Packages & Patches" section will be populated with the relevant patch levels once they are released.

For Business Intelligence Platform maintenance schedule and strategy see the Knowledge Base Article 2144559 in References section.
