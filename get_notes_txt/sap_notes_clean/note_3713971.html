Symptom

This SAP Note contains a correction for the domain of the classifier identifier that extends the output length from 60 to 255 characters. This domain is used in F4 help for "Location Classifiers" filters.

This change relates to the following apps that have the "Location Classifiers" filter:

  • Manage Locations (under the "Search" tab)
  • Manage Compliance Scenario (under the "Display by Location" tab)

Other Terms

location classifier incorrect value

Reason and Prerequisites

Reason: Error when selecting classifiers with names longer than 60 characters.

Solution

This SAP Note contains a correction for the selection of location classifiers with names longer than 60 characters.

The support packages are listed in the "Support Packages" section.
