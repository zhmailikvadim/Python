Symptom

Component: Foundation for EHS Management
Module: Amounts

When you review collected or calculated amount values in EHS Management apps like Monitor Data (App ID: EHENV_MDEF_AMOUNT_OIF), the history overview of amounts displays a large number of duplicate entries with identical remarks such as "Saved by the calculation carried out automatically" and identical values. These duplicate values cause performance issues when you handle amount values.

Other Terms

CL_EHFND_AMOUNTS_A_BASE

Reason and Prerequisites

The symptom occurs due to a program error.

See the correction instructions for prerequisites.

Solution

After implementing this SAP Note, the system does not save history entries for amounts with identical values and remarks

Alternatively, import the corresponding Support Package for the component extension of SAP EHS Management or the Feature Package Stack for SAP S/4HANA.
