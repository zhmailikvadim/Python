Symptom

In Timesheet Analytics - My Activities application, The Concurrent Employment functionality was not available.

Other Terms

F6687, Timesheet Analytics, My Activities, CE Enable, CE, Concurrent Employment

Reason and Prerequisites

Concurrent Employment functionality is now introduced.

Prerequisite: The SAP Note: 3543018 has to be implemented in the corresponding back-end system.

Solution

Apply corresponding support pack. Alternatively, implement the correction instructions attached to the SAP Note.
