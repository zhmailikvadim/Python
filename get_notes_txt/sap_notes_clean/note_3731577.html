Symptom

"Performance Analysis Tool" incorrectly opens "Performance Statistics and Analysis" report in SAC

Environment

SAP Analytics Cloud 2026.2.10

Reproducing the Issue

  1. Log in to the SAC tenant.
  2. Navigate to Performance -> Analysis Tool

Notice: The story name indicates “Performance Analysis Tool,” the content that opens is “Performance Statistics and Analysis”.

Cause

The issue is caused by a defect in the product.

Resolution

The issue will be fixed in a future** version of SAP Analytics Cloud. Please refer to maintenance document.

** in version 2026.02.12 for customers on a bi-weekly (fast track) release cycle.
** in Q2 for customers on a Quarterly release cycle (QRC).
