Symptom

You are using the Adjust Forecast app in SAP Customer Activity Repository applications bundle 5.0 FPS02, 5.0 FPS03, 5.0 FPS04, 5.0 SPS05, 5.0 SPS06, 5.0 SPS07, 5.0 SPS08, 6.0 SPS01 or 6.0 FPS02.

When you apply a filter on product, location, or a combination of these, the forecast corrections assigned to the selected products and locations are not displayed in some cases.

Reason and Prerequisites

Reason

Program error in ODataServices.controller.js.

Prerequisites

You have set up the Adjust Forecast app as described in the Common Installation Guide (for new installations) or the Common Upgrade Guide (for upgrade scenarios) for SAP Customer Activity Repository applications bundle 5.0 FPS02, 5.0 FPS03, 5.0 FPS04, 5.0 SPS05, 5.0 SPS06, 5.0 SPS07, 5.0 SPS08, 6.0 SPS01 or 6.0 FPS02. You can find both guides on SAP Help Portal at https://help.sap.com/docs/CARAB. Select your version at the top.

Solution

Implement the attached correction instructions in your front-end system.
