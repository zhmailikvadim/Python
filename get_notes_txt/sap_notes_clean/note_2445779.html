Symptom

The verification list master data (program name J_1VRTOOL_CHECK_VKUN) terminates or has a runtime that is far too long. The selection of existing contracts of the customers to be created is insufficient.

Reason and Prerequisites

The prerequisite for implementing the program corrections is patch level 0005 for KVsprint Release 5.700 (technical version 510_702).

Solution

  • Import transport request KV5K905211 into any client of your system. The transport request is attached to this SAP Note in the file KV5K905211.SAR.
