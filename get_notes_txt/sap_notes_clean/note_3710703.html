Symptom

This SAP Note contains an error correction of the OData service /MSG/API_X_ORG_OBJECT_READ_O4 for reading PD-ORG objects that is required as a technical pre-implementation step for further development.

Other Terms

Web Service

Remote API

Reason and Prerequisites

For the affected CDS views, the cardinality of some associations was specified incorrectly. If an association cannot be resolved, a structure with empty fields is returned.

Solution

History:
February 2, 2026: New

Troubleshooting description:
The OData service /MSG/API_X_ORG_OBJECT_READ_O4 has been adjusted.
The cardinalities of associations of the CDS views /MSG/R_X_ORG_Object and /MSG/R_X_ORG_ObjectHierarchy have been corrected. If an association cannot be resolved, "null" is returned.

Note that the services are currently not released for productive use. For this reason, non-compatible changes to the services may take place in the future. The services will be released at a later point in time.
