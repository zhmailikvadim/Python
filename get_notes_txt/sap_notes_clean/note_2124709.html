Symptom

The following exception is logged into the smdagent.trc or SMDSystem.log:

May 9, 2014 2:16:43 PM [Thread[ExDaemon:exmgmt:jobmgr_1,5,exmgmt:j...] Error      Exception occured when trigerring task for executor : exmpush in execution manager: Execution Manager[exmgmt:jobmgr] 1011980385 :

[...]

[EXCEPTION]
java.lang.IllegalThreadStateException
               at java.lang.ThreadGroup.addUnstarted(ThreadGroup.java:843)
               at java.lang.Thread.init(Thread.java:403)
               at java.lang.Thread.init(Thread.java:355)
               at java.lang.Thread.<init>(Thread.java:625)
               at com.sap.smd.server.exec.ExecThreadFactory.newThread(ExecThreadFactory.java:22)

Other Terms

Diagnostics Agent, SMD, IllegalThreadStateException, thread, EXM, executor

Reason and Prerequisites

A program error is present in the life-cycle of agent task executor, leading to registering a task on a thread which is not alive.

Solution

Deploy the LM-SERVICE Software Component, mentioned in the "Support Packages & Patches" section.
Before proceeding with the deployment, pay attention to the general recommendations of SAP note 1483508.
