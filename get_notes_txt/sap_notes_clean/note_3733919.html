Symptom

  • When running the MC.9 report, the material description is truncated and not fully displayed alongside the material code.
  • Adjusting the column width only allows up to 60 characters, which is insufficient for longer descriptions.
  • No error message is displayed; the report output is incomplete.

Environment

  • Sales And Distribution (SD)
  • Logistics Execution (LE)
  • SAP S/4HANA
  • SAP S/4HANA Cloud 
  • SAP ERP Central Component
  • SAP ERP
  • SAP Enhancement package for SAP ERP
  • SAP Enhancement package for SAP ERP, version for SAP HANA

Reproducing the Issue

  1. Run transaction MC.9.
  2. Enter a plant or a material code.
  3. Start the report.
  4. Observe that the material code and description are displayed, but the material description is truncated.
  5. Attempt to adjust the column width; note that it cannot exceed 60 characters.

Cause

As per the SAP note 2215424 (Material Number Field Length Extension - General Information), in SAP S/4HANA, the material number field length has been extended from 18 to 40 characters. SAP S/4HANA supports material numbers with up to 40 characters. Therefore, the maximum length of the fields that handle the material number has been extended from 18 to 40 characters.

Resolution

As note 516932 says it is sometimes possible to modify the standard report, so it is able to work with a character key figure in the range of 30 to 58 characters length. However, this adaption would be chargeable as consulting. Please also have a look at note 2232362. In this case exist no solution to display material number and description in full length. The limit is 60 characteristics.
