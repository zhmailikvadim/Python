Symptom

While using the "Mass upload" option in "Maintain Master User ID Mapping"  in SPRO, the system is uploading incorrect entries.

 

Environment

SAP Access Control 12.0

Reproducing the Issue

  1. Go to SPRO-> Governance, Risk and Compliance -> Access Control -> Maintain master user ID mapping
  2. Select the option  "Mass Upload" Button
  3. Upload the given file 

Cause

The File format was incorrect .

Resolution

The Format for the Mass Upload should be as the below given format i.e. with no Space, or #, or Tab between the values.

Keywords

Master user, Master ID, Mapping,Mass Upload, File format
