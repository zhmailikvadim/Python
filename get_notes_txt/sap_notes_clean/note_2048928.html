Symptom

An activation error occurs when activating CDS views. The following error occurs:

  • A VALUE WITH DATA TYPE SYSIBM.INTEGER CANNOT BE CAST TO TYPE VARGRAPHIC

Other Terms

DB2 ZOS CAST CDS ACTIVATION INTEGER VARGRAPHIC

Reason and Prerequisites

DB2 does not directly support CASTs from numerical types to VARGRAPHIC.

Solution

Please apply the attached Note correction or implement the required SAP_BASIS support package.
