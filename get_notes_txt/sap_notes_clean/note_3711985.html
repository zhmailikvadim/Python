Symptom

An application displays data in Hierarchical Sequential (HS) format. The application also implements code to be executed before or after the save layout command &AVE. If the Hierarchical Sequential Grid (HS-Grid) view is chosen to display the data, the code is not executed when a layout variant is saved.

Other Terms

ALV; ABAP List Viewer; SAP List Viewer; Grid; List;

Hierarchical Sequential; Hierseq; HS; HS-LIST; HS-GRID

Reason and Prerequisites

Reason

HS-Grid passes the internal code &AVE_IMPL instead of external code &AVE to the event handler.

Prerequisites

The following Support Package level is the minimum prerequisite:

Release Support Package
SAP_BASIS 816 SP 00
SAP_BASIS 758 SP 02

When you install an SAP Note, some prerequisites might be required. The prerequisites are SAP Notes that have already been released and that change the same context of the current one. Then, even if these prerequisites are not necessarily related to the symptom of this SAP Note, you must implement them due to technical reasons.

Solution

From now on, the scenario described in the "Symptom" section no longer occurs.

Delivery

As a general rule, SAP recommends that you install a solution by applying a Support Package. However, if you need to install the solution earlier: Use the Note Assistant (SNOTE) to implement the correction instructions.

Learn more at SAP Note Assistant.
