Symptom

When replicating Supplier Business Partner master data via web service, the error 'Mandatory partner function VN is missing for purchasing organization' is raised.

Environment

SAP S/4HANA Cloud Public Edition

Cause

The payload received in SAP S/4HANA Public Cloud does not contain the mandatory partner function VN for the purchasing organization.

Resolution

  • Review the payload sent from Ariba SLP to ensure that the mandatory partner function VN is included for the relevant purchasing organization.
  • Alternatively, adjust the configuration in SAP S/4HANA Cloud Public Edition by maintaining configuration activity '103125 – Define Partner Schemas in Supplier Master Record'  and set the partner function from mandatory to non-mandatory if aligned with business requirements.

See Also

Refer to SAP Knowledge Base Article 3072643 - How to Configure Partner Functions as Mandatory for a Supplier Account Group.
