Symptom

  • How to mass create Custom Tariff codes in S/4HANA.
  • What is the standard functionality to create and update Custom Tariff codes in S/4HANA.

Environment

  • SAP S/4HANA 
  • SAP S/4HANA Public Cloud
  • SAP S/4HANA Private Cloud
  • International Trade Management - Classification  

Reproducing the Issue

  • You wish to know what functionality is available for the maintenance of Custom Tariff codes in an S/4HANA system. 

Cause

Resolution

The options for tariff code maintenance in an S/4HANA public and private cloud systems are as follows: 

1) The recommended approach for tariff code maintenance is to use the apps below which provide direct integration with a data provider. This approach allows the mass creation and update of tariff code data in the system.

  1. 'Schedule Content Request to Data Provider - Customs Tariff Numbers' (F3569) -> With this app you can regularly schedule content requests to your external data providers to receive updated and new versions.
  2. 'Manage Content  from Data Provider - Customs Tariff Numbers' (F3581) -> With this app you can use this app to display and activate content from external data providers.  

2) The app 'Manage Customs Tariff Numbers' (F3122) is available in case you wish to maintain the tariff codes manually in the system. Regarding mass classification, the following apps available:

  1. 'Classify Products - Customs Tariff Numbers' (F31460).
  2. 'Reclassify Products - Customs Tariff Numbers' (F3147).
