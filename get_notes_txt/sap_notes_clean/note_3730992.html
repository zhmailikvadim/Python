Symptom

Some Payroll Function/Operation text entries still exist in the corresponding tables after deleting the Function/Operation in transaction PE04.

Other Terms

PE04, Payroll Function, Payroll Operation, T52A0_T, T52B8, T52BB, T52BP

Reason and Prerequisites

You delete a Payroll Function/Operation in transaction PE04.
Only Function/Operation texts in the logon language will be deleted from tables T52A0_T, T52B8, T52BB, T52BP.

Solution

Implement the corresponding correction instruction or the Support Package.

See help.sap.com for further information regarding Payroll Functions and Operations.

Remark: As a solution, if the user is now logged on in the original language of the Payroll Function/Operation to be deleted, all associated texts (in all languages) will be deleted when deleting the object. Otherwise, only texts in the logon language will be deleted.
