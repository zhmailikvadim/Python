Symptom

Some users'/countries' national ID card type data cannot be displayed in the story report.

Environment

  • SAP Employee Central
  • Reporting Data (EC core only)

Reproducing the Issue

  1. Attempt to query data for national id info via Personal Information/Biographical Information/National ID Information.
  2. Observe that the column "National Id Card Type" (person#nationalIdCard#CARD_TYPE) does not display data for certain countries.

Cause

en_US label is not maintained for centain countries' national id card type while you are using en_US as the system language.

Resolution

Please maintain corresponding language labels for national id card type for different countries.

For example, maintain below translation for US English under nationalIdCard_USA then run the report again.

Keywords
