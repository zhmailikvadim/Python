Symptom

When you choose the "Data Transfer" button in master data maintenance (PA30) for the infotype 0008 ("Basic Pay"), the system displays a window with the TVoeD/TV-L transfer data. However, the sequence of the fields is alphabetical, which proves impractical in actual use.

Other Terms

Basic pay, MPV00800, screen 2002, backup table, PDPBSTVOED_UEB

Reason and Prerequisites

Program adjustment

Solution

Implement the attached correction instructions or import the specified support package. As a result, the sequence is left unchanged (in the same way as for the DNeuG transfer tables PDPBSDNEUG_UEB and PDPBSDNEUG_UEB_T). Sorting is no longer performed.
