Symptom

When trying to edit the form of a workflow action within form designer, it's impossible to delete, edit or add sections.

Environment

  • SAP Sales Performance Management
  • Advanced Workflow

Reproducing the Issue

  1. Log in to Advanced Workflow as an Administrator
  2. Go to Setup -> Processes -> Workflows
  3. Click "on Edit" for a workflow
  4. Go to "Settings for default actions"
  5. Select Form designer for the affected action. 

Cause

This can be caused by corrupted data in the backend, often caused by imports using the migration manager between environments in two different versions. Importing from the 2602 release to future versions is known to cause this issue. 

Resolution

The data has to be manually fixed by the engineering team through SAP Support. Open a support case and inform the following information:

  • The affected workflow actions
  • The reproduction steps
  • The tenants that have affected workflow actions.
  • If any migrations were done recently
