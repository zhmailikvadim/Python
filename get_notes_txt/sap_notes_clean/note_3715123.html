Symptom

You execute the function "Check Settings" for consolidation of investments (C/I) (transaction CX6C3). The execution terminates with error message G01 171 ( "No dimension exists as the source for transfer to SAP S/4HANA" ).

Other Terms

G01171, check settings, consolidation of investments, ABCOI

Reason and Prerequisites

You are undergoing the greenfield setup of SAP S/4HANA for Group Reporting (S/4GR) according to SAP Note 3592099 or you have installed the "best practices" content for S/4GR according to SAP Note 3666042.

Solution

Import of Support Package or advanced correction by implementation of correction instruction
