Symptom

  • Try to start realtime replication through SAC connection failed with error
  • Error in DP agent log
    CloudDataIntegrationAdapter | CloudDataIntegrationLogger.info [] - *** - creating subscription <FactData_***> for </sac/***/FactData> to <http://***.sapanalytics.cloud/api/v1/dataexport/administration/Subscriptions>
    2025-03-10 12:44:12,668 [ERROR] [***] CloudDataIntegrationAdapter | CloudDataIntegrationLogger.error [] - *** - Failed to create subscription FactData_*** due to 301 Moved Permanently

Environment

SAP HANA Smart Data Integration

Cause

The remote source configuration is incorrect

Resolution

Update the remote source configuration to use "Protocol" as "HTTPS"

Keywords

CDIAdapter, SAC, subscription, realtime
