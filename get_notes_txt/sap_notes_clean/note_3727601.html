Symptom

The Income Tax Declaration objects have been all set as secured and permissions to the objects (and permissions to manage and view income tax declarations) have only been granted to the Employee Self Service permission roles. However, in the Latest People Profile the employees are still able to see the 'Income Tax Declaration' card and the declarations which have been made on other Employee's profiles. 

Environment

SAP SuccessFactors Employee Central 2H 2025

Reproducing the Issue

  1. Secure all Income Tax Related Objects
  2. Grant the necessary permissions only to Employee on Self Roles
  3. Using the Latest People Profile, Proxy as Employee A
  4. Go to Employee's B profile
  5. Income Tax Declaration Card is visible.
  6. The same behavior does not happen with this configuration when using the legacy Profile. 

Cause

This is a Known Issue.

Resolution

The fix for this issue is planned to be deployed on 1H 2026 release.

For release timelines, please review SAP SuccessFactors Product Release & Road Map Information.
