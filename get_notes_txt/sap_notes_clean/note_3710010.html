Symptom

During the data replication with the data replication framework (DRF) using DRFOUT, no messages are logged in DRFLOG for non-existent product.

Other Terms

DRFOUT, DRFIMG,  DRF_OBJ_REP_STA, BAL_LOG, DRF, SOAP, DRF/SOAP

Reason and Prerequisites

Error messages are not logged for non-existent product in DRFLOG.

Solution

Error message 'Product does not exist' will be logged in DRFOUT while replicating the data with DRF.

Implement the attached correction instructions.
