Symptom

  • Errors in the Technical Job Repository (SJOBREPO).
  • Inconsistencies in standard BC jobs defined in the Technical Job Repository.
  • Specific job inconsistencies:
  • APS_YIO_ORCHESTRATOR_RUN: Event ID does not exist in the system.
  • FCOUA_TDD_SIMULATE_CLEAR: BTC User Error.
  • SAP_CDS_VIEW_CHECK_COMPATIBLE: Report does not exist in the system.
  • SAP_CDS_VIEW_DELETE_PROVISION: Report does not exist in the system.
  • SAP_CDS_VIEW_PROVISIONING_RUN: Report does not exist in the system.
  • SAP_FIN_APM_IHB_FINALIZATION: Report requires a variant, but no variants are available in the system.

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental. 

Environment

SAP S/4HANA 2023
SAP S/4HANA 2025

Reproducing the Issue

  1. Navigate to transaction SJOBREPO.
  2. Perform a consistency check on the Technical Job definitions using transaction SE80.
  3. Observe the inconsistencies in the job definitions:
  • APS_YIO_ORCHESTRATOR_RUN: Event ID does not exist.
  • FCOUA_TDD_SIMULATE_CLEAR: BTC User Error.
  • SAP_CDS_VIEW_CHECK_COMPATIBLE: Report does not exist.
  • SAP_CDS_VIEW_DELETE_PROVISION: Report does not exist.
  • SAP_CDS_VIEW_PROVISIONING_RUN: Report does not exist.
  • SAP_FIN_APM_IHB_FINALIZATION: Report requires a variant, but no variants are available.

Cause

  • The following job definitions should not be present in S/4HANA 2023 and S/4HANA 2025 - These jobs are no longer necessary because the functionality they supported has been re-engineered and replaced. They have been removed since the last customer still using the old version has transitioned to the new version. Therefore, they are indeed no longer needed.
  • SAP_CDS_VIEW_CHECK_COMPATIBLE
  • SAP_CDS_VIEW_DELETE_PROVISION
  • SAP_CDS_VIEW_PROVISIONING_RUN

Resolution

  1. For Technical Job APS_YIO_ORCHESTRATOR_RUN. Implement SAP Note 3733608 - Inconsistent technical job definition for APS_YIO_ORCHESTRATOR_RUN
  2. For Technical Job FCOUA_TDD_SIMULATE_CLEAR: BTC User Error. Implement SAP Note 3730118 - 'BTC User error' due to empty DEFAULT user entry  in STJR_JOBD_ROOT
  3. For Technical Job SAP_FIN_APM_IHB_FINALIZATION. This job is no longer required in S/4HANA 2025 and should be deactivated by following the instructions provided in the 7th question of FAQ 3236399.
  4. For the Technical Jobs SAP_CDS_VIEW_CHECK_COMPATIBLE, SAP_CDS_VIEW_DELETE_PROVISION, and SAP_CDS_VIEW_PROVISIONING_RUN:

    Implement SAP Note 3714840, which provides a solution to delete these jobs from SJOBREPO. If the note is not available for download, raise a ticket to CA-GTF-EI-VP. A workaround is to deactivate them by following the instructions provided in the 7th question of FAQ 3236399.
