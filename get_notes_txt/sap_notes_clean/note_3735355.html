Symptom

When attempting to create a Vendor Claim using transaction code /IRM/IPPRA23, no claim is generated. However, the same process works correctly when executed from the workbench.

This issue is resolved by implementing the relevant SAP Note, which provides the necessary corrections.

Other Terms

 /IRM/IPPRA23, /IRM/IPPRASP, Vendor Claim, Schedule Claim

Reason and Prerequisites

This is caused by program error

For ECC, SAP Note #3735400 is available

Solution

Please implement the correction instructions attached. These corrections are only valid if you have applied the following support packs.

Release V4 - 2023 V2, 2023 V3 or Support Pack 1

Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object.
