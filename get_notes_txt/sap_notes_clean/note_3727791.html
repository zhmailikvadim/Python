Symptom

  • Refreshing a Web Intelligence document in BI launchpad keeps on loading for an indefinite time.
  • The same Web Intelligence document is refreshed successfully in Webi Rich Client.

Environment

  • SAP BusinessObjects Business Intelligence Platform 2025
  • SAP BusinessObjects Business Intelligence Platform 4.x
  • Web Intelligence 

Reproducing the Issue

  1. Launch the BI launchpad.
  2. Open the affected Web Intelligence report
  3. Refresh the report
    >>Observe the report is stuck in refreshing state.

Cause

The trace level for any of the processing servers among CMS, APS, AJS, WIPS are set to High in Central Management Console, which causes the performance degradation while refreshing the Web Intelligence report.

Resolution

  1. Open CMC
  2. Click on Servers.
  3. Right click on one of the processing servers.
  4. Select ‘Properties’
  5. Confirm the Trace Log level.
  6. If trace level is set to ‘High’, change it to ‘unspecified’
  7. Follow the above steps for all the processing nodes.

Keywords

Webi, Webi rich client, report, refresh, stuck
