Symptom

When launching the My Time Events App, there is no quota shown to the end-user in the Entitlements table

Environment

  • My Leave Requests (Fiori 2.0 / Version 3)
  • SAP Fiori 2.0 for SAP HCM
  • SAP S/4HANA

Reproducing the Issue

  1. Launch the My Leave Requests (Fiori 2.0 / Version 3) application
  2. Observe that no quota is displayed in the entitlements table

Cause

There are no entries maintained in the configuration table V_T556A_WEB for the Rule Group related to the users PERNR

Resolution

  1. Configure the table V_T556A_WEB to include the necessary entries for the PERNRs Rule Group

See Also

Refer to: SAP Note 0001486743 - Quota not visible Time accounts ESS.

Keywords

my time events, quota not displayed, entitlements table, V_T556A_WEB, PERNRs Rule Group, SAP Note 0001486743, configuration table, leave quotas, Fiori app.
