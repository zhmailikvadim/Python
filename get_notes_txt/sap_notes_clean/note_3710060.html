Symptom

The error “Error during script processing” occurs when viewing a Lumira document created using the Generic Analysis template

Other Terms

SAP Lumira Designer 2025, SAP Lumira Server 2025, BI 2025 Patch8 , Bookmark , Add Datasource , Generic Analysis template

Reason and Prerequisites

Steps to reproduce:

       Issue 1 Bookmark

  1. Open Lumira Document (created with Generic Analysis template) in Lumira Designer 2025 Client(execute locally or in BI Platform)
  2. Click on the Bookmark icon and select Manage Bookmark

  >> Observe that "Error during script processing" dailog box is seen

Issue 2 Add Datasource

  1. Open Lumira Document (created with Generic Analysis template) in Lumira Designer 2025 Client(execute locally or in BI Platform)
  2. Click on the Add Datasource icon .
  3. Select the connection and query and then click ok

        >> Observe that blank screen is seen with error "Error during script processing"

 

Solution

NOTE : Fix will be available in Lumira 2025 Patch 1

Workaround for an existing document in Lumira 2025

  1. Login to Lumira Designer.

     For BI Platform workflow: Right-click on the BI Platform document and select download the document.
  2. Open the downloaded document in Lumira Designer.
  3. Follow the steps outlined in the attached document “Lumira 2025_Generic_Analysis_Script Update” to update the script in your document.
  4. After applying the changes, save the document and execute the Bookmark and Add Data Source workflow in the document.

Workaround for newly created document in Lumira 2025

  1. Close the Lumria Designer client
  2. Open the Machine containing Lumira Designer
  3. Navigate to the following directory <<INSTALL_DIR>>\SAP Lumira\Lumira Designer\plugins
  4. In pulgins folder and locate the following folder com.sap.ip.bi.zen.content_25.0.0 and take backup to other location
  5. Download Lumira2025_Script_Fix.zip to local machine and extract it
  6. Navigate the following directory  <<INSTALL_DIR>>\SAP Lumira\Lumira Designer\plugins\com.sap.ip.bi.zen.content_25.0.0\templates\APPLICATIONS\READY_TO_RUN_MAIN\GENERIC_ANALYSIS and replace  attached content_biapp
  7. Navigate the following directory <<INSTALL_DIR>>\SAP Lumira\Lumira Designer\plugins\com.sap.ip.bi.zen.content_25.0.0\templates\templates\DOCUMENTS\READY_TO_RUN and replace attached Generic_Analysis.lumx file
  8. Start Lumira Designer client
