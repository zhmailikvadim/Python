Symptom

When Matrix manager selects 'Matrix reports only' within Display options of the succession org chart, NO reports appear even though there are Matrix reports for the same Matrix manager. 

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

SAP SuccessFactors Succession Management

Reproducing the Issue

  1. Go to succession org chart >> search for any user 
  2. Display options >> select Matrix Reports Only 
  3. Users/matrix reportees are not displayed in the Succession Org Chart.

     

Cause

The issue is caused by the option "Show custom relationships in the legacy Org Chart, home page and mobile People Profile" in admin tool "Company System and Logo Settings".

Resolution

  1. Access the admin tool "Company System and Logo Settings".
  2. Locate the option "Show custom relationships in the legacy Org Chart, home page and mobile People Profile".
  3. Disable this option to resolve the issue.
  4. Verify that users are now displayed correctly in the Succession Org Chart.
