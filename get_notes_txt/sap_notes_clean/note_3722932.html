Symptom

When you post an invoice in the source application for an electronic waybill an error occurs.

Other Terms

e-waybill India, Implementation Overview Note, eDocument, eDoc, eInvoicing, e-Invoicing, electronic invoice, electronic invoicing, eInvoice, e-document, e-Document, e-Docs, e-Doc, e-docs, e-doc, Document Compliance, DC, Document and Reporting Compliance, electronic document processing, e-waybill, GST, Goods and Service Tax, edocuments India, cloud edition, EDOINEWBCONFIGV, source field type

Reason and Prerequisites

Reason

You have maintained the Source Field Type as Table Field in the EDOINEWBCONFIGV view.

Prerequisites

You have implemented either of the following SAP Notes based on your requirement,

  • 3569762 - Electronic Document Processing for India: Electronic Waybill using SAP Document and Repor….
  • 2631687 - Electronic Document Processing for India: E-Waybill - Implementation Overview - SAP for Me

Solution

After implementing this SAP Note, you can post an invoice in the source application without an error.
