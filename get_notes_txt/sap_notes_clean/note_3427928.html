Symptom

When system is in upgrade, table TSTCT technical setting is showing in SPDD. The sharing type is different between original system and new version. It can not be adjusted in SPDD and can not be editable in SE11 even in change mode. 

Environment

SAP NetWeaver

Reproducing the Issue

SE11>input table TSTCT>Display>technical setting>change

Cause

Sharing type is defined in table DD09V

Resolution

* Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

The sharing type can not be changed manually in SE11 and can not be adjusted in SPDD. The technical setting shows in SPDD because other information in this screen is changed, not triggered by sharing type. Sharing type can only be changed by SAP due to system version upgrade. 

Keywords

sharing type, SPDD, adjust, edit, TSTCT, technical setting
