Symptom

Your personnel number has a mid-month prohibited activities. In addition, the personnel number has two contracts for the German Pension Reform (AVmG) with the same investment type and the same contract elements for each contract. 
Due to the mid-month prohibited activities, the amounts of the contract elements are assigned proportionally to the periods in the fictitious run for the prohibited activities. For this purpose, new modules are generated on the basis of the existing modules in payroll function DAAG (parameter 1 = BV parameter 2 = AVMG). The system incorrectly uses the same module as a copy template, which means that the assignment to the contract is incorrect.

Other Terms

 CL_HRPAYDE_AVM_BERECHNUNG

Reason and Prerequisites

 Program error UPDATE_P0699_E

Solution

Implement the attached correction instructions or import the specified support package.

After you import the HR Support Package or implement the correction instructions, you do not have to perform any further follow-on activities.
