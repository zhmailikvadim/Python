Symptom

You are using either VMODE 1 or VMODE 2 and have implemented Note "3606572 - Fixed /SHCM/RH_SYNC_BUPA_EMPL_SINGLE if status of all actions at system date is 0 - SAP for Me". The report "/SHCM/RH_SYNC_BUPA_FROM_EMPL" is running its daily update. A person with a hire date of the current day is not properly synched and the warnings "Employee &1 : There is no valid CP-P relationship maintained." and "Employee &1 : There is no valid CP-P relationship maintained." are displayed.

Other Terms

/SHCM/RH_SYNC_BUPA_FROM_EMPL, /SHCM/EMPLOYEE_BL 073, /SHCM/EMPLOYEE_BL 074

Reason and Prerequisites

Program error

Solution

Apply the corresponding support package. You may also apply the corresponding correction instructions.
