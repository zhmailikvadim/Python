Symptom

Cloud Connector Administrators require a method to backup and restore the configuration of SAP Cloud Connector (SCC), including all custom paths, certificates, and system settings.
Special attention is needed for the props.ini file, as it is handled differently during the restore process.

Environment

SAP cloud connector 2.16 - 2.18

Cause

Configuration changes, system migrations, or upgrades require a backup of the Cloud Connector settings.
During backup and restore, the props.ini file requires special handling because it contains internal Cloud Connector settings that may differ between installations or hosts.

Resolution

Special Handling for props.ini
  • The props.ini file from the backup is treated differently:

    If the props.ini from the archive differs from the currently used file, it is placed next to the original file as props.ini.restored.

  • To use the restored props.ini:

    1. Replace the existing props.ini file on the OS level with the props.ini.restored file.

    2. Restart the Cloud Connector instance.

This ensures that the restored configuration, including internal settings, is applied correctly.
