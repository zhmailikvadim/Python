Symptom

You use the Cross Reference Fiori tile and expect the Restore function to clear the search results and reset the screen to its initial state. However, when the Restore function is executed, the previous results remain displayed and are not cleared. This leads to inconsistent behavior and may cause confusion when performing new searches.

This note corrects the issue by ensuring that the Restore function properly clears the results and resets the Cross Reference Fiori tile as expected.

Other Terms

/VTA/BXRM, SEARCH_DATA_PROCESS, Clearing, /IRM/GWSM, /VTA/BXRUPM

Reason and Prerequisites

This is caused by program error

Solution

Please implement the correction instructions. These corrections are only valid if you have applied the following Support Packs.

Release V/4 –  2023V2, 2023 V3, and 2023 V3 Support Package 1.
