Symptom

You would like to understand if there is an option to remove or hide the Personalization option, which is available for the agents under their profile in SAP Sales and Service Cloud Version 2.

Environment

SAP Sales and Service Cloud Version 2

Reproducing the Issue

  1. Logon to your SAP Sales and Service Cloud Version 2 environment.
  2. Click on the User Profile.
  3. See the option Personalization is displayed.

In this scenario, you would like to remove it, preventing agents from making any changes to the UI.

Resolution

This can be achieved by removing the personalizationService under the agent business role, which would make the option to not be displayed for such agent.

Another possibility would be to keep the service enabled in the business role, but restrict its Write Access and set it as No Access.
