Symptom

When Asset Requisition is Approved, ERP sends back push error saying "Asset Class is not entered"

Cause

The issue occurs when Asset Requisition is created with Tax Line. This is expected behavior and when Asset PR is created, Ariba doesn't store AssetClass details for the Tax Line.
This results in ERP sending back push error.

Resolution

If you expect to change in this system design, kindly raise an Enhancement Request
