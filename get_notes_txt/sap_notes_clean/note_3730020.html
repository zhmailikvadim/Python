Symptom

SAP note 3568494 delivered two percent options (100%, 81,5%) for recording supplementary maternity leave on infotype 3415.

During the supplementary maternity leave, 100% of the benefit is paid, therefore the 81.5% value is not relevant in any case.

Other Terms

IT3415, Supplementary maternity leave percent, ABSTY 26, AWART 0520.

Reason and Prerequisites

SAP note 3568494 has been applied.

Solution

The issue has been corrected.

Please apply the correction by importing the Support Package specified for your release, or by using the Note Assistant.
