Symptom

This note prevents any change of timestamps in database table BCA_BCT_UPD_MGMT for extraction with Init with Data Transfer (Mode of Data Update = S (Simulation of Initialization of Delta Transfer)) for datasources using timestamps for delta extractions.

Other Terms

BW, 0TB, Business Warehouse

UPDMODE

Solution

Import the relevant Support Package that is specified for your release. As a manual advance correction, you can implement the attached correction instructions using transaction SNOTE.
