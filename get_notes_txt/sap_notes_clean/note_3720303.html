Symptom

  • Unable to replicate tables from SAP HANA to GCP using SAP SLT due to missing filter options for design-time table resources defined in .hdbtable files.
  • The filter options tab is not visible in SLT-LTRS-Performance Options, preventing filtering of records greater than a specified date. 

Environment

SAP Landscape Transformation Replication Server

Cause

Key column is in the middle of the table.

Resolution

SLT for non-ABAP sources does not support replication for tables that do not have key fields at the beginning of the table.
See:
SAP Note 1768805 - SAP Landscape Transformation Replication Server (SLT): Non ABAP-based Sources lists the requirements for all non-abap source systems
