Symptom

Error encountered when using LMS IRT: instanceRefreshError-TARGET_SCHEMA_DROP_FAILED

Environment

  • SAP SuccessFactors Learning
  • SAP SuccessFactors HCM

Reproducing the Issue

  1. Schedule a Full Copy (Instance Refresh) via the SuccessFactors Instance Refresh Center from the source to the target, including Learning.
  2. Download and review the the Instance Refresh execution logs and observe the error “instanceRefreshError-TARGET_SCHEMA_DROP_FAILED”

Cause

  • A stale database connection to the target tenant persisted after the instance refresh started, preventing the refresh activity from completing and resulting to the TARGET_SCHEMA_DROP_FAILED error and data mismatch. 
  • This is a known intermittent issue.

Resolution

The fix is set to be implemented on the next release (1H2026 - Preview: April 13 Production: May 15-16)

Workaround:

  1. Schedule the Learning refresh via the Instance Refresh Tool; the failure is intermittent and may not recur.
  2. If the error persists, request to proceed with a Learning Manual Refresh by submitting the completed refresh form - 2165346 - Learning Management System Manual Refresh Process - SAP for Me
