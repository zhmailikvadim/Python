Symptom

How to confirm the list of users that were actually deleted in Import User Data (Consolidated File)?
 

Resolution

You can accurately confirm the users and the exact number that were actually processed using either of the following methods:
  • Contacting Ariba support team:​ We will investigate the details and report to you.
  • Checking via the Audit Log on your end:​ You can directly and promptly check the actual users processed by following the steps below.
Steps to check the Audit Log:
  1. Navigate to Core Administration > Intelligent Configuration Manager > Manage Configurations > Continue, and open the Audit Log.
  2. Set the search filters as follows and click Apply.
    • Event type:​ Data modification
    • Application:​ Buyer
    • From/To:​ Specify the date range you wish to check
    • Action:​ UPDATE
    • Operation:​ User Deactivated
  3. The search results will display a list of users who were actually deactivated during the specified period. You can also export a report by clicking the Export results​ button in the lower right corner of the screen.
