Symptom

What is the order of security validation when enabling both SDK Validation and IP Restrictions at the same time in Customer Data Cloud (CDC)?

Environment

  • SAP Customer Data Cloud
  • Account Login

Resolution

The order of security validation is as follows:

  1. IP Blacklisting Validation
  2. GMID Validation
  3. SDK Versions Validation
  4. IP Whitelisting Validation
