    "Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental." 

Environment

SAP S/4HANA Public Cloud 

Cause

Changes in the Map Data Format in one line will not be transferred to the other line via merge or rebase.

Resolution

See Restricted Features in the parallel line as described in 3680907 - Release Note for SAP S/4HANA Cloud Public Edition, Parallel Line with 2602 Release

As described in the SAP note 

Changes in the Map Data Format in one line will not be transferred to the other line via merge or rebase. You can use the "Export" functionality of the Map Format Data app.

Download the format mapping configuration from the source line and choose the "Import" button to upload it to the target line.

For more information, refer to Importing and Exporting Format Mappings.
