Symptom

Budget supplement amount is not reflected in the Available Amount column within the same budget document.

Environment

SAP S/4HANA Cloud Public Edition 

Reproducing the Issue

  1. Access the Budget Documents app in SAP S/4HANA Cloud Public Edition.
  2. Create a budget supplement for a project or WBS element.
  3. Observe the Available Amount column and the Amount in Global Currency column.
  4. Note that the supplemented budget is displayed in the Amount in Global Currency column, while the Available Amount column does not show a summed value of the previous budget and supplemented amount.

Cause

  • This behavior is by design in SAP S/4HANA Cloud Public Edition.
  • The system does not rewrite or sum values in the document line when a supplement is created.
  • The supplemented amount is added as a delta to the overall released budget, and the Available Amount is recalculated at runtime.

Resolution

  1. Understand that the Available Amount column reflects the resulting budget after release and does not display a running total per line.
  2. Refer to the Amount in Global Currency column to view the delta of the current budget document (supplement, return, transfer, etc.).
  3. To check the summed budget, use one of the following apps:
  • Project Budget Report
  • Project Control – Enterprise Projects
  • Project Financial View
