Symptom

During sales order costing, message CK877 (Sales order costing not possible for recursive configurable materials) is raised, if the same configurable material is included in the multilevel BOM on different levels but with different configuration.

Other Terms

CUOBJ, VA01, VA02, CK51N, recursion

Reason and Prerequisites

The check for recursive structure (introduced by SAP Note 2805521) does not consider multilevel configutration properly.

Solution

Apply attached correction.
