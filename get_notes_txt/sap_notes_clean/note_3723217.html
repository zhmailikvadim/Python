Symptom

Note 3624809 provides functionality to create a cascading balance carryforward in ECC (or SL in S4). However, FM G_RWIN_CASCADING_BCF doesn't return a list of the required or performed updates.

Other Terms

G_RWIN_CASCADING_BCF

Solution

Implement correction instructions. From now on, you can call G_RWIN_CASCADING_BCF and get the list of required or performed updates (depending on input parameter TESTRUN) in exporting parameter BCF_RESULT.
