Symptom

Logic of the Gantt Chart percentage calculation. 

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental. 

Environment

SAP Integrated Business Planning

Reproducing the Issue

  1. Login to IBP,
  2. Go to Manage Processes app,
  3. Copy an existing process,
  4. Make sure in that there is one task in any of the process step,
  5. Save the process and let it run,
  6. Check its Gantt Chart when the process is finished.

Resolution

The Gantt Chart shows the percentages according to the completed tasks within the process steps. Eg.:

  • If there are 5 tasks within a step, one is Completed but the other four is Open - Gantt Chart will show 20% for this process step.
  • If there are 2 tasks within a step, one is Completed but the other one is Open - Gantt Chart will show 50% for this process step.
  • Process steps without task assignment will show 0% till the step reaches it's end. When the step is completed, its percentage is shown as 100%.
