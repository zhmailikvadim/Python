Symptom

You are not able to complete an Organizational Change as some transfer documents are missing. Looking at the documents posted in a reprocessing step you see that these documents actually do not change the balances.

One root cause of that is that you have amount and currency information in your system (either starting balance or line items) that is not used (i.e. the currency type is not active). A transfer will not handle these amounts / currency.

With this note existing invalid amounts / currency will be ignored. We are more resilent w.r.t. database inconsistencies.

Other Terms

Organizational Change, FINOC 089, FINSC_LD_CMP, invalid amount / currency

Reason and Prerequisites

Program Error

Solution

Implement the Support Package or Correction Instruction
