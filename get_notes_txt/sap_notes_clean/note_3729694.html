Symptom

When displaying XML file of JP eDocument, node "Payment_means_code" is blank.

Environment

  • SAP ECC
  • SAP S/4HANA

Reproducing the Issue

  1. Run T-code EDOC_COCKPIT
  2. Choose JP eDocument
  3. Display XML file of the JP eDocument

Cause

AIF value mapping setting not done.

Resolution

Please refer to following help portal to do the setting:

ECC:

Configuring UNCL4461 Code List (PAYMENT_MEANS_CODE) | SAP Help Portal

S/4HANA:

Configuring UNCL4461 Code List (PAYMENT_MEANS_CODE) | SAP Help Portal
