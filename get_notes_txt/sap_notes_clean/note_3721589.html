Symptom

I see an option labeled DocuSign (Legacy) in the signature provider dropdown when attempting to send a contract for signature.

Environment

signature integration set up, Ariba Contracts Module

Reproducing the Issue

  1. Navigate to CW.
  2. Go to the Documents Tab.
  3. Ensure the document intended for routing is marked as Published.
  4. Click the dropdown arrow next to the document and select Create New Task --> Signature

Cause

DocuSign (Legacy) is no longer supported by Ariba E-Signature process.

Currently, Ariba integrates with DocuSign via OAuth service

Resolution

  1. Set the parameter Application.ACM.Signature.DocuSign.Enabled to No.
  2. Ensure that both Application.ACM.Signature.DocuSignREST.Enabled and Application.ACM.Signature.DocuSignREST.OAuth.Enabled remain set to Yes, as those Parameters will make sure the OAuth signature integration works properly.
  3. Confirm that the DocuSign (Legacy) option no longer appears in the signature provider dropdown.
