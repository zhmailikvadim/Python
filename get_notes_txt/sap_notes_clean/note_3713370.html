Symptom

Currently, IBGI (Invoice Before Goods Issue) is not supported for WM-managed delivery in the public cloud.  

Environment

SAP S/4HANA Cloud Public Edition

Cause

Invoice Before Goods Issue can be processed via scope item 7UA; however, it is still not supported for outbound delivery involving EWM scenario. 

Resolution

If this feature is something you're interested in, please raise a feature request on the SAP Customer Influence Portal or vote for this existing request https://influence.sap.com/sap/ino/#/idea/366721.

See Also

2963059 - How to Submit Feature Requests for SAP S/4HANA Cloud Product
