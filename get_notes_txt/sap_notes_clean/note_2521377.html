Symptom

We are facing the following issue for course launch in IE11 even though system check is all OK.

While launching a course users are getting error that "Browser does not support HTML5"

Environment

SuccessFactors Learning Management System (LMS) - All Supported Versions

Cause

Issue is that some HTML 5 elements of the content are not supported by the current browser

Resolution

Content will need to be reviewed to confirm what browser requirements are needed to run the content.

Browser HTML5 support can be tested at https://html5test.com

Keywords

HTML5; Content; Browser; Browser does not support; IE11;
