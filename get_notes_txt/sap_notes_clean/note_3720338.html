Symptom

Certain containers in the itemAssignments API call are missing Key values which cause some applications such as HCI to present errors when attempting to call the API.

Environment

SAP SuccessFactors Learning 1H 2025 and 2H 2025

Cause

This is a Known Issue.

Resolution

The fix for this issue is planned to be deployed on 1H 2026 release.

For release timelines, please review SAP SuccessFactors Product Release & Road Map Information.

Workaround

No workaround is available.
