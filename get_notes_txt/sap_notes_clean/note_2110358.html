Symptom

When you invoice a tax statement (invoicing function for creating a statement for tax amounts), the business place (field BRANCH) in the import parameter I_FKKKO of a function module for the event 1190 is not filled. As a result, a business place-specific determination of the official document number becomes more difficult.

Other Terms

Tax invoice, official receipt
BRANCH, BUPLA, official document number, ODN, event 1190
Invoicing, SAP Convergent Invoicing
FKKINV

Reason and Prerequisites

There is a program error.

Solution

Implement the attached correction instructions or import the specified Support Package.
