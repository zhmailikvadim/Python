Symptom

Data must displayed in ABAP list viewer but the top of page part is blank in both custom and standard transactions/reports.

Environment

  • SAP GUI for Windows release independent
  • SAP NetWeaver
  • SAP NetWeaver Application Server for SAP S/4HANA
  • ABAP PLATFORM - Application Server ABAP

Reproducing the Issue

Many transactions where an HTML control is used (SM50, SBWP, SOST, RZ11, SE80 etc..) are affected.

Cause

3rd party product issue.

Resolution

Read and follow KBA 3704912 - Latest Webview2 Version 144.xxx causes problems with SAP GUI HTML control when using edge as browser control in SAP GUI for Windows.

See Also

KBA 3704912 - Latest Webview2 Version 144.xxx causes problems with SAP GUI HTML control when using edge as browser control in SAP GUI for Windows
