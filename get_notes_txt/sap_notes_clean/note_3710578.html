Symptom

The Personalization option is not available across S/4HANA Fiori applications, including Service Orders, Service Quotations, Service Contracts, Service Confirmations, and Service Requests, for any action (Search, Display, Edit, or Create). 

Environment

SAP S/4HANA

Cause

Missing customizing

Resolution

  1. Execute SAP Transaction SM31
  2. Select the view PERSCV_PROFILE
  3. Enable the personalization

4. Afterward, login in the UI and you will be able to display the option Personalization

Keywords

PERSC_PROFILE,  note 3645900 - Personalization Function Profile New Table
