Symptom

You would like to know whether SAP ASE is impacted by the CVE-2026-21932.

Environment

  • SAP Adaptive Server Enterprise (ASE) 16.x
  • SAP Adaptive Server Enterprise (ASE) 16.x for Business Suite

Resolution

This vulnerability is in AWT and JavaFX components and none of SAP ASE and its tools use the impacted components.
So SAP ASE is not impacted by this CVE-2026-21932.
