Symptom

You declare your exports from SAP GTS to the French customs system DELTA Export.

After you receive the release (IE529) in the context of a simplified export declaration, you send a correction message (IE513).

This is rejected with reference to the following rules using error message IE556: R0883

Other Terms

FR, France, DELTA, export, LRN, MRN

Reason and Prerequisites

Program error: During the transfer of the change message, the specified LRN and the declaration type refer to the simplified customs declaration.

Solution

Implement the attached correction instructions or import the specified support package.
