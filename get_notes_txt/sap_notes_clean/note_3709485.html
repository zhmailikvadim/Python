Environment

  • SAP ERP Plant Maintenance (PM)
  • SAP ERP Customer Service (CS)
  • SAP R/3
  • SAP R/3 Enterprise
  • SAP ERP Central Component
  • SAP ERP
  • SAP Enhancement package for SAP ERP
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Enter transaction code IW32.
  2. Enter the order number and press Enter.
  3. Navigate to the operation tab.
  4. Select the operation and delete it.
  5. Confirm the deletion by selecting "Yes."
  6. Save the maintenance work order.
  7. Dump Occurs

Cause

  • There is an inconsistency with the Operation status. 

Resolution

  1. Use correction report AVO_STAT_CORRECT_ALL in transaction SE38/SA38 from SAP Note 675240 to correct these Operations
  2. Run the report in TEST mode to identify the inconsistencies
  3. The report can then be run in UPDATE mode to correct these inconsistencies
