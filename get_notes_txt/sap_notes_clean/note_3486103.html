Symptom

In the SAP Ariba Sourcing that is integrated with SAP ERP or SAP S/4HANA using SAP Integration Suite, managed gateway for spend management and SAP Business Network, when the buyer implemented the ARBCIG_BOM_MASTERDATA Badi, an empty SPROXY was being sent.

Other Terms

ARBCIG_BOM_REPLICATE, CL_ARBCIG_PUBLISHBILLOFMATERIALS, PREPARE_PAYLOAD, ARBCIG_BOM_MASTERDATA, PUBLISH_BILLOFMATERIALS

Reason and Prerequisites

This issue occured due to erroneous code. As a result, even when the LS_BOM_REQUEST parameter data was cleared, an empty SPROXY was being sent.

Solution

The code has been enhanced to address the issue. A parameter has been provided in BAdi to skip sending SPROXY or ZIP file export generated with payload.

Required Action

Apply the SAP note.
