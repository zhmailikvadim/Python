Symptom

You use the Run Statutory Reports app to create the VAT Declaration (Modelo 303) for Spain (report category ES_VAT_DCL_M303_NATIONAL). The TXT file does not match the format published for the year 2026.

Other Terms

ES_VAT_DCL_M303_NATIONAL, Document and Reporting Compliance, DRC, new file format, 2026, new tax box, fuel deduction, CNAE

Reason and Prerequisites

This SAP Note is relevant only for Spain.

Starting January 1, 2026, the Spanish tax authority requires a new field and tax box [112] for fuel deductions, reorders some output tax and equalization tax boxes, extends the CNAE code to four characters, and updates the determination rules for box [111] in form 303.

Solution

Report ES_VAT_DCL_M303_NATIONAL is adjusted to create files using the new file structure starting January 1, 2026.
