Symptom

You delete a partner function at supplier subrange or additional data level, error  'A purchasing info record exists for supplier sub-range' (WY053) is erroneously raised

Other Terms

WY 053,  VENDOR_EXTENSION_CHECK, VMD_SUBRANGE_CHECK, CL_IM_VMD_SUBRANGE_CHECK

Reason and Prerequisites

Program error

SAP Notes 3242532 and/or 3290934 or corresponding support package are implemented

Solution

Implement the attached correction instruction or equivalent support package
