Symptom

ESL code can retrieve variable value in preview but does not work in text-only version emails.

Environment

  • SAP Emarsys Customer Engagement

Reproducing the Issue

  1. Select "Send text-only" option in Email Settings tab.
  2. Check the email preview to confirm ESL code works.
  3. Launch the email and observed the ESL code does not set the value.

Cause

  • The ESL code to set value for variable is within an HTML comment. As a result, it is skipped when building the text-only version of the email. The preview is only for the HTML version, so the ESL gets loaded to retrieve the value for variable.

Resolution

Use an ESL snippet to set value for variable instead.
