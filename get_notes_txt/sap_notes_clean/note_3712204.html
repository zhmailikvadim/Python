Symptom

In the Migration Cockpit for the direct transfer approach, you are using the scenario SAP ERP to SAP S/4HANA to migrate master recipe master. You have noticed that phases, relationship are not migrated.

Other Terms

Migration Cockpit, SAP ERP to SAP S/4HANA, Master recipe, Migrate Data Directly from SAP System, Migrate Your Data, Direct Transfer, Transferring Data Directly from an SAP System, Master Recipe, CNV_PE_S4_MASTER_RECIPE_CREATE, Master recipe migration failed

Reason and Prerequisites

Reason - Program error.

Prerequisites - SAP Note 3649434 is implemented in your target system.

Solution

Implement the note in the target system, create a new project and try to proceed with the migration.
