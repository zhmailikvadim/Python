Symptom

When filtering by Pricing Unit on list view, you get an Invalid SQL error.

Reproducing the issue

Prerequisites:

You are working with Web Client for SAP Business One and using the Item list view.

Scenario:

  1. Select the Items by Price view/variant.
  2. In the Pricing Unit (Price) filter, set the value to 6Pack.
  3. Choose the Go button.

Expected behavior:

  1. The filter displays UOM Entry (value=2 in the example).
  2. No error occurs after you press the Go button. 

Actual behavior:

  1. The filter displays 6Pack﻿.
  2. An Invalid SQL error appears.

Solution

SAP intends to provide a patch or patches to solve the problem described. The section References below will list the relevant patches once they become available, and the Related Info content of relevant patches available in the SAP Support Portal will show this SAP Note. Be aware that these references can only be set at patch release date. SAP delivers patches only for selected releases at its own discretion, based on the business impact and the complexity of the implementation.
