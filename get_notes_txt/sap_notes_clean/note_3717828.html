Symptom

Job Seeker Upload via API using JSON format fails with error: Required data is missing for file type.

Cause

This error occurs when the Job Seeker Upload custom connector name does not match the connector name specified in the REST API URL.

Resolution

Update the custom connector name to ensure it matches the connector name specified in the REST API URL.

  1. Log in as a configuration manager.
  2. Navigate to the Connector Catalog and search for the Job Seeker Upload connector.
  3. Open the connector and click Edit.
  4. In Configure > Enable, update the Connector Name field to match the connector name used in the REST API URL.
  5. Click Next and proceed through the configuration steps.
  6. Save the connector setup and re-trigger the API request to confirm the issue is resolved.
