Symptom

Buyer users that should approve timesheets want to do so from a mobile device but are unsure how to access or use the SAP Fieldglass Manager Hub mobile application.

Resolution

Buyer  users can approve timesheets directly from the SAP Fieldglass Manager Hub mobile app after downloading the app and signing in using either a QR code (recommended) or username/password. All pending timesheets can be accessed and actioned under the Work Items section.

1. Download the App

  • iPhone users: Apple App Store
  • Android users: Google Play Store
    The app allows managers to approve/reject timesheets, review worker details, manage job postings, and act on work items.

2. Sign-In Options

Recommended: QR Code Login

  1. Log in to SAP Fieldglass (desktop).
  2. Go to My Profile -> Manager Hub.
  3. Generate a QR code.
  4. Open the Manager Hub app -> scan the QR code.
  5. Create a PIN when prompted.

This is the fastest and simplest method.

Alternative: Username & Password Login

  1. Open the app -> tap the wrench icon -> choose Alternative Setup.
  2. Enter:
    • Server URL
    • Username
    • Password
      (All available under My Profile in the desktop version.)

3. Approving Timesheets

  1. Open the app.
  2. Navigate to Work Items.
  3. Select the timesheet awaiting action.
  4. Review hours and comments.
  5. Tap Approve or Reject.

Additional Notes

  • Mobile services must be enabled for your company’s SAP Fieldglass tenant.
  • No extra configuration is required specifically for timesheet approvals.
