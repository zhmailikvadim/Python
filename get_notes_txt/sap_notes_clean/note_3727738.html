Symptom

When user settings are changed in results recording transaction (for example: QE01 or QE51N), below error is raised:

"Enter a valid value" (message nr: 00002)

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Environment

  • SAP ERP Quality Management (QM)
  • SAP ERP Central Component
  • SAP ERP
  • SAP Enhancement package for SAP ERP
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA
  • SAP S/4HANA cloud private edition

Reproducing the Issue

  1. Call transaction QE01.
  2. Click button User settings.
  3. Change any indicator on the screen.
  4. Click Save.
  5. Error "Enter valid value" is raised.

Cause

A SET/GET parameter relevant for results recording (for example: QFA and QFM) has value 'x' (lowercase character) for the user in the user maintenance.

Resolution

Maintain value 'X' (uppercase character) for the corresponding SET/GET parameter in transaction SU01 (tab Parameters).

Keywords
