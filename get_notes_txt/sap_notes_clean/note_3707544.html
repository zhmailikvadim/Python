Symptom

  • The requirement is to disable the "create activity" functionality in the performance form while enabling activities in Continuous Performance Management (CPM).
  • In the Performance tab, when opening the form and navigating to the Goal section, the side panel displays the Linked activities section along with the "create activity" button which needs to be disabled.

Environment

  • SAP SuccessFactors Performance Management
  • SAP SuccessFactors Continuous Performance Management

Reproducing the Issue

  1. Navigate to the Performance tab.
  2. Open the performance form.
  3. Go to the Goal section within the form.
  4. Open any goal.
  5. Observe the side panel where the Linked activities section is displayed along with the "create activity" button.

Resolution

  1. Navigate to the Form template settings.
  2. Disable the option "Enable Linked Activities and Achievements in Performance Goal Section".
  3. Note that it is not possible to disable only the "create activity" button. The entire Linked activities section can either be shown or disabled based on above setting.
