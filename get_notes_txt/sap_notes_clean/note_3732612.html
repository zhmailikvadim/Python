Symptom

  • The customer expects inflation revaluation to be calculated based on the Supplier Invoice Date.
  • In the system, report J1AI calculates revaluation using the Asset Value Date instead.
  • The customer reports that this behavior does not meet the local legal/business expectation for inflation accounting scenarios in LATAM countries such as Argentina.
  • The question may arise for both:
    • assets under construction (AuC), and
    • assets that were not managed as AuC.
  • The customer may also ask how SAP determines the supplier invoice date for asset revaluation, since there is no dedicated supplier invoice date field in the asset master for this purpose.

Environment

  • SAP S/4HANA
  • SAP ERP
  • Inflation in Asset Accounting 

Reproducing the Issue

  1. Create an asset (AuC or non-AuC) and post a supplier invoice.
  2. Maintain an Asset Value Date that differs from the supplier invoice date.
  3. Run report J1AI for inflation revaluation.
  4. Observe that the calculation uses the Asset Value Date, not the supplier invoice date. For AuC, after capitalization, the calculation restarts from the capitalization Asset Value Date.

Cause

  • Standard behavior: J1AI uses the Asset Value Date as the base date for inflation revaluation for both AuC and non-AuC assets.
  • When an AuC is capitalized, it is treated as a new asset with a new start date (Asset Value Date) for revaluation.
  • J1AI does not read a separate supplier invoice date field from the asset master.

Resolution

  1. If the revaluation must be based on the supplier invoice date, enter the supplier invoice date as the Asset Value Date during asset creation or acquisition posting.
  2. For AuC scenarios, at capitalization ensure the desired base date (for example, supplier invoice date) is maintained as the Asset Value Date on the resulting capitalized asset.
  3. Run report J1AI to perform the inflation revaluation using the maintained Asset Value Date.
  4. Validate that the applied indices and periods align with the maintained Asset Value Date.
