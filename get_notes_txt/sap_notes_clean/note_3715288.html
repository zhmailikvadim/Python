Symptom

On the Adobe printout of tax form IFT-1/R that was generated by report HPLCIFT1, the REGON number of the payer appears in cell B.9, which is unwanted.

Other Terms

IFT-1/R(17), IFT-1/R(18), HPLCIFT1, REGON, HRPL_IFT1_V17, SFPI_HRPL_IFT1_V17.XML.

Reason and Prerequisites

Error in the form interface.

Solution

The form interface called HRPL_IFT1_V17 has been corrected. This interface is used by both versions 17 and 18. 
The issue affects only the printout but not the generated XML.

The change is included in the Support Packages listed below.

If you decide to implement change on your system before you import the relevant Support Package then:

  1. Log in in Polish language into your system.
  2. With the help of transaction SFP upload and override the existing Adobe form interface HRPL_IFT1_V17 from the file SFPI_HRPL_IFT1_V17.XML located in the attached ZIP file.
  3. Activate the form interface.
