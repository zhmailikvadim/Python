Symptom

  • When running a query with an active hierarchy, the result for the "Not Assigned" node is displayed as blank.
  • In flat presentation, the values that belong to the "Not Assigned" node show the result star (*).
  • With the hierarchy active, it is expected that the "Not Assigned" node also displays the star (*) as a result however, it appears as blank instead.

Environment

  • SAP NetWeaver
  • SAP BW/4HANA

Reproducing the Issue

  1. RSRT -> execute the relevant query
  2. "Not Assigned" node is collapsed and display blank result.

Cause

The system behaves as designed.

Resolution

  1. After query execution, "Not assigned" node is collapsed by design. Expanding the "Not Assigned" node, the value with result as star (*) will be inside that.
  2. On a technical perspective, the star (*) is not considered a result. For this reason, it is displayed only at the specific value where it occurs, but it is not carried over to the hierarchy node.
