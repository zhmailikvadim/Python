Symptom

You are using Reusable Filter Variants functionality of Integration & Exception Monitoring app in SAP Focused Run (FRUN). You are trying to configure a Reusable Filter Variant of the Alerting and Scenario filter type.

You observe that you do not find value help for any filter parameter in the Reusable Filter Variants->Alerting tab.

Other Terms

AIM, FRUN, Focused Run, Integration & Exception Monitoring, Reusable Filter Variants, Variants, Alerting Tab, Alerting and Scenario Filters.

Reason and Prerequisites

This issue occurs because of a program error.

Solution

Apply the attached correction instruction(s). After this fix, the filter parameters will start having value helps.
