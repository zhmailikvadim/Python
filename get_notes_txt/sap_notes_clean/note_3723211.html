Symptom

A Service Request has been opened with ECS as per the process outlined in Note:

2720477 - How to access ECS Service Requests - SAP ONE Support Launchpad / SAP for Me


It is required to escalate / speed up / expedite the existing service request.

Environment

SAP Enterprise Cloud Services

Resolution

To speed up processing contact the Customer Interaction Centre (CIC).  For details on how to contact the CIC follow https://support.sap.com/contactus

No new case is required to be opened to speed up the service request.  If you separately need to open a case with ECS please follow Note 3399927 (How to create a case when support is needed for SAP Enterprise Cloud Services (ECS) system) using component: PCO-OPS 


Escalate questions related to your RISE engagement by reaching out to PC3 Mailbox: sap.ecs.private.cloud.customer.center@sap.com
