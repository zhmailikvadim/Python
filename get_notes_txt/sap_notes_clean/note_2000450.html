Symptom

The following error message may appear in log file Collabconsole.txt from folder \\Collaboration\log

28-Oct-2013 15:34:16 (TP-Processor2_26)]: Login failed for uPerformSystem
[28-Oct-2013 15:34:16 (TP-Processor1_26)]: Login failed for Anonymous
[28-Oct-2013 15:34:16 (TP-Processor1_26)]: File name:uXAPI.
[28-Oct-2013 15:34:16 (TP-Processor1_26)]: Login failed for uPerformSystem com.intraspect.security.auth.AuthenticationException: Login failed for uPerformSystem

Environment

This may happen from uPerform 4.5

Reproducing the Issue

The error may appear when publishing documents

Cause

The user uPerformSystem has not been assigned a password.

Resolution

  1. Login to uPerform using a uPerform administrator account
  2. Go to Administration->Server Administration->Integration Settings->Integration Credentials
  3. Click on uPerformSystem
  4. Click on Edit Credentials
  5. Type in any password and click Save (Note: you will not need to remember this password)
  6. Click Up
  7. Repeat steps 3-5 for the SolutionManager account
  8. Republish document

Keywords

Login failed for uPerformSystem, uPerform, publishing
