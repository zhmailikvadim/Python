Symptom

• Error occurs when creating a new time slice for a valuation rule in the current fiscal year.  
• Error message displayed: "AY257 Interval in closed fiscal year: editing not possible".  
• Issue arises while changing the account assignment object.  
• Valuation rule uses a contract object as the valuation object.  
• Account assignment is derived from the contract object instead of being maintained in the valuation parameter.  

Environment

  • Flexible Real Estate Management (RE-FX)
  • Contract and Lease Management
  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA Finance
  • SAP S/4HANA

Reproducing the Issue

1. Open transaction RECN.  
2. Modify the account assignment object of the contract object in the objects tab of the contract.  
3. Add a new time slice for the valuation parameters in the open fiscal year.  
4. Observe the error message "AY257 Interval in closed fiscal year: editing not possible".  

Cause

• Error AY257 occurs due to the account assignment not being managed as time-dependent for the contract object.  
• Changes to the account assignment impact historic time slices of valuation parameters when derived from the contract object

Resolution

To resolve the error message, please proceed as follows:

1. Maintain the previous account assignment directly in the valuation parameters.  
2. Ensure it is no longer derived from the contract object.
