Symptom

During the initial stock transfer for materials in an FTZ, the goods value is missing in the internal customs process document. This SAP Note contains the correction for the GTS system.

Other Terms

/SAPSLL/BWHSTOCK_INIT_TRANS_R3

Reason and Prerequisites

Program error.

Solution

Implement the attached correction instructions and also implement SAP Note 2660999 in your feeder systems.
