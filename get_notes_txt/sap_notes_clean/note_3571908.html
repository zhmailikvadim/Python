Symptom

SupplierConsolidated_Delete file failed to import in Ariba with Error: Value for lookupKey VendorID is invalid which has been received from SAP ERP master data export program ARBCIG_MASTER_DATA_EXPORT.

Cause

Customized FIELD_ARIBA entry: Original vendor No,  under ARBCIG_FILEDMAP table caused the issue.

Resolution

Change FIELD ARIBA value in ARBCIG_FILEDMAP table in ERP to VendorID instead of Original vendor No as Ariba expect VendorID field for SupplierConsolidated_Delete.
