Symptom

  • An error occurs when attempting to sign in to Fieldglass via SSO. The error message states: The sign-in attempt has failed.

Reproducing the Issue

  1. Attempt to log in to Fieldglass via SSO using the username format username_company code.
  2. Observe the error message: The sign-in attempt has failed.

Cause

The issue was caused by an incorrect username format. The username format should follow the pattern username_company code for the particular buyer, but the user was using username_incorrect buyer code. For example, if the buyer/company code is xxyy, the username should be username_xxyy

Resolution

1. Verify that the username format being used for SSO login by other users is valid (i.e., username_company code). 
2. Update the username to the correct format username_company code. In this case, the correct format is username_xxyy.
3. Update the username by attempting to log in to Fieldglass via SSO. Confirm the login is successful.
