Symptom

The (FAIR-based) process "DL Automatic Assignment" (/IBS/CAB_AUTO_ASSIGN) does not generate a work item for a manual check for corrected CFS investor accounts for which the "Rule Exclusion" indicator is set and there is a balance change.

Reason and Prerequisites

The status "Corrected" of a CFS investor account in combination with the "Rule Exclusion" indicator is not taken into account when checking for the requirement of a work item in the business function "Select CF Investor Accounts" (/IBS/CAB_SEL_INVES_ACCOUNT). Therefore, no work item is created, even if the assignment status is set to "Check Assignment".

Solution

The status "Corrected" is set in the implementation /IBS/CL_CAB_SEL_INVES_ACCOUNT of the above-mentioned BF added.

In addition, the new parameter type "Ind. Corr. WI Supp. Exclude Rule Accts" (/IBS/CAB_AS_WI_ITEM_REPAIRED), you can change the work item in the constellation.
To do this, add the settings in the parameter service (transaction /FAIR/PARAMETER) as required after you implement this SAP Note. If the process is started with a parameter ID, a setting must be maintained.
