Symptom

Field ‘Reopen after TECO’ in transaction /PACG/ECM_SCSTD serves the following purpose: if enabled, reopen an activity after cancelling service order technical completion. This behaviour would only be performed for old activities (sales activities). Implement this note to use this functionality for new activities (E4C activities).

Other Terms

E4C, connector, FSM, TECO, activity

Reason and Prerequisites

This is caused by program error.

Solution

Please, implement this correction instruction.
