Symptom

Access to the Key Extensibility application is stuck in loading or failed with error and ATO / VIRUS SCAN / OData were configured properly.

Environment

  • SAP S/4HANA 
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Access to Key Extensibility application (such as Custom Analytical Queries, Custom Business Objects, etc.).
  2. The application will either be stuck in loading or failed with errors although ATO / VIRUS SCAN / OData were configured properly. For example, the console log notifies /sap/opu/odata/SAP/ANA_QUERY_DESIGNER/AdtSet error.

Cause

There can be a race condition when the app loads third-party dependencies which can cause the app to load infinite or stop and show the error.

Resolution

  1. Access the affected application. It will still show the error/loading page. This is a mandatory step.
  2. Try either of the below actions:
    • Navigate to other application (which use the same third-party dependencies), then back to the affected application.
    • On the affected application, reload the browser.

Note that this will work as workaround, and need to repeat the action whenever access the affected application.
