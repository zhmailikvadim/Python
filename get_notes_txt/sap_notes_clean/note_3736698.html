Symptom

When editing the hostname of the C4C communication system, an UI popup dump error is raised.

Environment

SAP Cloud for Customer

Reproducing the Issue

  1. Open the relevant communication system and edit the hostname.
  2. Observe that an error is displayed and the update does not complete when saving it.

Cause

A blank communication arrangement associated to the communication system exists, which leads to the error during the system communication update.

Resolution

  1. Identify the communication arrangements in the tenant and locate any blank (empty) communication arrangement.
  2. Delete the blank communication arrangement.
  3. Retry the system communication update and confirm that it completes without error.
