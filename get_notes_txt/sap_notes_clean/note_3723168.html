Symptom

When enrolling an employee into a benefit, the '???BENEFITS_PERSON_RELATIONSHIP_IS_NULL???' error is received.

Environment

SAP SuccessFactors Employee Central

Reproducing the Issue

  1. Log into environment
  2. Go to example user's profile
  3. Go to the 'Benefits' portlet/card, go to a Dependent-based benefit and try to enroll
  4. Receive error

Cause

The employee has an existing enrollment for this Benefit already (e.g. for a previous schedule period), with the Dependent noted within it - however, some of the Dependent's details (e.g. their name) are missing in the enrollment record.

Resolution

The existing enrollment needs to be found and updated so that the Dependent's details are all filled in appropriately (e.g. their name needs to be filled in)
