Symptom

Miscellaneous Invoice > Miscellaneous Invoice Details section shows Miscellaneous Invoice code.

The character length of the Miscellaneous Invoice code is set to 100.

You want to change the character length of the code.

Cause

This is intentional based on system design.

Resolution

There is no way to change the character length of the Miscellaneous Invoice code.

Keywords

miscellaneous invoice code misc invoice character length change restrict
