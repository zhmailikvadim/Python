Symptom

After changing the description of the billing element within a project in the "Plan Customer Projects" app, the updated description is not displayed correctly in relevant CDS Views, the "Manage Project Billing" app, and subsequent invoices. 

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Navigate to the Billing tab of the project in the "Plan Customer Projects" app
  2. Change the description of the billing element
  3. Observe that the previous value for description is still displayed in relevant CDS views, the "Manage Project Billing" app, and subsequent invoices

Cause

This is a known regression issue due to a defect in class CL_PS_STRUCTURE.

Resolution

The correction was delivered with HFC05 over the weekend of March 1st. For the existing incorrect data, open the project in the "Plan Customer Projects" app and update the description of the billing element in the Billing tab.
