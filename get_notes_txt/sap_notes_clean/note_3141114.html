Symptom

You are in the Manager Hazardous Substance Inventory App and want to add an additional Substance.

In the Add Substance dialog you want to search by entering a Material Number.
When you execute the search with a Material Number the search did not return any result. In the backend you get a dump.

Please note:
This SAP note does not update the SAP Gateway Service Builder project for service EHHSS_HSM_SI. It contains only the generated artefacts.
If you need an updated version of the project (e.g. to create own services based on it) you need to wait for the next service pack which contains the updated project.

Reason and Prerequisites

Reason: Program error.
