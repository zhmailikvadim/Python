Symptom

This SAP Note delivers a new report to support new Geppen Form (Paper) and the employees over the age of 70 with Individual Numbers.

For more details, you may refer to the following Launchpad page:

https://launchpad.support.sap.com/#/legalchangenotification/detail/GSCBAPJ-1737

<< Valid release >>
All Valid Releases

<< Contents of the legal change >>

  • New transaction PC00_M22_LSIE
  • New report RPLSIEJ0
  • New BAdI Definition PAYJP_RPLSIEJ0 in the enhancement spot PAYJP_SI_REPORT
  • New configuration in maintenance view V_T5JTU for the transaction above

Other Terms

RPLSIEJ0, PC00_M22_LSIE, PJP_SI_OFFICE, PJP_LSIE_COM, PJP_LSIE_EE, PJP_SELECT, RPLSIXJ0

Reason and Prerequisites

This is a Legal Change.

Solution

Apply the HR Support Package or implement the correction instructions together with Manual-Steps in this SAP Note.

After this revision, you use the new report RPLSIEJ0 to create new paper of Geppen form with the new format since year 2018.

The following new BAdI method provided in the SAP Note can be used to modify the entire data table for Geppen Form data, before output the data to the paper or in the ALV list.

  • IF_EX_PAYJP_RPLSIEJ0~EDIT_RECORDS

The old report RPCSIEJ0 can be used to generate paper form in the old format before year 2017.
