Symptom

  • After configuring the connection between SAP Service Asset Manager and FSM, the error message "CM-12: Cross service communication error" is displayed when using the 'Connection Test' option. 



Environment

  • SAP Service and Asset Manager 
  • Field Service Management

Reproducing the Issue

  1. Configure RFC destination to establish the oAuth connection between the backend and Field Service Management. 
  2. Use the following parameters: Target Host: auth.coresuite.com

Cause

  • The domain auth.coresuite.com is outdated and no longer supported for connecting SSAM with FSM.

Resolution

  1. Update the Target host field to the new domain: us.fsm.cloud.sap.
