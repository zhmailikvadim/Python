Symptom

This SAP Note enhances the payroll log generated by the HR-Malaysia: Process STD - Malaysian tax (LTAX) payroll function when you run Payroll driver for Malaysia (RPCALCL0).

The following logs are added into the payroll logs of payroll function LTAX when it calculates other tax employee deduction (/432) wage type in an original payroll run for payroll periods after 10.2009 for resident employees:

{Formula: Total paid accumulated STD + [STD for current
month excluding additional remuneration * ( n + 1) ]}

STD for month excluding add. rem. ...........         100.00 MYR
(n + 1) ........................................................        *  2.00

Projected STD in remaining months ............         200.00 MYR
Accumulated STD paid for previous months (X) + 300.00 MYR

STD for year(exclude current month add. rem.)    500.00 MYR

Refer to the online Monthly Tax Deductions calculator at https://calcpcbplus.hasil.gov.my/HITS_CE/MainScreen.

Other Terms

Monthly Tax Deduction, MTD, LTAX, HR-Malaysia: Process STD - Malaysian tax, Payroll log, RPCALCL0, Payroll driver for Malaysia, /432

Reason and Prerequisites

This is a documentation enhancement related to SAP Note 1380989.

This SAP Note has no impact on the payroll result.

This SAP Note has no impact on the payroll log of retrospective payroll run.

This SAP Note has no impact for Non-Resident employees.

Solution

You can apply HR Support Package or implement instructions in this SAP Note.
