Symptom

In the process "Start of Basic or Backup Supply for Gas (New SUP View)" (DE_B_GELIGAS_SBS_SUP), step 805 never ran because preconditions were missing.

This issue is relevant for the market role: SUP.

Other Terms

SBS, SOS, DE_B_GELIGAS_SBS_SUP, precondition, /US4G/3_0_SUP_SP07_NOTE_3707555, step 805, basic supply, backup supply, gas

Reason and Prerequisites

The error occurred because step 805 was missing the required preconditions.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the system runs step 805 whenever needed. Activate the BC set /US4G/3_0_SUP_SP07_NOTE_3707555. You need to rerun the process. You can verify the fix by checking that step 805 runs successfully.
