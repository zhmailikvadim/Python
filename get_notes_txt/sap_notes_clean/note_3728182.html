Symptom

I am synchronizing from Solution Manager to Signavio using BPM connector. In my sync protocol I can see the following warning message: 

Sync Skipped for ObjectType: DIAGRAM BPMN xml data is not available. Diagram was skipped with name XXXXXXXX and ID ZZZZZZZZ

Resolution

The warning means that the given diagram in Solution Manager doesn't have a proper BPMN XML data stored in Solution Manager and therefore cannot be synchronized to Signavio.

Solution Manager will create the missing BPMN XML data, on the download of process document as Microsoft Word file. This option is given in Solution Documentation menu 'Global functions' > 'Process Document' (option 'Download as Microsoft Word Document'). Please note that this export can take a lot of time depending on the size of the given solution. Using a scope could help narrowing the amount of data to process.
