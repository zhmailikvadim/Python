Symptom

Additional information for manual options for table splitting using EUCLONEDEFS_ADD.LST file for SUM DMO

Environment

  • SAP NetWeaver
  • SAP ERP
  • SAP S/4HANA

Reproducing the Issue

  1. Start the SUM tool and proceed DMO migration.
  2. using EUCLONEDEFS_ADD.LST file for manual options for table splitting

Resolution

  1. For SUM tool 1.1, edit file path: /<SUM_DIR>/abap/bin/EUCLONEDEFS_ADD.LST 
  2. For SUM tool 2.0, edit file path: /<SUM_DIR>/abap/control/EUCLONEDEFS_ADD.LST.

For detailed steps on editing file EUCLONEDEFS_ADD.LST refer to below note:

SAP Note 3386079 - SUM DMO scenarios: Manual Options for Table Splitting and Table Migration

See Also

    SAP Note 3474736 - Database Migration Option (DMO) of SUM 1.1 SP04
