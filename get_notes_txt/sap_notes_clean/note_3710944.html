Symptom

  • When the Web Intelligence report is refreshed using manually entered prompt values (for a value which is not present in backend database), the report returns no data.

Environment

  • SAP BusinessObjects Business Intelligence Platform 4.3
  • Web Intelligence (Webi)

Reproducing the Issue

  1. Log in to the BI Launchpad.
  2. Open the Web Intelligence (WebI) report that contains prompts.
  3. In the prompt screen, manually enter a value that does not exist for the corresponding prompt‑related dimension (i.e., a value that is not present in the database).
  4. Run/Refresh the report.
    >> Observe that the report returns no data.

Cause

  • In SAP BusinessObjects 4.3, the system enforces a stricter interpretation of prompt parameters.
  • If the value defined in the WebI report is invalid, the system executes the query with the wrong value, resulting in a blank report.

Resolution

This is by design behavior as the value does not exist for that prompt variable.

Keywords

Webi, Report, prompt, no value
