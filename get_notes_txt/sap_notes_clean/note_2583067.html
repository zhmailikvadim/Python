Symptom

This article explains what types of notifications are sent by Recruiting Marketing and how to change the alert template.

Environment

 SAP SuccessFactors Recruiting Marketing 

Resolution

Only the Job alerts are sent by RMK.

The rest of the emails are generated from RCM (Recruiting Management). This includes the Welcome email and password reset.

The job alert template configuration is done in CSB:

  1. Open Career Site Builder
  2. Open Layouts
  3. Select Job Alerts Email Template
  4. Populate the required content for each configured locale
  5. Save

Note that only tokens from the available list can be used in the email. No custom tokens can be created.
You can add some formatting to the token, but it needs to be included, like this example :
[[jobs ::bullet=&#8226;:: ::title:: - ::location::]]
