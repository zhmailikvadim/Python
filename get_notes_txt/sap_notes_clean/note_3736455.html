Symptom

  • When selecting Create in the reimbursement application, the error message displays: "SAP has sent invalid response."
  • The issue occurs in case of application start from Fiori Launchpad. 
  • The overview page loads, but the error appears only when attempting to create a request.

Environment

  • Product: SAP S/4HANA
  • Advanced Claims

Reproducing the Issue

  1. Open the reimbursement application.
  2. Click Create.
  3. Observe the error: "SAP has sent invalid response."

Cause

Resolution

  1. Follow the recommendations as mentioned in the below  KBA
  2. 3661016 - webdynpro application server error when creating claims in fiori launchpad.
  3. Recheck the application after following the steps.

See Also

  • Refer to: 3661016 - webdynpro application server error when creating claims in fiori launchpad.
