Symptom

You are using ABAP Data Integration to replicate the data into SAP Business Data Cloud. There are new S/4 HANA Data Products that are built based on SQL services. Transaction DHADM does not allow you to perform expert functions on the subscriptions created for those data products.

Other Terms

S/4HANA, ABAP Data Integration, SAP Business Data Cloud, APRS, DHADM

Reason and Prerequisites

The ABAP-based source system is an S/4HANA 2025 SP01 system.

Solution

Implement the corrections in the ABAP-based source system.

Update the respective roles that include DHADM transaction.

Refer to note 3572314 for more information.
