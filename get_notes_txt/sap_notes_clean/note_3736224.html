Symptom

When executing an application in SAP Lumira Designer 2025, the application fails with an “Unknown Error” message if a global variable with the Array property enabled is used.

Environment

Lumira designer 2025

Reproducing the Issue

  1. Create a simple application in Lumira Designer 2025.
  2. Create a global variable.
  3. Set the variable type to Array.
  4. Execute the application.

Cause

This issue is caused by a product defect in SAP Lumira Designer 2025 where global variables defined as Array types are not handled correctly during application execution, leading to a generic “Unknown Error” at runtime.

Resolution

The issue has been identified and fixed by SAP Development.

The correction will be delivered in:

  • SAP Lumira 2025 – Patch 1
  • Planned release: Q2 2026
  • Customers should apply Lumira 2025 Patch 1 or higher once available.

Keywords

Lumira 2025, array, global variable
