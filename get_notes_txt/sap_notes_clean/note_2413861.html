Symptom

Even in non-HANA 7.50 Systems, ADSO is offered in the data flow generation wizard.

Reason and Prerequisites

Program Error

Solution

  • SAP BW 7.50
    Implement Support Package 7 for SAP BW 7.50 (SAPK-75007INSAPBW) into your BW system. The Support Package will be available as soon as SAP Note 2386795 with the short text "SAPBWNews 7.50 BW ABAP SP7", which describes this Support Package in more detail, is released for customers

  • SAP BW 7.51
    Implement Support Package 2 for SAP BW 7.51 (SAPK-75102INSAPBW) into your BW system. The Support Package will be available as soon as SAP Note 2386797 with the short text "SAPBWNews 7.51 BW ABAP SP2", which describes this Support Package in more detail, is released for customers.  

In urgent cases you can use the correction instructions.

Before you use the correction instructions, make sure that you check SAP Note 1668882 and SAP Note 2248091 for transaction SNOTE.

This SAP Note might already be available before the Support Package is released. In this case, however, the short text still contains the term "preliminary version".
