Symptom

Implement, upgrade or install SAP Commerce Cloud, composable storefront 221121.7.

Other Terms

General Information 

Composable storefront is a lean, Angular-based JavaScript storefront for SAP Commerce Cloud. Composable storefront communicates with SAP Commerce Cloud exclusively through the Commerce REST API.

Summary of New Features in Composable Storefront 221121.7

Title

Notes

Framework Update: Angular 21

Take advantage of new features in Angular 21, such as standalone components, with the once-a-year update to the composable storefront framework.

For more information, see Update Release 221121.7.

Open Payment Framework Global Functions

When configuring your Payment Service Provider or Gateway, speed up the configuration of client-side scripts with the use of open payment framework global functions .

For more information, see Open Payment Framework Global Functions.

 

Reason and Prerequisites

Install and upgrade to the February 2026 release of SAP Commerce Cloud 2211-jdk21.7, or the February 2026 release of SAP Commerce Cloud 2211.49.
