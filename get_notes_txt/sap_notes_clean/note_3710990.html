Symptom

The following errors show up during the migration of HU stock under the “Warehouse Stock” migration object when an internal number range is used.  

Error in items assigned to this header. Stock items were not uploaded. - Message No. /SCWM/STOCK_UPLOAD 81
Please enter external numbers. - Message No. /SCWM/HU_WM 141

Environment

SAP S/4HANA Cloud Public Edition

Cause

Packaging material type configuration is incomplete.

One possible cause is that no internal number range interval has been assigned for the affected packaging material types.  

Resolution

  1. Go to configuration activity 103418 "Assign Number Range Intervals to Packaging Material Types".
  2. Assign an appropriate internal number range interval for the affected packaging material types.
  3. Save the configuration.
