Symptom

The learning notification ID "ApmTrainingOverdue" does not include details of overdue curriculum requirements.

Environment

SAP SuccessFactors Learning

Reproducing the Issue

  1. Configure the "Learning Expiration Email Notification" APM without user as recipients selected >
  2. Set item-level configurations to override the APM default settings >
  3. Assign a curriculum with requirements to a user >
  4. Run the "Learning Expiration Email Notification" APM job >
  5. Check the received notification email >
  6. Observe that the curriculum requirement details are missing from the notification.

Cause

The system is working as intended. 

If the item level notification configuration is selected to send to the user as a recipient (overriding the APM configuration), the curriculum requirement details are not included in the "ApmTrainingOverdue" notification.

Resolution

To include the curriculum requirements in overdue learning notification, ensure that the user is enabled as a recipient in the APM level.

However, if you would like to change the system behaviour, we recommend you to create an enhancement request or vote in the existing one 329263.
