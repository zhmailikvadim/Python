Symptom

In Item-based accounting, after maintained only AAGC or AAGM in entries of T-Code VKOA <001 Cust.Grp/MaterialGrp/AcctKey>, below error message will be shown after creating new service order.

  • Error message: "Item XX: No revenue account can be determined for pricing element XXXX"

Environment

SAP S/4HANA

Reproducing the Issue

  1. Maintain only AAGC/AAGM in T-Code VKOA <001 Cust.Grp/MaterialGrp/AcctKey>
  2. Open the Fiori Launchpad, App: Manage Service Orders.
  3. Create Service order
  4. The error message will appear.

Resolution

To use only AAGC/AAGM to determine revenue account, please also maintain the entries in T-code: VKOA 002 Cust.Grp/Account Key or 003 Material Grp/Acct Key based on the using circumstance.
