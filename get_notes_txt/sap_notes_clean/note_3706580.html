Symptom

The need to determine whether a Switch BC Set contains client-dependent data or cross-client (client-independent) objects. 

Environment

This KBA applies to all SAP Products based on NetWeaver and S4H.

Cause

The method is not known.

Resolution

  1. Execute transaction SFW_BROWSER
  2. Expand the tree to the Switch BC Set in question
  3. Double click the Switch BC Set 
  4. In screen Activation of Switch BC Sets in frame BC Set Information the Client-Dependency is shown
    Cross-Client
    or
    Client-Specific

or

  1. Execute transaction SE38
  2. Enter Program SCPR_ACTIVATE_SBCSETS
  3. Press icon Execute  (F8)
  4. Enter the name of the Switch BC Set in field BC Set: in frame BC Set Information
  5. Press icon Enter  (Enter)
  6. In screen Activation of Switch BC Sets in frame BC Set Information the Client-Dependency is shown
    Cross-Client
    or
    Client-Specific

Keywords

switch bc sets, transport request, inactive status, manual activation, cross-client, SCPR_ACTIVATE_SBCSETS, SAP Note 1641394, SAP Note 2517797, business configuration sets, bc sets activation
