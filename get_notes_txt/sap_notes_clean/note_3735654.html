Symptom

When adding items to a software collection in "Export Software Collection" application, there is no option visible to "Select All" items.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Open the "Export Software Collection" app and navigate to a software collection.
  2. Click the "Add Items" button.
  3. Observe that there is no "Select all" option.

Resolution

The application "Export Software Collection" does not offer a "Select all" feature by design to prevent very large collections that can lead to dependency and transport issues.
