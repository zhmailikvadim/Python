Symptom

This article describes functionality introduced in the May 2026 SAP Fieldglass release

You may notice changes in how time is calculated, displayed, or validated on the Resource Timesheet

Common scenarios may include:

  1. Time categories such as Standard Time, Overtime, or Double Time look different than before
  2. Time category values change after editing time
  3. The Submit button is unavailable when entering time
  4. Upload files fail with a Time Category Code message
  5. The Resource Timesheet Details page shows new sections or is missing information
  6. Holiday, premium day, or per diem time does not apply as expected
  7. Adjust Time Worked changes time category results

Reproducing the Issue

You may encounter this behavior when:

  1. Opening a Resource Timesheet that now displays time categories automatically
  2. Adding or removing time entries and seeing category values update
  3. Attempting to submit time when entered hours exceed allowed limits
  4. Uploading Resource Timesheet or time entry files that include time category values
  5. Viewing the Resource Timesheet Details page
  6. Entering time on holidays, premium days, or nonworking days
  7. Using Adjust Time Worked to add or remove hours

Cause

SAP Fieldglass introduced enhancements to improve how time is categorized and processed on the Resource Timesheet

As a result:

  1. Time categories are calculated earlier and displayed automatically
  2. Editing time can trigger immediate updates to time category results
  3. Submitting time requires entered hours to align with allowed category limits
  4. Upload files must use valid Time Category Codes
  5. The Details page displays category information only when enabled
  6. Holiday, premium, and per diem time depend on schedules and calendars
  7. Adjusting time recalculates applicable time categories automatically

Resolution

(New Behavior)

  1. Time categories may appear immediately when the Resource Timesheet opens
  2. Editing time may automatically update time category values
  3. The Submit button becomes available when entered hours match allowed limits
  4. Upload files must include the Time Category Code defined for your site
  5. Time Category breakdowns appear on the Details page when enabled
  6. Holiday, premium, and per diem time apply when schedules and calendars are set up
  7. Adjust Time Worked recalculates time categories based on added or removed hours
