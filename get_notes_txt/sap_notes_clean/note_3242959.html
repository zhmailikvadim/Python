Symptom

When the server starts up, the runtime error SYSTEM_SHMCONSTR_ILL_STATEMENT is logged while trying to build the CL_CRM_ORGMAN_SHM_BUFFER_AREA shared memory area.

Other Terms

SYSTEM_SHMCONSTR_ILL_STATEMENT, shared mameory area, area constructor, area compilation with system user, CL_CRM_ORGMAN_SHM_BUFFER_AREA

Reason and Prerequisites

A strict check in the Shared Objects Framework (BC-ABA-LA) results in a termination during the advance filling of a shared memory area. More information is available in note 2881127.

Solution

Implement the attached manual activity or install the equivalent Support Package.
