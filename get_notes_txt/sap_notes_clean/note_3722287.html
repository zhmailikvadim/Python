Symptom

500 internal server error occurs when executing the Follow-On order 

Category               ABAP programming error
Runtime Errors         UNCAUGHT_EXCEPTION
Except.                /PLMU/CX_FRW_SERVICES_ERROR
ABAP Program           /PLMU/CL_FRW_LAUNCHPAD_INT====CP
Application Component  CA-EPT-SPI-FSI

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental." 

Environment

  • SAP ERP Plant Maintenance (PM)
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Access the WDA application Change maintenance Order application 
  2. Access 'Additional Functions' ->'Create Follow-on Order'
  3. Observe the 500 internal server error with the error message: Error in Application Ul: Invalid parameter value provided

Cause

Missing configuration.

Resolution

  1. Go t-code :LPD_CUST
  2. From menu 'Launchpads'->'Available Application Types' ,Select checkbox of 'Object Based Navigation'

Keywords

Modify order application, /PLMU/CL_FRW_LAUNCHPAD_INT====CP, RAISE EXCEPTION TYPE /plmu/cx_frw_services_error
