Symptom

The employee-position relationship is not created in iDocs when PROOF is set to 'N'. This results in incomplete data synchronization and may affect organizational management processes.

Environment

  • SAP S/4HANA on-premise
  • SAP S/4HANA private cloud edition

Cause

A bug causes the employee-position relationship not to be created in iDocs with PROOF = N. This issue is addressed by SAP Note 3526541.

Resolution

Install SAP Note 3526541 to fix this bug. Although the note does not explicitly mention the issue with the employee-position relationship, it resolves the underlying cause that leads to this side effect. Ensure to retest the scenario after installing the note.

Keywords

iDoc synchronization, employee-position relationship, PROOF = N, SAP Note 3526541, organizational management, data synchronization
