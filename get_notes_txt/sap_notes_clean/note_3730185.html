Symptom

There is a need to change the property "jdk.serialFilter", but it is missing.

Environment

SAP JVM

Resolution

This property was introduced by Oracle in Java 9, and later Oracle ported it to Java 8. Hence as the SAP JVM is based in the Oracle JDK, only starting on SAP JVM 8.1.029 this property is present.

See Also

2219315 - Mapping of SAP JVM patches to Oracle JDK updates
