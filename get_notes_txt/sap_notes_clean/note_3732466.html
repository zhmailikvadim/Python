Symptom

Information is required about the installation of CPILint in SAP Integration Suite.

Environment

  • SAP Integration Suite

Resolution

CPILint is a GitHub project, not an official/standard SAP provided add-on, therefore there is no official Note or KBA to instruct about its installation.

For further information, check the Community post below and the GitHub project page and FAQ:

  • CPILint version 1.0.5 is here - SAP Community
  • GitHub - mwittrock/cpilint: Automating your SAP Cloud Integration governance since 2019.
  • FAQ · mwittrock/cpilint Wiki · GitHub

Note: this add-on is not provided nor developed by SAP. Therefore, SAP does not provide support for this tool or for problems in it or caused by it.
