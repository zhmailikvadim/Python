Symptom

You are running the SAP license audit with System measurement (USMM) and for unit 0350 no engines are used, therefore the result for these units is 0, but unit for database size returned with value.

Other Terms

Audit, cnv_mbt_ld_audit, 0350, Unit, database size, system measurement, sap license audit, LAW, USMM

Reason and Prerequisites

The database size is determined independently of existing or non existing engines.

Solution

With the correction the database size will only be determined if at least one engine is used.
