Symptom

This SAP Note addresses an issue with The Payment List of Withholding Income Tax (HKRCYEG0) report. The Leaving Date field on the legal receipt doesn't populate correctly for active employees when a BAdI implementation exists for BAdI HRPAD00_ENTRY_LEAVE.

Other Terms

YEA, Hiring Date

Reason and Prerequisites

This is a program error.

Solution

You can apply relevant HR support package or correction instructions in this SAP Note.
