Symptom

  • New button "Upgrade Required" in business data cloud tenants (BDC Cockpit, Datasphere, SAC) in SAP for Me.
  • Some of your solutions are using outdated configurations. While they remain functional, we recomment upgrading them to include the latest changes and ensure compatibility.

Environment

  • SAP Business Data Cloud tenants
    • BDC Cockpit
    • SAP Datasphere
    • SAP Analytics Cloud

Reproducing the Issue

  1. Open SAP for Me.
  2. Navigate to the Business Data Cloud (BDC).
  3. Press the "Upgrade" button.

Resolution

  1. What is this upgrade related to?
    A: This upgrade is required due changes in the Datacenter Selection when provisioning new tenants. 
    Additional upgrades might be caused by any change in the BDC Suite configuration, it's not exclusive for datacenter changes only. 

  2. What got changed?
    A: SAP has extended support to new datacenters. 
    Additional upgrades might be caused by any change in the BDC Suite configuration, it's not exclusive for datacenter changes only. 

  3. What will happen if I don't upgrade? Any negative impact? 
    A: If you keep your solution outdated it will not impact any BDC features.
    However, it might not possible to change/edit any licenses or quota for the tenant.

  4. Does the upgrade have/cause a downtime?
    A: It does not have. Updating the solution will not cause any downtime.
