Symptom

When a parent job is being called by a child job in a shell script the job fails with 

 RUN-050316: |Session Bat_BDA_PR2PO_EMEA_Shell Job terminated as last run of job_name  is in progress!
(14.3) 02-02-26 11:32:18 (R) (18392:15672) RUN-050304: |Session job_name Function call <raise_exception ( Job terminated as last run of Bat_BDA_PR2PO_EMEA is in progress! ) > failed, due to error <50316>: <Job terminated as last run of job_name is in progress!>.

Environment

All available SAP Data Services versions

Cause

The datastore configuration was pointing to an old Data Services 4.2 repository, causing the shell job to fail.

Resolution

  1. Update the datastore configuration to point to the correct repository.
  2. Reexport the jobs to generate new .bat files to address potential configuration mismatches.
  3. Execute the updated jobs and verify successful execution.

Keywords

error 50316, job terminated, Bat_BDA_PR2PO_EMEA, .bat-related jobs, datastore configuration, Data Services 4.2, Linux ECC system, job execution failure, shell job error, repository configuration
