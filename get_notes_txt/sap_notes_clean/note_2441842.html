Symptom

You execute report SMIGR_CREATE_DDL as part of a system copy/migration. The report does not generate SQL files (like DFACT.SQL) containing DDL statements for BW tables and indexes. This problem can only occur in Systems with SAP Basis 700 - 740.

Reason and Prerequisites

Coding bug.

Solution

Implement the Support package mentioned below. In urgent cases implement the attached correction instruction.
