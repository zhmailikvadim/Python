Symptom

Unable to add "Free Goods" functionality for Purchasing (STO).

Environment

Materials Management (MM)
SAP ERP Central Component
SAP ERP
SAP enhancement package for SAP ERP
SAP enhancement package for SAP ERP, version for SAP HANA
SAP S/4HANA

Reproducing the Issue

1.Create STO through ME21N.
2.Try to set the field free goods indicator for the item.

Cause

Missing Functionality.

Resolution

Free goods functionality is not supported in Stock Transport Orders with delivery processing because STOs create deliveries without sales order reference, which is a prerequisite for free goods determination. This is a standard SAP limitation.

2497104-Inter-company STO is setting the "Free item" flag automatically, SAP ERP & SAP S/4 HANA
