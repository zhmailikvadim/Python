Symptom

If the Internal Order hierarchy is replicated, the field AUFNR is not filled in the table HRRP_NODE_N. This causes further issues with the hierarchy CDS views.

Other Terms

Internal Order, F2918, Manage Global Hierarchies, AUFNR, HRRP_NODE_N

Reason and Prerequisites

Program error

Solution

Apply following manual correction instructions using transaction SNOTE.

Once done, please deactivate and activate all the global hierarchies of type Internal Order.
