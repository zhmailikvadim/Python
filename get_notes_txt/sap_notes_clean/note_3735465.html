Symptom

A query is required in Query Builder to list all Web Intelligence reports in folders that contain data.

Environment

SAP BusinessObjects Business Intelligence Platform 4.x

Resolution

Such query is not possible in Query Builder as Web Intelligence content information is not stored in the CMS database metadata.

See Also

 2622713 - SAP Scope of Support for QueryBuilder, AdminTools or other CMS Database Queries

Keywords

Query builder, Web Intelligence, wid, content, data.
