Symptom

  • I receive the below error messages when submitting a Profile Worker:
    • [Custom Field Name] is required.
    • The user Role does not have permissions to populate the custom field. Please contact your System Administrator for assistance.
  • I cannot view the required Custom Field (CF) either on the form page to enter a value

Cause

This error occurs when

  • The custom field is mandatory.
  • It can only be entered by selected roles.
  • The user does not have one of those roles.

Resolution

Adding the user's role to the CF will allow them to enter a value when they create a Profile Worker:

  1. Click Admin (gear icon) > Custom Field
  2. Update filters to locate affected Custom Field
  3. Click Roles > Add
  4. Check the checkbox for User Role
  5. Click Add

If you do not see or have the permission to do the above, please contact your Administrator for further assistance.
