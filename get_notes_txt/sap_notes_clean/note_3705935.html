Symptom

An EC workflow has been triggered, how to locate the triggered workflow id with information in manage workflow details page.

Environment

SuccessFactors Employee Central

Cause

Resolution

  1. Navigate to the workflow details page in SAP SuccessFactors Employee Central.
  2. Locate the workflow request ID in the website's URL link.
  3. Access the Data Inspector tool and find table "Table_EMP_WF_REQUEST". Use the workflow request ID to find its corresponding workflow configuration code.
  4. Navigate to the table "Table_FO_WF_CONFIG". Use the workflow configuration code as the internal ID to find the corresponding workflow code.
