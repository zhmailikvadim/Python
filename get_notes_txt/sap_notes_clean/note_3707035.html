Symptom

  • Below error encountered during implementation of SAP Note 3591743, a prerequisite for SAP Note 3655229.

Error message: "APIS FIUA_RECON_REP_MAIL_TEMPLATE SMTG" Unknown message with

Diagnosis

/:DEFINE &V2& = ''
/:DEFINE &V3& = ''
/:DEFINE &V4& = '' 

Environment

Note Assistant

Reproducing the Issue

Attempt to implement SAP Note 3591743 using SNOTE.

Cause

If a SAP note contains an object an the corresponding APIS object together (e.g. both objects are created by the note) then an error is raised for the APIS object as the referenced object does not yet exist.

Resolution

  1. First implement SAP Note 3538837 - Note implementation fails for APIS objects.
  2. Update SNOTE tool to the latest version, see notes:
    SAP Note 1668882 - Note Assistant: Important notes for SAP_BASIS 730,731,740,750,751,752,753,754,755,756
    SAP Note 2869143 - Composite note for handling of Digitally Signed SAP Notes in Note Assistant (SNOTE transaction)
    SAP Note 3093855 - Note Assistant: Important SAP Notes for the Revamped Note Assistant  
  3. Retry the implementation of SAP Note 3591743 again.
