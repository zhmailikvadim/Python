Symptom

There is a memory leak during loading of SDL Master Data with Data Loading Process in followind situation:

  • The Data Loading Process is run for change pointers for individual objects and change pointer status "Error". If no data to be loaded has been found in the aDSO or in the HANA database table or in FSDM or all objects in the package to be loaded are erroneous, pre-buffered existing SDL Master Data objects are not deleted from the buffer after commit work. ABAP memory consumption increases while the load run is active.

Other Terms

Data Loading Process, SDL Master Data Framework

Reason and Prerequisites

Program Error

Solution

Implement the attached correction or the corresponding support package
