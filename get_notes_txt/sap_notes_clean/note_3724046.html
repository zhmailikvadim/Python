Symptom

In the process "Receipt of Activation for Supplier Time Series from DSO (SUP View)" (DE_B_EQS_POD_ACT_DSO_RECEIVE_SUP), the system does not populate the child settlement unit external name from the top-level settlement unit external name.

This issue is relevant for the market roles: DSO, SUP.

Other Terms

EQS_SUP, settlement unit, external name, top-level settlement unit, child settlement unit, supplier time series, activation

Reason and Prerequisites

The system lacks the functionality to populate the child settlement unit external name from the top-level settlement unit.

Solution

Implement this SAP Note or import the relevant support package.

After implementing this SAP Note, the system automatically populates the child settlement unit external name from the top-level settlement unit when the external name in the child settlement unit is not filled.
