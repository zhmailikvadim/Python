Symptom

 Problems occurred during processing because the import parameters _TAB_MDI_PAS, _TAB_MDI_COH_PAS, and _TAB_MDI_INVID_PAS were defined as mandatory in the method DESERIALIZE_HELPER of the class /IBS/CLFM_CRED_INVM_DC.

Reason and Prerequisites

For certain processing scenarios, it is necessary that these parameters can be passed optionally to avoid errors in processing.

Solution

In the method DESERIALIZE_HELPER of the class /IBS/CLFM_CRED_INVM_DC, the import parameters _TAB_MDI_PAS, _TAB_MDI_COH_PAS, and _TAB_MDI_INVID_PAS were set to optional.
This adjustment was provided in the form of an SAP Note implementation so that no customer-specific modification is required anymore.
