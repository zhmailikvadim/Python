Symptom

• The Trial Balance - F0996 tile does not include header information (technical details, variables, and filters) when exporting to Excel format.  
• Header information remains available when exporting to PDF format.  

Environment

• S/4 HANA Private Cloud  
• S/4 HANA On-Premise  
• Reporting Apps  

Reproducing the Issue

1. Open the Trial Balance - F0996 tile.  
2. Export the data to Excel format.  
3. Verify that header information is missing in the exported file.  
4. Export the data to PDF format.  
5. Verify that header information is present in the exported file.  

Cause

The header information is excluded by default when exporting to Excel format.  

Resolution

1. Select the option `Export As`.  
2. Ensure that the flag `Include Header Data` is selected.
