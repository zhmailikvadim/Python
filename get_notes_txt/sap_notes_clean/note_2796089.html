Symptom

You executed background risk analysis and multiple result sets created. Transaction GRAC_EXPORT_RA or report GRAC_EXPORT_REPORT_SPOOL in back-ground mode is not exporting all the result set lines. In foreground mode the correct data exported.

Other Terms

GRC, GRAC_EXPORT_RA, GRAC_EXPORT_REPORT_SPOOL, background, wrong, result, overwrites, missing, data

Reason and Prerequisites

Coding error.

Solution

Implement the attached correction instructions or the corresponding Support Package.
