Symptom

After archiving a project structure, when using a transaction that allows to display archived objects, the following message is thrown:

  • BS001 - No status objects exist for &

Environment

  • SAP ERP Project System (PS)
  • SAP R/3
  • SAP R/3 Enterprise
  • SAP ERP Central Component
  • SAP ERP
  • SAP Enhancement package for SAP ERP
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA, on-premise
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Open transaction SARA
  2. Use archiving object PS_PROJECT to archive project structure
  3. Open a transaction that allows the display of archived object (i.e. CN41N)
  4. Open archived project structure

Cause

missing code correction

Resolution

Implement the following SAP Note

  • 3082498 - Long runtime when using Logical Database PSJ

Keywords

CN41, CN43N, BS 001, DBPSJGNI, performance
