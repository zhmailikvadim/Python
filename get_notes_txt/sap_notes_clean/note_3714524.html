Symptom

The COMDIS reasons of code list G_0089 are not configured for compliant reasons of gas grid usage billing (code list G_0081).

Other Terms

SUP, Gas Grid Usage Billing, Code List, G_0081, G_0089, COMDIS, COMDIS Code List, COMDIS Reason, /IDXGL/V_COMPL_R

Reason and Prerequisites

Missing configuration

Solution

Implement the corrections of this SAP Note or import the relevant support package.

This issue is relevant for the market role: SUP.

After updating customizing configuration manually, the COMDIS process for gas grid usage billing is activated.
