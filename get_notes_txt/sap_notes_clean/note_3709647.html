Environment

SAP HANA Platform Edition 2.0
SAP Netweaver
SAP S/4HANA

Cause

  • The SUM_RTT_MS|AVG_RTT_MS are unacceptably slow between SAP HANA server & SAP Client host.
  • A Roundtrip time > 0.7 ms is considered a below average value for network latency between HANA and the client.



Resolution

  • Check the current network health with the responsible infrastructure and hardware team.
  • Follow the troubleshooting and measurement guidelines from KBA 2081065 - Troubleshooting SAP HANA Network  > 'How-To'  > point 1.
