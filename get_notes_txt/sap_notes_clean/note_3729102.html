Symptom

The follow error occurs when attempting to add plant text in different languages via the Manage Product Master Data app or other apps like Change Material (MM02).

MRP Text for plant &2 can be maintained only with language &1
Message no. CI_DRAFTPRD_MESSAGE 849

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Open the product in Manage Product Master Data app or other apps like Change Material (MM02).
  2. Click on "Edit" and select plant.
  3. Select "MRP Text" under plants.
  4. Create a new plant text in any other language.
  5. Return to the previous screen and click on the "Save" button.
  6. The following error message appears:
  • MRP Text for plant &2 can be maintained only with language &1
    Message no. CI_DRAFTPRD_MESSAGE 849

Cause

This behavior is standard system functionality, which restricts material memo creation to the language defined by the plant language key.

Resolution

  1. Verify the plant language key setup in Configuration activity 'Define Plant' (SSCUI 105877).
  2. Confirm that the language key for the plant.
  3. Text can be maintained only for those languages for a particular plant which is defined in Configuration activity 'Define Plant' (SSCUI 105877).

See Also
