Symptom

You are attempting to set up the integration between Microsoft Copilot and SAP Joule but are encountering difficulties in meeting the required technical prerequisites.

Environment

  • SAP Business Technology Platform for Cloud Foundry Environment
  • Joule AI copilot

Resolution

If you have not already done so, review and follow the enablement resources below:

  • SAP Discovery Center Mission - Integrate Joule and Microsoft 365 Copilot 
  • Integrating Joule with Microsoft 365 Copilot | SAP Help Portal 

SAP also provides a free advisory service to assist customers with setting up a proof-of-concept implementation for the Microsoft Copilot integration with SAP Joule.

To request this free advisory service, please submit the form below requesting for:
