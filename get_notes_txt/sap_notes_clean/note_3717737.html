Symptom

Reference Document field in MATDOC (MKPF) refer to Data Element XBLNR1. However, this field in DMATDOCDRAFT table of the RAP BO refers to Data Element XBLNR.

Other Terms

 DMATDOCDRAFT, XBLNR, XBLNR1, MATDOC, CL_MMIM_CHECK_MATDOC_ITEM

Reason and Prerequisites

It is possible to increase the length of data elements, especially in Public Sector Solutions, including XBLNR1. When that happens, the data type mismatch between MATDOC and the draft table results in a runtime exception.

Solution

Reference Document (REFERENCEDOCUMENT) in DMATDOCDRAFT now refers to XBLNR1.
