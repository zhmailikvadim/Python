Symptom

This SAP Note provides you with corrections to the PRV_NF_DOCUMENTO view.

Currently, when you process transactions for one-time customers, the system displays incorrect data for the Municipal Code (COD_MUN) field.

Other Terms

DRC, Brazil, SPED, S/4HANA, PRV_NF_DOCUMENTO, Municipal Code, COD_MUN, City Code, Tax Jurisdiction Code, TAXJURCODE.

Reason and Prerequisites

This SAP Note is relevant only for Brazil.

You have implemented SAP Notes 3707803 and 3709184.

Solution

After implementing this SAP Note, the J_OTC join node in the PRV_NF_DOCUMENT view is updated to validate the data in the Tax Jurisdiction Code (TAXJURCODE) field from the J_1BNFCPD table. As a result, the data in the COD_MUN field is correctly populated for One-Time customer scenarios.

As a general rule, SAP recommends that you implement the Support Package specified for your release. However, if you need to install a solution earlier, use the SAP Note Assistant to implement the correction instruction. You can find more information about the SAP Note Assistant in SAP Support Portal, under SAP Note Assistant.
