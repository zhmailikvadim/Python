Symptom

  • The PLVA XDL file fails during processing.
  • Error message observed: "The imported Assignment, {0}, has an invalid Element".
  • Error note: E_70113_IMP_VARASSIGN_RULEELEMENT.

Environment

  • SAP Sales Performance Management
    • Incentive Management on Legacy HANA  
    • Incentive Management on Hyperscalers

Reproducing the Issue

  1. Drop the file with type PLVA - have references for variable assignments (ex. Quota) referencing another rule element (ex. fixed value) where the fixed value doesn't exist in the system.
  2. Evaluate the progress of the file processing.
  3. Under the logs for the pipeline run - notice the errors. 

Cause

  • Rule elements (Ex. fixed values)referenced in the variable assignment file do not exist in the system.

Resolution

  1. Create the missing rule elements referenced in the PLVA file.
  2. Use the PLVA file to assign the fixed values to objects.
