Symptom

  • Unable to reset password using Gmail account.
  • Reset email does not come through.

Environment

  • UAA
  • Authentication
  • Authorization
  • Trust Management

Reproducing the Issue

  1. Attempt to reset the password for a SAP Business Technology Platform (BTP) account using a Gmail account.
  2. Observe that the reset email does not come through.

Cause

Resolution

  1. Verify if the user is registered within the SAP systems.
  2. If the user is not registered, complete the registration process by visiting the following link: SAP.com: Registration 
  3. After completing the registration, verify that the user appears in the SAP ID service on SAP's side.
  4. Ensure the Gmail account is associated with a registered user in the SAP systems.
  5. Confirm the identity provider for the user and verify the user details in the SAP Identity Authentication Service (IAS).
  6. If the issue persists, schedule a session with the user and an administrator to troubleshoot further.
