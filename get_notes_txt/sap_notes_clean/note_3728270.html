Symptom

This note includes the following corrections in the S/4 Defense Disconnected system:

Issue: During Stock Return in Delta Load (DL) process involving multiple PGIs, the predecessor determination logic was not consistently selecting the correct latest material document, leading to incorrect document reference during subsequent processing (PGR / synchronization).

Resolution: Code changes were implemented to correct the predecessor selection logic.

The logic was enhanced to:

  • Fetch all relevant material documents correctly.
  • Sort material documents based on posting year and document number.
  • Ensure the latest material document is consistently selected as the predecessor when multiple PGIs exist.

Other Terms

Outbound Delivery, Predecessor Document, MatStock, Material Stock, S/4 Defense Disconnected.

Solution

Implement the corresponding CI as per the note.
