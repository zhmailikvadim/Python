Symptom

In the "Manage Requests - Long-Term Supplier’s Declaration" app, it is not possible to show the following fields for the "LTSD Requests" table:

  • Ready to Send tab: Created By, Changed On
  • Failed, Sent, and Completed tabs: Created by,  Created on
  • "Reminder" tab page: Created by 

As a result of the changes in this SAP Note, the specified fields are included in the column set of the relevant table and can be selected.

Other Terms

Preference, long-term supplier’s declaration, fcg.lls.ltsdrequest.manage, F4419, Manage Requests - Long-Term Supplier’s Declaration, 3.0.3, 4.0.5

Solution

Implement the attached correction instructions in the GTS UI system. In addition, implement SAP Note 3718281 in your SAP back-end system.

If the correction was implemented successfully, but the error still occurs in the UI, follow the instructions in SAP Note 2319491.
