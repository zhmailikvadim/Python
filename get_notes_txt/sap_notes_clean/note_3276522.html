Symptom

In the results list of the program Wage Statement for Workers' Compensation Associations (RPLBGND0), the system issues an incorrect membership number for individual employees if the following prerequisites apply:

  • For the employees, an alternative AI agency is entered in infotype 0029.
  • An absence exists at the start of the evaluation period. 

Reason and Prerequisites

Program error

Solution

The program correction is delivered in a Support Package. For an advance correction, implement the attached correction instructions.
