Symptom

When opening News Tile created in Fiori Launchpad Application Manager following error pops up:

"App could not be opened because the SAP UI5 component of the application could not be loaded - accompanied by 'Failed to load UI5 component for navigation intent "#<SemanticObject> -<Action>"' 

Environment

SAP Fiori Launchpad Frontend Content Admin Tools

Cause

Wrong "SAPUI5 Component ID" value in the configuration.

Resolution

Maintain "SAPUI5 Component ID" wit value: "WebContent" in case of standard News tiles.

Keywords

news tile,  app manager,  fiori launchpad,  SAPUI5 component ID,
