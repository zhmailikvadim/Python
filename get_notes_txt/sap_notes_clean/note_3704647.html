Symptom

BAdI BADI_FTR_CUSTOM_DERIVE is used to adjust the general valuation class (GVC) for a mirror transaction, but the GVC is not getting set accordingly during the creation. 

Environment

Treasury and Risk Management (TRM)

Cause

Configuration

Resolution

  • The BAdI BADI_FTR_CUSTOM_DERIVE will only be called in online processing for example when navigating to the "Admin Data" tab, but not during background processing.
  • Use the following customizing path to set the GVC for the to be created mirror deal
    • Financial Supply Chain Management
      -Treasury and Risk Management 
      --Transaction Manager 
      ---General Settings 
      ----Accounting 
      -----Settings for Position Management 
      ------Assign General Valuation Classes to Groups 

Keywords

BADI_FTR_CUSTOM_DERIVE, gvc, mirror
