Symptom

You want to implement note 3589823, but used note 3034718 in the past.

Other Terms

 KE 496 PAOBJNR

Reason and Prerequisites

Design error (A warning message without reference to an input field does not result in the program being continued with the source code after the MESSAGE statement. In the case of an information message, on the other hand, a dialog box appears, after whose confirmation the execution of the program continues after MESSAGE as expected.)

Solution

Please de-implement note 3034718 and implement note 3589823 and afterwards implement this note, which replaces note 3034718. Note 3034718 is not needed anymore.
