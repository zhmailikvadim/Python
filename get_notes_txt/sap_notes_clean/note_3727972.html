Symptom

  • HTML Button functionality is not visible for poland KSEF Einvoice

Environment

  • SAP ERP 
  • SAP S/4HANA
  • SAP Document and Reporting Compliance Service

Reproducing the Issue

  1. Run Transaction code EDOC_COCKPIT. 
  2. Navigate to the Poland KSEF EINVOICE NODE and select the eDocument.
  3. Search for the display HTML button.

Resolution

  1. HTML display functionality is no longer supported for KSeF 2.0, PDF visualization is currently supported for both outgoing and incoming invoices.
  2. To enable PDF functionality for KSEF, implement all the notes mentioned in the overview note 3279206 Electronic Document Processing for Poland KSeF: Implementation Overview SAP for Me in sequential order.

See Also

  1. SAP Note: 3279206 - Electronic Document Processing for Poland - KSeF: Implementation Overview - SAP for Me
  2. SAP KBA: 3346323 - eDocument Poland B2B(KSeF) Troubleshooting Guide - SAP for Me
