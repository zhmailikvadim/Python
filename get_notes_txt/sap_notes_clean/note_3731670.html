Symptom

When executing transaction '/IDXGL/PDOCCREATE' to send message ES101 for the Gas division, the system displays an error indicating that the Identification Logic is missing in the process step data.

Other Terms

Gas, ES101, 44001, Identification Logic, P_IDENTL, /IDXGL/PDOCCREATE, SUP.

Reason and Prerequisites

Missing functionality.

Solution

Please implement manually activity in SAP Note or import higher version of Support Package.

This note is relevant for SUP role.
