Symptom

In RFQ integrations executed through SAP Cloud Integration (CPI) header-level attachments are transferred successfully, but line item‑level attachments are failing with the error Error:Executable file extension is automatically disallowed. Please contact support with the Error Reference Number: ANERR-xxxxxx for more details 
 

Environment

Sap Business Network

Resolution

RFQ integration through SAP CPI supports only header‑level attachments. Line item‑level attachments are not transferred because CPI does not provide RFQ item‑level attachment structures or mapping logic.

Keywords

Flow, Envelope, .com, ERP, ZIP, Parsing, Trace, Exception, Encoding, Format
