Symptom

Texts for a Hierarchical Structure defined in a particular language might be deleted when endusers execute the transaction RSZTREE in another logon language.

Other Terms

BEx Query Designer, RSZTREE, CKF, RKF, Hierarchical display.

Reason and Prerequisites

Program error.

Solution

A manual activity might be required BEFORE implementation of the code correction.

1. Execute transaction SE37 and open the function RSZ_DB_COMP_TREE_DELETE.

2. Add Importing parameter I_DELETE_ALL with the properties:

Type = RS_BOOL

Default Value = RS_C_TRUE

Optional = TRUE ('X')

Pass Value = TRUE ('X')

3. Save and activate the function

4. Apply the attached code correction via transaction SNOTE.
