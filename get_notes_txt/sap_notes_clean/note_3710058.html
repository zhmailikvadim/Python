Symptom

I created a Home Page Announcement for the User Type Buyer. I do not know how I find out which User Roles have permission to see the announcement  

Resolution

All buyer users within the buyer company will have visibility on the Home page announcement created for user type Buyer

On User level there is User role Permission that affects this admin object

The permission is Show Home Page Announcement:

  • If the permission is set to Yes then the Home Page announcement will be visible for Buyer Users
  • If the permission is set to No then the Home Page announcement will not be visible for Buyer Users even though the User Role permission is provided
