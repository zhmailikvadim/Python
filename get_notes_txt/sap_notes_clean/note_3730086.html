Symptom

You are executing payroll (Transaction code: PC00_M40_CALC) for employees in your organization. You are using Gross Carry Forward (GCF) functionality for Provident Fund (PF) calculation and are using the customer exit 007. It is observed that the parameter AS-PARM1 is not available for usage in the customer exit. However, it should be imported for custom logic implementation.

Other Terms

HINCALC0, Code of wages 2025, INEPF, EXIT_HINCALC0_007

Reason and Prerequisites

Correction of legal function

This SAP Note is relevant only for India

Solution
