Symptom

  • The system is required to automatically generate a document populated with MDF data captured throughout the workflow after HR submits the final form, and send it to the end user as an attachment via a system notification.
  • The user seeks confirmation whether this is supported natively or if recommended workarounds exist.
  • No error messages or error codes are mentioned.

Environment

 SAP SuccessFactors Onboarding

Reproducing the Issue

Cause

The process after the HR team submits the final form is by design and cannot be modified.

Resolution

  1. Confirm that Offboarding 2.0 supports the described behavior: after HR submits the final form, the system automatically generates a document using the MDF data captured throughout the workflow and sends it to the end user as a notification attachment.
  2. Note that this post-submission process is not configurable; modifications to the behavior after HR submits the final form are not supported.
  3. If changes to this behavior are required (for example, custom variations or alternative formats beyond supported configuration), submit an enhancement request for future consideration.
