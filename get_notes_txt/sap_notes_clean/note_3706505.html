Symptom

In transaction SNOTE, a menu item "Aktuellste Version herunterladen" appears after completely implementing Note 3007273.

Environment

  • Transaction SNOTE
  • VALID FOR
    Software Component   SAP_BASIS
    Release 700          SAPKB70009 - SAPKB70035
    Release 701          Until SAPKB70120

    Release 752          Until SAPK-75203INSAPBASIS
    Release 753          w/o Support Packages

Reproducing the Issue

  1. Logon the SAP system via language EN(English).
  2. Open transaction SNOTE.
  3. Click the top menu "SAP Note".
  4. Observe the German menu item "Aktuellste Version herunterladen".

Cause

SAP Note 2965025 has not been implemented.

Resolution

Implement SAP Note 2965025 and perform the manual activities described in the SAP Note.
