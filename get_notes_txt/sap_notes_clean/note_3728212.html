Symptom

  • Users with an Admin role are receiving email notifications when an inquiry is raised from Document Distribution. However, they are unable to take any action—such as answering, approving, or disapproving—in response to these notifications.
  • When Admin users open the inquiry, the options to respond are not enabled.

Environment

  • SAP Sales Performance Management
  • Incentive Management on Hyperscalers

Reproducing the Issue

  1. Assign the Admin role to users.
  2. Generate and distribute plan documents in the Plan Communicator workspace.
  3. Raise an inquiry to an admin using the "Send to Administrator" action in the distribution.
  4. Observe that all users with this role receive email notifications but are unable to take any action on the inquiry.

Cause

The role is missing required permissions:

  • Administration → Users → Read
  • Organization → Participants → Read

Resolution

  1. Verify the permissions assigned to the role.
  2. Ensure the following permissions are enabled for the Sales Ops role:
    • Administration → Users → Read
    • Organization → Participants → Read
