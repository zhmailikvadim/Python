Symptom

When the currency in DPQS master data (/ACCGO/MAINT_DPQS) or Contract differs from the settlement currency, users have the option to negotiate the foreign exchange (FX) rate during the settlement process prior to release.
The current Settlement Workcenter calculates the DPQS amounts on a new Settlement Proposal only. If FX rates are negotiated after generating the proposal or while modifying an existing settlement, the system applies the updated values directly to the calculated DPQS amounts.
This results in inconsistencies when comparing the DPQS calculations between the Settlement Proposal after FX Negotiation and the DPQS adjusted Values obtained from simulation of the transaction /ACCGO/MAINT_DPQS.

Other Terms

ACM; Agricultural Contract Management; Settlement; FX Negotiation; DPQS; Settlement WorkCenter; /ACCGO/MAINT_DPQS

Reason and Prerequisites

Settlement Work Center applies the Negotiated FX directly to the DPQS Amount Calculated when the Settlement was proposed, instead of applying the FX first to the Settlement Price and then using the revised price to re-calculate the DPQS.

Please implement the below mentioned prerequisite notes in sequence mentioned below

  1. 3340908 - ACM S/4H: Incorrect DPQS amount getting calculated after FX negotiation
  2. 3339386 - ACM S/4H: Incorrect DPQS amount calculated in case of re-negotiated FX rate

Solution

 Import the support package or implement the correction instruction.

Steps to replicate the Issue before implementation of Note and Test the Solution after implementing the Note

1. Generate a Contract in USD and attach a DPQS in another Currency (JPY preferably, special Currency)

2. Create multiple LDCs in different Currencies (USD / JPY / CAD). This would generate multiple Settlements.

3. Generate proposal for each LDC and in the FX Negotiation tab on Settlement Unit, re-negotiate the FX rates.

4. Check the calculations and match the DPQS Adjustment amounts by Simulating the DPQS using T.Code /ACCGO/MAINT_DPQS

After the Note implementation the DPQS Calculations should be consistent between the Settlement Proposal / Settlement Update (after FX Negotiations) and DPQS Simulation in T.Code /ACCGO/MAINT_DPQS.
