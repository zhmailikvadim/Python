Symptom

You use the progress documentation and have entered the entry "*" (asterisk) in the selection in the "Care Unit" field in the departures view or the occupancy view. You notice that the system also transfers this entry to the table N2VDNOTE. This can lead to problems during the authorization check or to progress entries not being visible to users.

Other Terms

Progress Documentation, Progress Report

Reason and Prerequisites

Program error

Solution

The system now correctly no longer transfers the asterisk entry to the table N2VDNOTE.

Import the specified Support Packages to solve this problem.
You can implement the correction in advance.
