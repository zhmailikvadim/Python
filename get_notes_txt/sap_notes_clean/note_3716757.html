Symptom

In S/4 Hana version 2023, in the application F0940 "Cost centers-Actuals", the business transaction type description name is not appearing anymore.

Environment

  • SAP S/4HANA
  • SAP S/4HANA Finance
  • SAP Fiori
  • SAP Fiori for SAP S/4HANA Finance

Reproducing the Issue

  1. Run the app "Cost centers Actuals" Fiori application.

Cause

The issue is caused by the new tables FINSC_CUSTBTTYPE and FINSC_CUSTBTTYPT being empty, while the old tables finsc_bttype and finsc_bttype_t contain data.

Resolution

1. In transaction code SA38 or SE38 to access the ABAP Editor.
2. Execute Program "FINS_BUSTRNSCTN_CNTNT_RETRIEVE" in the program field.
3. Run the imported report and verify the report has finished executing, the standard description names will be populated in the new tables FINSC_CUSTBTTYPE and FINSC_CUSTBTTYPT.
4. Run the Cost Centers Actuals Fiori app again to verify that the business transaction type descriptions are correctly displayed.
