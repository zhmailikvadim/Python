Symptom

Corrections of this SAP Note must be implemented before you implement SAP Note 3726413.

Other Terms

FINWITH_TAXKEY, T059O, T059Z, WT_TAXKEY, Withholding Tax, India LC.

Reason and Prerequisites

Prerequisite of SAP Note 3726413.

Solution

Kindly follow the instructions given in post-implementation steps of this SAP Note.

Execute the UDO report NOTE_3731455.
Step 1: Choose Test Run and execute and check if the changes are
compatible
Step 2: Choose Update & Activate and execute, provide the transport
request, and provide the package as FQST, if prompted.
Step 3: Choose Show Logs and execute to check the log of the changes
made by the report.

ECC Customers - Please follow the steps to create Table Maintenence Generator for the view FINWITH_TAXKEY by referring the attachment 'TMG Steps.pdf'.
