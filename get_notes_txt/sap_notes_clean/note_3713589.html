Symptom

You want to create a payoff with transaction FN8X. This is not possible because you get the error message 'Message incomplete (severity, area, number, or exception missing)' (Message number BL203).

The problem occurs after the coding changes of SAP Note 3560713 had been implemented in the system.

Other Terms

FN8X, Payoff, BL203

Reason and Prerequisites

Program error

Solution

Implement the attached Correction Instruction (CI) or the relevant Support Package.
