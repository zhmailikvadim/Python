Symptom

When viewing a Web Intelligence (WebI) document details in BI Launchpad, the Last Run field in the details section always shows N/A, even after running or refreshing the report.

Environment

  • SAP BusinessObjects Business Intelligence Platform (BI) 2025
  • SAP BusinessObjects Business Intelligence Platform (BI) 4.3 
  • BI Launchpad

Reproducing the Issue

  1. Open BI Launchpad and locate a Web Intelligence (WebI) report.
  2. Right-click the report or click the three dots (More Actions) and open Details.
  3. Observe the Last Run field — it may show N/A.
  4. Perform one of the following actions:
    • Run the report in BI Launchpad and save it after refreshing.
    • Run the report in BI Launchpad and select "save as" option after refreshing.
    • Schedule the report,but let the schedule fail or not complete.
  5. Return to the Details section in BI Launchpad
    >> Notice that the Last Run field remains N/A, even though the report appears to have run.

Cause

  • The Last Run field continues to show N/A because no successful instance was created in the BI platform.
  • The Last Run field in BI Launchpad only reflects the last successful execution of a scheduled report that generates an instance.

Resolution

This behavior is by design

Keywords
