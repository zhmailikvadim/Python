Symptom

After running SNP Optimizer notice the error in the step 2 with message "Optimizer: input data inconsistent".

Environment

  • SAP Supply Chain Management
  • Advanced Planner and Optimizer
  • Supply Network Planning (SNP)
  • Planning
  • Interactive Supply Network Planning

Reproducing the Issue

  1. Maintain optimizer profile.
  2. Maintain interchangeability groups.
  3. Run Optimizer for the products in the interchangeability group.
  4. Notice the error in the step two of the Application log

Cause

The error is caused by the same start dates for the products defined in the interchangeability group. All product within one group must have different start dates.

To confirm this, in the Trace File search for sign "<e>" is performed and error "Violation of rule continuity" is found. It is important to get the name of interchangeability group for further reference.

Input log ET_CHAIN and is now being filtered by interchangeability gourp.

Notice the same start dates within interchangeability group.

Resolution

Edit the dates within the interchangeability group. Start dates within the group have to be minimum 1 day apart.
