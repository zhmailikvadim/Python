Symptom

  • The issue is being observed during the process of changing a customer from one account group to another.
  • When the change of account group is attempted through XD07 or LSMW, the error “Address‑Text jurisdiction code & Control‑ Natural person” is being displayed.

Environment

  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  • Navigate to XD07
  • Change account group from old to new account group
  • System will populate error "Address-Text jurisdiction code  &Control-  Natural person"

Cause

The tax jurisdiction code field was set as a required entry, so error was being displayed.

Resolution

1. Navigate to transaction OBD2 to check the account group field status. 
2. Ensure the tax jurisdiction code field is set to "Optional entry" for both account groups.
3. In XD02, go to General Data → Control tab.
4. Set the Natural Person indicator as required by the target account group. 
5. Save the changes before attempting the account group change.
6. Execute the account group change using XD07 with the updated customer data. 

Keywords
