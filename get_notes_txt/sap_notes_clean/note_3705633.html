Symptom

You download data from the Search and Process Workbench (/VTA/BDO10) to Excel and expect all displayed fields to be exported correctly. However, the Posting Status Header field appears correctly in the system but is blank in the Excel export. Additionally, the Rejection Reason column displays correctly in the system, but in Excel it contains only the two-letter code instead of the code with description.

This note corrects the behavior so that the Excel export includes the Posting Status Header and the Rejection Reason with both code and description.

Other Terms

CL_SALV_GUI_LEX_FORMATTER

Reason and Prerequisites

This is caused by program error

Solution

Please implement the correction instructions attached. These corrections are only valid if you have applied the following support packs.

Release V4 - 2023 V2 and 2023 V3

1. Perform the manual pre-implementation steps that are described below.

2. Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object.
