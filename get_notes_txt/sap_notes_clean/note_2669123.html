Symptom

The button Customer - 360º View is not visible when clicking in the Action button for an account as shown below :

Environment

Cloud for Customer integrated with S4/Hana.

Reproducing the Issue

  1. Go to CUSTOMERS work center.
  2. Select an account which also exists in S4/Hana.
  3. Click on Actions button.
  4. Check if button Customer - 360º View is missing

Cause

Configuration of the functionality has not yet been completed.

Resolution

  • Go to ADMINISTRATOR work center > Business Flexibility > Mashup Authoring.
  • Search for a Mashup with the name S/4 HANA 360 Object Page.
  • Replace the dummy URL with the URL of your S/4 system.
  • Close the Mashup tab after ensuring that the changes are saved and mashup is active.

This should enable the button as pictured.
