Symptom

In the process "End of Supply for Electricity (DSO View)" (DE_B_GPKE_DSO_EOS), the system does not determine the future supplier correctly for transaction code ZT4 in step 1500 (class /US4G/CL_CHK_GPKE_PROCESS-CHECK_FUTURE_SUPPLIER).

This issue is relevant for the market role: DSO.

Other Terms

 GPKE, DE_B_GPKE_DSO_EOS, End of Supply for Electricity, ZT4, /US4G/CL_CHK_GPKE_PROCESS-CHECK_FUTURE_SUPPLIER, step 1500, future supplier, DSO, transaction code

Reason and Prerequisites

The error occurred because the system did not determine the future supplier correctly for transaction code ZT4.

Solution

Implement the corrections of this SAP Note or import the relevant support package.
After implementing this SAP Note, the system determines the future supplier correctly for ZT4 in step 1500. The related emergency patch on the Cloud side is 1000012564. You can verify the fix by retesting the scenario and confirming that the future supplier is determined correctly.
