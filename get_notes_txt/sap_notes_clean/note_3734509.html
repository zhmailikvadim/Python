Symptom

You use Transaction Manager of Treasury and Risk Management. Your system is still using the legacy security framework for the related security positions. When a trade is created or edited, selecting the “Yield on Call” option from the 'Extras' menu dropdown may lead to an incorrect calculation result.

Other Terms

LFWTR_01F05, LFWTR_01F16, CALL_DATES_READ, YIELD_ON_CALL_DETERMINE, PEMKURS, PBASKURS_LONG, CANCELRATE, CANCELRATE_LONG

Reason and Prerequisites

Program Error

Solution

Preliminary correction via Correction Instruction or implementation of the corresponding Support Package.
