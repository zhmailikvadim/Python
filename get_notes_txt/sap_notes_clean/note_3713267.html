Symptom

You are using SAP Build Work Zone, standard edition and would like to know whether Shell Plugins are supported or not.

Environment

  • SAP Business Technology Platform
  • SAP Cloud Foundry
  • SAP Build Work Zone, standard edition

Reproducing the Issue

Deploy a Shell Plugin on SAP Build Work Zone, standard edition and you face issues using this service.

Cause

As per design.

Resolution

Shell Plugin is only supported for SAP Cloud Portal Service on Cloud Foundry.
SAP Build Work Zone, standard edition does not support Shell Plugins Apollo scenario usage. The recommended approach is the Local Entry Point (LEP) scenario, which is fully supported and aligned with the latest experience entity models (for example:  Pages, Spaces and Cards).
For more details, check:
Creating Local-Entry-Point Business Solutions
