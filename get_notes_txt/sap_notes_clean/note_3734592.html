Symptom

Message no. HRPAY99_PAYROLL_LOG010 ‘FUPARTT: Result of new coding does not match current coding (see long text)’ after running payroll driver. "Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental." 

Environment

  • Human Capital Management (HCM)
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP SuccessFactors Employee Central Payroll
  • SAP S/4 HANA

Reproducing the Issue

  1. Run payroll driver via t-code SE38. 
  2. Error message no. HRPAY99_PAYROLL_LOG010 displays in payroll log shown below.

Cause

Program issue

Resolution

Please implement SAP Note 3685318 - PARTT: Wrong error RP490 for gaps in IT0007 to fix the issue.

See Also

SAP Note 3685318 - PARTT: Wrong error RP490 for gaps in IT0007

Keywords

HRPAY99_PAYROLL_LOG010, FUPARTT, RPCALC*0, H**CALC0
