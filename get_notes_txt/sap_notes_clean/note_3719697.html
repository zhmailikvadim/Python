Solution

What’s New for SAP GRC UIDP Masking for SAP HANA (Support Pack - 00)

Centralized Configurations

  • Centralized management of masking and blocking configurations are being introduced via central component approach, allowing users to manage sensitive data settings centrally and distribute them across multiple connected systems.

Enhanced UI Masking Policy Framework

  • A Fiori-based Policy Builder is now available to streamline the policy creation process and deliver unified, modern user experience.
  • Enhanced UI Masking policies now allow for automated fallback actions, ensuring data protection is maintained even when required attributes are unavailable.
  • Integrated alerting functionality within UI Masking policies to provide administrators with visibility into missing critical attributes, facilitating timely troubleshooting and resolution.

Extending Blocking Functionality to WebDynpro Scenarios

  • Blocking capabilities to WebDynpro (WDA) scenarios are now extended to  include output blocking in List and Form UIs.

Multilingual Support

  • Translations in all major languages are now provided to ensure alignment with other Governance, Risk, and Compliance (GRC) products.

Migration of Customizing Data

  • You can now migrate existing UI Masking configurations from legacy solutions into the SAP GRC central system.

For more information on SAP GRC UIDP Masking for SAP HANA, please refer to https://help.sap.com/docs/SAP_GRC_FOR_SAP_HANA
