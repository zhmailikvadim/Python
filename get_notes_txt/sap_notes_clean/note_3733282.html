Symptom

If two DSO device change process trigger at the same key date, the meter changes from a single rate meter to a double rate meter with the same device number, there will be 2 entries in MI_DEV table, and 2 lines of meter display in APP Parameterization Instance UI.

Other Terms

Setup / Replacement of Devices (DSO View) (DE_B_IM_DEV_CHANGE_DSO), Setup / Replacement of Devices (SUP View) (DE_B_IM_DEV_CHANGE_SUP), new feed-in malo, WORKLIST

Reason and Prerequisites

This is a program error

Solution

Please implement the attached SAP Note or import higher version of Support Package.
