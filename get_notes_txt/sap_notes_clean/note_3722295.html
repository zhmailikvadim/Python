Symptom

  • Error message "Something went wrong while converting the story to Optimized Design Experience.
    Cannot read properties of undefined (reading 'planning:XXX " would appear when converting the specific classic story to the optimized story

Environment

  • SAP Analytics Cloud 2026.2.6

Reproducing the Issue

  1. Log onto SAC
  2. Open the problematic classic story 
  3. Navigate to Save button and choose "Convert to Optimized Design Experience"
  4. On the following "Convert to Optimized Design Experience" panel, select "Convert + Save as"
  5. After waiting for a while, observe the below error message would appear:
    Something went wrong while converting the story to Optimized Design Experience.
    Cannot read properties of undefined (reading 'planning:XXX)

Cause

The issue is caused by a defect in the product.

Resolution

This issue has been addressed in version 2026.2.9.
