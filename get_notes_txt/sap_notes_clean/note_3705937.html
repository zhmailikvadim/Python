Symptom

 SAP mobile development kit client for SAP Mobile Services

Other Terms

Release, features, fixes, mdk, client, patch, sdk, iOS, Android, tablet, phone, Web, browser

Reason and Prerequisites

SAP Mobile Services offers a revision to the mobile development kit client.

This patch release provides the following minor update and important bug fixes.

Common fixes across platforms

  • Improved handling of additional network connectivity error codes

iOS Specific:

  • Fixed app crashes when adding .txt attachment
  • Fixed scrolling issue in Grid Table with header in multiple sections situation
  • Fixed SimplePropertyFormCell redraw causing keyboard jump
  • Resolved appearance of text on navigation back button on phone
  • Fixed AttachmentFormCell multi column crash after attaching
  • Fixed AttachmentFormCell not resetting when out of view
  • Fixed the white screen issue on hotspot Wi-Fi without internet
  • Fixed ObjectTable left padding issue on iPad if Icons are defined
  • Updated iOS SDK to 24.12.11
  • Updated to Apple Xcode 26.2

Android Specific:

  • Fixed soft keyboard overlap issue for the last input field in Tabs and Bottom Navigation 
  • Fixed Crash with onLoaded null error
  • Added preventive fix for crash error related to BackStack access
  • Fixed OnPress event issue when reusing attachment control
  • Fixed a crash on opening ListPicker before the list is loaded properly
  • Fixed ProgressBar freezing issue
  • Updated Android certificates to support Android 13 and earlier devices 

Solution

For Custom MDK Mobile Client developer, download and use the latest MDK Client SDK installer from SAP Store, Service Marketplace.

For SAP Mobile Services Client user, download the latest SAP Mobile Services Client from the Apple AppStore or Google Play Store.

For MDK Web Application developer, login to SAP Business Application Studio, do your development and deployment in your browser.
