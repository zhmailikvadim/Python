Symptom

Interface extension of BAdI MDBPCVI_READER_NONCORE_EXT returns undesirable error message when no data exists in the corresponding country/region-specific table FI PL BP Classification.

Other Terms

FI PL, BP, PL SP, BP Classification

Reason and Prerequisites

Interface returns undesirable message and that should be removed.

Solution

Apply the appropriate Support Package or install the attached correction instructions
