Symptom

The button for reporting line in profile pop-up is announced as “button” by JAWS instead of a descriptive label.

Environment

SAP SuccessFactors Employee Central

Reproducing the Issue

  1. Navigate to the homepage on the SAP SuccessFactors Mobile application using iOS.
  2. Open the profile pop-up.
  3. Use JAWS assisted technology to interact with the reporting line button.
  4. Observe that the button is announced as “button” without a descriptive label.

Resolution

The issue will be fixed on 2602 release.
