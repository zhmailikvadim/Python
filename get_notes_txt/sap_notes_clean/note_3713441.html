Symptom

Following the SAP HANA migration, the e‑ZLA program (HPLUZLA1) may set the incoming record with an incorrect STATUS value on infotype 3421. As a result, absence records are not being created on infotype 2001.

Other Terms

HPLUZLA1, HPLUZLA2, e-ZLA, STATUS, IT3421, T7PLEZLA_ZST.

Reason and Prerequisites

Program error.

Solution

HPLUZLA1 report has been corrected.

Please apply the correction by importing the Support Package specified for your release, or by using the Note Assistant.
