Symptom

In the Agreement Request search screen in Fiori, the search help for the Division field is not functioning. When attempting to search for a Division using the value help, no values are displayed.


This note ensures that search help values are available for the Division field on the search screen.

Other Terms

SPART, TSPA, /IRM/IPAGRM, ui_dropdowns_rtab_prepare, Division Search help.

Reason and Prerequisites

This is caused by Program error.

Please implement the correction instructions. These corrections are only valid if you have applied the following Support Packs.

Release V/4 – 2022 V2, 2023, 2023 V2, 2023 V3 and 2023 V3 Support Package 1
