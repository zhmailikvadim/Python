Symptom

Error message: "Program RS_CI_DELETE Method 'CLEAN_DB_ENTRIES' is unknown or PROTECTED or PRIVATE." appears after transport. It fails with return code 8.

Environment

SAP NetWeaver - All BASIS releases.

Reproducing the Issue

  1. Attempt to transport program RS_CI_DELETE from source system to target system.
  2. Observe the transport request failing with return code 8.
  3. Check the syntax error in the source system for program RS_CI_DELETE.
  4. Error message: "Program RS_CI_DELETE Method 'CLEAN_DB_ENTRIES' is unknown or PROTECTED or PRIVATE.

Resolution

Transport the corrections provided with SAP Note 2329296 from source system (and all its prerequisites if not already performed) to the target system. 

See Also

SAP Note 2329296 -  Regular cleanup of database tables for certain Code Inspector checks
