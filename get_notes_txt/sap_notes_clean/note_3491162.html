Symptom

Error KF009 "No assignment made to value fld for PA transfer str. FI, assignment no. XX" even though costing based CO-PA is no longer active.

Environment

  • Controlling (CO)
  • SAP R/3
  • SAP R/3 Enterprise 4.7
  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA Finance
  • SAP S/4HANA

Reproducing the Issue

Error received when posting any doc to Account Based CO-PA/ Margin Analysis.

Cause

Incorrect sequence of steps for deactivating Costing Based CO-PA.

Normally deactivating CO-PA in KEA0 would also update the customising OKKP.

Resolution

  1. Please check that costing based CO-PA is not active in transactions KEA0, KEKE, & OKKP.
  2. If Costing based CO-PA is still checked in OKKP and it is not editable,

    Then please update the type of Profit analysis via table TKA00 via SE16N in Edit mode, and change entry with CO area KOKRS = XXXX, change ERGBR from 4 to 3 in order to just have Margin analysis activated.

  3.
