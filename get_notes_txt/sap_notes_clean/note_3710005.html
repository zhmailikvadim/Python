Symptom

After the upgrade to S/4HANA 2025 system, "Create Support Message" item in the Help menu is missing even though user has SAP_ALL authorization.

Environment

  • S/4HANA 2025 system

Reproducing the Issue

  • Logon S/4HANA 2025 system
  • Select menu Help

Cause

Coding error.

Resolution

Implement SAP Note 3689363 - Incorrect menu assignment for users with SAP_ALL authorization profile.

See Also

  • SAP Note 3689363 - Incorrect menu assignment for users with SAP_ALL authorization profile
  • SAP Note 2741257 - Display of help and system technical menus in S/4HANA Cloud Private Edition and on-premises
  • SAP Note 2739940 - Display of help and system technical menus in S/4HANA Cloud Public Edition

Keywords

Create Support Message, missing
