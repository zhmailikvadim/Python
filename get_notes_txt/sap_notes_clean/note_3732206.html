Symptom

Data export to CSV includes all dimensions, regardless of drill level settings applied to a chart in SAC

Environment

SAP Analytics Cloud 2026.2.6 

Reproducing the Issue

  1. Login to the SAC 
  2. Open the story
  3. Set the drill level in the chart to any higher level. 
  4. Click on the chart, select "Export" from more options, and generate the CSV file.  

Notice : All dimensions are present in the CSV file, even though the chart is not drilled down to the lowest level. 

Cause

By design behavior of the product

Resolution

Kindly submit an enhancement request to request this feature as per - 2424376 - How do you create an enhancement request and provide feedback for SAP Analytics Cloud?
