Symptom

A custom field is added to the Create Incoming Invoices (or Create Outgoing Invoices) app. The value help icon appears, but clicking it does not open the value help dialog.

Environment

SAP S/4HANA Cloud Public Edition

Resolution

The Create Incoming Invoices and Create Outgoing Invoices apps are based on classic GUI technology and do not support custom fields like native Fiori apps. The new value help extension for custom fields is not supported in these classic applications.

Keywords

classic GUI app limitation, Create Incoming Invoices, Create Outgoing Invoices, FB60, FB70, behavior, custom field, value help extension, value help not working, extensibility limitation, accounts payable Fiori, supplier invoice
