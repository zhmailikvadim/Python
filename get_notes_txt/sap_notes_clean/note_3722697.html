Symptom

When adding a new template in Approval Templates > Originator tab, click on the Add button. If no originator has been selected, the error message will appear:

"On 'Originator' tab, fill in 'User' column"



Environment

  • SAP Business One
  • SAP Business One, version for SAP HANA

Reproducing the Issue


  1. Go to Administration > Approval Process > Approval Templates.
  2. Add a new template in the Originator tab.
  3. Ensure no document producer (originator) is selected, even when the template name is valid.
  4. Click the Add button.
  5. The error message will appear: "On 'Originator' tab, fill in 'User' column."

Cause


  • System behavior by design.

Resolution


  1. Ensure the document producer (originator) is selected in the Originator tab before clicking the Add button.
  2. The error message will not appear once the originator is filled in.
