Symptom

A Partner Function within an Account gets deleted after replication.

Environment

SAP Cloud for Customer V1

Reproducing the Issue

  1. In C4C go to Customers
  2. Accounts
  3. Go to tab Account Team
  4. Create a partner function
  5. Save
  6. After a few seconds the partner function gets deleted

Cause

The inbound payload for API BusinessPartnerReplicationIn shows setting partnerFunctionsListCompleteTransmissionIndicator="true" within segment SalesArrangement.

Resolution

Setting partnerFunctionsListCompleteTransmissionIndicator is set to TRUE by default in S/4HANA. You need to set that setting to FALSE.

That can be achieved via implementation of BAdI MDG_SE_BP_BULK_REPLRQ_OUT.

Keywords
