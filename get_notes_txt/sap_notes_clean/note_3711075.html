Symptom

Component: Incident Management
Module: Investigation

You maintain guidelines without a group assignment in the customizing activity Specify Guidelines for Incident Category. When you use e.g. the value help to determine guidelines, the system does not display the entries without group assignment.

Other Terms

Incident Category, GROUP_CODE, EHHSS_INC_CAT_GROUP_CODE

Reason and Prerequisites

Reason:
Program error. The system filters guidelines based on group assignment. Guidelines without a group assignment are excluded from the results.

Prerequisites:
For information about the validity of the corrections, see the correction instructions.

Solution

After implementing this SAP Note, the system displays all guidelines in e.g. the value help for Incident Category, including those without group assignment. You can now select guidelines regardless of their group assignment status.

The "Support Packages" section lists the Support Packages that contain these corrections.

Alternatively, you can implement the attached correction instructions.
