Symptom

In sales document display or change, selecting the item and starting Configuration shows the error: "Configuration not possible for material: Reason 3".

Environment

  • SAP ERP
  • SAP S/4HANA
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Open a sales document e.g. in transaction VA03.
  2. Select the line item and start Configuration.
  3. Observe the error: "Configuration not possible for material: Reason 3".

Cause

The sales item’s configuration instance references a configuration profile that is currently locked/not released (status 2).

Resolution

  1. Identify the configuration instance, field VBAP-CUOBJ of the sales item.
  2. Use transaction CUTABLEINFO to check IBIN and determine CUCOCN.
  3. Then identify the configuration profile with the counter in table CUCO (field CUCON) that the instance uses.
  4. In table CUCO, verify the profile status.
  5. If it is locked/not released (status 2), release the profile in transaction CU42.
