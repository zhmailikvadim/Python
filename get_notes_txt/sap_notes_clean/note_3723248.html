Symptom

When you try to display the application log in the Fiori application F1520 "Reprocess Bank Statement Items", you receive the error message: "Unable to retrieve log data."
This issue occurs even though you have valid authorizations for the authorization object F_FEBB_BUK.

Other Terms

F1520, F_FEBB_BUK, authorizations, APL_RT/006

Reason and Prerequisites

Program error.

Solution

Implement the attached Correction Instruction or the mentioned Support Package.
