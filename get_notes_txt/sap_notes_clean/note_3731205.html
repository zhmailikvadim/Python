Symptom

In the unloading step of the Pick by Cart app, after scanning and confirming a pickHU, a manual click in the destination HU input field is forced before the next pick HU can be scanned.

Other Terms

Fiori App; Handling Unit (HU)

Reason and Prerequisites

Prerequisites
There are several pick HUs to be unloaded.
A scanner is used to enter the pick HUs into the destination HU input field.

Reason
The focus gets lost from the destination HU input field if a scanner is used.

Solution

Implement the correction instruction provided with this SAP note or update to the corresponding support package.
