Symptom

This is a FAQ about the two fields "Rate for taxes"(KURSF) and "Tax rate: LC"(CTXKRS) for transaction like FB60

Environment

  • SAP R/3
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP ERP Central Component

Cause

RTAX1-KUSRF(BKPF-TXKRS) is the the rate of tax amounts (but not tax base amounts) from document currency into local (T001-WAERS) currency.

To make this field ready for input, you need to run OBY6 to maintain the "1" for "Crcy transl. for tax.

RTAX1-CTXKRS(BKPF-CTXKRS) is the rate to convert tax and base amounts from document currency into country (T005-WAERS) currency. It applies only if country currency differs from local currency.

For example 

Local Currency = JPY

Document currency = EUR

Country currency = USD

Resolution

If the the document currency and local currency are the same JPY.

We could find that the field is not visible.
