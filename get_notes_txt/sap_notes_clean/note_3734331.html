Symptom

  • The Data source field is empty, unable to save with FWB 00026 in specific OLAP connections (created or old connections)-->Connections page, starting BI 2025 version
  • When populating the field, and pressing Save button, the following error is seen: 
    • 500: Could not commit object '<CONNECTION NAME>' because another object with the same name and type already exists under object 'Connections'. (FWB 00026)

Other Terms

OLAP CMC, edit, modify, change

Reason and Prerequisites

Environment

  • SAP BusinessObjects Business Intelligence (BI) Platform 2025 Patch 08

Steps to duplicate

  1. Log in to Central Management Console,
  2. Navigate to Connections,
  3. Open a specific OLAP connections (new created or old).
  4. Observe the errors mentioned in the symptoms.
