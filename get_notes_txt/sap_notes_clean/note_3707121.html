Symptom

Time is incorrect in the field "Scheduled Start Time" and "Scheduled End Time" on the list page of the application Manage Process Orders.

Other Terms

F4587, abap_system_timezone, MfgOrderScheduledStartDateTime, MfgOrderScheduledEndDateTime, P_PROCESSORDERMGMTWRKLISTCALC

Reason and Prerequisites

Program error - incorrect time conversion. 
The problem occurred when the system time in the back-end system differs from the Personal Time Zone.

Solution

 Please apply the attached correction.
