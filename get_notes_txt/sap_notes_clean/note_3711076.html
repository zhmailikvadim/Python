Symptom

You are filling the required injury's information in infotype Delt@ (0968) and some values are missing for specific fields.

Environment

  • Human Capital Management (HCM)
  • SAP R/3
  • SAP ERP Core Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA

Reproducing the Issue

  1. Try to inform a specific value in a given field of infotype 0968.
  2. The value is not available for selection.

Cause

The infotype's fields fetch the values from tables with delivery class 'C - Customizing, maintenance only by customer, no SAP imports', meaning each customer is responsible for their content. These tables are listed below:

  • T5ED1 - Place type
  • T5ED2 - Variance
  • T5ED3 - Type of work
  • T5ED4 - Specific physical activity
  • T5ED5 - Contact method
  • T5ED6 - Injured body part
  • T5ED7 - Description of the injury
  • T5ED8 - Material agents
  • T5ED9 - Worker employment codes

Resolution

Maintain the required value(s) in the corresponding table(s) according to Delt@ specifications.
