Symptom

Error "CPD X: Valid from DD.MM.YYYY is in the past" when changing Valid To for an active CPD in Revision status.

Other Terms

CPD, change Valid date, CPD revision.

Reason and Prerequisites

 Incorrect VALID TO/FROM validation during CPD save in method /TMCSL/CL_SRV_V_CYCP_DEFS_C->CHECK_CYCP_DEF_CONSISTENCY

Solution

 Add Validation for VALID TO/FROM in method /TMCSL/CL_SRV_V_CYCP_DEFS_C->CHECK_CYCP_DEF_CONSISTENCY

You are using the IS-OIL movement based netting and you have some MM movements for which to value to be invoiced is 0 or for wich you do not expect any invoice.

You do not want to have these specific movements reported in the Movement without financial sections of the Netting Proposal but in the Receipt section instead.

A new BADI: OIA_BADI_ROIAAGAN_PROC_MM_FI is implemented to that purpose.

Other Terms

add-on, IS-Oil, industry solution, Borrow/loan, Zero Fee, ROIAAGAN, ROIAAGAN_FORMS_PROC_AND_COLLEC, PROC_AND_COLLECT_MM_FINANCIALS, OIA_BADI_ROIAAGAN_PROC_MM_FI, lo_badi_proc_mm_fi, fill_oia12i, OILNET_MSEG, 

IF_BADI_PROC_MM_FI, O3B7, O3B8, Netting Proposal

Reason and Prerequisites

You are creating a BORROW/LOAN exchange for which the meterial cost is posted against the Logical Inventory and that has not any fees or for which fees value is 0 . In that case there might be 0 value invoice created ( without accounting documents) or no invoice is created.

Solution

Please apply the correction included in that note and create the BADI implementation to fill table OIA12I.

The Fees (G_OIAFE) table must be read from the memory via the following statment : IMPORT G_OIAFE = G_I_OIAFE FROM MEMORY ID 'GOIAFE'. G_I_OIAFE is defined as : DATA: G_I_OIAFE type OIAFE_TT.

To have the record saved in the receipt section of the netting proposal the OIA12I,  Item Category should be set to '6' (C_receipt)   I_OIA12I-ITCAT  = C_RECEIPT. Where C_RECEIPT is defined as DATA: C_RECEIPT VALUE '6'.

I_G_NETTING_ITEM_NO need to be incremented.

The logic to find which values should be saved into OIA12I can be extrapoled from Form FILL_OIA12I (ROIAAGAN_FORMS_FILL_OIA12I).
