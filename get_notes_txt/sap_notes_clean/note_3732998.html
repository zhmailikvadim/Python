Symptom

Error "CPD X: Valid from DD.MM.YYYY is in the past" when changing Valid To for an active CPD in Revision status.

Other Terms

CPD, change Valid date, CPD revision.

Reason and Prerequisites

 Incorrect VALID TO/FROM validation during CPD save in method /TMCSL/CL_SRV_V_CYCP_DEFS_C->CHECK_CYCP_DEF_CONSISTENCY

Solution

 Add Validation for VALID TO/FROM in method /TMCSL/CL_SRV_V_CYCP_DEFS_C->CHECK_CYCP_DEF_CONSISTENCY
