Symptom

You cannot create Inventory Transfer Request via Pick and Pack, but no error appears.

Reproducing the issue

Scenario:

  1. Go to Inventory → Pick and Pack → Pick Pack and Production Manager. Select the criteria and click OK.
  2. On Open Drawer, select a document and choose Create → Transfer Request.

Expected behavior:

Transfer Request opens.

Actual behavior:

Nothing happens. No error message appears and Transfer Request does not open.

Cause

System Limitation

Solution

This is a system limitation.

Workaround:

  1. Go to Administration → System Initialization → Document Numbering → Inventory Transfer Request.
  2. Check that a Series exists and is usable. For example, Last No. was not reached, it is not locked, you have authorizations to the group, and so on.
