Symptom

This KBA explains the Attachment display logic in "Manage Automatic Payments" or "Revise Payment Proposals"

Environment

Financial Accounting (FI)
SAP Fiori for SAP ERP
SAP S/4HANA
SAP Fiori for SAP S/4HANA Finance
SAP S/4HANA Cloud Private Edition

Resolution

Both Fiori applications F0770 Manage Automatic Payments and F0771 Revise Payment Proposals
display attachment indicator based on attachment existence on document header level (attachment object type = BKPF).

Other object types like FIPP are not supported

Keywords

FIPP, BKPF, Object Type, Attachments, F0771, F0770, "Manage Automatic Payments", "Revise Payment Proposal"
