Symptom

For an employee, the employment status returned by 46HIR feature is other than '1' (meaning that it is not a regular employment) and a part time parental leave is recorded in a certain period. When payroll runs for that period, then payroll operation PLZLA throws a short dump because of division by zero.

Other Terms

PLZLA

Reason and Prerequisites

Gap in the implementation

Solution

Payroll operation PLZLA has been corrected.

Please apply the correction by importing the Support Package specified for your release, or by using the Note Assistant.
