Symptom

When you execute RZ70, the report _sld_bcsys returns incorrect license data when the license version exceeds 3.0.

Other Terms

RZ70,SLIC_LIKEY_GET_ALL_LICENSES,SLIC_LIKEY_GET_ALL_EXT_LICENSE, System License

Reason and Prerequisites

This is due to a program error.

Solution

Implement the attached corrections to your SAP Solution Manager system by using the Note Assistant (transaction SNOTE).
