Symptom

This note is only relevant for IDEX-AT3 Customers.

In case a Contract Partner relates to multiple PODs, if the Sender receives multiple rejection messages, each Process creates it's own BPEM Case, actually only one BPEM is expected, and all other processes should be set to status 'S2'(Completed Rejected).

Other Terms

IDXAT2, IDXAT3, Contract Partner, Master Data, 2031, 2131, multiple, BPEM, Change Entry

Reason and Prerequisites

Program error

Solution

Please perform following activities in this order:

1. Execute the Manual activities
2. Implement the attached automatic correction instructions
