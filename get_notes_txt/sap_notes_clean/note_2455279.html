Symptom

Transaction QS41 prompts for customizing request unwanted

or

One of these IMG activities prompts for customizing request unwanted:
Maintain Catalogs
Define and Configure Catalog for Inspection Codes
Edit Catalogs
Maintain Catalogs for Usage Decisions
Define Catalog Types and Code Groups for Catalog Types

Environment

This KBA applies for all SAP Products based on Netweaver.

Reproducing the Issue

  1. Start transaction QS41
  2. Maintain data and press icon Save  (Ctrl+S)

or

  1. Execute one of these IMG-activities
    Maintain Catalogs
    Define and Configure Catalog for Inspection Codes
    Edit Catalogs
    Maintain Catalogs for Usage Decisions
    Define Catalog Types and Code Groups for Catalog Types
  2. Maintain data and press icon Save  (Ctrl+S)

Cause

The Recording routine is set to Standard recording routine

Resolution

  1. Logon to your development system
  2. Start transaction SE54
  3. Enter Table/View V_QPGR_KAT
  4. Press radio button Generated Objects
  5. Press button Create/Change
  6. Confirm this Information
    Make repairs in foreign namespaces only if they
    are urgent
  7. Set radio button no, or user, recording routine
  8. Press icon Save
  9. In the status line this warning message appears
    You are changing a function group which does not belong to you
    Confirm this message with pressing Enter on your keyboard

Repeat steps 2. to 9. with Table/View V_QPCD_KAT
