Symptom

  • Search on text for a Lov of Hierarchy Object is not functioning as expected and returns "No Value to display"
  • No issue observed searching with Key values
  • Also if manually navigated through the hierarchy in the LOVs, there is no issue.
  • No issue when searched in RSRT.
  • Same issue in HTML/Applet and Webi Rich client.

Other Terms

webi,web intelligence,bw,bex,hierarchy,lov,search,text,No Value to display,OLAP,BICS

Reason and Prerequisites

Environment:

  • SAP BusinessObjects Business Intelligence Platform 4.2
  • Web Intelligence (Webi)
  • SAP BW
  • OLAP BICS

Steps to Reproduce:

  1. Launch Web Intelligence
  2. Select SAP Bex as Data Source
  3. Choose a Query which has Hierarchy Object
  4. Apply a Query filter on it
  5. Search for Lov by text and observe it returns "No Value to display"
