Symptom

When changing the Due Date in BCM, the updated Due Date is not reflected in APM for large Payment Orders.

Other Terms

/PF1/CL_BCM_PP_PI_DUE_DATE

Reason and Prerequisites

An incorrect data type was used for the Payment Order key during parallel processing. An additional required conversion was not applied.
As a result, the Due Date update process was terminated in APM for large Payment Orders.

Solution

Code correction.

Please apply SAP Note or assigned Support Package
