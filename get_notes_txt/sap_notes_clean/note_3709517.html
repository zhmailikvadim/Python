Symptom

While navigating to the agreement from the Master Agreement Promotions in fiori, the system does not display the Description for direct agreements in the popup window when multiple agreements are maintained for a single promotion.

This note ensures that the description is displayed for all types of agreements.

Other Terms

/IRM/CL_GAGR_UI_SERVICES, FIELD_VALUE_CHECK, DOCNR, Promotions, Agreements Description

Reason and Prerequisites

This is caused by program error.

SAP Note #3709441 is available for ECC Releases.

Solution

Please implement the correction instructions attached. These corrections are only valid if you have applied the Support Packs.

Release V4 -  2021, 2022, 2022 V2, 2023, 2023V2,2023 V3 Base Package or Support Pack 1.

Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object.
