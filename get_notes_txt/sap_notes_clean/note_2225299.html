Symptom

No rollback after termination in RSDRI_ODSO_DELETE_RFC.

Other Terms

RSDRI_ODSO_DELETE_RFC

Reason and Prerequisites

Program error

In the case of a deletion, no ROLLBACK takes place, which can sometimes cause the data you were trying to delete to be deleted anyway despite an error in the SQL execution.

Solution

  • SAP BW 7.30

    Import Support Package 15 for SAP BW 7.30 (SAPKW73105) into your BW system. The Support Package is available once SAP Note 2199121 ("SAPBWNews NW BW 7.30 ABAP SP15"), which describes this SP in more detail, has been released for customers.
  • SAP BW 7.31 (SAP NW BW 7.3 EHP 1)

    Import Support Package 18 for SAP BW 7.31 (SAPKW73118) into your BW system. The Support Package is available once SAP Note 2206236 ("SAPBWNews NW BW 7.31/7.03 ABAP SP18"), which describes this SP in more detail, has been released for customers.
  • SAP BW 7.40

    Import Support Package 14 for SAP BW 7.40 (SAPKW74014) into your BW system. The Support Package is available once SAP Note 2207824 ("SAPBWNews NW BW 7.40 ABAP SP14"), which describes this SP in more detail, has been released for customers.
  • SAP BW 7.50

    Import Support Package 2 for SAP BW 7.50 (SAPK-75002INSAPBW) into your BW system. The Support Package is available once SAP Note2220381 ("SAPBWNews NW BW 7.50 ABAP SP02"), which describes this SP in more detail, has been released for customers.

 

In urgent cases, you can implement the correction instructions as an advance correction.

You must first read SAP Note 1668882, which provides information about transaction SNOTE.

To provide information in advance, the SAP Note(s) specified above might already be available before the Support Package is released. In this case, however, the short text of the SAP Note still contains the term "preliminary version".
