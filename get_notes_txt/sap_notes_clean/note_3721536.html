Symptom

  • After a recent SAP SuccessFactors instance refresh, the IAS Admin credentials are failing to authenticate.
  • IAS admin is unable to log in to the IAS system.
  • Error message: "Scheduled email could not be sent to XXXXXX with subject How to reset your password. Pool size: 300. Active count: 1. Reason: IDS Mailing Exception: Email could not be sent to XXXXX with subject How to reset your password Caused by: Sending the email to the following server failed: mailserver.colourtex.co.in:587 Caused by: STARTTLS is required but host does not support STARTTLS".

Reproducing the Issue

  1. Perform an SAP SuccessFactors instance refresh.
  2. Attempt to log in to the IAS system using IAS Admin credentials.
  3. Observe authentication failure and inability to access the IAS system.

Cause

  • The mail server does not support STARTTLS, which is required for sending password reset emails.

Resolution

  1. Verify the email service settings for password reset functionality within the IAS system.
  2. Contact the mail server service provider to review and correct the settings, ensuring STARTTLS support is enabled.
    Please make sure that custom mail server is supporting STARTTLS.
  3. Reset the IAS Admin password and attempt to log in using the new password.
