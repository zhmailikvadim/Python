Symptom

When you execute payroll posting program RPCIPE00 or RPCIPE01 and there is a posting imbalance error the posting document is created but it excludes the employees for which the imbalance error was detected.

Cause

If errors occur on PERNR evaluation, which means evaluation of payroll results, the affected PERNRs are rejected from posting run.
For all other PERNRs contained in the selection the posting run is done anyway.
The reason is that you could not prevent the whole payroll area from being posted as soon as one or few PERNRs have issues with their payroll results.

Resolution

Correct the errors and run payroll posting again
