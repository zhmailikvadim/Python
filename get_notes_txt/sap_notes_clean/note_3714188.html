Symptom

 You use the API transaction /IWFND/GW_CLIENT to post Group Journal Entries. You attempt to post an entry where the line items have no amounts (empty values). The system does not display any error message, and the process seems to finish without validation. However, if you perform the same posting manually using the Post Group Journal Entries app, the system correctly raises the error message: "Document was not posted."

Other Terms

PGJE, API, Empty value

Reason and Prerequisites

The issue is caused by program error.

Solution

Implement the attached correction instruction or import the specified Support Package.
