Symptom

An error occurs in "Manage Withholding Tax Items" activity in Greece Withholding Tax Report (report category GR_WHT_1039_1027) with the error message: 

"A technical error has occurred, please check the selected filters as no data was found for the view I_StRpManageWhldgTaxItemC."

Environment

  • Financial Accounting (FI)
  • SAP S/4HANA Finance
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

1. Open Run Statutory Reports in Fiori Launchpad.

2. Generate a new run for report category GR_WHT_1039_1027.

3. Access the "Manage Withholding Tax Items" activity.

4. Observe the error message.

Cause

Missing configuration of Official Withholding Tax Keys and Income Types required for Greece Withholding Tax Reporting.

Resolution

Maintain the Official Withholding Tax Keys and Income Types and assign them to the appropriate withholding tax codes according to the SAP Help Portal documentation:

  • Defining Withholding Tax Keys
  • Defining Income Types

See Also

  • Defining Withholding Tax Keys
  • Defining Income Types
