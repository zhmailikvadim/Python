Symptom

A grid usage bill and the corresponding reversal document are processed (INVMON).

The grid usage bill then contains values in the totals fields for the net amount of the budget billing amounts (PAYED_BETRW_NET) and the invoice (TOPAY_BETRW_NET).

In the reversal document, these fields are empty.

This does not lead to an error during the transfer and aggregation.

The different values can lead to inconsistencies during the evaluation in a connected BW system (DataSource 0UC_INVLINB).

Other Terms

INV_BW_INVLINB_DELTA_WRITE, 0UC_INVLINB

Reason and Prerequisites

There is a program error.

Solution

Implement the correction attached to this SAP Note or import the relevant support package.
