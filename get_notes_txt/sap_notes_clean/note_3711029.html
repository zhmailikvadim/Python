Symptom

In app Custom Fields, custom field are configured based on field FirstAcquisitionFiscalPeriod. But the custom field shows empty value in app Display Line Items in General Ledger.

Environment

SAP S/4HANA Cloud Public Edition

Cause

The fields are deprecated in CDS view I_FixedAsset. CDS view I_FixedAsset can’t be used to retrieve these data in custom extension.

Resolution

CDS view I_FixedAssetForLedger can be used.

Below is the link for I_FixedAssetForLedger: https://api.sap.com/cdsviews/I_FIXEDASSETFORLEDGER.

Keywords

Display Line items in General Ledger, CDS view, I_FixedAsset, I_FixedAssetForLedger, FirstAcquisitionFiscalPeriod
