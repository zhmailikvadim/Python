Symptom

  • The heatmap function in SAP Emarsys only works for single-language campaigns.
  • Multilanguage campaigns are not supported by the heatmap feature due to technical limitations.

Environment

SAP Emarsys Customer Engagement

Reproducing the Issue

  1. Utilize SAP Emarsys to create a multilanguage campaign.
  2. Attempt to access the heatmap functionality for the campaign.
  3. Observe that the heatmap feature is not available for multilanguage campaigns.

Cause

The heatmap feature in SAP Emarsys is technically limited to single-language campaigns due to the inability to generate a static "master" image for multilanguage campaigns where content varies significantly per recipient.

Resolution

  1. Note that the Visual Link Tracking (Heatmap) feature is designed for single-language campaigns only.
  2. For multilanguage campaigns, ensure that links are distinct per language to analyze performance.
  3. Export the click data and use the "link_name" column to identify performance across languages.
  4. Add IETF tags (e.g., banner_en-US) to links to facilitate data analysis.
  5. Build custom heatmap reports using Excel or BI tools to compare performance across different languages.

Keywords
