You may experience long runtime when creating or modifying core data for an oil material in transactions MM01 or MM02.

Other Terms

  • General keywords: add-on, IS-Oil, industry solution, HPM.
  • Problem-specific key words: material master, performance issue, MM01, MM02.

Reason and Prerequisites

Program error.

The function module CHECK_MATERIAL_OIL triggers many checks on stock tables and valuation that are unnecessary when no oil-related fields are modified.

Solution

Please implement the correction instruction.
