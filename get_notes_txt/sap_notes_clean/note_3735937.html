Symptom

"CSA Collector Framework" Monitoring is missing in the Self-monitoring Dashboard.

Environment

SAP Focused Run

Cause

The CSA Collector Framework Error alert is by default Inactive in Self Monitoring template

Resolution

Go to FRUN system:

1. Enter Tcode: MAI_TOOLS

2. Click on Activate Expert Mode

3. Tab Configuration

4. Self-Monitoring --> Template Maintenance for Self-Monitoring

5. In Technical System

6. Select Self-Monitoring for Technical System, the CSA Collector Framework Error alert is currently inactive

7. Copy standard template to a custom template, and check the System Monitoring checkbox for "CSA Collector Framework Error" and its event and metric. 

8. Go to FRUN launchpad --> System Monitoring Advanced Configuration

9. Click on Manual Template Assignment, select Monitoring Use Case "Self Monitoring". 

10. Please assign the newly created self monitoring template to the affected Managed Objects. 

11. Click on Reconfigure.

Keywords
