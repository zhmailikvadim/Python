Symptom

  • Outlook calendar integration in SAP Fieldglass is not functioning as expected.
  • Meetings are not visible in the calendar widget.

Environment

  • Integration

Reproducing the Issue

  1. Log in as the user who wants to schedule meeting in the SAP Fieldglass system.
  2. Select a job seeker and attempt to schedule an interview with the user in the admin role.

Resolution

Please find below steps to check the blockers in the calendar

  1. Provide application permissions in Outlook to the admin's calendar and set up the Outlook email in the admin's preferences.
  2. Login as the user > Open job seeker details 
  3. under Actions > Schedule interview > Select Interviewers drop down value > Click on + Add Interviewer
  4. Check the calendar now, it will show the blockers for the user selected with integrating the calendar integrations
