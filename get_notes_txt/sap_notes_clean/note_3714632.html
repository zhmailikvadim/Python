Symptom

For instance, a cancelled version exists. While reading instance details, It read cancelled version resulting into no instance data.

Other Terms

/APEU, /APEU/CL_MCM_COMMON_UTILITY, GET_INSTANCE_DATA_FROM_OBJECT, /APEU/INST_MODEL, Cancelled Instance,

Reason and Prerequisites

Program Error

Release 2508

Solution

Please implement the attached correction instruction.

Cancelled instance does not have valid from and valid to dates. When instance data is being read,  such cancelled instances gets read and stops system from executing fallback logic with valid to date as infinite date. We have added additional condition in Select query to restrict selection of cancelled instances.
