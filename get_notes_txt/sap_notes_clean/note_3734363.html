Symptom

When a customer/partner has a registered prefix name longer than 6 characters (resp. 8 characters with leading and trailing: '/'), then previously the error message:
"&1 not suitable as namespace for authorization objects"
comes up when trying to use this prefix as a namespace for authorization objects.

As this message does not describe the cause of the problem, the message text was adapted with this SAP Note to:
"Prefix name &1 is too long (maximum 6 characters; see SAP Note: 395083)"

Other Terms

UE-217

Solution

Import the relevant support package.

When using the Note Assistant to implement the correction instruction, the new message text is only available in orignal language: German
