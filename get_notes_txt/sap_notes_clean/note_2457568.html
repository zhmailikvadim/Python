Symptom

Unable to get any score in the IMPACT/LINEAGE workflow in SAP Information Steward 4.2

Environment

  • Any Windows Server platform
  • SAP BusinessObjects BI Platform 4.x
  • Any SAP Data Services 4.2
  • Any SAP Information Steward 4.2

Reproducing the Issue

  1. Implement and run the appropriate integrator on an RDBMS datasource
  2. Create a rule validation (with a score) on one of the table that include the RDBMS datasource
  3. Create a scorecard including this rule.
  4. Run a task rule that include the rule validation previously created
  5. In the scorecard the score is displayed in the detailed.
  6. Open the IS Metadata Management area
  7. Navigate to the appropriate table which include a validation rule and the score
  8. The score should be visible from the within the IS MM when a mouse over is done on the column navigation (in a tool tip) but it is not displayed.

Cause

The tooltip in Metadata Management will show only if the table is part of enterprise project in Information Steward. when creating a different project in Information Steward to do the same, it will not.

Resolution

  • Use "Enterprise project" as the unique project within SAP Information Steward

 

  • Raise an Enhancement request : 1515837 - How To: Enhancement Request Process - "Idea Place" to attach all tooltips belong to "Enterprise project" to any projects within SAP Information Steward
