Symptom

  • Sub-screens (Header, Item, Tax, and Tender) in the SAP CAR Workbench (/POSDW/MON0) fail to render.
  • The right-hand detail pane remains blank when selecting a transaction line in the main list.
  • No error message is displayed in the status bar.

Environment

  • Customer Activity Repository
  • CAR 4.0
  • CAR 5.0
  • CAR 6.0

Reproducing the Issue

  1. Execute Transaction code /n/POSDW/MON0.
  2. Enter selection criteria and execute to display the list of transactions.
  3. Select any transaction line from the results list.
  4. Observed Result: The right-hand detail pane is blank. No error message is displayed in the status bar.
  5. Expected Result: The sub-screens showing Header, Item, Tax, and Tender information should be visible.

Cause

  • Known issue with Edge Webview2 version 144.0.XX affecting SAP GUI HTML control when using Edge as the browser control in SAP GUI for Windows.
  • 3rd party product issue.

Resolution

Read and follow KBA 3704912 - Latest Webview2 Version 144.xxx causes problems with SAP GUI HTML control when using edge as browser control in SAP GUI for Windows
