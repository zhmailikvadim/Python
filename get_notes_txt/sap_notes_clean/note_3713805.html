Symptom

When a user uploads a file containing medication numbers, in certain scenarios the system finds that the sequence numbers are already present in the system and overwrites them with the next available sequence numbers.

Performance issues observed

Other Terms

ISLS, Intelligent Clinical Supply Management, Manufacturing, Manage Medication Lists, sequence numbers

Reason and Prerequisites

You use SAP Intelligent Clinical Supply Management or this SAP Note is a prerequisite of another SAP Note.

Solution

Implement the attached correction.

The system no longer overwrites sequence numbers. If duplicates are found, an error is raised.
