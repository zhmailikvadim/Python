Symptom

You are using the Advanced Scheduling Board application in Production Planning and Detailed Scheduling and you notice the following symptom occur:

S.No Symptom Release(s)
1. When you open network view charts for any charts other than resource chart, reset is not working. 2025 FPS1
2. When you do Custom select option from Gantt toolbar, objects are not selected in the same chart as user had selected. 2025 FPS1
3. Network View chart content from a previous variant persists after selecting a new variant. 2025 FPS1, 2025 FPS0, 2023 FPS3, 2023 FPS2, 2023 FPS0
4. The Go to first object and Go to last object actions in the toolbar do not work if the selected rows are not currently visible on the screen. 2025 FPS1, 2025 FPS0, 2023 FPS3, 2023 FPS2
5. Drag and drop operations to alternate resource containing special characters does not work as expected 2025 FPS1, 2025 FPS0, 2023 FPS3, 2023 FPS2, 2023 FPS0
