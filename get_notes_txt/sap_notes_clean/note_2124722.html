Symptom

When you add a partner, the system issues an error message stating that no other partner can be created in a certain role because the role must be uniquely assigned to a partner.

Other Terms

QIM_OVP_ISSUE QIM_OVP_ACTIVITY /IAM/OBJECT248

Reason and Prerequisites

You have flagged partner roles as unique for the issue type or activity type.

Solution

Implement the correction instructions.
