Symptom

  • Profit Center description is not displayed in the Fiori App – Incoming Sales Orders – Flexible Analysis.
  • Only the Profit Center number is displayed in the output.

Environment

  • Sales and Distribution (SD)
  • Fiori
  • SAP S/4HANA 
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Select Profit Center as a row in the Fiori display output.
  2. Place the cursor into the characteristic ‘profit center.’
  3. Click the button labeled ‘display.’
  4. Only one option is available, and only the Profit Center number is displayed in the output.

Cause

  • The underlying CDS views only provide the ID for the Profit Center field and do not include the description.

Resolution

The Profit Center description is not available in the standard Fiori App due to the design of the underlying CDS views.
