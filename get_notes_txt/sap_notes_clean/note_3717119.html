Symptom

  • Client would like to know how to use the Contract functionality on Service feature
  • What Configuration is necessary to have Contract on the Statement of Work?

Cause

  • Contract Type requires Rates 2.0 to be enabled on Company Configuration, since is incompatible with Rates 1.0. In addition, Statement of Work must be enabled to use Contract functionality
  • Contracts are Created and Revised only through Integrations, and to Upload Contracts using the Connector, Contract Types admin objects must be created in the user interface

Resolution

To enable Rates 2.0 in the SAP Fieldglass application, follow these steps:

  1. Reach out to your SAP Fieldglass Representative  to initiate the process of enabling Rates 2.0 for your company
  2. Request Company Configuration called Enable Rates 2.0. This configuration is essential for activating the new Rate structure
  3. After the configuration is enabled, you can start creating new Rates 2.0 admin objects, such as Rates, SOW Types and SOW Templates and Contract Types
  4. When you have everything setup you can enable to Connectors to Upload the Contract Files to SAP Fieldglass
