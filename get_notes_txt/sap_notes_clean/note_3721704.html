Symptom

When executing batch determination in T-code CO01/CO02, the system issues message CO350 Material & is not available in quantity & on &.

Environment

  • SAP ERP Shop Floor Control (PP-SFC) 
  • SAP R/3
  • SAP R/3 Enterprise
  • SAP ERP Central Component
  • SAP ERP
  • SAP Enhancement package for SAP ERP
  • SAP S/4HANA
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Open Component Overview in T-code CO01/CO02 and making batch determination.
  2. System issues message CO350.

Cause

Checking the value of 'Res./Purc. req.' for this component, it's set to 1(never).

Resolution

To solve this issue, it suggests to set the value of 'Reservation/Purch.Req.' to 3 in T-code OPL8.

Keywords

CO350, CO 350, Batch determination, OPL8, Res./Purc. req.
