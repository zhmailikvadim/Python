Symptom

  • An error occurs in the system log with the message: "Object sap_skins/corbu/images/menu_arrow_down-2.png is not valid in URL."
  • The system fails to display a small arrow in the top line of Web UI due to the system expecting a different file format for the image. 

NOTE: All information and pictures were taking from sample test system and do not represent actual data (any resemblance as such is purely coincidental)

Environment

  • SAP S/4HANA 
  • SAP Customer Relationship Management

Reproducing the Issue

  1. User logs in the Web UI system and the menu_arrow_down-2.png Saved Search image is not displayed correctly.
  2. BSP exception in SM21 as below (Message ID – BRA)
    Error message: Object sap_skins/corbu/images/menu_arrow_down-2.png is not valid in URL
    /SAP/BC/BSP/SAP/thtmlb_styles/sap_skins/corbu/images/menu_arrow_down-2.png
 

Cause

  • Run SE80
  • Open the BSP application THTML_STLS,
  • Open MIMEs -> Corbu -> images, the image menu_arrow_down-2.png does not exist. 

Resolution

  1. Download the existing object menu_arrow_down.png

     


  2. And again creating a new object with the name menu_arrow_down-2.png as shown below

  3. Now re-test your scenario.
