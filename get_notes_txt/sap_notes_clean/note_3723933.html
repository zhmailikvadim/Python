Symptom

  • After running MRP Live (T-Code: MD01N) in SAP GUI, the MRP Run Date is not saved in table MDKP anymore.
  • but In the App “Display MRP Master Data Issue,” the MRP Run Date is visible, we would like to understand how to capture the field "date/time creation" in this app.

Environment

  • SAP ERP Logistics (PP)  
  • SAP S/4 HANA, on premise
  • SAP S/4HANA Cloud Private Edition
     

Reproducing the Issue

  1. Run T-code: MD01N for the material
  2. Check table MDKP, no entry is updated
  3. Open App "Display MRP Material Master Issues" for the material, verify that the "date/time creation" is displayed.

Cause

No MRP list is written by MD01N in SAP S/4HANA as this is standard behavior.

Resolution

  1. Refer to SAP Note 0002268085, which explains that MRP lists were intended for checking the MRP result and are now replaced by real-time MRP apps in SAP S/4HANA.
  2. The app read MRP run time from table PPH_MRP_ISSUES, field TIMESTAMP(saved in GMT time-zone)

See Also

  • Refer to: SAP Note 0002268085 - S4TWL - MRP Live on SAP HANA - MD01N
