Symptom

When attempting to open a quote on Sales Cloud V2 end, that has only VIEW action allowed in CPQ Workflow, error is thrown informing that Edit actions is not available in that status.

Environment

SAP SALES CLOUD CPQ Integrated with SAP Sales Cloud V2

Reproducing the Issue

  1. Log into SAP SALES CLOUD V2
  2. Open an Opportunity
  3. Open a Quote in order to land on CPQ
  4. Check on CPQ landing page that error is thrown informing that Edit action is not available.

Cause

System Limitation.

Resolution

  • Issue: CPQ Integration with SAP SALES CLOUD V2 do not allow quotes to be accessed using View Action
  • Workaround: Configure on CPQ Workflow the Edit action and configure the proper permissions to users not modify fields on certain quotes
  • ETA: Not available yet.
