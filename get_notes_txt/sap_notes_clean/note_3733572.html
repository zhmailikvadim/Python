Symptom

Email notifications that are sent to users regarding Web Service Message Monitoring Errors - eg, Process integration error; BusinessPartnerERPReplicationIn. You wish to deactivate these.

Environment

SAP Cloud For Customer

Reproducing the Issue

  1. A web service message related to BusinessPartnerERPReplicationIn fails in the Web Service Message Monitoring view.
  2. Users who are subscribed to notifications receive an email titled with “Process integration error; BusinessPartnerERPReplicationIn”.

Cause

Resolution

  1. Use user-level control: In the Web Service Message Monitoring view, instruct users to unsubscribe from email notifications using the Subscribe/Unsubscribe option if they no longer wish to receive alerts.
  2. For non-productive tenants: In Business Configuration, open the “E-Mail and Fax Settings” activity and review the section “E-Mail and Fax Delivery in a Non-Productive System”; adjust settings to prevent outbound email delivery if system-wide suppression is required in test environments.
  3. A dedicated scoping toggle to globally disable Web Service Message Monitor email notifications in productive tenants is not available per current information; manage notifications via user subscriptions.
