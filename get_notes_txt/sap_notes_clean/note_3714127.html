Symptom

You have defined a default workbook in Analysis Office as described in the Analysis Office documentation.

Please refer to following link for more details:

To insert a data source into a default workbook | SAP Help Portal

You insert an input ready planning query in Analysis Office.

The data source does not appear input ready in Analysis Office and you cannot plan data.

Other Terms

planning, new lines, sequence, function

Reason and Prerequisites

This is a program error that occurs only when the data source is inserted from a different system than the one specified in the default workbook.

Solution

The correction will be delivered by a SAP Analysis for Microsoft Office Support Package.

In section "SP Patch Level" you can find information which Support Package contains the respective correction.

If the section "SP Patch Level" is not available or the information about which Support Package contains the correction is missing, this means that this has not been determined yet. In this case please re-check this note at a later point of time.

Delivery Information

See also note 1518359 for planned delivery schedule.

    Install the Support Package for SAP Analysis 2.8 for Microsoft Office.
