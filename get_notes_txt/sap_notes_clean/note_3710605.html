Symptom

You are SAP S/4HANA Group Reporting, private edition customer and use the API API_CNSLDTNGRPJRNLITEM

 

- for reading ACDOCU data in SAP Group Reporting Data Collection (GRDC) for Data Mapping and/ Or Forms.
- for extracting data to BW or other systems.

 

In contrast to the predecessor API  API_GRTRANSACTIONDATA you are missing the fields Project, Material, WBSElement and WBSElementDescription.

Other Terms

 API, ACDOCU, GRDC

Reason and Prerequisites

Field not added to CDS view A_CnsldtnGrpJrnlItem

Solution

Implement the correction instruction of this SAP Note or install the corresponding support package.
