Symptom

•    In the EWM KPI app (Warehouse KPIs - Operations, App ID F4024), some tiles fail to open.
•    Error: "Loading table data failed. Property EWM Warehouse ID not found in type EWM Warehouse."

Environment

  • SAP Extended Warehouse Management
  • SAP S/4HANA

Reproducing the Issue

  1. Enter transaction code /UI2/FLP.
  2. Search for the EWM KPI App.
  3. Attempt to open the tiles in the EWM KPI App.
  4. The following error message appears:
    “Loading table data failed. Property EWM Warehouse ID not found in type EWM Warehouse.”

Cause

Coding Issue

Resolution

1.    Implement the following note: 3588101 - OQ: DimF-Q for optional filter Part 2.
2.    Clear cached data as described in scenario 1 of KBA: 2743849 - Resource not found for the segment 'XXXXXXXX'
