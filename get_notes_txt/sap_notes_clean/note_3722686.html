Symptom

When executing "Activity-Based Costing - Production" with a reference run , the fixed process rates for the manufacturing processes of the new products that are not available in the reference run are not calculated.

Other Terms

Activity-Based Costing - Production, Reference Run, Fixed Process Rates

Reason and Prerequisites

Program error

Solution

History:

Install the code correction.

Description of Correction:

When executing "Activity-Based Costing - Production" with a reference run , the fixed process rates for the manufacturing processes of the new products that are not available in the reference run are calculated.

There are no functional dependencies.
