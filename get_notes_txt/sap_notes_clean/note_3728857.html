Symptom

When creating a Fiori Elements application (e.g., List Report) using the project wizard in SAP Business Application Studio (BAS), OData V4 services are not appearing in the service dropdown list.

Environment

- S/4HANA FPS 2023 On-Premise system
- Business Application Studio
 

Cause

The OData V4 services are not visible in BAS because the `/IWFND/CONFIG` service was not published in the target client.

In multi-client SAP landscapes, publishing a service in one client (e.g., client 100) does not automatically make it available in other clients (e.g., client 250). Each client requires explicit service publication.

Resolution

Publish the `/IWFND/CONFIG` service in client 250 as well. After publishing the service in the target client (ED3-250), the OData V4 services became visible in BAS.
