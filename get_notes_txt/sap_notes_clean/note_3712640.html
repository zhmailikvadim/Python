Symptom

During result analysis with sales order or WBS element, it's found the actual cost is shown as not included. And the system does not consider the cost posted on the cost element.

Environment

  • Controlling (CO)
  • Financial Accounting (FI)
  • SAP R/3
  • SAP R/3 Enterprise 4.7
  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA
  • SAP S/4HANA Finance

Reproducing the Issue

  1. Execute transaction KKA2
  2. Input the relative fields and run it

Cause

During the execution the system will try to get the line ID assigned to the cost element in transaction OKG5. And then it will check the update setting in the result analysis in transaction OKG4. In case the category for the line ID is not set correctly ( N, for instance), this kind of issue can occur.

Resolution

Please check the setting in transaction OKG4 for the relative line ID and make sure the correct category is assigned.
