Symptom

While performing a data migration to SAP S/4HANA 2023 via the Migration Cockpit, a discrepancy has been identified regarding the Service Contract OP 2023 object.

Specifically, the number of service contracts currently residing in the target system exceeds the volume provided in the XML source files/staging tables.

This SAP Note corrects this issue of service contract migration object so that the system can avoid creation of duplicates service contracts.

Other Terms

LTMC, Migration Cockpit, Migration Service Contract,  SIF_SRVC_CONTR_X, LTMOM

Reason and Prerequisites

The default path of the header structure relation is not correct.

Solution

Implement the following SAP Note's transport-based correction instructions (TCI's) to migrate the "Service Contract OP 2023" migration object:

Release Service Pack Solution
2023 SP00 - SP05 Implement TCI Note 3727955
