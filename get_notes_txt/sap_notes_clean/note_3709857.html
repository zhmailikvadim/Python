Symptom

It is mandatory to have the sub-business unit of Singapore Government Agencies as the buyer reference in customer e-invoices submitted through Peppol. This note enables Branch Code to capture the sub-business unit information.

For more information about the development delivered with this SAP Note, see the following What's New topics on SAP Help Portal: 

  • SAP S/4HANA and SAP S/4HANA Cloud Private Edition: Enabling Branch Code for Sub-Business Units of Singapore Government Agencies.
  • SAP ERP: Enabling Branch Code for Sub-Business Units of Singapore Government Agencies.

Other Terms

Business Partner, Singapore, Business Unit, Branch, Branch Code

Reason and Prerequisites

This is a legal change.

Prerequisites:

  • 3705924 - Creating Package for Branch Code to Contain Sub-Business Units Information
  • 3713259 - Electronic Invoice Processing for Singapore: Meet the New Requirement for Peppol Participant IDs in B2G Invoicing

Solution

Apply the corresponding support package or implement the correction instructions in this note.

For more information about the development delivered with this SAP Note, see the following What's New topics on SAP Help Portal: 

  • SAP S/4HANA and SAP S/4HANA Cloud Private Edition: Enabling Branch Code for Sub-Business Units of Singapore Government Agencies.
  • SAP ERP: Enabling Branch Code for Sub-Business Units of Singapore Government Agencies.
