Symptom

This NOTE resolves the following issues in Generic Data Distribution (GDD):

  • During the Data Transfer phase of Generic Data Distribution, certain objects are no longer available in the source system. In such cases, the GDD object status is now set to Completed with appropriate message.
  • The issue where the Reference Location status(derived as a dependent object) in GDD remained “In Process” even after all relevant queues were successfully completed has been corrected.
  • The multiple-selection option in F4 help of GDD reported has been restricted due to technical limitations. Users can now select only a single value at a time using F4 help.

Other Terms

S4 Defense Disconnected, GDD, Equipment, Functional Location

Solution

Implement the attached correction instructions provided with the NOTE.
