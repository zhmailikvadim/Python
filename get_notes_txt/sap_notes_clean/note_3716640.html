Symptom

Batch tab is not displaying batch/valuation details for online stock in stock lookup

Other Terms

  • SAP Service and Asset Manager
  • SAM
  • SSAM
  • ICMTANGOAMF10-42756
  • ICMTANGOAMF10-42863

Reason and Prerequisites

Reason:

MaterialBatches and Batch Indicator are empty for online stock search so existing code to display batch and valuation is not getting executed.

Prerequisites:

  • Apply SAP Note 3685651 
  • BAS or Visual Studio Code with SAM MDK project
  • `patch` command (available on BAS and macOS by default) via terminal or other patching utility
  • Read the guidelines on how to apply the patch and read the appendix section if you receive any error

Solution

MaterialBatches and Batch Indicator are queried from the backend for online stock search so existing code will display batch and valuation fields.

To apply the patch, download the attached 3716640_<version>.patch file and save it in a known location, preferably adjacent to your SAM.mdkproject.

If you are on Windows, see SAP Note 3533879 for how to install a .patch file in Windows. If you are on a Mac, follow the steps below:

  1. Open a terminal in the root of your SAM metadata (i.e. SAM.mdkproject/metadata or SAM.mdkproject/metadata/base.definitions)
  2. Run the terminal command `patch --no-backup-if-mismatch -p4 < path/to/3716640_<version>.patch`. 
  3. Verify in the terminal window that the following files have been modified and listed as patched, as indicated by the console output:

  • Rules/Inventory/Stock/GetBatchAndValuationCategoryStockRows.js
