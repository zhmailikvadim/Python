Symptom

IAM information system is not showing all applications available for the user in "XXX - Application" tab. 

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Access the IAM Information System application.
  2. Select the main entity as XXX.
  3. Navigate to the XXX - Application tab.
  4. Observe that only a few applications are listed.

Resolution

The tab "XXX - Application" only shows apps that belong to non-migrated business catalogs. This is only one part of the apps.

The other part can be viewed in the "XXX - IAM App" tab, where all apps of migrated catalogs (or directly assigned IAM Apps) are shown. This is a new feature of the IAM Information system.
