Symptom

You run the EA or EC Annual Tax Report (RPCTEAL0_01) report and leave the Selection section on the selection screen as empty. You enter Payroll period, Employer's Reference No, and Year of EA Tax Assessment. It takes an extremely long time in running the report. And the process might be terminated by runtime error: time_out.

After you've installed the SAP Note, the following error message occurs immediately:

Input Personnel Number or any other Selection Criteria at first!

Version 2:

Add a new "without Printing Breakdown List" checkbox on the selection screen of EA report.

Other Terms

EA or EC Annual Tax Report, RPCTEAL0_01, performance, 5L122, 14REP, dump, filter_empty_select, filter_molga, filter_empty_payroll, CP21, CP22A, TP3, 5L346

Reason and Prerequisites

This is performance fine-tuning of EA report and has no impact on the output of this report.

To run the report RPCTEAL0_01 as expected, you must enter parameters in one or more of the following fields in the Selection section on the selection screen:

  • Personnel Number  
  • Personnel Area
  • Personnel Subarea
  • Payroll Area
  • Organizational Unit

The report selects employees to be processed by the latest infotype record according to above inputs. Leaving the Selection section empty prevents the report from functioning as expected.

The input of Employer's Reference No. field cannot work as selection criteria of employees because its filtering is based on the entire year’s payroll results.

It is recommend to run EA report with small group of employees.

Solution

You can apply HR Support Package or implement the correction instructions in this SAP Note and carry out the manual activities attached.
