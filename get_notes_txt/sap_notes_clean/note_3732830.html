Symptom

Apprentices get some reward for their work. The Act on Apprenticeship determines minimum, but still non-taxable reward amounts for each grade in the school.

The reward amounts have been revalorized with 01.03.2026.

Other Terms

 FTAP1, FTAP2, FTAP3, payroll rule SITS, apprenticeship reward

Reason and Prerequisites

  • Legal Source: Informacija o usklajenih zneskih vajeniških nagrad published by the Chamber of Craft and Small Business of Slovenia.

Solution

  • Payroll constants for minimum and non-taxable amounts for apprenticeship reward (REK-O income type 1021) in T511K are delimited with date 01.03.2026 as follows:
    • FTAP1  Reward for apprentice 1st grade = 317,85 EUR
    • FTAP2  Reward for apprentice 2nd grade = 381,42 EUR
    • FTAP3  Reward for apprentice 3rd grade = 508,56 EUR

Implementation

  • Generally, SAP recommends that you install a solution by applying a Support Package. However, if you need to install a solution earlier, implement automatic correction instructions and provide manual post-implementation steps.
  • Note that documentation, transactions, translation and some customizing activities are only delivered via Support Package.
