Symptom

Filters for Orders and Returns are null in backoffice customer support cockpit

Reason and Prerequisites

Reason: Program Error

Steps to reproduce:

  1. Login backoffice as customer support agent or manager or administrator
  2. Open Customer Support Cockpit
  3. Open Orders/Returns
  4. Check Filters in the left hand
  5. Filters are null

Solution

Temporary workaround: Replace wrong widget id customersupportbackoffice.typenode.order.all and customersupportbackoffice.typenode.all.returns with correct id customersupportbackoffice_typenode_order_all and customersupportbackoffice_typenode_all_returns

Solution: The fix will be available in Commerce version 2211-jdk21.12
