Symptom

  • Error message: /SAPAPO/RELH040 'Invalid object' occurs when attempting to display a transportation zone hierarchy using transaction /SAPAPO/RELHSHOW.
  • The hierarchy exists in hierarchy tables, but the transportation zone itself does not exist.
  • The hierarchy does not exist in table /SCMB/V_TZN_SCU view.

Environment

  • S/4 HANA 2020
  • S/4 HANA 2021
  • S/4 HANA 2022
  • S/4 HANA 2023
  • S/4 HANA 2025

Reproducing the Issue

  1. Attempt to display a transportation zone hierarchy using transaction /SAPAPO/RELHSHOW.
  2. Observe the error message: /SAPAPO/RELH040 'Invalid object' happened.

Cause

  • The transportation zone was deleted before the hierarchy was deleted.
  • The hierarchy exists in hierarchy tables, but the transportation zone itself does not exist.

Resolution

  1. Re-create the transportation zone.
  2. Delete the hierarchy using transaction code /SAPAPO/RELHDELETE.
