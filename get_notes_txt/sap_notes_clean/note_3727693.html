Symptom

Rules are not copied when a new version of a Service Level Agreement is created.
Periodically, the runtime error GETWA_NOT_ASSIGNED is logged when changing, updating, or deleting rules or conditions.

Other Terms

F6183

Reason and Prerequisites

In some cases, updating the HANA buffer takes time. When an operation is executed immediately after a creation request, the system may not retrieve the newly created data. This can cause issues during update, delete, or display operations, resulting in rules or conditions not being displayed.

Solution

The BYPASSING BUFFER addition was implemented to read the data directly from the database.

Please apply SAP Note or assigned Support Package.
