Symptom

For closed-out agreements in the Fiori application, the description of the close-out reason is not displayed in the Close-Out tab, even though the value property is maintained as “Show Value and Description” in the UI profile.

This note ensures that the description is displayed for the close-out reason field.

Other Terms

 /IRM/IPBBACO, CORSN, /IRM/TIPACORSN, Show Value and Description, Close Out Reason

Reason and Prerequisites

This is caused by program error.

SAP Note # 3714024 is available for ECC Releases.

Solution

Please implement the correction instructions attached. These corrections are only valid if you have applied the Support Packs.

Release V4 -  2022 V2 Base Package, 2023, 2023 V2

1. Perform the manual pre-implementation steps that are described below.
2. Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object.
