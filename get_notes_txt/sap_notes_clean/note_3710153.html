Symptom

Why Purchase Requisition (PR) is stuck in Ordering status where we could see that every half an hour the Purchase Order (PO) is getting repushed.

The issue is not happening for all the PO.

Cause

Both Export Purchase Orders web service and Export Purchase Orders Asynchronously are enabled. 

Resolution

Either one of the integration that is Export Purchase Orders web service or Export Purchase Orders Asynchronously can be enabled.

If both are enabled, system will randomly pick and send the integration events which it finds first. 

Keywords

PR stuck in ordering, ordering both CIG and Webservice.
