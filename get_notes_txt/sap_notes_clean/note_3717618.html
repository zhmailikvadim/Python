Symptom

This SAP Note is the introduction of new separation criteria for collectors using collection by Settlement BIC.

Other Terms

/PE1/RN, Collection Scenario, separation criteria and Clearing Agreement.

Reason and Prerequisites

New functionality. The goal is to allow one clearing agreement per corporate, combined with a flexible and configurable set of separation criteria linked to the Collector definition, instead of creating multiple agreements or custom implementation.

Solution

Implement the correction instruction attached to this SAP Note. Alternatively, you can import the referenced Support Package.
