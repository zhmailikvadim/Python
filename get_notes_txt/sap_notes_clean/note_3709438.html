Symptom

  • Running Software Update Manager (SUM) tool for SAP S/4HANA conversion, the phase PREP_CONFIGURATION/INITSUBST request only Downtime parameters. It does not display input boxes for Uptime parameters.
  • The phenomenon is only relevant when using Software Update Manager (SUM) along with Database Migration Option (DMO) & System Move.

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental"

Environment

  • SAP S/4HANA 2023
  • SAP S/4HANA 2025
  • Software Update Manager (SUM)
  • SAP NetWeaver AS ABAP
  • Database Migration Option (DMO)
  • System Move

Reproducing the Issue

Repeat phase PREP_CONFIGURATION/INITSUBST from SUM.

Cause

This is standard feature in SUM and not a defect or bug.

Resolution

When using DMO with System Move, exporting data on the source system occurs during downtime, and all processes on the target system also runs in downtime. Therefore, it is not necessary to request uptime values during the INITSUBST phase.
