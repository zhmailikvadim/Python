Symptom

  • In SAP Analytics Cloud (SAC), for existing BW queries, the dimensions within the CDS view does not allow the selection of any presentation view options; all options are grayed out.
  • When creating a new query, the presentation view options are selectable.

Environment

SAP Analytics Cloud 2025.21.11 

Reproducing the Issue

  1. Log in to the SAC tenant.
  2. Navigate to the model associated with the issue.
  3. Go to Data Management and select the existing BW import job.
  4. Edit the query and proceed to the next step.
  5. In the dimension, check the dimension and attempt to select a presentation view.
  6. Observe that the presentation options are grayed out and cannot be selected in the existing import job query.
  7. Repeat the process for a newly created import job, and notice that the presentation view options are selectable.

Resolution

This behavior is by design. It is not possible to change presentations when editing an import job. For more information on this, please check our guide.
