Symptom

The 'Maximum condition value' maintained for the condition type in T-cd: VK13 is not considered in the price calculation for Service Contract and Service Order in S/4CRM.

Environment

SAP S/4HANA Cloud Private Edition

Resolution

The condition update is not supported in S/4 Service, the details can be found in the following documentation:

----------------------
Pricing Functions Not Supported in Service | SAP Help Portal

The following pricing functions are not supported in Service:
•    Tracking cumulative values with condition update
---------------------
