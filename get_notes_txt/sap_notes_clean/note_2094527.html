Symptom

  • In DBSQL 16.0 the result window is now editable in text mode.
  • DBISQL Build 16.2020 and earlier versions down to 11 and 12 where it is not editable.

Environment

  • SAP Adaptive Server Enterprise (ASE) 16.0
  • SAP DBISQL ( Database Interactive SQL- JAVA) after Build 16.2020

Reproducing the Issue

  1. Go to tools/options/sap ase
  2. Change result mode to text
  3. Run anything
  4. Observe that you can delete from the results window

Cause

This issue has been addressed in SAP CR 773094. Its description reads, " DBISQL 16.0 allows results to be editable from the results window.".

Resolution

Upgrade ASE to version 16.0 GA PL05 or higher where SAP CR 773094 is fixed.

Keywords

773094,  CR773094
