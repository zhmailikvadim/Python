Symptom

  • When using the inbox and selecting manage substitution, searching a user's full name returns no values. 
  • You can only search by first or last name, or part of the name to return any values.

Environment

  • SAP NetWeaver 7.4
  • SAP NetWeaver 7.5
  • SAP S/4HANA 1511
  • SAP S/4HANA 1610
  • SAP Fiori Launchpad

Reproducing the Issue

  1. Login to any customer FLP (with Inbox confiigured) or go to  https://www.sapfioritrial.com/sites?helpset=trial&sap-client=001
  2. Click 'My Inbox'
  3. Click the User/Guest Icon
  4. Click 'Manage My Substitutes'
  5. Click 'Add New Substitute'
  6. As an example in the trial url: Search for James Abbott - No result
  7. Search with either James or Abbott - The name James Abbot comes up

Cause

  • Known issue with missing functionality in My InBox
  • Using an older version of SAPUI5 (especially in 1.28.XX)

Resolution

  1. Update, update to the latest 'My Inbox' version(and patch) and check if issue still exists.
  2. If issue still exisits, implement SAP note 2456014 in your system and check the behavior of the user search. The note should ensure that also a selection by firstname and lastname is supported.
