Symptom

The "Landing Page Settings" menu is no longer available within Learning Administration.

Environment

SAP SuccessFactors Learning

Reproducing the Issue

  1. Log into HCM > navigate to Learning Administration >
  2. At System Administration > Application Administration >
  3. Observer that the menu 'Landing page Setting' is no longer displayed.

Resolution

The legacy Learning homepage was retired on November 14, 2025, through the 2H2025 release (when the b2511 version was delivered). As a result of this retirement, the related settings are no longer available within Learning Administration. 

Only the new custom pages built using Custom Pages are available for learners on the new Learning Home Page.
