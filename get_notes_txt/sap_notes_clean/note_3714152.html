Symptom

You execute the function "Recalculate Absences" on the detail screen of the infotype "Maternity Protection/Parental Leave" (infotype 0080) in HR Renewal (PAO application).

  • The system incorrectly recalculates only those absences whose subtypes are flagged with "Propose Absence" in the view V_T554V. It would be correct to recalculate all recorded absences.
  • The "Recalculate Absences" function is available in both create and edit mode. It would be correct that the function is only available in edit mode.
