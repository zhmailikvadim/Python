Symptom

In the Manage Launchpad Pages apps, you try to upload multiple pages from a JSON file. You receive a success message even though only one page was uploaded and created in a target system.

Other Terms

Download, upload, missing transport, F4512

Reason and Prerequisites

Due to a program error, the selected transport request is wrongly overwritten after uploading the first object. This leads to an error in the page of the file.

Solution

Apply the attached correction instructions or install the latest support package.
