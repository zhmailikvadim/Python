Symptom

This SAP Note corrects an error when simulating or processing real estate contracts.

Other Terms

Simulation or processing aborts. The signature of the error in the SLG1 log is as follows.

Dispatcher adviced to Cancel processing due to exceptions raised in parallel processes.
Dispatcher adviced to Cancel processing due to exceptions raised in parallel processes. Details in CX_SHDB_PFW_EXCEPTION->P_COLL
An exception with the type CX_SY_OPEN_SQL_DATA_ERROR was raised, but was not handled locally or declared in a RAISING clause.
'.../...' is not a valid value for C(13,0).

Reason and Prerequisites

Program error.

Solution

Apply the attached correction instruction.
