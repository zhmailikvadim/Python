Symptom

In the process "Creation of a 100% Supplier Assignment to a Feed-In Installation (DSO View)" (DE_B_MPES_OBS_FOR_DS_OBLIGATION_DSO), service provider data is not fully populated in the UBD data in step 0350.


This issue is relevant for the market role: DSO.

Other Terms

MPES, feed-in installation, supplier assignment, UBD data, service provider, EOS process, DE_B_MPES_OBS_FOR_DS_OBLIGATION_DSO

Reason and Prerequisites

The error occurred because UBD data triggered by the EOS process is structured differently than data triggered by reports.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the system correctly populates UBD data for service providers, even for incomplete entries. You can verify the fix by retesting the scenario and checking that service provider data is fully populated in the UBD data.
