Symptom

The Employee Time records are shown as updated by v4admin and the last modified date as the Period End Processing(PEP) run date.

Environment

SAP SuccessFactors Employee Central: Time Management

Reproducing the Issue

  1. Login to instance.
  2. Navigate to Manage Data-> Open Employee Time record.
  3. Check the Last modified date and Updated by user.
  4. Records are updated by v4admin on the same date as Period End Processing run.

Resolution

This is expected. During Period End Processing, when time accounts get closed the Employee Time Records should be set as non-editable. Hence, the Editable field in Employee Time object is set to No. Due to this change, the last updated by is shown as v4admin and last updated date as the Period End Processing run date.
