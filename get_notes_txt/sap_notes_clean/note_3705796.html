Symptom

  • Error occurs when attempting to access the Mitigating Controls menu.
  • Error: 500 Internal Server Error   Derefencing of the NULL reference
  • ST22: ABAP programming error with runtime error OBJECTS_OBJREF_NOT_ASSIGNED_NO and exception CX_SY_REF_IS_INITIAL.
  • Attempts to extend the organization's validity date were unsuccessful.

Environment

  • Access Risk Analysis

Reproducing the Issue

  1. Navigate to Tcode: NWBC.
  2. Select SAP_GRC_NWBC.
  3. Access Master Data.
  4. Attempt to open the Mitigating Controls menu.

Cause

  • The issue is believed to be caused by mitigation controls linked to an organization with an expired validity period.

Resolution

  1. Temporarily extend the validity date of the expired organization linked to the mitigation control.
  2. Remove the mitigation control associated with the expired organization.
  3. Revert the organization's validity date back to its original expired status.
  4. Ensure that the data in the relevant tables is valid and not expired to prevent inconsistencies.
