Symptom

You cannot execute program 'Employment Tax (IRPF) reports' (RPC190E0) for different legal persons at the same time.

Environment

  • Human Capital Management (HCM)
  • SAP R/3
  • SAP ERP Core Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA

Reproducing the Issue

Try to generate any tax report for different legal entities.

Cause

The program does actually allow the selection of different legal entities but the Spanish Tax Agency only allows one taxpayer's CIF/NIF per file, you may refer to Modelo 190 Presentación mediante fichero for details. In order to meet this requirement, form CHECK_JUPER reads table 'Company tax data' (T5E02) to check if the CIF/NIF of those legal entities is the same, rejecting that/those whose CIF/NIF is different:

*---------------------------------------------------------------------*
*       FORM CHECK_JUPER                                              *
*---------------------------------------------------------------------*
*       Rutina de validación de parámetros de selección               *
*---------------------------------------------------------------------*
FORM check_juper
     USING pv_dynpro_number TYPE syst-dynnr
  CHANGING p_forma          LIKE s_format
                          p_formcr LIKE s_formcr
                          p_dehac  LIKE s_dehac
                          p_adhac  LIKE s_adhac.


* Determinar y chequear que todas las personas jurídicas seleccionadas
* tengan el mismo NIF/CIF de empresa.
  DATA: nif_emp LIKE t5e02-finnr,
        ldknz_emp LIKE t5e01-ldknz,
        lv_juper LIKE JUPER.

  DATA: ok_juper like sy-subrc.

  ok_juper = 4.

* Do the checking only inside the opened tab to avoid locking the user
* without the possibility to change the screen
  IF pv_dynpro_number <> '1000'.
    RETURN.
  ENDIF.

* En caso de error
  LOOP AT juper.
*   NIF
*   Adapting the NIF check to reject intervals where
*   first and last legal persons have different NIF;
    lv_juper = juper-low.
    IF juper-option = 'EQ'.
      PERFORM re5e02 USING    juper-low
                     CHANGING returncode.

    ELSE.
      IF juper-option = 'BT'.
        PERFORM re5e02 USING    juper-high
                       CHANGING returncode.
        lv_juper = juper-high.
      ENDIF.
    ENDIF.
*   Si la persona jurídica no está mantenida en la T5E02
    IF NOT returncode IS INITIAL.
      MESSAGE e302 WITH 'T5E02'
                         lv_juper
                         space
                         space.
    ENDIF.
*   Si el NIF de la empresa no está mantenido en la T5E02
    IF t5e02-finnr IS INITIAL.
      MESSAGE e016 WITH text-e18 lv_juper space space.
    ENDIF.
*   Si el nif de una persona jurídica no es igual al de las anteriores
*   salvo en el caso del primer loop
    IF NOT nif_emp IS INITIAL AND nif_emp NE t5e02-finnr.
      MESSAGE e016 WITH text-e19 space space space.
    ELSE.
      nif_emp = t5e02-finnr.
    ENDIF.

    SELECT * FROM t001p WHERE juper EQ lv_juper
                          AND molga EQ pesu0_molga.
    IF NOT sy-subrc IS INITIAL.
      MESSAGE e302 WITH 'T001P'
                        lv_juper
                        space
                        space.
        exit.
      ELSE.
      PERFORM re5e01 USING    t001p-werks
                              t001p-btrtl
                     CHANGING returncode.

      IF NOT returncode IS INITIAL.
        MESSAGE e302 WITH 'T5E01'
                           t001p-werks
                           t001p-btrtl
                           space.
          exit.
        ELSE.
          IF s_format NE t5e01-ldknz.
          ELSE.
            ok_juper = 0.
            exit.
        ENDIF.
      ENDIF.
    ENDIF.
    endselect.

  ENDLOOP.

Resolution

In the case of legal entities with different CIF/NIF, it is necessary to run the program separately for each one.
