Symptom

The system stream data from Info Provider for totals records into reporting Info Provider, with reporting mode standard by default. With the SAP Note 3072115, you can customize additional reporting modes (e.g. Restatement) to be streamed to the reporting Info Provider. You cannot influence the reference year and period for the reporting mode restatment.

Reason and Prerequisites

Missing customizing for reference year and period.

Solution

Implement the attached correction instructions or import the specified Support Package.

Once the SAP Note is installed, go to the transaction UC00. Navigate to "SAP BW/4HANA Business Consolidation Add-On: Configuration Menu" -> "Reporting" and execute "Multiple Selection of Consolidation Groups in Reports".

In the "Consolidation: General Settings" add a new setting "Additional reporting modes for Reporting InfoProvider" with value for requested additional reporting mode (e.g. R as restatement). Enter the the reference year and period after the reporting mode separated by the comma. For example the entry could look like as follows:

R,2022012,2023012

It means reporting mode restatement with reference year periods 2022012 and 2023012
