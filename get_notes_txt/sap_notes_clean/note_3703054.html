Symptom

After executing report RC_COLLECT_ANALYSIS_DATA, a dump with the following header information is found in ST22:

Category               ABAP programming error
Runtime Errors         SYNTAX_ERROR
ABAP Program           /SDF/SAPLFDQ_ASSESSMENT
Application Component  SV-SMG-DVM

Syntax error in program "/SDF/SAPLFDQ_ASSESSMENT

Environment

  • SAP ERP 6.0

Reproducing the Issue

  1. Execute report RC_COLLECT_ANALYSIS_DATA.
  2. The report fails with a SYNTAX_ERROR.

Cause

The manual steps for prerequisite SAP Note 2909538 have not been completed. 

Resolution

The manual post-implementation steps below must be performed in every system that the SAP Note 2909538 is implemented or transported into. The manual steps must be performed before implementing SAP Note 2972792.

After implementing SAP Note 2909538, proceed as follows:

  1. Run transaction SA38.
  2. Enter the report name "/SDF/NOTE_2909538".
  3. Execute the report in the "Step 2: Update & Activate” mode.
