Symptom

The institute for the charge system in the hospital (InEK) has updated the 25th November  2025 of the Annex to the Agreement on the Transfer of Data According to Sec. 21 Para. 4 and Para. 5 KHEntgG on 12/15/2026.

With this SAP Note, SAP provides the adjustments specified there for the file PFLEGEPERSONAL of the program RNAP21K01 for the hospital-related data for the annual data delivery for the year 2025  according to Sec. 21 Para. 1 and the  first  mid-year data delivery for the year 2026 according to Sec. 21 Para. 3b No. 3 KHEntgG.

The first adjustments for these data deliveries were made with SAP Notes 3687954 and 3700569.

These adjustments are also part of the listed Support Packages in March 2026.

Please note:
Adjustments to the creation of the case-related data of Section 21 KHEntgG for the data year 2025 are not part of this SAP Note.

Other Terms

RNAP21K01, P21, §21, header data, PFLEGEPERSONAL.CSV

Reason and Prerequisites

Legal change

Solution

Implement the prerequisite SAP Notes and then implement the attached correction instructions in this SAP Note or import the specified support package.

After you implement the correction instructions, you can use the program RNAP21K01 to correctly create the hospital-related  data for the annual data delivery for the year 2025  according to Sec. 21 Para. 1 and the  first  mid-year data delivery for the year 2026 according to Sec. 21 Para. 3b No. 3 KHEntgG also for the file PFLEGEPERSONAL.

Note that if you change the selection screen, the saved selection variants for program RNAP21K01 may no longer be valid. If this is the case, the system issues message DB634 - Variant & of program & is obsolete when the program is executed with an existing selection variant. If necessary, you can repair the saved variants using the program RSVARDOC_NEW displayed in the long text of the message.
