Symptom

When there’s no comment for an absence, the placeholder description in Time Forms doesn’t align correctly with other absences of the same Time Type.

Environment

  • SAP SuccessFactors Employee Central
  • SAP SuccessFactors HCM Suite 

Reproducing the Issue

  1. Navigate to the user profile;
  2. Access the Time Management section;
  3. Generate Time Forms;
  4. Review unaligned comments in generated Time Forms for a specific Time Type.

Resolution

  • This is a known issue and our engineering team are currently working on a fix for. The preview environment is set for April 13, 2026, and the production environment is set for May 16, 2026;
