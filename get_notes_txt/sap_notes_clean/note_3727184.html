Symptom

You are using the Analyze Forecast app in SAP Customer Activity Repository applications bundle 5.0 FPS04, 5.0 SPS05, 5.0 SPS06, 5.0 SPS07, 5.0 SPS08, 6.0 SPS01, or 6.0 FPS02.

After the chart is displayed, you click on the View Product Details link to navigate to the Product Details page. The page loads slowly, or the data is not displayed. In some cases you get the SQL error 339: invalid number.

You have set up the Analyze Forecast app as described in the Common Installation Guide (for new installations) or the Common Upgrade Guide (for upgrade scenarios) for SAP Customer Activity Repository applications bundle 5.0 FPS04, 5.0 SPS05, 5.0 SPS06, 5.0 SPS07, 5.0 SPS08, 6.0 SPS01 or 6.0 FPS02.
