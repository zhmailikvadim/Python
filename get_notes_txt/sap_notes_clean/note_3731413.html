Symptom

SAP Note 3682030 cannot be implemented due to error related to class CL_EDOC_GCO_ECOM_GEN_INC.

The method "SET_ADDITIONAL_DATA" is not declared or inherited in class "LTD_EDOC_MESSAGE".

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

  • SAP ERP 
  • SAP S/4HANA 
  • SAP Document and Reporting Compliance, cloud edition
  • SAP S/4HANA cloud, Private edition

Reproducing the Issue

  1. Start transaction SNOTE.
  2. Attempt to implement SAP Note 3682030.
  3. Observe that activation fails with an error referring to class CL_EDOC_GCO_ECOM_GEN_INC.

Cause

The updated GCO eCommunication framework was not present; required foundation objects were missing, causing classes, methods, BAdIs and filters to be unavailable during activation.

Resolution

  1. Implement SAP Note 3616236 – Correction for Follow-Up Actions in full (including all manual and automatic corrections). 
  2. Verify the note log to ensure all steps of Note 3616236 are correctly applied and active.
  3. After Note 3616236 is active, run SNOTE again. implement SAP Note 3682030.
