Symptom

While submitting the Strategy and Plan document for approval, getting the message: “An error occurred! Submit the document for approval again. Error saving spend” and the submission fails. 

Reproducing the Issue

1.Login to CATMAN instance

2.Click on Develop Strategy and Plan >Select any category needed for approval

3.Click Submit for approval

Cause

The reason for getting this error is because there is no spend data for the selected purchasing category (or any categories at all) and the strategy has category spend profile as mandatory tool.

Resolution

In order to avoid this error, kindly follow any one of the alternatives below,

1. Select a strategy that does not require category spend profile to exist or

2. Upload spend data correctly.

Keywords
