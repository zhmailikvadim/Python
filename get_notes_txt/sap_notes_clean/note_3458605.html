Symptom

In addition to the existing CDS view /FOM/V_ARO_UNDL (DDL Source /FOM/DDL_ARO_UNDL), which for example can be used to do reporting from an ARO point of view, we have added the new CDS View /FOM/V_UNDL_ARO (DDL Source /FOM/DDL_UNDL_ARO), which for example can be used to do reporting from an underlying object point of view. 
You for example can use the new view to check, if the underlying objects are linked to an ARO.

Other Terms

ARO, Asset Retirement Obligation, Underlying Object, CDS View

Reason and Prerequisites

This note is only applicable for the SAP S/4HANA release of SAP ARO (SAP S/4HANA for asset retirement obligation / software component FOM 801).

Solution

Implement the attached correction instructions or import the relevant support package.
