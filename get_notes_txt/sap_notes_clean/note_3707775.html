Symptom

Encountering "403 forbidden" error message when calling API for CDM sourcing content.

Access Denied. Please contact your Organization Admin.

Cause

Resolution

The error is caused because of the lack of the IP address of the call in the API.

In the developer Portal navigate to the:

  1. Manage section > Search for the API Key.
  2. Access Actions > Select Allowed IPs.
  3. Insert the IP Address > Enter the IP address from your call.
  4. Save the changes.

Once you complete these steps, the API should function correctly with the following Curl command:

- `curl --request GET \`
  - `--url 'https://openapi.ariba.com/api/sourcing-cdm-content/v1/prod/cdmForJoule?realm={REALM}' \`
  - `--header 'accept: application/json' \`
  - `--header 'apikey: {API Key}' \`
  - `--header 'authorization: Bearer {Token}'`
