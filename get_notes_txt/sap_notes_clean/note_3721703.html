Symptom

The Document fails in Managed Gateway for Spend & Network(ISMG) with error "Payload received exceeded the size limit 40MB - SenderID AN11XXXXXXX..
Suggested Action : Please resend the document within the allowed size limit 40
MB size limit.ErrorCode:CIG-PLT-00658"

Resolution

The maximum size of a cXML order, ship notice, order confirmation, or invoice document is 40 MB or 10,000 line items. All other cXML documents are limited to a maximum size of 4 MB or 1,000 line items. These maximums do not include the size of any attachments.

The maximum size of an Excel file created by running an Upload/Download job for a planning process job type is 40 MB or 10,000 rows.
