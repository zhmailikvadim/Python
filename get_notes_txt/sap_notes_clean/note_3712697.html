Symptom

Create an access request. Add a role which has child role in role mapping. The child mapped role does not have provisioning allowed in one of connector but it is allowed for all other connectors.

When adding parent role to access request, its adding child mapped role as well from connector where provisioning is not allowed .

Other Terms

GRC 12.0, Role, Role mapping, Access Request, Provisioning not allowed

Reason and Prerequisites

The code was not considering provisioning settings for child mapped role.

Solution

Implement attached correction instructions.
