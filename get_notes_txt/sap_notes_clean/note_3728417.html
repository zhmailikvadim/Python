Symptom

  • When implementing a CA-GTF-CSC-EDO* SAP Note, there are syntax errors in method VALIDATE_MESSAGE.
  • The error is the following:

Method VALIDATE_MESSAGE
The identically named component "IS_PROCESS_INSTANCE_RESPONSE[]-GCOS_ECO
M_AIF_GEN_RESPONSE-PROCESS_INSTANCE-FOLLOWUP_ACTIONS" cannot be
converted to "LS_PROCESS_INSTANCE[]-GENERIC_PROCESS-PROCESS_INSTANCE-FOL
LOWUP_ACTIONS".

Environment

  • Electronic Document Processing Framework
  • SAP S/4HANA
  • SAP Document Reporting and Compliance service

Reproducing the Issue

  1. Open transaction code SNOTE.
  2. Attempt to implement the SAP Note.
  3. Observe the error with method VALIDATE_MESSAGE

Cause

Missing the implementation of SAP Note 3642502 - Integration with SAP Document and Reporting Compliance, cloud edition: Enhancements to the Integration with the cloud edition.

Resolution

  1. Implement SAP Note 3642502 - Integration with SAP Document and Reporting Compliance, cloud edition: Enhancements to the Integration with the cloud edition.
  2. After implementing the SAP Note above, resume the implementation of the SAP Note that you were initially implementing.

See Also

Refer to: SAP Note 3642502 - Integration with SAP Document and Reporting Compliance, cloud edition: Enhancements to the Integration with the cloud edition.
