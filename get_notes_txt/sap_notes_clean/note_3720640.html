Symptom

When processing deliveries, the system raises a runtime error indicating that the interface method IF_LE_SHP_TM_INT~GET_ADV_SHP_RECV_RELEVANCE_MM is not implemented. The short dump or error log points to class TH_SHP_DELIVERY_PROC_DOUBLE as the caller.

Other Terms

Syntax error, IF_LE_SHP_TM_INT~GET_ADV_SHP_RECV_RELEVANCE_MM , method implementation missing, ASR, TH_SHP_DELIVERY_PROC_DOUBLE

Reason and Prerequisites

The class TH_SHP_DELIVERY_PROC_DOUBLE implements interface IF_LE_SHP_TM_INT, but the method GET_ADV_SHP_RECV_RELEVANCE_MM was introduced in a later Logistics Execution/Transportation Management integration update.

As a result, the class does not fulfill the interface contract, causing a runtime error when the method is called.

Solution

Implement this SAP Note or the relevant feature package. Implementing this SAP Note adds the missing interface method IF_LE_SHP_TM_INT~GET_ADV_SHP_RECV_RELEVANCE_MM in class TH_SHP_DELIVERY_PROC_DOUBLE, resolving the runtime error.
