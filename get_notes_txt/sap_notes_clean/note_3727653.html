Symptom

I want to route approvals based on how users answer the questions in my wizard, but I cannot find an option to send the wizard itself through an approval group or create approval routing directly from the wizard.

Resolution

You can route approvals based on the answers in your wizard by linking the wizard to a Decision Form and using scoring to trigger conditional approval rules. 

How wizard answers can control approval routing: 

Wizard answers do not route approvals directly.
Instead, each answer contributes a numeric score, and the total score determines which approval path is used.

For example, if you assign scores such as:

  • Yes = 1
  • No = 2

And a user answers four wizard questions with: No, Yes, Yes, No, the total score is 6.

You then create an Approval Rule with a condition such as:

  • Wizard Total Score ≥ 5

After creating the rule, you assign it to a level in the Approval Group that contains the approver you want to receive requests meeting this condition.

  • If the score meets the condition, the form routes to that level’s approver.
  • If the score does not meet the condition, the system skips that level and moves to the next one (or finishes routing if no levels remain).

This makes the wizard’s answers drive which approval levels are triggered or skipped.

This example provides context for the configuration steps below.

Create questions for your wizard

  1. Click Admin Configuration (gear icon) > Question
  2. Click New and set Module to Wizard
  3. Enter a question value
  4. Set Answer Type to User Defined Pick List
  5. Add each answer option and assign a numeric Score > Click Add
  6. Repeat for all questions you want to include

Build your wizard

  1. Click Admin Configuration (gear icon) > Wizard
  2. Click New > Set Starting Point to Decision Form
  3. Click + Add Questions or Fields and select the questions you created
  4. Tick the checkboxes > Click Add Selected
  5. Configure each answer’s action depending on your workflow

Set up approval rules that react to wizard scores

  1. Click Admin Configuration (gear icon) > Rule Engine
  2. Click New > Set Type to Approval Group
  3. Set Module to Decision Form
  4. Click + Add Condition
    • Add Use Scoring
    • Add Wizard Total Score
  5. Click Add
  6. Set Use Scoring to Yes
  7. Adjust the Wizard Total Score condition (Equals, Greater Than, Greater Than or Equal To) to match the routing you need
  8. Click Add
Create the Approval Group
  1. Click Admin Configuration (gear icon) > Approval Group
  2. Click New > Set Module to Decision Form
  3. Under Configuration, select a Role > Assign the correct Approval Rule
  4. Click Add
  5. Associate appropriate business units

Create the Decision Form template

  1. Click Admin Configuration (gear icon) > Decision Form Template
  2. Click New and enter a name
  3. In Types, tick applicable procurement options
  4. In Rules, tick Use Scoring
  5. Click Add
