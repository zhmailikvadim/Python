Symptom

The transaction /SMB/CONFIG_GUIDE_UI ends with runtime error  GETVA_NOT_ASSIGNED in the method /SMB/CL_CONFIG_GUIDE_OUTPUT-GET_DDIC_LENGTH_BYTYPE.

Other Terms

 /SMB/CONFIG_GUIDE_UI_OUTPUT, GETWA_NOT_ASSIGNED, /SMB/CL_CONFIG_GUIDE_OUTPUT===CP, /SMB/CL_CONFIG_GUIDE_OUTPUT===CM00V, GET_DDIC_LENGTH_BYTYPE

Reason and Prerequisites

The root cause of this error can be a wrong definition of the relate content file.

Solution

Please check and correct the definition of the content file.

To prevent the occurrence of this runtime error, it is recommended that you install the following code correction provided by SAP. This correction will replace the runtime error with an error message.
