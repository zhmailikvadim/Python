Symptom

As described in the Regulatory Change Manager, under Changes: Taxable limits for social security and unemployment insurance 2026, this SAP Note provides you with the relevant system changes to address the new taxable limits for social security and unemployment insurance for 2026 published by Superintendencia de Pensiones (SP).

Solution

The Payroll Parameters (V_T511P) table view has been updated to reflect the annual adjustment of the taxable limits for pensions, health, work accident insurance, and unemployment insurance.
