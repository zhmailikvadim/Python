Symptom

Currently AL11 only allow to navigate between folders by double clicking them. However when the folder is too big or is a network folder it may take too long to display data.

Other Terms

AL11, Performance, Direct Access

Solution

A new option was introduced called "Direct Acess" which allow to select a direct path to a folder or file navigating directly to it. This option is available via menu Goto -> Direct Access or shortcut key Shift + F8
