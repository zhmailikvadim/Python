Symptom

UI is not responsive and displays the following message "We'll be back soon! Sorry for the inconvenience but we're performing some maintenance at the moment. We'll be back online shortly!"

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

  • SAP Sales Performance Management
  • Incentive Management

Reproducing the Issue

  1. Access the environment.
  2. Attempt to log in.
  3. Observe that the page indicates the environment is under maintenance.

Cause

Root cause to be analyzed by Support and Operation team via case ticket.

Resolution

To assist the investigation and speed up the solution, here are some key items that can greatly assist the Support team to investigate:

  • Start time of when the issue started;
  • How many users are being impacted;
  • Provide HAR traces:
    • SAP Note 3503700 - How to to collect HAR (HTTP Archive) files for support - SAP for Me
