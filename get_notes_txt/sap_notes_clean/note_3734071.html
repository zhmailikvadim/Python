Symptom

When performing a Form-Fit-Function (FFF) class component exchange, the system does not validate the component status before
executing the exchange. Components that have already been assembled, have pending goods movements, posted GI quantities, completed
operation activities, open defects, collected IC data, active disassembly/reassembly records, or serial number groups can be
exchanged without restriction.

Other Terms

FFF, Form Fit Function, component exchange, assembly

Reason and Prerequisites

Program Error.

Solution

Apply Correction Instruction.
