Symptom

In some circumstances RFBILA00 and RFBILA00N are used in the context of other solutions. The programs are called and the data is processed in the BADI FI_BILA_OUTPUT. This note creates an additional hidden parameter for RFBILA00 and RFBILA00N in which the calling application can specify which country should be used as a filter value for the BADI in case no (unique) country can be derived from the involved company codes.

Solution

Implement Support Package or correction instructions.
