Symptom

  • The option to Allow Worker to Edit Cost Allocation field is not available for rate schedules with service types Services and Temp & Services
  • I am using Rate Schedule 2.0

Resolution

The rule Allow Worker to Edit Cost Allocation is not supported on Rates 2.0 enabled Rate Schedule for service type Services and Contingent and Services. This rule is only applicable for rate schedules with service type Contingent.
