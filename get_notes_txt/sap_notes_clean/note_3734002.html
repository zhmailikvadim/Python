Symptom

Environment

SAP S/4HANA 2025

Reproducing the Issue

In the Fiori App F1506A “My Time Events” Open the “Detailed Entry” tab and try to select a date. The drop-down list for date selection closes immediately after clicking, and no date can be selected.

Cause

Code error that we already resolved 

Resolution
