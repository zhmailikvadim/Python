Symptom

When uploading CP22 .txt file to the Tax Portal, it is displayed the error "Format error on field: Current Correspondence Address indicator". "Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental." 

Environment

  • Human Capital Management (HCM)
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP SuccessFactors Employee Central Payroll
  • SAP S/4 HANA

Reproducing the Issue

  1. Run T-code PC00_M14_CT22 or execute program RPCT22L0 via T-code SE38.
  2. Fill in all mandatory fields, then execute.
  3. Upload the downloaded file to the tax portal. An error occurred during the upload process.

Cause

Missing configuration in CP22 report.

Resolution

Please follow the rules below when filling out the output format for the CP22 report

  • "Date of Joining Between....." is a mandatory field.
  • When “Download File” checkbox is selected , “Correspond Address Indicator “ is a mandatory field.
  • For "Correspondent Address indicator":
    • If the "Correspondence Address indicator" is set to 1, the "Present Correspondence Address" may be left blank and its value is ignored.
    • If the "Correspondence Address indicator" is set to 2 or 3, the "Present Correspondence Address" is a mandatory field.
  • When “Download File” checkbox is selected and infotype 0023 (Other/Previous Employers) exists, the "Previous Employer Address" is a mandatory field.
