Symptom

You double click a transaction in the SAP menu. This message is raised in the status line: 
Transaction  does not exist
There exists a row in table TMENU01R for the corresponding NODE_ID. The transaction code exists, if you display the transaction in transaction SE93. If you enter the transaction code in the ok code field, the transaction is executed successfully.
The transaction cannot be found in the menu, if you try to find it with icon Find  (Ctrl+F), with using option In technical Name.

Environment

This KBA applies for all SAP Products based on NetWeaver and S4H

Reproducing the Issue

Execute a transaction in the SAP menu.

Resolution

  1. Execute transaction SE43
  2. Enter the corresponding area menu, where the issue appears
  3. Navigate over the menu: Edit -> Refresh Buffer
  4. Updating buffer for area menu .... appears in the status line
  5. Retest the behavior
