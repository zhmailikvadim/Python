Symptom

A blank screen shows when attempting to view a form from the mobile client. After doing a clean reset / initial sync, this blank screen persists.

Other Terms

SAP Asset Manager

SAP Service and Asset Manager

SAM

SSAM

SDF

SAP Dynamic Forms

ICMTANGOAMF10-40314

Reason and Prerequisites

Basic understanding of MDK based applications

Solution

unzip 3714352.patch.zip temporarily to the SAPAssetManager metadata folder of your application, creating 3714352.patch

Open a terminal at this location and apply this patch with

patch -p1 < 3714352.patch

This will modify

  • Rules/Common/CompleteReset.js
  • Rules/Forms/SDF/InitialTransmit.js

After the patch is applied, delete the temporary 3714352.patch file.

Finally, deploy your new metadata to apply these changes.
