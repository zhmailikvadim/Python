Symptom

There is a requirement to configure or maintain values for these fields Customer Condition Group 1 and Condition Group 2 in Customer Master Data.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Go to Manage Customer Master Data App.
  2. Open the Business Partner (Customer).
  3. Navigate to Additional Information.

Cause

Resolution

Maintain the required values for Condition Group 1 and Condition Group 2 using configuration activity 103530 – Define Condition Groups. 

Follow below steps for maintaining customer Condition group in Manage Your Solution' app:

  1. Open 'Manage Your Solution' app.
  2. Navigate to 'Configure Your Solution'.
  3. Choose configuration activity '103530 - Define Condition Groups'.
  4. Display & select the relevant project.
  5. Maintain the Values for Condition Groups.

Incase while using SAP Central Business Configuration (CBC) :

  1. Log on to the project experience in SAP Central Business Configuration.
  2. Select the relevant project.
  3. Navigate to the configurations activity tab. 
  4. Search for the configuration activity '103530 - Define Condition Groups'.
  5. Select the configuration activity to navigate directly to the SAP S/4HANA Cloud system.
  6. Display & select the relevant project.
  7. Maintain the Values for Condition Groups.
