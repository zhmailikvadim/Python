Symptom

While in the LMS UI and attempting to create a new external site, receive an error message with generic ticket when clicking submit.

Environment

SAP SuccessFactors Learning

Reproducing the Issue

PreRequisites: IAS for Learning-only users enabled & active AND Cloud Identity > Real-time sync enabled

  1. Navigate to Learning Administration > System Administration > Application Administration > Sites & Click Add New
  2. Fill out required fields (ensuring selection are valid & active) & Click Submit - Incur Error.

Cause

With real-time sync enabled, LMS will have the IAS & IPS try to create the site within those systems and certain settings may be preventing to do so.

Resolution

  1. Ensure all configuration authorizations are enabled for the "SAP LMS - <tenant>" admin user on the IAS, see KBA 3248892  
  2. Ensure the latest certificate for this admin was updated, see KBA 3247849
    *If not using converged but rather NEO IAS/IPS ensure the IPS certificate is also updated

See Also

  • 3248892 -  "A new Learning-only user could not be created" - Learning IAS Real Time User Sync
  • 3247849 - Learning IPS - mTLS Certificate Expiration [2024]
