Symptom

  • Fields in Employment Details are not visible in the Latest People Profile despite the required permissions being granted. Instead, message "There’s nothing to show yet" is being displayed;
  • The issue is specific to the Latest People Profile; Employment Details are displayed correctly in the Legacy People Profile.

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental." 

Environment

SAP SuccessFactors Employee Central

Reproducing the Issue

  1. Access People Profile using Latest People Profile;
  2. Navigate to the Job Data Category;
  3. Attempt to view the Employment Details card;
  4. Observe that the fields in Employment Details are not visible and the message "There’s nothing to show yet" is being displayed instead.

Cause

Only Termination Details fields' permissions have been granted.

In Latest People Profile, it is not enough to have permission for Employment Details MSS to view the fields. You need to grant permission for each field.

Resolution

Give permission for at least one field from Employment Details section (besides of Employment Details MSS).

To confirm which fields are part of each section, please refer to Employment Information | SAP Help Portal and Termination Information | SAP Help Portal.
