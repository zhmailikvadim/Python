Symptom

You create wage statements for employees as a form in accordance with the guidelines of the wage standard ELM as of Version 4.0. To do this, execute the program Wage Statement 2005 (CE Version) (report: RPLLAWC2_CE, transaction: PC00_M02_LLAW2_CE) or Wage Statement 2005 (report: RPLLAWC2, transaction: PC00_M02_LLAW2).

After you implement the corrections from SAP Note 3706563 (ERC2: Improvements to employer statement), the place of issue is no longer printed on the generated document.

Other Terms

 Wage statement, LAW, ELM, location, CL_HRPAYCH_ADDRESS, CL_HRPAYCH_ERC

Reason and Prerequisites

For the employer statement valid as of 2026, the address retrieval of the employer location has been modified. As a result, the place of issue in the wage statement is no longer output correctly on the form.

Solution

Import the HR Support Package relevant for your release. If you have to implement the solution in advance, use the SAP Note Assistant to implement the correction instructions relevant for your release. You can find more information about the SAP Note Assistant on SAP Support Portal at https://support.sap.com/notes.

After you implement this SAP Note, the place of issue in the document is output correctly again.
