Suddenly, SC does not display any information, which was accessible a few minutes before

At AS log in debug level appears:

DBG> Search, User/12344c88149e6a49336fa1c0c7799/09CC6931 DOES NOT have access to VoiceMailService/...

Environment

SAP Contact Center 7.0

Reproducing the Issue

  1. Install new SAP CCtr system
  2. Restore Database backup from a working SAP CCtr system
  3. IA works fine
  4. SC display information without any issue
  5. After some minutes, SC does not display information anymore

Cause

Incorrect configuration at workstation which has started SC

Resolution

  1. SC uses certificate authentication as the primary method and userid+pwd as secondary.
  2. Turn SSO off for Admins:
  3.      uncheck “use Client Certificate (CoS) for Client Authentication” from Admin COS Virtual Unit
