Symptom

  • During the implementation of CA-GTF-CSC-EDO-FR SAP Notes such as 3481252, the following syntax errors are occurring:.
    • "Type 'CL_EDOCUMENT_FR_INV' is unknown"
    • "Type 'EDOC_FR_INV' is unknown"

Environment

  • France Electronic Invoicing
  • SAP ERP
  • SAP S/4HANA
  • Document Reporting and Compliance service

Reproducing the Issue

  1. Implement a SAP Note.
  2. During implementation/activation, observe syntax error showing the missing objects CL_EDOCUMENT_FR_INV and EDOC_FR_INV.

Cause

Missing implementation of SAP Note 3413564.

Resolution

  1. Review and implement SAP Note 3413564 - Electronic Document Processing for France via Approved Platform: Implementation to deliver class CL_EDOCUMENT_FR_INV and type EDOC_FR_INV.
  2. Activate the corrected objects and verify that no syntax errors remain.

See Also

  • Refer to: SAP Note 3413564 - Electronic Document Processing for France via Approved Platform: Implementation
  • Refer to: SAP Note 3481252 - Electronic Reporting for France: Implementation
