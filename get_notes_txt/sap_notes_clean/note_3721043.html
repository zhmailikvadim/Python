Symptom

The error message "Item is inconsistent/missing in /SCWM/QUAN; GUID_PARENT &1, GUID_STOCK &2" (Message ID /SCWM/UI_DIFF 055) occurs when posting differences in Analyze Differences app.  

In the meanwhile, the following stuck queue message for the physical inventory document appears in Warehouse Monitor app:

Item *: Enter quantity or set ISSUE_ALL

Environment

SAP S/4HANA Cloud Public Edition

Cause

There is a database inconsistency in /SCWM/QUAN table combined with the physical inventory items.

Resolution

The inconsistency can only be corrected by SAP Support at backend.

Create a case to SAP under the component SCM-EWM-PI-2CL to correct the inconsistency.

Keywords

item inconsistent, missing item, difference analyzer error, physical inventory document, SMQ2 error, warehouse monitor, enter quantity, set ISSUE_ALL, inventory processing error.
