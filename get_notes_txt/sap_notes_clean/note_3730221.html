Symptom

  • After selecting the Add custom slots button, focus does not land on the combo box options for the From and To fields  in the Interview scheduling page when using VoiceOver on iOS devices.
  • Expected behavior: users should be able to interact with and select options from the combo boxes.

***Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.***

Environment

SAP SuccessFactors Recruiting Management

Reproducing the Issue

  1. Recruiting>Navigate to Interview scheduling and select any candidate to schedule.
  2. Select Add custom slots to open the dialog.
  3. Attempt to open and navigate the From and To combo boxes using VoiceOver on an iOS device 
  4. Observe that focus does not land on the drop-down options, preventing interaction.
