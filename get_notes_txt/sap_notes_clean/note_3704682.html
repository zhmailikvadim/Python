Symptom

  • Onboarding document doesn't generate for the onboardee.
  • Placeholder mapping error observed in the Form Id: ABC.
  • Error in Business Process:
    • FormId: ABC, Locale: XXX, User: XXX, AsOfDate: XXX, FormTemplateCode: XXX, TemplateMappingId: XXX
    • Error message: "Error while resolving token XXX for DIRECT BINDING [key=EMPLOYEE_HRIS_GO_INVALID_FIELD_OR_ASSOCIATION, values=[toDataCollectionUserConfig.cust_address.cust_ContactAddress, ONB2Process]] Messages:[key=EMPLOYEE_HRIS_GO_INVALID_FIELD_OR_ASSOCIATION, values=[toDataCollectionUserConfig.cust_address.cust_ContactAddress, ONB2Process]]"

Environment

SAP SuccessFactors Onboarding

Reproducing the Issue

  1. Navigate to Business Process under To-Do
  2. Find the document mapping error.

Cause

  • Placeholder paths in the mapping are not maintained.
  • Associations for cust_address under Object Definition: Onboarding data collection user configuration are inactive.

Resolution

  1. Navigate to Configure Object Definition > Object Definition: Onboarding data collection user configuration.
  2. Scroll down to Associations and find the cust_address.
  3. Click details.
  4. Confirm whether the status of the cust_address association is active.
  5. If not, activate it.
  6. Go to Maintain Onboarding and Offboarding Document Templates.
  7. Find the affected Form Id: ABC and open the mapping.
  8. Configure the missing placeholder paths.
  9. Select the appropriate object.
  10. Restart the Business Process and trigger the document generation again.
