Symptom

While performing Check action in Third-Party Inventory Notification, the error occurs: Item YY: Transfer group ID permitted for transfer only (YY is Item ID).

Environment

SAP Business ByDesign

Reproducing the Issue

  1. Go to Third-Party Logistics work center.
  2. Go to 3PL Inventory Management view.
  3. Select Inventory Notifications sub-view.
  4. Search for Inventory Notification document ID: 'ABC'.
  5. Choose Edit.
  6. Observe that Notification Status is Inconsistent.
  7. Choose Check button.
  8. Error message appears: Item YY: Transfer group ID permitted for transfer only.

(ABC represent Inventory Notification ID)

Cause

  • If the inventory notification is created with the movement direction as Inventory Receipt and the reason as Inventory Difference, the Item Group ID is not required.
  • The Item Group ID is required only when an inventory notification is created with the reason as Transfer.

(In the error message, the Transfer Group ID refers to the Item Group ID)

Resolution

To avoid the error, remove the Item Group ID as follows,

  1. Go to Third-Party Logistics work center.
  2. Go to 3PL Inventory Management view.
  3. Select Inventory Notifications sub-view.
  4. Search for Inventory Notification document ID: 'ABC'.
  5. Choose Edit.
  6. In line items tab, remove the data maintained in Item Group ID.
  7. Choose Check button.
  8. The Inventory Notification becomes Consistent.

(ABC represent Inventory Notification ID)

Keywords
