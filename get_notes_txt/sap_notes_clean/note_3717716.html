Symptom

In the scenario of the re-determination without certain substitutes introduced with note 3658178 and the scenario with reduced request introduced with note 3680716 an error can occur in the Review Availability Check Result Screen (RACR) when changing the confirmations of a requirement.

Other Terms

aATP; ABC; RACR

Reason and Prerequisites

The reason is that the Alternative Determination UUID is not always filled.

Solution

Apply the correction instructions for this note or upgrade to an SAP S/4HANA support package that contains this correction.
