Symptom

************************************************************

WARNING: This is a CA-FL (Field Logistics) specific note. If you DON'T have

Field Logistics activated in your system, this note does not apply to you.

************************************************************

The Field Logistics BAdI (FLOG_CHG_SUPPLY_PROCESS) includes the method IF_FLOG_CHG_SUPPLY_PROCESS-DETERMINE_SUPPLY_TYPE. This method influences the supply process based on the Supply Type configured in the Customizing settings. Currently, an incorrect supply process is being determined when the relevant operation details are passed to the BAdI implementation from  the Maintenance Order.

Other Terms

CA-FL, CA-FL-GR, Field Logistics, Supply Process, Maintence Order, Supply Type

SAP S/4HANA OP 2023 FPS0, FPS1, FPS2, FPS3,SPS4

Reason and Prerequisites

  Program Error.

Solution

Kindly implement the attached correction instructions or install the corresponding support package.
