Symptom

When you declare a DSN for an employee, the system does not generate an error message if the total number of working days that you declare in all blocks 53 (Type 01, Unit 40) for a month exceeds the number of calendar days in that month.

Other Terms

ACTI (payroll cluster), REMU, block 53

Reason and Prerequisites

Reason for triggering the error message: The payroll cluster ACTI contains inconsistencies.

Solution

After implementing this SAP Note, the system generates an error message if the total number of working days that you declare in all blocks 53 (Type 01, Unit 40) for a month exceeds the number of calendar days in that month.
There are a few exceptions for which it is acceptable to have the number of working days exceed the number of days in the month. The system lists them in the long text of the error message.

The correction included within this SAP Note is part of the HR Support Packages referenced below.
If you need the correction before the HRSP is available, implement the attached correction instructions. Note that correction instructions are a short-term solution and not a substitute for the Support Package.
