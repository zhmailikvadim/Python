Symptom

  • SQL statement for database object /SAPSLL/LCLICI was executed with an error
  • Unique constraint violated: Table(/SAPSLL/LICHDTN), Index(/SAPSLL/LICHDTN~SK) 
  • SQL Error Code: 301

Environment

  • Global Trade Services (GTS)
  • Global Trade Services / Logistics Services (SLL)
  • SAP Global Trade Services
  • SAP Global Trade Services, Edition for Hana

Reproducing the Issue

  1. Run the conversion to GTS Edition for SAP Hana 2023 and the data conversion task INS_LCLICI_2_LICHDTN  was unsuccessful.

Cause

  • Duplicate entries in the table /SAPSLL/LCLICI referencing the same legal control master record but with different GUIDs.

Resolution

1. Execute the following SQL statement in the SQL editor of transaction DBACOKPIT to identify the affected records:

select lclici.mandt, lclici.guid_lclic, ctsnum.ccngn, ctsnum.stcts from "/SAPSLL/LCLICI" as lclici
       inner join "/SAPSLL/CTSNUM" as ctsnum
         on lclici.mandt    = ctsnum.mandt
        and lclici.guid_cts = ctsnum.guid_ctsnum
      group by lclici.mandt, lclici.guid_lclic, ctsnum.ccngn, ctsnum.stcts
      having count( * ) > 1

2. Delete one of the duplicate records from the table /SAPSLL/LCLICI.

3. Repeat the XPRA phase of the upgrade process.

Note: If upgrading to GTS Edition for SAP HANA 2023 SP 5 or lower, please also implement the notes below during the appropriate SUM phase of the next upgrade:

  • 3606717 - XPRA: Data conversion task INS_LCLICI_2_LICHDTN was not executed successfully
  • 3656135 - XPRA: Data conversion task INS_LCLICI_2_LICHDTN was not executed successfully
