Symptom

Why is the supplier unable to see the Create Correction Service Sheet option for a Service Entry Sheet (SES)?

Cause

The transaction rule Allow suppliers to create correction service sheets is not enabled in the buyer’s SAP Business Network account.

Resolution

Enable the transaction rule Allow suppliers to create correction service sheets in the appropriate set

  • If the supplier is part of a Supplier Group, enable the rule in Supplier Group Transaction Rules

  • If the supplier is not part of any Supplier group, enable the rule in Default Transaction Rules

Once this rule is enabled, the supplier will see the Create Correction Service Sheet option under the SES and can create a correction SES accordingly.
