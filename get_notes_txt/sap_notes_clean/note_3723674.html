Symptom

The Storyboard is used to develop applications using graphical editors and wizards. The process fails and there is a need for more detailed traces for further investigation. 

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental." 

Environment

SAP Business Application Studio

Resolution

In SAP Business Application Studio:

  1. Open "Settings":



  2. Search for "Storyboard" settings, and update the Service Center logging level configuration to "debug":



  3. From the main menu, select View -> Output to open the "Output" view:



  4. Select the "vscode-wing-storyboard" output from the dropdown to view the storyboard logs:



  5. Go to the Storyboard and reproduce the issue

  6. If required by SAP Support, copy the output and attach it to the SAP case.

  7. Set the log level back to "warn" (default):
