Symptom

You are unable to perform the Manual Posting - Partial Transfer of Individual Material due to the error Posting not possible; acq. cost of indiv. mat. less than transf. amnt.

Environment

SAP Business ByDesign.

Reproducing the Issue

  1. Go to the Fixed Assets work center.
  2. Under Fixed Assets select Fixed Assets View.
  3. Select the Asset > Manual Posting > Partial Transfer of Individual Material.
  4. Fill all mandatory fields and click post.

Error message: Posting not possible; acq. cost of indiv. mat. less than transf. amnt.

Cause

This error appears when the book value of the asset is less than the amount that you entered for the partial transfer as on the date of your partial transfer amount.

Resolution

For instance if you have selected a posting date of DDMMYY, you need to check on this particular date what is the net value of the asset and based on that you need to adjust the posting date in the Manual Posting - Partial Transfer of Individual Material accordingly.

  1. Go to the Fixed Assets work center.
  2. Reports > Lists > Fixed Assets - Line Items.
  3. Fill the required details for your source assets and click Go.
  4. To view the Results > Show Settings > Characteristics and For Characteristic Settings > Select: Fixed Asset Class for Result Row, select > Show.

With this you can see the type of the transaction posting and the balance of the asset.

For example: on DDMMYY due to the business transaction type <Transfer of Current-Year Acq Costs> the result is zero but on the date DD1MMYY due to the business transaction type <Goods Receipt from Supplier> there is some balance which has come into the asset.

The system gives the error if the posting date is DDMMYY because of the zero net book value and if the posting date is DD1MMYY the system does not give the error because there is some book value which can be used for the Manual Posting - Partial Transfer of Individual Material.

Keywords
