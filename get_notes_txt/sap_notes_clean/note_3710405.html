Symptom

The "Discover" option for the API is inaccessible and error looks like - The issue is with the backend service. Contact the backend service owner. [Request ID: XXXXXXX]

Environment

  • SAP BTP Integration Suite
  • SAP BTP APIM

Reproducing the Issue

  1. Attempt to access the "Discover" option within the API portal UI
  2. Observe the error message -The issue is with the backend service. Contact the backend service owner.
    [Request ID: XXXXXXX]

Cause

This is a limitation and Discovery functionality is not supported for all kinds of API Provider.

Resolution

Note that the "Discover" functionality is not supported in API Management (APIM). Provider discovery works for CPI and any other target system exposed via SAP Gateway.
