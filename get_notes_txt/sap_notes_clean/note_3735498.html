Symptom

If IHB notifications are configured to be sent via e-mail, the attachment type is always set to PDF or XML.

It is not possible for the customer to send attachments in a different format.

Other Terms

/PF1/CL_OPM_OUTPUT_STREAM_MAIL

Reason and Prerequisites

In the output stream class, the attachment type is hardcoded (PDF or XML).

Solution

Code correction.

The attachment type is determined from the object list field REF_ANY_1 if it is maintained. Otherwise, the default value XML is used.

Please apply SAP Note or assigned Support Package.
