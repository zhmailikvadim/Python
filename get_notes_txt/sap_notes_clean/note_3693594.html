Symptom

While submitting a new Poland FA3 schema invoice, the error message "Attributes of interface PL_KSEF_SEND_REQUEST are not maintained" is displayed.

Environment

  • SAP ERP
  • SAP S/4HANA
  • SAP Document and Reporting Compliance Service

Reproducing the Issue

  1. Access transaction code EDOC_COCKPIT.
  2. Attempt to submit a Poland FA3 schema invoice.
  3. Observe the error message: "Attributes of interface PL_KSEF_SEND_REQUEST are not maintained".

Cause

  • The system attempts to fetch the record for the interface ID with version 0002, but the entry is maintained with table EDOINTDRCATTRDET is version 0003.

Resolution

  1. Maintain the Interface version 3 with interface ID PL_KSEF_SEND_REQUEST in SM30 view EDOINTVERSIONV.
  2. Refer to the SAP Help Portal for further details: Maintaining eDocument Interfaces Version | SAP Help Portal 
  3. Ensure the following note 3676968 - Electronic Document Processing for Poland - KSeF: FA (3) Schema - SAP for Me is implemented and UDO Report NOTE_3676968 is executed.

Note: Before implementing above note 3676968, ensure pre-requisites notes are implemented first. For a complete prerequisite list, refer to SAP Note 3279206 Electronic Document Processing for Poland - KSeF: Implementation Overview.
