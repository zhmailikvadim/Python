Symptom

 When we try to download the Excel files, then in the downloaded file author name is showing as ‘SAP WebAS’ instead of the Excel owner’s name.

Other Terms

 Author name, SAP WebAS, Excel file download, Excel owner name, /VTA/CL_BFTP_SERVICES, EXCEL_XSTRING_PREPARE

Reason and Prerequisites

 This is caused by Program error.

SAP Note#3724104 is available for V/4 release.

Solution

Please implement the correction instructions attached. These corrections are only valid if you have applied the Support Packs.

Release 6.4 – Base Package & Support Package 1
Release 6.3 – Base Package

1. Perform the manual pre-implementation steps that are described below
2. Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object
