Symptom

Translation functionality is not available for Data Analyzer content.

While the system displays the message "Translation settings can only be changed when the file is open" (similar to Stories), the Enable Translation option remains unavailable, and no translation settings can be found even after opening the Data Analyzer file.

Environment

SAP CLOUD GENERIC

Reproducing the Issue

  1. Create a Data Analyzer (Insight) object.
  2. In My Files, select the created Data Analyzer object and click Edit Details.
  3. Go to Translation and check the Enable translation option.
  4. The Enable translation button is disabled and cannot be clicked.

Cause

Data Analyzer content is not part of the list of translatable objects in SAP Analytics Cloud.

Translation features are only supported for specific object types such as Stories, Models, Public Dimensions, and Analytic Applications. Therefore, the observed behavior is expected by design.

Resolution

The translation feature cannot be enabled for DAN.

If translation is required, it is recommended to use supported objects (e.g., Stories or Analytic Applications)

Please feel free to initiate an enhancement request for this specific feature as outlined in the KBA 0002424376 - How do you create an enhancement request and provide feedback for SAP Analytics Cloud? accessible on the SAP ONE Support Launchpad. 

You can find more information how  the « Idea Place" is working in the KBA 0001515837
