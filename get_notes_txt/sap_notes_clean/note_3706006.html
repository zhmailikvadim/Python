Symptom

You wish to configure deduction and counting rules for cross midnight shifts

Environment

  • SAP ERP
  • Release independent

Cause

configuration

Resolution

  1. The IMG path for this configuration is 
    Time Management > Time Data Recording and Administration > Managing Time Accounts Using Attendance/Absence Quotas > Quota Deduction Using Attendances/Absences
  2. Cross midnight shifts will use the configuration set for the day where the shift started. This means that a shift starting on Monday and ending on Tuesday will use the Monday setup.

Keywords

configuration, T554C, attendance, absence
