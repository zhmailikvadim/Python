Symptom

When directly attempting to select and delete a sub-node under a model in BD64, an error message is displayed stating "Action cannot be carried out on the nodes selected". Additionally, when selecting a sub-node and pressing F2, the message "No details available for the selected nodes" is shown.

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental. 

Environment

  • ABAP Platform
  • SAP NetWeaver
  • SAP Web Application Server for SAP S/4 HANA

Resolution

Using the "TEST" model as an example:

        1.First, click the edit button and switch to edit mode. Then, expand the sub-node that needs to be deleted. In this example, the sub-node is "APOCLNT801".

2. Delete all maintained Message Types and BAPIs within the sub-node. In this example, remove the entries "ACLPAY" and "DIRDEB."

        3. After deletion, click the save button, then re-enter Tcode BD64. Then the sub-node APOCLNT801 is deleted.
