Symptom

  • The app "Post Cash Journal Entries (FBCJ)" in SAP S/4HANA Cloud Public Edition does not support segregation of access for "Save" and "Post" actions.
  • Require separate access authorizations for Save and Post actions.

Environment

SAP S/4HANA Cloud Public Edition.

Reproducing the Issue

  1. Access to Post Cash Journal Entries app with one of the required business roles SAP_BR_CASH_SPECIALIST, SAP_BR_EXTERNAL_AUDITOR or SAP_BR_GL_ACCOUNTANT.
  2. Observe that the same authorized user can both save and post the cash journal entry.

Cause

The authorization to perform Post and Save actions in Post Cash Journal Entries app is deemed as Write access. A check to distinguish these actions is not supported in SAP S/4HANA Cloud Public Edition.

Resolution

It is confirmed that currently system does not support segregating access authorizations for Save and Post actions in Cloud Public Edition.

Users are encouraged to submit a Feature request through the Customer Influence website.
