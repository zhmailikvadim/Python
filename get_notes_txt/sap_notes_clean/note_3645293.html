Symptom

You use SAP S/4HANA for manufacturing logistics. You want to set a user status for a tour in EWM Monitor. If you do not refresh data of this tour before changing user status, standard tour status values can be reset.

Other Terms

Manufacturing Logistics, Tour status, User status, EWM Monitor

Reason and Prerequisites

You use SAP S/4HANA for manufacturing logistics or this SAP Note is a prerequisite of another SAP Note.

Solution

Implement the attached correction.
