Symptom

This note refines the note 3647281 for complex document flows. 
In case there is a predecessor document for the returns process and there are more than one returns order in the document flow of that predecessor document, it can happen the rejection reason is still greyed out even though the returns delivery item of the relevant returns order item has no more quantity.

Other Terms

ABGRU, field selection, VBELV

Reason and Prerequisites

Prerequisite: more than one returns order in the same document flow

Solution

Implement the correction instruction.
