Symptom

The following link points to a documentation about enhancing the authorizations of user TMSADM:
https://help.sap.com/saphelp_autoid70/helpdata/en/ee/aea9186d9e11d2b41a006094b9ea64/content.htm?no_cache=true .

Other Terms

TMS Trusted Services, S_CTS_ADMI, authorization object

Reason and Prerequisites

Documentation is wrong.

Solution

The documentation about enhancing the authorizations of user TMSADM is wrong. The authorizations of user TMSADM must not be enhanced.
For technical reasons, it is not possible to delete this documentation.

This note invalidates the documentation. Please ignore the documentation.
