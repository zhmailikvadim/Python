Symptom

The sort option "Keep Left Columns Grouped" is grayed out in aggregated Table widget Story report.

The sort option "Keep Top Rows Groups" is grayed out in non-aggregated Table widget Story report.

User wants to know if it is possible to disable/uncheck the sort option "Keep Left Columns Grouped" or "Keep Top Rows Groups" in Table widget in Story report.

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental. 

Environment

SAP SuccessFactors HXM Core 2211
SAP SuccessFactors People Analytics

Reproducing the Issue

  1. Go to Report Center
  2. Edit the Story report
  3. Click on one of the columns in aggregated Table widget
  4. Right click> Sort Options> "Keep Left Columns Grouped" is grayed out
  5. Click on one of the columns in non-aggregated Table widget
  6. Right click> Sort Options> "Keep Top Rows Grouped" is grayed out

Resolution

“Keep Left Columns Grouped” is grayed out in aggregated Table widget as it is always grouped there, so it is not possible to untick. But for non-aggregated Table widget, “Keep Top Rows Grouped” option should not be there. These options will be removed in the next releases.
