Symptom

When submitting or checking an installation reply for Process Scenario 13( Replace Meter and SMGW ), the system raises the error message "You must not connect any devices of type Meter for process scenario 13". This message is wrong. Scenario 13 allows any number of connected meters, so long as there is at least one removed and installed meter as well. The additional connected meters can occur in a complex IMS, where one SMGW is connected to several meters. Those meters that are not replaced have to be added as "connect" in the installation reply to re-connect them to the new SMGW.

Other Terms

Installation Reply, Process Scenario 13, Device Change, DE_B_IM_DEV_CHANGE

Reason and Prerequisites

Relevant for market role: MOS

Solution

Please implement the correction instruction.
