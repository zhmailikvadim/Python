Symptom

ABAP Dictionary mass activation during upgrade / SP-update via SUM (phase MAIN_SHDRUN/ACT_UPG ) uses only a maximum of 6 parallel processes.

Other Terms

parallelization ACT_UPG

Reason and Prerequisites

The parallelization can be influenced by entries in table DDPAMSCTRL. The entry with SCENARIO = A is relevant for the upgrade only and is configured to use 6 parallel processes only.

See SAP Note 3048182 for parallelization parameters in standard scenario (SCENARIO = P) .

Solution

The new parametrization parameter will be deliverd with SP and via upcoming DDIC Fixbuffer which will be included in upgrade procedure automatically. In the meantime, please adjust the entry in DDPAMSCTRL in the shadow system via transaction SE16. The value of field MAXTASKS for scenario A should be set to 12.
