Symptom

This note corrects the following issue related to the Personal Information (Fiori 2.0) app 'My Addresses' for country version Belgium ('12'):

An error with message "In the context of Data Services an unknown internal server error occurred" occurs when the search help assigned to field Postal Code (PSTLZ) is accessed.

The error is related to properties LockIndicator and IsSearchhelp being defined of type Edm.String instead of Edm.Boolean.

Other Terms

HCMFAB_MYADDRESSES, ValueHelp_12PostalCode

Solution

Import the Support Package attached to this SAP Note or implement the corresponding correction instruction using the Note Assistant.
