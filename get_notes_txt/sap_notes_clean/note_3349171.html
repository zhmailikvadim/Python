Symptom

SAP Service & Asset Manager's allows the user to configure the delta sync to be periodically triggered using the paramterer 'PERIODIC'. This however is not triggered unless the threshold time period has passed. (THRESHOLD_PERIOD) Applying this note will make sure a delta sync is triggered every time period specified by the parameter 'PERIOD'.

Other Terms

  • SSAM
  • PERIODIC SYNC
  • THRESHOLD_PERIOD
  • DELTA SYNC
  • AUTO SYNC
  • AUTO_SYNC

Reason and Prerequisites

  • Access to WebIDE/BAS or Visual Studio code with current application definitiions
  • Can deploy application changes

Solution

    1. Download and unzip 3348832.zip
    2. Move AutoSyncLibrary.js to Rules/ApplicationEvents/AutoSync
    3. Overwrite the original file
    4. Follow the normal deployment process
