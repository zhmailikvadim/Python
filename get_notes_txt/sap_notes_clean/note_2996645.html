Symptom

Symptom

When you create a home delivery round using transaction JV04, the system does not create the associated delivery round.

Other Terms

Entry created in delivery round business partner (JVTBEZGP), but associated entry in delivery round (JVTBEZRND) not created, JV333

Reason and Prerequisites

Program error

Solution

Implement the attached correction instructions or import the specified support package.

For this scenario, the user must select communication group(s) from the list page or object page and perform relevant actions. After performing the actions, standard toast messages  such as Object Created, Do you want to delete object? were displayed

Other Terms

Manage Communication Group, JIT, Toast Message, JIT, Just-In-Time

Reason and Prerequisites

With the changes in the respective application, the framework related toast messages are displaying, currently

Program error.

Solution

Implement the correction instruction attached with the note.

After implementing this SAP note, the following toast messages display during create/delete:

  • Delete communication group (&)?
  • Communication group deleted
  • Communication group created
