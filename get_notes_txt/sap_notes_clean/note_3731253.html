Symptom

There is a syntax error dump in SADL related internal classes (CL_SADL_ABQI is an example):

Category               ABAP programming error
Runtime Errors         SYNTAX_ERROR
ABAP: Program          CL_SADL_ABQI==================CP
Application Component  BC-ESI-ESF-BSA

The gateway client also shows the following error:

An exception with the type CX_SY_REF_IS_INITIAL was raised, but was not handled locally or declared in a RAISING clause.
Exception in program CL_SADL_ABQI==================CP

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

  • SAP S/4HANA
  • SAP S/4HANA Public Cloud

Cause

Caused by a custom modification to an internal class.

Resolution

To identify the custom change, check the class, for which the exception/dump occurred in SE24. In this case it is CL_SADL_ABQI.

  • Compare its versions by each section (public, protected, private) by using Version Management. See KBA 2339632.
  • Check the Last Change user by navigating to the Properties tab of the class in SE24.

See Also

  • SAP Note 7 - Error caused by customer modification/development
  • KBA 2339632 - How to use Version Management
