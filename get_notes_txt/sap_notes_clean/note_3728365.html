Symptom

This article discuss the available options in case the customer would like to roll back a Cancelled Payout process for whatever reasons.

Environment

  • SAP Sales Performance Management
  • Agent Performance Management

Reproducing the Issue

Example:

  1. Started a Payout process
  2. Cancelled the Payout process before it got finished
  3. Observed that the cancelled Payout is no longer visible on the list of running Payouts

Resolution

  1. Navigate to the Payout menu under Processing (accessible via either the Manager portal or the Integration portal)
  2. Locate the specific Payout Run ID that is in Cancelled / Pending status
  3. Click on the Payout Run ID number in the row to open a new window
    1. Click the Reverse button - at the bottom of the newly opened window - to initiate the reversal process (this rolls everything back to the start of the Payout and a new Payout process needs to be initiated when it suits)
    2. If the the customer wishes to start the Payout again after completing the reversal process, click the Rerun button located at the bottom of the screen (this rolls everything back to the beginning - or another point in the Payout process - and then starts the Payout again)
  4. In the new screen that opens, select OK to confirm and start the selected process
