Symptom

It is necessary to discover the CDS View which contains a determined field for the Sales and Services area apps. 

Environment

SAP S/4HANA Cloud Public Edition

Resolution

All the CDS Views, which are released for usage, in the Sales and Services areas can be reviewed in the SAP Help Portal documentation below:

  • SAP Help Portal - CDS Views for Sales
  • SAP Help Portal - CDS Views for Service

In case the CDS View or the specific field are not released yet, it is necessary to create a Feature Request so the responsible development team can analyze if it is possible to release the field or not. (in case it gets enough votes in the Customer Influence Portal).

See Also

  • SAP KBA 2963059 - How to Submit Feature Requests for SAP S/4HANA Cloud Public Edition
