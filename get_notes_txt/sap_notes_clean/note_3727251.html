Symptom

  • Client error retrieving JWT token when attempting to connect to Task Center.
  • Error message: "Client error retrieving JWT token. Call to XSUAA was not successful!. Server URI https://<domain>/api/v2/auth/token/oauth/token. Http status code 404. Response body 'Sorry, the page you are looking for does not live here. If you think the error is yours, please try again. If it's our fault, we'll get right on it.

Please contact us to describe what happened.'

  • Error message: "There was an error while processing request for task operation. Error while obtaining a user token. Your request cannot be performed at the moment. Please try again later and if the error persists, contact your administrator and provide the following information: Error requesting user token with grant_type 'urn:ietf:params:oauth:grant-type:jwt-bearer': Client error retrieving JWT token. Call to XSUAA was not successful!. Server URI https://com/api/v2/auth/token/oauth/token. Http status code 404. Response body '
  • Underlying Post to the API returns "sap.taskcenter.internal.server.error" with a 500 Internal Server error status code

Environment

SAP Task Center

Reproducing the Issue

  1. Attempt to connect to the Task Center in SAP Advanced Work Zone.
  2. Observe the client error when retrieving the JWT token.
  3. Check the network trace and identify the failure at the server URI: https://<domain>/api/v2/auth/token/oauth/token.
  4. Review the HTTP status code, which returns a 404 error.

Cause

Misconfiguration in the destination

Resolution

Check the destination parameters in the SAP Task Center: Connect SAP Build Work Zone Advanced Edition and SAP Task Center 

See Also

Refer to: SAP Task Center: Connect SAP Build Work Zone Advanced Edition and SAP Task Center
