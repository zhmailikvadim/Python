Symptom

In the job scripts, when trying to Add link to package script or Relink to package script, the linked package script couldn't open unless you reopen the job.

Other Terms

IDM, Identity Management, Job Scripts, DevStudio, Package Scripts

Reason and Prerequisites

You are using SAP Identity Management 8.0 SP06

Solution

Apply the patch for SAP Identity Management Developer Studio available for download at https://tools.hana.ondemand.com/#idm. Follow the installation instructions.
