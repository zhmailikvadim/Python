Symptom

Does SAP Afaria 7.0 support Microsoft SQL Server 2019 as a database?

Environment

  • SAP Afaria 7.0
  • Microsoft SQL Server 2019

Resolution

As of the published date of this KBA, the latest Afaria 7.0 Service Pack is 33.

The System Requirements for this release can be found here: SAP Afaria System Requirements

Current Microsoft databases supported are:

  • Microsoft SQL Server 2016 Enterprise edition
  • Microsoft SQL Server 2016 Standard edition
  • Microsoft SQL Server 2014 Enterprise edition with SP2
  • Microsoft SQL Server 2014 Standard edition with SP2

Therefore Afaria 7.0 does NOT support using Microsoft SQL Server 2019 at this time.

Keywords

SQLServer, MS SQL
