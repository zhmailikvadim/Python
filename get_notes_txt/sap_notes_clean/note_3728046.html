Symptom

The following issues resolved as part of this SAP Note -

1) In the Assortment screen, the configuration maintained for default view type and view type order was not respected in the UI.

2) In the Assortment screen, when there were many products and the user selected a product at the end of the list, the selection was not cleared after Save or Cancel actions.

3) In the KPI Monitor, data was displayed with more than 2 decimal places.

4) The Plan ID was not visible in the CBP screens when the user navigated from the Detail Promotion screen via the Plan hyperlink.

5) In the Detail Promotion screen, the system displayed a promotion scenario linking warning message even though the scenario was not applicable.

Other Terms

Assortment, default view, Selection, Plan Id, KPI Monitor, Promotion, scenario link, warning message

Solution

Please implement the given correction instruction.
