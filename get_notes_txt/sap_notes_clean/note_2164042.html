Symptom

Configure People Profile displays "Error occurred", with errorID: errorId=9a240e0f-e5f0-4519-85bf-b8fe73226ef1 AND fingerprint=047139855fa84166fc8a138d372d8a3388c4f46d

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

SAP SuccessFactors People Profile

Reproducing the Issue

  1. Go to Configure People Profile, "Error Occurred" pop up box displays;



  2. Check General Settings > Name Format or Legal Entity are missing for some elements.

Cause

Description for Name Format or legal entity name format is mandatory in Manage Data.

Resolution

  1. Go to Manage Data;
  2. Search for the Name Format or Legal Entity. Make sure that there is description for it;
  3. If none, click on Take Action, and populate the description.
