Symptom

In the process "Sending of Standard Load Profile (DSO View)" (DE_C_EQS_SLP_SEND_DSO), the process logs were not displayed and the profiles were getting updated incorrectly.

This issue is relevant for the market role: SUP.

Other Terms

EQS_SUP, /US4G/EQS_SLPV, /US4G/MREDIREF, EEDMSYNPROFALLOC, standard load profile, process logs, profile updates

Reason and Prerequisites

The error occurred because of incorrect exception handling.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the system displays accurate error logs and handles the import of profiles efficiently.

You need to retrigger the process. You can verify the fix by checking that the process shows accurate error logs or updates data correctly, depending on the tested functionality.
