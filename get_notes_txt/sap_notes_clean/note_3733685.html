Symptom

  • An AI Agent does not run the AI Tool as expected.
  • When executed in the Playground, its status stay as Pending or Processing.

Environment

SAP CX AI Toolkit

Reproducing the Issue

  1. Go to the AI Agents tab
  2. Open the AI Agent
  3. Go to the Step 5 (Playground)
  4. Select an object (Opportunity, Quote, Ticket, or any other)
  5. Click on Execute

Cause

The AI Tool has some special character in the name, and special characters are not accepted.

Resolution

  1. Remove the special character from the AI Tool name
  2. Execute the AI Agent again.
