Symptom

Syntax errors are encountered in Include programs /CPRA/LAN_LINKSF01 and /CPRA/LAN_LINKSF03 during compatiblity testing of S/4 HANA 2025 FPS 01 with addon CPRA 200 SP 01.

Other Terms

CPRA

Reason and Prerequisites

After the S/4 Hana upgrade, there were updates on the number of fields in the include structure OIU_CREATION which caused incompatibility between the /CPRA/S_OIU_PR_DN_UI structure and OIU_PR_DN table structure for a direct move of data code logic.

Solution

Replaced the move assigment (equal sign '=') with MOVE-CORRESPONDING statement.
