Symptom

  • The list of all audit events includes "Truncation of Audit Table" as event 58
  • However, no audit logs are generated for this option even when it is configured.

Environment

  • SAP Adaptive Server Enterprise (ASE)
  • Auditing

Reproducing the Issue

Configure ASE to audit event 58.

Execute either of the following commands that perform audit table truncation and note that no audit record was generated for event 58

  • 1> truncate table sysaudits_01
    2> go
  • 1> sp_configure "current audit table",2,'with truncate'
    2> go

Cause

The issue has been identified and logged under SAP CR 640973. Its description reads, "No audit records are produced for audit event 58 when it is configured and an audit table is truncated."

Resolution

CR 640973 is fixed in:

  • SAP ASE 16.0 SP03 PL07 

Workaround:

  • Audit table truncation can be tracked as a generic truncate table operation via event id 64.
