Symptom

  • Assistance is needed to create deep links to custom MDF cards on the Latest People Profile.
  • Uncertainty about what values to use for the <objectType> and <assignmentId> parameters in the deep link.

Environment

  • SAP SuccessFactors HCM Suite
  • Metadata Framework (MDF)

Cause

Resolution

Identify the MDF object used by the custom card:

  • In Configure Object Definitions, open the MDF object and copy the value of the field “Code”.
    This value is the objectType to be used in the deep link.

Determine the assignmentId (optional):

  • Use the Assignment UUID stored per user in the User Data File (UDF). This value is the assignmentId to target a specific user’s profile.
  • If you do not set the assignmentId, the deep link will open the profile of the logged-in user.
