Symptom

After implementing the SAP Note 3462256 and 3466868, you can not select all items in the item table by clicking the checkbox on the left-top of table on the SAP Fiori application Account Balance Monitor.

Other Terms

US4G, DSO, MOS, Account Balance Monitor, Select All, com.sap.cd.us4g.MonitorAccountBalance

Reason and Prerequisites

Performance Issue

Solution

Please implement the correction instruction in the SAP Note or import the higher version of Support Package.

This SAP Note is relevant to the market role: DSO, MOS.
