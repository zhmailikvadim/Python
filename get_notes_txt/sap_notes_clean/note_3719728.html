Symptom

You want to perform system conversion to SAP S/4HANA. You need to delete obsolete derivation customizing steps as preliminary step. After executing the dedicated deletion report RFFMEC_DELETE, you establish that standard transparent tables have been deleted in addition to obsolete derivation rule tables.

Other Terms

Derivation strategy FMEC (Expenditure Certification): no more used within SAP S/4HANA based PSM releases

Report RFFMEC_DELETE (Deletion of Obsolete FMEC strategy steps)

Transparent table deleted wrongly.

Reason and Prerequisites

Program error : only obsolete generated Derivation Rule tables must be deleted with Derivation Steps.

Solution

Implement the attached correction or corresponding support package.

Ensure that you have restored the mistakenly deleted tables, that were used in obsolete Derivation Strategy TABLE steps type: you can refer to the application log of the deletion report execution.
