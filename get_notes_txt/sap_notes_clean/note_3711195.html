Symptom

The value of tag AccptncDtTm in message type /TIPS_R25_PACS.008.001.08 causes an XML validation error due to a regular expression pattern mismatch.

Other Terms

OPM, PACS.008, Instant Payments, Converter, TIPS, Rulebook 2025

Reason and Prerequisites

The generated timestamp in XML format may contain trailing zeros in the fractional part of the time. This occurs because the timestamp is directly converted from the internal SAP long timestamp format.
With TIPS Rulebook 2025, the regular expression pattern for timestamps was updated and no longer allows trailing zeros in the fractional part of the timestamp.

Solution

Please implement provided correction instructions or corresponding Support Package.
