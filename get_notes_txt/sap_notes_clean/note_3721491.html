Symptom

User need clarity on the following queries during Joule onboarding:

  • From which version of SAP S/4HANA Cloud, private edition Joule is supported?
  • Whether SAP S/4HANA on-premise systems can be enabled for Joule?
  • What is the minimum SAPUI5 version required per release for successful Joule features?
  • Which type of communication user should be used for backend connectivity during the Joule setup?

Environment

  • SAP BTP, Cloud Foundry runtime and environment
  • Joule
  • SAP S/4 HANA Cloud, private edition

Resolution

  • Joule is available for SAP S/4 HANA Cloud Private Edition 2021 onwards but with some exceptions. All capabilities of Joule may not be supported from 2021 but from a higher release. For the list of capabilities available per release of SAP S/4 HANA Cloud Private Edition, please refer to Release-Specific Capabilities and SAP Note 3523238.
  • Joule is available for SAP S/4HANA Cloud Private Edition only within Rise with SAP and in an SAP managed data center setup, but not all on-premise environment.
  • The communication user must be created as a "Communication" user type.
  • To be able to use the Joule capabilities, the minimum UI5 version required per release of the SAP S/4HANA Cloud Private Edition system must be used, as given in the following table.

    SAP S/4HANA Cloud Private Edition ReleaseMinimum UI5 Version Required

    2021

    1.96.33

    2022

    1.108.33

    2023

    1.120.0 or Latest

  • Refer to Constraints for Joule in SAP S/4HANA Cloud Private Edition for more information.
