Symptom

Service orders are not created with status Released from a Maintenance Plan in spite of the flag "Do not Released" was not marked

Environment

SAP S/4HANA

Cause

Use enhancements (BAdIs) to update fields or resolve errors. The system may release or fail to release the Service Order depending on whether the enhancement is called before release.

Resolution

For example, some mandatory fields, such as Customer Reference, cannot be updated directly from the maintenance plan. In such cases, you can use the BAdI ORDER_SAVE to update them automatically. 

The Service Order was initially created with an Error status, preventing the system from releasing it. Although the errors were subsequently resolved via BADI, the system still does not release the order. 

In these specific case, system will not release the service order.
