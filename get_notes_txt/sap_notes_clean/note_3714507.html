Symptom

Unable to delete Approval Process linked to approved sales quote in sap sales cloud version 2

Environment

  • SAP Sales Cloud Version 2
 



Reproducing the Issue

  1. Go to User Menu
  2. Go to Settings
  3. Go to All Settings
  4. Go to Business Flow 
  5. Go to Approval
  6. Delete the Approval Process for Sales Quote
  7. Failed to delete the approval process as it is being used in the following salesQuote(s)
     
    [XXXX, YYYY, ZZZZ]

Cause

Resolution

Note down the Sales Quotes from the Error message

  1. Go to Navigation Menu
  2. Go to Sales quotes
  3. From the List View, delete the Sales quotes from the Error message received from approval process screen
  4. Go to User Menu
  5. Go to Settings
  6. Go to All Settings
  7. Go to Business Flow
  8. Go to Approval
  9. Select Entity Sales Quotes
  10. Delete the Approval Process. It will be deleted now.

Keywords
