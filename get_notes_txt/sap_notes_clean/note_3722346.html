Symptom

In BTP DMS UI (earlier known as Document Center), users are unable to see all folder types.

System considers only default folder type maintained in the customizing below:

SAP Customizing Implementation Guide -> Cross-Application Components -> Document Management -> Integration with SAP Document Center -> Profile Settings for SAP Document Center

Other Terms

Doc center, DMS_MDOC_PROFILE, Folder Type, FOL, Maintain Default Settings, all folder types

Reason and Prerequisites

Code error

Solution

Implement the attached correctioin instructions.

Note: Now the application will show all the folder types defined and is structured under 'EDIPUBLICROOTFOLDER'. If a folder contains a file and is also a document structure then only the folder is displayed.
