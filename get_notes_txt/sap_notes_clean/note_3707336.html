Symptom

Business rule is not retrieving company code for validation purposes, the Rule retrieves null instead of company code during execution. Issue occurs when hiring candidates have future hire dates.

Environment

SAP SuccessFactors Onboarding

Reproducing the Issue

  1. Configure a business rule to validate company code in the Recruit New Hire Step.
  2. Assign the rule to a custom MDF object.
  3. Enable rule trace to monitor execution.
  4. Attempt to hire a candidate with a future hire date.
  5. Observe that the rule retrieves null instead of the company code.

Cause

MDF framework does not support fetching future job information, including company code.

Resolution

Please submit an enhancement idea to the Customer Influence for consideration, if this is an important feature/function you would like to see made in Onboarding in a future release. For how to raise an enhancement idea, refer to the KBA 2090228 - How to Submit Ideas for SAP SuccessFactors Products.
