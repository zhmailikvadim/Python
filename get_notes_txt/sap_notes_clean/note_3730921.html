Symptom

The requirement is to add custom fields in the Post Cash Journal Entries - FBCJ app.

A custom field created in the Accounting: Coding Block business context is not visible in the app and cannot be enabled via layout editing or the Custom Fields app User Interface tab.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Create a custom field in the business context Accounting: Coding Block (FINS_CODING_BLOCK) using the Custom Fields app (F1481).
  2. Open the Post Cash Journal Entries - FBCJ app and try to display or add the custom field in the entry screen via layout adaptation.
  3. In the Custom Fields app (F1481), navigate to the User Interfaces tab and attempt to enable the field for the Post Cash Journal Entries - FBCJ app or its data source.
  4. Observe that the custom field is not available to be enabled and cannot be added in the app.

Cause

The Post Cash Journal Entries - FBCJ app currently does not support extensibility for adding custom fields.

Resolution

Vote for the existing improvement request Improvement Request Details - Customer Influence to prioritize this capability.
