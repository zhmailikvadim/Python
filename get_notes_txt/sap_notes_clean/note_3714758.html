Symptom

In CO summarization (transaction KKBC_HOE_H), the termination AMDP_EXECUTION_FAILED occurs when you call the object list. The termination occurs in the method CL_FCO_AMDP_KKBC=>READ_TOTALS_BY_NODE.

Other Terms

KKBC_HOE_H, CO summarization, CL_FCO_AMDP_KKBC, READ_TOTALS_BY_NODE, memory overflow, memory, AMDP_EXECUTION_FAILED, object list

Reason and Prerequisites

You want to process a very large amount of data.

Solution

Implement the corrections. This optimizes the use of the memory in the method CL_FCO_AMDP_KKBC=>READ_TOTALS_BY_NODE. In principle, however, such terminations cannot always be prevented. We recommend that you reduce the amount of data to be processed by making a corresponding selection.
