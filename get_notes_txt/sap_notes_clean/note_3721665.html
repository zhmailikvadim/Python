Symptom

You want to delete the incorrect document record from the eDocument Cockpit. However, you are unable to do that due to an error 'The action is not allowed for any of the selected eDocuments'. 

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Open the eDocument Cockpit APP.
  2. Select the eDocument.
  3. Choose the button Cancel, then the mentioned error occurs.

Cause

Due to tax requirements from the government in Taiwan, the system has been designed such that it does not allow users to cancel eDocuments that have the processing status 'eDocument Created'.

Resolution

To proceed with canceling this sort of eDocuments, it is recommended to submit it using a dummy Government Uniform Invoice (GUI) number.

Update the field Document Reference ID in the Accounting Document with the dummy GUI number. Or update the field Reference in the Billing Document with the dummy GUI number. Then submit the eDocument.

Upon successful submission, the system will allow the eDocument to be canceled.
