Symptom

When attempting to run the Historical Firefighter Assignments report under "Reports and Analytics > Emergency Access User Management Reports", the message "No POWL object types are assigned to you. Please inform your system administrator." is displayed.

Environment

  • GRC Access Control

Reproducing the Issue

  1. Start NWBC.
  2. Navigate to the tab "Reports and Analytics > Emergency Access User Management Reports".
  3. Click on the link for "Historical Firefighter Assignments".
  4. A new pop-up window appears with the error message: "No POWL object types are assigned to you. Please inform your system administrator."

Cause

  • Missing configurations.

Resolution

Perform following steps to address it:

  • Check POWL Type definition:
    • Start transaction POWL_TYPE.
      • Check the existence of the following entry:
        • POWL Type ID: GRAC_FFARC
        • Description: Historical Firefighter Assignments
        • Feeder class: CL_GRAC_POWL_SPM_FFARC
        • "Sync. call" and "No Msg. Wrapping" is unchecked
      • If it does not exist, please create it and save it.
  • Check POWL Type - Role assignment:
    • Start transaction POWL_TYPER.
      • Check the existence of the following entry:
        • Application ID: GRAC_FFARC
        • Role: empty
        • POWL TYPE ID: GRAC_FFARC
      • If it does not exist, please create it and save it.
  • Check POWL Query definition:
    • Start transaction POWL_QUERY.
      • Check the existence of the following entry:
        • Query ID: GRAC_FFARC
        • Description: All
        • POWL Type ID: GRAC_FFARC
        • "Sync. call", "Layout":  empty,
        • Refresh Type: Manual only
      • If it does not exist, please create it and save it.
  • Check POWL Query - Role assignment:
    • Start transaction POWL_QUERYR.
      • Check the existence of the following entry:
        • Application: GRAC_FFARC
        • Role: empty
        • Query ID: GRAC_FFARC
        • Everything else: empty
      • If it does not exist, please create it and save it.

Also confirm whether the manual steps of note 0003105586 are correctly performed.
