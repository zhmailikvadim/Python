Symptom

Time valuation alerts can not retrieve variables. 

Environment

  • SAP SuccessFactors Employee Central - Time Sheet
  • SAP SuccessFactors Time Tracking

Cause

Time valuation alerts currently support only pure text and do not have a tag available to retrieve other variables. 

Resolution

    This is a known function gap, please raise an enhancement request to have the feature considered in the future release. 

    For detailed steps on how to raise an enhancement request, refer to: KBA 2090228 - How to submit enhancement ideas for SAP SuccessFactors Products.
