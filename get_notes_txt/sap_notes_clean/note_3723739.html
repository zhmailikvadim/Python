Symptom

  • Cascading List of Values are not filtered in crystal report based on BEx Query.
  • Cascading Value help based on User Exit is not working.

Other Terms

cascading list of value lov filtered CR

Reason and Prerequisites

Environment

  • SAP Crystal Reports 2025 Patch 8
  • SAP BW (DSL)
  • BEx Query

Steps to Reproduce

  1. Open crystal report in the designer.
  2. Refresh the report.
  3. When prompted select a value.
  4. Open for the next parameter the value help.
  5. Observe the LOVs are not filtered based on the previous selection.
