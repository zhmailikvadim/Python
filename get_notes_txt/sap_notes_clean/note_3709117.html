Environment

  • SAP BusinessObjects Business Intelligence Platform (BI) 4.2 / 4.3 / BI 2025
  • All Supported OS

Reproducing the Issue

  1. Upgrade SAP BO
  2. Note all schedules are failing
  3. Check in event viewer logs: 
    Faulting application name: WIReportServer.exe, version: 14.3.4.5434, time stamp: 0x68a9f9a4
    Faulting module name: c3_cdbcl.dll, version: 14.3.4.5434, time stamp: 0x68a9ed45
  4. Check if faulting Module is present in the system or not

Cause

  • The upgrade process might have failed to properly install or update c3_cdbcl.dll, leaving a corrupted/incomplete, or mismatched version. 

Resolution

  1. Take a backup of c3_cdbcl.dll in the non-working environment
  2. Copy the valid c3_cdbcl.dll file from a working environment
  3. Restart SIA (Optional)

Keywords

c3_cdbcl.dll, faulting module, 0xc0000005
