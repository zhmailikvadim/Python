Symptom

A false SPS block can be set in the delivery in case of a HANA database and the usage of address versions. Due to the different sorting of the HANA database, the new address is compared with the address version. Then the delivery is blocked due to address data.

Other Terms

C2S8000031320, C2S 8000031320, SPS, VL01N, address versions, HANA

Reason and Prerequisites

Program error.

Solution

Implement the correction instruction.
