Symptom

  • In transaction KEKF, the "new entries" and "copy" options under the "Edit" menu are greyed out and unavailable.
  • The system displays entries for different fiscal years for the same operating concern and controlling area.

Environment

  • Contolling(CO) 
  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA Finance
  • SAP S/4HANA

Reproducing the Issue

  1. Open transaction KEKF.
  2. Navigate to the "Edit" menu.
  3. Observe that the "new entries" and "copy" options are greyed out and cannot be selected.

Cause

Once there has been an assigned to the controlling area to the operating concern in transaction code KEKK, the entries will automatically be displayed in transaction code KEKF.
Therefore, there is no need to use the 'new entry' function in KEKF. Please first check if there has been an assignment to the controlling area to the operating concern in KEKK.

Resolution

  1. Assign the controlling area to the operating concern using transaction code KEKK.
  2. Verify the controlling area settings in transaction code OKKP (Activate components/control indicators).
  3. Perform detailed settings by fiscal year in transaction code KEKF based on the controlling area settings established in transaction code OKKP.
