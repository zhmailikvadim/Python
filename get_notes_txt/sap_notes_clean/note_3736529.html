Symptom

  • When the user's language is set to a regional dialect (e.g., de-CH, fr-CH, it-CH), the suggested file name during export uses the original, untranslated document name instead of the localized one.

Other Terms

WebI, webintelligence, TMT, translate

Reason and Prerequisites

Environment:

  • SAP BusinessObjects Business Intelligence platform 2025
  • Web Intelligence
  • Linux

Steps to Reproduce:

  1. Create a Web Intelligence document
  2. Add a cell with the formula =GetLocalized("some text")
  3. Import the document into Translation Management Tool
  4. Add a dominant language (e.g., de-DE) and translate the document name and the text string
  5. Re-export the document to BI platform
  6. In LaunchPad open the document with the user language set to the dominant language
  7. The GetLocalized cell shows the translated text
  8. Export the document - the suggested filename is localized
  9. Change the user language to a dialect (e.g., de-CH)
  10. Open the document again
  11. The GetLocalized cell still shows the translated text, but the suggested export filename is no longer localized
