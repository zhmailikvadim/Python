Symptom

  • An error occurs when attempting to preview calculation view node-level data in Business Application Studio (BAS).
  • Error message: "intermediate view creation error".

Environment

HANA Editors in Business Application Studio

Reproducing the Issue

  1. Attempt to preview calculation view node-level data in Business Application Studio (BAS).
  2. Observe the error messages "intermediate view creation error".

Cause

  • The .hdiconfig file configuration is incorrect, specifically the format of the version details. The file contains "plugin_version" instead of the required "minimum_feature_version", which causes inconsistencies.
  • The .hdiconfig file shows:

    "plugin_version": "2.x.xx.x", 

Resolution

As detailed in: The SAP HDI Container Configuration File | SAP Help Portal 

  1. Update the .hdiconfig file to include the correct format which should start with "minimum_feature_version": "100x", this indicate the project is based on BAS, the binding target is a cloud instance, usually HANA cloud.
    --for example
    {
        "minimum_feature_version": "1009", 
  2. The .hdiconfig file must be located in the root folder of the database module.
