Symptom

  • When opening a story in SAP Analytics Cloud via Files, it is possible to see the table option: "Open in Excel Add-in"
  • When opening the same story by copying the URL in a new browser or tab, the table does not show the option: "Open in Excel Add-in"
  • The option "Open in Excel Add-in" is available when accessing the story directly in SAP Analytics Cloud but disappears when the story is opened using the URL in a new browser tab or browser.

Environment

SAP Analytics Cloud 2024.21.14

Reproducing the Issue

  1. In SAP Analytics Cloud, create a story.
  2. Add a table widget.
  3. Check the table View Mode options: Enable Excel Add-in.
  4. Save and view the story.
  5. Select the table and click on the three dots in the upper right corner of the table.
  6. In the menu, notice the following option displays: Open in Excel Add-in.
  7. Copy the URL of the story and open it in a new browser tab or in a new browser.
  8. Select the table and click on the three dots in the upper right corner of the table.
  9. In the menu, notice the option "Open in Excel Add-in" is not available.

Cause

  • When opening a story from SAP Analytics Cloud directly, the option "Open in Excel Add-in" is available in the table because it opens the story in Full Viewer mode.
  • When copying the URL of the story and opening it in a new browser tab or new browser, the story opens in Lite Viewer mode, which does not support the "Open in Excel Add-in" option in the table context menu, as documented in the SAP Analytics Cloud Help under the section: Lite Viewer Usage Restrictions

Resolution

  • To view the table option: "Open in Excel Add-in", switch to the full Optimized Story Experience viewer (Full Viewer).
  • To switch to the Full Viewer, add the following parameter to the story URL: ?mode=optimized
