Symptom

SAP Business Technology Platform (BTP), Cloud Foundry (CF) Extension Center fails with error:


************************************

An error occurred while retrieving the services list

************************************

Screenshot:

Environment

  • SAP Business Technology Platform (BTP)
  • Cloud Foundry (CF)

Reproducing the Issue

Login to SAP BTP subaccount and access the Extension Center (where you can view a list of extensions and information about the secrets, config maps, and execution logs for an extension). This activity fails due to the above stated error.

Cause

At times such issues can occur due to Database/CPU overload resulting multiple flapping in the Extension Center application.

Resolution

Retry after couple of hours.
If the issue persists, report a case in the BC-CP-XF-SRT component with the below details:

  • Subaccount region - e.g. EU10

  • Subaccount ID - e.g. 22222222-1111-2222-33333-44444 (for more details, check SAP KBA ##3007643 - How to find the subaccount ID in SAP BTP Cockpit)

  • Step by step screenshots of how to reproduce the error

  • Attach the Browser network trace. Clear the browser cache (check https://www.refreshyourcache.com). 
    Now reproduce the issue and attach the browser console log after error reproduction. You can save the console log by pressing "Save as..." in console. More info: SAP KBA ##1990706 - 'How to capture a HTTP trace using Chrome browser'.
