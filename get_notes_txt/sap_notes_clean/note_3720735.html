Symptom

  • After a declaration is archived, the icon for accessing attachments is no longer displayed.
  • Attachments (PDFs etc.) cannot be accessed via an icon in the toolbar.

Environment

  • SAP Global Trade Services 11.0
  • SAP Global Trade Services Edition for Hana

Reproducing the Issue

  1. Navigate to "System Administration -> Archiving -> Display archived documents" via transaction code "/SAPSLL/MENU_LEGAL".
  2. Attempt to display attachments for an archived document.
  3. Observe that the icon for accessing attachments is not displayed.

Cause

This is a system defect.

Resolution

Please implement the following SAP Note.

SAP Note 0003381265 - CUHD Archiving: The attachments are deleted before running ILM destruction

Keywords

archiving, attachments, icon missing, toolbar, display archived documents, GTS, CUHD archiving, ILM destruction
