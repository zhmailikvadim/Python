Symptom

Collection ID is displayed to users who are in a different locale from the newly created Collection's locale. Existing collections do not have this issue.

Environment

SAP SuccessFactors Learning

Reproducing the Issue

  1. Create a collection as an admin (admin locale English UK).
  2. Recommend it to a user via Assignment Profile.
  3. Check it in the user side with same locale as admin> Title OK.
  4. Now changed the use locale to Spanish.
  5. Back to the Home Page, now the ID is displayed.

OR

  1. As a user, create a new collection.
  2. Now change the user's locale.
  3. Check the locales again.
  4. It will show "No title" in the referred locale.

Resolution

This is currently under investigation by Engineering team.
