Symptom

You have configured Audit Logging in the SAP Business Technology Platform (SAP BTP) Cloud Foundry environment (https://cockpit.btp.cloud.sap).
You would now like to understand the retention period for Cloud Foundry audit event logs (such as changes to org users, space users, and roles etc.), that are retrieved using the command:
**********************************************************
cf curl "/v3/audit_events"
**********************************************************

Environment

  • SAP Business Technology Platform (BTP)
  • Cloud Foundry (CF) environment
  • Audit Log Service

Reproducing the Issue

Configurations have been done as per Audit Logging in the Cloud Foundry Environment to retain logs. However logs for Cloud Foundry native audit events are stored only for a short period of time.

Cause

As per present architecture.

Resolution

The Cloud Foundry (CF) event logs (which are accessible using the CF command line interface), have a retention period of 14 days.
This value comes from the Cloud Foundry side and cannot be changed.
