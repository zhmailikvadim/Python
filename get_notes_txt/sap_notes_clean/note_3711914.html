Symptom

As described in Regulatory Change Manager tool under Changes: New formats for personified accounting forms ADV and SZV-K and removal of form ADV-6-1, the layouts of the forms ADV-1, ADV-2, ADV-3 in both PDF and XML formats have been changed as of December 30, 2025. This SAP Note provides you with the relevant system changes to generate the valid output of these legal forms.

Other Terms

HRULPFP1, HRULPFP2, HRULPFP3, HRPAYRU_HRULPFP3, HRPADRUPFR_1, HRPADRUPFR_2, HRPADRUPFR_3, HR_RU_ADV1_2026, HR_RU_ADV2_2026, HR_RU_ADV3_2026, АДВ-1, АДВ-2, АДВ-3, АДВ

Reason and Prerequisites

SFR Order № 1461 (Приказ Фонда пенсионного и социального страхования Российской Федерации от 17.11.2025 № 1461 "Об утверждении форм и форматов сведений, используемых для регистрации граждан в системе индивидуального (персонифицированного) учета‚ и порядка заполнения форм указанных сведений"), published on December 19, 2025 by the Pension and Social Insurance Fund of the Russian Federation (SFR).

You have implemented SAP Note 3720831 with prerequisite objects (UDO).

Solution

As a general rule, SAP recommends that you install a solution by applying a Support Package. However, if you need to implement a solution earlier, use the SAP Note Assistant to implement automatic and manual correction instructions. For more information about the SAP Note Assistant, see the SAP Support Portal under SAP Note Assistant.

Note that after installation of HR Support Package some customizing records for the table view V_T7RU0290IDCODE will be imported into client 000 only. In case of need, the changes have to be replicated to the table view in other client(s) by the user/customer manually.

Note that documentation, translations and Customizing activities are only delivered via Support Package.
