Symptom

When you implement this SAP Note, the test environment for unit tests in BW becomes more convenient.

Other Terms

CL_RSO_UT, CL_RSO_UT_UTIL, T_RETMSG_2_T_MSG

Reason and Prerequisites

Program

Solution

  • SAP BW/4HANA 2.0
    Import Support Package 16 for SAP BW/4HANA 2.0 (SAPK-20016INDW4CORE) into your BW system. The support package is available once SAP Note 3381469 ("SAPBWNews SAP BW/4HANA 2.0 SP16"), which describes this SP in more detail, has been released for customers.

  • SAP BW/4HANA 2021
    Import Support Package 08 for SAP BW/4HANA 2021 (SAPK-30008INDW4CORE) into your BW system. The support package is available once SAP Note 3381413 ("SAPBWNews SAP BW/4HANA 2021 SP08"), which describes this SP in more detail, has been released for customers.

  • SAP BW/4HANA 2023
    Import Support Package 01 for SAP BW/4HANA 2023 (SAPK-40001INDW4CORE) into your BW system. The support package is available once SAP Note 3372694 ("SAPBWNews SAP BW/4HANA 2023 SP01"), which describes this SP in more detail, has been released for customers.

In urgent cases, you can implement the correction instructions as an advance correction.

You must first read SAP Notes 1668882 and 2248091, which provide information about transaction SNOTE.

To provide information in advance, the SAP Note(s) specified above might already be available before the support package is released. In this case, however, the short text still contains the term "preliminary version".
