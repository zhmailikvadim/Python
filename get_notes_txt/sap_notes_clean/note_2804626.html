Symptom

For CTP defined as deductions with CTP transfer for non-regular sign, there is a tolerance for declaring both CTPs in a same period.  In case you have a deduction and a refund of such of contribution in a same month, one CTP disappears. This issue occurs only with details of aggregated contributions options 03 - Aggregated contributions in GIP-MDS format with amounts and 04 - Aggregated contributions in GIP-MDS format.

Other Terms

CTP 671, CTP 801, CTP 668, CTP 669.

Reason and Prerequisites

This is a program error.

Solution

These corrections will be available with the next HR Support Packages referenced below.
