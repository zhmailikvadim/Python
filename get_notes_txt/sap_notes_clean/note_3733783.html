Symptom

CPI connection fails and In CPI Monitor Message Processing errors with similar information below are perceived:

Connect timed out: destination-configuration.<region host>/IP

In Cloud connector following is perceived on the scc_core.trc in DEBUG level:

 2025-04-09 09:12:04,532 -0000#ERROR#com.sap.core.connectivity.tunnel.client.application.ApplicationClientConnectTask#Thread-9123#          #Unexpected exception while establishing application tunnel connection for tunnelId: account:///<subaccount ID>
io.netty.channel.ConnectTimeoutException: connection timed out: connectivitynotification.cf.br20.hana.ondemand.com/4.228.118.21:443

Environment

SAP Cloud Connector Release Independent

Reproducing the Issue

Perform a CPI connection and verify error message 

Cause

The host and IP mentioned in the error message is one of the hosts/IPs from  Prerequisites | SAP Help Portal  that should be whitelisted in firewall and proxy for the respective region host.

Sincee it is not whitelisted, proxy or firewall is blocking the connection

Resolution

Check  Prerequisites | SAP Help Portal and whitelist all the hosts for the respective region of the subaccount.
