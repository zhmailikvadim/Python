Symptom

 When attempting to post a parked document in transaction FBV0, the error "Field BKPF-REINDAT is not an input field" (Message no. 00347) appears.

Environment

  • SAP ERP
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition 

Cause

• In the SAP standard system, the field REINDAT is available only for specific country solutions where it is legally required.
• It is also active when Public Sector Management (PSM) Prompt Payment Act functionality is enabled.
• The field remains inactive unless these specific conditions are met.

Resolution

 If the field REINDAT is required for specific information, consider using the document header field or the reference field as alternatives.
