Symptom

System cannot create the  purchase document due to the error  "Change indicators could not be uniquely determined in material mapping"

Environment

SAP S/4HANA

Cause

Missing customizing

Resolution

Update the S/4 HANA Service customizing :

SAP Customizing Implementation Guide -> Service -> Transactions -> Settings for Service Transactions -> Integration -> Logistics Integration -> Set Purchasing Document Types for Logistics Integration

 

with the your Purchase Document type

Keywords

M_011, logistic integration, BAPI_PR_CREATE
