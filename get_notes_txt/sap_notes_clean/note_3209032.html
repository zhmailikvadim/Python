Symptom

HANA upgrade fails with following error:

./hdbinst: /lib64/libc.so.6: version 'GLIBC_2.14' not found (required by ./hdbinst)

Environment

  • Redhat 6.10
  • Target HANA version: HANA 2.0

Reproducing the Issue

Execute command: hdbinst

Cause

Current OS version does not meet the minimum requirement of HANA database.

Resolution

Upgrade OS to at least Redhat 7 so as to meet the minimum requirement as listed in following note.

2235581 - SAP HANA: Supported Operating Systems

See Also

2235581 - SAP HANA: Supported Operating Systems

Keywords

hdbinst, GLIBC_2.14, not found
