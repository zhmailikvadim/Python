Symptom

System randomly displays the message "The system is busy processing data" when clicking on the Save icon after modifying data in the system.

Environment

SAP S/4HANA Cloud Public Edition

Cause

Resolution

This behavior is expected, as some process requires additional backend processing (such as generating new IDs and validating related data). During this time, the SAP Fiori interface automatically displays a busy indicator to inform the user that the system is still processing the request and to prevent further actions until the operation is completed.
Additionally, the appearance of this message may vary depending on the system load at a given time. If multiple transactions are being processed simultaneously, the system response time can increase, which may trigger the busy indicator.
