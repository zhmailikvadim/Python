Symptom

  • Smart Insight in graphs does not work when there are customizations in hierarchies (e.g., modified or user-defined hierarchy structures).
  • Smart Insight in graphs does not work when calculated fields are used to create a hierarchy.

Environment

  • SAP Analytics Cloud (Enterprise)
 

Reproducing the Issue

  1. Log in to the tenant using the Access4 Support User.
  2. Open the relevant Story.
  3. Navigate to the page containing the Dashboard.
  4. Right-click on the Chart/Bar-Chart and observe that the Smart Insight functionality is unavailable.

Cause

The issue occurs due to a documented limitation in SAP Analytics Cloud.

Smart Insights are not available when working with custom hierarchies or calculated hierarchies, as these structures exceed the supported framework for hierarchy complexity.

Resolution

  1. Confirmed that the behavior is expected and documented in SAP Help.
  2. Refer to the SAP Help documentation for Smart Insights limitations: Smart Insights | SAP Help Portal.

An enhancement request to add the functionality into SAP Analytics Cloud (SAC) has been submitted by the customer.

Please cast your vote here: Improvement Request Details - Customer Influence.
