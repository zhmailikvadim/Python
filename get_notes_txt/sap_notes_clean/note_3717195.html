Symptom

  • Onboarding and compliance documents fail to generate without triggering a BPE Error.
  • Tasks appear completed in the dashboard, but no signature for I9 is generated, causing E-Verify to not trigger.
  • The issue impacts compliance deadlines and production.
  • Error message: "Error while executing SCA command [com.successfactors.onb2compliance.service.process.ComplianceDeleteProcessTask] Caused by: com.successfactors.sca.ServiceApplicationException: Given task id invalid for deletion."

Environment

SAP SuccessFactors Onboarding

Reproducing the Issue

  1. Navigate to BPE Manage Process Tasks.
  2. Attempt to generate onboarding and compliance documents.
  3. Observe that no BPE Error is created, and tasks appear completed in the dashboard despite missing signatures for I9 and E-Verify triggers.

Cause

The error occurs due to an invalid task ID for deletion, which prevents the compliance process from completing successfully.

Resolution

1. Navigate to Manage Data -> Compliance Process and search for the affected user's ID.

a. Select the record with Compliance Process Type as "I-9".  

b. Click "Take Action" -> "Make Correction".  

c. Mark the compliance process status as "Compliance process cancelled" and save.  

2. Navigate to Manage Data -> Create New -> Trigger I-9 Flow.

a. Provide the value for the user and process.  

b. Select "Correct Form I-9 Section 1 Data" from the dropdown "Trigger Type" and save.  

c. This will trigger the I9 Section 1 for the candidate.
