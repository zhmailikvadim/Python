Symptom

When printing Purchase Orders or Scheduling Agreement for domestic operations, the Ship-to country name is not displayed in the printout.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Open the ME22N (Change Purchase Order) app.
  2. Select Other Purchase Order button.
  3. Add a domestic purchase order.
  4. Press Enter.
  5. Open Messages.
  6. Select the PRINT output line and choose Display Document.
  7. Review the Ship-to address in the print preview.

Note that the country is not shown for domestic cases.

Cause

For domestic operations the Address Line containing the Country is not copied from master data, meaning that system will compare the countries and if it is the same it will not display in the Output as it is not necessary accordingly to International Postal Rule.

Resolution

System behavior.
