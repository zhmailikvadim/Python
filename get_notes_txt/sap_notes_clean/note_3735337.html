Alternatively the error can be corrected in translation object:

Object type (shortcut): SRT4
Object name: MP344100                                2000

(Please check if the texts fit to your Support Package level!)

Please use transaction SE63 to translate the text(s).

  1. Enter /oSE63 in the command field (transaction field) and press Enter
  2. Enter Object type (see above) in the command field and press Enter
  3. Enter Object name (see above) in the field Object Name
  4. If required change Source and/or Target language
  5. Press Edit
  6. Enter or correct the translation in the text field below the source text
  7. Choose Save to save your translation(s)
  8. Choose Back (F3) to leave the translation editor

110910 Deleting the language load
