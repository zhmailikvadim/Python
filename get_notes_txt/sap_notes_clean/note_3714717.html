Symptom

  • I would like to have different Shifts in a Job Posting and give the suppliers, the option of submitting Job Seekers for specific Shifts
  • I would like know how Shift Manager works
  • I would like a alternative for creating multiple Work Orders for the same worker, if they work on different Shifts

Resolution

SAP Fieldglass has a functionality called Shift Manager. With this, you can create a Job Posting with several Shift Positions, each shift can be valid for one or more days of the week, have specific start and end times, and have specific rates associated to it. Suppliers will be able to select the shifts that they would like to submit a Job Seeker to. And when hiring the workers, the Buyer can select which shifts they want to assign to the workers to.
