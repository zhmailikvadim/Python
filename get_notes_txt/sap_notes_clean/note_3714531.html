Symptom

Requester user already has Contract Manager permission, Auto Invoice option is still not visible for fixed and recurring fee items in contract request. 

Environment

SAP Ariba Buying & Invoicing

Cause

The ERS feature is not enabled in the realm, which is the prerequisite to display the Auto Invoice option for recurring fee.

Resolution

Enable the ERS feature by enabling the following parameter(s): 

  • Application.Procure.EnableERS 
  • Application.Procure.EnableERSForSAP (also required for SAP variant)

Keywords

auto-invoice, recurring item
