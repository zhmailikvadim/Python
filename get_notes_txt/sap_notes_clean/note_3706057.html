Symptom

Error message displayed: "Document key information incomplete: 0 records found" (Message no. FAGL_REPORTING_EN011) in transactions FBL1H or FBL5H.

Environment

  • SAP S/4HANA
  • SAP S/4HANA Finance

Reproducing the Issue

  1. Execute Transaction FBL1H or FBL5H to display the line items.
  2. In the output, click the hyperlink for any document number to drill down.
  3. Observe the error message displayed: "Document key information incomplete: 0 records found".

Cause

This is caused by a program error.

Resolution

Please implement the following KBA note and check it again.

 3570485 - FBL1H/FBL5H - Document key information incomplete Error for parked items

See Also

SAP note 3570485 - FBL1H/FBL5H - Document key information incomplete Error for parked items 

Keywords

FBL1H, FBL5H, document key information incomplete, FAGL_REPORTING_EN011, FI/PCA inconsistency, parked items, layout issue, coding error.
