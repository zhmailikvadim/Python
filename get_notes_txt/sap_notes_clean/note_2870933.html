Symptom

When BOM component having quantity in decimal (example quantity as 1.4). Even after attaching the full quantity at router step in routing maintenance, component still visible in unassembled component area showing remaining quantity as 0.0.

Other Terms

Routing steps, BOM Components

Reason and Prerequisites

Steps to reproduce:
1.Open Material Maintenance, create a Material. (20191129_MAT_1_COMPONENT, 20191129_MAT_1)
2.Open BOM Maintenance, create a BOM (20191129_BOM1) with the Assy quantity 1.4
3.Open Routing Maintenance, add the BOM and click to save.
4.Retrieve the previously created Routing. (20191129_ROUTING_1).

Actual result: Unassembled quantity shows 0.0 in routing maintenance.

Solution

The fix for the issue has been done in a form of code correction and will be delivered in core component.

Impact: Routing maintenance.
