Symptom

  • WebI reports based on BEx Query do not refresh current data on Prompts with SAP BI Mobile App
  • When applying prompt the report continues to reflect saved data

Other Terms

NW, business, warehouse, Web, Intelligence

Reason and Prerequisites

Environment

  • SAP BusinessObjects Business Intelligence Platform (BI) 4.1 SP06
  • SAP BI Mobile App
  • SAP BW

Reproducing the Issue

  1. Have a WebI report based on BEx Query with prompts
  2. Once the reports is opened on SAP BI Mobile App change a prompt value, such as "Between Dates" (click on Filter button)
  3. Click on Done
  4. The results are not changing and the saved report is displayed

Solution

  • This issue has been addressed and fixed with 6.3.10 IOS Patch release. SAP BusinessObjects Mobile 6.3.10 IOS version is available in Appstore.
