Symptom

Not able to integrate SAP Business Application Studio with GitHub Copilot via the button 'Set up Copilot' in the BAS status bar. Error message: 

An error occurred while setting up Copilot. Would you like to try again?

The extension 'GitHub.copilot' cannot be installed because it was not found.

Environment

  • SAP Build Code
  • SAP Business Application Studio

Reproducing the Issue

  1. Click on the GitHub Copilot icon in the taskbar of SAP Business Application Studio and click on button 'Set up Copilot'
  2. When prompted, use GitHub credentials to authenticate to the GitHub repository.
  3. After successful authentication with the GitHub repository, the error message appears: "The extension 'GitHub.copilot' cannot be installed because it was not found."

Cause

GitHub Copilot is currently not supported in Build Code / BAS.

Resolution

GitHub Copilot is currently not supported in SAP Build Code / BAS. It is appeared in the status bar by mistake, and needs to be removed.

While SAP is actively working on enabling GitHub Copilot. Once it is officially released, it'll be shared in the What’s New channel of the help document.
