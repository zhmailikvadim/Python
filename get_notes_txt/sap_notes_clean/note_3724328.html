Symptom

In SAP S/4HANA for manufacturing logistics, you use the EWM check monitor to run the Complete Tours with Confirmed WO for Unloading check. If multiple tours share the same route ID, the system may display an error and stop the mass processing of tours.

Other Terms

Manufacturing logistics, tour, EWM check monitor

Reason and Prerequisites

You use SAP S/4HANA for manufacturing logistics or this SAP Note is a prerequisite of another SAP Note.

Solution

Implement the attached correction.
