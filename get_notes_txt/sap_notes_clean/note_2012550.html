Symptom

The conditional formatting property is based on the description of the dimensions or measures, rather than the UID.

The chart only displays descriptions and has no knowledge of the UID. This causes an issue for two reasons:

  • Descriptions are not guaranteed to be unique
  • Descriptions could potentially change depending on the language

If descriptions are not unique, for example, if there are two measures with the same description, only the first one will ever be formatted.
If descriptions change, conditional formatting will not be successfully applied.




Reason and Prerequisites

Program error

Solution


Ensure that all measures and dimensions have unique descriptions.
