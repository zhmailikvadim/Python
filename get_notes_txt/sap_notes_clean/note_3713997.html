Symptom

When generating the withholding certificate from the 'Employment Tax (IRPF) reports' program (RPC190E0), a dump occurs when trying to view it.

Environment

  • Human Capital Management (HCM)
  • SAP R/3
  • SAP ERP Core Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA

Reproducing the Issue

  1. Run the 'Employment Tax (IRPF) reports' program (RPC190E0) by checking the 'Generate certificate' box on the 'Deductions certificate' tab.
  2. Enter an output device to view the document and press 'Print Preview'.
  3. A dump occurs in the PDF_OUTPUT form.

Cause

When attempting to retrieve the form to print, it does not exist in the transaction 'Form Builder: Entry Point' (SFP) and therefore an error occurs in:

METHOD FORM_SELECT_FOR_GENERATE.

  SELECT SINGLE interface lastdate lasttime INTO (e_interface, e_lastdate, e_lasttime) FROM fpcontext
    WHERE name = i_name AND state = i_state.

  IF sy-subrc <> 0.
    RAISE EXCEPTION TYPE cx_fp_api_repository
      EXPORTING
        textid = cx_fp_api_repository=>object_not_existing
        name   = i_name.
  ENDIF.

ENDMETHOD.
