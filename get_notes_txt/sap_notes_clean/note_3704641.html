Symptom

After posting flows for a security transaction the portfolio cannot be changed within the transaction anymore in FTR_EDIT. 

Environment

  • Treasury and Risk Management (TRM)
  • SAP ERP
  • SAP ERP Central Component
  •  SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
    SAP Financial Supply Chain Management
  • SAP S/4HANA
  • SAP S/4HANA Finance
  • SAP S/4HANA Cloud Private Edition

Cause

For security transactions, once the cash flow has been posted only fields under Memos and Partner Assignment remain editable.

Resolution

Kindly execute a portfolio transfer with TPM82 or use fields under 'Memos' in FTR_EDIT as internal reporting attributes, as these fields are never locked by the system.
