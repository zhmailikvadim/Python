Symptom

After upgrading S/4HANA 

  • Error occurs while executing the currency translation task in BPC monitor.
  • Posting failed due to documentation type configuration issues.

Environment

  • Real-Time Consolidation

Reproducing the Issue

  1. Attempt to execute the currency translation task in the BPC monitor.
  2. Observe the error message indicating posting failure.

Cause

  • The table for RTC Document type is RTC_P100C.
  • Document Type 03 lacks a Business Transaction Type compared to other Document Types (00, 01, 02, 04, 05, 06, 07, and 08).

Resolution

  1. Access the RTCDT transaction.
  2. Re-save the Document Type: XX for the concerned model.
  3. Execute the process again in the Consolidation Monitor.
  4. Confirm if the issue is resolved.
