Symptom

You have a statement and it has different runtimes in two systems; e.g. PRD is slow, QAS is fast.

Environment

SAP HANA.

Reproducing the Issue

Run the statement in both systems and compare the runtimes.

Cause

The statement may use different plans and engines.

Resolution

  1. Find out the hash of the affected statement. More details in KBA 3322936 - How to Find The SQL Statement Hash for A SQL Statement String.
  2. Run data collector against the statement in both systems. More details in KBA 3444934 - How to execute "HANA_SQL_StatementHash_DataCollector" for a specific SQL.
  3. Examine the data collector results. More details below:
    • 3272558 - How to check the engine of a statement on SAP HANA.
    • 3327150 - SQL Statement Collection: "HANA_SQL_StatementHash_DataCollector" report for SAP HANA.
    • 3215082 - Truncated result for data collector statement - HANA_SQL_StatementHash_DataCollector.
