Symptom

In the processes "Receipt of Request Without Dependencies from DSO to MOS as DSO (44162, 55162)" (DE_B_MDC_REQ_ALLOC_N_DEP_DSO_MOS_DSO) and "Forwarding of Non-Settlement-Relevant Change from Authorized DSO to MP as DSO (11160/55160/44160)" (DE_B_MDC_N_SETTL_REL_FORWARD_AUTH_DSO_MP), the process description texts contain obsolete references to process IDs 55162, 55160, and 11160.

This issue is relevant for the market role: DSO.

Other Terms

DE_B_MDC_REQ_ALLOC_N_DEP_DSO_MOS_DSO, DE_B_MDC_N_SETTL_REL_FORWARD_AUTH_DSO_MP, UTILMD 44162, UTILMD 44160, process description, allocation request, non-settlement-relevant change

Reason and Prerequisites

The error occurred because the process descriptions in the configuration contained incorrect text.

Solution

Please implement the corrections of this SAP Note by importing BC set /US4G/3_0_DSO_SP08_NOTE_3728660(102-SP08), /US4G/3_0_DSO_SP07_NOTE_3728660(102-SP07), /US4G/3_0_DSO_SP00_NOTE_3728660(103-SP00) or the relevant Support Package (SP).

After implementing this SAP Note, the system updates the process descriptions to remove obsolete text. The description for process DE_B_MDC_REQ_ALLOC_N_DEP_DSO_MOS_DSO changes to "MDC: Receipt of Request Without Dependencies from DSO to MOS as DSO (44162)". The description for process DE_B_MDC_N_SETTL_REL_FORWARD_AUTH_DSO_MP changes to "MDC: Forwarding of Non-Settlement-Relevant Change from Authorized DSO to MP as DSO (44160)". You can verify the fix by checking that the process descriptions no longer contain the obsolete process ID references.
