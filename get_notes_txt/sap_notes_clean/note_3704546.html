Symptom

Approval for a payment batch is not possible for all the dedicated approvers, because it is pending on one specific approver

Environment

  • Bank Communication Management (BCM)
  • Financial Accounting (FI)
  • SAP ERP
  • SAP ERP Central Component
  • SAP enhancement package for SAP ERP 605 and higher
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA all packages
  • SAP S/4HANA Finance

Cause

This system behavior happens when one of the possible approvers reserves the workitem in SBWP

Resolution

  • Go to BNK_MONI and check the 'approval list' for the batch . ( Or use report  RBNK_COM_SHOW_BTCH_APPR_LST )
  • There will be one name and user ID of the person who reserved the related workitem

The user can simply make the workitem available for other approvers by following below steps:

  • Go to SBWP
  • Select Inbox--> Workflow
  • Add 'ID' field in 'Change layout'
  • Search for the workitem in question

  • Highligh the workitem and choose 'Replace' button
