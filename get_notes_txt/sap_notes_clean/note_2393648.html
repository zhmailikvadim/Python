Symptom

Bulgarian legal requirement is that region fields (VEIAV-BESTIREGI or VEIAV-URSPRSREG) are filled with three characters abbreviation of region. After applying SAP Note 2152350 or appropriate SP, both fields are filled with 2 characters abbreviation from table T005S, field HERBL (State of manufacture), if this field is maintained in the table. This is not correct. This SAP Note reverts the functionality back and after applying it, region fields are always filled with value of T005S-BLAND (Region (State, Province, County)).

Other Terms

Localization Bulgaria, Intrastat, Destination region, Region of origin, /BGLOCS/SD_RMIMST00, BESTIREGI, URSPRSREG

Reason and Prerequisites

SAP Note 2152350 and 2271234 is completely implemented.

Solution

As a general rule, SAP recommends that you install a solution by applying a Support Package. However, if you need to install a solution earlier, use the Note Assistant to implement the correction instruction. You can find more information about the Note Assistant in SAP Service Marketplace, under service.sap.com/note-assistant.
