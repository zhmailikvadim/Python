Symptom

  • You are operating SAP Extended Warehouse Management as of Release 9.5 (SAP EWM9.5)
  • You are evaluating the integration of SAP EWM9.5 with SAP S/4HANA Cloud Public Edition using the following scope items and communication scenario SAP_COM_0247:
    • 2VN 
    • 2VO 
    • 2VM 
    • 2VJ 
    • 2VK 
    • 2VL 
    • 2YL

Environment

  • SAP S/4HANA Cloud Public Edition 2608 and higher 
  • SAP Extended Warehouse Management 9.5
  • SAP EWM9.5
  • Decentralized Warehouse Management System (WMS) Integration for Public Cloud

      Cause

      • Communication scenario SAP_COM_0247 will be deprecated in the future.
      • As a result, no additional features will be delivered for this communication scenario.
      • The following scope items associated with the communication scenario SAP_COM_0247 will also be deprecated simultaneously: 
        • 2VN 
        • 2VO
        • 2VM
        • 2VJ
        • 2VK
        • 2VL
        • 2YL

      Resolution

      • Please consider one of the alternative deployments outlined in 2993430 - SAP S/4HANA EWM Best Practice Content - SAP for Me. 

      See Also

      • 3236184 - SAP_COM_0247 transfers custom fields to EWM with incorrect field names - SAP for Me
