Symptom

You run the integration for SV as Classic BOM. The DM request does not include a reference designator (partial payload) for a component that has subitems. Although the response shows success, the subitem data, such as quantity, is not updated.

Other Terms

PLMSI, PLM System Integration, Classic BOM, SV, Structure Version, Reference Designator

Reason and Prerequisites

Since the DM request does not include a Reference Designator, the processing classes for Classic BOM subitems do not run even when a BOM item has subitems.

Solution

Implement this SAP Note to correct the issue with incorrect partial maintenance of Classic BOM items.
