Symptom

What is SAP Integration Suite, Managed Gateway for Spend Management and SAP Business Network Self Help Feature? How do I run it?

Resolution

SAP Integration Suite, Managed Gateway for Spend Management and SAP Business Network Self Help Feature is a tool that consolidates key configurations for the Managed Gateway for Spend & Network into a single view, simplifying diagnostics and configuration checks. This allows for quick setup verification, reducing time and preventing oversights. 

What does it do?

The tool checks:

  1. If the right SAP add-ons and versions are installed
  2. If key technical components and customizations are present.
  3. If enhancements and workflow connections are active.
  4. If important integration parameters are set.
  5. If advisory notes or recommendations are available for your setup.

How do you use it?

  1. Run the program ARBCIG_SHCONFIGCHK in transaction code SE38.
  2. On the selection screen, choose your solution area and the type of transaction you want to check.
  3. Click to run the check. Review the results, which show what’s OK and what needs attention.
  4. You can export the results to a spreadsheet or print them for your records

Note: To run this program, you need to have the SAP role with authorization objects ARBCIG_PRG and ARBCIG_SPR assigned to your user in transaction PFCG. If you do not have this role, contact your SAP security or system administrator to request access.
