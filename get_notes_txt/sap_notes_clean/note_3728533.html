Symptom

The hierarchy nodes in the t-codes WMATGRP01, WMATGRP02, or WMATGRP03 are not sorted as expected.

Environment

•    SAP ERP
•    SAP S/4HANA
•    SAP enhancement package for SAP ERP
•    SAP enhancement package for SAP ERP, version for SAP HANA
•    Retail

Reproducing the Issue

  1. Create an article hierarchy with multiple nodes, or open an existing article hierarchy in t-code WMATGRP02/WMATGRP03.
  2. Check the nodes and observe that they are not sorted in order.

Cause

The value in the field WRF_MATGRP_PARAM-CDT_SORT_FLG determines how the system sorts the nodes in the hierarchy.

  • If the value is 01, the nodes are sorted by their description or name.
  • If the value is 0, the nodes are sorted by their number.

Resolution

To sort the nodes in the hierarchy, check the following configuration in the system:

Navigate to T-code SPRO → Logistics-General → Article Hierarchy → Edit Control Parameters.

  • In the 'Sorting' field, set the value to '1 Description' to sort the hierarchy nodes based on their descriptions.
  • Alternatively, leaving the 'Sorting' field blank will sort the hierarchy nodes by their node numbers.
