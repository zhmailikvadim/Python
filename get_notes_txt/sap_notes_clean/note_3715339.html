Symptom

  • An incompleteness error occurs in  an Export Declaration preventing users from proceeding with export declaration filing.
  • The incompleteness error message displayed is: "Ult consignee must have a value other than &."
  • Error /SAPSLL/CUS_ECC540 is displayed.

Environment

  • Global Trade Services / Logistics Services (SLL)
  • SAP Global Trade Services Edition For HANA

Reproducing the Issue

  1. Open relevant Export Declaration via the "Manage Export Declarations" app or via transaction /n/SAPSLL/CUS_SGL03E.
  2. Run Check Messages (for M0410 Export Declaration Option 2 (US)).
  3. Incompleteness check error is issued: "Ult consignee must have a value other than &." (/SAPSLL/CUS_ECC 540)

Cause

Often this can be due to the Message ID being incorrect.

Resolution

  1. This error can often arise due to incorrect assignment of the message ID to the corresponding message. 
  2. Access SPRO: SAP Global Trade Services, edition for SAP HANA-> Customs Management-> Communication Processes-> Messages for Communication Processes.
  3. For example, 
    Locate the Message ID AES2 for the message M0410.
  4. Change the Message ID from AES2 to AES4.
  5. Save the changes and retry the export declaration filing process.
  6. Ensure the correct message ID is active/assigned for the correct Message.
