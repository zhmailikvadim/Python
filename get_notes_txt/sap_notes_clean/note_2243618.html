Symptom

----------------------------------------------------------------------------------------------
SAP EM scenario adjustment for S4Hana MATNR with length 40
----------------------------------------------------------------------------------------------

With SAP S/4HANA, the length of Material Number (MATNR) data elements changed from 18 to 40 characters. To enable this change in SAP Event Management (SAP EM), existing customers migrating to SAP S/4HANA must consider the following areas:

  • Info and Control Parameters
  • System Parameters
  • Alert Framework
  • Extraction Function Modules
  • Business Warehouse (BW) Integration

As the interface from any supported SAP application to SAP Event Management is a generic interface which supports parameters in the application interface:

  • Control Parameter length 200 in the application interface -> length 60 chars in SAP EM
  • Info Parameter length 200 in the application interface -> length 255 chars in SAP EM

Control parameters from the application interface can be converted into system parameters within SAP EM. As system parameters always refer to an extension table in SAP EM which exists in the DDIC, you must ensure that these parameters are not reduced in length.

In the following sections we provide guidance on what needs to be checked if you have developed your own custom visibility processes.

Currently, the only visibility processes affected are those in which the following parameter is used:

-    Material Number (MATNR): length 18 -> 40 characters

 

You find a guide which helps you to identify the areas in the corresponding visibility scenario in the SAP Community Network: https://scn.sap.com/docs/DOC-69771

Please also check SAP Note 2238445 - Integration of Supply Chain Execution Platform Applications to SAP S/4HANA, on-premise edition

 

Other Terms

MATNR, S4HANA, Material Number length 40, Compatibility Mode, BW extraction

Solution

Ensure that you have implemented note 2193232 or installed the corresponding support package. Then use the guide (https://scn.sap.com/docs/DOC-69771) to check if you have to adapt your custom tracking scenarios as described in the guide.
