Symptom

Abap Development Tools (ADT) show an exception when trying to open the Function Module SWF_CPWF_CALL_IN_BG.

Other Terms

 F1, SWF_CPWF_CALL_IN_BG, ADT

Reason and Prerequisites

The Interface Contract of the Function Module SWF_CPWF_CALL_IN_BG contains the invalid value F1.

Solution

Install the corresponding support package or implement the assigned correction instruction.
