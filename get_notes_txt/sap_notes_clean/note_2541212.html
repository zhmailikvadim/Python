Symptom

How to schedule a Work Mode on instance level of SAP HANA Databases (HDB00.DBTypeForSAP.HDB.SystemHome.<host>)

Environment

  • SAP Solution Manager 7.1
  • SAP Solution Manager 7.1, version for SAP HANA
  • SAP Solution Manager 7.2

Resolution

  1. SAP Solution Manager 7.1
    Work Mode scheduled on Database is not inherited by any other component.
    This is the design in Solution Manager 7.1 and there won’t be further development in 7.1 to change this design

  2. SAP Solution Manager 7.2
  • For Solution Manager 7.2 SP03-SP05, look up the following KBA to learn how to schedule work modes for HANA DB instances: 2605186 - Defining work modes for HANA Database Instances in Solution Manager
  • Starting from Solution Manager 7.2 SP06, the database instances are included into the dependents of HANA DB, which means that WM on HANA DB on Instance Level will be available.
