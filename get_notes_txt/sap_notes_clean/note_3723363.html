Symptom

You have an issue with the SAP S/4HANA Data Migration content delivered for SAP S/4HANA 2023 that is related to an error with the content

This Note is valid for:

  • SAP S/4HANA - Migrate Your Data Migration Cockpit - "Transferring Data Using Staging Tables"

Other Terms

Data migration, Customer, Supplier, SIF_CUSTOMER_2, SIF_VENDOR_2, Migration cockpit.

Reason and Prerequisites

You have installed SAP S/4HANA 2023 (SP00-SP03)

You are using SAP S/4HANA migration cockpit

You are using the pre-delivered SAP S/4HANA Data migration content without any modifications

Solution

The Transport-based Correction Instruction (TCI) included with this SAP Note fixes the following issues listed in the table below. For detailed description on the issues see the linked SAP Notes.

Technical Name Type Issue SAP Note

SIF_CUSTOMER_2

SIF_VENDOR_2

 Migration Object Missing Content Not required

Be aware that the TCI only corrects the related objects of SAP delivered content. As a result, the generated migration objects will be updated automatically.

Note: If you have modified/copied your object, the correction is not done within your modified/copied object.

You may refer KBA 2543372 - "How to implement a Transport-based Correction Instruction".
