Symptom

The CL_FP_CONV_PDFA class is missing when implementing SAP Peppol note 3664984 - Electronic Invoice for Peppol Countries/Regions - Submission by E-Mail: Corrections.

Environment

  • SAP ERP
  • SAP S/4HANA
  • SAP Document and Reporting Compliance service

Reproducing the Issue

  1. Begin the implementation of Peppol note 3664984 in the system.
  2. Error "CL_FP_CONV_PDFA is unknown" during object activation.

Cause

The CL_FP_CONV_PDFA class is delivered by SAP_BASIS only, and if the error is presented during the SAP note implementation, then the SAP_BASIS is outdated.

Resolution

Upgrade the SAP_BASIS support package to a new version and implement the SAP note again.
