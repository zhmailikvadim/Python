Symptom

You use the program CORUPROZ or the transaction CO1P to process preprocessing records that were created by confirmations.

On the selection screen, the "List Display" indicator is missing in the "Output Criteria" section.

Other Terms

CO1P, CORUPROZ, P_OLD

Reason and Prerequisites

In the context of visual harmonization, the "List Display" indicator was hidden. However, since it is only an additional option, it is not necessary to hide the indicator.

Solution

Implement the source code corrections as described in the correction instructions.
