Symptom

You are trying to use W00017 app (Process Maintenance Order WebDynpro app) with ARIBA catalog and have setup the catalog configuration in 'Web Services and Settings'. You are able to use this catalog with the GUI app and Fiori app but for the WebDynpro app, the catalog configurations are not set.

Other Terms

S/4HANA, Procurement, IW32, W00017, ARIBA, Web Services, Settings

Reason and Prerequisites

You are following the ARIBA configurations from note 3619013 and the issue isn't occurring for the GUI/Fiori app.

Solution

Please check with the ARIBA team on whether the configurations have been setup properly. The 'Web Services and Settings' configurations in S/4 Procurement are used by the GUI and Fiori apps but with older apps, this is not always the case. This is currently a known limitation.

  • Function module called for the W00017 app: EAM_PROCMT_CREATE_OCI_SESSION and LTD_MMPUR_OCI_INTEGRATION
