Symptom

Custom fields are displayed in Journal Entry Analyzer app, but they are not displayed in Journal Entry Analyzer (New Version) app.

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Environment

SAP S/4HANA Cloud Public Edition

SAP S/4HANA Cloud 2602 and higher version

Cause

To boost performance and add new features, the Journal Entry Analyzer (New Version) uses a new underlying query. Custom fields are disabled by default.

Resolution

To use the fields in the new version, custom fields for source C_GLACCOUNTLINEITEMQUERY needs to be reactivated. To do this, in Custom fields app, for the custom field, click “Enable Usage”.
