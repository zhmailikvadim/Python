Symptom

When attempting to maintain data with transaction OPS7 or with transaction S_ALR_87005385, in a client with another client role as Production. Information appears: Client xxx has status 'not modifiable'. Data are not modifiable.

Environment

This KBA applies to all SAP Products based on NetWeaver and S4H.

Reproducing the Issue

  1. Execute transaction OPS7
  2. Information appears: Client xxx has status 'not modifiable'.

or

  1. Execute transaction S_ALR_87005385
  2. Information appears: Client xxx has status 'not modifiable'.

or 

  1. Execute transaction SPRO
  2. Press button SAP Reference IMG
  3. Execute one of these IMG activities
    Specify Applicants for WBS elements
  4. Information appears: Client xxx has status 'not modifiable'.

Resolution

Follow manual instruction of KBA 3138512 - SM30 | SPRO | Client XXX has status 'not modifiable' | Client role Customizing or Test
The object names is V_TCJ05

Keywords

Specify Applicant for WBS Elements, OPS7
