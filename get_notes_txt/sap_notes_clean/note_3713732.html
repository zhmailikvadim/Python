Symptom

This SAP Note provides you with corrections to the following reports:

  • EFD ICMS IPI (/TMF/EFD) report:
    • Register 0450: The system does not retrieve the text description correctly from the /TMF/D_COD_INF_D table for the Additional Information Code (COD_INF) field.
    • Register C191: The system does not generate this register correctly because the Run ID (P_RUN_ID) parameter mapping is missing.

Other Terms

Brazil, SPED, S/4HANA, EFD ICMS IPI, 0450, C191.

Reason and Prerequisites

This SAP Note is relevant only for Brazil.

You have implemented SAP Note 3713723.

Solution

After implementing this SAP Note, the updates listed below are made to your system:

  • /TMF/EFD report:
    • Register 0450: The system updates the CA_TXT_COMPL calculated field on the J_P_COD_INF node to retrieve the text description from the /TMF/D_COD_INF_D table as priority. As a result, the system now generates register 0450 correctly.
    • Register C191: The system adds the P_RUN_ID parameter to the EFD_C191_014 view and maps it correctly in the EFD_C191_013 view. As a result, the system now generates register C191 correctly.

As a general rule, SAP recommends that you implement the Support Package specified for your release. However, if you need to install a solution earlier, use the SAP Note Assistant to implement the correction instruction. You can find more information about the SAP Note Assistant in SAP Support Portal, under SAP Note Assistant.
