Symptom

For a participant contract of a hierarchy that is also attached to at least one other hierarchy as a participant, the system transfers error message BCA_CNSP_ACPOOL092 to the PPO order as many times as there are participant contracts in the hierarchy. Example: A hierarchy contains 5 participant contracts. A participant contract is attached to two hierarchies. The identical message is not issued only once, but five times.

Other Terms

S_MC_CHECK, CL_MCM_PP_PACKAGE_BCA_MCCH, BCA_CNSP_ACPOOL092, BCA_MCCH_RUN_PP

Reason and Prerequisites

There is a program error.

Solution

Implement the attached correction instructions or import the specified Support Package.
