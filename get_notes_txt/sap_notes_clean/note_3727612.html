Symptom

Despite dropDownEnabled=true on System Administration > Configuration > Search Selectors > Security Domain, the dropdown is not displayed for Programs

Environment

SAP SuccessFactors Learning

Reproducing the Issue

  1. Access Learning Admin >
  2. System Administration > Configuration > Search Selectors > Security Domain > set dropDownEnabled=true >
  3. Access System Administration > Configuration > Search Selectors > Programs > set dropDownEnabled=true >
  4. System Configuration > LMS_Admin > adminFioriHorizonEnabled=true >
  5. Programs > Add New > Search for Security Domain >
  6. Dropdown was not enabled

Cause

The Engineering team has confirmed that the program page in learning Administration is built on a legacy UI framework, that page has never supported dropdowns, and therefore, the setting dropDownEnabled=true will not be applicable to it. 

Resolution

The Product Engineers are working to improve the admin experience for setting up programs by updating the page to a new UI framework, but this doesn't currently have a date on the roadmap.
