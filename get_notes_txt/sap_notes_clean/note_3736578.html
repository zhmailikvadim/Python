Symptom

  • Dropdown attributes in a products configuration have multiple values selected when only one value should be selected.
  • Usage of simple and or formula rules in a product configuration resulting in more than one value being selected in a dropdown atribute.

Environment

SAP CPQ

Reproducing the Issue

  1. Configure a product with a rule, Simple or Formula rule
  2. Configure the product to have a dropdown attribute associated with it in its configuration.
  3. Choose a value, see configuration summary.
  4. Choose another value in same dropdown and notice in configuration summary the two values selected are present,
  5. This is incorrect as only one value should be the selected value.

Cause

Bug.

Resolution

Bug fix required and tentative release version of 2608.

Keywords

Bug, Dropdown, Selected, Value, Configure, Configuration, Product
