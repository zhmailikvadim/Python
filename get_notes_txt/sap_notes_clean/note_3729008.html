Symptom

This SAP Note is relevant only for the country version Germany.

You create an XML file for the Federal Statistical Office for hospital statistics for the year 2025 ff.

The Federal Statistical Office rejects the file with format errors in the field labels for major medical devices: PedCat and major medical devices: HiRise, since these are specified in uppercase letters in the XML file as MG_PEDCAT or. MG_HIRISE

Other Terms

RNSKHSTATV2020, MG_PEDCAT, MG_HIRISE, MG_PedCat, MG_HiRiSE.

Reason and Prerequisites

Program error in legal change

Solution

Implement the attached correction instructions or import the specified support package.

Execute the program RNSKHSTATV2020 again for the reporting year to get an XML file with the correct field labels MG_PedCat and MG_HiRiSE.
