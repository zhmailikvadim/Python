Symptom

In the process "Preparation and Waiting for Sending of Meter Reading (MOS MaLo View)" (DE_B_MET_PREPARE_SEND_MR_MOSMALO), when you select the option "Cancel Process" in the BPEM case raised by step 0550 (Exception Code: EX_MET_MR_RESULT_NOT_SENT), the system creates an unnecessary TDoc with BMID BM_CLOSE_CLOUD_PROC_FROM_BPEM that fails to be sent.

This issue is relevant for the market role: MOS.

Other Terms

MET, DE_B_MET_PREPARE_SEND_MR_MOSMALO, BPEM case, TDoc, BM_CLOSE_CLOUD_PROC_FROM_BPEM, EX_MET_MR_RESULT_NOT_SENT, /US4G/CL_EXCP_PROC_SOLVE_WIM, SEND_CANCEL_PROCESS_WTDOC, meter reading, MaLo

Reason and Prerequisites

The error occurred because of a program error.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the system no longer creates an unnecessary TDoc when you select the option "Cancel Process" in the BPEM case raised by step 0550. You can verify the fix by testing whether the unnecessary TDoc is triggered again when you select the option to cancel the PDoc in the BPEM case raised by step 0550.
