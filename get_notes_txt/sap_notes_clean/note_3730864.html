Symptom

What's the purpose of role SAP_GRDC_RDM_READ_ALL_JOB_RUNS? 

Environment

Group Reporting Data Collection

Reproducing the Issue

  1. Open the Run Data Mapping app in Group Reporting Data Collection.
  2. Observe the warning about the role SAP_GRDC_RDM_READ_ALL_JOB_RUNS and questions arise regarding its purpose and impact on viewing other users’ job runs.

Cause

The role SAP_GRDC_RDM_READ_ALL_JOB_RUNS is a newly introduced catalog role that governs visibility of run data mapping jobs started by other users;

Users without this role can only see their own jobs.

If some specific admin user need to check all Data Mapping job, this admin user must have this role based on help Define and Bundle Roles on SAP Business Technology Platform

 

Resolution

  1. Identify which users (for example, administrators or monitoring roles) need to view run data mapping jobs started by other users.
  2. Assign the catalog role SAP_GRDC_RDM_READ_ALL_JOB_RUNS to those users.
  3. Verify that these users can now see all relevant run data mapping job runs in the app.
  4. Note that not assigning this role does not stop or impact the execution of jobs; it only limits visibility to the user’s own jobs.
