Symptom

  • When attempting to use any application dropdown, the dropdown list becomes unstable and flickers, making it difficult to select different options.
  • This behavior is observed only on devices with touch screen functionality. 

Environment

  • Business Technology Platform - BTP
  • Cloud Portal Service on NEO - CPP on NEO

Reproducing the Issue

  1. Access the application.
  2. Navigate to the dropdown menu.
  3. Attempt to select options using a mouse-click.

Cause

CSS rule with "hover" since touch screens do not support hover events

Resolution

Remove the following CSS rule in the Theme Designer to resolve the issue

megaMenuV2.megaMenuV2Small.megaMenuContainer li:hover 

Our recommendation is:
Either remove this rule and additional rules related to hover
or
Define separate CSS rules for touch screens.
