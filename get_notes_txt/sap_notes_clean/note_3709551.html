Symptom

You maintain a Maintenance Order in transaction IW31 or IW32 in the Basic Order View mode. When you add a Technical Object (Equipment or Functional Location) the partners are not copied from the Technical Object into the Order.

Other Terms

IHPA, IW31, IW32, IW38, PARNR, Partner function, Partner determination, PM_PARTNER_DETERMINE

Reason and Prerequisites

Programming error. It is a prerequisite to maintain some partners for the Technical Object.

Solution

Apply corrections.
