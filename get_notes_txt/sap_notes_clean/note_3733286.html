Symptom

In the sequencing of complex component assemblies in takt-based planning, the queue name is generated incorrectly using the routing number instead of the Reference Operation Set (ROS) in the ROS-specific scenario, leading to non-sequential processing and inconsistencies.

Other Terms

JIT, JIT-S2P, Complex Components, queue, sequencing, REM, Routing, Reference Operation Set

Reason and Prerequisites

This is a program error.

Solution

Implement the correction instructions of this SAP Note or import the relevant Support Package to correct the system behavior.
