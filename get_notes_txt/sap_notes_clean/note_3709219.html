Symptom

When you access the Manage Journal Posting Policies app (App ID: F8539) in systems running SAP S/4HANA 2022, you notice that translations are missing on the screen.

Other Terms

Intelligent Journal Upload, IJU, Translation for SAP S/4HANA 2022, Manage Journal Posting Policies, F8539

Reason and Prerequisites

Translations are not available in the system for the Manage Journal Posting Policies app in SAP S/4HANA 2022.

Solution

After implementing this SAP Note, the system displays the correct translations on the screen of the Manage Journal Posting Policies app in SAP S/4HANA 2022.
