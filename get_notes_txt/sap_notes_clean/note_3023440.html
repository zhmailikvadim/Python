Symptom

When using transaction VA06 with the single filter "Missing Invoice from Vendor for Third-Party Delivery" it is possible if multiple documents are selected that some documents will be shown that do not match the criteria. If the document is specified individually with the "Sales Document" filter the problem does not occur.

Other Terms

 SD_OSO_MONITOR

Reason and Prerequisites

 There is a program error.

Solution

Please install the support package described in this Note. In urgent cases you may choose to install the supplied correction instructions.
