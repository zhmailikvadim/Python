Symptom

Customer object SCP1 name cannot be assigned to package. Object is in the name range reserved for customers. The object cannot be assigned, since it belongs to a different namespace.

Cause

Object SCP1 is in the name range reserved for customers. 

Resolution

Assign the object to a customer package whose name begins with Y or Z, or if you want to create the object as a test object, assign package $TMP.

Keywords

name range, TO140, TO 140, Editing is terminated,
