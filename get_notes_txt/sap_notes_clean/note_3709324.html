Symptom

The "Identifier" tagged under Manage Mobile Users

Reproducing the Issue

  1. Login to instance
  2. Search Manage Mobile Users
  3. Admin Center > Mobile Settings > Manage Mobile Users

Resolution

 The Identifier is a random, system-generated value for each activation. If a user activates the SF app on the same device 5 times, there will be 5 rows in that table.

Keywords

Identifier, Mobile, Manage Mobile Users
