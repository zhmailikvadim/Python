Symptom

  • It's not possible to create attachment in WB21 even through the GOS button is visible.
  • Attachment list option is not available in the GOS functionality.

Environment

  • Global Trading Management in SAP ECC
  • Global Trading Management in SAP S/4HAHA

Reproducing the Issue

  1. Run WB21 to create a new trading contract.
  2. Check GOS attachment for the new document.

Cause

The Generic Object service (GOS) function is not available during the creation of trading contract via WB21:

  • This occurs due to the absence of an object key, which is only generated for saved documents. 
  • When creating a document via WB21, the document number, which serves as the object key, is not assigned until the document is saved. The GOS requires this object key to associate uploaded attachments. 

Resolution

  1. Save the trading contract document in WB21.
  2. Access transaction WB22 or WB23 to create attachments using the GOS functionality.
  3. Once attachments are created, the Attachment List option becomes visible. 

Keywords

WB22, WB23, WTEW, Create Note, Create External Document
