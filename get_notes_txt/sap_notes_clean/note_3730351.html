Symptom

  • When the goal plan and performance form are not synchronized (auto-sync disabled), deleting a goal from the goal plan removes the delete option (trash icon) for that goal on the performance form in the Latest Form Experience.
  • In the Legacy Form UI, the delete icon for the same goal remains available.

Environment

  • SAP SuccessFactors Performance Management
  • SAP SuccessFactors Goal Management

Reproducing the Issue

NOTE: Prerequisites - Prior form launch, the "Synchronize objectives from an objective plan and a review form" is disabled in Goal section. Latest Form Experience is enabled. 

  1. Open an active performance form that contains goals pulled from a goal plan.
  2. Delete one of the goals from the goal plan after the form is launched.
  3. Return to the performance form, locate the same goal, and observe that the goal cannot be deleted. 
  4. (Optional) Switch to the legacy form UI and observe that the delete icon is present.

Cause

This is a known issue. 

Resolution

This issue in tentatively planned to be fixed with 2H26 release. 

For release timelines, please review SAP SuccessFactors Product Release & Road Map Information.

WORKAROUNDS: 
  • Disable Latest Form Experience
  • Switch the performance form UI to the legacy form experience, open the form, delete the non-existing goal from the form, then switch back to the latest form experience.
  • Enable "Synchronize objectives from an objective plan and a review form" in Manage Templates prior form launch.
