Symptom

You receive an error message during submission of FACe for country of Spain in App : eDocument Cockpit.

Tax authority returned error: Error code: 411

Description: No existe o inactiva la Oficina Contable asociado

Environment

S/4HANA Cloud

Cause

This issue is normally related to CPI certificate.

Resolution

  1. Go to the eDocument App
  2. Go to History and download the file type ‘SIGNEDDOC’
  3. Upload this ‘.xsig’ file in the Tax Autority portal link and check the validation errors.
  4. You will see list of errors. In this case, please check your certificate.

Keywords

Spain, B2G scenario, eFactura, eDocument, FACe, Tax authority, Error, 411
