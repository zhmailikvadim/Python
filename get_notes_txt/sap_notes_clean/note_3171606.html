Resolution

Issue

Due to Purchasing (Procurement) Unit assignment, this user could not see any data in reports. He imported new data for "Responsible User Assignment" (in Data Import/Export), but this did not solve the issue.

Cause

The changes made in "Responsible User Assignment" do not take effect immediately in reporting. This data, loaded into reports using the regular Ariba Spend Management (ASM) Data Load (Administration > Reporting Manager > Schema Manager), requires completion of the next ASM Data Load to finish. Wait for it to finish.

Additional Information

For further troubleshooting, you can check whether the correct data appears in the sourcing side, analysis table, ariba.analytics.dimension.UserOrgAccessMap.
