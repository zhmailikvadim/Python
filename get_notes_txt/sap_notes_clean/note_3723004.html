Symptom

If you transfer and save a medication before admission into an order of the "Administration", "Dispensing" or "Prescription Print" purpose, the system currently displays a warning window, which informs the user that the original order is ended and the patient does not receive any medication doses.

This warning window does not offer any added value in the scenario described and is also incorrect in the process of the conversion to inpatient medication.

Other Terms

Smart Medication

Reconciliation

Renewed Order

Reason and Prerequisites

Program error

Solution

As a result of this SAP Note, this warning window no longer appears if you transfer a medication before admission into an order of the "Administration", "Dispensing", or "Print Prescription" purpose, and save this order.

If you change an active order of the "Administration" purpose and save the resulting subsequent order, the system still displays this warning window.

Import the specified Support Packages to solve the problem. You can implement the corrections in advance.
