Symptom

By system design, the fields 'Journal Entry Date' and 'Posting Date' are always filled with the dates in the app "Repost Line Items - Cost Accounting (F5549)". 
It is requested the default value to be blank for fields 'Posting Date' and 'Journal Entry Date'. 

Environment

SAP S/4HANA Cloud Public Edition

Cause

This is by system design. 

Resolution

The feature request ID: 370574 is raised for this topic. Vote this request if the same feature is required.
