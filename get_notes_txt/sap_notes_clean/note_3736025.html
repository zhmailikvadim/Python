Symptom

User creates a Lean SES using the App 'F2027 - Manage Service Entry Sheets LEAN', the Lean PO contains a service  (EKPO_PRODUCTTYPE '2' Service)

Environment

SAP S/4HANA

SAP S/4HANA Cloud Private Edition

Cause

Validation checks performed for Service Item (EKPO-PRODUCTTYPE = '2' ) in CL_MM_PUR_SES_HELPER method CHECK_PO_ITEM_VALID   -> CL_VAL_SES_ITEM_SVC_PO method CHECK_PO_REFERENCE

Resolution

*       only PO items where a goods receipt and an invoice receipt are expected (EKPO-WEPOS = X, EKPO-REPOS = X)     Message MM_PUR_SES160

*       No PO items that doesn't have GR-based IV  (EKPO-WEBRE)      Message MM_PUR_SES173

*       No PO items that contain delivery costs (entry in table prcd_elements with condition category 'B')      Message MM_PUR_SES207

*       No PO items that are blocked or deleted (EKPO-LOEKZ     Message MM_PUR_SES188
