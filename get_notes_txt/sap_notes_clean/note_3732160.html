Symptom

I want to store Service Entry Sheet (SES) and Invoice attachments received from SAP Business Network through Managed Gateway For Spend & Network in SAP ECC or SAP S/4HANA using ArchiveLink.

Environment

SAP ECC
SAP S4Hana

Cause

Attachments received from SAP Business Network are processed using class CL_ARBCIG_ATTACHMENT and are stored using GOS or DMS in SAP ECC or S4Hana.

Resolution

ArchiveLink storage for attachments received through Managed Gateway For Spend & Network is not supported in standard SAP design.

Attachments are handled as follows:

  • The system processes attachments using class CL_ARBCIG_ATTACHMENT
  • The system stores attachments using GOS or DMS in SAP ECC or S4Hana.

If you require ArchiveLink-based storage you must implement custom development on the SAP ECC or S4Hana side.

Keywords
