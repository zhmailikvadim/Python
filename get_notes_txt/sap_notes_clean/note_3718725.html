Symptom

This SAP Note provides you with adjustments in the Inflation Accounting for Asset Accounting solution for the Inflation Management of Fixed Assets (J_1AA001) report. It addresses the incorrect calculation of the Remaining Net Book Value when the asset has a residual value.

When you run the J_1AA001 report for an asset with a residual value, the system calculates using the latest revaluation amount instead of the amount for the whole period.

Other Terms

Inflation Management, INMA, Inflation, Inflation Accounting, Inflation FI, Asset Accounting, J_1AA001, J1AI, J_1AAINFL_REVERSE, Residual Value, Remaining Net Book Value.

Reason and Prerequisites

You have implemented previous SAP Notes listed in 3502825 - Inflation Accounting for Asset Accounting and General Ledger Accounting: Overview Note.

Solution

After implementing this SAP Note, the system considers the revaluation amount for the whole period when you run the J_1AA001 report for the residual value scenario.

SAP recommends that you implement a solution by using a Support Package. If you implement a solution earlier, use the Note Assistant to implement the correction instruction. You can find more information about the SAP Note Assistant on the SAP Help Portal, under SAP Note Assistant.

To implement the solution, perform manual activities and implement the correction instructions that are valid for your release.
