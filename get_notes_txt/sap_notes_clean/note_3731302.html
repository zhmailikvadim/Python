Symptom

As described in the regulatory change manager under Discontinuation of EFD-Contribuições 2026.

This SAP Note provides you with relevant system changes to address Technical Note 011_2026 - Discontinuation of EFD-Contribuições, published on February 03, 2026.

For more information, see also Nota Técnica 011_2026 - Descontinuidade da EFD-Contribuições - Orientações para os contribuintes.pdf. Other Terms

Brazil, SPED, S/4HANA, EFD CONTRIBUIÇÕES, PCO, LEGAL CHANGE, C500, C501, C505, C509, C600, C601, C605, C609.

Reason and Prerequisites

This SAP Note is relevant only for Brazil.

You have SAP Note 3725400 applied in your system.

Solution

After you implement this SAP Note, you can report the PCO (/TMF/PCO_REPORT) report according the changes in Technical Note 011.

As a general rule, SAP recommends that you implement the Support Package specified for your release. However, if you need to install a solution earlier, use the SAP Note Assistant to implement the correction instruction. You can find more information about the SAP Note Assistant in SAP Support Portal, under SAP Note Assistant.

Note that the documentation of certain objects is delivered with the corresponding Support Package only.

List of Affected Objects:

  • PCO_C500_006, PCO_C501_005, PCO_C505_005 and PCO_C509_005 views
    • The CA_COD_MOD calculated field was created to convert the value in the COD_MOD field from 75 to 29.
    • The CA_COD_MOD calculated field was created to convert the value in the COD_MOD field from 76 to 28.
  • PCO_C600_005, PCO_C601_005, PCO_C605_005 and PCO_C609_005 views
    • The CA_COD_MOD calculated field was created to convert the value in the COD_MOD field from 75 to 29.
    • The CA_COD_MOD calculated field was created to convert the value in the COD_MOD field from 76 to 28.
