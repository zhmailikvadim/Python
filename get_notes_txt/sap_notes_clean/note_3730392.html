Symptom

The CostCenter included in the Import Requisition request payload is not set when the requisition is created in SAP Ariba Buying.

  • CostCenter field is set as Null in the Requisition created.
  • Import Requisition is through webservice/CIG integration.
  • Applicable only for realms integrated with SAP ERP systems (VSAP variants).
  • Scheduled task SetUserSetBitForRequisitionImportPull is already run.

Cause

The CompanyCode is missing in the payload for the CostCenter section. The standard design expects the CompanyCode inside the CostCenter structure, without it, the value is not set.

Resolution

The structure for CostCenter should be passed with CompanyCode as below to have it set on the Requisition.

 <urn:CostCenter>
           <urn:CompanyCode>
                 <urn:UniqueName>CCXXXXX</urn:UniqueName>
            </urn:CompanyCode>
      <urn:UniqueName>XXXXX</urn:UniqueName>
</urn:CostCenter>
