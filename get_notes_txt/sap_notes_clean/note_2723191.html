Symptom

User wants to see only controls that have performance plan assigned in Manual Control Performance Results report.

Environment

  • SAP GRC Process Control 10.1
  • SAP GRC Process Control 12.0

Reproducing the Issue

1. Launch NWBC and navigate to Assessments work center.

2. Open Manual Control Performance Results and check that the whole list of controls is displayed on the report.

Cause

 This is the standard report behavior.

Resolution

As explained on KBA 2721254 - Difference between Manual Control Performance Details and Results reports, the standard behavior of the Manual Control Performance Results, is to show which controls have been performed in the, organization structure. This way, the whole control list that belongs to each organization is displayed and user can track this information.

However, customer can customize the standard behavior by changing the walking strategy on the following way:

1.Go to SPRO->Governance, Risk and Compliance->Reporting->Maintain Report Configuration

2.Select Report GRPC_CTRL_PERF and navigate to Walking Strategy

3.For Position 50, in column Store Node-> change this to 'Do not Store'.

4.Save the changes and re-run the report.
