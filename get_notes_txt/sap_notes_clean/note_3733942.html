Symptom

Audit details for Agency user access changes in SuccessFactors

Require information on users who create/enable/disable/modify Agency user access, along with the following details:

  • Name of the user who performed the action
  • User ID
  • Date and time when the access was created or updated

Environment

SAP SuccessFactors Recruiting Management

Resolution

It is not possible to have audit report for agency users in successfactors.

You can only run an Adhoc report that shows the agency ID to which a user is linked and there is no audit logs available for the agency users. https://me.sap.com/notes/2837392

See Also

https://me.sap.com/notes/2837392
