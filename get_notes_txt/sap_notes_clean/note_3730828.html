Symptom

User is creating or editing a Brazil Nota Fiscal NF-e document.
The value of the Benefit code (CBENEF) is being deleted on J1B1N and J1B2N after saving the document.

Environment

  • SAP ERP
  • SAP S/4HANA

Cause

Wrong XML version being used.

Resolution

The mentioned field CBENEF is related to XML 4.0. Therefore, check whether your document was set for using previous versions like 2.00 or 3.10, hence the field is not available for entering in the overview and will not work properly even if you enable it in the item details. Due to this, the logic clears the values even if they are determined.
To solve this issue, please use XML 4.0 and it should solve your issue.
