Symptom

During Load profile allocation there are two different checks:

  • If load profile allocation required - Load profile allocation is not determined based on settlement procedure. 
  • If configured load profile is valid profile  - Only synthetic profiles were allowed for load profile allocation.

Other Terms

/APEU, MCM, Load profile, SLP, RLM, Load profile allocation,  CHECK_SLP_INPUT, CHECK_SLP_ASSIGNMENT

Reason and Prerequisites

Program Error.

Release 2602.

Solution

Please implement the attached the correction instruction.

After implementation of correction,

  • If load profile allocation required - Load profile allocation is determined based on settlement procedure. If load profile allocation configuration matches with Settlement procedure, then only load profile allocation will happen. 
  • If configured load profile is valid profile  - Restriction for only SLPs are removed.
