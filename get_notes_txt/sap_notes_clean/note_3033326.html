Symptom

To close an auditable item, a user first chooses the Close button and then Release. There is currently no Submit step between the two actions. In the scenario where submitting auditable items is enabled, the Submit button should be availabe after closing an auditable item so that the user can submit it to another person who will perform the Release action.

Other Terms

Auditable Item, Submit, Close,

Reason and Prerequisites

 In the scenario where the Submit step is enabled in the workflow of working on auditable items, the Submit button should be availabe after closing an auditable item so that the user can submit it to another person who will perform the Release action.

Solution

Apply this note.
