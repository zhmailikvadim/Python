Symptom

In SBPA, the irpa_excel SDK provides an activity named Set Sensitivity, which allows users to assign Microsoft Sensitivity Labels to Excel documents.

However, users observe that similar functionality is not available for other Microsoft Office applications, such as:

  • Microsoft Word
  • Microsoft Outlook
  • Other Microsoft Office products

As a result, users are unable to set Microsoft Sensitivity Labels for non‑Excel documents using SAP Intelligent RPA SDKs.

Environment

SAP Build Process Automation

Cause

The 'Set Sensitivity' activity is currently implemented only in the irpa_excel SDK. Support for Microsoft Sensitivity Labels is limited to Excel documents based on the current SDK design and supported APIs.

Resolution

  • At present, SAP Build Process Automation supports setting Microsoft Sensitivity Labels (Set Sensitivity) only for Excel documents through the irpa_excel SDK. There is no equivalent activity available for Word, Outlook, or other Microsoft Office applications. This behavior is by design.

  • Users can submit a request to have this feature included in the product for future use, as outlined in the referenced KBA:
    3463121 - How to Submit Feature Request Ideas for SAP Build Process Automation
