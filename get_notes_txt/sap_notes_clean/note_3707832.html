Symptom

You have an employee whose employment ended during the year. The following year, the employee receives a supplementary payment that affects all codes of block 44. In the payroll function FCUMC, the system incorrectly sets the last accumulation indicator for the last active period and the last period of the previous year for all codes in block 44, except code 001 (balance of apprenticeship levy). This behavior causes inconsistencies in the table T5FDSNCUMCO2.

Reason and Prerequisites

This SAP Note addresses a program error.

Solution

After implementing this SAP Note, the system sets the last accumulation indicator only for the matching accumulation number in the last period of the previous year. This applies to all codes in block 44, except code 001 (balance of apprenticeship levy).

These corrections are available with the HR Support Packages referenced below.
