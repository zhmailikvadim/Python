Environment

SAP Focused Run

Reproducing the Issue

  1. Check the metric status in FRUN System Monitoring UI.
  2. Increase log severity of Heartbeat application:
    1. Open NetWeaver Administrator (http://<host>:<port>/nwa) -> Troubleshooting -> Logs and traces -> Log Configuration -> Next to "Show" choose "Tracing Locations".
    2. Navigate in the Location tree to : "com.sap.engine.heartbeat.ServerGRMGApplication".
    3. Set tracing location "com.sap.engine.heartbeat.ServerGRMGApplication" to Debug severity.
    4. Check the errors in developer trace files (defaultTrace*.trc)

Resolution

  1. Apply the patch from SAP note 2620218 and 2658857.
  2. Execute the manual activities from SAP Note 2416119 - Improved security for outgoing HTTPS connections in SAP NetWeaver.
  3. If the issue persists, follow steps 3-16 in 2658857.
  4. For further queries regarding 2620218, 2658857, and 2416119, contact the BC-JAS-ADM-MON team.
