Symptom

In the GPKE process, the system fetches incorrect device data because the process date is missing. This issue affects the process with Process ID DE_B_GPKE_NIN_DSO.
This issue is relevant for the market role: DSO.

Other Terms

DE_B_GPKE_NIN_DSO, DSO, device data, process date, NIN, GPKE, device time slices

Reason and Prerequisites

The error occurs because of a program error that causes the system to fetch device data without considering the process date.

Solution

Implement the corrections of this SAP Note or import the relevant support package.
After implementing this SAP Note, the system fetches the correct device data based on the process date. You can verify the fix by running the process and checking that the device time slices are correct
