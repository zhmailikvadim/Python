Symptom

You are reversing an OTC deal with transaction FTR_EDIT. The system throws an exception. The exception is assigned to class 'CX_SY_REF_IS_INITIAL' and was not caught in procedure "CHECK_TRANSACTION" "(METHOD)".

Other Terms

THX, Hedge Management, Hedge Accounting, Treasury, FTR_EDIT

Reason and Prerequisites

Program Error

Solution

Implement the note or the corresponding support package.
