Symptom

During the posting of master agreement to change the existing agreement, the period profile maintained in the master agreement is not copied to the target agreement when the ‘Copy Periods’ checkbox is enabled in the postings popup screen.

This note ensures that the period profile is successfully copied to the target agreement.

Other Terms

Master agreement Postings, /IRM/IPBB_AGREEMENT_CHANGE, it_fields, /IRM/IPPQMAM, FPRPF, CB_CPAR

Reason and Prerequisites

This is caused by program error

SAP Note#3732055 is available for ECC Release

Solution

Please implement the correction instructions attached. These corrections are only valid if you have applied the Support Packs.

Release V/4 - 2021, 2022, 2022 V2, and 2023

 Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object.
