Symptom

I am a supplier who is unable to enable the Reason Code Upload Connector in Configuration Manager

Cause

The Reason Code Upload is only for Buyer Companies

Resolution

  1. It is not possible to set up the Reason Code Upload for Suppliers
  2. You will need to set up the Reason Codes through the UI
