Symptom

When you change the values of specific fields (e.g., Inspection Duration) within an inspection rule in Extended Warehouse Management (EWM), the system does not generate a change document. Consequently, you cannot track these modifications via the technical table CDHDR/CDPOS.

Other Terms

/SCWM/QRSETUP, QIE Inspection Rule, decentral

Reason and Prerequisites

Program error.

Solution

 Implement correction instruction.
