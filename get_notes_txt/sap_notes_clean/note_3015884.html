Symptom

This SAP Note provides a new version of the Plug-In class for the de-installation of software component version SCM_BASIS 7.13 (or higher), especially in the context of SAP EWM 9.3 (or higher) installed as Add-On to SAP ERP. The software component SCM_BASIS is contained in the SAP EWM product.

The Plug-In class is used in SAINT/SPAM from version 63 on.

Other Terms

Uninstall, deinstall, SAINT, SCM Basis, Extended Warehouse Management, 940

Reason and Prerequisites

Add-on deinstallation

Solution

Check information in SAP Note 2407103 "Condition technique and pricing enhancements for add-on De-Installation" and implement the correction instructions as mentioned below.
