Symptom

Check failure below happens during ATC run:

Check only partially executable, due to missing prerequisites

    Usage of APIs    BC-DWB-CEX-CCM

        objectClassifications_3TierModel.json is deprecated. Please use objectClassifications_SAP.json

Environment

SAP Netweaver Release >= 758

Reproducing the Issue

  • run ATC check with the check variant which has check option "Usage of APIs" enabled

Resolution

  1. Execute transaction code SCI
  2. Enter the used check variant and click Change button
  3. Click Attributes icon of check option "Clean Core" -> "Usage of APIs"
  4. Change the URL of "Classic API Data Source" from url(https://raw.githubusercontent.com/SAP/abap-atc-cr-cv-s4hc/refs/heads/main/src/objectClassifications_3TierModel.json)
    to url(https://raw.githubusercontent.com/SAP/abap-atc-cr-cv-s4hc/refs/heads/main/src/objectClassifications_SAP.json)  

See Also

  • SAP Note 3635743 - Rename ATC Check Category from "3-Tier Model" to "Clean Core"
