Symptom

You execute the foreign currency valuation in Contract Accounts Receivable and Payable FPW1. You receive error SG105 "Enter rate & / & rate type & for & in the system settings" when you use several company codes with different valuation methods.

Other Terms

SAPFK100, SG 105, 105(SG)

Reason and Prerequisites

This is a program error.

Solution

Implement this SAP Note or import the specified Support Package.
