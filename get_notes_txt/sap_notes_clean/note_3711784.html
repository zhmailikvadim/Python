Symptom

When Process Multiple Material Services is provided with storage location on destination which is not extended to plant and storage location combination for material then Production Service does not prompt with error and allows the execution.

Other Terms

ACM; Agricultural Contract Management; Production Service; Multiple Material; Simulation; Validation; MARD.

Reason and Prerequisites

Programming error.

Solution

Validation added to check valid material/palnt/storage location combination in MARD table for destination storage locations of Multiple Material Services.

Implement the attached Correction Instruction(s) associated with this SAP Note.
