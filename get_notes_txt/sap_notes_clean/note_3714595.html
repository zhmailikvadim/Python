Symptom

In the application Transportation Cockpit: Profile and Layout Sets, while deleting a requirement document in the execution document hierarchy, system raises a popup with the warning message "Do you really want to delete the Road Freight Order?"

Other Terms

Freight Unit, Road Freight Order Hierarchy, Air Booking Hierarchy, Ocean Booking Hierarchy, Rail Freight Order Hierarchy, Transportation Unit

Reason and Prerequisites

 Program error

Solution

 Implement the correction instruction or upgrade to relevant Support Package.
