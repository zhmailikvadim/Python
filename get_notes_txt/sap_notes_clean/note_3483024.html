Symptom

In case that with a first run the "Post Components" batch job could not post (due to configuration issue) the related material documents for the components, after correcting the related configuration, in the second run the missing material documents for components are not anymore created.

Other Terms

Post Components, failure handling

Reason and Prerequisites

Program error

Solution

History:

Version 1 of 17 June, 2024: New creation

Install the code correction.

Description of Correction:

After new execution of the "Post Components" and the correction of the related configuration issue, the material documents for components which failed in the previous run are succesfully posted. Additionally, in the application log of "Post Components" there is a success message displayed for each material documents posted for the components.

Functional Dependencies:

There are no functional dependencies.

Manual Changes:

There are no manual activities.
