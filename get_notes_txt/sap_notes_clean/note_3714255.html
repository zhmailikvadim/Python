Symptom

This SAP Note describes the steps to update the obsolete data element used in the authorization field BL_CAT of the authorization object F_BL_BLCAT. The data element BCA_DTE_BL_CATEGORY has been replaced with BCA_DTE_BLNG_INVC_BILLCATEGORY.

Other Terms

SU20, SU24, F_BL_BLCAT, Billing Authorization Object

Reason and Prerequisites

Program Error

Solution

Import the relevant Support Package for your release.
For release S4TRBK 2.0, S4TRBK 1.0 and Banking Services 9.0 please carry out the Manual Activities given below.
