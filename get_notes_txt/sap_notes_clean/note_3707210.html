Symptom

Unclear if quality management (QM) process or feature is supported on an EWM managed storage location and what are the release requirements for the EWM system as well as the QM system.

Other Terms

SAP EWM, Quality Management, QM, EWM-QM integration

Reason and Prerequisites

You want to use a QM process or feature on an EWM storage location. You want to know if the feature is supported with your current EWM as well as QM system.

Solution
