Symptom

You want to control whether the Messages conversation popup is displayed or hidden in sourcing events.

Resolution

The parameter Application.AQS.RFX.EnableMessageConversationPopUp is a self-service configuration parameter that controls the visibility of the Messages conversation popup in the user interface.

The behavior of this parameter is as follows:

  • When the parameter is set to Yes, the Messages conversation popup is displayed in the UI.
  • When the parameter is set to No, the Messages conversation popup is hidden.

The default value of this parameter is Yes, which means the Messages conversation popup is displayed by default.
