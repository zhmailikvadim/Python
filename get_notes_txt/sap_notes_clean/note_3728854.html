Symptom

 To process deferred income in JMHZ, it is necessary to create WTs for: (1) compensation for an uninsured bonus; (2) compensation for unfair termination of employment; and (3) compensation for ELDP-related unfair termination of employment.

Other Terms

Payroll, WTs /3OE, /3ON, /3OP, M5OP, value 8 in proicessing class 47, values ' ', 0, 1 in processing class 62, rule CZ14

Reason and Prerequisites

  • This SAP Note is relevant only for the Czech Republic.

Solution

1. To enter the uninsured bonus for ZMR/DPP type from deferred income of an inactive employee with DPP or ZMR, use WT /3ON from IT0015.

2. To enter compensation for unfairly terminated employment, use WT M5OP. The amount from WT M5OP is included in WT /3OP in the payroll.

3. For entering information about compensation for invalidly terminated employment for ELDP use WT /3OE.


Rule CZ14 was modified for the calculation, and new values ​​were created for processing classes 47 and 62.

Note: The method of entering data WTs /3OE, /3ON, M5OP for deferred income is part of the SAP documentation for JMHZ.
