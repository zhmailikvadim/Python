Symptom

CoreFix 1 for Developer 4.8.1 fixes the following problems:

  1. Flow Service Editor:
    The "Pipeline" tab disappears from the lower half of the Flow Editor.

  2. .NET Client Library (DotNetClientLibrary.dll):
    The Service invocation fails with a System.ArgumentException without an obvious reason.

Reason and Prerequisites

The symptoms above have been caused by the following issues within previous CoreFixes or the initial shipment of Business Connector Developer 4.8.1:

  1. The  "Pipeline" tab disappears, when deleting two or more Flow steps that do not need the "Pipeline" tab due to a wrong update element in the code.  As a workaround, a UI refresh could be triggered, e.g. by simply switching between Services.

  2. The exception is thrown, if in the HTTP communication between a .NET Client application and the Business Connector Server an already existing Cookie is updated.

In addition, CoreFix 1 provides the following enhancements:

  1. The Certificate Toolkit now allows the generation of private keys of size 4096 bits.
  2. The Certificate Toolkit now allows adding Subject Alternative Names (SAN) to a certificate signing request.
  3. The Certificate Toolkit can now extract the contents of PKCS#12 files (certificate files of format .p12 and .pfx). The certificate chain and the private key contained in the PKCS#12 file are saved as separate files in DER format, so that they can be used directly in the SAP Business Connector Server, e.g. for outbound HTTPS, for HTTPS Listeners, or for WebSocket RFC.

Solution

You can find the most recent CoreFix for Business Connector Developer 4.8.1 in the "Software Updates" section of the Business Connector download page on SAP Support Portal (https://support.sap.com/sbc-download?anchorId=software-updates). Note that all CoreFixes are cumulative, meaning each release contains all fixes and enhancements of the previous releases. For this reason, when upgrading, you only need to install the latest one.

After you have downloaded the BCD481CoreFixP_1-20007203.zip file, you can install it by unpacking it into the <developer> directory. In this case, <developer> stands for the installation directory, for example, C:\sapbc481\Developer on Windows or /opt/sapbc481/Developer on Linux and AIX.

In addition, refer to further installation instructions in the CoreFixReadme.txt contained in the zip file.

After you have started the Business Connector Developer, you can verify that the installation was successful by choosing "Help -> About Developer" from the menu. The version should now be displayed as "Version 4.8.1 (1042 + CoreFix 1)".
