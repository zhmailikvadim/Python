Symptom

When Forecast Mode is set for a document via VFX3 transaction, and multiple invoices are being posted to accounting in sequence, an error in posting in one particular invoice causes the Forecast Mode to be applied to consecutive invoices as well incorrectly.

Other Terms

S4; CM; Commodity Management; VFX3; Forecast mode; Invoice; Accounting; Invoice anticipation

Reason and Prerequisites

Program error.

Solution

Please implement the correction instruction(s) associated with the note.
