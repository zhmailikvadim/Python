Symptom

While opening report data via broadcast you are not able to use the "Report navigation"

Environment

  • C4C, ByD

Reproducing the Issue

  1. Go to Business Analytics work center
  2. Go to Broadcasts View
  3. Open the broadcast and run it
  4. Open broadcast throw mail
  5. There is no option for report navigation

Cause

Broadcasts doesn't support report navigation directly

Resolution

This is system limitation.

Broadcasts doesn't support report navigation directly from the mail. Broadcasts are generally used to view the data from report and they don't support to navigate to other work centers.

We can only access report navigation throw Design reports.

If this requirement is needed in further releases you can create a ticket in influence portal:https://influence.sap.com/sap/ino/#/campaign/1867
