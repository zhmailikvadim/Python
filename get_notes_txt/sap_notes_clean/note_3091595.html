Symptom

Running Landing Page, you realize that icons are missing in some lanes, like Search.
For example, if an employee is searched, you'll get an error on HTTPWatch/F12 trace regarding 'Person.gif':

Trace errors:

  1. 404 error
  2. 403 error

Environment

  • Human Capital Management (HCM)
  • HR Renewal 
  • SAP enhancement package 6 for SAP ERP 6.0
  • SAP enhancement package 7 for SAP ERP 6.0
  • SAP enhancement package 6 for SAP ERP 6.0, version for SAP HANA
  • SAP enhancement package 8 for SAP ERP 6.0

Reproducing the Issue

  1. Landing page is opened using an url similar to the following one:
    http://[hostname:port]/sap/bc/ui5_ui5/sap/arsrvc_suite_pb/main.html?page=SPB_LANDING_PAGE
    OR
    http://[hostname:port]/sap/bc/ui5_ui5/sap/arsrvc_suite_pb/main.html?page=HR_SELFSERVICES
  2. If you open any lane with images, like Search, you can check that images aren´t displayed.

Cause

Relevant PAO business functions or services aren´t activated on the system.

Resolution

  1. Please make sure that business functions from HCM_PAO_CI_1 to HCM_PAO_CI_9 are activated on your system.
  2. If business functions are already activated on your system, please confirm that icon paths haven´t been changed on launchpad role HRPAO, instance ACTIONS.
    You could find standard icon paths on your client 000, tcode LPD_CUST: "/sap/public/HRRenewal/PB/OrgUnit_32.png", "/sap/public/HRRenewal/PB/Person_32.png", and etc..
  3. Please make sure the services in the trace is activated in Tcode SICF, like service HRRenewal in this example(403 error in trace).
