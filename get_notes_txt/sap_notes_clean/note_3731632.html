Symptom

  • Supplier location data missing when imported prior to the Supplier
    • Error reported for supplier integration in downstream process.
    • Supplier appears in the csv file, but integration still reports an error.
    • Error Message: supplier is not specified for supplier location

Environment

  • Product: SAP Ariba Buying
  • Core Administration Supplier Management

Reproducing the Issue

  1. Import supplier location data before importing the corresponding supplier master data via Data Import/Export.
  2. Observe that downstream integration reports an error for the supplier with message: supplier is not specified for supplier location

Cause

Supplier location data was imported before the supplier master data, causing a dependency mismatch during integration.

Resolution

  1. Verify the import sequence for supplier-related data (supplier first, then supplier locations).
  2. If supplier locations were imported before the supplier, import the supplier master data.
  3. Re-import the supplier location data after the supplier exists.
  4. Reprocess or validate the integration to confirm the error no longer occurs.
