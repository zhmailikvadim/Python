Symptom

  • How to set "Obsolete" in PFCG as the checkbox is gray out.
  • Why "obsolete role" checkbox is set in the PFCG screen, but the role can still be used.

Environment

  • SAP NetWeaver
  • SAP NetWeaver Application Server for SAP S/4HANA
  • ABAP PLATFORM - Application Server ABAP
  • ABAP Authorization and Role Administration

Cause

  • The "obsolete role" checkbox is designed for informational purposes only and does not deactivate the role.

Resolution

1.This flag can be set via the menu (Role -> Role obsolete) 

2.The indicator is for informational purposes only and does not deactivate the role.
The information can be verified using F1-help on the obsolete checkbox. 

Keywords
