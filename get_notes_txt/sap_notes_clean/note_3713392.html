Symptom

You declare your exports from SAP GTS to the Swiss customs system Passar.

This SAP Note changes the XML definitions and the corresponding mapping in SAP GTS for the new message versions of the following messages:

  • Export declaration, NE015 v5

  • Correction of export declaration, NE013 v5

  • Correction of export declaration, NE069 v3

  • Confirmation of export data, NE021 v3 (not used functionally in SAP GTS)

Solution

Implement the attached correction instructions or import the specified support package.

Implement the additional SAP Notes for the changes from the new version in SAP Note 3720167 and the adjustment of the incompleteness check in SAP Note 3720168.

Changes to the mappings in SEEBURGER BIS are not required for the nine message versions.
