Symptom

When processing file format CFONB120c France using transaction code FEB_FILE_HANDLING, the system displays the message FEB_BSIMP001: "Cannot split into individual account statements." 

Environment

  • Electronic Bank Statement
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA

Reproducing the Issue

  1. Place file format CFONB120c France in the AL11 directory.
  2. Execute transaction code FEB_FILE_HANDLING.
  3. Attempt to process the file.
  4. Observe the error message FEB_BSIMP001 in the log: "Cannot split into individual account statements."

Cause

File format CFONB120c France is not supported in transaction FEB_FILE_HANDLING.

Resolution

SAP Help Documentation:https://help.sap.com/docs/SAP_ERP/6211bf1accac4f2e813e7f8e5fc19e99/1e44639767de4d13b6a28004e26c1efd.html?version=6.18.latest&q=FEB_FILE_HANDLING 

The following formats are supported in transaction FEB_FILE_HANDLING:

  • MT940, MT942
  • BAI
  • XML formats camt.053.001.02 and camt.054.001.02 

For formats that are not supported the transaction FF_5 could be used.
