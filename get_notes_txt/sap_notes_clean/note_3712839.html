Symptom

When using the "Copy to Clipboard" and then the "Paste" functionality in the Notifications and Objects table in Fiori application Manage Maintenance Orders (F5241), an error occurs when a functional location is copied.
The "Technical Object Type" column shows "Equipment", even though the technical object is a functional location.
This leads to error IM003 "Equipment ... does not exist" on save.

Other Terms

IM003, F5241

Reason and Prerequisites

Program error.

Solution

Implement this SAP Note.
