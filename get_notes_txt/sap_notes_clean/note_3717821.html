Symptom

Components which had to be deleted when corresponding operation changed effectivity and was moved to a different sequence in the next MPD file revision appear in the production order, and proper components fo not appear

Other Terms

MSFR14: Planning Routing

Reason and Prerequisites

Program error. Model-unit effectivity is used.

Solution

Implement the attached correction instruction. the system will create a data correction report mpe_pln_routing_fix_cmp. Run this report for the planning routings where the issues occur, first in test mode to check what is going to be done, and if it looks reasonable, not in test mode. The report will fix the routing by deleting the obsolete PLMZ entries.
