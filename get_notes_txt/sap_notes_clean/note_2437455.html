Symptom

The table IEFFZ for the function module 'EFFEKTIVZINS_BERECHNEN' contains the payment structure that is used as the basis for the effective interest rate calculation. The payment date (field DDISPO) has an inclusive indicator (field SDINCL).

Irrespective of whether the inclusive indicators are now set for all table entries or not set for all table entries, the effective interest rates should be identical. However, they are not.

Other Terms

SAPLFV90, FV90

Reason and Prerequisites

At the time of the last payment, an interest capitalization entry is generated in the program. However, the inclusive indicator is not passed on. This is incorrect.

Solution

Implement the attached program corrections or import the relevant support package.
