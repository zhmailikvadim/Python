Symptom

Why is no Test Scenarios available in the TEST Central for SAP Ariba strategic sourcing solutions in Managed Gateway For Spend & Network Portal ?

Environment

Managed Gateway For Spend & Network
SAP Ariba Sourcing

Resolution

There are no test scenarios available in Test Central for Managed Gateway for Spend & Network when using SAP Ariba Sourcing solutions. This is because Test Central integration is currently not supported for SAP Ariba Procurement and SAP Ariba Strategic Sourcing solutions.

At present, Test Central is supported only for SAP Business Network solutions.

For more information, please refer to the attached Overview Guide: Prerequisites and Restrictions | SAP Help Portal.
