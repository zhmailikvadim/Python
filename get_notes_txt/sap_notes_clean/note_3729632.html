Symptom

  • The change log of the scope items within a specific period of time needs to be checked.
  • Which scope items and countries were added via Define Scope App within a specific time period?

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Environment

  • SAP Central Business Configuration
  • SAP S/4HANA Cloud Public Edition

Resolution

1. Make sure your CBC user has the role Auditor (SAP_CBC_CONSUMPTION_AUDITOR). For more information, see SAP Help Portal - Authorization.

2. Click into "Change Logs" in the bottom left corner of the main interface of CBC.

3. Select the time range firstly. In the "Application Areas" filter, select "Scoping". Select the CBC workspace. 

Note: If there is no option for "Scoping" in the "Application areas" filter after setting the time range, it indicates that no operation related to Define Scope app was performed within this time period.

4. In the Details column, you can choose Open to get more information on the task performed.
