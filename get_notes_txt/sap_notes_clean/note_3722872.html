Symptom

It is not clear how the baseline payment date is determined in SD side.

Environment

  • Sales And Distribution (SD)
  • Logistics Execution (LE)
  • SAP R/3
  • SAP R/3 Enterprise 4.7
  • SAP ERP Central Component
  • SAP ERP
  • SAP Enhancement package for SAP ERP
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Resolution

Firstly, system will use the Billing date(VBRK-FKDAT) to fill the Baseline payment date(XACCIT-ZFBDT).

Then, system will check if there is a Fixed value date(VBRK-VALDT), if so, system will use this Fixed value date to update the Baseline payment date(XACCIT-ZFBDT).

Keywords

baseline payment date, ZFBDT, VBRK-FKDAT, XACCIT-ZFBDT, VBRK-VALDT, LV60BF00, ACCOUNTING_HEAD_LINE
