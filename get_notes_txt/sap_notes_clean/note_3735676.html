Symptom

 The Fiori Processes and Forms Analytics application triggers a runtime error during SELECT processing and the page does not load correctly.

Other Terms

PNF, Analytics,  HCM Processes, F6686, CL_HRFIO_ASR_PROCESS_ANALYTICS

Reason and Prerequisites

Method CL_HRFIO_ASR_PROCESS_ANALYTICS~MODIFY_USER_FILTER populates an incorrect filter condition. This invalid filter is later used in the SELECT statement, leading to an inconsistent WHERE clause and causing the runtime error

Solution

Apply the attached correction instructions alternatively apply corresponding support packages.
