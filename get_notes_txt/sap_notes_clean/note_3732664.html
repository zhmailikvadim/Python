Symptom

You are using the Case Find functionality in SAP Case Management. You will find that the time-related fields in the result list do not reflect the user’s local time zone. Instead, they appear in the System Time, leading to confusion regarding the actual timing of case activities.

Other Terms

Case Management, Case Find, Case Search, Created On Time, Changed On Time, Closed On/Completed Time, User timezone, Local timezone, System timezone.

Reason and Prerequisites

This issue is due to program error.

Solution

Implement the SAP Note or upgrade to the relevant Support Package.
