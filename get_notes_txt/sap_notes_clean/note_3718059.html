Symptom

  • In the Solution Process "Define User Decision" of Clarification Cases (e.g., Case Category UG02), it is not possible to track which user decision was selected during the pop-up screen.
  • There is no possibility to understand or trace what exactly happens to the Process Document through the selected User Decision.

Environment

SAP S/4HANA Utilities Master Data Updates Option for Germany

Reproducing the Issue

  1. Navigate to the Solution Process "Define User Decision" within Clarification Cases (e.g., Case Category UG02).
  2. Select a user decision in the pop-up screen.
  3. Observe that there is no mechanism to track or trace the selected user decision within the process.

Cause

Resolution

The solution of this issue is currently under investigation. Please follow this KBA for updates.
