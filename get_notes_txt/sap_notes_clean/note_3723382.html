Symptom

  • In the Manage Outgoing Checks app, both payee number and payee name are visible in the on-screen report output.
  • When the report is downloaded, only the payee number is included, and the payee name is missing.

Environment

SAP S/4HANA Cloud Public Edition

 



Reproducing the Issue

  1. Run the Manage Outgoing Checks app.
  2. Download the report.

Resolution

This is standard behavior of the Manage Outgoing Checks app, where the payee name is not displayed in the downloaded report. Displaying the payee name instead of the payee number can cause issues with identification, as the payee name is not as unique as the payee number.
