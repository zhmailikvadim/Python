Symptom

The report /SAPAPO/PROD_ORD_CNV doesn't convert planned orders with scope settings display only.

Additional, for such elements no log information are provided.

Other Terms

GET_SCOPE_FROM_MATNRLOCNO  /SAPAPO/CL_PPDS_SCOPE_SRV:

Flexible integration /SAPAPO/RRP_ORDER_IS_CONVTABLE

Reason and Prerequisites

Orders with the restriction of the display only are sorted out. This is incorrect

Solution

Implement the attached correction instructions or import the Support Package relevant for your release.

With the program corrections delivered in this SAP Note, an additional check is introduced.

Implement SAP Note 3712227 as a prerequisite.
