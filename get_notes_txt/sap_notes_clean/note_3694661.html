Symptom

You use the Pricing Grid in Fiori and expect search help (F4) to be available for input fields. However, Fiori search help is not available for any fields in the Pricing Grid, preventing users from selecting values using F4.

This note corrects the behavior by enabling Fiori search help for all applicable fields in the Pricing Grid.

Other Terms

/IRM/IPPDM, /IRM/LIPPDMF0G, GRID_HELP_FIELDS_GET, /IRM/LIPPDMF0I, INPUT_HELP_PROCESS, /IRM/LIPPDMF0U, UI_GRID_DATA_PREPARE, /IRM/LIPPDMTOP, /IRM/IPPD_INPUT_HELP_PROCESS, /IRM/UCL_IPPDM_HANDLER, /IRM/IF_GUI_CALLBACK~INPUTHELP_PROCESS, /IRM/IF_GUI_CALLBACK~TYPEAHEAD_PROCESS
