Symptom

This SAP note provides you with relevant system changes to prepare the following events with limited scope for alphanumeric CNPJs:

  • S-1070 - Tabela de processos administrativos/judiciais (administrative/legal proceedings table)
  • S-1260 - Comercialização da produção rural pessoa física (marketing of rural production, natural person)
  • S-1270 - Contratação De Trabalhadores Avulsos Não Portuários (Hiring Of Temporary Workers, Non-Dockworkers)
  • S-2210 - Comunicação de Acidente de Trabalho (Work Accident Communication) 
  • S-2220 - Monitoramento da Saúde do Trabalhador (Monitoring of employee health)
  • S-2231 - Cessão/Exercício Em Outro Órgão (Assigment In Another Entity)
  • S-2240 - Condições ambientais do trabalho - fatores de risco (Environmental working conditions - risk factors)
  • S-2400 - Cadastro de Beneficiário - Entes Públicos - Início (Beneficiary registration – public entities - beginning)
  • S-2405 - Cadastro de Beneficiário - Entes Públicos - Alteração (Beneficiary registration – public entities - update)
  • S-2410 - Cadastro De Benefício - Entes Públicos - Início (Benefit Registration – Public Entities – Beginning)
  • S-2416 - Cadastro De Benefício - Entes Públicos - Alteração (Benefit Registration – Public Entities – Update)
  • S-2418 - Reativação De Benefício - Entes Públicos (Benefit Reactivation – Public Entities)
  • S-2420 - Cadastro De Benefício - Entes Públicos - Término (Benefit Registration – Public Entities – Expiration)

Other Terms

Alphanumeric CNPJ; event structures

Reason and Prerequisites

Reason

According to the Normative Instruction nº 2229, October 15 of 2024, starting from July 2026, CNPJs may be alphanumeric. Therefore, system adjustment are necessary to ensure correct reporting of this information.

Prerequisites

You have applied the following SAP Note as prerequisite: 3700999 - eSocial - Alphanumeric CNPJ - Changes in standard events.

The following Support Package level is the minimum prerequisite:

Release Support Package
SAP_HRCBR 6.04 (EhP6 and EhP7) HR SP J1
SAP_HRCBR 6.08 HR SP B9
S4HCMCBR 1.00 HR SP 08
S4HCMCBR 1.01 HR SP 00
S4HCMCBR 1.02 HR SP 00

For more information, access the SAP ERP Human Capital Management - Support Package Schedules.

When you install an SAP Note, some prerequisites might be required. The prerequisites are SAP Notes that have already been released and that change the same context of the current one. Then, even if these prerequisites are not necessarily related to the symptom of this SAP Note, you must implement them due to technical reasons.

Solution

This SAP Note delivers changes in the following structure fields to prepare the system for alphanumeric CNPJs:

  • HRPADBR_EFDE_S_ADMIN_LAWSUIT (field EMPLOYER-INSC_NUM)
  • HRPADBR_EFDE_S_PAY_OSSR (field EMPLOYER-INSC_NUM)
  • HRPADBR_EFDE_S_PAY_OSSR (field EMPLOYEE_INFO-LINK_SUCESSION-PREVIOUS_CNPJ)
  • HRPADBR_EFDE_S_PAY_OSSR (field PAYMENT-LINK_SUCESSION-WORKPLACES-WRKPLC_INFO-INSC_NUM)
  • HRPADBR_EFDE_S_PAY_OSSR (field PAYMENT-LINK_SUCESSION-PY_PREVIOUS-PERIODS-WORKPLACES-WRKPLC_INFO-INSC_NUM)
  • HRPADBR_EFDE_S_PAY_OSSR (field PAYMENT-LINK_SUCESSION-RRA-LAWYER_INFO-LAWYER_INSC_NR)
  • HRPADBR_EFDE_S_PAY_BEN_OSSR (field EMPLOYER-INSC_NUM)
  • HRPADBR_EFDE_S_PAY_BEN_OSSR (field PAYMENT-WORKPLACES-WRKPLC_INFO-INSC_NUM)
  • HRPADBR_EFDE_S_PAY_BEN_OSSR (field PAYMENT-PY_PREVIOUS-PERIODS-WORKPLACES-WRKPLC_INFO-INSC_NUM)
  • HRPADBR_EFDE_S_PAY_BEN_OSSR (field PAYMENT-RRA-LAWYER_INFO-LAWYER_INSC_NR)
  • HRPADBR_EFDE_S_RURAL_MKT (field EMPLOYER-INSC_NUM)
  • HRPADBR_EFDE_S_RURAL_MKT (field TRADER_WORKPLACE-CNPJ)
  • HRPADBR_EFDE_S_NONDOCK_WORKR (field EMPLOYER_INFO-INSC_NUM)
  • HRPADBR_EFDE_S_WORK_ACCIDENT (field EMPLOYER-INSC_NUM)
  • HRPADBR_EFDE_S_WORK_ACCIDENT (field WORK_ACCIDENT-ACCIDENT_PLACE-ACC_INSC_NUM)

  • HRPADBR_EFDE_S_ER_CESSION (field EMPLOYER_INFO-INSC_NUM)
  • HRPADBR_EFDE_S_ER_CESSION (field CESSION_INFO-DESTINATION_INSC)
  • HRPADBR_EFDE_S_WRKPLC_COND (field ER_INFO-INSC_NUM)
  • HRPADBR_EFDE_S_WRKPLC_COND (field ENV_HAZARD-RISK_INFO_X-INSC_NUMBER)

  • HRPADBR_EFDE_S_BENEFIT_REG (field BENEFIT_INFO-SUCESSION-CNPJ)

  • HRPADBR_EFDE_S_BENEFIT_REACT (field EMPLOYER_INFO-INSC_NUM)

  • HRPADBR_EFDE_S_BEN_CESSATION (field CESSATION_INFO-CPNJ)

Important: If you have implemented any of the mentioned events, review the use of the specified fields above to avoid activation and runtime errors.
