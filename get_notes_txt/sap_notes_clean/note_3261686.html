Symptom

You are interested in release information for the Administration and Management Console (AMC) for SAP Adaptive Server Enterprise (SAP ASE).

Other Terms

AMC, Sybase, DBACockpit, DBA Cockpit, ASE Cockpit, Sybase Control Center, SCC, SAP ASE, SAP Adaptive Server Enterprise, Administration and Management Console

Reason and Prerequisites

AMC version 1.0 and 1.1 was released with SAP ASE 16.0 SP04. AMC version 1.2 was released with SAP ASE 16.1 SP00.
Support Packages for AMC were released with SAP ASE patch levels of SAP ASE 16.0 SP03, SAP ASE 16.0 SP04 and SAP ASE 16.1 SP00.
