Symptom

The focus around links or buttons on the career are not visible enough for accessibility compliance.

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

SAP SuccessFactors Recruiting Marketing

Resolution

Per accessibility team, SAP is targeting WCAG 2.2 A & AA criteria and this is not one of those criteria.

Per design team, this should not be changed unless it is configurable by customer, which would be an enhancement request. More information on how to submit an Enhancement Request can be found in KBA 2090228 - How to Submit Ideas for SAP SuccessFactors Products.
