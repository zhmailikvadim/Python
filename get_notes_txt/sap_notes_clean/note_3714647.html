Symptom

  • The Custom fields are not being updated for Crossboarding candidates when hired into the system.
  • These dates are correctly updated for new hires.

Environment

SAP SuccessFactors Onboarding

Reproducing the Issue

  1. Crossboarding has been Initiated through RCM.
  2. Observe that custom fields are not being updated for the candidate, even after passing the values from RCM

Cause

Expected Behavior

Resolution

This is expected behavior. Custom objects are not updated in the Employee Central (EC) profile for Crossboarding candidates.

If this functionality is important to your organization, we recommend submitting an Enhancement Request in the SAP Customer Community for consideration in a future Onboarding release.

For guidance on how to submit an idea, please refer to SAP Note 2090228: How to Submit Ideas for SAP SuccessFactors Products
https://launchpad.support.sap.com/#/notes/2090228
