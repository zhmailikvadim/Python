Symptom

When you run the WAB Monitor (RPUWABN0) report, the system displays error message "IT 0059 not available: Determination of necessity of revision not possible (HRPAYNL_EN 284)". This message appears when the employee is hired in the future.

Other Terms

WAB Monitor, RPUWABN0, PC00_M05_UWAB

Solution

After implementing this SAP Note, the system no longer displays an error message for employees without the Social Insurance NL (0059) infotype who are hired in a future date.

Delivery

Import the HR Support Package relevant for your release. For more information, see SAP ERP Human Capital Management Support Package Schedules or SAP S4 HCM Support Package Schedules.

SAP recommends that you implement the Support Package. However, if you need to install a solution earlier, use the SAP Note Assistant.
