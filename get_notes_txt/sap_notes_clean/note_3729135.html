Symptom

 This is a technical adjustment for the display functionality of the analytical view of the VAT return.

Other Terms

 CZ_VAT_DCL, CZ_VAT_DCL_CONTROL, CZ_VAT_DCL_CONTROL_V2, VAT Return

Reason and Prerequisites

 Function consolidation

Solution

Implement the automatic correction instructions.

As a general rule, SAP recommends that you implement the Support Package specified for your release. However, if you need to install a solution earlier, use the SAP Note Assistant to implement the correction instruction. For more information see SAP Note Assistant.
