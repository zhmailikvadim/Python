Symptom

You have applied SAP Note 2635706 to solve the invalid session handling.

After update to 7.50 SP16 you observe the symptoms described in SAP Note 2635706:

- Some requests fail with HTTP 401 or 500 codes

- In the traces you found for the failed requests time stamp exceptions like:

javax.servlet.ServletException: java.lang.RuntimeException: com.sap.engine.services.servlets_jsp.server.exceptions.WebInvalidSessionException: The session has already been invalidated.

Reason and Prerequisites

The upper behavior is side effect from SAP Note 2784278 - New Feature: XSRF token support for REST Adapter Servlet

Solution

With this SAP Note a program correction in the REST adapter has been introduced that will handle the invalid sessions properly.

Apply the patch matching your Support Package version as listed in the Support Package & Patches table below. Follow the instructions described in SAP Note 952402.
