Symptom

  • An error occurs when attempting to update the internal status for a custom status in Opportunities.
  • Error message: "Cannot invoke 'String.equalsIgnoreCase(String)' because the return value of 'com.sap.crm.opp.common.entity.CustomStatusCode.getLifeCycleStatus()' is null."
  • HTTP 500 Internal Server Error; code "0000"; target "java.lang.NullPointerException".
  • Changes cannot be saved.

Environment

SAP Sales Cloud version 2

Reproducing the Issue

  1. Click on User Menu
  2. Navigate to Settings
  3. Go to All Settings
  4. Go to Opportunities
  5. Go to Status
  6. Attempt to change the internal status for a custom status which is initially blank.
  7. Save the change and observe the error and that the save does not complete.

Cause

Lifecycle status was not assigned to custom statuses due to incomplete migration of Business Configuration data from V1 to V2.

Resolution

  1. In the V1 system
  2. Go to Transition WC
  3. Go to Data Transfer WCV
  4. Go to Opportunities
  5. Select Opportunity and initiate “Transfer All BC” to transfer all Business Configuration settings for Opportunity to V2.
  6. Wait until the transfer completes.
  7. In the V2 system, navigate to Settings > All settings > Opportunities > Status and verify that custom statuses have lifecycle/internal statuses assigned.
