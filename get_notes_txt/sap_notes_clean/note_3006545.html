Symptom

Creating a new service quotation leads to an update error "Error in updating object CRMS4D_SERV_I". As a result the service quotation is not saved.

Reason and Prerequisites

The problem only occurs, if the service order referring to the service quotation is created in one LUW. This is e.g. possible by using the PPF action CRM_SERVICE_QUOTE_SERV_ORD_FUP.

Solution

Please apply the attached code correction.
