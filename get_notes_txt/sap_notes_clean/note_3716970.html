Symptom

The order of search conditions appear is different in the prompt window when refreshing a Web Intelligence report after upgrade from 4.3 to BI 2025.

Other Terms

WebI, prompts, BI2025, BW, upgrade

Reason and Prerequisites

Environment : 

  • SAP BusinessObjects BI Platform BI 2025 Patch 5
  • Reporting database: SAP BW

Steps to reproduce :

  1. Login to BI launchpad
  2. Open a Web Intelligence report
  3. Refresh the report
  4. The Prompt Window pops up.
    >> Observe that the order of search conditions are different between 4.3 & BI 2025.
