Symptom

In loyalty management, the tier expiry report RLOY_TIER_EXPIRY_REPORT terminates if there is a large dataset.

Other Terms

RLOY_TIER_EXPIRY_REPORT, BATCH_MAX_NO_WP, maximum number of tasks, loyc_eng_param, CRM, activities, LOYD_MSH_MEMS, LOYD_MA_GEMATTR

Reason and Prerequisites

The maximum number of tasks processed in parallel is not taken into account. You enter the maximum number of tasks in Customizing under

Marketing -> Loyalty Management -> Processing Settings -> Define Loyalty Engine Parameters" using the parameter BATCH_MAX_NO_WP.

Solution

Implement the attached correction instructions in your system.
