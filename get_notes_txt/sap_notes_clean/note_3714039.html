Symptom

After searching for a word or phrase on the Catalog tab, the search result UI does not include the option to filter the results by keywords.

Environment

SAP Ariba Buying, SAP Ariba Guided Buying

Reproducing the Issue

In Ariba:

  1. Go to the Catalog tab.
  2. Search for a word or phrase.
  3. See the search result filters in the UI do not contain keywords as an option.

Cause

parameter Application.Catalog.DisableFacetKeyword is set to "Yes"

Resolution

Contact SAP Support to disable the parameter.

See Also

This parameter is enabled to mitigate search UI performance issues connected with having multiple keywords and large catalogs and is set to "No" (is disabled) by default.

Keywords
