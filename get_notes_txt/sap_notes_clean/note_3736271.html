Symptom

My reports display duplicate rows with different values for the same question, making the data unclear.

Resolution

To resolve this, add the Respondent field to your report to identify which user has provided each response. The fact SM Project Questionnaire Response is designed to fetch answers from all respondents within the same questionnaire version.

For more information regarding this behavior, you can check the documentation about the prepackaged report Supplier Management Questionnaire Response Report which uses this fact.
