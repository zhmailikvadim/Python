Symptom

After running MRP Live in transaction MD01N, the stock/requirements list is restricted to 100 materials, however, a larger number of materials were planned.

Environment

  • SAP ERP Version for SAP HANA
  • S/4 HANA
  • Material Requirements Planning - PP-MRP

Reproducing the Issue

1. Run MRP Live in transaction MD01N for a selection larger than 100 materials.

2. After the planning is complete, select the 'Stock/Requirements List' button.

3.  You'll notice that the list will only display 100 materials.

Cause

This system behavior was implemented within SAP Note 1943856.

A restriction of 100 materials was added to the 'Stock/Requirements List' from transaction MD01N in concern of performance, since it takes a long time to start up transaction MD07 when a lot of materials are involved.

Resolution

This is the standard behavior of the system. The alternative is to enter transaction MD07 directly with the suitable selection criteria.
