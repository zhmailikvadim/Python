Symptom

Execution of report /BA1/RBR_WAYC_DISPLAY - "Display Weighted Average Yield Curves" leads to runtime error COMPUTE_POW_RANGE (exception CX_SY_ARITHMETIC_OVERFLOW), if the weighted average yield curve contains discount factors smaller than 5E-15. These discount factors are shown as zero values in /BA1/RBR_WAYC_DISPLAY because the data type for displaying discount factors on the screen only has 14 decimal places.

However, the runtime error can be avoided by calculation of interest rates from floating point discount factor values.
