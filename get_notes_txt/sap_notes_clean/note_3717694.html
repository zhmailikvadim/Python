Symptom

This article provides information about the available process to mass cancel or restart messages in AIF.

Environment

  • ABAP Platform
  • SAP S/4HANA
  • SAP NetWeaver

Resolution

AIF is a monitor that monitors different types of interfaces and there are the so called 'Engines' in AIF configuration to deal with different types of interfaces. For example, a SOAP-based interface doesn't have the same engine responsible for cancel/restart than an IDoc-based interface. Therefore, AIF doesn't have a generic program/transaction to mass cancel/restart all types of interfaces.

However, such process can de achieved directly in /AIF/IFMON > Message Summary > Mass Cancel / Mass Restart for most interfaces. Such option requires a recipient to be assigned. For more information refer to SAP KBA 3358732 - AIF - How to assign AIF recipients to users?

For AIF own XML interfaces (/AIF/T_FINF table APPL_ENGINE 002), it's also possible to use the /AIF/XML_RESTART transaction for mass restart or mass cancel. But this report requires an entry in the /AIF/PERS_QMSG table for the MSGGUID. So the checkbox "From Index Table" must be marked.
