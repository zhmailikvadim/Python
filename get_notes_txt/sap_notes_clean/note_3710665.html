Symptom

  • A critical vulnerability (CVE-2025-68161) has been identified in the PI adapter related to the log4j library.
  • The vulnerability affects Apache Log4j versions below 2.25.3, with the installed version being 2.23.1.
  • Path of the affected file: /usr/sap/SID/J00/j2ee/cluster/bin/ext/com.sap.aii.adapter.ws.cxf.lib/lib/org.apache.logging.log4j-log4j-core-2.23.1.jar.

Environment

  • SAP NetWeaver 7.50

Cause

The vulnerability CVE-2025-68161 is caused by the use of Apache Log4j versions below 2.25.3

Resolution

1. CVE-2025-68161 is regarding the component Socket Appender, so those who use this component may be affected.
In PI/PO, Socket Appender is not used. Thus gladly, CVE-2025-68161will not affect PI/PO systems

2.PI/PO use Log4j and it has been updated regularly. But PI/PO do not use every functionality in Log4j and do not use Socket Appender.

3. Since PI/PO do not use Socket Appender's functionality, the vulnerability that is found in it does not affect. 
For more information: https://nvd.nist.gov/vuln/detail/CVE-2025-68161
