Symptom

  • The error message is being received when attempting to approve an item
  • The value for the Custom Field listed in the error message is selected through a dropdown

Cause

The value selected for the Custom Field in question is not valid

Resolution

Reject the item and update the value for the Custom Field in question to a valid one

Keywords

error, invalid entry, custom field, dropdown value, approval issue, reject item, update value, invalid selection, statement of work, approval process, error message, invalid data, dropdown error, custom field error, value update
