Symptom

While recording result for an inspection point relevant inspection lot via BAPI BAPI_INSPOPER_RECORDRESULTS an error is raised and written in RETURNTABLE:

QE456 - The inspected sample size does not correspond to the planned sample size.

Environment

  • SAP ERP Central Component
  • SAP ERP
  • SAP Enhancement package for SAP ERP
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA, on-premise

Cause

A possible cause is an incorrect input in the BAPI table SAMPLE_RESULTS.

Resolution

Check the inspection scope and sample size for the affected characteristic in its sampling procedure. If there is a fixed scope, match the value in BAPI input field SAMPLE_RESULTS-VALID_VALS with the defined sample size and retest the process with the BAPI.
