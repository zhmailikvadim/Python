Symptom

The Day Balance in the Time Statement display negative hours when multiple time types, such as Time Off in Lieu and actual working time, are recorded simultaneously on the same day.

Environment

SAP Business ByDesign

Reproducing the Issue

  1. Navigate to the Time Administration work center.
  2. Open Employee View.
  3. Search for the affected employee.
  4. Specify the month when the issue is first observed.
  5. Review the Time Statement.
    A negative Day Balance occurs when Time Off in Lieu is applied alongside recorded working hours, as the system deducts the time off while not counting the actual hours worked on that day.

Cause

This is the excepted System Behavior. The system reflects the value corresponding to the hours entered in Time in Lieu, resulting in a negative value in the Day Balance field. 

Credited Hours Calculation

  •  If no leave is recorded:
     Credited Hours = Actual Working Hours
  • If leave is recorded:
    Credited Hours = Planned Working Hours 

Balance calculation when a combination of time types is used

  • Balance = Credited Hours – Planned Hours – Time in Lieu – Leave Hours

Resolution

When Time in Lieu time type is used, Actual working hours recorded on the same day are not considered as working hours. These hours are ignored in the calculation.

Keywords

Negative hours, Time Statement, Day Balance, Actual Working Hours, Time off in Lieu
