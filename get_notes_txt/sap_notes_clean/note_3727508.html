Symptom

The Sourcing full project page keeps loading when we click on Export Excel icon in the Project Message Board. 

Cause

The issue occurs because the project title exceeds the system limit of 30 characters, which prevents the spreadsheet from being downloaded when exporting from the Project Message Board.

Resolution

Follow the steps below to resolve the issue:

  1. Verify the length of the project title in the full project view.

  2. If the title exceeds 30 characters, update the project title so that it is 30 characters or fewer.

  3. After updating the title, attempt to export again from the Project Message Board.

Note: If the issue persists after updating the project title, contact SAP Support for further assistance.
