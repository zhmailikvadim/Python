Symptom

  • Unable to export custom analytical query to Excel.
  • Export button is missing when previewing a query in the "Custom Analytical Queries and Services" application.
  • Export button is missing when previewing a query in the "View Browser" application.
  • Export button is missing when previewing a query from the "Manage KPIs and Reports" application > Multidimensional Reports > Show preview.

Environment

SAP S/4HANA Cloud Public Edition 2602

Reproducing the Issue

  1. Open the "Custom Analytical Queries and Services" application.
  2. Navigate to the query.
  3. Preview the query.
  4. Observe that the export button is missing.

Cause

Key user extensibility applications (such as Custom Analytical Queries and Services, View Browser, and Manage KPIs and Reports) have changed the default preview from Web Dynpro Grid to Multidimensional Analysis report.

The export function has been removed in Multidimensional Analysis (MDA) preview mode due to enhancements in the export feature, including async exporting and support for comments (cell, page).

Resolution

The export functionality remains available in customer-created applications.

Please refer to the SAP Help Documentation under Multidimensional Analysis: Export Data to Spreadsheet and PDF | SAP Help Portal for detailed instructions on the new export features.

        

In the context of Developer Extensibility, Developers can create a new application variant and release a new Multidimensional Analysis app based on the query. For more information, please refer to Generating Multi-Dimensional Analysis Applications | SAP Help Portal 

     

In the context of Key User Extensibility, key users can use the "Manage KPI and Reports" app to create a new Multidimensional Analysis app based on the queries created with the Custom Analytical Queries and Services application. Please refer to the following documentation: How to Create Multidimensional Analysis | SAP Help Portal.
