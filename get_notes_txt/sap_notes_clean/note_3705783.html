Symptom

While upgrading SAP CR Content from 16.X and 21.X from SLD, there is a need to confirm which specific packages are required to be downloaded from SAP Market place.

Environment

  • System Landscape Directory(SLD);
  • SAP CR Content(CR).

Resolution

Following are the package names and the respective order the import must be carried out from the SLD.

1. Update from 16.x to 16.11
    Support portal entry point: SAP CR CONTENT 2020
    File name: CRDelta161611_0-70005202.ZIP

2. Update from 17.x to 17.12
    Support portal entry point: SAP CR CONTENT 2021
    File name: CRDelta171612_0-70006262.ZIP

3. Update from 18.x to 18.11
    Support portal entry point: SAP CR CONTENT 2022
    File name: CRDelta181611_0-80007066.ZIP

4. Update from 19.x to 19.08
    Support portal entry point: SAP CR CONTENT 2023
    File name: CRDelta191608_0-80008225.ZIP

5. Update from 20.x to 20.05 
    Support portal entry point: SAP CR CONTENT 2024
    File name: CRDelta201605_0-70008421.ZIP

6. Update from 20.05 or 21.x to 21.y (latest version)
    Support portal entry point: SAP CR CONTENT 2025
    File name: CRDelta2116*.ZIP


All details are also available in SAP note 669669.
