Symptom

You are performing the Database content refresh with SWPM for system with product version S/4HANA 1809. An error occurred as below.
 
Error:
Last error reported by the step: Execution of the command "R3ldctl -h" finished with status TST_ERROR

Environment

SAP S/4HANA 1809 or higher.

Reproducing the Issue

Run the Database content refresh option with SWPM.

Cause

Your system is with S/4HANA 1809. But you are using SWPM 1.0, which is incorrect.

Resolution

For product S/4HANA 1809 or higher version, SWPM 2.0 should be used.
 
Please cancel the current SWPM run and start a new run with SWPM 2.0.

Keywords

R3ldctl, TST_ERROR
