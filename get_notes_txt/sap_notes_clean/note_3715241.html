Symptom

Despite having the necessary business catalogs and IAM app assigned to the configuration role, the configuration activities are not visible in the CBC system and following error is encountered - No matching results found for the given search criteria. Please modify the search criteria and try again.

 

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

Pre-requisites:

  • Configure Number Range for order types (106461) - Business Catalog: SAP_CA_BC_IC_LND_EAM_ORD_PC, IAM App ID: S_ALR_87000028_TRAN
  • Assign Number Range to Order Types (106462) - Business Catalog: SAP_CA_BC_IC_LND_EAM_ORD_PC, IAM App ID: S_ER9_52001758_TRAN

 

  1. In the CBC workspace, click the tab Configuration Activities.
  2. Search for the Configuration activity: 106461 and 106462.
  3. Observe the error message: "No matching results found for the given search criteria. Please modify the search criteria and try again."

 

Cause

There is no separate configuration activity for production orders number range.

Resolution

Activate the relevant PM scope items 4HI and 4HH in the CBC system.

Upon completion, SSCUI 106461 - Configure Number Range for Order Types and SSCUI 106462 - Assign Number Range to Order Types will become available.
