Symptom

In the 3D load plan, item labels cannot be displayed when drag and drop functionality is active. An error message is displayed when both settings are enabled in the Customizing (IMG activity "Define Layouts for 3D Load Plan").

Reason and Prerequisites

Program error

Solution

Implement the attached correction instructions or install the corresponding Support Package.

The error message in the Customizing is removed and item labels can be enabled together with drag and drop.
