Symptom

You call the drug master (transaction N1MEFM) and want to search for drugs using the assigned material number. However, this search is not executed, because no search criteria is entered for the drug, for example, the drug name or description.

Other Terms

Drug Master, Transaction N1MEFM, Medication

Reason and Prerequisites

There is a program error

Solution

The specified patch corrects this problem.

For an advance correction, implement Note 2889407.
