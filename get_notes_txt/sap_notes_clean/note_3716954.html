Symptom

SAP Note 3573491 delivered the adjustments in IT3670 for the year-end legal change 2025/26, including for the UI notification (address of insured person). According to the new guidelines, only address registrations for addresses outside Austria can be transferred - this applies only to the ÖGK.

If the check class CL_HRPA_INFOTYPE_3670_01 and, in particular, its derivation CL_HRPA_INFOTYPE_3670_GKK_01 is also used for other SI agencies (for example, Insurance Institution for Public Service Wage and Salary Earners, Railways and Mining - Railways Mining), the system issues the error message regarding Addresses in Austria are also triggered for these SI agencies.

Other Terms

IT3670, UI notification, ELDA

Reason and Prerequisites

Program gap

Solution

The correction is provided by HR Support Package (via CLC if necessary). You can implement an advance correction by implementing the correction instructions using SNOTE.
