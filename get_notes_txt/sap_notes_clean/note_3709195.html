Symptom

  • UTCTOLOCAL function is not working properly in the embedded SAP Analytics Cloud in SAP Sales Cloud.
  • Error message: "Formula could not be computed: Function 'UTCTOLOCAL' expects parameter 1 to be a timestamp value. Please complete or rewrite the formula."

Environment

  • SAP Sales Cloud and SAP Service Cloud
  • Embedded usage of SAP Analytics Cloud

Reproducing the Issue

  1. Log in to the SAP Sales Cloud and SAP Service Cloud.
  2. Navigate to System Settings -> All Settings -> Analytics -> Design Stories.
  3. Click "New" to create a new story.
  4. In Query Designer, drag and drop the table "Cases".
  5. In Select Columns, choose columns with a timestamp data type, e.g., "Initial Review Completed On" or "Initial Review Due To".
  6. Click "Create Calculated Columns" and navigate to the "Calculation Columns" tab to create a new column.
  7. Use the UTCTOLOCAL function in the formula editor as follows:
    UTCTOLOCAL([Data Sources#Case#Initial Review Completed On], "CET").
  8. Observe the error message: "Formula could not be computed: Function 'UTCTOLOCAL' expects parameter 1 to be a timestamp value. Please complete or rewrite the formula."

    Cause

    • The issue has been confirmed as a product defect.

    Resolution

    • The issue has been planned to be fixed in 2026.08 version.
