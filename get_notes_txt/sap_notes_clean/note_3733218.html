Symptom

How to print reservation documents in app Manage Manual Reservations?

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

SAP S/4HANA Cloud Public Edition

Resolution

Currently there is no automatic printing available for reservations as reservation documents may be changed again after creation. Instead, a PDF Print button was introduced. 

  1. In "Manage Manual Reservations", click the Print PDF button to generate the output item.
  2. Open the "Manage Output Items" app and locate the reservation output item. Select the item and choose "Send" to dispatch it for printing.
  3. Verify the item in the "Maintain Print Queues" app.
