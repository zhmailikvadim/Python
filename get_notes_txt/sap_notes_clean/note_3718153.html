Symptom

The field for gearbox type exists in the model master database table (/DBE/V_MODEL), but it's not visible in the /DBE/VMODEL transaction.

Other Terms

VSS, Vehicle Model Master, /DBE/VMODEL, gearbox, /DBE/C_VEH_GBOX

Reason and Prerequisites

Missing functionality.

Solution

Apply the attached note and run the attached correction program /DBE/NOTE_3718153 to resolve this issue (Step 2 & Step 3). Following is recommended, when executing the program:

  • When prompted for a package, enter /DBE/VEHICLE_MASTER
  • When seeing a message "Object ... can be created in a package of namespace /DBE/ only", press button "Further for All''. 

The note also introduces functionality that automatically updates vehicle Generic Options based on the new columns in the /DBE/C_VEH_GBOX table.
