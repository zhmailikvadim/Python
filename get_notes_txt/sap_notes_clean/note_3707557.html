Symptom

When you follow note 3625033 and you create documents like A/R Invoice, you find Create QRCode From is empty after the document is added to system.

Reproducing the issue

Prerequisites:

You are working in Argentina localization.

Scenario:

  1. You follow note 3625033 to create documents like A/R Invoice .

  2. After the document is added to the system, go to the Accounting folder and check the Create QRCode From field.

Expected behavior:

Create QRCode From is filled.

Actual behavior:

Create QRCode From is empty. 

Solution

SAP intends to provide a feature package or patch to solve the symptom that is described in this SAP Note. The section References below lists the related feature package or patch once they become available. Be aware that References can only be confirmed at feature package or patch release date. SAP delivers feature packages or patches only for selected releases at its own discretion, based on the business impact and the complexity of implementation.
