Symptom

  • The tile displays the URL instead of the intended title in SAP Fiori Launchpad.
  • The user cannot change the title of the tile, as it keeps referring to the URL.

Environment

SAP Fiori Launchpad ABAP Services

Reproducing the Issue

  1. Navigate to the SAP Fiori Launchpad App Manager (FLPAM).
  2. Configure a tile with the "URL App" type.
  3. Provide the required Target URL for the tile.
  4. Observe that the tile displays the URL instead of the intended title.

Cause

Resolution

  1. Navigate to the SAP Fiori Launchpad App Manager (FLPAM).
  2. Locate the "Target Application Title" field within the launchpad app manager.
  3. Specify the desired title for the application in the "Target Application Title" field.
  4. Save the changes and verify that the tile displays the correct title in SAP Fiori Launchpad.

See Also
