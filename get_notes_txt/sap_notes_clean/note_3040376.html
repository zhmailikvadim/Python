Symptom

You are executing program FAGL_FC_TRANSLATION. The log contains warning message FR600 "Error in Batch-Input Field" although the translation documents are created as direct postings using the Accounting Interface and Batch-Input is not used anymore in the current release.

Other Terms

FAGL_FC_TRANS POSTING_SL_FIELDS COBL

Reason and Prerequisites

Program design. The warning message FR600 with text "Error in Batch-Input Field" is obsolete since Batch-Input is not used anymore. The warning message appears in subroutine POSTING_SL_FIELDS referring to the fields of structure COBL which was formerly used for Batch-Input data purposes.

Solution

Implementation of the correction instruction.
