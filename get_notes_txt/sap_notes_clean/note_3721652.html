Symptom

You'd like to copy inspection results but cannot find copy inspection results function in Record Inspection Results (F1685A) app.

Environment

SAP S/4HANA Cloud Public Edition

Cause

This is standard behavior.

The copy inspection results function can be called at the following times:

  • At inspection lot creation

  • During inspection lot processing

Resolution

  1. Make sure the results data origin for transferred data has been specified in configuration activity 103947 - Make Settings for Results Copy and Operation Completion.
  2. Choose Edit -> Copy Inspection Results after sample calculation in Change Inspection Lot app to manually start the copy inspection results function.
