Symptom

When a user has several tabs or windows opened in the same session and logs off, only the current tab or window is logged off.

Reason and Prerequisites

Step to reproduce

1) Log in to Financial Consolidation Web HTML5

2) Go to Package List and open one or more packages

3) Go back to the Browser window or tab with the Package List and log off

The other windows or tabs stay connected.

From Financial Consolidation 10.1 SP01, the other tabs or windows in the same session will be logged off and will display the login page within 1 minute.

Solution

Update your SAP Financial Consolidation installation to a Support Package, Patch or Release where the issue is fixed. See the Support Packages & Patches section for details and available patches.
