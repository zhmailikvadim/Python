Symptom

You run the Individual Income and Social Tax Declaration (Form 200) report. The resulting XML and ALV output contain incorrect data for the Social Insurance Contribution paid by the employee (SICE). For civil contractors who are subject to SICE, this data displays with the reversed sign.

Other Terms

HRPAYKZ_F200_DECLR, RPCPAYKZ_F200_DECLR, F0081, F0082, F0083, F3051, F3052, F3053, FG05R, FG05Z, FG1AI

Solution

As a general rule, SAP recommends that you implement the Support Package specified for your release.
However, if you need to install the solution earlier, use the SAP Note Assistant to carry out manual correction instructions. For more information, see SAP Note Assistant.

Note that after installing the SAP Support Package, some customizing records are only imported into client 000. If necessary, users of other clients must manually replicate the changes for the following records from client 000 to their clients. For more information, see the attached file Manual Instructions.
