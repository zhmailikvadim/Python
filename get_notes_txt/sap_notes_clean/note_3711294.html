Symptom

  • The issue occurs in the SAP Cash Flow Analyzer App (F2332) where currency options are unavailable for selection, and the search help for currency does not function.  
  • All necessary services for the app have been activated, and authorizations have been appropriately adjusted.  

    “Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.”

Environment

SAP S/4HANA 1610 onwards

Reproducing the Issue

  1. Open Cash Flow Analyzer app (F2332)
  2. Try to select a currency using the search help feature.  

Cause

The proposal search functionality for input fields is not configured.  

Resolution

  1. Run transaction SDSH_CONFIG.  
  2. Enable "Use proposal search for input fields".  
  3. Save the configuration.
