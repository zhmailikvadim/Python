Symptom

In the program DFS_DIS_IBQ_TRANS_PROC_PROG, Creation from time cannot be empty else it throws an error. When the job is scheduled to run at 12 midnight (set the dynamic variant), Creation time is set to 00:00:00.

This NOTE contains fix to overcome the above mentioned scenario to allow processing of job/program when the creation time is set to 00:00:00.

Other Terms

Defense Disconnected S4, DFS_DIS_IBQ_TRANS_PROC_PROG

Solution

 Implement the attached correction instructions provided with the NOTE.
