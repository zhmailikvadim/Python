Symptom

  • Transport of communication scenario not showing in Quality system.
  • Communication scenario is visible in ADT when adding the Quality system in Eclipse but is not visible while creating communication arrangements in Quality system.

Environment

Maintain Communication System and Arrangement

Reproducing the Issue

  1. Create a communication scenario.
  2. Transport the communication scenario to the Quality system.
  3. Attempt to create communication arrangements in the Quality system.

Cause

Transport may fail when the Communication Scenario is marked as scope dependent.  This option is not intended for development of own communication scenarios.

Resolution

  1. Unmark the scope dependency of the Communication Scenario.
  2. Re-transport the Communication Scenario to the Quality system.
