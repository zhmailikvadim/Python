Symptom

The Add Logic options is missing when following the tutorial Build a Maintenance Notification App with GenAI-Powered SAP Build Code Assistant to Add Logic to your Application Service

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Environment

SAP Build Code

Reproducing the Issue

  1. Following the tutorial Build a Maintenance Notification App with GenAI-Powered SAP Build Code Assistant

  2. Try to do Add Logic to your Application Service on Extend the Maintenance Notification App

Cause

There are some differences between the tutorial and the application because the application has been updated.

Resolution

The Add Logic option becomes available only after creating a service action or function.
