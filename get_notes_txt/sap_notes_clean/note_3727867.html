Symptom

When attempting to execute the app Manage Fixed Assets (New Version), the following error message is displayed: "Unit XXX is not created in language AA.".

Environment

SAP S/4HANA Cloud Public Edition.

Reproducing the Issue

  1. Access the new version of the app Manage Fixed Assets.
  2. Set the logon language to a non-English language (e.g., Chinese).
  3. Click "Execute" and the following error message is displayed:"Unit XXX is not created in language AA.".

Cause

A language-dependent key is not maintained for the internal unit of measure in the respective language.

Resolution

Verify that the unit of measure exists in the configuration activity "Check Units of Measurement" for the logon language where the error arises.

Keywords

unit of measure, language, manage fixed assets
