Symptom

When we select a candidate and click “Mass Complete My New Hire Tasks”, then select the required task and choose the option “Add New Hires”, we can see a duplicated user available for selection.

Environment

SAP SuccessFactors Onboarding 

Reproducing the Issue

  1. Proxy as the responsible user for the equipment tasks.
  2. Search for the candidate in the onboarding system.
  3. Select the onboardee and click on "Mass Complete My New Hire Tasks".
  4. Select the required equipment task (e.g., computer).
  5. Choose the option "Add New Hires".
  6. Observe that the same user is duplicated and available for selection.

Cause

The logic for mass completion picks all candidates with equipment activity. If multiple equipment activities exist, the system applies the same new hire's name across all equipment activities.

Resolution

This is a known gap with the mass completion feature for new hire equipment activity. The logic picks all equipment activities associated with the candidates, resulting in the duplication of the new hire's name for each equipment activity.

To change this behavior, it is required to submit an enhancement request. Please refer to this KBA: How to submit enhancement ideas for SAP SuccessFactors Products
