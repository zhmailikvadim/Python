Symptom

For construction savings contracts with other contract partnerships as holders for which there is no CFS, the building savings contract display (transaction /IBS/BSC_SHOW) terminates in an uncontrolled manner.

Reason and Prerequisites

In the BSC BO, the first row of an empty table is accessed.

Solution

The implementation /IBS/CL_PRM_BO_BDS_IFM_BSC, method WOP_DATA_FILL of the above-mentioned BO is corrected.
