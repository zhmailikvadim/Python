Symptom

This SAP Note delivers a correction to properly populate and display the column Plant at the Pending Documents report (EDOC_BR_PENDING_DOC transaction).

Other Terms

eDocument, eDoc, eInvoicing, e-Invoicing, electronic invoice, electronic invoicing, e-document, e-Document, e-Docs, e-Doc, e-docs, e-doc, Brazil, Brazil eInvoice, Inbound, NF-e, Nota Fiscal Eletronica, Nota Fiscal Eletrônica, Electronic Nota Fiscal, Pending Documents, EDOC_BR_PENDING_DOC

Reason and Prerequisites

You have implemented the SAP Note 3711451 (eDocument Brazil Inbound NF-e: Prerequisite objects for SAP Note 3711472).

This SAP Note is relevant only for Brazil.

Solution

After you implement this SAP Note, the Plant column will display the correct information, and a new Business Place column is added to the report.

Note: The values in the Plant and Business Place columns will be corrected only after you implement this SAP Note. The data you have in your system before implementing this SAP Note will remain unchanged.

Use transaction SNOTE to implement the attached correction instructions.
