Symptom

After enabling the Unified Data Model, a category page shows incorrect jobs or some expected jobs are missing

Environment

SAP SuccessFactors Recruiting Marketing

Cause

The job list is not matching with the rule archive.

Resolution

When enabling the Unified Data Model, existing category rules are migrated. However it is essential to review the new Job List filters. (equivalent of previous category rules)

During this review, ensure that the filters align with the original rules. If a Job List filter does not match a legacy rule, take corrective action by either updating the filter or considering deactivating the page if you are enable to duplicate the previous conditions.
