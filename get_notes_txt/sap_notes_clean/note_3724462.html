Symptom

You are unable to create an external Sales Quote from an Opportunity.
When attempting to do so, the following error message is raised:

"Order block Sales Order Block is set for account (“XYZ”) in master data."

Environment

Integration for SAP Sales/Service Cloud

Sales Cloud Version 2

Reproducing the Issue

  1. In System Administration, configure an Order Block under Customer → Order Block for a specific account’s sales data.
  2. Navigate to an Opportunity.
  3. Attempt to create an external Sales Quote by selecting + under Sales Quotes.
 
The system raises an error message:"Order block Sales Order is set for account in master data."

Cause

The system prevents the creation of external Sales Quotes from Opportunities when an Order Block is maintained in the account’s Sales Data.

This behavior results from validation logic that checks the account’s master data (MasterDataBlockingReasons).
If an Order Block is found, the system blocks the creation of follow‑up external documents.

Resolution

External Sales Quotes cannot be created from Opportunities if the account has an Order Block configured.
This is expected system behavior.

Additional details:

  • An Order Block is designed to prevent saving the following transactions:

    • Quotes
    • Sales Orders
    • Service Tickets
    • Contracts

  • A Sales Support Block, on the other hand, prevents saving:

    • Activities
    • Leads
    • Opportunities

A Delivery Block alone does not prevent the creation of internal or external quotes by default.

For external follow‑up document creation (e.g., external Sales Quote in SAP S/4HANA), any Order Block detected in the account master data correctly prevents the outbound call.
