Symptom

This SAP Note contains information related to the Intercompany integration solution 2.0 PL25.

Solution

Patch Installation:

This patch can be used to directly upgrade, update or install from scratch, Intercompany integration solution 2.0.

Compatible SAP Business One versions (only these combinations of versions are approved and supported):

  • SAP Business One 9.2 PL10
  • SAP Business One 9.2 PL10, version for SAP HANA
  • SAP Business One 9.3 PL01
  • SAP Business One 9.3 PL01, version for SAP HANA
  • SAP Business One 9.3 PL02
  • SAP Business One 9.3 PL02, version for SAP HANA
  • SAP Business One 9.3 PL03
  • SAP Business One 9.3 PL03, version for SAP HANA

Patch Content:

This patch includes all corrections and enhancements from the previous patches of Intercompany integration solution 2.0 as listed chronologically in the associated info file.

For more information about the corrections contained in this specific patch, see the SAP Notes listed in the References section.
