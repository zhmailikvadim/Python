Symptom

  • Error TPM_TLV1017 with text “Scheduled bus. trans. with posting date before key date”.
  • Unable to post amortization of bond issue discount and effective interest rate (EIR) interest for month-end.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Execute Run Valuation – TPM1 or related amortization posting.
  2. Observe that the process stops with error TPM_TLV1017 “Scheduled bus. trans. with posting date before key date”.

Cause

Position flows for the affected business transaction(s) are in status “Scheduled”, so postings cannot proceed for the given key date.

Resolution

  1. Execute the App Fix, Pay and Post flows (TBB1) to update the position flows to Fixed status.
  2. Additionally execute the Process Business Transaction (TPM10) to see the transactions to be Fixed/Posted for the securities class.
  3. Execute Run valuation (TPM1) for the relevant securities class and securities account to generate the amortization postings.
  4. In the parameters select Product Group: Securities,
    Provide the Securities class and Securities account.
    Provide the Key Date Valuation : (Date prior to the calculation date of the securities deal)
