Symptom

You use the clinical work station (CWS) in the Outpatient Clinic/Service Facility view type (007) to manage your outpatient clinic data. When you perform a partial update of the KLAP, you notice that the system does not set the MOD (modification) indicator correctly in the method EXIT_DISPLAY of the BAdI N1_WP_LSTAMB.

If the treatment order is active, the indicator is set for all CWS entries with active treatment authorization instead of only for the entries that triggered a partial update.
If the treatment authorization is not active, the indicator is set for all CWS lines.

Other Terms

CWS, Outpatient Clinic/Service Facility view type, EXIT_DISPLAY, N1_WP_LSTAMB

Reason and Prerequisites

Program error

Solution

The system now correctly sets the MOD indicator for the method EXIT_DISPLAY in the BAdI N1_WP_LSTAMB only for the lines of the clinical work station (CWS) that were changed.

Import the specified Support Packages to solve the problem. You can implement the corrections in advance.
