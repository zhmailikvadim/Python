Symptom

You are declaring export declaration to Netherlands DMS using SAP GTS.

SAP GTS does not create one export declaration containing all items.

Instead, it creates multiple export declarations, where each declaration contains a maximum of 99 item lines.

Other Terms

Netherlands, Dutch, Export, limit, 99, 999, items, /SAPSLL/CL_ITLMT_UTIL, GOVERNMENTAGENCYGOODSITEM

Reason and Prerequisites

Program error.

Solution

This note increase the limit to 999 items per declaration.

Implement the correction instructions that are relevant for your release.
