Symptom

When you process bursaries, the system displays an incorrect bursary exemption limit in the display log for beneficiaries with disabilities or higher education (National Qualifications Framework Levels 5–10).

Other Terms

BURSARY, RELATIVES, M603, EXEMPTION LIMIT, RPCMSTW0, RPCALCW0, PC00_M16_CALC

Reason and Prerequisites

Program error.

This SAP Note is relevant only for South Africa localization.

Solution

After implementing this SAP Note, the system displays the correct bursary exemption limit for employee beneficiaries.

As a general rule, SAP recommends that you implement a solution by implementing the Support Package specified for your release.

However, if you need to implement this solution earlier, implement the attached correction instructions using the SAP Note Assistant. You can find more information about the SAP Note Assistant on the SAP Support Portal, under SAP Note Assistant.
