Symptom

SAP GUI for HTML implements the Cumulative patch UnifiedRendering 1710 to incorporate corrections that were provided by the UnifiedRendering team.

UnifiedRendering is the HTML rendering technology on which SAP GUI for HTML is based. This technology is also used by other SAP frameworks, such as Web Dynpro. In case you experience an issue caused by the UnifiedRendering framework and this issue has been corrected with the Cumulative patch UnifiedRendering 1710 (as documented in SAP Note 2523075), please follow the instructions within this note.

Solution

To resolve this issue please do the following:

Determine the patch number of the SAPEXE SAR package with the highest patch number for your kernel version on the SAP Support Portal. If the SAPEXE patch number is higher than or equal to the highest patchlevel number you see in the section Support Packages & Patches of this SAP note for your SAP kernel version, you should download and install the SAPEXE SAR package. Otherwise, download and install the most recent DW.SAR archive.

Important: If there is no entry with the patch number yet, please re-check this SAP note at a later point in time. In this case the patch is not yet available for download and still in production or validation. The patch number will be added automatically once the corresponding patch is available on SAP Support Portal. You can subscribe to a SAP note to get informed when it is updated.

You can find up to date information about the status of the patch delivery in the following SAP notes:

FAQ:

Expected delivery dates of SAP kernel patches

  • SAP Kernel patches are usually delivered weekly or bi-weekly on Friday depending on the release. From the patchlevel of latest delivered patch you can roughly conclude when the patch you need will become available.

Downloading SAP Kernel patches

  • All patches are available in the Software Downloads section on SAP Support Portal (http://support.sap.com/patches). Detailed information on the exact location of the patches can be taken from note 563161. SAP Note 330793 explains how to download patches from SAP Support Portal.

Installation of SAP kernel patches

  • SAP note 19466 explains how install SAP Kernel patches.
