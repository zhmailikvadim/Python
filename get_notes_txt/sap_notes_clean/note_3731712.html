Symptom

You execute report RFAWVZ40N. The list header as well as the PDF header show an incomplete name for the reporter (only ADRS-NAME1 and not ADRS-NAME1 and ADRS-NAME2). The XML file contains the correct  name of the reporter from the address fields NAME1 and NAME2.

Other Terms

RFAWVZ40N, RFAWVZ40N_LIST, RFAWVZ40N_PDF, NAME1, NAME2

Reason and Prerequisites

Missing functionality

Solution

Implement the attached coding correction or the respective support package.

The list header as well as the PDF header will then show the expected full name of the reporter like in the generated XML file.
