Symptom

When you run the MDR report with _AUTHORITY-CHECK OBJECT '/US4G/MDR', the system displays an incorrect error message because of a field length mismatch.
This issue is relevant for the market roles: SUP, DSO.

Other Terms

MDR report, AUTHORITY-CHECK OBJECT '/US4G/MDR', field length, error message, SUP, DSO

Reason and Prerequisites

The error occurs because the field length for _AUTHORITY-CHECK OBJECT '/US4G/MDR' is incorrect in the system.

Solution

Implement the corrections of this SAP Note or import the relevant support package.
After implementing this SAP Note, the system no longer displays a field length mismatch error message when you run the MDR report with _AUTHORITY-CHECK OBJECT '/US4G/MDR'.
You can verify the fix by running the MDR report and confirming that no error message appears.
