Symptom

 When we try to download/Export the TPR accrual data from a Vizi operation Report in an excel format, then the data which has currencies amount with KRW, TWD, then that currency amount data is downloading with decimals which is incorrect. It should be downloaded without decimals only.

Other Terms

Vizi operation Report, Excel data, TPR accrual data, Currency amount, KRW, TWD, Decimal issue.

Reason and Prerequisites

This is caused by program error

Please implement the correction instructions attached. These corrections are only valid if you have applied the Support Packs.
