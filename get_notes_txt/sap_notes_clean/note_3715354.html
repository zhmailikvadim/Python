Symptom

  • The feature toggle "Rebuild of Documented Credit Decision (FIN_FSCM_REBUILD_DCD)" must be activated to use the "Manage Documented Credit Decisions (F5587A)" app.
  • The feature toggle "Rebuild of Documented Credit Decision (FIN_FSCM_REBUILD_DCD)" is not found in the configuration activity Activate Features or the CBC system.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Attempt to locate the "Rebuild of Documented Credit Decision (FIN_FSCM_REBUILD_DCD)" feature in the configuration activity.
  2. Search for the feature in the CBC system.
  3. Observe that the feature is not available in either location.

Cause

Feature toggle "Rebuild of Documented Credit Decision (FIN_FSCM_REBUILD_DCD)" was not assigned to a scope in the SAP S/4HANA Cloud 2602.

Resolution

The feature toggle "Rebuild of Documented Credit Decision (FIN_FSCM_REBUILD_DCD)" will be delivered as Continuous Feature Delivery (CFD 2602.02).
