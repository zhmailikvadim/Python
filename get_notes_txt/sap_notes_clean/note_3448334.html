Symptom

You are not able to access the System Landscape Directory (SLD) from the URL https://serveraddress:40000/ControlCenter.

The system displays the following error message: "NetworkError when attempting to fetch resource".

Image/data in this SAP Note is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Reproducing the issue

You are using the Firefox browser and trying to access the SLD from the URL https://serveraddress:40000/ControlCenter.

You get the error message: "NetworkError when attempting to fetch resource".

Cause

Due to CORS (cross-origin resource sharing), the SLD is blocked in the Firefox browser.

Solution

SAP intends to provide a feature package or patch to solve the symptom that is described in this SAP Note. The section References below lists the related feature package or patch once they become available. Be aware that References can only be confirmed at feature package or patch release date. SAP delivers feature packages or patches only for selected releases at its own discretion, based on the business impact and the complexity of implementation.

As a workaround, you can use the latest version of the Google Chrome browser or Microsoft Edge browser to access the SLD URL.
