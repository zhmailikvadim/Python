Symptom

I want to know if I need to manually update the URLs under Ariba side as part of the upcoming domain decommissioning (EV33942093).

I am unsure if SAP automatically migrates these endpoints or if I must update them manually in Ariba solution.

Environment

  • Managed Gateway for Spend&Network
  • SAP ERP Central Component (ECC)
  • SAP S/4HANA
  • SAP Ariba Buying and Invoicing
  • SAP Ariba Sourcing
  • SAP Business Network

Resolution

You do not need to manually update the endpoints in:

  • SAP Ariba Buying and Invoicing
  • SAP Ariba Sourcing
  • SAP Business Network

The SAP Engineering team updates any required endpoints on the Ariba side as part of the domain migration process.

You must only focus on updating the endpoints in your backend system (SAP ERP or SAP S/4HANA), if applicable.

For the required backend changes, follow: SAP Note 3508782 - How do I update new domain URLs for the SAP Integration Suite, Managed Gateway for Spend Management and SAP Business Network application in my SAP ERP or SAP S/4HANA environment? - SAP for Me
