Symptom

  • An error occurs when attempting to update the credit limit for customer using T-Code FD32 
  • Error message received: "Maximum individual limit should only be ***" (Error message no. F4724).

Environment

  • SAP R/3
  • SAP S/4HANA

Reproducing the Issue

  1. Navigate to T-Code FD32.
  2. Enter Customer
  3. Select Credit controlling area 
  4. Check the status checkbox.
  5. Attempt to update the Credit limit value.

Cause

  • The issue arises due to a configuration mismatch in the credit limit settings for the customer.
  • The permitted maximum individual limit is incorrectly set to in the "Central Data" tab for the specified customer. 

Resolution

  1. Navigate to T-Code FD32 
  2. Enter the customer number and select the credit controlling area.
  3. Open the "Central Data" tab.
  4. Adjust the permitted credit limit to the desired value.
  5. Save the changes.
