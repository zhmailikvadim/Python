Symptom

This note includes the following corrections in the S/4 Defense Disconnected system:

Issue: Material availability check (ATP) error is raised while changing the Delivery date in return STR created from Receiver system.

Resolution: New condition has been added to check if the quantity in Stock Transfer Request(STR) is greater than the available stock before raising the ATP check error.

Other Terms

STR, Stock Transfer Request, ATP Check Error, Material Availability, S/4 Defense Disconnected.

Solution

Implement the corresponding CI as per the note.
