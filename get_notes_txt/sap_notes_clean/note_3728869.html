Symptom

  • The Fiori home page appears blank, and no content is displayed.
  • Checking the network trace the URL `<domain>/sap/ushell_abap/bootstrap/evo/abap.js` results in a 404 error.
  • Clicking on the URL `<domain>/sap/bc/ui5_ui5/ui2/ushell/resources/sap/ushell_abap/bootstrap/evo/core-ext-light-1.js` returns the message: "SAPUI5 is not maintained in the system. Check appl. log details."
  • Note 3599156 is implemented, but the `SUI5_Content` package is missing in SE80.

Environment

SAP Fiori Launchpad ABAP Services

Reproducing the Issue

  1. Attempt to access the Fiori Launchpad home page.
  2. Observe that the page appears blank with no content displayed.
  3. Access the URL `/sap/ushell_abap/bootstrap/evo/abap.js` and confirm the 404 error.
  4. Access the URL `/sap/bc/ui5_ui5/ui2/ushell/resources/sap/ushell_abap/bootstrap/evo/core-ext-light-1.js` and confirm the message: "SAPUI5 is not maintained in the system. Check appl. log details."

Cause

The issue occurred because the SAPUI5 version was removed or overwritten during a system restore or rollback of a note. Specifically, note 3599156 was implemented, but the required `SUI5_Content` package is missing in the repository.

Resolution

  1. Verify if the server was refreshed or if a UI5 patch was uninstalled.
  2. Confirm that the system has been updated to the new method by checking if SAP Note 3599156 has been implemented.
  3. Open SE80 and select Package in the dropdown and verify SUI5_CONTENT is not present
  4. Install the required SAPUI5 version to restore functionality:
  • Apply the latest version of the installed SAPUI5
  • See SAP Note 3155948
  1. Test the Fiori Launchpad home page to ensure the issue is resolved.
