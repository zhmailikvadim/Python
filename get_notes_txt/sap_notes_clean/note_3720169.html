Symptom

  • The default KPI "sum of Application run cost by business capability" is a group-by KPI type.
  • User attempted to create a custom group-by KPI, but this is not showing up in the dropdown menu.

Environment

SAP LeanIX KPI (Key Performance Indicator)

Resolution

  1. Currently, there is no resolution, as it's currently not supported to create custom group-by KPIs, also not by the SAP LeanIX customer support team.
  2. It is recommended to submit the idea to the LeanIX product roadmap for consideration by the product team.
  3. Refer to the LeanIX product roadmap submission link: LeanIX Product Roadmap.

See Also

Refer to: LeanIX Product Roadmap.
