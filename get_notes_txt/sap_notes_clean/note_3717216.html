Symptom

You are using the Egypt VAT Return statutory report in the Run Statutory Reports app. You encounter a memory allocation error when generating the VAT Return Form 10 for a large number of sales documents . As a result report output is not generated.

Other Terms

VAT, Form 10, Egypt VAT Return, memory allocation error, Document and Reporting Compliance, Statutory Reports, EG_VAT_DCL

Reason and Prerequisites

Performance improvements need to be made to the report so that it can handle report generation for a large number of sales documents.

Solution

 After implementing this SAP Note, the system successfully generates the VAT return Form 10 without encountering any memory allocation error.
