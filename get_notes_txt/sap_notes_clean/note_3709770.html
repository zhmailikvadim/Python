Symptom

In the process "Receipt of SLP/TEP/PSET from DSO (SUP View)" (DE_B_EQS_SLP_RECEIVE_SUP), the process terminated because of missing exception configuration and process linkage.

This issue is relevant for the market role: SUP.

Other Terms

EQS_SUP, DE_B_EQS_SLP_RECEIVE_SUP, SLP, TEP, PSET, exception configuration, process linkage, BC set

Reason and Prerequisites

The error occurred because the system was missing exception configuration and process linkage.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the system includes the required exception configuration and process linkage by activating the BC set /US4G/1_0_DSO_SP07_NOTE_3709770.

You need to retrigger the process. You can verify the fix by checking that it runs successfully.
