Symptom

Not able to open Manage Bank Accounts App due to output management viewing error as attached below :

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental." 

Environment

SAP S/4 HANA Cloud Private Edition 2023

Reproducing the Issue

  1. Open Manage Bank Accounts App.
  2. Click on Create

Cause

Cache Invalidation

Resolution

Run the below reports:

  1. Run program /UI2/GET_APP_DESCR_REMOTE_DEV with system alias 'S4FIN' via SE38.
  2. Run report /UI5/APP_INDEX_CALCULATE in Execution mode and Single Repository for "FIN_BANK_ACC
  3. Run transaction /n/UI2/APPDESC_GET_ALL without Test Mode
  4. Run report /UI2/INVALIDATE_GLOBAL_CACHES and /UI2/INVALIDATE_CLIENT_CACHES to clear the cache

Keywords

Output Management , check for file not found or parse error , Manage Bank Accounts , View.XML
