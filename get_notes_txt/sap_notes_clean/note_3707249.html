Symptom

In transaction SCFD_EUI, it is not possible to enable a field with semantical type Amount or Quantity. Although the field referenced currency/unit code field is contained in a CDS view, there is still a warning shown like the following:

Ensure database reference field &1 is available in CDS view &2.

Other Terms

Custom Field, ABAP Managed Field, SCFD_EUI, Reference Field, Amount, Quantity, Foreign Table, RAP

Reason and Prerequisites

1. The aount/quantity field is contained in a different database table than its referenced currency/unit code field.

2. The business context contains RAP CDS views, which are marked as "Automatically enabled".

Solution

Apply this note. Afterwards restart transaction SCFD_EUI and enable the database field.

"FUNCTION DHCDC_RT_SCHEDULE_JOBS DID NOT FINISH IN 300 SECONDS"
"CANNOT START FUNCTION DHCDC_RT_SCHEDULE_JOBS: 2"
"RFC task SCHEDULE_JOBS already open."

Other Terms

S/4HANA, ABAP Data Integration, SAP Business Data Cloud, APRS, CDC, Delta Reconstruction, DHADM, Access Plan Calculation, transfer job, observer job, ACP Daemon

Reason and Prerequisites

The ABAP-based source system is an S/4HANA 2025 FPS00-SP01 system.

The ABAP-based source system is an S/4HANA 2023 FPS00-SP05 system.

The ABAP-based source system is an S/4HANA 2022 FPS00-SP07 system.

The ABAP-based source system is an S/4HANA 2021 FPS00-SP09 system.

Solution

Implement the corrections in the ABAP-based source system.
