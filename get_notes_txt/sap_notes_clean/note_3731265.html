Symptom

A mileage leg which is not the leg with the highest index is copied. For example there are 2 legs existing. The first one is copied. In this case the created copy will be supplied not with the custom field values from the copied leg 1 but with the customer field values from the mileage leg with highest index 2.

Other Terms

copy, custom fields, kilometer, mileage,

Reason and Prerequisites

Program error.

Solution

Implement the SAP Note or the Support Package.
