Symptom

PTCH-48539 delivered a fix so managers could register team members for classes when "Manager Can Register Others" was enabled, even if self-registration for users was turned off. The patch also enabled access to individual reports and external learning requests from Team View. This article provides more details for this enhancement patch.

Environment

SAP SuccessFactors Learning

Resolution

Managers are able to register others even though users are not able to self-register

As a Manager, I want to register learners into a class when the “Manager Can Register Others” configuration is enabled, so that I can manage enrollments for my team members even when self-registration is disabled for learners.

Permissions:

  • Manager role with “Manager Can Register Others” permission enabled at the class level
  • Access Registration Permission

Prerequisites:

  • The class exists and is open for registration or waitlisting
  • The manager has access to the class via the library
  • The “Manager Can Register Others” configuration is enabled on the class
  • The "User Can Self-Register" configuration is disabled on the class

External Learning Request (SF-182) Link in Team View

Allow managers to generate individual reports directly on Team View. The manager can select an individual's more menu and select View Report and View External Learning Requests.

Permissions:

  • Access Direct Reports
  • Access Team's Learning

Known Issues / Limitations

  • Links for Reports and External Learning Requests still rely on the legacy flow.
  • Clicking these links opens a new browser tab instead of navigating within the current context.
  • Some pages may be missing breadcrumbs, which can impact navigation and make it harder for users to understand their current location.
