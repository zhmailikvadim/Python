Symptom

The navigation menu remains in hierarchy mode despite configuration changes made to "Organize Work Centers and Work Center Views".

Environment

SAP Cloud for Customer

Reproducing the Issue

  1. Log in to the system.
  2. Observe that navigation items are displayed in hierarchy mode instead of flat mode.

Cause

The company settings do not have the option "Enable Flat List Navigation Menu" activated.

Resolution

  1. Access the system settings.
  2. Navigate to "System Settings".
  3. Select the relevant role.
  4. Enable the option "Enable Flat List Navigation Menu".
  5. Ensure all required roles are configured to enable flat navigation menus.

See Also

KBA 3221627 - Hierarchy View Not Displayed To Users
