Symptom

Once billing document ABC has been posted, the various characteristics of the billing document are greyed out and can no longer be edited.

Where ABC is a sample billing document ID.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Log onto SAP S/4HANA Cloud Public Edition.
  2. Open the app Change Billing Documents.
  3. Search for Billing Document: ABC.
  4. Select Continue.

Observe all the fields are read-only and cannot be altered.

Cause

You cannot alter an invoice once it has been processed and posted, because doing so can create inconsistencies throughout the system and result in incorrect reporting. This is the expected system behavior. 

Resolution

If you need to change the details of an invoice you will need to rollback the process and ensure any documents preceding and succeeding the invoice have consistent data.
