Symptom

When generating the EFD ICMS/IPI report with either DRC Brazil or the Tax Declaration Framework (TDF), it is observed that field 0200-COD_BARRA is incorrectly populated. This issue occurs for materials where this field should remain blank.

Environment

  • SAP S/4HANA
  • DRC Brazil
  • Statutory Reporting
  • Tax Declaration Framework (TDF)

Cause

The system is designed to populate field 0200-COD_BARRA in the EFD ICMS/IPI report using the data from the material master field MARA-EAN11.

The issue occurs because the MARA-EAN11 field has been incorrectly filled for materials that should not have a barcode. As a result, this text is transferred to the report, when the field should be blank.

Resolution

To resolve this issue, ensure the material master field MARA-EAN11 is left blank for materials that should not have barcode populated at 0200-COD_BARRA.
