Symptom

Can SAP Replication Server 16.1 replicate to Oracle 19? 

Environment

  • SAP Replication Server (SRS) 16.1
  • SAP Replication Server Options for Oracle (RSO) 15.7

Cause

  • Mainframe Connect 15.0 SAP 3408368 - End of mainstream maintenance for Sybase EnterpriseConnect Data Access 15.7
  • ECDA 15.7 SAP note  (use this also for DCO)
  • 3059657 - Using SAP Data Integrator/SAP Data Services as an Alternate Solution to SAP Replication Server Heterogeneous Replication (from DB2 UDB, Microsoft SQL Server or Oracle) to SAP ASE
  • ECO ExpressConnect for Oracle 

Resolution

We don’t have new development specific for ECO, hence we don’t have any testing to Oracle for 16.1. 

See Also

3099155 - SAP Replication Server, real-time loading edition 16.0 Mainstream Maintenance Support Statement

Keywords

ECO, 16.1
