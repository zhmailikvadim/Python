Symptom

When the rule Enable traffic light bidding is set to Yes in an event, the project owner enters item price range values at which suppliers will be shown a green, yellow, or red traffic light icon to indicate how competitive their bid is. Do suppliers have the ability to see what values the project owner has set for each traffic light's price range?

Resolution

If the rule Show lead bid to all participants is set to Yes or After participant's first response is accepted, suppliers can see the value ranges you defined for green, yellow, and red in the bidding rules. This provides suppliers with immediate feedback and enables them to improve their bids and ensure their bid is considered during the selection process.

If you do not want suppliers to see the traffic light range values, you can consider setting the rule Show lead bid to all participants to No, or the rule Show bid graph to all participants to No.

See Also

You can also confirm whether traffic light range values are visible to suppliers in your event by using the Preview as supplier option. 

Please refer to Enable traffic light bidding (Event Rules Reference) for more information on traffic light bidding functionality. For instructions on how to enable and set up traffic lights, please see Enabling Traffic Light Bidding in English Reverse Auctions in Guided Sourcing.
