Symptom

The "Ready for Warehouse Processing" status is not set for a document in which both the product and the package are partially used in packaging.

Other Terms

ICR, inbound cargo registration, goods receipt, ready for warehouse processing

Reason and Prerequisites

You use SAP S/4 HANA for manufacturing logistics or this SAP Note is a prerequisite of another SAP Note.

Solution

Implement the attached correction.

  • Duplicate entries are being allowed through upload but not creating through the workbench.
  • When there is a line with an error and you select the “Transfer Error” option, there is no error message in the Excel error file.

Other Terms

Flexible Group, Upload , Duplicate, Error File

Reason and Prerequisites

This is caused by program error.

Note : SAP Notes#3143178 , #3141122, #3194798 are available for V/4 Release

Solution

Please implement the correction instructions attached. These corrections are only valid if you have applied the Support Packs.

Release 6.1 - Base Package

Release 60E - Support Pack 1, Support Pack 2, Support Pack 3, Support Pack 4, Support Pack 5, Support Pack 6 and Support Pack 7

Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object.
