Symptom

Color inconsistency occurs for identical measures in pie charts after rank changes in SAC

Environment

 SAP ANALYTICS CLOUD 2025.21.9

Reproducing the Issue

  1. Login to sac
  2. Create a new story 
  3. Add a pie chart with measures, dimension and color palette
  4. Add a new pie chart with different measure but same dimensions and color palette.
  5. Change the rank 

Notice: The color representing the same measure in both charts is inconsistent.  

Cause

Colors are applied based on the sequence in which data is loaded  

Resolution

Manually assign colors to individual values to ensure color consistency across pie charts.
