Symptom

How to change the "To" domain in email destination for all recurring instances with Java SDK?
Example: from ...@abc.com to ...@sap.com

Environment

SAP BusinessObjects Business Intelligence Platform 4.x

Resolution

To change the receiver's email address domain for all Crystal and Webi recurring schedules, use the attached code "change_destination_domain.jsp".
