Symptom

In trip expense statements and during processing of separation allowance periods, the "Deductions" tab page is available for entering meals that were provided to the traveler. Under the table of meals, there are the buttons "All breakfasts", "All lunches", "All dinners", "All nights for" and, if necessary, "All food vouchers" for faster entry.
Depending on which meals have already been selected in the table, these buttons either have the function Select All Fields of the Corresponding Type or Deselect All Fields. However, this function is indicated only by an icon on the button. In addition, no tooltips are defined for the buttons. As a result, the functions of the buttons of screen readers cannot be perceived and the buttons are not accessible.

Other Terms

buttons, select, delete, remove, accessibility, screen reader, readable, tooltip, breakfast, lunch, dinner, food coupon

Reason and Prerequisites

Buttons on the "Deductions" tab page are not accessible.

Solution

Import the specified HR Support Package or implement the attached correction instructions.
If required, also refer to the activities in the manual correction instructions.

After you implement the corrections, tooltips are assigned to the buttons mentioned above that adapt to the functions of the buttons. For example, the tooltip for the breakfast deductions is "Select all breakfasts" or "Delete all breakfasts".
