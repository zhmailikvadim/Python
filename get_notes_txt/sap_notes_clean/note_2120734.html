Symptom

After you implement SAP Note 2115158, you can no longer create new routings from SAP ME in the ERP system. The creation of the routing fails and the system issues message CO_MES_INT 060 ("Operation assignment of the components could not be determined (code 1)").

Other Terms

CO_MES_ROUTING_CREATE_PUSH, component assignment, BAdI, CO_MES_INT060, IF_CO_MES_INT_ROUTING_INBOUND~BEFORE_ROUTING_MAINTENANCE

Reason and Prerequisites

There is a program error in SAP Note 2115158. This SAP Note is also part of the following Support Packages:

SAPKH60514
SAPKH60615
SAPKH61609
SAPKH61708

Solution

Implement the attached correction instructions or import the specified Support Package.

After you implement the correction, errors that occur when determining the existing component assignment for the routing no longer result in the termination of routing creation. The system issues message CO_MES_INT 060 as a warning in the log.
