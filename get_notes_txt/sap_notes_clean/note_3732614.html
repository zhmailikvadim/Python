Symptom

HTTP 500 Internal Server Error occurs in Promotion Management when exporting a WebI report to LCMBIAR in BI 2025 Patch 8.

Environment

SAP BusinessObjects Business Intelligence platform 2025 Patch 8



Reproducing the Issue

  1. In Promotion Management, create a new job and add objects.
  2. Select any WebI report.
  3. After the screen finishes loading, an HTTP 500 Internal Server Error occurs.

Cause

It should be related to the bug in BI 2025 Patch 8.

3724948 - Publications fail / cluster communication fails after upgrade to 4.3 SP05 Patch 400 / BI 2025 Patch 800

Resolution

The issue does not occur in any versions other than Patch 8.

See Also

3724948 - Publications fail / cluster communication fails after upgrade to 4.3 SP05 Patch 400 / BI 2025 Patch 800
