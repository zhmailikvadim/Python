Symptom

  • Report sides are cut off when report is printed from Crystal Reports desginer

Environment

  • SAP Crystal Reports 2011
  • SAP Crystal Reports 2013
  • SAP Crystal Reports 2016

Reproducing the Issue

  1. Create Crystal report
  2. In the printer driver set "Letter" page type
  3. Print it to A4 pages
  4. Observe that sides are cut off

Cause

  • The "Letter" and "A4" paper types have different side ratio
  • If the paper type is not set properly in the printer driver, the page won't be scaled properly

Resolution

  1. When printing, go to preferences
  2. Change the paper format to the actually used one (A4 for example)
  3. Apply changes
  4. Print the report
