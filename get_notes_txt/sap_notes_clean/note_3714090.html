Symptom

  • Error Message No V1227 - New Pricing is no longer possible.
  • The system does not pick the latest price from the Info Record for price-changed purchase orders that are partially delivered.
  • This issue occurs specifically for partially delivered purchase orders where the latest price should be determined automatically.

Environment

  • SAP S/4 HANA Cloud Public Edition

Reproducing the Issue

  1. Create a Purchase Order.

  2. Post a partial delivery for the Purchase Order in MIRO.

  3. Update the price in the Purchasing Info Record.

  4. Verify that the partially delivered Purchase Order does not automatically adopt the updated price from the Info Record.

  5. Attempt to manually apply the new price by selecting Update under the Conditions tab in the Purchase Order.

  6. System will give Error Message no. V1227 - New pricing is no longer possible.

Cause

The automatic price update is not possible after GR is already posted.

Resolution

  • This is standard behavior. The net price can be changed manually, but should check whether this change possibly requires to cancel follow-on documents.

See Also

  • SAP Note 3168171 - New price determination is no longer possible automatically after follow-on documents
