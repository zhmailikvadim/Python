Symptom

Data Point Comments disappear after refreshing master data in the model in SAC.

Environment

SAP Analytics Cloud 2025.21.11

Reproducing the Issue

    1. Login to SAC.
    2. Open the story.
    3. Add Data Point comments.
    4. Now go to the source model.
    5. Update the data source: Data Management - Import Data From - File - Finish Mapping,
    6. Re-open the story/refresh the page.
    7. Notice, that after the data source refresh Data Point Comments disappeared.

    Cause

    This issue is currently under investigation by development.

    Resolution

    This issue is currently under investigation by development.
