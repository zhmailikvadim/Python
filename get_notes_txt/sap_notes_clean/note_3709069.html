Symptom

You are creating warehouse tasks for a Production Material Request (PMR). You want to take over the date Requirement Start Date for Component (SCMPREQ) from the PMR item as Planned Closing Time (PICK_COMP_DT) into the warehouse task. To achieve this behavior, you are using the example class /SCWM/CL_EI_MFG_STAGE_WT_SO in a BAdI implementation. The code does not always fill the date as expected.

Other Terms

/SCWM/ES_MFG_STAGE, /SCWM/EX_MFG_STAGE_WT_SO, /SCWM/IF_EX_MFG_STAGE_WT_SO

/SCWM/EI_MFG_STAGE_WT_SO, /SCWM/CL_EI_MFG_STAGE_WT_SO

PICK_COMP_DT, SCMPREQ

Reason and Prerequisites

Program error

Solution

Implement the attached Correction Instruction (CI) or the relevant Support Package.
