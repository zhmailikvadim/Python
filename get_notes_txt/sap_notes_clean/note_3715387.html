Symptom

  • Syntax error encountered: "Include report LEDOT6T00 not found" during the implementation of SAP Note 3651634.
  • SAP Note 3651634 remains incompletely implemented.
  • SAP Note 3651634 is a prerequisite for SAP Note 3647347.

Environment

  • SAP S/4HANA
  • SAP ERP

Reproducing the Issue

  1. Attempt to implement SAP Note 3651634.
  2. Encounter syntax error: "Include report LEDOT6T00 not found".

Cause

Resolution

  1. Proceed with the implementation of SAP Note 3647347 - Enable Creation of Electronic Documents for the Goods Receipt for Purchase Order in MIGO.
  2. Then execute the UDO report NOTE_3651634.

Keywords

syntax error, LEDOT6T00, SAP Note 3651634, SAP Note 3647347, electronic document processing, S/4HANA, FRAMEWORK_S4CORE_11.xml, SAP Note Analyzer, UDO report, NOTE_3651634
