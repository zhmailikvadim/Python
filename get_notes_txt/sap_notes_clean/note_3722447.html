Symptom

In lean services procurement integration between SAP S/4HANA and SAP Business Network Commerce Automation using the SAP Integration Suite, managed gateway for spend management and SAP Business Network, add-on for SAP S/4HANA, SES creation was failing when submitting an SES for a lean service from the supplier's Business Network. Due to this failure, the SES was not getting created.

When sending Purchase Order for Leasn Service, the Vendor is sent to SBN with leading zero, causing issue.

Other Terms

CL_S4CINT_SES_IMPL, CL_S4CINT_PURCHASEORDER_IMPL

Reason and Prerequisites

The issue occurred because of an erroneous code.

Solution

The code has been corrected to resolve the issue, and the SES is being created successfully.

The payment amount is migrated with incorrect decimal places.

Other Terms

SAP S/4HANA Migration cockpit, S4HANA, Direct Transfer, TRM Interest Rate Instrument, Decimal places, amount field

Reason and Prerequisites

Program error.

Solution

Implement this SAP note in all participating source systems and create a new project.
