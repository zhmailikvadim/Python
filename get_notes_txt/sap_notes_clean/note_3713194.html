Symptom

Email Notification Error "(FAIL: email failed)" in Access Request Audit Log when using Graph API.

Environment

SAP Cloud Identity Access Governance 2.0

Reproducing the Issue

You have configured IAG notifications using Graph API. However you receive the following error message:

Notification for request <request#> was sent to <email.address> (Requester) (FAIL: email failed).

Resolution

  1. Ensure that you have configured the bpmworkflowruntime_mail destination in BTP as described in the IAG Admin Guide for Microsoft Graph API  Configuring Notification E-Mail Server | SAP Help Portal 

  2. Ensure that you have all the required permissions on the Graph API side in azure configuration. For example, Mail.Send permissions should be granted with type 'Application' instead of 'Delegated'. You can refer to the screenshots below.
