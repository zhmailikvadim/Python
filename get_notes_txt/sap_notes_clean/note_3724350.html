Symptom

  •  WebIntelligence processing server crashes when viewing documents with option "Auto-merge dimension" enables

Other Terms

webi, webintelligence, WIPS, restart, coreinfo, shut down, unexpected, Termination, crashing, stop, calculation

Reason and Prerequisites

Environment:

  • SAP BusinessObjects Business Intelligence platform 4.3 Support Package SP05 Patch3
  • Web Intelligence

Steps To Reproduce:

  1. Create document in a version inferior than SAP BusinessObjects Business Intelligence platform 4.3 Support Package SP05
  2. Add multiple dataproviders and make sure to add some numeric dimensions as well
  3. Enable "Auto-merge dimension" option in document properties and save the document
  4. Open the document in SAP BusinessObjects Business Intelligence platform 4.3 Support Package SP05 Patch 3 or later
    1. >> Web Intelligence processing server crashes

Cause:

  • This issue is caused by a defect in the product.

Solution

This issue is fixed in the patches listed in the "Support Packages & Patches" section below.
Due to different internal processes, the "Support Packages & Patches" section of the SAP Note will be populated as below:

  • for fixes delivered in "Support Packages": immediately (even if the "Support Package" is planned to be released several months later)
  • for fixes delivered in "Patches": ONLY once the "Patch" is released AND available on SMP (not before)

For Business Intelligence Platform maintenance schedule and strategy see the Knowledge Base Article 2144559 in References section.
