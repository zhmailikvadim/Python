Symptom

When trying to add a subaccount in regions running on SAP Cloud Infrastructure(e.g. Europe (Rot) SAP EU Access) to Cloud Connector manually, the region cannot be found in the value help on Cloud Connector UI.

Environment

SAP Cloud Connector prior to version 2.19.0.

Reproducing the Issue

  1. Attempt to add a subaccount on Cloud Connector UI manually
  2. Look for the region from the list.
  3. The region cannot be found in the list

Cause

The regions on SAP Cloud Infrastructure are not added to Cloud Connector prior to version 2.19.0.

Resolution

Use one of the following approach to solve the issue:

  • Upgrade Cloud Connector to the latest version
  • Follow KBA 3501400 to add the subaccount using authentication data
  • If Cloud Connector cannot be upgraded or authentication data cannot be downloaded from BTP Cockpit at the moment, a workaround is to create a custom region with the hostname listed in the document below:
    Region Hosts and IPs

    For example, for region 'Europe (Rot) - SAP (EU Access Only)', create a custom region with host 'mtls.eu02-l-c.uc-live-eu.shoot.live.k8s-hana.ondemand.com', and select this region when manually adding the subaccount to Cloud Connector.
