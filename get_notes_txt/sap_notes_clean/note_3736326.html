Symptom

Pixel Perfect Talent Card (PPTC) templates display titles only in English even when the user’s language is set to French (Canada).

Environment

SAP SuccessFactors Employee Profile: Pixel Perfect Talent Card

Reproducing the Issue

  1. Set the user interface language to French (Canada).
  2. Open a user’s People Profile.
  3. Select All actions and choose Create talent profile.
  4. In the dropdown, observe that the PPTC titles remain in English instead of French.

Resolution

PPTC template titles (file names) are not localized by user language. Renaming a PPTC sets a single static title used for all locales, and locale-based title translations via PPTC XML are not supported.

Kindly create a enhancement request via influence tool (https://www.sap.com/germany/about/customer-involvement/influence-adopt.html) -> SAP Continous Influence -> SAP SuccessFactors Employee Central (https://influence.sap.com/sap/ino/#/campaign/85).  

Please refer  https://launchpad.support.sap.com/#/notes/2090228
