Symptom

The GR-Based IV flag is not populated automatically while creating Purchase order using ME21N.

Environment

•Materials Management (MM)
•SAP ERP Central Component
•SAP ERP
•SAP enhancement package for SAP ERP
•SAP enhancement package for SAP ERP, version for SAP HANA
•SAP S/4HANA

Reproducing the Issue

  1. Create a PO with a valid info record
  2. Check PO item level and found GR-Based IV flag is blank.

Cause

  1. There is an info record used here where the GR-Based IV is not set.
  2. System will copy GR-Based IV flag during PO creation by following this sequence: reference document -> info record -> vendor master -> Manually set

Resolution

Set the GR-Based IV flag in the info record.

Keywords

PO, GR-Based IV, EKPO-WEBRE, WEBRE, me13, me23n
