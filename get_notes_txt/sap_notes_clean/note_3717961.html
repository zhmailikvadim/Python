Symptom

Error message: "Material not found in plant (Check entry)" - SAP Message CO010 occurs when attempting to generate an internal service order in the ARM process.

Environment

  • Sales and Distribution (SD)
  • SAP ERP 
  • SAP ERP Central Component 
  • SAP enhancement package for SAP ERP 
  • SAP enhancement package for SAP ERP, version for SAP HANA 
  • SAP S/4HANA 
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Access transaction VA03.
  2. Open the ARM sales order.
  3. Navigate to Returns Overview.
  4. Attempt to restart document creation for the in-house repair order.
  5. Observe the error message "Material not found in plant (Check entry)" - SAP Message CO010.

Cause

Custom configuration in requirement class is not foreseen in the ARM process. Specifically, the assembly type is configured with order type, which deviates from the standard ARM configuration.

Resolution

  1. Review the configuration of requirement class.
  2. Compare the custom configuration with the standard ARM configuration delivered by SAP. See KBA 2967776 -   Advanced Returns Management: Description of Delivered Configuration 
  3. Ensure that the assembly type and order type settings align with the standard ARM configuration (Item category IRR2 Requirement type SE and Requirement class 039) where these fields are left blank.
  4. If modifications are required to suit specific requirements, consult with a local SAP consultant for assistance in adjusting the configuration.
