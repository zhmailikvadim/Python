Symptom

When an Agreement Request is posted, the rule changes made in the document are not captured in the generated outbound IDoc. As a result, the outbound IDoc is created without the required rules segments.
This SAP Note addresses the underlying issue by correcting the rules comparison and data handling logic, ensuring that all rule changes are correctly captured and included in the outbound IDoc during Agreement Request posting.

Other Terms

IDoc, Rules Segment, Outbound, Agreement Request, /IRM/IPBBARM

Reason and Prerequisites

This is caused by program error

Solution

Please implement the correction instructions attached. These corrections are only valid if you have applied the following support packs.

Release 6.4 Base Package and Support Pack 1

Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object.
