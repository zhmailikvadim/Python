Symptom

Test failed for the default configuration 'XXXX' of datastore: 'XXXX':
ODBC call <SQLDriverConnect> for data source <XXXXXXXX> failed: <[SAP AG][LIBODBCHDB SO][HDBODBC] General error;-10719 Connect failed (invalid SERVERNODE 'XXXXX')>. Notify Customer Support.

Environment

SAP Cloud Integration for data services

Cause

Two environment variables were not correctly set:
ODBCINI=
HANA_DRIVERMANAGER=

Resolution

Ensure environment variables set correctly.

Example:
ODBCINI=/usr/sap/XXXXX/CPI-DS/bin/ds_odbc.ini 
HANA_DRIVERMANAGER=/usr/sap/XXXXX/hdbclient/libodbcHDB.so
