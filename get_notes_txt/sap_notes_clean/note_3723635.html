Symptom

Error: HTTP response 403: Forbidden. Access denied to system XXXXX.XXX.XXXX.XX:XXXXX. In case this was a valid request, ensure to expose the system correctly in your cloud connector.' when communicating with https://ADDRESS_IS_SET_VIA.HEADER”

Cause

The error occurs because the requested backend system host and port is not exposed in the SAP Cloud Connector configuration.

  • The backend system is not added under Cloud To On-Premise.

  • The virtual host/port mapping is incorrect.

  • The protocol mismatch (HTTP vs HTTPS).

  • The subaccount ID is incorrect.

  • The Cloud Connector tunnel is disconnected.

Resolution

To resolve this issue, ensure that you are using the SAP-owned subaccount while configuring the SAP Cloud Connector.

For detailed SAP Cloud Connector configuration steps, see SAP KBA 3440354 - How do I setup the cloud connector?
