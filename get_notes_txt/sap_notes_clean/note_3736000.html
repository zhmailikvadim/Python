Symptom

The application currently uses DDIC SQL views (IPPPRODNSUPRVISR and IPPWORKCENTER), which need to be replaced with their corresponding CDS entities (I_ProductionSupervisor and I_WorkCenter). This adjustment ensures alignment with modern ABAP development guidelines.

Reason and Prerequisites

Standard SAP development guidelines require consuming the CDS entity directly rather than the generated DDIC SQL view. Because the underlying SQL views are being phased out in favor of CDS entities, the application code must be adjusted to maintain compatibility and adhere to best practices.
