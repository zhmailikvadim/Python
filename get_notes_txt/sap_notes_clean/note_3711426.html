Symptom

You are using UDF forecast and have location products where there is a consumption reference assigned, and the UDF forecast and variance are retrieved for the dynamic minimum stock calculation. The UDF variance values retrieved are undefined. In F&R, the undefined variance values result in the calculation of the dynamic minimum stock also being undefined.

The dynamic minimum stock is calculated according to the following formula:

Dynamic Minimum Stock    =    Dyn. Min. mean value    x    weight of Dyn. Min. mean value / 100    +   Dyn. Min. deviation    x    weight of Dyn. Min. deviation / 100

Other Terms

UDF, variance, UNDEF, dynamic minimum stock, F_FORECAST_UDF, FORECAST_UDF.saf, /FRE/FU_CL_FRP_UDF, CONVERT_VAR_TO_DEC20_6, MAP_UDF_RESULTS, SalesVarianceAll, SalesVarianceExcludeItemPredictors, SalesVarianceExcludeStorePredictors, SalesVarianceExcludeAllPredictors

Reason and Prerequisites

If there is no sales history for a product location due to the consumption reference assigned, UDF provides undefined values in the 4 variance values for dynamic minimum stock.

Solution

Implement the corresponding correction instruction.

The variance value of the dynamic minimum stock will now be set to 0 if the UDF variance is undefined. The calculation of the dynamic minimum stock will therefore be based only on the mean value of the dynamic minimum stock.
