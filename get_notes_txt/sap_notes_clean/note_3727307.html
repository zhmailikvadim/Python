Symptom

  • When running the Single-Stage Inventory Optimization function, it fails with a 'user error' and the additional error: "Missing Forecast inputs".
  • It is not understood why this error occurs because the forecast is not an input to the function, per the Help portal.

Environment

SAP Integrated Business Planning (SCM-IBP) for Supply Chain

Cause

  • Check the application job for any filter that limits the run to an internal product/location.
  • Though forecast data is not direct input, the planning filter/subnetwork combination must still include records that are in the Demand Stream master data (planning level WKPRODLOCCUSTGROUP).
  • The algo (AFL) for the Single-Stage function still expects Demand Stream and customer forecast inputs to be able to construct the network up to the filtered product/location.

Resolution

  • Construct the filter in such a way that Demand Stream records are included.
  • Additionally, Development is reviewing the log messages to make sure it is as clear as possible.
