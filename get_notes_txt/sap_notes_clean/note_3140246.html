Symptom

Symptom

You are running SAP LT Replication Server and you would like to replicating an object. This note will enhance the method that gets the metadata information of the object in order to return some additional information. This will be used later in the central system to provide some additonal messages or information to the user.

Other Terms

LT Replication Server, SLT, replication, replicator, trigger, realtime, S/4HANA On-Premise 2020-2021

Reason and Prerequisites

  • The source system is on S/4HANA 2021 using SAP NetWeaver 7.56 and SAP_ABA 75G FPS00-01.
  • The source system is on S/4HANA 2020 using SAP NetWeaver 7.55 and SAP_ABA 75F FPS00-SPS03.

Solution

Implement the required corrections on the source and SLT system in accordance with the correction instructions.
Replication jobs have to be stopped during SAP Note implementation: no

Manual Steps: None.

New BADIs for these fields have been added also

Solution

As a general rule, SAP recommends that you install a solution by applying a Support Package. However, if you need to install a solution earlier, use the Note Assistant to implement the correction instruction. You can find more information about the Note Assistant in SAP Service Marketplace, under service.sap.com/note-assistant.
