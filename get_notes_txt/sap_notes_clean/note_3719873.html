Symptom

  • Attempting to delete sequence from the CT method results in an warning message: "Entries to be deleted: 1 of 1 of these with dependent entries - all entries or You can either delete all entries or only those which have no dependent entries".
  • The system does not specify which dependent objects need to be considered for deletion of sequence.

Environment

  • SAP Fiori for SAP S/4HANA Finance
  • SAP S/4HANA Finance for Group Reporting

Reproducing the Issue

  1. Attempt to delete sequence from a CT method.
  2. Observe the warning message like below: "Entries to be deleted: 1 of 1 of these with dependent entries - all entries or You can either delete all entries or only those which have no dependent entries".

Cause

The issue is caused by dependent objects stored in TF411 regarding the difference items, which prevent the deletion of sequence directly.

Resolution

  • When select 'Without dependent re', then all the relevant data of that sequence in the table TF410(Currency Translation Methods: Entries) will be deleted.
  • When select 'All Entries', then all the data related to this sequence in both TF410 and TF411(Translation Methods: Entries for Subassignments) will be deleted.
