Symptom

You run Trial Balance for a specified branch and you choose to show opening balance. In some cases business partners are not displayed in the Trial Balance, despite they have postings from period prior to the report dates range.

Reproducing the issue

Prerequisites:

You are working in a database where Multiple Branches is enabled (Administration → System Initialization → Company Details → Basic Initialization).

You have defined few branches (Administration → Setup → Financials → Branches).

Scenario 1:

  1. Business Partner BP1 is created and assigned to Branch 1 (Business Partner → Business Partner Master Data → General → Branch Assignment).
  2. BP1 has the following transaction:
    Journal entry (JE) 101, 31.12.2019, 500.00 GBP, JE line assigned to Branch 1.
  3. Go to Reports → Financial Reports → Financials → Trial Balance, fill selection criteria as follows:
    - Select BP checkbox
    - Date: Posting Date, From: 01.01.2020, To: 31.12.2020
    - Don't select Hide Zero Balanced Acct checkbox
    - Don't select Hide Acct with No Postings checkbox
    - Select Opening Balance for Period checkbox
    - Select     OB from Start of Company Activity radio button
    - Choose Expended button → Select Branch 1 in Branch field and choose OK.
  4. Choose OK to execute the trail balance report.

Expected behavior:

BP1 is displayed in the Trial Balance report showing open balance of 500.00 GBP.

Actual behavior:

BP1 is not displayed in the Trial Balance report.

 

Scenario 2:

  1. Business Partner BP2 is created and assigned to Branch 2 (Business Partner → Business Partner Master Data → General → Branch Assignment).
  2. BP1 has these transactions:
    Journal entry 201, 31.12.2019, 500.00 GBP, JE line assigned to Branch 2.
    Journal entry 202, 21.10.2020, 600.00 GBP, JE line assigned to Branch 2.
  3. Go to Reports → Financial Reports → Financials → Trial Balance, fill selection criteria as follows:
    - Select BP checkbox
    - Date: Posting Date, From: 01.01.2020, To: 31.06.2020
    - Don't select Hide Zero Balanced Acct checkbox
    - Don't select Hide Acct with No Postings checkbox
    - Select Opening Balance for Period checkbox
    - Select     OB from Start of Company Activity radio button
    - Choose Expended button → Select Branch 2 in Branch field and choose OK.
  4. Choose OK to execute the trail balance report.

Expected behavior:

BP2 is displayed in the Trial Balance report showing open balance of 500.00 GBP.

Actual behavior:

BP2 is not displayed in the Trial Balance report.

 

Scenario 3:

  1. Business Partner BP3 is created and assigned to Branch 3 (Business Partner → Business Partner Master Data → General → Branch Assignment).
  2. BP3 has these transactions:
    Journal entry 301, 31.12.2019, 500.00 GBP, JE line assigned to Branch 3.
    Journal entry 302, 15.01.2020, 700.00 GBP, JE line assigned to Branch 3.
  3. Go to Reports → Financial Reports → Financials → Trial Balance, fill selection criteria as follows:
    - Select BP checkbox
    - Date: Posting Date, From: 01.07.2020, To: 31.12.2020
    - Don't select Hide Zero Balanced Acct checkbox
    - Don't select Hide Acct with No Postings checkbox
    - Select Opening Balance for Period checkbox
    - Select OB from Start of Company Activity radio button
    - Choose Expended button → Select Branch 3 in Branch field and choose OK.
  4. Choose OK to execute the trail balance report.

Expected behavior:

BP3 is displayed in the Trial Balance report showing open balance of 1200.00 GBP.

Actual behavior:

BP3 is not displayed in the Trial Balance report.

Solution

This is correct system behavior.

When there are no BP transactions matching selection criteria / expand selection criteria in dates range selected for Trial Balance, BP will not be shown in the Trial Balance.
