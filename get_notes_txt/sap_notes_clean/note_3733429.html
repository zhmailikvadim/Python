Symptom

The costs displayed in the "CO Sales and Distribution" allocation control do not match the expected costs. Personnel costs that are reported to the parent cost center via a personnel pool cost center are also displayed here as personnel costs of the parent cost center.

The primary costs cannot be recognized in the report because they are mixed with the secondary costs.

Program error

Solution

Implement the source code corrections.

Troubleshooting Description:

This SAP Note introduces two new columns in the report that display the primary and secondary costs.

In addition, the SPRO parameter VK1 “Clearing Control CC Sales Ignore PPKST for Parent CC” (SAP Dairy Management by msg -> Dairy Cross Functions -> General Settings -> System Settings) enables you to ignore HI entries that report personnel costs from the personnel pool cost center to the parent cost center. For this, the parameter must be added and activated in the specified Customizing.
