Symptom

When deploying a project in SAP Build Process Automation, the following error appears:

Errors occurred while deploying an artifact to a runtime. (component: Process Builder)
Error during execution in tenant context
### Error updating database. Cause: com.sap.db.jdbc.exceptions.JDBCDriverException: SAP DBTech JDBC: [287]: cannot insert NULL or update to NULL: Not nullable "TYPE_" column
### The error may exist in org/flowable/job/service/db/mapping/entity/Job.xml
### The error may involve org.flowable.job.service.impl.persistence.entity.JobEntityImpl.insertJob-Inline
### The error occurred while setting parameters
### SQL: insert into ACT_RU_JOB (...) values (..., NULL, ...) 

Environment

SAP Build Process Automation

Resolution

  • This error indicates a temporary issue at SBPA's cloud side.
    • For further information about outages and RCAs, the Cloud Availability Center is the correct channel. Refer to KBA 2443785 - How to receive email alerts for SAP BTP availability.
  • For any further queries, reach out to the Account Executive. Refer to KBA 2626344 - Who is the SAP Account Manager for my company?
