Symptom

  • An error occurs on "Step 1b - write message to message queue".
  • Error message: javax.script: java.lang.Exception: com.sap.it.op.b2b.monitor.api.B2BSubmitException: Payload is not set on BusinessDocumentPayload attached to document with id <>@ line <>in OrphanedInterchangeErrorEvent.groovy, cause: com.sap.it.op.b2b.monitor.api.B2BSubmitException: Payload is not set on BusinessDocumentPayload attached to document with id <>.

Environment

  • B2B Scenarios with Trading Partners
  • Cloud Integration
  • Integration Suite

Cause

The issue was caused by the payload being set as File Attachment instead of raw Data.

Resolution

  1. Verify the payload settings in the deployed agreement.
  2. Change the payload setting from File Attachment to raw Data.
  3. Test the configuration to ensure the error is resolved.
