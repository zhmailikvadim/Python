Symptom

SAP note 3701276 provides a correction so that a warning message is displayed when creating a Shop Floor Routing if this is required by the material status.

The warning message shall also be issued in release check when changing a shop floor routing, when copying a Shop Floor Routing and when creating a new version of a shop floor routing.

Other Terms

MSFR4, MSFR2, Shop Floor Routing, Routing type V, message CP 089

Reason and Prerequisites

Coding error.

Solution

Implement the correction instruction.

The warning will not be raised in order-specific routings.

The check on the material status is only executed in Shop Floor Routings.

When creating a new Shop Floor Routing, the warning will be displayed immediately. When copying a Shop Floor Routing or creating a new version of a Shop Floor routing, the warning will not be shown immediately, but only in the release check. In contrast to standard routings, shop floor routings have lifecycle management (via versions), so it is sufficient to display the warning in the release check.
