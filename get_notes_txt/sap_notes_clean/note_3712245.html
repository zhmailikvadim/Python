Symptom

When you try to reverse an AuC settlement of an investment measure, error KD 211 'no COBRB found 0000001' is issued.

Other Terms

 AUC JHR RKA_AUC_DELETED_DOUBLE KD042 KD211

Reason and Prerequisites

Business function 'Universal Parallel Accounting' is active.

The investment measure covers multiple fiscal years and settlement reversals have been executed. AuC settlement rules which have not yet been reversed have been deleted.

You want to use a program RKA_AUC_DELETED_DOUBLE from note 3446288 to recreate the deleted AuC rules, but the program recreates the rule with the wrong LFDNR.

Solution

Implement the correction instruction and run the program RKA_AUC_DELETED_DOUBLE to recreate the deleted AuC rule.
