Symptom

  • The 'Possible Match' button is listed on the Profile Workers in question
  • The Profile Workers in question have different Security IDs
  • The 'Security Information' Company Configuration is enabled

Cause

The first and last names of the Profile Workers in question are the same

Resolution

  • When the 'Security Information' Company Configuration is enabled, the application will consider the first and last names OR the Security IDs of the Profile Workers in question
  • If either the first and last names or the Security IDs are same, the 'Possible Match' button will be listed on the Profile Workers in question
