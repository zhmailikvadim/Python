Symptom

Error "This external code xxx already exists, please try another one" pops up during incremental import of organization information

Environment

  • SuccessFactors Employee Central

Cause

Check the external code mentioned in the error message, and try to pinpoint this external code in manage data, you will find that the external code is actually incorrect.

For example one letter should be in lower case, but it is in upper case in the import file.

Resolution

Correct the external code of the specific organization
