Symptom

Customer specific search help for WBS objects does not find any results.

Other Terms

(PLM-41642),  /DSCSAG/SMSH_RESULT_GET

Reason and Prerequisites

SAP search help engine handle TREX- based search helps and usual ones differently. In the Note 3228775 has been implemented workaround to fix this difference and support TREX based search helps together with regular search helps. Later testing, however, discovered that this workaround has side effects and does not working properly in 100% regular search helps (including customer specific implementations with search help exits).

This Note restrict solution from 3228775 to TREX based search helps only - i.e. now regular search helps handled in SAP ECTR as before.

Solution

Function module /DSCSAG/SMSH_RESULT_GET has been adjusted.

Hardcoded the "filtering out" only for search aids that have "TREX" in their name.
