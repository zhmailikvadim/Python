Symptom

It is not possible to create a custom field with a Date (DATS) data type in the "Accounting: Coding Block business context (FINS_CODING_BLOCK)". 

Environment

SAP S/4HANA Cloud Public Edition

Cause

In the "Accounting: Coding Block business context (FINS_CODING_BLOCK)", note that this functionality is limited to 20 fields, with a maximum of 400 characters in total. Only the "CHAR" and "NUMC" data types are supported for this purpose and not "Date" type DATS is offered.

Resolution

For scenarios requiring a "Date" type, consider using the "Accounting: Journal Entry Item(FINS_JOURNAL_ENTRY_ITEM)" business context instead.
