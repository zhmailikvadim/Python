Symptom

When doing costing for production order, incorrect error message is shown in the log like MM124 "Valuation data for material & for sales order item & & does not exist".

Environment

  • Financial Accounting (FI)
  • Controlling (CO)
  • SAP R/3
  • SAP R/3 Enterprise
  • SAP ERP Central Component
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA Finance
  • SAP S/4HANA

Reproducing the Issue

  1. Execute T-Code CO02;
  2. Input order number and press Enter;
  3. Press  Costing button.

Cause

This issue is caused due to missing of note 2881728.

Resolution

 Apply note 2881728 in the system.

If customer system version has already covered note 2881728, please apply note 3157727.

See Also

  • SAP Note 2881728 - Wrong error message collected in cost calculation log for production order
  • SAP Note 3157727 - Dynamic creation of sales order cost component split in case of cost component structure change

Keywords

CO01, CO02, CSER, KL013, CR028, work center, routing, COR1, COR2, process order, MM, 124
