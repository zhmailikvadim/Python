Symptom

It is expected that as of April 27, 2026, the technical conditions for the electronic transfer of E-Bilanz messages will be changed in accordance with the specifications of the regulator: Software products that use the regulator component "ERiC" (ELSTER Rich Client) in Version 42 or older will no longer be accepted by the receiving server. You must use ERiC in Version 43 (or higher) instead.

Information: This change applies regardless of the used taxonomy version.

Other Terms

SAP Disclosure Management, E-Bilanz, DM, ERiC, ELSTER, SAP ERP client for E-Bilanz

Reason and Prerequisites

You use SAP Disclosure Management 10.1 Stack 2301 or an older version of SAP Disclosure Management (including 10.0).

-or-

You use SAP ERP client for E-Bilanz 1.0 SP03 Patch 22 (Build 1.0.3.65) or an older version of SAP ERP client for E-Bilanz.

Detailed information about the ERiC release cycle is available here.

Solution

ERiC Version 43 is used in the following releases:

  • SAP ERP client for E-Bilanz 1.0 SP03 Patch Level 23 (SAP Note 3691933)
  • SAP Disclosure Management 10.1 Stack 2302 (SAP Note 3722894)

 

Ensure that your SAP Disclosure Management and SAP ERP client for E-Bilanz installations are updated to a version that supports ERiC 43 (or higher) in good time before April 2026.
