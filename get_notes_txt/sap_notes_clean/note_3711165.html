Symptom

You use S/4HANA Group Reporting and you want to create new document type with Posting Level 00.

Once you try to save the new document type, an error message appears:

Message no. G0483: Document type for the posting of reported financial data is not unique.

Other Terms

CHECK_DOCTY, CACTA = '9'

Solution

Import of Support Package or advanced correction by implementation of correction instruction.
