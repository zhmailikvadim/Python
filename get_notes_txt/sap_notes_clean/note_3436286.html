Symptom

Several DataSources for BW extraction are no longer available with S/4HANA Finance as part of the product version SAP S/4HANA, on-premise edition 1511 and above

Other Terms

SFIN, extractor

Reason and Prerequisites

See the reasons given below.

Solution

The DataSources listed below have been deactivated:

  • 0FI_AA_001
  • 0FI_AA_002
  • 0FI_AA_003
  • 0FI_AA_004
  • 0FI_AA_005
  • 0FI_AA_006

Those DataSources were already outdated in ECC. Data sources 0FI_AA_11 and 0FI_AA_12 can be used instead.

0FI_AA_11: DataSource 0FI_AA_11 can be used but has the limitation that group assets are no longer supported (see note 2748735).

0FI_AA_12: Usage of DataSource 0FI_AA_12 requires implementation of SAP note 2408988. Refer to note 614804 concerning supported methods for simulated depreciation values.

Other Terms

HBRCCED0; CCED; Income Declaration; Informe de Rendimentos; Pensão; Alimony; Garnishment

Reason and Prerequisites

You have installed SAP note 2434339.

Solution

The changes described in this note will be included in an HR Support Package, as indicated in the item "Reference to Support Packages". 
 
The support package includes the following objects:

  • Changes in class CL_HR_BR_CCED

As a general rule, SAP recommends that you install a solution by applying a Support Package or CLC. However, if you need to install a solution earlier, use the Note Assistant to implement the correction instructions. You can find more information about the Note Assistant under https://support.sap.com/kb-incidents/notes/assistant.html.
