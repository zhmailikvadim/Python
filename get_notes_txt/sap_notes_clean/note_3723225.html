Symptom

  • No results are displayed when searching for deleted and dispatched handling units.
  • A memory overflow dump occurs when searching for deleted and dispatched handling units without filters.
  • The message "Cannot search dispatched or deleted handling units using the specified criteria" appears when additional filters are added to the selection criteria.
  • The issue occurs while using the BAdI /SCWM/EX_CORE_GM_UPDATE. 

Environment

  • SAP S/4HANA
  • SAP Extended Warehouse Management

Reproducing the Issue

  1. Open the /SCWM/MON transaction.
  2. Navigate to the node "Documents --> Handling units".
  3. Select the "Dispatched and Deleted Handling units" option.
  4. Press F8.
  5. Observe a memory overflow dump.
  6. Attempt to add additional filters such as product number, and note the message "Cannot search dispatched or deleted handling units using the specified criteria". 

Cause

  • If the method get_flag is implemented in the BAdI /SCWM/EX_CORE_GM_UPDATE, it may return X, indicating an attempt to read GI/GR-related historical data or deleted HUs from the support table /scwm/hu_dlv, which is not properly populated in this case.
  • The message "Cannot search dispatched or deleted HUs using HU Attributes or Stock Attributes" is standard behavior. During development, HU attribute and stock attribute searches for deleted and dispatched HUs were disabled. Many deleted HUs lack core information, such as creation dates, as they may have been temporary pick HUs. When deleted or posted flags are checked, the attribute section is immediately greyed out. 

Resolution

  1. Change the flag in the BAdI /SCWM/EX_CORE_GM_UPDATE to abap_false to ensure all database tables are read and data is displayed correctly. This method runs only during the monitor query and does not affect other processes.
  2. Run the report /SCWM/RP_TRACK_HU to build the helper table. Ensure the table is populated during each warehouse task confirmation and goods movement update using BAdIs.
  3. Refer to SAP Note 2232280 for detailed implementation instructions.
