Symptom

You hit an known issue described in APAR IT27940 (for Db2 11.1) or APAR IT31823 (for Db2 10.5).

You can not update the fix pack right now, and want to know how to temporarily turn off auto reorg for ITC tables.

Environment

SAP on IBM Db2 for Linux, UNIX, and Windows

Resolution

  1. Enter the web browser-based DBA Cockpit.

    * Make sure you use web browser-based dbacockpit. SAPGUI dbacockpit does not have this function.


  2. Configuration => Automatic Maintenance => Tab "Automatic Reorg" => Configure




  3.  Step 7 "Reorg" => Tick out "Reclaim Size for MDC/ITC Tables" => Continue => Execute (in next screen)



Please be noted that this is only a temporary workaround. You still need to apply the fixpack which includes the fix as soon as possible.

See Also

975352 - DB6: Configuring DB2 Auto Reorg for Space Reclamation and Index Cleanup

Keywords

auto reorg, exclude, turn off, ITC
