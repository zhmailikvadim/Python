Symptom

When users press Edit button in Approve Bank Payments to edit the Due Date or Instruction Key, there is no "Save" button to save these changes.

Other Terms

Approve Bank Payments, Save Button, F0673A

Reason and Prerequisites

New feature.

Solution

This SAP note is providing backend support for the new "Save" button delivered by frontend related SAP note 3706756.

For this solution to work, please implement a front-end related SAP note 3706756.

Implement the attached Correction Instruction or the relevant Support Package.
