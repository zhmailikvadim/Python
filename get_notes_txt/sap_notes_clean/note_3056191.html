Symptom

You are using CIF to create/change workcenter capacity details in SAP S/4HANA. You recognise that functionality to update workcenter description and functionality to delete intervals and shifts are not available.

Other Terms

Workcenter

Workcenter Capacity

Workcenter Description

Capacity Interval

Capacity Shifts

Reason and Prerequisites

New functionality and defect fixes done as part of this note.

prerequisite notes:

3066163

2996892

Solution

With this fix, new functionality is added to change description of existing workcenters and also to delete interval and shifts of existing workcenter capacities.

The methods are internally used by CIF.

As general rule, SAP recommends that you install a solution by applying a Support Package. However, if you need to install a solution earlier, use the Note Assistant to implement the correction instruction.
