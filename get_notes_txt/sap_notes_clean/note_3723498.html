Symptom

  • In Web browser, Error message: "500 Internal server error: view element with UIBB_CONTAINER_1 in view preview already exist"
  • Category               ABAP programming error
    Runtime Errors         UNCAUGHT_EXCEPTION
    Except.                CX_WDR_RT_EXCEPTION
    ABAP: Program          CL_WDR_VIEW===================CP
    Application Component  BC-WD-ABA-RUN
  • Affected application directs to /sap/bc/webdynpro/sap/configure_component

Environment

  • Floorplan Manager

Reproducing the Issue

  • Display configuration on SE80, web FPM_OIF_COMPONENT, and search for an application

Cause

  • Program error

Resolution

  • Implement the corrections provided in SAP Note 3718378 - FLUID OIF/GAF: Exception CX_WDR_RT_EXCEPTION -> View element with ID UIBB_CONTAINER_1 in view PREVIEW already exists
