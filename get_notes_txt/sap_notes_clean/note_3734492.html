Symptom

  • Values are truncated to 7 decimal places in SAC story

Environment

  • SAP Analytics Cloud 2026

Reproducing the Issue

  1. Create a story based on a data source that contains values with more than 7 decimal places, such as 1.123456789.
  2. Add a chart or table to the story.
  3. You'll notice that the value is displayed as 1.1234568, indicating that only 7 decimal places are shown.

Cause

  • The symptom is by design. 

Resolution

  • With the current system settings, maximum 7 decimal places are displayed.
  • If you require more decimal places, please submit an enhancement request by referring to the KBA below.
    2424376 - How do you create an enhancement request and provide feedback for SAP Analytics Cloud?

Keywords

cut off, more than, less than, part
