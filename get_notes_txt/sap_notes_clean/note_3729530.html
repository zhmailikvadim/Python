Symptom

Some of your outbound replication API's in SAP Cloud for Customer fail with the following error:

CM_APCRM_OPP_REPLICATE_OUT:An exception occurred.Business instance could not be determined

Environment

SAP Cloud for Customer

Reproducing the Issue

1. Logon to the C4C tenant
2. Navigate to General Settings > Web Service Message Monitoring
3. Filter as follows:
    - Error Text = CM_APCRM_OPP_REPLICATE_OUT:An exception occurred.Business instance  could not be determined
    - Party = [Communication System ID] 

Cause

The target communication system is not flagged as 'SAP Business Suite'.

Resolution

  1. In the C4C tenant navigate to Administrator> General Settings > Communication Systems
  2. Select the target communication system
  3. Click on Edit
  4. Turn on toggle 'SAP Business Suite'
  5. Change to tab 'System Instances'
  6. Enter a name in field 'Business System ID' making sure this value is unique, i.e. not used in any other communication system.
  7. Save and Close.

Keywords

CommunicationSystemID, BusinessSystemID, System Instance ID, Business System ID
