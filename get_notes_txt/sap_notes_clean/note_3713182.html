Symptom

You use Transaction Manager of Treasury and Risk Management. You manage Trade Request from TPITRO. When fulfill a trade request, the system may incorrectly issue message FTR_TPI_CORE(706) and FTR_TPI_CORE(707). The customizing setting for the two customizable messages is ignored.

Other Terms

MM_TREQ_CHECK_HELPER, READ_CUSTOMIZED_MESSAGE, FTR_TPI_CORE 097,  FTR_TPI_CORE 098, FTR_TPI_CORE 706, FTR_TPI_CORE 707

Reason and Prerequisites

Program Error

Solution

Preliminary correction via Correction Instruction or implementation of the corresponding Support Package.
