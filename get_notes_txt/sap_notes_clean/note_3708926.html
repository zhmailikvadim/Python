Symptom

Unable to delete report variable created under Data Mode in Web Intelligence document, getting below error message.

  • The resource of type "Variable" with identifier "CDP1.L1" does not exist.

Environment

  • SAP BusinessObjects Business Intelligence Platform 2025 Patch 06
  • Web Intelligence

Reproducing the Issue

  1. Login to BI launchpad.
  2. Create a new Web Intelligence document based on efashion Universe.
  3. Switch to 'Data mode'.
  4. Click on the three dots side the Query > Create a variable with any formula. (e.g.: =Upper (State))
  5. Attempt to delete the variable under Object Panel.
  6. Notice below error:
    • The resource of type "Variable" with identifier "CDP2.L2" does not exist

Cause

This behavior appears a product defect in current version.

Resolution

This issue is no more reproducible on SAP BI 2025 Patch 07 and appears to be fixed.

Keywords

data mode, webi, variable, variable does not delete, not exist
