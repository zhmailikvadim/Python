Symptom

Text file generated from Fiori App "Run Statutory Reports", or from report RPFIES_M347 for model 347, in one of the SAP products mentioned in the "Product" tab of this KBA, is rejected in Spanish Tax portal for 2025  

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental. 

Environment

  • SAP S/4HANA
  • SAP ERP Central Component
  • SAP enhancement package for SAP ERP 

Reproducing the Issue

Trying to upload at AEAT site the txt file generated from Statutory report ES_M347, or from report RPFIES_M347, results in several errors: 

  1. FRECH    El fichero se rechazó.No se importó ningún registro
  2. E010330 Caracteres no válidos "Número de convocatoria BDNS"

Cause

The steps to obtain the txt file might have been performed before the adaptation of the SAP software to the legal change for 2026.

Resolution

  1. Implement "Legal change" SAP Note 3708125 "RPFIES_M347, ES_M347 – File Format Changes for 2025 Reporting"
  2. And, either:   
    • Perform a new run at "Run Statutory Reports" Fiori app (for report category ES_M34), and download a new txt file.
    • Or execute transaction SE38 for report RPFIES_M347, and download the new txt file with report RFASLDPC (or transactions FIES_M347 and IDTW_DME_DOWNLOAD).

See Also

  • There are known restrictions for Legal Changes delivery.
  • On 31.12.2025, some releases, including SAP ERP 6.0 and SAP ERP 6.0 EHP 1 to EHP5, SAP S/4HANA Simple Finance, S/4HANA 1709, 1809, 1909 and 2020, concluded their "mainstream maintenance" phase, and entered on "customer specific maintenance" (CSM). For this reason, "Legal Change" SAP Note 3708125 "RPFIES_M347, ES_M347 – File Format Changes for 2025 Reporting" lacks of corresponding corrections instructions for mentioned products on CSM.
  • If having doubts regarding maintenance strategy for SAP applications and Legal Changes, please read SAP Note 52505 "Support after end of mainstream maintenance or extended maintenance"  
  • "SAP Business ByDesign" is a product handled by a different SAP team and component. Companies using "SAP Business ByDesign" that need to contact SAP Support for problems related to model 347, should choose Component SRD-FIN-GLO-TAX.
