Symptom

  • Error message: "Field 'M_ICFAUTH' is unknown" in class CL_HTTP_CLIENT.
  • Syntax error occurs in program "CL_HTTP_CLIENT================CP".
  • The issue prevents downloading the SAP Note 3591163 in SNOTE.

    Syntax error in program "CL_HTTP_CLIENT================CP        ".

Error analysis
    The following syntax error has occurred in the programme
     CL_HTTP_CLIENT================CP        :
    The pole ‘M_ICFAUTH’ is unknown. It is not included in any of the specified tables

Environment

  • SAP NetWeaver
  • SAP NetWeaver Application Server for SAP S/4HANA
  • ABAP PLATFORM - Application Server ABAP

Reproducing the Issue

  1. Attempt to download the SAP Note 3591163.
  2. Observe the syntax error message: "Field 'M_ICFAUTH' is unknown" in program "CL_HTTP_CLIENT================CP".

Cause

  • The attribute "M_ICFAUTH" was added in SAP Basis which is mentioned in the SAP Note 3591163.

Resolution

  1. Implement SAP Note 3089413 [CVE-2023-0014] Capture-replay vulnerability in SAP NetWeaver AS for ABAP and ABAP Platform.
  2. Implement SAP Note 3591163 [CVE-2025-42875] Missing Authentication check in SAP NetWeaver Internet Communication Framework.
  3. If the issue persists, uninstall and reinstall SAP Note 3591163.

See Also

SAP Note 3591163 - Missing Authentication check in SAP NetWeaver Internet Communication Framework.

SAP Note 3089413 - Capture-replay vulnerability in SAP NetWeaver AS for ABAP and ABAP Platform.
