Symptom

Warning message “Your Manager is not licensed.” is raised despite the fact that license is still valid.

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental." 

Environment

On-Premise SAP Enable Now Manager, HANA Edition.

Cause

Database entries can not be correctly read.

Resolution

Make sure connectivity from Manager to Database is working, if the password of the database user have expired then following entries can be found in the Manager log, in that case update the password and test again.

Failed to query process sync states
com.sap.db.jdbc.exceptions.JDBCDriverException: SAP DBTech JDBC: [516]: user is forced to change password: alter password required for user <DB USER>

If connection to database is fine and password validity also been check but still the same error is raising, reproduce the error again and collect the Manager Debug trace then report a case to KM-SEN-MGR and attach the trace file.

1861900 - Enabling Tracing for SAP Enable Now On-Premise Manager
