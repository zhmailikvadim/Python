Symptom

The user starts the transaction "Create Loss", fills all required fields, and confirms with Enter. The system terminates with a runtime error. The short text of the runtime error is "Access using a 'NULL' data reference is not possible".

The user starts the transaction "Create Account". The transaction does not display the tree control.

Other Terms

/MSG/R_U1

/MSG/R_A1

Dump

DATREF_NOT_ASSIGNED

Reason and Prerequisites

Prerequisite:
The tree control is activated using the setting "Environment - Settings - Tree Control".

Reason:
The system tries to read a data record that does not yet exist.

Solution

History:
February 16, 2026: New

Troubleshooting Description:
The system does not select data that does not yet exist. The tree control is displayed correctly and no program termination occurs.
