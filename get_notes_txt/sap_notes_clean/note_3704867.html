Symptom

The Goods Movement queue is stuck with the error:

Account determination for entry &1 &2 &3 &4 &5 not possible
Message no. M8147

Environment

  • SAP Extended Warehouse Management (SCM-EWM)
  • SAP Extended Warehouse Management Interface (SCM-EWM-IF)
  • SAP Service Parts Management (LO-SPM)
  • Logistics Execution (LE)

Reproducing the Issue

Execute SMQ2 and the queue will be stuck with the above error.

Cause

Missing Customizing

Resolution

The queue is stuck because the system cannot find a corresponding entry in the T030 table for the values specified in the queue error:

Table fieldvalue in the queue error
Chart of account&1
Transaction key (= Posting transaction)&2
Valuation grouping code&3
Account grouping code&4
Valuation class&5

Check and maintain the above combination and try to process the queue and try to re-process the queue.

Keywords

M8147, Account determination, SMQ2, T030, Customizing, Goods Movement
