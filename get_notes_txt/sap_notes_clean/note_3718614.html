Symptom

Some selected Items cannot be copied from the Service Call Related Documents Details window to Sales-related new documents A/R Invoice and A/R Credit Memo. You can only copy the following documents:

  • Sales Quotation
  • Sales Order
  • Delivery
  • Return Request
  • Returns
  • Transferred to Technician
  • Returned from Technician

Reproducing the Issue

Scenario:

  1. Go to Service → Service Call and create a Service Call document.
  2. On the Related Documents tab, click the Details button.
  3. In the Service Call Related Documents Details window, choose to add New Document.
  4. Select the Sales Order option and create a Sales Order document for any 2 or 3 items.
  5. Update Service Call and on the Related Documents tab, click the Details button again.
  6. Select any Item row in the Service Call Related Documents Details window and choose to add New Document.
  7. Select the A/R Invoice or A/R Credit Memo option and click the OK button.

Actual behavior:

Selected Item Details will not copy over into the A/R Invoice and A/R Credit Memo documents.

Cause

Functionality description

Resolution

This is the correct system behavior.

Highlighting items in the Service Call Related Documents Details window and pressing to create an invoice opens a grid on the invoice but no item is automatically added. The purpose is to avoid creating an independent invoice for an item that was already delivered for a related service call. In such cases the invoice would do an additional stock transaction.

To add the A/R Invoice and A/R Credit Memo document, open the base documents from the Related Documents tab and use the Copy To option.
