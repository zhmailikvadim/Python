Symptom

User is not able to find an export button in "Display Imported Master Data" app

Reproducing the Issue

  1. Open the "Display Imported Master Data" app in SAP Ariba Invoicing
  2. Attempt to export the displayed data.
  3. Observe that there is no export button available for exporting the data.

Cause

Resolution

Currently SAP Ariba Invoicing does not provide an export button for exporting displayed data in "Display Imported Master Data" app.

How do I create an improvement or enhancement request for SAP Ariba Invoicing?
