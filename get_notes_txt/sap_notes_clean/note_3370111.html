Symptom

This note enhances the MAPI account creation by introducing a new flag to create a draft account (data element BCA_DTE_FLG_AC_CREATE_DRAFT).

For a complete overview of the new feature Account in Draft Status in feature package 06 of CB4HANA 100, please check consulting note 3363425.

Solution

Import the relevant Support Package that is specified for your release. As a manual advance correction, you can implement the attached correction instructions using transaction SNOTE.
