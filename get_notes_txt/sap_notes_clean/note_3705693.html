Symptom

SAP Landscape Management 3.0 SP36 Patch02

Other Terms

SAP LaMa, SAP Landscape Management, Patch, LaMa Correction

Reason and Prerequisites

In SAP Landscape Management 3.0 the following fixes have been provided with Patch36 for Support Package 02:

Enterprise Edition Only

Standard and Enterprise Edition

  • FIX: Null Pointer Exception: While trying to read the field 'name' of a null object loaded from local variable 'inputRef' during a process execution.
  • FIX: NoClassDefFoundError: Failed to connect to Azure connector after LaMa SP 36 update.
  • FIX: Azure connector - Error in attaching ip to nic on same host.

 

Solution

  1. To safeguard your data against any corruptions during the implementation, export your existing configuration data. For more information, see Exporting the Configuration Data.
  2. Optional: Back up your SAP Landscape Management system.
  3. Download the software from https://launchpad.support.sap.com/#/softwarecenter -> Support Packages & Patches ->  By Alphabetical Index (A-Z) -> L -> SAP LANDSCAPE MANAGEMENT <STD/ENT> -> SAP LANDSCAPE MGT <STD/ENT> 3.0 -> COMPRISED SOFTWARE COMPONENT VERSIONS -> VCM LVM 3.0 -> latest VCMxx_x-xxxxxxxx.SCA. You may need also the latest VCM LVM CR 3.0 and latest SAP LANDSCAPE MGT <STD/ENT> 3.0 archives.
  4. Deploy the components with Software Update Manager (SUM).
  5. Restart the SAP NetWeaver AS JAVA engine.
  6. Validate the change in SAP Landscape Management, by choosing the quick link "Info" from the top menu, then check the SAP Landscape Management edition.


For more information, see:

  • SAP Landscape Management, Installation Guide on SAP Help Portal at https://help.sap.com/viewer/lama_instguide.
  • SAP Landscape Management, Application Help on SAP Help Portal at https://help.sap.com/lama.
