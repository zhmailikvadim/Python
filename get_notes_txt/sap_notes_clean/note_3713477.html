Symptom

Errors occur when attempting to download files in .TXT, .XLS, and .CSV formats. During the data download process, the file name preparation and title description functions are not executing correctly, so getting filename wrongly and data not getting properly.

Other Terms

 /VIZI/OPER, file name preparation, data download, different formats

Reason and Prerequisites

This is caused by Program error.

Please implement SAP Note#3572140 as a prerequisite note for V/4 2021 and 2022 releases.

Solution

Please implement the correction instructions. These corrections are only valid if you have applied the following Support Packs.

Release V/4 – 2021, 2022

Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object
