Symptom

Nothing happens when agency is submitting candidate.

Environment

SAP SuccessFactors Recruiting Management

Reproducing the Issue

  1. Log into Agency Portal.
  2. Click on "Add Candidate".
  3. Fill in candidate details.
  4. Submit. Nothing happens.

Cause

Engineering has identified that the issue is cause by misconfiguration on resume file size limit.

Resolution

The maximum file size settings on the server has been updated and issue is now fixed. Agencies should now be able to successfully submit candidates.
