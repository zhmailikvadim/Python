Symptom

This document shows the steps to add an extension field (KUT) in the Contracts work center view (OWL) as example.

Environment

SAP Business ByDesign

Reproducing the Issue

  1. Go to the Sales Orders work center.
  2. Go to the Contracts view.
  3. The extension field ABC is not available in the Contracts list.  (ABC represents the extension field name)

Cause

Extension field ABC is added in the contracts document but not added in the Contracts view yet. 

Resolution

Follow the steps below to add the extension field in the view:

  1. From Key User Settings option, enter the Adaption Mode.
  2. Go to Sales Orders - Contracts view.
  3. From Adaptation, enter Extension Fields list.
  4. Select the field ABC, and select 'Is Already Added' checkbox.
  5. Save and Publish the change.

The extension field ABC is then added and available in the Contracts work list.

Keywords

extension field; KUT field; work center view; OWL; contracts view.
