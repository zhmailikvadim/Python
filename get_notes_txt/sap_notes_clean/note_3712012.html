Symptom

You run the Payroll Driver (USA) (RPCALCU0) report for an employee who has multiple Work Tax Authorities (IT0208); after per-authority amounts are rounded to cents, the rounding remainder (0.01) is added to the first found tax authority, causing a wagetype /7XX to be 0.01 instead of 0.00.

Example

  • IT0208 (Work Tax Authorities and split):
    • KY09 — 40% (Tax Level C -> KY5I)
    • KY0H — 40% (Tax Level C -> KY0H)
    • KY20 — 20% (Tax Level C -> KY0A)
  • Wage type 1002 (gross wages sum for period): 4,240.96

Calculation steps

  1. Prorate by work-tax percentages:

    • KY5I: 4,240.96 * 40% = 1,696.384
    • KY0H: 4,240.96 * 40% = 1,696.384
    • KY0A: 4,240.96 * 20% = 848.192

  2. Round each amount to two decimals (cents):

    • KY5I → 1,696.38 (down, lost 0.004)
    • KY0H → 1,696.38 (down, lost 0.004)
    • KY0A → 848.19 (down, lost 0.002)

  3. Sum of rounded amounts:

    • 1,696.38 + 1,696.38 + 848.19 = 4,240.95

  4. Rounding difference vs original gross:

    • Original total = 4,240.96
    • Sum rounded = 4,240.95
    • Difference = 0.01

System behavior that produces the observed problem:

  • In order to add the 0.01 value back, the system sums it to the first Tax Authority for that tax level, which is KY0A. Then, its pay-period-wages in TXRT changes from 848.19 to 848.20. The BSI build routines use TXRT, so BSI receives:
    • KY0A PW = 848.20
    • KY20 PW = 848.19

Observed results:

  • /451 KY0A = 0.00
  • /751 KY0A = 0.01 (unexpected small amount due to the 0.01 added to KY0A).

Other Terms

RPCALCU0; Tax earnings; /7XX; IT0208.

Reason and Prerequisites

Reason

The system does not preserve the expected per-authority taxable totals when prorating produces fractional-cent amounts and the rounding-correction routine unconditionally adds the cent difference to the first tax authority found for the tax level, causing an unexpected 0.01 on that authority.

Prerequisites

The following Support Package level is the minimum prerequisite:

S4HCM 101 HR SP 06

When you install an SAP Note, some prerequisites might be required. The prerequisites are SAP Notes that have already been released and that change the same context of the current one. Solution

From now on, the system adds the remainder cent to the tax authority with the largest amount in the same tax level.
