  • Report RSDU_TABLE_CONSISTENCY reported "Table Placement expected but not maintained"

Environment

SAP HANA, platform edition

Reproducing the Issue

Execute the DTP

Cause

TABLE_PLACEMENT is not maintained.

Resolution

Refer to SAP Notes for setting up the TABLE_PLACEMENT:

SAP Note 1908075 - SAP BW on SAP HANA: Table Placement and Landscape Redistribution

SAP Note 1900822   Table Placement in BW/4HANA, S/4HANA, SAP BW Powered by SAP HANA or SAP Business Suite Powered by SAP HANA

After table_placement is in place, rerun the report RSDU_TABLE_CONSISTENCY and use the repair option to fix the inconsistency error. (KBA 2569097 - How to use report RSDU_TABLE_CONSISTENCY [VIDEO].)
