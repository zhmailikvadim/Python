Symptom

While adding an additional measure and opening the Measure node in the Rows or Column section of the navigation panel a giving below error message:

500 Internal Server Error

Adapter error in C_TBL_HIERARCHICAL_CELL "CV_HR_COL_NAME" of view "FPM_BICS_GRID_RUIBB.V_NAVIGATION": Property "hierachicalState" of adapter has an invalid value. This error can occur if you, for example, are using incorrect constants or no constants at all.

Environment

  • SAP NetWeaver Application Server for SAP S/4HANA
  • SAP S/4HANA Cloud Public Edition 2208
  • SAP Integrated Business Planning for Supply Chain 2208 
  • SAP Web Dynpro FloorPlanManager

Cause

Programming error.

Resolution

Apply SAP Note 3208293 - Dump when adding additional measures
