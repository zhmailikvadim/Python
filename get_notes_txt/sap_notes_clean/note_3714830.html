Symptom

  • The SC_003 constraint in Advanced Workflow limits the amount of operations to be executed in a script, with the default value being 1.000.000 (one million). This constraint may be increased via a SAP Support Case. 
  •  Error message: Script <script_name> reached the maximum number of operations allowed. (#SC_003) 

Environment

  • SAP Sales Performance Management
  • Advanced Workflow

Reproducing the Issue

Execute a script with more than a million operations.

Cause

This constraint is implemented because a large number of operations can cause performance issues in the servers.

Resolution

  1. Firstly consider if there are any optimizations to be done in the script to prevent breaching a million operations. Refer to the following article for more about best practices and optimization:

    Help Portal: Coding Standards & Best Practices | SAP Help Portal 

  2. If there are no optimizations to be done, raise a SAP Support Case to request an increase in the number of operations. The following information needs to be contained in the case's description:
    1. Script Name
    2. Error Message extracted from System Logs, Script Execution Logs or Debug Tool
    3. A description of the script's general purpose
    4. Optimizations attempted
    5. Step-by-step operations on how to reproduce the error
