Symptom

A transport is assigned to a Cloud ALM Feature, but the transport attribute CALM_CDM_FEATURE_ID is not set in the transport.

Environment

  • SAP Cloud ALM 1.0
  • SAP ABAP systems

Reproducing the Issue

  1. Create a Transport request with transaction code SE01 or SE10;
  2. Assign the transport to a Cloud ALM Feature;
  3. Verify if the transport attribute CALM_CDM_FEATURE_ID is set in the transport under 'Properties' tab.

Resolution

This attribute will only be assigned to the transport when it is created directly from the Feature. 
It will not be set for locally created transports that are later assigned to a Feature.
This is the standard behavior.
