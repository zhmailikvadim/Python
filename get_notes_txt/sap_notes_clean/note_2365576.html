Symptom

A SAP System Upgrade on a SAP system with DB2 for Linux, UNIX, and Windows breaks in phase PREP_INSTALL/RUN_RSPTBFIL_SPACE. Report RSPTBFIL (running in batch mode) stops with a DBIF_REPO_SQL_ERROR short dump. The shortdump refers to an SQL error "-55555".  The database error text (from the SAP DBSL library) is like

"DbSlPrepareDB6: Unable to find a reusable entry in stmt_cache (out of handles)"

Other Terms

SQL -55555 statement handle

Reason and Prerequisites

An SQL prepared statement in class CL_DD_DB_VIEW_DB6 is not closed correctly. Statement handles stay alive. The error can appear in other places as well.

Solution

Install SPs mentioned below or apply attached correction instruction.
