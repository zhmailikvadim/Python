Symptom

Error:  “ HTTP operation failed invoking https://xxx-xxx.xxxxxxxxx.xxxx/xxxxxxx with statusCode: 403" Occurs When Sending a Message from Network to an External System.

Cause


The HTTP 403 Forbidden error occurs when the external system server understands the request but refuses to authorize it. 

  • The user or service account lacks sufficient permissions to perform the requested operation.
  • Required roles or authorizations are missing in the target system, preventing access.
  • The request is attempting to access a restricted resource or endpoint.

Resolution

  • Verify User/Service Account Permissions.
  • Ensure the user or technical account has the necessary roles and permissions to access the resource or endpoint.
  • Assign any missing authorizations in the target system to enable access.
