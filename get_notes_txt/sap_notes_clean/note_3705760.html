Symptom

  • The Agent Desktop does not display all contacts associated with an incoming phone number.
  • Only 10 contacts are shown in the Search Result List, although the phone number is linked to more (e.g., 37 contacts).
  • No visible pagination or "Show more" button is provided to access additional results.

Environment

  • SAP Service Cloud Version 2 1.0
  • SAP Sales Cloud Version 2 1.0 

Reproducing the Issue

  1. Log into the tenant with any user.
  2. Open the Agent Desktop with CTI integration.
  3. Receive an incoming call from a phone number that is linked to multiple contacts (e.g., 37).
  4. Observe the Search Result List that appears in the Agent Desktop.
  5. Only 10 contacts are initially shown.

Cause

  • The system is working as designed.
  • To optimize performance, the Agent Desktop initially loads a limited number of results (default: 10). 

Resolution

  • The search result list loads 10 contacts by default to improve load times and performance.
  • Additional contacts will be dynamically loaded as the user scrolls down.
  • This is an infinite scrolling mechanism rather than pagination or a manual "Show more" option.
  • Additional Information:
    • No configuration is currently available to change the default number of contacts shown.
    • For enhancement requests or changes to this behavior, please submit feedback via the Customer Influence Portal.
    • For all roadmap features, please visit: SAP Roadmap Explorer.
      • Please note that any forward-looking roadmap item is subject to change and timelines are tentative in nature.
      • When it's available, the information will be shared via the What's New documentation with links.
