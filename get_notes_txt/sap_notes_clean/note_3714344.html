Symptom

  • After the upgrade from SAP S/4HANA 2021 to 2023, the option “Show Usage in Roles” in FLPCM_CUST is taking significantly longer to load compared to earlier.
  • Security team is experiencing delays when viewing roles.

Environment

SAP Fiori Launchpad

Cause

  • The issue is caused by the method /UI2/IF_FLP_CONT_MGR~RESOLVE_COMPOSITE_ROLE, specifically the code related to the function `/ui2/cl_pfcg_utils=>get_roles_of_composite_role`. This can be confirmed from ST12 trace.
  • The code responsible for the delay was removed by applying SAP Note 3642185 - Composite roles with derived roles are missing in design time.

Resolution

  1. Apply SAP Note 3642185 - Composite roles with derived roles are missing in design time.
  2. Verify the resolution by testing the "Show Usage in Roles" option after applying the SAP Note.
    It will depend on the number of roles the app is in.
