Symptom

You want to update the commons-fileupload library to the latest possible version, compatible with java8, which is commons-fileupload 1.6.0.

Other Terms

IDM, Identity Management, commons-fileupload

Reason and Prerequisites

You are using SAP Identity Management 8.0 SP08.

Solution

Apply the attached patch provided in the Support Packages & Patches section of this SAP Note.

Make sure all components are updated to the corresponding SP level before applying the patch.
