Symptom

SPDD transport request number is not updated in file <DIR_TRANS>/bin/umodauto.lst.

Environment

SUM

Reproducing the Issue

Perform SPDD adjustment at ACT_UPG phase.

Cause

For upgrades with target release S4HANA 2021 or higher, release the tasks and transport after the conversion is complete. 

Resolution

During next SUM run for following systems, either manually add SPDD transport request number into umodauto.lst before upgrade or input SPDD TR number during next SUM run.
