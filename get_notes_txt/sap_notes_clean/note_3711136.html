Symptom

You are using ABAP Data Integration with SAP Business Data Cloud, SAP SLT, SAP Datasphere or SAP Data Intelligence. You created a CDS view on top of a table with replacement object. And the table is partitioned. This combination delivers duplicate and incomplete data during initial load.

Other Terms

S/4HANA, ABAP Data Integration, SAP Business Data Cloud, APRS, CDC, Delta Reconstruction, DHADM, Access Plan Calculation, transfer job, observer job, ACP Daemon

Reason and Prerequisites

The ABAP-based source system is an S/4HANA 2025 FPS00-SP01 system.

The ABAP-based source system is an S/4HANA 2023 FPS00-SP05 system.

The ABAP-based source system is an S/4HANA 2022 FPS00-SP07 system.

The ABAP-based source system is an S/4HANA 2021 FPS00-SP09 system.

Solution

Implement the corrections in the ABAP-based source system.
