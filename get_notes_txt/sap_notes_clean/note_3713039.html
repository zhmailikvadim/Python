Symptom

The Responsible Service Organization, also known as the Service Organization Unit, cannot be specified for a service order using an API and is also not passed back from any API.

Other Terms

API_SERVICEORDER, ServiceOrderRequest_In, ServiceOrderBulkRequest_In, ServiceOrderConfirmation_Out, ServiceOrderConfirmationBulk_Out, ServiceOrderNotification_Out, I_ServiceOrderTP

Reason and Prerequisites

The field was not part of any communication structures. The field is only supported, if the Enterprise Organization Management has not been enabled.

Solution

The field ResponsibleServiceOrganization was added to the communication structures for the OData, SOAP, and PaaS APIs for the service order.

To get this feature, please upgrade to FSP2.
