Symptom

In transaction IW32 in the Fiori context, choosing Create Follow-On Order for an operation opens a Web Dynpro application that does not offer Service copy and therefore cannot copy a billable order. For billable maintenance orders, the navigation should open SAP GUI transaction IW31 instead of the Web Dynpro Create Maintenance Order application.

Other Terms

IW32, CL_LSAPI_MANAGER, sap-ui-tech-hint

Reason and Prerequisites

The navigation does not pass the GUI hint for billable orders; the correction adds parameter sap-ui-tech-hint=GUI when the order is billable.

Solution

Implement the attached corrections.
