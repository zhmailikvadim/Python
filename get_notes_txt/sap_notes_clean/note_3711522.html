Symptom

You use Transaction Manager of Treasury and Risk management. You try to create a foreign exchange transaction with activity 'Order' from FTR_CREATE or TX01. You find the activity is automatically switched to 'Contract'.

Other Terms

CL_FTR_GUI_ENTRY, CHECK_ENTRY, XORDER, IF_TTM_UI_CONTROLLER~MV_CREATE_AS, IF_TTM_UI_CONTROLLER=>GC_CREATE_AS-ORDER

Reason and Prerequisites

Program Error

Solution

Preliminary correction via Correction Instruction or implementation of the corresponding Support Package.
