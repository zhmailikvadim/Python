Symptom

SAP Note 2651735 - HRNO - SPK - July 2018 contains an UDO report NOTE_2651735. Within this report, SAP-BASIS is incorrectly evaluated which prevents the execution in some versions of the system.

Other Terms

HRNO, NOTE_2651735

Reason and Prerequisites

•This SAP Note is relevant only for Norway (HRNO)

•HRSP SAPK-60831INSAPHR, SAPK-604A3INSAPHR, SAPK-600D6INSAPHR (or newer)

Solution

Source code was maintained to solve the problem, which is described in symptom.

Implementation

  • Generally, SAP recommends you to install a solution by applying a Support Package. However, if you need to install a solution earlier, use the Note Assistant to implement the correction instruction. For more information about the Note Assistant, see SAP Service Marketplace, under service.sap.com/note-assistant.
  • Note that documentation, transactions, and Customizing activities are only delivered via Support Package.
