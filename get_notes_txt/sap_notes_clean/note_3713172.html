Symptom

  • Period posting control in question is open for all accounts (type = "+") 
  • However period posting control is closed for sub account types 'K' or 'D' (suppliers/customers)
  • 'Manage posting periods' in does not restrict postings for account type 'K' or 'D' for the closed period

Environment

  • SAP S/4HANA Cloud Public Edition
 

Reproducing the Issue

  1. Run the 'Manage Posting Periods' app and check the period control settings for account type 'K'. You see that it is closed for period 012.2025 (for example)
  2. At the same time, the posting control for all accounts (+) is open for period 012.2025
  3. Post a vendor invoice using example app 'create incoming invoices' (FB60)
  4. Observe that posting is successful in period 012.2025 for this supplier posting, despite the period being closed for suppliers 'K'. A posting error is expected. 

Cause

The likely cause of this scenario is that the posting control has not been properly maintained for the adjustment periods in app 'Mange posting periods'. 

Example: 

  • In this case the adjustment periods for account type + is correctly set from 013.2025 to 016.2025
  • However for the customers and suppliers sub account types, the adjustment periods are open from 2020 until 2030!
  • This will result in the ability of the system to post customer and supplier postings in period 012.2025 for example even though this is not allowed as per the 'Normal' posting control. 

Resolution

The solution to this problem is to properly maintain your adjustment periods control for all account types. Typically, the adjustment periods should only be open for a single fiscal year eg. period 013 to 016 2025.
