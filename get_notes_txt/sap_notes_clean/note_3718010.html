Symptom

On Overview page, 'Hours Logged Today' KPI header shows time for all users instead of showing time just for the current user.

Reason and Prerequisites

Reason: 

Missing metadata logic to show only the current user time

Prerequisites:

  •  BAS or Visual Studio Code with SAM MDK project
  • `patch` command (available on BAS and macOS by default) via terminal or other patching utility
  • Read the guidelines on how to apply the patch and read the appendix section if you receive any error
  • Review the SAP MDK support Matrix SAP Note 2933065  for your version of SSAM to determine the correct SAP Notes to apply before this patch

Solution

To apply the patch, download the attached 3718010-<version>.patch file and save it in a known location, preferably adjacent to your SAM.mdkproject.

If you are on Windows, see SAP Note 3533879 for how to install a .patch file in Windows. If you are on a Mac or on BAS, follow the steps below:

  1. Open a terminal in the root of your SAM metadata (i.e. SAM.mdkproject/metadata or SAM.mdkproject/metadata/base.definitions)
  2. Run the terminal command `patch --no-backup-if-mismatch -p4 < path/to/3718010-<version>.patch`.
  3. Verify in the terminal window that the following files have been modified and listed as patched, as indicated by the console output:
