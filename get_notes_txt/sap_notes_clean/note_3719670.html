Symptom

When attempting to submit a requisition, there is a validation error on the line item SAPPlant must be set.

Cause

This error is triggered when the line item does not have an SAP Plant value. This value is set from the Requester's user profile. If the Requester does not have an SAP Plant assigned to them, all line items created by them will have a null value for SAP Plant.

Resolution

Modify the requester's user profile to include a value for SAP Plant. This can be done via CSV import file or via the UI under Core Administration > User Manager.

Keywords

SAP Plant, SAPPlant
