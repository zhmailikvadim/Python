Symptom

After the release of SAP Note 3695676, there are some questions regarding the changes.

  • 3695676 - SAP Datasphere Outbound & Inbound IP Addresses Changes for Multiple Availability Zone Enablement

Environment

SAP Datasphere

Resolution

  1. Will the Datasphere Tenant URLs be affected by this?
    No, the SAP Datasphere tenant URLs remain unchanged.
  2. Will the IP Address changes have any implications on OpenSQL Access or DPAgent communication?
    No. There is no impact on Open SQL access or DP Agent communication.
  3. Will external partners using Datasphere APIs via Oauth Clients?
    No. External partners consuming SAP Datasphere APIs via OAuth clients are not affected.

Keywords

Datasphere, IP, Allowlist, Data Flow, Replication Flow
