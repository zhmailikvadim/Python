Symptom

When running the IP document migration, the system is expected to include accruals for all migrated items. However, during accrual migration, the system incorrectly excludes credit lines. As a result, these lines are not present in the calculation runs that are created.

This note corrects the behavior so that accrual migration considers all billing document types, ensuring that credit lines are also included.

Other Terms

/IRM/IPMGM, IP, Migration, Calculation run, Items, Accruals, Credit, Threshold

Reason and Prerequisites

This is caused by program error.

Note : SAP Note#3729226 is available for ECC release.

Solution

Please implement the correction instructions. These corrections are only valid if you have applied the following Support Packs.

Release V/4 – 2023 V2, 2023 V3 and Support Pack 1

Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object
