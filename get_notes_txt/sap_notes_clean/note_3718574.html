Symptom

After adding a document in the "attachments" tab, the source of this file is marked as "Generic Object Service".

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Access the Manage Sales Orders Version 2 app.
  2. Click the button "Edit" of the Sales Orders or click in the button "attachments" before saving the newly created Sales Order.
  3. Upload any file.
  4. See that the source of the file is marked as GOS (Generic Object Service).

Cause

It is possible to upload the types of files:

  • JPG
  • PNG
  • AVIF
  • JPEG
  • PDF
  • MP4
  • PPTS
  • CSV
  • XLMS
  • TXT
  • XML
  • ZIP
  • And others

Resolution

At the moment, GOS is the only document source accepted to be uploaded into Sales Documents attached files.

It is not possible to activate other sources or to change the original source to a new configuration.

In case a new source is required, it is necessary to create a feature request via the SAP Customer Influence portal.
