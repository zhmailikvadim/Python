Symptom

When performing Technical Completion (TECO) on a billable maintenance order in IW42, the linked service order item does not receive the expected Completed status.
As a result, the service order item remains Released even though the maintenance order is technically completed.

Other Terms

IW42, TECO, CO_ZV_ORDER_POST, Service Order Item, Status Update

Reason and Prerequisites

The reason is a program error.
A billable order processing block in CO_ZV_ORDER_POST was executed only when a specific activity flag was initial.
In IW42, this flag is set, which caused the update logic for billable service orders to be skipped.

Solution

Apply attached Correction Instruction.
