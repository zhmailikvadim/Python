Symptom

CALL LIBRARY/PROGRAM results in SQL5016 - Qualified object name &1 not valid.

Reason and Prerequisites

The DBSL parser gets "confused" by CALL LIBRARY/PROGRAM without trailing parenthesis.

Solution

Please apply the LIB_DBSL patch listed in the Support Packages & Patches section of this note.

The work-around is to add parenthesis:

EXECSQL.

  CALL LIBRARY/PROGRAM ()

ENDEXEC.
