Symptom

The SAP Note 3727875 (Tax Reform: IPI Tax Rate Updates for NF-e Documents) requires this SAP Note to be implemented as a prerequisite.

Other Terms

Brazil, Outbound, Nota Fiscal Eletronica, Electronic Nota Fiscal, Brazil eInvoice, Cloud, NF-e, LC 224/2025, SAP Document and Reporting Compliance, cloud edition, SAP Document and Reporting Compliance, outbound invoicing option for Brazil to submit Electronic Invoice (NF-e) outbound documents

Reason and Prerequisites

This SAP Note is relevant only for Brazil.

This SAP Note contains a report and automatic correction instructions to create/modify objects that are used in the solution delivered via SAP Note 3727875 (Tax Reform: IPI Tax Rate Updates for Outbound Documents).

Solution

Use transaction SNOTE to implement the attached correction instructions.

Version Date Reason
1 March 18, 2025 Released for Customer

As a general rule SAP recommends that you implement the Support Package specified for your release. However, if you need to install a solution earlier, use the SAP Note Assistant to implement the correction instruction. You can find more information about the SAP Note Assistant in SAP Support Portal, under SAP Note Assistant.
