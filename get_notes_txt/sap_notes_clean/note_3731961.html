Symptom

A guide on how to configure the dashboards to show data for a specific period and/or time range. 

Environment

  • SAP Sales Performance Management
  • Incentive Management
  • Incentive Management on Hyperscalers
  • Incentive Management on Legacy Oracle
  • Incentive Management on Legacy HANA 

Reproducing the Issue

Cause

Dashboards can be configured to show only released periods, which would allow the end users to only view data for specifically released periods. 

Resolution

To enable Released Periods, please follow the steps below: 

  1. Access the Global Settings in the Incentive Management Portal;
  2. Go to the tab "Dashboard Personalization Settings"
  3. Enable the “Released periods” option so end users can view dashboards for released periods only.
  4. Enable the "Show only released periods matching by positions date range"

When the Enable released periods setting is enabled, and the Show only released periods matching by positions date range option is enabled, all manual periods that have a release date within the date range for the selected position are displayed.

How to Release a Period for viewing in dashboards:

  1. Navigate to Dashboards > Released Periods.
  2. Choose a Processing Unit from the dropdown list.
  3. Choose a Calendar from the dropdown list.
  4. Choose a Period from the dropdown list.
  5. Click the Release Period for Dashboards button.
