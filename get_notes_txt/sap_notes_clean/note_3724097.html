Symptom

There is a requirement on how to identify if a statement is supported in current environment and how to get its usage method.

Environment

ABAP system

Resolution

Search the statement in transaction 'ABAPHELP', and then search the statement in the pop up. If there is result for the searching statement and it is not marked as obsolete, then it is support in current environment. The usage of the searching statement will show in the search result as well.

Keywords

Syntax; support; statement; abaphelp
