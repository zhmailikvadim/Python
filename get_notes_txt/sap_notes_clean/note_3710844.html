Symptom

CDOTE dump or error 'Wrong Document Quantities from Upstream System' with CDOTE Cancellation.

Replication steps:

1. Create a trading contract from WB21.

2. Create multiple price fixations with CDOTE price orders from WB22.

3. Receive the fill for all CDOTE price orders.

4. Go to CMM_MP_MASS_PRICING. Perform the cancel with CDOTE using multiple price lots.

5. Go to CMM_OF_PROCESS. Enter the CDOTE order and execute.

6. The system shows either a dump or the error 'Wrong Document Quantities from Upstream System' on the fill process.

Other Terms

Agricultural Contract Management; ACM; CDOTE; CMM_MP_MASS_PRICING; CDOTE Cancellation; CMMFDOF_ORDER_FILL 257

Reason and Prerequisites

 Program error

Solution

Import the relevant support pack or apply the correction instruction(s).
