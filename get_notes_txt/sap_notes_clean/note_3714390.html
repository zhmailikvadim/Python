Symptom

 

When receiving items managed by serial numbers set to On Release Only and with the Automatic Serial Number Creation on Receipt option enabled, the Bin Location Allocation field becomes non-editable. As a result, users cannot specify the bin location where the items will be stored.

Environment

SAP Business One

Reproducing the Issue

Scenario:

  1. Go to Inventory → Item Master Data to create a new item, set Manage Item by as 'Serial Numbers', Management Method as 'On Release Only' and tick the checkbox Automatic Serial Number Creation on Receipt.
  2. Go to Inventory → Inventory Transactions → Goods Receipt to create a goods receipt for the item created in Step 1, select a bin location warehouse.

Actual behavior:

The Bin Location Allocation field is greyed out and not editable.

Resolution

  • This is the correct system behavior.
  • Add the receiving document to enable the system to automatically create the serial numbers and store the items in the warehouse's default bin location. Afterwards, create an Inventory Transfer to move the items to the desired bin locations.

Keywords
