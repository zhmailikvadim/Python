Symptom

  • The New Delegation Request Card on the Home Page is not functioning as expected following the 2H2025 release.
  • The card appears unchanged and does not reflect the updated functionality.

Environment

SAP SuccessFactors HCM Suite

Reproducing the Issue

  1. Navigate to the Home Page in the system.
  2. Attempt to access the New Delegation Request Card feature.
  3. Observe that the functionality remains the same as the previous version and does not reflect the updates from the 2H2025 release.

Cause

Expected behaviour from the system.

Resolution

New Delegation Request Card is only available for:

  • Platform Delegation: Users, typically managers, can delegate their performance review tasks to others for a specified time period. Access the Platform Delegation tool under Settings/  Delegation

  • If Delegation tool is intended to be used otherwise than the above scenarios, then it is expected behaviour that it will not work.
