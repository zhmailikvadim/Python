Symptom

The Flow type field (BSEG-VBEWA) is not changeable via Function module FI_DOCUMENT_CHANGE. Despite being set as changeable in OB32.

The error message F5A194 appears when executing FI_DOCUMENT_CHANGE via SE37.

Environment

  • Financial Accounting (FI)
  • SAP R/3
  • SAP R/3 Enterprise
  • SAP ERP Central Component
  • SAP ERP SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA Finance
  • SAP S/4HANA

Reproducing the Issue

  1. Execute the function module FI_DOCUMENT_CHANGE via SE37.
  2. Attempt to change the Flow type field.
  3. Observe error message F5A194.

Cause

The field BSEG-VBEWA (Flow type) cannot be changed via FI_DOCUMENT_CHANGE.

Resolution

This is the standard system behavior.

Keywords

flow type, bseg, fi_document_change, ob32, error f5a194, se37, field modification, accounting interface
