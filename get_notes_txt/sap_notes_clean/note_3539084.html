Symptom

You execute the consolidation of investment(C/I) task. The error message no. G06020 is shown in the log telling that investment or equity data was collected for a consolidation unit after the period of its first consolidation.

( This note also applies to other issues caused by the situation where you upload and reverse the same C/I reported data within the same period. )

Reason and Prerequisites

  1. You have set the master data for first consolidation of a consolidation unit A to a certain period and year.
  2. During the data preparation for the C/I task, you have maintained investment or equity reported data for the unit A with activity 01 (first consolidation) in a period X later than  the time mentioned in the point 1.
  3. Afterwards, you have reversed the reported data mentioned in point 2 via manual posting so that in total no investment or equity data of unit A is available in the period X for the first consolidation.

Solution

 Import the support package or implement the advance correction using the correction instructions.
