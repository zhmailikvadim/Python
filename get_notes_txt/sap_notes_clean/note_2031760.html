Symptom

When you start the Generic Object Services (GOS), the system does not display the usual list of services in the toolbar or the menu. Instead of this, only one single (inactive) service is visible. This error occurs sporadically; sometimes, all services exist and other times, only the one service exists.

An additional service was entered in the table SGOSATTR but this service was not integrated correctly into the existing structure. This may be the service 'J3RF_IMCD', which arrived in the system after EhP7 Upgrade or SP update in EhP7.

Other Terms

SGOSATTR, J3RF_IMCD, SGOSM

Reason and Prerequisites

This problem is caused by an error in the Customizing table SGOSATTR.

Solution

Perform the following manual steps.

Call Transaction SM30 to maintain the table SGOSATTR (or use transaction SGOSM). Remove the incorrect service 'J3RF_IMCD' and save changes.
