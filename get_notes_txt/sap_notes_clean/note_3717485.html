Symptom

If you insert a moved Solution Documentation element at a marked as deleted elements, then a dump happens.

Other Terms

CL_SMUDK_TOOL_CONTRACT, CL_SMUDK_BINST_COMMANDS=>IF_SMUDK_INSTANCE_COMMANDS~MOVE_NODES, marked as deleted

Reason and Prerequisites

To do a insert of a moved element at a marked as deleted silbing element, then a dump happens. This is a program error.

Solution

Please implement the current correction instruction of the SAP Note with transaction SNOTE. After that, the error should be solved and a dump should not happen again.
