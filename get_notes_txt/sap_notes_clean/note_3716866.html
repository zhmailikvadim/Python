Symptom

The error message "No items are valid on dd.mm.yyyy" is displayed when attempting to view a assembly using the CS73 transaction.

Environment

  • SAP ERP Logistics (LO)
  • SAP R/3
  • SAP R/3 Enterprise
  • SAP ERP Central Component
  • SAP ERP
  • SAP Enhancement package for SAP ERP
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Open transaction CS73.
  2. Enter WBS Element , Material , Plant, and BOM Usage 
  3. Open part tick a component .
  4. Double-click assemble.
  5. Select BOM Usage and continue.
  6. Error message "No items are valid on dd.mm.yyyy" (Message no. 29064)

Cause

The issue occurs because there is no BOM item maintained for the WBS BOM.

Resolution

Using  transaction CS71 or CS72 to create or change the WBS BOM for the WBS element. Ensure that at least one BOM item is maintained for the WBS BOM
