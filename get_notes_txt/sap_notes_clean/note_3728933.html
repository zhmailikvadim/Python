Symptom

You are trying to use the features "View design-time data" and "Validate" after the 2603 Kyma release. After clicking the button, the popup displays the following messages:

  • debug/designtimedata
  • validate/validate

Other Terms

SAP Cloud Integrations for data services - Server 2603 Kyma

Reason and Prerequisites

A regression code issue affected the loading of this functionality.

Solution

The correction for this server issue is delivered in SAP Cloud Integration for data services version 2603 (1.1.2.1).
