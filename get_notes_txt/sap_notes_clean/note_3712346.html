Symptom

You are using Employee Central Payroll with PTP employee replication.

One employee has a Higher Duty assignment where the Allowance Computation Basis is set to "Not Applicable".

The replication fails with the error message "Currency of Higher duty &1 and of Substantive &2 must be the same."

Other Terms

Higher Duty, IT0509, HRSFEC_HDREP 009

Reason and Prerequisites

Program error.

Solution

Please implement the correction instruction via SNOTE.
