Symptom

Questionnaires are assigned to a bucket in the tab 'Field Service Configuration' tab under the Miscellaneous tab. Here, by default application shows only 3 rows. If there are more entries, then the scroll bar needs to be used which is not very user friendly.

Other Terms

Questionnaires assignment in bucket, number of rows displayed in Field Service Configuration tab, /RPM/BUCKET_DETAILS, VI_FIELD_SER_CONFIG, Visible_Row_Count

Reason and Prerequisites

Usability enhancement

Solution

 After the implementation, users will be able to see 10 rows in the 'Field Service Configuration' tab.

Implement the corrections assigned to the SAP Note.
