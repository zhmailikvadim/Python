Symptom

  • Upload failed with the error message: 'xxxx is required'
  • xxxx is a Custom Field, so it can be different for each buyer

Cause

The custom fields that appear on the upload error are configured as mandatory and were missing in the upload

Resolution

  1. For each affected detail in the upload, provide valid values for all mandatory custom fields.
  2. Resubmit the upload and confirm it processes without errors.
