Symptom

When a plausible meter reading is not available in the Meter Reading Value Complaint process, a runtime error "RAISE_SHORTDUMP" occurs after maintaining the plausible meter reading. This error is caused by a duplicate key in the check result.

Other Terms

US4G, MET, Meter reading value complaint, DE_B_MET_MRV_COMPLAINT_MALO, runtime error, RAISE_SHORTDUMP, /US4G/CL_IM_BADI_DATEXCONN002, CHK_ACTIVE_COMPLANIT_PROC, duplicate key, check result

Reason and Prerequisites

Program error.

Solution

Please implement the correction instructions mentioned in this SAP Note or import higher support packages.
