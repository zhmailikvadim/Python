Symptom

In Slovenia, some legal reports have been redesigned by authorities in the last few years. SAP developed and delivered new programs for such redesigned reports. The old programs became unusable as they do not provide correct output. SAP doesn't provide support for these programs anymore and this fact must clearly be presented to customers.

Other Terms

obsolete legal reports, REK-1, REK-2, REK-1A, BOL0, BOL1, DIP, M-4, ZAP-M

Reason and Prerequisites

  •  Some legal reports have been redesigned by authorities causing obsolescence of many SAP reports.

Solution

In below listed ABAP programs, following action is added:

  1. When user runs the program, popup window is displayed with the following text (or corresponding translation):
    "This program is obsolete and not supported by SAP anymore. Use <program-name>, transaction <transaction-name> instead."
  2. Program then runs as is, still giving to user an option to get data.

Caution: These reports most likely display many incorrect data, some of them might even end-up with error. SAP gives no support for problems in these reports, user cases will be rejected. SAP also doesn't take any responsibility for potential damage or incorrect decisions based on data displayed with these reports. After some time (years), running these reports will be prevented or the code from these programs (all or selected ones) might even be removed. In the country menu (transaction PC62), these reports were moved to branches named "Obsolete" ("Zastarelo") in the tree structure.
