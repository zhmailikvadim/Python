Symptom

  • When printing a warehouse order from transaction /SCWM/MON, all pages are included in the spool order. 
  • When replicating the process in /SCWM/RFUI, only the first page is considered.

Environment

  • SAP Extended Warehouse Management
  • SAP S/4HANA

Reproducing the Issue

  1. Enter Tcode: /N/SCWM/RFUI
  2. Initiate Warehouse Order Printing

Cause

Incorrect configuration 

Resolution

  1. Navigate to Tcode: SPPFCADM. 
  2. Select the application (/SCWM/WME).
  3. Open the action profile (/SCWM/WO).
  4. Choose the standard action definition (WO_SINGLE).
  5. Update the processing time for the standard PPF action to 'Processing when saving document'.
