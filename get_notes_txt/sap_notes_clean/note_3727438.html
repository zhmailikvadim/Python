Symptom

While navigating to Price component section after clicking on Edit for Single Price lot from Process ACM Price lots Fiori Application, a memory allocation dump occurs.

Replication steps:
1. Open Process ACM price Lots FIORI application.
2. Filter with contract number.
3. Select a price lot and click Edit button.
4. Navigate to Price component tab.
5. The system shows either a dump or the error due to memory allocation issue.

Other Terms

Agricultural Contract Management; ACM; Advanced CPE; Process ACM Price lots Fiori Application

Reason and Prerequisites

Performance

Solution

Import the relevant support pack or apply the correction instruction(s).

This SAP Note delivers changes to use Contract number and item along with Source lot id to get the price lot details.
