Symptom

How do I generated a report that exports user and role data? 

Environment

SAP Business Network

Cause

User and role data exports must be generated directly by the customer from the UI using the available reporting functionality.

Resolution

User and role data can be exported directly from the SAP Ariba account using the standard UI functionality.

  1. Log in to your buyer or supplier account.

  2. Click on your initials in the top-right corner

  3. Navigate to Settings > Manage Users

  4. Choose the appropriate tab depending on the data required:

    Manage Users > to export user details (First Name, Last Name, User Name, Email ID, Corporate User Name, etc.)

    Manage Roles > to export role assignments

1. Click the table options menu (gear icon) 

2. Select the required columns

3. Click Export all rows

Note: The export must be performed separately for Production and Test environments.
