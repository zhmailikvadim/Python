Given name match standards in English domains (EN_AU, EN_GB, EN_US) are incomplete or blank when compared to the previous release.

Other Terms

Keywords

DQ DS CPB DC IS

Environment:  

SAP Data Services 4.2 SP6 or later
SAP Information Steward 4.2 SP6 or later

Reproducing the Issue:  

Output match standards for Linda, Eve, Clara, Steve, Alex, Sandy, Sam, Denny, Larry, and Jax using Data Services 4.2 SP6 and later.  Compare these to what was output in Data Services 4.2 SP5 or earlier.

Solution

Workaround: Manually add the missing match standards via the Cleansing Package Builder.

This is a known issue and devleopment is planning to fix this in a future release.
