Symptom

You are planning to use OR using SAP ECS Managed SAPRouter. 

You would like to know the supported use cases and limitations of SAP ECS Managed SAPRouter.

Resolution

Below are the use cases supported by ECS Managed SAPRouter connectivity, along with its limitations for SAP SuccessFactors Employee Central Payroll systems.

Use CaseOffered in ECSRemarks
SAPGUI ConnectivityYesCustomer on-prem network to EF ECP system, SAPGUI connectivity over SNC tunnel.
RFC Connectivity (ECP <--> ECS Backend System)YesRFC to/from ECS-S4 to ECP systems for payroll posting, cost centre replication, etc. 
Single Cheque Printing (Customer on-premise print server port = 515)NoDue to technical limitations, the ECS Managed SAPRouter hosted in the ECS network cannot communicate with the customers' on-prem print server over port 515.
