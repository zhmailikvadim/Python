Symptom

  • The submit button remains disabled when attempting to complete a custom task with an attachment in the Onboarding module.

Environment

    SAP SuccessFactors Onboarding

    Reproducing the Issue

    1. Log in as the user.
    2. Navigate to "Your Onboarding Checklist".
    3. Select a custom task.
    4. Attach a file to the task.
    5. Observe that the submit button remains disabled and cannot be clicked.

    Cause

    • The visibility of external code in the tasks object is set as 'not visible' or 'read only'.
    • Required property should be set to No

    Resolution

      1. Navigate to Admin Center → Configure Object Definitions.
      2. Search for the custom object related to the custom task.
      3. Select Take Action → Make Correction.
      4. Click on Details for externalCode.
      5. Set the Required property to No.
      6. Change the visibility of external code to 'Editable'.
      7. Save the changes.

      Keywords

      submit button, disabled, onboarding, custom task, attachment, visibility, external code, editable, checklist, task submission.
