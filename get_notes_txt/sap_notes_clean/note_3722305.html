Symptom

  • SuccessFactors Joule option is not visible during proxy login.

  • Joule panel does not load when accessing the system through Proxy Now.

  • No error message is displayed, but Joule functionality is unavailable in the proxied session.

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental. 

Environment

  • SAP SuccessFactors HCM Suite

Reproducing the Issue

  1. Log in to the SAP SuccessFactors instance.

  2. Navigate to Proxy Now.

  3. Select a target user and initiate proxy login.

  4. Once logged in as the proxied user, check for the SuccessFactors Joule option in the interface.

  5. Observe that the SuccessFactors Joule option is not available during the proxy session.

Resolution

  • SuccessFactors Joule does not support proxy mode in SAP SuccessFactors. This is the expected system behavior.
  • Joule functionality is available only when users log in directly with their own credentials.
  • Accessing Joule while proxying another user is not supported; therefore, administrators or other users cannot use Joule on behalf of another person through proxy login.
