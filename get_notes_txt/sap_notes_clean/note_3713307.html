Symptom

The setVariableValue() API in Dropdown.onSelect() event does not work after select dropdown value, while change the variable manually through variable editor is working. 

Environment

SAP Analytics Cloud

Cause

setVariableValue script API will wait for all widgets ready in the background loading. Switch to Viewport loading can help improve performance and solve unresponsive issue.

Resolution

In the Edit toolbar, select Data Refresh ->Loading Optimization Settings -> Viewport loading

Keywords
