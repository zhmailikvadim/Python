Symptom

In SAP HANA Studio modeler, you notice that some nodes have two columns while others have only one column. You want to understand how to control this behavior. 

  • Two columns


  • One column

 "Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Environment

  • SAP HANA, platform edition 1.0
  • SAP HANA, platform edition 2.0

Reproducing the Issue

  1. Open a calculation view HANA Studio.
  2. Select a node (e.g JOIN).

Cause

The second column is displayed when the “Show Description” display option is enabled.

Resolution

Follow below steps:

  1. Right-click the node name.
  2. Enable or disable "Show Description" to control the column display.
