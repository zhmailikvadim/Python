Symptom

Execution Flow icon  is missing in the Real User Monitoring navigation bar. 

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

SAP Focused Run

Reproducing the Issue

  1. Start the application from the Launchpad -> Advanced User Monitoring -> Real User Monitoring.
  2. Select the systems in the scope selection.
  3. Attempt to open the Execution Flow page in the left-side Navigation Bar. 

Cause

This page is only visible for users having authorization to display sensitive data e.g. having role SAP_FRN_APP_RUM_ALL. 

Resolution

Assign the role SAP_FRN_APP_RUM_ALL to the logon user. Once the role is assigned, the Execution Flow icon will become visible in the navigation bar.
