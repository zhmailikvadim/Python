Symptom

In the "General" and "Linked Documents" tabs of the Display Archived Cloud Documents app, some field label descriptions and translations don't match those in the Monitor Process Documents and Monitor Transfer Documents apps. These inconsistencies can cause confusion.

Other Terms

F5805, Display Archived Cloud Documents, Monitor Process Documents, Monitor Transfer Documents, General, Linked Documents.

Reason and Prerequisites

The lables were not updated in this App.

Solution

Implement the SAP Note or install further Support Package.

This note is for the backend system, please implement the note 3713337 which contains the UI part changes to your frontend system.
