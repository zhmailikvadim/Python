Symptom

When a user edits the Purchasing Unit field on the Invoice or IR Header and clicks on search, there are no values in the search results.

Reproducing the Issue

Parameter OrganizationalVisibilityPartitioning must be enabled for this issue to be observable.

Cause

The Requester of the document does not have responsibility for the PU "All" within the Invoice Header Editor group.

Resolution

Either assign the Requester responsibility directly for PU All within the Invoice Header Editor SYSTEM group, or ensure that one of the Requester's custom groups is a child of the Invoice Header Editor SYSTEM Group.

Keywords

Purchasing Unit, Procurement Unit
