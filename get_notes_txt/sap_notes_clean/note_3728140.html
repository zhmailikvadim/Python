Symptom

Error Bank Area &1 has no processing date in Manage Payment app for ORP item. Error occurs despite the bank area posting date having been set in the system, the system does not fetch the date.

Other Terms

F3647, /PF1/MSG_IHB_EOD_DTS(016), APM-IHB, IHB, In-House Banking.

Reason and Prerequisites

Issue occurs as authorization is not bypassed.

Solution

Coding change done to bypass authorization. Implement this SAP Note to fix the authorization issue.
