Symptom

Swedish Absence Splitter payroll function SNAB - For absences that start the day before a holiday but are not full-day absences, the system incorrectly triggers a payroll warning on the holiday itself, indicating that no work schedule exists for that day. This should not occur because holidays should not be considered for absence continuation. The absence should resume only on the next working day.

Other Terms

SNAB, 1SD split, Absence, Karensavdrag, Payroll

Solution

We have modified the processing of planned hours so that holidays are now excluded from consideration. As a result, the system skips holidays when determining absence split continuation.

Important Configuration Note:

  • For non-working holidays, the day type must be set to a value other than 0.
  • If an employee is scheduled to work on a holiday, the day type must be set to 0.
