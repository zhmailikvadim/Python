Symptom

External codes of Foundation Objects, Generic Objects, and picklist option IDs are not visible in the job info page after updating to the latest people profile.

Environment

Successfactors Employee Central

Reproducing the Issue

  1. Update to the latest people profile.
  2. Navigate to the job info page.
  3. Observe that the Foundation Objects, Generic Objects, and picklist option IDs external code is not visible unless you open the edit page.

Cause

External codes for Foundation Objects, Generic Objects, and picklist option IDs do not appear by default.

As an administrator, you can use this admin setting Show external codes for all users to control whether users can show or hide external codes. 

Resolution

  1. Navigate to Settings.
  2. Select People Profile Data Display
  3. Enable the "Show External Codes"
