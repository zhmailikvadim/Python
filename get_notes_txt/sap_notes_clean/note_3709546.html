Symptom

You reprint the interest note through FPI4 and send an error message in event 2060. When you start the printout on the first screen with the interest number entered, the system issues the error message correctly. However, when you reprint after displaying the interest, the system issues error message 0K534 "Program errors" instead of the expected message.

Other Terms

FPI4, Print, Correspondence, REUSE_ALV_HIERSEQ_LIST_DISPLAY, K_KKB_HIER_SEQU_LIST_DISPLAY

Reason and Prerequisites

Program error

Solution

After implementing this SAP Note, the system issues the correct error message from event 2060 when you reprint the interest note through FPI4, regardless of whether you start the printout on the first screen or after displaying the interest.

Implement the attached program correction or import the relevant support package.
