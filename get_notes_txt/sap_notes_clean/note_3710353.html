Symptom

  • Issue with posting Vendor invoice via Advance intercompany scenario.
  • An error occurs during the posting of the Vendor there is a small difference between Sales Order Amount and Purchase Order item amount in MIRO.
  • Error message displayed: "MM_IV_INTERCOMPANY001 - Supplier invoice X was saved with errors. Manual processing required."

Environment

  • Materials Management (MM)
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA
  • Advanced Intercompany Scenario

Reproducing the Issue

  1. Attempt to create an advanced intercompany supplier invoice
  2. In the App Monitor Value Chain, select the Supplier Invoice
  3. On the Message Log tab, verify the displayed error message and details.
  4. Message confirms that exist a small amount difference between Sales Order and Purchase Order
  5. Invoice is Parked

Cause

  • This is the expected system behavior.
  • A small difference in the amount between the Sales Order and Purchase Order requires user confirmation. 

Resolution

Open the supplier invoice in a business application and edit the document manually:

  1. Run T-code MIR4 and input the invoice number.
  2. Select the "Edit: Display/Change" option.
  3. Navigate to the line item level and locate the column labeled `Correction ID`.
  4. Update the `Correction ID` from `1 unclarified error: Park invoice` to `No error`.
  5. Adjust the line item amount and header amount to ensure consistency.
