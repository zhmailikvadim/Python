Symptom

This KBA introduces how to enable Infotype Change Logs  (JP Localization). 

Environment

  • Japan
  • Human Capital Management (HCM)
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for HANA
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition
  • SAP SuccessFactors Employee Central Payroll

Cause

Resolution

To ensure that infotype change logs are properly recorded, configure the following tables:

  1. V_T7JP_UPDINF: Add the necessary infotype.
  2. V_T7JP_UPDSUB: If the infotype includes a subtype, add the relevant subtype. For infotypes without a subtype (e.g., IT0001), this step can be skipped.
  3. V_T7JP_UPDOPN: Specify the operations that should be tracked.
  4. V_T7JP_UPDFLD: Add the fields to be monitored for changes.

After completing the above configuration, infotype change logs will be recorded. These logs can be displayed by running the report RPCUPDJ0.

For more details, please refer to KBA 2714544 - Infotype log enhancement (Page 38~41).
