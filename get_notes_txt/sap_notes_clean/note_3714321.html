Symptom

Where does the stock go if accepted usage decision is made for the early inspection lot before posting goods receipt(GR) to the production order?

It is to differentiate the scenario of 'what happens if usage decision is made before full GR is posted' in test case [TEST CASE] Inspection lot creation: Early lot inspection | SAP Help Portal.

Environment

  • SAP ERP Quality Management (QM)
  • SAP ERP
  • SAP R/3
  • SAP R/3 Enterprise
  • SAP ERP Central Component
  • SAP Enhancement package for SAP ERP
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA, on-premise
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Make master data settings.  
  2. Create a production order for the material. 
  3. Release the production order and an early lot should be created at release. 
  4. Make usage decision(UD) for the early inspection lot.
  5. Make GR for the production order. SPRQ status is set and stock goes to quality inspection. 

Resolution

It is by standard design that the stock will go to quality inspection although accepted usage decision has been made for the early inspection lot before posting goods receipt to the production order. 

It is different from the test case [TEST CASE] Inspection lot creation: Early lot inspection | SAP Help Portal in which the suggested process is:

  1. Avoid making a usage decision for this type of early inspection lot before any goods receipts have been posted. 
  2. Make a usage decision after full goods receipt is posted for the order.
