Symptom

  • Error: The viewer could not process an event. The viewer could not get the page. [CRSDK00001087]
    ---- Error code:0 [CRWEB00000119] When opening Crystal for Enterprise document in BI Launchpad .
  • No issue when opening the document in SAP Crystal Reports for Enterprise designer .

Environment

  • SAP BusinessObjects Business Intelligence 4.3 Support Package 3
  • All supported OS system

Reproducing the Issue

  1. Log on BI Launchpad as Administrator .
  2. Open a Crystal For Enterprise report in BI Launchpad .
  3. An error is displayed:

    Error: The viewer could not process an event. The viewer could not get the page. [CRSDK00001087]
    ---- Error code:0 [CRWEB00000119]

Cause

An issue occurs on the server side when the system tries to open a Crystal Reports for Enterprise document with subreport(s).

Resolution

The issue has been fixed in BO version 4.3 SP3 Patch 3.
