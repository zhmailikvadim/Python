Symptom

  • Error message: "Configuration for output type missing" encountered while using the app "Schedule Accounts Receivable Jobs" (F2366) when selecting job template-Customer Balance Confirmation

Environment

Financial Accounting (FI)
SAP Fiori for SAP ERP
SAP S/4HANA
SAP Fiori for SAP S/4HANA Finance
SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Use the app "Schedule Accounts Receivable Jobs" (F2366) and encounter the error message "Configuration for output type missing" for Customer Balance Confirmation.

Cause

  • The application object FIN_FO_BLNC_CNFRM is exclusive to the S/4HANA Cloud Edition. SAP S/4HANA output management is not supported for this job/program(SAPF130D_OM) in the private cloud/on-premise environment.

Resolution

  1. For S/4HANA On-Premise or Private Cloud, utilize one of the following balance confirmation programs:
  • SAPF130D: Customer Balance Confirmation (SAPscript)
  • SAPF130D_PDF: Customer Balance Confirmation (PDF)
  • SAPF130K: Vendor Balance Confirmation (SAPscript)
  • SAPF130K_PDF: Vendor Balance Confirmation (PDF)

Keywords
