Symptom

How to update old numeric point chart to configure secondary binding provided introduced in QRC1 2026 in SAC?

Image / Data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental  

Environment

SAP Analytics Cloud 2026.2

Resolution

  1. Login to SAC
  2. Open a story containing numeric point charts created before the 2026 QRC 1 Release.
  3. Go to Edit mode
  4. Select the numeric point chart and open the builder panel
  5. Switch to Styling option.
  6. Choose the update widget option.
