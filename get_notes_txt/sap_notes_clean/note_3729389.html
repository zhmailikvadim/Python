Symptom

Application using function TSTR_GET_NEXT_VALID_TIME observe that it is not gernating invalid time stream segments on need.

Other Terms

 TTSTR, TTSTRCOMP, TTSTRCOMPH

Reason and Prerequisites

SAP Note 3716845 introduced a new optional paramter which controls whether the function is allowed to use the old logic, which generates the segments, or uses a new logic which just returns the existing ones based on input data.

Solution

Implement this note or corresponging support package.
