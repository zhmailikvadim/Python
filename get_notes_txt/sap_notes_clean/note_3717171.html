Symptom

  • An activation error "Method 'GET_ODN_TYPE_FROM_INT_TYPE' is unknown or PROTECTED or PRIVATE" occurs during implementing corrections delivered via SAP Note 3605804 using SNOTE.
  • Same Syntax Error also occurs when performing a check on method IF_IDFI_ODN~GET_ODN_VALUE in SE24.

Environment

SAP S/4HANA 

Reproducing the Issue

  1. Access the system and implement corrections delivered via SAP Note 3605804 using transaction SNOTE.
  2. Attempt to activate the object after implementation.
  3. Observe the error message: "Method 'GET_ODN_TYPE_FROM_INT_TYPE' is unknown or PROTECTED or PRIVATE."

Cause

Method "GET_ODN_TYPE_FROM_INT_TYPE" is missing in the system because the note that delivers this method has not been implemented.

Resolution

  1. Implement SAP Note 3565130 to resolve the missing method "GET_ODN_TYPE_FROM_INT_TYPE".
  2. If additional prerequisite notes are required, refer to SAP Note 3565339 for the complete note implementation sequence and solution guide.
  3. Re-implement SAP Note 3605804 after ensuring all prerequisite notes are properly applied.
