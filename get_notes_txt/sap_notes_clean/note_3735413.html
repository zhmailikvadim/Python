Symptom

  • Contingent workers are not excluded when sessions are created using the “Mass Create Calibration Sessions” tool with a validated CSV.
  • When creating sessions through “Manage Calibration Sessions,” the system behaves as expected and shows an error message when attempting to add contingent workers.

Environment

  • SAP SuccessFactors Performance & Goals

Reproducing the Issue

  1. Enable the feature “Exclude Contingent Workers from Session Creation” in Admin Center → Manage Calibration Settings → Global Settings.
  2. Use “Mass Create Calibration Sessions” with a validated CSV that includes contingent workers; observe that sessions are created and contingent workers are not excluded.

Cause

Known issue.

Resolution

Note that this is a known issue that has been fixed in the 2H 2026 release.
