Symptom

The error "Unexpected response received. Contact technical support." is displayed for eDocument from the Country Poland.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Open the eDocument Cockpit app.
  2. Select any available eDocument for Poland.
  3. Choose Go to… -> Application Log.

The error “Unexpected response received. Contact technical support.” is displayed.

Cause

Authorization inconsistency in the communication arrangement for Poland due to missing authentication or an outdated certificate.

Resolution

The communication arrangement maintained for Poland must be reviewed to verify whether any authentication settings are incorrectly maintained or if any certificates are outdated.
