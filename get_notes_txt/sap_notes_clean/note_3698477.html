Symptom

  • The system displays an incorrect tenure end date for a Consolidated Worker.
  • The tenure date shown extends beyond the expected tenure period.

Cause

  • The tenure policy configuration excludes gaps in tenure.
  • A gap exists between assignments for the Consolidated Worker.
  • Because the policy excludes gap days from the tenure calculation, the system extends the tenure end date beyond the expected value.

Resolution

  • Review the tenure policy custom lookup header for Include gaps in tenure? value.
  • If necessary, update the tenure policy setting to include gaps in tenure by changing the value to Yes.
  • Re-trigger the tenure policy on the Consolidated Worker via Edit action or Tenure Policies on Documents Upload.
  • Verify that the tenure end date updates to the expected value.

Note: Ensure compliance requirements are considered before making changes to the tenure policy. This adjustment impacts all records matching the keys in the lookup.
