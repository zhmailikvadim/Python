Symptom

When attempting to access the object worklist (OWL) of a custom service, the following error message is displayed:

  • Error Code: custom.400 Message: No outbound configurations found. Please check with your administrator.

Environment

  • SAP Sales Cloud Version 2
  • SAP Service Cloud Version 2

Reproducing the Issue

  1. Log in to the affected tenant.
  2. Navigate to the Custom Service.
  3. Observe the error message: 
    Error Code: custom.400
Message: No outbound configurations found. Please check with your administrator.

Cause

The issue occurs because the "Query" option, which is necessary for fetching all records in list view, was not selected under the "Configure Host" settings.

Resolution

To resolve this issue, please follow these steps:

  1. Navigate to the Profile Icon.
  2. Go to System Settings > All Settings.
  3. Select Custom Services.
  4. Next to the affected Custom Service, click on Configure Host.
  5. Under Operations, add "Query".
  6. Click Save.

After completing these steps, the error should no longer be visible.
