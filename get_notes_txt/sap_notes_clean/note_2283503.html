Symptom

When an entry from Buyer's worklist is skipped, the record is updated in the MILS. history database table. However there is no indicator in this table to identify if this record was skipped.

Other Terms

FMS, DMP, FMS Buyer worklist, /PSFMS/MILS_HIST

Reason and Prerequisites

New Requirement

Solution

Please implement automatic correction instruction. You can also implement support package SAPK-60007INPSFMS/SAPK-60407INFMFMS, if available.
