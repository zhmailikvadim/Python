Symptom

During the process of eInvoice submission in Germany DRC, the system raises error "[BR-16]-An Invoice shall have at least one Invoice line (BG-25)" in eDocument Cockpit.  

Environment

  • SAP S/4HANA Cloud Public Edition
  • SAP Document and Reporting Compliance, Cloud Edition

Reproducing the Issue

  1. Go to the eDocument cockpit application.
  2. Select the source document.
  3. Click on submit.
  4. Error message appears.

Cause

Missing or incorrect TAX_CATEGORY configuration. 

Resolution

To resolve this error, the Tax Category must be maintained in the TAX_CATEGORY Value Mapping. 

The following steps outline the resolution procedure:  

  1. Go to the Configure Value Mapping app.
  2. Enter the /EDUBL namespace and the TAX_CATEGORY value mapping. Choose Run.
  3. Choose Switch between display and change mode.
  4. Map the tax codes you want to use to the values accepted by Peppol.
  5. Save your changes.

Note: For more information on the values accepted by Peppol, check Duty or tax or fee category code.
