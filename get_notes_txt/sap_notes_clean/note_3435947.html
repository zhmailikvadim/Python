Symptom

In the context of check ID #03, the conversion readiness check reports invalid time stamps in database tables, which prevent the technical upgrade to SAP GTS, edition for SAP HANA.

Other Terms

/SAPSLL/READINESS_CHK_V8000, CHECK-ID #03, /SAPSLL/CUIT, /SAPSLL/CUITDAT, /SAPSLL/LCLIC, RPTTS, RPTDT, DATAB, /SAPSLL/TIME_STAMP_REPAIR

Reason and Prerequisites

This problem is caused by a program error.

Solution

Implement the correction instructions for your release.

Then start the report /SAPSLL/TIME_STAMP_REPAIR in background processing in an update run.
