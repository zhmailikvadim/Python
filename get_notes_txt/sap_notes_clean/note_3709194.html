Symptom

  • A security scan, using some third party scanning tool, reports a potential security vulnerability impacting a file which appears to below to the 
  • In this case, the CVE-2025-68161 
  • Suspect vulnerable file path is: /usr/sap/<SID>/<Instance>/j2ee/cluster/bin/ext/com.sap.aii.adapter.ws.cxf.lib/lib/org.apache.logging.log4j-log4j-core-2.23.1.jar
    Installed version : 2.23.1
    Fixed version : 2.25.3

Environment

SAP NetWeaver 7.5

Cause

  • SAP Security Notes & News - all Security Notes are documented here.
  • SAP Security Patch Day - January 2026 - these are displayed on a monthly basis showing all new Security Notes released each month. Did you check here?
  • Search: SAP for me - always check for any existing SAP Security Note or KBA on the CVE topic, prior to opening a Support Case. 

Resolution

  1. This is not a vulnerability that impacts on the SAP product, the Socket Appender is not used here. 
  2. There is no security risk by this. 
  3. However, SAP will update the impacted jar file to a version which does not have the vulnerability. 
  4. This will be available with an upcoming Support Pack (SP35 in this case).
