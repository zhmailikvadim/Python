Symptom

Customer is getting "currency conversion fail" error message when trying to modify and save story.

Environment

SAP Sales Cloud and SAP Service Cloud Version 2

Cause

The currency conversion may be not set.

Resolution

Set the currency conversion like the documentation below:

https://help.sap.com/docs/CX_NG_SALES/ea5ff8b9460a43cb8765a3c07d3421fe/0b8c99cff1604ede87f109026d018344.html?q=currency+conversion#loio79428ba03dbb460a817511592dabfaa9
