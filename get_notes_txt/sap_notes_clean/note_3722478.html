Symptom

Dump when clicking on Consistency button in VCM Display Chain via LE_SHP_ICO_SUPPORT

Category: ABAP programming error

Runtime Errors: SAPSQL_DATA_LOSS

Except.: CX_SY_OPEN_SQL_DATA_ERROR
ABAP Program: SAP_ICO_VCM_DISPLAY
Application Component: LE-SHP-DL

Short Text
Data was lost while copying a value.

What happened?
Error in the ABAP application program.

The current ABAP program "SAP_ICO_VCM_DISPLAY" had to be terminated because it found a statement that could not be executed.

Other Terms

LE_SHP_ICO_SUPPORT, VCM Display Chain, Consistency button, SAPSQL_DATA_LOSS, SAP_ICO_VCM_DISPLAY, IC_SALES, IC_STOCKTRANFER, Value Chain Monitor, VCM

Reason and Prerequisites

Programming error, This issue occurred while attempting to fetch Supplier Invoice data from table VBRP, which is incorrect.

Solution

Implement this note to correct the code in SAP_ICO_VCM_DISPLAY program.
