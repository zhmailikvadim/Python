Symptom

When attempting to add partner functions in Custom Partner schemas for the Supplier Master, the system displays the error message 'Cannot use this partner role'.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Go to Implementation Activities app.
  2. Search for Configuration Activity 103125 - Define Partner Schemas in Supplier Master Record.
  3. Create a custom Partner Schema in the Partner determination procedure.
  4. Navigate to Partner Functions in Procedure.
  5. Add partner function to custom Partner Schema.
  6. Encounter error: 'Cannot use this partner role Message no. CZ327'.

Resolution

  • In SSCUI 103125 – Define Partner Schemas in Supplier Master Record, only the partner types LI, PE, and AP are supported. Other standard partner functions are not supported in this configuration activity, which is a standard system limitation.
  • To avoid this issue and add a custom partner function to the schema, ensure that the custom partner function is maintained as a supported partner type in SSCUI 101286 – Set Up Partner Determination for Customer and Supplier Master.
