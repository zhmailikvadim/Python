Symptom

In Process DE_B_IM_EMBED_DISPR_CALLER (version 0001) step 1000, the check ID CHECK_IM_IMS_RELEVANT has returned "Non-IMS" as check result although Meter and SMGW device is involved.

Other Terms

DE_B_IM_EMBED_DISPR_CALLER, Version 0001, DSO

Reason and Prerequisites

Program error.

It checks all device in the installation. Actually only Meter and SMGW device should be checked.

Solution

Implement the correction instruction.
