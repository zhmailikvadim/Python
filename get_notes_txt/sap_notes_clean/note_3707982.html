Symptom

French translation for button 'Not started' in FIORI app F2049 Manage RFQs is missing F2049.

Environment

  • Materials Management (MM)
  • SAP S/4HANA
  • SAP Fiori

Cause

Incorrect configuration.

Resolution

 Enter transaction SE63:

  • Enter Object Type: REPT

  • Class: CL_MM_PUR_RFQ_APPROVER_EXIT

  • Source Language: enUS (English)

  • Target Language: frFR (French)

Then edit and update the translation maintained for ‘Not Started’.

Keywords

French, translation, F2049, Manage RFQ, Manage RFQs, CL_MM_PUR_RFQ_APPROVER_EXIT, Not Started
