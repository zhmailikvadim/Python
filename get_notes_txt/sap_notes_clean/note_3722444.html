Symptom

Object link for Inspection Lot (QALS) is not visible in the SAP Document Management System (DMS) functionality

Environment

  • SAP Document Management System (CA-DMS)
  • SAP S/4HANA on-premise
  • SAP S/4HANA Cloud Private Edition 

Reproducing the Issue

  1. Transaction CV01N
  2. Select Document Type
  3. Go to 'Object Links' tab and select 'Inspection Lot'

Cause

Resolution

  1. Transaction SPRO
  2. Activate the functionality in customizing at 'Quality Management -> Basic Settings -> Maintain Settings at Client Level' using the 'Document Link to Insp. Lot' switch  
  3. In Customizing for document management under "Cross-Application Components -> Document Management -> Control data -> Define Document Types", define at least one document type that allows the assignment of inspection lots (object QALS for object link). Specify Dynpro 500 for the object link.
  4. If the QALS object type is not visible in transaction DC10 via the search menu:

a. Add the QALS object type in the customizing at:

Cross-Application Components -> Document Management -> Control Data -> Maintain Screen for Object Link
with the following entry:

  • Object: QALS
  • Object description: Inspection Lot
  • Number: 500

b. Afterward, select this object type in transaction DC10 to maintain it to a document type.
