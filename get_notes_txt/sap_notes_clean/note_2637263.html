Symptom

The current setting for "dsi_keep_triggers" is turned on for the Active connection of WarmStandBy pair.

Environment

SAP Replication Server (SRS) 15.7 and above

Resolution

By default, the "dsi_keep_triggers" setting is on for the Active database and off for the Standby database in a WarmStandBy pair.
