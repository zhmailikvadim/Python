Symptom

You are working in the transportation cockpit and introduce via the page layout customizing a tooltip for an application toolbar button choice. Additionally, you create a new page layout or restore the pushbutton data for existing page Layout(s). The transportation cockpit did not display defined tooltip for the Button Choice. 

Other Terms

TC, /SCMTMSPLN, toolbar, button choice, tooltip

Reason and Prerequisites

The problem is caused by a program error.

Solution

Implement the attached correction instructions or import the specified Support Package.
