Symptom

Is there an option to upgrade HANA version to 2.0 in BTP(Business Technology Platform) / SAP Cloud Platform?

Environment

  • SAP Cloud Platform
  • NEO
  • BTP

Cause

SAP HANA 1.0 reaches end-of-maintenance in June 2021, which also affects the SAP HANA systems based on SAP HANA 1.0 running in the SAP Cloud Platform Neo environment.
EPM-MDS plugin must be installed right after HANA2.0 upgrade on NEO for live connectivity to work. Here is a blog that can walk you through it: SAP Analytics cloud connection to HANA2 on BTP Neo

Resolution

  1. The current available HANA Version on NEO platform is 1.00.122.29.
    Update Database Systems:

    https://help.sap.com/viewer/d4790b2de2f4429db6f3dff54e4d7b3a/Cloud/en-US/93f9728df6c842d1aed4bcab3ecd44c1.html

    Work with SAP HANA 1.0
    Work with SAP HANA tenant and single-container database systems and SAP HANA version 1.0.

  2. As a first milestone release towards this goal, we are pleased to announce the availability of SAP HANA version 2.0 on the Neo environment.

    You will now be able to provision new SAP HANA 2.0 systems

  3. SAP HANA 2.0: Welcome to SAP Cloud Platform, Neo environment

  4. The target platform will be HANA 2 SPS5 and the minimal supported version is HANA 1.00.122.29+
    Also a SDC-2-MDC migration will be necessary for the update. This is self-service for customer, will be enabled first as Beta Feature.

  5. It is now also possible to upgrade 32Gb tee shirt size to HANA 2.0, this can be prepared by the Operations team as part of the OS migration to SLES 12 SP 5.

  6. 2960608 - Converting SAP HANA XS to SAP HANA MDC in the SAP Cloud Platform, Neo Environment
    2946614 - End-of-Maintenance of SAP HANA 1.0 on SAP Cloud Platform

  7. As of the release of SAP Cloud Platform on January 14, 2021, SAP HANA 2.0 is available for update in the Neo environment and the conversion is generally available, which allows customers to do the following:

    • Customers running tenant database ("MDC") systems that are running on SLES 12 SP 5 can update their SAP HANA 1.0 systems to SAP HANA 2.0 using the Update functionality in the SAP Cloud Platform cockpit.
    • Customers running single-container ("XS") systems can convert their systems into MDC systems. After the conversion, they will be able to update their SAP HANA 1.0 systems to 2.0 using the Update functionality in the SAP Cloud Platform cockpit. 
  8. 2469025 - How-To: Upgrading to SAP HANA 2.0
  9. EPM-MDS plugin must be installed right after HANA2.0 upgrade on NEO for live connectivity to work. Here is a blog that can walk you through it: SAP Analytics cloud connection to HANA2 on BTP Neo​​​​​​​
