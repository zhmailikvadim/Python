Symptom

 SAP Adaptive Extensions: Fixes for Patch 92

Other Terms

SAP Adaptive Extensions, SAPACEXT, sapacext, storage library, platform library, LaMa HADR, hook, illegal character, Linux Cluster, pacemaker cluster

Reason and Prerequisites

Fixes for Patch 92

  • LaMa HADR hook version validation not working for secondary or stopped instances
  • Database refresh failed for Windows system wrong default profile selected

    Solution

    Obtain the SAPACEXT.SAR files from SAP Support Portal at https://launchpad.support.sap.com/#/softwarecenter -> Support Packages and Patches -> By Category -> SAP Technology Components -> SAP Adaptive Extension

    To install, upgrade, or downgrade SAP Adaptive Extensions, see SAP Help Portal at https://help.sap.com/docs/SAP_ADAPTIVE_EXTENSIONS.

    For more information about installing, upgrading, or downgrading SAP Adaptive Extensions using the enterprise edition of SAP Landscape Management, see Managing SAP Adaptive Extensions Installations on SAP Help Portal.

    • For more information about the enterprise edition, see SAP Help Portal at http://help.sap.com/lama.
    • For more information about the standard edition, see SAP Help Portal at http://help.sap.com/lamastd.
