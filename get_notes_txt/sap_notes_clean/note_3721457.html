Symptom

 Difficulty in determining commodity codes or service codes (specific to Italy) when no material is available in the document.

Environment

  • S/4HANA
  • S/4HANA Private Cloud 
  • Intrastat Reporting 

Cause

Classification relies on material master data, and materials without this data lack association with commodity codes or service codes.

Resolution

  • Commodity codes or service codes cannot be determined automatically via standard without material master data.
  • Use BADI_ITRASTAT_SELECTION to manually determine these codes.
  • Refer to note 3364197 for detailed instructions on using the BADI.
    • In BADI_INTRASTAT_SELECTION, establish an association between the purchase order number, purchase order item, and the commodity code or service code which must be assigned.
