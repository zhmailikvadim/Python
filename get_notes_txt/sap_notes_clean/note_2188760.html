Symptom

When GRAC_RISK_ANALYSIS_WOUT_NO_WS web service is executed, report format 3 and 4 for the risk analysis were not considered. Only the report format value 2 was considered.

Other Terms

Access Control, CL_GRAC_IDM_INBOUND_SERVICES, GRAC_RISK_ANALYSIS_WOUT_NO_WS.

Reason and Prerequisites

A fixed value was set for report format.

Solution

Implement the correction instruction.
