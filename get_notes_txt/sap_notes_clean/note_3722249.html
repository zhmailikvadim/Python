Symptom

This SAP Note addresses the following symptoms.

1. The Submit action is incorrectly enabled when a document is in Sent to DRCC or Acknowledged by DRCC status. Performing this action leads to a Rejected by DRCC status because the document already holds an Accepted, Rejected, or Failed status in the Document and Reporting Compliance Cloud Edition. Additionally, the Get Service Status action fails to retrieve the most recent status for the affected documents.
For more details on how to resolve the status mismatches, please refer to the Solution section of this SAP Note.

2. After resubmitting documents that are in Rejected by KSeF or Rejected by KSeF (Offline) status, the Display XML action continues to show the previous version of the XML instead of the updated one.

Solution

Date Version Reason
02.03.2026 1 Released for Customers
25.03.2026 2 Correction for SAP APPL 606

Import the Support Package specified for your release or implement the attached correction instruction.

SAP recommends that you implement the Support Package specified for your release. However, if you need to install a solution earlier, use the SAP Note Assistant to implement the correction instruction. You can find more information about the SAP Note Assistant in SAP Support Portal, under SAP Note Assistant.

You must also implement the customization activities shared in the attachment SAP Note 3722249 Customization Steps PDF.

Once the corrections are available in your system, please follow the steps below to resolve the status mismatch.

1. Open the Electronic Document Cockpit or Manage Electronic Documents application.

2. Select the Electronic Documents with status mismatch and perform More -> Get Status to pull the latest status from Document and Reporting Compliance Cloud Edition.

3. Once the latest status is received in the Business System, you can take the necessary follow up actions - Send to Customer/More -> Display PDF on documents in Accepted by KSeF status, Retry on documents in Failed to Send status and Submit on documents in Rejected by KSeF status.
