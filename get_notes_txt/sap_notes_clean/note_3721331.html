Symptom

The "HSN Code" and "Tax Value" are not being fetched in the XML file for the purchase order form template "YY1_MM_PUR_PURCHASE__ORDER_IN".

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Access the "Manage Purchase Order" app.
  2. Navigate to the "Output" tab.
  3. Select "Print Preview" for the purchase order.
  4. Observe that the HSN Code and tax value are missing in the XML file for the form template "YY1_MM_PUR_PURCHASE__ORDER_IN".

Resolution

To resolve the error, maintain the following configuration:

Replace the gateway service FDP_EF_PURCHASE_ORDER_GLO_MENA_SRV with FDP_EF_PURCHASE_ORDER_GLO_GEN_SRV for the form template “YY1_MM_PUR_PURCHASE__ORDER_IN”.
