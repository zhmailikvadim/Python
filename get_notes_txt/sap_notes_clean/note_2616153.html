Symptom

  • Is is possible to silently uninstall Lumira? 
  • Whenever running the silent uninstall parameters, the pop-up to confirm the uninstallation occurs

Environment

  • SAP Lumira Desktop 1.X
  • SAP Lumira Discovery 2.X
  • Windows

Reproducing the Issue

  1.  Enter an uninstall string in the registry such as "C:\Program Files\SAP BusinessObjects Lumira\Lumira Discovery\setup.exe" -u product.hilo-4.0-core-32

Cause

  •  Silent uninstall is not possible in Lumira 

Resolution

  • This is a by design feature
  • For Business Objects Enterprise products there is no silent uninstall feature
