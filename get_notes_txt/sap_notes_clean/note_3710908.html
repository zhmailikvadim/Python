Symptom

In the System Monitoring application, the SAP Content Server metric "Content Server Info" is not collecting information, and the following error message is displayed: "com.sap.solman.agent.mai.api.ConfigCollectorException: Parameter PORT must not be empty".

Environment

SAP Focused Run 4.0 and above

Reproducing the Issue

  1. Register and configure the respective SAP Content Server in FRUN.
  2. Navigate to the System Monitoring application.
  3. Check the status of the metric "Content Server Info".
  4. Observe that the metric is gray rated and the error message "Parameter PORT must not be empty" is displayed.

Cause

The HTTP port is missing in the Landscape Management (LMDB) application.

Resolution

Review the system registration according to the steps below:

  1. Ensure that the system is correctly registered according to the SAP Content Server page of the SAP Focused Run Expert Portal.
  2. In the Landscape Management (LMDB) application, check if the Content Server HTTP port is maintained in the TCP/IP tab.
    1. If no HTTP port is maintained, please follow the steps in the "Add Content Server Port" section of the SAP Content Server Monitoring Guide to add it manually.
    2. If the HTTP port is already maintained, please open a case for the SV-FRN-INF-MAI component, providing the SDA logs in debug for "com.sap.solman.agent.mai".
      • To activate the SDA debug and download the logs, please follow the steps of the KBAs 2696231 and 2954399.
