Symptom

Differences of adding an existing SAC tenant to Formations rather than creating a new tenant. Are there any limitations to this?

Environment

SAP Analytics Cloud features specifically related to BDC.

Resolution

  • Product features inside SAC (stories, planning, modeling, visualizations, etc.) do not change based on how the tenant was created.
  • ONLY existing CF SAC Tenants can be used in BDC formation or you can create a new SAC tenant within the Business Data Cloud Suite.  
  • Under BDC Suite, SAC is available as an application that can be provisioned. No matter how the tenant is created, it can be integrated with other BDC components via BDC formation.

For more details on Formations, please refer to: Creating SAP Business Data Cloud Formations
