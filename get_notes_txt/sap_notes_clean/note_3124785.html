Symptom

You receive error message CK322 during sales order costing.

Other Terms

VA01, VA21, VA22, CK55, VBR, VAY, VAX

Reason and Prerequisites

Program error caused by SAP Note 2878012 in case of sales order items with sales order controlling (i.e. sales order item as controlling object). Due to this, the system is searching for a consumption account using GBB/VBR instead of GBB/VAY.

Solution

Apply attached correction.
