Symptom

  • Transaction code FAGL_FCV is posting foreign currency valuations to additional ledger incorrectly.
  • Ledger group contains only ledger "0L", but figures are being posted to additional ledger incorrectly.

Environment

  • SAP S/4HANA
  • SAP S/4HANA Finance

Reproducing the Issue

  1. Start transaction FAGL_FCV.
  2. Enter necessary parameters.
  3. Run the valuation and review the generated postings and assigned ledger.

Cause

A ledger group with the same name as "0L" was created, under which both ledger "0L" and the additional ledger were maintained; this configuration might cause valuation postings to appear in an unintended ledger.

Resolution

  1. Implement SAP Note 3580480 to adjust the ranking logic for the target ledger group in valuation posting.
  2. After implementation, run FAGL_FCV with the same parameters and verify that postings are created only in the intended ledger.
