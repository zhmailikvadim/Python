Symptom

When creating a new entry in the "Define Warehouse-dependent IOT Settings" customizing activity by copying from an existing Inspection Object Type (IOT), the system replicates all data fields. This may include parameters that are not applicable to the new entry, potentially leading to inconsistencies or errors during follow-on processing.

Other Terms

IOT, QM, embedded, decentral, /SCWM/QIOTWM

Reason and Prerequisites

 Program error.

Solution

 Implement correction instruction.
