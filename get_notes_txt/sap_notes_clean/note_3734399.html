Symptom

  • Leave encashment amounts does not appear in Form-16 although they are entered in infotype 0015/0416 and visible in the payroll result.
  • The issue persists despite the evaluation class 6 being correctly maintained for the gratuity wage type.

Environment

  • Human Capital Management (HCM)
  • SAP R/3
  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • Payroll India

Reproducing the Issue

  1. Run the Form 16 transaction code: PC00_M40_F16.
  2. Select the relevant variant.
  3. Check whether the specific Leave encashment value is showing in the 'Emoluments Paid' section of Annexure.

Cause

Incorrect configuration of the custom leave encashment wage type.

Resolution

  1. Go to SM30 t-code.
  2. Pass the view name as V_512W_D.
  3. Click on the maintain button.
  4. Pass the HCM Localization value as '40'.
  5. Locate the leave encashment wage type used for posting the amount.
  6. Set Processing Class 30 to value ‘T’ .
  7. Save the changes.
  8. If still issue persists , check the entry of Leave encashment wage type on table V_T54C3. Entry needs to be present for both cumulation type 'C' , 'Y'.
