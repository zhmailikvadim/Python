Symptom

  • Unable to find "Manage route map" under Admin Center.
  • Routing map permission is missing in Role-Based Permissions (RBP).
  • No option "Routing map" under Administrator Permissions -> Manage Templates.

Environment

SAP SuccessFactors Performance Management

Reproducing the Issue

  1. Navigate to Admin Center.
  2. Go to Manage Permission Roles.
  3. Open any role.
  4. Under Administrator Permissions -> Manage Templates, observe that Routing map permission is missing.

Cause

  • Workflow option is not enabled in provisioning.

Resolution

  1. Enable Workflow option from provisioning, as it is one of the prerequisites for showing the route map under RBP.
  2. Note: Access to provisioning is restricted for customer. Contact the implementation partner or Account Executive for provisioning-related tasks. For non-implementation tasks, contact Technical Support.

See Also
