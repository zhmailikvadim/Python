Symptom

No customs declaration is displayed in the worklist of app 'Display Export Declarations', after implement Note 3601056 to improve performance.

Other Terms

USE_HEX_PLAN, Cardinality, I_CustomsDocumentReferenceDoc,  I_ExportDeclaration

Reason and Prerequisites

Program Error

Solution

Apply the note or Upgrade SP Levels

When you search for risk assessments in EHS Risk Management using the location-based search, the system displays the location structure in an outdated UI tree. This UI tree has display issues when you resize the screen.

Other Terms

WDCC, EHHSS_UI_RAS_RSK_LOC_TREE

Reason and Prerequisites

A program error causes this issue.

For prerequisites, see the correction instructions in this SAP Note.

Solution

After implementing this SAP Note, the system displays the location structure in an updated tree format that adjusts correctly to screen size changes.

Alternatively, you can import the corresponding support package for the component extension of SAP Environment, Health, and Safety Management or the Feature Package Stack for the SAP S/4HANA system.
