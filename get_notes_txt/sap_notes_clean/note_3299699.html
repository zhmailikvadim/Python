Symptom

How do I enable DMS-16627 - Support for RFQ Lean Service Hierarchies in the S/4HANA Sourcing with SAP Ariba Sourcing (4BL) Integration Scenario?

Resolution

To enable DMS-16627, the following two parameters must be set to True in the Intelligent Configuration Manager:

  • Application.ACM.S4Hana.ComplexServiceHierarchyRFQSupport
  • Application.ACM.ComplexServiceHierarchy.Enabled

The default value for both parameters is False.

DMS-16627: Support for RFQ Lean Service Hierarchies in the S/4HANA Sourcing with SAP Ariba Sourcing (4BL) Integration Scenario.

Enable support for RFQ lean service hierarchies in the S/4HANA Sourcing with SAP Ariba Sourcing (4BL) integration scenario - Determines whether RFQ lean service hierarchies can be used in the SAP Ariba Sourcing integration with SAP S/4HANA Cloud 4BL integration scenario.

Enable support for the use of lean service hierarchies in RFQs - Determines whether RFQ lean service hierarchies can be used in SAP Ariba Sourcing.
