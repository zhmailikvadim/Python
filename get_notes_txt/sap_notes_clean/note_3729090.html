Symptom

The commodity code on a planned Service Entry Sheet (SES) item does not match the corresponding Purchase Order (PO) line item.

Cause

The item is a catalog item, and the catalog item is mapped to a different commodity code than that of the PO line item.

Resolution

If the item is a catalog item, before submitting the Purchase Requisition (PR), ensure the commodity code on the catalog item corresponds to the item commodity code present on the catalog.
