Symptom

The status of the eDocuments Chile DTE are being displayed as "Sent to SII Acknowledged".

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Go to the eDocument Cockpit App.
  2. Select the Chile DTE.

The status of the eDocuments are displayed as "Approved By SII (SII Email)".

Cause

According to the documentation for the Process Statuses for Chile DTE, the eDocument status is updated to "Approved By SII (SII Email)" to match the approval status sent by SII through the acknowledgement e-mail.

The Process Statuses for Chile DTE Documentation can be accessed through the following steps:

    1. Go to the eDocument Cockpit App.
    2. Select the Chile DTE.
    3. Select the Help button (Exclamation Mark Icon).
    4. Select Process Statuses for Chile DTE link.

    Status: Approved by SII (SII E-Mail) (APSE)
    Explanation: The eDocument status has been updated to match the approval status sent by SII through the acknowledgement e-mail.    
    Actions: You can send the eDocument to your customer

    Resolution

    When a Chile DTE eDocument has the Status as "Approved By SII (SII Email)", it is recommend to perform the action "Send to Customer" in order to complete the process for the eDocument.

    1. Go to the eDocument Cockpit App.
    2. Select the Chile DTE.
    3. Mark the eDocument with Status as "Approved by SII (SII E-Mail)".
    4. Select Send to Customer button.

    The Status of the eDocument should be displayed as Sent to Customer.
