Symptom

There is no description for the ERT - Material Serial Number

Environment

SAP Service and Asset Manager all versions

Reproducing the Issue

  1. Navigate to the work order List
  2. Work order 4027420
  3. Open operations list
  4. Select "check for additional damage"
  5. Open Parts list
  6. Select "Intake Assembly SN 0300"
  7. Issue Part
  8. Select "Serial Number"
  9. Serial Number list with only serial number without Material description

Cause

This is the current design

Resolution

If you would like to open an enhancement request, please see below note 3033176.
