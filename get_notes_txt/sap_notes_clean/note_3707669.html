Symptom

In the application "My Situations - Extended" the operations assinging/closing and other operations on a Situation Instance need more then 3 seconds to complete.

Reason and Prerequisites

During these operations the Situation Handling framework is using metadata of the affected Business Objects which is obtained from the runtime api .
This metadata is used multiple times in various methods to perform the operation and so the Situation Handling framework caches this metadata on DB and in memory.
Unfortunately the memory cache was not used and the metadata was loaded multiple times in this case.

Solution

Please apply the attached correction instructions. This ensures that the memory cache is properly used.
