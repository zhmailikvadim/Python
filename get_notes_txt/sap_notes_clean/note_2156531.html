Symptom

When clicked on create incident, and entering injured tab. On adding injured person, all these tab like basic/injury/summary are misaligned.

Environment

  • SAP ERP
  • SAP ERP Central Component
  • SAP Enhancement package for SAP ERP
  • EHSM 4.0

Reproducing the Issue

1. Logon NWBC
2. Navigate to EH&S Management workcenter
3. Choose Incident and click service Create Incident
4. On the step 2 Injured,  click adding Injured Person, Basics/Injuries/Protecivtive Equipment/Summary are misaligned.

Cause

This is due to program error in Web Dynpro Floorplanmanager.

Resolution

Implement BC note 2039794 or upgrade support package level as mentioned in the note.

Keywords
