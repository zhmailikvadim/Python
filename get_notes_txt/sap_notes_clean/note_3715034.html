Symptom

The data extraction logic of datasource 0TB_TOC_CHARGE_TRAN does not support an extraction with Init with Data Transfer (Mode of Data Update = S (Simulation of Initialization of Delta Transfer)). This is required for the delta initialization process.

Other Terms

BW, 0TB, Business Warehouse

UPDMODE

Solution

The data extraction is enhanced to allow a data extraction with Init with Data Transfer.

Import the relevant Support Package that is specified for your release. As a manual advance correction, you can implement the attached correction instructions using transaction SNOTE.

Please also implement SAP notes 3715057 and 3715485.
