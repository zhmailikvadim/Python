Symptom

BBP: EA63 terminates with error TU-555 "No lines exist in budget billing plan &1 for contract &2"

EA29 Printing error E9-011 "BBP not available" / EA62 E61D TU-555

Reproducing the Issue

EA63 EA62 -> TU-555

EA40 / EA60 ->  E9-011 E9011
E61D ->  TU-555 TU-226  TU555 TU226
EA29 E9011 "BBP 540004319101 doesn't exit"

Cause

The involved budget billing plan is inconsistent since only header data are still available (table EABP)

Resolution

You may try to delete this budget billing plan using E61D. If this doesn't work then I recommend to review the attached note 0000546570 and its report ZANADEL_EABP - although it is quite old, but it is still valid. Please use this report for correcting inconsistent budget billing plan data, but please try it in test/backup system at first.
