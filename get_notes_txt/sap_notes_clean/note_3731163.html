Symptom

When managing substitution in SBWP transaction, the button "Substitutes for positions" does not exist in the SAP Quartz and Belize theme, meaning only the "Personal Substitutes" view is available.

 For example, the Classic theme shows the button "Substitutes for positions".

Other Terms

SBWP, Edit Substitution

Reason and Prerequisites

The GUI status SAPLRHW2 SUBST_N is old.

Solution

Apply attached correction instruction or import corresponding support package.

Alternatively, to display the button, you need to adapt the Fiori Usage of the function code.
Open the status SAPLRHW2  SUBST_N in the menu painter SE41 and double click on the function code SUBCH (field name GV_0200_SUBCH_TEXT) to open the function attributes.
In the dropdown menu for "Usage (Fiori)" you have "Default" value, change it to "Application-Specific Action" .
