Symptom

You wonder how to activate the Development Console in SAP Sports One.

Environment

  • SAP Sports One
  • Google Chrome

Cause

In order to get more detailed information about unexpected system behaviour or unclear error messages you can make background actions visible and share them.

Resolution

  1. Open the Customize and control dialog in Google Chrome (3 dots symbol, top right corner of the screen).
  2. Via More tools select Developer tools.
  3. The developer tools appear in an additional window on the right side of your screen.
  4. Go to the Application tab.
  5. Choose the Clear storage section.
  6. Click on the Clear site data button.
  7. Go to the Network tab.
  8. Now reproduce the issue you are facing.
  9. When done save the log via right click on the lower part of the window and Save as.
  10. Attach the log file to the case.
