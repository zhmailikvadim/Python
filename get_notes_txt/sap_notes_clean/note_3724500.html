Symptom

  • On the mobile app, the “My Calendar” label (under More) and certain Time Off-related notifications (e.g., “February”) remain in English despite the user and device being set to French (France) language.
  • Label keys exist and French translations are correctly maintained in Manage Languages; English Debug shows the “My Calendar” key is valid, but the translation isn’t applied on mobile.
  • Attempts: Reinstalling
  • The issue is only happening in the mobile app.

Environment

  • SAP SuccessFactors Employee Central: Time Off
  • SAP SuccessFactors HCM Suite 

Reproducing the Issue

  1. Log in to the mobile app
  2. Make sure that the language is set in French
  3. In Android operating system
     - The month of February is not translated to Fevrier.
  4. In iOS
    -  The My Calendar is  not translated to Mon calendrier.

Cause

The strings are not added to be part of the translations.

Resolution

The strings will be added to be part of the translations; it will be available in next application update which is H2605.

Production Schedule Date: May 13, 2026
