Symptom

  • SAC live connection is failing.
  • RFC error NIECONN_REFUSED(-10) encountered when creating connection from t-code /SSB/UTIL_SAC_S4HANA.

Environment

SAP S4H embedded SAC UI integration

Reproducing the Issue

  1. Configure SAC live connection using transaction /n/SSB/UTIL_SAC_S4HANA.
  2. Check the connection.

Cause

  • Invalid hostname configuration in transaction /n/SSB/UTIL_SAC_S4HANA.

Resolution

  1. Run transaction /n/SSB/UTIL_SAC_S4HANA.
  2. Select the "Edit Connection" option.
  3. Locate the specific connection and choose "Edit" for the hostname.
  4. Update the hostname to use the SAC tenant URL. Ensure that the Web Dispatcher or S4 system URL is not used for this configuration.
