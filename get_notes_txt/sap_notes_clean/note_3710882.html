Symptom

  • In the SAP Work Zone instance, the sub-navigation background menu is in grey color/theme on the desktop version however it displays in black on mobile or tablet browsers, which shows inconsistency.

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Environment

SAP Workzone 

Reproducing the Issue

  1. Access SAP Work Zone system via desktop browser and observe the sub-navigation background menu color/theme displayed in grey.
  2. Access the same system via mobile or tablet browser and observe the sub-navigation background menu color/theme displayed in black.

Desktop Browser version:

Mobile/tablet browser Version:

Cause

The behavior observed, where the background of the sub-navigation menu differs between desktop and mobile browsers, has been confirmed as expected behavior. This is part of the current design and functionality of the system as per Engineering.

Resolution

As confirmed by the Engineering team, discrepancy on the Sub navigation workzone background menu theme or color is an expected behavior for example if color/theme is set up in grey for Desktop browser then on the mobile/tablet browser it will display black. This is currently part of the system design and functionality. This is to Improve contrast and readability on smaller screen devices.
