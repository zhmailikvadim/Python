Symptom

  • User wants to use a generic email address for the system owner of the Datasphere and/or Business Data Cloud.
  • User wants to know which email address  can be used by the system owner of the Datasphere and/or Business Data Cloud.

Environment

Provisioning of Datasphere and/or Business Data Cloud tenants.

Reproducing the Issue

While provisioning a Datasphere and/or Business Data Cloud tenants the email address for system owner must be informed

Cause

Resolution

Any email address can assign as system owner, but it must be valid in your corporate email system and it must be assigned to a Datasphere and/or Business Data Cloud user account.
