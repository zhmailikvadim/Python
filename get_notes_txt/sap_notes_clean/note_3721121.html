Symptom

F4 help is not functioning in the Fiori App "Manage Workflows for Condition Contracts" (App ID F3680). 

Environment

  • Condition Contract Settlement in S/4HANA

Reproducing the Issue

  1. Execute Fiori app "Manage Workflows for Condition Contracts" (App ID F3680).
  2. Create a new workflow
  3. Navigate to Start Conditions.
  4. Select Condition Contract Type; no Matchcode is displayed in the input field.

Cause

  • Configuration issue.
  • The relevant OData service must be active for the search help values to display correctly in Manage Workflow. 

Resolution

Activate the OData service LO_SETMAN_WFVH (Settlement Management Workflow Value Helps) using the transaction /IWFND/MAINT_SERVICE in the backend system.

Keywords

Matchcode, Match code, F4 value help
