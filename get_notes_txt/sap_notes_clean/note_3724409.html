Symptom

In the Central Finance scenario cross-company documents are parked in the source document. During the replication AIF displays the error message FINS_CFIN_AC_DOC160 'Document was already posted with &1 &2 &3' where the referenced document is not related to the currently processed document.

Environment

  • S4CORE 108
  • S4CORE 109

Reproducing the Issue

1. Park cross-company documents in the source system
2. Display error message in target system AIF Error Handling

Cause

Replication of parked cross-company documents is not supported.

Resolution

Cancel the replication of parked cross-company documents in AIF Error Handling.

To prevent the replication of these documents implement the following SAP Notes in the source system:
1. 3653688 - Intercompany parked documents
2. 3471345 - Cross company parked document transfer check
