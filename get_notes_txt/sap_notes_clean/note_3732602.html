Symptom

When you generate a DANFE in the eDocument Cockpit (EDOC_COCKPIT) for an NF-e XML file, the name of the issuer does not appear in the Nome tag value. Only the Nome Fantasia tag is displayed.

Other Terms

Inbound, NF-e, Nota Fiscal Eletronica, Nota Fiscal Eletrônica, Electronic Nota Fiscal, Brazil, Brazil eInvoice, InfoAdProd, DANFE, EDOC_COCKPIT, eDocument Cockpit, EDOC_BR_DANFE_OP, EDOC_BR_DANFE,  NF-e Items, Nome, Nome Fantasia, Issuer

Reason and Prerequisites

This SAP Note is relevant only for Brazil.

Solution

After you implement this SAP Note, the Nome tag is displayed correctly in the DANFE.

Use the SNOTE transaction to implement the attached correction instructions.
