Symptom

Oracle databases are having issues with Oracle SQL Patch [DEPRECATED] (S00441) and this extractor runs into an error state

Environment

SAP Focused Run 5.0

SAP Focused Run 4.0

Reproducing the Issue

  1. Navigate to CSA Administration and select the affected system.
  2. From the list of scheduled extractors, filter and locate S00441.

Cause

Extractor S00441, which collects data for the store “Oracle SQL Patch”, has been deprecated and is no longer maintained.

Resolution

Extractor S00441 (“Oracle SQL Patch”), has been deprecated and is no longer maintained. The same data is now available via an alternate store, “Oracle SQL patches installed”, which is collected by a different extractor S00467.

Therefore, the reported failure of extractor S00441 does not impact data availability, as the information continues to be collected through the supported extractor.

If you prefer not to see the failed extractor anymore, extractor S00441 can be disabled for the affected systems by following the steps below:

Steps to switch off an extractor for a specific system:

Navigate to CSA Administration and select the affected system.
From the list of scheduled extractors, filter and locate S00441.
Click the Configuration icon → Edit → Start Condition → Settings → set it to Switched off.
Retrigger the extractor or wait for the next scheduled run.
