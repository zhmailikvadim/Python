Symptom

It takes a long time to get the data in the App, or it displays "loading" all the time and cannot get results in Manage Financial Transaction (F6157) App, or even get a dump saying "CX_SADL_DUMP_DATABASE_FAILURE" in the backend.

Other Terms

F6157, R_FinTransCashFlowActivity, VTBFHAZU, C_FinTransactionManage, CX_SADL_DUMP_DATABASE_FAILURE

Reason and Prerequisites

This issue is caused by CDS View performance. If there are so many daily fixing entries(E.g. Interest Rate Adjustment/Foreign Exchange Rate Adjustment), these entries are selected but useless in Manage Financial Transaction App.

Solution

Implement the SAP Note or corresponding Support Package.
