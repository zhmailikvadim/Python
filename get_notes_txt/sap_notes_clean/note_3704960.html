Symptom

  • The Active Period date of a Supplemental Task is not respected, and the task appears with a one-day delay.
  • The end user’s time zone is not considered when evaluating the Active Period date.

Example:

  • A Supplemental Task is created with an Active Period date of January 16, 2026. When the Active Period is reached at midnight (which corresponds to January 15, 2026, 4:00 PM UTC), a user located in a UTC+8 time zone does not see the Supplemental Task on the homepage, even though it is already January 16, 2026 in their local time.
  • After 8 hours, the same Supplemental Task is also not visible to a user located in the UTC time zone at midnight, even though the date is already January 16, 2026 according to UTC.

Environment

SAP SuccessFactors HCM Suite

Reproducing the Issue

  1. Access the "Manage Home Page" tool
  2. Under "Cards", go to "Supplemental Tasks"
  3. Click "Add Task"
  4. Select any category and keep the task enabled 
  5. Set the "Due Date" to the following day
  6. Set "Number of days to show pending tasks before they‘re due:" to 1 
  7. Note that the Active Period is now set to start on the current day
  8. Navigate back to the Home Page
  9. Observe that the Supplemental Task is not visible, even though the Active Period is set to start on the current day

Cause

  • In the backend, the Supplemental Task card is created with an incorrect Active Period value.
  • There is a known issue where the system uses the UTC time zone instead of the end user’s local time zone to determine when the task is displayed on the homepage.
  • Additionally, the system incorrectly evaluates the end of the day rather than the start of the day when calculating the Active Period.

Resolution

The fix for this issue is planned for the 1H 2026 release that is on 13th April (Preview) and 16th May (Production).

Workaround:
When configuring Supplemental Tasks, set the Active Period to start one day earlier than the intended start date. This adjustment ensures the task is displayed to end users on the expected day.
