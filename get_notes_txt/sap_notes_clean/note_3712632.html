Symptom

This article explains how to handle incorrectly completed I-9 processes, including how to correct errors, retrigger I-9 flows, and manage audit trail updates.

  • I-9 or E-Verify was incorrectly completed
  • Required steps were missed or completed in error

Environment

SAP SuccessFactors Onboarding

Resolution

Correcting I-9:

1. Trigger a New I-9 Flow:

  • Go to Manage Data > Create New > Trigger I-9 Flow
  • Enter employee’s name in User
  • Select Trigger Type (e.g., I-9 Reverification)
  • Enter Process ID if available (leave blank for ONB 1.0 migrated users as users do not have the ID)
  • Click Save

    A new task is assigned to the employee or I-9 Responsible users.

2. Handle Previous Task:

  • Go to Manage Data > Search for Compliance Process
  • Find the record using Compliance Process ID / User ID
  • From Take Action, select Make Correction
  • Change Compliance Process Status to Cancelled
  • Click Save

See Also

Other compliance tasks can be restarted using: Restart Compliance Process
