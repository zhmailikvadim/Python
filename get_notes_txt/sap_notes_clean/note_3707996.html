Symptom

In the Fiori App "My Purchase Requisitions - New" (Fiori ID: F1639A), entering text for one PR item automatically causes it to appear and update in other PR items.

Environment

  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. In the Fiori App F1639A, creating a PR which has multiple PR items.
  2. Changing the text in one item results in the text being synchronized across all items.

Cause

This is a system error due to the missing of some standard notes.

Resolution

Implement the following notes if they are missing from the system:

3620323 - Note text not cleared
3660000 - Navigation issue when not full screen
3652216 - BAdI MM_PUR_S4_PR_MODIFY_ITEM does not update on frontend
3640623 - F5 triggers Cross-Catalog Search (My Purchase Requisitions app)
