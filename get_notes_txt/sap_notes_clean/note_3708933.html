Symptom

  • When trying to create a meter reading order in EL28 an error is raised.
  • Message No. EL478: Device xxx not installed in installation yyy on zz.zz.zzzz.

Environment

  • SAP for Utilities as Part of SAP ERP, release IS-UT 600 and higher.
  • SAP S/4HANA, release IS-UT 801 and higher.

Reproducing the Issue

  1. Run transaction EL28.
  2. Attempt to create a Meter Reading Order.
  3. Observe the error message: "Device xxx not installed in installation yyy on zz.zz.zzzz" (Message No. EL478).
  4. Verify in transaction EL31 that a Meter Reading Order (MRO) already exists for the same date for the device.
  5. Repeat the process for a different date, e.g. 30.10.2025, and observe that the MRO is created successfully.
  6. Attempt to create another MRO for the same date, e.g. 30.10.2025, and observe the same error message: "Device xxx not installed in installation yyy on 30.10.2025" (Message No. EL478).

Cause

  • The method ADD_MR_REASON_IN_OPEN of the enhancement spot ISU_MR_INSTLN might be incorrectly programmed.
  • The issue occurs when a Meter Reading Reason (MRR) is selected in transaction EL28, and a Meter Reading Order (MRO) already exists for the same date for the device.

Resolution

  1. Verify if the method ADD_MR_REASON_IN_OPEN of the enhancement spot ISU_MR_INSTLN has an active implementation.
  2. Test the behavior by deactivating the implementation of the method ADD_MR_REASON_IN_OPEN.
  3. Confirm that the error message EL478 is no longer raised incorrectly.

Keywords

EL-478, EL 478, meter reading order
