Symptom

During purchase order or delivery creation, the plant is not yet determined. As a result, when we open the value help for the commodity code the search is not restricted by country and all commodity code records available in the system are loaded. In systems with a large number of commodity codes, this can lead to a memory dump.

Other Terms

VL02N, ME02N

Reason and Prerequisites

Program error

Solution

Implement the correction instructions added to the note.
