Symptom

  • Error occurs during the installation of Java Application Server of SAP NetWeaver 7.5 with SWPM on MaxDB.
  • Error message: "Error during execution of command identifier 'DB_ENUM' : com.sap.sdb.sllib.api.exceptions. Package JDBC is not installed!"

Environment

MaxDB database, SAP NetWeaver 7.5

Reproducing the Issue

  1. Start the installation of SAP NetWeaver 7.5 using SWPM on MaxDB.
  2. Select the option: (SAP NETWEAVER 7.5) > SAP NetWeaver 7.5 > MaxDB > Installation > Application Server Java > Standard System > Standard System.
  3. Proceed with the installation steps until the error occurs during the execution of the command identifier 'DB_ENUM'.
  4. Observe the error message: "Error during execution of command identifier 'DB_ENUM' : com.sap.sdb.sllib.api.exceptions. Package JDBC is not installed!"

Cause

The database instance (MaxDB) is not installed prior to the installation of the Java application server.

Resolution

  1. Install the MaxDB database instance with SWPM first before proceeding with the installation of the SAP NetWeaver 7.5 Java application server.
  2. Once the database instance is successfully installed, restart the installation process for the Java application server using SWPM.
