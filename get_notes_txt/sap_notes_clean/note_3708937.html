Symptom

You use Transaction Manager of Treasury and Risk management. You use investment funds in Brazil and report Month End Tax Calculation for Funds (J1BTRMFME). You want to run reversal of J1BTRMFME and empty screen without any message is displayed.

Other Terms

Investment Fund, Brazil Tax Calculation, J1BTRMFME, Reversal

Reason and Prerequisites

 The issue is caused by program error.

Solution

Implement the attached correction instructions and perform post implementation steps or import the relevant support package.
