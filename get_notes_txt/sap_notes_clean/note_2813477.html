Symptom

  • Setup SSO with Work manager with ECC 6.0 system as backend.
  • No tcode CERTRULE to add certificate.   
  • Is there any alternative for this?

Environment

  • SAP Work Manager 6.5
  • SAP ERP Central Component(SAP ECC) 6.0

Reproducing the Issue

1. run /nCERTRULE in backend system

2. error "Transaction CERTRULE does not exist"

Resolution

use RZ10  with  login/certificate_mapping_rulebased profile.
