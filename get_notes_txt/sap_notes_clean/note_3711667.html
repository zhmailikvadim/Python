Symptom

 When a contract contains multiple line items, the "Approve ACM Trading Contract" application displays the same 'Delivery Period From' and 'Delivery Period To' values across all rows in the list view, even when the line items have different delivery periods.

Other Terms

LO-AGR; Agricultural Contract Management; Approve ACM Trading Contract; Delivery From; Delivery To; F4379

Reason and Prerequisites

This issue occurs because the key field defined for the list display entity (List Screen Table) in the "Approve ACM Trading Contract" application is 'Contract Number'.
As a result, the Fiori UI framework overwrites non-key-related data with the latest record when multiple entries share the same key.

Solution

Import the relevant support pack or apply the correction instruction(s).
