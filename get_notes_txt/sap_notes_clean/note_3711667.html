Symptom

This SAP Note resolves the issue of GDD: Missing possibility to resend already send Objects with active switch DFS_GDD_DT.

During processing of the GDD report, an issue was identified, GDD does not allow data that has already been sent to be sent again to the same system. To address this, we created DFS_GNRC_DISTR_SUP transaction code, which has its own authorization object. This transaction provides the same options as the standard GDD transaction (DFS_GENERIC_DISTR), enhanced with an additional checkbox, “Resend Objects.” This checkbox allows users to explicitly control whether previously sent objects should be resent or not.

Other Terms

Defense Disconnected S4, GDD, Generic Data Distribution, DFS_GENERIC_DISTR, DFS_GNRC_DISTR_SUP

Solution

Implement the attached correction instructions provided with the NOTE.
