Symptom

Tendering Process set to "Freight RFQ Based, Award Automatically", but the Automatic Award does not work. After evaluation, the quotation remains in status ‘Evaluated, Award Pending’ instead of being awarded.

Other Terms

ACCEPT_RESP, /SCMTMS/AYS_POPUP 011, PROC_CONTROL, RFQ

Reason and Prerequisites

Program error.

Solution

Implement the attached correction instructions or install the corresponding Support Package.
