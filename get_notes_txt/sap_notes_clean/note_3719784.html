Symptom

How to view the change document for user privileges/role-assignment changes in IDM?

Environment

IDM 8.0

Resolution

Utilize the view idmv_bw_old_values_and_links for analysis.

Example query:
select * from idmv_bw_old_values_and_links where mskey = <affected_MSKEY_of_Role>

The view provides key columns to identify changes:

  • "userID" or "changedby" — indicates the user responsible for the change.
  • "aValue" — contains the MSKEY of the privilege.

Keywords

change document, IDM, role, privilege, idmv_bw_old_values_and_links
