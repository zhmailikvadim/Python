Symptom

You use API_INSPECTIONLOT_SRV to record results for an inspection point. There is a characteristics with control indicator "documentation required". The "InspectionResultText" is therefore required when recording result.

According to note 3503850 API API_INSPECTIONLOT_SRV doesn't trigger valuation on characteristic/inspection lot level (QAMR), so you are using additional PATCH request, but status in QAMR table is not updated to 5. You receive the following error: 'Documentation required, closing not possible for charac &1 of lot &2'

Other Terms

API_INSPECTIONLOT_SRV/A_InspectionCharacteristic(...)/to_InspSmplResult, API_INSPECTIONLOT_SRV/A_InspectionResult(...), QECOPY 050, InspResultIsDocumentationRqd, InspectionResultText, A_InspSampleResult, PRUEFBEMKT, QASR, QAMR 

Reason and Prerequisites

Programming error.

Solution

Implement the correction instructions.
