Symptom

An Active Return can be triggered more times for its Recipient Item, even for this Recipient Item there is already on Active Return order created.

Other Terms

/PE1/CL_EH_E_ACTIVE_RETURN->CHECK_ACTIVE_RETURN

/PE1/CL_EH_P_ACTIVE_RETURN->CREATE_ACTIVE_RETURN

Reason and Prerequisites

Program error.

This issue has been found when selecting 'Prioritize Active Return' for definition of Response Type of Active Return, but this fix is generic also for the normal Active Return process.

Solution

Before creating the Active Return, one more check is added, to check whether behind the Recipient Item already one existing Active Return created.

Please implement the provided Correction or the corresponding Support Package.
