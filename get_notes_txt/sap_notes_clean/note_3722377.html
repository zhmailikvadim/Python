Symptom

When executing Annex16 report (RAIDJP_ANNEX16), it's time-consuming for large amount of assets.

Other Terms

Annex16, RAIDJP_ANNEX16

Reason and Prerequisites

This is a performance issue. Sort mt_anka_t095 and Sort mt_ankp_t095 statement was called for each asset, which resulted to a performance issue.

This SAP Note is a continual correction of SAP Note 3342354.

Solution

Apply the corresponding support package or implement the correction instructions in this note.
