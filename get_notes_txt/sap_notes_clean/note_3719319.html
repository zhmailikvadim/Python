Symptom

  • In the Data Dictionary the tables (INST_ID2, INST_ID3) are present, However Unable to add these fields (INST_ID2, INST_ID3) to the Connector configuration
  • Changes made to the connector configuration are not saved after clicking "Apply Change".

Environment

SAP SuccessFactors Learning

Reproducing the Issue

  1. Navigate to Learning Administration.
  2. Go to System Administration > Configuration > System Configuration > Connectors.
  3. Attempt to add new settings (INST_ID2, INST_ID3) in the connector configuration under scheudleoffering connector configuration.
  4. Click "Apply Change".
  5. Observe that the system does not save the changes.

Cause

Expected behavior.

Resolution

  1. Verify that the data dictionary being checked is for database tables and columns, as it does not directly correlate to usable settings in connectors.
  2. Refer to the connector workbook for valid settings that can be updated in the connector configuration.
  3. Note that as per the Connector Workbook, the settings (INST_ID2, INST_ID3) cannot be updated to the specified connector.
