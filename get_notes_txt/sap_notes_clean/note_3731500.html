Symptom

  • In the generic withholding tax report (program RFIDYYWT), the ALV column for G/L account shows no values.
  • The issue is observed when running report S_P00_07000134.

Environment

  • SAP S/4HANA
  • SAP ECC
  • Withholding Tax (Reporting)

Reproducing the Issue

  1. Start report S_P00_07000134 (Generic Withholding Tax Reporting).
  2. Enter selection parameters
  3. Run the report and review the ALV output: the G/L account column is empty.

Cause

The G/L account field in the generic withholding tax report is not populated generically by design; it is used differently across countries. Although HKONT is available in table WITH_ITEM, it is not mapped to the ALV G/L field by default.

Resolution

  1. Refer to the KBA on adapting the report output: 3409397 RFIDYYWT - How to change the output data 
  2. Implement the user exit/enhancement described in the KBA for program RFIDYYWT to populate the ALV G/L account field using WITH_ITEM-HKONT in accordance with local reporting requirements.
  3. Test the change in a non-production system to confirm that the G/L account column displays the expected values.
