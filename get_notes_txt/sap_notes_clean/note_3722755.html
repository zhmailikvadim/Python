Symptom

The maximum number for allocation rules in the SAP S/4HANA Cloud Private Edition system has been expanded from three digits to seven digits (9999999). 
There is a concern about the maximum number of distribution rules.

Environment

  • Controlling (CO)
  • SAP S/4HANA
  • SAP S/4HANA Finance
  • SAP S/4HANA Cloud Private Edition
  • SAP Fiori for SAP S/4HANA

Reproducing the Issue

  1. Navigate to SAP Customizing Implementation Guide (IMG).
  2. Go to Controlling -> Internal Orders -> Actual Postings -> Settlement -> Maintain Settlement Profiles.
  3. Observe field TKB1V-MAXBR (Maximum Number of Distribution Rules) 's length is 7 digits .

Cause

Standard behavior

Resolution

Since S4/HANA OP1909 - 104, it is possible to maintain more distribution rules with 7 digits. The sequence number of distribution rule (LFDNR) in table COBRB is of length 7.

While on lower releases (S4 included), the maximum number of digits is 3. 

Keywords

Distribution rule, Settlement profile, MAXBR, COBRB, LFDNR
