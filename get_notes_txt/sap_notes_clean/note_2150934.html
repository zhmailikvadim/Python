Symptom

When you process an inbound IDoc that was initiated by the Belgian customs server PLDA, the system terminates the processing with the runtime error CONVT_OVERFLOW.

Other Terms

PLDA, /SAPSLL/CVRTQ, CONVT_OVERFLOW, CX_SY_CONVERSION_OVERFLOW, /SAPSLL/SAPLLEG_ECC_IP, /SAPSLL/LLEG_ECC_IPF12, PARSE_2_HD_AAR

Reason and Prerequisites

This problem is caused by a program error.

Solution

Implement the correction instructions that are relevant for your release.
