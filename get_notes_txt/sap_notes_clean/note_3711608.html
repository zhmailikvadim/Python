Symptom

When executing the BG_SAFT_MONTH report, there is a SAPSQL_DATA_LOSS dump in CL_SAFT_BG_TAX_TYPES.

Other Terms

BG, SAFT, BG_SAFT_MONTH

Reason and Prerequisites

This SAP Note is only relevant for Bulgaria.

Implement Note 3703066 before implementing this Note.

Solution

As a general rule SAP recommends that you implement the Support Package specified for your release. However, if you need to install a solution earlier, use the SAP Note Assistant to implement the correction instruction.
