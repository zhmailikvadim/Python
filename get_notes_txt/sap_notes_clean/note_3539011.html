Symptom

Symptom

In the Capacity Plan Simulation app when there's a space or special character in the Simulation ID then the In-app navigation fails

Other Terms

Capacity plan simulation, F3934

Reason and Prerequisites

Whenever a simulation ID contains special character then the encoding of the URL is needed.

Solution

Please install this SAP Note

Error: WAARDE ELEMENT ONTBREEKT

Reference: Item[1]/Commodity/GoodsMeasure/TariffQuantity

Original field value: Missing measurement units, additional unit DTNZ

Other Terms

NL, DMS, import, customs duties

Reason and Prerequisites

The error occurs, for example, for weight-based or complex customs duty rates for which a quantity must be reported in a special unit of measurement.

Solution

Implement the attached correction instruction.

If there is a quantity in a supplementary unit at item level, this is transferred if the unit of measurement corresponds to that from the customs duty rate.

If this is not the case, the quantity is transferred from the "Duty Quantity" field. The quantity must be specified in the special unit of measure of the customs duty rate.
