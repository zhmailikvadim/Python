Symptom

Wrong information is displayed in the header of XML files of Slovenia EC Sales List - Form PD-O and Form RP-O in case plants abroad is enabled.

Other Terms

SI_ECSL_RPO, SI_ECSL_PDO, SI, EC Sales List, DRC, Document and Reporting Compliance

Reason and Prerequisites

This SAP Note is relevant only for Slovenia.

This SAP Note is applicable only for SAP Document and Reporting Compliance.

Solution

As a general rule, SAP recommends that you install this solution by applying a Support Package. However, if you need to install it earlier, use the Note Assistant to implement the correction instructions.
