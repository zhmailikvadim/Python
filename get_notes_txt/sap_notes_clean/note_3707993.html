Symptom

Authorization Object W_WBRK_STS is not restricting user behavior in TCODE/FIORI app WZR2.

Environment

Condition Contract Management in SAP S/4HANA

Cause

Missing configuration.

Resolution

Go to transaction WLFA -> Basic Settings -> Specify extended settings for settlement management -> Authorization Check on Application Status Change and add a new entry. This check box should add an ACTVT = 43 to authorization object W_WBRK_STS.

Keywords

W_WBRK_STS, WZR2, ACTVT, 43, Settlement Document, Status, Release, Authorization
