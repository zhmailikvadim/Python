Symptom

This SAP Note contains the BC set for the table TNDCV and is relevant only in connection with SAP Note 3713998.

This covers the following topic:

Depending on the setting, you can create scoring data for a patient with outpatient cases for each movement/visit day (for all cases or for each outpatient case (day/case view)).

If a visit day falls on an admission or discharge day for an inpatient case  of the patient, the system does not create any scoring data for this outpatient movement day.

If necessary, no outpatient ELDA/P321 admission notification is created for this.
