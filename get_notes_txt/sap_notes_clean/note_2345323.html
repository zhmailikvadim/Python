Symptom

When the function EPMSaveComment is used in a report and an "Insert Members" is performed, the existing comments disappear after inserting members.

Other Terms

EPM add-in 10.0 SP26 Patch 01, resolved issues, FC (for Financial Consolidation), BPC (for Planning and Consolidation), SAP BW (INA Provider)

Solution

This has been corrected in this Support Package.

The sheet option "Perform Worksheet Refresh on Expand/Collapse and Insert Members" needs to be selected.

This option is not compatible with the user option "Refresh only Expanded/Inserted members".
