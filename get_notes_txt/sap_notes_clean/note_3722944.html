Symptom

  • SAP Note 3423115 - 'Singapore Withholding Tax: Addition of new fields - Legal change' has added fields to the ALV output.
  • Fields "PAYEE'S BUSINESS STRUCTURE"  is generated without values in the report from the Run Statutory Reports app and it is not clear where to maintain the data.

Environment

• Financial Accounting (FI)

• SAP S/4HANA Finance

• SAP S/4HANA

Reproducing the Issue

  1. Access the Fiori app - Run Statutory Reports.
  2. Choose Report Category - SG_WHT_IR37 and generate a new run.
  3. Open the output and observe that the field "PAYEE'S BUSINESS STRUCTURE" is empty.

Cause

The "PAYEE'S BUSINESS STRUCTURE" field is empty because the recipient type's description is not maintained.

Resolution

Update the "PAYEE'S BUSINESS STRUCTURE" information in the recipient type's description by maintaining the relevant configuration in the following SPRO path:

Financial Accounting → Financial Accounting Global Settings → Withholding Tax → Extended Withholding Tax → Basic Settings → Check Recipient Types.
