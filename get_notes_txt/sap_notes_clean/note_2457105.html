Symptom

You create an outbound delivery using  stock transport order. You do goods issue for the outbound delivery.

 Now you try to generate SPED output which creates an inbound delivery. You notice that inbound delivery is not created .

Other Terms

Vl31n , vl32n ,vl33n, SPED , outbound delivery , inbound delivery

Reason and Prerequisites

This is caused due to program error.

Solution

Implement the note 2457108 before implementing the current note.

After implementing the note 2457108 , implement all the correction instructions mentioned in the note
