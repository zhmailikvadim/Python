Symptom

  • Error encountered during the execution of the command: `Set-OrganizationAddInOverrides -ProductId 8a512e2b-c04e-4c06-9235-11b6cd59f584 -AppDomains "https://hcs.cloud.sap", "https://mycompany-dev.eu10.analytics.cloud.sap", "https://mycompany-qm.us10.sapanalytics.cloud", "https://mycompany-test.eu10.hanacloudservices.cloud.sap"`.
  • Error message: `FullyQualifiedErrorId: FailedWriteToExchange.Microsoft.OrganizationAddInPowerShell.Cmdlets.SetOrganizationAddInOverrides`.

Environment

SAP Analytics Cloud, add-in for Microsoft Office

Reproducing the Issue

  1. Open the PowerShell and connect to your Office365 tenant by running the command Connect-OrganizationAddInService.
  2. execute the command: `Set-OrganizationAddInOverrides -ProductId 8a512e2b-c04e-4c06-9235-11b6cd59f584 -AppDomains "https://hcs.cloud.sap", "https://mycompany-dev.eu10.analytics.cloud.sap", "https://mycompany-qm.us10.sapanalytics.cloud", "https://mycompany-test.eu10.hanacloudservices.cloud.sap"`.
  3. Observe the error message: `FullyQualifiedErrorId: FailedWriteToExchange.Microsoft.OrganizationAddInPowerShell.Cmdlets.SetOrganizationAddInOverrides`.

Cause

The error message is related to Microsoft Powershell.

Resolution

Contact Microsoft Support to investigate why 'Set-OrganizationAddInOverrides' command throws error message in Powershell.
