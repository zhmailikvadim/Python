Symptom

You notice that, after selecting a warranty while updating a Registered Product, the Warranty Start Date and Warranty End Date fields are populated, but the Warranty field remains blank / empty in the Registered Product header.

Environment

SAP Service Cloud Version 2.

Reproducing the Issue

  1. Logon to the SAP Service Cloud Version 2 tenant.
  2. Navigate to Registered Products work list.
  3. Open any record.
  4. Expand the header section.
  5. Click to edit the field Warranty.
  6. Select any active warranty in the pop-up.
  7. See the fields Warranty Start Date and Warranty End Date populate the data, but the field Warranty remains empty.

Cause

This may happen in case the Warranty has no description maintained for the logged in language, which causes it to be displayed as blank, when in reality it has been populated as expected.

Resolution

Ensure the description is maintained for all warranties in the language you are logged in, and upon selecting a warranty for a registered product, the field will then display the data properly.
