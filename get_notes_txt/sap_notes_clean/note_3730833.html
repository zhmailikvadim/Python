Symptom

  • Installed system details are not reflecting same in the SAP Support portal (Maintenance planner).
  • Discrepancy observed between the installed version of products in the Maintenance planner and the product version installed in the system.

Environment

  • Product: SAP S/4HANA
  • Maintenance Planning Tools

Cause

For some products, the same component levels are included in the different SPS level of the products. Maintenance planner calculates the higher SPS to ensure compatibility with the upgrade target. For example, the same components level is inlcuded from SAP FIORI FRONTEND SERVER 6.0 SPS01 to SPS05.

Resolution

Check the listed components level in he Maintenance Planner are same as the system compoennts level, and in such situation, the products version discrepancy will not impact the upgrade process.
