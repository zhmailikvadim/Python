Symptom

Order is created and it is showing in Operation bucket later we assigned /Maintenance event /Revision to the work order and now it is showing in both operational and event based planning bucket. 

Environment

  • SAP ERP Plant Maintenance (PM)
  • SAP S/4HANA 2021, on-premise 
  • SAP S/4HANA 2022, on-premise
  • SAP S/4HANA 2023, on-premise
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Create a maintenance order.
  2. Open app F3888 or F3888A.
  3. Select one of the operational bucket and find the order available in it.
  4. Assign maintenance event to the order header.
  5. Then this order is also available in the event-based bucket.

Cause

Expected behavior.

Resolution

Operational buckets are available to give overall availability of orders and its respective components.

For event based buckets orders will only be displayed when an order is assigned to Event. For operational buckets, orders will be displayed just based on the scope defined for the bucket.
