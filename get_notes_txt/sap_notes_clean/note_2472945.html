Symptom

When using Custom Analytical Queries, only 5 Queries are shown.

Environment

NetWeaver 7.50

Cause

This is expected behavior.

Resolution

Custom Analytical Queries App does not show all the SAP Queries, it shows only SAP Released (Or Allowlisted) Analytical Queries and that's why few Queries are shown.

The following URL: http://www.sap.com/documents/2015/07/2ad59b27-347c-0010-82c7-eda71af511fa.html
