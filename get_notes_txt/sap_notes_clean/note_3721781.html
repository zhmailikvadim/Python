Symptom

When you run transaction /AIF/CONTENT_COMP to compress multiple interfaces during deployment, translation entries (for example, interface description) are written to the transport request for only one interface. As a result, descriptions for the other interfaces are not extracted in the target systems.

Other Terms

/AIF/CONTENT_COMP, Text Table, Interface Description

Reason and Prerequisites

This is a programming error.

Solution

Implement the attached correction instructions or corresponding support package.
