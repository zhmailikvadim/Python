Symptom

  • A technical error occurs while submitting a form.
  • Error message: "A technical error occurred."

Environment

SAP Marketing Cloud Edition

Reproducing the Issue

  1. Navigate to the form embedded in your external website.
  2. Fill and attempt to submit the form.
  3. Observe the error message: "A technical error occurred."

Cause

The SAP backend is not sending the required CORS headers to allow requests from the production domain.

Resolution

  • URL settings for the SAP Cloud Integration Service may not be configured correctly.
  • To resolve this, you will need to adjust the settings in accordance with our documentation for externally hosted Landing Pages.
  • For detailed guidance on how to align the settings properly, please refer to the documentation available on the SAP Help Portal:
    SAP Marketing Cloud Documentation.
