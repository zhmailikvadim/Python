Symptom

When accessing Work Zone on a mobile browser and navigating to the "View All Workspaces" page, the left and right arrow navigation buttons on the "Favorites" carousel are not functioning properly.

Environment

SAP Work Zone Mobile

Reproducing the Issue

  1. Go to Work Zone using a mobile browser.
  2. Click on "Workspaces" > "View All Workspaces".
  3. Check the "Favorites" section.
  4. Observe that the left and right arrow navigation buttons on the "Favorites" carousel do not work.

Cause

This is a known issue. The carousel does not activate because the number of favorites displayed in the mobile view is lower than the threshold required to trigger the "next page" function.

Resolution

Our engineering team has confirmed that the root cause of this issue is that the mobile view displays fewer Favorites when the page loads. They plan to fix this issue to properly support mobile devices.

Please set this KBA as a favorite (star icon on the top right corner) to receive updates on the fix. Refer to KBA 2171560 to understand how to get notified when a KBA is created/updated.
