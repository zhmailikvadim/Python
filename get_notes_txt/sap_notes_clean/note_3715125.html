Symptom

There is an already connected Quality SAP Integration Suite (API Management) tenant to the PROD Developer Hub. 

Environment

SAP Integration Suite – API Management
SAP BTP Cockpit

Reproducing the Issue

Now want to disconnect the PROD Developer Hub from the Quality System Integration Suite and instead connect the Quality System Developer Hub to the Quality System Integration Suite. 

Cause

 This change is currently not supported.

Resolution

SAP Integration Suite does not support disconnecting API Management from an existing Developer Hub. 
Once a connection is established, it cannot be removed to create a new connection to a different centralized Developer Hub.
Please follow the help.
