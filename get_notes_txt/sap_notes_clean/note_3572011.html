Symptom

If a related date resulting from a CIC/IIS notification is scheduled after the "Change B/M Data" business transaction has been executed, this date should have the fine control value 2. However, if the message is included immediately and other business transactions are executed in addition to the "Change B/M Data" business transaction, the system does not determine this fine control value correctly.

Other Terms

Auto line of business

Time model

Reason and Prerequisites

Program error

Solution

To correct the problem in advance, implement the source code corrections.
