Symptom

It is required to transport the Financial Statement Version created in Manage Global Hierarchies app to Test/ Production system. This article provides a guide on how to achieve this. 

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental." 

Environment

SAP S/4HANA Cloud Public edition. 

Resolution

Use the Export/Import functionality, this ensures the structure is identical across all tenants.

Exporting an FSV

  1. Open the ‘Manage Global Hierarchies’ app.
  2. Find and select the FSV need to be transported. 
  3. Export to Spreadsheet:
        ○ Use the ‘Export/Import’ option.
        ○ Choose ‘Export to Spreadsheet’. 
        ○ This will download the FSV details to a spreadsheet file on your local device.

Importing an FSV

  1. Access the desired system to import the FSV, such as a TEST system.
  2. Open the ‘Manage Global Hierarchies’ App
  3.  Import from Spreadsheet:
        ○ Use the ‘Export/Import’ option.
        ○ Select ‘Import from Spreadsheet’.
  4. Upload the spreadsheet file you exported earlier.

For more information, kindly refer to Manage Global Hierarchies | SAP Help Portal and Edit in Spreadsheet | SAP Help Portal
