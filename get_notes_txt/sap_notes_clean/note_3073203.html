2. Enhancements and Legal Changes

ERiC component for E-Bilanz reporting has been updated to version 33.4.4.0
The E-BILANZ PLUG-IN 10.1 of DM Stack 1802 contains ERiC version 33.4.4.0. With this version you can submit productive filings with E-Bilanz taxonomy 6.4.
