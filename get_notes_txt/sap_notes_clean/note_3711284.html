Symptom

In the 'Post Incoming Payments' and 'Post Outgoing Payments' apps, the system displays the warning message 'Partial payment amount greater than net amount' Message No.: F5590. Even after maintaining message control settings in customizing to change this warning message to an error, the system continues to display it as a warning.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  • Go to Customizing IMG Activities
    Financial Accounting --> Financial Accounting Global Settings --> Tools --> Change Message Control
  • Locate message F5590 using the application area and message number from the technical information.
  • Change the message type from Warning to Error.
  • Save the configuration.
  • Open the Post Incoming Payments or Post Outgoing Payments app.
  • Enter a partial payment amount greater than the net amount.

The message still appears as a warning and does not stop posting.

Cause

In SAP S/4HANA Cloud Public Edition, message control customization is restricted for certain accounting and fiori app specific messages. 

Resolution

The functionality to change the message control for accounting-related messages such as F5590 is not supported in SAP S/4HANA Cloud Public Edition. Therefore, the configuration  maintained to change the warning message F5590 to an error message will not affect the system behavior. SAP recommends submitting a Feature Request via the SAP Customer Influence portal.
