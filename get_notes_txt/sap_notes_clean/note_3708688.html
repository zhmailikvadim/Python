Symptom

  • An error occurs when attempting to call the API endpoint https://XXXXXXXX.eu20.alm.cloud.sap/api/calm-documents/v1/Documents, resulting in a "401 Unauthorized" response.
  • Connection with SAP Analytics Cloud (SAC) cannot be established.
  • The issue involves scopes calm-api.documents.read, calm-api.documents.write, and calm-api.documents.delete in the SAP Cloud ALM API service instance created in BTP.

Environment

  • SAP Analytics Cloud 2026.2.1 (Server) 
  • SAP Analytics Cloud 2026.2.2 (Client)

Reproducing the Issue

  1. Create a "SAP Cloud ALM API" service instance in BTP with calm-api.documents.read, calm-api.documents.write, and calm-api.documents.delete scopes.
  2. Attempt to call the API endpoint https://XXXXXXXX.cloud.sap/api/calm-documents/v1/Documents.
  3. Observe the "401 Unauthorized" error response and inability to establish a connection with SAC.
  4. The issue does not happen to postman.
