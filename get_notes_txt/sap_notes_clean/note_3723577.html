Symptom

You need to output Tax Invoice PDF forms and Universal Transfer Document (UTD) PDF forms that comply with the letter № СД-4-3/11730@ of the Federal Tax Service of Russia, dated December 26, 2025. The forms are output using the old output management.

This change impacts the following PDF forms:

  • J_3RINVOICE_2021
  • J_3RINVOICE_RNPT
  • J_3R_UTD_2021
  • J_3R_UTD_RNPT

After you implemented this SAP Note:

  • The system adds a new line "5b" to the PDF forms. This line is populated with the number and date of the invoice issued upon receipt of the down payment when the current invoice is issued against that down payment.
  • The system extracts this information from the "External Number" and "External Date" fields of the down payment.

Other Terms

Line 5b, Tax Invoice PDF Form, RU, Russia, J_3RINVOICE_2021, J_3RINVOICE_RNPT, J_3R_UTD_2021, J_3R_UTD_RNPT, J_3RF_INV_A, UTD, Universal Transfer Document

Reason and Prerequisites

Legal Requirement.

This SAP Note provides you with the relevant system changes to address the legal change: 

Invoice and UTD (printout form) with rate 22%. Recommended formats from January 1,2026

Solution

Install the appropriate Support Package mentioned in this SAP Note or implement the attached correction instructions with manual post-implementation steps.

For more information about the development delivered with this SAP Note, see the What's New topic on SAP Help Portal: Updates to Tax Invoice and Universal Transfer Document Printing Forms (2026)
