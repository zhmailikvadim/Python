Symptom

You execute one or more master data consistency checks in transaction RSRV for the time characteristic 0FISCPER. The checks fail with the following error:

  • Syntax error in GP_ERR_RSDMDTMPLCHECK, row 1,031 (-> long text)
  • Characteristic 0FISCPER: Error when generating check program with template RSDMD_CHECKINC
  • Unknown column name "FISCVARNT"

Solution

The master data checks are incorrectly enabled for time characteristics. Following the implementation of this SAP Note, time characteristics are not permitted for the master data consistency checks.

As an alternative to implementing this SAP Note, you can import the following Support Package:

 

  • SAP BW 7.50
    Import Support Package 5 for SAP BW 7.50 (SAPK-75005INSAPBW) into your BW system. The Support Package is available once SAP Note 2318815 ("SAPBWNews NW BW 7.50 ABAP SP5"), which describes this SP in more detail, has been released for customers.

 

In urgent cases, you can implement the correction instructions as an advance correction.

You must first read SAP Notes 1668882 and 2248091, which provide information about transaction SNOTE.

To provide information in advance, the SAP Note(s) specified above might already be available before the Support Package is released. In this case, however, the short text of the SAP Note still contains the term "preliminary version".
