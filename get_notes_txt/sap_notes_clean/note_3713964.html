Symptom

Component: App Support
Module: App Inspector

You use the App Inspector app on the SAP Fiori Launchpad. When you select the link in the SAP Fiori ID column, the system opens the SAP Fiori Apps Reference Library but displays an error message.

Other Terms

FAL

Reason and Prerequisites

The symptom occurs due to a program error.

Prerequisites are available in the relevant correction instructions.

Solution

After implementing this SAP Note, the system uses the correct link to the SAP Fiori Apps Reference Library.

Alternatively, import either of the corresponding Support Packages for the component SAP_UI or the Feature Packages Stacks for SAP S/4HANA.
