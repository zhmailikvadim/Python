Symptom

 In the process "New Installation (SUP View)" (DE_B_GPKE_NIN_SUP), the process key date (proc_date) is deleted after receiving a rejection in step 900.This issue is relevant for the market role: SUP.

Other Terms

 GPKE, New Installation, Contract Start Date, proc_date, Rejection, Step 900, DE_B_GPKE_NIN_SUP

Reason and Prerequisites

 The error occurred because in step 900, the contract start date field was empty, and the system attempted to change the key date to match the contract start date, resulting in the deletion of the process key date.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the system includes a check to verify whether the contract start date is present before attempting to update the process key date. This prevents the unintended deletion of the proc_date field. You can verify the fix by checking that the proc_date field is not deleted after receiving a rejection in the new installation process.
