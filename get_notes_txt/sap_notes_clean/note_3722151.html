Symptom

Is it possible to hide or disable the RFQ Edit Request button using API flow in Guided Buying?

Environment

Ariba Guided Buying

Cause

As per design, it is not possible to hide the Edit Request button in Guided Buying when the following parameters are enabled.

PARAM_ENABLE_EDIT_RFQ
PARAM_ENABLE_ENHANCED_RFQ_INTEGRATION 

For the RFQ API integration, these parameters are mandatory. 

Resolution

Submit an Enhancement Request (How do I create an Enhancement Request?).

Keywords

RFQ, Guided Buying, RFQ Edit Request, API Flow, Hide Edit Request, Disable Edit Request
