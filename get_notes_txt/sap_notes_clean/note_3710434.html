Symptom

  • Error encountered while importing table <table name> from S/4HANA system into the datastore in SAP Data Services Designer.
  • Error while importing table with special data types: Error message: "Cannot import the metadata table . Parser detected an unknown data type".

Environment

SAP Data Services

Reproducing the Issue

  1. Log on to SAP Data Services Designer.
  2. Connect to the repository.
  3. Right-click on the datastore.
  4. Select "Import By Name" and enter the table name <>.
  5. Observe the error message: "Cannot import the metadata table . Parser detected an unknown data type".

Cause

Some data types in the table are not supported by the SAP Data Services engine and are not convertible.

Resolution

There is not any solution, as some tables are not directly usable in DS.

Workarounds:

  • Create a CDS view upon the problematic table and import the CDS view instead.
  • Create a custom 'Z' view to store the data and import it into the Designer. This step needs converting the data first into storable data types: Data type mapping for BW/4HANA | SAP Help Portal
