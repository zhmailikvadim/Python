Symptom

You change the posting date in an A/P or A/R document that can be paid, and you receive the following system message:

Newly entered posting date relates to another posting period. Do you want to update G/L accounts accordingly?

Environment

  • SAP Business One
  • SAP Business One, version for SAP HANA

Reproducing the Issue

  1. Create an A/P or A/R document that can be paid, as an example, an A/R Invoice. 
  2. Change the posting date to a date in a different posting period. Tab out of the posting date field. 
  3. A pop up System Message window appears with the mentioned system message. 

Cause

G/L Account Determination is done by period. A posting date change can change the posting period of a document. 

Resolution

The system message advices that by clicking "Yes", you will update relevant G/L Accounts in the document. 

You can choose "No", no change will be made.
