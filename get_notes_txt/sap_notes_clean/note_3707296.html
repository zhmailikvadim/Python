Symptom

In App Manage Validation/Substitution Rules for Journal Entries (F4406) the cost center cannot be substituted with blank value and the profit center cannot be replaced with another value.

Environment

SAP S/4HANA Public Cloud Edition 2602

Reproducing the Issue

  1. Open App Manage Validation/Substitution Rules for Journal Entries (F4406)
  2. Add parameters in General Information and Precondition
  3. In the Substitution tab try to set operator for Cost Center 'is empty' OR Set operator Profit Center "equal to' xxxxxxxx but target fields are not shown.

Cause

Cost Center

In the App Manage Validation/Substitution Rules for Journal Entries (F4406) the field CostCenter is configured as not being overwritable. The value is only set if the field has not been set yet and thus contains an initial value. Consequently, it is not possible to clear the field.

Profit Center

There is no single point to substitute the profit center available.

The profit center is for example substitutable for service documents (see Manage Substitution/Validation Rules - Service Documents), but not in such a generic use case.

Resolution

Cost Center:

In the enhancement spot FIN_ACDOC_SUBSTITUTION, BAdI interface IF_FIN_COBL_SUBSTITUTION, the field cost center is a changeable field which can be set back to its initial value.

Profit Center: 

This field cannot be substituted in this App.
