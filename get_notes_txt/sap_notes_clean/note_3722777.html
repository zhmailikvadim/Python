Symptom

Despite disabling the [Enter] function key in the configuration, pressing the physical Enter key on the keyboard still triggers the function.

Environment

  • SAP S/4HANA
  • SAP Extended Warehouse Management (SCM-EWM)
  • SAP Extended Warehouse Management Radio Frequency (SCM-EWM-RF)

Reproducing the Issue

  1. Open the transaction code /SCWM/RFUI.
  2. Log into RFUI (Warehouse Number, Resource and Device).
  3. Follow the menu path: Outbound delivery -> Packing -> Manually Repack. Note that this behavior may also occur in other RF processes. 
  4. Press the physical Enter key on the keyboard.

Cause

  • The Enter key is a default navigation tool in dialog/module pool programming, automatically invoking the corresponding PAI function module when pressed.
  • Configuration settings allow disabling certain keys/buttons, but the Enter key cannot be disabled due to its essential role in standard design. 

Resolution

The Enter key is a default navigation tool in dialog/module pool programming and cannot be disabled because it is essential for navigating between screens and performing actions such as verifying the destination bin or progressing to the next screen.
