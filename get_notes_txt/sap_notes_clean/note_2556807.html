Symptom

You are migrating your Tax Groups to Access Sequences. The migration program gives you errors like these:

  • Access sequence , application V does not exist

  • Access sequence , application TX does not exist

Other Terms

Brazil, J_1B_EXT_ACC_SEQ, migration, TAXBRA, CBT, J_1BCONDMAPV

Reason and Prerequisites

  • This note is relevant only for Brazil.
  • You have implemented SAP Note 2245489.

Solution

If the internal code ICMSNONCONTRIBUTOR is assigned to any Tax Group (like ICMS) in view J_1BCONDMAPV, errors will occur during the migration. Please execute the manual steps in the attached document to make sure that the ICMSNONCONTRIBUTOR internal code is properly configured.

  • As a general rule, SAP recommends that you install a solution by applying a Support Package. However, if you need to install a solution earlier, use the Note Assistant to implement the correction instruction. You can find more information about the Note Assistant in SAP Service Marketplace, under service.sap.com/note-assistant.
  • Note that documentation, transactions, and Customizing activities are delivered via Support Package only.
  • The description of the solution is in the attached document Manual Instructions for SAP Note 2554394.pdf. To implement the solution, you must carry out manual activities (see attachment). This note does not have correction instructions, only manual activities.

List of affected objects
