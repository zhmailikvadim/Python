Symptom

  • User restrictions are not working when running a job template for SAP Cloud Integration for data services.
  • Users from one plant are able to upload data for out of their permission scope

Environment

  • SAP Integrated Business Planning
  • SAP Cloud Integration for Data Services

Cause

  • Data integration jobs will be considered done by the configured integration user and not by the user that runs the application job

Resolution

Control the user permission with planning filters restrictions and job templates restrictions to avoid the user to run an unwanted integration job.
