Symptom

In SAP CPI, a createIf function has been added to an IFlow mapping. When the scenario is run, the following error occurs:

com.sap.aii.mappingtool.tf7.MessageMappingException: Runtime exception during processing target field mapping /<IFlow Name>/<Mapping Name>/<Header Name>. The root message is: Exception:[com.sap.aii.mappingtool.tf7.rt.BehaviorInvocationException: Cannot cast "" to boolean.]

Environment

SAP Integration Suite

Reproducing the Issue

Create an Iflow mapping that utilizes a createIf function.

Cause

The createIf function will create an empty parameter if the required input is provided. This will cause the boolean function to fail, as it expects either a "TRUE" or "FALSE" value.

Resolution

The scenario will require an AND function in order to work.

This should be added after the createIf function, with whatever should occur in the event of the required input being provided.

See Also

3653876 - Usage of AND Standard Function in SAP Cloud Integration Message Mapping
