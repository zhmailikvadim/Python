Symptom

In tcode s_p99_41000062, the value/periodic unit price is displayed as zero, which is incorrect.

Environment

  • SAP S/4HANA

Reproducing the Issue

  1. Run transaction code S_P99_41000062.
  2. Check the value/periodic unit price field which displays zero value.

Cause

Program error

Resolution

  1. Implement the following SAP Notes:
  • 0003472283
  • 0003595796

     2. Reverse the settlement step of in tcode CKMLCP

     3. Execute the settlement again. The value based on periodic unit price will be updated correctly.
