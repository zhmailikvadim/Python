Symptom

You want to use the mass processing function to convert 210 planned orders into collective orders. In the function parameters, you choose "Process Only Leading Orders". As a result, the system only converts collective orders for which the leading order corresponds to the selection conditions.

The system selects the planned orders correctly. However, the conversion does not take place.
