Symptom

You are using Real-Time Integration (RTI) from SAP ECC EHP4 or EHP5 to SAP IBP. These versions are not supported anymore by RTI.

If the serializer in the SM59 connection to the SAP IBP system in the 'Special Options' tab is set to other value than the supported 'Fast serializer', then the planned order and production order integration to the SAP IBP system might fail or results in unexpected data in the component's batch and other fields starting from the IBP version 2602.

Please note: Do not install the note before your SAP IBP system is upgraded to version 2602.

Other Terms

Classic serializer, /IBP/CIF_ORDER_INBOUND, CIF_ORDER_INP_IBP, CIFBATCH

Reason and Prerequisites

The RTI integration is set up in a non-recommended way

Solution

Implement the note to change the component's batch field lenght to 20 from 10, to make it compatible with SAP IBP.
