Symptom

A custom query created for Tickets does not appear in the query dropdown for specific users/roles when creating a homepage quick link card.

Environment

SAP Cloud for Customer

Reproducing the Issue

  1. Go to Homepage
  2. Create a "Quick Link" tile
  3. Select Tickets Object
  4. In the "Query" dropdown, the required custom query is not available

Cause

Different user roles access different Ticket work center views, one is Service Work Center -> Tickets work center view; another one is Tickets Work Center. Need to confirm which view user/role assigned and if the custom query visible in this view.

Resolution

Go to Work Center and View Assignment, compare the work center assigned for working case and non-working case. Confirm the assigned work center view included the custom query.
