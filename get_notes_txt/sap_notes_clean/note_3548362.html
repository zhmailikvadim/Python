Symptom

You want to exchange electronic documents for Mexico and need to customize the Application Interface Framework (AIF).

You are in version 108 and have issues when uploading and trying to activate BC Sets.

Other Terms

Mexico, Electronic Document Processing for Mexico, eDocument, eDoc, eInvoicing, e-Invoicing, electronic invoice, electronic invoicing, e-document, e-Document, e-Docs, e-Doc, e-docs, e-doc, Document Compliance, DC, Document and Reporting Compliance, electronic document processing, business configuration, customizing

Reason and Prerequisites

Reason: 

This SAP Note is relevant only for Mexico.

This is a central source updated with BC Set content delivered in previous SAP Notes in the table below.

Prerequisite: 

You have implemented the Electronic Document Processing Framework in your system. Refer to the implementation instructions provided in the Help Portal for SAP ERP or SAP S/4HANA.

You have implemented the full Mexico solution as indicated in 2526771 (SAP_APPL) or 2565791 (S4CORE)

Solution

Perform the following steps for each set in the table.

  1. Download the zip file with all the .bcs files according to your system version in the Attachments section of this SAP Note.
    • The file SAP_APPL.zip corresponds to all SAP_APPL versions.
    • The file S4CORE 100-107.zip corresponds to S4CORE from version 100 to 107.
    • The file S4CORE 108-higher.zip corresponds to S4CORE starting from version 108.
  2. Upload the BC Set to your development system using the SCPR3 transaction. For more information, see the BC Set Upload section in BC Set Editing Functions.
  3. Activate the BC Set. For more information, see Activate BC Sets on the SAP Help Portal.
  4. Perform the manual steps described in the document attached to the original SAP Note for the SAP Notes marked with a * symbol.
