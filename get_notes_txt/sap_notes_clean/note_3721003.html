Symptom

  • I do not see the Maximum duration for a Requisition, Work Order, and Work Order Revision (Days) field in the Job Posting Template preview.
  • The field appears correctly when I edit the template.

Cause

  • The Job Posting Template was created or updated through upload with the maximum duration value set to 0.
  • A value of 0 effectively disables the duration rule for that template, which suppresses the field from displaying.
  • When editing the template in the UI, the system automatically defaults a value from the Contingent Type if the stored template value is 0, creating the impression that a duration was originally configured.

Resolution

If the Maximum duration field does not appear in the preview, the template’s stored duration value is 0.
To activate the duration rule and make the field visible, you update the template so the duration value is greater than 0.

Follow these steps to correct the template:

  1. Sign into your SAP Fieldglass account
  2. Click Admin Configuration (gear icon) > Download Data
  3. Select the Job Posting Template Download option
  4. Enter ALL for the download parameter > Click Download
  5. Open the file and locate the column: Set maximum duration for a Requisition, Work Order, and Work Order Revision (Days)
  6. Replace any 0 values with the correct duration value.
  7. Save the file.
  8. Return to Admin Configuration (gear icon) > Upload Data
  9. Click Choose File, select the updated file, and click Upload
Optional
If you want future templates or template edits to inherit a consistent duration value, lock the duration field on the Contingent Type.
When locked, uploads that attempt to set a different value (including 0) fail with a validation message indicating the field cannot be changed.
