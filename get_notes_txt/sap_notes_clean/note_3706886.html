Symptom

The below information messages can appear when selecting adaptation changes in a a transport request:

  • Blocked layout change of type ChangeMashupStatus was excluded.
  • Blocked layout change of type ConfigMashupURL was excluded.
  • Blocked layout change of type AddRemoveAssignedObject was excluded.
  • Blocked layout change of type AddRemoveWocView was excluded.

Environment

SAP Cloud for Customer

Reproducing the Issue

  1. In the Available Transport Objects table of the transport request, click Select button.
  2. Select the Adaptation Changes Transport Object Type and click OK.

Cause

These info messages appear because certain adaptation changes are not transported, as they are not relevant in this context and will be ignored for a plain adaptation changes transport.

Resolution

Please review when each layout change type will be transported:

ChangeMashupStatus: Will be transported if the linked mashup will be transported
ConfigMashupURL: Will be transported if the linked mashup will be transported
AddRemoveAssignedObject: Will be transported if the linked offline print template will be transported
AddRemoveWocView: Will be transported if the linked custom workcenter will be transported
