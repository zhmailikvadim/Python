Symptom

When you revert a process with an open non-remote BPEM case, the system does not close the BPEM case automatically. The case remains active. This issue also occurs with other similar actions that should close BPEM cases.

Other Terms

Close BPEM cases, Process Document

Reason and Prerequisites

Program Error

Solution

Implement the Correction Instructions below.

After implementing this SAP Note, the system closes BPEM cases automatically when you revert a process or perform similar actions. The correction addresses a program error in /US4G/IF_BPEM_CLOSE_POST_ACT.
