Symptom

When you perform an LAW consolidation, the system displays users who are no longer valid in the component system.

Other Terms

USMM

Reason and Prerequisites

The actual error is located in the result file of component systems that use the SAP_BASIS release with the Support Packages SAPK-75004INSAPBASIS and SAPK-75003INSAPBASIS. If multi-client or multi-system users exist in this system, the referenced, chargeable users are also written to the result file, even though the multi-client or multi-system users are no longer valid.

Solution

Implement the correction instructions or import the relevant Support Package.
