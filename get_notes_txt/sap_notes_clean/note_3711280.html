Symptom

The class /US4G/CL_IM_BADI_CONFIG_ACC is defined as final and cannot be redefined.

This issue is relevant for the market roles SUP, DSO, and MOS.

Other Terms

/US4G/CL_IM_BADI_CONFIG_ACC , MCM

Reason and Prerequisites

The error occurred because of a programming error. The class was incorrectly marked as final, which prevents redefinition.

Solution

Implement the corrections in this SAP Note or import the relevant support package.

After implementing this SAP Note, the system removes the "Final" flag from class /US4G/CL_IM_BADI_CONFIG_ACC, allowing the BAdI to be redefined.

Retry redefining the BAdI of class /US4G/CL_IM_BADI_CONFIG_ACC. You can verify the fix by checking that the BAdI of class /US4G/CL_IM_BADI_CONFIG_ACC is redefinable.
