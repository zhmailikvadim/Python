Symptom

  • When a Web Intelligence (Webi) report is opened via hyperlink passing all the required parameters, the report displays as blank for some users.
  • Prompts are passed from one report to another when opened through hyperlink, but the table does not display the data on the report.

Environment

  • SAP BusinessObjects Business Intelligence platform 4.3
  • Web Intelligence

Reproducing the Issue

  1. Open a Web Intelligence (Webi) report.
  2. Refresh the report.
  3. Click on the cell containing a hyperlink to another document.
  4. The document opens in a new window but does not display any data for some users.

Cause

  • The issue is specific to a single report following its migration, potentially due to corruption during the migration process.

Resolution

After recreating the parent report (affected), which was migrated, no issue was observed thereafter.
