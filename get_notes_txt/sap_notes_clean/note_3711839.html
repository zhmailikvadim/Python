Symptom

You execute the Trade Funds Budget Workbench using transaction /IRM/IPTFBM and enter the fund type, sales organization, scenario, validity dates, and fund center. After executing the transaction, when you change the periodicity of the flexible period, you notice that the period description is truncated and the last digit of the year is missing.

This behavior is incorrect. This SAP Note corrects the issue to ensure that the period description is displayed correctly.

Other Terms

/IRM/CL_IPTF_DATA, IPTFBM, Period, Description, /IRM/IPTFBM, Flexible group, Truncated, Truncation

Reason and Prerequisites

 This is caused by program error.

Solution

Please implement the correction instructions. These corrections are only valid if you have applied the following Support Packs.

Release V/4 – 2022 V2, 2023, 2023 V2, 2023 V3 and Support Pack 1

Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object
