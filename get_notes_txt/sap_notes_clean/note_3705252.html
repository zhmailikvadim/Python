Symptom

- Transport request selection is empty/read-only when clicking "Publish"
- Http Trace: "NO_TRANSPORTS" returned from request to https://sapsrvwfd.nbgdmzqa.gr/sap/bc/lrep/actions/gettransports/
- Unable to proceed with adaptation publishing due to missing transport request

Cause

- A transport request must be created in the front-end system before the adaptation can be selected from the transport menu.

Resolution

1. Create a transport request in the front-end system.
2. Ensure the transport request is available and selectable from the transport menu.
3. Refer to the documentation for detailed steps: [ Transporting Your Adaptations | SAP Help Portal ]

Keywords

Transport request, Adaptation publishing, NO_TRANSPORTS error, SAP NetWeaver
