Symptom

When creating an Offer Letter for a candidate an error occurs:

errorId=***********, fingerprint=456b28ad0f10109d95a04834a633c728b8dcb681, timestamp=*******, errorMessage=An application error occurred. Please try again later, or notify support (with the error information of errorID=**********, timestamp=*******, and the version information from the current screen) if the issue keeps occurring., server=******

Environment

SAP SuccessFactors Recruiting Management

Reproducing the Issue

  1. Go to Recruiting
  2. Job Requisitions >> Candidates
  3. Select a candidate >> Take action
  4. Error occurs: errorId=**********, fingerprint=456b28ad0f10109d95a04834a633c728b8dcb681, timestamp=*******, errorMessage=An application error occurred. Please try again later, or notify support (with the error information of errorID=**********, timestamp=*******, and the version information from the current screen) if the issue keeps occurring., server=******

Cause

The internal logs clearly show that the business rule configured to preselect the offer letter template is failing to set the offer letter template due to incomplete business rule configuration.

Resolution

The issue stems from an incomplete configuration maintained for the business rule to preselect the offer letter template. Please follow the steps outlined in the KBA  3455683 - Preselect Offer Letter Templates based on Rules - SAP for Me for the complete rule setup.

See Also
