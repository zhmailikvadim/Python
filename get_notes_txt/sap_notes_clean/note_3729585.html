Symptom

  • The "Jump To" feature from the Total line is not functioning when values are passed in a range.
  • Error message displayed: "There was an error while trying to open the data source. Please choose another data source."
  • Passing the same value as a member works without any issue.
  • Passing the same value as a range from Data Analyzer works without any issue.

Environment

  • SAP Analytics Cloud (Enterprise)
 

Reproducing the Issue

  1. Log in to the SAC tenant.
  2. Open the Story and input the parameter "xxx" with values ranging from `xxx` to `xxx`.
  3. Click "Set".
  4. From the "Grand Total line", use the "Jump To" feature for "xxx".
  5. Observe the error: "There was an error while trying to open the data source. Please choose another data source."
