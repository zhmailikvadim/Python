Symptom

Is it possible to use the Consent Statement created from Customer Data Cloud (CDC) on the Recruiting career site?

Environment

  • SAP SuccessFactors Recruiting Management
  • SAP Customer Data Cloud

Reproducing the Issue

For Customer Data Cloud:

  1. Go to Customer Data Cloud console > Data Privacy > Consent Statements.
  2. Edit the ID "terms.RCM_EXTERNAL_CAND".
  3. Add a new Localization Template and set it as Default.

For Recruiting Career Site:

  1. Access the Recruiting career site.
  2. Create a new account.
  3. Select a Country/Region of Residence.
  4. Click on "Read and acknowledge the data privacy statement".
  5. Observe that nothing appears after clicking.

Cause

The consent statement created in Customer Data Cloud (CDC) cannot be used on the Recruiting career site.

Resolution

To create the consent statement for SuccessFactors Recruiting, use the 'Data Privacy Statement' feature in SuccessFactors. This will be synchronized to Customer Data Cloud (CDC).

The Data Privacy section in Customer Data Cloud (CDC) will be removed starting from the 1H 2025 release. As part of it, users will not be able to create consent statements in Customer Data Cloud (CDC).
