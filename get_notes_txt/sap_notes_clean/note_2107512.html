Symptom

Some errors from the backend are not well described to the user or swallowed. It can make it difficult to notice the error or find the solution.

Reason and Prerequisites

Too short or too general text messages for errors. Missing code to write the errors in the respective trace files.

Solution

We provide a patch for the described problem that improves error texts and error tracing to the default trace files. The correction is part of the following (and newer) Support Packages:

  • PI 7.30 SP14
  • PI 7.31 SP16
  • PI 7.40 SP11

For older Support Packages, the correction is available as a patch. Please update the software components of the Adapter Engine to the patch levels maintained in section "Support Packages & Patches" of this note or use newer versions.

The archives and the support package stack guide can be found on the SAP Service Marketplace as described in SAP Note 952402.
