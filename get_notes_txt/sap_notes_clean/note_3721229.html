Symptom

When the process "Sending of Deactivation of Assignment Authorization (PRS View)" (DE_B_EQS_ASSAUTH_DACTV_SEND_PRS) is triggered through transaction /US4G/EQS_ASSGN_AUTH, the system updates the deactivation end date with the time 00:00:00 instead of 23:00:00.

This issue is relevant for the market roles: PRS, DSO.

Other Terms

US4G, EQS, assignment authorization, /US4G/EQS_ASSGN_AUTH, end date, end time, deactivation

Reason and Prerequisites

The error occurred due to a technical issue in the system.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the system updates the assignment authorization with the end date and the correct time of 23:00:00. You need to retrigger the assignment authorization with the withdraw option. You can verify the fix by checking that the end date is updated with the time 23:00:00.
