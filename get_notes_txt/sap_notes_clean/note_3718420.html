Symptom

The IMG path view are missing after implementing relevant SAP Notes for SAF-T Bulgaria.

The following menu path for SAF-T Financial Data report in Bulgaria configuration is not available in SPRO: 

Customizing for Cross-Application Components under: General Application Functions ->  Document and Reporting Compliance -> Country/Region-Specific Settings ->  Bulgaria  Statutory Reporting ->  SAF-T 

Environment

•Financial Accounting (FI)

•SAP S/4HANA Finance

•SAP S/4HANA

Reproducing the Issue

  1. Implement collective SAP Note 3676192  - SAF-T Bulgaria: Collective Note (SAP S/4HANA) and its prerequisites.

  2. Try to navigate to the above IMG path.

  3. Observe that the node for SAF-T Bulgaria is not visible in customizing.

Cause

The node for SAF-T Bulgaria is delivered with a Support Package (SP). If the SP is not available in the system, the node will not appear in customizing.

Resolution

Use SM34 views mentioned on SAP Help Portal or in attachments of notes to configure the SAF-T Bulgaria node manually.
Refer to SAP Help Portal for detailed steps on using transaction SM34 views to access the relevant activity by utilizing view cluster SRFVC_RPG_ENT.
