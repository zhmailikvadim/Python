Symptom

This Knowledge Base Article guides how to create a Service Catalog Request for Prophecy Provisioning.

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Note: Steps related to access and where you can create service requests are mentioned in the following article KBA: 3302846 - Incentive Management: How-To Submit a Service Catalog Request 

Environment

SAP Incentive Management on Hyperscalers

Resolution

In SAP for Me Page, the user to Navigate to the ‘Create a Service Request’ to submit a request following these steps:

  1. Choose 'Services and Support' on the home page;

  2. Click on Create Service Request



  3.  A Service Catalogue with different Service options is displayed as below.


     


  4. Select the SPM catalogue and click Go. This will display the services under the SPM catalogue, including Prophecy.



  5. Select the SPM: PROPHECY PROVISIONING Service to proceed. 



  6. Click on Add Service option.




  7. Click on Select a SAP tenant to select either of SAP Incentive Management or SAP Agent Life Cycle Management or SAP Agent Performance Management Additional Test Tenants.




  8. Select a Tenant and a Confirmation screen display. Check the Confirmation check box and complete the request. This will create a Service Request in Sap4Me.
