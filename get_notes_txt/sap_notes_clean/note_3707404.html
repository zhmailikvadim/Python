Symptom

Splash Screen is not being displayed when login into SuccessFactors mobile app even when it was configured previously.

Environment

SAP SuccessFactors HCM Suite

Reproducing the Issue

  1. Access the SuccessFactors instance.
  2. Go to Admin Center > Enable Mobile Features > Branding.
  3. Set the Background image for portrait and landscape.
  4. Login SuccessFactors mobile app and check that the Splash Screen is not being displayed.

Cause

The image size and aspect ratio is not being set correctly when the Splash Screen was set up.

Resolution

In order to configure correctly this feature follow the next help guide considerations: Customizing the Splash Screen in the Mobile App | SAP Help Portal

Help guide - Customizing the Splash Screen in the Mobile App | SAP Help Portal
