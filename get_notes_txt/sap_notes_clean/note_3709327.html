Symptom

The rule of Preselect Offer Letter Templates does not work after saving the offer letter as draft.

Environment

SAP SuccessFactors Recruiting Management

Reproducing the Issue

  1. Set a Preselect Offer Letter Templates rule.
  2. Go to Recruiting, select an application who does not have an offer letter.
  3. Take Action > Offer > Offer Letter.
  4. The rule works as expected. Then save the offer as a draft.
  5. Back to application page and change the condition field of the Preselect Offer Letter Templates rule.
  6. Repeat the offer letter create operation: Take Action > Offer > Offer Letter.
  7. The offer letter template cannot be changed following the rule. And there has a warning message displaying:
    “Warning
    This draft of the offer letter was last modified by XXX on DD/MM/YYYY. Any changes you make will overwrite the current draft. ”

Cause

This is the expected behavior.

Resolution

The rule of Preselect Offer Letter Templates only works when creating a new offer letter. 

If needing the offer letter template to be selected by the rule, please click Cancel, rather than Save as Draft. 

Once you save the draft, the rule will no longer run. Please change the offer letter template manually.
