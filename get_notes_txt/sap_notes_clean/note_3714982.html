Symptom

The "Reservation Hand Over" and "Cancel Order" buttons are disabled in the "Hand Over Orders" App (App ID F46390), even though the picking request has been processed in the "Process Picking Requests" App (App ID F4643).

 



Environment

  • SAP ERP
  • SAP S/4HANA
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • Retail

Reproducing the Issue

  1. In the "Process Picking Requests" App, input scan barcode number and process the Picking Request.
  2. Go to the "Hand Over Orders" App, the "Reservation Hand Over" and "Cancel Order" buttons are disabled.

Cause

It is standard system behavior that the buttons are disabled if the order has not been withdrawn from the shelf (withdraw status is 'No') in the Hand Over Orders App.

Resolution

To enable the "Reservation Hand Over" and "Cancel Order" buttons, first go to the "Handover Status" button in the Hand Over Orders App and withdraw the order from the shelf.
