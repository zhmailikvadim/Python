Symptom

You are using transaction /SAPSLL/IS_MM in a ECC feeder system to send data to SAP GTS, Edition for SAP HANA. Items with weigths that are less than 1G are not currently supported.

Current correction adds support for sending weights, that are lower than 1G, to GTS.

Other Terms

Intrastat, SAP GTS Edition for SAP HANA, GTS, MEIS, /SAPSLL/IS_MM, NETWEI, NWUOM, EIGENMAAK, EIGENMASS, EIGENMAEH, tare mass, low weight

Reason and Prerequisites

This issue is caused by a program error.

SAP Note "3718398 - Support for low weights" is a prerequisite for SAP GTS Edition for SAP HANA and must be implemented in the GTS system.

Solution

Implement the correction instructions provided with this note or upgrade to the relevant Service Pack.
