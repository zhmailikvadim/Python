Symptom

How do I send a password reset email as a buyer to the supplier user?

Resolution

To send a password reset email

  1. Click Manage > Administration > Supplier and Customer Manager > Users.
  2. Search for the user and select it.
  3. Click Generate Passwords and a popup will appear 
  4. To view the temporary password, check the Display temporary passwords. You can use this password to reset the user’s account password manually.
  5. Click OK. The system automatically sends a password reset email to the user as well.
