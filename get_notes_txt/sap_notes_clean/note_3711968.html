Symptom

Explanation on how the Approvers for PAM Log Review are determined

Environment

SAP Cloud Identity Access Governance 2.0

Resolution

The PAM stages operate as follows:

  1. Role Owner reviewers are defined in the Maintain Privileged Access application.
  2. Manager reviewers are determined in the IAS application as the manager of the employee who created the PAM log. If no manager is defined, a user from the IAG_WF_MANAGER group is assigned for this stage.
  3. Security reviewers are defined by membership in the IAG_WF_ADMIN group.
