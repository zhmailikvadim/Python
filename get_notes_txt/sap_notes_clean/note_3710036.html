Symptom

The following error is displayed when saving a template to the server:

File scanning error.

An exception was raised

Report error ID <ID> to the system administrator

Environment

  • SAP Integrated Business Planning
  • SAP IBP, add-in for Microsoft Excel

Cause

The issue is caused by the IBP system's virus scanner setup.

Resolution

Open a case to the "SCM-IBP-OPS-SRV" component and ask for a health check on the affected system's virus scanner.

Keywords
