Symptom

Accessing the Time Statement the results are not appearing. In the overview there are 0 worked hours and PDF is also with empty data.
An administrator can see the result for the same employee with the same Form by running transaction HRFORMS.
Other employees have no issue.

Environment

HCM Fiori

Reproducing the Issue

Open My Time statement app and check the PDF

Cause

The user has no authorization of infotype 0008.

The employee would not see the result even if they used the HRFORMS transaction.

Resolution

Please follow the KBA 1917599 Cannot print time statement without infotype 0008 read authorization by running transaction HRFORMS

Keywords

HCMFAB_MYTIM_ST, hcm.fab.mytimestatements , My Time Statements
