  • The same user was able to login in CMC (Central Management Console)

Environment

  • SAP BusinessObjects Business Intelligence platform BI 4.x / 2025

Reproducing the Issue

  1. Login to BILP
  2. Blank screen will load after logging in.

Cause

  • This issue appears because user has corrupted preferences.

Resolution

Note: Please take a Backup of the CMS database before running this script.

  1. Create a group with only that user

  2. Follow the steps in KBA-1659566 - How to Apply Preferences to All User Accounts in BI 4.x programmatically [Video].

  3. Then use that script where the user to copy from is a working user, and the user group to copy to is the group containing only that user.
