Symptom

  • Analytical model validation displays the error message: "Leveled Hierarchy Attribute missing."
  • Error message specifies: "Leveled Hierarchy Attribute '<attribute>' of source '<dimension>' is missing.

Environment

  • SAP Datasphere

Reproducing the Issue

  1. Open the Analytic Model in the Data Builder.
  2. Navigate to the model validation section.
  3. Observe the error message: "Leveled Hierarchy Attribute is missing."

Cause

  • Attributes used in the hierarchy need to be selected in the Analytic Model.
  • The attribute might be added to the hierarchy, after the analytic model was deployed, but it was not selected in the analytic model.

Resolution

  1. Select the attribute in the analytic model to resolve the error. 
  2. Depending on the business scenario removing the attribute from dimension or adding into analytic model would be other alternatives.
