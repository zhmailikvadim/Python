Symptom

An error shows up when you would like to open or approve/decline the External Learning Request Verification Approval Card:

externalDetailMessage=You do not have the permission to access the approval request.

Environment

SAP SuccessFactors Learning

Reproducing the Issue

  1. Log in to SuccessFactors HCM Suite
  2. Find the external learning request verification Approval Card on the Home Page
  3. Open the card or try to approve/decline the request
  4. An error message is displayed: You do not have the permission to access the approval request.

Cause

An administrator verified the external learning request on behalf of the user in Learning Administration and the request was submitted with "submit as approved". In this scenario, the card create/delete events are published in the same transaction. As a result, card service may receive these events out of order and cause the card to remain on the home page for other approvers.

Resolution

Our Engineering Team is targeting the upcoming 1H2026 major release to provide a permanent fix for this issue.

Please follow this KBA for further updates.

Note for SAP Support: Please see internal memo for further information.
