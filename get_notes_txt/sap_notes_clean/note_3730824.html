Symptom

• Organizational authorization restriction in Fiori App F2279 – Manage Output Items does not work for a test user limited to a single company code.
• Irrelevant entries from other companies are displayed in the app.
• No specific error messages or error codes are provided. 

Environment

  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Cause

Resolution

As outlined in documentation : Manage Output Items | SAP Help Portal
for Authorization Object CA_OC_ORI:
OC_APPL_TY: Application Object Type (Business Object)
OC_OUTP_TY: Output Type
OC_CHANNEL: Output Channel
OC_ORGU_ID: Output Management: Organizational Unit ID     >> SenderOrganizationalUnit           >>> APOC_D_OR_ITEM-SENDER_ORG_UNIT_ID
OC_ORGU_TY: Output Management: Organizational Unit Type  >> SenderOrganizationalUnitType    >>> APOC_D_OR_ITEM-SENDER_ORG_UNIT_TYPE
OC_ORG_ID: Output Management: Organization ID                >> SenderOrganizationalCenter        >>> APOC_D_OR_ITEM-SENDER_ORGANIZATION_ID
OC_ORG_TYP: Output Management: Organization Type           >> SenderOrganizationalCenterType >>> APOC_D_OR_ITEM-SENDER_ORGANIZATION_TYPE
ACTVT: Activity
