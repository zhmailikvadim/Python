Symptom

The field 'Var.Portion Type' is missing while trying to execute cost center plan assessment cycle of shared expense.

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Go to 'Manage Allocations - New Version' App
  2. Select an Allocation Cycle 
  3. Go to 'Segment'
  4. Expand and select 'Receiver Basis'
  5. Could not find the field 'Var.Portion Type'

Cause

The reason why the cycle is not visible in the script is because the assessment’s localization plan cycle differs from the DE configuration, and the correct setting wasn’t applied.

Resolution

  1. Go to the App 'Manage Allocations - New Version' (F3338A), select 'Cost Centers' as the Allocation Context and the expected Allocation Cycle, the 'Go'.


  2. Select the Allocation Cycle and choose '>' to access the 'General Information' section.

  3. Scroll down, to find 'Segment' (Derivation Rules section) and choose '>' to expand.


  4. In the next page, select 'Edit' and under the 'Rules' tab, change the 'Receiver Rule' to "Variable Portions", then select 'Save'.
    [Note: Select 'Yes' if a pop-up asks "Do you want to perform the elements clean up now?”]


  5. Next, move on to 'Senders' section, choose 'Edit' and 'Save' if needed, checking the correct 'Account Number Group' and 'Cost Center' used.

  6. Then, in the 'Receivers' section, choose 'Edit', 'Add' and the 'Add Receiver(s)' window opens.


  7. Select 'Cost Center' and choose 'Add'.

  8. Create the following entries and select 'Save' after:
    • Value Type: Select via dropdown "Group"
    • Group: Select via dropdown search


  9. Next, go to 'Receiver Basis' section and click 'Edit'

  10. And in the 'Var. Portion Type' select "SKF-1 Plan" via dropdown
    [Note: Select 'Yes' if a pop-up asks "Do you want to perform the field clean up now?”]

  11. The 'Receiver Basis (2)' window appears, then select the 'Plan Category' and 'Stat. key figure' and click on 'Check Segment' and 'Save'
