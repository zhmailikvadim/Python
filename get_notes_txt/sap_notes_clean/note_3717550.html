Symptom

In app "Manage Characteristic Catalogs" (F3829) are the selection options of continous value groups in table "Continuous Value Definition" distributed over different rows. One selection option shall be displayed in one row.

Other Terms

aATP, advanced Available-To-Promise,

Solution

Implement the attached correction instruction.

After implementation of the note the new app coding will be operative after batch job /UI5/APP_INDEX_CALCULATE is executed.

If you cannot wait until execution of this batch job, you may start the corresponding program /UI5/APP_INDEX_CALCULATE manually, enter MANCHARCCTLGS1 in tab strip "Single SAPUI5 Repository Only" and press execute button.
