Symptom

You are receiving an error while trying to implement SAP note 3212219:
LHRPAYUS_V_T5UTKZT00 not found

Environment

For customers using BSI TaxFactory 11.0 OnPremise: minimum Cyclic H level.
For customers using BSI TaxFactory 11.0 SaaS: the solution is already available on the TaxFactory instance
S4HCMCUS  100     
SAP_HRCUS 608 

Resolution

1. Report NOTE_3212219 must be run in the mode "Generate Maintenance Dailog" for performing this step you need to have a modification key for the user which is running the report and run it until the log is green.

2. Even after running the report the issue persists, you can create the view V_T5UTKZ using transaction SE54 following the steps:

  1. Open transaction SE54.
  2. Select the V_T5UTKZ view.

  3. Select Generated Objects and Delete

  4. Select all the options that appear and delete them all.
  5. If a pop-up appears saying that the includes LRHPAYUS_V_T5UTKF00, .. I00 ... T00 are being used, click on the delete button.
  6. After deleting, keep the Generated Objects option selected and click on Create/Change.


  7. It will show pop-up informing the objects don’t exist and asking for creation confirmation.
  8. Fill in as shown below:


  9. Click on the Create button at the top of the screen.
  10. Still in the Generated Objects screen, go to Environment -> Generate Time-Dep.

  11. Open SM30 transaction and check if view V_T5UTKZ is working.
