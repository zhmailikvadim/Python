Symptom

The DocuSign envelope is sending the email to all recipients at the same time instead of routing it sequentially.

Environment

SAP SuccessFactors Onboarding

Reproducing the Issue

  1. Initiate the onboarding process that includes the DocuSign e-signature workflow.

  2. Review the envelope delivery behavior.

  3. Observe that the envelope is sent to all recipients simultaneously for signature instead of following the configured sequential routing order.

Cause

This is expected behavior. The DocuSign envelope is configured to send notifications to all recipients simultaneously rather than using sequential (routing order) delivery.

Resolution

At this time, the onboarding process does not support ordered (sequential) signatures in DocuSign.

As a result, the email notification is sent simultaneously to both the manager and the candidate.

In the current system version, this behavior is expected, as signatures are triggered in parallel for all assigned users. It is recommended that an enhancement request be submitted so the Product Management team can evaluate the feasibility of adding this functionality.
