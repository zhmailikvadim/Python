Symptom

  • Error displayed in Monitor Value Chain app: "Events not processed. Contact administrator. Message No. VCM_UI006".Diagnosis: Events are in the event queue and not delivered. System Response: The business process cannot be automatically processed.
  • Error message when attempting to change mode for Sales Order: "Document updates are in progress. Try changing your document later".

Environment

  • Sales and Distribution (SD)
  • Logistics Execution (LE)
  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA
  • Value Chain Monitoring Framework

Reproducing the Issue

  1. Go to VA01/Create Sales Order.
  2. Enter Order type
  3. Enter Sold to party
  4. Select Sales Area
  5. Enter Material
  6. Enter Order quantity
  7. Go to Shipping tab
  8. Maintain Shipping type
  9. Save Order
  10. Observe that the order saves with a delivery block
  11. Go to Monitor Value Chains app
  12. Error displayed: "Events not processed. Contact administrator. Message No. VCM_UI006".

Cause

The RFC destination is not configured in the system, which causes the events to remain in the queue and not be delivered. 

Resolution

Configure RFC destination following the KBAs 3649164 and 3385233

See Also

  • KBA 3649164: VCM events are not triggered correctly
  • KBA 3385233: How to set up a workflow destination using “Configure RFC Destination” in SWU3
