Symptom

  • The fields "Excise Division" (J_1IEXDI) and "Type of Vendor" (J_1IVTYP) are not available for selection in T-Code MASS when using object type LFA1.
  • The standard mass processing tools, including the MASS transaction and Fiori applications such as "Start Mass Processing Business Partner", do not accommodate these specific fields.

Environment

  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Navigate to T-Code MASS.
  2. Select the object type LFA1 and execute.
  3. Choose table LFA1, then go to the Fields tab to select the field "Type of Vendor" (J_1IVTYP).
  4. Observe that the field "Type of Vendor" (J_1IVTYP) is not available for selection.
  5. Similarly, attempt to locate the field "Excise Division" (J_1IEXDI) for mass update using T-Code MASS, but this field is also not available.

Cause

The described issue is standard SAP behavior.

Resolution

The fields "Excise Division" (J_1IEXDI) and "Type of Vendor" (J_1IVTYP) are not supported for selection in T-Code MASS when using object type LFA1. 
The standard mass processing tools, including the MASS transaction and Fiori applications such as "Start Mass Processing Business Partner" do not accommodate these specific fields.
Certain vendor master fields require specialized handling that falls outside the scope of standard mass processing functionalities. 

As the described issue would be consider standard behavior, to address your requirement, a custom development approach would be necessary.
For S/4HANA systems, we recommend utilizing class CL_MD_BP_MAINTAIN method MAINTAIN to update Business Partner data centrally.
