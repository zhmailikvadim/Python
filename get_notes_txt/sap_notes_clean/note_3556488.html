Symptom

After DI Cloud maintenance, there is no delta data being written to target ADL container. The error raised by RMS may look like:

005056BD2C331EDE829C2F252F6B0A3C: Last response received at: 11/4/2024, 9:16:10 AM,    roundtrip failed: failed to read message - roundtrip failed: failed to read message - websocket: close 1011 (internal server error): request failed: rc=1, msg="no response from the source system after 119 seconds", when closing: websocket: close sent

Environment

SAP Data Intelligence Cloud

Cause

The issue is due to there is no support by the target platform ADL for concurrent atomic writes.

Resolution

This is fixed in DI Cloud 2404.13.47.
