Symptom

When an Adv. Shipping and Receiving notification was received to create or update a TM document like a consignment order with a package item, for which an issue was detected, a document error will be created. Additionally for the child items a reference document error will be created even they could be referenced to a requirement item. But the reference document error were created without information about the corresponding requirement / referenced document.

Other Terms

Adv. Shipping and Receiving notification, Consigment Creation, Consignment Update, Reference error

Reason and Prerequisites

Program error

Solution

Implement the attached correction instruction or install the corresponding Support Package.
