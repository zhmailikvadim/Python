Symptom

  • PDF documents embedded in SCORM content are opens in the SAP SuccessFactors Mobile App but are not scrollable.
  • The issue has been observed in the SuccessFactors Mobile App for iOS device. 

Environment

  • SAP SuccessFactors Mobile App
  • SAP SuccessFactors Learning

Reproducing the Issue

  1. Access the SAP SuccessFactors Mobile App on an iOS device. 
  2. Open the course and launch the content.
  3. Click the embedded PDF document.
  4. Observe that the PDF opens but scrolling is not possible.

Cause

This issue has been identified as a bug in the SAP SuccessFactors Mobile App. 

Resolution

The resolution for this issue has been implemented. It is scheduled to be delivered as part of the Mobile 2603 release. The MobTrain 2603 is currently scheduled to be released on March 17, 2026.

  • The fixed version of the SuccessFactors Mobile app will be 16.1.3.
  • The iOS app is under Apple's review and will be released as soon as it is approved.
