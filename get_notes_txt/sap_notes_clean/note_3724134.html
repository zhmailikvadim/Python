Symptom

The settlement partners maintained for depcodes are cleared in the Performance tab when the agreement is opened in change mode. However, the values are visible when the agreement is opened in display mode.
This note resolves the issue by ensuring that the settlement partner values remain visible in both display and change modes.

Other Terms

/IRM/IPA, PART_INFOCUS_AGR_FILL, Agreement Workbench, Settlement Partner, Performance Tab, Auto Participant

Reason and Prerequisites

This is caused by program error.

Note : SAP Note# 3724157 is available for ECC release.

Solution

Please implement the correction instructions. These corrections are only valid if you have applied the following Support Packs.

Release V/4 – 2021, 2022, 2022 V2, 2023, 2023 V2, 2023 V3 and Support Pack 1

Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object
