Symptom

Collaboration tab is not available on the change version of requisition.

Resolution

In the initial version, the requisition is created as a Collaboration requisition, which enables supplier interaction and displays the Collaboration tab.

However, in a subsequent (change) version, if the “Re-Collaborate” field is set to “No”, the system interprets this as no further supplier interaction being required. As a result, the Collaboration tab is no longer displayed for that version of the requisition.

The Collaboration tab is only visible when supplier collaboration is active or required for that specific version of the requisition.
