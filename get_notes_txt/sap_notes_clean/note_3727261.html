Symptom

 In the process "End of Supply for Gas (DSO View)" (DE_B_GELIGAS_EOS_DSO), the rejection checks for the data type definition (DTD) G_0007 need to be updated to align with the current GPKE document requirements. The embedded process "Check Rejection of Deregistration for End of Supply Process" (DE_B_GELIGAS_DSO_EOS_G_0007) at step 1200 requires updated validation logic.

This issue is relevant for the market role: DSO.

Other Terms

G_0007, DE_B_GELIGAS_EOS_DSO, DE_B_GELIGAS_DSO_EOS_G_0007, DTD, GPKE, embedded process, deregistration, rejection codes, Z14, Z08, E14, E17

Reason and Prerequisites

This SAP Note provides a new development change to align the rejection checks for DTD G_0007 with the updated GPKE document requirements.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the system uses the updated embedded process DE_B_GELIGAS_DSO_EOS_G_0007 (version 2604) at step 1200, which includes the revised rejection checks for G_0007 as specified in the GPKE document.

You can verify the fix by testing scenarios Z14, Z08, E14, and E17, and checking that the rejection codes Z14, Z08, E14, and E17 are sent correctly in the Cloud Pdoc at step 100 (Send EoS Rejection to Old Supplier).

This SAP Note is related to emergency patch 2000009185 on the Cloud side.
