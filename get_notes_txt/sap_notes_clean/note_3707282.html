Symptom

Runtime errors occur in transaction ST22 "SYNTAX_ERROR". The supposedly erroneous Include doesn't seem to have a connection to the main program displayed
Shortly before the time of the runtime error, you can see entries of the following form in the Developer Trace files:

*GENER* program <prog> is known to have a syntax error.

Other Terms

There is an error in the SAP kernel.

Reason and Prerequisites

In some cases, the information about whether syntax errors were detected for a program during generation is not updated correctly.

Solution

Apply the kernel patch specified under "Support Packages & Patches".

The patch text is "ABAP Generation: wrong information about syntax error"
