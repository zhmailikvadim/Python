Symptom

No data is visible in RSA7 for the following datasources after the init/delta run: 

  • 0RT_PA_TRAN_CONTROL – POS Analytics: Sales Transactions and Test Transactions
  • 0RT_PA_TRAN_GDS_MOV – POS Analytics: Goods Movements
  • 0RT_PA_TRAN_LOYALTY – POS Analytics: Loyalty Program
  • 0RT_PA_TRAN_MOV_FIN – POS Analytics: Financial Transactions
  • 0RT_PA_TRAN_TOTALS – POS Analytics: Totals Records

Environment

  • BW only - Point of Sales Inbound Processing Engine 
  • SAP Customer Activity Repository

Reproducing the Issue

  1. Open transaction RSA7.
  2. Verify the datasources:
  • 0RT_PA_TRAN_CONTROL – POS Analytics: Sales Transactions and Test Transactions
  • 0RT_PA_TRAN_GDS_MOV – POS Analytics: Goods Movements
  • 0RT_PA_TRAN_LOYALTY – POS Analytics: Loyalty Program
  • 0RT_PA_TRAN_MOV_FIN – POS Analytics: Financial Transactions
  • 0RT_PA_TRAN_TOTALS – POS Analytics: Totals Records

Execute the delta process for the datasources.
Observe that no data is extracted for the datasources listed above.

Cause

Lack of goods movements

Missing BW task

Resolution

To proceed, BW tasks (0004 for example) need to be run for goods movement transactions, financial transactions, and totals transactions.
