Symptom

Perceived payroll process slowness

Reason and Prerequisites

You run payroll calculation through programs RPCALC*0 or H**CALC0

Solution

This note delivers a new program called RH_PAYR_PERF_CHECK and its corresponding transaction, that measures the speed of the payroll driver in periods per second

Please check KBA https://me.sap.com/notes/3732217 for more information about this application
