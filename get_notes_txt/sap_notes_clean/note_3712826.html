Symptom

  • Lookup function doesn't return expected results
  • Lookup function return NULL/blank values while you believe it should return your expected data

Environment

SAP Cloud Integration for data services

Reproducing the Issue

  1. In dataflow transform, define lookup function for a field. Defining the "default_value" of lookup as '' or Null
  2. Run task, it observed that some values are blank or Null for this field

Cause

The conditions defined in the lookup function are not satisfied

Resolution

Validate the data in both the lookup file/table and the source table. If there are data couldn't satisfy the conditions defined in the lookup, the default value Null are returned
