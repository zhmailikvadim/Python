Symptom

In the MIRO Fiori App, a dropdown option is available next to the PO Reference field, allowing selection of different reference options related to the purchase order.

In the Fiori App F0859 (Create Supplier Invoice), the dropdown option next to the PO Reference field is not available.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Open the Create Supplier Invoice App (F0859).
  2. Navigate to the PO Reference tab.
  3. Search for the dropdown that should display Goods/Service Items.
  4. Observe that the dropdown is missing.

Cause

The Fiori app Create Supplier Invoice (F0859) provides a simplified interface and does not support all functionalities available in Create Supplier Invoice – Advanced (MIRO).

Resolution

Use the Create Supplier Invoice – Advanced (MIRO) app for invoice entry if reference options such as the dropdown next to the PO Reference field are required.

Confirm that the dropdown functionality described is standard system design not available in the F0859 app.

See Also

Refer to: KBA 3037629 - Missing Material field in Create Supplier Invoice app
