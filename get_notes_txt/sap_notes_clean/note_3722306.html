Symptom

Getting error: "You do not have start authorization for R3TR IWSV" when calling the OData servicet with communication user

Environment

SAP S/4HANA Public Cloud 

Reproducing the Issue

  1. Call the OData API in SAP S/4HANA Public Cloud system
  2. Observe the system returning the error message: "You do not have start authorization for R3TR IWSV xxxx, return code 4".

Cause

Missing Authorization.

In SAP S/4HANA Public Cloud system, the authorization for OData API is aessignaed to user by creating a communication arrarngement and bind it to a communication user.

Resolution

  1. Create the communication arrangement in the system and set the communication user.
  2. Publish the communication arrangement , the authrization will be assigned to the communication user.
