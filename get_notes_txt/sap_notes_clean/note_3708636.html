Symptom

  • An error is displayed in the Monitor Rating but the error details are not visible.

Environment

Subscription Billing

Reproducing the Issue

  1. Access the Monitor Rating in SAP Subscription Billing.
  2. Observe that an error is displayed in the Monitor Rating.
  3. Attempt to view the error details, but they are not visible.

Cause

  • The Rating monitor worklist fetches only the first 500 entries, and none of them includes the error.
  • Currently, there is no direct filter for warning/error messages.

Resolution

  1. Open the Monitor Rating in SAP Subscription Billing.
  2. Select all message types in the filter except the "no price found" type.
  3. Locate the error details.

Keywords

Monitor Rating, Subscription Billing, Rating Monitor Worklist, Error Filter, Message Types, No price Found.
