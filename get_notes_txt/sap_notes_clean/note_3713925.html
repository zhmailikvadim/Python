Symptom

You run the Aging Report - Accounts Receivable (transaction IDCNAR) or the Aging Report - Accounts Payable (transaction IDCNAP) reports for China and Japan. You have run the report with the range search for Customer Account or Vendor Account and with Profit Center or Segment filtered. The system returns no results.

Other Terms

Aging Report - Accounts Receivable, Aging Report - Accounts Payable, IDCNAR, IDCNAP, RFIDCN_AR_AGING, RFIDCN_AP_AGING, China, Japan, CN, JP

Reason and Prerequisites

This is a program error.

The reports select data with an incorrect filter clause.

Solution

Apply the corresponding support package or implement the correction instructions in this note.
