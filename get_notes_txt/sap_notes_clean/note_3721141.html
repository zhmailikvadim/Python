Symptom

As of March 01, 2026, the payroll system no longer complies with the new and revised 2026/2027 tax table and PAYE-related modifications. This SAP Note provides you with the relevant system changes to ensure legal compliance for the 2026/2027 tax year. You can learn more about this in the regulatory change manager tool under 'Budget speech changes effective March 01, 2026'.

Other Terms

Tax Rates, Tax Rebates, Salary Packaging: ZA Receiver Rates, V_T5W2R, V_T5W2S, V_T5W7F, MKRR, PFCAM, MAETC, MADTC

Reason and Prerequisites

Legal change.

This SAP Note is relevant only for South Africa localization.

Solution

After implementing this SAP Note, the payroll system complies with the 2026/2027 tax table and PAYE-related modifications effective March 01, 2026

1. Tax Table Changes for Tax Year 2027:

View V_T5W2R - Tax Rates

  • Enter the transaction code SM30 in the command field. 
  • Enter V_T5W2R in the View Name field and choose Maintain. 
  • Enter 2027 in the Tax Year field when the system prompts. 
  • Choose New Entries and create the tax rate entries as shown below.

2. Tax Free KM Reimbursed Rate Changes

V_T511P

  • Enter the transaction code SM30 in the command field.
  • Enter V_T511P in the View Name field and choose Maintain.
  • When prompted for the Country Grouping, enter the value as 16.
  • Select Constant MKRR and choose Delimit.
  • When prompted for the Start Date, enter the start date as 01.03.2026.
  • Create the entry as shown below.
Constant From To Amount Currency
MKRR 01.03.2026 31.12.9999 4,95 ZAR
  • Save the changes.
