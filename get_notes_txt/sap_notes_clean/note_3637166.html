Symptom

You process order RAI and you get the error message 067(FARR_PROCESS_CHECK) "Finalization date earlier than start date". The error is displayed even if the start date is before finalization date.

Other Terms

FARR, Revenue Accounting, POB Cancellation, 067, FARR_PROCESS_CHECK

Reason and Prerequisites

Program error.

The issue appears only if you changed the start date. The SAP Note is relevant only for Classic Inbound Processing.

Solution

Implement this SAP Note or upload the corresponding Support Package.
