Question

Why do I get "The refresh token is invalid" error?

Answer

OAuth Refresh Tokens are valid for 45 days from the issue date. Using the Refresh Token after 45 days from issue date we get “404 Bad Request” error with below error message

{

    "error": "invalid_grant",
    "description": "The refresh token is invalid"
}

If we get this error we should re generate new Refresh Token using the steps from the link Generate Token
