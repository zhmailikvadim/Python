Symptom

You create the WHT confirmation for employees (HRFORM: SAP_QSTC1_CH01, transaction PC00_M02_HRF_QSTC1).

If the SSL number of a payroll unit has changed over time, the system may still display an obsolete number on the form instead of the SSL number that is valid for the selection period.

Other Terms

Withholding tax, WHT confirmation, HRFORM, SSL number, T5C2Q, payroll unit, ELM, FILL_FORM_WORKPL_ADDR, ABRNR, CL_HRPAYCH_HRF0_QSTC1

Reason and Prerequisites

During the selection of the SSL number maintained in the table T5C2Q, the system did not check against the evaluation period. As a result, the obsolete SSL number may have been read when the SSL number of a payroll unit was changed.

Solution

Import the HR Support Package relevant for your release. If you have to implement the solution in advance, use the SAP Note Assistant to implement the correction instructions relevant for your release. You can find more information about the SAP Note Assistant on SAP Support Portal at https://support.sap.com/notes.

After you implement this SAP Note, the SSL number that is valid at the end of the selected evaluation period is always printed on the form on the form.
