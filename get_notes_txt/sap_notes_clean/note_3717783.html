Symptom

Salary Range and Guideline display in Right-to-Left order.

For example, when set display format set as min - def - max and employee's local currency code is SAR, the Range will display as max - def - min in the Executive Review and Worksheet.

Environment

  • SAP SuccessFactors Compensation
  • SAP SuccessFactors Variable Pay

Resolution

This is expected behavior for RTL(Right to Left) language currency, such as (Arabic or Hebrew currency like SAR or ILS). The browser will auto apply RTL to Arabic or Hebrew formatted currency. So it will display as max-def-min when you set the format as min-def-max.
