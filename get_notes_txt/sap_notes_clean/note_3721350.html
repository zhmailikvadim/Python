Symptom

There is no released function for determining all matching identities and corresponding lives for an identity from the database.

Other Terms

  • Movement Processing 
  • Life Matching
  • Function: GET_IDENTITIES_FROM_DB

Reason and Prerequisites

Reason:

When the search for matching or similar identities (life) was adjusted and a BRF+ function was added, the previous function was replaced. 
The new function returns only the most suitable or similar identity/life.
However, there is no longer a released function that starts the identity search and returns all possible matching identities and lives as results.

Solution

History:

March 3, 2026: New

Troubleshooting Description:

During the life search, the function GET_IDENTITIES_FROM_DB was created, which allows the search for identities from the database using the ID type and ID number or using the name, date of birth, and gender. 
All identities found, and thereby indirectly the corresponding lives, are returned.
