Symptom

The Inland Revenue Authority of Singapore (IRAS) has issued official updates to Form IR8A, Appendix 8A, and Appendix 8B for the Year of Assessment 2026.

This SAP Note updates the system’s current PDF form layouts to align with the latest IRAS requirements. The updated layouts include:

  • IR8A layout: HR_SGTAXIR8A_2026
  • IR8E layout: HR_SGTAXIR8E_2026

Key text changes are as follows:

  • In Appendix 8A: Revised the description for item h from 'Driver [Annual Wages X (Private / Total Mileage)]' to 'Driver [Annual Cost X (Private / Total Mileage)]'.
  • In Appendix 8B: Added the following fields at the head: 'Tax Reference No : _______________________ Full Name of Employee as per NRIC / FIN / Passport :_______________________'

Other Terms

IR8A, IR8E, PDF Layout updated

Reason and Prerequisites

This is a legal change update.

Solution

Apply the HR Support Package to get this solution.

Alternatively, you may apply manual correction instruction with this SAP Note if the changes are required immediately.

For more information about the development delivered with this SAP Note, see the entry for this SAP Note in the What's New topic on SAP Help Portal.
