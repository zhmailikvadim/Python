Symptom

You are working with the Last Mile Distribution solution. You are trying to settle a route, but the settlement run fails with the message "An error occurred. Contact the help desk to check transactions ST22/SM21".

However, the information are not enough to resolve the issue.

Other Terms

Last Mile Distribution, LMD, Settlement, Timeout, Settlement Error, ST21, SM21, SAP Fiori, F4242, With Errors

Reason and Prerequisites

In some cases, additional information on the actual source of the problem might be available to be displayed in the Settle Route Data app on the Log tab.

Solution

To identify the root cause, more detailed information might be provided in the Settle Route Data app on the Log tab.

Apply the attached correction instructions to use the described enhancement.
