Symptom

  • Backend error occurs when double-clicking on GL account lines in the account assignment screen.
  • The system becomes unresponsive after the error.
  • Error message: “Backend error occurred – control not found on batch step #2”.

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

  • SAP GUI for HTML / WebGUI
  • SAP Kernel 793
  • SAP Kernel 916

Reproducing the Issue

  1. Open the S/4HANA system using SAP GUI for HTML / WebGUI
  2. Navigate to the Account assignment screen. Transaction: F-05  (issue has also been observed with transactions: FB50, FB60, F-01, F-02 and CR02 )
  3. At the bottom of the screen. In the section: 'First Line Item' Hit F4 on the: Account field
  4. Then select 'S: G/L Account description code.'
  5. Enter '1000' in the COMPANY CODE and hit search.
  6. Double click on ANY of the RESULTS.  
  7. Observe the backend error message and the screen becoming unresponsive.

Cause

Regression from the correction in the SAP Note: 3696500 - User defined buttons missing in valuehelp

Resolution

Apply the Kernel listed in the SAP Note 3708147 - Valuehelp with error on toolbar button
