Symptom

  • Unable to execute interface InvoiceRequest_In to create a Carrier Invoice for a Freight Booking
  • Error message :/SCMTMS/SFIR 005 "Enter freight settlement document type" is raised while processing InvoiceRequest_In
  • Error message : /SCMTMS/SFIR 148 "Document type does not belong to category 50" is raised while processing InvoiceRequest_In

Environment

  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition
  • SAP Transportation Management

Reproducing the Issue

  1. Execute interface InvoiceRequest_In with the details in the payload
  2. Following error are raised during Carrier Invoice creation:
    /SCMTMS/SFIR 005 “Enter freight settlement document type”
    /SCMTMS/SFIR 148 “Document type does not belong to category 50”

    Cause

    Error is raised due to missing customization of the FSD type for Carrier Invoice (05) and the Default type flag not set

    Resolution

    1. Open the customizing path: SPRO -> Transportation Management -> Settlement -> Freight Settlement -> Define Freight Settlement Document Types.
    2. Create or adjust the required FSD type so that:
      The document category is set to Carrier Invoice (05).
      The Default type flag (default type) is set.
    3. Save the changed and re-run the interface InvoiceRequest_In
