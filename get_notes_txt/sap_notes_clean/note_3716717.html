Symptom

When attempting to preview a correspondence document in the 'Create Correspondence' app, the system fails to generate the PDF and displays one or more of the following error messages:

  • Output Management could not generate the requested PDF document.
  • FIN_CORR/342 Output Management could not generate the requested PDF document.
  • CX_FI_CORRESPONDENCE Form template name can't be initial.
  • CX_FI_CORR_REQUEST_ERROR Output Management could not generate the requested PDF document.
  • /IWBEP/CX_MGW_BUSI_EXCEPTION Could not retrieve the PDF document from the Output Management.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Navigate to the 'Create Correspondence'(F0744A) app.
  2. Create or execute a new correspondence with identifier SAP09.
  3. Open the PDF Preview tab.
  4. The system displays the error messages and the PDF is not generated.

Cause

Incorrect configuration in 'Output Parameter Determination'(OPD) app.

Resolution

  1. Open the 'Output Parameter Determination'(OPD) app.
  2. Locate the Form Template Determination step for the relevant internal document.
  3. Remove the parameter setting: variants=" ".
  4. Then change the Correspondence Identifier from SAP09 to S09LE (Ledger View) or S09EN (Entry view) based on the requirements.
  5. Save the configuration.
  6. Test again by previewing the PDF in the Create Correspondence(F0744A) app.
