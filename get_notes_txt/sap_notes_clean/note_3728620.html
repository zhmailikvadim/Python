Symptom

Setting Up SAP Ariba Buying

SAP Ariba Buying requires certain configurations to be completed before you add items to a cart and create a purchase request.

Other Terms

SAP Ariba Buying, SAP Ariba Buying, base edition, procurement expansion package

Buying Profiles, Purchase Request

Reason and Prerequisites

In some cases, you might encounter errors while setting up SAP Ariba Buying. The following SAP Note provides information in setting up SAP Ariba Buying.

Solution

General

When setting up SAP Ariba Buying, if you do not see the tile SAP Ariba Buying Suite on the My Product Packages tab, please raise an incident using the component S2P-P2O-SET.

New or Existing Customers

This SAP Note provides information about setting up SAP Ariba Buying for new and existing customers.

For more detailed information about setting up, see Initial Tasks in Setting Up SAP Ariba Buying with SAP for Me.
