Symptom

It is observed that BOM components marked with the Fixed Quantity indicator in S/4HANA are not appearing in the IBP PDS Component after RTI execution.  Only non–fixed-quantity components are transferred.  No error or queue issue occurs during RTI.

Environment

SAP Integrated Business Planning

Reproducing the Issue

  1. In CS02, set the component’s Fixed Quantity checkbox.
  2. Run CFM2 or CFM9 to transfer data to SAP IBP.
  3. Observe that the Component is not transferred for components flagged as Fixed Quantity, while non-fixed components are transferred.

Cause

Currently, the integration of BOM components with the Fixed Quantity indicator is not supported by Real-Time Integration (designed behavior).

Resolution

There is an active feature request regarding this functionality on the customer influence portal.
Kindly review the details and participate in the discussion by visiting the following link: https://influence.sap.com/sap/ino/#/idea/298393
