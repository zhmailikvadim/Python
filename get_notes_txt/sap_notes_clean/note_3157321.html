Symptom

After you implement SAP Note 3051781, it is possible to place several individual postings in the release at the same time or to place an individual posting in the release even though other transactions already exist to be released.

This SAP Note ensures that this is no longer possible.

Other Terms

Workflow, VDARL-SFGBEW

Reason and Prerequisites

Program error

Solution

Implement the attached correction instructions or import the specified support package.
