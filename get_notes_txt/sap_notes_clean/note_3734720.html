Symptom

You want to migrate the HANA DB from one server to another. So, you are going to perform a Homogeneous DMO without System Update. After selecting the scenario category without stack XML file, the option "DMO without System Update" is not displayed.

Environment

NetWeaver ABAP based systems.

Reproducing the Issue

Start SUM and select the scenario category without stack XML file.

Cause

According to the DMO central Note (example as below), the supported target versions for Homogeneous DMO are listed below.
 
3620679 - Database Migration Option (DMO) of SUM 2.0 SP25
Homogeneous DMO (SAP HANA Database to SAP HANA Database)
* Supported target systems
    * SAP S/4HANA (on-premise deployment)
    * SAP S/4HANA Cloud, private edition
    * Systems based on SAP S/4HANA Foundation
 
If your current system version is out of this scope and you are performing "DMO without System Update", it is not supported for Homogeneous DMO. So, the option is not displayed.

Resolution

As described above, if the current system version is out of the supported target versions, "DMO without System Update" is not supported for Homogeneous DMO.
 
As an alternative option, please consider performing migration with a standard System Copy based on DB backup/restore method via SWPM.

See Also

For more information about upgrade/update troubleshooting, refer to our System Upgrade and Update WIKI.
