Symptom

During the delivery of a treaty (BAPI), the system incorrectly issues the error message "A distribution plan is missing for treaty <VTGNR>, generation <VTGJJ>, section <BESTNR>".

Other Terms

/MSG/R_V_VTG2

588

Reason and Prerequisites

 Prerequisites:
- Life treaty
- Existing installment schedule or reinstatement schedule
- Several accounting units for a section
- An accounting unit is marked as deleted when the treaty (BAPI) is delivered.

Reason:
- When checking the valid accounting units for a section, the system uses the accounting units to be deleted.

Solution

History:
February 19, 2026: New

Troubleshooting description:
To check whether a distribution plan exists, the system considers only undeleted entries in the accounting units, installment plans, and reinstatement schedules.
