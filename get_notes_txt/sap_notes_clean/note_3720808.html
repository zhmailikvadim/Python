Symptom

You are using the mass processing function of the Migration Workbench (MWB) to migrate data. The migration process stops after 96 hours when it reaches the maximum runtime limit.

Other Terms

 MWB, DMIS, NZDT, DTL

Reason and Prerequisites

You are using the mass processing function of the Migration Workbench (MWB).

MWB / DMIS functionality is used on one of the following releases:

  • DMIS 2018 SP11 - SP14
  • DMIS 2020 SP07 - SP10
  • DMIS 2025 SP00
  • SAP S/4HANA 2023 FPS00 - SPS05
  • SAP S/4HANA 2025 FPS00 - FPS01

Solution

 Implement the attached corrections using transaction SNOTE.
