Symptom

  • Error message: "AR103" related to Bank key length.
  • Issue occurs during the processing of outbound messages for invoices - IDOC.

Environment

•Sales And Distribution (SD)
•Logistics Execution (LE)
•SAP R/3 •SAP R/3 Enterprise 4.7
•SAP ERP Central Component
•SAP ERP
•SAP Enhancement package for SAP ERP
•SAP Enhancement package for SAP ERP, version for SAP HANA
•SAP S/4HANA
•S/4HANA Cloud Private Edition

*** The Updated value: ***

  • Product: SAP ERP
  • Outbound Messages Invoices

Reproducing the Issue

  1. Execute transaction code VF01.
  2. Display the outbound message for the sales invoice.
  3. Trigger the output (EDI message) for the affected billing document.
  4. Observe the error message "AR103" in the output condition log.

Cause

The issue occurs because the bank key number is checked against the country determined from table T012, considering the billing document company code.
If bank settings in OY17 for this country are not configured properly, message AR103 is raised.

Resolution

  1. Configure the bank key number for desired country in table T012, considering the billing document company code.
  2. Verify the settings in OY17 for country-specific configurations to ensure the correct bank key length.
