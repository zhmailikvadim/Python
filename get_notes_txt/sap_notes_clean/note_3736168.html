Symptom

The DG-1 report HPLSDG10 throws a short dump when processing a personnel number of a part-time employee whose hire date occurred shortly before the end of the reported month.
For an employee hired on the last day of the reported month, the short dump occurs only if the employee’s FTE is as small as 15%.
For an employee hired on the day preceding the last day of the reported month, the short dump occurs only if the employee’s FTE is as small as 7%.

Other Terms

GUS DG-1, HPLSDG10.

Reason and Prerequisites

Program error.

Solution

The error has been corrected.

Please apply the correction by importing the Support Package specified for your release, or by using the Note Assistant.
