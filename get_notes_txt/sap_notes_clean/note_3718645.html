Symptom

You maintain the Membership List Hierarchy and expect it to be allowed only for the specified Account Groups as per configuration. However, the system allows users to maintain the Membership at the item level for other Account Groups that are not configured for this functionality. This results in inconsistent data maintenance and behavior that does not align with the defined business rules.

This note corrects the issue by restricting Membership List Hierarchy maintenance strictly to the configured Account Groups and preventing updates for unauthorized groups at the item level.

Other Terms

/IRM/GMLM, Trade Organization

Reason and Prerequisites

 This is caused by program error

Solution

Please implement the correction instructions. These corrections are only valid if you have applied the following Support Packs.

Release V/4 –  2021, 2022, 2022V2, 2023, 2023V2, 2023 V3 and 2023 V3 Support Package 1.
