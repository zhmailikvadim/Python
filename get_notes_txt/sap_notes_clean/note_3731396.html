Symptom

  • CBC activity Define Source Application Accounting is missing.
  • Unable to access to SSCUI Define Source Application Accounting.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. In Central Business Configuration (CBC), search for SSCUI ID 105487 (Define Source Application Accounting).
  2. Observe that the configuration activity is not found in the results.

Cause

Scope item J78 is required to have access to the configuration activity Define Source Application Accounting (ID: 105487).

Resolution

Ensure the activation of scope item J78, refer to Advanced Cash Operations (J78).

See Also

Refer to: Scope item activation request for SAP Central Business Configuration
