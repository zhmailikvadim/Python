Symptom

  • An anonymized task appears in To-Do or Home Page Cards related to Employment Profile Selection. The task does not display the candidate’s name (anonymized).
  • A rehire-related task remains open (stale) and needs to be completed or closed.

Environment

SAP SuccessFactors Onboarding

Resolution

Follow the steps below to identify the candidate associated with the anonymized task and close any stale rehire workflow task:

  1. Open the anonymized task > Copy the URL of the task.

  2. Paste it into a text editor for easier review.

  3. Locate the following parameters in the URL: Example: triggerId=a1b2c3d4e5f7g8h9

  4. Navigate to Manage Data > Search for Process Trigger > Paste the copied Trigger ID > Locate the BPE Process Instance ID > Copy this ID.

  5. Go to BPE Monitor Process > Filter by the Process Instance ID > Identify the unfinished task. Note: The rehire task may not appear in this initial filtered view.

  6. Copy the Start Time of the unfinished task. Update the filter: Start Date After: One day before the identified start time / Start Date Before: One day after the identified start time > Apply the filter.
  7. Identify the Rehire Workflow: Use Ctrl + F to search for the copied start date/time. > Locate another unfinished task corresponding to the Rehire workflow. > This is the impacted task > Copy the Process Instance ID (first column).

  8. Complete the Stale Task: Navigate to BPE Manage User Tasks > Filter using the copied Process Instance ID > Select the Rehire Verification Task > Complete the task.
