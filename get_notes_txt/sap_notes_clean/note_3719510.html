Symptom

You use the 'Manage Corporate Income Tax Hierarchies' app to maintain classification rules. You want to add Journal Entry Field node with Equal To or Not Equal To options with empty Value From but the system doesn't allow you to do it because Value From field is mandatory and should contain at least one symbol.

Other Terms

F7885, Manage Corporate Income Tax Hierarchies, F5732, Manage CIT Classification, Corporate Income Tax, CIT, CIT_CLASSIFICATION

Reason and Prerequisites

Incorrect program behavior

Solution

Install the appropriate Support Package mentioned in this SAP Note or implement the Correction Instructions and Manual Activity.

The SAP Note makes the Value From field optional. To configure the rule 'equal to empty' select Equal To option and keep Value From field blank. Pay attention that values like '', "", ' ', " " are considered as separate values and they are not empty values.
