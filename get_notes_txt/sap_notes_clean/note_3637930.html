Symptom

The following error occurs when generating an External SAP HANA View for Queries:

>>> External SAP HANA view: Unsupported time restriction for non-cumulatives  (Message No. RS2HANA_VIEW091)

Environment

  • SAP NetWeaver 7.5
  • SAP BW/4HANA

Reproducing the Issue

  1. Selection option ''External HANA View' in Query designer.
  2. Generate the Query.
  3. Check the Query.

Cause

In versions following those mentioned in SAP Note 2233537, when generating an External HANA View for a query with Non-Cumulative key figures and the query includes a variable filter on Time characteristics (e.g., 0CALDAY, 0CALYEAR, etc.), then the query must contain at least one additional Time characteristic that meets the following two conditions:

  • There is a constant filter. (e.g. [1,2] or [100,200], etc.)
  • There is no variable filter. 

Resolution

Add a constant filter in another Time characteristic, and make sure that there i no variable filter on it.
