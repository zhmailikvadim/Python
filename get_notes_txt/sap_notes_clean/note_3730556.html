Symptom

When attempting to generate a PDF based on a document from SAP Business One Sales app, the following error message is displayed:

Processing failed

java.lang.RuntimeException: Bad URL /sap//IMCC/srv/pa/crystal/LoadCR?DocCode=xxxx

Environment

SAP Business One, version for SAP HANA

Reproducing the Issue

Prerequisites:

  • The SAP Business One Sales app is in use.

Scenario:

  1. Log in to the SAP Business One Sales app.
  2. Select a document such as a sales order, and generate a PDF report.

Expected behavior:

The PDF report is displayed correctly.

Actual behavior:

The following error message is displayed:

Processing failed

java.lang.RuntimeException: Bad URL /sap//IMCC/srv/pa/crystal/LoadCR?DocCode=xxxx

Cause

The Analytics Platform URL is incorrectly configured.

Resolution

1. Verify the Analytics Platform URL https://xxxx:port/Enablement is accessible from the mobile browser.

2. If logging in to SAP Business One Sales app using an external address, ensure the external address of the Analytics platform is maintained under System Landscape Directory > External address tab.

Keywords

bad URL, PDF, report, processing failed
