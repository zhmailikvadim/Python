Symptom

Creation or Activation of an archiving process (DAP) for an aDSO fails with error message:

Error in structuring definition for archiving object BWXXX
Message no. RSARCH154

Error when activating Data Archiving Process XXX
Message no. RSO404

Could not activate data archiving process
Message no. RSDA506

 

Environment

  • BW/4HANA
  • SAP NetWeaver

Cause

In transaction RSDANLCON no nearline connection has been defined. Therefore, the system tries to create a DAP as ADK, which is not support for aDSOs, only NLS is supported.

Resolution

Define a nearline connection in transaction RSDANLCON.

Further information on how to set up a NLS connection can be found in the online documentation: Creating Near-Line Storage Connections
