Symptom

  • Use SAP HANA Smart Data Integration(SDI) with SAP Integrated Business Planning for Supply Chain(IBP) to integrate with External System
  • License required for SDI?
  • One DP agent required for each IBP instance?

Environment

SAP HANA Smart Data Integration
SAP Integrated Business Planning for Supply Chain

Resolution

  • License is not required for SDI, check detail setup in help document
  • One DP agent installation is required for One IBP instance

Keywords

DP Agent, SDI, IBP, file, bapi, license, DP agent
