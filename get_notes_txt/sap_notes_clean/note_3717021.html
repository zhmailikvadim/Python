Symptom

When repointing a Web Intelligence report, qualification of object gets changed and associated merged dimension objects get unmerged.

Other Terms

merge, web intelligence, BI Laucnhpad, 4.3, webi, repoint

Reason and Prerequisites

Environment:

  • SAP BusinessObjects Business Intelligence Platform 4.3 Support Package 05 Patch 03
  • Web Intelligence (WebI)
  • All Supported Operating Systems

Steps to Reproduce:

  1. Log in to BI launchpad.
  2. Create a Web Intelligence report with 3 queries with one query pointing to efashion Universe (.unx) & others based on FHSQL (using eFashion connection).
  3. Add 3-4 objects to each query, including "Month" as a common object
  4. Ensure Month is set as a dimension in all queries
  5. Merge the Month objects across queries
  6. Perform "Change Source" operation.
    >> Notice that after changing source month object changes from dimension to measure & month object becomes unmerged.
