Symptom

  • Cannot edit threshold ranges from the measure label cell.

Environment

SAP Analytics Cloud

Reproducing the Issue

  1. Open the SAC story based on the BW live model.
  2. Attempt to edit threshold ranges from the measure label cell.
  3. Threshold option is grayed out .

Cause

This issue was caused by a product bug in SAP Analytics Cloud.

Resolution

The issue has been fixed.
The correction is available starting from 2026 QRC1 patch.

Refer to: 3671358 - How to improve performance BW thresholds on 2nd structure in SAC.
