Symptom

  • After creating a custom metric, when trying to apply "Drill Down By" settings located under "Calculation Settings", the setting is grayed out and is not possible to set. 
  • The issue occurs regardless of the initial configuration of the unit of measure for the metric.

Environment

Sustainability Control Tower

Reproducing the Issue

  1. Create a custom metric in the "Manage Metrics" app. 
  2. Edit the metric and attempt to apply "Drill Down By" settings. 
  3. Observe that the option is grayed out. 

Cause

When custom metrics use calculation definitions, the calculation variants used in the calculation need to use measures that have SAP or Custom dimensions in common. If measures are used that have no common dimensions across each measure, it will result in the "Drill Down By" section being grayed out. 

For example, if there are two calculation variants, one which is using measure "Headcount Covered by Region" and the other which is using measure "Headcount covered by worker representatives", the "Drill Down By" setting will work as each of these measures have dimensions in common such as "Absence Type", "Age Group", "Business Location", etc.

Resolution

To use the "Drill Down By" setting, make sure the measures used in the calculation variants have dimensions in common. 

To verify dimensions used by the measure, please do the following: 

  1. Open "Manage Measure" application
  2. Compare the dimensions used by each measure. Search for the measures used in the calculation variant and compare the dimensions under each measure.
