Symptom

In QPC transactions, when a work center is assigned to an operation having control key reference indicator is marked, user can change the control key for the operation. However, these changes are not retained when the task list is saved.

Ideally, users should not be able to modify the control key when reference indicator is marked in work center.

Other Terms

Task List, Inspection Planning, QPC02, QPC, Control Key, Work Center, Edit Control Key, Control Key Reference indicator, STEUS_REF, LINSPECTION_PLANF01, LINSPECTION_PLANF09.

Reason and Prerequisites

Program error.

Solution

Implement attached corrections.
