Symptom

When you execute the transaction for Sales Rebate (/IRM/IPCRM) and Claims (/IRM/GCRM), you find the change log documents sorted from oldest dates to newest dates.

Other Terms

/IRM/GCDM, Sales Rebate, Claim, Change log, IPCRM, GCRM, Sorting

Reason and Prerequisites

This is caused by the program error.

NOTE: For V/4 release please implement SAP Note#3040046 (From V4 1610 Base to 2020 Base pack)

Solution

Please implement the correction instructions. These corrections are only valid if you have applied the following Support Packs.

Release 60E - Support pack 3, Support Pack 4, Support pack 5, Support pack 6 and Support Pack 7

Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object.
