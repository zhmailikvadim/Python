Symptom

This SAP Note solves the following issues:

  • You already entered the 3-character Incoterms code, but still getting this error message:

         Incoterms must not exceed 3 characters.

  • In a billing document, when entering a tax exemption reason on the Item Texts tab in the item data, you select a language different from the company code's language. Later, when you attempt to display the electronic invoice in XML format or submit it, the following error message appears:
    Provide tax exemption reasons in the source document.

Other Terms

e-invoice, eInvoicing, e-Invoicing, electronic invoice, electronic invoicing, Malaysia eInvoice, MY eInvoice, e-document, eDoc, e-doc, document compliance, digital compliance, DC, document and reporting compliance, electronic document processing, IRBM, LHDN, Inland Revenue Board of Malaysia

Solution


Version Date Changes
1 March 12, 2026 Released for customers

Import the Support Package specified for your release or implement the attached correction instruction.

As a general rule, SAP recommends that you implement the Support Package specified for your release. However, if you need to install a solution earlier, use the SAP Note Assistant to implement the correction instruction. For more information, see SAP Note Assistant.
