Symptom

What is virus scan profile /EHFND_RAP_GRAPHIC/VALIDATE_BINARIES?

Environment

  • SAP ERP Environment, Health and Safety (EHS)
  • SAP R/3
  • SAP R/3 Enterprise
  • SAP ERP Central Component
  • SAP ERP
  • SAP Enhancement package for SAP ERP
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Resolution

Virus scan profile /EHFND_RAP_GRAPHIC/VALIDATE_BINARIES is part of S/4HANA for product compliance. The virus scan profile checks if a binary is a valid SVG graphic and does not contain any embedded script.
