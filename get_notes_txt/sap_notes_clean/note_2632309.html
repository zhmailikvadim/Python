Symptom

The  customizing Maintain Conversion Trigger is used to define a conversion mode and configuration ID for a document type and workstation application. However, the key "Doc type + Sequence + Application" however does not allow the precise identification of one entry for certain CAD models like NX where the part and the assembly files are stored into the same document type and WS application. In such case, when a conversion is triggered for part the assembly conversion job also gets created with VEG. This results in two parallel jobs getting created.

Other Terms

Duplicate VEG jobs, conversion, VEG, 2396742

Reason and Prerequisites

Some corrections were mssing in S4CORE 100 release system in the note 2396742.

Solution

Implement the attached correction instructions.
