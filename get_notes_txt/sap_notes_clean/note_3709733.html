Symptom

With transaction code TPM44 accruals are calculated for interest rate instruments using conditions with the interest calculation type:

  • Average linear interest calculation
  • Average compound interest calculation

If the interest amount of a flow is accrued which has a calculation period longer than one day, the interest amount is accrued linearly. Instead the amount for the accrual must be calculated using the given interest calculation type.

Other Terms

550, 580, risk free rates, SOFR, SONIA, TONA, ESTER

Reason and Prerequisites

Programming error

Solution

Implement the correction instructions with transaction code SNOTE. If prompted, implement the manual corrections instructions.
