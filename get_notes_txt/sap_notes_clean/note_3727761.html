Symptom

You use the budgeting workbench (transaction code FMBB) with the accessibility mode active. You observe that when selecting 'Carry Over', or 'Distribution', 'Revenues Increasing the Budget', the process value switches to 'Transfer' process value.

Other Terms

Usability improvement, accessibility mode is active.

Reason and Prerequisites

Side effect of SAP Note 3387752 - FMBB: Group header fields, accessibility mode is active.

Solution

After applying the following correction, when the accessibility mode is active, the selected Budgeting process is kept in the field area of the screen. The following SAP Note will correct the side effect of SAP Note 3387752 - FMBB: Group header fields, when accessibility mode is active.
