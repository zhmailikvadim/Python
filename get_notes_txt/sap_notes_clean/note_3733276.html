Symptom

When using a Formatted Search (FMS) on a Sales Order (SO) line, an error is triggered for an assembly-type Bill of Materials (BOM). 

Error: "Value in Quantity Cannot Be Zero". 

Reproducing the issue

  1. Create an Assembly-type BOM.
  2. Create a FMS with the following properties:
    • Query: SELECT $[$38.12.11] = 'Y'
    • Assign to Field: Inventory UoM on sales order
    • Auto-Refresh When Field Changes: Item No.
  3. Create an SO for the assembly BOM.
  4. Press the Tab key to move focus away from the Item No. field.

Expected Result

No error should occur.

Actual Result

The error message appears "Value in Quantity Cannot Be Zero".
