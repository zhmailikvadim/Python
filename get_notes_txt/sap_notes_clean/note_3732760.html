Symptom

You are working with the app "W0075 - Statistical Key Figures - Actuals" or with a custom query based on the CDS I_StatisticalKeyFigureItem

The SKF data created for other objects than "Cost Center" are not shown.

Other Terms

authorization, W0075, I_StatisticalKeyFigureItem, DCL

Reason and Prerequisites

The CDS cube I_StatisticalKeyFigureItem contains the DCL based on the authorization object K_CSKS. If the user has the authorization only for certain cost centers, but not for empty cost center, then the data from the cube are filtered out. This is often the case when SKF data is created for other objects than "Cost Center" (like WBS element, Order, etc)

Solution

 Apply the attached correction or relevant SP
