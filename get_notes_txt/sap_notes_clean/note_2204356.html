Symptom

You are encountering the error message below when you run /CAR/ACTIVATE_HANA_CONTENT in a new system installation:

Table SAP_CAR:/CAR/CONDTYPEMAP not found.

Table SAP_CAR:......

......

The message lists all tables with authoring schemas SAP_CAR, SAP_DDF, SAPOSA.

Other Terms

schema,SAP_CAR,SAP_DDF,SAPOSA

Reason and Prerequisites

The activation program was creating the schema mappings after the table check. This causes this issue in a new system installation.

Solution

Please apply this note or the relevant release.
