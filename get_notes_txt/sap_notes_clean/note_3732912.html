Symptom

  • SAP S/4 HANA - Transaction code CKMLCP shows dump in post closing step in ABAP program CL_FCO_COGS_SPLIT_DOCUMENT====CP.
  • Runtime Errors ASSERTION_FAILED in method SET_ITEMS of class CL_FCO_COGS_SPLIT_DOCUMENT====CM00C.
  • The main program was "FCML4H_POST_CLOSING". 

Environment

  • SAP S/4 HANA
  • Controlling (CO)

Reproducing the Issue

  1. Enter transaction CKMLCP (costing run).
  2. Execute the post closing step.
  3. ABAP programming error appears.

Resolution

  1. Implement SAP Note 3697462 ASSERTION_FAILED in method SET_ITEMS of class CL_FCO_COGS_SPLIT_DOCUMENT.
  2. If the above note doesn't resolve your issue, please contact SAP Support and check if pilot notes 3713152 or 3310168 are relevant.
