Symptom

  • Transaction IWBK does not return results when searching with material number.
  • Searching with a combination of order number and material number also does not return results.
  • Searching with order number alone returns results as expected.

Environment

  • SAP R/3
  • SAP R/3 Enterprise
  • SAP ERP Central Component
  • SAP ERP
  • SAP Enhancement package for SAP ERP
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA, on-premise
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Execute transaction IWBK with an order number --->  Material availability list is displayed.

  2. Chose a material number from the list, then return to the selection screen.

  3. Enter the selected material number in the 'material' field (SERMAT), then execute IWBK again. ---> Material availability list is empty

User is expecting to see orders where the selected material is assigned as a component.

Cause

Expected behavior:

The 'material' field (SERMAT) is associated with the technical object within the order (Equipment Construction type material). It is not relevant for the materials (components) reserved for the orders.

Resolution

Use the 'Material (reservation)' (MATNR) field to search orders based on the assigned components.
