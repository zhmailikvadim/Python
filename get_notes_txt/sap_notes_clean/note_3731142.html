Symptom

In the latest version of DELTA I/E, SAP has not provided PDF Form Layouts for EAD, Proof of Exit, etc.

Reproducing the Issue

Cause

DELTA I/E is implemented as a fully dematerialized customs process; therefore, new dedicated printable PDF forms for DELTA I/E are not provided by design.

Resolution

  1. Use the Movement Reference Number (MRN) to reference shipments; customs authorities access the declaration data directly in their systems.
  2. Where a readable document is operationally needed, print the technical XML messages exchanged with the customs authorities, including their responses, and provide these as needed.
  3. Monitor regulatory updates for DELTA I/E to verify any future changes that could introduce printable artifacts.
  4. Please refer to SAP Note 3661630 for the latest changes related to DELTA Export and SAP Note 3552772 for DELTA Import.
