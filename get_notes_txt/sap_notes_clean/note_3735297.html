Symptom

In the process "Receipt of Activation of Assignment Authorization (DSO View)" (DE_B_EQS_ASSAUTH_ACTV_DSO), the process stops at step 0200 and does not proceed to step 8950.
This issue is relevant for market role DSO.

Other Terms

EQS, Settlement, MaBiS, DE_B_EQS_ASSAUTH_ACTV_DSO, precondition, PDoc

Reason and Prerequisites

The system stops the process because an incorrect precondition is configured for step 8950.

Solution

Implement this SAP Note or import the relevant support package.

After implementing this SAP Note, the system corrects the precondition for step 8950 and completes the process successfully. Import the BC set /US4G/3_0_DSO_SP01_NOTE_3735297. Run the affected PDoc again. To verify the correction, confirm that the PDoc completes successfully.
