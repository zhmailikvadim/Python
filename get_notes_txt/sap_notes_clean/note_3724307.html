Symptom

Running SAP Disclosure Management 10.1 with client PCs using:

  • Microsoft Office 2016
  • Microsoft Office 2019

Other Terms

Word, Excel, PowerPoint

Reason and Prerequisites

The software vendor providing Microsoft Office 2016 and Microsoft Office 2019, Microsoft Corporation, has ended support for these Office versions by Oct 14, 2025. Technical support or security patches are no longer provided and Microsoft does not recommend to continue to use these Microsoft Office versions in production. (Source: https://www.microsoft.com/en-us/microsoft-365/office-2016-2019-end-of-support)

SAP adjusts to this by discontinuing support for the installation and usage of SAP Disclosure Management 10.1 using the named product versions.

Solution

Running SAP Disclosure Management 10.1 with client PCs using

  • Microsoft Office 2016
  • Microsoft Office 2019

will be no longer supported from Q4/2026.

 

SAP advises all customers to update to

  • Microsoft Office 2021 or newer

 

Note:
Microsoft Office 2021 and newer versions are supported both under the classic Microsoft license model and the newer Microsoft 365 license model - no cloud features supported. The product Microsoft Office 365 (cloud) is not supported.
