Symptom

While creating MBS pegging in nomination, LDC ticket remains in partially processed status.

Other Terms

ACM; Agricultural Contract Management; Nomination; MBS pegging; LDC

Reason and Prerequisites

Program error.

Solution

Import the support pack or implement the attached correction instructions.

Below are the steps followed to test the scenario:

1) Create a Nomination with 3 contracts.
2) Peg line 10,20 and 20,30 with MBS pegging.
3) Create LDC load event.
4) Create 4th contract.
5) Add 4th contract on nomination.
6) Peg line 30,40 with MBS pegging.
7) Ticket for third contract should be completely processed.
