Symptom

The "created by" field displays only userid and not the full name.

Other Terms

GRAC 10.1 ARQ, ACCESS REQUEST, Search, CREATED BY, Not, Showing, Full Name.

Reason and Prerequisites

The access request search does not show the full name in "Created By" column for the user created through End User Logon. This is because the end user logon id was not considered while searching the user details of the "Created by" relating to the End User logon id.

Solution

Please implement the correction instruction.
