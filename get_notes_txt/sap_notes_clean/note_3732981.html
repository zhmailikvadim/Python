Symptom

  • When importing a newer version of a software collection to production system, the import status shows "Outdated" for several earlier software collection versions.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Import a newer version of a software collection to the production system.
  2. Open the import/execution log or the software collection list in production and observe that earlier versions of the same collection are now marked with status "Outdated."

Cause

The "Outdated" status on earlier software collection versions occurs because a newer version that includes all items was successfully imported. This is expected standard behavior when exporting all items from a software collection.

Resolution

  1. Verify that the latest software collection version was successfully imported and contains all required items.
  2. Review the execution messages for the latest import to confirm that earlier versions were set to "Outdated" as part of standard processing.
  3. No further action is required for the older versions; continue using the latest imported version.
