Symptom

You are running a WD ABAP application. During runtime you open the right mouse click context menu. In that special case additional entries have been added to the standard WD menu items which will be visible in the context menu. After that you click in any "empty" region aside the place where you opened the context menu before. Then a dump OBJECTS_OBJREF_NOT_ASSIGNED occurs.

Reason and Prerequisites

Programm error

Solution

Please implement the attched correction instruction or the corresponding support package.
