Symptom

• Tasks with a due date set to today or a future date do not appear under the "My Upcoming Activities" section on the Home screen.  
• Appointments with a future start date/time are displayed correctly.  

Environment

  1. SAP Sales Cloud and SAP Service Cloud Version 2
  2. SAP Sales Cloud Version 2
  3. SAP SAP Service Cloud Version 2

Reproducing the Issue

1. Navigate to any Opportunity.  
2. Create an Activity of type Task.  
3. Set the Due Date to Today or a Future Date.  
4. Save the activity.  
5. Navigate to the Home Screen.  
6. Check the "My Upcoming Activities" section.  
7. Observe that the Task is not displayed.  

Cause

  • The "My Upcoming Activities" card is designed to display only calendar-based activities.
  • Calendar-based activities include:  
    • Activities with a future Start Date/Time. 
    • Activities that are part of the user's calendar. 
  • Tasks: 
    • Are based on Due Date. 
    • Do not have a Start Date/Time. 
    • Are not treated as calendar entries.  

Resolution

1. This is the expected system behavior.  
2. No configuration or standard setting is available to include Tasks in the "My Upcoming Activities" section.
