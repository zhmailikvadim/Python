Symptom

After updating field MARC-MAABC by running Transaction MC41 (program RMCBAB25) change logs in transaction MM04 are missing.

The problem occurs in a S4HANA-system where the retail switch ISR_RETILSYSTEM is active, but actually there are no retail articles present in table MARA.

Other Terms

MC41

Reason and Prerequisites

Program error.

Solution

The attached correction solves the problem.
