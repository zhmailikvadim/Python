Symptom

There has a requirement to print the document to a specific tray. 

This article provides the setting example.

Environment

Version Independent.

Resolution

Accoriding to the KBA below, it is possible to set the trace by changing the XDC file.

0002111085 - Tray control in SAP

Here are the steps:

  1. Downloading the XDC file by running report RSPO0022.
    See KBA : KBA 0003409776 - XDC file editing example 
  2. Click on "XDC Administration". In next screen, select the current used XDC, for example hppcl5e.xdc , and click on Download XDC file to save the file to desktop. 
  3. Open the XDC file with NotePad, and add the command below in "<deviceInfo></deviceInfo> " node.
          <inputTrays>
                    <inputTrayDefinition name="Tray1" medium="<PaperTypeInTemplate>" trayType="Preprinted" trayNumber="1"/>
          </inputTrays> 
  4. Make sure to change the tray number accoridingly.
  5. Make sure the name and id in xdc node is unique. 
    See KBA : 0003381875 - Same parameter generate different print files 
  6. Save the XDC file with a customized name start with Z.
  7. Upload the XDC file in by running report RSPO0022(XDC Administration)
  8. Assign the XDC file to the device type.
    See KBA : 0003348871 - Assign XDC file to Device Type 

If the input tray is not clear, it is necessary to check with printer vendor.

During test, it is possible to run report RSPO0020 and input the spool number. Then in next screenshot, check the print file about the Tray related command.

For example :

  • &l4H for tray 1
  • &l1H for tray 2
  • &l5h for tray 3
  • &l8h for tray 4
