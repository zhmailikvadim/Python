Symptom

When attempting to preview or print journal entries in the 'Accounts Document Output -China' app, it does not function properly.

Environment

SAP S4/HANA Cloud Public Edition

Reproducing the Issue

  1. Open the Accounts Document Output -China app.
  2. Enter the document ID, fiscal year, company code and other parameters, or simply select the document type.
  3. Run the process.

Cause

This is a known error that was identified after the 2602 release.

There are problems when printing the following journal entries:

  • Asset depreciation postings
  • Ledger specific postings
  • Foreign currency valuation postings
  • CO internal postings

Resolution

A hotfix is being actively prepared resolve this issue. The resolution is planned for release by March 15th.
