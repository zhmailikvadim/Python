Symptom

You are using the Simple System Integration app in SAP Focused Run (FRUN) to configure Integration & Exception Monitoring. 

While auto assigning Configuration Variants, you observe an error in the Configuration Status with the message Configure Integration Monitoring finished with error. This occured when the system found a configuration variant that had no monitoring filters defined.

Other Terms

SAP, FRUN, AIM, Integration & Exception Monitoring, Configuration Variant, Simple System Integration, Configure Automatically, Configure Integration Monitoring finished with error

Reason and Prerequisites

The issue occurred because of a programming error.

Solution

Implement the attached correction instructions.

After the fix, such configuration errors will be classified as Warnings in the Simple System Integration App.
