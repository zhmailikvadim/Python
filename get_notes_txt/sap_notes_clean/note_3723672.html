Symptom

There is uncertainty about whether the migration to SAP Cloud Identity Authentication Service (IAS) affects external users who are created and managed directly within SAP SuccessFactors Learning.


Environment

SAP SuccessFactors Learning

Resolution

  1. External users created and managed directly within SAP SuccessFactors Learning, who do not access the system via login, are unaffected by IAS migration.
  2. IAS migration impacts only users who authenticate through the login process, and external Learning users are excluded from this authentication flow.
  3. The current method for creating and managing external Learning users within LMS remains unchanged and is fully supported.
  4. No additional actions are required regarding these users.

Keywords
