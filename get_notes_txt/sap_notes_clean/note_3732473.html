Symptom

When creating a purge request, the legal entity for which you want to purge users from is not appearing in the search.

Environment

SAP SuccessFactors HCM Core

Reproducing the Issue

  1. Navigate to Data Retention Management > Create new purge request > Select DRTM Master Data Purge.
  2. Attempt to search for the legal entity.
  3. Observe that no legal entity is available to select as expected.

Cause

The Legal Entity is inactive. Inactive legal entities cannot be selected for DRTM purge requests.

Resolution

  1. This is expected behavior
  2. If a purge must be performed for users associated with an inactive legal entity, temporarily set the legal entity to Active and run the purge.

See Also
