Symptom

This SAP Note provides the enhancement for the retroactive calculation of severance payment.

Below is an example scenario:

  1. An employee left the company in 2025/02.
    The Severance Payment (new) MBSP wage type had been maintained with a value of 10,000 in the Additional Payments (0015) infotype.
  2. The regular payroll run of 2025/02 was run (/491 = 10,000). 
  3. An allowance wage type was maintained in the Additional Payments (0015) infotype in 2025/02. 
  4. In 2025/03, the regular run of March was run. And the retroactive run for February was triggered.

In the previous solution, the amount in the Taxable Severance Payment (/491) wage type was zero in the period 2025/03 for 2025/02.

Now after you apply this SAP Note, when you run the Payroll Driver, Version China (HCNCALC0) report for the employee above, the amount of /491 in the retroactive period of February will be 10,000 instead of zero in the IT and RT tables.

Other Terms

CNL5, CNL9, /491

Reason and Prerequisites

This is an enhancement. You have applied the SAP note 3644114.

Solution

Do either of the following:

  • Apply the Support Package.
  • Carry out the manual activities described in the attachment ManualSteps3705811.pdf.
