Symptom

The release of a maintenance order with many externally procured components exceeds the processing time limit, causing the transaction to be aborted. Therefore, the order cannot be released. The system aborts the release with the runtime error TIME_OUT.

Reason and Prerequisites

The STXH table is read very frequently in MM when an order is released. This note reduces the read operations on this table is significantly.
