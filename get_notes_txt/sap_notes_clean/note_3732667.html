Symptom

In Sales order item, tag COPA segment,  'order' not populated for sales order number. 

Environment

  • Controlling and Profitability Analysis (CO-PA)
  • SAP ERP
  • SAP ERP Central Component
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA Finance
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Cause

The issue results from expecting the sales order to populate CO-PA characteristic 'order'-RKAUFNR. In standard, RKAUFNR refers to internal (CO) orders; in SD flows the sales order reference is stored in ACDOCA field KDAUF and does not appear as RKAUFNR.

Resolution

Do not expect the sales order to populate the CO-PA characteristic “Order (RKAUFNR)” in SD-driven profitability segments; this characteristic is for internal (CO) orders.

For SD billing/accounting documents, usually verify the sales order reference in ACDOCA field KDAUF.
