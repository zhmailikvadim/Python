Symptom

Synchronization between Joule System and SAP Ariba Buyer System fails with below error:

Synchronization between Joule System [SYSTME_NAME] and SAP Ariba Buyer System [SYSTEM_ID] has failed.
– The facilitator SAP Cloud Identity Services Facilitator working on behalf of SAP Ariba Buyer System [SYSTEM_ID] returned a technical error for assignment with ID [ASSIGNMENT_ID]:
"An error has occurred during assign formation notification processing. Correlation Id: [CORRELATION_ID]. IAS operation has failed with code: OPERATION_FAILED and message: An error occurred while modifying application with id [APPLICATION_ID] in tenant. IAS Response: Invalid user input. Duplicated consumed api names are not allowed."



*Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.  

Environment

  • Joule
  • SAP Ariba Buyer System

Resolution

  1. Access the IAS tenant using an admin account.
  2. Navigate to the Joule application (the same one used in Formation).
  3. Go to Trust → Application APIs → Dependencies
  4. Delete the dependency corresponding to the application SAP Ariba Buyer System [SYSTEM_ID]
  5. Resynchronize the formation.

Keywords
