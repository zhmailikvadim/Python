Symptom

 Include a new version of the jgit library.

Reason and Prerequisites

Adding libraries.

Solution

Delivered with the latest version of KERNEL for kernel release 793.

It can be found in the SAP software center under the component SAP KERNEL 7.93 64-BIT UNICODE and is available as of the kernel with patch level 350.

It can also be downloaded and installed separately, as it is provided as separate patch: abap2vcs_350-70007807.sar

It can be installed without restarting the system, but it must be exchanged in both executable directories of each application server (similar to exchanging the transport tool R3trans, see note 19466).
