Symptom

When comparing the line items and total values of Fiori app F0963 "Cost Centers - Actual" and app F2217 "Display Line Items in General Ledger" there are differences.

Environment

  • Controlling (CO)
  • SAP S/4HANA 2022
  • SAP S/4HANA 2023
  • SAP S/4HANA Cloud Private Edition
  • SAP Fiori for SAP S/4HANA 2022
  • SAP Fiori for SAP S/4HANA 2023

Reproducing the Issue

  1. Access app F0963 - Cost Centers Actual
  2. Insert filters: cost center hierarchy, node, G/L hierarchy, period etc.
  3. Check amounts displayed
  4. Select the "navigate to" button
  5. Select "display line items in general ledger"
  6. Amount shown is different

Cause

The selection from one application is not completely taken over to the other application.

Resolution

Implement SAP Note 3521763 - Navigate to issue in Web Dynpro Grid application

Keywords

fiori, cost center, actual, display line items, general ledger, navigate to, line item, totals, selection, application, hierarchy, node, filter, F0963, F2217
