Symptom

You use transaction HRPBCM ("Change Personnel Budget Plan Management") to edit a budget element ("BU Element"). However, you notice the following problem on the "Basic Data" tab page (PBC_BAS).

  • In the combo box for the "Budget Structure Element Type" field , the system does not display all values that you would expect based on the settings made in the IMG activity PBC_FTE_0050 ("Maintain Hierarchy of Budget Structure Element Types"). 

Other Terms

PBC_BAS; scenario PBC_; IT1500

Reason and Prerequisites

This problem occurs in particular if the BU element is assigned to different BU elements over time.

Solution

Implement the attached correction instructions or import the specified Support Package.
