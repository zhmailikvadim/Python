Symptom

  • Multiple Users got one same item reassigned.
  • In Learning Plan Audit report, some have reassignment records while others don't.

Environment

SAP SuccessFactors Learning

Reproducing the Issue

1. Learning Administration > Item entity > Edit the Online Content Settings > Save

2. Reassignment Assistant is triggered > Reassign the item to all the users in the list

3. Run User Learning Plan Audit report for these users > Some have reassignment records while others don't.

Resolution

This is the current expected behavior.

If user has started the course, there will be CBT data to re-assign.

If user hasn't started the course, there won't be any CBT data to re-assign.

Learning Plan Audit report will only fetch reassignment records when there is CBT data.

Keywords

Reassignment Assistant, Online Content settings, NOT STARTED
