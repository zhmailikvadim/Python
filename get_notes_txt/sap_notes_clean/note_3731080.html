Symptom

The payroll driver (HFICALC0) log should display yellow warning message if the employee´s pay results produce a claim (/561 Claim) and there is no /560 net pay.

In such case, the user should validate the result and make the necessary corrections to avoid the claim, as recommended to ensure correct legal reporting.

Solution

This SAP Note corrects issues described in the Symptom section.

  • New payroll function and operation FIMSG is created to support this and future similar enhancements.
  • New maintenance view V_T7FIMSG for configurable messages is created. The respective message text with ERRID = CLM is delivered.
    • This maintenance view can be enhanced by customers by changing ERRTY to make message stronger (Error) or suppressed if not important.
  • Template schema FIND is enhanced to display a warning message when a claim was created. 

As a general rule, SAP recommends that you install a solution by applying a Support Package. However, if you need to install a solution earlier, use the Note Assistant to implement the correction instruction. You can find more information about the SAP Note Assistant in SAP Support Portal, under SAP Note Assistant.

To implement the solution, you must carry out manual activities (see the attached Manual_steps_3731080_pre.docx, Manual_steps_3731080_post.docx) and implement the correction instructions that are valid for your release.
