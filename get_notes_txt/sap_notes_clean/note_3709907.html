Symptom

The user can not edit goals in the goal plan or the performance form, despite having write permission.

Environment

SAP SuccessFactors Goal Management

Cause

The goals were created via import as read only.

Resolution

To make these read-only goals editable, update them through a goal import following below steps:

  1. Go to "import performance goal".
  2. Select the goal plan from dropdown of "Available List of Templates".
  3. Click on "Generate CSV header".
  4. In the downloaded import file, input only the below fields:
    • ^TYPE:   OBJECTIVE    
    • ACTION:    UPDATE
    • ID:   Enter the internal id of the goals (this can be retrieved from an ad hoc report with the goal management domain)
  5. Leave the rest of the fields blank.
  6. Save the file.
  7. Import performance goal (do not check the read-only option).

Try the solution on one or two goals first to check test the effect, before applying it to all affected goals.
