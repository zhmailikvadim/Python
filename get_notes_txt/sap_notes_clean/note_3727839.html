Symptom

  • I have a Custom Field that has exactly the same name as the Default Value of a Custom Text
  • The Custom Field is being replaced by the Custom Text in the Report Form page

Cause

Custom Fields with the same name as a Default Value Custom Text, will be replaced by the Custom Text on the report form page.

Resolution

Update the Custom Field name to not match the Default Value of the Custom Text.

To update a Custom Field Name: 

  1. Sign in on SAP Fieldglass
  2. Navigate to Admin Menu (Gear Icon)
  3. Click Custom Field
  4. Search and open the Custom Field by clicking on its name
  5. Click Edit
  6. Update the name
  7. Click update

Keywords

Custom Field, Report, Reporting, Custom Text
