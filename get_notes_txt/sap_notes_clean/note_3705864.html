Symptom

You click the Proforma document link in Fiori and expect the system to navigate to the Proforma document. Instead, the system navigates to the normal search Fiori screen rather than opening the Proforma document. This note corrects the behavior so that clicking the Proforma document link opens the corresponding Proforma document directly in Fiori.

Other Terms

Proforma document navigation, Activities, Email, User defined link.

Reason and Prerequisites

This is caused by program error

Solution

Please implement the correction instructions attached. These corrections are only valid if you have applied the Support Packs.

Release 6.1 - Base Package

Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object.
