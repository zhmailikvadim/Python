Symptom

Invest.id (BSEG-XINVE) is not displayed in the Down Payment (DP)  item. The screen No. is SAPMF05A 0304.

Environment

  • Financial Accounting (FI)
  • SAP ERP
  • SAP ERP Central Component
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA 
  • SAP S/4HANA Finance
  • SAP S/4HANA Cloud Private Edition
  • SAP Fiori for SAP S/4HANA

Reproducing the Issue

  1. Navigate FB03 to display an item with posting key 29
  2. BSEG-XINVE is not displayed in this screen

Cause

This is caused by field status configuration

Resolution

  1. Access transaction T-code: OB14 or OB41 
  2. Set the  field status of "Payment Term" to optional.

Keywords

invest.id, BSEG-XINVE, missing field, DP screen, special GL indicator, OB14, OB14, payment term, field status, advance payment, down payment.
