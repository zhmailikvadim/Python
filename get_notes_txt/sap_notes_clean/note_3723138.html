Symptom

When changes are made to the promotion rules in a revision such as deleting modifying the rate in the promotion grid, after the revision is saved and posted, the updates are reflected in the master agreement. However, the corresponding Agreement is not updated with these changes.

Other Terms

/IRM/IPPQM, promo_posting_status_update, Promotion Postings, pmact

Reason and Prerequisites

This is caused by Program error.

Solution

Please implement the correction instructions. These corrections are only valid if you have applied the following Support Packs.

Release V/4 – 2023 V3 and 2023 V3 Support Package 1

Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object
