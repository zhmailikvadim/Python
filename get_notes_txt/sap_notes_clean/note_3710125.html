Symptom

In the Promo Volume Plan of the Fiori Revision and Master Agreement, you may notice that the Total Baseline value does not match the Value shown in the Promotion Details tab. This Note corrects the mismatched values.

Other Terms

Baseline, Promotion, Memory Derivation, /IRM/IPMXVARM, /IRM/IPMXDM, In-Memory Derivation, No Duplicate Records Exist, Unique Data Exists, /IRM/LIPMXDMF0T, /IRM/IPMXDM, /IRM/LIPMXVARMF0T, /IRM/IPMXVARM

Reason and Prerequisites

This is caused by program error.

SAP Note #3710069 is available for V4 Releases.

Please download and implement the latest version of SAP Note #3604131 as pre requisite note.

Solution

Please implement the correction instructions attached. These corrections are only valid if you have applied the Support Packs.

Release 6.4 Base Package.

Release 6.2 Base Package.

Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object.
