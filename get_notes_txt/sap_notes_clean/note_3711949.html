Symptom

An issue related to formula data was identified in the Smart CPE SDM job CL_SDM_CPED_MIGRATE_COMPLETE, where formula input fields are not correctly migrated. The issue with SDM job has been fixed with the note 3696180 but if the SDM job is already complete before this note was released, such records are converted during Advanced-CPE adoption.

Other Terms

CPE, Smart CPE, Advanced-CPE adoption, Data Conversion.

Reason and Prerequisites

This is a program error.

Solution

Apply the correction instructions.
