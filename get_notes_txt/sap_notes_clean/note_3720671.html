Symptom

Looking for a different approver determination process.

  • The approvers needs to be updated based on the leave dates selected on the leave creation screen.
  • This change should happen before the leave request is submitted to see and change the approver according to the selected leave dates.

 

Environment

  • SAP S/4HANA Cloud Private Edition 2023
  • SAP S/4HANA All Versions 
  • My Leave Requests (Fiori 2.0 / Version 3)
  • SAP ERP

Reproducing the Issue

  • Click on Create Request.

  • Enter a future Start Date for the request.

  • The system displays all approvers based in the system date.

Cause

By standard, The approvers it is updated based the current system date. 

Resolution

This Behavior can be change Via BADI HCMFAB_B_MYLEAVEREQUESTS . Just set parameter EV_RELOAD_APPROVERS_DATECHANGE of method GET_CONFIGURATION to 'X'

Keywords

Leave Request, FIORI, Approver, HCMFAB_B_MYLEAVEREQUESTS
