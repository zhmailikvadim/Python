Symptom

Wrong message long text for VF 510.

Other Terms

VF510, VF510, V_VBRK_VKA, V_VBRK_FKA

Reason and Prerequisites

Error in VF 510 message long text.

Solution

Message long text is corrected.

The translation cannot being provide via this SAP Note. You can adjust the translation manually:
Enter SE63 in the command field (transaction field) and press Enter
Enter Object type NA in the command field and press Enter
Enter Object name VF510 in the field Object Name
Source language enUS and choose your Target language
Press Edit
Enter or correct the translation in the text field below the source text: 'V_VBRK_FKA'
Choose Save to save your translation(s)
