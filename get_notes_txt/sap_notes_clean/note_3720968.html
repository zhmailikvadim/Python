Symptom

You access a Form Bundle that contains a Transparent Table Form as a subtable with thousands of entries.
When you load the Form Bundle, the system displays the form on the user interface after several minutes.

Other Terms

FMCA_WDY_FPF, FMCA_WD_FORM_TAB_TEMPLATE, TAX RETURN, TAX REGISTRATION, TAX REFUND, SET_FORMDATA

Reason and Prerequisites

Prerequisites: You create the Web Dynpro component with the FMCA_WD_FORM_TAB_TEMPLATE template.
Reason: The set_formdata method causes a performance issue in the system.

Solution

Implement this SAP Note by following the attached correction instructions, or install the Support Package that contains this correction.
You adjust your Web Dynpro components that you created with the template FMCA_WD_FORM_TAB_TEMPLATE to use the corrected logic.
