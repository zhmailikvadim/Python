Symptom

Field within Extended Address in SAP Ariba Supplier Lifecycle and Performance needs to be set as mandatory so that user does not respond with empty value to it

Resolution

Yes, field within Extended Address can be set as mandatory. All the fields within extended address are optional by default and needs field configuration.

You can raise a case with SAP Support from SAP for ME to request the extended address fields to be set as mandatory.
