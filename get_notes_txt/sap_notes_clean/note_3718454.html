Symptom

This SAP Note is relevant for you only if your system has HR Release 6.08 or higher.

When you execute the SI: Retrieval of Data from SI Agencies program (RP_PAYDE_B2A_SV_COLLECTOR) or the Test Report for Communication with GKV and DSRV program (RPUSVHD1), the system issues the following error message:

"<KOMMSERVER>/<BNRAG_DL>: E200 - Es ist ein Fehler bei der Verarbeitung der Inhalte der eXTra-XML-Datei aufgetreten: Unzulaessige Kombination Procedure/DataType (DataType "Abholung_MTOM" anstatt "Abholung_BASE64") -> #F-Daten" (HRPAYDEB2A 001)

Both programs (database update) also write the error message to the Notification Tool as an open task.

Due to the error, the system does not collect any confirmations/responses on the affected communication server.

Reason and Prerequisites

In transaction SOAMANAGER, for the logical port of the company number from the error message, you have not maintained the Optimized XML Transfer field with the value MTOM.

Solution

If this error occurs, check the settings in SOAMANAGER for the logical port of the affected communication server (GKV or DSRV) and adjust them.
