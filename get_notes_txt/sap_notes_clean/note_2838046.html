Symptom

When trying to open the 'Display Job List' App within the Fiori Launchpad, a '500 Internal Server Error' is raised

Environment

  • SAP ERP Enterprise Portal (EP)
  • SAP R/3
  • SAP R/3 Enterprise
  • SAP ERP Central Component
  • SAP ERP
  • SAP Enhancement package for SAP ERP
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA, on-premise

Reproducing the Issue

Open the Fiori Launchpad and open the 'Display Job List' App. A '500 Internal Server Error' message is displayed and there is an OBJECTS_OBJREF_NOT_ASSIGNED DUMP in the Back-End system

Cause

This is caused by a Program Error

Resolution

Implement the corrections from SAP Note 2767837

Keywords

CX_SY_REF_IS_INITIAL, CL_POWL_UI_HELPER=============CP, RENDER_DELTA_TABSTRIP, SAPMHTTP, W0020
