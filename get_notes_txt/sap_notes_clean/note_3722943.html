Symptom

You are using the Supply Assignment(ARun) functionality.

During immediate supply assignment process, when immediate delivery and release check option are enabled for sales orders , the system behaves as follows: If an item fails the field check, its quantity is excluded from the release rule calculation. This exclusion can result in a partial delivery being generated for the sales order, even if the overall release rule check would otherwise fail.

Other Terms

Supply Assignment, ARUN, Immediate Assignment, Release check, field check

Reason and Prerequisites

Prerequisite

- For the sales document type and sales area, you enable immediate assignment and release check required.

- You turn on immediate delivery.

- You set the release rule at document header level to 100%.

Reason
This issue is because of a program error.

Solution

The Program corrections are contained in the tab correction instruction of this note. You need to implement the corrections in your system.
