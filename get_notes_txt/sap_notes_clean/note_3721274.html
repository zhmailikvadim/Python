Symptom

The user is working on Monitor Receipts application. The user's login language is different from English.

When a user attempts to open “Heuristics”, the correct heuristics and their names are not displaying.

According to correct behavior, the logic for determining heuristic availability and display should not depend on the existence of a description.

Other Terms

Monitor Receipts, Heuristics, Login Language, PPDS

Reason and Prerequisites

The logic allows the selection of the Heuristics even if does not contain the corresponding description in the language the user is using when logging in.

Solution

Implement the attached Correction Instruction or import the relevant Support Package into your system.
