Symptom

When is custom logic triggered during lead creation?

Environment

  • SAP Sales Cloud Version 2
  • SAP Sales and Service Cloud Version 2

Reproducing the Issue

  1. Go to Extensibility Administration 
  2. Select the Lead 
  3. Go to the Determinations/Validations tab
  4. Create the custom logic with Pre Hook/Post Hook
  5. Go to Lead -> Click +
  6. Enter the data and Save 
  7. This custom logic sometimes doesn't work 

Resolution

Only code fields(exclude multi value code) and Object Value Selector(OVS) based fields like account/owner/primaryContact/individualCustomer/salesOrganization changes will immediately trigger the custom logic. For other changes, the custom logic will be triggered only while pressing the save button.
