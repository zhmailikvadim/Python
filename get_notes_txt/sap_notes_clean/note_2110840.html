Symptom

The special character text element is not maintained for the login language, the learner creation is not been successful.

Other Terms

XLSO, Extended Learning Solution, Special character, email

Reason and Prerequisites

The special character in text element available english langauge. But learner creation is done with different logon language, then the validation of special character will not allow to create the learner.

Solution

The logon langauge do not have special character text maintained, then the special character check for the email name will be done against default english langauge text element maintained.
