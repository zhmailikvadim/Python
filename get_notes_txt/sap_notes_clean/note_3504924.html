Symptom

You use transaction FPW2 or the report SAPFK200 in Contract Accounts Receivable and Payable. After the upgrade to SAP_BASIS Version 758 or higher, the Excel download in the list of log records for foreign currency valuation is no longer possible: The "Spreadsheet" entry is missing in the context menu when you right-click the table.

Other Terms

Spreadsheet, EXCEL, &XXL, context menu

Reason and Prerequisites

Changes in the ALV framework

Solution

Implement this SAP Note or import the specified Support Package.
