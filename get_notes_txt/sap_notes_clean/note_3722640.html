Symptom

There are technical errors in Software Update Manager (SUM) related to missing repository objects. The following packages interface are again delivered for the topic Payment Template Archiving:

R3TR PINF FS_TB_ARCHIVING_PAYMTMP_IF
R3TR PINF FS_TB_BVW_PAYMTMP_EXITS_IF

Other Terms

 SUM, Module CHECK_DELIVERY_FOR_COMPONENTS, TRBK 2.0,  Objects without delivery transport found for Software Component 'CB4HANA', CB4HANA

Reason and Prerequisites

Program Error

Solution

Import the relevant Support Package for your release. There is no need to manually implement this note in advance using transaction SNOTE.
