Symptom

  • During the manual activities of implementing SAP Note 3688319 - [CVE-2026-24328] Open Redirection vulnerability in Business Server Pages Application (TAF_APPLAUNCHER), the warning message: "Creating the alias in the non-original system could result in database errors Read Notes 1037771 and 1031500 - Do you want to create the alias here anyway?" displays.

Environment

SAP Solution Manager 7.2

Reproducing the Issue

  1. Start the SNOTE transaction.
  2. Perform manual activities for implementing SAP Note 3688319.
  3. Encounter the warning message: "Creating the alias in the non-original system could result in database errors Read Notes 1037771 and 1031500 - Do you want to create the alias here anyway?"

Cause

Known issue

Resolution

The warning message is expected behavior. Proceed with selecting "Yes" to continue.
In the next pop-up to select Package enter AGS_TST_WEB and save.
