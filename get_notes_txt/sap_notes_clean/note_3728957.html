Symptom

When creating the service entry sheet in transaction ML81N, the error message "Inconsistent with FI/CO document: different document structures" AAPO007 is encountered.

Environment

• Materials Management (MM)
• SAP S/4HANA

Reproducing the Issue

  1. Access transaction ML81N.
  2. Enter the Purchase Order (PO).
  3. Create a service entry sheet.
  4. Click on service selection and enter the required details.
  5. Select the line item and click on "Adopt Service".
  6. Release and save the service entry sheet.
  7. Observe the error message "Inconsistent with FI/CO document: different document structures".

Resolution

Implement the below note and re-test the issue:
3540072 - Error message AAPO007 when creating a service entry
