Symptom

Would like to know how Fulfillment Rate is calculated in Sales Contract.

Environment

  • SAP Sales Cloud Version 2
  • SAP Sales Cloud and SAP Service Cloud Version 2

Reproducing the Issue

  1. Navigate to the Sales Contracts work center.
  2. Open Sales Contract A.
  3. Observe that there is one Fulfillment Rate field under the Products section.

Resolution

The formula for calculating the Fulfillment Rate is release/target * 100. eg: If the target quantity is 0, the Fulfillment Rate will be set to 0.
