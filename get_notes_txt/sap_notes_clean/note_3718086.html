Symptom

VAT Return report RFUMSV00 for Slovenian KIR/KPR calculates values of Tax posting documents 100 times lower than actual tax values on versions lower than S4CORE 103 for OBRAVNAVA 2 documents (documents prom previous periods not yet being processed in update run of RFUMSV00 report).

Tax Payable Posting is required and Batch Input Session flag is selected.

Other Terms

Slovenia, Book of Incoming Invoices, Book of Outgoing Invoices, KIR, KPR, DDV-O, RFUMSV00, /CCEE/SIFI_DMEE_KIR_KPR, DMEE UMS1 SI_DDV_KIR_KPR, HWBAS, HWSTE, DMBTR

Reason and Prerequisites

Error fixing 

Note 3702969 has been implemented.

Solution

Apply corresponding support package or implement attached correction instructions.
