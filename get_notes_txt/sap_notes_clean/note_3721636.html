Symptom

The Purchase Order is successfully created, but upon navigating to Environment → AC Commitment Documents, the system displays the message: "No subsequent document found in Accounting".

Environment

  • SAP S/4HANA 
  • SAP ERP 
  • SAP enhancement package for SAP ERP 
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4 HANA Cloud Private Edition

Reproducing the Issue

  1. Run transaction ME21N in SAP S/4HANA.
  2. Create a Purchase Order and save it successfully.
  3. Navigate to Environment → AC Commitment Documents.
  4. Observe the message: "No subsequent document found in Accounting". 

Cause

Commitment Management was not active for PM Order 

Resolution

  1. Activate Commitment Management at the Controlling Area level using transaction OKKP.
  2. Activate Commitment Management at the Order Type level using transaction OIOA/KOAO for the relevant PM Order Type.
