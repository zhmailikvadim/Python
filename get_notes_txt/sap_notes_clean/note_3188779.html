Symptom

Resolution

Previously, when a buyer integrated with SAP ERP or SAP S/4HANA using SAP Integration Suite, managed gateway for spend management and SAP Business Network ran the ARBCIG_MASTER_DATA_EXPORT program in a test run, using external commands with a specific path, an error occurred. The following error message was displayed:

'mv: target 'tmp/xxxxx' is not a directory'

For more information, please see SAP Note 3142752.
