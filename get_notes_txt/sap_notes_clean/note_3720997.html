Symptom

Poland outbound invoice failed with the error in the history tab as Invalid token

Cause

The KSeF token set up under Country Configuration tab is an invalid token number.

Resolution

Supplier should work internally to gather a valid KSeF token number and update the same under Country Configuration tab in Ariba Network. 

To update the set-up follow the steps below:

  1. Login to SAP Ariba Network Supplier account
  2. Navigate to Initials->Company Profile
  3. Open the tab Country Configuration->Create
  4. Choose the radio button for KSeF token and create an entry
  5. Save

Support does not have visibility to Supplier's KSeF Portal to confirm if the KSeF token provided by the supplier in Ariba Network account is a valid one or not.
