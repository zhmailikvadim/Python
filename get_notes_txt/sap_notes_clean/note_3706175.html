Symptom

If you open the check results to see the alerts from the connected clinical checking system and you switch to different tab pages, the system displays the last selected tab page initially opened the next time you open the check results. It is expected that the "All" tab page is always displayed as open, in order to be able to see all warnings from the clinical checking system.

This affects the "Check Results" function from the application toolbar in Smart Medication, as well as the check results in the "Checks" chapter in the Smart Medication order dialog.

Other Terms

Smart Medication

Clinical Checking

Alerts

Reason and Prerequisites

Program error

Solution

After you implement this SAP Note, the system initially always displays the "All" tab page as open.

Import the specified Support Packages to solve the problem. You can implement the corrections in advance.
