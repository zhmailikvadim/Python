Symptom

  • Field "Special Stock" in VL01N is not editable after upgrading the system from S/4HANA 2020 FPS02 to 2023 FPS01.
  • Field "Special Stock" is greyed out (non-editable) in the item details on the Stock Movement Data tab of VL01N (Outbound Delivery Creation).

Environment

  • SAP R/3 
  • SAP R/3 Enterprise
  • SAP ERP
  • SAP ERP Central Component
  • SAP enhancement package for SAP CRM, version for SAP HANA
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Run VL01N
  2. Enter the affected shipping point / sales order
  3. Press "Enter"
  4. Observe that field "Special Stock" is not editable (greyed out).

Cause

  • Field "LIPS-SOBKZ" is greyed out because it is typically copied from the sales order and the system makes it read-only in the delivery.

Resolution

  1. Run transaction SE18 and enter BADI "LE_SHP_DELIVERY_PROC".
  2. Create an implementation or use an existing one.
  3. Implement the method "CHANGE_FIELD_ATTRIBUTES". 
  4. Activate the BADI implementation  

See Also

Refer to SAP Notes or Knowledge Articles related to BADI "LE_SHP_DELIVERY_PROC" and field attribute modifications in SAP S/4HANA delivery processing.
