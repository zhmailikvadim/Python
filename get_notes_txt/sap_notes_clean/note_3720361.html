Symptom

A few users report an error when trying to access 'View My Tax Statements' in SuccessFactors Employee Central:

"Technical Exception not available
User [user_name] does not exist in this period"

Environment

  • SAP ERP
  • SAP S/4HANA
  • SAP SuccessFactors Employee Central Payroll
  • HR Renewal

Cause

The application HRESS_A_REP_CA_TFR is called via mashup in SuccessFactors Employee Central to display the Canada Year End forms for employees.

In order to fetch the form for the respective employee, the application will fetch the PERNR that is associated with the EC user.

To link the EC user to the PERNR in the backend, a record of infotype 105 (IT0105), Subtype 0001, is created, specifying the EC user in field 'System user name'.

Resolution

In transaction PA20, check if PERNR has IT0105 correctly linking to the SuccessFactors EC user.
