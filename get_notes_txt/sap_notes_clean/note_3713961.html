Symptom

  • Request for information on accessing SAP CX AI Toolkit tenant from various devices.
  • Inquiry about the existence of a dedicated application for SAP CX AI Toolkit on mobile/tab devices.

Environment

SAP CX AI Toolkit

Reproducing the Issue

Not applicable (How to Request).

Cause

A dedicated SAP CX AI Toolkit app does not exist for mobile/tab devices.

Resolution

  1. SAP CX AI Toolkit does not have a dedicated application for mobile/tab devices as of now.
  2. SAP CX AI Toolkit can be accessed via web browsers from any electronic device that supports web browsing.
