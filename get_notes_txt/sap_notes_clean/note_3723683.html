Symptom

The app "Monitor Predictive Accounting" (fin.pra.monitor) does not include the fields "Sales Office" and "Sales Group" in the analytical list page or in the filter section.

Other Terms

Monitor Predictive Accounting, fin.pra.monitor, F3828, FAC_MONITORING_PRA

Reason and Prerequisites

Missing functionality.

Solution

Apply this note. The fields "Sales Office" and "Sales Group" are then available as optional columns in the table and as optional filter criteria, which can be added manually via the table settings and adapt filters dialogs.
