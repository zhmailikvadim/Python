Symptom

Why I cannot participate by clicking on the button Email Response for single line item event?

Reproducing the Issue

  1. Open realm
  2. Navigate to the Sourcing Project
  3. Click supplier tab>Select supplier>Click send invitation email
  4. Open the mail > click on button email response> you can see no subject to participate through email

Resolution

If the customer realm has the Parameter(Application.ACM.EmailBiddingWithStandardExcel), Feature (SS-13636) and Rule (Allow participants to submit bids by email) set then it will be using excel sheet only.

[Note: If Parameter Application.ACM.EmailBiddingWithStandardExcel is turned off and we click on the button Email Response, then participation can be done for single line item.Since this parameter is enhancement so it is not recommended to turn it off. In case you want to disable this parameter, DSC can raise a case to support.]
