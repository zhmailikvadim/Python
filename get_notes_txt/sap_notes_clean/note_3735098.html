Symptom

You trigger the outbound event for POL as PEO Routing Notification. The event fails because the system does not fill the before image and after image data for POL as PEO Routing in the outbound event.

Other Terms

PLMSI, PLMSIFND, PLM System Integration, POL, PEO, Notification, Before Image, After Image, Outbound

Reason and Prerequisites

The outbound event expects before image and after image data for POL as PEO Routing, but this data is not needed in this scenario.

Solution

Implement this SAP Note to ensure that the system skips reading the before and after images for POL as PEO Routing outbound event.
