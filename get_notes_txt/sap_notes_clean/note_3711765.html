Symptom

You are using transaction FPE2 to change the payment method on document level. This leads to error message 0 133: Payment method &1 is not defined for paying company code &2.

Other Terms

FPE2, PYMET, TFK042E, paying company code, >0 133.

Reason and Prerequisites

The system performs the payment method check for the company code of the contract account and not for the paying company code on document level.

Solution

After implementing this SAP Note, the system performs the payment method check for the paying company code on document level.

Implement the correction using the automatic correction instruction or wait for the next support package.
