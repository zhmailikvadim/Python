Symptom

When trying to migrate project using migration tool however system throws the following error:

"End constraint not permitted for summary task".

Environment

SAP Business ByDesign

Reproducing the Issue

  1. Go to Business Configuration workcenter
  2. Go to Overview
  3. Select Migration of Projects activity.
  4. Select Migrate projects using the migration tool
  5. Select the file
  6. Execute step by step
  7. Error appears. 

Cause

The duration is not maintained in the task level under the Duration column of the migration template.

Additionally, ensure that when the project is created, the given dates in the expenses, revenues and tasks must be in the range of the project start date and end date.

Resolution

For the Project task details, duration days need to be maintained for the summary tasks based on the Project Start date and End date.
