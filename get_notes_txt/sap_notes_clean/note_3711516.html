Symptom

Failed and in-process messages cannot be deleted in bulk, retention settings allow only completed and inactive status messages to be deleted automatically.

Environment

  • SAP Digital Manufacturing Cloud 2511

Reproducing the Issue

  1. Navigate to the Integration Message Dashboard in SAP Digital Manufacturing Cloud.
  2. Attempt to delete failed and in-process messages in bulk.
  3. Observe that the system does not allow bulk deletion of these messages.

Cause

Resolution

Please check these public APIs：

  • https://api.sap.com/api/sapdme_integrationMessage/resource/
  • https://api.sap.com/api/sapdme_integrationMessage/path/setToInactive

See Also
