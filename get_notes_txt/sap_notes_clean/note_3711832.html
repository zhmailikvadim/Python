Symptom

When attempting to create a Custom Code List, the following error messages are displayed: 

  • Invalid user input
  • Code List ID can contain only alphanumeric characters. Special characters are not allowed."

Environment

  • SAP Sales Cloud Version 2
  • SAP Service Cloud Version 2

Reproducing the Issue

  1. Click on the User Icon.
  2. Navigate to the System Settings > All Settings.
  3. Go to the Extensibility section.
  4. Select Custom Code Lists.
  5. Click on the Create button.
  6. Enter a Name like SAP_TESTING_V3.
  7. Observe the error message.

Cause

The error occurs because the Custom Code List name input validation restricts the use of spaces or special characters. The system has enforced a limitation on the use of special characters in the CodeList ID field to ensure consistency and avoid technical issues that previously arose when special characters, such as underscores, were included.

Resolution

To resolve this issue, ensure that the Custom Code List name adheres to the input validation requirements by removing any spaces and special characters. Use only alphanumeric characters.
