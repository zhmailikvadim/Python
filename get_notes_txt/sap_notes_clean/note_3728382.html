Symptom

As an administrator, I attempt to configure different banner images for each PU or Landing Page, but the banner always updates globally for the entire realm.

Cause

The Banner Image in Guided Buying is a global realm‑level setting. It cannot be customized per PU or per Landing Page.
Resolution

Resolution

This is expected system behavior.
Different banner images per PU or per Landing Page are not supported.
All banner updates will apply globally across the realm.
