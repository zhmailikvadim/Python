Solution

Import the Support Package specified for your release or implement the attached correction instructions.

As a general rule, SAP recommends that you implement the Support Package specified for your release. However, if you need to install a solution earlier, use the SAP Note Assistant to implement the correction instruction. You can find more information about the SAP Note Assistant under SAP Note Assistant.

Re-extract Data

After you have implemented this SAP Note, delete and recreate extraction data to correctly fill out the extraction tables. If you do not execute this step, errors and inconsistencies may persist.

Note: if you implement an SAP Note sequence, you will only be required to execute this step after all SAP Notes have been implemented.
