Symptom

A DataSphere tenant is set up with replication flows that import tables from S4HANA. Currently, attempts are being made to establish a connection between the DataSphere database and Power BI/Excel via an ODBC connection.

After executing an "ODBC test connection," the following error message appears:

  • [SAP AG][LIBODBCHDB SO][HDBODBC] General error;10 authentication failed: Detailed info for this error can be found with correlation ID 'XXXXXXXXXXXXXXXXXXXXXX'

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Environment

  • SAP HANA Cloud
  • SAP Datasphere

Cause

Improper IP Allow listing configuration for HANA Incoming Requests

Resolution

First ensure the password does not include the following specific characters: []{}(),;?*=!@.

To resolve the issue, utilize the correct corporate VPN and be sure you have added this system to the IP Allow list rather than a single standalone IP address.
