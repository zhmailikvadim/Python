Symptom

  • The program J_1AF_UPDATE_VENDOR (transaction S_EB5_05000371) does not evaluate all documents from the vendor on the selected period resulting on zero or incomplete Total Accumulated value.
  • The issue arises when payment is made for vendors classified as "Monotributistas" based on resolution RG 2616.

Environment

  • SAP ERP
  • SAP S/4HANA
  • Withholding Taxes for Argentina

Reproducing the Issue

  1. Execute transaction S_EB5_05000371 for Argentina.
  2. Evaluate payments for vendors classified as "Monotributistas".
  3. Observe that the cumulative amounts are not reported correctly.

Cause

  • The absence of the WHT Code configuration within the vendor's master data caused the issue.
  • Since SAP Note 1382897, any WHT Types configured without Code are deleted on the roll of eligible scenarios to be processed by program J_1AF_UPDATE_VENDOR.

Resolution

  1. Update the vendor's master data to include the WHT Code from the applicable WHT Types for your vendor.
  2. Test the transaction S_EB5_05000371 to confirm that invoices and cumulative amounts are reported correctly.
