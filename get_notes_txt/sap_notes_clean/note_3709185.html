Symptom

  • Period End Processing job could not process time accounts as the follow-up account is already closed.
  • In the log file, an error is shown as expected but no error message that points to the closed account. 

Environment

  • Product: SAP SuccessFactors Employee Central
  • Calendar Jobs

Reproducing the Issue

  1. PEP for time account type which started in simulation mode for Dec 31, 2022.
  2. If follow up account is not closed, there is a well defined error message.
  3. After closing the follow-up account, I got an error state without text:
  4. "TAT","co20140702","co20140702","AccountTester Joe","fe6ebcf7098c47a983fabe9baf0b76d7","2022-01-01","2022-12-31","No","","","","","ERROR","","Yes" without indicating what is the error.

Cause

Closed follow-up time account causes an error during the PEP job execution.

Resolution

We have enhanced the system behavior that once system detects a closed follow up account, after running PEP, error states: Follow-up account is closed.
