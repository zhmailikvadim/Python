Symptom

In vital sign entry you create two examinations for the same examination time.

In one of the examinations you have entered vital signs, which lie within the warning or alarm range.
In the other examination, all vital signs lie within the normal range.

However, in the Vital Signs UIBB for this examination, the system also incorrectly uses the ! or !! Icons to display
that vital signs have been entered in the warning or alarm area.

Other Terms

Clinical Measurements, Smart UI, Value Range, Normal Range, Warning Range

Reason and Prerequisites

Program error

Solution

Import the specified support packages to solve this problem.
For an advance correction, implement SAP Note 3214207.
