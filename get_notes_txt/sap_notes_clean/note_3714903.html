Symptom

  • During migration of Customer Master data from SAP ECC to SAP S/4HANA using Migrate Your Data, partner function assignments are observed.
  • Partner functions are automatically created in S/4HANA for multiple customers, even though no partner functions exist in ECC.

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Environment

  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Perform data migration using Migration Cockpit
  2. Compare partner function assignments in ECC and S/4HANA.
  3. Observe there are more assignments in S/4HANA than in ECC

Cause

Partner functions were added automatically based on the partner procedure assigned to account group.

Resolution

Find the partner procedure assigned to customer or vendor account group:

  1. Run transaction OVT0 (customer) or OMSG (vendor)
  2. Double-click on the relevant account group
  3. The partner procedure assigned to customer account group appears in the field "PartnDet.Proc.", as shown below:



  4. The partner procedures assigned to vendor account group can be found as shown below:

Inside the partner procedure assigned to the account group, it is possible to check what partner functions are defined as mandatory (automatically determined).

For customers:

  1. Run transaction VOPAN
  2. Select "Cust.Master"
  3. Click Display
  4. Select the partner procedure 
  5. Double-click on "Partner Functions in Procedure"
  6. Partner functions with checkbox in column PM set, "Partner Mandatory", set will be determined automatically when creating or changing the customer.

For vendors:

  1. Run transaction SPRO
  2. Navigate to SAP Customizing Implementation Guide > Materials Management > Purchasing > Partner Determination > Partner Settings in Supplier Master Record
  3. Execute activity "Define Partner Schemas in Supplier Master Record"
  4. Select the partner procedure (partner schema)
  5. Double-click on "Partner Functions in Procedure"
  6. Partner functions with Mandatory flag set will be determined automatically when creating or changing the vendor

If it is needed to mass delete partner functions, utilize the API class CL_MD_BP_MAINTAIN and method MAINTAIN within a custom program, as described in SAP Note 2417298.
