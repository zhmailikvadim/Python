Symptom

With the coding corrections of note 2342497 being contained in the system, there are problems with changing or posting FI documents, which were parked via an accounting BAPI like function module 'BAPI_ACC_DOCUMENT_POST', if the parked FI documents should contain withholding tax data, but do not.

Other Terms

parking, accounting BAPI, 'BAPI_ACC_DOCUMENT_POST', withholding tax data, FI_WT_ACI_PROCESS_WITH

Reason and Prerequisites

The coding corrections of note 2342497 are contained in the system, and also the coding corrections of note 1847676 are contained in the system.

Solution

Please implement the attached correction instruction in your system.
