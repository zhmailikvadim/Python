Symptom

Below error occurs during the preparation step using the migration object "Customer" in the Migrate Your Data app. 

  • Enter country/region key in the conversion rule &1, region is &2

          Message No. CNV_DMC_SIN  123

Environment

  • SAP S/4HANA Cloud Public Edition
  • SAP S/4HANA Cloud Private Edition
  • SAP S/4HANA

Cause

The issue arises when the State (Region) field is populated, but the Country/Region field is left empty in the "Contact Person" sheet.

Resolution

Ensure that the Country/Region field is filled whenever the State (Region) field is populated.

See Also
