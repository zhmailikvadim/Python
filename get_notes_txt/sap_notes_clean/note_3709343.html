Symptom

You use BAPIs to create requests with transaction locks or to change the transaction locks of requests. One of the following symptoms occurs:

  • The business locks are only transferred in the database table DFKKORDERLOCKS, while the corresponding fields in the database table DFKKORDERPOS are not updated.  
  • If you use edit mode 02 (Change) for a business lock, the change is not applied.
  • If you use an invalid processing mode for a business lock, the system does not issue an error message.
  • If you pass an unknown business lock, the system does not issue an error message.
  • If you pass multiple business locks of the same category for the same item, the system does not issue an error message.
