Symptom

SAP Groupware Connector 7.0 is not synchronizing content. The following error is output in Proxy logs.

FATAL EWSSyncCore.ExchangeGWStub .ctor [] - Failed to Initialize the ExchangeGWStub
Exception: System.Runtime.Serialization.SerializationException
Message: End of Stream encountered before parsing was completed.

Environment

SAP Groupware Connector 7.0 is installed.

Cause

  • SyncUser.xml file is corrupted.
  • SyncUser.xml file is too huge.

Resolution

  1. Rename the file "SyncUser.xml" in path "...\SAP\SAP Groupware Connector 7.0\Exchange\Proxy\Storage".
  2. Restart services.

Keywords

CRM, Groupware Connector, exchange, sync, SyncUser, EWSSoapxmls, MW, xml, Office 365 Exchange, GW
