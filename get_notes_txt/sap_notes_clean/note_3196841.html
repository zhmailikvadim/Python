Symptom

After setting Customer Page in Fiori BI Launch Pad Preferences , the user's Home Page shows as Personal Folder

Environment

  • SAP BusinessObjects Business Intelligence platform (BI) 4.3

Reproducing the Issue

  1. Create a User and Add the User to a Group.
  2. Go to CMC Home, click Users and Groups.
  3. Expand Group Hierarchy.
  4. Right-click the Group, and then click Fiori BI Launch Pad  Preferences.
  5. Uncheck No Prefernces Defined.
  6. Click Customer Page.
  7. Select Folders as Landing Page.
  8. Click Browse Folders, select a public folder.
  9. Logon BI Launchpad by the user, even the column shows Public Folders information, but the title is still Personal Folders.

Cause

The symptom should be caused by system defect.

Resolution

The symptom could no be reproduced in BI4.3SP2Patch3

Keywords

BILaunchpad, customized, BI Launchpad, faverate
