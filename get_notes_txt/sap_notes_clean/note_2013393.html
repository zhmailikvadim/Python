Symptom

Entering an invalid (not existing) function location or equipment

  • in the section 'Object References and Dates' of a change request or
  • in the section 'Related Objects' of a change request or an activity

does not lead to an error message.

Together with empty but mandatory fields (e.g. Description, Reported By) there will be incorrect error messages after filling these mandatory fields.

Other Terms

Management of Change, Object Reference

Reason and Prerequisites

Program error

Solution

Apply correction instruction
