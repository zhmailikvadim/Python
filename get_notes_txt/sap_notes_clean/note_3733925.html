Symptom

The feature Automatically Extend SOW Worker End Dates to Match SOW End Date allows the end dates of workers on a Statement of Work (SOW) to be automatically adjusted to match the end date of the SOW when it is revised.

Cause

  • Ensure the Allow change to Worker end dates without Worker Revision company configuration is enabled.
  • Enable the Automatically Extend SOW Worker End Dates to Match SOW End Date rule on the SOW type or template.

Resolution

Detailed Steps to Enable and Use the Feature
 
  1. Enable Company Configuration:
    • Go to the company configuration settings.
    • Enable Allow change to Worker end dates without Worker Revision.
  2. Enable SOW Rule:
    • Navigate to the SOW type or template settings.
    • Enable Automatically Extend SOW Worker End Dates to Match SOW End Date.
  3. Create or Revise SOW:
    • Open the SOW for revision.
    • Change the end date of the SOW.
    • Save the changes.
  4. Verify End Date Extension:
    • Check the SOW workers' end dates to ensure they match the new SOW end date.
