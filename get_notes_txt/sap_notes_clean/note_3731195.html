Symptom

This note enhances the RFC function module FSTB_ACCT_VRFY_GET_BY_ACCT_EXT for Bank Account Verification to return account holder names from the following sources for the requested account:

  • 01: Business Partner Name
  • 02: Account Alias Names
  • 03: Business Partner Additional Names

Other Terms

BankAccountVerification, BAV, AccountHolderNames,

Packages FS_TB_ACCT_VERIFICATION_AL and FS_TB_ACCT_VERIFICATION_MAIN

Reason and Prerequisites

New requirement

Solution

A new exporting parameter ET_ACCOUNT_HOLDER_NAME is added to the function module FSTB_ACCT_VRFY_GET_BY_ACCT_EXT containing the account holder names.

Import the relevant Support Package for your release. You can use transaction SNOTE to implement the correction instructions relevant for your release in advance.
