Symptom

You are working with visit lists in the Last Mile Distribution solution. The OData service Visit List (LASTMILEVISITLIST) lets you maintain a free-text driver ID at visit-item level, which can differ from the driver ID on the visit list. This means the actual driver might not exist as a business partner in SAP S/4HANA, which prevents IRF from functioning correctly.

Other Terms

LMD, F5221, person-related data, LastMileVisitListDriverID, LastMileVisitItemActDriverID, IRF, LMD_VL_BIL_API_VisitList, val_actdriver_matches_vl, api_lastmilevisitlist

Reason and Prerequisites

The issue is caused by a programming error.

Note that the new message LMD_VL_MSGS 165 contains only the orginal English text. If you need translations in additional languages, you must provide the translated text yourself.

Solution

Please apply the attached correction instructions to resolve the described issue.
