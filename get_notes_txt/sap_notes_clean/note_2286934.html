Symptom

This SAP Note provides a correction to the check for direct and indirect quotation.

Reason and Prerequisites

If you do not change the risk currency and invoice currency in the new business and they are the same as the local currency, the system determines the exchange rate between the local currency and the risk/invoice currency as -1. This is incorrect. The exchange rate must be determined as 1. This SAP Note corrects the error.

Solution

To correct the problem, implement the source code corrections.
