Symptom

After implementing Note 3689097 for JPK KR PD, customers encountered an error in the report definition.

Other Terms

JPK_KR_PD, PL_SAFT_KR_PD

Reason and Prerequisites

Before proceeding with the implementation of SAP Note 3715142, customers must first apply SAP Note 3689097. This step ensures that all necessary components are available and properly configured.

Solution

After implementing SAP Note 3715142, please verify that the report PL_SAFT_KR_PD can be generated successfully.
