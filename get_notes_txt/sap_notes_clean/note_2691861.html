Symptom

When you create an additional insured person, the system checks whether the role ID was entered. If not, the insured person is not created.

Other Terms

- Creation of an insured person

Reason and Prerequisites

Program error

Solution

History:

Version 1 from 9/10/2018: New　

Implement the source code corrections.
Support Package SAPK-70020INMSG / SAPK-80203INMSG solves the problem.　

Solution:

When you create an additional insured person, the system checks whether the role was entered. If it was, an additional insured person is created.

Technical effects:

See the "Solution" section.　

Manual changes:

You do not have to perform any manual tasks.
