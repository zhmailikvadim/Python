Symptom

  • When removing Incoterms and Incoterms Location 1 Description in the Manage Purchase Contracts app, the Incoterms Location 1 Description reappears.
  • Error message is shown: “Incoterm does not exist (Please check your input)” with code 06/485.

Environment

  • SAP S/4HANA
  • SAP Fiori

Reproducing the Issue

  1. Open the Fiori app Manage Purchase Contracts (F1600A).
  2. Open an existing purchase contract and switch to Edit.
  3. Remove the Incoterms and the Incoterms Location 1 Description.
  4. Press Enter or Save.
  5. Observe that the error “Incoterm does not exist” appears and the Location 1 Description reappears.

Cause

  • A required software correction was missing; the related note was not implemented in the system.

Resolution

  1. Implement note 3566294.
  2. Clear the caches as described in note 2319491.

See Also

  • Refer to: SAP Note 3566294
  • Refer to: SAP Note 2319491
