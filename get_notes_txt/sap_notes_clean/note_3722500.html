Symptom

You use SAP Travel Management for S/4 HANA Fiori application "My Travel Requests (Version 4) for Business Traveler" or "Travel Requests (Version 4) for Travel Assistant".

In the travel request application, you create a travel service to the destination and add a return trip, after saving the trip request you can see that departure country/region description is not displayed but only the country/region code is displayed.

Expected vs Actual Behavior

Expected Result: The Departure country/region Description to be displayed in the field after saving the travel request

Actual Result: Only the Departure country/region Code is displayed without the country/region description/text

Deviation: Country/region Name is not fetched correctly based on the country/region code

Other Terms

Travel Requests, MTR

F6015 'My Travel Requests for Business Traveler'

F6015A 'My Travel Requests (Version 4) for Business Traveler'

Reason and Prerequisites

Programming Error

Solution
