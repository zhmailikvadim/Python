Symptom

  • After upgrading from 4.3 to BI 2025 non admin users are not able to build new reports based on the OLAP connection due to missing BW BEx connections and unable to use the BEx query.

Note: “Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.“ 

Environment

  • SAP BusinessObjects Business Intelligence (BI) Platform 4.x/2025
  • Any supported OS

Reproducing the Issue

  1. Perform upgrade from 4.3 to BI 2025.
  2. Attempt to build a new report using BW BEx connections with non-admin users.
  3. See the BEx connections are not visible for non admin users.

Cause

  • The "Download Connection Locally" security right is missing for the non admin users for the OLAP Connection type.

Resolution

  1. Navigate to the CMC - Connections
  2. Click on Manage - Top-Level Security - All Connections (click OK on pop up)
  3. Select the desired user/group click on Assign Security - Click on Advanced - Add/Remove Rights
  4. Expand the System section - Click on OLAP Connection
  5. Grant the "Download connection locally" security right 
  6. Click on Apply and OK

Keywords
