Symptom

This SAP Note contains information about planning the installation and upgrades of the ABAP add-on ST-OST.

Other Terms

Solution

  1. General information
  • See the general information from SAP Note 70228.
  • You cannot uninstall ABAP add-ons.
  • You can install ABAP add-ons only on certain SAP releases.
  • Upgrades with ABAP add-ons
    1. In systems in which an ABAP add-on is installed, you can upgrade only to SAP releases that are supported for this add-on.
    2. Note that there is a delay between the delivery of the SAP standard releases and the release of the corresponding add-on releases.
    3. If an add-on upgrade is connected with a change of the SAP release, it is integrated into the SAP upgrade (repository or system switch). To carry out this upgrade, you require an additional add-on upgrade CD or DVD in addition to the SAP standard upgrade CDs or DVDs. If the SAP standard upgrade is carried out without this additional CD or DVD, the add-on will no longer work after the upgrade. The entire SAP system is inconsistent.
      For more information about this problem, see SAP Note 33040.
    4. If you are scheduling an upgrade or the import of an enhancement package, note that you should not import the latest Support Package Stack in the source release directly before the upgrade to ensure that the required, equivalent Support Package is already available in the target release.
      For more information about this, see SAP Note 832594 and the following information on SAP Service Marketplace.
      http://support.sap.com/sp-stacks -> SP Stack Information -> SP Stack Strategy

    1. SAP Product Availability Matrix
      The following information about the add-on in the SAP Product Availability Matrix is available on SAP Service Marketplace at http://support.sap.com/PAM 
      - Availability
      - End of maintenance
      - Release for products with enhancement packages
      - Language support
    2. Download
      The software is available on SAP Service Marketplace at http://support.sap.com/SWDC
        -> Search for Software Downloads
            -> ST-OST

      Search options: Only Software Components and Software Component Versions
    3. The add-on does not contain any modifications.
      You can also perform the installation if you have already imported more Support Packages into your system than are specified in the section "Required Support Packages".
