Symptom

A FMPO assignment is delimited from an organizational measure. Another FMPO assignemnt is created with the same organizational measure and a time gap exists between "Planned Delimitation Date" of the first assignment and "Valid Start Date" of the new assignment. After activation of the organizational measure the first assignment is not delimited correctly and the "Planned Operation" still exists.

Other Terms

S/4 HANA Defense & Security, Manage Force Elements, Manage Positions, FMPO, Delimitation

Reason and Prerequisites

Program error.

Solution

Implement the correction instruction.
