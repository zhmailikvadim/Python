Symptom

In view V_512W_O, the system selects cumulations 47 and 60 for wage type /406 (Rent Paid to Employer by Employee), which is incorrect.

After implementing this SAP Note, the system no longer selects cumulations 47 and 60 for wage type /406 in view V_512W_O.

Other Terms

 V_512W_O, /406, cumulation 47, cumulation 60

Reason and Prerequisites

The system configuration incorrectly associates cumulations 47 and 60 with wage type /406.

Solution

Implement the attached manual activities or the relevant Support Package.
