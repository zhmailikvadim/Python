Symptom

After selecting one or more enitities in the Released status and going through the Mass Actions > Approve process the selected entries are not approved.

Environment

Fiori Consolidation Release Monitor

Financial Real Time Consolidation

SAP Business Planning and Consolidation Embedded - All versions

Reproducing the Issue

  1. Access the Consolidation Data Release Monitor.
  2. Enter context information.
  3. Select entries that have a status of released
  4. Go to Mass Actions > Approve.
  5. Select Initialization options and press OK

Cause

The Mass Actions option is a sequence of steps. For example,
Step 1 - Initialize Package
Step 2 - Currency Translation
Step 3 - Vaidate
Step 4 - Release
Step 4 - Defer
Step 4 - Approve
Notice that Step 4 can have multiple choices, Release, Defer, or Approve. As the Mass Actions has to go through the steps sequentially and it has reached the end of the sequnce (Step 4) it will not go back and rerun the entire sequence, nor can it start at the end point of the sequence.

Resolution

Use the Approve button to the right of the Mass Actions button to change the status from Released to Approved.
