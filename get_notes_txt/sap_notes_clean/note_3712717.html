Symptom

  • Multiple invoices are being generated.
  • When attempting to generate the invoice summary in T-code ISJP_CR, no pending documents are displayed.

Environment

  • SAP R/3
  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA Finance
  • SAP S/4HANA

Reproducing the Issue

  1. Run T-code ISJP_CR
  2. Input company code, payer, closing date
  3. FI document is not included in monthly invoice

Cause

 The document type and posting key targeted for Invoice Summary Processing are not customized.

Resolution

Set the document type, posting key, and special GL indicator that are targeted for Invoice Summary Processing.

IMG: Financial Accounting > Accounts Receivable and Accounts Payable > Business Transaction > Country-Specific Settings > Japan > Invoice Summary: Standard Customizing Settings > Define Line Items
