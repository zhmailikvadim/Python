Symptom

An error occurs during employee replication from S4HANA, displaying the message: "Title 2 does not exist" or "Title 1 does not exist"

Environment

SAP Sales & Service Cloud V2 Integration with SAP S4HANA

Reproducing the Issue

  1. Go to COD_INT_MENU in S4HANA system.
  2. Select Load or Resend Employees.
  3. Fill the logical system and execute the Employee Replication from SAP S4HANA to Sales & Service Cloud V2.
  4. Observe the error message "Title 2 does not exist." / "Title 1 does not exist" in Message Monitoring.

Cause

Missing or incorrect maintenance of external-to-internal Value Mapping for FormOfAddressCode.

Resolution

  1. Ensure the Value Mapping for FormOfAddressCode and the linked Value Mapping Group is maintained in the corresponding communication configuration
  2. If Value mapping is maintained with Mapping Rule "Pass Through" - then ensure the values sent from SAP S4HANA are properly mapped to corresponding Internal Values of SAP Sales & Service Cloud V2

See Also

SAP Note 3414212 - Error Due to Value Mapping in C4C V2
