Symptom

  • A Work Order is in 'Accepted' status that needs to be closed
  • The 'Close Work Order' option is not listed under the 'Actions' Menu

Cause

The 'Close' User Permission for the Work Order module is not enabled for the relevant User Role 

Resolution

Once the 'Close' User Permission for the Work Order module has been enabled for the relevant User Role, the 'Close Work Order' option will be listed under the 'Actions' Menu on the Work Order
