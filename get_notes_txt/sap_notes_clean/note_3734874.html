Symptom

When a location material with production planning integration mode (PPIM) = 0 (No Detailed Planning and Scheduling) is integrated into Production Planning and Detailed Scheduling (PP/DS), the ATP category NI/NY for external planned orders is assigned, instead of the ATP category AI/AJ/AY for normal planned orders.

Other Terms

Synchronized production planning, production planning integration, SAP S/4HANA Production Planning and Detailed Scheduling (PP/DS)

Reason and Prerequisites

Program error

Solution

Implement the correction instructions attached with this SAP Note or wait for the next support package.
