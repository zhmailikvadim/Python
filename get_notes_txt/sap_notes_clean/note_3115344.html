Symptom

You are running SAP LT Replication Server and you face one of the following issues:


1. You are replicating data from an Informix database and the trigger creation fails with an error: "A syntax error has occured".

Other Terms

LT Replication Server, SLT, replication, replicator, trigger, realtime, DMIS 2011

Reason and Prerequisites

SAP LT Replication Server is installed on release DMIS 2011 SP13.

Solution

Implement the required corrections on the source and SLT system in accordance with the correction instructions.
Replication Jobs have to be stopped during note implementation: no
