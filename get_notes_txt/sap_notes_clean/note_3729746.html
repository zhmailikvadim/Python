Symptom

When maintaining IT0001 via PA30 or displaying IT0001 via PA20, the text for position is blank in the screen even the T528T has the corresponding entry. "Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Environment

  • Human Capital Management (HCM)
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for HANA
  • SAP S/4HANA
  • SAP SuccessFactors Employee Central Payroll

Reproducing the Issue

  1. Maintain or display IT0001 via T-code PA30/PA20.
  2. The text for position is blank in the screen.

Cause

Generally, the PA0001-OTYPE should be stored as 'S' when an employee is hired into the system. If the PA0001-OTYPE for the personnel number is accidentally left empty in the PA0001 database, the text for the position cannot be displayed.

Resolution

Create a custom report to update the PA0001-OTYPE to 'S' for the affected personnel numbers.

Keywords

PA0001-OTYPE, T528T, position text, PLSTX.
