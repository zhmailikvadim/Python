Symptom

A sales order exists which contains VAS items determined only from the condition tables. After this, VAS items are deleted for the key figures in the VAS SD condition tables. Transaction FSH_VAS_RD is executed to redetermine the VAS items once more from the condition tables. An error message is raised 'Item is missing in tale YVBAP' in the problem log.

Other Terms

V1 505, updkz = charu, XVBAP, YVBAP, FSH_VAS_RD, FSH_VAS_SD.

Reason and Prerequisites

Reason: Program error

Prerequisites:

1. The VAS condition records for the key figures are deleted completely in FSH_VAS_SD.

2. Sales order item has VAS determined only from the VAS condition tables.

Solution

Implement the attached corrections.
