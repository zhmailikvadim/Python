Symptom

You create a Production Operation List with components on the operation or suboperation level. You then update this Production Operation List with a partial payload, and the components collection (assignedComponents) is missing.  As a result, the components assigned to Production Operation List are deleted.

The expected behavior is that if the components collection is not provided in the payload, the existing components should remain unchanged and must not be deleted.

Other Terms

PLMSI, PLM system integration, POL, Production Operation List, Components, assignedComponents, Partial Payload

Solution

 Implement this SAP Note to resolve the symptom.
