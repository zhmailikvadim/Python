Symptom

When changing component field in app Maintain bill of material, Runtime error "RAISE_SHORTDUMP" occur. However the same dump did not happen on app Maintain bill of material(version 2)

Dump information

  • Category: ABAP programming error
  • Runtime Errors: RAISE_SHORTDUMP
  • Exception: CX_RAP_REUSE_SRV_HOST_CALLBACK
  • Development Object: NONT BILLOFMATERIALITEM
  • Application Component: LO-MD-BOM

Environment

SAP S/4HANA Public Cloud Edition

Reproducing the Issue

  1. Access the application Maintain bill of material.
  2. Modify the component quantity.
  3. Update the long text associated with the item.
  4. Observe the runtime error "RAISE_SHORTDUMP" occurring during these operations.

Cause

It is known issue.

Resolution

The issue will be fixed in 2602.1(RTH - 12th March) 

Keywords
