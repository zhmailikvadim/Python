Symptom

Document links under Course Home are not user friendly. When multiple document links are included, the system places them one in front of the other without any separation.

Environment

SAP SuccessFactors Learning

Reproducing the Issue

  1. In the Admin side, create some Document Links (ideally 3 is fine).
  2. Create an ILT item.
  3. Schedule a class.
  4. Place the document links created in both the class and the item
  5. Add the item to a library that the test user has access to.
  6. Go to the user side.
  7. Search for the item in the library
  8. Access its course details page and class details page.
  9. The document links will appear all together in a single row without any separation.

Cause

Product defect.

Resolution

Engineering team provided a fix to reorder to links to make it user friendly and easy to distinguish between the links added.

The fix will take place the upcoming release b2605. Please find the release dates below:

  • Preview: April 13
  • Production: May 15 - 16

You may find more release details in this Customer Community link.
