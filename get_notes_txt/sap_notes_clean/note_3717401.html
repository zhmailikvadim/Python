Symptom

The error message displayed on the default error page is language-dependent. While the meaningful original error message is displayed when the language is set to English, a respective translation of the generic error message "Message E XX XXX cannot be displayed in plugin mode HTTPS" (or similar) is displayed when the language is set to any other language.

Please note that this Note only addresses the language-independent behavior. There are cases in which the actual error cannot be displayed on the error page and a generic error message is displayed instead. If the error message does not change when switching the language to English updating the kernel patch level will most likely not change the behavior.

Solution

Updating the kernel to the patch level indicated in this SAP Note will change the behavior to be language-independent. The original error is always displayed when available, regardless of the language settings. Thereby, the error is displayed in the language set if available (and in the original lanugage of the error if not).
