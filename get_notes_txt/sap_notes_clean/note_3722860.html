Symptom

An employee left the company on 06/09/2025. You delimit the Contribution Plan HK (0346) infotype record on 06/08/2025.

You create a new Contribution Plan HK (0346) infotype record from 02/01/2026 to 02/28/2026. The system calculates MPF incorrectly in the correction run for February 2026.

Other Terms

MPF, IT0346

Reason and Prerequisites

 This is a correction of legal function. You have applied SAP Note 3694572.

Solution

Apply the Support Package or implement the corresponding correction instructions.
