Symptom

When running App Network Activity Overview (ID F1970) the Field Material is no longer available in Filter 

Environment

  • Fiori UI for Project System (PS)
  • SAP S/4HANA, on-premise
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Run App
  2. Enter data
  3. Select Filter 
     Field is no longer available  

Cause

The Field Materials ( MATNR )have been depreciated
  

Resolution

You can use the app ‘Material component overview’.    ID : F1971 


*The new Fiori application in 2023 FPS03 that replaces ‘Material component overview’ is the Fiori application ‘Product Overview’
