Symptom

Alternative Cost Distribution portlet not working as expected for users without any data in latest people profile.

The summary card remains 'loading' on the preview.

Environment

SuccessFactors Employee Profile 2h2025

Reproducing the Issue

  1. Go to profile of user without any data in this portlet
  2. Summary card remains loading

Cause

This is a known defect.

Resolution

A code correction has been created for this issue.

The engineering team has confirmed that the issue is resolved and will be released on 13-Apr-2026 for Preview and on 15-May-2026 for Production.
