Symptom

The MBC message is not created and an error is logged indicating that no Sender ID is determined.

Other Terms

/PF1/MSG_MBC017

/PF1/CL_OPM_OUTPUT_STREAM_MBC

Reason and Prerequisites

The Sender ID is not configured either in the House Bank settings or in the MBC configuration.

An exception is raised in APM if the Sender ID is not determined. As a result, the MBC message creation process is terminated.

Solution

The Sender ID is optional for MBC processing. Therefore, the error logging has been removed in cases where the Sender ID is not determined, and the process is no longer terminated.

Please apply SAP Note or assigned Support Package.
