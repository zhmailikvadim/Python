Symptom

A sales order is created using the Fiori app "Create Sales Orders - Automatic Extraction." In Customizing, manual product selection has been activated and the corresponding condition records have been specified. Despite these configurations, the option to choose between alternative materials is not available.

Environment

  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

Create sales orders via Fiori app "Create Sales Orders - Automatic Extraction."

Resolution

This is the standard system behavior.

This Fiori app creates the sales order in the background and product substitution does not work in the background.

See Also

SAP Note 1702275 - Manual product selection and background processes.

Keywords

product substitution, background, Fiori app
