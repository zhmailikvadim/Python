Symptom

  • The retention period for "Action Logs" in SAP Cloud ALM is unknown.
  • No official documentation is available describing the retention period or whether it is configurable.

Environment

SAP Cloud ALM

Reproducing the Issue

  1. Access the SAP Cloud ALM tenant.
  2. Navigate to "Administration".
  3. Open "Landscape Management".
  4. Select any system and click on "Action Log".
  5. Review the logs displayed.

Cause

Resolution

In Landscape Manager (LMS), the retention period for Action Logs is 90 days. The retention period is currently not configurable.

See Also

Refer to: 3416639 - How to configure the data retention (housekeeping) in SAP Cloud ALM Health Monitoring.
