Symptom

When performing invoice run in app 'Create Invoices for Rent' for rental contracts, error RERAIV045 - 'No relevant cash flow items were found in the period' is shown and invoices are not generated.

OR

When performing periodic posting in app 'Periodic Posting: Contracts', error RERAPP012 - 'No cash flow planned records found in selected period' is shown and periodic posting cannot be executed.

Environment

SAP S/4HANA Cloud Public Edition

Cause

  • The system optimizes cash flow items for long-running contracts and does not save all items to the database during the initial contract setup.
  • This can lead to cash flow items not being recognized during periodic posting/invoice run for lease contracts.

Resolution

  1. Access the app 'Cash Flow Update for Contract'.
  2. Select the affected contracts and enable the 'Force regeneration' parameter.
  3. Run the cash flow update in 'update mode' (ensure the 'Simulation' checkbox is unchecked).
  4. Perform Invoice run/Periodic posting process again.
  5. Verify that the cash flow items are recalculated and the error is resolved.
