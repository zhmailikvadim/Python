Symptom

Incorrect flexible constraints heuristics results for the product having co-product as its component.

Other Terms

PP/DS, Production Planning and Detailed Scheduling, PP/DS for SAP S/4HANA, Synchronized Planning for Production, Production Planning Integration, SAP Integrated Business Planning for Supply Chain(SAP IBP), Flexible Constraints Heuristics, Planning for Std. Lots(Flex. Constr.), SAP_PP_FLXC, /SAPAPO/PPT1, SAPAPO/RRP3.

Reason and Prerequisites

When doing the GR in current bucket for the planned order/production order having co-product, finished product GR quantity is updated incorrectly in the confirmed quantity table as the co-product quantity is also getting updated as main product quantity, this is causing incorrect results in flexible constraints heuristics run for finished product.

Solution

Implement the following correction instruction or install the corresponding support package.

As general rule, SAP recommends that you install a solution by applying a Support Package. However, if you need to install a solution earlier, use the Note Assistant to implement the correction instruction.
