Symptom

  • Application error occurs when creating a new Time Recording Profile Visualization.
  • Error message: "An application error occurred. Please try again later, or notify support (with the error information of errorID=XXX, timestamp=XXX, and the version information from the current screen) if the issue keeps occurring."

Environment

SAP SuccessFactors Employee Center - Time Sheet

Reproducing the Issue

  1. Navigate to Manage Data.
  2. Select Create New.
  3. Choose Time Recording Profile Visualization.
  4. Enter any Time Recording Profile (TRP).
  5. Observe the application error message.

Cause

  • This is a Known Error. 
  • The system checks for the existence of Time Tracking entities (such as the Start of Time Record), but this functionality should also be available to customers who do not have the Time Tracking feature enabled.

Resolution

Permanent Fix:

This fix for this issue is planned to be deployed on 1H 2026 release.
For release timelines, please review SAP SuccessFactors Product Release & Road Map Information. 

Workaround

Customers who have a Time Tracking license can use the feature without the permanent fix.
However, there is no available workaround for customers who do not have a Time Tracking license.
We apologize for any inconvenience this issue may cause and appreciate your understanding while we work on a permanent resolution.
