Symptom

You are running the Accounts Receivable periodic job "SAP Digital Payments: Advice Processing". 
Multiple (virtual) Bank Statements are created even if the selection is restricted to one Merchant Alias.

Other Terms

Digital Advice, Schedule Accounts Receivable Jobs 

Reason and Prerequisites

You are using SAP S/4HANA Public Cloud Edition 2602 or higher.

The Digital Advice program was changed to improve the system performance. With this change the Bank Statements are splitted per Merchant ID (even, if they assigned to the same Merchant Alias).

Solution

Information only.  The reconciliation functionality has not been changed.
