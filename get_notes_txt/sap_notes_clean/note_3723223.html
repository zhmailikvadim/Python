Symptom

This article explains how to sync Users from SuccessFactors HCM to SmartRecruiters and highlights some important details about the process.

Environment

SmartRecruiters for SAP SuccessFactors

Resolution

Important Details:

  • User synchronization happens only from SuccessFactors to SmartRecruiters.
  • The synchronization starts after enabling the Enable SmartRecruiters switch in Provisioning.
  • After the first sync, the + Add new user button under SmartRecruiters → Settings → User management is disabled.
  • The sync duration depends on the number of Users and may take several hours.
  • After enabling the Provisioning switch, the sync job runs every hour.
  • Users are created in SmartRecruiters with Employee as the System Role.
  • User details in SmartRecruiters (such as Name and Email) cannot be changed in SmartRecruiters. Changes must be done in SuccessFactors.

Required User Data

The following fields must have values for the sync to work:

  • email
  • firstName
  • lastName
  • userName
  • personUUID
  • status

If any of these fields are missing, the User will not be synced.

Email Address Rules

Each SmartRecruiters User must have a unique Email Address:

  • The same Email Address (for example, user@company.com) can be used for only one User in SmartRecruiters.
  • If the same person needs access to multiple environments (for example, Production and Sandbox), different Email Addresses must be used.
    • Example:
      • user@company.com (Production)
      • user+sbx@company.com (Sandbox)
  • Email Addresses must be unique across all SmartRecruiters environments.
  • Email Address cannot be empty.

If Users share the same Email Address or if the Email Address is missing, the sync will fail.

How to Run the Sync Manually

If the sync fails or needs to run immediately:

  1. Go to Admin Center → Scheduled Job Manager
  2. Find the job Synchronize Users from SuccessFactors HCM to SmartRecruiters
  3. Select View Details
  4. Click Run it now

Access to SmartRecruiters

After Users are synced, access must be granted in permissions:

  1. Go to Admin Center → Manage Permission Roles;
  2. Find and edit the role that you want to give access to;
  3. Under User Permissions, select SmartRecruiters;
  4. Select the permission Access SmartRecruiters.

Frequently Asked Questions:

  • Can I see details of the sync job (for example, number of Users imported)?
    • No. The sync job does not show details about the data being processed. This may be added in future updates.
  • Can I see the full list of Users imported from SuccessFactors?
    • No. There is no option to view the full list of imported Users in SmartRecruiters.
