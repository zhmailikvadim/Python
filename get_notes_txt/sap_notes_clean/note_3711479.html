Symptom

When trying to generate eSocial event s-1010, you receive the error message "Nenhuma entrada na tabela T512W para XXXX", where XXXX is a wage type. 
When you check the table T512W, the entry is correctly created for the mentioned wage type.

Environment

  • HR renewal 

  • SAP S/4HANA 

  • SAP ERP  

  • SAP SuccessFactors Employee Central Payroll 

Cause

The error message "Nenhuma entrada na tabela T512W para XXXX" occurs when there is a delimitation in table T596J for the mentioned wage type that precedes the start date of the wage type in table T512W. 

This misalignment between the two tables is the root cause of the issue.  

Resolution

To resolve this issue, it is necessary to review and adjust the customizing settings to ensure that the entries in tables T596J and T512W are properly aligned. 
If a wage type is delimited in table T596J, the same wage type must exist in table T512W with an appropriate start date that matches the delimitation date in T596J.
