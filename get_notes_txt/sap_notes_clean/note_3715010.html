Symptom

You want to implement the corrections described in SAP Note 3715064. These corrections require additional changes that are usually handled as manual tasks. This note is used to cover these changes automatically.

Other Terms

FSM, Connector, User, /PACG/ECM_USROL, Inactive

Reason and Prerequisites

This note implements an ABAP report that performs the required changes.

Solution

1. Implement the attached correction instructions.

2. Run report /PACG/ECM_PACG_NOTE_3715064 or /PACG/ECM_S4PACG_NOTE_3715064 depending on your system version.

3. Implement the note 3715064.
