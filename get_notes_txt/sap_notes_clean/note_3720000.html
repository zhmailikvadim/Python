Symptom

 You want to implement the SAP Note 3510060 - "DC: Bottled Gas Scenario"

Solution

Implement this SAP Note in a separate mode and NOT in a queue with SAP Note 3510060.

AFTER the implementation, execute the manual activity as described in the post-implementation instruction. This execution will create or change DDIC-objects needed for SAP Note 3510060; if they would be missing, the implementation of that SAP Note will fail.
