Symptom

In an application that supports the Manage Views feature e.g. Manage Service Orders, the option to save views is missing.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Go to any supporting application e.g. Manage Service Orders.
  2. Select a parameter(s) in any of the fields.
  3. Click on the Select View dropdown next to the name of the current view.

The "Save as" option does not appear.

Cause

The "UI5FLEX_ENABLE_VARIANTS" parameter in the Manage Launchpad Settings application is configured as "NONE", which prohibits all users from using the Manage Views feature.

Resolution

To save views the "UI5FLEX_ENABLE_VARIANTS" parameter must be configured as either "PRIVATE" or "PUBLIC"

  1. Manage Launchpad Settings.
  2. Select Edit.
  3. Set the "UI5FLEX_ENABLE_VARIANTS" to either PRIVATE or PUBLIC.
  4. Save.
