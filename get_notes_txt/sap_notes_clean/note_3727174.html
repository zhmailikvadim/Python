Symptom

Symptom

In the process "Integration of Meter Operator Contracts" (DE_B_MOSB_INTG_PROC), step 0040 (Fetch Master Data) fails to fetch the original contract with type CMDC during a device change from CM to MM/IM. This failure prevents the original contract from being terminated.

This issue is relevant for the market role: MOS.

Other Terms

Integration of Meter Operator Contracts, DE_B_MOSB_INTG_PROC, Fetch Master Data, /US4G/CL_CHK_MOSB_PROCESS->CHK_INTG_FETCH_CTR, CMDC, device change, CM, MM, IM

Reason and Prerequisites

Program error.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the system successfully fetches the original contract with type CMDC in step 0040 (Fetch Master Data) during a device change from CM to MM/IM, allowing the original contract to be terminated. You need to retest the process. You can verify the fix by testing that the original contract termination completes successfully during a device change from CM to MM/IM.

Other Terms

LE-TM Integration; Logistic Integration

Reason and Prerequisites

Because of a programming error the relevant field was not mapped.

Solution

Implement the correction instruction attached to this Note or install the corresponding support package. Afterwards, the relevant field is mapped and the Package Material Type shown on the UI.
