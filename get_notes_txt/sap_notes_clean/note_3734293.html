Symptom

Component: Product Safety / Dangerous Goods Management
Module: Report Shipping / Dangerous Goods Documents

When you run transaction CVD1 (Edit Report Shipping Orders) in the system, the system displays no label or an incorrect label on the user interface for certain tremcard parameter values.

Other Terms

CVD1
REPDELVTRM
REPSHIPTRM
RDGCOUNTRYREGLANG VALDAT
UMB

Reason and Prerequisites

Reason
A program error in the label determination logic causes this behavior.

Prerequisites
For information about the validity of the corrections, see the correction instructions.

Solution

After implementing this SAP Note, the system displays the correct label on the user interface in transaction CVD1 for all relevant parameter values.

The Support Packages section lists the support packages that contain the corrections.

Install a listed support package or implement the attached correction instructions.
