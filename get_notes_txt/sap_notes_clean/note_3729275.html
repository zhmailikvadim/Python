Symptom

  • When running report SCWN_PREREQUISITE_CALC_SWI or selecting "Display Prerequisite SAP Notes" option in transaction SNOTE to display the prerequisite SAP Notes of a specific SAP Note, a time-out error occurs.
  • Reference to KBA 3587778 to increase the maximum runtime of a work process does not resolve the time-out error.
  • The error is caused by the report SCWN_PREREQUISITE_CALC_SWI entering an infinite loop. 

Environment

  • Note Assistant(SNOTE)

Reproducing the Issue

  1. Open transaction SNOTE.
  2. Select "Display Prerequisite SAP Notes" option to display the prerequisite SAP Notes of a specific SAP Note.
  3. Observe the short dump caused by a time-out error.

Or 

  1. Open transaction SA38/SE38.
  2. Execute the report SCWN_PREREQUISITE_CALC_SWI.
  3. Enter a specific SAP Note to display its the prerequisite SAP Notes.
  4. Observe the short dump caused by a time-out error.

Cause

  • Program error.

Resolution

Implement SAP Note 2718168 - SAP Note Assistant: Troubleshooting Reports run for a long time.
