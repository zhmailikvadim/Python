Symptom

  • New buttons to add Efforts, Material, Mileage and Expenses (EMME) appeared on the activity main screen in the FSM mobile App unexpectedly.
  • Users reported this issue suddenly, and the FSM administrator could not see the change even when checking the release notes of the most recent release.

Environment

SAP Field Service Management Mobile Applications, version 2511 or later

Reproducing the Issue

  1. Access the activity main screen in the Windows App.
  2. Observe the appearance of new buttons labelled "Create Effort" or "Create Expense" or "Take it" below the header information of the activity.

Cause

  • This feature was released in app version 25.11.0 (November 2025).
  • The buttons do not appear initially in old or existing screen configurations as they were not configured.
  • Upon editing the screen configuration once, the section for the quick-action-bar was added, causing the buttons to appear.

Resolution

  1. Review the release notes for app version 25.11.0 for the announcement on the new feature.
  2. Review the documentation of the Quick Action Bar feature for details on functionality and configuration.
  3. Understand that the feature is opt-in and becomes active upon editing the screen configuration. (See screenshot below)
  4. Note that a message appears on the screen when the quick-action-bar section is added during configuration edits.
  5. If deactivation is required, consult the documentation or SAP Support for further guidance.
