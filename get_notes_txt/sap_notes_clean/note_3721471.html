Symptom

  • Non admin users are able to access and delete the instances from the Instance Manager.
  • Non admin users should not be able to delete the instances from the Instance Manager.

Environment

  • SAP BusinessObjects Business Intelligence Platform 4.x & 2025
  • All supported OS

Reproducing the Issue

  1. Login to CMC with non admin user
  2. Navigate to Instance Manager 
  3. Delete button is visible

Cause

  • "Delete Objects" and "Delete Instances" rights are granted for the non admin users.

Resolution

  1. Navigate to CMC - Folders - Manage - Top level security - All Folders
  2. Click on Add Principals - select user/group - click on Add and Assign Security - Advanced - Add/Remove Rights
  3. In General section - deny the "Delete objects" and "Delete instances" rights - Apply & OK
  4. The Delete option is no longer available in the Instance Manager.

Note: Users are restricted from deleting objects as well.

Keywords

Instance Manager, delete, Objects, instances
