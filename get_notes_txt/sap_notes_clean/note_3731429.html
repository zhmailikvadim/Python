Symptom

Freight Order is missing or not visible for invoicing within the SAP Business Network for Logistics.

Resolution

To ensure the Freight Order is correctly transmitted to the carrier for invoicing, the shipper must manually trigger the settlement request within the SAP Transportation Management system by following these steps:

Calculate Charges > Update Execution Status to in-execution > Save the Document

This action triggers the outbound integration message that pushes the Freight Order data to the SAP Business Network for Logistics.
