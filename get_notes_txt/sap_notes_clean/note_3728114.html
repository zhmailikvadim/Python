Symptom

  • Error occurred during the TRANSMIT step: The exchange of electronic documents or report submission failed while performing the TRANSMIT action due to an unexpected error in the country specific process. Retry the operation, and if the issue persists, open a support request. Refer to the Monitoring and Troubleshooting section in the Help Portal documentation for the SAP Document and Reporting Compliance, cloud edition. 
  • Error occurred during the PUSH step: Connection to Business System D2S200 returned HTTP status code: 500
  • Error occurred during the TRANSMIT step: Java Script Error PL_KSeF_EInvoicing_DRCce_HTTP_Adapter_Router Process_68042103 CallActivity_68042123 [placeholder for image]javax.script.ScriptException: java.lang.Exception: CommunicationSettingsException: |XXXXXXXXXX is not configured in Partner Directory.|@ line 25 in GetCommunicationSettings.groovy|Process_68042103/CallActivity_68042123

Environment

  • SAP ERP 
  • SAP S/4HANA
  • SAP Document and Reporting Compliance Service

Reproducing the Issue

  1. Configure EInvoice for KSeF using SAP Document and Reporting Compliance (DRC).
  2. Attempt to submit an edocument to the KSeF portal.
  3. Observe the error message during the TRANSMIT step indicating an unexpected error in the country specific process.
  4. For plant abroad scenarios, observe that the system fetches the foreign VAT ID instead of the Polish VAT ID, causing the error.

Cause

  • For plant abroad scenarios, the system fetches the VAT ID based on the foreign country instead of the Polish VAT ID, leading to the error.

Resolution

  1. Implement SAP Note 3703478 to address the plant abroad scenario issue.
  2. Ensure all prerequisites mentioned in the overview note 3279206 Electronic Document Processing for Poland KSeF: Implementation Overview SAP for Me are implemented before proceeding.

Note: If the note 3703478 is already implemented, skip this solution and submit an incident to the component CA-GTF-CSC-EDO-PL.
