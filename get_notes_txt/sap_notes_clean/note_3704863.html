Symptom

After navigating from Balance sheet/income statement app to Display G/L Account Balances app, the loading never finishes. 

Environment

SAP Fiori for SAP S/4HANA

Reproducing the Issue

  1. Access the Balance Sheet/Income Statement app.
  2. Attempt to navigate to the Display G/L Balances link from Period Balances.
  3. Observe that the link does not open and displays the "502 Bad Gateway" error.

Cause

Sap note is missing.

Resolution

  1. Implement SAP Note 3465518 - Display G/L Account Balances navigation infinite loading.
  2. After implementing the note, clean the caches by recalculating the index of the app as described in SAP Note 2449635 for project FIN_GL_ACC_BAL (tab 'Single SAPUI5 Repository Only').

See Also

SAP note 3465518 - Display G/L Account Balances navigation infinite loading
