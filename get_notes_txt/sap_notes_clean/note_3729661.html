Symptom

  • An error occurs while executing transaction ABUMN to post an asset transfer.
  • Error message: AA335.

Environment

  • FIN Asset Accounting​
  • SAP ERP​
  • SAP ERP Central Component
  • ​SAP enhancement package for SAP ERP​
  • SAP enhancement package for SAP ERP, version for SAP HANA​
  • SAP S/4HANA ​
  • SAP S/4HANA Finance​
  • SAP S/4HANA Cloud Private Edition​

Reproducing the Issue

  1. Execute transaction ABUMN.
  2. Attempt to transfer a complete asset.
  3. Reference the current-year acquisition.
  4. Observe error message AA335.

Cause

  • The error AA335 occurs due to a mismatch between the asset values being transferred.
  • The asset contains only 'Prior-Year values,' but the transfer references 'Curr.-Year Acquisition.' or vice-versa.

Resolution

  1. Review the asset values in transaction AW01N to confirm the type of acquisition values associated with the asset.
  2. In transaction ABUMN, select 'Prior-Year Acquis.' instead of 'Curr.-Year Acquisition' for the transfer.
  3. Execute the asset transfer operation again.
  4. Verify that the error message AA335 no longer occurs.
