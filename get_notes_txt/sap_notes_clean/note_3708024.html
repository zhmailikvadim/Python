Symptom

The master data values like payment term, incoterms etc are not available for the answer type master data in the supplier template.

Environment

Supplier Lifecycle & Performance Registration

Reproducing the Issue

  1. Access the Supplier Lifecycle & Performance Registration system.
  2. Navigate to the Supplier Registration template configuration.
  3. Open the questionnaire either internal or external
  4. Locate the field
  5. Attempt to select the value for the field from the master data uploaded.
  6. Observe that the fields of master data are not available in the questionnaire.

Cause

  • Due to the incorrect system ID maintained.

Resolution

  1. Ensure that the system ID in the SM administration matches the system ID in the master data.
  2. If discrepancies are found, correct the system ID in the Supplier Management administration or reload the master data for the system ID maintained in Supplier Management administration.
