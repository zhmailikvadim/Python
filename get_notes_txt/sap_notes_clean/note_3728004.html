Symptom

During the sales document creation or change, when clicking the "Propose alternatives" button, there will be message VP119 “There are no alternatives available for partner XXX”. However, there are several such partners inside the partner function tab of customer master data.

Environment

  • Sales And Distribution (SD)
  • SAP ERP Central Component
  • SAP ERP
  • SAP Enhancement package for SAP ERP
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. create the sales order VA01
  2. enter the order type and sales area
  3. enter the sold-to party
  4. go to header partner tab
  5. select one partner function , for example the "Ship-to" party 
  6. click the "Propose alternatives" button

Cause

The relevant partner function is no change possible/not modifiable.

Resolution

Check the partner determination procedure settings via transaction VOPAN, and then remove the "No Modifiable" flag if required.

Keywords

VP119, Propose alternatives, V_TPAER_VK-AENDB, AENDB, TPAER, SD_PARTNER_ALTERNATIVES_SELECT
