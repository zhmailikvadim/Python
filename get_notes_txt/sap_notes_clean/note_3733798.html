Symptom

An error occurs during the activation of "DDL Source FGL_BCF_RUN_INFO" while applying SAP note 3134505.

The error messages are:
--------------------------------------------------
"The column TL_LEDGER is unknown"
"The column TL_DBCNT_PL is unknown"
"The column TL_DBCNT_BS is unknown"
"The column TL_NUM_IMSG is unknown"
"The column TL_NUM_SMSG is unknown"
"The column TL_NUM_WMSG is unknown"
"The column TL_NUM_EMSG is unknown"
"DDL source FGL_BCF_RUN_INFO could not be activated"
--------------------------------------------------

Environment

  • SAP S/4HANA
  • SAP S/4HANA Finance

Reproducing the Issue

  1. Apply note 3134505.
  2. Attempt to activate "DDL Source FGL_BCF_RUN_INFO".
  3. Observe the activation error and messages indicating unknown columns.

Cause

Resolution

  1. Ensure that note 3153144 - "BCF: Missing balance carryforward for technical ledger" is completely implemented (this note delivers the artifacts for FGL_BCF_RUN_INFO).
  2. Reapply note 3134505.
  3. Activate DDL source FGL_BCF_RUN_INFO again and confirm that no "unknown column" errors occur.

See Also

  • SAP Note 3153144 - BCF: Missing balance carryforward for technical ledger
  • SAP Note 3134505 - BCF: Always derive fiscal year variant from customizing
