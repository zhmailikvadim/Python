Symptom

For the determination of a goods receiving point, if you have maintained the same priority (TWLVZ-VPRIO) for more than entry of combination between plant and storage location, system will set the goods receiving point alphabetically.

Environment

  • Sales And Distribution (SD)
  • Logistics Execution (LE)
  • SAP R/3
  • SAP R/3 Enterprise 4.7
  • SAP ERP Central Component
  • SAP ERP
  • SAP Enhancement package for SAP ERP
  • SAP Enhancement package for SAP ERP, version for SAP HANA

Reproducing the Issue

  1. Go to VL31N
  2. Enter PO number
  3. Press Enter
  4. Go to Header
  5. Check the Receiving Point at Shipment tab 

Cause

 Having the same priority for the same combination of plant and storage location has not been taken into account so far.

Resolution

Apply the Note 2964267.

Keywords

TWLVZ, VPRIO, LT_TWLVZ, WANST, VL31N, goods receiving point
