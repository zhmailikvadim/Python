Symptom

After implementation of SAP Note 3416449, the auto-checkin from webgui (ITS mode) is not possible and instead suggests to check-in manually. Even when the user performs the check-in manually and chooses the latest file, the error regarding auto check-in reappears.

Other Terms

 file selection, file browser, auto checkin, TDWS, check-in flag.

Reason and Prerequisites

Pre-requisite: The switch 'WGUI_FL_BRWSR_CHECKIN' is enabled as per SAP Note 3416449.

Solution

 Implement the instructions attached.
