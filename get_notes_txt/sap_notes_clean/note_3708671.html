Symptom

  • An error occurs while enabling Best Practice (e.g. J48).
  • "Creating a new entries is not allowed"
  • "Activation of customizing object V_T030_LDT with errors"

Environment

SAP S/4HANA

Reproducing the Issue

Attempt to enable Best Practice in SAP S/4HANA.

Cause

The issue occurs because BCTOOLS_CUST_VARIANT_APPLIED has not been activated.

Resolution

  1. Activate BC Set BCTOOLS_CUST_VARIANT_APPLIED manually in transaction SCPR20 in every client where BP content activation is planned; SAP Note: 3633481.
  2. Re-activate the activities that were previously failing.

See Also

Refer to: 0003633481 - SAP Best Practices for SAP S/4HANA Cloud Private Edition 2025 FP-Stack 00 (10/2025) Content Activation Note - SAP for Me
