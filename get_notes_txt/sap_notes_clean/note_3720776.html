Symptom

You want to update the patch level of the installed 1.120 SAPUI5 release in an ABAP system.

Other Terms

SAPUI5 version 1.120.43, UI5 version 1.120.43, SAPUI5 version patch 1.120.43, UI5 version patch 1.120.43, SAPUI5 1.120.43, UI5 1.120.43

Reason and Prerequisites

Program error.

SAP_UI 758 SP01 or higher is installed

Solution

Implement the transport-based correction instruction (TCI) specified in this SAP note.
You can find more information about handling transport-based correction instructions in SAP Note 2187425.

SAPUI5 resources no longer reside in the MIME Repository, and there may be obsolete content that needs to be deleted. The following steps are optional and can be performed if you need to clean up legacy MIME content:

  • Start transaction RZ11.
  • Increase the following parameters to 120m:
    • rdisp/scheduler/prio_normal/max_runtime
    • rdisp/scheduler/prio_high/max_runtime
  • Start transaction SE38 and run the report /UI5/MIME_CLEANUP.
  • Ensure the system is configured with at least 6 dialog work processes before running the report.
  • After the report completes, restore the parameters in RZ11 to their defaults:
    • rdisp/scheduler/prio_normal/max_runtime = 60m
    • rdisp/scheduler/prio_high/max_runtime = 11m
