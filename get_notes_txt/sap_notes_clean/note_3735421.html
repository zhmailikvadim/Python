Symptom

You are applying note 3697099 and you have queries regarding implementing as your system is out of mainstream maintenance.  

Environment

 SAP Customer Relationship Management - SAP CRM 2007 (with SAP CRM UIF 600) 

Reproducing the Issue

  1. Download (do not implement) the SAP Note 3697099.
  2. Check the implementation status message shown by the note (for example, "can be implemented" or "cannot implement").

Cause

  • The system release/UI component is out of mainstream maintenance (eg- SAP CRM 2007), so no code correction is provided in the note for that release; only the manual workaround applies.

Resolution

  1. Verify the note applicability by downloading the note (do not implement) and reviewing the implementation status message.
  2. Follow the manual workaround described in the note to deactivate service CRM_IC_ISE.
  3. If service CRM_IC_ISE is not available or is not active in the system, no action is required and the note can be ignored.
