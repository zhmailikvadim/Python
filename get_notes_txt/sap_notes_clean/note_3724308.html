Symptom

When accessing Cost Center Budget report in Fiori, a message is displayed: "invalid data is displayed (some measures have different units)", and no results are shown.

Environment

  • Controlling (CO)
  • SAP S/4HANA
  • SAP S/4HANA Finance
  • SAP S/4HANA Cloud Private Edition
  • SAP Fiori for SAP S/4HANA

Reproducing the Issue

  1. Execute the Cost Center Budget Report in the Fiori App.
  2. Input controlling area and fiscal year as mandatory parameters.
  3. Message appears: "invalid data is displayed (some measures have different units)"

Cause

The controlling area includes company codes with multiple currencies. Due to this configuration, the Fiori app is unable to display the multiple currencies on the same data point. 

Resolution

Use filters such as Company Code or Budget Carrying Cost Centers, then the Fiori app will display correctly.
