Symptom

  • The issue occurs within Analytics Stories where clicking the 'Data Refresh' button does not update the data; changes are only reflected after refreshing the browser session.
  • The issue began after the Upgrade 2602.
  • No getResponse calls are sent when pressing 'Data Refresh'.

Environment

  • SAP Integrated Business Planning (IBP) 2602 

    • Manage Analytics Stories Application

Reproducing the Issue

  1. Navigate to "Manage Analytic Story".
  2. Open the story in view mode.
  3. Click the 'Data refresh' button.
    => UI is reloaded but data is not refreshed.
  4. Open Developer Tools and navigate to the Network tab.
  5. Press the 'Data Refresh' button again.
    => No getResponse calls are observed in the network trace.

      Cause

      This issue is currently under investigation by development.
