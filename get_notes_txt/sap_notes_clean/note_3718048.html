Symptom

What are the supported special characters in a team member rules (TMR) file?

Resolution

The supported special characters in a TMR file include:

Asterisk/wildcard (*): Matches any value for that field.

Tilde (~): Used for range expressions on numeric fields.

We can also use colons (:) when adding multiple users or groups in one row, as separators.

Example: user1:user2:user3.
