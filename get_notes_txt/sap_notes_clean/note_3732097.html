Symptom

  • Refresh scheduled vis instance refresh tool has status failed with status validation failed. 
  • The refresh scheduled with successfully validation is getting reverted. 

Error Codes : 
1. TENANTREFRESH_REQUEST_CONTAINS_INVALID_DATA
2. Encouterred Exception while fetching validation info

Environment

SAP SuccessFactors HCM Suite

Cause

Incorrect CMW Time Format configured in IRT is causing this issue. 

Resolution

Engineering team has fixed the issue. Please retry to schedule the refresh.
