Symptom

In a non-CWM system the Function Module 'MB_SELECT_STOCK_IN_TRANSIT' and the subroutine SELECT_PREPARE_SIT_RESULT unnecessarily calculate CWM-specific data. This may lead to an increased runtime in transactions using this Function Module, such as MB5SIT. In an ST12 ABAP trace of the transaction calling this Function Module you notice an expensive statement involving internal table LT_/CWM/VBFA_MSEG (such as "READ TABLE LT_/CWM/VBFA_MSEG").

Other Terms

LMBBSF02, SIT, stock in transit

Reason and Prerequisites

The Business Function LOG_MM_SIT is active.

The Business Function /CWM/CM_1 is not active

Solution

The correction is delivered with the corresponding Support Packages. For advance access implement the Corection Instructions.
