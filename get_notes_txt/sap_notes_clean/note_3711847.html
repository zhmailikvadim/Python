Symptom

In freight unit building, the bin packing heuristic worst-fit-decreasing was used up to now to assign items to freight units in scenarios that don't allow item splits. This heuristic could lead to the following issues:

  • Wasted space within freight units resulting in suboptimal capacity utilization
  • Higher number of freight units than expected because an inefficient item-to-bin assignment caused the unnecessary creation of additional freight units

Therefore, the default bin packing heuristic has now been changed to the first-fit-decreasing bin packing heuristic. This heuristic has the following benefits:

  • It reduces the wasted space within freight units.
  • It avoids the creation of unnecessary additional freight units.
  • It improves the overall system performance.

To allow customers to choose between the different bin packing heuristics via enhancement, a new parameter, /SCMTMS/CL_FU_BUILDER_CORE=>MV_FUB_ALGORITHM, has been introduced.

Other Terms

Freight Unit Building, FUB, Bin Packing, First-Fit-Decreasing, Best-Fit-Decreasing, Worst-Fit-Decreasing, MV_FUB_ALGORITHM, /SCMTMS/CL_FU_BUILDER_CORE

Reason and Prerequisites

Program Error

Solution

Implement the attached correction instruction or the mentioned support package.
