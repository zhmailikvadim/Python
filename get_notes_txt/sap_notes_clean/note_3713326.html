Symptom

The expected change processes (e.g., Cancel) are not available in the Processes assignment block. When checking the system status of this Subscription Contract Document, the I1030 (Contains Errors) exists in the document, but no error message existing in the document.

Environment

SAP S/4HANA Cloud Private Edition

Resolution

To resolve this issue, please execute the report CRM_MESSAGES_CHECK for the Transaction ID (not the contract number) and the transaction type with the 'Execute update' field selected, then the document will be fixed and the Cancel process should be available.

Keywords

"Change Processes", "Change Process", "Process", "Cancel"
