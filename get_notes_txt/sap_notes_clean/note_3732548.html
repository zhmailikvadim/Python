Symptom

You approve a workflow task created for posting a journal entry from import group journal entries and you receive the error message:

"Following error occurred in validation phase of posting process: Document type XX is not allowed for manual posting".

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Go to the app My Inbox - Group Journal Entries.
  2. Approve the task to post the journal entry.

An error occur: "Following error occurred in validation phase of posting process: Document type XX is not allowed for manual posting".

Cause

The submitter of the task have not set the Global Parameters.

Resolution

Once the submitter set the Global Parameters, all relevant tables will be updated and the error will not occur anymore.
