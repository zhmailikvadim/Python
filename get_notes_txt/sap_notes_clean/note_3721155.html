Symptom

 The following issues are addressed:

- Taxes calculation is performed for documetns not in scope of the report

- Calculated exchange rate causes error: not enough space for packed decimal

Other Terms

 Bulgaria, BG, SAF-T

Reason and Prerequisites

This SAP Note is relevant for Bulgaria SAF-T.

Please implement Note 3703066 before implementing this Note.

Solution

As a general rule SAP recommends that you implement the Support Package specified for your release. However, if you need to install a solution earlier, use the SAP Note Assistant to implement the correction instruction.
