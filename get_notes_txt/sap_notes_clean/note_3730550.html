Symptom

You are unable to edit quick selections which have the flag "Provided by SAP", e.g. SAP Default, SAP Default Retail UI, SAP Default for mobile devices

The pencil icon and the add / delete buttons are not shown.

Environment

SAP Customer Checkout manager, cloud edition

SAP Customer Checkout manager

Cause

Quick selections of type "Provided by SAP" can only be edited by SAP, for example as part of a new patch or feature pack.

Only quick selections which do not have the flag "Provided by SAP" can be edited.

Resolution

If you want to use a quick selection flagged "Provided by SAP" you can create a copy of it by tapping the 3 dots next to the magnifying glass.

Keywords
