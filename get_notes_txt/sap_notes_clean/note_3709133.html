Symptom

  • While creating the Web Intelligence report based on any UNX universe, below error is observed:
    >> WIS 30270 An internal error occurred while calling the 'processDPCommandsEx' API. (Error: ERR_WIS_30270) 
  • Also, while refreshing the existing Web Intelligence documents based UNX universe, below error is seen:
    >> The action cannot be performed.

Environment

  • SAP BusinessObjects Business Intelligence 4.3 Support Package3 Patch11
  • Web Intelligence

Reproducing the Issue

  • Login the BI Launchpad.
  • Select Application > click Web Intelligence.
  • Select the data source as Universe.
  • Select any UNX universe from the list.
  • Observe the error: 
    >> WIS 30270 An internal error occurred while calling the 'processDPCommandsEx' API. (Error: ERR_WIS_30270) 

Cause

The Adaptive Processing Server hosting DSL Bridge service could either be stopped or disabled. 

OR the service could have been corrupted.

Resolution

  • Ensure all the Adaptive Processing Server hosting DSL Bridge service are running and enabled. 

Keywords

dsl, wis30270, processDP, action cannot be performed, dsl disbaled.
