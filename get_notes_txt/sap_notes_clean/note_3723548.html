Symptom

The reporting authority has issued a revised annual statistics PDF template. In this updated version, several data fields previously required have been removed. Consequently, the existing system template no longer matches the official legal format.

Other Terms

HBGLSTY0; Annual Statistics; Legal Change

Reason and Prerequisites

This SAP Note is only relevant for Bulgaria(HRBG)

Solution

The standard PDF template HR_BG_HBGLSTY0 has been updated to reflect these regulatory changes. Obsolete fields have been removed to ensure the generated output is fully compliant with the new requirements introduced by the authority.

Implementation

As a general rule SAP recommends that you implement the Support Package specified for your release. However, if you need to install a solution earlier, use the SAP Note Assistant to implement the correction instruction. You can find more information about the SAP Note Assistant in SAP Support Portal, under SAP Note Assistant.
