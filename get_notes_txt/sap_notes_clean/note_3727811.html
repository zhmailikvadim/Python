Symptom

  • The latest homepage experience has been enabled to the instance.
  • There is no option to upload company branding or theme on the banner.

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

SAP SuccessFactors HCM Suite

Cause

  • The banner image at the top of the page is removed in the latest homepage experience.
  • This is intended behavior by design:

In the legacy home page, a large banner image served as the primary branding element at the top of the screen. In the Latest Home Page experience, SAP SuccessFactors has transitioned to a cleaner, action-oriented design.

Resolution

Changes Included in the Latest Home Page Experience

Learn about key differences between the latest home page experience and the previous experience, for both users and administrators.

When you enable the latest experience at Company System and Logo Settings  Enable the latest Home Page experience.

    User Experience:

    No banner image

    The banner image at the top of the page is removed. In the latest experience, banner cards images can be used in custom content, but they're not used as branding for the overall home page.

    In the legacy experience, a large banner image is shown at the top of the page, before all content and actions.
