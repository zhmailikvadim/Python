Symptom

  • An error occurs during the implementation of SAP Note 3594333 due to a missing dependency.

  • The system throws the error message: “Runtime object for &TABLE& could not be read.”

  • While attempting to implement SAP Note 2685547 as part of troubleshooting, the activation fails.

  • Activation error repeatedly shows: “The column APMBANKACCOUNTSTATUS is unknown.”

  • The implementation attempt was carried out in the S4D100 development system.

  • All prerequisite SAP Notes mentioned in SAP Note 3594333 are marked as obsolete.

  • The system connection was successfully opened, and required credentials were maintained.

Environment

  • SAP S/4HANA Cloud Private Edition 2023 (Support Package: 108)

Reproducing the Issue

  • In the development system, attempt to implement SAP Note 3594333.

  • During the activation phase, the system throws the error:
    “Runtime object for &TABLE& could not be read.”

  • As a follow-up, attempt to implement SAP Note 2685547.

  • The activation of SAP Note 2685547 fails with the repeated error:
    “The column APMBANKACCOUNTSTATUS is unknown.”

Cause

  • Activation of SAP Note 3594333 failed, and the subsequent deimplementation also resulted in an error, leaving the system in an inconsistent state.

  • Reattempting the note implementation triggers the error “Runtime object for &TABLE& could not be read.”

  • Following SAP Note 2685547 was not successful, as activation fails with repeated errors stating “The column APMBANKACCOUNTSTATUS is unknown,” indicating missing or inconsistent database objects.

Resolution

  • Execute report RUTDDLSACTSNOTE with SAP Note 3594333, and then proceed with the implementation of the SAP Note.
  • If the issue persists after following the first step, proceed with the second step and check whether the CDS view I_APMPAYMENTITEM is missing in the error log.
  • If yes, open the CDS view I_APMPAYMENTITEM and update line 2 and line 142 as per the provided screenshot.
