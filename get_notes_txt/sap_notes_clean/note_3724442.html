Symptom

  • Comp-a-ratio and range penetration fields are not visible in the compensation details page when using the Latest People Profile.
  • These fields were visible in the old people profile or through History.

Environment

SAP SuccessFactors Employee Central

Reproducing the Issue

  1. Navigate to the compensation category.
  2. Open the compensation information card and click "View All".
  3. Observe that the fields "Comp-a-ratio" and "Range penetration" are missing on the compensation details page.
  4. Open the History to view the values.

Cause

The behavior occurs because the fields "Comp-a-ratio" and "Range penetration" are currently maintained only in the Country-Specific Compensation Information.

Resolution

Expected System Behavior. To make the fields appear in View All section it's necessary to add these fields in the general Compensation Information HRIS Elements.
