Symptom

After enabling the Latest People Profile clicking "Full Profile" opens the employee profile in the same tab instead of a new tab, disrupting navigation.

Environment

  • Compensation
  • Latest People Profile

Reproducing the Issue

1. Enable the Latest People Profile.
2. Navigate to the Compensation process and click "Full Profile."
3. Observe that the profile opens in the same tab instead of a new tab.

Cause

The "Full Profile" opening behavior current system design.

Resolution

It is current system design. However, as a workaround in full profile right click the link and "Open in a new tab."
