Symptom

You run the task Reported Data  Validation and you receive the error: "Rule group not assigned for S1080 / Y10 / 006"  

S1080 stands for the ID of the task and Y10 is the version.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Go to the Data Monitor app.
  2. Run the task Reported Data Validation.

You received the error message: "Rule group not assigned for S1080 / Y10"

Cause

You have not assigned the validation task to a rule group.

Resolution

In the app '"Assign Rule Group for Balance Validation", you need to assign the task S1080 / Version Y10 to a rule group.

Keywords

Reported Data Validation, Rule Group, Task S1080, Data Monitor, Assign Rule Group for Balance Validation.
