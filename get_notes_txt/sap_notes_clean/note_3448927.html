Symptom

Currently in the standard solution of Education Grant (0965), advance recovery amounts are incorrectly reported in the ARREAR table during the payroll simulation. This is because the assignment number is not read from the Additional Off-Cycle Payments (0267) infotype for existing advance payments.

Other Terms

  • Education Grant
  • Infotype 0965
  • Assignment number
  • Advance recovery

Reason and Prerequisites

Correction of legal function.

This SAP Note is relevant only for non-profit organizations.

Solution

To get the changes, import the Support Package specified for your release.

Alternatively, apply the attached correction instructions.

As a general rule, SAP recommends that you implement the Support Package specified for your release. However, if you need to install a solution earlier, use the SAP Note Assistant to implement the correction instruction. You can find more information about the SAP Note Assistant in SAP Support Portal, under SAP Note Assistant.
