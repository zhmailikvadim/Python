Cause

  • Problem is related to executing more than one SELECT statement, each containing COMPUTE clause(s).
  • The issue has been identified and logged under SAP CR 786821. Its description reads, "When executing SQL statements where there are more than one SELECT with COMPUTE clause, error occurs: "System.ArgumentException: Item has already been added. Key in dictionary: '2'  Key being added: '2'".

Resolution

This issue has been addressed in:

  • SAP SDK 15.7 SP 136 and higher
  • SAP SDK 16.0 SP02 PL02 and higher

OCS, ADONET, 78682, CR#786821, CR786821
