Symptom

  • Need to restart or rerun data collection in Process Insights
  • Data collection requires manual refresh outside of scheduled Daily/Weekly updates
  • Process Intelligence data needs to be updated on demand

Resolution

To rerun data collection in Process Insights, follow these steps:

  1. Navigate to transaction /SDF/PINS_SETUP
  2. Expand section 6. Additional options
  3. Select Restart Data Collection
