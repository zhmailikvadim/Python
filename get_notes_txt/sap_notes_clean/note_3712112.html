Symptom

  • Replication of BOM updates from SAP S/4HANA to SAP Digital Manufacturing Cloud (SAP DMC) fails when using the "Replicate by Object Selection" app.
  • SAP DMC log indicates the presence of a blank item node generated by SAP S/4HANA for deleted BOM items.

Environment

  • SAP S/4HANA Cloud Public Edition
  • SAP Digital Manufacturing Cloud

Reproducing the Issue

  1. Update a BOM in SAP S/4HANA.
  2. Use the "Replicate by Object Selection" app to replicate the BOM update to SAP DMC.
  3. Observe the replication failure in SAP DMC.

Cause

The ERP transaction 003 Delete Routing is not supported in DM. 

Resolution

The only way to delete any master object in DM is to delete it with appropriate public API call or from UI.
