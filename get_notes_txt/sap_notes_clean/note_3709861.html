Symptom

Whenever an exception is raised in POS Inbound, payload will not get staged for the subsequent tests in SPROXY and AIF will not log any message.

Other Terms

Point-of-sale, POS, POS Inbound, Billing document, Aggregated Sales, Sales-per-receipt

Reason and Prerequisites

This is due to a program error.

Solution


Kindly implement the attached correction instructions.

After implementation, payload will get staged and AIF will log an error message for all the exceptions raised in POS Inbound.
