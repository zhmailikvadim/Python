Symptom

System throws the Error "Reduction of value domain not permitted" when attempting to reduce the End value of a product property.

Environment

SAP Business ByDesign

Reproducing the Issue

  1. Go to the product development work center.
  2. Select the product properties view.
  3. Select all properties from the dropdown menu.
  4. Search for the relevant product property.
  5. Select and choose Edit.
  6. Enter the property value (End) with a value less than the current value.
  7. While saving the system throws the error message: Reduction of value domain not permitted.

Cause

Once a product property is saved, the existing property values, ID, and format are set to read-only.

For integers, decimal numbers, and quantities, it is possible to edit the values, but reducing the value ranges is not permitted.

Resolution

This is the expected system behavior.
