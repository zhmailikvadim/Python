Symptom

This SAP Note is relevant for you only if you use SAP S/4 HANA Utilities for EEG Billing of German Electricity or the TSO notification solution under SAP EEG Billing for German Electricity Feed.

The start date of an IMS device is not taken into account within the billing period. If the metering procedure is changed, the system must create a new settlement subperiod so that the correct compensation code can be determined.
