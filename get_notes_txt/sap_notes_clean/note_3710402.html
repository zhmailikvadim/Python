Symptom

The system allows the creation of billing documents using despite the application of required billing blocks to specific business partners.

The following error message is expected: "*** is blocked for billing. (VF068)." However, the invoice can be created, there is no error that prevents the creation. 

Environment

  • Sales And Distribution (SD)
  • SAP R/3
  • SAP R/3 Enterprise 4.7
  • SAP ERP Central Component
  • SAP ERP
  • SAP Enhancement package for SAP ERP
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA
  • S/4HANA Cloud Private Edition

Reproducing the Issue

  1. In transaction code BP apply Billing Block in Sales area for the partner(s).
  2. Attempt to generate the Billing Document using the Sales Order via Transaction Code VF01.
  3. Observe that the system allows the creation of the Billing Document despite the applied billing block.

Cause

The issue can occur if only the partner(s) other than the Payer have billing block set, but the Payer has no billing block. As a result, the system allows the creation of billing documents despite the block applied to the other partner(s). The billing creation is only prevented if the Payer is blocked for billing. 

Resolution

  • Apply a billing block to the "Payer" in Transaction BP.
  • Verify that the system prevents the creation of billing documents once the "Payer" is blocked.
