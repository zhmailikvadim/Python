Symptom

No option to "Award One Time Payments" in One Time Payments Card in Latest People Profile

Environment

  • SAP SuccessFactors Employee Profile - Latest People Profile
  • SAP SuccessFactors Employee Central

Reproducing the Issue

  1. Access old UI for Employee Profile
  2. Access One-Time Payments section
  3. There is a button at the bottom of the section that is named "Award One-Time Payment"
  4. Access the new UI for Employee Profile
  5. Access One-Time Payments section
  6. The button is no longer available

Cause

Button is no longer supported in the Latest People Profile

Resolution

Button for "Award One-Time Payment" is associated with EC Quick Action Template One-Time Payments. This option is still available via Profile > All Actions section.
