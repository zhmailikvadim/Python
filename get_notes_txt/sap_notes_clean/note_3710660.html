Symptom

You use the Product Compliance functionality.

In the system, the exemption for a substance declaration does not propagate to the parent component.

 

Other Terms

CORRECT_EXEMPTED_DECLARABLES, CL_EHPRC_CHECK_SUBST, Application Code, Exemption ID

Reason and Prerequisites

The symptom occurs due to a program error.

The correction instructions list the prerequisites.

Solution

After implementing this SAP Note, the system propagates the exemption for the substance declaration to the parent component. In the International Material Data System, exemptions are called Application Codes.

Alternatively, import either of the corresponding Support Packages for the component extension of SAP EHS Management or the Feature Packages Stacks for SAP S/4HANA.
