Symptom

Notification tasks cannot be completed in EAMS. Moreover, any changes to a EAM notification are not saved to the database when the status of one of its tasks is set to completed (TSCO). This behavior occurs via EAMS or for example via the "Change Maintenance Notification" Web Dynpro app.
