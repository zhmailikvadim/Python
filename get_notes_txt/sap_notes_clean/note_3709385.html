Symptom

Labels print successfully via Monitor but fail when using the transaction code /SCWM/PACK.

The issue impacts all handling units associated with Manufacturing Orders.

Environment

  • SAP Extended Warehouse Management
  • SAP S/4HANA

Reproducing the Issue

  1. Perform Create HU using the transaction code /SCWM/PACK.
  2. Attempt to print the label.

Cause


Configuration error

Resolution

  1. Navigate to Tcode: SPPFCADM. 
  2. Select the application (/SCWM/WME).
  3. Open the action profile (/SCWM/HU).
  4. Choose the standard action definition (HU_CONTENT_GENERAL_AND_RF).
  5. Update the processing time for the standard PPF action to 'Immediate Processing'.
