Symptom

  • How to increase the retention period for inactive subscriptions
  • How to decrease the retention period for inactive subscriptions
  • How to change the retention time for inactive subscriptions
  • Subscriber ID XXX (SLT) for object XXX (type TABL) will expire in XX days

Environment

  • ABAP CDC Engine
  • SAP Landscape Transformation Replication Server

Cause

The default retention period for inactive subscriptions is 180 days

Resolution

The parameter CDC_SUBS_INACT_MAX_RTNTN_DAYS can be increased/decreased to change the retention period for subscriptions - 3530049 - ABAP Integration – Settings for Retention Period

Keywords

CDC, SLT, Subscriptions, expire
