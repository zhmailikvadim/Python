Symptom

Plant description is displayed instead of Plant code in 'Filter By: Plant' screen in Fiori app 'Stock - Single material (F1076)'

Environment

  • SAP S/4HANA
  • SAP Fiori 

Reproducing the Issue

  1. Go to Fiori app 'Stock - Single material (F1076)'
  2. Enter Material
  3. Click on Filter button
  4. Click on 'Plant'
  5. 'Filter By: Plant' screen pops up
  6. Only description of the plant is shown

Resolution

This is not an error or bug in the Fiori app 'Stock - Single material (F1076)' but it is the standard design.

Keywords
