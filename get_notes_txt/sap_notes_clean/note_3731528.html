Symptom

  • Follow the manual instructions provided in SAP Note 2831943 - Reconciliation on ECC: Compare BSEG with Index for Fields.
  • An error occurs when attempting to enter "Modification Name" values such as FIN_FB_RECON220 and FIN_FB_RECON.

Environment

  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA

Reproducing the Issue

  1. Implement SAP Note 2972792 for Financial Data Quality readiness.
  2. Complete the manual steps in SAP Note 2831943 - Reconciliation on ECC: Compare BSEG with Index for Fields.
  3. Create messages in message class FIN_FB_RECON and add the required long text.
  4. Enter "Modification Name" values such as FIN_FB_RECON220 or FIN_FB_RECON.
  5. Encounter an error when entering the Modification Name.

Cause

Resolution

You can create the modification, save it in a package, and enter the long text.

When naming, it should look like in which package it could be saved could be read from i-button. As a general rule, they should be in the same namespace, for example, a Z-object in a Z-package.
