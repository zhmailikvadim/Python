Symptom

In an SAP S/4HANA 2023 system or newer, you are attempting to use the classic report S_P00_07000134 for Colombian Withholding Tax reporting. You find that this transaction is not the correct solution or does not provide the required output for your legal reporting needs.

Environment

  • SAP S/4HANA
  • SAP S/4HANA Private Cloud Edition

Reproducing the Issue

  1. Start transaction S_P00_07000134.
  2. Select a variant for Colombia withholding tax (for example, an income tax withholding variant).
  3. Attempt to generate the report using Output Control and Lists.

Cause

The transaction S_P00_07000134 is an older, classic report that has been replaced in modern SAP S/4HANA versions. The standard and supported solution for statutory reporting, including Colombian Withholding Tax, is the SAP Document and Reporting Compliance (DRC) framework.

Resolution

To comply with Colombian withholding tax requirements in SAP S/4HANA, you must use the new reports available within the DRC framework instead of the obsolete transaction S_P00_07000134.

Follow the guidance in the SAP Notes below to implement the correct solution:

  1. Main Implementation Note: Start with the overview note which provides a step-by-step guide for implementing DRC for Colombian Withholding Tax.

    • SAP Note 3216115 - DRC Colombia Withholding Tax Reports Implementation Overview

  2. General DRC Information: This note explains the overall strategy of replacing older legal reports with the DRC framework.

    • SAP Note 2480067 - Replacement of Existing Legal Reports with 'SAP Document and Reporting Compliance - Statutory Reports'.

  3. DIAN350 Form Enablement: For generating the specific layout of the DIAN350 interactive form, implement the following notes.

    • SAP Note 3534754 - DRC Enablement DIAN350
    • SAP Note 3536286 - Generic Table for Maintaining Double Tax Agreement Countries

After implementing the relevant notes, you will access the new reporting functionality through the "Run Statutory Reports" Fiori app.
