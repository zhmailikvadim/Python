Symptom

We have observed that Personal Information is visible in the Onboarding module when the user's time zone is set to IST (India Standard Time). However, when the time zone is changed to Pacific Time, the Personal Information section displays the message: “There’s nothing to show yet.”

 

Environment

SAP SuccessFactors Onboarding

Reproducing the Issue

  1. Login as candidate with the URL and username and password and go to Personal data collection step
  2. Under personal information we can find the data the candidate submitted.
  3. Now, change the time zone in your system to Pacific and try to view the data under personal information.
  4. It shows “There’s nothing to show yet”.

Resolution

This issue has now been fixed. If you are still facing the same problem, please create a case with SAP Support and mention the component LOD-SF-OBX for further investigation.
