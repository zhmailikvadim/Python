Symptom

Türkiye E-Ledger report (TR_ELDGR_REP) file needs to be uploaded into a 3rd party service provider as example into SOVOS. After successful report, the created file is raising error in SOVOS system. The error messages can vary depending on the situation.

Environment

  • SAP S/4HANA 1809
  • SAP S/4HANA 1909
  • SAP S/4HANA 2020
  • SAP S/4HANA 2021
  • SAP S/4HANA 2022
  • SAP S/4HANA 2023
  • SAP S/4HANA 2025

Cause

A third-party report submission is required because the local authority does not accept files submitted directly from any Enterprise Resource Planning (ERP) system. This is the reason why SAP solution "stops" with file creation for TR E-Ledger (TR_ELDGR_REP).

Resolution

TR_ELDGR_REP file is intended for the legal authority and is designed to meet that requirement. SAP does not provide provider-specific formats; instead, it delivers a harmonized format that can be used by all service providers. As a result, service providers must adapt to SAP’s format, which is why SOVOS sometimes encounters issues with the report file.

Thus in such situation always get in touch with SOVOS (or any other 3rd party service provider).
