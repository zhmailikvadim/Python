Symptom

The standard service Get_Inspection_Single_Results_History used in a custom production process fails and returns HTTP 500 error when retrieving inspection results history.

Below error message will show:

"Character N is neither a decimal digit number, decimal point, nor "e" notation exponential mark."

Environment

SAP Digital Manufacturing

Reproducing the Issue

  1. Open the Monitor app in the production system.
  2. Insert or locate the relevant process instance.
  3. Check the service error, or call the Get_Inspection_Single_Results_History API with parameters for an inspection characteristic that contains an invalid measured value (for example, 'NaN').

Cause

An invalid measured value "NaN" was recorded for a quantitative characteristic and stored in the database; retrieving it through Get_Inspection_Single_Results_History triggers a number format error.

Resolution

  1. Use the Record_Inspection_Single_Results API to overwrite the invalid 'NaN' value with a valid numeric value, setting overwriteResult=true. Refer to: Record_Inspection_Single_Results API documentation.
  2. Re-run the Get_Inspection_Single_Results_History API to confirm successful retrieval without HTTP 500.
  3. Prevent recurrence by reviewing and adjusting the production process logic to avoid sending non-numeric values (such as 'NaN') for quantitative characteristics when recording single results.
