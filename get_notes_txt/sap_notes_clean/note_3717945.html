Symptom

The standard migration template for maintenance orders excludes service package fields by design. 

Environment

  • SAP R/3
  • SAP R/3 Enterprise
  • SAP ERP Central Component
  • SAP ERP
  • SAP Enhancement package for SAP ERP
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA, on-premise

Reproducing the Issue

Attempt to migrate service package fields.

Cause

Works as designed.

Resolution

  1. The standard migration template for maintenance orders excludes service package fields by design. 
  2. For more information please see: PM - Maintenance order

Keywords

SAP maintenance, migration template, exclude fields, service package exclusion, maintenance order template, SAP enhancement package, field migration, template design, service package migration
