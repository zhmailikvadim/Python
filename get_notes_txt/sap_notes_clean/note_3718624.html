Symptom

You run the Reconciliation - P.I.E.R. Report (PC00_M07_CPIE) transaction with the Deficiency only option selected, and the system does not display employees with only CPP2 deficiencies in the output.

Other Terms

RPCPIEK0, RPCPIEK0_CE

Reason and Prerequisites

Reason

The system does not include the employees with CPP2 deficiencies when building the PIER report output.

Prerequisites

The following Support Package level is the minimum prerequisite:

Release Support Package
S4HCM 102 HR SP 00
S4HCM 101 HR SP 13
S4HCM 100 HR SP 26
EA-HR 6.08 HR SP D7
EA-HR 6.07 HR SP G0
EA-HR 6.06 HR SP H1

For more information, access the SAP ERP Human Capital Management - Support Package Schedules (SAP_HR) or SAP S/4HANA On Premise Human Capital Management (S4HCM).

When you install an SAP Note, some prerequisites might be required. The prerequisites are SAP Notes that have already been released and that change the same context of the current one. Then, even if these prerequisites are not necessarily related to the symptom of this SAP Note, you must implement them due to technical reasons.

Solution

From now on, the system includes the employees with CPP2 deficiencies when building the PIER report output.
