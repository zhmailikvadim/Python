Symptom

The below error message occurs when transporting content between two SAC tenants using Content Network Storage in SAP Analytics Cloud:

 "Couldn’t import object "Model name": Data Motive Service unavailable."

Environment

SAP Analytics Cloud.

Reproducing the Issue

  1. Log in to SAP Analytics Cloud.
  2. Navigate to Transport > Import > Content Network Storage.
  3. Select the package.
  4. Select the Overwrite objects and data option.
  5. Click on Import
    >> Observe the error: "Couldn’t import object "Model name": Data Motive Service unavailable."

Cause

The issue occurs because the username used to perform the content transport contains special characters.

Resolution

Perform the content transport using a user account with a username that does not contain special characters.
