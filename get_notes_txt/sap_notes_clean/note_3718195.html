Symptom

Execution of job CALM HEARTBEAT (program /SDF/CALM_HEARTBEAT) is failing with message errors below

  • ICM_HTTP_SSL_PEER_CERT_UNTRUSTED
  • Job canceled after system exception ERROR_MESSAGE

Environment

  • SAP Netweaver 7.4
  • SAP Netweaver 7.5
  • SAP ERP 6.0
  • SAP S/4HANA

Reproducing the Issue

  1. Open transaction SM37
  2. Re-execute CALM job for program /SDF/CALMHEARTBEAT

Cause

Missing certificates in transaction STRUST

Resolution

Error ICM_HTTP_SSL_PEER_CERT_UNTRUSTED in during the execution of CALM HEARTBEAT job means that a certificate is missing. Check chapter Technical Prerequisites of SAP NetWeaver Application Server for ABAP (7.40 and higher), including SAP BW/4HANA

  • Profile parameter ssl/client_ciphersuites is defined as described in section 7 of SAP Note 510007 .

  • DigiCert Global Root G2 has been imported in STRUST under SSL Client (Anonymous) and SSL Client (Standard).

  • DigiCert TLS RSA4096 Root G5 has been imported in STRUST under SSL Client (Anonymous) and SSL Client (Standard).

Be careful for the correct certificate name that is DigiCertTLSRSA4096RootG5.crt and not DigiCertRSA4096RootG5.crt 

Keywords
