Environment

  • S/4 Hana
  • BTP Cloud Platform 

Reproducing the Issue

  1. Run a connection test in transaction OAC0
  2. Note short dump in transaction ST22

Cause

As part of the connection test, Oauth 2.0 functionality is included in the program logic. Due to a missing correction such a short dump may be provoked

Resolution

  1. There is a missing correction in the system 
  2. Apply SAP note 3545936 - Fix call to null reference in CL_HTTP_CLIENT->SET_OAUTH2C_INFO
