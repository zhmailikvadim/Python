Symptom

You are implementing Brazil CIAP solution but don't use the TDF addon functionality and want to know if the license for TDF is required on this scenario.

Environment

  • Tax Management Framework Brazil
  • SAP ERP
  • SAP S/4HANA

Resolution

There are two options depending on the scenario. A TDF or DRC license is necessary to use CIAP.

  • Scenario 1- SAP ECC System: TDF license is required, as Brazil DRC statutory reporting is not available for ECC.
  • Scenario 2 - SAP S/4HANA system: Either TDF or DRC license can be used to enable CIAP. 

For licensing inquiries, please contact the Account Executive.
