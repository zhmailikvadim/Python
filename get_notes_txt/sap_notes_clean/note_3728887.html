Environment

  • SAP IQ 16.1 SP05 PL15
  • SAP IQ 16.1 SP05 PL16

Cause

The issue has been logged under SAP CR 831877. Its description reads, "IQ Server hanging post PL16 upgrade".

Resolution

  • SAP CR 831877 is scheduled to be addressed in a future SAP IQ Service Pack/Patch Level. Please mark this KBA as a Favorite to be automatically notified when the CR status is changed.

  • Workaround in 16.1 PL15 and PL16 :

    set option PUBLIC.CORE_Options199='OFF' ;

    and restart IQ server.

  • In higher patch levels, the default setting will be 'OFF'.

See Also

KBA 2171560 about how to be notified of new or updated SAP Notes or KBAs in the ONE Support Launchpad.
