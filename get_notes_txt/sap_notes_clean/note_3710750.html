Symptom

While using the RKKBALVI report to transport program layouts, you would like to transport more than one layout per report call in Report Callback program field

Environment

  • ABAP List Viewer 
  • SAP NetWeaver - All BASIS releases.

Resolution

The limitations of the SAP standard functionality, as described in SAP Note 643330, are acknowledged. ALV layouts are specific to their programs, and SAP does not presently furnish a supported mechanism for the bulk migration of layouts across diverse programs or at scale. This represents a known product impediment in both ECC and S/4HANA.

See Also

SAP Note 643330 - ALV layout: Import and export of layouts
