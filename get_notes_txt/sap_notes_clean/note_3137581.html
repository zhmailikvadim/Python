You are trying to upload material prices with app Upload Material Inventory Prices.
If the price unit digit is equal to or above 4 (for example 1000), the system will not be able to upload the value.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Open app Upload Material Inventory Prices
  2. Download empty template and fill in values (make sure that price unit has 4 digits)
  3. Upload the file

Cause

It's a system error that it can't determine the number format correctly.

Resolution

Development team is working on fixing the issue.
For workaround, you can change the Decimal Format in the configuration and upload again:
