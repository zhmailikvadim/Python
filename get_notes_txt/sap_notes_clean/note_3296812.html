Symptom

  • What are the image format supported?
  • What are the pictures file format supported in Lumira?
  • What are the pictograms and shapes file format supported in Lumira Desktop?

Environment

  • SAP Lumira 1.x

Resolution

  • SAP Lumira Desktop support the following file format:
      
    • For Pictures: jpg, jpeg, png, gif
       
    • For Pictograms: svg   ( Limitation: Transparency is not supported. ) 
        
    • For Shapes: svg

Keywords

LUM, Graphic image support, image format support
