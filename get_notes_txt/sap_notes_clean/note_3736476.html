Symptom

The user will face multiple issue in CRMD_ORDER. But the same issue cannot be reproduced in WUI or Fiori.

Environment

  • S4CRM: Business Transaction Framework

Reproducing the Issue

  1. Start transaction CRMD_ORDER.
  2. Some issue will occur like memory dump.

Cause

Transaction CRMD_ORDER is not supported in S/4HANA. S/4 Service and CRM processing must be performed via WebClient UI (WUI) or Fiori, as documented in note 1118231.

Resolution

  1. Refer to note 1118231 to confirm the supported user interfaces for CRM/S/4 Service.
  2. Use the WebClient UI (WUI) or the relevant Fiori applications for all S/4 Service and CRM processes instead of CRMD_ORDER.
  3. Repeat the same search or transaction in WUI or Fiori and validate that the result is returned without a dump.
  4. If the issue can be reproduced in WUI or Fiori, document the exact steps and collect relevant traces and dumps, then contact support.

See Also

Refer to: 1118231 - Supported user interfaces for SAP CRM
