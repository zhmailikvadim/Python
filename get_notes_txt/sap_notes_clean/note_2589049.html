Find ENHANCEMENTS

Other Terms

CHECK program for ENHANCEMENTS

Reason and Prerequisites

You cannot use the Basis tools to search specifically for enhancement implementations in the customer namespace in the given programs; the following program YY_CHECK_ENHANCEMENTS can deliver this.

Solution

For the source code of the report, see the attachment CODING_YY_CHECK_ENHANCEMENTS.

In addition, use SE11 to create the structure YYENH_TAB (see attachments). For GUI status 100 in the program YY_CHECK_ENHANCEMENTS, also see the attachments.
