Symptom

Information about maximum number of LDAP user stores that can be configured in SAP Cloud Connector.

Environment

SAP Cloud Connector

Resolution

  • SAP Cloud Connector supports the configuration of up to three LDAP user stores for authentication.
  • The order of the stores is significant when performing user authentication. During logon, the authentication process evaluates the configured LDAP user stores in sequence to determine where the user credentials are validate.
