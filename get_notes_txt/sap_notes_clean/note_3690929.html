Symptom

  • Unable to update the code list mapping for the DeliveryPriorityCode.
  • Error encountered n web service message monitor "Solve Configuration Issue in Code List Mapping for field DeliveryPriorityCode with value AB" for Service Interface CustomerOrderReplicationIn. 

Environment

SAP Cloud For Customer

Reproducing the Issue

  1. Login C4C tenant.
  2. Go to Business configuration -> Overview -> Code List Mapping for Integration with External Applications and Solutions .
  3. open maintain code list mapping -> Deliveryprioritycode 
  4. Observe you cannot add new code .

Cause

The standard DeliveryPriorityCodes maintained in the system are:

  • 1 - Immediate
  • 2 - Urgent
  • 3 - Normal
  • 7 - Low
  • New Delivery Priority Codes were not created under the "Maintain Delivery Priorities" section, which is required for updating the code list mapping for DeliveryPriorityCode.

Resolution

  1. Navigate to Business Configuration > Overview > Accounts > Maintain Delivery Priorities.
  2. Create new Delivery Priority Codes as needed.
  3. Once the new delivery codes are created, add them under the code list mapping for DeliveryPriorityCode.
