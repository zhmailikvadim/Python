Symptom

Error message: "Reference rule &1 group &2 has no key target field" occurs during transformation activation.

Environment

SAP NetWeaver

Cause

Reference to SAP Note 2078558 - SP35: Reference Rules for non key target fields are created

Resolution

The mentioned rules must be deleted and re-created.

See Also

KBA 2746778 - Reference rule &1 group &2 has no key target field 0RECORDMODE
