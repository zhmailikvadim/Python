Symptom

In the process "Receipt of Meter Value Reversals" (DE_B_MET_MR_REV_MOS), step 0700 displays an error message instead of applying the reverse status to delivery notes that were migrated from a legacy system.

This issue is relevant for the market role: SUP.

Other Terms

SUP, MET, DE_B_MET_MR_REV_MOS, Receipt of Meter Value Reversals, Reverse Delivery Note, /US4G/CL_EA_ACCESS, /US4G/IF_EA_ACCESS~REV_EA_DOC, /US4G/BADI_DELV_NOTE, /US4G/CL_DEF_BADI_DELV_NOTE, IS_EA_DOC_BILLED

Reason and Prerequisites

Program error.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, you can implement method IS_EA_DOC_BILLED in BADI /US4G/BADI_DELV_NOTE and develop your own check logic to handle reversed delivery notes according to your business requirements.

You can verify the fix by implementing the method "IS_EA_DOC_BILLED" in BADI /US4G/BADI_DELV_NOTE and confirming that migrated delivery notes receive the correct reverse status without generating error messages.
