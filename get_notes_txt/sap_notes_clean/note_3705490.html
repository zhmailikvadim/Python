Symptom

You use the SAP Fiori interface of R4CM. When you call the app, the system does not display any data. You find out that the user does not have authorization for the corresponding service.

If you now want to give the user this authorization, the system issues an error message stating that the service does not exist.

For example, the error message "TADIR service /CGS/CALL_DEFAULT_REPORT_SRV 0001 does not exist".

Other Terms

Ticket 1000007349, ADHOF, CGSUI5, ODATA

Reason and Prerequisites

The reason for the error is missing entries in table TADIR. These are created by importing the component CGS. Under rare circumstances, these entries are not created correctly.

Solution

Import the attached transport AO7K901458.SAR. This contains all IWSV entries of the component CGS.

This transport contains the entries with patch 2503/2603.
