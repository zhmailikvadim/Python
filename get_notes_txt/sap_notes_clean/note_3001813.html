Symptom

(WINDOWS)

When you run the SAP IQ installer from the command line or GUI, the following error occurs while trying to load the 'sylapij.dll' library:

The installer failed to load E:\Installer\IQ16.1_sp04.08\server\archives\sylapi\sylapij.dll.
Please verify you have installed Microsoft Visual C++ 2008 Service Pack 1 Redistributable Package
and the SAP IQ Server Suite 16.1 SP04.08 (64-bit) installer image is complete.

Applies to the following versions of the product:

  • SAP IQ 16.1 SP 04, Revision 08 onwards

Solution

To fix this error, install the MS Visual C++ 2015 redistributable package (x64). SAP provides a copy of this package in the server\archives\ms-redist-2015 folder as well.
