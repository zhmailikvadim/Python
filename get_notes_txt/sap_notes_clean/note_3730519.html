Symptom

  • Unable to export Orders or Transactions from the application; the export options window does not appear.
  • Browser shows: "Aw, Snap" and "Something went wrong while displaying this webpage. Error code: Out of Memory".

Environment

  • SAP Sales Performance Management
  • Incentive Management on Hyperscalers

Reproducing the Issue

  1. Open the application and navigate to the Orders or Transactions workspace.
  2. Select any order or transaction record.
  3. Start the export action for the selected record(s).
  4. Observe that the export selection dialog does not open and the browser displays "Aw, Snap" with "Error code: Out of Memory".

Cause

Extremely large data volume (more than 200 million orders). Exporting without filters causes excessive memory usage on the front end, resulting in an out-of-memory condition.

Resolution

  1. Apply filters to narrow the selection before exporting (for example, filter by processing unit or business unit), then export the filtered results.
  2. If needed, export from unaffected areas (such as Organization data or Rule Elements) while applying appropriate filters.
  3. A permanent fix (COM-131901) is planned for the 2604 release. Refer to release notes and upgrade when the release becomes available.
