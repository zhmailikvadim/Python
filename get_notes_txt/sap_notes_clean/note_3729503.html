Symptom

Middle Name and Additional Last Name fields are not available for selection in the filter criteria during data export.

Environment

  • SAP Sales and Service Cloud version 2

Reproducing the Issue

  1. Go to User Menu.
  2. Navigate to Settings.
  3. Select All Settings.
  4. Go to Data Import and Export.
  5. Select Import and Export Data.
  6. Choose Export.
  7. Select the entity as Contact.
  8. Open the filter options.
  9. Observe that the Middle Name field is not available for selection.

Cause

  • The field is not marked as filterable for the Contact entity under Extensibility Administration.

Resolution

  1. Go to User Menu.
  2. Navigate to Settings.
  3. Select All Settings.
  4. Open Extensibility Administration.
  5. Navigate to Contact Person Service.
  6. Select Contact Person.
  7. Go to Field Query Properties.
  8. Uncheck "Show Only Custom Fields".
  9. Locate the Middle Name field.
  10. Set it as filterable.
  11. Click Save.
