Symptom

Data for Investment Projects were not visible within the Project Financial Booklet - Enterprise Projects & Project Financial Booklet – Professional Services app.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

Navigate to Project Financial Booklet - Enterprise Projects & Project Financial Booklet – Professional Services app then display the data.

Cause

Data for Investment Projects are not displayed because the project profile is classified as an Investment Project, which are considered outside the scope of these apps. 

Resolution

It is recommended that the detailed information provided on the SAP Help Portal be reviewed via the following links: 

Project Financial Booklet – Professional Services | SAP Help Portal

Project Financial Booklet - Enterprise Projects | SAP Help Portal

Keywords

Investment Project, Project Financial Booklet - Enterprise Projects, Project Financial Booklet – Professional Services
