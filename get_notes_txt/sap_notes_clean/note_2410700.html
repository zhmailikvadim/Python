Symptom

You execute a step in EWM which updates delivery items of different categories at the same time. The step is aborted with a dump in class /SCWM/CL_DLV_MSG_LOG_DB.

For example in TX. /SCWM/PACK you perform difference on a received cross-docking HU, this step should update both the inbound and outbound delivery items at the same time.

Other Terms

DBSQL_DUPLICATE_KEY_ERROR, CX_SY_OPEN_SQL_DB,  /SCWM/CL_DLV_MSG_LOG_DB=======CP, /SCWM/CL_DLV_MSG_LOG_DB=======CM002, INSERT, /SCWM/CL_DLV_MSG_LOG_DB, DIFD, TDD, ID3, GRC, goods receipt correction

Reason and Prerequisites

Program error.

Solution

Implement the code changes or the corresponding support package.
