Symptom

When configuring connections to SAP Cloud Platform using SAP Cloud Connector the following error message is received:

"Unable to connect: A different SAP Cloud Connector instance is already connected to this account"

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental." 

 

Environment

SAP Cloud Platform: SAP Cloud Connector

Cause

There is an existing SAP Cloud Connector connection to SAP Cloud Platform for this account with the same location ID. Only a single SAP Cloud Connector can be connected to a given account with a specific location ID.

Resolution

This behavior is a limitation inherent to the system design. Each SAP Cloud Connector can be integrated with a singular SAP Cloud Platform account using the same location ID. Consequently, simultaneous connections from multiple Cloud Connectors with identical location IDs are not supported.

In the event that an SAP Cloud Connector connection must be terminated, a new connection can be initiated from a different SAP Cloud Connector client once the initial connection has been disconnected.

For cases requiring remote disconnection, this can be achieved via the Cockpit interface, provided that the Cloud Connector is version 2.8.0.1 or later.

Caution: The remote disconnection option should be considered a last resort. This action is particularly relevant when the existing Cloud Connector’s details have been misplaced, and a new Cloud Connector must be connected, while the active one is preventing access to the account.
