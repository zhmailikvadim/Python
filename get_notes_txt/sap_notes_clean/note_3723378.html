Symptom

You are interested in translating custom tabs to ensure they are accessible and understandable to users across different languages.

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

  • SAP Sales Cloud Version 2
  • SAP Service Cloud Version 2

Reproducing the Issue

1. Navigate to an Entity Screen (e.g. Account, Case, Opportunity etc.).
2. Open an instance. 
3. Click on the User Icon > Start Adaptation.
4. Click on the Pencil next to the Tabs.
5. Select Create Tab.
6. Add a Name.
7. Click on Apply.

Here is an example of a Custom Tab named "IS_CH_Translation" for illustrating the process of the Translation:

                                                                      

                                                                           

Now, you would like to understand how you can maintain translation for this Custom Tab to ensure they are comprehensible to users in different languages. 

Resolution

In order to translate Custom Tabs labels, you need to access the Language Adaptation feature and maintain required texts by following these steps:

1. Click on the User Icon.
2. Go to the System Settings > All Settings.
3. Go to the Language Adaptation.
4. Select an Application (i.e., entity you have created the Custom Section).
5. Select a Source Language.
6. Select a Target Language.
7. Click on Next.
8. Go to the Field Labels tab.
9. Look for the Custom Tab Name into the Source Language Text.
10. Maintain a Text on the "Target Language Text" column:

                                                                  

                                                                                        

11. Save it.

                                                     

These custom tabs names have now been successfully translated. These changes ensure that users can view and understand labels in their preferred language, enhancing the accessibility and usability of the application.
