Symptom

In the following processes, both the package ID and location bundle processes were triggered during the DSO change process:

  • Start of Supply for Electricity (DSO View) (DE_B_GPKE_DSO_SOS_REG)
  • End of Supply for Electricity (DSO View) (DE_B_GPKE_DSO_EOS)
  • Start of Basic or Backup Supply for Electricity (DSO View) (DE_B_GPKE_DSO_SBS)
  • Start of Supply for Feed-In (DSO View) (DE_B_MPES_DSO_SOS_REG)

This issue is relevant for the market role: DSO.

Other Terms

GPKE, package ID, location bundle, 55690, /US4G/BADI_MDC_LBS_TRIG, /US4G/CL_DEF_BADI_MDC_LBS_TRIG, DSO change process

Reason and Prerequisites

The system triggered both processes because the earlier logic was designed to do so.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the system provides BAdI /US4G/BADI_MDC_LBS_TRIG, where you can implement custom logic to trigger the location bundle process (55690).

You need to retrigger the process. You can verify the fix by implementing custom logic in the BAdI and checking that the location bundle process is triggered as expected.
