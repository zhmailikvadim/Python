Symptom

During the creation of JIT Delivery Schedules, the system evaluates all relevant Sales Scheduling Agreements based on standard processing logic. The standard system does not provide an option to exclude specific Sales Scheduling Agreements from this evaluation. As a result, customers who need to omit certain agreements from the JIT Delivery Schedule creation process cannot do so using standard functionality.

Other Terms

Next Generation Just-In-Time processing, JIT Delivery Schedule, NJIT, S2C, Sales Scheduling Agreement

Reason and Prerequisites

Advance development

Solution

Implement the correction instructions provided in this SAP Note or import the relevant Support Package to enable the enhanced processing logic.

After applying the correction, the system provides you an enhancement option to exclude specific Sales Scheduling Agreements during the creation of JIT Delivery Schedules. Via this enhancement implementation you could exclude the Sales Scheduling Agreements that need not be considered for current processing.
