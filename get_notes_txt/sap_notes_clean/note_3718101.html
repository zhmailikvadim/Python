Symptom

Error message: ERROR in row with record key - XXXX , Invalid values for Work Item Assignment thrown while closing a user via upload

Environment

Integration - Uploads

Reproducing the Issue

  1. Attempt to close user  XXXX via the Upload User integration.
  2. Observe the error message: Invalid values for Work Item Assignment.

Cause

  • The upload file does not include the Work Item Assignment column, which is required for closing a user.
  • The system defaults the Work Item Assignment to the user's primary supervisor, who does not have the required permissions to reassign job postings or work orders owned by the user being closed.

Resolution

  1. Ensure the upload file includes the Work Item Assignment column with valid values.
  2. Verify the permissions of the primary supervisor assigned to the user being closed.
  3. If the primary supervisor lacks the necessary permissions, update the user role permissions to allow the role to own work items.
