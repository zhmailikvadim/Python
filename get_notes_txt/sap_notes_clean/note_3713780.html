Symptom

  • Manager can be opened but 503 error is raised while trying to open courses via Learner view.
  • Content creation or updates triggered from Producer fails due to error 503.

Environment

On-Premise SAP Enable Now Manager, HANA Edition.

Cause

  • Network connectivity issue
  • Tomcat is crashed or down.
  • Database entries can not be correctly read.

Resolution

  • Check Network connectivity, especially when VPNs are used or SSL configuration have just been applied, try to disable them and test again.
  • Restart Tomcat and connect again
  • Make sure connectivity from Manager to Database is working, if the password of the database user have expired then following entries can be found in the Manager log, in that case update the password and test again.

Failed to query process sync states
com.sap.db.jdbc.exceptions.JDBCDriverException: SAP DBTech JDBC: [516]: user is forced to change password: alter password required for user <DB USER>

If none of the solutions mentioned above helps with the situation, try to reproduce the error then collect the Manager Debug trace then report a case to KM-SEN-MGR and attach the trace file.
