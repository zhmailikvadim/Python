Symptom

  • Plan Communicator document distributed no longer visible or downloadable from the distribution tracking workspace.
  • Attempts to view or download the document result in a blank page, and the download does not complete.
  • The issue has been identified to occur specifically with participants that have been end-dated.

Environment

  • SAP Sales Performance Management
  • Incentive Management on Legacy Oracle

Reproducing the Issue

  1. Access Incentive Management Admin > Plan Communicator
  2. Attempt to view or download a Plan Communicator document distributed for participant that is already end-dated from the distribution tracking workspace.
  3. Observe that the document is not visible or downloadable, and a blank page is displayed.

Cause

The root cause of this error is that the participant has an end date. It is happening only in the Oracle Legacy environment.

Resolution

The issue has been confirmed, and a code correction (COM-130458) has been developed, which is scheduled to be deployed during release 2604:

  • Lower Environment: Saturday, April 11, 7:00 AM – 1:00 PM UTC
  • Production Environment: Saturday, April 18, 7:00 AM – 1:00 PM UTC

In the meantime, use the following workaround:

  • If documents are of type Plan: Select the distribution row from the distribution tracking page and click on "Export as PDF" button. This will effectively do the same as clicking "Download PDF" for that distribution.
  • If documents are of type Other: Change the effective end date of the participant to "End of time" and then use "Download PDF" as these documents cannot be converted to PDF with the "Export as PDF" button.
