Symptom

It is not possible to select multiple lines to create down payment request in app: Monitor Purchase Order Down Payments

Environment

  • SAP S/4HANA 
  • SAP FIORI

Reproducing the Issue

  1. Open the “Monitor Purchase Order Down Payments” Fiori app.
  2. Attempt to select multiple purchase orders or multiple line items for down payment creation.
  3. Try to create a down payment request for the selected entries.
  4. Observe that only one purchase order/line item can be processed at a time; multi-selection is not available.

Cause

  • The application is designed to process down payment requests one purchase order at a time; multi-selection and mass creation are not supported in SAP S/4HANA Cloud Public Edition.

Resolution

Using the app to create down payment requests individually per purchase order or line item, as multi-selection is not supported.

See Also
