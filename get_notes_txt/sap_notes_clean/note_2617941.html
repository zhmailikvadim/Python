Symptom

The message type 'Error' is not used in the API for changing customs waybills if the customs waybill is locked by another user at the time of the change.

Other Terms

/SAPSLL/API_6800_CFB_CHG, CFB, /SAPSLL/API_6800_CFB_SYNCH

Reason and Prerequisites

Program error

Solution

Implement the correction instructions that are relevant for your release.
