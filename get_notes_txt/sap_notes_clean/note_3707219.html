Environment

SAP Focused Run

Reproducing the Issue

Check sm21 for dumps

Cause

The dump triggers due to an incorrect timestamp stored in the table COF_IQS

Resolution

To resolve this issue, please execute the report 'COF_IQS_RELEASE', which will clear the incorrect timestamp and prevent the runtime error. Additionally, implement the latest collective corrections for CSA Collector Framework for your FRUN system version.
