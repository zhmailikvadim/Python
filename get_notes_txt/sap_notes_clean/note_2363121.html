Symptom

You use the data migration to Simple Finance. The main migration activities have already been performed and you have set the migration status to "finished" - all according to the sequence of steps in the Simple Finance migration IMG branch.

Now you want to perform the so-called post-migration activies. The system sends you the error message FINS_FI_MIG 735 , "No migration activities allowed after migration has been completed". Of course, this does not make sense at all, since you cannot perform the post-migration activies.

Other Terms

Transfer Application Indexes, Fill Due Dates in FI Documents, Fill the Offsetting Account in FI Documents, Enrichment of Balance Carryforward,

Reason and Prerequisites

Program error. The program treats the post-migration activities as main migration activies. The solution is to exclude after-migration activities from the check which raises the error FINS_FI_MIG 735.

Solution

Apply the attached correction instruction.
