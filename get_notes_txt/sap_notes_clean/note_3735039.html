Symptom

The repair of DTP Request writing into an Open Hub Destination does not propagate all data packages after a repair of a failed request.

Other Terms

DTP Open Hub Destination Repair

Reason and Prerequisites

There is a program error.

A repair for Open Hub Destinations of Type: 'File' is not possible. For Open Hub Destinations of Type: 'Database Table', only certain settins allow the repair option.

Open Hub Destination Type: 'File':

  • Repair in general not possible

Open Hub Destination Type: 'Database Table':

Key Write Mode Repair Option
Semantic Key Delete data and insert data sets Repair not possible
Semantic Key Keep data and overwrite data sets Repair possible
Semantic Key Keep data and insert data sets Repair possible
Technical Key Delete data and insert data sets Repair not possible
Technical Key Keep data and insert data sets

Repair possible




Solution

To solve the issue please apply the Support Package as mentioned below. In urgent cases you can use the correction instructions.
