Symptom

You are using the SAP S/4HANA Migration Cockpit or the Migration Object Modeler to migrate data to SAP S/4HANA.

You are using a large number of background jobs for your migration project. However, the jobs are waiting a long time to be used for the simulation process or for the migration process.

This issue can occur if a job locks a table used by the migration object, for example table REPOSRC or IUUC_REPL_CONPAR. The remaining jobs must wait longer than expected before they can process the migration object.

Other Terms

Migration Cockpit, MC, Migration Object Modeler, LTMOM, Migration, S/4HANA  2023, S/4HANA  2025

Reason and Prerequisites

  •  SAP S/4HANA
    • Migrate Your Data – Migration Cockpit
      • Migrate Data Using Staging Tables 

  • This SAP Note is valid for:

    • SAP S/4HANA 2023
    • SAP S/4HANA 2025

Solution

Implement the required corrections in the SAP S/4HANA system in accordance with the correction instructions.
