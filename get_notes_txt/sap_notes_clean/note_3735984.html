Symptom

Is there a way to remove standard and custom fields in the connector file format?

Cause

  • The connector file format is designed to display a complete list of all available native and custom fields by default.
  • It does not support dynamic filtering or exclusion of fields based on specific connector configurations. 

Resolution

Connector file format provides a comprehensive view of all native fields available within the connector. Currently, there is no option to filter or remove standard or custom fields based on connector configurations.
