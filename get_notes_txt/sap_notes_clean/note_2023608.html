Symptom

In the standard system as currently delivered, you have activated Online W-2 and have maintained your customizing settings in order to print different PDF files for paper and online copies. When you execute the Tax Reporter (PU19) transaction for employees who have chosen to receive their forms online, the system is also producing a paper copy. Consider the following example:

  • You have configured the W-2 Employee copy (HR_F_W2_MULT_11) to be used for paper forms.
  • You have configured the W-2 Self-sealer form (HR_F_W2_MULT_SS) to be used for the online forms.
  • You run Tax Reporter for employees 1 and 2. Employee 1 has chosen to receive a paper copy and 2 has chosen to receive an online form.
  • Currently, Tax Reporter produces the W-2 Employee copy for both employees. Employee 2 should only receive the Self-sealer form. 

Other Terms

RPCTRPU0, T5UX7, UTRW2, HCM_LOC_CI_29.

Reason and Prerequisites

Program error.

Solution

This Note allows different forms to be produced for paper and online copies. The system will validate all forms which have been maintained either with option "PDF Spool" or "PDF File" in the Tax Form Definitions (V_T51T8) view cluster and which have been configured as employee copies (value "EMPE") in the Alternate forms details (V_T5UX7) table view.

As general rule, SAP recommends that you install a solution by applying a Support Package. However, if you need to install a solution earlier, use Note Assistant (SNOTE) to implement the correction instructions.

You can find more information about the Note Assistant in SAP Service Marketplace, under service.sap.com/note-assistant.
