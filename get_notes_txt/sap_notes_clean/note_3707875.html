Symptom

  • Outbound file processing generates a blank file named "This PC" instead of the desired output.
  • Files are generated as "tar.gz" with a "This PC" file inside.
  • Issue occurs only when processing a single register in the outbound file.
  • Files are being generated with zero bytes.

Environment

  • SAP Sales Performance Management
  • Incentive Management on Legacy HANA 
  • Incentive Management on Hyperscalers

Reproducing the Issue

  1. Run a process that generates an outbound file.
  2. Ensure the output file contains only a single row.
  3. Observe that the file is generated as "tar.gz" with a blank file named "This PC" inside.

Cause

Windows Explorer has difficulties handling "tar.gz" files, particularly when the file contains only a single register.

Resolution

  • For resolution, both options below can be used to open the files:
    • Use another application to decompress these files.
    • Change the compression mode for these files using the Express Data Loader "Edit File Type" option.
