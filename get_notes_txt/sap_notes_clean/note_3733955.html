Symptom

The Note provides functionality for the BOM Subitems Compatibility mode.


You customize your scenario with SAP_SVasClassicBOM mapping by ID = X. You manually create BOM subitems for your components. You send SV components with a reference designator. After you implement this SAP Note, subitems map with the sent reference designator by installation point:

  • In case SAP subitems installation point = DM referenceDesignator value - the system creates a KM for the subitem and updates the data according to the DM values.
  • In case SAP subitems installation point <> DM referenceDesignator value - the system creates a KM for the subitem and deletes it from SAP because it is not provided in the DM.

Reason and Prerequisites

Note 3732436 is installed.

Solution

Implement this SAP Note to enable mapping of subitems with the reference designator by installation point.
