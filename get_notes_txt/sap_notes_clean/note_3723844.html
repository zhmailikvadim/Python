Symptom

There is a need to understand where in the communication flow Proxy server definition on SCC (SCC -> "Configuration" -> "Cloud" -> "HTTPS Proxy") is used.


       

Note: Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

SAP Cloud Connector(SCC) release independent;
Third-party proxy server.

Resolution

The proxy definition from SCC -> "Configuration" -> "Cloud" -> "HTTPS Proxy" is limited and takes effect only for connections between SCC and BTP hosts, it does not interfere in connections between SCC and backend systems.
