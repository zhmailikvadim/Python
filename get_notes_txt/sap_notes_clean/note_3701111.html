Symptom

The newly created IC table is not visible in the IC tab after being copied from the template library/favorites under the following scenario:

The user opens the IC tab for Activity 1 in the detail workspace, then navigates to the structure workspace and pastes the IC table into Activity 2 from the template library in the structure workspace without opening the IC tab for Activity 2. As a result, the newly added IC table is not displayed in the IC tab of Activity 2.

Other Terms

PEO, IC, Inspection Characteristics, IC Table, MSFR4, Favourites, Copy, Paste, Drag, Drop

Reason and Prerequisites

Program Error.

Solution

Implement this SAP Note to resolve the above-mentioned issue.
