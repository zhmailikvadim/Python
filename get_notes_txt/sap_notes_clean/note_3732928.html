Symptom

  • The background job SAP_WORKFLOW_RESTART no longer completes and is canceled.
  • The job log shows the message: "Job canceled after system exception ERROR_MESSAGE".

Environment

  • SAP Business Workflow
  • SAP Web Application Server for SAP S/4 HANA
  • ABAP PLATFORM - Application Server ABAP

Cause

Authorization OBJECT 'S_WF_ADM'; 'WF_GROUP' = 'ADMIN'; 'ACTVT' = '16' is checked. If the check fails, workflow system jobs cannot be scheduled. 

Resolution

  • Grant the user who executes scheduler job SAP_WORKFLOW_RESTART with the missing authorization (OBJECT 'S_WF_ADM'; 'WF_GROUP' = 'ADMIN'; 'ACTVT' = '16' ).
  • The standard workflow role SAP_BC_BMT_WFM_SERV_USER_PLV01 also contains the expected authorization values. This role can also be assigned to the user that is used to run workflow system jobs.
