Symptom

Message "No input help is available" is displayed when using search help (F4) in BP tcode.

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Environment

  • SAP S/4HANA
  • SAP S/4HANA Private Cloud Edition

Reproducing the Issue

  1. Go to the BP tcode to create a business partner.
  2. Choose role FLCU01 (Sales), go to the Sales and Distribution tab, and maintain a sales area.
  3. In the Orders tab, click the search help (F4) button for field Item Proposal.
  4. Observe that the message "No input help is available" is displayed.

Cause

In SAP S/4HANA, VBUK has become obsolete (refer to SAP Note 2349135), therefore no standard search help is available for the Item Proposal (KNVV-VSORT) field in the business partner master.

Resolution

If the search help is needed, the user may consider implementing a custom solution for the F4 search help.
