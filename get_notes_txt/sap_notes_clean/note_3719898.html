Symptom

This KBA explains how to display Transaction No. and Transaction Date fields in the PCB II Form for Malaysia.

Environment

  • Human Capital Management (HCM)
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP SuccessFactors Employee Central Payroll
  • SAP S/4 HANA

Resolution

  1. Verify if the required data for "Transaction No" and "Transaction Date" exists in table T5LTC.
  2. If the data is missing, update the table using report SAPM59L0. Ensure the month is entered as two digits (e.g., 02 for February).

Keywords

PCB II form, transaction no, transaction date, table T5LTC, SAPM59L0, T5LTC-RCTDT, T5LTC-RCTNO
