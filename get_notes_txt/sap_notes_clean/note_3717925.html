Symptom

You are using the Migration Workbench (MWB) as part of an NZDT project. The "Runtime write" information is not saved for cluster tables.

Other Terms

 DMIS, MWB

Reason and Prerequisites

MWB / DMIS functionality is used on one of the following releases:

  • DMIS 2018 SP14
  • DMIS 2020 SP10
  • DMIS 2025 SP00
  • SAP S/4HANA 2023 FPS00 - SPS05
  • SAP S/4HANA 2025 FPS00 - FPS01

Solution

Implement the attached correction using transaction SNOTE.
