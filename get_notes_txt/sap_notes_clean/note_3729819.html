Symptom

Insight Cards Show "Card configuration error" or "500 error" in My Home page.

Environment

  • SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Navigate to the My Home page.
  2. Open the Insights Cards.

Cause

Resolution

For the configuration error/500 error cards, try the following steps to resolve the issue.

    1. Delete the card from My Insights: Home page->User menu->My Home Settings->Insights Cards->Go to the detail of the given card and press Delete button. 
    2. Perform 'Empty cache and hard reload' in the browser. In Chrome it is done by triggering dev tools through F12, right click on the refresh button and left click on empty cache and hard reload.
    3. Go to the overview app containing the deleted card. If the given card works correctly add it again to My Insights.
