Symptom

'Devline does not exist' error is thrown while importing objects into ESR

Other Terms

Devline does not exist, ESR, import, DB error

Reason and Prerequisites

Most of the time, this error happens after a system copy or upgrade of PI system. Here, the error is due to partial execution of migration after the system copy that results in database inconsistency.

The root cause of the issue was a code bug.

Solution

Since the code base has been updated with the latest fix, please download the patch and apply to the system.

If the issue persists even after application of the patches, please execute the SAP Note:1244593.
