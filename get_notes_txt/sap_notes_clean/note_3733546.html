Symptom

A web channel is set to 'show only on category pages'. On accessing the category page on the website, the campaign is not placed.

Environment

  • Product: Emarsys

Reproducing the Issue

  1. Create an embedded web channel campaign with placement option 'show on category pages only' and select a zone on a category page
  2. Activate the campaign
  3. Navigate to a matching category page and observe that the campaign is not placed

Cause

The websites category page is implemented in a way that it is not recognized as a category page - the reason might be a missing “category” web extend event. 

The “category” web extend event qualifies the page to be considered as a category page:

The same is valid for web channel campaigns set to 'show only on category pages', in that case the "view" web extend event qualifies the page as product page:
