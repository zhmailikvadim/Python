Symptom

Execution of the program '/PE1/R_AR_ITEM_DATA_DELETE' results in a short dump.

Other Terms

/PE1/R_AR_ITEM_DATA_DELETE, TCode /PE1/PI_AR_DELETE, Table /PE1/DB_ITEM_ARC, Program termination, SAPSQL_PARSE_ERROR.

Reason and Prerequisites

Program error. Sequence of the fields in the select query had a mismatch with the sequence of fields in the underlying database table.

Solution

Implement the correction instruction attached to this SAP Note. Alternatively, you can import the referenced Support Package.
