Symptom

In a Multidimensional Data Grid based application, you manually enter a valid value in a hierarchy node input field or select a node from the hierarchical value help.

The value is reported as invalid on executing the query.

Other Terms

Web Dynpro Data Grid, FPM BICS Grid, Hierarchy Search Help

Reason and Prerequisites

Coding Error

Solution

Implement the attached correction instructions and the prerequisites or apply the corresponding support package.
