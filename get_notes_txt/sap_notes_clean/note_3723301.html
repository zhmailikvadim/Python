Symptom

  • Encountering the "Disconnected from FTP server" error while executing certain processes, such as file sweep.
  • Processing logs display the following error: [EXCEPTION] java.lang.IllegalStateException: java.lang.IllegalStateException: java.lang.RuntimeException: RuntimeException occurred while executing the batch for PSID 'InsertInFile'. Message: java.sql.BatchUpdateException: ORA-00001: unique constraint (DATABASE.INDEXName) violated  

Environment

  • SAP Sales Performance Management
  • Agent Lifecycle Management

Reproducing the Issue

  1. Open file sweep search on the environment UI.
  2. Run the file sweep for the respective entity.
  3. Observe the following error: "Disconnected from FTP server".

Cause

The Entity Index referenced in the processing logs, "ORA-00001: unique constraint (DATABASE.INDEXName) violated", is causing the issue. It exists in the "InFile" table, and the custom code is conflicting with the creation of a new InFile.  

Resolution

  1. Identify the team responsible for the creation of the customized Entity Index. 
  2. Review the code associated with the Entity Index.
  3. Perform the necessary changes to resolve the conflict with the creation of a new InFile.
