Symptom

  • A new email trigger is required when a new hire completes the compliance task after uploading the P45 attachment and submits the Form before the signature step.

Environment

SAP SuccessFactors Onboarding 

Reproducing the Issue

  1. A new hire uploads the P45 attachment.
  2. The new hire completes the compliance task.
  3. The new hire submits the Form before the signature step.

Cause

The system does not currently support the requested functionality for email triggers in the described scenario.

Resolution

  1. Submit an Enhancement Request to have this feature considered for future releases.
  2. Refer to SAP Note KB0135759 for additional information regarding the current system functionality.

See Also

Refer to: SAP Note KB0135759 - Individual solution without a specific SAP Note
