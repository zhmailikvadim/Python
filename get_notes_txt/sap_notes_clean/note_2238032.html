Symptom

If you specified an item type that uses reference fields (for example, item type CSC uses the "Reference Document" and "Reference Item" fields) in the item overview, it is not possible to navigate to the initial screen due to the failed check of required fields.

This SAP Note also provides an enhancement spot for the creation of a separate confirmation prompt.
