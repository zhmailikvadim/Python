Symptom

In the 'Plan Raw Material Deliveries' app, when a process order is cancelled, its reference in the raw material planning day is not cleared, therefore the respective day cannot be reused in the event of a new process order.

Other Terms

Plan Raw Material Deliveries, cancel process order, raw material planning day

Reason and Prerequisites

Program error

Solution

History:

Version 1 of 02.03.2026: New creation

Install the code correction.

Description of Correction:

The field in the raw material planning day entry where the reference to the process order is stored is now cleared as soon as the process order is cancelled. As a consequence, the link between the two is broken, and the raw material planning day could be reused in the eventuality of a new process order being created.

Functional Dependencies:

No dependencies.

Manual Changes:

No manual changes.
