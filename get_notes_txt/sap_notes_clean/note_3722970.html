Symptom

- An error occurs during supplier integration, displaying the message: Delivery Service type is Missing in SRT_MONI
- The error is suspected to be caused by a blank value in the PO Box service type field under the Address question in the Registration Questionnaire.

Cause

The PO Box service type field in the Address question of the Registration Questionnaire was left blank, causing the error Delivery Service type is Missing

Resolution

Kindly ask supplier to update the value in PO Box service type field under the Address question in the Registration Questionnaire and re-submit the response.

Keywords

Delivery Service type is Missing, PO Box Service Type, Address, Postal Address, Extended address, SRT_MONI
