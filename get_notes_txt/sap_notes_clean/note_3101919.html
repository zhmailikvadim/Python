Symptom

You are using the datasource 0CRM_CIC_CTI, but don't want all the data to come to BW system. You can then add the following paramater CRM_IC_CTI_EXT_NEW = X. With this parameter, the extractor will only fetch the data not yet send to BW.

Other Terms

0CRM_CIC_CTI, Memory issue, Timeout issue

Reason and Prerequisites

Usage of 0CRM_CIC_CTI, changing BW system

Solution

Please apply the attached correction instructions

Please execute the manual instructions and add the user parameter "CRM_IC_CTI_EXT_NEW= X" to all the users that work with the extractor.
