Symptom

GST Report -ANX Report cannot be opened with Limited Financial License in HANA

System Error Message: You cannot open window [23400444] GST Report -ANX  with your current license

Reproducing the issue

Prerequisites:

  • India (IN) Localization Company
  • SAP Business One version 10.0 for SAP HANA

Scenario:

  1. Log on in SAP Business One with a user with Limited Financials License
  2. Go to Menu → Financials → Financial Reports → GST Report -ANX

Expected behavior:

GST Report -ANX window to select criteria opens for user with Limited Financials License

Actual behavior:

  • GST Report -ANX Report cannot be opened with Limited Financial License in HANA
  • System Error Message: You cannot open window [23400444] GST Report -ANX  with your current license

Solution

SAP intends to provide a feature package or patch to solve the symptom that is described in this SAP Note. The section References below lists the related feature package or patch once they become available. Be aware that References can only be confirmed at feature package or patch release date. SAP delivers feature packages or patches only for selected releases at its own discretion, based on the business impact and the complexity of implementation.
