Symptom

You want to implement the most current correction notes addressing issues with Documents - Handling Unit node.

With the Note Analyzer function in SNOTE, you can import the most relevant notes as an XML file into the SNOTE transaction and implement all notes relevant for your Release and Support Package, which have not yet been implemented. In order to use the Note analyzer, you need to implement 3200109 and pre-requisites. To implement the missing notes for stock inconsistencies, please follow the steps given in the solution chapter.

Other Terms

HUMO; Handling Unit history; documents; monitor; node; PDI; PDO; reference; WT; status

Solution

tep 1 : Implement SAP Note 3200109.

Step 2 : Launch transaction SNOTE . In the menu bar select Goto -> other tools -> Launch Note Analyzer

Step 3 : Upload the file XML file attached to this note. You can process the Notes using the Note Analyzer now. For more details on usage of Note Analyzer refer to SAP Note 3200109.

Step 4 : Test your processes and if successful move the implemented notes to your productive system

Please note that the XML file attached to this note might be updated, if new important notes are created. So in order to stay on a most current level of corrections, you might check back to see if a new version of the note is available.
