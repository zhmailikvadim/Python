Symptom

  • Error message "Acknowledgment for PO item contains date variance" (Message No: ME795) is issued when attempting to create an inbound delivery in the background;
  • ME795 is issued as warning when creating the inbound delivery manually using transaction VL31N.

Environment

  • Sales And Distribution (SD)
  • Logistics Execution (LE)
  • SAP S/4HANA
  • SAP S/4HANA Cloud 
  • SAP ERP Central Component
  • SAP ERP
  • SAP Enhancement package for SAP ERP
  • SAP Enhancement package for SAP ERP, version for SAP HANA

Reproducing the Issue

Attempt to create an inbound delivery in the background.

Cause

  • Note 3167238 is missing in the system.
  • LIKP-VERUR and LIKP-LIFEX are not empty during the inbound delivery creation process. For purchase order to inbound delivery scenarios, these fields are expected to be empty.

Resolution

  1. Apply note 3167238 in the system;
  2. Make sure that fields LIKP-VERUR and LIKP-LIFEX are empty during the inbound delivery creation process;
