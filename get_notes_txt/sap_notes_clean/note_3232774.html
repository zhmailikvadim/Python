Symptom

An error such as the following occurs:

Failed to instantiate [java.lang.Class]: Factory method 'currentLsnClass' threw exception; nested exception is java.lang.IllegalArgumentException: Invalid version format: 12.11 (Ubuntu 12.11-0ubuntu0.20.04.1). Should be major.minor.patch.

Other Terms

Replication, SDI, target, log file

Environment:

  • SAP HANA SDI 2.0 SP06 Patch 2x

Reason and Prerequisites

The adapter expected a version string such as 13.4 but reveived 12.11 (Ubuntu 12.11-0ubuntu0.20.04.1)

The version string format can involve descriptions other than a version number, basing only on version string parsing was not correct.

Solution

This correction is delivered in the release(s) listed in the Support Packages and Patches section of this SAP Note.
