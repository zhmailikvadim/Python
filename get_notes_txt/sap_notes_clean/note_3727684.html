Symptom

Valid during period 1.4.2026 – 30.9.2027 an Employer's Contribution is temporary lowered for Employees in an age of 18-22 years at start of income year. For year 2026 this means Employees born year 2003-2007.

For such employees the Employer's Contribution is calculated with a rate of 20.81% for gross income that does not exceed 25.000 SEK per calendar month. For gross amounts above this limit, standard Employer’s contribution of 31.42% should be paid.

Other Terms

EC, Employer's Contribution, ECONY, HR_SE_CALC_ECONT

Solution

The new rate of 20.81% used in EC calculation for younger Employees is valid from 1.4.2026 and is stored under the constant ECONY in table T511K (please follow the document with manual steps attached to this SAP note to apply these changes).

The calculation of age necessary for Employer's Contribution reflects the fact that Employee's age needs to be defined at the start of each income year for legal purposes.

Other Terms

Max Parallel, ODH, Package Size

Reason and Prerequisites

ODH process was designed to run in a batch work process. So, to enable parallel processing via RFC in dialog, ODH tool provides a new configuration parameter 'Max Parallel'.

Solution

  1. Import the relevant Support Package or implement the correction instructions.
  2. In ODH transaction:
    1. Mark the ODH class.
    2. Hit the button 'Details(Analysis)'.
    3. Hit the button 'Show Parameters'. Here you choose 'Max Parallel'.
    4. Select a rule that you want to change its 'Max parallel'.
  3. If you set the 'Max Parallel' to more than 1 (default), the parallelization processing will run in dialog work process. In this case, you need to modify the 'Package size' parameter to avoid timeout.

Note: The default value of Max Parallel is 1 which means the processing run in batch.
