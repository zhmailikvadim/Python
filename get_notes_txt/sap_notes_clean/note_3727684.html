Symptom

You run the Programma di calcolo della retribuzione ITALIA (RPCALCI0) report for part-time employees. The system does not apply the part-time percentage limit maintained in the Grado di occupazione (BSGRD) field of the Retribuzioni di base (0008) infotype when it calculates the cap on monthly integration for events.

Other Terms

Transaction: PC00_M19_CALC

&ITASS_PLAF (for more information, see SAP Note 2327917)

Plafonamento, IT008

Reason and Prerequisites

Reason

The system considers only the amount maintained in the Tabella periodi convenzionali (V_T5ITIG) table when it calculates the cap limit in the Elaborazione delle assenze (ITASS) payroll function with parameter PLAF.

Prerequisite

The following Support Package (SP) level is the minimum prerequisite:

ReleaseSupport Package
S4HCM 102 HR SP 00
S4HCM 101 HR SP 06
S4HCM 100 HR SP 19
SAP_HR 6.08 HR SP D0
SAP_HR 6.04 (EhP6 and EhP7) HR SP K2

Solution

From now on, the Adjust plafonamento limit for absences (ADJUST_PLAFONAMENTO_LIMIT) method of the BADI: valutazione delle assenze con XNAB/ITASS (HRPAYIT_XNAB) BAdI allows you to adjust how the system calculates the cap limit for employees.
