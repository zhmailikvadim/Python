Symptom

This SAP Note is a technical delivery of 2 new fields: Time of Document Generation (DOTIM) and Date of Document Generation (DODAT) in the Documents and Certificates (0290) infotype.

Another SAP Note requires this SAP Note to be implemented as a prerequisite.

Other Terms

PS0290, PS02904, HRPAY99RU_DOC_GEN_TIME, HRPAY99RU_DOC_GEN_DATE

Reason and Prerequisites

Install this SAP Note only if another SAP Note requires this one as a prerequisite.

Solution

As a general rule, SAP recommends that you implement the Support Package specified for your release.

However, if you need to install the solution earlier, use the Note Assistant to carry out automatic correction instruction. You can find more information about the SAP Note Assistant in SAP Support Portal, under SAP Note Assistant.
