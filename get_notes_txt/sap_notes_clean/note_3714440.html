Symptom

In the app "Manage Long-Term Supplier's Declarations - Inbound", there are no options to see if a supplier is third-country.

This feature was available in GTS 11.

Environment

SAP Global Trade Services, edition for SAP HANA

Reproducing the Issue

  1. Access the Fiori app Manage Long-Term Supplier's Declarations - Inbound (App ID: F5689)
  2. Attempt to locate an indicator for if a supplier is third-country

Cause

In the inbound process if the supplier is a third-country supplier it is not considered at all for the exchange of LTSDs.

Resolution

Currently there is no equivalent solution for this functionality in GTS edition for SAP HANA.

As a workaround:
An entry can be created in the Preference Properties, where a flag will be displayed indicating if the given supplier is a third-country supplier or not.
