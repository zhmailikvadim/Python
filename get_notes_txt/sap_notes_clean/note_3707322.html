Symptom

After upgrading to SAP S/4HANA 2025 FPS00, you attempt to send multiple service objects with serial numbers assigned back to the customer. To do so, you create one outbound delivery the Manage In-House Services or Manage In-House Service Objects app.

Although the outbound delivery is successfully created, the delivery document displays only one of the serial numbers instead of all serial numbers assigned to the service objects.

Reason and Prerequisites

This issue applies if the following has been specified:

  • Service objects that do not have the special stock indicator Customer Stock (B) assigned. 
  • If SBOUTDEL (Send Back to Customer: Create Outbound Delivery) has not been specified in the Define Profiles for Logistics Integration Customizing activity in Customizing for Service under Transactions -> Settings for In-House Service.

This issue has been caused by a program error.

Solution

Implement this SAP Note.
