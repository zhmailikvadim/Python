Symptom

In New Mexico Royalty, an error occurs when attempting to create an auto suspense exemption in the SAP ERP system using transaction code O3UGR_ASUE. The issue arises when the entity is no longer tied to a Master record, resulting in the inability to bring both old and new entity records into ASUE. The error message displayed is: "Entities not found or missing Master Data." 

Environment

SAP ERP 

Reproducing the Issue

  1. Access the SB0 system.
  2. Use Transaction code: O3UGR_ASUE.
  3. Click Create.
  4. Select State Entity and click Enter.
  5. Enter with the description.
  6. Enter with the State Entity selection criteria.
  7. Click Find Entities button.
  8. Observe the message: "Entities not found or missing Master Data."

Resolution

Functionality is working correctly as per design. Entity needs to be assigned to master data first and then only it can be used in the ASUE. In case you have requirement of changing the entity after reporting, then master data can be changed with the new entity, and it will generate the PPN (Prior Period Notification) and will report to new entity. 

Keywords

ASUE (auto suspense exemption); NM ROY; NM Master Data.
