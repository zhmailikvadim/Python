Symptom

In the output of the CO_WHT_INCOME_CERT report, there are missing documents.

Environment

  • SAP S/4HANA

Reproducing the Issue

  1. Open Fiori launchpad
  2. Open "Run Statutory Reports" app
  3. Execute the CO_WHT_INCOME_DCL report
  4. Check the report output
  5. Observe missing documents that should be reported

Cause

 The report CO_WHT_INCOME_DCL is designed to consider only documents where the associated Business Partner has the tax type configured as "CO1".

Resolution

 Ensure that the tax type of the BP associated with the missing document is set to "CO1".
