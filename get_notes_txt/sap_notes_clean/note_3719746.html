Symptom

The payee name (name of the receiver/recipient) is too short.

Other Terms

RPCDCP00, HRPAD_PAYEE_NAME, HRPAY00_COLDME_ALV_CONTENT, HRPAY00_COLDME_ALV_INFO, RPMREP00, INDX(HR)

Reason and Prerequisites

- You implemented SAP note 3719443.

- You use report RPCDCP00 for collective transfers.

Solution

  • Implement the attached correction instructions or import the HR Support Package relevant for your release.
  • In case you are using report RPMREP00 to do distributed reporting, then all your systems need to have the same SP level.
  • After implementing the note and before running the report RPMREP00 you may go and remove the entries in database table INDX for RELID = 'HR' and PGMID = 'SCHEDULR'.

As solution we now use the new data element HRPAD_PAYEE_NAME for component ETEXT in structures HRPAY00_COLDME_ALV_CONTENT and HRPAY00_COLDME_ALV_INFO. (Old data element was EMTXT.)

Remark:

The cluster table INDEX (for RELID = HR) is only used temporarily during the runtime. The export is done in form DATA_EXPORT (include RPCDCI00) and the back-conversion (of the imported data) is done in form REDEFINE (include RPCDCI00). Thus, both sides use the same structure, so that the above structural change will work.
