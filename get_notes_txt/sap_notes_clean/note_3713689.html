Symptom

How to display the decimal values of a certain length in a story.

Resolution

Open CSM and set "Scale" to at least 3 to show the third integer value of the decimal.

This determines how many of the decimal values are passed into the table.

Example:

"VALUE": {

"@Common.Label": "Deposit Value",
"type": "cds.Decimal",
"scale": 2,
"@Analytics.Measure": true,
"@Measures.ISOCurrency": {
"=": "UNITSYMBOLFORVALUE"
},
"@Aggregation.default": {
"#": "SUM"
}

In Embedded Analytics, open the Styling Menu and look for Decimal Places.

Enter the desired decimal places to show on the story.
