Symptom

When maintaining an inspection plan in T-code QP02, the operation short text does not translate according to the user’s logon language.

Environment

  • SAP ERP Quality Management (QM)
  • SAP ERP
  • SAP R/3
  • SAP R/3 Enterprise
  • SAP ERP Central Component
  • SAP Enhancement package for SAP ERP
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA, on-premise
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. log in SAP GUI in EN, and maintain an inspection plan with English operation short text in T-code QP02.
  2. log in SAP GUI with other languages such as ZH, the operation short text does not translate according to the user’s logon language.

Cause

This is standard SAP system behavior.

Inspection plan operation short text is not multilingual.
Its description field (PLPO‑LTXA1) is not language‑dependent, meaning:

  • The system stores only one version of the text
  • The text always appears in the language in which the plan was created
  • QP01/QP02 do not offer language‑dependent maintenance for operation descriptions

Resolution

It is standard behavior that inspection plan operation short text cannot be translated into multiple languages. 
 
A possible workaround: MICs (QS21/QS23) support multiple languages

Keywords
