Symptom

The Note 3401762 status becomes to 'completely implemented' automatically after downloading it, no transport request included it.

Environment

Note Assistant

Reproducing the Issue

Cause

The corrections have already been implemented into the system. 

Resolution

  1. The note 3401762 is included in Transport-based Correction Instruction (TCI) package SAPK10801JCPS4CORE, which is corresponding to sap note 3521388.
  2. Import the TR which includes the note 3521388 into the next systems in the landscape, corrections from sap note 3401762 are included and will be imported automatically with the TCI of note 3521388.
