Symptom

  • A Custom Analytical Query generated a CDS query view with below annotations on a field:

    @AnalyticsDetails.query.displayHierarchy: #OFF
    @AnalyticsDetails.query.hierarchyBinding : [{type :#CONSTANT,value :''}]

  • The field is defined with header time dependent hierarchies.
  • When run the CDS query, a hierarchy on this field is processed unexpectedly in query runtime.
  • The performance of the query is degraded due to hierarchy processing.

Other Terms

time dependent hierarchy directory, hierarchy binding, empty constant, initial

Reason and Prerequisites

The CDS query view is generated incorrectly. After this note, the two line annotation combination is ignored in Analytic Engine.

Solution

The issue is fixed in ABAP Platform Cloud 2602 05 and 2602 04 with Hotfix 1000037394, number 90004 (2026).
