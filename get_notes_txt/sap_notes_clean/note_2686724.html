Symptom

You are using Manage WorkCenter Capacity for SAP. You notice that when you are changing the language to any other language there is an error state.

Other Terms

Manage WorkCenter Capacity

WorkCenter Capacity

Reason and Prerequisites

Program Error

Solution

As a general rule, SAP recommends that you install a solution by applying a Support Package. However, if you need to install a solution earlier, use the Note Assistant to implement the correction instruction.
