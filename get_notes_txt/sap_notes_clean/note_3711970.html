Symptom

  • Error AY135 – It is not possible to deactivate in fiscal year &1

Environment

  • Contract and Lease Management
  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA Finance
  • SAP S/4HANA

Reproducing the Issue

  1. Proceed to close the internal order using KO02 (order closure).
  2. System initially displays the message: "It is not possible to deactivate in fiscal year &1"

Cause

  • Fiscal Year change did not took place in FIAA as it did on FIGL side comparing tables T093C and FAGL_CARRY_FORW.

Resolution

  1. Execute FAGLGVTR to align tables T093C and FAGL_CARRY_FORW.
  2. Once tables aligned proceed to KO02 and close the mentioned order.

Keywords

AA558, AY135, internal order settlement, asset fiscal year, internal order closure.
