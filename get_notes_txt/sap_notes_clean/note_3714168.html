Symptom

  • List of Values showing both 'Text' and 'Key' for a parameter in the prompt window in Crystal Reports designer.
  • Only the description of the values should appear for the LOVs.

Environment

  • SAP Crystal Reports 2020
  • SAP Crystal Reports 2025
  • SAP BW

Reproducing the Issue

  1. Open a report in Crystal Reports designer.
  2. Refresh the report.
  3. Prompt window pops-up.
  4. Select to see the List of Values for the Parameter.
  5. Observe Text and Key are both displayed.

Cause

'Prompt With Description Only' option was set to 'False'.

Resolution

  1. Open Crystal Reports designer.
  2. Right click on the parameter and Edit.
  3. In the Edit Parameter window set 'Prompt With Description Only' option to 'True'.
  4. Click on 'OK'.
  5. Refresh the report.
  6. Observe only the Description is seen for the Parameter.
