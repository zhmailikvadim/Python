Symptom

The access conditions of CDS view "C_EHSBizUsrWthPrsnlWrkAgrmtVH" are inherited from entity "I_BusinessUserBasic" and activity "03" is replaced by activity "F4." This causes an error. The issue comes from authorization object "B_BUP_PCPT," in whose list of activities, activity "F4" is missing.

Other Terms

Access control I_BusinessUserBasic, authorization object B_BUP_PCPT

Reason and Prerequisites

Reason: Program error.

Prerequisites: The validity of the correction is maintained in the correction instructions.

Solution

The Support Packages that contain this correction are listed in the "Support Packages" section.
Alternatively, you can implement the correction instructions of this SAP Note.
