Symptom

When customer (external) demand is uploaded with the external demand capture service (from table /SAPAPO/EXTDMS), demand for those customer locations where the product does not exist are uploaded. This should not be allowed.

Other Terms

eSPP, /SAPAPO/EXTDMS, dealer product, dealer demand upload, customer demand

Reason and Prerequisites

Program error

Solution

Please implement this Note with the correction. It will implement the check for product location existence on the dealer, and an error log message in case demand does not comply. Also a new error code for not processed dealer demand is implemented.
