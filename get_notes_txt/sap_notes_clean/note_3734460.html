Symptom

What are the requirements for a Learning item to appear on the 'Advance My Proficiency' section in Growth Portfolio?

Environment

SAP SuccessFactors Learning

Resolution

Learning courses will be recommended in the Growth Portfolio if the following requirements are met:

  1. The item is correctly associated with the attribute in Learning.
  2. The item is available in a library that the user has access to.
  3. The item is not assigned to the user.
  4. The user has never completed this item.
