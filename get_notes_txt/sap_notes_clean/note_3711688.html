Symptom

Spotlight/Employee Details option missing in People Search when the Latest Shell Bar is enabled. 

Environment

SAP SuccessFactors HCM

Reproducing the Issue

  1. Search for a user in People/Action search.
  2. Notice that the Spotlight/Employee Details icon is available allowing a preview of the employee profile. 
  3. Go to Company System & Logo Settings. 
  4. Enable "Enable the latest SAP shell bar". 
  5. Now when searching for a user in People/Action search, the Spotlight/Employee details icon is missing. 

Cause

This is a Known Error.

Resolution

This fix for this issue is planned to be deployed on 1H2026 release. 

For release timelines, please review SAP SuccessFactors Product Release & Road Map Information.
