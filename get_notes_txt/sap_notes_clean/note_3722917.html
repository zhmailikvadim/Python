Symptom

 Error in message /SAPAPO/CURTO_API number 058.

Other Terms

/SAPAPO/CURTO_API, 058, PDS, BAPI

Reason and Prerequisites

In the message "No continuous consumption has been maintained for PDS &1, op. &1, mode &3", &1 appears twice. The second one should be &2.

Solution

Please implement this SAP Note.
