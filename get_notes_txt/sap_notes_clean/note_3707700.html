Symptom

You have implemented one of the BAdI methods:

  • REMI_GET_DATA (Get Remi DMFA data to be stored in cluster table BEEVL)
  • DDDT_GET_DATA (Get DDDT DMFA data to be stored in cluster table BEEVL)
  • WAPM_GET_DATA (Get WAPM Dmfa data to be stored in cluster table BEEVL)

of BAdI BADI_HRPAYBE_BEEVL of Enhancement Spot HRPAYBE_PAYROLL_SPOT to create one of the sections

  • REMI (REMES Information (90438))
  • DDDT (Deduction Detail Occp. (90250))
  • WAPM (Indemnity WAPM (90011))

in the DmfA Declaration (RPCDMFB0).

The error messages raised in the above mentioned BAdI methods are not displayed in the log of the Payroll Driver Belgium (RPCALCB0).

Other Terms

RPCALCB0, BEEVL, RPCMASB9_BEEVL, CREATE_BEEVL_FIELDS, CL_HRPAYBE_BEEVL_BL, PY_CREATE_BEEVL_FIELDS

Reason and Prerequisites

You are using the Payroll Localization Belgium.

Solution

Import the Support Package specified for your release or implement the manual correction instruction.

As a general rule SAP recommends that you implement the Support Package specified for your release.

However, if you need to install a solution earlier, use the SAP Note Assistant to implement the correction instruction.

You can find more information about the SAP Note Assistant in SAP Support Portal, under SAP Note Assistant.
