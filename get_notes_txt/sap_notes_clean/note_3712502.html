Symptom

New XML format of monthly tax statement for year 2026 is presented by authority.

Other Terms

Report HSKCDPHP, XML output.

Reason and Prerequisites

This SAP Note is relevant only for Slovakia (HRSK).

Solution

This SAP Note delivers new XML format of monthly tax statement for year 2026. Newly, tag "01b" is added to the XML file.
Updated PDF form of monthly tax statement for year 2026 was already delivered in SAP Note 3696927 (but is relevant for Public sector only). Customers that want to have latest PDF form of monthly tax statement (even if changes are not relevant for them) can implement the SAP Note 3696927.

 

Implementation

  • As a general rule, SAP recommends that you install a solution by applying a Support Package. However, if you need to install a solution earlier, use the Note Assistant to implement the correction instruction. You can find more information about the Note Assistant in SAP Service Marketplace, under service.sap.com/note-assistant.
