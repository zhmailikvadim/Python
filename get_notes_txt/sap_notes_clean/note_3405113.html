Symptom

The name of the current logged on user is shown in the MIGO titlebar.

Other Terms

USNAM, SAPLMIGO, MIGO_GR, MIGO_GI, MIGO_GS, MIGO_GT, MIGO_GO

Reason and Prerequisites

Reason: It was defined in the design of transaction MIGO to show the name of the user in the titlebar.

Solution

The display of the name from the current logged on user in the MIGO titlebar is not a required function to use the transaction. The name of the user is therefore removed from the titlebar. If you want to know the current logged on user, you can display the current users ID in the GUI status bar accordingly.

Implement the attached corrections or import the relevant Support Package.
