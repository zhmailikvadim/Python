Symptom

Is it possible to set a default font type in the Comments field on the 360 Multi-rater Review?

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

SAP SuccessFactors 360 Multi-rater

Reproducing the Issue

On the 360 Multi-rater Review, scroll to a Comments field.

The default font type is displayed as Andale Mono:

Resolution

No, there is no way to set the default font type.

The font type list is in alphabetical order and the "Andale Mono" font type is the first listed.
