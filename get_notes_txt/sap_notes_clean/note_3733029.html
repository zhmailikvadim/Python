Symptom

Learning search results show only item code in Opportunity Marketplace

Environment

SAP SuccessFactors Opportunity Marketplace

SAP SuccessFactors Learning

Reproducing the Issue

  1. Set the system language to English UK.
  2. Go to Opportunity Marketplace.
  3. Search with a keyword in the common search bar.
  4. Open the Learning sub tab/filter.
  5. Observe that all courses in the search results display the item code instead of the item title.
  6. Switch the system language to another language, repeat the steps, and observe that course titles are shown correctly.

Cause

In Opportunity Marketplace, a learning item title is shown only if the item is available in the user’s locale; when search matches exist only in other locales (for example, item available in English US while the user’s locale is English UK), the UI shows the item ID instead of the title.

Resolution

As a temporary workaround, switch the user’s display language to a locale in which the item is available to see the title.

Product Engineering is investigating a solution.
