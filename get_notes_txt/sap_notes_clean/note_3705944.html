Symptom

  • The user tries to open SAP Build Work Zone, advanced edition from the standard edition using a Local URL application assigned to a tile.

  • When the tile is launched, the following authentication error appears: "Identity Provider could not process the authentication request received. Delete your browser cache and stored cookies, and restart your browser. If issues persist, contact your administrator."

Environment

  • SAP Build Work Zone, Standard Edition
  • SAP Build Work Zone, Advanced Edition

Reproducing the Issue

  1. Create a tile to launch SAP Build Work Zone Advanced Edition from SAP Build Work Zone Standard Edition 
  2. Attempt to launch the tile and encounter error. 

Resolution

Embedding SAP Build Work Zone, Advanced Edition into the Standard Edition is not supported, as they are separate services with distinct architectures, features, and pricing models.

Keywords

 SAP Build Work Zone, Standard Edition, Advanced Edition, URL tile, Intra network , IAS
