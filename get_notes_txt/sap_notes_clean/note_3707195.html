Symptom

When launching the Excel Add‑in, users are prompted to install an update. However, the update process cannot be completed and it prevents the Add‑in from being used.

Environment

SAC Excel Add-in

Reproducing the Issue

Try to update the SAC Excel Add-in once using it. 

Resolution

Based on the symptoms described, this issue does not seem to be related to SAC tenant configuration or user authorization but rather related to Microsoft Office.

You can first check that the Microsoft Store is not locked internally.
 
Then, if the issue persists after manually clearing the cache in Excel, please reach out to Microsoft Support for further assistance by clicking on the Help tab and selecting 'Contact Support'.
