Symptom

After completing the BI2025 upgrade from BI 4.3, following issues are seen

  • Install succeeded and CMC is available.
  • After updating the 2025 license, and enabling all servers and restarting the APS services.
  • 2 of the APS processes APS.core and APS.Analysis cannot be started / either running with errors / or stopped. 
  • Restarting these services or removing select services does not help nor show failure

Environment

  • Windows Server 2022 Standard
  • SAP BusinessObjects BI Platform 2025 Patch 7 

Reproducing the Issue

  1. Log in to CMC
  2. Select Servers 

Cause

Upon checking the installdata/logs folder - a log uninstalldu-platform.services.cms.cpp.cms_shared-4.0-core-64.14.4.0.5542.log - which suggested to reboot the system to complete the process.

Resolution

Rebooting the system finished the install process and both APS processes (APS.core and APS.Analysis)  started up with everything working as expected.
