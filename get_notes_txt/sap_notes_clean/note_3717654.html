Symptom

More information about how to mass update several or all Replication Flows with a single command is required.

Environment

SAP Datasphere

Resolution

A mass update for Replication Flows in SAP Datasphere is currently not supported. Each replication flow must be pause or resumed individually.

Several ideas requesting this enhancement already exist in SAP Datasphere's Influence channel: 341485, 337471, 352858 and 366394, for example.
