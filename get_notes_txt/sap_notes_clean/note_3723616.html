Symptom

Component: Incident Management
Module: Report Incident

When you run the Report Incident (F1992) app, the system displays all the incidents that you created. If you have no authorization to read the incident data, the system displays an "index out of range" error message.

Other Terms

CL_EHHSS_INC_REPINC_BO_ACCESS, READ_INCIDENTS, READ_BASIC_INFO_DATA, CL_EHS_INC_REPORTINCID_DPC_EXT

Reason and Prerequisites

Reason:
Program error.

Prerequisites:
For information about the validity of the corrections, see the correction instructions.

Solution

After implementing this SAP Note, the Report Incident app runs without the error message.

The "Support Packages" section lists the Support Packages that contain these corrections.

Alternatively, you can implement the attached correction instructions.
