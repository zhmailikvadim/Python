Symptom

In /AIF/CUST->Error Handling->Define Interface-Specific Features->Define Key Fields for Multi. Search, if a keyfield is source structure and Relevant for Recipient Determination, then the recipient cannot be successfully identified if the source structure data is changed by 'Init Function Before Mapping' or 'Init Class Before Mapping'. Within this note, the source value changed by the Init function or class are correctly identified for recipient determination.

Other Terms

Key field, Recipient Determination, Init Function Before Mapping, Init Class Before Mapping

Reason and Prerequisites

 This is a code defect.

Solution

Implement attached correction instruction or corresponding support package.
