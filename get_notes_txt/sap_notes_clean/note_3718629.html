Symptom

You use Transaction Manager of Treasury and Risk management. You manage cross currency interest rate swaps and have manually edited an interest or a derived flow(net payment). You want to undo the manual edit but the error FTR_GUI 035 "Change flow(s) function cannot be undone" is raised.

Other Terms

 620, SAEND

Reason and Prerequisites

This problem is caused by a program error.

Solution

Implement the attached correction instructions or import the relevant Support Package.
