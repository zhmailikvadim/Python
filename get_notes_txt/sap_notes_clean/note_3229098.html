Symptom

The Controls that is maintained as "Event Relevant" is still picked up during the Planning Assessment via Planner. Ideally, it should only select "Time Frame" relevant.

Environment

SAP Process Control(all versions)

Reproducing the Issue

  1. NWBC
  2. Assessment 
  3. planner

Cause

Application behavior.

Resolution

  1. Determine whether the control is automated or manual and identify the date or event initiation type.
  2. Plan the self assessment of such controls accordingly.
  3. Ensure control owners receive the work item for the self assessment process.
  4. Execute all subsequent steps, similar to other manual or automated date initiation controls.
  5. Note that no automated schedules will be initiated, and no business rules will be performed in this scenario.

Keywords

event, time, trigger, determination, control, plan, planner, schedule, timeframe, time-frame.
