Symptom

The following behaviour occurs in SAP Analytics Cloud (SAC):

  1. Configure SMTP server by following guide: Configure Custom SMTP Server
  2. Email about the successful configuration is received
  3. Create a script to generate emails, receive notification that it is created
  4. Notice that no email is received.

Environment

SAP Analytics Cloud

Resolution

  • Go to User settings.
  • Navigate to ''Notifications'' tab.
  • Enable '' System Notifications'' checkbox.
