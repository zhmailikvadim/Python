Symptom

  • Go to the Modeler UI app in SAP Data Intelligence;
  • Reports of the error:
    • The modeler screen in SAP Data Intelligence is getting cut off and not rendering properly.
    • When loading the modeler app for the first time after login, the graph editor or RMS editor does not open or load on the screen.
    • Clicking on the maximize icon allows the graphs to show up, but the screen remains cut off.
    • When opening the configuration of any operators in the graph, the configuration displays, but the screen gets cut off again upon closing the configuration.
    • Unable to switch to JSON view of the graph.

Environment

SAP Data Intelligence Cloud

Reproducing the Issue

  1. Log in to the production tenant of SAP Data Intelligence.
  2. Open the modeler application.
  3. Attempt to load any graph or RMS editor.
  4. Observe that the graph editor or RMS editor does not open or load on the screen.
  5. Click on the maximize icon and observe that the graphs show up, but the screen remains cut off.
  6. Open the configuration of any operators in the graph and observe that the configuration displays.
  7. Close the configuration and observe that the screen gets cut off again.
  8. Attempt to switch to the JSON view of the graph and observe that it is not accessible.

Cause

The application state file for the user in the tenant has likely become corrupted or stuck in an invalid layout configuration.
This prevents the system from rendering the Graph/RMS editors correctly and causes the screen to cut off.

Resolution

  1. Log in as the technical user or an administrator with access to the user's workspace.
  2. Open the System Management application.
  3. Navigate to the Files tab.
  4. Select "My Workspace".
  5. Navigate through the folder hierarchy: files -> vflow -> general.
  6. Locate the folder named ".user".
  7. Delete the ".user" folder.
  8. Refresh the browser or log out and log back in.
  9. Verify that the modeler screen renders with the default factory layout, resolving the rendering and JSON view issues.
