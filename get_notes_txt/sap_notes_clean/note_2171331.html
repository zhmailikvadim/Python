Symptom

If the main page uses plug-ins (ActiveX, Java, Adobe) a PopupWindow may disapear behind the plug-in content when the user opens a LayeredControl (PopupMenu, DatePicker, etc.)

Other Terms

Web Dynpro Java, Web Dynpro ABAP, WDJ, WDA, SAP GUI for HTML, Unified Rendering, Lightspeed,

Solution

To fix this issue, please see the description for your basis framework:

Important Remark: In case of using an Enterprise Portal with a custom theme it might be necessary to update the eppserv.sca (7.0x) or ep-basis.sca (>= 7.1x)  (see note 1446099).

A) Web Dynpro ABAP:

See related note section for the corresponding ABAP note. Notes start in general with "Corrections for unified rendering ... <release>"

If there is no ABAP note available it means the note is not release yet. The release date is in most cases end of the month. In this case, please re-check this note at a later point of time or contact SAP.

B) Web Dynpro JAVA:

The issue described above will be fixed by a rendering related Web Dynpro for Java patch. In section "SP Patch Level" you can find information, which patches contain the respective correction. Please install this patch or a newer one (Patches are always cumulative).

If the section "SP Patch Level" is not available or the information about which patch contains the correction is missing, this means that this has not been determined, yet. In this case, please re-check this note at a later point of time.

Downloading Web Dynpro for Java Patches All Web Dynpro for Java patches are available on SAP Service Marketplace. Note 330793 explains how to download patches from SAP Service Marketplace.

Release Specific:

NW04S/ NW 7.00:

From Web Dynpro perspective, we recommend you to apply both SAPJTECHF.sca and SAPJTECHS.sca. So if any one of the archive is not mentioned in the 'SP Patch Level' tab, it is recommended to apply the same, which is available in the service market place.

In addition, to avoid breaking dependencies between the rest of the Java core components, it is also advisable to patch the components mentioned in SAP KBA 1893013

Deploy the sca's directly using JSPM. Please refer the link to the official documentation for JSPM: http://help.sap.com/saphelp_nw70/helpdata/EN/1f/c45b4211aac353e10000000a1550b0/frameset.htm

NW 7.10 & 7.11:

The patch is included in the FRAMEWORK.SCA.

NW 7.20 and above:

The patch is included in the AJAX-RUNTIME.SCA. We recommend to install the newest available patch level of the FRAMEWORK.sca and WDRUNTIME.sca.

Deploy the sca’s directly using SUM. You can find its documentation in the SAP Service Marketplace at: http://service.sap.com/sltoolset -> Software Logistics Toolset (latest version) -> Software Update Manager (SUM).

In addition, to avoid breaking dependencies between the rest of the Java core components, it is also advisable to patch the components mentioned in SAP KBA 1794179.

C) SAP GUI for HTML

The patch will be shipped with a new kernel version. See related note section for the corresponding SAP GUI for HTML note. If there is no SAP GUI for HTML (Kernel) note available it means the note is not release yet. In this case, please re-check this note at a later point of time or contact SAP.
