Symptom

There are various differences between the old and new FIMA. This SAP Note provides an overview of the effects of these differences in the loan module.

Interest Calculation Methods and Dates

The dates are determined for the new FIMA on the basis of the calendar days. The interest calculation method does not influence the determination of the days. If you use the 360/360 method, the calculation date (DVALUT) can still be the 31st of a month. This is then handled during the correct daily calculation so that the working days (ATAGE) for the old and the new cash flow are the same.

See SAP Note 1674670.

Inclusive View

In most cases, the old FiMa uses the inclusive view. If you enter the calculation date exclusively (inclusive indicator is not set), FIMA converts it to the inclusive view, that is, usually one day before the inclusive view. This is a technical necessity of the old FiMa. The new FiMa does not take this restriction into account. This means that if you have set the calculation date exclusively, the flow has the calculation date according to the condition as the exclusive date.

See SAP Note 1674670.

Parallel Interest Conditions

For the old FIMA, parallel interest conditions for the calculation of the interest were first summarized and later split again. In the new FIMA, each interest condition is calculated independently. As a result, the interest conditions can now also have different interest calculation methods or interest calculation types. This is not possible for the old FIMA.

Interest per disbursement (accrued interest)

If the 'Interest per Disbursement' indicator is set for the product type, the indicator SSTCKKZ = 'X' is set for all disbursement flows. For the old FIMA, this indicator is not set for the interest of the first disbursement or the first interest flow of the period. This indicator is set for the interest flows that belong to the later disbursements/disbursements in the period.

For the new FIMA, the disbursement flows also receive the indicator SSTCKKZ = 'X'. In addition, the "Flow Record Group" field (SBEWEGRP) is filled. All interest flows that belong to a disbursement also receive the indicator SSTCKKZ and the flow record group of the corresponding disbursement is in the Flow Record Group field. As a result, the interest can be uniquely assigned to the respective disbursement.

Accrued Interest for Capital Outflows

For the old FIMA, the interest calculation was converted from the accrued interest method to the balance method for capital outflows within the period (for example, a payoff). In this case, accrued interest is generated for the capital outflows for the new FIMA. See also SAP Note 3291464.

Capitalized Charges

For the old FIMA, capitalized charges had to be defined with the calculation category 'VZ' and the flow category 'TB'  (see SAP Note 1572786). For the new FIMA, the charges are capitalized with flows with the calculation category and the flow category TB and the "Immediate Settlement" indicator (see SAP Note 3057235). The calculation category 'VZ' is no longer supported in the new FIMA.

Risk-Free Rates (RFR) 

Only supported in the new FIMA. This function was implemented with SAP Notes 2880124 (for CML), 2932789 (for TRM), and 2939657 (common basis for CML and TRM).

Totals Conditions

Only supported in the new FIMA. This function was implemented with SAP Note 3383600.

Loans according to the actual principle

The actual principle 2 is not supported in the new FIMA.
The actual principle 1 was not initially provided in the new FIMA. SAP Note 3073852 enabled this.

Imputed annuity

This was not initially provided in the new FIMA. Was implemented with SAP Notes 3514189 and 3521358.

General

With SAP Note 2907389, the FIMA connection can be preassigned in Customizing for the product type and can also be displayed in the contract.

Solution

This SAP Note describes the basic differences between the old and new FIMA in the loan. Before you use the new FIMA in the loan, you should be aware of these differences and check/compare the results of the cash flow calculation.
