Symptom

When trying to run the Class connector in LMS, it is throwing the below error:

Scheduled Offering item details doesn't match with provided item details.

Environment

SAP SuccessFactors Learning

Reproducing the Issue

  • Upload the connector file to the SFTP>
  • Access Learning Administration>
  • Go to System Administration> Connectors> Class Connector>
  • Schedule the connector to run>
  • After the connector runs, click View Results> View Details>
  • The exception is displayed.

Cause

The value included in the feed file for the class connector for field COL_NUM99_VAL, belongs to some other item and not the item mentioned in the feed file.

Resolution

This is because the  COL_NUM99_VAL of the affected item entity is duplicated with another Class's Legacy ID/COL_NUM99_VAL.

In order to fix the issue, please validate the Custom field Value 99 for the respective item in the feed file. To do this, follow the below steps:

  • Navigate to Learning Activities>Classes>
  • Click on Add/Remove Criteria and select Custom field 99
  • Delete the "Start Date After" field
  • Do a blank search with the COL_NUM99_VAL value from the feed file. In other words, if your class connector feed file has "21" as the value for "COL_NUM99_VAL", search for 21 in this screen.
  • You will get the corresponding class/item details.
  • Correct the feed file with these item details. 
  • Rerun the connector
