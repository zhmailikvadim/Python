Symptom

This SAP Note provides you with changes to the Mexico DIOT Report, Format with 54 Fields (report category MX_VAT_DCL) in the Run Statutory Reports app for SAP Document and Reporting Compliance. When you run the MX_VAT_DCL report, the system does not include documents posted in a fiscal year different from the reporting date.

Other Terms

DRC, Mexico, DIOT, MX_VAT_DCL

Reason and Prerequisites

This SAP Note is relevant only for Mexico.

To implement Mexico VAT Compliance Reporting, you need to install the Document and Reporting Compliance (DRC) framework. For more information, see Document and Reporting Compliance (formerly known as ACR).

Solution

After implementing this SAP Note, the system considers the posting date from the tax transfer document and its relevant documents, specifically the invoice and payment.

For more information about the development delivered with this SAP Note, see
the What's New topic on SAP Help Portal: Definition of Business and Transaction Types for Suppliers in DIOT Report, Format with 54 Fields.
