Symptom

We can see the data in Check Data Availability (Usage Logging), but Available data for selected systems cannot be downloaded in tabular form or readable form.

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

SAP Solution Manager 7.2

Reproducing the Issue

  1. Run transaction SM_WORKCENTER
  2. Custom Code Management
  3. Check Data Availability (Usage Logging)

Cause

It is not an issue/error, but normal behavior.

Resolution

It is not an issue/error, but normal behavior. The data can only be displayed on the web page and cannot be downloaded or exported.

Please refer to Custom Code Management User guide on page 75 to know about the function of Check Data Availability – Usage Logging.
