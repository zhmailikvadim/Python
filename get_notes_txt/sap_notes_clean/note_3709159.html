Symptom

  • Data Service job fails when using SAP application datastore.
  • Error message:
    R3C-150605: |Data flow XXX
                                                           The SAP job was canceled for host <XXX>, job name <XXX>, job count <XXX>, job log
                                                           from SAP <20260108, 100846, Job started
                                                           20260108, 100846, Step 001 started (program XXX, variant &0000000000025, user ID XXX)
                                                           20260108, 100846, File \path\XXX.dat cannot be opened
                                                           20260108, 100846, Job canceled after system exception ERROR_MESSAGE
                                                           >.

Environment

  • SAP Data Services
  • SAP application

Cause

The parameter "Working directory on SAP server" in datastore was incorrectly configured.

Resolution

Inside SAP application datastore, parameter 'Working directory on SAP server' should be setup with a directory on the SAP application server where the software writes intermediate files. Not a directory on DS server.
