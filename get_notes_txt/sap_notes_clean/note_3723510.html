Symptom

When you execute the report SI: Get Data from SI Agencies  (RP_PAYDE_B2A_SV_COLLECTOR), tasks are generated in the Notification Tool (NT) for return codes of the authority. These do not always have a link to the report SI: Display Inbound Processes  (RP_PAYDE_B2A_SV_LIST).

Examples include:

  • I005: Data is available, but not all communication partners could be reached

  • I006: No data exists, whereby not all communication partners could be reached

Solution

Implement the attached correction instructions or import the specified support package. After you import the HR Support Package or implement the correction instructions, you do not have to perform any further follow-on activities.

After you implement the corrections, there is always a link to RP_PAYDE_B2A_SV_LIST.
