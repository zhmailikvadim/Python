Symptom

  • Submit button for Data Locking Task is not working.
  • Nothing happens after selecting Submit for the data locking task.
  • Console error appears: "Uncaught (in promise) TypeError: Cannot read properties of undefined (reading 'concat')".

Environment

SAP Analytics Cloud 2026.2.10

Reproducing the Issue

  1. Open a Calendar Data Locking task assigned to you.
  2. Select Submit.
  3. Observe that nothing happens and a console error is logged.
  4. Verify that submitting other task types (for example, general task) works as expected.

Cause

Resolution

The solution of this issue is currently under investigation. Please follow this KBA for updates.
