Symptom

In the SAP Ariba Buying and Invoicing that is integrated with SAP ERP or SAP S/4HANA using SAP Integration Suite, managed gateway for spend management and SAP Business Network, ISU for rejected invoices sends the DOC_DATE as the rejection date due to the CHDHR entry, causing ISU failures in CIG. This results in the invoice status not being updated correctly.

Cause

The error occurred due to erroneous code.

Resolution

Now, the code has been enhanced to resolve the problem. For more information, please see SAP NOTE 3706595.
