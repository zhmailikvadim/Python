Symptom

When adding an Activity to an existing Network the message CN053  Network does not have activity account assignment is raised.

Environment

  • SAP ERP Project Systems (PS)
  • SAP ERP Central Component 6.0
  • SAP ERP 6.0
  • SAP enhancement package for SAP ERP
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA, on-premise
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Run CJ20N Or CN22.
  2. Open Project or Network.
  3. Add Activity to Network.
  4. Save or Calculate Costs.
    ( Edit => Cost => Plan/Actual => Activity/Element) 

Cause

Incomplete configurations 

Resolution

In OPUV: Review network type configuration and activity account assignment settings.
( Ensure you check then network type parameters for the combination of Network type and Plant you are using  )

Also/Or in OPUU  

Check that you have maintained the proper Control key for the network. Make any necessary changes. 


Keywords
