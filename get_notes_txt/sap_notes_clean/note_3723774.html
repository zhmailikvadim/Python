Symptom

  • Running /SDF/RC_START_CHECK after implementing the latest version of SAP Note 2502552 - S4TC - SAP S/4HANA Conversion & Upgrade new Simplification Item Checks results in the error: "CLS4SIC_TM_ASR_NON_SUBSTOP_HDL not found; check the implementation status of SAP Note 3674107".
  • SAP Note 3674107 - Simplification Item Check for Adv. SR relevant Stops w/o SubStops is marked as implemented in SNOTE.
  • Object CINC CLS4SIC_TM_ASR_NON_SUBSTOP_HDLCCAU is implemented/active.
  • Object CLAS CLS4SIC_TM_ASR_NON_SUBSTOP_HDL is implemented/active.

Environment

  • SAP S/4HANA 2020
  • SAP S/4HANA 2021
  • SAP S/4HANA 2022
  • SAP S/4HANA 2023
  • SAP S/4HANA 2025

Reproducing the Issue

  1. Execute report /SDF/RC_START_CHECK.
  2. Observe the error message: "CLS4SIC_TM_ASR_NON_SUBSTOP_HDL not found; check the implementation status of SAP Note 3674107".

Cause

Program error.

Resolution

  1. Implement SAP Note 3693999 - Correction for SI07: Check for Adv. SR relevant Stops w/o SubStops.
  2. Verify if the issue persists after implementing the note.
