Symptom

  • Searching for a user by name in SAP Analytics Cloud while editing team members only displays the user ID in the search results. 

Environment

  • SAP Analytics Cloud 2025.21.9

Reproducing the Issue

  1. Login to SAC
  2. Open Security => Teams. (in new User Interface)
  3. Select a team.
  4. Click on members of that team.
  5. Enter the full username in the search box.
  6. Observe multiple results are displayed, showing only User IDs. 

Cause

  • By design behavior

Resolution

  1. The current design of SAC only displays the user ID in the "Users" page, and there is no option available to show the display name in the new user interface at this time. 
  2. Kindly vote on the influence request at Improvement Request Details - Customer Influence
