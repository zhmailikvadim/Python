Symptom

The note provides soolution for following issues:

  1. In some cases, the KSeF ID was not read from the FI document text. This issue affected the V7M(3), KR_PD, and ST_KR structures.
  2. Incorrectly defining the fixed asset type when no fixed asset class was directly assigned in the configuration.
  3. Determination of the the fixed asset retirment/sale document.
  4. Added a warning if the KSeF ID is missing when the fixed asset was retired after February 1, 2026, and then sold.
