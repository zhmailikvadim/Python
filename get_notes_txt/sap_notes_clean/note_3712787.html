Symptom

While creating prolongation order from the MAPIs, Last Installment EOT Flag was missing from Inpayment Agreement Structure which was impacting the Loan's End Date.

Other Terms

BCA_MAP_OR_PROL_CREATE_ACTIVAT, BCA_STR_OR_MAP_PROL_CREATE, BCA_STR_OR_MAP_IPA_DATA ,LAST_INST_EOT

Reason and Prerequisites

Program Error

Solution

Implement this correction via SNOTE or upgrade to relevant support package.
