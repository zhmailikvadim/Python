Symptom

 You are using real-time integration (RTI) between the SAP ERP system (either SAP ECC or SAP S/4HANA) and SAP Integrated Business Planning (IBP). When you run the report RCIFMTDE, you notice that on the IBP side, the sent location products or MRP area materials are processed in a single queue. This processing does not take into account the filter size setting from the CFI3 transaction.

Other Terms

SAP IBP Real-Time Integration, RTI, RCIFMTDE

Solution

 To ensure that the queues process the number of units according to the settings specified in CFI3, implement the relevant SAP Note or apply the corresponding support package. This will align the queue processing with the desired configuration.
