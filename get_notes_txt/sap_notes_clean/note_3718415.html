Symptom

This SAP Note introduces the updated version of the VAT Return (Version 5.12), in compliance with Order No ЕD-7-3/1227@ from February 02, 2026, and effective from Q1 2026. The updates apply to Sections 8 and 9 (Purchase Ledger, Sales Ledger).

In this update, the following new elements have been introduced:
New hierarchies: RU8_5, RU815, RU9_5, RU915

Other Terms

Sales Ledger, Purchase Ledger, J_3RF_PURCHASE_LEDGER_X, J_3RF_SALES_LEDGER_X, FIARRU_SALES_LEDGER, FIAPRU_PURCHASE_LEDG, XML, 5.12, VAT Return

NO_NDS.8_1_003_01_05_12_01, NO_NDS.81_1_003_02_05_12_01, NO_NDS.9_1_003_03_05_12_01, NO_NDS.91_1_003_04_05_12_01

RU8_5, RU815, RU9_5, RU915

Reason and Prerequisites

Legal change.

New legal requirements according to Government Regulation with Order No ЕD-7-3/1227@ from February 02, 2026, valid from Q1 2026.

This SAP Note provides you with the relevant system changes to address the legal change: Changes: VAT Return and electronic formats 2026

Solution

Apply the appropriate Support Package or install the attached correction instructions with manual post-implementation and pre-implementation steps.

After installing this SAP Note do the following:

SM30, view J_3RF_VATXMLCST, Change. Add records for company code as follows:
