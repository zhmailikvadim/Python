Symptom

When deploying the project or any modified object, it fails with below error.

“Error: Cannot find module ‘core-util-is’.”

Environment

  • Business Application Studio
  • SAP HANA Cloud

Reproducing the Issue

Deploy the project or objects.

Cause

You are running hdi-deploy version 4.8.1 or 4.9.0, which has known issues.

Resolution

The latest version 4.9.1 of the hdi-deployer has been published to NPM to fix the issue with core-util-is.
You may consider deleting the package-lock.json file and the node_modules folder in db module, then build again for the new version to be downloaded.

Deployer is available in artifactory and npm: https://www.npmjs.com/package/@sap/hdi-deploy?activeTab=versions
