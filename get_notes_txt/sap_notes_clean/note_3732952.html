Symptom

When click on button save in "Define Balance Validation Tasks" error message FINCS_BV_RGASGMT003 occurs.

Environment

SAP S/4HANA Finance for Group Reporting

Reproducing the Issue

  1. Set global parameters then use transaction code FINCS_BV_TASK or customizing for SAP S/4HANA for Group Reporting -> Data Collection for Consolidation -> Define Balance Validation Tasks 
  2. Enter the new validation tasks without assignement of document type selection or currency
  3. Try to save the new entries.
  4. Observe that the system raises error FINCS_BV_RGASGMT003.

Cause

Missing mandatory configuration in the validation task definition (at least currency not selected) when creating new validation tasks in FINCS_BV_TASK.

Resolution

  1. Open transaction FINCS_BV_TASK.
  2. For each of the new validation tasks maintain the currency 
  3. Choose the appropriate currency per task based on your configuration (for example local currency or group currency).
  4. Save it
