Symptom

A discrepancy is observed between the results shown in the campaign reports within SAP Emarsys and the results returned by the query executed using open data. The difference can be seen in metrics like Sent, Opened, Delivered and so on.

Environment

SAP Emarsys

Cause

The discrepancy is expected and is explained by the SAP Help article Differences between Trends and Open Data. Please read the linked document for a more details explanation but to summarize:

  • Open Data reflects contact deletions, which results in smaller KPIs over time as more contacts are deleted.
  • The email analysis data in Emarsys is collected for 30 days and aggregated, meaning data is only added but not deleted.

Resolution

  • Avoid attempting to reproduce Trends reporting using Open Data, as Trends is based on special conditions that cannot be recreated.
  • Rely on the organization's own interpretation, which suits individual account requirements better than the generalized Trends reporting.
  • Extract data and manipulate it according to the organization's definition of valid metrics, such as opens or clicks. Apply filters and build queries tailored to specific needs.
  • Consider using the API as an alternative for obtaining numbers. Please refer to SAP Emarsys API: Bulk response summary.
