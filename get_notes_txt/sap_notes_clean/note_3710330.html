Symptom

  • With the file versions feature, story designers are now able to save different versions of optimized story since SAC 2026 QRC1 Update  

Environment

  • SAP Analytics Cloud 2026.2

Resolution

  • Since SAC 2026 QRC1 update, users are now able to save different versions of optimized story.
  • For each version, users will know who saved it and when. Also, users wil be able to open any version in read-only mode. 
  • If it is required to revert to an earlier version, it is able to easily select and restore it, making it the most current version for editing.
  • For detailed information, please refer to help document Review, Restore, and Delete File Versions | SAP Help Portal.
