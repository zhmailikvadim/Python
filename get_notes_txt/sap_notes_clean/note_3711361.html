Symptom

  • A supplier has responded to a SOW Bid, but I have not received any notifications
  • I have email notifications enabled on my user profile
  • Messaging is enabled for the buyer company for SOW Bid Response - Evaluate

Cause

User role permissions are disabled for SOW Bid Response - Evaluate

Resolution

An Administrator or Program Office user can enable the user role permission:

  1. Navigate to the Admin configuration menu (gear icon) > Click User Role
  2. Click the desired user role > Click Edit
  3. Check the box for SOW Bid Response module - Evaluate
  4. Click Update
