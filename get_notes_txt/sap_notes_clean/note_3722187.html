Symptom

Possibility to create a Process Variant which will only contain the Compliance Forms Task.

Environment

SAP SuccessFactors Onboarding

Cause

System limitation.

Resolution

A process variant with only the Compliance forms is currently not supported.

The system requires either a Personal Data Collection Step, or an Additional Data Collection Step before the Compliance Task step.

This is a system limitation.

If you believe this is a potential enhancement to the product, please consider raising and enhancement request. For more information -  2090228 - How to submit enhancement ideas for SAP SuccessFactors Products - SAP for Me
