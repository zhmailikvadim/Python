Symptom

Notes 3646233 and 3708215 implemented the check when sending the notification record. If for an item the commodity code in the material master differs from the commodity code in the item, the notification record is not sent, and the error message is shown. The check is carried out only when the item has a BTI (binding tariff information), i.e. an authorization C626.

However, it is not checked if BTI situation was changed. The notification record must not be sent, and the error message must be shown, if BTI in the notification record and BTI in the material master differ.

Other Terms

6000165759, INC44792661, C2SFIBU, vZTA, BTI, STAWN, C626.

Reason and Prerequisites

Program error.

Prerequisites: notes 3646233 and 3708215 are implemented.

Solution

Implement changes according to the correction instructions.
