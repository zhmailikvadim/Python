Symptom

An error message for missing data for fields is raised at the activation step during mass processing of materials and the error message for Material X is incorrectly displayed against a different material in the error log.

Environment

  • SAP Master Data Governance- Consolidation and Mass Processing

Resolution

  • All Change Request errors that occur during saving are assigned to the last master data object written to the particular Change Request. Not all errors may be caused by specific master data objects.
