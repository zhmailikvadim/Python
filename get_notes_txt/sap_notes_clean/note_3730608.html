Symptom

  • Error message appears in purchase order Messages: "Error sending EDI message EDI_ORDERREQUEST_OUT_SPRX for purchase order".
  • Purchase order output IDoc shows an error.
  • Issue occurs after creating/saving and then checking the Messages of the newly created purchase order.

Environment

  • Product: SAP S/4HANA

Reproducing the Issue

Post purchase order
Failed output or error: Error sending EDI message EDI_ORDERREQUEST_OUT_SPRX for purchase order

Cause

Missing customizing. 

Resolution

  1. In SPRO: Cross-Application Components > General Application Functions > SAP Application Interface Framework > Interface Development > Define Namespace, define the "/MMEDI" namespace.
  2. In SPRO: Cross-Application Components > General Application Functions > SAP Application Interface Framework > Interface Development > Define Interfaces, maintain the following for outbound purchase order request:
  • Interface: ORDREQOUT; SAP data structure: EDI_PURCHASE_ORDER_MESSAGE; raw data structure: EDI_PURCHASE_ORDER_REQUEST; record type: PURCHASE_ORDER_REQUEST; set the "corrstruct" checkbox.
  • Init function before mapping: EDI_MM_PO_OUT_PROCESS.
  • Proxy class outbound: CO_EDI_ORDER_REQUEST_OUT; proxy method: ORDER_REQUEST_OUT.
  • Application engine: Proxy; Persistency engine: XML; Selection engine: AIF Index tables; Logging engine: Basic application log.
  1. Verify proxy and AIF implementation:
  • Ensure ABAP interface II_MMPUR_PO_REQUEST_IN has an implementation class CL_MMPUR_PO_REQUEST_IN for proxy PurchaseOrderRequest_In (XI Procurement).
  • In table /AIF/T_FINF, ensure there is an entry (field PROXYCLASSNAMEXI) for CO_EDI_ORDER_REQUEST_OUT.
  1. Run transaction /AIF/CONTENT_EXTRACT and enter scenario SAP_COM_0238. Optionally enable "Customizing to be recorded" and "Configuration to be recorded" if transports are required.
  2. Reprocess the purchase order output and confirm the message is sent successfully via the proxy/AIF. If the error persists, review the AIF logs and application log for details and adjust the above settings accordingly.
