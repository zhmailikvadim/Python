Symptom

"Looks like the data isn't loading" error is raised when loading Temporary Work Schedule and Temporary Holiday Calendar details screen on SF Mobile App on Android mobile devices. 

As a result of this error, it is not possible to remove temporary work schedules nor temporary holiday calendars even though latest SF Mobile app has been installed for Create and Delete Temporary Work Schedules in Mobile and Create and Delete Temporary Holiday Calendars in Mobile .

It works fine with iOS mobile devices.

Environment

  • SAP SuccessFactors Employee Central - Time Off
  • SAP SuccessFactors Employee Central - Time Sheet
  • SAP SuccessFactors Time Tracking

Reproducing the Issue

  1. Log on SAP Mobile App.
  2. Click on View My Calendar.
  3. View all Temporary Changes.
  4. Click into any existing Temporary Work Schedule or Temporary Holiday Calendar.
  5. Then the screen shows "Looks like the data isn't loading" with "Something went wrong. Please try again.".

Resolution

This is confirmed to be a defect on Android device only, and we are now working on a fix of it.
It is currently planned to deploy the fix with 2605 release. 

Before the fix is deployed, please use the SF web version instead to view or delete temporary changes including temporary work schedule and temporary holiday calendars.
