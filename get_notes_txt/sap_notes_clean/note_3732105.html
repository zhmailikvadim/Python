Symptom

  • Mass Activity (EBW_DQ_SS) is running and send Data to Delta Queue
  • Mass Activity finished without any error message
  • Check of Data in Delta Queue shows no billing Dcoument

Environment

  • SAP S/4HANA
  • SAP ERP
  • SAP Data Sphere

Reproducing the Issue

  1. Close reconciliation keys using FPF2.
  2. Run EBW_DQ_SS.
  3. Check RSA3 and observe that data exists but the billing number is missing.
  4. Open ODQMON and observe that no data is available and no subscriptions are present for the data source.

Cause

  • Billing Documents are different from IS-U Billing Dcouments
  • IS-U Sales Statistic just made for IS-U Billing Dcouments

Resolution

  • Use of different Data Sources to extract Billing Documents different from IS-U Billing Documents
