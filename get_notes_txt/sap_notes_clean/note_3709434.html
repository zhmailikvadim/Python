Symptom

The Service Definition for the interfaces Service Entry Sheet Request and Invoice Request is created with incorrect URLs, causing the transaction to fail with a 401 error.

Other Terms

S4CINT_INTERFACE_SETUP, S4CINT_IF_MASTER

Reason and Prerequisites

The issue occurred due to erroneous code that generated the wrong URLs.

Solution

The code has been corrected. The Service Definition now uses the correct URLs, resolving the issue.
