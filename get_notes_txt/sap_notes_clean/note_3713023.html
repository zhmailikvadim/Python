Symptom

  • DSN generates an error in the DSN block 41 during a change of collective agreement.
  • The old code for DSN field 41 no longer exists, leading to the error.

Environment

France

Cause

The old code for DSN field 41 is no longer available, causing the system to generate an error during its creation.

Resolution

  1. Override the generation of DSN field 41 by customizing the mapping in V_T5F99FMAP.
  2. Add the field COCOL in the CONT section of table T5F99FMAP_FIELDS.
  3. Map the old value to the new one effectively in V_T5F99FMAP.
  4. Test the changes to ensure the expected result is achieved.

Keywords

sap, dsn field 41, collective agreement change, error, old code missing, v_t5f99fmap, t5f99fmap_fields, mapping, override, france, sap error; S21.G00.41.011, collective agreement codes.
