Symptom

During the loading of the SAP ME Activity Manager, the system initializes UI libraries that use jQuery 2.2.4, an outdated version that is no longer maintained.

Other Terms

SAP ME, Welcome page

Reason and Prerequisites

Steps to Reproduce:

1. Open the SAP ME Home page.
2. Open Developer Tools in the browser.
3. Inspect/trace the UI files being loaded.
4. Observe that the system is using jQuery version 2.2.4 within the Activity Manager module.

Actual Result: The system loads an older jQuery version (2.2.4), which an analysis tool flags as outdated.

Expected Result: SAP ME should load a supported and up‑to‑date version of jQuery.

Solution

The solution is delivered as a code‑level correction. The system now imports jQuery version 3.7.1, an actively maintained release that provides improved stability, modern functionality, and updates not present in older branches.

Impact

The code change implemented to address the symptom above is isolated to Welcome page.
