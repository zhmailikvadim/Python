Symptom

Error "502 Bad Gateway error" when deleting large number of account members in SAC

Environment

SAP Analytic Cloud 2025.21.11

Reproducing the Issue

  1. Log in to SAP Analytic Cloud
  2. Open Model
  3. Try to Delete all account members 

Notice: Error "502 Bad Gateway"

Cause

Timeout issues arise during the deletion process when attempting to handle an excessive number of members simultaneously.  

Resolution

  1. Delete account members in smaller batches. 
  2. Ensure smoother processing by reducing the number of members deleted simultaneously.
