Symptom

You use Change Request Management to control your software distribution.

You have performed some transport actions, for example, create transport of copies. After that you save your document on the CRM WebClient UI, but the transport action logs are not added as the text log after saving.

Other Terms

ChaRM, Transport, Log, Save, ToC, Critical Object, Allowlist Object, SM_CRM, CRM_UI

Reason and Prerequisites

This is due to a program error.

Solution

Implement the attached corrections to your SAP Solution Manager system by using the Note Assistant (transaction SNOTE), or import the relevant Support Package.
