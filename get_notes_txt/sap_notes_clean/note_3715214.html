Symptom

You are using Production Planning and Detailed Scheduling (PP/DS) for SAP S/4HANA 2025 FPS1. You want to visualize and evaluate production planning relevant KPIs interactively while working on a PP/DS application.

Other Terms

PP/DS, Production Planning and Detailed Scheduling, PPDS, PP/DS for SAP S/4HANA, PP/DS Analytics, KPI Monitoring

Reason and Prerequisites

  • Ensure that you are using SAP S/4HANA 2025 FPS1.
  • The minimum required revision of SAP HANA 2.0 is Support Package 07 Revision 77.
  • Perform all the steps given in the Solution section of the SAP Note 2407589.

Solution

Overview

"Monitor Production Plan Performance" app provides production planners with real time visibility into key performance indicators (KPIs) relevant to interactive production planning. It enables immediate access to KPI calculations based on the current planning context, helping users validate whether their decisions are progressing toward targeted production outcomes.

Please refer to the help page Monitor Production Plan Performance for more details.

Recommendations

SAP recommends restricting the evaluation time range and object selection in the source PP/DS application to business needs and the analysis period, because navigation to the Interactive KPI Monitor and KPI rendering depend on the application context, particularly the number of selected PP/DS objects.

Restrictions

  • This application is not available as a standalone app and cannot be launched from the SAP Fiori Launchpad for the moment. It is accessible only when opened from PP/DS application via SAP GUI or VH apps (for example: Resource Planning Table, Product Planning Table, and Detailed Scheduling Planning Board).
  • Since the main purpose of the app is to read the KPIs from the context of a PP/DS application and analyze the planning situation based on those performance indicators, filters are not enabled and displayed in the app.
  • Saving as variant functionality in the app is disabled as the app is used interactively with context based on the PP/DS application and is bound to a current simulation session.
  • The KPIs in the tile are plotted on a grey scale chart and are not associated with any Thresholds definition for now.
  • The KPIs related to resource capacity are delivered as fixed set of KPIs for the release SAP S/4HANA 2025 FPS1.
