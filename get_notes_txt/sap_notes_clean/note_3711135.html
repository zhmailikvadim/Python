Symptom

In the PDF output of the HSKPSTM0 report, data is missing in section III. "Zrážky zo mzdy" in the "v prospech" column when the report is generated for more than one employee.
Also the PDF output of the HSKCDPRZ Confirmation of payment there is a missing text in the section 12 in the brackets "(vyznačí sa x)" instead of "(vyznačí sa x a dátum)".

Other Terms

HRSK, HSKPSTM0, PDF, HR_SK_HSKPSTM0_ZAM.

Reason and Prerequisites

This SAP Note is relevant only for Slovakia (HRSK).

Solution

This SAP Note delivers corrections of PDF output data binding in report HSKPSTM0 and PDF output text in report HSKCDPRZ confirmation of payment  described in the chapter Symptom.

- In report HSKPSTM0 correction of the text has been made in text "(§76a Zákonníka práce)" in the PDF output and Word template. Due to that a new Word template with this correction has been included.
