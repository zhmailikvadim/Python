Symptom

When trying to add skills from resume in Growth Portfolio, there are no results displayed, even though skills are standardized and configuration is properly set. 

Environment

SAP SuccessFactors Talent Intelligence Hub 2H 2025

Reproducing the Issue

  1. Go to Growth Portfolio from the Home page.
  2. Choose View All next to the skills section and navigate to the Skills page.
  3. Choose Add from Resume.
  4. Choose Browse and upload your resume.
  5. See that no skills are being displayed. 

NOTE: Prior the above steps, make sure configuration is set as per guide, and skills are standardized. 

Cause

This is a Known Issue.

Resolution

The fix for this issue is planned to be deployed on 1H 2026 release.

For release timelines, please review SAP SuccessFactors Product Release & Road Map Information.
