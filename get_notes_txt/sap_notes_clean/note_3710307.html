Symptom

You run the Tax Reporter (PU19) transaction for the Wage and Tax Statement for Nebraska (HR_F_MMREF_1_NE) file, and in the state account number in position 248 of Record RS, the system reports an incomplete state account number.

Example:

Account number configured in V_T5UTI:

Auth TT. Employer ID number Description
NE 001 199-99999 Withholding Tax

State account number reported in RS record in position 248: 00000000000009999999.

Other Terms

PU19; Tax Reporter; V_T51T8, RPCTRPU0.

Reason and Prerequisites

Reason

The issue occurs when the state account number has more than 7 digits due to a customizing configuration.

This issue is a side effect caused by the solution delivered by SAP Note 3522874 - TR: NE W2 incorrect file format for State Account Number in Record RS

Prerequisites

The following Support Package level is the minimum prerequisite:

Release Support Package
S4HCM 102 HR SP 00
S4HCM 101 HR SP 26
S4HCM 100 HR SP 39
SAP_HR 6.08 HR SP F0

For more information, access the SAP ERP Human Capital Management - Support Package Schedules (SAP_HR) or SAP S/4HANA On Premise Human Capital Management (S4HCM).

When you install an SAP Note, some prerequisites might be required. The prerequisites are SAP Notes that have already been released and that change the same context of the current one. Solution

From now on, the system reports the complete state account number in Record RS in tax form HR_F_MMREF_1_NE when it has more than 7 digits.

Consider the example from the symptom section:

State account number reported in RS record in position 248: 00000000000019999999
