Symptom

This KBA explains where the F4 help value comes from for the Format field when importing bank statements in "Manage Incoming Payment Files"

Environment

SAP S/4HANA

Reproducing the Issue

1. Import a bank statement file in "Manage Incoming Payment Files"
2. Check the F4 help value

Resolution

Technically speaking there exist 2 data source for the format displayed in "manage incoming payment files"  for the format F4 help.
1. The customized format configured in SPRO->Financial Accounting->Bank Accounting->Business Transactions->Payment Transactions->Electronic Bank Statement->Configure Formats for Import (SAP Fiori)
2. Formats which are configured in "Map Format data for incoming files from bank", Mapping Type: BSED & Status "Released".
