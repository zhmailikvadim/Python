Symptom

You are cancelling electronic documents for Mexico and want to correct UUID reference regarding case sensitive.

Other Terms

MX, eDocument, eDoc, eInvoicing, e-Invoicing, electronic invoice, electronic invoicing, e-Document, e-Docs, e-Doc, Document Compliance, Document and Reporting Compliance, electronic document processing

Reason and Prerequisites

Reason: 

This SAP Note is relevant only for Mexico.

You have SAP Notes 3152003 and 3252833 applied in your system.

Prerequisite: 

You have implemented the Electronic Document Processing Framework in your system. Refer to the implementation instructions provided in the SAP Help Portal for SAP ERP or SAP S/4HANA.

For a complete list of all corrections, check the following SAP Notes:

  • SAP Note 2526771 - eDocument Mexico: Overview Note (SAP ERP) 
  • SAP Note 2565791 - eDocument Mexico: Overview Note (SAP S/4HANA)

Solution

After you implement this SAP Note, you will cancel documents without erros.

SAP recommends that you implement the Support Package specified for your release. However, if you need to install a solution earlier, use the SAP Note Assistant to implement the correction instruction. You can find more information about the SAP Note Assistant in SAP Support Portal, under SAP Note Assistant.
