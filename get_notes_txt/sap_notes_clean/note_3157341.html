Symptom

Info object /ERP/GL_ACCT does not contain any data or only 1 entry in master data table /B631/PGL_ACCT, but there is data in SKA1 table.

Environment

  • Controlling (CO)
  • SAP S/4HANA
  • SAP S/4HANA Finance
  • SAP S/4HANA Cloud Private Edition
  • SAP Fiori for SAP S/4HANA

Resolution

The InfoObject /ERP/GL_ACCT is part of the BPC optimized content /ERP/SFIN_PLANNING. The InfoObject is defined with a virtual hierarchy where the data is taken via the implementing class and not from some special master data table.

For example, you can see that in transaction RSH1, the financial statement version GICG is displayed for the InfoObject /ERP/GL_ACCT. This is because the financial statement version GICG has been replicated in transaction HRRP_REP. If the HRRP* tables are empty (nothing is replicated), the data is selected directly from master data. However, as soon as there is at least one hierarchy replicated the data are since then selected from HRRP* tables.

Keywords

/ERP/GL_ACCT, /B631/PGL_ACCT, SKA1, /ERP/SFIN_PLANNING, HRRP*, HRRP_REP, RSH1, SE16, SE16N
