Symptom

Querying a virtual table with a (N)CLOB column returns the error "Requested to write attribute values of type nclob to OutputValueWrapper".

Applies to: SAP HANA 1.0 SPS 12 - resolved in SPS 12 rev 19

SAP HANA 2.0 SPS 000 through 03 - resolved in SPS 02 rev 20 and SPS 03 rev 30

Other Terms

 SAP HANA database, SDA, CLOB, NCLOB

Reason and Prerequisites

This behavior occurs if the virtual table contains incorrect metadata. See columns DATA_TYPE_ID and CS_DATA_TYPE_ID in the TABLE_COLUMNS table.

Solution

For SPS 12, upgrade to the latest revision and then drop and recreate the virtual table.

For SPS 00 or 01, upgrade to the latest revision of SPS 02 or O3, then drop and re-create the virtual table or refresh the definition..

For SPS 02 or 03, upgrade to the latest revision and then drop and re-create the virtual table or refresh the definition.
