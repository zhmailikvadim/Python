Symptom

In the Fiori Launchpad, some tile subtitles are not translated and cannot be maintained via transaction SE63. The subtitle "Submitted" for the Screen Maintenance Requests app (Fiori ID: F4072) and the subtitle "Open" for the My Maintenance Requests app (Fiori ID: F4513) are currently treated as a numeric unit, which is defined as a non-translatable field in the metadata of dynamic launcher tiles.

Other Terms

Screen Maintenance Requests, F4072, My Maintenance Requests, F4513, Descriptor Item ID: 42010AEE2A7C1EEAA7E33CDFE5F76D95, Descriptor Item ID: 42010AEE2A7C1EEAA7E33CDFE5F7ED95, TILE_INFORMATION, Translation, English, Package: I2D_EAM_WORKREQ_OVERVIEW_A1, Package: I2D_EAM_WORKREQ_OVERVIEW_A2, MM_APP, SE63, PM-FIO-WOC-MN

Reason and Prerequisites

Program error.

Solution

Apply corrections of the SAP Note or change the metadata field type (in transaction MM_APP) from "Unit of Value" to "Tile Information" so that the subtitles can be translated (and maintained in SE63).
