Symptom

This SAP Note is a collective Note that provides you with guidelines, a list of frequently asked questions and SAP Notes, and information for the implementation of the Czech Republic EC Sales List (CZ_ECSL) report in SAP S/4HANA.

Make sure you check this SAP Note regularly for an updated list of frequently asked questions and relevant SAP Notes.

If you need further support, open an incident on SAP Support Portal under the FI-LOC-ECS component.

Other Terms

Czech Republic, Czechia, ECSL, EC Sales List, CZ_ECSL, FAQ, Implementation Leading Note, SAP Document and Reporting Compliance, DRC, SAP S/4HANA, Statutory Reports

Reason and Prerequisites

Reason

You want to understand, install, configure and/or use Czech Republic EC Sales List in SAP S/4HANA.

Prerequisites

Check SAP Note 3033455 to find out more information about SAP Notes that are required for statutory reporting in SAP S/4HANA.
