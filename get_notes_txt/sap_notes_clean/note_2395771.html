Symptom

During the generation of a DataStore object (advanced), an error occurs. The error message "Operation &3 could not be carried out for &1 &2" (RS_EXCEPTION 120) occurs. The error occurs because no key fields are defined in the Open ODS view. To generate a DataStore object (advanced), key fields must be defined in the Open ODS view.

Other Terms

RS_EXCEPTION120; Generate DataFlow

Reason and Prerequisites

Program error

Solution
