Symptom

  • Sales Order integration fails with the error "Could not create planning objects" in RTI
  • In Settings for Order-Based Planning app, "Create Only Sales Orders with Planning Objects" is set to No 

Environment

  • SAP Integrated Business Planning for Supply Chain

Cause

  • A database procedure used for planning object ID generation was invalidated, leading to "Could not create planning objects" during sales order inbound.
  • Planning area artifacts and dependencies require reactivation to restore valid procedures.

Resolution

  1. Reactivate the planning area with dependencies.
  2. After successful activation, reprocess the failed inbound sales order messages/queues.
