Symptom

You are using SAP S/4HANA Cloud, public edition. 

You want to extend the existing SAP S/4HANA Migration Cockpit data migration object Routing via Model Your Migration Objects - Migration Object Modeler app. 

Environment

SAP S/4HANA Cloud Public Edition 

Cause

The extension of SAP S/4HANA Migration Cockpit data migration object Routing (SIF_ROUTING_2) is restricted. 

The migration object Routing is based on the BAPI BAPI_ROUTING_CREATE. 

Resolution

The following API structures and fields are currently not used by the migration object Routing. 

They can be used to extend the data migration object. 

Other fields and structures than those mentioned below are not allowed to be used. 

Using other fields can cause serious issues and is outside of any support and warranty by SAP. 

Be aware that the values you then provide within the migration template must have the correct SAP S/4HANA Cloud value and format.
