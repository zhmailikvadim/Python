Symptom

The EarlyWatch report for a HANA database instance under section backup and recovery stating that no recent backup has been successfully completed.

When you check the HANA instance, you can confirm that there have been successful backups completed recently.

Environment

SAP HANA Platform Edition

Reproducing the Issue

Observe alert in Earlywatch report that no recent full backup has been successfully completed even though checking the HANA instance confirms there have been completed backups.

You may notice that it only lists a few days yet the column "Successful Data Backups" does have an entry.

Cause

The backup catalog has been cleaned or a job configured to keep less than a week's backup data.

The EarlyWatch report needs to see at last a week's worth of backup data.

Resolution

Maintain at least one week of backup data in the backup catalog
