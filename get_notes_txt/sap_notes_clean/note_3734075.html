Symptom

You encounter an error as below during the S/4HANA Conversion in phase PREP_INIT/SPACECHK_INI!

# Find the detailed information in log 'MSSSPACE.LOG':
3 ETQ398XSQL: if @disk_free_MB < 0
3 ETQ398XSQL: set @disk_free_MB = 0
3 ETQ398XSQL: end
3 ETQ398XSQL: select @disk_free_MB as disk_free_MB
4 ETQ010 Date & Time: 
1EETQ008 Error message: dbrc = 99: 'all DB errors, except for ...'
1EETQ009XDB code 300
1EETQ009XVIEW SERVER PERFORMANCE STATE permission was denied on object 'server', database
1EETQ009X'master'.  

Reproducing the Issue

Run SUM tool.

Cause

SUM is trying to query SQL Server system views, but the DB user does not have sufficient privileges. 

Resolution

You need to grant the required permission at the SQL Server instance level.

Run this on the SQL Server (as a sysadmin) to the <SID> DB user

GRANT VIEW SERVER PERFORMANCE STATE TO [<SID>];

Keywords

HANA, S/4HANA, PREP_INIT/SPACECHK_INI, SQL Server, Microsoft
