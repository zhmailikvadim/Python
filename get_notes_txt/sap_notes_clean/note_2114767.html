Symptom

Whenever the user was attempting to use the Action Buttons for entries in the Opt Out Codes List, only the first record was being retrieved for editing instead of the selected row.

Other Terms

Display Home Page; 360 fact sheet

Reason and Prerequisites

The Data Collection was not being passed through the outbound plug to the inound plug of the target view. Instead the target view was retreiving all the records belonging to the collection and displaying the first one by default.

Solution

The selected record is being passed to Custom Controller of the target node which was earlier being filled with the entire collection of records. Please implement the note to correct the issue. The fix will be available as part of FRMCM release 200 SP 01.
