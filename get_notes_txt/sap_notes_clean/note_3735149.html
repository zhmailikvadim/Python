Symptom

Is ASE 16.0 impacted by CVE CVE-2025-24813 and CVE-2023-46604?

Environment

SAP Adaptive Server Enterprise (ASE) 16.0 & 16.1

Cause

This refers to (external content) CVE 2025-24813 and CVE 2023-46604.

Resolution

  • ASE 16.0 is not impacted by these CVEs.
    • The below 2 CVEs are related to FOSS used by AMC component within ASE. 
  • CVE-2025-24813 - Not affected because AMC does not support file upload.
  • CVE-2023-46604 - Not affected because AMC does not use ActiveMQ.

See Also

CVE, vulnerabilities, Apache Tomcat Path Equivalence Vulnerability, Apache ActiveMQ Deserialization of Untrusted Data Vulnerability
