Symptom

You configure an exception for a certain check result in the check group configuration. The system creates a corresponding exception document or clarification case after check execution. However, you cannot add the messages related to the check to the exception document or clarification case.

Other Terms

APE, Check Messages, Exception, Clarification Case, BPEM

Reason and Prerequisites

The feature of adding messages from check execution to the exception is not supported in APE.

Solution

After implementing this SAP Note, the system supports adding messages from check execution to exception documents and clarification cases.

Add the EXPORTING parameter ET_CHECK_MESSAGE TYPE /APE/T_CHECK_MESSAGE to your check method. For example:

```
METHODS your method
    EXPORTING
        et_check_result TYPE /ape/t_check_result
        et_check_message TYPE /ape/t_check_message
    RAISING
        /ape/cx_exception.
```

If the check result is linked with an exception code in the check group configuration, the system adds the messages related to the check in the et_check_message to the exception.
