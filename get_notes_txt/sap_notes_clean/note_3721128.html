Symptom

You run the report ARUN_SDO_TEMPORARY_ASSIGNMENTS for a supply assignment relevant article.
The system selects already completed third-party orders as open requirements for temporary supply assignment.

Other Terms

Temporary Assignment, ARUN_SDO_TEMPORARY_ASSIGNMENTS, ARUN

Reason and Prerequisites

Reason

This issue is a program error.

Solution

The program corrections are contained in the tab correction instruction of this note. You need to implement the corrections in your system.
