Symptom

When saving a purchase order, an error occurs: "Error in external tax system: (5002) External: 5002 - VertexInvalidTaxAreaIdEx"

Environment

SAP S/4HANA Cloud Public Edition.

Reproducing the Issue

  1. Open a purchase order.
  2. Mark the deletion indicator for a limit item, or add a new line item to an existing purchase order.
  3. Choose Check or Save.
  4. Observe the error: "Error in external tax system: (5002) External: 5002 - VertexInvalidTaxAreaIdEx".

Cause

The tax jurisdiction code is time-dependent and not accepted by the tax partner. 

Resolution

  1. Identify the tax jurisdiction code determined for the affected purchase order/item.
  2. Contact the external tax partner (for example, Vertex) to verify whether the jurisdiction code is valid and has not expired.
  3. Redetermine the tax jurisdiction code in the document or adjust the relevant master data so a valid code is used.
