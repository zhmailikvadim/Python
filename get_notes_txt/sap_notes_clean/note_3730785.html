Symptom

Clicking the arrow on the extreme right of the New Posting icon in the Consolidation Data Release Monitor triggers:

"App could not be opened either due to an incorrect SAP Fiori launchpad configuration or a missing role assignment.
Failed to resolve navigation target "#RealTimeConsolidation-releaseCockpit?...This is most likely caused by an incorrect SAP Fiori launchpad content configuration or by a missing role assignment." 

Environment

  • SAP Business Planning and Consolidation Embedded with RTC (Real-Time Consolidation), version for SAP NetWeaver

Reproducing the Issue

  1. Open and login to the Fiori Launchpad URL
  2. Navigate to the Consolidation Data Release Monitor (app F2620).
  3. Enter parameters: 

                Model: CONSOL
                Version: <Version>
                Entity: <Entity>
                Fiscal Period/Year: <Period/Year>

  4. Click Go.
  5. Click the arrow on the extreme right of New Posting → error appears.

           

Cause

  • The error occurs due to missing OData services required for Real-Time Consolidation (RTC).
  • Additionally, the user who loads Release Cockpit must have the role SAP_BR_CONSLDTN_SPECIALIST assigned.

Resolution

  • The user who loads Release Cockpit must have the role SAP_BR_CONSLDTN_SPECIALIST assigned
  • The following services need to be registered and activated (via /IWFND/MAINT_SERVICE):

ZCONS_VALIDATION_RULE_SRV 1 Real-Time Consolidation Validation Rule CONS_VALIDATION_RULE_SRV 0 Not Supported Routing-based

ZCONS_DATA_MASS_APPROVAL_SRV 1 Consolidation Data Mass Approval CONS_DATA_MASS_APPROVAL_SRV

ZCONS_DATA_RELEASE_COCKPIT_SRV 1 Real-Time Consolidation Data Release Cockpit CONS_DATA_RELEASE_COCKPIT_SRV

  • There are also those services for having RTC working:

ZCONS_VECRESULT_SRV 1 Validation Rule Result CONS_VECRESULT_SRV 0 Not Supported Routing-based ZCONS_VALIDATION_METHOD_SRV 1 Validation Method CONS_VALIDATION_METHOD_SRV 0 Not Supported Routing-based ZCONS_DOCLIST_SRV 1 Consolidation Used Document List CONS_DOCLIST_SRV

ZCONS_LIB_FINANCIAL_STATEMENT_SRV 1 Contains all reusable financial statement components for RTC CONS_LIB_FINANCIAL_STATEMENT_SRV 

ZCONS_LIB_VALIDATION_RESULT_SRV 1 Real-Time Consolidation Validation Result Library CONS_LIB_VALIDATION_RESULT_SRV
