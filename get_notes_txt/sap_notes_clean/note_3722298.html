Symptom

In the process "Receipt of Invoice" (DE_B_CP_INVC_RECV), rejection error AF4 raised in DTD check E_0406 from the market communication system, even though the price catalog already existed in the business system.

This issue is relevant for the market role: SUP.

Other Terms

PriceSheetSenderDiffGUE, AF4, price catalog, EBD, E_0406, DTD check, BM_INVDOC_CREATE_REQ, receipt of invoice

Reason and Prerequisites

Program error.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the system reads the price catalog data into step 0100-0030 UBD data, and the AF4 rejection error is no longer raised. You need to trigger the DTD recheck process. You can verify the fix by checking that the rejection reason table is cleared after the DTD recheck process finishes in transaction code INVMON.
