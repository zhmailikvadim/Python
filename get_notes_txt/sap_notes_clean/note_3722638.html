Symptom

When navigating from the Manage Force Elements app and using the Copy to Force Element in the background, the Valid From field does not display or convert the date in the correct format.

Other Terms

 S/4HANA Defense and Security, Manage force elements, Copy to Force Element in the Background

Reason and Prerequisites

 Relevant code added for component change.

Solution

The date format is working properly in the Copy to Force Element in the Background.

Please also implement the backend Note 3717757 version 2 for the fix to work correctly.

As a general rule, SAP recommends that you install a solution by applying a Support Package. However, if you need to install a solution earlier, use the Note Assistant to implement the correction instruction. You can find more information about the Note Assistant in SAP Service Marketplace, under service.sap.com/note-assistant
