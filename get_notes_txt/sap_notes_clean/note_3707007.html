Symptom

When using CLFMAS IDOC, there is error: Message No. C1149 - Error during IDoc processing - see log for details.
Checking the log in SLG1 with Object CAPI & Subobject CAPI_LOG, it can find message C1003 - characteristic not found or not valid.
However, the characteristic exists in system.

Environment

  • SAP ERP Logistics (LO)
  • SAP ERP Classification (CA)
  • SAP R/3
  • SAP R/3 Enterprise
  • SAP ERP Central Component
  • SAP ERP
  • SAP Enhancement package for SAP ERP
  • SAP S/4HANA
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA Cloud Private Edition

Cause

The characteristic is not assigned to the corresponding class in CL02.

Resolution

Assign the characteristic to the class in CL02.

Keywords

CTMS_DDB_SET_VALUE, READ TABLE icind, LS_IMP_VALUES-ATINN, KSSK, KSML, KLAH, CABN
