Symptom

  • The mobile app theme does not respect the configured theme values: with background set to black and font set to white, the search bar displays employee names in bold black, making them invisible against the black background.
  • The issue occurs despite the font color being configured as white in the theme settings.

Environment

SAP Successfactors HCM Suite

Reproducing the Issue

  1. Open the mobile app and navigate to the global search bar.
  2. Perform a user search or any search
  3. Observe that the search bar appears in bold black text on a black background, instead of white.

Cause

A code-related problem in the mobile app’s handling of theme settings.

Resolution

The Engineering team has identified the root cause and developed a patch; the fix is scheduled for deployment in the H1 2026 release.
