Symptom

In the log file of a document with 'Failed to Push' status in Monitor Document section of Peppol Exchange app, the error message 'Server returned HTTP response code: 401 for URL' is being thrown.

Environment

  • SAP ERP
  • SAP S/4HANA Private Cloud Edition
  • SAP Document Reporting and Compliance service

Cause

Missing authorization.

Resolution

Ensure the user configured in the Destination connected to the backend system possesses all the authorizations specified in the 'User' section of page Setting Up Connectivity | SAP Help Portal. Additionally, run an SU53 log when re-processing the invoice to identify if any further authorizations are missing.
