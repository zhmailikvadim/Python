Symptom

In the Fiori app TOENE (Hedge Management Cockpit), selecting one line in the result table and choosing Hedge Request > FX Hedge Request, raises the message: "Select exactly one cell". The same navigation works in the classic SAP GUI, but fails in SAP GUI for HTML (webgui).

Environment

  • SAP S/4HANA
  • SAP GUI for HTML

Cause

A defect in SAP GUI for HTML grid selection handling causes CL_GUI_ALV_GRID->GET_SELECTED_CELLS to return all cells of the selected row instead of the individually selected cell, preventing single-cell selection logic. This is documented in SAP Note 3716874.

Resolution

  1. Refer to SAP Note 3716874 for the status and technical details of the correction.
  2. Apply the corresponding SAP kernel patch listed in Note 3716874 to fix the ALV single-cell selection behavior in SAP GUI for HTML.

To be notified when Note 3716874 is updated with patch availability, refer to KBA 2171560.
