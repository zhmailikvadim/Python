Symptom

You run the WCR Manager (report RPCWCRN1, transaction PC00_M05_CWCR1).

After you select the Manual Update option, you cannot display or modify the external costs, even though you have the required authorizations.

The system checks whether table V_WCR_UPDATE has the required authorization via authorization object S_TABU_NAM. If the table name check fails, the system does not check authorization object S_TABU_DIS to verify whether you have authorization to this table group.

Additionally, the system displays an incorrect error message ("no display authorization") when you try to change data.

Other Terms

WCR, RPCWCRN1, PC00_M05_CWCR1, S_TABU_NAM, S_TABU_DIS, V_WCR_UPDATE

Solution

After implementing this SAP Note, the system performs the authorization check for S_TABU_NAM first. If this check fails, the system then checks S_TABU_DIS to verify whether access is granted via the table group.
