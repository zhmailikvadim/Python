Symptom

You are going to implement Note 3051466 and 3092620. You find that, they are prerequisite Note for each other. It is confused in which sequence should they be implemented.

Environment

NetWeaver ABAP based systems.

Reproducing the Issue

Implement Note 3051466 and 3092620.

Resolution

Please implement Note 3051466. Its prerequisite Note 3092620 will be implemented automatically.
