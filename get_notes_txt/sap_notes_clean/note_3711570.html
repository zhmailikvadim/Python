Symptom

  • Schedule publication for story shows as Partially Successful
  • Emails for this schedule publication have been successfully delivered to some recipients. However, for some other recipients, this email for schedule publication failed to be delivered with below error message:
    "Info: Email delivery failed due to SMTP authorization error. Please verify the email server credentials and configuration. "
  • This SAC system uses the Default Email Server

Environment

  • SAP Analytics Cloud 2025.21.11

Reproducing the Issue

  1. Log onto SAC
  2. Navigate to the Calendar of this SAC system
  3. Enable the Publication Admin Mode for this Calendar
  4. Check these "Partially Successful" schedule publication from Feb 2, 2026
    => Observe that status of these schedule publication would show as "Partially Successful"
  5. Check the Detail of these "Partially Successful" schedule publication
    => Observe that the Generation Status of this schedule publication is successful. And this schedule publication may succeed delivering to some recipients.
    => However, also observe that this schedule publication may fail to deliver to some recipients with error message "Email delivery failed due to SMTP authorization error. Please verify the email server credentials and configuration. "

Cause

These issues originated from network issues on CN40 landscape due to which the email deliveries are affected. Authentication to the SMTP is failing due to the network issue. 

Resolution

This issue has been resolved by the fix delivered in the infrastructure level.
