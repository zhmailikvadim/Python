Symptom

  • Item description is displayed on the Item Details page of the catalog item by default.
  • Users may want to configure catalog item descriptions to appear on the Catalog search results page.
  • Steps to make the item description visible on the search results page are unclear.

Resolution

  1. Login as a customer administrator to parent realm and go to Intelligent Configuration Manager.
  2. Enable the parameter Application.Catalog.EnableDescriptionInSearchConfiguration
  3. Now, go to Manage -> Core Administration -> Catalog Manager -> Catalog Display Customization.
  4. Navigate to the Fields section -> Find the field Description -> Edit.
  5. Enable the toggle for In Search Results  and click Save.

Prerequisites:

Catalog Display Customization must be enabled under Catalog Manager in your site.

If you would like to have this enabled, kindly follow the steps mentioned in this article - How to enable the menu Catalog Display Customization?
