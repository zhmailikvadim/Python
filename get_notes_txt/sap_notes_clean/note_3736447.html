Symptom

The material has special procurement type 30 set and during Costing in transaction CK11N, it's found that there is no G line item generated. 

Environment

  • Controlling (CO)
  • SAP ERP
  • SAP ERP Central Component
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA
  • SAP S/4HANA Finance
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Execute transaction CK11N
  2. Check the itemization in the result screen

Cause

In the Overhead tab of the valuation variant, the field 'overhead on Subcontracted Materials' is not marked so the system will not calculate the overhead. 

Resolution

Please mark the field  'overhead on Subcontracted Materials' in the valuation variant and check it again. Also please check the the plant-specific setting of the valuation variant.

Keywords

CK11N, CK24, CK40N, production costing, special procurement type, overhead, Mark, release, subcontract, subcontracting.
