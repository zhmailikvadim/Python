Symptom

Within the "Delegate My Workflows" Quick Action, the placeholder text "Select" in the dropdown field is displayed in English, even when the system language is set to another language. 

Environment

SAP SuccessFactors HCM Core

Reproducing the Issue

  1. Set the system language to non English language such as German.
  2. Proxy as a user with managerial permissions.
  3. Navigate to "Delegate My Workflows" in the Quick Actions section of the home page.
  4. Turn on automatic delegation.
  5. Specify the period during which workflows are automatically delegated.
  6. Choose how to delegate workflows -> Delegate workflows by groups.
  7. Observe that the field to select a group appears in English.

Cause

It has been identified that the translation for the placeholder text was missing from the delegate workflow group quick action input.

Resolution

This fix for this issue is planned to be deployed on 1H 2026 release.
For release timelines, please review SAP SuccessFactors Product Release & Road Map Information.
