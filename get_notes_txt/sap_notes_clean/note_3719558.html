Symptom

During payroll calculation, the system generates an incorrect link from block 53 (ACTI) to block 51 (REM).

The system creates an ACTI record with the wrong link to REM. This causes an error in DSN when the system uses this data to generate block 51 and child block 53.

Other Terms

RPLDSNF0, Payroll, REM, ACTI, FACTI, FCHEC.

Reason and Prerequisites

This is a correction for payroll function FCHEC.

The error occurs only when:

1. The system creates REM parent records with empty values as dummy parents for the ACTI record.
2. Multiple WPBP splits exist for the period.

Solution

After implementing this SAP Note, the system generates the correct link from the ACTI record to the corresponding REM parent record.

The correction enclosed within this SAP Note will be included in HR Support Packages referenced below. If you need the correction before the HRSP is available, you may apply the attached correction instructions.
It should be emphasized however, that correction instructions are only a short-term solution, not a substitute for the Support Package.
