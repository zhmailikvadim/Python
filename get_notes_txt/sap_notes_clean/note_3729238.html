Symptom

  • BAS -> Simplified Git, doesn't ask to put username&password or token
  • Would like to know where BAS -> Simplified Git saved username&password or token

Environment

SAP Business Application Studio

Reproducing the Issue

BAS -> Simplified Git, clone Repository from Git, but not asking to put username&password or token

Cause

BAS saved and reuse Git username&password or token under /home/user/.git-credentials

Resolution

Check /home/user/.git-credentials, it store username&password or token.

See Also

.git-credentials, .gitconfig, Simplified Git
