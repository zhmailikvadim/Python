Symptom

ERGANI2 requests new version of XML file WTO (WTO_v2).

Other Terms

ERGANI, ERGANI2, HGRLE2V0, WTO

Reason and Prerequisites

This SAP Note is relevant only for Greece.

Solution

Report HGRLE2V0(ERGANI II: EEs with variable work schedule + changes/absences of all) has been updated.
Two new fields added to xml based on absence quotas:
* f_year
* f_req_days

However, if you need to install a solution earlier, use the Note Assistant to implement the correction instruction. You can find more information about the Note Assistant in SAP Service Marketplace, under service.sap.com/note-assistant.
  • Please, note that documentation, transactions, and Customizing activities are only delivered via Support Package
