Symptom

 Future SAP Notes require this SAP Note to be implemented as a prerequisite, and/or you encountered problems installing SAP Note 3656279

Other Terms

SAFT, SAF-T, RPFIEU_SAFT, FIEU_SAFT, Luxembourg, SAF-T LU, FAIA, SAF-T(LU), DDIC, Technical Note, Interface Note.

Reason and Prerequisites

This SAP Note is relevant only for Luxembourg.

This SAP Note is a preparation SAP Note that provides DDIC objects for further developments.

If this SAP Note is required as a prerequisite for later fixes, you must implement it.

Solution

Import the Support Package specified for your release or implement the attached correction instructions.
