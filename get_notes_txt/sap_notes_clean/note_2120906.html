Symptom

What component software and license needs to be downloaded to setup replication from Oracle to HANA database with SAP Sybase Replication Server?

Environment

  • SAP Replication Server (SRS)
  • SAP Replication Server Options (RSO)

Resolution

Structure Diagram

Oracle Instance-> Replication Agent for Oracle(RAO)->SAP Replication Server(SRS with ECH)->HANA Instance

Components and Software

SAP Replication Server (SRS)

ExpressConnect for SAP HANA Database (ECH) which is bundled in SAP Replication Server
software and is installed with the SRS install.

Replication Agent for Oracle (RAO) which is in Replication Server options (RSO) package

Visit SAP support  portal at SAP Service Marketplace (SMP) http://service.sap.com to
download the above component software. It is recommend to download the latest support packages
of these components.

For example,

To download SRS 15.7.1 SP204, click download software->support packages and
patches->Browse Download Catalog->Sybase Products->SYBASE REPLICATION SERVER HANA
EDITION 15.7->SYBASE REPLICATION SERVER->SAP REPLICATION SERVER 15.7-><select your
platform-> choose SP204

To downlaod RAO 15.7.1 SP204, click download software->support packages and patches->Browse
Download Catalog->Sybase Products->SYBASE REPLICATION SERVER HANA EDITION 15.7->SYBASE
REP SERVER OPT ORACLE->SAP REP SRV OPT ORACLE 15.7->SAP REPLICATION AGENT 15.7-><select
your platform>-> choose SP204


Licenses

Replication Server license - REP_SERVER
ExpressConnect for SAP HANA Database license - REP_EC_HANA
Replication Agent for Oracle - RAO_SERVER

To generate licenses, visit https://support.sap.com/keys-systems-installations/keys.html
If a problem is encounterd with license generation,open an case using XX-SER-LIKEY-
SYB component.

Keywords

ECH, RAO, RepServer, "Replication Agent", RepAgent
