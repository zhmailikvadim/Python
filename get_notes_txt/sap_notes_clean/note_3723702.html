Symptom

This SAP note corrects problems with the aggregation of the bulk indicator for AIG components. If an alternative item group is changed or created, the bulk indicator can be set. It can then reset the bulk indicator from material master for some of the components. If a bulk component is assigned to an AIG, the aggregated bulk indicator will be set to true first for the new AIG as a default.

Other Terms

MPE_RTG_UI_HLP, screen 614, Alternative Item Group

Reason and Prerequisites

Coding errors.

Solution

Implement the correction instructions.
