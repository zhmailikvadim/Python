Symptom

  • Error message: "Note 0002755591 : no entry for DMIS 2011 SP 000000" when running report "CNV_NOTE_ANALYZER_SLT" from SE38.
  • Note 3016862 is applied with the latest version in the system.

Environment

SAP Landscape Transformation Replication Server (SLT)

Reproducing the Issue

  1. Implement SAP Note 3016862 as per SAP documentation.
  2. Perform pre and post manual activities as instructed.
  3. Run the report "CNV_NOTE_ANALYZER_SLT" from SE38.
  4. Observe the error message: "Note 0002755591 : no entry for DMIS 2011 SP 000000".

Cause

The report CNV_NOTE_ANALYZER_SLT is designed for DMIS 2011 SP11 or higher versions, while the system DMIS version is too old to use the report.

Resolution

As a workaround, reference note 2675613 to find the release note of the current DMIS version, apply the required notes as mentioned in the release note.
