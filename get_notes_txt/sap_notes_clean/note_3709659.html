Symptom

You are working with Treasury and Risk Management. For a specific security product and transaction type, you have set the Proc. Category to 00001 Order - Contract. As a result, when you create a deal for this product and transaction type, the cash flow correctly has a posting status of 1 ("flagged for posting"), since no settlement is required.

However, when you exercise an OTC security option deal, the newly created security deal (with the same product and transaction type) incorrectly has a posting status of 0, and the cash flows are blocked with the reason "Activity does not allow posting."

Other Terms

CL_TTM_SEC_CASHFLOW, PREPARE_FOR_CREATE

Reason and Prerequisites

Programming error.

Solution

Please implement the SAP Note or the Support Package.
