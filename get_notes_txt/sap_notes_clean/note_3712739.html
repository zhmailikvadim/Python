Symptom

  • When editing Contact Information (Email and/or Phone Information) or Job Relationship Information in the Latest People Profile with the latest Editing UI enabled, mandatory fields do not display the "*" indicator.
  • Although the mandatory "*" indicator is not shown, the system prevents saving and displays the following error message if a required field is left empty: "This value is required <field-name>"

Environment

  • SAP SuccessFactors HCM Suite
  • SAP SuccessFactors Employee Central

Reproducing the Issue

  1. Ensure that the Latest People Profile is enabled.
  2. Go to Configure Latest People Profile.
  3. Select General Settings.
  4. If not already enabled, enable the option:
    "Enable the latest Editing and History UI for Employee Central HRIS entities on Full Profile (Except Compensation Information)".
  5. Navigate to the Employee Profile of any user.
  6. Edit Contact Information using the edit (pencil) icon.
  7. Observe that mandatory fields (for example, Email Address) do not display the mandatory "*" indicator.
  8. Go to Manage Business Configuration and verify that the same field is configured as Mandatory = Yes.

Cause

This is a known issue.

Resolution

The fix for this issue is planned to be deployed on 1H 2026 release.

For release timelines, please review SAP SuccessFactors Product Release & Road Map Information.

Workaround

Temporarily disable the option "Enable the latest Editing and History UI for Employee Central HRIS entities on Full Profile (Except Compensation Information)".
