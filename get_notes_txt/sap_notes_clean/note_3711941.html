Symptom

  • I9 compliance migration request from Onboarding 1.0 to Onboarding.
  • No candidates available to perform test migration.
  • Inquiry regarding prerequisites for enabling "rehire verification" in Onboarding 1.0.
  • Inquiry regarding the impact of enabling "rehire verification" on Onboarding 1.0 processing.
  • Inquiry regarding the possibility of performing a mock test run in production.

Environment

SAP Successfactor Onboarding 1.0

Reproducing the Issue

  1. Access the Super admin.
  2. Perform the activity related to I9 compliance migration.

Cause

  • I9 migration.

Resolution

  1. Refer to the SAP guide for detailed steps on migrating Form I-9 data from Onboarding 1.0 to Onboarding:

a href="https://help.sap.com/docs/successfactors-onboarding/implementing-onboarding/migrating-form-i-9-data-from-onboarding-1-0" Migration of Form I-9 data from Onboarding 1.0 to Onboarding /a.  

  1. Ensure the "rehire verification" is enabled in Onboarding 1.0 as a prerequisite.
  2. Confirm that enabling "rehire verification" will not impact Onboarding 1.0 processing.
  3. Note that candidate data will not be deleted during the I-9 migration.
  4. Complete all inflight candidates in Onboarding 1.0 prior to performing the I-9 migration, as this is considered best practice.
  5. If a test system is not available, the migration can be carried out directly in the production system based on the requirements.
