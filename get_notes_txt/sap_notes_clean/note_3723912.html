Symptom

You get an Error: Employee XXXXXX is already the Organizational Unit YYY-ABCD from yyyy-mm-dd to yyyy-mm-dd when you are trying to add an employee with employee ID ZZZZZZ

Environment

SAP Sales Cloud and SAP Service Cloud Version 2

Reproducing the Issue

  1. Go to User Menu
  2. Go to Settings
  3. Go to All Settings
  4. Go to Company
  5. Go to Organizational Structure
  6. Open the Organizational Unit
  7. Go to Employees facet
  8. Go to Employees section
  9. Click on Add
  10. Add an employee with ID ZZZZZZ
  11. Error: Employee XXXXXX is already the Organizational Unit YYY-ABCD from yyyy-mm-dd to yyyy-mm-dd

Cause

There is an employee with ID XXXXXX maintained twice in the Employees section

Resolution

  1. Verify if the employee with ID XXXXXX is already assigned to the Organizational Unit.
  2. Remove any duplicate entries of the same employee in the Organizational Unit.
  3. Reattempt to add the employee with ID ZZZZZZ to the Organizational Unit.
  4. This time the error does not show up.
