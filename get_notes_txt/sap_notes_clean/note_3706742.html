Symptom

  • Differences are observed between the values reported in  Monthly income Declaration (DMR) (RPCMIDP0) and HR-PT: Individual Income Declaration (DIR) (RPCIIDP0).

Resolution

This KBA provides a list of checks and potential root causes that may explain these discrepancies. Please review the points described below:

  1. Check constant DIR 15 in table view V_T511K: Ensure that constant DIR 15 is active. Since 10.2020, SAP no longer supports report RPCIIDP0 if this constant is inactive (as communicated in GUSP July 2020).

  2. Verify subapplication TMNV is inactive: Confirm if subapplication TMNV in table view Validity Intervals for Non-Statutory Subapplications (V_T596D) is inactive. When TMNV is active, the validation process used in DMR is skipped in the DIR execution, which may lead to differences between DMR and DIR.

  4. Check for custom implementations in BAdIs and Enhancements: Ensure that no customer-specific logic is causing discrepancies. Review any custom code in:
    • BAdI HR_PT_MID_RP
    • Enhancement EXIT_RPCIIDP0_001
