Symptom

You execute the program payroll driver - Austria (RPCALCA0) for several persons.

Even though the yearly limit for special payments (2019: 10.440 euros) has not yet been reached, social insurance contributions for individual employees are not displayed for special payments.

Cumulative values of the SI (internal table SV_CRT) have multiplied.

Other Terms

IM_SV_CRT

CRT

BVA

GKK

mBGM

Reason and Prerequisites

This SAP Note is relevant only for Payroll Austria as of January 01, 2019.

You have implemented SAP Note 2755639. If there is a change from GKK to BVA or from BVA to the regional health insurance fund (GKK), this results in a backup of the totals table for multiple duplication if several persons with a combination of this kind are settled one after the other.

Solution

Implement the attached correction instructions or import the specified Support Package.
