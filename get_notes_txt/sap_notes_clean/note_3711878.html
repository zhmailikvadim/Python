Symptom

  • Translation cannot be added to survey answer options.
  • Only login language is available when attempting to add translation entries to survey answer options.
  • Globe button does not allow selection of other languages.

Environment

SAP Cloud for Customer

Reproducing the Issue

  1. Login to the system in English.
  2. Navigate to the Surveys workcenter.
  3. Search for and open target Survey.
  4. Go to the Design Facet and click Edit.
  5. Open the target question, scroll down to Answer options, and click Add.
  6. Enter text and click the Globe Icon.
  7. Observe that only English is available, and it is not possible to add another language.

Cause

Language is determined by the login language.

Resolution

  1. Log into the system using a different language, e.g., Deutsch.
  2. Navigate to the Surveys workcenter and open the target survey.
  3. Go to the Design Facet and click Edit.
  4. Open the target question, scroll down to Answer options, and click Add.
  5. Enter text and click the Globe Icon.
  6. The language shown is Deutsch.
  7. Enter translation and save.
  8. Repeat for desired language translations
