Symptom

You use the balance confirmation in PDF format for customers or vendors.
To send the balance confirmation by e-mail, use the Open FI process 00002410 to fill the e-mail addresses.
You define a large number of e-mail addresses so that the additional fields FINAA-INTAD*ADD are used.

After the PDF document has been displayed, the 'Send Internet Mail' dialog box appears.
However, only as many e-mail addresses are displayed (and also used) as fit in the fields FINAA_INTAD_CC and FINAA_INTAD_BCC, that is, only 130 CHAR. If necessary, a mail address is even truncated after the 130 characters.

Other Terms

FI_PREVIEW_PDF, SOST
F17P SAPF130D_PDF 
F18P SAPF130K_PDF 
F17OM SAPF130D_OM
F18OM SAPF130K_OM

Reason and Prerequisites

 SAP Notes 3509100 and 3512599

Solution

Implement the attached correction instructions in your system or import the relevant support package.
The dialog box now contains an ALV list with recipient type and mail addresses, as described in SAP Note 3512599.
