Symptom

Current Nested filters disappear when adding a new filter to them in web intelligence document.

Other Terms

lost, loose, empty, webi,

Reason and Prerequisites

Environment

  • SAP BusinessObjects Business Intelligence (BI) Platform 2025 Patch 08
  • SAP BusinessObjects Business Intelligence 4.3 Service Pack 05
  • Web Intelligence

Steps to duplicate

  1. Login to BI Launchpad.
  2. Create a new webi document.
  3. Add multiple objects to the Result objects.
  4. Run the report.
  5. Create a table containing those objects.
  6. Select the table -> show report element data -> filter.
  7. Add four filters to the table
  8. After adding the 4 filters, click on manage filters.
  9. Set the existing filters in the following way: [Filter1 OR Filter2] AND [Filter3 AND Filter4] (You can freely choose AND or OR, as long as you have relevant data).
  10. Apply and save.
  11. Now drag and drop a new filter in the filters panel.
  12. All the filters will go missing.
