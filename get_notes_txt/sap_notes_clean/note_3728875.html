Symptom

During SAP EHP upgrade using Software Update Manager (SUM), the extraction phase fails with "DBSL error 99 (db code -1): Connect failed" when executing SAPuptool.EXE dbversion command.




Environment

  • Windows
  • Oracle 19c
  • Software Update Manager (SUM)

Reproducing the Issue

  1. EXECUTING E:\usr\sap\<SYSNAME>\SUM\abap\bin\SAPuptool.EXE dbversion
  2. 2 ETQ399 Connecting to database 'ora'.
  3. 3EETQ008 Error message: DBSL error 99 (db code -1): Connect failed

Cause

Incorrect ORACLE_HOME environment variable - pointing to wrong Oracle installation directory

Resolution

1: Verify and Set ORACLE_HOME Environment Variable

Set the ORACLE_HOME environment variable to the correct Oracle 19c installation path:

ORACLE_HOME = <DRIVE>:\oracle\<DBSID>\19


2: Add Oracle bin Directory to PATH

Add %ORACLE_HOME%\bin to the system PATH environment variable:

  1. Open System Properties > Environment Variables
  2. Edit the PATH variable
  3. Add: %ORACLE_HOME%\bin
  4. Ensure this path is positioned before any other Oracle-related paths
