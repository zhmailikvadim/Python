Cause

Program error or required parameter missing.

Resolution

If applicable, implement SAP Note 3517094 - CX_ODQ_OBJECT_NOT_FOUND error in ODQ_QUEUES_CLEANUP

Also, ensure to fill both parameters MODEL and QUEUE while running the program.
