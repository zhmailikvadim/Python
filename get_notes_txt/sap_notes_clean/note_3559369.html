Symptom

  • The "Make Bank Transfers - Create with Templates" application does not display previously made bank transfers.
  • The application does not allow for any creation of new bank transfers.
  • The screen appears blank when launching the application.
  • App Make Bank Transfer - Create with Templates error message is displayed.
    "You have not entered anything. Enter at least an amount."

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Launch the "Make Bank Transfers - Create with Templates" application.
  2. Observe that the screen is blank and does not display any previously made bank transfers.
  3. Attempt to create a new bank transfer.

Cause

Resolution

Ensure the prerequisites are implemented before using the "Make Bank Transfers - Create with Templates" application.

  1. Confirm the following configuration settings are made in the Manage Your Solution app:
  • Clearing Accounts for Outgoing Transfers
  • Clearing Accounts for Incoming Transfers
  1. Ensure the bank account master data is maintained in the Manage Bank Accounts app.
  2. Ensure the payment method data is configured.
  3. Define templates in the App Define Bank Transfer Templates app before using this application.
