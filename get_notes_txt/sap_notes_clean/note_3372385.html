Symptom

You want to cancel an invoice that has a related ePayment. This operation should only be allowed in case the related ePayment has been cancelled first. However, this validation does not happen.

Other Terms

Mexico, Convergent Invoicing, CI, eDocument, eDoc, Cockpit, Cancel, ePayment

Reason and Prerequisites

This SAP Note is relevant only for Mexico and Convergent Invoicing.

Solution

After applying this SAP note, when you try to cancel an invoice that has a related ePayment that is not cancelled, you receive a message "Document cancellation not possible. First, cancel the ePayment XXX".

As a general rule, SAP recommends that you implement the Support Package specific for your release. However, if you need to install a solution earlier than that, use the SAP Note Assistant to implement the correction instruction. You can find more information about the SAP Note Assistant in SAP Support Portal, under SAP Note Assistant.

To apply the solution, you must implement the correction instructions that are valid for your release.
