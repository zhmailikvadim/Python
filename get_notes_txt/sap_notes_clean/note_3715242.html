Symptom

In the process "Sending of Quotation (MOS View)" (DE_C_MOSB_QUOT_MOS), a configuration error caused the transfer document (TDoc) of business message ID (BMID) BM_MOSB_QUOT_INFO_PROC_START to start a new quotation sending process in the business system instead of assigning it to an existing process when the existing process had an error in step 0015.
This issue is relevant for the market role: MOS.

Other Terms

MOSB, Quotation, DE_C_MOSB_QUOT_MOS, BM_MOSB_QUOT_INFO_PROC_START, BM_MOSB_QUOT_INFO_PROC_START_FROM_REQT, Information of MOS Billing Quotation Process Start by Request for Quotation, TDoc, BMID

Reason and Prerequisites

Configuration error.

Solution

Please implement the corrections of this SAP Note by importing BC sets /US4G/3_0_MOS_SP07_NOTE_3715242 and /US4G/3_0_MOS_SP08_NOTE_3715242 or the relevant Support Package (SP). For relevant information regarding your market communication system, see and implement emergency patch 2000011412.
After implementing this SAP Note, the system assigns the TDoc of BMID BM_MOSB_QUOT_INFO_PROC_START to the existing process, even if the existing process has an error.
You need to retest the process.
You can verify the fix by checking that the TDoc of BMID BM_MOSB_QUOT_INFO_PROC_START is assigned to the existing process, even if the existing process has an error.
