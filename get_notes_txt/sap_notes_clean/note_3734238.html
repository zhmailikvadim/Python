Symptom

You attempt to use field selections from the table “History per Purchasing Document” in the Automated Document Maintenance application, but the fields are not available for selection. The system does not provide access to these fields within the application as expected.

This note corrects the behavior by enabling the availability of “History per Purchasing Document” fields for selection in the Automated Document Maintenance application.

Other Terms

Automated Document Maintenance, Calculation Bucket,  /IRM/IPG_AUTO_DATA_MAINTAIN, Target, Source, Field Mapping

Reason and Prerequisites

This is caused by program error

Solution

Please implement the correction instructions attached. These corrections are only valid if you have applied the following support packs.

Release 6.1 – Base Package
