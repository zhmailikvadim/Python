Symptom

  • When attempting to create a new journal entry template in the Fiori app Manage Journal Entry Templates PP F3803 an error message displayed:
    Sorry, a technical error occurred. Please try again later.
    In the context of Data Services an unknown internal server error occurred
  • In the gateway error log (transaction /N/IWFND/ERROR_LOG) the following error message is present for service FAC_MANAGE_POSTING_TMPL_SRV:  A row was to be added that would have produced a duplicate of the key UX_FC_FIELD.

Environment

  • SAP S/4HANA
  • SAP S/4HANA Finance
  • SAP S/4HANA Cloud Private Edition

 

Reproducing the Issue

1. Navigate to the app Fiori F3803 - Manage Journal Entry Templates.
2. Select the option "Create".
3. In the pop-up, select the option "Post General Journal Entries" and click "Create".
4. Update the fields "Template ID" and “Access Level”.
5. Click the "Step 2" button and the error appears.

Cause

Program error.

Resolution

Implement SAP Note 3646343 - [SAP Fiori] "Manage Journal Entry Templates" app terminates with technical error  

Keywords

/IWFND/CM_CONSUMER122, row, added, duplicate, key UX_FC_FIELD, FAC_MANAGE_POSTING_TMPL_SRV, /IWCOR/CX_DS_INTERNAL_ERROR , context, Data Services, unknown, internal, server, error, CX_SY_ITAB_DUPLICATE_KEY, CL_FIN_ODATA_COBL_FLD_MAP_HLP=CP, CL_FIN_ODATA_COBL_FLD_MAP_HLP=CM005, Manage Journal Entry Templates, F3803
