Symptom

A role is automatically assigned to newly created user in SAC 

Environment

SAP Analytic Cloud 2026.2.5 

Reproducing the Issue

  1. Login to SAC
  2. Navigate to security --> users 
  3. Click on create users
  4. Provide user information and proceed to the next step without assigning a role or team.  
  5. Click on save
  6. Search for the created user in Security --> Users

Notice: A role is automatically assigned to the created user. 

Cause

The "Use as Default Role" option is enabled in the Role Configuration for the assigned role, which results in the automatic assignment of the role to newly created users when no role is manually selected.   

Resolution

  1. Navigate to SAP Analytics Cloud > Security > Roles. 
  2. Select the affected role and open Role Configuration.
  3. Disable the "Use as Default Role" option for the relevant roles.
