Symptom

  • Unable to install BI 4.3 SP04 client tools due to "Suppressed Reboot" error during Checking Prerequisite when running setup.exe.
  • There is no PendingFileRenameOperations subkey under HKEY_LOCAL_MACHINE\System\CurrentControlSet\Control\Session Manager.

Environment

  • SAP BusinessObjects Business Intelligence platform Client Tool 4.x

Reproducing the Issue

  1. Start the BI 4.x client tool installer on Windows (run setup.exe as administrator).
  2. During the prerequisite check, observe "Suppressed Reboot" reported and the installation does not continue.

Cause

  • Excessive temp files in the user's TEMP folder where the installer extracts content.

Resolution

  1. Clear the user's TEMP folder contents (the folder used by the installer to extract files).
  2. Launch setup.exe again using Run as administrator and proceed with the installation.

See Also
