Symptom

Unable to export the data from app 'Display Actual Costing Result' even by specifying one material.
The document is too large to be downloaded. Please filter by material. 

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Access the app 'Display Actual Costing Result'.
  2. Attempt to export data using the custom export button.
  3. Observe the error message indicating that the document is too large to download.

Cause

Error "Specified Company Codes assigned to more than one Controlling Area" raised.
For legacy code for F6073, there're explicit logic to block filter value with pattern and the case company code assigned to more than one controlling area(COMPANY_CODES_FROM_SAME_CA allowed only).

Resolution

New app to replace this one will be delivered in next release CE2608.
So currently there's no plan to enhance this app, as a workaround, kindly consider to keep the Company Code field as empty.
