Symptom

Should SAP HANA Systems configured with the Fast Reboot Option have the /hana/tmpfs directory excluded from antivirus scans. 

Environment

SAP HANA, platform edition

Resolution

  • Implementing antivirus software on a system using SAP HANA Fast Restart Option (/hana/tmpfs) requires careful configuration to avoid critical performance degradation or system instability. The /hana/tmpfs file system, which stores critical database memory data in RAM for rapid restarting, is highly sensitive to I/O interference. 
  • It is strongly recommended to exclude the entire /hana/tmpfs directory structure from real-time and scheduled antivirus scans. Because this file system exists solely in RAM to accelerate database restarts, scanning these files provides no security benefit (as it is not persistent storage) and causes severe performance overhead.
