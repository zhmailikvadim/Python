Symptom

  • The "Stalled For Days" field under Manage Workflow Requests is not appearing in the Story Reports

Environment

SAP SuccessFactors HCM Core

Reproducing the Issue

  1. Login to instance
  2. Navigate to Report Center
  3. Create a story report
  4. Look for the Workflow Schema
  5. Notice that the Stalled For Days field is not showing

Cause

Expected Behavior

Resolution

The "Stalled for Days" under the Manage Workflow Request is considered an on the fly as its data is changing from time to time and the content on this field is not being saved in the database.

Hence, we recommend creating a calculated column that will show data in this field.

To have this field populate, you can use the technical guide for Employee Central - Advanced Reporting Standard Report Technical Guide for Open Workflow Requests: Employee Central Advanced Reporting- Technical Guide for Standard Reports in Employee Central Page 140

The guide above shows to to create a calculated column for Stalled for Days:

Stalled for Days: 

  • {TODAY – Last Step Approved Date} when ‘Last Step Approved Date <> 12/31/9999’
  • Otherwise: {TODAY – ‘Workflow Wf Request-Wf Request Created Date’}
