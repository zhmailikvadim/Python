Symptom

  • In optimized story, story designers are now able to set invisible filter members on table with New Table Build Experience enabled since SAC 2026 QRC1 update

Environment

  • SAP Analytics Cloud 2026.2

Resolution

  • Since SAC 2026 QRC1 update, on table widget filter, users are now able to set invisible filter members after switching this table to the New Table Build Experience in optimized story.
  • It is possible to specify the visibility of each selected member and their child members by selecting "Create Visibility Filter", accessible when hovering over the filter token of the dimension, measure, or account. 
  • In the Set Invisible Filter Members dialog, select the filter members required to hide from the table and confirm the selection. 
  • Invisible members do not appear in the table. However, unlike members that are filtered out of the table, their values are still aggregated to parent members and can be affected when a parent member is adjusted.
  • For more detailed information, please refer to help document Filters.
