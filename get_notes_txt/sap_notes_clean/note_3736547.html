Symptom

  • Error displayed: The choice ”” does not contain the specified code delimiters. in the question where code delimiter is set as []

Reproducing the Issue

  1. Use a questionnaire that contains a list of choice field with code-delimiter options (for example entries formatted like [1] …, [2] …, [A] …).
  2. Set the code de-limiter as []
  3. Observe that the list of choice values is empty when the questionnaire is exported
  4. When editing the question in template, observe the error: The choice””does not contain the specified code delimiters.

Cause

One or more list of choices were not saved and were blank even though they appeared in the edit question screen, so they did not contain the specified code delimiters, which triggered the error. 

Resolution

  1. Open the questionnaire in template and edit the question.
  2. Delete the list of choice values.
  3. Re-add the list of choice values, ensuring each choice includes the required code delimiters consistently (for example, enclosed in square brackets like [1], [2], [A], etc.).
  4. Save the changes to persist the list of choices.
  5. Export the questionnaire again and confirm the choice values are present and the error no longer appears.
  6. If the issue persists, copy and paste the entire list of choices again, save, and re-validate/export.
