Symptom

Whenever any changes (add, replace, and delete) done to attachments in Project, the change log has no details to be displayed under Project.

Environment

SAP Business ByDesign

Reproducing the Issue

  1. Go to Project Management work center.
  2. Go to Projects view.
  3. Edit a project.
  4. In Project Plan tab, select a project task.
  5. Go to Attachments subtab, add/replace/delete an attachment.
  6. Save the change.
  7. Go to Project Overview tab, Changes subtab.

There is a new change log generated but no details available.

Cause

This is a known functional gap.

Currently, there is no change details available for any changes done to attachments in project.

Resolution

Development team has taken up the functional gap in the development backlog and is planning to enhance it in future release.

If there is any new progress, it will be updated here.

Keywords

Project, Change Log, Attachment, Missing
