Symptom

Unable to store data in custom table column due to insufficient column length.

Environment

SAP CPQ

Resolution

To modify column length, sql command has to be executed on database. For this user has to create a case for CPQ support team with following details:

  • Target tenant url.
  • Custom table name, column name.
  • SQL command to update database.
  • Explicit approval to execute the sql command on database.

Keywords

custom table, column, length, insufficient, data, values, CPQ, SAP
