Symptom

The SAP Note addresses the issue with Network Graph for OA Relationships. The error happens with the Shop Floor Routing where some operations have the same operation number and there are existing Relationships between Operation Activities. The Network Graph for OA Relationships are launched from Display/Change mode of the Shop Floor Routing within msfr4 transaction.

Other Terms

PEO; MSFR4; Network Graph for Relationships; Relationships Graphic; Shop Floor Routing

Reason and Prerequisites

Program error.

The Shop Floor Routing should have operations with the same operation number and there are existing Relationships between Operation Activities.

Solution

Apply the attached correction instructions
