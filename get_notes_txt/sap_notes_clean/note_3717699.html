Symptom

Error message "More than one item with 100% usage probability found in alt. item grp &1" (Message No. CM_BOM_V_UI 032) is raised when saving effectivity changes for items in an Alternative Item Group (AIG) within a Versioned BOM.

Other Terms

Versioned BOM, Alternative Item Group, AIG, effectivity change error

Reason and Prerequisites

Program error.

Solution

Implement the attached correction instructions.
