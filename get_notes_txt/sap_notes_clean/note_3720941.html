Symptom

  • Recipe IDs  (SEARCH_TERM field) are missing in the standard database table /PLMB/RCP_RECIPE for migrated recipes from ECC to S/4HANA.
  • Recipe IDs are also missing in the Manage Recipes home page and Advanced Search..
  • Newly created recipe IDs are available in the database table.

Environment

  • SAP ERP
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Compare the database tables in ECC and S/4HANA; observe that migrated recipe IDs (SEARCH_TERM field) are missing in /PLMB/RCP_RECIPE table in both environments.
  2. Check the Manage Recipes home page and Advanced Search; observe that the migrated recipe IDs are missing.

Resolution

  1. Execute the report /PLMB/RCP_SEARCH_TERM_GENERATE in S/4HANA, setting the "Update existing search terms" flag during execution.
  2. The SEARCH_TERM field in /PLMB/RCP_RECIPE table should now be populated for the recipes.
