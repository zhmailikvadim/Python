Symptom

It is not possible to create the inbound delivery via IDoc, as the system issues the error VL 561 "Essential transfer parameters are missing in record: xxx". This is triggered by the warning ME 160 "Notified quantity exceeds quantity still to be notified & &", that the system does not deal with correctly.

However, the behavior can also occur with other warnings.

Other Terms

ME_CONFIRMATION_CHECK_QUANTITY, ME_CONFIRMATION_VIA_EDI, ME160

Reason and Prerequisites

The reason is the implementation of SAP Note 1954496 or DIMP SAP Note 2014241. These SAP Notes cause the system to save the warning and exit the function module ME_CONFIRMATION_CHECK_QUANTITY.

Solution

Implement the correction instructions.
