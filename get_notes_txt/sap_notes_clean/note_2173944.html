Symptom

Wrong translation

Other Terms

SE63, Access Management, Role Management, Role Maintenance, Role Mass Maintenance, Role Import

Reason and Prerequisites

Translation error from English to Japanese

Solution

Translation of objects using transaction SE63 (see further below) / Delivery with Support Package


Transport Object:

PgID Object Name
LIMU DTED GRAC_ROLE_STATUS

 
Package: GRAC_ROLE_MANAGEMENT
Software Component: GRCFND_A
Application Component: GRC-AC


English Text: Role Status
Wrong Target Text: Role Status
Correct Target Text: ロールステータス


The correction will be delivered with support package:

Planned delivery:

SP10 for GRCFND_A V1100 <SAPK-V1110INGRCFNDA>

Please refer to the SAP Support Package Stacks page in the SAP Service
Market Place: http://service.sap.com/sp-stacks


Alternatively the error can be corrected in translation object:

Object type (shortcut): DTEL
Object name: GRAC_ROLE_STATUS

(Please check if the texts fit to your Support Package level!)

Please use transaction SE63 to translate the text(s).

  1. Enter /oSE63 in the command field (transaction field) and press Enter
  2. Enter Object type (see above) in the command field and press Enter
  3. Enter Object name (see above) in the field Object Name
  4. If required change Source and/or Target language
  5. Press Edit
  6. Enter or correct the translation in the text field below the source text
  7. Choose Save to save your translation(s)
  8. Choose Back (F3) to leave the translation editor


Please refer to the following notes:

585116 Translation after installation of WebAS SP25
110910 Deleting the language load


Note:

Translations performed in a customer system may be overwritten in the following instances:

  • If the object happens to be part of a future support package
  • Upgrade to a new release
  • Import of new language CDs from Support Releases
