Symptom

If you provide a country code to the importing parameter 'I_BANKLAND' of the function module 'BCA_API_IBAN_CHECK_IBAN', the country code from the importing parameter 'I_IBAN' does not get checked. This has now been corrected.

Other Terms

Function Module BCA_API_IBAN_CHECK_IBAN

Solution

Import the relevant Support Package that is specified for your release. As a manual advance correction, you can implement the attached correction instructions using the transaction SNOTE. There are no manual activities.
