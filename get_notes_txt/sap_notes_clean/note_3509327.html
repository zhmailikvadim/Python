Symptom

  • When using the "Manage My Data" command in the SAP SuccessFactors Platform within Microsoft Teams, individual commands such as "view addresses" and "view cost center" are not recognized.
  • The system returns the message "I'm sorry. I don't recognize your command. Type 'help' to see what I can do."

Environment

SAP SuccessFactors HCM Suite

Reproducing the Issue

  1. Use the "Manage my data" command in Microsoft Teams.
  2. A list of quick actions is displayed.
  3. Clicking on any of the individual actions works as expected.
  4. However, when using the individual commands listed in Commands Supported for the SAP SuccessFactors App in Microsoft Teams, the system does not recognize these commands.

Cause

The issue arises because EC view commands are only available when locale is set to en_US on BizX. Please also note commands for Quick Actions continue to work regardless of the locale setting.

Resolution

The product management team is currently working to enhance this functionality to support additional English locales. You can view the complete list of supported commands on the Commands Supported for the SAP SuccessFactors App in Microsoft Teams.

The following commands are supported only when Employee Central Quick Actions are properly configured. These commands are expected to work seamlessly unless you customize the task names in the Admin Center. In such cases, users must use the customized task names, visible in the Manage My Data View list, as the commands to trigger the corresponding tasks.

The SAP SuccessFactors App for Microsoft Teams is designed to work with a specific set of commands, as detailed in the SAP Help Guide under "Commands Supported for the Application". If you use a command that is not on this list, you may encounter issues where the command does not function as expected. However, if Natural Language Processing (NLP) is enabled, the app can interpret and map your command to deliver the correct result.

For example, if you type "View My Data," the system will recognize and map this command to return the result for 'Manage My Data.
