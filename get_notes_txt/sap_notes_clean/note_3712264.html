Symptom

"Choose layout" window repeatedly appears in ALV grid after right-click

* Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

SAP S/4HANA 2023

Reproducing the Issue

  1. Execute transaction.
  2. Press F9.
  3. Click "Choose view" 
  4. Select one view
  5. Right-click anywhere in the ALV grid.
  6. "Choose layout" screen keeps popping up.

Cause

Resolution

Implement SAP Note - 3588505 - ALV: Wrong function is executed when interacting with the ALV

Keywords

ALV，choose layout, view, right-click, right click, pop up, window, screen.
