Symptom

  • When a compound structure is split into distinct performance obligations using the user interface and a subsequent change of the related sales order leads to the re-creation of that structure, the compound POB item is incorrectly passed to the Contract Management interface using interface parameter IT_POB_UPD instead of using interface parameter IT_POB_NEW.
  • When a Bill-of-Material is added to an existing sales order, the linked performance obligation associated with the Bill-of-Material item is not created in the contract.
  • When the material of a sales order is replaced by a different material that is no longer compound relevant, error message 'Compound POB x does not have a non-distinct, lower-level POB'. A linked POB was incorrectly passed to the Contract Management interface using interface parameter IT_POB_UPD instead of using interface parameter IT_POB_NEW. The compound performance obligation was not passed to the Contract Management interface using interface parameter IT_POB_DEL.

Other Terms

Revenue Accounting, POB

Reason and Prerequisites

This is a program error.

Solution

Implement the correction instructions.
