Symptom

  • Extended System ID is missing for a system in Transport Domain within Landscape Management Database (LMDB).
  • The issue persists even after following the recommendations described in SAP Note 2381495.
  • The Extended System ID temporarily appears after adjustments but disappears again after some time.
  • The job SAP_LMDB_NOTIFY_LDB_<ID> completes successfully without errors, but the Extended System ID remains missing.
  • Duplicate technical systems with the same System ID (SID) are identified in the System Landscape Directory (SLD).

Environment

 SAP Solution manager 7.2

Reproducing the Issue

    1. Open transaction LMDB in SAP Solution Manager.
    2. Navigate to Technical Systems.
    3. Search for the relevant System ID (SID).
    4. Open the technical system.
    5. Select Details → Properties → Transport Domain.

    Observe that the Extended System ID is missing for the system in the transport domain "DOMAIN_<XXX>".

    Verify in SLD

    1. Open SLD Home.
    2. Navigate to Technical Systems.
    3. Use the Search field and enter the relevant System ID (SID).Execute the search.
    4. Check whether multiple technical systems exist with the same SID (for example, "XXX" and "XXX000000001").

    Cause

    Duplicate technical systems with the same SID exist in the SLD, which prevents the LMDB from uniquely determining and assigning the extended system ID to the transport domain. even though the job "SAP_LMDB_NOTIFY_LDB_0000000001" running successfully without any errors.

    Resolution

    1. Follow the procedure described in SAP Note 1694004 Dealing with duplicate technical system names (SIDs)
    2. Review the System Landscape Directory (SLD) for duplicate technical systems with the same System ID (SID).
    3. Clean up obsolete or duplicate technical system entries in the SLD.
    4. Ensure that only one active and valid technical system with the relevant SID remains in the SLD/LMDB.
    5. Perform SLD–LMDB synchronization.
    6. Verify in LMDB → Transport Domain that the Extended System ID is correctly determined and assigned to the transport domain.
