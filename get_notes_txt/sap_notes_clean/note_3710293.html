Symptom

  • Error message: RES-20105
  • Error message Cannot read from or write to an unknown file. Specify the file name for file format.

Cause

The dataflow's reference to the file format object has been unintentionally broken.

Resolution

  1. Perform an operation to update the file format definition in CI-DS. For example, a column's data type could be modified, the file format's description could be updated, etc.
  2. The change of Step #1 could be undone afterwards; the only goal is that some operation was performed to update the format in the CI-DS database.
  3. Test the task again to confirm resolution.

Keywords
