Symptom

Correction of note 3669124 - minimal posting date is set wrong for contracts. You have contracts or security with archived data but the it is still processing even though you have zero balance on it (there are no CBD entries but PCDSTAT entries exist). This could affect on other regular processing contracts or securities and on certain point of time system will set wrong minimal posting date for contracts which are still processed and have zero balance.

Other Terms

 INIT(/BA1/CL_AL_BR_LBA_CALC_SERVICE)

Reason and Prerequisites

You run S/4HANA for financial products subledger release 3.0 with SP0-SP04.

Solution

Implement the attached Correction Instructions or import the specified Feature/Support Package.
