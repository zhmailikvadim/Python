Symptom

Following a system upgrade, AFS Filter button is not working in MD04.

Environment

  • SAP ERP 
  • IS-AFS (Apparel & Footwear Solution) 

Reproducing the Issue

  1. T Code : MD04
  2. Enter material and plant
  3. Try to select a grid value by AFS Filter or grid value and it’s empty

Cause

This functionality was not enhanced for AFS.

Resolution

AFS introduces grid-specific planning (e.g., size/color matrix), which is not supported in the standard MD04 transaction. Standard MD04 is designed for generic materials.

So, transaction /AFS/MD04 was introduced specifically for AFS as if you attempt to use AFS filters in MD04 will result in missing or inconsistent data display, this is expected behavior, not a system defect.
