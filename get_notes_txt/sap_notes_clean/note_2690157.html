Symptom

The BW Authorization role '#' value is not shown in authorization log.

Other Terms

Authorization log, '#' value, LRSEC_CHECKSF01, log error

Reason and Prerequisites

Program error

Solution

  • SAP BW 7.40
    Implement Support Package 18 for SAP BW 7.40 (SAPKW74018) into your BW system. The Support Package will be available as soon as SAP Note 2445563 with the short text "SAPBWNews 7.40 BW ABAP SP18", which describes this Support Package in more detail, is released for customers.
     
  • SAP BW 7.50
    Implement Support Package 9 for SAP BW 7.50 (SAPK-75009INSAPBW) into your BW system. The Support Package will be available as soon as SAP Note 2467627 with the short text "SAPBWNews 7.50 BW ABAP SP9", which describes this Support Package in more detail, is released for customers

  • SAP BW 7.51
    Implement Support Package 4 for SAP BW 7.51 (SAPK-75104INSAPBW) into your BW system. The Support Package will be available as soon as SAP Note 2467673 with the short text "SAPBWNews 7.51 BW ABAP SP4", which describes this Support Package in more detail, is released for customers. 
  • SAP BW 7.52
    Spielen Sie das Support Package 1 für SAP BW 7.52 (SAPK-75201INSAPBW) in Ihr BW-System ein. Das Support Package ist verfügbar, wenn Hinweis 2487153 mit Kurztext "SAPBWNews NW BW 7.52 ABAP SP1", der dieses SP detaillierter beschreibt, für Kunden freigegeben ist.

 

  • SAP BW/4HANA 1.0
    Implement Support Package 5 for SAP BW/4HANA 1.0 (SAPK-10005INDW4CORE) into your SAP BW/4HANA system. The Support Package will be available as soon as SAP Note 2458236 with the short text "SAPBWNews SAP BW/4HANA 1.0 SP05", which describes this Support Package in more detail, is released for customers. 

 

In urgent cases you can use the correction instructions.

Before you use the correction instructions, make sure that you check SAP Note 1668882 and SAP Note 2248091 for transaction SNOTE.

This SAP Note might already be available before the Support Package is released. In this case, however, the short text still contains the term "preliminary version".
