Symptom

When you have a sales document created by the standard APIs such as the Sales Order A2X, you have the error "500 Internal Server Error" and the error "MODIFY SCREEN: Invalid load ID" on SM21 transaction.

Other Terms

Reason and Prerequisites

This problem is caused by a program error. The SAP Note 3674001 is also needed but not listed as pre-requisite, implement it.

Solution

Implement the attached correction.
