Symptom

When trying to access SSCUI/Configuration Activity "Set Up Payment Medium Formats per Company Code" ID 102216, the following error messages are raised: 

  • No authorization to start the UI for transaction S_ER9_11000794
  • No Authorization for transaction S_ER9_11000794 of IMG Activity - VC_T042ZA_FORMAT

Environment

  • SAP Central Business Configuration (CBC)
  • SAP S/4HANA Cloud public edition

Reproducing the Issue

  1. In the CBC workspace, click the tab "Configuration Activities".
  2. Search for the Configuration Activity: 102216.
  3. Click on the entry.
  4. Click the button "Go to Activity".
  5. The error message is raised.

Cause

CAUSE 1 - Missing Business Catalog.

OR

CAUSE 2 - SSCUI is specialized for certain countries.

Resolution

CAUSE 1 - Missing Business Catalog.

  1. Ensure that the business catalog SAP_CA_BC_IC_LND_FIN_AP0_PC is assigned to the BPC_EXPERT business role (or an equivalent role used in your system).
  2. Additionally, to enable editing capabilities, make sure that Write Access is granted as Unrestricted for the BPC_EXPERT business role (or the equivalent role in use).

CAUSE 2 - The SSCUI is available only for the following countries: AE,AR,AT,AU,BE,BG,BM,BR,BS,BW,BZ,CA,CH,CL,CN,CO,CW,CY,CZ,DE,DK,DO,EG,ES,FI,FR,GB,GG,GH,GR,HK,HR,HU,ID,IE,IL,IN,IT,JE,JP,KE,KR,KW,KY,KZ,LB,LU,MU,MW,MX,MY,MZ,NA,NG,NL,NO,NZ,OM,PE,PH,PL,PT,QA,RO,RS,RU,SA,SE,SG,SI,SK,TH,TR,TW,TZ,UA,UG,US,VE,VG,VN,ZA,ZM,ZW.

The SSCUI will not be available for countries that are not listed above.

If neither Cause 1 nor Cause 2 resolves the issue, please create a support case under component FI-AP-AP-B-2CL and provide the affected CBC user, as described in SAP KBA 2838282.
