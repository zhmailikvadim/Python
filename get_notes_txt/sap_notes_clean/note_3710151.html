Symptom

You run the Calculo nómina y vacaciones  ( CO ) (HCOCALV0) report for an employee in the following scenario:

  • The employee has vacation days stored in Vacaciones (CO) (IT0834).
  • The employee has vacation absence in Absentismos (IT2001).
  • The employee receives a non-salary payment that is subject to law 1393 in Pago complementario (IT0015).

In this scenario, the IBC calculated during simulation differs from the IBC calculated during the real payroll run, causing the vacation absence to be incorrectly valuated in simulation.

Example

  • The employee has vacation days stored in Vacaciones (CO) (IT0834).
  • The employee has vacation absence in Absentismos (IT2001).
  • The employee receives a non-salary payment that is subject to law 1393 in Pago complementario (IT0015).

During vacation simulation:

  • The system calculates the vacation base using an IBC of 17,930,814 COP, because the non-salary payment is not considered in the calculation of the IBC.

In the real payroll run for the same period:

  • The system applies law 1393, and the non-salary payment above the legal amount expected is included, resulting in an IBC of 21,906,199 COP.

As a result, the vacation amount calculated in simulation is lower than the amount calculated in the real payroll run, even though both calculations should consider the same IBC.

Reason and Prerequisites

Reason

This issue occurs because the system skips the non-salary base incidence logic related to law 1393 during vacation simulation due to a fixed control that prevents its execution.

Prerequisites

The following Support Package levels are the minimum prerequisites:

Release Support Package
S4HCM 102 HR SP 00
S4HCM 101 HR SP 04
S4HCM 100 HR SP 17
SAP_HR 6.08 HR SP C8
SAP_HR 6.04 (EhP6 and EhP7) HR-CEE SP K9

For more information, access the SAP ERP Human Capital Management - Support Package Schedules.

When you install an SAP Note, some prerequisites might be required. The prerequisites are SAP Notes that have already been released and that change the same context of the current one. Then, even if these prerequisites are not necessarily related to the symptom of this SAP Note, you must implement them due to technical reasons.

Solution

This issue is resolved by allowing the non-salary base incidence calculation to be executed during vacation simulation based on a customizing setting, ensuring that the same IBC is used in both simulation and real payroll runs.
