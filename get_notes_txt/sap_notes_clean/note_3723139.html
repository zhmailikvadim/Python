Symptom

On the infotype Fiscal Data (0101), the description of the different values for the field Marital status (P0101-BVFAM) in the section Data on employee, does not have the same meaning, if you compare the English to the French or the Dutch version.

Important to know for every topic, which has to do with French or Dutch texts, when using the Belgian Payroll, is the fact that all objects are developed in language English, afterwards the translation to Dutch and French is done. Whenever a Dutch or a French term is not correct, there can be two reasons, either the translation is not correct, or the initial English term was not correct.

This overview shows, the different descriptions, currently used for the values of the field Marital status (P0101-BVFAM).

With this SAP Note 3723139, the descriptions used in English have been reviewed, documentation and values are using the same description. The translation is adapted accordingly.

Cohabitation légale

The documentation of field Marital status (P0101-BVFAM) has been reviewed, as the marital status is no longer reported in the BELCOTAX - Yearly Income Declaration (RPCBWAB0) via field 2.019.
