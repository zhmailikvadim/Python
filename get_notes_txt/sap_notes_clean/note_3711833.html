Symptom

The long text for the data element (CRM_ISX_MSG_DIST_STEP_PREREQ - Step Sequence for Prerequisite Document Distribution Step) was missing the information that the prerequisite steps are only possible within the same step category. This text was updated by accessing the documentation from the IMG node 'Define Settings for Document Distribution'.

Other Terms

Subscription Order Management, ODI Framework, Document Distribution, Prerequisite Step Sequence

Reason and Prerequisites

Missing information in IMG long text

Solution

The long text in the IMG node has been updated with the text highlighted in the attached pdf. The updated documentation will be available in SAP S/4HANA release 2023 SPS06 and 2025 FPS02.

The 'Define Settings for Document Distribution' IMG node is available in the Customizing path 'SAP Customizing Implementation Guide > Service > Transactions > Settings for Subscription Transactions > Document Distribution > Define Settings for Document Distribution'.
