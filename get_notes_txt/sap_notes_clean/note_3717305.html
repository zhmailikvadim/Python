Symptom

Missing cost component description and showing <not assigned> in the "Analysis" columns of Edit Costing Runs (CK40N) in English version. 

Environment

SAP S/4HANA Cloud Public Edition

Cause

Missing descriptions for Cost Component Groups in the English language version during the configuration "Define Cost Component Structure (SSCUI 102427)". 

Resolution

  1. Log in to the system using the English language settings.
  2. Open the configuration "Define Cost Component Structure (SSCUI 102427)".
  3. Edit the existing Cost Component Groups to add English names.
  4. Export and import the customizing changes across the system landscape.
  5. Open any existing Costing Run while using English language settings and verify that the columns in the layouts now display the actual names instead of <not assigned>.
