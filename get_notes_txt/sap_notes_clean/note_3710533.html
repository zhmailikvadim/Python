Symptom

  • There is requirement to custom form with MICR font and print from S4 HANA public cloud system

Environment

  • SAP S/4HANA Cloud system

Resolution

  1. Ensure MICR font is installed on the printer side.(MICR-enabled printer or install MICR DIMM/SIMM module, MICR toner, MICR toner sensor...)
  2. Upload MICR font to  Adobe Document Services by using "Adobe Document services configuration" App.
  3. Download standard form template from App "maintain form templates" and add the text field by using the "MICR Std" font from font list.
  4. Create print queue from App "maintain print queues" with PDF format.
  5. Assign the print queue to the printer which has installed the MICR font in CPM.
  6. Assign the uploaded ADS form template and the print queue to the corresponding application document.
  7. Print application document and test the print result.
