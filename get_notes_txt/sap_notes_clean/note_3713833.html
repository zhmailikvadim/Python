Symptom

In SLGD, it is possible to filter out the application logs to review the desired entries. Upon changing the layout to customize the display of logs and messages, the option to save the ALV layout  is unavailable.

Environment

SAP S/4HANA

Reproducing the Issue

  1. Execute SLGD to access the application log.
  2. Attempt to change and save the ALV Layout in the results.

Cause

Program error

Resolution

The issue is resolved by implementing the guidance provided in SAP Note 2857126 

See Also

2857126 - SLGD - Save and choose the layout of the ALV log with the detailed messages

Keywords

SLGD, ALV layout
