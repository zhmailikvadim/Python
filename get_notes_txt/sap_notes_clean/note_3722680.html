Symptom

You want to update the patch level of the installed 1.136 SAPUI5 release in an ABAP system.

Other Terms

SAPUI5 version 1.136.15, UI5 version 1.136.15, SAPUI5 version patch 1.136.15, UI5 version patch 1.136.15, SAPUI5 1.136.15, UI5 1.136.15

Reason and Prerequisites

Program Error

SAP_UI 816 SP00 or higher is installed

Solution

Implement the transport-based correction instruction (TCI) specified in this SAP note.
You can find more information about handling transport-based correction instructions in SAP Note 2187425.
