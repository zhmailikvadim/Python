Check regularly for new SAP Notes that might interest you.

For issues not covered in this SAP Note, create a customer case on the SAP Support Portal in the CA-GTF-CSC-EDO-AE component.

Other Terms

Reason and Prerequisites

Reason

You want to implement, configure, and use electronic document processing for United Arab Emirates.

Prerequisities: 

  • You have subscribed to the Peppol Exchange service of SAP Document and Reporting Compliance, cloud edition. Please get in touch with your SAP representative for further details.

  • You have implemented the Electronic Document Processing Framework in your system. Refer to the implementation guide attached to SAP Note 2134248 for instructions. You can also refer the Help Portal:

    • SAP S/4HANA

    • SAP ERP

  • You have implemented SAP Note 2783795, which contains the general Peppol functionality.

  • You have implemented SAP Notes 3654706 and 3652290 which contains United Arab Emirates Business Partner and Company Code Legal Identifiers.

Note on Area Menu and IMG Updates: 

Official SAP documentation cites menu paths on the SAP Easy Access screen and technical names of maintenance objects assigned to the SAP Implementation Guide (IMG; transaction SPRO). 

Any updates to the menu on the SAP Easy Access screen and to the IMG require a given support package to be installed. 

If you have not installed the corresponding support package, the updated menu path is not available in your system. 

To access the required objects, call the transaction code assigned to area menus or enter the technical name of the maintenance object in the maintenance transaction, such as SM30 for maintenance views. The object names that you need to use are covered in the documentation.

Note about translations: Translations are only delivered with support packages.
