Symptom

The changes that are made with this interface SAP Note are the prerequisite for implementing additional SAP Notes.

Remark: This SAP Note does not contain the complete scope for the deletion of personal data in Advance return for tax on sales/purchases (Portugal) reporting. This functionality is only implemented with a Support Package (ERP 617, Support Package 8).

Other Terms

This is a technical SAP Note, an interface SAP Note,RFUVPT00

Reason and Prerequisites

This interface SAP Note is a preparatory SAP Note that does not interrupt operation. Implement this SAP Note if it is a precondition for subsequent corrections.

Solution

You should only implement this SAP Note if it is a precondition for other SAP Notes.

Execute the manual pre-implementation activities specified below, and then use transaction SNOTE to implement the automatic corrections.
