Symptom

After performing a system - refresh, backups are no longer visible in SAP HANA Studio except for those created after the refresh.

Environment

SAP HANA, platform edition

Cause

The backup catalog in SAP HANA Studio is reinitialized or reset during a system refresh, system copy, or restore. This process prevents the target system from accidentally overwriting or deleting the source system's production backups in shared storage. As a result, backups created prior to the refresh are no longer registered in the active backup catalog of the refreshed system.

Resolution

Please note that the behavior of SAP HANA Studio displaying only backups created after a system refresh, is expected and confirmed by SAP.
