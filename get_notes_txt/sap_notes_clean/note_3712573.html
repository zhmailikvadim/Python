Environment

SAP IQ

Cause

The issue has been logged under SAP CR 831756. Its description reads, "IQ node crashing on s_packedRec::AcceptDataFieldFromHDVal ".

Resolution

There is currently no target SAP IQ Service Pack scheduled for this issue (SAP CR 831756). It has been assigned to our Product Group. Please mark this KBA as a Favorite to be automatically notified when the CR status is changed.

Workaround :
   SET TEMPORARY OPTION Dml_Options14 = 8;

Apply the workaround by setting "SET TEMPORARY OPTION Dml_Options14 = 8" at the beginning of the stored procedure or query involved.
