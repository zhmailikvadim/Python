Symptom

From 2305 release(HANA legacy), HANA customers can download EXT custom code for PROCEDURE/LIBRARY/FUNCTION/TRIGGER via Touchless Deployment API.

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

SAP SuccessFactors Sales Performance Management

    • Incentive Management on Legacy HANA

Resolution

This is to download custom code from EXT schema(HANA only) for below object types :

  • PROCEDURE
  • LIBRARY
  • FUNCTION
  • TRIGGER 

1. API to fetch Procedure via API tool: 

https://xxxx.callidusondemand.com/TrueComp-SaaS/services/rest/touchlessdeployment/downloadCustomCode?OBJECTTYPE=PROCEDURE&OBJECTNAME=procedurename

2.  API to fetch Procedure via command Line TDClient script:

TDClient.cmd -o codeDownload -type PROCEDURE -object procedurename
