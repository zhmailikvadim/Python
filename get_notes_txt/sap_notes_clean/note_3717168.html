Symptom

  • Connection to Microsoft Fabric using the MssqlLogReaderAdapter (or MSSQL connection type in Datasphere) works, and it is possible to replicate data via snapshots.
  • However, trying to enable real-time replication fails with the following error message in the DP Agent logs:
    • DPFramework | WorkerThread.processRequest [] - com.sap.hana.dp.adapter.sdk.AdapterException: [Driver error] Failed to add the first subscription. Error: java.sql.SQLException: DROP TRIGGER is not supported.Failed to drop triggers of

Environment

SAP HANA Smart Data Integration

Reproducing the Issue

  1. Establish a connection to a Microsoft Fabric source using the MssqlLogReaderAdapter.
  2. Attempt to perform real-time replication.
  3. Observe the error message indicating failure.

Cause

Real-time replication is not supported when connecting to Microsoft Fabric because the MssqlLogReaderAdapter requires privileges such as CREATE TRIGGER and DROP TRIGGER to exist in the source database. MS Fabric does not support standard DDL/DML trigger.

Resolution

It is only possible to replicate data via snapshots when connecting to Microsoft Fabric via MssqlLogReaderAdapter.
