Symptom

If you run report /UI5/UPD_ODATA_METADATA_CACHE, there are next error messages.

     Service has errors: /sap/opu/odata/sap/FAR_CUSTOMER_LINE_ITEMS
     An exception was raised

     Service has errors: /sap/opu/odata/sap/MD_PRODUCT_OP_SRV/
     An exception was raised

Environment

Fiori Launchpad

Reproducing the Issue

Run report /UI5/UPD_ODATA_METADATA_CACHE

Cause

These two service which raise exception has other OData services dependencies so they will not be accounted for Context token calculation.

     1. FAR_CUSTOMER_LINE_ITEMS has dependency with FARP_VALUE_HELPS_SRV.
     2. MD_PRODUCT_OP_SRV has dependency with LO_VCHCLF.

If an OData service is extended by another OData service and an entry is present  table /IWBEP/L_MGW_MDE,  this exception is valid and can be ignored.

If the OData service has  system alias other than LOCAL , you will have to check for the table in the backend system(System Alias-Double click and do a remote logon).

Resolution

You can ignore these exceptions as they are valid and expected.
