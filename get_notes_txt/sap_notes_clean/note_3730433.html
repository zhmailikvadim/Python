Symptom

Role /UIMC/ALL is a template role that provides all authorizations for centralized configuration for SAP GRC UIDP Masking, this role will be delivered in SCV SAPGRC 100 SP01.

Other Terms

SAPGRC 100, SAP GRC UIDP Masking for S/4HANA, Role for centralized masking configuration

Reason and Prerequisites

Role is not delivered in SAPGRC 100 SP00, this note is applicable only for SAPGRC 100 SP00

Solution

This note does not include automatic correction instructions. The steps to create the required role are outlined below:

  1. Create the role /UIMC/ALL using transaction PFCG.

  2. Maintain the following details in role /UIMC/ALL:

    • Description: Master Role with all authorizations for GRC UI data protection masking.

    • Authorization Objects:

      • Add authorization object /UIMC/CONF and set the ACTVT field to Full authorization.
      • Add authorization object S_TABU_DIS, set DICBERCLS to /UIMC/UIM, and set ACTVT to Full authorization.

    • Menu:
      Add the following transactions to the role:
      /UIMC/MIGRATION, /UIMC/POLICY_BUILDER, /UIMC/SYNC, and /UIMC/UPD_VALUEHELPS.

  3. Save and generate the role.
