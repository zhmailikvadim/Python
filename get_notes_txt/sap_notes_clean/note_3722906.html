Symptom

 In the process "Reversal and Recreation of Future Parameterization Instance (MOS View)" (DE_B_IM_MANG_VALID_MI_PAR), the system incorrectly triggered the process DE_B_IM_PARAM_CREATE in step 0750 "Check Whether Parameterization Needs to Be Created or Changed."

Other Terms

 US4G, IM4G, DE_B_DISPR_IM_PROCESS, DE_B_IM_MANG_VALID_MI_PAR, DE_B_IM_PARAM_CREATE, DE_B_IM_PARAM_CHANGE, parameterization instance

Reason and Prerequisites

 The error occurred because of a programming error.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the system triggers DE_B_IM_PARAM_CHANGE for existing installations. The CREATE process is only run for an initial installation. You can verify the fix by checking that DE_B_IM_PARAM_CHANGE is triggered for existing installations.
