Symptom

I'm executing the program ARBCIG_CONTRACT_EXPORT in my SAP ERP and I would like to understand why it is exporting line items flagged for deletion.

Environment

Managed Gateway for Spend&Network (ISMG)

Cause

The program ARBCIG_CONTRACT_EXPORT uses the function ARBCIG_CONTRACT_GETDETAIL to get the item details. The ISMG function ARBCIG_CONTRACT_GETDETAIL calls the core Material's Management (MM) BAPI BAPI_CONTRACT_GETDETAIL to get the item details.
As per current design, the MM BAPI BAPI_CONTRACT_GETDETAIL is designed to return all items, regardless of their deletion status. Deleted items are marked with ITEM-DELETE_IND as 'L'.

Resolution

Implement a customization to workaround. A possible solution is to implement the BADI ARBCIG_CONTRACTDET Method POSTPROCESSING to customize and ensure the correct item is sent in the Table: ITEM[].
