Symptom

When attempting to add a Filtered List View to the Home Page, you may notice that the Save button remains greyed out even if all mandatory fields have been selected.

Environment

  • SAP Sales Cloud Version 2
  • SAP Service Cloud Version 2

Reproducing the Issue

  1. Go to the Home Page.
  2. Enter Adaptation Mode and select "Main Layout".
  3. In the home page, click on "Create Card".
  4. Select "Application".
  5. Then, select "Filtered List".
  6. Choose "Guided Selling" and click "Add".
  7. Using the pen icon, select:
    • Filter: All Opportunities;
  8. Select one Primary Field.
  9. Select one Secondary Field.
  10. Observe that the "Save" button is not clickable.

Cause

A minimum of two Secondary Fields must be selected for the 'Save' button to be enabled. The maximum limit is also set at two fields.

Resolution

To resolve this issue, ensure that exactly two fields are selected for the Secondary Fields.
