Symptom

Incorrect consumption of prepayment request occurs when modifying the prepayment request (PPR) percentage in settlement.

Reproduction Steps
1.Create and approve/save a trading contract price using transaction WB21.
2.Add one or more prepayment request (PPR) lines to the contract.
3.Execute transaction PPMGCO_PMT_REQPOST to post the prepayment request.
4.execute the payment run using transaction F110.
5.Create one or more LDCs using transaction /ACCGO/LDC_WC.
6.Create a final settlement using transaction /ACCGO/STL_WC.
7.Open the final settlement document in change mode.
8.Modify the Prepayment Request (PPR) percentage.
9.Observe that the prepayment consumption amount is not recalculated correctly.

Other Terms

 PPMG; Prepayment Agreement; PPR; Prepayment request, ACM, /ACCGO/STL_WC

Reason and Prerequisites

Program error.

Solution

Import the relevant support pack or apply the correction instruction(s).
