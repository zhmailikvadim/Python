Symptom

You want to make inspection for delivery process and related inspection types have been activated in the material master data, for example, inspection type 10 for outbound delivery and inspection type 05 for lean return delivery.
However, after the delivery is created, no inspection lot is generated.

Environment

SAP S/4HANA Cloud Public Edition

Cause

Inspection lot origin is not assigned to the corresponding delivery type in configuration activity 105135 - Assign Inspection Lot Origin to Delivery Type.

Resolution

Assign the proper inspection lot origin to the inspection type in the configuration activity. For example, assign inspection lot origin 05 to delivery type LR for lean return delivery.

Keywords

inspection type 05, lean return delivery, inspection lot, inspection origin, Scope Item BKP, QA32, movement type 653,  Assign Inspection Lot Origin to Delivery Type
