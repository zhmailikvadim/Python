Symptom

This article describes functionality introduced in the May 2026 SAP Fieldglass release

Job postings created for another user display differently than expected on the homepage or Job Posting Dashboard

You may notice one or more of the following:

  • Job postings created for someone else do not appear on the homepage
  • The homepage and the Job Posting Dashboard show different job postings
  • Filters that were previously selected are no longer applied
  • A person’s name is displayed next to a job posting and it is unclear why
  • The Job Posting Dashboard layout looks the same after recent changes

Reproducing the Issue

This behavior may be observed when:

  1. Switching between My Job Postings and Job Postings Created on Behalf of Others on the homepage
  2. Opening the Job Posting Dashboard directly instead of using the homepage link
  3. Signing out and later signing back in
  4. Reviewing job postings that were created for another user
  5. Comparing the homepage view and the Job Posting Dashboard layout

Cause

SAP Fieldglass introduced enhancements that improve visibility into job postings created on behalf of others

As part of this update:

  1. The homepage allows switching between personal job postings and postings created for others
  2. Filters behave differently based on how the Job Posting Dashboard is accessed
  3. Selected filters remain active only during the current session
  4. Additional information is shown to identify who a job posting was created for
  5. The Job Posting Dashboard layout remains unchanged

Resolution

Follow these steps to view job postings created on behalf of others:

  1. On the homepage, use the job postings card to switch between:
    • My Job Postings
    • Job Postings Created on Behalf of Others
  2. To keep the same view when opening the Job Posting Dashboard, select Go to Job Posting Dashboard from the homepage card
  3. If filters are no longer applied, reselect them after signing in again
  4. When a person’s name appears next to a job posting, it identifies who the posting was created for
  5. The Job Posting Dashboard layout remains the same, with additional filtering options available to refine results
