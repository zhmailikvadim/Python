Symptom

The formation is created with type Integration with SAP Business Data Cloud in SAP4ME.
SAP Help Portal Documentation: Creating SAP Business Data Cloud Formations

Errors occurred while synchronizing:

Synchronization between xxxx and xxxx  has failed.
– xxxx System xxxx  returned a technical error for assignment with ID xxxx :
"Operation: assign has failed. Receiver Tenant State: INITIAL, Formation Tenant State: READY, Receiver Tenant Application Namespace: xxxx. Error: handleSecondSynchronizationRequest rejected - assignedTenant.deploymentRegion (cf-eu10) is different from receiverTenant.deploymentRegion (cf-eu20)"

*deploymentRegion might be differ in your error message

Environment

SAP Business Data Cloud

Reproducing the Issue

  • SAP Help Portal Documentation: Creating SAP Business Data Cloud Formations

Cause

The Formation members are located on different datacenter.

Resolution

Make sure that the SAP Business Data Cloud, SAP Datasphere and SAP Analytics Cloud are based on the same region.
