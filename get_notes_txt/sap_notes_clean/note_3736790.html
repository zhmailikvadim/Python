Symptom

You've mistakenly assigned an employee to the wrong personnel area. You need to correct this assignment. This KBA provides guidance on how to correct the personnel area for the employee. "Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Environment

  • Human Capital Management (HCM)
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for HANA
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition
  • SAP SuccessFactors Employee Central Payroll

Resolution

 For example, an employee is incorrectly assigned to the personnel area JP02 through the action type '02 Organizational Reassignment'. The correct personnel area should be JP01. Follow these steps to make the correction:

  1. Go to t-code PA30 and input the pernr.
  2. Copy the latest IT0000 record - action type '02 Organizational reassignment'.
  3. Modify the personnel area from JP02 to JP01.
  4. Click the 'Save' button.
  5. When prompted by the pop-up window, click 'Yes' to confirm the changes. 
  6. Save the new IT0001 with correct personnel area.
