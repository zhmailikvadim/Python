Symptom

You want to post Peppol invoice without Downpayment request and you get a Dump while posting.

Other Terms

Peppol, Belgium, Convergent Invoicing, CI, UBL, eDocument, eDOC, Cockpit, eDocument Cockpit, edoc_cockpit, e-invoice, e-invoicing, electronic invoicing, AIF

Reason and Prerequisites

This note is valid only for Peppol.

Solution

You can create a Peppol eDocument without Downpayment request.

As a general rule, SAP recommends that you install a solution by applying a Support Package. However, if you need to install a solution earlier, use the SAP Note Assistant to implement the correction instruction. You can find more information about the SAP Note Assistant in SAP Support Portal, under SAP Note Assistant.
