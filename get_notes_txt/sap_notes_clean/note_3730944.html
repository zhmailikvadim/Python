Symptom

  • Issue when implementing note 3701228 – Income Tax Bill 2025 related to "V_T7INI10_EXTRACT".
  • Prerequisite note 3727345 was successfully implemented without errors.
  • Error message: "Issue with Implementation of SAP Note 3701228 - Income Tax Bill 2025 'V_T7INI10_EXTRACT' is unknown."

Environment

  • SAP SuccessFactors Employee Central Payroll
  • Human Capital Management (HCM) (Payroll India - Public Sector)
  • SAP ERP 6.0
  • SAP ERP Core Component 6.0
  • SAP enhancement package 4 for SAP ERP 6.0
  • SAP enhancement package 4 for SAP ERP 6.0 on SAP enhancement package for SAP NetWeaver 7.0
  • India

Reproducing the Issue

  1. Implement prerequisite note 3727345 successfully.
  2. Start implementing note 3701228 – Income Tax Bill 2025.
  3. Observe that "V_T7INI10_EXTRACT" is reported as unknown and the error message appears.

Cause

Resolution

  1. Review the implementation instructions of note 3701228 and complete step 3 to activate the relevant programs.
  2. Execute Transaction SE38.
  3. Enter the report name as NOTE_3701228.
  4. Execute the same, You will find screenshot as below: 
  5. Perform the Step 3 also.
