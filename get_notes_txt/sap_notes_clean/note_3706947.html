Symptom

  • Task completes successfully, but the ABAP program is not uploaded automatically to the SAP system.
  • Generated ABAP program is not found in the SAP system after task execution.

Environment

SAP Cloud Integration for data services

Reproducing the Issue

  1. In the default configuration of sap application datastore, ensure that the ABAP execution option is set to "Generate & Execute".
  2. In the dataflow editor, select "Generate and view ABAP report" (start icon) from the icons listed above the data flow editor.
  3. Select the option "Load ABAP program to SAP Application server".
  4. Select "OK".
  5. Run the task successfully.
  6. In SAP system, run Tcode SE38 and verify that the generated ABAP program is not present.

Cause

The user configured in the SAP application datastore lacks of permissions

Resolution

Assign SAP ALL profile to the user
