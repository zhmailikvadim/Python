Symptom

The application tile for the "Manage Outgoing Documents" application is not accessible in the fiori launchpad. 

Environment

S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Open fiori launchpad.
  2. Search for the application "Manage Outgoing Documents".

Resolution

  • The "Manage Outgoing Documents" application is currently only available in the India GSTR1 Report.
  • Refer to Manage Outgoing Documents | SAP Help Portal for more information.

Keywords
