Symptom

Agent determination for the manual KRI entry WF starts already on the parent object of the KRI instance. This is fine for the standard scenarios. However, in case the customer enhance the solution and enable the role assignment on the KRI instance itself, the agent cannot be determined by the standard WF properly.

Reason and Prerequisites

Program error

Solution

Implement corresponding correction instructions.

This note doesn’t change the standard WF behavior, but prepares the standard solution for easier customer enhancement.
