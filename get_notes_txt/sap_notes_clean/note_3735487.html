Symptom

Fixed assets with real estate key assigned cannot be part of organizational changes. If you enter such an asset into the user interface, the system will send error FINOC 860.

Nevertheless, the system tries to add such assets automatically. This SAP Note will remove this internal inconsistency.

Other Terms

Error message FINOC 860 in simulation, activation or processing run.

FINOC 860 is justified if you enter a fixed asset with real estate key in the user interface. This SAP Note deals with not adding such assets by software (which is an internal inconsistency).

Reason and Prerequisites

Program error.

Solution

Apply the attached correction instruction.
