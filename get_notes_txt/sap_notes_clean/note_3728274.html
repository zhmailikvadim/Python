Symptom

When using the Display Measuring Point Web Dynpro Application and selecting the Advanced Search Help, the Functional Location and Equipment columns are empty, even though a Functional Location or an Equipment is defined.

Other Terms

W0031, search help, IMPM_ES_ADVANCED_E, ESH, TPLNR, TPLNR_EXT, IMPM_ES_ADVANCED, EQUNR, EQUNR_EXT

Reason and Prerequisites

Incorrect mapping

Solution

Implement the attached correction instruction or the corresponding Support Package.
