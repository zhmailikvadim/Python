Symptom

You are using the Master Data Quality apps and navigate to the BRFplus Workbench to edit objects and implement your validation rules or derivation scenarios. You notice that the BRFplus Workbench is in 'Simplified BRF+ Workbench' mode. In this mode, you are unable to use certain objects, which are enabled in the full version of the BRFplus Workbench.

Other Terms

Master Data Quality, Business Rule Framework, Simplified Workbench, Expression not supported

Solution

The SAP-delivered Fiori catalogs, as part of the SAP_*steward pfcg roles, contained navigation targets for both the Simplified and full BRFplus Workbench, leading to issues when using these roles as is. Implementing this note or applying the corresponding support package will adjust the SAP-delivered content to once again navigate all customers to the full version of the BRFplus Workbench.
