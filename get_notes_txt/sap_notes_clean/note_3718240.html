Symptom

 Bullet Chart fails when colour palette is set

Other Terms

Lumira, Bullet Chart

Reason and Prerequisites

  1. Open Lumira Designer document using the Analysis template
  2. Switch to a chart; a column chart is displayed by default.
  3. Change chart types, observe that there are no issues
  4. Switch multiple times between column, bar, pie, bullet, waterfall, and line charts — everything works as expected.
  5. Now set the colour palette
    1. Choose any of the provided palettes or create a custom one — the issue consistently occurs.
  6. Change the chart type to a Bullet chart and observe the problem: although the Bullet icon is selected, the chart component does not re-render and fails to load properly.

Solution

Fix will be available in Lumira 2025
