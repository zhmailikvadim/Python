Symptom

Users are unable to locate previously generated SAP HANA Explain Plans and require a way to retrieve them from the SQL Plan Cache.

Environment

SAP HANA 2.0

Resolution

  1. Search for the SQL statement in the M_SQL_PLAN_CACHE view

Use a query such as:
SELECT  
STATEMENT_STRING, 
STATEMENT_HASH,
PLAN_ID,
LAST_EXECUTION_TIMESTAMP FROM M_SQL_PLAN_CACHE
WHERE STATEMENT_STRING LIKE '%your_statement_text_here%'

     2. Identify the corresponding PLAN_ID for the SQL statement you want to analyze.

     3. Display the Explain Plan that was used at execution time by running:

         EXPLAIN PLAN FOR SQL PLAN CACHE ENTRY <PLAN_ID>
