With release SAP S/4HANA 2025 Service the maintenance view 'CRMS4V_PM_OT_MAP' cannot be used anymore.

Environment

SAP S/4HANA 2025

Resolution

With SAP S/4HANA 2023 release a new maintenance views have been delivered

see: https://me.sap.com/notes/0003433802

Starting with S/4HANA 2025

The old view cannot be used anymore.

Just  to revamp this customizing. The new views are more intuitive and user friendly. The path for customizing activity within the SAP Reference IMG has remained the same between the SAP S/4HANA 2022 and 2023 releases, thus providing users with a seamless transition between the old and new views.

Only maintenance view in SAP S/4HANA 2025 and beyond

  • CRMS4V_MS_PM_MAP
  • CRMS4V_PM_MS_MAP

For more information - see the SPRO documentation under:
