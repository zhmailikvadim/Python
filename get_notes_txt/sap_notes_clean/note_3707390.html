Symptom

When attempting to enable the Data Provisioning Server in HANA Cloud Central, you receive the following error message:

"You do not have enough memory for the additional capabilities you have selected. The selected capabilities require the SAP HANA database to have at least 32GB."

 "Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Environment

SAP HANA Cloud

Reproducing the Issue

  1. Attempt to enable the Data Provisioning Server in HANA Cloud Central.
  2. Observe the error message indicating insufficient memory.

Cause

The memory allocation for this HANA instance is less than 32GB.

Resolution

Increase the memory allocation to at least 32 GB.
