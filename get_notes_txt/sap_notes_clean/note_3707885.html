Symptom

Files show without a file size value in iContent Management tool.

Environment

SAP SuccessFactors Learning

Reproducing the Issue

  1. Go to Learning Administration 
  2. iContent Management Tool
  3. Check the column size file - Notice that some of contents do not have any file size (value is empty). 

Cause

Product defect.

Resolution

According to Engineering team analysis, there are updates that are not pushed to some DB tables that impact the update of the content displayed in the tool. The fix will ensure that the updates are performed accurately and multiple times a day to prevent old contents to display.

The fix will take place the upcoming release b2605. Please find the release dates below:

  • Preview: April 13
  • Production: May 15 - 16

You may find more release details in this Customer Community link.
