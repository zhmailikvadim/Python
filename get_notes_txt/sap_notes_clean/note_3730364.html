Symptom

This SAP Note provides the technical prerequisites and release validity dates, from which the new Rulebooks and Standards Releases are valid. These prerequisites are needed for the following Rulebook changes:

  • SEPA 2026 (EBA and BuBa)
  • SWIFT MX CBPR+ Standards Release 2026
  • TARGET2 Rulebook 2026
  • TIPS (TARGET Instant Payments) Rulebook 2026

Other Terms

Rulebook 2026, SR2026, Standards Release 2026, R26, R2026, /PE1/V_TRANS_PER, release validity

Reason and Prerequisites

Legal change.

Solution

Implement the correction instruction attached to this SAP Note. Alternatively, you can import the referenced Support Package.
