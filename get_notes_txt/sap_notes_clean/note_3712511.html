Symptom

There is a business requirement to prevent users from taking screenshots of SuccessFactors when accessing it via the web.

Environment

SAP SuccessFactors HCM Suite

Resolution

Screenshot capture is not a feature controlled by applications (such as web browsers), but rather by the operating system. For this reason, this type of restriction can be configured on mobile apps, but not on SuccessFactors Web.

To verify whether this is possible, the question should be directed to the browser vendors, such as Chrome, Mozilla Firefox, Safari, and others.
