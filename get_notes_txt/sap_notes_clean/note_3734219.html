Symptom

In the Fiori application F0673A Approve Bank Payments, column Payee shows misleading information. In case that alternative payee is set for business partner, Payee column displays mixed information from supplier and alternative payee.

Other Terms

F0673A, payee, supplier, business partner, alternative payee

Reason and Prerequisites

Program error.

Solution

This SAP Note adjusts how Payee/Supplier is displayed in the Fiori application F0673A Approve Bank Payments:

  • Column "Payee" has been renamed to "Supplier" - naming is now unified with F2388 Monitor Payments, BNK_MONI and BNK_APP,
  • Description in parentheses for "Supplier" column displays Supplier Name (REGUH-NAME1) instead of Payee Name (REGUH-ZNME1),
  • In order to display Alternative Payee, user can check columns "Payee Name" or "Payment Recipient",
  • Information in contact popover has been limited to address only. Phone and e-mail is no longer available due to duplicates and performance reasons - for more information see SAP Notes 3481084 and 3334449.

Implement the attached Correction Instruction or the mentioned Support Package.

IMPORTANT: This is front-end part of the correction, that requires back-end part. If you implement this SAP Note, make sure that SAP Note no. 3734220 is also implemented.
