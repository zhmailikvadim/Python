Symptom

This SAP Note describes the implementation of a new feature.

In an order-specific routing, for the components in the Component List Workspace the lead time Offset, the lead time offset for operation and its unit of measure shall be entered and transferred to the production order when the order-specific routing is relased.

Other Terms

MSFR7, NLFZT, NLFZV, NLFMV, MPE_RTG_UI screen 0611

Reason and Prerequisites

This SAP Note does not describe a correction, but the implementation of a missing feature.

Solution

Change the DDIC structures and the database table MPE_RTGV_ORD_CMP and extend the UI screen as described in the correction instructions.
