Symptom

A customer can make a reservation before purchasing a ticket. After the money have been paid the reservation need to be converted.

When doing this an error message appears in the errorlog 

ERROR: Error in document: ZVMTS 000000000 XYZXYZXYZX | General ledger account &V1& is not defined in company code xyz

Environment

SAP Event Ticketing, Cloud version

Reproducing the Issue

logon to SAP Event Ticketing
select Company
go to System
Export Interfaces
Export
select the export interface for the reservation
Export Status: Export error
Click: Find

Cause

The General Ledger account &V1& is not maintained in the company code determined by the system. As a result, the account cannot be used for the selected item.

Resolution

Please contact the department responsible for maintaining the General Ledger accounts.
