Symptom

You have applied SAP Note 2128151 - W-2 PR / W-2c PR - Box 16 code E, which delivers legal changes that affect Forms W-2 PR / W-2c PR. Those legal changes are valid only from tax year 2014 onwards. However, after applying SAP Note 2128151, when you run Tax Reporter (PU19) for prior years (such as 2013), the results are affected.

Other Terms

T145, EXSE, TEXA, TEXC, HR_F_MMREF_2_PR.

Reason and Prerequisites

Reason

Correction of legal function.

Prerequisites

You have applied the following SAP Notes:

  • 2063247 - Year End 2014 Phase I for U.S. Tax Reporter
  • 2128151 - W-2 PR / W-2c PR - Box 16 code E

Solution

As general rule, SAP recommends that you install a solution by applying a Support Package. However, if you need to install a solution earlier, use the Note Assistant (SNOTE) to implement the correction instructions

You can find more information about the Note Assistant in SAP Service Marketplace, under service.sap.com/note-assistant.
