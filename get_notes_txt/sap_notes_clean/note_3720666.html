Symptom

In the process "Integration of Meter Operator Contracts" (DE_B_MOSB_INTG_PROC), when both the supply scenario for MALO POD and MEMA relationship end, and the end date plus one day is passed as the process date, the integration process discards the timeslice or reports an error instead of terminating it.
This issue is relevant for the market role: MOS.

Other Terms

MOSB, Meter Operation Service Billing, MALO POD, MEMA relationship, timeslice, termination scenario, supply scenario

Reason and Prerequisites

The error occurred because of a program error.

Solution

Implement the corrections of this SAP Note or import the relevant support package.
After implementing this SAP Note, the integration process correctly recognizes MOS termination scenarios and terminates the target timeslice.
You can verify the fix by running the integration process with a process date set to the end date plus one day and checking that the system terminates the target timeslice correctly.
