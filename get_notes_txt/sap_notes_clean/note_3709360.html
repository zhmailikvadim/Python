Symptom

Missing field group data in table T082G while in configuration it is correctly maintained.

Environment

  • Financial Accounting (FI)
  • Asset Accounting (FI-AA)
  • SAP R/3
  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA Finance
  • SAP S/4HANA 

Cause

Unintended system behavior.

Resolution

Perform the following steps to resolve the issue:

  1. Start transaction SM34 and open view cluster VC_T082G_10 (IMG activity: 'Define Screen Layout for Asset Master Data') in maintenance mode.
  2. The available layouts are displayed. Navigate to the issued screen layout. Do not make any changes.
  3. Return to the previous screen (F3) and select SAVE button.

In the background the view logic automatically updates the records in T082G.

See Also

SAP Note 118233 - Table inconsistencies for screen layout rules

Keywords

T082G, field group, V_T082G_10, VC_T082G_10, screen layout, OAVM, table inconsistency
