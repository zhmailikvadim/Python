Symptom

The Planned Order API fails to determine requirement segment and stock segment of new component. The API shows the error message SGT_01(008) for the component.

Other Terms

SAP Fashion Management, FMS, Fashion Management Solution, Segmentation, Planned Order, Requirement Segment Value, Stock Segment Value, SGT_01(008), Enter requirement segment for, MD_PLDORD_CHANGE_ALL_BAPI, MD_NEW_COMP_CHECK_BAPI

Reason and Prerequisites

Reason -

This problem is caused by a program error.

Solution

Implement the correction instruction provided with this SAP Note.
