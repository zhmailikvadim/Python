Symptom

I would like to update "Include in MSP Fee Calculation" flag in a Rate Component but the flag is greyed out.

Reproducing the Issue

  1. Sign in as Administrator user
  2. Navigate to Admin Menu > Rate Component
  3. Open the affected Rate Component > Edit
  4. "Include in MSP Fee Calculation" cannot be edited.

or

  1. Sign in as Configuration Manager user
  2. Click on Rate Component
  3. Open the affected Rate Component > Edit
  4. "Include in MSP Fee Calculation" cannot be edited.

Cause

The Rate Component flag "Include in MSP Fee Calculation" cannot be edited via the UI by either the Administrator or the Configuration Manager once the Rate Component has been created

Resolution

  1. Create a new Rate Component with the correct setting for "Include in MSP Fee Calculation".
  2. Associate this new Rate Component with the Rate Component Group via Configuration Manager user.

Should you need the new Rate Component to be included in an existing Work Order, you must revise the Work Order, and then the new Rate Component will be associated. Please note that Work Orders will be affected only after they have been revised.
