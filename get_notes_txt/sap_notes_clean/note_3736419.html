Symptom

How to change LOB garbage collection time.

Environment

SAP HANA database

Resolution

It is controlled by parameter:
indexserver.ini -> [lobhandling] -> garbage_collect_daily_schedule_s

As explained in KBA 2169283 - FAQ: SAP HANA Garbage Collection

Offset in seconds from the beginning of each day when column store LOB garbage collection (volume-scan) should take place (once per day, e.g. 3600 -> 1:00 a.m.)

If you want to change it to 2:00 a.m. Please set it to 7200, and 3:00 a.m. please set it to 10800.

Keywords

LOB, Garbage collection
