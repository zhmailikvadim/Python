Symptom

Missing qRFC and bgRFC data are missing in Health Monitoring Communication Tab,

Environment

SAP Cloud ALM 2.0

Resolution

The following qRFC and bgRFC metrics are updated in SAP Cloud ALM, if there is data available for the last 7 days.

Check tables QRFC_I_UNIT and QRFC_O_UNIT, these tables contain the bgRFC content for queued units.

In transaction SBGRFCMON, search for units from the last 7 days (transactional and queued processing), the source tables are TRFCQOUT and TRFCQIN.
