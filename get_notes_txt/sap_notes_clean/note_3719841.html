Symptom

You are using multiple level configuration in MRP3 view for material variant.
You are wondering if there is any function module supporting configuration update for it.

Environment

  • SAP ERP Logistics (LO)
  • SAP R/3
  • SAP R/3 Enterprise
  • SAP ERP Central Component
  • SAP ERP
  • SAP Enhancement package for SAP ERPS
  • AP S/4HANA
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA Cloud Private Edition

Cause

In standard system, there isn't a way to do this.

Resolution

It may try to make use of CUCB_SET_CONFIGURATION followed by CUCB_CONFIGURATION_TO_DB and COMMIT WORK.
How these modules work, you can see in debugger, when doing multilevel configuration e.g. in sales order.

If further details or explanation is needed, please get in touch with a consultant or own developer because the topic here belongs to consulting scope which is out of the scope of SAP standard support.

Keywords

lower-level, higher-level, MATERIAL_SAVE_CONFIGURATION, CUXM_SET_CONFIGURATION
