Symptom

In the (FAIR-based) process "Create CFS Report" (/IBS/CAB_REPT_CREATE), the addresses of the partners valid on the position date of the process run are used to generate the reporting data instead of the addresses that are currently valid.

Reason and Prerequisites

To read the addresses, the system uses the position date of the process instead of the current business day BASS (CMU date), which means that the system does not determine the currently valid addresses, but rather the valid addresses.

Solution

The implementation /IBS/CL_CAB_REPT_CREATE_R_DATA of the BF "Determine CFS Data" (/IBS/CAB_REPT_CREATE_R_DATA) of the above The process is adjusted so that the CMU date for the CMU of a (user-defined) building savings contract is determined for each partner and used to determine the valid address of the relevant partner.
