Symptom

  • Get errors when attempting to create an SAP Application datastore.
  • Warning: "Cannot establish database connection. 'NT RFC DLL <SAPNWRFC.DLL> not found. Please make sure that the product is installed correctly.' (BODI-1111348)"

Other Terms

ds, sap, gui, SAPNWRFC, 1111348

Reason and Prerequisites

Environment

  • Windows 64-bit
  • SAP Data Services XI4.3 and higher version

Cause

 SAP GUI is not installed at the Designer's Windows OS.

Solution

Install sapgui 740 at the DS job server's OS.
