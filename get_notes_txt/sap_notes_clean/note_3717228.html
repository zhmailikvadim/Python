Symptom

Inquiry on how to add BasicAuth credentials to the configuration.

Environment

Admin Console (Settings, Administration, Queries / Reports)
Log Connector

Cause

Authorization is currently not supported for the HTTP generic log provider configuration by the Log Connector.

Resolution

To request this feature, submit a feature request on the CDC Influence portal.

See Also

Refer to: Sending log data via the log connector

Refer to: KB0010255 - How do I submit a product enhancement or feature request for SAP Customer Data Cloud?
