Symptom

You are using Function Module MPLAN_CHANGE or mplan_api->mplan_change method to change both the Scheduling Indicator to 'Time - key date' and the cycle unit.

Other Terms

stich, fabkl, mmpt, cycles, update, zeieh, zykl1, offset

Reason and Prerequisites

Key date scheduling checks were performed on the data found in the database instead of the updated cycle.

Solution

Implement the attached correction instruction or the corresponding Support Package.
