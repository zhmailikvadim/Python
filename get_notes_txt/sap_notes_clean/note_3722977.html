Symptom

In the Freight Order Worklist, when executing the "Send to Carrier" action on multiple Freight Orders, the system fails to process any of them if even one order fails validation.

Environment

  • SAP TM 9.0+
  • S/4HANA 1709+

Reproducing the Issue

 1. Navigate to the Freight Order Worklist.
 2. Select multiple Freight Orders (FO).
 3. Trigger the "Send to Carrier" action.
 4. Observe that if one Freight Order fails validation, the system does not perform this action on any of the selected FOs.

 

Cause

This is the standard system behaviour.

Resolution

In SAP Transportation Management, worklists are designed to operate on an "all or nothing" principle during Send to Carrier Action. This means that the action will only be executed if it can be performed successfully for all selected documents. If any of the documents encounter an error, the process is rolled back and none of the actions are applied. This approach is intended to maintain data consistency and is the current standard pattern of the application.
