Symptom

  • eDocument is stuck in "Acknowledged by DRCC" status, and the automatic push response is not being updated. 
  • The final status of the eDocument, such as Accepted or Rejected by KSEF, is not received until the "Get Service Status" button is clicked in the eDoc Cockpit.

Environment

  • SAP ERP 
  • SAP S/4HANA 
  • SAP Document and Reporting Compliance, Cloud edition

Reproducing the Issue

  1. Run Transaction code EDOC_COCKPIT. 
  2. Submit the eDocument for Poland KSEF
  3. Observe that the eDocument is stuck in "Acknowledged by DRCC" status.

Cause

The push response is not functioning correctly from the DRC Cloud edition to the backend business system. 

Resolution

Verify and cross check all configuration points mentioned in the below help portal

  • SAP S/4HANA: Establishing Connection from Cloud Edition to Business System | SAP Help Portal
  • SAP ERP: Establishing Connection from Cloud Edition to Business System (Electronic Documents and Data Submission Only) | SAP Help Portal
