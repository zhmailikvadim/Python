Symptom

Under the "Approvals" section in SAP SuccessFactors Learning, the "External Completion Verification" option displays the German word "skip" in the next row.

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

SAP SuccessFactors Learning

Reproducing the Issue

  1. Log into HCM >
  2. Navigate to the Learning homepage >
  3. Go to the "Approvals" section >
  4. Observe the "External Completion Verification" option is displaying the German word for "Skip" (that is, "Überspringen") in the next row.

Cause

This is a Known Issue.

Resolution

The fix for this issue is planned to be deployed on 1H 2026 release (b2605).

For release timelines, please review SAP SuccessFactors Product Release & Road Map Information.
