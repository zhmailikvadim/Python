Symptom

When the voluntary tax contribution percentage for a Non-Executive Director (NED) changes during the tax year, the system continues to use the previous tax contribution percentage for tax computations.

Other Terms

NON EXECUTIVE DIRECTOR, NED, RPCMSTW0, NEDPC, CST, WTAX, RPCALCW0, PC00_M16_CALC

Reason and Prerequisites

Correction of legal function.

This SAP Note is relevant only for South Africa localization.

Solution

After implementing this SAP Note, the system will calculate the voluntary tax for a Non-Executive Director (NED) using the correct tax percentage.

As a general rule, SAP recommends that you install a solution by implementing the Support Package specified for your release.

However, if you need to install this solution earlier, please implement the attached correction instructions using the SAP Note Assistant. You can find more information about the SAP Note Assistant in SAP Support Portal, under SAP Note Assistant.
