Symptom

You are using the ABAP Cross Trace for the component "Managed Runtime (BSP Framework)". When recording a trace, a MOVE_CAST_ERROR runtime error occurs.

Other Terms

BOPF, ABAP X Trace, ADT, /BOBF/IF_RCTX_DEPENDENT_INST, CX_SY_MOVE_CAST_ERROR, Buffer

Reason and Prerequisites

Missing error handling when attempting to cast a node-specific buffer class to the interface /BOBF/IF_RCTX_DEPENDENT_INST.

Solution

Install the Support Package or implement the attached correction instruction.
