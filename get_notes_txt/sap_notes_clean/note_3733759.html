Symptom

  • Unable to remove class in Material Master due to error 'Assignment & cannot be deleted' CL551
  • Stock of material has been reduced to zero but still unable to remove class of class type 023 in the Material Master

Environment

SAP S/4HANA Public Cloud

Reproducing the Issue

  1. Open the app Change Material.
  2. Input Material Number.
  3. Go to Classification view.
  4. Select the class and press delete button.
  5. System shows error 'Assignment & cannot be deleted' CL551.

Cause

The material still has classified batches.

Resolution

There are two options.

Option 1: Delete the relevant classification used in classified batch with this class.

  1. Delete the classification in Change Batch app (MSC2N) under 'Classification' tab for all classified batches with this class.
  2. Delete the classification assignment from material master in the change material app

Option 2: Archive all the classified batch with this class.

  1. Make the inventory at the batch level to zero.
  2. Archive the batch using the archiving object LO_BATCH
  3. Delete the classification assignment from material master.
