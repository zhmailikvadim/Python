Symptom

Error: "HTTP operation failed invoking https://xxx-xxx.xxxxxxxxx.xxxx/xxxxxxx 401 Unauthorized Error" When Sending Messages from Network to External System

Cause

HTTP 401 Unauthorized error arises due to incorrect or missing authentication credentials when communicating with external systems. 

Resolution

  • Verify and correct the authentication configuration maintained under System Connection of the carrier tenant in the Authentication Details section.
  • Confirm the username and password are correct if using Basic authentication.
  • Ensure the API key is valid and active if using API Key authentication.
  • Check that the OAuth client ID and client secret are correct if using OAuth 2.0 authentication.
