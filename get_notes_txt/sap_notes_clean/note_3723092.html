Symptom

 Relevant files missing in the document info records (DIR) required to run the business process 7VG.

Reason and Prerequisites

The files are required to run the business process described in the sceneario 7VG.

Solution

Perform the following steps to upload the attached files in the respective document inforecords (DIR):

  1. Download the files attached to this SAP Note.
  2. Use either the backend transaction CV02N or the FIORI App 'Manage Document' (F2733) to access the DIR - ID: D2O_09077205423_C; Document Type: PEO; Part: 000; Version: 00.
  3. Go to the section to manage originals, then upload the file 'D2O_09077205423_C.vds' downloaded in step 1 and check it in the content server 'DMS_C1_ST'.
  4. Change the status of the document info record to 'Released'.
  5. Repeat the step 2 through 4 for the DIR:  ID: D2O_SAFETYINS_01; Document Type: QC2; Part: 000; Version: 00 using the file 'D2O_SAFETYINS_01.pdf'.
  6. If you have activated the Germany country version (DE) version of the best practice:
    1. Repeat the step 2 through 4 for the DIR:  ID: D2O_09077205423_C_SFRMPD; Document Type: MPD; Part: 000; Version: 00 using the file 'D2O_09077205423_C_SFRMPD_DE.xml'.
    2. Repeat the step 2 through 4 for the DIR:  ID: D2O_09077205423_C_RWRKMPD; Document Type: MPD; Part: 000; Version: 00 using the file 'D2O_09077205423_C_RWRKMPD_DE.xml'.
  7. If you have activated the United States of America country version (US) version of the best practice:
    1. Repeat the step 2 through 4 for the DIR:  ID: D2O_09077205423_C_SFRMPD; Document Type: MPD; Part: 000; Version: 00 using the file 'D2O_09077205423_C_SFRMPD_US.xml'.
    2. Repeat the step 2 through 4 for the DIR:  ID: D2O_09077205423_C_RWRKMPD; Document Type: MPD; Part: 000; Version: 00 using the file 'D2O_09077205423_C_RWRKMPD_US.xml'.

After executing the steps the files are uploaded to the relevant DIRs and the DIRs are in released status.

Note: This note is relvant only for S/4HANA version 2025 FPS01 onwards. For version 2025 FPS00, use the SAP Note 3602590.
