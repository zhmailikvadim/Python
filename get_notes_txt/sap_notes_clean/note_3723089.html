Symptom

  • During the Export Preparation step using the SWPM tool (1680045), the step "runR3szchkInParallel" fails with the error message: "The parallel execution of R3szchk has failed. Check the log file(s) R3szchk_.exe.log".
  • The log file contains the error message: "ERROR: invalid or incomplete option '-E'".

Environment

  • OS/DB Independant.
  • SAP NetWeaver 7.0x

Reproducing the Issue

  1. Start the SWPM tool.
  2. Choose the Export Preparation option.
  3. Observe the error message in the log file: "ERROR: invalid or incomplete option '-E'".

Cause

  • The old SAP Kernel version R7.20/V1.6 used during the Export Preparation does not support the option "-E". This option is available in SAP Kernel versions 7.2x or higher.

Resolution

  1. Update the SAP Kernel to the latest supported version for the system. Refer to the PAM (Product Availability Matrix) for supported Release/OS/DB/Kernel combinations.
  2. Start the SAPinst and Export Preparation step using the root user only (for Linux systems) and follow SAP KBA 2445952 for Windows Systems. Do not use the SIDadm user.
