Symptom

The balance carryforward transaction FAGLGVTR fails with the error message: FINS_BCF_EN 088   Retained earnings account &1 is a reconciliation account (type &2).

Environment

  • Financial Accounting (FI)
  • SAP S/4HANA Finance
  • SAP S/4HANA

Reproducing the Issue

Execute the balance carryforward transaction FAGLGVTR.

Cause

  • The error is caused by new logic checks introduced in SAP Notes 3328691 and 3386219 in SAP S/4HANA. These checks ensure that retained earnings accounts are not customized as open item managed or reconciliation accounts during the balance carryforward process.

Resolution

  1. Create a new General Ledger (GL) account that is not configured as a reconciliation account and not open item managed
  2. Assign the newly created GL account as the retained earnings account in transaction OB53.
  3. Transfer the balances from the old retained earnings account to the new GL account using adjustment entries or appropriate balance transfer methods.
  4. Rerun the balance carryforward transaction FAGLGVTR for all affected fiscal years.

See Also

SAP Notes 3328691 and 3386219
