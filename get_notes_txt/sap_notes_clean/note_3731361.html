Symptom

The net value shows zero in the invoice list output.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Open the output print preview for the invoice list document.
  2. Observe that the net value in the printed document is zero.

Cause

An incorrect form template is determined for the invoice list output.

Resolution

Use the correct invoice list template SDBIL_IL_STANDARD_DE.

Keywords

invoice list, print preview, net value zero, output document, form template, SDBIL_IL_STANDARD_DE, SDBIL_CI_STANDARD, billing output, output parameter determination, form determination, localization, billing, invoice list template
