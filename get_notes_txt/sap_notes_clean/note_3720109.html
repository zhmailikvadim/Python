Symptom

For the Verification of Payee Scheme API implementation, X-CSRF Token is not part of the request header parameter as per the standard API documentation and hence it's not required.

Other Terms

X-CSRF Token, VOP, X-Request-Id.

Reason and Prerequisites

Standard SAP Framework has a default implementation for checking X-CSRF Token for its API implementations. This check needs to be skipped for the VOP Scheme API Implementation, as the VOP API Standard documentation does not mention the usage of the X-CSRF Token. To mitigate security threat, if any, the idempotency check for each of the API requests has been implemented using X-Request-Id, a mandatory Request Header parameter.

Solution

Implement the correction instruction attached to this SAP Note. Alternatively, you can import the referenced Support Package.
