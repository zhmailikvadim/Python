Symptom

You navigate to the Master Agreement, open the Promotion tab, and click on the Direct Agreement number generated based on the Promotion. You expect the system to open and display the corresponding Direct Agreement content. However, when clicking the Direct Agreement number, the system does not respond and no navigation occurs. The link is inactive and does not open the related document.

This note corrects the behavior by ensuring that the Direct Agreement number link properly navigates to and displays the associated Direct Agreement content.

Other Terms

/IRM/IPPQM, Navigation Issue, Master Agreement, Unresponsive

Reason and Prerequisites

This is caused by program error

Please implement SAP Note#3706384 as pre-requisite for 2021 & 2022.

Solution

Please implement the correction instructions. These corrections are only valid if you have applied the following Support Packs.

Release V/4 – 2021, 2022, 2022 V2, 2023, 2023V2, 2023 V3, and 2023 V3 Support Package 1.

Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object
