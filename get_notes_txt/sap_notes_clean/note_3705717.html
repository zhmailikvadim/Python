Symptom

A custom field with type text or list with length 1 is converted to edm type boolean in OData services

Other Terms

The field uses a domain for typing, which has FLAG or BOOL in its name.

Reason and Prerequisites

Wrong heuristic by gateway framework

Solution

Apply this note. Afterwards reset the custom fields runtime buffer by running report CFD_INVALIDATE_RUNTIME_BUFFER or via transaction SCFD_FIELDVIEWER.
