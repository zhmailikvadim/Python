Symptom

When manually entering a value in the filter criteria field for the master data object (MDO) instead of selecting from the dropdown list, the error message "The server does not support the functionality required to fulfill the request" is displayed.
This issue occurs for all filter input fields when attempting to manually search for a value.

Environment

SAP Digital Manufacturing 2511

Reproducing the Issue

  1. Open the custom SFC(KPI) Report application.
  2. Navigate to a filter criteria field for the master data object (MDO).
  3. Manually enter a value into the filter criteria field instead of selecting from the dropdown.
  4. Observe the error message: "The server does not support the functionality required to fulfill the request."

Cause

The system works as designed, currently it is not supported to add field values manually at this point.

Resolution

Currently the drop down list should be used to add values, manual input is not supported. To have this functionality in some future release an enhancement request can be addressed at https://influence.sap.com/sap/ino/#/campaign/3537.
