Symptom

When you click on the hyperlink for the settlement document, it does not display the settlement document; instead, it shows the following value in the screen against the settlement document field:

{ampersandSETTL_DOC_IDampersand}

You had implemented the manual instructions from note 3503870, including the code correction.

Environment

  • Settlement Management
  • Agency Business
  • Condition Contract Management
  • Condition Contract Settlement
  • SAP S/4 HANA

Resolution

Review the manual instructions from note 3503870 and replace the text "ampersand" with the symbol "&".

Keywords

SETTLEMENT MANAGEMENT, SETTLEMENT DOCUMENT, WS02000445, MYINBOX, HYPERLINK, MY INBOX, LINK
