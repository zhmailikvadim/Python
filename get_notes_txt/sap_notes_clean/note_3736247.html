Symptom

  • Additional line items are posted on certain working capital accounts during the cutback in a specific period and year.
  • Gross billable amounts from second cutback run are correct, but discrepancies are observed in the first cutback values.
  • The issue occurred only during the first cutback after JVA activation and enrichment of historical transactions.

Environment

  • Joint Venture Accounting (JVA)
  • Financial Accounting (FI)
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA
  • SAP S/4HANA Finance

Reproducing the Issue

  1. Activate Joint Venture Accounting and enrich historical transactional data (for example, from earlier fiscal years up to the go-live year).
  2. Ensure affected working capital/material accounts and their cutback mapping accounts have “balance in local currency” active.
  3. Perform the first inception‑to‑date (ITD) cutback run for the first cutback period.
  4. Review cutback postings, observe additional line items on the working capital accounts not included in gross billable postings and discrepancies specifically in the first cutback period, while periods from second cutback run appear correct.

Cause

The issue is caused due to activation of inception‑to‑date (ITD) for historical data, causing the cutback program to pick up all billable costs ever posted.

Resolution

  1. Reverse the impacted cutback run using transaction GJCB_REVA.
  2. Update the Cutback Run ID (ACDOCA-CBRUNID) for historical billable items to prevent unintended ITD pickup by running transaction code GJMIGCBR in each company code where ITD is active (use parameters as per business needs; typically defaults).
  3. Re-run the cutback for the affected period after completing GJMIGCBR.
  4. Validate the result by checking processed billable amounts in ACDOCA (for example, via SE16N) and confirming no unintended historic amounts are included.
  5. Review cutback results and confirm alignment with gross billable postings.
