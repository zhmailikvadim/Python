Symptom

  • You are not able to select the payroll frequency on the 'Setup activity list' application.
  • F4-help of 'Frequency' field is showing 'No Value'.

Environment

  • Human Capital Management (HCM)
  • Payroll Control Center (PCC)
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP SuccessFactors Employee Central

Reproducing the Issue

  1. Logon to PCC portal.
  2. Navigate to Set-up Activity List application.
  3. Start creating a setup activity list.
  4. Attempt to select the payroll frequency on the setup activity list: the expected frequency cannot be selected or is not available.


Cause

Incorrect configuration in payroll period-related tables t549r_cust and t549n.

Resolution

  1. Go to SM30 t-code.
  2. Pass the view name as V_T549R_CUST.
  3. Click on the maintain button.
  4. Maintain correct Period-Parameter with applicable HCM Localization value.
  5. Review T549N  table via SE16N t-code to ensure corresponding payroll period entries exist and are consistent with V_T549R_CUST.
