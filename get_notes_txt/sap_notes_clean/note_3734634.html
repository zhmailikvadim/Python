Symptom

One specific Visit Type A cannot be deactivated in Visit Type settings.

Environment

  • SAP Sales Cloud Version 2
  • SAP Sales Cloud and SAP Service Cloud Version 2

Reproducing the Issue

  1. Go to Settings.
  2. Search for Visits.
  3. Select Visit Types.
  4. Attempt to deactivate Visit Type A.
  5. It is greyed out.

Cause

Visit Type A is selected as the default Visit Type under Settings > Visits > General, and the system does not allow deactivation of the default Visit Type.

Resolution

  1. Navigate to Settings > Visits > General.
  2. Change the default Visit Type to any other Visit Types (not A).
  3. Save the change.
  4. Go to Visit Type settings.
  5. Deactivate Visit Type A as required.
