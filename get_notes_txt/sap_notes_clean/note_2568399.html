Symptom

In transaction VL31N or VL01N you receive the error message "Material subject to serial number not possible for external warehouse" (L9 501).

Other Terms

EWM, L9501, /SPE/V_T340_E

Reason and Prerequisites

Due to a program error the below IMG activity disables the usage of serial numbers for deliveries which need to be distributed to an external warehouse management system.

IMG activity:
Logistics Execution/ Extended Warehouse Management Integration/ Basic Setup of Connectivity/ Configure SAP EWM-Specific Parameters.

Solution

Independent from the correction of the IMG activity, you may already correct your Customizing as follows:

  1. Start transaction SM30 and enter view /SPE/V_T340. Click button "Maintain".
  2. Set checkbox in column "SN Dec. WM" in all lines with value "Extended Warehouse Management" (E) in column "Comm. WM".
  3. Save and transport your changes.

Correct the IMG activity as follows:

  1. Follow the description of the manual pre-implementation steps.
  2. Apply the correction instruction
