Symptom

You create posting results with help of report RPCIPE00 or RPCIPE01.

The document generated by the system contains substitutions which you reach via transaction PCP0 for display.

  1. When you display the substitutions, a line is displayed with the value for a cost center even though the cost center, e.g. from table C1 which was valid but which has a segment assigned which is replaced by master cost center.
  2. In addition to point 1 above, another line is displayed were the replaced segment replacement is shown. 

 All lines contain the value "Replaced by: BAPI_CODINGBLOCK_PRECHECK_HR".

Other Terms

RPCIPE00, RPCIPE01, RPCIPS00,  PPDST, COSUB, seg_sub, Segment, Cost center, Employee Cost Center substitution

Reason and Prerequisites

Display logic of Substitutions.

Solution

Implement the correction instructions or import the relevant Support Package to correct the error.
