Symptom

User cannot find existing airports or ports when searching in the Manage Locations app (F6225). Further create function for new locations (airports/ports) is not possible. 

Environment

SAP S/4HANA Public Cloud Edition

Reproducing the Issue

  1. Navigate to the Manage Locations app.
  2. Click on "Go".
  3. The search list does not show any locations.

Cause

The tenant type used here is Customizing Tenant. This tenant type does not have any demo data or pre-delivered locations. Only the Starter systems have the demo data.

Resolution

From the Manage Locations app, it is possible to create locations such as ports, airports, railway stations, and rail junctions. You can also display, edit, and delete these locations. To do so; 

- Go to Manage Locations app
- Click on Create button 
- Select the Location Type. (For example - Shipping point, Port, Airport, Rail Junction etc.) 
- Now enter the required Location manually
- Click on Continue 
- Under General Information tab, Define Language (Add descriptions for locations in multiple languages)
- Address details and Geocoordinates (Calculate or recalculate geocoordinates for specific locations using the Calculate Geocoordinates button)
- Enter Identification information and click on Create button.
