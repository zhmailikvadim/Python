Symptom

After blocking a user from accessing the system, the user count in the "Manage Workforce" application stays the same as before blocking.

Environment

SAP S/4HANA Cloud Public Edition

Resolution

Blocking a user does not delete them from the system, therefore the user count will not reduce as technically the user still exists within the system.

Only when the ILM Data Destruction process has been complete, then the user will officially be deleted from the system and the count in "Manage Workforce" will reflect this.
