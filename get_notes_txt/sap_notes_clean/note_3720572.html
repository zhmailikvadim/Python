Symptom

This SAP Note contains a list of the scope items that are planned for availability in local versions other than Germany and USA with the 2608 release of SAP S/4HANA Cloud Public Edition.

This SAP note is subject to changes and is specific to the 2608 release of SAP S/4HANA Cloud Public Edition.

Other Terms

Globalization, Scope items, Country/Region, Local versions

Solution

This SAP Note contains an excel file that you can find as an attachment. The file lists scope items planned for availability for the respective countries mentioned in this Note with the S/4HANA Cloud Public Edition 2608 release.

This SAP Note is cross-referenced in the roadmap explorer at https://roadmaps.sap.com/board?range=2026Q1-2026Q1&q=cross%20country%20scope%20items&PRODUC… .

Also to view details of all new Localizations topics planned for release in Q1, 2026, please refer https://roadmaps.sap.com/board?range=2026Q3-2026Q3&PRODUCT=67837800100800007389&FT=GLOB#Q3%202026 in roadmap explorer.

Since the content of this note relates to the planned delivery of scope items for a future release, the legal disclaimer mentioned in the roadmap explorer also applies to the SAP Note.

Please note that not all scope items available for a given country/region are localized to comply with business and legal requirements in the given country/region. Some scope items might be used only for standard business processes and the preconfigured content might have to be adapted to meet the requirements in a country.
