Symptom

The employee is a martyr's dependent. The record in the Additional Personal Data (0529) infotype with the certificate number for martyrs' dependents is maintained with start date in the next month, not the current month.

When you run the Payroll Driver, Version China (HCNCALC0) report for this employee in the current month, the system displays the CN_DTR data in payroll function P0529. This causes the system to calculate the tax reduction in payroll function CNTRD incorrectly.

After you apply this SAP Note, for the above-mentioned employee, the CN_DTR data in P0529 for the current month is blank. Therefore, the system does not calculate the tax reduction in CNTRD.

Other Terms

CN_DTR, P0529, CNTRD, Martyrs' Dependents

Reason and Prerequisites

This is a program error.

Solution

Apply the corresponding SAP Support Package or the correction instructions in this SAP Note.
