Symptom

You are a system administrator and try to import transports containing SAP S/4 HANA MC DT objects into your system. You get an error message that you are not authorized to do it. Import is not possible.

Other Terms

Transport, AFTER_IMPORT

Reason and Prerequisites

Transport after import logic contains authorization checks that are not always required.

Solution

Implement the attached correction using transaction SNOTE or import the relevant Support Package in the original SAP S/4HANA system. Import required transport again.
