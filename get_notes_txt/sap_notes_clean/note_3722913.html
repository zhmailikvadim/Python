Symptom

  • PDF files cannot be opened in SAP Engineering Control Center (ECTR) 
  • You don't get an error message
  • The log file may display the following lines: 
    • cmd after filename replace: "C:\CAD\ECTR\basis\sys\win\bin\dsc_LaunchRegisteredApplication.exe" "C:\Users\[xxx]\ECTR\filespace\495059\addfiles\N-5CG3343JJ3\QXC\010\495059\ro\E03000000000000030000046046400PRT\SAP_CID_300000460464_E03_PRT_V00.pdf" -view`.
    • The file dsc_LaunchRegisteredApplication.exe` is not located in the expected folder C:\CAD\ECTR\basis\sys\win\bin\ but is instead located under \folder_location

Environment

SAP Engineering Control Center (SAP ECTR)

Reproducing the Issue

  1. Start SAP Engineering Control Center (ECTR).
  2. Attempt to open a PDF file.

Cause

The configuration settings for the file path of `dsc_LaunchRegisteredApplication.exe` are incorrect.

Resolution

  1. Verify the configuration settings for the file path of `dsc_LaunchRegisteredApplication.exe`.
  2. Update the configuration settings as follows:
  • Replace:
  • `plm.appl.startCommand.PDF = "C:\CAD\ECTR\basis\sys\win\bin\dsc_LaunchRegisteredApplication.exe" "$(filename)" -view`
  • `plm.appl.startCommand.TXT = "C:\CAD\ECTR\basis\sys\win\bin\dsc_LaunchRegisteredApplication.exe" "$(filename)" -auto`
  • With:
  • `plm.appl.startCommand.default = dsc_LaunchRegisteredApplication.exe "$(filename)"`
  • `plm.appl.startCommand.PDF = dsc_LaunchRegisteredApplication.exe "$(filename)" -view`
