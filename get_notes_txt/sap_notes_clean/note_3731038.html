Symptom

Working with SAP Cloud ALM - Landscape Management you attempt to Maintain Corporate Group (CCC) S-User function, then you receive below error

Can't save S-User. - Connection test failed with http Status: Error while connecting to the support backbone. It might be due to invalid username or password. Try again with different login details.

Environment

SAP Cloud ALM

Reproducing the Issue

  1. Open the Maintain Corporate Group (CCC) S-User function.
  2. Add an S-User and attempt to save; the operation fails with a connection test error to the support backbone.

Cause

Invalid credentials for the CCC S-User or use of unsupported special characters in the S-User password can prevent connection to the support backbone.

Resolution

  1. Reset the password of the CCC S-User used in Maintain Corporate Group (CCC).
  2. Ensure the new password uses only supported characters as outlined in the password policy; refer to KBA 2174416 - Creation and activation of Technical Communication Users - SAP for Me.
  3. Re-enter the updated S-User credentials in the Maintain Corporate Group (CCC) S-User function and save.
  4. If the error persists, verify that the S-User is active and not locked, then retry.
