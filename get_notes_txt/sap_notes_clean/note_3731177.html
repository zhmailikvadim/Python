Symptom

You are working with routes in a mobile scenario within the Last Mile Distribution solution. When you download a route with the OData service Mobile Data for Last Mile Distribution (LMD_MA) and the route is no longer valid due to its lifecycle status, the system returns an unclear error message. The message does not indicate the reason for the failed download.

Other Terms

LMD, Mobile Adapter, LMD_MA, Route

Reason and Prerequisites

When the specified selection criteria, such as a route UUID, do not match any data, the system returns an empty response. This empty response causes subsequent issues. The correction contained in this SAP Note ensures that the system returns a clear business error message.

Solution

 Implement the attached correction instruction.
