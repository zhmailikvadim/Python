Symptom

When saving the changes in business configuration, the transport pop-up but cannot find the expected transport request.

Environment

SAP S/4HANA Cloud Public Edition

Cause

1.The specific transport is not under the name of current user or it does not contain a task under the name of the current user.

2.There is a restriction that only the first 100 transport requests are shown in the option list.

Resolution

For the cause 1:

Create a task under the specific transport or assign the transport to current user.

For the cause 2:

Use another user with less customizing transport requests or to release or cancel some of the customizing transport requests.
