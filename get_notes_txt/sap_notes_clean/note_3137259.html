Symptom

You got the following error during system upgrade/update at phase MAIN_NEWBAS/PARRUN_CONF_REFCHAR:

*******************

A4 EUPGBA 041 Report name ...: "RNGC_CONV_REF_CHAR" Component: "CA-CL" "(Classification)"
A4 EUPGBA 006 Log name ......: "RNGC_CONV_REF_CHAR_000.FPA"
A4 EUPGBA 007 Start time ....: "24.10.2022" "16:10:08"
A4 EUPGBA 002 " "
A4 EUPGBA 001 -------------------------------------------------------------------------
A4 ECFD_TRN_ADAPTER 009 Could not add object "DDLS" "ZRC_NODES000_DDL" to object directory
A2EEUPGBA 031 Exception raised at "CL_CFD_TRANSPORT_ADAPTER======CP", "CL_CFD_TRANSPORT_ADAPTER======CM003" line "53"

*******************

Environment

All SAP NetWeaver ABAP systems.

Reproducing the Issue

Repeat the phase.

Cause

Object cannot be added to the object directory. In transaction SE06, the global setting and the namespaces of the object are set as “not modifiable”.

Resolution

For avoiding this issue please implement SAP Note 3219730 - Avoid Local Object Registration at Dynamic CDS View creation

Or do this steps as workaround: 

  1. Go to SE06 and change the global setting to “modifiable”.
  2. Find out the namespace of the object and set it to “modifiable”.
  3. Repeat the phase in SUM.
