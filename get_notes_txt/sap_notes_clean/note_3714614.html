Symptom

During installation or upgrade of BI on Linux, the process fails with the error:
setup.engine/perl/bin/perl: cannot execute binary file: Exec format error

Environment

  • SAP BusinessObjects Business Intelligence Platform 4.x
  • SAP BusinessObjects Business Intelligence Platform 2025
  • Linux

Reproducing the Issue

  1. Install the installer package and  extract it.
  2. Start the installation using setup.sh.
  3. Observe that the install failed with the error:
    setup.engine/perl/bin/perl: cannot execute binary file: Exec format error

Cause

An installation package intended for a different OS version was mistakenly used.

Resolution

Refer to the following steps to download the correct installation package:

  1. Visit the Software Center - SAP for Me .
  2. Navigate to SBOP BI PLATFORM 4.3 -> SBOP BI PLATFORM SERVERS 4.3.
  3. Download the installation packages corresponding to the target version and OS.
