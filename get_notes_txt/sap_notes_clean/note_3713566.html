Symptom

The DRC for Domestic Sales Purchase List for Romania (Report Category: RO_DSPL) is not enabled for reporting documents posted via Contract Accounting.

Other Terms

RO_DSPL, FICA, CA, Contract Accounting, DRC, RO_RPG_ENT, RO_DSPL_V3, CL_DSPL_RO_EVENT_HANDLER, CL_DSPL_RO_BRF, C_RO_STRPDOMESTICSLSPURLISTQ, C_RO_STRPDOMESTICSLSPURLISTC, C_RO_CASTRPDMSTCSLSPURLISTQ

Reason and Prerequisites

This SAP Note is relevant only for Romania.
This SAP Note is applicable only for SAP Document and Reporting Compliance.

Prerequisite:

3664502 - FICA DRC: Generic CDS for building tax items in contract accounting for DRC reports - I

3674478 - FICA DRC: Generic CDS for building tax items in contract accounting for DRC reports - II

3677854 - FICA DRC: Generic CDS for building tax items in contract accounting for DRC reports - III

3713171 - FICA DRC: Generic CDS for building tax items in contract accounting for DRC reports - IV

3713060 - RO DSPL 394: DDIC Technical change to AccountingDocument (DRC)

3713058 - RO DSPL 394: Technical change to AccountingDocument (DRC)

3617107 - Romania: FKK_ID_RO_REP Package Creation

Solution

As a general rule, SAP recommends that you implement the Support Package specified for your release. However, if you need to install a solution earlier, use the SAP Note Assistant to implement the correction instruction. You can find more information about the SAP Note Assistant in SAP Support Portal, under SAP Note Assistant.

Refer the configuration steps in Setting Up the Domestic Sales Purchase List and Configuring the Tax Box Mappings for Domestic Sales Purchase List

In Configuring the Tax Box Mappings for Domestic Sales Purchase List settings, assign the tax code/Transaction key/Document type relevant for contract accounting to the relevant TaxBox.

To generate the report follow steps in Generating the Domestic Sales Purchase List

The Contract Accounting Documents relevant for each run will be updated to the log table DFKK_SRF_REP_ADV.

You can also display the analytical view by clicking Analyze Data on the right-hand side of your screen after you generate the report. You will have Two options for Financial Accounting and Contract Accounting(if the run contains CA documents). You can then check the data and dynamically change the analytical view for the displayed fields.
