Symptom

  1. In the report RPLHZPT1, it's unable to download local file, in case there is missing setup for the entry 111 in the table T5T1K.

Other Terms

  • RPLHZPT1

Reason and Prerequisites

Source code changes have been done in RPLHZPT1 report to fix mentioned error.
