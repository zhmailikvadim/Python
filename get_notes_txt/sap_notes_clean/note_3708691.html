Symptom

When I setup a connection between Signavio Process Intelligence and my SAP SAP ECC/S4HANA system, and run a preview on the table in the Process Data Pipeline source data, the values have been rounded. Instead of being shown to three decimal places, the values are rounded to two decimal places.

Cause

The table data preview function in Process Intelligences' Source Data, shows the data as it appears in the Source Data. It carries out no transformations on the data.

Resolution

Please ensure that you are referencing the exact same record in your source data. As there can be millions to billions of records in your source data, is can be very easy to confuse one record for another.

If you are sure of this however, please let us know by creating a support case with Signavio Support via the SAP For Me platform: https://me.sap.com/servicessupport
