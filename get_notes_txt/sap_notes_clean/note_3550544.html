Symptom

Change date determination of MDX change event BILLINGINS is not correct when the change is done via some transactions, for example, EG34 and EC30.

Other Terms

MaCo, /UCOM/CL_MDX_CHGTRK_BILLINGINS, DET_CHANGE_DATE_FROM_ITEM

Reason and Prerequisites

Program error

Solution

Please implement the correction instructions using SNOTE or import higher version of Support Package.
