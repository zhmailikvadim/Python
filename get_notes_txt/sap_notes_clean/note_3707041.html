Symptom

You use SAP Process Orchestration(PO) and REST Receiver channel. You configure the channel as selecting Support Attachments with type multipart/form-data and module parameter multipartEnforced with value true. You are observing that the multipart Content-Type header is changed to multipart/related instead of configured multipart/form-data.

Other Terms

multipart/form-data, multipartEnforced

Reason and Prerequisites

That is a side effect of SAP Note 3567885.

Solution

With this SAP Note a code correction is introduced, that corrects the upper behaviour.

Apply the patch matching your Support Package version as listed in the Support Package & Patches section of this SAP Note. Follow the instructions described in SAP Note 952402.
