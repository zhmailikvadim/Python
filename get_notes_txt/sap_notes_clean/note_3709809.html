Symptom

In the process "Integration of Meter Operator Contracts" (DE_B_MOSB_INTG_PROC), step 0090-0030 added a new time slice to a terminated contract instead of to the new or active contract.
This issue is relevant for the market roles: MOS, SUP.

Other Terms

MOSB, Integration of Meter Operator Contracts, DE_B_MOSB_INTG_PROC, timeslice, terminated contract, active contract, meter operator contract

Reason and Prerequisites

Program error.

Solution

Implement the corrections of this SAP Note or import the relevant support package.
After implementing this SAP Note, the system adds the new time slice to the new or active contract.
You can verify the fix by checking that the new time slice is added to the new or active contract and not to the terminated contract.
