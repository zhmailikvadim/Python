Symptom

When making a Physical Inventory Count for a serialized material the first serial number entered is not displayed in the list of all entered serial numbers when the material is viewed

Other Terms

  • SAPIM
  • Inventory Manager
  • Physical Inventory
  • Serial Number

Reason and Prerequisites

  • User must have access to the Eclipse Editor with Agentry Plugin containing the Inventory Manager application.
  • User must have access to deploy new Application definitions to the SMP server hosting the Inventory Manager application.

Solution

  1. Download the attached SAPNote-3068351-IM.agxz file from this note.  Save it to a location where it can be accessed from the Eclipse editor with Agentry Plugin.
  2. Launch the Eclipse Editor with Agentry Plugin containing the Inventory Manager definitions.
  3. Export the current definitions to save as a backup.
  4. Right click on the Inventory Manager project in the Eclipse Editor and choose Compre With->Other Agentry Application.  The Agentry Compare tab will open.  Maximize the tab.
  5. Click the Select Session button and select Manage Compare Sessions drop down.  Click the Browse button and navigate to select the file downloaded in step 1.  Click Finish
  6. Enter a name for the compare session and click OK.  The definitions will load and the comparison windows will display.
  7. Select the top check box on the right window to select all different definitions.
  8. Click the Import button to import all new definitiions.
  9. Click the save box to save the definitions.
  10. Close the Agentry Compare window.
  11. Publish the new definitions to your development server and test.
  12. After testing is complete, publish the application to your production environment
