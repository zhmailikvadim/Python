Symptom

The popup windows are different when exporting an Excel file. 
Sometimes, the popup window needs to select desktop directory to save the file or shows different save file window. 

Environment

 SAP S/4HANA 2023

Reproducing the Issue

* Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

  1. Execute transaction
  2. List>Export>Spreadsheet
  3. Export as window or Save file window pops up

Resolution

The different popup windows are the expected result when exporting an Excel file.
The ALV triggers a GUI function that displays a popup allowing the user to select the file path/directory. SAP GUI shows different popups depending on whether scripting is enabled.

If scripting is not active and the platform supports the standard file dialog, SAP GUI displays the native Windows file dialog.
If scripting is active, SAP GUI switches to an alternative dialog.

This behavior is explained in the following SAP Notes:
587202 – Restrictions when using SAP GUI Scripting (see bullet point 9)
1974590 – File save dialog during scripting 
2038061 – File open dialogs cannot be handled in SAP GUI Scripting

From ALV component side, there is no adjustment as the system is functioning as designed.
