Symptom

You use the transaction GM_RRB_IND in the context where Grants have been submitted to the billing process without the new feature for GL Account billable. See note 1832216 New selection of billable records (Grantee Management).

In case you continue to use these Grants with the new feature installed and in case you set the GL Account as billable or as not billable, you observe inconsistent results.

method IF_EX_DIP_FILTER_OLD_DLIS~CHECK_DLI (Check Dynamic Item)
