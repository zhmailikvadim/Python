Symptom

When you execute a wage type processing ESS scenario, the system fails with a system termination.

Other Terms

PS, PS_DIR, HR_INDVAL_READ_INFOTYPE, RP_FILL_WAGE_TYPE_TABLE, RP_EVALUATE_INDIRECTLY.

Reason and Prerequisites

When you execute the ESS scenario, a program termination occurs.

During the processing of data of infotype 0014 (Recurring Payments and Deductions), the program tries to read data records and valuate the wage types they contain. However, these are already contained in the table PS_DIR and originate from another infotype.
Since duplicate entries are not allowed at this point, the process is terminated with an error message.

Solution

 Import the relevant HR Support Package into your system or implement the correction instructions.
