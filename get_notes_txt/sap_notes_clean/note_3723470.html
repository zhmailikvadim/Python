Symptom

In the process "Receipt of Invoice" (DE_B_CP_INVC_RECV), step 0070 creates only one municipal discount line as the total line when creating an invoice document, instead of creating multiple municipal discount line items.
This issue is relevant for the market roles: SUP, DSO, MOS.

Other Terms

Billing Enhancement, Invoice Document, Municipal Discount, Subtransaction, INVMON, /US4G/DISC_AD, /US4G/V_DISC_AD, /US4G/CL_INV_PAYM_CUST_ACCESS, /US4G/CL_UBD_INVDOC, BUILD_BAPI_INV_LINE_MUNI_DISC, /US4G/ISU_DEREG_INV_DATA_120

Reason and Prerequisites

Program error.

Solution

Implement the corrections of this SAP Note or import the relevant support package.
After implementing this SAP Note, the system creates multiple municipal discount line items in the invoice document. You need to retrigger the process to create a new invoice document.
You can verify the fix by checking that multiple municipal discount line items are created in the invoice document and that the invoice document can be processed successfully.
