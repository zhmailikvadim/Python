Symptom

Requirements cards in the new learning home page are not showing decimals for CPE hours.

Environment

SAP SuccessFactors Learning

Reproducing the Issue

  1. Go to the new learning home page >
  2. Required Learning >
  3. Find any Requirements card >
  4. Click on Complete Requirement >
  5. The CPE hours are not showing decimals.

Resolution

The Engineering team has provided a code fix for the reported scenario, the fix will be available in the learning release of 1H 2026 / 2605.

The 1H 2026 / 2605 Learning Release schedule dates will be posted in the Product Release & Road Map Information page.
