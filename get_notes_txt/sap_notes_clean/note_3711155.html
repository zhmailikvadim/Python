Symptom

  • In transaction FEB_BSPROC, the GSBER field is not automatically populated in the "on account" tab despite the user parameter GSB being properly maintained.

Environment

  • SAP R/3
  • SAP ERP
  • SAP S/4HANA

Reproducing the Issue

  1. Access transaction FEB_BSPROC.
  2. Navigate to the "on account" tab.
  3. Observe that the field GSBER is not populated automatically despite the user parameter GSB being properly maintained.
  4. Navigate to the "assignment" tab and observe that the field is populated.

Cause

FEB_BSPROC does not read SU3 GSBER for the "on account" tab.

Resolution

  1. Implement FI substitution to set GSBER during posting.
  2. Utilize BTE 1650 to default GSBER.
