Symptom

 In the Provisioning Access Approval email generated from Manage Provisioning Access, the approver email is blank.

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental. 

Environment

SAP SuccessFactors HCM Suite

Reproducing the Issue

  1. Login with an admin user with access to Manage Provisioning Access;
  2. Go to Admin Center -> Manage Provisioning Access;
  3. Click on the '+' sign on the top right;
  4. Enter your partner email address;
  5. The approver email field in the Provisioning Access Approval email is blank.

Cause

The approver's email address is not maintained in the UDF(User Data File), so the approver email cannot be generated in the email.

Resolution

  1. Export the User File through ''Employee Export'' and check whether the approver's email field has a value. If it is blank, you need to add a real and valid email address;

  2. Re-add the partner's email.
