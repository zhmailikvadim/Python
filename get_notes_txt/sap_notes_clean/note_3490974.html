Symptom

You have an issue with the SAP S/4HANA Data Migration content delivered for SAP S/4HANA 1909 that is related to an error with the content

This Note is valid for:

       • SAP S/4HANA - LTMC

             o "Transferring Data Using Files"
             o "Transferring Data Using Staging Tables"

Other Terms

Data migration, CO - Cost center, SIF_KOSTL_MASTER, Migration cockpit.

Reason and Prerequisites

You have installed SAP S/4HANA 1909 (SP00 - SP10)

You are using SAP S/4HANA migration cockpit

You are using the pre-delivered SAP S/4HANA Data migration content without any modifications

Solution

The TCI included with this SAP Note fixes the following issues listed in the table below. For detailed description on the issues see the linked SAP Notes.

Technical Name Type Issue SAP Note 

SIF_KOSTL_MASTER

 Migration object

The domain and search help within the mapping rule MAP_JVOTYP are incorrect. In the value mapping of the field "Joint Venture Object Type" wrong values are displayed.

 3485591

Be aware that the TCI only corrects the related objects of SAP delivered content. As a result, the generated migration objects will be updated automatically.

Note: If you have modified/copied your object, the correction is not done within your modified/copied object.

You may refer KBA 2543372 - "How to implement a Transport-based Correction Instruction".
