Symptom

During creation of a shop floor order it is not checked whether the selected shop floor routing matches the serialization settings of the finished material. Due to this missing check it is possible to have for a serialized material an order created based on a non-serialized shop floor routing and vice versa.

Other Terms

Shop Floor Item, MPE_SFI, SFI_CONF, SERNP

Reason and Prerequisites

Missing check at order creation.

SAP Note 3637652 is a prerequisite.

Solution

Apply correction instruction.
