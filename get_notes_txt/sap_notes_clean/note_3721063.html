Symptom

  • The Worker Country / Region of Origin field is not editable via "Edit Worker" or "Revisions" in the Work Order (Statement of Work) module.
  • The Work Order Revision upload does not support editing of the "Worker Country / Region of Origin" field.

Environment

Work Order (Statement of Work)

Reproducing the Issue

Tried editing the Worker Country / Region of Origin field via "Edit Worker" in the Work Order, via revise from supplier end and through Work Order Revision upload functionality. The field doesn't show up to edit.

Cause

Resolution

  1. This is a as designed behavior where the native field Worker Country / Region of Origin is not editable via "Edit Worker" or "Revisions" and that the Work Order Revision upload does not support editing of this field.
  2. The solution for this would be for the buyer to Unlock the security information for the worker. Go to worker >actions >unlock security information.
  3. Then, the supplier can navigate to worker > Actions > Edit Security Information and select the option to edit the Worker Country / Region of Origin field along with the security ID.
