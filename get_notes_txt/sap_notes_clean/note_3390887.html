Symptom

User wants to activate Fiori 2.0 theme, but add "?sap-ui-guideline=FIORI20" to url is too troublesome and no authorization to add parameters in SU3

Environment

  • SAP Customer Relationship Management (CRM)
  • SAP Enhancement Package 4 for SAP CRM 7.0 SP11 (RTC October 2018)
  • Or SAP Enhancement Package 4 for SAP CRM 7.0 SP10 with the installation of
    notes as per the prerequisites in SAP Note 2628638

Resolution

You can enable the SAP Fiori visual theme in Customizing in two different ways

  • You can assign the FIORI20 Function Profile ID to existing roles. Using this approach allows you to turn the SAP Fiori visual theme on and off on a role basis.
    1. You can assign the FIORI20 profile to a business role in SPRO->Customer Relationship Management->UI Framework- >Business Roles -> Define Business Role.
    2. Select the business role. Navigate to 'Assign Function Profiles’.
    3. Add FIORI20 profile with empty value.

 

  • By adding function type Fiori20 Profile to an existing function profile.
    1. You can assign the function type in SPRO->Customer Relationship Management->UI Framework- >UI Framework -> Technical Role Definition -> Define Function Profile.
    2. This will activate the Belize theme for all roles using this function profile
