Symptom

You are using an ABAP Test Cockpit (ATC) setup with a central ATC check system (also called reference check system). Exemptions for specific checks (e.g. "Usage of APIs") that are requested from one of the satellite systems do not apply after creation.

Other Terms

Code Inspector

Reason and Prerequisites

Due to a bug the exemptions are created with an incorrect checksum version which leads to them not being recognized during runtime.

Solution

Implement the correction instructions or apply the corresponding support package in your central ATC check system (reference check system). The exemptions in question have to be requested again after the corrections are applied.
