Symptom

In the configuration data model of the SSC REST API, three new fields are now introduced namely kbDate in the Configuration data model, length and numberDecimals in the Characteristic data model.

Other Terms

SSC (Solution Sales Configuraion), REST API, createConfiguration , endpoint, configuration, characteristic.

Solution

New additions to the configuration data model and the characteristic data model has been introduced. Field "kbDate" will hold the value of the date sent as part of the createConfigInput request to the create configuration endpoint. If not set, it will take the current date as its value.

Fields "length" and "numberDecimals" are introduced in the characteristic data model. These are fields are populated with the master data values.
