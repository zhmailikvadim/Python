Symptom

Type ahead functionality of the Material field in the app Manage Central Purchase Contracts does not return any suggestions from connected system, although the material can be found using Value Help.

Environment

  • SAP Ariba Central Procurement
  • SAP S/4HANA

Cause

The system design does not automatically determine the connected system from which the material should be fetched when a material is entered in the material field.

Resolution

Use the Material Value Help to locate the desired material:

  1. Open Material Value Help.
  2. Specify the relevant connected system ID in the filters.
  3. Search for the material.
  4. Select the material from the value help.

Keywords
