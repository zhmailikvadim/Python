Symptom

Runtime Errors BCD_BADDATA occurred in program MPV50700 when using T-CODE PA30 or PA20 to view infotype 0220. "Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental." 

Environment

  • Human Capital Management (HCM)
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP SuccessFactors Employee Central Payroll
  • SAP S/4HANA

Reproducing the Issue

  1. Feature IVWID is configured with return value "13".
  2. Execute T-CODE SM30, V_T582V is configured to use view "V507" and Dialog Module "RP_V507".
  3. Execute T-CODE PA20 to check Infotype0220, the dump occurred.

Cause

Incorrect configuration in table V_T582V or feature IVWID can cause issues. The field IVWID in PA0003 is derived from feature IVWID. The system uses the return code from PA0003-IVWID to fetch data from table V_T582V. If the view in V_T582V is V507, this view will be used to display infotype 0220. View V507 is intended for the Public Sector.

Resolution

Please check the following points:

  • Feature IVWID
    1. Execute T-CODE PE03 to edit feature IVWID and ensure it return BLANK.
    2. Run report RPUFACEVIEKN to update the field IVWID in table PA0003.

  • View V_T582V
    1. Execute SM30, edit view V_T582V.
    2. Remove the value from field View and Dialog Module.
