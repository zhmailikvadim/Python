Symptom

In MOS device change for a gas meter, the system parameter SU_P_TE341 should be read before applying a proposed value from table TE341 for measured pressure of a meter. This logic must apply in two places:

1. when a gas meter is installed for which no measured pressure is supplied in the installation reply or in EGERS

2. when simulating the measured pressure for a new gas meter for deciding if the old meter can be replaced by the new meter

Other Terms

DE_B_IM_DEV_CHANGE

Reason and Prerequisites

Measured pressure for old and new meter must be equal in order to allow device replacement. If they differ, the old meter is removed and the new one is installed.

SU_P_TE341 is used to propose the TE341 value in manual device installation (EG31). In the context of IM4G device change, the TE341 value is used automatically if there is no value in the installation reply or in EGERS.

Solution

Implement the correction with /nSNOTE.
