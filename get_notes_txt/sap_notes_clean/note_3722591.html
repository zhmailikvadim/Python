Symptom

  • Links within PDF documents hosted in Work Zone are not clickable in the document viewer.
  • Downloaded documents retain link functionality, but links are missing in the preview.

Environment

  • SAP Build Work Zone, Advanced Edition
  • SAP SuccessFactors Work Zone

Reproducing the Issue

  1. Login to Work Zone.
  2. Navigate to any workspace.
  3. Upload any PDF document with a link it.
  4. Open the PDF in the document viewer.
  5. Observe that the highlighted link is not clickable within the viewer.

Cause

By default, the PDF preview in SAP Build Work Zone renders the document as an image. In this rendering mode, text cannot be selected and embedded hyperlinks are not clickable. This is expected system behavior when the PDF Viewer feature is not enabled.

Resolution

The feature “Preview PDF documents using PDF viewer” can be enabled in the Admin Console under Feature Enablement → Features. Once enabled, PDFs are rendered using the PDF viewer, allowing inline viewing with clickable links.
