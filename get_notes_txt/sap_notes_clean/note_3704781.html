Symptom

  • Central repository checkout fails.
  • Error message: "ATTEMP TO INSERT DUPLICATE KEY ROW IN OBJECT AL_LANG WITH UNIQUE INDEX AL_LANG_INDX1".

Environment

SAP Data Services

Reproducing the Issue

  1. Open Designer.
  2. In central repository, select the project.
  3. Select the checkout option.
  4. Observe the error message: "ATTEMP TO INSERT DUPLICATE KEY ROW IN OBJECT AL_LANG WITH UNIQUE INDEX AL_LANG_INDX1".

Cause

Resolution

  1. Delete the issue project in the Central repository.
  2. Re-add the project into the Central repository.
  3. Attempt to check out the project again.
