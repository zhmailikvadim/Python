Symptom

  • Message SOCM_ACTION_LOG 110 doesn't have the information of original transport request for transport of copy (ToC);
  • The ToC # is not correctly showing in text log after 7.2 SP09 upgrade;
  • Change document text log no longer contains transport of copy id but rather the Transport Request ID: Transport of Copies <Transport Request ID>;

Other Terms

COPY_ALL_ENH, Transport of Copy, ToC, Text Log.

Reason and Prerequisites

Imporvement for message texts.

Solution

Implement the attached corrections into your SAP Solution Manager system by using the SAP Note Assistant (transaction SNOTE), or import the relevant support package stack.
