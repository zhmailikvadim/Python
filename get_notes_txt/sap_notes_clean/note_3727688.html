Symptom

This note delivers ABAP enhancements to notfication, equipment and functional location handlers to support presync functionality, when enabled in configuration.

Other Terms

S4MOBADDON-8364, /MERP/CL_PM_NOTIFICATION_OD, /MERP/CL_PM_FUNCLOC1_OD, /MERP/CL_PM_EQUIPMENT1_OD, Pre Sync ABAP Objects, Optimised Delta Sync

Reason and Prerequisites

Reason: ABAP enhancements required to support pre sync functionality.

Prerequisite: Prior to installing this note, please ensure the Mobile Foundation layer ABAP objects SAP note 3581875 is installated in the system.

Solution

Please apply this note in SAP system.
