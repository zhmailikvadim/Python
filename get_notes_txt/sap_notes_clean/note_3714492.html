Symptom

Enabled Latest People Profile in "Company System and Logo Settings" as well as 'Onboarding General Settings'. As an employee, the profile is coming in latest people profile. But in PDC step, as an onboardee, the screen is not showing the latest people profile. 

Environment

SAP SuccessFactors Onboarding

Reproducing the Issue

Login as a external onboardee and see the submitted portlets not showing as per the Latest People Profile configuration.

Cause

we should not associate any New Hire template and Business Rule that is used to assign new hire template.

Resolution

1. Please note that for the latest PPX read-only page to appear, no New Hire template should be associated.

2. Additionally, the Business Rule should not be used to assign a New Hire template.

3. Kindly remove both the Business Rule and the associated New Hire template. Once removed, you should be able to view the desired page as expected.

Keywords
