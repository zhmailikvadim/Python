Symptom

  • Error encountered during BP activation in J58 (J61 BB).
  • Activation of customizing object V_T079M_M failed with error: "Purchasing organization 0001 is not defined".
  • Entry XXXXX was not copied from V_T079M_M of the Business Configuration Set.

Environment

  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Navigate to transaction /n/smb/bbi.
  2. Select the solution and click on "Implement Assistant", then activate.
  3. Observe the error message indicating the issue with BP activation. (e.g. under J61 SIMG_CFMENUOLMSOMFK)

Cause

  • Missing configuration data for purchasing organization in the target client.

Resolution

  1. Transport the missing purchasing organization data from client 000 to the target client.
  2. Ensure that the baseline customizing is copied during client setup.
  3. If the configuration is not required, skip the step or change the status to green and proceed with activation.
