Symptom

This cumulative patch 2603 for UR-c2 collects several Unified Rendering bug fixes that were reported since the last cumulative patch 2602.

Please be aware that this note cannot be implemented. In order to apply the latest UR patches, you need to install the corresponding UI technology patch. Please see instructions below under "Solution".

Please see the complete list of issues that has been fixed:

  • Carousel: Fixed an issue where the control is incorrectly displayed in Right-to-Left (RTL) mode.
  • ClientInspector: Fixed an issue where ClientInspector failed to retrieve data for custom themes when the ThemeFallbackID did not match the base SAP standard theme.
  • DatePickerSlider: Fixed an issue where the time stamp past 12 pm in the 12 hour format cannot be conveniently entered.
  • FileUpload: Upload via fetch
  • Icon: Fixed an issue where (SVG) icons are not shown for visual theme Belize.
  • Page: Fixed an issue where an error page was briefly displayed during navigation before the target application loaded.
  • PopupMenu: Fixed an issue where sub-menus either failed to open or appeared empty in specific SAP GUI for HTML screens.
  • PopupWindow: Fixed an issue where sending invalid semantic events caused Web Dynpro Abap to crash during the opening or closing of popups.
  • ThemingPreviews: Fixed an issue where UR previews appeared blank during the initial load.

Other Terms

Web Dynpro ABAP, WDA, SAP GUI for HTML, Unified Rendering, UnifiedRendering, UR, UR-c2

Reason and Prerequisites

Using WD ABAP starting from SAP_UI 7.57, and SAP GUI for HTML from kernel release 7.85. For WD ABAP 7.00 and 7.01 or SAP_UI version < 7.58, and SAP GUI for HTML with kernel releases < 7.85 a different process is in place.
