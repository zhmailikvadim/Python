Symptom

Can I use one Purchasing Unit to connect both SAP ECC and SAP S4HANA in Managed Gateway For Spend & Network during a phased migration?

Environment

Managed Gateway For Spend & Network Portal
S4Hana
ECC

Resolution

Simultaneous ERP connections are supported. However, you must separate integration by Purchasing Unit and project.

Standard Architecture

  • If you use Purchasing Unit PU1 for SAP ECC, create a project connected to SAP ECC.

  • If you use Purchasing Unit PU2 for SAP S4HANA, create a separate project connected to SAP S4HANA.

  • You cannot use PU1 for both SAP ECC and SAP S4HANA.

Required Setup During Phased Migration

  1. Keep the existing project connected to SAP ECC for PU1.

  2. Create a new project in Managed Gateway For Spend & Network for SAP S4HANA.

  3. Use a separate Purchasing Unit for SAP S4HANA transactions.

  4. Deploy both projects.

This ensures correct backend routing and aligns with standard Managed Gateway For Spend & Network architecture.
