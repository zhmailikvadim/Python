Symptom

Upon checking the FPS report (RPCFPSG0), there is no radio button or check box of the Final Submission that note 3678330 - PY-GB: Final submissions on FPS do not display XML correctly

is referring to.  

The note says: "When you run the FPS report (RPCFPSG0) with the "Final Submission" flag ticked, the XML displayed shows a display error. " - XML Error "Error on line 1" 

Environment

  • Human Capital Management (HCM)
  • SAP R/3
  • SAP R/3 Enterprise 4.7
  • SAP ERP Core Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP SuccessFactors Employee Central integration to SAP Business
  • SAP SuccessFactors Employee Central Payroll

Resolution

 The final submission field is only displayed if you enter the last period of the period.. e.g. if you enter 12 for a payroll area that is monthly, then the field is displayed. 

Keywords

3678330,  RPCFPSG0, FPS, Final Submission, XML Error "Error on line 1"
