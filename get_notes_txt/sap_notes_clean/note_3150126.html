Symptom

You use the Run Compliance Reports app to create the VAT Declaration for Singapore (report category SG_VAT_DCL, SG_VAT_F5). You would like to see one of the following fields in Analyze Data:

  • Reconciliation Key ﻿(﻿CAReconciliationKey﻿)﻿
  • Reference Document Type (ReferenceDocumentType)
  • ﻿Report Run Key (﻿StatryRptRun﻿)

Other Terms

C_SG_STRPTAXRETURNQUERY, I_SG_STRPTAXRETURNCUBE, SG_VAT_DCL, SG_VAT_F5, ACR, CAReconciliationKey, ReferenceDocumentType, StatryRptRun, Document and Reporting Compliance

Reason and Prerequisites

 This SAP Note is relevant only for Singapore.

Solution

The field CAReconciliationKey﻿, ReferenceDocumentType and StatryRptRun was added to the CDS views ﻿﻿C_SG_STRPTAXRETURNQUERY and ﻿﻿I_SG_STRPTAXRETURNCUBE.

Version Date Reason
1 15.02.2022 Released for all customers
