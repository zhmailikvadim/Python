Symptom

In Include LCOKO1FAB syntax error "The data object 'GC_MSG_NMBR' does not have a component called 'NO_VALID_OPER_IN_SFR'" is shown. Activation of the Include is therefore not possible and short dumps may occur.

Other Terms

MPE, CO01, CO02, CO03, SAPLCOKO1, Runtime error, IF_MPE_EXEC_SFO_C

Reason and Prerequisites

Program error. The issue occurs after implementing SAP Note 3372771.

Solution

Apply correction instruction.
