Symptom

In the “Custom Analytical Queries and Services” app, the hierarchy enabled dimensions do not show "Hierarchy Node" selection type in the Filter tab, unlike "Customer Analytical Queries" app.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Open the "Custom Analytical Queries and Services" app.
  2. Create a new query.
  3. Enable hierarchy for a dimension.
  4. Go to the Filter tab, the "Hierarchy Node" selection type not available. 

Cause

Resolution

To apply a hierarchy node filter to a dimension in the "Custom Analytical Queries and Services" app, a parameter must be used. 

Follow these steps to enable the hierarchy node selection type:

  1. In the Model tab, enable hierarchy for the required dimension.
  2. In the Parameter tab, create a parameter for that dimension.
  3. Navigate to the Filter tab, for the hierarchy enabled dimension, select "Filter with Fixed Values" and add the created parameter.
  4. Return to the Parameters tab, select the parameter created in Step 2.
  5. The option "Filter with Hierarchy Node" will now be available for selection in the Parameters tab.
