Symptom

You are in travel expenses application with MOREI (01) German - in PR05 or in Web DynPro, you have entered only one destination which is the main destination. When you want to display the simulation or results in PDF, this destination does not appeard. In German the main destination has to be displayed by default because of a legal requirement.

Other Terms

PTRV_EXPENSE_FORM

LPTRA_WEB_MAINU05

PTRA_WEB_EXPENSE_FORM_DATA_GET

PR05

Reason and Prerequisites

There is a program error.

Solution

Implement the correction instructions or apply the corresponding support package.
