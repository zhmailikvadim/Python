Symptom

You are using the 4 weekly payroll frequency and your payroll calendar is not in line with the one used by the tax office.

If you run the report Crisis Tax Calculation (RPCLATN0), the Kenteken 710 is created for the wrong FOR period.

Other Terms

Pseudo Final Levy High Wage

Reason and Prerequisites

You are running the payroll on a 4 weekly bases.

31.3.2014 is not in your payroll period 03.

Solution

Import the HR Support Package specified for your release or implement the attached correction instructions manually.

As a general rule, SAP recommends that you install a solution by applying a Support Package. However, if you need to install a solution earlier, use the Note Assistant to implement the correction instruction.
