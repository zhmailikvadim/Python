Symptom

This KBA describes how to manually change Social Security Allowance capping in Thailand payroll solution.

Environment

  • Human Capital Management (HCM)
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP SuccessFactors Employee Central Payroll
  • SAP S/4 HANA

Resolution

According to SAP Note 3691207 - LC-TH: Social Security Contribution Wage Base Change 2026, the maximum social security contribute wage base has been increased to 17,500THB in 2026. The social security allowances should be updated to 17,500 * 5% *12 = 10,500THB. However, Social Security Office (SSO) decides to keep the social security allowances for 2026 at 9,000 THB. Refer to the link:https://www.prachachat.net/general/news-1952625.

If local SSO holds different requirements of the capping, follow the steps to manually to update the SS allowance amount:

  1. Go to T-Code SM30.
  2. Enter V_T511P for country grouping 26, and then click Maintain.
  3. Locate constant SOSEC and delimit according to the corresponding phase. For example, for Phase 1: January 1, 2026 - December 31, 2028:
    ConstantPayroll ConstantFromToValueCrcy
    SOSECSocial security allowance01.01.201331.12.20259,000THB
      01.01.202631.12.202810,500THB

Note that if the same happens in the rest phases, refer the same steps to manually adjust.
