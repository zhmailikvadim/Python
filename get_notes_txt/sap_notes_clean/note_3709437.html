Symptom

After M10, M20 steps, you are checking some documents in CKMB or CKM3OLD:
- not all materials are displayed correctly when compared to table MLDOC lines - fields BELNR POSNR KALNR.

Other Terms

MLDOC, BELNR, POSNR

Reason and Prerequisites

There is bug in creation of BELNR POSNR combination, which should be always unique per Material Plant (MLDOC-KALNR).

Solution

1) Implement this SAP Note.
2) Rerun migration step M10.

Alternatively test for some packages only:
3) via report FINS_MASS_DATA_TEST
or
4) via report FCMLHELP_SIMULATE_MIGR_MLDOCS
