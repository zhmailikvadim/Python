Symptom

Physical inventory document is generated for some materials even though there is no goods movement generated. 

Environment

  • Materials Management (MM)
  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA

Reproducing the Issue

When checking the result in MICN/RM07INC1, there are several materials determined under the field "a phys. inv. doc. can be generated". However, there is no goods movement for these materials. 

Cause

The issue is related to batch management. Materials that are not batch-managed are counted at the material level and are automatically included in the physical inventory document, even when no stock is available.

Resolution

It is standard system behavior.
