Symptom

  • You are using the Process Integration (PI) or Process Orchestration (PO) Enterprise Services Repository (ESR) or Integration Directory tools
  • You wish to increase the font size of the text in the applications Integration Builder or ESR

Environment

  • SAP NetWeaver
  • SAP Process Integration

Resolution

  1. Launch IB or ESB
  2. Right-click on the application icon in the task bar
  3. Right click on the option Java (TM) Web Launcher
  4. Select Properties
  5. Select Run this program in compatibility mode for and select  [ Windows 7 ]
  6. Click Change high DPI settings
  7. Select Override high DPI scaling behavior ... select System
  8. Click OK
  9. close and re-launch the IB or ESB
