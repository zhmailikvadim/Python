Symptom

User can navigate from Table view into Detail view via linkages in Table view. For example, when the user clicks content in column 'Notes' in Table view, the respective tab 'Notes' in Detail view should be displayed.

Other Terms

PPM, Project Management, Portfolio and Project Management

Reason and Prerequisites

After applying note 2200606, the navigation from linkages in Table view to tabs in Detail view works incorrectly, for example, user click the content in column 'Notes' in Table view, tab 'Basic Data' will be displayed.

Solution

Apply this note.
