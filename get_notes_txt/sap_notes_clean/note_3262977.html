Symptom

Function module LE_DELIVERY_SET_TO_BUFFER does not fill buffer GT_HANDLE_FOR_VBELN when "automatic packing using packing proposal" is switch on for Delivery Type (TVLK-AVERP = X).

Under certain circumstances, this can lead to problems in Commodity Management (index table CMM_VLOGP is not filled with Outbound Delivery just created).

Reason and Prerequisites

Correction

Solution

Implement the correction instructions of this SAP Note or import the relevant Support Package (S4OP_2022_FPS1).
