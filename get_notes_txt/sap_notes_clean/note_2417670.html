Symptom

You are using an application with a sap.ui.unified.Calendar control in a mobile device.

When swiping to navigate through the App, if the finger touches a day within the calendar, the day is selected.

Environment

  • SAPUI5
  • User Experience
  • Controls
  • sap.ui.unified.Calendar

Reproducing the Issue

  1. Create an application using sap.ui.unified.Calendar control
  2. Add enough controls in the View until you need to swipe through your app to see them all
  3. Open the App in a movile phone
  4. Swipe to navigate through your App touching the control
  5. Days are selected if swiping over them

Example:

Cause

There was a product issue and the swaping interaction was not handled properly in the control

Resolution

The issue has been fixed in the following SAPUI5 versions:

  • 1.44.5
  • 1.42.8
  • 1.38.15
  • 1.28.42
