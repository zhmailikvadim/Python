Symptom

Is it possible to default an offer letter template based on the value in the Candidate Application?

Environment

SAP SuccessFactors Recruiting Management

Reproducing the Issue

  1. Access a candidate in a job requisition to whom you are sending an Offer Letter to
  2. Take action -> Offer -> Offer Letter
  3. The Offer letter should be auto populated based on the value in the candidate application

Resolution

Currently such functionality is not available in the product.
User will have to use the drop down and select an offer template they wish to send to the candidate.  

You may submit this as an enhancement request to engage our Product Team. Please refer to KB Article: 2090228 - How to Submit Ideas for SAP SuccessFactors Products.
