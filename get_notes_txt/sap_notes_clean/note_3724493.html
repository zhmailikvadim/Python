Symptom

User group details (name, email, license attribution) are no longer displayed in Process Manager under Setup > Manage users & access rights > User groups

  • "Feature sets" are shown instead of individual user names and group membership information
  • Previously visible license attribution details (to whom and from which group) are missing from the Setup area

Resolution

The functionality for viewing detailed license and user attribution information has been moved to the SAP Signavio Process Collaboration Hub.

To access this information:

  1. Navigate to the Collaboration Hub
  2. Go to Reporting
  3. Select the License Usage Dashboard

The License Usage Dashboard provides workspace administrators with a comprehensive overview of license usage, including detailed user and group attribution information previously available in the Process Manager Setup area.

Additional dashboards available in this area:

  • Usage Dashboard – for detailed analysis of workspace usage
  • Process Metrics Dashboard – for analysis of process model usage

Additional features:

  • Apply filters to customize your view
  • Export data as CSV files
  • Share dashboards with other users

For more detailed information, refer to the official documentation: Process Model Dashboards
