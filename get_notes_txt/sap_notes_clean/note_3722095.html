Symptom

When attempting to submit an invoice, the operation fails with the error message Tax items without any related G/L or PO account items must be deleted to submit the invoice. Use the delete button on the tax row. This occurs on invoices where Line Item Matching is enabled.

Cause

Each tax entry on the invoice must be linked to a corresponding posting line (GL posting or PO posting) by matching both the tax code and tax jurisdiction. The system found at least one tax entry that either:

  • Has no TaxCodeId defined, or
  • Has no matching posting line with the same TaxCodeId and TaxJurisdiction combination.

Resolution

  1. Open the invoice and review all tax entries.
  2. For each tax entry, verify there is a corresponding GL posting or PO posting line with:
    • Matching tax code.
    • Matching tax jurisdiction.
  3. Either:
    • Add the missing posting line that should carry the orphaned tax, or
    • Remove the unlinked tax entry if it was added incorrectly.
  4. Retry the Submit operation.

Keywords
