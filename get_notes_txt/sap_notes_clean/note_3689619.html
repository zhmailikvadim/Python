Symptom

 In the process "Start of Supply for Feed-In (DSO View)" (DE_B_MPES_DSO_SOS_REG) version 2508, the business status under the process header is not getting updated.

Other Terms

Business Status, Proc Ended, Confirmed, MPES, 2508, Last Step, Start of Supply for Feed-In (DSO View)

Reason and Prerequisites

Program Error

Solution

This SAP Notes is relevant for the following role: DSO

BC Set /US4G/3_0_DSO_SP07_NOTE1_3689619 to be activated

1. After implementing the note. Run transaction SCPR20.

2. Input BC set "/US4G/3_0_DSO_SP07_NOTE1_3689619" .

3. Click "Activate" button or choose BC Set->Activate option, from Menu
