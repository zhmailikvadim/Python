Symptom

How to download SySAM 2.5.0 for PowerDesigner

Environment

  • SAP PowerDesigner 16.7 SP09 and higher
  • Sybase Software Asset Management (SySAM) 2.5.0

Resolution

  1. Access the SAP Software Downloads portal.
  2. Select SAP POWERDESIGNER 16.7 - Maintenance Product.
  3. Select SAP POWERDESIGNER 16.7.
  4. Once the screen updates, click on POWDES167009_0-80005353.ZIP to download the PowerDesigner 16.7 SP09 software.
    The downloaded file will contain the following: PowerDesigner_16.7.SP09.8174.exe, PowerDesigner_16.7.SP09.8174_DVD.zip, SQLAnywhere 17.0 SP1 PL19 For PowerDesigner 16.7 SP09.zip, and SYSAM 2.5.0.zip.
