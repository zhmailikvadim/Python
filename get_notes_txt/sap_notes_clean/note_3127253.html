Symptom

While using Output Parameter Determination, the following error occurs: In zero downtime mode BRFplus objects must not be created or changed.

Environment

S/4HANA Cloud

Reproducing the Issue

Steps:
1. Open App "Output Parameter Determination"
2. Click button "Edit" 
3. Error pops up - In zero downtime mode BRFplus objects must not be created or changed.

Cause

Due to S/4 HANA upgrade functions are not available.

Resolution

Wait for upgrade to finish or

For more details, opening a case on XX-S4C-OPR-INC or XX-S4C-OPR-SRV-UPG.

See Also

SAP Note 3110278 - Restrictions for Zero Downtime Option for BRF+

Keywords

BRF+, Workbench, Output, Parameter, Determination, Cloud
