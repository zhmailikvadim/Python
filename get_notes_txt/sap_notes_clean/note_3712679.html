Symptom

Pre-requisites and steps to enable the Supplier Profile Summary feature in SAP Ariba Supplier Lifecycle and Performance (SLP)

Resolution

Supplier Profile Summary is a self configurable feature which has to be provisioned and configured by an Administrator on customer side.

The below prerequisites are to be met or performed in the given sequence.

Your SAP Ariba realm needs to be enabled with SAP Ariba Supplier Lifecycle and Performance entitlement and should be hosted on United States or European Union data centres.

  1. Step1 - Setup Single Sign-on (SSO) between SAP Cloud Identity Services - Identity Authentication and your SAP Ariba site.
    Check if there is a tenant assigned to your customer ID by accessing the SAP Cloud Identity Services - Tenants application at https://iamtenants.accounts.cloud.sap/Information published on SAP site. For more information, see Viewing Assigned Tenants and Administrators.

    Share the blog below with your SAP Cloud Identity Services administrator or IT team managing Single Sign-on (SSO).
    Overview: SAP Ariba SSO with SAP Cloud Identity Se... - SAP Community 

  2. Step2 – SAP Ariba Supplier Management to be provisioned in SAP for ME by Ariba Cloud administrator.
    Procedure for provisioning the SAP Ariba Supplier Management Entitlement in SAP for Me

  3. Step3 – Carryout post provisioning Single Sign-On (SSO) configurations if you're using a corporate identity provider with identity federation through SAP Cloud Identity Services - Identity Authentication. If you're using Identity Authentication as your identity provider, you can skip these steps.
    Completing Post-Provisioning Steps for Identity Federation with a Corporate Identity Provider 

  4. Step4 - Enable the feature is SM Administration > Configuration Parameters.
    Procedure for enabling the feature toggles
