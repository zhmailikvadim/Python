Symptom

'500 Connection Timed Out' error message after trying to add a proxy interface in the User-Defined Search Soamanager's feature. A short dump similar to the one below can be found in the ST22:

Category               ABAP programming error
Runtime Errors         TIME_OUT
Web Dynpro Component   C_XPATH_TREE
ABAP: Program          /1BCWDY/3NBNVOZSMB36F7UAU6VM==CP
Application Component  BC-ESI-WS-ABA-RT
Date and Time          31.01.2026 00:00:00

Environment

  • ABAP Web Services
  • SAP NetWeaver
  • SAP NetWeaver Application Server for SAP S/4HANA
  • ABAP PLATFORM - Application Server ABAP

Cause

The proxy object to be added it too big and it is not possible to use large proxy objects for the User Defined Search. This is a limitation.

Resolution

User Defined Search is only possible for small proxy objects.
