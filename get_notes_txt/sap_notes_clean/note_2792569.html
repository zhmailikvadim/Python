Symptom

According to present legislation, such as GDPR, data subjects have the right to get information regarding their personal data undergoing processing.

The Information Retrieval Framework (IRF) allows you to search for and retrieve personal data of a specified data subject. The search results are displayed in a comprehensive and structured way. To use the IRF, customers must set up a data model.

You can find further details about IRF in the SAP Help Portal (https://help.sap.com) by searching for Information Retrieval Framework.

Other Terms

Data Protection and Privacy, GDPR, Information Retrieval Framework, IRF, SAP note 2743548, LOY_MSH

Reason and Prerequisites

Business Function CA_DTINF_FW must be activated in your system.

Solution

BAdI implementations are used to adjust relationships between tables when an IRF model for a certain ILM object is created. With this note we deliver implementations of BAdI BADI_DTINF_ILM_OBJ_TABLES for the ILM objects listed below. You will find further information about the BAdI itself in SAP note 2743548.

In certain cases, it is necessary to influence the behavior of the framework during data collection when linking two tables. This is done by assigning a function module to such a link in a certain method of the BAdI.
