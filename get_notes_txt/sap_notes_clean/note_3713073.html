Symptom

You are a customer of Group reporting and want to schedule automatic postings via the fiori application F5572 Schedule Jobs for Consolidation Tasks.

This template is not selectable.

Other Terms

F5572, SAP_FIN_CS_AUTOMATIC_POSTING, Intent, LADI, Launch App Item descriptor

Reason and Prerequisites

Missing template in launchpad item descriptor.

Solution

Implement the correction instruction of this SAP Note or install the corresponding support package.
