Symptom

In the Transportation Cockpit, the screen content is only partially translated into the user's logon language. Some field names are displayed in English. 

Environment

S/4HANA 2023

Cause

Program error.

Resolution

To solve the issue, implement Note 3587358 - FC: Field HEADER_TEXT is filled incorrectly in data base table /SCMTMS/D_FCA_C.

Keywords

TC, FC, Field Catalog, language, translate, column, field
