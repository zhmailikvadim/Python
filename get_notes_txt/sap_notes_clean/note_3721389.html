Symptom

During the configuration of SAP Service and Asset Manager, the following OMDOs (oData Mobile Data Objects) do not have any associated handlers:

  • SAMxxxx_WORK_ORDER_GENERIC_EAM
  • SAMxxxx_PHASE_CONTROL_CODE
  • SAMxxxx_WORK_ORDER_PHASE_CONTROL

Environment

  • SAP Service and Asset Manager (SSAM)
  • S/4 HANA 2021 (S4FND 106, S4CORE 106, and IS-UT 806) 

Cause

System version does not support EAM functionality.

Resolution

For lower release S/4 systems, certain features may not be available, which can result in missing class errors in the configuration panel. This is expected behavior and should not impact synchronization, provided that the feature is deactivated at the application level.

When upgrading to a newer S/4 release, the system will automatically adapt to include the missing classes, resolving these errors without manual intervention.
