Symptom

When collect planviz, it prompts multiple bind values and you are struggle with inputting them one by one.

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental." 

Environment

  • SAP HANA database 1.0 
  • SAP HANA database 2.0 
  • SAP HANA Studio

Reproducing the Issue

  1. collect planviz
  2. mutilple bind values are requested

Resolution

  1. To input all bind values at once, right-click at the blank area and there is a "Add Parameter Values " button.
  2. Summarize all parameters and use "," to separate every parameter .
  3. After press ok, all parameters will be inputted at once.  If you are using DBACOCKPIT to collect planviz, you can refer to 3441557.
