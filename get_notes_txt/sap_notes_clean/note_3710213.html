Symptom

You are processing an inbound delivery in RF Receiving of Handling Units. The delivery item has already been packed and putawayed. Although an overdelivery tolerance is defined that should allow receiving additional quantities, the system issues the error message "E: Delivery quantity exceeded".

Other Terms

REHU, unloading, overdelivery, tolerance, item finished, /SCWM/RF_EN491, /SCWM/RF_EN 491

Reason and Prerequisites

Code error.

Solution

Implement the attached code changes or import the corresponding support package.
