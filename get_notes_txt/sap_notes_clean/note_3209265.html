Symptom

To activate your licenses on an on-premise license server in offline mode.

Environment

Tricentis License Server

Resolution

This chapter explains how to activate your licenses on an on-premise license server in offline mode. To administer your licenses, you must have Tosca Commander installed on a machine that has a valid connection to the license server. Check the following Tricentis documentation for more details:
