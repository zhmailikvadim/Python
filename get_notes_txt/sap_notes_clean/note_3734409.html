Symptom

You notice that the create button (+) is grayed out in the MTA Provider Configuration under the Email Settings of your SAP Service Cloud Version 2 or SAP Enterprise Service Management environment.

Environment

SAP Sales and Service Cloud Version 2.

SAP Enterprise Service Management.

Reproducing the Issue

  1. Click on the User Profile.
  2. Click System Settings.
  3. Navigate to All Settings.
  4. Click MTA Provider Configuration.
  5. See the + is grayed out.

Cause

The Provider Type within the MTA Configuration is maintained as Default instead of SMTP.

Resolution

To enable the create (+) button under the MTA Provider Configuration settings, it is required to change the Provider Type within the MTA Configuration and maintain it as SMTP.

Once this is done, the button will become available to be used.
