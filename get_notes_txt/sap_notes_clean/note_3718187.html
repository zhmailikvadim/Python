Symptom

  • Global Assignment Details portlet is not visible in Configure Latest People Profile after migration from Legacy People Profile;
  • The migration job result indicates: "The configuration is not applicable in the latest People Profile."

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental." 

Environment

SAP SuccessFactors Employee Central

Reproducing the Issue

  1. Navigate to Company System and Logo Settings;
  2. Click on “Migrate Legacy People Profile Configuration.”;
  3. Go to Scheduled Job Manager, open Job Details, and download the results file;
  4. Open the generated .csv file and verify that the Global Assignment Details portlet was not migrated. The message “The configuration is not applicable in the latest People Profile.” is displayed;
  5. Navigate to Configure Latest People Profile > Job Data;
  6. Confirm that the Global Assignment Details portlet is not displayed there.

Cause

Global Assignment Details and Employment Details share the same portlet space.

The system automatically toggles visibility based on the employment type: Global Assignments display Global Assignment Details, while all other employment types display Employment Details.

Resolution

Expected behavior.

In Configure Latest People Profile, only Employment Details portlet is visible. Global Assignment Details replaces Employment Details in Global Assignment employments.
