Symptom

Before the 2602 release, users could maintain profitability segment attibutes for non-billing WBS elements via the Settlement Rule button, in either the Project Control - Enterprise Projects or the WBS Element app.

With the 2602 release, we will introduce the new UI approach to create profitability segment attributes for billing WBS elements. Since profitability segment attibutes were only to be assigned for those WBS elements that are marked as billing elements, the initial apprach was to delete profitability segment attributes maintained for WBS elements that are not marked as billing elements.

Other Terms

Maintain settlement rules

Reason and Prerequisites

The profitability segment attributes also for those WBS elements that are not marked as billing elements should not be deleted when customer systems are upgraded to 2602.

Solution

We will deliver a fix to prevent the deletion of the profitability segment attributes for non-billing WBS elements for the production system upgrade.
