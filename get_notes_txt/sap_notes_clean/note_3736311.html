Symptom

  • In Transaction /AIF/CONTENT_COMP, error 'The AIF-Interface /MMEDI/INVREQIN/001 does not belong to the deployment scenario SAP_COM_0224' is raised
  • Error message no. /AIF/ICD_CONTENT 042 is raised while following the steps provided in the note 2787264

Environment

  • SAP S/4HANA 1809 release and higher

Reproducing the Issue

  1. Goto Transaction /AIF/CONTENT_COMP
  2. Enter these details:
    Deployment Scenario ID : SAP_COM_0224
    Workbench Request : Transport Request number
    Namespace : /MMEDI
    Interface Name : INVREQIN
  3. Execute and error is raised

Cause

As mentioned in the attachment "AIF Content activation.docx" in the note 2787264, AIF content has to be activated only if the content is not already available in the system.

Resolution

Please follow the Step 2 provided in the "AIF Content activation.docx" from the note 2787264, only when the Step 1 was not successful. In case, content is not already available in the system as per the Step 1, then please execute Step 2 using which it can transport the Compressed Content from the Development System.

If Content is already available in the system as per the Step 1, then Step 2 can be skipped and proceed with the subsequent steps.
