Symptom

Time Account Balance appears on the Global Assignments Details card in Latest People Profile.

Environment

  • SAP SuccessFactors HCM Suite
  • SAP SuccessFactors Employee Central

Reproducing the Issue

  1. Go to Job Data
  2. Global Assignment Details card
  3. View All
  4. See the Time Account Balance is displayed as part of the Global Assignment information.

Cause

This is working as designed.

Resolution

The Time Account Balance section displays all time accounts that are valid as of the global assignment end date. All Time Accounts open/in progress/valid until the assignment end date will appear in this section. 

For example, if the employee's assignment end date is on 01/01/2035 and only one account is valid on that date (for example “Time in Lieu Of” account, which remains valid through the year 9999) is displayed in the Global Assignment card in the Time Account Balance section. If any other accounts (e.g., recurring time accounts in the year 2034/2035)  are valid on the assignment end date, they will also appear in this section.
