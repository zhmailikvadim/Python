Symptom

You execute the payroll for employees from the construction industry for whom the prerequisites for an equalization calculation are met in accordance with SAP Note 2699917. When determining the number of working days, the system does not take into account days on which the employee worked and had no lost time due to seasonal reduced working hours.

Other Terms

RPCALCD0, double taxation convention DTC, TEEA due to employment abroad, payroll function DST AT APER, class CL_HRPAYDEST_DBA_AUSGLEICH, BAU main construction industry, SKUG seasonal reduced hours compensation

Reason and Prerequisites

In the construction industry, the standard working times are recorded in table ZT. These must not be taken into account as a non-working day when evaluating the DBA working days.

Solution

The check whether a working day exists is enhanced with the exception of the standard working time.

Implement the attached correction instructions or import the specified support package.

After you import the HR Support Package or implement the correction instructions, you must perform the following activities.

If you have an employee from the main construction industry who falls under a double taxation convention and receives seasonal reduced hours compensation, run payroll again for this employee in December.
