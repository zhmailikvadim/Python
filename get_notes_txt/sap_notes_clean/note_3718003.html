Symptom

Interest calculation with FINT or FINTAP uses wrong document type and hence wrong number range when posting interest documents.

Other Terms

T033E-BLART T033F-ATRR1

Reason and Prerequisites

You have configured document types in the Posting Specifications (T033F-ATRR1), because you do not want to post all interest documents with the same document type (T033E-BLART). This functionality was first provided by note 3135521, which contains an error.

Solution

Repair the source code.
