Symptom

While scheduling application job via external scheduler Redwood, error message "APJ_RT/106: You cannot schedule the job for user 'XXX'" appears in the log and the job cannot be scheduled.

Environment

SAP Integrated Business Planning

Reproducing the Issue

  1. Ensure communication system, communication user and communication arrangement have been prepared correctly.
  2. Create business user "XXX (XXX is User ID)" that is expected to schedule jobs in IBP from the external scheduler Redwood.
  3. Attempt to execute jobs from external scheduler Redwood.
  4. Observe the error message "APJ_RT/106: You cannot schedule the job for user 'XXX'" in the process log.

Cause

Possible causes:

  • Business user doesn't have required business catalog: SAP_CORE_BC_APJ_JCE
  • User ID "XXX" in the request in Redwood does not match the User ID in IBP system. 

Resolution

User need to check if the business user has required business catalog SAP_CORE_BC_APJ_JCE assigned to its business role.

Additionally, if the business catalog is assigned and the error message still appears, user need to compare the User ID in both IBP and the request in Redwood to make sure the User ID is consistent.
