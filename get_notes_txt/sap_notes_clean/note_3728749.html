Symptom

After the implementation of a UDF related SAP Note, it is observed that some related HANA objects have become inactive. 

Environment

SAP Customer Activity application bundle - Unified Demand Forecasting

Cause

This can be caused by an incorrect activation sequence of HANA objects during the SAP Note implementation process. 
This behaviour has been linked to previous versions of the SNOTE transaction, which did not consistently enforce the required activation order.

Resolution

As a general best practice, it is recommended that the SNOTE is kept up to date.

To prevent and resolve, please ensure that the following SAP Notes are implemented in your system or upgrade the SNOTE to the latest version:

3653531 - Syntax Error while Implementing SAP Note containing AMDP Class Methods
3654552 - Recalculate EDITORDER After Adding a Parameter via SAP Note

See Also

3408293 - UDF related HANA Object not deployed.
