Symptom

Symptom

You notice an incorrect minimum wage in your employee's payslip. The symptom occurs especially if you have a split in the Planned Working Time (0007) infotype in the given payroll period. In this case, the system considers only the most recent entry in the infotype.

Other Terms

Minimumloon, IT0007, NLWAS

Solution

After implementing this SAP Note, the system will consider all relevant entries in the Planned Working Time (0007) infotype within the payroll period. Please notice that the decision of using this infotype for the calculation of minimum wage is determined by the Sequence for Determining Number Worked Hours for Minimum Wage (NLMWH) feature.

Delivery

Import the HR Support Package relevant for your release. For more information, see SAP ERP Human Capital Management Support Package Schedules or SAP S4 HCM Support Package Schedules.

SAP recommends that you implement the Support Package. However, if you need to install a solution earlier, use the SAP Note Assistant.



Other Terms
