Symptom

Based on the feedback from LPM and customers, Field 20 must be filled in when running report HGCYTX0 in Taxable Income mode

Additionally, in case of tax advance, some values were converted to EUR, which is incorrect.

Other Terms

Taxable Income; Tax; HBGCYTX0

Reason and Prerequisites

This SAP Note is only relevant for Bulgaria (HRBG)

Solution

Report HBGCYTX0 has been adjusted to also fill in field 20 in case of taxable income.

Report HBGCYTX0 has been adjusted to not convert some values in specific cases(Specifically tax advance) in 2025 report.

Implementation

As a general rule SAP recommends that you implement the Support Package specified for your release. However, if you need to install a solution earlier, use the SAP Note Assistant to implement the correction instruction. You can find more information about the SAP Note Assistant in SAP Support Portal, under SAP Note Assistant.

Versions

Version 1 (26.1.2026) - Initial release, Report HBGCYTX0 has been adjusted to also fill in field 20 in case of taxable income.

Version 2 (27.1.2026) - Report HBGCYTX0 has been adjusted to not convert some values in specific cases(Specifically tax advance) in 2025 report.
