Symptom

You use transaction RESRMO to change the results of a sales-based settlement.

In the list of selected sales-based settlement results, you right-click a row.

In the context menu that is called with the right mouse button, the "Spreadsheet" option is not displayed.

It is impossible to export the table.

Other Terms

Sales-based rent, sales-based settlement, sales-based rent settlement, RESRMO, sales-based settlement results, ALV export, Excel, spreadsheet

Reason and Prerequisites

As of S4CORE Release 108, this function is no longer available in the ALV standard.

Solution

Instead of a context menu that is called with the right mouse button, the "Export" button is now displayed in the application toolbar. This SAP Note adds this standard ALV function to the application toolbar. Implement the attached correction instruction for your release or import the corresponding support package.
