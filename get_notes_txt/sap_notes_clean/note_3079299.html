Symptom

  • Story keeps loading data in explorer views after changing chart to another chart type or to a table in SAP Analytics Cloud (SAC)



Environment

  • SAP Analytics Cloud (Enterprise) 2021.7.7

Reproducing the Issue

  1. Open the problem story
  2. Go to explorer view
  3. Go to the problem page
  4. Change chart type to other chart type or table
    => Note the story hangs, it keeps loading data

Cause

The issue is under investigation by developer group.

Resolution

The issue is under investigation by developer group.
