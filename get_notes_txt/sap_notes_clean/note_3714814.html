Symptom

  • The header in report transactions PT62 and PT64 is missing, resulting in an empty report output header. 
  • The issue occurs when starting the report in PT62 or PT64 with any variant. 
  • No error message or short dump is generated during the transaction. 
  • The data output is accurate, but the header area is blank. 
  • The Day Balances screen in SAP Time Management (transaction code PT_BAL00) is experiencing an issue where the header section appears blank. Previously, this section displayed Day Balances, the selected date range, and the number of selected employees, but this information is no longer visible.

Environment

  • Human Capital Management (HCM)
  • SAP R/3
  • SAP ERP Core Component
  • SAP ERP
  • SAP enhancement package for SAP ERP

Reproducing the Issue

  1. Start transaction or report. 
  2. Select any variant. 
  3. Run the transaction or the report.

Cause

Issue is not caused by any SAP Time Management reports and transactions

Resolution

  1. Ensure that in SAP GUI Logon options under "Interaction Design" > "Control Settings" > "HTML Control", Internet Explorer is selected instead of Edge (based on Chromium).
  2. Refer to SAP Note 3704912 for detailed instructions on resolving the issue.
