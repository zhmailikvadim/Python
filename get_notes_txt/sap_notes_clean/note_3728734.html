Version 2 (17.03.2026): General Availability (No changes from Pilot Version)

SAP Note 3692644 (PY-GB: SSP 2026-27) introduced significant changes to SSP from April 6th 2026. One of which was that, if an employee has a zero value of Average Weekly Earnings (AWE) and the 'Transitional Protection' rules do not apply, then the weekly payment rate should be zero. But that rule was not being applied correctly.

This SAP Note corrects this issue.

Other Terms

GBSXP, SSP, AWE

Solution

EITHER

Apply the relevant HRSP

OR

Apply the correction instructions attached to this SAP Note, via SNOTE.
