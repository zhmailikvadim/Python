Symptom

  • The network team has raised a VAPT (Vulnerability Assessment and Penetration Testing) concern regarding MongoDB.
  • MongoDB has not been installed, configured, or used in the SAP BusinessObjects environment.
    However, MongoDB-related drivers or binaries are still present on the server.

Environment

  • SAP BusinessObjects Business Intelligence platform 4.x, BI 2025
  • All supported OS

Resolution

  1. The SAP BO installer includes an InstallCache that stores binaries for all features to support future operations like modify or repair installations. Even unselected features like MongoDB are cached by design, but this cache is not used in production workflows and does not pose a security risk since the feature isn't active.
  2. To partially remove intermediate cache versions, use the SAP BusinessObjects ONE Uninstaller. For complete removal, perform a fresh installation on a new system with the latest version.

Keywords
