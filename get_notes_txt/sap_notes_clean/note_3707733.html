Symptom

If the tranche's end date is open-ended, then under certain circumstances the start date of the drawdown will not be checked for validity, since neither the start date (fulfilled) nor the end date (not fulfilled) of the tranche may be initial.

Reason and Prerequisites

You have configured your system so that it is also possible to maintain an open-ended option for the CML contracts.

Solution

Implement the attached Correction Instruction (CI) or the relevant Support Package.
