Symptom

After the latest GUI update, when reports such as FBL1N, FBL5N, FBL3N and FAGLL03 are executed are seen with Missing headers including Supplier Name, Customer Name, and Company Code and the implementation of SAP Note 3514409 is not valid for this scenario.

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Environment

  • SAP S/4HANA

Reproducing the Issue

  1. Execute FBL*N report.
  2. Observe that header information, including Supplier Name, Customer Name, and Company Code, is missing.

Cause

  • The issue is caused by the latest Webview2 Version 144.xxx, which leads to problems with SAP GUI HTML control when using Edge as the browser control in SAP GUI for Windows.

Resolution

  1. Refer to KBA 0003704912 "Latest Webview2 Version 144.xxx causes problems with SAP GUI HTML control when using edge as browser control in SAP GUI for Windows".
  2. Follow the solution, Use either webview2 runtime version 143.xxx or switch the browser control to "Internet Explorer". 

  3. And keep a track of KBA - 0003704912 for further updates.
