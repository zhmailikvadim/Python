Symptom

When adding a new product hierarchy sub-node, the following error message is received: "Please correct your inputs. Value & already exists in this hierarchy. Specify a different value."

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Open Manage Product Hierarchies app.
  2. Open a Product Hierarchy.
  3. Navigate to a node.
  4. Click on "Add".
  5. Search and select a value that already exists in the product hierarchy.
  6. Click "Save".
  7. Error message "Please correct your inputs. Value & already exists in this hierarchy. Specify a different value." appears.

Cause

This is the standard system behavior. 

In the product hierarchy, a node can only be assigned (or be part) once within a single hierarchy structure. 

The system does not support maintaining multiple occurrences of the same node or subnode within the same product hierarchy.

As a result, duplicate entries of identical nodes/subnodes in one hierarchy are not allowed

Resolution

As a workaround, it may be useful to maintain the same node in multiple product hierarchies.

Creation of a new product hierarchy can be done through following steps:

  1. Open Manage Product Hierarchies app.
  2. On the Hierarchy List, click on "Create" (top right).
  3. Fill mandatory fields.
  4. Open the newly created Product Hierarchy.
  5. Maintain the desired new hierarchy with the same value as other product hierarchies.
  6. Save and activate such hierarchy.
