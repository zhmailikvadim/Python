Symptom

The wrong HIE-ID is used for hierarchy flat file loading.

Other Terms

SDL, HIER, HIE-ID, HIEID, flat file, loading, hierarchies, InfoPackage, A version, T version

Reason and Prerequisites

Program error

Solution


  • SAP BW 7.30
    Import Support Package 19 for SAP BW 7.30 (SAPKW73019) into your BW system. The support package is available once SAP Note 2554973 ("SAPBWNews NW BW 7.30 ABAP SP19"), which describes this SP in more detail, has been released for customers.

  • SAP BW 7.31 (SAP NW BW 7.3 EHP 1)
    Import Support Package 24 for SAP BW 7.31 (SAPKW73124) into your BW system. The support package is available once SAP Note 2653359 ("SAPBWNews NW BW 7.31/7.03 ABAP SP24"), which describes this SP in more detail, has been released for customers.

  • SAP BW 7.40
    Import Support Package 21 for SAP BW 7.40 (SAPKW74021) into your BW system. The support package is available once SAP Note 2653455 ("SAPBWNews NW BW 7.40 ABAP SP21"), which describes this SP in more detail, has been released for customers.

  • SAP BW 7.50
    Import Support Package 14 for SAP BW 7.50 (SAPK-75014INSAPBW) into your BW system. The support package is available once SAP Note 2668631 ("SAPBWNews NW BW 7.50 ABAP SP14"), which describes this SP in more detail, has been released for customers.

  • SAP BW 7.51
    Import Support Package 8 for SAP BW 7.51 (SAPK-75108INSAPBW) into your BW system. The support package is available once SAP Note 2671143 ("SAPBWNews NW BW 7.51 ABAP SP8"), which describes this SP in more detail, has been released for customers.

  • SAP BW 7.52
    Import Support Package 4 for SAP BW 7.52 (SAPK-75204INSAPBW) into your BW system. The support package is available once SAP Note 2689693 ("SAPBWNews NW BW 7.52 ABAP SP4"), which describes this SP in more detail, has been released for customers.

  • SAP BW 7.53
    Import Support Package 1 for SAP BW 7.53 (SAPK-75301INSAPBW) into your BW system. The support package is available once SAP Note 2653494 ("SAPBWNews NW BW 7.53 ABAP SP1"), which describes this SP in more detail, has been released for customers.

 

In urgent cases, you can implement the correction instructions as an advance correction.

You must first read SAP Notes 1668882 and 2248091, which provide information about transaction SNOTE.

To provide information in advance, the SAP Note(s) specified above might already be available before the support package is released. In this case, however, the short text of the SAP Note still contains the term "preliminary version".
