Symptom

The Custom Fields and Logic app displays an endless loading screen when accessed.
When checking the error log in Gateway, the message is 'No System Alias found for Service <> and user <>'.

Environment

SAP S/4HANA

Reproducing the Issue

Access the Custom Fields and Logic app.

Cause

  • The services APS_CUSTOM_FIELD_MAINTENANCE_SRV, APS_DATA_SOURCE_EXTENSION_SRV, and SBLE_BADI_CTX_REGISTRY_SRV were missing in the system.
  • The system alias was not maintained

Resolution

  1. Use T-Code SICF to check the activation of the services APS_CUSTOM_FIELD_MAINTENANCE_SRV, APS_DATA_SOURCE_EXTENSION_SRV, and SBLE_BADI_CTX_REGISTRY_SRV.
  2. Use T-Code /n/IWFND/MAINT_SERVICE to maintain the system alias for the services.
