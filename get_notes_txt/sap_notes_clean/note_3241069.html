Symptom

The following Public sector payroll relevant corrections are delivered in this SAP Note:

  • No SI contribution was calculated for Public sector employees for foreign currency supply if employment was terminated during foreign service and before the end of the month, this was corrected.

Hungarian description can be found attached.

Version information

Version 1: Original version

Other Terms

R/3 HR CEE, Hungary, Payroll 2022, public sector foreign currency supply

Reason and Prerequisites

The prerequisite for the correction:

HR-CEE/SAP_HR SP05/2022 - HR-CEE/SAP_HR SP09/2022 package or the relevant CLC package for Hungary.

Solution

---------------------------------------------------------

Installation steps
---------------------------------------------------------

The modifications consist of one part:
1. Correction Instructions (attached to SAP Note 3241070)

1. Apply the Correction Instructions of SAP Note 3241070 with the Note Assistant (transaction SNOTE; more information on service.sap.com/note-assistant) and execute the further implementation steps in the SAP note.

IMPORTANT!
- Log on to the requested clients in HU language and run the newly created report in Update mode (in all requested clients): RPUUDOH0_MANDT_3241070
- Make sure that the created or modified objects are all active.

The modifications will also be delivered with the HR-CEE/SAP_HR SP10/2022 package and the relevant CLC package for Hungary.
