Symptom

When first time user choose as Spreadsheet in Download option then very next screen Export As popup Screen appear and

in that screen, user not able to select information in 'Format drop down' input option like 'Google Sheets' or 'Microsoft Excel'. etc.,

 

Other Terms

 Vendor Type, File share, OData File share, Google Workspace, Google

Reason and Prerequisites

 Prerequisites note: 3220522 - For Release 757 and Software Component   SAP_BASIS

Solution

In this Export As Tab need to show as format like "GOOGLE' / MICROSOFT/ SAP "based on Repository name and find out respective Vendor Type and that vendor type information shows here.

For showing this Vendor type information in file share OData property exposed as ("FileShareVendorType": "GOOGLE").

Example:

Transaction: GUITVH (Visual Harmonization Test App) → ALV Grid Tab → In Download Option as 'Spreadsheet' → Export AS Popup window --> Format Drop Down need to select which Vendor type format on file will be downloaded like as follows.

Format (drop down input option):

1. Google Sheets

2. Microsoft Excel. etc.,


Limitation in Fileshare Layer:

  1.  When first time user press Export to button then back end UI Team will execute OData call to get all fileshare Details, so that particular time File share ABAP Layer will get respective all fileshare repository vendor name and match with Enumeration to get respective Vendor type information will be stored into table CMIS_L_FS_EXTEN  (File Share Extension).

                 UI Team make OData Call: /sap/opu/odata4/sap/ui_cmis_myfiles_04/srvd/sap/cmis_myfiles/0001/FileShares?$orderby=FileShare desc

      2.   Now fileshare Transaction V_CMIS_l_FS will be synced with fileshare Extension Table  CMIS_L_FS_EXTEN  (File Share Extension)

      3.  2nd time on-wards while press Export to button then user will get all the Fileshare information, so that UI Team can able to show format drop down input show as Google Sheet based on Vendor Type.
