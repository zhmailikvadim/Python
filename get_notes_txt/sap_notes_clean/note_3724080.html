Symptom

 In the process "Start of Supply for Electricity (New SUP View)" (DE_B_GPKE_SOS_SUP), step 8950 had an incorrect precondition with step 2000 using "OR" logic instead of "AND" logic.
This issue is relevant for the market role: SUP.

Other Terms

GPKE, DE_B_GPKE_SOS_SUP, step precondition, step 8950, step 2000, BC set, start of supply

Reason and Prerequisites

The error occurred because the precondition for step 8950 with step 2000 was configured with "OR" logic in the customer system instead of the correct "AND" logic.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the system corrects the precondition for step 8950, ensuring it uses "AND" logic with step 2000. Import the BC set /US4G/3_0_SUP_SP08_NOTE_3724080 or /US4G/3_0_SUP_SP07_NOTE_3724080, depending on your support package level. You can verify the fix by retesting the start of supply scenario and confirming that step 8950 executes with the correct precondition logic.
