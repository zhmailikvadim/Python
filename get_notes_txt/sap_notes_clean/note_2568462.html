Symptom

When you recalculate the depreciation (transaction AFAR), when you recalculate the depreciation within the migration (depreciation), or when you call the Asset Explorer (AW01N), the runtime error GETWA_NOT_ASSIGNED occurs in the program CL_FAA_DC_SEGMENTS, method _HLPSEG_GET_CO_YR_AND_PRD.

Other Terms

FINS_MIG_STATUS, AW01N, short dump, dump

Reason and Prerequisites

You use changeover method 4 in a depreciation key that is assigned to the fixed asset that has just been calculated.

Solution

Implement the attached correction instructions or import the specified support package.
