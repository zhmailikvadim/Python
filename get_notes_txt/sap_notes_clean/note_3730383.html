Symptom

The Data Collection Entry plugin in a Touch POD does not support the 'Enter' hotkey to add data collection values.

Other Terms

DC500

Reason and Prerequisites

Steps to reproduce:
1. Create an SFC.
2. Create a Data Collection with 2 or more parameters.
3. Attach the SFC to the Data Collection.
4. Open POD - SFC TOUCH.
5. Retrieve the SFC.
6. Open the list of data collection.
7. Fill 1 parameter and press 'ENTER'.

Expected result: The 'add' process will execute, resulting in an error message stating that the other fields are empty.
Actual result: Nothing has changed.

8. Fill in all parameters and press 'ENTER'.

Expected result: The view will close, and the data collection parameters will be added.
Actual result: Nothing has changed.

Solution

This is a rework of the implementation of SAP Note 3589546, as the previous blocked the functionality of the ENTER key in various PODs.

New solution is provided in the form of a code correction. Key pressed listener has been added for the Data Collection Touch view to evaluate 'add' button click and process adding data collection set parameters.

Impact

The code change implemented to address the symptom above is isolated to POD Touch view.
