Symptom

There is no possibility to define constant values checks or to define deviating activity codes or authority fail messages for custom authority checks.

Other Terms

Extensibility

Reason and Prerequisites

This SAP Note extends the authorization check BAdI /SCMTMS/BADI_AC_DEF with new methods SET_AUTH_OBJECT_CONST_VALUES() and SET_AUTH_OBJECT_ADMIN().

Solution

Implement the correction from the correction instruction or implement the Support Package of the SAP Note.

Further documentation is available in the BAdI documentation (in English), after having executed program /SCMTMS/NOTE_3040897.
