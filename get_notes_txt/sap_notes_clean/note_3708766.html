Symptom

  • A CDS Analytics app (such as Group Reporting app) gets time out on S4C 2602 when run it with year dependent fiscal variant
  • On backend system, a dump is generated as below:
    Category               ABAP programming error
    Runtime Errors         ITAB_DUPLICATE_KEY
    ABAP: Program          CL_RSMD_RS_0FISCPER===========CP
    Application Component  BW-BEX-OT

Other Terms

0fiscper, 0fiscvarnt, year dependent fiscal variant, fiscal period text

Reason and Prerequisites

Program error.

Solution

The issue is fixed in ABAP Platform Cloud 2602 04 and 2602 03 with Hotfix 1000040700, number 56742 (2026).
