Symptom

You are not able to enter data on posting level 0C due to the maximum selection restriction specified in the breakdown category. System is raising error message FINCS_ACCTG_INTEG / 100 or G00 / 790.

This issue can occur in Flexible Upload, GRDC form and GRDC data mapping.

Other Terms

DOCTY, RLEVL, Item &3: &1 &2 is not in maximum selection &4

Reason and Prerequisites

The issue is caused by program error.

Solution

Implement the attached correction instruction or import the specified Support Package.
