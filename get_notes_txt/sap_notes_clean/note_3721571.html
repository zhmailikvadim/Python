Symptom

Workcenters sidebar dropdown automatically close upon being opened.

Environment

SAP Cloud For Customer

Reproducing the Issue

  1. Open C4C.
  2. On the sidebar used to navigate workcenters select any workcenter.
  3. Click somewhere on the workcenter name to open the dropdown to select one of the OWL's available for the specific workcenter. ( e.g. select the activities workcenter to open the dropdown with appointments, phone calls etc.).
  4. Notice how sometimes the dropdown for the OWL selection will open and then immediately close.

Cause

The “>” icon was not used to open the workcenter.

Resolution

It is a system specification that you must use the “>” icon to expand or collapse any workcenter. This is the intended system design.

If it is a requirement to alter the above behavior a influence request must be raised in the influence portal.

See Also

Refer to KBA 2531827- Functionality Currently not Available for further steps on the influence portal process.
