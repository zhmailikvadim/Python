Symptom

 When creating an OLAP connection with the language set to Swedish (Sweden), it is automatically changed to Swedish (Finland).

Other Terms

Swedish, OLAP, Language

Reason and Prerequisites

Environment:

SAP BusinessObjects Business Intelligence Platform 2025 Patch 8

Steps to Reproduce:

  1. Go to Central Management Console (CMC)
  2. Click New OLAP Connection
  3. Click Next
  4. Select "SAP BW" in Network Layer
  5. Select "Swedish (Sweden)" in Language
  6. Click Create Connection
  7. Right-click on the created connection > Organize > Edit
  8. Note that the language has been automatically changed to Swedish (Finland)

Cause:

The cause is currently under investigation by Development.

Solution

 The issue is currently under investigation by Development.
