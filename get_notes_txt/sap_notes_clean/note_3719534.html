Symptom

In PDS integration processing, message 141 in message class CURTO1 is missing, resulting in an empty or incomplete system message.
During run of report CUSLNTRTO_CIF_REPORT, the checkbox for “Lock PDS if error occurs” displays only the technical field name GP_LOCK instead of the descriptive text.
The incorrect or missing texts appear in the selection screen.

Other Terms

CURTOADV_CREATE, GP_LOCK, TRDIR, Text elements, PDS creation, CURTO1, SE91

Reason and Prerequisites

The system delivery was missing the message text for message 141 in message class CURTO1 and the corresponding text element for field GP_LOCK.
Both omissions resulted in incomplete or technical-only texts during runtime.

Solution

Apply the attached Correction Instructions.
The corrections introduce the missing message text into message class CURTO1 and add the missing text element for the GP_LOCK checkbox.
