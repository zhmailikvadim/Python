Symptom

This article describes functionality introduced in the May 2026 SAP Fieldglass release

You may notice that some configuration options you previously used in Configuration Manager → Company Configuration are no longer visible

You might be wondering:

  • Why a configuration setting you used before is missing
  • Whether your access or permissions changed
  • Why the system still behaves the same even though a setting is no longer visible
  • Why different environments appear to behave differently

Reproducing the Issue

You may experience this when:

  1. You go to Configuration Manager → Company Configuration and cannot find a setting that was previously available
  2. You attempt to update configurations related to:
    • Service Orders mobile app behavior
    • Invoice generation for reversed or revised time sheets
    • Assignment Cost Center behavior
    • Supplier registration without Single Sign‑On (SSO)
    • Login settings such as Security Assertion Markup Language (SAML) 2.0, Two‑Factor Authentication (2FA), or LinkedIn sign‑in
  3. The system continues to follow a previous configuration even though the option is no longer displayed
  4. Two environments behave differently because one was previously configured and the other was not

Cause

SAP Fieldglass has removed certain high‑risk or legacy company configuration options from the Company Configuration area of Configuration Manager as part of ongoing product improvements

These updates help:

  • Reduce the risk of misconfiguration
  • Improve system stability and consistency
  • Modernize how configuration behavior is managed

If a configuration value was set in the past, it continues to apply even though the setting is no longer visible

Newly created environments may not have historical configuration values and may behave differently

Resolution

What to expect:

  1. A missing configuration setting is expected behavior
    • Certain configuration options were intentionally removed from Configuration Manager

  2. Existing behavior may continue
    • If your environment previously used a configuration, the system will continue to follow that setting

  3. Different environments may behave differently
    • Environments with prior configuration history may not behave the same as newer environments

  4. Configuration changes are no longer available through the user interface
    • These settings can no longer be adjusted directly in Configuration Manager

If you believe a configuration change is required for your business needs, please contact SAP Fieldglass Support for guidance
