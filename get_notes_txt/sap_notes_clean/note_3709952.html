Symptom

When creating time valuation decision rule, there is no navigation path to access the "Biographical information" section, which is required to retrieve data, e.g. the date of birth.

Environment

  • SAP SuccessFactors Time Tracking

Reproducing the Issue

  1. Create time valuation decision rules in SAP SuccessFactors Time Tracking.
  2. Attempt to navigate to the "biographical information" section.
  3. Observe that the navigation path to the "biographical information" section is missing.

Cause

This is a Known Error.

Resolution

This fix for this issue is planned to be deployed with the patch b2511p19. 

For patch timelines, please review SAP SuccessFactors Patches Knowledge Base. 

Workaround
No workaround is available.
