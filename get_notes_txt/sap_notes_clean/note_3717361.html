Symptom

API_PRAWELLCOMPLETIONVOL(PRA Well Completion Volumes)

You are using PRA Well completion volumes API to create well completion volumes, during creation you receive following messages, which are incorrect:

  • Message "does not exist" when volume source code is missing and,
  • Message "Measurement type not available for object WCMP" when measurement type does not have valid readings.
