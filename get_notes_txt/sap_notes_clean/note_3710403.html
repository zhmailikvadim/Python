Symptom

When upgrading SAP S/4 2020 system to S/4HANA 2023, got below error at phase PREP_INPUT_KERNEL.
**********************************************
===Tool: disp+work===
  
  *  *Issue:* Solvable issue
  *  *Kernel Type:* exe (/usr/sap/SID/SUM/abap/exe)
  *  *Action:* Update the download directory /xxxxxxxxxxxxxxxx

Variable   |Current             |Required|Status
================================================
Release    |916                 |        |ok
------------------------------------------------
Patch Level|71                  |100     |error
------------------------------------------------
Date       |Oct 11 2025 00:06:15|        |ok
------------------------------------------------
**********************************************

But kernel 916 patch 100 is not released yet.

Environment

SUM 2.0
SAP S/4HANA ABAP system

Reproducing the Issue

Repeat the SUM phase.

Cause

The source system S/4HANA 2020 is using kernel version 916. Please check PAM to get the supported kernel of the system.

Resolution

For the start version other than S/4HANA 2020, the solution is also the same.

  1. Reset the SUM.
  2. Downgrade the kernel version to 793 for the source system. Please refer to Note 19466 for more details.
  3. Upload the system info xml file to the MP. Then generate a new stack xml file. Please refer to Note 2287046 for more details.
  4. Use the newly generated stack xml file to start the SUM from scratch.
