Symptom

Versions:

  • 23.2.2026: Manual correction related T5A1M and T5A1L.

Symptoms:

  1. File structure: From the internal correct internal display, four characters instead of two characters are reserved for the record type in the file.
  2. File structure: When you fill the sequence number of a service, nine is saved from the internal display instead of the number that was actually calculated.
  3. SI number in notification: The SI number contains two blanks between the checksum part and the part resulting from the date of birth.
  4. The length of DDS and DDSOL is incorrect: the last character from field AKTZE is truncated.
  5. Payroll operation ASOP1 does not appear to exist and personnel calculation rule A011 cannot be activated. 

Other Terms

SOPENS, stock notification, export

Solution

Symptom 1 and 2 are corrected by the manual correction: Changes in T5A1L and  in T5A1M.
Symptom 3 and 4 are corrected automatically by implementing the source code corrections using transaction SNOTE.
For symptom 5, you can find some notes in the manual instructions. If this does not correct the symptom, open a new message with reference to this SAP Note and symptom 5.

Import the corresponding HR Support Package into your SAP system.

For an advance correction, proceed as follows:

  • First, implement the advance corrections described in the manual instructions.
  • Then upload the attached correction instructions to your SAP system using transaction SNOTE.
