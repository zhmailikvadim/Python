Symptom

When a business user selects a Freight Order in the Transportation Cockpit and clicks "Set to Confirmed by Carrier", the following error message is displayed:

    • "Select at least one ocean freight booking" (/SCMTMS/UI_MESSAGES 361)

Environment

SAP S/4HANA embedded TM up to the 2023 release

Reproducing the Issue

  • Open Transportation Cockpit
  • Select at least one Freight Order
  • Click on  button "Set to Confirmed by Carrier"

Cause

Although, the button is available for Freight Orders quadrants as well, the action "Set to Confirmed by Carrier" is supported only for Freight Bookings. In the standard system, enabling this action for Freight Orders in the Transportation Cockpit is available only as of the SAP S/4HANA 2025 release.

Currently, removing these buttons in earlier releases is not possible. Follow the resolution below to prevent users from executing the action for FOs. 

Resolution

To prevent the action in Freight Order scenarios, ensure that the button is not displayed in the Transportation Cockpit. To do this:
  1. Open the affected layout in Page Layouts for Transportation Cockpit.
  2. Go to the details of the Freight Order–relevant quadrant.
  3. Deselect the "Set to Confirmed by Carrier" button.
  4. Save the changes.
If you want to ensure that this button is not displayed at all for selection in Page Layouts for Transportation Cockpit:
  1. Open SPRO → Transportation Management → Planning → General Settings → Define Settings for Page Layouts.
  2. Go to Content Objects.
  3. Select the relevant Freight Order–related content object.
  4. Go to Content Toolbar Buttons.
  5. For button AID_TOR_LIST_SET_CONF_ACCEPT, unselect the 'Available' checkbox.
  6. Save the changes.
  7. Execute the app Synchronize Page Layouts with Customizing.

    If you also want to enable the action for Road FOs, please contact your technical team to support this custom requirement.
