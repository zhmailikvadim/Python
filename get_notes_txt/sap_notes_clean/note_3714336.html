Symptom

I create an award from a Sourcing Event for two vendors, however the Purchase Order (PO) is only created for one of the vendors.

Environment

SAP ERP

Reproducing the Issue

  1. Create an award for Sourcing Event in Ariba for more than one vendor;
  2. Send it to external system;
  3. Check in the ERP that the PO was only created for one of the vendors.

Resolution

The system currently does not support awarding a single sourcing event to multiple suppliers for QuoteMessage integration, this is expected behavior of the product. For more detailed information refer to the product Guide: Integration Gateway Overview Guide - Restrictions.

A customization would be necessary for this functionality to work, check your requirement with your consulting partner.
