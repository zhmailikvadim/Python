Symptom

How to display Pending Tasks Quantity on Each Tile in Work Zone?

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental. 

Environment

  • SAP SuccessFactors Work Zone
  • SAP Build Work Zone Advanced Edition

Resolution

It is technically possible to achieve the desired functionality. Two steps are required:

  1. A destination link needs to be configured between the cockpit and e.g. Travel & Expense. This server should provide an API that returns the number of pending tasks.
  2. The Tile should be changed to a dynamic tile and the dynamic data should be configured correctly (see attached screenshot). 

Keywords

SAP Build Work Zone, standard edition, pending tasks, tile display, dynamic tile, API, destination link, cockpit, Travel & Expense.
