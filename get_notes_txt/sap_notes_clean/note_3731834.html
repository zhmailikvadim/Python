Symptom

On the Swagger UI page, the Products API is labelled as deprecated.

Environment

SAP CPQ

Cause

To avoid confusion, there is no plan to decommission the Products API, this should still work for the foreseeable future. The Products API is optimized primarily for UI use cases.

Resolution

For external service integrations, it is strongly recommended to use the ProductsIntegration API, this has been specifically designed and optimized for that purpose.
The strategy is to keep the ProductsIntegration API continuously up to date with all new features intended for external consumption, ensuring better long-term compatibility and support.
