Symptom

With app Edit Costing Run or Edit Actual Costing Run, you are able to start the step with background processing. It's not clear how to check the background job status.

Environment

SAP S/4HANA Cloud

Reproducing the Issue

Start background process in app Edit Costing Run or Edit Actual Costing Run.

Cause

Check Resolution part.

Resolution

You can always check the background job status with app "Application Jobs".

Keywords

background, BTG, SM37, status, job.
