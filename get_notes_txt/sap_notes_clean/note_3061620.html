Symptom

This SAP Note provides guidelines for the installation of SAP Document Compliance (eDocument) for Germany Settlement Management.

Other Terms

eDocument, Germany, eDoc, eInvoicing, e-Invoicing, electronic invoice, electronic invoicing, Germany eInvoice, e-document, e-Document, e-Docs, e-Doc, e-docs, e-doc, Document Compliance, Digital Compliance, Document Compliance Framework, eDocument Framework, XRechnung, x-Rechnung, ZUGfERD

Reason and Prerequisites

Prerequisites

  • You need to subscribe to Peppol Exchange Service for the Cloud Edition of SAP Document Compliance that includes the usage right for the on-premise connector. Please get in touch with your SAP representative for further details.

  • You have installed SAP Document Compliance, on-premise edition. This option has a limited runtime usage right for the Application Interface Framework (AIF). For more information, refer to SAP Note 2378414.

  • You have installed the Document Compliance Framework in your system. Refer to the installation guide attached to SAP Note 2134248 for installation instructions.

  • You have installed eDocument for EU Peppol in your system. Refer to SAP Note 2783795 for installation instructions.

  • You have installed eDocument Germany - EU eInvoicing: Installation Overview in your system. Refer to SAP Note 2690949 for installation instructions.
  • You have implemented SAP Note 3050161 - Document Compliance: Installation Information for Integration of Settlement Management Documents.
  • You have installed eDocument EU Peppol Settlement Management: Installation Overview in your system. Refer to SAP Note 3061642 for installation instructions.

Solution

Date Version Reason
28.05.2021 1 Released for Customers

As a general rule, SAP recommends that you implement the Support Package specified for your release. However, if you need to install a solution earlier, use the SAP Note Assistant to implement the correction instruction. You can find more information about the SAP Note Assistant in SAP Support Portal, under SAP Note Assistant.

For software components S4CORE 104 and higher releases, install the following SAP Notes:

    1. 2988434  - eDocument Germany - EU eInvoicing for Settlement Management Documents

Product Documentation

Application Help Documentation

The online help for eDocument for Germany is published in the application help for country/region versions on the SAP Help Portal on the release day of the relevant support packages. You can find a preliminary version of this documentation attached to this SAP Note.

System and Customizing Documentation

Changes in the Customizing (IMG) structure (in SPRO transaction) are delivered with support packages (SPs) only. If you do not install the SPs regularly in your system, you can refer to the attachment of the SAP Note 2170013 for a preliminary version of the most important Customizing documents. The documentation attached to this SAP Note is subject to adjustment if changes in the software require that the documentation is updated.

Note about translations: Translations are only delivered with support packages.

Data Management Documentation

For information about features related to Data Protection and Privacy, such as data archiving, data destruction, blocking and deletion of customer and supplier data, refer to SAP Note 2876151 - Documentation About Data Management in eDocument (Applies to SAP ERP and SAP S/4HANA).
