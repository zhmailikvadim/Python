Symptom

In class CL_IM_EI_BADI_FSH_ART_CHK method IF_EX_BADI_MATERIAL_CHECK~CHECK_DATA_RETAIL the old data model MARC has been used to fetch the plant details. Since the old data model is prohibited in SAP S/4 HANA, the old data model has to be changed with the new data model.

Other Terms

MARC, V_MARC_MD, CL_IM_EI_BADI_FSH_ART_CHK, IF_EX_BADI_MATERIAL_CHECK~CHECK_DATA_RETAIL

Reason and Prerequisites

This is a program error.

Solution

Replace old data model MARC with new data model V_MARC_MD.

Implement the solution attached with this note.
