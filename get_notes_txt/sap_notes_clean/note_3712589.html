Symptom

Remittance is failing in CIG with the error message:

BusinessNetwork response is :"Conflict" and ErrorCode:"409" and ErrorResponse:"Error:Payment Proposal (xxxxxxxxxxxxxxxx) already has Remittance (xxxxxxxxxxxxxxxx) associated with it. Please contact support with the Error Reference Number: ANERR-xxxxxxxxxxxxxxxxxxxx for more details.. Suggested Action : Document failed in Business Network.ErrorCode:CIG-PLT-00644.Please review the Support Note 189049 (https://support.ariba.com/item/view/189049) for solution

Resolution

System only update a Payment Proposal once. Therefore, the Remittance must not contain any Payment Proposal that has already been sent in a previous Remittance.
