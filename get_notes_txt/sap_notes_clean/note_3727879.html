Symptom

Requirement is to understand whether Content Agent Service (CAS) (plan: application) on target subaccount is a necessary service in order to transport artifacts in Cloud Integration using Cloud Transport Management Service (cTMS). 

Environment

  • SAP Integration Suite
  • Cloud Integration

Resolution

If using cTMS for transporting artifacts and deploy-service for importing content from CTMS to CPI, the CAS application plan is not required in this scenario.

To determine if a Cloud Application Services (CAS) subscription exists:

  • Log in to the SAP BTP Cockpit.
  • Navigate to Global Account -> Entitlements.
  • Click on Entity Assignments (or the user's relevant location).
  • Search for the service named Cloud Application Services.
  • If CAS is entitled to the account, the following details will be visible:
  • Service: Cloud Application Services
  • Plan: Usually application
  • If the service is not listed, it indicates that the global account does not have the entitlement assigned.
