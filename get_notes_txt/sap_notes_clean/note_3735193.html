Symptom

An error appears after selecting Analyze Data button in a statutory reporting task: "The result set does not contain any relevant data." and a warning can appear: "Result set for query XXX is empty."

*XXX respects for CDS view name.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Open Run Statutory Reports app.
  2. Generate the report.
  3. Select the corresponding reporting form.
  4. Click Analyze Data.
  5. In the new window, the above error and warning occurs.

Cause

Missing country/region configuration in the Define Enhanced Tax Box Structure activity, which prevents the analytics view from returning data.

Resolution

Please refer to the following steps to resolve the issue:

  1. Open the Define Enhanced Tax Box Structure activity (102273).
  2. Maintain the required Tax Box settings for the relevant country/region.
  3. Save the configuration and retry Analyze Data in the Run Statutory Reports app.

See Also

Refer to: 3700823 - No Data Returned After Generating Report in Run Statutory Reports App
