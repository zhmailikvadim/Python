Symptom

In the settings for the LTSD process, you have defined a number of days in the "Days Before Validity Period" field.

Despite this, for an LTSD whose validity starts in the future, the system does not take into account billing documents whose issue date is before the validity start date. This results in a negative statement.

Other Terms

 DAYS_BEFORE_ISSUE_PERIOD, /SAPSLL/V_TSDCTR.

Solution

Implement the attached correction instructions or import the relevant support package.

After you implement this SAP Note, the setting defined in Customizing is also taken into account for future LTSDs. However, if there are several preference statements in this LTSD, the billing document is compared only with the first preference statement. All later preference statements remain undefined.
