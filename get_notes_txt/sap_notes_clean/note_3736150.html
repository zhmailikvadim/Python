Symptom

An error is received when creating a job in SmartRecruiters from SuccessFactors Position Org Chart

"Invalid input parameters for creating job in SmartRecruiters"


Environment

SmartRecruiters for SAP SuccessFactors

Reproducing the Issue

  1. Go to a position in Position Org Chart
  2. Click on "Create job in SmartRecruiters"
  3. An error occurs "Invalid input parameters for creating job in SmartRecruiters"

Cause

"jobTitle" is a mandatory field and "jobTitle" is being passed as null in the request to SmartRecruiters from Position Org Chart

Resolution

The standard field "jobTitle" must be configured in the Position Org Chart, the "jobTitle" field must also contain a value.

The value should also be visible in the Position UI otherwise it will get passed as null in the request to SmartRecruiters from Position Org Chart.

To resolve the issue

  1. Ensure "jobTitle" contains a valid value.
  2. Ensure that "jobTitle" is configured to display in the position UI by selecting it in Admin Center-> Org Chart Configuration.
