Symptom

In the new version of Manage Fixed Assets, the description linked to a custom field is not displayed alongside the custom field ID in Adapt UI.

Environment

SAP S/4HANA Cloud Public Edition 

Cause

This is a known issue.

Resolution

The issue will be corrected in hotfix 2602.7.
