Symptom

  • After server reboot, Adaptive Processing Server and Web Application Container Server are in failed state.
  • APS and AJS are continuously generating core dumps and JVM logs when they get restarted.

Environment

  • SAP BusinessObjects Business Intelligence platform 4.x
  • AIX

Cause

  • This issue is specific to AIX OS.
  • The xLC (C/C++ runtime) level on AIX needs to be compatible with SAP kernel.

Resolution

This issue has been escalated to development for further investigation - please see SAP Note 2278764 for further details

Workaround

  • Please review and verify the required xLC (C/C++ runtime) library level for AIX on KBA 1780629.
  • Any xLC (C/C++ runtime) level higher than 12.1.0.4 will not be compatible with BOE 4.x.
  • Downgrade it to 12.1.0.x accordingly.

See Also

  • 1780629 - AIX: Minimal OS Requirements for SAP Kernel
  • 2068737 - Adaptive Processing Server & Web Application Container Server Failed
