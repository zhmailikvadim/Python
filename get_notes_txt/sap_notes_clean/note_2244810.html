Symptom

You have deleted the value for the weighting. When you view the field in display mode, you see a zero in the field. When you view the field in change mode, you see that the field is blank. You do not find any explanation in the documentation as to how the system calculates the weighting in this case.

Other Terms

PLMD_AUDIT

Reason and Prerequisites

The documentation is unclear. For fields with this data type (INT2), the initial value zero is displayed differently in display and change modes for technical reasons. A change to this behavior is not possible for technical reasons.

Solution

Implement the correction instructions to change the documentation.

Note that the translation is delivered with a Support Package.
