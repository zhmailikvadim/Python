Symptom

This SAP Note enhances the following functionalities in year end adjustment calculation to support the special deduction for common life insurance for employees with dependents under age 23.

  • Payroll function P0146
  • Payroll operation JYEAC
  • Personnel calculation rules JY03 and JY08

Other Terms

RE5J40, RE511K, RE5J39, RE511P, RE5J20, RE5J18, CL_HRPAYJP_T5J40, /Y5S, /YS5

Reason and Prerequisites

This is a part of legal changes. You need to implement the changes in SAP Note 3713824 in advance.

Solution

You can apply the relevant HR Support Package or implement the rule changes together with the correction instructions in this SAP Note.

This SAP Note changes how the system calculates the deduction amount for common life insurance in year-end adjustment based on the Dep. under Age 23 checkbox on the life insurance tab of the infotype 0146 record. When you select this checkbox, the system calculates the deduction amount from the special deduction table for employees with dependents under age 23. Otherwise, the system calculates the deduction amount from the normal deduction table.
