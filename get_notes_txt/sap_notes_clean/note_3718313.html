Symptom

The SAP Note contains corrections relevant for ST-A/PI 01W_731 SP01 & SP02 in context of FRUN CSA.

The following corrections are included for the different use cases:

Store S00011 - Gateway registry & S00012 - Gateway security - Values are truncated

The values were truncated due to a 128-character limitation in the current function module. This has been fixed by using the appropriate function module, which retrieves the full value up to 250 characters.
