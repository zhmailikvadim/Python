Symptom

During the LIFT with CDOTE for an Inter/Intra company contracts from mass pricing work center, after running the transaction CMM_OF_PROCESS, the CDOTE order status shows 'Filled' in the Fiori application 'Manage Commodity Derivative Orders' but shows the message type as 'E' from message class 'LOG_CMM_PRC_PSM'.

Replication Steps -

-Create an Inter/Intra company contract with two NFE price fixations.
-Edit the contract and price the two NFE lots with the CDOTE.
-Run the CDOTE cycle to fill the future conditions of two lots.
-Create a nomination with reference to contract.
-Create a LDC with reference to nomination.
-Perform Lift with CDOTE using Mass Pricing Workcenter.
-Run the CDOTE cycle, after running the transaction CMM_OF_PROCESS, open the CDOTE Order in the Fiori application 'Manage Commodity Derivative Orders'.
-Error message type and message class 'LOG_CMM_PRC_PSM' with number '090' shows under assigned messages tab.

Other Terms

Agricultural Contract Management; ACM; LIFT; CDOTE; Manage Commodity Derivative Orders; FPIP (Pricing with Internal Partner)

Reason and Prerequisites

Program Error

Solution

Import the relevant support pack or apply the correction instruction(s).
