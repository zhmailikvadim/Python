Symptom

In transaction FB02 you change a document line of a FI document posted from a goods movement. The document items contains an internal order as account assignment and OLC functionality is switched off in your system. The document change will automatically clear the fields AUFPL and APLZL from the document item, even though there was no change done on these fields or the order.

If you are using Central Finance, this can lead to error message FINS_CFIN_AC_DOC 156 in the CFIN target system when the document change is replicated from source to target system.

Other Terms

FINS_CFIN_AC_DOC 156, FINS_CFIN_AC_DOC156, AUFPL, APLZL, COBL, FB02, FI_DOCUMENT_CHANGE

Reason and Prerequisites

Program error.

Solution

Implement the attached correction instruction in your system.
