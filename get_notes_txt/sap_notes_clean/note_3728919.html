Symptom

  • The hierarchy depth in the LeanIX workspace is set to X levels, which is insufficient for documenting parent-child relationships beyond this limit.
  • Users encounter an issue where they cannot add child objects to level X objects due to the hierarchy depth restriction.

Environment

LeanIX workspace settings

Cause

The hierarchy depth in the LeanIX workspace was configured to a default maximum, which limited the ability to create deeper parent-child relationships.

Resolution

  1. Create a Support ticket to request more hierarchy levels
  2. Once Support has configured the levels, to increase the hierarchy for fact sheet types, make the necessary adjustments via the LeanIX Model configuration for each fact sheet type.

See Also

Refer to: LeanIX Model Configuration Guide
