Symptom

In the Timesheet Analytics – Activities application:

  • There was no provision to set default filter on calendar date during the initial load.
  • Records with zero hours were shown incorrectly.
  • Canceled records were displayed.

Other Terms

F7189, Timesheet Analytics, Activities, hrfio_b_timesheet_act_anlyts, set_def_filter_calendar_date

Reason and Prerequisites

  • Default Filter Functionality was not available
  • Zero hour and canceled records were retrieved incorrectly while loading the data.

    Prerequisite: The SAP Note: 3734848 has to be implemented in the corresponding front-end system.

Solution

Apply corresponding support pack. Alternatively, implement the correction instructions attached to the SAP Note.

Steps to Implement Default Filter for Calendar Date:

  • Implement the method: SET_DEF_FILTER_CALENDAR_DATE of BAdI HRFIO_B_TIMESHEET_ACT_ANLYTS.
  • Set the desired default filter value within this method as per your requirement.
  • If the method is not implemented or no configuration is provided, the system will automatically default the Calendar Date filter with Year To Date (YTD).
