Symptom

Would like to change the description for standard Business Partner relationships.

Environment

  • SAP Sales Cloud Version 2
  • SAP Sales Cloud and SAP Service Cloud Version 2

Reproducing the Issue

  1. Go to Settings.
  2. Search for Business Partners.
  3. Choose Relationships.
  4. Click on a standard Business Partner relationship.
  5. In the Business Partner 1/2 section, the description could not be changed.

Cause

Business Partner relationship translation needs to be done via language adaptation.

Resolution

  1. Go to Settings > Language Adaptation.
  2. Select the application: businessPartnerRelationship – sap.crm.md.service.businessPartnerRelationshipService.
  3. Open the Code List tab.
  4. Select both “Business Partner Relationship Role” and “Directed Business Partner Relationship Role”.
  5. Search for the relevant relationship (for example, “Employee Responsible”).
  6. Add or adjust the translations for the entries AT46-1 and AT46-2 to ensure consistency with AT46.
  7. Save and verify in an Account that the roles now display with the expected translated labels.
