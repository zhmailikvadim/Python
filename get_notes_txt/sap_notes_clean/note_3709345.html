Symptom

I need to send the Contract Workspace Header Fields to the ERP without CLID.

Resolution

     The Contract Workspace Header Fields can be sent to the ERP without CLID in the below two ways.

  1. Utilize SAP Ariba API
  2. Use Web Services

     Note: There is no way to send the Contract Workspace Header Fields to the ERP without CLID using CIG
