Symptom

The clearing document for the MeMi document is missing in table /US4G/MEMI_DOC due to migration issues.

This issue is relevant for the market role: DSO.

Other Terms

MeMi, /US4G/RP_MEMI_CLEAR_DOC_CORR, /US4G/MEMI_DOC, CLEARING_DOC_NO, AUGBL, Clearing Document, migration

Reason and Prerequisites

Program error (migration issues).

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, you can use the correction report /US4G/RP_MEMI_CLEAR_DOC_CORR to correct the erroneous data.

You need to run the correction report /US4G/RP_MEMI_CLEAR_DOC_CORR to update the missing clearing documents.

You can verify the fix by checking in table /US4G/MEMI_DOC that the clearing document for the MeMi document has been updated.
