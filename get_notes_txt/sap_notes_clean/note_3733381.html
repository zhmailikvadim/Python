Symptom

You run the Maintain HR Master Data (PA30) transaction code. Press F4 in the Personnel Number field or choose Find by -> Person -> Collective Search Help. A search help based on the Alternative Names (0182) infotype is available.

Other Terms

Infotype 0182, Alternative Names, M_PREMAN, PREMAN, PREM, search help

Reason and Prerequisites

This is a legal change.

Solution

You can apply HR Support Package or implement the correction instructions in this SAP Note.

The new search help will be available for all country groupings that use the Alternative Names (0182) infotype.
