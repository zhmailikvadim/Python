Symptom

Component: App Support
Module: App Support 

You display an app in the SAP Fiori Launchpad and want to check it using the App Support tool (Fiori ID: F4914). You receive an error message:

"The app type "" is currently not supported by App Support tool." (/UI2/ADE_MSG 001)

Other Terms

/UI2/ADE_MSG001

Reason and Prerequisites

Reason:
The error occurs after implementing SAP Note 3668888.

Prerequisites:
For information about the validity of the corrections, see the correction instructions.

Solution

After implementing this SAP Note, the system identifies the correct app type again.

The "Support Packages" section lists the Support Packages that contain these corrections.

Alternatively, you can implement the attached correction instructions.
