Symptom

In the In-Force Business Configurator, the template "Payer of the CBC amount" is defined for the contract. A default value has been defined for the template using the static attribute DEFAULT_PARTNER, for example, DEFAULT_PARTNER = 1 "Preassignment with policyholder".

The contract is automatically included for the product.

Despite preassignment in the in-force business configurator, the payer of the Cash-Before-Cover amount must be entered manually.

Other Terms

CBC, CBC payer

Reason and Prerequisites

The relevant function is missing.

Solution

This is delivered with the next Support Package. For an advance correction, implement the correction instructions.
