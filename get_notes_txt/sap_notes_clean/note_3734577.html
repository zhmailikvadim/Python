Symptom

In the Fiori app "Display Meter Readings," when you select the value help for the parameter "Qty Determ. Status," the value range for the status of the quantity determination doesn't correspond to the value range of the domain E_QD_STAT. When you select a value from the value help and search for data, the app doesn't return any results.

This issue is relevant for all market roles.

Other Terms

Display Meter Readings, Fiori app, Qty Determ. Status, quantity determination status, E_QD_STAT, CDS view, /US4G/C_QuantityDetStatus_VH, value help, parameter

Reason and Prerequisites

The error occurred because of a program error in CDS view /US4G/C_QuantityDetStatus_VH.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the system displays the correct value range in the value help for the parameter "Qty Determ. Status." You can verify the fix by selecting a value from the value help and confirming that the app returns the expected search results for the parameter "Qty Determ. Status."
