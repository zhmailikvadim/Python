Environment

  • SAP Adaptive Server Enterprise (ASE) 16.0 for Business Suite
  • SAP Adaptive Server Enterprise (ASE) 16.0

Cause

SUM upgrade fail with below pre-requisites such as extend implicit conversion

Resolution

Run the following isql command and repeat the SUM

sp_configure "extend implicit conversion", 1
go
