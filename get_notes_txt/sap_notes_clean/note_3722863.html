Symptom

  • Clarification is requested regarding SAP's official support for AEM Forms Designer as a replacement for Adobe LiveCycle Designer 11.0.
  • Inquiry about the compatibility of SAP GUI (Transaction SFP) integration with AEM Forms Designer
  • Looking for additional information about the SAP maintenance and product strategy for Adobe LiveCycle Designer delivered by SAP. 

Environment

Forms Processing

Reproducing the Issue

  • Transition to Adobe Experience Manager (AEM) Forms Designer and inquire about SAP's official support for AEM Forms Designer as a replacement for Adobe LiveCycle Designer.
  • Test SAP GUI (Transaction SFP) integration with AEM Forms Designer and facing error

Cause

Resolution

 SAP does not contain the program Adobe Experience Manager, check directly with Adobe support in order to obtain it .

 Bear in mind that AEM is a competitor product and it is an Adobe Solution.

Using the Adobe Livecycle Designer for SAP solutions, which is available in the SAP service marketplace.
This is a branded product that is not related to Adobes product lifecycle.

We recommend to upgrade your Designer version to the most recent version.

The currently most recent available version is ADOBE LIVECYCLE DESIGNER for SAP solutions 11.0.

It is only support from SAP when use the tool/Product delivered by SAP - which is Adobe LiveCycle Designer for SAP solutions.
