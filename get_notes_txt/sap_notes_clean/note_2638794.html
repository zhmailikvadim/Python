Symptom

This is additional note to SAP Notes EDAG - June 2018 to provide a generated maintenance view to disable finance tax reporting.

Other Terms

RPCEDGV0, T7NOEDG_FTXOFF, V_T7NOEDG_FTXOFF

Reason and Prerequisites

  • This SAP Note is relevant only for Norway (HRNO) 
  • HRSP SAPK-60831INSAPHRCNO, SAPK-604A3INSAPHRCNO, SAPK-600D6INSAPHRCNO (or newer)
  • Prerequisites SAP Notes from previous SPs: 2493370, 2509710, 2522547, 2509791, 2509792, 2524295, 2524296, 2524297, 2536506, 2552359, 2564716, 2571481, 2571482, 2573087, 2580652, 2585460, 2596543, 2596544, 2600558, 2596545, 2618586, 2608315, 2622193 must be implemented before this SAP Note

  • Prerequisite SAP Notes from current SP (must be implemented before this SAP Note): 2647816, 2642128

Solution

This SAP Note provides a specific generated maintenance option for database view “V_T7NOEDG_FTXOFF”. Each customization must start on the first day of a month and end on the last day of a month (not in the middle). Only allowed value is ‘’ (FTX enabled) or ‘X’ (FTX disabled).

The view customizes the table „T7NOEDG_FTXOFF“. This table contains, whether enterprise + business is responsible to report Finance tax via EDAG reporting. For such record, empty value can be set to allow finance tax reporting, or „X“ value to disable finance tax reporting. A record with empty „business“ field will be valid for all businesses in the enterprise, which does not have the specific record. If there is neither record for combination enterprise + business nor general record for enterprise, finance tax will be reported, because it is not disabled.
