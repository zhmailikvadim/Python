Symptom

Can we configure a default currency at the supplier organisation/profile level which gets auto-applied for events the supplier is invited to?

Can we import the default currency field while importing data into Supplier Master data?

Cause

Resolution

Both Classic and Guided Sourcing event currency rules originate at the template level only.

Multi-currency events are configured by setting the event rule Allow participants to select bidding currency to Yes. In multi-currency events, the template author or event owner also configures the following items in the event rules.

This gives the supplier an option to choose the currency they would like to submit their bids while choosing the Lots/Line items. At the supplier level, we do not have the option to set up a default currency at the Supplier organisation level.

If the option for Suppliers to choose their bidding currency is provided, they would need to select the same once they access the event. If the rules is disabled, they would need to bid in the event currency only.

As per standard process we do not configure Supplier Master data to contain default Supplier Currency.

You can reach out to the Best Practice Center/Consulting or SAP Store to verify if they can assist you with your required configuration.
