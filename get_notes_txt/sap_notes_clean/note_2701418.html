Symptom

  • "[IllegalArgumentException] <pageIndex> (Error: RWI 00011)(Error: INF)" error is shown intermittently when refreshing Web Intelligence document through a document link (OpenDoc) in Web Intelligence HTML interface.
  • Issue is specific to BI Launchpad HTML interface and it's not reproducible using BI Launchpad Java Applet interface or in Web Intelligence Rich Client (WRC).

Environment

  • SAP BusinessObjects Business Intelligence Platform 4.2 SP5
  • Web Intelligence (HTML interface)

Reproducing the Issue

  1. Open the problematic Web Intelligence document.
  2. Click on Refresh.
  3. Fill in the prompt values.
  4. Click on a document link.
  5. A new browser tab will be opened with the child report.
  6. Error message appears: "[IllegalArgumentException] <pageIndex> (Error: RWI 00011)(Error: INF)"

Cause

  • This issue is caused by a defect in the product

Resolution

  • The issue is not reproducible using BI 4.2 SP5 Patch 6 and higher releases.

Workaround

  • The issue is not reproducible in BI Launchpad Java Applet interface or in Web Intelligence Rich Client.
