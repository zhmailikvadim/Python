Symptom

In transaction BP, you maintain the company code data for a business partner in a customer/vendor role (FLCU00/FLVN00). Under Customer: Payment Transactions/Vendor: Payment Transactions, you enter a customer/vendor that is not assigned to the same company code as the Permitted Payer/Permitted Payee in the dialog box that appears. The system does not issue an error message about invalid data maintenance or only when you have exited the dialog box and check or save your data.

Other Terms

BP, CVI, company code

Reason and Prerequisites

Program error

You get an error issued during saving when SAP Note 3709260 is implemented.

Solution

Implement the attached Correction Instruction or the mentioned Support Package.
