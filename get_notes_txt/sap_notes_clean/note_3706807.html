Symptom

While implementing SAP Note 3043063, in the activation phase the following errors occur for DDLS P_RU_BPBALANCE1:

  • The column ReversalReferenceDocumentCntxt is unknown
  • The column OriginalReferenceDocumentNum is unknown
  • The column OriginalReferenceDocumentYear is unknown

There are additional errors in DDLS P_RU_BPBALANCE0, P_RU_BPBALANCE2, C_RU_BPBALRECNCLN and C_RU_BPBALRECNCLNITM:

  • The column CUSTOMER is unknown
  • The column SUPPLIER is unknown
  • The entity P_RU_BPBALANCE0 required parameter P_REPORTINGENTITY
  • Parameter P_REPORTINGENTITY is not defined for entity P_RU_BPBALANCE1

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

  • SAP S/4HANA 1809 - below support package SAPK-10307INS4CORE
  • SAP S/4HANA 1909 - below support package SAPK-10405INS4CORE
  • SAP S/4HANA 2020 - below support package SAPK-10503INS4CORE

Cause

The errors in DDLS P_RU_BPBALANCE1 occur, as DDLS P_RU_JRNLENTRRVSD is missing the highlighted fields.

The missing fields are delivered by SAP Note 3042758, which is not implemented.

The errors in the dependent DDLS objects occur, as the inactive version of P_RU_BPBALANCE1 can't be activated due to the errors.

Resolution

  1. Implement SAP Note 3002174, if it has status 'Can be implemented':
    • This is a prerequisite of SAP Note 3042758.
    • Cancel the implementation in the object activation phase.
  2. Implement SAP Note 3042758 to eliminate the syntax errors in P_RU_BPBALANCE1:
    • If the error "Unable to read object DDLS P_RU_JRNLENTRRVSD from the local repository" occurs while initiating the implementation, of 3042758, manually reactivate Data Definition P_RU_JRNLENTRRVSD in SE80, then try to implement 3042758 again.
  3. Finish the implementation of SAP Note 3043063.
