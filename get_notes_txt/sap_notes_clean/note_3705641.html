Symptom

After activating global setting 0012/0002 "Non-Sequential Phase Processing", the flag "Non-Sequential Phases" is visible in Project Management projects but not in project templates.

Environment

  • SAP Portfolio and Project Management
  • SAP Portfolio and Project Management for SAP S/4HANA
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition 

Reproducing the Issue

  1. Maintain an entry for global setting 0012 / 0002 "Non-Sequential Phase Processing" with value "X" in customizing "Override Default Global Settings".
  2. Verify that the flag "Non-Sequential Phases" is visible in operative projects.
  3. Verify that the flag "Non-Sequential Phases" is not visible in project templates.

Cause

The required field control configuration for displaying check box "Non-Sequential Phases" exists for operative projects (object DPO) but not for project templates (object DPT).

Resolution

Maintain the missing field control customizing for project templates (object DPT):

SPRO
> SAP Portfolio and Project Management
 > Project Management
  > Global Enhancements to Project Elements
   > Set Up Field Control

  1. Select field control 0SAP or the relevant customer field control.
  2. In the node "Object Category in Field Control", select object DPT "Project (Template)".
  3. Navigate to the node "Field Conditions (per Field)".
  4. Create an entry specifying meaningful conditions for the external attribute NONSEQ_PHASES in External Structure DPR_TS_API_PROJECT_T.
  5. Save the field control configuration.
  6. Follow the instructions in SAP Knowledge Base Article Nr. 2193819 to clear field memory area CL_DPR_FC_MAP_AREA.

See Also

SAP Note Nr. 2162433 describes the functionality "Non-Sequential Phases for Projects" and related customizing settings.
