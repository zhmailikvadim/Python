Symptom

  1. You are getting an error while trying to enter a value in the code field of a custom fine tuning activity: XX is not a Valid Value for this Field. Enter a Valid Value, where XX is a the code value entered in the field.
  2. The value you are trying to enter in any of the column in the custom fine-tuning activity is getting removed on Saving the activity, after entering a value.

Environment

  • SAP Cloud Application Studio
  • SAP Business Bydesign
  • SAP Cloud for Customer

Reproducing the Issue

  1. Login to frontend UI.
  2. Navigate to Business Configuration work center.
  3. Select Implementation Projects view.
  4. Open the Activity list.
  5. Find and open the custom finetuning activity.
  6. Click on Add Row button to add a new value.
  7. When you enter a value in the code field, error appears.

Cause

If the UI is modified by you and the CCTS type is set as code then you will not be able to add a new row from the fine tuning activity.

Resolution

  1. Login to SDK.
  2. Open the relevant solution.
  3. Navigate to the BC view component of the fine tuning activity.
  4. Double click on the file to open it in the UI designer.
  5. Select the field where you have entered the value in the UI.
  6. Go to the properties tab and check the CCTS type which is maintained as code.
  7. Go to Data Modeling and select the content of the field i.e. Root - BC_SETS - Code- Content
  8. Change the CCTS type from Code to Text in the Properties
  9. Save and Activate.
  10. Deploy Business Configuration of the solution.
  11. You will be able to add rows.
