Symptom

This document lists the supported operating systems for the SAP HANA Client 2.28.

For other SAP HANA Client versions see SAP Note 3165810 - SAP HANA Client Supported Platforms.

For SAP HANA see SAP Note 2235581 - SAP HANA: Supported Operating Systems.

Solution

This note supplements the information available on the SAP Product Availability Matrix (PAM). The PAM does not have fields for the full range of client platforms and requirements, so this note provides more complete information on product availability for the SAP HANA Client Interfaces. In case of disagreement, the PAM is the official statement.

X = supported

On Linux x86_64, the Node.js driver requires the GCC 6 runtime files libstdc++.so.6 and libgcc_s.so.1, included with RHEL8 and SLES12.  The minimum version of libstdc++.so is libstdc++.so.6.0.21.  See the Operating System Classification section below on how to check compatibility for required kernel, glibc, and libstdc++.so versions.

On Linux ARM64, the client is compatible with ARMv8.2-a instructions, including LSE extensions.

* For HANA 2 SPS 08 and later servers, the JDBC driver requires a JDK 8 version released on or after July 2022 (SAP JVM 8.1.089 or Oracle JDK 8u341) in order to support TLS 1.3.

* The SAP HANA Client on Linux requires libgcc_s1 (C compiler runtime library). Most modern distributions and systems include this library.

Use the following commands to check version compatibility:

  • To check the Linux kernel version, run the command "uname -r".
  • To check the glibc version, run " ldd --version".
  • To check the libstdc++.so version is compatible, run "ls -al /usr/lib64/libstdc++.so*"
  • To check the version symbols inside the libgcc_s1 library, run a command like "strings /lib/libgcc_s.so.1 |grep GCC_ | sort -V |tail -1" and look for the symbol "GCC_4.8.0".
  • To check the version symbols inside the libstdc++.so.6 library, run a command like "strings /usr/lib64/libstdc++.so* | grep ^GLIBCXX_[0-9] | sed -e 's/^GLIBCXX_//' | sort -V |tail  -1".  To check whether the node driver is supported, look for the string "3.4.21".
