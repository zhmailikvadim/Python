Symptom

Time variance and overdue days are calculated using calendar days only.

Environment

•Materials Management (MM)
•SAP S/4HANA

Reproducing the Issue

1. Open the Supplier Evaluation by Time app (F1664A).

2. For each purchase order item, review the fields:

Overdue Days

Time Variance

Observe that the calculation subtracts the two dates directly in calendar days, ignoring the factory calendar.

Cause

Factory calendar enablement for Supplier Evaluation by Time is a new feature available from release 2023; earlier releases calculate using calendar days do not support this logic.

Resolution

  1. Use SAP S/4HANA 2023 or higher, where factory calendar support for Supplier Evaluation by Time is available.
  2. In Customizing, navigate to: SPRO > Materials Management > Purchasing > Purchase Order > Supplier Evaluation > Factory Calendar Setting.
  3. Enable the factory calendar settings so that statistical delivery date and goods receipt posting date are considered in the Time Variance calculation.
  4. If on a release prior to 2023, note that this feature is not downported to lower releases. Will require an upgrade to 2023 or higher to use this functionality.
