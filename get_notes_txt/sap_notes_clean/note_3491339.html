Symptom

  • New/Existing OLAP connection shows blank page once opened from Information design tool.
  • Unable to view OLAP connection details in IDT whereas the same connections can be accessed from CMC.
  • Error captured in logs:

Unable to find servers in CMS CMSserver.com:6400 and cluster @xyzcluster with kind pjs and service SecurityTokenService. All such servers could be down or disabled by the administrator. (FWM 01014)

Environment

  • SAP BusinessObjects Business Intelligence 4.x
  • Windows Server 2019

Reproducing the Issue

  1. Open IDT.
  2. Insert a repository session. 
  3. Open any OLAP connection.
  4. Notice the page on the right hand side is blank.

Cause

  • Security Token Service is missing from Adaptive Processing Server
  • Security Token Service 

Resolution

Create a new Adaptive processing server containing Security Token Service Or add Security Token service in existing Adaptive Processing Server.

Keywords

IDT, blank page, OLAP connection, STS, APS
