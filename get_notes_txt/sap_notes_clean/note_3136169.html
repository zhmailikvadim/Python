Symptom

You use the batch record function. You can jump to other transactions from the batch record. You are able to display the material master record, batch master record, batch where-used list, process order, and inspection lot. The system does not perform an authorization check.

Other Terms

Batch record, authorization, CALL TRANSACTION, ATC

Reason and Prerequisites

Missing authorization checks.

Solution

Implement the source code corrections according to the correction instructions.
