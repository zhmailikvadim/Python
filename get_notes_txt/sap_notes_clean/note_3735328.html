Symptom

  • In the Manage Outgoing Checks app, the "Void Reason" dropdown shows empty/blank descriptions when the logon language is, for example, ZF - Chinese Traditional.
  • When the logon language is EN - English, the same dropdown shows the correct descriptions.
  • No error messages are displayed.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Switch the system language to ZF - Chinese Traditional.
  2. Open the Manage Outgoing Checks app.
  3. Start the check void process.
  4. Open the "Void Reason" dropdown and observe that descriptions are not displayed.
  5. Switch the system language to EN - English.
  6. Repeat the same steps and observe that descriptions are displayed correctly.

Cause

Missing ZF - Chinese Traditional translations for some check void reasons in the backend table.

Resolution

  1. Confirm which languages are required.
  2. Contact the SAP contact person for further assistance.
