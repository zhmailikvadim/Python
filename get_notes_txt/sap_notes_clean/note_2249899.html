Symptom

Below details needs to be mentioned, when creating incidents related to Responsive UI Online / Offline issues

  • Are you facing issue in RUI(browser) or RUI (app) or from Mobile device app.
  • Is this issue in Online / Offline mode?
  • If it is from RUI (app), you are using it from laptop or from device?
  • Incase if issue is from mobile device app, you are using native app or RUI extended app.
  • Provide mobile device version (iPad 4, Samsung Galaxy etc.) and Operating system used.
  • Provide mobile app version (long press on the SAP logo in login page).

In case of Offline issues along with above details:

  • Business user who faced issue in offline mode in case of sync issues.
  • Screenshot and sync log (if available) for sync errors.
  • Data you tried to sync (Accounts/ Contacts / Appoitments etc).
