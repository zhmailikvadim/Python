Symptom

Some values applied through Mass Action in Executive Review are not saving.

Environment

  • SAP SuccessFactors HCM Suite
  • SAP SuccessFactors Compensation

Reproducing the Issue

  1. Open a template in Executive Review
  2. Click on Mass Action
  3. Add the value to be updated and apply
  4. Refresh the page and notice value is still not there

Cause

Expected behavior when the page is refreshed.

Resolution

As best practice it is recommended to close the page after a Mass Action. Once job is complete, reopen it and values will be updated.
