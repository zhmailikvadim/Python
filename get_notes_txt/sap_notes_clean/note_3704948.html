Symptom

Some old transactions were created before search suggestions were added as a possible input help. Since then some of the elementary search helps can handle this new feature, but not all.

Other Terms

/SCMB/MDL_PROD_BASE, 3682225

Reason and Prerequisites

New developments possibly working with old search helps.

Solution

Install this SAP Note,
or if no automatic correction instructions exist, then install the manual correction instruction steps.

**

Disclaimer:
the type-ahead fuzzy search cannot be activated for all search help input fields. It only has a chance to work with elementary search helps. It cannot and will not work with a variety of other search helps types, like collective or freestyle or specifically coded search help exist.

SAP is not obligated to provide additional input help features for already existing delivered software.
This SAP Note should not be looked upon as a precedent for such.
Activating this feature requires some performance from the DB, based on used database provider this may be noticeable.
Activating this feature for these two specific search helps was safe, because these are not complex simple search helps used in multiple places.
Activating this feature for further search helps in out of scope at this time.
