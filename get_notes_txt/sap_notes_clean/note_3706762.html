Symptom

Multiple issues faced when implementing SAP Note 3638716, as mentioned below:

  1. “TABLEFRAME” does not exist.
  2. In SRM_SERVER 713, a syntax error occurs in class /SAPCRM/CL_CH_WD_OUTBOUND_HDLR.
  3. Manual steps for maintaining hook_url and target_url in table SAPSRM_CTLG_LIST.

Environment

  • SAP enhancement package for SAP Supplier Relationship Management 

Reproducing the Issue

The issue occurs while implementing the SAP Note 3638716.

Cause

Since we are importing a new table SAPSRM_CTLG_LIST in the system, this issue with the TABLEFRAME is seen.

Resolution

1. For the popup when implementing the note - "TABLEFRAME does not exist":

  • From SRM_SERVER 700 to 713 , when implementing the note you get a popup "TABLEFRAME does not exist." You may ignore this message, kindly go ahead and activate Table SAPSRM_CTLG_LIST.
  • For SRM_SERVER 714, please check the note 3712781 - Follow-on correction for Note 3638716

2. The program NOTE_3712781_713 is created in your system. Follow the steps described in the note 3712781 - Follow-on correction for Note 3638716

3. Post implementation of the Note, while maintaining the values for the table SAPSRM_CTLG_LIST:

  • Maintain the HOOK_URL and TARGET_URL in Table SAPSRM_CTLG_LIST, with domain as per below format:
    protocol://host:port
  • Usually it is identical to the portal URL.

For example – SRM Portal is of below format:

https://host:port/../portal or http://host:port/../portal

Then maintain hook_url = https://host:port  or http://host:port

and target_url = https://host:port  or  http://host:port

In case of multiple portals connected to SRM system, maintain hook_url and target_url for all the portals, in the table SAPSRM_CTLG_LIST.

  • For catalogs being called from NWBC or SRMNXP scenario, similarly maintain the NWBC url or SRMNXP url domains for both the hook_url and target_url in table SAPSRM_CTLG_LIST.

Ensure the hook_url and target_url values in table are maintained in lowercase.

Note: If the SAPSRM_CTLG_LIST table is not maintained, then catalog will not launch.
