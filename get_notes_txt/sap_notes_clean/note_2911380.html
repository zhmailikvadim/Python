Symptom

  • Updating Web Intelligence Documents from unv to unx universes using the BI Automation Framework (2906634) fails if the Data Source for one of the Query is missing
  • Performing Change Source directly from Web Intelligence works fine

Other Terms

webi,web intelligence,unv,unx,change,source,BI Automation Framework,missing

Reason and Prerequisites

 Environment:

  • SAP BusinessObjects Business Intelligence Platform 4.2 SP05 onwards.
  • Admin Console
  • Automation Framework
  • Web Intelligence (Webi)

Steps to Reproduce:

  1. Create a web Intelligence report based on Excel datasource and efashion.unv
  2. Save the report to the repository
  3. Delete the Excel source from the repository
  4. From Automation Framework Create a scenario containing Logon, change source, logout
  5. Save and Run the scenario=> It fails

Solution

  • This issue is fixed in the patches listed in the "Support Packages & Patches" section below.

    The "Support Packages & Patches" section will be populated with the relevant patch levels once they are released.

    For Business Intelligence Platform maintenance schedule and strategy see the Knowledge Base Article 2144559 in References section.
