Symptom

In the process "Processing of Disconnection Request from SUP (DSO View)" (DE_B_GPKE_DISC_DOC_DSO),This occurs when a master data issue prevents step 1800 from completing. After five minutes, step 2850 (Receive process end notification from maco) closes the Pdoc, despite step 1800 remaining active.

This issue is relevant for the market role: DSO.

Other Terms

GPKE, DE_B_GPKE_DISC_DOC_DSO, disconnection request, ORDRSP 19116, step 1800, master data update, MDR

Reason and Prerequisites

The error occurred because the precondition for step 2900 was incorrect, allowing the process to complete before step 1800 finished.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the system ensures that the process completes only after finishing the master data update in step 1800 and all subsequent steps. The BC set /US4G/3_0_DSO_SP00_NOTE_3733924 contains the necessary corrections. No further action is required. You can verify the fix by checking that the process completes only after step 1800 and all subsequent steps finish successfully.
