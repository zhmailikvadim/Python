Symptom

The tool report "Confirm Clarification Cases" does not work as expected. The restriction to the clarification case type is taken into account only in combination with the entry of one or more clarification case IDs. In addition, the output icons for the clarification case types "Order" and "Delivery" are swapped.

Reason and Prerequisites

The prerequisite for implementing the program corrections is patch level 1905 for DMsprint 2.x (software component CDM, Release 200_702 or 210_751H).

Solution

  • Implement the attached correction instructions.


Corrected as of: DMsprint 2.x Patch 1911
