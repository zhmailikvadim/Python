Symptom

  • Searching with short words e.g. for task-ID's or project names in SAP Cloud ALM using the "all" search option results in an error.
  • Universal Search Error – Valid Object IDs Not Found - Defect 7
  • Searching with multiple words works correctly.

Reproducing the Issue

  1. Open SAP Cloud ALM.
  2. Set the search option to "all".
  3. Search for short words
  4. Observe the error that appears.

Cause

We have identified the root cause of the issue and are currently working on a fix.

This KBA will be updated once a fix is available.

Resolution

**UPDATE**

The issue is resolved.
