Symptom

This announcement contains information about the legal change for Bank Payments, published on 22 November 2025 by the Nordic Payment Council and Swedish Banking Association. These organizations represent the Swedish banks and the file format settings for swedish payments that SAP and its customers must adhere to. 

The following changes will be made by all Swedish banks, and banks with Swedish subsidiaries offering payment transactions through the Swedish banking network.  The legal change is mandatory. The effective date is November 15, 2026.​ 

 

Cause

The following payment services will no longer be offered by the banks: 

Bankgiro Inbetalningar (Including Image files) 

RFFOSE_A 

Payment Medium (Sweden) Autogiro (Direct Debit) for Sweden 

Bankgiro service Leverantörsbetalningar (Supplier payments) 

 

RFFOSE_B 

 

Payment Medium (Sweden) Domestic Vendor via Bankgiro 

Bankgiro service Leverantörsbetalningar (Supplier payments) 

 

RFFOSE_H 

Payment Medium (Sweden)  

Foreign Vendor via Bankgiro 

Postgiro Värdeavi (Money order) 

 

RFFOSE_P 

 

Payment Medium (Sweden) Domestic Vendor Postgiro 

 

Resolution

This means if you are currently using such a format, you need to transition to the new file formats provided by SAP, configure and implement these file formats 

All outgoing payments for salaries or to make vendor payments/customer refunds via an SAP S4 HANA public cloud, private cloud or ECC system must use the ISO 20022 formats. This includes bankgiro format SE_BANKGIROT which follows the ISO 20022 format. They can be found in DMEEX configuration t-code at the following location. 

Forms and Payment Medium Formats | SAP Help Portal
