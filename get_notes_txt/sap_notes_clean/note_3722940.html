Symptom

When you declare a DSN for an employee, the system does not generate an error message if the total number of working days that you declare in all blocks 53 (Type 01, Unit 40) for a month is negative.

When you declare a DSN for an employee, the system generates error message 344 if the total number of working days that you declare in all blocks 53 (Type 01, Unit 40) for a month exceeds the number of calendar days in that month.
There are a few exceptions for which it is acceptable to have the number of working days exceed the number of days in the month. Details in the long text of the error message are not clear.

Other Terms

ACTI (payroll cluster), REMU, block 53, S21.G00.53, S21.G00.53.002, working days, message 344, 345, HRPAYFR_DSN, Jours calendaires négatifs,  prorata du PMSS, error message

Reason and Prerequisites

Reason for triggering the error message: The payroll cluster ACTI contains inconsistencies.

Solution

After implementing this SAP Note, the system generates:
1) Error message 345 if the total number of working days that you declare in all blocks 53 (Type 01, Unit 40, 'C' current or 'R' replace) for a month is negative.
There are a few exceptions for which it is acceptable to have a negative total of working days in the month. The system lists them in the long text of the error message.

2) Error message 344 if the total number of working days that you declare in all blocks 53 (Type 01, Unit 40, 'C' current or 'R' replace) for a month exceeds the number of calendar days in that month.
Details in the long text of the error message are slightly adjusted.
