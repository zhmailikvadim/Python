Symptom

    Date format discrepancy in calculated dimension when exporting story to CSV in SAC

    Environment

    SAP Analytic Cloud 2025.21.11



    Reproducing the Issue

    1. Login to SAC
    2. Create a story report 
    3. Create a calculated dimension with date data type
    4. Verify that the report displays the date format ( eg : YYYY-MM-DD ) within SAC.
    5. Schedule publication of the report as a CSV file and download it from the zip file.

    Notice: The date format in the CSV file is changed ( eg : MM, DD, YYYY).

    Cause

      This is by design behavior of the product. 

      Resolution

      Kindly submit an enhancement request to request this feature as per - 2424376 - How do you create an enhancement request and provide feedback for SAP Analytics Cloud?
