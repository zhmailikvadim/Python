Symptom

Customer implements core SAP Note 3632490 (partial GR cancellation produces superfluous "cancellation of service") and needs to check with the side effects on DIMP related coding.

Other Terms

ERP2005, Enhancement Framework

Reason and Prerequisites

You are using DIMP and are about to implement core SAP Note 3632490.

Solution

To achieve consistent system behavior, implement the correction provided with this SAP Note together with the correction provided with core SAP Note 3632490.
