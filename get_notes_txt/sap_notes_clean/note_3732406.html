Symptom

When running the Tax Card report via job, the differences between values in the infotype 0205 and the received tax card might not be compared properly, leading to the B2A status Success - No changes and no update to the infotype, even in cases where there were changes to the tax card.

Other Terms

Tax Card, HFIUTAX_CARD_API

Solution

The issue described in section Symptom is resolved.

Additionally, the logging of B2A processing is now retained in the B2A Manager display after the execution is finalised. The information about processing can be found in the column Log.

Implementation

As a general rule SAP recommends that you implement the Support Package specified for your release. However, if you need to install a solution earlier, use the SAP Note Assistant to implement the correction instruction. You can find more information about the SAP Note Assistant in SAP Support Portal, under SAP Note Assistant.
