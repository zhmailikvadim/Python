Symptom

A user is unable to login via SSO into an Incentive Management Environment. 

Environment

  • SAP Sales Performance Management
  • Incentive Management on Legacy Oracle
  • Incentive Management on Legacy HANA 

Reproducing the Issue

  1. Attempt to login to the environment as the problematic user.
  2. Observe that the user is directed to the logon screen instead of being logged on via SSO.

Cause

This issue is caused due to a record not being present in the User Management space inside of Cloud Identity Services (IAS). In order to confirm this, the users ID should be searched in the User Management section of Cloud Identity Services.

Resolution

If the user is not present in the User Management section, it is due to the user not having an email present in their Participant record inside Incentive Management. Below are the steps to fix this issue: 

  1. Add the user's email address to the Participant record in Incentive Management.
  2. Run a re-sync job in IAS (Cloud Identity Services).
  3. Validate that the user can now access the environment via SSO.
