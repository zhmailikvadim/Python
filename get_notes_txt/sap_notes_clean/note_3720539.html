Symptom

Our Hub users cannot generate any report via the Collaboration Hub. The window to select the diagrams is always empty.

Resolution

If Collaboration Hub users have only the rights to view the published version of the diagram (H-rights), they will not be able to create reports. By granting these Collaboration Hub users additional the R-rights to view unpublished diagrams, they will be able to generate the necessary reports.
