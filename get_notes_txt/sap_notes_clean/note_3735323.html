Symptom

The system doesn't allow grid level update within a billed period via dialog transaction EEDM11. However, the system incorrectly allows grid level updates via MDX update within a billed period.

This issue is relevant for all market roles.

Other Terms

MaCo, /US4G/CL_UBD_DATA_POD_PERIOD, UPDATE_EUIGRID

Reason and Prerequisites

Missing functionality

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the system prevents grid level updates via MDX within a billed period, ensuring consistent behavior with transaction EEDM11. You can verify the fix by attempting to update the grid level within a billed period via MDX and confirming that the system raises the same error message as in transaction EEDM11.
