Symptom

There is a need to confirm if whenever upgrading SCC on Windows server platform it requires its re-installation, or the upgrade can be performed in one go.

Environment

  • SAP Cloud Connector(SCC);
  • Windows Operating Sytstem.

Resolution

Differently from SCC upgrade on Linux(which can be upgraded in one go replacing SCC package), when upgrading SCC on Windows it requires a two steps upgrade:

  1. Uninstall the Cloud Connector as described here and make sure to retain the existing configuration;
  2. Reinstall the Cloud Connector within the same directory as per page here;

Note: Before accessing the SCC Web UI, clear the  browser cache to avoid any unpredictable behavior due to the upgraded UI.
