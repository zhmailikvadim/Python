Symptom

When completing a maintenance order via Fiori app, the option 'Complete Assigned Notifications' flag is empty (not selected as default)

Environment

  • SAP ERP Plant Maintenance (PM)
  • SAP ERP
  • SAP S/4HANA, on-premise
  • SAP S/4HANA, Cloud Private Edition

Reproducing the Issue

  1. Run the app 'Find Maintenance Orders'
  2. Select a open order
  3. Select 'Change Status - Complete Technically'

Cause

This is the expected behaviour in Fiori apps

Resolution

Some adjustments can be made to the completions options offered:
Use transaction EAM_USER (in backend system) and set automatic Notification completion at tab 'EAMS Web UI'
Set a option for 'Complete Technically' and 'Complete (Business)'
- Display popup with options
- Always complete notifications
- Never complete notifications

Keywords

F2173, EAM_ORD_MANS1, EAM_ORDOP_MONS1, default
