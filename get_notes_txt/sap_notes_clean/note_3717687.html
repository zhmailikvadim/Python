Symptom

 There is no option to revert errored modification for SFO order from servicing side.

Other Terms

G-Invoicing, SFO

Reason and Prerequisites

missing functionality

Solution

Please implement this SAP Note or relevant support pack.

Please run Report /GINH/REVERT_SFO_ERROR_MOD to revert errored modification from servicing side. User should have following authorization to run this report.

Auth-Object:- S_PROGRAM

P_GROUP: -/GINH/REVERT_SFO_ERROR_MOD

P_ACTION:-BTCSUBMIT and/or SUBMIT
