Symptom

You want to transfer some KUT fields or extension fields from registered products to your V2 system, however the "Transfer Extension Field Definition" is disabled even though you can select the fields you want to transfer.

Environment

SAP Cloud For Customer

Reproducing the Issue

  1. Open Data transfer tool
  2. Navigate to the object for which you want to transfer the custom fields
  3. Select the fields you want to transfer
  4. The "Transfer Extension Field Definition" button is disabled

Cause

DTT tool has a check to ensure that BO metadata transfer done at least once before users can transfer extension fields directly using "Transfer Extension Field Definition button".

Resolution

Perform a test data transfer for the specific BO. Once an initial transfer is perform, the "Transfer Extension Field Definition" button will be available.

Keywords

DTT, data transfer, disabled, greyed out, BO, registered product, Transfer, Extension, Field, Definition
