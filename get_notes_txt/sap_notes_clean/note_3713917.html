Symptom

You want to reissue an outgoing check in the Fiori application F1578 – Manage Outgoing Checks.
After confirming the dialog, nothing happens.
You can see the following error in the console log:
"o.getValueState is not a function"

Other Terms

F1578, reissue, check, error, unresponsive, getValueState

Reason and Prerequisites

Programming error. Incorrect element ID causes undefined function call.

Solution

Implement the attached Correction Instruction or the mentioned Support Package.

Afterwards reindex the application on your frontend (gateway) system as described in SAP Note no. 2227577 and clean the caches according to SAP Note no. 2017117.
