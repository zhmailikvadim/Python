va10n10058:/sybase_0058/sandbox2_REP/RS157.> oslevel -s
6100-09-04-1441

Environment

SAP Replication Server (RS) 15.7.1 SP 300

Cause

SRS 15.7.1 SP300  is being installed on an incorrect version of the AIX operating system.

Resolution

  • SRS 15.7.1 SP300 is the current major release of Replication Server and requires AIX OS 7.1.   
  • See the 15.7.1 SP 300 Replication Installation Guide for Unix  and search for a 'System Requirements' link for AIX and other platforms:

 AIX 7.1

Before you install SAP Replication Server, install the IBM XL C/C++ Runtime for AIX and the SMP Runtime Libraries required for your version of AIX. Obtain the libraries from your IBM AIX operating system installation media.
These libraries are also available from the IBM AIX Web site:

    • The IBM XL C/C++ Runtime for AIX is available on the Web site under “Latest updates for supported IBM C and C++ compilers.”
    • The SMP Runtime Libraries for AIX are available on the Web site under “IBM XL C/C++ Enterprise Edition for AIX, Runtime Environment and Utilities.”

To verify that the libraries are installed, issue:

  • source $SYBASE/SYBASE.csh
  • cd $SYBASE/$SYBASE_REP/bin ldd ./repserver
