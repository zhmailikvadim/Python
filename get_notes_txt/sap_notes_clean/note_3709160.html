Symptom

You execute SI legal reports to generate XML files successfully, but these XML files couldn't be displayed in B2A Manager.
When you select 'SI' in Area filed, you will get the following message without any file output.

  • Message no. HR_PAY_B2A021: "There are no notifications for this selection and HCM localization Japan".

Environment

  • Human Capital Management (HCM)
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP SuccessFactors Employee Central Payroll
  • SAP S/4HANA

Reproducing the Issue

  1. Access transaction code PB2A.
  2. Select Area SI.
  3. Observe that there are no notifications for the selection and HCM localization Japan.

Cause

  • Default start date is set one year earlier than system date. The target period is earlier than default start date.

Resolution

  1. Navigate to the menu bar and select 'Process' button.
  2. Choose 'Change period'.
  3. Adjust the start date to be earlier than the target period.
  4. XML files will be filtered and outputted.
