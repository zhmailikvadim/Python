Symptom

  • Unable to edit the protected section of the class during SPAU activity.
  • Error message encountered: "You cannot edit the section directly in modification mode".
  • The modification in a previous upgrade SPAU activity cannot be modified in the current SPAU activity.

Environment

SAP NetWeaver Application Server for ABAP

Reproducing the Issue

  1. Navigate to SPAU activity in SAP S/4HANA.
  2. Attempt to edit the protected section of the class.
  3. Observe that the system prevents the modification and displays the error message: "You cannot edit the section directly in modification mode".

Resolution

Follow the instructions provided in the performance assistant message associated with the error:

  • Click on the '?' icon in the pop-up error message.
  • Navigate to the Class Builder and choose Edit -> Modification Operations -> Switch Off Assistant.

Attempt to edit the protected section of the class again after switching off the Modification Assistant.

See Also
