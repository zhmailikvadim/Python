Symptom

  • When trying to create a new crystal report for a Date field which has empty key value as '#' at SAP BW side, following error message appears:

    'Database Connector Error: 'Failed to run query.java.lang.String cannot be cast to java.util.Date'

  • Crystal Reports cannot handle '#' as Date field, it reads the value as String.
  • Other client tools like Information Design Tool, Web Intelligence and Crystal Reports for Enterprise can handle '#' as Date field.

Other Terms

 string cannot be cast date # empty value

Reason and Prerequisites

Environment

  • SAP Crystal Reports 2020 SP5 Patch 4
  • SAP Crystal Reports 2025 Patch 8
  • SAP BW (DSL)

Steps to Reproduce

  1. Create a Date field at SAP BW side and have empty values, which key value appear as '#'.
  2. Open Crystal Reports designer.
  3. Create a new crystal report.
  4. Add that Date field to the report.
  5. Observe error message appears.
