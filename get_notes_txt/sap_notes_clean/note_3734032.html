Symptom

  • Unable to send attachments while creating a Purchase Requisition using web service.
  • Error message: "The incoming message has invalid Attachments with the following errors: Attachment with Content Id <xxxxxxxxx xx> is not available"

Cause

  • The attachment reference uses a content id (cid)value enclosed in double quotes and/or has spaces between cid and its value, which invalidates the attachment handling.

Resolution

  1. Remove double quotes around the Content ID (cid) value in the attachment reference.
  2. Ensure there are no spaces between cid and its value.
  3. Do not require the cid value to match the file name; it can be different.
