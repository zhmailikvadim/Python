Symptom

Symptom

SAP Transportation Management (SAP TM): when a railcar unit or a container unit contains more equipment at the highest level, the capacity information in General Data screen is misleading - it refers only to one of them (randomly chosen).

Reason and Prerequisites

Program error.

Solution

Implement the attached correction instructions or install the corresponding Support Package.

After the correction, the capacities are correctly cumulated into the corresponding fields in General Data screen. Additionally, these fields become read-only in General Data, if more than one main item is defined in the Equipment screen.

Do not run new initial replication if there is one still running in the system.

It is recommended to implement the SAP Note before running replication for Bank.

Other Terms

Bank, MDI, bank master data, SAP Master Data Integration, CL_MDI_BM_DRF_OUT

Reason and Prerequisites

You want to improve performance for the SAP Master Data Integration inbound and outbound process.

Solution

Install attached Correction Instructions via transaction SNOTE or implement corresponding Support Package.
