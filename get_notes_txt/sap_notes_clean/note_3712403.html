Symptom

You execute the transaction DFS_DIS_NR_MAINT to transfer number range interval from central system to remote system. You receive an error 'Target System Not Found'.

The issue occurs during one of the following scenarios.

  1. If the RFC destination maintained does not have the same name as that of the logical system
  2. If the RFC destination does not exist
  3. If the RFC destination is not configured as per the System and Landscape Configuration Maintenance 

Other Terms

Target System not available, DFS_DISCONNECTED/009, Number Range Transfer, Invalid RFC Destination, Defense & Security, S/4 Defense Disconnected

Reason and Prerequisites

Programming Error

Solution

For symptom 1 please implement the attached Correction Instructions.

For other symptoms, please maintain the RFC destination in spro -> Defense & Security -> Common Settings -> System Landscape Maintenance.
