Symptom

The purpose of RFA Replicate InfoObject Master Data is to replicate hierarchies and master data from BW InfoObjects into environment fields.

Earlier to this enhancement, in the case where you wanted to replicate any data from BW on HANA application suite, you were required to create and maintain function module ZNXI_NW_GET_IOBJ_MD on the remote BW system, were you wanted to replaicate data from.

With this current enhancement, the logic of function module ZNXI_NW_GET_IOBJ_MD is narrowed to only retrieving hierarchy data from remote BW on HANA systems, in the specific case where remote hierarchy classes are used. For the rest of the cases (InfoObject master data and standard hierarchies on BW on HANA, or InfoObject masterdata and any type of hierarchies on BW4/HANA), the RFA works without the help of function module ZNXI_NW_GET_IOBJ_MD.

Other Terms

LOD-PER-OP Rel 3.0 SP 23

Reason and Prerequisites

Enhancement

Solution

In order to get the solution, please implement the automatic correction instructions using transaction SNOTE and perform the manual steps.
