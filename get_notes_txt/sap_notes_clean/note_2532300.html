Symptom

In ACT_UPG phase, the following error can be found:

  ==

  1EEDO519X"DDL Source" "BSAD_DDL" could not be activated
  1EEDO519 "DDL Source" "BSAK_DDL" could not be activated
  1EEDO519 "DDL Source" "BSAS_DDL" could not be activated
  1EEDO519 "DDL Source" "BSIS_DDL" could not be activated
  1EEDO519 "DDL Source" "FAGLBSAS_DDL" could not be activated
  1EEDO519 "DDL Source" "FAGLBSIS_DDL" could not be activated

  ==

  Or Error:

  ==

  1EEMC705 Key fields are not together at the beginning of the view

   ===

Environment

  • SAP Basis 750 up to patch level 09
  • SAP Basis 751 up to patch level 04

Reproducing the Issue

Repeat the phase.

Cause

These all are CDS views. For all these kinds of views, all the key fields should be stored together at the beginning of the view.

Resolution

Implement the correction instructions of SAP Note 2474810 - ABAP CDS: view with UNION and preserved key.

If SNOTE is locked, then implement the correction instruction "0020751259 0000073013" manually on the shadow instance.

Repeat the phase afterwards.

See Also

SAP Note 2474810 - ABAP CDS: view with UNION and preserved key
