Symptom

How to maintain the tax mapping in view EDO_PL_MAP_TAXV2V for Poland KSeF 2.0.

Environment

  • SAP ERP
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition
  • SAP Document and Reporting Compliance, Cloud Edition

Reproducing the Issue

  1. Run transaction SM30
  2. Open the view EDO_PL_MAPTAXV2V

Cause

How-To

Resolution

  1. Refer to below Help Portal documentation:
    1. SAP ERP: Mapping Tax Codes to Invoicing Elements Version 2 | SAP Help Portal 
    2. SAP S/4HANA: Mapping Tax Codes to Invoicing Elements Version 2 | SAP Help Portal 
  2. Follow the guidance to maintain the required tax code mapping in EDO_PL_MAPTAXV2V according to Poland KSeF requirements.
