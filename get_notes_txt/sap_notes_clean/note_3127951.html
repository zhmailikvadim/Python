Symptom

This SAP Note contains information about planning the installation, upgrades and support packages of the ABAP add-on UIACS 150.

Other Terms

Release strategy, UIACS 150, Maintenance Information, SAP ASSURANCE & COMPLIANCE 1.5

Solution

 

  1. General information
  • See the general information from SAP Note 70228.
  • Please check chapter Uninstallation for this UIACS and find further instructions there. Further uninstallable Add-ons are listed centrally in SAP Note 2011192.
  • Upgrades with ABAP add-ons
    • In systems in which an ABAP add-on is installed, you can upgrade only to SAP releases that are supported for this add-on.
    • Note that there is a delay between the delivery of the SAP standard releases and the release of the corresponding add-on releases.
    • If S/4 conversion is planned on a system with add-ons, then the add-on upgrade is integrated in the conversion process. To carry out this conversion, you require the target release add-on relevant packages in addition to the SAP standard upgrade packages.If you are scheduling an upgrade or the import of an enhancement package, note that you should not import the latest Support Package Stack in the source release directly before the upgrade to ensure that the required, equivalent Support Package is already available in the target release.
    • For more information about this, see SAP Note 832594 and the following information on SAP Support Portal.
      http://support.sap.com/sp-stacks -> SP Stack Information -> SP Stack Strategy
  1. Overview
    • SAP Product Availability Matrix
      The following information about the add-on in the SAP Product Availability Matrix is available on SAP Support Portal at https://apps.support.sap.com/sap/support/pam
      - Availability
      - End of maintenance
      - Release for products with enhancement packages
      - Language support

    • Download
      SAP Solution Manager's cloud-based Maintenance Planner enables easy and efficient planning of all changes in your SAP system landscape such as updates, upgrades, and new installations.Therefore, separate installation/upgrade media is no longer required as the Maintenance Planner is automatically calculating all software packages needed for your installation/upgrade. But, in case you attempt to download the packages manually, the software is available on SAP Support Portal at https://launchpad.support.sap.com/#/softwarecenter. Insert the technical name of the software component in the search field of the Software Download Center. The technical name of the software component can be found in the Installation Guide (in the Product Availability Matrix) or Installation Note.

    • The add-on does not contain any modifications.
      You can also perform the installation if you have already imported more Support Packages into your system than are specified in the section "Required Support Packages".
