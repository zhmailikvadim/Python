Symptom

If the results list of a report is updated, for example, by a restriction using a search or by changing the selection criteria, the screen reader does not display any information. Therefore, the user does not know that the result has changed.

Other Terms

UIS4H 2025.0.23, UIS4HOP1 12.2.24, UIS4HOP1 10.0.36, F4940A, ReportResult.controller.js

Solution

Implement the described correction or import the relevant Support Package. For release 2025 the translations of the new texts are included in the corresponding support package only.
