Symptom

During the manual re-pack all HU items process in /SCWM/RFUI transaction, Error message "Function code FCPMIN not defined for Flow; contact your system administrator" /SCWM/UI_RF022 is displayed.

Environment

  • SAP S/4 HANA
  • SAP Extended Warehouse Management

Reproducing the Issue

  1. Login to transaction /SCWM/RFUI 
  2. Follow the menu Outbound Processes --> Packing --> Log on to packing station. 
  3. Enter the work center and select the menu Manual re-pack all HU items (Logical transaction PAIRAL will be called)
  4. Enter the source HU 
  5. Error message /SCWM/UI_RF022 is displayed.

Cause

Missing customizing.

Resolution

  1. Navigate to the following path in SPRO Customizing.  
  2. SPRO --> SCM Extended Warehouse Management ---> Extended Warehouse Management ---> Master Data ---> Work Center ---> Define Work Center.
  3. Enable the flag " RF: HU ready for input " for the relevant warehouse and work center.

Keywords

manual repack, /SCWM/UI_RF022, PAIRAL, RF: HU ready for input, FCPMIN, work center, manual re-pack.
