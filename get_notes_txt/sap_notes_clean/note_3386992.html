Symptom

Cannot Insert or Delete the entries on a maintainance view in the transaction SM30

Environment

ABAP Platform

Reproducing the Issue

Run SM30, try to insert entries in a view.

Cause

The settings restriction of the view.

Resolution

Go to SE11, change the access option to "read, change, delete and insert". If this is a standard object, the option change could be a modification.

Keywords

SM30, SE11, insert, delete, view, change, modification, maintainance
