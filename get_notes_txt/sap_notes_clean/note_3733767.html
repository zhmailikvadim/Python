Symptom

  • According to SAP note 2938650 the fixes are applied For ST component 7.2 SP-11 later versions
  • SAP security note 2938650 addresses CVE-2020-26836 on Solution Manager 7.2 (Trace Analysis).
  • Security experts suggest similar behavior may exist in other ABAP systems although the specific Solution Manager note cannot be implemented there.

Environment

  • ABAP systems (including Solution Manager 7.2 and other ABAP-based systems)
  • Trace Analysis

Reproducing the Issue

Vulnerability CVE-2020-26836

Resolution

  1. If the vulnerability effects for Solution Manager 7.2 SP-level 11 and lesser. refer to and implement the corrections in SAP Note 2938650.
  2. If the Solution manager SP-level is greater than 11, you can ignore the vulnerability as the note is already implemented in the latest versions

If vulnerability occurs for Other Managed Systems that are registered in Solution Manager

  1. Follow the Resolution Steps as mentioned in the Following KBA
  2. 0003143650 - The logoff parameter redirecturl is marked as a security vulnerability

  • refer to: KBA 3143650 - The logoff parameter redirecturl is marked as a security vulnerability
