In Field Logistics Initiate Returns by Product app, enhanced Confirmation popup on click of the save button.

In the Initiate Returns-By product App, once the user initiates the return of the product, a popup is displayed with Stock Transport Order number and on choosing "Yes", it checks for the below condtions:

1.If multiple stock transport order numbers are created for the same product, then app navigates to the List Report of the Initiate Returns by Product application with filtered reference document number.

2.If single stock transport order number is created for a product, then app navigates to the object of the Initiate Returns by Product application.

If you choose "No" , the app refreshes the object page.
