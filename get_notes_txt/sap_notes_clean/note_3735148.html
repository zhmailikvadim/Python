Symptom

  • Restricted measures that filter on a hierarchy-based time dimension do not apply the selected interval correctly; when a single-year range is used, totals still include values from other years.
  • When the hierarchy time dimension is added to table rows, values from multiple years appear even though the restricted measure is set to a single-year range.

Environment

  • SAP Analytics Cloud 2026-02-21 - 2026.2.5 -> 2026.2.6.

Reproducing the Issue

  1. Open a story based on a Datasphere live model that includes a hierarchy-based time dimension.
  2. Create or edit a restricted measure: select a base measure, add the hierarchy time dimension as a filter, and choose a range/interval restricted to a single year.
  3. Apply and observe that the total is not limited to the selected year; values from other years remain included.

Cause

This issue is currently under investigation by development.
