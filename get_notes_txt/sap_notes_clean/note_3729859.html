Symptom

The approver name shown on the save & submit popup shall be changed based on customer specific rules. In S/4HANA Fiori app MTE V3 there is no suitable BAdI available to change or remove the name.

Other Terms

approver, my Travel and Expenses,

Reason and Prerequisites

Missing feature.

Solution

Implement the SAP Note or the Support Package. Then the enhancement spot FIN_TRV_ENH_TRAVELEXPENSE will contain the new BAdI definition FIN_TRV_APPROVER_NAME with method CHANGE_APPROVER_NAME. Employee number of traveler and general trip data are imported. The approver name is provided as changing parameter. The value can be changed. If the value is cleared, no approver name will be shown on the submit popup.
