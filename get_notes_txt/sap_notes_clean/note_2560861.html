Symptom

For a new proposal run in F110 the proposal log is not shown after running the proposal.

Environment

  • Financial Accounting (FI)
  • SAP R/3 Enterprise 4.7
  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA

Reproducing the Issue

  1. Go to transaction F110
  2. Enter the parameters
  3. Execute proposal run
  4. Check Proposal log then below popup appears

Cause

Due to missing check boxes in F110.

Resolution

  • Select the required checkboxes in below screen of F110

  • Please make sure to enter Vendor/Customer numbers in below screen to get detailed payment logs for particular Vendors/Customers.

             

  • Rerun the proposal and check the logs.
