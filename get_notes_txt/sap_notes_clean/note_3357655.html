Symptom

I am unable to revoke time sheets because there are no reasons to select

Cause

No reasons for time sheet revocation were created by the buyer

Resolution

Reasons for time sheet revocation can be created by buyer administrators and buyer users with the correct authorizations using the steps below:

  1. Click Admin Configuration (gear icon) > Reason
  2. Click New 
  3. Set Module to Time Sheet
  4. Set Type to Revoked
  5. Enter the new reason Name
  6. Click Add
