Symptom

  • You are trying to pull W4 (federal forms) data via odata api from SuccessFactors.
  • You are checking odata api dictionary however you cannot find entity related to W4.

Environment

  • SAP SuccessFactors HXM Suite
  • Integration Center

Reproducing the Issue

- You are checking Odata API dictionary and you do not find entity related to W4.

Cause

W4 is not an entity but rather a field within SF PM related forms.

Resolution

You can find W4 related data under ComplianceFormData entity.
