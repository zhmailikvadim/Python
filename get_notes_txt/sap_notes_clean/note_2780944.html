Symptom

You use a trip provision variant for the German public sector. In the field control (transaction FITVFELD or the "Define Schema and Individual Field Control" IMG activity), you have maintained a different field control for the schema for a separation allowance with daily return (schema TR in the delivery customizing) for screen "0100 Daily Return: Per Diems".

The field control is ignored when the per diems screen is called for a separation allowance period.

Other Terms

Separation allowance, separation allowance event, separation allowance period, screen 0100, FITVPS_TG, required entry field, optional entry field, hide, tr_tg, pubsec_germany

Reason and Prerequisites

Program error

Solution

Import the specified HR support package or implement the attached correction instructions.
