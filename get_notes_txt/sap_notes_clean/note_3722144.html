Symptom

When the flag "Include Expired Title Groups in SCIM API Response," which was added in release 2602, is disabled, users are being incorrectly marked with "inactive" titles.

Environment

  • SAP Sales Performance Management
  • Incentive Management on Hyperscalers

Reproducing the Issue

  1. Enable flag "Include Expired Title Groups in SCIM API Response";
  2. Users that are active might be marked as 

Cause

Currently, Incentive Management uses two queries to fetch participant-role assignments. Both queries filter by effective end dates that are set in the future; however, they do not account for participants with no effective end dates. As a result, active participants are being mistakenly excluded from the inherited role query results.

Resolution

SAP Incentive Management Engineering team is already aware of the issue and a fix is planned to be delivered on release 2605.

For release dates, please find SAP SPM 2026 Release Schedule.
