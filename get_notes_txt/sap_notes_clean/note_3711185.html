Symptom

The reconciled invoice document from our SAP Business Network account is not delivered to our Buyer ERP system via the integrated buyer side SAP Managed Gateway application.

Reproducing the Issue

Please follow the below steps to check the document routing configuration in the Business Application IDs and End Points section:

  1. Sign in to your buyer SAP Business Network account
  2. Click User-Initials at top right corner for Account Settings
  3. Select Business Application IDs and End Points (cXML and OData Setup)
  4. Select your application ID in the System ID column
  5. Select tab End Point > Configure Document Routing 
  6. Verify if the Invoice Reconciliation document type routing is correct

Cause

The End-Point configured for the Invoice Reconciliation document in the Configure Document Routing section for the given System-Id is not of your buyer side SAP Managed Gateway URL, but with some other end-point. Hence, this document will not be routed to the SAP Managed Gateway and eventually not be sent over to you buyer ERP as well.

Resolution

The correct end-point pointing to the buyer SAP Managed Gateway which is integrated with your ERP system must be selected for the Invoice Reconciliation document type option in the Configure Document Routing section for the given System-Id.
