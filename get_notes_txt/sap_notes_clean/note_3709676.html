Symptom

  • Printing the "Dashboard" view in a Calibration session results in a blank PDF.
  • The downloaded PDF does not display any content.

Environment

  • SAP SuccessFactors Calibration

Reproducing the Issue

  1. Navigate to Calibration from the Homepage and open a session.
  2. Access the Dashboard view and print the view using the print button.
  3. Open the downloaded PDF; it appears blank.

Resolution

The issue is planned to be fixed within the new 1H 2026 release upgrade:

- in preview systems on April 13th 2026;

- in production systems on May 15th 2026.
