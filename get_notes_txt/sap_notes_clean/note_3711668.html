Symptom

Payroll stops with error "Hire Date Not Known for Legal Person <juper>". This happens when running a correction payroll for an employee who didn’t show up after hiring. As described in the SAP’s Process manual for the wage return, the employee has an action record Incorrect entry (70) in infotype Actions (IT0000-MASSN). The employee has also been assigned to the special employee group (AUSNA) No Tax, Wage nor Pension Return (888) in infotype Fiscal Data (NL) (0060).

Other Terms

Wage Return, NLLA0, RPCMLAN0, RPCALCN0, PC00_M05_CALC

Action 70

Reason and Prerequisites

With this SAP Note, the fields Entry Date (LA-INDDA) and Leaving Date (LA-UITDA) will be properly filled for no-show employees.

Solution

Delivery

Import the HR Support Package relevant for your release. For more information, see SAP ERP Human Capital Management Support Package Schedules.

As a general rule SAP recommends you to implement the Support Package. However, if you need to install a solution earlier, use the SAP Note Assistant to implement the correction instructions. For more information, see SAP Note Assistant.
