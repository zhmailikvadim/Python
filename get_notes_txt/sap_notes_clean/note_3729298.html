Symptom

Filters set in the Table Options menu during project/document search are reset after navigating away and returning to the original search.

Resolution

This is expected system behavior. Default columns are predefined for each project/document type search in Search results and Recently Viewed, and filters can be applied using the Table Actions menu.

However, these filters are not retained and are reset to default after navigating away and returning to the original search.
