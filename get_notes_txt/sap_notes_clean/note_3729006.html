Symptom

In a multidimensional analytic Web Dynpro grid application, you have characteristics other than measures in the columns axis, and more than two columns in the row headers, and cell merging activated. That means that the headers for column headers already start in column 2, and are merged all the way to the rightmost header column.

When exporting the data to spreadsheet without cell merging, these header cells are repeated multiple times.

Other Terms

BICS Grid, Header Segment

Reason and Prerequisites

Programming Error

Solution

Apply this note or the corresponding support package.
