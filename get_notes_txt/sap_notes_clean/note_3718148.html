Symptom

Would like to edit the notifications for Discovery.

Reproducing the Issue

If trying to access it via Settings > Notifications > Discovery, it may not load correctly.

Cause

It must be accessed directly via User Notifications & Preferences.

Resolution

  1. Click in the user initials,
  2. Settings,
  3. User Notifications & Preferences,
  4. Discovery Notifications will appear, click in "Edit",
  5. Apply the needed changes and save it.

Keywords
