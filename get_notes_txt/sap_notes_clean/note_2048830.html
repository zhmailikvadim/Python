Symptom

  • Error ME 154 "Not possible to determine any components" shows up when creating subcontract PR (Purchase Requisitions) by using BAPI_PR_CREATE.
  • This error occurs even though note 1589772 already exists in your system.

Environment

  • Materials Management (MM)
  • SAP ERP 6.0
  • SAP ERP Core Component 6.0
  • SAP enhancement package 4 for SAP ERP 6.0
  • SAP enhancement package 5 for SAP ERP 6.0
  • SAP enhancement package 6 for SAP ERP 6.0
  • SAP enhancement package 7 for SAP ERP 6.0  

Reproducing the Issue

Any process which calls the BAPI to create the PR, or through T-code SE37 as follows:

  1. In T-code, execute BAPI_PR_CREATE.
  2. Insert data in PR Header and Item, set ITEM_CAT to L (subcontracting).
  3. Execute.
  4. You will see the error message ME 154 in return table.

Cause

The PR with subcontracting category can not be created in BAPI without business function LOG_MMFI_P2P activated.

Resolution

Please refer to SAP On-line help: MM, Integration Materials Management and Financial Accounting for more details about business function LOG_MMFI_P2P.

Keywords

ME 154, BOM
