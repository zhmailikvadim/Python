Symptom

You are using the transaction /SCWM/STAGE to create stagign warehouse task for multiple PMR documents together that contains a Cross Order relevant component item with the same material. In such a case the system should summarize all the required quantitys for all similar components. Instead the staging WT will contain the quantity of the last time processed PMR item.

Other Terms

PMR, cross order staging, quantity merge, summarization, sum qty, staging WT, quantity not summarized

Reason and Prerequisites

Program error

Solution

Implement the correction instruction or the equivalent support package.
