Symptom

In CAT2 transaction, if time is recorded for a non-working day or public holiday (with zero hours) and has not been approved yet, then user cannot cancel or delete that specific entry.

Other Terms

CAT2, Public Holiday, Non-Working Day, SAPLCATS, Zero hours

Reason and Prerequisites

For non-working days or public holidays, target hours are set to zero, and any time entries recorded on these days are logged as zero hours. Hence, removal requires a corresponding time difference between the recorded hours.

Solution

Implement the note or import the corresponding support package.
