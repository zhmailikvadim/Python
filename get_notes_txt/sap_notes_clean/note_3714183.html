Symptom

You like to define a data controller for bank area in transaction DC_MAINTAIN_DC. You notice that the assignment to the data controller is rejected by the message:  DATA_CTRL013 (Bank Area (00004) with ID '<your bank area>' is already assigned to data controller '<data controller with any other bank area assigned>').

You notice that all bank areas with the same country/region code must be assigned to exactly one data controller.

Other Terms

DC_ENTITY_RELATIONS, Data Controller, tbkk01, banks, DC_MAINTAIN_DC, Entity Type, DATA_CTRL013, Bank Area,

Reason and Prerequisites

Program error

Different Bank Areas with the same country/region code

Solution

Implement the attached Correction Instruction (CI) or the relevant Support Package.
