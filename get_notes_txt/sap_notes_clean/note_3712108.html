Symptom

Understanding the purpose of flag ADR6- DFT_RECEIV

Environment

  • Sales and Distribution (SD)
  • Logistics Execution (LE)
  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA

Resolution

  • Regarding the field DFT_RECEIV: there is no significance of this field for ADR6(Email).  Only FLGDEFAULT is valid for ADR6 which says that this entry is default entry.
  • DFT_RECEIV field is valid for ADR2 only, if a telephone is SMS-Enabled this field will be set. otherwise this field will be empty.  Please refer the SAP Help Portal below for details.
