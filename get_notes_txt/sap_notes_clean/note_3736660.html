Symptom

  • Cannot reset approved SAP Cloud ALM Requirement.
  • How to reset approval status for SAP Cloud ALM Requirement?
  • How to make changes to a Requirement with "Approved" approval status? 

Environment

  • SAP Cloud ALM

Reproducing the Issue

  1. Logon to SAP Cloud ALM.
  2. Select the Implementation tab.
  3. Select the Requirements app.
  4. Select a Project and other necessary data to filter the Requirements list.
  5. Locate a Requirement with Approval Status value "Approved".
  6. Open the Requirement.
  7. Enter "Edit" mode.
  8. Some fields cannot be edited (e.g. Description) while the Requirement has the approval status "Approved".

Resolution

  1. Logon to SAP Cloud ALM.
  2. Open the Requirements app.
  3. Open a Requirement with approval status "Approved".
  4. Click the "overflow" button (3 horizontal dots, "...").
  5. Select the "Reset Approval" option.
