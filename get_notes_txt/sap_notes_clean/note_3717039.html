Symptom

The extension extHookOnEditOverview is working only for the initial screen, all extra lines afterwards don't get this draft mark.

Environment

Fiori 2.0 HCM

Resolution

Be sure the SAP Note is implemented 3539438 - My Timesheet Fiori 2.0: Bug Fixes and Improvements

In controller extension: extHookOnEditOverview, add the below code line as well to set draft for new entries:

this.oEmptyRecordTemplate.SetDraft = true;

Keywords

hcm.fab.mytimesheet, HCMFAB_MY_TIME, F3074
