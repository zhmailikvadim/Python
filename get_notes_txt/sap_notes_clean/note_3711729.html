Symptom

After you run the Balance Sheet / Income Statement report in document and reporting compliance for Taiwan. On the preview page of the Income Statement, the value of the Company Code field is missing.

Other Terms

Taiwan Income Statement, financial statement, DRC, TW

Reason and Prerequisites

This is a program error.

The Company Code field is not mapped in the report definition.

Solution

Apply the corresponding support package or implement the correction instructions in this note.
