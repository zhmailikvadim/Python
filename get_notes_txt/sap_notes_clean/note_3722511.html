Symptom

The user can see the names of all Service Units displayed on the User Access Portal (UAP), even though they are only assigned to one Service Unit in Cloud Control Center (CCC).

Reproducing the issue

Prerequisites:

  • The Cloud Control Center (CCC) contains multiple Service Units, each with a different version.
  • A user is assigned exclusively to a tenant associated with one specific Service Unit (for example, SU1).

Scenario:

  1. Log in to the User Access Portal.
     
  2. Check the Service Unit names displayed under the List of SAP Business One Web Servers section.

Expected behavior:

Only the Service Unit SU1 is displayed.

Actual behavior:

The names of all Service Units are displayed.

Solution

SAP intends to provide a feature package or patch to solve the symptom that is described in this SAP Note. The section References below lists the related feature package or patch once they become available. Be aware that References can only be confirmed at feature package or patch release date.
