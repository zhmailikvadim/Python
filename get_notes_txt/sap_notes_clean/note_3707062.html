Symptom

When you use the excel export functionality for a simulation with a large number of components, you notice that rows are repeated numerous times in the resulting excel file.

Other Terms

F4450, pMRP, Component View, excel export, download, duplicates, duplicate rows

Reason and Prerequisites

A mistake in the code lead to rows being processed multiple times when the number of components is very large.

Solution

 Implementing this note will fix the issue.
