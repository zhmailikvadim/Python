Symptom

In Start Backup Supply Electricity (Distributor View) process(Process ID: 1091), Process 1300 (Transfer Transaction Data) is only started with Alternative process ID "DE_SBS_BSUP".  Actually starting 06.06.2025 the Alternative process ID "DE_SBS_DSO_ELEC" should be replacing "DE_SBS_DSO".

Other Terms

1300, MSCONS, 1091, GE_CHK_0011,/IDXGC/RP_MSC_PDOC_CRT_Z27, /IDEXGE/CL_CHECK_METHOD_ADD, CHECK_TRANSACTION_REASON_MSC, DE_SBS_DSO_ELEC, DE_SBS_DSO, DSO.

Reason and Prerequisites

Program Error.

Solution

Please implement the automatic corrections or import higher relevant support packages. This SAP Note is relevant for roles: DSO.
