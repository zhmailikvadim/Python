Symptom

When editing the font size in job profile section from a requisition, once clicking in preview the font size return to the default one.

Environment

SAP SuccessFactors Recruiting Management 

Reproducing the Issue

  1. Access Job Profile tab from a requisition
  2. Select one section
  3. Try to modify the font size
  4. Click preview
  5. The alteration was not saving

Resolution

Currently this scenario is under analysis by our engineering.
