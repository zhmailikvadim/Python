Symptom

  • An error occurs in the SAP Integration Suite's  interface during outbound connection using OData adapter.
  • Error message: "com.sap.it.rt.adapter.http.api.exception.HttpResponseException: An internal server error occured: HTTP Request failed with error : Connection refused: <>. The MPL ID for the failed message is : <>".

Environment

  • Integration Suite
  • Cloud Integration

Cause

  • The `Process in Pages` setting was disabled in the OData Receiver, causing multiple record retrieval requests to fail.

Resolution

  1. Enable the `Process in Pages` setting in the ODataV2/V4 Receiver.
  2. Verify that the configuration includes the Looping Process Call, as this is essential for proper operation.
