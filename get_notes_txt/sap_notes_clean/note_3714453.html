Symptom

This SAP Note adds wage types to the following PE51 forms: these wage types display the new premium amounts in the remuneration statements and the payroll journal for Child and Child-Rearing Support Fund contributions.

  • JF01: /607
  • JF02: /617
  • JF03: /627
  • JJS1: /607, /617, and /627
  • JJD1: /607, /617, and /627

For similar changes to HR Forms, refer to KBA 3728098.

Other Terms

PE51, Payslip, Monthly Payroll, RPCEDTJ0, RPCLJNJ0

Reason and Prerequisites

This is a part of legal changes. You need to apply SAP Note 3709289 first.

Solution

You can apply the HR support package or the manual changes in this SAP Note.

For more information about the development delivered with this SAP Note, see the following What's New topics on SAP Help Portal:

  • SAP ERP: Child and Child-Rearing Support Fund
  • SAP S/4HANA and SAP S/4HANA Cloud Private Edition: Child and Child-Rearing Support Fund
