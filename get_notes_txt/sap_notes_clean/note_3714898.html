Symptom

  • Unable to access process dashboard in Process Intelligence.
  • Despite having all permissions assigned to view a process, dashboard, and investigations, the user cannot see the investigation under processes.
  • Error message received: Failed to execute 'removeChild' on 'Node': The note to be removed is not a child of this node.

Environment

SAP Signavio Process Intelligence

Reproducing the Issue

  1. Go to Process Intelligence.
  2. Navigate the Processes in the left side pane.
  3. Error is received: Failed to execute 'removeChild' on 'Node': The note to be removed is not a child of this node.

Cause

Translations is enabled at the browser level.

Resolution

1. Disable the translation option in the browser being used. (Ex. Microsoft Edge)

Microsoft Edge: Go to Settings and more > Settings > Languages and verify the "Offer to translate pages that aren't in a language I read" option is enabled but needs to be disabled. 

Google Chrome: Settings -> Languages -> Use google translate, check if option is enabled. 

2. Reload page and or reopen in a new browser session to confirm error is no longer reproducible.
