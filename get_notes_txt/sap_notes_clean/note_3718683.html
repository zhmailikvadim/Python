Symptom

Can Xcelsius service running in only BW Java system be removed?

Environment

  • SAP BusinessObjects Business Intelligence platform 4.3
  • SAP BusinessObjects Business Intelligence platform 2025
         
  • MS Windows Server

Cause

  • This service is no longer needed. Xcelsius dashboards is a Flash-based application and has been "out of support"  since the end of life of Flash in 2020
  • For reference, see the SAP Knowledge Base Article: 2579982 - SAP BusinessObjects Dashboards (Xcelsius) EOL & roadmap )

Resolution

The Xcelsius service can be removed from BW Java system as Xcelsius dashboards as a product is no longer maintained or supported.

See Also

SAP Note 3640185 - [CVE-2025-42874] Denial of service (DOS) in SAP systems.

Keywords

Xcelsius service, SAP systems, BI components, SP update rejection, SUM manually prepared directory,  Maintenance Planner, Netweaver
