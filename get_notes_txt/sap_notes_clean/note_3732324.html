Symptom

After upgrading the system, the Fiori app "Manage Internal Order" does not display the success message toast upon creating an internal order.

The success message is only visible when hovering the cursor over it and disappears when the cursor is moved away. 

Environment

•Controlling (CO)
•SAP S/4HANA
•SAP S/4HANA Finance
•SAP S/4HANA Cloud Private Edition
•SAP Fiori for SAP S/4HANA

Reproducing the Issue

  1. Open the Fiori app.
  2. Search for "Manage Internal Order."
  3. Select Create.
  4. Check and save.
  5. Observe that the success message toast is not displayed immediately in the bottom-left corner and only becomes visible when hovering the mouse over that area.

Resolution

To resolve this issue, perform following steps:

1) Invalidate Fiori Launchpad caches

The following transactions should be executed in the backend system: /UI2/INVALIDATE_CLIENT_CACHES

Both Invalidate Global Caches and Invalidate User-Specific Caches should be selected.

/UI2/CHIP_SYNCHRONIZE_CACHE

/UI2/DELETE_CACHE_AFTER_IMP

These actions ensure that Launchpad and UI5 caches are refreshed after the system upgrade.

2) Clear browser cache

The browser cache should be cleared, or a hard refresh (Ctrl + Shift + R) should be performed.

The scenario should then be tested again.

3) Regenerate UI5 themes

If the behavior continues, the UI theme should be regenerated using transaction: /UI5/THEME_TOOL

The application should then be tested again.
