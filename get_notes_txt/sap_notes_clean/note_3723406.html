Symptom

  • The tile F0545A is missing in the system.
  • Only the target mapping for the tile is available

Environment

  • Materials Management (MM)
  • SAP S/4HANA
  • SAP Fiori

Reproducing the Issue

Access launchpad try to find App: F0545A 

Cause

My Inbox - Approve Supplier Invoice is not a tile-kind-of app. It is an adaptation of the standard My Inbox app to the Supplier Invoice scenario.

Resolution

Please create the tile manually and add the corresponding app launchers provided in the Fiori Apps Library:

Approve Supplier Invoices        scenarioId=SIV_RELEASE&sap-ach=MM-FIO-IV        

as explained in this note:

2827346 - How to configure My Inbox scenario-specific tile for Task Gateway providers - SAP for Me

Please also check  SAP Note 2371741 for more information
