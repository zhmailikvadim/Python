Symptom

This SAP note corrects the labels of following USC related data element:

  • PIE_USC_TR_MR1
  • PIE_USC_TR_MR2
  • PIE_USC_TR_MR3
  • PIE_USC_TR_MR4
  • PIE_USC_TR_WR1
  • PIE_USC_TR_WR2
  • PIE_USC_TR_WR3
  • PIE_USC_TR_WR4
  • PIE_USC_TR_YR1
  • PIE_USC_TR_YR2
  • PIE_USC_TR_YR3
  • PIE_USC_TR_YR4

Other Terms

IE, USC, label, PIE_USC_TR_MR1, PIE_USC_TR_MR2, PIE_USC_TR_MR3, PIE_USC_TR_MR4, PIE_USC_TR_WR1, PIE_USC_TR_WR2, PIE_USC_TR_WR3, PIE_USC_TR_WR4, PIE_USC_TR_YR1, PIE_USC_TR_YR2, PIE_USC_TR_YR3, PIE_USC_TR_YR4, UDO, NOTE_3728793

 

Solution

To resolve this error in your system, apply the source code corrections located in the Correction Instructions section of this SAP Note. Alternatively, apply the HR Support Package(s) indicated for your release under Reference to Support Packages.

On SAP_HR 604 and SAP_HR 608 the automatic correction creates UDO report: NOTE_3728793, it has to run in Update&Activate mode via SE38 transaction.
