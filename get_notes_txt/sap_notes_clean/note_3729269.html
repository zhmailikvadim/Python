Symptom

Message no. 3G141 'According to control record for PArea &1, payroll is not completed' is triggered when executing posting program RPCIPE00. "Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental." 

Environment

  • Human Capital Management (HCM)
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP SuccessFactors Employee Central Payroll
  • SAP S/4 HANA

Resolution

Message no. 3G141 is triggered when following criteria is met:

  1. Type of off-cycle reason in table T52OCC is O (others) or blank.
  2. Payroll status in transaction PA03 is not ‘Exit payroll’.
  3. System type is production.

Please check above configuration.
