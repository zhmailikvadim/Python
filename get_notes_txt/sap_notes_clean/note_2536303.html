Symptom

It takes a long time to load library contents over 30k.

Other Terms

Library.

Reason and Prerequisites

Performance issue.

Solution

Please upgrade component BPC4HANA100 to Support Package 03.

Other Terms

MPP-optimized DSO data activation, DB6_MPP_DSO_TABLES_COLOCATED, RDR index, RSDU_FASTSTORE_CHECK_UNIQU_DB6, BLU Acceleration, currently committed, CS, DataStore object

Reason and Prerequisites

All of the following are true:

  • You use BLU Acceleration for some SAP BW objects and therefore you have configured your database and application server to use currently committed semantics (database configuration parameter CUR_COMMIT=ON, SAP profile parameter dbs/db6/dbsl_ur_to_cc = *).
  • You also use the Db2 database partitioning feature and have configured your standard DSOs so that the MPP-optimized data activation procedure is used.
  • You have set the RSADMIN parameter DB6_MPP_DSO_TABLES_COLLOCATED to YES.

Solution

    Implement Support Package 20 for SAP BW 7.30 (SAPKW73020) into your BW system. The Support Package will be available as soon as SAP Note 2733813 with the short text "SAPBWNews 7.30 BW ABAP SP20", which describes this Support Package in more detail, is released for customers.
     
  • SAP BW 7.31 (SAP BW 7.3 EHP 1)
    Implement Support Package 25 for SAP BW 7.31 (SAPKW73125) into your BW system. The Support Package will be available as soon as SAP Note 2717533 with the short text "SAPBWNews 7.31 BW ABAP SP25", which describes this Support Package in more detail, is released for customers.
     
  • SAP BW 7.40
    Implement Support Package 22 for SAP BW 7.40 (SAPKW74022) into your BW system. The Support Package will be available as soon as SAP Note 2718124 with the short text "SAPBWNews 7.40 BW ABAP SP22", which describes this Support Package in more detail, is released for customers.
     
  • SAP BW 7.50
    Implement Support Package 15 for SAP BW 7.50 (SAPK-75015INSAPBW) into your BW or AS ABAP system. The Support Package will be available as soon as SAP Note 2731707 with the short text "SAPBWNews 7.50 BW ABAP SP15", which describes this Support Package in more detail, is released for customers
