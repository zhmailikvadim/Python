Symptom

The virtual registers will be installed and changed if the system parameter DEVICE_DIR_USAGE_TYPE is set as '1-Only Devices'. During device change process 3070, the virtual register is removed one day later than device change. This is wrong activity.

Other Terms

DEVICE_DIR_USAGE_TYPE, 3070, 0410

Reason and Prerequisites

 This is a program error.

Solution

Please implement the correction instruction in the SAP Note or install further Support Package.
