Symptom

When sending INVOIC message for MeMi or Grid Usage Billing, wrong PoD is determined in case the external UI of PoD is changed.

Other Terms

US4G, Overtake and Undertake Quantities Billing, MeMi Document, Grid Usage Billing, GUB, Budget Billing, INVOIC, PoD, MaLo, External UI, EXT_UI, EUITRANS, LOC+172, 31002, 31004, 31005, 31006, Key Date, /US4G/CL_UBD_INVDOC_UTIL_GU, /US4G/CL_UBD_INVDOC_UTIL_MEMI, GET_EXTUI_FROM_INTUI, DSO

Reason and Prerequisites

 Program Error

Solution

Please implement the automatic corrections or import a higher relevant support package.
