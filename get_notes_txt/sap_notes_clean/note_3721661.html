Symptom

  • SAP transaction SQ01 fails and results in a dump when attempting to open, modify, or delete a specific query.
  • The issue occurs during the preparation of the system for conversion to S/4HANA.

Environment

SAP NetWeaver Application Server for ABAP

ABAP Platform

Reproducing the Issue

  1. Navigate to transaction SQ01.
  2. Attempt to open, modify, or delete the query.
  3. Observe that the transaction fails and results in a dump.

     Category               ABAP programming error
     Runtime Errors         ASSIGN_LENGTH_0 
     ABAP Program           SAPMS38R

Cause

Resolution

  1. Execute the report RSAQ_REPAIR_DBFF.
  2. Verify if the issue persists after running the report.
