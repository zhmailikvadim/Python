Symptom

The exchange rate field for flows can be saved to the database with a positive or a negative value, however in the external data transfer it was previously not possible to transfer negative values.

This note adds a new field to the sender structure (S_KURS2). If a minus sign (-) is entered in this field, the external data transfer will save the value in KURS2 as a negative exchange rate. If the the field is left blank, KURS2 will be saved in the same was as previously, with a positive value.

Other Terms

KCLJ KCLT KCLL EDT J11SDAB

Reason and Prerequisites

This was not previously provided

Solution

Implement the attached correction instructions or import the specified Support Package

If you have already defined the transfer structure using transaction KCLL, you have to delete this and then create it again so that the changes take effect.
