Symptom

  • The "Settings for Change History" Fiori tile does not open.
  • The tile remains on a loading screen showing three dots indefinitely.
  • No error message or error code is displayed.

Environment

  • Product: SAP Integrated Business Planning
  • Settings for Change History Web UI

Reproducing the Issue

  1. Open the Fiori launchpad.
  2. Select the "Settings for Change History" App.
  3. Observe that the screen shows three dots and the application does not load.

Cause

The reason is that no key figure in the planning area has change history enabled. 

Resolution

Activate the change history at least for one key figure
