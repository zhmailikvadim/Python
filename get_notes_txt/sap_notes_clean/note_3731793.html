Symptom

In the Report Incident app (F1992) on mobile devices, the fields for entering person names, such as Injured Person (Was anyone injured?) and Witness (Were there any witnesses?), do not provide value help. Users enter names as free text.

After implementing this SAP Note, the system displays value help for these fields on mobile devices. Users can select business partners from the system.

Other Terms

Incident Management, Person value help, Create Incidents, Injured Person, Witness

Reason and Prerequisites

Prerequisites: For information about the validity of the corrections, see the correction instructions.

Solution

The support packages that contain this correction are listed in the "Support Packages" section.

Alternatively, you can implement the correction instructions of this SAP Note.
