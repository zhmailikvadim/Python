Symptom

  • Error message: "Task type train is not supported for SAPGENAI"
  • AI-assisted sales order fulfillment for Private Edition encountered issues during intelligent scenario training

Environment

  • Intelligent Scenario (ISLM)
  • SAP S/4HANA Cloud Private Edition 2023

Reproducing the Issue

  1. Attempt to train the intelligent scenario SD_INT_AI_GPT or SD_INT_AI_GEMINI.
  2. Encounter error message: "Task type train is not supported for SAPGENAI".

Cause

  • Perform Sales Order Field Changes and Resolve Fulfillment Issues | SAP Help Portal 
  • Connection mapping of the Intelligent Scenario is not yet done.

Resolution

To train and active the ISLM required by Perform Sales Order Field Changes and Resolve Fulfillment Issues | SAP Help Portal , first, Pre-Requisites setup of ISLM is required. Then, Configure an Intelligent Scenario setup is required. Before performing Configure an Intelligent Scenario, make sure Requesting Access to SAP Business Technology Platform (SAP BTP) is done.
