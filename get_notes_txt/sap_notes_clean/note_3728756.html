Symptom

When you exclude invoices during mass processing of source documents using transaction /IRM/IPCRCLBSMP, you expect the agreement accrual bucket to be updated with all invoices except the excluded ones. However, the system updates the bucket with only the excluded invoices instead of excluding them. This results in incorrect bucket data and processing behavior.

This note corrects the behavior by ensuring that excluded invoices are properly ignored and only the intended invoices are processed in the calculation bucket.

Other Terms

/IRM/IPCRCLBSMP, exclusion issue, mass processing, source documents, excluded invoices, calculation bucket update, incorrect processing

Reason and Prerequisites

This is caused by program error

Please implement SAP Note#3630903 as a Prerequisite note

Solution

Please implement the correction instructions attached. These corrections are only valid if you have applied the Support Packs.

Release V/4 -  2023 V2, 2023 V3 and 2023 V3 Support Pack 1

Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object.
