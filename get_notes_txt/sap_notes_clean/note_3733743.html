Symptom

This article describes functionality introduced in the May 2026 SAP Fieldglass release

You may notice changes in how your Statement of Work (SOW) list views appear in SAP Fieldglass.
These changes can include:

  • The Sequence # column no longer appearing in list views
  • Purchase Order (PO) lines appearing grouped with related events and charges
  • Differences between what buyers see and what suppliers see
  • More fields appearing on the SOW details page than in list views
  • Some lines appearing grouped while others are not

These changes may make the SOW layout look different than what you were previously used to

Reproducing the Issue

This behavior can occur when you:

  • View SOW list views after a recent update
  • Review SOWs that contain multiple PO lines, events, or charges
  • Compare buyer list views to supplier list views
  • Open the SOW details page after reviewing a list view
  • Work with SOWs that include a mix of items with and without Event Hierarchy (EH)

Cause

Recent enhancements in SAP Fieldglass improved how SOW information is displayed and organized:

  • When your organization uses Buyer‑Defined Sequence Number, the system hides the system‑generated Sequence # in buyer list views to reduce duplicate information
  • Supplier views are not affected and continue to display Sequence #
  • The SOW details page always displays all sequence fields to provide full visibility
  • PO lines, events, and charges are automatically grouped when they are connected through Event Hierarchy (EH)
  • Items not associated with Event Hierarchy continue to display using existing behavior, which can result in mixed grouping within the same SOW

Resolution

What This Means for You

  • If Sequence # is not visible in buyer list views, this is expected behavior when Buyer‑Defined Sequence Number is enabled
  • You can manage which columns appear in list views using Configure List Views → Events
  • Differences between buyer and supplier views are expected and intentional
  • The SOW details page is designed to show more information than list views
  • Grouping improves readability but applies only to items connected through Event Hierarchy

If something does not match your expectations, contact your SAP Fieldglass administrator or SAP Fieldglass Support for assistance.
