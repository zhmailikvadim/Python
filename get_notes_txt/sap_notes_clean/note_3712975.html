Symptom

This note delivers several Unified Rendering related fixes for the frameworks WD JAVA and SAP Enterprise Portal (only theming related).

Find the complete list of fixed issues in SAP note 3703267 - Cumulative patch for Unified Rendering 2602. See also reference section.

Other Terms

Web Dynpro Java, WDJ,  Unified Rendering, UnifiedRendering, UR, Lightspeed

Reason and Prerequisites

This note is valid for:

  • WD JAVA starting from 7.30 and above. 
  • SAP Enterprise Portal starting from 7.30 and above in case of using a custom theme.

For WD AJAV/Portal < 7.30 a different process is in place.

Solution

As part of regular intense quality- and patch creation tasks, all Unified Rendering fixes will be collected and shipped about every four weeks. Detailed information about the process can be found in SAP note 2500800 - General information about cumulative patches for Unified Rendering.

Common information about the WD JAVA and Portal fix proces ca be found in SAP note 2503680 - WD JAVA & Portal: Unified Rendering Update - Instructions And Related Notes.

The issue listed under section "References" will be fixed by the rendering related Web Dynpro for Java patch. In the section "Support Packages & Patches" of the note you will find the relevant patches for the different NetWeaver releases and SPs.

The software components that will install the correction for the reported issue are “AJAX-RUNTIME” and “SAP-BASIS” (or “FIORI LAUNCHPAD PORTAL” for NetWeaver 7.50). Additionally, it will be necessary to install all the software components that were found by the dependency analysis.

Install this patch or a newer one (patches are always cumulative).

The above SAP note will be updated automatically with the respective patch number once it is available on the SAP Support Portal. If the section " Support Packages & Patches" or your NetWeaver release and SP are missing then the patch creation is still in progress. In this case, please re-check this note at a later time.

You can subscribe to the aforementioned note in order to be notified when it is updated with the according WD Java patch number. How to subscribe to a note is explained in SAP Knowledgebase article 2171560 - How to be notified of new or updated SAP notes or KBA's in the ONE Support Launchpad.

Using a custom theme:

In case of using a custom theme in Enterprise Portal it might be necessary to

  • update ep-basis.sca (>= 7.30 && <= 7.40)  or EPFLP.sca (>= 7.50 SP2)
  • regenerate the custom themes including style adjustments

If these steps are not done it can happen that the custom theme is not used by the backend or looks distorted.
