Symptom

Mail Adapter of SAP NetWeaver Process Integration (PI) makes use of Open Source Nimbus JOSE + JWT Library. For security and compatibility reasons, this Open Source was updated.

Other Terms

Open Source, Netweaver, PI PO, Mail Adapter, Sender Channel, Receiver Channel, JWT

Reason and Prerequisites

Open Source Libraries are regularly updated.

Solution

Code changes have been performed to include libraries with latest versions.

To get this feature available in your Support package you need to apply the patch matching to your support package version as per the instructions in the NetWeaver Support Package Stack guide which can be found on the SAP Service Marketplace as described in SAP Note 952402.
