Other Terms

Reason and Prerequisites

This SAP Note contains a report to automate the creation/modification of objects that are used in the solution delivered via SAP Note 3701228. The list of created/modified objects can be found in the "Solution" section.

The implementation of this SAP Note alone has no impact on your existing functionality until the solution SAP Note is applied.

SAP Note 3701228 requires this SAP Note to be implemented as a prerequisite.
