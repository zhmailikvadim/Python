Symptom

You have configured a Planning Profile to set the preferred position of an empty stage (e.g., "Add at end of capacity documents").

However, when you open the Transportation Cockpit and access the Change Planning Settings dialog box, the Preferred Position of Empty Stage field incorrectly displays the default value ("Add at beginning of capacity documents") instead of the value from your selected Planning Profile.

Reason and Prerequisites

Program error

Solution

Implement the attached correction instructions or install the corresponding Support Package.
