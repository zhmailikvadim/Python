Symptom

If a 'Number' type Data Field is set up and assigned to a NC Data Type then a Nonconformance Code, the 'Nonconformance Data Entry' POD plugin will let the operator scroll the field using the mouse wheel, which can also result in negative numbers.

Environment

SAP Digital Manufacturing

Reproducing the Issue

Prerequisites:

  • Set up a 'Number' type Data Field in the Manage Data Fields app.
  • Set up an NC Data Type using the Data Field from before in the Manage Data Types app.
  • Set up a Nonconformance Code with the Data Type from before in the Manage Nonconformance Codes app.
  • Have an SFC ready to process Nonconformance against.

Reproduction:

  1. Open a POD with Nonconformance Data Entry plugin.
  2. In the Nonconformance Data Entry plugin, select the NC Code with the Number Data type.
  3. Click into the Data Field and scroll.
  4. Result: Number goes up or down based on scroll direction and can go into negative numbers.

Cause

The Data Field type 'Number', is a simple numeric field which can be filled with negative numbers and is scrollable in POD plugin(s).

Resolution

This is working as designed.
