Symptom

When you execute report Job Applications Transferred from SuccessFactors bizX (RH_SFI_RECRUIT_PROCESS_APPL, Transaction: HRSFI_RCT_HIRE) to display all the applications Transferred from Successfactors and click on 'Start Action' for a candidate. At the end of the hire action, the candidate's record is still displayed on the screen with status Start Action. When you press the Refresh button, the record now disappears from the screen. Ideally it should have disappeared after completion of Hire Action.

Other Terms

Hire, HRSFI_RCT_HIRE, SFSF, Integration, Applicant, SuccessFactors, RH_SFI_RECRUIT_PROCESS_APPL, BizX, RH_SFI_RECRUIT_PROCESS_APP_L02, Start Action

Reason and Prerequisites

This is due to program error. Refresh of list was happening after Hire Action.

Solution

To resolve this error in your system, apply the source code corrections contained in the attached Correction Instructions of this Note.
