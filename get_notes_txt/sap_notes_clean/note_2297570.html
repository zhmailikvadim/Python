Symptom

Until now, the system does not provide consumer select hooks for these use cases:

- get segment result selection

- get segement dynamic content

Other Terms

GSEG, CSEG, Segmentation, SEG, ADT, Audience Discovery Targeting, CEI, select consumer

Reason and Prerequisites

Downport of functionality

Interface IF_GSEG_SELECT_CONSUMER gets 4 new methods:

  • BEFORE_RESULT_SELECTION          Adapt meta data before selection of segment/BB result SQL
  • AFTER_RESULT_SELECTION           Use meta data after selection of segment/BB result SQL
  • BEFORE_SEGMENT_DYNAMIC_CONTENT   Adapt meta data before selection of segment content
  • AFTER_SEGMENT_DYNAMIC_CONTENT    Use meta data after selection selection of segment content

Find the method documentations in the attachment section of this note.
