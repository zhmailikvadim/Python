Symptom

  • The reference catalogue for IT components is missing data.
  • For example: a missing provider, or a missing version of a provider.

Environment

SAP LeanIX - IT component reference catalog

Cause

Resolution

  1. Missing data in the IT component reference catalogue should be requested via the production workspace.
  2. Refer to the help center article titled "Report Missing Data in Catalog Items" for instructions:

Report Missing Data in Catalog Items.

  1. Review the response times per IT component tier in the documentation:

Tiers for IT Components in the Catalog.
