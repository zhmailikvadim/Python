Symptom

You require information on Audit Logging for an Enterprise Cloud Services (ECS) system.

Environment

Enterprise Cloud Services

Resolution

ECS Customers can navigate to:
https://help.sap.com/docs/RISE_WITH_SAP_PRIVATE_CLOUD_EDITION

When you open the page if you are not logged on you will see:

"This page is intended for SAP private cloud customers only. Log in using the padlock icon on the right side of the page to view content. For access, send a message to: SAP_Private_Cloud_Help_Portal_Admin@sap.com - include your customer ID, email address and/or S-user"

You must "Log in using the padlock icon on the right side of the page to view content" then:


Under section "Security" find document:
ABAP SAL - Audit Logging in Private Cloud


To create a service request for an ECS system use "Create Service Request" as outlined in the Note below:
2720477 (How to access ECS Service Requests - SAP ONE Support Launchpad / SAP for Me)

To create a case for an ECS system see Note:
3399927 (How to create a case when support is needed for SAP Enterprise Cloud Services (ECS) system)
