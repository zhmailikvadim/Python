Symptom

  • An external sales order ID is displayed in SSCV2 but is not clickable, preventing direct navigation to the external system.

Environment

  • SAP Sales Cloud Version 2
  • SAP Service Cloud Version 2

Reproducing the Issue

  1. Open sales quote that has an external order created from it.
  2. Navigate to the "Sales Orders" section.
  3. Observe that the external order ID is displayed but is not clickable.

Cause

Missing configuration of external links

Resolution

  1. Navigate to settings -> Integration -> External Link.
  2. Click on "+" to add a new configuration.
  3. Set "Sales object" to "Sales Order".
  4. Select the appropriate communication system from the drop-down menu.
  5. Add the external link:
    • For Public Cloud it would be in format https://(S4 URL)/ui#SalesOrder-manageV2?SalesOrder=${displayId} .
    • For S/4 private Cloud : https://tenant.s4hana.ondemand.com/ui#SalesOrder-display?&SalesOrder=${displayId}
  6. Save the configuration.
