Symptom

  • There are 4 folders available in XDL's SFTP
  • Each folder has their own set of permissions

Environment

  • SAP Sales Performance Management
  • Incentive Management on Hyperscalers

Cause

For each folder there is a set of permissions available by default.

Resolution

  • Inbound folder: 
    • List
    • Download
    • Upload
    • Overwrite
    • Rename Files
  • Outbound folder:
    • List
    • Dowanload
    • Rename Files
    • Delete Files
  • Archive folder:
    • List
    • Download
  • Badfiles folder:
    • List
    • Download

Keywords

Hyperscalers, SFTP, K8S, XDL, Permissions, Folders, Folder, Permission, inbound, outbound, archive, badfiles
