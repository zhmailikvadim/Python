Symptom

You use Easy Cost Planning to create plan costs for WBS elements (tcode CJ20N or CJ9ECP) and update of ACDOCP is enabled.
After saving the data you cann see that the cost estimate has ben created and table COSP and/or COSS have been updated, but plan data is missing in table ACDOCP.

Other Terms

CJ20N, CJ9ECP, ACDOCP

Reason and Prerequisites

If an error is raised during ACDOCP update then the data might not be written into ACDOCP. There is no information message about this error.

Solution

Implement the attached correction. Afterwards the user will be notified by an information message if ACDOCP update fails.
