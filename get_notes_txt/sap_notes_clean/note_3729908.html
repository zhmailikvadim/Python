Symptom

When you create a waste transportation document, the setting Transporter Is Required in the Specify Waste Document Types customizing activity isn't respected. The system always requires at least one transporter, regardless of the configuration.

Reason and Prerequisites

Reason: Program error.

The validity of the correction is maintained in the correction instructions.

Solution

The support packages that contain this correction are listed in the "Support Packages" section.

Alternatively, you can implement the correction instructions of this SAP Note.
