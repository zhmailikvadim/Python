Symptom

  • HRAC (HRA/CLA/COA) application is not appearing in the dropdown.
  • Employee not able to see the various IT declaration types in the dropdown in Payroll India ESS application.
  • Only one application "Previous Employment Tax Detail" is coming in dropdown for IT declaration types.
  • Section 80C Deductions application is not appearing in the dropdown.

Environment

  • Human Capital Management (HCM)
  • Payroll India ESS ITD
  • SAP R/3
  • SAP ERP Core Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA

Reproducing the Issue

  1. Employee Logs into the Portal.
  2. Access the Employee Self Service tab.
  3. Access the Benefit and Payment tab.
  4. Access the "IT declaration".
  5. Dropdown doesn't contain required IT declaration types e.g. HRA

Resolution

  1. Go to Transaction SM30.
  2. Select the view V_T77WWW_GATYP.
  3. Enter Solution Type as '00002'.
  4. Enter General application Type say example : 'HRAC'.
  5. Tick the 'IS ACTIVE' field as shown below:
