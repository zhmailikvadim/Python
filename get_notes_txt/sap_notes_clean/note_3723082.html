Symptom

When changing the invoice date or posting date in transaction FV60, which, based on the system settings, changes the currency translation date, the system displays error SG105 "Enter rate & / & rate type & for & in the system settings". This occurs even though the correct exchange rates are maintained in OB08. Specifically, the error message does not display the rate type.

The error can occur in cross-company code postings in document currency other than local currency, where company codes have the same local currency:

  • In parked documents when changing dates via FBV0/FBV2/FBV3
  • In new documents in FV50/FV60/FV70 before parking, when values are entered and confirmed, and then the dates are changed
  • When the exchange rate is manually changed in the Local Currency tab for a parked document

Other Terms

KURST, KUTY2, KUTY3, exchange rate type, FSKB_SUBSCREEN_ITEMS_IMP_PAI, foreign currency, SG 105

Reason and Prerequisites

Program error.

Solution

As a general rule, SAP recommends that you implement the Support Package specified for your release. However, if you need to install a solution earlier, use the  SAP Note Assistant to implement the correction instruction. You can find more information about the SAP Note Assistant in SAP Support Portal, under SAP Note Assistant.
