Symptom

  • I have a custom field that automatically populates, but the value needs to be updated.
  • The default value of a custom field needs to be updated, but I don't know how.

Resolution

First off, identify the custom field that requires an update, then:

  1. Log into your account.
  2. Click Admin configuration> From Configuration click Custom Field.
  3. Search for the relevant custom field, make sure it's the Primary Custom Field, and open it.
  4. Find the Default Value that required an update.
  5. Click Edit and make the needed changes.
  6. Click Update to save the changes.
