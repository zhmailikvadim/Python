Symptom

You use Transaction Manager of Treasury and Risk Management. You create a Letter of Credit (Product Category 850) or Bank Guarantee (Product Category 860) transaction. You run TBB1 to post some, but not all flows. You rollover the transaction. In FTR_EDIT Change mode, in the Cash Flow tab, Reverse mode incorrectly displays posted flows from a previous transaction activity.

Other Terms

CL_FTR_CF, IS_FLOW_REVERSABLE, FTRTLC02, FTRTBG02

Reason and Prerequisites

This problem is caused by a program error.

Solution

Preliminary correction via Correction Instruction or implementation of the corresponding Support Package.
