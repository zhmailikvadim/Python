Symptom

Dump appears during the processing of such an inbound delivery where the ERP communication indicator is set to B on header level and there is no ERP delivery reference existing.

Other Terms

/SCWM/CL_ERP_INT_CONF GET_CTRL_PARAM

Reason and Prerequisites

The issue occurs if the PDI contains batch hierarchy in such case unnecessary CI message is being determined for the main item, which triggers the "Send ERP Delivery Messages" PPF and it's execution will generate the short dump due to missing ERO reference.

Solution
