Symptom

In Air Freight Booking on the 'Booking' tab, when "Freight Agreement/ Version" field is filled with valid details, there is a link created next to the input field stating the document number.
When this link is clicked, the following error message is displayed instead of navigating to the document: "No business document type exists for document; navigation not possible".

Other Terms

FAG_ID_TXT

Reason and Prerequisites

Program Error

Solution

Implement the changes as described in the attached correction instruction or install the relevant support package
