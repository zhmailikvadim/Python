Symptom

You may see the below error when 7 bytes or less characters are specified as System Administrator's Password in ASE installation.

Configure New SAP ASE
---------------------
SAP ASE Name (Default: TESTHOST): testase
System Administrator's Password:
Confirm System Administrator's Password:
The 'System Administrator's Password' must have at least 8 characters.

Environment

SAP Adaptive Server Enterprise 16.1 SP00 PL01

Cause

The "minimum password length" configuration option is changed from 6 to 8 on SAP ASE 16.1 SP00 PL01.

Resolution

If you would like to use 7 bytes or less characters as System Administrator's Password, please operate as follow:

  1. Specify 8 bytes or more characters as System Administrator's Password in ASE installation
  2. Change "minimum password length" configuration option to 7 or lower

    1> sp_configure "minimum password length",6
    2> go

  3. Change System Administrator's Password
    In the below sample, sa login password is changed from sybase12 to sybase

    1> alter login sa with password sybase12 modify password sybase
    2> go

Keywords

System Administrator's Password, minimum password length
