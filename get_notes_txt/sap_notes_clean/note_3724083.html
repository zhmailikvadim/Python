Symptom

When you set an outbound message with the parameter class /APE/CL_PARAM_TRANS_OUT->MV_TO_BE_AGGREGATED, the message is in status 'Queued (QUEUE)' instead of 'To Be Aggregated (AGGR)'.

Other Terms

Monitor Messages, To Be Aggregated, AGGR, /APE/CL_PARAM_TRANS_OUT, MV_TO_BE_AGGREGATED

Reason and Prerequisites

Program Error.

Solution

Implement the SAP Note or install further Support Package.
