Symptom

You post goods movements for production orders with picking list of transaction MB26. Incorrect goods movements for multiple reservations occur and the system reacts with a popup 'Error posting the goods movements'.

After you push the 'Yes' button to correct the errors, the system does not display all incorrect goods movements.

Other Terms

Pick list, MB26, PP_PICK_LIST, production order, process order

Reason and Prerequisites

The issue has been caused by a program error.

Solution

Implement the attached correction instructions.
