Symptom

Item currency for the currency conversion was incorrectly determined when multiple Requirement Coverage Requests supplied. Issue occurs after SAP Note 2877226 due to changed sy-tabix.
As result, corrections from SAP Note 2084753 always process the 1st entry.

Additionally, RFC interface specific special exceptions (SYSTEM_FAILURE, COMMUNICATION_FAILURE and OTHERS) were removed from BBP_STATUS_READ FM interface declaration.

Other Terms

META_STATUS_READ, BBP_GET_STATUS_2, VAL_PO_E, E 009 Error during currency conversion.

Reason and Prerequisites

Program error.

Solution

Implement the correction instructions from this SAP Note or upgrade to the corresponding Support Package.
