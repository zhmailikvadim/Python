Symptom

You are using the SAP Fiori App - Advanced Scheduling Board (ASB) in Production Planning and Detailed Scheduling (PP/DS) for SAP S/4HANA and you notice the below issues in 2025 FPS1 release

  1. When executing custom select option, objects are not selected back in the same chart in which user had initially selected them.
  2. Conversion indicator is missing for operations in resource/operation network charts.

Solution

As general rule, SAP recommends that you install a solution by applying a Support Package. However, if you need to install a solution earlier, use the Note Assistant to implement the correction instruction.

Please implement the front end Note 3721392 as a post implementation step.
