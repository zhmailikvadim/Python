Symptom

When editing a recurring deduction record related to benefits, the forward propagation behavior does not happen as expected. 

Environment

SAP SuccessFactors - Employee Central

Reproducing the Issue

  • Make any change that would trigger the forward propagation
    • If the values are different, the expected behavior would be to stop the forward propagation
  • See that the forward propagation did not work and all the records are propagated

Resolution

This is expected behavior. Forward propagation is not supported for changes derived from Benefit Events. 

An enhancement request can be raised for this and it will be evaluated for future releases (learn more at 2090228 - How to submit enhancement ideas for SAP SuccessFactors Products).
