Symptom

Users need to download existing metadata from an SAP Embedded Analytics application but are unsure how to do so.

Environment

  • SAP Sales Performance Management
    • Embedded Analytics

Resolution

  • Construct the metadata URL using the following format and paste it into a web browser:

    https://<sampletenant>.authentication.<dc>.hana.ondemand.com/saml/metadata

    where sampletenant represents the SAP Embedded Analytics tenant subdomain.

    DC represents the data center (region) where the tenant is hosted.

  • The data center can be identified from the SAP Embedded Analytics application URL, for example:

    https://<sampletenant>.<dc>.sapanalytics.cloud

  • Accessing this URL in a browser will automatically download the SAML metadata of the SAP Embedded Analytics application.
