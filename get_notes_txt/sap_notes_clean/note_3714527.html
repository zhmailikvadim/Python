Symptom

Create a code list restriction for a field of any entity (e.g: contacts, guided selling etc.). It works in the UI but in the Outlook Server Side Add-In the same code list restriction is not working. 

Environment

 SAP Sales and Service Cloud V2

Cause

Sales and Service Cloud V2 Outlook Add-In does not support code list restriction. 

Resolution

Code list restriction are not supported in the add-in as they work on the UI layer only. This is a system behavior.
