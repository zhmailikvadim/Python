Error messages

The Electronic File Manager runtime and Electronic File Manager: Format Definition now provide clearer and more actionable error messages. You can see more precise context and root‑cause details, and duplicate messages are avoided.

 

Electronic File Manager: Format Definition - Conditions Section Resizing

When editing complex conditions for a node, it is now possible to resize the conditions section so that all conditions are visible on the screen.

 

Minimum Supported SAP Business One Version in Mapping

When a mapping is loaded into the Electronic File Manager: Format Definition, a new read-only property Minimum SAP Business One Version Required is displayed in the main format explorer node properties. This value shows the earliest SAP Business One version that is needed to open and run the created mapping.

 

Since Electronic File Manager: Format Definition remains backward compatible, you can continue opening older mappings. When you save a mapping, Electronic File Manager: Format Definition updates the mapping version (model) to the latest version. When the mapping is saved, a system message informs you about the SAP Business One version required to run the updated mapping.
