Symptom

In app Monitor Purchase Contract items , the search help of material group filter does not return any data

Environment

  • SAP Fiori
  • SAP S/4HANA

Reproducing the Issue

  1. Access Monitor Purchase contract Items app
  2. Click the search help button of filter material group
  3. no data returns

Cause

  • The issue is because that access user is missing authorization for authorization object M_MATE_WGR

  • Search help of material group is from CDS View : C_MM_MaterialGroupValueHelp, in the DCLS of C_MM_MaterialGroupValueHelp
    authorization object M_MATE_WGR is checked with activity '03' or 'F4'

Resolution

Please assign user for authorization object M_MATE_WGR with '03' or 'F4' activity

See Also

@EndUserText.label: 'role_label'

@MappingRole: true

// 3120323 - added MaterialAuthorizationGroup instead of MaterialGroup for Auth Check

define role C_MM_MaterialGroupValueHelp {

grant select on C_MM_MaterialGroupValueHelp

where ( MaterialAuthorizationGroup ) =

aspect pfcg_auth ( M_MATE_WGR,

BEGRU,

actvt = '03' )

or

( MaterialAuthorizationGroup ) =

aspect pfcg_auth ( M_MATE_WGR,

BEGRU,

ACTVT = 'F4' )

;

}
