Symptom

Component: Foundation for EHS Management
Module: Chemicals

You cannot extend the following application actions in the business object Chemical (Technical ID: EHFND_CHEMICAL) on the node Revision:
- Set the status to historic
- Set the status to in process
- Set the status to new
- Set the status to released
- Set the status to void

Other Terms

EHFND_CHEMICAL, SET_STATUS_VOID_REVISION,SET_STATUS_RELEASED_REVISION, SET_STATUS_NEW_REVISION, SET_STATUS_IN_PROCESS_REVISION, SET_STATUS_HISTORIC_REVISION, CL_EHFND_CHM_REV_A_SET_STATUS

Reason and Prerequisites

The configuration of Business Object Chemical, node Revision contains an error.

For prerequisites, see the correction instructions.

Solution

After implementing this SAP Note, the system enables you to enhance the application actions of the Chemical Revision.

Alternatively, import the corresponding Support Packages for the component extension of SAP EHS Management or the Feature Package Stacks for SAP S/4HANA.
