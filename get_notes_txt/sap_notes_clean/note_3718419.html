Symptom

You maintain company code cata for a customer or a vendor. The field "Clearing with vendor / customer" (KNB1-XVERR / LFB1-XVERR) is not available on the GUIBB if you have not maintained the field Supplier / Customer (KNA1-LIFNR / LFA1-KUNNR) before.

Other Terms

MDG-CUS MDG-SUP MDGC MDGS

KNB1-XVERR LFB1-XVERR

KNA1-LIFNR LFA1-KUNNR

Reason and Prerequisites

The behavior has changed with S/4HANA. Now you will get a warning if you set the flag "Clearing with vendor / customer" (KNB1-XVERR / LFB1-XVERR) and the field supplier / customer (KNA1-LIFNR / LFA1-KUNNR) is empty.
In ECC releases, the field has not been visible just as in MDG.
With this correction, the MDG UI will now behave based on the underlying ERP version.

Solution

Apply the attached correction instructions or the related support package to the affected system(s).
