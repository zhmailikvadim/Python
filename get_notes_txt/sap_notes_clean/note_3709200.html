Environment

SAP Fiori Launchpad 2022

Cause

The issue is related to chip cache update. 

Resolution

To fix the issue we can clear the cache by running Report /UI2/INVALIDATE_GLOBAL_CACHES in SE38.

To prevent issue happen again we need apply the note 3295872.

3295872 - Tiles are not displayed due to issue in Chip Cache update
