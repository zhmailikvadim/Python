Symptom

When you enter the “Manage Environments” page, you find that the “Create” button is greyed out and disabled.

Environment

SAP BusinessObjects planning and consolidation 10.1 ,version for SAP NetWeaver(BPC 10.1 NW).

SAP Business Planning and Consolidation 11.0, version for SAP BW/4HANA.

Reproducing the Issue

  1. Login to BPC start page ->Enter the “Manage Environments” page.
  2. You find that the "Create" button is greyed out and disabled.

Cause

For BPC standard model, "Create" button is disbled by design.  The "Create" button is only for BPC embedded model.

Resolution

1, If you want to use Standard model, please use the "Copy" button to copy an environment from EnvironmentShell.

2,If you want to use Embedded model, please follow the steps in the note 1919631 to set the two parameters 'BPC Embedded Model Active' and "Deep HANA Integration Active".
