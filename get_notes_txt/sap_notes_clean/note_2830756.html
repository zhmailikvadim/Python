Symptom

The PTF check for SAP on IBM i indicates that a PTF group has an obsolete state even though the state is actually current. If the command CHKR3PTF is used, error message SAP0214 is displayed: "Old version of a Fixpack, product 5770SS1. Order PTF <PTF identifier>". For ABAP transactions DB4PTFCHK and DBACOCKPIT, the following is displayed for the status of the PTF group: "The PTF could not be found."

Other Terms

AS/400 OS/400 OS400 iSeries i5/OS Old version of a Fixpack SF99740 SF99704 SF99739 SF99730 SF99703 SF99729 SF99727 SF99720 SF99702 SF99719 SF99717

Reason and Prerequisites

This is a programming error. If, in a system, the installation status of the PTF groups is displayed for more than one level (service attribute PTFGRPLVL has a value above 1), the PTF check investigates the PTF level only for one level of the PTF group in question. If this is not the highest level, the PTF is displayed as "not found".

Solution

Check the PTF level with the operating system command WRKPTFGRP or implement the ILE patch levels assigned to this SAP Note.
