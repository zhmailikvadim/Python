Symptom

When attempting to create a bank account using the Manage Bank Accounts, only the Current Bank Account option is visible. Other contract types, such as Saving Bank Account, are missing from the selection.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Navigate to the Manage Bank Accounts.
  2. Attempt to create a house bank.
  3. Observe that only the current bank account option is available, and the other bank account types are not visible.

Cause

This behavior is due to the Enhancement for Bank Account Contract Type Validation. By default, the system restricts bank account creation to the Current Account contract type unless the contract type function is explicitly enabled.

Resolution

To enable the selection of other contract types, the following configuration steps must be performed:

  1. Navigate to the SSCUI Activate Features and activate the feature FTGL_BAM_BKACCT_CONTR_TYPE_VAL.
  2. Access the SSCUI Define Basic Settings (101025) and ensure the Enable Bank Account Contract Types option is selected.
  3. In the SSCUI Manage Field Status Groups (104059), assign appropriate field status groups to bank account contract types under Assign Field Status Grps to Bank Acct Master Data. This ensures that mandatory fields for specific contract types are correctly defined.
