Symptom

  • An error occurs when opening transaction CAT2 or CAT3 in GUI with a specified language settings. 
  • The error message displayed is LR047 'The unit "hour" could not be determined' 
  • The system suggests maintaining the missing entries in customizing. 
  • The issue does not occur when using other GUI languages such as German, English, or French. Issue particularly with one language (for example Italian: "Non è stato possibile determinare unità per l'ora")

Environment

  • Human Capital Management (HCM)
  • SAP R/3
  • SAP R/3 Enterprise 4.7
  • SAP ERP Core Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA

Reproducing the Issue

  1. Login to SAP in GUI with specified language (IT, DE, FR, EN & ES etc) settings. 
  2. Open transaction CAT2 or CAT3
  3. Error message LR047 appears.

Cause

Missing entries for MSEHI are not visible in the table T006A for the specified language. 

Resolution

  1. Verify the absence of entries in the table T006A for the specified language corresponding to MSEHI. 
  2. Maintain the same from Client 000 (Golden Client )
  3. Copy the maintained entries to the working client to resolve the error.
