Symptom

A transaction cannot be started if the referenced RFC destination contains in field "Target System" the message server port instead of a system ID.

Other Terms

SAP Business Client; SAP BC; NWBC; SM59

Reason and Prerequisites

The “target system” field of an RFC destination of type 3 can contain in field "Target System" either the system ID or a message server port, and until now only the system ID was evaluated in SAP Business Client correctly.

Solution

To use such an RFC destination with the SAP Business Client, you must install the SAP Business Client patch attached to this SAP Note as well as the NWBC ABAP Runtime Patch 94, which enables correct detection of the port and provides it to the client accordingly.

Please install the patch mentioned in section "Support Packages & Patches" or a newer one (SAP Business Client patches are always cumulative).

If the patch, which contains the correction from this SAP Note, is not delivered yet you can check the following SAP Notes for the expected delivery dates:
