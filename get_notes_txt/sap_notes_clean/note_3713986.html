Symptom

You want to implement Electronic Reporting for France via an Approved Platform (Invoice and Payment List). This SAP Note contains its prerequisites and a list of SAP Notes that you must implement.

Check regularly for new SAP Notes that might interest you.

For issues not covered in this SAP Note, create a customer incident on the SAP Support Portal under the CA-GTF-CSC-EDO-FR-RE component.

Other Terms

Reason and Prerequisites

Reason

You want to implement, configure, and use Electronic Reporting for France via an Approved Platform (Invoice and Payment List).

Legal requirements in France mandate that companies use electronic reporting as proof of invoice and payment transactions.

For more information, see Regulatory Change Manager.

Prerequisites for Electronic Reporting and Invoicing:

  • You have subscribed to SAP Document and Reporting Compliance, cloud edition. Contact your SAP Account Executive for details.

  • You have implemented the latest version of Electronic Communication Framework in your system. See SAP Note 3540443 for the implementation guide.

  • You have implemented the latest version of Electronic Document Processing Framework in your system. See SAP Note 2134248 for the implementation guide. 

  • Relevant for SAP ERP Only: You have implemented SAP Application Interface Framework in your system. See the "Integration with Application Interface Framework" section in the implementation guide for the Electronic Document Processing Framework (SAP Note 2134248).

  • You have implemented the Integration with SAP Document and Reporting Compliance, cloud edition by implementing the following SAP Notes:

    • 3385482 - Integration with SAP Document and Reporting Compliance, cloud edition
    • 3442326 - Integration with SAP Document and Reporting Compliance, cloud edition: Additional Features
    • 3549679 - Integration with SAP Document and Reporting Compliance, cloud edition: Improved Error Message Handling
    • 3668355 - Correction for integration with SAP Document and Reporting Compliance, cloud edition
    • 3616236 - Correction for Follow-Up Actions
    • 3642502 - Integration with SAP Document and Reporting Compliance, cloud edition: Enhancements to the Integration with the cloud edition

Prerequisites for Electronic Reporting only:

  • Relevant for releases as of SAP S/4HANA OP2021 and above: You have implemented the latest version of Statutory Reporting Framework in your system. See the following SAP Notes: 
    • Please refer to the ‘Getting started’ section in the Help portal to get information on setup and administration info.
    • Review and implement the following SAP notes as per your release

    • 3703762 - Manage Electronic Documents: Enhancement for Tax Reporting Date and Process Category

  • You have implemented the latest version of Electronic Document Processing for France via Approved Platform in your system. See SAP Note 3413565 for the implementation guide.

Solution

After implementing the SAP Notes listed in the Prerequisites section, implement the following SAP Notes specific to Electronic Reporting for France, which are required for generating reporting XML files:

   Follow these steps to implement SAP Notes for Electronic Reporting for France using the Note Analyzer:

  1. Download the relevant XML file (FR_RE_S4CORE_1.xml or FR_RE_SAP_APPL_1.xml) from the attachment section.
  2. Launch transaction SNOTE.
  3. In the menu bar, select Goto -> Other Tools -> Launch Note Analyzer.
  4. Upload the XML file to your system and choose Analyze (F8).
  5. Navigate to SAP Note Check Phase.
  6. Choose the first scope.
  7. In the table, mark the column Comment.
  8. In the menu, choose Sort in Ascending Order.
    • As a result, a list of relevant SAP Notes will be displayed in the correct sequence to be implemented.
  9. Implement the SAP Notes that are in status Can be implemented.

Note on Area Menu and IMG Updates:
Official SAP documentation cites menu paths on the SAP Easy Access screen and technical names of maintenance objects assigned to the SAP Implementation Guide (IMG; transaction SPRO).
Any updates to the menu on SAP Easy Access screen and to the IMG require a given support package to be installed.
If you have not installed the corresponding support package, the updated menu path is not available in your system.
To access the required objects, call the transaction code assigned to area menus or enter the technical name of the maintenance object in the maintenance transaction, such as SM30 for maintenance views. The object names that you need to use are covered in the documentation.

Note about translations: Translations are only delivered with support packages.
