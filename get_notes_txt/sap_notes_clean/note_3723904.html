Symptom

When using DRFOUT and  replication object PS_WBS_OUT  not all WBS are been replicated. 

Environment

  • SAP ERP Project System (PS)
  • SAP R/3
  • SAP R/3 Enterprise
  • SAP ERP Central Component
  • SAP ERP
  • SAP Enhancement package for SAP ERP
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA, on-premise

Reproducing the Issue

Run DRFOUT 
Enter Replication Model eg PS_WBS_OUT 
Execute 
In  Targe System not all WBS have been created 

Cause

The WBS not replicated are missing "Account Assignment" flag.

Resolution

The Field  Account assignment  (PRPS-BELKZ) must be set on the WBS , then they will be replicated. 

Keywords

CFIN , CLASS: CL_PS_DRF_WBS_OUTB  METHOD: READ_COMPLETE_DATA, DRF_SERVICES_OUTBOUND, Data Replication Framework ,CL_PS_DRF_WBS_OUTB, READ_COMPLETE_DATA, EPPM_WBS_INTEGRATION .
