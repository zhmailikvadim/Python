Symptom

 In the process "Start of Supply for Electricity (DSO View)" (DE_B_GPKE_DSO_SOS_REG), step 2700 for updating the BP relationship was missing from the system.

This issue is relevant for the market role: DSO.

Other Terms

GPKE, DE_B_GPKE_DSO_SOS_REG, BP Relationship Update, Step 2700, deregistration message, 2508

Reason and Prerequisites

Program Error

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the system includes step 2700 in the process by activating the BC set /US4G/3_0_DSO_SP07_NOTE_3717357, ensuring that the BP relationship is updated correctly. You need to retest the process. You can verify the fix by triggering the Start of Supply for Electricity process (DSO View) with a deregistration message and checking that the BP relationship is cleaned up or updated as expected.
