Symptom

We can see the RAR contracts as soft deleted in the mapping table but it is not visible in 'Monitor Revenue contracts' Fiori app.

Environment

Revenue Accounting and Reporting

Reproducing the Issue

1. Execute " Monitor Revenue contracts" Fiori app .

2. Search for the soft deleted contract and you get message " No data found".

Resolution

  • The system is working as designed. The Fiori app “Monitor Revenue Contracts” does not display soft deleted contracts.
  • To view soft deleted contracts, use the Fiori app “Manage Revenue Contract.”
  • This app includes an additional selection parameter:
    • Go to Adapt Filters
    • Enable the filter “Include Soft Deleted”
    • Set the value to Yes or No as required.
    • You should now be able to view soft deleted contracts.
