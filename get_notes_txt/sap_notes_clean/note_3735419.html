Symptom



  • After running the payment proposal, the due date is displayed for the item in the EPIC screen.
  • The due date determination logic requires clarification.


Environment


  • China

Reproducing the Issue


  1. Run the payment proposal for vendor invoices in EPIC.
  2. Check the due date of the proposal items.

Cause



Resolution


  1. Review the setting “Payment per Due Day” in FBZP (Payment methods in company codes).
  2. If not checked, EPIC uses the Posting Date entered in the payment proposal run as the Due Date.
  3. If checked, EPIC uses the original invoice due date as the Due Date.
