Symptom

During Unified Package Building (UPB), performance degradation can occur due to excessive write operations to the application log. This behavior appears independently of the packaging engine used — Package Building (PB), Packaging Specification (PS), or Packing Instruction (PI).
Each time a message is raised by the packaging logic, the system triggers an immediate application log write. Since the application log framework is not optimized for mass‑processing scenarios, this results in a large number of individual write actions and can significantly increase runtime.

Other Terms

Unified Package Building

Solution

Implement the attached Correction Instruction or the mentioned Support Package.
