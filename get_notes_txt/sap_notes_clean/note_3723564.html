Symptom

While creating a correction item via the 'Manage In-House Bank Account Balances' app, you find that the posting date of the created item does not correspond with the current bank area posting date.

Other Terms

/PF1/IHB_SET_TRANS_D, F6341

Reason and Prerequisites

This is due to a coding error. Existing code passes the current system date as posting date.

Solution

Please apply this SAP Note or the assigned Support Package.
