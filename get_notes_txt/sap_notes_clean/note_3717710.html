Symptom

  • The logic is not clear how value is copied from main item to subitem.
  • How to control which field is copied. 

Environment

  • Materials Management (MM)
  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA

Reproducing the Issue

Create PO with subitems in ME21N and check the target value. 

Cause

System always call FM ME_COPY_FIELD_TO_SUB_ITEM to copy value. In this FM, system searches entry with the relevant table, field and key in table TMSI3. If the entry exists, the value will always be copied. The key is determined in TMSI1 based on subitem category.

Resolution

Configure the entry in table TMSI1 and TMSI3 to control the copy rule for subitem. 

Keywords

subitem, component
