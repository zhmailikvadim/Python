Symptom

In the "Manage Automatic Payments" app, the payment has been generated, but the payment medium has not.
The following error appears in the log: "Payment method & in company code && does not use the PMW".

Environment

SAP S/4HANA Cloud Public Edition

Cause

The default payment medium format (PMW) was not maintained in the configuration "Set Up Payment Methods for Each Country/Region for Payment Transactions" (ID: 101972) for country/region && and payment method &.

Resolution

In configuration "Set Up Payment Methods for Each Country/Region for Payment Transactions" (ID: 101972), maintain the default payment medium format for the mentioned country/region and payment method.

Keywords

Automatic Payments, Payment Medium, PMW, Payment Format, Default Payment Medium Format, Bank Determination, Payment Batch, Payment File, Set Up Payment Methods for Each Country/Region for Payment Transactions
