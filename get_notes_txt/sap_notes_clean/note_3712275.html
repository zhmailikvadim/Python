Symptom

You execute transaction TOENE ('Hedge Management Cockpit for Hedging Area') and start to create a Hedge Request.

The program immediately terminates with a short dump.

Other Terms

CONVT_NO_NUMBER
CX_SY_CONVERSION_NO_NUMBER
CL_TOE_NOE_CTRL===============CP
Short dump
TOENE

Reason and Prerequisites

There is a program error.

The issue only occurs when a field in Columns 'Company Code' or "Risk Free Currency' is selected while creating the Hedge Request.

Solution

Implement the correction instruction or import the relevant support package.

With this SAP Note, Error Message FTOE_CORE 222 ('Requirements for navigation to a hedge request not fulfilled') is produced instead.
