Symptom

How to enable parameter for Support for Line-Item Numbers in PRs?

Resolution

This is a self service parameter available for the admin user in guided buying PARAM_ENABLE_LINEITEM_NUMBERING.

Displays line item numbers next to line items on purchase requisitions and purchase orders. If set to false (default), line item numbers do not display next to line items If set to true, line item numbers display next to line items.

Line item numbers aren't supported in sites with the 2NV and 3EN integration scenarios.
