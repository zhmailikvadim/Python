Symptom

You call BAPI_ALM_ORDER_MAINTAIN to create or change an order. When you do this, the technical object transfers an invalid maintenance planner group to the order header.

Other Terms

Equipment; functional location; INGPR; INGRP; PLGRP; CAUFVD; IBAPI; IBAPI_H_CREATE_PROPOSAL_INTERN; V_T024I; ITOB; IW151; IW 151

Reason and Prerequisites

The error occurs if the maintenance planner group maintained for the technical object is not valid for the order.

Solution

Implement this SAP Note.
