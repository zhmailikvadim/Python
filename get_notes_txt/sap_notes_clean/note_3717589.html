Symptom

Quick Actions display header text in dark color, while others display it in white color when using the Latest Home Page Experience.

Environment

SAP SuccessFactors HCM 

Reproducing the Issue

  1. Login to the system with provided access.
  2. Navigate to the home page.
  3. Click on one of the Quick Action.
  4. Observe that the header text is white.
  5. Click on another Quick Action.
  6. Observe that the header text is dark.

Cause

Known issue. 

Resolution

  • This fix for this issue is planned to be deployed on 1H 2026 release.
  • For release timelines, please review SAP SuccessFactors Product Release & Road Map Information.
