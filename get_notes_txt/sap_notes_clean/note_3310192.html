Symptom

Variable values for compounded characteristics either appear with a doubled slash delimiter, i.e. '//', or, if selected from the (F4) input help, become rejected. The issue usually affects node variables for hierarchies along compounded characteristics.

Other Terms

plausibiltiy check, internal external value

Reason and Prerequisites

According variable values by mistake might be reformatted for external display, although they already are in the external format.

The issue is caused by a program error.

Solution

  • SAP Business Warehouse (SAP BW) 7.30
    Implement Support Package 17 for SAP BW 7.30 (SAPKW73017) into your BW system. The Support Package will be available as soon as SAP Note 2342209 with the short text "SAPBWNews 7.30 BW ABAP SP17", which describes this Support Package in more detail, is released for customers.
     
  • SAP BW 7.31 (SAP NW BW 7.3 EHP 1)
    Implement Support Package 20 for SAP BW 7.31 (SAPKW73120) into your BW system. The Support Package will be available as soon as SAP Note 2367669 with the short text "SAPBWNews 7.31 BW ABAP SP20", which describes this Support Package in more detail, is released for customers.
     
  • SAP BW 7.40
    Implement Support Package 17 for SAP BW 7.40 (SAPKW74017) into your BW system. The Support Package will be available as soon as SAP Note 2365926 with the short text "SAPBWNews 7.40 BW ABAP SP17", which describes this Support Package in more detail, is released for customers.
     
  • SAP BW 7.50
    Implement Support Package 6 for SAP BW 7.50 (SAPK-75006INSAPBW) into your BW system. The Support Package will be available as soon as SAP Note 2346340 with the short text "SAPBWNews 7.50 BW ABAP SP6", which describes this Support Package in more detail, is released for customers

  • SAP BW 7.51
    Implement Support Package 1 for SAP BW 7.51 (SAPK-75101INSAPBW) into your BW system. The Support Package will be available as soon as SAP Note 2345927 with the short text "SAPBWNews 7.51 BW ABAP SP1", which describes this Support Package in more detail, is released for customers.
