Symptom

The graphic view of an ALV Grid display is opened and changes made to the initial grahical settings e.g. chart type, color definitions, element descriptions. The display is then saved as a layout variant. When the saved layout variant is re-loaded, the graphic view is displayed as expected, but the graphical settings revert to their initial state.

This issue arises only when the ALV Grid Bridge architecture is active.

Other Terms

ALV; ABAP List Viewer; SAP List Viewer; Grid;

Reason and Prerequisites

Reason

The graphic configuration of the layout is loaded incompletely, causing the initial settings to be used.

Solution

From now on, the scenario described in the "Symptom" section no longer occurs.
