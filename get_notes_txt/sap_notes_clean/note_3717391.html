Symptom

In the process "Device Installation and Finalization of Technical Master Data" (APEU_B_MCM_TMD_PROCESS), step 0340 (Post Acknowledgement to Cloud) fails when the device transaction date is in the future.

Other Terms

US4G, device installation, Post Acknowledgement to Cloud, device transaction date, MCM, technical master data

Reason and Prerequisites

Post acknowledgement to cloud step is getting failed if device transaction date is in future.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, step 0340 completes without an error, and the process APEU_B_MCM_TMD_PROCESS continues and completes successfully.

For existing error cases, you need to reverse step 0340 and rerun the process to reprocess the step. You can verify the fix by checking that the process completes successfully.
