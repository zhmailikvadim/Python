Symptom

You use the Fiori App F3616 Currency Exchange Rates to monitor exchange rates

You display Detail of one Currency Exchange Rate.

You realize that in the Change Log no data are displayed and instead there is an error 'Runtime Error: 'RAISE_SHORTDUMP'. The OData request processing has been abnormally terminated...'.

In the transaction ST22 you can also find dump CX_SADL_DUMP_DATABASE_FAILURE with SQL code 339 and in the System Log (Transaction SM21) you can see error '...attribute value is not a table...' during PRE access to table C_CRCYEXCHRATECHANGELOGRECORD.

Other Terms

F3616, CX_SADL_DUMP_DATABASE_FAILURE, 339, C_CRCYEXCHRATECHANGELOGRECORD, LOGKEY, attribute value is not a table

Reason and Prerequisites

Program error.

Solution

Implement automatic correction instruction from this SAP Note or implement the corresponding Support Package.
