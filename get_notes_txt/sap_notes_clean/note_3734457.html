Symptom

  • After a system upgrade, ABAP dumps are recorded in transaction ST22 indicating an exception in class LCX_RAP_EVENT_RUNTIME.
  • The errors are related to missing bgRFC inbound destination 'BGPF'.

Environment

SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Perform a system upgrade to a new release.
  2. Run RAP-based processes or applications.
  3. Open transaction ST22 and observe ABAP dumps referencing LCX_RAP_EVENT_RUNTIME.

Cause

The customizing for bgRFC inbound destination 'BGPF' is missing.

Resolution

  1. Review the guidance in SAP Note 3457926 to understand the required bgRFC settings.
  2. Maintain the bgRFC inbound destination 'BGPF' manually as described in the Note.
  3. Save the configuration and ensure the destination is active.
  4. Re-test the previously failing RAP process.
  5. Confirm in ST22 that no new dumps of type LCX_RAP_EVENT_RUNTIME occur.
