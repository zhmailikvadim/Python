Symptom

Dump occurs when searching authorization object without fields with in PFCG:

Category               ABAP programming error                                                       
Runtime Errors         MESSAGE_TYPE_X                                                               
ABAP: Program          SAPLOLEA                                                                     
Application Component  BC-FES-CTL            

|Short Text                                                                                        |
|    The current application has intentionally triggered a termination with                        |
|    a short dump.                                                                                 |


|What happened?                                                                                    |
|    The current application program has detected a situation that should                          |
|    not occur. A termination with short dump has therefore been triggered                         |
|    by the key word MESSAGE (type X).                                                             |

Environment

  • SAP NetWeaver
  • SAP NetWeaver Application Server for SAP S/4HANA
  • ABAP PLATFORM - Application Server ABAP

Reproducing the Issue

  1. Access PFCG
  2. Navigate to Authorization tab
  3. Select "Change Authorization Data"
  4. Search for an authorization object without fields within the role editing interface
  5. Observe that a dump occurs during the search process

Resolution

Implement Note 3684924 - PFCG: Suche in Berechtigungsdaten führt zu Laufzeitfehler

Keywords

PFCG, authorization object, MESSAGE_TYPE_X, SAPLOLEA, BC-FES-CTL, runtime error, role editing, authorization data, ABAP programming error, short dump
