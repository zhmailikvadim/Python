Symptom

In Business Data Request process, segment SG8 RFF+MG under segment SG8 SEQ+Z05 is missing when sending message ‘UTILMD for master data (CD013) - GPKE / GeLi Gas’ if basic device category is communicator(Z05).

Other Terms

8021, SG8 RFF+MG, SG8 SEQ+Z05, CD013, 11035, /IDEXGE/CL_DP_UTILMD_009, COMMUNICATION_DEVICE_DATA_SEQ

Reason and Prerequisites

This is a program error.

Solution

Please implement the correction instruction in the SAP Note or install further Support Package.
