Symptom

  • Validation error encountered in the "Manage Integration Workflows" app while customizing the "Production Order Goods Receipt" workflow (Outbound).
  • Attempting to update the "Identifier" field to map it to the SFC information instead of the material results in the system blocking the action with the validation error: "Please shorten this text to 40 characters or less."
  • Inconsistency observed where the standard value delivered for the "Identifier" field is "/erpGoodsReceiptRequest/materialDocumentItem/material," which is 53 characters long, conflicting with the validation rule of 40 characters.

Environment

  • SAP Digital Manufacturing

Reproducing the Issue

  1. Access the "Manage Integration Workflows" app.
  2. Navigate to the "Production Order Goods Receipt" workflow (Outbound).
  3. Attempt to update the "Identifier" field to point to the relevant SFC path in the payload.
  4. Observe the validation error: "Please shorten this text to 40 characters or less."

Cause

Resolution

The Identifier in the "Manage Integration Workflows" app does not control the identifier in the Integration Message Dashboard. Instead, it refers to the Application Message ID in SAP Cloud Integration, which means there is no direct relationship with the Identifier in this scenario.  

Regarding tracing messages using the SFC rather than the material, this functionality is not supported under the current product design.
