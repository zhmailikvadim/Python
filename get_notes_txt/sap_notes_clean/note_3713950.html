Symptom

  • CMC Freezing During Publication scheduling/rescheduling.
  • Issue is occurring only with Load balancer URL.
  • Bypassing the Load Balancer URL allows the workflow to function normally.

Environment

  • SAP BusinessObjects Business Intelligence Platform 4.x
  • All supported OS.
  • Load Balancer.

Reproducing the Issue

1. Attempt to schedule/reschedule a publication in the CMC.
2. Observe that the CMC freezes and no further actions can be performed.

Cause

  • The issue is likely due to misconfigured or missing Sticky Session (Session Affinity) settings.
  • When Sticky Sessions are not properly set up, users may experience unexpected logouts or disruptions in web applications like CMC or BILP.

Resolution

  • In this Customer case, they were using Azure Loadbalancer.
  • The issues was fixed after changing the Stickiness Type to "Load Balancer Generated Cookie" in the Load Balancer configuration.
  • This change will ensure proper session persistence, allowing the publication workflow to function as expected.
