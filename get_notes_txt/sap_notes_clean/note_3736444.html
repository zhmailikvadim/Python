Symptom

Materials are set to price control V (moving average price), the request is in a collective production order scenario with automatic goods movements, the child order’s cost is rolled up to the parent order. 

Environment

SAP S/4HANA Cloud Public Edition

Cause

Currently the function is not supported in SAP S/4HANA Cloud Public Edition. 

Resolution

Submit a feature request according to 2963059 - How to Submit Feature Requests for SAP S/4HANA Cloud Public Edition - SAP for Me .
