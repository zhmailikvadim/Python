Symptom

How to add pricing elements in the pricing header section and product table in Sales Quote and Sales Order?

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

  • SAP Sales Cloud Version 2 
  • SAP Sales and Service Cloud Version 2

Reproducing the Issue

  1. Go to Navigation menu -> Sales Quotes/Sales Order 
  2. Open one Sales Quote/Order 
  3. Start adaptation 
  4. Click pencil button for the header section
  5. Click Add from list 
  6. The fields related to Pricing Elements are not available 
  7. Click Pencil button for Product Table 
  8. Select "Show Hidden Elements"
  9. The fields with general name "Pricing Element 1", "Pricing Element 2", "Pricing Element 3".......

Resolution

  1. Go to Settings -> Pricing Configuration -> Pricing Element Mapping 
  2. Select the internal/External drop down list 
  3. Click + button 
  4. Select the target pricing procedure  
  5. Select the Condition type that you would like to show in the header and product table.
  6. Please note that it supports up to 9 item level manual conditions and 5 header manual conditions.
  7. Go to the target sales order/sales quote detailed view 
  8. Make sure the same pricing procedure was determined as the same one in step 4 
  9. Go to Header level/Product Table 
  10. These manual conditions are available.
  11. Start Adaptation 
  12. The adaptation only supports the field sequence change. Please note that hide/delete won't take effect. If the corresponding price elements need to be hidden, please go back to step 5 and delete the target price elements. If the visibility needs to be controlled based on the business role, please refer to the help documentation -> Internal and External Pricing Configurations-> Configure Authorizations for Pricing.
