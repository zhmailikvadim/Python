Symptom

An error occurs in the app "Physical Inventory Analysis" with the message: "Chart cannot be rendered: Multiple units for measure REF_QV_CELL_2".

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Access the app "Physical Inventory Analysis".
  2. Attempt to use the "Graphical Display" functionality.
  3. Observe the error message: "Chart cannot be rendered: Multiple units for measure REF_QV_CELL_2".

Cause

This is the standard behavior of Web Dynpro Reporting.

As the error message informed, the chart can only display measures with the same unit (currency keys or unit of measures), because the chart will be meaningless if the measures are based on different units/currencies in one graphic.

Resolution

The selection must be made in such a way that only the same units appear.
