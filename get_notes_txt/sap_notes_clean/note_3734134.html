Symptom

Incorrect tracing factor shown in KEU5; tracing factors appear doubled in special periods when using cumulative/aggregate tracing factors; messages GA190 and KF444 observed.

Environment

•Controlling (CO)
•SAP R/3
•SAP R/3 Enterprise 
•SAP ERP Central Component
•SAP ERP
•SAP enhancement package for SAP ERP
•SAP enhancement package for SAP ERP, version for SAP HANA
•SAP S/4HANA Finance
•SAP S/4HANA

Reproducing the Issue

  1. In KEU5 (or KEU3), run Check cycle.
  2. Go to Receiver weighted factor and run the Select product check.
  3. Run the allocation for a special period using cumulative/aggregate tracing factors.
  4. Observe that receiver tracing factors appear doubled and review the error log showing GA190 and KF444.

Cause

Resolution

  1. Apply SAP Note addressing special-period tracing factors: implement SAP Note 539125 - Tracing factors in special periods.
  2. Test in a development or test environment: run the relevant KEU5/KEU3 cycle in a special period with cumulative/aggregate tracing factors and confirm that tracing factors are no longer doubled or incorrect.
