Symptom

  • Changes in (text) associations are not detected by recreating or redeploying the Analytic Model

Environment

  • SAP Datasphere

Reproducing the Issue

  • Associations are added to Dimension 
  • Dimension is shared from "Space A" with another space "Space B"
  • An Analytic Model is created on "Space B".
  • Analytic Model is not able to see the associations.

Cause

  • The target tables of the associations are not shared with the other space where the Analytic Model is created on.

Resolution

  1. Share the target table(s) of association with the Analytical Model space.
  2. Re-deploy the Analytical Model after sharing the associations.

Keywords

association, analytic model, text association, deployment, dimension, association sharing, SAP Datasphere, missing text association, text
