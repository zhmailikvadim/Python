Symptom

Message no. COOS126: “Cannot split order &1. Error records relating to cost distribution exist.” is triggered when attempting to split order. 

Environment

SAP S/4HANA Cloud Public Edition

Cause

Check there is error for the order or child order in the app "Manage Product Costing Errors". 

Resolution

Simulate the orders in the app "Manage Product Costing Errors". 

If the simulation is successful, trigger the activity cost redistribution by using the template 'Redistribute Activity Costs in Order Split: Event-Based Production Order' in the app "Schedule Product Costing Jobs". 

Keywords

COOS126, order split, error records relating to cost distribution, activity cost distribution, event-based postings, Manage Event-Based Posting Errors, F5132, Redistribute Activity Costs in Order Split, Schedule Product Costing Jobs, production order split blocked, child order costing errors
