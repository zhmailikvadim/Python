Symptom

When adding activities to a program agenda error pops up saying "You have exceeded the number of activities that a program can contain 200"

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

SAP SuccessFactors Learning

Reproducing the Issue

  1. Go to Learning Administration
  2. Go to Programs
  3. Open an existing program in Learning administration and navigate to the Agenda.
  4. Ensure the agenda already contains 200 activities in total across all sections.
  5. Create a new section in the agenda.
  6. Attempt to add a new activity (for example, a link) to any section.
  7. Observe the error: "You have exceeded the number of activities that a program can contain 200".

Cause

This is system-expected behavior: A )rogram agenda supports a maximum of 200 activities in total across all sections. Attempts to add more activities trigger the error.

Resolution

  1. Verify the total number of activities in the program agenda across all sections; if it is 200, the limit has been reached.
  2. Remove one or more existing activities so the total falls below 200.
  3. Add the new activity after reducing the total count.
  4. Alternatively, distribute content across multiple programs to stay within the 200-activity limit per program agenda.
  5. If a higher limit is required, Submit an enhancement request via the Customer Community Portal. Provide business reasons and gather support through Kudos for the request. The Engineering team will evaluate the feasibility of implementing this feature in the future. 2090228 - How to submit enhancement ideas for SAP SuccessFactors Products - SAP for Me
