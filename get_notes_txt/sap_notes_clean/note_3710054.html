Symptom

You have applied the SP that contains note 2048531, which makes it possible to enable changing time-dependent bank accounts that are linked to a SEPA mandate. You want to maintain the value in table BPCONSTANTS or have done so already. In the value help for the key there is no text next to the key "CVISTD".

Other Terms

SEPA mandate, business partner, CVI, customer vendor integration, time-dependency

Reason and Prerequisites

 Text was not delivered in the original language German, therefore translations could not be delivered, either.

Solution

 Follow the manual steps of this note or apply the SP.
