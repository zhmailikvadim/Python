Symptom

  • During the migration of the data object "Warehouse Storage Bin", the following issue occurs: "Key Field is initial and the record is ignored".
  • Migration object items are skipped and not migrated.

Environment

SAP S/4HANA Public Cloud

Reproducing the Issue

  1. Attempt to migrate the data object "Warehouse Storage Bin" using the migration cockpit.
  2. Observe that the error message "Key Field is initial and the record is ignored" appears.
  3. Verify that all key fields are filled.
  4. Notice that the migration object items are skipped and not migrated.

Cause

Resolution

  1. Ensure the "Storage Type" field is maintained in the provided file.
  2. Re-upload the file with the corrected "Storage Type" field.
  3. If the issue persists, verify the system link and project name to confirm the correct system is being accessed.

See Also

Refer to: SAP Help Documentation on Migration Cockpit
