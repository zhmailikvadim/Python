This SAP Note provides you with changes in HANA calculation views and ABAP code to address the following issues:

ECD

  • The values for functional currency are not considered in I155 and I355 filters causing missing lines in the SPED output.
EFD ICMS IPI
  • K100 register is reported even when you do not have any information to be reported in block K.  

EFD Contribuições

  • PCO_0120 register cannot be reported for January - November.

Other Terms

TDF, Brazil, SPED, ECD, EFD ICMS IPI, EFD Contribuições.

Reason and Prerequisites

This SAP Note is relevant only for Brazil.

You have implemented SAP Note 3198840.

You have upgraded your system to SAP TDF Support Package 16.

Solution

After implementing this SAP Note, the system will be updated as follows:

ECD

  • Values for CA_VL_SLD_INI_AUX, CA_VL_DEB_AUX_FINAL, CA_VL_CRED_AUX_FINAL fields in EFD_I155 Calculation View and VL_CTA_AUX field in EFD_I355 calculation view will be considered in corresponding filters.

EFD ICMS IPI

  • The K100 register will be generated only if any other register in block K was generated.

EFD Contribuições

  • The PCO_0120 register will allow reporting different months than December.
