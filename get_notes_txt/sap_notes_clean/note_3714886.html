Symptom

  • When executing the OData service in /IWFND/GW_CLIENT, the system returns the error: “HTTP receive failed: Client connection to …”
  • HTTP Receive failed: Client connection to XXXXXXX via proxy YYYYYYY broken. Request with ZZZ bytes sent 0 seconds ago. 

Environment

  • SAP NetWeaver 7.5 and higher 
  • SAP S/4HANA 1709 and higher 

Reproducing the Issue

  1. Login to the SAP GUI and execute /n/iwfnd/gw_client
  2. Pick any ODATA Service and click on Execute

Cause

Proxy name is mentioned in the client settings and this proxy is not reachable 

Resolution

  1. Login to the SAP GUI and execute t-code SICF 
  2. Click on Execute
  3. Click on Client-->Proxy Settings
  4. Click on the HTTP Protocol, remove the Host Name, Port and click on Continue
