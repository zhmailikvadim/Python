Symptom

In the process "DE_B_MDC_MET_RECV_BDR_DSO" (MET cluster), in step 0300, the system did not send the preliminary load shape value even when it was valid because the final load shape value was not available. Business message 17102 is involved.

This issue is relevant for the market roles: DSO.

Other Terms

 MET, GAS, DE_B_MDC_MET_RECV_BDR_DSO, BDR, Business message 17102, /US4G/CL_DEF_BADI_MDC_PROCESS, RETRIEVE_PROFILE_VALUES, step 0300, load shape, preliminary load shape value, final load shape value, DSO

Reason and Prerequisites

 The error occurred because the system required a final load shape value and prevented sending a valid preliminary value when the final value was not available.

Solution

 Implement the corrections of this SAP Note or import the relevant support package.
