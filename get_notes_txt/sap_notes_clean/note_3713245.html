Symptom

In Timesheet Analytics - My Compliance:

  • The Concurrent Employment functionality was not available.
  • Header KPI was not reflected and few other fixes.
  • Compliance Percentage calculation was not enabled.
  • No Default Filter Persist on load.

Other Terms

F8342, Timesheet Analytics, My Compliance, CE, Concurrent Employment,  Compliance Percentage, OCP

Reason and Prerequisites

  •  Concurrent Employment functionality is now introduced.
  •  Header KPI was not fully functional.
  •  Default filter was not available.

    Prerequisite    : The SAP Note: 3609516 has to be implemented in the corresponding back-end system.

Solution

 Apply corresponding support pack. Alternatively, implement the correction instructions attached to the SAP Note.
