Symptom

 Copying an ABAP Report Launchpad fails with dump - error message Function module "LSAPI_GET_FLP_DATA" not found.

Category

 ABAP Programming Error
Runtime Errors CALL_FUNCTION_NOT_FOUND
ABAP Program SAPLAPB_LAUNCHPAD
Application Component BC-MUS-LP

Other Terms

LPD_CUST, Menu 'Launchpad -> Read from other system via RFC'

Reason and Prerequisites

The function called does not exist in the target system. This may occur if the support package version of NWBC differ between the systems.

Side effect of SAP note 3672931 - SAP NWBC ABAP Runtime Patch 92

Solution

Implement the attached correction instructions or import the specified Support Package.
