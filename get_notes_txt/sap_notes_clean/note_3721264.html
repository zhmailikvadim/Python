Symptom


The HU API is unable to create a Handling Unit (HU) when the same material is provided with two or more different batches within a single HU.
During HU creation, if multiple line items contain the same material number but different batch numbers, the API returns an error or rejects the HU creation request.

Other Terms

Handling Unit Management, MAP_ITEM, CL_LO_HU_HANDLING_UNIT_MANAGE.

Reason and Prerequisites
