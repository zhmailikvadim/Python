Symptom

This SAP Note makes the following corrections for the CDP BDN when filling the Anonymous Birth field:

  • Following requests from customers and feedback from the registry office and the Federal Ministry of the Interior, the Anonymous Birth field with the following values is provided for this interface:
    • A - Anonymous Birth
    • B - Baby flap
    • F - Findelkind
    • T - for stillbirth
    • E - for N/A
  • The default sets the value E if it is not an anonymous birth. Only if it is an anonymous birth can other forms, such as e.g. T for stillbirth if the mother explicitly wants this. Only in this case would the Anonymous Birth field be supplied with the value T in the case of a stillbirth. In the case of a stillbirth for which the mother is known, this field should not be filled.
  • Special queries for setting this field can be overridden at any time in BAdI /ISHCDP/AT_BDN_DATA in method FILL_SDG04V10.
