Symptom

  • Goal description field is not appearing in performance form pdf view.

Environment

  • SAP SuccessFactors Performance Management
  • SAP SuccessFactors Goal Management

Reproducing the Issue

  1. Generate a PDF of the form containing the "Goal Details" tab.
  2. Verify that the "Goal Details" tab does not have description field in print/pdf form view.

Cause

The "Description" field in the goal plan is a custom field, which is not supported in the legacy form view.

Resolution

Enable the latest form view to display the "Description" field in the Performance Form Print/PDF view.

Keywords

goal details tab, form pdf, sap successfactors, performance & goals, description field, custom field, legacy form view, xml configuration, pdf output, enable latest form view
