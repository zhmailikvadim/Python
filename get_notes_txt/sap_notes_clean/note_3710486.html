Symptom

  • An error occurs when editing the PM form as "You do not have permission to access the form."
  • The error occurs when changing the rating for competency/goal during calibration sessions.
  • The error also occurs when trying to change the Overall form rating.
  • Despite the error message, the form can still be saved with changes.
  • Issue is not reproduced in legacy performance form.

Environment

  • SAP SuccessFactors Performance & Goals
  • SAP SuccessFactors Calibration

Reproducing the Issue

  1. Proxy as user who is a participant of calibration session (not involved in PM route map)
  2. Go to calibration session > open the session > List View > Edit the PM form 
  3. Try to edit the competency rating (Item rating permission has been granted for all roles)
  4. Observe the error message "You do not have permission to access the form."
  5. Save the form, which will save successfully despite the error message.

Cause

The issue is caused by the business rule configuration in the form template for the "Field change" action.

Resolution

The fix for the issue (error when editing competency rating field) is planned to be fixed through patch which will be fixed at 2026-02-10 for preview  and production instance at 2026-02-12. 

The fix for the issue (error when editing overall form rating) will be fixed through different patch and can be tested for preview instance after 2026-02-17, and production instance after 2026-02-19. 

For patch timelines, please review SAP SuccessFactors Patches Knowledge Base

Workaround:

Remove the business rule assigned for the "Field change" action from the form template.
