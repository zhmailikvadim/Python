Symptom

  • Download Manager displays a blank screen.
  • Download Basket does not display files.
  • Download Manager shows a blank screen after login. 

Environment

  • Download Manager
  • Technical Communication S-user

Reproducing the Issue

  1. Navigate to Download Manager.
  2. Log in using a Technical S-user.
  3. Notice that Download Manager shows a blank screen.

Cause

Download Manager does not support the use of Technical Communication S-users.

Resolution

Use a regular S-user account to log in to Download Manager instead of a Technical Communication S-user account. 
 
Note: Only regular S-user accounts are supported by Download Manager. For detailed information on the differences between regular S-user and Technical Communication S-user accounts, refer to KBA 2668288.

If the problems with Download Manager persist even after successfully logging into Download Manager using a Regular S-user, create a case under component XX-SER-SWFL-FCMS for further investigation and include the following information:

  • Attach a screenshot of the issue.
  • Attach the Download Manager trace files as per the steps explained in KBA 2662384.
  • Add the S-user ID and password to the Customer Remote Logon Depot as described in KBA 1773689 so SAP can test the issue.
