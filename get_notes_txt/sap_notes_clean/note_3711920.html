Symptom

The system reclassifies the GST rules as Fatal in the May 2026 release by the Infocomm Media Development Authority (IMDA).

This affects invoice and credit note document types for GST category codes SR, SRCA-S, SRCA-C, ZR, SRRC, SROVR-RS, SROVR-LVG, or NA.

Other Terms

eDocument, eDoc, eInvoicing, e-Invoicing, electronic invoice, electronic invoicing, Singapore eInvoice, Convergent Invoicing, FICA/CI.

Solution

Import the Support Package specified for your release or implement the attached correction instruction.

As a general rule, SAP recommends that you implement the Support Package specified for your release. However, if you need to install a solution earlier, use the SAP Note Assistant to implement the correction instruction. For more information, see SAP Note Assistant.
