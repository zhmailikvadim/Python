Symptom

Error "The Create button is disabled in the Product Specific Configuration phase. Please use the Edit button in the Scope section to make necessary changes and access the app" Occurs in Configure Standard Order Type App in Central Business Configuration.

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Environment

  • SAP Central Business Configuration
  • SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Open the Configure Standard Order Type activity in CBC workspace.
  2. While the CBC workspace is in the Product-Specific Configuration phase, attempt to click create button.
  3. Observe that the Create button is disabled. Error "The Create button is disabled in the Product Specific Configuration phase. Please use the Edit button in the Scope section to make necessary changes and access the app" Occurs.

Cause

You cannot create sales document types during the Product Specific Configuration phase.

Resolution

  1. Open the Define Scope App or Set Up Organizational Structure App.
  2. Select Edit to switch the phase from Product-Specific Configuration back to the Scope and Organizational Structure phase.
  3. Click complete activity button in Define Scope App or Set Up Organizational Structure App.
  4. Now you can edit Configure Standard Order Type App and set status for it. SAP Help Portal - Create a Sales Document Type
