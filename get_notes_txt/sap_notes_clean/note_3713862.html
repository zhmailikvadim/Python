Symptom

After the SP24 upgrade, when the program ARBCIG_IF_SETUP_P2P is executed, it fails while generating the service name for ReceiptExportRequest with the following error:

Could not create service name for Receipt Export Request ( Error when creating configuration BUYERGOODSRECEIPTCREATE_ASYN_I for DT object BUYERGOODSRECEIPTCREATE_ASYN_I. Error when reading variant "BUYERGOODSRECEIPTCREATE_ASYN_I" )

Cause

The SP24 upgrade causes an inconsistency in the web service configuration for the ReceiptExportRequest.

Because of this inconsistency, the program ARBCIG_IF_SETUP_P2P cannot generate the required service configuration and fails.

Resolution

To resolve this issue execute report WEBI_CHECK_REPAIR for the affected service definition to fix the inconsistency, according to SAP Note 1847901 - Repair of service definitions.

If the issue still persists, please file an service request with the component BC-ESI-WS-ABA.

See Also

Refer to SAP Note 2170390 for additional details.

Keywords
