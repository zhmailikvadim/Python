Symptom

You are using dependent components in SAP CRM in conjunction with solution configuration (SSC) in SAP CRM.

One of the dependent components of a solution product is configurable.

When the system creates the dependent component in the quote, order, provider order, it does not create the default configuration for the component.

Reason and Prerequisites

The combination of solutions and configurable dependent components was not supported.

Solution products can (equivalent to the dependent components) include products as dependent components which are not part of the bill of material of the solution product.

In order to remain fully compatible with the IPC and allow customer to continue to use their models and dependent component masterdata, we have implemented this behavior.

Solution

The solution consists of an ABAP correction and a Java correction which need both to be present in the system.
