Symptom

When you change the selected shortage definition in the object page view of F0251A - Manage Material Coverage, the stock/requirements list and the stock availability chart are updated, but the shortage information in the header and the "First Shortage" section are not.

Other Terms

F0251A, Manage Material Coverage, shortage quantity, day's supply, daysofsupplyduration, time till shortage, stock/requirements list, shortage definition

Reason and Prerequisites

The data bindings for those parts of the UI were not set up correctly, so that the data was loaded only once with the initial shortage definition, but not updated by changing the selection.

Solution

Implement this note to fix the issue.
