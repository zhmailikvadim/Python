Symptom

Component: Foundation for EHS Management
Module: Location structure 

When you search for risk assessments or compliance scenarios in EHS using the location-based search, the system displays the location structure in an old UI tree. This UI tree has display issues when you resize the screen.

Other Terms

CL_EHFND_INCL_SUBLOC_HIER_FDR

Reason and Prerequisites

A program error in the system causes the symptom.

You find the prerequisites in the relevant correction instructions.

Solution

After implementing this SAP Note, the system enables you to switch to the location structure form with a new UI tree that provides enhanced features and properties.

Alternatively, you import the corresponding Support Package for the component extension of SAP Environment, Health, and Safety Management or the Feature Package Stack for the SAP S/4HANA system.
