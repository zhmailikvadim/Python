Symptom

Enabling a feature via Configuration Variant which is disabled in the base configuration. You then received sync error 'Entity set <entity> does not exist' with the $metadata call

Error is due to $metadata XML does not contain the entities. MAIF MPC class does not consider configuration variant settings.

Other Terms

Configuration Variant, metadata, MAIF, MPC class, S4MOBADDON-8006, TCI

Reason and Prerequisites

Programming Errors

Solution

Implement attached correction instructions.

This is a TCI verson of SAP Note 3701068.
