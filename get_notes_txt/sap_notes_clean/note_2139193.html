Symptom

Flag "Allow RFx response modification" is not automatically  set for Outline and Services items.

Environment

  • SAP enhancement package for SAP Supplier Relationship Management

Reproducing the Issue

1- Create RFx

2- Go to Items tab and set flag "Allow RFx Response Modification" at the header overview

3- Add any outline or service

4- Flag is not set automatically for these two item types

Cause

Switch is missing in the customizing.

Resolution

When the Outline and Services are added, there is a check in the code (/sapsrm/cl_sw_service_proc_1=>serv_proc_ge_is_active) where if this switch is not set true, the header flag is not read.


To activate this switch, follow the steps:

- Go to tx SFW5 and activate Business Function SRM_CONT_IMPR

- Go to SPRO -> SRM -> SRM Server -> Cross App -> Service Procurement -> Activate Service Procurement Innovations -> set the SRM_701_SERV_PROC_GE "Active"


Once this is done, you should have the flag set automatically by adding Outline and Services.
