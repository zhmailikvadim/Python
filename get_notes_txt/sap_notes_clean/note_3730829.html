Symptom

When provisioning to target system SAP BTP Platform Members (Cloud Foundry), the IPS job finishes with the following error:

  • error=Invalid organization. Ensure that the organization exists and you have access to it

Environment

Identity Provisioning Service (IPS)

Reproducing the Issue

  1. Configure a target system of type "SAP BTP Platform Members (Cloud Foundry)" in Identity Provisioning.
  2. Run a provisioning job.
  3. Observe that the job fails with error "Invalid organization. Ensure that the organization exists and you have access to it".

Cause

The target system property btp.cf.pm.landscape is misconfigured (wrong technical key/wrong separator, etc.).

Resolution

Make sure that the value entered for property btp.cf.pm.landscape is correct.

For property btp.cf.pm.landscape it is necessary to enter the technical key or the host name of the landscape in which your multi-environment subaccount with enabled Cloud Foundry environment is located.

For example: If the API endpoint is api.cf.eu10-002.hana.ondemand.com, then for property btp.cf.pm.landscape enter value cf-eu10-002
