Symptom

  • An error occurs when using the Latest People Profile with a custom MDF object.
  • The error message displayed is: "No applicable renderer found." 

** Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.**

Environment

SAP SuccessFactors HCM Suite

Reproducing the Issue

  1. Navigate to the custom MDF in People profile
  2. Observe the error message "No applicable renderer found" displayed in the MDF object.

Cause

  • Broken fields in the MDF object.
  • Issues with the UI configuration. 

Resolution

  1. Set the broken field (e.g., effectivestartdate) to "not visible" in the existing UI to temporarily resolve the issue.
  2. Create a new UI to check if the issue disappears. If the issue is resolved, it indicates that the old UI is broken.
