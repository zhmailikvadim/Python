Symptom

You would like to consume the business event for Bank Payment Transaction Counter.

Other Terms

SAP Transactional Banking for SAP S4HANA 2.0, Business event, Payment Transaction Counter

Reason and Prerequisites

Special Development

Solution

Below notes have to be implemented in sequence to raise an event for Bank Payment Transaction Counter.

1. 3709633: This note contains all the RAP objects that are needed to trigger Bank Payment Transaction Counter business event.

2. 3716713: With this note the existing counter functionality is enhanced to enable the triggering of the Bank Payment Transaction Counter business event.

3. 3716714: With this note the existing payment item functionality is enhanced to enable the triggering of the Bank Payment Transaction Counter business event.

4. 3721948: With this note the customer can decide the number of entries(payload size) that are to be sent in the Bank Payment Transaction Counter business event.

Please refer the solution section of the above notes carefully for more information and implement the manual activities mentioned in the notes.
