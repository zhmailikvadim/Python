Symptom

Catalog was deactivated and History Tool shows User as System. How do I check when and who has deactivated the catalog?

Resolution

In order to track Deactivation catalog action, we can use our Audit Logs functionality to find this information.
If we go to Site Manager > Audit Logs, we can set the Event Type as Admin (tasks/files) and select the desired time-frame to check.
You should see entries like below, where we will display the User, who has performed the catalog deactivation:
"Initiated deactivation tracker for "Catalog Name", Version "Version Number"

Keywords

Catalog deactivation, subscription deactivated
