Symptom

You use the valuation function for leasing contracts in Real Estate Management. You execute valuation postings (RECEEP). You want to determine the trading partner (affiliated company) in the line items by using a logic that differs from the standard logic. This SAP Note provides a new method for the BAdI BADI_REEX_FI_BAPI.

However, the BAdI is not called if no main contractual partner is assigned in the contract. This is required, however.

Other Terms

RFRECEEP, VBUND, TRADEID, BSEG-VBUND

IFRS leasing, lease-in, leasing contract, valuation, right-of-use asset, RoU, new leasing standard, US GAAP, HGB

Reason and Prerequisites

Cause: An additional exit is provided.

This SAP Note is relevant for you only if the function for valuating leasing contracts in RE has already been activated in the system or if the SAP Note is a technical prerequisite for another SAP Note.

Solution

Implement the attached source code corrections and, if necessary, create a customer-specific implementation for the BAdI BADI_REEX_FI_BAPI, method°MODIFY_TRADEID_FOR_EVALUATION. Then implement the attached source code corrections.
