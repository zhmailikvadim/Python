Symptom

You approve a supplier registration request, not using Change&Approve but directly the approve button at the POWL or the approve button at the workitem detail screen.

In this case, you are not assigned as purchaser responsible to a created supplier.

Please note that there is a manual post-implementation step to be performed.

Other Terms

Business Partner, Potential Supplier, Bidder, Supplier, Contact Person, Master Data, Registration Request, Approval, Workflow, Purchaser Responsible,

Reason and Prerequisites

Program error

Solution

Implement the changes as of the correction instruction, or apply the corresponding support package.

Please note that there is a manual post-implementation step to be performed.
