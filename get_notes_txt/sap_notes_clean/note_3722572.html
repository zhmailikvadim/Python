Symptom

You are using program FAGL_FC_TRANSLATION for the Currency Translation. You would like the program to populate the key date to field ZUONR (assignment number in the document line) in order to make it more transparent which documents were posted for which translation run. However, the field is only filled automatically when the field status of the target account requires input as 'mandatory'. The field remains initial if it is set 'optional' by the field status.

Other Terms

FAGL_FC_TRANSLATION ZUONR field status

Reason and Prerequisites

Program design.

Solution

Implementation of the correcdtion instruction or corresponding Support Package.

The correction creates a new parameter in selection screen section "Posting Selections". If you would like ZUONR to be populated as well in case of being defined 'optional' by the field status set this parameter accordingly.
