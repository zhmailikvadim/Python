Symptom

You are currently using the Croatian company code for your business operations.

When using the Fiori app “Clear Incoming Payments” or any other application to clear accounting document, a fiscalization number is being generated even in simulation mode.

Other Terms

HR, Croatia, Fiscalization Law, Fiscalization Process, Simulate, Clear Incoming Payments, Clear accounting document, Accounting document,ACCDOC_SIMULATE , CL_FDC_ACCDOC_POST============CP, /CCEE/HR_FILL_BUPLA.

Reason and Prerequisites

Pregram error

Solution

Implement the Correction Instructions relevant for your Release. This SAP Note valid only for S/4 releases.

After implementing the Note, Fiscalization for zero net value in billing document is deactivated if force numbering is not activated.

As a general rule, SAP recommends that you install a solution by applying a Support Package. However, if you need to install a solution earlier, kindly use the Note Assistant to implement the correction instruction. You can find more information about the Note Assistant in SAP Service Marketplace, under service.sap.com/note-assistant.
