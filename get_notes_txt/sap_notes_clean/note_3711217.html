Symptom

Error message: C+ 721 No data is available for material xxxxxxx in period xxx  executing  transaction CKMLCP.

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Environment

  • S4HANA
  • SAP ERP
  • SAP enhancement package for SAP ERP

Reproducing the Issue

  1. Transaction CKMLCP.
  2. Input costing run and period.
  3. Error logs >  C+ 721 No data is available for material xxxxx  in period xxx .

Cause

Missing Material ledger  header data

Resolution

  1. In order to check if there are inconsistencies, go to transaction SE38 and run/execute report MLHELP_SAPRCKMU (ECC release) / report  FCMLHELP SAPRCKMU (S4HANA release) or transaction CKMC.
  2. Enter material and plant  and see if any inconsistency is reported.
  3. In order to repair these inconsistencies; Run report FML_REPAIR_ML_MASTERDATA located in SAP Note 3024883 
  4. In case inconsistencies are not resolved  with the report FML_REPAIR_ML_MASTERDATA ; contact SAP Support.
