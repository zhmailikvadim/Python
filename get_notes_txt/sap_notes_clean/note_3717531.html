Symptom

  • Error message: "Declaration Type does not exist"
  • Error code: Message Class /SAPSLL/CUS_ECC Message Number 101
  • Issue occurs during the incompleteness check in supplementary customs declaration (SCD) processing
  • The error prevents the SCD document from being completed and submitted to ATLAS.

Environment

  • Customs Document Processing
  • Global Trade Services (GTS)
  • Global Trade Services / Logistics Services (SLL)

Cause

  • The system does not determine the declaration type in the supplementary customs declaration due to the missing code.

Resolution

  1. Implement SAP Note 3577371 - DE ATLAS 10.1: System does not determine declaration type in supplementary customs declaration.
