Symptom

 Not able to download products from promotion or a plan into excel. This Note contain text of the Note 3716283.

Other Terms

Product Picker, Export products from a promotion Picker export to excel.

Reason and Prerequisites

Export functionality is not available

Solution

This note contain the text for the note 3716283, which provides a new functionality in the product picker in plans and promotions to allow exporting product information into excel.
