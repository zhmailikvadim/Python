Symptom

If a measurement reading is created in FSM with remarks longer than 40 chars, then on the ECC/S4 side in table IMRG an entry with the long text is created. However, for such an entry the time stamp (the field INVTS) is missing. Due to the lack of the time stamp, the measurement reading cannot be displayed in transaction IK03.

Other Terms

FSM, E4C, Connector, Measurement reading, IK03, IRMG, /PACG/ECM_MPREAD_IN

Reason and Prerequisites

Changes have been implemented in method IDOC_INPUT, class /PACG/ECM_CL_S_IDOC_I_MPREAD.

Solution

Please, implement this correction instruction.
