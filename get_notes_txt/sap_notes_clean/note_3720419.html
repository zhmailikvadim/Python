Symptom

  • Newly created Custom Attribute cannot be used for synchronization with Solution Manager.
  • The "Used In" field remains empty when selecting/adding the new attribute combination in the synchronization project wizard.

Reproducing the Issue

  1. Create a new custom attribute in the Business Process Model Connector for SAP Signavio solutions.
  2. Attempt to synchronize the attribute with Solution Manager.
  3. Navigate to the Synchronization Projects Wizard in BPMC.
  4. Select the attribute mapping and add the mapping for the customized attribute.
  5. Observe that the "Used In" field remains empty.

Resolution

  1. Reapply the changes made to the attribute mapping in the synchronization project wizard.
  2. Ensure the "Include" button in the bottom right corner is clicked to navigate to the next screen and save the changes to the server.
  3. If the values used for the attribute are preserved in the browser cache, delete the attribute first on the screen, re-add it properly, and then click the "Include" button to save the value to the database.
