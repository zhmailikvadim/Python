Symptom

This KBA explains how to use an SAP-generated CSV file to upload to the IRAS myTax portal.

Environment

  • Human Capital Management (HCM)
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP SuccessFactors Employee Central Payroll
  • SAP S/4 HANA

Resolution

For the Singapore Payroll solution IR8A report in myTax format, the latest SAP Note 3682025 - PY-SG: Legal Change for IR8A MyTax YOA 2026 has updated the solution to provide a CSV file that incorporates IRAS requirements.

There are two ways to create an IR8A Excel file for uploading to the IRAS myTax portal:

  • Using the IRAS myTax Excel template:
    1. Obtain the IRAS myTax template from the IRAS website.
    2. Open the template .xlsx file.
    3. Click on "Data," then select "From Text/CSV."
    4. Select the SAP-generated .csv file, refine the data, and then click "Load."
    5. Repeat step 4 for each sheet if applicable.
    6. Once complete, upload the Excel file to the IRAS portal.
  • Using a blank Excel format:
    1. Create a blank Excel file.
    2. Repeat steps 3 to 6 from the above method.
