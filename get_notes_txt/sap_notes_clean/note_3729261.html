Symptom

You use the following monitor applications to track documents:
- Monitor Process Document
- Monitor Transfer Document
- Monitor Message
- Monitor Exception Document

On the object page, you select the Linked Documents tab to view related document information. The Linked Documents table does not display the business object of the linked process document. You must open each linked process document individually to see this information.

Other Terms

Monitor Process Document, Monitor Transfer Document, Monitor Message, Monitor Exception Document, Linked Documents, Business Object

Reason and Prerequisites

Advance Development.

Solution

Implement the SAP Note or install further Support Package.
