Symptom

The EAM Create Maintenance Request Fiori app has a field called 'Failure Effect'  (Default - Catalog 6 ) (As of SAP S/4HANA Cloud Private Edition 2023 FPS02).

The current 'effect' field in SSAM (operational effect)  and the 'Failure Effect' introduced with SAP S/4HANA Cloud Private Edition 2023 FPS02 are not the same. This note introduces a new property in SSAM to support this scenario.

Other Terms

S4MOBADDON-8195, SSAM, /MERP/CL_PM_NOTIFICATION_OD, Maintenance Request, Failure Effect.

Reason and Prerequisites

New functionality.

Solution

Please apply the attached correction instructions in SAP system.
