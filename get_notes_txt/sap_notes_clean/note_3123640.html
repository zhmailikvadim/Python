Symptom

The new date "Accounting event date" or "Дата факта хозяйственной жизни" has been added to the MX-3 form according to the Ministry of Finance order #62 from 16 April 2021 FSBU 27/2021.

"Accounting event date" is filled by default from Actual Goods Movement Date.

It can be changed before output in BAdI FIRU_MX1_MX3_OUTPUT.

Other Terms

MX-3, SDDEL_MX3_RU, CL_FIRU_FDP_V2_MX3_HELPER, FIRU_S_MX3_HEADER, FDP_V2_MX3_RU, FIRU_MX1_MX3_OUTPUT.

Reason and Prerequisites

Legal Change

Solution

 Install the appropriate Support Package mentioned in this SAP Note or implement manual pre-implementation steps, correction instructions and manual post-implementation steps.
