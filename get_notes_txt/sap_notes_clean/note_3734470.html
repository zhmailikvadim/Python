Symptom

Custom fields are not available for selection in KPI Cards at the home page.

Environment

SAP Sales and Service Cloud Version 2

Reproducing the Issue

  1. At Home page start adaptation mode.
  2. Click on the pencil button for the KPI Card.
  3. Click on the dropdown for the KPI Field.
  4. Notice only standard fields are shown. 

Cause

The custom field is not added to the entity view under advanced filter.

Resolution

In order for the custom fields to appear in the KPI card please add the fields to the advanced filter of the entity using adaptation mode. For example, if the KPI card is for Leads, then on advanced filter of the Leads view the custom field should be added.

  1. Open Navigation menu.
  2. Go to the affected entity view e.g. Leads.
  3. Open advanced filter.
  4. Start adaptation mode.
  5. Click on the pencil button for the advance filter.
  6. Click on add from list.
  7. Click on > for Extensions.
  8. Add the wanted custom fields.
  9. Click on Apply.
  10. End adaptation mode.
