Symptom

This SAP Note optimizes the handling of additional IR8A Appendix 8B records.
When the IR8A report generates an additional form, any Appendix 8B record that was already reported in the original form and has no changes will not necessary to be included in the additional form.

Other Terms

Appendix 8B

Reason and Prerequisites

This is a correction.

Solution

Apply the HR Support Package to get this solution.

Alternatively, you may apply correction instruction with this SAP Note if the changes are required immediately.
