Symptom

When an admin makes a class inactive and there are users enrolled in it, the users can still see the class on their Learning Plan.

Environment

SAP SuccessFactors Learning

Reproducing the Issue

  1. Assign/Register Users in a class 
  2. Make the Class inactive 
  3. Go to the User's Learning Plan 
  4. It still shows as "enrolled".

Resolution

This is working as designed, the Admin should take care of the registrations. This is part of LMS Best Practices.
