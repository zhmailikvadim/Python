Symptom

When using RFA Replicate CO Master Data you cannot replicate hierarchy data for a field listed on the Rules tab, and you get an error message.

For the following fields: Company Code, Vendor, Customer, Business Area, GL Account, Material Number, the function replicates only master data. Cost Object Hierarchy data is not relicated. An error message informs the user about the fact that hierarchy data is not avaialable.

Other Terms

LOD-PER-OP Rel 3.0 SP 27

Reason and Prerequisites

Program error

Solution

 In order to get the solution, please implement the automatic correction instructions using transaction SNOTE.
