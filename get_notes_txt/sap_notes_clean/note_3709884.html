Symptom

In the process "Receipt of SLP/TEP/PSET from DSO (SUP View)" (DE_B_EQS_SLP_RECEIVE_SUP), the system doesn't allow you to import the same profile version again.

This issue is relevant for the market role: SUP.

Other Terms

Profile version, SLP, TEP, PSET, /US4G/IF_BADI_EQS_SLP, /US4G/IF_BADI_EQS_SLP~CHECK_EQS_SLPVERS_EXIST, BAdI, enhancement

Reason and Prerequisites

The error occurred because the profile version import logic didn't support multiple imports for the same version.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the system includes a BAdI method (/US4G/IF_BADI_EQS_SLP~CHECK_EQS_SLPVERS_EXIST) that allows you to implement your own enhancement for handling profile version imports.

You need to include your custom enhancements and retrigger the process. You can verify the fix by checking that the process works as required and allows reimporting the same profile version.
