Symptom

  • Setup build apps, but unable to see C4C destinations in Integration tab of application.
  • Configured BTP Destination, but the destination does not appear in Build Apps.

Environment

SAP Build Apps

SAP Sales Cloud and SAP Service Cloud Version 2

Reproducing the Issue

  1. Configure BTP Destination for Mashup and Discovery Service.
  2. Attempt to view the destination in Build Apps under Integration.
  3. Observe that the destinations are not available in Build Apps.

Cause

  • The user is using the Default Plan, which does not include support for SAP Sales Cloud and SAP Service Cloud Version 2 in SAP Build Apps.  See https://help.sap.com/docs/build-apps/service-guide/setup-sap-build-apps-with-sap-sales-cloud-version-2.

Resolution

Only the Standard Plan support SAP Sales Cloud and SAP Service Cloud Version 2.
