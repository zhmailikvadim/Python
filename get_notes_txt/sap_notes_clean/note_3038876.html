Symptom

  • You have activated business function HCM_LOC_CI_110 as per note 2967001 and 2998641.
  • Test posting run for test payroll results ends with error  3G163 "payroll result" "could not be read from the puffer"

Environment

  • Human Capital Management (HCM)
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA

Cause

  • The error in simulation is caused by the entries missing in table PCL2 with relid XT

  • Those entries are generated when payroll is executed

Resolution

  • Then how to generate payroll results with RELID XT in PCL2, it require the BFs HCM_LOC_CI_110 and run test payroll in PCC after activated the BFs

  • If the above doesn't work for your, please search for other notes with the same error and if you find no solution, please open an case to SAP Support.
