Symptom

Error to login via Single Sign On (SSO) in the SAP Sales cloud v2 mobile app: “Invalid SAML configuration. Please contact your system administrator.”

Environment

SAP Sales Cloud Version 2

Reproducing the Issue

Cause

The SAP Sales cloud mobile app supports one SSO configuration. If there are multiple configurations, choose one as the default.

Resolution

  1. Login to SAP Sales Cloud V2 system.
  2. Navigate to settings.
  3. Open Identity provider configuration view.
  4. Under Trusted Identity Providers, set one of them as Default.

See Also
