Symptom

You run the PREVIRED Interface (HCLCPRVD0) report with the Gratuity (P_02) payment type for an employee in the following scenario:

  • In the Planned Working Time (0007) infotype's, the Part-Time Employee (TEILK) field is unchecked.

In the scenario described above, the system generates the PREVIRED output with the Tipo de Jornada (93) field set to zero.

Other Terms

&PREVIRED - For further information, see SAP Note 1730752 - CONSULT: List of tags for Chile notes

PC00_M39_RPRV; DAQ;

Reason and Prerequisites

Reason

This issue happens because the Employee journey type (JORTY) field is not included in the Gratificación - CCAF (GCAF) identification section of the PREVIRED Interface (PRVD) form.

Prerequisites

The following Support Package levels are the minimum prerequisites:

Release Support Package
S4HCM 102 HR SP 00
S4HCM 101 HR SP 06
S4HCM 100 HR SP 19
SAP_HR 6.08 HR SP D0
SAP_HR 6.04 (EhP6 and EhP7) HR SP K2

For more information, access the SAP ERP Human Capital Management - Support Package Schedules.

When you install an SAP Note, some prerequisites might be required. The prerequisites are SAP Notes that have already been released and that change the same context of the current one. Then, even if these prerequisites are not necessarily related to the symptom of this SAP Note, you must implement them due to technical reasons.

Solution

From now on, the system correctly populates the employee's journey type.
