Symptom

In the Direct Agreement, the header display profile fields are not being shown in the Overview tab. Although the configuration and data are maintained, the corresponding header fields are not visible on the screen.

This note ensures that the display profile fields are correctly shown in the Overview tab.

Other Terms

Header Display Profile, /IRM/IPAGRM, screen_modify_0117, /IRM/IPSDAM, /IRM/IPPDAM

Reason and Prerequisites

This is caused by program error.
Note : SAP Note#3722323 is available for ECC release.

Solution

Please implement the correction instructions. These corrections are only valid if you have applied the following Support Packs.

Release V/4 – 2023 V2, 2023 V3 and Support Pack 1

Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object
