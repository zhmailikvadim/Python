Symptom

Execution of task 'Determine Usage of Involved Objects' of an INPLACE conversion fails with the error even the root cause has been solved:

"Parameter for task 'Checklist for Usage of Involved Objects' missing; maintain task before execution"

Environment

SAP NetWeaver 7.5

Resolution

Before starting the tasklist run again, the root cause why there are errors has to solved. In order to continue with the tasklist run, the following steps are necessary:

  1. Call the tasklist run in change mode.
  2. Call the hotspot of 'Checklist for Usage of Involved Objects".
  3. Double click on the check button in order to perform a check again.
  4. In case the status change from red to yellow/green, make sure that the resolved flag is set.
  5. SAVE the tasklist run.
  6. Continue with tasklist run.
