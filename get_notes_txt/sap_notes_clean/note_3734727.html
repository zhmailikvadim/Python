Symptom

It's required to have option for scheduling backup/DB jobs eg: consistency check from DBACOCKPIT.

NOTE : This is a JAVA only system on SAP HANA

Environment

SAP HANA DB Version 2.0

Cause

DB13 won't be available for JAVA only system.

Resolution

This is possible, but it is necessary to have an ABAP Stack system where one can run DBACOCKPIT. In this DBACOCKPIT one can integrate external HANAs (e.g. the HANA of a JAVA only system) and afterwards one can schedule Backups for this external HANAs

NOTE : The ABAP stack system must have release 7.02 or higher. Every ABAP stack system equal/higher 7.02 has a DBACOCKPIT/DB13.as HANA system

Keywords
