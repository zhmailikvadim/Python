Symptom

  • An error occurs when attempting to manually add the configuration object SAP_FIN_CO_RB_COST_CENTER as described in note  3655718
  • The system returns the error message: "SAP_FIN_CO_RB_COST_CENTER (Review Booklet) does not exist."

Environment

  • SAP S/4HANA Cloud Private edition

Reproducing the Issue

  1. Implement SAP Note 3655718 and apply automatic correction instructions.
  2. Perform manual steps as described in the SAP Note.
  3. Attempt to manually add the configuration object SAP_FIN_CO_RB_COST_CENTER in a workbench transport request.
  4. Observe the error message: "SAP_FIN_CO_RB_COST_CENTER (Review Booklet) does not exist."

Cause

  • Configuration Missing

Resolution

  1. Retest after executing transaction ACTIVATE_REVIEW_BOOK in each relevant system and client.
  2. Ensure that the ABAP Development Tools (ADT) version being used is up to date.
