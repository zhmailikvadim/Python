Symptom

User would like to know how to rename "SAP CPQ Sales Quote" button in Opportunity.

Environment

  • SAP Sales Cloud Version 2

Reproducing the Issue

  1. Go to Guided Selling.
  2. Open any Opportunity.
  3. Navigate to the Sales Quotes section and select + (create).
  4. Observe the button labeled "SAP CPQ Sales Quote".
  5. User would like to know how to rename it. 

Cause

Resolution

  1. Open Language Adaptation.
  2. Select the application: "Guided Selling - sap.crm.service.guidedSellingService".
  3. Go to Field Labels.
  4. Locate the label key "lbl_create_cpq_quote".
  5. Enter the desired text in the Target Language Text field.
  6. Save the changes.
