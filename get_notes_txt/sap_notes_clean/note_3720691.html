Symptom

When trying to update quantities in CS02 with dot for decimal places, error message appears: 00088 - "Input must be in the format _._._._~,_V".

Environment

  • SAP ERP Logistics (LO)
  • SAP R/3
  • SAP R/3 Enterprise
  • SAP ERP Central Component
  • SAP ERP
  • SAP Enhancement package for SAP ERP
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA, on-premise
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Transaction CS02.
  2. Select material.
  3. Choose plant.
  4. Change component quantity for a material for example to 1.2
  5. Save.
  6. Error 00088 is being raised.

Cause

Incorrect configuration of decimal notation in transaction SU3.

Resolution

  1. Access transaction SU3 or navigate to System > User Profile > Own Data.
  2. Open the "Defaults" tab.
  3. Locate the "Decimal Notation" field.
  4. Change the decimal notation to "1,234,567.89" to allow the usage of a dot (.) for decimal places.
  5. Save the changes to the user.
  6. Log off from the system to apply the changes.
  7. Log back into the system, error should not appear anymore
