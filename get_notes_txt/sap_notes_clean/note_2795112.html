Symptom

You use SAP GTS to send an import message from a customs import declaration to the Danish customs system.

The customs system sends an error message because a zero number of packages is not allowed.

Message reference: Item 2/0 ,Sub item2
Error: 1250 ; Feltet er ikke udfyldt korrekt
Kolli_ant

Other Terms

FOE02, FUE04 or STF07, 02, 03 ,07, SKAT, Denmark, import, The field is not filled in correctly, Kolli_ant﻿

Reason and Prerequisites

Program error

Solution

Implement the correction instructions.

You can then enter the supplementary item on an item with zero number of packages as usual. When sending the message, the system ensures that the number of packages are declared from the main item.

Reason and Prerequisites

Erroneously shipped table entry

Solution

Implement the attached correction instructions.

Then execute the report RFFM_TFB03T_DEL_BA with transaction SE38 or SA38. You must unflag the checkbox 'Select Data Only (No Deletion)' to delete the entry 'BA' from the database. If you need to remove this entry from all system clients, you must also unflag the checkbox 'Only Current System Client'.
