Symptom

The Award Confirmation tab does not appear in the event monitoring interface. Confirmation message data includes awarded participant names, contact names, and award confirmation IDs. 

Cause

  • The Award Confirmation tab is displayed only when the RFQ origin is S/4HANA. When the origin is other than S/4HANA then the tab does not appear.
  • If the backend system does not send a confirmation response after processing the award, the tab (when applicable) does not update with processing status and the Audit Log shows only a generic message.

Resolution

  1. Verify if the feature DMS-15883 is enabled.
  2. Ensure the RFQ origin is set to S/4HANA because the Award Confirmation tab is available only for S/4HANA Cloud Public Edition originated RFQs. Please note this functionality is not available for SAP S/4HANA.
  3. Review and adjust integration configuration so that RFQs/events originate from S/4HANA when Award Confirmation visibility is required.
  4. Confirm the backend system sends a confirmation response back to Sourcing after award processing (success or failure) so the tab can display the status or error details.

If further configuration guidance is needed, engage with your Best Practice Team or Consulting to validate and enable the required integration setup.
