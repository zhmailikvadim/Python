Symptom

Extrinsic fields [isSpltPayment] and [splitPaymentDescription] for InvoiceDetaiRequest document type missing in Managed Gateway for Spend & Network mapping tool


Environment

S4HANA
Managed Gateway for Spend & Network 

Cause

You cannot find extrinsic fields [isSplitPayment] and [splitPaymentDescription] for InvoiceDetailRequest in Managed Gateway mapping tool

Resolution

To resolve this issue, perform the below steps by navigating to Managed Gateway For Spend & Network Portal :

  1. Open the Mapping Tool and navigate to the required mapping configuration.

  2. Add a new condition for Split in the mapping structure.

  3. Click on OK to save the condition.

  4. Verify that the Source Condition with Extrinsic Name = isSplitPayment is visible in the mapping.

  5. If required, add an additional condition for SplitDescription following the same process as stated above.

  6. Save and deploy the mapping & validate the mapping configuration to ensure the changes are correctly applied.
