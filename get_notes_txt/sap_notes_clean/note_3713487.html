Symptom

You use transactions FP04 and FP04M to process write-offs for the same document. The system calculates different tax amounts in FP04M, resulting in incorrect values.

Other Terms

FP04, FP04M, tax, VAT, value-added tax, HANA_FP04M

Reason and Prerequisites

Cause:

This is a program error.

Prerequisites:

  • The document which will be written off contains multiple identical general ledger line items.

Solution

After implementing this SAP Note, the system calculates tax amounts correctly in transaction FP04M.

The corrections can be implemented in advance. The corrections are normally delivered with the relevant support package.
