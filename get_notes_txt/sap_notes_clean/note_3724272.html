Symptom

When trying to select a foreign trade classification number in the classification search help, the runtime error CX_SY_CONVERSION_NO_NUMBER occurs.
The argument '*00' cannot be interpreted as a number.

Other Terms

/SAPSLL/SAPLLLNS_SERVICES, GET_NMBR_OF_ASSGD_MEASURES, CX_SY_CONVERSION_NO_NUMBER, product classification, classification, search help, classification help,, import code number, tariff numbers, commodity codes

Reason and Prerequisites

Program error.

Solution

Carry out the manual steps below and implement the correction instructions assigned to your release.
