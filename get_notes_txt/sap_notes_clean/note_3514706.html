Symptom

Instead of displaying a default initial authorization group, 'No Data' is displayed by clicking on each groups in 'Authorization' tab of 'Business Process Improvement - Administration' tile.

* Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

Solution Manager 7.2 SP08 or higher

Reproducing the Issue

Solution Manager 7.2 SP08:

  1. Transaction SM_WORKCENTER
  2. 'Business Process Improvement - Administration' tile
  3. Authorization tab
  4. Click on the icon with 2 persons at the left top of the screen
  5. Select 'Key Figure Groups' or other groups in the dropdown menu
  6. One initial group which name starts with 'SAP_DEFAULT_ALL_*' should be created for each groups but 'No data' was displayed instead.

As of Solution Manager 7.2 SP09:

  1. Transaction SM_WORKCENTER
  2. 'Business Process Improvement - Administration' tile
  3. Authorization tab
  4. Click on the icon with 3 lines at the left top of the screen
  5. Select 'Key Figure Groups' or other groups in the dropdown menu
  6. One initial group which name starts with 'SAP_DEFAULT_ALL_*' should be created for each groups but 'No data' was displayed instead.

Cause

Services and OData services for Business Process Improvement were not activated/created correctly.

Resolution

  1. Create new OData services referring to the number 3) of the note 2424585.
  2. Activate the services listed in the number 4) of the note 2424585.
