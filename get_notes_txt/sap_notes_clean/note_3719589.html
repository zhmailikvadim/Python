Symptom

You create and send a statement of earnings for maternity pay and receive an error confirmation with error DSLWK20 (RECEIVER NUMBER is not the actual recipient of the notification). This error occurs for part-time employees.

Other Terms

Social Insurance

Reason and Prerequisites

Since the information about actual health insurance is regularly available in the system for part-time employees, statements of earnings for maternity pay must be reported as of DSLW Version 13 and for situations as of 1/1/2026 using the payroll program as part of the EEL procedure.

Solution

The program error is corrected.

In the future, the correct receiving office will be determined.

Implement the attached correction instructions or import the specified support package.

After you import the HR Support Package or implement the correction instructions, you do not have to perform any further follow-on activities.
