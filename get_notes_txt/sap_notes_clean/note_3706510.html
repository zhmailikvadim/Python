Symptom

  •  An error occurs when using the a custom field in the Local Job Info of a country. The error message states: "custom-string is a deleted field. Adding this field to the configuration again can lead to corruption of data and regression in API because this field includes existing data."
  •  The field is successfully used for other countries.

Environment

SAP SuccessFactors Employee Central

Reproducing the Issue

  1. Manage Business Configuration.
  2. Select Job Info a specific country.
  3. Attempt to add a new custom field.
  4. Receive error.

Cause

  • This warning is valid when field is configured in global HRIS Element and country specific configuration.
  • We will enhance the system warning to consider this scenarios. 

Resolution

  • SAP Engineering team plan to fix this issue with an estimate delivery of the 2H 2026 release.
  • Please follow this KBA for further updates.
