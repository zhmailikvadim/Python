Symptom

When attempting to update the "Valid To" date in the source list to a date earlier than the current date, the system displays the error message: "Valid until date must not be in the past."

Environment

  • Materials Management (MM)
  • SAP ERP Core Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA

Reproducing the Issue

  1. Open transaction ME01.

  2. Enter the Material ID and Plant.

  3. Press Enter to proceed.

  4. Attempt to update the Valid To date to a past date.

  5. Observe the system error message: "Valid until date must not be in the past."

Cause

Standard system design

Resolution

Error message 06345 cannot be changed to a warning type, as it is hard-coded as an error in the system. It is therefore recommended to maintain the "Valid To" field with a date in the present or future to avoid this error.
