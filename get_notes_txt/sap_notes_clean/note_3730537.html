Symptom

When attempting to partially return subcontracted material to a supplier using transaction MIGO, the system displays error message 8B288: 'Enter an alternative amount'.

Environment

  • SAP ERP
  • SAP S/4HANA

Reproducing the Issue

  1. Open transaction code MIGO
  2. Select "A02 Return delivery" and "R02 Material document"
  3. Inform the reference document number and hit enter
  4.  Check item as ok and click on "check" button

Cause

This error occurs when attempting a subcontracted material reversal in MIGO without selecting the correct document type and cancellation method.

When posting the document, the system executes function module J_1B_IM_NF_EXBASE_CHECK and for the RAISE exbase_required to trigger, ALL conditions must be met:

  • pstyp = '3' 
  • L_mseg_kzbew is populated (not blank)
  • L_vm07m_xstor is a reversal movement (populated)

Current Behavior: When the variable 'reversal' equals '2' and the field 'L_vm07m_xstor' is empty, the validation condition fails, triggering error 8B288.

Resolution

Process an actual reversal transaction (not a standard return) in MIGO by selecting the following in order:  

  1. Select the action type 'A03 Cancellation' and document type 'R02 Material Document'
  2. Enter the reference document number and all required fields
  3. Click Check to validate the actions performed
