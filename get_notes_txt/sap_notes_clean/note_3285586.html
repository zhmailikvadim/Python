Symptom

Payroll for December 2022 with pay date in January 2023 is calculated in HRK. As valid currency on a pay date of the salary is EUR, payslip should also display amounts in EUR.

Other Terms

EURO changeover, payslip, EUR, HRK, alternative currency

Reason and Prerequisites

  • EURO changeover in Croatia on January 1, 2023.
  • Customer runs payroll for December 2022 in payroll currency HRK.
  • Payslip for pay date in January 2023 should display amounts in EUR.
  • This SAP Note is for information purposes only and cannot be implemented.

Solution

Problem appears only once and only at customers, who calculate payroll for 12/2022 in HRK. Complete solution for the problem is not provided by SAP out of the box. Using built-in payslip functionality "Alternative currency" as-is will take the exchange rate HRK -> EUR from currency customizing (TCUR* tables), but on a start date of payroll period (01.12.2022). As Croatia is using variable exchange rate HRK -> EUR until end of the year (fixed rate is enforced on 31.12.2022 with infinite validity), exchange rate will be incorrect for the aimed purpose.

There are other tools and techniques, how to meet requirement described in the symptom. Three options are listed here.

  1. Create new payslip by copying existing one and set conversion module 'KE' on each field with amount.
    • Amount in field with conversion module 'KE' will be converted from HRK to EUR, using fixed rate entered in T7HRRE table (key EURO_CONVERSION / FIXED_RATE).
  2. Temporarily set the exchange rate to fixed one.
    • Set exchange rate in currency customizing to fixed rate on date 01.12.2022. Exchange rate type 'M' is the right one in most cases.
    • Run the payslip program with option Alternative currency = EUR.
    • Set fixed exchange rate only for the time of running the payslip program.
  3. Create new artificial currency
    • In currency customizing, create new currency, which will have only one, fixed exchange rate on a date 01.12.2022.
    • Run the payslip program with option Alternative currency = <new currency>.
