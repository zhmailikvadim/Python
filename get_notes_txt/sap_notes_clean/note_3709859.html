Symptom

  • Users are unable to upload attachments in Freight Unit, Freight Order, or related Transportation Management apps.
  • Unable to upload attachments to the App Display Freight Unit.
  • Not able to attach files in output management FO/FU
  • The "Document Type" dropdown is empty, and no options are available for selection.
  • As a result, attachments cannot be uploaded.

Environment

  • SAP S/4HANA Cloud, Private Edition 2022
  • SAP S/4HANA Cloud, Private Edition 2023

Reproducing the Issue

  1. Open a Freight Unit or Freight Order.
  2. Navigate to the "Attachments" section or Output Management -> Attachments
  3. Switch to "Edit" mode by clicking the button in the upper left corner.
  4. Select "File" and attempt to upload an attachment.
  5. Observe that the "Document Type" dropdown is empty, and no options can be selected.

Cause

The business object is configured to use an attachment schema (for example, DEFLT_ATS), but no attachment types are assigned to this schema in customizing. When the attachment framework evaluates the schema and finds zero allowed attachment types, the system cannot determine any valid document types. As a result, the Document Type dropdown remains empty in the UI.

Resolution

Assign at least one attachment type to the attachment schema used by the business object.

Customizing Steps:

  1. Open the SAP system and navigate to the following customizing path:

SPRO/SIMG -> Cross-Application Components -> Processes and Tools for Enterprise Applications -> Reusable Objects and Functions for BOPF Environment -> Dependent Object Attachment Folder -> Maintain Attachment Schema -> Assign Attachment Types.

  1. Select the attachment schema used by the object (for example, DEFLT_ATS).
  2. Assign at least one attachment type to the attachment schema used by the business object.
  3. Save the changes.
  4. Test it again to confirm that the "Document Type" dropdown is now populated and attachments can be successfully uploaded.

After this change: The Document Type dropdown is populated. Attachment upload works as expected.
