Symptom

  • Inquiry regarding the creation of a custom CDS view based on the ACDOCU table.
  • No CDS view template for the ACDOCU table is available on the SAP Help Portal, while templates for ACDOCA and BSEG/BKPF tables are provided.
  • Request for guidance on a recommended CDS view template or example for the ACDOCU table.

Environment

SAP S/4HANA Cloud Public Edition

Cause

In SAP S/4HANA Cloud Public Edition, the table ACDOCU (Group Reporting – Universal Document) is a backend consolidation table that is not released for customer extensibility.

ACDOCU is technically complex, contains a very large number of fields, and is used internally by SAP for Group Reporting processing. For reasons of system stability, data consistency, and upgrade safety, direct access to ACDOCU via custom CDS views is restricted and not supported in the public cloud.

As a result, SAP does not provide CDS view templates and does not support the creation of custom CDS views directly on ACDOCU in the public cloud.

Resolution

    Creating custom CDS views directly on table ACDOCU is not supported in SAP S/4HANA Cloud Public Edition.

    To consume Group Reporting data, please use SAP-delivered, released CDS views for Group Reporting, which expose selected consolidation data in a supported and upgrade-safe manner.

    Refer to the official documentation CDS Views for Group Reporting to identify the appropriate CDS views for reporting or analytical requirements.

    This behavior is by design.
