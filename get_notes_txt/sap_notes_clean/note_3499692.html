Symptom

• The service /sap/opu/odata/UI2/INTEROP is not found in the DEV system when attempting to add it.
• The issue persists even after testing with user parameter WCF_IGNORE_ENHANCEMT with value A.

Environment

SAP Fiori Launchpad ABAP Services

Reproducing the Issue

1. Attempt to add /sap/opu/odata/UI2/INTEROP service in the DEV system.
2. Observe that the service is not found in the system.

Cause

Wrong name of service

Resolution

The correct service to be added should be /UI2/INTEROP instead of /sap/opu/odata/UI2/INTEROP.

Keywords

SAP S/4HANA, Service, DEV System, /UI2/INTEROP, /sap/opu/odata/UI2/INTEROP, Service not found.
