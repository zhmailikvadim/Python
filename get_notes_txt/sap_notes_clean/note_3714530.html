Symptom

Benefit program is correctly configured. However, its not displaying in the benefit overview screen.

Environment

SAP SuccessFactors Employee Central - Global Benefits

Reproducing the Issue

  1. Go to any employee profile who is eligible for the benefit program.
  2. Click on Go to Benefits.
  3. The benefit program is not shown. 

Cause

Benefit open enrollment is configured for the legal entity. When open enrollment is configured, benefit program is not supported.

Resolution

  • It is a system limitation currently. 
  • For updates to design, please raise an enhancement request by following KBA 2090228 on "How to submit an enhancement request" as process.
