Symptom

I have Approved the Supplier Request but I still do not see the option to create an Engagement Request.

Resolution

To create an Engagement Request from a Supplier Request you will need to have the below features and parameters enabled:

  • CreateEngagementRequestFromSupplierRequest
  • EnableEngagementRequestDocumentTypes
  • SRUpdateRolesAndPermissions

Also within SM Admin the feature toggle SM-4976 and parameter Application.SM.EnableERCreationInSupplierRequest will need to be enabled.
