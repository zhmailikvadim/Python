Symptom

The below error message occurs when transporting content between two SAC tenants using Content Network Storage with the option “Overwrite objects and data” in SAP Analytics Cloud:

"Import is disabled as the data format is incompatible"

Environment

SAP Analytics Cloud.

Reproducing the Issue

  1. Log in to SAP Analytics Cloud.
  2. Navigate to Transport > Import > Content Network Storage.
  3. Select the package.
  4. Go to the Import Options tab.
  5. Select the Overwrite objects and data option.
  6. Select the Include Data option.
    >> Observe the error: "Import is disabled as the data format is incompatible."

Cause

The "Enhanced Data Compression" option enabled during package export via Content Network Storage makes the data format incompatible with the target tenant

Resolution

To solve this issue, disable the "Enhanced data compression for Content Network packages" toggle on the source tenant:

  1. Log into the source SAC tenant with an administrator account.
  2. Navigate to System > Administration > System Configuration.
  3. Toggle off "Enhanced data compression for Content Network packages."
  4. Export the package (see Exporting Content for details).
  5. In the target tenant, import the newly exported package. For more information, see Importing Content.
  6. To re-enable compression (optional):
  1.  After the import is successful, return to the source tenant.
  2. Go to System > Administration > System Configuration.
  3. Enable the setting to resume enhanced compression for future exports.
