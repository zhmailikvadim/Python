Symptom

In the app "Manage Usage Records", some fields are visible, such as "Custom Reference". However, when exporting data via excel, the fields are not available in the exported Excel file.

Environment

SAP Subscription Billing

Reproducing the Issue

  1. Open the Manage Usage Records app.
  2. Open the list view settings.
  3. Select the some fields, such as Custom Reference.
  4. Confirm the fields are visible in the list view.
  5. Export the data to Excel.
  6. Observe that the fields are not included in the exported Excel file.

Resolution

This is a constraint for the export functionality. 

The export option in the "Manage Usage Records" app does not export data for fields that support multiple values (for example, custom references).
