Symptom

  • The field set in the clearing rule cannot be derived in F110
  • The field can be derived in manual posting.

Environment

  • SAP R/3
  • SAP S/4HANA

Reproducing the Issue

  1. Set clearing rule and Z field in T-code OBIA.
  2. Post in manual posting like FB05 and observe the field can be derived.
  3. Run F110 and observe the field cannot be derived in F110 posting.

Cause


Custom field set in the clearing rule is not populated by the SAP Standard Transaction F110

Resolution

  1. The custom field set in the clearing rule is not populated by SAP Standard Transaction F110
  2. Consider using manual posting methods like FB05 to derive the field.
