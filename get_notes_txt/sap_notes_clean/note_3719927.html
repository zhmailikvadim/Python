Symptom

  • Error when provisioning SAP Cloud Identity Services (SCI) in Identity Provider Administration Tool.
  • Error message displayed: "Tenant provisioning failed. Please try again later. If this error persists, please see KBA 3576392."

Environment

  • Identity Provider Administration tool

Reproducing the Issue

  1. Log in to the Identity Provider Administration Tool.
  2. Select your tenant card and click on "+ Add SAP Cloud Identity Services"
  3. Click on "Provision a New SAP Cloud Identity Services Tenant".
  4. Observe the error message: "Tenant provisioning failed. Please try again later. If this error persists, please see KBA 3576392."
