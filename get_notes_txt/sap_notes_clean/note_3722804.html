Symptom

The UI5 Gantt chart fails to load when the SAP UI5 version is 1.136 or higher the issue is caused by the UI5 bootstrap not executing, which prevents the required Gantt chart from getting loaded.

Other Terms

Web Planning Board, Gantt, UI5, MRSS, Bootstraping

Reason and Prerequisites

This was due to the bootstrap process not being triggered. The note is valid from version 1.136

Solution

Apply the automatic corrections.
