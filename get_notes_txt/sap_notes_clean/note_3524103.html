Symptom

In Customizing, under "Preference Management -> Vendor Declarations -> Define Settings for LTSD Process", you have selected the option "Send One E-Mail with All Forms".

During the creation of an outbound LTSD in the "Cross-Company Code Stock Transfer" process (Intercompany (IC) Stock Transfer), the system schedules the PPF action "Send Collective Forms by E-Mail". However, the PPF action is not required for the IC process.

Other Terms

cross-company-code stock transfer, ﻿intercompany﻿, IC, SLL_PPF_PRE_EMAIL_SCHED_COND﻿

Solution

Implement the attached correction instruction.

After you implement the corrections from this SAP Note, the PPF action "Send Collective Forms by E-Mail" is no longer scheduled for outbound LTSDs in the "Cross-Company-Code Stock Transfer" process.
