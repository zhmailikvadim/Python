Symptom

In an SAP HANA system, high CPU utilization is observed in SAP HANA Studio, SAP HANA Cockpit, or operating system monitoring tools (for example, such as top, nmon, or vmstat).
However, SAP HANA Alert ID 5 (Host CPU Usage) is not raised, or is raised inconsistently.

Typical observations:

  • CPU usage appears high in monitoring graphs
  • No corresponding SAP HANA alert is raised
  • The monitoring behavior is interpreted as incorrect

The behavior can be observed especially during short CPU peaks or temporary workload spikes.

Environment

SAP HANA DB Version 2.0

Cause

This behavior is expected and does not indicate a malfunction of SAP HANA monitoring.

SAP HANA Alert ID 5 is evaluated by the SAP HANA monitoring and alerting framework. The alert is not based on instantaneous CPU utilization values reported by operating system tools.

The alert evaluation is based on internal monitoring data processing, including:

  • monitoring sampling intervals
  • aggregated statistics collection
  • persistence of the threshold violation
  • alert evaluation cycles

Short CPU peaks or transient workload spikes visible in SAP HANA Studio or operating system tools may therefore not trigger Alert ID 5.

Operating system monitoring tools report CPU utilization differently than the SAP HANA monitoring framework, which evaluates statistical monitoring data collected by SAP HANA monitoring services.

As a result, visible high CPU usage does not automatically lead to the generation of Alert ID 5.

Resolution

No corrective action is required if the system is operating normally.

When reviewing the alert behavior, the following aspects are relevant:

  • the alert threshold configuration
  • the alert check interval
  • the duration of the CPU usage
  • the availability of monitoring data

Alert ID 5 is only raised if the CPU usage exceeds the configured threshold for a sustained period according to the SAP HANA monitoring framework.

SAP HANA alerts are intended for system health monitoring based on the internal monitoring framework.
