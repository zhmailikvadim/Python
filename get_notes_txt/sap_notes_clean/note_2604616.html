Symptom

Filtering worklists for values with special characters e.g. hyphen “-“ returns no results.

Environment

Business ByDesign

Reproducing the Issue

  1. Choose filter icon in Worklist.
  2. Use special character for filter criteria and press enter.

Cause

In SAP Business ByDesign the search syntax supports the following operators:

Resolution

If you want to use a character string as a filter and this character string contains one of the special characters (e.g. C-1025) you need to place the entire character string in quotation marks (e.g. "C-1025").
