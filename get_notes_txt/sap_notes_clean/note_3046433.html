Symptom

SAP Note 3006456 delivered an incorrect Customizing entry in the table view V_T582ITVCLAS ((Assign Check Classes and Define Permissibility) for the infotype 0118 (Child Allowance).

Other Terms

V_T582ITVCLAS, infotype 0118, CL_HRPADDE_INFOTYPE_0118, CL_HRPA_INFOTYPE_CONTAINER

Solution

Import the Support Package assigned to this SAP Note and then reconcile the table T582ITVCLAS in your customer client with client 000 if you have previously reconciled the incorrect entry for the infotype 0118 from SAP Note 3006456.
