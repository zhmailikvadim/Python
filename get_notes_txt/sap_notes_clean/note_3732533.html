Symptom

ECC and SRM in-structure field name difference leading to the partial data loss during RFC parameters mapping. Issue introduced via the SAP Note 3715464.

As result, the new GRs can't be posted after return even if the PO doesn't have FINAL_ENTRY set, the SC still has FINAL_ENTRY set.
Additionally, the PO Header and Item Links are missing from SC.
