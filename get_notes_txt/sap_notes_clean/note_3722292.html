Symptom

  • Unable to use file or file_format_group as target when using GBQ as source in CIDS.

Environment

  • SAP Cloud Integration for data services
  • Google BigQuery

Reproducing the Issue

  1. Create a task and select GBQ datastore as source.
  2. For target, there is no option to select File or File Group

Cause

This is by-design.

Resolution

When using GBQ as the source, only IBP is supported as the target. Refer to SAP Cloud Integration for data services PAM (Page 7) for detailed information: SAP Cloud Platform Integration PAM.

To request an enhancement for this functionality, refer to: KBA 0002991676 - How to Submit Ideas for SAP Cloud Integration for data services.

See Also
