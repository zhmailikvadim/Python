Symptom

While trying to update employee details by using "Maintain Employees" application (F2288A) in SAP S/4HANA system, you observed that the "Contact Information" tab is not editable. e.g. E-mail Address. 

 

Environment

SAP S/4HANA

Reproducing the Issue

Fiori app Maintain Employees, F2288A

Cause

Workplace Address Attribute Source Definition set as 'Default'.

Resolution

To enable editing, change the 'Source' to -> 'Business Partner'. 

Steps:

Start Transaction SPRO
-> SAP Reference IMG
-> Cross-Application Components 
-> SAP Business Partner
-> Additional Customizing for SAP Business Partners
-> Workplace Address Attribute Source Definition

(View TBZ_V_EEWA_SRC)

See Also

SAP Note 2570961 – Simplification item S4TWL – Business User Management , section "Maintenance of the Business User Workplace Address".

SAP KBA Note 3094750 - Business User Management - FAQ
