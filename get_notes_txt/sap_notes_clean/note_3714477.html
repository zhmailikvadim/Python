Symptom

Update Richtext Editor version to TinyMCE6 for the ITSM Apps Create Incident, My Incidents and Resolve And Dispatch.

"TinyMCE version 4 is used as editor. This version is no longer supported by TinyMCE and will be removed in future releases, therefore using it is at your own risk. Please consider upgrading to TinyMCE version 6."

Other Terms

RTE, Deprecated

Reason and Prerequisites

ST-UI SP 19 or 20

SAP UI Version >= 1.71.71

Solution

Please implement this SAP Note to solve the described issue.

After the note implementation, please make sure to reset the application index: execute report /UI5/APP_INDEX_CALCULATE via transaction SA38, go to tab "Single SAPUI5 Repository only", enter "AI_CREATE_INC" as repository and execute the report. Repeat it for SAPUI5 Repositories "AI_MY_INCIDENTS" and "AI_RES_DISP_INC".

You may also have to reset your local browser cache.
