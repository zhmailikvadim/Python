Symptom

  • The Check Tool - Document template for Print Forms service is consistent. (ONB2CheckHCPTemplatesExistInBizx) shows job failure.
  • Issues Found under System Health.

Environment

SAP SuccessFactors Onboarding

Reproducing the Issue

  1. Go to Check Tool.
  2. Click System Health tab and select Document template for Print Forms service is consistent. (ONB2CheckHCPTemplatesExistInBizx).
  3. Run Check.
  4. Observe that the job fails with X issues Found.

Cause

This is a system behavior.

Resolution

  • Document template for Print Forms service is consistent. (ONB2CheckHCPTemplatesExistInBizx)
    • This check only validate if there is any template which exist in PFS side but is not available on BizX side. This does not create any issue and can be ignored.
  • Please check another Check Tool - Document template is consistent. (ONB2CheckBizxTemplatesExistInHCP) , which validates if all the BizX onboarding templates are available in PFS or not. If any template missing here, it may create issue in document generation.
