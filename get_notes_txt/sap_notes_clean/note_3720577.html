Symptom

In the process "Receipt of Profile Definitions from DSO (SUP View)" (DE_C_EQS_PFDEF_DSO_RECEIVE_SUP), the BAdI /US4G/IF_BADI_UPD_PROF_DEF~UPDATE_PROFILE_DEFINITION is missing the importing parameter PROF_CAT_INT.
This issue is relevant for the market role: SUP.

Other Terms

EQS_SUP, Receipt of Profile Definitions from DSO, BAdI, UPDATE_PROFILE_DEFINITION, PROF_CAT_INT, importing parameter, profile definition

Reason and Prerequisites

The importing parameter was not added to the BAdI.

Solution

Implement the corrections of this SAP Note or import the relevant support package.
After implementing this SAP Note, the system includes the parameter PROF_CAT_INT as an importing parameter in the BAdI /US4G/IF_BADI_UPD_PROF_DEF~UPDATE_PROFILE_DEFINITION.
You need to retest the process. You can verify the fix by checking that the parameter PROF_CAT_INT is available in the BAdI.
