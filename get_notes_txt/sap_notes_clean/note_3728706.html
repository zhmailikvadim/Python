Symptom

When using file upload option to update tax classification data, which is not maintained yet, there is error "You have not fully maintained the tax classifications".

Environment

  • Master Data Governance on S/4 HANA on-premise 1709
  • Master Data Governance on S/4 HANA on-premise 1809
  • Master Data Governance on S/4 HANA on-premise 1909
  • Master Data Governance on S/4 HANA on-premise 2020
  • Master Data Governance on S/4 HANA on-premise 2021
  • Master Data Governance on S/4 HANA on-premise 2022
  • Master Data Governance on S/4 HANA on-premise 2023

Reproducing the Issue

1. Open File upload functionality in MDG UI

2. Select entity type "Tax Classification for Sales"

3. Select relevant fields and load file

4. Create change request and move to next step

5. The error appears in final step "Check and Execute"

Cause

Program error

Resolution

Please implement SAP Notes below:
3379493 - MDG-M: Tax Classification of a country with multiple tax categories is not updated correctly
3463478 - Getting unexpected 'Check the tax classification data' message in MDG material
