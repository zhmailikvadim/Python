Symptom

You want to split freight unit stages based on a default route, but the corresponding default route is not found.

During the definition of the default route, you have deleted the first or last stop of the default route. The new first or last stop of the default route is not considered during the default route search.

Other Terms

LOC_POS

Reason and Prerequisites

Program error

Solution

Implement the attached correction instructions or install the corresponding Support Package.
