Symptom

IHB account creation is not possible when the opening date is earlier than system date, but later then the posting date of the Bank Area.

Other Terms

IHB, Account, opening date, Bank Area posting date, CheckOpeningDate, validation

Reason and Prerequisites

There is a requirement to create an IHB account whose opening date is earlier than system date, but later then the posting date of the Bank Area.

Solution

During opening date validation, the system compares the date with the Bank Area posting date instead of using the obsolete comparison with the system date.
Please apply SAP Note or assigned Support Package.
