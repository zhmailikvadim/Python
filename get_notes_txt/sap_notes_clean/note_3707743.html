Symptom

The domain PTRV_USEFL has only one position (type Character with length 1). The letters of the Latin alphabet as well as the digits from 0-9 are completely assigned, as are the most common special characters in the German language.
In addition, it may be helpful in the future to be able to automatically assign usage indicators (useflags) to a specific country grouping (MOLGA).

Other Terms

VWKZ, extension, four characters, adjustment, domain, output length, usage indicator for trip expense type, USEFL_APPEND

Reason and Prerequisites

No more free standard characters in the value range of the domain.

Solution

The domain PTRV_USEFL and its fixed value attachment (fixed value append) USEFL_APPEND are extended to four characters with this correction.
Future use stipulates that the first two places of the fixed values describe the country grouping (MOLGA) to which the respective usage indicator is assigned. The rear two digits are then available for further differentiation.

To implement the corrections, import the specified HR Support Package or implement the attached correction instructions.
