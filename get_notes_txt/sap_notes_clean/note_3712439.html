Symptom

Where to find information about /hana/backup storage options in SAP HANA System Replication (HSR)

Environment

SAP HANA PLATFORM EDITION 2.0

Resolution

  • SAP HANA supports file-based backups to NFS-mounted file systems. 1642148 - FAQ: SAP HANA Database Backup & Recovery) - "15. Does SAP HANA Database support file-based backups to NFS-mounted file systems?"
  • In HSR, the backup location should be accessible after takeover and the former primary must stop writing log backups if the same location is used. 2165547 - FAQ: SAP HANA Database Backup & Recovery in an SAP HANA System Replication Landscape - "4.What needs to be considered regarding the backup location?"
  • Shared storage solutions and local disks are infrastructure choices; ensure exclusive access, fencing, and performance per storage vendor best practices.
