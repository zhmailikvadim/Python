Symptom

  • When adding products in guided selling, the name field displays the UUID instead of the product name.

Environment

  • SAP Sales Cloud Version 2

Reproducing the Issue

  1. Open SAP Sales Cloud Version 2 in German language.
  2. Go to Navigation Menu
  3. Go to Guided Selling.
  4. Click on + icon
  5. Go to Products section
  6. Click on Add Products Button
  7. Observe that the name column displays the UUID instead of the product name.

Cause

The Product does not have the German name maintained

Resolution

  1. Go to Navigation Menu
  2. Go to Products
  3. Open the Product
  4. Go to Names Section
  5. Click on Add button
  6. Add a Name and select the Language
  7. Now we will be able to view the Product Name in the given language during Opportunity creation
