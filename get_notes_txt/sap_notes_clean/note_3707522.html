Symptom

When running SUM, at phase MAIN_UPTRANS/INSTTEMPLIC_PUPG, see below error message

4EETQ399 162 wlikey_sign_for_installer: rc of SsfLibSign: 5 = SSF_API_SIGNER_ERRORS = there are signer errors.
4EETQ399 310 wlikey_sign_for_installer: uResult = 23 = SSF_API_UNKNOWN_PROFILE = security profile not found.
1EETQ399 Last generated error message is: wlikey_sign_for_installer failed, see phase log
1EETQ399X'INSTTEMPLICPUPG.LOG' for details!

Environment

SUM

Reproducing the Issue

Running SUM till phase MAIN_UPTRANS/INSTTEMPLIC_PUPG

Cause

Can not install temporary license

Resolution

Follow below actions:

1.exit SUM

2.kill "sapup" process on OS

3.register SUM to hostagent and restart SUM

4.repeat SUM phase

Keywords

MAIN_UPTRANS/INSTTEMPLIC_PUPG, signer errors, security profile not found
