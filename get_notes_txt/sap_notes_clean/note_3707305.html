Symptom

  • An incompatibility issue is identified during readiness checks for upgrading the S/4 system to version 2025.
  • The component SAP S/4HANA Utilities for German Billing Enhancements 2305 is not compatible with the planned upgrade.
  • Documentation indicates that version 2602 is planned for Q1 2026, and clarification is requested regarding its compatibility with S/4HANA 2025 and specific versions (00, 01, or both).

Environment

SAP S/4HANA Utilities for German Billing Enhancements

Reproducing the Issue

  1. Perform readiness checks for upgrading the S/4 system to version 2025.
  2. Identify that the component SAP S/4HANA Utilities for German Billing Enhancements 2305 is flagged as incompatible.
  3. Review the documentation to find that version 2602 is planned for release in Q1 2026.

Cause

Resolution

  1. Refer to the help portal for updates on release dates and related information.
  2. Monitor the portal for finalized updates and relevant details regarding version 2602 compatibility with S/4HANA 2025.
