Symptom

The direct manager is able to edit the dependents in the benefits section of employees despite having view-only permissions related to dependents data.

Environment

  • SAP SuccessFactors HCM Suite
  • SAP SuccessFactors Employee Central

Reproducing the Issue

  1. Proxy as the manager and search for a direct report
  2. Navigate to the benefit section of the employee's profile
  3. Go to Benefits Section > "Manage Family Members"
  4. The 'Edit' button is available and the manager can edit health check and smoking information despite having view-only permissions related to dependents

Cause

Currently, it is not possible to restrict the edit button which gives access to edit the health check and smoking information of the dependents. 

It is only possible to restrict the edition of dependents data from dependents section, such as name and date of birth, for example. 

Resolution

If you would like to have this option added to the system you can open an enhancement request following the article 2090228 - How to submit enhancement ideas for SAP SuccessFactors Products. 

This way other customer can vote and our Product Management Team can consider it for future updates.
