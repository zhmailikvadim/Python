Symptom

Error message: "Invalid Access to a DB Table" (Message Class: edocument000) when saving an invoice in transaction VF01.

Environment

  • SAP ERP
  • SAP S/4HANA
  • SAP Document and Reporting Compliance service

Reproducing the Issue

  1. Create a document in transaction VF01.
  2. Attempt to save the invoice.
  3. Observe the error message "Invalid Access to a DB Table" being triggered.

Cause

The eDocument Class Name for the country DE is not maintained in the EDOCOUNTRYTCH table.

Resolution

  1. If SAP DRC is being used for Germany, ensure that the eDocument Class Name for DE is correctly maintained in the EDOCOUNTRYTCH table.
  2. Save the changes and attempt to save the invoice again in transaction VF01.
  3. Confirm if the error is resolved.
