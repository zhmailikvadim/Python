Symptom

  • The Field Masking functional component of Data Protection and Privacy allows you to fully or partially mask a field’s value when it's presented on the user interface.
  • When a field is defined as masked, the value is always initially presented in its masked format. The user’s security permissions dictate if they can view and/or modify the full field value. 
  • The following topics are discussed in the help documents:
    • Security
    • Where Masking Is Applied in the System
    • Where Masking Isn't Applied in the System

Environment

  • SAP SuccessFactors Sales Performance Management
    • Agent Lifecycle Management

Resolution

This Knowledge Base article provides a link to the official SAP Help documentation for the Agent Lifecycle Management (ALM) Field Masking.
