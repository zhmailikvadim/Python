Symptom

Error "Output Management service unknown exception" occurs when the user tries to send an e-mail from a Sales Quote.

Environment

  • SAP Sales Cloud Version 2

Reproducing the Issue

  1. Go to Sales Quotes workcenter.
  2. Open any Sales Quote.
  3. Click + Create.
  4. Select E-mail.
  5. The error is displayed.

Cause

Missing form template configuration in Output Management Settings for the Sales Quote entity.

Resolution

  1. Open Settings. 
  2. Navigate to Output Management Settings.
  3. Choose Select Form Template.
  4. Open the Sales Quote entity.
  5. Add a rule by clicking the add button (+) and set a value for the templateId column.
  6. Save the rule.
  7. Activate the template using the Activate button.
