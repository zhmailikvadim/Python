Symptom

Component: Product Safety
Module: Global Label Management

You print labels with transaction CBGL_MP01. You change the value for the field Print Immediately using the search help. The system displays the info message 'Value is not applied' (DH864) . The value is still applied when you select Continue.

When you select the initial value (Do Not Print Immediately), the system does not change the description text accordingly.

Other Terms

GLM, GLM+, GLMplus, Immed. Prnt, FLGPRIMM, FLGPRIMMTEXT

Reason and Prerequisites

Reason
Program error

Prerequisites
For information about the validity of the corrections, see the section Correction Instructions.

Solution

After implementing this SAP Note, the system no longer displays the info message. The system shows the correct description text.

The section Support Package specifies the Support Packages that contain the correction.
Alternatively, you can implement the attached correction instructions.
