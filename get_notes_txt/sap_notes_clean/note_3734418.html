Symptom

When a Purchase Requisition (PR) is copied in the Fiori Manage Purchase Requisitions – Professional app, the Document Type field cannot be changed in the new PR.

Environment

  • SAP S/4HANA
  • SAP Fiori
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Open the Fiori Manage Purchase Requisitions – Professional app
  2. Select an existing purchase requisition
  3. Choose Copy
  4. Observe that the document type is copied from the previous PR and cannot be edited in the new requisition.

Cause

System is working as designed.

Resolution

The Manage Purchase Requisitions - Professional app does not allow to edit the Document Type when copying a document if the PR contains at least one item.
