Symptom

  • It is only possible to add objects assigned to $TMP to a transport request with a new package assignment

Other Terms

BW-MT, Modeling Tools, Eclipse, Transport

Reason and Prerequisites

Program Error

Please note this code correction requires the BW Modeling Tools 1.27 PL1 or higher. Earlier frontend versions will not consume the adjusted functionality.

Solution

To solve the issue, apply the following Support Packages:

  • SAP BW/4HANA 2021
    Implement Support Package 14 for SAP BW/4HANA 2021 (SAPK-30014INDW4CORE) into your BW/4HANA system. The Support Package will be available as soon as SAP Note 3625395 with the short text "SAPBWNews SAP BW/4HANA 2021 SP14", which describes this Support Package in more detail, is released for customers.
      
  • SAP BW/4HANA 2023
    Implement Support Package 8 for SAP BW/4HANA 2023 (SAPK-40008INDW4CORE) into your SAP BW/4HANA system. The Support Package will be available as soon as SAP Note 3639252 with the short text "SAPBWNews SAP BW/4HANA 2023 SP08", which describes this Support Package in more detail, is released for customers.

In urgent cases you can use the correction instructions.

Before you use the correction instructions, make sure that you check SAP Note 1668882 for transaction SNOTE.

This SAP Note might already be available before the Support Package is released. In this case, however, the short text still contains the term "preliminary version".
