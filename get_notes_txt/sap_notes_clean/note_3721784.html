Symptom

During a client copy, you can use the profile to select which SAPscript objects (texts, styles, forms) are to be copied. There are two settings here that are relevant for SAPscript objects:

  • Copy of Customizing Data (COPY_MANDT_DATA):
    • All SAPscript texts with text object TEXT are copied.
    • If the source client is not 000, all SAPscript forms and styles are copied.
  • Copy of application data (COPY_APPL_DATA):
    • All SAPscript texts are copied.
    • If the source client is not 000, all SAPscript forms and styles are copied.

If neither of the two settings are set, no SAPscript objects should be copied. Instead, it can happen that all SAPscript objects are copied.

Solution

Implement the correction instructions contained in this SAP Note or import the relevant support package.
