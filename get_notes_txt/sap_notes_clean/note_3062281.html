Symptom

The option "Not Requested" is selected for Empty Provisioning or Empty Return fields in the Calculation Sheet. During Charge Calculation in the Order, the charge lines which has the Containers with the "Empty Provisioning" or "Empty Return" marked as "Requested" does not get determined.

Other Terms

Empty Provisioning, Empty Return, Provision, Return, Requested, Containers, resolution base

Reason and Prerequisites

Program error. This note reverses the changes brought in by note 2784430

Solution

Please implement the correction instructions attached in the SAP Note or upgrade to the latest Support Package.
