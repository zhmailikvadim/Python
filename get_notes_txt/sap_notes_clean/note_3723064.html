Symptom

  • I would like to create a report using external data, but I am not sure if this is possible in SAP Fieldglass.
  • I cannot find an option to upload or integrate external data into the reporting module.

Resolution

SAP Fieldglass reports can only include data fields that exist within the system database.

Only standard fields, calculated fields (formulas), custom fields, or system-configured fields available in SAP Fieldglass can be selected and displayed in reports.

Data that is external to SAP Fieldglass, or information that is not stored as a system field, cannot be added to reports.
