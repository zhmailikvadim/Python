Symptom

How to view the last login time for a user in SAC?

Environment

SAP ANALYTICS CLOUD 2026.2.10

Resolution

  1. Login to SAC
  2. Go to Security > Users 
  3. Click on select column
  4. Select "Last connected on"
  5. Ok
