Symptom

  • You use the Standard Fiori Application F4171– Convert Planned Orders to convert Planned Orders to a Production Order or a Process Order individually or collectively.
  • You have enabled the Segmentation functionality in your system by activating the Segmentation switch SFSW_SEGMENTATION_04. You have materials/articles which are relevant for segmentation.
  • For such materials/articles, when you use the fiori application to convert the Planned Order to a Production or a Process Order, the Segmentation information such as the Stock Segment is not visible in the application
  • With this SAP Note; we have now enabled the stock segment field in the application to be displayed.

Other Terms

F4171,Convert Planned Orders,Convert Planned Orders Fiori Application with segmentation fields

Reason and Prerequisites

Reason: Advanced Development

Solution

The Changes are contained in the tab Correction Instruction of this note. You need to implement this corrections in your system

Note: This enhancement will only be visible if you have switched ON business function ISR_RETAILSYSTEM.
