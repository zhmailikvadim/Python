Symptom

Login with UME administrator and want to make change to user, but the button "modify" is grey out, and not able to make any change.

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Environment

SAP NetWeaver Java 

Reproducing the Issue

  1. Login with AS JAVA administrator
  2. Go to UME
  3. Modify user button is grey out

Cause

Data Source file has marked read only to true.

Resolution

Change the Readonly value from true to false.

  1. Go to UME ->Configuration
  2. Modify Configuration
  3. Download current Data Source File
  4. Check below part:
    <dataSource id="xxx" className="xxx" isReadonly="false" isPrimary="true"> 
  5. Save the change and restart all the server nodes.
