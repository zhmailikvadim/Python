Symptom

The following symptoms occur in connection with the foreign valuation valuation using transaction FPW1M:

  • You move a run from transaction FPW1M using the options "Adjust According to Interval of Run Data" or "Adjust According to Number of Days": the key date is not shifted.
  • You move a run using the report RFKK_MA_SCHEDULER: The key date is not shifted.
  • You move a run with the option "Shift to End of Month": The key date is not shifted to the end of the month.

If you use the "Specify Posting Data" setting in the basic settings for foreign currency valuation, the statements also apply to the fields that can be entered, such as the posting date.

Other Terms

SAPFK100 FPW1M

Reason and Prerequisites

This is a program error.

Solution

After you implement this SAP Note, the symptoms no longer occur.
