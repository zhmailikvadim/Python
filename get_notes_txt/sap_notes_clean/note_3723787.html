Symptom

In certain scenarios, applications request access tokens for SAP Generative AI Hub and require a backend-controlled mechanism to obtain them.

Other Terms

PLM-42194, /DSCSAG/PLMAI_UAA_TOKEN_GET

Reason and Prerequisites

In some system constellations, authorization handling and secure credential usage may require backend execution.
An SM59 destination and access to the XSUAA token endpoint are prerequisites.

Solution

A new RFC-enabled function module /DSCSAG/PLMAI_UAA_TOKEN_GET was added.
Authorization handling was introduced.
Token retrieval via an SM59 HTTP destination was enabled.
