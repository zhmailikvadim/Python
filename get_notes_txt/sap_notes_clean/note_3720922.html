Symptom

  • The system does not allow updating the address in SAP Transportation Connection Points.

Environment

  • SAP ERP
  • SAP ERP Central Component
  • SAP R/3
  • SAP R/3 Enterprise
  • SAP enhancement package for SAP CRM, version for SAP HANA
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Navigate to Logistics Execution → Routes → Define Routes → Define Transportation Connection Points.
  2. Search for the relevant address entry.
  3. Update the address
  4. Save the changes.

Cause

The affected path is only relevant to define a route and not for updating the address

Resolution

  1. Assign the address maintained in table ADRC to a Business Partner (XD02 / XK02 / BP).
  2. Run Transaction Code: SM30 or SM31.
  3. Enter Table/View: V_T5UZC (for US ZIP Code validation).
  4. Click "Maintain".
  5. Add new entries for the Country Key, State, and the ZIP Code range.
  6. Save the changes.
