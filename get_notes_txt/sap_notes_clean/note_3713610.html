Symptom

  • Request to enable email notifications for system maintenances or upgrades.
  • Uncertainty regarding the location or option to enable these notifications.

Environment

  • SAP Cloud for Customer core applications
  • Integration with SAP CRM and enhancement package for SAP CRM
  • Integration with SAP ERP and enhancement package for SAP ERP

Reproducing the Issue

  1. Attempt to locate an option to enable email notifications for system maintenances/upgrades within SAP Cloud for Customer core applications.
  2. Observe that the option is not readily identifiable.

Cause

Notifications for applications and systems maintenances/upgrades are only sent to IT contacts.

Resolution

  1. Verify that the intended recipient is listed as an IT contact within the system.
  2. Ensure the IT contact information is correctly configured in SAP Cloud for Customer core applications.
