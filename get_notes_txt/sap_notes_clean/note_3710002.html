Symptom

When a sales document is created and the sold-to party is entered, the proposal search is not triggered. This issue also occurs when entering the ship-to party.

Environment

  • Sales And Distribution (SD)
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

In transaction VA01 or VA02, when the cursor is placed in the Sold-to Party field and the customer name is directly typed, the proposal search is not triggered.

Resolution

  1. In SE11, select the search help SD_DEBI and then choose DEBIA from the included search helps.
  2. Verify whether DEBIA is active in the system. If it is not active, please activate the DEBIA search help.
  3. Under the Definition tab, in the Enhanced Options section, ensure that the fields "Autosuggest in Input Fields" and "Multi-Column Full Text Search (Database Specific)" are enabled.
