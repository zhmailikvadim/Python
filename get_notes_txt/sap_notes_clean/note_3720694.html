Symptom

When you execute the report ELStAM Procedure: Update Master Data Using ELStAM  Procedure (RPCE2VD0_IN), the log table Infotype 0012 Tax with the changes in the infotype Fiscal Data D (0012) due to the processing of a notification is incorrect. The special rules are always displayed as initial in the line with the new IT12 data if the special rule is not L (new vesting period).

Solution

Implement the attached correction instructions or import the specified support package. After you import the HR Support Package or implement the correction instructions, you do not have to perform any further follow-on activities.

After you implement the corrections, the logging is corrected.
