Symptom

When you navigate from Cash Flow Analyzer to Display Cash Flow Item list, with Liquidity Item Name or Liquidity Item Description column without Liquidity Item column, you notice a difference between header amount and total amount in display cash flow item screen.

Other Terms

F2332

Reason and Prerequisites

Limitation. If only the column Liquidity Item Description or Liquidity Item Name is selected, the related Liquidity item will not be transmitted to detail page and the issue would occur. For the Liquidity Item value to be transmitted to CFA detail page, there are two following options:

- The column for Liquidity Item is part of the table result (along with Liquidity Item Name or Liquidity Item Description)

- The Liquidity Item is inputted in CFA filter field (without necessarily having selected Liquidity Item column)

Solution

For consistent results, it is either required to select Liquidity Item column along with Liquidity Item Name or Liquidity Item Description columns or to enter a value in the Liquidity Item filter of CFA.
