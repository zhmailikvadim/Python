Symptom

 Testing OLAP connection with a name containing brackets [ ] throws the error "RWS 00070: Internal server error (RWS 00070)"

Other Terms

BILP, BI, Source, BEX, SAP, BW, WEB INTELLIGENCE, Refresh, Query, CMC

Reason and Prerequisites

Environment

SAP BusinessObjects Business Intelligence Platform 2025 Patch 7

Reproducing the issue 

  1. Log in to CMC using a user account whose name contains brackets [ ]
  2. Navigate to Connections
  3. Select the connection and click on Edit 
  4. In the Authentication Settings section, select Use Single Sign-On (SSO) as the authentication mode
  5. Click on Test Connection and observe the following error message:RWS 00070: Internal server error (RWS 00070)
