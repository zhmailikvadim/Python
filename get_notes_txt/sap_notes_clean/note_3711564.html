Symptom

For Fiori Applications 'Asset Transactions' and 'Asset Depreciation Lists', the value of Acquisition Year and Acquisition Month is empty, this issue is corrected. And now below fields are avaliable:

  • First Acquisition On( Asset Value Date of the First Posting )
  • FY 1st Acquisition( Fiscal Year in Which First Acquisition was Posted )
  • Per. 1st Acquisition( Period in Which First Acquisition was Posted )
  • Capitalized On( Asset Capitalization Date )

Other Terms

Acquisition Year, Acquisition Month, Asset Transactions, Asset Depreciation Lists

Reason and Prerequisites

Program errors.

Solution

As a general rule, SAP recommends that you install a solution by applying a Support Package. However, if you need to install a solution earlier, use the Note Assistant to implement the correction instruction. You can find more information about the Note Assistant in SAP Service Marketplace, under service.sap.com/note-assistant.
