Symptom

 My Access Approvals FIORI App Enhancements.

Other Terms

My Access Approvals, Access Control 2026, Access Request Approvals.

Reason and Prerequisites

Following Features are enabeld in My Access Approvals FIORI App.

1.Line Item Level Comment.

2.Forward on Request Level & Line Item Level.

3.Request Reject, Hold, Release & Return-To.

4.Mitigation Assignment & Creation.

5.Admin Settings : User Parameter & User Group.

6.Add/Remove Roles,System & Existing Assignments.

Prerequisite: UISAPGRC - SP00 Software Component.

Solution

Please apply this SAP Note in FIORI System.  Backend Service SAP Note 3722799 should be implemented in GRC Box.
