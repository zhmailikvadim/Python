Symptom

  • Discrepancy observed between SAP Enable Now usage and the SAP for Me Customer Insights Dashboard.
  • Manager > Reports > Server Statistics display a different amount of active users than the Customer Insights Dashboard for SEN.

Environment

  • SAP Enable Now - Manager
  • SAP for Me Customer Insights Dashboard

Cause

Resolution

The discrepancy in the reports was caused by the migration of SAP Enable Now from the Neo platform to the Kyma platform in 2025. This platform change temporarily impacted reporting data. The SAP Enable Now development team has addressed and resolved this issue, so it should not occur again in the future.
