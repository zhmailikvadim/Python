Symptom

The processing of migration processing units for BOPF-Based-Master-Data or System-and-User-Settings is running into a system exception GETWA_NOT_ASSIGNED, if the TM Migration Tool was executed with processing in parallel tasks in SAP S/4HANA.

Other Terms

GV_MIGR_OBJECT; bo_process_unit

Reason and Prerequisites

Due to a program error the migration object name is not correctly passed over into processes for parallel processing in SAP S/4HANA. As consequence, the processing units cannot be processed, and the system is running into runtime error GETWA_NOT_ASSIGNED.

The problem does not occur for Customizing or for BAPI-Based-Master-Data, or in case of sequential processing in SAP S/4HANA.

Solution

Implement the attached Correction Instruction (CI) or the relevant Support Package.
