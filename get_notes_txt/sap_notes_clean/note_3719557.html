Symptom

'Journal Entry' not appearing in the Fiori Launchpad Search dropdown.

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental." 

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Use the global search bar.
  2. Enter a valid Journal Entry or Invoice number.
  3. No results are returned.

Cause

Issue  on upgrade 

Resolution

Temporary Workaround:

Use app Manage Journal Entries - New Version (App ID F0717A) to search for the document number. 

Solution: HFC05, planned release date 26.02.2026.
