Symptom

When trying to use the Generator 'OData Web API Service' in Eclipse ADT on custom tables, the error "Package does not exist." appears.

Environment

  • ABAP platform
  • RAP - OData Design Time, Draft, Event Enablement

Reproducing the Issue

  1. In Eclipse/ADT, start Generate ABAP Repository Objects.
  2. Select the generator "OData Web API Service".
  3. Choose a custom table as the referenced object.
  4. Run the generation and observe the error "Package does not exist.".

Cause

A validation problem in the OData Web API Service generator (referenced object validations not executed) in releases prior to SAP_BASIS 758 SP02.

Resolution

  • Method 1:
    • Implement SAP Note 3479745 in the ABAP system to correct the validation issue.

  • Method 2:
    1. Alternatively, upgrade the system to SAP_BASIS 758 SP02 or higher where the problem is fixed.
    2. After implementing the note or upgrading, rerun the OData Web API Service generator.

    See Also

    SAP Note 3479745 - Referenced object validations not executed
