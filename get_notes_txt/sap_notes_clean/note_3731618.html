Symptom

You are working with the Web Dynpro API /SCWM/MFG_CONSUMPTION. Your PMR document contains backflush component items also, not just the standard components, so the API will show you all components to consume despite the backflush components can not be consumed manually. These component items are unnecessary, can not be consumed, therefore they have to be removed from the list.

Other Terms

backflush component, backflush, PMR, PCSO

Reason and Prerequisites

Program error

Solution

Implement the correction instruction or the equivalent support package.
