Symptom

In the Fiori application, item-level functions are currently available in the Calculation tab. However, these functions are not relevant in this context and can lead to user confusion during processing.

To improve usability and avoid incorrect actions, the item functions are removed from the Calculation tab. This ensures a more intuitive user experience by displaying only the relevant options for calculation-related activities.

Other Terms

calculation tab, Fiori, Reprice, Reprocess, /IRM/GCRM, function removal

Reason and Prerequisites

This is caused by program error

SAP Note#3719329 is available for V4 Release

Solution

Please implement the correction instructions attached. These corrections are only valid if you have applied the Support Packs.

Release 6.2 - Base Package
