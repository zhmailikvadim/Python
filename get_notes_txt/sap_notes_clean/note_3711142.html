Symptom

When Device & Register data for Installation is being read with Key date, it returns installation which is not valid on given key date.

Other Terms

APEU, Device, EASTL, AB, BIS, Key Date, GET_DEVICE_REGISTER_DATA, ISU_DB_EASTL_FORALL_ANLAGE

Reason and Prerequisites

APEU, 2508

Solution

Please implement the attached correction instruction.
Key-date handling was corrected by explicitly, when calling ISU_DB_EASTL_FORALL_ANLAGE, ensuring that only devices valid on the specified key date are selected.
