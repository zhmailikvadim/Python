Symptom

When you create a new financing in the FIN dialog, no check table is defined in the "Contract Unit" field. As a result, you can enter values that are not contained in the Customizing table /IBS/CBSC_CMU.

Reason and Prerequisites

The error message for this problem can be avoided and the process then runs with an incorrect value.

Solution

The problem has been solved by adding the check table /IBS/CBSC_CMU to the structures in which the contract unit is saved.
