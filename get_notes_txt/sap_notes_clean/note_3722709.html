Symptom

You display the upload logs of the preference rules and notice that the "Version Number" column is empty in the overview of the logs. However, the version number is displayed in the log itself.

In addition, the columns Number and Packages are displayed that are not required in connection with the upload of the preference rules.

Other Terms

/SAPSLL/PREF_RULES_UPL, /SAPSLL/SHOW_UPLOAD_PROTOCOL, /SAPSLL/PREF_RLS_UPM.

Reason and Prerequisites

Program error

Solution

Implement the attached correction instructions or import the specified support package.

After you implement this SAP Note, the version number is saved for preference rules uploaded in the future (transaction /SAPSLL/PREF_RLS_UPL) and is therefore also displayed in the overview of the logs. The two columns Number and Packages are no longer displayed.
