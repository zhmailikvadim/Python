Symptom

I have added a measure field in my SAP Ariba Spend Analysis report but I am not able to apply a filter on it.

Resolution

There is no option to apply a filter on measure fields in SAP Ariba Spend Analysis reports.

Currently we only can sort measure fields in ascending or descending order. Click on the measure field you can see only the following option:

Sort this column:

  • Ascending
  • Descending

It would be considered as an Enhancement Request having filters on measure fields in SAP Ariba Spend Analysis reports.

See Also

For further information regarding reporting filters, please check SAP documentation: SAP Ariba Procurement Solutions > Running Analytical Reports > Report Filtering and Navigation.

If you feel having that functionality is essential for your business, please submit an Enhancement Request.
