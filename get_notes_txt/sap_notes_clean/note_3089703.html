Symptom

You are using SAP Solution Manager Incident Management in SAP Solution Manager 7.2.

In the CRM UI, you choose "Manage Substitute". The substitution popup opens, but you don't see the Add button for "Business Partner for Whom I Substitute".

Other Terms

Substitution, Authority-Check, 'B_BUPA_RLT', cl_ai_crm_bp_helper=>chek_subst_auth, CL_AIC_IM_S_VMYSUBTITUTIO_IMPL --> IF_BSP_WD_DYN_BTN_CONTROL~GET_LOCAL_TOOLBAR_BUTTONS

Reason and Prerequisites

The authority check for the authorization object 'B_BUPA_RLT' fails. Ideally, there is no need to select any specific business role if the user has the authorization to substitute.

Solution

Implement the attached corrections to your SAP Solution Manager system by using the Note Assistant (transaction SNOTE).
