Symptom

Customers, using the Holiday Pay calculation (schema BHLP) in the Payroll Belgium (RPCALCB0) in the payroll schema B000, should not execute one of the calculation rules B01A or B01B in payroll schema BT00.

The execution of one of the calculation rules B01A or B01B in payroll schema BT00, is linked to the use of Holiday Pay calculation (schema BHOL). This is the old Holiday pay calculation method and is no longer supported. For this reason calculation rules B01A and B01B should no longer be processed in payroll subschema BT00.

Other Terms

RPCALCB0, /861, B000, BT00, BHLP, BHOL

Reason and Prerequisites

You are using Payroll localization Belgium.

Solution

  • As a general rule SAP recommends that you implement the Support Package specified for your release.
  • However, if you need to install a solution earlier, use the SAP Note Assistant to implement the correction instruction.
  • You can find more information about the SAP Note Assistant in SAP Support Portal, under SAP Note Assistant.
