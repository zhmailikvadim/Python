Symptom

The Preferred Item flag is not visible in SAP Ariba Guided Buying, however it appears as expected in the Ariba Spend Management View. The flag either does not appear in search results or is not shown when opening an item detail page.

Cause

The flag is displayed only in the Enhanced Product Detail GB View. In other views, the flag remains hidden because the required configuration parameters are not enabled. Without these parameters turned on, the system cannot display the Preferred Item indicator, regardless of how the items are catalogued or tagged.

Resolution

Enable the following parameters based on the desired display behavior:

  • To display the Preferred Item flag in Search results:
    Enable the two dedicated parameters that control the visibility of the Preferred Item indicator within the Guided Buying search experience.
    • PARAM_ENABLE_IMPROVED_SEARCH_EXPERIENCE
    • PARAM_ENABLE_IMPROVED_SEARCH_EXPERIENCE_2

  • To display the Preferred Item flag on the Item Detail page (when opening an item in Guided Buying):
    Enable the additional parameter that controls the flag visibility at the item detail level.
    • PARAM_ENHANCED_PRODUCT_DETAIL_VIEW

All parameters must be configured at the appropriate site/realm level. After enabling, allow time for the configuration to propagate before testing.
