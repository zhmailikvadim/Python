Symptom

Error message KE007 appears when regenerating operating concern for client-specific objects.

KE007 - Operating concern XXXX cannot be used in client CCC.

Additional error message in Action Log can be observed:

KH022 - Entry 2001000 901201 is already contained in Table T237.

Environment

  • Controlling and Profitability Analysis
  • SAP S/4HANA Finance
  • SAP S/4HANA

Reproducing the Issue

  1. Access transaction KEA0.
  2. Select Operating concern XXXX.
  3. Navigate to the Environment tab and click Activate button for the Client-specific part.
  4. Observe the occurrence of error message KE007.

Cause

The root cause for the problem is an unsorted table for building the initial entries for the key figure scheme.

Resolution

  1. Implement SAP Note 3690425 - KEA0: Message KH022 when generating the environment.
  2. Follow the process for regenerating the operating concern as described in the SAP Help Portal page: Regenerating an Operating Concern
