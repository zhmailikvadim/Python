Symptom

Hierarchy variable is not applied to the characteristic within the Analysis for Office workbook even if the hierarchy has already been selected for the variable. The hierarchy is displayed as a flat list instead of being applied correctly.

Environment

  • SAP Analysis for Microsoft Office
  • SAP Analysis for Microsoft Office, edition for SAP Analytics Cloud

Reproducing the Issue

  1. Open the Analysis for Office
  2. Execute the affected query
  3. Set the variables in the query
  4. Observe the affected hierarchy is displayed as a flat list instead of being applied correctly.

Cause

Program error.

Resolution

Implement the SAP note below in the affected system.

3708502 - Wiederöffnen einer Arbeitsmappe: Hierarchie ist unerwarteter Weise aktiv
