Symptom

  • For several CDS views in Customer and Internal Project Management, a switch to other CDS views is recommended.
  • This is published in the “What’s New in SAP S/4HANA Cloud Public Edition 2508” update.
  • However, certain fields are missing in the alternative CDS views. For example:
  • Fiscal Year - which exists in the original CDS View I_ResourceAssignmentPlanItem - is missing from all alternative views like e.g. I_ProjectDemand.
  • This creates concern about the practical value of the new versions following deprecation.

Environment

S/4HANA Cloud Public Edition

Cause

  • The new views are built on the PROJECTDEMAND table, which is a common table used for both CPM and Enterprise Projects.
  • The old view are based on /CPD/PLANDATA, which is specific to CPM projects only.

Resolution

  • The original views are not depreciated.
  • Continue using the original CDS views for CPM Projects, if preferred.
  • To use the alternative views, no data migration is required.
  • Old and alternative views will be in sync (store the same data, however the new CDS views are more granular).
  • As new applications are built over new models (which is harmonized for all project types) it is recommending to move to new models.
