Symptom

While trying to execute in the BW/4HANA conversion cockpit the step to align with the task list run the following issue occurs:
Connection closed (no data)

Environment

SAP NetWeaver

Cause

  • This error means that there is an issue with the RFC connection between sender and receiver even if the activity "Define RFC Destinations" in Package Settings has been executed successfully. 
  • This can also happen because the tool checks the connection when executing an activity, but the status of the activity "Define RFC Destinations" is not modified. 
  • Sometimes it occurs due to out of sync between BW/4HANA conversion cockpit tool migration process (through transaction RSB4HCONV) and task list monitor (transaction STC02) during BW/4HANA conversion.

Resolution

Rerun activity "Define RFC Destinations" in Package Settings and make sure the connection is working.

Keywords

RSB4HCONV, STC02, SAP_BW4_TRANSFER_REMOTE_PREPARE, BW/4HANA conversion cockpit, task list monitor, out of sync, 3.x objects warning, cannot proceed, short dump, memory increase, heat memory, remote conversion, BW-B4H-CNV
