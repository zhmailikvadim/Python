Symptom

  • The issue pertains to the printing of documents in sales order.
  • The standard SAP print program correctly populates the overall total value, but the individual line item price details are not being printed.
  • The issue affects the outputs for the following documents and following program:
    • Quotation : SD_SDOC_PRINT01, SD_SDOC_FORM01
    • Contract : SD_SDOC_PRINT01, SD_SDOC_FORM01
    • Standard Order : SD_SDOC_PRINT01, SD_SDOC_FORM01
    • Invoice : SD_INVOICE_PRINT01, SD_INVOICE_FORM01

Environment

  • Sales And Distribution (SD)
  • SAP ERP Central Component
  • SAP ERP
  • SAP Enhancement package for SAP ERP
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. check the preview of output in sales order
  2. the overall total value populates correctly, but the individual line item price details are not being printed (being printed as 0).

Cause

  • The 'print type' is not set for the relevant pricing procedure in T-code V/08.

Resolution

  1. Access T-code V/08 in the SAP system.
  2. Navigate to the relevant pricing procedure settings.
  3. Set the 'print type' for the pricing procedure.
  4. Save the changes.
  5. Retest the behavior to confirm the resolution.

Keywords
