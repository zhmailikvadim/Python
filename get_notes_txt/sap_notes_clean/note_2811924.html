Symptom

Use DSConnectionManager.sh to edit a DSN, When changing the port the following error is returned:

Error: Input must match the pattern: ^(3\d{2}15|3\d{2} [4-9] [0-9]) 

Environment

  • SAP Data Services 14.2.x
  • Linux

Reproducing the Issue

  1. Run DSConnectionManager.sh
  2. Edit an existing database source
  3. Select the source to ammend
  4. Specify the User Name
  5. Specify the Password
  6. Specify the Driver
  7. Specify the Host Name
  8. Specify the Port

Cause

The Incorrect port is being entered.

Resolution

Confirm the correct port your Database is listening on and configure the DSN with this port.
