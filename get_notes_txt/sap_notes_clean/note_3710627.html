Symptom

ERP vendor ID is not updated in supplier 360 profile, even after sending KeyMappings from S4-Hana Public Cloud using MDI Integration(Master Data Integration).

Reproducing the Issue

  1. Create a supplier in Ariba.
  2. request_in is triggered 
  3. received a confirmation_out and keyMapping payloads on Ariba

Cause

MDI parameters are not enabled. 

  • Application.SM.BPKM.KeyMappingTypeCodeForBP

  • Application.SM.BPKM.KeyMappingObjectSchemaCodeForBP

  • Application.SM.BPKM.KeyMappingObjectSchemaCodeForBPUuid

  • Application.SM.BPKM.KeyMappingTypeCodeForSup

  • Application.SM.BPKM.KeyMappingObjectSchemaCodeForSup

  • Application.SM.BPKM.KeyMappingEnabled

  • Application.SM.MDI.MdiPrimaryBpId

  • Application.SM.MDI.MdiEnabled

Resolution

Enable the self-configuration MDI parameters. 

Follow the link below to configure the parameters. 

If the parameters are not visible, please reach to support to enable SM toggle "Enable MDI integration".
