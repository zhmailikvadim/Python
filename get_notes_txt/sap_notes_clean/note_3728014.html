Symptom

When maintaining IT0007 via PA30, the 'WSR Finder' button is not visible on the screen. "Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Environment

  • Human Capital Management (HCM)
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for HANA
  • SAP S/4HANA
  • SAP SuccessFactors Employee Central Payroll

Reproducing the Issue

  1. Open T-code PA30.
  2. Input a personnel number and create IT0007.
  3. On the screen, the button 'WSR Finder' button is not visible.

Cause

This is a configuration issue.

Resolution

  1. Open T-code SM30 and go to V_T588M.
  2. Select the corresponding entry of MP000700 based on the return value of feature P0007.
  3. Set the BUTTON_WSR_FIND to an optional field.
  4. Then the WSR Finder shows in the IT0007 screen.

Keywords

WSR Finder, button, BUTTON_WSR_FIND, T588M, P0007, MP000700.
