Symptom

  • Document Input Control display as Report Input control in Web Intelligence when using a variable.
  • Issue is only when the input control is based on a variable, the definition of which is a constant (Eg: =” ” or =”test”).

Other Terms

WebI

Reason and Prerequisites

Environment:

  • SAP BusinessObjects Business Intelligence Platform 4.3 SP02 Patch 03
  • Web Intelligence (Webi)

Steps to Reproduce:

  1. Open any existing Web Intelligence Document from the environment.
  2. Switch to Design Mode.
  3. Create a new variable with =” ” formula , Name it and save.
  4. Click on the Input Controls => New => Select the Variable.
  5. Set the current target as Document.
  6. Click on OK
    >> the Input control is created as Report Input control instead of Document Input Control.

Cause:

  • This issue is caused by a defect in the product.

Solution

  • This issue is fixed in the patches listed in the "Support Packages & Patches" section below.

    Due to different internal processes, the "Support Packages & Patches" section of the SAP Note will be populated as below:

    • for fixes delivered in "Support Packages": immediately (even if the "Support Package" is planned to be released several months later)
    • for fixes delivered in "Patches": ONLY once the "Patch" is released AND available on SMP (not before)
