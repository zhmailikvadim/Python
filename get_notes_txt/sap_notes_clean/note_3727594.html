Symptom

It is not possible to edit the Posting Date before posting to the Posting Area 1 on a bank statement item in the status Posting Not Started, or it is not possible to change the Fiscal Period before posting to the Posting Area 2.

Other Terms

F1520, BUDAT, MONAT

Reason and Prerequisites

Attempt to initiate the posting to the Posting Area 1 in the application Reprocess Bank Statement Items, or attempt to post in a specific Fiscal Period in the Posting Area 2.

This SAP Note requires another SAP Note "3724967 - Reprocess Bank Statement Items - Editing the Posting Date before triggering posting in the Posting Area 1 BE" to be implemented on the backend server.

Solution

This SAP Note introduces a new field Posting Date in the dialog window Open Item on the worklist in the application Reprocess Bank Statement Items, and a new field Fiscal Period on the detail screen of a bank statement item.

As a general rule, SAP recommends that you install a solution by applying a Support Package. However, if you need to install a solution earlier, use the Note Assistant to implement the correction instruction. You can find more information about the Note Assistant in SAP Service Marketplace, under service.sap.com/note-assistant.
