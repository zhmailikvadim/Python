Symptom

  • Navigate to Security Center
  • When clicking X509 Certificate a pop up error will show up.

  • Following this error, the X509 Certificate tab will become disabled.

Environment

SAP Successfactors HCM Suite

Reproducing the Issue

  1. Navigate to security center.
  2. Click on the "X509 Certificates" tab.
  3. Error message appears.

Cause

The user does not have the required "Admin access to MDF OData API" permission assigned to their role.

Resolution

  1. Navigate to `Admin Center -> Manage Permission Roles`.
  2. Select the role assigned to the affected user.
  3. Assign the `Metadata Framework > Admin access to MDF OData API` permission to the user.
  4. Save the changes and verify if the user can access the "X509 Certificates" tab without encountering the error.

See Also

Refer to: KBA KB0858164 - Admin with no RBP Permission can Access the Application Security Feature
