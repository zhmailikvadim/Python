Symptom

Receiving error message no. M7124 - 'Selection is incomplete due to missing authorization' when trying to execute MI24

Environment

  • Materials Management (MM)
  • SAP S/4HANA

Reproducing the Issue

  1. Go to MI24
  2. Enter required details
  3. Execute
  4. Error message appears

Cause

In optimized S/4HANA reporting, there is no direct authorization check performed in ABAP.

Instead, the authorization object is used to build a positive WHERE condition directly in the SQL SELECT statement to retrieve only the data the user has authorization for.

Resolution

Assign the following authorization objects to the business roles:

  • M_ISEG_WDB with ACTVT = '03'.
  • M_ISEG_WDB with ACTVT = '04'.

Keywords

MI24, missing authorization, M7124, M_ISEG_WDB
