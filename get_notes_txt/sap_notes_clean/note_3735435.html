Symptom

The time duration is displayed differently on the Overview/library page and in the Course/item Details page within SAP SuccessFactors Learning.

Examples:

  • Overview page (in library): displays the duration in minutes (24 minutes)

  • Item detail page:  displays the duration as 0.40h, which is not a common or user-friendly time notation.

Expected Behavior:
The time format should be consistent across both pages and displayed in a standard, readable format.

Environment

SAP SuccessFactors Learning

Reproducing the Issue

  1. Access Learning Administration > Learning Activities > Items > view the item details of item 
  2. Duration field is 0.40 hours
  3. Return to Home Page > Access My Learning
  4. Browse Library and search for the item
  5. From the preview of the item, the duration is 24 m

Cause

The item duration displayed in Library View is dynamic, and it changes depending on the duration of the item, whereas in the item details page, duration is always displayed in hours.

Resolution

The item duration displayed in Library View is dynamic, and it changes depending on the duration of the item, whereas in the item details page, duration is always displayed in hours.

In above example, the duration in the item details page is 0.40 hours, which translates to 24 minutes in the library view.

There is currently no configuration to change this behavior and hence, it cannot be changed.

If you require this to be implemented in the future road map of the application in the admin side, you could raise this in the community portal via a new thread/query for an enhancement.

Enhancement: You could always raise this as an enhancement request in the Customer community portal, and with business reasons, our engineering team would look into this request based on the Kudos that it garners, along with the reason for the request, and they might implement. This usually takes time. Check KBA 2090228 that explains this.
