Symptom

Information regarding APIs for SAP Cloud Connector (SCC).

Environment

SAP Cloud Connector On-Demand/On-Premise Connectivity

Resolution

SAP Cloud Connector includes a collection of APIs that allow users to read various types of monitoring data. Refer to the official SAP guide for a complete list of available APIs, details on their usage, response properties, and example requests at: SAP Cloud Connector Monitoring APIs Guide.

See Also

Help Portal -  Monitoring APIs.
