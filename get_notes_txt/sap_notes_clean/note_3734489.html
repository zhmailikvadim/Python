Symptom

This article describes functionality introduced in the May 2026 SAP Fieldglass release

Customers reviewing a Statement of Work in SAP Fieldglass see expand and collapse arrows on some rows in the Characteristics tab and want to understand why:

  • Only some rows can be collapsed
  • Collapsed rows expand again after refreshing the page
  • Sorting changes the order of rows
  • Expand and collapse arrows are not visible in some views
  • There is no indication showing how many items were hidden

Reproducing the Issue

This behavior may occur when:

  • Only some rows in the Statement of Work Characteristics tab display expand or collapse arrows
  • Refreshing the page causes all rows to expand again
  • Sorting the list changes the order of rows at the same level
  • Expand and collapse arrows are not visible in certain views
  • Collapsing a group does not show how many rows were hidden

Cause

This behavior is part of an enhancement in SAP Fieldglass designed to organize Statement of Work Characteristics using a structured hierarchy:

  • Only rows that belong to a defined hierarchical structure can be expanded or collapsed
  • Refreshing the page resets the view so all rows are expanded
  • Sorting rearranges rows at the same level while keeping child rows grouped under their parent
  • Expand and collapse controls appear only in the horizontal view of the Characteristics tab
  • Hidden item indicators may vary depending on the environment

Resolution

The following behavior is expected when using expand and collapse groups in SAP Fieldglass:

  1. Rows that can be collapsed
    • Only rows that are part of a defined hierarchical structure display expand and collapse arrows

  2. View resets after refresh
    • Refreshing the page expands all rows, which is expected behavior

  3. Sorting behavior
    • Sorting may reorder rows at the same level, but child rows always remain grouped under their parent

  4. Availability of expand and collapse
    • Expand and collapse controls appear only in the horizontal view of the Characteristics tab

  5. Hidden item indicators
    • Depending on your environment, SAP Fieldglass may or may not display how many items are hidden when a group is collapsed
