Symptom

When we are performing the mass processing for custom fields/standard fields, the changes are not shown in 'Show changes' or 'show all changes' after activation of the process and the change documents are empty.

Environment

  • SAP Master Data Governance-Consolidation and Mass Processing

Reproducing the Issue

  1. Launch Fiori Launchpad
  2. Select Manage Mass Process app
  3. Create a process
  4. upload a file
  5. Continue till activation step
  6. Click Show changes
  7. Changes are empty 

Resolution

  • Ensure the Data Element of that field has change document enabled in SE11 transaction for change documents to be captured for those fields.

See Also

  • Review SAP Note 3253663 - "Show Changes" button or "Track Mass Changes" application doesn't show any data when CMP activation step is completed. 

Keywords

Show Changes, Track Mass Changes, data element, change document, show all changes, MDC, CMP, mass process, consolidation
