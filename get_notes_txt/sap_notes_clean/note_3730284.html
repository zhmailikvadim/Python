Symptom

You are using migration workbench functionality to calculate access plans for a migration object. You are using the logical operator "NLIK" for the static delimitation of an access plan. However, the following error occurs in the Migration Workbench (transaction MWB):

"An error has occurred while parsing a dynamic entry"

This issue is caused by an SAPSQL_PARSE_ERROR runtime error. You can view this information in transaction ST22.

Other Terms

MWB, DMIS, DTL, access plan calculation, SAPSQL_PARSE_ERROR

Reason and Prerequisites

MWB / DMIS functionality is used in one of the following releases:

  • DMIS 2011 SP25 - SP28
  • DMIS 2018 SP11 - SP14
  • DMIS 2020 SP07 - SP10
  • DMIS 2025 SP00
  • SAP S/4HANA 2023 FPS00 - SPS05
  • SAP S/4HANA 2025 FPS00 - FPS01

Solution

Implement the attached corrections using transaction SNOTE.
