Symptom

The ViZi OPER report is generating an SQL Syntax Error for fields that do not have corresponding source fields maintained in the data source.

Specifically, when a field is defined as a Currency Key field without an assigned source field, the system constructs an invalid SQL query during runtime. This results in an incorrectly generated query statement, ultimately causing a query execution failure.

Other Terms

OPER, Currency key, Query

Reason and Prerequisites

This is cuased by program error.

If you are in 2023 V2, Please implement the latest version of SAP Note #3457178 as pre requisite note.

Solution

Please implement the correction instructions attached. These corrections are only valid if you have applied the Support Packs.


Release V4 - 2023 V2, 2023 V3 Base Package or Support Pack 1.

Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object.
