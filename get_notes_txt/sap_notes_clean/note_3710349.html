Cause

SAP Note 3499462 does not contain a data declaration for

Class:     CL_QM_INSPUSGDSC_MANAG_DPC_EXT
Method:    /IWBEP/IF_MGW_APPL_SRV_RUNTIME~EXECUTE_ACTION.

Resolution

  1. Ensure that SAP Note 3499462 is completely de-implement from the affected system.
  2. Implement SAP Note 3636751 into the affected system (contains the data declaration).
  3. After step 2. is completely finished, implement SAP Note 3499462.
