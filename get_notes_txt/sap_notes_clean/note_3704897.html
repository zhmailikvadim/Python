Symptom

In the process "Receipt of Delivery Note for Grid Usage Billing (SUP View)" (DE_B_GUB_SUP_DELV_NOTE), steps 0130-0030 searched for the profile value only for the entire delivery note period instead of monthly. As a result, the monthly profile value was not fetched and filled into the UBD (Utility Billing Document).

This issue is relevant for the market role: SUP.

Other Terms

Billing Enhancement, Delivery Note, Basic Price, Profile Value, Calculate Energy Quantity, /US4G/CL_PRST_DTD_DELV_NOTE, ENHANCE_STEP_DATA, /US4G/CL_DTD_CHK_E_0456_DATA, ADJUST_CALC_ENERGY_QTY, UBD, CALC_ENERGY_QUANTITY_DTD

Reason and Prerequisites

Program error.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the system fetches the monthly profile value and fills it into the UBD. You need to receive the business message from the market communication system again.

You can verify the fix by checking that the monthly profile value is fetched and that the field CALC_ENERGY_QUANTITY_DTD in the UBD contains a value.
