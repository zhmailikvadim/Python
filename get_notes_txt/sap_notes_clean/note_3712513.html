Symptom

For a specific plant, the Subcontract PO type is flagged with Activate Immediate Assignment, but the supply assignment does not occur. Users must manually assign physical stock to the Subcontract PO via ME2ON.

Environment

  • S/4HANA Retail & Fashion Management (LO-RFM) 
  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Cause

This function is not supported during immediate assignment.

Resolution

Supply assignment of subcontract PO components to future stock is only possible via the MSAD app and the Supply Assignment API.

Keywords

supply assignment, immediate assignment, MSAD, subcontract PO, API, ME20N, stock, physical stock
