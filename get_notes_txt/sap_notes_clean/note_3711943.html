Symptom

After an upgrade to S/4HANA 2023 (SAP_UI 758), users of the Classic My Inbox (App ID: F0862 / Technical Component: SAP_CA_FIORI_INBOX) experience the following issues:

  • Clicking the Open Task button for workflow tasks (e.g., SAP_WFRT) triggers no action or navigation.

  • The issue occurs "silently" with no visible error message on the UI.

  • The browser console reveals a specific JavaScript TypeError: Uncaught TypeError: sap.ui.getCore(...).getConfiguration(...).getStatistics is not a function

  • Other deprecation warnings appear in the logs related to sap.ui.core.ID and sap.ushell.URLParsing.isIntentUrl.

  • The backend workflow functions correctly via the backend transaction SBWP.

Environment

SAP S/4HANA

Cause

  • The issue is caused by the use of deprecated and removed SAPUI5 APIs within the older versions of the My Inbox application.
  • Specifically, the method getStatistics() was removed in newer SAPUI5 versions (included in SAP_UI 758), causing the Component-preload.js of the Fiori app to crash when attempting to resolve URL parameters for task navigation.
  • The reported environment of this scenario was My Inbox version = 100.39.3, which does not contain the necessary code adjustments to handle the API removals in S/4HANA 2023.

Resolution

To resolve this issue, the My Inbox UI component must be patched to a version that is compatible with the updated SAPUI5 framework.

  1. Apply SAP Note 3462177: This note specifically addresses the getStatistics is not a function error.

  2. Minimum Version Requirement: Update the My Inbox UI component to at least version 100.39.6.

  3. Recommended Version: Per SAP Note 2676990 (Central Note for My Inbox), S/4HANA 2023 supports patches up to 100.39.31 and higher. It is recommended to apply the latest available patch for the 100.39.x stream to ensure all deprecation issues are addressed.

FAQ and Additional Clarification:

  1. Is this fix for "Classic" or "Task Center" based My Inbox? The fix (Version 100.39.6+) is applicable to the My Inbox app itself. Even if you are using the Classic configuration, the underlying UI5 component must be updated to remain compatible with the SAP_UI 758 platform.
  2. Is Classic My Inbox (F0862) still supported? Yes, it remains supported; however, it requires these specific patches to function on S/4HANA 2023 due to the underlying framework changes. 
  3. Should we migrate to Task Center? While SAP Task Center is the strategic direction for cross-environment workflow integration, the immediate fix for the "Open Task" button failure is a component patch to the existing My Inbox installation.
