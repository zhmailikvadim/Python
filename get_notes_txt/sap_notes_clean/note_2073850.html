Symptom

You have a pricing document (e.g. CRM Sales Order) and enter a manual price condition without any amount. Afterwards the entered condition line is not editable but it should be editable.

Other Terms

None.

Reason and Prerequisites

Program error.

Solution

This Java correction is supported as of SAP_AP 700 Support Package 33.
For Support Package 14 - 32 a Java patch will be available. Please refer to the SAP note 880749 how to import this patch.
