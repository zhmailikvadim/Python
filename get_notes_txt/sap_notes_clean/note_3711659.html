Symptom

The Master Data Cleanup Wizard does not find an item with a specific property selected.

Environment

SAP Business One

Reproducing the Issue

Prerequisite:

1. SAP Business One FP 2408 or higher version is used.

2. Select property no.:50 for Item X today (Item Master Data → Properties tab).

Scenario:

1. Run the Master Data Cleanup Wizard on the same day when property no.:50 was selected (Administration → Utilities → Master Data Cleanup Wizard).

2. In step 1 of the wizard select the Start New Master Data Cleanup Run option.

3. In step 2 of the wizard select the Item Master Data option.

4. In step 3 of the wizard select the Additional Filters checkbox and click on the Ellipsis icon. The Additional Filters window is opened.

5. Select the checkbox Find items with only the selected properties and select the property 50. Click OK, then click on Search.

Expected behavior:

Item X is not found by the wizard.

Resolution

From the version SAP Business One FP 2408, in case the update date of an item is the same date as the running date of Master Data Cleanup Wizard, the item will be filtered out in the selection and will not be found by the wizard. This is correct behavior as per system design and due to requirements added by the Audit Trail Report functionality.
