Symptom

On starting a Multidimensional Data Grid based application with a default variant and bookmark a message is raised, if the variables in the bookmark differ from the saved values in the variant.

This message should only be reported on initially loading the application, but not if the application is reloaded or a saved tile/bookmark/shared application state is loaded. That is fixed with this note.

Other Terms

Web Dynpro Data Grid, FPM Bics Grid

Reason and Prerequisites

Coding Error

Solution

Implement the attached correction instructions or apply the corresponding support package.
