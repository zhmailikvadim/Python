Symptom

You have released a document with several assigned services. You then create a new document version and link the new document version with a subsequently entered service. However, the system assigns this service to the first document version.

Other Terms

parameterized, document category, PMD, version, release

Reason and Prerequisites

There is a program error during the separate assignment of multiple services to individual document versions.

Solution

The specified patches solve this problem.
For an advance correction, you can implement these correction instructions.
