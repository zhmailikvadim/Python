Symptom

  • After purchasing a mixed subscription that includes single games and a hospitality table plan, tickets appear under “my tickets” in the online shop but cannot be selected or downloaded.
  • The print button is missing in “my tickets”.
  • No error message is displayed.

Environment

SAP Event Ticketing - Online Shop

Reproducing the Issue

  1. Open the online shop.
  2. Select a mixed subscription that combines single games and hospitality tickets
  3. Complete a purchase.
  4. Navigate to “my tickets” and try to select or download the tickets; observe that the print option is missing.

Cause

  • Ticket layout was not assigned to the corresponding events in the mixed subscription, which prevented the print option from appearing in “my tickets”.
  • A combined “passbook/print@home” shipping method configuration caused issues; assigning “passbook” and “print@home” separately resolved the missing print option.
  • In the mixed subscription is "force booklet print" set up

Resolution

  1. Assign a valid ticket layout to each event included in the mixed subscription 
  2. Review event shipping methods:
    1. setup “passbook/print@home” as specialized shipping method or
    2. setup only “passbook”  as specialized shipping method
  3. Verify mobile ticketing configuration is consistent:
    1. Confirm mobile ticket settings in each event.
    2. Confirm the shipping methods support mobile/print formats as intended.
    3. Confirm the shop settings allow printing in the self-service “my tickets” area.
  4. If reservations are used, ensure the setting to display reservations under “my tickets” aligns with the intended behavior; otherwise, only completed bookings will show with print options.
  5. Ensure that "default booklet print" is set up in the mixed subscription
