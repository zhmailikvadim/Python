Symptom

After the implementation of Note 3634494, a noticeable decrease in system performance occurs within the RAP environment. The issue manifests as increased response times during data processing or service execution. Investigation reveals that a mandatory database read operation was introduced that is redundant for RAP-based services but necessary for Web Dynpro ABAP (WDA) applications.
