Symptom

When running the BELCOTAX - Yearly Income Declaration (RPCBWAB0) report for fiches 281.20,  for zone 2.030 is not correctly generated. Now this zone will be done in a similar way than the zone 2.031 of fiche 281.10.

This means that the zone 2.030 of fiche 281.20, will generated independently of the zone 2.069 of the same fiche, and use the evaluation class 14 values 06, 52 and 53 accordingly.

Other Terms

BOW, BOWA, BELCOTAX - Yearly Income Declaration (RPCBWAB0).

Reason and Prerequisites

Program error.

Solution

As a general rule, SAP recommends that you install a solution by applying a Support Package.

However, if you need to install a solution earlier, use the Note Assistant to implement the correction instruction.

You can find more information about the Note Assistant in SAP Service Marketplace, under service.sap.com/note-assistant.
