Symptom

Error message: "The field MARC-DISMM is defined as a required field; it does not contain an entry" encountered when using BAPI_MATERIAL_MAINTAINDATA_RT to create material, even though a value is passed.

Environment

  • SAP ERP Logistics (LO)
  • SAP S/4HANA, on-premise
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Use BAPI_MATERIAL_MAINTAINDATA_RT to create material in T-code: SE37.
  2. Pass a value for the MARC-DISMM field for plant.
  3. Observe the error message: "The field MARC-DISMM is defined as a required field; it does not contain an entry".

Cause

If you have maintained reference plant for Stores/DCs, you also need to maintain MARC entry for the reference plant.

Resolution

Please ensure that the MRP type is maintained for the reference plant for Stores/DCs.

See Also

Refer to: 3377977 - Missing views/fields or error message 'The field & is defined as a required field; it does not contain an entry' (MG 144).
