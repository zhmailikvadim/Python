Symptom

MRKO postings are not updating table FIWTIN_ACC_EXEM

Environment

  • Financial Accounting
  • SAP R/3
  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA
  • SAP S/4HANA Finance
  • This is relevant for India Localization or Country India Version (CIN) component.

Resolution

Currently there is no localization on MRKO postings, hence the table FIWTIN_ACC_EXEM is not getting updated.

Keywords

MRKO, MIRO, TDS, Exemption, India, FIWTIN_ACC_EXEM, 194Q, FI-LOC-FI-IN, FI_LOC-FI-IN-WHT
