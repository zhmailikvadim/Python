Symptom

When creating a Price Review using a Review Model and selecting a specific validity date range, the system displays pricing records in the pricing grid that fall outside the selected validity period. This SAP Note resolves the issue by ensuring that only records whose validity dates fall within the selected period are displayed in the pricing grid.

Other Terms

/IRM/LGPRMF0A, Price review Validity, /IRM/IPPDM, Review Model Validity

Reason and Prerequisites

This is caused by Program error.

Please implement the SAP Note#3467926 as prerequisite note for V/4 2022 V2, 2023 and 2023 V2 releases.

SAP Note#3713881 is available for 60E release.

Solution

Please implement the correction instructions. These corrections are only valid if you have applied the following Support Packs.

Release V/4 – 2022 V2, 2023, 2023 V2, 2023 V3 and 2023 V3 Support Package 1

Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object

3713881
