Symptom

Error Message "Valuation type could not be determined for the stock segment" No. SGT_01250 in MIGO or during the PO creation.

Environment

  • S/4HANA Retail & Fashion Management (LO-RFM) 
  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Cause

The valuation type is not maintained for the material’s segmentation value in the material master.

Resolution

Update the material master (MM02) to maintain the valuation type for the specific stock segment.

Keywords

stock segment, valuation type, material master, SGT_01250, PO, MM02, MIGO
