Symptom

User creates a Purchaser Order Response manually from the POWL for a Purchase Order, but the item longtext is not copied over from the PO to the POR correctly.

Other Terms

POR manual confirmation, PCO, Purchase Order Response, supplier text, longtext not copied, CREATE_POR_ITEM, CREATE_PCO_DB, vendor text

Reason and Prerequisites

This is caused by a program error. The  item longtext should be copied over from the PO to the POR.

Solution

Implement the correction instructions or the corresponding support package
