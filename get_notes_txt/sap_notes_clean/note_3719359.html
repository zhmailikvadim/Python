Symptom

Trip is missing in PR05 overview even you find the trip number in table PTRV_HEAD.

Environment

SAP Travel Management - on premise

Reproducing the Issue

  1. Create a trip new in SAP Travel Management.
  2. Navigate to transaction PR05.
  3. You are missing the trip you just created in PR05 overview page but trip number is shown in TE-Cluster PRTE / table PTRV_HEAD.

Cause

user settings in table PTRV_USERPREFS 

Resolution

Adjust the 'User Settings' in PR05 or TRIP (stored in table PTRV_USERPREFS) 

1. Start transaction PR05 or TRIP. Click the icon 'settings' and change the date range to 'ALL' 

3. When opening PR05 all trips are displayed
