Symptom

Unauthorized syntax error if language addition EXCEPT is used

Other Terms

﻿KERNELCOR﻿﻿﻿, CORRESPONDING, Include, Suffix﻿

Reason and Prerequisites

If the language addition EXCEPT is used, an unexpected syntax error occurs if the EXCEPT clause contains a named include with a suffix.

Solution

This error has been corrected as part of a kernel patch.

The patch comment is "CORRESPONDING: unexpected syntax error for EXCEPT﻿".﻿

This correction is delivered with the following SAP kernel archives:

  • Hotfix - file ﻿﻿DISP+WORK﻿﻿
  • ﻿SP Stack Kernel - files SAPEXE.SAR and SAPEXEDB.SAR ﻿﻿

Follow our recommendations for selecting and applying the kernel patch:

  1. Import the latest SP Stack Kernel﻿ if it contains the correction. For a list of current SP Stack Kernels, see SAP Note ﻿﻿2083594﻿ ("SAP Kernel Versions and SAP Kernel Patch Levels").
  2. Otherwise, the corrections will be provided with the next SP Stack Kernel. Apply the hotfix only if you need the corrections quickly.
  3. The correction might not be available for kernel releases that are no longer maintained. Perform an upgrade to the relevant downward-compatible kernel (DCK); see SAP Note ﻿2083594 (SAP Kernel Versions and SAP Kernel SAP Levels).
  4. Check the regression note for the target patch level before applying the patch. For detailed information, see SAP Note ﻿﻿﻿1802333﻿ (Finding information about regressions in the SAP kernel).
  5. For instructions on how to download and apply kernel patches, see SAP Note 19466﻿ (Downloading SAP kernel patches).

For detailed information about our recommendations, see the following strategy document: ﻿Update Strategy for the Kernel of the Application Server ABAP in On Premise Landscapes﻿.
