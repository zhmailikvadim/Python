Symptom

The application log for the update contains error messages due to incorrect processing steps. However, these process steps were executed again in an interval repetition and were processed successfully. No further action is required for the error messages.

Other Terms

Basis line of business

Update

Update policies or contracts

/PM0/ABP_TM_MSG 088

/PM0/ABP_TM_MSG 089

Reason and Prerequisites

The interval repetition was not taken into account.

Solution

For an advance correction, implement the source code corrections.

The system now issues the messages /PM0/ABP_TM_MSG 088 and 089 as warnings if the interval is repeated.
