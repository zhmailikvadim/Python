Symptom

When dismantling an equipment with RAP Entity Manipulation Language, the equipment is removed from the superior technical object but the address remains.

Other Terms

 EAM, CL_EAMS_BO_EQUI, CL_EAM_RAP_API_EQUI, BAPI_EQUI_DISMANTLE, read_masterdata_itob, message IE 063,

Reason and Prerequisites

This is due to programming error.

The Equipment buffer was not reread before equipment was checked for changes comparing the old object record. This caused the address field to remain on the Equipment and was overwritten by the following save.

Solution

The correction adds the necessary buffer read so that the Equipment data is compared and updated correctly.

Please install the corrections in this SAP Note or upgrade to equivalent Support Pack.
