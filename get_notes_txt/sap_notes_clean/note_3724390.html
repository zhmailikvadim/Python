Symptom

During CO document replication in CFIN, there is the error message

System error (COKEY_ENTRY_NOT_FOUND &)
Message No. FINS_CO_2_ACDOC001

Environment

  • Financial Accounting (FI)
  • Central Finance (CF)
  • SAP S/4HANA Finance
  • SAP S/4HANA
  • SAP ERP

Reproducing the Issue

Replicate a CO Document from Source System to Target System in the Central Finance(CFIN) landscape.

Cause

Missing value in table "COKEY" for the field "HRKFT" during CO document replication in the Target system.

Resolution

Ensure that the entry for the replicated CO Document in field "CO subkey - HRKFT" is available in the Target System table "COKEY".
