Symptom

When attempting to activate a Privileged Access ID, the activation fails. A message redirecting you to this KBA is displayed in the log.

Environment

SAP Cloud Identity Access Governance 2.0

Cause

To mitigate security vulnerabilities and inconsistencies in the PAM process, the system verifies that all mandatory SAP Notes are implemented. If any required note is missing, the activation is blocked.

These SAP Notes contain mandatory:
 
Security corrections
Stability improvements
Functional fixes for Privileged Access Management

Resolution

Implement all SAP Notes listed below:

  • 3228601
  • 3327858
  • 3353902
  • 3434812
  • 3528087
  • 3575316
  • 3586699
  • 3593493
  • 3604073
  • 3606297
  • 3610975
  • 3613465
  • 3618392
  • 3631022
  • 3663834
  • 3666084

Once all mandatory notes are implemented, the activation process will complete successfully as before.

Additional Information:

The list of required notes may change as new corrections become available. Always ensure that the latest version of each note is implemented.
