Symptom

Pick List Value Upload fails with error: File Error : Number of values entered does not match the number of Custom Lookup segments. 

Cause

The segment count is correct; however, the issue occurs when the custom lookup values in the upload file are not separated using the delimiter defined in the custom lookup configuration. 

Resolution

Verify the delimiter configured in the custom lookup and ensure the upload file uses the same delimiter.

  1. Sign in to SAP Fieldglass.
  2. Click Admin Configuration (gear icon).
  3. Navigate to Configuration > Custom Lookup.
  4. Search for and select the relevant custom lookup.
  5. In the Delimiter field, identify the configured delimiter.
  6. Update the upload file to separate custom lookup values using the identified delimiter, then reattempt the upload.
