Symptom

This SAP Note doesn't affect your business functionality. It is required as a prerequisite for the SAP Note 3707258. Do not install it separately from this SAP Note.

Reason and Prerequisites

This SAP Note contains reports to automate the creation or modification of objects used in the solution provided by the SAP Note 3707258.

Solution

Use transaction SNOTE to implement the attached correction instructions before you implement the primary SAP Note 3707258. The transaction SNOTE installs dictionary objects and the reports NOTE_3707258, NOTE_3707258_EAHR in your system. After that, implement the primary SAP Note 3707258.

As a general rule, SAP recommends that you implement the Support Package specified for your release. However, if you need to install the solution earlier, use the SAP Note Assistant to implement automatic correction instructions. For more information, see SAP Note Assistant.
