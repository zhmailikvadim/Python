Symptom

•   Cannot create any Web Intelligence document based on Google BigQuery universe with ODBC middleware, error below is displayed:
>> "Error: An internal error occurred while calling "processDPCommandsEx"API”
•   Also, Web Intelligence Server crashes with following error:"Your Web Intelligence idle connection has expired. Close this document and reopen it."

Environment

•  SAP BusinessObjects Business Intelligence Platform 4.3
•  Web Intelligence
•  ODBC SIMBA Driver Big Query 3.1.6.1026
•  ODBC SIMBA Driver Big Query 3.1.5.1022 

Reproducing the Issue

  • Create a relational connection based on Bigquery (either with ODBC SIMBA Driver Big Query 3.1.6.1026 or ODBC SIMBA Driver Big Query 3.1.5.1022) in Information Design Tool.
  • Create a universe based on that connection.
  • Log on to BI Launchpad.
  • Create a Web Intelligence document based on newly created universe based on Google BigQuery.
  • Refresh the query.
  • Observe the error:
    >> "Error: An internal error occurred while calling "processDPCommandsEx"API”

Cause

The driver Bigquery ODBC 3.1.6.1026 and 3.1.5.1022 is not certified with current SAP BusinessObjects Business Intelligence Platform 4.3.

Resolution

Create a connection in Information Design Tool based on Google BigQuery using Google BigQuery ODBC 3.0.4.1008 driver.

Keywords

Bigquery, big query, google, simba, crash, close, stop, restart, webi
