Symptom

Post upgradation the CURTP field is blank in table V_GLPOS_N_GL_CT

Environment

  • Financial Accounting (FI)
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA Finance
  • SAP S/4HANA

Reproducing the Issue

  1. Go to SE16N
  2. Pass table name as V_GLPOS_N_GL_CT
  3. Pass any value in the field CURTP

Cause

The currency and currency key fields are intentionally returning null values in the latest versions

Resolution

The fields will be already available in any of the field given below.

CURRKEY_00    Update Currency for General Ledger Totals Data
CURRVAL_00    Update Currency Value for General Ledger Totals Data
CURRKEY_0D    Document Currency Key
CURRVAL_0D    Document Currency Value
CURRKEY_10    Company Code Currency Key
CURRVAL_10    Company Code Currency Value
CURRKEY_11    Company Code Currency Key, Group Valuation
CURRVAL_11    Company Code Currency Value, Group Valuation
CURRKEY_12    Company Code Currency Key, Profit Center Valuation
CURRVAL_12    Company Code Currency Value, Profit Center Valuation
CURRKEY_30    Group Currency Key
CURRVAL_30    Group Currency Value
CURRKEY_31    Group Currency Key, Group Valuation
CURRVAL_31    Group Currency Value, Group Valuation
CURRKEY_32    Group Currency Key, Profit Center Valuation
CURRVAL_32    Group Currency Value, Profit Center Valuation
CURRKEY_40    Hard Currency Key
CURRVAL_40    Hard Currency Value
CURRKEY_50    Index Currency Key
CURRVAL_50    Index Currency Value
CURRKEY_60    Global Company Currency Key
CURRVAL_60    Global Company Currency Value
