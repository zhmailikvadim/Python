Symptom

An error occurs during saving of a subscription billing item in a solution order.
Error message: Quantity could not be converted from &1 to &2 (Error ID: V1297)

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Open the "Manage Solution Order" app.
  2. Create a new solution order.
  3. Add a subscription billing item.
  4. Verify that pricing is determined.
  5. Save the solution order and observe the V1297 error.
  6. Check the product in the "Rate Elements" assignment block
  7. Search the product using MM03 in S/4HANA Cloud system
  8. Observe the error "The material & does not exist or is not activated" (Message ID: M3305)

Cause

A product referenced by the solution order (for example, used in rate elements or pricing) was not synchronized to the S/4HANA Cloud system, so its unit of measure could not be determined, resulting in error V1297.

Resolution

  1. Identify all products directly or indirectly referenced by the solution order (including products used in rate elements and pricing).
  2. Check in product master data that each referenced product exists in the system and that its base and alternative units of measure are available.
  3. If any referenced product is missing, replicate or synchronize it to the S/4HANA Cloud system from the source/master system.
