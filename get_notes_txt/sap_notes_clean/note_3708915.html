Symptom

The FIFO Valuation Extract generated from the app "Create FIFO Valuation Extract" does not contain any data.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Navigate to the app "Create FIFO Valuation Extract".
  2. Attempt to execute the FIFO Valuation for specified materials.
  3. Observe that the generated list does not contain any data.

Resolution

In Order to see list of data. Ensure that the below steps are followed: 

  1. Access the app Flag material for FIFO valuation.
  2. Run the transaction using material type, company code, and plant.
  3. Ensure all materials are selected in the result.
  4. Access the app Create Extract Documents.
  5. Run the transaction and verify the result.
