Symptom

In the frame of a remote conversion to BCS4HANA, the import of the BCS transport terminates with dump MESSAGE_TYPE_X in class CL_UG_MD_GEN_BUILDING_PLAN. The import may also fail later due to attempt to write data of InfoObject 0BCS_AREA to BW.

Other Terms

AREAID, REGISTER_FIELD_CLOSURE, RSPMPROCESS, Process Type TRANSFLOCK, 0BCS_AREA

Reason and Prerequisites

Program error.

Solution

Apply attached source code correction or apply corresponding support package to your new BCS4HANA system, prior to importing BCS transport.
