Symptom

  • The field 'Max. Storage Period'(MARC-MAXLZ) is not visible in 'General Plant Data / Storage 1' View for some materials.
  • Message M3245 "No plant-specific data exists" is displayed when accessing Storage views. 

Environment

  • SAP R/3 
  • SAP R/3 Enterprise 
  • SAP ERP Central Component
  • SAP ERP
  • SAP Enhancement package for SAP ERP 
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA, on-premise 
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Transaction MM02/MM03.
  2. Select 'General Plant Data / Storage 1' View.
  3. Enter Plant.
  4. The field 'Max. Storage Period' is not visible. Message "No plant-specific data exists" is displayed.

Cause

The 'General Plant Data / Storage 1' View has not been extended to the plant yet (Maintenance status 'L' is missing in MARC-PSTAT).

Resolution

  1. Open transaction MM01.
  2. Enter Material.
  3. Select the 'General Plant Data / Storage' view.
  4. Input the plant.
  5. You will be able to see the plant level fields and receive an information message stating 'material already exists and will be extended'.
  6. Save. This will update MARC table with status 'L' for Storage and the field 'Max. Storage Period' will be displayed in transaction MM03.
