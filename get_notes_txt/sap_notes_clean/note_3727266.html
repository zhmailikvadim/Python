Symptom

I need to add a User as observer of a task in Guided Sourcing Project/Request so this User will receive notifications about the task.

Resolution

To add Observers to a task in a Guided Sourcing Project or Request you should follow the steps below.

1. Open the Sourcing Project or Sourcing Request

2. Scroll down the tab Tasks > Open the task you want to add an observer to

3. Go to Task Details > Edit > Observers > Select Observers

4. Select the Users you want to observe the task.

5. Save it.

If you need this change to be applicable for all Sourcing Projects or Requests, please add the observers at template level.
