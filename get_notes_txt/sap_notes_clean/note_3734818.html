Symptom

The requirement is to pass Additional Data Collection custom object information to the Employee Profile.

Environment

SAP SuccessFactors Onboarding

Resolution

  • It is recommended to utilize the hire template in PDC to pass custom object information to the Employee Profile following Configuring Hire Template for Access New Hire | SAP Help Portal.

  • Enhancement requests can be submitted following KBA 2090228. 

  • The use of other methods can be discussed with consulting resources.

See Also

  • KBA 2090228 - How to submit enhancement ideas for SAP SuccessFactors Products
  • Configuring Hire Template for Access New Hire | SAP Help Portal
