Symptom

You use Focused Build Test Steps to design and execute manual test cases with SAP Solution Manager. In Test Steps Designer you want to use the translation functionality to translate your test cases into different languages.

This SAP Note provides you corrections and a new BAdI example implementation (/SALM/TM_TS_TD_TRANSLATE_IMPL2) for SAP Translation Hub. The previous implementation (/SALM/TM_TS_TD_TRANSLATE_IMPL) used the SAP Translation Hub on Neo environment for translation. This will not work from 2029 onward due to the sunset of the SAP Business Technology Platform, Neo environment. Please refer to SAP Help: SAP Translation Hub - Migrating from Neo to Multi-Cloud

Reason and Prerequisites

Focused Build, Test Steps, Test Steps Designer Translation, SAP Translation Hub, Machine Translation

Solution

Implement this SAP Note and follow the attached documentation in case you want to make use of the new BAdI example implementation.
