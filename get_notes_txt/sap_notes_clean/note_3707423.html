Symptom

Automatic PR to PO Conversion is an existing report which is used as scheduled program to convert the approved PR items into a PO based on the criteria specified. Now the same report shall support the conversion of Enhanced Limit PR items into a PO as well .

Environment

  • SAP S/4HANA 2023

Reproducing the Issue

  1. Approve a purchase requisition in the system.
  2. Attempt to convert the approved requisition into a purchase order using the Process Purchase Requisitions application.
  3. Observe that the process requires manual intervention, leading to delays.
  4. Attempt to use the ME59N transaction for automation, and observe that Enhanced Limit requisitions are not supported.

Cause

New Feature Introduced in SAP Note  3435267 - Auto PO Creation for Enhanced Limit Item PR.

Resolution

Implement  SAP Note 3435267 - Auto PO Creation for Enhanced Limit Item PR.

Afterwards, they can execute and plan ME59n for Enhanced Limit PR lines to generate auto-POs as well
