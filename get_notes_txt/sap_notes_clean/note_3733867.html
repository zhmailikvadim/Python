Symptom

Users search for a BAPI to create a customer by reference (copying from an existing customer).

Environment

  • Sales and Distribution (SD)
  • Logistics Execution (LE)
  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA

Reproducing the Issue

Attempt to find or use a BAPI that creates a customer by reference (copying from an existing record).

Resolution

There is no dedicated BAPI for "create customer by reference"; this capability is provided as a UI-only feature.
