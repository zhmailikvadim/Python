Symptom

Changes to subnodes (for example: products/preference statements/assigned documents) of existing LTSD requests, inbound or Outbound LTSDs do not lead to an update of the fields "Changed By" and "Changed On".

If, for example, you use transaction /SAPSLL/LTSDREQ_1 (Creation of LTSD Requests) to change LTSD requests, these changes are not visible in the "Manage Requests - Long-Term Supplier's Declaration" app using the specified fields.

Solution

Implement the attached correction instructions.

The changes in this SAP Note also add the "Created By" field for LTSD requests. In addition, implement SAP Note 3718665 in your SAP GTS UI system to update the SAP Fiori app "Manage Requests - Long-Term Supplier's Declaration".
