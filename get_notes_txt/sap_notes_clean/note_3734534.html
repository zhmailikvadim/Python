When importing records through the External Time Data object or when an administrator records a time type, workflows configured for specific time types (SRA – Single Record Approval) are triggered automatically.

For example, a workflow associated with an “Extra” time type is initiated upon record creation. In certain business scenarios, this automatic triggering is not desired, such as when an employee attempts to record and save the same time type.

Environment

SAP SuccessFactors Time Tracking 

Resolution

  1. Create a Workflow as below (Sample) in Manage Organization, Pay and Job Structures
  2. Go to Configure Object Definitions.
  3. Select Object.
  4. Object Definition: Attendance Request (AttendanceRequest) 
  5. Assign save rule as per your requirement.

  7. Now when we proxy as employee 'D Positive' or this employee logs in and record the time type SP_Extra-SRA, the approval will not go to Manager and the time type record will be auto approved (Set to Approved status).
