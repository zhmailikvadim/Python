Symptom

You execute the archiving write program of Logistics Invoice Verification (RM08RW47 or RM08RARC) and receive the runtime error ITAB_ILLEGAL_SORT_ORDER. Some of the invoices to be archived contain their own conditions or vendor conditions.

Other Terms

Dump, RM08RW47, RM08RARC, KONV, PRCD_ELEMENTS

Reason and Prerequisites

This is a program error.

Solution

The correction will be delivered with the specified support packages. Implement the relevant correction instructions as an advance correction.
