Symptom

Upload button is not visible in the Manage Locations app for the bulk upload. 

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Go to Manage Locations app
  2. Observe that the Upload button is missing

Cause

Upload capability is currently not provided in the Manage Locations app (feature not available).

Resolution

In the current release mass upload function is not available in the Manage Locations app and it can be taken as a new requirement. To address feature requests for SAP S/4HANA Cloud, Public Edition, you can use the SAP Continuous Influence process described here (https://community.sap.com/topics/s4hana-cloud/continuous-influence). More information can be found also in SAP Note 2963059 - How to Submit Feature Requests for SAP S/4HANA Cloud Product.

Please use session "SAP S/4HANA Cloud for Supply Chain Logistics, public edition" (https://influence.sap.com/sap/ino/#/campaign/2897) to address specific feature requests for the Transportation Management business processes.

Note: 

Users can make use API services to create, read, update, and delete locations in SAP S/4HANA Cloud Public Edition using this synchronous inbound service. (https://api.sap.com/api/CE_MANAGELOCATION_0001/overview)
You can perform these activities from a source system outside SAP S/4HANA Cloud Public Edition system. However one location created at a time and the service is based on the OData protocol.
