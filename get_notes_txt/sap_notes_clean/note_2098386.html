Symptom

When selecting a condition in the financial condition overview, another condition of the same condition list is displayed in the details view in some constellations (e.g. in f9cogr1, f9cogr2, f9cogr3).

Other Terms

FICOO0, FICO_OV_PROCESS_PBO, FICOO0F07, PBO_1000_UPDATE_FIELDCAT

Reason and Prerequisites

The problem is caused by a source code error.

Solution

Import the relevant Support Package for your release.

You can use Transaction SNOTE to implement the following correction instructions in advance. There are no manual activities necessary.
