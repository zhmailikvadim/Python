Symptom

You follow the steps in note 3429227 to use enhancement spot CTVB_CHANGEDOC_CHAR_DES.
There is error "Enhancement spot CTVB_CHANGEDOC_CHAR_DES does not exist" when you try to open it in SE18.

Environment

  • SAP ERP Logistics (LO)
  • SAP ERP Classification (CA)
  • SAP S/4HANA
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA Cloud Private Edition

Resolution

Implement the note first and then follow the manual activity in note.

3429227 - CT04: Record changes of descriptions on characteristic and/or values level
