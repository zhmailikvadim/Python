Symptom

According to Sec. 4a of the General Pension Act (APG), the sixth control calculation is to be suspended if an employee receives a partial pension (see also EStG Sec. 77, Para. 4a, first clause, exception reason i).

Other Terms

Wage type /4J3, wage type /4JV, AJST0, HRPAYAT_AJST0_001

Solution

For the hiring action for partial pensioners (or for employees already in partial pension), enter wage type /4J3 with a set number field in infotype 0014 for the entire duration of the work relationship. The wage type is stored in the results table and causes wage type /4JV to be generated in payroll function AJST0. This, in turn, suspends the sixth control calculation. In the context of queries, you can use /4J3 to see in the result that this employee is a partial pensioner. Perform retroactive accounting for the affected persons accordingly.

The correction is provided by HR Support Package (via CLC if necessary).
You can implement an advance correction by performing the manual activity and implementing the automatic correction instructions using SNOTE.
