Symptom

For an employee, you have created an EEL notification reason 11 ("Remuneration Statement PI Transitional Pay Med. Rehab). If you enter a leaving for this personnel number at a later point in time, the system reverses the notification Reason 11 and creates a new notification. This is created with the error message "Infotype 0651, subtype 1/2 does not exist at start of absence" with the status With Errors.

Other Terms

EEL, health cure, submission reason 11

Solution

The "Employed Until" field (BVBIS) in the data module DBLT (Additional Data Calculation of Transitional Pay) is taken from the comparison. As a result, the entry of a leaving no longer leads to a reversal and new notification of notifications with reason 11. This procedure corresponds to the specifications of the procedure description for the reimbursement of notifications as part of the data exchange for compensation benefits according to Sec. 107 SGB IV in the version valid as of January 1, 2026. For more information, see section 3.15.3).
