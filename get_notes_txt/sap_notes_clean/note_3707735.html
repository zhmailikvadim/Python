Symptom

This SAP Note contains modifications related to the Employment Certificate (RPCFGLH0) report
and related objects.

Detailed description can be found attached.

Version information

Version 1: Original version

Other Terms

R/3 HR CEE, Hungary, RPCFGLH0

Reason and Prerequisites

The prerequisite for the correction:
HR-CEE/SAP_HR/S4HCM SP11/2025 - HR-CEE/SAP_HR/S4HCM SP02/2026

Solution

---------------------------------------------------------

Installation steps
---------------------------------------------------------

1. Only for versions 600 and 604! Apply the Correction Instructions of the SAP Note
3709700 with the Note Assistant (transaction SNOTE; more information on 
service.sap.com/note-assistant) and execute the further implementation steps in the SAP note.

IMPORTANT!
- Run the newly created report in "Update ..." mode: RPUUDOH0_3709700
- Make sure that the created or modified objects are all active.

2. Apply the Correction Instructions of the SAP Note 3707735 with the Note Assistant
(transaction SNOTE; more information on service.sap.com/note-assistant)

3. Upload the modified smartform HR_HU_FOGLIG26 (attached to the current SAP Note 3707735):

  • Extract the smartstyle and the smartforms from the Smartform_3707735.zip file.
  • Refer to the attached Upload_Modifed_Smartform_Style.pdf document for steps
    to upload the form.

The modifications will also be delivered with the HR-CEE/SAP_HR/S4HCM SP03/2026 package
(and the equivalent CLC package for Hungary).
