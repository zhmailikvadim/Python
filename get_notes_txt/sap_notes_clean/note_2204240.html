Symptom

In report HSKCELD0 absence 0537 ('Otcovska dovol. bez mat.') needs to be reported with sign 'A', not sign 'MD' as has e.g. absence 0525. Modification is needed.

Other Terms

Report HSKCELD0, absence 0537.

Reason and Prerequisites

This SAP Note is relevant only for Slovakia (HRSK).

Solution

This SAP Note delivers modification of report HSKCELD0 to report absence 0537 with 'A', not sign 'MD'.

 

Implementation

  • As a general rule, SAP recommends that you install a solution by applying a Support Package. However, if you need to install a solution earlier, use the Note Assistant to implement the correction instruction and perform manual implementation steps described in attached file 'Manual Instructions of Note 3346143.docx'.
