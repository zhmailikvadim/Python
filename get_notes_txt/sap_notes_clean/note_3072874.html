Symptom

If a variable is accessed in write-only mode with no read access, this indicates a program error. The extended program check returns a warning in such cases: DAT 0908.

However, if a table expression is used to access rows of an internal table in the form of a method call, this warning is output without justification:

itab[ ... ]->meth( ).

or

itab[ ... ]-ref->meth( ).

Other Terms

SLIN

Reason and Prerequisites

Program error in the extended program check.

Solution

Implement the correction.
