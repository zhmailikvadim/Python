Symptom

You are trying to access the profile of an employee using the Latest People Profile, through "Full Profile" button on Spotlight. After clicking to access, an error is displayed and no information about the user appears: "You do not have permission to view this profile".

Environment

SAP SuccessFactors Employee Central

Reproducing the Issue

  1. On the Search bar, search for a user that you have permission to access
  2. Click on spotlight and open it to access "Full Profile"
  3. After clicking, see that the error is displayed: "You do not have permission to view this profile" 
  4. If you search the same user through People Picker on employee profile or with the Latest Employee Profile disabled, you can normally access the data.

Cause

The user has an invalid per-person-uuid, not system-generated.

Resolution

This KBA will be updated once there is a response from engineering team.

The per-person-uuid is a system-generated field that cannot be modified. Do not try to change the value of this field via Import or add a new user via Import with a custom value on this field. If a custom value is used, Spotlight and integrations with other systems are affected.

If you already have users affected by it, please add the users again in the system without customizing the per-person-uuid and purge the existing user with the data issue.
