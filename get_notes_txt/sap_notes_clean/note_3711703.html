Symptom

You execute the IMG activity "Define Time-Dependent Period Controls". This IMG activity contains subobjects. The system displays "No text found for the transaction" instead of the correct name of the subactivity. The correct name of the subactivity is "Assign Time-Dependent Period Controls to Dep. Keys".

Other Terms

ORFA_PER_ZEIT: Define Time-Dependent Period Controls; T090A

Reason and Prerequisites

Missing documentation

Solution


The corrections are delivered by means of a support package. Import the current support package to obtain the missing text. 
The IMG activity can be executed without any problems despite missing text.  
For depreciation keys with the indicator "Period control per fiscal year", you can use this IMG activity to specify the period control differently from the period control method for selected company codes and fiscal years.
