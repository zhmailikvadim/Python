Symptom

It's noticed that ICF services /default_host/sap/bc/abap/toolsdocu and /default_host/sap/public/bc/abap/toolsdocu are missing in S/4HANA 2025 system during the configuration of ABAP Back-end for ABAP Development Tools for Eclipse.

Environment

  • S/4HANA 2025 system

Reproducing the Issue

  • Execute transaction code SICF
  • Search for services /default_host/sap/bc/abap/toolsdocu and /default_host/sap/public/bc/abap/toolsdocu

Resolution

Activating ICF nodes below is required only for systems release lower than 755. The activation is not necessary in S/4HANA 2025 systems.

  1. For the ABAP - Keyword Documentation view
    • /default_host/sap/public/bc/abap/docu
    • /default_host/sap/bc/abap/docu
  2. For the ABAP Problem Description and ABAP Element Info views 
    • /default_host/sap/public/bc/abap/toolsdocu
    • /default_host/sap/bc/abap/toolsdocu
