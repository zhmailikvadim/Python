Symptom

Multiple GST return submission tasks displayed incorrectly in Singapore GST Return Submission.

Environment

  • SAP S/4 HANA
  • SAP S/4 HANA Finance

Reproducing the Issue

  1. Open the 'Singapore GST F5 Report'.
  2. Set the reporting date in the selection parameters.
  3. Observe that the system displays GST Return Submission report along with the selected reporting date.

Cause

Maintain the wrong activity key for SG_VAT_F5.

Resolution

  1. Change the activity key to SG_VAT_DCL_ON_V4_S and SG_VAT_F7_ON_V4_C.
  2. Test the report again to confirm the resolution.
