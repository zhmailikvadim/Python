Symptom

Customer does not want certain data to be triggered by CT, and does not want to see any data generated with a CT indicator of 1 or 4 in the CT logs or ACDOCU table.

Image/Data is from SAP internal systems, sample data, or demo systems.  Any resemblance to real data is purely coincidental.  

Environment

  • SAP Fiori for SAP S/4HANA Finance
  • SAP S/4HANA Finance for Group Reporting

Reproducing the Issue

Execute CT in Data monitor.

Cause

Some FS-ITEMs will still generate data for CT indicator 4 even if they are not triggered by any sequence in the CT method, but customer does not want to calculate the reference amount.

Resolution

Try to assign a specific document type to these data, as the example shown in the screenshot. Make sure that local currency and group currency are enabled and no translation is enabled. At this point, the data uploaded using this DT will no longer be triggered by CT.

Please note that if you forget to enable the option 'Post in group currency', the data will still be triggered by CT.
