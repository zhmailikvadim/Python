Symptom

EOD In-House Bank - G/L Transfer Job is unable to post GL with message

'Payment item &1 &2 &3 already transferred to FI'

Other Terms

IHB, DB_ITEM_FI, FIBTI, EOD In-House Bank - G/L Transfer Job

/PF1/CL_IHB_BC_PRX_FIBTI

E007/pf1/ihb_fi

Reason and Prerequisites

The presence of records with status 02 (posted) in the DB_ITEM_FI table caused an error during the GL posting process.

Solution

An additional check was added to determine whether posting needs to be performed.

The logic of the check is as follows:

  • If the finance notification UUID has already been generated on the GL side and the GL document failed, then posting is skipped.
  • If the finance notification UUID has not been generated on the GL side, then posting continues.
  • If the finance notification UUID  has not been saved in the FI table, but the finance notification UUID has been generated on the GL side, then posting continues.
