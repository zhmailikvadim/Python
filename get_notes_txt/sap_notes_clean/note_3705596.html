Symptom

The eDHR Report stopped functioning after the installation of SAP ME patch level 48. Clicking the menu item for the report results in a blank page with no loading time or errors.

Other Terms

SAP ME, eDHR Report

Reason and Prerequisites

Steps to Reproduce:

  1. Open SAP ME.
  2. Navigate to and click on the eDHR Report.

Actual Result: The report does not open and displays a blank screen.

Expected Result: The report displays and functions normally.

Analysis: This issue is caused by problems with the report project build.

Solution

A code correction has been implemented. As a result, the report project builds correctly and the report functions as expected.

Impact

The code change is isolated to the eDHR report's functionality and has no impact on other system features.
