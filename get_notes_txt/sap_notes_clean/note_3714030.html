Symptom

  • After upgrading to S4CORE 108 SP04, an error occurs related to Enrichment & Validation (E&V) Check IDs 213 and 217, which are missing from the configuration under "Define Enrichment and Validation Check Functions."
  • The missing Check IDs are still maintained under "E&V Check Set configuration" but are absent from the standard checks listed in SAPHELP.
  • Error message: "Error while reading EV check ID 000213 & 000217 appeared."
  • Occurrence: During IPO processing.

Environment

SAP S/4HANA 2023

Reproducing the Issue

  1. Manage Payment then filter using Creation date.
  2. Go to respective IPO.
  3. Display LOG.
  4. Error while reading EV check ID 000213 & 000217 appeared.

Cause

The issue is caused by the removal of E&V Check IDs 213 and 217 in S4CORE version 108.

Resolution

  1. Confirm that E&V functions are no longer available in S4CORE version 108.
  2. The checks should be removed from any check-set.
  3. Remove Check IDs 213 and 217 manually from the Check Sets configuration as they are no longer supported in the updated version.
