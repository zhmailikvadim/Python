Symptom

  • When opening a Currency Conversion Table, observe that not all records are visible
  • See following message in SAC "This currency rates table has a large number of rows. The application has loaded only a sample of the available data to optimize performance. To see more data, apply a sort or a filter."

Environment

SAP Analytics Cloud 

Reproducing the Issue

  1. Log into SAC Tenant 
  2. Open Currency Conversion Table 
  3. Observe not all records are visible and see the following message "This currency rates table has a large number of rows. The application has loaded only a sample of the available data to optimize performance. To see more data, apply a sort or a filter."

Cause

  • This behaviour was implemented to help improve performance. It provides a snapshot of the available records

Resolution

  • To see all records, apply a sort or a filter to see more data
