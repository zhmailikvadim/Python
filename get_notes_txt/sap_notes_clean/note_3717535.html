Symptom

Approvals on an SAP Fieldglass Statement of Work (SoW) do not update the corresponding correctly in SAP Ariba Buying.

Cause

OriginatingSystem is not sent as Fieldglass in the web services integration message.

Resolution

OriginatingSystem must be sent as Fieldglass in the web services integration message; no other string is supported, as the SAP Ariba Procurement software is coded to look for the exact string Fieldglass to correctly handle SAP Fieldglass requisitions.
