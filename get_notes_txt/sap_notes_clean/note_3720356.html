Symptom

The field Service Start Date appears as Unclassified in Purchase Requisition report in SAP Ariba Analytical Reports.

Resolution

This is expected behavior in the analytical reports, since some Requisitions have Line Items, it can contain different types of line items. 

Take, for example, PR01. If we filter it in the report, the Service Start Date will appear as Unclassified because the line item type that is automatically displayed in the report for PR01 does not have a Service Start Date associated.

To correct this, please add the Line Item Type field as a detail field to the report. Another line for the same PR01 will then appear. This time including the Service Start Date.
