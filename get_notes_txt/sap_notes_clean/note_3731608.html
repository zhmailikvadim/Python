Symptom

The form generated after the onboarding signed the Document Flow for New Hire Tasks is not automatically visible in the EC profile post conversion.

Environment

SAP SuccessFactors Onboarding

Reproducing the Issue

  1. Onbordee completed Document Flow for New Hire Task
  2. There is a signed form generated
  3. User is converted to active employee
  4. Signed form is not visible in the EC profile

Resolution

This is currently expected behavior. The form can be accessed via Manage Documents or from the Onb Dashboard > Candidate > Actions > Documents.
Enhancement requests can be submitted following KBA 2090228.
