Symptom

When attempting a selective deletion of the data of an info object the following error occurs:

Overlap between deletion criteria and archiving criteria

Message No. RSO_RES_ADSO095

Diagnosis

The operation (delete or update) detects an overlap between the filter criteria and the archiving criteria of the COLD-store / the data archiving process ("DAP").
If the DSO uses a temperature schema with COLD-store, then it is possible to activate the feature "exception updates" and after that the operation can be execute on records, which are currently stored in the COLD-store.
For DSOs using an explicit data archiving process, this feature is not supported.

Procedure

Check whether it is possible to activate the "exceptional updates".
If different time-characteristics are used as filter and as criteria for the COLD-store / archive, then it sometime seems incorrect to output this error message because of the logical dependency of the times characteristics.
In this case, please check the time-consistency via transaction RSRV and if the time is not consistent, then please read SAP Note 3040267 "Deletion criteria and archiving criteria overlap (DBMAN245)", which explains this special case more in detail.

However:

  • The info object does not use Cold-Store, and;
  • There is not data archived or all archived data has being deleted, and;
  • The time-consistency check shows that the object is in fact consistent.

Environment

  • SAP BW/4HANA;
  • SAP NetWeaver.

Reproducing the Issue

  1. Manage the DSO;
  2. Go to 'utilities';
  3. Choose selective deletion;
  4. Define the selective deletion criteria;
  5. Start the process.

Resolution

To solve this issue:

  1. Go to transaction RSDAP;
  2. Enter the name of the DSO;
  3. Delete the DAP for the DSO.

If the issue still persists, follow these additional steps:

  1. Implement SAP Note 0003542530 - Delete obsolete Data Archiving Processes;
  2. Use the report RSDA_DELETE_DAP_WO_CONN to remove any remnants of the DAP object.

See Also

SAP Note 0003542530 - Delete obsolete Data Archiving Processes
