Symptom

User ran a settlement for an order and received an error message: RW010 "FI/CO interface: object type not clear" when you execute CO88 or KO88

Environment

  • Controlling (CO)
  • SAP S/4HANA

Reproducing the Issue

  1. Run transaction KO88 or CO88. 
  2. Insert the effected order or plant, period and fiscal year.
  3. Execute the program.
  4. System displays error RW010 message: "FI/CO interface: object type not clear"

Cause

The price differences are distributed to two items in different valuation views due to different plus/minus signs.

Resolution

Implement the SAP Note 3500346 - KKS9: Unnecessary PRD split, according to your support package.

See Also

3428636 - KKS9: Price difference split enhancements

Keywords
