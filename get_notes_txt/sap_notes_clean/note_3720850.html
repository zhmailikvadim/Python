Symptom

The Check Rule field is not visible in the BP / UKM_BP transaction (UKM000 role) under the Credit Profile Tab.

Environment

  • SAP enhancement package for SAP ERP, version for SAP HANA All versions
  • SAP S/4HANA
  • SAP S/4HANA FINANCE

Reproducing the Issue

  1. Access the BP transaction.
  2. Enter Business Partner number
  3. Select BP Role UKM000.
  4. Navigate to the Credit Profile Tab.
  5. Observe that the Check Rule field is not visible.

Cause

CHECK_RULE field exists in field group 2432 in transaction BUS2.

Resolution

Remove the CHECK_RULE field from field group 2432 in transaction BUS2.

See Also

KBA 3666831 - Fields in transaction UKM_BP under 'Credit Profile' tab are missing
