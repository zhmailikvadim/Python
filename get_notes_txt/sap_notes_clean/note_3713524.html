Symptom

  • When navigating to the file upload section, screen readers can repeat the same information across multiple focus points, making it unclear to users what element triggered the action.
  • After adding a file, the feedback message is unnecessarily long and partially redundant.
  • The screen reader announces “Confirm Confirm” even though there is no confirmation step on the view.
  • Date information for the uploaded file is also read, which is not visually present.

Environment

SAP SuccessFactors Recruiting Management

Reproducing the Issue

  1. Use a screen reader and add or remove files to a Candidate application
  2. Observe unnecessary or incorrect screen readings  

Cause

The issue is caused by accessibility violations in the file upload section, including redundant feedback messages, duplicate interactive elements, and focus jumping to unrelated page elements during file removal.

Resolution

This issue will be fixed as part of the 1H 2026 release. This will be released on 13-Apr-2026 for Preview and on 15-May-2026 for Production.

Keywords

A11y, JAWS, NVDA, WCAG
