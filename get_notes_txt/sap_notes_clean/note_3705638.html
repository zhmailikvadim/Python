Symptom

  • An error occurs when attempting to raise an access request for Firefighter ID using a template based request.
  • The error message displayed is: "There is already an open request for system xxxxxx by user id xxxxxx."
  • This issue prevents the submission of new requests due to unresolved pending requests.
  • Users are unable to locate or terminate the previous request.

Environment

SAP Access Control 12.0 for SAP S/4HANA

Reproducing the Issue

  1. Attempt to submit an access request for Firefighter ID using a template based request.
  2. Receive the error message: "There is already an open request for system xxxxxx by user id xxxxxx."

Cause

  • The issue occurs because the user ID has a pending request in the Access Management module under NWBC. 
  • The request is in a decision pending status.
  • pending status, preventing new submissions.

Resolution

  1. Navigate to GRFNMW_MANUAL_INSTANCE_CANCEL.
  2. Locate the pending request with user id xxxxx.
  3. Abort the request using the GRFNMW_MANUAL_INSTANCE_CANCEL program.
