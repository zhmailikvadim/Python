Symptom

While raising of a BOR (Business Object Repository) event wrong event receivers will be triggered.

This problem occurs only if  BOR object type has "Type Status" = "modeled". 

Reason and Prerequisites

The event manager determines object supertypes. If the BOR object has status "modeled" then the object type  table is empty. The select on linkage table returns all object types.

Solution

Please apply the code correction.
