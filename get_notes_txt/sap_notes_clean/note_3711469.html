Symptom

  • You attempt to configure one or more parameters in a Sender or Receiver adapter in an iFlow in Cloud Integration, but an error similar to the following is returned - 

    You cannot configure dynamic parameters in the receiver adapter. All configurations in the adapter must be static. 

Environment

  • Cloud Integration
  • SAP Integration Suite

Reproducing the Issue

  1. Access an iFlow in the designer section of your tenant
  2. Attempt to configure some of the parameters in either a sender or receiver adapter with dynamic expressions

    For example, attempt to configure the 'address' parameter with the expression ${property.property_name} or ${header.header_name}

Cause

For certain adapter types, some parameters may support dynamic configuration, while others may not. For the parameters which support such configuration, this will be stated in the official SAP Help Documentation for the adapter.

However, there is a further consideration which must be taken into account. As outlined in SAP Note 2903776 - SAP Integration Suite – Service Plans and Upgrade Paths, there are limitations in place on Cloud Integration tenants which are using the 'Starter Edition' service plan. These limitations include the following - 

No dynamic parameter allowed in the sender and receiver system configuration in the adapters.

Resolution

  1. Use the SAP Help Documentation to verify whether it is supported to configure the parameter dynamically for the respective adapter type in which you are attempting to do so.
  2. If it should be possible according to the documentation, but you are encountering the message advising that 'All configurations in the adapter must be static', verify whether the tenant is using a 'Starter Edition' service plan.

    If it is, then in order to use dynamic expressions in the configuration of Sender or Receiver adapters, it will be necessary to upgrade the tenant to a service plan whereby such configurations are supported.
    For details on how to upgrade the service plan, see the section 'Service Plan Upgrade' in SAP Note 2903776 - SAP Integration Suite – Service Plans and Upgrade Paths
