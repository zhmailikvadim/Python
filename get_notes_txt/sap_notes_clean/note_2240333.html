Symptom

The next Support Package introduces improvements in CRM-IU. To implement these improvements, changes to the existing source code are required.

To make sure that you can implement subsequent SAP Notes without problems, the existing and the new source code are decoupled. As a result, we document the changes to the existing source code in this interface note as well as in the related correction instructions.

Other Terms

This interface note is a preparation note that does not cause any interruptions.
If this SAP Note is required for subsequent corrections, you should implement it.
For the validity of the corrections, refer to the correction instructions.

Solution

Implement the attached correction instructions or import the specified Support Package.
