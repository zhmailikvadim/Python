Reason and Prerequisites

Error in coding.

Solution

The correction is delivered with the kernel archives:

  • ENSERVER patch - enserver.sar
  • SP stack kernel files SAPEXE.SAR and SAPEXEDB.SAR 

The corrections are contained in all patch levels that are equal to or greater than the patch level specified in the "Support Package Patches" section for the relevant kernel release.
If there is no entry there, the patch has not yet been delivered. The patch level will be entered automatically once the patch is available for download. You can subscribe to this SAP Note to be notified when it is updated.

Follow our recommendations for applying a kernel patch:

  1. Import the latest SP Stack Kernel if it already contains the corrections. For a list of current SP Stack Kernels, see SAP Note 2083594 (SAP Kernel Versions and SAP Kernel Patch Levels).
  2. Apply the hotfix only if it is a serious error that is not yet corrected by the latest SP Stack Kernel. 
  3. Check the regression note for the desired patch level before applying the kernel patch. For detailed information, see SAP Note 1802333 (Finding information about regressions in the SAP kernel).
  4. For instructions on how to download and apply a kernel patch, see SAP Note 19466 (Downloading SAP kernel patches).

For detailed information about our recommendations, see the strategy document: Update Strategy for the Kernel of the Application Server ABAP in On Premise Landscapes.
