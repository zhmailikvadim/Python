Symptom

You logon Manage Location Clusters Fiori app in English and edit a location cluster set. In Clusters tab, you select two clusters and click Compare button. In Cluster Comparison page, click Compare button, a graph appears. Click a point in the graph, the details of the Criteria and the cluster are displayed. This is the correct behavior. If you logon the Fiori app with a language other than English, the details of the Criteria and the cluster are not displayed.

Other Terms

Manage Location Clusters, Compare

Reason and Prerequisites

This is a program error.

Solution

Implement this note.

** Changes to the content of the SAPUI5 ABAP repository require the index to be updated using the calculation report '/UI5/APP_INDEX_CALCULATE'. When the correction instruction is applied via a note, please ensure this report is executed either manually or by an existing scheduled periodic background task after the note is applied. See note: 2227577 - Recalculation of application index after implementing a note.
