Symptom

You receive an error: "Error loading template" when the BI report is being executed in WebUI.

Environment

  • SAP Customer Relationship Management 7.0
  • SAP Customer Relationship Management 7.0 on SAP enhancement package 1 for SAP NetWeaver 7.0
  • SAP enhancement package 1 for SAP CRM 7.0
  • SAP enhancement package 2 for SAP CRM 7.0
  • SAP enhancement package 3 for SAP CRM 7.0
  • SAP enhancement package 4 for SAP CRM 7.0
  • SAP enhancement package 2 for SAP CRM 7.0, version for SAP HANA

Reproducing the Issue

  1. Log on the Web UI in the CRM system
  2. Select one business role
  3. Select the name of the Web report and click on it
  4. Afterwards the following error is shown: Error loading template 0TPLB_CRM*

Cause

BI system cannot be reached as Standard BAPI destination is improperly configured

Resolution

  1. Go to Tcode CRMC_UI_BI and double click on 'templateID'
  2. Remove 'logical system name' from the field BI System
  3. Go to Tcode BD97
    • Select logical system CRMBWDEST when configuring default BI system or
    • Select the BI logical system name after the displayed report
  4. Click on "Add Standard BAPI destination" and assign the coressponding BI 'logical system name' to CRMBWDEST.
