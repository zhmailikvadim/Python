Symptom

  • Status information in the "Sort by" section of the My Reporting dialog is read aloud incorrectly by screen readers (e.g., JAWS).
  • "Is active" is announced for all list items, even those not selected, causing confusion.
  • The status "Is active" should only be read aloud for elements that are actually selected.

Environment

SAP UI5 Controls Balkan team

Reproducing the Issue

  1. Navigate to the "Sort by" section in the My Reporting dialog.
  2. Use a screen reader (e.g., JAWS) to navigate through the list items.
  3. Observe that "Is active" is announced for all items, regardless of selection status.

Cause

Resolution

  1. Update the SAPUI5 version to the latest patch, specifically version 1.120.41.
  2. Verify if the issue persists after the update.
  3. Refer to the following note for guidance on performing the patch update:

SAP Note 3155948 - ABAP SAPUI5 patch version update.

See Also

  • SAP Note 3155948 - ABAP SAPUI5 patch version update
  • SAP Note 2346367 - How to upgrade the SAPUI5 library in ABAP
