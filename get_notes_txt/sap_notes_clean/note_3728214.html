Symptom

When IW31/32 SAP transaction code is called in SAP GUI and try to call Ariba Catalog it does not open and only a blank screen opens. 

Environment

  • SAP S/4HANA 2021
  • SAP S/4HANA 2022
  • SAP S/4HANA 2023
  • SAP S/4HANA 2025
  • 2025 (Private Cloud)
  • 2025 (On Premise)

Reproducing the Issue

  • Call SAP transaction IW31/IW32
  • Create Order example Order type PM01
  • Components Tab 
  • Call Ariba Catalog 
  • Try to Logon with the User 
  • Blank Page opens not able to logon to Ariba Catalog 

Cause

When secure OCI is used from GUI example SAP transaction IW31/IW32 connecting to ARIBA for security purposes unless we have a user in ARIBA it will not allow us to login.

Resolution

Maintain SAP User ID for the Ariba User. Make sure Ariba User exist. 

Keywords

SAP User ID, Catalog, Ariba , IW31 call Catalog
