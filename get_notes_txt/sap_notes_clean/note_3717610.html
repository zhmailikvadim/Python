Symptom

You are processing chargeback transaction in an advice using SAP Digital Payments add-on and the memo line is empty in the bank statement as they are not automatically processed.

Other Terms

FCC_DP_ADVICE_V2, chargeback

Reason and Prerequisites

The issue can occur due to one of the following reasons:

  • The PSP Adapter does not support automatic chargebacks.
  • The original charge cannot be found in S/4HANA using the provided transaction ID because the charge was created externally. In this scenario, the charge is initiated outside of the Digital Payments add-on, so the transaction ID referenced in the chargeback transaction of an advice does not exist in the S/4HANA system. As a result, automatic processing of the chargeback using the transaction ID fails, since the system cannot match the chargeback to any internal transaction record.

Solution

Program error. Apply the attached correction instruction.
