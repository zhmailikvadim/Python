Symptom

Incorrect mapping of XML namespace in SWIFT MX outbound file.

Other Terms

XMLNS, NSURI, SEPA XML, Namespace.

Reason and Prerequisites

Issue was is side effect of fix made in 3678155 and 3684294.

Solution

If Document tag already contains the namespace attribute, the implemented in above mentioned notes is to be skipped.

Implement the correction instruction attached to this SAP Note. Alternatively, you can import the referenced Support Package.
