Symptom

The statistical report RPSSBKS0 is designed to read time data from the Infotype 2010. In case of customizing assigned with parameter N data should be read for current period. This is not working, data from IT2010 is not read properly

Other Terms

SCB Statistics, RPSSBKS0, EC time

Solution

Reading data from IT2010 in report RPSSBKS0 was changed to properly read the data from current period when assigned with parameter N in form in table T512E.

Version Date Reason
01 2025-11-07 Released for all customers
