Symptom

  •  Sort applied on a block in a Web Intelligence report is not removed when the sorted object is replaced by drag and drop.

Other Terms

WebI, webintelligence, html, dhtml, java, applet, rich, client, wrc

Reason and Prerequisites

Environment:

  • SAP BusinessObjects Business Intelligence platform 4.2
  • Web Intelligence

Steps To Reproduce:

  1. Logon to BI LanchPad or Web Intelligence Rich Client
  2. Open the Formatting Sample demo document
  3. Apply a descending Sort on Margin
  4. Drag and drop the Quantity Sold measure from the Available Objects on the table to replace Margin
  5. Open the Sort dialog box
    1. >> The Sort on Margin is still applied to the block

Cause:

  • This issue is caused by a defect in the product.

Solution

This issue is fixed in the patches listed in the "Support Packages & Patches" section below. 
The "Support Packages & Patches" section will be populated with the relevant patch levels once they are released.
