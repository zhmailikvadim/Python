Symptom

Thailand branch code change log history is not displayed in RFDABL00 or RFKABL00 after upgrading the system from ECC to S/4HANA.

Environment

  • SAP S/4 HANA
  • SAP S/4 HANA FINANCE

Reproducing the Issue

  1. Upgrade SAP ECC system to S/4HANA system.
  2. Migrate vendor/customer  master data from ECC to S/4HANA system by transaction code FITH_BC_MIG (Branch code migration for Thailand).
  3. Access the report RFDABL00 or RFKABL00.
  4. Observe that all changes of the Thailand branch code change log history were missing.

Cause

It is standard behavior. If no new changes related to the branch code were made in BP after migration, no change log will be shown in the report RFDABL00 or RFKABL00.

Resolution

  1. Confirm that the issue is due to standard system behavior.
  2. If new changes related to the branch code are made in BP after migration, the change log will be displayed in the report RFDABL00 or RFKABL00.
