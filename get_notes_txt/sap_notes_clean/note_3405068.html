Symptom

Navigate to the Activities screen when current language is Portuguese, but the application still displays label contents in English

Environment

  • SAM 2105 with MOBASSTMGRAND00_0-80006144 client
  • Device: Android SAMSUNG S20FE

Reproducing the Issue

  1. Enables functionalities of meter component, like disconnect the meter
  2. Navigate to the Activities screen when current language is Portuguese
  3. The application still displays label contents in English

Cause

  • Product issue

Resolution

  • Deploy new definitions to replace RelatedActivities.page and localization files in the definitions.zip file attached

Keywords

Asset Manager, Meter, Translation, Translated, Language, Portuguese, English, Activities
