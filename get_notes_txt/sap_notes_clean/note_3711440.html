Symptom

This KBA describes the steps on how to update the Operator (Recruiter) field for a Job Requisition using Integration Center.

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental." 

Environment

SAP SuccessFactors Recruiting Management

Resolution

Note: The Operator Team is not upsertable and updateable through Integration Center.

  1. Create a CSV UTF file with the minimum mandatory fields: Job Req ID, UserSysID, Username, and isDraft:

  2. Upload the file in SFTP Server.

  3. Go to Integration Center → Create → select Scheduled CSV Input Integration.

  4. Search for JobRequisition under Entity Name and select it.

  5. Click Configure Fields → Upload Sample CSV, then upload the CSV created in Step 1.

  6. After uploading, ensure to follow the mapping with destination fields and set Operation to "Upsert Multiple": 

    • Under the JobRequisition entity:


      isDraft:

    • Under the Operator (Recruiter) Navigational Property:



  7. Configure the Source Settings according to the SFTP credentials where the file is uploaded.

  8. Save the integration and click Run to execute the job.
