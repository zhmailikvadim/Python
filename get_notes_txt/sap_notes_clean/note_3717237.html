Symptom

When working with ocean freight bookings, specifically when you add or change the 'Equipment Group' or 'Equipment Type' for a local container, the 'Quantity' field of the associated container item automatically resets to '1'. This behavior occurs even if a different quantity was previously maintained.

This issue is observed as a side effect following the implementation of SNote '3445725 - Inconsistencies in Ocean Freight Booking between Container Resource and Equipment Type'.

Other Terms

DET_ITEM_TR_AM, QUA_PCS_VAL, QUA_PCS_UNI, EQUI

Reason and Prerequisites

Program error.

Solution

Please implement attached correction instruction or the corresponding Support Package.
