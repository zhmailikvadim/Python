Symptom

After a system update it is not possible anymore to add an IBAN in a one-time payable/receivable item, after the account number has been entered. Instead of the IBAN popup, an empty Bank Details Assistant screen does appear.

Other Terms

FB01, FB02, FB60, CPD, BF00 020, BF00020, AR 034, AR034

Reason and Prerequisites

The correct approach is to enter either only an IBAN or only an account number. Before the update it was still possible to add an IBAN to the account number, activating the EXIT_SAPLFCPD_001. This workaround however does not work anymore, due to a program error.

Solution

Update: To avoid program code inconsistences, make sure the Note 2465413 is applied in your system, before applying this Note.

Apply the correction using the correction instructions or the corresponding support package.

Inform the affected user, that either the IBAN or the account number has to be entered. Not both the information.

It is strongly recommended to inactivate the user-exit EXIT_SAPLFCPD_001. The system will then raise the error message AR 034 (Enter an account number or an IBAN), to inform the user about the right procedure.
