Symptom

When attempting to post the Collective Agreement Request from the work bench, the system raises the error message “You are not authorized to post.” As a result, the posting process cannot be completed successfully.

This note ensures that the Collective Agreement Request can be posted successfully.

Other Terms

/IRM/IPAGRM, collective_req_post, /IRM/IPAGR 230, Collective Request Post, V_IP_SAGRQ

Reason and Prerequisites

This is caused by Program error.

Solution

Please implement the correction instructions. These corrections are only valid if you have applied the following Support Packs.

Release V/4 – 2023 V3 and 2023 V3 Support Package 1

Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object
