Symptom

Here is some information about the compatibility between SAP Process Integration/Orchestration (PI/PO) with SAP S/4 HANA system (On-premise system / Cloud Private Edition).

Environment

  • SAP S/4HANA
  • SAP NetWeaver 7.5
  • SAP Process Integration
  • SAP Process Orchestration

Resolution

  • SAP S/4HANA On Premise 2020 and S/4HANA Cloud Private Edition or higher is compatible with SAP PI/PO 7.50.
  • Converting an ERP System to SAP S/4HANA On Premise 2020 and S/4HANA Cloud Private Edition or higher, or upgrading one SAP S/4HANA On Premise and S/4HANA Cloud Private Edition 1709, 1809, 1909, 2020, 2021, 2022 and 2023 to the latest version of SAP S/4HANA On Premise and S/4HANA Cloud Private Edition (currently 2023) does not require to upgrade SAP PI/PO system, if this system has version 7.50. In case SAP PI/PO system is lower than version 7.50, you must upgrade SAP PI/PO system upfront to version 7.50.
  • Upgrading SAP PI/PO to version 7.50 does not require implementing a higher product version for SAP S/4HANA On Premise or S/4HANA Cloud Private Edition.
  • In version 7.50 the classical PI Dual Stack was replaced by a Dual Usage Type. ABAP and Java stack are running on separated instances. Therefore regarding SAP PI/PO 7.5, there are the following 3 options that can be considered. For more details, refer to SAP Note 1515223 - SAP Process Integration and SAP Process Orchestration: Release Recommendation for more details. 

    1)Dual usage type installation 
    2)Advanced Adapter Engine Extended (AEX)
    3)Process Orchestration
