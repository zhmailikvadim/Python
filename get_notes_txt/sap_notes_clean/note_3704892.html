Symptom

As part of S/4HANA 2025 FPS01, Business Add-Ins (BAdIs) were provided to allow customers to configure the logic used for automatic and manual scheduling with the Manage Schedules app.

As part of this SAP Note, the following improvements are provided in this area:

  • The BAdI definitions are included in SAP Customizing Implementation Guide under Plant Maintenance and Customer Service -> Maintenance and Service Processing -> Resource Scheduling.

  • Documentation was added for each of the BAdIs that can also be accessed from SAP Customizing Implementation Guide.

Other Terms

 F3326, Scheduling Logic Rules (BADI_RSH_LEVELING_RULES), Scheduling Logic Sequence (BADI_RSH_LEVELING_SEQUENCING), Scheduling Strategy (BADI_RSH_LEVELING_STRATEGY)

Reason and Prerequisites

 NA

Solution

Implement the automatic corrections as part of this Note.
