Symptom

When a production order/process order is realigned in the Capacity Scheduling Table or Schedule Production app, the error message “Order XXX could not be scheduled due to a technical error” appears in the log. The same error occurs during 'Save and Realign' and Dispatch actions.

Other Terms

Capacity Scheduling Table, Schedule production, F3770, Process orders, Capacity Demand Factor, return error code 59, Scheduling failure type 6 (No activity slot found matching all requirements)

Reason and Prerequisites

If calculated capacity demand factor of a phase/operation is greater than the value 1.000, then live cache returns an error code '59' and scheduling failure type 6.

Solution

Implement the attached corrections.
