Symptom

After you have implemented SAP Note 3729504, some fields in the CDS views /PLGA/C_TSO_EQUIPMENT and /PLGA/C_TSO_FUNCTIONALLOCATION have been replaced by constant values. This SAP Note replaces the constant values again with valid association paths.

Other Terms

Meter-to-Cash 2025 FPS01 Activation /PLGA/C_TSO_EQUIPMENT /PLGA/C_TSO_FUNCTIONALLOCATION

Reason and Prerequisites

In FPS01, in the CDS view I_LinearAssetManagementData, the association _LinearDataObjectType has been renamed to _ObjectType. The CDS views mentioned above are selected using the association path with the previous name. This resulted in the syntax errors and was temporarily corrected by SAP Note 3729504 so that FPS01 could be activated.

Solution

Implement this SAP Note. The two above CDS views should then work as usual again.
