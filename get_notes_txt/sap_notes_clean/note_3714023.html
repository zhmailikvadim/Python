Symptom

In the process "Sending of Request for Calorific Value and Volume Correction Factor (SUP View)" (DE_B_MET_SUP_CALV_VC), the AMOUNT_STATUS field in table /US4G/EADOC is not updated correctly according to the meter reading Quantity Determination status in steps 0380-0610.This issue is relevant for the market role: SUP.

Other Terms

AMOUNT_STATUS, /US4G/EADOC, QD, GAS, MET, Meter Reading Quantity Determination, calorific value, volume correction factor, DE_B_MET_SUP_CALV_VC, /US4G/CL_IM_BADI_DATEXCONN002, retrieve_gasf_eadoc

Reason and Prerequisites

The error occurred because the parameter es_gasf_eadoc was not set correctly in method /US4G/CL_IM_BADI_DATEXCONN002~retrieve_gasf_eadoc.

Solution

Implement the corrections of this SAP Note or import the relevant support package.
