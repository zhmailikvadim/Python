Symptom

After archiving data using the Financial Consolidation archiving tool, is it possible to restore the archived data?

Environment

SAP Financial Consolidation 10.1

Resolution

The archiving tool provides the ability to restore archived data. Once restored, the archived data can be accessed and utilized as needed.
