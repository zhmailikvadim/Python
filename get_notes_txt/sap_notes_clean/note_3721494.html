Symptom

Note 3697499 takes qals-gebeh and qals-anzgeb if Badi QPLEXT_LOT_CREATE makes modifications to them. This introduced a side-effect of overwriting standard properties with empty values.

Other Terms

GEBEH, ANZGEB, CA-QIE, EWM, ERP, QALS, QPLEXT_LOT_CREATE, SAP Note 2575361, Inspection Lot, MRP, ATP, S/4HANA, Note 3697499

Reason and Prerequisites

Programming error.

Solution

Implement Correction Instruction. Check was moved so empty values are not overwriting standard values with empty values.
