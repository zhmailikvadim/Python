Symptom

Error iaik.security.ssl.SSLException: Peer sent alert: Alert Fatal: handshake failure is received when check connection to a NON-SAP System 

Environment

SAP Cloud connector Release Independent

Reproducing the Issue

  1. Login to SAP Cloud Connector UI;
  2. Click on Cloud to On-Premises;
  3. Click on “Check Availability of internal host”;
  4. Check Result shows “Not Reachable”;
  5. Click on “Show Details” Icon;
  6. Error iaik.security.ssl.SSLException: Peer sent alert: Alert Fatal: handshake failure is perceived.

Cause

Backend system requests a certificate for connection and there is no certificate in Cloud Connector or the target host is not covered in the Subject CN DN or Subject Alternative names of the certificate

Resolution

Follow the steps from SAP documentation below:

Configure Trust | SAP Help Portal

Initial Configuration (HTTP) | SAP Help Portal
