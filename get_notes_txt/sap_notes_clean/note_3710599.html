Symptom

You are using Configuration Variants functionality of Integration & Exception Monitoring app in SAP Focused Run (FRUN).

You observe that you are unable to remove or delete any previously added filter parameter from the Configuration Variants->Monitoring tab. The value reappeared when the screen was refreshed after saving.

Other Terms

AIM, FRUN, Focused Run, Integration & Exception Monitoring, Configuration Variants, Variants, Monitoring Tab, Monitoring Filters.

Reason and Prerequisites

This issue occurs because of a program error.

Solution

Apply the attached correction instruction(s). After this fix, the filter parameters that you remove or delete in the Monitoring tab are saved successfully.
