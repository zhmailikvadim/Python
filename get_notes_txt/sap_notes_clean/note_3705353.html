Symptom

  • Refreshing the Growth Portfolio - Skills View page results in the word "Capabilities" being displayed for a brief moment.
  • This issue occurs every time the page is refreshed.
  • The issue is categorized as a minor UI glitch.

Environment

Growth Portfolio

Reproducing the Issue

  1. Navigate to the Growth Portfolio.
  2. Click "View All" on skills.
  3. Observe the "Skills" label at the top left of the page.
  4. Refresh the page and note that the word "Capabilities" is displayed briefly before reverting back to "Skills".
  5. Repeat the refresh action to confirm the behavior.

Cause

The issue was caused by a conditional setup where the default attribute type was displayed until the current attribute type value was loaded.

Resolution

Fix has been created and will be included in the next release, scheduled for April 13th for the test instance, May 15th for the production instance.
