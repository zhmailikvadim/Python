Symptom

When creating a Simple Rfx document in Product Sourcing I get an error: An error has occurred, please try again.

Cause

The message error points for a non existent template for Simple Rfx events.

Resolution

For you to create a template for Simple Rfx events, you should select the capacity type for the event as simple at template level.

To create an event with simple capacity type, follow the steps below.

1. Click Manage > Templates 

2. Click Actions > Create > Template

3. Select Sourcing Project > Quick Event

4. Choose Classic or Guided Sourcing and the type of event we desire

Now with the template created, go to Documents

1. Click the dropdown button next to the title of the document

2. Click Edit

3. Go to Capacity Type

4. Select simple

Publish template.
