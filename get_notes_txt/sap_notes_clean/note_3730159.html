Symptom

WARNING: This is an SAP Tax Declaration Framework Note. If you do not have SAP TDF installed in your system, this SAP Note does not apply to you. SAP TDF requires a specific license.

This SAP Note provides you with corrections to the EFD ICMS IPI (/TMF/EFD) report. This Note addresses the scenarios listed below:

  • The system does not generate register D761 due to UF filter restrictions.
  • The system does not generate register K200 correctly when using the Inventory Data (/TMF/D_INVENT) table for multiple entries of the Code of Controlled Book Account Charged/Credited (COD_CTA) field for the same Item Code (COD_ITEM) field.

Other Terms

TDF, Brazil, SPED, EFD ICMS IPI, D761, K200, INVENTARIO.

Reason and Prerequisites

This SAP Note is relevant only for Brazil.

You have implemented SAP Note 3721961.

You have upgraded your system to SAP TDF Support Package 20.

Solution

After implementing this SAP Note, the updates listed below are made to your system:

  • Register D761: The system removes the UF validation from the filter expression in register D761. As a result, the system generates register D761 correctly.
  • Register K200: The COD_CTA field becomes a key field in the /TMF/D_INVENT table and on the J_SHADOWING join node of the ctr/INVENTARIO view. As a result, the /TMF/D_INVENT table accepts multiple COD_CTA field entries for the same COD_ITEM field, and the system generates register K200 correctly.

As a general rule, SAP recommends that you implement the Support Package specified for your release. However, if you need to install a solution earlier, use the SAP Note Assistant to implement the correction instruction. You can find more information about the SAP Note Assistant in SAP Support Portal, under SAP Note Assistant. Note that documentation, transactions, and customizing activities are only delivered via Support Package.
