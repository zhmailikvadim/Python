Symptom

When attempting to create a billing document, the error "For down-payment requirements, only billing rule 4 or 5 can be used" occurs. 

Environment

SAP S/4HANA Cloud Public Edition 

Reproducing the Issue

    1. Open App "Create billing documents - VF01".
    2. Select Billing Type DPR1.
    3. Enter SD Document Number and save. A pop-up will appear: "No billing documents were created. Please see log."
    4. Click on "Log".
    5. Observe the error message: "For down-payment requirements, only billing rule 4 or 5 can be used. message VF217".

    Cause

    Missing configuration in SSCUI Maintain Date Category for Billing Plan Type.

    Resolution

    1. Access SSCUI Maintain Date Category for Billing Plan Type.
    2. Select the billing plan type (for example, 01 - Milestone Billing).
    3. Maintain billing rule 4 or 5 for the selected billing plan type.

    See Also

    Refer to: SAP Help Portal for further guidance on SSCUI configuration.
