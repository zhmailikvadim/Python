Symptom

Error message SG105: "Enter rate Curr1/Curr2 and rate type M for DD/MM/YYYY in system settings" is occurring.

Environment

SAP S/4HANA Cloud Public Edition

Cause

The Currency Exchange Rate mentioned in the error message is not maintained in the app 'Currency Exchange Rates'.

Resolution

Maintain Currency Exchange Rate in the App 'Currency Exchange Rates'.

In case the error message appears like "Curr1 / ", it is most likely that the system failed to capture the Local Currency or the Group Currency maintained, when creating the Currency Exchange Rate, use either the Group or the Local Currency.

Keywords
