Symptom

When you use the firm/unfirm functionality from the list report in F4170 Manage Planned Orders, the selected planned orders do not change and you receive an error message like 'Base UoM conversion failed for Material MRP Area'.

Other Terms

F4170, manage planned orders, planned order, PO, firm, firming, unfirm, uom conversion, base uom conversion failed, unit of measure

Reason and Prerequisites

The attributes of the planned order to be changed were being read from the wrong source.

Solution

This note ensures that unit of measure conversion is done using data from the database.
