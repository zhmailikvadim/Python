Symptom

Supplier Contacts data is not visible on the New Supplier Profile Summary page despite supplier users being available for the supplier in the application.

Reproducing the Issue

  1. Log in and click on Supplier Management tab.
  2. Search for a supplier and click on the Title.
  3. Observe that the Supplier Contacts card is empty.

Cause

Resolution

For Supplier Contacts to be available within the New Supplier Profile Summary, Supplier Data API should have the PrimaryContact field enabled.

Please refer to steps below:

  1. Manage > SM Administration > Config Fields.
  2. Select Type of data as Supplier.
  3. Select API as Supplier Data API.
  4. In the field below, select PrimaryContact.
  5. Click Save.
