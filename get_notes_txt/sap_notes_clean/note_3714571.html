Symptom

Error "XXX for Object payComponent is not a consistent value with the associated Object Country/Region with value XXX" pops up when saving compensation information.

Environment

SuccessFactors Employee Central

  • Compensation Information
  • Recurring Pay Components

Reproducing the Issue

  1. Edit any user's compensation information and click on save.
  2. Error "XXX for Object payComponent is not a consistent value with the associated Object Country/Region with value XXX" pops up.

Cause

Missing pay component data in manage data->the relevant country/region.

Resolution

  1. Navigate to the country/region referred to in the error massage.
  2. Add a pay component to this country/region.

Keywords

Pay component, country/region, japan, compensation data, recurring payments, error message, recurring pay component, manage data, missing data, missing pay component, XXX for object payComponent is not a consistent value with the associated object country/region with value XXX
