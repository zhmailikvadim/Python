Symptom

When executing CNV_NOTE_ANALYZER_MC_EXT for Migration Approach: Migrate Data Directly from SAP System, the error message "Screen output without connection to user" occurs.

Environment

SAP S/4HANA Private Cloud

Resolution

  1. In the Note Search Area of the CNV_NOTE_ANALYZER_MC_EXT program , it will list the RFC destination.
  2. Navigate to SM59 and open the relevant RFC.
  3. Check the user that is assigned in the RFC.
  4. Navigate to SU01 and open the relevant user.
  5. Change the user type from 'system' to 'service'
  6. Run the CNV_NOTE_ANALYZER_MC_EX program again.

Keywords

Direct Transfer, CNV_NOTE_ANALYZER_MC, Migration Cockpit, Note Analyzer
