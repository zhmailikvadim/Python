Symptom

  • An error occurs when executing a BOBJ report based on the Datasphere model with a date range prompt. The first run of the report in a new session returns the error:

    • "BW System returned state 'NO_DATA_AVAILABLE'. Message = Characteristic Summary Date does not support interval selections All selections for characteristic Summary Date are invalid and have been discarded"
  • The issue occurs when an interval/range selection is applied to the Summary Date characteristic, which supports only single-value selections.
  • After subsequent runs of the report no issue occurs.

Other Terms

Web intellignece, datasphere, refresh

Reason and Prerequisites

Environment:

  • SAP BusinessObjects Business Intelligence Platform 2025 Patch05 Version : 14.4.0.5460 
  • Datasphere
  • Web Intelligence
  • Windows 

Steps to Reproduce:

  1. Log in to BI Launchpad.
  2. Open the existing report based on Datasphere.
  3. Try to refresh the document.
  4. Notice refresh fails with below error message.

    • BW System returned state 'NO_DATA_AVAILABLE'. Message = Characteristic Summary Date does not support interval selections All selections for characteristic Summary Date are invalid and have been discarded

  5. Now refresh the report again and enter same date range prompt.
  6. Notice now the report is getting refreshed.
