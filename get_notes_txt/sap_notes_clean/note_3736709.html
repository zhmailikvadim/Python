Symptom

While generating a migration object in LTMOM, these errors are raised:

Synchronization failed; key fields have been changed for table <staging table name>
Error while altering staging table <staging table name>
Error during execution of SQL statement ()
ABAP Dictionary differences detected for object <migration object name>
Migration object <migration object name> cannot be generated due to DDIC differences

Environment

S4/HANA 2021 onwards

Reproducing the Issue

  1. Go to t-code LTMOM
  2. Select the migration object
  3. Click on the button Generate Runtime Object (top menu)
  4. Errors are raised in the bottom of the page

Cause

Modified migration object usually custom fields were added to the template.

Resolution

Use the standard template without adding custom fields to it.
This is most likely due to the migration object been modified. Unfortunately we from SAP support cannot provide any support on custom fields as they are not standardly delivered by us.

Please check in this page if the migration object you are using supports custom fields:
Available Migration Objects

Even if the used migration object supports custom fields, SAP support is not responsible for this.
