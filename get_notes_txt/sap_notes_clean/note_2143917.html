Symptom

An AO workbook is opened after it has been precalculated. A popup is displayed that the workbook needs to be repaired.

Some Styles of the query result set may be lost.

Other Terms

Scheduling

Reason and Prerequisites

The workbook contains "protected" cells, i.e. the cell format property "Protection" is activated. Please note, that the error occurs although the sheet itself is not protected. When deactivating the data grid property "Apply Style", in order to retain custom styles for the query, these styles are lost.

The error was introduced with SP 10, where the "Apply Style" property was activated for precalculation.

Solution


The correction will be delivered by a SAP BusinessObjects Analysis, edition for Microsoft Office BIP Add On Support Package.
In section "References" you can find a "...Release Note" whith the information which BIP Add On Support Package number contains the respective correction.
