Symptom

Transaction - /PRA/RP_WORKPLACE, /PRA/UTX_MASTER

New oil type code “3 – Light” is added for UTX master data. Also, a warning message is added if API gravity is greater than 44.1 and the oil type code is not equal to 3.

Other Terms

General keywords:  IS-Oil, Industry Solution, PRA, Oil and Gas Upstream, Production and Revenue Accounting, IS-OIL-PRA.

Problem-specific keywords: IS-PRA, S4CEXT, PRATXUT, /PRA/CL_UTX_LEGACY_DATA_MU, /PRA/CL_UTX, PROCESS_BUILD_REPORT, /PRA/CL_UTX_LEGACY_DATA_MU, VALIDATE_OIL_TYPE, /PRA/UTX_OIL_TYPE, oil type code, 3 - Light, API gravity.

Reason and Prerequisites

Program error.

Solution

Apply automatic correction instructions attached with the note.
