Symptom

You use one of the following CDS Views in a development object:

  • I_PhysInvtryDocItem
  • I_PhysInvtryDocHeader
  • I_MaterialDocumentHeader (on OP1909)
  • I_MaterialDocumentItem (on OP1909)
  • I_MaterialDocumentHeader_2 (from OP2020 onwards)
  • I_MaterialDocumentItem_2 (from OP2020 onwards)

When running the Package Check for the development object, you receive a Package Error, because the CDS View is not exposed in a Package Interface.

Solution

The assignment of the CDS Views to Package Interaces will only be delivered with the corresponding Support Packages.
