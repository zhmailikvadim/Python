Symptom

• The cost center used in position data is invalid.

• When adding a new record to a position via Manage data<Position or Manage Position, the inactive cost center does not appear in the dropdown. 

• Position data containing an inactive cost center can be imported without any error.  

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

SAP SuccessFactors Employee Central

  • Position Management

Resolution

This happens only when position data is changed via import.

Create an Evaluate save rules in Position Object to raise error in such scenario.

Configure it under Save Rules in Configure Object Definition<Position.
