Symptom

Component: Foundation for EHS Management
Module: Persons

When you open a location in the Manage Locations app and navigate to the Person Responsible tab, the system displays an error if you try to add a person whose name contains an opening bracket '('. The error states that the user doesn't exist.

Other Terms

CL_EHFND_BU_SU_CONV_EXIT_HELPR, IF_EHFND_BU_CONVERSION_EXIT, INPUT_CONVERSION

Reason and Prerequisites

A program error in the system causes the symptom.

You find the prerequisites in the relevant correction instructions.

Solution

After implementing this SAP Note, the system no longer displays an error when you assign a user with brackets in their name.

Alternatively, you import the corresponding Support Package for the component extension of SAP Environment, Health, and Safety Management or the Feature Package Stack for the SAP S/4HANA system.
