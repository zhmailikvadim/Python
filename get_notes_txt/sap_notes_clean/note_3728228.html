Symptom

When trying to reverse an other flow for a trade finance product in FTR_EDIT in the cashflow tab the reverse option is not available.

Environment

  • SAP S/4HANA
  • SAP S/4HANA Finance

Cause

Programming error

Resolution

Implement SAP Note: 3678177 - Missing Reversal mode for FTR_EDIT 

See Also

3678177 - Missing Reversal mode for FTR_EDIT

Keywords

FTR_EDIT, Trade Finance, Reversal
