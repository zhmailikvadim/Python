Symptom

  • While creating a Maintenance Planner plan, the following error appears: "The technical system [ABAP] contains the following add-ons that are compatible with SAP S/4HANA 2025 - 01 (02/2026) FPS. However, the add-ons are not entitled for your user. Please contact your SAP accounts executive, for further details on your software entitlement."
  • The add-on product version listed is "APPLIC INTERFACE FW EXT 01-APPLIC INTERFACE FW EXT".
  • The system contains "AIFGEN 700 0000 - Application Interface Framework License Check Tool".

Environment

  • SAP Application Interface Framework
  • SAP S/4HANA

Reproducing the Issue

  1. Start Maintenance Planner.
  2. Plan an update to SAP S/4HANA 2025.
  3. Observe the error indicating that add-ons are not entitled for the user.

Cause

The AIFGEN add-on (Application Interface Framework license check) is installed. Maintenance Planner detects the add-on and requires a valid entitlement; without it, the upgrade planning shows the entitlement error. 

Resolution

  1. Determine whether custom AIF interfaces are required:
  2. Run transaction /n/AIF/IDXTBL to review all AIF interfaces and namespaces in the system. Customer namespaces start without "/".
  3. If custom AIF interfaces are not required, remove the AIFGEN add-on. Refer to SAP Note 3082641 - Uninstalling the add-on AIFGEN.
  4. If custom AIF interfaces are required, obtain the appropriate entitlement/license for AIF by contacting the SAP accounts executive.
  5. After removing AIFGEN or resolving entitlement, rerun Maintenance Planner and repeat the upgrade planning.
