Symptom

You are using the Business Transformation Center and want to apply the latest corrections and features which are delivered as part of ST-PI 740 SP34.

Other Terms

BTC

Reason and Prerequisites

ST-PI 740 SP24 or higher

Solution

Please implement the Transport-based Correction Instruction (TCI) attached. This update includes bug fixes and ensures the application runs smoothly at all times.

Improvement Topics

  • Enhanced features for support/helpdesk applications

  • Auto-retry error handling with adaptive large portion/package size handling

  • Portion read endpoint: return empty response instead of error when no portions exist

  • PreselectionTask.Create - option to enrich auxiliary field with concatenated field values

  • auto-retry enhanced with package size adaption in case of memory related dump

  • Resolved cross-application race condition (with Portioning) in background job allocation

  • improved preselection report pattern choice

New features

  • Introduce new use case DataManagement to perform KPI-based Data Health Checks

  • PreselectionTask.Create - option to submit an allowed report instead of using SQL

  • Preselection Table Viewer - transaction to view contents of preselection tables

  • Preselection Data Read endpoint - enhance with option for GROUP BY

  • Preselection Data Read endpoint - enhance with option to read standard staging table

Note: To implement a TCI Note, the SNOTE transaction of the ABAP system must be enabled for the same. To enable SNOTE for TCI refer to the below resources.

2187425 - Information about SAP Note Transport based Correction Instructions (TCI) - SAP for Me

Cheat Sheet - Enabling SNOTE for Digitally Signed SAP Notes and for TCI
