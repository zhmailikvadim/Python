Symptom

  • The DRC Mexico Chart of Accounts report is generating the XML output file with element Desc in English.
  • The Accounts descriptions at master data is maintained in Spanish.
  • The user is logged in Spanish.

Environment

  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition
  • DRC Mexico Statutory Reporting

Reproducing the Issue

  1. Log in to the Fiori system using Spanish language.
  2. Attempt to generate the DRC Mexico Chart of Accounts XML output file.
  3. Observe that the XML file is generated in English instead of Spanish.

Cause

  • The language setup for the Financial Structure Version (FSV) within the global hierarchy is English.
  • This can be checked at table HRRP_ATTR_DIR, Attribute Name (HRYATTRNAME) for your FSV (HRYID) should be set as E (English).

Resolution

  1. At Fiori Launchpad, click on your Profile icon and click Settings.
  2. Click Language and Region.
  3. Change the Language and Region to Spanish.
  4. Click Save.
  5. Open the "Gestión de jerarquías globales" app / "Manage Global Hierarchies app" - Fiori ID: 2918.
  6. Search for the Financial Statement Version, i.e.: EAMX.
  7. Click "Desactivar" / "Deactivate".
  8. Click "Editar" / "Edit".
  9. In the dropdown box, change the language from "English" to "Spanish".
  10. Click "Grabar" / "Save".
  11. Click "Activar" / "Active".
  12. Run the Mexico Chart of Accounts Reporting again.
