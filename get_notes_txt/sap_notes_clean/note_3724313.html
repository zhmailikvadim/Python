Symptom

This SAP Note fixes an issue with the file names of vendor invoices that you retrieve from KSeF. The system generates two files for each invoice: an XML file with a unique name based on the KSeF number and a PDF file with a static name (SUPPLIER_INVOICE_LOCAL_FORMAT_PDF_PL). The static name of the PDF file makes it difficult to identify, unlike the XML file.

This SAP Note fixes the naming of the PDF file so it follows the same naming convention as the XML file. Electronic documents that you created before you implement this note don't have PDF files with unique file names.

Reason and Prerequisites

For a complete prerequisite list, refer to SAP Note 3279206 (Electronic Document Processing for Poland - KSeF: Implementation Overview).

Solution


After you implement this note, PDF files on your system follow the same naming convention as the XML files.
