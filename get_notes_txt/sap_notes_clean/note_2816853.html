Symptom

The OPS delivery request is processing in EWM, and the OPS document is initially created, but the staging bay is not determined automatically by the system, however the necessary entry is created in /SCWM/PSASTAGE for the appropriate Entitled/PSA pair. This entry contains the bin, but it is not determined. Additionally you can experiance different behaviour on different application servers.

Other Terms

Staging bay, staging bin, OPS, PDO, movement control, 504, 507, BOPF, /BOPF/CONF_UI, /SCWM/CL_DLV_MOVE_CTRL_I->/BOPF/IF_FRW_DETERMINATION~EXECUTE,  /SCWM/CL_DLV_STAGDATA_DOOR_I->DO_EXECUTE

Reason and Prerequisites

Missing BOPF determinitation dependecy

Solution

Implement the manual correction instruction, or the equvalent support package.
