Symptom

Component: Product Safety
Module: Report shipping

You used the IMG activity "Specify Parameter Values to Be Saved" to specify parameters not to be deleted when setting a report shipping order to "Historical". The system does not transfer these parameter values to the corresponding subsequent shipping orders.

Other Terms

SUB_CALL
CVDDP
Parameter
Parameter values

Reason and Prerequisites

Reason
Program error. Only the parameters marked as key parameters are taken over, i.e. the parameters specified in the IMG activity are not taken into account when creating the subsequent shipping order.

Prerequisites
For information about the validity of the corrections, see the correction instructions.

Solution

The section "Support Packages" specifies the support packages that contain the corrections.

Alternatively, you can implement the attached correction instructions.
