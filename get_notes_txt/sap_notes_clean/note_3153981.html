Symptom

Under the "Help Us to Know You Better" card to complete the Personal Data Collection task, Additional Data Collection task and e-signature task for data collection etc. Is it possible to hide if any of these tasks are not used in the process?

Environment

SAP SuccessFactors Onboarding

Reproducing the Issue

  1. Initiate onboarding
  2. complete review new hire data
  3. try to complete personal paper work step
  4. Click the ‘Help Us to Know You Better’ tile

Resolution

  • It is not possible to hide task which is not part of the business process.
  • If the tasks are skipped then it can automatically be marked as skipped but it is not configurable to hide.
