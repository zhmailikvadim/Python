Symptom

For old dropdown listbox (DDLB) without Type-ahead feature (the input field of the DDLB is editable so you may type partial text to search for DDLB entries for selection), you have to click twice to open the listbox.

Other Terms

DDLB, Dropdown Listbox, Typeahead, Type-ahead, Find-ahead, Findahead.

Reason and Prerequisites

This is a program error.

Solution

Please implement the attached Correction Instructions or the mentioned Support Package.

WARNING: This Note contains JavaScript code.

If you do not have Note 1855979 and 1920638 or equivalent SP installed, then please do not forget to erase your browser cache (temporary internet files) and server cache, and close all local browser instances before restarting the application again.

To delete caches, please check with Note 2050322.
