Symptom

While trying to retrieve logs in the Reprocess Bank Statement Items app (F1520), there is an error shown: Unable to retrieve log data

Environment

  • SAP S/4HANA
  • SAP Fiori

Reproducing the Issue

  1. Go to Reprocess Bank Statement Items app (F1520) app
  2. Select a line item and navigate to the log section
  3. Error is shown: Unable to retrieve log data

Cause

This is due to insufficient authorizations to access the log details, which can be checked using transaction code SU53

Resolution

  1. Add the relevant authorization objects as per SU53 transaction code
  2. Reset metadata and cache according to KBA: 2319491
  3. Follow steps from KBA: 2449635 to avoid cache issues and re-calculate UI5-index
