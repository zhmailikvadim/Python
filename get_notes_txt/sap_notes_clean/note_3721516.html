Symptom

Element nsSAFT:UOMTableEntry occurs 0 times. Number of occurrences must be >= 1.

Environment

•Financial Accounting (FI)

•SAP S/4HANA Finance

•SAP S/4HANA

Reproducing the Issue

  1. Access the Run Statutory Reports app.
  2. Attempt to generate the BG_SAFT_MONTH report.
  3. Observe the error messages indicating missing occurrences of specific elements in the XML file.

Cause

Bulgarian Base Conversion Factor field in T-Code "V_BGSAFT_UOMMAP" is either empty or set to zero.

Resolution

      Ensure the Bulgarian Base Conversion Factor field in "V_BGSAFT_UOMMAP" is not empty or zero.
