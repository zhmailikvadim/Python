Symptom

  • Users encounter the error message "application error cannot read properties of undefined reading 'getselecteditem'" when opening any model in SAP Analytics Cloud
  • The error occurs when users navigate to a new story and select a model
  • Users are able to view the model and data under Data Analyzer without encountering any errors
  • Removing all Data Warehouse-related roles in SAC from the affected teams or users resolves the issue

Environment

  • Story Design & Visualizations

Reproducing the Issue

  1. Open story and select any model as a data source
  2. Observe error message
  3. Navigate to a model and observe the same error message appear "application error cannot read properties of undefined reading 'getselecteditem'"
  4. Remove all Data Warehouse roles from the affected user and/ or remove the user from the teams with any Data Warehouse roles

Cause

  • The issue is currently under investigation by development 
  • As a workaround ensure that users and the Teams that the user is assigned to have any Data Warehouse roles in SAP Analytics Cloud removed

Resolution

  • The resolution is under investigation by development
