Symptom

Usually each project member will focus on one object link type which is essential to his/her tasks, e.g. the plant material object link type is essential for design tasks. There shall be a separate tab which just contains data of one essential object link type.

This SAP Note create the interface IF_DPR_OBL_OPERATIVE for the new separate tab which is implemented in note 2180813.

Other Terms

Operative Object Link, Operative Object Link Tab, Object Links, Interface Note
