Symptom

  • Inquiry about the existence of an OData API provided by SAP for importing data into the table CATSDB.
  • Existing BAPI (BAPI_CATIMESHEETMGR_INSERT) is known, but an API with similar functionality is preferred for implementation.

Environment

  • SAP ERP 
  • Time Sheet

Resolution

Currently, there is no OData API provided by SAP for direct CRUD operations on the CATSDB table.

The supported approach is to use the existing BAPIs exposed via custom wrapper services if API-based consumption is required. The following Tier 2 BAPIs are available for use:

  • BAPI_CATIMESHEETMGR_INSERT
  • BAPI_CATIMESHEETMGR_CHANGE
  • BAPI_CATIMESHEETMGR_DELETE
  • BAPI_CATIMESHEETRECORD_GETLIST

SAP plans to evaluate and potentially develop dedicated OData APIs for CRUD operations in the timesheet. This initiative is tentatively included in the roadmap for the last quarter of the year, subject to feasibility assessment and planning considerations.

For further details on the currently supported BAPIs, refer to the Cloudification Repository Viewer: Cloudification Repository Viewer.

It is recommended to follow up toward the end of the year for updates on the roadmap status.
