Symptom

This KBA guides you how to output wage types for Child and Child-Rearing Support Fund Contributions in HR Forms, after implementing SAP Note 3714453 - LC2026: PE51 Forms for Child and Child-Rearing Support Fund Contributions.

Environment

  • Human Capital Management (HCM)
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP SuccessFactors Employee Central Payroll
  • SAP S/4HANA

Resolution

  1. Edit HR Form: SAP_PAYSLIP_JP via transaction HRFORMS.
  2. Add Wage types from InfoNet DEDUC_LEGAL -> WAGETYPE -> Selections tab:
  3. Add Wage types /607, /617, /627 to the appropriate positions.
    • /607 should be placed after /315.
    • /617 should be placed after /345.
    • /627 should be placed after /365.
    • Please note that "HR form: SAP_PAYSLIP_JP" is just an example. Please be aware that the content and order of wage types in your system might be different from what is shown in the screenshot.
