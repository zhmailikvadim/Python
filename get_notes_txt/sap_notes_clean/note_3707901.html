Symptom

In the "Manage my Timesheet" application, an error message "Enter the valid work package ... details" is received on "Save" or "Save & Submit". 

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Attempt to make "Save & Submit" time entries in "Manage my Timesheet" application.
  2. Encounter the error message "Enter the valid work package ...  details".

Cause

  • There is a mismatch between work item assignment in task and work package role, leading to inconsistencies in task and work item linkage.
  • Example: The task settings still display a work item, but the work item assignment has been removed from the corresponding role in the work package.

Resolution

  • Make sure that an appropriate role—including the required work item—is assigned to the work package.
  • Example: Restore the original work item assignment to the relevant role within the work package.
