Symptom

When trying to perform a change, of any kind, in a sales order which was created automatically from a solution order, of type SOR, the following error is displayed:

  • 'You cannot process a sales order of type SOR (solution order).' (Message no. V2 449)

The attempt of change can be done directly, e.g. via app 'Change Sales Orders' (VA02), or indirectly, via any other app that would edit any field in the sales order, e.g. 'Manage Documents - Trade Compliance' to remove or set a compliance block.

Environment

SAP S/4HANA Cloud Public Edition

Cause

Sales orders of type SOR are not changeable in sales. The creation of display-only sales order of type SOR has been deprecated; however, any existing sales orders of type SOR remain unchanged and unchangeable.

Resolution

Currently, in SAP S/4HANA Cloud Public Edition, solution order integration with sales was enhanced by allowing the generation of sales orders of type OR, standard orders. To activate changeable sales orders for solution orders, raise a support case to the application component: CRM-S4-SOL-SLO.

If the scope item Solution Order (4GT) is not in use yet, by activating it now, the system creates changeable sales orders of type OR (standard orders) by default.
