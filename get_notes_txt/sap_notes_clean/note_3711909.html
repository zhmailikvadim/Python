Symptom

  • In SAP SuccessFactors Onboarding, is it possible to change the label of standard onboarding steps (for example, the Data Collection step labeled “Provide Additional Data”) to a custom name?
  • Is it possible to change/customize disclaimers displayed in Onboarding Step? 

Environment

SAP SuccessFactors Onboarding

Resolution

Labels were previously changed via Manage Language in Provisioning. This functionality has now been migrated to the UI, allowing customers to perform the changes directly from the interface where they have access.

Kindly follow the KBA from the Platform team for detailed instructions. The general steps are outlined below: 2576546 - Using the Manage Language Feature to Change Labels in SuccessFactors

Steps:

  1. Access SuccessFactors.
  2. In the search bar, search for Manage Languages.
  3. Under the Locale list, select the desired language under Custom Translation.
  4. Search for the label you would like to modify.
  5. Please note that the search is case-sensitive. For example, if the label is "Provide Additional Data", it must be searched using the exact capitalization.
  6. Modify the label and add your Custom Translation.

Please note that changes may take some time to appear in the system. In most cases, the update is reflected within 24 hours.
