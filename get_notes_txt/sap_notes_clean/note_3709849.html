Symptom

When an administrator retrieves user login records via the User Login Report in Manage General Audit, they will notice that logout records are also included in the report. 

 Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

SAP SuccessFactors HCM Suite

Resolution

Q1: What is this record?

A1: This is a logout record. There are two types of Logout Methods: Time-Out and User-Initiated.

Time-Out is the record of automatic logout caused by session timeout

User-Initiated is the record of manual logout triggered by users.

Q2: If multiple tabs are opened and operated in the same browser, will the system record the logout corresponding to each tab? Is there a specific list specifying which modules' session records are included in the User Login Report?

A2: A logout will trigger the termination of various backend sessions. We cannot provide a specific list of which sessions generate records and which do not, as this depends on the design of each module. 

Therefore, the login report may not necessarily record the activities of all tabs opened by end users.
