Symptom

I received a notification regarding the migration of Managed Gateway for Spend&Network from the SAP BTP Neo environment to the SAP BTP Cloud Foundry environment, is it necessary to add IP addresses to my firewall allowlist? If so, what are the relevant IP addresses?

Resolution

If you are using SAP Cloud Connector and you leverage IP Allow Listing, it is necessary to add IP addresses to firewall allowlist. 

Use the following keyword to search for the list of IP addresses under Region column in the Regions and API Endpoints for SAP Cloud Foundry Environment resource:

  • For China data center, use cn40.
  • For United Arab Emirates (UAE) data center, use ae01.
  • For North America (NA) data center, use us10.
  • For European Union (EU) data center, use eu10.

All IP addresses listed under NAT IPs column and LB IPs column should be added to firewall allowlist.
