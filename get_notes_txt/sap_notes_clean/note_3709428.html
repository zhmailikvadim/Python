Symptom

Error CK157 "Company code &1: Release too early" is issued during price release in T-code CK24.

Environment

  • Financial Accounting (FI)
  • SAP R/3
  • SAP R/3 Enterprise 4.7
  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA

Reproducing the Issue

  1. Run T-code CK24.
  2. Enter the material, plant, period.
  3. Execute T-code CK24.
  4. The error CK157 is issued.

Cause

A cost estimate  is attempting to be released in a future period. The system compares the system date with the date for which you are trying to release to. The release of the cost estimate is only possible in the current period.

Resolution

SAP therefore recommends to only execute the release of the standard cost estimate when the system-date matches with the period for which
the release should take place. The release of the cost estimate is only possible in the current period.

Keywords

CK157, CK24, CK40N, Standard price release future, Release too early
