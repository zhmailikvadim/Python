Symptom

You want to use functionality of the Migration Workbench (MWB) on an SAP system with one of the following DMIS releases:

  • DMIS 2025

Other Terms

You want to use functionality of the Migration Workbench (MWB) with the DMIS 2025 add-on.

Reason and Prerequisites

MWB, DTL, replication, migration, selection

Solution

This SAP Note provides the list of corrections for the Migration Workbench (MWB) using DMIS 2025.

For an automated verification of all required corrections that are listed in the tables below, you can use the Note Analyzer. For more information, see SAP Note 3016862.
