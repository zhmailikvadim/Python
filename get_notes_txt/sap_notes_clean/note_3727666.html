Symptom

  • When releasing a Central Integration Scenario in SOAMANAGER, the following issue occurs:
    • Error when reading variant "X" of service definition "Y": You have no display authorization

Environment

  • ABAP Web Services
  • SAP NetWeaver
  • SAP NetWeaver Application Server for SAP S/4HANA
  • ABAP PLATFORM - Application Server ABAP

Reproducing the Issue

  1. Navigate to SOAMANAGER in the central system.
  2. Select the "Service Administration" tab.
  3. Open the "Integration Scenario Configuration".
  4. Click on "Search".
  5. Open the affected scenario.
  6. Click on "Release" twice and keep refreshing until the error occurs.

Cause

Missing authorization object S_BTCH_ADM for the user maintained at the management connection.

Resolution

Assign authorization object S_BTCH_ADM with field BTCADMIN as D for the user maintained at the management connection.

To check the user:

  1. in the central system, at SOAMANAGER, go to the Management Connections tab
  2. open Management Connections
  3. display the affected managed system and click on the Logical Port (starting with CON...)
  4. the username will be shown here
