Symptom

When creating an invoice using the Trading Execution Workbench (WTEW), the saved invoice document number does not appear in the message log.

Environment

  • Global Trading Management in S/4HAHA

Reproducing the Issue

  1. Create an invoice using WTEW. 
  2. Save the invoice document.
  3. Check the message log to verify if the invoice number is displayed. 

Resolution

Implement the following correction notes:

  • 3435628 - Creating an invoice with WTEW does not show the invoice number in the log
  • 3720339 - WTEW: Creating an invoice from PO does not show the invoice number in the log

Keywords

Invoice Verification, MM invoice, MIRO, CL_WB2_ACTION_LIV_CREATE_PO, CL_WB2_ACTION_LIV_CREATE_GR
