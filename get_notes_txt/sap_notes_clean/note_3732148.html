Symptom

It is not possible to add long text to an inspection plan in Fiori APP Manage inspection plans(F3788)

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Environment

  • SAP S/4HANA, on-premise

  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

In Fiori APP Manage inspection plans(F3788), there is no available button to add long text to an inspection plan.

Cause

This is by standard design.

Resolution

This is a missing functionality of the standard system currently.

Only the field ‘’Inspection Plan Description' is available for maintaining short text.

As a workaround, the APP Change Inspection Plan(QP02) can be used.
