Symptom

The following error message occurs when doing any posting in the system

No Target Chart Of Accounts is defined for company code '&&&&' 

Message no. F1307

Environment

  • Financial Accounting 
  • SAP ERP
  • SAP ERP Central Component
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA
  • SAP S/4HANA Finance

Cause

Check the entry in table T001O for company code in question.

The issue can be caused by missing entry in table T001O.

Resolution

Check the settings in table T001O (transaction OBA7) for a target Chart of accounts.

If logging is active in the client, then use SCU3 to see when and how the customizing was changed. If logging is not active, consider activating it.
