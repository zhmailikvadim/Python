Symptom

The purpose of this KBA is to explain how the user can check the query technical name where a specific Fiori report app is based on.

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

  • SAP BW/4HANA
  • SAP NetWeaver

Resolution

  1. Opening the Fiori report app, the standard design for report apps will appears
  2. After, go to tab "Query Information". Then you can check the "Query Technical Name" under section "Technical Information":
