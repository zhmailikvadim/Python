Symptom

Use Fiori Catalog name to find all the apps in it with the Pages it is maintained on.

Environment

SAP Fiori Launchpad 2023

Cause

How to find page id by Catalog

Resolution

There is no standard report that lists a catalog together with all its apps and the pages where they’re used.

  • Via FLP Content Manager CONF/CUST you can open the catalog to see its assigned apps in the detail view. For each app in the detail view you can use the "Display Usage in Pages" button to see where it appears. Note: this must be done per app.

  • Alternative for Pages:

Call transaction /UI2/FLP_TAB_BROWSER
Navigate to Pages > Page > Tile (Table /UI2/PGSECAITIC)
Filter the column TILE_CATALOG_ID with your catalog ID, and you’ll get the PAGE_IDs that use tiles/apps from that catalog.
