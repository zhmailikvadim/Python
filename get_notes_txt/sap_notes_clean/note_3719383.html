Symptom

You are trying to change payment related fields, for example the payment terms in transaction FB02. Although they are not open for input on the line item detail screen, the fields are highlighted. This behavior is misleading.

Other Terms

FB02, FB09, SAPMF05L, PBO_DYNPRO_MODIFIKATION, BSEG-ZTERM, BSEG-ZFBDT, BSEG-ZLSCH, BSEG-ZSLPR, APAR_EBPP_TYPE_PAYMENT_METHOD, SAPMF05L 0301, T042ZEBPP

Reason and Prerequisites

Program Error

The issue happens if the used payment method is maintained as Biller Direct payment method.

Solution

Implement the attached Correction Instruction (CI) or the relevant Support Package.
