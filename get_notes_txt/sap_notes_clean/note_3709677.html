Symptom

Web IDE is slow and freezing and needs to be restarted and restaged.

Environment

SAP Web IDE, personal edition

Reproducing the Issue

Open WEB IDE and create a new Calculation view or take an existing Calculation view and try to preview it but it is slow and it freezes.

Resolution

1. To restart di-builder use the below command (restart):

$ XSA restage-at-startup

$ XSA restart

These XSA commands restart not only XS advanced but also all applications.

2. To restage and restart di-core execute the commands:

xs restrage di-core

xs restart di-core

3. To restage applications bound to outdated runtimes to ensure the latest runtime version is used execute:

xs restage --outdated

4. To remove outdated application runtime not bound to any applications execute:

xs delete-runtime --outdated

5. To clear disk space occupied by stopped/crashed application instances execute:

xs delete-app-instances
