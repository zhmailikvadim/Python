Symptom

You want to know how to upload multiple Purchase Requests at a time.

Environment

SAP Business ByDesign

Reproducing the Issue

  1. Go to Purchase Requests and Orders work center.
  2. Go to Purchase Requests View.
  3. Enable All Purchase Requests in the drop-down.
  4. Click New Request Button.
  5. This is time consuming when it is required to create multiple Purchase Requests.

Resolution

Web Services is available to create Purchase Requests in bulk.

All the APIs in this link: https://api.sap.com/package/BYD/soap

The Purchase Request WSDL is under Manage Purchase Request : https://api.sap.com/api/PSM_ISI_R_II_SRM_PR_MBO/overview

Keywords
