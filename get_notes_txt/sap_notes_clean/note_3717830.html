Symptom

In Compare Authorized with Actual Materials (AAC) Fiori application, the stock and other actuals information is not displayed for FMPO's when the valid date of the assignments are in the past.

Other Terms

AAC, Compare Authorized with Actual Materials, Flexible Material Planning Object, FMPO, Material Indicator, Assignment, BAdI, UN, Unauthorized, Force Element, S/4 HANA D&S, Defense and Security.

Reason and Prerequisites

This is an enhancement.

Solution

Implement the attached correction instruction for the solution.

BAdI is provided to modify the results based on filter. 

BADI_DFS_AUTHZDACTL_COMPRN_V2->MODIFY_AUTHORIZED_BY_FILTER has to be implemented to modify the records displayed in the AAC. 

Example class (CL_DFS_AUTHZDACTL_COMPRN_V2) is provided in the BAdI - This has the sample code for displaying the UN records for FMPO's, ensuring it is not skipped due to assignments of the past (from the entered current date).

The sample code is present in the section - Sample Code 2.
