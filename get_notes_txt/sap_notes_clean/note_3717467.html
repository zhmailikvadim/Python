Symptom

  • Payment terms, payment blocks and payment methods are missing
  • The Master Data file upload was processed and the records were created successfully, but still no Payment terms are available to be selected in the invoices
  • The import of master data was successful, but the imported data was zero

Cause

Business System ID is missing in Master Data File Upload

Resolution

You must maintain a business system with the logical identifier of your Own SAP Cloud System (e.g. 0A1AAA1). In SAP S/4HANA Cloud Public Edition, you can access app Communication Systems, use the filter Own SAP Cloud System and find this value in communication system details.

  1. Navigate to 'Master Data File Upload | SAP Help Portal' app and click on 'Maintain Business System' button to add a new business system
  2. Enter the following information in Add/ Edit Business System
    1. Business System ID 
    2. Business System Description 
    3. Business System Type
  3.  Click Save changes

The Business System ID will now be available for selection in the Select Source System dropdown at the time of master data file upload.

  1. Upload the master data file (e.g. Payment Term) selecting the new system ID
  2. Once the job is finished successfully, replicate this data again but via the Schedule Import of Master Data | SAP Help Portal app
  3. Check the job logs
  4. The number of imported items should be the same as the Master Data file upload

After that the Payment master data should be available for selection in the invoices.
