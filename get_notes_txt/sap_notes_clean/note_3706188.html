Symptom

You want to implement important current correction notes that correct problems during the local currency changeover (LCC) in leasing.


With the Note Analyzer functionality in SNOTE, you can import the most relevant SAP Notes as XML files into transaction SNOTE and implement all SAP Notes relevant for your release and support package that have not yet been implemented. In order to use the Note Analyzer, you need to implement SAP Note 3200109 and fulfill the prerequisites mentioned therein. To implement the missing SAP Notes for RE-FX Leasing – Integration of Financial Accounting in Leasing, perform the steps described in the "Solution" section.

The last update was made on January 22, 2026. SAP Notes released later are not (yet) contained in the XML file.

Solution

Perform the following steps in your development system:

Step 1: Implement SAP Note 3200109 if the note has not yet been installed

Step2:Go to transaction SNOTE. In the menu, choose "Goto -> Other Tools -> Launch Note Analyzer".

Step 3. Upload the XML file attached to this note. You can now process the SAP Notes using the Note Analyzer. For more information about how to use the Note Analyzer, see SAP Note 3200109.

Step 4: Test your processes and move the implemented SAP Notes to your production system after successful testing.

Please note that the XML file attached to this SAP Note might be updated when new important SAP Notes are created. So, to stay on the latest maintenance level, you can check from time to time whether a new version of the SAP Note is available.
