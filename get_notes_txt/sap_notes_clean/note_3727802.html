Symptom

The Offboarding Dashboard displays the process status as "Completed" even though the process is still open.

Environment

SAP SuccessFactors Onboarding.

Reproducing the Issue

  1. Terminate an employee in the system.
  2. Go to the Offboarding Dashboard.
  3. Search for the terminated user.
  4. Notice that the dashboard shows the status as "Completed" even before the employee starts the review step.

Cause

The Employer Data Review step is skipped for employer.

Resolution

The Offboarding Dashboard is showing the employer tasks, not employee tasks. When a user is terminated and the Employer Data Review step is skipped, there are no pending employer tasks, so the status appears as Completed. Once the employee completes their task and an employer task is generated, the status changes from Completed to On Track.

In conclusion:

  • If the Employer Data Review step is active in the Offboarding process, the status will show as "On Track."
  • If only the Employee Data Review step is active, the status in the Offboarding process will show as "Completed."
