Symptom

After posting goods movements for production order, go to CO03 menu "Go to" > "Documented Goods Movements" and no material documents entry are displayed here although the goods movements can be checked in table MSEG and the indicators about documentation of goods movements are all set in t-code OPL8.

Environment

  • SAP ERP Production Planning (PP)
  • SAP R/3
  • SAP R/3 Enterprise
  • SAP ERP Central Component
  • SAP ERP
  • SAP Enhancement package for SAP ERP
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA, on-premise
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Post goods movements for a production order through making confirmation or MM transaction.
  2. Go to t-code CO03 to check the material documents in menu "Go to" > "Documented Goods Movements".
  3. No entry displayed in "Documented Goods Movements" overview.

Cause

Not use standard layout to display the result.

Resolution

Firstly, check the relevant setting in OPL8 as KBA 1898443 described. If there is no missing customizing but issue still occurs, please use standard layout 000000000001 to display the result.

Follow the path in "Documented Goods Movements" overview: Click "choose layout" button > "Choose layout" option > click "000000000001 standard layout" (currently use other layout). Then the material document entries should be displayed in "Documented Goods Movements" overview.

Keywords

material document entry, COR3, COR4, goods movements documentation
