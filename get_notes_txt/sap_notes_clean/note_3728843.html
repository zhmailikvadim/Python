Symptom

In the Flexible Material Planning Object application, generic validation checks are not available for fields added via application extensibility.

Other Terms

Flexible Material Planning Object, FMPO, Variant, Models, Extensibility, Checks, Validations, S/4HANA Defense & Security.

Reason and Prerequisites

This is an enhancement for generic validation checks, that are not available for fields added via application extensibility.

Solution

Please implement the correction instruction for the solution. A new BADI 'BADI_DFS_FMPO_VLDT_EXTN_FIELDS' has been added for field level validation which is made available for all the sections of the FMPO application.
