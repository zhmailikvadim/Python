Symptom

Missing details on the Monitor Production Plan Performance app:
Location:
SAP S/4HANA -> Production Planning and Detailed Scheduling (PP/DS) -> Fiori Apps in PP/DS -> Monitor Production Plan Performance

 The online documentation is missing information about the key features of the app and supported KPIs. You can find this information in the Key Features and Supported KPIs sections in the attached .pdf file. The documentation also doesn't contain information on accessing the app. This information can be found in a separate .pdf file attached .

Other Terms

 KPI monitor, Resource capacity, KPI

Solution

The missing details about the key features, supported KPIs, and access to the app have been added to the attached .pdf file and will be published on the SAP Help Portal as of release SAP S/4HANA 2025 FPS02.
