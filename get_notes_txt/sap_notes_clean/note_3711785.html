Symptom

When trying to change an existing report in the reporting app, the data preview can prevent any activities in the edit screen when dealing with a high data volume in the track or if expensive enrichments are implemented in the used reporting pattern by the customer.

Other Terms

FCCOREUI ; Reporting ; preview ; runtime ; freeze ; report ; pattern

Reason and Prerequisites

Data preview is always getting loaded when entering the edit page of a report. This is getting disabled and the data preview is only triggered when the generate preview button is pressed.

Solution

Please implement the note and all of its prerequisits.
