Symptom

When running a DTP, duplicate or inconsistent records appear in the SAP BW system after loading data using a ABAP runtime transformation.

Environment

  • SAP NetWeaver 7.5x
  • SAP BW/4HANA

Resolution

  1. Implement all of the notes as per the note analyzer as per SAP Note 2603241 - Overview and summary of the most important SAP Notes in the context of BW transformations with SAP HANA and ABAP execution;
  2. Verify if the transformation contains more than one rule group besides the standard group and analyze if the key fields are mapped the same between the rule groups. In case non-cumulative is used, inventory data verify if the rule group consignment stock is used. Check if the consignment stock information is really necessary or not. If it is not necessary, it is possible to remove that rule group;
  3. Test the process by removing any InfoSources and linking the transformation directly to the source and target;
  4. Test by removing any Start, End or Field routines.
