Symptom

There is a need to confirm if SAC subaccounts mantained and managed from SAC Data Source Configuration portal are eligible for the auto certificate refresh option.

Environment

  • SAP Cloud Connector(SCC);
  • SAP Analytics Cloud(SAC).

Resolution

The subaccount auto certificate refresh option is built and limited to subaccounts running specifically within SAP BTP in a multi-cloud environment, therefore subaccounts managed from SAC Data Source Configuration are not eligible for the auto-renewal option.
