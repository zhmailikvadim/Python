Symptom

  • Issues when trying to activate SAP Field Service Management – Crowd Workforce.
  • The Crowd Workforce tile is visible in SAP FSM but the three tabs/sections are empty.
  • No entries can be selected or maintained.
  • No configuration or productive data can be created.
  • In the Partner Portal, only example/demo data is displayed.
  • A system message indicates that the data shown is not productive.
  • No real partner, technician, or service data can be configured.

Environment

FSM Crowd Workforce

Reproducing the Issue

1. Log on to SAP Field Service Management.

2. Navigate to Crowd Workforce.

3. Open the Crowd Workforce application.

4. Observe that:

  • All three tabs/sections are empty.
  • No configuration or data can be selected or created.

5. Navigate to the Partner Portal.

6. Observe that:

  • Only example/demo data is displayed.
  • No productive configuration is possible.

Cause

Missing Crowd Type value under Admin - Company - Maintenance - Crowd Type, preventing activation of Crowd Workforce functionality.

Resolution

  1. Navigate to Admin - Company - Maintenance - Crowd Type in the system.
  2. Update the Crowd Type value to "Single Tenant".
  3. Test the resolution by verifying the functionality of Crowd Workforce.
  4. Confirm that the three tabs/sections in the Crowd Workforce application are populated and allow configuration or data creation.
  5. Verify that the Partner Portal displays productive configuration and real partner, technician, or service data.
