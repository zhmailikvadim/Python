    https://help.sap.com/docs/connectivity/sap-btp-connectivity-cf/configure-access-control-rfc

Environment

  • CAP with Node.js
  • load-balancing RFC

Cause

CAP plugin implementation: CAP does not currently support load-balancing RFC. 

Resolution

  1. CAP plugin implementation: CAP does not currently support load-balancing RFC. 
  2. It expects to find "jco.client.ashost", "jco.client.sysnr" and "jco.client.client" and does not consider other properties for the RFC clients connection parameters.
  3. Customer can  raise this as a feature request via https://influence.sap.com/.
