Symptom

After a BPC upgrade (for example to SAP_BW 750 SP15), issues are encountered after running Data Manager package (using process chain /CPMB/IMPORT_MASTER or /CPMB/IMPORT_IOBJ_MASTER) to import master data in Update mode:

  • an ABAP dump GETWA_NOT_ASSIGNED is generated in ST22 transaction
  • the hierarchy information is cleared
  • changes are noticed in the structure of a different hierarchy than the one that is imported through the package

Environment

  • SAP Business Planning and Consolidation 10.1, version for SAP NetWeaver
  • SAP Business Planning and Consolidation 11.0, version for SAP BW/4HANA
  • SAP Business Planning and Consolidation 11.1, version for SAP BW/4HANA

Reproducing the Issue

  1. Log on to EPM Add-in
  2. In Data Manager, execute a package (using process chain /CPMB/IMPORT_MASTER or /CPMB/IMPORT_IOBJ_MASTER) to load master data in update mode

Resolution

Implement the following Notes:

  • Note 2801782 (Fix error when import master data in update mode)
  • Note 2814672 (Hierarchy information is cleared in update mode)

Keywords

IMPORT_MASTER, IMPORT_IOBJ_MASTER, "update mode"
