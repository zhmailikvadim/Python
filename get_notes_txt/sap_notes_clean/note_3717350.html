Symptom

  • Cannot able to Edit Standard PCR as per the note instruction
  • Unable to change the PCR due to "Person responsible - SAP" in attributes 
  • As per the note manual steps could not able to edit PCR due to error P0505 "Maintenance may only be performed by the person responsible for the structure" or
    P0511 "Only the person responsible may change responsibility for the structure". 

Environment

  • Human Capital Management (HCM)
  • SAP ERP Payroll India
  • SAP R/3
  • SAP ERP Core Component
  • SAP ERP
  • SAP enhancement package for SAP ERP

Reproducing the Issue

  1. Try to Edit SAP Standard PCR
  2. System does not allow to edit, due to "Person responsible - SAP" in attributes

Cause

Resolution

Please note there are two ways to resolve the issue :

1. Check the instructions given in the Note 0000131272, Steps to change the Standard PCR

or

2.

a. Copy the standard rules into Z-rules

b. Make changes in the Z-rule as per the Note.

c. Comment the standard rule in your payroll schema.

d. Insert the modified Z-rule in place of the standard rule in the payroll schema.
