Symptom

  • Onboarding login language is incorrectly set to English instead of German.
  • All candidates receive the value "German" when application data is saved, as per the business rule.

Environment

SAP Successfactors Onboarding

Reproducing the Issue

Log into SuccessFactors to find that the language is English instead of German.

Cause

Default Language in Provisioning is set to English.

Resolution

Go to  Company Settings  - Provisioning  - "Default Language" and set preferred language.
