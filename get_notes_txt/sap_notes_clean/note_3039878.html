Symptom

You are trying to add an extension field to the org unit screen, but cannot find this option in the Adaptation menu.

Environment

 SAP Cloud for Customer

Reproducing the Issue

  1. Go to the Organizational Management work center.
  2. Access the Org Structures work center view.
  3. Open any org unit.
  4. In the new screen, click the user icon at the top-right corner.
  5. Select option to Start Adaptation.
  6. Click the pencil icon over any section to edit the same.
  7. In the adaptation menu at the right side, click Add.
  8. There is an option to add a Mashup, but no option to add a Field.

Cause

According to the Organizational Management developers, the entirety of the Org Unit business object and its associated UI elements are not extensible as per the framework's design.

Resolution

This is currently the expected system behavior.

Keywords

Org Unit, Extension Field, Add, Unable
