Symptom

Component: Incident Management
Module: Customizing

In Customizing (Transaction SPRO) under Environment, Health and Safety -> Incident Management -> Injury/Illness Log, when you create customizing for the listed objects and run them, the system generates the following dump:

Category                          ABAP programming error
Runtime Errors                PERFORM_NOT_FOUND
Except.                             CX_SY_DYN_CALL_ILLEGAL_FORM
ABAP Program                 SAPLEHHSS_CUSTOMIZE2
Application Component   EHS-SUS-IM

Other Terms

 PERFORM_NOT_FOUND, CX_SY_DYN_CALL_ILLEGAL_FORM, SAPLEHHSS_CUSTOMIZE2, DISABLE_CONVERSIONS, EHHSSV_INC_WF_WI, EHHSS_BO_INC_CFG

Reason and Prerequisites

Reason:
Program error introduced by Note 3691583

Prerequisites:
For information about the validity of the corrections, see the correction instructions.

Solution

After implementing this SAP Note, the system no longer generates the PERFORM_NOT_FOUND dump when you create and run customizing for Injury/Illness Log objects.

The "Support Packages" section lists the Support Packages that contain these corrections. 

Alternatively, you can implement the attached correction instructions.
