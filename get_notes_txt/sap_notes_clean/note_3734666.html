Symptom

The error message "Pricing error: check intercompany pricing procedure" is a general error message, which is triggered after the invoice simulation in method IF_FVC_ICO_POST_INT_DATA_PROVI~GET_PC_TP_PRICE (class: CL_FVC_ICO_POST_INT_DATA_PROVI) and is misleading.

Other Terms

 IF_FVC_ICO_POST_INT_DATA_PROVI~GET_PC_TP_PRICE, CL_FVC_ICO_POST_INT_DATA_PROVI, error message, SEGVAL, Advanced intercompany, EC-PCA-TP

Reason and Prerequisites

 The error message was misleading and too generic

Solution

Enhanced the message handling and subtituted the generic message with more concrete one.
