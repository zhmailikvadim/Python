Symptom

This note contains information about SAP MII 15.5 SP02 release.

Other Terms

SAP MII, MII 15.5 SP02, OEE, MII OEE, workbench.jar, workbench.zip, file upload restriction, MII 15.5 SP01 Release note, MII release note

Reason and Prerequisites

SAP MII 15.5 SP02 release information , XMII, MII_ADMIN and MFG MII Content, OEE MII, OEE ERP INTERFACE 15.0, Catalog OData Services, MDO OData Services, KPI OData Services

 

Solution

SAP MII 15.5 SP02 contains following changes.

  • SAP MII is moving from Odata4j to Olingo Library for odata interface generation. Impact areas are MII IlluminatorOdata Interface, Odata Modeller, and all the OEE UI e.g. OEE dashboard, Configuration for worker UIs, Application Launchpad. For Illuminator ODATA format change, kindly check the note 3594355 
  • Security related correction for GHAS and Check Max issues. 
  • Applets executon in Microsoft Internet Explorer or Edge are no longer supported as per earlier communications note 2440728 - MII Visualization Applets and 3181292 - End of mainstream support for MII applets. If you are using applets then 1st move all those to UI5 based visualization objects and then upgrade to MII 15.5 SP02.
  • Addition of new stonger encryption algorithms i.e. AES-128, AES - 192 and AES - 256 in Security Services -> Encryption. Please take the back up of all the configuration and projects before doing MII 15.5 SP02 upgrade due to change in Encryption algorithm. 

  • Removal of EXIF data of pictures during uploaded unless a clear consent or other legal ground exist

  • Additional system property to maintain the allowed CSP header in MII. Check the note 3567906 - Content Security Policy addition as per product standard

  • Additional system property as Remove Exif data from JPG/PNG file (REMOVE_EXIF_METADATA) to Controls the removal of EXIF data from the image when uploaded in SSCE or MII Workbench. Check this note 3566054 - Option to remove Exif data from uploaded pictures

  • All corrections since MII 15.5 SP01 release.

Here are some of the other important change done in lower releases which may have impact when moving to 15.5 SP02

Please check note 2959802 - Remove User Attributes from PropertyAccessServlet which will have impact on your custom development as some of the user session properties are removed due to security reasons.

Please take the latest Software archives for XMII, MII_ADMIN and MFG MII Content from Service Market Place for MII 15.5 SP02 to use the latest features.

Some general information 

Please don't deploy MFG MII Content software archive on JAVA stack and OEE ERP INTERFACE 15.0 on ABAP stack if you are not using OEE functionality of MII product.

To check OEE add on (OEE_ERP 150 component) compatibility with different S/4HANA releases, kindly check the note 2214409 

Also it is not mandatory to upgrade OEE ERP INTERFACE 15.0 component if you are not using OEE functionality of MII while upgrading the JAVA component. If you are using OEE funtionality in MII then upgrading the OEE ERP INTERFACE 15.0 is important as it brings some of the correction done in latest SPs.
