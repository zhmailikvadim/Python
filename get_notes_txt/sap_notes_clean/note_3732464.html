Symptom

If it is set in the contract type that the accruals reversal takes place by new lines in the accruals index WCOCOF (TWCBCONTRTYPE-ACCR_INDEX_UPD_SETTL = '1'), the index date (WCOCO-BUDAT) is not filled for postings from the accruals reversal (WB2R_RAC / WB2R_RAV / WB2R_RAP). This does not lead to any problems during processing, but it is a hindrance in reporting. With this SAP Note, the date is filled with the validity end date of the contract in this case.

Reason and Prerequisites

This is due to a program error.
