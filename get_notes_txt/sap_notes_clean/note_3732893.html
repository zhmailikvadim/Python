Symptom

During output no data is being outputted when the field is in "Only Text" mode.

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

  • SAP R/3
  • SAP ERP Core Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA

Reproducing the Issue

Open any ad hoc query and output data where there is no corresponding text value given.

Resolution

With the help of report RS_TEST_IDENTIFY_TEXT, any HR table can be checked which field has a text field assigned.
After running the report, the table name needs to be written and select the bottom radio check.


After the output, it can be seen which field has readable text assigned to it. The source of the text is in the same row as the field.


To display the value in a specific field, change the setting from "Only Text" to "Only Value" after right‑clicking the field.
