Symptom

You declare your exports and imports to the Dutch customs system, DMS, using SAP GTS

You receive an error from Dutch customs with the message: SAMETYPE_SUPPLEMENTARYUNIT_INCOMPATIBILITY.

This SAP Note includes program enhancements for IDoc outbound processing, enabling weight values to be specified with up to six decimal places.

Other Terms

Global Trade Services, Customs Management, Dutch, Netherlands, DMS, SUPPLEMENTARYUNIT, WEINET, WEIGRO

Reason and Prerequisites

Program error.

Solution

Implement the attached correction instructions.

Update your mapping according to point 33 in Note 2300489.

By implementing the correction instructions and updating the mapping as per point 33, you should be able to declare weight values with up to six decimal places.
