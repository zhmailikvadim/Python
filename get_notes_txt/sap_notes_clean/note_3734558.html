Symptom

When BP telephone number is maintained, ADR2-HOME_FLAG gets updated with 'X'

Environment

  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition
  • SAP Netweaver

Resolution

This is standard system behavior. HOME_FLAG will always stay at 1st entry and will be moved to next entry if the 1st entry is deleted.  

FLGDEFAULT is expected to change as per the standard flag changes on UI. The telephone number which has FLGDEFAULT will be copied to ADRC. HOME_FLAG has no functional significance. FLGDEFAULT will be treated as default/standard entry.
