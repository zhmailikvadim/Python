Symptom

When trying to unload / Receipt HU by Delivery, system creates a new line item in the delivery document as it is not accepting packing multiple items into a single HU.

Environment

  • SAP Extended Warehouse Management (SCM-EWM)
  • SAP Extended Warehouse Management Radio Frequency (SCM-EWM-RF)

Reproducing the Issue

  1. Execute tcode /n/SCWM/RFUI
  2. Enter Warehouse no and Resource
  3. Unloading by Delivery / Receiving of Handling Units by Delivery
  4. Enter Inbound delivery No
  5. Create a new HU or use existing HU
  6. Pack same materials belonging to different line items in the inbound delivery document
  7. New line item is created in the delivery document, but system is not accepting packing multiple items into a single HU

Cause

Standard design limitation in the RF framework. Unfortunately packing of multiple items into 1 HU for unloading in RFUI is not supported.

Resolution

The RF Unloading/Receiving HU by Delivery will support only 1 item : 1 HU scenario.

Keywords

RF Unloading, Receipt HU by Delivery, Packing multiple line items, New line item created
