Symptom

Business role considered for provisioning in access request even if, role is in process of methodology without complete or approved.

Environment

Access Control

  • Access Request

Cause

Business role versioning not activated

Data in database table GRACROLERELACTV and GRACBRMVERSEXC were empty, which means versioning is not working and not considered, hence irrespective of state of Business role, it will be considered for provisioning in access request form.

Resolution

Execute program GRAC_SETUP_BRM_ACT_VERSN_TABLE from TCODE SE38

FYI - The above mentioned program should run only once and this program will process the data set up for existing business roles and get them ready for usage with versions. Once executed this report entries would be filled with versioned Business role and new roles which will be set to complete state.
