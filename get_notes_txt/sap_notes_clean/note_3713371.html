Symptom

In the process "Receipt of Request for MaLo Identification from SUP (DSO View)" (DE_B_GPKE_MALO_IDENT_RECV_DSO), technical resource–related data is not fetched and prepared in the UBD data in step 0300.

This issue is relevant for the market role: DSO.

Other Terms

GPKE, MaLo Identification, Technical Resource, UBD-RESOURCE, DE_B_GPKE_MALO_IDENT_RECV_DSO, API message

Reason and Prerequisites

The error occurred because of a program error.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the system fetches and prepares technical resource–related data in the UBD data.

You need to reimport the request API message and check the outbound response API message. You can verify the fix by checking that the outbound response API message contains data related to the technical resource.
