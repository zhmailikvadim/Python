Symptom

  • Error message displayed: "Enter valid executing server"
  • Message number: /CLMDV/DVF_TEST_SPEC090
  • Unable to maintain the executing server for source and target systems in the DTV tool during system maintenance.

Environment

  • Data Validation Framework

Reproducing the Issue

  1. Open the DTV tool.
  2. Open the project 
  3. Click on "Maintain Systems".
  4. Attempt to maintain SAP ECC as the source system and S/4HANA as the target system.
  5. Try to maintain the executing server for both systems.

Cause

  • The executing server for the SAP ECC system is not available in the S/4HANA system. Only the executing server available in the current system can be maintained.

Resolution

  1. Ensure that the executing server being added is available in the current system.
  2. For data validation using the DTV tool:
  • Extract source system data in the SAP ECC system prior to system conversion.
  • After system conversion, extract target system data in the S/4HANA system.
  • Perform evaluation once data extraction is completed for both systems.

     3. Upgrade to the latest DTV version in upcoming cycles. Refer to SAP Note ##3657674 for DTV Tool 1.7 delivery.
