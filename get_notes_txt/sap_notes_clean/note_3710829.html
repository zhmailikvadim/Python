Symptom

Via SAP Note 3697080 the customizing entries in the table:

  • Assign Description texts to BELCOTAX forms, BOW (T5BX6) 

have been provided for the fiches 281.xx for the income declaration of the income year 2025, to be declared in 2026:

The customizing entries are required to generate the fiche 281.xx using report FICHES 281.xx - Printing documents (RPCBWPB0).

The delivery of SAP Note 3697080 was based on the draft version of the fiches 281.xx published at that time by the Federal Public Service FINANCE, which were available in French and in Dutch.

In the meanwhile the final version of the fiches 281.xx has been posted in French, Dutch and German. For the French and Dutch version no differences to the draft version have been detected. For the German version many texts have been updated by this SAP Note 3710829.

Other Terms

RPCBWPB0, T5BX6, T5BX8

Reason and Prerequisites

Legal changes to the fiches 281.10, 281.11, 281.13, 281.14, 281.17, 281.18 and 281.20.

You can find the fiches on the Belgian government website using the following links:

  • https://financien.belgium.be/nl/ondernemingen/personeel_en_loon/bericht_aan_schuldenaars
  • https://financien.belgium.be/fr/entreprises/personnel_et_remuneration/avis_aux_debiteurs
  • https://financien.belgium.be/de/unternehmen/personal-und-lohn/mitteilung-schuldner

You can find instructions on how to fill the forms using the link http://minfin.fgov.be/belcotaxonweb.html.
