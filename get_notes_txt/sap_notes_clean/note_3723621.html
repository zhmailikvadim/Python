Symptom

You use the control variable in Organizational Management and have defined a corresponding activation date for this in the switch T77S0 switch HRFPM OMBAD.

In the "Check Budget..." function, the key figures in the overview area are currently only displayed as full-time equivalents (FTE). To be able to use this function better, the corresponding key figures should also be displayed as weekly working hours in the overview area.

Reason and Prerequisites

Missing function

Solution

Implement the attached correction instructions or import the specified support package.

To activate the display of the weekly working hours, you must then create the T77S0 switch HRFPM/ZOMST and assign the value 'X'.

Note, however, that this can also lead to noticeable performance problems.
