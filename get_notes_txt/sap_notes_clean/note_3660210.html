Symptom

Report "CL_ATP_USAGE_MAT_AATP_CHK_GRP=CP" ends with  SYSTEM_MEMORY_ERROR dump.

Other Terms

CL_ATP_USAGE_MAT_AATP_CHK_GRP=CP.

MARC

SYSTEM_MEMORY_ERROR

Reason and Prerequisites

The memory error occurs when table MARC is hudge.

Program error.

Solution

Apply the note.

Copy and execute the report using the attached file" Z_ATP_MAT_USAGE_ADD". It will add the parameter and desactivate the usage.

To reactivate the usage Copy and execute the report using the attached file" Z_ATP_MAT_USAGE_DEL".
