Symptom

The mxmc-install.sh script does not function properly.

The mxmc-install.sh and the util-functions.sh are using the program "grep" with the parameter "-P" which is only available for GNU grep and not for IBM AIX grep.

The mxmc-install.sh uses BASH as shell-program which is not available as default on IBM AIX.

 Steps to reproduce the issue(incl. used master data and/or business documents numbers):

 run mxmc-install.sh

Other Terms

SAP Identity Management, IDM, mxmc-install.sh, GREP, IBM AIX

Reason and Prerequisites

Regular expressions are used in both mxmc-install.sh and mxmc-update.sh

Solution

Apply the attached patch provided in the Support Packages & Patches section of this SAP Note. Make sure all components are updated to the corresponding SP level before applying the patch.

After applying the patch, regular expressions that are used in both mxmc-install.sh and mxmc-update.sh are removed and replaced with hardcoded values.

GREP - P is no longer used.
