Symptom

When reopening a previously saved pivot table after approximately 15 minutes and attempting to modify the report or refresh the connection, Microsoft Excel crashes.

Reproducing the issue

Scenario:

  1. Open SAP Business One Excel Report and Interactive Analysis and log on to the company database.

  2. Navigate to the Interactive Analysis tab.

  3. Create a new Pivot Table by selecting the required report.

  4. Add several columns from the right-hand panel.

  5. Save the pivot table to the repository and/or save the file via Excel → File → Save or Save As.

  6. Close the file.

  7. Wait at least 15 minutes.

  8. Reopen the saved report.

  9. When prompted, click Enable Content.

  10. Attempt to add a new column to the pivot table or refresh the connection.

Expected behavior:

SAP Business One Excel Report and Interactive Analysis successfully establishes the connection, and the report can be edited without any issues.

Actual behavior:

Microsoft Excel crashes when attempting to modify the pivot table or refresh the connection, and you are unable to edit the report.

Solution

SAP intends to provide a feature package or patch to solve the symptom that is described in this SAP Note. The section References below lists the related feature package or patch once they become available. Be aware that References can only be confirmed at feature package or patch release date. SAP delivers feature packages or patches only for selected releases at its own discretion, based on the business impact and the complexity of implementation.
