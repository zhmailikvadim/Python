Symptom

When checking the IDoc generated in the FI system via transaction WE02 after executing Posting to Accounting in the HR system, the following issue occurs:

  • Some IDoc fields—such as company code, GL account, cost center, etc.—are missing in the IDoc in the FI system, even though they are present in the corresponding IDoc created in the HR system.

  • The IDoc in FI ends in the following error:

    Required field GL_ACCOUNT was not transferred in parameter ACCOUNTGL
    Message No. RW602

Environment

  • HR renewal
  • SAP S/4HANA
  • SAP ERP
  • SAP SuccessFactors Employee Central Payroll

Reproducing the Issue

  1. Perform the document transfer from the HR system using transaction PCP0.
  2. Verify the IDoc in the HR system and confirm that the relevant fields (e.g. GL_ACCOUNT) are correctly populated.
  3. Check the corresponding IDoc in the FI system using transaction WE02.
  4. Observe the issue:
    • Some fields (e.g. GL_ACCOUNT) are missing in the IDoc in the FI system.
    • The IDoc contains the following error message:
      “Required field GL_ACCOUNT was not transferred in parameter ACCOUNTGL (Message No. RW602)”

Cause

  • A conversion rule is configured in transaction Maintain IDoc Conversion Rules (BD79) that sets the affected field to an initial value during outbound IDoc creation. As a result, the field is removed from the generated IDoc.




  • After the correction introduced in SAP note 3280430 - EDT: Unassigned values not set to initial:

    "With the delivery of this SAP Note (see section support packages), conversion rules which are defined with 'Set at initial value' are executed correctly: The receiver fields are initialized and hence are missing in the IDocs."


Resolution

  1. Review your conversion rules in transaction BD79 and confirm they are not setting fields to an initial value.
  2. Refer to the solution in SAP Note 3280430 – EDT: Unassigned values not set to initial.
  3. Refer to the solution in SAP Note 3634478 - Modifications when posting payroll results to accounting.
