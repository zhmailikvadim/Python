Symptom

HANA system information are not saved in H_CustomerMasterData table in RSP database.

Environment

Remote Support Platform for SAP Business One

Reproducing the Issue

  1. Install RSP Studio and configure Partner WebDAV in RSP.
  2. Run the System Status Report (SSR) on the customer system.
  3. Upload the SSR to the partner RSP.
  4. Go to database RSPSrv and check table H_CustomerMasterData, or run query SELECT *  FROM [RSPSrv].[dbo].[H_CustomerMasterData]
  5. Observe that system information for SAP HANA systems (for example, DataFileSize, LogFileSize) is missing, while SQL system information is present.

Cause

The SSR does not collect certain database metrics for SAP HANA systems. These details are collected only for Microsoft SQL Server databases.

Resolution

This is by design.
