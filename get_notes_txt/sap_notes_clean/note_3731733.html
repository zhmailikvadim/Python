Symptom

After you implement Note 3217953, the creation of a credit limit request is checked against authorization object ‘F_UKM_SGMT’ and ‘ACTVT’ FIELD '01’ (create). After the corrections are applied the system with check it against ‘ACTVT’ FIELD '02’ (update).

Other Terms

SCASE, UKM_CASE, K_CR, F_UKM_SGMT, authorization

Reason and Prerequisites

Program error

Solution

Implement the attached corrections in your system or import the relevant Support Package.
