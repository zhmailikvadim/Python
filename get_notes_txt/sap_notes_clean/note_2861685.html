Symptom

An investment measure (order or WBS element) has been created which has an asset under construction (AUC) assigned. Although an AUC has been assigned to the investment measure and the status AUC is set when releaseing the investment measure, there is no distribution rule available with the settlement type AUC available in the settlement rule. There is no distribution rule in the settlement rule available at all, although the AUC distribution rule is expected to be available by default.

Environment

Controlling (CO)
SAP ERP 
SAP Enhancement package for SAP ERP
SAP Enhancement package for SAP ERP, version for SAP HANA
SAP S/4HANA
SAP S/4HANA Finance

Cause

System design. The AUC distribution rule will be available in the settlement rule only after the first settlement has been executed. As soon as the first settlement to AUC has been performed, the AUC distribution rule will be set with the relevant dates filled (first used, last used, etc.)

Resolution

The fact that the AUC distribution rule will only be available in the settlement rule after the first settlement is executed is a fact that cannot be changed. If you need to see the AUC which is assigned to the investment measure, you can proceed as follows:

  • For orders: In transaction KO03 follow the menu path: Extras -> Asset under Construction
  • For WBS elements: In the CJ20N project builder select the relevant WBS element and follow the menu path: Edit -> WBS Element -> Investment programs -> Asset under construction.
    If transaction CJ03 is used, the AUC can be displayed with a double click to the relevant WBS element and the use of the button "Asset under construction". Alternatively, after selecting the WBS element in transaction CJ03 with following the menu path "Extras -> Asset under construction", the AUC can be reached as well.
