Symptom

You are using a Fiori application (e.g. "Stock - Multiple Materials") and click the jump link to navigate to the Batch Information Cockpit from a specific batch instance.

The selection details are not pre-filled.

Other Terms

 BMBC, BICMSF

Reason and Prerequisites

A precise navigation is unfortunately not possible due to a technical limitation when it comes to cross-app navigation between Fiori applications and traditional transactions that use a dynamic selection screen like BMBC in this case. It is currently not on the roadmap to correct this behavior for this legacy transaction.

Solution

The recommended action is to use the navigation to Manage Batches when it comes to working with batch master data. Manage Batches is a native Fiori and future-proof application.
