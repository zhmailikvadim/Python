Symptom

In the SAP HCM SAP Fiori app "My Paystubs (Version 3/SAP Fiori 2.0)" duplicate entries are displayed in the list of paystubs.

Other Terms

GBX01HR5, HCMFAB_MYPAYSTUBS_SRV, GT_RGDIR, CX_HCMFAB_MYPAYSTUBS, T77S0, CCURE PAYxx, CCURE MAINS

Reason and Prerequisites

Prerequisite:

  • Concurrent Employment is active (switch CCURE MAINS ("Concurrent Employment Main Switch") = 'X' in table T77S0)
  • Concurrent Employment Payroll is active (e.g. switch CCURE PAYUS ("USA: Start Date CE Payroll") set to a certain date)

 

Reason:

Program error.

Solution

The corrections contained in this SAP Note ensure that only the paystubs of the selected assignments are displayed.

Paystubs of other available assignments can be displayed via the button "Switch Assignment".

 

Import the relevant Support Package into your system.

For an advance adjustment to the behavior, implement the attached correction instructions.
