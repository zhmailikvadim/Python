Symptom

This note solves several errors during SD-TM integration.

- When integrating an SD document into Transportation Management and the associated Shipping point is not given or has an deviating adress, this does not stop the current integration process.

- When integrating a Customer Returns, the above check is wrongfully performed for the source location instead of the destination location, what usually leads to a wrongfully failed check.

Other Terms

Sales and Distribution; Transportation Management; /SCMTMS/CL_SD_TM_INTEGRATION; ADDITIONAL_CHECKS;

Reason and Prerequisites

Program Error.

Solution

Implement the corrections of the attached correction instructions or apply the corresponding support package.
