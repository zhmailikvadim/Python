Symptom

You use transaction ISSR_UPLOAD_TPT to import a BVI data sheet into statutory reporting. The total of the fund shares should be 100%.

This SAP Note delivers a plausibility check for this. If deviations exist, the program now issues warning message ISSR 563 ("Total share of a fund must be equal to 100%"). If a warning is issued, the BVI data sheet is saved anyway. You can check it and import it again after performing a correction.

Other Terms

LC2021, ISSR_UPLOAD_TPT, RISSR_UPLOAD_TPT, look through, BAPI_ISSR_ASSET_CREATE, BAPI_ISSR_ASSET_CHANGE

Reason and Prerequisites

Design

Solution

Implement the attached correction instructions or import the specified Support Package.
