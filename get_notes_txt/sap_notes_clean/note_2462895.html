Symptom

Crystal Reports Designer toolbar icons are displayed in a small size when using a large screen resolution (for example, 4K).

There is no option on the toolbar to enlarge the icons.

Reproducing the issue

Scenario:

Run Crystal Reports 2013 SP05 on a 4k monitor.

Expected behavior:

Crystal Reports leverages high DPI settings on such monitors so that the icons are properly scaled.

Actual behavior:

The Crystal Reports menu is almost unusable because the icons are too small.

Cause

Future Version Candidate (FVC)

Solution

Correction of system behavior is not feasible in the currently supported versions of SAP Business One. The error has been recorded and is a candidate to be fixed in a future version.
