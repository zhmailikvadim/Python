Symptom

  • The EWA Workspace application is not responding.
  • It keeps loading when accessing the URL https://me.sap.com/app/ewaworkspace.
  • Other SAP EarlyWatch Applications in SAP for Me are affected as well

Environment

SAP EarlyWatch Alert in SAP for Me

Cause

There is an issue with a web service affecting SAP EarlyWatch Alert applications in SAP for Me - we are working on a fix.

Resolution

A fix has been released in backend system and the EWA Workspace service is available again since 3/27.

  1. Use the EarlyWatch Alert Dashboard: EarlyWatch Alert Dashboard as workaround
  2. To access reports, use: EarlyWatch Alert Reports.
  3. To find solutions, use: EarlyWatch Alert Solution Finder.
  4. As a workaround, access the the EWA workspace with the following URL instead https://me.sap.com/ewa/filter/081B04B1F94B3C493796B7A08B67290E
