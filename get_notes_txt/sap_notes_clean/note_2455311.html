Symptom

In Customer Returns Process, you observe that error messages with message number /SPE/INSPRES036 and /SPE/INSPRES046 occur in inbound queue monitor, if returns material authorization (RMA) is not activated in IMG.

Other Terms

SMQ2, SPE, RMA, Customer Returns Management, ERP, EWM, IMG, Enhanced Returns, Inspection, Follow-up activity

Reason and Prerequisites

Prerequisites:

  • Customer Returns Oder is created in ERP Sales and Distribution (SD)
  • Returns Delivey is created and distributed automatically to EWM system
  • Quality Inspection is required in EWM, requiring EWM to communicate the inspection results with ERP

Solution

In IMG path Logistics - Execution -> EWM Integration -> Customer Returns Processing -> Activate Enhanced Returns Processing,

you should maintain an entry, with which the checkbox RMA must be ticked, along with all corresponding sales order type/sales organisation combination.
