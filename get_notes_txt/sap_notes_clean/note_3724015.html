Symptom

Mismatch Between Originator and Recipient Item Amounts After Adjusting the Amount Using the Manual Repair Payment Application

Other Terms

 Repair Payment, Transaction Amount

Reason and Prerequisites

Permission to edit the Transaction Amount field for a Recipient Item in the Repair Payment Application leads to a mismatch between Originator and Recipient Item amounts after the amount is adjusted using the Manual Repair Application.

Solution

Editing of the Transaction Amount and Transaction Currency fields has been disabled in the Repair Payment Application.

Please apply SAP Note or assigned Support Package.
