Symptom

Error message "Follow-up activity 'In-House Service' is only valid for non-valuated stock" (CX_MSR_TRC130) occurs during inspection when follow-up activity 0026 or 0027 is chosen.

Environment

  • Sales and Distribution (SD)
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Create an ARM return order and the relevant follow-up documents.
  2. Start the inspection step (transaction MSR_INSPWH).
  3. Enter the logistical follow-up action as 0026 or 0027 (In-House Repair Service)
  4. While entering the inspection result the system displays the error message: "Follow-up activity 'In-House Service' is only valid for non-valuated stock".

Resolution

The error message CX_MSR_TRC130 is referencing a customizing option to enable inspection step with valuated stock. However, this applies only for follow-up activity 0016 and 0017.
Follow-up activities 0026 and 0027 are intended strictly for non-valuated stock. This is according to the standard design.

See Also

  • In-house repair follow-up activities are also not foreseen for use with cross-company returns.
  • For further reference:

SAP Note 1879120 - FAQ: External Repair with Advanced Returns Management

13. Is it possible to trigger a repair for valuated stock?

In general the repair process is destined for use with non-valuated stock as this will be returned to the customer once repaired and hence the ownership of the stock remains with the customer.

Despite the above, the possibility to set In-house/External Repair as a logistical follow-up activity (0016, 0017 follow-up activities) during the material inspection (transaction MSR_INSPWH) has also been provided for valuated stock. SAP Note 1780677 describes the required Support Package level as well as the necessary customising for activation of this functionality.
