Symptom

Unable to find test script for Workforce Enablement (1FD) under LOB Human Resources > HR Master Data and Connectivity in SAP Best Practices for SAP S/4HANA Cloud Public Edition.

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental. 

Environment

SAP S/4HANA Cloud Public Edition

Cause

This scope item enables the maintenance of workforce master data in SAP S/4HANA Cloud Public Edition. It is a prerequisite for other apps and integrations, and does not provide additional documentation.

Resolution

  • Scope item 1FD is activated by default and serves as the foundation for using apps such as Manage Workforce or integrating with Master Data Integration.
  • For integration purposes, additional settings may be required, which are detailed in the integration setup guide.
  • Note that scope item 1FD includes only automated content and does not require further manual steps.
