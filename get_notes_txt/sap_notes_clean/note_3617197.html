Symptom

WARNING: This is a CA-FL (Field Logistics) specific note. If you DON'T have

In Field Logistic Returns by Maintenance Order App, when Return to Stock action is performed, the valuation type field value help shows blank entries even if the item is not a valuated material. These additional entries are incorrect for the valuation type.

When Return to Stock action is performed in the Returns by Maintenance Order App (F4682), with Initiate Returns checkbox selected in the pop-up dialog, STO is generated with status "Return Requested" which is incorrect.
