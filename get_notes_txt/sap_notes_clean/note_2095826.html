Symptom

The output language of SRM Purchase Order.

Environment

  • SAP Supplier Relationship Management (SRM) 7.0 and higher
  • SAP enhancement package for SAP Supplier Relationship Management

Resolution

In standard SRM, the output language of purchase order is determined always by the preset language of supplier.

You can check supplier language in a purchase order on tab Overview, by checking supplier properties.

 

 

You can change the output language using the BBP_OUTPUT_CHANGE BAdI.

This BAdI offers the interface IF_OUTPUT_MEDIUM_CHANGE and the method CHANGE_OUTPUT_LANG, where the print preview language and the final output language can be set separately.
