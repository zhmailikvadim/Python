Symptom

  • Unable to access display group journal entries app (F2573). The application keeps loading indefinitely.
  • No error messages are displayed.
  • No error logs are registered in the back-end or front-end.
  • Error message observed in F12 developer tools: "function '/GetFeatureSupport' not found in the metadata - EventProvider sap.ui.model.odata.v2.ODataModel sap.ui.model.odata.v2.ODataModel".

Environment

  • Group Journal Entry for S/4HANA Group Reporting.

Reproducing the Issue

  1. Open the application F2573 (Display Group Journal Entries).
  2. Observe that the application keeps loading indefinitely.

Cause

Resolution

  1. Reload the metadata for the service "FINCS_JRNLENTR_LIST_SRV" of the application F2573 in transaction "/n/IWFND/MAINT_SERVICE".
  2. If the issue persists, implement the suggested SAP Note "3711111 - DGJE - Hide 'Merged Journal Entries' tab when Customer sign feature is enabled".
  3. Verify if the issue is resolved after implementing the SAP Note.

See Also

  • Refer to: Consolidation | SAP Help Portal.
  • Refer to: SAP Note 3711111 - DGJE - Hide 'Merged Journal Entries' tab when Customer sign feature is enabled.
