Symptom

This error should occur only if the database is Oracle.

In this case, the two views OIJ05_V_NOM_SIM and OIJ05_V_NWB are missing from the database following the upgrade.

Other Terms

Upgrade with Oracle database, RSDB02CK.ELG

Reason and Prerequisites

The CREATE statements of the specified views are extremely large. For this reason, the views cannot be created in the Oracle database.

This problem is rectified by SAP Note 2467500, but this is not available during the upgrade.

Solution

AFTER the upgrade, implement SAP Note 2467500 and activate both views with the program RUTDDLSACT.
