Symptom

You use transaction SU10 to create users. After filling out the "Logon Data" tab, you enter data on the "Roles", "Profiles", or "Systems" tab and save the users.
When you check the users, you notice that except for the user group, none of the data you entered on the "Logon Data" tab has been saved.

Reason and Prerequisites

This is a follow-on error from SAP Note 1663177.

Solution

Implement the correction instructions using the Note Assistant or import the relevant Support Package.
