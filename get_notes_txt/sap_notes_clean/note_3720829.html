Symptom

You use the migration cockpit object 'HCM - Object' (SIF_HCM_Object) uploading an XML migration file to your Migration Cockpit object 'HCM - Object'.

Your XML Migration file contains values for Object ID's whose contains more than one Object (Infotype 1000) record. If you execute now the 'Simulate' or 'Migrate' step and the business logic raise any  error, this error lead then to short dump CX_HRBAS_INVALID_BUFFER _STACK. An example of error is that the provide Object ID belongs not to any external number range interval for number range object 'RP_PLAN' and therefore the 'Simulate' or 'Migrate' step will raise error message 5A 366 (The number indicated is not in any valid external interval) and as mentioned this will lead to short dump CX_HRBAS_INVALID_BUFFER _STACK.

Other Terms

CX_HRBAS_INVALID_BUFFER _STACK, 5A 366, 000 DMC_RT_MSG, RP_PLAN, HCM - Object, SIF_HCM_Object

Reason and Prerequisites

Programming error.

Solution

Implement the attached correction instructions or import the relevant support package.
