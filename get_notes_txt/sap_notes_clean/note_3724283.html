Symptom

In SAP Business Application Studio, Fiori or Adaptation Project creation from template fails when connecting to the backend ABAP system. There is a need for more detailed traces for further investigation. 

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental." 

Environment

  • SAP Business Application Studio
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Resolution

To understand the root cause of the issue, please follow these steps:

  1. In the BAS terminal, refresh destination list using command:

    curl localhost:8887/reload

  2. Generate destinations list using command:

    curl $H2O_URL/api/listDestinations -o dests.json

    The generated dests.json file contains a list of all the destinations available on your subaccount.

  3. For S/4HANA on-Premise or Cloud Private Edition backend systems, connect to the SAP Cloud Connector:

    • From your SCC UI, select Connector, select the Subaccount where BAS is subscribed)
    • Under "Log and Trace files" section, edit and set the "Cloud Connector Trace Level" to "Debug" and tick option "Tunnel Traffic Trace":



  4. In BAS, set the "Application Wizard" trace logging level to "debug" as per KBA 3660568

  5. Reproduce the issue and set the trace logging levels back to the default values.

  6. In BAS, open command palette via View -> Find Command and enter "Fiori: Open Environment Check" -> Select "Check Destination" -> Select your destination -> Enter credentials if prompted -> Select "Save and view results" -> Choose a location to save the generated zip

    • After selecting the destination, it might take some seconds until the next prompt is displayed. Please wait.
    • Download the generated zip from the chosen location.

  7. Collect and attach:

    • dests.json file containing the list of all the destinations
    • Generated Cloud Connector traces "scc_core.trc" and "tunnel_traffic_<account id>_on_<landscapehost>.trc"
    • Generated "Application Wizard" traces
    • Generated zip file by the Fiori Environment Check
