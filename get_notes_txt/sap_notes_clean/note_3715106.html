Symptom

In the SAP Fiori app "Create Sales Orders – Automatic Extraction" (F4920), the Extracted Net Value in the sales order header is displayed with an incorrect amount.

The Extracted Net Value is transferred and displayed without applying proper external to internal currency amount format conversion.

For example:

  • The Extracted Net Value in the header is displayed as:
    7.654.494.000 HUF (Incorrect)

  • However, the Net Amount extracted by Document AI is:
    76,544,940.0 HUF (Correct)

Other Terms

F4920,Create Sales Orders – Automatic Extraction,Extracted Net Value,Header Net Value,Currency Amount Format,Decimal Shift,Document AI,CL_SD_SOR_DATA_ANALYZER,CHECK_DATA_CONSISTENCY

Reason and Prerequisites

Program Error

Solution

Implement the attached correction instructions or upgrade the specified Support Package.
