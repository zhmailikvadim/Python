Symptom

This KBA introduces how to add additional standard search help for field Personnel Number in t-code PA20/PA30.
"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Environment

  • Human Capital Management (HCM)
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for HANA
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition
  • SAP SuccessFactors Employee Central Payroll

Resolution

The search help available for the Personnel Number field varies based on the user parameter 'MOL'.

  • This parameter can be maintained via t-code SU01. 
  • Here are the examples of search help options depending on the 'MOL' setting:

     For MOL set to '41', following search help is available:

However, there may be scenarios where the standard search help linked to each user group does not meet specific business requirements. For instance, users from molga 41 do not have access to the search help PREMI, yet there is a business need for it.

To address such requirements, utilize the BADI HRPAD00_PREM to create a custom implementation. This will allow adding the expected search help for the required molga. By implementing this BADI, the search help can be tailored to align with business needs.

  • To have an overview of all standard search help for Personnel Numbers, run t-code SE11 and display search help PREM:
     

 Please proceed with creating the BADI implementation to incorporate the desired search help for the target molga.
