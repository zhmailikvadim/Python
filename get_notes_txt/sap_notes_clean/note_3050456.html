Symptom

Salesforce test connection fails in integration hub if the password for salesforce account contains a special character.

Reproducing the issue

Prerequisites:

You are using Intercompany Integration Solution PL41 or lower.

Scenario:

Salesforce test connection fails in integration hub if the password for salesforce account contains a special character.

Expected Result:

Salesforce test connection is successful.

Actual Result:

Test connection fails.

Solution

The issue described above has been fixed in the Intercompany integration solution PL42 for SAP Business One, and higher versions.
