Symptom

In the process "End of Supply for Electricity (SUP View)" (DE_B_GPKE_EOS_SUP), the INFO_EVENT_CODE field was missing in the UBD data in step 8950.


This issue is relevant for the market role: SUP.

Other Terms

 MPES, End of Supply for Electricity, DE_B_GPKE_EOS_SUP, INFO_EVENT_CODE, UBD data, BC set

Reason and Prerequisites

The error occurred because of a logic error in the system.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the system makes the INFO_EVENT_CODE field available in the UBD data in step 8950. You can activate the BC set /US4G/3_0_SUP_SP08_NOTE_3721698 to resolve the issue. You need to retrigger the process. You can verify the fix by checking the UBD_DATA and confirming that INFO_EVENT_CODE is available in step 8950.
