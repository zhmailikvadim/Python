Symptom

Bullet Chart fails when colour palette is set

Other Terms

Bullet Chart,colour palette, BI Platfrom

Reason and Prerequisites

  1. Build new Lumira Document and use analysis template
  2. Create the crosstab and publish to BI platform 
  3. Login to BI platform and  open the document ,Change to chart, you see a column chart
  4. Filter on Measures to get two valid measures
  5. Change chart types, observe no issues
    1. We change to column, bar, pie, bullet, waterfall, line many times and nothing goes wrong
  6. Now set the colour palette
    1. Choose any provided or create your own, the issue always occurs
  7. Change to Bullet chart and observe the issue that it doesn’t load - icon is on bullet but chart component does not re-render

Solution

Fix will be available in Lumira 2025
