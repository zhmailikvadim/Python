Symptom

Short dump MESSAGE_TYPE_X in the method GET_OBJ_REF_SRC_STATIC of the class CL_RSBK_CMD_TPL_X.

Other Terms

MultiProvider CompositeProvider

Reason and Prerequisites

Program error. The call is made for a request/DTP having as source a MultiProvider which was convertes into a CompositeProvider.

Solution

  • SAP BW 7.50
    Implement Support Package 11 for SAP BW 7.50 (SAPK-75011INSAPBW) into your BW system. The Support Package will be available as soon as SAP Note 2544313 with the short text "SAPBWNews 7.50 BW ABAP SP11", which describes this Support Package in more detail, is released for customers

  • SAP BW 7.51
    Implement Support Package 6 for SAP BW 7.51 (SAPK-75106INSAPBW) into your BW system. The Support Package will be available as soon as SAP Note 2544283 with the short text "SAPBWNews 7.51 BW ABAP SP6", which describes this Support Package in more detail, is released for customers. 

  • SAP BW 7.52
    Implement Support Package 1 for SAP BW 7.52 (SAPK-75201INSAPBW) into your BW system. The Support Package will be available as soon as SAP Note 2505912 with the short text "SAPBWNews 7.52 BW ABAP SP1", which describes this Support Package in more detail, is released for customers.  

In urgent cases you can use the correction instructions.

Before you use the correction instructions, make sure that you check SAP Note 1668882 and SAP Note 2248091 for transaction SNOTE.

This SAP Note might already be available before the Support Package is released. In this case, however, the short text still contains the term "preliminary version".
