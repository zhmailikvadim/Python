Symptom

JDBC sender adapter throws the error "com.sap.it.rt.adapter.jdbc.exceptions.JDBCException: Root Cause: --xxxxxx : java.lang.NegativeArraySizeException: -xxxxxxx" when querying data from a database.

Environment

  • SAP Integration Suite
  • Cloud Integration

Cause

The error is thrown when the query returns a huge amount of records.

Resolution

Reduce the number of records returned per query (for example, split the extraction into smaller batches) to avoid processing a very large single result set.
