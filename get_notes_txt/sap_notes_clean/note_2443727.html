Symptom

You execute the SPED EFD ICMS/IPI report (J_1BEFD transaction) with the BAdI: Filling of Selected Fields in Electronic Fiscal File (BADI_J_1BEFD->SET_BLOCK_K_REG_K200) implemented, however the system rejects the value entered in the K200 register and doesn’t show it in the report.

Other Terms

SPED, EFD, BLOCK K, K200, BADI_J_1BEFD, SET_BLOCK_K_REG_K200, J_1BEFD

Reason and Prerequisites

This SAP Note is relevant only for Brazil.

Solution

With this note, the values entered in the BAdI: Filling of Selected Fields in Electronic Fiscal File (BADI_J_1BEFD->SET_BLOCK_K_REG_K200) are displayed according to the K200 register.
