Symptom

  • I want to migrate from SAP BTP NEO to SAP BTP Cloud Foundry (CF)
  • I am not sure how to configure the new CF subaccounts in SAP Cloud Connector
  • I need guidance to identity the provider (IdP) configuration
  • I do not know how to add Cloud Foundry subaccounts 

Environment

  • Managed Gateway for Spend&Network
  • SAP Business Technology Platform (BTP)

Resolution

To proceed with the migration, ensure you complete the required configuration steps in SAP BTP Cloud Foundry. To complement the migration, you need to review the guides and KBA below for detailed step-by-step instructions: 

SAP Integration Suite, Managed Gateway for Spend Management and SAP Business Network - SAP BTP NEO to CF Migration 

KBA KB1321448 - Steps to add cloud foundry subaccounts of Managed Gateway for Spend & Network

Use a Custom IDP for Subaccount Configuration | SAP Help Portal
