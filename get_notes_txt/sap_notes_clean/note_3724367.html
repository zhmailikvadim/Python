Symptom

In a Web Dynpro Multidimensional Analytic Grid application, on the Query Information Tab, there is information on variables and filters for the current result set. For any value being excluded, a "!" character indicates that the condition is inverted (logical NOT)

Values that are excluded from a variable via conditions are displayed without the "!" character.

Also, in case an excluded value has an excluding operator (like "IS NOT" or "NOT GREATER THAN"), the "!" character can be doubled.

BICS Grid
