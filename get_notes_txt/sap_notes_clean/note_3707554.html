Symptom

When Goods Receipt has been done for a CFG process order, a CFG subcontracting purchase order or a batch that came into the system via an initial upload, the batch status may be set to Restricted. In the SAP Fiori app Manage PCNs, the restricted stock quantity is not displayed in the Stock column in either table on the Batches tab in display mode, nor on the Batches tab in edit mode.

Other Terms

ISLS, Manufacturing, Intelligent Clinical Supply Management, Manage PCNs, Restricted Stock.

Reason and Prerequisites

You use SAP Intelligent Clinical Supply Management or this SAP Note is a prerequisite of another SAP Note.

Solution

Implement the attached correction.
