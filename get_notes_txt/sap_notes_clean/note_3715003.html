Symptom

This SAP Note addresses the following symptoms.

1. You have re-processed a document in Failed to Send status from Resolve Errors application provided by Document and Reporting Compliance Cloud Edition. You observed that the documents do not change the status to Accepted by KSeF or Rejected by KSeF.

2. You have re-processed a document in Failed to Send status from Resolve Errors application provided by Document and Reporting Compliance Cloud Edition. You observed that the documents do not change the status to Accepted by KSeF or Rejected by KSeF and you have executed Retry/Resubmit action. The document is blocked in Send to DRCC status.

For more details on how to resolve the status mismatches, please refer to the Solution section of this SAP Note.

This SAP Note delivers a new status Sent to Customer (Accp Offln) for Electronic Document Processing for Poland - KSeF.

Solution

Date Version Reason
22.02.2026 1 Released for Customers

Import the Support Package specified for your release or implement the attached correction instruction.

SAP recommends that you implement the Support Package specified for your release. However, if you need to install a solution earlier, use the SAP Note Assistant to implement the correction instruction. You can find more information about the SAP Note Assistant in SAP Support Portal, under SAP Note Assistant.

You must also implement the customization activities shared in the attachment SAP Note 3715003 Customization Steps PDF.

Once the corrections are available in your system, please follow the steps below to resolve the status mismatch.

1. Open the Electronic Document Cockpit or Manage Electronic Documents application.

2. Select the Electronic Documents with status mismatch and perform More -> Get Status to pull the latest status from Document and Reporting Compliance Cloud Edition.

3. Once the latest status is received in the Business System, you can take the necessary follow up actions - Send to Customer/More -> Display PDF on documents in Accepted by KSeF status, Retry on documents in Failed to Send status and Resubmit on documents in Rejected by KSeF status.
