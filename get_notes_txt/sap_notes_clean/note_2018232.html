Symptom

You are performing a conversion project using SAP Landscape Transformation (version 2, Add On DMIS_2011).

Symptom

In Transfer Posting, different Segmentation values are provided at source and destination for respective Batches.

Batches created both for source and destination hold the same Segment value

Other Terms

Transfer Posting, Incorrect Segmentation, Receiving, Batch, Inventory Management

Reason and Prerequisites

Program Error

Solution

Implement the Correction instruction attached with this note

Other Terms

SAP Landscape Transformation, SAP LT, DMIS, DMIS_2011, ADT, Transformation Summary, PIFD, Storage

Reason and Prerequisites

Due to some different problem you want to enable old storage for the Transformation Summary. This note provides a possibility to control the type of used storage at transformation package level.

Solution

Install Service Pack 08 of Add On DMIS_2011 or implement provided correction in the control system (usually Solution Manager system).

To enable old storage for your package maintain a package parameter CNV_ADT_PIFD_ENABLED using transaction CNVMBTPARAMS.

Enter:

  • Parameter Name = CNV_ADT_PIFD_ENABLED
  • Value Low = X

Other values ought to remain unchanged.
