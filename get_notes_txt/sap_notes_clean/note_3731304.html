Symptom

In "Project Control - Professional Services Projects" app, 'No Data' is displayed when searching a customer name containing special characters (for example '&') in 'Customer' field. After a while, error displayed as "Value XXX does not exist".

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Open the Project Control – Professional Services Projects app.
  2. In the Customer field, type a customer name that includes a special character (for example, '&'), No Data displayed even though the customer is valid.
  3. After a while, error displayed as "Value XXX does not exist".

Cause

  • The Customer field only accepts a customer ID for direct entry; You can use the customer name to search and select the customer, by this way, 'Customer' field will be filled in correctly without any error.
  • Special characters are not supported in customer name search at present, which leads to validation errors when such characters are used.

Resolution

  • Replace the special character with space and search the full customer name.
    For example customer full name is "A&A BBBB CCCC", searching for "A A BBBB CCCC".
  • Search for the customer ID directly.
