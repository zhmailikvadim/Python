Symptom

  • You are using the SAP Analytics Cloud (SAC) Stories built using HANA views for Forecast-Related Analytics in SAP CARAB (5.0 FPS01 or 5.0 FPS02 or 5.0 FPS03 ).
  • When you open the stories, you notice that the texts are dispalyed only in English irrespective of the logon language.

Other Terms

Forecast-Related Analytics, Demand Forecasting, Replenishment Planning, Dashboard, SAP Analytics Cloud, SAC, Performance, SAC story, SAP CARAB 5.0 FPS01, SAP CARAB 5.0 FPS02, SAP CARAB 5.0 FPS03, SAP Replenishment Planning, SAP CAR, UDF, Unified Demand Forecast, model, modeling, demand modeling, historical sales, demand forecasting, forecast, unit sales,

Reason and Prerequisites

You have implemented note 3246496, 3170396 and 3245740 to improve the rendering of these analytical stories.

Solution

The translations will all be available with SAP CARAB 5.0 FPS04 and later releases.

Alternatively, if you are unable to wait for the fix in the above support/feature packages, or if you are not able to implement the support/feature packages, then you can also translate the texts manually as follows:

Please refer to the SAP Support Package Stacks information in the Support Portal for information on the availability of these support packages: https://support.sap.com/en/release-upgrade-maintenance/maintenance-information/schedules-for-maintenance-deliveries.html

Translation Object type (shortcut): HOTS

Object names: Please find all the relevant the object names and the translated texts in English, German, French and Italian in the MS Excel file attached to this SAP Note.

Please use transaction SE63 to translate the text(s).

  1. Enter /oSE63 in the command field (transaction field) and press Enter
  2. Enter the object type (see above) in the command field and press Enter
  3. Enter the object name (refer to the document attached) in the field Object Name
  4. If required change Source and/or Target language
  5. Press Edit
  6. Enter the required translation in the text fields below the source text fields (Refer to the attached excel file for the translation texts)
  7. Choose Save to save your translation(s)
  8. Choose Back (F3) to leave the translation editor

Note:

Translations performed in a customer system may be overwritten in the following instances:

  • If the object happens to be part of a future support package
  • Upgrade to a new release
  • Import of new language CDs from Support Releases
