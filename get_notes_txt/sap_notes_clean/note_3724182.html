Symptom

In the cost breakdown, the UBE components are exploded. However, the UBE portions of the corresponding levels cannot be easily identified because these UBE lines are not displayed.

Solution

Implement the source code corrections.

Troubleshooting Description:

After you implement this SAP Note, you can use Customizing SAP Dairy Management by msg -> Dairy Cross Functions -> General Settings -> System Settings to activate new parameters.

UB1: If activated, the UBE is displayed at the highest level of the cost breakdown

UB2: If activated, the UBE entries are displayed in the lower levels of the cost breakdown.
