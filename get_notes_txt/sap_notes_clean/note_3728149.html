Symptom

  • How do I change the System Owner role in SAP Business Data Cloud?
  • The assigned System Owner role for SAP Business Data Cloud has left the organization and they need to reassign it to someone else. How would they go about that?
  • SAP Business Data Cloud license purchased, but System Owner user can't login to the system

Environment

SAP Business Data Cloud

Resolution

  • Preferred solution is to follow SAP Help Portal Documentation: Transferring the System Owner Role
  • Please note - It is not possible to transfer the System Owner role to a user who has not logged on for the first time yet.
  • If the System Owner cannot access a new SAP Business Data Cloud system you can contact your Account Manager and request to change the Contact IT person in your contract.
  • You can't request the change via Technical Support case as Technical Support cannot authenticate that you are authorized to request that change.
