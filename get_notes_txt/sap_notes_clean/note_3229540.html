Symptom

Error: Target not found - Cannot set properties of undefined (setting 'oView')

Environment

  • SAP S/4HANA 2020
  • Combine Revenue Contracts app

Cause

FIN_RARREUSELIB node is inactive in SICF

Resolution

  1. Go to transaction SICF
  2. Search for node FIN_RARREUSELIB under the "ui5_ui5" path
  3. Activate the node
  4. Clear the browser cache and reload
