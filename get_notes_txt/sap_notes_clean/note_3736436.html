Symptom

Cloud process DE_C_MET_MRV_COMPLAINT_MALO can't send an APERAK message for the incoming message MSCONS 13017.

Other Terms

MET, MSCONS, APERAK, DE_B_MET_MRV_COMPLAINT_MALO, Device, meter reading complaint

Reason and Prerequisites

Device information isn't retrieved from the backend process DE_B_MET_MRV_COMPLAINT_MALO.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

In process DE_B_MET_MRV_COMPLAINT_MALO, device information can be retrieved in step 0110 and send to cloud system in step 0150.
