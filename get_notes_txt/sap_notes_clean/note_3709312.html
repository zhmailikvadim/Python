Symptom

  • Story URL is not working properly when setting lang parameter as English
  • When having lang in parameter, the variable parameter can be ignored sometimes.

Environment

  • SAP Analytics Cloud Enterprise 2025.21

Reproducing the Issue

  1. Create a story URL with variable parameter and lang=en or lang=en_us
  2. Access the story URL and sometimes the story is loaded without correct content which should be filtered by the variable parameter.
  3. Access the story URL by removing the lang=en, the story content is loaded correctly according to the variable parameter value.
  4. Access the story URL by setting lang to other value, eg: lang=de, the story content is loaded correctly according to the variable parameter value.

Cause

The issue is caused by product bug.

Resolution

The issue will be fixed in 2026.2.10 version

Keywords

lang=en-US, language, SAC story URL parameter, incorrect variable values.
