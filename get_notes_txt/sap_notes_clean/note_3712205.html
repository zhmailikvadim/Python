Symptom

Make change in OKEQ, the TR is not generating and the data getting saves automatically without prompting of Transport request. 

Environment

  • Controlling (CO)
  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
    • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA
  • SAP S/4HANA Finance
  • SAP S/4HANA Cloud Private Edition
  • SAP Fiori for SAP S/4HANA

Reproducing the Issue

OKEQ, make any change, after saving, the data getting saves automatically without prompting of Transport request. 

Cause

system setting

Resolution

Please check object V_TKA07_CL  setting in SOBJ.

Keywords

SOBJ, current setting, transport request, automatic, OKEQ.
