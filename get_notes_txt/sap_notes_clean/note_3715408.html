Symptom

In the latest people profile, when navigating via all actions the link does not open in a new tab/window.

Environment

SuccessFactors Employee Profile 2511

Reproducing the Issue

  1. Navigate to all actions on latest people profile.
  2. Select any action e.g. Employment Details.
  3. Observe the action opens in the same window.

Cause

Working by design.

Resolution

This is expected behavior.

The Full Profile in the latest People Profile is a single‑page view and cannot be configured to open in a new window at this time.

See Also

If you wish to see this feature introduced, we kindly encourage you to submit an enhancement request on the same via 2090228 on “How to submit an enhancement request”.
