Symptom

When creating an Agreement request from Data Objects, the schedules are not generated correctly. The schedule data passed from the Data Object is not properly considered, resulting in all schedule items being assigned to the first schedule ID instead of being distributed as intended.
This SAP Note provides the necessary corrections to address and resolve this issue.

Other Terms

Agreement Request, Data Object, Schedules, Data Model, Document Map

Reason and Prerequisites

 This is caused by program error.

Note : Please implement SAP Notes#3569822, 3629377 and 3715672 as a Pre requisite note

Solution

Please implement the correction instructions. These corrections are only valid if you have applied the following Support Packs.

Release V/4 – 2023 V2

Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object
