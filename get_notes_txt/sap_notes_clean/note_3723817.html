Symptom

Default behaviour and tag labels are displayed in Talent Intelligence Hub - Attributes tab in view mode. 

In Edit mode behaviour and tag labels are translated to the current system locale.

Environment

SAP SuccessFactors Talent Intelligence Hub

Reproducing the Issue

Attributes tab:

  1. Behaviour and tag is created and mapped to a competency in English.
  2. All translations are maintained for the behaviour, tag and competency.
  3. Switch the locale to another one.
  4. Go to TIH - Attributes tab and view the Competency.
  5. Competency is displayed in current locale (translated), but the tag and behaviour is in English (not translated).
  6. Click Edit the competency and see that the behaviour now is appearing in current locale (translated), but the tag is not.

Behaviours and tags tab:

  1. Behaviour and tag is created and mapped to a competency in English.
  2. All translations are maintained for the behaviour and tag.
  3. Switch the locale to another one.
  4. Go to TIH - Behaviours or Tags tabs and view the behaviour or tag.
  5. It appears in current locale (translated).
  6. Click Edit and Click Translate buttons.
  7. Do not change anything, click Cancel.
  8. Behaviour or tag name changes to default label (English, not translated).
  9. Refresh the page - current locale is displayed again.
