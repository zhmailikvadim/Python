Symptom

When attempting to change the Confirmation Control Key, but an error occurred: "MEPO031: Confirmation control key cannot now be changed".

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Open Change Purchase Order- Advance (ME22N)
  2. Item details
  3. Change confirmation control key

"Confirmation control key cannot now be changed Message no. MEPO031"

Resolution

In standard functionality, it is not possible to change the Confirmation Control Key once follow-on processes have been performed. If the Confirmation Control Key is changed, it leads to inconsistencies in the system.

Remove the line item from the inbound delivery using transaction VL32N. After removing the line item from the inbound delivery, proceed to change the Confirmation Control Key as required.
