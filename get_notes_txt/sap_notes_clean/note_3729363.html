Symptom

When we are trying to navigate to the Processors tab for the activities in the Fiori level, we are unable to see the Processors tab. This happens only if we previously view the activities which don’t have the processors. But firstly, if we view the activities which has processors, then in this case we will be able to see the processors tab for all the other activities as well.

Other Terms

Processor tab, Activities, Fiori, /IRM/GACM, DOCUMENT_METADATA_PREPARE

Reason and Prerequisites

This is caused by program error

Solution

Please implement the correction instructions. These corrections are only valid if you have applied the following Support Packs.

Release V/4 – 2023V2, 2023 V3 and 2023 V3 Support Package 1.

Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object
