Symptom

All the data required for currency conversion is available for a payment item. The system does not need to query the exchange rate. However, the system still rejects the posting of the payment item by issuing error message /FSCEX/GENERAL 003 if no exchange rates are maintained for the currency combination of transaction currency and account currency.

Other Terms

/FSCEX/APPL_EXCHANGE_AMOUNT, /FSCEX/GENERAL003, /fscex/cx_al_rate_det_failed, /FSCEX/AL_CURRENCY_EXCHANGE, DETERMINE_EXCHANGE_RATE, /FSCEX/IF_EX_CURR_TRANSLATION, e_exchange_rate, e_exchange_rate_tmstmp, g_con_return_import_err, s_determine_exchange_rate, /fscex/cl_al_currency_exchange, CONVERT_TO_LOCAL_CURRENCY

Reason and Prerequisites

There is a program error

Solution

Implement the attached correction instructions or import the specified Support Package.
