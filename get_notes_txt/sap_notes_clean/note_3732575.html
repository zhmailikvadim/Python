Symptom

  • An issue occurs during repo scan with repair, where it consistently reports repairing 633 or 634 items. However, subsequent scans indicate the same number of items still need repair.
  • The repo scan output file displays warnings such as:
    Relation 'ServiceContainer-Server' from object ID 989 has an invalid target (Object ID = 320848). Allow the application to delete the dangling relation.
    Relation 'Service-Server' from object ID 1148 has an invalid target (Object ID = 337024). Allow the application to delete the dangling relation. 

Other Terms

Reposcan, Repair

Reason and Prerequisites

Environment:

  • SAP BusinessObjects Business Intelligence Platform BI 2025 Patch 800
  • All Supported Operating System

Steps to Reproduce:

  1. Run the reposcan command for scanning.
  2. Observe that 633 or 634 items are flagged for repair.
  3. Execute the repair command, which indicates successful repair of inconsistencies.
  4. Run the scan command again and observe that the same number of items are flagged for repair.
