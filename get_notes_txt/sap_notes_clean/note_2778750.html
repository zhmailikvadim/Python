Symptom

SAP Customer Checkout manager 2.0 FP06 PL19

Other Terms

SAP Customer Checkout manager 2.0 FP06 PL19 can used for a fresh installation or an update of an existing version

Reason and Prerequisites

SAP Customer Checkout manager 2.0 FP06 PL19 contains the following changes

Solution

SAP Customer Checkout manager 2.0 FP06 PL19 contains the following changes:

Fix: Delete receipt performance on MSSQL, update receipt with less payment items.

Fix: Delete receipt with referenced salesItems: In some cases it can occurs that a salesItem that shall be deleted in restaurant mode is referenced by another salesItem. This reference is cleared now, before the

      receipt is deleted.

Fix: For sales items in receipt App. The shows only the limited number of sales items.

Fix: Added ddl script to execute missing updatescript.

Fix: HANA issue with SQL-Update in Server Update_20SP03_0-9.

Fix: UPDATE statements which do not affect any row will fail with the exception

Fix: Software version of ping monitoring events is not updated.

Fix: Send always monitoring messages were stored even if monitoring was disabled. This is changed now.
