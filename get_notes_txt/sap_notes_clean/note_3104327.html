Symptom

The following error occurs during the remodeling of a DataStore object:

RSO_DTO 001: "Metadata inconsistency in method '&1'. Location: '&2'"

The arguments are "_CHECK_INPLACE_REPART_POSSIBLE" and "-1-".

Other Terms

DataStore object (advanced), ADSO, DTO, Data Tiering Optimization, partitioning

Reason and Prerequisites

This error occurs only if the temperature schema is either 'HWP' (HOT-WARM at partition level) or 'HWCP' (HOT-WARM-COLD at partition level) and if dynamic partitions are used. In addition, a HASH partitioning must be used at the second level of the DB partitioning.

Solution

  • SAP BW/4HANA 2021
    Import Support Package 01 for SAP BW/4HANA 2021 (SAPK-30001INDW4CORE) into your BW system. The support package is available once SAP Note 3092812 ("SAPBWNews SAP BW/4HANA 2021 SP01"), which describes this SP in more detail, has been released for customers.

In urgent cases, you can implement the correction instructions as an advance correction.

You must first read SAP Notes 1668882 and 2248091, which provide information about transaction SNOTE.

To provide information in advance, the SAP Note(s) specified above might already be available before the support package is released. In this case, however, the short text of the SAP Note still contains the term "preliminary version".
