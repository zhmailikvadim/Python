Symptom

When you retrieve the details of a delivery document, you find that the batch/serial number information is not returned in the service layer response.

Reproducing the issue

Scenario:

  1. In SAP Business One client, create a delivery document with one or two batch managed Items.
  2. In SAP Business One client, create a return document for the delivery document created in step 1.
  3. In SAP Business One client, cancel the return document created in step 2.
  4. Try to retrieve the details of the delivery document created in step 1 using service layer as below:

GET /b1s/v1/DeliveryNotes(385)

Expected behavior:

Batch/serial number information is correctly returned in the service layer response in the case if you have cancelled the return document created for the delivery document.

Actual behavior:﻿

Batch/serial number information is not returned in service layer response in both the cases if you have cancelled the return document created for the delivery document.

Solution

SAP intends to provide a feature package or patch to solve the symptom that is described in this SAP Note.
