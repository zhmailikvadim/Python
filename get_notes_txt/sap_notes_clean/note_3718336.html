Symptom

The "New Entry" button is greyed out in the configuration activity "Define Common Divisions (ID: 102181)".

Environment

SAP S/4HANA Cloud Public Edition

Resolution

For each combination of "Sales Organization + Division", there should only be one "Customer-Division" acting as the source of master data. Allowing multiple entries for the same combination would create inconsistencies in the system, as it would not be able to determine which division should act as the source of master data.
