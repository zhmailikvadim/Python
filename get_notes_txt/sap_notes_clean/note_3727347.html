Symptom

  • An error occurs when attempting to execute query after importing the bionbi universe from different machine. The error message displayed is: "Invalid object name"

Environment

  • SAP Business Intelligence Platform 4.3 Service Pack 03 P11 or later

Reproducing the Issue

  1. Import the Sample bionbi from promotion management.
  2. Login to IDT.
  3. Open bionbi universe from repository.
  4. Drag any object in result object and Refresh
  5. Observe the error "Invalid object name Properties"

Cause

  • The issue is related to the preloaded tables in the SAP-provided Standard universe, which are not functioning correctly due to SDK and Virtual link getting broken.

Resolution

Import the lcmbiar file from same machine to avoid any issues also check connection is pointing to CMS (in some cases auditing database credentials are used and create issue) and refresh structure.

also the issue is not noticed in 4.3 sp4
