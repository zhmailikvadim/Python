Symptom

 In the standard system as currently delivered, you run the Payroll Driver USA (RPCALCU0 or RPCALCU0_CE) report for an employee who lives and/or work in New Jersey and has a supplemental payment. When the supplemental payment is processed, the system incorrectly sends the base wages in the BSI interface, resulting in an incorrect withholding tax calculation for New Jersey.

Other Terms

New Jersey, NJ, withholding tax, tax type 01, base wages, BW, PC00_M10_CALC, PC00_M10_CALC_SIMU, RPCALCU0, RPCALCU0_CE

Reason and Prerequisites

Correction of legal function

Solution

As a general rule, SAP recommends that you install a solution by applying a Support Package. However, if you need to install a solution earlier, use the Note Assistant (SNOTE) to implement the correction instruction. You can find more information about the Note Assistant in SAP Service Marketplace, under service.sap.com/note-assistant.
