Symptom

In CDS view C_MaintOrderComponentDEX field PurReqnOrResvnGeneration (Purchase Requisition or Reservation Generation) has an incorrect semantic annotation which indicates it is values are of type boolean (@Semantics.booleanIndicator: true). However, field PurReqnOrResvnGeneration has values 1/2/3/4. This might result in incorrect values of extracted/accessed data on consumer side e.g. NULL values.

Other Terms

MaintenanceOrder, Maintenance Order, Maintenance Order Component, Boolean Indicator

Solution

Apply the correction instruction. Annotation Semantics.booleanIndicator is removed for field PurReqnOrResvnGeneration.
