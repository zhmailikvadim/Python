Symptom

Install/Uninstall Equipment Superior Object exchange is not triggered. For example. An equipment is installed at the function location but the functional location is not replaced on device.

Apply the SAP note and then follow the manual instructions to update the Agentry SAP Framework Configuration (Transaction '/Syclo/ConfigPanel').

Other Terms

syclo, exchange, equipment, functional location, superior object, install, uninstall,  ERPADDON-516

Reason and Prerequisites

Program Error

Solution

Apply this SAP note in effected systems and then following manual instructions.
