Symptom

When executing sanctioned party list screening to check document addresses in background, the maximum memory consumption is too high after a while. As a consequence, program has been terminated due to no more memory available.

Other Terms

/SAPSLL/SPL_CHSB2MLO, /SAPSLL/SPL_CHSC2MLO, C2M, B2M, /SAPSLL/SPL_SAVE_SPL_RESULTS, TSV_TNEW_PAGE_ALLOC_FAILED.

Reason and Prerequisites

Program Error

Solution

Implement the attached corrections or import relevant support package.

Reason and Prerequisites

If you do not use the infrastructure for extraordinary remuneration, the corrections contained in this SAP Note have no relevance for your EEG Billing installation.

Please note that the infrastructure for extraordinary remuneration as such does not provide you with any executable functions in EEG Billing.

Solution

Implement the corrections only if you actually use the infrastructure for extraordinary remuneration.

First, implement the manual correction instructions. You can then implement the automatic correction instructions.
