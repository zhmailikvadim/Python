Symptom

- The issue occurs specifically in the Balance Sheet/Income Statement (F0708) app.
- When using the tab labeled "Balance Sheet ", the data for Balance Sheet  is displayed on the screen but does not appear in the PDF output.
- Clicking "Create PDF Preview" in the "Balance Sheet " tab results in only header information being displayed in the PDF file. 
- Using the "All Accounts " tab and clicking "Create PDF Preview" displays financial statement versions and amounts hierarchically in the PDF file.

Environment

• S/4 On Premise or Private Cloud  
• Reporting Apps  

Reproducing the Issue

1. Run the Balance Sheet/Income Statement (F0708) app.  
2. Navigate to the tab labeled "Balance Sheet".  
3. Verify that Balance Sheet assessment data is displayed on the screen.  
4. Click "Create PDF Preview" and observe that the PDF file only displays header information.  
5. Switch to the "All Accounts" tab and click "Create PDF Preview".  
6. Observe that the PDF file displays financial statement versions and amounts hierarchically.  

Cause

Incorrect definition of the Financial Statement Versions causes an unexpected display of the PDF review.  

Resolution

Please correct the definition of the FSV per the directions from note 0001154817 in order to get the expected results in the Fiori app.

See Also

0001154817 - Creating financial statement versions
