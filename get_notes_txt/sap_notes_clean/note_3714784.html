Symptom

After updating your ABAP or S/4HANA system, or upgrading a SAP UI technology (such as SAPUI5 or Unified Rendering), you encounter one of the following issues:

  • Custom themes are no longer displayed correctly.
  • Publishing or rebuilding a theme results in compilation errors, such as LessCompilationException or missing variables (e.g, variable <xyz> is undefined).

Other Terms

UI Theme Designer, Theme, Custom Theme, Upgrade, SAPUI5, UR, Unified Rendering, CRM Web UI, Compilation Exception, LessCompilationException, Variable Undefined, Web Dynpro ABAP, SAP GUI for HTML, S/4HANA, On-Premise

Reason and Prerequisites

The CSS generated for your custom theme is not compatible with the updated versions of the UI technologies.

Solution

  1. Ensure that the newest version of UI theme designer is installed, as described in SAP Note 1852401. 
  2. Make sure the latest version of the theming base content and Unified Rendering is installed. Refer to SAP Note 2090746 for instructions on installing the newest versions using TCI.
  3. Publish or rebuild your custom themes with the UI theme designer (transaction /UI5/THEME_DESIGNER) to genrate CSS for your custom theme that is compatible with the updated UI technology versions.
