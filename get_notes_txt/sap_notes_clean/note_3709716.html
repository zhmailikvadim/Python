Symptom

The FSM Activities fiori app consist of the two views:

  • Overview screen (general view)
  • Detail screen

Each FSM Activity contains multiple properties, including remarks.
The issue observed was:

  • Remarks were not displayed in the overview screen, even though they existed and were visible in the detail view.

This resulted in incomplete activity information being shown to end users, reducing the usability of the overview screen.

Other Terms

FSM, Connector, E4C, FSM Activity, Remarks

Reason and Prerequisites

The problem was caused by a coding issue in the Fiori application logic for the overview screen.

Solution

The application code has been corrected, please implement attached correction instructions.
