Symptom

With SAP Note 2333187, you created new ABAP Dictionary objects in your system. The corresponding source code is now created via the implementation of this SAP Note.

If you implement this SAP Note, checks for the new General Ledger are deactivated that were managed for company codes in the classic General Ledger.

Other Terms

FINS_RECON, R21, R23, R24, FINS_RECON(072), FINS_RECON072, 072, FINS_RECON037, FINS_RECON(037), 037, FAGLFLEXA, FAGLFLEXT, new General Ledger

Reason and Prerequisites

Program error

Solution

Import the relevant Support Package or implement the correction instructions in your system.
