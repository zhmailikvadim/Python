Symptom

List of changes

  • Updated job code list
  • Employment working hours will be rounded to one decimal starting from reporting period 01.2026

  • The solution in this SAP Note for reporting pay dates in A‑messages when the dates are in the past has been redesigned. The updated solution reports delta changes for garnishment amounts and withheld tax amounts that result from retro-calculated payrolls. If possible, it reports these amounts using the employee’s (PERNR) last pay date in the reporting period. If not, it keeps the original pay dates.
    Note: Because of this change, please review any manual delta corrections you created earlier. Either remove them (the new solution produces the correct output) or adjust them if the last pay date in the reporting period is not correct.
