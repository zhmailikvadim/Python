Symptom

Error "Running transaction **** is not allowed in ECP" is shown when running a transaction in an ECP system.

Environment

  • SAP SuccessFactors Employee Central Payroll

Reproducing the Issue

Run a transaction code in an ECP system.

Resolution

Several transactions are blocked for Employee Central Payroll. The blocked transactions can be split into the following categories:

TypeDescriptionExample: Blocked Transactions
Security Relevant Security Relevant transactions are part of this category. E.g. changes to audit log settings and ability to execute OS commands poses security risks and deviates from security standards maintained for ECP systems.SM49, SM69
System Stability RelevantTransaction that could harm the system stability are part of this category. E.g. transactions for client administration and file upload/download poses a risk for the system stability.CG3Y, CG3Z
