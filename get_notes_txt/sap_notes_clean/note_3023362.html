Symptom

This SAP Note provides fixes for the following issues:

  • Net value computation issue in DR/CR Mix/Net matching.
  • One to One suggestion rule for Proposed matching was not considering 1% amount tolerance to match the transactions.

Reason and Prerequisites

This SAP Note can be applied to SAP Liquidity Management for Banking (LMS) 6.3 SP1.

Solution

  • Updated the DB view to correct the calculation of net value in DR/CR mix matching.
  • Updated the DB views and procedures to allow 1% amount tolerance for Proposed matching for One to One suggestion rule.

Affected Files:

  1. <INSTALL_BASE>/applicationserver/webapps/lmsui/ fs.lms.manualmatchingwithdrcrmix/view/ ManualMatchingWithDrCrMix.view.xml
  2. <INSTALL_BASE>/applicationserver/webapps/lmsui/ fs.lms.manualmatchingwithdrcrmix/controller/ ManualMatchingWithDrCrMix.controller.js
  3. <INSTALL_BASE>/applicationserver/webapps/lmsui/fs.lms.regularmatching/controller/RegularMatching.controller.js
  4. <INSTALL_BASE>/applicationserver/webapps/lmsui/Component-preload.js
  5. <INSTALL_BASE>/applicationserver/webapps/lmsui/WEB-INF/classes/com/sap/lms/gateway/data/ManualMatchingCRDRMixDataHandlerImpl.class
  6. <INSTALL_BASE>/applicationserver/webapps/lmsui/WEB-INF/classes/com/sap/lms/gateway/data/SuggestedMatchingDataHandlerImpl.class

Affected Jars:

  • com.sap.lms.dataaccess-6.3.1-FINAL.jar

  5. Replace RegularMatching.controller.js file to “<INSTALL_BASE>/applicationserver/webapps/lmsui/fs.lms.regularmatching/controller/” location.
  6. Replace Component-preload.js file to “<INSTALL_BASE>/applicationserver/webapps/lmsui/” location.
