Symptom

MDG EAM 2023, 2024,

When you are searching and adding EAM objects to MOCR change request along with Equipment or Functional Location which are already locked in inheritance change request(table /UGI/EAM_INHER), you are unable to add the objects which are not locked in the CR to MOCR.

Other Terms

MDG EAM, Add to MOCR, /UGI/EAM_INHER, Inheritance Change Request

Reason and Prerequisites

Program Error

Solution

Implement the correction instruction

When you are searching and adding the EAM objects to MOCR change request along with Equipment or Functional Location which are already locked in inheritance change request, you are unable to add any of these objects to MOCR.
