Symptom

Users in a form route map are able to see routing comments by others users, even in those steps where the comments were not for them.

Environment

SAP SuccessFactors Performance Management

Reproducing the Issue

  • Open the performance inbox
  • Form Information
  • Audit trail shows comments from other users, including route steps the user was not part of

Cause

This is a Known issue.

Resolution

Known issue, resolved for 1H 2026.

Please tag this KBA to be notified of future updates:

  • 2171560- How to mark a KBA or SAP Note as a favorite and get notified when it is updated - SAP for Me https://me.sap.com/notes/2171560 

Workaround

As a partial mitigation, an admin can enable the Disable Info button setting in the Form Template Settings page to block access to audit-trail comments. This action blocks audit-trail access for all users, not only the affected ones.
