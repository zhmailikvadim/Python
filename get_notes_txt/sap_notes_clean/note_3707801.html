Symptom

When copying an Opportunity, the Probability (%) field is either empty or has a different value in the new Opportunity.

Environment

SAP Sales Cloud Version 2

Reproducing the Issue

  1. Open any Opportunity where the Probability (%) field has a value
  2. Copy the Opportunity

Result: in the new Opportunity, the Probability (%) field is blank or has a different value to the original Opportunity.

Cause

The Probability (%) field is based on the Opportunity's Sales Phase.  When copying an Opportunity, the Sales Phase is reset, so if the new Opportunity is in a different Sales Phase it will have a different Probability.

Resolution

This is the expected behaviour of the system.

Keywords

copy opportunity, opportunity probability, sales phase
