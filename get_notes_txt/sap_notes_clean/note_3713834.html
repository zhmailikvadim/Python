Symptom

After entering the company code, date and other parameters in the Aging Report - Accounts Payable - China app, the report will be generated. Upon reviewing the details, it was discovered that some data were not included.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Generate the aging report in SAP S/4HANA Cloud Public Edition.
  2. Observe that the journal entry is missing from the generated report.

Cause

When generating the aging report using this app, the "special G/L indicator" was selected for filtering. As a result, some journal entries were excluded because they did not have the selected value.

Resolution

The special G/L indicator is manually entered during the posting process. It is necessary to ensure that the value of this field has been set correctly according to the business requirements. Or, during the generation of the aging report, modify the filtering criteria so that these journal entries are included.

Keywords
