Symptom

There are a few reports that were designed for internal SAP use.  Using them on production systems could cause unexpected result.

Other Terms

CRM_FM_FPO_AGR_CONSISTENCY, CRM_FM_FPO_CONSISTENCY

Reason and Prerequisites

Customers should avoid using reports that were designed for SAP internal use.

Solution

Please apply the correction via SNOTE.

This SAP Note restricts the use of SAP internal reports.
