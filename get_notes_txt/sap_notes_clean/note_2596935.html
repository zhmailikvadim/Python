Symptom

  • No site change is possible after logon to SAP ME.
  • List of sites available is empty.

Environment

  • SAP Manufacturing Execution (MFG-ME)

Reproducing the Issue

  1. Logon to NetWeaver Administrator (NWA).
  2. Create/copy a user, add a default site for it and save.
  3. Logon to SAP ME with this user.
  4. Add it to the proper user group at User Maintenance and save.
  5. Try to change site - there is no site to change for.

Cause

The user should be assigned to a user group by site_admin user, not by the user itself.

Resolution

  1. Logon to SAP ME with site_admin.
  2. Go to User Maintenance, retrieve the user created/copied.
  3. Assign it to the proper user group and save.
  4. Logout with site_admin and logon with the user mentioned.
  5. The sites available will be visible for the affected user and site switch is possible.

Keywords
