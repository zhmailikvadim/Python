Symptom

  • SQL statement for database object "/SAPSLL/CTSTXT" was executed with an error.
  • XPRA error during GTS upgrade.
  • GTS E4H Conversion stuck in XPRAS_AIMMRG.

Environment

  • Global Trade Services (GTS)
  • Global Trade Services / Logistics Services (SLL)
  • SAP Global Trade Services
  • SAP Global Trade Services, Edition for Hana

Reproducing the Issue

  • Run the conversion to GTS Edition for SAP HANA 
  • The conversion is not possible due to the following error:
  • SQL statement for database object "/SAPSLL/CTSTXT" was executed with an error (SQL Error Code: "2.048")

Cause

Missing Time zone in SAP HANA

Resolution

  • The issue arises from the use of time zones that are not compatible with the SAP HANA Database or are not recognized by default. 
  • Please add the time zones like described in the SAP Note ##1791342 - Time Zone Support in SAP HANA

 

See Also

SAP Note: 1791342 - Time Zone Support in SAP HANA

Keywords
