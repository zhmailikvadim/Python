Symptom

No standard SAP-delivered field is available for maintaining or transmitting the KPD code for article master or billing item.

Environment

  • Document Compliance Croatia
  • SAP ERP
  • SAP S/4HANA
  • SAP Document and Reporting Compliance service

Reproducing the Issue

  1. Attempt to maintain KPD code.
  2. Observe that no standard SAP-delivered field exists for maintaining or transmitting the KPD code for article master or billing item.

Cause

Missing maintenance of Value Mapping HR_ACCOUNT_CLASSIF.

Resolution

  1. Configure value mapping HR_ACCOUNT_CLASSIF in transaction /N/AIF/VMAP for Namespace /EDOHR.
  2. Maintain the respective GL account used for FI Invoices in value mapping to ensure the KPD code is populated.
  3. If the value mapping does not exist, refer to KBA 3682215 - Missing value mapping entries in transaction code /AIF/VMAP for namespace /EDOHR in DRC Croatia - SAP for Me.

See Also

  • Refer to SAP Note: 3679698 - Croatia Electronic Invoicing - Electronic Document Processing - FAQ
  • Refer to KBA: 3682215 - Missing value mapping entries in transaction code /AIF/VMAP for namespace /EDOHR in DRC Croatia
