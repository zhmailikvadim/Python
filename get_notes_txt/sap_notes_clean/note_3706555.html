Symptom

You are trying to cancel failed messages from the Web Service Message Monitoring but the Cancel button does not work for some messages.

Environment

SAP Cloud for Customer V1

Reproducing the Issue

  1. Navigate to the Web Service Message Monitoring.
  2. Select the message to be cancelled.
  3. Click on Cancel.
  4. Result: The status of the message does not change to 'Canceled'.

Cause

Some messages cannot be cancelled from the UI.

Resolution

Create a support case if you want SAP to cancel those messages on your behalf.

See Also

Web Service Message Monitoring | SAP Help Portal

Keywords

Web Service Message Monitoring, WSMM, Asynchronous Messages, Cancel Failed Messages
