Symptom

Work instructions having links which contain special characters are not displayed correctly in the Perform Operation Activity application.

For example, "&nbsp" is displayed for empty space, "&lt" is displayed for "<" and "&gt" is dispalyed for ">".

Other Terms

Work Instruction, Perform Work, Perform Operation Activity, Links, Special characters, References

Reason and Prerequisites

Program Error

Solution

After implementing this SAP Note, links in the work instruction are displayed correctly
