Symptom

I want to customize the Billable Per Diem, at time sheet level.

Currently when a per diem value is entered on the work order this automatically applies to every day hours are submitted on a time sheet.

I want to know if there a way for it to only apply for certain days

Resolution

Currently there is no possibility to edit it in the product.

In case this is an important feature for the business, you need to submit this idea via enhancement request (https://me.sap.com/notes/11). So, this idea will be evaluate by Product Manager team, and can be further consider for future releases.

Keywords

customize Billable per Diem, updated/edited Billable per Diem
