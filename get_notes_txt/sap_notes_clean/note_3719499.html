Symptom

Transportation Management in SAP S/4HANA: when navigating from document flow to an EWM Inbound Delivery, the target application starts, but requests the user to "add an additional filter".

Reason and Prerequisites

With SAP FIORI FES 2023 FOR S/4HANA FPS2 (or with SAP Note 3446733), the app descriptor of Fiori App F1706 ("Change Inbound Deliveries") was changed: the parameter for Warehouse was renamed to EWMWarehouse. Therefore, any calling applications must ensure that the expected parameter name is sent.

Solution

Implement the attached correction instructions or install the corresponding Support Package.

If you use S/4HANA On-Premise 2023 FPS0 or FPS1, then apply the attached correction instructions only if you applied the correction instructions of SAP Note 3446733.
