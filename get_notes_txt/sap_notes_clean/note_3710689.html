Symptom

Error 403 occurred during API file transfers from Workflow to Agent Performance Management, resulting in a session timeout.

Environment

  • SAP Sales Performance Management
  • Agent Performance Management

Cause

As described in the HTTP Response Codes documentation available in the SAP Help Portal, the 403 error indicates that one or more of the authorization edits failed. This failure could occur for one of several reasons:

  • "System APIs" Feature is not enabled in the target PRODUCER instance.
  • Cald-sessionid not included in HTTP request header. Note that successful authentication will return the Cald-sessionid to use in subsequent API calls.
  • Cald-sessionid does not reference a valid session in the PRODUCER instance.
  • Cald-sessionid references a session that has expired.
  • Cald-sessionid is valid and active but the Client IP of the request does not match the Client IP of the authentication request resulting in the session creation.
  • Authenticated User does not have access to the "System APIs" AppArea.
  • Authenticated User belongs to a Security Group that is not authorized to access the specified API.
  • Client IP address is not within the range specified by the system.api.ip.whitelist option (This edit is omitted when the option is not configured meaning any client IP address is enabled access).

Resolution

It is recommended to verify the items mentioned in the section above, especially those that reference Cald-sessionid. These items can justify a 403 error related to timeout.
Considering this, it is recommended that:

  1. The retry logic for API posts be reviewed - if during the posting process the call identifies a terminated session, the retry logic should make it fetch the active session to successfully post.
  2. Review the system.api.session.timeout option - verify whether the default option of 30 minutes is sufficient for the processes to be implemented via API.
    1. Further explanations about the system.api.session.timeout option can be found in the SAP Help Portal: Session Timeouts.
