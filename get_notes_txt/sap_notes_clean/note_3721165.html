Symptom

When you use transaction ADCO99 to set the completion status for service management orders and select the "Free Selection" key to choose additional selection conditions, the system sets incorrect selection criteria. This leads to erroneous results and performance issues because the system selects more records than expected.

Other Terms

Service orders, Maintenance orders, EAM orders, System status, CO99, RIARCORVS, IW31, IW32, IW33

Reason and Prerequisites

Program error.

Solution

Implement the correction instructions attached in this note.
