Symptom

Long text on component level saved via API_MAINTENANCEORDER, Fiori application Manage Maintenance Orders (F5241), or the WebDynpro applications Create/Change Maintenance Order (W0017) is displayed incorrectly in transactions IW32/IW33.
The first 40 characters of the long text are not displayed.
This only happens for components where a material number is specified.

Other Terms

API_MaintenanceOrder, F5241, Component Long text, MATK, RESB-POTX1, IW32, IW33, longtext

Reason and Prerequisites

Program error.

Solution

Implement this SAP Note.
