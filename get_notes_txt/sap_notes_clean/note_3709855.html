Symptom

• In the billing document, the 'Services for Object' button is not available.
• Transactions VF03 do not display the 'Services for Object' icon.
• When selecting System -> Services for Object, the following error message is displayed: "No service available Message no. SGOS_MSG002".

Environment

  • Sales And Distribution (SD)
  • Logistics Execution (LE)
  • SAP R/3
  • SAP R/3 Enterprise 4.7
  • SAP ERP Central Component
  • SAP ERP
  • SAP Enhancement package for SAP ERP
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA
  • S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Open the billing document in VF03.
  2. Observe that the 'Services for Object' button is not displayed in the top left-hand corner of the billing document header. 

Cause

• 'Services for Object' is only available for Dialog Users.
• The User Type 'Service' is maintained for the user in SU01.

Resolution

1. Open transaction SU01 for the applicable user.
2. Change the User Type in Logon Data to 'Dialog'.
