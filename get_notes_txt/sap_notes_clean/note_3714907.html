Symptom

  • Error codes 404 and 500 are encountered during the preloading of JavaScript (JS) and JSON files.
  • The issue manifests when accessing the webide interface for SAP Incentive Management.

Environment

SAP Incentive Management

Reproducing the Issue

  1. Enable Zscaler on your system
  2. Access the webide interface for SAP Incentive Management.
  3. Log in to the system.
  4. Observe the errors displayed on the home page during loading.

Cause

  • SSL Inspection interference: Modifications to the certificate or header of sub-resources by SSL Inspection can cause the browser to reject files with a network error leading to 404/500 errors.
  • MIME Type mismatches: Incorrect MIME type interpretation for JSON or JS files by the origin server or intermediary systems can result in security filters blocking the content.
  • PAC file or tunneling issues: Discrepancies in routing requests between nodes due to PAC file configurations or missing authentication tokens can block sub-resources.
  • Browser-side JavaScript disabled: Restrictions or disabling of JavaScript in the browser can lead to authentication or preloading script failures.

Resolution

  1. Add the specific application domain to the SSL Inspection Bypass list to prevent interference.
  2. Ensure the server sends `application/json` for JSON files and `application/javascript` for JS files to avoid MIME type mismatches.
  3. Use the Web Check tool to verify that all requests are routed to the same Service Edge and resolve PAC file or tunneling issues.
  4. Verify that JavaScript is enabled in the browser settings and ensure no strict "NoScript" style extensions are active.
