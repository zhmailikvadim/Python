Symptom

According to Malaysia Service Tax Policy No. 2/2025 (Amendment No. 3) dated 23 January 2026, certain rental or leasing services are subject to an 8% statutory service tax rate with a 2% exemption. This results in an effective tax charge of 6%. Users need to configure the tax box mappings to ensure correct reporting in the SST-02 forms, where:
•    The taxable service value should be declared under the 6% tax rate.
•    The same service value should also be declared in Field Field 18(c)(3) – Other Exempted Services to reflect the 2% exemption.

This note provides configuration guidance to report this tax exemption scenario correctly in the SST-02 return forms submitted to the tax authority in Malaysia.

### Background

The Royal Malaysian Customs Department has issued guidance under Service Tax Policy No. 2/2025 that allows the following invoicing approach for the 2% service tax exemption:

Invoicing Method: Rental value with 8% service tax shown, followed by a 2% exemption adjustment, resulting in net 6% service tax charged.

For SST-02 reporting purposes:

  • The net taxable service value should be declared under the 6% tax rate in Part C.
  • The service tax collected at 6% (after exemption) should be reported in relevant service tax fields. 
  • The 2% exemption amount should be reported in Part D, Field 18(c)(3) – Other Exempted Services

### Step-by-Step Configuration Guide

Through proper configuration of tax groups and tax box mappings, users can achieve the correct SST-02 reporting without requiring system enhancements. The following configuration has been tested and validated in system QA7_931.

Test Example Reference:

- Tax Code: “A2”

- Sample Invoice: “1800000005”

- Test System: “QA7_931”

The configuration involves two main steps in the Define Enhanced Tax Box Structure (ID: 103011) configuration activity:

1. Define Tax Groups (mapping tax codes to the internam tax groups)

2. Assign Tax Groups to Tax Boxes (mapping internal tax groups to tax boxes)

### Configuration Overview

Tax Groups Required:

  • “SST_SER_E8” for 8% service tax base amount
  • “SST_SER_E2” for 2% exemption amount

Tax Boxes to be Configured:

  • Box “10” for Part C: Total Service Tax Charged
  • Box “11C_B” for Part C: Taxable Service Value at 6%
  • Box “11C_T” for Part C: Service Tax at 6%
  • Box “18C3” for Part D: Other Exempted Services (Field 18(c)(3))

### Configuration Steps

#### Step 1: Define Tax Groups (Map Tax Codes to Tax Groups)

Create new tax codes for services with 8% rate in the Define Tax Codes for Sales and Purchases (ID: 101016) configuration activity and map the tax code to the relevant tax groups in the Define Enhanced Tax Box Structure (ID: 103011) configuration activity.

Configure the following tax groups:

Tax Group: SST_SER_E8

- Tax Code: `A2` (or your tax code for services with 8% rate)

- Transaction: `MWS` (Service Tax)

- Document Type: *(as applicable)*

- Free Classification: `8.00` (8% rate)

Tax Group: SST_SER_E2

- Tax Code: `A2` (same tax code)

- Transaction: `MWS` (Service Tax) 

- Document Type: *(as applicable)*

- Free Classification: `2.00` (2% exemption rate)

Note: The validity dates in the tax groups must match the validity dates of the tax codes.

#### Step 2: Assign Tax Groups to Tax Boxes

Access the configuration via the Define Enhanced Tax Box Structure (ID: 103011) configuration activity.

Create or update the tax box structure for Malaysia SST02 with the following assignments:

Key Configuration Points:

Tax Box 11C_T has two entries:

  - First entry with `SST_SER_E8` (no operation) adds the 8% base tax

  - Second entry with `SST_SER_E2` with `-` (minus operation) subtracts the 2% exemption

  - Result: Net 6% service tax displayed

Tax Box 18C3 captures the service value subject to exemption for Part D reporting.

#### Step 3: Save and Activate

1. Save your tax group and tax box structure configurations.

2. Activate the new version of the tax box structure.

3. Set the validity period to align with the effective date of the legal change.

#### Step 4: Testing and Validation

a) Post a test transaction using the configured tax code

    • Create a test document with the tax code (e.g., A2).

    • Ensure the transaction posts successfully.

b) Run the SST02 report

    • Execute the Malaysia SST-02 return for the test period.
    • Review the output XML or report display.

c) Verify the following in the SST02 report:

    • Part C, Tax Box 10: Shows the total service tax collected (at 6% effective rate).
    • Part C, Tax Box 11C_B: Shows the taxable service base amount.
    • Part C, Tax Box 11C_T: Shows the net service tax amount (8% statutory rate - 2% exemption = 6% collected).
    • Part D, Tax Box 18C3: Shows the service value subject to exemption (the base amount on which 2% exemption applies).

### Expected Result

After configuration, your SST-02 report will correctly show:

    • The service value and tax at the effective 6% rate.
    • The 2% exemption reflected in Field 18(c)(3) as required by tax authorities.

### Key Points to Remember

    • The configuration involves two separate steps: defining tax groups and assigning tax groups to tax boxes.
    • The same tax code (`A2`) is mapped to two different tax groups (`SST_SER_E8` and `SST_SER_E2`).
    • Tax Box `11C_T` has two assignments with different arithmetical operations (one positive, one negative with `-`).
    • The negative operation (`-`) on `SST_SER_E2` in Box `11C_T` performs the exemption calculation
    • No system enhancement is required - standard customizing configuration is sufficient.
    • This configuration follows the Configuring Tax Box Mappings for SST Report as documented in SAP Help Portal.

### Additional Notes

    • This configuration is based on the guidance from Service Tax Policy No. 2/2025 (Amendment No. 3) for rental and leasing services.
    • The configuration follows SAP's standard DRC (Document and Reporting Compliance) framework
    • Tax group validity dates must match the tax code validity dates.
    • For Malaysia e-Invoice (MyInvoice) alignment, the invoice should clearly show the 8% statutory rate with the 2% exemption deduction.
    • It is recommended to maintain a separate tax code specifically for services subject to this exemption for easier reporting and reconciliation.
    • Always perform thorough testing in your development/test environment before implementing in production.
    • Coordinate with your tax advisors to ensure compliance with all SST02 reporting requirements.
