Symptom

SPDD adjustments were performed in the ACT_UPG phase when the SUM process was started in Prerequisite Check Extended mode.

There is no option to export the SPDD TR.

Environment

SAP ABAP system.

Cause

The shadow instance is not a complete system.

Resolution

The SPDD TR can only be exported after system switch in Porstprocessing roadmap.

The system is already on target release when the SPDD TR can be released in an upgrade procedure.

SPDD TR export is not possible when the SUM is running in the Prerequisite Check Extended mode. Since it only runs system check, no actual system upgrade is performed.

Keywords

SUM, SOFTWARE UPDATE MANAGER, check mode, SPDD, adjustment, export, TR
