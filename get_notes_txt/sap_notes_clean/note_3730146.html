Symptom

  • Detailed error message in ABAP report /fldq/ad_rept_display_errlog shows
    - Error Code:  DQ07:   Data Quality microservice returned an error. Contact your system administrator
    - Location Code: 130 : CALL_DQ_ENGINE_ADR_VALIDATE method of CL_DQ_ENGINE_DQAAS
    - 401 - Invalid client certificate
  • Service key has many BEGIN/END certificate tags - multiple certificates appear in the JSON.

Environment

  • SAP Data Quality Management, version for SAP Solutions
  • SAP Data Quality Management, microservices for location data

Cause

The correct certificate was not exported from STRUST and used in the creation of the service key.

Resolution

Export the S/4 HANA or MDG certificate from STRUST and use it in the creation of the service key.
