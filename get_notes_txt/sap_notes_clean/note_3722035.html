Symptom

  • In Fiori App F4006 (Upload Material Inventory Prices), when attempting to download a template, error message "Chosen Combination is not correct" arises.
  • The error appears in red, and the "Download" button is disabled.

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

SAP S/4 HANA Finance

SAP S/4 HANA

Reproducing the Issue

  1. Navigate to the Fiori app "Upload Material Inventory Prices" (App ID F4006).
  2. Click on the "Download template" button.
  3. Select the option "With existing material prices".
  4. Enter the following details:
  • Company code.
  • Plant.
  • Currency type/Ledger/Valuation view.
  • Value date.
  • Materials.

     5. Click on the "Download" button.

     6. Observe the error message "Chosen Combination is not correct" displayed in red, and the "Download" button is disabled, as per below image: 

Cause

Program issue. Normally, due to missing leading zeros for a check (specially when trying to download a template with filtered materials if the material number is numeric).

Resolution

Implement SAP Note 3643277: "[F4006] does not work with filtered material number".
