Symptom

After maintaining information in the "Data Sources" section of the Relationship Intelligence setup and saving successfully, clicking to edit the Data Sources again results in the previously entered data being cleared and the fields turning red.

Environment

  • SAP Cloud for Customer - Relationship Intelligence Add-on
  • SAP Sales Cloud Version 2

Reproducing the Issue

  1. In Cloud for Customer, navigate to Intelligent Sales Execution workcenter. 
  2. Go to Relationship Intelligence Add-on.
  3. Edit the Data Sources section and enter the necessary data.
  4. Save the information.
  5. Click to edit the Data Sources section again.
  6. Observe that the previously entered data is cleared, and the fields become red.

Cause

The "Data Sources" section displaying blank fields is expected behavior for this configuration.

Resolution

The setup is saved successfully despite the fields turning blank upon re-editing. Note that this behavior is expected for this configuration and does not result in functional impact.
