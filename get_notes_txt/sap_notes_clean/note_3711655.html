Symptom

Unable to send the project owner details from Ariba to the ERP system through an Outbound/Request_In payload.

Cause

Resolution

  1. There is no standard supplier field mapping available to transfer the project owner from Ariba to ERP through an Outbound/Request_In payload.
  2. As a workaround, configure a new question with a custom generic mapping.
  3. Manually answer the newly configured question.
  4. Trigger an Outbound/Request_In payload.

Keywords

project owner, outbound payload, ERP integration, outbound request_in, username mapping, custom generic mapping
