Symptom

"Enterprise Manager heartbeat" metric displays a value of "0 Heartbeat is missing in System monitoring. 

In the sapdpcpush.log following error is showing:

6/04/25 10:02:29 PM UTC [ERROR] [Manager.SapPush.EFP] failed to push 10000 metrics

Environment

SAP Focused Run

SAP Solution Manager 7.2

Reproducing the Issue

Check status of metric "Enterprise Manager heartbeat"

Cause

The EM has reached the default limit of 10000 metrics, so some metrics may be suppressed.

Resolution

Edit sap/EFP.e2emai.properties and add this line:

"alertqueue.max.entry.count=20000"

This will increase the limit of metrics that can be collected.
