Symptom

  • With this SAP Note, we provide changes that are usually treated as manual activities within SNOTE (for example, packages, ABAP dictionary objects, message classes).
  • The implementation of this SAP Note has no impact on your existing functionality.
  • After applying this SAP Note, you need to implement the primary SAP Note 3705056. 

Other Terms

UDO

Reason and Prerequisites

This SAP Note is only relevant for Payroll Belgium (PY-BE).

You need to implement SAP Note 3705056.

Solution

Implement the attached correction instructions by using transaction SNOTE, before applying the primary SAP Note 3705056. By doing so, the report NOTE_3705056 gets installed in your SAP system.
