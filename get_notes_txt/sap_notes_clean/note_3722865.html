Symptom

Dump occurs when creating a new Custom Code Analysis project.

Category               ABAP programming error                                                                                                                                                                                                                                                      

Runtime Errors         RAISE_SHORTDUMP                                                                                                                                                                                                                                                             

Except.                /BOBF/CX_FRW_FATAL                                                                                                                                                                                                                                                          

Development Object     CLAS CL_SCA_DS_REMOTE_CONNECTION                                                                                                                                                                                                                                            

Application Component  BC-DWB-TOO-ATF

Access denied for SCA_REMOTE_DATA_VERSION on *. Expose the function module in your Cloud Connector.

Resolution

In SAP Cloud Connector, all function modules that cloud system should be able to call in the on-premise system need to be included in the allowlist.

For the Custom Code Migration app, a complete list of all required functional modules is provided.

Refer to the SAP Note "2861842 - Custom Code Migration in SAP BTP ABAP environment: Set up SAP Cloud Connector".

Keywords

Custom Code Analysis, Cloud Connector, function module, access denied, RAISE_SHORTDUMP, /BOBF/CX_FRW_FATAL, CL_SCA_DS_REMOTE_CONNECTION, SCA_REMOTE_DATA_VERSION
