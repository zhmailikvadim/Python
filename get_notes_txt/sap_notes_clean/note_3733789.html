Symptom

  • PPIM and DPS horizon values are not integrating correctly between S/4HANA and IBP (values revert to PPIM = 0 and DPS horizon = 0 after transfer)
  • Updates sent using the report SYPP_MARC_EXT_MNT appear in IBP logs but do not result in changes to the location-product
  • Manually updated values in IBP are overwritten back to 0 after sending from S/4HANA
  • No error messages were mentioned

Environment

  • SAP Integrated Business Planning for Supply Chain
  • SAP S/4HANA

Cause

Resolution

  1. Ensure all relevant installation notes are fully implemented. For guidance, review KBA: 3648021 - IBP: RTI - Multiple Planning and Multiple CIF (MultiCIF - MCIF) scenarios
  2. In SM30, open table/view SYPP_V_CONFIG and ensure both options are enabled:
    1. Activate Production Planning Integration with IBP.
    2. Multiple Use of the Core Interface.
  3. In transaction MM02 (material master), on the "Advanced Planning" tab for the relevant location-product, ensure the “Advanced Planning” indicator is selected.
  4. Verify that DPS horizon is only defined for location-products with PPIM “Detailed Scheduling Delegated (1)” or “Short-Term Production Planning and Detailed Scheduling Delegated (3)”. Adjust PPIM accordingly before transferring.
  5. Review the guidance for changing PPIM and related attributes in IBP master data; refer to: Changing Master Data.
  6. Ensure the BAdI implementation for CIF material mapping is in place: method CHANGE_MATERIAL of BAdI CIF_MATERIAL_MAP_IBP should be implemented according to example SYPP_CIF_MAT_MAP_IBP (“Enrich Data Before Transfer to IBP”).
    1. You need to highlight all rows that you want to send.
    2. Method CHANGE_MATERIAL of BAdI CIF_MATERIAL_MAP_IBP  needs to be implemented (for details read the system documentation of the report)
    3. The materials needs to be in an active integration model.
  7. Run transaction /SAPAPO/SYPP_HLT_CHK to validate that all configuration and master data prerequisites for the chosen PPIM are met for the affected location-products.
