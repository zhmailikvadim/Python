Symptom

In SAC data analyzer when accessing the Technical Details via the Info button, the Information Provider is displayed using its technical name (ID) instead of the maintained description. 
This results in the same value appearing for both the Name and Description fields, despite different descriptions being defined in the backend system.

Environment

SAP ANALYTICS CLOUD

Reproducing the Issue

  1. Login into SAC
  2. Open Data analyzer created from BW query
  3. Open the Technical Details section
  4. Notice in the information provider , the name and description are the same 

Cause

This behavior is by design

Resolution

This behavior is currently by design in SAP Analytics Cloud. The Technical Details section shows the technical name (ID) of the BW Information Provider instead of the description.

If you need the description to be displayed, you can submit an enhancement request as mentioned in the SAP KBA: 2424376 - How do you create an enhancement request and provide feedback for SAP Analytics Cloud? accessible on the SAP ONE Support Launchpad. 

You can find more information how  the « Idea Place" is working in the KBA 1515837
