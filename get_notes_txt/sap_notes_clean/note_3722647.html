Symptom

  • In transactions ME22N and ME23N, the system allows navigation to the earmarked funds document via double-clicking on the earmarked funds number in the simple account assignment classification screen.
  • Navigation to the earmarked funds document is not possible via double-clicking on the earmarked funds number in the multiple account assignment classification screen.

Environment

  • Materials Management (MM)
  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA

Reproducing the Issue

  1. Open transactions ME22N or ME23N.
  2. Navigate to the multiple account assignment classification screen.
  3. Attempt to double-click on the earmarked funds number to access the earmarked funds document.
  4. Observe that navigation is not possible from the multiple account assignment classification screen.

Cause

  • The multiple account assignment classification screen is designed as an ALV overview for distribution purposes. This screen does not support double-click navigation or direct access to field details due to its technical design and underlying UI technologies.

Resolution

  • You need to open the account assignment detail view or navigate through the item account assignment tab.  
    • For accessing field information correctly from the multiple account assignment screen, please follow these steps in multiple account assignment screen:  
      - Select the relevant line.  
      - Click on Account Assignment Details using the icon or button provided.  

    This is standard system behavior and does not indicate an issue.
