Symptom

Users are able to select the link to enroll into the benefits (even if there is no benefit in the process which they are eligible for), 
They then receive an error message when they click 'Continue': "You are not eligible for any benefits"

Environment

SAP SuccessFactors Employee Central

Reproducing the Issue

  1. Go to the benefits section of the profile for a sample user
  2. Click the link to enroll into the open enrollment benefits in the Benefits Actions card
  3. After clicking the link you are directed to review the dependents
  4. Click the ‘Continue’ button
  5. The following error message appears:“You are not eligible for any benefits"

Cause

Currently, the system evaluates rules individually for each benefit within the process, which causes performance issues.

Due to this limitation, it is not possible to determine a process’s eligibility in advance before displaying the open enrollment link. Since it is not feasible to evaluate all the benefit-level rules beforehand, the open enrollment link is displayed to the user even if they are not eligible for the process.

The error message will then appear after trying to move through the open enrollment process as the user is then deemed not to be eligible for any of the benefits.

Resolution

An enhancement has been raised to help introduce an eligibility rule at the Benefit Enrollment Process Configuration level. You may request your CSM to monitor 'BEN-22135' for the status of the enhancement.
