Symptom

  • The "Certificate of Residence Indicator" field in the report PE_WHT_PDT617 displays the value "0" instead of the the value "1".
  • The "Residence Certificate Number" field in the report PE_WHT_PDT617 displays the value "0".

Environment

SAP S/4HANA Cloud Public Edition.

Reproducing the Issue

  1. Go to Run Statutory Reports app.
  2. Execute the Peru Foreign Vendor Withholding Tax Return - PE_WHT_PDT617 report.
  3. Observe that the "Certificate of Residence Indicator" and "Certificate of Residence Number" fields show the value 0.

Cause

Identification Number with ID Type "PE_RES" is not maintained for Business Partner.

Resolution

  1. Maintain the Identification Number with ID Type "PES_RES" for the affected Business Partner under Manage Business Partner Master Data app.
