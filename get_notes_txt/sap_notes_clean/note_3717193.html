Symptom

  • In SAP S/4HANA 2025, the field 'Source ID' is displayed in transaction SCU0 Comparison instead of the 'R/3 connection' field as of SAP S/4HANA 2025.
  • Users are unable to perform customizing comparison using the 'RFC Connection' in SCU0.

Environment

  • Customizing Cross-System Tools

Reproducing the Issue

  1. Run transaction SCU0 in SAP S/4HANA 2025.
  2. Observe that the 'Source ID' field is displayed instead of the 'RFC Connection' field in Comparison.

Cause

  • The SACF scenario 'BC_CC_COMPARE - Customizing Comparison - Maintenance Objects' is active in the system, which leads to the display of the 'Source ID' field for customizing comparison via communication arrangements setup between local and remote systems.

Resolution

  • Verify if the SACF scenario 'BC_CC_COMPARE - Customizing Comparison - Maintenance Objects' is active in the system using transaction SACF.
  • Deactivating this SACF scenario would switch back the field to 'R/3 connection', which is able to be used to perform comparison as before with SM59 destinations. 
  • Refer to KBA Note 3700798 to deactivated it: Set the scenario "BC_CC_COMPARE" under the Production Scenario option to status "I" (Inactive) using transaction code SACF.
