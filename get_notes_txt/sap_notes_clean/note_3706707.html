Symptom

In method /UCOM/CL_ISU_DEVICE->/UCOM/IF_ISU_DEVICE~SET_BASE_REGS_READING_REL the meter reading relevance of registers is changed during device processes. In this method, the register group should not play any role. In particular, the method should set NABLESEN = '' even if the register group has the setting of NABLESEN = 'X', because the register group is just a template for setting register data. Once a register is installed it should no longer play any role what the register group setting was.

Reason and Prerequisites

The reading relevance of registers is influenced in IM4G by the system parameters

IM4G_DISABLE_MR_DEACT_BASE_REG

IM4G_MME_DISABLE_MR_NON_BILL

and by the register billing data customizing. If a register is to be set billing relevant it must first be set to reading relevant.

Solution

Implement the correction with /nSNOTE
