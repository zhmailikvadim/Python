Symptom

When the original good receipt document from PO is archived, user can not cancel the material document or create returns material document reference to the original material document.

Environment

  • Materials Management (MM)
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA
  • SAP ERP
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Go to t-code MIGO.
  2. Choose A03- Cancellation + R02- Material Document or A02 - Return Delivery + R02 - Material Document and input the reference Material Document.
  3. The error pop-up M7 063 Document XX does not exist in fiscal year XXXX.

Cause

The material document is no longer available for transaction due to the fact that it is already archived.

Resolution

Create PO Returns to the Vendor.

Keywords

M7 063, M7063, archived, document, material document, no longer exist
