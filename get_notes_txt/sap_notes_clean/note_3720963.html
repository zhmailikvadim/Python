Symptom

  • Job SAP_ICF_ERROR_LOGGING (report RS_STARTUP_ICF_LOGGING_DAEMON) fails in SAP S/4HANA 2025;
  • The job log says "Internal session terminated with a runtime error RAISE_EXCEPTION (see ST22)";
  • In the ST22 dump, the Short Text says that exception condition "CONFIGURATION_FAILED" was triggered.

Environment

  • SAP NetWeaver
  • SAP NetWeaver Application Server for SAP S/4HANA
  • ABAP PLATFORM - Application Server ABAP
  • SAP S/4HANA 2025

Cause

The job fails due to an authorization issue. The user starting the job does not have the required authorization to create the startup configuration for a system daemon.

Resolution

Assign the authorization object S_ADF_CFG with activity 1 and 3 to the user starting the job. The id of the daemon is "HTTP_ERROR_LOGGING".
