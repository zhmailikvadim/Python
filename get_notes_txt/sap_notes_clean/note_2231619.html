Symptom

While provisioning multi user request, the Audit log shows 'Invalid date format' error.

Environment

GRC Access Control 10.0/10.1

Reproducing the Issue

1. Go to Access Management -> Multi user request

2. Import the multi user file and submit the request.

3. Approve the request.

Cause

Multiple user file date format field was incorrect.

Resolution

Drop down fields should have the ID/Key value.  As date format is a drop down field in SU01, it should be passed with the key value of the entries under the Date format column.

e.g. for DD/MM/YYYY in the excel file specify the value as 1.
