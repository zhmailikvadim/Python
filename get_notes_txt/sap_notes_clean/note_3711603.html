Symptom

When a production order or planned order is selected through the Production Order Information System, the fashion fields (Season, Season Year, Theme, Collection) are displayed initially. However, after clicking the Refresh button on the results screen, these fields become blank.

Other Terms

FSH_SEASON, FSH_SEASON_YEAR, FSH_THEME, Master production order, Master combined order

Reason and Prerequisites

Reason

This problem is caused by a program error.

Solution

Implement the correction instruction provided with this SAP Note.
