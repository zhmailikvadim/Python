Symptom

In the Confirm Project Tasks app, no warning is displayed when an Actual Start Date or Actual Finish Date is entered in the future.

Other Terms

F0284; ppmprotskcnfs1;

Reason and Prerequisites

Program error

Solution

Implement the attached Correction Instruction or the mentioned Support Package.

Afterwards, reindex the application on your frontend (gateway) system as described in SAP Note 2227577 and clean the caches according to SAP Note 2017117.
