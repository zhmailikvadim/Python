Symptom

This note contains internal metric collection introduced to support SAP Cloud ALM Landscape Management (LMS). These LMS metrics are collected automatically and are not visible to users in the /SDF/ALM_SETUP transaction.

Other Terms

SAP Cloud ALM

ST-PI

/SDF/ALM_SETUP

N/SDF/ALM_SETUP

Set up integration with SAP Cloud ALM

Setup Managed Components (SAP NetWeaver Application Server for ABAP)

LMS (Landscape Management)

Reason and Prerequisites

  • Activation: For every destination successfully registered in /SDF/ALM_SETUP, the Heartbeat Job automatically enables the LMS use case.
  • Collection cadence: A background collection runs every 12 hours and sends required LMS metrics (for example, Transport Domain and the list of clients) to support other SAP Cloud ALM use cases.
  • Deactivation: When a destination is deregistered, LMS data collection is stopped together with any other use-case data collections.
  • Visibility: LMS metric collection use case is internal to SAP Cloud ALM and is not displayed in /SDF/ALM_SETUP.

Solution

Install the latest ST-PI support package. No additional manual steps are required.
