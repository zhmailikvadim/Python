Symptom

  • You are using the Process Maintenance Order web dynpro applications (e.g., Change Maintenance Order).
  • When checking the Person Responsible Field F4 Search Help, "Nothing Found" is returned.



Environment

  • SAP ERP Plant Maintenance (PM)
  • SAP ERP Customer Service (CS)
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Open the “Change Maintenance Order” screen.
  2. Go to the “Operation Data” tab.
  3. Open the F4/value help for “Person Responsible” and observe no values returned.
  4. The same behavior can be observed on the Header screen.

Cause

  • The search help does not return any persons responsible due to the following reasons:
    • The search help only includes persons assigned to a work center.
    • If no work center is provided, no data is returned.
    • If no persons are assigned to the work center, no data is returned.

Resolution

  1. Navigate to Work Center Display/Change:
    a. Use transaction IR02 (Change Work Center) or IR03 (Display Work Center).
    b. Enter the plant and work center ID from your maintenance order operation.
  2. Review HR Assignment:
    a. Go to the "Capacities" tab in the work center.
    b. Select "Capacity" at the bottom of the screen.
    c. Click the "HRMS" (HR Assignment) button at the top.
  3. Verify Personnel Assignments:
    a. Check if personnel are properly assigned to the work center.
    b. Ensure assignments have valid date ranges covering your maintenance order dates.
