Symptom

You run a database search for recipes. The system does not fill the "Recipe" column. To fill the column, you run the report /PLMB/RCP_SEARCH_TERM_GENERATE, but the system does not fill the corresponding field SEARCH_TERM of the table /PLMB/RCP_RECIPE.

Other Terms

PLMWUI, /PLMI/RCP_ELM_DB, /PLMB/RCP_ELM_DMZ_DB

Reason and Prerequisites

There is a program error.

Solution

Implement the correction instructions.
