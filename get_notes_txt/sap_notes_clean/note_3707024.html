Symptom

The automation script may fail when executing the reference Business Scenario BKZ due to specific changes in the process steps.

Environment

SAP S/4HANA Cloud Public Edition 2602 HFC02

Reproducing the Issue

Automation Test execution might fail for above listed business scenarios (BKZ) due to the changes in Process Step "Create Billing Documents".

Cause

The mentioned step is part of 2602 release, due to Label change of a button "Add", this step has failed. And this change had come post the internal cut-off timeline. Resulting in the failure of this process step having HFC02 version of 2602.

Resolution

The standard script with adaption changes will be available in SAP S/4HANA Cloud Public Edition Release 2602 HFC04.

As an alternative or workaround, create a custom copy of BKZ and change the actions 31, 33, 35 from "Add" to "Add Condition". Make necessary changes in Data binding and execute.

Keywords

BKZ, Sales Order Processing with Invoice List and Collective Billing, label
