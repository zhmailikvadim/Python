Symptom

  • The VDAB job board study code field displays an outdated picklist.
  • The latest study levels are not available for selection in the study code field.

Environment

SAP SuccessFactors Recruiting Posting

Reproducing the Issue

  1. Create or edit a job intended for posting to VDAB.
  2. Open the study code field in the job.
  3. Observe that the latest study levels has outdated options.

Cause

the VDAB study code picklist was outdated and required an integration update.

Resolution

Our engineering team has updated the study code field in our system.

  1. Edit each affected job.
  2. In the study code field, select a new value from the updated picklist (latest study codes are now available).
  3. Then repost the job to VDAB.

To avoid this issue in the future a solution will be set to dynamically retrieve latest data on regular basis.
