Symptom

Users cannot consolidate multiple requirement documents into one Freight Booking document when using the “Create Capacity Document Based on Groups” option.

Environment

  • SAP Transportation Management
  • S/HANA embedded TM

Reproducing the Issue

  1. Open Transportation Cockpit
  2. Select multiple Freight Units
  3. Click on 'Create Capacity Document based on Groups'

Cause

Condition-based document type determination is set in the Planning Profile under Capacity Document Creation Settings.

Resolution

This is standard behavior. For more information, see the "Consolidation of multiple Freight Units is not possible if document type determination is set to "Condition-Based" section on the Support Content page.
