Symptom

Open transaction GRAC_EXPORT_RA. Select a 'JOB ID', maintain a target file path and execute the report. The generated report file does not contain Risk level field. The issue is occurring when you want to export the Technical View of risk analysis report in 'Detail' or 'Summary' format.

Other Terms

GRC, Access Control, GRAC_EXPORT_RA, GRAC_EXPORT_REPORT_SPOOL, Risk level, missing

Reason and Prerequisites

The returning fields are marked with incorrect mask. (Risk type instead of Risk level)

Solution

Implement the attached correction instruction.
