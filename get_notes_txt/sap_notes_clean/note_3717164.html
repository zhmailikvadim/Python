Symptom

while applying breaks in web intelligence report appears duplicate data.

Environment

  • SAP BusinessObjects Business Intelligence Platform 4.3 Support Package 05
  • Web Intelligence

Reproducing the Issue

1. Logon to BI launchpad.
2. Open the web intelligence report.
3. Right click on the report.
4. Select the "apply break" option.
5. Duplicate values  appear within the report.

Cause

  • Changing the duplicate values setting from "display first" to "display all" causes all duplicates to be displayed in the report.

Resolution

  • Select the report containing the column where break is applied.

Keywords

web intelligence , breaks , report , duplicate.
