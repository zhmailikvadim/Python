Symptom

You are using SAP S/4HANA Migration Cockpit on SAP S/4HANA 2025 FPS00 and you are running report CNV_PE_MC_CHECK_SETUP. The report raises the following warning:  "DMIS component 2011_1_731(0028) not compatible with S4CORE component 109(0000)". The warning is inocerrect since DMIS 2011 with an SP higher than SP27 is compatible with SAP S/4HANA 2025 FPS00.

Other Terms

S4HANA, Migration Cockpit, Setup Check, DMIS, Compatibility

Reason and Prerequisites

You are running SAP S/4HANA Migration Cockpit on release SAP S/4HANA 2025 FPS00 and the source system is running DMIS 2011 higher than SP27

Solution

This SAP note resolves the incorrect DMIS compatibility check. Implement this note in the S/4HANA system and repeat the execution of report CNV_PE_MC_CHECK_SETUP.
