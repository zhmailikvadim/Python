Symptom

After publishing an RFQ, no option is available to edit the RFQ.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Open the Manage RFQ App.
  2. Create an RFQ.
  3. Publish the RFQ.
  4. Observe that no option is available to edit the published RFQ.

Cause

Standard system behavior.

Resolution

  1. Understand that once an RFQ is published, it cannot be edited.
  2. Refer to the app documentation for alternative actions:

         Manage RFQs | SAP Help Portal

  • Manually change the details of quotations in status "In Preparation" or "Submitted" if agreed with the supplier outside the bidding platform (via phone or email).
  • This feature is available only for Ariba Quote Automation process and Request for Price process.
