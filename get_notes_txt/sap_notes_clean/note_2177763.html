Symptom

You go to SAP NetWeaver Administrator → SOA → Monitoring → Communication Channel Monitor → Communication Channels tab. You select a channel from the list of channels (different than the first one) and open the Schedule subtab. Then you change the selected tab of the main table to Channel Availability, and then select tab Communication Channels again.

The data displayed in the Schedule subtab should correspond to the selected channel from the main table (after switching back to tab Communication Channels, this is the first one in the list), but instead, the UI displays data for the channel that was originally selected.

Other Terms

SAP NetWeaver Administrator, Channel Monitor, Communication Channels, PI, SOA, NWA, Schedule

Reason and Prerequisites

Program error

Solution

Patches for this problem are provided for the Software Components specified in the Support Packages & Patches section of this SAP Note. The archives can be found at SAP Service Marketplace as described in SAP Note 952402.
