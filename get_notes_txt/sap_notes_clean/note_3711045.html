Symptom

Request for an overview of user assignments across multiple companies.

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental." 

Environment

  • SAP Field Service Management (FSM)

Resolution

  1. Login to Admin Portal and navigate to the user detail page



  2. Use the following API to access user assignments:
    https://{{env}}.{{baseUrl}}/api/master/v1/accounts/{{accountId}}/users/{{userId}}/erpUsers/

  3. Refer to the example API usage in screenshot:
    https://eu.fsm.cloud.sap/api/master/v1/accounts/123/users/ABC/erpUsers
