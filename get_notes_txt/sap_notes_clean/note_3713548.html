Symptom

You are using the Migration Workbench or the SAP S/4HANA Migration Cockpit and the short dump “Exception condition 'CONCATENATE_ERROR' triggered.” occurs when the runtime objects are generated for a migration object.

Other Terms

SAP S/4HANA, SAP S/4HANA Migration Cockpit, MWB, DMIS

Reason and Prerequisites

MWB / DMIS functionality is used on one of the following releases:

  • DMIS 2011 SP25 - SP28
  • DMIS 2018 SP11 - SP14
  • DMIS 2020 SP07 - SP10
  • DMIS 2025 SP00 
  • SAP S/4HANA 2023 FPS00 - SPS05
  • SAP S/4HANA 2025 FPS00 - FPS01

Solution

Implement the corrections in the MWB source system.
