Symptom

You use transaction FBL1H, FBL3H, or FBL5H. Your layout contains the field _DATAAGING (Data Filter Value for Data Aging).

If you call FBL1N, FBL3N, or FBL5N for individual lines, a runtime error occurs.

Other Terms

_DATAAGING, FBL1N, FBL3N, FBL5N, SAPDBDDF, SAPDBSDF, SAPDBKDF, RFITEMAP, RFITEMGL, RFITEMAR, RDDFHDB, RKDFHDB, SAPSQL_PARSE_ERROR, CX_SY_DYNAMIC_OSQL_SYNTAX, runtime error, dump, FBL1H, FBL3H, FBL5H,

Reason and Prerequisites

Program error

Solution

Implement the attached correction instructions in your system or import the relevant support package.
