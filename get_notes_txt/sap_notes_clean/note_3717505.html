Symptom

  • Delivery packing cannot be changed due to error HUDIALOG047 when de-assigning Handling Unit (HU)  
  • Error message: "Handling unit &1 is packed - only entire HUs can be reassigned." HUDIALOG 47

Environment

  • Logistics Execution (LE)
  • SAP R/3
  • SAP R/3 Enterprise 
  • SAP ERP Central Component
  • SAP ERP
  • SAP Enhancement package for SAP ERP
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition 

Reproducing the Issue

  1. Access transaction VL32N and enter the delivery number 
  2. Attempt to de-assign one HU
  3. Error message raised 

Cause

If you intend to de-assign one HU in a hierarchy from the delivery, then the entire HU hierarchy would need to be de-assigned. 

Resolution

  1. De-assign all handling units simultaneously from the delivery.
  2. The error is raised as an informational message. 
  3. Press back. Note that all HUs are now de-assigned. 
  4. Save the delivery. 

Keywords
