Symptom

  • The error message “G/L acc. in transport request XXXX (user CB998XXXX) locked – import missing” is displayed when attempting to activate Open Item Management (OIM) for G/L account XXXX in the company code XX.
  • The original transport request XXXX was released to the test and production system without a destination, as it contains semantic objects.
  • A new transport request (XXXX) was created as a copy of the original one and transported to Test and Production, but the issue persists.

Environment

 SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Attempt to activate Open Item Management (OIM) for G/L account XXXX in the company code XX.
  2. Observe the error message: “G/L acc. in transport request XXXX (user CB998XXXX) locked – import missing.”
  3. The Original TR was released without maintaining the Target system so it does not exist in Test and Production system under Import Collection App. 
  4. Created a new transport request as a copy of the original one (XXXX) and transport it to Test and Production.
  5. Verify that the issue persists despite the new transport request.

Cause

  • Issue with Upgrade.

Resolution

Development team has identified the root cause and have completed their correction in regards to issue. This correction is planned to be delivered with Hotfix Collection HFC07. The implementation is planned to take place in the next maintenance window at 29th. of March. 

Solution: HFC07, Planned Release date 29.03.2026.
