Symptom

It is possible to create planned independent requirements with a requirement plan containing lowercase letters. Requirements created in this way cannot be selected using the requirement plan in md61/63, because only uppercase letters are allowed there.

Other Terms

Maintain PIRs, planned independent requirements, F3445, requirement plan, lowercase, uppercase, all caps, reqplan

Reason and Prerequisites

This is a standard from SAP GUI (see note 685886) that was not carried over to Fiori.

Solution

This note ensures that PIRs created or uploaded using the app have their requirement plan converted to uppercase.
