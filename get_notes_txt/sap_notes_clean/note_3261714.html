2) You encounter dump Syntax error in program "SAPLSUG2 " while executing transaction CNV_TDMS_HCM_PRESEL.

Other Terms

HCM Object Selection, TDMS HCM, Transfer, PA, PD, CNV_TDMS_HCM_PRESEL, LT HCM, SAPLSUG2, GC_RELEASE_UNKNOWN

Reason and Prerequisites

1) In case of HANA system the function module UPGI22_GET_SINGLE_COMPREL may not exist.

2) Program Error in SAPLSUG2

Solution

Apply the correction instruction of this note in the sender/execution system and run the activity again. If HCM Object Selection needs to be executed in any other system with similar symptoms, then apply the correction instruction in those systems as well.
