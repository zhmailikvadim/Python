Symptom

  • When refreshing a Web Intelligence report or changing filters in the SAP Mobile BI app for iOS, the Done button to confirm prompt selection is not visible
  • The prompts are mandatory or optional prompts configured as BEx variables in the BEx query

Other Terms

Environment:

SAP Mobile BI app for iOS 6.0.8

Steps to Reproduce:

  1. Create a Webi report from a BICS connection on a BEx query containing optional prompts
  2. Open the report from the SAP Mobile BI app
  3. Refresh the report for the desired parameters
  4. Attempt to filter the report again by selecting new parameters
  5. The Done button is not visible on the prompt selector

Solution

Issue is fixed in SAP Business Objects Mobile 6.1.11.
