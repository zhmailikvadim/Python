Symptom

When you choose "Back" in the navigation tree, a dump occurs after retrocession was executed for the displayed life.

Other Terms

The associated message is "Access via NULL object reference not possible".

Reason and Prerequisites

You access the navigation tree (LRM dialog) by double-clicking (forward navigation) on a successfully processed movement.

Solution

History:

7/1/2025: New

Implement the source code corrections.

Solution:

Following retrocession, the navigation stack is emptied. You can therefore navigate back to the current life successfully.

Technical effects:

None

Manual changes:

You do not have to perform any manual tasks.
