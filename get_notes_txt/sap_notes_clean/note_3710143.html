Symptom

In S/4HANA Group Reporting, the Import Group Journal Entry (IGJE) app terminates with an error indicating Memory allocation failed.

The system cannot complete database operations due to a memory allocation failure on the view P_IMPORTJOURNALENTRIES.

Apply this correction only if SAP Note "3564015 - IGJE: Memory Allocation Fails" does not resolve the issue in your system.

Other Terms

Hint to use PREAGGR_BEFORE_UNION, IGJE, fin.cons.jrnlentr.upload, FIN_CS_ENTRUP, F3073, SAP note 3564015,

RAISE_SHORTDUMP, CX_SADL_DUMP_DATABASE_FAILURE, CL_SADL_SHORTDUMP, TRIGGER_DATABASE_FAILURE_DUMP.

Reason and Prerequisites

Program error.

Since HANA versions may vary between customer environments, some hints may be more effective than others.

Solution

Please follow correction instructions to implement this SAP Note.
