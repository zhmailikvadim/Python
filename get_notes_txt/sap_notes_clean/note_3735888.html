Symptom

A specific file from Informatica Integration is needed.

Environment

  • SAP Sales Performance Management
  • Incentive Management on Legacy Oracle

Resolution

  1. Create a new ticket with SAP Cloud Support with medium or low priority;
  2. In the title, specify the request. For example, "ODI, Need file from Informatica Integration ";
  3. In the description of the case, please specify the following:
    1. Tenant ID and environment (if applicable);
    2. The specific filename(s) that is needed.

See Also
