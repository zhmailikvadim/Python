Symptom

GRAC_PFCG_AUTHORIZATION_SYNC gets cancelled at the main GRC system with error message "Error in RFC; 'An error has occurred while parsing a dynamic entr'"

At the Decentralized system, dump with Runtime error SAPSQL_PARSE_ERROR is shown

Environment

SAP Access Control 12.x

Reproducing the Issue

Execute the GRAC_PFCG_AUTHORIZATION_SYNC job

Cause

Program error

Resolution

Implement following note at the plug-in system: 3119752 - Authorization data Synchronization issue, SAPSQL_PARSE_ERROR in GRC plug-in system

Keywords

PFCG, sync, SAPSQL_PARSE_ERROR, CX_SY_DYNAMIC_OSQL_SEMANTICS, exception, error, parsing, dynamic, /GRCPI/CL_GRIA_AUTH===========CP, an error has occurred while parsing a dynamic entry
