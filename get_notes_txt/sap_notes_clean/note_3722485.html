Symptom

While importing a corrupted BACS file, which has an incorrect counter compared to CONTRA file in the trailer (UTL1) for example, the process will not be stopped and no dummy order created.

Other Terms

/PE1/CL_FH_BACS_TXT_ADAPTER->/PE1/IF_BADI_FH_XML_TXT_ADAPTR~CONVERT_TXT_BLOCK_INTO_XML, /PE1/FH_CONV_XML_IMP-089.

Reason and Prerequisites

Improvement of process.

Solution

After implementing this SAP Note, if a corrupted file will be imported, which contains incorrect counter, a dummy order will be created.

Please implement the provided Correction or the corresponding Support Package.
