Symptom

Material change log is not displayed in t-code CA_SHOW_CD but it shows in change document tables CDPOS and CDHDR.

Environment

  • SAP ERP Logistics (LO)
  • SAP R/3
  • SAP R/3 Enterprise
  • SAP ERP Central Component
  • SAP ERP
  • SAP Enhancement package for SAP ERP
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Make a material change via t-code MM02, for e.g. change description.
  2. The change is recorded in t-code MM04 and the changes can also be seen in tables CDHDR and CDPOS.
  3. Enter t-code CA_SHOW_CD.
  4. Enter MATERIAL in change document object, along with other data such as user ID.
  5. Execute.
  6. No changes are shown.

Cause

This is standard system behavior. Change Document Object "MATERIAL" is not included in the change documents displayed in transaction code CA_SHOW_CD.

Resolution

Transaction code MM04 (Display change documents) to view change documents for a material.

Check out SAP NOTE 11 for raising a feature request to be considered in future product development. The requests which are voted the most are considered for functionality development.
