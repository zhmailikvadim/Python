Symptom

There is an issue with the calculation of tax bonus for children for the year 2026 in report HSKCALCM. This issue occurs only when the employee’s tax base (/431) is greater than 3,144.95 EUR and less than 3,285.95 EUR, and the employee applies for the child bonus (Infotype 0021).
In such cases, results for January may be incorrect.

Other Terms

HRSK, HSKCALCM, Tax bonus for children.

Reason and Prerequisites

This SAP Note delivers corrections described in the chapter Symptom.

Solution

This Note delivers correction described in the chapter symptom.

After application of this Note please perform following steps:
1. Check whether employee's Tax Base (/431) is greater than 3,144.95 EUR and less than 3,285.95 EUR.
2. For those Employee's check if they also apply for the child bonus (Infotype 0021).
3. These employees have to be recalculated into January 2026. (using wage types /99* is not necessary in this case)
