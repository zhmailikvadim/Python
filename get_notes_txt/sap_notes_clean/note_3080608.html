Symptom

alter user and sp_iqpassword is showing the error message only first 172 characters and reporting the following error. 

Msg 13467, Level 16, State 0
SQL Anywhere Error -1044: Invalid password: Valid password chars: A-Z, a-z, 0-9 and @$_#. Space/comma not valid. First char must be lower or upper case letter. At least 1 n

Environment

SAP IQ 16.1

Reproducing the Issue

sp_iqpassword 'password','abcde123',mf_dummy
go

Msg 13467, Level 16, State 0: SQL Anywhere Error -1044: Invalid password: Valid password chars: A-Z, a-z, 0-9 and @$_#. Space/comma not valid. First char must be lower or upper case letter. At least 1 n (1 row affected)

Cause

This issues has been identified and logged under SAP CR 825258. Its description reads, "Long messages may be truncated by the SAP IQ server. Substrings that are used to compose an error or warning message are truncated to 128 characters. The original truncation length was 249 characters and this has been restored."

Resolution

SAP CR 825258 is fixed ;

  • SAP IQ16.1 SP04 PL10 and higher
  • SAP IQ16.1 SP05 PL01 and higher (to be)
