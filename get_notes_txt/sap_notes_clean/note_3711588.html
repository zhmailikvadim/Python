Symptom

  • Inquiry regarding the deletion of shadow tables ADRCS2 and ADR2S2, which containing personal data.
  • There is concern about potential issues that may arise after deleting data from these tables.

Environment

  • Sales and Distribution (SD)
  • Logistics Execution (LE)
  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA

Cause

  • Shadow tables ADRCS2/ADR2S2 are temporary tables used during the copying of customizing data such as plant and company code data.
  • The data in these tables is automatically deleted once the copying process is completed.

Resolution

The data in these tables is automatically deleted once the copying process is completed. However, if you still find any data in these tables, you can manually delete it.
