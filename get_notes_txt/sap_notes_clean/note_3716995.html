Symptom

The system permits the user to modify the external code of a country or region object using any three characters without any validation.

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

SAP SuccessFactors Employee Central

Reproducing the Issue

1. Navigate to Manage Data in the system.
2. Search for "Country/Region."
3. Open any country object.
4. Attempt to edit the External Code field with any three characters.

Resolution

This is an expected system behavior. 

Keywords

SAP, Manage Data, Country/region, external code, LOD-SF-EC-FOO
