Symptom

How Daylight saving time is set in Chile on 2026, 2027 and 2028?

Resolution

There are fixed summer time rules configured in system until 2025.

As there is no fixed rules on 2026, 2027 and so on, the variable summer time rule applies until 2028. This can be checked from transaction STZBC.
We may create a fixed rule for 2029, but it's early to create it now.

The DST works for Chile if the latest time zone settings are imported from note 198411.
