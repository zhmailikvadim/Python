Symptom

After posting Freight Settlement Document (FSD) from Freight Settlement Document Posting transaction (/ACMTM/FSD_POST_MONI) which has confirmation status 'Cost Distribution Document Creation Failed', the confirmation status changes to 'Cost Distribution Document Creation Started' status. The Cost Distribution Document is not created and the status does not change after that.

Other Terms

FSD; Cost Distribution Document Creation Started; /ACMTM/FSD_POST_MONI; Freight Settlement Document Posting;

Reason and Prerequisites

Program error.

Solution

Implement attached correction instructions.

After note implementation, please post the FSD which has confirmation status as 'Cost Distribution Document Creation Failed' from the FSD Posting transaction (/ACMTM/FSD_POST_MONI) and the FSD Confirmation status should no longer get stuck in 'Cost Distribution Document Creation Started'.
