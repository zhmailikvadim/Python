Symptom

  • When refreshing the web intelligence, and there are data security profiles, variables defined on the data foundation of this universe.
  • If you change one prompt, then other permission selected in the prompt will be cleared.

Environment

  • SAP BusinessObjects Business Intelligence Platform 4.3 Support Package 3
  • Web Intelligence (WebI) 

Reproducing the Issue

  1. Log on BILaunchpad.
  2. Open the webi report.
  3. The prompts window appears, select available permissions.
  4. Select other prompt values which are variables defined on the data foundation.
  5. Change the value for first prompt, notice the selected previously prompt values will be reset.

Cause

This is by design.

Resolution

This is by design.

Keywords

BI 4.3; WebI; BI Launchpad; SAP BusinessObjects Business Intelligence platform 4.3; Web Intelligence; prompt; data security profile; permission; reset; clear; variable
