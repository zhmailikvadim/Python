Symptom

You use the Run Compliance Reports app to create the VAT return for Switzerland (report category CH_VAT_DCL). When you upload the XML file with the version 1.0, you receive an error message stating that only version 2.0.0 is valid for submission to the Swiss tax authorities.

Other Terms

CH_VAT_DCL,  Document and Reporting Compliance, DRC, ACR, version 2.0.0

Reason and Prerequisites

This SAP Note is relevant only for Switzerland.

Solution

After implementing this SAP Note, the Run Statutory Reports app generates an XML file with version 2.0.0 for the Swiss VAT return.

Version Date Reason
1 February 20, 2026

Release to all customers

Import the Support Package specified for your release or implement the correction instruction.

As a general rule SAP recommends that you implement the Support Package specified for your release. However, if you need to install a solution earlier, use the SAP Note Assistant to implement the correction instruction. You can find more information about the SAP Note Assistant in SAP Support Portal, under SAP Note Assistant.

The related online help is published on SAP Help Portal on the release day of the relevant support packages.
