Symptom

Foreign key for customizing tables T8JZ and T8J0IA

Other Terms

JVA, Joint Venture, Joint Venture Accounting

Reason and Prerequisites

Table foreign key error

Solution

For table T8J5F:

The foreign key check for the field BUKRS is changed from T8JZ to T001.

For table T8J0IA:

There is no foreign key maintained for the field "BILLING_LAYOUT". The foreign key is now maintained to table "T8J0MA"
