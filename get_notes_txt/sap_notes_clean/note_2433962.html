Symptom

This SAP Note contains the solution needed to implement the changes in rental subsidy that is part of the new Compensation Package proposed by ICSC(A/70/30) and approved by GA resolution 70/244 on the United Nations common system.

As part of this SAP Note, the following object is delivered:

  • Entitlement: UNRSC (Unified Rental Subsidy Calculation) 

Other Terms

  • Rental Subsidy
  • Infotype 0962
  • UNRSC (Unified Rental Subsidy Calculation)
  • CL_HRPAYUN_FUP0962->EXECUTE_NEW

Reason and Prerequisites

Legal Change

Solution

Relevant code changes have been made to implement the update in rental subsidy mentioned in the 'Symptom' action. Perform the following steps to get the solution:

1. Follow the manual steps mentioned in the document 'Manual Steps for Rental Subsidy' attached with the SAP Note.

2. Apply the attached correction instructions.

Alternatively, apply the relevant support package.

Entitlement: UNRSC

This entitlement indicates the date from which the revised rental subsidy allowance is valid. Note that the implementation date can be different from the validity date.

This implementation date is used in method EXECUTE_NEW of class CL_HRPAYUN_FUP0962 to determine whether the revised rental subsidy allowance should be processed or not.

Note : Please maintain entitlement 'UNRSC' in the 'V_T7UNPAD_GA_IMP' view with the effective date of implementation.
