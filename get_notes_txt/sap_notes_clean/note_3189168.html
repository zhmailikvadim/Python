Symptom

You are working with SAP Solution Manager and want to enable System Monitoring to check SAP Web Dispatcher issues.

Environment

  • SAP Solution Manager 7.2

Reproducing the Issue

  1. Go to transaction solman_setup
  2. Expand Application Operations
  3. Click on System Monitoring
  4. Select Step 5 Define Scope-Select Managed System
  5. Press Next to go to Step 6 Setup Monitoring

Resolution

In order to enable System Monitoring correctly, first the product needs to be registered in LMDB with correct Product Assignment and updated parameters information.

  1. Make sure Outside Discovery is enabled using SAP Host Agent method

    2556432 - Switch Outside Discovery from Diagnostics Agent to SAP Host Agent

  2. Register SAP Web Dispatcher in LMDB and enable Diagnostics Relevance flag:
    1. 2363407 - Missing EFWK Templates in SAP Solution Manager 7.2
    2. 2190760 - Product Instance SAP NETWEAVER 7.X : SAP Web Dispatcher has no Diagnostics Template
    3. Maintenance of Product in the System Landscape WIKI
    4. 2270826 - Activity "Introscope Host Adapter" ended with errors for SAP Web Dispatcher

      This KBA will show which Product version needs to be selected to register the product Manually, for example:

      • SAP WEB DISPATCHER 7.40, 7.41, 7.42, 7.49
        -Product Version: NW AS ABAP 7.4
        -Product Instance: SAP Web Dispatcher

      • SAP WEB DISPATCHER 7.45
        -Product Version: SAP NetWeaver 7.5
        -Product Instance: SAP Web Dispatcher

      • SAP WEB DISPATCHER 7.52
        -Product Version: NW AS ABAP 7.52 
        -Product Instance: SAP Web Dispatcher

      • SAP WEB DISPATCHER 7.53, 7.54
        -Product Version: NW AS ABAP 7.52
        -Product Instance: SAP Web Dispatcher

      • SAP WEB DISPATCHER 7.73
        -Product Version: ABAP PLATFORM 1809
        -        Product Instance: SAP Web Dispatcher

      • SAP WEB DISPATCHER 7.77
        -Product Version: SAP S/4HANA 1909
        -        Product Instance: SAP Web Dispatcher
      • SAP WEB DISPATCHER 7.81
        -Product Version: SAP S/4HANA 2020
        -        Product Instance: SAP Web Dispatcher

      • SAP WEB DISPATCHER 7.85
        -Product Version: SAP S/4HANA 2021
        -        Product Instance: SAP Web Dispatcher

      • SAP WEB DISPATCHER 7.89
        -Product Vesrion: SAP S/4HANA 2022
        -Product Instance: SAP Web Dispatcher

      • SAP WEB DISPATCHER 7.93
        -        Product Vesrion: ABAP PLATFORM 2023
        -Product Instance: SAP Web Dispatcher
  3. Once the Product is correctly registered in LMDB, run Managed System Configuration (all steps need to report successful status).
  4. Execute SOLMAN_SETUP / Application operations / System Monitoring / step 5 select MO / step 6
  5. For the Web Dispatcher Managed Object - select template 'SAP Web Dispatcher 7.1 and higher'
  6. Run apply and activate.
  7. In case of Cannot connect to service sapstartsrv errors, please review KBA  2348537 - How to troubleshoot error "cannot connect to service sapstartsrv" in Solution manager 7.2, make sure Diagnostics Agent user has authorizations to call sapcontrol web methods.
