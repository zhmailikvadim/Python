Symptom

  • Option "Auto-Size And Page Table Vertically" is now available on table with New Table Build Experience enabled in optimized story since SAC 2026 QRC 1 update
  • It is now able to use the Pagination in Stories containing table with New Table Build Experience

Environment

  • SAP Analytics Cloud 2026.2

Resolution

  • Since SAC 2026 QRC1 update, story designers are now able to select the option "Auto-Size And Page Table Vertically" on table with New Table Build Experience enabled.
  • There is a Reporting section on your Builder panel if table is on a canvas page. In this section, there is an option called Auto-Size And Page Table Vertically.
  • If this option is selected, table will expand or shrink automatically. If necessary, the application will split the table over multiple pages to fit the content. 
  • For more detailed information, please refer to help document Reporting and Use Pagination in Stories | SAP Help Portal.
