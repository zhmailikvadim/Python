Symptom

The system uses the new value already in the day processing before End-of-Period processing on that date.

Other Terms

Reason and Prerequisites

You run S/4HANA for financial products subledger

- release 3.0 with SP04 or lower, or
- release 2.0 with SP02 - SP07.

There's an error in the software.

Solution

Implement the attached Correction Instructions or import the specified Feature/Support Package.
