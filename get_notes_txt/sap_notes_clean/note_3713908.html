Symptom

This SAP Note adds a new checkbox, Dep. under Age 23 (P0146-CLIDP), to the Life Insurance tab in the Y.E.A. Data JP (0146) infotype.

Other Terms

MP014600, CLIDP

Reason and Prerequisites

This is a part of legal changes. You have implemented the changes in SAP Note 3713824.

Solution

You can apply the relevant HR Support Package or implement the manual changes together with the correction instructions in this SAP Note.

This new checkbox described in the symptom part applies to target years 2026 and later. Select this checkbox to calculate the common life insurance deduction with the special deduction amounts for employees who have dependents under age 23.

If you select this checkbox for target years 2025 and earlier, the system returns an error.  If you select this checkbox and no dependent under age 23 exists in the Family Member/Dependents (0021) infotype, the system displays a warning.
