Symptom

When attempting to print from the Output Management tab of a freight document in TM (for example the Road Waybill), the system terminates with a runtime error. In the short dump you find the text that the current ABAP program "/SCMTMS/CL_LOGISTICS_DATA_PROVCP" had to be terminated because it found a statement that could not be executed. The runtime error occurs when text types are involved in the freight document which is configured with ERP text determination.

Other Terms

Logistics Integration, FO, /SCMTMS/TOR_PRINT_ROAD, ASSIGN_REFERENCE_EXPECTED, PPF,

Reason and Prerequisites

There is a program error when class /SCMTMS/CL_LOGISTICS_DATA_PROV is called during printing.

Solution

Implement the corresponding correction instructions or install the relevant support package instead.
