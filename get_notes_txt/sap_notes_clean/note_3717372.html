Symptom

You need to implement a new tax box reporting structure in response to Romania’s VAT changes effective from January 2026.

Other Terms

Romania, VAT 300, legal change, RO_VAT_300, RO_VAT_300_V5, DRC, Document and Reporting Compliance

Reason and Prerequisites

This update follows a legal change valid from January 2026. This SAP Note is valid only for Romania.

Solution

As a general rule, SAP recommends that you install this solution by applying a Support Package. However, if you need to install it earlier, use Note Assistant to implement the correction instructions.

Post-implementation Instructions:

A. Copy already existing tax box structure (in case you already configured it in SAP Note 3667668):

  1. Copy the existing tax box structure in the view cluster VG_CUST in transaction SM34.

  • Locate the tax box structure with Country/Region: RO, Tax Box Structure Type: 5 and Type Description: Romania VAT 300 Declaration - 08/2025.
  • Make a copy of the structure with validity from 01.01.2026.

  2. Delete the nodes that are no longer used from the new tax box structure.

  • List of obsolete nodes: R12_3_1, R12_3_2, R24_1, R24_2, R25_3_1, R25_3_2, R69_1, R69_2, R70_1, R70_2, R71_1, R71_2, R73_1, R73_2, R74_1, R74_2, R75_1, R75_2, R76_1, R76_2, R77_1, R77_2

      Note: Start from the Assignment of Tax Groups to Tax Boxes view. Continue with Tax Groups and Tax Boxes views. Save after maintaining each view.

  3. Update tax box names in the Tax Boxes view.

B. Create new tax box structure from scratch (in case you did NOT configure it in SAP Note 3667668 as a part of the previous legal change):

  1. Run transaction SM30 for the view V_VG_TBS_CT.
    a. Select Edit
    b. Enter RO (Romania) in the Country field.
    c. Click Apply to proceed.
    d. On the initial screen, add a new entry for Romania.
      I.  Tax Box Structure Type: 5
      II. Name: Romania VAT 300 Declaration - 08/2025

  2. Activate the BC set attached to this note to create a new Tax box reporting structure.

      Note: Ensure the BC Set provided with the SAP Note is correctly imported before activation.
      Note: For the BC Set upload, use package GLO_FIN_IS_VAT_RO.

  3. Maintain the new tax box structure in the view cluster VG_CUST in transaction SM34.

  • On the Tax Groups view, assign the correct tax codes and transaction types to the corresponding tax groups.
