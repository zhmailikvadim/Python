Symptom

In Start Process Process Overview application, the “Affected” column truncates the name and PERNR if the full name exceeds ~28 characters.

  • If the name length ≤ 28, the display Name (PERNR) is correct.
  • If > 28, the PERNR is cut off.

Issue also visible in the form header (“Affected Employee”) when opening the process.

Other Terms

affected , start process, truncated, truncate, cut

Reason and Prerequisites

Reason: There was a programming error

Solution
