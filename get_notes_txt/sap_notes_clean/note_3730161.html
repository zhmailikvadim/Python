Symptom

User expects to have a link for Account field (when created from SSI) as it is when appointment is created in C4C directly. But instead no record appears in Account field at OWL. 

Environment

SAP Cloud for Customer

Reproducing the Issue

  1. Create Appointment using server side add-in. 
  2. Check the Appointment content in the C4C.
  3. The Standard field "Account" is empty in the UI view. 

Resolution

This possibility is not supported as out-of-the-box behavior. Nevertheless it is possible to access Influence Portal and suggest an improvement in this scenario if you believe it could best serve your business purposes: Influence Page
