Symptom

Onboarding process not started for the crossboardee after changing the job information in the employee profile.

Environment

SAP SuccessFactors Onboarding

Reproducing the Issue

  1. Navigate to employee profile
  2. Select a user which should be crossboarded
  3. Edit the job information via filling the required fields, such as position, supervisor, weekly hours 
  4. After saving these the onboarding process should start and the crossboardee should be present in the onboarding dashboard

Cause

A user has been appointed as the supervisor who is ineligible to be one.

Resolution

Ensure the appointed supervisor is an internal employee. Active external users, such as new hires currently in the onboarding process or inactive users are not eligible to be supervisors, hence the crossboardee won't be present in the onboarding dashboard.

Keywords

supervisor, hiring manager, onboarding dashboard
