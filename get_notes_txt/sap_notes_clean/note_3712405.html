Symptom

In the SAP HCM Fiori app "My Leave Requests (Version 3/SAP Fiori 2.0)", some texts have not been translated correctly into Slovakian, Romanian or Bulgarian.

Other Terms

UIHR002, HCMFAB_LEAV_MAN, hcm.fab.myleaverequest, 1.0.207, i18n.properties, i18n_sk.properties, i18n_ro.properties, i18n_bg.properties

Reason and Prerequisites

Prerequisite:

  • You launch the SAP HCM Fiori app in Slovakian (SK), Romanian (RO) or Bulgarian (BG).

 

Reason:

Program error

Solution

The correction contained in this SAP Note adjusts the translations accordingly.

 

Import the relevant Support Package into your system.

For an advance correction, implement the attached correction instructions.
