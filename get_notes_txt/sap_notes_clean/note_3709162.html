Symptom

While trying to save business configuration you are receiving following error appears:

  • In Dynamic Group Filters, the field [Field ID] isn't visible.
    OR
  • In Dynamic Group Filters, the field reference [Field ID] is invalid

Environment

  • SAP SuccessFactors Employee Central
  • SAP SuccessFactors Employee Profile

Cause

The mentioned field is not present in Standard Field configuration. Maybe it was present in configuration but later removed from configuration.

Resolution

Please add or remove the referenced field in standard field data model. Please remove the reference field from the DG filters.
