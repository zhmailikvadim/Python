Symptom

In Payroll Control Center - Manage Team Application, when adding a user to Team members or Team leaders takes very long time or even no response.

Other Terms

PCC, hrpy_pcc_tm_1, performance, can't add user

Reason and Prerequisites

The selectDiablog UI5 controller will re-load all entries after user has been selected when multiple selection is allowed. The re-load will cause performance issue and take very long time when there are many users in system.

Solution

We disabled the multiple selection when adding user to Team members or Team leaders, now only one user can be added for each operation.

 Implement the corrections in your system or import the relevant Support Package.

 Due to the correction is about UI codes, after apply this SAP Note, you need execute report /UI5/APP_INDEX_CALCULATE for HRPY_PCC_TM_1 and clear your web browser local cache to make the UI changes work.
