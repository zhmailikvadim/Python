Symptom

Previously, in the Legacy People Profile - it was possible to mask key pay statement figures such as gross and net salary. After upgrading to the Latest People Profile - you would like to implement this functionality.

Environment

  • Pay Statement Integration with Employee Central Payroll
  • Direct Pay Statement

Reproducing the Issue

  1. Access the Latest People Profile in SAP SuccessFactors Employee Central Payroll.
  2. Navigate to the Pay Statement card within the Payroll Portlet.
  3. Observe that the payroll key figures are displayed without masking.

Resolution

  • The functionality of the Latest People Profile differs from the Legacy People Profile. In the Legacy People Profile, all payroll key figures were displayed masked, and users could choose to show or hide them.
  • In the Latest People Profile, due to the changed navigation pattern where users can actively select a payroll category, the payroll key figures are no longer masked.
    • More information on the differences between Legacy and Latest People Profiles, can be accessed via Notable Differences in Existing Features | SAP Help Portal
