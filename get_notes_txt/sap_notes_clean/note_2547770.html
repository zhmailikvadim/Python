Symptom

Improvement Request

Improve Autorizations: In SAP Solution Manager we can have many SEA analyses and we would like to restrict the SEA analysis access by SID.

Request Reason

Only authorized people should have access to this data.

Other Terms

Customer Connection, CustConn,

Reason and Prerequisites

SAP Solution Manager 7.2 Scope and Effort Analyzer is in use.

Solution

Improvement

A function to control the visibility of scope and effort analyses has been added.

Benefit

Only authorized users are able to see the analyses.

Delivery

SAP Solution Manager 7.2 FPS06. Configuration via SAP Solution Manager Configuration - Scenario: Scope and Effort Analyzer.
