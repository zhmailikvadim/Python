Symptom

 In the process "Receipt of Meter Reading (SUP View)" (DE_B_MET_MR_INFO_SUP), in step 0200-0070-0045, meter reading MSCONS for TAF2/TAF7 (multi-tariff) cannot be posted via the MRP framework.

The Note is relevant to SUP and DSO and MOS

Other Terms

MET, TAF2, TAF7, multi-tariff, MSCONS, MRP framework, /US4G/IF_MRP_ANALYZE~DETERMINE_ENERGY_FEED_INSTLN, /US4G/CL_MRP_ANALYZE, billing-relevant, feed-in register, reading attribute

Reason and Prerequisites

The error occurred because in the MRP framework, a reading attribute indicates whether the value transmitted via MSCONS is for a feed-in register. This check is performed for each reading value and considers whether the register is billing-relevant. If a register is marked as "not billing-relevant," the reading attribute for this value is not set to "IS_FEED" or "NOT_FEED."

Solution

Implement the corrections of this SAP Note or import the relevant support package.
