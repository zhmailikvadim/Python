Symptom

  • In single-day changes in the time sheet, allowance displays correctly in the Time Recording Change Document.
  • In multiple-day changes scenarios, allowance does not display correctly (incorrect type and/or value) in the Time Recording Change Document.

Environment

  • SAP SuccessFactors Time Tracking

Cause

This is a Known Error.

Resolution

This fix for this issue is planned to be deployed on 1H 2026 release.
For release timelines, please review SAP SuccessFactors Product Release & Road Map Information.

Workaround
No workaround is available.
