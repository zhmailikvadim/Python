Symptom

  • Hierarchy variables are not displayed in the InfoProvider view.

  • Browsing for Text and Formula variables is not possible with BW-MT 1.27 PL2

Other Terms

BW-MT, Modeling Tools, Eclipse, Query Designer

Reason and Prerequisites

Program Error

Please note this code correction requires the Query Designer version 1.27 PL1 or higher of the BW Modeling Tools to use the functionality.

Before applying this correction it's strongly recommended to check the consulting note 2133546 for previous query related corrections and the order how they should be applied.

Solution
