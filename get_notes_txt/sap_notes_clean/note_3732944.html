Symptom

When navigating to item detail pages from the tab 'All Objects', the system forces a Refresh. This results in significant browser latency and reduced application performance.

This issue occurs after implementing the changes from the SAP Note 3680118.

Other Terms

F2097A, F2097, UI5, performance, All Objects, detail page, ST12, F12, S/4, PLM, CR

Reason and Prerequisites

Program Error. Performance.

Prerequisite SAP Notes: 3680118, 3712790.

Solution

 Use transaction SNOTE to implement the corrections delivered with this SAP Note in the affected Change Record UI system or import the relevant Support Package.
