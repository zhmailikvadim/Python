Symptom

  • Error messages occur during the archiving job execution for MM_ACCTIT_WRI.
  • Relevant error messages include:
  • BA237: Maximum number of archive files reached for current archiving session.
  • 00564: Job canceled after system exception ERROR_MESSAGE.

Reproducing the Issue

  1. Execute the program MM_ACCTIT_WRI with a selection variant
  2. Monitor the job execution result in transaction SM37.

Cause

  • The archiving job attempts to process an excessively large volume of data in a single run, resulting in the maximum number of archive files being reached.

Resolution

Split the archiving run into smaller packages to reduce the volume of data processed in a single session.
(For example, divide the archiving process by company code and/or date.)

Keywords

MM_ACCTIT, MM_ACCTIT_WRI, BA237, 00564, ba237, mm_acctit, mm_acctit_wri
