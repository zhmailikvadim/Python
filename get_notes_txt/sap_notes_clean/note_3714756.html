Symptom

How to update the table T024L - Laboratory/office for material

Environment

  • SAP ERP Logistics (LO)
  • SAP R/3
  • SAP R/3 Enterprise
  • SAP ERP Central Component
  • SAP ERP
  • SAP Enhancement package for SAP ERP
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA, on-premise
  • SAP S/4HANA Cloud Private Edition

Resolution

The T024L table can be accessed and updated through Transaction OD30 or with the SPRO Customizing path:

  1. Transaction SPRO
  2. IMG
  3. Logistics General
  4. Global Trade Management
  5. Trading Contract
  6. Document Management 
  7. General Data
  8. Settings for Storage Systems
  9. Define Laboratories / Design Offices
