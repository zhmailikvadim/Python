Symptom

You retroactively terminate an employee's contract in the payroll system. When you run the RPLDSNF0 report for this payroll period, the "replace" block 51/type 002 does not appear.

Other Terms

- RPLDSNF0 report

- REMU

Reason and Prerequisites

This issue occurs in the system when:

- Block 51/type 002 has a zero amount (for example, employees in contract suspension).

- Block 51/type 002 is linked to one or more activities (block 53) of type 01 or 02 that are calculated with a non-zero amount.

Solution

After implementing this SAP Note, the system correctly displays the "replace" block 51/type 002 in the RPLDSNF0 report when it is linked to activities with a non-zero amount.
