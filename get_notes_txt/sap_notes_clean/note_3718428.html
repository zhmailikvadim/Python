Symptom

Is it possible use integration center to export onboarding PDFS to STPF?

Environment

SAP SuccessFactors Onboarding

Resolution

  • Integration Center does not support exporting Onboarding PDFs to SFTP by standard functionality.
  • Users are advised to submit an enhancement request for this feature. Refer to KBA: 2090228 - How to submit enhancement ideas for SAP SuccessFactors Products.

See Also

3246373: Outbound Integration with Attachment as the Format Type is not returning data - Integration Center
