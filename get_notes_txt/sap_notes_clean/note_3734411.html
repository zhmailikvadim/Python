Symptom

After change Position value, the error message related an "You have selected a position that can only be assigned to one employee at a time" pop-up appears only after saving and not when changing the Position value.

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

  • SAP SuccessFactors HCM Suite
  • SAP SuccessFactors Employee Central
    • Job Information

Reproducing the Issue

  1. Access Job Data
  2. Click in Edit (Pencil Icon) under Job Information
  3. Choose a Effective Date
  4. Change de Position for an already used value
  5. Save
  6. Error message pop-up

Cause

This behavior is currently expected in Latest People Profile and is planned to be fixed in an upcoming release. 

Resolution

The issue is planned to be fixed in the next release to the test environment on April 13, 2026, and to the production environment on May 16, 2026.

Note: The behavior doesn't occurs in Legacy People Profile.
