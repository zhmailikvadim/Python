Symptom

You declare your import declaration in SAP GTS to customs system DMS.

You specify a guarantee on Transportation tab in your declaration for inward processing. Authorization C505 is created in the XML message with authorization ID corresponding to the guarantee ID. This results in error DK2031 saying, that authorization number does not exist or is invalid.

Other Terms

Denmark, XML, /SAPSLL/LLEG_ECC_DATAF56, dk_dms_im_create_auhorizations, C505, DK2031

Reason and Prerequisites

Program error.

Solution

Implement the correction instructions.
