Symptom

In the process "Receipt of Business Data Request (DSO View)" (DE_B_MDC_RECV_BDR_DSO), step 5000 encounters an error because the division check is hardcoded in the class method /US4G/CL_CHK_MDC_PROCESS-CHECK_BDR_SPA_AVAILABILITY.

This issue is relevant for the market role: MDC.

Other Terms

BDR, Receipt of Business Data Request, /US4G/CL_CHK_MDC_PROCESS, CHECK_BDR_SPA_AVAILABILITY, division category, hardcoded division

Reason and Prerequisites

Program error. The error occurred because the division was hardcoded in the class method /US4G/CL_CHK_MDC_PROCESS-CHECK_BDR_SPA_AVAILABILITY.

Solution

Implement the corrections of this SAP Note or import the relevant support package.
After implementing this SAP Note, the system identifies the correct division category and processes as expected. You need to retrigger the process.
You can verify the fix by checking that the system identifies the correct division category and doesn't encounter an error.
