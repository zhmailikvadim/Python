Symptom

  • Error encountered during the upgrade of MaxDB for the content server from version 7.9.08.40 to version 7.9.11.10.
  • The following error happens:
    Switching state to ONLINE
    Cannot switch database mode, error during dbm command:
    db_online
    ERR
    -24622, ERR_AUTOLOADSYSTAB: Automatic loading of the System Tables failed
    24909, ERR_DBAUNKNOWN: SYSDBA unknown
    Looking for error messages in knldiag
    No error messages found in knldiag

Environment

  • Target MaxDB is between 7.9.11.00 and 7.9.11.10

Reproducing the Issue

  1. Attempt to upgrade MaxDB of the content server from version 7.9.08.40 to version 7.9.11.10.
  2. Observe the error during the upgrade process.

Resolution

Follow the instructions provided in 3585685: Upgrade to SAP MaxDB Version 7.9.11 changes the network port of the database.

See Also

  • Refer to: 3585685: Upgrade to SAP MaxDB Version 7.9.11 changes the network port of the database
