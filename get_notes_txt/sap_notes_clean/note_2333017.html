Symptom

In transaction /SAPAPO/SCC_TQ1 (/SAPAPO/QTA1) when toggling between 'Display' and 'Change' mode the transactions locks the whole model.

Other Terms

/SAPAPO/LOO_TR_QUOTAI06, authority_enqueue, authority_enqueue_multi

Reason and Prerequisites

There is a different locking approach used for toggling then for the transaction startup, which is corrected with this note.

Solution

Please implement this note to get the correction.
