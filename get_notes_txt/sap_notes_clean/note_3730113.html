Symptom

A how-to guide on how to export and download specific configurations for Inbound/Outbound files via Express Data Loader. This is especially useful for importing custom file type setups from one environment to another.

Environment

  • SAP Sales Performance Management
  • Incentive Management on Hyperscalers
  • Incentive Management on Legacy HANA 

Resolution

How to export and download File Type Setups regarding Inbound/Outbound files:

  1. Navigate to File Types Setup
  2. Select if you wish to see the File Types for Inbound or Outbound
  3. Select the checkbox of the File Type you wish to export
  4. Press the "..." button next to the "Create New" button
  5. Click "Export Selected"

A download will begin with the exported file type setup configurations which you can import into another environment.
