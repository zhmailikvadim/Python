Symptom

Symptom

You processed an Organizational Change and reset the clearing of open items. The corresponding transfer documents still contain the clearing information.

Other Terms

Organizational Change, FINOC, Reset Clearing

Reason and Prerequisites

Program Error

Solution

Implement the Support Package or Correction Instruction

"Requested content hash not found"

"Configuration is not maintained"

Reason and Prerequisites

The behavior of the Exposure report /UI2/CDM3_EXP_SCOPE has been corrected and the displayed messages have been adjusted.

Solution

Implement the correction instructions and perform the manual activity or implement the according Support Package.

After implementing the code corrections, you need to expose the CDM content again to correct the content by using transaction /UI2/CDM3_EXP_SCOPE.
