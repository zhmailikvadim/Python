Symptom

End of Purpose Check of Consent Management can cause problems for other programs.

Other Terms

BUTSORT, bupa_pre_eop, cl_cm_bupa_eop_check

Reason and Prerequisites

Dates declared as '' can lead to problems in other programs. Since an empty string is not a valid date. To solve this issue these dates are now declared as initial.

Solution

Apply the Correction Instructions to fix this problem.

If you have already entries in BUTSORT table with initial values you can delete them by do the steps descripted in the manual correction instruction.
