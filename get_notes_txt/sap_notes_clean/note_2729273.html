Symptom

You are using the Fiori app ‘Migrate Your Data’ to migrate data to SAP S/4 HANA.

After this migration scenario has been hidden by SAP, any existing projects that use this scenario are also hidden by mistake.

Currently, if there is only one available migration scenario in the system, then the migration scenario is not displayed in the header when creating, viewing or editing a project. After you implement this SAP Note, if only one migration scenario is available in the system, the scenario is displayed in the header whenever you create, view or edit a project.

Other Terms

Migrate Your Data, Migrate Data Directly from SAP System, S/4HANA 1809, Migration, Posting Engine, F3473

Reason and Prerequisites

You are using SAP S/4HANA 1809 FPS01.

Solution

Implement the corrections outlined in this SAP Note in the SAP S/4HANA back-end system in accordance with the correction instructions. In addition, to resolve this issue, you must implement SAP Note 2731854 in the S/4HANA front-end system.
