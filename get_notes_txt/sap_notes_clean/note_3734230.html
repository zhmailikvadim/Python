Symptom

In the DRC Cloud Edition "Resolve Errors" app, the following message is shown for the affected eDocument ID:
"Error occurred during the PUSH step, detailed information: Connection to Business System <destinationName> returned HTTP status code: 503 / 504"

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

  • SAP ERP
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition
  • SAP Document and Reporting Compliance, Cloud Edition

Reproducing the Issue

  1. Launch transaction edoc_cockpit
  2. Submit an eDocument
  3. Check the logs in DRC Cloud Edition - Resolve Error application

Cause

This error may occur due to blocked or misconfigured network connectivity (firewall/proxy) between DRC Cloud Edition and the backend system.

Resolution

Check the tunnel blocker in the firewall configurations in the BTP Connectivity Proxy, as follows:

  • Access the Cloud Connector and choose your BTP subaccount. Select the Monitor (Cloud to On-Premise) menu.
    • Access the Most Recent Requests tab and check if there are logs on this view.
    • If there are no logs, you need to update your firewall configuration according the your region, se follows:
      • In your SAP BTP account, access the Overview menu and get the region of your subaccount (e.g. EU10 / EU11)
        • For EU10:
          • URL: cp.a1c4955.kyma.ondemand.com
          • PORT: 443
        • For EU11:
          • URL: cp.c-39821b9.kyma.ondemand.com
          • PORT: 443
        • Alternatively, you can use the egress IP. In that case, the IP address remains unchanged for the cloud edition.
