Symptom

When you implement SAP Security Note 3565506 "[CVE-2026-0511] Multiple vulnerabilities in SAP Fiori App (Intercompany Balance Reconciliation)", you face the following issues during the implementation:

  • WAPP RECONREPRU_S1 FRAGMENT/BPEMAILSDIALOG.FRAGMENT.XML does not exist; create it

  • WAPP RECONREPRU_S1 UI5D76DD861549513A2BC852CAFDBD483AE14D6C717 does not exist; create it

  • WAPP RECONREPRU_S1 UTIL/SENDTOBP.JS does not exist; create it

Other Terms

Reconciliation report, Russia, RECONCILIATION_REPORT, F1819

Reason and Prerequisites

Missing objects

Solution

Implement automatic correction instructions of this SAP Note or implement the mentioned Support Package.

2 29.01.2026 Correction instructions have been added for UIAPFI70 400, 600,700, 800.
