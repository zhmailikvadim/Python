Symptom

You want to use the same username and password to connect to multiple On-Premise systems from SAP BTP with SAP Cloud Connector

Environment

SAP BTP, Cloud to On-Premise

Resolution

Follow the checklist below:
1. Ensure each on-premise system is exposed and reachable.
2. Use the same Basic Authentication username and password in each destination.
3. Ensure each on-premise system has the same username with the same password, and that the user has sufficient roles and permissions.
4. Test the destination connection
     4.1 Going to SAP BAS,
     4.2 creating a project
     4.3 Select the data source to connect to a system.
