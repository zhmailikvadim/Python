Symptom

Understanding Proxy Management

If employee "A" proxies as Employee "B", then "A" is now in the perspective of "B" and will follow "B's" permission for all the features.

However, in Proxy Management, it is possible to limit so that Employee "A" can access only a subset of what Employee "B" has permission to access.

Example 1:

  • B is permission-ed to see Total Goal Management, some Admin Tools and Performance Forms in RBP.
  • In Proxy Management, A is configured to access only B's Total Goal Management and Admin Tools.
  • When A proxies as B, A can search end user related pages like 'add a goal", "Manage Templates"  in action search, but CANNOT access Performance Forms related features

 

Example 2:

  • B is permission-ed to see Total Goal Management, some Admin Tools in RBP.
  • In Proxy Management, A is configured to access only B's Total Goal Management, Admin Tools,  and Performance Forms 
  • When A proxies as B, A can search pages like 'add a goal", "manage templates" in action search, but CANNOT access Performance Forms related features

 

Example 3:

  • B is permission-ed to see Total Goal Management, some Admin Tools and Performance Forms in RBP.
  • In Proxy Management, A is configured to access B's Total Goal Management, Admin Tools,  and Performance Forms 
  • When A proxies as B, A can search pages like 'add a goal", "manage templates" in action search and also access Performance Forms related features

On this scenario, there were few gaps that have been addressed in the 2111 Release.

Environment

SAP SuccessFactors Performance and Goals

Resolution

The following Proxy Management gaps have been fixed in the 2111 Release:

  • Performance Management data can be accessed in Report by the user who doesn't have PM module permission
  • Performance Management data can be accessed on EP by the user who doesn't have PM module permission
  • With only a certain form template permission granted, the user can read other form information on home page
  • Performance Management data can be accessed in SAC by the user who doesn't have PM module permission
  • Performance Management data can be accessed in Calibration by the user who doesn't have PM module permission
  • Performance Management data can be accessed via OData by the user who doesn't have Performance Management permission
  • Performance Management data can be accessed via DWR call by the user who doesn't have Performance Management permission
  • Goal name and Goal rating are visible when Goal Management proxy is NOT granted on a PMU form
