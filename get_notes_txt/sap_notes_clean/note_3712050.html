Symptom

  • When running the Survey Response (CSV) report with a date filter, the report returns blank results.
  • Running the report without filters returns several records as expected.

Environment

SAP SuccessFactors Learning

Reproducing the Issue

  1. Access Learning Administration >
  2. Navigate to Reports and select the Survey Response (CSV) report >
  3. Run the report without filters and confirm that it returns several records as a result >
  4. Run the same report but add a date filter >
  5. Check the report results and confirm that they are blank.

Cause

  • The completion status used for the affected item has the survey flags disabled.
  • If the survey flags are not enabled for the completion status and the survey is initiated from the class, the record learning history process will not trigger the survey process. As a result, the item completion details will not be populated in the PA_STUD_SURVEY database table.
  • Since this is the table used for the Survey Response report and the completion details are not available, filtering the report by date will not return results for the records where the completion date is blank.

Resolution

To ensure the completion details are reflected on the survey report, we recommend enabling the following settings on the completion status under References > Manage Users Learning > Completion Status:

  • Include history records with this completion status for Course Feedback Surveys Assignments
  • Include history records with this completion status for Follow-up Survey: Surveys Assignments
