Symptom

The “spilindex” has been surpassed. As a result, a number of reduction wage ceilings are to be updated as of July 1st, 2024.

This impacts the structural reductions and Target group reductions.

Other Terms

RPCDMFB0, 3000, T5BVJ, MMRS1, MMRS9, MQRS7, MQRS8, GGMMI

Reason and Prerequisites

As intermediate instruction the Social Security (RSZ/ONSS/LSS) has published changes for Q3 / 2024.

  • Dutch:

https://www.socialsecurity.be/employer/instructions/dmfa/nl/latest/intermediates#aanpassing-van-loonplafonds-verminderingen-21-12-2023

  • French:

https://www.socialsecurity.be/employer/instructions/dmfa/fr/latest/intermediates#adaptation-des-plafonds-salariaux-reductions-21-12-2023

Solution

As a general rule SAP recommends that you implement the Support Package specified for your release.

However, if you need to install a solution earlier, use the SAP Note Assistant to implement the correction instruction.

You can find more information about the SAP Note Assistant in SAP Support Portal, under SAP Note Assistant.

Note

For the target group reduction for older employees in the Brussels region exceptions apply for the interim sector. These exceptions are not supported, as to our best knowledge none of our customers pays employees in this sector.

Within the intermediate instructions an update as of 01.10.2024 is mentioned for payroll parameter MQRS7. This future delimitation is included in the manual correction instructions.
