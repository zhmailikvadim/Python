Symptom

The web channel campaign configured to be displayed for contacts part of a segment, but for some contacts it isn't displayed.

Environment

SAP Emarsys Customer Engagement

Reproducing the Issue

  1. Create a web channel campaign (like overlay) with recipient source as "Segment".
  2. Save the campaign and activate.
  3. Access the web site and log-in with one of the contacts part of the segment.
  4. The pop-up is not displayed for this contact, event though the contact is identified by the web extend command, but it is displayed for other contacts part of the segment.

Cause

The possible cause is that the affect contact is sharing the same value for the predict contact identification (Email or External ID) with a different contact in the system. Therefore the web channel can not identify the contact exactly.

Resolution

Use the contact segmentation to find if the contact share the same value for the predict contact identification (Email or External ID) with a different contact in the system.

If multiple contacts share the same value, remove or merge those contacts to keep only one contact with this value for predict contact identification. 

Keywords

web channel, not displayed, segment, identified
