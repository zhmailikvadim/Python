Symptom

This SAP Note resolves the issue of Missing Plant/Sloc View data when distributing Material via Initial Data Distribution and Delta Distribution.

  • During Initial Data Distribution, an issue was identified in the remote system where the plant and storage location view data were not synchronized with the material. To address this, the system now retrieves all Plant/SLOCs used in the Subscription ID, as well as the plants defined in the Plant filter within the MAIF configuration.
  • During Delta Distribution, an issue was identified in the remote system where the plant and storage location view data were not synchronized with the material. To address this, the system now retrieves all Plant/SLOCs available in the target system, as well as the plants defined in the Plant filter within the MAIF configuration.

Other Terms

S4 Defense Disconnected, Material, Plant, Storage Location, Sloc

Solution

Implement the attached correction instructions provided with the NOTE.
