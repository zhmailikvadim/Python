Symptom

SAP user experience updates are now available in Production systems.
(Previously, they were only available in Preview systems.)

In 2H 2025, we provided early access to a set of upcoming SAP user experience updates, so that you could start testing and preparing to adopt them.
In 1H 2026, those updates are now available in all systems, both Preview and Production. They include both visual and functional changes in the following places:
  • Shell bar
  • Navigation menu
  • Search for people or actions
  • Notifications
  • User menu
  • User settings

Environment

SAP SuccessFactors HCM Suite

Resolution

SAP user experience updates provide an HCM experience that's more consistent with other SAP applications, based on SAP standards. You can now adopt them anytime, on your own timeline, before the become the default experience in all systems.

To enable the SAP user experience updates:

  1. Go to Admin Center; or on Search bar
  2. Company System and Logo Settings
  3. Enable > Enable SAP user experience updates

For test purposes, you can turn the updates on and off as often as you need to, with no lasting impact on the system.
