Symptom

Up to now, the system only checked the settings of the logical ports in the Test Report for Communication with the GKV and DSRV (RPUSVHD1).

This SAP Note also checks the settings of the logical ports when sending files using the B2A Manager and when retrieving files using the report SI: Retrieval of Data from SI Agencies  (RP_PAYDE_B2A_SV_COLLECTOR).

This is to prevent, for example, that data from the production system is incorrectly sent to the test environment of the DSRV or incorrect requests are sent to the communication server.

If errors occur in the settings of the logical ports, the transfer or request is prevented. In this case, check the settings with the Test Report for Communication with the GKV and DSRV (RPUSVHD1) and correct the incorrect setting.

In addition, the system now tries to issue a more specific error message if a technical error occurs when files are sent to the DSRV.

Other Terms

eXTra 1.5, RP_PAYDE_B2A_SV_COLLECTOR, B2A Manager

Solution

Implement the attached correction instructions or import the specified support package. After you import the HR Support Package or implement the correction instructions, you do not have to perform any further follow-on activities.
