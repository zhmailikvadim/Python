Symptom

When using IDCNDOC to print accounting vouchers, the vouchers are not printed with the expected sequence.

Environment

  • Financial Accounting (FI)
  • SAP R/3
  • SAP R/3 Enterprise 
  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA Finance
  • SAP S/4HANA

Resolution

To print the vouchers not with the default sequence, try filling in the 'Sorting Field for Batch Input' to adjust the sequence.

Keywords

IDCNDOC, FI document, Accounting Voucher, Print Sequence
