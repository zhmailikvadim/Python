Symptom

The longer the product compliance event processing (PCEP) background job SAP_EHFND_PCEP runs, the more the processing time for each product compliance task and the memory consumption of the background job increases. Depending on the number of processed tasks, this can potentially lead to "out of memory" dumps.

Reason and Prerequisites

Reason:

Memory leaks within the processing of product compliance events/tasks.

Prerequisites:

For information about the validity of the corrections, see the correction instructions.

Solution

This issue is mitigated in the patches listed in the "Support Packages & Patches" section below by periodically restarting the PCEP background job SAP_EHFND_PCEP.

Alternatively, you can implement the attached correction instructions.
