Symptom

Attempting to resubmit an invoice with the status "FAILED TO SEND" in KSEF 2.0 results in the error message: "The action is not allowed for any of the selected eDocuments. Message no. EDOCUMENT094."

Environment

  • SAP ERP
  • SAP S/4HANA
  • SAP Document and Reporting Compliance Service

Reproducing the Issue

  1. Run Transaction code EDOC_COCKPIT
  2. Attempt to resubmit the invoice with the status "FAILED" in KSEF 2.0.
  3. Observe the error message: "The action is not allowed for any of the selected eDocuments. Message no. EDOCUMENT094."

Resolution

  1. Implement SAP Note 3702882, which was released to correct the error encountered when attempting to submit a document.
  2. After implementing the note, test the scenario by attempting to do a Retry action button in edoc_cockpit for the failed invoice.
  3. Ensure all prerequisites mentioned in the overview note 3279206 Electronic Document Processing for Poland KSeF: Implementation Overview SAP for Me are implemented before proceeding.

Note: If the note 3702882 is already implemented, skip this solution and submit an incident to the component CA-GTF-CSC-EDO-PL.
