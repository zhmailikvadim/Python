Symptom

  • An error is encountered when SAP Note 3542282 is attempted to be implemented in transaction SNOTE. The error message “NOT_VALID does not exist” is displayed.

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental. 

Environment

  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  • Open transaction SNOTE.
  • Attempt to implement SAP Note 3542282. 

Cause

  • The syntax error is caused because the exception “NOT_VALID”, which is defined in the method IF_EX_TB037_CHECK~TB037_CHECK, is not present in the system.
  • It is indicated that a prerequisite Note is missing.

Resolution

SAP Note 3518900 is to be implement, before implementing SAP Note 3542282.

Missing object will be added in the prerequisite Note.

Keywords

3518900, 3542282, NOT_VALID, IF_EX_TB037_CHECK~TB037_CHECK
