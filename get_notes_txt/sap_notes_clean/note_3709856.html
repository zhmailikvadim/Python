Symptom

This KBA describes the steps of configuring e-stamp and e-signature to be displayed on ITF1A and 50BIS report.

Environment

  • Human Capital Management (HCM)
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP SuccessFactors Employee Central Payroll
  • SAP S/4HANA

Resolution

  1. Upload e-stamp and e-signature file in Transaction Code SE78.
  2. Maintain feature 'THSTM' return value for the corresponding employee groups. The return value is the e-stamp and e-signature names in above step.
    Note: If feature 'THSTM' is not used, ITF1A and 50BIS report program will look for default e-stamp and e-signature file name in the format of 'STAMP + client number' and 'SIGNATURE + client number'. For instance, if the current logon client number is 100, the report will search for images named 'STAMP100' and 'SIGNATURE100'.
