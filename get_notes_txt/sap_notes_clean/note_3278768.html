Symptom

How to use Fiddler to collect HTTP and HTTPS traces.

  • SAP support requests a Fiddler trace.
  • Fiddler is a third-party troubleshooting tool that can trace web traffic activity (HTTP requests and responses). It can be downloaded for free from http://www.telerik.com/fiddler
  • It can be used to get information on:
    • The client web requests and web server responses
    • To capture the traffic being generated to the BI Platform.
    • The amount of time dedicated to backend processing vs. client-side processing.
    • The content of any errors returned from the web server.
    • To troubleshoot issues with SAP HANA HTTP connections.

Environment

  • Lumira Designer 2.x
  • S/4HANA

Resolution

  1. Download Fiddler from http://www.telerik.com/fiddler and install it on a machine which is on the same network as the computer where Lumira Designer is installed (usually installed directly on the same PC Lumira Designer).
  2. Close all the applications on the machine to avoid capturing irrelevant information.

  3. Make sure Lumira Designer is closed, then open SapLumiraDesigner.ini and add the 4 +2 bold lines at the same position:

     jre/bin/SapLumiraDesignerVM.exe
    -aadProxyHost
    localhost
    -aadProxyPort
    8888
    -vmargs
    -XX:+HeapDumpOnOutOfMemoryError
    ...
    -Dsun.net.client.defaultConnectTimeout=88000
    -Dhttp.proxyHost=localhost
    -Dhttp.proxyPort=8888
    -Xmx4g

  4. Launch Fiddler and make sure that network capturing is active ("Capturing" must be displayed in the lower left corner of the Fiddler window).
  5. For HTTPS traffic, the following must be done, otherwise fiddler will not see any of the content:
    1. Select Tools->Fiddler Options
    2. Click the HTTPS tab
    3. Ensure 'Capture HTTPS CONNECTs' and 'Decrypt HTTPS traffic' are checked
    4. Click OK
  6. Open Lumira Designer and the application where the issue occurs, then reproduce the scenario.
  7. After the issue has been reproduced, go back to Fiddler, click the file menu and de-select "Capture Traffic".
  8. Choose "File > Save > All Sessions..." from the Fiddler menu.
  9. Zip the Fiddler trace file (.SAZ file) and forward it to SAP.
    Note: The file needs to be zipped to be able to attach it to the case.
