Symptom

The below runtime error occurs with exception CX_SY_CONVERSION_CODEPAGE when applying filters to fields of type raw (binary).

Category               ABAP programming error
Runtime Errors         CONVT_CODEPAGE
Except.                CX_SY_CONVERSION_CODEPAGE
ABAP Program           CL_ABAP_CONV_OUT_CE===========CP
Application Component  BC-ABA-LA
Date and Time          dd.mm.yyyy hh:mm:ss

Environment

  • Dynpro
  • CUA engine

Cause

The code for RAL logging of select options did not included proper handling for raw fields. Due to this the code page converter will be called with binary data that might not be a valid Unicode string which triggers the exception. With this note code for properly handling the raw datatype in select options is delivered.

Resolution

Implement the SAP Note 3686657 - CX_SY_CONVERSION_CODEPAGE occurs when RAL logging is activated for RAW fields
