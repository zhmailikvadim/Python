Symptom

The header text of the query changed while stretch the width of the column manually. The text will read the text setting in corresponding data element instead. The issue only happens for some fields and some language. 

Environment

ABAP system

Reproducing the Issue

  1. Execute the query.
  2. stretch the column.
  3. The header text will change to short text/long text as set in the data element.

Cause

Improper text translation, the translation of the header is longer than 40 characters.

Resolution

Adjusting the text translation to a shorter text in transaction code SQ07 (less than 40 characters).

Keywords

Query; header text; long text; short text
