Symptom

SPLIT_FILENAME identify the file extension incorrectly.

Environment

  • SAP Netweaver
  • SAP S/4 Hana

Reproducing the Issue

Call the function module SPLIT_FILENAME, then pick a file.

Cause

File name has several dots.

Resolution

SPLIT_FILENAME works simply by tracking the "." and captures the next three characters as the file extension. For that to work properly, please make sure there are no other dots contained in the file name, besides the one that actually separates the extension.

For example:

my.file.txt = will fail, extension would be "fil"

Rename to my_file.txt for example, then it would correctly pick "txt"
