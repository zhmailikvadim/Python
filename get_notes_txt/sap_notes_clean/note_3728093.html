Symptom

When reviewing Journal Entries for production order postings, inconsistent behavior is observed regarding the WBS Element field:

  1. For some production orders, the WBS Element is populated as a statistical account assignment in the journal entry.

  2. For other production orders, the WBS Element field remains blank in the journal entry.

  3. In both cases, a WBS Element is visible in the Assignment tab of the Display Production Order app.

Environment

SAP S/4HANA Cloud Public Edition

Cause

The difference in behavior is determined by the origin of the production order and the stock valuation method:

Scenario 1: Engineer-to-Order (ETO) with Valuated Project Stock

The production order is part of a managed ETO process flow:

Sales Order - WBS Element - Planned Order - Production Order

In this scenario, the production is specifically for Valuated Project Stock. Because the order is intrinsically linked to the project structure from its inception, the WBS Element is correctly passed to the Journal Entry as a statistical account assignment during postings.

Scenario 2: Make-to-Stock (MTS) with Manual Assignment

The production order is a standard order intended for normal (anonymous) plant stock. The WBS Element seen in the Assignment tab was manually entered after the production order was created. In this case, the order is not technically considered "Project-related" for stock valuation purposes. Therefore, the assigned WBS Element is not carried over as a statistical account assignment in the financial postings.

Resolution

To confirm if a production order is intended for Valuated Project Stock, you must verify the technical attributes of the order item.

 - Access the Query Browser or Custom CDS Views app.

 - Open the CDS View: I_ProductionOrderItem.

 - Filter by the Production Order number.

 - Check the value in the field InventorySpecialStockType (Special Stock Type).

If the value is "Q", the order is for Valuated Project Stock (ETO Scenario).

If the field is blank, it is a standard Make-to-Stock order.
