Symptom

When trying to create new APIM service instances, it fails with the error below:

Service broker error: Service broker apimanagement-apiportal failed with: Service instance provision failed with error : Could not find credentials service 

Environment

  • SAP API Management

Resolution

Try to create the service instance again after some time (the next day).

If the creation fails again or if urgent solution is needed, kindly open a case in the component OPU-API-OD-DT.

In the case description please provide:

  1. Subaccount id;
  2. Screenshots of the issue.

Keywords

Could not find credentials service, Service broker error, Service broker, apimanagement-apiportal, Service instance provision failed with error, APIM, subaccount, instance, API Management
