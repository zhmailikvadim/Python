Symptom

Is X11 officially supported by SAP Data Services?

Environment

  • X11 (X Window) 
  • SAP Data Services 4.X

Cause

SAP Data Services and its predecessor versions were never exclusively certified for X11 authentication.

Resolution

Since SAP Data Services does not officially support X11, it is not recommended to use it, as there is no guarantee that it will work functionally.
