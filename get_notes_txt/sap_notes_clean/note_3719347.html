Symptom

When creating a condition table and saving it, the error message "Too many fields selected (larger than 510 bytes)" occurs and the condition table cannot be saved.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Access the configuration activity: "Create Condition Tables for Pricing in Sales".
  2. Select some fields and save the condition table.
  3. The error message occurs and the condition table cannot be saved.

Cause

The total length of the key fields included in the table exceeds the system limitation-510.

Resolution

It can be considered to adjust the length of the containing custom fields or adjust the condition table to make sure the total length of all key fields in the condition table not exceed 255.

Keywords

Condition Table Creation, Condition Table, Field Length Limitation, Key Field, Custom Field, Create Condition Tables for Pricing in Sales.
