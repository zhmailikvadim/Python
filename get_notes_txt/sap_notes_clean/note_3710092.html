Symptom

Following the deployment of the new versions of Chrome 142 and Edge 143, an issue is observed during the punchout process in SAP SRM Catalog.

The following error occurs when launching the application:
“The connection is blocked because it was initiated by a public page to connect to devices or servers on your private network.”

Other Terms

LNA, PNA, Chrome/Edge, SRM Catalog, Private Network Access, Local Network Access

Reason and Prerequisites

Apply Prerequisite Note 3686552 - SAP NWBC ABAP Runtime Patch 93

Solution

Please implement the note or import relevant support package.
