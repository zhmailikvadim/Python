Symptom

You want to change a PM /CS Order. You start the change from the PM / CS Notification transaction (IW22/ IW52), and navigate to change Order transaction (IW32).  After changing the Order data there is no confirmation popup before navigating to the notification transaction.

Other Terms

IW52, IW32, Service Notification, service order, Save, External call, Order Notification integration

Reason and Prerequisites

Reason:

Program error

Solution

Implement the attached correction. After implementing this SAP Note the order changes can be saved only from notification if the user navigates from Notification and performs order change.
