Symptom

When processing a payment advice note in INVMON, the document status was set to Incorrect due to an error in determining the correct item in the aggregated posting document.

This SAP Note is relevant for the market role: DSO

Other Terms

Grid Usage Billing, Payment Advice Note, Aggregation Posting Document, Open Item, Document Status, Error Message >0 372, Class /US4G/CL_ADVICE_UTIL_GU, Method /US4G/IF_ADVICE_UTIL_GU~GET_OPEN_ITEM_FOR_BS

Reason and Prerequisites

Program error.

Solution

Please implement the corrections of this SAP Note or import the relevant Support Package (SP).

After this fix, the payment advice note can be processed as usual.

To verify the fix, process the payment advice note again and check that the document status is set to On Hold instead of Incorrect.
