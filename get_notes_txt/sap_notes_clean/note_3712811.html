Symptom

When you request metadata from an APE API service, such as '/APE/API_PROCDOC'(Process Document) or '/APE/API_TRANSDOC'(Transfer Document), you might encounter status code 500 with error message "Invalid EDM name 'Process Step'".

Other Terms

/APE/API_PROCDOC, /APE/API_TRANSDOC, Process Document, Transfer Document.

Reason and Prerequisites

 Program Error.

Solution

 Implement the SAP Note or install further Support Package.
