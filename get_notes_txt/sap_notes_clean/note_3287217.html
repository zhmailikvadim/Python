Symptom

You're using PLM System Integration functionality for mapping DM Product Version to SAP Material Master. You've maintained in the configuration 'Define Conditions for Outbound', value for the field 'Value From' and set the flag 'Value From Any' to true for a given attribute. Alternatively, the same settings is also maintained for the field 'Value To' and 'Value To Any'. This is not correct, as only one option should be allowed.

Other Terms

PLM System Integration, PLMSI, PLM System Integration for SAP ERP, PLM System Integration for S/4HANA OP, Define Conditions for Outbound, Map DM Product Version to SAP Material Master.

Reason and Prerequisites

Programming Error.

Solution

Implement the Note with the attached correction instructions and listed manual activities.
