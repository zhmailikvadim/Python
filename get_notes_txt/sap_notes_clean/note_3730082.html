Symptom

Per-position pipeline is picking up old position names along with the new ones while running

Environment

  • SAP Sales Performance Management
  • Incentive Management on Hyperscalers

Reproducing the Issue

  1. Go to Position screen.
  2. Create a position in December 2025 period.
  3. Version it to January 2026 period with a new name.
  4. Go to Pipeline Screen.
  5. Run a Per-position Pipeline.
  6. See the name of Old version and new version appear in the Pipeline details.

Cause

A UI defect causes the pipeline details to display the incorrect position version name in some situations.

Resolution

Engineering team confirmed the issue and the fix is expected to be resolved in the 2605 release scheduled for May 16, 2026.
