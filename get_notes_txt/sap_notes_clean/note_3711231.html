Symptom

  • When accessing the managed system in System Monitoring, the application attempts to load the file '/sap/public/bc/ui5_ui5/1/resources/sap/galilei/ui5/Editor.js', resulting in a 404 - Not Found error.
  • Error observed in developer tools: "Uncaught Error: failed to load 'sap/galilei/ui5/Editor.js' from /sap/public/bc/ui5_ui5/1/resources/sap/galilei/ui5/Editor.js: 404 - Not Found".
  • The issue persists even when accessing the Solution Manager directly, bypassing the web dispatcher.
  • The path '/sap/public/bc/ui5_ui5' and all components below this tree section are active in SICF.

Environment

Solution Manager 7.2

Reproducing the Issue

  1. Open the System Monitoring application in work center.
  2. Click on a managed system within the monitoring interface.
  3. You will see blank screen after selecting managed system
  4. Open developer tools and look for path /ui5_ui5/1/resources/sap/galilei/ui5/Editor.js

Cause

Program Error

Resolution

  1. Refer to SAP Note 3708314 for troubleshooting steps and resolution.
  2. Ensure that the path '/sap/public/bc/ui5_ui5' and its components are correctly activated in SICF.
  3. Deactivate and reactivate the "Application Operations/System Monitoring" gateway services.
  4. If the issue persists, open remote connections to the Solution Manager system for further investigation.

See Also

  • Refer to SAP Note 3708314 for detailed resolution steps: SAP Note 3708314.
  • Refer to SAP Knowledge Base Article KB0123130 for additional troubleshooting guidance: KB0123130 - Troubleshooting steps for blank screen in Monitoring UI's - SAP Solution Manager 7.2.
