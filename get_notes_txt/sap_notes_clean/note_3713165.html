Symptom

After you upgrade the system to SAP S/4HANA OP2023, the Itemization for Posting Totals (transaction FPT5) and the Itemization for G/L Documents from FI-CA (transaction FPT5) or FPG5 show the totals but do not display the assigned document lines for archived documents.

Other Terms

FP-GL, FP-SUM, CO-PA, DFKKSUM, RFKKGL20, PAOBJNR, FPT5, DFKKSUM

Reason and Prerequisites

Reason:With Release S/4HANA OP2023, the data type of the profitability segment is changed from numeric to character-like (field PAOBJNR, change from NUMC10 to CHAR10). As a result, the initial value changes from '0000000000' to SPACE.

Prerequisites: You archived the Reconciliation Key with PAOBJNR value '0000000000'.

Solution

After implementing this SAP Note, the system correctly displays the assigned document lines for archived documents in transactions FPT5 and FPG5.

Implement the corrections in accordance with the attached correction instructions or import the relevant Support Package.
