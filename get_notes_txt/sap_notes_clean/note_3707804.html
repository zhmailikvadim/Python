Symptom

******************************************************************************************************************************************************************************************************************

WARNING: This is an SAP Tax Declaration Framework Note. If you do not have SAP TDF installed in your system, this SAP Note does not apply to you. SAP TDF requires a specific license.
******************************************************************************************************************************************************************************************************************

This SAP Note provides you with corrections to the following reports:

  • EFD ICMS IPI (/TMF/EFD) report:
    • Register C170 and C190 do not generate the ICMS Aliquot (ALIQ_ICMS) fields correctly when the Fiscal Situation Code (CST_ICMS) field equals to 061 for ICMS single-phased transactions.
    • The system does not generate Register D761. 
  • ECF (/TMF/ECF) report:
    • Register 0000 generates an error when you enter a value greater than 100 in the Scission Remaining Patrimony (PAT_REMAIN_CIS) field.

Other Terms

TDF, Brazil, SPED, EFD ICMS IPI, ECF, 0000, C170, C190, D760, D761.

Reason and Prerequisites

This SAP Note is relevant only for Brazil.

You have implemented SAP Note 3698679 and 3702926.

You have upgraded your system to SAP TDF Support Package 20.

Solution

After implementing this SAP Note, the updates listed below are made to your system:

  • /TMF/EFD report:
    • Register C170: The CA_ALIQ_ICMS calculated field on the J_C170 node is updated to generate the ALIQ_ICMS field with a value equal of 0 when the CST_ICMS field is equal to 061 in the EFD_C170 view. As a result, the ALIQ_ICMS is generated correctly.
    • Register C190: The CA_ALIQ_ICMS calculated field on the P_REMOVE_NULL_VALUES node is updated to generate the ALIQ_ICMS field with a value equal of 0 when the CST_ICMS field is equal to 061 in the EFD_C190_013 view. As a result, the ALIQ_ICMS is generated correctly.
    • Register D760: The Document Date (DT_DOC) field is enabled in the EFD_D760 view, allowing the system to insert data into the /TMF/D_SERV_NF_T buffer table. As a result, the buffer table is correctly populated and the D761 register is generated as expected.
  • /TMF/ECF report:
    • Register 0000: The CA_PAT_REMAIN_CIS calculated field on the J_EMPRESA node is updated. The system enforces an upper limit of 100; all input values exceeding this threshold are automatically set to 100 in the EFD_0000 view. As a result, the register no longer generates an error when the PAT_REMAIN_CIS field is greater than 100.

As a general rule, SAP recommends that you implement the Support Package specified for your release. However, if you need to install a solution earlier, use the SAP Note Assistant to implement the correction instruction. You can find more information about the SAP Note Assistant in SAP Support Portal, under SAP Note Assistant.
