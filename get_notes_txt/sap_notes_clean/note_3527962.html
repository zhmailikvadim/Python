Symptom

  • When attempting to upload a PLQU file for a monthly quota in SAP SuccessFactors Incentive Management, the file does not accept the defined monthly period in the calendar (e.g., 1/31/2024)
  • Specifically, if the start date is set as the first day of the month (e.g., 1/1/2024) and the end date as the last day of the same month (e.g., 1/31/2024), the PLQU file errors out.
  • However, if an overlapping date is used (e.g., start date as 1/1/2024 and end date as 2/1/2024), the PLQU file works without errors.

Environment

SuccessFactors Incentive Management

Reproducing the Issue

  1. Prepare a PLQU file for a monthly quota.
  2. Set the start date as the first day of the month and the end date as the last day of the same month.
  3. Attempt to upload the PLQU file in SAP SuccessFactors Incentive Management.

Cause

The PLQU file requires the effective start date and effective end date to be the first day of each month  (e.g., start date as 1/1/2024 and end date as 2/1/2024 for Jan 2024).

Resolution

  1. Set the effective start date as the first day of the mentioned month.
  2. Set the effective end date as the first day of the next month.
  3. Upload the modified PLQU file in SAP SuccessFactors Incentive Management.
