Symptom

A dashboard widget shows values with a "CAD" prefix (for example, CAD 1234) for the CAD unit type, while another unit type shows only the numeric value (for example, 1234).

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

  • SAP Sales Performance Management
    • Incentive Management

Cause

The unit type symbol configuration is missing for the unit type in Incentive Management.

Resolution

  1. Access Incentive Management Admin and navigate to Global values > Unit Types.
  2. Check the Symbol setting for the relevant unit type. 



  3. Update the configuration:
    • Define the desired symbol for the unit type to display a prefix. 
    • Leave the symbol blank for the unit type to hide the prefix. 
  4. Save the changes.
