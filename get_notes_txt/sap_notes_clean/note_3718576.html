Symptom

VAT Return report RFUMSV00 for Slovenian KIR/KPR declaration fails with following dump on versions lower or equal to than SAP_FIN 720:

CALL_FUNCTION_PARM_UNKNOWN

Function parameter "ID_BUDAT" is unknown.

Called function is FI_SPLIT_TAX.

Dump occurs in method IF_EX_FI_TAX_BADI_016~UPDATE_TAX_POSTING of class /CCEE/CL_IM_SIFI_TAX_016 when Tax Payable Posting is required and Batch Input Session is required.

Other Terms

Slovenia, Book of Incoming Invoices, Book of Outgoing Invoices, KIR, KPR, DDV-O, RFUMSV00, /CCEE/SIFI_DMEE_KIR_KPR, CALL_FUNCTION_PARM_UNKNOWN, ID_BUDAT, FI_SPLIT_TAX

Reason and Prerequisites

Error fixing 

Note 3684270 and has been implemented.

Solution

Apply corresponding support package or implement attached correction instructions.
