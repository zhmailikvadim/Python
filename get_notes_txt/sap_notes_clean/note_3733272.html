Symptom

  • Deleting the failed service key returns “deletion failed”.
  • Error details include “Service broker failed to delete service binding for instance: Service broker error: Service broker it-broker-rt failed with: Error occurred during deletion of binding <id> of instance <id>"

Environment

  • SAP Integration Suite
  • Cloud Integration

Resolution

  • The issue is currently under investigation, and the root cause has been identified. A platform patch addressing this behavior is scheduled to be applied over the coming weekend (28th - 29th March 2026)
  • Once the patch deployment is completed, we kindly request you to retry deleting the service key.
  • If the issue persists after the patch update, please create a ticket under the component LOD-HCI-PI-CON-SOAP.
