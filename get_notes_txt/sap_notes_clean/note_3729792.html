Symptom

 When starting a job, some fields starting with "custom" are not present in the job field list.

Environment

SAP SuccessFactors HXM Suite

  • Integration Center

Reproducing the Issue

  1. In Manage Business Configuration, set the Enabled setting to No for fields starting with "custom"

  2. Open Integration Center and create a new integration using the EmpJob entity（example）.
  3. Observe the disabled field is not available in field list.

Cause

Fields starting with "custom" and having the Enabled setting set to No are not displayed in the OData API Data Dictionary.

As a result, they do not appear in the Integration Center, which is the intended system behavior. 

Resolution

Change the Enable from NO to Yes in Manage Business Configuration

The the fileds will show in OData API Data Dictionary, you can now choose the filed in integration center.
