Symptom

You observe abnormal behavior when you use filters in the Communication Channels tab tables in PI Commincation Channel Monitoring: Availability Time Planning. The abnormal behavior is triggered when a schedule is being saved.

Other Terms

ATP, Availability, Time, Planning, Channel, Monitoring, Filter

Reason and Prerequisites

 Program error.

Solution

Please, update the software components to the patch levels maintained in section "Support Packages & Patches" of this SAP Note or use higher versions.

For more information on how to find the archives on SAP Support Portal, see SAP Note 952402.
