Symptom

Billing documents have been archived in ECC system. Is it possible to open them in S/4HANA system, such as via billing transaction code VF07?

Environment

  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Cause

Resolution

As the data models are different in ECC and S/4, this is not supported in standard.

See Also

Note 2198647 - S/4 HANA: Data Model Changes in SD

Keywords

BA111, archived billing documents, archive file does not exist , SD_VBRK, archiving billing, ERP, S/4 HANA, S4 system
