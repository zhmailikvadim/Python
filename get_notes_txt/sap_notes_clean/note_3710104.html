Symptom

The note provides consulting solution that enables the Retail Sale Price value maintained in specific condition type in Pricing Procedure to be made available to the Tax Procedure based on condition subtotal '5' within ME21N and MIRO transactions.

This is a pre-requisite for supporting Retail Sale price based GST tax calculations for Tobacco.

Other Terms

GST, IN, subtotal 5, GRWRT, M/08, OBQ3, Tobacco GST, India, ME21N, MIRO,J_1I4_COPY_TAX_ITEM_INDEX, J_1I4_COPY_PO_DATA, IGST, SGST, CGST, UGST,

Reason and Prerequisites

Consulting solution.

Solution

Kindly note this is a consulting solution.

The code in the attached correction instruction uses condition subtotal '5' option based on field KZWI5. Please choose appropriate subtotal option from 1 to 6 that suits your requirement and is currently not being used for other business contexts.

Please refer attached document with steps on required manual code changes to support alternate KZWI other than KZWI5 provided in the correction instruction.
