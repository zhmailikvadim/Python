Symptom

This SAP Note provides the ERP_UOM, ERP_UOM_ISO, PO_ORG, and PO_GRP parameters for the Match Document process to ensure that these fields are validated. If any of these fields are not populated, the Match Document process returns the Not Validated status.

Reason and Prerequisites

This SAP Note is relevant only for Brazil.

Solution

 Use transaction SNOTE to implement the attached correction instructions.
