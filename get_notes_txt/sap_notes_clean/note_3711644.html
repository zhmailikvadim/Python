Symptom

While configuring the preparation of consolidation group changes in T-CD: CXEK, an error message is displayed indicating that posting level 22 is not available.

Environment

  • Group Reporting
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

Go to T-CD: CXEK, try to create a new Document type.

Cause

Incorrect transaction code used, posting level 22 can only be selected in t-cd: CXEL.

Resolution

Go to transaction code CXEL, or Customizing node: SAP S/4HANA for Group Reporting->Preparation for Changes in the Consolidation Group->Define Document Types for Cons. Group Change in Consolidation Monitor
