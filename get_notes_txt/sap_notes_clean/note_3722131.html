Symptom

When running the "Manage Draft Benefit Enrollments", the result comes with the status "Not Deleted" and description ""This record can’t be deleted because an enrollment window is currently active for this benefit".

Environment

SAP SuccessFactors Employee Central

Cause

This is currently the expected behavior. The job will not delete the benefit enrollment drafts if the enrollment window is open for the current period, or if there is an enrollment exception window also open.

Resolution

As a workaround for this behavior, the customer can change the enrollment dates on the relevant period, to make sure the window is closed. Run the job again and, after the drafts are deleted, change the dates back to the required ones.
