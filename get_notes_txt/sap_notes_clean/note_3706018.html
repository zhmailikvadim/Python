Symptom

In SAP Business Clients dialog for changing the properties of a system connection there is an input field for specifying the URL of the connection entry. When the validation of this URL field fails, sometimes erroneously the text, which should be only read by the screen reader, is displayed in the menu popup as well.

Other Terms

SAP Business Client; SAP BC; NWBC; Error Text; Error Message; URL Field; Input Field;

Solution

Please install the patch mentioned in section "Support Packages & Patches" or a newer one (SAP Business Client patches are always cumulative).

If the patch, which contains the solution from this SAP Note, is not delivered yet you can check the following SAP Notes for the expected delivery dates:
