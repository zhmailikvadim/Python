Symptom

You are processing imported preliminary journal entries with the register process for a source system with the business domain 'External Subledger'. In order to complete the processing and clear all process controller status, Day End-BT processes need to be processed too.

Other Terms

Transaction for Register Process: /BA1/BR_REGISTER
Register Substep '06 Register Imported Preliminary Subledger Journal Entries'
Business Domain 'External Subledger': 50

Reason and Prerequisites

There is a program error.

Solution

Apply the correction instruction or the respective support package.
