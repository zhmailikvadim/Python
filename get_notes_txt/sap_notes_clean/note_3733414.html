Symptom

You are doing a performance analysis and want to check when a problematic SAP HANA SQL query is compiled and executed?

Environment

SAP HANA 2.0

Resolution

You can run this diagnostic SELECT statement:

SELECT "STATEMENT_STRING", "IS_VALID", "LAST_INVALIDATION_REASON", "STATEMENT_HASH", "USER_NAME", "PLAN_ID", "EXECUTION_COUNT", "PREPARATION_COUNT", "LAST_EXECUTION_TIMESTAMP", "LAST_PREPARATION_TIMESTAMP"
FROM "M_SQL_PLAN_CACHE" WHERE "STATEMENT_STRING" LIKE '%<part of your statement>%'
AND "HOST" = '<host_name>'
AND "USER_NAME" = '<database user name>';
