Symptom

When carry out mass change for batch of maintenance orders, error in the log states: 'Selected Orders cannot be processed together, remove Order ****'. 

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

  • SAP S/4HANA Cloud Private Version
  • SAP S/4HANA 2021
  • SAP S/4HANA 2022
  • SAP S/4HANA 2023

Reproducing the Issue

  1. RunTransaction Code IW38 and select some orders.
  2. Select the Mass Change option to do mass change for orders from different order types.
  3. Receive error 'Selected Orders cannot be processed together, remove Order ****'. Message No. BL001

Cause

The issue arises due to the system's design where it is not possible to have orders with different procurement modes processed in the same session. This conflicting situation triggers the error message.

Resolution

  1. Avoid mass editing the orders with different procurement modes.
  2. Specify the order type with the same procurement modes. 

    Verify Procurement Mode is used for this order type, per the following customizing path:

    SPRO

    Plant Maintenance and Customer Service

    -> Maintenance and Service Processing

         -> Maintenance and Service Orders

             -> Functions and Settings for Order Types

                 -> Procurement

                      -> Activate Enhanced Procurement Mode

  3. It's advisable to the 'Save Variant' feature in IW38 to process the order type in same procurement modes in same session.
