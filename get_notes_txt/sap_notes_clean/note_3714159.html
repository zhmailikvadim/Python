Symptom

How to add a new tab in Dashboard to display documents related to only Pcards? User would like to see documents in To-Do specific to only PCards (Charges, Charge Reconciliation, PCard Order).

Cause

Resolution

The below steps needs to be performed to add a new Tab in the Dashboard.

  1. On the dashboard, click the dashboard configuration icon and choose "Tab Set Options" and "Add New Tab".
  2. Enter a title for the new dashboard tab in the "Tab Name" field.
  3. In the "Document Types" field, select "Restrict the type of documents displayed on this tab" to choose which types of documents are displayed on the dashboard tab.
  4. If "Restrict the type of documents displayed on this tab" is selected, specify the document types displayed on the dashboard:
  5. Choose "Display" to make a document type visible on the dashboard tab.
  6. Select PCard Charges, PCard Charge Reconciliation, PCard Order from the list.
  7. Click "OK" to save the settings.
