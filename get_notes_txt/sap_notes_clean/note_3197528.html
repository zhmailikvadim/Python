Environment

SAP Analysis for Microsoft Office

Reproducing the Issue

  1. Try to start Analysis Office;
  2. The error is shown and AO is not started;

Cause

The Excel version is not supported or Visual Basic for Applications (VBA) is disabled. VBA is a prerequisite for running AO.

Resolution

  • Make sure that the Excel version installed in the machine is a supported version according to Microsoft's page Update history for Microsoft 365 Apps (listed by date). The Excel version can be found via Excel -> File -> Account as shown below:

  • There are several ways that VBA might be disabled. See this page from Microsoft Learn. VBA enablement can be checked in the Developer ribbon of Excel. The Visual Basic button has to be enabled:
