Symptom

Despite having both Global administrator and subaccount administrator access, the user is unable to view the Work Zone service instance that has been created.

Environment

SAP Build Work Zone, Standard edition

Cause

The user must have the 'Org Manager' role in order to view the Work Zone service instance.

Resolution

The 'Org Manager' role is mandatory for viewing Work Zone service instances, regardless of other administrative privileges. Follow the steps outlined in the SAP KBA '3249765' to add an 'Org Manager' to a Cloud Foundry org.

Keywords

Work zone, service, instance, instance not visible, service binding key, instance, access missing
