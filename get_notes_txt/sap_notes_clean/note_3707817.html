Environment

SAP HANA Platform Edition 2.0

Cause

The errors indicate an inconsistency with the backup file on persistence level

Resolution

Take a fresh backup of the Tenant DB or full Instance and verify if it is recoverable from a persistence layer check prior to recovery using these resources :

  • 1873247 - Checking recoverability with hdbbackupdiag --check
  • Checking Whether a Recovery is Possible
