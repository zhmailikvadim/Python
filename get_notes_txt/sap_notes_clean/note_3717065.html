Symptom

You can use the report SBAL_STATISTICS_NEW to create statistics that you can use, for example, to determine which application has created the most application logs in the system and why these logs have not yet been deleted. SAP Note 3555222 describes the various options of this report in more detail.

However, it is currently not possible to use the report to determine which application has created the most entries in the tables BALDAT and BAL_INDX.

Solution

Implement the correction instructions contained in this SAP Note or import the relevant support package.

After you implement this SAP Note, the report provides the new selection parameter "Get Number of Table Entries". If you set this parameter, the report also determines the number of entries in tables BALDAT and BAL_INDX.

Caution: Setting this parameter increases the runtime of the report.
