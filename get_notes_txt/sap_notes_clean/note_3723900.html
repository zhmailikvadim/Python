Symptom

  • Users encounter a permission error when attempting to add a Pay Component Non Recurring
  • Error message displayed: "You do not have the necessary permissions to perform these changes. Please contact your system administrator".

Environment

  • SAP SuccessFactors HCM Suite
  • SAP SuccessFactors Employee Central
    • Compensation information
    • Pay Component Non Recurring

Reproducing the Issue

  1. Navigate to Compensation.
  2. Select Pay Component Non Recurring 
  3. Attempt to add a pay component for the intended user
  4. Error message displayed: "You do not have the necessary permissions to perform these changes. Please contact your system administrator."

Cause

The user currently lacks a required permission to perform the intended action.

Resolution

  1. Grant the "Pay Component Non Recurring Edit Action (Edit)" permission.
  2. In the Legacy People Profile, the permission "Pay Component Non Recurring (Edit)" alone is sufficient to execute the action.
  3. However, in the Latest People Profile, the user additionally requires "Pay Component Non Recurring Action (Edit)" to execute the same action
