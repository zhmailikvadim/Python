Symptom

BOP isn’t allocating from the storage location defined in the supply assignment rule.

Environment

  • S/4HANA Retail & Fashion Management (LO-RFM) 
  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Cause

The supply source in the selection rule is storage location stock, but the material is batch-managed. Supply Source is S (Storage Location Stock). 

Resolution

If the material is batch-managed, the Supply Source should be set to C (Batches), and the Attribute should restrict by Storage Location.

Keywords

BOP, allocation, storage location, selection rule, supply source, material, assignment, batch management
