Symptom

Cycle containing basic sets cannot be uploaded via KSEUD.

The system does not recognize setclass 0000.

Because the setclass cannot be determined correctly, the selection on set tables does not work. This result in error GA2 / 501 (Group & does not exist in controlling area &) or GA2 / 109 (No objects were transferred).

Other Terms

KSEUD, Allocations, GA2 501, GA2 109, ACYCLEUDF02, GJG1, KSU1, KSV1

Reason and Prerequisites

Cycle segment contains set created in GS01.

Program error.

Solution

Implement the following correction instructions or apply relevant support package.
