Symptom

In G-Invoicing backend, when edocument automation is activated, a function module '/GINB/FM_DO_EDOC_ACTION_BG' is triggered by bgRFC for the edoc actions, and a message will be added in the application log with EDOC guid 'Action <action> is triggered via bgRFC'.

This message will be added in the edocument only when using Configurable Constant 'GC_EDOC_BGRFC_MSG_ACTIVE' is set to 'X' and is inactive by default.

Other Terms

G-Invoicing Edocument automation, ERP configurable constant

Reason and Prerequisites

To enable the addition of message 'Action <action> is triggered via bgRFC' only when configuration constant 'GC_EDOC_BGRFC_MSG_ACTIVE' is set to 'X'

Solution

Implement the attached SAP Note.
