- Query tab is missing for users assigned to the reconciler or auditor role even though it is enabled in the recon configuration (UI elements)

- Selected filter value is not shown in the header filter field after having selected it via value help

- Filtering not working for field names that contain a '/' slash

- Value help symbol is shown for filter fields without any value help maintained

- Manual matching not possible even when total amount of select items is zero
