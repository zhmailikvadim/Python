Symptom

User has planned MCP in recurrence for future but want to disable the same due to business requirement

Environment

GRC Process Control

Reproducing the Issue

  • Scheduled plan for MCP in recurrence

Resolution

GRPC_UPDATE_PLAN can be used

Execute the program GRPC_UPDATE_PLAN in SE38 and provide the timeframe and year.

Select and click on edit, then you can remove any of the selected objects.
