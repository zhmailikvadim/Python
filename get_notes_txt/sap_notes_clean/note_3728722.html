Symptom

  • Removing object from combined query in Web Intelligence document, results in a persistent pop‑up appears in the top‑left corner.

Other Terms

pop-up, webi, combined query, query panel

Reason and Prerequisites

Environment:

  • SAP BusinessObjects Business Intelligence Platform 4.3 Support Package 04 Patch 200
  • Web Intelligence

Steps to Reproduce:

  1. Open any Web Intelligence document containing Combined Query.
  2. Switch to Design Mode and open Query Panel.
  3. Now remove an object from one of the Combined Query (or from both).
  4. Click on Cancel operation. 
    >> Noticed that pop-up appears in top left side of screen.
