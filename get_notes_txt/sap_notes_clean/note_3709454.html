Symptom

You cannot find the Manage Journal Posting Policies (App ID: F8539) and Manage Journal Upload Cases (App ID: F8538) applications in SAP S/4HANA 2022.

Other Terms

Intelligent Journal Upload, IJU, Applications for SAP S/4HANA 2022, Manage Journal Upload Cases, F8538, Manage Journal Posting Policies, F8539

Reason and Prerequisites

These applications are not available in the system for SAP S/4HANA 2022.

Solution

With this SAP Note, you are informed that you can access the Manage Journal Upload Cases and Manage Journal Posting Policies applications in SAP S/4HANA 2022 SP07 or later versions.
