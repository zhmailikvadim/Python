Symptom

In the log file of a document with 'Failed to Push' status in Monitor Document section of Peppol Exchange app, the error message 'Service cannot be reached ' is being thrown. 

Environment

  • SAP ERP
  • SAP S/4HANA Private Cloud Edition
  • SAP Document Reporting and Compliance service

Cause

The URL path of the BTP Destination is missing the resource from Cloud Connector.

Resolution

Review the details of Help page Adding Resource to Connection | SAP Help Portal to retrieve the resource configured in Cloud Connector and add it in the URL Path section of the Destination, as mentioned in Help page Setting Up Connectivity | SAP Help Portal.
