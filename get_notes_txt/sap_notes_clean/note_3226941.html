Symptom

GJCBA: Error G4-427 incorrectly displayed

Other Terms

JVA, Joint Venture Accounting, G4427, G4 427, Profit center assigned to cost obj. does not match given PC

Reason and Prerequisites

When cutback is run, error G4-427 can be shown although the profit centers match. This is due to a code error.

Solution

Implement the attached code corrections with transaction SNOTE.
