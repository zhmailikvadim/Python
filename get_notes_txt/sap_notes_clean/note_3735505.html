Symptom

After you migrate the default integration patterns (INTEG_PAS4), the system creates an irrelevant entry in the Integration Pattern Assignment customizing for Classification (CLS). You need to delete this entry.

Other Terms

PLMSI, PLM system integration, migration, CLS, Classification, SAP_DMClassAsClassification

Solution

Implement this SAP Note to delete the irrelevant default integration pattern entry for classification from the customizing.
