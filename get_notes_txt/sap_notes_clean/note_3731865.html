Symptom

You run the Transfer to Accounting (Post documents) via transaction PCP0 or via report RPCIPP00.

The output log for the statistics is not translated.

Other Terms

RPCIPP00, Translation, text

Reason and Prerequisites

This issue is a side effect caused by the solution delivered by SAP Note 3679186.

Solution

Please apply the corresponding HR Support Package or the corresponding code changes.

When applying the code changes of the note, please also apply the manual activity described in under 'Manual Activity'.
