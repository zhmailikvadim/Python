Symptom

In the SAP Ariba Buying and Invoicing that is integrated with SAP ERP or SAP S/4HANA using SAP Integration Suite, managed gateway for spend management and SAP Business Network, when a Service PO with multiple account assignments is changed, the generated change IDoc still contains entries for the deleted account assignments under the limits data. The same incorrect data is sent to Ariba. 

Cause

This issue occurred due to an erroneous code.

Resolution

Now, the code has been enhanced to resolve the problem. The additional check has been added to exclude the deleted entries of Account assignment data. For more information, please see SAP NOTE 3676104.
