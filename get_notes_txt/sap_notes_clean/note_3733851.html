Symptom

  • When printing multiple vouchers in batch using IDCNDOC, the system assigns local voucher numbers (ODN) in a specific order, what is the order logic?

Environment


  • China
  • IDCNDOC

Reproducing the Issue


  1. Run IDCNDOC for a set of documents.
  2. Check the ODN assigned to the vouchers and review the sequence.

Cause


Consulting issue related to product design.

Resolution


  1. Apply the standard logic: the system sorts headers during processing using SORT gt_header BY user.
  2. The ODN order is sorted by document number and fiscal year.

Keywords


IDCNDOC, batch printing, local voucher number, ODN, official document number, sorting logic, order, belnr, gjahr, document number, fiscal year, localization, numbering sequence, voucher print, assignment order
