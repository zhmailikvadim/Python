Symptom

When you generate the S-2200 Cadastramento Inicial do Vínculo e Admissão/Ingresso de Trabalhador (initial registration of employment and employee hiring) or the S-2299 Desligamento (termination) events, the fields cnpjEmpregAnt/cnpjSucessora are always filled.

Other Terms

eSocial; Transfer;

Reason and Prerequisites

You have implemented SAP Note 2736009.

Solution

The fields cnpjEmpregAnt/cnpjSucessor will now only be filled for a transfer.

The changes described in this SAP Note will be included in an HR Support Package, as indicated in section Support Packages.

The Support Package includes the following objects:

  • Changed classes:
    • CL_HRPAYBR_EFDB_SUCCESSION

As a general rule, SAP recommends that you implement the Support Package specified for your release. However, if you need to install a solution earlier, use the SAP Note Assistant to implement the correction instruction. You can find more information about the SAP Note Assistant in SAP Support Portal, under SAP Note Assistant.
