Symptom

You are using transaction FPVA and do not have setup the exceptions. When you go to the menu  environment, the exceptions option is available even though it fails when you select it.

Other Terms

RFKKMAEX, TFK000C, xmaex, e083, Dunning exception log is not active (table TFK000C), goto menu.

Reason and Prerequisites

Program error.

Solution

This correction the Exceptions option will gray out in menu if the customizing is not setup.
Implement the attached program corrections or import the relevant Support Package.
