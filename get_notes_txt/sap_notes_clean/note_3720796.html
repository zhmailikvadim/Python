Symptom

You run the Cálculo de nómina española (RPCALCE0) program for an artist employee with the Tipo de CCC (T5E08-TPCCC) field equal to '007' who has a difference between the Base de cotización de contingencias comunes (/342) and Base de cotización de contingencias de AT y EP (/343) wage types. However, the system incorrectly calculates the Contribución del trabajador a desempleo (/370) and Contribución de la empresa a desempleo (/371) wage types.

Example:   

Proceed with the following steps to reproduce the symptom mentioned in this SAP Note.

Description

Payroll run for an artist employee with a difference between the /342 and /343 wage types

Preparation/Background

  1. Hire an artist employee.
  2. Configure the employee master data to present a difference between the /342 and /343 wage types.

Execution

Run the RPCALCE0 program for this employee.

Result

The system generates the /370 and /371 wage types using the /342 wage type amount.

Other Terms

&CALC - PC00_M04_CALC;

For more information, see SAP Note 1071282 - CONSULT: List of tags for Spain notes.

Reason and Prerequisites

Reason

This issue is a side effect caused by the solution delivered by the SAP Note 3473016 - ARTISTAS: Incorrect unemployement quota calculated using the /343 wage type.

According to the Spanish Social Insurance, for artist employees the unemployment base represented by the /370 and /371 wage types must be calculated based on the /343 wage type, except if the /343 value falls below the minimum base of /342, in which case the minimum base value is used instead.

Prerequisites

The following Support Package level is the minimum prerequisite:

Release Support Package
S4HCM 102 HR SP 00
S4HCM 101 HR SP 07
S4HCM 100 HR SP 20
SAP_HR 6.08 HR SP D1
SAP_HR 6.04 (EhP6 and EhP7)  HR SP K3

For more information, access SAP ERP Human Capital Management - Support Package Schedules.

Note:

When you implement an SAP Note, some prerequisites might be required. The prerequisites are SAP Notes that have already been released and that change the same context as the current one. Then, even if these prerequisites are not necessarily related to the symptom of this SAP Note, you must implement them for technical reasons.

Solution

From now on, the system calculates the /370 and /371 wage types for artist employees using the /343 wage type amount. However, if the /343 value falls below the minimum base of /342, the system uses that minimum base value instead.

Example:

Following the steps from the example of the Symptom section.

Result

The system generates the /370 and /371 wage types using the /343 wage type amount. If the /343 value is lower, the system uses the /342 minimum base.

Retroactivity:

To avoid problems with retroactivity, the changes were sheltered with the entry 'NOTE' '03720796' in the V_T5F99VD table view.
For further information, see SAP Note 515323 - CONSULT.: Protección de correcciones.
