Symptom

  • How to query the default BI Audit universe to track BO users who have updated reports or universes?

Environment

  • SAP BusinessObjects Business Intelligence 4.x
  • Web intelligence (WebI)
  • Information Design Tool (IDT)

Reproducing the Issue

  1. Log in BI Launchpad
  2. Create a report based on BI Audit Universe
  3. add the respective objects
  4. Observe there is no data present in the report who modified the universe/report.

Cause

  • User details must present in the BI Audit Universe and CMS Database 

Resolution

  1. Log into Central management Console
  2. Create a new user 
  3. Log in to Information Design tool
  4. Perform the changes report/universe
  5. Save the universe/report
  6. Log into BI Launchpad with Administrator account
  7. Create a report based on BI Audit universe
  8. Drag and drop the below objects
  9. Event ID ,Event Type ,User ,Object ,Start datetime ,Object Type
  10. Add the below object in the query result
  11. Object use IN list operator with required universe
  12. Run the report
  13. Expected results will retrieve in the report to track BO users who have last updated/Modified reports or universes

Keywords

BI Audit Universe, CMS ,User, last Modified
