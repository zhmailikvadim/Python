Symptom

  • Dynamic user lists are not visible or searchable in the "Specific users or user lists" section of access control for workspace tabs.
  • Only static user lists can be found in the access control settings.

Environment

  • SAP Build Work Zone, Advanced Edition
  • SAP SuccessFactors Work Zone

Reproducing the Issue

  1. Navigate to the Admin console > Users > User List > Dynamic User Lists.
  2. Create and publish a new Dynamic User List or observe existing Dynamic User Lists.
  3. Navigate to any Workspace
  4. Select the Settings icon > Choose a Tab > Expand the tab settings > Select Access Control > Choose "Specific users or user lists".
  5. Attempt to search for any Dynamic User List; none are found.
  6. Search for Static User Lists; they are visible and searchable.

Cause

  • A dynamic group must first be invited to the workspace before it can be added to a navigation tab's access control.

Resolution

  1. Ensure the dynamic group is invited to the workspace.
  2. Verify the dynamic group invitation status using the workspace settings.
  3. Once the dynamic group is invited, attempt to add it to the navigation tab's access control under "Specific users or user lists".
