Symptom

  • During RE-FX periodic posting (simulation and actual run), an error occurs: "Enter a name and city for one-time account postings."
  • Error code: F5 266

Environment

  • Flexible Real Estate Management (RE-FX)
  • Contract and Lease Management
  • SAP ERP 
  • SAP ERP Central Component
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA Finance
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Execute Periodic Posting with transaction RERAPP
  2. Posting fails with error F5 266: "Enter a name and city for one-time account postings."

Cause

  • Postings to one-time customers are not supported by the periodic posting transaction RERAPP

Resolution

  1. Use transaction RERAOP for postings involving one-time customers.
  2. Enter the address manually within transaction RERAOP.
  3. For periodic postings with transaction RERAPP or single postings from the RECN transaction, use regular customers instead of one-time customers.

Keywords
