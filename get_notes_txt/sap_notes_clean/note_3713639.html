Symptom

From January 2026 new version of VAT form and output XML structure is introduced in Croatia.

Other Terms

Croatia, VAT Changes 2023, PDV, U-RA, I-RA, /CCEE/HRFI_RFUVDE00, RFUMSV00, /CCEE/HRFI_RFUMS, J_3K_VATFORM2026, HR_URA_2023, Food Donations

Reason and Prerequisites

Legal change

Solution

Implement attached correction instructions including manual post-implementation steps.

There are new fields on the selection screen of Croatian VAT return report /CCEE/HRFI_RFUVDE00 in new Food Donations section:

Purchase value and Sales value in rates 5%, 13% and 25%.

These fields must be filled manually.

These are then poulated in the output XML file in tags:

  • Podatak701
  • Podatak702
  • Podatak703
  • Podatak704

 At the same time, there is new SAPScript form J_3K_VATFORM2026, having new section PODACI O OBAVLJENIM DONACIJAMA HRANE ( Information on food donation done ).

This section contains the same information as above mentioned output XML file.

Don't forget to set the new form name in the view V_T007F as described in attached manual steps file.
