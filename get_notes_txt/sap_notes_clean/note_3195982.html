Symptom

User is creating, changing, or deleting a specification in CG02. If the specification type is assigned to category ALLERGEN, NUTRIENT, PURE, REAL, RAW, STREAM, or PACKAGE, then the expected business event is not raised from CG02.

Note: Business events are correctly reported if the user is working with PLM Recipe Development WebUI or S4HANA using the Product Specification API, this only affects changes in CG02.

Other Terms

PLM Recipe Development Business Events; AT_SAVE Exit;

Reason and Prerequisites

This is a programming error.

Solution

Implement assigned correction instruction.
