Symptom

In the Find Test Case/Test Scenario function, the system does not display a message if the object does not exist.

Reason and Prerequisites

When searching for the test cases or test scenarios, you can also enter an ID for the search directly outside the search help. You can search for the entry using the "Search" button. If the test case or test scenario does not exist in the system, the user does not receive any information or message. Popup window remains open and no action is performed. The feeling is that the function does not work.

Solution

If no test case or test scenario was found for the ID entered, a message should be issued.
