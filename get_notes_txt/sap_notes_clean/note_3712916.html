Symptom

You encounter wrong system entries in the SAM4U Tool when navigating to Users --> Locked Users; these users might not exist in the system which are currently displayed.

Other Terms

Software Asset Management, SAM4U Tool, PCE, Locked Users

Reason and Prerequisites

When processing the data for the locked users it was missed to clear an internal table which contains all relevant system entries per user.

Solution

Implement this SAP Note to fix the above-mentioned performance issue.
