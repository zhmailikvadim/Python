Symptom

The probability settings configured for an alternative item group in the BOM are not being applied as expected. Despite setting a specific usage probability for each material, the system selects materials without regard to these probabilities. 

Environment

  • SAP ERP Production Planning (PP)
  • SAP ERP All versions
  • SAP ERP Central Component All versions
  • SAP R/3 All versions
  • SAP R/3 Enterprise All versions
  • SAP S/4HANA All versions
  • SAP S/4HANA Cloud Private Edition All versions
  • SAP enhancement package for SAP ERP All versions
  • SAP enhancement package for SAP ERP, version for SAP HANA All versions

Cause

The current configuration for the alternative item group strategy is set to "2" (100% check). Under this configuration, following the execution of the ATP check, the usage probabilities are automatically adjusted to either 100% or 0%. 

Resolution

To allow for adjustable probabilities, please changing the strategy to "1".
