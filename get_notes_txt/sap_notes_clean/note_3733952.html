Symptom

When the HHKCPFE0 report is executed in background mode or launched twice from the frontend, the global variant used for file-path validation will be reinitialized, resulting in an incorrect “Invalid File Path” error.

This SAP Note resolves the issue by preventing the unintended reinitialization and ensuring correct file-path validation.

Other Terms

HHKCPFE0_F02

Reason and Prerequisites

This is a program error.

Solution

Do either of the following:

  • Apply the Support Package;
  • Implement the corresponding correction instruction.
