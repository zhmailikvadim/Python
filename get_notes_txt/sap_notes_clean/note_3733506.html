Symptom

In the Perform Operation Activity application, duplicate tokens are automatically generated for inspection characteristics with single result control indicator when the following conditions occur simultaneously:

  • Result/Tokens are already recorded for the IC
  • Remark and valuation are entered/selected together in the same operation

Other Terms

Remark, Valuation, Single IC, Perform Work, Token, Data collection, Result recording, Work Instruction, Single Result

Reason and Prerequisites

When remark and valuation are entered together, Odata POST request payload for second query gets replaced with the response of the GET request of the first query and generates duplicate token in the result field.

Solution

Implement this SAP Note for the stated issue and also for the valuation refresh issue.
