Symptom

Clarification is required on whether updating supplier custom fields using the Risk Category Information API for Supplier Risk Exposure will overwrite existing values that were previously loaded using the manual CSV upload.

Cause

Resolution

The Risk Category Information API for Supplier Risk Exposure updates only the custom fields that are explicitly included in the request payload. Any custom fields omitted from the request remain unchanged.

For full details and usage instructions, consult the official API documentation for the Risk Category Information API for Supplier Risk Exposure
