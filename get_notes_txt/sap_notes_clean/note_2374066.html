Symptom

You are performing system copy using SAP Software Provisioning Manager (SWPM) or SAP Upgrade Manager (SUM). R3load breaks at import phase with the error message:

(TPL) ERROR: Unexpected token 'trcflg:' in '/usr/sap/T01/SUM/abap/exchange/tplpack/DDLDB2.TPL' l. 6

Other Terms

DB2 DDLDB2.TPL Unexpected token trcflg

Reason and Prerequisites

The error is caused by incompatible change in R3load, which was introduced with the note 2309736.

1. In case of SWPM use, the error occurs under the following condition:

Export archive is generated by R3load 7.42 or older, but the import is performed with R3load 7.45.

Note: normally the same kernel version is used for import as for export. 7.45 kernel is certified as a downward compatible kernel, therefore it is elideble for use in 7.42 environment.

2.In case of system upgrade (SUM) the error occurs in the phase EU_IMPORT during the import of upgrade DVDs.
