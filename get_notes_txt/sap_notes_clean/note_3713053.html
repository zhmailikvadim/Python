Symptom

Customers are experiencing issues when trying to implement SAP Note 3691869 due to the CIs issue with the object P_PL_SAFTTaxVatItemTotalAux, which is critical for the note's proper functionality.

Other Terms

Poland, SAF-T, Standard Audit File, PL, JPK_VAT, SAP Note 3691869, Object P_PL_SAFTTaxVatItemTotalAux, Sequence Order, Implementation Error

Reason and Prerequisites

The object P_PL_SAFTTaxVatItemTotalAux is required by SAP Note 3691869 but isn't present in the correct state.

Solution

  • Implement SAP Note 3713053:

    • Apply SAP Note 3713053 to your system, which includes the creation of the object P_PL_SAFTTaxVatItemTotalAux_v2. Follow the standard procedure for implementing the note, ensuring all components are thoroughly reviewed and activated.

  • Confirm Object Creation:

    • After implementing SAP Note 3713053, verify the activation of the objects CL_SAFT_PL_TAX_ITEMS and P_PL_SAFTTaxItemTotal in your system.
