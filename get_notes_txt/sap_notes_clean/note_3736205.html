Symptom

Error message: "E: Exception code SKWO cannot be used (/SCWM/EXCEPTION003)" when using exception code SKWO in putaway and inventory counting processes in /SCWM/RFUI.

Environment

  • SAP S/4HANA
  • SAP Extended Warehouse Management

Reproducing the Issue

  1. Execute the transaction /SCWM/RFUI and logon.
  2. Navigate to menu Manual Selection --> Selection by WO.
  3. Enter the WO (use warehouse order related to putaway and inventory counting) 
  4. Use exception code SKWO in the next screen short cut field.
  5. Error message /SCWM/EXCEPTION003 occurs.

Cause

 Exception code SKWO is not applicable to putaway and inventory counting processes in /SCWM/RFUI by standard design as of now.

Resolution

  1. SKWO exception code intended for picking process to skip the current warehouse order and proceed with next warehouse order in /SCWM/RFUI.
  2. Do not use exception code SKWO in putaway and inventory counting processes in /SCWM/RFUI; it is not supported by standard design.
  3. Review the standard function module /SCWM/RF_PT_EXCEPTION to know the applicable exception codes in putaway process. 
  4. Review the standard function module /SCWM/RF_INCOUN_EXCEPTION to know the applicable exception codes in inventory counting processes.
  5. Alternatively, use exception code SCLW to skip counting and proceed to the next warehouse order in inventory counting process.
