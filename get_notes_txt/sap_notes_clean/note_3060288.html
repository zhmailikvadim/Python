Symptom

Note: This note is only relevant for IDXAT2 Customers!

When calling the smart document from a different entity (such as for example from the process document) and then pressing button "Refresh", the original calling context is lost. Afterwards, after wanting to return to the processing document by pressing "Back" or "Return", the original calling context (ie. the process document) is lost.

Other Terms

IDXAT2, Smart Document

Reason and Prerequisites

IDXAT2 200 SP04

Solution

Implement this note or IDXAT2 200 SP05
