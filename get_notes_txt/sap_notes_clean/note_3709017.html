Symptom

When users select Update Taxes, nothing happens and taxes are not calculated.

Environment

Reproducing the Issue

  1. Site is configured for Third-Party Tax Calculation
  2. User clicks Update Taxes on requisition
  3. Nothing happens and taxes are not calculated

Cause

This scenario can occur when the <urn:ProcureLineItemCollection_TaxServiceAPIResponse_Item> is missing from the TaxServiceExportReply.

Resolution

Ensure the tax service returns the <urn:ProcureLineItemCollection_TaxServiceAPIResponse_Item> element within the response structure.

If this element is omitted, the application will not process tax calculations.

See Also

For more information on the correct response format, see the Sample Requests and Responses in Third-Party Tax Calculation.
