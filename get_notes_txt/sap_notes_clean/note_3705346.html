Symptom

You want to know whether it is possible to set retention for the internal audit policy MandatoryAuditPolicy.

Environment

SAP HANA, platform edition 2.0

Resolution

It is not possible to set retention for the internal MandatoryAuditPolicy. As described in Actions Audited by Default Audit Policy | SAP Help Portal, MandatoryAuditPolicy is a default audit policy which helps track the configuration change of audit logs. Generally, there will not be too many entries generated, so you do not have to delete them. However if required, you can manually clear the MandatoryAuditPolicy audit logs using the SQL statements below:

ALTER SYSTEM CLEAR AUDIT LOG FOR AUDIT POLICY "MandatoryAuditPolicy" UNTIL  '2026-xx-xx xx:xx:xx';
ALTER SYSTEM CLEAR AUDIT LOG FOR AUDIT POLICY "MandatoryAuditPolicy" ALL;
