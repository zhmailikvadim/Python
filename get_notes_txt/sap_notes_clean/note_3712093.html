Symptom

When the DSO sends an INVOIC message via REDISND1, the system raises error message /US4G/PROC_BILLING030 "(&1 (&2) is invalid in process &3)".

This issue is relevant for the market role: DSO.

Other Terms

GUB, REDISND1, /US4G/PROC_BILLING030, /US4G/CL_UBD_INVDOC, VAL_INVDOC_ITEM_FOR_GU, INVOIC message, discount percentage

Reason and Prerequisites

Program error.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, you need to re-send the INVOIC message via REDISND1.

You can verify the fix by checking that error message /US4G/PROC_BILLING030 "(&1 (&2) is invalid in process &3)" is not raised when the DSO sends the INVOIC message via REDISND1.
