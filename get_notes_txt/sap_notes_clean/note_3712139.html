Symptom

When creating a claim from the schedules of the purchase agreement, the system displays the error message 'Maintain Manual Condition Type'. This occurs despite the fact that the manual condition is maintained in the calculation procedure. As a result, the claim creation process is failing.

This note ensures that claims will be created successfully without any errors.

Other Terms

/IRM/IPA, condition_manual_check, condition_manual_check, /IRM/IPAGR 343, /IRM/IPPRASP, Manual Condition

Reason and Prerequisites

This is caused by program error.
Note : SAP Note#3712202 is available for ECC release

Solution

Please implement the correction instructions. These corrections are only valid if you have applied the following Support Packs.

Release V/4 – 2023 V2, 2023 V3 and Support Pack 1

Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object
