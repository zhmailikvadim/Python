Symptom

Candidate first name is not updated on the webpage for the Welcome header and the Get to know you better message.

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental. 

Environment

SAP SuccessFactors Recruiting Management 

Reproducing the Issue

  1. Go to customer career site
  2. Login
  3. Edit Profile
  4. Change the first name field to another name
  5. Save changes
  6. Note the name does not update on the header or message
  7. Signing out and back again updates these entries

Cause

This is expected behaviour, there is no metadata refresh pulled for the labels for these entries that include candidate name until Signing back in occurs.

Resolution

Product Management has confirmed that this is expected and there are no plans to enhance this behaviour.

For the updated name to appear the candidate will need to log out and back in again.
