Symptom

Unable to update data fields in floor stock during the inventory download (INVCON03).

Environment

  • SAP Digital Manufacturing Cloud 2602

Reproducing the Issue

  1. Download inventory
  2. No value in Data Attribute field in the manage floor stock

Cause

standard behavior

Resolution

DM currently does not support data type mapping (data fields and field values) for inventory download (INVCON03) when using individual inventory management mode.

Keywords

floor stock, data field update, SAP DM, digital manufacturing, v2 patch API, inventory management, data type mapping, INVCON03, XSLT segment, manufacturing execution, bug, error
