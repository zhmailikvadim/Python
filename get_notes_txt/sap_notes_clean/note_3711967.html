Symptom

 In the process "Start of Supply for Electricity (New SUP View)" (DE_B_GPKE_SOS_SUP), the system sent an incorrect supply scenario from S4 to cloud in step 500, causing errors in the cloud process.

This issue is relevant for the market role: SUP.

Other Terms

 GPKE, SOS, supply scenario, cancel, Start of Supply for Electricity, DE_B_GPKE_SOS_SUP, step 500

Reason and Prerequisites

 The error occurred because of a logic error in the system.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the system sends the correct supply scenario to cloud, and the cloud process flows smoothly. Import the BC set /US4G/3_0_SUP_SP08_NOTE_3711967. You need to rerun the start of supply process, especially when it involves termination of the old supplier. You can verify the fix by checking that the process flows to cloud smoothly without errors.
