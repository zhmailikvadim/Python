Symptom

Inactive tags are displayed in the Growth Portfolio on the Skill/Attribute itself despite being set to inactive.

Environment

SAP SuccessFactors Talent Intelligence Hub

Reproducing the Issue

  1. Add attributes in the Growth Portfolio.
  2. Select a specific attribute type.
  3. Observe that all tags, including inactive ones, are shown in the filter.

Cause

Though the tag is set to Inactive, the mapping between the Tag and Attribute still exists.

Resolution

A code correction has been created to fix the issue. With this fix, if a tag is linked to an attribute, the system will not allow changing its status to Inactive.

This change is planned for the 1H 2025 release. 

Preview: April 13
Production: May 15 - 16

For release timelines, please review SAP SuccessFactors Product Release & Road Map Information.

Workaround:

  1. Navigate to the Attribute screen in the Growth Portfolio.
  2. Remove the inactive tag from the attribute. This will remove the mapping between the attribute and the tag.
  3. Verify that the inactive tag no longer appears in the Growth Portfolio.
