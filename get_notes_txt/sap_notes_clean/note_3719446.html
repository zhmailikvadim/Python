Symptom

This SAP Note resolves the issue of Missing Plant/Sloc View data when distributing Material as a dependent object or via GDD.

  • During the processing of the GDD report, an issue was identified in the remote system where the plant and storage location view data were not synchronized with the material. To address this, the system now retrieves all Plant/SLOCs used in the Subscription ID entered on the screen, as well as the plants defined in the Plant filter within the MAIF configuration.
  • During synchronization of Material as a dependent object(to main objects like Work Order, STR, STO) an issue was identified in missing Plant and Storage location view data. To address this, the system now retrieves all Plant/SLOCs used in the Main objects(STR/STO/WO) and synchronize.

Other Terms

S4 Defense Disconnected, STR, Material, Plant, Storage Location, Sloc. GDD

Solution

Implement the attached correction instructions provided with the NOTE.
