Symptom

Direct Data Access (DDA) integration – the _updated segment data e.g. (segment_xxxxxx_updated) is missing in Customer Data Platform (CDP) Export while the corresponding _value is present.

However, DDA Purpose View of external schemas correctly exposes both _value and _updated for a segment e.g. segment_MarketingOptIn_value and segment_MarketingOptIn_updated fields.

Environment

  • SAP Customer Data Platform
  • Direct Data Access
  • Exports

Reproducing the Issue

Resolution

In the current design of CDP Export, external schema fields for segment update e.g. (segment_xxxxxx_updated) are not exported.
