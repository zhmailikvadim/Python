Symptom

  • Users are unable to search for cases using text from the case description field.
  • The functionality to perform keyword searches within the case description appears unavailable by default.

Environment

SAP Service Cloud Version 2

Cause

Resolution

To enable case search using the case description, follow these steps:

  1. Go to Settings
  2. Go to Extensibility
  3. Go to Case
  4. Go to Field Query Properties
  5. Untick the box labeled "Show only Custom Fields".
  6. Locate and expand the field labeled "description".
  7. Enable the "Searchable" property for the field content.
  8. Save the changes.
  9. Note. this will only work for cases created after the search has been enabled.
