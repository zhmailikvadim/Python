Symptom

The issue started around Feb.9.2026.

Export jobs were failing with error:
"The job failed due to an unexpected internal server error. Please contact your system administrator."

System Monitor showed:
EPM_SERVICE : {"errorCode":"INTERNAL_ERROR","isUserError":false,"jobMetrics":{"artificalStartDelay":"PT0S"}}
mappingID: XXX

Environment

SAP Analytics Cloud

Reproducing the Issue

1. In SAC, go to the Model page, go to Export jobs
2. Trigger the existing export job, but the job fails since Feb.09.2026
3. Re-create the same export job, same error

Resolution

The issue was fixed on February 11th, 2026.
