Symptom

  • EINE-NETPR field exists in transaction MASSOBJ but is not visible in transaction MASS field selection.
  • Entry for EINE-NETPR is confirmed in table MASSFLDLST but remains inaccessible for mass maintenance.

Environment

  • Materials Management (MM)
  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA

Reproducing the Issue

  1. Access transaction MASSOBJ and confirm the configuration for EINE-NETPR field exists.
  2. Check table MASSFLDLST in SM30 and confirm the entry for EINE-NETPR is present.
  3. Attempt to select EINE-NETPR field in MASS transaction; observe that the field is not visible.

Cause

The standard function module within the MASSOBJ transaction restricts sensitive fields, including NETPR, from being displayed in the MASS transaction to prevent unintended bulk price changes.

Resolution

  • The field NETPR is excluded from the field list of MASS in function module ME_LIMIT_FIELDS_INF. 
  • To update the net price and /or conditions for info records, you can consider using the function module ME_INFORECORD_MAINTAIN_MULTI.
