Symptom

You want to clean up entries in table /SMB/CONT_REF

Reason and Prerequisites

Table /SMB/CONT_REF is a system table and any change on it is regarded a modification.
This table contains the SAP Reference Content to be used in Solution Builder to activate Best Practices configuration. 
If you clean it up, you will not be able to import solution or installation data from reference content in solution builder.
Hence, we don't recommend you to clean up this table.

Solution

However, if you want to clean it up anyway, you could run report /SMB/INACTIVE_CONTENT_OVERVIEW.
Choose "Reference Content" and press execute button to display all reference contents.
Select the entries which you want to delete, and press DELETE button in the tool bar.
The deletion operation will be logged in system.
