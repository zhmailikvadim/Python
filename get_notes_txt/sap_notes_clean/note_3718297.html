Symptom

When you click the ATP check button in the sales quotation, the Review Availability Check Result screen does not appear.

Environment

SAP S/4HANA Cloud Public Edition

Cause

This is standard behavior as for sales quotation (item category AGN), the default schedule line category is BN. The ATP function is not active for this schedule line category by design.

Resolution

The workaround could be manually changing the schedule line category by following these steps:

  1. Go to configuration activity 105642 - Assign Schedule Line Categories to maintain the MSLCa column (For example: BP, BA which activate the ATP check functionality).
  2. Change the schedule line category in the sales quotation manually to the values maintained in step 1 (For example: change BN to BP).

Keywords

Sales Quotation, ATP Check, AGN, Schedule Line Category, BN, 105642, Item Category
