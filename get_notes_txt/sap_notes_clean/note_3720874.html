Symptom

Executing a selection from report VF31 - Output from Billing (SD70AV3A) in S/4 HANA results in an exception "[2724] Olap temporary data size exceeded 31/32 bit limit 8025634232 refs in 56 columns"

Reason and Prerequisites

The program is capable of generating an SQL statement that can not be processed using the OLAP engine. Use of the OLAP engine should only be used if necessary. In SAP Note 2856370 the OLAP engine was forced for certain queries. This Note removes this default behavior.

Solution

Implement the Support Package as specified by this SAP Note. In urgent cases you may choose to implement the attached correction instructions.

For those customers that require the use of the OLAP engine to improve the performance of their VF31 queries it is recommended to add the hint using the implicit enhancement spot at the end of the method CREATE_QUERY(CL_BDOC_SELECT_OUTPUT).

The previous behavior from SAP Note 2856370 can be implemented with this example ABAP as part of the end of method enhancement spot.
