Symptom

While doing settlement of condition contract, error message 'AR191' pop ups & not able to create settlement document.

Environment

  • Condition Contract Management
  • Condition Contract Settlement
  • SAP S/4 HANA

Reproducing the Issue

  1. WB2R_SC.
  2. Enter contract ‘XXXXX’ & settlement date as ‘XXXX’.
  3. Execute.

Cause

Tax Destination Country Configuration set up, Settlement Document Type Configuration set up, VAT Registration Number Format.

Resolution

Please set the message AR191 to a 'Warning' via transaction code OBA5.

For reference, please have a look the following note.

2461054 - VAT ISO Code is different from Customer/Vendor Country

See Also

2461054 - VAT ISO Code is different from Customer/Vendor Country

3167273 - Error in proposal of supplier VAT ID and country

Keywords

AR191, WB2R_SC, Settlement, Condition Contract Settlement.
