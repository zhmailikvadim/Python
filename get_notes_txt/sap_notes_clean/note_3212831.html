Symptom

When logging in the system with language A, the Feed facet is showing data with language B.

Environment

SAP Cloud for Customer

Reproducing the Issue

  1. Login in the system with language A.
  2. Navigate to Customers Work Center.
  3. Go to Accounts Work Center View.
  4. Open Account 123 (where 123 is the Account ID).
  5. Go to Feed facet.
  6. See the data is shown in language B. 

Cause

Feed is generated in the login language. If you login in the system with different language the existing feeds will not be translated, only the newly created feed will be in new language.

Resolution

You need to login in the system with the desired language and check the system language then generate a new feed.

Keywords

Language, Feed, English, translated, login, generate, Ticket, Account, facet
