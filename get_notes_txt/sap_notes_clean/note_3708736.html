Symptom

Unified Rendering for Web Dynpro ABAP must be updated in line with error corrections.
These corrections solve the errors described with SAP note  3703267 - Cumulative patch for Unified Rendering 2602

Other Terms

Unified Rendering (UR), WD ABAP, Web Dynpro, Rendering

Reason and Prerequisites

This problem is caused by a program error

Solution

In case that the TCI implementation with transaction snote leads to a 'CALL_FUNCTION_BACK_REJECTED' runtime error in program 'SAPLSTPA', use transaction snote_old to implement or reset the TCI of this note. If a popup appears showing the information 'Error during RFC connection to SAP, cannot update SAP Note' for snote_old, please accept in order to complete the TCI implementation. A correction for transaction snote is in process.

The related TCI Package contains corrections for Unified Rendering from SAPK-757 to SAPK-75709INSAPUI with the following change list:
