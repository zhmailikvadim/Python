Symptom

You encounter issues when importing SAM4U results from systems with a very large user population and the processing stops in status "Parse USMM" with a corresponding dump in the SAM4U system in components CL_LAW3_DATA_BUFFER and/or CL_LAW3_CONS_PROCESS.

Other Terms

Software Asset Management, SAM4U Tool, Extractor, CL_LAW3_DATA_BUFFER, CL_LAW3_CONS_PROCESS

Reason and Prerequisites

You have installed SAP TCI Note 3292019 - SAM4U Data Extractor. The correction will remove the 'Userlist' entries from the USMM package since they are not required to significantly reduce the size of the USMM data package.

Solution

Implement this SAP Note to fix the above-mentioned issue.
