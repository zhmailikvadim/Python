Symptom

You need clarification on whether cf-java-logging-support 3.x is still supported.

Environment

  • SAP Business Technology Platform (BTP)
  • Cloud Foundry (CF)

Reproducing the Issue

Cf-java-logging-support is an open-source library from SAP that provides structured logging and request-metrics instrumentation for Java applications. 

Resolution

Important points to note:

  1. cf-java-logging-support 3.x remains supported and production grade. Support continues at minimum until November 2026 (one year after the 4.0 release). 
  2. It receives security and bug fixes as per need.
  3. There won't be any new features released for cf-java-logging-support 3.x version.
