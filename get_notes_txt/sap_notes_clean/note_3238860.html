Symptom

The report to display log of access to unmasked card number (Transaction: BCA_CARD_DISP_LOG) shows the records with date and time in UTC timezone.

Other Terms

 BCA_CARD_DISP_LOG, RBCA_CARD_DISP_LOG, Log: Unmasked Display of Cards

Reason and Prerequisites

 The logic is such that the data is retreived from the database in UTC timezone and displayed as it is.

Solution

This change will convert the timestamp into the user's timezone in the report before displaying it in the ALV output.

Import the relevant Support Package that is specified for your release. As a manual advance correction, you can implement the attached correction instructions using transaction SNOTE.
