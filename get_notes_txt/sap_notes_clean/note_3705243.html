Symptom

The required permission to export 'Employee Data Replication Confirmation' and 'Employee Data Replication Confirmation Error Message'

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

  • SAP SuccessFactors HCM Suite
    • Employee  Data Replication Confirmation

Reproducing the Issue

1. Navigate to 'Import and Export Data' > Select 'Export Data' 
2. Select Generic Object  'Employee Data Replication Confirmation' Or 'Employee Data Replication Confirmation Error Message' 
Results: Cannot find these objects.

Cause

Missing "Employee Data Replication Confirmation" export permission.

Resolution

1. Please navigate to the user's permission role
2. Search 'confirmation'.
3. Grant the 'Export' permission under 'Data Replication Monitor' > 'Employee Data Replication Confirmation'
