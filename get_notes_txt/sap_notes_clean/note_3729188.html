Symptom

How to export Parametric Type Catalog attributes on Requisition/Purchase orders exports to the ERP? 

Reproducing the Issue

Create a PR and fully approve it. You can see that the catalog attributes are not exported out to the ERP on the below events,

  1. Export Purchase Order.
  2. Export Change Purchase Order.
  3. Requisition Real time budget export Request.
  4. Process Requisition Externally export. 

Cause

The standard export integration events does not support exporting the catalog attributes to the ERP. 

Resolution

A custom field can be added to the respective classes and the value of the catalog attributes can be derived into this custom field. This custom field can then be made exportable for all the necessary export events. Please contact SAP services for a customization request
