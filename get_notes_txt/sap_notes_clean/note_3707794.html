Symptom

You set up the overhead posting to inherit functional area of overhead cost lines from original costs (old logic which can be re-activated when you set the FIN_FLAG  KAZC_FKBER_DERIVE). This setting is not considered by event based overhead logic, empty functional area is posted (what can cause a posting error).

Other Terms

KZPR

Reason and Prerequisites

Program error.

Solution

Apply this SAP Note or corresponding Support Package.
