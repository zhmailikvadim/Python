Symptom

Users are unable to export Data Subject Information reports despite having permissions to access and run the reports.

Environment

SAP SuccessFactors HXM Suite

Reproducing the Issue

1. Login to the instance
2. Go to Data Subject Information
3. Run the Person Search
4. Notice that once the report is ready, the export option are grayed out 

Cause

The issue was caused by missing target population for the user trying to run the report and the subject user.

Resolution

  1. Ensure that the user running the report has target population for Employee export permission to the subject user.
  2. Ensure that the pre-requisite permissions mentioned on this help guide Running the Information Report | SAP Help Portal is granted to the user running the report. 

See Also

Running the Information Report | SAP Help Portal
