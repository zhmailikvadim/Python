Symptom

  • Space utilization doubled
  • The increase is impacting the threshold limit.
  • Metrics report shows significant increases between periods

Environment

  • SAP Sales Performance Management
  • Incentive Management on Legacy Hana
  • Incentive Management on Legacy Oracle

Reproducing the Issue

  1. Review the metrics report on the SAP for Me - Reporting page.
  2. Verify the storage metrics
  3. Observe the significant increase in backups and data metrics in a certain period

Cause

The root cause for the storage increase could not be determined as the concern was raised beyond the two-week retention window for metrics analysis.

Resolution

  1. Note that backups are retained for two weeks in the system as per the defined process.
  2. Data displayed in the metrics section reflects the current database content.
  3. Root cause analysis for storage changes is possible only if concerns are raised within two weeks of the metrics report publication.
  4. Refer to the Monthly Metrics Report on the SAP for Me - Reporting page for further insights.
