Symptom

The header item is returned to the top after scrolling and clicking choose header button. 

Environment

SAP S/4HANA

SAP_BASIS 758

Reproducing the Issue

* Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

  1. Execute transaction to get ALV list
  2. Go to layout screen for document
  3. Scroll the bar on the right.
  4. Select one item and press "Choose header"

         

       5. Position data is displayed for the selected document but the header data returns to the first document before scrolling.

Resolution

Implement SAP Note - 3711733 - ALV: Hierseq Grid scrolls to top after expanding row.

This is the pilot note. 

Keywords

choose header, scroll, header, document, return, go back, Hierseq Grid
