Symptom

You are using the Review Order Plan Items app in SAP Customer Activity Repository applications bundle 5.0 FPS03, 5.0 FPS04, 5.0 SPS05, 5.0 SPS06, 5.0 SPS07, 5.0 SPS08, 6.0 SPS01 or 6.0 FPS02.

You see that the Calculation Basis section for an order plan item in the Review Order Plan Items app does not display any data.

Other Terms

ROPI, Calculation Basis, Rounding Profile, Rounding profile text, Review Order Plan Items, SAP Customer Activity Repository applications bundle, CARAB, Replenishment Planning

Reason and Prerequisites

This issue occurs when there are multiple text descriptions in the table /DMF/RNDPRFL_TXT against a specific combination of rounding profile, language, and logical system for different plants.

Solution

Implement the attached correction instructions in your back-end system.
