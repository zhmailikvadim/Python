Symptom

You open SAP Identity Management Web UI, select the Manage tab and then the Advanced mode for searching entries. You provide at least one search criteria in the fields and choose the Reset button. After that, you choose the Go button.

The message 'Could not search for entries' appears.

Other Terms

IDM, Identity Management, Web UI, Manage Tab, Search, Advanced Search, Reset Button

Reason and Prerequisites

You are using SAP Identity Management 8.0 SP07.

Solution

Apply the attached patch provided in the Support Packages & Patches section of this SAP Note.

Make sure all components are updated to the corresponding SP level before applying the patch.

After applying the patch, this error will no longer occur. You will be able to search when you choose the Reset button before the Go button.
