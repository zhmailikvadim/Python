Symptom

After implementing the SAP Note 3668471 for the extended open item management clearing process, you get a syntax error in program CL_GL_ACCOUNT_SORT_KEY.
The type TT_POSTING_DATE_RANGE is unknown.


The clearing process is not affected, simply the activation of the classes and methods will result in the reported error without functional impact.

Other Terms

IF_GL_ACCOUNT_SORT_KEY, CL_GL_ACCOUNT_SORT_KEY, TT_POSTING_DATE_RANGE

Reason and Prerequisites

Side Effect of SAP Note 3668471.

Solution

Please implement the following correction instruction.
