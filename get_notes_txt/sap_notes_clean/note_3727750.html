Symptom

API_PRAWELLCOMPLETIONDOWNTIME(PRA Well Completion Downtime)

- You are trying to delete all the Downtime reason codes for a given PRA Well Completion downtime record using the action 'UpdateWellDowntime' of the standard API - API_PRAWELLCOMPLETIONDOWNTIME. The API allows this attempt and deletes all the downtime reason codes whereas the corresponding GUI transaction code (O3U_WCDT2)
does not allow the user to do so.

- Additionally, the API does not validate the downtime reason code being inserted or deleted when PRA Well Completion downtime record is updated using the action 'UpdateWellDowntime'. Expected behavior is to throw error if:

  • Downtime Reason Code being inserted already exists in the PRA Well Completion downtime record OR
  • Downtime Reason Code being deleted does not exist in the PRA Well Completion downtime record.
