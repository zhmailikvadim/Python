Symptom

When you execute the report ELStAM Procedure: Transfer XML Data to ELStAM Table (RPCE2SD0_IN), notifications are not assigned to the correct personnel number.

Reason and Prerequisites

  • Two personnel numbers exist in your system for an identification number.
  • For one of the personnel numbers, the executing user has no authorization to read the infotypes or the Customizing of the LSTC is not maintained correctly.

Solution

Implement the attached correction instructions or import the specified support package. After you import the HR Support Package or implement the correction instructions, you do not have to perform any further follow-on activities.

After you implement the corrections, the system issues a corresponding error message in the output log in this case, depending on the error constellation.

If this is a missing authorization, assign the required authorization to the executing user. With sufficient authorizations, a correct assignment can be made.
