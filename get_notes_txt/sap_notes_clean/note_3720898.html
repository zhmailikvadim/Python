Symptom

  • The transaction LM01 currently supports visibility for a maximum of six buttons on screen however the requirement is to have more buttons than that.

Environment

  • SAP ERP Central Component
  • Sales And Distribution (SD)
  • Logistics Execution (LE)
  • SAP ERP
  • SAP R/3
  • SAP R/3 Enterprise 4.7
  • SAP Enhancement package for SAP ERP
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Cause

In standard configuration, transaction LM01 allows visibility for a maximum of six buttons.

Resolution

If you require more than 6 buttons in LM01, you can regroup your transactions and make nested menus.
