Symptom

You are getting following error when trying to change the password of User TMSADM by using report TMS_UPDATE_PWD_OF_TMSADM.

  • Function module "TMS_SEC_CHANGE_PWD" not found.

 

 

 

Environment

SAP NetWeaver

Reproducing the Issue

Run report TMS_UPDATE_PWD_OF_TMSADM on the TMS domain controller, client 000.

Cause

This error message means that the Function Module "TMS_SEC_CHANGE_PWD" is missing in one system of your transport landscape.

Resolution

Please make sure the Function Module "TMS_SEC_CHANGE_PWD" is available in every system of your transport landscape.

To do this, please confirm that both of SAP Notes 1414256 and 1515926 have been applied in all of transport systems of the relevant system landscape.

(You can also implement Note 1691028 if necessary. This note is only for TMS domain links.)

Keywords

TMS_SEC_CHANGE_PWD, TMS_UPDATE_PWD_OF_TMSADM, TMSADM
