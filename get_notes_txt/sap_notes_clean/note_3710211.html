Symptom

"Infotype Actions cannot be deleted" error message when deleting infotypes for a Person via PU00

Environment

  • Human Capital Management (HCM)
  • SAP R/3
  • SAP ERP Core Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA

Reproducing the Issue

  1. Run Transaction PU00
  2. Select a Personnel number needed for deletion
  3. Select IT0000, IT0001, IT0002 and IT0003 but not all infotypes
  4. Execute the report and the error is thrown

Cause

You will receive the error message 'Infotype Actions cannot be deleted (Message no. PG036)', because you can delete infotype 0000, 0001, 0002 and 0003 only, when you delete all infotypes of the person. It is not allowed to delete only infotype 0000 or 0001 or 0002 or 0003 without deleting the rest of the infotypes for the employee

Resolution

Ensure all infotypes are selected when using PU00 when deleting IT0000, IT0001, IT0002 and IT0003
