Symptom

You calculate a surcharge agreed on a percentage basis or discount using the condition technique.

If the percentage value changes during the stay of the case (this can be the case for cases over the year or long patient cases), the surcharge or The discount is calculated on a time-scaled basis using different values.

You want to know which setting is to be made so that the surcharge or Discount is only billed with the percentage value that is valid at the time of admission of the case.

Environment

  • Industry-Specific Component Hospital (IS-H)
  • release-independent

Solution

This means that a surcharge agreed as a percentage or The condition type controlling the discount must have the pricing date E - order date (KOMK-AUDAT).

You can make this setting in Customizing under SAP Healthcare - Industry-Specific Components for Hospitals -> Patient Accounting -> Billing -> Pricing (Using Conditions) -> Conditions -> Define Condition Types.
Enter the value E - Order date (KOMK-AUDAT) for the corresponding condition type on the Control data 2 tab page.
