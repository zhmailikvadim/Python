Symptom

This article is intended to describe how to change the allowed size of the file(s) to upload for the 'File Upload' type Custom Field in Advanced Workflow

Environment

  • SAP Sales Performance Management
  • Advanced Workflow

Resolution

  • A Custom Field ('File Upload' type) created and used to upload files to Advanced Workflow cases
  • To control the size of the file(s) to upload, set the 'Max File Size (in KB)' parameter available under
    Setup > Fields & Forms > Custom Fields > search for the 'File Upload' type Custom Field that was previously created and click on 'Edit' button
  • On the Basic tab scroll down to  'Max File Size (in KB)' parameter and add the desired value in KB using only numbers without decimal or digit grouping symbol (By default, the maximum file size that can be attached to a case is 25 MB = 25600 KB)
