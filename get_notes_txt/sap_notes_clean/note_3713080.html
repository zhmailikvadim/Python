Symptom

  • The issue pertains to the "Get Feedback" functionality in SAP SuccessFactors Performance Management.
  • The request is to restrict the range of selectable providers and control the list of recipients who receive the "Get Feedback" requests.
  • There is no specific Performance Management configuration to restrict visibility; it is controlled by Role-Based Permissions (RBP).

Environment

  • SAP SuccessFactors Performance Management

Reproducing the Issue

  1. Access SAP SuccessFactors Performance Management.
  2. Attempt to use the "Get Feedback" functionality to search for providers or send requests.
  3. Observe that there is no restriction on the range of selectable providers or recipients.

Cause

The visibility of users in the "Get Feedback" functionality is entirely controlled by the "User Search" permission configuration in Role-Based Permissions (RBP). There is no separate configuration specific to Performance Management for restricting visibility.

Resolution

  1. Review the "User Search" permission in Role-Based Permissions (RBP).
  2. Adjust the target population settings within the permission role to align with the desired visibility requirements.
  3. Ensure that the requester has "User Search" permission for the specific target population they need to access.
