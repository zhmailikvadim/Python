Symptom

While trying to upload a file in Maintain Onboarding and Offboarding Document Templates, the system throws the below error:

"error - ChangeSet index 1 - ???HCP Error Message: Http Status Code 400???;File could not be uploaded to PFS"

Environment

SAP SuccessFactors Onboarding

Reproducing the Issue

  1. Navigate to Maintain Onboarding and Offboarding Document Templates
  2. Click on add new document template
  3. Upload the document
  4. Select any id, process type, name and category
  5. Make sure to select the locale as "English (DEBUG)"
  6. Upload
  7. Encounter the error message

Cause

The document was uploaded with the locale English (DEBUG)

Resolution

Upload the file in other language than English (DEBUG)

Keywords

english, debug, language, locale, file, document, form, Maintain Onboarding and Offboarding Document Templates, error - ChangeSet index 1 - HCP Error Message: Http Status Code 400; File could not be uploaded to PFS
