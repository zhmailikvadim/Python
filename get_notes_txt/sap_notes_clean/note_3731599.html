Symptom

When creating survey's, there is no option for 'Advanced Filtering' like there is in the inventory, resulting in not being able to select multiple fact sheets in a single relation filter.

There is a workaround, please refer to the resolution part of this KBA.

Environment

SAP LeanIX surveys

Cause

Surveys currently do not support Advanced Filters

Resolution

  1. In the survey’s filter settings, set the filter mode to “IS NOT”.
  2. Select the required fact sheets from the left-hand menu.
  3. Change the logical operator to “OR”, this keep the filters that were set in the previous step.
  4. If native Advanced Filters support is required in surveys, consider submitting an enhancement request to the product team.
