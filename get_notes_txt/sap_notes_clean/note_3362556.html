Symptom

SAP Cloud Platform Console Client for Neo commands fails with the below error:

*********************************
Error description: Error occurred during initialization of VM
java.lang.Error: java.lang.reflect.InvocationTargetException
Caused by: java.lang.reflect.InvocationTargetException
If you need help, provide the output of the command and attach the log directory [C:\SCP\SDK\tools\log]

*********************************

Environment

  • SAP Cloud Platform
  • SAP Cloud Platform Console Client

Reproducing the Issue

Run the Console Client for the Neo Environment and this activity fails with the above stated error.

Cause

Prerequisistes not met.

Resolution

Ensure that:

  1. Only the latest version of SAP BTP SDK for the Neo environment is used (can be downloaded from SAP Development Tools).
  2. The Prerequisites are met. More info: Setup the Console Client.
