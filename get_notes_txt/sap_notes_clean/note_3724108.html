Symptom

You get following error message when trying to generate calendar table from BAS:

Error while fetching schema details.

Environment

  • Business Application Studio

Reproducing the Issue

  1. Open SAP Business Application Studio.

  2. Follow the steps outlined in the official guide: Create Time-Related Tables.

  3. Attempt to generate or update the time data.

  4. The system returns the following error: "Error while fetching schema details".

Cause

It is caused by incorrect service binding.

Resolution

Following two options are available:

  • Check and correct the service binding (.env file)
  • Manually create the time table with SQL mentioned in the guide Generating TIME DIMENSION Data Using SQL.
    e.g.
    CALL SYS.UPDATE_TIME_DIMENSION('HOUR', 2021, 2026, 0, CURRENT_SCHEMA, 'M_TIME_DIMENSION', ?)
