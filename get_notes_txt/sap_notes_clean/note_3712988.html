Symptom

Even based overhead is posted for an activity allocation document. When you repost the activity allocation, event based overhead is also reposted, but reference to original document may be wrong in new overhead document. The system does not recognize that the overhead has been posted already, and overhead reposting (RTOHC_REPOST) posts overhead again, with correct reference this time.

Other Terms

KZPR RKLU KB65

Reason and Prerequisites

Program error.

Solution

Apply this SAP Note or corresponding Support Package.
