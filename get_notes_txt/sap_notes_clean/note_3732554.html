Symptom

You review the Agreement Posting Summary and expect the system to include data from completed buckets as part of the business requirement. However, when viewing the summary, the system does not display information from completed buckets. The standard system behavior excludes completed buckets from the Agreement Posting Summary, resulting in missing critical information.

This note corrects the behavior by ensuring that completed bucket data is included in the Agreement Posting Summary.

Other Terms

Billbacks, Claims, Completed Flag, Calculation Run, Calculation Bucket

Reason and Prerequisites

This is caused by program error.

Solution

Please implement the correction instructions. These corrections are only valid if you have applied the following Support Packs.

Release V/4 – 2022 V2, 2023, 2023V2, 2023 V3, and 2023 V3 Support Pack 1
