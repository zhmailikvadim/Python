Symptom

You have executed Forecasting based on constant EAC and closed the first period of Monthly planning. If you execute In-Period Forecasting on the second period, ETC and EAC values are set to zero.

Other Terms

Calculate ETC, Calculate EAC, /CPD/PFP_M_A1_PF51, In-Period Forecasting, Forecasting, Monthly Planning, Monthly Plan.

Reason and Prerequisites

Program Error.

Solution

Implement the manual correction provided with this SAP Note.
