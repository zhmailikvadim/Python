Symptom

Incorrect information in SAP S/4HANA 2025 product assistance for SAP Fiori Overview: Required Product Versions.

Location: SAP S/4HANA Product Assistance (version: 2025 (Oct 2025)) -> Technology -> SAP Fiori -> SAP Fiori Overview -> Set Up and Configure SAP Fiori -> Setup of SAP Fiori System Landscape -> Installation of a New Front-End Server -> Required Product Versions

This online documentation for SAP S/4HANA on the SAP Help Portal states in the Database section that SAP Fiori Front-end Server 2023 for SAP S/4HANA supports SAP MaxDB and Sybase ASE databases. This is incorrect and has been corrected in the attached PDF document:

Required Product Versions (p. 3)

Other Terms

SAP Fiori Front-end Server 2025 for SAP S/4HANA, database, unsupported, SAP MaxDB, Sybase ASE, documentation correction, incorrect information

Reason and Prerequisites

SAP MaxDB and Sybase ASE databases are not supproted by SAP Fiori Front-end Server 2025 for SAP S/4HANA.

Solution

Online documentation content on the Help Portal has been corrected for the latest release. Yet, previously published documentation content for version 2025 (Oct 2025) cannot be corrected in the online version. Hence, the content is corrected in the PDF document attached to this SAP Note.
