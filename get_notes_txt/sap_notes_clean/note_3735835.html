Symptom

When running CNMASSSTATUS (or BAPUs) , each Validation/ Substitution rule get called for each WBS 

Environment

  • SAP ERP Project System (PS)
  • SAP R/3
  • SAP R/3 Enterprise
  • SAP ERP Central Component
  • SAP ERP
  • SAP Enhancement package for SAP ERP
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA, on-premise
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Run CNMASSSTATUS 
    Database prof.  000000000001
  2. Enter Object Project/ WBS / Network 
     Select status to change 
     Set or Reset  
     Selection Output 
     Options 
  3. Execute 

Cause

CNMASSSTATUS program calls BAPI_BUS2054_SET_STATUS to set WBS element status.
Even if selecting a single WBS element in the program, the save shall call FM: CJWB_CHECK_BEFORE_COMMIT which shall read "All" the WBS elements of the project in auto Substitution/ Validation  program.

Resolution

Deactivate the Substitution/ Validations. 

Keywords

BAPI_BUS2054_SET_STATUS, BAPI_TRANSACTION_COMMIT,  BAPI_PS_PRECOMMIT.
