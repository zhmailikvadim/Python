Symptom

While running monthly payout , the payout process failed at at "Process Event After Finalize" Task with below error in logs:

AM][EXCEPTION--An Exception has occurred--] Invalid object name 'BrokerLicense'.

Environment

SAP SuccessFactors Sales Performance Management

    • Agent Performance Management Enterprise

Reproducing the Issue

  1. Login to Portal
  2. Search for FinalizePayout in top Search column.
  3. In this form, filter via failed payout or Payout name or RunID.
  4. Navigate to Task and search for task being failed.
  5. Open the logs and you will see error:

AM][EXCEPTION--An Exception has occurred--] Invalid object name 'BrokerLicense'.

Cause

The issue occurred during the Payout process because the Broker License functionality was disabled.

In the 2408 release, this code related to the Broker License feature was removed. 

Resolution

In Release 2402, the Broker License functionality was disabled because it was not being used, having been replaced by the Producer License functionality.

As no issues were reported from this change, all code, tables, and related components for the Broker License have been removed in this release.

This affects all related screens, posting logic, and the following options: 


process.val.product.brokerlicense

process.val.product.brokerlicense.hierarchy

process.brokerlicense.licenseid.edit

process.val.product.brokerlicense.reevaluate

process.val.product.brokerlicense.method

In the 2408 release, this code related to the Broker License feature was removed.
