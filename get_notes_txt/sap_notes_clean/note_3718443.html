Symptom

You are using the service 'BankCardContractProcessingManageBankCardContractIn - RENEW_BANK_CARD_CONTRACT' to renew the main card belonging to an account which also contains an additional card. While doing so, you receive the error message 'The card specified as a main card is an additional card'.

Other Terms

Message Class BCA_CARD, Message Number 053, Include LBCA_OBJ_CNSP_CARDFDO

Reason and Prerequisites

The reference card check in Include LBCA_OBJ_CNSP_CARDFDO does not work if the main and additional card numbers are partly masked resulting in identical masked card numbers.

Solution

In case the card numbers are masked, they will now be unmasked for the reference card check to ensure that their full values are used for the required check. 

Import the relevant Support Package that is specified for your release. As a manual advance correction, you can implement the attached correction instructions using the transaction SNOTE. There are no manual activities.
