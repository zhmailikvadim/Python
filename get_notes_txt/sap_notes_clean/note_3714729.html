Symptom

  • Error message: "F5808 - Field order is a required field for G/L account ..." encountered during asset transaction posting.
  • Issue may occur when using an offsetting account that requires an order, even after entering it in 'Addit.Acct Assignment Line Item' 

Environment

  • FIN Asset Accounting
  • SAP ERP
  • SAP ERP Central Component
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA
  • SAP S/4HANA Finance
  • SAP S/4HANA Cloud Private Edition
  • SAP Fiori for SAP S/4HANA Finance,
  • SAP Fiori for SAP S/4HANA,
  • SAP Fiori for SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Access any asset accounting transaction like ABZON.
  2. Enter an existing asset.
  3. Set Document date, Posting date, and Asset value date to 26.11.2025.
  4. Enter Amount and offsetting account etc. Attempt to post the transaction.
  5. As error occurs, additional account assignment 'order' is entered nevertheless it is not taken over in the document to be posted.

Cause

The issue arises due to additional account assignment requirements for the specified G/L account, which are not automatically fulfilled by the system.

Resolution

At asset transactions like ABZON, an internal order is entered as an account assignment in the entry screen. The order is then correctly derived according to the current customizing setting (note 684659) for EAUFN into the asset line items of the documents in all accounting principles.

Please mind that for APC postings in asset accounting the account assignment order (EAUFN) is only used for statistical costs. So it will only be derived from the master data to asset line items, but not the offsetting accounts like KTANSG or KTREST. Otherwise this would not make sense for statistical costs. Therefor the order is not derived into the offsetting account item as expected. 


If the order is still ought to be assigned, it can be done this when viewing the simulated documents. In the simulated document, each relevant line item can be edited by clicking click the button "Addit.Acct Assignment Line Item".
