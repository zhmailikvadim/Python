Symptom

When trying to add or create a new record for Pension Payout for a rehired user following error prevents you to saving:

???BENEFITS_REC_INFO_INTG_ALREADY_EXISTS_WITH_EXTERNAL_CODE???”

Environment

  • SAP SuccessFactors Employee Central
  • SAP SuccessFactors Global Benefits

Cause

The Enrollment ID linked in the BenefitsIntegrationRecurringInfo record is incorrect. This is causing an error when attempting to create enrollment records.


Resolution

Engineering team needs to run a script to update the BenefitsIntegrationRecurringInfo records with the correct Enrollment ID.
Please engage support team and provide approval to run script to correct the data.

Keywords

???BENEFITS_REC_INFO_INTG_ALREADY_EXISTS_WITH_EXTERNAL_CODE???”, enrollment, pension, payout, error, duplicate, unable, save, benefit,
