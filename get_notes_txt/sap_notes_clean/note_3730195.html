Symptom

When we send back performance form to the previous step we can comment in the comment box.

Is it possible to extract all the comments via reporting?

Environment

Performance Management

Reproducing the Issue

1. Login to the Instance

2. Go to Performance Management

3. Open the PM Form from the Inbox

4. Send the form back to the previous route step with comments

Resolution

When a performance form is sent back, the comments entered during that action are stored in the form’s audit/history (route map comments).

These send-back comments are not available in standard reports.
