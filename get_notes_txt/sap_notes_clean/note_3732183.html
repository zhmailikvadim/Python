Symptom

What is the SAP EPM Add-in used for?

Environment

  • SAP Financial Consolidation
  • SAP EPM Add-in

Resolution

The EPM add-in is designed to give access to SAP BusinessObjects EPM Solutions product data and perform reporting and analysis on this data.

Below is the complete list of the EPM products that can be used with EPM Add-in for both input and retrieval:

Data Retrieval
The EPM add-in enables you to analyze the data of the following OLAP data sources:
- Microsoft SSAS cubes, including SSAS cubes created with SAP BusinessObjects Financial Consolidation, cube designer.
- SAP NetWeaver BW InfoCubes, including BW InfoCubes created with SAP BusinessObjects Financial Consolidation, cube designer.
- SAP BusinessObjects Profitability and Cost Management models.
- SAP BusinessObjects Strategy Management models.
- SAP BusinessObjects Planning and Consolidation, version for SAP NetWeaver, models - version 10.0 and version 10.1.
- SAP Business Planning and Consolidation, version for SAP BW/4HANA, models - versions 11.0 and 11.1
- SAP BusinessObjects Planning and Consolidation, version for the Microsoft platform, models.
- SAP HANA views, including views created with SAP BusinessObjects Financial Consolidation, cube designer..


Data Input
You can also enter data against the following data sources:
- SAP NetWeaver BW InfoCubes
- SAP BusinessObjects Planning and Consolidation, version for SAP NetWeaver, models - version 10.0 and version 10.1.
- SAP Business Planning and Consolidation, version for SAP BW/4HANA, models - versions 11.0 and 11.1
- SAP BusinessObjects Planning and Consolidation, version for the Microsoft platform, models.
