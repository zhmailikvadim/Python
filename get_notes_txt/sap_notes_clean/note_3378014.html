Symptom

Manual LMDB resynchronization with SLD for a specific technical system is returning the following message error:

MAX(1) violated for role SAP_BCCentralServiceMessageServer.PartComponent. Can not associate a SAP_BCCentralServiceInstance with another SAP_BCMessageServer

Environment

SAP Solution Manager 7.2

Reproducing the Issue

  1. Open transaction LMDB
  2. Select tab Technical Systems
  3. Select the affected technical system and click in edit
  4. Select option Advanced
  5. Click in Resynchronize Technical system from SLD
  6. Select the source SLD and click OK

Cause

Inconsistent SAP_BCCentralServiceMessageServer entry in LMDB

Resolution

  1. Open transaction LMDB
  2. Click in LMDB Administration
  3. Select Instance Browser
  4. In CIM Class Name, insert SAP_BCCENTRALSERVICEMESSAGESERVER and click in Search.
  5. Click in column Reference String and select (User-Defined Filter...)
  6. For the field Reference string insert *SID*, where SID is the affected technical system and click in OK
  7. The corrupted entry will appear, select this entry, click in Delete it and Save

Keywords

MAX(1) violated, SAP_BCCentralServiceMessageServer, SAP_BCCentralServiceInstance, SAP_BCMessageServer
