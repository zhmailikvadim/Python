Symptom

  • Not able to pass the same partner function (e.g., Ship-To-Party) in the payload of API_BUS_SOLUTION_QUOTATION_SRV during Solution Quotation creation.
  • Error message: "Error while parsing an XML stream" occurs when running the payload. 

Environment

SAP S/4HANA

Reproducing the Issue

  1. Run the OData service /sap/opu/odata/sap/API_BUS_SOLUTION_QUOTATION_SRV/A_BusinessSolutionQuotation to create a Solution Quotation.
  2. Include multiple occurrences of the same partner function (e.g., two ShipToParty fields) in the request payload.
  3. Send the request and observe the response error: "Error while parsing an XML stream". 

Cause

  • API_BUS_SOLUTION_QUOTATION_SRV is designed to not support multiple occurrences of the same partner function when creating a Solution Quotation.
  • Partner fields such as SoldToParty, ShipToParty, and BillToParty are single-valued and cannot occur multiple times in the create payload. 

Resolution

Modify the payload to include only one value per partner function (e.g., a single ShipToParty, BillToParty, SoldToParty).
