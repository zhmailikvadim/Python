Symptom

When creating a Flexible Group (FG) request using an upload file with modified header dates, the system still creates the FG requests using the parent header dates. The updated dates provided in the file are not being considered by the system.

This SAP Note provides the necessary corrections to address and resolve this issue.

Other Terms

/IRM/GFGRUPLD, Allowed fields, Flexible Group request

Reason and Prerequisites

This is caused by program error.

SAP Note # 3733547 is available for V4 Releases.

Solution

Please implement the correction instructions attached. These corrections are only valid if you have applied the Support Packs.

Release 6.4  Base Package or Support Pack 1, 6.3 Base package

Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object.
