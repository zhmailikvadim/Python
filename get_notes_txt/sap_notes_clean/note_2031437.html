Symptom

With this note the SAPUI5 library com.sap.ui5.ux3 will be upgraded to version 1.20.9.

Other Terms

SAP UI development toolkit for HTML5

Reason and Prerequisites

SAPUI5 version 1.20.8 is installed.

Solution

Install the corresponding support package or the correction instructions.

As the central SAPUI5 note is becomming too large to be handled by SNOTE there is a separate note for every UI5 library. To get a running system all corresponding libraries must be installed in their correct version.

Please check note 2031259 to see which corrections belong together.
