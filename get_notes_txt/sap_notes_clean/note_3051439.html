Symptom

  • In SAP Event Management the Function Module /SAPTRX/BAPI_EH_GET_DATA_GEN is not released for Customer Use
  • Provide information on the use of BAPI /SAPTRX/BAPI_EH_GET_DATA_GEN 

Environment

  • SAP Event Management (SAP SCM-EM)

Resolution

The Function Module /SAPTRX/BAPI_EH_GET_DATA_GEN is unreleased but will be supported if any issues occur. This BAPI allows the use of flexible selection criteria.

The other BAPI which is released is /SAPTRX/BAPI_EH_GET_DATA. This is useful only if the Event Handler GUID's are known and can be used in the selection criteria.

Keywords

/SAPTRX/BAPI_EH_GET_DATA_GEN, /SAPTRX/BAPI_EH_GET_DATA, SAP Event Management, BAPI_EH_GET_DATA_GEN, BAPI_EH_GET_DATA, Event Handler GUID's
