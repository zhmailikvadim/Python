Symptom

You are implementing the output type GOODS_TRANSFER_POSTING in a S/4HANA Private Cloud system. This KBA provides additional information about its usage.

Environment

  • Materials Management (MM)
  • SAP S/4HANA

Resolution

The output type GOODS_TRANSFER_POSTING was developed and intended to be used on Public Cloud S/4HANA systems, not being part of the original development of the BRF+ Output Determination in Inventory Management. The existing documentation for it is hosted in the SAP Help page below:

  • Goods Movement for Transfer Posting (GOODS_TRANSFER_POSTING)

Output determination for transfer postings and other movements that are not delivered with a Print Indicator (T156-KZDRU) in their definition is not included in the standard configuration of SAP S/4HANA Private Cloud systems. This is why this output type is not listed in the default configuration described in SAP Note 2461075.

Although it is possible to configure that output type in a Private Cloud system, this is not supported by SAP Technical Support since deviates from the default and standard configuration. Cases regarding issues and/or questions about these additional configurations are considered consulting cases according to SAP Note 83020.
