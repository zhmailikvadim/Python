Symptom

Runtime Error ASSERTION_FAILED is raised when running CALCULATE CONTRACT LIABILITIES AND ASSETS (Program B)

Environment

  • Revenue Accounting (RA).
  • Financial Accounting (FI).
  • SAP ERP Central Component.
  • SAP ERP.
  • SAP enhancement package for SAP ERP.
  • SAP enhancement package for SAP ERP, version for SAP HANA.
  • SAP S/4HANA. 

Reproducing the Issue

  1. Execute program CALCULATE CONTRACT LIABILITIES AND ASSETS (PROGRAM B) - T-Code: FARR_LIABILITY_CALC and dump is generated>

Runtime Errors: ASSERTION_FAILED
ABAP Program: CL_FARR_CALC_LIAB_ASSET=======CP

Information on where terminated
The termination occurred in ABAP program or include
"CL_FARR_CALC_LIAB_ASSET=======CP", in
"IF_FARR_CALC_LIAB_ASSET~CALC_LIAB_ASSET_POSTED_ON_CONT". The
main program was "RBANK_PROC_START".

In the source code, the termination point is in line 74 of program or
include "CL_FARR_CALC_LIAB_ASSET=======CM009".
Program "CL_FARR_CALC_LIAB_ASSET=======CP" was started as a background job.

Cause

The dump is caused by missing customizing.

Resolution

Please follow these steps to add the missing customizing:

  1. Run transaction FARR_IMG
  2. Run the activity Revenue Accounting Contracts -> Configure Accounting Principle-specific Settings
  3. Set up Contract Balance Presentation (column called Presentat.). Select suitable option for you (either Unbilled Receivable/Deferred Revenue or Contract Liability/Contract Asset).
  4. Save the customizing
  5. Run the contract asset/liability calculation again.

The asset/liability should be calculated without dump
