Symptom

Adding the Functionality to set an External Status for a Payment Order via the Fiori Manage Payments app.

Other Terms

 External Status, Manage Payments, F3647, Payment Order

Reason and Prerequisites

There was no possibility to set the External Status via the Fiori Manage Payments app.

Please apply the SAP Note 3713533, for the related backend corrections and SAP Note 3715298 for prerequisite frontend corrections.

Solution

Please apply this SAP Note or assigned Support Package.
