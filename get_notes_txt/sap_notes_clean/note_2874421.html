Symptom

You are working with Items released for scheduling but even after the Ticket is released to ERP, the Demands related to those Items are visible on the Resource Scheduler.

Environment

SAP Cloud for Customer

Reproducing the Issue

  1. Go to the Service work center.
  2. Select the Tickets view and Open Ticket ID XYZ (XYZ represents the Ticket ID).
  3. Go to the Items tab.
  4. Select Item line 10 (10 represents the Line Item).
  5. Click on more > Release for Scheduling.
  6. Go to the Resource Scheduler work center.
  7. See that the Demand XYZ-10 is there.
  8. Now go back to Ticket ID XYZ.
  9. Click on Actions > Release to ERP.
  10. See that Ticket's and Items' Work Progress are set to Finish.
  11. Go back to the Resource Scheduler.
  12. See that the Demand XYZ-10 is still there.

Cause

Demands are shown until an Assignment is created for them.

Resolution

To make the Demand disappear from the list, you need to assign it to a Technician.

See Also

  • Assignments in Resource Scheduler.
