Symptom

  • Is it normal that the 'Late Run' option is always shown empty for Web intelligence reports in BI launchpad.
  • What does 'Late Run' option in BI launchpad do?
  • How to get the updated value for 'Late Run' option for web intelligence reports?

Environment

  • SAP BusinessObjects Business Intelligence 4.x
  • Web Intelligence

Reproducing the Issue

  1. Login into BI launchpad.
  2. Click on Document tab in the left side.
  3. Observe the option 'Last Run' appearing for webi reports.
  4. This option is seen blank.

Cause

 This is an Expected behavior.

Resolution

The option 'Last run' is for objects that can be scheduled. Schedule any report and the date will be updated under this option.
Check the BI Launchpad User Guide for details.
