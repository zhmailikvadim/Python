Symptom

When MOS executes parallelization variants in T-CODE /MOSB/MASS_PCC, error message >U045 (Variant & (&) does not exist) is raised which results that parallelization objects cannot be created.

Other Terms

/MOSB/MASS_PCC, Parallelization Objects, >u045, /MOSB/FM_CONTRACT_INTERVALS, MOS

Solution

Please implement the correction instructions mentioned in this Note or import higher Support Packages.

This SAP Note is relevant to the following role: MOS
