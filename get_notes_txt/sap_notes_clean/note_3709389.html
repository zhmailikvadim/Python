Symptom

In Cash Flow Analyzer application, you select filter Planning Level or Overdue Planning Level and define condition "not empty'. When navigating to Display Cash Flow Items list, you receive OData error, the data is not displayed.

Other Terms

F2332 ; Error 'null' is not a valid value for C(2,0) from Gateway Error /IWFND/ERROR_LOG

Reason and Prerequisites

 Program error.

Solution

Implement the attached Correction Instruction (CI) or the relevant Support Package.
