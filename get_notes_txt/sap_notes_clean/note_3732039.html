Symptom

  • In SAP Fiori Launchpad Application Manager we are unable to maintain the target mapping parameter as Mandatory with below error:
    Mandatory parameter "<parameter_name>" not maintained in tile "<tile_name>".
  • The parameter is not in tile.

Environment

S/4HANA 2023

Cause

If the parameter is mandatory in target mapping, the app will only be started by a tile having this parameter as well. Within a launchpad app descriptor item, all tiles must be able to start the app and therefore, all tiles must have all mandatory parameters.

Resolution

Therefore, you need to add the mandatory parameter to the tile before you can save it as mandatory.
