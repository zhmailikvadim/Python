Symptom

It is possible to retrieve the name of Job Relationships managers in a document using Rule mapping type. 

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental. 

Environment

SAP SuccessFactors Employee Central

Resolution

To have the name of the job relationships manager mapped to the document, please follow the steps:

  1. Go to Configure Business Rules > Create New Rule
  2. Select Scenario Document Generation Template Mapping Rule under Document Generation
  3. Select Base Object Job Relationships and add any ID and name
  4. In the IF part of the rule, ensure to filter the Relationship Type
  5. In the Then part of the rule, select Job Relationships > Name > Personal Information > then select the required field.
  6. See an example below:
  7. After creating the rule, go to Document Generation - Manage Document Template Mapping
  8. Select the document template, set mapping type as Rule and assign the rule under the respective field.
