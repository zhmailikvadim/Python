Symptom

  • You experience issues importing a table from an SAP Applications Datastore or SAP BW Source Datastore, including:
  • Data Services designer crashes
  • Dumps
  • Failure to import
  • Unusable table
  • Authorization issues
  • error messages
  • warnings

Environment

SAP Data Services 4.2

 

Reproducing the Issue

Import a table from an SAP Applications Datastore

Resolution

See the SAP Table Import Troubleshooting Guide

Keywords

BODS, SAPDS, functions, exception, RAISE_EXCEPTION, abap, dumps, crashing, tracing, warning, parser detected unknown, failed
