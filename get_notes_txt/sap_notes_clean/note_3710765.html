Symptom

You are working with route stock management in the Last Mile Distribution solution.

You want to set stock transfer events in Customizing for Logistics Execution under Last Mile Distribution > Route Stock Management > Specify Freight Order Types Relevant for Route Stock Transfer (LMD_ST_V_FOTYPE). However, the stock transfer events described in the documentation are not correct.

Other Terms

Customizing, route stock management, LOAD_BEGIN, LOAD_END, route stock transfer

Reason and Prerequisites

The issue was caused by outdated documentation.

Solution

The Customizing documentation has been updated and reflects the correct stock transfer events EP_LOAD_END and EP_UNLOAD_END. This change applies to release SAP S/4HANA OP2023 and later.

The changes will be visible with the next feature release. A PDF with the corrected documentation is attached.
