Symptom


Joule booster is being run and you encounter the below error:

*********************************
SAP S/4HANA Cloud System <system> returned a technical error for assignment with ID <ID>
An exception was raised. Communication scenario already in use. Error is for scenario <Communication scenario>. Correlation ID:
*********************************

Screenshot:

Environment

  • SAP Business Technology Platform (SAP BTP. URL: https://cockpit.btp.cloud.sap) 
  • SAP Cloud Foundry (CF)
  • Joule
  • SAP S/4HANA Cloud Public Edition

Reproducing the Issue

You Run the Joule Booster and you encounter the above stated error. 

Cause

The communication scenario has already been activated in the SAP S/4HANA Cloud Public Edition tenant and it is not possible to have another activation of the scenario.

Resolution

Check the below points:

  1. Deregister the SAP S/4HANA Cloud system from the BTP System Landscape and register it again, issuing the token with the “all communication scenarios” scenario group.
  2. Alternatively, register a new the SAP S/4HANA Cloud system with the “all communication scenarios” scenario group and use that system for the formation.
  3. If you would like to use the same Communication Arrangement, then it should be deleted from the backend. For more details check: Extend and Integrate Your SAP S/4HANA Cloud Public Edition- Communication Arrangements.
