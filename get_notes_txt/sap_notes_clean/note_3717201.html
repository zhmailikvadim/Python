Symptom

In the process "End of Supply for Electricity (DSO View)" (DE_B_GPKE_DSO_EOS), step 8950 is executed too late because it waits for precondition step 2600 to complete.
This issue is relevant for the market roles: DSO.

Other Terms

GPKE, End of Supply, DE_B_GPKE_DSO_EOS, Version 2508, Step 2600, Step 8950, Step 2450, Trigger 100% process,  precondition 2600

Reason and Prerequisites

The error occurred because of a configuration issue in the process.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the system executes step 8950 when precondition step 2600 is active. Import the BC set /US4G/3_0_DSO_SP07_NOTE_3717201 to apply the fix. You can verify the fix by triggering the process DE_B_GPKE_DSO_EOS Version 2508 and checking that step 8950 is executed at the correct time.
