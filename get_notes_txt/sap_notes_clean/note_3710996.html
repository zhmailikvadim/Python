Symptom

The user is using output type "GOODS_RECEIPT_LABEL" and attempting to print labels with the standard template RM07ETIKETT of type 2 – DDIC Interface. The standard program SAPM07DR and routine ENTRY_ETIE_PDF are utilized. However, when sending the output item to the printer, the following error messages are encountered:
APOC_OR_MESSAGES 260: No output possible for output item '2' due to inconsistencies.
APOC_OR_MESSAGES 152: The selected output device $ does not support PDF files.

Environment

  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Cause

• The output device is defined with a device type supporting the PDL type "ZPL".
• The S/4HANA output control framework requires the output device to have a PDL type = "PDF" to render the NAST Adobe PDF form "RM07ETIKETT".

Resolution

Use a different output device assigned to a device type with a PDL Type = "PDF".
