Symptom

  • When processing FF.5 with the 'Post Immediately' option, the system issues error message FV 215 stating that the option 'Generate batch input session' is deprecated.

Environment

SAP S/4HANA

Reproducing the Issue

  1. Navigate to transaction FF_5.
  2. Attempt to process with the 'Post Immediately' option.
  3. Observe that the system issues error message FV 215, indicating that the option 'Generate batch input session' is deprecated.

Cause

  • The issue may be caused by the use of old variants where the batch mode option (pa_xbdc) is ticked. This option is hidden in the upgraded system as it is deprecated.

Resolution

  1. Verify the variant being used:
    1. Execute FF_5 -> Goto -> Variants -> Display.
    2. Select the variant where the pa_xbdc field is marked.
    3. Click on the attributes option and attempt to edit the variant.
  2. Execute SE38 -> Program RFEBKA00, display the variant, and use the "Change Attributes" option to edit the variant.
