Symptom

This SAP Note is a composite SAP note for SAP Convergent Mediation by DigitalRoute 9.4 Support Package 3

Other Terms

SAP Convergent Mediation, SAP Convergent Policy, DigitalRoute, SAP CC, SAP Convergent Charging, SAP Hybris Billing mediation, SAP Hybris Billing service control, MediationZone.

Reason and Prerequisites

You want to use the solution SAP Convergent Mediation by DigitalRoute 9.4

Solution

SAP Convergent Mediation 9.4 by DigitalRoute Support Package 3 equals partner release number MediationZone 9.4.1.11, s. also SAP note 2925138.

The bug fix appendix can be found as PDF attached to the note. The most current version is available in the documentation portal InfoZone of DigitalRoute, use this link MZ_94_SP_1_ER11_Release_Notes.pdf, s. also SAP note 2924977. Please be aware that the bug fix appendix document included in the delivery file might meanwhile be updated by DigitalRoute and is outdated.

The package is available on SAP download center, check attachment 'How to find Support Package 2 for SAP CM BY DIGITALROUTE 9.4.pdf' on how to find it.

Be aware support packages for SAP Convergent Mediation by DigitalRoute always contains the full package scope (not only the corrections) and replaces the updated components of the current installation.
If you do not have a permanent license key check SAP note 2751421.
In case you encounter problems when installing the software, report an incident using component XX-PART-DRT-CMD.
