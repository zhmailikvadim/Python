Symptom

S4HANA and SAC users want to create a Live Direct connection to S4HANA Cloud (Live Data Connection to SAP S/4HANA Cloud Edition via OAuth | SAP Help Portal).

Steps from above help page were performed to configure S4HC communication system, and communication arrangement. However, SAC connection does not work and Network trace shows below error message for respective GetServerInfo request.

  • https://<SAC Host>/s4hcremotes/<SAC connection Name>/sap/bw/ina/GetServerInfo
  • unsupported_response_type,
    Invalid response!Response body is not application/json encoded

Environment

SAP Analytics Cloud 2026.2.5 

Reproducing the Issue

  1. Attempt to create an SAC Live Direct connection to S4HANA Cloud Edition via OAuth authentication.
  2. Connection cannot be established and Network Trace shows following error message.
    • https://<SAC Host>/s4hcremotes/<SAC connection Name>/sap/bw/ina/GetServerInfo
    • unsupported_response_type,
      Invalid response!Response body is not application/json encoded

Cause

This error message suggests there is a missing configuration step, which is setup same IdP for SAC and S4HANA Cloud.

Resolution

Proceed with configuration to setup same SAC IdP into S4HANA Cloud environment.
