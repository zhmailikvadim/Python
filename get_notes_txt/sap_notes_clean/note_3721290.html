Symptom

The transaction /STTP/SNR_MOVE considers active ranges for moving serial numbers to the shadow table. However, the operation should be allowed only for the ranges in status "Closed" or "Protected".

Other Terms

SNR, Status, /STTP/SNR_LIST, /STTP/SNR_LIST_A

Reason and Prerequisites

The reason is a coding error.

The note only applies to SAP Advanced Track and Trace for Pharmaceuticals.

Solution

Implement all correction instructions.
