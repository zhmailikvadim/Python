Symptom

You have created an extension field of type Indicator and you have added it to a report and the report selection. When you open the value help in the report selection, it only lists one value, for example # (unassigned), even though it should list # and X.

Environment

SAP Business ByDesign

Reproducing the Issue

  1. Go to the work center Business Analytics, view Design Reports.
  2. Open the report which contains the indicator field.
  3. In the report selection, open the value help of the indicator field.

It only shows the value #, and the value X is missing.

Cause

This is a current issue with the value help of indicator fields.

Resolution

We are preparing a correction which is scheduled to be done with the 1905 release. In the meantime, you can manually type the value X into the selection and then run the report.

Keywords
