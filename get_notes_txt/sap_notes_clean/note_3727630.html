Symptom

In the Pricing Workbench (Fiori), records were not displayed for a specific condition type due to a missing dictionary relation update between Country and Region in Price Review and the Pricing Workbench.
This note resolves the issue by updating the required dictionary relation, ensuring that the records are displayed correctly in both Price Review and the Pricing Workbench in Fiori.

Other Terms

/IRM/GPRPWM, Price records, Dictionary update,/IRM/IPPDM,/IRM/CL_GIGRID_SERVICES

Reason and Prerequisites

This is caused by program error

Please implement SAP Note#3667549 as a Prerequisite note

Solution

Please implement the correction instructions attached. These corrections are only valid if you have applied the Support Packs.

Release V/4 - 2023 V2

 Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object.
