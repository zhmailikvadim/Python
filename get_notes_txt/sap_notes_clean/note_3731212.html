Symptom

A recoursive call in the history manager for the history state can result in dump. During BOL reset, the method that activates retention of the collection needs to check the import parameter for activation to prevent the recoursive call.

Other Terms

CL_CRM_IU_IC_BOL_TREE_NODE_COL, SET_RESET_SURVIVAL_MODE, CL_BSP_WD_HISTORY_STATE_CONT, REMOVE_ALL_ITEMS

Solution

Please implement the correction instructions or install the corresponding support package.
