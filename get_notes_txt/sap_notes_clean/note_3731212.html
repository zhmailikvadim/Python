Symptom

When we do correction post diversion is done in Back-to-back( B2B ) scenario, the LDC release execution failed

Other Terms

B2B, Diversion, Correction, LDC Release

Reason and Prerequisites

For B2B scenarios, after we perform diversion, LDC release if giving issues after performing correction. Correction ticket was going into exception because of incorrect handling of nomination items, because of which load and unload items were not identified correctly, this is fixed now.

Solution

Implement the CI to fix the issue.
