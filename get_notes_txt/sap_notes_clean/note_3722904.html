Symptom

Error: an error occurred for the property locationAltKey: is not found 

Reproducing the Issue

  1. Report an ARRIVAL event from the MFE application.
  2. Observe that the event message contains the stopId but does not include the locationAltKey property.
  3. Check the event-to-action script in the Manage Models app.
  4. Property locationAltKey is not found. 

Cause

The MFE application relies on the stopId to correlate the reported event with the planned stops in the freight document. The locationAltKey is not explicitly passed in the payload of the actual event.

Resolution

  1. Update the Event-to-Action (E2A) script to handle the MFE data structure.
  2. Modify the script to check if the locationAltKey exists on the actual event.
  3. If the locationAltKey is missing, check if the actual event is matched to a planned event (usually via actualEvent.refPlannedEvent).
  4. Retrieve the locationAltKey from the referenced Planned Event (actualEvent.refPlannedEvent.locationAltKey).
  5. Test the updated script to ensure it resolves the error.
