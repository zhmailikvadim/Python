Symptom

  • In a Project Management project, navigate to tab "SAP DMS" and click on an easy document. 
  • A popup opens with the links to the services for accessing the document.
  • The popup only contains the service and link for "SAP Easy Document Management", the other services and transactions expected based on the customizing are missing. 

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

  • SAP Portfolio and Project Management (PPM)
  • SAP Portfolio and Project Management for SAP S/4HANA (PPM)
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition 

Reproducing the Issue

  1. Open a project in Project Management.
  2. Go to tab "SAP DMS".
  3. Click on a document link.
  4. Look at the links (transactions) which are available for use in popup "Object Links".
  5. The link to transaction "SAP Easy Document Management" exists. All other transactions which are maintained in the customizing for this object link are missing (e.g. 0CV02N, 0CV03N, 0CV04N, 0CV11, 0CV12, 0CV13).

Cause

  • A web server has not been maintained in the customizing for the DMS object links.
  • This is required for the services to work: without this customizing, the service will not be available.
  • The only exception is service "SAP Easy Document Management" (i.e. 0EASYDMS), which has a dedicated logic.

Resolution

  1. Access the customizing for the DMS Object Link and maintain a valid Web Server.
  2. The standard object type which is used is 0EASYDMS.
  3. In case a custom object type is used, then the following applies to the custom object type.
  4. SPRO
