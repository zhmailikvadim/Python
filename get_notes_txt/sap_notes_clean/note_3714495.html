Symptom

In Starter Development System, it's impossible to create a package with Transport Layer "SAP".

Error "Use transport layer" (Message No. TR427) is reported.

(Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental)

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Access the Starter Development system in ADT.
  2. Attempt to create a Z* package* with transport layer SAP.
  3. Encounter the error message "Use Transport Layer" (Message No. TR427).

Cause

The Starter Development system, which is used for evaluation and exploration of the SAP System, is a standalone system. So the Starter Development system does not have transport layers. It is only provided with the ZLOCAL package and ZLOCAL software component.

Resolution

The Starter Development system does not have transport layers.

It's only possible to use ZLOCAL package or create a sub package in it.

To export the development objects from the starter development tenant and import them into the development tenant, use the abapGit plugin.
