Symptom

In the Service and Asset Manager application, when adding a document to a work order/functional location/equipment, the document page will show all other local documents linked to that object.
This SAP Note will correctly remove other local documents linked to the object on the document add page.

Solution

Perform the following Steps:

  • Save the coressponding Asset Manager version of 3243941.zip file from the attachments on this note to your file system 
  • From the SAP WEB IDE or BAS, click on the MDK Development tab icon along the left side 
  • In the WebIDE workspace, right-click the MDK project name (SAP Asset Manager) and choose Import -> From File System 
  • Click the Browse button on the Import dialog and select the 3243941.zip file from your file system 
  • Check the Extract Archive check-box on the Import dialog, and click OK. 
  • On the import confirmation dialog, click Import 
  • Follow the normal deployment process to move note to production system
