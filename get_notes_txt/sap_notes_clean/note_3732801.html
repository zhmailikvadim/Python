Symptom

You use transaction FF67 to process manual bank statements. When you navigate to the "Statement Overview" (F8), the overview is displayed. This may be very long and therefore confusing.

There are no options for sorting or setting filters in this list.

Other Terms

FF67 manual bank statement EFART = 'M' SAPMF40K

Reason and Prerequisites

 Previously, this function was not provided.

Solution

Implement the automatic and manual correction instructions in your system or import the relevant support package.
Afterwards, you can sort data, set filters, and delete filters in the overview of manual bank statements.

You can also change, select, and save layout variants. 
You can use "Settings -> Layout -> Layout Management" to manage these variants and set them as initial variants in the transaction.
