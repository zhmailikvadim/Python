Symptom

During cleanup of deleted partitions in the context of data tiering movements (please refer SAP Note - 3645941), below runtime error is observed.

Category                  ABAP programming error
Runtime Errors         OBJECTS_OBJREF_NOT_ASSIGNED
Except.                    CX_SY_REF_IS_INITIAL
ABAP: Program         CL_RSO_DTO_SYNC_COLD==========CP

Other Terms

DTO, data tiering optimization, ADSO, advanced DataStore object, DELETED, RSOADSODTO

Reason and Prerequisites

Program Error. Symptom is observed only when RSADMIN parameter DTO_DROP_PART_<adso-pattern> is maintained as explained in SAP Note - 3645941.

Solution

To solve the issue please apply the Support Package as mentioned below. In urgent cases you can use the correction instructions.

Delivery Information

  • SAP BW/4HANA 2023
    Implement Support Package 09 for SAP BW/4HANA 2023 (SAPK-40009INDW4CORE) into your SAP BW/4HANA system. The Support Package will be available as soon as SAP Note 3678578 with the short text "SAPBWNews SAP BW/4HANA 2023 SP09", which describes this Support Package in more detail, is released for customers.

In urgent cases you can use the correction instructions.
