Symptom

Request to use migration tool to migrate quotes from quote 1.0 engine to quote 2.0 engine.

Environment

SAP SALES CLOUD CPQ

Resolution

Case should be submitted to SAP CPQ support team to enable the feature.

As whole migration to quote 2.0 engine should be treated as project, it's highly recommended to review: SAP CPQ Migration Guide to Quote 2.0 Engine | SAP Help Portal 

See Also

The migration tool is depended of the switch between quote 1.0 to quote 2.0, as per: 3511209 - How to request migration from Quote 1.0 to Quote 2.0 engine for CPQ - SAP for Me 

The Quote Migration Tool is the feature that is used for migration of Quote transactional data from Quote 1.0 to Quote 2.0, more information on: Quote Migration Tool | SAP Help Portal
