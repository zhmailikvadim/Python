Symptom

You run the Programa de cálculo de nómina Argentina (HARCALC0) report for an employee in the following scenario:

  • The employee received the SAC payment, and the benefit for the contribution discount was generated. 
  • In the month of the SAC payment, an AJUS off-cycle is executed.

In the scenario described above, the system is generating the Red.Cons. Ley 27430 - SAC (/CRS) wage type when it should not.

Other Terms

&SSOCIAL - For further information, see SAP Note 1152671 - CONSULT: List of tags for Argentina notes

PC00_M29_CALC;

Reason and Prerequisites

Reason

The problem happens because the system does not check if the employee has a change in the worked days. The reduction will be generated if such an event occurs.

Prerequisites

The following Support Package level is the minimum prerequisite:

Release Support Package
600 HR SP I7
604 HR SP F3
608 HR SP 81

For more information, access the SAP ERP Human Capital Management - Support Package Schedules.

When you install an SAP Note, some prerequisites might be required. The prerequisites are SAP Notes that have already been released and that change the same context of the current one. Then, even if these prerequisites are not necessarily related to the symptom of this SAP Note, you must implement them due to technical reasons.

Solution

From now on, the system does not generate the /CRS wage type.
