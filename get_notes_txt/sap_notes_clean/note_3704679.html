Symptom

How to export double quotes in the data via Integration Center

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

  • SAP SuccessFactors HXM Suite
    • Integration Center

Cause

Resolution

If we need export value - Test"AAA", we can use Integration Center calculation field to export - "Test""AAA""" 
Wrap the data with double quotes, and replace one double quote with two double quotes.

Refer to: 3390735 - Field value is returned enclosed by quotation characters when using Value Lookup Table - Integration Center
