Symptom

The new PPF action introduced with note 3440456 which provides a new recommendation item (Create New Outbound Delivery) to recreate an outbound delivery if the logistics availability is 'Sent Back by Repair Center' only works the first time.

Other Terms

In-House Repair, Deletion of Outbound Delivery, Repair Object, in-house service, service object, in-house service object; LIPS; LIKP

Reason and Prerequisites

Program Error as the delivery buffer was cleared between creation and saving the delivery.

Solution

Attached Correction Intruction.
