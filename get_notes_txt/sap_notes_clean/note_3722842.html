Symptom

During trade sales contract creation, when there is a change in the default values of Ship-To/Sold-To from header partner data and add the additional Ship-To/Sold-To details which is different from contract header partner data then system throws an error.

Replication Steps -

- Create a trade sales contract using WB21 transaction.
- Change the Ship-To/Sold-To default value from header partner data.
- Fill the item details and provide the Valuation Point, DPQS, Tolerance data.
- During optionaity data, add the additional Ship-To/Sold-To details which is different from contract header partner data.
- Click on Save. System throws an error message.

Other Terms

Agricultural Contract Management; ACM; Optionality; Partner;  Ship-To/Sold-To

Reason and Prerequisites

Program Error

Solution

Import the relevant support pack or apply the correction instruction(s).
