Symptom

You start "Step 1" in transaction SU25 using the menu "Profile Generator". You now choose "Back" (F3), "Exit" (SHIFT + F3 ) or "Cancel" (F12). Even though no action was performed, the system updates the time stamp for step 1.

Other Terms

SU25_COMPARE PRGN_STAT

Reason and Prerequisites

This is a follow-on error caused by the correction delivered with SAP Note 1691993.

Solution

Use the Note Assistant to implement the correction instructions or import the relevant Support Package.
