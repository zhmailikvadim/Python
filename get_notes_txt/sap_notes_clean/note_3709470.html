Symptom

BNK_BATCH_ITEM contains all the data records; however, the corresponding records are not visible in transaction code BNK_MONI.

Environment

  • Bank Communication Management
  • S/4HANA
  • SAP ERP
  • SAP Enhancement Package for ERP

Reproducing the Issue

  1. Run transaction BNK_MONI.
  2. Observe that batches are not displayed.

Cause

Related payment data is missing from REGUHM/REGUH tables.

Resolution

Transaction BNK_MONI in Bank Communication Management (BCM) depends on data from the database tables REGUHM and REGUH. Incomplete migration or deletion of entries in these tables can lead to an incomplete data display in BNK_MONI, thereby affecting the monitoring and execution of payment transactions. It is not possible to recreate the deleted REGU* entries. 

Deletion of payment data is only possible if the respective authorization has been granted.

Further details about reorganizing data in the payment program can be found in SAP Note 574350 - F110: Reorganization of Payment Data.

See Also

  • SAP note 574350 - F110: Reorganization of payment data.
  • SAP note 2006650 - BCM: No deletion of payment runs during authorization
