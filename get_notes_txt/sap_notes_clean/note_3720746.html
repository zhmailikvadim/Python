Symptom

Error occurs in the Production Order Without Material - Create (CO07) app when attempting to create a Production Order, and system termination occurs.

Error message displayed:
"No default value maintained for op. generation for order type xxx (mat )"
(Message no. CO831).

Environment

SAP S/4HANA Cloud Public edition

Cause

The reference operation set is empty and does not contain any operations, and default values for operation generation are not maintained in SSCUI 102534 for the selected order type in the relevant plant.

Resolution

  1. There is a selected reference operation set with operations maintained.
    If it is valid, the operations will be copied and no operation generated. Hence SSCUI 102534 – 'Define Default Values' configuration is not needed.

  2. The reference operation set is empty and has no operations maintained.
    If it is valid, the system allows the generation of an operation. For that, the customizing must be setup in SSCUI 102534 – 'Define Default Values'.

See Also

Refer to: SAP Note 3250589 - Error CO831 "No default value maintained for op. generation for order type &1 (mat &3)" when converting planned order to production order.
