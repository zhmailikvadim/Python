Symptom

You declare your imports to the Swedish customs server from SAP GTS.

The messages in the context of the customs warehouse use the XML data format. When XML messages are sent, they can be checked against the XSD schema of the message.
If the XSD schema is violated, the converter transfers an appropriate check log and IDoc to SAP GTS. This function is introduced with SAP Note 2730287.

This SAP Note enhances the outbound delivery check for customs processing in Sweden.

Solution

Implement the attached correction instructions or import the specified support package.
