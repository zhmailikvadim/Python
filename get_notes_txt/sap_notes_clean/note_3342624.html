Symptom

Service provider provide the names of the trusted Identity Providers and prompts the user to choose one.

Environment

SAP NetWeaver, SAP NetWeaver Application Server for SAP S/4HANA.

Cause

The SAML configuration at your Local Provider was set to manually choose the Identity Provider. 

Resolution

Select the default Identity provider and on Identity Provider Discovery, set selection mode to Automatic, that means the user will not need to select the default authentication provider. It will be selected automatically.

  • Start SAML 2.0 configuration through SAML2 transaction.
  • At the Local Provider tab, go to Service Provider Settings.
  • In edit mode, under Identity Provider Discovery Mode, Selection Mode field select it as Automatic.
  • Save.
