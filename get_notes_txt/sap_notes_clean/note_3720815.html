Symptom

It is not possible to use the following for Data Source Extension:

  • C_SALESDOCUMENT_F8712 (Sales Document)
  • C_SALESDOCUMENTITEM_F8712 (Sales Document Item)
  • C_SALESDOCUMENTFILTER_F8712 (Sales Document Filter)

Environment

SAP S/4HANA Cloud Public Edition 

Cause

The Help documentation is inaccurate and needs to be corrected.

Resolution

The fix for the documentation is planned for CFD 2602.2, which is scheduled for the 16th of April 2026. See the following 3688194 for more information on the Patch Collection Information.

Please note that SAP may alter the date of this fix without any further notice. This date is planned but not guaranteed.

Keywords
