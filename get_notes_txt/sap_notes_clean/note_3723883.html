Symptom

How do I map Mobile Number in Ariba to populate the Mobile Phone field under Address section in ERP?

Resolution

To ensure that the Mobile Number in Ariba populates the Mobile Number field under Address section in ERP, configure the supplier field mappings for mobile number synchronization as follows:

  1. Create a Yes/No answer type question to indicate whether the phone number is a mobile number using the supplier field mapping vendor.addressExt.telephone[0].mobilePhoneNumberIndicator
  2. Create another question to enter the phone number using the supplier field mapping vendor.addressExt.telephone[0].numberSubscriberID
  3. Ensure the following mapping logic:
    • If the Yes/No question is answered Yes, the value of the phone number question will populate the Mobile Phone field under Address section in the ERP system.
    • If the Yes/No question is answered No, the value of the phone number question will populate the Telephone field under Address section in the ERP system.
  4. Adjust the index [0] in the supplier field mappings as per the business requirements.
