Symptom

When the Gross Quality Loss Fee price component is added at the item level in a Sales Order, it does not appear in the Sales Order Volume with Price Component report.

Environment

SAP Business ByDesign

Reproducing the Issue

  1. Go to Sales Order work center.
  2. Select Sales Order view.
  3. In the drop-down, select All Orders.
  4. Search for Sales Order ABC (Where ABC represents the Sales Order ID).
  5. Click in Edit and View All.
  6. Go to Pricing and Invoicing tab.
  7. Under Pricing subtab, the respective line item.

Gross Quality Loss Fee is visible.

  1. Go to Sales Order work center.
  2. Select Reports view.
  3. In the drop-down, select All Report.
  4. Search for Sales Order Volume with Price Components report.
  5. Open the report and search for Sales Order ID ABC.

Quality Loss Fee is not visible.

Cause

This is the expected system behaviour.

Resolution

The FSI view contains a condition that skips price component 050 (Gross Quality Loss Fee), consequently, this price component is not visible in the Sales Order Volume with Price Component report.

Keywords

Does not appear, not available, missing, quality loss fee, 050, not visible.
