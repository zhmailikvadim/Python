Symptom

The Fiori app "Create Store Order" shows no articles because Init Load (WDBI_HPR) calculates the Valid From Date incorrectly. The correction from SAP Note 3573972 is only applied for Manual Load ('A'), not for Init Load ('I').

Solution

Function PACKETS_BUILD_FOR_ALL, must be extended so that the correction from SAP Note 3573972 also applies to Init Load ('I'), not just Manual Load ('A'). Changing if packet_info-dld_mode = 'A'. to if packet_info-dld_mode = 'A' or packet_info-dld_mode = 'I'. ensures that both load modes calculate the date correctly.
