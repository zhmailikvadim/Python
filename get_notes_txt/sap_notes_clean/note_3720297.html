Symptom

Multi-User Quote Collaboration has been activated, but the Impact Notifications are not functioning.

Environment

SAP CPQ

Reproducing the Issue

  1. Activate Multi-User Quote Collaboration in the system
  2. Make changes within the CPQ Quote object by multiple collaborators
  3. Observe that notification popups for impact changes are not displayed.

Cause

Possible Causes:

  • Backend feature hasn't been activated for Impact Notifications

OR 

  • It is activated but Responsive Templates need to be updated.

Resolution

Please create Support Ticket and reference this KBA.
