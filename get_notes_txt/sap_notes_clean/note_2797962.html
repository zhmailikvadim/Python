Symptom

There is a plan for the stack XML file, but the database host is separated from application server host, e.g. application server is on Windows while the database host is on LINUX. So there is question about which OS type should be selected in OS/DB selection step in Maintenance Planner.

Environment

Maintenance Planner

Reproducing the Issue

1. Plan the stack file for one system

2. Select the OS/DB files

Cause

The kernel files will be selected while selecting different OS/DB types.

Resolution

Select the OS type of the host where the Software Update Manager will be ran and select the DB type of the system.
