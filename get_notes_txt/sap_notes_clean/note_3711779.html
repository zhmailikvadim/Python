Symptom

 The system triggers a runtime dump when selecting Page Size or All Pages in the ALV Grid while using any language other than English.

Other Terms

Page Size, All Pages, VTA grid, /VTA/BRXM, Runtime error

Reason and Prerequisites

This is caused by program error

SAP Note#3711786 is available for ECC Release

Solution

Please implement the correction instructions attached. These corrections are only valid if you have applied the Support Packs.

Release V/4 - 2021, 2022, 2022 V2, 2023, 2023 V2, 2023 V3 and 2023 V3 Support Pack 1
