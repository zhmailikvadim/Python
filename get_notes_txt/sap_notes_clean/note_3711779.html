Symptom

In XML file of report HSKCELD0 authority allows up to 20 characters for field "Title". Until now, we only support up to 15 characters.

Other Terms

Report HSKCELD0, field Title, XML file.

Reason and Prerequisites

This SAP Note is relevant only for Slovakia (HRSK).

Solution

This SAP Note delivers modification of report HSKCELD0 to support length of field "Title" up to 20 characters (from 15 characters). Limitation of 20 characters is defined by authority (in latest valid XSD schema).

 

Implementation

  • As a general rule, SAP recommends that you install a solution by applying a Support Package. However, if you need to install a solution earlier, use the Note Assistant to implement the correction instruction. You can find more information about the Note Assistant in SAP Service Marketplace, under service.sap.com/note-assistant. 
  • For customers on the lowest releases EHP6, EHP7 (component HR-CEECSK 604) there are no instructions and the change needs to be applied via Support Package.
