Symptom

When you change the function /IBS/RD_ADD_DOCUMENT, a template from an existing document is used as the basis for a new document. However, the amount field is copied together with the template.

Reason and Prerequisites

This amount field is not deleted in the function and is therefore transferred to all new documents.

Solution

The function has been adjusted and the field for the amount has been deleted.
