Symptom

There is a DTP with a target characteristic which has time dependent attributes. The records with the same key items as the key times in the time dependent Q master data are loaded from source objects, but could not be updated to the target master Q table correctly, even the "Handle Duplicated Record Key" option has been set in the DTP update tab.

Environment

  • SAP BW/4HANA 2.0
  • SAP BW/4HANA 2021 

Cause

SAP program error

Resolution

Implement the following SAP Notes:

3341495 - BW4: Gaps in SIDs seen during master data attribute update.
3253251 - Classic master data update: Runtime optimization in "Generate Navigation Data" step ineffective
3281749 - BW4HANA: Master data update/load does not update attributes correctly

Keywords

Master data maintenance, CL_RSDMD_UPDATE_MASTER_DATA, Gaps in SIDs, RSDMD_SID_CREATE_CHECK1
