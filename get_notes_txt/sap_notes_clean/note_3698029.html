Symptom

The creation of a Retail Site in a country/region different from the Company Code in the Retail scenario (from the CBC Organization app) is not supported.

Environment

  • SAP Central Business Configuration
  • SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Attempt to create a Retail Site in a country/region different from the company code in the S/4HANA Public Cloud Retail system using the CBC Org app.
  2. Observe that the system restricts the creation of foreign Retail Sites.

Cause

Creating a foreign site (a site in a country/region that differs from the company code country/region) is not possible in CBC Org app: the foreign organizational structure set up is only for Foreign Plant and Foreign Shipping Point. You may find more information in SAP Help - Foreign Plant: If you choose a different country/region than the country/region of the plant's company code, or a country/region which is not in your scope, a Foreign Plant or Foreign Shipping Point checkbox appears and is automatically selected to indicate that it is a foreign plant or a foreign shipping point. 

Resolution

The creation of a foreign Retail Site is currently not supported. Please:

  1. Review SAP Note 3435102 - SAP S/4HANA Cloud 2508: Retail Release Restriction Note for details on the restriction.
  2. Refer to SAP Help documentation (SAP Help - Foreign Plant) for information on Foreign Plant and Foreign Shipping Point functionality.
  3. Vote for the improvement request 356175 on SAP Influence to advocate for the development of this feature.
