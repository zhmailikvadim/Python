Symptom

Running KE27 for the affected CO-PA line items returns error MLCCS031. Error text observed: "No valuation exists for material ... in period ...". The issue occurs specifically when activating the indictor "Periodic valuation" in KE27.

Environment

SAP ERP

Reproducing the Issue

  1. Open transaction KE27.
  2. Enable the "Periodic valuation" indicator and run the valuation.
  3. Observe that KE27 stops with error MLCCS031 and the COGS value does not update in KE24.

Cause

When "Periodic valuation" is selected in KE27, the system uses the CO-PA item’s goods issue date (WADAT, WADAT itself gets filled by VBRP-FBUDA) as the valuation period; if that period has no Material Ledger actual costing data, KE27 raises error MLCCS031.

Resolution

Run KE27 without the "Periodic valuation" indicator to valuate using the posting period.

See Also

Refer to: application help for the "Periodic valuation" indicator in KE27.
