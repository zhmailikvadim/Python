Symptom

  • While acting in proxy, you are not able to download reports in Report Center
  • Download only works if the proxy permission “Reports in Classic View” is granted

Environment

SAP SuccessFactors HCM Suite

Reproducing the Issue

  1. Go to Proxy Management.
  2. Grant proxy rights so that user A can proxy as user B.
  3. Add the proxy permission for Reporting > Report Schedules.
  4. Set start and end date for the proxy relationship and save.
  5. Proxy as user B.
  6. Go to Report Center > View Schedules.
  7. In a visible report schedule, open the three dots menu and select Download.
  8. Observe that nothing happens.
  9. Return to Proxy Management and additionally grant the proxy permission “Reports in Classic View.”
  10. Repeat steps 5–7.
  11. Download now works as expected.

Cause

The scheduled reports involved are of the Classic type. By design, Classic reports require the “Reports in Classic View” proxy permission for download actions when operating in proxy mode.

Resolution

In Proxy Management, add the proxy permission “Reports in Classic View” for the proxy user that needs to download Classic reports.

After saving the proxy assignment, test it by proxying to the target user and downloading the scheduled report from Report Center.
