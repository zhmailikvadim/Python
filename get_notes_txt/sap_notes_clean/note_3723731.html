Symptom

This SAP Note generates the required DDIC elements (and other bits) in your system before implementing SAP Note 3723731 and is only relevant to customers on SAP_HR Release 608, S4HCMCIE 100, S4HCMCIE 101 and S4HCMCIE 102.

Other Terms

UDO, DDIC Elements of SAP Note 3723731, NOTE_3723732, SAP_HR Release 608, S4HCMCIE 100, S4HCMCIE 101 and S4HCMCIE 102

Reason and Prerequisites

DDIC Elements of SAP Note 3723731, SAP_HR Release 608, S4HCMCIE 100, S4HCMCIE 101 and S4HCMCIE 102.

Solution

This SAP Note generates the required DDIC elements in your system before implementing SAP Note 3723731. But how this SAP Note is applied is dependent on your SAP_BASIS release.

The outcome of applying this SAP Note (and running the UDO program it creates) is dependent on your SAP_HR release and your SAP_BASIS release.

SAP_HR Release 608, S4HCMCIE 100, S4HCMCIE 101 and S4HCMCIE 102:

SAP_BASIS 702 SP18 (or higher)
SAP_BASIS 731 SP18 (or higher)
SAP_BASIS 740 SP15 (or higher)
SAP_BASIS 750 SP02 (or higher)
All Higher Releases of SAP_BASIS

Apply the aCI DDIC correction instructions in this SAP Note and the DDIC objects included in this SAP Note will be created automatically.

SAP_HR Release 608, S4HCMCIE 100, S4HCMCIE 101 and S4HCMCIE 102:

SAP_BASIS 702 SP17 (or lower)
SAP_BASIS 731 SP17 (or lower)
SAP_BASIS 740 SP14 (or lower)
SAP_BASIS 750 SP01
All Lower Releases of SAP_BASIS

Apply the correction instructions of this SAP Note, via SNOTE, to create the UDO program NOTE_3723732. Run the newly created report (NOTE_3723732) in Update mode, so that all the DDIC changes for SAP Note 3723731 are created and activated.
