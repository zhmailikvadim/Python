Symptom

You implement SAP Note 3583627 - Brazil Tax Reform: Alphanumeric CNPJ - Adjustment in BR1 Checks, but still unable to use the Alphanumeric CNPJ in BP transaction.

Environment

  • Localization Brazil
  • Materials Management (MM)
  • Sales and Distribution (SD)
  • Business Partner (BP)
  • Logistics.

Cause

Workaround of SAP Note 3583627 - Brazil Tax Reform: Alphanumeric CNPJ - Adjustment in BR1 Checks not performed.

Resolution

  1. Start transaction OY17.
  2. Select Brazil (BR) and locate the entry for Tax Number 1.
  3. Set Length to 14 characters.
  4. Set Checking Rule to 1, 3, or 5 to allow alphanumeric entries.
  5. Save the configuration.
  6. Retest in BP by entering an alphanumeric CNPJ.
