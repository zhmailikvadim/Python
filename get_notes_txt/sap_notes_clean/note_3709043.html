Symptom

In a Remote Conversion process, you run Generate Programs for Data Migration, Conversion Activity with the following error message:

  • Generate Conversion Program /1LT/BW900XXX_XXXXXXXXX for Table XXXXXXXXX
    • Syntax error in generation program.
    • Message: Type "E" is unknown.

Other Terms

Remote Conversion; Remote Migration; Remote Transfer; RSB4HCONV; CNVMBTPCT;

Reason and Prerequisites

The system does not process the DF16_DEC and DF34_DEC data types correctly, which results in the syntax error in the generated program.

Solution

From now on, the system handles the DF16_DEC and DF34_DEC data types correctly and defines decfloat16 and decfloat34 as the corresponding types.

Apply this SAP Note in both Sender and Receiver systems.
