Symptom

A primary key column cannot be nullable. In the SAP Web IDE Data Target node, the user can select the nullable checkbox on primary key columns. The nullable checkbox should be disabled if its a primary key column.

Other Terms

SAP Web IDE, Data Target, Table Types, Template Table, Primary Key, Nullable

Environment

SAP Web IDE for HANA 2.0 SPS03 Feature Revision 01

Steps to Reproduce

  1. Create a new flowgraph in SAP Web IDE
  2. Add Data Source with primary key
  3. Connect to any nodes with a Data Target
  4. Configure Data Target - either Template table or Table Type
  5. User can select nullable for primary key columns - This should be disabled

Solution

This correction is delivered in the release(s) listed in the Support Packages and Patches section of this SAP Note.
