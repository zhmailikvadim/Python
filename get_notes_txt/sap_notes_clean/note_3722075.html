Symptom

In PCW (FSH_PCW), when a Combined Master Production Order is opened in Change Mode, its linked Production Orders are not locked. This issue allows other users to modify those orders simultaneously, causing potential data conflicts.

Other Terms

PCW, Production Control Workbench, Lock issues, Combined Master Production Order, Master Production Order, Production Order

Reason and Prerequisites

Reason 

This problem is caused by a program error.

Solution

Implement the correction instruction provided with this SAP Note.
