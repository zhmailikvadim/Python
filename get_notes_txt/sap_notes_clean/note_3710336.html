Symptom

OpenAPI: Event management API returns 10 digit decimal places on Money fields

Resolution

Event Management API is currently designed to retrieve data from the database in its existing form, which defaults to 10 decimal digits.

We suggest submitting an enhancement request to better align with your business needs. For more detailed guidance on how to proceed with this request, please refer to article KB0397628.
