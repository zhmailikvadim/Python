Symptom

In SAP Business One, Web client, irrelevant fields appear in journal entry templates.

Reproducing the issue

Prerequisites:

You are working in the Ukraine localization.

Scenario:

In SAP Business One, Web client, go to Administration → Configuration → Accounting → Journal Entry Templates.

Expected behavior:

  • In the list view, choose Adapt Filters, and you find that Tax Code does not appear in the Adapt Filters window.
  • In the list view, choose the View Settings icon, and you find that Tax Code does not appear in the View Settings window.
  • In the detailed view of a journal entry template, the Automatic Tax and Deferred Tax checkboxes do not appear on the General tab.
  • In the detailed view of a journal entry template, choose the Settings icon on the Contents tab, and you find that Tax Posting Account, Tax Code, Tax Jurisdiction Type, and Tax Jurisdiction Code do not appear in the Select Columns window.

Actual behavior:

  • In the list view, choose Adapt Filters, and you find that Tax Code appears in the Adapt Filters window.
  • In the list view, choose the View Settings icon, and you find that Tax Code appears in the View Settings window.
  • In the detailed view of a journal entry template, the Automatic Tax and Deferred Tax checkboxes appear on the General tab.
  • In the detailed view of a journal entry template, choose the Settings icon on the Contents tab, and you find that Tax Posting Account, Tax Code, Tax Jurisdiction Type, and Tax Jurisdiction Code appear in the Select Columns window.

Solution

SAP intends to provide a feature package or patch to solve the symptom that is described in this SAP Note. The section References below lists the related feature package or patch once they become available. Be aware that References can only be confirmed at feature package or patch release date.
