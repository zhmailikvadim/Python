Symptom

Field CustomerStatisticsGroup (KNVV - VERSG) is not available in Odata API_BUSINESS_PARTNER 

Environment

  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Resolution

The data source for the Customer Statistics Group field is available starting from S4CORE109. To consume this field, system must be on S4CORE109 / S4HANA2025.
