Symptom

You would like to know the current available Schedule eDocument Jobs for the Country Greece.

Environment

SAP S/4HANA Cloud Public Edition

Cause

Currently the following Schedule eDocument Jobs are available in the SAP S/4HANA Cloud Public Edition:

eDocument (General): Processing eDocuments in the Background
By using the eDocument (General): Process Actions in the Background job, you can run specific process actions on selected eDocuments in the background automatically.

eDocument (General): Handling Incoming Messages
By using the eDocument (General): Handle Incoming Messages job, you can retrieve and process incoming messages (XML) related to eDocument process statuses from your application server or from external systems.
 
eDocument (General): Generating Summary of eDocuments
By using the eDocument (General): Generate Summary for eDocuments job, you can generate an eDocument that contains a summary of eDocuments for one company code.
 
eDocument (General): Setting eDocument Status to Completed
By using the eDocument (General): Setting eDocument Status to Completed job, you can change the statuses In process (yellow icon in the eDocument Cockpit) or Action required (red icon in the eDocument Cockpit) to Completed (green icon in the eDocument Cockpit) automatically.

Resolution

The Help Portal Schedule eDocument Jobs - General Documentation describes each Schedule eDocument Jobs with further details.
