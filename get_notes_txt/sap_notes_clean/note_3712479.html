Symptom

  • Transaction WRF_DIS_SEL displays a misleading message stating "Materials selected have been flagged for discontinuation" when errors exist on the article.
  • The check light turns green, indicating the article is flagged as discontinued, but the article is not actually flagged as discontinued in the tables, nor is the valid-to date set.

Environment

  • Retail
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Go to Transaction WRF_DIS_SEL.
  2. Specify the article and execute.
  3. Verify that errors exist on the article.
  4. Select the "Discontinue" option on the menu bar to attempt to discontinue the material.
  5. Observe the message "Materials selected have been flagged for discontinuation" and the check light turning green.
  6. Verify that the article is not actually flagged as discontinued, as tables are not updated and the valid-to date is not set.

Cause

Since it is possible to select multiple articles in the WRF_DIS_SEL t-code, the message "WRF_DIS_SEL" is only a general message.

If there is already an error message shown in the work list, the discontinuation will not be executed and the error will be shown in the inbox.

Resolution

Standard behavior.
