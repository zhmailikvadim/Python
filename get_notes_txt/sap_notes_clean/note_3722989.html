Symptom

In the process "Sending of 55653 from Responsible MOS to SUP" (DE_B_MDC_55653_MOS2SUP_SEND), the meter reading card group information is missing in step 1100 when sending the 55653 message to the supplier.

This issue is relevant for all market roles.

Other Terms

MDC_55653_MOS2SUP, meter reading card, feed-in scenario, /UCOM/IF_BADI_POD_DATA_ACCESS->GET_POD_MALO, /US4G/CL_IM_BADI_POD_DATA_ACC, GET_POD_MALO

Reason and Prerequisites

The error occurred because the system did not retrieve the market location POD for generation/feed-in scenarios.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the system retrieves the feed-in POD when only a feed-in POD exists in the master data in API /US4G/CL_IM_BADI_POD_DATA_ACC->GET_POD_MALO. In the BPEM case, select the *Refetch Data* option. You can verify the fix by checking that the system sends the 55653 message to the supplier in the cloud with the complete meter reading card group information.
