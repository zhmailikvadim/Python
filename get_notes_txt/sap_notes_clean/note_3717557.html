Symptom

Serial number (SERNR) field in Asset Balance report (RABEST_ALV01) is not as long as in table ANLA.

The field SERNR is expected to have a length of 18 characters, consistent with the asset master definition; however, it is currently defined with a length of 8 characters.

Environment

  • Financial Accounting (FI)
  • Asset Accounting (FI-AA)
  • SAP R/3
  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA Finance
  • SAP S/4HANA 

Cause

Incorrect data element type defined for SERNR in data type FIAA_SALVTAB_RABEST.

In table ANLA, the data element type of the field SERNR is defined as AM_SERNR. However, in the structure definition FIAA_SALVTAB_RABEST, the data element type assigned to SERNR is SERNR.

Resolution

Change the data element type in FIAA_SALVTAB_RABEST from SERNR to AM_SERNR in transaction SE11.

Correct configuration (transaction SE11)

> data type: FIAA_SALVTAB_RABEST > Field SERNR

  • Data element        AM_SERNR
  • No. of Characters   18
  • Output Length      18
