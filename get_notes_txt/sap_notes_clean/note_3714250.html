Symptom

When you save a confirmation using single-screen entry, the system checks whether the automatic goods movements require documentation batch management.
If an automatic batch determination takes place in EWM for material components to be backflushed during the confirmation, batches are not found even though they are available.

Other Terms

CO11N, single-screen entry, confirmation, production order
COR6N, single-screen entry, confirmation, process order
Documentary batch, OMCDB, material overview
Batch Determination EWM, Extended Warehouse Management

Reason and Prerequisites

A temporary lock is not reset.

The error occurs only if the "Documentary Batch" function is active in the system and if batch determination takes place automatically in EWM during the confirmation. The problem does not occur if the goods movement overview is already used before saving.

Solution

Implement the source code corrections in accordance with the correction instructions.
