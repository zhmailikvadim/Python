Symptom

When auxiliary package material information is sent from LE to TM , the Package Material Type is not being transferred. As a result, the Package Material Type remains blank in the details tap of the auxiliary package materials in the TM UI, even though this information exists in the master data.

Other Terms

LE-TM Integration; Logistic Integration

Reason and Prerequisites

Because of a programming error the relevant field was not mapped.

Solution

Implement the correction instruction attached to this Note or install the corresponding support package. Afterwards, the relevant field is mapped and the Package Material Type shown on the UI.
