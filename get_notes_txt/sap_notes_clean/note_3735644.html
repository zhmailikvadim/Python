Symptom

When planning to update SAP S/4HANA 2025 to FPS01 in the maintenance planner, SAP Fiori front-end server 2025 for SAP S/4HANA Support Package Stack 01 is not visible. 

Environment

  • Product: SAP S/4HANA 2025
  • Maintenance Planning Tools

Reproducing the Issue

  1. Open Maintenance Planner and create a plan for an update to S/4HANA 2025 FPS01.
  2. Add SAP Fiori front-end server 2025 for SAP S/4HANA to the planned stack.
  3. In the support package selection, observe that SP01 is not offered for SAP Fiori front-end server 2025 for SAP S/4HANA.

Cause

  • It is not planned to provide support package stacks for SAP Fiori front-end server 2025 for SAP S/4HANA beyond the initial shipment. Maintenance Planner calculates the SAP_UI 8.16 level according to the underlying S/4HANA/ABAP Platform stack.

Resolution

  1. Acknowledge that the absence of SP01 for SAP Fiori front-end server 2025 for SAP S/4HANA5 is expected behavior and not a Maintenance Planner defect, as no separate SP stacks are planned beyond the initial shipment.
  2. If a higher SAP_UI 8.16 level is required, plan and implement the corresponding S/4HANA 2025/ABAP Platform support package upgrade; Maintenance Planner will then calculate the SAP_UI level accordingly.
  3. Regenerate the maintenance plan focusing on the S/4HANA/ABAP Platform support package level to achieve the desired SAP_UI level.
  4. Refer to the official maintenance strategy and product-specific note for detailed guidance.
