Symptom

New Assets are created in SAP S/4 HANA and are pushed to SAP Ariba via Managed Gateway for Spend&Network (formerly CIG), however these new Assets are not available in Ariba

 

Cause

The events are not binding after being put in a queue because internally it checks the provided Check function module in event linkage.

Resolution

Remove the Check function module entry in event linkages and test the integration

Keywords

Asset Master Data, ARBCIG_MASTER_DATA_EXPORT
