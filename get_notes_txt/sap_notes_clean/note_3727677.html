Symptom

A user is not receiving requisition approval emails despite being active, added to the approval flow, and is the active approver.  Is there a specific group that may be causing this?

Resolution

Check if the user is in the group called, Requisition Approval Emails Disabled User.  Any user in this group will not receive requisition approval emails regardless of active status or email notification settings.

To verify, navigate to:

  • Core Administration
  • User Manager
  • Users
  • Search for the affected user
  • Click the All Groups tab

If the group is listed, they will need to be removed from this group to receive requisition approval notifications.
