Symptom

  • The report RP_HRSFEC_ExportPayrollResults in Employee Central Payroll (ECP) executes successfully but does not transfer payroll results for the current payroll period to Employee Central.
  • No error message is displayed during the report execution.

Environment

  • SAP SuccessFactors Employee Central Payroll

Reproducing the Issue

  1. Open transaction SE38.
  2. Execute report RP_HRSFEC_ExportPayrollResults.
  3. Use the variant or fill data manually
  4. Select Automated Transfer (OData).
  5. Execute the report.

Cause

In the "Manage Data" tool, when the "Search As Of" field is set, the system displays records that are effective as of the specified future date.

These records are referred to as future effective-dated records. 

Resolution

Adjust the date of the "Search As Of" for the EffectiveStartDate found in the ODATA Upsert xml file.
