Symptom

Entire oracle database is having poor job performance. Jobs are running slower than expected, and overall database response times are degraded.

 

Environment

Oracle database

Cause

Poor job performance on a database can be caused by:
- Underlying storage/disk I/O issues

Resolution

  • Generate and review AWR report 
  • Check File I/O Statistics in AWR Report
  • Average Read Time (Av Rd(ms)) is greater than 10 ms
  • Engage with disk/storage vendor to analyze the performance of the underlying hardware. Issues such as slow disks, contention in the storage array, or configuration problems could be causing these delays.

Keywords

I/O Statistics, AWR, Av Rd(ms), Performance, Oracle, Hardware, Slow, Degraded
