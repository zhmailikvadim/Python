Symptom

When running activity "Check primary and unique keys of tables to be transferred" the activitiy aborted with an error message and a short dump.

Other Terms

downtime minimization tool for SAP S/4HANA

Reason and Prerequisites

NZDT 1.0 SP01

Solution

Implement the automatic correction instructions on your sender, central and receiver system.

To take effect, this note has to be implemented before activity NZTR1_CHK_UIDX_PIDX_DLTA_TAB_R.
