Symptom

The following error message is shown in the payroll calculation log:

EN: No entry in table T7PL31 for key <space> for time <date>
PL: Brak wpisu w tabeli T7PL31 dla argumentu <spacja> w czasie <data>

The error occurs when for the calendar month of the payment date there is no record with the insurance title 01xx in the infotype 0515, and there are at least two records in the infotype 0413. This happens most typically in the next month after the end of a regular employment.

Other Terms

PLGEN PDHC, IT0413, PY-3326143.

Reason and Prerequisites

Error in the correction released with SAP Note 3326143, point 2, in the payroll function PLGEN PDHC.

Solution

Payroll function PLGEN PDHC has been corrected.

To activate this change, update the value of the old activation switch called PY-3326143 in table T7PL13 from value '1' to '2'.
Apply the correction by importing the Support Package specified for your release, or by using the Note Assistant.
