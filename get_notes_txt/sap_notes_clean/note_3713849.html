Symptom

  • The Execute Changes option is unavailable for Assignment Profile [ID], even though there are pending curriculum updates.
  • As the changes cannot be executed, the relevant curricula are not correctly assigned to the user.

Environment

SAP SuccessFactors Learning

Reproducing the Issue

  1. Login to Learning Administration
  2. Navigate to Assignment Profile
  3. Open the impacted Assignment Profile ID
  4. Execute Changes option is greyed out

Cause

Due to an internal calculation and dependent background job, curriculum additions or deletions may not be picked up by the system cache. This causes curricula to remain in Pending Addition or Pending Deletion status, preventing the Execute Changes button from becoming available.

Resolution

Remove the affected curricula from the Assignment Profile and add them again. After re-adding the curricula, the system correctly recalculates the pending changes and the Execute Changes button becomes available. Once enabled, execute the changes to successfully assign the curricula to the user.

Keywords

Execute Changes greyed out, Execute Changes disabled, Pending Addition, Pending Deletion. Curricula not assigned
