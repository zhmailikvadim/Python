Symptom

Note 3731276 (component MOB-APP-MAO-FND) contains updates to the MAIF Cloud Metrics classes on which this note (component MOB-APP-MAO-ERP) depends on. Please ensure note 3731276 is implemented in your system before implementing this note.

This note includes the following updates to the App Create transaction:

  1. Client Role - Used by the Metrics report to determine user counts (i.e., 0 user counts sent for non-productive systems). Defaulted to System Client Role defined in transaction SCC4. Stored in MAIF Product Table.
  2. Satellite System - Satellite Systems can be setup to avoid duplicate User Counts when multiple Production Systems are in use. Metrics from a Satellite Sytem will be retrieved via the Metrics Report executed in the Host System ensuring duplicate SAP Users are only counted once.
  3. Host RFC Destination - Used to create the Satellite Middleware Server in the Host System that will be used to retrieve the Metrics from the Satellite System.

Other Terms

SAP Service and Asset Manager, /MERP/CPMS_APPCREATE, /MERP/CORE_CPMS_APPCREATE_PROG

Reason and Prerequisites

Feature. Note 3731276 is a prequisite.

Solution

Please implement note.
