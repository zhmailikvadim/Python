Symptom

All users do not see the Latest Growth Portfolio UI even though Latest Career Worksheet is enabled.

Environment

SAP SuccessFactors Talent Intelligent Hub

Reproducing the Issue

  1. Confirm that both the Latest Career Worksheet and Opportunity Marketplace are enabled for the instance.
  2. Cross check on the system and some users are able to view the Latest Growth Portfolio UI whereas the other users are not able to view the Latest Growth Portfolio UI with Roles and Aspirations section.

Cause

In order for users to access and view the new landing page, they must have the "Opportunities Marketplace Access" permission enabled in Role-Based Permissions (RBP). Without this permission, the new landing page will not be visible.

Resolution

Ensure that the affected users have the "Opportunities Marketplace Access" permission enabled in Role-Based Permissions (RBP).

Kindly please note that an enhancement is planned for the 1H 2026 release to introduce a universal Growth Portfolio landing page feature, which will address this dependency.
