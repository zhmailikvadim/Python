Symptom

When deleting a VAS (Value Added Service) entry for a sales document item using API FSH_API_SALESDOCUMENT_CHANGE, the system correctly removes the VAS relevancy indicator(VBAP - FSH_VAS_REL) from the item. However, when a new VAS entry is subsequently added again for the same item using the same API, the system does not set the VAS relevancy indicator back on the item.

Other Terms

VAS, Value Added Service, VAS redetermination, VBAP - FSH_VAS_REL

Reason and Prerequisites
