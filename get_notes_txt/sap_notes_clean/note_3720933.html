Symptom

In the SAP SuccessFactors instance, the option "Admin Center → Security Center → Manage OIDC Auth Client Application" is missing and not visible.

Environment

SAP SuccessFactors HCM Suite

Reproducing the Issue

  1. Navigate to Admin Center → Security Center in the SAP SuccessFactors instance.
  2. Attempt to locate the option "Manage OIDC Auth Client Application".
  3. Observe that the option is not visible.

Cause

Prerequisites not followed.

Resolution

  1. Refer to the Authentication Using OpenID Connect (OIDC) guide for implementation details.
  2. Navigate to the step 2 section of the guide, which points to the Register Your Own Application to Communicate with SAP SuccessFactors HCM Suite with OpenID Connect chapter.
  3. Verify and ensure all prerequisites mentioned in the prerequisites section are met.
