Symptom

***WAITING FOR VALIDATION, if proven, change status to Released to Customer and remove this statement***

In CJ20N, the following dump is thrown when creating a new activity.

Runtime Errors         COMPUTE_INT_PLUS_OVERFLOW
Except.                CX_SY_OPEN_SQL_DB
ABAP Program           SAPLBSVU

Environment

  • SAP ERP Project System (PS)
  • SAP R/3
  • SAP R/3 Enterprise
  • SAP ERP Central Component
  • SAP ERP
  • SAP Enhancement package for SAP ERP
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA, on-premise
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. CJ20N
  2. Create Activity
  3. Dump

Resolution

Implement the following SAP Note: 49257 - Reference date for internal date format

Keywords

COMPUTE_INT_PLUS_OVERFLOW, CX_SY_ARITHMETIC_OVERFLOW, SAPLCXTP, Integer overflow while adding (type I), point_in_time = g_cxtp_days * c_seconds_per_day + g_cxtp_time.
