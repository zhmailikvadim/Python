Symptom

The system throws the error "Inspected sample size &1 & planned sample size &2 is not equal for IC" on completion of OA/SFI, though the inspected sample size is correct.

The issue is observed for serialized orders having more than 1 quantity.

Other Terms

PEO, Inspection Characteristics, IC, Sample Size, Operation Activity, Serialized

Reason and Prerequisites

Program Error.

Solution

Implement this SAP Note for correction.
