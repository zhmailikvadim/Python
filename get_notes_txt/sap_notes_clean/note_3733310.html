Symptom

 Download not working correctly from OPER for XLSX and CSV format

Other Terms

Download, XLSX,

Reason and Prerequisites

This is caused by program error.

if you are in 2022 V2 Base Package, 2023 V2, Please implement the latest version of SAP Note # 3704610 and # 3588142 as pre requisite notes.

Solution

Please implement the correction instructions attached. These corrections are only valid if you have applied the Support Packs.

Release V4 - 2022 V2 Base Package, 2023 V2, 2023 V3 Base Package or Support Pack 1.

Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object.

***V4 2025 Bug 219583: [Internal] Downloads are not working properly in S4B system.*                  IF sy-subrc EQ 0 AND <lwa_sectn_fields> IS ASSIGNED.                  IF sy-subrc EQ 0.                    APPEND INITIAL LINE TO <lwa_sectn>-fields ASSIGNING <lwa_sectn_fields>.                    CHECK <lwa_sectn_fields> IS ASSIGNED.***
