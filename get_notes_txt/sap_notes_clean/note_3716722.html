Symptom

  • Validation or system errors during Adjustment Item creation in Financial Control Adjustment Hub trigger Message Popover in bottom-left corner of wizard screen.
  • Users see high-level error messages but lack direct access to detailed technical logs.

Other Terms

  • Application Log Integration
  • FC-Adjustment Hub
  • Message Popover Navigation
  • Cross-App Navigation to F1487 (Application Logs app)

Reason and Prerequisites

Reason

  • Missing seamless integration between Adjustment Hub errors and standard SAP Fiori Application Logs app (F1487), slowing issue resolution.
  • Errors not linking directly to deep-dive logs, requiring manual navigation.

Prerequisite

  • SAP Fiori Application Logs app (F1487) configured and active in Fiori Launchpad (FLP). SAP Fiori Apps Reference Library
  • Users assigned technical/business roles for Application Logs access.
  • Intent-Based Navigation (Semantic Object/Action) for Application Logs maintained in FLP Designer.
  • Object and Sub-Object values maintained in table for the category (refer to SAP Fiori Apps Reference Library for details).

Solution
