Symptom

Unable to add Mentee group in both Supervised and Unsupervised Mentoring program , gives error "You have added 0 new mentees"

**Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental**

Environment

  • SAP SuccessFactors Mentoring
  • SAP SuccessFactors Succession and Development

Reproducing the Issue

  1. Navigate to Manage Mentoring Programs >> open any supervised or unsupervised mentoring program 
  2. Go to the relevant section to add mentees by group.
  3. Select any group
  4. Observe that the system does not allow the mentee group to be added and givers error "You have added 0 new mentees".

       

Cause

  • The issue is caused by a known defect in the 2H2025 release.

Resolution

  • Engineering confirmed this is a known issue from the 2H2025 release, and the fix is planned for the 1H2026 release. For release timelines, please review SAP SuccessFactors Product Release & Road Map Information.
  • As a workaround, engineering can manually add the mentee group to the program upon request.
  • User can add by group as expected before launching the program (under draft status).However, please note that before 1H2026, You cannot add by group for both mentor and mentee after launching it (for confirmed status) unless you add participant one by one 
  • we advise to add participants before launching the program ,while creating the mentoring program itself.
