Symptom

  • Todate Function does not work in Web Intelligence when changing the Preferred Viewing Locale to Japanese
  • #Error is diaplying for Todate() function when viewing WebI report by Japanese Preferred Viewing Locale

Other Terms

WebI, PVL, Function, feature, TODATE,

Reason and Prerequisites

Environment:

  • SAP BusinessObjects Business Intelligence Platform (BI) 4.3 Support Package  04 (SP04) Patch 14
  • SAP BusinessObjects Business Intelligence Platform (BI) 4.3 Support Package  05 (SP05) Patch 3
  • Web Intelligence (WebI)

Steps To Reproduce:

  1. Create WebI and define column like below.
    =ToDate("12/15/02 11:00:00";"INPUT_DATE_TIME")
    =ToDate("12/15/02 11:00:00";"MM/dd/yyyy")
  2. Logon BI launch Pad, set the Preferred Viewing Location to English.
  3. The Todate() function works fine.
  4. Set the Preferred Viewing Location to Japanese.
  5. Logoff and logon againt.
  6. View the Report.
    Note that the Todate function return #Error.
