Symptom

For FC 10.1 SP09 Patch levels prior to Patch 25, the minimum supported SAP HANA version is 2.0 SPS05. However, starting from FC 10.1 SP09 Patch 25 and higher, SAP HANA 2.0 SPS07 or higher is required as the minimum supported version.

Environment

FC 10.1 SP09

HANA 2.0

Resolution

Update your SAP Financial Consolidation server installation to a support package, patch or release where the issue is fixed.

See the Support Packages & Patches section for details and available patches.
