Symptom

You execute a Legacy Data Transfer. In the valuation area dependent data you have specified an amount in the field FX Valuation of Amortized Acquisition Value in Valuation Currency or the field FX Valuation of Acquisition Value in Valuation Currency. Even though you have set the Valuation with Reset indicator, the system does not generate a corresponding reset flow.

Other Terms

Position component 1011, position component 1030, RESET_VAL

Reason and Prerequisites

Program error.

Solution

Implement the attached Correction Instruction (CI) or the relevant Support Package.
