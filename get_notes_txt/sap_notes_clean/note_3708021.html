Symptom

  • The performance analysis tool did not capture the story access record which is based on live data connection to BW.
  • The access record by support user can be recorded, but the user access records are not recorded.

Environment

SAP Analytics Cloud (Enterprise)

Reproducing the Issue

  1. Log in to SAP Analytics Cloud.
  2. Open the Performance Analysis Tool and select the story.
  3. Set the date range for look back.
  4. Observe that only the support user accessed records can be found, while the other users' access records are missing.

Cause

The issue is caused by a defect in the product.

Resolution

This issue is fixed in version 2026 QRC1.
