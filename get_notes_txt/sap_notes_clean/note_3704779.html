Symptom

Add conditional suppress formula to a line, make sure the formula returns "false" (do not suppress), find it's always suppressed when export to pdf in Crystal Reports for Eclipse JAVA

Other Terms

Crystal Reports for Eclipse, JAVA, suppress, formula

Reason and Prerequisites

1. Add conditional "Suppress" formula for a line object in Crystal Reports designer, make sure the formula returns "false" (do not suppress);

2. Check the "Suppress" formula works as expected in Crystal Reports designer.

3. Export report to pdf in CR for Eclipse JAVA;

4. Find the line is missing. While it should be visible according to the formula.

Solution

This issue is fixed in SAP Crystal Reports, version for Eclipse SP32.

To download the latest Support Packages of SAP Crystal Reports, version for Eclipse, visit:

https://wiki.scn.sap.com/wiki/display/BOBJ/SAP+Crystal+Reports+version+for+Eclipse+-+Downloads
