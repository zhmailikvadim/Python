Symptom

Improper translation of 'To' (field RP50M-ENDDA) in program 'MP055200' screen '3000' into Japanese.

Other Terms

PA30, Infotype 0552; Program MP055200 Screen number 3000; RP50M-ENDDA

Reason and Prerequisites

 Text length is too short to display any appropriate Japanese texts.

Solution

In the Screen Painter Attributes of field 'RP50M-ENDDA' in program MP055200, screen 3000 the value of field 'Modify' has been changed to 'F' (according to MP055200, screen 2000), so that the text can be translated properly.  
This is delivered with the next Synchronization HRSP available in November 2023.
