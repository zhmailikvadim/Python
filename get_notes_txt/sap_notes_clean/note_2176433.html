Symptom

Write comments in any language for a Firefighter to which Firefighter ID it has been assigned to. The text will not be available for users who log in with another language.

Login with English language and write comment:




Login with German language, the text will not appear:

 

Environment

GRC Access Control 10.0 and 10.1 - Emergency Access Management (EAM)

Reproducing the Issue

  1. Run NWBC.
  2. Go to Emergency Access Maintenance and select option Firefighters.
  3. Assign FF ID to Firefighter and write comment to it.
  4. In case you are logged in with English language, the comment will not appear by logging in with another language.

Cause

This text field is language dependent, therefore users who log in with another language will not see the comment.

Resolution

Logon language used while adding the comments should be used everytime in order to view the comments. Or comments can be maintained in multiple languages after logon to the application in desired language.

Keywords

firefighter, firefighter id, comment, another language
