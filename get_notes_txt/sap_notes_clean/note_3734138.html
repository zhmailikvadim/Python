Symptom

In the LDC Workcenter, the Vendor field F4 Search Help is active for Released LDC, even though the field is disabled for manual input. This allows users to select the value and potentially lead to data inconsistent.

Other Terms

Agricultural Contract Management; LO-AGR-LDC; Vendor; Data inconsistency

Reason and Prerequisites

While manual input was disabled for "Released" status, the F4 Search Help was not suppressed, allowing users to select and update values.

Solution

Before implementing this note make sure note 3710558 is applied in system. then, apply automatic correction instructions from this SAP Note.
