Symptom

User have the following queries regarding data‑privacy protection for Joule, specifically related to the handling of user data.
  • Joule Base automatically applies privacy controls to user data,
  • Additional administration steps are required to activate privacy features, or
  • Specific privacy settings must be configured for the standard Joule agents provided by SAP.

Environment

  • SAP BTP, Cloud Foundry runtime and environment
  • Joule

Resolution

  • The Data Anonymization feature anonymizes text and text files based on your selected method for detected entity instances. The Data Pseudonymization feature is a reversible anonymization process that detects and pseudonymizes instances of entities based on your selected anonymization method.
  • User also receive a metadata report linking pseudonymized entities to the real entities. Available through the SAP Data Privacy Integration Service with Enterprise plan.
  • This AI offering is included with Joule Base at no additional cost. A subscription to Joule Base is required to use it.
  • To use this AI feature, the corresponding base product must be available and active in the user's system landscape.
  • For detailed technical information and resources for setting up the service, please check SAP Data Privacy Integration, Data Anonymization.
