Symptom

Within trailer unit, railcar unit or container unit you open the cargo table, select a cargo item and in the detail area you select tab 'Discrepancies and Events'. When trying to create or display a note or an attachment via link, error message APB_FPM_CORE112 'No valid target area found' is displayed.

Reason and Prerequisites

Program Error.

Solution

Implement the coding changes as described in the attached correction instruction or install the relevant support package.
