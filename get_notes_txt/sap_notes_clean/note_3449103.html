Symptom

In application Define Selections you want to set selections for your master data. By filtering for an already defined selection via the Smart Filterbar the filter "Selection with Single Field" is not considered.  Changing values of it does not change the behavior.

Other Terms

F3725, Fiori Launchpad, Master Data, Define Selections

Reason and Prerequisites

Due to a programming error the filter value was not recognized correctly.

Solution

Implement the correction instruction of this note or install the corresponding support package.
