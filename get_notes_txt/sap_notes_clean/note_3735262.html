Symptom

  • Add a new destination for Adaptive Job Server.
  • Logon to BI Launchpad and see this destination option is not added to the list for Send To option.

Environment

SAP Business Intelligence (BusinessObjects) Enterprise 2025

Reproducing the Issue

  1. Logon to Central Management Console (CMC) and navigate to servers.
  2. Find Adaptive Job Server, add a new destination. e.g. SFTP destination.
  3. Logon to BI Launchpad and find any document.
  4. Click 3 dots > send to.
  5. Cannot find this SFTP destination option just added.

Cause

  • It's necessary to reload Tomcat to make this option showing in Send To list.
  • The issue mostly happens after BI version upgraded to BI 2025 Patch X.

Resolution

The option will show after restarting Tomcat.

  • Windows Server: logon to Central Configuration Manager (CCM) and restart Tomcat
  • Linux Server: execute ./tomcatshutdown.sh and ./tomcatstartup.sh
