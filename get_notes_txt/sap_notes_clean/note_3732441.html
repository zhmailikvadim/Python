Symptom

  • When trying to change externalCode of pre delivered custom MDF object, following error is received - "You are not allowed to change the value of attribute Maximum Length for the field externalCode of this object"
  • Unable to change the externalCode of pre delivered custom MDF objects  

Environment

SAP SuccessFactors Onboarding

Reproducing the Issue

  1. Go to "Configure Object Definition Page"
  2. Search for pre delivered custom objects (ONB2CustomTaskData000, ONB2CustomTaskData001, ONB2CustomTaskData002 and so on)
  3. Click on Take Action -> Make Correction
  4. Try to Change the externalCode of the object, following error is received - "You are not allowed to change the value of attribute Maximum Length for the field externalCode of this object"

Cause

Expected Behavior

Resolution

This is currently an expected behavior. The external code changes are not allowed for pre delivered objects.
