Symptom

After migrating the application from Neo environment to SAP BTP Cloud Foundry (CF) environment due to Neo environment sunset, there is a requirement to redirect users from the old Neo application URL to the new Cloud Foundry application URL

Environment

Cloud Portal service on NEO environment

Reproducing the Issue

  1. Migrate the application from Neo to CF Environment.
  2. Attempt to redirect the old Neo URL to the new CF URL.

Resolution

  • URL redirection from the Neo environment to the CF environment is not supported.
  • The old Neo URL remains valid until 2028 unless the service is extended or manually disabled.
