Symptom

While Submitting Greece Classification non-deductable NAV credit invoice to myData system throws following Rejection error:

Error 332: "In case of VAT exemption sum of E3 classifications' amounts and sum of VAT classifications' vat amounts must be equal to invoice total net value plus taxes to be classified plus excluded vat amount."

Image/data in this KBA is from SAP internal systems, sample data, or demo systems and are for reference purposes only.  

Environment

  • SAP ERP
  • SAP S/4HANA
  • SAP Document and Reporting Compliance service
  • SAP Document and Reporting Compliance, cloud edition

Reproducing the Issue

  1. Launch EDOC_COCKPIT transaction.
  2. Enter the Classification eBook source key
  3. Submit it
  4. Display XML
  5. Check Application/Interface Logs

Cause

Incorrect classification amount for non-deductable credit invoice lines with Greece eBooks Classification process.

Resolution

To fix the issue, Implement the Below SAP Note to correct the Classification amount for NAV Non-Deductable Credit Scenarios:
 3654037 - Electronic Document Processing for Greece: Mapping Enablement and Code Correction XLVIII 

Please make sure you have already implemented all the latest notes as per sequence in overview note 0002930400.
