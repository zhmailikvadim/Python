Symptom

Australia pain.001.001.06 CGI Credit Transfer (AU_CGI_XML_CT) payment medium format is not accepted by bank. The bank accepts pain.001.001.03 version only. This format is currently not supported.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

Resolution

  1. Create a copy of the delivered AU_CGI_XML_CT format tree in the payment format mapping in Config Activity 100558 'Map Format Data for Payments' (use the delivered tree as a template).
  2. Edit the copied format and change the ISO constant from pain.001.001.06 to pain.001.001.03.
  3. Assign and use the custom AU_CGI_XML_CT variant for payment runs, then validate the generated XML with the bank’s requirements.
