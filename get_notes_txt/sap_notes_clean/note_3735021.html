Symptom

You download an operational report in both .XLSX and .CSV formats and expect the key and description field labels to match those configured in the reporting view. However, the downloaded files display field labels that do not align with the reporting view configuration.

This note corrects the behavior by ensuring that the key and description field labels in the downloaded reports match the reporting view configuration.

Other Terms

/VTA/CL_BSTMNT_SERVICES, Downloading OPER report, XLSX and .CSV format, Reporting View Descriptions

Reason and Prerequisites

This is caused by Program error.

Solution

Please implement the correction instructions. These corrections are only valid if you have applied the following Support Packs.

Release V/4 – 2022 V2, 2023, 2023 V2, 2023 V3, and 2023 V3 Support Package 1

Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object
