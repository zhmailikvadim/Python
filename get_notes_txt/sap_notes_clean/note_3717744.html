Symptom

A requestor creating supplier request gets the error below while entering an email address.

Error : Please enter in a valid email format

Cause

The email address value being entered is not meeting the validation pattern configured for the question in the supplier request template.

Resolution

Update the validation pattern in the Supplier Request template for email related question so that it can accommodate various email domains you have for the suppliers.
