Symptom

There has a requirement to generated XML data by CPM.

This article introduce the procedure about how to do it.

Environment

S/4HANA Public cloud.

Resolution

  1. Create a print queue with format XML in the "Maintain Print Queues" app.
  2. Set the option "Save to File" option in CPM, and set up the directory.

Further details how to setup the print queue and CPM can be found here .
