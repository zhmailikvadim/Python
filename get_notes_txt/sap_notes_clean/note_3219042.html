Symptom

You work with My Inbox application and have created a Scenario Specific tile. After the initial loading of My Inbox the Тask list is empty. If you open the Filter dialog and select the OK button, the tasks are loaded into the Task list.

Other Terms

Fiori, Launchpad

Reason and Prerequisites

Program error

Solution

Apply the referenced SAP Note corresponding to your My Inbox Support Package. Then clear the SMICM cache as described in SAP Note "2319491 - How to clean up the cache after applying changes that affect SAP Fiori apps", and clear the end-user browser cache.
