Symptom

When the KPI Object/PIC Analysis Query name is too long, during runtime the configuration button is not visible. Also the custom time period is not shown competely in the UI.

Other Terms

MII SSCE , SSCE KPI Object, PIC Analysis query

Reason and Prerequisites

The style properties were not added correctly, due to which the configuration button was not displayed.

Solution

The solution is provided in the Patch mentioned in Support Packages & Patches
