Symptom

This SAP Note provides you with the relevant system changes to allow the new field Indicativo de cumprimento parcial do aviso prévio in the Rescisão de contrato trabalho (BR) (0661) infotype.

Other Terms

Infotype 0661; Rescisão; Termination; Dismissal

Reason and Prerequisites

You have implemented SAP Note 2499842.

Solution

The new field in the infotype allows the following values:

  • 0 - Cumprimento total
  • 1 - Cumpr. parcial em razão obtenção novo emprego pelo empregado
  • 2 - Cumprimento parcial por iniciativa do empregador
  • 3 - Outras hipóteses de cumprimento parcial do aviso prévio
  • 4 - Aviso prévio indenizado ou não exigível

The changes described in this SAP note will be included in an HR Support Package, as indicated in the item "Reference to Support Packages". 
 
The Support Package includes the following objects:

  • Changes in structure PS0661
  • Changes in the screen 2000 of the module pool MP066100

As a general rule, SAP recommends that you install a solution by applying a Support Package or CLC. However, if you need to install a solution earlier, use the Note Assistant to implement the correction instructions. You can find more information about the Note Assistant under https://support.sap.com/kb-incidents/notes/assistant.html.
