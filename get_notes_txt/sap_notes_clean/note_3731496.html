Symptom

  • Edit a dataflow, inside a Query -> filter tab, there is severity error like:" Replication table <XXX> has an invalid filter expression: Unknown identifier".
  • Filters used in the Query has no logic or syntax error. Filter design is correct.

Environment

SAP Cloud Integration for data services

Reproducing the Issue

Cause

Resolution

  1. Open the affected dataflow and navigate to the filter where the error appears.
  2. Make a temporary, non-functional edit to the filter expression (for example, add and remove a space or similar harmless change), then save the dataflow.
  3. Revert the temporary change to restore the original filter expression and save the dataflow again.
  4. Reopen or refresh the filter and verify that the error message no longer appears.
