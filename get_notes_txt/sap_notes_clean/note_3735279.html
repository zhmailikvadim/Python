Symptom

  • Goal ratings and goal details are missing from Detailed Document Search when the latest performance form experience is enabled.
  • When the latest experience is disabled (legacy view), goal ratings and goal details appear as expected.

Environment

  • SAP SuccessFactors Performance Management
  • SAP SuccessFactors Goal Management

Reproducing the Issue

  1. Enable the latest performance form experience.
  2. Go to Report Center > Classic View > Detailed Document Search.
  3. Search for a completed performance form and generate the report.
  4. Open the form and review the goals section: goal ratings and goal details are not displayed.
  5. Disable the latest performance form experience and repeat the steps to see that goal ratings and details are displayed.

Cause

In the latest performance form detail page, goal access permission checks are enhanced and enforced.

Resolution

To view the goals in performance form or in detailed document report , respective user should have the access to user's goal plan and read access to all the goal fields.

If the goals are private, we have to ensure the private access permissions are also granted to the user who is viewing the form or generating the report.

You can add F role to resolve the above scenario.

For more information on F role, check KBA: 2077256 - Manage Templates - Goals not showing on PM form but comments are - Explanation on F Role Permission - Performance Management
