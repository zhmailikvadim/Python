Symptom

During legacy asset data transfer (AS91/ABLDT), values entered in company code currency are not converted into group currency in parallel depreciation areas (e.g. areas with currency type 30).

Environment

  • FIN Asset Accounting
  • SAP ERP
  • SAP ERP Central Component
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA
  • SAP S/4HANA Finance
  • SAP S/4HANA Cloud Private Edition
  • SAP Fiori for SAP S/4HANA Finance,
  • SAP Fiori for SAP S/4HANA,
  • SAP Fiori for SAP S/4HANA Cloud Private Edition

Cause

Legacy asset transfer transactions (AS91/ABLDT) are data takeover processes and not valuating postings.
Therefore, no automatic currency conversion is performed between depreciation areas.

Resolution

This is standard system behavior.

To ensure correct values in group currency areas:

  • Provide values explicitly for each depreciation area during migration, or
  • Perform subsequent valuation processes (e.g. depreciation run) to derive values

Currency conversion in Asset Accounting is only triggered by valuating transactions (e.g. asset postings or depreciation runs).
