Symptom

Intervals can be specified for the creation date or the change date of IDoc´s in different ALE reports, or transactions. Also multiple selections can be made.
However, if selection options for individual values are chosen, not all expected IDocs are selected.
If, for example all IDocs up to a certain change date are to be selected and just the selection in the form "< dd.mm.yyyy" is used, only the IDocs for the specified date are selected.

Environment

  • SAP Basis ALE
  • SAP NetWeaver
  • SAP Web Application Server for SAP S/4 HANA
  • ABAP PLATFORM - Application Server ABAP

Reproducing the Issue

Maintain in the date field a selection in the form "< dd.mm.yyyy".

Cause

Date has been entered in a format, which is not sufficient.

Resolution

If IDoc´s for a period are to be chosen, both date fields 'From' / 'To' are to be filled with a date. Detailed explanation and sample is delivered also in SAP Note 791185.
For the transactions WE02 / WE05 / WE09 a correction has been delivered with SAP Note 994135.
