Symptom

If you try to execute a TagStore query for an OPC-HDA agent to update a tag at a timestamp, where already a value exists, then you will get the error "value already exists".

Other Terms

PCo, MII, InsertReplace

Reason and Prerequisites

Program error

Solution

Download the installation program for SAP Plant Connectivity 15.0 with Support Package 05 Patch 10 from SAP Service Marketplace and install it in accordance with the instructions in the installation guide.

Pay particular attention to the following points before you start the installation:

  • In case of an upgrade installation, make sure that all agent instances have been stopped and that the Management Console is closed. After starting the installation, the system issues a warning message if there are still any running PCo applications that may impair the upgrade installation process.
  • If the installation program indicates a pending restart of the system, perform the restart before you continue with the installation of PCo.
