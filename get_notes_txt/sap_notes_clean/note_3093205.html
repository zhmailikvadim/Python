Symptom

This note is only relevant for IDXAT3 customers.

In case the distributor receives a consumption reqeust(Message: ABLEHNUNG_PT) which contains a wrong period, e.g. the DATE_FROM later than DATE_TO.

A rejection message with response code 82(Prozessdatum falsch) will be sent.

Other Terms

IDXAT2, 2522, CR_REQ_PT, Step 200, /IDXAT3/CA_CR_PT, /IDXAT3/CG_CR_PT_02, /IDXAT3/_192, VALID_DATE, INVALID_DATE, Rejection, Response Code, 82, Prozessdatum falsch

Reason and Prerequisites

 Program Error

Solution

1. Please import and activate the attached BC-Set;

2. Please implement the correction instrcution in this note.
