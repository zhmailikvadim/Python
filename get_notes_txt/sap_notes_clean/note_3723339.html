Symptom

This SAP Note provides you with corrections to the Peru Inventory and Balance Ledger (PE_INVENTORY_BALANCE_LEDGER) report in the Run Statutory Reports app for SAP Document and Reporting Compliance. It addresses missing journal entries in the report output, which have open items but are associated with a reversed vendor invoice.

When you run the PE_INVENTORY_BALANCE_LEDGER report, the system does not consider journal entries with open items that are associated with a reversed vendor invoice. As a result, the balances in the report are incorrect.

Other Terms

Peru, Document and Reporting Compliance, DRC, fka ACR, Inventory and Balance Ledger, Libro de Inventarios y Balances, Material Management, Supplier Invoice, Reversal

Reason and Prerequisites

This SAP Note is relevant only for Peru.

You have implemented previous notes listed in SAP Note 3206609.

Solution

After implementing this SAP Note, the system includes journal entries with open items that are associated with reversed vendor invoice in the report output. As a result, the balances in the PE_INVENTORY_BALANCE_LEDGER report are correct.

SAP recommends that you install a solution by applying a Support Package. However, if you need to install a solution earlier, use the Note Assistant to implement the correction instruction. You can find more information about the SAP Note Assistant in the SAP Support Portal, under SAP Note Assistant.

To implement the solution, you must carry out manual activities and implement the correction instructions that are valid for your release.
