Symptom

Environment

  • SAP S/4HANA 2025 FPS0
  • SAP S/4HANA Cloud Private Edition 2025 FPS0

Cause

Incorrect step sequence.

Resolution

There are following workarounds for current issue, please choose one of options:

Option 1: Add LR -> F8 copy control header via T-code VTFL manually refer to below screenshot, then continue best practice solution activation in /SMB/BBI.

Option 2: In /SMB/BBI, skip this step with error, re-activate this step with error after scope item BDD activation is completed.
