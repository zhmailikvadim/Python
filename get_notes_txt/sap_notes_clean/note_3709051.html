Symptom

  • When attempting to create a proxy assignment, the system displays one of the following error messages upon saving:

    • “Inactive username entered for the account holder”

    • “Inactive username entered for the proxy”

  • This occurs even though both users are active in the system

Environment

SAP SuccessFactors HCM Suite

Reproducing the Issue

  1. Navigate to proxy management.
  2. Specify the user who will act as the proxy.
  3. Specify the account holder the proxy will act on behalf of.
  4. Select the Modules.
  5. Enter the Start Date and the End Date.
  6. Click on Save.
  7. Observe the error message: "Inactive username entered for the account holder" or "Inactive username entered for the proxy"

Cause

There is an extra space character (leading or trailing) after the username of either:

  • the account holder, or

  • the proxy user.

This causes the system to interpret the username as invalid or inactive.

Resolution

  1. Re-enter the username for the account holder and/or proxy user.

  2. Ensure there are no leading or trailing spaces.

  3. Click Save again.

  4. The proxy assignment will be saved successfully, and the error message will no longer appear.
