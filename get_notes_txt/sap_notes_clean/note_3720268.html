Symptom

After the upgrade to BI 2025, the LDAP module fails to load with error message "The secLdap plugin failed to verify the server administration credentials. " 

Environment

  • SAP BusinessObjects Business Intelligence Platform 2025
  • Windows

Reproducing the Issue

  1. After upgrade to BI 2025
  2. Login to CMC
  3. Application >Authentication >LDAP
  4. Click on LDAP Authentication 
  5. Prior to the configuration page appearing
  6. Error message comes up "The secLdap plugin failed to verify the server administration credentials. " 

Cause

  • secLDAP_Pin..xml Dfo and secLDAP.xml Dfo are corrupted 

Resolution

  1. In <INSTAL DIR>/SAP Business object/ SAP Business object Enterprise XI 4.0 /dfo, find corrupted secLDAP_Pin.xml and secLDAP.xml DFO
  2.  Perform cleanup of this corrupted DFO's
  3. Refer this attached KBA for DFO cleanup
    Clean up default objects (DFOs) from the CMS repository [How-To]
  4. Restart the SIA (server intelligence agent) 

Note: Ensure that you have taken backup of the CMS database and FRS (filestore) before performing the DFO cleanup.
