Symptom

A piece of Equipment is created in a specific language i.e. EN (English). When accessing F2072 in a different language i.e. DE (German) and searching for this Technical Object, the Technical Object Description column is BLANK. The Technical Object column will also show the technical name only and no description.

Environment

  • SAP ERP Plant Maintenance (PM)
  • SAP S/4HANA, on-premise
  • SAP S/4HANA, Cloud Private Edition

Reproducing the Issue

  1. Logon in EN
  2. Open F2072
  3. Search for a piece of Equipment (Technical Object) where the Description is maintained in EN
  4. The Technical Object Description is displayed
  5. Switch to a different language i.e. DE
  6. Search for the same Technical Object
  7. The Technical Object Description is BLANK

Cause

  • The Technical Object Description will only be displayed if it has been maintained for the Logon Language
  • You can check Table EQUI_TEXT to see what language(s) the Equipment is maintained
  • If it is maintained for multiple languages i.e. EN and DE, the Technical Object Description will be displayed when you access the App in either EN or DE

Resolution

EAM Apps such as F2072 do not support the Primary Logon Language. The Equipment needs to be maintained in different languages so that this information will be displayed when using the App. This is the expected behaviour.

See Also

  • SAP Note 3513720 was previosuly created to remedy this but the solution was not feasible and was reverted with SAP Note 3534520
  • A correctly working solution would require a re-development of the existing functionality and hence, if there is a business requirement, this can be requested via a SAP customer influence request (SAP Note 11)
