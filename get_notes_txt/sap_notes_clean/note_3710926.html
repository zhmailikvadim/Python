Symptom

  • After an upgrade, the system is unable to create invoices through a specific process when a Business Partner has multiple addresses configured.
  • Error message: "Enter a valid Address Number".

Environment

  • SAP S/4HANA
  • SAP S/4HANA 2023
  • SAP S/4HANA Cloud Private Edition
  • SAP S/4HANA Cloud Private Edition 2023

Reproducing the Issue

Process an IDoc to create invoices.

Cause

One of the switches for BP Adoption - multiple addresses is switched on and there is a logic error.

Resolution

Implement SAP Note 3539648.

See Also

SAP Note 3539648 - Performance improvement for S/4HANA BP Adoption - multiple addresses
