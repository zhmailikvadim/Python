Symptom

Customer is attempting to subscribe to SAP Build Apps and Standard Plan using Booster.

Environment

  • SAP Build Apps
  • Mobile and Visual Cloud Function options

Reproducing the Issue

  1. Run the booster to set up SAP build code/build apps.
  2. Open the build lobby and create or open a project.
  3. Navigate to the mobile category/option.
  4. Observe the status "Not Subscribed".

Cause

The mobile option is only available with the Standard Plan, which has been deprecated; it is not included under the Consumption (CPEA) model.

Resolution

As part of the continued evolution and simplification of the SAP Build portfolio, SAP Build Apps is being deprecated as a standalone product. Moving forward, customers can leverage SAP Build unified offering, that combines application development, process automation, and business site creation into a single platform that supports low-code, code-first, and generative AI capabilities. Please see SAP Build Apps Deprecation.
