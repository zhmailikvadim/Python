Symptom

When you change the material group in purchase requisition of item category “B” Limit, the automatic G/L account derivation is not carried out again.

Environment

Materials Management (MM)
SAP ERP Central Component
SAP ERP
SAP enhancement package for SAP ERP
SAP enhancement package for SAP ERP, version for SAP HANA
SAP S/4HANA

Reproducing the Issue

1) Create purchase requisition through ME51N with item category B. 

2) Change PR item material group through ME52N.

3) Check G/L account.

Cause

Only Limit value changes and item description changes triggers a new account determination. 

Resolution

In standard SAP, If you change the material group in a created purchase requisition item of the category “B” Limit, the automatic G/L account derivation is not carried out again.
For a purchase requisition item of category B (Limit) the system uses the account-determination logic when the PR line is created, but it does not automatically re-run that derivation if you later change the material group.
Only when you change the Limit value and item description of line item it will trigger a new account determination. Material group change do not trigger new account determination.
