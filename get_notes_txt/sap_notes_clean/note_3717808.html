Symptom

MRP generates purchase requisition. When user informs fixed supplier, system still asks for purchasing organization as a mandatory field, even when having field selection key as optional. 

Environment

  • Materials Management (MM)
  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA

Reproducing the Issue

  1. Launch MRP through MD01N for a single material with a PIR.
  2. View in MD04 purchase requisition created and edit it.
  3. inform fixed supplier and save. It asks for purchasing organization

Cause

The purchasing organization field is mandatory during MRP PR creation as standard design.

Resolution

  • Maintain the default Purchasing organization in transaction OMKI for plant and create a Source list via ME01 for the material, assigning the plant and Purchasing organization as valid source.
  • to restrict the Purchasing organization for document type, use the BADI MD_CHANGE_MRP_DATA (change MRP data before PR creation) or MD_PURREQ_CHANGE (change data before PR post).

Keywords

Purchasing organization, MRP, PR, MD04, MD01N, OMKI
