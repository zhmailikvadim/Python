Symptom

The FM: FVD_IOA_UI_DETAILS_DISPLAY currently when called allows only displaying the IOA details in an ALV format but there is not option to suppress this and just return the data that will be displayed in the ALV.

Other Terms

FVD_IOA_UI_DETAILS_DISPLAY; IOA;

Solution

Implement the correction instructions.

With this note, new parameters are added - I_FLG_GET_DISPLAY and E_TAB_IOA_DETAIL.

If the I_FLG_GET_DISPLAY is set, then, the system suppresses ALV display and sends the IOA details back using the parameter E_TAB_IOA_DETAIL.
