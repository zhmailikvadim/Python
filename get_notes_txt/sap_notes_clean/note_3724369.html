Symptom

A custom button is added to transaction N/AIF/ERR in a namespace not related to Central Finance. Using this button displays the error message 'The selected AIF interface does not support this function' instead of the expected result from the custom process.

Environment

  • Financial Accounting (FI)
  • SAP S/4HANA
  • SAP S/4HANA Finance
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

1. Add a custom button to AIF Error Handling (transaction N/AIF/ERR) and implement custom logic related to the button
2. Click on custom button to execute custom logic under a namespace that is not related to Central Finance (/FINCF)
3. Error message 'The selected AIF interface does not support this function' is displayed

Cause

program error

Resolution

Implement SAP Note 3682922 - Issue with Custom "Skip Date Check" Button where error is displayed properly.

Keywords

AIF, BAdI, /AIF/V1_ACT, interface, namespace
