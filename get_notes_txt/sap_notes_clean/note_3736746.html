Symptom

When you go to SLG1 transaction with object SHCM_EE_INTEGRATION, you still can find past several years logs.

Environment

S/4 HANA OP with HR integration active status

Reproducing the Issue

  1. Go to transaction SGL1
  2. Execution with object SHCM_EE_INTEGRATION
  3. You still can find past several years logs of relevant employee

Resolution

After apply following notes:

3219365 - BF: HRALX LOGDY Implementation
3099471 - HRALX LOGDY Implementation

customer can control how long logs will be shown in SLG1 via switch HRALX LOGDY

This Switch HRALX LOGDY give the possibility to set the days the application log is protected against deletion as desired between 1 and 365 days, or if the Switch is set to 0/BLANK until 31.12.9999. 

You can go to transaction SM30 to create the switch under table T77S0 and create the relevant return value from 0/BLANK to 365, what customer need as customer own business requirement.
