Symptom

This KBA explains how to generate an English version EA form.

Environment

  • Human Capital Management (HCM)
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP SuccessFactors Employee Central Payroll
  • SAP S/4 HANA

Cause

As the Malaysian version is the one required for submission to the government, currently the system does not provide the English version.

Resolution

Prerequisite: The software Adobe LiveCycle Designer must be installed in the system.

  1. Run T-Code: SFP, input the standard form name for example HR_MY_TAXEA_2022, select the Form Object and click Copy or Press CTRL+F5.
  2. Input the custom form name for example ZEA_EN in "Copy To", create the custom form, save and activate.
  3. Change the form ZEA_EN, click Fullscreen Layout or press CTRL+F12, change labels from Malay into English.
  4. Run T-Code: SM30, edit view V_T799BSFT, find group HR_MY_EAPF, and change HR_MY_TAXEA_2022 to ZEA_EN. 
Note: This is a global configuration and impacts all employees.
