Symptom

System does not create an automatic purchase requisition for Item Category Group type "Expense items (SRVE)".

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Open App "Manage Service Order"
  2. Create a Service Order.
  3. Add item with Item Category Group Expense items (SRVE).
  4. Set Service Order item status to released and save.
  5. Select the Released Service Order
  6. Navigate to Transaction History
  7. Observe that no purchase requisition is created automatically.

Cause

This is standard system behavior.

Resolution

The automatic purchase requisition creation is supported only for procurement-relevant items, such as Service Part Items or Externally procured service items, and not for expense items like SRVE.
