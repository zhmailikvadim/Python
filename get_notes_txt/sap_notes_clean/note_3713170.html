Symptom

Due to the certificate renewal, an update was performed on the SAP API Management custom domain. As a post-processing step, redeployment of the API Management artifacts is mandatory

Environment

  • SAP API Management
  • SAP Integration Suite

Resolution

The API Proxy must always be deployed first, followed by product publication. In a self-service approach, data must be synchronized between design time and runtime, therefore, redeployment is required

Keywords

API Management, APIM, Integration Suite, Virtual Host, Custom Domain, VirtualHostRequests, Redeployment, Certificate renewal
