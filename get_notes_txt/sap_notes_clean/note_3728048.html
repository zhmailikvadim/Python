Symptom

A set of hierarchies was created in Manage Global Hierarchies, but only one hierarchy is visible in reconciliation cases.

Environment

SAP S/4HANA Cloud Public Edition

Cause

Resolution

  1. Copy the existing consolidation unit default hierarchy and edit it according to the required specifications.
  2. For creating a new hierarchy, note that the issue is under investigation and will be resolved in the May 2026.
  3. Refer to the Knowledge Base Article (KBA) for updates on the resolution timeline.
