Symptom

Not all categories are being displayed in the "Browse by category" filter

Environment

SAP SuccessFactors Learning

Cause

The Engineering team confirmed that categories must have a description in order to be displayed in the Browse By Category Dropdown. 

Resolution

Access Learning Administrations > Categories > Any the text in the "Description" field in the categories which are not displayed in the dropdown and save.

Keywords

category, dropdown, display, description, missing, filter
