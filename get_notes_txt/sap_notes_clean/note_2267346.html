Symptom

Symptom

The outbound processing of ORDERS / ORDCHG SIDocs is aborted if table YH050 does not contain entries for one or more profit centers of the purchase order items.

Other Terms

CIP2SAP SIDoc, IDoc, TTP1443, CR 1443, /SIE/SF_ED_SER_FIBU, GET_DEP_STRUC_4_PRCTR,  yyordit-dep_str

Reason and Prerequisites

Reason:
Program error in note 2234640 for TTP1443: During processing of EDI messages (SIDocs) the depth structure of a profit center shall be read from CIP2SAP customizing table YH050.

Prerequisite:
The issue occurs only if note 2234640 for TTP1443 is implemented.

Solution

Implement the correction instruction.

The correction from this SAP Note resets the input-readiness of the cells for %XT or %YT in the aforementioned situation.

Example:

The query contains the characteristics 'product range', 'product group', and 'product' in the rows, as well as the key figure structure with the elements 'turnover' and '%YT turnover'. The universal display hierarchy is used in the rows (in the Query Designer, this is the setting 'Display as Hierarchy'. The characteristics and structure are used in the following sequence in the rows: 'Product range', 'key figure structure', 'product group', 'product'. No totals are displayed in the result list for 'product range', but these are required for the inverse formula calculation with regard to %YT. In this case, the input-readiness of %YT in the key figure structure is reset.

Other Terms

Input-ready query, inverse formulas, %XT, %YT, universal display hierarchy, UHRY, PROCESS_EXEC_PLAN-03, PROCESS_EXEC_PLAN, CL_RSR_RRK0_PLAN
