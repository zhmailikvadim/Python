Symptom

  • After Support Pack (SP) Upgrade to WEBCUIF 748 SPS 19, when logging in to Web UI and clicking on any field a dashed outline appears.
  • This behavior was not happening before upgrade.


*DISCLAIMER: Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental. 

 

Environment

  • SAP Customer Relationship Management (CRM)
  • SAP enhancement package for SAP CRM
  • SAP enhancement package for SAP CRM, version for SAP HANA
  • SAP Solution Manager
  • Web UI.

Reproducing the Issue

  1. Login to Web UI.
  2. Click on any field, Dashed outline appears.

 

Cause

Its a new feature with WEBCUIF 748 Support Pack 19.

Resolution

The dashed focus is added to comply with the accessibility guideline where the focus must be visible to all users, therefore it can't be removed.

If User role is not using the standard skin there might be a reason why they do not get dashed focus.
