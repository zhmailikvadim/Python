Symptom

During Route Assembly, the error message 'Delivery &1 is not completely processed' (LMD_ROUTE_HUB/084) is displayed.
The error message indicates that the delivery has not been picked yet (overall status for picking in the outbound delivery).

For a route that should be assembled with lifecycle status 'In Execution',
the error message is raised appropriately to prevent the execution of follow-up processes before deliveries have been picked (e.g., LMD Settlement follow-up process).

For a route that should be assembled with lifecycle status 'Open' (Early Route Assembly to preview the route, e.g., for Freight Order Status 'Ready For Execution'),
the error message is also raised but is not required, since follow-up processes are not allowed in the system when the route has the lifecycle status 'Open'.
This note will resolve the issue: The error message will not be displayed for a route that should be assembled early with lifecycle status 'Open'.

Other Terms

LMD - Last Mile Distribution
