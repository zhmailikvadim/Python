Symptom

You find that the text for the delivery header is greyed out and you are not able to edit it in the Change Outbound Delivery app.

Environment

SAP S/4HANA Cloud Public Edition

Cause

The text in the delivery header is greyed-out and cannot be edited after posting the goods movement of the delivery. This is the standard system behavior. 

Resolution

Edit the text before posting the goods movement.

Keywords

Text, Posting the goods movement, Change Outbound Delivery, Greyed-out, Cannot edit
