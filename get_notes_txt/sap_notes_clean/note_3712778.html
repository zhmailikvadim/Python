Symptom

You want to activate XOPVW and XLGCLR for multiple G/L accounts together, using:

  • transaction code FAGL_ACTIVATE_OP (report FAGL_SWITCH_TO_OPEN_ITEM) or
  • transaction code FINS_ACTIVATE_OIM (report FINS_SWITCH_TO_OPEN_ITEM) or
  • report FINS_SWITCH_XLGCLR.

Environment

  • SAP R/3
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA Finance
  • SAP S/4HANA

Reproducing the Issue

  1. Go to the transaction code FAGL_ACTIVATE_OP (report FAGL_SWITCH_TO_OPEN_ITEM) or the transaction code FINS_ACTIVATE_OIM (report FINS_SWITCH_TO_OPEN_ITEM) or the report FINS_SWITCH_XLGCLR.
  2. Try to enter multiple G/L accounts.

Resolution

It is not possible to activate XOPVW or XLGCLR for multiple G/L accounts together. The activation can be done for one G/L account at a time, for multiple company codes. This is mentioned in the SAP Notes 2745769 and 2894095.
