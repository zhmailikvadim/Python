Symptom

Version 1.0.2 of SAF-T (BG) Monthly File is needed for upload to NRA

Other Terms

SAF-T Bulgaria monthly file, Bulgaria, BG, SAF-T, Tax Reporting, Audit, Standard Audit File, Legal Change, SAFT, Monthly File

Reason and Prerequisites

This SAP Note delivers structure of XML file version 1.0.2. It is only relevant for Bulgaria.

Prerequisites for Release 618:

  • implement note 3730119

Prerequisites for Release 110_604:

  • implement note 3730820
  • then implement note 3730119

Solution

As a general rule, SAP recommends that you install a solution by applying a Support Package. However, if you need to install a solution earlier, use the SAP Note Assistant to implement the correction instruction. You can find more information about the SAP Note Assistant in SAP Service Marketplace, under service.sap.com/note-assistant.SAP

Note: Check and install any newer notes released in component FI-LOC-SAF-BG.

​​​​​​​For more information about the development delivered with this SAP Note, see the following What's New topic on SAP Help Portal:

  • SAP ERP: Electronic Document Processing for Standard Audit File - Taxation (SAF-T) 

Recommendation: If you configured file splitting for version 1.0.1, perform the same configuration for version 1.0.2 (transaction SM30, maintain customizing view SAFTNV_FS_TRANS)
