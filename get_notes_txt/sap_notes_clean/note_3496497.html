Symptom

To remove a service from HANA DB, we can use alter statement and port number is possible to be requested:
ALTER DATABASE my_database REMOVE '<service>' AT LOCATION '<hostname>:<port>';
----------------------------------------------------------------------------
Refer to note:
3050423 - How to remove a service in SAP HANA


Environment

SAP HANA database.

Resolution

Following SQL can be used to check the port number of doctore:
select * from SYS.M_SERVICES where service_name = 'docstore'; 

Also you can log on to HANA STUDIO and open Landscape tab, look into Services tab, there is the list of all services and docstore can be found, port number can be confirmed via port column.
