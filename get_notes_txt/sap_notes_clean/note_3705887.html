Symptom

When an administrator is on the Manage Permission Groups page and opens Spotlight, the user profile visualization changes significantly.

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

 

Environment

SAP SuccessFactors HCM Suite

Reproducing the Issue

  1. Login to instance.
  2. Navigate to the Manage Permission Groups page.
  3. Open a user profile.
  4. Observe the differences in the user profile visualization when compared to profiles accessed from other landing pages.

Cause

This is a Known Issue.

Resolution

The fix for this issue is planned to be deployed on 1H 2026 release.

For release timelines, please review SAP SuccessFactors Product Release & Road Map Information.
