Symptom

With release S/4 HANA 2023 new event codes were introduced with the new event processing as per SAP Note 3485221.

Two issues can be observed.

1. You could not choose these new event codes when:

     - Reporting events from the document UI

     - Choosing the Last Expected Event in the document type customizing.

2. If a new event code is reported, on the document UI in the execution event detail, the technical event code is displayed instead of the description.

The above listed points can cause confusion in event reporting. This SAP Note provides a correction for the display of the events, but does not change the functionality.

Other Terms

EP*

Reason and Prerequisites

Program error

Solution

 Implement the changes as described in the attached correction instruction or install the relevant support package
