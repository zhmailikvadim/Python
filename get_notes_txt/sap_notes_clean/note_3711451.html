Symptom

SAP Note 3711472 (eDocument Brazil Inbound NF-e: Adjustments in Pending Documents report) requires this SAP Note to be implemented as a prerequisite.

Other Terms

eDocument, eDoc, eInvoicing, e-Invoicing, electronic invoice, electronic invoicing, e-document, e-Document, e-Docs, e-Doc, e-docs, e-doc, Brazil, Brazil eInvoice, Inbound, NF-e, Nota Fiscal Eletronica, Nota Fiscal Eletrônica, Electronic Nota Fiscal, Pending Documents, EDOC_BR_PENDING_DOC..

Reason and Prerequisites

This SAP Note contains a report and automatic correction instructions to create/modify objects that are used in the solution delivered via SAP Note 3711472.

Solution

Use the SNOTE transaction to implement the attached correction instructions before you implement primary SAP Note 3711472.

VersionDateReason
01 February 05, 2026 Released for Customer

As a general rule, SAP recommends that you implement the Support Package specified for your release. However, if you need to install a solution earlier, use the SAP Note Assistant to implement the correction instruction. You can find more information about the SAP Note Assistant in SAP Support Portal, under SAP Note Assistant.
