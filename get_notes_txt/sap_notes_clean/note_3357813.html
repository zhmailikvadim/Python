Symptom

You use Transaction Manager of Treasury and Risk Management. You try to create a new position in the system for a security ID with product type index-linked bond (04L). The position is handled with a position management procedure where Standard Book Value option is set for Valuation Base Amount. The valuation steps are executed in the following order: index valuation, forex exchange valuation and security valuation. In some cases the system might raise the error message TPM_TRL 063 Processing leads to inconsistent positions for case REPO.

Other Terms

TS01, FTR_CREATE, index-linked bonds, 04L, TPM_TRL_PROFIT_AND_LOSS_INDEX

Reason and Prerequisites

Program error

Solution

Implement the correction instructions or import the corresponding Support Package.
