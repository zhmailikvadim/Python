Symptom

Launching an interactive report returns the warning message

Specify reference currency; default is EURO

Environment

  • SAP Customer Relationship Management (CRM)
  • SAP enhancement package for SAP CRM
  • SAP enhancement package for SAP CRM, version for SAP HANA

Reproducing the Issue

  1. Open the CRM UI and choose Reporting
  2. Choose a report with underlying queries '/CRMBW/SVRQ_C01_Q0001' and '/CRMBW/CSAL_C01_Q0002'
  3. Receive the Warning message 'Specify reference currency, default is EURO'

Cause

This kind of message is only a warning and it means that some or all service requests do not have the currency into the header overview and by default the system gets it from the organization unit which is assigned to EUR currency.

Resolution

  1. Check if you have maintained the reference currency to the organization unit.
  2. If not, please assign the user to the same organization and check further.
  3. Interactive reporting is user specific, based on the Org unit assignments, currency will be considered.
