Symptom

No default implementation of the BAdI definition /PE1/BADI_RCL_ITEM exists.

Other Terms

/PE1/BADI_RCL_ITEM, /PE1/BADIIMPL_RCL_ITEM, /PE1/BADIEI_RCL_ITEM.

Reason and Prerequisites

Fallback class /PE1/CL_DEF_IM_BADI_RCL_ITEM of BAdI definition /PE1/BADI_RCL_ITEM was final.

Solution

Create a new default BAdI implementation /PE1/BADIIMPL_RCL_ITEM with implementation class /PE1/CL_DEF_IM_BADI_RCL_ITEM and activate it.
