Symptom

On status change of the task "Validation of Reported Data" on a group level, the status of units for its successive tasks in extension versions does not change accordingly.

For example, you have task "Validation of Reported Data" and its successive task "Currency Translation" both blocked for a consolidation group "P".

On unblocking the validation task on "P", it should unblock the currency translation task in reference and extension versions for the cons group "P" and all of its units.

But you observe that the unblocking is done only for reference version, not for extension version for the consolidation units.

Other Terms

CACTI 1080, CACTT 03, Validation of Reported Data, Data Monitor, Status management, extension version, automatic run

Solution

Import of Support Package or advanced correction by implementation of correction instruction.
