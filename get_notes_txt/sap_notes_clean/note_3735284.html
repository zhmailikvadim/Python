Symptom

When opening Embedded Analytics Configuration, a warning appears stating: “You are not authenticated. Close window to proceed to login”, preventing access to the configuration page.

Environment

  • SAP Sales Performance Management
  • Embedded Analytics
  • IAS

Reproducing the Issue

  1. Log in to the Incentive Management Portal.
  2. Navigate to and click Embedded Analytics Configuration.
  3. Observe the warning message: “You are not authenticated. Close window to proceed to login.”

Cause

  • The Identity Authentication technical user SAP_TUSER (or EmbeddedAnalytics_TENANTID_ReadOnly_System_USER ) used by Embedded Analytics Configuration was deleted and later recreated without the required read permission.
  • In addition, the recreated user generated a new client ID/secret that was not updated in the application, leading to authentication failures.

Resolution

  1. Log in to IAS (Identity Authentication Service).
  2. Go to Users & Authorizations.
  3. Select Administrators.
  4. Locate the technical user SAP_TUSER (or EmbeddedAnalytics_TENANTID_ReadOnly_System_USER).
  5. If the user is missing or was deleted:
    • Recreate the user with the same permissions as before (especially the required read permission), and
    • Open a Support case.
  6. Support must update the new Client ID and Client Secret on the application side.
  7. Provide the new Client ID and Client Secret to Support via email (the assigned processor will request them) for security purposes. (Do Not Share the Client Secret in the Case)
