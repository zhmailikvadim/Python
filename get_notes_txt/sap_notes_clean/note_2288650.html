Symptom

You change report variants with a large number of filter settings in SAP Environmental Compliance 3.0, and you save the changes. You find the following error message in the logs: Trying to recycle a non deleted record for INSERT INTO EM_REP_RESTR_MAP.

 

Other Terms

﻿xEM, EC, SAP EC, EP, SAP EP, EEM﻿

﻿Emissionsmanagement, Einhaltung von Umweltschutzauflagen, Umweltleistung, Energiemanagement, Berichtsvariante﻿

﻿Emission Management, Environmental Compliance, Environmental Performance, Enterprise Energy Management, Report Variant

 

Reason and Prerequisites

This symptom is caused by a program error.

You use SAP Environmental Compliance 3.0 with a Support Package or patch level lower than the level specified in the Support Packages & Patches section.

 

Solution

Download the complete solution for your Support Package level using the appropriate link in the Support Packages & Patches section and implement it in your system. When you do this, apply the other required patches as well. For more information, see SAP Note 1974464.
