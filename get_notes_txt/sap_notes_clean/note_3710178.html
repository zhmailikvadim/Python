Symptom

Post upgrade, PREVIEW is no longer working at  Fiori App F1596 "Carry Forward Balances" 

Environment

SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Launch Fiori App F1596 "Carry Forward Balances"
  2. Click on PREVIEW
  3. Error pop up displays:
    • Some technical errors have occurred. Please contact the technical support team.
    • 500 () HTTP request failed .................................................................In the context of Data Services an unknown internal server error occurred
  4. Going to transaction /IWFND/ERROR_LOG in the front end system, error /IWFND/CM_MGW033 and info "Value 'Preview' violates facet information 'maxlength=4'" is displayed related to Odata Service FAC_BALANCE_CARRY_FORWARD_SRV.

Cause

Fiori App F1596  "Carry Forward Balances" (that used Odata Service FAC_BALANCE_CARRY_FORWARD_SRV) was set to deprecated with S/4HANA 2021. It means that the corresponding Odata Service will not be updated further, so it might fail in higher releases.

Resolution

  • From the launchpad, the Balance Carry Forward can be performed using one of the following methods: 
  1. Use Fiori App F1927 "Schedule General Ledger Jobs" and create a Job selecting Job Template "Balance Carryforward" (FINS_FI_BCFJOB_V2)
  2. Or configure (and use) GUI App FAGLGVTR as explained in the SAP Fiori Apps Reference Library.
  • Alternatively, perform Balance Carry Forward from the backend system by means of transaction FAGLGVTR.

See Also

  • For details on Fiori App F1596 deprecation, refer to SAP Note 3291458 "F1596: Carry Forward Balances app - internal session terminates with runtime error tsv_tnew_page_alloc_failed" .
  •  KBA 3363404 "Recommendations to perform balance carryforward in Fiori environment"
