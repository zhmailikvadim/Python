Symptom

  • Users experience functional issues within SAP Fiori applications, including loading problems, UI rendering issues, and unexpected freezes during navigation or data entry.
  • The issue occurs specifically with Microsoft Edge Version 144.0.3719.82 (32-bit) and later.
  • Alternative browsers such as Chrome or newer 64-bit Edge versions do not exhibit the same issue.
  • Affected applications may display a popup stating “The given date object is invalid” or log the same error in the browser console.

Environment

SAP Fiori

Reproducing the Issue

  1. Open Microsoft Edge Version 144.0.3719.82 (32-bit).
  2. Log in to the system.
  3. Navigate through SAP Fiori applications.
  4. Observe rendering delays, UI glitches, or session freezes during navigation or data entry.

Cause

The issue is caused by a compatibility problem specific to Microsoft Edge Version 144.0.3719.82 (32-bit) and later.

Resolution

The issue is fixed with the Microsoft Edge stable Version 146.0.3856.62

If you are on a 64-bit system, please verify that the corresponding 64-bit edition of Microsoft Edge is installed. You can verify this by opening Microsoft Edge and entering about:version in the address bar.

    

Workarounds:

  1. Use the 64-bit version of Microsoft Edge instead of the 32-bit version, as the issue does not occur in the 64-bit version.
  2. Alternatively, use a different browser such as Chrome, which does not exhibit the same issue.
  3. For further assistance, contact Microsoft Edge Browser support to address the compatibility issue.
