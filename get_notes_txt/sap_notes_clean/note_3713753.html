Symptom

Please note that this SAP note is intended for customers who have upgraded their S/4HANA system using the S4CORE TCI note.

This note has the following changes:

  • Adds class name to CHARACTERISTIC VALUE entities
  • Adds a field CHAR_VALUE to CHARACTERISTIC VALUE entities so that all characteristic values of any data type is in the same field, which simplifies configuration mapping
  • PERNR value is missing from WOHEADER entity

Other Terms

S4MOBADDON-8171, MAIF, SAP Dynamic Forms, Classification, Characteristic, TCI

Reason and Prerequisites

Programming Error

Solution

Please implement the attached correction instructions
