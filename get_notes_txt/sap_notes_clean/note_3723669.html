Symptom

While performing the migration of the 'Enterprise Project' object the error "First maintain parent WBS element && in the 'WBS element' column" - (Message no. CNV_DMC_SIN456) occurs. 

Environment

SAP S/4HANA Cloud Public Edition. 

Cause

The value in the field "Parent Work Breakdown Structure" in sheet "Work Breakdown Structure (WBS)" is the same as value in the field "Enterprise Project ID" in the template file.

Resolution

  1. Open the template file used for the project creation.
  2. Verify the "Parent Work Breakdown Structure" value in the sheet "Work Breakdown Structure (WBS)".
  3. Ensure that the "Parent Work Breakdown Structure" value is not the same as the "Enterprise Project ID" value. 
  4. Input a different value for the field "Parent Work Breakdown Structure" derived from the field 'Work Breakdown Structure (WBS)' of the respective Enterprise Project.
  5. Refer to KBA 3128304 for additional details on maintaining hierarchy and sort numbers.
  6. Modify the template accordingly and reattempt the migration process.
