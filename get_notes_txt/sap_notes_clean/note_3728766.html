Symptom

Component: Product Safety
Module: Specification Management

You want to specify a new data origin in the IMG settings. To do so, you

  • Start transaction SPRO
  • Navigate to the IMG node 'Environment, Health and Safety -> Basic data and tools -> Tools -> Import and Export -> Basic settings for import and export -> Specify data origin' (on ERP systems) or
    'Product Safety and Stewardship -> Basic data and tools -> Tools -> Import and Export -> Basic settings for import and export -> Specify data origin' (on S/4HANA systems)
  • Select 'New Entries'
  • Insert a new entry
  • Select 'Save'

When you select 'Save', the system displays a runtime error PERFORM_NOT_FOUND with the exception CX_SY_DYN_CALL_ILLEGAL_FORM in function group SAPLC17L.

Other Terms

SAPLC17L

Reason and Prerequisites

Reason
Program error. You have implemented SAP note 3661590 or a support package which contains this note.

Prerequisites
For information about the validity of the corrections, see the correction instructions.

Solution

The section "Support Packages" specifies the support packages that contain the corrections.

Alternatively, you can implement the attached correction instructions.
