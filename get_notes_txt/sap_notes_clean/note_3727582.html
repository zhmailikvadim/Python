Symptom

  • In BILP with German language setting, creating a shortcut for any file results in an incorrect shortcut name.
  • The shortcut is created with the English prefix “shortcut to”.
  • The prefix is not translated into German, as expected for the selected language.

Other Terms

shortcut, BI Launchpad, German, Language, copy shortcut

Reason and Prerequisites

  • SAP BusinessObjects Business Intelligence Platform 4.3 Support pack 4 patch 7
  • SAP BusinessObjects Business Intelligence Platform 4.3 Support pack 2 patch 3
  • SAP BusinessObjects Business Intelligence Platform 2025 patch 5
  • OS: Windows

--Steps to reproduce—

  1. Login to BI Launchpad
  2. Navigate to Folders
  3. Browse to the required folder and open Reports
  4. Select any report
  5. Click on Organize → Copy Shortcut
  6. Navigate to the target folder
  7. Click on the More Option in the top‑right corner
  8. Select Organize → Paste Shortcut
