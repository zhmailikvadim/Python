Symptom

Filter "Document Type" is not available when using the job "Document Journal" in app "Schedule General Ledger Job - Italy". 

 

 

Environment

SAP S/4HANA Public Cloud Edition

Reproducing the Issue

  1. Open app "Schedule General Ledger Job - Italy" 
  2. Select "Document Journal" job
  3. No filter option for Document Type
 

 

Cause

Standard system behavior.

Choosing document type as a filter is not currently supported in the Document Journal job. 

 

Resolution

Use the original app "Document Journal". It is possible to filter document type after filling in the required fields and clicking "Dynamic Selections". 

 

 

 

Keywords

Document Journal, "Schedule General Ledger Job - Italy", F1927_TRAN, document type, filter, Dynamic Selections
