Symptom

Unable to edit or modify the purchase contract created from an external system, like Ariba, using the Manage Purchase Contracts Fiori app.

Environment

  • SAP S/4HANA
  • SAP Fiori

Reproducing the Issue

  1. Open the Manage Purchase Contracts app.
  2. Find the purchase contract.
  3. Click on Edit button.
  4. Error message: "Cannot edit pur. contract XXXXXXX as it is created from an external document" - APPL_MM_PUR_OA_CON046

Cause

The app is not designed to open contracts from external systems, such as Ariba integrations. 

Resolution

This is standard behavior. This code is not enabled to change the contract from an external system. 

CL_MM_PUR_CTR_UTIL -> IF_MM_PUR_CTR_UTIL~IS_EDIT_ENABLED 

    19          IF lv_statu EQ gc_ext_purchase_doc.
    20            et_messages = VALUE #( ( messageid     = gc_contract_msg_class
    21                                     messagenumber = gc_msg_no-msg_ctr_frm_ariba
    22                                     messagetype   = gc_error
    23                                     messagevariable1 = iv_ebeln ) ).
    24            RETURN.
