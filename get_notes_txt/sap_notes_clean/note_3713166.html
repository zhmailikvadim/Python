Symptom

 To Storage Location picker excludes From Storage Location value for 411K Consignment Transfer

Other Terms

  • SAP Service and Asset Manager
  • SAM
  • SSAM
  • ICMTANGOAMF10-42482

Reason and Prerequisites

Reason:

To Storage Location picker is filtering out the From Storage Location for 411K Consignment 

Prerequisites:

  •  BAS or Visual Studio Code with SAM MDK project
  • `patch` command (available on BAS and macOS by default) via terminal or other patching utility
  • Read the guidelines on how to apply the patch and read the appendix section if you receive any error

Solution

To Storage Location picker is not filtering out the From Storage Location for 411K Consignment 

 

To apply the patch, download the attached 3713166.patch file and save it in a known location, preferably adjacent to your SAM.mdkproject.

If you are on Windows, see SAP Note 3533879 for how to install a .patch file in Windows. If you are on a Mac, follow the steps below:

  1. Open a terminal in the root of your SAM metadata (i.e. SAM.mdkproject/metadata or SAM.mdkproject/metadata/base.definitions)
  2. Run the terminal command `patch --no-backup-if-mismatch -p4 < path/to/3713166.patch`. 
  3. Verify in the terminal window that the following files have been modified and listed as patched, as indicated by the console output:
  • patching 'Rules/Inventory/IssueOrReceipt/OnMovementTypeValueChanged.js'
  • patching 'Rules/Inventory/Validation/OnStorageLocationChanged.js'
