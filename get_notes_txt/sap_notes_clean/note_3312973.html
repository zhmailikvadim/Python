Symptom

The "Data Volume Management" application on SAP for Me displays the following message:

The user is not assigned to any data sets.

For more information about utilizing the new Data Volume Management functionalities in the cloud, refer to SAP Note 2698930.

Environment

  • SAP for Me platform.
  • Data Volume Management (DVM) application.

Reproducing the Issue

  1. Log in to the SAP for Me platform.
  2. Navigate to the "Data Volume Management" application.
  3. Observe the error message: "The user is not assigned to any data sets."

Cause

There are two possible reasons for this:

  1. The user may not have the required authorization to display the data. 
  2. There might be no DVM report generated yet.

Resolution

  1. Ensure that the S-User accessing SAP for Me has the authorization profile 'Service Reports and Feedback'.
  2. Verify that at least one DVM report has been generated. To generate a DVM report, refer to SAP Note 2716655.
