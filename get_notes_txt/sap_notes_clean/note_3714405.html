Symptom

In the S/4HANA 2025 release, you cannot extend the custom extension step with an embedded process in the Fiori app 'Configure Processes'. The system displays the error message "Please specify a valid embedded process" because the Trigger Process Cluster is not populated from the value help of the Trigger Process ID.

Other Terms

Configure Process, Extend Process, F5834A

Reason and Prerequisites

The value help binding uses an incorrect element name.

Solution

After implementing this SAP Note, the system uses the correct element name in the value help binding. You can extend the custom extension step with an embedded process in the Fiori app 'Configure Processes'.
