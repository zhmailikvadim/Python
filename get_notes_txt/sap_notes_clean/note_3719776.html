Symptom

When you generate the India GSTR1 report using the Run Statutory Reports app, the serial number is missing in the B2CL section.

Other Terms

India GSTR1, IN_GSTR1_DCL, Run Statutory Reports, B2CL

Reason and Prerequisites

The mapping in the document definition for GSTR1 was incorrect.

Solution

After implementing this SAP Note, the correct data source is mapped to the Serial Number field and assigns serial numbers to the respective sections.

As a general rule, SAP recommends implementing the Support Package specified for your release. However, if you need to install a solution earlier, use the SAP Note Assistant to implement the correction instruction. For more information, refer to the SAP Note Assistant.
