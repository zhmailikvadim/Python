Symptom

Is it possible to set an icon for each link in the Recommend links task, which is one of the Onboarding tasks in the system?

Environment

SAP SUCCESSFACTORS ONBOARDING

Reproducing the Issue

  1. Go to Manage Onboarding and Offboarding Tasks
  2. Choose an Onboarding Program
  3. choose and edit recommend link task

Cause

Expected behaviour

Resolution

This is not possible to add icon for recommend link as per the Product design.
