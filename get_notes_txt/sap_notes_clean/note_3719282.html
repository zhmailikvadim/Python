Symptom

Audit history for a User to identify when the Job Position was Updated/Changed in Learning 

Environment

SAP SuccessFactors Learning

Reproducing the Issue

1] Goto Learning Administration

2] People -> Users -> Search for any user

3] Check the User Details page -> Search for Job Position/Job Code

Resolution

Currently there are no standard reports available in Learning to fetch details regarding the history on when a Job Position/Code was updated.

    Customer can Build a Custom Report using the Plateau Report Designer (PRD) tool as per requirements.

    Refer to: 2171592 - Process to enable the use of Plateau Report Designer (PRD) - SAP ONE Support Launchpad

    However, Please note that Job Classifications/Code is created via User Connector SF, So Customer can check with their team internally on who updated this value in the input file while running the connector
