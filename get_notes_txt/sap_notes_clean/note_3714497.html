Symptom

In the Pricing Grid, the system validates custom product hierarchy fields even though they do not have a foreign key or check-table relationship.

Other Terms

/IRM/CL_GIGRID_SERVICES, Pricing Grid, foreign key

Reason and Prerequisites

This is caused by program error

Please implement SAP Note#3504871 as a Prerquisite note for 2023 V2 Release

Solution

Please implement the correction instructions attached. These corrections are only valid if you have applied the Support Packs.

Release V/4 - 2023 V2, 2023 V3 and 2023 V3 Support Pack 1

Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object.
