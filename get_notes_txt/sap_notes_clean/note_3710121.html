Symptom

When opening an inspection lot or saving inspection results in transaction QE71, it takes more time than expected.

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Environment

  • SAP ERP Quality Management (QM)
  • SAP ERP Central Component
  • SAP ERP
  • SAP Enhancement package for SAP ERP
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA cloud private edition
  • SAP S/4HANA 

Cause

A large number of inspection points are assigned to one single inspection lot. The number of inspection points assigned to the inspection lot can be checked in database table QAPP.

Resolution

To reduce the response time of the system, limit the number of displayed inspection points for the inspection lot. This is possible by using field 'Max. number of inspection points' in the selection screen of transaction QE71. If, for example, value 10 is entered to this field, this will have the result that only 10 inspection points will be displayed for the inspection lot on the next screen, and previously observed slow system response will not occur.
