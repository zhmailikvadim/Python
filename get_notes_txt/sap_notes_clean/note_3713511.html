Symptom

RAIL TARIFFS- PUBLIC TRANSPORT 2026

As of 01/02/2026 the railway tariffs change, however the increase has no impact on the reimbursement of railway cards. Employers who calculate the reimbursement as a percentage of the price of a railway card, will have an increase of the reimbursement amount.

  • Employer Reimbursement Public Transport by train 01.02.2026
    For the new period starting 01.02.2026 new values for reimbursement of the employer part of railway tickets have been published.

  • Railway Tariff 01.02.2026
    The Railway Tariffs have been increased as from February 1st 2026 and published. 

  • Table Rail Tariff for Commuting Expenses (B) (T5BC0)

    The values for Public Transport are grouped via the "MOD04" field = 02
    With payroll function Determine modifiers (MOD) the personnel calculation rule Value definition for payroll modifiers (BMO1) can be processed. With payroll operation Employee grouping (Belgium) (BMODI) the modifier MOD04 can be set to address table Rail Tariff for Commuting Expenses (B)(T5BC0).

    • The reimbursement values for transport are changed and should be updated in table Rail Tariff for Commuting Expenses (B) (T5BC0). 
      See fields:
      BEPMN, BEPKW, BEPJR, BEPPT, BEF06, BEF10, BEF80 & BEF12

    • The Railway tariffs themselves are changed and should be updated in table Rail Tariff for Commuting Expenses (B) (T5BC0).
      See fields:
      BBPMN, BBPKW, BBPJR, BBPPT, BBF06, BBF10, BBF80 & BBF12

RAIL TARIFFS- OTHER THAN PUBLIC TRANSPORT 2026

  • Employer Reimbursement Other Transport 01.02.2026
    For the new period starting 01.02.2026 new values for reimbursement have been published, thus the employer's reimbursement will change.

  • Table Rail Tariff for Commuting Expenses (B) (T5BC0)

    The values for Other Transport are grouped via the "MOD04" field = 01
    With payroll function Determine modifiers (MOD) the personnel calculation rule Value definition for payroll modifiers (BMO1) can be processed. With payroll operation Employee grouping (Belgium) (BMODI) the modifier MOD04 can be set to address table Rail Tariff for Commuting Expenses (B) (T5BC0).

    • The reimbursement values for transport are changed and should be updated in table Rail Tariff for Commuting Expenses (B) (T5BC0). 
      See fields:
      BEPMN, BEPKW, BEPJR, BEPPT, BEF06, BEF10, BEF80 & BEF12

    • The Railway tariffs themselves are changed and should be updated in table Rail Tariff for Commuting Expenses (B) (T5BC0).
      See fields:
      BBPMN, BBPKW, BBPJR, BBPPT, BBF06, BBF10, BBF80 & BBF12

Note

As of 01.02.2026 the Belgian Rail introduced a fare reform, home-to-work season ticket prices are capped beyond 120 km from February 1, 2026.

* En vertu de la réforme tarifaire de la SNCB, les tarifs des abonnements domicile-travail sont plafonnés au-delà de 120 km à partir du 1er février 2026.

Consequently, the entries in table T5BC0, relevant for distances, which are not relevant anymore, are delimited like all other entries, but as of 01.02.2026 these entries do not exist anymore. More details to be found in the attached overview.
