Symptom

Add-ons processes (addon.exe) appear in the Connected Clients list.

Reproducing the issue

Prerequisites:

Run the add-ons which use Single Sign-On method to connect to the SAP Business One company database as below:

int setConnectionContextReturn = 0;
int connectToCompanyReturn = 0;
string sCookie = null;
string sConnectionContext = null;
int lRetCode = 0;
oCompany = new SAPbobsCOM.Company();
sCookie = oCompany.GetContextCookie();
sConnectionContext = SBO_Application.Company.GetConnectionContext( sCookie );
if ( oCompany.Connected == true ) {
oCompany.Disconnect();
}
setConnectionContextReturn = oCompany.SetSboLoginContext( sConnectionContext );
connectToCompanyReturn = oCompany.Connect();

Scenario:

Go to Administration → Utilities and open Connected Clients form.

Expected behavior:

Add-ons processes do not appear in the Connected Clients list.

Actual behavior:

Add-ons processes appear in the Connected Clients list.
