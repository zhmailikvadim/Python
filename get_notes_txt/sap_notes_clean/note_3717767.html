Symptom

MDG RMP 2021

Generation of Variants for Article on Retail Mass Processing(RMP) having below consistencies during creation of Change Request.

- During the article creation process assigning a merchandise category and generating the informative characteristics for an Article, the characteristics from the first record are being copied to the second record

- During the process of creating a Generic Variants via Execl import file process, the variants are not getting generated when the characteristic values are maintained through Excel file

Other Terms

MDG RMP, Generate variants, Retail Mass Processing, Import Excel file in RMP

Reason and Prerequisites

Program Error

Solution

Implement the correction instruction
