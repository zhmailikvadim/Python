Symptom

If a variant article is not maintained for the plant but is included in Deviation Quantities, FM CSAI_BOM_MAINTAIN still saves those inconsistent deviation quantities.

Other Terms

Fashion BoM, FSH_BOM_VD, Fashion BoM Deviation Quantities for Components

Reason and Prerequisites

Reason 

This problem is caused by a program error.

Solution

Implement the correction instruction provided with this SAP Note.
