Symptom

When navigating from Display POD application to IQ03 application we need Serial Number to view details.

Other Terms

MCM, /APEU, UCOM, UIUCOM, Display POD, Display

Reason and Prerequisites

Our application is using field ManufacturerSerialNumber for navigation to transaction IQ03.

Solution

We are now using the field SerialNumber instead of field ManufacturerSerialNumber for navigation to equipment details page.
