Symptom

From the Travel Request Advances screen, the application allows to choose currency of accounting, but from the Approve Travel Request the Currency for the Advance is incorrectly displayed.

Other Terms

Travel, Advance, Travel Request, Request, Currency, Approval

Reason and Prerequisites

The ABAP structure "PTRV_REQ_FOR_APPROVAL" need to be enhanced with the new "ESTI_COST_CURR" Types "TRVWAERS".

Solution

Please apply the following note that will enhance the Advance functionality by taking into account the "Advances -> Accounting In Currency" or implement the corresponding Support Package.
