Issue

Attempting to retrieve a list of filtered catalog items with selectable custom fields through Catalog Content API causes the following error:

"msg": "Invalid select field : <field_name>, input is not valid for facet"

Resolution

Please raise a Case with SAP Ariba Support.

Cause

The fields created for Catalog Content API must be ExtensibleType and refinable in order to be retrievable by the API. This error indicates that the field being requested is not refinable.  Due to this, the schema that the API utilizes needs to be updated to have this change.

Additional Information

Reference: OpenAPI: How to query catalog content API
