Symptom

DTP extraction from an Advanced DataStore Object (ADSO) may result in an error message RS_EXCEPTION000 with the following text:

Invalid host type (DECFLOAT(IBM)) for parameter (6)

Reason and Prerequisites

The ADSO has a column of type ABAP Dictionary type DF34_RAW. This data type is not support by ADBC framework which is used for extraction.

Solution

  • SAP BW 7.50
    Implement Support Package 6 for SAP BW 7.50 (SAPK-75006INSAPBW) into your BW system. The Support Package will be available as soon as SAP Note 2346340 with the short text "SAPBWNews 7.50 BW ABAP SP6", which describes this Support Package in more detail, is released for customers

  • SAP BW 7.51
    Implement Support Package 1 for SAP BW 7.51 (SAPK-75101INSAPBW) into your BW system. The Support Package will be available as soon as SAP Note 2345927 with the short text "SAPBWNews 7.51 BW ABAP SP1", which describes this Support Package in more detail, is released for customers. 

 

In urgent cases you can use the correction instructions.

Before you use the correction instructions, make sure that you check SAP Note 1668882 and SAP Note 2248091 for transaction SNOTE.

This SAP Note might already be available before the Support Package is released. In this case, however, the short text still contains the term "preliminary version".
