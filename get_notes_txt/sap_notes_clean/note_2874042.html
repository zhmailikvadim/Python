Symptom

App Set Status to Closed - Orders - Collective (CO99) can't close Production Order / Process Order.
If you enter plant and press Execute, there will be blank in the result list.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

Open App Set Status to Closed - Orders - Collective (CO99). Input plant and click execute.

Cause

App Set Status to Closed - Orders - Collective (CO99) can only be used for CO Production types or maintenance types. If you enter order types for PP production orders, nothing will be selected.

Resolution

If you want to process Production Order / Process order in mass, please check KBA 2864245.

Keywords

Mass, collectively, deletion flag, deletion indicator, together, batch
