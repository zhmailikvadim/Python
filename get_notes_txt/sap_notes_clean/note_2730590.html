Symptom

It was not possible to save the data of a worksheet containing an EVDRE with the sorting option.

A popup was displayed with two possible messages.

"The data cannot be saved because an Excel sort is activated. Please remove the sort by using the Delete Level button in DATA --> Sort, refresh the worksheet, then save the data again."

Or

"The data cannot be saved because a sort is activated within the AutoFilter Excel feature. Please remove the sort by clicking DATA --> Clear, refresh the worksheet, then save the data again."

With this fix, you can now save the data.
