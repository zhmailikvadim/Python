Symptom

SAP Financial Information Management will fail to start if requirements for SAP Java Connector are missing.

Other Terms

fim sapjco3 sapjco

Reason and Prerequisites

To communicate with systems such as SAP Business Warehouse (BW), SAP Financial Information Management embeds SAP Java Connector (SAPJCO).

Solution

With latest patch of SAP Financial Information Management (SP18 Patch 13), you will need to install Visual C++ Redistributable v14.

Previous patches (SP18 Patch 11 and SP18 Patch 12) require Visual C++ 2013 Redistributable packages.

Check https://learn.microsoft.com/fr-fr/cpp/windows/latest-supported-vc-redist?view=msvc-170 for more details.
