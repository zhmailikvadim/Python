Symptom

  • Execution of task 'Verification of Active Packages' fails with:
    Load of tables failed 
    CNVLTRL_BW2_MSG1 527
  • Starting the 'Verification of Active Packages' again, the following error occurs:
    Package Number XXXXXX already exists in Sender System
    CNVLTRL_BW2_MSG1 010

Environment

SAP NetWeaver

Reproducing the Issue

Start the Verification Task within the Migration Tool.

Resolution

Maintain in Task 'Definition of RFC Destination' for the Sender system the value NONE. 

Keywords

"Verification of Active Packages"; CNVLTRL_BW2_MSG1;
