Symptom

Post setting up profile parameter is/SMTP/virt_server_1 to system's instance profile, same is not reflecting from SICF as per below.


Environment

  • SAPConnect SMTP plugin;
  • SAP AS Netweaver ABAP.

Cause

ICM has not been restarted post maintaining the parameter.

Resolution

As parameter is/SMTP/virt_server_1 is not dynamic, proceed restarting ICM post maintaining it in the system's profile.
