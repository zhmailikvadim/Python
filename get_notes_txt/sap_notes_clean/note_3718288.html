Symptom

Consumption values are not displayed in MD04.

Other Terms

MD04, MATDOC, MVER, DVER, T438A, PROVB, VERBRAUCH_LESEN, LM61KO10

Reason and Prerequisites

Prerequisites:

  • Within MRP type customizing you have maintained 'Material Doc.' (T438A-PRODB).
  • You want to display consumption values in MD04 based on Material documents

Reason:

Due to program error consumption periods are not calculated correctly and hence consumptions values are not calculated.

Solution

This correction will fix the consumption periods calculation logic.
