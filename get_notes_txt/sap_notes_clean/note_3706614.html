Symptom

When using the Excel functionality to download and upload modifications of a purchase info record, changing the value in the No Evaluated Receipt Settlement field allows the upload but results in an error when attempting to display the info record. Error message: "In the context of Data Services an unknown internal server error occurred. Error log entry: 'Property 'IsEvaluatedRcptSettlmtAllowed' has invalid value 'x'.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Use the Excel functionality to download a purchase info record template.
  2. Modify the No ERS field value from uppercase "X" to lowercase "x".
  3. Upload the modified template back into the system.
  4. Attempt to display the purchase info record in the Manage Purchasing Info Records app.
  5. Observe the error message: "In the context of Data Services an unknown internal server error occurred."

Cause

The database requires all field values to be in uppercase. Insufficient documentation led to improper usage of lowercase values in the Excel template.

Resolution

Ensure all field values in the Excel template are entered in uppercase before uploading.
