Symptom

In the process "Receipt of 55643 from Responsible MOS to DSO" (DE_B_MDC_55643_MOS2DSO_RECV) in the MDC cluster, step 1050 does not apply the 30-minute waiting period defined in customizing (MDC_WAIT_MD_CREATION).
This issue is relevant for the market roles: DSO.

Other Terms

MDC, DE_B_MDC_55643_MOS2DSO_RECV, MDC_WAIT_MD_CREATION, 55643, DSO

Reason and Prerequisites

The error occurred because the program contained a code error.

Solution

Implement the corrections of this SAP Note or import the relevant support package.
After implementing this SAP Note, the system applies the 30-minute waiting period from customizing (MDC_WAIT_MD_CREATION) in step 1050.
