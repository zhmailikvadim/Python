Symptom

  • When a dynamic date filter value is changed and the story is bookmarked, the value selected is not retained with the bookmark.
  • When viewing the story again and selecting the bookmark, it shows the default value for the dynamic page filter instead of the bookmarked value.
  • Other filters set for the story are retained when bookmarked, only the dynamic date filter values are not retained.

Environment

SAP Analytics Cloud 2026.2.6

Reproducing the Issue

  1. In SAP Analytics Cloud, create a story with a couple of widgets.
  2. Add a dynamic date filter.
  3. View the story, and change the value in the dynamic date filter.
  4. Bookmark this state.
  5. When viewing the story again, and select the bookmark that was created.
  6. Notice the dynamic date filter values were not saved with the bookmark.

Cause

The issue is currently under investigation by development.

Resolution

  • The issue is currently under investigation by development. 
        
  • Workaround: Set the story bookmarks option to: "All Components in the Story"
    1. Open the story in SAP Analytics Cloud in Edit mode.
    2. In the left pane, select the tab "Outline"
    3. Under Global Settings, click on the three dots on the right side of the option: "Bookmarks"
    4. In the right pane, for the option "Bookmark includes", select: "All Components in the Story"
    5. Click: "Done"
    6. Save the story.
    7. New bookmarks created after applying this change will retain the dynamic date filter values.
