Symptom

If the conversion exit BEGDA or ENDDA is used in the transformation TRFN, the following termination may occur when you save/activate/read the transformation:

ABAP programming error
OBJECTS_NOT_CHAR
SAPLOM_CONVERSION_EXITS

Other Terms

TRFN, CL_RSO_TRFN_MODEL, CL_RSEM_MODEL_OBJECT, CONVERT_VALUE_ANY, SYNCHRONIZE_CONVERTIBLE_VALUES, ANY, STRING.

Reason and Prerequisites

Program

Solution

  • SAP BW/4HANA 2021
    Import Support Package 14 for SAP BW/4HANA 2021 (SAPK-30014INDW4CORE) into your BW system. The support package is available once SAP Note 3625395 ("SAPBWNews SAP BW/4HANA 2021 SP14"), which describes this SP in more detail, has been released for customers.

  • SAP BW/4HANA 2023
    Import Support Package 09 for SAP BW/4HANA 2023 (SAPK-40009INDW4CORE) into your BW system. The support package is available once SAP Note 3678578  ("SAPBWNews SAP BW/4HANA 2023 SP09"), which describes this SP in more detail, has been released for customers.

In urgent cases, you can use the correction instructions as an advance correction.

Before you do this, you must refer to SAP Notes 1668882 and 2248091 for information about transaction SNOTE.

To provide information in advance, the specified SAP Notes may already be available before the Support Package is released. In this case, however, the short text still contains the word "preliminary version".
