Symptom

You are executing either a migration or conversion using Object-Based Transformation (OBT) with SAP Landscape Transformation (SAP LT). In case of an RFC issue the preselection version is switched from version 2 to version 1.

Other Terms

SAP Landscape Transformation, SAP LT, DMIS 2011, DMIS 2018, Object-Based Transformation, S4CORE, On-Premise, OP, Cloud Private Edition, Object-Based Transformation, OBT, Preselection,  Preselection Version1,  Preselection Version2

Reason and Prerequisites

Program error in OBT code.

Solution

Implement the attached correction using transaction SNOTE or import the relevant support package for OP / Cloud Private Edition into the control system / DMIS into the control system.
