Symptom

When you are creating a purchase order for a centrally managed purchase requisition using the app Process Purchase Requisitions Centrally, you are unable to edit the Currency field.

Other Terms

Process Purchase Requisitions Centrally, Manage Purchase Orders, F3290, F0842A, Central Purchasing, Create Purchase Orders Centrally

Reason and Prerequisites

Program error

Solution

With this fix, you can edit the Currency field at the header level while creating a purchase order for a centrally managed purchase requisition. It is available from SAP S/4HANA Cloud Public Edition 2602.1 release.
