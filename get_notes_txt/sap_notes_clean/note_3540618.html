Symptom

During a pull an error could be raised by transport tool R3trans with an error message which could looks like:

1AETW000 Exception FingerprintFileFault: Error while reading 'FINGERPRINT.json': Fingerprint for file
'%2FCBSA%2FC_WM_EINV.asx.json' does not exist
2EETW000 Please contact the SAP support.
exiting ...
R3trans finished (0016).

Reason and Prerequisites

This error can occur if the following conditions are given:

  • On an earlier commit, a complete table was deleted and pulled 
  • An own repository is used with fingerprint files

During the pull of the deletion of a complete table a necessary file was not handled correctly, which leads to the above error message.

Solution

The R3trans version, which includes the fix and shows the version: 15.11.24 or more recent one.
