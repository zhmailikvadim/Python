Symptom

Support Tools for Transportation Management.

Other Terms

BONAV

Reason and Prerequisites

Program Error.

The usage of transaction /SCMTMS/BONAV is intended only for support and error analysis purposes. Other usages are not recommended and in full responsibility of the customer.

Especially in productive environments, the tools must always be used with the greatest care!

Solution

Implement the attached correction instructions or install the corresponding Support Package.
