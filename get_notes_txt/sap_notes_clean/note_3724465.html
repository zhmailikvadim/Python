Symptom

You added customer fields to both LIKP and LIPS. The customer fields in LIKP and LIPS have same name but with different data types. This causes syntax error in testing class of CL_SD_DELIVERY_ACCESS after upgrade to S/4HANA.

Other Terms

CL_SD_DELIVERY_ACCESS, syntax error, TCL_SD_DELIVERY_ACCESS

Reason and Prerequisites

The issue is caused by a table join error in testing class in a special case when customer added own fields.

Solution

Implement the attached correction or upgrade to the corresponding support packages.
