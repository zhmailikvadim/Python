Symptom

Person External ID 'personExternalId' column does not show value in Growth Portfolio Export via Integration Center even though the job executor has all the mandatory Growth Portfolio and Integration Center permissions enabled.

Environment

SAP SuccessFactors Talent Intelligence Hub

Reproducing the Issue

1. Download the ICD file for Growth Portfolio Export from SAP Business Accelerator Hub.
2. Import ICD file and create new integration.
3. Add the Destination setting in the Integration Job and Run the Job.
4. In the file exported in the SFTP, notice that the Person External Id 'personExternalId" does not show any values.

Cause

The job executor does not have the RBP permission "General User Permission > User Search" enabled, which is required for the Growth Portfolio Export to function correctly.

Resolution

Enable the RBP permission "General User Permission > User Search" for the IC Job executor to resolve the issue with the "personExternalId" field.
