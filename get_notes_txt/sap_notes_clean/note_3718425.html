Symptom

In the process "Receipt of Delivery Note for Grid Usage Billing (SUP View)" (DE_B_GUB_SUP_DELV_NOTE), step 0130-0100 runs into an error when attempting to trigger the profile value complaint process if no profile value can be determined for the time period of the delivery note.

This issue is relevant for the market role: SUP.

Other Terms

Billing Enhancement, Delivery Note, Profile Value Complaint, /US4G/CL_PRST_TRG_COMP_PROC_PFDO_ACTION, OBIS Code Conversion, GUB_SUP, DE_B_GUB_SUP_DELV_NOTE

Reason and Prerequisites

The error occurred because the OBIS code conversion logic didn't work when no profile value could be determined for the time period of the delivery note.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the system converts the OBIS code correctly and triggers the profile value complaint process. You need to reprocess the error process step by handling the BPEM case. You can verify the fix by checking that step 0130-0100 runs successfully and triggers the profile value complaint process when no profile value can be determined for the time period of the delivery note.
