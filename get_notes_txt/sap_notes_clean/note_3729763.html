Symptom

With this NOTE,

Utility report is created to bring Push instance from COMPLETE to NEW status. Post that, this push instance will be used to retrigger the outbound queue.

Report        : DFS_DIS_PUSH_MANIPULATION

Transaction : DFS_PUSH_ST

Logging enabled with Log object DEFENSE_DIC with sub-object DFS_PUSH_MANIP.


Other Terms

S4 Defense Disconnected, Push Instance, MatStock

Solution

Implement the attached correction instructions provided with the NOTE.
