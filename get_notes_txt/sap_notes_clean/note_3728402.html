Symptom

-An error is encountered during IDOC replication from MDG to COMET ECC, and the message "Duplicate found, Object: , Key:" is displayed with IDOC Status 51 and Message Number AM359.

-The error is triggered when DRFOUT is performed from MDG to ECC.

Environment

    • SAP ERP
    • SAP enhancement package for SAP ERP
    • SAP enhancement package for SAP ERP, version for SAP HANA
    • SAP ERP Central Component
    • SAP S/4HANA
    • SAP S/4HANA Cloud Private Edition
 

Reproducing the Issue

1. Perform the DRFOUT operation from MDG to ECC.

2. Observe the IDOC error message: "Duplicate found, Object: , Key:" during replication. 

3. Check the address side for duplicate entries in the name field associated with the person number.

Cause

The error message is triggered because the entry in the name field has already been associated with a person number. Consequently, when a new entry is attempted, duplicate checks are performed by the system on the relevant fields, and the error message is generated.

Resolution

To resolve the issue, either of the following corrections may be applied:

  • The duplicate checks may be disabled, provided that the functionality of the system is not impacted.
  • Alternatively, a new name may be used instead of "PERSON" to avoid duplication conflicts.

Keywords

IDOC fails, Duplicate found, Status 51, AM359
