Symptom

Period in the header of form in report RPCSTLT2M (za období) is incorrect in case of multiple employment. Problem occurs if main personal number has employment status 0 in the whole processing period.

Other Terms

HRCZ, RPCSTLT2M

Reason and Prerequisites

Correction of legal function.

Solution

Issue was corrected in report RPCSTLT2M.

Installation instructions:

However, if you need to install a solution earlier, use the Note Assistant to implement the correction instruction. You can find more information about the Note Assistant in SAP Service Marketplace, under service.sap.com/note-assistant.
  • Note that documentation, transactions, and Customizing activities are only delivered via Support Package.
  • To implement the solution, you must implement the correction instructions that are valid for your release.
