Symptom

  • Non-admin roles have access to the IT component reference catalog.
  • These roles can link reference catalog items to existing IT components, which may result in incorrect links due to inadequate quality.

Environment

LeanIX Integration lifecyclecatalog

Resolution

  1. Navigate to the IT component meta model configuration in LeanIX.
  2. Access the 'permissions' tab within the configuration.
  3. Disable the create, read, update, and delete permissions for the Catalog Status field for roles that should not have access.
  4. Confirm that the IT-component reference catalog is no longer accessible for the restricted roles.

Keywords
