Symptom

You have recently upgraded to SAP Customer Activity Repository applications bundle 6.0 FPS02 and you are running UDF forecasting reports. The forecasting engine incorrectly classifies offers as Unsupported. This happens due to an error in the calculation view SAP.IS.DDF.UDF.CUBEOFFERTERMDETAIL.

Other Terms

Unified Demand Forecast, UDF, CARAB, SAP CARAB 6.0 FPS02, Forecast, SAP.IS.DDF.UDF.CUBEOFFERTERMDETAIL, Upgrade, Offer Term

Reason and Prerequisites

Reason: The calculation view SAP.IS.DDF.UDF.CUBEOFFERTERMDETAIL is not updated with the latest code corrections.

Solution

 Implement the attached correction instructions in your back-end system.
