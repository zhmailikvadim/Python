Symptom

When using report RFM_ASM_MISSING_DATA_GEN or the Fiori app "Postprocess Assortments" and selecting a large amount of data (i.e.or  option "Generate for All"), you receive memory related runtime errors.

Other Terms

 SYSTEM_NO_ROLL, TSV_TNEW_PAGE_ALLOC_FAILED

Reason and Prerequisites

This is a programming error.

Solution

Implement this SAP-note using transaction SNOTE.
