Symptom

Service Center is used to explore services from various service providers. The process fails and there is a need for more detailed traces for further investigation. 

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental." 

Environment

SAP Business Application Studio

Resolution

In SAP Business Application Studio:

  1. Open "Settings":



  2. Search for "Service Center" settings, and update the Service Center logging level configuration to "debug":



  3. From the main menu, select View -> Output to open the "Output" view:



  4. Select the "Service Center" output from the dropdown to view the Service Center logs:



  5. Go to the Service Center and reproduce the issue

  6. If required by SAP Support, copy the output and attach it to the SAP case.

  7. Set the log level back to "warn" (default):
