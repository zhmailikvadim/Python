Symptom

After enabling the Enterprise Search functionality in SAP Build Work Zone, search queries execute successfully and return results. However, when attempting to open any of the returned items, the system displays an error message instead of navigating to the expected content. 

Sorry, the app couldn't be opened.

The navigation target "#XXXXX&sap-app-origin-hint=XXX&sap-system=sid(XXX.XXX)" of the app could not be resolved. This can be caused, for example, by an incorrect content configuration, an incorrect or missing role assignment, or because your current device is not supported.

Environment

SAP Build Work Zone Standard Edition 

Reproducing the Issue

  1. Log in to the Work Zone Site.
  2. Search for any application.
  3. Click on the application received in the search result; the error occurs.

Cause

Incorrect alias mapping in the content provider.

Resolution

Verify the alias mapping of the Content Provider as described in the below document
Integration with Enterprise Search | SAP Help Portal
