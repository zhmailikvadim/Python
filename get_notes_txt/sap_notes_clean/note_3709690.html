Symptom

  • Tooltips (Help Text) are not functioning on application tiles within the Work Zone Homepage, workspaces, or work pages
  • Application tiles display tooltips correctly in the Applications area but fail to show tooltips when used in the Work Zone Homepage or other Workspaces

Environment

SAP SuccessFactors HCM Suite

Reproducing the Issue

  1. Create an Application tile and maintain both title and subtitle texts
  2. Add the new tile to the following locations:
    1. Applications Homepage
    2. Work Zone Homepage
    3. Workspace 
  3. Hover the mouse over the tile in all locations to test the tooltip functionality
  4. Notice that only in Applications tab tool tip works

Work Zone Home Page view:

Applications tab view:

      Cause

      The current behavior is consistent with the system's design. 

      Resolution

      In the redesigned version, tooltips are specifically intended to appear for truncated text within tiles. This means that tiles with text that is cut off will display a tooltip when hovered over, whereas tiles with fully visible text will not.
