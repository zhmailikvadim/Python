Symptom

  • In the Mobile Application Integration Framework Configuration Panel (/SYCLO/CONFIGPANEL), under the Client Globals tab, the Client Global Settings appears completely empty.

Environment

  • SAP Service and Asset Manager
  • Mobile Application Integration Framework Configuration panel (/SYCLO/CONFIGPANEL)

Reproducing the Issue

  1. Open ConfigPanel and navigate to the Client Globals tab; observe that the Client Global List is empty.

Cause

The Client Globals settings are not used in SAP Service and Asset Manager; it is only applicable to Agentry projects (for example, Work Manager), so the empty list is expected.

Resolution

This is expected behavior for SAP Service and Asset Manager.
