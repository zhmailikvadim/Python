Symptom

In the process "MOS Billing Request for Quotation" (MOSB0007), step 98 determines the wrong rejection code even though the leading MaLo is a feed-in.

This issue is relevant for the market roles: MOS, SUP.

Other Terms

MOSB, MOS Billing, /MOSB/CL_CHECK_METHOD, CHK_REQOTE_LEADING_MALO_FEEDIN, E_0207, A05, A06, rejection code, leading MaLo, feed-in, billing request for quotation

Reason and Prerequisites

The error occurred because the system checked the non-leading MaLo instead of the leading MaLo to determine if it is a feed-in.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the system determines the correct rejection code E_0207-A05 when the leading MaLo is a feed-in. You can verify the fix by checking that the rejection code E_0207-A05 is determined in step 98 of the process "MOS Billing Request for Quotation" (MOSB0007) when the leading MaLo is a feed-in.
