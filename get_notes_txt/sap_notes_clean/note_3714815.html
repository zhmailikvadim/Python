Symptom

To comprehend the importance of the start item parameter in the FCLM_FLOW_BUILDER_2 program.

Environment

  • SAP S/4HANA 2021
  • SAP S/4HANA 2022
  • SAP S/4HANA 2023

Reproducing the Issue

Run the program FCLM_FLOW_BUILDER_2 in SE38

Resolution

1. Program FCLM_FLOW_BUILDER_2 uses this parameter. It has three possible values:

  • ' ': stands for 'Related Items'.
    The Program will first fetch out all relevant documents based on Selection Criteria, and then search the start items for these relevant document items.
  • 'X' stands for 'Input'.
    The program gets start items from the input documents based on Selection Criteria.
  • 'Y' stands for 'Actual input'.
    The program will search for the start items whose certainty level is ACTUAL from the input documents based on Selection Criteria.

2. The running type Liquidity Analysis: Delta Run from Staging only uses the Related Item as its value. In this mode, the program will fetch out all relevant documents from the Staging table, and then search the start items for these relevant document items.
