Symptom

  • "Access to file denied - Message No. FES012" occurs when attempting to access a folder while running a transaction (e.g, custom [Z or Y] codes, AL11, etc.) in SAP GUI.
  • The error is generally linked to a specific directory path.
  • The error can occur in specific clients from same system.

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

  • SAP GUI for Windows
  • ABAP Platform

Cause

The error is not caused by SAP GUI. The file access is failing because of the lack of permissions/authorizations to create file on the OS level. The following information can be found in the SAP GUI Data Provider trace:

Mass Data Transfer(Error):    CreateFile failed: hr=0x80070005, lastError=0(Access is denied.)

Mass Data Transfer(Error):    CreateFile failed: hr=0x80070005, lastError=0(Access is denied.)

Mass Data Transfer(Error):    ConvertStreamToFile failed: hr=0x80070005(Access is denied.)

Mass Data Transfer(Error):    ConvertToFile failed: hr=0x80070005(Access is denied.)

Mass Data Transfer(Error):    CSAPDataProviderFormat::SaveDataToFile(FileName='C:\<file path>') =>hr=0x80070005(Access is denied.)

Resolution

  1. Involve the Administrator to review and change the authorizations to access the affected directory path.
  2. Ensure that the user has the necessary permissions to access and create files in the directory.
  3. In some cases, where the problem still persists on a specific SAP standard product, the application must be involved.

Taking a SAP GUI session trace for the following components can help into identify the issue: "Access Control", "Automation", "Context manager", "Control manager", "Frontend Services", "General", "Mass Data Tables", "Mass Data Transfer" with Timestamp and Detailed Output checkboxes enabled. Refer to KBA 2291242 - Tutorial "How to create a Session Trace" for guidance. 

In case the access error occurs directly with a system Managed by HEC, create a case under PCO-OPS component.
