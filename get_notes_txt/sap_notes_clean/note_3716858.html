Symptom

Correspondence form not generated for US 1042-S Withholding tax via DRC report .

Environment

  • SAP S/4HANA
  • SAP Fiori for SAP S/4HANA

Reproducing the Issue

  1. Access the "Run Statutory Reports" application.
  2. Select the report name "US_WHT_1042S" and choose the report entity.
  3. Attempt to generate the report , no correspondence form generation.

Cause

The withholding tax key CHP3 is not maintained

Resolution

  1. Define Withholding Tax Key as CHP3 in below path:-

    SPRO- > Customizing for Financial Accounting under Financial Accounting Global Settings-> Withholding Tax -> Extended Withholding Tax-> Report ->Country-Specific Functions -> United States of America -> Define Withholding Tax Keys.

  2. Assign CHP3 Withholding Tax key to respective Withholding Tax code in below path:-

    SPRO- > Customizing for Financial Accounting under Financial Accounting Global Settings-> Withholding Tax -> Extended Withholding Tax-> Report ->Country-Specific Functions -> United States of America -> Assign Withholding Tax Keys 
    Assign Withholding Tax Keys | SAP Help Portal 

  3. Define Official Withholding Tax Key under below path as per help document  Define Withholding Tax Keys | SAP Help Portal 

    SPRO-> Customizing for Financial Accounting under Financial Accounting Global Settings->  Withholding Tax -> Extended Withholding Tax -> Basic Settings -> Define Withholding Tax Keys.

  4. Maintain Official Withholding Tax key in below path based on income type.

    SPRO-> Customizing for Financial Accounting under Financial Accounting Global Settings-> Withholding Tax -> Extended Withholding Tax -> Calculation -> Withholding Tax Codes-> Define Withholding Tax Codes.
