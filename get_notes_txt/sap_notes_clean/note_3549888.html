Symptom

Being on public cloud its need to have a coloured bar that contained the system, client and type of system when they were connected via Fiori with a different colour per system (blue, green, pink). 

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Environment

SAP S/4HANA

Resolution

The system info bar can be enabled by the administrator to display above the shell bar.

  • The standard color for this bar is orange, but the colors can be defined by the administrator.
  • To display the system information bar, the new parameter SYS_INFO_BAR can be used.
  • To change the color of the system information bar, the new parameter SYS_INFO_BAR_COLOR can be used and the required color can be selected from the dropdown list.
