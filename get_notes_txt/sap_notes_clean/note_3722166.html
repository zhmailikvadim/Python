Symptom

Why engagement-type control status does not change after control status import in Ariba Supplier Risk?

Resolution

The control status import is not applicable for controls in the following scenarios:

  • The control is of type Engagement.
  • The control is currently pending or open in an active engagement for the supplier.
  • The control status is Expired.

See Also

Importing Status Data for Assessments and Risk Controls provides further details about prerequisites and restrictions related to the status import for controls and assessments.
