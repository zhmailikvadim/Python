Symptom

  • Batches were created with an incorrect Batch Class due to an incorrect Batch Class assignment in the Material data.
  • There is a business requirement to change the Batch Class that is already assigned to a Batch

Environment

  • SAP ERP
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition
  • Batch Management (LO-BM)

Resolution

Use transaction BMCHCL to perform Batch Conversion. 

    • Important to highlight that the new class must be an extension of the previous class.
    • This means that no characteristic is deleted from the new class, characteristics are only added to it in comparison to the old one.
