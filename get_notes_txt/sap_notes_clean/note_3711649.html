Symptom

  • The issue involves the inability to display tables /SCWM/TWCR and /SCWM/TWCRT via SE16n or SE16 in SAP S/4HANA.
  • Error message: "You cannot display /SCWM/TWCRT with the standard tools" (Message No. MO408).
  • Diagnosis: The maintenance status of the table or view is set in the ABAP Dictionary in such a way that display and maintenance using standard tools (SE16, SE17, SM30/SE54) is not allowed.

Environment

  • Controlling (CO)
  • SAP R/3
  • SAP R/3 Enterprise
  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA Finance
  • SAP S/4HANA

Reproducing the Issue

  1. Attempt to display table /SCWM/TWCR or /SCWM/TWCRT using SE16 or SE16n.
  2. Observe the error message "You cannot display /SCWM/TWCRT with the standard tools" (Message No. MO408).

Cause

The maintenance status of the table or view is configured in the ABAP Dictionary to restrict display and maintenance using standard tools such as SE16, SE17, SM30/SE54.

Resolution

  1. Navigate to SE11 → Change Mode → Delivery and Maintenance.
  2. Set "Data Browser/Table View Editing" to "Allowed".
  3. Activate the table status if it is inactive.
  4. Retry displaying the table using SE16 or SE16n.
