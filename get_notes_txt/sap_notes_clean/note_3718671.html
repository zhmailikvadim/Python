Symptom

An error occurs when attempting to create a Data Provider Profile in SAP Datasphere:
"You don’t have the necessary privileges to create and manage the Provider profile. Please contact your tenant administrator"

The issue can impact both regular users and the system owner, even if they have the required privilege (Update).

Environment

SAP Datasphere

Reproducing the Issue

  1. Access SAP Datasphere tenant
  2. Navigate to the Data Sharing cockpit 
  3. Go to my Data Provider Profile
  4. Attempt to create a Data Provider Profile.

Cause

The scoped roles required, Space Admin Update privilege, for creating and managing the Data Provider Profile were not assigned to the user attempting to create the profile.

Resolution

  1. Verify that the user attempting to create the Data Provider Profile has the necessary scoped roles assigned, Space Admin privilege.
  2. If the scoped roles are missing, assign them to the user
  3. After assigning the scoped roles, attempt to create the Data Provider Profile again.
  4. Confirm successful creation of the Data Provider Profile.
