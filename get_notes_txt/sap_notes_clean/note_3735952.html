Symptom

According to the Fiori App Library, with Monitor Order Progress app (CO46), you can select a sales order, process order, production order, or planned order and navigate directly to the order report in the Monitor Stock/ Requirements List app (MD04). However, when checking the order report in CO46 and MD04 respectively, you find that the results are different in the two apps.

Environment

SAP S/4HANA Cloud Public Edition

Cause

The results in the two apps are different because the overall profiles used are different. Different overall profiles can control different elements to be displayed on the order report.
By default, MD04 uses overall profile SAP000000001, while CO46 uses SAP000000002.

Resolution

Change the overall profile used in these two apps accordingly.

  • For CO46, the overall profile can be selected on the initial selection screen. 
  • For MD04, the overall profile used can be changed via "User Settings -> General Settings -> Order Profile".
