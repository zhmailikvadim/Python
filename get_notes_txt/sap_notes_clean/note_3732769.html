Symptom

  • When creating a Launchpad App Descriptor Item (LADI) with the launchpad application manager, the generated LADI ID is prefixed by a reserved namespace, but this does not ensure uniqueness across systems.
  • During migration of a Fiori front-end server from standalone (hub) to embedded, LADIs are overwritten due to duplicate LADI IDs existing in different systems.
  • LADI IDs are system-generated and cannot be manually changed.

Environment

  • ABAP platform
  • SAP Fiori Launchpad before SAP_UI 7.58 

Reproducing the Issue

  1. Create or maintain LADIs in separate systems (for example, a back-end system and a front-end server) where LADI IDs are generated with the same reserved namespace prefix.
  2. Prepare migration from standalone (hub) to embedded FES and transport content.
  3. In the target system, use transaction SE01 to review transport object lists and observe that UIAD objects with the same generated ID exist from different source systems.
  4. Import the transport and observe that existing LADIs are overwritten due to duplicate IDs.

Cause

In certain releases, the generated Launchpad App Descriptor Item ID can be truncated at the end due to the namespace, for example, before SAP_UI 7.58 SP04

Resolution

  1. Determine the SAP_UI release of the affected system(s).
  2. If SAP_UI 7.58 SP01–SP03 is in use, implement SAP Note 3572413-UIAD migration: Fails due to duplicate inserts (Optimization of SAP Note 3568848) This note strengthens the LADI ID generation logic and reduces the chance of duplicates.
  3. If already on SAP_UI 7.58 SP04 or higher, note implementation is not required; however, LADIs created before SAP_UI 7.58 SP04 may still be affected.
  4. To prevent overwriting original objects during import, review and set appropriate import options in the Change and Transport System (CTS/TMS) as described in KBA 3088775 - Meaning of transport import options in STMS.

See Also

  1. SAP Note 3572413-UIAD migration: Fails due to duplicate inserts (Optimization of SAP Note 3568848)
  2. KBA 3088775 - Meaning of transport import options in STMS
