Symptom

You edit a credit item in transaction FPCRPO. The "Pay" tab displays the system date as the due date, even though the credit item has a different due date. The system transfers this date when you change the document.

Other Terms

FPCRPO; enhanced credit processing; Pay; document change

Reason and Prerequisites

  • You use the enhanced credit processing.
  • The due date of the item is in the past.
  • You edit the item in transaction FPCRPO.
  • You have implemented SAP Note 1910043.

Solution

Implement this SAP Note or import the specified Support Package.
