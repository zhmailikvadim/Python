Symptom

You create and post an Agreement Request using the upload program and expect the file for Agreement Termination to process correctly, as the referenced agreement contains all required values. However, incorrect error messages appear during the upload, indicating missing or invalid data even though the agreement includes these values.

This note corrects the behavior by ensuring that valid agreement data is properly recognized and error messages are only shown when data is truly missing or incorrect.

Other Terms

/IRM/IPAGRM, /IRM/LIPAGRMCLS, LCL_DISPLAY_PROFILE_HANDLER IMPLEMENTATION, SCREEN_MODIFY_0105, /IRM/LIPAGRMF0S, /N/IRM/IPCRARUPLD

Reason and Prerequisites

This is caused by program error.

Note : Please implement SAP Note#3564955 and # 3669164 as a Pre requisite note for 2023 V2 and 2023 V3 Release

Solution

Please implement the correction instructions. These corrections are only valid if you have applied the following Support Packs.

Release V/4 –  2023 V3 and 2023 V3 Support Pack 1.

Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object
