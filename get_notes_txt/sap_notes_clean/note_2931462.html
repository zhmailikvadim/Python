Symptom

Transformation having constant mapping for field with NUMC datatype and value > 61 will result in an error while activating transformation.

Formula element is longer than 61 characters

Parsing of formula XXX failed

Other Terms

NUMC, 61, Formula, Constant, Transformation

Reason and Prerequisites

Program error

Solution

To solve the issue please apply the Support Package as mentioned below. In urgent cases you can use the correction instructions.

Delivery Information

  • SAP BW/4HANA 1.0
    Implement Support Package 17 for SAP BW/4HANA 1.0 (SAPK-10017INDW4CORE) into your SAP BW/4HANA system. The Support Package will be available as soon as SAP Note 2891199 with the short text "SAPBWNews SAP BW/4HANA 1.0 SP17", which describes this Support Package in more detail, is released for customers.
      

In urgent cases you can use the correction instructions.

Before you use the correction instructions, make sure that you check SAP Note 1668882 and SAP Note 2248091 for transaction SNOTE.

This SAP Note might already be available before the Support Package is released. In this case, however, the short text still contains the term "preliminary version".
