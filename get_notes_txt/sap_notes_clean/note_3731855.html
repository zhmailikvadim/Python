Symptom

Odata call for API /IBP/PLANNING_DATA_API_SRV in communication arrangement SAP_COM_0720 fails with error "You need authorization to display this object."

Environment

  • Generic OData V2 connectivity to IBP (service PLANNING_DATA_API_SRV)
  • SAP Integrated Business Planning

Cause

  • IBP OData authorization configuration issue: in communication arrangement SAP_COM_0720, the "Corresponding Business User" was not assigned to a business user with the required authorizations for the data requested, causing the API to return an authorization error.

Resolution

  1. In SAP IBP, open the Communication Arrangements app.
  2. Locate and open the communication arrangement SAP_COM_0720 used for PLANNING_DATA_API_SRV access.
  3. Set the "Corresponding Business User" to a valid business user that will represent the inbound communication.
  4. Ensure this business user has the necessary roles/authorizations.
