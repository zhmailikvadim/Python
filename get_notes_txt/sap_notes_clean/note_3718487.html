Symptom

You save a capacity document like the freight order. If the freight order has a huge amount of items, a runtime error can happen because of a memory overflow. This can also happen if you run the program for creating freight settlement documents with a high number of freight orders in the selection.

This is a follow-up note of note 3687314.

With this note, further improvements regarding memory consumption and runtime are provided.

Other Terms

/SCMTMS/CL_TOR_D_ROOT_BS, /SCMTMS/CL_LI_TOR_IDNT_BTD_CHG; /SCMTMS/SFIR_CREATE_BATCH

Reason and Prerequisites

There is a program error.

Solution

Implement the corrections of the attached correction instructions or apply the corresponding support package.
