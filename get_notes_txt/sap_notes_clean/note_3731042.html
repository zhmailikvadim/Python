Symptom

You use the infotype 0600 (Employer Statement) to create employer statements for your employees.

After you implement SAP Note 3722365 (ELM 5, LAW (CE): Missing location on wage statement), the name of the employer no longer appears on the form.

Other Terms

Name of employer, AGBE, employer statement, SFP, form

Reason and Prerequisites

Program error

Solution

Import the HR Support Package relevant for your release. If you have to implement the solution in advance, use the SAP Note Assistant to implement the correction instructions relevant for your release. You can find more information about the SAP Note Assistant on the SAP Support Portal under https://support.sap.com/notes.

After you implement this SAP Note, the name of the employer is determined again using the company name specified in the table/view V_T536A and printed correctly in the document.
