Symptom

There is a discrepancy in the invoice date for the same invoice between ARIBA and SAP ERP.
For Example, the Invoice Date is displayed as 1/8/2026 in Ariba invoice, but the value is showing 1/7/2026 in ERP. 

Environment

SAP Ariba Buying & Invoicing

Cause

This Invoice Date is shown as 1/8/2026 in the Ariba invoice UI, because system will convert this field to reflect your local time zone and display in UI. 

Actually this value is stored as 1/7/2026 PST in database. When system send out invoice to external ERP, system will sent out this value from database. 

Resolution

This is expected behavior.

Keywords

Date difference, ERP invoice, display
