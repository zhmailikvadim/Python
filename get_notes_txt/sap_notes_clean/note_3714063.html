Symptom

When executing the Deferred Tax report, one-time account data for customers or vendors is not displayed.

The following fields are not retrieved for one-time accounts:

  • STCD1 – Tax Number 1
  • STCD2 – Tax Number 2
  • ORT01 – City
  • PSTLZ – Postal Code
  • STRAS – Street
  • NAME1, NAME2, NAME3, NAME4 – Name

As a result, the corresponding information is missing in the ALV output of the report.

Other Terms

RFUMSV50, S_AC0_52000644, Deferred Tax Transfer, BSEC

Reason and Prerequisites

Incorrect program behavior

Solution

Implement automatic correction instructions of this SAP Note or implement the mentioned Support Package.
