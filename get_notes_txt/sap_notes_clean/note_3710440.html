Symptom

The /AIF/ namespace is missing after a system patching update.

Environment

  • Application Interface Framework for NetWeaver
  • SAP BTP, ABAP Environment
  • SAP S/4HANA

Reproducing the Issue

  1. In SNOTE, attempt to download any AIF relevant note.
  2. Observe the failure "Namespace /AIF/ does not exist"

Cause

The system patching/SAP update caused an inconsistency in the namespace tables (e.g., TRNSPACE, TRNSPACET, TRNSPACETT).

Resolution

  1. Manually add the missing namespace using the following steps:

a. Access transaction code SM30.

b. Open table V_TRNSPACE.

c. Create a new entry with the following details:

    • Namespace: /AIF/
    • Namespace role: C
    • Repair License: 27264839141722606756
    • SSCR Popup: Check
    • SAP Standard: Not check
    • Gen. Only: Not check

  2. Save the changes.
  3. Retry the operation.

Keywords

/aif/, namespace missing, TRNSPACE, TRNSPACET, TRNSPACETT, V_TRNSPACE
