Symptom

Upon entering the Reprocess Bank Statement Items - FEB_BSPROC app, different users see different initial screens.

User 1:

User 2:

“Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.“

Environment

 SAP S/4HANA Cloud Public Edition

Cause

The “Display Quick Start” option is enabled in the application for user 1.

Resolution

  1. Open the “Reprocess Bank Statement Items - FEB_BSPROC” application.
  2. Unselect the “Display Quick Start” option.
  3. Quit the application.
  4. Reopen the application.
  5. Confirm that the expected screen loads.

Keywords

Reprocess Bank Statement Items, FEB_BSPROC, initial screens, Display Quick Start
