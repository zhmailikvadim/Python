Symptom

Custom competencies are not pulling or showing in the 360 Forms. 

Reproducing the Issue

  1. Login to the instance
  2. Launch 360 Review form for any Test User
  3. Progress form to Evaluation stage ‘Provide Feedback’.
  4. View form as any participant 
  5. No custom competencies are visible
  6. Navigate to Manage Templates
  7. Custom competency option is grayed out, cannot select it

Cause

As per current system design, MTR forms only support standard attributes such as Skills and Competencies.

Resolution

As we are currently attempting to apply custom attributes on MTR forms, MTR forms do not support custom attributes at this time. They only support standard attributes such as Skills and Competencies.

To address this requirement in the future, our Engineering team has already raised an enhancement request with the Product team for further evaluation and consideration for the Future.
