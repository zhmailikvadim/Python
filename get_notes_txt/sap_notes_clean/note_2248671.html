Symptom

You create import freigt documents from the export freigh booking. The error message occurs: "Value Import for Traffic Direction does not comply with value Export that exists in Customizing"

Other Terms

shipper scenario, /SCMTMS/TRQ312

Reason and Prerequisites

Program error. The import FWO MOT on header level gets initialized thus the import FWO type determination fails.

Solution

Implement the attached correction instruction or install the corresponding support package.
