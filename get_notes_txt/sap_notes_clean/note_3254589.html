Symptom

You are applying a Routing rule to route the request line item based on the result of the risk analysis performed in the request. The routing rule is terminated with a memory dump and the workflow request remains in 'Decision Pending' state (is not routed to the next approver).

Other Terms

GRC, Access Control, Access Request, Workflow request, decision pending, stuck, route.

Reason and Prerequisites

Processing of the violations inside the request is not optimized.

Solution

Implement the attached correction instruction or update the relevant Support Package.
