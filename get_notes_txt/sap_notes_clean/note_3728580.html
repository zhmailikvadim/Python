Symptom

  • An error occurs when attempting to upload a mass file with Agreement Work on App Manage Workforce F6097A.
  • Error message: "Employee: Person External ID doesn’t exist.".

Environment

SAP S/4HANA Cloud Public Edition

Cause

The issue is caused by missing worker agreements for the respective workers.

Resolution

When importing the Service Cost Level, it is necessary to ensure there is a work agreement for the respective worker as per How to Import Worker Data | SAP Help Portal.

-> However, if you want to assign a worker to a cost center or enter a company code, you need to create a work agreement for the respective worker.
