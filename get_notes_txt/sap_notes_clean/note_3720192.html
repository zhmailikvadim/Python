Symptom

An action (for example a Process Chain Transport) leads to the error: Incompatible Call Rejected .... Called Incompatible Function :RSPC_API_CHAIN_GET_STARTCOND

Environment

  • SAP BW/4HANA
  • SAP Netweaver
  • SAP S/4HANA

Cause

The Function Module is blocklisted and the UCON-Check is active.

Resolution

Use the program RS_RFC_BLACKLIST_CUSTOM from SAP Note 2408693 and add the Function Module to the blocklist override.
