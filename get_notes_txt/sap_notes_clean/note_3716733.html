Symptom

It is not clear how the "Defect Class" is determined for the characteristic during the result recording process of the inspection lot.

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Environment

SAP S/4HANA Cloud Public Edition

Resolution

During result recording, the system applies defect classes based on the following logic:

  • If a defect type is specified in the inspection plan characteristic, the system selects the associated defect class. This can be checked in Edit Code Groups app for catalog 9.
  • If no defect type is specified in the inspection plan characteristic, the system proposes a default defect class from plant-level settings. This can be checked in configuration activity 102456 - Maintain Settings at Plant Level .
  • For qualitative characteristics with attribute codes, the system uses the defect class from the selected set. This can be checked in Edit Selected Sets app for catalog 1.
