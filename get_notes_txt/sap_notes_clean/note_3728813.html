Symptom

  • The BAdI method BEFORE_PRINT used in VSS print programs does not allow raising an exception.
  • When implementing this BAdI, it is not possible to properly interrupt the print process if an application error occurs.
  • Errors detected during the execution of the BAdI implementation cannot be propagated to the calling print program, which may lead to incorrect or incomplete printouts.
