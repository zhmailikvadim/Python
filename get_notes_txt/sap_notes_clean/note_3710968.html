Symptom

'WBSElementExternalID' can be found from 'EWM Staging fulfillment' and 'EWM Stock Change' messsage, however the value is empty in Manage Floor Stock app

Environment

  • SAP Digital Manufacturing 2602

Reproducing the Issue

  1. Create and Release the Production Order in S4H and Download it to DMC
  2. Create staging Request for materials in DM and sent it to EWM
  3. EWM gives materials from the storage to DM PSA zone and 2 integration messages created with information regarding<WBSElementExternalID>XXXX</WBSElementExternalID>: EWM Staging fulfillment, EWM Stock Chang
  4. Open the Manage Floor Stock app and see that the custom field "WBS_element_df" is not populated with WBS-element value 

Cause

standard behavior

Resolution

WBS is currently not supported in SAP Digital Manufacturing Cloud (DMC)

Keywords

WBS element, DMC, EWM, inventory ID, stock change
