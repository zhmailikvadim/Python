Symptom

Electronic Document Service (EDS) processes and communicates electronic documents and reports for SAP Business One and SAP Business One, version for SAP HANA.

Electronic Document Framework (EDF) generates electronic documents and electronic reports for SAP Business One and SAP Business One, version for SAP HANA.

Electronic Document Monitor (EDM) is an interactive report where user is able to see status and details of electronic documents and also to perform various contextual actions on them.

Electronic Document Import Wizard (EDIW) enables loading of external electronic documents from various sources for relevant protocols, including XML files and specific web services.

EDS and EDF work together in varying degrees depending on localization and electronic document and reporting protocols. EDM monitors overall status of electronic documents processed by EDF and EDS.

Changes are required for EDS, EDF, EDM and EDIW.

More information about the initial release and initial required setup of EDS with SAP Business One version 10.0 Feature Package 2008 is described in SAP Note 2952067. More information about the subsequent changes in EDS and EDF are described in SAP Notes 2983713, 3024085, 3057499, 3090810, 3112629, and 3130712.

Solution

SAP intends to provide a feature package or patch to solve the symptom that is described in this SAP Note. The section References below lists the related feature package or patch once they become available. Be aware that References can only be confirmed at feature package or patch release date. SAP delivers feature packages or patches only for selected releases at its own discretion, based on the business impact and the complexity of implementation.

 

Unique Index in Table ECM2

The column GUID in table ECM2 stores unique identifiers for electronic documents. A new unique index is introduced to table ECM2, consisting of this particular column only. The index prevents data inconsistencies that occurred infrequently in table ECM2 in the past. Now, each record in table ECM2 must contain a value in column GUID ("Null" is not allowed) and this value must be unique in ECM2. During the upgrade process:

  • Missing GUID values are generated in ECM2.
  • Duplicate GUID values are modified to become unique. Duplicates exist for only very specific document imports, when either original imported document drafts are marked as removed in table ECM2 or imported documents are un-linked from records in table ECM2.

For more information, see SAP Note 3149543.

 

Certificate Selection for XML Sender and XML Connector Processing Targets

In the Mexico localization of SAP Business One, for the protocol CFDI in Document Settings (follow path Administration → System Initialization → Document Settings → Electronic Document → CFDi → Processing Target) you can select a certificate for the following Processing Targets:

  • XML Connector
  • XML Sender

 

More Descriptive Error Messages in EDS

For several scenarios, EDS provides extended error messages in the logs, for example when importing certificates. More details are provided in extended messages, to enable easier investigation when problems occur.

 

Tax Service Through EDS in US Localization

In the US localization of SAP Business One, tax rates are obtained from external providers through specialized EDS connectors. Tax rates are kept up-to-date by the external providers and are specific to addresses and dates. Tax Service provides current tax rates.

For more information, see SAP Note 3194512.

 

Custom Search in Electronic Document Monitor

A new tab Custom Search is available in EDM. On Custom Search, you can search or filter existing electronic documents through selection criteria that are defined in custom filters:

You can define up to 4 custom filtering criteria and after selecting Refresh, EDM displays the relevant electronic documents that meet the criteria simultaneously. Filtering conditions can use all columns from table ECM2. After closing and reopening EDM, previous filtering criteria are restored. All filtering criteria can be erased by selecting You Can Also → Clear Filter.

 

Allow EFM File Encoding from UTF-8 to ANSI Thai

Files that are generated by EFM can be saved in ANSI Thai encoding. For more information, see also Customer Influence improvement request 250430.

 

Updated Type of Column ActType in Table ECM2

The column ActType in table ECM2 is updated to INTEGER from CHAR(1) for type in company databases. Related DI API EcmAction.Type and EcmActionTypeEnum are not updated and existing DI API scripts do not need to be recompiled.

 

Update from .NET Core 3.1 to .NET 6

EDS runtime (.NET SDK version) is updated from .NET Core 3.1 to .NET 6, which Microsoft defines as a Long Term Support (LTS) version.

 

EDS Memory Consumption and Performance with 100+ Company Databases

When EDS handles more than 100 company databases simultaneously, memory consumption and late-response issues may occur. These issues are partially addressed, and after EDS is running for some time (for example, more than 30 minutes), memory consumption and response time are stabilized.
