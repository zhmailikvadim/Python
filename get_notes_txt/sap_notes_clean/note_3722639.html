Symptom

************************************************************

WARNING: This is a CA-FL (Field Logistics) specific note. If you DON'T have

Field Logistics activated in your system, this note does not apply to you.

************************************************************

When performing a two-step goods receipt (GR), the first step using movement type 103/107 posts successfully in MIGO. However, for a purchase order that does not have an inbound delivery, the system prevents the posting of the second step (movement type 105/109) through the Process Receipts application.

Other Terms

CA-FL-GR, Field Logistics, 2 Step GR, 103, 105, Movement Type, PR app, GR.

SAP S/4HANA OP 2023 FPS02, OP 2023 FPS03, OP 2023 SPS04, OP 2023 SPS05

Reason and Prerequisites

 Program error.

Solution

 Kindly implement the attached correction instructions or install the respective support package.
