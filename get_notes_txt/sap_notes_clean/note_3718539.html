Symptom

 Non Admin users able to login to AdminTools

Other Terms

AdminTools, Non-Admin, Session, Active

Reason and Prerequisites

  1. Create a user (Test) do not make it a part of administrators group.
  2. Attempt to access http(s)://<Host>:<Port>/AdminTools/   from the browser. Provide the same system, username(Test) and password used for SAP BOBJ. Observe that login failed error with detailed error message
  3. Open a new tab on same browser login to CMC with Test user
  4. Come to the admin tool tab again and update the URL tohttp(s)://<Host>:<Port>/AdminTools/querybuilder/ie.jsp /Click the "back button" or "refresh" the error page
  5. Check you can access admin tools 

Solution

This issue is fixed in the patches listed in the "Support Packages & Patches" section below.

The "Support Packages & Patches" section will be populated with the relevant patch levels once they are released.

For Business Intelligence Platform maintenance schedule and strategy see the Knowledge Base Article 2144559 in References section.
