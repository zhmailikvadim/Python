Symptom

Error messages logged by the API JournalEntryBulkClearingRequest_In do not provide information about the specific Accounts Payable/Accounts Receivable (AP/AR) items that triggered the actual error.

Reproducing the Issue

  1. Use the Journal Entry - Clearing API: JournalEntryBulkClearingRequest_In for posting settlement against Payable or Receivable invoices.
  2. Attempt to clear several AP/AR line items.
  3. Observe that error messages logged by the API do not contain information about the specific AP/AR items that triggered the error.

Cause

It is not possible to locate the error message at the line item level because the journal entry has not yet been generated at the given point in teme.

Resolution

Alternatively, use the Fiori app "Post Incoming Payments" or "Post Outgoing Payments" for clearing operations.

See Also

Refer to: Post Outgoing Payments | SAP Help Portal.
