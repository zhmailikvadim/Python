Symptom

You have data in form sections and still is not generated Insights, you receive error message:

"Insights cannot be generated for this section because there is not enough data available. If insights were expected here, please contact your system administrator for assistance"

Environment

SAP SuccessFactors Performance Management

Reproducing the Issue

Click to generate Insights inside Performance form using feature:

https://help.sap.com/docs/successfactors-release-information/8e0d540f96474717bbf18df51e54e522/43e14337668748aaa97b482762065f38.html?locale=en-US

Resolution

We confirmed that generating insights requires concise, meaningful feedback containing more than a single attribute. The requirement is not based on word or character count, but on the quality and depth of the information provided.

For example, a comment such as “employee is the best” does not provide sufficient context for the AI to generate a meaningful insight. However, feedback that includes multiple relevant attributes (e.g., strengths and specific qualities) provides enough information for the system to generate a complete insight.

Additionally, the AI requires contextually relevant information. A comment may contain many words, but if the content does not align with performance-related insights, the system will not generate any entries.

Please also note that AI-powered insights currently support only data entered in:

  • Goal item/section comments

  • Competency/Skill item/section comments

  • Summary section

Custom elements, such as text areas, are not supported at this time.

In this case, the form does not contain data within the supported sections, which is why no insights are being generated.
