Symptom

  • After the 1H 2025 release, gaps btween the results are observed when searching for any employee or actions in the preview instances in DC 55.
  • The issue is observed only when the check tool is the current page opened in the browser.

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental. 

Environment

SAP SuccessFactors HCM Suite 

Reproducing the Issue

  1. Open the check tool in the instance.
  2. Perform a search for any employee or action using the global search bar.
  3. Observe the gap between the results.

Cause

The issue is identified to be specific to customers in DC55, in preview environments. The UI isn't rendering correctly, causing gaps in the search results.

Resolution

A patch is currently planned for 05/25 to fix this issue.
