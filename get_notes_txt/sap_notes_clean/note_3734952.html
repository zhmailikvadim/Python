Symptom

Address Number (EKKO-ADRNR) is populated in purchasing contracts (Outline Agreements) created or updated via integration with SAP Ariba, even though no manual address change is performed in the system. 

Environment

  • Materials Management (MM)
  • SAP ERP
  • SAP S/4HANA

Cause

Program error.

Resolution

Implement SAP Note 3362919 and validate the behavior using fresh test data.

 
 
 

Keywords

ECC, S4HANA, purchase, order, PO, address, ADRNR, number, change, update, save, saved, vendor, EKKO, ADRC
