Symptom

Symptom

Due to an incorrect scope assignment of the technical job definition APS_YIO_ORCHESTRATOR_RUN, the technical job repository may display an inconsistent state for the corresponding technical job definition.

Reason and Prerequisites

The scope of the technical job definition APS_YIO_ORCHESTRATOR_RUN was incorrectly assigned as scope‑independent instead of scope‑dependent.

Solution

To resolve the issue, install the SAP Note and perform the required manual activities.

Other Terms

RSPM_NEW_ERRORLOG_DELETE; RSBERRORCROSS; RSBTERRORCROSS; RSBERRORCROSSREQ; RSBTRRORCROSSREQ; RSBKCHECKBUFFER

Reason and Prerequisites

Progam error

Solution

To solve the issue please apply the Support Package as mentioned below. In urgent cases you can use the correction instructions.
