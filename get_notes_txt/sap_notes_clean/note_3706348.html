Symptom

  • In Disputes, there are two different statuses: "New" and "In Use."
  • Depending on the status, the fields that can be edited will vary.

Environment

  • SAP Sales Performance Management
  • Incentive Management



Cause

The status of a dispute template determines which fields can be modified. Templates marked as "In Use" have restrictions to maintain data integrity for active disputes.

Resolution

Actions Based on Dispute Status:  

  • For a dispute with "In Use" status:
    • Existing reason codes cannot be changed (they are locked).
    • New reason codes can be added.*
    • Descriptions can be modified.
    • The option to allow or disallow the creation of new disputes of this type can be enabled or disabled.
  • For a dispute with "New" status:
    • Additional Settings can be changed
    • Workflow Settings can be changed
    • Reason codes can be changed

* If the field to add a new Reason Code for a Dispute with status "In Use" is not available, please perform the steps below:

  1. Navigate to Incentive Management Admin.
  2. Select Plan Communicator.
  3. Click on "Disputes".
  4. Open the desired dispute template.
  5. Scroll down to the "Reason Code Configuration" section.
  6. Locate the two fields: "Code" and "Description".
  7. Click in the last box under "Description" – a new box to add a reason code will be displayed.
