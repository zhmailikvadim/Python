Symptom

While assigning a serial number with ID DEF for a product XYZ in the pick task you receive an error message. *DEF refers to SerialNo, XYZ refers to Product id.

Serial Number DEF already used.

Environment

SAP Business ByDesign

Reproducing the Issue

  1. Go to Outbound Logistics Work center.
  2. Navigate to Task Control view.
  3. Search for the relevant pick task and choose the Confirm button.
  4. In Confirm Pick Task screen select the product XYZ and enter the serial number DEF in Serial Numbers sub-tab.
  5. You receive above mentioned error message:

Serial Number DEF already used.

Cause

The serial number with ID DEF have been used for the same product XYZ in another line item of the same pick task.

Since you cannot ship a product with same serial number more than once in the same delivery you receive the reported error message.

Resolution

You will have to use a different serial number other than DEF for the product XYZ in the pick task.

Keywords
