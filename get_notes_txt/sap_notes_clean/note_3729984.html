Symptom

  • Cannot post Debit Memo Request in DP90.
  • Error message V1028: “Material has status: End of Life.”

Environment

  • Sales and Distribution (SD)
  • SAP ERP
  • SAP ERP Central Component
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Maintain a 'DChain-spec. status' in the Material Master and set the 'Valid from' as 01.02.2026 for example
  2. Open transaction DP90
  3. Enter a sales order and continue.
  4. System issues error V1028: “Material has status: End of Life.”

Cause

The status check (include: LV45EU02) uses the requested delivery date (VBAK-VDATU) as the reference.

Where:

FVMSTA = status
FVMSTD = status valid-from date
FSPERRDATUM = requested delivery date

If the material status valid-from date (FVMSTD) for “End of Life” is earlier than or equal to that requested delivery date, the system determines the status will be active on that date and raises V1028.

Resolution

Set the sales order's requested delivery date earlier than the “status valid from” date to prevent the error from being triggered.
