Symptom

The change record remote API fails when change number is created or updated as a result of changes in change record items.

Other Terms

PLM-CR, ECM, Change Number, A_CHANGERECORD, Remote API, API, Dumps, Behavior Illegal, RAP Contact Violation

Reason and Prerequisites

Program Error.

Sequence of SAP Note Implementation:

Implement the below mentioned SAP Notes in the same sequence one by one.

SAP Note #1: 3676701(Change Record Remote API - API Context Setting)

SAP Note #2: 3714980(Current SAP Note: Change Record Remote API - Restrict Events)

SAP Note #3: 3704201(Change Record Remote API - Raise Events)

Solution

Use transaction SNOTE to implement the corrections delivered with this SAP Note in the affected Change Record system or import the relevant Support Package.
