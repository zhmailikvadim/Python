Symptom

In the process "Device Installation/Replacement/Shutdown" (DE_B_MCM_DEV_INSTLN_PROCESS), the system creates a duplicate step 0060. This duplication causes the process to fail. The issue affects already-running production processes that are mirrored to test environments.

This issue is relevant for the market role: DSO.

Other Terms

MCM_DE, DE_B_MCM_DEV_INSTLN_PROCESS, MELO, device installation, device replacement, device shutdown, duplicate step, process step 0055, process step 0060

Reason and Prerequisites

The error occurred because of a program error.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the system no longer creates a duplicate step 0060, preventing the process failure. The BC set _US4G_MCM_3_0_2602_3730877 contains the necessary corrections.

You need to retrigger the process. You can verify the fix by checking that the process completes successfully without encountering an error in step 0060.
