Symptom

In the Manage Product Master Data application, the Purchasing Text is not being copied from the source material to the target material.

  1. During the copy process from the display page, the system raises the warning:
    “No authorization to edit the note type ‘Purchasing Texts’.”

  2. During the copy process from the search page, no error or warning is raised.

However, in both situations, the Purchasing Text is not getting copied.

Other Terms

F1602, STXH, STXL, BEST, MATERIAL

Reason and Prerequisites

Program error.

Solution

implement this note.
