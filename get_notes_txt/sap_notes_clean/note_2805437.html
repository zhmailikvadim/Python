Symptom

  • The text label for Confirm New Password is cut off when using Japanese locale
  • In Japanese locale, for Confirm New Password, it is shown Japanese as 新しいパスワードの確認
  • However, in the Web page, it only displays like 新しいパスワードの確 and there is not enough to display all the characters.

Other Terms

BI, BILaunchpad, launchpad, CMC, change, password, PWD

Reason and Prerequisites

Environment:

  • SAP BusinessObjects Business Intelligence Platform (BI) 4.2

Steps to reproduce:

  1. Log on to Central Management Console (CMC), and create a new Enterprise user.
  2. Make sure that User must change password at next logon is checked on.
  3. Launch browser like Internet Explorer 11, set the locale as Japan (Japanese).
  4. Log on to BI launch pad or CMC using this new user.
  5. Notice that the change password page comes. And the label of 新しいパスワードの確認 is cut off as there is no enough space to display all the characters.

Solution

This issue is fixed in the patches listed in the "Support Packages & Patches" section below.

The "Support Packages & Patches" section will be populated with the relevant patch levels once they are released.
