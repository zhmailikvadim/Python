Symptom

The translation date for the postings during the IHB daily closing process is incorrect.

For example, the posting date is February 27, the translation date should also be February 27. However, the translation date was set to March 1.

As a result, the March exchange rate was applied instead of the February rate, which led to an incorrect amount being recorded.

Other Terms

 /PF1/R_IHB_GL_TRANSFER

Reason and Prerequisites

Program error.

Solution

Apply the program correction of this note.
