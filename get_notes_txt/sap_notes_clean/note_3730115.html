Symptom

  • Health Monitoring displays only Heartbeat metrics for on‑premise systems.
  • ST‑PI of the managed system is SP33.
  • The metric RFC Connection Status (ABAP.rfc.monitoring) has been configured.

Environment

  • SAP Cloud ALM

Reproducing the Issue

  1. Patch the managed on‑premise ABAP system to ST‑PI SP33.
  2. Configure the RFC Connection Status metric (ABAP.rfc.monitoring) in Health Monitoring.
  3. Open Health Monitoring and observe that only Heartbeat metrics are shown for the system.

Cause

A newly introduced metric RFC Connection Status(ABAP.rfc.monitoring) causes issues that lead to only Heartbeat metrics being displayed.

Resolution

  1. The development is working on the fix for this problem and this KBA will be updates once the correction is delivered.
  2. As a workaround until the corrections are delivered, delete the existent configuration for RFC Connection Status (ABAP.rfc.monitoring) metric in Health Monitoring application.
