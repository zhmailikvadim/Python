  • Observer a runtime error -

Environment

  • SAP NetWeaver 7.3
  • SAP enhancement package 1 for SAP NetWeaver 7.3
  • SAP NetWeaver 7.4
  • SAP Netweaver 7.5
  • Reported case was in SAP NetWeaver Enterprise Portal 7.40
  • May occur in other releases

Reproducing the Issue

  1. Call the Enterprise Portal as the System Administrator
  2. Navigate to System Administration -> Transport -> transport Packages -> Import
  3. A runtime error appears on-screen

Cause

  • The import service has encountered some issues or has stopped responding
  • In the reported case restarting the import service resolved the issue

Resolution

  1. Call the NetWeaver Administrator (NWA) as the administrator user
  2. Navigate to Configuration -> Start & Stop > Java Applications
  3. Copy and paste 'com.sap.portal.pcd.admintools.import' into the field beside the funnel

  4. Press Enter
  5. Highlight 'com.sap.portal.pcd.admintools.import'
  6. Click on 'Restart'
  7. Navigate to the Import functionality again
