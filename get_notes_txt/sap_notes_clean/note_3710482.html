Symptom

An error 'No SOA service name is defined for interface KR_SEND_INVOICE_REQUEST' occurs while submitting an e-invoice for South Korea.

Environment

SAP S/4 HANA Cloud Public Edition

Reproducing the Issue

  1. Open the eDocument Cockpit App.
  2. Search for eDocument Source Document.
  3. Choose the button Submit, then you will observe the error message: 'No SOA service name is defined for interface KR_SEND_INVOICE_REQUEST'.

Cause

Missing Configuration of integration scenarios using the integration package SAP Document and Reporting Compliance: Electronic Invoicing for South Korea.

Resolution

It is necessary to configure integration scenarios using the integration package SAP Document and Reporting Compliance (DRC): Electronic Invoicing for South Korea. Detailed guidance on this setup can be found in the SAP Help Portal under the section Integrating With SAP Integration Suite.

For your convenience, here is the documentation link: Documents | SAP Document and Reporting Compliance: Electronic Invoicing for South Korea | SAP Business Accelerator Hub

Please ensure that the required services are set up to establish a connection between SAP DRC and the application service providers. Once this configuration is complete, kindly attempt to submit the e-invoice again.
