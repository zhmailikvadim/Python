Symptom

Component: Product Safety
Module: Specification Management

You are working in the specification workbench (CG02). You output the characteristic values for a specification in an Excel table using an output variant. Some of the characteristic data to be output consist of phrases. You want to output the phrases with the phrase key to Excel. This is not possible.

Other Terms

Export to Excel

Reason and Prerequisites

Reason
Program error.

Prerequisites
For information about the validity of the corrections, see the correction instructions.

Solution

After implementing this note, you have the following choices:

  • Output the phrase text
  • Output the phrase code
  • Output the phrase code and text, separated by a colon (:)
  • Output the phrase key

The first three choices were already possible before. This note adds the possibility to output the phrase key (<Phrase library>-<Phrase>).

The section "Support Packages" specifies the support packages that contain the corrections.

Alternatively, you can implement the attached correction instructions.
