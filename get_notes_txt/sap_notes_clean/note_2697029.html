Symptom

You cannot download the output file after applying SAP note 2681329.

Other Terms

&PREVIRED (further information see SAP Note 1730752);
PC00_M39_RPRV;

Reason and Prerequisites

The following Support Package level is the minimum prerequisite:

Release Support Package
604 SAP_HRCCL SP B2
608 SAP_HRCCL SP 39

(For more information, click "HR Support Package Schedule" in support.sap.com/maintenance-schedule)

Solution

Version Date Comments
2 October 15, 2018 Note released for customers

After applying this note the download of the output file will work as expected.

As general rule, SAP recommends that you install a solution by applying a Support Package. However, if you need to install the solution earlier, proceed as follows:

    1. Use Note Assistant (SNOTE) to implement the correction instructions

You can find more information about the Note Assistant in SAP Support Portal, under support.sap.com/kb-incidents/notes/assistant.html.
