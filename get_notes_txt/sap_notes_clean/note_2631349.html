Symptom

There is no reference at charge posting order available within the expert user interface (/PE1/PO_EXPERT) that connects the charge order to the item for which the charges where determined.

Other Terms

Reference, Charge Settlement, CHRG_PO.

Reason and Prerequisites

Missing functionality.

Solution

Preliminary correction according to the correction instructions included in this SAP Note or via Support Package.

In detail, the ordering party item of the charge posting order will be linked its predecessor.
