Symptom

This SAP Note provides you with corrections to the Peru Inventory and Balance Ledger (PE_INVENTORY_BALANCE_LEDGER) report in the Run Statutory Reports app for SAP Document and Reporting Compliance. It addresses the incorrect amounts reported in the Balance Sheet Accounts - Assets, Balance Sheet Accounts - Liabilities, Losses, and Profits fields in the electronic accounting book 3.17 Inventory and Balance Ledger (SUNAT code 031700).

When you run the PE_INVENTORY_BALANCE_LEDGER report, the system displays incorrect amounts in the Balance Sheet Accounts - Assets, Balance Sheet Accounts - Liabilities, Losses, and Profits fields.

Other Terms

Peru, Document and Reporting Compliance, DRC, fka ACR, Inventory and Balance Ledger, Libro de Inventarios y Balances, PE_INVENTORY_BALANCE_LEDGER, 031700

Reason and Prerequisites

This SAP Note is relevant only for Peru.

You have implemented previous notes listed in SAP Note 3206609.

Solution

After implementing this SAP Note, the system displays the correct amounts in the Balance Sheet Accounts - Assets, Balance Sheet Accounts - Liabilities, Losses, and Profits fields in the electronic accounting book 3.17 Inventory and Balance Ledger (SUNAT code 031700) within the PE_INVENTORY_BALANCE_LEDGER report.

SAP recommends that you install a solution by applying a Support Package. However, if you need to install a solution earlier, use the Note Assistant to implement the correction instruction. You can find more information about the SAP Note Assistant in the SAP Support Portal, under SAP Note Assistant.

To implement the solution, you must carry out manual activities and implement the correction instructions that are valid for your release.

Documentation for SAP S/4HANA for Peru is available in the system and in the SAP Help Portal. System documentation (such as field help, customizing, messages) is delivered via Support Packages. SAP Help Portal documentation is available at help.sap.com/viewer/s4h_op_peru.
