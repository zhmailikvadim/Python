Symptom

SAP mobile development kit client

Other Terms

Release, features, fixes, mdk, client, patch, sdk, iOS, Android, tablet, phone

Reason and Prerequisites

SAP Mobile Services offers a major revision to the mobile development kit client.

This release provides the following new features and enhancements on top of regular bug fixes.

  • Support Onboarding via Mobile Device Management (MDM) Application Configuration
  • Support Required Indicator for Segmented Form Cell Control
  • Support Display of user initials in Contact Cell
  • Added BadgeImage support on the Contact Cell
  • Added styling to `KeyValueItem` control
  • Enhanced ActionBar Caption Size Options
  • Added In-App camera option for AttachmentFormcell on Android
  • Support Offline OData Database upload
  • Updated sap-icon with latest icon set

Platform upgrades

  • Upgraded SAP BTP SDK for iOS to version 25.11.5
  • Upgraded SAP BTP SDK for Android to version 25.11.2
  • Upgraded NativeScript to version 9.0.4 
  • Upgraded XCode to version 26.3

This release of the mobile development kit client was verified on desktop web browser, tablets and phones.

Minimum OS versions:

  • iOS/iPadOS 26.x (It has been tested with Xcode 26.3 and iOS/iPadOS 26.3)
  • Android OS 9.0 or newer (It has been tested with Android 15 and 16)

Solution

For Custom MDK Mobile Client developer, download and use the latest MDK Client SDK installer from SAP Store, Service Marketplace.

For SAP Mobile Services Client user, download the latest SAP Mobile Services Client from the Apple AppStore or Google Play Store.
