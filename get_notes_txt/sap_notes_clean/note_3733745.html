Symptom

No option to view the deleted Sales Quotes 

Environment

SAP Business ByDesign

Reproducing the Issue

  1. Go To the New Business Work center
  2. Select the Sales Quotes View
  3. From Drop Down choose "All Quotes"
  4. Search with your sales quotes ID in the search tool bar
  5. Expected Sales Quote ID not found

Cause

This is the expected system behavior.

Resolution

The Sales Quote ID with status "Open" only can be deleted and there is no option to view the deleted sales quotes.
This is the current system behavior.

Keywords

 Sales quotes, status Open, Deleted, No Option
