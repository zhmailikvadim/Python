Symptom

You run HR-HK: Periodic MPF contribution report (format: eMPF) (HHKCPFE0) report to generate eMPF data for employees. Previously, report reads HKID for permanent residents and non-permanent residents from a subtype in the Personal IDs (0185) infotype. The subtype is configured at position 2-3 with the format "E<subtype>" in feature HR-HK: Configurable elements for IR56 reporting (27IRI).

This SAP Note supports to read HKID for non-permanent residents from the subtype configured at position 4-5 in feature 27IRI.

Other Terms

HHKCPFE0, eMPF, HKID, IT0185, 27IRI, Idnum

Reason and Prerequisites

This is a correction of legal function.

Solution

Apply the Support Packages or implement the Correction Instructions.
