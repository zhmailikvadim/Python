Symptom

You have installed UI Data Protection Logging for SAP S/4HANA.

The Product Version (PV) can be:

  • 1811 with software component version LOGS4H 100 or
  • 2011 with software component version UIDP 100.

In order to use the functionality for logging of SAP Fiori and UI5 (OData) applications, it is required to implement this modification.

Note: This SAP Note is referred to in Consulting Note 2605075 - Hooks for UI Data Protection Logging for SAP S/4HANA.

Other Terms

LOGS4H, OData Logging, Fiori Logging, UI5 Logging, SAP Gateway 2.0

Reason and Prerequisites

Software Component (SWC) LOGS4H and UIDP belong to the Add-on product UI Data Protection Logging for SAP S/4HANA that requires some hook points in standard coding.

Solution

Implement the automatic correction instruction provided.
