Symptom

The following ORA-07445 error was reported in the trace file:

> ORA-07445: exception encountered: core dump [PC:0x7FFEF55AF47C] [ABORT_PROCESS_TERMINATED] [] [PC:0x7FFEF55AF47C] [] []

Environment

  • Oracle database 12.2.0.1
  • Oracle database 18c
  • Oracle database 19c

Cause

In general, [ABORT_PROCESS_TERMINATED] is output when system resources are exhausted.

Example trace file:
Dump file F:\ORACLE\XXX\SAPTRACE\diag\rdbms\xxx\xxx\incident\incdir_188166\xxx_j000_6520_i188166.trc
Oracle Database 19c Enterprise Edition Release 19.0.0.0.0 - Production

Process Affinity    : 0x0000000000000000
Memory (Avail/Total): Ph:2988M/12287M, Ph+PgF:1M/32767M

The above example shows that the available page files (virtual memory) in the Oracle Database were "1MB".

Resolution

This is due to insufficient memory resources. Please secure enough memory resources.
