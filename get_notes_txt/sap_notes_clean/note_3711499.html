Symptom

Unassigned cost component shown in order cost reports app, such as 'Analyze Production Costs - Event Based' and 'Order Cost Details - Event Based'.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Open the app "Analyze Production Costs - Event Based".
  2. Navigate to the order's cost section.
  3. Observe that the cost components are displayed as '000(Unassigned)'.

Cause

Resolution

Map the G/L accounts under cost component structure in SSCUI 102427 - Define Cost Component Structure.
