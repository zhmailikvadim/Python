Symptom

When making changes to position from Manage Position, an error 'Saving of one position throwed a Exception' is displayed.

Environment

SAP SuccessFactors Employee Central: Position Management

Reproducing the Issue

  1. Login to instance.
  2. Open a position in Manage Positions.
  3. Make some changes and save.
  4. Error is displayed.

Cause

Issue is caused by a business rule which sets the matrix relationship to a value that already exists in the position.

Resolution

Add an additional IF condition in the rule to check if matrix relationship is Null and set a value only if there is no existing relationship of the same type and value.

Keywords

Saving of one position throwed a Exception, position error, duplicate matrix relationships, unique constraint violation, matrix relationships, error, duplicate value, existing matrix relationship
