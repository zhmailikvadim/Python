Symptom

Events are stuck in status 'In Flight' and there is an exception in transaction /iwxbe/error_log stating: Internal Error in Enterprise Eventing. When clicking on that entry, the child log states:
"Insufficient Flow Cntl window or link credits".

Other Terms

OPU-XBE, Enterprise Event Enablement, In Flight, Insufficient Flow Credits, Flow Cntl, Stuck

Reason and Prerequisites

There is an internal buffer overflowing caused by insufficient flow credits. Events that are not properly sent because of this error get stuck and are not retried.

Solution

Implement the Correction Instructions and restart the channel by right clicking it in /iwxbe/config and choosing "Reactivate Channel"
