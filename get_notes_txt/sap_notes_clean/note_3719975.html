Symptom

In BIB replication, Head of own organizational unit checkbox is not updated properly in transaction PPOME.

Environment

  • SAP SuccessFactors Employee Central integration to SAP Business Suite
  • SAP SuccessFactors Employee Central integration with SAP Human Capital Management for SAP S/4HANA
  • Org. object replication 
  • Business Integration Builder (BIB)

Reproducing the Issue

  1. Run org. object replication for position object. 
  2. Check the position in PPOME.

Cause

S - A003/A012 relationships are mapped to different O objects. 

Resolution

In template group configuration, infotype 1001, subtype A003(B003) and A012(B012): EC field ID should come from the same data source (Or EC field ID should be the same value).

Then the system will create S - A/B003 - O and S - A/B012 - O relationships. The O objects in these relationships are the same value.
