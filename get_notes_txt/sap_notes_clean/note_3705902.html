Symptom

When creating a Ship Notice (ASN) directly from the Purchase Order (PO) page in SAP Business Network, the system generates a single ASN with the cumulative quantity of all firm lines. However, I need to create separate ASNs for each firm line. Is there a way to achieve this from the PO page, or is there an alternative method to create individual ASNs per firm line?

Environment

SAP Business Network

Cause

This behavior is standard in SAP Business Network. When ASN creation is initiated from the PO page, the system aggregates the quantities of all firm lines and generates a single ASN. The PO page does not support creating separate ASNs for individual firm lines.

Resolution

To create separate ASNs for individual firm lines, use the Items to Ship tab:

  1. Navigate to the Items to Ship tab.

  2. Select the required firm line.

  3. Create the ASN for the selected firm line.

The Items to Ship tab supports line-level selection and allows creation of separate ASNs for each firm line.
