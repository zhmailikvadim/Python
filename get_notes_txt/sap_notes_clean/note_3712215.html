Symptom

Business user wants to schedule application job or create application job template from global templates of order-based planning, but the user is not able to see them.

Environment

SAP Integrated Business Planning

Reproducing the Issue

Use standard template "Order-Based Planning: Deployment Run" as an example:

  1. Go to Application Jobs app or Application Job Templates app.
  2. Create a new job or job template.
  3. While selecting template, user cannot find Order-Based Planning: Deployment Run (/IBP/RM_DEPLOYMENT_RUN_V2).
  4. Similarly, user is not able to see other existing templates that include this standard template.

Cause

Business catalog "SAP_IBP_BC_RMAPPLJOB_PC" is not assigned to the issued business user.

Similarly, other affected global templates are: 

  • Order-Based Planning: Activate Allocation Profile
  • Order-Based Planning: Activate Planning Level
  • Order-Based Planning: Copy Version Data
  • Order-Based Planning: Delete Version Data
  • Order-Based Planning: Supply Planning Run
  • Order-Based Planning: Load Planning Run information into Key Figures
  • Order-Based Planning: Unload or Delete Load Requisitions

Resolution

Assign the business catalog "SAP_IBP_BC_RMAPPLJOB_PC" to the affected users via their respective business roles.

Keywords

Application Job Template, Business Catalog, SAP_IBP_BC_RMAPPLJOB_PC, Deployment run, Application Jobs, Job Scheduling, Global Template, Order-Based Planning
