Symptom

Add-on RTLRAP 100 uninstallation with transaction SAINT or, during an upgrade, with the Software Update Manager (SUM). Here, the process with SAINT is described.

Other Terms

SAINT, add-on, RTLRAP, SAPK-10000INRTLRAP

Reason and Prerequisites

You have installed CARAB 1.0 as an embedded deployment along with ECC or S/4HANA in the same system. There are restrictions on installation / upgrade of Add-ons in those systems which requires a de-install of the corresponding components.

Solution

Please implement the attached correction instructions
