Symptom

Indonesia BPPU and BPNR withholding tax reports are failing during CoreTax validation due to invalid country code, incorrect tax rate format, and document description in the generated legal XML output.

Other Terms

Withholding tax, Indonesia, BPPU, BPNR, CoreTax, Tax rate, and Document.

Reason and Prerequisites

This SAP Note is only valid for country/region version Indonesia.

Solution

This note corrected the XML output by:

Formatting the tax rate.
Updating the country code to a valid 3-character code.
Updating valid document description as per CoreTax requirements.
