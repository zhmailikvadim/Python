Symptom

You are using SAP S/4HANA Migration Cockpit to do data migration. Relationships for object type BP (business partner) and CP (central person) is not generated after data migration to S/4HANA.

Environment

SAP S/4HANA

Resolution

Relationships for object types BP (business partner) and CP (central person) are not supported by Migration Cockpit object "HCM - Relationship" as per SAP Help Portal HCM - Relationship.

Execute the report /SHCM/RH_SYNC_BUPA_EMPL_SINGLE for all migrated employees to generate relationships for object types BP (business partner) and CP (central person).
