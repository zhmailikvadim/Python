Symptom

You are working with Hedge Management with parallel valuation area hedge accounting. When creating a Dedesignation or SWAP hedge request, users must assign requested amounts to hedging relationships. Users previously had to switch between different valuation areas to complete the assignment process. This resulted in a cumbersome and non-intuitive user experience.

Other Terms

TOENE, TOEHREQO, CL_TOE_HREQ_CTRL, CL_TOE_HREQ RETRIEVE_RESERVED_VOLUME, CL_TOE_HREQ_CTRL REFRESH_HREL_ALV, UPDATE_HREQ,
LCL_HREQ_HREL, INIT, UPDATE_DATA, CREATE_FIELDCAT, HANDLE_DATA_CHANGED, HANDLE_DATA_CHANGED_FINISHED

Reason and Prerequisites

Improved usability.

Solution

The functionality has been enhanced to consolidate all hedging relationships from different valuation areas into a single ALV table. Users can now view and manage all relevant hedging relationships in one unified interface, eliminating the need to switch between valuation areas.

Please implement the SAP Note or the Support Package.
