Symptom

On the freight Document maintenance screens you are changing  the value of multiple fields including carrier, then press enter.
Charge calculation is executed on the old values instead the new values. Corresponding messages do not reflect the current state of the document.

Other Terms

ASSIGN_TSP, UPDATE_TRANSPORTATION_CHARGES, CALC_TRANSPORTATION_CHARGES, mtr, vehicle, means of transport

Reason and Prerequisites

Program error.

Solution

Implement the attached correction instructions or install the corresponding Support Package.
