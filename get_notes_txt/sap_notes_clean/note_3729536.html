Symptom

You want to create quarterly or annual employment tax notifications for a payroll area in which no employees have existed since a specific month.

Environment

  • Human Capital Management (HCM) 
  • SAP ERP Core Component 
  • SAP ERP 
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA

Solution

When you execute the report RPCTAVD0 (Employment Tax Notification ELSTER), set the invisible parameter P_VJJGEN if you want to generate quarterly (periods 3, 6, and 9) or annual (period 12) employment tax notifications without current payroll results.

This parameter ensures that the report also generates the data for the quarterly registration or annual registration (period 41, 42, 43, 44 / 19) without payroll for personnel numbers.

The parameter can be activated for these special cases using a report variant as follows:

  1. Save a variant of the report RPCTAVD0 and position the cursor on the field name P_VJJGEN.
  2. In the "Selection Variable" field, select the entry T ("Table Variable from TVARVC").
  3. In the Name of Variable column (input using F4 only), choose ABAP_TRUE.
  4. If ABAP_TRUE does not yet exist in your system, create the variable with the value X (menu: Environment → Maintain Sel. Variables).
  5. After creation, ABAP_TRUE is available for selection variable T and the variant can be saved.
