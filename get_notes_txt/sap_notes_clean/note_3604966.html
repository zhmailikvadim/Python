Symptom

  • In the transaction code ME13, the Purchase Order Text appears empty.
  • The text only becomes visible after clicking on the empty box of Purchase Order Text.

Environment

  • Materials Management (MM)
  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA

Reproducing the Issue

  1. Access the inforecord ME12/ME13
  2. Goto texts
  3. Observe that the Purchase Order Text is empty.
  4. By double Click on the empty box of Purchase Order Text can only be viewed 

Cause

The issue arises due to the format key being set as "/:" instead of "*". The "/:" format key is a SAPscript control command which means the characters contained in the actual text line are not output as text but are regarded as a control command.

Resolution

  1. Access the transaction code ME12.
  2. Locate the Purchase Order Text section.
  3. Replace the format key "/:" with "*". 
  4. This change enables the text to be displayed correctly in the Purchase Order Text section.
