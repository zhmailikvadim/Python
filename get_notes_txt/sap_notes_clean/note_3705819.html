Symptom

  • Users are unable to see multiple event reasons when attempting to cancel onboarding in SAP SuccessFactors Onboarding.
  • The issue persists despite following the steps outlined in KBA 2915296 - Cancellation Reason is not listed while trying to Cancel Onboarding Process.
  • The issue affects all new users.

Environment

SAP SF ONBOARDING

Reproducing the Issue

  1. Navigate to Admin Center > Home > Onboarding.
  2. Search for the user profile (e.g., "NameTest").
  3. Select "Actions" > "Cancel Onboarding".
  4. Attempt to select a cancellation reason.

Cause

Event Reason association with the country field was not maintained, and "No Show" event reason values were not created.

Resolution

  1. Maintain the Event Reason under the country field in Manage Data.
  2. Ensure the Event Reason is associated with all relevant country objects.
  3. Create additional values in the Event Reason center to display more options in the dropdown.
  4. Refer to KBA 2897272 - No Show event reason not showing under Take Action Report No-Show - SAP for Me for further guidance.
