Symptom

  • Calculation Issue when using an Aggregation Time Valuation that contains a negative, fixed threshold.
  • Actual behavior: the system appears to ignore the negative threshold when attempting to get all hours above a negative threshold.
  • Expected behavior: the negative value should be added to the group above.

Environment

  • SAP SuccessFactors Employee Center - Time Sheet
  • SAP SuccessFactors Time Tracking

Reproducing the Issue

  1. Create Aggregation Time Valuation with a negative, fixed threshold
  2. During calculation, the system appears to ignore the fix negative threshold value when attempting to get all hours above a negative threshold.

Resolution

Due to potential side effects and associated implementation risks, this issue will not be addressed in the 1H 2026 release. Product Engineering is investigating a solution.
