Symptom

  • After executing transaction F-36, a screen titled "Overview Output Management: Output" appears;
  • Inquiry regarding disabling this screen.

Environment

SAP S/4HANA

Reproducing the Issue

  1. Execute transaction F-36.
  2. Observe the appearance of the "Overview Output Management: Output" screen immediately after posting the transaction.

Cause

This screen appears as part of the standard system functionality in SAP S/4HANA.

Resolution

  1. The screen can be bypassed by pressing the 'Back' button.
  2. Alternatively, complete the customizing for New Output Management to prevent the screen from appearing:
  • Navigate to the IMG activity and search for "Assign Form Templates".
  • Create the following entry in table APOC_C_FORM for APP Post Dated Check:
    • Application object type: FI_BILL_OF_EXCHANGE
    • Output type: BILL_OF_EXCHANGE_LOC
    • Form type: Output forms (Gateway interface)
    • Form template ID: FI_BILL_OF_EXCHANGE_LOC
