Symptom

  • While performing a schedule of a Webi report to a CSV output, the following message appears in the instance.
    • com.businessobjects.common.CdzClusterMgmt.ClusterFacadeException: IDL:CdzClusterMgmt/ClusterFacadeException:1.0
  • In the Webi Processing Server logs is possible to see the following error.
    • **ERROR:WICDZUserSessionImpl:ThrownException.errorMessage : ClusterFacadeImpl::updateUserSettings : [repo_proxy 17] SessionFacade::changeUserSettings - (SessionFacade::changeUserSettings) Sorry, you do not have the right to 'Change preferences' (ID: 84) for 'Username' (ID: 29451131). Please contact your administrator if you require this right.(hr=#0x80041e01) [WICDZUserSession_session.cpp;786]

Environment

  • SAP BusinessObjects BI Platform 4.3/2025
  • Windows/Unix

Reproducing the Issue

  1. Go to BI LaunchPad.
  2. Go to the folder section, select schedule option on a Webi report.
  3. Perform the schedule to a CSV output.
  4. Appears the following message in the instance after failed.
    1. com.businessobjects.common.CdzClusterMgmt.ClusterFacadeException: IDL:CdzClusterMgmt/ClusterFacadeException:1.0

Cause

You have a custom access level on the User/Group/Folder/File that can interfere with the schedule. 

Resolution

  1. The issue can be resolved by ensuring that there is no explicit "Deny" on the affected User/Group/Folder/File on the following rights:
    1. General - Change preferences
      General - Change preferences for objects that user owns
  2. Typically, for Administrator users and the Administrators group, these could be managed at the Users & groups top-level folder on the CMC.
    1. CMC > Users & Groups > Security at Top Level Folders > All Users
      CMC > Users & Groups > Security at Top Level Folders > All Groups

Keywords
