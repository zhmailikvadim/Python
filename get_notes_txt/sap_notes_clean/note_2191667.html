Symptom

For the transfer of the data deliveries to the health insurance funds, you still use the e-mail procedure. However, this transfer method is no longer permitted as of 1/1/2016. E-mail procedure users are informed by the receiving offices about the termination of the e-mail procedure.

Other Terms

E-mail, communication server

Reason and Prerequisites

Legal regulation:
In the fourth law for changing the fourth book of the German Social Code (SGB) and other laws, the application of the eXTra standard for the data transfer was made subject to binding regulation in all contribution and notification procedures as of January 1, 2016.

For the transfer of the data deliveries to the health insurance funds the use of the communications server was provided as an alternative to the transfer via e-mail. For newly introduced procedures, the communication server is permitted as the only transfer method. The following overview lists the procedures and the available transfer methods in the SAP system:

Solution

Convert the procedures that still use e-mail as a transfer method to the communications server at the latest by 12/31/2015. Make the relevant settings in Customizing of Payroll Germany under Communication with Authorities (B2A) -> Data Exchange with Social Insurance Agencies -> Settings for the Applications. The settings for the communication procedure are defined for the individual procedures as "Data for the Reporting personnel area".

Alternatively, you can implement the check or processing directly using the view V_T596M. The affected subapplications in the personnel area Reporting are listed in the table. Check whether the transfer via e-mail is still defined in these.

Recommendation SAP: If you still use e-mail for certain procedures, convert this to the communications server immediately. You should not wait with a conversion on the fiscal year change at this time due to the pending additional legal changes.

If you already use the communications server for one or several procedures, further technical settings during the conversion of a procedure required from e-mail to communication server are not required. If you are not yet using the communications server, refer to the required steps in Customizing of Payroll: Germany under "Communication with Authorities (B2A) -> Data Exchange with Social Insurance Agencies -> Technical Basis for Communication with PKCS#7 -> Communication server of the ITSG (statutory health insurance). In addition, the SAP Notes 1791191 and 1820665 are relevant for the use of eXTra 1.3.
