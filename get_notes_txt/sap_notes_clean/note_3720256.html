Symptom

  • When a user is activated as an "XMLRPC API user" in their user settings, coupon redemption becomes impossible.
  • Instead of the field for entering the coupon number, an error message appears stating "No coupon data entered."

Environment

SAP Event Ticketing, Cloud Edition

Reproducing the Issue

  1. Log on to SAP Event Ticketing.
  2. Select the relevant company.
  3. Select the event.
  4. Add a ticket to the cart.
  5. Select "Coupon" as the payment method.
  6. Attempt to pay:
  • When XMLRPC API user is enabled: Message "No coupon data entered" appears.
  • When XMLRPC API user is disabled: Coupon number can be entered as usual.

Cause

  • The issue arises due to a combination of permissions, differing business logic, different context parameters, missing default values in the API, and technical constraints.

Resolution

  • The system behavior is as designed.
