Symptom

After importing the PR with different Requester & Preparer, either of them is not able to open the PR in Guided Buying.
User is getting System Error by saying: You can come back later and try again, or report the issue to your administrator.

Cause

Not maintaining the Partitioned user data for that user in SAP Ariba Buying causing this issue. 

Resolution

Maintain the Partitioned user data in SAP Ariba Buying to resolve the issue. 

Keywords

System Error, PR, Requisition, Preparer, Requester, Importing PR, not able to access PR.
