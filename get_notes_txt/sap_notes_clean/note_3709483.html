Symptom

When attempting to maintain data with transaction AFAMS or with IMG activity Define Multi-Level Methods in a client with another client role as Production. Information appears: Client xxx has status 'not modifiable'. Data are not modifiable.

Environment

This KBA applies to all SAP Products based on NetWeaver and S4H.

Reproducing the Issue

  1. Execute transaction AFAMS
  2. Information appears: Client xxx has status 'not modifiable'.

or

  1. Execute transaction S_ALR_87015162
  2. Information appears: Client xxx has status 'not modifiable'.

or 

  1. Execute transaction SPRO
  2. Press button SAP Reference IMG
  3. Execute IMG activity Define Multi-Level Methods
  4. Information appears: Client xxx has status 'not modifiable'.

Resolution

Follow manual instruction of KBA 3138512 - SM30 | SPRO | Client XXX has status 'not modifiable' | Client role Customizing or Test
The object names are the following:
V_T090NS
V_T090NS_1

See Also

KBA 3138512 - SM30 | SPRO | Client XXX has status 'not modifiable' | Client role Customizing or Test
