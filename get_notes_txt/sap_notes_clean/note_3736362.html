Symptom

It's not possible to export the users list from SAP Cloud ALM 'User Management' application

Environment

SAP Cloud ALM

Reproducing the Issue

  1. Access the SAP Cloud ALM tenant.
  2. Open the 'User Management' application.
  3. Observe that there is no export users option.

Cause

The 'User Management' application does not include an export user list feature.

Resolution

  1. In the SAP BTP cockpit, open the relevant SAP Cloud ALM subaccount.
  2. Navigate to Security > Users.
  3. Use one of the available options to export to CSV:
    1. Export All Users.
    2. Export All Users with Role Collection.
  4. If you cannot access or do not see the subaccount, request to be added as an Administrator for the Global and/or Subaccount, then repeat steps 1–3.
