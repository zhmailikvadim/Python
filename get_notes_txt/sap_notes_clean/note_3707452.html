Symptom

You process the external forecast inbound. You will get an error "Missing weekly timestream for start time"

Other Terms

External Forecast, /DMF/OPIF_EFCST_INBOUND, /DMF/MSG_HL 802, Missing weekly timesteream for start time

Reason and Prerequisites

Program error.

We are searching for timestream_id EQ '' in table /dmf/tstrm_week. This scenario no longer occurs in CARAB 5.0 as the timestream ID's are populated in the /DMF/TSTRM_WEEK table.

Solution

Implement the correction instruction.
