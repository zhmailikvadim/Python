Symptom

Flash player is not supported after December 31, 2020. Flash contents running in Flash player are blocked by Abode from Jan 12, 2021.

If you are using BPM and you want to know the impact on BPM.

Environment

SAP NetWeaver 7.5

SAP Business Process Management

Cause

Fix was released for higher SP Version

Resolution

For the issue with System Overview please see the following notes:

 2981308 - New implementation of Flash based NWA Applications

 2477654 - Deprecation of embedded flash content in browsers affecting System Overview in NWA

Once you fix the issue with System overview, you will be able to navigate to the BPM System Overview.

If you cannot upgrade your system according to SAP Note 2981308 at the moment, you are still able to reach the BPM System Overview page. See KBA 3042982 - BPM System Overview cannot be opened

The BPM use Adobe Flash only in Process Visualization, that is accessible from Manage Processes in NWA or tasks in UWL.

The Flash has already been replaced with Java Script implementation in 7.50 SP16 PL0.

2774717 - New features delivered for SAP Business Process Management with SAP Netweaver 7.5 SPS16

See Also

3013115 Flash Player deprecated - when and where are the new implementations of Netweaver Administrator applications
