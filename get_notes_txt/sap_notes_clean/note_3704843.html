Symptom

Header Sender Address is removed from System > Administration > Notifications > Email Server Configuration > Default and Default header sender address is updated to data-analytics@notifications.sap.com

Other Terms

SAC, Email Server Configuration, Default SMTP, Header Sender Address

Reason and Prerequisites

Due to various security reasons, header sender address support for default SMTP is stopped from 2026 Q1.

Solution

From 2026 Q1 onwards the emails from SAC would be delivered with the default SAC header sender address i.e., data-analytics@notifications.sap.com The support for header sender address support for default SMTP is stopped.

Please switch to custom SMTP if you still want to use this feature.
