Symptom

When trying to create a transaction through FTR_CREATE the other flows tab is not visible for a money market transaction. 

Environment

  • SAP S/4HANA
  • SAP S/4HANA Finance

Cause

Configuration

Resolution

  • Root cause is that no flow type with flow category TF(Fixed Amounts) is assigned to the product type.
  • Kindly make sure to assign a flow with flow category TF(Fixed Amounts) within the below customizing path to the product type and transaction type.

Financial Supply Chain Management 
-Treasury and Risk Management 
--Transaction Manager 
---Money Market 
----Transaction Management 
-----Flow Types 
------Assign Flow Types to Transaction Type - MM Transactions 

Keywords

Other flow, FTR_CREATE
