Symptom

The employment indicator is visible despite the setting "show all active employments (for global assignment and concurrent employment) on org chart and global search results" being deselected in logo settings.

Environment

  • SAP SuccessFactors Employee Central - Global Assignment

Reproducing the Issue

  1. "show all active employments (for global assignment and concurrent employment) on org chart and global search results" is deselected in company system and logo settings.
  2. Proxy as a user having global assignment "AAA".
  3. Go to Employee profile.
  4. The tag “Global Assignment” is visible.

Cause

Resolution

This is an expected behavior in the Latest People Profile. The Latest People Profile UI does not respect this configuration “Show All Active Employments (for Global Assignment and Concurrent Employment) on Org Chart and Global Search Results” when it is deselected under Company System and Logo Settings.
