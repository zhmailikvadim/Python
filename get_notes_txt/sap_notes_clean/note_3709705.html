Symptom

  • An error occurs when attempting to view Crystal Reports through BI Launch Pad, while the same report can be viewed in CMC.
  • Error message: "The viewer could not process an event. Your request was refused because you do not have the right 'Right To Refresh' on report 'Report Path'. Please ask your system administrator to grant you this right. [] Error code:0 [CRWEB00000119]"
  • Report shortcut can be opened correctly.

Environment

  • SAP BusinessObjects Business Intelligence 4.x
  • SAP Crystal Server 2020

Reproducing the Issue

  1. Login to BI LaunchPad
  2. Click on Folder
  3. Open Crystal Report

Cause

  • Missing "View On Demand" rights for the user in BI Launch Pad.

Resolution

1) Login to the CMC
2) Browse to the report-> Right click on the report and select User Security
3) Select the User or Group to which the User belongs in order to provide rights and click on Assign Security
4) Add the View On Demand Access Right and click on apply.
5) Now the report should open successfully.

Keywords

CrystalReport, View on demand, Right to Refresh, open, BILauchPad
