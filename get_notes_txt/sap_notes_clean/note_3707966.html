Symptom

  • Language key (T005-SPRAS) missing for country code 'MY' (Malaysia).

Environment

  • SAP S/4HANA 2023

Reproducing the Issue

  1. Access transaction SE16.
  2. Open table T005.
  3. Check the Language key (T005-SPRAS) for country code 'MY'.
  4. Observe that the Language key is missing for country code 'MY'.

Cause

Default language key configuration is missing for country code 'MY' in S/4HANA 2023.

Resolution

  1. Maintain the missing language key 'MS' (Malay) for country code 'MY' using transaction code OY01.
  2. Save the changes and verify the update in table T005.
