Symptom

  • Transaction FD32 system shows a wrong or negative credit exposure for Media documents.
  • Program RVKRED07 does not reorganize Advertizing Management orders.

Environment

SAP for Media

Reproducing the Issue

  1. Go into transaction FD32.
  2. Enter a customer and the credit control area.
  3. Goto 'Extras' and select 'Sales values'.
  4. negative values will be displayed.

Cause

  • In general a negative sales value may result from a cancellation invoice.
  • Another possibility is the customizing setting regarding the Update Group. Actually, the open order value is reduced only if you have set Update Group 000012 for the corresponding credit control area.

Resolution

  1. Customizing Setting has to be checked regarding the Update Group in transaction OB45.
    Update group 00012 should be maintained for the credit control area.
  2. The values shall be deleted aid of deletion reports attach to this KBA.
  3. The values need to be reorganized by the following report:
    • RJ_JANEU - Reorganization of Order Data (AM)
    • RJ_JFNEU - Reorganization of billing Data (AM)

Recommendations for shorter runtime:
It is possible to delete the values for one or a set of business partners. In this case, the reports RJ_JANEU and RJ_FNEU may only run for the orders/billing documents of the specific business partner.
