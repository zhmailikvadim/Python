Symptom

An error occurs while running Transaction FDM_CREATE_PROPOSAL (Program RFDM4000), resulting in a dump with the message: MESSAGE_TYPE_UNKNOWN and description "Message Type ' ' is unknown."

 "Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental." 

Environment

  • Dispute Management
  • SAP ERP       
  • SAP enhancement package for SAP ERP
  • SAP S/4HANA      

Reproducing the Issue

  1. Execute Transaction FDM_CREATE_PROPOSAL.
  2. Observe the runtime error MESSAGE_TYPE_UNKNOWN with the message "Message type ' ' is unknown."

Cause

Program RFDM4000 is not run as a batch job.

Resolution

  1. Ensure that Transaction FDM_CREATE_PROPOSAL is executed as a batch job.
  2. Verify that the job is running in the background by checking the `sy_batch` value in the Function Module `BP_ADD_APPL_LOG_HANDLE`.
  3. Confirm that the `sy_batch` value is equal to X to avoid runtime errors.
