Symptom

You run an integration for POL as Classic Routing. Some of the sub-objects, for example, operations, have failed at the same time as the other operations are successful and are saved on the SAP side for the same POL.

Other Terms

PLMSI, PLM System Integration, transactional handling, TC, transactional constraint, all-or-nothing, POL, SAP_POLasRouting.

Reason and Prerequisites

You must implement SAP Note 3726363 before this SAP Note.

Solution

Implement this SAP Note to ensure the sub-objects group is maintained only if none of them fail for POL as Classic Routing.
