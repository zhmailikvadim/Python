Symptom

The postal code validation for infotype Bank Details (0009) was delivered via SAP Notes 3662683 and 3672661.
However, an issue occurred when a postal code was associated with multiple possible city names. In such cases, a valid postal code and city combination was incorrectly flagged as invalid.

This SAP Note corrects that behavior and ensures that the validation now works as intended.

Other Terms

ADRS_BANKS, P0009, IT0009, Postal Code Check

Reason and Prerequisites

This SAP Note is relevant only for Belgium.

Solution

  • As a general rule SAP recommends that you implement the Support Package specified for your release.

  • However, if you need to install a solution earlier, use the SAP Note Assistant to implement the correction instruction.

  • You can find more information about the SAP Note Assistant in SAP Support Portal, under SAP Note Assistant.
