Symptom

For certain users, the display format is not displayed in the National ID field when National ID information is added during the Add New Employee screen.

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental. 

Environment

SAP SuccessFactors Employee Central - Add New Employee

Reproducing the Issue

  • Go to Add new employee screen.
  • Fill in the mandatory fields and proceed to the National ID-Information section.
  • Select the country, Card type and check.
  • The display format is not visible.

       

Resolution

This is an expected behavior as per the current system design.

Workaround:  Once the National ID edit permission is granted to the logged-in user, the issue will be resolved.
