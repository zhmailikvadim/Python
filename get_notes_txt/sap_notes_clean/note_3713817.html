Symptom

SYNTAX_ERROR dump is triggered in class /SPE/CL_EWM_STAGING_INTF======CP

Dump details:

Category                                        ABAP programming error
Runtime Errors                                        SYNTAX_ERROR

Application Component                                        LE-IEW-X

Environment

Cross-Application Topics

Cause

The issue is caused by a custom structure, the custom structure is already used in another standard table.

Resolution

  1. Adjust the custom structure.
  2. Use enhanced fields starting with 'Z' or 'Y' when working with Append or Include operations in standard tables to ensure better compatibility and reduce the likelihood of similar issues in the future.
