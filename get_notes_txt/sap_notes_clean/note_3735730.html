Symptom

An error occurs in Manage > Supplier Evaluation when starting supplier migration from the UI. How do I retrigger the migration to complete the pending jobs?

Environment

SAP Ariba SMR - Supplier Evaluation

Reproducing the Issue

  1. Navigate to Manage > Supplier Evaluation.
  2. Start the supplier migration from the UI.
  3. Observe that an error occurs, or the migration completes with data errors.

Resolution

Please follow the below steps to retrigger the migration mode for Supplier Evaluations for the failed suppliers.

  1. Go to Manage > SM Administration > Migration
  2. Define Start Date and End Date
  3. Click Start Migration and verify the Migration status.
