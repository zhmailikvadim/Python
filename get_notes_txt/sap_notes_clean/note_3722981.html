Symptom

During the upgrade process, errors occur because the service definition ISU_C4C_PREMISE_TI is not updated with the enhanced fields. The missing fields lead to inconsistencies in the service metadata and activation failures during upgrade.

Other Terms

 SAP Cloud for Customer, C4C Utilities, Premise TI, ISU_C4C_PREMISE_TI

Solution

Adjust Service Definition

To adjust or regenerate an existing service definition, do as follows:
1.Go to transaction code SE80.
2.Select object type as Package from the drop-down and provide the value as ISU_C4C_PROXY.
3.Expand the enterprise services node and expand Service Definition.
4.Double click on the service ISU_C4C_PREMISE_TI to regenerate and select the Service Definition.
5.Choose Edit and then Syntax Check. In the dialog box that appears, choose Yes and then Activate.
