Symptom

Relative position of cell is not working properly when row in table is set to “Hide when empty” in Web Intelligence

Other Terms

webi, relative , cell, position, bi

Reason and Prerequisites

Environment:

  • SAP BusinessObjects Business Intelligence Platform BI2025 Patch 03
  • SAP BusinessObjects Business Intelligence 4.3 Service Pack 04
  • Web Intelligence (Webi)

Step to Reproduce:

  1. Create a new webi document.
  2. Select a data source that contains empty rows.
  3. Run the query.
  4. Create a table that contains empty rows when a condition is selected.
  5. Create a new cell in the report (named Test for example).
  6. Set the relative position of the newly created cell to 0 Left side of Table and 0 Bottom side of Table.
  7. Set the rows of the table to “Hide when empty”.
  8. Apply a filter that returns some empty rows.
  9. Notice that there’s a space between the Table and the cell even with the relative position set to 0.
