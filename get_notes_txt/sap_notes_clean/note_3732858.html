Symptom

You are using the Run Statutory Reports app to prepare VAT reporting for Russia.

According to the Order dated 18.12.2025 № ЕД-7-3/1227@ issued by the Russian Federal Tax Service, the VAT return format has been updated to version 5.12.

This SAP Note delivers the necessary corrections to enable the system to generate XML files in accordance with the updated VAT return format (version 5.12) and to support the changes introduced by the new legal requirements.

Other Terms

RU, Russia, SAP Document and Reporting Compliance – statutory reporting, formerly known as Advanced Compliance Reports, Russia VAT Return, RU_VAT_DCL, Run Statutory Reports, Run Advanced Compliance Reports, RU_VAT_DCL_V512_26Q1, 5.12

Reason and Prerequisites

Legal Change

This SAP Note provides you with the relevant system changes to address the legal change: Changes: VAT Return and electronic formats 2026

Solution

As a general rule SAP recommends that you implement the Support Package specified for your release. However, if you need to install a solution earlier, use the SAP Note Assistant to implement the correction instruction. You can find more information about the SAP Note Assistant in SAP Support Portal, under SAP Note Assistant.

New xml tag СумСрокПр1992 has been introduced in section 1 of the VAT return. The tag derives its value from line 041 maintained in the customization.

New xml tags РеалТов22, РеалТов122, and РеалТов18.03 have been introduced in section 3 of the VAT return. These tags derive their values from lines 003, 005, and 007 maintained in the customization respectively.

The system derives data for the new tag СвЧастСчФ in the section 8 using the external number and external date of the invoice cleared with the down payment.

Example:

  Down Payment Invoice Down Payment Clearing External Number of Invoice External Date of Invoice
Case 1 1400000001 1800000001 1600000001 INV1 01.01.2026
           
Case 2 1400000002 1800000002 1600000002 INV2 01.01.2026
  1400000002 1800000003 1600000003 INV3 02.02.2026

Case 1 Output:

<КнПокСтр НомерПор="1" НомСчФПрод="1400000001" ...>

    <СвЧастСчФ НомЧастСчФ="INV1" ДатаЧастСчФ="01.01.2026"/>

</КнПокСтр>

 

Case 2 Output:

<КнПокСтр НомерПор="2" НомСчФПрод="1400000002" ...>

    <СвЧастСчФ НомЧастСчФ="INV2" ДатаЧастСчФ="01.01.2026"/>

    <СвЧастСчФ НомЧастСчФ="INV3" ДатаЧастСчФ="02.02.2026"/>

</КнПокСтр>

The system derives data for the new tag СвЧастСчФ in the section 9 using the external number and date of the document specified in 'Payment Order' option for respective subscenario in VAT Return Version.
