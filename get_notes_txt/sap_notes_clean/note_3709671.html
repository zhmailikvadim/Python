Symptom

The application keeps loading after clicking on the create button.

Affected applications:

  • Manage Payment Advices - New Version (F2550A) | SAP Fiori Apps Reference Library
  • Maintain Asset Revaluation Index (F8119) | SAP Fiori Apps Reference Library

Environment

SAP S/4HANA Cloud Public Edition 2602

Reproducing the Issue

  1. Access the Manage Payment Advices - New Version application.
  2. Click on the Create button.
  3. The application keeps loading.

Resolution

A fix will be delivered with HFC03 (31.January-1.February).

SAP S/4HANA Cloud Public Edition, 2-System Landscape – Upgrade & Maintenance Schedule

SAP S/4HANA Cloud Public Edition, 3-System Landscape – Upgrade & Maintenance Schedule

Please note the release date might be subject to change.

       

Workaround:

Please reload the creation page by pressing F5 or clicking the browser reload button.
