Symptom

After you implement SAP Note 3669704, a termination may occur in rare cases when you display a work confirmation and remuneration confirmation. This can happen if a partial remuneration rate of 0% is to be entered.

Other Terms

AK notification, subtype 70,

Reason and Prerequisites

In the method GET_TEILENTGELT of the class CL_HRPA_INFOTYPE_0526_01, an entry is deleted in the table of partial remuneration using an index. In rare cases, this index can be null and therefore call an invalid command that leads to a termination.

Solution

The correction is provided by HR Support Package (via CLC if necessary).
You can implement an advance correction by implementing the automatic correction instructions using SNOTE.

Immediately before the deletion of the entry is called, the system checks whether the index is greater than zero. Only then is a deletion carried out.
