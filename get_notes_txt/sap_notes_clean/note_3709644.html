Symptom

Create a Service Confirmation with a tool item and change the start date at the item level. Complete the Service Confirmation and find the created Controlling document in the transaction history tab. Open transaction KSB5 and check this Controlling document. The date in this document will not be changed compared to the date shown in the UI.

Other Terms

Service Confirmation, SRVC, Tool Item, SVCT, BUS2000154, Controlling Document, Start Date.

Solution

Install the correction instruction.
