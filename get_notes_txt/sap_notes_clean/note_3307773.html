Symptom

In MDG UI, when user tries to validate the tax number, system throws an error: "Another Business Partner already has this tax number".

Environment

  • Master Data Governance on S/4 HANA on-premise 2020
  • Master Data Governance on S/4 HANA on-premise 2021
  • Master Data Governance on S/4 HANA on-premise 2022
  • Master Data Governance on S/4 HANA on-premise 2023

Reproducing the Issue

  1. Launch NWBC
  2. Search BP and edit
  3. Add a Tax number and click check
  4. Message "Another Business Partner already has this tax number" is displayed

Cause

Standard Behavior

Resolution

The message is not an MDG specific one. It is a result of the business partner validation that is executed in MDG in the same way as done in transaction BP. It is correct that MDG does not have any pop-up for displaying the duplicates. The pop-up of transaction BP cannot be re-used. 

Also the message cannot be suppressed. It still contains useful information. 

Keywords

MDG Business partner, Tax Number Duplication, Incomplete message. Validation, MDG-BP/C/S
