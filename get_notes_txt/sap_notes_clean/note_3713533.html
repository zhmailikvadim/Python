Symptom

Adding the Functionality to set an External Status for a Payment Order via the Fiori Manage Payments app

Other Terms

External Status, Manage Payments, Payment Order

Reason and Prerequisites

There was no possibility to set the External Status via the Fiori Manage Payments app.

Solution

Providing possibility to set the External Status via the Fiori Manage Payments app.

After implementation this note, implement notes number 3715298 and 3727365 to enable the UI part.

Please apply SAP Note or assigned Support Package
