Symptom

  • The SAP SuccessFactors mobile application does not display the new banner cards customized for the instance.
  • The issue occurs after enabling the latest homepage feature in SAP SuccessFactors.
  • The banner card feature is confirmed to be unavailable in the legacy homepage.
  • Disabling the latest homepage results in the banner card disappearing from both the homepage and the mobile app.

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

SAP SuccessFactors HCM Suite

Reproducing the Issue

  1. Enable the latest homepage feature in SAP SuccessFactors.
  2. Access the SAP SuccessFactors mobile application.
  3. Observe that the banner cards customized for the instance are not displayed.

Cause

Known issue. The New Home Page became GA (General Availability) in the 2H 2025 release, but Banners cards won't be supported on the Mobile app until the February (2602) release.

Resolution

As of February 18, 2026:

  • The latest mobile app has now been released in Playstore for Android devices.
  • The App store for iOS devices has not approved the latest mobile app yet.
  1. Monitor the availability of the latest mobile app update in the App store for iOS devices.
