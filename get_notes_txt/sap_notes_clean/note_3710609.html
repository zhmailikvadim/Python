Symptom

You manage a bond or an interest rate instrument using risk free rates. You created an accumulating condition to create an accumulating nominal interest flow. The condition for the nominal interest flow uses a spread. For the accumulating flow the following field is not filled: Average Interest Rate Spread

Other Terms

PAVGSPREAD, DZFEST, DPKOND, SOFR, SONIA, ESTR, TONA

Reason and Prerequisites

Design

Solution

Implement the correction instructions with transaction code SNOTE.

For the accumulating nominal interest flow the field Average Interest Rate Spread is filled using the information from the last snippet contributing to the accumulation.
