Symptom

You specify the global parameters in the Set Global Parameters app or in transaction CXCD. You open the Data Monitor and you get a message like: "Period ZZZ/YYYY is invalid for version X" (Message No. G00088).

Environment

SAP S/4HANA Finance for Group Reporting

Cause

The information message, that "Period ZZZ/YYYY is invalid for version X   Message No. G00088" is shown when you assigned a planning cycle to the relevant version (In IMG activity "Plan Consolidation"/"Assign Consolidation Cycle to Version") and the current period is not a consolidation period for this cycle (according to the Cons.Frequency set in IMG activity "Define Consolidation Cycle").

Resolution

The message is only an info-message and shall tell you, that for the assigned Cons.Cycle - that can be run automatically for this version - the period ZZZ/YYYY is not a valid one for the cycle. Nevertheless, you can still access the Data Monitor and you can run the manual activities there.

Keywords

Period is invalid for version, Data Monitor, G00088, Cons.Cycle, Cons.Frequency, Define Consolidation Cycle, Assign Consolidation Cycle to Version
