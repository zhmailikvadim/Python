Symptom

As described in the Regulatory Change Manager tool under Changes: Formation date and time for military registration document, the Cabinet of Ministers of Ukraine has updated the requirements for the content of military registration documents.

According to the legal change, military registration documents must now include the generation date and time of the military registration document.

This SAP Note provides the system changes required to enable the storage and maintenance of the generation date and time of the military registration document.

Other Terms

0290, PA0290, UAML, MP029000, 2366, DOTIM, DODAT

Reason and Prerequisites

Resolution No. 559 of the Cabinet of Ministers of Ukraine

Implement SAP Note 3731139 before implementing this SAP Note.

Solution

As a general rule, SAP recommends that you implement the Support Package specified for your release.

However, if you need to install the solution earlier, use the SAP Note Assistant to carry out  automatic and manual post-implementation correction instructions. For more information, see SAP Note Assistant.

Note that after installation of HR Support Package, it is possible that customizing records will be imported into client 000 only. In this case, adjust the customizing of your work client manually. For more information about the customizing, see the attached file Manual Post-Implementation Instructions for SAP Note 3731140.pdf.
