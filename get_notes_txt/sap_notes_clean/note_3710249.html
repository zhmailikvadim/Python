Symptom

  • I would like to understand what is the impact of changing the value of the Use Admin Structure field in a Rate Schedule
  • I would like to know what is the Use Admin Structure field used for

Reproducing the Issue

  1. Admin Configuration > Rate Schedule
  2. Open a Rate Schedule > Edit
  3. Change Use Admin Structure field to Yes or No
  4. Save

Resolution

When creating/updating a rate schedule, users will have two options for setting the Use Admin Structure field, Yes and No.

  • If Yes is selected, the rate schedule structure is copied from the Admin menu when a work order or work order revision is created. Therefore, if a change has been made to the rate schedule structure after a job posting has been created, the change is automatically updated on a work order or work order revision.
  • ­If No is selected, when a work order or work order revision is created, the rate schedule is copied from the job posting. If a change has been made to the rate schedule in the Admin menu, the change is not picked up.

When Yes is selected for rate schedules with a Service Type of Services, uploaded work order revisions will be updated to match the structure of the rate schedule admin object. The use of this functionality is limited to work order revisions that are uploaded using the Upload SOW Team Member Revision connector.
