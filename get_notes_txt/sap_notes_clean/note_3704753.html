Symptom

  • New hires do not appear in the search results but are visible in the Org Chart.
  • Employees appear as "Unknown" in the search engine despite having correctly filled personal information.
  • User synchronization issues cause delays of 20-30 minutes for employees to appear in SuccessFactors after integration.
  • Excessive time is required to update data for a single employee.

Environment

SAP SuccessFactors HCM Suite

Reproducing the Issue

  1. Search for employees in the directory using their first and last names or External Personal ID.
  2. Observe that some employees appear as "Unknown" despite having their personal information correctly filled.
  3. Search for employees in the Org Chart and note that their names appear correctly.
  4. Create an employee via integration and observe that it takes 20-30 minutes for the user to appear in SuccessFactors.

Cause

This is a Known Issue.

Resolution

The fix for this issue is planned to be deployed on 1H 2026 release. For release timelines, please review SAP SuccessFactors Product Release & Road Map Information.

Workaround

  • Use the legacy search functionality as a workaround.
  • Disable the "Enable Enhanced People Search in the Global Header and the Directory Search" setting in the instance configuration.
