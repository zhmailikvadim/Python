Symptom

  • Creating a realignment project to realign Key figures issues a warning"Aggregation mode Avg of key figure XXX does not fit the selected action."

Environment

  • Data Realignment

Reproducing the Issue

  1. Create a realignment Project to realign a key figure using aggregation mode "Average"
  2. In the Adjust Target Key Figure Values Option, select option such as "Sum up Source and Target" or "Sum up Source"
  3. Observe the warning message: "Aggregation mode Avg of key figure XXX does not fit the selected action."

Cause

The issue is related to the aggregation mode "Avg" of the affected key figure. This does not fit the selected action in the realignment project.

Resolution

For key figures using aggregation mode "AVG", the following 2 processing options

  • Sum up Source and Target
  • Sum up Source 

can not be used. An Average calculation of a key figure does not support the calculation of a SUM at the same time.
