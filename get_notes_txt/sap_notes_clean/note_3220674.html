Symptom

A tree control can react to right mouse clicks of the user with opening different context menus depending on where the user has positioned the mouse.

If a tree control is reacting to right clicks on individual tree items this normaly results in an item context menu being displayed. In tree controls where multi selection is enabled for nodes and items we have a special case where right clicking an item in the tree is supposed to result in a node context menu opening - as long as at least one other tree node is already selected.

In SAP GUI for HTML, the right click on an item unexpectedly results in an item context menu being requested which is a wrong behaviour in this particular multiselection case.

The problem for example occurs in transaction ECH_MONI_SEL.

Internet Transaction Server, ITS

Reason and Prerequisites

Program error

Solution

To resolve this issue please do the following:

Determine the patch number of the SAPEXE SAR package with the highest patch number for your kernel version on the SAP Service Marketplace (SMP). If the SAPEXE patch number is higher than or equal to the highest patch level number you see in the section Support Packages & Patches of this SAP note for your SAP kernel version, you should download and install the SAPEXE SAR package. Otherwise, download and install the most recent DW.SAR archive.

If there is no entry with the patch number yet, please re-check this SAP note at a later point in time. The patch number level will be added automatically once the corresponding patch is available on SAP Service Marketplace.
