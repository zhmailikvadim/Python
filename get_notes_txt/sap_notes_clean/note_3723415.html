Symptom

You are using the Egypt VAT Return statutory report( Report Category:- EG_VAT_DCL ) in the Run Statutory Reports app. When you use Manage Tax Items reporting activity and change the reporting date of even one document to a date before the posting date, the report output does not get generated.

Other Terms

VAT, Form 10, Egypt VAT Return,Document and Reporting Compliance, Statutory Reports, EG_VAT_DCL

Reason and Prerequisites

Changing a document's reporting date to a date before the posting date causes issues while fetching the data required for reporting, thus leading to failure in report output file generation.

Solution

Corrections have been made to prevent this issue from occuring and generate the VAT Retun Form 10 as expected.

After implementing this SAP Note, the system generates the the VAT return Form 10 file for Sales, Purchase when data exists in the sales and purchase sections.
