Symptom

You have contract accounts in C4C and require to export them with there related BP ID. The Data Workbench for Business Agreement object does not include BP ID data in the export so this would not be a valid option.

Environment

  • Cloud for Customer
  • SAP S4/HANA

Reproducing the Issue

  1. Open the C4C system.
  2. Navigate to the Utilities workcenter.
  3. Open the Contract account object.
  4. Open any contract account.
  5. Observe there is no option to export the contract accounts with linked BP ID.

Cause

System limitation, The contract accounts from C4C cannot be exported with the BP ID linked.

Resolution

Use BAPI FM BAPI_ISUACCOUNT_GETLIST in the corresponding ISU system to fetch the contract account data along with the linked BP ID.

Keywords

Contract accounts, BP ID, C4C, Export option, Utilities workcenter, Data Workbench, BAPI_ISUACCOUNT_GETLIST, ISU system, System limitation, linked BP ID
