Symptom

You use the Product Compliance functionality. The system shows following incorrect behavior.

In the Compliance Workbench, you use the Reset to Calculated function to remove substance declarations. The system displays a timeout error message.

 

Other Terms

CL_EHPRC_CW_OIF_DECSUB_LST

Reason and Prerequisites

A program error in the system causes the symptom.

You find the prerequisites in the relevant correction instructions.

Solution

After implementing this SAP Note, the system runs the function significantly faster.

Alternatively, you import the corresponding Support Package for the component extension of SAP Environment, Health, and Safety Management or the Feature Package Stack for the SAP S/4HANA system.
