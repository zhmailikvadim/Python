Symptom

Error "Item type UISC is not supported in the current environment" shows when trying to create Spaces and Pages in Fiori Launchpad

Environment

  • SAP Fiori Launchpad
  • SAP S/4HANA 2025

Reproducing the Issue

  1. Log in to the system
  2. Access the FLP using tcode /n/UI2/FLP
  3. Open Manage Launchpad Spaces/Pages
  4. Click on create for creating a Space/Page
  5. Fill in the required details
  6. A pop-up appears stating: "Item type UISC is not supported in the current environment"

Resolution

  1. Apply SAP Note 3621938 - Spaces+Pages: Save Terminates with Exception
  2. Clean cache according to 2319491 - How to clean up the cache after applying changes that affect SAP Fiori apps
