Symptom

Vulnerability identified: Apache Struts XML External Entity (XXE) (S2-069)(CVE-2025-68493).

Environment

  • SAP BusinessObjects 4.3 SP4+
  • Information Steward 4.3 SP4+
  • Apache Struts API version 2.5.x

Cause

Vulnerability found in Apache Struts API version 2.5.x

Resolution

In BI/IPS 4.3 SP5 the Apache Struts API was upgraded to a non-affected version (6.7) as per: 2364904 - Which version of Apache Struts does SAP BusinessObjects use? - SAP for Me

  1. Upgrade BI/IPS to 4.3 SP5+
  2. If you want to double check the API version upgraded: 
  3. Open File Explorer.
  4. Search for 'struts*.jar' file within the installed BusinessObjects folder, i.e. C:\Program Files (x86)\SAP BusinessObjects
  5. The search result will list multiple 'struts.jar' files, i.e. C:\Program Files (x86)\SAP BusinessObjects\tomcat\webapps\BOE\WEB-INF\eclipse\plugins\webpath.InfoView\web\WEB-INF\lib\struts.jar
  6. Copy and paste any 'struts.jar' file in a folder and extract it, or open struts.jar directly with tools such as WinRAR.
  7. There will be a 'META-INF' folder, inside the META-INF folder, open the 'MANIFEST.MF' file in any text editor.
  8. All details of the version are included in this file.
