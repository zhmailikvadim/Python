Symptom

If a visit day of an outpatient case falls on an admission or discharge day for an inpatient case  of the patient in the communication procedure P321/ELDA or during scoring, the system does not create an outpatient admission message or scoring package for this outpatient movement date.

Other Terms

ELDA communication, P321, ELDA, LKF model 2026, SCOX, scoring, outpatient movement, parallel inpatient case

Reason and Prerequisites

Prerequisites:

  • Country Version Austria (AT)
  • Use of the communication procedure ELDA (P321) 
  • Scoring LKF Model 2025 (Online, EDI Procedure SCOX, Scoring Reports, ) 
  • Note 3713998

Reason:

Development Request

Solution

Implement the program changes and perform the manual tasks.

The system now takes into account the Scoring Relevance checkbox delivered with SAP Note 3713998 - IS-H AT: LKFScoring - Scoring Relevance of an Outpatient Movement (see there) for an outpatient movement of an outpatient case when creating an outpatient admission message in the communication procedure P321/ELDA and when scoring outpatient cases.

P321/ELDA:

The system creates an outpatient admission notification for this movement/day if the movement is before the inpatient admission time or after the discharge time. This also applies to the absence start or end date. Note that for outpatient cases, only the first visit of a patient is reported for each case in the quarter. Also take your setting of the system parameter ELDA_ABEWA (ELDA - Outpatient Movement Exclusion for Parallel Stat. Case) into account.

LKF Model - Scoring:

The system takes this flagged outpatient movement into account during scoring if the movement is before the inpatient admission time point or after the discharge time. This also applies to the absence start or end date.

For other visiting days outside of an inpatient stay (incl. Admission/Discharge Day), you do not need to select this checkbox as before.

Outpatient movements within an inpatient stay (now exact to the time) of a patient are not taken into account during scoring, even though they have been selected.

Note also that all other exclusion reasons that do not refer purely to parallelism with an inpatient case remain valid.
