Symptom

The participation settings is not longer available for newly created programs.

Environment

SAP Engagement Cloud

Reproducing the Issue

  1. Go to Automation.
  2. Create a new automation program of any type.
  3. Notice the participation settings is not available, only the participation check node.

Cause

The Participation Settings is planned to sunset, with no date yet, but it's disabled in the newly created programs. 

Resolution

This is the expected system design.

The participation check node should be used instead: Configuring the Participation Check Node

See Also

 Configuring the Participation Check Node
