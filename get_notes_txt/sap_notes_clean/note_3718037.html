Symptom

Lot-generating transaction number is not visible in TPM12 and TPM13 for flows created by a portfolio transfer.

Environment

Transaction Management

  • Treasury and Risk Management (TRM)
  • EA-FINSERV
  • S/4CORE
  • SAP ERP
  • SAP ERP Central Component
  • SAP enhancement package for SAP ERP
  • SAP S/4HANA
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA Finance

Reproducing the Issue

  1. Execute transaction code TPM12 or TPM13 for a Security Class.
  2. For flows created by a portfolio transfer, Lot-generating Transaction Number field is blank.

Cause

Missing functionality: When conducting a portfolio transfer from a portfolio with lots, where a sale/purchase has occurred, the standard system does not track the origin of the new lot, resulting in the absence of the "Lot-gen. Transaction Number". This is valid regardless of the method (FIFO/LIFO) used.

Resolution

Refer to SAP Note 2967362 for guidance on submitting an enhancement request for On-Premise Product.
