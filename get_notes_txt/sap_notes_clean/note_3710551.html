Symptom

You have created a payment lot referencing a payment advice note. When the advice is received only after the payment has been made, the system creates a clarification case when the payment lot is posted. You want to initiate follow-up activities in event 0197. The event is not available on S/4HANA 2023.

Other Terms

FP-EVENT-197, FKK_SAMPLE_0197, FPAV, FP05

Reason and Prerequisites

Missing functionality.

Solution

Implement the attached correction instructions or import the relevant Support Package.

You can now use event 0197.

Sample module FKK_SAMPLE_0197_SEARCH_AMOUNT is available as a template to check whether there is an open clarification case for incoming payments that has the same payment amount as the payment advice note.

Sample module FKK_SAMPLE_0197_SEARCH_AVKEY is available as a template to check whether an open clarification case for incoming payments exists in which the number of the payment advice note was determined as a selection criterion.
