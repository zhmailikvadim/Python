Symptom

  • Installed Work Manager 6.1 and connected to ECC6.0 EhP7 with ISU extension
  • During ISU work order fetch, the following error was seen in the Work Manager logs:

    CX_SY_NO_HANDLER:An exception of type 'CX_SY_DYN_CALL_PARAM_MISSING'.

~~~~~~~~~~~Error~~~~~~~~~~~~~~~~~~~~

Environment

  • SAP ERP 6.0
  • SAP Work Manager 6.1

Cause

  • Traced the issue to the calling function module ISU_MAM_READ_MREAD_LIMITS
  • This is in /SMISU/CL_BTX_WORKORDER_DO->FILL_MREAD_TABLES. The function module call is missing parameter X_MASSREAD.
  • The EhP2/IS-UT SP6 system and the function module ISM_MAM_READ_MREAD_LIMITS does not have the additional parameter X_MASSREAD (not marked as optional)

Resolution

An SAP note was created to resolve this issue. See - SAP Note: 2046464 (under References).
