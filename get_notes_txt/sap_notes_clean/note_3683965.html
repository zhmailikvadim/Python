Symptom

  • The table AUSP is not updated automatically when a field in material master is changed in MM02 - the reference characteristic in classification does not update automatically.

Environment

  • Classification

Reproducing the Issue

  1. Create a material and classify it, ensuring that the characteristic is set with reference to a MARA field
  2. Change the field in material master in MM02 (without entering the classification screen).
  3. Observe that the entry in the AUSP table is not updated automatically.

Cause

  • Standard behavior in ECC indicates that the table AUSP is only updated when classification is performed, resulting in outdated characteristic values.

Resolution

Please follow the instructions in the attached PDF file. 
In this file the classification of materials will be updated. This method can also be used to update the classification of any other object (for example equipments). In this case, the program must be adopted accordingly. 

See Also

Please find the program Z_UPDATE_OBJ_CHARACTERISTICS which is mentioned in the PDF file attached.
