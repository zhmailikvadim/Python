Symptom

The notification message "Certificate validation disabled" appears in the Electronic Document Service (EDS) dashboard, even though certificate verification is enabled in the SAP Business One Server Components Setup Wizard.

Reproducing the issue

Prerequisites:

Certificate verification has been enabled in the SAP Business One Server Components Setup Wizard.

Scenario:

  1. Log into the EDS dashboard.
  2. Navigate to the Notification Inbox.

Expected behavior:

The warning message "Certificate validation disabled" should not be displayed.

Actual behavior:

The warning message "Certificate validation disabled" is displayed.

Solution

SAP intends to provide a feature package or patch to solve the symptom that is described in this SAP Note. The section References below lists the related feature package or patch once they become available. Be aware that References can only be confirmed at feature package or patch release date. SAP delivers feature packages or patches only for selected releases at its own discretion, based on the business impact and the complexity of implementation.
