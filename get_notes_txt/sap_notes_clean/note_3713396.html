Symptom

  • How to export an entire agent repository in ATL/XML format?
  • How to export a task/process in ATL/XML format?
  • How to collect agent diagnostic package using command line?

Environment

SAP Cloud Integration for data services

Resolution

  • Usage: ConfigureAgent.bat for Windows or ./configureAgent.sh for Linux. For Windows OS,  ensure open command prompt as administrator to continue.

  • -export[xml|atl] -repo[sandbox|prod] [-name[taskname]]

    • For example for Windows, ConfigureAgent.bat -exportatl -reposandbox -nameYourtaskname
    • For example for Linux, ./configureAgent.sh -exportatl -reposandbox -nameYourtaskname
    • Export ATL or XML from repo:
      -exportatl -reposandbox -nameYourtaskname ---- Export the task/process  Yourtaskname from the sandbox in  atl format.
      -exportxml -reposandbox -nameYourtaskname ---- Export the task/process  Yourtaskname from the sandbox in  DS XML format.
      -exportatl -reposandbox  ---- Export the entire sandbox repo in atl format.-exportxml -reposandbox  ---- Export the entire sandbox repo in DS XML format.
      -exportatl -repoprod  ---- Export the entire production repo in atl format.
      -exportxml -repoprod  ---- Export the entire production repo in DS XML format.

  • -agentdiagnostic[all|logsonly|systemdump|checkcerts]
    • For example for Windows, ConfigureAgent.bat -agentdiagnosticlogsonly
    • For example for Linux, ./configureAgent.sh -agentdiagnosticlogsonly
    • -agentdiagnostic   ----By default, run full diagnostic and collects log,conf and adapter into a zip file.
      -agentdiagnosticall   ----Run full diagnostic and collects log,conf and adapter into a zip file.
      -agentdiagnosticlogsonly   ----Only collects log,conf and adapter into a zip file.
      -agentdiagnosticsystemdump -----Run the network/system diagnostic and output to the Diagnostic_yyyy.mm.dd.hh.mm.ss.log file
      -agentdiagnosticcheckcerts -----Run the certificate diagnostic and output to the Diagnostic_yyyy.mm.dd.hh.mm.ss.log file
