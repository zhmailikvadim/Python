Symptom

You are running consistency check with CHECK_TABLE_CONSISTENCY.

The syntax to run consistency check is:

CALL CHECK_TABLE_CONSISTENCY ('<check_actions>', '<schema_name>', '<table_name>')

Environment

SAP HANA Platform

Resolution

The list of available check actions may differ from HANA revision to HANA revision.

You can list the available check actions with:

CALL GET_CHECK_ACTIONS('CHECK_TABLE_CONSISTENCY')

You can define multiple check actions by listing the actions separated by commas between the apostrophes e.g:

CALL CHECK_TABLE_CONSISTENCY ('<check_action1>,<check_action2>', '<schema_name>', '<table_name>')
