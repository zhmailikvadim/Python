Symptom

In the costing run step selection, structure explosion or costing you maintain the processing parameters (report variant). When pressing the F4 value help for the costing run field (or costing run date field) the transaction is terminated with a dump due to the ABAP program error PERFORM_CONFLICT_UC_STRUCT.

The same happens with the reports SAPRCK10, SAPRCK60 or SAPRCK62.

Other Terms

CX_SY_DYN_CALL_ILLEGAL_TYPE, CK40N, costing run, VALUE_REQUEST_CALLBACK

Reason and Prerequisites

Program error

Solution

Install the attached correction
