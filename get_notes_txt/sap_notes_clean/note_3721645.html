Symptom

Some Fields/Columns are Missing from layout in KE24/KE24N After system Upgrade to S/4HANA.

Environment

  • SAP S/4HANA Finance
  • SAP S/4HANA

Reproducing the Issue

  1. Go to T-code: KE24/KE24N input the detailed parameters
  2. Select the layout
  3. click "execute" button
  4. check the layout fields,  such as  Partner prof.segment, Partner subnumber are missing from the layout.

Cause

As per attached KBA 2718989(point 20), the fields listed are now set as technical fields and should not be displayed on output in Account Based Profitability analysis.
Technical fields and should not be displayed on output in Account Based Profitability analysis.
Please go to T-code: KEA5, input your operating concern=>click "Display" =>go to next screen=>MENU=>Extras=>Fixed fields
=>Tab "Technical fields"=>you will find the following fields, they are all Technical fields.

Resolution

This is current application design which cannot be changed and is used by many other customers.
