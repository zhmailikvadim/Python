Symptom

Solr can provide suggestions for searched words. You can use one of the following components to activate this feature:

  • SpellCheckComponent

  • SuggestComponentt

Environment

SAP Commerce Cloud 

Cause

By default, the Solr server uses SpellCheckComponent. You can change this configuration in the solrconfig.xml file.

Resolution

While SpellCheckComponent may work adequately in single-shard setups, it faces limitations in multi-shard configurations. We recommend you to use SuggestComponent instead of SpellecheckComponent for multi-shard configurations. Update the solrconfig.xml to replace SpellcheckComponent with the SuggestComponent. You may need to add or modify field types to optimize suggestion generation.

SuggestComponentt

To switch to the SuggestComponent component, in the solrconfig.xml file, comment out the following code blocks:

<requestHandler name="/suggest" class="solr.SearchHandler">
		<lst name="defaults">
			<str name="spellcheck">true</str>
			<str name="spellcheck.dictionary">default</str>
			<str name="spellcheck.onlyMorePopular">true</str>
			<str name="spellcheck.count">5</str>
			<str name="spellcheck.collate">true</str>
		</lst>
		<arr name="components">
			<str>suggest</str>
		</arr>
	</requestHandler>
