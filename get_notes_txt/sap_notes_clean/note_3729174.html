Symptom

Fiori app F4920 (Create Sales Orders - Automatic Extraction) is used to create sales orders from purchase order files in PDF or image formats (unstructured data). However the app cannot create sales order with reference to the contract.

Environment

  • Sales and Distribution (SD)
  • Fiori
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Cause

The Fiori app F4920 is not designed to create sales order with reference to the contract. It is limited to extracting information from unstructured data (e.g., PDFs) and mapping it to sales order fields.

Resolution

Use the other Fiori app, for example, Manage Sales Orders - Version 2 - F3893 to create the sales order with reference to the contract.
