Symptom

In the "Raw Material Balance" analytical application, the actual values for "Production" line shows incorrect percentages, even though the quantities themselves are correct. The values differ from those in the "Postprocess Documents" application, although it contains the correct values. This happens when different component materials are having the same component type ID.

Other Terms

Raw Material Balance, Postprocess Documents, Component Type ID

Reason and Prerequisites

Program error

Solution

History:

Version 1 of 18 March, 2026: New creation

Install the code correction.

Description of Correction:

The calculation logic was adjusted at the Actual Quantity level such that the result is also considering the number of component materials with the same component type ID.

Functional Dependencies:

There are no functional dependencies.

Manual Changes:

There are no manual activities.
