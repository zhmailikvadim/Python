Symptom

When creating/changing material with same EAN number already used for another material, there is no warning message "M3 348 - The EAN & is already used for the material &" in MDG change request.
The same warning message is shown when creating/changing material via backend transaction MM01/02.

Environment

Master Data Governance on S/4HANA 2023

Reproducing the Issue

  1. Create material in MDG change request
  2. Enter EAN number already used in other material
  3. No warning message shown during validation

Cause

Program error

Resolution

Please implement SAP Note 3478980 - MDG-M: EAN warning message not triggering after upgrade for MDG Material

See Also

3478980 - MDG-M: EAN warning message not triggering after upgrade for MDG Material

Keywords

EAN, M3 348, OMT4
