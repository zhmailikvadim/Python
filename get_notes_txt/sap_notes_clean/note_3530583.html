Symptom

After pressing the "check availability" button the system throws the following errors in Manage Sales Order Version 2 fiori app when starting to create a sales order:

  • Error message: "Product availability check request failed"
  • Error message: "Processing not possible: adv. ATP is inactive for item"

Environment

  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Open manage sales order V2 - Extended
  2. Click on create button
  3. Enter sales order type, sales organization, business partner, material
  4. check availability button
  5. system throws the errors

Cause

Advanced ATP is not activated for the checking group of the material. The advanced ATP and the ECC ATP are two different solutions. In Fiori Sales Order only the aATP is supported.

Resolution

For materials without advanced ATP transaction VA01 should be used.
