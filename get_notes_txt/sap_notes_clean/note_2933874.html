Symptom

Hyperlink for recorded defects is not visible in the defect facet of 'Perform Work' application, if the defect description is not available.

Other Terms

Defect, Defect Description, Perform Work, Process Defect, Defect List

Reason and Prerequisites

The hyperlink is enabled for description of the defect. Hence, while recording a defect, if description is not maintained, hyperlink will not be visible.

Solution

Implement this note to enable hyperlink for defect id, instead of defect description.
