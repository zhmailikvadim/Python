Environment

  • Framework: SAP SuccessFactors People Analytics

Reproducing the Issue

  1. Login into the instance with support access.
  2. Create a new story from the report center.
  3. Select connection type as SAPSFSFREP.
  4. Create Query
  5. Preview the query and observe data visibility.
  6. Click "Finish".
  7. Edit the data source again and observe the error messages (no query will be displayed).

(or)

  1. Drag and drop affected object
  2. The object takes a long time to load.
  3. When we click on the columns, they keep loading indefinitely and never display

Cause

  • The issue is related to one of the objects used in Query design.
  • Affected object has a high level of nesting with multiple associations, leading to significant response size expansion.

Resolution

  1. Review and simplify the structure of the affected object.
  2. Reduce deeply nested associations, particularly those involving country-specific corporate address expansions.
  3. Optimize the object model to decrease response size and improve performance.
