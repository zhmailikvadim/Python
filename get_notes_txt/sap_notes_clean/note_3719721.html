Symptom

  • In the tenant, when navigating to Home -> User Management -> Search for user with last name, no results are returned.
  • Searching with email or first name provides results.

Environment

Identity Authentication Service

Reproducing the Issue

  1. Navigate to Home -> User Management.
  2. Attempt to search for a user using their last name.
  3. Observe that no results are returned.
  4. Search using the user's email or first name.
  5. Observe that results are returned.

Cause

It is working as expected.

Resolution

  1. When performing a search, users can search by User ID, Global User ID, SCIM ID, Email, or Login Name.
  2. To search for last name, users must press "show filters" to enable the last name search functionality.
  3. Please check SAP Help Portal: Search Users | SAP Help Portal

See Also

Refer to SAP Help Portal: Search Users | SAP Help Portal
