Symptom

• When customizing BRF+ or OPD rules "Printer Settings", the output device cannot be selected or is filtered out from F4 help.

• Device class and Access method are maintained with allowed values as outlined in SAP Note 3057330. 

Environment

  • SAP S/4HANA
  • As of release SAP S/4HANA 2021
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Run the transaction BRF+, search for the corresponding object, navigate from Expression to Decision Table, select "Printer Settings," click on edit, select the print queue ‘LP01,’ click on folder box (F4 Help), and click on GO.




  2. For comparison, run the transaction BRF+, search for the corresponding object, navigate from Expression to Decision Table, select "Printer Settings," click on edit, select the print queue ‘LP01,’ click on folder box, and click on folder box under output Device.
  3. Run the transaction OPD, select the corresponding object and Determination Step "Printer Settings," click on edit, select the print queue ‘LP01,’ click on folder box, and click on GO.
  4. For comparison, run the transaction OPD, select the corresponding object and Determination Step "Printer Settings," click on edit, select the print queue ‘LP01,’ click on folder box, and click on folder box under output Device. 

Cause

• The device type assigned to the output device can be listed using Transaction SE37: function = RSPO_RETURN_DEVICES.
• The device type has not been customized with a Page Definition Language (PDL Type), which can be verified using Transaction SE37: function = ADS_GET_PRINTER_ATTRIBUTES.
• The list of output devices displayed in the "F4 Help" is filtered based on supported PDL types.
• When the function "ADS_GET_PRINTER_ATTRIBUTES" is run by the standard S/4HANA output control framework for either of these two output devices, it returns an value for the PDF type.

Resolution

1. Customize the device type with a supported PDL type. 

2. Page Definition Language (PDL) type is derived from the device type. Allowed values include PDF, PCL, PS, DPL, IPL, TPCL, ZPL, and CAB.

3. After applying these changes to the device type, you should be able to select either of these two output devices in the "F4 Help" from the BRF+ rule "Printer Settings."
