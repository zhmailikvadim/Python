Symptom

  • Error message is returned when starting the SySAM service:
       The SYSAM service on Local Computer started and then stopped. Some services stop automatically if they are not in use by other services or programs.
  • The SYSAM service is configured with Local System Account.

Environment

  • Sybase Software Asset Management (SySAM) 2.4.0, 2.4.2
  • SAP PowerDesigner (PD) 16.7
  • Microsoft Windows Server 2025

Cause

SySAM 2.4 binaries have not been tested on Windows Server 2025, which may lead to compatibility issues.

Resolution

Install SySAM 2.5.0, which is included with PD 16.7 SP09.
