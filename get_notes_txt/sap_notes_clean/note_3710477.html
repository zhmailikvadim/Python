Symptom

A Deletion Flag 'DLFL' status is set for an Order. All the frozen schedules (in which one or more operations of that Order is included) are deleted. This is prevented with this SAP Note.

After implementing this SAP Note, Frozen Schedules will get deleted only when the Order is set to Deletion Indicator 'DLT' status

Other Terms

RSH , F3326 , Manage Schedules , Frozen Schedules , DLFL (I0076) , DLT (I0013)

Reason and Prerequisites

Occurs due to program error

Solution

Apply the automatic correction
