Symptom

Is the SAP Cloud Print Manager for Business ByDesign supporting Windows Server 2025?

Environment

SAP Business ByDesign

Resolution

  • The SAP Cloud Print Manager for SAP Business  ByDesign supports all Operation systems like Windows 10 and 11 and all Server installations which are supported by Microsoft itself.
  • Any Operation system which reached the Microsoft EndOfLiveCycle is not supported.

Keywords

Cloud Print manager, server 2025, printing, CPM.
