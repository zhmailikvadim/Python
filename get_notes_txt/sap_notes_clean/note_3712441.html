Symptom

When you run report EDOC_GR_FI_CATEGORY_C (transaction EDOC_GR_CAT_C) to generate eDocument type C Category (Invoice type 17.1 - 17.6), the amount calculated at account level is incorrect in cases where credit entries exist. Credit postings are treated as absolute values, leading to an increased total rather than a net reduction.

Other Terms

Greece eBooks, myDATA, AADE, eDocument, e-Doc, EDOC_GR_FI_CATEGORY_C, EDOC_GR_CAT_C, CL_EDOC_MAP_GR_FI_CAT_C

Reason and Prerequisites

Program error.
This note amend the solution delivered by note 3003692 and subsequent. For list of all prerequisite notes please refer to the overview note 3123296.

Solution

As a general rule, SAP recommends that you install a solution by applying a Support Package. However, if you need to install a solution earlier, use the Note Assistant to implement the correction instruction. You can find more information about the Note Assistant in SAP Service Marketplace, under service.sap.com/note-assistant.
