  • These stack are usually printed during the same period of time.

Environment

  • SAP Replication Server (SRS) 16.0
  • SAP Adaptive Server Enterprise (ASE) 16.0 with Always-on Option (HADR)
  • SAP Adaptive Server Enterprise (ASE) 16.0 with High Availability Disaster Recovery (HADR) for Business Suite
  • SAP Replication Server (SRS) 16.1
  • SAP Adaptive Server Enterprise (ASE) 16.1 with Always-on Option (HADR)
  • SAP Adaptive Server Enterprise (ASE) 16.1 with High Availability Disaster Recovery (HADR) for Business Suite

Cause

  • Attempt of connection initiated by an external application (non-TDS protocol), likely a port scanning tool.
  • Disruption of the client process or interruption of network communication between the client and RMA.

Resolution

  1. Identify external programs or tools that may be initiating connections using non-TDS protocols, such as port scanning tools.
  2. Investigate and disable or reconfigure these external programs to prevent them from causing connection disruptions.
  3. Check for any network communication interruptions between the client and RMA.
  4. Resolve any network issues to ensure stable communication between the client and RMA.
