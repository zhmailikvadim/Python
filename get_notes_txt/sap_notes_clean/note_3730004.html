Symptom

Starting on March 19th 2026, substitutions can not be submitted via task Center because the user search is failing.

Error in the screen:

User details cannot be displayed at the moment.

Following request failing: /comsapbpminbox-service.comsapbpmtcinbox/task-center-service/v1/users?$search=...

error": {

        "code": "sap.taskcenter.internal.server.error",

        "message": "Failed to send audit log event: 403 Forbidden: \"{\"error\":{\"code\":\"0403\",\"message\":\"AlsRegion doesn't match\"}}\"",

        "details": []

    }

}

or

{
    "error": {
        "code": "sap.taskcenter.internal.server.error",
        "message": "Failed to send audit log event: 401 Unauthorized: [no body]",
        "details": []
    }
}

Environment

SAP Task Center

Reproducing the Issue

  1. Open Task Center
  2. Navigate to the substitution Management area
  3. click on add substitution
  4. Try to search for any user
  5. error message displayed

Cause

Regression issue.

Resolution

SAP is aware of the issue. The development team is working on providing a Fix as soon possible.
