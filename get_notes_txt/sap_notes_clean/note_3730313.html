Symptom

Part-time employees (mini jobbers) who are subject to PI incorrectly receive the indicator for the active pension (field Curr. Pension in payroll table ST with A). The prerequisite is that all other criteria for the asset-side annuity have been met.

Other Terms

Active pension, CL_HRPAYDE_AR_CHECK, SV_CHECK

Reason and Prerequisites

The part-time employee pays PI contributions

Solution

Implement the attached correction instructions or import the specified support package.

After you import the HR Support Package or implement the correction instructions, run payroll again for affected cases.

Part-time employees are determined using the SI attribute '05' in the infotype Social Insurance D (0013) and excluded from the asset-side annuity.
