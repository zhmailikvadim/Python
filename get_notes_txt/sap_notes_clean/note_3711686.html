Symptom

While using Asset Workbench functionality to create assets in Guided Buying, and it is possible to split a single line item with multiple quantities into several individual line items so that an individual asset can be created for each.

Cause

There is no such limit defined in Asset Workbench in Guided Buying.

Resolution

There is no such limit in Asset Workbench in Guided Buying. It respects the parameter: Application.GuidedBuy.Limits.RequisitionLineItemMax as the line items created in Asset Workbench are treated as normal line items only. Hence, whatever the value set in above parameter defines the maximum number of line items created in a PR. 

The default value is 200 and the maximum value is 1000 for this parameter.
