Symptom

You want to change the default language of S/4HC fiori launchpad from American English to others that are not listed in the User Profile-> Settings-> Language and Region e.g British English. 

Environment

SAP S/4HANA Cloud Public Edition

SAP Fiori Launchpad 

Reproducing the Issue

  1. Open the User Profile in the S/4HC fiori launchpad.
  2. Go to Settings.
  3. Open Language and Region.
  4. Select Language as English.
  5. Observe that the default language is displayed as American English and English (UK) is not listed separately.

Resolution

You have the flexibility to set the default language of launchpad to Browser Language. After that, maintain the browser language as English (United Kingdom), then the default language in User Profile-> Settings-> Language and Region will be shown as British English.

  1. In the S/4HC fiori launchpad, open the User Profile and go to Settings -> Language and Region.
  2. Set the language to Browser Language.
  3. In your web browser, select English (United Kingdom) as the browser’s language.
  4. Fully close all browser windows and restart the browser.
  5. Re-login the system and verify that the default language now displays as British English in User Profile-> Settings-> Language and Region.
