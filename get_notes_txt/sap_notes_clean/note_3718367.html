Symptom

Customizing to generate mirror transaction has been done in IMG activity "Intercompany Trading: Distribution of Mirror Transactions. However in transaction SM58, the below error messages can be found during debugging: 

  • E FTR_GUI 141 Fill the following required field: VTBFHAZU-NORDEXT 000000 VTBFHAZU-NORDEXT

Environment

  • Treasury and Risk Management (TRM)
  • SAP ERP
  • SAP ERP Central Component
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA Finance

Reproducing the Issue

  1. Tcode FTR_CREATE to create a transaction;
  2. Tcode FTR_EDIT to display the transaction, from Menu -> Environment -> Object link is greyed out.
  3. Tcode SM58 to check the RFC.

Cause

The configuration for the given transaction requires the external reference field (VTBFHAZU-NORDEXT) to be set as "Required". This causes an error during the mirroring process.

Resolution

  1. If possible, remove the external reference field (VTBFHAZU-NORDEXT) from being set as "Required".
  2. Alternatively,  Implement the modification described in SAP Note 682541.
  • This modification involves commenting the following line of method MIRROR_DEALDATA in the class CL_IM_FTR_TR_MIRROR_DEALS:

PE_MIRROR_TRANSACTION-TRANSACTION-A_ACTIVITY-NORDEXT 

  • Note: This modification represents a source code change. SAP does not accept responsibility for errors resulting from this modification, and it will not be available in higher releases. Refer to SAP Note 170183 for more information.
