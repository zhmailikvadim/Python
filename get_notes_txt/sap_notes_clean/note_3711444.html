Symptom

When running an existing import job in SAP Analytics Cloud, you get an error saying "Unable to connect to S/4HANA due to missing credentials".

It may be the case that if you hover your mouse over the "Refresh" button, it'll show you a message saying which user the import job and connection details belong to.

Environment

SAP Analytics Cloud 2025.21

Reproducing the Issue

  1. Go to the modeler and access your model
  2. Go to the Data Management tab
  3. Run the existing import job or check the past executions
  4. Notice that it fails with the error "unable to connect to S/4HANA due to missing credentials"

Cause

The import job is running with another user's credentials.

Resolution

If you hover your mouse over the "Refresh" button, it'll show you a message saying which user the import job and connection details belong to.

That user must share the credentials in the connection creation screen. If that is not possible or not within the business case, you must create a new import job.
