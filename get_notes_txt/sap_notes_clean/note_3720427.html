Symptom

• An error occurs when users attempt to navigate from the main results table in the Fiori App "Display G/L Account Balances" (F0707A) to the detail view using the amount links.  
• The application does not load successfully and displays a navigation error.  
• Error message: App could not be opened either due to an incorrect SAP Fiori launchpad configuration or a missing role assignment. Failed to resolve navigation target '#GLAccount-displayGLLineItemReportingView'  

Environment

• S/4 HANA On-Premise  
• Private Cloud  
• Reporting Apps  

Reproducing the Issue

1. Navigate to the main results table in the Fiori App "Display G/L Account Balances" (F0707A).  
2. Select the amount links to access the detail view.  
3. Note that the application fails to load and displays a navigation error.  

Cause

• The issue is linked to the target mapping for the intent "GLAccount-displayGLLineItemReportingView," which is necessary for navigation.  
• This target mapping appears to be unassigned to the user account.  

Resolution

1. Assign the user a role that includes a catalog, such as `SAP_TC_FIN_ACC_COMMON`, which contains the necessary target mapping for the intent.  
2. Alternatively, reference this target mapping in your custom catalog.
