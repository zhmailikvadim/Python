Symptom

The XML file format created by the Hungarian EC Sales and Purchases List report in the statutory reporting framework does not meet the legal requirements for the calendar year 2026.

Other Terms

Hungary, HU, S/4HANA, EC Sales List, EC Sales and Purchases List, 26A60, HU_ECSL_V1

Solution

When creating the EC Sales and Purchases List using the Run Statutory Reports app, the form ID is generated dynamically based on the last two digits of the calendar year with the A60 suffix (for example, 26A60 for 2026 and 27A60 for 2027).

For more information about the development delivered with this SAP Note, see the entry for this SAP Note in the What's New topic on SAP Help Portal: https://help.sap.com/docs/SAP_S4HANA_ON-PREMISE/83fb257602634259bf40b22d80c1f563/e605324da4704b199b99d8adbd3733b2.html

As a general rule, SAP recommends that you install this solution by applying a Support Package. However, if you need to install it earlier, it is recommended to utilize the most current version of Note Assistant to apply the corrective instructions. To ensure that Note Assistant is updated, please refer to SAP Note 875986 for NetWeaver version 7.0x or SAP Note 1668882 for NetWeaver versions 7.3x and higher.
