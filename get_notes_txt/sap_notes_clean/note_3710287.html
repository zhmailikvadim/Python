Symptom

  • I am working with accruable rates and the existing configuration is allowing workers to submit time against this rate.
  • I would like to understand if there is a way to restrict workers from adding time on accruable rates, when submitting their timesheets

Reproducing the Issue

  1. Create a Rate Schedule with accruable rates and associate this to a Job Posting Template
  2. Create a Job Posting and hire a worker
  3. When submitting a timesheet, the worker will have the option of entering time against an accruable rate

Cause

The accruable rate has the Allow Worker to Draw Down on Time Sheet field set as Yes. This field is displayed if Quantity is selected in the Accrue Quantity or Amount field. If Yes is selected, workers are able to submit time against this rate which will be deducted from their accrual balance.

Resolution

Create a new accruable rate with the Allow Worker to Draw Down on Time Sheet field set as No. This will restrict workers from entering time against the rates on their timesheets.

There will still be a Edit Worker Accruals option, under the Actions button of the work order, where a buyer user with enough permission can manage the accrual balance.

Note: Due to several complex correlations with other elements in the system, this Allow Worker to Draw Down on Time Sheet setting cannot be edited on existing rates.
