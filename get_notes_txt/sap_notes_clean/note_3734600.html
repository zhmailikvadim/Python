Symptom

 API_BILL_OF_MATERIAL_SRV to create a BOM item fails with errors:

  •  "Bulk material not allowed for items relevant to costing" (29/127) 
  •  "Consistency check when creating BOM item was not successful" (BOM/201)

Environment

SAP S/4HANA Cloud Public Edition

Cause

A defect introduced with release 2602.1 causes failures when creating a BOM item for below scenario

Production relevant ='X', costing relevant = '', bulk indicator = 'X'. 

Resolution

Consume the fix delivered with HFC08 (planned rollout on 16 April).

Keywords

Bulk material not allowed for items relevant to costing, Consistency check when creating BOM item was not successful, Production relevant ='X', costing relevant = '', bulk indicator = 'X', API_BILL_OF_MATERIAL_SRV
