Symptom

In the process "Device Configuration Change (DSO View)" (DE_B_IM_DEV_CONFIG_CHG), step 0030 "Collect Business System Data" raises an unnecessary clarification case for non-IMS cases because no parameterization instance was available. The system determines parameterization instance data for all cases, including non-IMS cases ,where this data isn't available.

This issue is relevant for the market roles: DSO, SUP.

Other Terms

IM4G, DE_B_IM_DEV_CONFIG_CHG, BM_UOP_CONF_CHG_REQ, IMS, parameterization instance, clarification case, Collect Business System Data

Reason and Prerequisites

The error occurred because the system determined the parameterization instance for all cases, including non-IMS cases where this determination isn't necessary.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the system determines parameterization instance data only for IMS cases. You can reprocess step 0030 from the clarification case. You can verify the fix by checking that step 0030 in the process DE_B_IM_DEV_CONFIG_CHG is processed without error for non-IMS cases.
