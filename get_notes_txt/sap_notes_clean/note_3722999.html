Symptom

Material memo text disappears after some time and does not store permanently. Status message displayed: "Text has been saved temporarily in the main memory, see long text"

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Environment

SAP S/4HANA Cloud Public Edition

Cause

The text is not yet stored in the text data base, but placed in a main storage area.

Resolution

  1. In the "Material Memo" tab, enter the memo text and save it;
  2. Click "Text → Exit" or "Goto → Back" to properly exit the text editor;
  3. Save the entire material master record to commit text to database;
  4. Go back to the Change Material screen;
  5. Go into this material again and check that the material memo still exists.
