Symptom

In SOS Gas DE_B_GELIGAS_SOS_DSO process step 0385 - 500 the customizing shows Trigger Process-ID DE_B_GPKE_DSO_SBS, which generates a Start of Basic or Backup Supply for Energy instead of Gas. We expect the customizing here to call DE_B_GELIGAS_SBS_DSO instead.

Other Terms

Start Linked Process: "Start of Basic or Backup Supply", DE_B_EMBED_TRIG_SBS

Reason and Prerequisites

Program Error

Solution

Please implement the correction instruction in the SAP Note, or import  the higher version of Support Package.

This SAP Note is for the following roles: DSO.
