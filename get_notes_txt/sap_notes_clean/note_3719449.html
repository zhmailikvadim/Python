Symptom

You are using Production Planning and Detailed Scheduling (PP/DS) for SAP S/4HANA and facing CIF failure and ST22 dump with error  "CONVT_NO_NUMBER" while posting Goods Receipt.

Other Terms

PP/DS, Production Planning and Detailed Scheduling, PP/DS for SAP S/4HANA, Synchronized Planning for Production, Production Planning Integration, Flexible Constraints

Reason and Prerequisites

CIF failures for Good receipts postings for orders with the error "CONVT_NO_NUMBER" due to high quantity

Solution

As general rule, SAP recommends that you install a solution by applying a Support Package. However, if you need to install a solution earlier, use the Note Assistant to implement the correction instruction.
