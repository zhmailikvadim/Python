Symptom

Checking the database options from SYSPTIONS table we can see an option is existing twice:

  • MPX_DAS_Communication_Enabled .
    and
  • Mpx_DAS_COMMUNICATION_ENABLE

Environment

SAP IQ 16.1

Resolution

This issue only appears in server that have been created older version (16.0 or earlier) and have been migrated to SAP IQ 16.1.
Actually the 2 options are not really identical, pay attention to the CASE : Mpx.... and MPX...

  • SAP IQ 16.0 -- Mpx_DAS_Communication_Enabled
  • SAP IQ 16.1 -- MPX_DAS_Communication_Enabled

The option name referenced by SAP IQ16.1 is MPX_DAS_Communication_Enabled. The setting value of Mpx_DAS_COMMUNICATION_ENABLE will be ignored.
