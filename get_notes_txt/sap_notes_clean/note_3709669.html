Symptom

An issue is encountered while Note 3067086  is being implemented and the error message “Not all corrections applied” is received in SNOTE. 

Environment

  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

Attempted to implement Note 3067086, after implementing the pre-requisite Note and followed all the manual activities outlined in Note 3088419. 

You will end up with the error message "Not all corrections applied" in SNOTE. 

Cause

All the required Notes are not implemented. 

Resolution

To resolve the issue follow the below mentioned steps:

  1. Implement Note 2968536 - Harmonize the behavior for Linking Customer and Vendor to a single Business Partner.
  2. The implementation of Note 3067086 is then to be reattempted.
  3. The Note 3067086  will be implemented successfully.
