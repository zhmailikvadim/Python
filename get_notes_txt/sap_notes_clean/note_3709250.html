Symptom

Error message: "To associate the attachment XXX with ID NNNNNN" occurs during Additional data collection step when attempting to save uploaded attachment.

Environment

SAP SuccessFactors Onboarding

Cause

In Attachment Field Configuration for Attachment, the Entity ID is set incorrectly.

Resolution

  1. Go to Object Definition and find the custom object
  2. Add a custom field under Fields. Enter the name of the field as cust_subjectUser, referring to Configuring Data Collection Object Entities Guide
  3. Click "Details" associated with the cust_subjectUser field.
  4. Select "User" from the Data Type dropdown.
  5. Select "Read Only" from the Visibility dropdown.
  6. Click "Done".
  7. Create a rule to set value, referring to Creating a Business Rule to Set up a Data Collection Object Entity for Read Audit and Change Audit 
  8. Go to Manage Data >> Attachment Field Configuration for Attachment

  9. Set the Entity ID to cust_subjectUser, referring to Attachment Field Configuration Guide
