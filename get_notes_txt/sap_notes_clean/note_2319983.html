Symptom

  • Try to open an Lumira document with the SAP BI SDK App 6.x , error message "Failed to download Lumira Viewer".
  • Same document open fine with SAP Mobile BI App Version installed over the app-Store

Environment

  • SAP BI Mobile 6.3.x SDK
  • SAP BI Mobile 6.4.x SDK
  • SAP BusinessObjects Business Intelligence Platform 4.1
  • SAP BusinessObjects Business Intelligence Platform 4.2
  • Lumira  Add-On 1.29,1.30,1.31

Reproducing the Issue

  1. open SAP BI Mobile 6.3.12 SDK on Xcode and run the app
  2. open a Lumira document
  3. message :"Failed to download Lumira Viewer"

Cause

  • Lumira viewer does not support Mobi SDK.

Resolution

Update: SAP Lumira is now supported through SDK  strating from SAP BI Mobile 6.5.x SDK
