Symptom

During payroll execution for certain staff members, the payroll run terminates with a dump CX_SY_PROVIDE_TABLE_NOT_SORTED. The issue occurs while processing Entitlement Validation Engine (EVE) entitlements for employees with multiple records in the Flexible Work Arrangements (3446) infotype, interrupting the payroll process.

Other Terms

  • FWA
  • 3446
  • EVE (Entitlement Validation Engine)
  • Entitlement
  • CX_SY_PROVIDE_TABLE_NOT_SORTED
  • GET_FWAEN
  • CL_HRPADUN_ATTRIBUTE_READER

Reason and Prerequisites

Program error.

This SAP Note is relevant only for non-profit organizations.

Solution

After implementing this SAP Note, the payroll runs successfully for all the staff members without any runtime error.

To obtain the changes, import the Support Package specified for your release.

Alternatively, apply the attached correction instructions.

As a general rule, SAP recommends that you implement the Support Package specified for your release. However, if you need to install a solution earlier, use the SAP Note Assistant to implement the correction instruction. You can find more information about the SAP Note Assistant in SAP Support Portal, under SAP Note Assistant.
