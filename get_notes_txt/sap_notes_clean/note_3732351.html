Symptom

You have used transaction code /BA1/BR_PCSTAT_RESET for re-processing of contracts. Then, in a later Period End processing, the system creates warning messages regarding incorrect Legal Entity-assignments "Incorrect properties...".

Other Terms

transaction code /BA1/BR_PCSTAT_RESET - Include Contracts in Day Processing Again

warning messages with message class /BA1/MSG_AL_PC, message number 101, like:

"Incorrect properties from contract &1 (cat. &2, part. key. &3 date &4)"

Reason and Prerequisites

code error

Solution

Please delete the inconsistent contracts (having wrong Legal Entities and process ID 4005) from DB-table /BA1/FC_REGCAP manually, e.g. via an own SQL statement.
