Symptom

Users need guidance on how to properly set up Joule Studio, a component of SAP Build, together with SAP Build Process Automation. They require step-by-step instructions to ensure both solutions are configured on the same Cloud Identity Service (CIS), correctly integrated, and properly linked.

Environment

  • SAP BTP, Cloud Foundry runtime and environment
  • SAP Build Apps
  • Joule

Resolution

  • Joule Studio is one of the components of SAP Build and enables to create and augment enterprise applications, process automation, and business sites with drag-and-drop simplicity. Allows the user to perform the Integration with SAP Build to support various SAP Build Services.
  • An outline of the prerequisites for both customers and partners. Understanding these requirements ensures proper activation and use of SAP Build services and avoids compatibility issues.
  • When setting up Joule Studio, there are two options available:
    • The first option is to run the booster to connect Joule with SAP Build Process Automation.
    • The second option is available if you have already set up and configured Joule, in which case you can add SAP Build Process Automation to a formation. For more information on different set-up scenarios, see Setup Scenarios for Joule Studio.
