Symptom

There's the requirement to add custom fields in an approval-related email template, e.g. SD_SLS_SO_APPROVAL_EMAIL_TMPL.

Environment

SAP S/4HANA Cloud Public Edition

Cause

Currently, in SAP S/4HANA Cloud Public Edition, it's not possible to extend approval-related email templates with custom fields or additional standard fields. 

The reason for it is that the data sources used for approval email templates can't be extended, thus it's not possible to add new fields to them. Additionally, it's not possible to change the data source of an email template.

Resolution

It's completely understandable how the possibility of adding additional fields or custom fields would be of great use for specific business, so please consider creating a feature request for this business requirement. This will inform the responsible developers as to what additional functionality should be considered for future releases.
