Symptom

When using Enterprise Search in the SAP Business One client application, documents from a numbering series that the users lack authorization for are still displayed.

Reproducing the issue

Prerequisites:

After initializing an SAP Business One company in the Analytics Enablement console, the Enterprise Search feature is available.

Scenario:

  1. Add one SAP Business One user to a user group.
  2. Create a new document series with the manager user.
  3. Remove authorizations to the new document series for user group configured in point 1.
  4. Create a new document on the new series with the manager user, and without a specified owner.
  5. Log on to the SAP Business One client application with the user from point 1.
  6. Search for the document created in point 4.

Expected behavior:

Enterprise Search should exclude marketing documents from a numbering series for which a user group does not have authorization.

Actual behavior:

Enterprise Search does not exclude marketing documents from a numbering series for which a user group does not have authorization.

Solution
