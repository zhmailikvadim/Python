Symptom

When making a change on a service contract at item level, the following error displays. 
BDR &1 cannot be rejected while it has the invoiced status.

Environment

SAP S/4HANA Cloud Public Edition

Cause

To keep consistency between a service contract and the corresponding billing document request, once making some changes on a service contract at item level, the system will trigger the deletion of the subsequent billing document request. If the subsequent billing document request is already invoiced and posted, it's not supported to be deleted anymore.

However, if the status of the billing document request was changed to "Invoiced" by generating a preliminary billing document, it can be solved by the steps in the Resolution part.

Resolution

  1. Find the preliminary billing document in the "Manage preliminary billing document" app
  2. If the status is "Finalized", click the "Set to In Progress" button to set the status back to "In Progress".
  3. Click the "Reject" button to reject the preliminary billing document.
