Symptom

Information on the installation of SAP Best Practices for S/4HANA (on premise) (Belgium) (BEV4)

Other Terms

Best Practices, S/4HANA

Reason and Prerequisites

This note is only relevant for the following SAP Best Practices version: SAP Best Practices for S/4HANA (on premise) (Belgium) (BEV4)

Solution

This SAP Note contains additional information on the installation and
documentation of the SAP Best Practices for S/4HANA (on premise) (Belgium) package

1. Software and Delivery Requirements and Administration guide for SAP S/4HANA solution implementation
Before you start the implementation of the SAP Best Practices for S/4HANA (on premise) (Belgium) package read carefully the following documents:

  • Software and Delivery Requirements documentation:

    https://support.sap.com/content/dam/SAAP/Sol_Pack/BP_OP_ENTPR/BP_OP_ENTPR_S4HANA1709_04_Requirements_EN_XX.docx

  • Administration Guide for SAP S/4HANA OP Solution Implementation: https://help.sap.com/viewer/S4HANA_1709_AdminGuide

2. Installation Preparation

2.1 SAP NOTES
Ensure that the following SAP Notes are implemented in addition to the notes already listed in the Software and Delivery Requirements document or listed in the Administration Guide for SAP S/4HANA OP Solution Implementation or listed in the FPS specific note that is listed in the collective note 2446206 "SAP S/4HANA, on-premise edition 1709 collective note for content activation": None.


2.2 Correction transport
All corrections will be offered as a collective transport request which will be attached to a FPS specific note that is listed in the collective note 2446206 "SAP S/4HANA, on-premise edition 1709 collective note for content activation".


2.3 Installation: Trouble Shooting (errors that might occur during activation)
See the Administration guide for SAP S/4HANA solution implementation for information on general error handling. It is very important to read, understand and follow the generic guidelines on how to handle known activation errors.

Detailed descriptions of workarounds per known activation error can either be found in a FPS specific note that is listed in the collective note 2446206 "SAP S/4HANA, on-premise edition 1709 collective note for content activation" or are attached to this note (see documents 1709_OP_nnn_XX.docx - nnn is the Building Block ID).

See the Administration guide for SAP S/4HANA solution implementation for information on general error handling. It is very important to read, understand and follow the generic guidelines on how to handle known errors.

Detailed descriptions of workarounds per known usage error can either be found in a FPS specific note that is listed in the collective note 2446206 "SAP S/4HANA, on-premise edition 1709 collective note for content activation" or are attached to this note (see documents 1709_OP_nnn_XX.docx - nnn is the Scope Item ID).
