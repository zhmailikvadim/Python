Symptom

Even though the XML file for the output document contains the correct data, some fields in the Adobe form still appear blank when the form is generated.

Environment

SAP S/4HANA Cloud Public Edition

Cause

There is a binding issue in the form template, caused by incorrect or missing configuration in the output document setup.

Resolution

This community post highlights the importance of the interface context and how to ensure that form fields display the correct data.
It explains the concept of data binding for custom output forms such as invoices, delivery notes, and purchase orders.

Refer to: Data Binding Basics for Print Forms - SAP Community
