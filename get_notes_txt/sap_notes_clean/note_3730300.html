Symptom

Old CPE Reference date routine 9999997 returns End date or current date ( whichever is higher ), but none of the Advanced CPE new routines
provides the same functionality. So, a new routine will be required to provide the same behavior.

Other Terms

Agricultural Contract Management; ACM; Reference date Routine; Advanced CPE Routine BC set;/ACCGO/NGCPE_REFDATEROUT_BCSET; Advanced CPE ;

Reason and Prerequisites

New routine need to be introduced in Business configuration set.

Solution

Import the relevant support pack or apply the correction instruction(s).

This SAP Note delivers new routine 19005 in Business configuration set - /ACCGO/NGCPE_REFDATEROUT_BCSET for Reference Date.
