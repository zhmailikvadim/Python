Symptom

  • How to solve errors like ERROR ( message:Failed to add duplicate collection element "/userdata". )

  • You get an Error (or a couple of them) in the LiveCompare Configuration Wizard logs similar to:

    ERROR ( message:Failed to add duplicate collection element "/userdata". )

    This error may be included in the logs generated after running the LiveCompare Configuration Wizard. 

  •  Scenarios that can cause these type of errors:
    - LiveCompare is already installed and correctly configured on your machine and you re-ran the LiveCompare Configuration Wizard. -> In this case the errors would be expected behaviour.
    - You previously had an installation of LiveCompare on the same machine, managed to uninstall it but it was not de-configured properly, ultimately leading to issues with configuring the new LiveCompare version. This would be an issue because the configuration of the old version would still be in place, preventing the new configuration to be carried out.
