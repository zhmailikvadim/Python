Symptom

You have created a location which always leads to the creation of an according suppy chain unit. Besides, the usage of the address for the supply chain unit has been stored internally.

When you now delete this location per report /SAPAPO/DELETE_LOCATIONS, the assigned roles of the suply chain unit remain persisted as well as the usage of the address for the supply chain unit.

Other Terms

SCU, /SCMB/TOEBCASS, ADRV

Reason and Prerequisites

program error

Solution

Implement the attached correction instructions or import the relevant support package.
