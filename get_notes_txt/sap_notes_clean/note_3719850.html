Symptom

Previously, in the IR56 reports, if the wage amount exceeded the maximum length limit, the program stopped and displayed an error.

This SAP Note adds validation for digit length in IR56 reports.

If the wage amount exceeds the limit, the system displays an error message and continues processing: "The wage type number exceeds the maximum length limit of nine digits."

Other Terms

PCTAXHKF, IR56

Reason and Prerequisites

This is an enhancement.

Solution

Do either of the following:

  • Apply the relevant Support Package.
  • Implement the corresponding correction instructions.
