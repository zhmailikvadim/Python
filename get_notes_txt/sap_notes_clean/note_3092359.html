Symptom

When you change rate data using transaction EG70, a termination occurs during saving and the system returns the error DBSQL_DUPLICATE_KEY_ERROR for the EABL table.

Other Terms

SAPLEL25, ISU_DB_EABL_UPDATE

Reason and Prerequisites

The device has several registers. The change of the rate data does not affect all registers. On the day before the activity date, there are already meter reading results with meter reading reasons 09 or 10.

Solution

Implement the correction instructions or import the support package.
