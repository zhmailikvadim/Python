Symptom

The system automatically determines the main packing instruction, but no dialog is shown to allow user selection, even when alternative packing instructions exist.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Set the "Packing Instruction Selection Dialog" and "Respect Alternative Packing Instruction" indicators in the customizing activity "Maintain Packing Transaction Profile".
  2. Trigger automatic packing process in the delivery.
  3. Observe that the system automatically determines the main packing instruction without displaying a selection dialog for alternative packing instructions.

Cause

The indicator for 'packing instruction selection dialog' specifies that the system displays a selection dialog box in automatically started packing instruction determination when the "Automatic start" indicator is flagged.

Resolution

Ensure that the "Start automatically" option is also checked for the corresponding profile e.g. 0001 inbound delivery in the customizing activity 103717 "Maintain Packing Transaction Profile".
