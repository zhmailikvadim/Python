Symptom

After manually restarting an onboarding process, the value input within Additional Data Collection step doesn't retain.

Environment

SAP SuccessFactors Onboarding

Reproducing the Issue

  1. Start onboarding process
  2. Complete the Additional Data Collection step
  3. Restart the onboarding process manually
  4. Notice the values that were previously given in the Additional Data Collection step are not saved.

Cause

This behavior is expected and aligns with the system’s design specifications.

Resolution

The Retain Onboarding Information After Manual Restart setting applies only to New Hire Data Review and Personal Data Collection.

Keywords
