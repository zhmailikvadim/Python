Symptom

  • When searching for master contracts in the Manage Sales Contracts (F1851) and Manage Sales Contracts – version 2 (F5987) apps, no results are found.
  • There is no dedicated Manage Master Contracts app, so master contracts would be expected to be handled in apps F1851 and F5987.

Environment

  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Open app Manage Sales Contracts
  2. Enter a Master Contract number in Sales Contract field and click on Go
  3. No results are returned

Cause

The apps F1851 and F5987 were not designed to include Master Contracts. They select from the CDS View I_SALESCONTRACT and there is a precondition on the sales document category:

"

define view I_SalesContract
as select from I_SalesDocument
...
where SDDocumentCategory = 'G';
...

"

In transaction VOV8, master contracts have a category value of '0' and therefore do not satisfy the above precondition.

Resolution

  • This is missing functionality: there is no dedicated app for master contracts.
  • Master contracts can be displayed using the VA43 app.
