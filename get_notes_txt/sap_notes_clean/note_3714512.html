Symptom

You are performing Client Copy. The following error occurred.
 
Error:
At least one table failed in SELTAB_SPLITTER
 
You can implement Note 3281364 to avoid any known issues as above. Before that, you want to know, whether it is OK to reproduce the issue with "Test Mode". And then run "Test Mode" again after implementing Note 3281364 to see whether the issue is resolved or not.

Environment

S/4HANA.

Reproducing the Issue

Run Client Copy with "Test Mode".

Resolution

This error will also occur in "Test Mode". It is OK to reproduce it with "Test Mode". And then run "Test Mode" again after implementing Note 3281364 to see whether the issue is resolved or not.
