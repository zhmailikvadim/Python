Symptom

When creating PAM ID, IAG tenant currently sends a connection name via RFC to your ERP or S/4HANA system.

Other Terms

IAG PAM Privileged Access Management Connector Connection

Reason and Prerequisites

Sending connection name from IAG tenant via RFC is not safe. To mitigate this, IAG has decided to use one unified 'SAP_IAG_RFC_PAM_LOGON' RFC connection.

Before implementing this SAP Note, please make sure you have completed steps from:

https://help.sap.com/docs/SAP_CLOUD_IDENTITY_ACCESS_GOVERNANCE/8927ff487e3e4520b3211167b7f06c31/f3e554c1fc7c4307a4ef8808ff359996.html?version=LATEST&locale=en-US

The same steps are also described in SAP Note 3389398

Solution

After completing the setup steps mentioned above, please implement the attached correction instructions.
