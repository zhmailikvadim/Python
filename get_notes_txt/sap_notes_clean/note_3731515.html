Symptom

However, it is currently not possible to hide individual country-specific input fields.

Other Terms

H99_B2AMANAGER, H99_B2AMANAGER_FORMS, form MODIFY_SCREEN

Reason and Prerequisites

In the B2A Manager, some input fields are provided for selecting the processes. Country-specific additional data can be added. However, up to now, it was not possible to hide standard input fields other than the country grouping (MOLGA).

Solution

Implement the attached correction instructions or import the specified support package.

After you import the HR Support Package or implement the correction instructions, you do not have to perform any further follow-on activities.

The individual countries can decide whether you want to add, hide, or show individual input fields.
