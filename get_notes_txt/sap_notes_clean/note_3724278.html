Symptom

When attempting to customize the Contacts section in the Account Overview, it is not possible to modify the fields displayed in the Card View.

For example: if the requirement is to replace the standard field "Function" with "Job Title" in the Card View of the Contacts table, it is not possible to modify it via Adaptation. 

Environment

  • SAP Sales Cloud Version 2
  • SAP Service Cloud Version 2

Reproducing the Issue

  1. Navigate to Accounts.
  2. Open any Account. 
  3. Enter Adaptation Mode.
  4. Scroll down to the Contacts section.
  5. Attempt to modify the Card View of any contact (e.g., try to change displayed fields).
  6. It is not possible to modify it. 

Cause

Resolution

According to Product Management, the guidelines for which changes will be supported in the Card View are currently still being evaluated by the Platform (UI) and UX teams. At the moment, there is no estimated timeline available for when these changes will be implemented.

Therefore, it is currently not possible to adapt the card view.
