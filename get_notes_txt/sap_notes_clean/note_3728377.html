Symptom

A redirect (parameter "icm/HTTP/redirect_X") was configured in the SAP Web Dispatcher, so the root URL ("/") gets redirected to a commonly used service, like:

  • WebGUI - /sap/bc/gui/sap/its/webgui
  • Fiori LaunchPad - /sap/bc/ui2/flp
  • or others

However, the redirect does not work.

An HTTP trace from the browser reveals that a "403 forbidden" response is received.

If the SAP Web Dispatcher is hosed on SAP ECS (former HEC), the ECS standard "service not available" page is received.

Environment

  • SAP Web Dispatcher

Cause

The root URL ("/") is not allowed on the permission file.

Resolution

The root URL ("/") is not allowed by the permission file (option "PERMFILE" from the parameter "icm/HTTP/auth_0").

Add the following line to the permission file:

S /

Caution: do not add a "*" at the end of the line. Otherwise, all URLs will be allowed, which is not desirable from a security perspective.
