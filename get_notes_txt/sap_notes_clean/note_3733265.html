Symptom

When Distributors (DSO) receive a 55643 (MeLo data) from the MOS containing a time-of-use code, an error occurs in step 5500-0005 (Create Parameterization Instance) of the process DE_B_IM_DEV_CHANGE_DSO(Setup / Replacement of Devices (DSO View)). The error message will be like "XXX is not a valid external number for a service provider".

Other Terms

ToU, DSO, DE_B_IM_DEV_CHANGE_DSO, Service Provider

Reason and Prerequisites

This is a program issue.

Solution

Implement this SAP Note.
