Symptom

After March 1, 2026, the system does not apply the correct National Minimum Wage rate. To address this issue, you must implement the SAP Note as described in the regulatory change manager tool under 'Change in national minimum wage hourly rate, effective March 01, 2026'. This SAP Note provides the relevant system modifications for the National Minimum Wage that take effect on March 1, 2026.

Other Terms

NMINW, T511P, V_T511P, National Minimum Wage, RPCALCW0, PC00_M16_CALC.

Reason and Prerequisites

Legal change.

This SAP Note is relevant only for South Africa localization.

Solution

After applying this SAP Note, the SAP system updates the value of the National Minimum Wage (NMINW) constant in the T511P table to ZAR 30.23, effective March 1, 2026.

SAP recommends that you implement the solution by using the Support Package specified for your release.

If you need to install this solution earlier, run the manual steps in 'Manual Instructions for 3713382.pdf' attached to this SAP Note.
