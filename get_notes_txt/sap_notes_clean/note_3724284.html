Symptom

  • A UI issue has been identified in the Company Structure Org Chart.
  • The issue involves a missing space between numeric counters and labels, which affects readability.

Environment

SuccessFactors Employee Central 2511

Reproducing the Issue

  1. Access Org Chart.
  2. Navigate to Company Structure > Department > Hierarchy Details.
  3. Observe issue of spacing.

Cause

This is a bug.

Resolution

  1. A code correction has been created under ECT-268418.
  2. The solution of this issue is currently under investigation. Please follow this KBA for updates.
