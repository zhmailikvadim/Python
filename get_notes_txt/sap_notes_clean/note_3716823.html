Symptom

When attempting to reschedule operations in app F2901, the user receives an error and the rescheduling action does not complete (malformed URI literal syntax).

Other Terms

UI5 Date/Time formatting, OData batch request, DateFormat

Reason and Prerequisites

Hardcoded date and time parsing/formatting patterns in the application logic do not consider user‑dependent locale settings.

Solution

Apply attached Correction Instruction. The correction adjusts the date/time handling to ensure that locale‑aware formatting is used and that the backend receives a valid timestamp. No additional post‑implementation steps are required.

For the op2023 version please refer to SAP Note 3721851.
