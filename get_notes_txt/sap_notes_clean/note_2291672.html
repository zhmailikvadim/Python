Symptom

The Process Type "Initial Activation and Filling of SAP HANA Index" is used to create a SAP HANA index for a Query as InfoProvider with external SAP HANA view.

Manually created SAP HANA views using this external SAP HANA view are invalid afterwards although the structure of the Query did not change.

Other Terms

RS2HANA

RSDDB

RSPC

RS2HANA_VIEW006; RS2HANA_VIEW017

Reason and Prerequisites

Program error

Solution

  • SAP BW 7.40

    Import Support Package 15 for SAP BW 7.40 (SAPKW74015) into your BW system. The Support Package will be available as soon as SAP Note 2252375 with the short text "Preliminary Version SAPBWNews NW BW 7.4 ABAP SP15", which describes this Support Package in more detail, is released for customers
  • SAP BW 7.50

    Import Support Package 4 for SAP BW 7.50 (SAPK-75004INSAPBW) into your BW system. The Support Package will be available as soon as SAP Note 2280438 with the short text "Preliminary Version SAPBWNews NW BW 7.50 ABAP SP4", which describes this Support Package in more detail, is released for customers.



    In urgent cases you can use the correction instructions.

    Before you use the correction instructions, make sure that you check SAP Note 1668882 and SAP Note 2248091 for transaction SNOTE.

    This SAP Note might already be available before the Support Package is released. In this case, however, the short text still contains the term "preliminary version".
