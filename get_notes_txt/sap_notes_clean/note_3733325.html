Symptom

Group currency and Local currency is shown on separate lines for the same data line in the output of any Group Reporting reports. For example in Group Data Analysis or Group Data Analysis with Reporting Rules. More details in 3192876 - FIN GR Analytics:Separation Logic.

Reason and Prerequisites

The separation logic was introduced because of Multi Group Currency feature.

Solution

Implement the attached Correction Instruction (CI) or the relevant Support Package.

After installation execute report FINCS_CUBE_PARAM_ADMIN with Parameter ID = USE_SEPARATION_LOGIC and initial Parameter value. With that the system will longer separate the data lines and you get one line with filled group currency and local currency. If you would like to enable the separation logic, either modify for USE_SEPARATION_LOGIC the value X or delete the parameter USE_SEPARATION_LOGIC.
