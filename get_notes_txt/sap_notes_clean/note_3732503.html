Symptom

  • Spaces are not available (service call has failed).
  • HTTP 500 error when accessing spaces.
  • [PoolAcquireError] Failed to acquire connection from pool using pool user REPODUMMY, requestUser is TENANTUSER_the id.
  • Original error message indicates: Database instance resize in progress, Code: 1894, SQL State: HY000.
  • HTTP Status: 500 error
  • Error Code: Technical Message: HANA Database instance resize in progress, Code: 1894, SQL State: HY000
  • Spaces are not available (service call has failed)

Environment

  • SAP Datasphere

Reproducing the Issue

  1. Open Space Management;
  2. Observe that the UI returns "Spaces are not available (service call has failed)" or an HTTP 500 error; backend logs may show a PoolAcquireError and/or "Database instance resize in progress" (Code 1894, SQL State HY000).

Cause

Regression

Resolution

Update on: March 25, 2026:

Related failed requests and runtime is back to normal, so no outages in US10 are expected.
Information regarding RCA will still be updated in this document.
