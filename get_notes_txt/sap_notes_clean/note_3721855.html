Symptom

SAP ME Database and Data Migration for Upgrade CTC task appears to hang or run for an extended period without progress.

Environment

  • SAP Manufacturing Execution

Reproducing the Issue

  1. Start the SAP ME Database and Data Migration for Upgrade CTC task that is running longer than expected.

Cause

As the CTC task has been running for a long time, the session times out.

Resolution

If you expect the CTC task to migrate a high volume of data, you must consider increasing or deactivating timeouts to prevent termination of the CTC task. In particular, it concerns the following timeouts: TransactionTimeout, ICM connection timeouts, and   Application specific timeout defined at NW layer. For more information, see SAP Note 1825121  

However if the session times out look for stuck sessions related to the CTC migration task and terminate them.
