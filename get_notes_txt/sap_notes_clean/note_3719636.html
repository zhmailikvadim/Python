Symptom

The following menu path for SAF-T Financial Data report in Bulgaria configuration is not available in SPRO: 

  • In case of classic GL solution: Financial Accounting -> General Ledger Accounting -> Business Transactions -> Closing -> Report -> Statutory Reporting: Bulgaria -> SAF-T
  • In case of new GL solution: Financial Accounting (New) -> General Ledger Accounting (New) -> Periodic Processing -> Report -> Statutory Reporting: Bulgaria -> SAF-T

Environment

  • SAP ERP
  • SAP_FIN 618
  • Bulgaria

Reproducing the Issue

  1. Implement collective note 3702420 - SAF-T Bulgaria: Collective Note (SAP ERP) and its prerequisites.
  2. Try to navigate to the above IMG path.
  3. Observe that the node for SAF-T Bulgaria is not visible in customizing.

Cause

The node for SAF-T Bulgaria is delivered with a Support Package (SP). If the SP is not available in the system, the node will not appear in customizing.

Resolution

  1. Upgrade SP to the latest.
  2. Alternatively, use SM30 views mentioned on SAP Help Portal or in attachments of notes to configure the SAF-T Bulgaria node manually.
  3. Refer to SAP Help Portal for detailed steps on using SM30 views for configuration.

See Also

Refer to SAP Note 3702420 - SAF-T Bulgaria: Collective Note (SAP ERP).
