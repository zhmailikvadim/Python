Symptom

IMG Configuration Nodes are missing for Personnel Cost Planning and Simulation

Environment

  • Human Capital Management (HCM)
  • Personnel Cost Planning and Simulation
  • SAP enhancement package 5 for SAP ERP 6.0
  • SAP enhancement package 6 for SAP ERP 6.0
  • SAP enhancement package 7 for SAP ERP 6.0
  • SAP enhancement package 8 for SAP ERP 6.0

Reproducing the Issue

  1. Open transaction SPRO
  2. Select node Personnel Management
  3. Click on Personnel Cost Planning and Simulation subnode
  4. There are missing configuration nodes

Cause

  • Business Function HCM_HCP_CI_1 has not been activated
  • EA-HR Extension has not been activated

Resolution

Business Function HCM_HCP_CI_1 should be active.
It can be checked in transaction SFW_BROWSER or SFW5.

Important:
HCM_HCP_CI_1 is available as of SAP enhancement package 5 for SAP ERP 6.0.
Business Function HCM_HCP_CI_1 will activate only new functionalities in PCP and change the Detail Planning application from BSP-UI to Web Dynpro.

For more information about the BF, please read the available documentation in transaction SFW5

EA-HR extension should be active as well.
It can be checked via transaction SM31, Table: TBE11 or transaction SFW5
