Symptom

You create a new version as a copy of an existing version. When you do this, the system issues the error message "Please select a routing" and the field for the routing group is not ready for input.

Other Terms

SFR, shop floor routing, MPE_RTG_UI 001

Reason and Prerequisites

This is caused by a program error.

The external number range of the number range object ROUTING_V is not maintained.

Solution

Implement the correction instructions.
