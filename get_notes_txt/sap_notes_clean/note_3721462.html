Symptom

When trying to add jobs to an existing job group in Focused Run Job & Automation Monitoring, the following error is returned: "Jobs not added as they are already part of the group"

Environment

SAP Focused Run 5.0 FP01

Reproducing the Issue

  1. Transaction FRUN

  2. Click on Job & Automation Monitoring

  3. Open the Configuration of the Job Group

  4. Try to add existing jobs to the group and an error will appear

Resolution

Implement the latest version of note 3608306 in the Focused Run system

See Also

3608306 — FRUN 5.0 FP01 Job & Automation Monitoring front-end correction

Keywords

Focused Run, Job & Automation monitoring, Job Group , Jobs not added as they are already part of the group
