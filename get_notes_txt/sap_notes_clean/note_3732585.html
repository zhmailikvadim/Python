Symptom

  • Invoice consolidation for Taiwan e-invoicing is confirmed to be supported only in s/4 HANA system(minimum supported release OP2020).

Environment

  • SAP ERP

Reproducing the Issue

N/A

Cause

Consolidation-related processes are not supported in SAP ECC system, and there are no current plans to add this functionality.

Resolution

  1. Please submit a Customer Influence (CIP) request for this requirement; refer to: 0000000011.
  2. As an alternative, consider adopting s/4 on-premise 2020 (op2020) or higher, where consolidation-related solutions are supported.

See Also

  • Refer to: 0000000011
  • Refer to: Maintaining Tax Number for Business Partners
