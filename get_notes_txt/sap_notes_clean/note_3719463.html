Symptom

Reading of EDM profile configuration depends on device & register data of the Installation. As installation data is not read correctly, this leads to no EDM profile creation.

Other Terms

/APEU, /UCOM, MCM, DER, Device, EASTL, AB, BIS, Key Date, GET_DEVICE_REGISTER_DATA, ISU_DB_EASTL_FORALL_ANLAGE, Installation, EDM Profile,

Reason and Prerequisites

APEU

2602

Solution

Please implement the attached correction instruction.

Key-date handling was corrected when calling FM ISU_DB_EASTL_FORALL_ANLAGE, ensuring that only devices valid on the specified key date are selected.
