Symptom

You are not able to add text in Portal reply option as it is grayed out in Responsive UI /Fiori Client.

Environment

SAP Cloud for customer

Reproducing the Issue

  1. Login to Fiori Client/ Responsive UI
  2. Go to Service Work Center
  3. Go to Tickets view
  4. Open any ticket of source 'Internet'
  5. Navigate to 'Interactions' tab
  6. Click on New or Reply button
  7. Navigate to Portal option
  8. Text box is grayed out

Cause

Necessary scoping question is not selected in your system.

Resolution

  1. Log in to HTML5 UI
  2. Go to Business Configuration Work Center
  3. Go to Implementation view
  4. Open Edit Project Scope
  5. Navigate until step 4 Questions
  6. Select Service-> Customer Care
  7. Select Communication for Customer Care
  8. Flag the question "Would you like to extend your customer service scenarios to include an external customer service portal?"
  9. Click on Next and Finish

Keywords

Portal reply, grayed, Fiori, RUI, ticket
