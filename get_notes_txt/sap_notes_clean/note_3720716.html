Symptom

Error "The file type does not match the connector type entered in the URL" when running upload via API

Cause

The error message "The file type does not match the connector type entered in the URL" occurs when the file type being sent from the API does not match the file type selected in the connector. 

Resolution

  1. Take the name of connector from the API url 
  2. Log in in Fieldglass with the CM user and locate the connector from Connector Catalog
  3. On the view screen the file type is not displayed
  4. Go to Edit the connector settings and ensure the file type matches the format being sent via the API (CSV or JSON).
  5. Save the changes and try again the API upload.
