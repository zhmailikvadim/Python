Symptom

In the Talent Pool, while using the "Import Successor" feature with the action "TRANSFER," user details are transferred except for the Notes field.

Environment

SAP SuccessFactors Succession Management

Cause

Expected Behavior.

Resolution

  1. Note that this behavior is by design: the NOTE field is not processed for the TRANSFER action in Import Successors.
  2. If notes are required, import the nomination using ADD or UPDATE action including the NOTE field to create or modify the note.
  3. Alternatively, after performing the TRANSFER, update the Notes field via the user interface.

See Also

refer to: KBA 2105083 - Importing Successors in SuccessFactors
