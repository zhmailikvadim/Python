Symptom

Bank details are present at BP level in Payment Transactions tab, but missing in customer (KNBK) and vendor (LFBK) tables.

Environment

  • SAP ERP
  • SAP ERP Central Component
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Execute the DRFOUT transaction in the source system to replicate BP data to the target system.
  2. Verify the replication status in SRT_MONI; status shows as "Delivered" without errors.
  3. Check the target system table for bank details; data is missing.

Or

  1. Execute BP transaction
  2. Open a Business Partner
  3. Change any field
  4. Save it

Cause

A synchronization error occurred during CVI.

Resolution

  1. Run the MDS_PPO2 transaction and input the specified details in the selection criteria:
    • Object Type: BUS1006
    • Object Key: Enter the BP number
  2. Execute the transaction. If any errors are displayed, it indicates synchronization issues between BP and Customer/Vendor. These errors prevent customer/vendor data from being saved.
  3. Identify and resolve the root cause of the synchronization errors.
  4. Initiate synchronization from BP to Customer to update the missing data in the target system table.
  5. Perform BP replication again, save any data in BP transaction or use the MDS_LOAD_COCKPIT transaction to ensure successful synchronization.
