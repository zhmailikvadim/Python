Symptom

Certain fields (eg. district name, street 5, etc.) were not retaining the existing value while changing the connection object.

Other Terms

APEU, UCOM, DER, /UCOM/CL_ISU_CONNOBJ, CONNOBJ, Change,  /UCOM/CL_ISU_CONNOBJ~MODIFY, /UCOM/IF_ISU_CONNOBJ, /APEU/CL_PRST_CHNG_CONNOBJ

Reason and Prerequisites

 Program error

Release 2508

Solution

Please implement the attached correction instruction.

The retrieval of old connection object data was not present for all the fields so created a new method to retrieve the old data in the modify method.
