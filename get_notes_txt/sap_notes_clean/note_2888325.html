Symptom

Current SAP Note delivers necessary updates for IT3425 HR Renewal 2.0 functionality.

Functionality has been released to Support Pack already in 01/2020. But note wasn't released by mistake.

For comprehensive information about SAP solution for eBOL and eNDM process and its implementation see collective SAP Note 2879935.

Other Terms

HRSI, infotype 3425, eBOL, sickness certificate, HR Renewal.

Reason and Prerequisites

  • This SAP Note is relevant only for Slovenia

Solution

This SAP Note contains configuration changes needed for Renewal functionality of new infotype 3425.

Implementation

  • As a general rule, SAP recommends that you install a solution by applying a Support Package. However, if you need to install a solution earlier, use the Note Assistant to implement the correction instruction. You can find more information about the Note Assistant in SAP Service Marketplace, under service.sap.com/note-assistant
  • Note that documentation, transactions, translation and customizing activities are only delivered via Support Package
  • SAP Note 2872058 is the prerequisites before implementation of current SAP Note
