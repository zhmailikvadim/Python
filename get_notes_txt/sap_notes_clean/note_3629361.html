Symptom

In Display/Post Group Journal Entries application in S4HANA Group Reporting, you face performance issues or memory exception.

Other Terms

 C_CnsldtnJrnlEntryItemTP, C_CnsldtnJrnlEntr, C_CnsldtnJrnlEntryTP, SAP Note 3633203, PGJE

Reason and Prerequisites

You implemented SAP Note 3633203 - DGJE: Working with group journal entry might create not enough memory dump.

This note adds optimization to CDS view C_CNSLDTNJRNLENTRYITEMTP which is shared between PGJE and DGJE app. It also brings new build of BSP Project for PGJE application.

Solution

 Import of Support Package or advanced correction by implementation of correction instruction.
