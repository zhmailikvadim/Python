Symptom


--- Description ---

Follow the below KBA to configure Corba SSL, however, when trying to generate CA-Signed certificate request using GenPSE.exe command, nothing will be generated.

  • 3026364 - How to configure Corba SSL using a CA-signed certificate for SAP BI 4.2 SP05+ - SAP for Me

--- Steps to Reproduce ---

  1. Follow KBA 3026364 for the Corba SSL configuratoin in Windows Server
  2. Run the below command to generate the CA-Signed certificate request:
    GenPSE.exe gencsr csr.p10 server.key passphrase.txt ssl.cnf
  3. However, no files will be generated after running the command
