Symptom

  • Live Profile in Employee Profile Section is not visible for employees in Legacy People Profile, despite correct permissions being granted.
  • Some users can see the Section.
  • Employee Views Permission has been given for the Section.

Environment

SAP SuccessFactors HCM Suite

Reproducing the Issue

  1. Create a Section in Employee Profile as per KBA 2208066 - How to create a Custom MDF Portlet in the Employee/People Profile.
  2. Notice that, despite following the KBA, the Section is not visible to users.

Cause

This behavior is related to lack of permission for the object(s) inside the Live Profile in Employee Profile Section.

If users only have Employee Views permissions, without having object permissions as well, the section will not become visible to the users at all, as there is nothing to be shown inside it.

Resolution

Ensure that object permissions (Miscellaneous Permissions) are also granted to the respective users.
