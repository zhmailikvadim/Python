Symptom

  • User is not able to get the input control option to refresh the data
  • Normal users are unable to use input controls and refresh reports in SAP BusinessObjects BI 4.x 

Environment

  • SAP BusinessObjects Business Intelligence Platform (BI) 4.x/BI 2025
  • All supported OS

Reproducing the Issue

  1. Log in to BI Launchpad using a non‑administrator account.
  2. Open a Web Intelligence report that is built on a universe and try to refresh it.
  3. Attempt to refresh a report with input controls using a user.
  4. Observe that the input control option is not visible to the user.
  5. Repeat the same steps using an administrator account and confirm the input control is visible and functional. 

Cause

  • The user profile does not have the right for input control data for the web intelligence report.

Resolution

  1. Grant the required permissions to the affected users via the Central Management Console (CMC).
  2. Login to Central Management Console (CMC).
  3. Navigate to Applications → Web Intelligence.
  4. Navigate to User Security → Select users.
  5. Click on Assign Security → Advanced tab → Add/Remove Rights.
  6. Select Application rights → Web Intelligence → Specific Rights for Web Intelligence.
  7. Enable Right: Interactive: Reporting – Create and edit report filter.
