Symptom

  • No data is available in the MX_VAT_DCL for DIOT with 54 fields generated in Fiori.

Environment

  • SAP S/4HANA
  • DRC Mexico
  • Statutory Reporting

Cause

Missing configuration for Tax Grouping Versions.

Resolution

  1. Access the maintenance views V_T007K and V_T007L using transaction SM30.
  2. Specify "MX" as the country and set the Tax Grouping Versions as described in the documentation provided on the SAP Help Portal. 
  3. Add each Tax Code relevant to the business scenario. For additional guidance, refer to example at this other article from SAP Help Portal.
  4. Retest the report generation process to verify the solution.
