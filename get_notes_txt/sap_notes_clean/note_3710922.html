Symptom

Financial statement leaf item is not shown with G/L Account in the same field.

Environment

As of S/4HANA 2021

Reproducing the Issue

  1. Access the WDA app W0161 "Balance Sheet/Income Statement - Multidimensional".
  2. Attempt to display financial statement leaf item/GL account in the same field.
  3. Observe that the expected layout is not applied.

Resolution

  1. Open the Axis Properties dialog in the WDA app "Balance Sheet/Income Statement - Multidimensional".
  2. Select the option "Display Rows as Hierarchy".
  3. Set Option "Results Row" as Top.

Keywords

W0161, balance sheet, income statement, multidimensional, layout issue, reporting apps, financial statement leaf item, gl account, axis properties, display rows as hierarchy.
