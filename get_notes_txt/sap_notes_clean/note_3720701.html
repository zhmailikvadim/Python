Symptom

For already posted document transaction J1INPP does not work with FI document to update section code.

Environment

  • SAP ERP
  • SAP R/3
  • SAP ERP Central Component
  • SAP S/4HANA
  • SAP S/4HANA Finance
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • This is relevant for India Localisation or Country India Version (CIN) component.

Reproducing the Issue

  1. For a FI posted document, try to attempt to update the section code using transaction J1INPP.
  2. Observe the error message: "No Financial Document Found."

Cause

Transaction J1INPP could not be used to updated section code for already posted FI document.

Resolution

  1. Transaction J1INPP is designed to update section codes in documents posted by MM-related transactions, such as MIRO, and not for FI documents.
  2. To update the section code for an already posted FI document, a custom zprogram has to be written to update BSEG-SECCO filed.
