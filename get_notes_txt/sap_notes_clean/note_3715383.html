Symptom

During the customer-to-Business Partner conversion, a short dump ‘POSTING_ILLEGAL_STATEMENT’ occurs in the update task module /PACG/SAPLECM_UPDATE_CVICCP. The reason of the function module is remapping of FSM Customer Contact Person Identifiers.

Other Terms

FSM, E4C, Connector, Customer, Contact, Person

Reason and Prerequisites

This issue is caused by a database deadlock that triggers an explicit ROLLBACK WORK statement.

Solution

Please, implement this correction instruction and run report /PACG/ECM_PACG_NOTE_3715383 using t-code SE38 (or SA38).
