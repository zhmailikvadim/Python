Symptom

You run Price Simulation using a Display Profile and expect the Field Description to be read with the long label and displayed correctly in the download output. However, the system does not read the Field Description with the long label and therefore does not display it in the downloaded file.

This note corrects the behavior by ensuring that the Display Profile reads the Field Description with the long label and displays it properly in the download.

Other Terms

/IRM/GPSM, /IRM/LGPSMF0I, ITEM_ADDNL_DATA_FCAT_PREPARE

Reason and Prerequisites

 This is caused by program error

Solution

Please implement the correction instructions. These corrections are only valid if you have applied the following Support Pack(s):

Release V/4 – 2022 V2, 2023, 2023 V2, 2023 V3 and 2023V3 Support pack 1.

Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object
