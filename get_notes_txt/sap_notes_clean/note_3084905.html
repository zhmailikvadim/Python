Symptom

When printing a spool from SP01, the following error occurs.

  • Output cannot be redirected to this printer(see long text)
  • TD366

Reproducing the Issue

Reprint a spool request from SP01 via an output device defined with a different device type than the one that was used to create the spool.

Cause

The reason for this message is that you try to reprint a spool request via an output device which is defined with a different device type than the one in the output device used to originally create  the spool request. This would not create a good result with the print as the print control commands and format action commands embedded in the print data can be very different for different device types.

Resolution

If it is necessary to reprint a spool request from SP01, then this should be done via an output device which is defined with same device type that was used to originally create the spool.
