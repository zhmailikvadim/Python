Symptom

I need to change the Worker Pay Type for a Work Order that is currently in Created status.

Resolution

The supplier must first decline the Work Order. Once declined, the status changes to Declined, which makes the Work Order editable again. At that point, the Worker Pay Type can be updated by the buyer user and the Work Order can be resubmitted for supplier approval.
