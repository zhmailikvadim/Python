Symptom

  • How Super Admins enforce MFA
  • How to add MFA as mandatory to all or some S-users in your company
  • How to force MFA for individuals
 

Environment

SAP for Me > Users & Contact dashboard > User Management

Reproducing the Issue

As a Super Admin you want to enforce Multi‑Factor Authentication (MFA) for your S-user in your company.

Resolution

As of January 2026, a new feature is available that enables customers to enforce Multi‑Factor Authentication (MFA) across their company.
With this enhancement, Super Admins can centrally manage and strengthen account security by mandating MFA for users within their organization.

As a Super Admin, you have three options to enforce MFA, depending on your administrative needs.

Via User List Page

  1. The Customer Super Admin login to Sap for Me and opens the User Management Tool.
  2. Navigates to the User List page.
  3. Searches, filters, and selects one or multiple users.
  4. Enforces Multi‑Factor Authentication (MFA) for the selected users.

Via User Details Page

  1. The Customer Super Admin opens a specific user from the User List.
  2. On the User Details page, enforces MFA for that user.

Via Request User Page

  1. The Customer Super Admin login to Sap for Me and opens the User Management Tool.
  2. Click Request User button to create new S-user
  3. In the Request User page, the super user has the ability to enable/enfore MFA for a given user before the user has been created

Once a Super Admin enforces Multi‑Factor Authentication (MFA) for a S-user, the system automatically sends an email notification to the affected user.

The notification:

  • Informs the user that MFA has been enforced by their company administrator
  • Explains why MFA is required
  • Provides clear instructions on the steps the user must take to complete MFA setup

This communication ensures that users are aware of the change in advance and understand the required actions before their next login.

Email notification to user.

Provides notification on MFA enablement and instructions on what steps the user has to take to set it up on their end.

The user is forced to set up MFA method on their next login.
