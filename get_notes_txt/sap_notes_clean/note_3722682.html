Symptom

You are trying to enable legacy fields or newly added DB fields with domains containing conversion exits via SCFD_EUI as Custom Fields for usage in app "Custom Fields" and you encounter the following error messages regarding unallowed usage of conversion exits:

Conversion Exit ... specified on domain ... not allowed in S/4 HANA

Other Terms

unallowed Conversion Exit, Custom Field, SCFD_EUI, ABAP Managed Field

Reason and Prerequisites

Usage of Conversion Exits in S/4 HANA is restricted and only allowed for certain conversion exits as in the context of OData services they lead to various issues.

You can find a detailed explanation in blog Conversion Exits and the ABAP RESTful Application Programming Model (RAP)

Solution

Usage of unallowed conversion exits for Custom Fields or Data Source Extensions is not possible until Release S/4HANA 2025.
With this Release new annotation AbapCatalog.typeSpec.convExit: null for CDS extends gets introduced which ignore the usage of conversion exits in OData services.
