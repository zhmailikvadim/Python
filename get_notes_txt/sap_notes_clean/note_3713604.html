Symptom

You are using the Run Statutory Reports app to create the JPK_VAT report and receive the error message: "Query OUTPUT_TAX_ITEM_CLASS returns more than one record. Use query row key in data mapping."

Other Terms

Poland, SAF-T, Standard Audit File, PL, JPK_VAT, Run Compliance Reports, Run Statutory Report, document and reporting compliance features, SAP Document and Reporting Compliance, statutory reporting, statutory report, compliance report

Reason and Prerequisites

This SAP Note is relevant only for Poland.

Solution

Import the Support Package specified for your release or implement the attached correction instruction. Apply SAP Note 3713604 to your system. This note involves correcting the Query OUTPUT_TAX_ITEM_CLASS and integrating its mapping at the report definition level. Please follow the standard procedure for implementing the note, and make sure all components are carefully reviewed and activated.

As a general rule, SAP recommends that you implement the Support Package specified for your release. However, if you need to install a solution earlier, use the SAP Note Assistant to implement the correction instruction. You can find more information about the SAP Note Assistant in the SAP Support Portal, under SAP Note Assistant.
