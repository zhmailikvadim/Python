Symptom

You use Fiori App F0717A Manage Journal Entries - New Version to manage and analyze Journal Entries.

You try to display corresponding Intercompany Transactions.

You realize that Journal Entries are displayed in the Fiori App F0717 Manage Journal Entries - Old Version.

Other Terms

F0717A, fin.gl.journalentry.manage, FIN_JRNLENTRMNG, onNavigationTargetsObtained, F0717A, AccountingDocument-manage, AccountingDocument-manageV2

Reason and Prerequisites

Program error.

Solution

After implementation of the SAP Note please clean the caches by recalculating the index of the app (as described in the SAP Note 2319491) for project FIN_JRNLENTRMNG (tab 'Single SAPUI5 Repository Only'). Please make also sure to re-calculate the index for all dependencies. For SAP standard apps, dependencies such as central libraries should be listed under Fiori Apps Library > Implementation Information > Configuration > SAPUI5 Application in the Technical Name column.

Once the SAP Note is implemented system displays after clicking on the Intercompany Transaction number popup, where you can select, whether you want to display Intercompany Transactions in the Fiori App F0717A Manage Journal Entries - New Version (click the Intercompany Transaction number) or in the Fiori App F0717 Manage Journal Entries - Old Version (click the link with the same name).
