Symptom

Upon maintaining a Service Organization within an S/4HANA Service Opportunity, the Service Organization entry is not retained; it disappears after the opportunity is saved and subsequently reopened.

Other Terms

Opportunities, Service Organization, CRMS4S_OPPT_C_ORGMAN, CRMS4D_OPPT_H, CRMS4D_OPPT_I, Disappear.

Reason and Prerequisites

Program error.

Solution

  1. If you are running S/4HANA OP2025, implement the automatic attached correction instruction.
  2. Depending on your SAP_BASIS SP level, implement the automatic attached correction instruction or the manual steps as described in the following:
    • If your system is running SAP_BASIS 702 with Support Package SAPKB70218 or higher, implement the automatic attached correction instruction. For systems on lower Support Package levels, implement the attached manual steps.
    • If your system is running SAP_BASIS 731 with Support Package SAPKB73118 or higher, implement the automatic attached correction instruction. For systems on lower Support Package levels, implement the attached manual steps.
    • If your system is running SAP_BASIS 740 with Support Package SAPKB74015 or higher, implement the automatic attached correction instruction. For systems on lower Support Package levels, implement the attached manual steps.
    • If your system is running SAP_BASIS 750 with Support Package SAPK-75002INSAPBASIS  or higher, implement the automatic attached correction instruction. For systems on lower Support Package levels, implement the attached manual steps.
    • If your system is running SAP_BASIS 751, 752, 753, 754, 755, 756, 757, 758, 816 or 824 implement the automatic attached correction instruction.
