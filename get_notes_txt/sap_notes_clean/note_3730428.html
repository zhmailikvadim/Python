Symptom

In JIT customizing, specific action settings are required to support the intended processing flow. In the standard system, these entries may not be fully available or may require manual maintenance. As a result, customers may experience missing or incomplete action configurations, which can affect JIT‑related processing or lead to inconsistencies across systems during configuration transport.

Other Terms

NJIT, Next Generation Just-In-Time, Customizing, Action configuration

Reason and Prerequisites

Missing configuration entries/settings

Solution

Implement the BC Set delivered with this SAP Note to add and update the required action settings in JIT customizing.
The BC Set provides the necessary configuration entries so that customers can deploy the correct action setup consistently across systems without manual maintenance.

After activating the BC Set, the updated action settings shall be available for use in the relevant JIT processes.

Below are the updates available with this Note.

  • The action levels for the actions Confirm Packing for Materials (S1_CG_PACK) and Confirm Packing for Slots (S1_PG_PACK) will be updated to Component Group Level.
