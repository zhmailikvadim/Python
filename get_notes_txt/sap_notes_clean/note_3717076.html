Symptom

In SM21 the following error is displayed after the system has been updated to ST-PI 740 SP33
"Database error 257 at OPC"
"> sql syntax error: incorrect syntax near "@mv_instance": line"

Reproducing the Issue

  1. Upgrade the system to ST-PI SP33
  2. Many errors "Database error 257 at OPC" are displayed in transaction SM21.

Cause

Code error.

Resolution

  1. Implement the latest version of SAP note 3639977 - Collective corrections as of ST-PI 7.40 SP32 and ST-PI 7.58 SP00 for SAP Cloud ALM.
  2. Access the Cloud ALM setup transaction(/n/sdf/alm_setup).
  3. Click in 'Activate Use-Cases', deactivate the 'Health Monitoring' and confirm.
  4. Wait for 10 minute and reactivate it.

See Also

https://support.sap.com/en/alm/sap-cloud-alm/operations/expert-portal/setup-managed-services/setup-abap-onprem.html
