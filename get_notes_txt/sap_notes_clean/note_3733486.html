Symptom

In the Fiori application, filter validation should handle empty values appropriately based on the expected behavior.

However, when a filter value is set as empty, the validation fails and prevents further processing.
This note will fix the issue so that filter validation works correctly even when the value is empty.

Other Terms

Fiori filter, validation failure, empty value, filter issue, UI validation, Fiori error

Reason and Prerequisites

This is caused by program error

SAP Note#3733472 is available for ECC Release

Solution

Please implement the correction instructions attached. These corrections are only valid if you have applied the Support Packs.

Release V/4 - 2022 V2, 2023, 2023V2, 2023 V3 and 2023 V3 Support Pack 1

 Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object.
