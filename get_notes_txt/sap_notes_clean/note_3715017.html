Symptom

Error "Scope Item 78L must be active and Communication Arrangement configured." on submission process for Spanish legal reporting (SII) in edocument cockpit.

Environment

  • SAP S/4HANA Cloud Public Edition
  • SAP Document and Reporting Edition, Cloud Edition
 

Reproducing the Issue

  1. eDocument cockpit application.
  2. Select the source document.
  3. Submit the document, goes to Sent to Interface requested state.
  4. Error occurs : "Scope Item 78L must be active and Communication Arrangement configured."

Cause

Missing scope item 78L.

Resolution

  1. Ensure that scope item 78L is activated for Document and Reporting Compliance (DRC) cloud edition.
  2. Maintain the communication scenario SAP_COM_0854 as per Integrating All Other Processes and Countries/Regions with SAP S/4HANA Cloud Public Edition | SAP Help Portal. 



See Also

Refer to - Checking Requirements for Document and Reporting Compliance Features | SAP Help Portal.
