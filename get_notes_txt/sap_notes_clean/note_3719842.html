Symptom

Error /SCWM/ERPINTEGRATION884 is shown in COGI.

Enter the warehouse number.

Message No. /SCWM/ERPINTEGRATION884

Environment

  • SAP Extended Warehouse Management (SCM-EWM);
  • SAP Extended Warehouse Management Interfaces (SCM-EWM-IF);
  • SAP ERP Production Planning (PP)
  • SAP R/3
  • SAP R/3 Enterprise
  • SAP ERP Central Component
  • SAP ERP
  • SAP Enhancement package for SAP ERP
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA, on-premise

Cause

The error "/SCWM/ERPINTEGRATION884" occurs due to the warehouse number not being maintained in the system's configuration (LPK3) for the Destination Bin Assignment.

Resolution

Maintain warehouse number for the Destination Bin Assignment in LPK3.

Keywords

COGI, /SCWM/ERPINTEGRATION884, goods movement posting, SAP S/4HANA, warehouse number, LPK3, consumption posting error, synchronous goods movements
