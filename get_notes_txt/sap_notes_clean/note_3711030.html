Symptom

Run IC elimination task in consolidation monitor but cannot find any document posted.

Environment

S/4HANA Intercompany Matching and Reconciliation

Reproducing the Issue

  1. Open the consolidation monitor.
  2. Run IC elimination task 
  3. No posting in ACDOCU.
  4. ICMR Elimination Simulation tool has data in step 300

Cause

The matching method uses the data source only reads data from ACDOCU, for example CDS view ICA_CONSJOURNALENTRIES_3.

Resolution

For interunit elimination, use a data source that combines ACDOCA and ACDOCU, such as SC_JOURNAL_ENTRIES_01, to ensure data consistency and finer granularity.
