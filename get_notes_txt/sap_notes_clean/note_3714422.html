Symptom

When some manual steps of SAP Note are being followed, there is step to maintain the Data Type under structure, but it is grey and can not input value, for example in the Note 3233591, it is mentioned:

CONT_DESC->Choose Pre-defined type under column data Type -> Give value as 'STRING'. 

Environment

SAP ABAP systems

Cause

The Data Type should be maintained in build-in type.

Resolution

Choose the Built-In Type button after opening the structure, and then add value under Data Type.

See Also

3233591 - Note Analyzer: Improvements in the UI and the addition of a Content Field 

Keywords

Snote, data type, grey, edit, manual ,steps, pre-defined, activate
