Symptom

The error FAGL_SWITCH_OP093 (Account &&&&&&&&&&/&&&& is or was a material stock account; activation not possible) is displayed in transaction code FAGL_ACTIVATE_OP (report FAGL_SWITCH_TO_OPEN_ITEM) or in transaction code FINS_ACTIVATE_OIM (report FINS_SWITCH_TO_OPEN_ITEM).

Environment

  • SAP R/3
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA Finance
  • SAP S/4HANA

Reproducing the Issue

  1. Go to the transaction code FAGL_ACTIVATE_OP/FINS_ACTIVATE_OIM (report FAGL_SWITCH_TO_OPEN_ITEM/FINS_SWITCH_TO_OPEN_ITEM).
  2. Enter the G/L account and company code.
  3. Execute.

Cause

The error FAGL_SWITCH_OP093 is displayed because material postings exist in the G/L account. These postings can be found in the BSEG table by filtering with BSEG-KOART = 'M'.

Resolution

G/L accounts having material postings cannot be converted to open item managed accounts.
