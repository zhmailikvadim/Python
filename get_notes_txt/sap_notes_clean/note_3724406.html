Symptom

You use clinical documents to manage information about your patients. If you call the selection of text modules (F4) in a long text field at the end of a line that is longer than 255 characters, you notice that the system displays an error message:

Runtime error: GETWA_NOT_ASSIGNED_RANGE
ABAP Program: CL_ISHMED_LTE_EDITOR==========CP

Other Terms

PMD, Parameterized, NMED

Reason and Prerequisites

Program error

Solution

A runtime error no longer occurs in a long text field if you call the selection of text modules (F4) at the end of a line that is longer than 255 characters.

Import the specified Support Packages to solve the problem. You can implement the corrections in advance.
