Symptom

  • When viewing Prerequisites from the Item details page in user Library search, the sort order does not match the order which appears in the prerequisites tab of the item in learning administration.
  • Is there a way to change the sort order of the prerequisites?

Environment

SAP SuccessFactors Learning

Reproducing the Issue

  1. Add prerequisite items to an item in LMS admin.
  2. Check the item from the user side via library search.
  3. Observe that the items do not appear in the same order as configured in the admin.

Resolution

  • From the user side, when viewing the item details page, prerequisites will be ordered in the sequence they were added to the item, with the first items added appearing first and items added later appearing later in the list - It is not possible to change the order here.
  • In LMS admin, it is possible to choose the sort order by Title, ID, and Type - To do this, select the 'Sort' button in the top right of the prerequisites tab. Choose the sort order and select 'OK'
