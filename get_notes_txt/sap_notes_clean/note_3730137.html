Symptom

This SAP Note applies to the Fiori app Mass Change of Sales Scheduling Agreements (F5278) in the SAP S/4HANA OP2022 release.

When attempting to enter a value for a field to be changed, the error "enter a valid value" is raised, even though the entered value is valid.

Other Terms

MCC, Mass Change Cockpit, Mass Change of Sales Documents, Mass Change of Sales Scheduling Agreements, F5278, Input Validation, Shipping Point

Reason and Prerequisites

This is a program error.

Solution

Apply this SAP Note or upgrade to the listed Support Package.
