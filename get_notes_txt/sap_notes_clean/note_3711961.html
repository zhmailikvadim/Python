Symptom

  • The pencil icon is no longer appearing in custom MDF portlets in the Latest People Profile
  • Users without CORRECT permission cannot see the pencil icon in the custom MDF objects displaying in Latest People Profile

Environment

SAP SuccessFactors HCM

Cause

This is caused by a change in the RBP permissions for the Latest People Profile in which the pencil icon is strictly linked to the CORRECT permission for the custom object. In the legacy version, users with CORRECT and/or INSERT permission would see the pencil icon. 

Resolution

Our engineer team identified it as an issue and is working to revert to the legacy behavior in which either INSERT or CORRECT permissions would make the pencil icon available in the custom MDF portlet. 

The fix is planned for the next release on 1H26. 

Workaroud:

  • From the custom MDF portlet, click on the clock icon to access the history > go to Insert New Record.
