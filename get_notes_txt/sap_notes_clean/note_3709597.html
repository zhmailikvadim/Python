Symptom

  • A SAP standard technical table is growing on a HANA Database, and requires cleanup and housekeeping to reduce the table size in memory and on disk.
  • Upon checking for a solution in KBA 2388483 - How-To: Data Management for Technical Tables , there is no entry or section present for the particular table of interest.

Environment

SAP HANA Platform Edition 2.0

Resolution

1) Go to one of the Transaction Codes DBACOCKPIT-DB02-ST04 and lookup the table name within the "Tables/Views" section of the "Diagnostics" folder

2) Under the "SAP ABAP Dictionary Information" section you will see the application owner under the "Application Component" field.





3) Contact the SAP support component listed there, to request guidance on table cleanup, since SAP standard table housekeeping is always performed via the official tools and reports on application layer and not native SQL.
