Symptom

You use Transaction Manager of Treasury and Risk Management. You manage an ABS/MBS security position (product category 042, Bonds with installment repayment). You create a new redemption schedule after an existing redemption schedule has been fixed. An error message RDPT215 about “Repayments cannot be made before &1” may be issued when you save this new redemption schedule.

Other Terms

CL_RDPT_SCHEDULE, CHECKREDEMPTIONS, TRANCHE, FLG_RETROSPC_ADJ, RDPT 215

Reason and Prerequisites

Program Error

Solution

Preliminary correction via Correction Instruction or implementation of the corresponding Support Package.
