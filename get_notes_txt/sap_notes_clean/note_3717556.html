Symptom

  • The extracted SAP Web IDE Personal Edition installation package does not contain the expected executable files, such as orion.exe and eclipsec.exe.
  • Users are unable to locate the required files in the eclipse folder after extracting the ZIP file using Windows Explorer.

Environment

SAP Web IDE platform

Reproducing the Issue

  1. Download the SAP Web IDE Personal Edition installation package provided by Support.
  2. Extract the ZIP file using Windows Explorer.
  3. Navigate to the eclipse folder within the extracted directory.
  4. Observe that the orion.exe and eclipsec.exe files are missing.

Cause

The issue occurs due to compatibility problems between the ZIP file and the Windows Explorer extraction utility, which fails to extract certain files from the package.

Resolution

  1. Download the SAP Web IDE Personal Edition installation package from the provided link.
  2. Use an alternative extraction utility, such as 7-Zip or WinRAR, to extract the contents of the ZIP file.
  3. After extraction, navigate to the eclipse folder within the extracted directory.
  4. Verify that the orion.exe and eclipsec.exe files are present.
  5. Follow the installation and setup instructions provided in the SAP Web IDE Personal Edition documentation:

a. Navigate to `C:\SAPWebIDE` and open the eclipse folder.

b. Double-click the orion.exe file to start the Application Server.

c. Access the SAP Web IDE Personal Edition via the URL: http://localhost:8080/index.html.

d. Create a new account and log in using the defined credentials.
