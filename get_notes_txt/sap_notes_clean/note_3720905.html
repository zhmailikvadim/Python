Symptom

The implementation of this SAP Note is required for the consistency of identification documents processing after implementation of SAP Note 3720958.

Other Terms

IT0290, 0290, T7RU0290CODE, DCCAT

Reason and Prerequisites

  • Some objects that required for future localization development must be shipped in advance
  • You have implemented SAP Note 3720958

Solution

As a general rule, SAP recommends that you implement the Support Package specified for your release. However, if you need to install a solution earlier, use the SAP Note Assistant to implement automatic correction instructions. You can find more information about the SAP Note Assistant on the SAP Support Portal, under SAP Note Assistant.
