Symptom

When provisioning users to SAP Build Work zone, standard edition, the following error occurred:

  • Caused by: Connector cannot process more entities due to irreparable error
    Caused by: All target systems are marked as invalid due to the following reasons: [Could not retrieve OAuth token Caused by: Unexpected character ('<' (code 60)): expected a valid value (JSON String, Number, Array, Object or token 'null', 'true' or 'false')
    at [Source: REDACTED (`StreamReadFeature.INCLUDE_SOURCE_IN_LOCATION` disabled); line: 1, column: 1]]

Environment

  • SAP Cloud Identity Service
  • SAP Build Work zone, standard edition

Cause

There are mandatory properties in the IPS target system for SAP Build Work zone standard edition being configured incorrectly.

Resolution

Refer to the documentation for SAP Build Work Zone, standard edition to verify all the mandatory properties are configured correctly.

Especially the OAuth2TokenServiceURL should follows the pattern: <uaa.url>/oauth/token.  For example: https://ips-cflp-woaealle.authentication.sap.hana.ondemand.com/oauth/token.
