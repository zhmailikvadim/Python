Symptom

The new supplier view does not display supplier contacts in the dashboard card on the UI.

Environment

  • UI/Dashboard
  • NGSM, new supplier profile

Reproducing the Issue

  1. Access the realm of SAP Ariba Supplier Lifecycle and Performance.
  2. Navigate to the supplier click on the supplier to view new supplier profile.
  3. Observe that supplier contacts, certifications, performance or other data are missing on the card.

Cause

Configuration for displaying primary contact field was not enabled under SM Administration > Configurable Fields > Supplier Data API.

Resolution

  1. Navigate to SM Administration > Configurable Fields.
  2. Select Supplier and Supplier Data API.
  3. Enable the configuration for the desired field.
  4. Verify if the data are displayed correctly.
  5. For certificates still not loading, further investigation is required.

See Also

Ensure the correct URL is used in SM Administration > BTP Application
