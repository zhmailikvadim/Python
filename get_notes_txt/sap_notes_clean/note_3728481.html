Symptom

  • An error occurs during the “Retrigger” action for the “Position Follow-Up Processes” alert type.
  • The error message is: "An error occurred during the “Retrigger” action for the “Position Follow-Up Processes” alert type. Select the alert and choose the “Retry” action."
  • Retriggering failed domain events does not clear the issue.
  • Event Type: sap.sf.foundation.alerts.actionExecution.created.v1

Environment

SAP SuccessFactors Employee Central

Reproducing the Issue

  1. Navigate to Admin Alerts.
  2. Open Failed Events for Position Follow-Up Processes.
  3. Observe multiple failed events.
  4. Attempt to rerun the failed events; the issue persists.

Cause

Resolution

The solution of this issue is currently under investigation by engineering team. Please follow this KBA for updates.
