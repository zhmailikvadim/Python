Symptom

You are working with BAPI function modules for payment details to process Current Account Style Instrument deals. BAPI terminates with error message FTR0 898 "No BAPI support available for transaction ".

Other Terms

BAPI_FTR_PAYDET_CHANGE
BAPI_FTR_PAYDET_CREATE
BAPI_FTR_PAYDET_DELETE
BAPI_FTR_PAYDET_GETLIST

Reason and Prerequisites

Programming error.

Solution

Please implement the SAP Note or the Support Package.
