Symptom

The status of a send request in fiori app 'Monitor Email Transmission' is still on 'W' for 'waiting', while the status of the same send request has already been changed in transaction SOST.

Other Terms

SBCS_MAIL_OUT, SMTP, SCOT

Reason and Prerequisites

The Fiori app 'Monitor Email Transmission' and the monitoring transaction SBCS_MAIL_OUT both use a different set of database tables where the email metadata is stored, compared to the transaction SOST or the obsolete Fiori app 'Display Email Transmission'. Therefore, it was possible for the database tables to become out of sync in certain error situations.

Solution

Implement the correction using the Note Assistant or import the corresponding support package.
