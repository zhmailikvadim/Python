Symptom

Performance period start date and end date are not copied from the reference Purchase Order when creating a Service Entry Sheet using the application "Manage Service Entry Sheets - Lean Service".

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Open the application "Manage Service Entry Sheets - Lean Service".
  2. Create a service entry sheet using a reference purchase order.
  3. Observe that the performance period start date and end date are not copied from the reference purchase order to the service entry sheet.

Cause

System Standard Design.

Resolution

  1. The described behavior is standard in SAP S/4HANA Cloud Public Edition.
  2. The Performance Period in the Service Entry Sheet (SES) is not automatically copied from the referenced Purchase Order (PO).
  3. The Performance Period must be entered manually during SES creation.
  4. The system ensures that the entered Performance Period remains within the Start Date and End Date maintained in the PO.

  • KBA 3655589 - Performance period restriction to PO-defined range.
