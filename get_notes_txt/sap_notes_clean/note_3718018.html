Symptom

You review or maintain contact types for suppliers or customers. You notice a missing description for contact type R1.

 

Other Terms

V_TPFK, TPFK, TPFKT VTEXT, R1

Reason and Prerequisites

Incomplete delivery of customizing settings causes this symptom.

You find the prerequisites in the relevant correction instructions.

Solution

After implementing this SAP Note, the system displays the correct description "Compliance Contact" for contact type R1 on the user interface.

Alternatively, import the corresponding Support Package for the component extension for SAP Environment, Health, and Safety Management or the corresponding Feature Pack Stack for SAP S/4HANA.
