Symptom

Popup with message "License expiration date: XX.XX.XXXX(in X days)" appears when login even if the new license has been installed.

Environment

SAP NetWeaver

SAP NetWeaver Application Server for SAP S/4HANA

ABAP PLATFORM - Application Server ABAP

Cause

License buffer is inconsistent.

Resolution

Reset License Key buffer via function module SLIC_RESET_BUFFER_ENTRY in transaction SE37.

The Product ID should appear in transaction SLICENSE.  Check the table for "Digitally Signed Licenses" and see the "SW Product" column. The table in transaction SLICENSE shows for all application servers consistently the license and expiration date (generally appearing as 31.12.9999) in the "Current Settings" section.  

Keywords

license, license expire, new license installed, slicense, logon
