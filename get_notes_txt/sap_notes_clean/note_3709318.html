Symptom

Error message: "Posting failed. Condition type is incorrect for tax reclaim code XX." displayed while posting expense report in the Concur Posting Documents Monitor app.

Environment

S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Access the Concur Posting Documents Monitor app.
  2. Attempt to post an expense report.
  3. Observe the error message: "Posting failed. Condition type is incorrect for tax reclaim code XX."

Cause

The tax rate is not maintained in the tax configuration, which renders the condition type invalid in the SAP system.

Resolution

  1. Review the tax authority configuration in the Concur system.
  2. Adjust the configuration to ensure consistency between the tax settings in Concur and SAP systems.
  3. Maintain the tax rate for the corresponding line in the SAP system.

Keywords

posting failed, condition type incorrect, tax reclaim code, concur, s/4hana cloud public edition, tax rate missing, expense report error, sap system configuration, tax authority adjustment.
