Symptom

User try to process a PMR document item, and receive the following error message:

"Warehouse process type &1 not allowed for document category PWR.

Message No. /SCWM/DELIVERY108."

Environment

  • SAP Extended Warehouse Management (SCM-EWM);
  • SAP Extended Warehouse Management Production Material Request (SCM-EWM-PMR);
  • SAP S/4HANA.

Reproducing the Issue

  1. Go to /SCWM/PMR;
  2. Enter the document number;
  3. Try to process the item;
  4. Error is raised.

Cause

Program error resulting from the implementation of SAP Note 3418999 - No WPT mapped from PSASTAGE to SAPEXT_P, which caused incorrect WPT determination in the PMR item.

Resolution

Implement SAP Note 3549169 - Initial WPT on PMR item determined faulty after implemention of note 3418999.

See Also

Keywords
