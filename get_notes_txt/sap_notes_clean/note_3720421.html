Symptom

  • In the Transport->Monitor page, an export job was in Executing status for 8 hours and eventually changed to Failed status.
  • Multiple failed jobs are listed in the Monitor page, each with an execution time of 8 hours.
  • The reason for the issue is unclear.
  • Users are unable to cancel the export job when it is stuck in Executing status.

Environment

  • SAP Analytics Cloud (SAC) 2026.2.6
  • Content Network

Reproducing the Issue

  1. Log in to SAP Analytics Cloud using the relevant access credentials.
  2. Navigate to Transport->Monitor.
  3. Locate the export job with the specified Job ID .
  4. Observe that the job remains in Executing status for 8 hours and eventually changes to Failed status.

Cause

The issue is currently under investigation by the development.

Resolution

The solution of this issue is currently under investigation. Please follow this KBA for updates.
