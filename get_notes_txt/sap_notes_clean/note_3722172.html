Symptom

Payment request is set to Scheduled status instead of transitioning to Sending status.

Reproducing the Issue

  1. Create an invoice in the Ariba.
  2. Reconcile the invoice.
  3. Observe that the payment request transitions to Scheduled status instead of Sending status.
  4. Verify that the invoice reconciliation document is not exported to the external system (CIG > SAP ECC).

Cause

Parameter Application.Invoicing.SendIRtoAN is enabled. Per documentation "If Application.Invoicing.SendIRtoAN is disabled, all invoice reconciliation documents are exported to an external ERP in OK2Pay file format using the SAP Ariba integration toolkit.". So if the parameter ie enabled, invoice reconciliation documents are not exported to an external ERP in OK2Pay file format.

Resolution

Disable parameter Application.Invoicing.SendIRtoAN.
