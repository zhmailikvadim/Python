Symptom

  • Universe design Tool hangs when performing the integrity check for Universe
  • In the logs, each SQL Statement checking take almost 2 minutes.

Environment

  • SAP BusinessObjects Business Intelligence platform 4.3 SP2
  • Universe Design Tool (UDT)
  • Oracle 19c 64 bit software driver.

Reproducing the Issue

  1. Upgrade the Oracle database to Oracle Exa 19C
  2. Open the UDT.
  3. Import the universe.
  4. Check Integrity for all.
    Note that the integrity checking hangs, it cannot complete.

Cause

  • Go to Oracle Server monitor, found the SQL history, note that each SQL send from UDT took 2 minutes.
  • Oracle server was upgraded, the connectInit for the old Oracle server is not working for new Oracle server.

Resolution

  • Remove the ConnectionInit from UDT connection define area.

Keywords

universe,slow, UNV, check, integrity.
