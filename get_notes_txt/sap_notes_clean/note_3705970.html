Symptom

This SAP Note contains information about planning the installation, upgrades and support packages of the ABAP add-on ST-PI 758

Solution

 

  1. General information
  • See the general information from SAP Note 70228.
  • In general it is not possible to uninstall ABAP add-ons. Uninstallable Add-ons are listed centrally in SAP Note 2011192.
    Please check chapter Uninstallation for this [Add-on] and find further instructions there. Further uninstallable Add-ons are listed centrally in SAP Note 2011192.
  • Upgrades with ABAP add-ons
    • In systems in which an ABAP add-on is installed, you can upgrade only to SAP releases that are supported for this add-on.
    • Note that there is a delay between the delivery of the SAP standard releases and the release of the corresponding add-on releases.
    • If an add-on upgrade is connected with a change of the SAP release, it is integrated into the SAP upgrade (repository or system switch). To carry out this upgrade, you require an additional add-on exchange package in addition to the SAP standard upgrade packages. If the SAP standard upgrade is carried out without this additional exchange package, the add-on will no longer work after the upgrade. The entire SAP system is inconsistent.
      For more information about this problem, see SAP Note 33040.
    • If S/4 conversion is planned on a system with add-ons, then the add-on upgrade is integrated in the conversion process. To carry out this conversion, you require the target release add-on relevant packages in addition to the SAP standard upgrade packages.If you are scheduling an upgrade or the import of an enhancement package, note that you should not import the latest Support Package Stack in the source release directly before the upgrade to ensure that the required, equivalent Support Package is already available in the target release.
      For more information about this, see SAP Note 832594 and the following information on SAP Support Portal.
      http://support.sap.com/sp-stacks -> SP Stack Information -> SP Stack Strategy

  1. 2. ST-PI 758

    1. Installation
      Name of the Installation Package (AOI):
      SAPK-758AGINSTPI

      ST‑PI 758 – Supported Product Versions

      The ST‑PI 758 component is supported starting from the following product versions:

      • SAP S/4HANA 2023 FPS 00
      • SAP BW/4HANA 2023 FPS 00
      • SAP S/4HANA Foundation 2023 FPS 00

      Please select ST‑PI 758 as the target release during the planning phase in the Maintenance Planner.
