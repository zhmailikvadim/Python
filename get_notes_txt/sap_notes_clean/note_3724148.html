Symptom

The Odata datastore fails to be saved with error ” The Data Services Request could not be understood due to malformed syntax”

Other Terms

CI-DS, HCI, Odata, connection, URL, /IWFND/ERROR_LOG, SAP Gateway, error, log

Reason and Prerequisites

Environment

  •  SAP Cloud Integration for Data Services 1.0.11.xxxx

Cause

 The Odata Datastore's URL format "https://xxxxx:<port>/sap/opu/odata/sap/xxxxx/xxx?$format=json" is incorrect

Solution

Change the URL to correct format like: "https://xxxxx:<port>/sap/opu/odata/sap/xxxxx" and the issue will be resolved.
