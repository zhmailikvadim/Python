Symptom

When trying to download the document report in Run Statutory Reports APP, system pop up the information "XXX Cannot be downloaded". The document is generated successfully in the document format "Table View".

(XXX represent for the document name)

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Open "Run Statutory Reports" application.
  2. Select the applicable reporting task.
  3. Select the report generation activity.
  4. Click on "New Run" to generate the report.
  5. File generated is of the format "Table View".
  6. Message "cannot be downloaded" is observed while trying to download the report.

Cause

Currently, the document format is "Table View" which means no file created or archived in the content, and the document cannot be downloaded with "Table View" foramt.

Resolution

If desire to download the content, kindly refer to the following steps:

  1. Select the document name.
  2. Click on Analyze Data button.
  3. Click on the button "Export" to save the table data in ".xlsx" format.
  4. Then the content should be downloaded to your device.
