Symptom

In Slovenian KPR/KIR report the generated file header displays values in tags <TUJEC1> and <TUJEC2>.

Environment

  • SAP S/4HANA
  • SAP ERP

Resolution

If plants abroad is activated and VAT ID is provided in T001N-VMZL1, then <TUJEC1> will display the foreign payer country code and <TUJEC2> will display the foreign payer VAT registration number.

In case <TUJEC> tags are not needed, remove the VAT ID from it T001N-VMZL1.
