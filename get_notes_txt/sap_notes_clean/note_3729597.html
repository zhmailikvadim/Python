Symptom

A request is made to delete specific files and/or folders from the FTP

Environment

  • SAP Sales Performance Management
  • Agent Performance Management 

Resolution

  1. Up until now, SAP Support handled this request alongside the SAP Operations Team
  2. Following the FTP Agent Performance Management migration set to occur on April 4, 2026, for Non-Production, and June 6, 2026, for Production, the customer will need to themselves administer and remove any files/folders they no longer want/need, since SAP Support does not have access to the FTP.
