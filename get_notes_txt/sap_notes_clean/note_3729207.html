Symptom

When  display XML file of JP eDocument in EDOC_COCKPIT, following error happens:

  • No name exists for Customer &1(EDOCUMENT_EU 010)

Environment

  • SAP ECC
  • SAP S/4HANA

Reproducing the Issue

  1. Run EDOC_COCKPIT
  2. Choose JP eDocument
  3. Display XML file

Cause

"Name" not maintained for related customer.

Resolution

In ECC, please maintain "Name" for customer in T-code XD02.

In S/4HANA, please maintain "Name" for customer in T-code BP.
