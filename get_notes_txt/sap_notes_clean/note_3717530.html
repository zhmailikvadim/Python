Symptom

The Profile Photo, About Me, and My Name sections are not displayed on the Complete Profile (Start Completing Your Profile) page.

Environment

SAP SuccessFactors Onboarding

Reproducing the Issue

  1. Log in as the new hire.
  2. Open the To-Do panel > Start Completing Your Profile.
  3. Issue: The Profile Photo, About Me, and My Name sections are not displayed.

Cause

Missing settings.

Resolution

Please ensure that the following settings are enabled in:

  • Configure People Profile > General Settings:
    • Allow employees to edit background image
    • Allow employees to upload an audio file for name pronunciation
    • Allow employees to upload an About Me video
    • Allow employees to enter introductory text
  • Configure Latest People Profile > Configurations > General Settings:
    • Allow employees to add name pronunciation audio
    • Allow employees to add "About Me" videos
    • Enable self-description
