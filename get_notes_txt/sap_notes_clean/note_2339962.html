Symptom

During an automatic expiry, the system should book forward up to the end date of the policy component + one day. However, the system creates period values only up to the end date of the policy component.

Other Terms

- Movement processing

- Update

Reason and Prerequisites

Program error

Solution

History:

Version 1 from 7/7/2016: New　

Implement the source code corrections.
Support Package SAPK-70013INMSG corrects the problem.　

Solution:

During an automatic expiry, the system books forward up to the end of the policy component + one day.

Technical effects:

See the "Solution" section.

Manual changes:

You do not have to perform any manual tasks.
