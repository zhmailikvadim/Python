Symptom

In the process "Sending of Complaint Notification" (DE_C_CP_COMPL_NOTE_SEND), the system doesn't include free text segments FTX+Z14 (Included Budget Bill) and FTX+Z16 (Quotation Item Number) in complaint message 33003 for specific complaint reasons.
 
This issue is relevant for the market role: SUP.

Other Terms

REMADV, FTX+Z14, FTX+Z16, complaint reasons, E_0406-A74, E_0406-A75, E_0406-A76, E_0406-AC5, E_0210-A21, E_0264-A21, E_0515-A21, E_0519-A21, E_0266-A21, E_0517-A21

Reason and Prerequisites

The error occurred because the indicators for free text types weren't set for the relevant complaint reason entries in view /US4G/V_COMPL_R. Without these indicators, the system was unable to include the required free text segments in the complaint message for related DTD checks.

Solution

For this SAP Note, you need to manually activate the indicators for the complaint reasons in view /US4G/V_COMPL_R. To access this view, go to Customizing for SAP Utilities → Intercompany Data Exchange → Utilities Solutions for Germany → Billing Enhancements → Bill and Payment Advice Note Processing → Define Complaint Reasons.
 
Set the following indicators:

  • Select the "Indicator to Free Text Type IncludedBudgetBill(Z14)" checkbox for complaint reasons E_0406-A74(MissingPrepaidInvoiceReferenceSumGUE), E_0406-A75(PrepaidInvoiceAlreadyClearSumGUE), E_0406-A76(PrepaidInvoiceNotConfirmSumGUE), and E_0406-AC5(MissingPrepaidInvoiceSumGUE).
  • Select the "Indicator to Free Text Type QuotationItemNo(Z16)" checkbox for complaint reasons E_0210-A21(MissingArticleIDSumMOSB), E_0264-A21(MissingArticleIDSumESABE), E_0515-A21(MissingArticleIDSumCFBDE), E_0519-A21(MissingArticleIDSumCFBSE), E_0266-A21(MissingArticleIDSumESABEA), and E_0517-A21(MissingArticleIDSumCFBDEA).
    You can verify the fix by checking that the system includes free text segments FTX+Z14 and FTX+Z16 in complaint message 33003 for the relevant DTD checks.
