Symptom

  • When implementing ONB 2.0 without using I-9 or E-Verify, the system does not provide a completion message to new hires after they finish all tasks.
  • The ONB panel does not close or redirect to the home page upon task completion.
  • No email is sent to new hires confirming the completion of the process.
  • The issue occurs consistently when new hires complete their tasks in ONB 2.0.
  • No error messages are displayed.

Environment

 SAP SuccessFactors Onboarding

Reproducing the Issue

  1. New hire logs into Onboarding 2.0.
  2. New hire completes data collection and compliance tasks.
  3. Observe that no completion message is generated, the panel remains open and no redirect occurs, and no completion email is sent.

Cause

The current product behavior does not provide an overall completion confirmation (message, panel close/redirect, or email) when I-9 and E-Verify are not used in Onboarding 2.0.

Resolution

  1. Acknowledge that this is the current system behavior; there is no configuration or built-in option to send a completion message, close the panel, or redirect when I-9/E-Verify are not used.
  2. If this behavior does not meet business needs, submit an enhancement request to have completion messaging/redirect added for non–I-9/E-Verify scenarios.
  3. Monitor product release notes and this KBA for any future updates that introduce this functionality.
