Symptom

Dump OBJECTS_OBJREF_NOT_ASSIGNED happened when executing application transaction for special company codes. 

Category               ABAP programming error
Runtime Errors         OBJECTS_OBJREF_NOT_ASSIGNED
Except.                CX_SY_REF_IS_INITIAL
ABAP: Program          CL_SALV_GUI_GRID_V_TOOLBAR_IDACP
Application Component  BC-SRV-ALV

Environment

SAP S/4HANA 2023

SAP_BASIS 758 SP01 

Reproducing the Issue

  1. Navigate to SAP FAGLGVTR screen.
  2. Perform the annual balance carry forward operation.
  3. Observe the runtime error "OBJECTS_OBJREF_NOT_ASSIGNED" occurring during the process.

Cause

Resolution

Reinstall SAPGUI and upgrade to the latest version. 

Keywords

OBJECTS_OBJREF_NOT_ASSIGNED, dump, company code, ALV, CL_SALV_GUI_GRID_V_TOOLBAR_IDACP
