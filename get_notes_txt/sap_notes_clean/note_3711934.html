Symptom

  • The displayed value is incorrect, or the field is missing entirely from the payslip.
  • The same issue does not happen with the standard delivered payslip.

Environment

  • Human Capital Management (HCM)
  • SAP R/3
  • SAP ERP Core Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA

Reproducing the Issue

In HRFORMS, either the field is entirely missing or the payslip form displays unexpected results.

Cause

The %GLOBAL_INIT in the relevant include, such as /1BCDWB/SAPLSM0000, either populates the field incorrectly or does not print it at all, based on the customer’s ABAP code.

Resolution

To resolve the issue, please check, debug, and adjust the custom ABAP code as necessary to ensure the correct functionality.

See Also

 It is recommended to utilize the ANST transaction and source code functionality to identify where in the ABAP code a field is calculated.
