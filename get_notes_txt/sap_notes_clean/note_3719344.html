Symptom

In the application, the selection screen contains two input filters: Service Document and Service Document Item.

In the previous system behavior, when users entered values in these fields, the value help returned all corresponding service documents and document items without considering the root service documents ( Control Flag = H ).

As a result, documents with different Control Flag values were displayed together, which did not align with the intended business logic

Other Terms

Service Settlement Hierarchy, Service Document, Service Document Item

Reason and Prerequisites

Enhanced filter implementation.
