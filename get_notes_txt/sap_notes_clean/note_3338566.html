Symptom

In the Latest Version of Goal Management, there are goal cards showing in the landing page when viewing the goal plan.

Its functionally has been modified to provide better experience to users as it displays the goals in Tiles/Card format in the Goal Plan with limited space/area to show few fields.

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

Product version: SAP SuccessFactors HXM Suite Core 2305
Sold product: Successfactors Goal Management Module

Resolution

These card fields are dependent on the goal plan configuration. They cannot be hidden from the card only. If the field (except for name and goal type) is hidden in the goal template, it will be hidden in the card as well.

The goal fields attributes viewdefault & showlabel are responsible to display the fields in the Goal plan by default.

Following the configuration in the goal plan template xml, all the other fields can be hidden except the goal name and the Personal/Team goal fields.

Only the following fields will show if configured:

  1. Name
  2. Personal/Team goal type indicator
  3. Due Date, if overdue, it will show the number of days it is overdue
  4. %Complete
  5. Status
  6. Weight

Note: When a user adds a new goal, they are allowed to add both personal goal and team goal, then the field will appear automatically to differentiate the type of these two goals. 

All other fields such as metric, description, start date, and other custom fields are currently not supported in the goal card.

An enhancement request for this feature has already been submitted. If you believe it would be beneficial, please consider casting your vote to support it. You can refer: Goal card fields - add Weighted Success Rating 

Goal Weight Display 

Goal weight is displayed in both the goal plan header and on goal cards, in the latest Goal Management.

Users should be granted the read permission for the weight field in Goal XML configurations.
