Symptom

You run flexible upload. Some of the uploaded values have multiple spaces. The system raises error message UCD1 022 (&1 &2 is not defined) which complains about non-existing value. However, this message is not justified, as the uploaded value does exist.

Other Terms

UCD1(022)

Reason and Prerequisites

The issue is caused by program error introduced in BCS/4HANA release.

Solution

Implement the attached correction instruction or import the specified Support Package.
