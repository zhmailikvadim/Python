Symptom

The system is not creating the PM inspection lot of origin 14 upon order release automatically, nor manually from menu in transaction IW32.

The configuration is done properly, also as per KBA

2627091 - Warning 'No insp. lot was generated Release order anyway?' within IW32.

Still the lot is not created.

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

  • SAP ERP Central Component
  • SAP ERP
  • SAP Enhancement package for SAP ERP
  • SAP S/4HANA, on-premise
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Call transaction IW32 for affected order.
  2. Press ENTER.
  3. Goto menu 'Order > Functions > Inspection Lot > Create inspection lot'.
  4. SAVE.
    (Order saved with number & - Message no. IW080)
  5. Inspection lot is not created, the order has status:
    ILNC - I0182 - Inspection lot not created
    instead of
    ILAS - I0281 - Inspection lot assigned

Cause

Missing customizing.

Resolution

Navigate to following customizing path:

IMG(transaction QCC0 or SPRO)
>Quality Management
>>Quality Inspection
>>>Inspection Lot Creation
>>>>Define Default Values for Inspection Type

Ensure that an entry for inspection type 14 exists (or the custom inspection type used for origin 14).
