Symptom

This SAP Note is relevant for you only if you use SAP EEG Billing for German Electricity Feed.

It should be possible to assign an existing business partner to a generating unit as part of the new installation process, even though the generating unit cannot yet have a move-in for an installation.

This is to support the early process of the grid operator check, otherwise there is a risk of executing the grid operator check too late.

In addition, business partner data in the GU-LIST should not be taken into account if there is no active contract in the update period.

Other Terms

ENBIL, business partner

Reason and Prerequisites

ENBIL 606 SP23 or lower

Solution

Implement this SAP Note or import ENBIL 606 Support Package 24.
