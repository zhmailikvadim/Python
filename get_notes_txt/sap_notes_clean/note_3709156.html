Symptom

  • Errors occurs during activation of BI Content "/ERP/SFIN_PLANNING".
  • Error message: "System error in program CL_FCOM_IP_IOBJ_ACCESS-CONSTRUCTOR and form DataSource 0STKEYFIG_TEXT, Error Message No. BRAIN299"
  • Error message: "Exception 'System error in program CL_FCOM_IP_IOBJ_ACCESS-CON' in 'SAPLRRMS,LRRMSU13,91' when passing type 'IOBJ', Error Message No. RSO323".

Environment

  • Controlling (CO)
  • SAP S/4HANA
  • SAP S/4HANA Finance
  • SAP S/4HANA Cloud Private Edition
  • SAP Fiori for SAP S/4HANA

Reproducing the Issue

  1. Attempt to activate BI Content "/ERP/SFIN_PLANNING".
  2. Encounter error messages during the activation process.

Cause

Resolution

  1. To set up SAP BPC optimized for S/4 HANA, follow the instructions provided in SAP Note 1972819.  
  2. Ensure that the business function OPS_PS_CI_1 (Step 2) is activated.  
  3. After activation, proceed with the subsequent steps outlined in the SAP Note 1972819. 

See Also

SAP Note 1972819 - Setup SAP BPC optimized for S/4 HANA Finance and Embedded BW Reporting
