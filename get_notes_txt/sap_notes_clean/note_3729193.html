Symptom

User wants to use Fiori app F4557 for handling units in onPrem or private cloud;

Environment

  • Sales And Distribution (SD)
  • Logistics Execution (LE)
  • SAP S/4HANA
  • SAP S/4HANA Cloud 
  • SAP ERP Central Component
  • SAP ERP
  • SAP Enhancement package for SAP ERP
  • SAP Enhancement package for SAP ERP, version for SAP HANA

Resolution

We are not having any dedicated Fiori application such as F4557 with a function set like HUMO in onPrem or private cloud. For S/4 onPrem and private cloud releases you might want to refer to the HUMO WebGUI application as of the business application catalogue. For further questions regarding configuration and use of HUMO with WebGUI, please open case under component BC-FES-WGU or BC-FES-ITS.
