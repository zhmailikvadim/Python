Symptom

Incorrect screen behavior can be observed in the WDA tile Display measuring point.

  1. Unable to see General Data.
  2. Failed to load Display Measuring Point even the existing point can be checked in SAP GUI.
  3. Several blocks are not displayed.

Environment

  • SAP S/4HANA 2023, on-premise 
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Open WDA app 'Display measuring point'.
  2. Enter existing point and hit enter.
  3. Unable to see several part of the application.

Cause

Incorrect index was used to hide UIBBs on support package S4 2023 support package SP00 to SP01.

Resolution

For S4 2023 support package SP00 and SP01 please implement note 3471425 - EAMS: Missing UIBBs in Display Measuring Point application.

See Also

3471425 - EAMS: Missing UIBBs in Display Measuring Point application

Keywords
