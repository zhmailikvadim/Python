Symptom

You want to know what's new in SP Stack Kernel 722 PL 400.

Other Terms

SP Stack Kernel, Hotfix, 722, 722_EXT, PL400, PL 400, PL#400, patch level 400, Release Notes

Reason and Prerequisites

SAP recommends to use the latest SP Stack Kernel 722 that is available in the download area of the SAP Support portal.

All information in this SAP Note also applies to the respective 722 EXT Kernel patch level.

This SAP Note contains the list of new features/enhancements that were included since the release of Kernel 722.  Please see SAP Note 2133909 for the complete list of new features in the SAP Kernel 722.

Below you will also find the list of known issues with this kernel patch level (if any). This list is being continuously updated.

Solution

IMPORTANT CHANGES WITH THIS VERSION

The SP Stack Kernel 722 PL 400 replaces the previous SP Stack Kernel 722 PL 300. Besides the enhancements and new features described below, it contains many corrections that improve the performance, stability and security of the kernel. All issues reported in SAP Note 2448512 as "Known regressions in kernel 7.22 patch level 300" have been removed.

In addition fixes for the following issues are included:

AWS:

  • The hardware key on Amazon AWS might not be verified due to a corrupted document or signature error. For details see SAP Note 2449062.

CCMS:

  • In SM50 you might see a high number of work processes waiting for semaphore 38. For details see SAP Note 2469083.

DP:

  • Due to an unstable network connection, frontend printing might lead to a hanging spool process. For details see SAP Note 2447577.

In addition, the Common Crypto Library was updated to version 8.5.14. For details see SAP Note 2481365.

NEW FEATURES

New features and enhancements listed below are introduced with the current SP stack kernel 722 PL 400 compared to the previous SP Stack Kernel 722 PL 300:

  • DB2 z/OS: Support for DB2 adde, see SAP Note 2428077
  • DBSL: Support for increased maximum number of rows in Oracle 12, see SAP Note 2395242
  • see also Parameters changed and added section below.

Parameters changed

  • none

Parameters added

  • alert/MTELOGLEVEL and alert/MTELOGFILTER with empty default, see SAP Note 2433763

Parameters removed

  •  none

KNOWN ISSUES

  • In case of a severe issue, we will update this SAP Note as well as the "SAP Kernel Important News" blog on SCN https://wiki.scn.sap.com/wiki/display/SI/SAP+Kernel:+Important+News.

For the list of all known regressions in SAP SP Stack Kernel 722 PL 400, please refer to SAP note 2513278.
