Symptom

  • Rehire using a new personnel number does not work because IT0185 does not allow the same Aadhaar ID to be reused.
  • Error message details:
    Message class: HRPADIN01
    Message No. 404
    Short text: This Aadhaar ID has already been allocated to personnel no & .

Environment

  • Human Capital Management (HCM)
  • SAP ERP Payroll India
  • SAP R/3
  • SAP ERP Core Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA

Reproducing the Issue

  1. Attempt to rehire a separated employee using a new personnel number.
  2. Enter the Aadhaar ID in IT0185 (stored under subtype-06).
  3. Observe that the system displays the error message: "This Aadhaar ID has already been allocated to personnel no & ."

Cause

Note that as per standard system behavior, a unique ID (e.g., Aadhaar ID) cannot be assigned to two different personnel numbers (PERNR).

Resolution

  1. Remove the Aadhaar Number from IT0185 (stored under subtype 06) for employee.
  2. Refer to SAP Note 2576508 - PY-IN: Storing of Aadhaar Numbers in HCM systems for further details.
