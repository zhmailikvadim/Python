Symptom

 In certain cases, the indicator MEHRL is set incorrectly in the table NGEB.

Reason and Prerequisites

To clean up the affected data, you can use the report ZHC_NGEB_MEHRL_KORR. The report was created in LC1 and enables the identification and correction of the affected data records.

Solution

The data is provided as an SAP transport in the usual form as a ZIP file with K and R file.
