Symptom

An email campaigns containing personalization rules is not delivered to the full recipient source but showing a 'no content' error in the email analyzes.

Environment

  • SAP Engagement Cloud 

Reproducing the Issue

  1. Create an automation with a recurring entry node.
  2. Use an email template that consumes personalization rules for product/content recommendations.
  3. Run the automation and compare the Send Email node count to trends and the email report; observe large discrepancy and many cancellations with "no content".

Cause

Only contacts with recent email communication are eligible for personalization rules. Using personalization rules for contacts who have not received any email in the last 180 days and/or sending before the personalization rule calculation has finished leads to "no content" cancellations, creating a large gap between the send node entrants and actual emails sent.

Resolution

  1. Ensure recipients meet personalization rule prerequisites: target only contacts who have received any email within the last 180 days.
  2. Before starting the calculation of each personalization rule, send an email to the intended audience to synchronize contact data, then wait until the rule calculation has finished.
  3. Add a pre-filter segment before the email node to include only contacts eligible for personalization rules (for example, “received any email in the last 180 days”).
  4. Implement branching in the automation: one branch for eligible contacts using the personalized email, and another branch for ineligible contacts sending an alternative email without personalization rules.
  5. After implementing the above, re-run and compare the Send Email node count with trends and the email report; the discrepancy should reduce to expected levels.

See Also

Please refer to the following Help File

  • Personalization Rules - Overview
    • Requirements and good to know
      • Personalization rules work only for contacts who have received a campaign (any email communication) in the last 180 days
      • Before starting the calculation of a personalization rule, you have to send an email to the target of your campaign. This is needed for the correct synchronization of the contact data before the calculation.
    • Known issues and limitations
      • If a campaign using Personalization rules is sent before calculation has finished for that rule, it will not be delivered and the "no content" error message will be displayed.
