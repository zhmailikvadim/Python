Symptom

A new legal change concerning the mapping of the Bank Transaction Code is introduced with the DK "Anlage 3" Version 3.0 (specification of the data formats) - see DK change requests CR FS-15-01 (Bank Transaction Code ISO-Domain), CR FS-15-02 (BkTxCd on Entry-Level), CR-FS-15-08 (Issuer BkTxCd).

This SAP Note contains the following new functionality:

  • Possibility to maintain the ISO-Domain Code (Domain Code, Family Code and Subfamily Code) in the customizing (/PE6/V_PR_TR_CD)
  • Mapping of the ISO-Domain in the DK and ISO CAMT.054 converter (the CML CAMT.054 converter is not affected)

Other Terms

BkTxCd, CR FS-15-01, CR FS-15-02, CR-FS-15-08, /PE1/CL_OPM_CAMT054_DK, /PE1/CL_OPM_CAMT054_ISO, DK

Reason and Prerequisites

Legal change.

Solution

Implement the correction instruction attached to this SAP Note. Alternatively, you can import the referenced Support Package.

Please note that this SAP Note contains manual pre- and post-implementation steps.
