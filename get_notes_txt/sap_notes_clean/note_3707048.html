Symptom

You are implementing a Note. One or more corrections instructions are not displayed.
 
In this document, Note 3525557 is an example.
 
 
Note 3525557 has 2 correction instructions which are valid for the current system. However, only the correction instruction for manual activity is displayed. The auto correction instruction is not displayed. Thus, the changes included in it cannot be implemented.

Environment

NetWeaver ABAP based systems.

Reproducing the Issue

Implement Note via SNOTE.

Cause

The SPAM queue is still not finished. For example, there is unfinished TCI package implementation or support package implementation.

Resolution

1, Finish the SPAM queue or restore the system to the status before the SPAM queue import started. This is to make sure the SPAM queue is empty.
2, Implement the Note again.
