Symptom

  • Error messages occur when generating the report "German Z3 Data Extraction Files" in App Run Statutory Reports.
  • The report was generated with errors for Trial Balance, Accounts Receivables, Accounts Payable, Journal Entries, Tax Line Items, GL accounts, Company Codes, Customers, Suppliers, and Tax Groups.
  • Errors are similar to: Query ID *** in query filter *** of report definition DE_AUDIT_Z3 not found.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Navigate to the App Run Statutory Reports.
  2. Create the report "German Z3 Data Extraction Files".
  3. Observe that the report is generated with error messages for multiple components except GL Accounts - Line Items.

Cause

The issue can be caused by a high amount of data in the extraction result.

Resolution

  1. Reduce the reporting period to a shorter duration (e.g., 3 months).
  2. Generate the report again.
  3. Verify that the error messages no longer occur.
