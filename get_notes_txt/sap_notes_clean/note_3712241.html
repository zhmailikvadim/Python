Symptom

You exchange sequences of a shop floor order in CO02. After exchanging the sequence, execution is not locked for the operations of the exchanged sequence and a worker could start work of an OA of such operation as long the shop floor order is not saved. This SAP Note prevents this behavior by locking execution of exchanged operations.

Other Terms

MPE, Exchange Sequences, Execution Lock

Reason and Prerequisites

Program error.

Solution

Apply correction instruction.
