Symptom

In a Risk Manager Non-Life policy, there is a premium schedule for an assumed participation. When you display the relevant installments of this premium schedule, the program terminates during the determination of the short texts and long texts for the accounting frequency if the determination fails.

Other Terms

ACCFREQU

/MSG/H_GENERIC

022

Reason and Prerequisites

Reason:

If an error occurs, the system tries to issue an error message, which, however, has not been defined.

Prerequisites:

The fiscal year end of the assigned treaty is not equal to December 31.

Solution

History:

Version from Tuesday, October 01, 2019: New

Implement the source code corrections.

Support Packages SAPK-70022INMSG and SAPK-80303INMSG correct the problem.

Solution:

If the short text and long text for the accounting frequency cannot be determined, the system issues a suitable error message.

Manual changes:

Before you implement this SAP Note, perform the manual pre-implementation steps.
