Symptom

  • Error CX005 'No configuration profile selected for configurable object' raised when creating a Sales / Production / Purchase Order for a Material
  • Configurable Material has a Configuration Profile assigned which is Released and valid but still the error "No configuration profile selected" is raised.

Environment

  • SAP ERP Logistics (LO)
  • SAP R/3
  • SAP R/3 Enterprise
  • SAP ERP Central Component
  • SAP ERP
  • SAP Enhancement package for SAP ERP
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA, on-premise 

Cause

The Ehancement CUCQ0001 is activate ( User Exit  EXIT_SAPLCUCQ_001 ) but contains no code or the custom code doesnt return a configuration profile in the internal table YCUCO

Resolution

Please either deactivate the Enhancement or change the coding in the Exit to return the table YCUCO with a configuration profile

Please refer to the documentation in transaction SMOD that gives sample coding that you can use.
