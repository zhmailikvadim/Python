Symptom

Open Manage Workflows for Purchase Requisitions App, there is no Create button for Overall Release of Purchase Requisition (WS02000458) or Release of Purchase Requisition Item (WS02000471) scenario.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Open Manage Workflows for Purchase Requisitions app.
  2. Create button is missing.

Cause

There are 2 possible causes:

  1. The scenario was already imported from another system: The scenario has been transported from the development system to the current system. In this case, the scenario is locked, because all changes applied in the current system, would be overwritten by a new transport from another system.
  2. The scenario was exported to a file: In the Manage Workflow Scenarios app, the workflow scenario has been exported to a .xml file (this includes all workflows belonging to the scenario). This also leads to the scenario being locked.

Resolution

Using the Manage Workflow Scenarios app, unlock the scenario if it is locked because of an import or export.

Note: Once unlock the modification, any user with enough authorization can make changes to the corresponding workflows (in production system). We highly recommend using transport functionality instead of developing on the productive system.
