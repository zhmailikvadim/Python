Symptom

In the Quick Financial Forecasting, you add a new lineitem by clicking on insert button and add the value in ETC fields. The system terminates with the dump "Row types in an internal table and a work area not compatible".

Other Terms

QFP, Fiscal Period, Monthly, Insert, Forecasting, New entry, 500 Internal Server Error, OBJECTS_WA_NOT_COMPATIBLE.

Reason and Prerequisites

Reason: Program error.

Prerequisites: The object where you want to insert a new records does not have any existing planning.

Solution

 Implement the SAP Note.
