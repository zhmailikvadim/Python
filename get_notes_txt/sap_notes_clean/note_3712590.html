Symptom

When distributing documents in Plan Communicator, the system is skipping the position's manager approval to allow participants to view their documents.

Environment

SAP Sales Performance Management

Reproducing the Issue

  1. Navigate to Plan Communicator.
  2. Access the Distributions section.
  3. Select a document.
  4. Generate and distribute the document.
  5. Select the title of the distribution.
  6. Distribute the document.
  7. Observe that the distribution skips the manager approval step and directly routes to the participant for acceptance.

Cause

This is a Known Issue.

Resolution

The fix for this issue is planned to be deployed with the patch 2604. For patch timelines, please review SAP SPM 2026 Release Schedule - SAP Community 

Workaround

No workaround is available.
