Symptom

html href does not work in texts in CR for Eclipse JAVA

Other Terms

CR for Eclipse, java runtime, Gateway API, html href, website link

Reason and Prerequisites

1. Configure a formula in layout in Crystal Reports designer. This formula is a website link.

2. Click preview then click the website link in java app, it failed.

3. In Crystal Reports designer, it works well and jumps to the website.

Solution

This issue is fixed in SAP Crystal Reports, version for Eclipse SP32.

To download the latest Support Packages of SAP Crystal Reports, version for Eclipse, visit:

https://wiki.scn.sap.com/wiki/display/BOBJ/SAP+Crystal+Reports+version+for+Eclipse+-+Downloads
