Symptom

A job with an ODP Extractor source fails with an error message similar to:

10196  5436  R3C-150413  2/2/2026 7:00:11 PM  |Data flow New_DataFlow|Reader Query
10196  5436  R3C-150413  2/2/2026 7:00:11 PM  RFC CallReceive exception <RFC_ABAP_RUNTIME_FAILURE-(Exception_Key: SYNTAX_ERROR)- Syntax error in program
10196  5436  R3C-150413  2/2/2026 7:00:11 PM  CL_RODPS_SEGMENT_FLATTENING===CP        .[SAP NWRFC 754][SAP Partner 750 ][SID][SAPAPPSERVER][DSUSER][4103]>.

Environment

  • SAP Data Services 2025
  • SAP Applications Datastore ODP source

Reproducing the Issue

  • Execute a job containing an ODP Extractor source

Resolution

Apply SAP Note 2598154 - System-local ODP consumer support to your SAP Applications source system.

See Also

SAP Note: 2598154 - System-local ODP consumer support

Keywords

ODPAPI, RODPS, REPL, extractors, BODS, SAPDS
