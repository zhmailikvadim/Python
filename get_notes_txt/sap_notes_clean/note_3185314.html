Issue

Why am I receiving the specified field does not exist when integrating to an Ariba Web Service?

Resolution

Adjust your SoapRequst file to remove any fields not included in the WSDL found at Integration Manager > Integration Configuration in your Buyer or Sourcing realm

Cause

This error is thrown when the formatting of the XML SoapReuqest file does contains an XML element that is not included in the WSDL rules file

Additional Information

the specified XXX does not exist
