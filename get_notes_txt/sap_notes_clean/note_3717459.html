Symptom

  • "Display Parked Journal Entries" does not display line items when simulated.
  • The issue is observed only in FLP (SAPGUI for HTML / WebGUI) and not in the Desktop SAPGUI.
  • No error messages or error codes are shown - just a BLANK page

Environment

SAPGUI for HTML / WebGUI

SAP Kernel 793

SAP Kernel 916

Reproducing the Issue

  1. Execute transaction FBV3 (Display Parked Document).
  2. Enter the following details:
  • Company Code: xyz
  • Fiscal Year: 202*
  1. Click on "Document List."
  2. Fill in the following details:
  • Company Code: xyz
  • Fiscal Year: 202*
  1. Hit "Execute."
  2. Double-click on Document "abc". 

  3. Change the document to edit mode (this is the FV53 transaction ): goto 'Menu' -> 'Document' -> 'Change' 
  4. Simulate the document (this is the FV50 transaction): goto 'Menu' -> 'Document' -> 'Simulate' 
  5. Blank Page is displayed as per below screenshot (The same steps in SAPGUI an ABAPlist is displayed)




Cause

SAP Note 3629293 - Fullscreen container is not shown 

Resolution

  1. Apply the Kernel correction as outlined in SAP Note 3638863 - ABAPlist not displayed
