Symptom

  • No permission is granted to see Biographical Information of an inactive user, yet Biographical card is reflecting in full profile.
  • In legacy version, system is behaving as expected. 

Environment

SAP SuccessFactors Employee Central

Reproducing the Issue

  1. Make sure no permission is granted to see Biographical Information of the target user
  2. Go to the user's profile 
  3. For the inactive employment, Biographical Information is reflecting
  4. Switch to legacy and observe that Biographical Information is not visible as per RBP configuration 

Cause

Read permission is false for employment.

Resolution

The issue has been confirmed as a defect and will be addressed in the 1H2026 Release.
