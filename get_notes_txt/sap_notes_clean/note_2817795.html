Symptom

You want to write a positive transport request which contains all objects that are in scope within the Custom Code Migration App

Other Terms

 CCM, transport request, scoping

Solution

Install the mentioned support package or apply the attached correction instruction in the system and enter the following values in the table SYCMA_CONFIG to enable the positive transport request.

  • ﻿property = CREAT_TREQ
  • value = X﻿

Caution: Either a negative transport request (containing all objects that are not in scope) or a positive transport request (containing all objects that are in scope) can be create in the Custom Code Migration app. There is only one button in the UI to create the transport request and this note will not add a additionally button or change the actual button. The table entry just change the behaviour of the existing button. The change is also applied for all users.
