Symptom

  • When creating a new live model with direct BW connection enabled SAML SSO, a pop-up window appears for entering credential.
  • After entering credential it's possible to see the BW queries.

Environment

SAP Analytics Cloud (Enterprise) 2026.2.6

Reproducing the Issue

  1. Create a new live direct BW connection with SAML SSO enabled in SAC.
  2. Create a new live model via this connection.
  3. After selecting the connection, a pop-up window appears for entering IDP user credential.
  4. After entering credential it's possible to see the BW query data.

Cause

This case is currently under investigation by development group.

Resolution

This case is currently under investigation by development group.
