Symptom

You get an error when activating a dependent transient projection view. The error states that the database statement for the transient projection view could not be generated.

Other Terms

DDIC, PROJ_TRA, ACT, RADMASUT1_C3, MA_REPLACE_VIEWS, DB

Reason and Prerequisites

Transient projection views do not have a database artifact.

A bug in the code for processing the activation of transient projection views did not skip the database statement generation for the entity.

Solution

Implement this note.
