Symptom

In applications that are based on the Floorplan Manager (FPM) an unnecessary and too technical accessibility description is read out by the screen reader when accessing GUIBBs, such as the GUIBBs Form, Form@GL2.0, Form Repeater, List, List ATS, Tree, Search, Chart, Vizframe, Visual Business, or Launchpad. In the case of the GUIBB Form@GL2.0, for example, this accessibility description is 'Form UI Building Block'.

Other Terms

Floorplan Manager, accessibility, ARIA, screen reader

Reason and Prerequisites

Program error.

Solution

Implement the attached corrections or import the relevant Support Package.
