Symptom

  • Loss of diagram linkages occurs when dictionary items are moved to a different dictionary category.
  • Exporting and importing dictionary items removes all diagram linkages.

Cause

Resolution

Dictionary doesn't support MOVING Dictionary items via Excel export and import.

What happens is that NEW entries are created on import or (if importing into a different category and "update only" mode is used). Items are ignored that already exist in a different category.

So, when you import Dictionary items and use the "Update existing and create new ones", then Dictionary will create new ones and these are not linked in any diagram yet, and that's why the diagram linkages are empty.

This is expected behavior.
