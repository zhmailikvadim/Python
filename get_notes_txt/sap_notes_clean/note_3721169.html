Symptom

In the process "Sending of Split Allocation List (Gas)" (DE_B_MM_SEND_SPLIT_ALLOC_LIST), the MeMi allocation list information was not persisted on the sender side.

This issue is relevant for the market role: DSO.

Other Terms

MeMi Billing, Gas, Allocation List, DE_B_MM_SEND_SPLIT_ALLOC_LIST, /US4G/ALLOC_HDR, /US4G/ALLOC_ITEM, BC set

Reason and Prerequisites

The error occurred because of a function gap in the system.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the system runs a new version of the process "Sending of Split Allocation List (Gas)" (DE_B_MM_SEND_SPLIT_ALLOC_LIST) and persists the MeMi allocation list information in the new process step 0100. The BC set /US4G/3_0_DSO_SP08_NOTE_3721169 contains the necessary corrections. You can verify the fix by checking that the new version of the process runs and that the MeMi allocation list information is persisted in step 0100.
