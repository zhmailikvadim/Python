Symptom

  • For projection function set to None for a Measure in an OLAP BICS Universe, the following error is encountered after performing Check Integrity:
    "The aggregation function cannot be empty."

Other Terms

idt, check integrity, aggregation function, empty, projection function, none, olap, bex

Reason and Prerequisites

Environment:

  • SAP BusinessObjects BI Platform 4.3 SP04 Patch 7
  • Information Design Tool (IDT)
  • BEx Query

Steps to Reproduce:

  1. Log in to Information Design Tool (IDT).
  2. Create a Universe (.unx) on top of internal G75 connection.
  3. In the Business layer, create a measure object and set the Projection Function to ‘None’.
  4. Perform a Check Integrity operation in the universe.
  5. You will notice that under 'Check Business Object OLAP Validity', the following error is encountered:
    "The aggregation function cannot be empty."
