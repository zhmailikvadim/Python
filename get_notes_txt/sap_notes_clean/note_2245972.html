Symptom

The below error message appears when the portfolio item is moved from one bucket to another. This happens when the related operative project is in Completed/Closed/Locked status and the DFM synchronization is active.

'An error occurred when changing the application object. <item ID>'

Other Terms

Item, Error message, Reassign Bucket, Move to, Project, Completed/Closed/Locked status, DFM synchronization, CL_RPM_INM_OUTBOUND.

Reason and Prerequisites

Program Error.

Solution

Implement the correction instruction provided with the SAP Note.
