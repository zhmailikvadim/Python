Symptom

Subcontracting PDS integration fails in SAP IBP with the following error message:

  • "Transportation lane <parameters>: Input data contains new record but not all required attrs are present. "

Other Terms

IBP, Real-time integration, RTI, Subcontracting, PDS, transportation lane, tlane, tlane-x, tlanex, x-structure, x structure, x-fields, checklist, required attributes, BAdI, no_load, lv_no_load, no load, empty

Reason and Prerequisites

During the integration of subcontracting PDS into SAP IBP, the checklist for the transportation lane (tlanex structure) is empty. The required fields are not set to 'X'.

It can occur when multiple subcontracting PDSs are included in the integration model, and at the last processed PDS, the no_load parameter is set to true in the BAdI.

Solution

Implement the note or apply the corresponding support package.
