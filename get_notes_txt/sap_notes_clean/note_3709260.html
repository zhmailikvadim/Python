Symptom

Adding an alternative payee or payer to a BP with customer or supplier role at company code level. No error is raised if the master data used as alternative payee/payer is not defined for the current company code

example:

Customer B is not extended to company code 0001

During maintenance of customer A from BP transaction at company code 0001, you try to assisgn customer B as Alternative Payer.

Without this SAP note, no error would raised

Other Terms

BUKRS,

KNZA, EMPFD,  F2 154

LFZA, EMPFK, F2 164

Reason and Prerequisites

Program error

Solution

Implement the attached correction instruction or equivalent support package
