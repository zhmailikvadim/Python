Symptom

  • During the ADS configuration wizard on an on‑premise SAP NetWeaver Java system, an error occurs when maintaining the HTTP destination in Java.
  • Error message: "Instance of URLAddress object is valid if it consists at least the protocol and the host name."
  • I ABAP system FP_PDF_TEST_00 works, while FP_CHECK_DESTINATION_SERVICE fails with : "SYSTEM ERROR: ADS: com.adobe.ProcessingException: com.adobe.ProcessingException: XMLFM Exception during render: com.adobe.ProcessingException(200.101)."

Environment

  • Product: SAP NetWeaver
  • Forms Processing

Reproducing the Issue

  1. Run the ADS configuration wizard.
  2. Navigate to the step to maintain the HTTP destination in Java.
  3. Provide only host details (hostname or IP) without specifying the required port.
  4. Observe the error: "Instance of URLAddress object is valid if it consists at least the protocol and the host name."

Cause

  • Incorrect host and port details were provided for the Central Services Registry Proxy during the configuration wizard.

Resolution

Make sure you can access this URL with user ADS_AGENT and password.
http://<ABAP server>:<ABAP http port> /sap/bc/fp/form/layout/fp_test_00.xdp

Use ABAP http port that can be found under T-code SMICM, Goto -> Services
