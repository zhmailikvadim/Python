Symptom

The Log Details button isn't available in the Track Sales Order application for a sales order that has a delivery block but no associated outbound delivery.

Other Terms

ISLS, Intelligent Clinical Supply Management, Track Sales Order Application, Log Details

Reason and Prerequisites

You use SAP Intelligent Clinical Supply Management.

Currently, the Log Details button appears only in the Delivery Issues in Sales Order tab, which is specific to outbound deliveries. Because of this, the Log Details function isn't available in the following scenario:

  • When a sales order has a delivery block but the outbound delivery hasn't been created yet.

Solution

The Log Details button is now available on the Delivery Order tab in the Track Sales Order application.

Implement the attached correction.
