Symptom

  • Formula field returns the date as MM/YYYY format when the report output is XLSX
  • When the report output is CSV, the formula field displays a different format, such as Oct-25

Reproducing the Issue

  1. Generate the report output in XLSX format and observe the correct MM/YYYY format for the date
  2. Generate the report output in CSV format and observe the discrepancy in the formula field, which displays the date in a different format, such as Oct-25

Cause

If you open the CSV file using a text editor, you'll notice that the actual values are populated accurately in the MM/YYYY format according to the formula. However, when opened in Excel, the format may change due to Excel's automatic conversion settings.

Resolution

You need to manually adjust the date format after importing the CSV into Excel by using cell formatting options to achieve the desired MM/YYYY format.
