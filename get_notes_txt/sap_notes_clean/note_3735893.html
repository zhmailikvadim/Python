Symptom

  • While activating Best Practices in Implementation Assistant, the process stops with an error in scope item 5XM.
  • Error message: "Field UEPST not found"

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental. 

Environment

  • Sales and Distribution (SD)
  • Logistics Execution (LE)
  • SAP Credit Management (FSCM)
  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA

Reproducing the Issue

  1. Execute transaction code /n/SMB/BBI.
  2. Choose a solution from Solution list.
  3. Click on Implementation Assistant.
  4. Click on Activate Icon. 

Resolution

1. Start transaction /SMB/BBI.

2. Click on a solution from Solution list.

2. Go to SI DE_5HP_OP->BB 5XM-> Click activity V_SDSLS_DOC_VCM

3. Click on Definition

4. Click Retrieve
