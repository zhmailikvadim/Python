Symptom

  • The new directory search does not respect the target population permissions.
  • When users search for custom fields, they can view search results for employees outside their target population.
  • This behavior allows employees to access information they do not have permission to view, revealing sensitive data.

Environment

SAP SuccessFactors Employee Central - Employee Directory Search

Reproducing the Issue

  1. Access the directory search as a regular employee.
  2. Search for a value.
  3. Observe that multiple employees appear in the search results, even if the access user does not have permission to see those information on the employees' profile. 

Resolution

 

The issue has been confirmed as a defect and will be addressed in the 1H2026 Release.
