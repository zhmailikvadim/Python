Symptom

You run the Programa de cálculo da folha de pagamento - Brasil (HBRCALC0) report. However, when the system recomposes the taxable income for the current month, it incorrectly considers taxable income amounts from the previous month instead of only using the current month.

Example:

Consider an employee with the following payroll cluster:

Payroll Period 
RTRIB (Taxable Income)
01.2026
BRL 5,000.00

When you run the HBRCALC0 for 02.2026, the employee contains the following payroll cluster:

Payroll Period  RTRIB (Taxable Income)
01.2026 BRL 5,000.00
02.2026 BRL 11,000.00 (BRL 6,000.00 + BRL 5,000.00)

Note: The RTRIB for payroll period 02.2026 totals BRL 11,000.00, which is the sum of BRL 6,000.00 earned in the current month (02.2026) plus BRL 5,000.00 carried over from the previous month (01.2026).

Other Terms

BRTI; payroll function; taxable income; recomposition; RTRIB; payroll cluster; previous month;

Reason and Prerequisites

Reason

When the system searches for previous payroll clusters to perform taxable income recomposition, it includes taxable income values from previous months instead of limiting the search to the current payroll period only.

Prerequisites

The following Support Package level is the minimum prerequisite:

Release Support Package
SAP_HRCBR 6.04 (EhP6 and EhP7) HR SP J1
SAP_HRCBR 6.08 HR SP B9
S4HCMCBR 1.00 HR SP 08
S4HCMCBR 1.01 HR SP 00
S4HCMCBR 1.02 HR SP 00

For more information, access the SAP ERP Human Capital Management - Support Package Schedules.

When you install an SAP Note, some prerequisites might be required. The prerequisites are SAP Notes that have already been released and that change the same context of the current one. Then, even if these prerequisites are not necessarily related to the symptom of this SAP Note, you must implement them due to technical reasons.

Solution

When you run the HBRCALC0 report, the system correctly recomposes the taxable income for the current month using only the current month amounts, without considering taxable income amounts from previous months.

Example:

Considering the scenario from the Symptom section, when you run the HBRCALC0 for 02.2026, the employee now contains the following payroll cluster:

Payroll Period  RTRIB (Taxable Income)
01.2026 BRL 5,000.00
02.2026 BRL 6,000.00
