Symptom

DMS attachments are deleted from S/4HANA Public Cloud but the deletion does not happen from the Repository immediately

Environment

  • SAP S/4HANA Cloud Public Edition
  • Document Management (DMS)

Cause

In order to avoid any database inconsistencies especially for a deletion and subsequent rollback, the DMS attachment deletion will be done as part of a daily technical job run.

Resolution

The technical job RSIR_CONTENT_UNMARK_PRELIM is scheduled to run every night at midnight in S/4HANA Public Cloud. During this run, any attachments that were deleted more than one day ago are permanently removed from the repository.
