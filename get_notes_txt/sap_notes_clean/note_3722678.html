Symptom

The logging for the archiving object HRCATGARNI is incomplete. The system only displays entries from infotype 131.

Other Terms

ILM, HRCATGARNI, garnishment, log, BAdI

Reason and Prerequisites

The system does not take into account all affected infotypes in the log.

Solution

The correction is provided by HR Support Package (via CLC if necessary).
You can implement an advance correction by implementing the automatic correction instructions using SNOTE.

After you implement the corrections, the system displays all affected infotypes in the log.

To do this, the system uses a standard implementation of the BAdI HRARCH_B_ADDITIONAL_LOG with the name HRARCH_GAR_ADDITIONAL_LOG.
