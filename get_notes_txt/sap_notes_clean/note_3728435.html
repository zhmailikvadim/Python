Symptom

Note 3636830 or corresponding support package is implemented. Errors during event POST in the accounting interface (DOCUMENT_POST) may be lost or overwritten, leading to the POST not being rolled backed as expected.

This can lead to unexpected data inconsistencies depending on the error which was ignored.

Possible examples: missing g/l view (ACDOCA) although the entry view (BSEG) is posted, missing entry view although g/l view is posted, missing secodary documents or inconsistent data within one of the tables.

Other Terms

 RWCL ACDOCA CL_FINS_ACDOC_ADAPTER CL_FINS_ACDOC_POSTING_EVENTS FINS_COMMIT_DETECTION POST TRPWR DOCUMENT_POST

Reason and Prerequisites

Program error: Code changes in note 3636830 overwrite the sy-subrc after TRWPR call.

Solution

Implement correction instructions or corresponding support package.
