Symptom

Purchasing OData V2 services are deprecated as of SAP S/4HANA 2023 and should be replaced by OData V4 services.

However, after upgrading to SAP S/4HANA 2023 or higher, OData V4 services are not available in transaction /IWFND/MAINT_SERVICE.

Environment

  • SAP S/4HANA 2023 or higher
  • OData APIs

Cause

The new OData V4 services for Purchasing are not automatically registered during the system upgrade. Also, transaction /IWFND/MAINT_SERVICE is only applicable to OData V2.

Resolution

ODATA V4 services need to be registered in transaction /n/IWFND/V4_ADMIN (not /n/iwfnd/maint_services).

For detailed registration steps, refer to:

 KBA 2948977 - How to Register OData V4 Service in SAP Gateway System?

See Also

Customizing and extensions implemented for OData V2 services are not automatically available in OData V4 services.

If custom fields or extensions are required, refer to:

  • KBA 3447178 - Custom field not available in Purchase Order API V4
  • KBA 3613327 - Extension to Api_purchaserequisition_2 not Reflected in Odata Api - S/4HANA
