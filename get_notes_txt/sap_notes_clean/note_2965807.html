Symptom

You want to import a configuration that was previously exported in XML format into the Management Console. However, the import fails with an error message if the configuration contains an OPC UA destination system as the destination of a notification.

Other Terms

KeyNotFoundException, Management Console, Plant Connectivity, export, import

Reason and Prerequisites

Program error

Solution

The error is corrected with Patch 1 for SAP Plant Connectivity 15.4 with Support Package 01 (PCo 15.4.1).

Download the current installation program for your release from SAP Service Marketplace and install it in accordance with the instructions in the installation guide. The current version of the installation guide is available on the PCo documentation page on the SAP Help Portal under https://help.sap.com/pco. You can use the installation program for the PCo patch to update a PCo version that is already installed or to set up a new installation if PCo is not yet installed on the target host.

Prior to the installation, also refer to the installation note 2926478 for PCo 15.4 Support Package 1.
