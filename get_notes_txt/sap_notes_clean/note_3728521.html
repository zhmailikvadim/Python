Symptom

Multiple vendors are displayed under partitioned data in Ariba Downstream.

Environment

SAP Ariba Core Administration Supplier Management

Reproducing the Issue

  1. Open vendor in downstream.
  2. Navigate to partitioned data.

Cause

Resolution

  1. Access the Intelligent Configuration Manager workspace.
  2. Navigate to the Parameter Management section.
  3. Locate the parameter "Check partitioned supplier data f|or multiple associations on import".
  4. Set the parameter to "Yes" to validate the data and ensure that a partitioned supplier cannot be associated with another common supplier if previous associations exist.
  5. Save the changes.
  6. Regularly review the partitioned supplier data to maintain data integrity.
