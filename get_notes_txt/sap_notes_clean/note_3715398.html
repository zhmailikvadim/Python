Symptom

You use Transaction Manager of Treasury And Risk Management. In the Additional tab in FWZZ, the system allows values belonging to other Attribute IDs to be entered in Short Text or Long Text fields without triggering a validation error.

Other Terms

FWZZ, Additional Tab, Short Text Field, Long Text Field, VWPT_ATTR_L, VWPT_ATTR_S, Attribute ID,

Reason and Prerequisites

Program Error

Solution

Implement the correction instructions or import the corresponding Support Package.
