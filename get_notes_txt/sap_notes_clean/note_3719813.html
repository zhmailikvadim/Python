Symptom

FPIP Cancel with CDOTE Order error 'Unexpected number of response lines' for an intracompany contract.

Replication Steps:

1. Create an intracompany trading contract with flat price fixation from WB21.

2. Go to transaction WB22. Create a cancel with the CDOTE order using the CDOTE order type FPIP.

3. While the fill process system shows the error 'Unexpected number of response lines' or dump.

Other Terms

 Agricultural Contract Management; ACM; CDOTE; Cancel; FPIP; LOG_CMM_PRC_PSM015

Reason and Prerequisites

  Program error

Solution

 Import the relevant support pack or apply the correction instruction(s).
