Symptom

In BELCOTAX - Yearly Income Declaration (RPCBWAB0) for income year 2025, this SAP Note delivers following changes according to the latest brochure and *.xsd:

  • The content in xml tag "f10_2228_relanceovertimehours2025" of Fiche 281.10 and the xml tag "f20_2228_relanceovertimehours2025" of Fiche 281.20 should be maximum 7 characters length.
  • Removal of the validation rule regarding the pension bonus (2.088) in fiche 281.11.

Apart from the above changes, missing customization for fiche 281.17, 281.30 and 281.50 regarding the xml tag "f2105_birthplace" is provided in attached "SAP_NOTE_3707240_Customization_Overview.xlsx".

Other Terms

BOW, BOWA, Zone 2.228, Zone 2.088, f2105_birthplace,  BELCOTAX - Yearly Income Declaration (RPCBWAB0), BELCOTAX-ON-WEB, Belcotax

Reason and Prerequisites

 This SAP Note is only relevant for Payroll Belgium.

Solution

  • Generally, SAP recommends that you implement the HR Support Package specified for your release.

  • However, if you need to install a solution earlier, use the SAP Note Assistant to implement the correction instruction.

  • You can find more information about the SAP Note Assistant in SAP Support Portal, under SAP Note Assistant.
