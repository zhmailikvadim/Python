Symptom

incompletion logs show an item has missing value for some fields, for example, goods issue date in schedule line. However, the item actually does not exist any more.

Environment

  • Sales and Distribution (SD)
  • SAP ERP Central Component
  • SAP ERP
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Open transaction VA02 or VA03 for the sales order.
  2. From the menu, choose Edit > Incompletion log.

Cause

The issue is caused by inconsistency.

Resolution

  1. Add a new item line with same item number, press Enter key via VA02.
  2. Remove this item line, then save the sales order.
