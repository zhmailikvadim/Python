Symptom

You try to send a Change Record with Related Items with a specific Item Relation Type. No relation is maintained in Customizing under Cross-Application Components -> PLM-Systemintegration -> DM Change Object to SAP Change Record -> Map DM Change Object to SAP Change Record -> Define Relation Type. You receive the error message "No valid configuration found for change record type &1" (/PLMS/BO_CO_CORE 004) or "No valid configuration found for change object type &1" (/PLMF/BO_CO_CORE 003).

The error message is not sufficiently descriptive and does not clearly indicate that the required customizing is missing.

Other Terms

PLMSI, PLM System Integration, Change Object, CO, Change Record, CR, Reference Item, Solution Item, Item Relevance, Relation Type

Solution

 Implement this SAP Note to resolve the symptom.
