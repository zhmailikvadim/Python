Symptom

The system does not allow the migration of opening balances for Asset Under Construction (AUC) assets with individual line items during migration.

Environment

SAP S/4HANA Public Cloud

Reproducing the Issue

  1. Open Migrate Your Data -Migration Cockpit app
  2. Upload Fixed asset - Postings migration object. 
  3. Observe the system does not allow the migration of opening balances for Asset Under Construction (AUC) assets with individual line items during migration.

Cause

For assets under construction linked to an investment project (where the indicator "Assets Under Construction for Investment Measures" is set), neither manual nor automatic legacy data transfer with the S/4HANA migration cockpit is supported.

Resolution

Only independent assets under construction have line item management. 

Transfer the line items using the special transaction type for prior-year acquisitions (900 - Takeover open items APC (AuC)).

Refer to the help document for more details: Transferring Assets Under Construction.
