Symptom

When opening Fiori app F5425 (Manage Routings), startup fails with the errors: "Error while processing building block FilterBar" and "TypeError: Cannot read properties of undefined (reading 'map')".

Environment

  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Open Fiori App F5425 (Manage Routings).
  2. Observe that the app fails to load at startup and displays the error.

Cause

This issue is caused by program error.

Resolution

Implement SAP Note 3697073 – “F5425 - Manage Routings Fails to load at startup - Part 2” to resolve the issue.

If not yet applied, also implement following SAP Notes:

3696802 – “F5425 - Manage Routings Fails to load at startup - Part 1”

3652946 – “Addition of Change number parameter to Manage Routing app”

Since this is a front-end correction, ensure to follow the below steps after the implementation of the notes:

  1. Execute the report /UI5/APP_INDEX_CALCULATE in gateway system
  2. Choose the tab 'Single SAPUI5 Repository Only'
  3. Enter the UI5 repository ROUTING_MANS1 and execute
  4. Perform Empty cache and hard reload in the browser before retesting the application -> To do this, kindly first press F12 and  then right click on the refresh button -> select clear cache and hard reload here.
