Symptom

In the process "Decommissioning of REM Full Feed-In Installation" (DE_B_MCM_MKT_PROCESS_INST_SHUTDOWN), the system attempts to trigger a deregistration in MaCo in step 0040. This results in an error, because no MaCo deregistration is required for a feed-in installation.
This issue is relevant for the market role: REM.

Other Terms

DE_B_MCM_MKT_PROCESS_INST_SHUTDOWN, REM full feed-in installation, MaCo, step 0040, decommissioning, deregistration, MCM process

Reason and Prerequisites

The error occurs because the system incorrectly triggers a MaCo deregistration for a feed-in installation. The program catches an information message as an exception, which causes the MCM process to go into error.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the system no longer triggers an error when processing the decommissioning of a REM full feed-in installation. The process completes without attempting an unnecessary MaCo deregistration.

You can verify the fix by running the decommissioning process for a REM full feed-in installation and confirming that the process completes without error in step 0040.
