Symptom

Component: Recipe Management
Module: Specification Management

You use Specification Management in the PLM Web user interface (PLM Web UI) to access specification data. The CBUI_BO API is used in the background to access this data.

You inherit data from one specification to another one. Using the SAP GUI the data is displayed completely independent of being inherited or not while in the Web UI instances not being inherited having the same sort order are not displayed.

Important remark about CBUI_BO APIs:
Do not confuse CBUI_BO APIs with CBUI APIs which are used in specification management (for example, in transaction CG02BD). CBUI_BO APIs are used internally for Recipe Development Web user interface.

Other Terms

CBUI_BO
Inheritance
RC1R0INH
Valuation instance

Reason and Prerequisites

Reason
Program error. The error is caused by SAP note 3485203. This SAP note did not take into account that there might exist non-inherited valuation instances having the same sort order as inherited ones.

Prerequisites
For information about the validity of the corrections, see the correction instructions.

Solution

The section "Support Packages" specifies the support packages that contain the corrections.

Alternatively, you can implement the attached correction instructions.
