Symptom

The customs authority in the Netherlands has introduced the new customs system DMS 4.0 for exports.

Adjustments to the data retrieval for the electronic messages of the customs export declaration are required.

Other Terms

Customs Management, CuMa, customs declarations, DMS, AGS, migration, Belastingdienst, customs, import, export, Aangiftesysteem, AGS3

Reason and Prerequisites

Legal change

Solution

Implement the correction instructions.

For information about using the complete function for the new customs system DMS, see the central SAP Note 3277213.

You want to set up Location Core Data Services (CDS) Views as Data Sources for Data Extraction in Environment Management or Incident Management.
You cannot find CDS Views for Locations objects.

Other Terms

Reason and Prerequisites

Reason:
Missing functionality.

Prerequisites:

Solution

The "Support Packages" section lists the Support Packages that contain these corrections.

Alternatively, you can implement the attached correction instructions and execute the post manually correction instructions.
