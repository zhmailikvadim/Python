Symptom

With note 3201422 (change request TTP 2746), transaction YYPREF_STPREFS_DET (report YYPREF_STPREFS_DETERMINE) was created. The transaction allows writing of stated preferences to the table YYPREF_STPREFS. Messages for deliveries and invoices are evaluated.

The current note makes some changes in transaction YYPREF_STPREFS_DET.

Creation date (selection parameter "Created on") is obligatory.

Selection parameter "Ship-to party country" is added.

Deliveries to a country that is not a ship-to part, not an EU country and not a preference country are not considered.

Items with empty preferences are not considered.

Other Terms

C2S4500001240, C2S 4500001240, 4500001240, TTP 3182, TTP3182, 3182, #3182, C2SFIBU, YYPREF_STPREFS, YYPREF_STATEMENTS.

Reason and Prerequisites

Change Request 4500001240, TTP 3182.

Prerequisites: Note 3201422 (Change Request 4500000612 TTP 2746) is implemented.

Solution

Implement changes according to the correction instructions.

Please note: Additional customizing may be required to activate CR.
