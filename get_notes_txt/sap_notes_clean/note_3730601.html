Symptom

You use Intrastat solution for Bulgaria in S/4HANA or SAP GTS(Global Trade Services). Statiscal value and Supplementary unit when zero or not maintained are written into the declaration file as zero. However, the authorities requires the reporting of null when the Statiscal value or Suplementary unit is  zero else the declaration file is rejected.

Other Terms

S/4HANA, GTS, GTS Edition for HANA, Intrastat, Bulgaria, Statistical Value, STVAR, Supplementary unit, SPQUR, /ECRS/RP_EDIT, Manage Intrastat Declaration.

Reason and Prerequisites

This is a legal requirement.

Solution

Implement the attached correction.
