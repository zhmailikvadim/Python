Symptom

In the Manage Compliance Requirements app, you are using a large number of equations (more than 10,000), which may lead to slowed system performance.

The following actions in the application may be affected:

  • Status change button to "Released" or "Under Revision"
  • "Back to Compliance Requirement" button
  • "Save" button on the "Equations" tab 
  • Modification of equations and their inputs and results

This SAP Note resolves the issue by optimizing the logic executed when the status of a requirement is changed. The logic for setting property conditions of equation, input, and result nodes has been improved. In addition, the consistency check logic for equations has been optimized.

Other Terms

EHS Compliance Requirement, Manage Compliance Requirement, Equations, F5038.

Reason and Prerequisites

Reason: These improvements enhance the overall performance and help prevent long processing times, memory errors, timeouts, and system dumps when users work with the application.

Prerequisites: The validity of the correction is maintained in the correction instructions.

Solution

The Support Packages that contain this correction are listed in the "Support Packages" section.

Alternatively, you can implement the correction instructions of this SAP Note.
