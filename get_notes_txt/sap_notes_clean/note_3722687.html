Symptom

You are using the Manage Purchase Requisition Professional app to create or edit a Purchase Requisition. You choose to create an item with account assignment category 'P' (Project). Even though you enter non / a wrong WBS element, the system does not return the error "Enter Work Breakdown Structure Element (WBS Element)".

Other Terms

F2229, WBS Element, ME083, ME 083, CL_MMPUR_DO_ACC_CHECK_ENRICH, WBS_ACC_CHECK

Reason and Prerequisites

Reason: Program error

Prerequisite: The Project field has been set as "Mandatory" in OME9 for Account Assignment Category P (Project).

Solution

Implement the attached Correction Instructions into your System.
