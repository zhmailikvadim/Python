Symptom

Due to a program error, it was possible in the past that an order-specific routing became inconsistent.

When trying to delete the inconsistent order-specific routing, the deletion unsuccessfully terminates with an error message MESSAGE_TYPE_X_TEXT in program SAPLMPE_OAN_POSTS.

Other Terms

MSFR7

Reason and Prerequisites

The following two SAP Notes are missing in the system:

3207387 - Move order update to an earlier point in time during OSR release

3216545 - Ignore deleted operations in the release procedure

Due to the missing SAP Notes it was possible that the operation activity network layer of an order-specific routing was released by accident.

Solution

First implement the SAP Notes 3207387 and 3216545, if not already done.

Implement this SAP Note. This SAP Note provides the repair program MPE_RTGV_HF_REPAIR_OSR for an inconsistent order-specific routing.

Execute program MPE_RTGV_HF_REPAIR_OSR in transaction SE38. Enter the Routing Group, Counter and Version of the order-specific routing.

(Remark: When executing the program in test mode, no database updates are performed.)

On the result screen you can see if the order-specific routing was successfully updated. Afterwards, it should be possible to delete the inconsistent order-specific routing.
