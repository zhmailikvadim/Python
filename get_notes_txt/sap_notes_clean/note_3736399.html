Symptom

When sending email for a correspondence that contains many recepients, each recepient will get an email request.

Environment

SAP S/4HANA

Reproducing the Issue

1. Create Correspondence and click "Email Preview"
2. Input multiple recepients and click "Send Email"
3. Check it in SOST, each recepient gets an email request

Resolution

It's a standard behavior that each recepient will get a email request.
