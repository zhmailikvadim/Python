Symptom

In the Agreement Request, the schedule item data is not properly displayed for single line-item schedules. Due to this issue, the Event and Trigger icons are not shown in the Schedule tab.

This SAP Note corrects the item data handling logic to ensure that the schedule item data is passed correctly. After implementing the note, the Event and Trigger icons will be displayed properly in the Schedule tab.

Other Terms

/IRM/IPAGRM, /IRM/IPA_AGR_EVNT_MAINTAIN, Assigning Event, Trigger, Event

Reason and Prerequisites

This is caused by program error

SAP Note#3716678 is available for ECC Release

Please implement SAP Note#3695810 as a Prerequisite note for 2023 V2, 2023 V3 and 2023 V3 Support Pack 1

Please implement SAP Note#3630797 as a Prerequisite note for 2023 V2 Release

Solution

Please implement the correction instructions attached. These corrections are only valid if you have applied the Support Packs.

Release V/4 - 2023 V2, 2023 V3 and 2023 V3 Support Pack 1

 Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object.
