Symptom

  • Inquiry regarding the potential replacement of SIS table S060 with a CDS view or its possible deprecation in the future.
  • Query about the future of SIS table S060 in the context of migration from ECC to S/4HANA.

Environment

  • Sales And Distribution (SD)
  • SAP R/3
  • SAP R/3 Enterprise 4.7
  • SAP ERP Central Component
  • SAP ERP
  • SAP Enhancement package for SAP ERP
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Resolution

  1. Infostructure S060 will not be replaced with a CDS view.
  2. The classic SD rebate process is obsolete in S/4HANA, and there are no plans for any changes to this infostructure.
