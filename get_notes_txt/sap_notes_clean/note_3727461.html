Symptom

  • Error message MLCCS099: "Internal error in FORM/FUNCTION CKMO_UPDATE_ORD_HIST_MLCCS in position GT_MLCCS_ORD_HIST EMPTY with RC 1 for KALNR" occurs during goods receipt upon declaration of production order.
  • Price determination ID mismatch between MBEW (value: 3) and CKMLHD (value: 2) for the material.

Environment

  • Controlling (CO)
  • SAP ERP
  • SAP S/4 HANA

Reproducing the Issue

  1. Perform goods receipt (GR) for production order.
  2. Observe the error message MLCCS099: "Internal error in FORM/FUNCTION CKMO_UPDATE_ORD_HIST_MLCCS in position GT_MLCCS_ORD_HIST EMPTY with RC 1 for KALNR".
  3. Check the material details in transaction MM04.
  4. Confirm the Price determination change with Transaction MM02 or CKMM

Resolution

  1. Confirm implementation of Helpdesk, if Helpdesk reports are not implemented, follow KBA 2314493 - How to install the Material Ledger Helpdesk reports in 10 steps in R/3** New transaction FCMLHELP** 
  2. Execute the report FCMLHELP_SAPRCKMU adding flag "Check statistical values" to confirm the inconsistency.
  3. Open a case under CO-PC-ACT component referencing this KBA and ask SAP Technical Support to make corrections.
