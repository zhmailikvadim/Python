Symptom

You are using Focused Build applications Mass Change application and observe the following issue:

  In Mass Change application, you see that when the wave choosen in cross wave for Defect/Defect Correction is having the text of wave as  TaskType "Test Planning & Execution" instead of Task Type as "Wave"

Other Terms

S1TM, S1DM, Defect Correction, Cross Wave, REQUESTED_PHASE, PHASE_LEVEL1, /SALM/ITPPM

Reason and Prerequisites

You have installed ST-OST 200 SP12, SP13, SP14, SP15

Solution

This SAP Note provides a report /SALM/NOTE_3539753  to update the entries of table /SALM/ITPPM.
Once the note is implemented, execute the report /SALM/NOTE_3539753  by providing the object IDs of all the defect corrections
