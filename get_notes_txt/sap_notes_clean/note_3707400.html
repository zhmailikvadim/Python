Symptom

CUPS is automatically adding a start banner page containing the text "#pdf-banner" to the printed documents.

Environment

  • Print and Output Management

Reproducing the Issue

Printing with CUPS, unwanted page PDF-BANNER is also printed

Cause

This page is generated by the operating system and has nothing to do with the SAP cover page.

Resolution

To deactivate this title page, set the profile parameter "rspo/host_spool/burst_pages" to 0 in the instance profile.

As this parameter rspo/host_spool/burst_pages is not standard, probably receive a "parameter is not known" message in RZ10.

 This is only a warning message because it is not a standard parameter but the SAP kernel uses it nevertheless.

See Also

SAP Note 77540 - "Unwanted title pages during printing (1)"
