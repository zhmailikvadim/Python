Symptom

Hot keys are missing in collective search help DFS_MAT1_SHLP for DFS_SHLP_FMPO_ACCPART, DFS_SHLP_FMPO_MODEL, DFS_SHLP_PLTFRM_MATNR, DFS_SHLP_PRODUCT_EXTWG and DFS_SHLP_MAT0M.

Other Terms

S/4HANA Defense & Security, D&S, Logistics, Hot Key, Search Helps, Elementary Search Helps, DFS_MAT1_SHLP

Reason and Prerequisites

This is an enhancement for better usability.

Solution

Hot Keys have been added to all included search helps in DFS_MAT1_SHLP based on the availability of suitable hot-key options. Please implement the attached correction instruction for solution.

You declare your export or re-export declaration and you are using SUMA document on item level, to generate <PreviousDocument> in the XML message. You do not declare any Packaging for this document in SAP GTS, but the <Packaging> element is a part of the <PreviousDocument> with 0 value. This causes a validation error and the declaration is not sent to customs.

Other Terms

Denmark, XML, /SAPSLL/LLEG_ECC_DATAF34, ECC_VIEW_MODIFY_DK_DMS_GEN, SUMA, Packaging, WriteOff, QuantityQuantity

Reason and Prerequisites

The problem is caused by a program error.

Solution

Implement the correction instructions.
