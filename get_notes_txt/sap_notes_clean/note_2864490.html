Symptom

Scheduling of a Workbook can fail under certain circumstances if variable values have been passed that are dependent on each other.

Error 352 is listed in the log file.

Other Terms

scheduling, precalculation, 352, variables, prompts, compoundment, compound

Reason and Prerequisites

Program error

Solution

If an error occurs when setting a Variable Value, this correction synchronizes the Variables and repeats the setting in the event of an error.
In this case, the log file outputs information:

"Setting the variable value was not valid. For this reason, the variables are synchronized and the setting of the value is repeated."

-

The correction will be delivered by a SAP BusinessObjects Analysis, edition for Microsoft Office BIP Add On Support Package.
In section "References" you can find a "...Release Note" whith the information which BIP Add On Support Package number contains the respective correction.
