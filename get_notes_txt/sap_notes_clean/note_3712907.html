Symptom

While posting the Master Agreement promotions in an agreement change scenario, after modifying the promotion validity dates, the system throws a validity error. This issue occurs during the posting process when the promotion dates are adjusted.
This note ensures that no validity error occurs during the posting of Master Agreement promotions.

Other Terms

/IRM/CL_IPPQ_PROCESS, ACTIVITY_CHANGE_DATA_PREPARE, Promotions mass posting, /IRM/GPRM 002, GS_XCMO_HEADER-DATBI

Reason and Prerequisites

This is caused by Program error.

SAP Note#3712906 is available for V/4 release.

Solution

Please implement the correction instructions attached. These corrections are only valid if you have applied the Support Packs.

Release 6.4 – Base Package & Support Package 1
Release 6.3 – Base Package
Release 6.2 – Base Package

Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object
