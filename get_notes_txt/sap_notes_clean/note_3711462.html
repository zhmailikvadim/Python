Symptom

Using Analytical Reporting API, Synchronous call you receive the error: "Error": "Document nesting depth (1001) exceeds the maximum allowed (1000)

Cause

The issue is caused by the absence of a filter in the viewTemplate configuration leads to the inability to retrieve all values, as there is a limit of 1,000 records per API call.

Resolution

  1. Verify the viewTemplate configuration and ensure it includes the necessary filter to narrow down the data being retrieved.
  2. If it not includes is necessary to include them and run the update viewTemplate endpoint.
  3. Test the API call to confirm the resolution of the issue.
  4. Ensure that the call will get only the maximum of 1000 for Synchronous calls.
