Symptom

Same as note 3728657, when you are trying to process warranty IDocs using the function module IDOC_INPUT_WNTYIV_WTY10 in your S/4 system, you encounter a runtime dump. Due to this dump, the IDoc processing fails and the warranty claim cannot be processed through the IDoc.

Other Terms

Warranty management system, BAdI, comment, OBJECTS_OBJREF_NOT_ASSIGNED_NO, CX_SY_REF_IS_INITIAL

Reason and Prerequisites

Program error

Solution

Implement the correction instructions attached in this note and also in the note 3728657.
