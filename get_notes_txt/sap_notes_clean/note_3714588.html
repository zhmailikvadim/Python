Symptom

When you carry out the following actions to process dangerous goods items on logistics documents, the ABAP runtime error BEHAVIOR_STATEMENT_ILLEGAL occurs:

  • Delivery: You have created an outbound delivery. In the Pick Outbound Delivery app, you have entered a picking quantity and saved your document. When choosing Post GI, system responds with the runtime error.
  • Transportation Management: You have created a freight order and set the execution status to Ready for Transportation Execution. When saving the document, system responds with a runtime error.

Reason and Prerequisites

Reason: The generation function of dangerous goods information contains a program error.

Solution

After implementing this SAP Note, the system no longer displays a runtime error when you process dangerous goods items on logistics documents.
