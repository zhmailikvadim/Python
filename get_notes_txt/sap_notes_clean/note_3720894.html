Symptom

How to enable Tactical Sourcing with API flow in Guided Buying?

Environment

SAP Ariba Guided Buying

Resolution

To enable Tactical Sourcing with API flow, you must enable the following parameters in Guided Buying.

1.PARAM_ENABLE_ENHANCED_RFQ_INTEGRATION 

2.PARAM_ENABLE_EDIT_RFQ 

Steps to manage this parameter in GB is described below:

1. On the Guided Buying home page, choose Admin.

2. In the Parameters section, click Manage parameters.

3. In the Parameter settings table, click Change value next to the parameter you want to update.

4. Enter the value as true, and then click Change value.
