Symptom

You run the Collection of Personnel Cost Planning Data for Org. Objects (RHHCP_DC_ORGOBJECT) program and open the Data Collection Methods pop-up. However, there are two Data Collection Methods called "Infotype: Planned Compensation (1005)".

Other Terms

&PSE - PHCPDCOO

For more information, see SAP Note 1071282 - CONSULT: List of tags for Spain notes.

Reason and Prerequisites

Reason

The available Data Collection Methods are listed in the Method Implementations Personnel Cost Planning (V_T77HCP_INTF) table. The following entries have the same name:

Data Collection Method (V_T77HCP_INTF-INTERFACE) Name (V_T77HCP_INTF-IFTXT)
1005 Infotyp Sollbezahlung (1005)
6028 Infotyp Sollbezahlung (1005)

Note: As the original language of table V_T77HCP_INTF is German, the entry names are described in German.

Prerequisites

The following Support Package level is the minimum prerequisite:

Release Support Package
S4HCM 102 HR SP 00
S4HCM 101 HR SP 06
S4HCM 100 HR SP 19

For more information, access SAP ERP Human Capital Management - Support Package Schedules.

Note:

When you implement an SAP Note, some prerequisites might be required. The prerequisites are SAP Notes that have already been released and that change the same context as the current one. Then, even if these prerequisites are not necessarily related to the symptom of this SAP Note, you must implement them for technical reasons.

Solution

From now on, the system displays a unique name for the 6028 data collection method, allowing you to distinguish it from the 1005 method:

Data Collection Method (V_T77HCP_INTF-INTERFACE) Name (V_T77HCP_INTF-IFTXT)
6028 IT Sollbezahlung (1005) (f. PY-ES-PS)
