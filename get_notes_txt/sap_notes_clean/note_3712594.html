Symptom

When trying to access a Maintenance Order via the Change / Display Maintenance Order Apps, there is a 500 Internal Server Error (The ASSERT condition was violated)

Environment

  • SAP ERP Plant Maintenance (PM)
  • SAP S/4HANA, on-premise
  • SAP S/4HANA, Cloud Private Edition

Reproducing the Issue

Open a Maintenance Order via W0017 (Change or Display Maintenance Order)

Cause

Program Error

Resolution

Ensure the following SAP Notes are implemented:

  • 3454927 - F1511A - Dump when creating Maintenance Request with emergency context
  • 3641743 - Phase and Subphase Calculation for Planning Plant / Order Type / Overall Status Profile Combination
  • 3654999 - Report & Repair Malfunction status inclusion at operation level
  • 3667006 - [FIX]: Deletion of an Operation dumps when saving the Maintenance Order

Keywords

ASSERTION_FAILED, CL_EAM_OVERALL_STATUS_FACTORY=CP, GET_SESSION_INSTANCE
