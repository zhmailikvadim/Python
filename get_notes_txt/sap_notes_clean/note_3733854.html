Symptom

Scheduleline Buffer contains wrong dates for STO Substitute Schedulelines in Consolidation.

This is purely an internal, technical symptom that does not have any impact on the correctness, performance, or scope of the Advanced ATP functionality.

Other Terms

aATP; ABC; Consolidation

Reason and Prerequisites

Although this correction does not have any visible effect for a user, it contributes to the foundation required for providing and correcting functionality in aATP.

Solution

Apply the correction instructions for this note or upgrade to a SAP S/4HANA support package that contains this correction.
