Symptom

Device data is not available in BAdI during device installation which prevents maintaining device data during RLM installations with DIR creation.

Other Terms

APEU, 2508, DIR Installation.

Reason and Prerequisites

APEU, 2508

Solution

Please implement the attached the correction instruction.

A new method SET_DIR_AUTO_DATA has been added to the BAdI /APEU/BADI_MCM_MDO_AUTO_DATA to enable maintenance of device data. The method provides /APEU/S_MCM_DEVICE_DETAILS as an importing parameter and ISU07_AUTO_GERT as a changing parameter, allowing device data to be set.
