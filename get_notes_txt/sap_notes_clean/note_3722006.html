Symptom

  • HDLFS space is not visible in the Data Sharing Cockpit when attempting to create Custom Data Products
  • The storage type is filtered by default to SAP HANA Database (Disk and In-Memory) spaces, preventing the selection of HDLFS spaces.

Environment

SAP Datasphere

Reproducing the Issue

  1. Go to the "Data Product"
  2. Select "Artifact Space"
  3. Observe that is no possible to search for HDLFS space since it is automatically filter by "Storage type: SAP HANA Database (Disk and In-Memory)"

Cause

Resolution

Select the visibility option "Formations" in the Data Product profile, as described in Creating Data Products for SAP Business Data Cloud | SAP Help Portal
