Symptom

How to access the builder panel for a table in view mode in SAC?

Environment

SAP Analytic Cloud 2026.2.6

Resolution

  1. Login to SAC
  2. Open a story in edit mode
  3. Open the builder panel for a table 
  4. From table properties, check enable quick builder
  5. Save the story
  6. open story in view mode
  7. Click on more actions of a table
  8. Open quick builder
