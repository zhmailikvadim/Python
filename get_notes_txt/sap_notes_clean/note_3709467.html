Symptom

The system does not allow the creation of new custom sections within the Compensation Information portlet (or other Standard portlets) in the Legacy People Profile.
 

Environment

  • SAP SuccessFactors Employee Central

Reproducing the Issue

1. Navigate to Admin Center.
2. Go to Configure People Profile.
3. Select the Compensation Information portlet (or other Standard portlets).
4. Add custom fields as per business requirements.
5. Attempt to create a new section within the portlet to differentiate or organize the custom fields.
6. The system does not provide any option to add new custom sections in the Compensation portlet (or other Standard portlets).
 

Resolution

Currently, it is not possible to customize the compensation portlet (or other Standard portlets) by adding additional sections to separate the fields.

You can raise an enhancement request following the KBA 2090228 - How to Submit Enhancement Requests for SAP SuccessFactors Products
