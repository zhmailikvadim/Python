Symptom

Budget availability control (AVC) for Internal Orders is not triggered during asset-related procurement.

Environment

  • Controlling (CO)
  • SAP R/3
  • SAP R/3 Enterprise 
  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA Finance
  • SAP S/4HANA

Cause

The issue occurs when the required configuration for passing the Internal Order account assignment from asset procurement to Controlling is not properly maintained.

Additionally, if the CO/PP Order field is suppressed in the field status group, the Internal Order account assignment cannot be recorded during the posting.

Resolution

Check the following configuration settings.

  • Maintain G/L Account Setting

    1. Execute transaction FS00.

    2. Open the asset reconciliation account used for the asset posting.

    3. Activate the indicator Record account assignment.

  • Maintain Field Status Group

    1. Execute transaction OBC4.

    2. Identify the Field Status Group assigned to the reconciliation account.

    3. Adjust the field status for CO/PP Order from Suppress to Optional entry.
