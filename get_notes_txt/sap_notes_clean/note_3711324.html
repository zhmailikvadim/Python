Symptom

  • Customized employee information is truncated when exporting Org Chart.
  • Information from custom field on Org Chart Cards is truncated and not displayed completely when exporting the Org Chart.

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

SAP SuccessFactors HCM Suite

Reproducing the Issue

  1. Go to Admin Center > Org Chart Configuration > Company Org Chart.
  2. In the Choose the information shown on Org Chart cards drop-down list, select a field to display its information on cards. For example, Customizable Field 7 (custom07 field).
  3. Populate this field in the User Data File (UDF).
  4. Go to Org Chart.
  5. Search for a user who has this field populated.
  6. Hover the mouse above this user's card to view the complete information from custom07 field.
  7. Export the Org Chart to PDF format.
  8. Observe that if the information is too long, it is truncated, it is not completely displayed.

Resolution

The Engineering Team plans to redesign the PDF export in the 2H 2026 Release. In the meantime, it is recommended using shorter text.
