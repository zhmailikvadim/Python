Symptom

Incorrect constant 'BILL_PROCEDURE' used instead of 'BILLING_PROCEDURE' in /APEU/CL_MCM_CONFIG_ACCESS → GET_REGISTER_DATA method prevents BILLING_PROCEDURE from being recognized in the priority list and customizing table.

Other Terms

 APEU ,2508

Reason and Prerequisites

 Program Error , Register Data

Solution

The constant in class /APEU/CL_MCM_CONFIG_ACCESS was corrected from 'BILL_PROCEDURE' to 'BILLING_PROCEDURE', enabling the BILLING_PROCEDURE field to be properly recognized in the priority list and customizing table for meter register data reading.
