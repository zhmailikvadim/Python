Symptom

When posting asset retirement in T-code ABAVN, the cost center is not appearing in the gain & loss GL Line item.

Error message  KI235 "Account & requires an account assignment relevant to cost accounting" can be triggered.

Environment

  • Financial Accounting (FI)
  • Asset Accounting (FI-AA)
  • SAP S/4HANA Finance
  • SAP S/4HANA

Reproducing the Issue

  1. Run T-code ABAVN
  2. Insert all the parameters
  3. Simulate

Cause

The cost center associated with issued asset is blocked for revenues in T-code KS03, which prevents it from being populated in the GL line item during asset retirement.

Resolution

  1. Check the gain & loss account in FS03-> Go to the "Control Data" tab-> cost element category is defined as "11" (Revenue) 
  2. Check the cost center in asset master data via T-code AS03
  3. Verify the settings in T-code KS03 for the this cost center.  Navigate to the "Control" tab to confirm whether the cost center is blocked for revenue postings. 
  4. If there is a need to unblock the cost center to allow revenue postings, you may untick the indicator in T-code KS02, please consult with your local consultant team as this involves customization settings.
