Symptom

 An iframe embedded into Manage KPIs and Reports fails to load SAP Analytics Cloud content and displays an error message indicating that access to systems on the local network is blocked by the browser.

Other Terms

Google Chrome, Microsoft Edge, Local Network Access, LNA, Iframe, Service Worker, SAC, SAP Analytics Cloud, S/4HANA, Manage KPIs and Reports

Reason and Prerequisites

This issue is related to browser security changes regarding Local Network Access (LNA).

For a general description of the browser behavior and possible administrative mitigations, see the central SAP note 3683742.

Solution

This SAP Note provides a correction for SAP Analytics Cloud content viewed on the configuration tab in Manage KPIs and Reports application.
For issues with the SAP S/4HANA applications created using Manage KPIs and Reports that launch SAP Analytics Cloud content from Fiori Launchpad, SAP Note 3695649 needs to be implemented.

After implementing the correction, the behavior described above is resolved provided that the prerequisites and browser settings described in the central note are fulfilled.

For additional scenarios, limitations, and temporary workarounds, refer to SAP note 3683742.
