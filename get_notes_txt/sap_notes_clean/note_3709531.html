Symptom

In the Latest Applicant Workbench, the following areas have table headers or tab labels that are not readable due to low text contrast:

  • Manage Interviews -> Interviewers
  • Change History -> More -> Old Value / New Value

Environment

SAP SuccessFactors Recruiting Management

Reproducing the Issue

Issue 1:

  1. Open any application.
  2. Navigate to Audit Information > Change History.
  3. Click "More" when the displayed value is long.
  4. Observe that the "Old Value" and "New Value" headers are not readable unless they are highlighted.

Issue 2:

  1. Open any application with completed interviews.
  2. Open the action menu and select "Manage Interviews".
  3. Notice that the "Interviewers" tab is not readable until it is highlighted.

Cause

This is a Known Issue.

Resolution

The fix for this issue is planned to be deployed on 1H 2026 release.

For release timelines, please review SAP SuccessFactors Product Release & Road Map Information.
