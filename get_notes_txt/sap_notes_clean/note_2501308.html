Symptom

Images in BW BEx web reports are exported to PDF though the requirement is not to export the images to PDF.

Environment

  • SAP Business Warehouse 7.31 Support Pack 19

Reproducing the Issue

  1. Execute a BW BEx Web Application containing images.
  2. Click on Export to PDF button.

Resolution

  1. Go to transaction SE38.
  2. Enter the program name SAP_RSADMIN_MAINTAIN and execute.
  3. Then you will get "Maintenance of Table RSADMIN" where choose the radio button "INSERT" and enter the object "RSWR_EXPORT_INTERNAL_ITEM" and value "FALSE".
  4. Click on "Execute" button.
  5. Restart the Java server to see the changes in the portal.

Keywords

RSWR_EXPORT_INTERNAL_ITEM, RSADMIN, image, Export to PDF, BW BEx reports
