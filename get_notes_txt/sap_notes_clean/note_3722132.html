Symptom

When creating a publication in SAP Analytics Cloud, it is not possible to select the "Change Story Filter" option when either the CSV or XLS formats are selected.

Environment

  • SAP Analytics Cloud

Reproducing the Issue

  1. Create a publication from the report in SAP Analytics Cloud.
  2. Select either CSV or Excel format
  3. Observe that you are not able to select the "Change Story Filter" option as it is greyed out.

Cause

This behavior is by design.  As the tooltip which appears when you mouse over the "Change Story Filter" option says, this option is disabled when selecting a CSV style format.

Resolution

As a workaround, you can create a bookmark for each set of story filters that you want to create a publication on, and then select the bookmark which has the specific parameter value set for each iteration of the publication that you want to run.
