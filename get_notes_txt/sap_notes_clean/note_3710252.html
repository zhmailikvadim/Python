Symptom

  • Outbound file is failing immediately after the trigger file is dropped.
  • Error message: SAP DBTech JDBC: [1281]: wrong number or types of parameters in call: too many arguments

Environment

  • SAP Sales Performance Management
  • Incentive Management on Hyperscalers
  • Incentive Management on Legacy HANA 

Reproducing the Issue

  1. Set up a simple output file type and stored procedure.
  2. Drop a trigger file.
  3. Observe immediate failure of the process.

Cause

One possible cause is that the procedure was created without input parameters, so the trigger file must be empty; however, the file that was dropped has content.

Resolution

  1. Open the trigger file and confirm that it is empty.
       Sometimes the file may contain hidden characters.
  2. Delete the content or create a new empty file.
  3. Re-drop the new empty file.
