Symptom

When activating a Job Catalog Entry in an ABAP Cloud environment, the activation fails with an error indicating that the Job Catalog Entry references development objects that are not permitted in the ABAP for Cloud Development language version.

The error may occur even if the referenced objects are C1‑released, and are valid for consumption from standard ABAP environments.

Other Terms

CDS value help

Reason and Prerequisites

In this scenario, the activation fails due to an internal program error within the SAP framework.

Solution

Implement the correction instruction or install the corresponding support package that contains the fix.
