Symptom

When planning to S/4HANA 2025 in the MP, SAP FIORI FOR SAP CARAB 6.0 FPS02 is selected. But get below error message.
***************************
"ABAP Queue Check resulted in no valid queue. Result status 'Partial Delivery P'"
"There is no usable Inst./Upg. Package available for Add-On UICAR001 rel. 600"
***************************

Environment

Maintenance Planner

Reproducing the Issue

Add SAP FIORI FOR SAP CARAB 6.0 in the MP.

Cause

SAP FIORI FOR SAP CARAB 6.0 cannot be installed in SAP S/4HANA system. It can be installed in SAP S/4HANA Foundation based system.

Resolution

Please do not select SAP FIORI FOR SAP CARAB 6.0 when planning to SAP S/4HANA system in the MP.

For more information about upgrade/update troubleshooting, please refer to System Upgrade & Update Troubleshooting Wiki.
