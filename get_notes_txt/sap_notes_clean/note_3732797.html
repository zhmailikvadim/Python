Symptom

Symptom

During the execution of a process chain of a DTP, terminations occur sporadically:

"You are not authorized to use this transformation"
Message no. RSTRAN011

After you implement the corrections from this SAP Note, the system tries to trace the cause.

After you implement the corrections in the system, you must add the following entry to the table RSDHA_ADMIN:

RSDHA_ADMIN  
FIELD RSDHA_TRACE_EXCEPTION
VALUE CX_RSTRAN_NOT_AUTHORIZED

Afterwards, the table RSDHA_TRACE should contain entries of the following type, which can be used to restrict the cause under certain circumstances.
