Symptom

Unable to receive alerts from SAP Edge Integration Cell using SAP Alert Notification Services (ANS).

Environment

  • SAP Integration Suite
  • Edge Integration Cell
  • SAP Alert Notification service for SAP BTP

Resolution

To receive alerts from SAP Edge Integration Cell, only the SAP Alert Notification service needs to be enabled in the subaccount where SAP Integration Suite and Edge Integration Cell are being used. 

You can refer to the official SAP Help documentation for detailed information,

  • Alerting with SAP Alert Notification Service for SAP BTP | SAP Help Portal
  • Alerting | SAP Help Portal 

Keywords

prerequisites, SAP Alert Notification Service, alerts, Edge Integration, notification, Integration suite
