Symptom

Joule does not give any response and no errors are displayed when giving a command. 

Environment

  • SAP SuccessFactors HCM Suite
  • SAP SuccessFactors Employee Central

Reproducing the Issue

  1. Open Joule;
  2. Attempt to change something (e.g. Change “To Be Hired” Status of Position);
  3. No response and no error messages - nothing happens;

Cause

There are two templates configured with the same use case in Manage Data. 

Resolution

  1. Go to Manage Data > Employee Central Quick Action Template;
  2. Search for the template of the action tested in Joule;
  3. See there are more than one for the same use case type;
  4. Inactivate or delete one of them.
