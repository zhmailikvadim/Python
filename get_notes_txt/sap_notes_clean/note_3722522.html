Symptom

  • The system does not allow the selection of specific WBS elements during the creation of a sales order.
  • The error message displayed is: "This WBS element cannot be assigned."

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Attempt to create a sales order in the system.
  2. Try to assign a WBS element.
  3. Observe the error message: "This WBS element cannot be assigned."

Cause

The affected WBS elements currently do not have any revenue recognition key assigned.

Resolution

  1. Verify the revenue recognition key settings for the affected WBS elements.
  2. Assign the appropriate revenue recognition key to the WBS elements in the system.
