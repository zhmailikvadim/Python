Symptom

  • Cannot use "Upgrade to New Product Version" for VC products.
  • Version of the product remains outdated despite updating the effective date on the quote.

Environment

SAP CPQ connected with Variant Configuration

Reproducing the Issue

  1. Create a quote and add a VC product
  2. Create a new version of the product in the backend and sync it to CPQ
  3. Adjust the quote effective date to match the new version
  4. Upgrade to new Product Version not available, even though all settings are properly done on Setup.

Resolution

  • Upgrade to new Product Version is only supported on Quote 2.0
  • If you are using 2.0, configuration is properly done and it still not showing up the action, a configuration needs to be done by CPQ Support Team
  • Raise a case to component CEC-SAL-CPQ, mentioning the issue and granting approval to proceed with the change.
