Symptom

There is 0GB memory in the Quota of Cloud Foundry Spaces

Environment

Cloud Foundry

Cause

No Cloud Foundry Products have been purchased

Resolution

In order to work in the SAP Cloud Foundry environment, you need to have relevant products in your account. The main one is Application Runtime, which is basically memory and also other resources and applications. Currently, you have no SAP Cloud Foundry products purchased. Therefore, you cannot create an quota plans.

In order to purchase Cloud Foundry products, please contact your SAP Account Executive. They will take it from there.
