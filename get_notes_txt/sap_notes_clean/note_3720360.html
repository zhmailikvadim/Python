Symptom

  • Users who are provisioned with the Joule Embedded Entitlement (8018168) may receive an email notification from SAP regarding the product sunset of the associated Cloud Test, Demo, and Development environment.

    Sample Email:

    Dear Valued Partner,

    We value you as an active subscriber of the below cloud test and demo licensing service:
    cloud test, demo, and development for Joule embedded entitlement (8018168)

    SAP has undertaken the effort to simplify and evolve its portfolio. This effort results in the product sunset of the above service, therefore, your subscription for this product will not be renewed at the end of your subscription term.

    You can transition to successor offerings, which expand capabilities:  

    • 8020592 | cloud test, demo, and development for Joule Base

    • 8019670 | cloud test, demo, and development for SAP AI Unit

  • User would like to clarify whether the existing embedded Joule configurations and entitlements will automatically migrate to above service offerings, or if they need to purchase entitlements independently?

Environment

  • SAP BTP, Cloud Foundry runtime and environment
  • SAP Build Process Automation
  • Joule

Resolution

  • In Joule Studio, No components will be removed from SBPA as long as the tenant remains active. SAP does not anticipate any disruption to existing or deployed skills or Agents within the environment.

  • As a recommended best practice, user should export and backup all custom skills. Refer to the Export a Project SAP Help documentation for detailed instructions.

  • Existing embedded Joule configurations and entitlements will not be automatically transitioned to successor offerings like 8020592 | cloud test, demo, and development for Joule Base or 8019670 | cloud test, demo, and development for SAP AI Unit.

  • Users must procure the successor offerings separately. 

  • Once the service is provisioned in the subaccount, all necessary configuration and setup steps must be completed manually by the user.
