Symptom

  • SAML error encountered while attempting to access embedded analytics.
  • User encounters the below error

Environment

  • SAP Sales Performance Management
    • Incentive Management on Hyperscalers
    • Embedded Analytics
    • IAS

Reproducing the Issue

  1. Attempt to access embedded analytics through SAP Incentive management Portal.
  2. Encounter SAML error preventing access.

Cause

The issue is with the wrong certificate configured in IAS for Embedded Analytics 

Resolution

Review the below article to validate and correct configurations needed for the SSO setup:

  • Setting Up SAP SuccessFactors Incentive Management on SAP IdP | SAP Help Portal
