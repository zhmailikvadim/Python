Symptom

Typeahead search result shows wrong material number in WEBGUI MM03 or Display Material app.

The '/' character in typeahead result is displayed with '\' in front. For example, for material number 'TEST/01', typeahead search shows 'TEST\/01'.

The same works correctly in GUI MM03.

Environment

  • SAP S/4HANA, on-premise
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Open Display Material app.
  2. Enter material number which contains character '/'. For example 'TEST/01'.
  3. Typeahead search result displays material 'TEST\/01'.

Cause

Program error.

Resolution

Please check the solution in SAP note 3661542 - Fix issues with /, ' and & characters in typeahead result.

If you have any questions about kernel upgrade, please open a support case to BC-FES-ITS.

Keywords

Typeahead, search, wrong material number, WEBGUI, MM03, Display Material, character '/', special character
