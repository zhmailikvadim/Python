Symptom

  • The Cloud Transport System is being used for SAP Integration Suite, and there is a requirement to block users from performing manual imports of integration suite changes due to audit requirements for segregation of duties.
  • Concerns were raised regarding the enforcement of the Cloud Transport System not being fully configured to block manual imports.

Environment

  • SAP Integration Suite
  • Cloud Integrations

Resolution

  1. The roles within SAP Integration Suite do not allow permissions for export and import to be controlled separately. 
  2. Recommend submitting a feature request through the Influence Campaign for future enhancements to address this functionality. Influence Campaign: Influence Opportunity Homepage - Customer Influence (sap.com)  

Keywords

manual import, cloud transport system, sap integration suite, segregation of duties, developer roles, export permissions, import permissions, audit requirements, feature request, influence campaign, stop import
