Symptom

Even though a field of the generating unit list is to be ready for input for a specific energy type, it is locked for input. This affects, for example, the field REF_PRODUCTION (reference yield for wind turbines), which must be ready for input for generating units to which the energy type WINDON (Wind Onshore)  or WINDOFF (Wind Offshore)  is assigned, but it is not.

Other Terms

Transaction /ENBIL/GU02, transaction /ENBIL/GU_LIST_ATTR

Reason and Prerequisites

Program error
