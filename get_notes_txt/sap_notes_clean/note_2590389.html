Symptom

After upgrading SAP 3D Visual Enterprise Viewer 9.0 on an existing installation, there is a situation where ECTR cannot find the Viewer. ECTR shows a message, '“SAP 3D Visual Enterprise Viewer (amd64) was not found”. Viewer does not show in ECTR panel.

Other Terms

VEV, VEV 9.0 FP04, VEV9.0FP04

Reason and Prerequisites

Workaround for ECTR message after upgrade of SAP 3D Visual Enterprise Viewer 9.0.

Solution

Please follow these steps to recover from this situation:

  1. Double click the SAP 3D Visual Enterprise executable to initialise the Viewer install wizard.
  2. If checkbox for Viewer is unchecked then, you are free to install: check the box, click next until installation completes, then start ECTR. Viewer should now be available in ECTR.
  3. If checkbox is checked, then you will need to uninstall Viewer:
    1. uncheck the box, and click next until uninstallation completes.
    2. Close the install wizard & Go to Step1.
