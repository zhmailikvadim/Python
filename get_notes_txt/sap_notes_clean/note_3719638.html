Symptom

Health Savings Account (HSA) enrollments are not being inactivated due to error "Value is a required field/association in Object One-Time Information for Benefit Integration."

Environment

  • SAP SuccessFactors HCM Suite 
  • SAP SuccessFactors Employee Central

Reproducing the Issue

  1. Navigate to the Benefits Admin Overview
  2. Open the enrollment of Health Savings Account (HSA) for an user
  3. Insert a new record and change the effective status to "Inactive"
  4. Save the record and observe the error message: "Value is a required field/association in Object One-Time Information for Benefit Integration."

Cause

Resolution

Engineering team is planning to deliver the fix in the next release 1H2026. The deployment in preview will be in April 13th and in production in May 15th. 

If after the deployment you are still facing issues please contact support.
