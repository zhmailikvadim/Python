Symptom

Symptom

SAP Note 1540940 (handling of paid, required intermediate placements) describes how such employment is to be represented in the infotype "Social insurance" (0013).
If the employee is then insured in a farmers' health insurance scheme, you would have to enter a health insurance fund in the supplementary fund field in the infotype 0013 to which the contribution amounts are to be paid because the farmers' health insurance schemes do not participate in the compensation procedure according to the German Law for Compensation of Employer’s Expenses for Continued Pay (AAG).
However, the system issues the following error message:

  • Do not enter a supplementary fund

Other Terms

Social insurance, placement students, SI contribution, farmers' health insurance scheme

Reason and Prerequisites

There is a program error.

Solution

The changes are delivered in an HR Support Package. For the connection between the HR Support Package and the technical name specified in the SAP Note under "Support Packages", see SAP Note "Q&A: New HCM Legal Change delivery process (CLC)" (1232082).

Implement the attached correction instructions or import the specified Support Package.

The error message indicates that the field ATZHL of MARA_AUSP must not be initial.

Other Terms

MDC, CMP, Mass Processing, ATZHL, Classification, Characteristic Values

Reason and Prerequisites

Program error.

An initial value (0) for field ATZHL is allowed and must be supported.

Solution

Implement the correction instruction.
