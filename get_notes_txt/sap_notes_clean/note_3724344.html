Symptom

In the Actual Maintenance Costs app (F3567), the field label 'Cost Element Hierarchy' appears instead of 'G/L Account Hierarchy', although both refer to the same underlying data. The field 'Type of Technical Object' is displayed with inconsistent naming compared to other applications.

Other Terms

Actual Cost Analysis, Maintenance Order Costs, F3567, F4603, C_MAINTORDERACTUALCOSTQ, C_MAINTORDACTLPLNDCOSTQUERY, @EndUserText, label, quickinfo, TechnicalObjectType, GLAccountHierarchy

Reason and Prerequisites

Program error. The labels for these two fields were inconsistent across multiple applications.

Solution

Implement this SAP Note.

After implementing this SAP Note, the following two labels will change in the Fiori app Actual Maintenance Costs:

  • Cost Element Hierarchy -> G/L Account Hierarchy
  • Type of Technical Object -> Object Type

In the Fiori app Maintenance Order Costs (F4603) the quickinfo for the 'Object Type' changes to 'Type of Technical Object'.
