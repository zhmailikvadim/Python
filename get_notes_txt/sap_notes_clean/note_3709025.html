Symptom

When creating an in-house service as a follow-up to a returns order, the creation of the corresponding goods movement for the in-house service fails.
In the Returns Overview of the returns order, an incorrect output format for the material ID is displayed for the in-house service.

Other Terms

in-house repair

Reason and Prerequisites

The issue can occur if a material/product number template has been specified in Customizing for Logistics General under Material Master -> Basic Settings -> Define Output Format of Material Number.

Solution

Implement the SAP Note.
