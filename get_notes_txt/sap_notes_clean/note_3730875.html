Symptom

  • When you run T-code FAGLL03H, warning message FAGL_REPORTING_EN022 is raised.

    --FAGL_REPORTING_EN022 - BAdI method is obsolete >> see note 2489364. V_GLPOS_C_DEB_CT

Environment

  • SAP S/4HANA
  • SAP S/4HANA Finance

Reproducing the Issue

  1. Open T-code FAGLL03H;
  2. Enter the parameters in the selection screen and click Execute;
  3. Warning message FAGL_REPORTING_EN022 is raised.

Cause

The method ADD_FIELDS of BAdI FAGL_LIB is no longer supported, leading to an invalid-method message.

Resolution

  1. In the existing BAdI FAGL_LIB implementation used by FAGLL03H, remove or comment out all coding in method ADD_FIELDS.
  2. Run the FAGLL03H line item browser again and confirm the message no longer appears.
  3. To add fields to the line item browser in supported releases, refer to the guidance in SAP Note 2512883.

See Also

SAP note 2512883 - Financials Line Item Browsers: BADI FAGL_LIB - Update
