  • In the Data Services 14.2 SP10 releases, the job fails with a BAD SYSTEM MESSAGE instead. 

Other Terms

Keywords

DS, REST, C4C, ACCESS_VIOLATION, JSON2NRDM

Environment

  • Data Services 14.2.10 Patch 4 and higher
  • Windows and Linux job servers

Reproducing the Issue

  1. Create a job that invokes a REST Web Services function with q request/respond in the JSON data.
  2. Send two records to the Web Service function call.  Job will fail with the error listed in the symptoms section.

Reason and Prerequisites

The issue is caused due to a defect in Data Services for REST Web Services.

Solution

The correction is delivered in the release(s) listed in Support Packages & Patches section of this SAP Note.
