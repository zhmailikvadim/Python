Symptom

There's the requirement to enable the 'Header Billing Plan' indicator, but the field cannot be found in some scenarios, e.g.: when the item category used is CPHD.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Go to 'Manage Sales Orders - Version 2' (ID: F3893)
  2. Enter a sales order
  3. Enter one of the items
  4. The billing plan tab cannot be found, so the 'Header Billing Plan'.

For specific item categories, e.g. CBAO, billing plan tab and indicator are available.

Cause

It's not possible to enable the indicator 'Header Billing Plan' to item categories that don't use billing plan. If the item category doesn't use billing plan at all, the indicator for billing plan at header level cannot be set.

 

Resolution

For item categories relevant to billing plan, e.g. CBAO, the app 'Manage Sales Orders - Version 2' can be used to check the 'Header Billing Plan' indicator. This indicator can be changed, to on or off, while there are no billing dates created.
