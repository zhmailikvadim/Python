Symptom

The system does not run embedded process steps that you add to the extension process step if the extension process step contains a check group without any executable checks.

Other Terms

APE, Step Extension, Embedded Process

Reason and Prerequisites

When a check group has no checks or sub check groups that can be executed, the system does not return any result during the process run. As a result, the process step does not run the logic of the DO_ACTION section.

Solution

After implementing this SAP Note, the system correctly runs embedded process steps in extension process steps, even when the check group contains no checks that can run.
