Symptom

    • My Calendar view loads but does not display any entries.  
    • Team Calendar view remains blank.  
    • No error message is shown when the issue occurs.  
    • Refreshing the page does not resolve the problem.  
    • Users are unable to access scheduled activities or planned items via the Calendar view.  

    Environment

    • SAP Sales Cloud and SAP Service Cloud Version 2
    • SAP Sales Cloud Version 2
    • SAP Service Cloud Version 2
    • SAP Enterprise Service Management

    Reproducing the Issue

    1. Log in to the system.  
    2. Click on the Menu option.  
    3. Select the Calendar entity.  
    4. Observe that the Calendar view is empty, displaying no entries or data.

    Cause

    The Calendar view relies on Appointment and Visit business objects, which are included in the SAP Sales Cloud Version 2 and SAP Service Cloud Version 2 license scope.  
    In tenants licensed only for SAP Enterprise Service Management (ESM):  
    • Appointment and Visit objects are unavailable.  
    • Calendar functionality does not display any entries.   
    • The Calendar view remains blank as per design.  

    Resolution

    1. This behavior is expected due to the current license and scoping configuration.  
    2. To enable Calendar functionality:  
       a. Ensure the tenant has SAP Sales Cloud Version 2 or SAP Service Cloud Version 2 licensing.  
       b. Activate the relevant scoping elements (Appointments and/or Visits).  
    3. If Calendar functionality is needed, contact your SAP account representative to discuss licensing and scope activation options.
