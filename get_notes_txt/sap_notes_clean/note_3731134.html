Symptom

Information about transitioning to Version 2 and associated licensing requirements.

Environment

SAP Sales & Service Cloud

Cause

SAP C4C Version 2 is a new-generation product and not a direct technical upgrade of Version 1. Therefore, embedded AI capabilities are not available for retrofit into Version 1.

Resolution

  • SAP C4C Version 2 (SAP Sales Cloud Version 2 / SAP Service Cloud Version 2) is a strategic platform with embedded AI capabilities and ongoing innovation.
  • Transitioning from Version 1 to Version 2 requires a migration project, which includes:
    • System assessment
    • Configuration redesign (if required)
    • Data migration
  • This transition is not an in-place upgrade.
  • Licensing:
    • SAP C4C Version 2 requires separate licensing.
    • Customers should contact their SAP Account Executive to discuss commercial conversion or licensing options.
  • For additional details, refer to the relevant SAP Note. “SAP FOR ME”
