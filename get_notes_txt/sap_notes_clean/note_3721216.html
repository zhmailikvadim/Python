Symptom

While creating or updating a worker in Manage Workforce application, following error is encountered: 

  • Email is already used with Employee #WORKERID. Enter a unique email.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Open "Manage Workforce" app.
  2. Open the profile of the worker.
  3. Click on edit.
  4. Make any necessary change.
  5. Save the changes.

Cause

Email ID used by the worker is not unique.

Resolution

  • Email ID of every worker should be unique.
  • While updating or creating a new worker, assign a unique email ID.

Keywords

MANAGE WORKFORCE, EMAIL, CA-WFD-INT, CA-GTF-BUM, SAP_PC, SAP_1C, MDI, JB1, SOAP API
