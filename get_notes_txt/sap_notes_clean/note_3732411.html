Symptom

Unwanted occurrence of duplicate records in Manage In-House Bank Payment items Fiori app

Other Terms

IHB, Company Code, I_APMPaymentItemWithOrderInfo, duplicate records, Manage In-House Bank Payment items Fiori app, F6339

Reason and Prerequisites

Retrieving associated data using non‑key fields, which led to unintended record replication.

Solution

Changing the conditions for retrieving data via the association in I_IHBTransactionItemCmpst. Using a key field instead of a composite key.

Please apply SAP Note or assigned Support Package
