Symptom

  • User wants to integrate Joule AI with their SAP ERP system (SAP ECC) require clarification on whether Joule AI supports integration with SAP ECC. 
  • User are unable to identify any official documentation, configuration options, or product capabilities that indicate support for Joule AI integration with on‑premise ECC systems and are seeking guidance on supported system requirements and compatibility.

Environment

  • SAP BTP, Cloud Foundry runtime and environment
  • Joule

Reproducing the Issue

Cause

The Joule functionality is planned exclusively for cloud solutions, whether they are private (Edition 2025 and higher) or public cloud. User can check it here Introducing the Next Era of ABAP Development.

Resolution

  • Joule’s capabilities are currently supported only in SAP S/4HANA Cloud environments. Integration with on‑premise SAP ECC systems is not available at this time.
  • For an overview of SAP solutions that can be integrated with Joule, please refer to the documentation available on the SAP Help Portal under Integrating Your SAP Solution.
