Symptom

When executing a payroll Brazil report such as HBRCALC0, the selection screen fields are displayed incorrectly.

Environment

  • HR renewal 

  • SAP S/4HANA 

  • SAP ERP  

  • SAP SuccessFactors Employee Central Payroll 

Cause

The HR report class assigned in the program attributes of the underlying standard reports is not the SAP standard.

Resolution

  1. In transaction SE38, open the executable program behind each affected transaction.
  2. Choose Program > Attributes and locate the HR report category setting, in the following path: SE38 -> HBRCALC0 -> Características -> Classe de report HR -> Dds. mestre (infotipos) .
  3. Set the HR report category to the SAP standard value. For example, for report HBRCALC0, the standard value is " __M00001".
  4. Save and activate the changes.
  5. Repeat for all affected reports and retest each transaction’s selection screen.
