Symptom

  • Column profile task fails after S/4HANA is upgraded from 2022 to 2023.
  • [E] 2025-02-10 15:11:17.589 [TaskUtility][pool-6-thread-1] java.lang.NullPointerException: while trying to invoke the method java.lang.String.length() of a null object loaded from local variable 'str'

Other Terms

Keywords

IS, S/4HANA upgrade, profile task, NullPointerException, 'str'

Reason and Prerequisites

Environment

  • Information Steward version 14.3.3.263
  • SAP S/4HANA 2023 FPS01

Reproducing the Issue

  1. Upgrade S/4HANA from 2022 to 2023
  2. Create a new IS connection with S/4HANA 2023
  3. Import tables from the new S/4HANA connection into IS -> DI
  4. Create a column profile task with the imported table.
  5. Run task fails.

Solution

This issue arises only after updating the connection server details through the ConnectionUpdateUtility available in the SAP KBA: 2592327 - How to change a connection in Linux - Information Steward 4.x
This issue has been fixed and available with SAP KBA: 2592327 with lastest version of ConnectionUpdateUtilility.
