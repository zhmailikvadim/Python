Symptom

LDC reversal from the Document Flow WorkCentre is not possible. The system indicates that the settlement must be invoice-canceled to allow LDC reversal.

Environment

SAP Agricultural Contract Management

Reproducing the Issue

  1. Navigate to the Document Flow WorkCentre (/ACCGO/DFW_WC - Agricultural Contract Management -> Tools -> Document Flow WorkCentre).
  2. Attempt to reverse LDC using the established workflow steps.

Cause

The invoice document was reversed but not canceled, leaving the settlement status as “Invoiced,” which prevents LDC reversal.

Resolution

  1. Cancel the invoice as part of the settlement reversal process in settlement work center (/ACCGO/STL_WC - Agricultural Contract Management -> Contract Settlement -> Settlement WorkCentre).
  2. After canceling the invoice, perform the application reversal.
  3. Once the application is reversed, execute the LDC reversal.
