Symptom

Runtime error "ASSERTION_FAILED" is thrown when transferring a cross-company code CO posting in a CFIN target system. The CO posting contains more than one transactions that are posted in different transaction currencies.

Other Terms

CL_FINS_ACDOC_POSTING_EVENTS, SET_GLACCOUNT_TYPE_AND_XOPVW, CL_FINS_ACDOC_INTF_CHECK, CREATE_CLR_ITEM_UNIQNE_SIGN, CFIN, BUV, BSCHL, KOART, KUNNR, LIFNR, SHKZG

Reason and Prerequisites

 Program error

Solution

Please implement the attached correction instruction in your CFIN target system.
