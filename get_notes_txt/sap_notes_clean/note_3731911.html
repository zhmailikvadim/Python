Symptom

This article provides the correct flow to Cancel Prolonged I9 Reverifications.

Environment

SAP SuccessFactors Onboarding

Resolution

  1. Start the Pre‑Migration Report: Begin by initiating the Pre‑Migration Report for the affected employee records. This report will determine whether there are any records that require migration.
  2. Verify whether records need migration. After generating the report, review the results to confirm if any records are listed for migration: If no records appear no further migration steps are required. If records do appear proceed to the next step.
  3. Submit the Migration Request: As per the standard process, you must submit the migration request first. Once the migration is submitted, the system will generate the first migration result.
  4. After the initial migration result is generated select the option: "Cancel prolonged I‑9 reverification processes" 
  5. After the prolonged reverification processes are cancelled, you will be able to complete the remaining migration steps without interruption.
