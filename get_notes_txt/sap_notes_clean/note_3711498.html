Symptom

Initiate Onboarding button displays inconsistent labels in the detail page and list page of the Redesigned Workbench.

***Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.***

Environment

SAP SuccessFactors Recruiting Management

Reproducing the Issue

  1. Navigate to the job application page, do initiate onboarding action for a candidate
  2. Observe that the onboarding process fails in the Onboarding Module
  3. Switch back to the job application page. In the job application details page, the initiate onboarding button label displays as "Initiate Onboarding"
  4. In the job application list page, the label displays as "Reinitiate Onboarding"

Resolution

This issue will be resolved in the 1H 2026 version.

The new version will be released on the following dates:
-Apr.13 for Preview instance
-May 15 for Production instance

RCM-158700, INC19951747, initiate onboarding, application pipeline, UXR
