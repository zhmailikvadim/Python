Symptom

The BW target Datastore can't import “Master Transfer Structures” or “Transaction Transfer Structures” from BW/4 HANA server.

Environment

SAP BusinessObjects Data Services XI4.2 SP14

Reproducing the Issue

  1. Create a BW target Datastore DS1 to access to BW/4 HANA server.
  2. Double click “Master Transfer Structures” or “Transaction Transfer Structures”, RFC error "<RFC_ABAP_MESSAGE, Action cancelled. Not supported in SAP BW/4HANA.)- The current application has intentionally triggered a termination with a short dump>" occurs.

Cause

The BW target datastore may have multi configurations, and it doesn’t know whether the current BW system is a BW/4HANA or a classic BW. Therefore, transfer structures folders won’t work for BW/4 HANA and BW4 datastore objects won’t work for classic BW.

Resolution

This is a product limitations by design.

Keywords
