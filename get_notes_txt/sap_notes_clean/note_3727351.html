Symptom

  • Error message: "No update performed due to an error."
  • Detailed error message: "Unit MV_ATTR1 is not defined for activity type MV_ATTR2, please use MV_ATTR3" (Message no. FCO_COST_RATE007).

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Perform the formal execution of the Actual Cost Rate Calculation.
  2. Click the 'Save' button for actual cost rate calculation.
  3. Observe the error message: "No update performed due to an error."

Cause

The activity type unit was changed, however, there's control record already exist for the combination of cost center/activity type/activity unit year dependent.
The change of activity type unit cause different activity unit data in different year.

Resolution

To avoid such error, SAP Support can help check if there's such inconsistent data exist and help remove such control record to allow the actual cost rate calculation.
The precondition to correct the data is to ensure 'No cost rate record exist and No actual posting exist for the cost center/activity type combination'.
