Symptom

You are trying to use Joule Capabilities for Sales after implement all the mandatory notes listed in 3459573 and have the all the mandatory Business Roles assigned to the SAP user:

  • SAP_BR_INTERNAL_SALES_REP
  • SAP_BR_ANALYTICS_SPECIALIST
  • SAP_BR_BILLING_CLERK
  • SAP_BC_WEBSERVICE_CONSUMER

After that, it still not working and you are enrolled on RISE program (PCE Rise Customer contract).

Other Terms

API_SALESDOCUMENTS_AI_SRV, API_SALES_DOCUMENT_AI, Business Function, PRC_GENERIC_CONF, ODATA_SD_SDOC_AI, CL_API_SDOC_AI_DPC_EXT, PRC_D_AI_SCOPE, PCE_GENERAL_AI_SCOPE, AI, GPT, Joule, Sales and Distribution, SD, API, ISLM, Generative AI, genAI, SAP S/4HANA, ISLM, Intelligent Scenario, SD_INT_AI_GPT, SD_INT_AI_GEMINI.

Reason and Prerequisites

You need to perform all the mandatory steps listed on this note's solution.

Solution

  1. Go to SFW5 transaction (KBA 1641394 can be used as reference, the SAP user needs specific authorization)

       1.1 - Search for the Business Function "PRC_GENERIC_CONF" (delivered by the SAP Note 3437407)
       1.2 - Enable the checkbox on the column "Planned Status"
       1.3 - Click on the "Activate Changes" button on the toolbar
       1.4 - There is no impact on enabling this BF (SWF5 -> Impact Analyzer). It delivers a feature switch and it's a reversible Business Function meaning the S/4HANA doesn't need to be restarted. This step also needs to be done on the system that transaction SCC4 shows as "Customizing"

  2. Go to SCPR20 transaction and activate the BC Set "PRC_GENERIC_CONF" (Business Configuration Sets)

  3. Go to ISLM_CONN_MAP transaction to do the ISLM Backend Setup

    3.1 - Click New and enter one of the SD Intelligent Scenarios you want to setup, like "SD_INT_AI_GPT"
    3.2 - Enter the OAuth 2.0 Client Profile and the AI Core Service Key from your BTP account
    3.3 - Make sure you click the Connection Check button before Save
    3.4 - The ISLM will trigger the activation of this Intelligent Scenario and the next step will be to activate the scenario using the Fiori Launchpad and the ISLM apps.This task will take approximately 20 minutes to complete

  4. Proceed to the ILSM setup on the Frontend (Fiori Launchpad):

    4.1 - Open the Intelligent Scenario Management (Fiori App F4470)
    4.2 - Search for the ODATA_SD_SDOC_AI package
    4.3 - Click on one of the Intelligent Scenarios displayed
    4.4 - Select the Training and click on Deploy
      - Some systems require the user to enter the modelVersion, you can enter latest and this will ensure that the setup will use the last available modelVersion for a specific LLM Model
      - Example: modelName: gpt-4o
                       modelVersion: latest
      - In case latest is not accepted, you need to pick the latest version from the SAP Note 3437766
      - Once the Deployment is done, use the button Activate to activate the deployment For All users in the system. This might take 30 minutes.
