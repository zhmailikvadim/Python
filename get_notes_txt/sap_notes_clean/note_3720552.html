Symptom

When you run the Reference List GBA Section 14 Loans, the system doesn't evaluate loans where the parameter 'Position Grouping According to Paragraph 14 GBA Open Loan Commitments' contains a value.

Other Terms

FNK0, 32

Reason and Prerequisites

Programming error

Solution

After implementing this SAP Note, the system correctly evaluates loans where the parameter 'Position Grouping According to Paragraph 14 GBA Open Loan Commitments' contains a value when you run the Reference List GBA Section 14 Loans. Implement this SAP Note with transaction code SNOTE.
