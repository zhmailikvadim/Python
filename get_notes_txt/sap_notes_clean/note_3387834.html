Symptom

Time out dump occurs when remodeling request of ADSO gets stuck in yellow state.

Category               ABAP programming error
Runtime Errors         TIME_OUT
ABAP Program           CL_SQL_STATEMENT==============CP
Application Component  BC-DB-DBI

Environment

SAP BW/4HANA 2.0

Reproducing the Issue

  1. T-code RSMONITOR or report RSCNV_MONITOR;
  2. Search for the USER *;
  3. Select the affected ADSO;
  4. Hit Start Request button.

Cause

Program error: when executing a remodeling rule based on Advanced Data Store Objects (ADSO), it may result into TIME_OUT dump and the status of the remodeling rule persisting as "still running".

Resolution

Implement the following note using transaction SNOTE:

3387044 - TIME_OUT dump occurred when executing remodeling rule

Keywords

remodeling, time_out dump, time out,  CL_SQL_STATEMENT==============CP,  CL_SQL_STATEMENT, adso, yellow, state, yellow state, stuck, still running, CL_RSCNV_MANAGER
