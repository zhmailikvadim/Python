Symptom

  • Error 'Property Id is mandatory' while creating IT584 Subtype 0001

  • Message no. HRPADIN01525 

  • Make Property Identifier (Property ID) field as non mandatory or hide

Environment

  • Human Capital Management (HCM)
  • Payroll India
  • SAP R/3
  • SAP ERP Core Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA

Reproducing the Issue

Following Step 5 (HOW TO HIDE PROPERTY IDENTIFIER FIELD - Hide from Backend (R/3) PA30) as per

KBA 3559089 - Property Identifier in ESS Payroll India IT Declaration Application

Error "Property Id is mandatory" is thrown while maintaining Income From Other Sources for House Property :

     

    Cause

    If you are not activating PRLIN (Property ID Linkage) i.e. value 0, the field property ID is hidden. And when INSLF = 1 is maintained, property ID field is technically empty as its hidden, and we get the property ID is mandatory error.

    Resolution

    1. Check if PRLIN is set to 0 as per Step 5 (HOW TO HIDE PROPERTY IDENTIFIER FIELD) from KBA 3559089 - Property Identifier in ESS Payroll India IT Declaration Application.
    2. Check the value of constant INSLF (V_T511K in SM30).
    3. If the value of this constant is set to 1, Change it to 0.
    4. Save.
    5. Reproduce the issue again.

    Solution Section

    NOTE- You are advised to consult your legal counsel before applying these changes in your system.

    As a part of the solution the following objects have been changed/introduced:

    • Change in constant - Income from house property (INHPR): Effective from 01.04.2025, this constant should be maintained as 2.
    • New constant - Two Self Occupied Properties (INSLF): This constant is used to determine the number of self-occupied properties an employee can declare. It has below two values:
      • 1 - Maximum of two self-occupied properties will be allowed
      • 0 - One or more self-occupied properties will be allowed based on the values configured for Income from house property (INHPR).

    More details, go through the above SAP Note.
