Symptom

This is the release note for Agentry SDK 7.3.0.18

Other Terms

Agentry

Reason and Prerequisites

SAP is pleased to announce the latest release of Agentry SDK 7.3.0.18

Solution

Agentry SDK 7.3.0.18 resolves defects and security related issues.

This release of the SDK is compatible with Agentry Cloud Server (ACE).

Android Details:
The Agentry Android client is compatible with Android 10 through Android 15 and has been validated against Android 15 and tested against Android 16.
For developers extending the Agentry Android client with OpenUI extensions, the Android targetSdkVersion and compileSdkVersion are set to API level 35 (Android 15).
Starting with Agentry SDK 7.3.0.4, the Agentry Android client has migrated to the AndroidX library and no longer uses the deprecated Android Support Library.  Developers writing their own custom OpenUI extensions must also migrate those extensions to the AndroidX libraries. This also allows OpenUI extensions to use 3rd party libraries that require AndroidX libraries.
Developers extending the Agentry Android client or building a branded Agentry client release are recomended to use Android Studio Ladybug | 2024.2.2.
Starting with the Agentry SDK 7.3.0.7 release, the Android .apk file is signed with SAP's official signing certificate. Users of Agentry SDK 7.3.0.6 or earlier will have to first uninstall the Agentry Client prior to installing 7.3.0.7 due to the change in signing certificates. Attempting to upgrade from an earlier version to 7.3.0.7 will fail with the following error message:  "App not installed as package conflicts with an existing package". Customers that brand their own Agentry client release using the Agentry SDK will not be affected.

iOS Details:
The Agentry iOS client has been validated against iOS 26 and iPad iOS 26.
It is compatible with iOS 17, iOS 18, iPad OS 17, and iPad OS 18.
Agentry SDK 7.3.0.18 requires Xcode 26 for Agentry client branding and Agentry OpenUI extension development. Xcode 16 or earlier is no longer supported.

Windows Details:
The Agentry WPF client is compatible with Windows 10 and Windows 11.
