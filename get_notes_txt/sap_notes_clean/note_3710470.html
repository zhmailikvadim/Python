Symptom

When processing a CHAPS PACS 008 out , determination of ISO XML was wrong when payment scope is customer.

Other Terms

SWIFT MX, PACS008 OUT, InstdAgt, InstgAgt.

Reason and Prerequisites

While mapping Swift MX PACS008 out few tags were missing due to wrong ISO XML being picked at run time.

Solution

Implement the correction instruction attached to this SAP Note. Alternatively, you can import the referenced Support Package.
