Symptom

Resolution

Issue

When checking out a requisition in guided buying the approval flow preview shows the following error:

Approval process was not successfully determined. Please retry.

Solution

Change the unsupported direct link for the External Site landing page tile to one of the supported web jumper links in SAP Ariba Buying solutions.

To change the link in guided buying:

  1. Go to the guided buying Admin page.
  2. In the Landing pages section, click Manage using UI.
  3. Select the Purchasing unit where the External Site tile is configured.
  4. Edit the tile.
  5. In the Target Url field, change the direct link to a supported web jumper link.
  6. Click Save.
Cause

A new user session is created when accessing a guided buying External Site landing page tile (Action tile type in the JSON landing page file) that is configured with an unsupported direct link to SAP Ariba Buying solutions, such as the following target URL examples:

  • http://[customer_site_name].procurement.ariba.com
  • https://s1.ariba.com/Buyer/Main/?realm=[customer_site_name]

External Site landing page tiles always open the target URL in a new tab in your browser. When returning to the original browser tab with guided buying opened and navigating to the checkout page for a requisition, the approval flow lookup action doesn't find the previous session, leading to the error.
