Symptom

With this SAP Note, the Personal Calendar Generation (RPTGENB0) report uses this field in combination with the customizing in "Define Absence Category Attributes" (T5BP8) for parameter "Function module to perform exceptions/custom processing" (T5BP8-EXFUN). In the standard delivery this parameter is set to function module HR_BE_ABS_EXCEPTIONS_ILLNESS for illnesses. The evaluation of KENN2 is only used in this function module.

The counting of the first year of incapacity is interrupted when there is a gap of at least 14 calendar days between two adjacent absence periods (illness or professional accident). Otherwise the counting is not interrupted and continues.

You can use this field to force the reset of the counter of the first year of incapacity. When you add a value to this field, the counting starts again at the begin date of this absence period. You can use an arbitrary value for that field.

Other Terms

KENN2, Invalidity, RPTGENB0, LHRPADBE_ABSENCESF14, HR_BE_ABS_EXCEPTIONS_ILLNESS

Reason and Prerequisites

This SAP Note is relevant only for Belgium.
