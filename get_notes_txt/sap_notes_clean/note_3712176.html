Symptom

When you cancel an electronic invoice in Malaysia, the system displays error message "Cancellation failed due to the time window for cancellation was closed.", that does not provide sufficient detail about the cancellation failure.

After implementing this SAP Note, the system displays detailed information about cancellation failures in the application log.
