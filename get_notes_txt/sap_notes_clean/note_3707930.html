Symptom

SAP Solution Tools Plug-In (ST-PI) contains a function module that does not perform the necessary authorization checks for authenticated users, allowing system information to be disclosed. This vulnerability has a low impact on confidentiality and does not affect integrity or availability.

Other Terms

Access control, Authorization profile, CVE-2026-24313

Reason and Prerequisites

Missing authorization checks in the SAP Solution Tools Plug-In (ST-PI) allowed authenticated users to access certain functions that exposed system information.

Solution

Necessary authorization checks were implemented in the affected Function Module so no sensitive information is disclosed.

Please Implement the Correction Instructions or Support Packages referenced by this SAP Security Note.
