Symptom

When the payment plan amount is changed, the service updates payment plan items for which invoices have already been generated, which is not the expected behavior.

Other Terms

SAP Service Cloud Version 2, SAP C4C, Cloud for Cusotmer, Payment Plan, Payment Plan Edit, ISU_C4C_V2_PAYMENT_PLAN

Solution

The code corrections are delivered via Automatic Code Correction. After implementing this SAP Note, please follow the steps described in the SAP Help documentation to adjust the service definitions.

If the external view of the service definition displays fields and structures in capital letters, delete the service definition and recreate it. The SAP Help documentation provides the necessary steps for both adjusting and recreating the service definition.

Affected Service Definitions:

ISU_C4C_V2_PAYMENT_PLAN
