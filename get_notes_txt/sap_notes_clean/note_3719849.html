Symptom

Short Dump : ITAB_DUPLICATE_KEY encountered during change the production version (PV) in transaction ME12.

Category: ABAP programming error.

Runtime Errors: ITAB_DUPLICATE_KEY.

ABAP Program: SAPLCURTOMDCH.

Application Component: SCM-APO-INT-MD-PDS.

Environment

  • SAP S/4HANA

Reproducing the Issue

  1. Open transaction ME12 to access the Purchasing Info Record (vendor–material) with the info category set to subcontracting. 
  2. Modify the production version (PV) and save the changes. 
  3. Navigate to transaction ST22 to verify if the error ITAB_DUPLICATE_KEY is present. 

Cause

  • The program did not consider PDS types in a SELECT statement.

Resolution

Refer to SAP Note 3460253 - Issue updating PIRs via ME12.

See Also

  • Refer to SAP Note 3485134 - Report: Deletes duplicates from CURTO_PDS_CHG which have a processed pair (same MATNR, LIFNR).
