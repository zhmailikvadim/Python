Symptom

You run Payroll Driver for an employee who has sickness after leaving. The following exception has been raised:

Category ABAP programming error
Runtime Errors COMPUTE_FLOAT_ZERODIVIDE
Except. CX_SY_ZERODIVIDE
ABAP Program CL_HRPAYRU_ADJ_SICKNESS_CASE==CP

Short Text

Division by 0 (type F) in program "CL_HRPAYRU_ADJ_SICKNESS_CASE==CP"

Error analysis
An exception has occurred which is explained in more detail below. The exception is assigned to class 'CX_SY_ZERODIVIDE' and was not caught in procedure "PROCESS_SFSS" "(METHOD)", nor was it propagated by a RAISING clause.
Since the caller of the procedure could not have anticipated this exception, the current program was terminated.
The reason for the exception is:
In the current program "CL_HRPAYRU_ADJ_SICKNESS_CASE==CP", an arithmetic operation ('DIVIDE', '/','DIV' or 'MOD') attempted to use operands of type F to divide by 0.

Other Terms

PC00_M33_CALC, HRUCALC0, HRUCALC0_CE

Solution

Import the Support Package specified for your release or implement the attached correction instruction.

As a general rule SAP recommends that you implement the Support Package specified for your release. However, if you need to install the solution earlier, use the Note Assistant to carry out automatic correction instructions. You can find more information about the SAP Note Assistant in SAP Support Portal, under SAP Note Assistant.
