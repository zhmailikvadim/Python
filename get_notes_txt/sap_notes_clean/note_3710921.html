Symptom

  • Unable to find the S/4hana Private Cloud option in the Step 3 - "Select Integration" stage dropdown during activation of Joule Booster.



  • In global account Systems, S/4HANA Private Cloud system is already auto discovered.

Environment

SAP Business Technology Platform

Reproducing the Issue

  1. Run the Joule Booster to integrate S/4HANA Private Cloud system with Joule.
  2. Navigate to the Step 3 - "Select Integration".
  3. Attempt to locate the S/4HANA Private Cloud option in the dropdown menu.
  4. Observe that the S/4HANA Private Cloud option is not available.

Cause

In Joule Booster, Step 3 - "Select Integration" aims to enable Joule Icon in the selected system, Step 4 - "Select Capabilities" aims to enable capabilities of the selected system in Joule tenant.

For S/4HANA Private Cloud, icon enablement is not done via the booster, so it will not be visible the Step 3 - "Select Integrations".

Resolution

Ensure that it is visible in Step 4 - "Select Capabilities", tick "S/4HANA Private Cloud" system at Step 4, and proceed to complete the running of Joule booster.

After running the booster, follow the post booster steps as stated in the documentation for icon enablement: Integration with SAP S/4HANA Cloud Private Edition | SAP Help Portal.
