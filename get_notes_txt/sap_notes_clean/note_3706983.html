Symptom

Shot dump GETWA_NOT_ASSIGNED occurs when a user with restricted activity list accesses Set Up Teams Application in Payroll Activities.

  • Category ABAP programming error
  • Runtime Errors GETWA_NOT_ASSIGNED 
  • ABAP: Program CL_PYC_TMS_CONFIG_FACADE======CP
  • Short Text: Field symbol has not been assigned yet.

Other Terms

Payroll Control Center, PCC, Payroll Activities, Activity Lists, Set Up Teams, P_PYT_TM, GETWA_NOT_ASSIGNED, CL_PYC_TMS_CONFIG_FACADE======CP

Reason and Prerequisites

This is a program error.

Solution

Import the related Support Package or implement the attached correction instruction.

 

Other Terms

IDXGC, /IDXGC/CL_PROCESS_DOCUMENT, BUILD_PDOC_FROM_ARCHIVE_DATA, BAL_GLB_MEMORY_REFRESH, /IDXMM/DOC_DISPLAY.

Reason and Prerequisites

In method BUILD_PDOC_FROM_ARCHIVE_DATA of class /IDXGC/CL_PROCESS_DOCUMENT, FM BAL_GLB_MEMORY_REFRESH is called to refresh the business application log completely to make sure the archived BAL will be loaded correctly. But it impacts the current session BAL, which can't be accessed anymore and causes the short dump.

Solution

Please implement the corrections of this SAP Note or import relevant Support Package.

This SAP Note is relevant for the following roles: DSO
