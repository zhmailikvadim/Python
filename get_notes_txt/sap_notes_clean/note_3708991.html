Symptom

If the following conditions were met for texts of EAM Work Order Header, EAM Work Order Operation or EAM Work Order Suboperation:

  • The Short Text (i.e., Description) was not empty.
  • The Short Text did not match the second line of the Long Text. In other words, the Short Text did not match the second line of the Long Text returned by FM READ_TEXT.

During requests handled by the EAM WO oMDO handlers that resulted in changes to the Long Text, the Short Text was copied into the corresponding Long Text as the second line.

Other Terms

S4MOBADDON-8096, SAP Note 3706856, /MERP/CL_PM_EAM_WORKORDER_OD, /MERP/CL_PM_EAM_MOBILE_STS_OD, /MERP/CL_PM_EAM_WO_TRANSFER_OD, EAM Work Order long text, EAM WO short text is copied to long text, EAM WO description is copied to long text

Reason and Prerequisites

 Programming error.

Solution

Apply the attached correction instructions.
