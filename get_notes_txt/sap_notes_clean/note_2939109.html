Symptom

Unified rendering for Web Dynpro ABAP must be updated in line with error corrections.
These corrections solve the errors described with SAP Note '2934230 - Cumulative patch for Unified Rendering 2007'.

Other Terms

Unified Rendering (UR), WD ABAP, Web Dynpro, Rendering.

Reason and Prerequisites

This problem is caused by a program error.

SAP Note 2187425  describes how to prepare your system for the use of TCI(Transport based Correction Instructions) and how this SAP Note can be used in the SAP Notes Assistant ( transaction SNOTE ).

The TCI  technology is mandatory in release SAP_UI 753 for updating Unified Rendering. 

For more information about the UR Update Process please read SAP Note 2090746.

Solution

The related TCI Package contains corrections for unified rendering from SAPK-75300INSAPUI to SAPK-75307INSAPUI with the following change list:

urchangelist 326347
jschangelist 326346
urversion 10.30.7.326347.0
Code line/UR Patch UnifiedR_03_REL/2007
Spam level needed check note 2187425
Deinstall Allowed
TCI Created 15.07.2020

Also refer to the comments in SAP Note 1411980.
