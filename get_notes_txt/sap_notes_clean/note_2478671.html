Symptom

Push deployment from contract packager does not create a stock transfer.

The corresponding application log contains the following messages:
Warning: Push deployment from inactive contract packager (contract packager name).
Total available quantity sent to location (master facility name).

This occurs when the master facility is a consolidated location.

Other Terms

SPP
/SAPAPO/DEPL_MESS175
/SAPAPO/DEPL_MESS176

Reason and Prerequisites

Program error
