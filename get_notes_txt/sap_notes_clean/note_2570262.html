Symptom

While creating a new BPF template, the target context model cannot be filled during the creation of new action where the selected target action is "Work Status".

Environment

SAP Business Planning and Consolidation 11.0, version for SAP BW/4HANA

Reproducing the Issue

  1. Create a new BPF template
  2. Create new activity and create a new action
  3. Press button “Create” in “Performer Definition” to define the activity itself.
  4. Press button Add Hyperlinks
  5. Select Target action as "Work Status" and then "Set Work Status"

--> The model list is empty

Cause

The work status is not enabled for the affected model.

Resolution

Enable the work status in Administration --> Features --> Work Status Configurations.
