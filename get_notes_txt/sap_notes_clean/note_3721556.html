Symptom

In the Manage Asset Retirement Obligations Fiori application, the value help for the Interest Rate Key field (int_rate_key) does not function correctly.

Users may experience one of the following behaviors:

  • The value help button is not displayed at all
  • The value help button appears but clicking it shows the message: "Right now, there are no visible columns in the table. Please select the columns you need in the table settings."
  • The value help dialog opens but displays no data

This issue occurs in the following areas of the application:

  • Obligation Header (General Data section)
  • Accounting Principle-specific settings

The value help for Inflation Rate Key (inf_rate_key) works correctly.

Other Terms

int_rate_key, /FOM/C_OblHeader, /FOM/C_OblAP, /FOM/C_VH_INTEREST_RATE, /FOM/WORKBENCH_SRV, Asset Retirement Obligation, ARO, Interest Rate Key, Value Help, F4 Help, OData, CDS Annotation, @Consumption.valueHelpDefinition

Reason and Prerequisites

The issue is caused by an invalid character (trailing space) in the @Consumption.valueHelpDefinition annotation in the following CDS views:

  • /FOM/C_OblHeader (lines 79-81)
  • /FOM/C_OblAP (lines 46-48)

The annotation incorrectly references the value help entity with a trailing space:

@Consumption.valueHelpDefinition: [ { entity: { name: '/FOM/C_VH_INTEREST_RATE ', element: 'id' } } ]

This trailing space after /FOM/C_VH_INTEREST_RATE causes the OData metadata generation to fail, resulting in the Common.ValueList annotation not being created for the int_rate_key field.

The issue manifests in SAP S/4HANA 1909 and later releases due to stricter annotation processing in the OData framework. Systems running SAP S/4HANA 1809 are not affected.

Solution

As a general rule, SAP recommends that you implement the Support Package specified for your release. However, if you need to install a solution earlier, use the SAP Note Assistant to implement the correction instruction. You can find more information about the SAP Note Assistant in SAP Support Portal, under SAP Note Assistant.

After successfull implementation, clear the OData metadata cache using transaction /n/IWFND/CACHE_CLEANUP.
