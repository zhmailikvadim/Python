Symptom

In the item overview on switching to the tab 'Financial Information', the button 'Financial Planning' is not visible. This issue occurs after implementing the corrections from the SAP Note 3018094.

Other Terms

Overview page, alignment, Capacity Planning, Financial Planning, button, Capacity Information, Financial Information, horizontal gutter, transparent container, alignment, help, FIN_CONTAINER, HIDE_CONTAINER_HEADER

Reason and Prerequisites

Program Error.

Solution

Use transaction SNOTE to implement the corrections delivered with this SAP Note in the affected PPM system or import the relevant Support Package.
