Symptom

Error: Enter a valid date interval in line EXEMPT received when syncing a supplier to ERP.

Cause

You encounter this error when values for Validity Period Start Date and End Date for Withholding Tax Exemptions are not provided when Withholding Tax Exemptions are enabled. 

Resolution

You need to ensure that values for Validity Period Start Date and End Date for Withholding Tax Exemptions are provided when Withholding Tax Exemptions are enabled. 

See Also

Supplier field mapping for Validity Period Start Date and End Date for Withholding Tax Exemptions are vendor.companyCode.withholdingTaxApis.withholdingTaxExemptionValidityPeriodStartDate and vendor.companyCode.withholdingTaxApis.withholdingTaxExemptionValidityPeriodEndDate
