Symptom

  • Acquiring data in Lumira Desktop from a direct connection to BW (Business Warehouse) results in the error:
    "Data Acquisition is not supported for this query. Check the query settings"

Other Terms

BW query with drill characteristics > 99

Reason and Prerequisites

Environment:

  • SAP Lumira Desktop 1.31.7
  • Windows Server 2012 R2

Steps to reproduce:

  1. Go to File > New
  2. Select "Download from SAP Business Warehouse"
  3. From the "Connect To" drop down menu select "Local BW System"
  4. Enter the BW system credentials
  5. Select a "BeX Query"
  6. Select all of the dimensions and measures
  7. Click the "Create" button
  8. Error occurs stating "Data Acquisition is not supported for this query. Check the query settings".

 

Solution

For resolution of this issue, see Note:Error for query with many drill-down characteristics. https://launchpad.support.sap.com/#/notes/2363340

127 is the maximum number of characteristics which can be added to an axis, see Note 2264336 - Maximum number of characteristics on axis .https://launchpad.support.sap.com/#/notes/2264336
