Symptom

Structure of APD file for TEKA has been changed.

Other Terms

APD, TEKA, HGRUAPD0, HGRUAPD4, HGRUAPD5

Reason and Prerequisites

This SAP Note is relevant only for Greece.

Solution

Since the changes to TEKA APD file are very close to EFKA APD file, report HGRUAPD5 (ASCII file CSL01 creation for TEKA employees) is now deprecated and has been deactivated.

Report HGRUAPD4 (IKA ASCII File) can again be used for both EFKA and TEKA APD files. Report has been updated and for TEKA file line type 3 is different.

Report HGRUAPD0 (IKA APD records creation) has been adjusted as well. In certain cases of TEKA with specific combination of KAD/KPK fields that are not present on the TEKA file but not explicitly cleared. Those fields are saved to the table, but not reported by the HGRUAPD4.
