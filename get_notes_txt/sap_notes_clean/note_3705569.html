Symptom

When trying to do a import for the AI Assisted Skills Standardization process Custom label (C) does not retain the custom names and system behavior is not consistent with portal  AI-Assisted Skills Standardization | SAP Help Portal 

Environment

SAP SuccessFactors Talent Intelligence Hub

Reproducing the Issue

  1. You follow the steps for the AI-Assisted Skills Standardization | SAP Help Portal
  2. Import .ICD file of "Skill Standardization Import", available in here: Integration Center Definition | SuccessFactors Talent Intelligence Hub Library Import Templates | SAP Business Accelerator Hub 
  3. Notice that Custom label (C) does not retain the custom names or behavior is not consistent with  AI-Assisted Skills Standardization | SAP Help Portal 

Cause

Known Error

Resolution

This scenario is caused by issues with the .ICD file. Our Engineer team are updating the correct file in Integration Center Definition | SuccessFactors Talent Intelligence Hub Library Import Templates | SAP Business Accelerator Hub

Workaround

In the meanwhile, you can use the .ICD file that is available in the attachment of this KBA.
