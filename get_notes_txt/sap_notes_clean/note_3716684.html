Symptom

When attempting to approve the termination workflow for Manager A, the following error message is displayed:

"This operation cannot be performed due to errors in other records."

    Environment

    SAP SuccessFactors Employee Central

      Reproducing the Issue

      1. Initiate the termination process for Manager A and discover an unknown direct reportee in the system.

      2. Click Save. A termination workflow is automatically triggered.

      3. Attempt to approve the workflow. The error message is displayed: "This operation cannot be performed due to errors in other records."

      Cause

      Using Data Inspector tool to examine Table_USERS_SYSINFO for users where Manager = A, it was identified that Manager A has a direct reportee with the following characteristics:

      • User ID: AAA
      • Username: BBB
      • Valid: e (means it is a Onboarding external user)
      • Name Field: Unknown

      This incomplete user record prevents the termination workflow approval from processing.

      Resolution

      1. Verify the status of the direct reportee.
      2. Determine whether the direct reportee needs to be hired or purged.
      3. After verifying the above, reattempt to process the termination workflow again.

      Keywords

      data inspector, onboarding user, termination process
