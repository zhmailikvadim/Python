Symptom

Fiscal year is not equal to calendar year. Prior year exchange rate is the latest exchange rate of previous calendar year instead of the latest exchange rate of fiscal year. 

Environment

SAP S/4HANA Finance for Group Reporting

Reproducing the Issue

  1. Set "prior year exchange rate" option in one of the currency translation sequence
  2. Run currency translation in consolidation version where the fiscal year is not equal to calendar year
  3. The prior year exchange rate is derived from previous calendar year

Cause

Designed behaviour. 

Resolution

Use this function only if you need the latest exchnage rate from pervious calandar year  

Keywords

currency translation, prior year exchange rate, exchange rate discrepancy, fiscal year variant, task failure, prior year ER option, designed behavior, exchange rate table.
