Symptom

In SAP Solution Manager, database-related background jobs and monitoring collectors periodically fail (typically every hour) with an SQL error.

The following error message is reported:
SQL error text: "numeric overflow"
SQL code: 314

Other Terms

Solution Manager, SolMan, DBA Cockpit, DBACOCKPIT, SQL error 314, numeric overflow, database monitoring, background job failure, periodic job error, database collector

Reason and Prerequisites

The error is caused by a numeric overflow during execution of periodic Solution Manager database monitoring collectors.

Internal SQL statements in DBA Cockpit exceed the numeric range of database-specific data types when calculating monitoring data, resulting in SQL error code 314.

Solution

Implement the correction instructions provided with this SAP Note.
