Reason and Prerequisites

Program Error

Solution

To solve the issue please apply the Support Package as mentioned below. In urgent cases you can use the correction instructions.

Delivery Information

  • SAP BW/4HANA 2021
    Implement Support Package 14 for SAP BW/4HANA 2021 (SAPK-30014INDW4CORE) into your SAP BW/4HANA system. The Support Package will be available as soon as SAP Note 3625395 with the short text "SAPBWNews SAP BW/4HANA 2021 SP14", which describes this Support Package in more detail, is released for customers.
  • SAP BW/4HANA 2023
    Implement Support Package 09 for SAP BW/4HANA 2023 (SAPK-40009INDW4CORE) into your SAP BW/4HANA system. The Support Package will be available as soon as SAP Note 3678578 with the short text "SAPBWNews SAP BW/4HANA 2023 SP09", which describes this Support Package in more detail, is released for customers.

In urgent cases you can use the correction instructions.

Before you use the correction instructions, make sure that you check SAP Note 1668882 and SAP Note 2248091 for transaction SNOTE.

This SAP Note might already be available before the Support Package is released. In this case, however, the short text still contains the term "preliminary version".
