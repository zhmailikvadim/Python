Symptom

  • Additional field is defined as clearing rules as described in consulting note 3436397 to enable them for the reduced OIM field catalog.
  • However, the field is not carried forward as expected. 

Environment

  • SAP S/4HANA
  • SAP S/4HANA Finance

Reproducing the Issue

  1. A field is active in the BCF field catalog.
  2. This field is also defined as clearing rule.
  3. This field is not carried forward to next fiscal year for open-item-management account in balance carryforward.

Cause

One of the reason is the program error.

Resolution

Please implement the following sap note and check it again.

SAP Note 3712324 - BCF: Additional clearing rules not considered in reduced OIM field catalog as expected
