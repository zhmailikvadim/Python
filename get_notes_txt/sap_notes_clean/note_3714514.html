Symptom

You run Seniority reporting (RPCPAYRU_SZVR) for a reporting area with a large number of employees. The report runs in background for a long time before providing a result.

Other Terms

HRPAYRU_SZVR

Reason and Prerequisites

Infotype buffer is not initialized on starting processing a new employee.

This solution applies to both new and old infotype frameworks.

Solution

As a general rule, SAP recommends that you implement the Support Package specified for your release.
However, if you need to install the solution earlier, use the SAP Note Assistant to carry out automatic correction instructions. You can find more information about the SAP Note Assistant in SAP Support Portal, under SAP Note Assistant.
