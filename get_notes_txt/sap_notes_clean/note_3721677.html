Symptom

I set the Enable Multi-Factor Authentication for a user to Yes. The Ariba system automatically changed the value to No after some time.

Cause

Data replication in Multi-ERP configurations replicates Shared User data from parent site to child site.

Resolution

We need to set the value of Enable Multi-Factor Authentication for a user to Yes in the parent site and allow the data replication to transfer the value to the child site.
