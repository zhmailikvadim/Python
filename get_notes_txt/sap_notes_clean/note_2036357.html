Symptom

Component: Product Compliance
Module: BOMBOS Transfer

You are in Compliance Workbench and want to start a BOMBOS transfer for a product or component. You switch to the BOM overview dialog and start the BOMBOS transfer asynchronously. If the system cannot start the BOMBOS transfer you get a message like "You can complete a collaboration only if you release the revision".

This SAP note replaces this message with the correct one.

Other Terms

EHSM300SP07PRC
EHSM400SP03PRC
EHSM500SP01PRC

CM_EHPRC_COMP_MGM_UI 014

Start BOMBOS

You can complete a collaboration only if you release the revision

Reason and Prerequisites

Reason:
Program Error

Prerequisites:
The validity of the correction is maintained in the correction instructions.

Solution

The Support Packages that contain this correction are listed in the "Support Packages" section.

Alternatively, you can implement the correction instructions of this SAP Note.
