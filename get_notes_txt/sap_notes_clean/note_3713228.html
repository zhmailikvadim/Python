Symptom

When a message is stored in FCOT_EB_LOG during real-time (event-based) overhead calculation and the original document is then reversed, the message remains in the log.

Other Terms

Event-based overhead calculation, Real-time overhead calculation, KZPR, FCOT_EB_LOG

Reason and Prerequisites

 Missing functionality

Solution

This note implements a functionality that ensures that the message linked to original document is deleted when the original document is reversed. Implement this note or apply corresponding SP.
