Symptom

The ATC Error - Security Checks for ABAP (CVA) - Ungültige Validierung über CL_ABAP_DYN_PRG (tautologische Wertprüfung) / Invalid validation via CL_ABAP_DYN_PRG (tautological value check) in the package FS_TB_DATA_STORE_CN is addressed in this  SAP Note.

Other Terms

Tautological Value Check, FS_TB_DATA_STORE_CN, ATC Error, Security Checks for ABAP (CVA), FSTB_DS_AC_DATA_SET, FSTB_DS_MC_DATA_SET, FSTB_DS_CA_DATA_SET

Reason and Prerequisites

ATC error fix

Solution

Import the relevant Support Package for your release. You can use transaction SNOTE to implement the correction instructions relevant for your release in advance.

There are no manual activities.
