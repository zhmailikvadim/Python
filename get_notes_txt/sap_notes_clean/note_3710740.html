Symptom

  • External interface for Budget Master is unable to retrieve and manage available budget data due to the absence of the Project Available Budget API.
  • Integration challenges are occurring as a result of the missing API.

Environment

SAP S/4HANA Cloud Public Edition

Cause

Currently, SAP S/4HANA Cloud Public Edition does not provide a standard public API to retrieve Project Available Budget (available budget = budget minus actuals and commitments). This functionality is handled internally by SAP through Budget Availability Control and is not exposed via transactional OData APIs.

Resolution

SAP recommends using analytical reporting (CDS views / analytical queries) to consume budget, actuals, and commitments data, or relying on SAP’s internal budget checks as the system of record.

The situation you are referring to does not represent an error in current delivery of S/4HANA Cloud but a new feature request. SAP Support deals with fixing errors in S/4HANA Cloud and does not cover feature requests. 

To see new functionality introduced in the future you are encouraged to create an improvement request. SAP Customer Influence enables customers to submit improvement requests and to openly vote on them.
