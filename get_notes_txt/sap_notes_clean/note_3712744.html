Symptom

Symptom

Transportation Management in S/4HANA: When not providing any stop information and the system fails to automatically determine the correct stop, only a generic error message is raised. After implementing this SAP Note a specific error message pointing out the exact problem is raised for this scenario.

Other Terms

/scmtms/cl_tor_a_root_repevent; Event Reporting;

Solution

Implement the attached Correction Instruction (CI) or the relevant Support Package.

Other Terms

F4435, POS_MANS1, Manage Positions, Requirement facet, S/4HANA Defense and Security.

Reason and Prerequisites

Relevant code added for component change.

Solution

In the Requirement facet of the Manage Positions app, Qualification level id dropdown in Qualification table is updated.

As a general rule, SAP recommends that you install a solution by applying a Support Package. However, if you need to install a solution earlier, use the Note Assistant to implement the correction instruction. You can find more information about the Note Assistant in SAP Service Marketplace, under service.sap.com/note-assistant.
