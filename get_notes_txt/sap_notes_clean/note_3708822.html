Symptom

  • Users want only Active Accounts to be displayed by default in the Accounts view when selecting All Accounts.
  • There is no standard “Active Accounts” default filter available in the UI. 

Environment

  • SAP Sales Cloud Version 2
  • SAP Service Cloud Version 2
  • SAP Sales Cloud And SAP Service Cloud Version 2

Reproducing the Issue

  1. Navigate to Menu → Accounts
  2. Click on Query / Saved Filters
  3. Review the list of available standard filters

Result:

There is no standard filter available to display Active Accounts only.

Cause

  • The system does not offer a standard default filter for displaying Active Accounts in the Accounts view.
  • Default filters can only be configured using saved queries, not directly through standard filters.  

Resolution

1. Enable Adaptation Mode.  
• Turn Adaptation Mode ON.  
• This step ensures the filter is visible to all users.  
• If Adaptation Mode is OFF, the filter will only be visible to the current user.  

2. Create a Custom Query.  
• Navigate to Accounts.  
• Create a new query with the condition:  
• Status = Active.  
• Save the query with a descriptive name, such as Active Accounts.  

3. Save Query for All Users.  
• Save the query while Adaptation Mode is ON to make it globally accessible.  

4. Set the Custom Query as Default Filter.  
• In the Accounts view:  
• Open Saved Filters.  
• Under Default Filter, select the newly created Active Accounts query.  
• Ensure Adaptation Mode remains ON during this step.  

5. Save the Changes.  
• Confirm and save the default filter settings.  

Keywords

Active Accounts, Default Filter, Saved Filters, Custom Query
