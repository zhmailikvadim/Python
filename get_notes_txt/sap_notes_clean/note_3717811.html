Symptom

The file for AI reporting does not contain the expected sender.

Environment

  • SAP Release S4HCMCDE, SAP ERP 6.0 and higher
  • Human Resources / HR / HCM

Reproduce Issue

A file generated using the report RPCUVHD0_OUT does not contain the expected sender that is defined for the reporting personnel area in the subapplication UVAB.

Cause

For the subapplication UVAB, the grouping to the reporting personnel area was defined with a feature in the table view V_T596L.
The structure of the feature does not correspond to the value PME01.

Solution

Check and update the definition of the used characteristic.

  1. Call feature maintenance using transaction PE03.
  2. Use the attributes to branch to the structure of the characteristic.
  3. The structure name for decision fields must be PME01.

Feature DZUBT can be used as a template because it is defined with structure PME01.

Keywords

RPCUVHD0_OUT, UVAB, PME01, sender
