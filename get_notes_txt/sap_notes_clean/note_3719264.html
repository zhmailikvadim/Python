Symptom

How to enable Standard Shipping Address for Request for Quotes?

Resolution

With this feature, you can configure your site to display the user's chosen address on request for quotes (RFQs). You can chose to automatically display the user's default shipping address on RFQs, or allow users to chose the shipping address for RFQs.

General Prerequisites

You must enable the new guided buying parameter:

PARAM_ENABLE_RFQ_SHIP_TO_ADDRESS_IN_USER_CHOSEN_ENTITY

IDPARAM_ENABLE_RFQ_SHIP_TO_ADDRESS_IN_USER_CHOSEN_ENTITY
Default valuefalse
 

Defaults the ship-to address on RFQs to the user's chosen address, instead of using the user's default ship-to address.

If set to false (default), the user's default ship-to address is used by default on RFQs.

If set to true, the user's chosen address is used by default on RFQs.
