Symptom

User is requesting CIDR range for SFTP host to connect using external tools.

Environment

  • SAP Sales Performance Management
  • Incentive Management on Hyperscalers

Reproducing the Issue

Cause

The user is attempting to do an external connection to SFTP.

Resolution

The dropbox SFTP hosts are externally accessible, so no CIDR range configuration is required. In case of persistent connection issue, please provide the following details when opening an SAP Case:

  1. If connection issues persist, provide the connection details and configuration parameters used for analysis.
  2. Capture and share screenshots or recordings of any error messages encountered.

See Also

  • SAP Note 3527121 - Troubleshooting Connection Issues via SFTP: IP Firewall Configuration
