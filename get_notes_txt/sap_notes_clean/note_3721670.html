Symptom

This KBA explains how to display the field 55 and 56 in BIR2316 form on ESS.

Environment

  • Human Capital Management (HCM)
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP SuccessFactors Employee Central Payroll
  • SAP S/4 HANA

Resolution

The feature 48EOY enables employees to view the End of Year Tax Statement (BIR2316) for the Philippines via Employee Self‑Service (ESS). The variant configured for this feature affects the report shown in ESS.

  1. Run T-Code: PE03 and check the variant maintained in feature 48EOY. By Default variant SAP&EOYT is used.
  2. Ensure that the "Substitute Filing" and "E-Signature" options are selected in the variant SAP&EOYT or custom variant maintained in the feature.
  3. Test the configuration in the environment to ensure fields 55 and 56 are correctly populated in the BIR2316 form on ESS.
