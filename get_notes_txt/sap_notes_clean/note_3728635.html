Symptom

When a POS Inbound financial transactionis scheduled, interface throws 'balance in transaction currency' error when the accounting document is generated. This error is occured whenever the interface payload has total amount value less than or equal to 1

Other Terms

Point-of-sale, POS, POS Inbound,POS Financial Transactions

Reason and Prerequisites

Program error

Solution

Kindly implement the correction instruction. After implementation, accounting document will be generated without any error.
