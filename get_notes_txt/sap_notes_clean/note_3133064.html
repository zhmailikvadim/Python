Missing function

Solution

Implement the attached corrections using SNOTE or by importing the referenced support package.

Explanation of the solution

Transactions are displayed in the menu tree only if the executing user has start authorization and the transactions are not locked.
If all transactions of a node cannot be executed, the node itself remains visible in the menu.
