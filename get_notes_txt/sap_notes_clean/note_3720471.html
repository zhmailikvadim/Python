Symptom

The procedure configured in the Price Sheet Workbench with the 'On Check' option is not being executed during price review generation.

Other Terms

Price Sheet Workbench, On Check, price review generation

Reason and Prerequisites

This is caused by Program error.

SAP Note#3720431 is available for 60E release.

Solution

Please implement the correction instructions. These corrections are only valid if you have applied the following Support Packs.

Release V/4 – 2022 V2, 2023, 2023 V2, 2023 V3 and 2023 V3 Support Package 1

Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object
