Symptom

  • The worker is unable to utilize the 'Daily Verification' feature via the SAP Fieldglass Time Entry mobile app.
  • The mobile app does not provide an option for workers to submit time for verification, which is available in the desktop application.
  • The issue occurs when attempting to submit time for verification using the mobile app.

Environment

Timesheet (Statement of Work) - Submission

Reproducing the Issue

  1. Enable the 'Daily Verification' option on the SOW Type.
  2. Log in as a worker via the desktop application and submit time for verification successfully.
  3. Attempt to log in as a worker via the SAP Fieldglass Time Entry mobile app.
  4. Observe that the option to submit time for verification is not available.

Cause

The SAP Fieldglass Time Entry mobile app does not support the 'Daily Verification' feature, as it mirrors the desktop application's functionality without offering additional features.

Resolution

  1. The 'Daily Verification' feature is not available in the mobile app as per design.
  2. Workers must use the SAP Fieldglass desktop application to access the 'Daily Verification' feature.
  3. SAP Fieldglass Time Entry mobile app is designed to manage time sheets and track hours, but certain features like 'Daily Verification' are not available on the mobile platform. 
  4. If desired, submit an enhancement request for the feature to be considered in future releases.
