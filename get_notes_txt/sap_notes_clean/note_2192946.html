Symptom

You are using a HANA DataSource and encounter one of the following problems:

1: Data Replication using a Data Provisioning Agent fails. The HANA DDL to create a "Remote Subscription" is missing a closing bracket in the WHERE-clause.

2: The HANA Datasource contains a field with HANA Data type 'BLOB'. The Data Preview fails with database error "string is too long: length can't exceed maximum length(16384bytes) at function __trex_field_BString__"

3: Activation of a HANA DataSource fails with error "Wrong specification of Length or Scale". The error is not shown when pressing the "Check" button.

 

Other Terms

HANA DataSource

Reason and Prerequisites

Program error

Solution

To solve the issue, apply the following Support Packages:

  • SAP BW 7.50

    Import Support Package 1 for SAP BW 7.50 (SAPK-75001INSAPBW) into your BW system. The Support Package will be available as soon as SAP Note 2192427 with the short text "Preliminary Version SAPBWNews NW BW 7.50 ABAP SP1", which describes this Support Package in more detail, is released for customers.



In urgent cases you can use the correction instructions.

Before you use the correction instructions, make sure that you check SAP Note 1668882 for transaction SNOTE.

This SAP Note might already be available before the Support Package is released. In this case, however, the short text still contains the term "preliminary version".
