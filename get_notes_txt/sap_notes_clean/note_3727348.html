Symptom

When viewing a custom MDF object on the Latest People Profile, the "Show Empty Fields" functionality works for some groups but does not expand for others.

“Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Environment

SAP SuccessFactors HCM Suite

Reproducing the Issue

  1. In the Latest People Profile, navigate to the affected custom MDF object.
  2. Observe that some groups expand when you click "Show Empty Fields" while other groups do not.

Cause

A forward slash (/) in the group’s Title field prevents the "Show Empty Fields" functionality from expanding for that group.

Resolution

  1. Go to Manage Configuration UI.
  2. Select the UI for the affected custom MDF object.
  3. Locate the affected group and edit its Title to remove the forward slash (/). Replace the forward slash with an alternative character such as a vertical bar (|) or a backslash (). Example:
    • Test3/Test3 -> Test3|Test3
    • Test3/Test3 -> Test3\Test3
  4. Save the changes.
  5. Verify the change by returning to the Latest People Profile, opening the Custom MDF Object, and confirming the "Show Empty Fields" control expands for the group.
