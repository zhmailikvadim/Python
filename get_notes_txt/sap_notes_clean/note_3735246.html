Symptom

When MDG Change Request is getting approved in system and idoc is initiated to target system, MDG system is not sending KNVVM segment to the target system.

Environment

  • SAP Master Data Governance
  • SAP S/4 HANA

Reproducing the Issue

  1. Approve the relevant record in the source MDG system so that a DEBMAS IDoc is initiated to the target system.
  2. Monitor the outbound DEBMAS IDoc and observe that the KNVVM (E1KNVVM) sales area segment is missing.
  3. Alternatively, follow the steps contained in the attachment “E1KNVVM_Segment Issue.docx”.

Cause

Code corrections

Resolution

Please implement below SAP Note to resolve this issue: 

  •  3449570 – “Missing sales area segment in DEBMAS idoc type”.

See Also

  • 3449570 - Missing sales area segment in DEBMAS idoc type
  • 3428671 - MDG-C/S: IDoc Replication - Corrections for Governance Scope
