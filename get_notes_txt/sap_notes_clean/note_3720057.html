Symptom

Inconsistent event registry table update for IC scenario during the LDC load event.

Other Terms

ACM Error; Event registry update error; Event registry error for IC( Inter/Intra company ) scenario; /accgo/t_evtreg update error;

Reason and Prerequisites

Event registry table had inconsistent update for IC scenario when application document for sale & purchase go to finally applied status during   the LDC load event creation and did not upate even after creation of LDC unload event.

Solution

Implement the automatic correction instruction(s) to resolve the issue.
