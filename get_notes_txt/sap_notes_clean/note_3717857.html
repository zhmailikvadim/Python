Symptom

02/24/2026: Missing correction instructions added

For the field "CI Children", the system sometimes does not print a number, even though this has an effect on the EE contribution of the CI contribution rate for persons with statutory insurance as of age 23.

Other Terms

CL_HRPAYDE_CEDT_SI_INFO->GET_VALUE, EV_KZ,

Reason and Prerequisites

The remuneration statement suppresses the number for which this does not affect the employee's contribution to the CI contribution rate. (Privately insured persons, mini-jobers, younger than 23 or before 1940 born.)

Currently, the value is transferred from the SI payroll table in all other cases. However, it can also be empty if it should actually be the number 0 for the remuneration statement.

Solution

Implement the attached correction instructions or import the specified support package.

As a result, the empty value from the SI payroll table is displayed as 0 CI children in the remuneration statement if it is relevant for the EE contribution to the CI contribution rate.

After you import the HR Support Package or implement the correction instructions, you do not have to perform any further follow-on activities.
