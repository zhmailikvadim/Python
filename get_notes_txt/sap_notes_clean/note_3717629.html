Symptom

An error occurs when login to Fiori Launchpad

Login using URL parameters is not possible. No login cookie exists.

image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

• S/4 HANA 2023
• SAP Fiori Launchpad ABAP Services

Reproducing the Issue

1. Log in to Fiori using HTTP protocol.

Cause

Incorrect parameter configuration

Resolution

1. Verify the parameter configuration.  
2. Set the parameter icm/HTTP/samesite = None.  

See Also

3400023 - Logon with URL parameter not possible; logon cookie is missing
