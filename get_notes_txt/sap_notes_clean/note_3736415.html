Symptom

An error occurs in the application job at the step of copy operator, displaying an error message: "planning filter XXX is invalid" (XXX stands for the planning filter ID).

Environment

SAP Integrated Business Planning

Reproducing the Issue

  1. Start the copy operator execution that references a planning filter.
  2. Observe that the job fails and shows the error "planning XXX filter is invalid".

Cause

The planning filter referenced by the copy operator is not shared with the business user who runs the job.

Resolution

  1. Identify which planning filter(s) the application job uses and throws the error.
  2. Verify the authorizations of the user executing the job to access those planning filter(s).
  3. Grant the required permissions to the planning filter(s) by sharing the planning filter(s) with the user, either in Planning Filters app or in Content Administration app.
  4. Confirm that the executing user has read/use access to the filter(s) and save the changes.
  5. Re-run the application job and verify it completes successfully.

Keywords

Copy operator, Application job error, Invalid planning filter, Insufficient authorization, Authorization issue,
