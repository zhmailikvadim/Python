Symptom

While working in Fiori App F5211 and attempting to submit a change to Sales Order item you receive the popup message "Job Submission Failed".  The HTTP response to the request while submitting the job has status code 500 with message  "In the context of Data Services an unknown internal server error occurred."

Reason and Prerequisites

The name of the Profit Center is time-dependent. The existing CDS view did not eliminate names from expired validity periods causing a duplicate key exception during creation of the job data.

Solution

Implement the Support Package specified in this SAP Note which is valid for your release. In urgent cases you may choose to implement the attached Correction Instructions.
