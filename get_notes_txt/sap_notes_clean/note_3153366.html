Symptom

You check relationship data, for example by web service inbound processing or the single object maintenance user interfaces.

The system raises a short dump "DATA_LENGTH_0, CX_SY_RANGE_OUT_OF_BOUNDS".

Reason and Prerequisites

Software Error

Solution

Apply the attached correction instructions or the related support package to the affected system(s).

Other Terms

gc_comctrl_trans_ext, gc_comctrl_trans_ext_nocheck, E065, APO_BAPI064

Reason and Prerequisites

The reason is a missing functionality.

Solution

Implement the attached correction instruction or install the corresponding support package.

After implementing the note, value 3 (external commit) or value 6 (external commit without check) can be used. Be aware, that external commit control can only be used, when a planning version is entered into parameter PLANNING_VERSION. If you want to use value 6, be aware that no checks are executed and you have to make sure that everything is consistent.
