Symptom

When running the Risk Analysis with the ‘Consider Org Rule’ checkbox selected, the organizational rule is not highlighted as an exception, even though the corresponding SOD risk is triggered.

Environment

SAP Access Control 10.x
SAP Access Control 12.0

Reproducing the Issue

1. Create functions

2. Create Risk with functions

3. Create Org rule with two Org Level.

4. Run Risk analysis

5. Risk analysis result : Org rules not detected

Cause

The organizational rule is triggered when both company code is available same time. 

Resolution

1. Change condition to OR instead of AND

OR

2. Separate Organization Level to different Org Rules

Keywords

Org rule, Consider Org Rule, Org Level, Organization Level, Organization Rule
