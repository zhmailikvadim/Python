Symptom

Note 3646233 implemented the check when sending the notification record. If for an item the commodity code in the material master differs from the commodity code in the item, the notification record is not sent, and the error message is shown. However, the check does not take into account that the commodity code can be maintained in the delivery foreign trade data.

Other Terms

6000160910, INC44503966, C2SFIBU, vZTA, BTI, STAWN, MARC, YYAHSD.

Reason and Prerequisites

Program error.

Prerequisites: note 3646233 is implemented.

Solution

Implement changes according to the correction instructions.
