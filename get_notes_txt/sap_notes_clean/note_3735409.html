Symptom

  • Users are unable to Add Goal Comments on the Performance Form in the Latest Experience for Performance Management Forms
  • In Legacy Form Experience Users CAN Add Goal Comments

Environment

  • Product: SAP SuccessFactors Performance & Goals
  • Forms & Templates

Reproducing the Issue

  1. Set Goal Section Permissions as Disabled for All in All Steps. 

  2. Launch Form

  3. In Legacy Performance Form - User CAN Add Goal Comments



  4. Turn on Latest Performance Form Experience - User CAN NOT Add Goal Comments

Cause

When Using The Latest Experience Performance Forms, Users Must Have Enabled Section Permission.
Form Section Permissions take precedence over Goal Plan Field Permissions. 
This Behavior is By Design and Expected Behavior.

Section Permissions

Permission

XML Value

Description

Hidden

hidden

Users can't view the section.

Disabled

disabled

Users can view the section, but can't edit it.

Enabled

enabled

Users can view and edit the section.

The permission type only applies to permitted roles in some or all steps.

NOTE: Form Section Permissions take priority over Field Permissions.

Resolution

In Latest Performance Forms Experience - User Must Not Have Disabled Section Permission If They Want To Add Comments To Goal Within Goal Section. 

Form Section Permissions take priority over Field Permissions.

Make Sure Section Permission is Enabled.
