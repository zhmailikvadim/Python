Symptom

Error on submission of Poland invoices or pulling Incoming supplier invoices in the Manage Electronic Documents application: 

  • No SOA service name is defined for interface PL_KSEF_SEND_REQUEST
  • No SOA service name is defined for interface PL_KSEF_IN_QUERY_REQUEST

Environment

  • SAP S/4HANA Cloud Public Edition
  • SAP Document and Reporting Edition, Cloud Edition
 

Reproducing the Issue

  1. Access Manage Electronic Documents application.
  2. Select the edocument.
  3. Submit the document.
  4. Error is triggered.

Cause

Missing configuration for the 2.0 communication/integration setup.

Resolution

  • Configure the Communication arrangement with scenario ID SAP_COM_0854.
       Refer to: Integrating All Other Processes and Countries/Regions with SAP S/4HANA Cloud Public Edition.

  • Configure the SSCUI "Define Process Communication Through Cloud Services" 105552 as Cloud Edition: Standard Integration.
        Refer to : Defining Process Communication Through Cloud Services (Electronic Documents only) | SAP Help Portal

  • Activate the feature toggle GSLOGGEN_DRC_PL_TAA in SSCUI "Activate Features" 106573.
       Refer to :  Activate Features | SAP Help Portal. 

Note: Once the above configuration is performed, retest the scenario by creating a new document.
