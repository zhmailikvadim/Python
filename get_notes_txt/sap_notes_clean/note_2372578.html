Symptom

When checking the FQM flows, customer may notice some cleared documents (after payment) can be built into One Exposure, which is unexpected from normal behavior. So the "Finding Unexpected FI Document" tool is enhanced that, all the cleared documents with Flow Type 50/60/70/80 will be detected out by the tool. Being pushed into delta table, this kind of document can be rebuilt and deleted by Flow Builder.

Other Terms

Flow Builder Configuration Tool; Finding Unexpected FI Document

Reason and Prerequisites

There is some programming error from Flow Builder

Solution

Please apply the note.
