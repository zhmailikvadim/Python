Symptom

  • Error occurs in onboarding 1.0 orientation step when accessing an activity.
  • Error message: "Please return to the home page to continue accessing your account. If the error still appears, close the browser and re-start Onboarding 1.0. Error timestamp: XXXXXX"

Environment

SAP SuccessFactors Onboarding 1.0

Reproducing the Issue

  1. Go to Onboarding 1.0.
  2. Initiate a candidate and navigate to the Orientation step to open an activity.
  3. Observe the timestamp error and error page.

Cause

An unhandled System.OutOfMemoryException during initialization of 'XpressHR.Controls.XHRPage' (type initializer exception) resulted in a timestamp error page.

Resolution

  1. Perform an OData refresh for Onboarding 1.0 
  2. Have the user re-open the Orientation step activity from the Onboarding dashboard and confirm the activity loads without the timestamp error.
