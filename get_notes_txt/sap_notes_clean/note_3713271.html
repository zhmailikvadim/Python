Symptom

In the process "Merging of Supplier Clearing Lists (SUP View)" (DE_B_EQS_CLSUP_MERGE_RECEIVE_SUP), the system encountered an error when saving the clearing list data.

This issue is relevant for the market role SUP.

Other Terms

EQS, supplier clearing list, merging, inbound data

Reason and Prerequisites

The error occurred because of a program error.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing the SAP Note, the system saves the clearing list data successfully, and the process completes without errors.

You need to receive the supplier clearing list from the DSO again. You can verify the fix by checking that the process completes successfully without errors.
