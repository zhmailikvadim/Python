Symptom

When executing the calculation bucket mass processing report /IRM/IP_CLB_MASS_PROCESS** via transaction SE38, the Function field (Accrue/Settle) on the selection screen does not allow any selection.


As a result, users are unable to choose the required function and proceed with the mass processing of calculation buckets.

This issue occurs consistently when the report is executed directly from SE38.


This SAP Note resolves the issue.

Other Terms

SE38, Report, Mass Processing, Calculation Bucket, /IRM/IP**_CLB_MASS_PROCESS, Accrue, Settlement

Reason and Prerequisites

This is caused by program error.

Please implement the latest version of SAP Note #3652842 as pre requisite note.

Solution

Please implement the correction instructions attached. These corrections are only valid if you have applied the Support Packs.


Release V4 - 2020 Base Package, 2020 V2 Base Package, 2021, 2022, 2022 V2, 2023, 2023V2, 2023 V3 .

Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object.
