Reproducing the Issue

Cause

The issue has been logged under SAP CR 831810. Its description reads, "Signal 8 on stu_sortrange_to_hist".

Resolution

  • There is currently no target SAP ASE Service Pack scheduled for this issue (SAP CR 831810). It has been assigned to our Product Group to be evaluated. Please mark this KBA as a Favorite to be automatically notified when the CR status is changed

  • Workaround: set auto_temptable_stats off
