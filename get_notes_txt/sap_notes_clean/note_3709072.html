Symptom

You run the integration for SV as Classic BOM with Field and Value Mapping set to fill BOM subitem text. Although the response shows success, the subitem text is not filled. Field and Value Mapping fills subitem text only when the default value is used.

Other Terms

PLMSI, PLM System Integration, Classic BOM, SV, Structure Version, Reference Designator, FVM, Field Value Mapping

Solution

Implement this SAP Note to correct the issue with FVM for Classic BOM sub-items.
