Symptom

  • ASE tables having roundrobin partition scheme shows data skew.
  • Need to remove the data skew and rebalance the partitions.

Environment

SAP Adaptive Server Enterprise (ASE)

Resolution

Use one of the following options:

Option 1: Re-partition the table using the ALTER TABLE command:

  1. alter table <tablename> partition by roundrobin 1
  2. alter table <tablename> partition by roundrobin <number_of_partitions>

Option 2: Create and drop a clustered index on the table. 

Option 3: Re-create the table using the SELECT INTO command:

  1. select * into <temp_tablename> from <tablename>
  2. drop table <tablename>
  3. select * into <tablename> partition by roundrobin <number_of_partitions> from <temp_tablename>

Option 4: Reload the data into the table using the BCP utility:

  1. bcp out the table data to a file
  2. truncate the table
  3. decide the from and to range of the datafile for different partitions so that rows are more or less equally divided
  4. bcp in with slice number and first and last row from the data file for the partition decided in the above step

Keywords

alter, re-partition, repartition, create, recreate, bcp, load, reload, clustered, index
