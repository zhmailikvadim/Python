Symptom

Error message 8A 128 "Document: Date error: Document date XXXXXX inconsistent with last-assigned date YYYYYY." occurs when cancelling the billing document for Spain.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Open the "Manage Billing Documents" app.
  2. Cancel the billing document.
  3. Error message "Document: Date error: Document date XXXXXX inconsistent with last-assigned date YYYYYY." occurs. 

Cause

The date that was entered for the document is earlier than the last assigned date of the official document number. As per Spain legal requirement, document can not be posted with a date that is earlier than the date of document that has already been posted.

Resolution

  1. Open the "Change Billing Documents"(VF02) app.
  2. Locate the billing document to be canceled.
  3. Navigate to "Billing Document -> Cancel".
  4. Change the billing date and then proceed again.
