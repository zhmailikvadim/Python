Symptom

Unable to modify multi-action task recurrence after the end date in SAC

Environment

SAP ANALYTICS CLOUD 2025.21.9 

Reproducing the Issue

  1. Login to SAC
  2. Navigate to calendar
  3. Select a Multi‑Action Task and open details panel

Notice: The Edit option for the recurring schedule is disabled.

Cause

By design behavior of the product

Resolution

Kindly submit an enhancement request to request this feature as per - 2424376 - How do you create an enhancement request and provide feedback for SAP Analytics Cloud?
