Symptom

In case of Remote-Central-Remote scenario, for a Docment that is known to both the Remote systems, when changes done for Document on Remote 1 are transferred only till Central and not synchronized to Remote 2.

With this NOTE changes for Document are transferred from Remote 1 to Central to Remote 2. 

Other Terms

Defense disconnected S4, Document, RCR, REM-CEN-REM

Solution

Implement the attached correction instructions provided with the NOTE.
