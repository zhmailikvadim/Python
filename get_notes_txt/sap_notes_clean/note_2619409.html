Symptom

When you try to create Employee, system throws below error message:

Employee ID missing

Environment

SAP Business ByDesign

Reproducing the Issue

1. Go to Business Configuration work center
2. Open Migration of Employee Data - <country>
3. Select Manually enter employees
4. Fill the ID and First Name/Last Name
5. Select Next

Cause

The cause for the issue is that the Migration Object "Employee Data- <Country>" is not enabled for migration

Resolution

Based which country you are hiring Employee, you can enable the respective Migration Object.

1. Go to Business Configuration work center
2. Select Overview view
3. Search and Open the fine tuning activity "Enable Data Migration per Object"
4. Check the checkbox for "Enabled" for Employee Data - <country>
5. Save the changes

Keywords

Employee id missing
