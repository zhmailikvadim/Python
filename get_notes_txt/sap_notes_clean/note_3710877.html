Symptom

  • Cloud ALM Health Monitoring is not functioning as expected.
  • The error message "Something Went Wrong, please reload and try again" is displayed when attempting to access Health Monitoring.
  • Cloud ALM tenants hosted in US10 and EU10 datacenters are affected.
  • The issue started on 02/02/2026

Environment

SAP Cloud ALM - Health Monitoring

Reproducing the Issue

  1. Navigate to the Health Monitoring feature within SAP Cloud ALM.
  2. Attempt to access the Health Monitoring functionality.
  3. Observe the error message "Something Went Wrong, please reload and try again" displayed on the screen.

Cause

An issue was identified with direct memory buffer allocation, causing connection requests to the Cloud ALM Health Monitoring Application to fail.

After the direct memory buffer was increased by the SAP Cloud Operations team the issue should be resolved

Resolution

The issue should be resolved as of 10:00 UTC on 3rd February
