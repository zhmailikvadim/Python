Symptom

Starting from April of 2026, to continue receiving emails for your tenant(s), ensure that there is an active subscription within the Cloud Service Availability Notifications application on SAP for Me. 

Environment

SAP Cloud Integration for data services

Reproducing the Issue

To view the current subscriptions for the account / user, use the following link: Systems & Provisioning - Get Notified.

Resolution

Using the Cloud Service Availability Notifications application on SAP for Me, it is possible to receive timely updates regarding SAP Cloud Services, including but not limited to planned and unplanned downtimes and customer communications. (This does not include task execution notifications nor agent downtime notifications, however.)

 Users are able to:

  • Set preferences about which types of notifications are required to receive for cases of both planned and unplanned downtimes
  • Set customer communication notifications
  • Manage the existing subscriptions and add new ones. Super administrators can also specify additional notification recipients.
  • Create bundles of Cloud Services / Systems for a process-oriented Outage Communication. See the  New Cloud Outage Notification Subscription User Experience in SAP for Me 
  • Subscribe relevant users for outage communication notification emails related to a bundle. 

Additional resources:

  • Details of the Cloud Availability Dashboard are explained in the following article: Cloud Availability in SAP for Me.
  • To view the Get Notified user guide, check the following document: Cloud System Notification Subscriptions Get Notified User Guide.
