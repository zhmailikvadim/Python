Symptom

Starting March 9, 2026, the PINT-MY specification v1.3.0 takes effect. We have made the necessary adjustments to ensure the system can generate Peppol customer invoices that comply with this new version.

The transition to this new version does not impact how you create and send Peppol invoices to customers. Just continue with the existing process.

For more information, refer to the following What's New documents on SAP Help Portal:

  • SAP S/4HANA and SAP S/4HANA Cloud Private Edition: PINT Customer Invoices: Compliance with PINT-MY Specification v1.3.0
  • SAP ERP: PINT Customer Invoices: Compliance with PINT-MY Specification v1.3.0
