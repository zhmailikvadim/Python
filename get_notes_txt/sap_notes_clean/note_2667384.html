Symptom

You run the Mensaje de Afiliación (RPCAFIE0) report for an employee that has an Anotación Base Adicionales (ABA), Modificación Bases Adicionales (MBA) or an Alta sucesiva (MA) affiliation movement. The Fechas de Control de Trabajadores (FCT) segment, which is mandatory for those movements, is not being filled.

Other Terms

&AFI - PC00_M04_MENS_AFI (for more information, see SAP Note 1071282 - CONSULT: List of tags for Spain notes.)

Reason and Prerequisites

The report was not considering the FCT segment as mandatory for ABA, MBA or MA movements.

You have to apply SAP Note 2644141 - AFI: Additional bases segment (RED 01/2018) as prerequisite.

Good to know: when you install an SAP Note, some prerequisites might be required. The prerequisites are SAP Notes that were already released and that change the same context of the current SAP Note. So, even if these prerequisites are not necessarily related to the symptom of the current SAP Note, you should implement them due to technical reasons.

The following Support Package level is the minimum prerequisite:

Release Support Package
6.08 HR SP 37
6.04 HR SP A9
6.00 HR SP E3

(For more information, click "HR Support Packages Schedule" in support.sap.com/maintenance-schedule)

Solution

From now on, the FCT segment can be filled for ABA, MBA and MA affiliation movements.

Good to know: during the piloting phase, this SAP Note may be updated and versioned depending on feedback from pilot customers. After this SAP Note changes to status Released for Customer, it will not be changed anymore, unless there is a very specific situation that demands changes. In this case, a reason for the change will be specified in the SAP Note.
