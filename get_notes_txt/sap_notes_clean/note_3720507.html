Symptom

In employee data replication, there is an error message "Enter a valid value" (message: 00 002).

Environment

  • SAP SuccessFactors Employee Central integration to SAP Business Suite
  • SAP SuccessFactors Employee Central integration with SAP Human Capital Management for SAP S/4HANA
  • Employee Master Data Replication
  • Business Integration Builder (BIB)

Reproducing the Issue

  1. Run employee master data replication. 
  2. Check SLG1 log.

Cause

The replication result does not exist in DD07T table. 

Resolution

  1. Set a breakpoint in method CHECK_STRUCTURE (CL_HRPA_FOREIGN_KEY_CHECK).
  2. Get the tabname, fieldname and <field> which caused the error message. 
  3. Run function DDUT_INPUT_CHECK in SM37 and find the corresponding DOMNAME. 
  4. Find all possible values in DD07T table. 
  5. Check and compare the inbound value and all possible values in DD07T. 
  6. If the inbound value is correct, please maintain corresponding entry in DD07T. Otherwise, please correct the inbound value. 

Keywords
