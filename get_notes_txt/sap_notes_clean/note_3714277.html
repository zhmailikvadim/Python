Symptom

The system truncates the text of the Tax Situation for CBS and IBS when you create or edit a Nota Fiscal in the sales process flow or in the Nota Fiscal Writer (J1B1N or J1B2N) transactions. The system removes characters from the end of words.

Other Terms

Legal Change, Brazil, Brazilian Tax Reform, Nota Fiscal Text, Tax Situation, CBS, IBS

Reason and Prerequisites

This SAP Note is relevant only for Brazil.

 You have SAP Note 3619306 or 3632130 applied in your system.

Solution

After you implement this SAP Note, the system displays the text of the Tax Situation for CBS and IBS on multiple lines without truncating words or removing characters at the end of lines.

As a general rule SAP recommends that you implement the Support Package specified for your release. However, if you need to install a solution earlier, use the SAP Note Assistant to implement the correction instruction. You can find more information about the SAP Note Assistant in SAP Support Portal, under SAP Note Assistant.
