Symptom

When executing the process in Simulation mode, certain errors are not detected and the simulation completes successfully. However, when the process is subsequently executed in Post mode, errors occur and the affected record is no longer visible in the processing list.

Other Terms

Event-based overhead posting, Overhead calculation, Simulation mode

Reason and Prerequisites

Certain errors were not detected during Simulation mode, allowing the simulation to complete successfully. This behavior has been corrected, and all relevant errors are now consistently detected and displayed during the simulation.

Solution

Implement correction instructions from this SAP Note or import corresponding SP.
