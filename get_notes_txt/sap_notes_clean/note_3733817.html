Symptom

This SAP Note delivers additional enhancements  for the IMICC solution referred in SAP Note 3513060.

Wage Type Reporter(HTRICM00) is adjusted to accomodate the changes to new payroll solution to support:

  • Juper split due to Workplace/Company Change
  • R&D/Dangerous work(Document codes 29, 32, 35) which split with V0 table VTYPE B.
  • New /DXX wage types for excessive amount.

Other Terms

IMICC, ICMAL

Reason and Prerequisites

This SAP Note is relevant only for payroll solution of Turkyie.

Please make sure that your system is patched up to the following minimum prerequisite Support Package level

The following SAP Note explains why this step is important

3181624 - SAP Notes Implementation Prerequisites for HCM Turkey - SAP ONE Support Launchpad

This SAP Note is a prerequisite for:

  • Note 3733818 - HRTR: IMICC - Enhancement for Payroll Wage Type Reporter - [2/3] DDIC 
  • Note 3700431 - HRTR: IMICC - Enhancement for Payroll Wage Type Reporter - [3/3] Code 

Solution

Function Group HRPAYTR_ICM_MV is adjusted accordingly.
