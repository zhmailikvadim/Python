Symptom

In report RKKBCAL2 (or transaction code S_P99_41000111), the settings defined under “Extras > Selection Scope” are not being considered during execution.

Environment

  • Controlling (CO)
  • SAP R/3
  • SAP R/3 Enterprise 
  • SAP ERP
  • SAP ERP Central Component
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA

Reproducing the Issue

  1. Start the transaction code S_P99_41000111 or report RKKBCAL2.
  2. Fill-in all required parameters.
  3. Go to Extras > Selection Scope.
  4. Choose one option and confirm.
  5. Execute.
  6. The results shown in the screen are not considering the setting chosen in the step #3.

Cause

Program error.

Resolution

Install the SAP Note:

3691993 - RKKBCAL2 does not consider the setting of 'Selection Scope'

Keywords

RKKBCAL2, S_P99_41000111, Selection Scope, Cost Estimates, Costing Run.
