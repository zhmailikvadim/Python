Symptom

While adding “Cost assignment” via Edit/pencil icon, the error “No permission” is occurring.

Environment

SAP SuccessFactors Employee Central 

Reproducing the Issue

  1. Navigate to Employee's profile.
  2. Click the Edit/pencil icon to edit “Employee Cost Assignment”.
  3. Fill all the required details and Click on Save.
  4. Error “No permission” occurs.

Resolution

  • This issue will be fixed in 2H 2026 release
  • Workaround:
    • In Latest people profile -> Take action -> Manage Cost Assignment -> Add the cost assignment.
    • (OR) Through Legacy People profile.
