Symptom

Why is expression editor not available in data source chooser of custom form?

Cause

Due to how custom forms gather the data from the users while creating a form document there is not a way of having the expression editor, only in guided buying the usage of this function will be available for data source choosers.

Resolution

You can only default values from data source chooser using the Set Value option, the expression editor will only appear for other types of fields, such as, Input Field.
