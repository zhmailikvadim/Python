Symptom

Transaction code VCH_NEW_BUNDLING doesn't exist.

Environment

  • S/4HANA Cloud Private Edition 2025
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA

Reproducing the Issue

  1. Set "VCH_NEW_BUNDLING" in command field.
  2. Press ENTER.
  3. The message is issued "Transaction VCH_NEW_BUNDLING" does not exist.

Cause

The behavior for AVC sales order (SET) scenario configuration models was changed.

Resolution

There is no need to have transaction code VCH_NEW_BUNDLING for S/4 release 2025.

In lower releases than S/4 2025, the transaction VCH_NEW_BUNDLING is available to switch to the new behavior.

See Also

In consulting SAP Note 3190519 the change is described in detail.

Keywords

VCH_NEW_BUNDLING, AVC, Sales Order, SET, configuration profile, LO-VCH
