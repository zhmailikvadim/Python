Symptom

  • You have run into errors when using SAP Enterprise Threat Detection 1.0 Support Package 01.
  • You want to upgrade the HANA version underneath your installation of SAP Enterprise Threat Detection to HANA 1.0 SPS09.
  • You have upgraded the HANA version underneath your installation of SAP Enterprise Threat Detection to HANA 1.0 SPS09 and now the launchpad and some tiles of SAP Enterprise Threat Detection do not work anymore.

Other Terms

SAP Enterprise Threat Detection, SAP HANA, support package, upgrade, compatibility

Reason and Prerequisites

  • This SAP Note contains information about the corrections provided with SAP Enterprise Threat Detection 1.0 Support Package 01 Patch level 01.
  • Due to some changes in SAP UI5 and Fiori launchpad you will have to take a few manual configuration steps in order to be able to run SAP Enterprise Threat Detection 1.0 Support Package 01 on HANA 1.0 SPS09.

Solution

Compatibility information for SAP Enterprise Threat Detection support packages and SAP HANA revisions can be found in SAP Note 2137018

Following aspects are addressed in Patch level 1 in Support Package 1 of SAP Enterprise Threat Detection:

  1. Various Bug fixes (including adjustments to UI behavior)
  2. Log Learning (e.g. properly recognizing attribute value pairs)
  3. Display log of resolve user identity
  4. Automatic and more efficient deletion of old logs 
  5. colored counts in Launchpad tiles
  6. Mass activation/de-activation of Patterns
  7. Launchpad automatic refresh every minute
  8. Launchpad refresh when navigating back to Launchpad

The following steps need to be performed manually after installing Enterprise Threat Detection on SAP HANA revison 9x or after upgrading SAP HANA revision 8x to revision 9x in an existing installation Enterprise Threat Detection. Otherwise Launchpad tiles are not visible and functional (see also attachment Launchpad.pdf)

  1. Assign sap.hana.uis.db::SITE_DESIGNER role to an HANA administration user
  2. Open http://<your server>:<your server port>/sap/hana/uis/clients/role-editor/RoleEditor.html?scenario=onPremise&siteId=sap.secmon.ui.mobile.launchpad%7CETDLaunchpad
  3. Login with HANA administration user enhanced above
  4. Choose role "sap.secmon.db::EtdUser"
  5. Assign catalog "SAP Enterprise Threat Detection Widgets"
  6. Choose "Groups" tab
  7. Assign Group "SAP Enterprise Threat Detection Main Group"
