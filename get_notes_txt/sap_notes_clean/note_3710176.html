Symptom

When, or after implementing SAP Note 3639977 you receive the following syntax error in the Class /SDF/CL_CALM_TASK_HM_COLL_ABAP.

Method /SDF/IF_CALM_TASK_HM_COLLECTOR
"<FS_DATA_RESULT>" was already declared.

Other Terms

CALM, CRUN

Reason and Prerequisites

This is caused by inconsistencies in version 23/24 of SAP Note 3639977.

Solution

  1. Please make sure to fully implement the latest version of SAP Note 3639977
    If syntax errors occur during or after implementation, kindly ignore them
  2. Ensure via a syntax check of class /SDF/CL_CALM_TASK_HM_COLL_ABAP that you're the class is having above mentioned error
  3. implement attached correction instructions
  4. disable health monitoring for at least 5 minutes and enable it again
