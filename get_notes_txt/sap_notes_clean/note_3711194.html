Symptom

In the process "Receipt of SLP/TEP/PSET from DSO (SUP View)" (DE_C_EQS_SLP_RECEIVE_SUP), the business object was not populated correctly.

This issue is relevant for the market roles: SUP, DSO.

Other Terms

EQS_SUP, DE_C_EQS_SLP_RECEIVE_SUP, business object, SLP, TEP, PSET, standard load profile, time-of-use profile, profile set

Reason and Prerequisites

Program error.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the system populates the business object correctly.

You can verify the fix by checking that the business object contains the expected data in the receipt of SLP/TEP/PSET process.
