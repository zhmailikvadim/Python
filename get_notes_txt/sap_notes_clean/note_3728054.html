Symptom

In the process "Preparation and Waiting for Sending of Meter Reading (MOS MaLo View)" (DE_B_MET_PREPARE_SEND_MR_MOSMALO), the system fails to create a meter reading order in step 0350. The system displays the message: "Failed to create meter reading order for device XXX, register XXX with meter reading date XXXX.XX.XX" (Message /US4G/PROC_METERING248).

Other Terms

MET, MOS MaLo View, DE_B_MET_PREPARE_SEND_MR_MOSMALO, /US4G/MET_MR_DOC, Message /US4G/PROC_METERING248, /US4G/CL_CHK_MET_PROCESS, CHECK_READABLE_DISCRETE_REG, /US4G/CL_PRST_DET_MR_ORDER, GET_MISSING_MR_ORDERS, meter reading order

Reason and Prerequisites

The error occurred because step 0300 (Check if Meter Reading Document or Meter Reading Order Exists) did not exclude non-discrete registers. The system then tried to create a meter reading order for a non-discrete register, which caused step 0350 to fail.

Solution

Implement the corrections of this SAP Note or import the relevant support package.
After implementing this SAP Note, the system excludes non-discrete registers in step 0300 and creates meter reading orders only for discrete registers. The process runs without an error and step 0350 completes.
You need to run the process document that failed again starting at step 0300.
You can verify the fix by confirming that the system creates a meter reading order for the discrete register and that the process continues after step 0350.
