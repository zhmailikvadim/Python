Symptom

In SE01, SE09 or SE10, while checking the objects, you see that the button called 'Move object' is missing. It is also noticed that in another system this button is available.

The button is highlighted below:

Environment

SAP Netweaver and S/4HANA

Reproducing the Issue

  1. Go to SE01, SE09 or SE10
  2. Search for any request
  3. Access the screen where you can see all objects in a request

Cause

This is a feature that was added by the note 2849134.

Resolution

Please implement the SAP note below or upgrade to the respective support package version:
2849134 - SE09: Moving objects between requests

It is only possible to move objects between requests and tasks of the same type and with the same target.
In addition, multiple objects can be moved in parallel in the object list editor.
Objects that are locked in a request multiple times cannot be moved. In this case, you must first release the tasks in question or summarize the piece list.
