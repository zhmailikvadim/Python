Symptom

In F0798 "Create Billing Documents" app -> Billing Settings, "Display billing documents after creation" button is grayed out. It cannot be set to ON.

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Log on to Fiori launch pad
  2. Open F0798 "Create Billing Documents" app
  3. Click on "Billing Settings"
  4. "Display billing documents after creation" button is disabled

Cause

Missing authorization

Resolution

Assign standard business role SAP_BR_BILLING_CLERK to User ID via SU01.
