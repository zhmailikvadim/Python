Symptom

You run the Cálculo de nómina española (RPCALCE0) report with Visualizar fomrulario RH checked for any period of 2023. The Equidad intergeneracional trabajador (/3N0) wage type is beign shown at the payslip and its amount is being doubled summed in the total deductions amount.

Other Terms

&CALC - PC00_M04_CALC

Intergenerational Equity Mechanism Contribution, HR Forms, payslip, RPCEDTE0.

For more information, see SAP Note 1071282 - CONSULT: List of tags for Spain notes.

Reason and Prerequisites

This issue is a side effect caused by the solution delivered by the SAP Note 3277435 - CALC: Intergenerational Equity Mechanism contribution.

The wage type creation of SAP Note 3277435 has copied unexpected table entries of Asignación claves concepto nómina a CC-nómina acumulación (T596I) or Asignación CC-nóminas a CC-nóm.acumulación (tab.clientes) (T596J) tables.

﻿Note:

When you install an SAP Note, some prerequisites might be required. The prerequisites are SAP Notes that have already been released and that change the same context of the current one. Solution

The customizing of T596I or T596J tables must not have any entry related to the wage type /3N0. The evaluation class value of evaluation class 2 of /3N0 wage type entry of the Clases de tratamiento, acumulaciones y clases de evaluación (V_512W_D) table view has changed to zero.
