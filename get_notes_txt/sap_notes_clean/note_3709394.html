Symptom

  • No subtypes are added to a fact sheet type that needs subtypes

Environment

SAP LeanIX - meta model configuration

Resolution

  1. Access the meta model configuration of the fact sheet type in question.
  2. Click on 'Add Subtypes'
  3. Add the technical key under 'Values', click on the '+' icon to add the value
  4. Save the configuration by clicking on the 'create' button
  5. Refer to the following help center article for detailed steps:

a href="https://help.sap.com/docs/leanix/ea/configure-workspace-meta-model-v4?locale=en-US#add-fact-sheet-subtypes-to-several-fact-sheet-types" Add Fact Sheet Subtypes to Several Fact Sheet Types /a
