Symptom

You want to download Accounts through Sales organization filter using Data impex tool

Environment

SAP Sales Cloud and SAP Service Cloud Version 2

Reproducing the Issue

  1. Go to User Menu
  2. Go to Settings
  3. Go to All Settings
  4. Go to Data Import and Export
  5. Go to Import and Export Data
  6. Select Export 
  7. Select Entity as Account
  8. Click on Apply Filter Button
  9. Sales Organization field is not available

Cause

This is the current system behaviour

Resolution

  1. Go to Navigation Menu
  2. Go to Accounts
  3. Select Advanced Filter
  4. Select the Sales Organization
  5. Click Apply
  6. Click Export button 
  7. Select File Type
  8. Click Export
  9. Note: Maximum number of records that can be exported is 10000.

Keywords

export, account, sales, organization, data, impex
