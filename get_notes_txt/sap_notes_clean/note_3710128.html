Symptom

The custom field was successfully created and published in the Shipping: Delivery business context, however, it is not available as a filter in the “Create Outbound Deliveries from Sales Orders” application.

Environment

SAP S/4HANA Public Cloud

Reproducing the Issue

  1. Create and publish a custom field in the “Shipping: Delivery” business context.
  2. Attempt to use the custom field as a filter in the “Create Outbound Deliveries from Sales Orders” application.

Cause

The "Create Outbound Deliveries from Sales Order" app is designed to support only standard fields provided by SAP on the selection screen.

Resolution

Since this functionality is currently not available, we recommend submitting a feature request through the SAP Customer Influence portal. This allows SAP Product Management to evaluate the requirement for potential inclusion in upcoming releases. 

For reference, you can find more information on submitting feature requests for SAP S/4HANA Cloud Public Edition here:
2963059 - How to Submit Feature Requests for SAP S/4HANA Cloud Public Edition
