Symptom

You would like to know which database is supported for BW DB Connect as source system.

Environment

  • SAP BW/4HANA
  • SAP NetWeaver

Reproducing the Issue

  1. Tr-cd:RSA1 and goto Source system.
  2. Right click on 'DB Connect' and 'Create'.

Resolution

Please check the following list and notes.
