Symptom

ErrorMessage: App could not be opened either due to an incorrect SAP Fiori launchpad configuration or a missing role assignment.Details: { "info": "Failed to resolve navigation target \"#ReportingTask-manageTaxItems? This is most likely caused by an incorrect SAP Fiori launchpad content configuration or by a missing role assignment." ,"#ReportingTask-manageTaxItems?

Environment

  • SAP S/4HANA 2020
  • SAP S/4HANA 2021
  • SAP S/4HANA 2022
  • SAP S/4HANA 2023

Reproducing the Issue

  1. Open the Run Statutory Reports Fiori application.
  2. Enter the required details: Reporting Entity Name, Report Category Name, and Reporting Year, then click Go.
  3. Select a reporting task and execute the report run.
  4. Click on the activity MANAGE TAX ITEMS.
  5. Error Message:
    The app could not be opened, either due to an incorrect SAP Fiori launchpad configuration or a missing role assignment.

    ErrorMessage: App could not be opened either due to an incorrect SAP Fiori launchpad configuration or a missing role assignment.Details: { "info": "Failed to resolve navigation target \"#ReportingTask-manageTaxItems? This is most likely caused by an incorrect SAP Fiori launchpad content configuration or by a missing role assignment.",  "#ReportingTask-manageTaxItems?

Cause

App Implementation steps and the Replication of business catalog required

Resolution

  1. Front-End Server : Activate SAP UI5 Application

    Component

    Technical Name

    SAP UI5 Application

    FIN_MNG_TAX_ITM

    Generic Drill-Down Application

    /sap/bc/ui5_ui5/sap/fin_mng_tax_itm

     

  2. Front-End Server :  Activate OData Services

    Component

    Technical Name

    OData Service (Version Number)

    FIGL_MANAGE_TAX_ITEMS_SRV

     

    For more information about activating OData services, see Activating OData Services.

  3. Check the user is assigned with report specific authorizations as guided in the help document link: Country/Region-Specific Role Authorizations .

  4. Replication of catalog SAP_TC_FIN_GLO_BE_APPS in front-end GUI system.

    Run the report program /UI2/GET_APP_DESCR_REMOTE_DEV in SE38 transaction with the following parameters in the front-end GUI system:

    Replication system alias         : S4FIN
    Backend technical catalog ID  : SAP_TC_FIN_GLO_BE_APP
    Replication mode                   : full
    Testmode                              : unchecked
