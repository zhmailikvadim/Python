Symptom

The calculation of the PTF withdrawal using the program "Recalculate Conditions Risk Manager" does not take into account all relevant accounts. As a result, not all PTF withdrawal accounts are created.

Other Terms

/MSG/CL_H_FI_P_CALC

DUMMY_FOR_PTF_WITHDRAWL

/MSG/H_FI_P_BATCH_COND

ABRHFGK

Reason and Prerequisites

There is a program error.

Solution

History:
December 22, 2018: New

Implement the source code corrections.

Support Package SAPK-70020INMSG / SAPK-80203INMSG / SAPK-80301INMSG corrects the problem.

Solution and technical effects:
The correction ensures that the system correctly takes into account the accounting frequency for the PTF withdrawal. As a result, all relevant accounts will be processed and all PTF withdrawal accounts will be created.

Manual tasks:
You do not have to perform any manual tasks.
