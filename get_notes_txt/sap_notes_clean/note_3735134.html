Symptom

From CE2502 onwards, it has been observed that certain validations are not working on the SSCUIs in the Procurement area for SAP S/4HANA Cloud Public Edition.
This meant that it was possible to enter invalid data in the purchasing configurations, which could have lead to inconsistent data.

Environment

Reproducing the Issue

Go to the Manage Your Solution app and try any of the SSCUIs mentioned in the list above.
Try to enter some invalid data into the fields. The SSCUI will not show any errors even if the data is incorrect/inconsistent.  

Alternatively, if in the SAP Central Business Configuration, navigate to the Implementation Activities application.
Additional details about this app can be found in Implementation Activities | SAP Help Portal. 

Cause

The issue occurred after a code change provided as part of SAP Central Business Configuration activation.
The change was intended only for use cases where the SSCUIs are running via Central Business Configuration, and customers want to branch/merge/rebase between Main project line and Parallel project line or vice versa.
Due to the program error, the validations that are supposed to be raised when the SSCUIs are running with a dialog processor (a user that actively interacts with the SSCUI) were also affected. 

Resolution

A correction for this is delivered with hotfix CE2602.8, the missing validations are back in place. 
After this hotfix, users can see that the validations are back for the SSCUIs listed in the table.
If incorrect entries have already been created or you have some inconsistent data as a result of this and want to correct it, check whether the delete option is enabled for those SSCUIs, and correct the data.
If the delete option is not available, please get in touch with your SAP Representative.  

A similar issue was reported for SSCUI 102909 and a hotfix 2602.03 was already released to fix that issue.
Details are available in KBA 3703916. 3703916 - SSC UI 102909 (IMG: SIMG_CFMENUOLMEOMEC): Namespace restriction for custom purchasing document types - SAP for Me
