Symptom

In the process "Sending of Deregistration Due to End of Supply for Electricity (DSO View)" (DE_B_GPKE_DEREG_DSO), the Deregistration DSO View process does not send a Transaction Reason Additional (TRANS_REASON_ADD) as ZW6or ZW7 when triggering the Start of Backup Supply process. 
This issue is relevant for the market role: DSO.

Other Terms

GPKE, DE_B_GPKE_DEREG_DSO, Deregistration, Start of Backup Supply, Transaction Reason Additional, TRANS_REASON_ADD, ZW6, ZW7,

Reason and Prerequisites

The error occurred because the TRANS_REASN_ADD was missing when the Start of Backup Supply process is triggered via the Deregistration process.

Solution

Please implement the automatic corrections or import higher relevant Support Packages.
This SAP Notes is relevant for the following roles: DSO

After implementing this SAP Note, the Start of Backup Supply process should be triggered with the correct transaction reason additional. You can verify the fix by retesting the scenario.
