Symptom

To clean up CIP2SAP, Siemens IT has decided to eliminate the obsolete function modules:

/SIE/SF_IPC_PROCESS_XML

/SIE/SF_IPC_REPLACE_ALL

Deprecated objects and source code belonging to this functionality will be deleted and should not be used anymore.

Please see note 2200353 for more details.

ATTENTION:

Check your customer-specific code and see note 2200353 for more details.

In case you still need the functionality or parts of it in customer-specific-functionality:

  • make sure to copy the required source code/objects to your own namespace beforehand;
  • adapt your customer-specific development accordingly.

Without adaptation, the deletion of CIP2SAP coding/objects may result in various critical errors within customer-specific code or data inconsistencies.

Be aware that adaptation of customer-specific-functionalities is within your responsibility.

Other Terms

 Obsolete function, CIP2SAP, C2S 4500001210, 4500001210, Deletion.

Reason and Prerequisites

Functionality has become obsolete.

Solution

 Implement changes according to the correction instructions.
