Symptom

SAP has released note 2999028 to update the PhilHealth Premium Contribution Table for year 2021. Which is based on PhilHealth Circular No. 2020-005 as well as official statement published on December 29th 2020.

While PhilHealth has published another official statement on Jan. 5th 2021, which decided to collect Philhealth premium contribution based on 3% and ceiling P60,000, which is same as year 2020.

So this note reverted the changes delivered by note 2999028.

Other Terms

Deferment of PhilHealth contribution , V_T7PHSC

Reason and Prerequisites

This is a legal change.

Solution

As a general rule, SAP recommends that you install a solution by applying a Support Package.

However, in urgent case, please follow the manual instructions:

For Customers that HAVEN'T applied note 2999028, please ignore this note and do nothing.

For customers that have applied note 2999028, please follow attachment 'Note 3010245 Manual Instruction.pdf';
