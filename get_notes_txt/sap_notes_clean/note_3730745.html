Symptom

  • After CIDS Kyma migration, some users cannot logon CIDS.
  • This issue doesn't happen to all users. Some users work fine.

Environment

SAP Cloud Integration for data services

Cause

Users who are not added to Identity Authentication (IAS) or to the customer’s identity provider (IdP) cannot log in to the new environment.

Resolution

New Kyma CI-DS uses IAS. All CI-DS users must be added into IAS if not added before.
