Symptom

New hire is not appearing in Manage Pending Hires even after completing all steps.

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."
 

Environment

SAP SuccessFactors Onboarding

Reproducing the Issue

  1. Go to the Onboarding dashboard.
  2. Search for the affected candidate.
  3. Confirm that all onboarding tasks are completed.
  4. Go to Manage Pending Hires.
  5. Observe that the candidate is not appearing.

Cause

Hire Status in Manage Data > Process remained as Not Started.

Resolution

  1. Navigate to Manage Data.
  2. Search for the Process ID for the affected candidate.
  3. Select Take Action > Make Correction.
  4. Change the Hire Status field from Not Started to Ready To Hire.
  5. Save the record.
  6. Wait approximately 5 minutes.
  7. Verify that the candidate now appears in Manage Pending Hires.
