Symptom

When an inbound delivery is created via IDoc or ASN message using purchasing document information and quantities - without providing the product ID (MATNR) - an error occurs during the save process. As a result, the inbound delivery cannot be saved successfully.

Other Terms

Inbound Delivery, SOAP, DELIVERYREQUEST_IN, IDoc, ASN, MATNR

Reason and Prerequisites

The system triggers error ME 726 - "An error has occurred during unit of measure conversion" during processing.
This occurs in the context of the function module ME_CONFIRMATION_CHECK_QUANTITY, related to field LIPS-VRKME.

Solution

Implement the correction provided in the correction instruction of this SAP Note, or apply the corresponding Support Package that contains the fix.
