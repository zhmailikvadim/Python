Symptom

This KBA provides best practices when creating a new version for a Lease-In Contract in SAP Business ByDesign.

The aim of following the best practices is to avoid the system to fail during Lease-In Contract valuation.

Environment

SAP Business ByDesign

Resolution

When creating a new version for any contract, it must be ensured that the lease term of the latest version covers at least one complete posting period.

Examples:

  • Example 1 (Incorrect):

Latest version start date = 15 Jan 2026

Latest version end date = 31 Jan 2026

This is incorrect.

The system will fail during valuation in such cases because the version does not cover a full period.

 

  • Example 2 (Correct):

Latest version start date = 01 Jan 2026

Latest version end date = 31 Jan 2026

This is correct.

 

  • Example 3 (Correct):

Latest version start date = 15 Dec 2025

Latest version end date = 31 Jan 2026

This is correct, as it spans more than one full period.
