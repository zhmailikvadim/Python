Symptom

  • How to make the attachment mandatory during the actual investment declaration in IT declaration application.
  • Application should not skip the attachment step.

Environment

  • SAP SuccessFactors Employee Central Payroll
  • Human Capital Management (HCM) (Payroll India - Public Sector)
  • SAP ERP 6.0
  • SAP ERP Core Component 6.0
  • SAP enhancement package 6 for SAP ERP 6.0
  • SAP enhancement package 7 for SAP ERP 6.0 on SAP enhancement package for SAP NetWeaver 7.0
  • SAP enhancement package 8 for SAP ERP 6.0 on SAP enhancement package for SAP NetWeaver 7.0
  • India

Reproducing the Issue

  1. Employee Access IT declaration application.
  2. Select IT Declaration Type.
  3. Attachment details comes for entering the data.
  4. Click on Next.

Cause

Missing BADI Implementation HRPBSIN_VALIDATE_REQUEST.

Resolution

  1. Execute Transaction SE18, Enter the BADI name HRPBSIN_VALIDATE_REQUEST.
  2. Check whether the attachment table ITB_FDATA is initial.
  3. If ITB_FDATA is initial, append a hard error to CTB_ERRTB to block Next/Submit until a document is uploaded.

    Sample Code as below : 

    DATA: ls_error  TYPE bapiret2.

    IF itb_fdata IS INITIAL.

      ls_error-type      = 'E'.
          ls_error-id        = 'XYZ'.
          ls_error-number    = '001'.
          ls_error-message   = 'Attachment should be mandatory'

    APPEND ls_error TO ctb_errtb.
    ENDIF.
