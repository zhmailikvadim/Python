Symptom

During Period-End Processing (Contract), including the Classify P&L process step, the derecognition postings for FX OCI are incorrect for positions classified under accounting methodology “08 – Fair Value Through OCI with Recycling (Investment)”.

Reason and Prerequisites

The derecognition logic does not correctly include the current accruals balance when calculating FX OCI adjustments.

Solution

Please implement the attached correction instruction for your release, or install the corresponding Support Package that contains the fix.
