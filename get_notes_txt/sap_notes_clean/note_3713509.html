Symptom

In SSCUI 101071(Setting Up Your Compliance Reporting), what values must be maintained for the following parameters when configuring Spain withholding tax reports (Models 111, 115, and 216):

  • ES_WHT_SOFTWARE_VERSION (official four-digit version)
  • ES_WHT_SOFTWARE_COMPANY_TAX_ID (official 9-digit developer tax ID)

Environment

SAP S/4HANA Cloud Public Edition

Cause

Resolution

The required values for ES_WHT_SOFTWARE_VERSION and ES_WHT_SOFTWARE_COMPANY_TAX_ID depends on the submission method used for the legal file (online submission vs. manual upload via tax authority portal).

If the report is submitted via online communication, maintain the following values in SSCUI 101071(Setting Up Your Compliance Reporting):

  • ES_WHT_SOFTWARE_VERSION = v001
  • ES_WHT_SOFTWARE_COMPANY_TAX_ID =  Tax ID of the company submitting the report. 

The value maintained for ES_WHT_SOFTWARE_COMPANY_TAX_ID must exactly match the Tax ID contained in the generated text file.
