Symptom

You execute a change for a subscription contract and in the change order header you can change the external reference number (= purchase order number) and the external reference date (purchase order date). If you do so and the changes are copied back to the contract, the purchase order in the contract is also changed for items which are not part of the change order.

Other Terms

PO_NUMBER_SOLD, PO_DATE_SOLD

Reason and Prerequisites

Program error.

Solution

In a change order it is not allowed to change the external reference on header level. It must always be changed on item level. After implementing this note, the relevant fields are set to read-only.
