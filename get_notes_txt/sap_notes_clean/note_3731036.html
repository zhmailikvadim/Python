Symptom

  • An existing SAP HANA Cloud Instance cannot be added to a BDC Formation to use Dataproducts with SAP HANA Cloud.
  • The system is not listed in BDC Customer Landscape -> Systems.

Environment

HANA Cloud Services HANA Cloud

Reproducing the Issue

  1. Attempt to add an existing SAP HANA Cloud Instance to a BDC Formation.
  2. Navigate to Customer Landscape -> Systems.
  3. Observe that the instance is not listed.

Resolution

  • Make sure that all the requisites detailed in the following SAP HANA Cloud documentation are met:
    How Data Products Become Available in SAP HANA Cloud | SAP Help Portal
  • The following note is important:
    "SAP HANA Cloud Central must be running in the subacccount because it acts the data product gateway for all SAP HANA database instances in the subaccount when joined in a formation with a BDC system."
