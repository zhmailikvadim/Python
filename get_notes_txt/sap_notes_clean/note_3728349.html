Symptom

While executing BAPI BAPI_INSPECTIONPLAN_CREATE the task list (inspection plan) is created without any error, however, the formula is not saved for the calculated characteristic.

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

  • SAP ERP Central Component
  • SAP ERP
  • SAP Enhancement package for SAP ERP
  • SAP S/4HANA, on-premise
  • SAP S/4HANA Cloud Private Edition

Cause

Incomplete input for the BAPI BAPI_INSPECTIONPLAN_CREATE.

Resolution

Ensure that the control indicator field FORMULA_IND (Calculated Characteristic) in table INSPCHARACTERISTIC is set to X. Then system interprets the characteristic as calculated and the entered formula in field FORMULA_FIELD_1 will be copied properly.
