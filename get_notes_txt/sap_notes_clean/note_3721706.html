Symptom

The process "Sending of Calculation Formula (DSO View)" (DE_B_MET_CALCULATION_FORMULA_DSO) doesn't support sending calculation formulas from DSO Old to DSO New.

As specified in the UTILTS Anwendungshandbuch error correction from December 11, 2025, this function should be supported.

This SAP Note is relevant for the market role: DSO.

Other Terms

DSO Change, DSO New, DSO Old, MET, Sending of Calculation Formula (DSO View), DE_B_MET_CALCULATION_FORMULA_DSO, Receipt of Calculation Formula (DSO View), DE_B_MET_RECV_CALCULATION_FORMULA_DSO, UTILTS 25001, /US4G/CL_WIM_API, /US4G/IF_WIM_API_PROC~TRIG_CALC_FORMULA_PROC_DSO, /US4G/CL_CHK_MET_PROCESS, CHECK_UPD_CAL_FRM_IS_NEEDED, CHECK_CALC_FORM_RECIPIENT, CP_SEND_PROC_CANCEL_INFO

Reason and Prerequisites

The functionality wasn't implemented to comply with the UTILTS Application Guide requirements that allow DSO Old to send calculation formulas to DSO New.

Solution

Implement the corrections of this SAP Note or import the relevant Support Package (SP).

After implemention, the system supports sending calculation formulas from DSO Old to DSO New. The following changes are included:

- A new process "Receipt of Calculation Formula (DSO View)" (DE_B_MET_RECV_CALCULATION_FORMULA_DSO) is added.
- The process step 9810 "Inform Market Communication System That Process Can Be Canceled" (CP_SEND_PROC_CANCEL_INFO) is obsoleted from process DE_B_MET_CALCULATION_FORMULA_DSO.

Import BC set /US4G/3_0_DSO_SP08_NOTE_3721706 to activate the configuration changes.

With this NOTE and emergency patch EP/VP 2000011461, DSO Old can send calculation formulas to DSO New.

You can verify the fix by testing the sending of calculation formulas from DSO Old to DSO New and confirming that the process completes successfully.
