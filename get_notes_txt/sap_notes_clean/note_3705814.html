Symptom

On a Windows platform, a Java agent and supporting software are installed on the customer’s system, and the Java agent is consumed by SAP Analytics Cloud to support the following features:

  • Acquire data from On-Premise SAP BW systems
  • Acquire data from On-Premise SAP BPC MS systems
  • Acquire data from On-Premise SAP Universe
    • SBOP 4.3 requires version 1.0.354 or higher.
  • Acquire data from On-Premise SAP ERP Central Component 6.0
  • Acquire data from On-Premise SAP HANA Views
  • Acquire data from On-Premise JDBC Datasources
  • Acquire data from On-Premise File Server
  • Export data to On-Premise File Server 

Reason and Prerequisites

SAP strongly recommends applying maintenance shipments (on-premise application integrations) regularly at least every 6 months. For more details see the following SAP Note: SAP Note 2658835

It is also recommended that you upgrade to the latest SAP Analytics Cloud agent version 1.0.457 to take advantage of new improvements and bug fixes for all data acquisition types.

The agent’s required minimum version is 1.0.449.

List of Changes:

  • JDBC Async plugin default will use the previous solution.  To turn on, define the C4A_AGENT_CLOUD_RESILIENCE environment variable, phase 1

Solution

  1. Download SAP Analytics Cloud Agent Simple Deployment Kit from SAP Software Downloads.
  2. Unzip the contents to your local file system, and follow the instructions in the README.txt included in the .zip, and the Post-Setup Guide available in the deployment folder.
  3. Please upgrade to this version at the earliest time as this version contains an improvement for better handling of large data volume for BW acquisitions, and please ensure that Tomcat has at least 4GB of memory. Please refer to this SAP Note to increase the agent Tomcat memory. 3664717 - Tomcat Edit Service Configuration Tool for Tomcat9_SACAgent

Note: Upgrading to SAP Cloud Connector 2.17 or later requires minimum version 2.13 (see SAP Note 2539713).  If you are upgrading to this version from 1.0.118 to 1.0.333, please upgrade to 1.0.443 first before upgrading to the latest SAC Simple Deployment Kit.
