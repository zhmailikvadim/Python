Symptom

  • How to update / assign / re-assign a template in Solution Manager 7.2 System Monitoring
  • Template is still assigned to a managed object which has since had a version upgrade 

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Environment

SAP Solution Manager 7.2

Resolution

  1. Execute transaction SM_WORKCENTER
  2. Navigate to Application Operations => System Monitoring => Step 5: Define Scope
  3. Select system => click next and move to Step 6: Setup Monitoring
  4. Select the Managed Object => Click Edit
  5. Select option Restore Defaults
  6. Assign the correct template from the available list => Save the details
  7. Perform the Apply and Activate for all managed objects

  8. Refresh the browser (i.e. by clicking on F5)
  9. Navigate to step 6 again and check that the correct assignment exists
