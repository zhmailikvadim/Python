Symptom

When using learning APIs, results, all results are being displayed, instead only the results that follows the case sensitive filter.

Environment

SAP SuccessFactors Learning

Cause

ODATA search APIs are default case insensitive, and currently there is no option for users to enable or disable case sensitivity.

Resolution

For case sensitivity, note that the current behavior of Learning ODATA APIs does not support case-sensitive filters.

Check in the next releases documentation if the LRN-207711 is included to identify when this feature will be available.

Keywords

SuccessFactors, Learning, ODATA, API, case sensitivity, filter
