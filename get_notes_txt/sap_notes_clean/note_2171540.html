Symptom

When using a Planning and Consolidation for Microsoft (BPC MS) connection in the SAP EPM Add-in for Microsoft Office, no user credentials are required and the connection is automatically established using the current Windows account.

Environment

  • Business Planing and Conslidation for Microsoft version 10
  • Business Planing and Conslidation for Microsoft version 10.1

Reproducing the Issue

In the EPM Add-in, create a new connection for BPC Microsoft.

Cause

If BPC is installed using the Windows Authentication mode, the EPM Add-in will automatically connect to the server using the current Windows user account credentials.

Resolution

In order to force the EPM Add-in to prompt for the user name and password when connecting to BPC for Microsoft in Windows Authentication mode, the current Windows account should not be logged on to the same domain with BPC application server.

Keywords
