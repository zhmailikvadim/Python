Symptom

  • Is SAML SSO authentication possible for Crystal Reports Designer?
  • Is is possible to use Multi-Factor Authentication (MFA) for Crystal Reports Designer?

Environment

  • SAP BusinessObjects Business Intelligence Platform 4.3
  • SAP BusinessObjects Business Intelligence Platform 2025
  • Windows
  • Linux / Unix 

Cause

Resolution

  • This is not possible due to Crystal Reports Designer being a native Windows client that connects directly to BI Platform (not through the web tier).
  • It uses the CMS authentication plugins and only supports the following methods:
    • Enterprise, Windows AD, LDAP, SAP.
  • SAML SSO and MFA prompts (i.e., Okta/Entra/Azure AD browser flows) aren't possible as Crystal Designer doesn’t handle web redirects or MFA challenges; SAML-based SSO is only for BI webapps (BI Launch Pad, CMC, OpenDocument) not the desktop Designer itself. 
  • Additionally, Multi-Factor Authentication (MFA)is something that is handled on the Identity Provider (IdP) side.

See Also

For setting up SAML SSO/MFA with a BI webapp, please refer to: 2791348 - How To: Configure Front-End SAML Authentication on BI 4.3 - SAP for Me.
