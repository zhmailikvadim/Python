Symptom

This SAP Note provides relevant system changes to further prepare the S-2230 - Afastamento temporário (Temporary absence) event for alphanumeric CNPJs.

Other Terms

Alphanumeric CNPJ; event structures; eSocial; S-2230;

Reason and Prerequisites

Reason

According to Normative Instruction nº 2229, October 15, 2024, starting from July 2026, CNPJs may be alphanumeric. Therefore, system adjustments are necessary to ensure correct reporting of this information.

Prerequisites

You have implemented the following SAP Note as prerequisite: 3728895 - Prerequisite objects for SAP Note 3731328.

The following Support Package level is the minimum prerequisite:

Release Support Package
SAP_HRCBR 6.04 (EhP6 and EhP7) HR SP J1
SAP_HRCBR 6.08 HR SP B9
S4HCMCBR 1.00 HR SP 08
S4HCMCBR 1.01 HR SP 00
S4HCMCBR 1.02 HR SP 00

For more information, access the SAP ERP Human Capital Management - Support Package Schedules.

When you install an SAP Note, some prerequisites might be required. The prerequisites are SAP Notes that have already been released and that change the same context of the current one. Then, even if these prerequisites are not necessarily related to the symptom of this SAP Note, you must implement them due to technical reasons.

Solution

The system supports alphanumeric CNPJs in all fields of the S-2230 event. The event can process and display CNPJs that contain both numeric and alphabetic characters, ensuring compliance with the Brazilian government regulation effective July 2026.

Important:

If you have implemented any of the following BAdI methods, review the use of the CNPJ field to avoid activation and runtime errors:

  • HRPAYBR_EFD_TEMP_ABSENCE - IF_HRPAYBR_EFD_TEMP_ABSENCE->FILL_TEMP_ABSENCE_INFO;
  • HRPAYBR_EFD_TEMP_ABSENCE - IF_HRPAYBR_EFD_TEMP_ABSENCE->FILL_UNION_INFO.

Additionally, for more information regarding the implementation of eSocial BAdI check the Consulting SAP Note 3721621.
