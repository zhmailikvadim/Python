Symptom

How can I find my API application in the SAP Developer Portal?

Cause

Resolution

Follow the steps below to search for an API application in the SAP Developer Portal:

  1. Log in to the SAP Developer Portal.
  2. In the top navigation menu, select Manage > Applications.
  3. In the search bar, type the exact API Application Name.
  4. Click on the magnifying glass icon to trigger the search.
  5. Review the search results to locate your API application.
