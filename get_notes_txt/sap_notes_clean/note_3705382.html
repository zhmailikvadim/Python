Symptom

This SAP Note introduces the option to change the owner and approver of a task during an active workflow. This provides greater flexibility when you need to adapt to organizational changes or correct assignments without interrupting the workflow. Previously, you could only change the assignee.

The new Update Responsibilities button offers the following options, depending on the task type: Change Assignee, Change Owner, and Change Approver.

This feature applies to tasks of the types Task and Task with Approval that have a status of Due or Overdue.

This SAP Note contains the back-end implementation of this feature. For the required front-end changes, see SAP Note 3711203.

Other Terms

Monitor Tasks, Task Instance, Owner, Approver

Reason and Prerequisites

Reason: New functionality.

Prerequisites: The validity of the correction is maintained in the correction instructions.

Solution

The Support Packages that contain this correction are listed in the "Support Packages" section.
