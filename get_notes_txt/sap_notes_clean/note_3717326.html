Symptom

You use the new F4 structure search for budget structure elements delivered with SAP Note 3625516 in the system.

You notice the following problems:

  • For structurally restricted users in the BU structure, the system issues status message 5a244 "No objects found". In this case, the dialog for selecting other search helps is no longer sent.

Reason and Prerequisites

If no other budget structure element type can be derived from the application context, when you select the search help "Structure Search Overall Budget", the system selects the budget structure element type in hierarchy maintenance of the budget structure element types (V_HRFPM_T7PM3_H) as the highest budget structure element type for determining the initial objects for the search help.

In the case of structurally restricted authorizations, this may lead to the problem described above.

Solution

Implement the attached correction instructions or import the specified support package.
