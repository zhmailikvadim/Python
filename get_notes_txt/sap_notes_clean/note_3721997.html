Symptom

External onboardee received a system-generated email, however the PasswordResetUrl token was not populated, causing the password reset link to be missing or non-functional.

Environment

SAP SuccessFactors Onboarding

Reproducing the Issue

  1. Complete NHDR for onboardee and system triggers (ONB) Start Onboarding process template
  2. External onboardee receives the notification, but the email does not contain the required password or specific URL to set/reset the password.

Cause

  • The (ONB) External User Welcome Message Trigger is disabled. 
  • The [[PasswordResetUrl]] token is only available for the (ONB) External User Welcome Message Trigger.
  • No standard template in Onboarding meant for external candidates includes [[PasswordResetUrl]] token except the The (ONB) External User Welcome Message Trigger.

Resolution

Ensure that the (ONB) External User Welcome Message Trigger in the system is enabled, and that the [[PasswordResetUrl]] token is configured in the body of the email template.
