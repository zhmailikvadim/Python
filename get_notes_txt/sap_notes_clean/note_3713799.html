Symptom

The odata v4 API does not provide entities to maintain long texts for products. 

Environment

  • SAP ERP Logistics (LO)  
  • SAP S/4 HANA on premise
  • SAP S/4 HANA Cloud Private Edition

Reproducing the Issue

  1. Attempt to use the OData V4 API API_PRODUCT to maintain long texts for products.  
  2. Observe that there are no entities available for long text maintenance.  

Cause

Missing functionality. 

Resolution

The odata v4 API API_PRODUCT does not provide long text entities. Instead the odata v2 API API_PRODUCT_SRV can be used for long text maintenance purpose.
