Symptom

  • In CTI Configuration → Communication Provider Configuration, the following behavior is observed:
    • The "+" (Plus) button is disabled, preventing the addition of a new communication provider.
    • The Save button is grayed out, preventing saving changes even for an existing provider.
    • Administrators are unable to add a new CTI provider or modify existing provider configuration.
  • This issue occurs even when the user has administrator privileges.

Environment

  • SAP Service Cloud Version 2 1.0
  • SAP Sales Cloud Version 2 1.0

Reproducing the Issue

  1. Navigate to the settings section of the communication provider configuration in CTI.
  2. Attempt to add a new communication provider.
  3. Observe that the plus button is grayed out and cannot be clicked.
  4. Attempt to save the configuration; observe that the save button is also grayed out.

Cause

  • Missing or incorrect configuration.
  • The tenant is configured to use Change Project transport functionality.
    • In such landscapes:
    • The Target system is read-only for configuration changes.
    • Configuration changes must be performed in the Source system.
  • Changes are then replicated to the Target system through the Change Project transport mechanism.
  • Because of this restriction, configuration actions such as adding or modifying Communication Providers for CTI are disabled in the Target system.

Resolution

Perform the configuration changes in the Source system and transport them to the Target system.

    1. Log in to the Source tenant.
    2. Create a Change Project .
      • Once you create a change project, you can make configuration changes in the Settings page.
      • After making necessary configuration changes, publish and transport your project. 
    3. Navigate to Settings → CTI Configuration
    4. Go to Communication Provider Configuration.
    5. Click the "+" (Plus) button to add the provider or modify the existing configuration.
    6. Save the changes.
    7. Transport the changes to the Target tenant using Change Project. Follow the steps outlined under the Publish, Transport, and Activate Change Project online help document.
