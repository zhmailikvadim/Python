Symptom

The application F0708 stops working after implementation of the frontend note, or any frontend note which has a prerequisite note 3652989.

Environment

As of SAP S/4HANA 2020

Reproducing the Issue

1. Open application F0708 Balance Sheet/Income Statement
2. Run for any parameters
3. No data returned

Cause

If the note 3653370 - Balance Sheet/Income Statement: Company Code is not correctly copied navigating to G/L Account Balances - Backend is not implemented, the application stops working after the frontend note implementation.

Resolution

Implement SAP Note 3653370 - Balance Sheet/Income Statement: Company Code is not correctly copied navigating to G/L Account Balances - Backend

Keywords

F0708, Balance Sheet/Income Statement, no data, stops working, empty result, 3652989, 3653370
