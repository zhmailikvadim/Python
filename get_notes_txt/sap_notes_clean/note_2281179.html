Symptom

When generating any connector run detail report, in the "Completion Report" section it shows the warning message "The connector run has resulted in more than X exceptions getting generated. Please address the issues indicated and re-run the data file."

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

SAP SuccessFactors Learning

Cause

In the connectors configuration ID, the value defined for the flag "connector.report.detail.numberOfExceptionsDisplayed" is smaller than the number of exceptions.

Resolution

Increase the value of the setting "connector.report.detail.numberOfExceptionsDisplayed" in connectors configuration ID.

Steps:

  1. Access Learning Administration Side >
  2. System Administration > Configuration > System Configuration > CONNECTORS >
  3. Search for setting "connector.report.detail.numberOfExceptionsDisplayed" and increase the value.
  4. Note: The default value is 5000
  5. Click on "Apply Changes" >
  6. Generate the connector report again >
  7. It will display the number of records under "Message and Exceptions" section defined in the configuration above.
