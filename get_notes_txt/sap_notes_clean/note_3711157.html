Symptom

While releasing the Outbound Delivery, system throws a 500 Error (Dump).

Environment

SAP Business ByDesign

Reproducing the Issue

  1. Go to Outbound Logistics work center.
  2. Go to Outbound Deliveries view.
  3. Search and Select the Outbound Delivery ABC. (ABC represents the Outbound Delivery ID)
  4. Choose 'Release'.
  5. The system dumps.

Cause

  • The Outbound Delivery is associated with a Project Stock Order.
  • The Project Task which is linked to the Project Stock Order is in 'Closed' status, causing the system to throw a 500 Error (Dump) while releasing the Outbound Delivery.

Resolution

Change the status of relevant Project Task to “Completed” by following the below steps,

  1. Go to Project Management work center.
  2. Go to Projects view.
  3. Search for the Project DEF. (DEF represents the Project ID)
  4. Choose Edit
  5. Under Project Plan tab, go to Gantt Chart sub-tab
  6. Select the Task ID XYZ. (XYZ represents the Task ID)
  7. Change the status of Project Task to "Completed" by selecting 'Revoke Closure' from the dropdown.
  8. After performing these steps Release Outbound Delivery.
