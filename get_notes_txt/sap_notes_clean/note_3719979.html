Symptom

Lifecycle phase changes do not trigger the automation when lifecycle dates are altered

Environment

SAP LeanIX - Automations

Reproducing the Issue

  1. Set up an automation in LeanIX with the 'Lifecycle Phase Is Reached' trigger
  2. Manually alter lifecycle dates within the factsheet so that it changes to the set lifecycle phase.
  3. Observe that the automation does not trigger as expected.

Cause

When lifecycle dates within the factsheet are manually altered, the automation does not trigger because the criteria for the automation are based on lifecycle phase changes, not date changes.

Resolution

  • Set the lifecycle phase date to two days in the future to allow the lifecycle phase to change automatically.
  • Note that lifecycle date changes are not supported with automations and recommend submitting this idea to the product roadmap for future consideration.
