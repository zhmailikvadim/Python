Symptom

Sorting Workspaces by Description doesn't sort alphabetically

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Reproducing the Issue

  1. Once you're in Workzone, click on Explore All Workspaces
  2. Within the table, click on the "Description" label in the header of the table 
  3. Clicking on "Description" does not sort them alphabetically
  4. But if you click on "Name" or "Last Activity", it works as expected

Cause

Root Cause: The description condition is not supported by the backend service.

Resolution

Engineering Team will apply a fix in R2602A Release which is set on March 4, 2026.
