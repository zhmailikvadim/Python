Symptom

When release fails for any order in a PCN, the entire PCN release fails. However, label print orders are not created for the orders that were successfully released before the failure occurred.

Other Terms

ISLS, Intelligent Clinical Supply Management, Manufacturing, Clinical Supply Orders, Validate Clinical Batches and Expiry Dates, Release, Label Print Orders

Reason and Prerequisites

 You use SAP Intelligent Clinical Supply Management or this SAP Note is a prerequisite of another SAP Note.

Solution

 Implement the attached correction.

A report is created which can be accessed via transaction /ctco/label_print. The prerequiste of the report is that order should be released already .Input the process order / purchase order , purchase order item and it will trigger label print process. The status can be viewed in the app - Manage Label Print Orders.
