Symptom

In Payroll Control Center, using report PYT_CFG_CONTAINER_COMPARE to update container with new or delete entry will cause data inconsistence. This should be prevented.

Other Terms

PCC, Payroll Control Center, PYT_CFG_CONTAINER_COMPARE, Update Container

Reason and Prerequisites

New or delete entry will impact the hierarchy in container.

Solution

Implement the corrections in your system or import the relevant Support Package.
