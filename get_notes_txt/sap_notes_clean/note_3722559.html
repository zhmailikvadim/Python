Symptom

In WG22 T-Code unable to assign Characteristics Profile for Material Group due to error Message No. WW030 

Environment

  • SAP ERP Logistics (LO)
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA, on-premise
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

1. T-Code WG22.
2. Enter Material Group
3. Input Characteristic profile field
4. Click on the "Assign Profile" button.
5. Click on the "Display Profile Assignment" button. 
The system displays the error message (Message No. WW030).

Cause

Program Error

Resolution

Implement SAP Note 3709429

See Also

https://me.sap.com/notes/3709429

Keywords

WW030, WG21, WG22
