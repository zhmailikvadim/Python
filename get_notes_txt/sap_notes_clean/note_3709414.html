Symptom

In the process "Termination of Assignment of MOS" (DE_B_GPKE_TERM_OF_ASSGN_MOS), step 0100 (Start Process: Receipt of Information Termination of MOS assignment and Receive Supply Scenario Request) runs into an error with the message "Could not determine a business object using the current strategy group STRGRP_GEN_PROC_MAP" because the BMID BM_INIT_SUSC_READ_REQ was not maintained in table /US4G/PROC_MAP against the Process ID( proc_ext_id - DE_C_GPKE_TERM_OF_ASSGN_MOS and proc_int_id -  DE_B_GPKE_TERM_OF_ASSGN_MOS).

This issue is relevant for the market role: MOS.

Other Terms

GPKE, DE_B_GPKE_TERM_OF_ASSGN_MOS, /US4G/PROC_MAP, STRGRP_GEN_PROC_MAP, BM_INIT_SUSC_READ_REQ, BMID, strategy group, business object, TDoc, Supply Scenario Response

Reason and Prerequisites

The error occurred because table /US4G/PROC_MAP was missing entries for the BMID BM_INIT_SUSC_READ_REQ against the Process ID(  proc_ext_id - DE_C_GPKE_TERM_OF_ASSGN_MOS and proc_int_id -  DE_B_GPKE_TERM_OF_ASSGN_MOS ).

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the system completes step 0100 (Send Supply Scenario Response from Business System) successfully.
Import the BC set /US4G/3_0_SUP_SP07_NOTE_3709414 to add the missing table entries. You can verify the fix by checking that step 0100
completes successfully and that the system receives the TDoc successfully.
