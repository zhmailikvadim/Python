Symptom

  • Header value in some of the analytical cards on the Fiori overview page is displayed as 0.
  • The issue occurs when the Fiori launchpad settings for Language and Region are set to certain Decimal Formats e.g. 1.234.567,89 or 1 234 567,89.
  • The values are displayed correctly when the Decimal Format is set to: 1,234,567.89.
  • UI5 version is 1.120.xx

Environment

  • Fiori elements v2 Overview Page Analytical Cards
  • SAP UI5 version 1.120.xx.

Reproducing the Issue

  1. Open any Overview Page (OVP) with decimal KPI header and target value.
  2. Set different Decimal Format options in the Fiori launchpad settings for Language and Region:
  • Set Decimal Format to 1,234,567.89: Values are displayed correctly.
  • Set Decimal Format to 1.234.567,89 or 1 234 567,89: Values are displayed as 0.

Cause

This is a know issue for UI5 version 1.120.xx.

Resolution

The fix for this issue is available from 1.120.43 onwards, which is pending release.
