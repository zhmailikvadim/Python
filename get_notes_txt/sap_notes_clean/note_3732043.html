Symptom

In process Receipt of Quotation for MOS Billing (SUP View) (DE_B_MOSB_QUOT_SUP), when receiving the "Cloud Information End" information of step 8100, the system raises an error due to a duplicated Malo, this is not correct.

This issue is relevant for the market role: SUP.

Other Terms

Supplier, MOS, MaLo, MOSB_SUP, /US4G/AP_BILLING(521), /US4G/CL_CTR_QUERY, /US4G/IF_CTR_QUERY~SEL_MOSB_QUOTATION_ALL, Cloud Information End Receipt of Quotation for MOS Billing (SUP View) (DE_B_MOSB_QUOT_SUP)

Reason and Prerequisites

Program error.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the process runs successfully without raising errors for duplicated MaLo. You can run the Exception Document with solution "Reprocess the current process step".
You can verify the fix by checking that the process completes without error and the duplicated MaLo error no longer appears.
