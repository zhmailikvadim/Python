Symptom

  • Error message displayed: 'Expected Value' is too high. Please enter a lower value.
  • Message number: MMPUR_SPROC006

Environment

SAP S/4hana Public Cloud Edition

Reproducing the Issue

  1. Attempt to create a limit Purchase Order in app 'create purchase order - advanced' (ME21n) .
  2. Enter an expected value and an overall limit value.
  3. Observe the error message: 'Expected Value' is too high. Please enter a lower value. (MMPUR_SPROC006) 

Cause

  • When entering the limit amounts the system compares the entered limit value with the net price of the item (The price condition value).

Resolution

  1. Maintain the required price for the item in the Purchase Order.
  2. Ensure that the expected value defined in the system does not exceed the price condition value.
  3. If necessary remove the limit item category 'E' then add the net price and re-enter the limit item category.
