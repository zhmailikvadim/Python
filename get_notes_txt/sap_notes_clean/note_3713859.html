Symptom

System upgrade is running with DMO option in SUM(Software Update Manager).

The target HANA database information is required during the procedure.

Environment

SAP ABAP System.

Resolution

The target HANA database must be prepared before the DMO procedure reaches the phase HDB_MIGCONFIG.

In the phase HDB_MIGCONFIG, the target database information is asked and validated by SUM process.

Keywords
