Symptom

Premium beneficiary expenses are not displayed correctly in the transaction "Display Building Savings Contract" (/IBS/BSC_SHOW), while they are displayed correctly in the transaction "Housing Premium - Information" (/IBS/PRM_BDS_OVERVW). This situation occurs for partners with several building savings contracts within an application if premiums are fixed for some but not for all construction savings contracts (of the application).

Reason and Prerequisites

In the display format of the data, the above mentioned constellation is not taken into account correctly.

Solution

The display formatting in the "BO Wop in Building Savings Contract" (/IBS/CL_PRM_BO_BDS_IFM_BSC, method WOP_DATA_FILL) is corrected.
