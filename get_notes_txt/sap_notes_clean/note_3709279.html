Symptom

  • The Branch Code field is not available in the standard upload template in Fiori app “Import Supplier Invoices”.
  • After uploading the completed template, during simulation or review, there is no option to enter or change the Branch Code.
  • As a result, users are unable to select the required branch code for vendors with multiple branches, unlike the behavior in FB60 or MIRO.

Environment

  • SAP S/4HANA
  • SAP S/4HANA Finance

Reproducing the Issue

Use the Fiori app “Import Supplier Invoices” and observe the following:

  • The Branch Code field is not available in the standard upload template.
  • During simulation or review, there is no option to enter or change the Branch Code.

Cause

The Branch Code field is not available in the “Import Supplier Invoices” app, and this is considered standard system behavior.

Resolution

  1. Confirm that the Branch Code field is not available in the “Import Supplier Invoices” app, as this is standard system behavior.
  2. If enabling this functionality is essential for business processes, submit a change request to the product team.
  3. The product team will evaluate the change request and determine whether adjustments to the current design are feasible.

See Also

Refer to SAP Note 11 for guidance on submitting change requests to the product team.
