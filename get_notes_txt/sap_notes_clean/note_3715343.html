Symptom

The screen freezes during the registration process for a large number of users and throws the error message:

  • "An error occurred while processing your request. Reference #221.a8a4c117.1765793677.1127bcba https://errors.edgesuite.net/221.d6d70b17.1769171563.8c21231c"

After refreshing the page, only a few users are successfully registered, while others are not.

Environment

SAP SuccessFactors Learning

Reproducing the Issue

  1. Create a class in Learning Administration side > 
  2. Navigate to the Registration tab >
  3. Add Users > Filter by User ID (Filter Type: Any) >
  4. Select a large amount of users >
  5. Click Add >
  6. Observe that the screen freezes and after some time, an error similar to "An error occurred while processing your request. Reference #221.a8a4c117.1765793677.1127bcba" appears >
  7. Refresh the page and note that only a few users are successfully registered, while others are not.

Cause

This is a limitation in the code causing timeout during the registration process for a large number of users.

Resolution

The Product Managers are aware of this scenario and are studying the possibility of a future product enhancement to the system.

As a workaround, please try one of the following alternatives: 

  • Register users in smaller batches (e.g., approximately 100 users at a time).
  • Alternatively, register all users at once and keep the application session active after the exception occurs.
  • Ensure the session remains active until all registrations are completed or the administrator logs out of the application.
  • If the error persists, refer to the Import Data feature at Learning Administration > System Administration > System Management Tools > Import Data > Entity Type > Registrations for bulk registration.
