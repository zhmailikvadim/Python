Symptom

Some special GL Indicator are not available in Reprocess Bank Statement Items/app ID F1520

Environment

  • Payment Apps

Reproducing the Issue

  1. Open the Reprocess Bank Statement Items app (F1520) and select a bank statement item for processing.
  2. Navigate to the Post on Account tab.
  3. Attempt to choose a Special G/L indicator for the posting.
  4. Observe that certain Special G/L indicators (for example, types A and W) are not available in the selection list.

Cause

Certain Special G/L indicators (types A and W) are intentionally not supported in the Reprocess Bank Statement Items app. The app filters out these types by design; type A is intended for down payments and type W for bills of exchange, which must be handled in dedicated applications.

Resolution

There is no planned development in this area, as there are already KBAs that describe this limitation and state that it is intended to use dedicated applications for these transactions.
