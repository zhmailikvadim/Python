Symptom

The creation of a generating unit of the energy type WINDOFF (wind offshore)  with a commissioning date later than 1/1/2023 terminates with the error message No billing attributes are assigned to operand assignment VGEEGMPSAN (/ENBIL/BI 082). 

Other Terms

Transaction /ENBIL/GU

Reason and Prerequisites

Missing customizing

Solution

Maintain the missing billing attribute for energy type WINDOFF for operand assignment VGEEGMPSAN as follows:

  1. Call transaction SM34.
  2. In the "View cluster" field, enter /ENBIL/VC_ENTYPE and choose "Maintain".
  3. Select the WINDOFF energy type and double-click Compensation Version in the Dialog Structure area.
  4. Select version 7 and double-click Operand Assignment  in the Dialog Structure area.
  5. Select the operand assignment VGEEGMPSAN and double-click Billing Attributes in the Dialog Structure area.
  6. Choose New Entries.
  7. In the Billing Attribute column, use the F4 help to select the characteristic Exceedance of Maximum Duration AFVM and SAP as the configuration source.
  8. Save the change in a transport request and transport it to all your system/clients where this Customizing is required.
