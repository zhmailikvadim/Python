Symptom

Using the CDS Providers Explorer View on Modeling tools, the folders appear in alphabetic order.

Other Terms

Transient Provider, sorting, CDS providers

Reason and Prerequisites

Standardize the folder orders according to the Project Explorer

Solution

The order of the folders will match the Project Explorer view.

QUERY - REP
CALCULATED KEY FIGURE - CKF
RESTRICTED KEY FIGURE - RKF
SELECTION - SEL
STRUCTURE - STR
SELECTION - SOB

To solve the issue, apply the following Support Packages:

  • SAP BW 7.55
    Implement Support Package 12 for SAP BW 7.55 (SAPK-75512INSAPBW) into your BW system. 
     
  • SAP BW 7.56
    Implement Support Package 10 for SAP BW 7.56 (SAPK-75610INSAPBW) into your BW system.
     
  • SAP BW 7.57
    Implement Support Package 08 for SAP BW 7.57 (SAPK-75708INSAPBW) into your BW system.

  • SAP BW 7.58
    Implement Support Package 6 for SAP BW 7.58 (SAPK-75806INSAPBW) into your BW system.
     
  • SAP BW/4HANA 2021
    Import Support Package 14 for SAP BW/4HANA 2021 (SAPK-30014INDW4CORE) into your BW system.
      
  • SAP BW/4HANA 2023
    Import Support Package 9 for SAP BW/4HANA 2023 (SAPK-40009INDW4CORE) into your BW system.
