Symptom

  • Repeating values are displayed in the “Your Municipality” field on the Pennsylvania Residency Certification form.
  • Repeating values are displayed in the “Your Tax Collection District” field on the same form.
  • The issue occurs across multiple browsers and for all users.
  • The issue persists after clearing cache, logging out/in, and restarting the browser.
  • The number of repeated “Your Municipality” values matches the number of imported PA_USCompliance_Municipality picklist versions with different effective dates.
  • No error message is shown.

Environment

 SAP SuccessFactors Onboarding

Reproducing the Issue

  1. Log in as Onboardee.
  2. Navigate to the Pennsylvania Residency Certification form under Compliance tasks.
  3. Scroll to the “Pennsylvania Residency Certification Information” section.
  4. Set “Your County” 
  5. Set “Your Tax Collection District” to UPPER PERKIOMEN S D.
  6. Observe that “HEREFORD TWP” appears multiple times in “Your Municipality”.

Cause

Multiple effective‑dated versions of the PA_USCompliance_Municipality picklist (and related parent picklist) are present in the instance; a product defect causes the form to display combined entries across versions, resulting in duplicates.

Resolution

  1. Temporary mitigation (may reduce or remove duplicates in some instances):
  2. Go to Picklist Center.
  3. Clear the “As of date” filter (leave it blank).
  4. Search for the picklist “PA_USCompliance_Municipality”.
  5. Keep only one version active and deactivate all other versions with different effective dates.
  6. If applicable, repeat the same for the parent picklist used by “Your Tax Collection District”.
  7. Re-test the form.
  8. If duplicates persist after the mitigation:
  9. A product fix is planned (engineering code correction referenced internally as OBX-41268) and is scheduled for the 1H 2026 release.
  10. Monitor release notes and apply the update when available, then retest the form.
  11. Optional post-fix step:
  12. Re-import the mandatory compliance picklists following the referenced KBA to ensure consistency, then validate the form behavior.
