Symptom

Component: Health and Safety
Module: Edit Person Assignment

You maintain person assignments in the App "Edit Person Assignments" (EHHSS_UI_PERSON_ASSIGN_OIF) and search for HR Organizational Positions on the "By Position" tab. The search with fields "Valid to" and "Valid from" behaves inconsistently compared to the search help on the "By Location" tab when you add a position.

Other Terms

Search help, F4 Help, BEGDA, ENDDA

Reason and Prerequisites

A program error in the system causes the symptom.

The relevant correction instructions contain the prerequisites.

Solution

After implementing this SAP Note, the system provides all search options for fields "Valid to" and "Valid from".

Alternatively, you import the corresponding Support Package for the component extension of SAP Environment, Health, and Safety Management or the Feature Package Stack for the SAP S/4HANA system.
