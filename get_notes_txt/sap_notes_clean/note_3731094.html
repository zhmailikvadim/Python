Symptom

The changes delivered as part of SAP Note 3565199 are not fully reflected in the system. As a result, the expected behavior is not achieved in certain scenarios.

Other Terms

  • RPLPBSQ8, RPLPBSQ8_ALV
  • RPLPBSQ8_DATA_ALV, RPLPBSQ8_FORM_ALV, RPLPBSQ8_MAIN_ALV

Reason and Prerequisites

 Some delta changes from SAP Note 3565199 were not correctly included or transported into the affected release.

Solution

 This SAP Note provides the missing corrections and ensures that all intended changes from SAP Note 3565199 are properly implemented.

As a general rule SAP recommends that you implement the Support Package specified for your release. However, if you need to install a solution earlier, use the SAP Note Assistant to implement the correction instruction. You can find more information about the SAP Note Assistant in SAP Support Portal, under SAP Note Assistant.
