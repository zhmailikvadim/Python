Symptom

1. After successful execution of the undo approve action, the subsequent buyoff step is not identified correctly

2. When two different users perform an action on the same step, the step for the next course of action is not identified correctly

3. The validation of the processor is not performed correctly

4. Upon execution of the undo-reject action, buyoff steps fail to revert to their previous status

Other Terms

Manage Buyoffs, Transfer Buyoff, Undo Buyoff

Reason and Prerequisites

Program error

Solution

Implementation of this SAP note will resolve all the listed issues
