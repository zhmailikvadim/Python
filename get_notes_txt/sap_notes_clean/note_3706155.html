Symptom

When you select the Without Qty Posting checkbox in a GST debit memo, you find that the Gross Profit calculation is incorrect.

Reproducing the issue

Prerequisites:

  1. You are working in India localization.
  2. Go to Main Menu → Administration → System Initialization → Document Settings → General tab and select Calculate Gross Profit.
  3. Go to Main Menu → Administration → System Initialization → Document Settings → General tab and make sure that Base Price Origin is set to Item Cost.
  4. Go to Main Menu → Administration → System Initialization → Document Settings → General tab and make sure that the Calculate % Gross Profit field is set to Profit/Sales Price. 

Scenario:

  1. Create a GST debit memo and select an inventory item with non-zero Item Cost.
  2. Check that Gross Profit is calculated.
  3. Select the Without Qty Posting checkbox.
  4. Add the document.

Expected behavior:

The base price for the Gross Profit calculation is set to zero.

Actual behavior:

The base price for the Gross Profit calculation is not set to zero.
