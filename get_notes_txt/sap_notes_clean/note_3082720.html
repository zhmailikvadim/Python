Symptom

Module: EH&S WWI, EH&S GLM

You use the "SAP EH&S WWI HVP" printer driver.

You use a vertical text field (text box).

In the bar code parameters, you use the ROTATE parameter to rotate the bar code 90° or 270°.

However, the bar code is rotated twice (180°) and may appear upside down or truncated.

Other Terms

﻿EHSWWISP54﻿

EHS EH&S WWI GLM GLM+ GLMplus EHS_SERBC EHS_SERLB

Reason and Prerequisites

Reason:
There is a program error in hvpDisp.dll.

Prerequisites:
EH&S WWI, SP49-SP53 (SAP Note 2751048)

Solution

A correction is available with EH&S WWI 3.2 SP54. Apply this patch or a more recent patch. SAP Note 568302 provides information about the availability of patches, as well as information about storing, downloading, and applying them.

The following files were changed:
hvpDisp.dll, Version 1.3.55 from 7/1/2021 or newer.
