Symptom

Reading multiple entries from Object Relations using the persistency class consumes too much memory and reduces the performance of reading.

This slows down the archiving processing, for example of archiving object /PE1/ORD2. The reading from Object Relations causes long time running of job, which also affects other normal processes.

Other Terms

/PE1/CL_BPE_D_OBJECT_RELATION

Reason and Prerequisites

Performance issue.

Solution

The binary search is applied to improve the performance.

Please implement the provided Correction or the corresponding Support Package.
