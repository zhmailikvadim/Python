Symptom

As described in SAP ONE Support Launchpad – Regulatory Change Manager, this SAP Note provides you with the relevant system changes to address the new minimum contribution amounts, presented by the Orden PJC/297/2026, de 30 de marzo, por la que se desarrollan las normas legales de cotización a la Seguridad Social, desempleo, protección por cese de actividad, Fondo de Garantía Salarial y formación profesional published on March 31, 2026.

For more information, see Orden PJC/297/2026 and RED 5/2026. Check the disclaimers section below regarding these external websites.
