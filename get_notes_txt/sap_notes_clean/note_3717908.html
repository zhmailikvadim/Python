Symptom

Mapping for various cost obejcts are missing for Travel Request and Expense Report Obligation when posting these documents via SAP Concur PSM. Following additional cost objects are supported with the implementation of this note:

- Funds Center

- Commitment Item

- Internal Order

- Maintenance Order

- Production Order

- QM Order

Other Terms

 ICS, Concur, Travel Request, Expense Report Obligation, PSM

Solution

In case your system is on 2023 FPS3 or lowever, please check the pre-requisite note 3566936 and implement the mentioned pre-requisite notes accordingly before applying this note.
