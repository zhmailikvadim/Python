Symptom

Please note that this SAP note is intended for customers who have upgraded their S/4HANA system using the S4CORE TCI note.

The dynamic forms pre-processing event handler /merp/cl_sdf_evt_msrment_read uses the method initialize_evt_entity_data to fetch event data. During this event, it changes the event entity model depending on the configuration and the object the form is assigned to.

However, this change to the event entity model is not reset after each call of initialize_evt_entity_data, which means the event entity model may be incorrect after the first run of the method. This causes incorrect pre-processing results for the EQUIPMENT_TO_FORM and FUNCLOC_TO_FORM entitites.

Other Terms

S4MOBADDON-8445, MAIF, SAP Dynamic Forms, Pre-processing, Equipment, Functional Location, Event, TCI

Reason and Prerequisites

Programming Error

Solution

Please implement the attached correction instructions
