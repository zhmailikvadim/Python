Symptom

You run the Libro de Sueldos Digital (RPC_PAYAR_F931) report for multiple biweekly employees:

  • One of the given employees was hired in the middle of the period dates for which the report is run.

In the scenario described above, the system generates multiple files when it should generate only one.

Other Terms

&LIBRODIGITAL - For further information, see SAP Note 1152671 - CONSULT: List of tags for Argentina notes

PC00_M29_F931

Reason and Prerequisites

Reason

This issue is a side effect caused by the solution delivered by SAP Note 3663906 - [AR] F931 Split of files for hired employees.

Prerequisites

The following Support Package levels are the minimum prerequisites:

Release Support Package
S4HCM 102 HR SP 00
S4HCM 101 HR SP 04
S4HCM 100 HR SP 17
SAP_HR 608 HR SP C8
SAP_HR 604 (EhP6 e EhP7) HR SP K0

For more information, access the SAP ERP Human Capital Management - Support Package Schedules.

When you install an SAP Note, some prerequisites might be required. The prerequisites are SAP Notes that have already been released and that change the same context of the current one. Then, even if these prerequisites are not necessarily related to the symptom of this SAP Note, you must implement them due to technical reasons.

Solution

From now on, the system will correctly generate one file in the scenario described above.
