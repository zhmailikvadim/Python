When we execute the purchasing document preprocessor program RM06EV70 for a purchase order with more than one line item, goods receipt and invoice documents are created for all -error 'Invoiced quantity not the same as GR quantity' is thrown.

Other Terms

RM06EV70, EKBE

Reason and Prerequisites

The logic introduced in note  2071064 did not consider building the entries in the purchase order history table for the right purchase order item number.

Solution

The logic is modified to consider the position number of the purchase order item.
