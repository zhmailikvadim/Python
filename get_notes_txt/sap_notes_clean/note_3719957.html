Symptom

We would like to confirm:

  1. Whether there is any supported enhancement or configuration that allows inclusion of custom SuccessFactors tokens in the IAS activation email.

  2. Whether additional flexibility is planned regarding subject line and sender name configuration.

Environment

SAP Success factors onboarding

Resolution

  1. Regarding the inclusion of custom SuccessFactors tokens in IAS activation emails, this feature is currently not supported. However, if this functionality is critical for your requirements, you have the option to raise an enhancement request for consideration.

     2. Concerning the configuration of the subject line and sender details in IAS emails, I am pleased to confirm that these elements can be customized.
         You can find detailed instructions on how to configure email templates by visiting the following link:

https://help.sap.com/docs/cloud-identity-services/cloud-identity-services/configuring-e-mail-templates?locale=en-US&version=LATEST
         

This guide provides step-by-step information to assist you with the setup.
