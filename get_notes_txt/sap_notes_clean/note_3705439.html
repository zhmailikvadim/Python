Symptom

The Fiori app “Create Maintenance Plan”, as well as other Fiori apps such as “Manage Maintenance Plan” , do not allow the creation of Maintenance Plan SM. 

For example, the “Create Maintenance Plan” Fiori app displays the following plan categories:

while the SAPGUI transaction IP01 gives all values

Environment

SAP S/4HANA

Reproducing the Issue

  • Create a " Service with Advance Execution" maintenance plan in FIORI using Fiori Apss "Create Maintenance Plan" : Maintenance Plan category SM is not available.
  • Create a "Service with Advance Execution"  in SAPGUI transaction IP01 : Maintenance Plan Category SM is available

Cause

Missing feature

Resolution

The SM maintenance category is only supported in SAP GUI (transaction IP01)  not supported in Webdynpro Application ("Create Maintenance Plan" or FIORI applications ("Manage maintenance Plan")

Keywords

IP01, F5325 , CRM, Services
