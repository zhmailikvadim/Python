Symptom

  • Invalid entry is displayed when attempting to use the 'Project Manager' filter in the app 'Project Financial Booklet – Professional Services'.
  • Error message indicates 'Invalid entry' for the value defined in the Dimensions column.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Open the app 'Project Financial Booklet – Professional Services'.
  2. Attempt to use the 'Project Manager' filter by selecting a value in the Dimensions column.
  3. Observe that the filter displays 'Invalid entry' for the selected value.

Cause

In the Page Filters, the parameters selected under 'Variables' are utilized to display data within the app. 
These selections serve as the foundation for filtering and organizing the displayed information.  

Additionally, the 'Dimensions' column filters values based on the data defined by the selected 'Variables'.
If a value specified in the Dimensions column does not exist within the selected Variables, this results in an "Invalid entry" message.
This behavior is intentional and aligns with the system's design.

Resolution

Ensure that the data selected in the 'Variables' aligns with the values intended to be filtered in the 'Dimensions' column. Adjust the filter criteria to match the selected data in the 'Variables' to avoid invalid entries.
