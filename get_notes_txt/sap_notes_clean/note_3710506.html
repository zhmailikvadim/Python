Symptom

Users attempted to search for documents in CV04N using special characters such as ','>, @, + - '. However, no results were displayed because the search string was not processed correctly.

Other Terms

SF_SPECIALCHAR_DELETE, 'RESPECT_HYPHEN'

Reason and Prerequisites

String contains a delimiter that was not properly handled during manipulation.
Only '-' should not be used as a delimiter because it can indicate the continuation of a string.

Latest version of SAP Note 3259577 is pre-requisite for this Note.

Solution

 Please implement the corrections in this SAP note and implement the latest version pre-requisite note 3259577
