Symptom

Skills/Competencies do not appear as an option when creating a Development Goal using the option "Create from Interests" available in AI-Assisted Goal Creation

Environment

SAP SuccessFactors Career Development Planning 

SAP SuccessFactors Talent Intelligence Hub 

Reproducing the Issue

  1. Proxy as the user experiencing the issue.
  2. Navigate to the "Goals" section from the dropdown menu.
  3. Select "Development"
  4. Go to the Development Plan and click "Create Goal"
  5. Select the option "Create from interests".
  6. Observe that only one attribute type appear, and skills/competencies are not available as an option.

Cause

Translations not configure for the affected attribute type in Talent Intelligence Hub.

Resolution

You need to add the translations to all the locales for the affected attribute type. You can do this by following the steps below:

  1. Go to Talent Intelligence Hub> Attribute Types
  2. Select Skill/Competency attribute type.
  3. Click Edit attribute type
  4. Click on the globe icon
  5. Add a name for all the locales listed there.
  6. Click save.

After this both the Competency and Skills attributes should be available during the Development Goal creation.
