Symptom

When accessing trend portlet of a user with multiple employments in the latest people profile, below error message occurs:

 Looks like something went wrong. Please try again. If the issue persists, contact your system administrator.

Environment

SAP SuccessFactors Performance Management

Cause

This is a known issue and the error occurs under these conditions:

  1. The latest people profile is enabled.
  2. The subject user (whose profile is being viewed) has multiple employments.
  3. The accessing user has permission for the trend element of only one of the subject user’s employments, but not for the others.

Resolution

The fix for this issue is planned to be deployed on 1H 2026 release.

For release timelines, please review SAP SuccessFactors Product Release & Road Map Information.

Workaround

Grant the accessing user permission for the trend element of all employments of the subject user.
