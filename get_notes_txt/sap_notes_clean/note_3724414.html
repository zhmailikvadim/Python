Symptom

  • The query concerns the presence of a mandatory Bill Rate field on a Fixed Price Statement of Work (SOW)
  • The 'Presented' amount field is mandatory for suppliers when adding SOW workers
  • Suppliers are required to enter a Rate despite the configuration settings indicating "Allow rates on SOW Worker roles without invoicing" is set to "No"

Cause

As designed

Resolution

The Rate field is mandatory and the 'Presented' field is displayed based on the following conditions:

  • Rate schedule should not be enabled
  • Time sheet should be enabled

Additionally, one of the following conditions must be met:

  • Rate grid should be enabled
  • Allow team payment flag should be enabled

This is a designed behavior When the Rate Grid is enabled, the 'Presented' field is present and mandatory
