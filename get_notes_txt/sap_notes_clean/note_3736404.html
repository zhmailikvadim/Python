Symptom

Request to check any reports (CDS views) for item texts in Purchase Order.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

Cause

Currently this is a Missing functionality.

There's a existed request in Customer Influence Portal can be voted: Improvement Request - Customer Influence

Resolution

There're two workarounds:

  1. Use ODATA API to GET PO item text: /sap/opu/odata4/sap/api_purchaseorder_2/srvd_a2x/sap/purchaseorder/0001/PurchaseOrderItem(PurchaseOrder='XXX',PurchaseOrderItem='XX')/_PurchaseOrderItemNote
  2. Use a custom output form to display the item text in the output: FIELD TYPE 2: STANDARD FIELD AVAILABLE ON FORM: S/4HANA Cloud Output Management - Purchase Order - SAP Community. Before proceeding, follow the KBA 2900438 - (Missing Purchase Order Header/Item Text in Output Forms) to ensure that the necessary text in the item is successfully maintained in SSCUI 102270 "Define Texts for Purchase Order."
