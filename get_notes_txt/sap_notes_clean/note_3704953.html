Symptom

When trying to add an attachment to a Sales Quote via the Server Side Add-In via Outlook, the "Attach" option is missing.

Environment

SAP Cloud for Customer

Reproducing the Issue

  1. Open Outlook.
  2. Select an email with attachments.
  3. Enable the Server Side Add-In by clicking on "View Context of SAP Cloud for Customer".
  4. Search for a Sales Quote.
  5. Observe that there is no option to attach any attachment to Sales Quotes.

Cause

This is a current limitation of the Server-Side Outlook Add-In functionality.

Resolution

  1. Submit an enhancement request to product management to address the limitation. KBA 2531650 - New Customer Request for Additional Feature or Function in SAP standard software
  2. Or as a workaround:
  • Search for the relevant Sales Quote in the server-side add-in.
  • Click on the three dots in the top-right corner ("...").
  • Select "Open in SAP".
  • Add the relevant attachment to the Sales Quote directly in the SAP Cloud for Customer system.
