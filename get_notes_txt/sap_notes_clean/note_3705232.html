Symptom

There is a need to confirm if SMTP port 25/TCP is required while setting up inbound functionality for SAPConnect SOIN queue.

Environment

  • SAPconnect;
  • SAP AS Netweaver ABAP.

Resolution

To confirm whether SMTP port 25/TCP is required or not, go to T-code SMICM -> Goto -> Services, and confirm which SMTP port is defined for SAPconnect, following image is an example where SMTP port that will be used is 25000.
