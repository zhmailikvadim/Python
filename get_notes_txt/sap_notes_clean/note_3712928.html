Symptom

Is SAP BW/4HANA supported with FIM 10 application ?

Environment

SAP Financial Information Management ( FIM ) 10.0

Resolution

  • Data Services 4.2 SP12 patch03 and later supports BW/4HANA 2.0 and later.
  • As the FIM application is based on Data Services repositories supportability, BW/4HANA 2.0 is supported by FIM.
  • The development team did not test BW/4HANA with FIM. However, since this scenario is supported by DS, it can be customized within DS, which is a fully supported use case.
