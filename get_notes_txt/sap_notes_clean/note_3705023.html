Symptom

Symptom

User executes a Depreciation Simulation report (Transaction AR18N / report RASIMU_ALV01) in current year with simulation variant.

The processing takes long time or ends with TIMEOUT or memory DUMP.

Other Terms

RASIMU_ALV01, TIME_OUT, performance, MT_INSTANCE_BUFFER, AR18N, SYSTEM_NO_ROLL, DUMP, memory

Reason and Prerequisites

Program error.

In some scenarios, the MT_INSTANCE_BUFFER is not reset, causing performance issues or memory dump for high number of processed assets.

Solution

Implement the attached correction instruction or upgrade to the corresponding Support Package level.

Other Terms

HRSK, PDF, HR_SK_HSKCDPRZ, HR_SK_HSKCDPP0, HR_SK_HSKCDPP0_V23, Report HSKCDPP0.

Reason and Prerequisites

This SAP Note is relevant only for Slovakia (HRSK).

Solution

This SAP Note delivers new PDF forms described in the chapter Symptom.

Tax certificate for 2026:

- A new line 8a has been added to the form. It captures the previous year’s amount for the children’s sports benefit (wage type M570).
- In accordance with the Legal Change on levy deductibles for trainers, the amount and the number of months of levy deductibles for trainers are displayed on the new line 10.

Tax payment confirmation for 2025:

- According to Directed portion of paid income tax Legal Change, the two new fields 11 and 12 are created.
