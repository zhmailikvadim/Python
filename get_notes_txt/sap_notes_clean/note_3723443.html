Symptom

In the Manage Balance Validation Rules and Groups app, when you import data from an Excel file to a target rule group, and the imported data belongs to a different rule group with a different ID, the system deletes the target rule group without any notification.

Other Terms

Manage Balance Validation Rules and Groups, Upload banlance validaiton rule group,  Import

Reason and Prerequisites

Importing incompatible rule group data to the target rule group is not allowed. The system is expected to raise an exception message in such cases.

Solution

 Implement this Note to the back-end system.
