Symptom

After an upgrade to S4CORE 109, when attempting to print the inspection instruction, no printout is produced and no spool request is generated.

Other Terms

output management, configuration

Reason and Prerequisites

Incorrectly delivered setting of the table APOC_I_OBJ_TYPE. As of S4CORE 109, these settings can no longer be edited by customers in the view APOC_I_OBJ_TYPEV. Therefore, this correction must be carried out using the attached report Z_APOC_INSPECTION_LOT_CORRECT.

Solution

Run the attached correction report
