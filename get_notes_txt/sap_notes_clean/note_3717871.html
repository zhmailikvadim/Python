Symptom

  • The background job FCLM_FLOWBUILDER_JOB is repeatedly cancelled in the SAP system.
  • Error message: "Error in processing of the output parameter "ET_EX" (Message no. 1, Message type: E)

Environment

  • SAP S/4HANA 1709
  • SAP S/4HANA 1809
  • SAP S/4HANA 1909
  • SAP S/4HANA 2020

Reproducing the Issue

  1. Start the background job FCLM_FLOWBUILDER_JOB in the system.
  2. Observe the error message: "Error in processing of the output parameter 'ET_EX'" (Message no. 1, Message type: E).

Cause

Programming error

Resolution

  1. Ensure that Flow Builder notes are implemented as per the system release status. Refer to SAP Note 2530138 -> Resolution section.
  2. Regenerate the loading class in transaction FCLM_FB_UTIL as per SAP Note 2369432.
  3. Use customer namespace (Z*) for FI & MM loading classes as recommended in SAP Note 2369432.
  4. Retest the scenario after implementing the above steps.
