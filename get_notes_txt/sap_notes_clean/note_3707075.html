Symptom

  • The user name filter search button in 360 Executive Review is unresponsive.
  • Users can still be searched manually by text.
  • Other filters on the same screen are functioning correctly.

Environment

SAP SuccessFactors 360 Reviews 

Reproducing the Issue

  1. Navigate to Forms > 360 Executive Review.
  2. Click on the button next to the "user name:" input field.
  3. Observe that the button does not bring up a screen with users.
  4. Confirm that other filters on the screen are working correctly.

Cause

This is a Known Issue.

Resolution

This issue will be corrected in b2605 (1H 2026 release) with the following rollout schedule:

  • Preview: April 13

  • Production: May 15–16

For release timelines, please review SAP SuccessFactors Product Release & Road Map Information.
