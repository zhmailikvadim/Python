Symptom

Navigation bar shows in OMP Open Job Posting detail page in iOS and Android app, which is unexpected

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

  • SAP SuccessFactors Mobile
  • SAP SuccessFactors Opportunity Marketplace

Reproducing the Issue

  1. Log in to the Mobile App (iOS example)
  2. Navigate to the Opportunity Marketplace (Menu: Opportunities)
  3. Click to view detail of a Open Job Posting from “Discover Open Job” section
  4. User will see the following page, with the navigation bar showing up

Resolution

This is unexpected from requirement perspective. However, since the page is integrated with a hybrid page, it cannot be removed by Mobile Native code.

SAP Engineering team is planning to fix this in a future release. However, there is no ETA for removing it at this moment.

Please monitor this KBA for updates on potential dates for a fix on this issue.
