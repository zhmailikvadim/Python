Symptom

  • On the Windows application, the effort.task field is not mandatory;
  • The same field is required on the other mobile applications;

Environment

FSM Mobile - Windows

Reproducing the Issue

  1. Open the Windows application.
  2. Navigate to "Efforts".
  3. Click on "Create".
  4. Observe that the "Task" field is not mandatory.
  5. Repeat the same steps on other mobile applications.
  6. Notice that the same field is mandatory in those applications.

Cause

On the screen configuration, the "Requiredness" of the effort.task field is set to "Default". The default behavior is different on Windows from the other mobile platforms. On Windows, the default behavior is optional, while on the other mobile applications, is mandatory.

Resolution

Change the "Requiredness" of the effort.task field to "Required". This will ensure that this field will be mandatory on all applications.
