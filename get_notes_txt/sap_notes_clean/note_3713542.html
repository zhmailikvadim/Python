Symptom

While closing a situation instance in My Situations-Extended application, a shortdump in class CL_SIT2_DCTX_CAPTURER, method PROCESS_INSTANCES may occur.
This shortdump prevents closing the instance.

Other Terms

Situation Handling - Extended framework ; My Situations - Extended

Reason and Prerequisites

Prerequisites :

1. You are using the extended framework of Situation Handling.

2. You are using situation types where data context capturing is enabled.

Reason : The shortdump occures only if the situation type to which the situation instance belongs has been deleted before closing all instances. This leads to inconsistent

type information at the time of capturing of the data context and this causes the shortdump.

Solution

Please apply the attached correction which adds a check to skip processing the situation type in such cases and thus, prevents the shortdump.
