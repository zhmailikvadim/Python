Symptom

  • Error: The basket content could not be read. The following exception occured: Unexpected char 60 at (line no=1, column no=1, offset=0)
  • Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

SAP Download Manager

Reproducing the Issue

  1. Open the Download Manager application 
  2. Enter the S-user ID and the corresponding password.
  3. Notice an error at the bottom of the screen is displayed: "The basket content could not be read. The following exception occured: Unexpected char 60 at (line no=1, column no=1, offset=0)

Cause

A performance issue may occur within the download when a large number of files (over 1,000) are added to the Download Basket. 

Resolution

Add or download files in batches of no more than 500-700 files at a time.

If the issue persists, create a support case for XX-SER-SWFL-FCMS and include the following:

  • The error message and/or screenshot of the error.
  • The user ID that experiences the issue
  • Attach the Download Manager trace files as per the steps explained in KBA 2662384.

SAP will fix the issue so users can log into Download Manager without any issues.
