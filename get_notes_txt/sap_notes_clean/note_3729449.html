Symptom

In the process Sending of Deregistration Due to Supply Shutdown (DSO View) (DE_B_GPKE_DEREG_DSO), when a device ended before the deregistration process key date, the device end date was not sent to the supplier. This resulted in an internal APERAK because of the missing DTM+206 field.

This issue is relevant for the market role: DSO.

Other Terms

UTILMD 55007, Device End Date, Deregistration, DTM+206, APERAK, Process Key Date, Device UBD Data, Valid To Date

Reason and Prerequisites

The error occurred because the system used the process key date to query active devices instead of the device end date. This caused issues when the device had already ended by the process key date.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the system fetches the active device based on the device end date. You can verify the fix by triggering the Deregistration process and checking that you receive the device component in the UBD data with the DTM+206 field populated with the "Valid To" date for the device.
