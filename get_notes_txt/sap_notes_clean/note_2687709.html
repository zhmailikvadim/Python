Symptom

You are using the latest SAP Identity Management Developer Studio plugins. You get the following warning in the Eclipse Error Log:

Removing part descriptor with the 'com.sap.idm.ic.rcp.views.JobsTabularView' id and the 'Jobs' description. Points to the invalid 'bundleclass://org.eclipse.ui.workbench/org.eclipse.ui.internal.e4.compatibility.CompatibilityView' class.

Other Terms

IDM, SAP Identity Management, Eclipse, plugin, JobsTabularView, warning

Reason and Prerequisites

You are using SAP Identity Management 8.0 SP06

Solution

Apply the patch for SAP Identity Management Developer Studio available for download at https://tools.hana.ondemand.com/#idm. Follow the installation instructions.
