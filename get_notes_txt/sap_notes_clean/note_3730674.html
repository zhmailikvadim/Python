Symptom

In the Perform Maintenance Job app (F5104A), on the Components and Services tab under the Planned Components part, after clicking the Return button, entering a decimal value in the Returned Quantity field is not possible. The field accepts only whole numbers through the step input control. The behavior occurs consistently when returning goods.

Other Terms

F5104A, PM-FIO-WOC-JC, Perform Maintenance Job, StepInput, Returned Quantity, decimals

Reason and Prerequisites

The reason is missing functionality. The input configuration for Returned Quantity uses a whole-number step that prevents decimal entry.

Solution

Install this SAP Note. The correction adjusts the input configuration to allow decimal entry with a defined step and display precision in app F5104A. No additional steps are required.
