Symptom

Changing the color of font or tiles in matrix grid or in Theme Manager, the font or tile color will not change in the Dashboard or Executive review.

Environment

SAP SuccessFactors Calibration

Cause

Expected behaviour as per current system design. Dashboard tiles and font colour cannot be changed. Tiles will appear in blue colour by default. 

Resolution

You may submit an enhancement request as outlined in the KBA. 2090228 - How to Submit Ideas for SAP SuccessFactors Products

See Also

2090228 - How to Submit Ideas for SAP SuccessFactors Products

Keywords

Calibration, calibration module, dashboard view, color change, blue color, enhancement request, fixed color, customization, calibration sessions, default color.
