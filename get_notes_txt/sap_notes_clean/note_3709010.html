Symptom

You use report Payroll Simulation (NL) (RPCNETN0) to simulate the effects of master data changes on the net income of an employee. Your employee is hired in the middle of the Payroll Period selected on the selection screen. The system displays the error message 'Hire Date Not Known for Legal Person xxxx' in the payroll function Store Deduction Data in Wage Return (NLLA0). The message is irrelevant and should not be displayed.

Other Terms

 RPCNETN0, PC00_M05_CNET

Solution

Delivery

Import the HR Support Package relevant for your release. For more information, see SAP ERP Human Capital Management Support Package Schedules or SAP S4 HCM Support Package Schedules.

SAP recommends that you implement the Support Package. However, if you need to install a solution earlier, use the SAP Note Assistant.
