Symptom

When you process a customs import declaration, the system displays data for a reconciliation summary on the "Transportation" tab, even though a reconciliation summary is not intended in this document context.

Other Terms

/SAPSLL/SAPLCUHD_LOAD_IFACE, /SAPSLL/CUHD_LOAD_0120_S-INCCY, 0120, reconciliation summary

Reason and Prerequisites

The problem is caused by a program error.

Solution

Implement the correction instructions that are relevant for your release.

Other Terms

QM_FMEA

Reason and Prerequisites

There is a program error.

Solution

Implement the correction instructions.

 

After the implementation of the correction, the system displays the status "Rejected".
