Symptom

This service request may only be provided if the target tenant lacks any admin user.

Environment

  • SAP Sales Performance Management
  • Advanced Workflow

Resolution

  1. If the environment already has an admin user, this request should be conducted as self service, as described in:
    SAP For Me: 3484881 - How to provide administrator access to a user - SAP for Me 
  2. If the environment has no admin users, open a case for SAP Support containing the following information:
    1. Tenant
    2. Login name 
    3. E-mail
