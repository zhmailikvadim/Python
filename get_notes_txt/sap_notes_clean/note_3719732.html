Symptom

  • This article provides comprehensive details on the Data Extraction Process for tenant decommissioning in SAP Agent Connection. 
  • It includes step by step instructions for customers to request data extraction and outlines the necessary procedures to ensure compliance with cloud off boarding and termination requirements.

Environment

  • SAP Sales Performance Management
  • Agent Connection

Cause

Contract termination, tenant decommissioning.

Resolution

  1. Agent Connection (AX) is not a repository; all data resides in integrated environments such as Agent Performance Management (APM).
  2. Follow the outlined documentation of other applications to raise a data extraction request.
