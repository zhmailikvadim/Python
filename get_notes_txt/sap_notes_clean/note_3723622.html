Symptom

Reprocessing Bank Statements with FEB_BSPROC, you get the error message SG105 about a missing currency exchage rate.

Other Terms

FEB_BSPROC_DETAIL_FE-FWAER

Solution

Note 3337174 solves the same issue if the currency of the bank account is the same as the currency of the company code. Otherwise please implement the code changes of this note via SNOTE of Support Package.
