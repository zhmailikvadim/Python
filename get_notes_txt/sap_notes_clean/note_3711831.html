Symptom

Transaction PAAH.

If you are using SAP GUI for Windows, after click on button "Persons along organizational structure"  the popup "Choose Organizational unit" will open on top of the screen.

In SAP GUI for Java the popup is not visible, it will only be displayed once you resize the window or press F3(Back) several times.

Other Terms

RH_TYPE_STRUC_HELP, macOS

Reason and Prerequisites

Function RH_TYPE_STRUC_HELP uses a call of obsolete function WS_QUERY.

Solution

Apply attached correction instruction or import corresponding support package.
