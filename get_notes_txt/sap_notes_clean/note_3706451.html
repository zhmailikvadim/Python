Symptom

You want to update/upgrade the Java stack/SP of your SAP Solution Manager 7.2 system, but not the ABAP or ST SPs.

Other Terms

Upgrade 7.40 to 7.50 for SAP Solution Manager dual stack, Update of NW Java 7.50 SP for SAP Solution Manager dual stack

Reason and Prerequisites

Your Support Package level /release of the NW Java stack of your SAP Solution Manager 7.2 is outdated and/or older than 2 years.

Solution

As of March 2026, it is possible for all SAP Solution Manager 7.2 systems based on ST SP03 and higher to independently update/upgrade to NW Java 7.50 release/latest available SP.
