Symptom

Unable to change IT0009 due to error message 'Change in the payroll past not possible. Message No. PG710'.
"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Environment

  • Human Capital Management (HCM)
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for HANA
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition
  • SAP SuccessFactors Employee Central Payroll

Reproducing the Issue

  1. Payroll area 'J0' is set to 'Exit Payroll' status for payroll period 2025.09.
  2. Run t-code PA30 to change the following IT0009.
  3. Observe that error message PG710 is triggered: "Change in the payroll past not possible".

Cause

  • 'Reaction to Entry in Payroll Past' (V_T582A-EPAST) is set to 'E' for IT0009 in V_T582A. 
  • The error occurs if the data change date falls in the past payroll.

Resolution

Changing IT0009 for a past payroll period is not permitted, as this is a standard system behavior designed to prevent financial discrepancies and compliance issues. Please ensure that IT0009 is maintained in accordance with the system's requirements.
