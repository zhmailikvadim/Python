Symptom

A new integration needs to be developed using the SAP Cloud Integration Kafka adapter with a requirement to leverage Schema Registry functionality. 

Environment

  • SAP Cloud Integration
  • BTP Integration Suite

Resolution

  • The Schema Registry functionality is currently not supported by the standard Kafka adapter in SAP Cloud Integration.
  • For more details, refer to the SAP Community blog: Cloud Integration - What You Need to Know About the Kafka Adapter - SAP Community.
  • If this capability is required, submit a feature enhancement request via the SAP Influence Portal by following the instructions outlined in KBA 3078373 – How to Submit Ideas for Cloud Integration or SAP Integration Suite.

Keywords

schema registry, kafka adapter limitation, sap integration suite, cloud integration.
