Symptom

A talent flag (for example, businessSegment) is not available as a filter in the Talent Pools page configuration. 

Environment

SAP SuccessFactors Succession & Development

Reproducing the Issue

  1. Try to add a talent flag (for example, ID: businessSegment) as a filter in the Talent Pools page configuration.
  2. Observe that the talent flag does not appear in the list of available filters.

Cause

Not all talent flags are supported for use as filters in the Talent Pools page configuration.

Resolution

  • It is confirmed that some talent flags, such as the businessSegment standard field, cannot be added as filters in Talent Pools due to system limitations.
  • Only fields listed in Manage Talent Pool Field Settings are available for selection as filters; all other fields cannot be added.
