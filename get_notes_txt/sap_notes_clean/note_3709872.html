Symptom

  • Unable to assign G/L accounts to the Profit and Loss result node in the FSV within the Manage Global Hierarchies application.
  • The option to add G/L accounts and nodes for P&L is not visible in the application.

Environment

SAP S/4HANA Cloud Public edition

Reproducing the Issue

  1. Access the Manage Global Hierarchies application.
  2. Attempt to create a Financial Statement Version (FSV).
  3. Create a Balance Sheet and add the G/L accounts successfully.
  4. Attempt to add G/L accounts and nodes for the P&L Result node.
  5. Observe that the option to add G/L accounts to the P&L Result node is not available.

Cause

It is expected system behavior that G/L accounts cannot be directly assigned to node 5 (P&L Result).

Resolution

Node 5 (P&L Result) is a special, non-editable results node whose content and balance are automatically calculated by the system.

Manual editing and assignment of G/L accounts is only possible for the following special nodes:

  • Assets
  • Liabilities and Equity
  • Financial Statement Notes
