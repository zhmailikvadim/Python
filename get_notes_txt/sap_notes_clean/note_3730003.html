Symptom

  • Delivery address is greyed out and cannot be changed in the Purchase Requisition automatically created from the Sales Order.
  • The issue occurs with custom Schedule Line Category and Item Category based on settings in VOV6.
  • In ME52N, the delivery address field is not editable, while it can be changed after converting the PR to a Purchase Order (PO).
  • The requirement is to enable changing the delivery address at PR level.

Environment

  • Materials Management (MM)
  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA

Reproducing the Issue

  1. Create a Sales Order with a specific Item Category and Schedule Line Category.
  2. Allow the system to automatically generate a Purchase Requisition at the schedule line level.
  3. Open the PR in ME52N and observe that the delivery address field is greyed out and not editable.

Cause

  • By design: PRs created with sales origin (creation indicator ESTKZ = “V”) use the Sales Order as the lead document. For third-party items, the delivery address in the PR is derived from the Sales Order ship-to party and is not changeable in the PR. For non–third-party items, the address defaults from the plant and is not connected to the Sales Order ship-to address; therefore, it is not changeable in the PR.

Resolution

  • In MM if the creation indicator of a PR has a sales origin (ESTKZ = 'V') we do not offer the possibility to change the delivery address in MM, because the sales order is the lead document. This is the case for any purchase requisition created from a sales order it is not possible to change the delivery address by design. 
  • When dealing with a real 3rd party case (Item category 'S') the delivery address in the PR should be that of the ship to party in sales. This 3rd party process is driven by item category CB1 and schedule line WS. In this 3rd party case, sales send the customer number to the purchase requisition and MM take the address from the customer number. A user can change the ship to party address in the sales order and it will update the delivery address in the PR directly as a result. It is not possible to change the delivery address from the PR.
