Symptom

You connect PCo to an OSISoft PI system via an Asset Framework source system. After browsing tags for a tag-based notification scenario that are defined as attributes in the PI system, the resulting subscription items have data types that PCo does not support, such as: OSIsoft.AF.Asset.AFAttribute, OSIsoft.AF.Asset.AFElement, OSIsoft.AF.Asset.AFEnumerationValue, or OSIsoft.AF.Asset.AFFile. If you define output expressions that use the subscription elements, and then want to start the agent instance, PCo displays conversion errors.

Other Terms

OSiSoft PI AF

Reason and Prerequisites

Program error

Solution

The following patches fix the above mentioned error:

  • Patch 2 for PCo 15.5 with Support Package 04
  • Patch 35 for PCo 15.4 with Support Package 03

Download the current installation program from the SAP Service Marketplace and install it in accordance with the instructions in the installation guide. The current version of the installation guide is available on the PCo documentation page in the SAP Help Portal at https://help.sap.com/pco. You can use the installation program for the PCo patch to update a PCo version that is already installed or to set up a new installation if PCo is not yet installed on the target computer.

Before the installation, refer to the installation notes 3203375 (for PCo 15.5.4) and 2990278 (for PCo 15.4.3).
