Symptom

In the process "Shutdown/Dismantling of IMS" (DE_B_IM_DEV_REMOVAL), step 0250 raises the error "No implementation was selected for the current BADI."

This issue is relevant for the market role: MOS.

Other Terms

IM4G, DE_B_IM_DEV_REMOVAL, /US4G/CL_PST_IM_DEV_REMOVAL, METHOD DO_ACTION, Device, Operation type, Process Scenario, BADI

Reason and Prerequisites

The error occurred because unused statements were needlessly called in the process.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the system no longer raises the BADI implementation error in step 0250. You can verify the fix by checking that step 0250 (Device Removal) completes successfully in the process "Shutdown/Dismantling of IMS" (DE_B_IM_DEV_REMOVAL) without displaying the error.
