Symptom

How to Configure Maintain Business Systems During Master Data File Upload in SAP Ariba Invoicing?

Resolution

Steps to Maintain Business Systems in Master Data File Upload (SAP Ariba Invoicing)

Users with the CIM_Replication_Admin role can add or maintain Business Systems using the Maintain Business System option within the Master Data File Upload app.

  1. Navigate to the Maintain Business System Screen
    • Open the Master Data File Upload app.
    • Click Maintain Business System.
    • To add a new business system, provide the following details:               
Field NameValue
 Business System ID A valid business system ID. Should be is equal to Own SAP Cloud System in SAP S/4 HANA Cloud.
 Business System Description A description of the new business system.
 Business System Type

The business system type. Select from the following options in the dropdown:

  • SAP S/4HANA Public Cloud
  • SAP S/4HANA Private Cloud/ECC
  • Other

How to check Business System ID?

You can go to any communication arrangement in  SAP S/4 HANA system and check for the Own SAP Cloud System ID.

    2. Click Save Changes.

Note : The Business System ID will now be available for selection in the Select Source System dropdown
