Symptom

In the "Calculate Actual Quantity" batch job, a technical error is triggered when trying to process a material document classified as "Component Adjustment", while also having having cancellations (done in MIGO).

Other Terms

Short dump, critical error, Postprocess Documents, classify material document

Reason and Prerequisites

Program error

Solution

History:

Version 1 of January 21, 2026: New creation

Install the code correction.

Description of Correction:

The code changes contained by this note prevent the "Calculate Actual Quantity" batch job from crashing while a material document classified as "Component Adjustment" is being processed.

Functional Dependencies:

There are no technical dependencies.

Manual Changes:

No manual work is required.
