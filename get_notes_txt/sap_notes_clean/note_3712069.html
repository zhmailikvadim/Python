Symptom

 Why is my Supplier Request stuck in Submitted status instead of being automatically approved?

Cause

This can happen when incompatible supplier field mapping are being used to store supplier names, like vendor.vendorInfo.name1 with  SM-14213: Enable supplier full name  feature enabled or the opposite scenario, having this feature disabled and use the vendor.vendorInfo.fullName supplier field mapping instead.

Resolution

Ensure that none of the supplier management questionnaires in the system contain questions mapped to individual numbered name fields, vendor.vendorInfo.name1 through vendor.vendorInfo.name4 if you have SM-14213: Enable supplier full name  feature enabled.
