Symptom

When executing an ALV-based report in background and generating spool output, the report summary does not display the selection screen fields and their corresponding values. Only the "Sort Criteria" and "Data statistics" information is shown in the output. The selection parameters used during report execution are not included in the printed output.

Environment

SAP S/4HANA

Cause

This behavior is related to the ALV print configuration. If the parameter "ALV Selections" is not enabled in the Print Properties settings, the selection screen fields and values are not printed in the spool unless this option is explicitly activated. 

Resolution

To restore the full summary (including selection fields and values):

  1. Go to Print Properties
  2. Choose Defaults
  3. Set 'ALV Selections' to YES
  4. Click Apply Settings

Important: Confirming the popup without clicking Apply Settings does not save the changes.

This setting can also be saved as a default (for dialog and/or background), so it applies to future executions.
