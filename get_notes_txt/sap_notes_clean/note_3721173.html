Symptom

In DSO device change process (DE_B_IM_DEV_CHANGE_DSO), if one register will billing install in multiple MaLos, in other word, there're more than 1 MaLo in related_pod in UBD data, process can't correctly determine IM4G metering product for multiple MaLos.

Other Terms

DSO device change, DE_B_IM_DEV_CHANGE_DSO, related_pod, determine IM4G metering product, IM4G MP

Reason and Prerequisites

This is a program error

Solution

Please implement the attached SAP Note or import higher version of Support Package.
