Symptom

  • A user is unable to sign in to Cloud Foundry through SAP HANA Database Explorer.
  • The login flow loops back to an Auth error page, preventing authentication.
  • Error message: "Sign-in failed. Something went wrong. Please try again. If the failure persists, please contact SAP Support."

Environment

HANA Cloud Services HANA Cloud

Reproducing the Issue

  1. Access SAP HANA Cloud via SAP HANA Database Explorer in the BTP subaccount.
  2. Attempt to sign in to Cloud Foundry through SAP HANA Database Explorer.
  3. Authentication fails.

Cause

Due to the account settings when logging into K8s CF Edition, the authorization information cannot be retrieved from BTP.

Resolution

  1. Navigate to the following URL: [https://uaa.cf.<region>.hana.ondemand.com/profile] i.e.: https://uaa.cf.ap11.hana.ondemand.com/profile or https://uaa.cf.jp10.hana.ondemand.com/profile.
  2. Select all checkboxes on the page.
  3. Click the "Update" button to save the changes.
