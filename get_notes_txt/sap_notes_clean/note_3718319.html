Symptom

When assigning a Task List to a Maintenance Order in the Manage Maintenance Order application (F5241) and selecting the "Delete existing operations" option, the newly added operations are not numbered sequentially from the beginning. Instead, they retain the numbering sequence from the last existing operation before deletion, leading to incorrect operation numbering.

Other Terms

F5241, R_MAINTENANCEORDERTP, Maintenance Order, Task List

Reason and Prerequisites

Coding issue.

Solution

Apply this SAP Note.
