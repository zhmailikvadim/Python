Symptom

In the application Manage Material Assignments for Force Elements (DFS_MCHG_ASSGMT), customer extended fields are not processed in transaction DFS_MCHG_ASSGMT, causing incorrect and inconsistent data display in ALV output.

Other Terms

BADI, BADI_DFS_ASSGMT_ALV_FIELDS, DFS_MCHG_ASSGMT, S/4 HANA D&S, Defense and Security.

Reason and Prerequisites

This is an enhancement.

Solution

BADI is provided for extended fields handling - BADI_DFS_ASSGMT_ALV_FIELDS -> SET_FIELDCATALOG , SET_EDITABLE_FIELDS and FILTER as DFS_ASSIGNMENT_TYPE.

Implement the attached correction instruction for the solution.
