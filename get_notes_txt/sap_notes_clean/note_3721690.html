Symptom

After creating custom interactions of type “Z” with an assigned EXECUTION_RUN_KEY, the Campaign Performance metrics displayed in the Overview tab do not match those shown in the Automation and Performance tabs.

Other Terms

NumberOfExecutedInteractions, CommunicationMedium, InteractionType, Performance, EXECUTION_RUN_KEY, C_MKT_AGGRGDCMPGNSUCCESSQ

Reason and Prerequisites

NumberOfExecutedInteractions metric from Z interaction types where included in the calculation of the Campaign Overview tab, this note will remove them from the calculation.

Solution

Implement the attached Correction Instructions.

Ensure that the SAPUI5 Application Index is updated before you test the correction. To do this manually, you can execute report /UI5/APP_INDEX_CALCULATE. See SAP Note 2227577 for more information.

Ensure that you also implement all the prerequisites.
