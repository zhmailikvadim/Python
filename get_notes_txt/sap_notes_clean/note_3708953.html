Symptom

You want to display a custom error page for 403 errors for an icf service without changing the error page of 404 errors.

Solution

Implement note, go to the icf serivce and select "error pages", a tab called "Forbidden" should be shown, you can edit your custom error page under it.

Note that an error page for 403 erros can also be customized in the "Not accessible" tab, but also affects 404 erros.

The behaviour is as follows:
- If both tabs have an error page configured, "Not accessible" will be used for 404 errors and "Forbidden" for 403 errors.
- If "Not accessible" is configured and "Forbidden" is not, "Not accessible" will be used for 404 and 403 errors.
- If "Forbidden" is configured and "Not accessible" is not, "Forbidden" is used for 403 errors and the default page for 404 errors.
