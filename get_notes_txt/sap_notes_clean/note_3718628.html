Symptom

After QRC1 2026 is applied to your SAP Analytics Cloud tenant (version 2026.02), you are seeing a popup when clicking or interacting with a model which is stored in datasphere.  The popup URL will be similar to <SAC Tenant>/mobileapp/oauth2/api/v1/authorize

Environment

  • SAP Analytics Cloud 2026.02

Reproducing the Issue

  1. Log into your SAC Tenant
  2. Create a Story
  3. Add a widget and get prompted to select a model
  4. On the left, under the "Datasphere" section, select a datasphere server
  5. See popup

Cause

In QRC1 2026, SAC switched over from SAML to OAuth + PKCE when making Direct live connections to Datasphere. The new method uses token based authorization and the token fetching is handled in the popup window. In order to support third party cookie blocking, the popup is used instead of an invisible iframe.

Resolution

This behavior is by design.
