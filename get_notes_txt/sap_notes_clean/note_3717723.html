Symptom

Contracts that have been assigned the status 'Awaiting Amendment Approval' are currently not displaying or accessible within the 'Approve ACM Trading Contracts' Fiori application. This prevents authorized users from reviewing, evaluating, and approving pending contract amendments through the designated application interface.

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental." 

Environment

  • Sap Agricultural Contract Management
  • SAP S/4HANA2023

Reproducing the Issue

  1. Open the Approve ACM Trading Contracts Fiori Application.
  2. Attempt to filter contracts with the status 'Awaiting Amendment Approval'.
  3. Observe that contracts with this status are not available.

Cause

Configuration data not maintained for Status Group 'N', Application Status 'I' and Next Status 'A'. 

Resolution

  1. Maintain the configuration data for Status Group 'N', Application Status 'I' and Next Status 'A' in table `/ACCGO/T_STATUS` .

Sap Agricultural Contract Management, Approve ACM Trading Contracts, Awaiting Amendment Approval, Configuration
