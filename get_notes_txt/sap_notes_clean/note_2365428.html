Symptom

SAP Note 2041815 ("Public Services regulation, state of Hesse: 2nd Public Services Regulation Modernization Act (DRModG) XXXIV (full child-rearing bonus in addition to the minimum pension for dependents)"), delivered the payment of the full bonuses in the case of the minimum pension.

With this change, the bonuses in the case of a surviving dependent's pension are to be paid using the wage types O5A2, O5B2, or O5D2 in the same way as the bonus according to section 50c BeamtVG. If the amount of the surviving dependent's pension is above the minimum pension, the bonuses pare paid as part of the wage type O052 for a widow or the wage type O053 for an orphan. This is not consistent.

Solution

Import the relevant HR Support Package or proceed in accordance with the correction instructions. The correction is activated with the subapplication VABR and is valid in the standard system as of January 1, 2018. If you want to implement the correction in advance, make the relevant entry for subapplication VABR in the view V_T596D. In this case, the entries in the view V_T7PBSCA3B for the field names

VERS_BEZUG-HINTERBL-ZAHLBETRAEGE_HINT_ZUSCHL_P50A,
VERS_BEZUG-HINTERBL-ZAHLBETRAEGE_HINT_ZUSCHL_P50B,
VERS_BEZUG-HINTERBL-ZAHLBETRAEGE_HINT_ZUSCHL_P50D

that are described in the manual correction instructions must not be delimited as of January 1, 2018, but as of the date you have selected.

After you implement the SAP Note and activate the subapplication, the bonuses are paid within the pension wage type O052 for widows and O053 for orphans. The payment wage types for the bonuses are no longer created.
