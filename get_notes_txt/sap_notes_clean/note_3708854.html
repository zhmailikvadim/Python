Symptom

Replication of Product Master from OnPrem/Private Cloud to S/4HANA Public Cloud 2602 fails with "Value is longer than the maximum permitted length1" for the field OriginalBatchManagementRequiredIndicator 

Environment

  • SAP S/4HANA Cloud Public Edition 2602
  • SAP S/4HANA Cloud Private Edition
  • SAP S/4HANA

Reproducing the Issue

  1. Access the app Technical Error Log
  2. Error Value is longer than the maximum permitted length1:"false" appears 

Cause

Program error

Resolution

Please implement the note 3612572 - DRF/SOAP: Indicator for original batch management sent as Boolean value in the MDG / Private Cloud System and start the replication again

Keywords

API_PRODUCT_SRV , PRODUCTMDMBULKREPLICATEREQUEST , SAP_COM_0009 , ProductMDMBulkReplicateRequest_In , WSDL  , CONSTRAINT_VALUE : 1 ,  "Value : false" , Boolean value
