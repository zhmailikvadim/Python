Symptom

The FA_VAT XML file submitted to the KSeF portal contains all the required information. However, the KSeF number (Tax Invoice Number) is not included in the attached FA_VAT XML file.

Cause

  • KSeF Number:
    The KSeF number is the unique identifier assigned to each electronic invoice in Poland's KSeF system. It is essential for tracking and identifying invoices within the system.

  • FA_VAT XML:
    The FA_VAT XML is the legal and official document that contains the tax-related details of the invoice. However, it does not include the KSeF number in the attachment.

Resolution

As per the system design, the KSeF number is not included in the FA_VAT XML. It is instead present in the following locations:

    • Invoice User Interface (UI): The KSeF number is displayed on the invoice UI where users can view the details of the invoice.

    • Invoice cXML: The KSeF number is available in the cXML format, which is part of the overall invoice communication.

    • SBN-Generated PDF: The KSeF number is shown in the PDF generated by the SBN that accompanies the invoice.

The KSeF number is generated and associated with the invoice during the integration process with the KSeF system. However, the FA_VAT XML document itself does not include this number, as it is considered a separate piece of information relevant for the display and validation of invoices, rather than a part of the tax document itself.
