Symptom

A reference to a customer scheduling agreement is stored in the export declaration as a reference to a sales order. The symptom occurs for export declarations that were created with reference to SAP Transportation Management.

Other Terms

ExportDeclarationSUITERequest_In, SalesOrderReference, 114, 734, BusinessTransactionDocumentTypeCode, /SAPSLL/BTDTC

Reason and Prerequisites

This problem is caused by a program error.

Solution

Implement the attached correction instructions.

In addition, implement the correction instructions from SAP Note 3713129 that are relevant for your release in the relevant SAP Transportation Management system.

Install the corresponding patch for the XI CONTENT if required:

  • XICNTGTSCORE3712968 for software component version XI CONTENT GTSCORE 200
  • XICNTGTSCORE3712968 for software component version XI CONTENT GTSCORE 300
