Symptom

Uploading the company logo from file management or personal file explorer doesn't work on notification email customization.

Environment

SAP SALES CLOUD CPQ

Cause

If you're trying to customize a quote approval notification email with HTML for example, you won't be able to use the file path of your personal computer or the company logo stored in CPQ under File Management. Instead, it is recommended to use a public image URL.

Resolution

Use a public company logo.
