Symptom

Posting of plant in overhead documents is active in the system. When plant of a CO object in overhead debit line is different from plant of a CO object in overhead credit line, the overhead posting is canceled with a runtime error UNCAUGHT_EXCEPTION for the exception CX_FATAL_EXCEPTION.

Other Terms

UNCAUGHT_EXCEPTION, KZPI, WERKS, KGI2, CO43

Reason and Prerequisites

Program error.

Solution

Apply this SAP Note or corresponding Support Package.
