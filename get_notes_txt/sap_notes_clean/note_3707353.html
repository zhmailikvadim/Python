Symptom

Within Learning Administration > Library record > QuickGuides tab, is possible to see the available QuickGuides. How does the column 'View Count' work? Are there any additional settings required to capture the count?

Environment

SAP SuccessFactors Learning

Reproducing the Issue

  1. Log into HCM > navigate to Learning Administration >
  2. Learning Activities > Libraries > search for the required entity and open it >
  3. At the QuickGuides tab, notice that users published some quick guides >
  4. Users are accessing these quickguides via library, but still the 'View Count' column shows 0.

Resolution

The SAP Engineering team has developed a code fix for the reported scenario, which is currently scheduled to be delivered within the patch 2511p17, according to the schedule available at SAP SuccessFactors - Patch Details.

For VSaaS customers, this means that the fix is planned to be delivered as part of the b2605 release version, as per the schedule available at Learning Validated SaaS Release Schedule.

Please set this KBA as favorite in order to be notified on future updates to this document, as per 2171560.
