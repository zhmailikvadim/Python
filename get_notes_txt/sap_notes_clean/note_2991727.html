Symptom

Once a Yard Request has been created by a loading/unloading instruction from Transportation Management (TM) System, it can't be updated from TM. The system will report the following errors:

Field "LENGTH" cannot be changed since it is read only

Field "WIDTH" cannot be changed since it is read only

Field "HEIGHT" cannot be changed since it is read only

Field "TU_MAX_WEIGHT" cannot be changed since it is read only

...

Yard Order Item *: Loaded Weight must be greater than 0

Yard Order Item *: Tare Weight must be greater than 0

...

Field "MAX_WEIGHT" cannot be changed since it is read only

Field "MAX_WEIGHT_UOM" cannot be changed since it is read only

Other Terms

/SAPYL/CL_API_YARD_ORDER=>MAP_API2BO_ATTR
/SAPYL/CL_CHECK_YO_DIMENSIONS=>/SAPYL/IF_CHECK_YO_DIMENSIONS~CHECK_DIMENSION
/SAPYL/CL_LDAP_MAP_YARD_ORDER=>MAP_ITEM_YR
/SAPYL/CL_YO_D_ITEM=>SET_ATTR_PROPERTY4GROUP
/SAPYL/CL_YO_D_ITEM=>SET_ITEM_DIMENSION_FIELD_CNTRL
/SAPYL/CL_YO_V_ITEM_C=>CHECK_DIMENSION_UOM

Reason and Prerequisites

 Program error

Solution

Install this correction instruction
