Symptom

  • In the invoice item detail tab, the additional data tab does not display the modified customer group fields (KVGR1 to KVGR5) updated via CMOD transaction. Instead, it shows the default labels (Customer group 1 to Customer group 5).
  • The issue occurs in screen 6106 of program SAPMV60A in SAP S/4HANA.

Environment

SAP S/4HANA

Reproducing the Issue

  1. Go to transaction VF03.
  2. Select an invoice.
  3. Select the item
  4. Open the additional data tab.

Cause

The "Modification of Field" setting for VBRP-KVGR* fields in S/4HANA is set to 'F', which requires direct modification of screen fields instead of dictionary-based changes.

Resolution

The previous modifications for the data elements KVGR1, KVGR2, KVGR3, KVGR4, and KVGR5 should be evaluated again. If the text changes are intended to be kept across all systems, the “Modification of Field” indicator can be set back to 2.

If the modifications have already been performed, SPAU/SPDD must be used during future upgrades to adjust them. This is the inherent nature of modifications.
