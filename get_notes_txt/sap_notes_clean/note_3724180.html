Symptom

When we set up the field 'Detailed Description' as a mandatory field and leave it empty, and save we have a validation error message -

"Please complete all required fields". But the exact field is not highlighted which is causing the error.

Other Terms

Change Record, Detailed Description, Mandatory, Error, Required, Field, error, highlight.

Reason and Prerequisites

Program Error.

Solution

Use transaction SNOTE to implement the corrections delivered with this SAP Note in the affected Change Record system or import the relevant Support Package.
