E-Separation Solution Implementation For Global Scenario

Environment

  • Payroll India E Separation
  • SAP ERP 6.0
  • SAP ERP Core Component 6.0
  • SAP enhancement package 6 for SAP ERP 6.0

Reproducing the Issue

Your organization has a global presence and you are looking for a single global separation application which should cater to all separation related activities such as approvals, counseling, due clearance etc.

Resolution

The document 'E-Separation Global.PDF' explains the step by step configuration for one sample country version 'Canada' (Country Grouping = 07). You can find the document in Attachment section.
