Symptom

The (WAVE-based) process "Deadline Expiry Due to WWV (12 Months)" (PRM_BDS_PR_DEADLINE_PROOF_RU) does not transfer a document used to monitor the period for proof of residential use to the status "Completed" after the deadline expiry. Instead, the status is set to "Requested".

Reason and Prerequisites

When you set the status of such a document in the task "Set Document Status to Completed (06)" (PRM_BDS_RU_TK_RDC_STATE_CHNG) of the aforementioned Process uses the wrong value for the new status.

Solution

The setting of the document status is corrected.
