Symptom

After an SAP NetWeaver or SAP BW/4HANA upgrade, BW query fails with the following ABAP dump:

---------
Category               ABAP programming error
Runtime Errors         OBJECTS_NOT_COMPATIBLE
ABAP Program           CL_BICS_MD_SELECTOR===========CP
---------

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Environment

  • SAP NetWeaver
  • SAP Business Warehouse
  • SAP BW/4HANA
  • SAP S/4HANA

Reproducing the Issue

  1. Upgrade BW system. 
  2. Run any affected query.
  3. Get the ABAP dump.

Cause

This is most likely caused by an inconsistent Program Buffer (PXA).

Resolution

Refer to the solution from SAP Note 3708173: Known Issues > ABAP runtime error OBJECTS_NOT_COMPATIBLE.
