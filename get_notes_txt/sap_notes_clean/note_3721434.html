Symptom

In the processes DE_*_MDC_SUP_DSO_SETTL_REL_W_DEP_SUP and DE_*_GELIGAS_SOS_SUP, the fields MSG_DATE and MSG_TIME are missing in the UBD structure for gas profile definitions.

For electricity processes such as DE_*_EQS_PFDEF_DSO_RECEIVE_SUP, these fields are populated.

However, for gas, the fields are not populated in \UBD_DATA\PROFILE_DEFINITION[*]\DATA[*].

Environment

SAP IS-U

Cause

The MSG_DATE and MSG_TIME fields are only used in the import processes (like DE_*_EQS_PFDEF_DSO_RECEIVE_SUP, for electricity) to verify if the imported profile definition is the latest version.
In Switching-related gas processes, only references to existing profile definitions are made, so these fields are not required nor available.

Resolution

This is the standard system design.
The fields MSG_DATE and MSG_TIME are not populated for gas-related switching processes because they are not required or used in these scenarios.
No further action is required.
