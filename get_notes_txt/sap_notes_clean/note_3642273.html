Symptom

This SAP Note provides you with corrections to the EFD Contribuições (/TMF/PCO) report.

Currently, when you fill the Goods incorporated into fixed assets (/TMF/D_BENSENCAR) shadow table and run the /TMF/PCO report, the system leaves the VL_OPER_AQUIS field in the F130 register empty.

Other Terms

 Brazil, SPED, F130, EFD Contribuições.

Reason and Prerequisites

This SAP Note is relevant only for Brazil.

You have implemented SAP Note 3652114.

Solution

After implementing this SAP Note, the system uses the correct value filled in the /TMF/D_BENSENCAR shadow table for the VL_OPER_AQUIS field in the PCO F130 register.

As a general rule, SAP recommends that you implement the Support Package specified for your release. However, if you need to install a solution earlier, use the SAP Note Assistant to implement the correction instruction. You can find more information about the SAP Note Assistant in SAP Support Portal, under SAP Note Assistant. Note that documentation, transactions, and customizing activities are only delivered via Support Package.

You must follow the correction instructions below to implement the solution.
