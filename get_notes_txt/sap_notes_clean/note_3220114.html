Symptom

You are using the promotion pricing service, which is an SAP HANA XS advanced (XSA) application.

You want to configure or update the central deployment of the promotion pricing service and therefore download the latest XSAC component from the SAP support portal.

Note that the SCV file, mentioned in the Common Installation Guide and the Common Upgrade Guide on the SAP Customer Activity Repository application bundle, does not represent the latest version.

Other Terms

SAP Service Marketplace, Omnichannel Promotion Pricing,  XSACOPPPPS03_<patch level>-80004642.ZIP ,  XSACOPPPPS02_<patch level>-80004642.ZIP,  XSACOPPPPS04_0-80004642.ZIP, SAP Support Portal, Update the PPS SAP HANA XSA Application, Configure the Central Deployment, SAP Customer Activity Repository applications bundle 5.0 FPS03

Reason and Prerequisites

The SCV file mentioned in the Common Installation Guide and Common Update Guide is not the latest version.

Solution

The correct SCV file to be used for the configuration or update of omnichannel promotion pricing with CARAB 5.0 FPS03 Patch 0 is XSACOPPPPS04_0-80004642.ZIP.

For more information see the updated documentation attached to this note.
