Symptom

When trying to start ./DPAgentService start the following error appears:

/DPAgentService: /lib64/libstdc++.so.6: version `CXXABI_1.3.9' not found (required by ./DPAgentService)

./DPAgentService: /lib64/libstdc++.so.6: version `CXXABI_1.3.9' not found (required by ./DPAgentService

Reason and Prerequisites

Ennvironment:

  • Linux Redhat 6.5, 7.4 and 7.6
  • SDI Agent 2.4.0.0

Solution

First install the following package:

  1. yum install compat-sap-c++-6

Then follow the next steps to link the symbolic link to the package that was just installed.

  1. cd /lib64
  2. ls -l libstd* - to show what the current library is linked to
  3. ls -l /opt/rh/SAP/lib64 - so if you see that it is pointing to an older version then please remove it by following step 4.
  4. sudo rm libstdc++.so.6 - to remove the old file
  5. sudo ln -s /opt/rh/SAP/lib64/compat-sap-c++-6.3.1.so libstdc++.so.6 - to create the new symbolic link to point to the compat sap
