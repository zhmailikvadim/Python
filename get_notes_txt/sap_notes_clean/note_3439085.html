  • This can happen to any parameter which has layer restriction

Environment

SAP HANA, platform edition 2.0

Cause

Mentioned parameter will have layer restriction and a default value [GREEN DOT] is added.

Resolution

Please perform one of the below steps to fix this issue-

  • Double click to open the parameter. In the particular layer, click on "Restore Default" and then click on Save.

OR

  • Execute the below script-
    ALTER SYSTEM ALTER CONFIGURATION ('<sample.ini>', '<layer>') UNSET ('<section>', '<parameter>') WITH RECONFIGURE';

Keywords

Parameter, layer restriction, GREEN DOT
