Symptom

On October 29, 2025, the Federal Central Tax Office (BZSt) announced a change regarding the file format for ELMA5. From December 1, 2025, reports can only be submitted in XML format. All details are announced here: https://www.bzst.de/SharedDocs/Kurzmeldungen/DE/2025_Kurzmeldungen/10_Oktober/20251030_zm_elma.html.

Solution

SAP intends to provide a feature package or patch to solve the symptom that is described in this SAP Note. The section References below lists the related feature package or patch once they become available. Be aware that References can only be confirmed at feature package or patch release date. SAP delivers feature packages or patches only for selected releases at its own discretion, based on the business impact and the complexity of implementation.

 

In the Company Details form on the Accounting tab, there are additional fields for entering the sender ID, the registration number, and the user account ID of the BZSt. The field "Benutzerkonto ID (BoP)" (BZSt User Account) must be filled.

In the menu, there is an additional item labeled Zusammenfassende Meldung (XML).

    

 

The wizard will guide you through generating the file.

 

For the interim period, SAP Note 3681115 has attached GEP file as a temporary workaround. After the release, only the complete implementation will be supported on new SAP Business One versions and the workaround GEP files will not be maintained.
You can upload the new file under Administration → Setup → General → Electronic File Manager.
