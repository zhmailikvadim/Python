Symptom

You are using the report IUUC_CREATE_ARCHIVE_OBJECT to enable SLT data migration from the archiving objects. This report's execution fails at the stage of runtime objects generation with one or more syntax errors.

Other Terms

LT Replication Server, SLT, replication, replicator, trigger, realtime, DMIS 2018, DMIS 2020, DMIS 2025

Reason and Prerequisites

SAP LT Replication Server is installed on:

- DMIS 2018 SP12-SP14 or

- DMIS 2020 SP08-SP10 or

- DMIS 2025 SP00

Solution

Install this note in your SLT system.
