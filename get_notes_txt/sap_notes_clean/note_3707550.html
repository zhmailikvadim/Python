Symptom

The Belgian customs system IDMS responds to an import declaration with a rejection message (IE556) that refers to the rule BE0163.

The field /IE415B/GoodsShipment/Destination/regionOfDestination is checked.

After you implement this SAP Note, the region of destination is mapped to one of the following values if the country of destination is 'BE':

1 -> Flemish region (Antwerp, East Flanders, West Flanders, Limburg and Flemish Brabant)
2 -> Walloon region (Hainaut, Liège, Luxembourg, Namur and Walloon Brabant)
3 -> Brussels Region (Brussels Capital)

Other Terms

BE, IDMS, import, region, /SAPSLL/REGAR

Reason and Prerequisites

Program error

Solution

Implement the attached correction instructions or import the specified support package.
