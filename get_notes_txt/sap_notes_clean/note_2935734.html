Symptom

Report PK_DELETE_VENDOR_CUSTOMER deletes personal data in area of Kanban. This report should not be used for this purpose. The functionality is replaced by data destruction object PP_KANBANCC_DESTRUCTION (documented here) together with the End-of-Purpose Check for personal data (documented here).

Reason and Prerequisites

Report PK_DELETE_VENDOR_CUSTOMER deletes customer data related to the process "Kanban with SD" and supplier data in message conditions for summarized JIT calls. In case you are not using these processes, the note is not relevant.

Solution

After applying the correction, the report will abort immediately issuing an error message referring to the information given in this note.
