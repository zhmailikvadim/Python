Symptom

You work with materials that are set as Co-Products.

You use PLMSI SV FVM Customizing for ITM_CLASS_DATA KZKUP. You send two BOMs, where each one is a Co-Product of the other:


BOM1 - PV1
 Component1 - PV2 - KZKUP = 'X'

BOM2 - PV2
  Component2 - PV1 - KZKUP = 'X'

When you run the process, you see the error "BOM is recursive".

Other Terms

29/110, CS_RC_RECURSIVITY_CHECK, Structure Version, Co-Product, items

Solution

Implement this SAP Note to allow recursive creation for Co-Product functionality.
