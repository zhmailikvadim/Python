Symptom

You are working with the Test Data Refresh feature, however when trying to execute the system readiness check the following error message appears:

"Cannot run system readiness check. The current refresh scenario is not supported" 

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Access the Check Your System Readiness app.
  2. Click on the New Check button.

Cause

The system readiness check is being performed in the target test/quality system, which is not supported.

The system readiness check must be performed on the source productive system.

Resolution

The system works as expected

SAP Help Center - Check Your System Readiness
