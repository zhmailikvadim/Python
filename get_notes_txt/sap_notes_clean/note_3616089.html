Symptom

 The request condition table configured within the price sheet is not being accurately referenced during the regeneration process.

Other Terms

/IRM/GPRM

Reason and Prerequisites

 This is caused by Program error.

Please implement SAP Note#3371381 for V/4 2021,2022, 2022 V2 and 2023 releases.

SAP Note# is available for 60E release.

Solution

Please implement the correction instructions. These corrections are only valid if you have applied the following Support Packs.

Release V/4 –  2021, 2022, 2022 V2, 2023, 2023 V2 and 2023 V3

Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object
