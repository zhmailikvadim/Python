Symptom

In display mode, the system displays the delete icon active on all screens of the Workers Compensation (WoCo) loss management.

Reason and Prerequisites

Prerequisite:
Use of Workers' Compensation (WoCo) Loss Management

SAP Note 3732160 is implemented

Reason:
The delete icon is displayed regardless of the editing mode.

Solution

History:

March 27, 2026: New

Troubleshooting Description:
When each screen is loaded in Workers Compensation (WoCo) Loss Management, the application checks whether the display mode is active. If the system detects this mode, the delete icon is grayed out.
You can delete an entry in workers Compensation (WoCo) loss management only in processing mode.
