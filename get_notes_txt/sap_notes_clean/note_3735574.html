Symptom

The Back button in Change Maintenance Order web dynpro app navigates to the FIORI launchpad instead of the app's initial selection screen.

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Environment

  • SAP R/3
  • SAP R/3 Enterprise
  • SAP ERP Central Component
  • SAP ERP
  • SAP Enhancement package for SAP ERP
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA, on-premise
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Open the Change Maintenance Order app from the Fiori launchpad.
  2. Enter a valid maintenance order and continue to the processing screen.
  3. Press the Back button on the launchpad header.

  4. The app navigates to the Fiori launchpad instead of returning to the initial order selection screen.

Cause

Standard behavior: 

Web Dynpro applications are not able to utilize the FIORI launchpad back button due to technical limitations.

Resolution

The initial order selection screen of Change Maintenance Order can be reached without leaving the app by refreshing the browser page.

Press 'F5' to refresh the page or use the 'Refresh' button on the browser window.
