Symptom

In Hedge Management Cockpit for Hedging Area (TOENE) you create a hedge request. If no hedging classification is entered and check, save or submitt is triggered, program will stop with dump OBJECTS_OBJREF_NOT_ASSIGNED_NO Execption CX_SY_REF_IS_INITIAL in ABAP Program CL_TOE_HA_VERSION.

Other Terms

OBJECTS_OBJREF_NOT_ASSIGNED_NO, CX_SY_REF_IS_INITIAL, CL_TOE_HA_VERSION, short dump, TOENE

Reason and Prerequisites

Program error from note 3642879.

Solution

Implement the attached Correction Instruction (CI) or the relevant Support Package.
