Symptom

In Fiori "My Travel and Expenses V2" application, if you create a cost assignment for Mileage or Additional destination, you will get a short dump such as "The application has encountered an unknown error: 500 HTTP request failed".

Other Terms

PAOC_TRV_COMMON_V2, cost assignment,  cost object, CL_TRV_BUSINESS_DATA, CREATE_COSTASSIGNMENT, lt_costdist_default,  ITAB_LINE_NOT_FOUND CX_SY_ITAB_LINE_NOT_FOUND.

Reason and Prerequisites

 Program Error.

Solution

 Apply the SAP Note or corresponding Support Package.
