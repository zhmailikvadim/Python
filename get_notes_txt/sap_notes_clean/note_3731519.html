Symptom

You declare your import declaration in SAP GTS to customs system DMS.

You use partner with the type 'Customer' (PATYP = 'CUSERW') and procedure code different than 42 or 63. Additional fiscal reference is created with the role FR2, which is only allowed for procedure codes 42 and 63.

Other Terms

Denmark, DK, XML, import, Additional fiscal reference, Dometic Duty Tax Party, FR2

Reason and Prerequisites

This note corrects an error in note 3717907

Solution

Implement the correction instructions.
