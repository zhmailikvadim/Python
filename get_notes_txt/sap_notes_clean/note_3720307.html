Symptom

  • Memory short dump occurs while running the report: /SALM/TRACEABILITY_MATRIX.
  • Error message: TSV_TNEW_PAGE_ALLOC_FAILED.

Environment

  • SAP Solution Manager 7.2

  • Focused Build

Reproducing the Issue

Execute the report /SALM/TRACEABILITY_MATRIX 

Resolution

  1. Ensure the complete implementation of the relevant SAP Master Note for Focused Build - Test Suite Dashboard (fiori based). For example, if ST-OST SP15 is deployed, the involved correction would be 3539838 - Focused Build - Test Suite Dashboard (fiori based): Corrections for ST-OST 200 SP15.
  2. SE38 report /UI5/APP_INDEX_CALCULATE - select Full Calculation - hit enter 2x to skip the warnings and execute
    clear all caches using
    - tx /n/IWFND/CACHE_CLEANUP
    - tx /n/IWBEP/CACHE_CLEANUP
  3. re-test /SALM/TRACEABILITY_MATRIX report
