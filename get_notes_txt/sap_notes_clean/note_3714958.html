Symptom

When reviewing the "Manage Outbound Deliveries - VL06O" app, there are no values displayed for fields such as delivery quantity, plant, storage location, etc.

Environment

SAP S/4HANA Cloud Public Edition

Cause

When VL06O app is run the Header View is displayed by default.

Resolution

  1. Switch the view from "Header view" to "Item view".
  2. Verify that item-level data is displayed.

Keywords

vl06o, outbound deliveries, header view, item view, sap s/4hana cloud public edition, delivery management, blank field values, display wrong view
