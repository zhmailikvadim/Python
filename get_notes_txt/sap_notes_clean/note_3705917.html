Symptom

 In the process "MDC: Receipt of Change with Dependencies from DSO to SUP as SUP (11117/55117/44117)" (DE_B_MDC_W_DEP_DSO_AUTH_SUP_SUP), step 2600 (Trigger Device Change Process) starts version 0001 of the process "Setup / Replacement of Devices (SUP View)" (DE_B_IM_DEV_CHANGE_SUP) instead of the correct version 2504.

Other Terms

DE_B_MDC_W_DEP_DSO_AUTH_SUP_SUP, DE_B_IM_DEV_CHANGE_SUP, step 2600, Trigger Device Change Process, device change, setup, replacement of devices, BC set, version 0001, version 2504

Reason and Prerequisites

The error occurred because the configuration was missing.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the system triggers version 2504 of the process DE_B_IM_DEV_CHANGE_SUP correctly. Import the BC set /US4G/3_0_SUP_SP07_NOTE_3705917. You can verify the fix by triggering the process "MDC: Receipt of Change with Dependencies from DSO to SUP as SUP" and checking that the correct process version 2504 is started.
