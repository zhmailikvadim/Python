Symptom

In the SAP Fiori app Medication Number Status, Study IDs and Protocol IDs are nont included in exported files.

Other Terms

ISLSUI, Manufacturing, Medication Number Status, F7467, download

Reason and Prerequisites

You use SAP Intelligent Clinical Supply Management or this SAP Note is a prerequisite of another SAP Note.

Solution

Implement the attached correction.

You must also implement SAP Note 3723568, which includes the related backend corrections.

Run calculation of SAPUI5 Application Index:

    - Run report /UI5/APP_INDEX_CALCULATE via transaction SE38.

    - Enter the SAPUI5 repository '/CTUI/MED_ANALYZE_LR' on the Single SAPUI5 Repository Only tab, and run it.
