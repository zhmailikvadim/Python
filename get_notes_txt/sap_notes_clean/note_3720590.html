Symptom

In transaction IL05 or IH06 when managing Layouts in 'Settings->Layouts->Current...' you notice that the View Tab is missing.

Other Terms

EAM, IL05, IH06, ALV, Change Layout, View tab, Layout variant, LVC, SALV, PF-STATUS, GUI status, Function code, Toolbar, Context menu, Menu Painter (SE41), RIIFLO20,

Reason and Prerequisites

The reason is a program error.
ALV PF-STATUS and related function availability definitions in the application did not comply with the stricter ALV UI checks mentioned in Note 3468780, which led to the suppression of the “View” tab in the dialog.

Solution

Apply attached Correction Instruction via SNOTE.
The correction aligns the ALV PF-STATUS/toolbar and function mapping with the tightened checks so that the “View” tab becomes visible again in the Change Layout dialog of IL05/IH06.
No manual post-implementation steps are required.
