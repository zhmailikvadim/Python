Symptom

When copying form settings between users, the source user’s UI configuration template is not copied.

Environment

SAP Business One

Reproducing the Issue

Scenario:

  1. Log in as the source user.
  2. Go to Sales Order → Form Settings → Table Format and note the current column order.
  3. In Utilities, create a UI configuration template for form Sales Order: go to Form Settings → Table Format, change the column order, and assign the template to the source user.
  4. Open User Settings for the source user, click Copy Form Settings, select a target user and Sales Order as the form, then click Copy.
  5. Log in as the target user and open a sales order.

Expected result:

The source user’s UI configuration template is copied and applied to the current sales order's form settings.

Actual result:

  • The source user’s UI configuration template is not copied.
  • The source user’s original form settings are copied to the target user, even if they were overridden by the UI configuration template.

Resolution

This is the correct behavior.
