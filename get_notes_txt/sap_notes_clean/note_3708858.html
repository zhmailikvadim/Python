Symptom

  • We created the Buyer Code in error
  • After it was created, we decided to change the Buyer Code

Environment

Test and Production Environments.

Cause

Once the tenant database is created, it is not possible to update the data.

Resolution

Once a tenant is created, their assigned buyer code cannot be changed.

To verify other options, please contact your Customer Success Partner (CSP).
