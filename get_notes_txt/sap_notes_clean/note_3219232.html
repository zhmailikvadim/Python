Symptom

 Disturbing temporary data in marketing tables for marketing sets, target groups and profiles.

Other Terms

segmentation

Reason and Prerequisites

 There is a program error.

Solution

Import the relevant Support Package or implement this note to correct the error.

Due to performance reasons when deleting marketing objects usually only a delete_flag is set.

Therefore you need to clean from time to time the tables and remove temporary data.

For doing this, please execute the report CRM_MKTTG_PURGE_TEMP_DATA.

When there are still temporary data in TG-item table you get informed. Than please do one of the following:

The cleaning of the TG item table can be done using either crm_mkttg_tg_del_items (for single small TGs)

or with CRM_MKTTG_PURGE_TG_I (which purges in packages). This cleaning may take long depending of the amount of data.

After cleaning the TG item table, please execute CRM_MKTTG_PURGE_TEMP_DATA again to clean the header table as well.
