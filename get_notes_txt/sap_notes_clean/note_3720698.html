Symptom

  • An error occurs when billing interval customers after completing the move-in process.
  • The error message indicates missing profile values up to a specific time - Error code AJ496. 

Environment

  • SAP S/4HANA, release IS-UT 801 and higher
  • SAP for Utilities as Part of SAP ERP, release IS-UT 600 and higher

Reproducing the Issue

  1. Install the meter at the installation for example at 10:00 AM using transaction EG31.
  2. Complete the move-in process on the same day using transaction EC50E.
  3. Maintain profile values starting with the same time as meter installation using transaction EEDM01. 
  4. Attempt to bill the customer for the first month after move-in using transaction EASIBI.
  5. Observe the error message indicating missing profile values up to 10:00 AM (error AJ469).

Cause

System design:

  • Billing starts at 00:00 on the first day of the billing period, requiring profile values to be available from this time.
  • The standard billing period spans from 00:00 on the start day to 23:59 on the end day. 

Resolution

  1. Use BAdI ISU_BI_RTP_TOU method IF_ISU_BI_RTP_TOU~CHNG_TIME_AT_RTP to modify the billing period.

Keywords

interval consumer billing, profile values missing, AJ496, AJ-496, AJ 496, Energy Data Management, profile assignment, ISU_BI_RTP_TOU
