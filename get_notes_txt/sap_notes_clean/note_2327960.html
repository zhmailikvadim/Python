Symptom

  • In a payroll correction (PAYTY = "B") to a month with (regular and Retirement bonus) withholding tax, the HCOTMRF0 report is showing double the retirement bonus withholding tax base.
  • The HCOCERT0 report is presenting a dump when the user uses the option 'Y' into decimal representation field in own default values.

Other Terms

Colombian Payroll - Tax - TMRF

Reason and Prerequisites

The HCOTMRF0 report was updated in order to show the correct retirement bonus withholding tax base

The HCOCERT0 report was modified to correct the dump presented
