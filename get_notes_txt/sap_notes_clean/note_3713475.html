Symptom

 SAP Solution Manager 7.2 ST-CONT 200 mainstream maintenance

Other Terms

Information on maintenance strategy of ST-CONT 200 for SAP Solution Manager 7.2.

Reason and Prerequisites

This SAP Note describes the maintenance strategy of ST-CONT 200 for SAP Solution Manager 7.2.

Solution

To enable planning security and support a sufficient transition phase from SAP Solution Manager 7.2 to SAP Cloud ALM, SAP provides mainstream maintenance for SAP Solution Manager 7.2 until December 31, 2027.

According to the information summarized in SAP Note 3255311, Application Operation within SAP Solution Manager 7.2 will transition to Customer-Specific Maintenance starting Jan. 1. 2028.

If you are using Application Operations with the delivered ST‑CONT 200 content package in SAP Solution Manager 7.2 from SP03 through SP13, then the final delivery version available for these Support Packages is: ➡️ ST‑CONT200 – Delivery Version 66. Version 66 is the last content delivery update provided for all SAP Solution Manager systems running on SP03–SP13 with ST‑CONT 200.

After the release of STCONT20066 - Version 66, the ST‑CONT 200 content package will continue to be delivered only for SAP Solution Manager 7.2 starting with Support Package SP14 and higher.

This delivery approach remains in place until the end of mainstream maintenance for SAP Solution Manager 7.2 in 2027.
