Symptom

How do I locate the value for the required x-anId header parameter used in SAP Ariba API endpoints?

Resolution

The x-anId header parameter expects the ANID associated to the realm you have configured the API. You can locate the ANIDs associated to an application by following the steps below:

  1. Sign in to the SAP Ariba API Developer Portal.
  2. Navigate to the Applications page.
  3. Select the application you are currently using.
  4. You will be able to locate the Ariba Network ID (ANID) for the relevant realm/environment in the Realm(s) used by this Application section.
  5. Use this ANID as the value of the x-anId request header.
