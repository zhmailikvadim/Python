Symptom

  • Error when uploading CSF succession data model in provisioning: An application error occurred. Please try again later, or notify support if the issue keeps occurring.
  • In Manage Business Configuration, no country value in the dropdown list of country field in localized HRIS element

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

SAP SuccessFactors Employee Central

Cause

Picklist csfCountry is not available in the tenant. Ideally the picklist is preconfigured, but for reasons like instance refresh or improper data handling, might cause unavailability of the picklist.

Resolution

Export the picklist from an instance that has the values maintained, and import the same into the affected instance.

OR

Create the picklist manually to reflect the country specific requirement.

2751695 - Country Field in globalInfo Does Not Display Any Values
