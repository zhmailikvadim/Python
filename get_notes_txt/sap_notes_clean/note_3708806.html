Symptom

You run the Tax Reporter (PU19) transaction for the W-2 PR - Wage and Tax Stat. Puerto Rico or W-2C Processing Puerto Rico forms and the electronic file output displays incomplete names. Letters with accents are removed.

Example

Raúl Juliá outputs as "Ral Juli"

Other Terms

PU19.

Reason and Prerequisites

Reason

The logic implemented to the fields in forms HR_F_MMREF_S_PR, HR_F_MMREF_1_PR, and HR_F_MMREF_2_PR removed certain special characters from the files. Accented characters (e.g.: á, ñ) were treated as special characters and removed.

This issue is a side effect caused by the solution delivered by SAP Note 3688712 - Year End 2025 Phase III for U.S. Tax Reporter.

Solution

From now on, the character-handling logic preserves alphabetic characters. The system normalizes accented letters by removing only the accent, while special characters such as periods, hyphens, and commas continue to be removed.
