Symptom

  • In the test and production environment it is not possible to perform the "Insert Line" option in app Manage Journal Entry Number Ranges.

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental." 

Environment

SAP S/4HANA Cloud Public Edition 

Reproducing the Issue

  1. Logon to the test or production environment.
  2. Access the Manage Journal Entry Number Ranges app.
  3. The option to 'Insert Line' is not maintainable.
  4. The same is true in the Quality or Test environment.


See screenshot below: Customizing, Test, Production from left to right. Insert line is only available in Customizing. 

Cause

The insert line option is considered additional configuration which requires transport through the 3SL (3 system landscape)

Resolution

  1. Logon to the customizing/development environment.
  2. Access the Manage Journal Entry Number Ranges app.
  3. Select 'Insert Line'.
  4. Maintain the additional line as per the business requirement.
  5. Save.
  6. Save this to a transport request and import the changes to the target system.
