Symptom

  • On the RMK career site, using the Keyword search for a posted job requisition returns a message stating that no jobs matching this value exist.  
  • The issue occurs despite the job requisition being properly posted and visible on the career site

Environment

SAP SuccessFactors Recruiting Marketing

Reproducing the Issue

  1. The job requisition is properly posted on the career site and can be found.
  2. Use its Job Req ID as Keyword
  3. Get result page saying that there is currently no jobs matching.

Cause

Database issue

Resolution

Open a ticket to the Support team so we can request a correction in the database to our Operations team

Keywords

RMK, job requisition, keyword search, Recruiting Marketing, no matching results, career site, database correction, job posting, Real Time Job Sync, search issue, job req ID
