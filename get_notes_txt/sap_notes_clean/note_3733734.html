Symptom

You display or edit a Form Bundle in the Form Processing Framework Web Dynpro application.
You deselect a form in the Form Overview tab on the user interface. The system reselects the form.

Other Terms

FMCA_WDY_FPF, FMCA_FORMS, TAX RETURN, TAX REGISTRATION, TAX REFUND

Reason and Prerequisites

Prerequisite: SAP Note 2027584 is implemented in the system.

Reason: The WDDOBEFORENAVIGATION method sets the lead selection of the FMCA_FORMS context before navigation, so the system reselects the form in the Form Overview tab.

Solution

After implementing this SAP Note, the system keeps the form deselected in the Form Overview tab. Implement this SAP Note by following the correction instructions in the attachment, or install the Support Package that contains this correction.
