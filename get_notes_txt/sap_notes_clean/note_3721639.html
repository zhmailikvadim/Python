Environment

SAP BW/4HANA

Cause

  • The ABAP DAEMON is configured for a different application server name in table ABAP_DAEMON_RT.
  • Go to table ABAP_DAEMON_RT and navigate to column APPLICATION_SERVER, the name of the application server is wrong.

Resolution

  1. Go to transaction SMDAEMON.
  2. Click on "System-Wide List view".
  3. In the menu bar, choose "ABAP Daemon" > "Check Daemon Table".
  4. Confirm the pop-up to delete the obsolete entries.
  5. The daemon runtime table is checked. All outdated entries are removed.
