Symptom

In the move-in process , the system reads inactivated data from table ETTIFN, resulting in an error when calculating the Article-ID.

Other Terms

GPKE, ETTIFN, operand value, Article-ID, /US4G/CL_DEF_BADI_ARTICLE_ID~GET_OPERAND_VALUE, /US4G/CL_DEF_BADI_ARTICLE_ID~GET_OPERAND_VALUES, End of Supply for Electricity

Reason and Prerequisites

The error occurred because the logic for updating table ETTIFN was changed, causing the system to read inactivated operand values.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the system reads the correct operand value from table ETTIFN, ensuring accurate Article-ID calculation. You can verify the fix by triggering the process (DE_B_GPKE_DSO_EOS), manually updating the operand value, and then triggering the move-in process to check that the Article-ID is calculated correctly.
