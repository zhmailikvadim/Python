Symptom

  • Catalog creation in Launchpad Content Manager (/UI2/FLPCM_CUST) fails with the error: “Catalog ‘ZXXX’ was not created because an error occurred.”
  • Catalog creation in Launchpad Content Manager (/UI2/FLPCM_CUST) fails with the error: “Catalog CAT: ZXXX: could not read”

Environment

SAP S/4HANA 2021 and higher

Reproducing the Issue

  1. Login to the SAP GUI screen, execute /n/UI2/FLPCM_CUST
  2. Click on Create, Enter New ID and Enter New Title 
  3. Click on Continue

Resolution

Implement SAP Note 3542599 and then invalidate the global caches in accordance with SAP KBA 2319491 

Keywords

Catalog, Launchpad Content Manager, UI2, FLPCM, Could not read, was not created because an error occurred, Z Catalog, Fiori, FLP, Launchpad, Gateway, NetWeaver, S4HANA, SAP GUI, /UI2/FLPCM_CUST
