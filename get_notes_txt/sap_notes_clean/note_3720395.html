Symptom

  • A data migration phase EU_CLONE_MIG* failed.
  • The option "Re-prepare: Keep the current status" is available in SUM UI.
  • New table split options are expected for the failed tables.

Environment

SAP ABAP System

Cause

New table split options are expected.

Resolution

1. Refer to the SAP Note 3386079 and set the new split options.

2. Repeat the failed step in SUM UI with the option "Re-prepare: Keep the current status".

See Also

3386079 - SUM DMO scenarios: Manual Options for Table Splitting and Table Migration

Keywords

table split, manual table split, EU_CLONE_MIG_DT_RUN,EU_CLONE_MIG_OPTDMO_INI_RUN,EU_CLONE_MIG_SOT_RUN
