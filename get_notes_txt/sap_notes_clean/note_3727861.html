Symptom

Business authorization setup at SAP IBP.

Environment

SAP Integrated Business Planning

Resolution


1. For authorization to use AI features in SAP IBP, your business role must contain the Natural Language Interface Services (SAP_IBP_BC_DS_NLI_PC) business catalog
this is the prerequsite for Joule.
In addition, 
2. For authorization to use AI-assisted inventory analysis, your business role must contain the Inventory Algorithm Analysis Component (SAP_IBP_BC_INV_ALGO_PC) business catalog.
3. For authorization to use AI-assisted MRO inventory analysis, your business role must contain the MRO Inventory Analysis Component (SAP_IBP_BC_MRO_COMP_PC) business catalog.
4. For AI-Assisted Supply Optimization Analysis, there is no dedicated catalog, but SAP_IBP_BC_DS_NLI_PC needs to be set Unrestricted or Restricted. The TSOPT_EXPL value must be maintained in the Generative AI Features (IBP_AIUCSE) field for the Generative AI Features (IBP_GENAI) restriction type.
5. For AI-assisted forecast analyses, Forecast Details Component (SAP_IBP_BC_FCST_LOG_COMP_PC) is necessary


Please note

  • If the value for the Plug-In Activation State property of this communication arrangement SAP_COM_0882 is empty, this means that Joule is available for all users of that system. To give access only to specific users, change the property value to ROLE and add the User Interface - Joule Plugin (SAP_CORE_BC_UI_JOU_PC) business catalog to the users' business roles.

  • If the value for the Plug-In Activation State property of this communication arrangement is ROLE, Joule is only available for users who have the User Interface - Joule Plugin (SAP_CORE_BC_UI_JOU_PC) business catalog in their business roles.
