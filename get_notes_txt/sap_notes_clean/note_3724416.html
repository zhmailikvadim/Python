Symptom

When removing an assignment of a hierarchy node that is defined as an interval, the upper boundary value of the interval is not passed to the validation logic. Because this boundary is missing, the system interprets the change as a single-node update rather than an interval, leading to incorrect or incomplete validation behavior.

Other Terms

CL_USMD_MDF_HRY_ADAPTER, CL_USMDZ7_RS_ENTITY, ADJUST_HIERARCHY, Validation, Hierarchy, Remove, Interval

Reason and Prerequisites

The upper boundary value of the interval was not passed to the delta table.

Solution

To apply the fix, implement this SAP Note in your system.
