Question

How should the ChargeExceptionTypeValidation.csv file be loaded into the Ariba system?

Answer

In case of a Federated Process Control (FPC) realm, the ChargeExceptionTypeValidation.csv should be loaded into the parent realm through the Data Import/Export option. Since it uses a replicated adapter source, loading the modified file directly into the child realm will not update the values. 

In case of non-FPC realms, the file can be loaded directly into the realm through the Data Import/Export option. 

  1. Go to Manage -> Core Administration
  2. Go to Data Import/Export under Site Manager. 
  3. Switch to the Export tab and search for Export Charge Exception Type Validation Data.
  4. Make the necessary modifications with the help of the values defined in the Data Dictionary.
  5. Go to the Import Tab and search for Import Charge Exception Type Validation Data.
  6. Upload the file and click on Run.
