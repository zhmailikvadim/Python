Symptom

  • Income Tax Bill 2025 - published on February 8, 2025 by the Lok Sabha, Government of India.

  • New wage code Rules 2025 for India - legal change The Code on Wages, 2019, published on July 18, 2019 by the Lok Sabha, Government of India. 

  • New Income Tax rule as notified by CBDT dated 20/03/2026.

  • Income tax rules effective from 1 April 2026 

  • Meal & Voucher increase from Rs 50 to Rs 200 per meal

  • Bengaluru, Hyderabad, Pune and Ahmedabad cities eligible for 50% HRA exemption 

  • The perquisite value is revised to ₹5,000 per month for engine capacity up to 1600 cc and ₹7,000 per month for engine capacity above 1600 cc.

  • The tax-free limit for interest-free or concessional loans is revised to ₹200,000.

  • Education and Hostel Allowance Changes

  • Landlord Validation: For HRA claims, employees must declare if the landlord is a relative.

Environment

  • Human Capital Management (HCM)
  • Payroll India
  • SAP R/3
  • SAP ERP Core Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA

  • Make sure you have applied the latest versions of the SAP Notes above so that all changes are fully implemented and error-free.
  • There are the minimum Support Package (SP) levels required to implement the SAP Notes. The solution must be implemented on these versions or higher, as it is not feasible to implement it on lower versions.
