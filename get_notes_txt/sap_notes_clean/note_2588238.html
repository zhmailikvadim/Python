Symptom

JOB PRE-RUNNING ACTION FAILED - FAILING JOB -- JCS-122035: Unable to persist: JCS-102200: Runtime limit Maximum for process <process id> already exists

Environment

SAP Business Process Automation 9.0 by Redwood

Cause

A pre-running action that causes a process to reschedule would fail if the Process Definition defines a Runtime Limit with the error "Runtime limit already exists"

Resolution

As per note 2254370 this is known issue and fixed in Version 9.0.16.0
