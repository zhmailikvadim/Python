Symptom

New laws in certain countries related to gender values require the HR system to be compliant. These legislations mandate specific gender values. In Employee Central certain technical change have to be made to comply with this.

Examples of gender values based on new legislations:

  • India: Male, Female, Transgender
  • Germany: Male, Female, Diverse

Other Terms

Employee Central Gender

Reason and Prerequisites

The standard field “Gender” in Employee Central can have either 2 or 5 values.

These are:

  • Male, Female
  • Male, Female, Unknown, Undeclared, Others

The values Unknown, Undeclared are used when the gender values are not known to the admin or not declared by the employee. In certain countries like above there are specific values which can be referred as others. This field in Employee central is not a picklist and cannot be customized. New picklist values cannot be added to this field. Successfactors talent modules have only Male and Female as possible values.

Solution

The IDP document provides recommendation on how to handle additional gender values in Employee Central in a step-by-step way for complying to recent legal regulations in countries like India and Germany. It also provides a guidance on how to handle mobility scenarios considering countries with different gender regulations.

Refer to IDP Employee Central: Recognizing Gender Diversity, also available on SAP Help portal, Partner Edge and Customer Community.
