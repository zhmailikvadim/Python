Symptom

When Latest Performance Form Experience enabled, Manager is unable to view subject ratings provided by the Employee; Side-by-side view is not working.

Environment

SAP SuccessFactors Performance & Goals

 

Reproducing the Issue

1. Proxy as a Manager and access the form.

2. Only Manager rating (item-rating) is displayed, Side-by-side view is not working.

Cause

The Manager has no access to Gap Analysis or Ratings from Others.

Resolution

Per the guide EZ Rater (Latest Form Experience) | SAP Help Portal:

When users have permission for Gap Analysis and Ratings from Others, they automatically have the side-by-side view and gaps in the form sections. 

Make sure permissions are granted to access both Gap Analysis and Ratings from Others.
