Symptom

Error message: Invalid values for Completed By Username is thrown while using Activity Item Completion Upload 

Environment

Integration - Uploads

Reproducing the Issue

  1. Attempt to mark activities as completed via Activity Item Completion Upload
  2. Enter a valid username in the Completed By Username field.
  3. Ensure the activity is in Pending status.
  4. Submit the transaction.
  5. Observe the error messages: Invalid values for Completed By Username

Cause

Completed By Username is not a mandatory field for action Complete

Resolution

  1. Remove the values entered in the Completed By Username field.
  2. Refer to the user guide for activity item completion upload for further clarification.
  3. This field is only mandatory for when action is Verify or Reject not for action complete.

See Also

Refer to: SAP Fieldglass Integration User Guide
