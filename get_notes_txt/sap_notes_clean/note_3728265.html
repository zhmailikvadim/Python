Symptom

Via Public API, it is possible to create bulk materials with only the logistics role transported set. This shall not be possible. This SAP note provides a new validation to prevent creating such bulk materials.

Other Terms

Chemical Compliance Information Public API, logistics roles, transported only, validation

Reason and Prerequisites

This symptom is caused by a program error.

Solution

Implement this SAP Note or import the corresponding Feature Packages Stack for SAP S/4HANA.
