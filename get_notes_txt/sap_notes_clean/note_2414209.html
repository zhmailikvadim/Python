Symptom

You post or activate a flow in the Business Operations Workplace. A termination occurs and the system issues the error message "Internal error; Calculated nominal is inconsistent; CoCode &1, ID &2" (TPM_CAL 004).

The error occurs only for contracts with formula-based conditions.

Other Terms

Distributor, TPM_CAL 004, TPM_CAL004

Reason and Prerequisites

Program error

Solution

Implement the attached correction instructions or import the specified Support Package.
