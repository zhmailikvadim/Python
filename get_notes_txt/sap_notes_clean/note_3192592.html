Symptom

  • Details of a Spring Framework vulnerability were published under CVE-2022-22965. It is also known as SpringShell or Spring4Shell.
  • Is there any impact on ASE or exposure to this issue in Replication Server?

Environment

SAP Replication Server (SRS)

Cause

This is a third-party security issue which, at time of writing, is documented in these external websites:
https://spring.io/blog/2022/03/31/spring-framework-rce-early-announcement
https://nvd.nist.gov/vuln/detail/CVE-2022-22965

NOTE : the content of these sites are not within the control of SAP and may change!

Resolution

SRS is not impacted by or exposed to this CVE.


For CVE-2022-22965 to be exploited it requires JDK 9, which RS does not use.

SAP's Security team is actively monitoring this CVE and will provide updates if further details come available.
