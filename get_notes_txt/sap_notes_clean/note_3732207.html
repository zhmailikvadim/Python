Symptom

  • The program RM06REOS terminates with error stating - "No component exists with the name "LKOMS", but there is a component with similar name "LKOMP"."

Environment

  • SAP ERP 6.0

Reproducing the Issue

  1. Go to transaction SE38 in SAP GUI.
  2. Enter program RM06REOS and choose Display.
  3. Execute Syntax Check (Ctrl+F2).
  4. System raises error: "No component exists with the name "LKOMS", but there is a component with similar name "LKOMP"."

Resolution

  1. To fix this issue in aggregation of confirmation, please use report RM06CMPs, which is automatically called in report RM06REOS. 
  2. Prerequisites - Please note that the above solution is specific to aggregation of confirmation records. 

Keywords

RM06REOS, "LKOMS", "LKOMP", aggregation of confirmation records
