Symptom

  • In the new layout of the latest people profile, the Effective Date is no longer visible on the MDF object summary card

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Environment

SuccessFactors HCM Suite

Reproducing the Issue

  1. Log in to instance
  2. Go to an employee profile
  3. Select any Custom/MDF object
  4. Observed that the new layout card view where the Effective Date is no longer displayed.

MDF Summary card showing on the old people profile

MDF Summary card showing on the Latest People profile

Cause

The Effective Start Date and External Code (for the root object) are not displayed on the summary card in the new layout. This is an expected behavior as per the MDF configurations for the Latest People Profile.

Resolution

This is an expected behavior where the Effective Start Date and External Code (for the root object) are not displayed on the summary card in the new layout as per the MDF configurations for the Latest People Profile.
