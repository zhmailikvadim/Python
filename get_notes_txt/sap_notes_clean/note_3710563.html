Symptom

Error "Transport cannot be released from SAP S/4HANA quality system" while releasing the transport.

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental." 

Environment

  • SAP Central Business Configuration 
  • * Valid for 2SL systems only

Reproducing the Issue

  1. Open CBC system.
  2. Navigate to the workspace.
  3. Attempt to release transport.
  4. Observe error message indicating failure to release transport.

Cause

Mandatory activities before releasing the transport has to be completed first.

Resolution

  1. Verify status of the below activities are in completed state or not in the workspace.
    • Request Production System". 
    • Assign Production System".
    • Configure Your Business Processes".
  2. Ensure that the activities are completed.
  3. Attempt to release the transport again.

For more information follow SAP Central Business Configuration Help Portal - Releasing Configuration into Production System (2-System Landscape).
