Symptom

This SAP KBA is to describe the difference between security_audit log and security log of the SAP NetWeaver Application Server (AS) Java.

Environment

SAP NetWeaver Application Server (AS) Java

Resolution

The security_audit log records creation, modification and deletion of users, groups and roles, while security log contains important security events, such as successful and failed user logons.

Can find the security audit log and the security log on path:
usr/sap/<sid>/<instanceID>/j2ee/cluster/serverx/log/system/security_audit_xx.x.log 
usr/sap/<sid>/<instanceID>/j2ee/cluster/serverx/log/system/security_xx.x.log

For more information, please see Security Audit Log of the AS Java
