Symptom

You want to post a goods receceipt of a structured article with a quantity of 0; you set the Delivery Completed indicator.

When you try to post that goods receipt, you get the error message "M7 001 table XPXVAF no entry 0001"

Other Terms

MIGO M7007

Reason and Prerequisites

Program error in note 3362011: The check implemented in that note was too restrictive.

Solution

Implement the corresponding support package or implement the correction instruction
