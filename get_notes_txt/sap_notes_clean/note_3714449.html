Symptom

  • Application error in called function module.
  • An exception with the type "cx_sy_itab_line_not_found" was raised, but was not handled locally or in a RAISING clause.
  • The table does not contain a row with the specified values.
  • Records cannot be posted to the target system; processing aborted.
  • Error during transfer; processing aborted.
  • Error code: sy-subrc = 4.

Environment

  • SLT Landscape Transformation Replication Server
  • Replication in CFIN scenario

Cause

The records in Source table have field "TRANSACTION_TYPE" as "Z - Delete Parked Document". This is related to the replication of parked documents. The target is unable to process those records because the related code for the parked documents doesn't exist there. 

Resolution

  1. Implement SAP Note 3467805 - Parked documents in Central Finance System - overview and ensure all prerequisites mentioned in the note are applied in both source and target systems. Or upgrade the target system S/4 HANA version to include the necessary corrections for handling transaction type "Z".
  2. If replication of parked documents is not required, create SLT filters to exclude parked documents from replication.
