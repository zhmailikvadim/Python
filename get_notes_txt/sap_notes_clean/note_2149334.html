Symptom

When RFIDYYWT is executed for Generic Withholding tax reporting, it  results in a dump.

Other Terms

RFIDYYWT, IDWTCOMPCD

Reason and Prerequisites

This happens because few of the fields in the company code structure  are not able to hold the value.

Solution

The Data Element of the following fields in the company code structure has been modified.

1. DMBTR
2. WT_QSSHH
3. WT_QBSHH
4. WT_QSFHH
5. WT_TAXHH
6. WT_AMEXCLVAT
7. WT_AMRBASE
8. WT_AMREXMP
9. SKNTO

 

Refer to manual activity section to apply the solution.
