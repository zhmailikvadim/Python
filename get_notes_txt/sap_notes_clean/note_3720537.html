Symptom

During the simulation of migration object 'Enterprise Project', below errors are thrown:

  • Project #ProjectID already exists, CJ 10.
  • WBS element #ProjectID already exists, CJ 20.
  • This Project ID exists. Enter another one, PSS4_MSG 6.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Open application 'Migrate your Data'.
  2. Select respective migration project.
  3. Select migration object 'Enterprise project'.
  4. Simulate.

Cause

Using Enterprise Project ID in both Project General Data sheet as well as in the WBS sheet, will cause this issue.

Resolution

  1. Review the template to identify the Enterprise Project ID* if it is used in both the Project General Data sheet and the WBS sheet.
  2. Remove the Enterprise Project IDs from the WBS sheet.
  3. Simulate the migration again using the correct template.
