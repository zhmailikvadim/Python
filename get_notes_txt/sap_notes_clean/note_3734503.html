Symptom

When checking the custom fields in Record Inspection Results - Hierarchical Worklist (QE51N) app, the custom fields are greyed-out and not editable.

Environment

SAP S/4HANA Cloud Public Edition

Cause

This is system standard design.

In Record Inspection Results - Hierarchical Worklist app, it's possible to include the extension fields of the business context Quality Management: Inspection Lot. However, the contents of the fields can only be displayed but cannot be changed. 

Resolution

Change the custom fields in Change Inspection Lot app as a workaround.
