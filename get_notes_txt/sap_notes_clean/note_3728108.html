Symptom

In Fiori app. Manage Documents (F2733) the ACL authorization tab is missing 

Environment

  • SAP Document Management System (CA-DMS)
  • SAP S/4HANA on-premise
  • SAP S/4HANA Cloud Private Edition 

Reproducing the Issue

  1. Open Manage Documents Fiori app.
  2. Select Document 
  3. The ACL authorization tab is missing

Cause

  • The current "Manage Documents" Fiori app does not offer the ACL authorization tab in the standard design
  • ACL functionality is supported in SAP S/4HANA but only for GUI and WebGUI (SAP GUI for HTML) transactions

Resolution

Consider raising this limitation as an idea for improvement to influence the SAP PLM and DMS portfolios. Refer to: SAP Influence Campaign.

Keywords

Manage Documents, Fiori app, ACL authorization, SAP GUI, CV03N, ACL functionality, ACL user group, Document Management, S/4HANA, ACL Access Control list, Document authorization, WebGUI
