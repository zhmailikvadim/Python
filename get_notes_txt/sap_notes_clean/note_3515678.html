Symptom

The OF rows of Indonesia VAT OUT report of Indonesia are not appearing in the CSV file

Environment

  • SAP S/4HANA
  • SAP S/4HANA Finance

Reproducing the Issue

  1. Go to run Compliance Reports；
  2. Choose Report Name: ID_VAT_OUT；
  3. Click line item Period；
  4. Click New Run button；
  5. Fill up Fiscal Year and Document Number；
  6. Click Run button；
  7. The OF row could not be displayed.

Cause

 If the tax codes are different between billing document and invoice document, the OF row could not be displayed. 

Resolution

Please notice this is a standard system behavior, if the tax codes are same between billing document and invoice document, users can see OF rows in the CSV file.

Keywords

VAT_OUT; ACR; Indonesia; S4HANA; OF row.
