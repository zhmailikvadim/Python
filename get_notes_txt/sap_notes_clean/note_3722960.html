Symptom

In the process "Sending of 55137 from Authorized DSO to SUP" (DE_B_MDC_55137_DSO2SUP_SEND), the cloud outbound UTILMD 55137 payload contained an incorrect period in the RFF+Z47 segment.

This issue is relevant for all market roles.

Other Terms

MDC, UTILMD 55137, RFF+Z47, receiver, data quality, consolidation logic, DE_B_MDC_55137_DSO2SUP_SEND, authorized DSO

Reason and Prerequisites

The error occurred because the consolidation logic didn't consider the receiver's period when calculating data quality.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the system generates the outbound UTILMD 55137 payload with the correct period in the RFF+Z47 segment.

You need to retrigger the process.

You can verify the fix by checking that the data is sent to the market partner with the correct period in the RFF+Z47 segment.
