Symptom

If there are no condition types for extended withholding tax in the settlement document, the data for the extended withholding tax for the vendor/customer is transferred to Accounting so that the calculation takes place there.

Up to now, however, this occurs only if the document also contains a tax posting. However, there are countries for which no tax may be calculated (for example, for rebate settlement) and therefore no tax code is set (for example, Indonesia). Here, the withholding tax data must also be determined for tax-relevant documents (indicator "Calculate Taxes" in the settlement process type) if there is a G/L account line for the vendor/customer line item.

Reason and Prerequisites
