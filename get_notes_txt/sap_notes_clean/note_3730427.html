Symptom

There is the need to provision users and groups from Microsoft Entra ID to SAP Cloud Identity Service

Environment

  • Identity Provisioning Service (IPS)

Reproducing the Issue

Cause

Resolution

There are two approaches to provision users from Microsoft Entra ID to SAP Cloud Identity Service:

  • Approach 1: There is no need to create IPS source and target systems for this approach.
    • Configure SAP Cloud Identity Services for automatic user provisioning with Microsoft Entra ID 

  • Approach 2: The source and target systems need to be created in IPS tenant.
    • Provision users from Microsoft EntraID to SAP Cloud Identity Services

See Also

  • Source system - Microsoft Entra ID
  • Target system - Identity Authentication
  • KBA 3297563 - How to configure Identity Provisioning with Microsoft Entra ID as source system and Identity Authentication as target system
