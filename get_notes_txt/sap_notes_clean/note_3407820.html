Symptom

I cannot make any changes to Request For Quotations (RFQ).

Cause

  • The ordering party has elected not to receive proposals for alternate prices or dates.
  • The time on the RFQ has run out.
  • The shipper may have closed out the RFQ.

Resolution

If you are not able to make changes to an RFQ, please try the following:

  • Check if the ordering party has elected not to receive proposals for alternate prices or dates, outlined here, Accepting a Freight Request for Quotation with a Change.
  • Check if time has run out on the RFQ.
  • Check if the ordering party has closed out the RFQ.
