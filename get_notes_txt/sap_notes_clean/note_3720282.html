Symptom

Component: Product Safety
Module: Service administration

When you start transaction CGSADM to manage EH&S WWI and Expert services, the system issues the error message GW 016: Exception &1 not checked; GWY_READ_CONNECTIONS.

After this error occurs, if any application performs an RFC call, the system displays all such calls with the status 'on hold' in the transaction SM50. To resolve this, you must restart the Gateway service.

Other Terms

EHS gw/monitor
Gateway Monitor SMGW

Reason and Prerequisites

Reason
Program error.

Prerequisites
For information about the validity of the corrections, see the correction instructions.

Solution

The section "Support Packages" specifies the support packages that contain the corrections.

Alternatively, you can implement the attached correction instructions.
