Symptom

You are processing a Credit Limit Request or a Documented Credit Decision in the related Fiori app and you get the error message "In the context of Data Services an unknown internal server error occurred".

AND

You are viewing a Credit Limit Request or a Documented Credit Decision in the related Fiori app and the log section shows the message "No items available".

AND

In the SAP Gateway Error Log you see messages with these characteristics:

  • Application Component: FIN-FIO-CCD-CR
  • Service Name: either UI_CREDITLIMITREQUEST_MAN or UI_CREDIT_DECISION_MANAGE
  • Error Text: "Property 'CaseLastChangedOn' has invalid value '20251215083560'"

Other Terms

F5602, Manage Credit Limit Requests, F5587, Manage Documented Credit Decisions, /IWFND/ERROR_LOG

Reason and Prerequisites

The CONV timestamp conversion from long format to short format can round up, which can produce invalid seconds value 60.

Solution

Please apply the correction instructions.
