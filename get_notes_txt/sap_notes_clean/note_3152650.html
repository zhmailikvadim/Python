Symptom

In the T-Code: REISCDCF, it is possible to mark the cash flow items as posted manually for the desired periods. But after that, the the amount of the local currency shows as ZERO for those cash flow items.

Environment

  • Financial Accounting (FI)
  • SAP R/3
  • SAP R/3 Enterprise 
  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA Finance
  • SAP S/4HANA

Reproducing the Issue

  1. Create an contract in with T-Code: RECN;
  2. Check the cash flow with T-Code: REISCDCF, mark the cash flow as posted manually;
  3. Check the cash flow again, will find  the amount of the local currency is Zero.

Cause

It is the standard system behavior. The local amounts in the cash flow items are filled only when the items are posted with a posting process, but not if they are just marked as posted.

Resolution

If it is necessary,  you can have a custom logic by using BADI_REIS_LIST to display the local currently amount in T-Code: REISCDCF for marked as posted cash flow items.
