Symptom

When trying to save a new infotype 0088 record with the 'Died' field left blank, the following error message is displayed incorrectly on the HR Renewal/ Mashup screen - or in PA30 when the New Infotype Framework is activated:

T5GPBS_CS_CODES, Code Set &1 (&2) does not contain code &3 on &4 table T5GPBS_CS_CODES

Other Terms

IT0088, HR Renewal, Mashup, ECP, New Infotype Framework, NITFW, ASPP_MBDOD, Died, Baby Died, e007(hrgb), HRGB/007, Code Set SXP_MSTLL (KEY1) does not contain code  on [00.00.0000] table T5GPBS_CS_CODES, Code Set &1 (&2) does not contain code &3 on &4 table T5GPBS_CS_CODES

Reason and Prerequisites

Prerequisite SAP Note: 3691082 - PY-GB: Changes to the IT0088 renewal screen (SAP-HR) 

Solution

Please implement the correction instruction(s) attached to this SAP Note.
The correction will be also delivered with the SPs mentioned below.
After implementing the relevant listed SP, this SAP Note will no longer need to be implemented.
