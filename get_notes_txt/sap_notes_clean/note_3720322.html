Symptom

Poland SAF-T JPK_VAT report generated with error:

"Value is less than or equal to the min. permitted value 0"
Element name: tns:Pouczenia

Environment

SAP S/4HANA Cloud Public Edition.

Reproducing the Issue

  1. Navigate to the app "Run Statutory Reports".
  2. Select the report name "Poland SAF-T - JPK_VAT Monthly".
  3. Observe the error message: "Value is less than or equal to the min. permitted value 0"  tns:Pouczenia.

Cause

The terms and conditions of the SAF-T PL VAT Declaration Manual Input fields activity were not accepted.

Resolution

Mark the checkbox "I accept the terms and conditions" under Legal Disclosure (POUCZENIA) when running the SAF-T PL VAT Declaration Manual Input fields activity.

Refer to the Help Documentation Generating the SAF-T JPK_VAT with Declaration (JPK_V7M) - Monthly Report for additional details.
