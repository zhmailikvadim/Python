Symptom

When performing a Cloud Connector upgrade to version 2.18.2, you receive error message below:

error: Failed dependencies:

        libstdc++.so.6(GLIBCXX_3.4.20)(64bit) is needed by com.sap.scc-ui-2.18.2-6.x86_64

        libstdc++.so.6(GLIBCXX_3.4.21)(64bit) is needed by com.sap.scc-ui-2.18.2-6.x86_64

 

Environment

SAP Cloud Connector 2.18.2

 

Reproducing the Issue

Upgrade of Cloud Connector to version 2.18.2 as per SAP HELP Upgrade | SAP Help Portal

Cause

You are using an Operating System that is not compatible with Cloud connector version 2.18.2

Resolution

Check the compatibility of the Operating System with the Cloud Connector version on Prerequisites | SAP Help Portal  and use an OS that is compatible with Cloud Connector version.

Please also check KBA 3669949 - Operating System upgrade when running an SAP Cloud Connector(SCC)
