Symptom

When trying to save a new Function ID with special characters the error below is triggered:

  • Message no. 122 - Invalid characters in the function ID"

Environment

  • SAP Access Control 12.0
  • SAP Access Control 12.0 for SAP S/4HANA

Reproducing the Issue

  1. Go to NWBC
  2. Access Management
  3. Function
  4. Name function with special characters
  5. Add permissions and Actions
  6. Save

Cause

Standard behavior from application

Resolution

After GRC 10.0 Function IDs have a naming convention, they can be 8 character long and only accepts the following characters:

-/ABCDEFGHIJKLMNOPQRSTUVWXYZ_0123456789

See Also

For further information upon Function IDs naming and creation process, refer to documents below:

  • SAP Help Portal - Access Control: Creating a Function
  • 3723341 - Generate rules not working for function with Special characters
  • SAP Note 1722872 - Function ID cannot be created with colon ":" character
