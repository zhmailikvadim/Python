Resolution

You can find Enterprise Continuous Testing by Tricentis 2024.2 Patch 11 at the following location:

  1. Go to Software Center - SAP for Me
  2. Select tab: SUPPORT PACKAGES & PATCHES
  3. Navigate to By Category -> SAP SOLUTION EXTENSIONS BY PARTNERS
  4. Now, search for SAP ENTRPRISE CNTNUS TSTNG OP
  5. Select SAP ENTRPSE CNTNS TSTNG OP 2024.2
  6. CLOUD SERVICE MANAGER 2024.2  ->
    CSM_2024.2_11_11-70009123.EXE
    SP 00 PL 11 for CLOUD SERVICE MANAGER 2024.2  --  This is TOSCA Server

    TRICENTIS TOSCA COMANDER2024.2 ->
    TOSCA_COMANDER_11_11-70009122.EXE
     
    SP 00 PL 11 for TRICENTIS TOSCA COMANDER2024.2  -- This is TOSCA Commander
