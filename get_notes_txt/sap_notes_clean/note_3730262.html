Symptom

You get error "One or more objects are dependent upon this object" when trying to delete a HANA object for example a HANA calculation view.



Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

  • SAP HANA, platform edition 1.0
  • SAP HANA, platform edition 2.0

Cause

The object to be deleted has been referred by another DB object for example, HANA calculation view, analytic privilege, etc.

Resolution

Read KBA 3729730 - How to check the object Dependencies for Inactive Base Objects in HANA XS classical Repository to determine the dependencies with other objects, remove the dependencies, and try the deletion again.

Keywords

One or more objects are dependent upon this object, HANA object cannot be deleted, HANA view cannot be deleted
