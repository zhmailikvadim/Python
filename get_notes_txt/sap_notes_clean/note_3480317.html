Symptom

Some changes made in Printer dialog window do not work, when printing from SAP Business One.

For example, orientation, watermark or color settings.

Reproducing the issue

Scenario:

  1. Run a crystal report in SAP Business One. For example, preview a sales order with any crystal report template.
  2. Go to File → Print, Printer window is opened.
  3. Select a local or redirected printer, click Preferences. 
  4. Change some default settings. For example, change orientation, add a watermark or change the color. 
  5. Save the changes in Preferences and click Print.

Expected behavior:

Changes made in step 4 are reflected in printout.

Actual behavior:

No change in printout.

Cause

System Limitation

Solution

This is a system limitation.

Workaround:

Please change the settings in the layout or Report and Layout Manager, do not change it from Printer dialog box.
