Symptom

It is necessary to add custom fields to the dynamic selection in F.01 transaction or FC10N.

Environment

  • SAP ERP
  • SAP ERP Central Component
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP_FIN 618

Reproducing the Issue

  1. Open transaction F.01 or FC10N.
  2. Attempt to create a variant for the report RFBILA00.
  3. Check the dynamic selection drop-down for the custom field
  4. Observe that the field is not available under dynamic selection.

Cause

In RFBILA00 or RFBILA00N only fields that are added to the totals table FAGLFLEXT, can be extended to FAGLFLEXT_FS  structure, which makes them available in dynamic selection.

Resolution

  1. Confirm that the New General Ledger functionality is activated by checking the table FAGL_ACTIVEC in transaction SE16 and ACDOCA table is not yet used.
  2. Make sure that the custom field is added to FAGLFLEXT table, which is read by F.01 and FC10N.
  3. Extend FAGLFLEXT_FS structure in SE11 with the custom field.

See Also

If you are on HANA and using ACDOCA please refer to the following note instead: https://me.sap.com/notes/2551817
