Symptom

In an S/4HANA system, the following SWU3 customizing activities cannot be executed manually or automatically without errors:

  • Schedule Background Job for Workflow Deadline Monitoring
  • Schedule Background Job for Work Items with Errors
  • Schedule Background Job for Condition Evaluation
  • Schedule Background Job for Clearing Report

The traffic lights remain red.

Other Terms

 

Old job name  New job name (S/4HANA)
SWWDHEX SAP_WORKFLOW_DEADLINE
SWWERRE SAP_WORKFLOW_SYSTEM
SWWCOND SAP_WORKFLOW_RULE
SWWCLEAR SAP_WORKFLOW_REORG

 

Reason and Prerequisites

This problem can only occur in an S/4HANA system. SAP Note 2185265 ignores the renaming of the workflow system jobs in accordance with SAP Note 2092495.

Solution

Implement the correction instructions.

Other Terms

Dump, FAGLGVTR, OBJECTS_OBJREF_NOT_ASSIGNED, CL_IM_FIAA_BW_DELTA_UPDATE, CL_IM_FIAA_BW_DELTA_UPDATE====CP, IF_EX_FIAA_BW_DELTA_UPDATE~UPDATE_POSTING_720

Reason and Prerequisites

Program error

You use BW delta extraction in FI-AA (implementation for the BAdI FIAA_BW_DELTA_UPDATE is active).

Solution

Implement the attached correction instructions or import the relevant support package.
