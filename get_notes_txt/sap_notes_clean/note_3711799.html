Symptom

This SAP Note contains a read OData service for Customizing and application data of losses in reinsurance, including the data for the linked external references. This SAP Note also contains an enhancement of the OData service for reading loss groups.

The main application is to enable the consumer of the service to read loss data, including the basic data, the linked loss group, the assigned cedents and coverages (treaty/single risk coverages), the responsibilities (PD-ORG), the external references, and the object assignments.

Other Terms

Web Service

Remote API

Reason and Prerequisites

Read claim data in reinsurance.

Troubleshooting description:
The following service exists in the system after you implement this SAP Note:

  • /MSG/API_LM_LOSS_MANAGE_O4: Loss - Manage

The following service has been adjusted:

  • /MSG/API_LM_LOSSGROUP_READ_O4: Loss Group - Read

Note that this service is currently not released for productive use. For this reason, non-compatible changes to the service may take place in the future. The service will be released at a later point in time.
