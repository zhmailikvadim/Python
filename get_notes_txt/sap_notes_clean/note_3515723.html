Symptom

The requirement is to void many (eg. over 1,500) cheques on average.

Resolution

There is no standard way to perform mass void in one run. The requested functionality is not available in the standard.

If a custom solution in Z namespace is needed, create a Z report that would call statement CALL TRANSACTION for FCH8 or FCH9 transaction multiple times with appropriate parameters based on the requirements. This way mass void of cheques can be achieved.

Keywords
