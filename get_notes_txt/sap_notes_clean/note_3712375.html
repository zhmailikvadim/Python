Symptom

In S/4HANA 2022 release version, when attempting to create a Handling Unit with serial numbers where the Serial Number Profile is configured with value 1 for the Stock Inconsistency Check flag, and is defined as a warning in the configuration. However, despite this, the Handling Unit API is returning an error.

Other Terms

S/4HANA OP2022, Handling unit, HU API, Serial Number Profile, HU_RAP 051, Stock Inconsistency Check

Reason and Prerequisites

Program Error

Solution

As a general rule, SAP recommends that you install a solution by applying a Support Package. However, if you need to install a solution earlier, use the Note Assistant to implement the correction instructions. You can find more information about the Note Assistant in service.sap.com/note-assistant.
