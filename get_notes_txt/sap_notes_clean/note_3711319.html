Symptom

Offers imported from external systems to SAP PMR via the function module /DMF/OPIF_OFFER_INBOUND cannot be transferred to ERP or S/4HANA. Offers are created and successfully saved with the status "Released", but the transfer button is not enabled for such imported offers.

Environment

SAP Promotion Management, SAP CARAB

Reproducing the Issue

  1. Import offers to SAP CAR using the function module /DMF/OPIF_OFFER_INBOUND.
  2. Ensure the logical system is assigned to the offers.
  3. Verify that the offers are saved in the database with the status "Released."
  4. Transfer button is not enabled

Cause

The logic which prevents to transfer these imported offers is in class: /PRM/CL_OFFER_PWL_ACT_TRANSFERCP method: SET_CONTROL_PROPERTIES. The root cause why the transfer button is not enabled is due to the value of offer_tcd which is 02 (imported - complete). 

Resolution

The coding in SET_CONTROL_PROPERTIES method allows only offer_tcd 01 and empty for transfer, therefore it is not supported to transfer these offers in standard system.
