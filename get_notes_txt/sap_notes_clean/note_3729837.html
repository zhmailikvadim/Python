Symptom

Firefighter Log sync job dumps when it tries to collect UIDP logs from a plug-in system where GRCPINW component installed.

Other Terms

SAP GRC for SAP HANA, Access Control, GRAC_SPM_LOG_SYNC_UPDATE, Firefighter Log sync job, dump, ST22, CALL_FUNCTION_REMOTE_ERROR.

Reason and Prerequisites

 UIDP logs do not exist in the older plug-ins (V1200_750, V1100_700). When the code attempts to check them, it fails and results in a short dump.

Solution

Implement the attached correction instruction or update the relevant Support Package.
