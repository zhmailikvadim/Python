Symptom

When acting as a delegate and attempting to register a team member into a class from the Team View page, by clicking the 'Register' button, the message 'You cannot access the target page' is displayed.

Environment

SAP SuccessFactors Learning

Reproducing the Issue

  1. Log into the system >
  2. Navigate to the Learning Homepage >
  3. Select 'Act as' and choose the manager the user is a delegate of >
  4. Navigate to 'Team View' >
  5. Select a team member who has an ILT item assigned with a future class scheduled >
  6. Click on the 'Register' button >
  7. Observe the message 'You cannot access the target page' being displayed.

Cause

The 'Access Learning Plan' permission is missing under the 'Learning' section in the user's assigned role.

Resolution

Add the 'Access Learning Plan' permission to the user's assigned role.
