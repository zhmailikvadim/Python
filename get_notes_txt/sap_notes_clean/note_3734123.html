Symptom

This is a UI issue of capability picker. The competency names are overlapping each other in the capability picker. 

Cause

This is a known issue. 

Resolution

The fix for this issue is planned to be deployed in the 1H 2026 release (b2605), coming to Preview on April 13th 2026, and to Production on May 15th-16th.

For further information on release timelines, please review SAP SuccessFactors Product Release & Road Map Information.
