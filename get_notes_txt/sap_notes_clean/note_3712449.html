Symptom

When you try to retrospectively create eDocuments manually for documents that have already been posted, and you attempt to create eDocuments from multiple source documents that all clear the same invoice (e.g. in case of partial payments), the system creates an eDocument for only one of the source documents.

Reason and Prerequisites

Reason

This SAP Note is relevant only for Croatia.

Prerequisites:

First implement all the prerequisite requirements listed in the SAP Note 3499841 - Electronic Document Processing for Croatia: Implementation Overview

Solution

As a general rule, SAP recommends that you implement the Support Package specified for your release. However, if you need to install a solution earlier, use the SAP Note Assistant to implement the correction instruction. You can find more information about the SAP Note Assistant in SAP Support Portal, under SAP Note Assistant.
