Symptom

The forwarding agent is not picked up in transaction vt01n during shipment creation.

Environment

  • SAP ERP
  • SAP ERP Central Component
  • SAP R/3
  • SAP R/3 Enterprise
  • SAP enhancement package for SAP CRM, version for SAP HANA
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Start transaction VT01N.
  2. Enter Transport Planning Point: 3003.
  3. Enter Shipment Type: YDOM.
  4. Add a Forwarding Agent and observe that it is not picked up automatically.

Cause

The shipment route needs to be maintained for the Forwarding Agent to be triggered automatically

Resolution

  1. Run transaction 0VTC.
  2. Select the affected Route.
  3. Enter/maintain the default Forwarding Agent for the selected entry.
  4. Save the customizing.
  5. Test again in VT01N by creating a shipment with the relevant Transport Planning Point and Shipment Type to confirm the Forwarding Agent is now picked up.
