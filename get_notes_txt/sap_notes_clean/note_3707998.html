Symptom

  • After promoting Crystal reports from 4.2 to 4.3, the reports are not working
    • In Crystal Reports designer, get the error: Unknown Database Connector ErrorDetails: Failed to retrieve List of ValuesPrevious Error:Error returned from returned from Web Intelligence server: Component:WISErrorCode: 10901ErrorType: USERMessage:The following database error occurred: ORA-12154: TNS:could not resolve the connect identifier specified. For information about this error, please refer to SAP Knowledge Base Article 2054721 on the SAP Support Portal. (IES 10901)Prefix:ERR
    • In CMC, get the error: The viewer could not process and event. Failed to open the connection.XXXXX [] ----- Error code:0 [CRWEB00000119]
  • The report is based on UNV universe
  • The Oracle connection is working in Information Design Tool
  • No issue when creating a new Crystal report based on the same UNV universe 

Other Terms

crystal reports, CR, oracle, connection, ORA-12154, TNS, middleware, connection server, client

Reason and Prerequisites

Environment

  • SAP Crystal Reports 2020 Support Pack 04 Patch11
  • SAP BusinessObjects Business Intelligence Platform 4.3 SP04 Patch11
  • Oracle client

Steps to reproduce

  1. Promote a Crystal Report from 4.2 to 4.3 using a BIAR file, including all the dependencies (UNV universe and Oracle connection)
  2. Install 64-bit Oracle client and configure the TNS name in tnsname.ora file on BI 4.3 server machine
  3. Launch Crystal Report Client tool
  4. Log on to BI server
  5. Open the report
    =>>Fail to open with below error message: Unknown Database Connector ErrorDetails: Failed to retrieve List of ValuesPrevious Error:Error returned from returned from Web Intelligence server: Component:WISErrorCode: 10901ErrorType: USERMessage:The following database error occurred: ORA-12154: TNS:could not resolve the connect identifier specified. For information about this error, please refer to SAP Knowledge Base Article 2054721 on the SAP Support Portal. (IES 10901)Prefix:ERR
