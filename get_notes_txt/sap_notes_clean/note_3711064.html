Symptom

  • Error : "Navigation to this application is not supported" occurs when using copy functionality in Application 'Manage Routings'.
  • Create a Routing using the AI-Assisted Material Search fails with  "Navigation to this application is not supported"

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Go to Application 'Manage Routings'.
  2. Launch search without criteria.
  3. Select randomly a routing.
  4. Click on Copy button.
  5. Choose to create another group.
  6. Click Copy button.
  7. Observe error message: "Navigation to this application is not supported". 

Cause

Copy Routing and Find Similar routing IAM applications 'F9111_TRAN' and 'PPH_PRODNRTG_COPY_01_TRAN' is not activated in the used in Business roles of Business Catalog SAP_SCM_BC_PROC_ENG_MC.

Resolution

Follow the below steps to activate the IAM apps in the Business Catalog 'SAP_SCM_BC_PROC_ENG_MC' :

  1. Go to App 'Business Catalogs'.
  2. Search for business catalog ID: 'SAP_SCM_BC_PROC_ENG_MC'.
  3. Go tab Used in Business roles.
  4. Open each Business role ID one by one and activate the IAM Apps 'F9111_TRAN' and 'PPH_PRODNRTG_COPY_01_TRAN'.
  5. Once the IAM Apps are activated and saved, Retest the scenario.
