Symptom

  • Unable to add a Rule Assignment to a CDS view
  • Unable to add a field in LTRS to a CDS view
  • Cannot add CDS view; see long text

Environment

SAP Landscape Transformation Replication Server

Cause

The CDS view is not extraction enabled

Resolution

The CDS view must be enabled for extraction by having one of the two lines below - 

  • Analytics.dataExtraction.delta.changeDataCapture.automatic

  • Analytics.dataExtraction.delta.changeDataCapture.mapping
