Symptom

In Managed System Configuration step 4 "Assign Diagnostics Agent" function "Check All Host Agents" leads to the following error message:

"Host Agent check could not be performed; check details"

Other Terms

CX_SY_NO_HANDLER : An exception with the type CX_SY_RANGE_OUT_OF_BOUNDS occurred, but was neither handled locally, nor declared in a RAISING clause

Solution

Implement the correction instructions attached to this note or install the relevant support package.
