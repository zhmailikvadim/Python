Symptom

You reverse an activity type allocation in different period than the original posting period (cross period reversal). The frozen prices are different in the reversal period and in the original posting period. An error message occurs during in the order settlement or the cost center is not balanced after you run the actual costing run.

Other Terms

MLCCS009

Reason and Prerequisites

As as side effect of note 3199370, the check which detects reversal of cross periods posting was not executed.

Solution

Implement the attached corrections.
