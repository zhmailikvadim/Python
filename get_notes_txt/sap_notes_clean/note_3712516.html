Symptom

You use the documentation work station to manage your patient data. You first enter the data for the first next of kin of a patient, including the address and telephone data. You then also enter the data for another next of kin, but now without the address and telephone data. You then notice that the clinical overview now incorrectly displays the same address data and the same telephone number as for the first next of kin.

Other Terms

DWS, documentation work station, NV2000

Reason and Prerequisites

Program error

Solution

The system now correctly no longer transfers the fields for the address and telephone data of the first next of kin for the second next of kin of a patient.

Import the specified Support Packages to solve the problem. You can implement the corrections in advance.
