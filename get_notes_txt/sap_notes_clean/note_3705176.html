Symptom

While attempting to install or renew the Adobe Reader Rights credential, the following error occurs in NetWeaver Administrator (NWA):

“An object with the specified key (field Alias) already exists”

Environment

SAP Netweaver All releases

Reproducing the Issue

  1. Log in to NetWeaver Administrator (NWA).

  2. Navigate to:

    • Configuration → Security → Certificates and Keys

  3. Attempt to create or import a new Adobe Reader Rights credential.

  4. System throws the error:

    • “An object with the specified key (field Alias) already exists"

Cause

An older, expired Adobe Reader Rights credential (PFX/P12) already exists in the system.

Resolution

  1. Log in to NetWeaver Administrator (NWA).

  2. Go to:

    • Configuration → Security → Certificates and Keys

  3. Perform the following cleanup steps:

    • Remove old Adobe Reader Rights credentials from:

      • Manage P12 Files

      • Remove Selected Object tab

  4. Ensure no old credential with the same Alias remains in the system.

  5. Import the new Adobe Reader Rights credential (PFX/P12) again.
