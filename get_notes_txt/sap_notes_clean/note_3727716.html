Symptom

  • An error occurs when scheduling a Web Intelligence  report with WIDX format to the BI Inbox destination. The error message is: "write error. [Unable to get SI_MIME_TYPE property value]: [CrystalEnterprise.Managed]".

Environment

  • SAP BusinessObjects Business Intelligence Platform 2025
  • All Supported Operating System

Reproducing the Issue

1. Log in to BI Launchpad.
2. Open any Web Intelligence report and click Schedule.
3. In Report Features, go to Formats and select WIDX. 
4.Under Destinations, choose BI Inbox .
5. Click Schedule.

Cause

Due to the large file size of WIDX exports, the BI Inbox destination is not supported for this format.

Resolution

This is by system Design. Due to the large size and structure of the WIDX output format, the BI Inbox destination is not supported, which results in the scheduling failure.
