Symptom

In the Transportation Cockpit, you perform the following steps:

  1. Select several Freight Units
  2. Select one or more railcar resource
  3. Choose Load Consolidation → Selected Entries Only
  4. Load Consolidation Mode in load planning setting is 'capacity-based'

The load consolidation fails with the error message "Select at least one resource or equipment type."

The error occurs even though valid capacities with assigned resources have been selected.

Other Terms

Transportation Cockpit, Load Consolidation, Railcar

Reason and Prerequisites

Program Error

Solution

Implement the attached Correction Instruction or the mentioned Support Package
