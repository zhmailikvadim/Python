Symptom

Users encounter the error FCO_SETTLEMENTRULE/044 when creating a settlement rule via API_PROJECTSETTLEMENTRULE.

Environment

SAP S/4HANA Cloud Public Edition

Cause

The issue is caused by a program error.

Resolution

The issue has been acknowledged, and work is currently being carried out by the development team to identify the root cause and develop a correction. Currently the hotfix is under extensive testing and is projected to be deployed along with the Hotfix Collection 2602/8 on 19.04.2026.
