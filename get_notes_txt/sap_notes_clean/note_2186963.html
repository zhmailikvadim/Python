Symptom

HTML Viewer - Applying format on chart category axis or chart value doesn't work

Other Terms

WebI

Reason and Prerequisites

Steps To Reproduce:

  1. Open any Web Intelligence document in HTML viewer
  2. Format chart value with a number format (ex $12345)
  3. Reopen the format chart and select number format, you will notice only DEFAULT in the combo box
  4. Open any Web Intelligence document in HTML viewer
  5. Format chart category axis and change any parameter font, color
  6. Reopen the format dialog box and change any value
  7. Notice the OK and APPLY  remains disabled

Solution

The issue has been fixed in the patches listed in the "Support Packages & Patches" section below.
