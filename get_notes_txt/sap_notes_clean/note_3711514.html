Symptom

Steps on how to automatically deactivate inactive users on SAP Enable Now.

Environment

SAP Enable Now - Manager

Cause

Resolution

  1. Navigate to SAP Enable Now Manager.
  2. Go to Administration > Server settings > User management.
  3. Enable the option for automatic user deactivation.
  4. Set the desired time/interval for user deactivation.
  5. This configuration will automatically deactivate users who have not logged in within the specified time, thereby freeing up licenses.

Keywords

sap enable now, active users, retired users, user deactivation, user management, automatic deactivation
