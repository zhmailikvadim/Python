Symptom

When using the Generate Documents for E-Sign action via Docusign, neither the 'Sent to customer' nor 'Viewed by customer' flags are set, even if the document is opened in Docusign.

Environment

SAP CPQ

Reproducing the Issue

  1. Open Quote
  2. Generate Documents for E-Sign
  3. Generate Document 
  4. send the proposal to yourself using the option Send to Docusign

Cause

As per the current functionality, the “Sent to Customer” flag is updated only when the email is triggered through the Generate Document action flow. The “Viewed by Customer” flag is updated once the customer opens the shared link.

Resolution

Please note that these flags are not supported with the DocuSign integration, so they will not be updated for documents sent via DocuSign.
