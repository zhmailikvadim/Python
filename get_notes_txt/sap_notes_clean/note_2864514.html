Symptom

Work Center POD crashes with the error:

Fatal error while rendering UI component com.sap.ui.faces.component.sap.UIData with ID templateForm:tcArea:threepanelh:areaA:WorklistView:worklist_table in UIViewRoot /com/sap/me/wpmf/client/template.jsp. javax.el.ELException: java.lang.NullPointerException: while trying to invoke the method java.lang.String.equalsIgnoreCase(java.lang.String) of a null object loaded from local variable 'actualRouter'

Other Terms

Shop Order, worklist, Trackable SFC

Reason and Prerequisites

The issue may occur after the following steps:

  1. Set up POD to use Shop Order type worklist.
  2. Create Trackable SFCs.
  3. Open POD and try to work with created SFC.

Actual result: component render error displayed.

Solution

Solution provided as code correction. The error in POD has been fixed. However, it is not possible to work with trackable SFCs when Shop Order based worklist used. This type SFCs are not grouped by Shop Order.

Impact: POD.
