Symptom

This is an extension of note 3721464. With this note the determination of the use scenario is enabled also for the (not recommended) case where the field Offsetting Needs in Register in IMG activite Define Classification has the value Undefined

Reason and Prerequisites

This note is relevant for

  • Enhanced AFI Logic, formerly Accounting for Financial Instruments Based on Expected Cash Flows (ECF)

Solution

Implement the attached correction instructions or the corresponding support package
