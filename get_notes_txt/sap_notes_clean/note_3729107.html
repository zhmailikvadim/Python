Symptom

  • Deleted purposes and subscriptions are still appearing in user profiles within the "Details" tab in SAP customer data platform.
  • Removal of attributes from the customer schema did not remove relations to subscriptions from customer profiles.

Environment

SAP customer data platform

Cause

Removing deleted purposes and subscriptions data from profiles is not currently supported by SAP customer data platform.

Resolution

  1. Recreate the deleted purposes and subscriptions.
  2. Map the recreated purposes and subscriptions.
  3. Ingest events to revoke consent for the recreated purposes and subscriptions.
  4. Note that complete removal of deleted purposes and subscriptions from profiles is not supported.
  5. If required, submit a feature request through the appropriate channels to enable this functionality.
