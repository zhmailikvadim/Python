Symptom

  • The Batch field is not visible under page reference object when create or change quality notification in Transaction Code QM01/QM02.

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.  

Environment

  • SAP ERP Quality Management (QM)
  • SAP R/3
  • SAP R/3 Enterprise
  • SAP ERP Central Component
  • SAP ERP
  • SAP Enhancement package for SAP ERP
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA, on-premise
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Open QM01 (for example : Notification Type Q2).
  2. Input the material number.
  3. The Batch field becomes invisible.
  4. Input plant value and click enter, the batch field becomes visible.

Cause

Standard behavior, this material batch is managed on plant level (T-code OMCE).

Resolution

When material number is specified in the QM notification, the visibility of field batch number (VIQMEL-CHARG) depends on whether batch management is active for the material. 

If batch management is configured at the plant level, as defined in T-code OMCE, both the material and plant values are required to determine whether the material is managed in batches.
