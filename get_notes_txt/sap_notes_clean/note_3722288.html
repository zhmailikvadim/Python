Symptom

In the t-code KKPY , there is error message KV104 issued (Inconsistent units of measure).

Environment

  • SAP ERP
  • SAP S/4HANA

Reproducing the Issue

  1. Enter t-code KKPY
  2. Enter selection screen details
  3. Execute KKPY

Cause

The message KV104 for PCC is due to a change of the output UoM.

Resolution

Please check cost report KKBC_ORD cumulatively for PCC, the header shows actual output quantities for mixing UoM.

Keywords

KKPY, UoM, PCC
