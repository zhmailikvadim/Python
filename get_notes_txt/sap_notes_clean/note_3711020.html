Symptom

While adding members using the Quick Entry option in Fiori, the system allows duplicate members to be added without displaying the expected duplicate validation error message.

This note ensures that an error is displayed for duplicate entries added via quick entry.

Other Terms

/IRM/GMLM, document_data_process, ui_data_update, Quick Entry, /IRM/GML 007

Reason and Prerequisites

This is caused by program error.

Note : SAP Note#3710553 is available for ECC release.

Note : Please implement SAP Note#3605653 as a Pre requisite

Solution

Please implement the correction instructions. These corrections are only valid if you have applied the following Support Packs.

Release V/4 – 2022 V2, 2023, 2023 V2, 2023 V3 and Support Pack 1

Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object
