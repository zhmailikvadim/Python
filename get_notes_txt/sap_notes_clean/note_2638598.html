Symptom

Error message when running SAT, which has the error text like:

The following error occurred during the runtime measurement:
XXXXXXXX 4: trace level stack empty. Current record: ID=, SUBID=, Text=

Environment

SAP ABAP System

Cause

This is a program error

Resolution

Apply the kernle patch as mentioned in SAP Note 2613587.

See Also

  • SAP Note 2598287 - SE30: Trace record inconsistency for W message if automation calls were issued previously
  • SAP Note 2613587 -  Aggregated ABAP runtime measurement is aborted when performing an exit command  (7.49 PL 420 - 424)

Keywords

SE30, SAT, ST12, trace level, stack, empty, runtime, measurement
