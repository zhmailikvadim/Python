Symptom

The "delete" option is not available for archiving object AM_STEUER.

Environment

  • Financial Accounting (FI)
  • Asset Accounting (FI-AA)
  • SAP R/3
  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA Finance
  • SAP S/4HANA 

Cause

The deletion of archived data is controlled by the archiving object AM_ASSET.

Archiving object AM_STEUER cannot be run independently via transaction SARA, as it utilizes the same write program as archiving object AM_ASSET. Processing occurs automatically through archiving object AM_ASSET when the “Customizing Snapshot” indicator is selected on the AM_ASSET selection screen.

Resolution

Use the "delete" option in archiving object AM_ASSET.
