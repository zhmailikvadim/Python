Symptom

There is a message appears at the bottom of the layout management screen in both SAP GUI for Windows and WEBGUI. 

Report:BCALV_TEST_FULLSCREEN_EVENTS,Log_Group:,Handle:

* Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

SAP S/4HANA 2023

Reproducing the Issue

  1. Go to SE38
  2. input BCALV_TEST_FULLSCREEN_EVENTS
  3. Execute
  4. Settings
  5. Layout
  6. Layout management 

Resolution

This is actually a change as standard behaviour, and it is available since SAP BASIS 757. 

Implement note 3710742 - ALV: Tech Info moved from End-of-List to hidden columns in Layout Administration

This is Pilot Release note.
