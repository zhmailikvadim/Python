Symptom

The documentation for the following Customizing activities in the Customizing for Transportation Management (TM) is outdated or incomplete:

  • Define Transportation Relevance for Purchasing Documents
  • Define Transportation Relevance for Sales Documents
  • Define Transportation Relevance for Delivery Documents

Other Terms

LE, LE-SHP, Integration, SD, MM, PUR

Reason and Prerequisites

The activities can be found in the Customizing for TM under Integration -> Logistics Integration -> Internal TM Component Integration.

Solution

Implement this SAP Note or the relevant support package.

This SAP Note contains changes to the existing documentation that aim to better describe the activity and provide more information about the implementation logic and fields that are avialable in it.
