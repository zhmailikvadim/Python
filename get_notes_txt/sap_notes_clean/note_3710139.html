Symptom

When running report FP_TEST_IA_01, the following warning appears:
"This document is digitally signed. Some signatures couldn't be verified. View signatures"

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Environment

  • SAP S/4HANA
  • SAP S/4HANA FOUNDATION
  • SAP ERP
  • ABAP Platform
  • SAP NetWeaver
  • SAP ERP Central Component
  • SAP Solution Manager

Reproducing the Issue

Run report FP_TEST_IA_01 in transaction SE38 to test the generation of an Interactive Form.

Cause

The PDF controller configured in the SAP GUI is set to Microsoft Edge, which is not supported for previewing Interactive Forms. In terms of browsers, we can only ensure Interactive Forms can be previewed as expected in the Internet Explorer browser or Microsoft Edge with IE Mode ON, as per SAP Note 1918612.

Resolution

In order for this warning not to appear, change the PDF controller in GUI to Internet Explorer as described in the KBA below:

3258783 - How to change the PDF controller in SAP GUI

The message "This document is digitally signed. Some signatures couldn't be verified. View signatures" is just a warning and can be ignored. This does not mean that the Digital Signature is not working. To actually test the Digital Signature, report FP_PDF_TEST_07 can be used instead, as described in the following KBA:

2814767 - How to use report FP_PDF_TEST_07
