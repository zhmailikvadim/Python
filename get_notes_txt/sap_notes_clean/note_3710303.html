Symptom

In applications that are based on the Floorplan Manager (FPM) and use the GUIBB Form@GL2.0 or the GUIBB Form Repeater, there might be a runtime error ITAB_DUPLICATE_KEY (Exception CX_SY_ITAB_DUPLICATE_KEY) in special layout cases, if the property 'Layout Design Type' is set to the value 'Responsive (CSS Grid)' in the respective GUIBB configuration.

Other Terms

Floorplan Manager, GUIBB, Form@GL2.0, Form, FPM_FORM_UIBB_GL2, Form Repeater, FPM_FORM_REPEATER_UIBB

Reason and Prerequisites

Program error.

Solution

Implement the attached corrections or import the relevant Support Package.
