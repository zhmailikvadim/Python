Symptom

In the following processes, APAREK checks failed in certain scenarios:

- Receipt of Deregistration Request for Gas (SUP View) (DE_C_GELIGAS_DEREG_SUP)
- Receipt of Termination of Supplier Assignment for Gas (SUP View) (DE_C_GELIGAS_TERM_OF_ASSGN_SUP)
- Receipt of Deregistration Request for Electricity (SUP View) (DE_C_GPKE_DEREG_SUP)
- Receipt of Termination of Supplier Assignment for Electricity (SUP View) (DE_C_GPKE_TERM_OF_ASSGN_SUP)

This issue is relevant for the market role: SUP.

Other Terms

 APAREK, POD_NOT_EXIST, Proc date, key date, GPES, GELIGAS, deregistration, termination of supplier assignment

Reason and Prerequisites

The error occurred because of a logic error in the APAREK check implementation.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the APAREK checks behave as expected. You need to retrigger the process. You can verify the fix by checking that the APAREK checks pass as expected.
