Symptom

Is there any way to increase the max size of the webadmin.log and webservices.log files?

Environment

SAP Data Services 4.x

Resolution

Customizing the rollover size is not supported for these files:

  • webadmin.log: fixed size 5 MB; maximum file count 3.
  • WebServices.log: fixed size 5 MB; maximum file count 1.

To retain content longer for troubleshooting, please regularly back up webadmin.log, which contains all content from WebServices.log.
