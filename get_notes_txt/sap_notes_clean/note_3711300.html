Symptom

This KBA explains the expected results of Average Demand Interval (ADI) after running a Forecast Automation Profile. It provides clarity on how ADI values are determined based on the input time series data.

Environment

  • SAP Integrated Business Planning (SCM-IBP)

Cause

The ADI result is influenced by the number of zeros and present in the input time series data. 

Resolution

Please note that in this example the number of required periods is set to 5, which is the default value.

  1. If the input time series contains only nulls, the expected value of ADI is null since the number of required periods is not reached. 
  2. If the input time series contains only zeros, the expected value of ADI is zero.
  3. If the input time series contains only nulls and there is one time period that is non-zero, the expected value of ADI is null because the number of required periods is not reached.
  4. If the input time series contains only zeros and there is one time period that is non-zero, the expected value of ADI is 1.
