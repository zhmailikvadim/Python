Symptom

In Fiori app "Custom Fields and Logic", unable to delete the Custom Field with the error status "Deletion Failed". The messages have "DDIC Mass Activation: Activation failed".

In SAP GUI, transaction SCFD_LOG, click on the error and it contains the messages.

"Object name: FINDS_GLPOS_C. The column ZZ1_ABC_COB is unknown"

"Object name: FINDS_GLPOS_V. The column ZZ1_ABC_COB is unknown"

Environment

  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Access to "Custom Fields and Logic" app
  2. Delete a custom field and receive an error message "DDIC Mass Activation: Activation failed"

Cause

The custom field is still used in the CDS-Views FINS_GLPOS_C / FINS_GLPOS_V.

Resolution

Follow the resolution as specified in 3405871 - Removing the customer-specific fields using FGL_MIG_REGENERATE_CDS_GLPOS.

Keywords

FINS_GLPOS_V, FINS_GLPOS_C, delete custom field, DDIC Mass Activation, Activation failed
