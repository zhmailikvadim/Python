Symptom

  • What should be the minimum length of the characters?
  • What should be the maximum length of the characters?

Environment

  • SAP BusinessObjects Business Intelligence Platform 4.3/2025
  • Windows
  • Linux / Unix

Reproducing the Issue

  1. Log in to the CMC.
  2. Navigate to Users and Groups, then click on User List.
  3. Select the required user and open the Properties page.
  4. In the password fields, try entering a password longer than the supported maximum limit.
  5. The system displays the error: "The new password is too long. Passwords cannot be longer than N characters."
  6. Also try entering a password shorter than the minimum allowed length.
  7. The system displays the error: "Your password must be at least 6 characters long."

    Cause

    • The Business Intelligence Platform 4.x does not accept the password if it is set beyond the given limit.

    Resolution

    • Minimum supported password length: 6 characters.
    • Maximum supported password length: 255 characters.
    • Attempting to exceed 255 is not supported.
