Symptom

Customer requaested to include information related to Risk Owners to the informations created in time of execution for the Rules Download and Upload programs.

All rule set in text Formats

Other Terms

All rule set in text Formats, GRC-SAC-ARA, GRC Rules Upload/Download, SAP GRC AC 10.1, GRAC_DOWNLOAD_RULES, GRAC_UPLOAD_RULES

Reason and Prerequisites

Customer requaested to include information related to Risk Owners to the informations created in time of execution for the Rules Download and Upload programs.

All rule set in text Formats

Solution

Deploy this note to update GRAC_DOWNLOAD_RULES and GRAC_UPLOAD_RULES Programs with feature enabling use of Risk Owners information in time of Rules Download and Upload.
