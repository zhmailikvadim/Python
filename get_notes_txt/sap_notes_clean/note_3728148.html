Symptom

  • IDOCs are failing with error message no M3 152 "Company code XXX is not set up for Materials Management".
  • Update records terminated with error message "Company code XXX is not set up for Materials Management".
  • Function Module: MB_UPDATE_TASKS status: Update was terminated.

Environment

  • SAP ERP Logistics (LO)
  • SAP R/3
  • SAP R/3 Enterprise
  • SAP ERP Central Component
  • SAP ERP
  • SAP Enhancement package for SAP ERP
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Execute transaction CKM3.
  2. Observe the error message "Company code XXX is not set up for Materials Management" appearing immediately without selecting anything on the screen.
  3. For batch users, check IDOC postings in transaction WE02. Error log will show the error message no. M3152.

Note: The error message can occur with different interfaces/reproducible steps.

Cause

OMSY configurations were missing for the relevant company codes in the affected server instance or they are inconsistently maintained.

Resolution

  1. Verify OMSY configurations for the affected company codes in the system.
  2. Maintain relevant fiscal years and periods(such as initial period) for material management data for the company codes in transaction OMSY.
  3. In case OMSY configurational data is missing from specific servers, execute the command "/$TAB" to refresh the server buffer and synchronize the OMSY data.
