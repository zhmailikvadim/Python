Symptom

When open items are derived using a manually entered payment advice, or when they are derived using processing rules using payment advice, the discount amount may be used in the wrong currency (local currency). A bank statement item is then not possible to complete automatically due to a non-zero balance.

Other Terms

F1520, SKNTO, WSKTO

Reason and Prerequisites

Using a payment advice with discounts in a currency different from the local currency.

Solution

This SAP Note changes the used field with the discount amount. It uses the amount in the transaction currency instead of the amount in the local currency.

As a general rule, SAP recommends that you install a solution by applying a Support Package. However, if you need to install a solution earlier, use the Note Assistant to implement the correction instruction. You can find more information about the Note Assistant in SAP Service Marketplace, under service.sap.com/note-assistant.
