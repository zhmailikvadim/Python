Symptom

Step "Activate BW Content for RCA" fails, and checking the job log of CCMS_BI_SETUP the following error occurs:

ERROR: Source system <SID>CLNT<NNN> is not active for this BW

Environment

SAP Solution Manager 7.2

Reproducing the Issue

  1. t-code SOLMAN_SETUP
  2. Mandatory Configuration
  3. Basic Configuration
  4. Configure Basic Functions
  5. Activate BW Content for RCA

Cause

BW Source System isn't activated. There should be many other errors as well, such as error when activating Datastore objects, or error activating element.
However, the above error about Source System has to be resolved firstly as the important prerequisite.

Resolution

  1. Follow the steps below to activate the BW source system:
    t-code RSA1
    ->Source System
    ->BW
    -><SID>CLNT<NNN>
    ->right click on the line and select Activate
    ->In the Replicate Metadata? dialog, select Replicate as Well.
    ->In the following screens, confirm the default settings displayed on the screen.
  2. Double confirm the activation successful in t-code RSTCO_ADMIN. If the status is red, please click on the activate button and wait until the background job finishes. 

  3. During the Source System activation in RSA1, if an error like "The RFC connection for destination <SID>CLNT<NNN> 
  4. Go to SM59 -> ABAP Connections -> Double Click <SID>CLNT<NNN>
  5. Select Connection Test -> if Successful
  6. Go to Utilities -> Test -> Authorization Test -> if this fails with Error -> Authorization for Trusted RFC between SAP Solution Manager and BW - System
  7. Check: Application-Specific Security Guide - Second Level User Role
  8. Re-run step "Actiavte BW Content for RCA".
