Symptom

For the standard form template SD_SLS_ORDER_CONFIRM for a certain language (e.g., French), the value of the "Delivery Date" field is missing. Besides, the "Payment Terms" field is missing.

Environment

SAP S/4HANA Cloud Public Edition

Resolution

  1. Regarding the missing of value for the "Delivery Date" field, the solution is to maintain the context for the related language (e.g., French) via the configuration activity "Maintain Display Type for Date/Period (ID: 106369)".
  2. Regarding the missing of the "Payment Terms" field, the solution is to maintain it for the related language following the below steps:
    (1) Log onto the system with the related language (e.g., French);
    (2) Open the configuration activity "Maintain Terms of Payment (ID: 102934)" and locate to the related payment terms;
    (3) Maintain the "Sales Text" field for this payment terms.
