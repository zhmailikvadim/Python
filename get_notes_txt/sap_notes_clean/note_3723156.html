Symptom

You switch from the component tab in an EAM order to another tab. After switching back to the component tab, the scroll position on the component tab is not retained. The system scrolls to the first component. This is SAP standard behavior.

Other Terms

IW31, IW32, IW33, COMK 0020, COMK0020, COMK, CO_MK_PROCEED_MAT_PROV, KOMPONENT_LIST, RC27X-INDEX_AUF, INDEX_AUF, RC27X, TCTRL_3020, SCROLL_TAB.

EAM-Auftrag, PM-Auftrag, IH-Auftrag, Instandhaltungsauftrag, CS-Aufrag, SM-Auftrag, Serviceauftrag.

Reason and Prerequisites

Ursache:
Missing functionality.

Solution

Implement the correction instructions.

Implementing this SAP Note does not change the current system behavior. This means that the system continues to scroll to the first component. SAP Note 3725851 describes how you can adjust the system behavior so that the scroll position is retained on the component tab page.
