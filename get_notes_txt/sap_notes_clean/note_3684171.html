Symptom

  • Users are unable to locate the advanced search option for filtering pipelines in the calculation workspace.
  • This causes difficulty in filtering pipeline runs effectively. 

Environment

  • SAP SuccessFactors Sales Performance Management
    • Incentive Management
    • Incentive Management on Hyperscalers

Reproducing the Issue

  1. Navigate to the Calculation Workspace.
  2. Attempt to locate the advanced search option for filtering pipeline runs.
  3. Observe that the option is not visible or accessible. 

Cause

UI overhaul - the advanced filters are now accessible under configure summary view for the Calculation workspace.

Resolution

  1. Navigate to the Calculation Runs workspace.
  2. Select the History section.
  3. Configure the Summary View by choosing the Filter option.
  4. Select all required fields for filtering.
  5. View filters displayed at the top of the run list.
  6. Save filters by clicking on the field, setting conditions, and selecting the save option.
  7. Use the "Manage" option under "My Views" to save filters for future use.
  8. For further details on advanced search, refer to the help document linked in the See also section.
