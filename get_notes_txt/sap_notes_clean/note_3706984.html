Symptom

You are using the Egypt VAT Return statutory report in the Run Statutory Reports app to generate your VAT return Form 10 (نموذج 10 ضريبة مضافة ). The report data analytics did not include supplier invoices.

Other Terms

VAT (Value Added Tax), EG_VAT_DCL, statutory reporting, SAP Document and Reporting Compliance, report data analytics

Reason and Prerequisites

The report data analytics for Egypt VAT Return report did not include supplier invoice data.

Solution

After implementing this SAP Note, the system includes supplier invoice data in the Egypt VAT Return report data analytics. You can now analyze supplier invoices alongside other VAT-related information using the Analyze Data action in the Run Statutory Reports app.

For more information on the Egypt VAT Return report, see the report documentation on the SAP Help Portal: Egypt VAT Return
