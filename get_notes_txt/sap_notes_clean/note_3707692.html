Symptom

You open tab test plan - test case selection and a dump appears in ABAP program CL_SMT_ACT_TPLN_RT_QRY_TC_TREE". The error is due to duplicated test plan nodes.

Other Terms

There is already a line with the same key, ITAB_DUPLICATE_KEY, Copy To Other Test Plan, CL_SMT_ACT_TPLN_RT_QRY_TC_TREE

Solution

The present note provides a report SMT_COR_TPLN_NODE_DUPLICATE_1 which helps to correct the data inconsistency (to be run in transaction SE38). Apply the attached source code corrections or the corresponding support package.

To prevent the issue to happen again, implement SAP Note 3706608 "Duplicate Test Plan Nodes After Copy to Other Test Plan" or implement the corresponding support package.
