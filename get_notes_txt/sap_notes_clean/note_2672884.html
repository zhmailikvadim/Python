Symptom

SAP Note 2667519 needs changes to text elements, which cannot be implemented by transaction SNOTE. Instead, run the herewith provided report NOTE_2667519 via transaction SE38 after implementing the master note 2667519.

Other Terms

NOTE_2667519, Text Elements Report RSR_CACHE_RSRV_CHECK_ENTRIES

Reason and Prerequisites

Changes for text elements are necessesary for report RSR_CACHE_RSRV_CHECK_ENTRIES.

Solution

  • SAP Business Warehouse (SAP BW) 7.30
    Implement Support Package 19 for SAP BW 7.30 (SAPKW73019) into your BW system. The Support Package will be available as soon as SAP Note 2554973 with the short text "SAPBWNews 7.30 BW ABAP SP19", which describes this Support Package in more detail, is released for customers.
     
  • SAP BW 7.31 (SAP NW BW 7.3 EHP 1)
    Implement Support Package 24 for SAP BW 7.31 (SAPKW73124) into your BW system. The Support Package will be available as soon as SAP Note 2653359 with the short text "SAPBWNews 7.31 BW ABAP SP24", which describes this Support Package in more detail, is released for customers.
     

 

In urgent cases you can use the correction instructions.

Before you use the correction instructions, make sure that you check SAP Note 1668882 and SAP Note 2248091 for transaction SNOTE.

This SAP Note might already be available before the Support Package is released. In this case, however, the short text still contains the term "preliminary version".
