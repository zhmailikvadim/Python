Symptom

  • Error message: “Changes failed. You can’t save the template because it contains one or more field IDs reserved for internal use. Please update the XML template and upload it again.”
  • An error occurs when attempting to update the Start Date directly in the XML file and uploading the XML via Provisioning.
  • Error message: “The system encountered error(s) when importing the file. Error: You cannot upload the template because it contains a field ID reserved for internal usage: status.”

Environment

SAP SuccessFactors Career And Talent Development.

Reproducing the Issue

  1. Attempt to change the "Start Date" in "Manage Templates" and click "Save". Observe the error message: “Changes failed. You can’t save the template because it contains one or more field IDs reserved for           internal use. Please update the XML template and upload it again"
  2. Update the "Start Date" directly in the XML file and attempt to upload the XML via "Provisioning". Observe the error message:“The system encountered error(s) when importing the file. Error: You cannot upload the template because it contains a field ID reserved for internal usage: status.”

Cause

The field ID “status” was configured incorrectly in the Development Goal Plan XML template.

Resolution

Update the field ID from “status” to “state” in the development goal plan XML .

Note: Create a new Development Goal Plan template instead of modifying the existing template dates to avoid potential data inconsistencies.
