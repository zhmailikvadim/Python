Symptom

  • Not able to create new contact user 
  • Cannot create new contact to assign new finance role
  • Create button is not clickable
  • New contact user cannot be created because the Create button is grey-out to click

Environment

  • SAP for Me > Finance & Legal
  • SAP Finance Contacts

Reproducing the Issue

  1. Login to SAP for Me 
  2. Access Finance & Legal
  3. Try to create new contact user in SAP Finance Contacts card
  4. After entering all the required fields, the Create button is not clickable

Cause

The user with the entered email address has been maintained as contact user with some finance role(s) already.

Resolution

Customer now has the option to create new contact and assign finance role(s) to this new contact using 'Create Contact' in SAP Finance Contacts card.

When creating new contact using 'Create Contact', there will be a duplicate check performed against the email, in order to prevent creating a duplicate role.

If there is a duplicate, the 'Create Contact' button will not be clickable and be grey-out. 

Please search the existing contact with the email address and use the existing contact for finance role assignment.
