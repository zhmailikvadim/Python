Symptom

Error "The job profile is currently in workflow." received when trying to import Role-Competency mapping file.

Environment

SAP SuccessFactors Job Profile Builder

Reproducing the Issue

  1. Create a Role-Competency mapping file.
  2. Import the file through the Import and Export Data Tool or from Manage Job Profile Content Import/Export.
  3. Check the MDF Data Import job from Scheduled Job Manager.
  4. Notice that the job fails with the error "The job profile is currently in workflow."

Cause

  • There are pending Job Profile workflows in "Draft" or "Pending" status, this can be verified in Manage Job Profile > In-Workflow Job Profile. 
  • The roles associated with these in-progress workflows are included in the Role-Competency mappings being imported, which leads to the issue.

Resolution

To resolve the issue, either:

  • Complete the pending Job Profile workflows so they are no longer in "Draft" or "Pending" status,

OR

  • Remove from the import file any roles that are linked to Job Profiles with change requests currently in workflow.
