Symptom

You use the ABC template to add costs in material or order costing. In the costing result (itemization) you can see that costs of internal activities from ABC template are missing.

Other Terms

ABC template, process costs, plan cost calculation

Reason and Prerequisites

Due to a program error the ABC template might not add an internal activity if you have maintained a CO version for internal activity prices (in the valuation variant) which is not a plan version.

Solution

Implement the attached correction.
