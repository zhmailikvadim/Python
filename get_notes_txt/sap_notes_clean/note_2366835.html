Symptom

This technical note creates the NF-e Number index and Document Number index in J_1BNFE_ACTIVE table specific for SAP HANA database.

  • The implementation of this SAP Note does not have an impact on your existing functionality. 
  • This SAP Note delivers changes that are usually treated as manual activities within SNOTE (for example, packages, ABAP dictionary objects, message classes).

Other Terms

NF-e, Nota fiscal, Brazil, J_1BNFE_ACTIVE, NFNUM9, DOCNUM9.

Reason and Prerequisites

  • This SAP Note is only valid for Brazil. 
  • This SAP Note is only valid for Hana Database.

Solution

  • Implement the manual steps described in the manual activities for your release.
  • Be aware that this SAP Note does not solve any issue related to businesses scenario.
