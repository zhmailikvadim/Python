Symptom

The system allows you to map inbound fields from a lower-level entity to a higher-level entity, such as from the operation level to the header level, for POL as PEO routing. This mapping should not be possible.

Other Terms

PLMSI, PLMSIFND, PLM System Integration, POL, Production Operation List, PEO, PEO Routing, POL as PEO Routing, Inbound Field Mapping, Extension Fields

Solution

Implement this SAP Note to correct the issue.
