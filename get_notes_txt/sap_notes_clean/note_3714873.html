Symptom

You run the FPG5 Itemization for G/L documents from FI-CA for a reconciliation key where the totals and/or FI-CA documents have been archived. The system does not display any records on the screen.

Other Terms

FPG5, FI-CA, G/L documents, itemization, reconciliation, archive

Reason and Prerequisites

Prerequisites:

In the system, the FI-CA documents and total records have been archived, but the G/L documents remain in the database.

Solution

After implementing this SAP Note, the system displays the relevant records on the screen when you run the FPG5 Itemization for G/L documents from FI-CA, even if the reconciliation key or FI-CA documents have been archived.

To resolve this issue, implement this SAP Note in accordance with the attached correction instructions or import the relevant Support Package.
