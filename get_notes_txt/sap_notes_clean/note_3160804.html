Resolution


Diese Fehlermeldung wird erzeugt, wenn die beiden Systemen nicht mit dem neuen Parameter "jwtsecret" verbunden sind. Bitte führen Sie dementsprechend folgende Schritte durch:

  1. Stoppen Sie beide Tomcats
  2. Öffnen Sie die effektif-onpremise.properties (Tomcat-Workflow\webapps\ROOT\WEB-INF\classes)

  3. Fügen folgenden Parameter hinzu und speichern anschließend die Datei:

    RDark

    (Bitte ersetzen Sie den Parameter "SHARED_SECRET_CHANGE_ME" durch ein von Ihnen gewählten Wert)

  4. Öffnen Sie die configuration.xml (Tomcat Process Manager\conf)

  5. Fügen folgenden Parameter hinzu und speichern anschließend die Datei:

    RDarkSHARED_SECRET_CHANGE_ME]]>

    (Bitte ersetzen Sie den Parameter "SHARED_SECRET_CHANGE_ME" durch den von Ihnen gewählten Wert, der im Workflow-System hinterlegt ist)

  6. Starten Sie beide Tomcats

Weitere Informationen finden Sie in unserem Administratorenhandbuch:

  • Workflow: https://docs.signavio.com/adminguide/workflow/de/update-notes.html#updating-the-configuration-file

  • Process Manager: https://docs.signavio.com/adminguide/editor/de/installation/install_solman.html#configuring-the-connector (nur der Parameter "sharedJwtSecret" ist wichtig)
