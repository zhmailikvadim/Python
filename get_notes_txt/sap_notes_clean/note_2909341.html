Symptom

Error below happens when opening excel file which is exported from story in SAC.

We found a problem with some content in 'XXX.xlsx". Do you want us to try to recover as much as we can? If you trust the source of this workbook, click Yes.

Environment

  • SAP Analytics Cloud (SAC)

Reproducing the Issue

  1. Create model based on Excel file.
  2. Create story based on the model.
  3. Select Scale as Billion and make the value in story displayed as +0.
  4. Export the story to XLSX Excel file, check Include Formatting option.
  5. Open the exported excel file, error below happens.

    We found a problem with some content in 'XXX.xlsx". Do you want us to try to recover as much as we can? If you trust the source of this workbook, click Yes.

Cause

This issue is caused by product bug.

Resolution

This issue will be fixed in 2020.08 version.
