Symptom

  • There are 2 HANA Cockpit with the same System ID (SID).
  • SSO is enabled for a resource for one Cockpit. When enabling SSO for the second cockpit, the SSO for the first cockpit breaks and does not work anymore.

Environment

  • SAP HANA 2.0 SPS03 or lower
  • SAP HANA XS Advanced 1.0.129 or lower
  • SAP HANA Cockpit 2.0

Cause

The current HANA version does not allow multiple JWT providers in parallel.

Resolution

Update HANA to SPS04 or higher and XSA to 1.0.130 or higher.

Keywords

xsa, sso, establish, HANA, Cockpit, SID, unable to, JWT, provider
