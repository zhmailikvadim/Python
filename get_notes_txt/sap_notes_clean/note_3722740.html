Symptom

  • After exporting HTML from Web Intelligence (webi) and opening individual report HTML files in Microsoft Edge, a "Translate page from French?" pop-up message appears.
  • This does not happen when using Google Chrome or Mozilla Firefox.

Other Terms

Keywords: translation pop-up, translate page from, edge browser, automatic translation

Reason and Prerequisites

Environment

  • SAP BusinessObjects Business Intelligence Platform 2025
  • Browser: Microsoft Edge

Reproducing the Issue

1. Create a Web Intelligence Document.
2. Schedule or download on-demand HTML (zip file).
3. Extract the zip file.
4. Navigate to the numbered subfolder and open report.html using Microsoft Edge.
5. Observe the translation pop-up message.

Cause

The cause is currently under investigation by Development.

Solution

The issue is currently under investigation by Development.

Workaround is to disable Automatic Translation

  1. Go to Settings of Edge browser
  2. Languages
  3. In "Translation" section disable "Offer to translate pages that aren't in a language I read".
