Symptom

Duplicate EMAIL_OPENED interaction may occur if the Link Tracking job, which runs every minute, creates the interaction before the API cl_cuan_ce_ia_post( )->import_interactions executes its operation.

Other Terms

SAP Marketing 2024, CUAN_MKT_EXEC_LINK_TRACKING, CUAN_MKT_EXEC_LISTENER.

Reason and Prerequisites

Side effect of SAP note 3705713.

Solution

This note will reduce the execution frequency of Link Tracking job to 5 minutes by default. This value is customizable by executing the program CUAN_MKT_EXEC_CTRLP_EDITOR and inserting the parameter ID named  LINK_TRACKING_MINUTE_FREQUENCY and the value of your choice. However we do not recommend to change that default value.

Apply the correction instruction.
