Symptom

This note prepares for the implementation of new operation versions for business object DepositAccountContract.

Before implementing note 2306760 you must have implemented this note. This note delivers a report that creates the necessary objects (such as DDIC, etc.) which cannot be otherwise automatically created via SNOTE.  It's execution is described in the manual implementation step of note 2306760!

These objects are necessary for the following new operations:

Service Interface BankAccountContractProcessingDepositAccountContractIn_V1:
  UpdateDepositAccountContractAsBulk
  CreateDepositAccountContractAsBulk

 Service Interface BankAccountContractProcessingDepositAccountContractOut_V1:
  ConfirmDepositAccountContractUpdateAsBulk
  ConfirmDepositAccountContractCreationAsBulk

Service Interface BankAccountContractProcessingManageDepositAccountContractIn_V1:
  CheckDepositAccountContractUpdate
  UpdateDepositAccountContract
  CreateDepositAccountContract
  CheckDepositAccountContractCreation

Solution

Import the relevant Support Package for your release.
