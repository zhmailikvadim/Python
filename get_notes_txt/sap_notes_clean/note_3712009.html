Symptom

When executing Task 2141 (Calculate Group Shares) in the Consolidation Monitor, the system raises the following error:

"The cumulated direct share of investor <Investor Unit ID> in the investee unit <Investee Unit ID> is 200.00000000 percent. However, this value must lie between zero and one hundred percent."

However, in Group Data Analysis, the ownership percentage appears to be correctly maintained as 100%, and no ownership quantities are posted in the current posting period.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Open the Consolidation Monitor app.
  2. Fill in the required parameters.
  3. Locate the Consolidation Group and execute the Calculate Group Shares task.
  4. Observe the error message: 

"The cumulated direct share of investor <Investor Unit> in the investee unit <Investee Unit> is 200.00000000 percent. However, this value must lie between zero and one hundred percent.

Message no. G00619

Diagnosis

The year-to-date value of the direct share of the investor <Investor Unit ID> in the investee unit <Investee Unit ID> is 200.00000000 percent. However, this value must lie between zero and one hundred percent.Procedure

Correct the additional financial data for investment."

Cause

Task 2141 calculates group shares based on cumulative consolidation quantities (CnsldtnQuantityInBaseUnit) for the entire fiscal year, not only on the quantities posted in the current period.

Ownership-related consolidation quantities may exist in multiple posting periods, including:

  • Posting Period 000, created by Balance Carryforward, which initializes the ownership position at the beginning of the fiscal year
  • Subsequent posting periods, where ownership quantities may be posted again due to ownership determination, data load, or manual postings

If ownership quantities are posted in more than one period without corresponding reversals, the cumulative ownership percentage may exceed 100%, which triggers the validation error.

Resolution

Ensure that ownership-related consolidation quantities result in a net cumulative ownership between 0% and 100% for the fiscal year.

Verification steps:

  1. Open the Display Group Journal Entries app
  2. Filter by:
    Consolidation COA
    Version
    Fiscal Year
    Range of Posting Period
    Consolidation Unit
    Investee Unit
  3. Check the following fields:
    Posting Period
    FS Item
    CnsldtnQuantityInBaseUnit
  4. Review ownership-related entries across all posting periods, in particular posting period 000 and early fiscal periods.

Additional Information:

If ownership quantities were carried forward and then re-posted, ensure that the carried-forward quantities are reversed

Avoid duplicate ownership postings across posting periods

After correction, rerun Task 2141 (Calculate Group Shares)

Group share calculation is based on consolidation quantities, not only on ownership master data

Posting period 000 is a valid posting period and is included in consolidation processing

Group Data Analysis may show correct values per period, while Task 2141 validates cumulative values
