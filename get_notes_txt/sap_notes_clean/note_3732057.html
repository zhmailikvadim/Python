Symptom

This note fix following issues.

1. Webservice for 'MeterProfileCreate' on SMGW should be completed before CTB install on CR.

2. Waiting for iMS Parameterization Process completed is not mandatory for CTB install on Melo, in case of Parameterization Instance status is Installed or Productive or no webservice needed, there is no need to create deadline waiting for iMS Parameterization Process.

3. Adjust the precondition of step 0409 in process DE_B_IM_DEV_CHANGE since step 0409 should not run before 0385 completed in case of 0385 exists.

Other Terms

DE_B_IM_DEV_CHANGE, DE_B_IDEV_CTB_CHANGE, Control box, Control Resource, /US4G/CL_CHK_IM_PROCESS,  CHK_CTB_LEADING_POD,  CHK_IF_CTB_INST_START, CHK_IF_CTB_IN_TRIGGER, CHECK_IF_CTB_CHG_CONTINUE

Reason and Prerequisites

Program and customizing error.

Solution

Please implement the correction instruction in the SAP Note or import the higher version of Support Package.

The main changes in process are the following:

  • Add step 0500 and 0508 in process DE_B_IM_DEV_CHANGE for checking If CTB Install process for CR need to wait MeterProfile WS on SMGW
  • Add step 0700 in process DE_B_IDEV_CTB_CHANGErelay on the step 0508 above in case of CTB install on CR
  • Adjust logic for step 0500 in DE_B_IDEV_CTB_CHANGE considering the webservice is needed or not.
