Symptom

 In the process "Receipt of Meter Readings as Part of Device Change or DSO Change Process" (DE_B_MET_MR_INFO_DVCHG), when the system receives meter readings for device replacement, it attempts to find a suitable meter reading order in step 0100 and a relevant device change process document in step 0120. If the meter reading order is found in step 0100 but the device change process document is not found in step 0120, the system raises a BPEM case in step 0120. Subsequently, the system displays the error message "Point of delivery is missing while preparing the MR values" in step 0300-0050.

This issue is relevant for the market role: DSO.

Other Terms

 MET, DE_B_MET_MR_INFO_DVCHG, meter reading order, device change process document, device replacement, point of delivery, BPEM case

Reason and Prerequisites

 The error occurred because of a process configuration issue.

Solution

 Implement the corrections of this SAP Note or import the relevant support package.
