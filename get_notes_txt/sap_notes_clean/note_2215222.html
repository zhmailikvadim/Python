Environment

  • SAP enhancement package 1 for SAP NetWeaver 7.3

Cause

This issue occurs if the code correction from the SAP Note 1790818 has been applied in the receiver system but the manual corrections have not

Resolution

  1. Implement the manual corrections detailed within the SAP Note 1790818 alongside the code corrections

Keywords
