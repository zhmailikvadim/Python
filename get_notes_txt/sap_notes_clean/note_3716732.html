Symptom

The function "read PP master data" is not available for mass processing in transaction code COHV.

Environment

  • SAP ERP Production Planning (PP)
  • SAP ERP All versions
  • SAP ERP Central Component All versions
  • SAP R/3 All versions
  • SAP R/3 Enterprise All versions
  • SAP S/4HANA All versions
  • SAP S/4HANA Cloud Private Edition All versions
  • SAP enhancement package for SAP ERP All versions
  • SAP enhancement package for SAP ERP, version for SAP HANA All versions

Cause

This is a standard system design.

Resolution

The "read PP master data" function is not supported by T-Code COHV. As an alternative, the following method can be used:

  1. Utilize the BAPI BAPI_PRODORD_CHANGE for reading the master data.
  2. Use the relevant fields in the parameter structures: EXPLOSION_DATE and EXPLODE_NEW.
