Symptom

  • sap process insights not functioning after a background user was changed.
  • Direct connection to authentication endpoint failed with error: NIECONN_REFUSED(-10).
  • OTR-Based Exception of Class: /SDF/CX_CALM_CONNECTOR.

Reproducing the Issue

  1. Change the background user and register a new user with all necessary roles.
  2. Delete the HTTP connection and maintain a new connection.
  3. Attempt to connect using t-code SDF/ALM_SETUP.
  4. Observe the error: Direct connect to authentication endpoint failed with NIECONN_REFUSED(-10).

Cause

Resolution

Please be advised to use /SDF/PINS_SETUP rather than /SDF/ALM_SETUP, which is outdated.

Also, to connect SAP On-Premise systems to Signavio, the prerequisites stated in Direct System Connection for SAP On-Premise Systems | SAP Help Portal need to be met, to successfully install Process Insights.
