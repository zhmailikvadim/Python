Symptom

Upport of Note 3376355. Material Type value "ERSA" is set as the default filter on adding an unplanned component for Material Value Help Type "All Materials".

Other Terms

 F5104A, ERSA, Unplanned Component, Perform Maintenance Jobs

Reason and Prerequisites

Not Applicable

Solution

Implement this note to remove 'ERSA' as the default material type.

If you need to set a default material type, please see note 3376387.
