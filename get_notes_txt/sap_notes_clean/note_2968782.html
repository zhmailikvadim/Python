Symptom

How to disable users who have not logged on to Enterprise in since a specific date using JAVA SDK ?

Environment

  • SAP BusinessObjects Business Intelligence platform 4.x

Resolution

Use the attached jsp code, it disables the users who haven't logged on to Business Objects since the date specified in the date variable "d1", except Administrator and Guest.

To run the code, refer to the steps in KBA 2932452 - Steps to Create / Deploy Web Application in Tomcat for BI4.x Java SDK ?
