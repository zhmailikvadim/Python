Symptom

  • Is dumping and loading a database with named pipes supported?
  • Named pipes are mentioned as compatible for a backup device in the "Reference Manual: Commands" under "Restrictions for dump database"
  • This information is not present in ASE 15.7 documents

Environment

  • Adaptive Server Enterprise 15.7
  • Adaptive Server Enterprise 16.0

Resolution

  • Named pipes with database dump and loads are supported
  • Was first introduced in ASE 15.0.2
  • Was first documented much later in time so is not present in older documentation

Keywords

named pipes, named pipe
