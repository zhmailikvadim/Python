Symptom

When you intend to create Agreement Request via Master Request using transaction /IRM/IPPQM, the system should allow you to do so successfully. However, while doing so if you face Error Message or Runtime Error; this note will fix that issue.

Other Terms

Master Request, Agreement Request, Error Message "Price record with identical attributes exist", SLG1, /IRM/IPPQM, /IRM/IPBBARM, /IRM/IPCBARM, /IRM/IPCRARM, /IRM/IPSIARM, /IRM/IPPRARM, /IRM/GPRM002

Reason and Prerequisites

This is caused by program error.

Solution

Please implement the correction instructions attached. These corrections are only valid if you have applied the Support Packs:

Release  60E - Support pack 2

Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object.
