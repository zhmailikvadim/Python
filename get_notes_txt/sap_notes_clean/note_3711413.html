Symptom

1. In the RPCZPLT0 report, statutory employees who were not insured in the processed period, but were insured in a month in the past to which a retro calculation was performed, were reported as insured. Statutory employees without retroactive calculations were reported correctly.

2. When creating the electronic file for health insurance companies, a single file was created for all processed health insurance companies and not a file for each processed health insurance company.

3. When creating a form based on organizational assignment (i.e., the selection screen parameter "'X' Form.dle VS payer/Org.Př." = ' ' ), a form with zero values ​​was created. When creating a form based on the variable payer symbol (i.e., the selection screen parameter "'X' Form.dle VS payer/Org.Př." = 'X' ), the form was created correctly.

Other Terms

RPCZPLT0, number of insured employees.

Reason and Prerequisites

  • This SAP Note is relevant only for the Czech Republic

Solution

This SAP Note contains corrections as described in Symptom section.

Important: Apply current version of SAP Note 875986 for NW 7.0x or 1668882 for NW 7.3x and higher. Otherwise an old version of SNOTE can lead to missing coding changes when applying SAP Notes.

  • As a general rule, SAP recommends that you install a solution by applying a Support Package. However, if you need to install a solution earlier, use the SAP Note Assistant to implement the correction instruction. You can find more information about the SAP Note Assistant in SAP Service Marketplace, under SAP Note Assistant.
  • Note that documentation, transactions, and Customizing activities are only delivered via Support Package.
