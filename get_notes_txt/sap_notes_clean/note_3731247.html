Symptom

You have recently upgraded to SAP Customer Activity Repository applications bundle 6.0 SPS01 and you are executing the UDF F&R-API: Function Module /DMF/UFC_RETRIEVE_RESULTS_FR. You experience that the UDF - F&R-API takes a long time for execution.

Other Terms

Forecasting and Replenishment, Unified Demand Forecast, UDF, Performance,  SAP CARAB 6.0 SPS01.

Reason and Prerequisites

Reason: Code is not optimized of the methods - DETERMINE_FC_SOURCE and FETCH_DATA of class /DMF/CL_UFC_GET_RESULTS_FR.

Solution

Implement the attached correction instructions in your back-end system.
