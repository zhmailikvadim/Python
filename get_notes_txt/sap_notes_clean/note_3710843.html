Symptom

You would like to use the transaction CKMM for articles.

Other Terms

CKMM, price determination control, article master, MBEW-STPRS, MBEW-VERPR, MBEW-VPRSV, T130F-KZCPY, T130F_C-KZCPY

Reason and Prerequisites

Actual costing is active in a retail scenario.

Solution

Transaction CKMM does not support the copy handling of articles. Thus, transaction CKMM can only be used if the copy handling for the standard price (MBEW-STPRS), the periodic unit price/moving average price (MBEW-VERPR) and the price control indicator (MBEW-VPRSV) is deactivated. This means that the field KZCPY in tables T130F respectively T130F_C must not be marked for those 3 entries.
