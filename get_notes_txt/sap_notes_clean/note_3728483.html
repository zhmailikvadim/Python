Symptom

I would like to request access or authorization to edit and manage custom fields within SAP Ariba, specifically for the Sourcing and Contracts modules. When I try to access the configuration area, the options to edit or manage these fields are not available.

Environment

SAP Ariba Sourcing
SAP Ariba Contracts

Cause

The feature PLICM-4976 (Selectively Migrate Field Configurations) is not enabled in your realm. This feature is required to manage custom field configurations using the Customization Deployment and Enablement Tool (CDET).

Resolution

To gain the ability to manage custom fields, you must request that SAP Support enable a specific feature for your site.

  1. Create a Service Request with SAP Support.
  2. In the problem description, state that you need to enable the feature PLICM-4976 (Selectively Migrate Field Configurations).
  3. Specify that this is required to manage custom fields for the Sourcing and Contracts modules.
  4. Once SAP Support confirms the feature is enabled, you will have the necessary authorization to manage custom fields via the Customization Deployment and Enablement Tool (CDET).
