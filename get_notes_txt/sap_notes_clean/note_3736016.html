Symptom

Project header field customization such as hiding out of the box fields in project create view

Environment

SPM Projects

Reproducing the Issue

Resolution

Certain out of the box project header fields such as Copy from Project and Predecessor Project are locked by design and cannot be customized. Therefore, it is not possible to hide these fields in the project creation view.
