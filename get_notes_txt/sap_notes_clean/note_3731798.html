Symptom

While launching the UI Theme Designer, it opens in Google Chrome instead of the configured default browser in Windows

Environment

UI Theme Designer

Reproducing the Issue

  1. Launch the UI theme designer via /UI5/THEME_DESIGNER.
  2. Observe that the tool opens in Google Chrome instead of the Windows default browser.

Resolution

Implement the SAP Note 3580158 - Use default browser when starting UI theme designer to resolve the issue

Keywords

ui theme designer, /UI5/THEME_DESIGNER, default browser, opens in chrome, windows default, Google Browser
