Symptom

When processing Sales Quote, the "Submit for Approval" option grayed out.

Environment

  • SAP Sales Cloud Version 2
  • SAP Sales and Service Cloud Version 2

Reproducing the Issue

  1. Open the Navigation menu.
  2. Open Sales Quotes view.
  3. Open a Sales Quote.
  4. The option "Submit for Approval" is grayed out.
  5. Click on "Check Consistency" option.
  6. See error message: "Order block [Order Block Reason] is set for party ID [Business Partner Number] with role Account in master data."

Cause

The system allows changes to the Sales Quote even when the Account has an Order Block but does not allow Submission for Approval.

Resolution

If an account has an Order Block in place, the system marks any associated Sales Quote as inconsistent. As a result, submitting the Sales Quote for Approval is not permitted - the "Submit for Approval" option will be grayed out.

Approval workflows require the Sales Quote to be in a consistent state.
