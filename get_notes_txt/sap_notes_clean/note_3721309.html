Symptom

While user is attempting to integrate an SAP S/4HANA Private Cloud system provisioned through the SAP Cloud Appliance Library (CAL) with SAP Joule, the S/4HANA system does not appear under the SAP Business Technology Platform (BTP) Cockpit, preventing successful integration.

Environment

  • SAP BTP, Cloud Foundry runtime and environment
  • Joule

Resolution

  • If the SAP S/4HANA system belongs to one of the below two supported categories, it is eligible for Joule integration.
    SAP S/4HANA Cloud, Private Edition
    SAP S/4HANA Cloud, Public Edition

    Systems provisioned through the Cloud Application Library (CAL), however, are not supported for Joule integration.

  • For the complete list of SAP products integrated with Joule, please refer to the documentation: Integrating Your SAP Solution.
