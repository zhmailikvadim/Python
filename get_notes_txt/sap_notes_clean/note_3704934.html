Symptom

Symptom

In Timesheet Analytics - My Activities and Timesheet Analytics - Activities (S/4 HANA) applications, fields of Consolidated Time Recording (CTR) like Time Type, Allowance Type, Absence Type were not available for analysis and display.

Other Terms

F7189, F6687, Timesheet Analytics, Activities, My Activities, CTR, Consolidated Time Recording.

Reason and Prerequisites

Relevant fields were not exposed from the respective CDS Entities.

Solution

 Apply corresponding support pack. Alternatively, implement the correction instructions attached to the SAP Note.

Other Terms

S/4HANA Defense & Security, D&S, Logistics, PLVAR, Plan Version

Solution

Objects have been adapted to fetch the correct Plan Version from CL_DFS_COMMON_UTIL => GET_ACTIVE_PLVAR.

Please implement the attached correction instruction for solution.
