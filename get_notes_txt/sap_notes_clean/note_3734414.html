Environment

  • SAP HANA Platform Edition 2.0
  • 3rd party Backint Tool

Cause

The root cause of the issue lies with the third-party BACKINT configuration.

Resolution

Please coordinate with your third-party BACKINT vendor to review the current BACKINT configuration and address the issue accordingly.

  • Change the Log Backup Destination Type to the default local filesystem as a temporary workaround to ensure log segments are successfully backed up again.
