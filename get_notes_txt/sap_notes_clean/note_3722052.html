Symptom

During the mass processing of business partners by uploading the file no errors were shown in validation step but the activation step failed in background due to data error 'Entry 000 does not exist in FSH_OSS' and the dump is raised as below :
Category               ABAP programming error
Runtime Errors         UNCAUGHT_EXCEPTION
Except.                CX_FSBP_PARTNER_INCONSISTENT
ABAP: Program          FSBP_API_ADAPTER_INTERN=======CP
Application Component  FS-BP  

Environment

  • SAP Master Data Governance-Consolidation and Mass Processing

Resolution

  • Ensure there are no values for fields FSH_** in the upload file when the Business switch RFM_FASHION in SFW5 is not enabled. 

Keywords

FSBP_API_ADAPTER_INTERN, CX_FSBP_PARTNER_INCONSISTENT, FSH_OSS, Entry 000 does not exist in FSH_OSS, FSBP_GENERIC 007, FSBP_API_ADAPTER_INTERN,
