Symptom

A purchase order has been posted GR with Movement Type 107. After that, an inbound delivery is created by this purchase order.

When posting GR for this inbound delivery, an error message occurs.

Error Message: Inadmissible movement type  - Acceptance at vendor set for PO item (Message ID: M7429)

Environment

SAP S/4HANA Cloud Public Edition

Cause

In the inbound delivery, the default Movement Type for GR is 101.

The process for Goods Movement Type 101 contains the process for Goods Movement Type 107 and 109.

Resolution

Post GR for the inbound delivery with Movement Type 109 after posting GR for the purchase order with Movement Type 107.

Keywords

M7429, Movement Type, 101, 107, 109, SAP S/4HANA Cloud Public Edition, Purchase Order History, Inbound Delivery
