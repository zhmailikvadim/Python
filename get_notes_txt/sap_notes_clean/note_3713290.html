Symptom

  • How to create a user to access Query Builder only but no access to CMC/BI Launchpad

Environment

  • SAP BusinessObject Business Intelligence Platform :4.2 SP09 patch 1400+, 4.3 SP02 patch 1100+, 4.3 SP03 patch 0300+ ,4.3 SP04 patch 0000 and onwards.
  •  All Supported Operating System

Reproducing the Issue

  1. Login to CMC
  2. Create a user in CMC.
  3. Add the user to the Administrators user group.
  4. Go to Folders.
  5. Remove access to Folders for the user.
  6. Logon BI Admin Tools by the new created user.
  7. Run query to check a report : select * from CI_INFOOBJECTS where SI_ID='****'.
  8. No result.

Cause

  • Refer to: SAP Note 3404706 - Non-Administrator users unable to log on to AdminTools

Resolution

  1. Login to CMC ( Central Management Console) by administrator.
  2. Create a user in CMC.
  3. Add the user to the Administrators user group.
  4. Go to Applications.
  5. Right click Central Management Console and select User Security.
  6. Click Add Principals then select the new created user, and click the Add and Assign Security.
  7. Click Remove Access in the Assign Security page.
  8. Click OK and see No Access for the user in the User Security: CMC page.
  9. Go back to the Applications page.
  10. Right click BI Launch pad and select User Security.
  11. Click Add Principals then select the new created user, and click the Add and Assign Security.
  12. Click Remove Access in the Assign Security page.
  13. Click OK and see No Access for the user in the User Security: BI Launch pad page.

See Also
