Symptom

How to use the Refresh Structure option in Information Design Tool (IDT)

  • How to Refresh Structure table information from an universe

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Environment

  • SAP BusinessObjects Business Intelligence platform 4.3 SP3 and above
  • Information Design Tool (IDT)

Resolution

There are 2 ways to use the Refresh Structure option in IDT: 

From IDT menu:

  1. Open IDT; 
  2. Select the universe and open the data foundation;
  3. In the IDT Menu, click in Actions;
  4. Click in the option Refresh Structure.

From Data Foundation:

  1. Open IDT; 
  2. Select the universe and open the data foundation;
  3. Select one table > right click > click in the option Refresh Structure

OR 

  1. Open IDT; 
  2. Select the universe and open the data foundation;
  3. Select table folder > right click > click in the option Refresh Structure
