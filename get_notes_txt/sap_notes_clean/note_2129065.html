Symptom

Release Restriction Note for SAP Predictive Analytics 2.1

Solution

BI-RA-PA

  • Open-Source R and Expert Analytics
    • If you have configured the application with an existing R installation and you want to install the latest R version using the “Install and Configure R” option from the application, restart the application and install R before working with analyses.
    • You cannot use date columns as strings in the data that is passed to the custom R component. Therefore, we recommended to filter the date column from the dataset or use the as.date function in R script.
  • Creation and Execution of Analyses in Expert Analytics
    • While creating an analysis, you cannot drag and add a component between two connected components. Instead, you need to select the connector between the two connected components and double-click the required component in the component list.
    • If you try to rerun a CSV Writer component without selecting the Overwrite, if exists option, you will see the successful execution message instead of execution failure. However, the analysis is not executed in the backend if there are no changes to the properties of the CSV Writer component. You need to go back and select the Overwrite, if exists option and run the analysis again.
  • Publishing stories built using predictive datasets to SAP Lumira Cloud is not supported.
  • Changes made in Prepare and Compose tabs of Expert Analytics
    • If you apply filter or sort conditions, or create a new calculated measure in the Prepare or Compose tabs, the corresponding changes are not retained in the Predict tab.
  • When running an analysis using an Auto algorithm, an error occurs if there are column names in your data containing a forward slash "/". To work around this, change the column names in the Prepare room by replacing the forward slash "/" with an underscore "_" before running the analysis.
  • When exporting a SAP Automated Predictive Library (APL) model as a stored procedure, the first character of the model name cannot be a number.
  • Visualization issues occur with online R component. For more information, see SAP Note: 2129066
  • Expert Analytics can hang at splash screen without errors. For more information, see SAP Note: 2129068
  • Unsupported columns in table types
    • Processing of data may fail while using Predictive Analysis Library (PAL) components in Expert Analytics since the Application Function Library (AFL) does not support certain columns in table types.

    • The following are the supported CSTYPE (column store type)/SQLTYPE/DIMENSION combinations from AFL:

      • "INT" / "INTEGER"
      • "FIXED8_19_0" / "BIGINT" ("FIXED8_19_0" means fixed with 8 bytes and 19 digits)
      • "DOUBLE" / "DOUBLE"
      • "STRING" / "CLOB"
      • "STRING" / "NCLOB"
      • "STRING" / "VARCHAR" / ...
      • "STRING" / "NVARCHAR" / ...
      • "DAYDATE" / "DATE"
      • "SECONDTIME" / "TIME"
      • "LONGDATE" / "TIMESTAMP"
      • "SECONDDATE" / "SECONDDATE"
    • Note: Not all of these columns may be supported with PAL. For more information, see the PAL documentation at http://help.sap.com/hana_one/ -> SAP HANA Predictive Analysis Library (PAL) Reference.
  • Note: Online and offline auto algorithms provide similar functions, but are not identical and may generate different models.
  • Data manipulation functions are not supported when using Spark SQL in Data Manager module of SAP Predictive Analytics.
  • Problems occur when accessing HANA analytical views with SAP Predictive Analytics (Automated Analytics). For more information, see SAP Note: 2128272
  • Restrictions exist when using SAP Predictive Analytics (Expert Analytics) with HANA SPS09. For more information, see SAP Note: 2128915
  • In Expert Analytics, HANA CHAID algorithm performance is inversely proportionate to the number of distinct values for categorical features in the training dataset. As a result, when using Expert Analytics, if you do not find the performance of CHAID optimal for your use case, it is recommended to use other decision tree algorithms such as HANA C4.5. Note: Additional configurable parameters will be exposed in a future release of Expert Analytics to allow a threshold parameter that will help stop merging of categories beyond a specified threshold. This will allow use of CHAID algorithm with all kinds of datasets.
