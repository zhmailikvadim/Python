Symptom

Error occurs while executing the report Project Time Recordings.

Error: Failed to open report "Internal Server Error" - report an incident.

Environment

SAP Business ByDesign

Reproducing the Issue

  1. Go to Project Management work center
  2. Go to Reports view
  3. Select the report Project Time Recordings
  4. View With Web Browser
  5. Click on Apply
  6. Error appears

Cause

The issue occurred due to the large data and system cannot display all the projects in the list. However if you filter based on the Projects and then execute, system will not throw any error. 

Usually the error can be raised in the following circumstances:
- The number of records that is requested in the report is too high.
- The report is running out of memory due to the number of records that are processed in combination with the modeling of the custom data source / report.
- The report is running for too long and results in a time-out. 
This is an expected behavior.  

Resolution

Filter the data based on the projects to avoid the error while using Project Time Recording reports.
