Symptom

On UDO created with a general/image type UDF, the picture is not displayed in User-defined fields window. Double-click opens System Message window "Picture '...' was not found".

Reproducing the issue

Prerequisites:

Have defined table "T_IMAGES" with Object Type: Master Data; that has related user defined field "F_IMG" with Description "Image", type "General", Structure "Image".

Scenario:

  1. Menu - Tools - Customization Tools - Objects Registration Wizard... - add a new object with UniqueID "UDO_IMAGES", Name "UDO_IMAGES", Type "Master Data", Table Name "T_IMAGES", Default Form and Header Line Style and column "U_F_IMG" selected.
  2. Open "UDO_IMAGES" and User-Defined Fields window (Ctrl+Shift+U).
  3. Upload an image to the Image field.

Expected Behavior:

Picture previewed in UDF window.

Actual Behavior:

Picture not displayed, double-click opens System Message window "Picture '[picture_file_name]' was not found".

Solution

SAP intends to provide a patch or patches in order to solve the problem described. The section Reference to Related Notes below will list the specific patches once they become available. The corresponding Info file of the patches in SAP Service Marketplace will also show the SAP Note number. Be aware that these references can only be set at patch release date. SAP will deliver patches only for selected releases at its own discretion, based on the business impact and the complexity of the implementation.
