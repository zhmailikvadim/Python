Symptom

You wish to delete a wrong Best Practice solution before the complete activation:
For example: Errors are showing up when attempting to activate the best practice solution and you noticed that you are using the wrong version.

Might errors described in this KBA below appear while activating the wrong version of the solution:
3714046 - /SMB/DEFINITION045 when activating Best Practices solution

Environment

All S4/HANA versions

Reproducing the Issue

Try to activate the solution in t-code /n/smb/bbi and errors show up.

Cause

After an attempt to activate the wrong solution, the same has to be delete completely before activating the correct one.

Resolution

Be aware that this only can be done with wrong solutions that were not activated completely!
To delete the entire solution, please follow this steps below:

  1. Go to /n/smb/bbi
  2. Click on the wrong solution
  3. Delete button (top menu)
  4. A popup will appear, select Sol&ScopItms (solution and scope items) button
  5. After the deletion, go to Building Block Builder 
  6. Hit Display<-->Change button then it will switch to change mode
  7. On top menu, go to Scope Item > Delete Unassigned
  8. And on top menu again, go to Building Block > Delete Unassigned

With that, the solution is completely deleted.
