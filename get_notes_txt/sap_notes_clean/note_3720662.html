Symptom

• In the Global Trade Services (GTS) system, users want to know if it is possible to screen only parts of the Name field.  
• For example, if one BP has Name1, Name2, Name3, and Name4, can the system screen only Name1 and Name2?  

Environment

  • SAP Global Trade Services Edition for Hana

Resolution

  • The system does not support partial screening on the Name field in its standard functionality.  
  • The Name field is treated as an object containing multiple name parts (e.g., Name1, Name2, Name3, and Name4).  
  • During the screening process, the system considers information at the object level and screens the entire Name object from the Sanctioned Party List (SPL) master data.  
  • Partial screening would lead to hit loss and is not considered a reasonable compliance solution.  

Keywords

partial screening, sanctioned party list screening, SPL screening, SAP Global Trade Services, compliance management, SAP S/4HANA
