Symptom

  • Users can access CSB via BizX but after a few minutes working in CSB, they are logged out and sent back to the home page.
  • No error message or error code is displayed.

Environment

SAP SuccessFactors Recruiting

Reproducing the Issue

  1. Log into Bizx
  2. Access CSB successfully via Manage Career Site Builder
  3. Work for a few minutes.
  4. Observe that the session ends and the user is redirected to the SF home page.

Cause

Resolution

The solution of this issue is currently under investigation. Please follow this KBA for updates.
