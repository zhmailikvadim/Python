Symptom

When trying to create a new Job Requisition in the Recruiting page an error occurs:

"Your request has not been processed successfully. Please try again."

Environment

SAP SuccessFactors Recruiting Management

Reproducing the Issue

  1. Recruiting
  2. Job Requisition
  3. Click on "Create new"
  4. An error occurs: "Your request has not been processed successfully. Please try again."

Cause

This is a data sync issue, as it is not getting the Job Requisition Template from Provisioning.

Resolution

The issue can be resolved by re-enabling the Job Requisition template in Provisioning.

Keywords
