Symptom

You encounter problems in Consolidations in the SAM4U tool which are of type "Consolidation by Email" or "Consolidation by Full Name" where the list of roles and profiles for the users are empty when trying to display the details.

Other Terms

Software Asset Management, SAM4U Tool, SAM4U, PCE, Users and Roles

Reason and Prerequisites

You have installed SAP TCI Note 3646933 - SAM4U Solution | Fiori Application | V2 and other prerequisite corrections.

Solution

Implement this SAP Note to fix the above-mentioned error.
