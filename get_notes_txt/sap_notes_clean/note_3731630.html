Symptom

When submitting the Poland eDocuments to KSEF, following error occurs:

"You can have only one RFC active entry in transaction GCO_RFC_CONF"
Message no. GCO_ECOM_MESSAGES501


Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

  • SAP ERP
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition
  • SAP Document and Reporting Compliance, Cloud Edition

Reproducing the Issue

  1. Launch T-Code EDOC_COCKPIT.
  2. Select Poland KSeF eInvoice eDocument.
  3. Click on Submit.

Resolution

Follow the below steps as per SAP Help portal to resolve the issue.

SAP S/4HANA - Activating RFC for SAP Document and Reporting Compliance, Cloud Edition

SAP ERP - Activating a Standard RFC Connection
