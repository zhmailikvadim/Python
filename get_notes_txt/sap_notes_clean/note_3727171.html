Symptom

When trying to create new Rate Components, going to Admin Configuration > Rate Structure > Rate Components, there is no option to create a new Rate Component (New button is missing).

Environment

All Environments

Reproducing the Issue

  1. Admin Configuration > Rate Structure > Rate Components
  2. Verify the is no New button

Cause

  • Company is using Rates 2.0
  • When working with Rates 2.0 structure alone, additional rate components cannot be created from the Rate Component List page and new rate components are created when you create a rate component group.

Resolution

To create new Rate Components

  • If the Rate Component already exists
    1. Go to Admin Configuration > Rate Structure > Rate Component Group > New
    2. Complete all required fields
    3. Click + Add Rate Components > Click + New Component
    4. Complete all required fields > Add
    5. Repeat steps 3 and 4 to create more Rate Components
  • If the Rate Component does not exists
    1. Go to Admin Configuration > Rate Structure > Rate Component Group > Open the desired group
    2. Click Edit
    3. Click + Add Rate Components > Click + New Component
    4. Complete all required fields > Add
    5. Repeat steps 3 and 4 to create more Rate Components

You can also refer to SAP Help Center > SAP Fieldglass > Rate Component Group (Rates 2.0) for details on how to create these items.

See Also
