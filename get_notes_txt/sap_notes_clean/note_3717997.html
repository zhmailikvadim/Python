Symptom

  • Using Analytical Table -  sap.ui.table.AnalyticalTable
  • Using Fiori Elements v2 - List Report - Table type AnalyticalTable
  • Using Analytical List Report
  • The expand/collapse state of the table nodes does not get persisted to the View (personalization).

Environment

Fiori Elements v2

Analytical List Report

Analytical Table

Resolution

Due to technical limitations, it is currently not supported to persist the expand/collapse state of the analytical table nodes to a View (personalization).
