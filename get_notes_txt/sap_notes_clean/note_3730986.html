Symptom

How to extend customer records with credit management data.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

Cause

Resolution

  1. Use the migration object “Customer - extend existing record by credit management data” to create credit management data for existing customers.
  2. Create the required credit management profile data using this migration object.
  3. Create the required credit management segment data using this migration object (for example, segment assignments).
  4. Note: Updating or overwriting existing fields or records is not possible with this migration object; it can only add new credit management data to extend existing customer records.
