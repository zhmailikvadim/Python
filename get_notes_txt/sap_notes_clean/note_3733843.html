Symptom

  • The Payroll Control Center (PCC) page displays blank under all tabs after entering username and password.
  • The issue persists when opening the PCC via the direct UI5 application link.
  • Browser console shows: "Uncaught Error 'failed to load 'hcm/py/pcc/config/process/Component.js' from ./Component.js: 404-Not found'".
  • Browser console shows: "Failed to load resource: the server responded with a status of 403 (Forbidden)" while loading sap-ui-core.js from /sap/public/bc/ui5_ui5.

Environment

  • Human Capital Management (HCM)
  • Payroll Control Center (PCC)
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP SuccessFactors Employee Central

Reproducing the Issue

  1. Logon to PCC.
  2. Navigate to any PCC tab.
  3. Observe that the tabs render blank.
  4. Now you have tried to open console by hitting F12 button. There you can observe the console error message.

Cause

Inactive ICF (SICF) service node /default_host/sap/public/bc/ui5_ui5, causing 403 responses for UI5 core resources and subsequent UI load failures. It is a configuration issue.

Resolution

  1. Go to t-code SICF.
  2. Locate and activate the service node /default_host/sap/public/bc/ui5_ui5. Ensure that the public UI5 service is active.
  3. Verify and activate any required child nodes under /sap/public/bc/ui5_ui5 as applicable.
