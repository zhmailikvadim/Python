Symptom

  • Dynamic user lists cannot be selected on Work Space user control
  • Only individual users and static lists are available for selection

Environment

  • SAP Build Work Zone, Advanced Edition
  • SAP SuccessFactors Work Zone

Reproducing the Issue

  1. Go to a Work Space
  2. Go to Overview - Access Control
  3. Specific users or user lists:
  4. Dynamic user lists cannot be selected

Cause

  • This is an expected behavior, since assigning a dynamic user list in access control requires that the dynamic user list has already followed the workspace.
  • Static user lists can be selected by default on public workspaces because the allowed user lists include all static user lists.

Resolution

  • Ensure that the dynamic user list has followed the workspace to appear in the access control search.
  • If the dynamic user list does not appear during search, delete the search text entirely to view all available options, including the dynamic user list.
  • Verify that there are no spaces or extra characters in the search input, as this may affect the search behavior.
