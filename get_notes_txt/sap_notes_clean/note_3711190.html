Symptom

When activating the OData service /MERP/SAP_SRV_ASSET_MANAGER_2511, the activation fails with the following error:

"Metadata not loaded – See /IWBEP/ERROR_LOG in backend – Data object "WarehouseInboundDelivery" not found"

Environment

SAP Service and Asset Manager

Reproducing the Issue

  1. Open transaction /IWFND/MAINT_SERVICE.
  2. Choose Add Service.
  3. Select the OData service MERP/SAP_SRV_ASSET_MANAGER_2511.
  4. Attempt to activate the service.
  5. Activation fails with error Metadata not loaded.

Cause

An unsupported OData entity is active in ConfigPanel, which causes metadata generation to fail during activation of the OData service.

Resolution

  1. Open ConfigPanel.
  2. Navigate to OData Model Configuration.
  3. Search for the relevant application
    (for example: SAP Service and Asset Manager XXXX).
  4. Search for the entity WarehouseInboundDelivery.
  5. Select the entity.
  6. Deactivate the entity.
  7. Save the configuration.
  8. Retry activating the OData service /MERP/SAP_SRV_ASSET_MANAGER_2511.
