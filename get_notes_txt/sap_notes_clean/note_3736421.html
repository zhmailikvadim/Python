Symptom

When opening InternalJobHistory cardlink on performance form, the deeplink fails to load detail view and 400 error is thrown.

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

SAP SuccessFactors Performance Management

Reproducing the Issue

  1. Latest People Profile is enabled and cardlink for InternalJobHistory is configured in the form's section introduction.
  2. Go to the performance form, click the cardlink for internal job history and observe that a 400 error is returned.
     

Cause

This is a Known Issue.

Resolution

The fix for this issue is planned to be deployed on 1H 2026 release.

For release timelines, please review SAP SuccessFactors Product Release & Road Map Information.

Workaround

No workaround is available.
