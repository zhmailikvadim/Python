Symptom

When using a Conversion Module for TrackingIDs or AOIDs the Function Module cannot be determined. TheTemplate Function Modules are missing in the View for conversion template modules /SAPTRX/V_CONVTP.

Solution

Maintain the following entries for View /SAPTRX/V_CONVTP:

- Run transatction sm30.

- Enter the following entries:

AI Application Object ID input conversion /SAPTRX/AOID_IN_CONV_TMPL
AO Application Object ID output conversion /SAPTRX/AOID_OUT_CONV_TMPL
TI Tracking ID input conversion /SAPTRX/TRXID_IN_CONV_TMPL
TO Tracking ID output conversion /SAPTRX/TRXID_OUT_CONV_TMPL


- save the view maintenance.
