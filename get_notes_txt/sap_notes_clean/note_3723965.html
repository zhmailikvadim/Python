Symptom

In the process "Setup / Replacement of Devices (DSO View)" (DE_B_IM_DEV_CHANGE_DSO_PASV), step 5500-0030 incorrectly set the usage status to 'DR' for the MELO product instead of the MALO product during the initial creation of the Parameter Instance (PI).

This issue is relevant for the market roles: DSO.

Other Terms

US4G, MET, Parameter Instance, PI, MALO, MELO, usage status, Intelligent Metering Processes

Reason and Prerequisites

The error occurred because of a program error.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the system correctly sets the usage status to 'DR' for the MALO product during the initial creation of the Parameter Instance (PI). You need to retest the process. You can verify the fix by checking that the usage status for the MALO product is set to 'DR' during the initial creation of the Parameter Instance.
