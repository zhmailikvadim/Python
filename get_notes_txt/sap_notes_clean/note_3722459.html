Symptom

  • Usernames behind the user IDs in the Workflow Administration app are missing while forwarding the work items. Only CB user IDs are visible when selecting a user to forward to.
  • Difficulty in forwarding work items due to missing usernames.

Environment

  • SAP Business Workflow
  • SAP NetWeaver
  • SAP Web Application Server for SAP S/4 HANA
  • ABAP PLATFORM - Application Server ABAP

Reproducing the Issue

  1. Access the Workflow Administration app in SAP S/4HANA Cloud Public Edition.
  2. Attempt to forward workflow tasks to users based on their names.
  3. In the next step, select a user to forward the work items to.
  4. Observe that only user IDs are displayed instead of usernames, making it difficult to identify the appropriate user for forwarding invoices.

Resolution

  1. In Workflow Administration app (App ID: SWI_ADM_CLOUD) while forwarding the work items, by default the "S: Search Term" is used to search the user, if the user name is not listed please use the "User by Address Data" search function to filter and identify the forwarding user. The usernames should be displayed with this function.
  2. Alternatively, utilize the Flexible Workflow Administration app to forward work items. This app includes the functionality to display usernames during the forwarding process.
