Symptom

The user is not able to display/maintain any PI-Sheets or Execution Sheets. Only a blank screen appears. From the Chromium developer tools, it is observed that essential files have not been loaded from the Web Repository.

As a result, production activities based on PI-Sheets and Execution sheets are at complete stop.

Other Terms

CO60, CMX_XSH_OBJ_WORKLIST, COEBR, COEBR_ENH

SAP GUI for Windows, GUI 770, 800, Edge, Internet Explorer, Browser Control, blank, empty, content WebView2 runtime, 143, 144

Reason and Prerequisites

The reason is a yet unknown issue with the latest Webview2 Version 144.xxx

Solution

The only available solution as of yet is to not upgrade to the latest WebView2 release 144.xxx but stay with release 143.xxx or below.

The situation is already being investigated on component BC-FES-CTL and by Microsoft, see KBA 3704912.
