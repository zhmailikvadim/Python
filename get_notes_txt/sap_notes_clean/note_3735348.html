Symptom

The KBA explains how to download .txt Files in the background for the Malaysia legal report."Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Environment

  • Human Capital Management (HCM)
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP SuccessFactors Employee Central Payroll
  • SAP S/4HANA

Resolution

Take the CP8D .txt file as an example.

Steps:

  1. Run T-Code: PC00_M14_NCTEA or run program RPCTEAL0_01 via T-Code: SE38.
  2. Enter all mandatory fields, then go to "Program" and choose "Execute in Background".
  3. Fill out all required fields in the background-job dialog, then continue. Save the job according to the business requirements.
  4. Monitor the job via T-Code: SM37. If it fails, inspect the job log and spool.
  5. Verify the file exists on the application server via T-Code: AL11 or ask the basis team to check the OS directory. 

Note: File download path is defined via T-Code: FILE.

  1. Select Local File Path: HR_MY_FILE_PATH and click "Assignment of Physical Paths to Logical Path".
  2. Define Physical path for both UNIX and WINDOW_DT server.
  3. In a standard system,
    • Physical path "/tmp/<FILENAME>" is assigned to Syntax group "UNIX".
    • Physical path "C:\temp\<FILENA" is assigned to Syntax group "WINDOWS NT".

In this example, when the job is run in the background, the file is saved to the /tmp directory. This can be confirmed by selecting the "Tape" option on the selection screen.
