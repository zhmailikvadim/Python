Symptom

When you execute an SQL command of the form "INSERT <dbtab> FROM TABLE @itab", the system does not raise an exception if the operation leads to duplicate entries in the database table. Only SY-SUBRC is set.

Other Terms

KERNELCOR 

Reason and Prerequisites

 Error in the core.

Solution

This correction was originally delivered with the kernel patch level listed in section "Support Package Patches" of this SAP Note. All later kernel patch levels also contain this correction. Note that only the latest kernel patch is available for download. It contains all corrections delivered so far. Therefore, you can download and apply the latest kernel patch at any time to implement this correction.

If no kernel patch level is entered in section "Support Package Patches", the patch is not yet available. The entry appears with the delivery of the correction. Check this SAP Note to see whether new kernel patch levels have been entered.

This correction is delivered with the following SAP kernel archives:

  • Hotfix - file dw.sar
  • SP Stack Kernel - Files SAPEXE.SAR and SAPEXEDB.SAR

Follow the SAP recommendations for selecting and installing the kernel patch:

  1. Apply the latest SP Stack Kernel if it contains the correction. For a list of the current SP stack kernels, see SAP Note 2083594 (SAP kernel versions and SAP kernel patch levels).
  2. Otherwise, the correction will be provided with the next SP Stack Kernel. Apply the hotfix only if you need the correction at short notice.
  3. The correction may not be available for kernel releases that are no longer maintained. Upgrade to the corresponding downward-compatible kernel (DCK), see SAP Note 2083594 (SAP kernel versions and SAP kernel patch levels).
  4. Check the regression note for the target patch level before applying the patch. For detailed information, see SAP Note 1802333 (find information about regressions in the SAP kernel).
  5. For instructions on how to download and apply kernel patches, see SAP Note 19466 (Downloading SAP kernel patches).

For detailed information about SAP recommendations, see the strategy document: Update Strategy for the Kernel of the Application Server ABAP in On Premise Landscapes.
