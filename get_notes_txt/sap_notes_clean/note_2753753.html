Symptom

System date convertion to company timezone has been used in ODN class CL_FICA_ODN_XX.

Other Terms

System Date, Date, Timezone, Chronological Numbering

Reason and Prerequisites

Only relevant if your country can use ODN setting 'Assign Document Classes to Document Types and Origin Keys' with System Date. For most cases there is no difference between the server date and the converted date to company timezone.

Solution

As a general rule, SAP recommends that you install a solution by applying a Support Package. However, if you need to install a solution earlier, use the Note Assistant to implement the correction instruction. You can find more information about the Note Assistant in SAP Service Marketplace, under service.sap.com/note-assistant.
