Symptom

You use the report "Info System: Rental Objects (transaction: REISRO). If the rental object does not have an organizational assignment, you want to derive an organizational assignment from the hierarchy.

Other Terms

IS, BADI_REIS_LIST

Reason and Prerequisites

Deriving the organizational assignment from the hierarchy is very time-consuming and may result in long runtimes.

Solution

Implement the advance corrections in accordance with the correction instructions, or import the relevant Support Package.

If you want to execute a derivation for the rental object, you can do so in the BAdI "BADI_REIS_LIST" using the sample implementation in the class CL_EXM_IM_REIS_LIST_CN_PRCTR for the profit center, the business area, and the functional area.
