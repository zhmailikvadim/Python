Symptom

in table T582Awhen changing the time constraint to 'T' for an infotype (ie:0783); error message "Time constraint 'T' is not possible for infotype xxxx-Message No. 54218" is generated

 

Environment

  • Human Capital Management (HCM)
  • SAP R/3
  • SAP R/3 Enterprise 4.7
  • SAP ERP Core Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA

Reproducing the Issue

  1. go to SM30
  2. enter table view V_T582A
  3. Edit
  4. select required ITXXXX
  5. change time constraint to T- error below occurs

Cause

Standard

Resolution

  1. Message is generated because in table T777D the infotype is set with no time constraint table as per standard design

NB: table T777D is set to Delivery class E Control table, SAP and customer have separate key areas; which means changing standard entry is not allowed
