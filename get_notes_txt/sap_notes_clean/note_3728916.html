Symptom

Late Package Building is used for Package Item creation in TM. When repacking happened in EWM and Load End is reported, LDAP Notification processing ends in error messages like "Item &1 cannot be reassigned" and "Processing canceled; data received is not valid".

Other Terms

TM, UH, LDAP, /SCMTMS/TOR141, /SCMTMS/INB_AGENT002, ITEM_REASSIGN

Reason and Prerequisites

This is caused by a program error.

Solution

Implement the solution below or apply the corresponding Support Pack.
