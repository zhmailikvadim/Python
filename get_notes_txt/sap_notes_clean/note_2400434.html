Symptom

Define Planners screen under compensation template- replace job code with other fields from standard User Directory File

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Environment

SAP SuccessFactors Compensation Management

Reproducing the Issue

1. Under Compensation click on 'Compensation Home'

2. Open any template and go to Define Planners via Manage Users

3. The search of the planner shows his name and his job code, as below:

   

Resolution

Please be informed that the Define Planner screen is hard coded and it is not possible to add any field of User Directory File, like Division/Department/Location,etc along with planners name and job code.

Keywords

Planner field, Define Planner
