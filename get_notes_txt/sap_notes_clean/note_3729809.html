Symptom

You process an off-cycle run with reason 0108 (Sick-leave Calculation) after an advance off-cycle run, for example, with reason 0101 (Monthly Advance Earned on Date) or 0102 (Monthly Advance % of Monthly Salary). If the employee pays alimony or a penalty, the system calculates an incorrect reservation amount.

Other Terms

V_T512W_O, PE01, /O7D, UAO3

Solution

As a general rule, SAP recommends that you implement the Support Package specified for your release.

However, if you need to install the solution earlier, use the SAP Note Assistant to carry out automatic and manual post-implementation correction instructions. For more information, see SAP Note Assistant.

Note that after installation of HR Support Package, it is possible that customizing records will be imported into client 000 only. In this case, adjust the customizing of your work client manually. For more information about the customizing, see the attached file Manual Instructions for SAP Note 3729809.pdf.

This SAP Note delivers changes to Payroll objects. For a list of the current and previous versions of the relevant objects, see the attached document Payroll Objects of SAP Note 3729809.pdf.
