Symptom

  • You have updated an existing plan version description in transaction OKEQ based on new business requirements.
  • When executing transaction F.01 (Financial Statements), the plan version field still displays the old description.
  • The newly updated description is visible and correct in Controlling (CO) transactions, such as KP26.

Environment

  • Controlling (CO)
  • SAP R/3
  • SAP R/3 Enterprise 
  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA Finance
  • SAP S/4HANA

Cause

The update performed in transaction OKEQ applies exclusively to the Controlling (CO) module. Transaction F.01 belongs to the Financial Accounting (FI) module and retrieves the plan version description from a different data table (T894T).

Resolution

Access transaction GLPV to maintain the FI plan versions.

Keywords

F.01, OKEQ, GLPV, plan version description, financial statements report, plan version text, FI planning version, CO planning version
