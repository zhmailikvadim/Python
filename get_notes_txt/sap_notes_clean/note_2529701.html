Symptom

When entering a string of maximum length of an input field and adding additional characters afterwards, no error message is thrown even though the string is too long.

Other Terms

SID key figure, field length

Reason and Prerequisites

The input field is based on a characteristic as key figure.

Solution

  • SAP BW 7.40
    Implement Support Package 19 for SAP BW 7.40 (SAPKW74019) into your BW system. The Support Package will be available as soon as SAP Note 2526318 with the short text "SAPBWNews 7.40 BW ABAP SP19", which describes this Support Package in more detail, is released for customers.
     
  • SAP BW 7.50
    Implement Support Package 10 for SAP BW 7.50 (SAPK-75010INSAPBW) into your BW system. The Support Package will be available as soon as SAP Note 2505921 with the short text "SAPBWNews 7.50 BW ABAP SP10", which describes this Support Package in more detail, is released for customers

  • SAP BW 7.51
    Implement Support Package 5 for SAP BW 7.51 (SAPK-75105INSAPBW) into your BW system. The Support Package will be available as soon as SAP Note 2467673 with the short text "SAPBWNews 7.51 BW ABAP SP5", which describes this Support Package in more detail, is released for customers. 

  • SAP BW 7.52
    Implement Support Package 1 for SAP BW 7.52 (SAPK-75201INSAPBW) into your BW system. The Support Package will be available as soon as SAP Note 2505912 with the short text "SAPBWNews 7.52 BW ABAP SP1", which describes this Support Package in more detail, is released for customers. 

  • SAP BW/4HANA 1.0
    Implement Support Package 7 for SAP BW/4HANA 1.0 (SAPK-10007INDW4CORE) into your SAP BW/4HANA system. The Support Package will be available as soon as SAP Note 2526413 with the short text "SAPBWNews SAP BW/4HANA 1.0 SP07", which describes this Support Package in more detail, is released for customers. 

 

In urgent cases you can use the correction instructions.

Before you use the correction instructions, make sure that you check SAP Note 1668882 and SAP Note 2248091 for transaction SNOTE.

This SAP Note might already be available before the Support Package is released. In this case, however, the short text still contains the term "preliminary version".
