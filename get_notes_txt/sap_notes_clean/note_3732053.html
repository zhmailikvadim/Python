Symptom

In the process "Start of Supply for Electricity (New SUP View)" (DE_B_GPKE_SOS_SUP), step 1100-0030 contains check group CHKGRP_GPKE_SUPN_SOS_CA with check ID CHK_CP_DET_CA_BY_BP. The action items BP_NOT_FOUND and CA_NOT_FOUND_FOR_BP were missing an exception code assignment, preventing proper error handling.

This issue is relevant for the market role: SUP.

Other Terms

GPKE_SUP, Start of Supply for Electricity, DE_B_GPKE_SOS_SUP, CHKGRP_GPKE_SUPN_SOS_CA, CHK_CP_DET_CA_BY_BP, BP_NOT_FOUND, CA_NOT_FOUND_FOR_BP, EX_GPKE_SUPN_SOS_CA, contract account, business partner

Reason and Prerequisites

The error occurred because the action items BP_NOT_FOUND and CA_NOT_FOUND_FOR_BP in check ID CHK_CP_DET_CA_BY_BP lacked an exception code assignment.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the system assigns the exception code "SOS process: CA check" (EX_GPKE_SUPN_SOS_CA) to the action items BP_NOT_FOUND and CA_NOT_FOUND_FOR_BP. Import BC set /US4G/3_0_SUP_SP08_NOTE_3732053 or /US4G/3_0_SUP_SP07_NOTE_3732053, depending on your support package level. You can verify the fix by retesting the scenario and confirming that the exception code is correctly assigned to these actions in step 1100-0030.
