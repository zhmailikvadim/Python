Symptom

  • While entering proposed contribution values in info type 585 - Sec 80 deductions, the values are not getting saved
  • Values entered in info type 585 for Section 80 deductions are not being saved
  • Values in info type 586 are saving correctly but not in IT 585.
  • You are trying to maintain HR master data (Transaction code: PA30) for employees in your organisation. While updating Section 80 details in the Section 80 Declarations (IT0585) infotype, the values are not getting stored in the HR Master Database table (PA0585). However, these values should be saved in the database table (PA0585).
  • Info type 585 values not getting saved

Environment

  • Human Capital Management (HCM)
  • Payroll India
  • S4HCMCIN 100
  • S4HCMCIN 101
  • SAP R/3
  • SAP ERP Core Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA

Cause

NOTE: The issue is observed in releases 100 and 101 only.

Resolution

Through SNOTE, Implement SAP Note 3415814 - Section 80 Deductions Information not Stored in HR Master Database Table

See Also

SAP Note 3415814 changes have been shipped as part of following SP's
