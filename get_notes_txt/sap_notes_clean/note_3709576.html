Symptom

You integrate the Production Operation List with an alternative sequence that contains one operation. The predecessor of the operation in the alternative sequence is the last operation from the standard sequence. The last operation in the standard sequence has predecessors that are the penultimate operation from the main sequence and the operation from the alternative sequence. As a result, the system incorrectly determines the penultimate operation from the standard sequence as the branch operation for the alternative sequence in the SAP routing.

Other Terms

 PLMSI, PLM system integration, POL, Production Operation List, Sequence, Alternative, Standard, Branch, Operation

Solution

Implement this SAP Note to resolve the symptom.
