Symptom

RFQ release strategy not determined when using characteristic CEKKO-GNETW (net order value) with ranges or greater-than (>)

Environment

  • Materials Management (MM)
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Configure an RFQ release strategy that includes the characteristic CEKKO-GNETW (net order value) with a greater-than condition (for example, >= a threshold) or a numeric interval.
  2. Create an RFQ (for example, via ME41) whose value meets the configured threshold or falls within the configured interval.
  3. Save the RFQ.
  4. Display the RFQ (for example, via ME43) and check the release strategy; observe that the release strategy is not determined.

Cause

Limitation in Standard

Resolution

This is a standard limitation when net order value (CEKKO-GNETW) is used as a characteristic in RFQ release strategy customizing (for example, with value > 0 or defined ranges), the RFQ value is not considered for release determination. Target value (ekko-ktwrt) is entered but will not be considered for RFQs.
