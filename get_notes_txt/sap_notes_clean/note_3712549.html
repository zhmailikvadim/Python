Symptom

You want to set up Loans & Advances replication to IT0045 but doesn't know how.

Environment

  • SAP SuccessFactors Employee Central integration to SAP Business Suite;
  • Employee Master Data replication;
  • Point To Point (PTP).

Cause

IT0045 is not included in the standard PTP replication framework between Employee Central and Employee Central Payroll systems. The standard replication program (transaction HRSFEC_PTP_EE_REPL) does not automatically transfer Loans and Advances data to this infotype. 

Resolution

To replicate Loans and Advances data to IT0045, implement a custom logic process through the following BAdIs:

  • HRSFEC_B_CE_PROCESS_EMPLOYEE
  • HRSFEC_B_CE_PROCESS_ET

For more information, please check SAP Help Portal | Implementing BAdIs for Updating the Infotype Framework.
