Symptom

  • ERROR: com.crystaldecisions.thirdparty.org.omg.CORBA.UNKNOWN minor code: 0x0 completed: No (COR-10869)
  • Error when opening projects in Data Insight
  • Errors opening different tabs in Information Steward

Environment

SAP Information Steward

Cause

The EIMAdaptiveProcessingServer (EIMAPS) cache has become corrupted.

Resolution

Clear the EIMAPS cache following the steps in KBA 2064254.

Keywords

IS, MM, Metapedia, Metadata Manager, Corba
