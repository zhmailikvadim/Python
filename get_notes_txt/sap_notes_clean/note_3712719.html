Symptom

This SAP Note adds the contribution amounts for the Child and Child-Rearing Support Fund to Wage Ledger.

Other Terms

 RPCEWGJ1, /607, /617, /627

Reason and Prerequisites

This is a part of legal change. You need to apply SAP Note 3709289 firstly.

Solution

Apply the corresponding HR Support Package or implement the correction instructions in this SAP Note.
