Symptom

In Self Service Composition Environment for Boolean type selected tag from PIC, if you do some changes and then Save in 'Configure Tag Display' then it fails and we will have the following error in the logs of browser console:

Uncaught TypeError: Cannot read property 'getValue' of undefined, TypeError: Cannot read property 'getValue' of undefined at validateSettings..........

Other Terms

MII 15.0, SSCE, Boolean tag, PIC, Configure tag display

Reason and Prerequisites

This issue was caused due to the program error.

Solution

The issue has been fixed and released in the patches listed in the SP Patch Level section. Apply the proper patch for your MII version.
