Symptom

When downloading multiple agreements using the mass download transaction /IRM/IPBBAGDNLD, all agreement rules are not captured in the generated Excel file. This occurs due to a bug in the Excel processing logic during mass download, resulting in missing rules for some agreements.

Other Terms

Class-  /VTA/CL_BFTP_SERVICES , EXPORT_ROW_PROCESS, deleting duplicate keys

Reason and Prerequisites

 This is caused by Program error.

SAP Note#3712312(IP) is available for 60E Release 6.4 – Base Package & Support Package 1

Solution

Please implement the correction instructions attached. These corrections are only valid if you have applied the Support Packs.

Release 6.4 – Base Package & Support Package 1

Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object
