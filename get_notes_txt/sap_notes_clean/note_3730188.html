Symptom

During query on a partitioned table error message is reported:

  • Database error text: SQL message: [ASE Error SQL2835]The procedure tree is old. Use the sp_remap procedure to remap all the procedures in this database.
  • Sql error 2835   performing FET on table
  • Msg 14313, Level 11, State 2:
    Server 'SID', Line 1:
    Partition condition for object 1321231, indid 0, ptncondid 1321546 was not found in database 4.

Environment

  • SAP Adaptive Server Enterprise (ASE) 15.7 for Business Suite
  • SAP Adaptive Server Enterprise (ASE) 16.0 for Business Suite

Resolution

During low workload window or no use of the partitioned tables:

Force recreate the partition conditions for all tables runing with sapsa or for standalone systems with sa user replacing the <SID> for dbname of the error :

use <SID>

go

dbcc traceon(3604)

go

dbcc upgrade_object(<SID>,"condition","force") 

go

Keywords

14313,2835 ,ASE Error SQL2835
