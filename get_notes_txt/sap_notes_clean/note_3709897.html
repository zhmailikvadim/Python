Symptom

  • When resized zoom to 200% using browser settings, some text becomes cut off or visually truncated, for example, text within forms and dropdown fields is no longer fully visible, making it difficult for users to read or interact with the content properly.

  • Violation reason: 

    WCAG 2.0 - Ensure 200% zoom works with no content/function loss

    WCAG 2.1 - Adds reflow requirement at 400% zoom for small viewports (mobile-first accessibility). 

***Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.*** 

Environment

SAP SuccessFactors Recruiting Management

Reproducing the Issue

  1. Log in as an External Candidate to the Career site.
  2. Increase browser size to 200%.
  3. Observe in the job search filters, there are texts truncation and misalignment of dropdown arrows.

Cause

Resolution

Engineering team has fixed this and will be available as part of 1H2026 release.
