Symptom

  • How to Edit all the Manual Retrofit Objects at once in SAP Cloud ALM?
  • Update the Implementation Status and Target Transport in Manual Retrofit Objects at once. 

Image/data in this KBA is from SAP systems, sample data, or demo systems. Any resemblance to real data is purely coincidental

 

Environment

  • SAP Cloud ALM 1.0
  • SAP Cloud ALM 2.0

 

Reproducing the Issue

 

Resolution

If the Category of your source transport is only Manual, you have to process the objects of the source transport manually.

  1. Open the detail view of the source transport that only has the category Manual. This shows you the objects that have to be retrofitted manually.



  2. In Edit Mode, you can edit your objects, change the Implementation Status, edit and document target transports. To use the mass edit option, follow the below steps.
     
    1. Select all the Objects for which you want to Edit the Status in bulk:



    2. In the Edit pop-up, select "Enter New Value" and enter the desired values in both the "Implementation Status" field and the "Target Transport" fields. 



    3. Click on OK and then Save. The Changes are now applied across all the selected objects.
