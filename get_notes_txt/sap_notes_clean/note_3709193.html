Symptom

  • EarlyWatch Alert (EWA) is created manually for a managed system.
  • The manually created EWA report is generated successfully.
  • However, the regular weekly EWA is not generated afterward.
  • The expectation is that the manual EWA behaves the manual EWA to behave the same as the scheduled EWA.

Environment

SAP Solution Manager 7.2

Resolution

  1. Use manual EWA creation only for one-time or ad-hoc analysis.
  2. For regular weekly monitoring:
  3. Ensure EWAs are generated via the scheduled EWA process
  4. Do not rely on manual execution
  5. Allow the next scheduled EWA cycle to run automatically.
  6. No corrective action is required if the manual EWA report was generated successfully.
