Symptom

In SAP Solution Manager 7.2, you have configured email notifications to be sent when specific attributes of an Incident or Transaction—such as Status or Support Team—are changed. However, the notification email does not include the updated data from the corresponding Incident or Transaction. This issue occurs only if the “Send E-Mail” command was used previously and only under certain specific conditions.

Other Terms

Solman, Incident, Email, Business Activity, GDC, Global Data Context, Reset, Initialized.

Reason and Prerequisites

The Global Data Context (GDC) data is not properly initialized.
This behavior is caused by a program error.

Solution

Implement the attached correction instruction.
