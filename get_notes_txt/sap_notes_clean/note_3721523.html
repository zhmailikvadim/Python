Symptom

  • An error occurs when attempting to copy or maintain an IT0008 record for Indian employee.
  • Error message displayed: "Message No.: RP129 - Wage type model changed on ."

Environment

  • Human Capital Management (HCM)
  • SAP R/3
  • SAP ERP Core Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • Payroll India

Reproducing the Issue

  1. Go to t-code PA30.
  2. Now pass the PERNR details and info-type as '0008'.
  3. Click on the create button.
  4. After fulfilling all the required fields, click on the save button.
  5. Error message RP129 is triggered. 

   

Cause

Incorrect customization of wage type permissibility.

Resolution

  1. Go to SM30 t-code.
  2. Now pass the view name as V_511_B.
  3. Click on the maintain button.
  4. Pass HCM Localization value as 40.
  5. Set the value 1 for the applicable Employee subgroup grouping field for all those wage type which has been used on IT0008 screen.
