Symptom

HCM Fiori applications are facing the issue that an incorrect manager is determined, if an employee’s position is assigned as a manager for a different organizational unit than his own.

Other Terms

I_HCMOrgHierManager, I_HCMOrgHierDirectReport, I_HCMOrgHierColleague

Reason and Prerequisites

The listed CDS Views use an incorrect implementation leading to an incorrect determination of a manager when an employee’s position is also assigned as a manager for a different organizational unit.
All HCM S/4HANA Fiori applications that use the above-mentioned CDS Views are affected.

Solution

Implement the attached Correction Instruction (CI) or the relevant Support Package.
