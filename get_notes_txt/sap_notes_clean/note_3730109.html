Symptom

Your try to calculate payroll for employee with amount more than 100.000.000,00 Rub. You see an error CX_SY_ARITHMETIC_OVERFLOW.

Other Terms

HRUCALC0, HRUCALC0_CE, RUTD

Solution

As a general rule, SAP recommends that you implement the Support Package specified for your release.

However, if you need to install the solution earlier, use the SAP Note Assistant to carry out automatic and manual post-implementation correction instructions. For more information, see SAP Note Assistant.

This SAP Note delivers changes to Payroll objects. For a list of the current and previous versions of the relevant objects, see the attached document Payroll Objects of SAP Note 3730109.pdf.
