Symptom

  • The XML file generated via report /CEECV/ROFI_394_2025 does not include the op11 field for certain documents.
  • Error R233.5 occurs when attempting to upload the XML file to the tax authorities via the DUK integrator program.

Environment

  • SAP ERP
  • SAP S/4HANA

Reproducing the Issue

  1. Generate the XML file for Declaration 394 via report /CEECV/ROFI_394_2025.
  2. Attempt to upload the XML file into the DUK-integrator system.
  3. Observe that error message R233.5 appears.

Cause

Missing configuration in table /CEECV/ROFI_NC. Tax codes required for generating the op11 field in the XML file are not maintained in the table.

Resolution

  1. Access table /CEECV/ROFI_NC.
  2. Maintain the missing tax codes associated with the erroneous documents.
  3. Retest the scenario to ensure the op11 field appears in the XML file for the relevant tax codes.
  4. Upload the XML file to the tax authorities to confirm the issue is resolved.

Keywords
