Symptom

When processing billing documents through transaction /IRM/IPCRCLBSMP, you expect calculation buckets to be updated only for valid billing documents. However, the system is also updating calculation buckets for cancelled billing documents, which leads to incorrect bucket data.
This note corrects the behavior by ensuring that cancelled billing documents are excluded from calculation bucket updates.

Other Terms

/IRM/IPCRCLBSMP, cancelled billing document, calculation bucket update, invoice cancellation, bucket processing issue

Reason and Prerequisites

This is caused by program error.
Note : SAP Note#3716695 is available for ECC release.

Solution

Please implement the correction instructions. These corrections are only valid if you have applied the following Support Packs.

Release V/4 – 2022 V2, 2023, 2023 V2, 2023 V3 and Support Pack 1

Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object
