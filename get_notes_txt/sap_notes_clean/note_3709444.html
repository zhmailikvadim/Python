Symptom

During order creation, you get an error saying "Field xyz (structure xyz) does not belong to the xyz structure."

Other Terms

PDS, /SAPAPO/RRP3, characteristic mapping, /SAPAPO/CUCDP_REFMAP, /SAPAPO/RRP518, /SAPAPO/CL_PDS_PPE_FILTER, EVAL_FILTER, /SAPAPO/CULL008, /SAPAPO/CULL007, /SAPAPO/CULL006, /SAPAPO/CULL005

Reason and Prerequisites

The issue arises because Data Dictionary fields are being modified using object dependencies, but the required mapping has not yet been maintained. These dependencies must first be mapped by transaction /SAPAPO/CUCDP_REFMAP.

Solution

The complete process is described in Consulting Note 610873.
