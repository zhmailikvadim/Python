Solution

Solution Description

After you implement this SAP Note, the system enables the calculation of IPI tax, considering a tax rate with 4 decimal places in the purchasing and goods movement processes. Plus, usage of tax rate 4 decimals is enabled for II (imposto de importação) in the creation of NF Writer (J1B1N transaction) process.

This SAP Note provides a detailed solution description. It outlines the changes, requirements, and system adjustments needed to accommodate these new tax obligations in SAP systems.

The system has changed as follows:

  • Mapping of the new IPI tax rate 4 decimals to nota fiscal 
  • Proposed pricing procedure changes (e.g.: TAXBRA as standard) 
  • New proposed pricing elements like condition types 
  • Inform tax rate with 4 decimal places for II in NF writer creation
