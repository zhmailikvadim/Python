Symptom

During item deletion, events referencing the deleted item need to be recalled. However, it could happen that another event, which is not referencing the item, will be recalled. For example: Proof of Delivery is recalled instead of the Delivery of Item event.

Other Terms

ref_exec_key, toritmuuid, TOR-ITEM_TR-DET_ITEM_TR_AM, TOR-ROOT-REPORT_EVENT, POD, ITEM_DELIVERY

Reason and Prerequisites

Program Error.

Solution

Implement the attached correction instructions or import the specified Support Package.
