Symptom

This note describes the restrictions known for SAP BusinessObjects Lumira, server version for BI Platform 2.0 for SP01

Solution

1) Unable to retain bookmarks after you save the same bookmark in a different folder

If you save an existing bookmark in a different folder from where the document was saved earlier and then reopen that document, the bookmarks are not available in the document.

Also, bookmarks are not retained after scheduling the document or performing Save As in the document.

Also, after opening a Multiple dataset document in SAP BusinessObjects Lumira, server version for BI Platform, and creating filters using both datasets and bookmarking it, when you delete one of the filters and refresh the document, the Bookmark is not saved correctly.

2)      OpenDocument parameters does not work for a few parameters

The parameters for Part mode and opening a document in same or new window is not supported.

3)      Unable to view the document in BI Launch pad, after you schedule and save the document in Inbox

In BI Launch pad, after you schedule a document and save it in the Inbox folder, you cannot view that document.

4)  Unable to change the font type for numbering

In a story, if you try to add a Text containing numbers and then try to change the font size, only the text reflects the font change and not the numbers.

5) Charts do not retain the bookmark state after removing the filtered dimensions from the charts

After creating visualizations using blended datasets, if you apply filter on a dimension, all the charts created using that dimension gets filtered. Now after bookmarking that document, if you try deleting the filter and reloading the bookmark, the chart does not contain the filtered dimension and hence does not retain the bookmark state.

6) Issue in saving the document having bookmark with different document name

For a document that has bookmark, when you try to save the document with different name and then reopen it, you cannot find the bookmark in that document.

7) DSL is not working for unx with data federation

DSL (Data Semantic Layer) / Universes are currently not supported on BI Platform.

General Restrictions:

  •  SAP BusinessObjects Lumira, server version for BI Platform is not supported on Linux platform.
