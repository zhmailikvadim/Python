Symptom

User Profile Change Requests (UPCR) Reporting in Ariba B&I

Cause

User Profile Change Requests (UPCRs) are not a reporting fact and no standard API or Report exposes Field-level change details in bulk.

Resolution

User Profile Change Requests (UPCR) are not included as a Reporting Fact within the Ariba reporting framework. As a result, Field-level changes cannot be exported through Analytical Reports.

Although UPCR Field-level details can be viewed individually within the document, which is not exposed for bulk Reporting or through Standard APIs.

Currently, SAP Ariba does not offer:

  • A Reporting fact for User Profile

  • A Standard Report for UPCR field-level changes

  • An API capable of exporting UPCR changes in bulk, as the Operational and Analytical Reporting APIs focus primarily on procurement transactions (such as Requisitions, Purchase Orders, and Invoices)

Therefore, this behavior is confirmed to be a Product limitation rather than a Configuration issue.

If bulk export is required, consider submitting a product enhancement request through the standard channels
