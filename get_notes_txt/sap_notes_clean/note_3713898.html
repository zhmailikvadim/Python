Symptom

You run the ICMR task. The execution status is updated in the consolidation monitor after the execution. Nevertheless, there is no log available in the App “Display Intercompany Reconciliation Log”.

Solution

Import of Support Package or advanced correction by implementation of correction instruction. You can then check the execution messages directly after the execution is completed or via the last log. Please do not use the App “Display Intercompany Reconciliation Log”.
