Symptom

This SAP Note provides relevant system changes to prepare the Empregos simultâneos (0437) infotype for alphanumeric CNPJs that become effective July 2026.

Other Terms

Alphanumeric CNPJ; Infotype 0437; Master Data; HR Renewal; Empregos simultâneos;

Reason and Prerequisites

Reason

According to the Normative Instruction nº 2229, October 15, 2024, starting from July 2026, CNPJs may be alphanumeric. Therefore, system adjustments are necessary to ensure correct reporting of this information.

Prerequisites 

You have applied the following SAP Note as prerequisite:

  • 3730575 - Prerequisite Objects for SAP Note 3720943, 3732954 and 3730656;
  • 3720943 - Alphanumeric CNPJ - Adapt Empregos simultâneos infotype (0437);
  • 3732954 - Prerequisite Objects for SAP Note 3730656.

The following Support Package level is the minimum prerequisite:

Release Support Package
EA-HR EHP 6.07 HR SP E2
EA-HR EHP 6.08 HR SP B9
S4HCMCBR 1.00 HR SP 08
S4HCMCBR 1.01 HR SP 00
S4HCMCBR 1.02 HR SP 00

For more information, access the SAP ERP Human Capital Management - Support Package Schedules.

When you install an SAP Note, some prerequisites might be required. The prerequisites are SAP Notes that have already been released and that change the same context of the current one. Then, even if these prerequisites are not necessarily related to the symptom of this SAP Note, you must implement them due to technical reasons.

Solution

The 0437 infotype supports alphanumeric CNPJs, ensuring compliance with the Brazilian government regulation effective July 2026.
