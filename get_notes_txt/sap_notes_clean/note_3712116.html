Symptom

In the delivery customizing client of the S/4HANA system, the default ACL setting is currently marked as "ACL with inheritance". This is incorrect. The setting should be changed to "No ACL". If the customer chooses to implement an ACL framework later, they can then adjust the settings to their preference.

Other Terms

ACL, OPSA, TCJ41

Reason and Prerequisites

Program error.

Solution

Implement the Manual instructions mentioned in this note.
