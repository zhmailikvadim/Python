Symptom

The PO number in the "External Reference" field in a service contract is not transferred to the "Custom Reference" field under the "PO Data" tab in the subsequent billing document request.

Environment

SAP S/4HANA Cloud Public Edition

Cause

To transfer the external reference to the subsequent BDR, the corresponding copying control needs to be set.

Resolution

Change the indicator "Reference Number" from E to A for the copying control of BSC to SVTB using the configuration activity "Define Copying Control for Sales Document to Billing Document" (ID: 102762). 

Keywords

po number, external reference, service contract, bdr, billing document, copying control, reference number, billing document request, define copying control, copying control, 102762
