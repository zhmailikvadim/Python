Symptom

This SAP Note provides you with the following corrections to year-end infotypes when Concurrent Employment (CE) is activated, and the Eligibility field is set to Permitted in All Circumstances (3) in table view V_T582ITVCLAS:

  • When you use transaction PA30 to edit the following infotypes, the fields Applicant Name, Relationship, and Registration No. don't automatically update after you select a different applicant name from the F4 help pop-up window:
    • Expense Information (0881)
    • Donation Expense (0858)
    • Medical Expense (0812)
  • The overview screen for the Expense Information (0881) infotype now displays records individually.

Other Terms

Infotype

Reason and Prerequisites

This is a program error.

Solution

You can implement the relevant correction instructions in this SAP Note.
