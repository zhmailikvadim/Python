Symptom

Lock functionality disables the ‘Post’ functionality in the calculation run. This SAP Note will enable the postings even if the calculation run is in display mode.

Other Terms

Enable Post in Display Mode, Calculation run Type, Calculation run, Participants Data area, Accrue, Reverse Accrue, Interim Settle, Reverse Interim Settle, Settlement, Reverse Settlement, Settlement Adjustment, /IRM/IPCI, EPDIS, /IRM/IPxxPCRM, /IRM/IPxxPCRCMP

Reason and Prerequisites

This is caused by program error.

SAP Note # 3705264 is available for ECC Releases.

Please implement the latest version of SAP Note #3703400 as pre requisite note.

Solution

Please implement the correction instructions. These corrections are only valid if you have applied the following Support Packs.

Release V/4 – 2023V2

1. Perform the manual pre-implementation steps that are described below
2. Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object
