Symptom

The program RS_ABAP_INIT_ANALYSIS does not exist in the system.

Environment

  • SAP NetWeaver application server for ABAP
  • ABAP Platform

Reproducing the Issue

  1. Execute SE38
  2. Run the program name RS_ABAP_INIT_ANALYSIS
  3. The message "Program RS_ABAP_INIT_ANALYSIS" does not exist is displayed.

Resolution

Use RS_ABAP_SETUP_ANALYSIS instead

See Also

KBA 2270689 - Remote Analysis (for source system)
