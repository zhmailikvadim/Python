Symptom

On SAP releases 757, 758 and 816 (2022 onwards) SQL based Service Bindings are not enabled for SNOTE versioning.

Other Terms

SQL Service Binding, SNOTE for Service Binding, SRVB, Version Management

Reason and Prerequisites

SQL webAPI based service binding is available from 2022 release but the SNOTE versioning was not supported. This note enables SQL based service binding from 757 (2022 release onward). INA and ODATA bindtypes for SNOTE versioning in service binding are supported from 816 (2025 release)

Solution

Apply the note for enabling SNOTE for SQL based service bindings (757 release onward)
