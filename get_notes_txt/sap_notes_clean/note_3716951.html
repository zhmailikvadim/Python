Symptom

You use the Run Statutory Reports app to create the VAT Declaration (Modelo 303) for Spain (report category ES_VAT_DCL_M303_NATIONAL) for January 2026.

When you upload the TXT file in the Spanish Tax Agency's pre‑production or production portal, you may notice that several errors are shown, such as:

  • FRECH – El fichero se rechazó. No se importó ningún registro.
  • E010450 – Caracteres no válidos 'Adquisiciones intracomunitarias - Cuota [11]'
  • E010460 – Caracteres no válidos 'Base imponible [12]'
  • …
  • E030280 – Caracteres no válidos 'Resultado - Pago a cuenta… [112]'

Other Terms

 Document and Reporting Compliance, Spain, 303, VAT Return, January 2026

Reason and Prerequisites

This SAP Note is relevant only for S/4HANA Cloud Public Edition for Spain.

The Spanish Tax Agency has activated the new Form 303 format for fiscal year 2026, starting with the January submission.
Your system does not yet contain the updated version of the ES_VAT_DCL_M303_NATIONAL report required to generate the TXT according to the new layout.
Therefore, the Tax Agency rejects the uploaded TXT file.

Solution

The official deadline for submitting the monthly VAT Return (Form 303) for January 2026 is March 2, 2026, as published by the Spanish Tax Agency
(see: https://sede.agenciatributaria.gob.es/Sede/ayuda/calendario-contribuyente/calendario-contribuyente-2026/calendario-anual/marzo/hasta-2-marzo.html).

If you wish to perform early testing, you can choose one of the following options:

  1. Wait for the updated report version.  The latest version of ES_VAT_DCL_M303_NATIONAL, including the 2026 format, is expected to be available by March 1, 2026.
  2. Perform manual entry in the Tax Agency portal.  For urgent cases, you may manually copy the values from the Preview screen into the Tax Agency’s legal form.
  3. Manually reorganize the TXT file generated by the Statutory Report Run
    This file generated by the "Run Statutory Reporting" app has to be manually restructured as defined by the Tax Agency in the format  (see https://sede.agenciatributaria.gob.es/static_files/Sede/Disenyo_registro/DR_300_399/archivos_26/DR303e26v101.xlsx):
    • From position 0  till  457 of the original TXT file +
    • Character "0" +
    • from position 457 till  496 of the original TXT file +
    • from position 1284 till 1323 of the original TXT file +
    • from position 496 till 793 of the original TXT file +
    • from position 1323 till 1362 of the original TXT file +
    • from position 793 till 1284 of the original TXT file +
    • from position 1363  till 1885 of the original TXT file +
    • from position 1885 till 2232 of the original TXT file +
    • from position 2316 till 2332 of the original TXT file +
    • from position 2231 till 2282 of the original TXT file +
    • "00000000000000000" +
    • from position 2282 till 2316 of the original TXT file +
    • from position 2332 till 2348 of the original TXT file +
    • from position 2469 till 2471 of the original TXT file +
    • from position 2349 till 2452 of the original TXT file +
    • from position 2471 till end of the original TXT file
