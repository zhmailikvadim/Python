Symptom

Report Assignment Overview is not Available

Environment

 SAP Business By Design

SAP Cloud For Customer

Reproducing the Issue

1. Go to  Business Analytics Work Center
2.Go to View Design Reports
3. Click on Assignment Overview

You can able to see the report assignment overview but that assignment overview is not present in any report

Cause

Currently there is no report available to see the Report Assignment Overview.

Resolution

You can only able to see the Report Assignment Overview in the Business Analytics Work Center only. There is no other way to see or extract that data.

Keywords

Business Analytics, Report Assignment Overview, Assignment overview report.
