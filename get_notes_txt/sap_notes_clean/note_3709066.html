Symptom

In the app F4006 the message "Material & does not exist for combination &/&/&." is displayed.
(It is from message class FML_UPLMATINVPR)

However, the material exists for combination Material/Currency Type/Ledger/Valuation Type, it is misleading message.

Existence of material can be checked in tables MBEW or CMATINVPR, alternatively in transaction MM03.

Other Terms

IF_FML_UMIP_PROCESS_UPLOAD~CHECK_INVTRY_VALUATION

Reason and Prerequisites

Clear authorization message is missing.

Solution

- Implement this SAP Note at your back-end system.

- New message 027 "No authorization for Plant &." is introduced.

- If message is generated, then Authorizatoin must be granted or plant must be removed from excel file.
