Environment

  • Cloud Integration

Reproducing the Issue

iFlow is reporting the error above under 'Manage Integration Content' in the Web UI.

Cause

The folder listed on the mail adapter configuration does not exist on the mail server.

Resolution

Update the mail adapter configuration and point to a folder which exists on the server.
