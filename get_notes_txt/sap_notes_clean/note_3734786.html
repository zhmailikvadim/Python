Symptom

Under Adjust employee compensation to tariff changes(Manage Pay Increase), future hires are not included in employee group.

Environment

SAP SuccessFactors Employee Central - Compensation Information

Reproducing the Issue

  1. Open Adjust employees compensation to tariff changes (Manage Pay Increase) and create a new pay increase.
  2. Enter the effective pay scale change date.
  3. Create or select an employee group that uses a dynamic group filter on pay scale group set “extend by n days”.
  4. Simulate or update the list of impacted employees.
  5. Observe that future hires within the extension window are not listed when using the employee group.
  6. But while choosing the pay scale group directly includes future hires are included.

Cause

Resolution

This is an expected behavior for employee group.
