Symptom

This SAP Note corrects following issues when executing the Social Risk Declaration (RPCSRDB0) report for scenario AL01 (Declaration of incapacity allowances):

  • when sending a declaration AL01 (zima0001) the tag <RiskStartingDate> is rejected with code <ErrorID>00408-003</ErrorID>.

Other Terms

AL01, SRD, RPCSRDB0

Reason and Prerequisites

  • This SAP Note is relevant for Belgium (PY-BE) only.
  • You are executing the Social Risk Declaration (RPCSRDB0) report for scenario AL01 (Declaration of incapacity allowances).     

Solution

If an error(s) occurs during simulation payroll then all declarations being prepared for the related employee(s) are cleared and an errror message is displayed (No payroll result simulated; check RPCALCB0 for pernr xxxx).
