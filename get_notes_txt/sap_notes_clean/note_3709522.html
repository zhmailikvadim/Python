Symptom

In the process "Request for Quotation (MOS View)" (DE_B_MOSB_REQT_MOS), the system could not create a new time slice when there were only obsolete status time slices in the contract. This occurred in step 0082.

This issue is relevant for the market role: MOS.

Other Terms

MOSB, DE_B_MOSB_REQT_MOS, /US4G/CL_CHK_MOSB_PROCESS, CHK_CREATE_NEW_TS, Time Slice, /US4G/AP_BILLING, Price catalog, Request for Quotation

Reason and Prerequisites

Program error.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After the fix, the system runs through step 0082 successfully and creates the new time slice. You need to run the BPEM case.

You can verify the fix by checking that the process "Request for Quotation (MOS View)" runs successfully.
