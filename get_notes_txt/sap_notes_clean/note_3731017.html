Symptom

After enabling the Latest People Profile (PPX), the Emergency Contact and Dependent Information UI displays Address fields as mandatory.

Environment

  • SAP SuccessFactors Employee Central - Emergency Contact Details
  • SAP SuccessFactors Employee Central - Dependents
  • SAP SuccessFactors Employee Profile -Latest People Profile

Reproducing the Issue

  1. Navigate to People Profile for a user.
  2. Open Personal Data.
  3. Go to Emergency Contact or Dependent.
  4. Click Add.
  5. Observe that the Home Address section fields appear mandatory.

Resolution

  • This is the expected behavior in the Latest People Profile. When selecting the Edit icon, full details for all emergency contacts become editable directly, and the address section displays with mandatory indicators.
  • Workaround - Temporarily disable the option "Enable the latest Editing and History UI for Employee Central HRIS entities on Full Profile (Except Compensation Information)" in Configure Latest People Profile.
  • This is planned to be fixed in 2611 release (tentative).
  • Please refer to this Enhancement reference and contact your CSM to track this reference number - ECT-268069.
