Symptom

You are working with transportation cross docking (TCD) relevant deliveries.
If you release a wave which contains several TCD relevant deliveries, a handling unit (HU) mapping error can happen and the following errors are raised:
"Error during WT creation for wave item x y z" (/SCWM/WAVE021)
"No CD-WTs for Wave Item x y z created" (/SCWM/WAVE179)

The same HU mapping problem happens if you try to create the warehouse tasks for TCD relevant deliveries at the same time in the warehouse task creation transaction.

Other Terms

/SCWM/TODLV_O, /scwm/wave,

Reason and Prerequisites

Program error.

Please apply the source below.
