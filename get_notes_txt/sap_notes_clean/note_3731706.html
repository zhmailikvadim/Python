Symptom

Cost breakdown templates get corrupted when updated/created via the mass maintenance option. This behavior is the same in both customer and SAP Standard application variants and with different cost breakdown templates.

After using the mass maintenance option to create cost breakdown for multiple items, the template is shown correctly only in the item 1, for other items it is corrupted .
