Symptom

When attempting to add attributes (competencies or skills) to a development goal from the Goal Creation menu in the Development Plan/Goal, the system displays a “No Results Found” message, even though competency and skills data are available in the system.

Environment

  • SAP SuccessFactors Career Development Planning
  • SAP SuccessFactors Talent Intelligence Hub

Reproducing the Issue

  1. Navigate to the Goal Creation menu for the Development Plan/Goal.

  2. Attempt to add attributes by selecting either Competencies or Skills.

  3. Observe that the system displays “No Results Found” or “0 results”, despite relevant competency or skills data being available.

Cause

The issue occurs due to the configuration in the Development Plan template, which excludes competencies from forms, the absence of a current role assigned to the user, and the lack of tags configured in the template. 

Resolution

To address the issue, perform one of the following actions:

  • Configure a current role for the user and map competencies to this role
  • Include competencies by tag: 
    <competency-filters>
      <exclude type="forms"/>  <exclude type="roles"/>
      <include type="library" match="tag name"/>
    </competency-filters>
  • Add switch <switch value="on" for="show-competency-browser"/> in development template to load ALL competencies and skills in the instance.
