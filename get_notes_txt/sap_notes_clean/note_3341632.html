Symptom

Automated emails sent via Workforce software (ACT notifications, leave requests, etc) are not being received by employees with Gmail accounts. 

Environment

SAP Time and Attendance Management by WorkForce Software

Resolution

For Time off Request emails and Approval Deadline emails to generate, you need to make sure of the following:
1. System Scheduler is turned on.
2. Emails are enabled in Policy Records > Setup > Environment Profile > NOT_PROD
3. Allow Sending Email Message is enabled in Policy Records > Setup > System Setup > Messaging Tab
4. Add email address to appropriate fields. Email addresses must exist on APP_USER.EMAIL_ADDRESS record for User (Group Role) related emails, and on EMPLOYEE_MASTER.EMAIL_ADDRESS for employee related emails.
