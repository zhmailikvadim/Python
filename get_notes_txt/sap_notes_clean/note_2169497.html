Symptom

A disconnection document is created or changed (EC85, EC86).

A disconnection or reconnection order is created and an "Order Code" (ORDERCODE) and "PM plant order" (ORDERWERK) entered.

The values entered is saved only (table EDISCACT) when switch ISU_AMI_EHP5 is active (SFW1, SFW_BROWSER).

Other Terms

EC85, EC86, EDISCACT, ISU_AMI_EHP5

Reason and Prerequisites

Missing functionality.

Solution

Implement the corrections attached to this SAP Note or import the relevant Support Package.
