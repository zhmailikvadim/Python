Symptom

In the Outbound Delivery application in SAP S/4HANA Cloud Public Edition, the Serial Numbers tab is not available at item level during the picking process. Even though the material is configured with an active serial number profile in the product master data, the system does not display the Serial Numbers tab in the outbound delivery document.

Environment

SAP S/4HANAN Cloud Public Edition 

Reproducing the Issue

  1. Maintain a Serial Number Profile in the product master (e.g., DHE15).
  2. Create an Outbound Delivery for the relevant product.
  3. Open the outbound delivery in the corresponding application.
  4. Navigate to the item details.
  5. Execute the picking process for the item.
  6. The Serial Numbers tab is not displayed at item level in the outbound delivery.
     
     
     

Cause

The Serial Numbers tab is not displayed as a separate tab in the outbound delivery item view in SAP S/4HANA Cloud Public Edition.

This is standard system behavior, where serial number maintenance is not exposed as a visible tab but instead is accessible via the application menu.

Resolution

To access and maintain serial numbers in the outbound delivery, please follow the path below:

Menu -> Extras -> Serial Number

This option allows you to view and assign serial numbers for the selected delivery item.
