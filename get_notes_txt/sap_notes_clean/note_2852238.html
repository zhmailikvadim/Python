Symptom

System is displaying the error message "Error while reading the valuation price for material &1"  when you're using a non-stock managed product (material type NLAG) .

 

Environment

SAP S/4HANA Cloud Public Edition Service

Reproducing the Issue

  • Create a new item Service Part in your service order
  • Press enter
  • System will display the above error message

Cause

Design of scenario : "Procurement Integration".

Resolution

Purchase integration works with non-stock material with source of supply and without source of supply.

  • For material with source of supply (e.g. InfoRecord)  the entire process can be automated to auto trigger the PR and create a PO using the job. The inforecord needs to be created with the corresponding condition price.
  • For material without source of supply,  the price condition  PMP0  must be updated for the product. It is mandatory to pass the valuation price for material without source of supply or system will throw an error ""Error while reading the valuation price for material &1" and do not allow release of the item.
