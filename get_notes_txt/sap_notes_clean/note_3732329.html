Symptom

Registration of a custom domain in Custom Domain Manager fails.  
• Error message displayed: Tenant does not exist.  

Image / Data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental  

Environment

  • SAP Business Technology Platform
  • Custom Domain Service

Reproducing the Issue

  1. Open Custom Domain Manager.
  2. Attempt to register a custom domain in another subaccount.
  3. Enter the target subaccount.
  4. Observe the error message: "Tenant does not exist." 

Cause

The target subaccount does not have a subscription to the Custom Domain Manager, which is required to create SaaS routes.

Resolution

The subscription of a Custom Domain Manager in the target subaccount is a required step to create SaaS routes there. 

Subscribing to the CDM in the target subaccount should not result in any cost for you without having certificates in place there. So this should not affect the operations.
