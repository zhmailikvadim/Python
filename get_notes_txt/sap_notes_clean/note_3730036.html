Symptom

You use Smart Chart 2.0 to manage the vital signs of your patients.
If you enter a vital sign in the quick entry function "+Create Measurement" with a date/time that is in the future,
you notice that the system incorrectly adjusts the date/time automatically,
instead of displaying a corresponding error message.

Other Terms

Vital Signs, Vital Signs, Vital Signs, VS, CLIM, CLIO

Reason and Prerequisites

Program error

Solution

The system now correctly displays an error message
if you enter a date/time that is in the future for a vital sign.

Import the specified Support Packages to solve the problem.
For an advance correction, you can implement SAP Note 3729969.
