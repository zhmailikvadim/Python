Symptom

When users pick a subprocess code besides '01' in step 1, step 2 does not show the expected header fields, even after refreshing or navigating.

Other Terms

Financial Control Adjustment Hub, subprocess selection issue, header fields missing.

Reason and Prerequisites

This happens due to setup limits that only enable full fields for the default '01' subprocess; other codes need extra configuration and code adjustment. Requires access to Adjustment Hub app and basic customizing rights.

Solution

1. Implement the sap Note
2. Update setup to include those fields for all subprocesses.
3. Refresh the app cache and test again.
4. Check SAP Notes for quick fixes matching this issue.
