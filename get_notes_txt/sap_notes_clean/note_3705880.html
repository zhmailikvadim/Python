Symptom

Activate scope item XX_28B_OP in T-code /n/SMB/BBI.

Error message /SMB/BBPRJ_INST123 appeared.

Environment

SAP S/4HANA Finance for Group Reporting

Reproducing the Issue

  1. Go to Solution Builder by using T-code /n/SMB/BBI.
  2. Activate scope item XX_28B_OP.
  3. Error message /SMB/BBPRJ_INST123 appeared.

Cause

Didn't activate XX_1SG_OP in advance.

Resolution

If you want to use Group Reporting, you need to activate XX_1SG_OP firstly. In order to make sure the necessary table or other configuration is in your system.

It means that you need to make sure XX_1SG_OP is active before you active XX_28B_OP. 

Keywords
