Symptom

Input field BUSINESSPLACE is missing for the RFA of types FI_AP and FI_AR.

Other Terms

LOD-PER-OP Rel 3.0 SP27

Reason and Prerequisites

 Enhancement to include Businessplace as input field

Solution

In order to get the solution, please perform the manual steps.

After implementing the manual steps, you will find the following fields on the Rules tab of RFA type FI-AP and FI-AR:

RFA FI-AP: accountp[ 1 ]-businessplace

RFA FI-AR: accountr[ 1 ]-businessplace
