Symptom

Cascading effect on input control stops functioning when linked analysis is applied to filter specific charts.

Environment

SAP Analytics Cloud

Reproducing the Issue

  1. Create 2 Input Control A and B.
  2. Create 2 Charts A and B.
  3. Enable Cascading effect on Input Control A and B.
  4. Set Link Analysis for Input Control A to Chart A.
  5. Set Link Analysis for Input Control B to Chart B.
  6. Cascading effect will not take affect.

Resolution

The issue occurs due to the design limitation of SAP Analytics Cloud. When using cascading effects and linked analysis, the widget selected for linked analysis must be the same for the cascading effect to function correctly.
