Symptom

How to export Business Rules?

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

  • SAP SuccessFactors Employee Central
  • SAP SuccessFactors HXM Suite

Resolution

You can export business rules through Import and Export Data:

  1. Navigate to 'Import and Export Data';
  2. Select "Export Data' from the Drop Down Menu;
  3. Select Generic Object "Rule";
  4. Include Dependencies = Yes;
  5. Include immutable IDs = No;
  6. Exclude Reference objects = No;
  7. Select all data records = Yes;
  8. Click 'Export';
  9. Next, click on 'View Result' or navigate to 'Admin Center' > 'Scheduled Job Manager';
  10. Once the Job is has completed, you can click on 'View Details' and 'Download Status' (this will be a Zip File);
  11. Then you can unzip the file, and find the file called 'Rule.csv'.

Result: This will be a CSV file with all the rules in the system. On the "Body" column the rule logic can be found (all conditions) - opening the file in a text editor such as NotePad ++ may be easier to read it.
