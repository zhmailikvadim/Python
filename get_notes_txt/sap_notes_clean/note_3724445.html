Symptom

  • External users bypass the approval workflow during class registration despite approval configurations being set up.
  • Approval workflows function correctly for internal users but are bypassed by external users.

Environment

SAP SuccessFactors Learning

Reproducing the Issue

  1. Configure a separate setup for external site users, including a dedicated site page, library, user accounts, and security domain.
  2. Set up an approval process on the item/class level.
  3. Log in as an external site user and request registration for a class.
  4. Observe that the external site user is automatically registered and added to the participant list without triggering any approval workflow or sending notifications to approvers.

Resolution

  • Please note that the current system design does not support approval workflows for external site users.
  • As a result, external site users will bypass the approval process even if an approval workflow is configured at the class or item level.
  • If this functionality is important for your use case, you may refer to the existing improvement request (Functional Approval Processes for External Users) and vote for it.
