Symptom

I migrated a custom form from the test realm to the production realm, but it is not appearing in the Search menu.

Cause

The migrated form has not been replaced in the production environment. Forms must be replaced after migration in order for them to become visible in the Search menu. 

Resolution

To resolve this issue, perform the following steps in the production realm:

  1. Open the migrated form.
  2. Select Revise.
  3. Select Publish.
  4. Choose Replace to overwrite the existing version.

After completing these steps, the form should be displayed correctly in the Search menu.
