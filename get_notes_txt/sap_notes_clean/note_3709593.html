Symptom

When the Relevance type is Create in a Master Recipe item, the Material, Plant, Production Version, and Profile fields can be edited only after a 20-second delay.

Other Terms

Manage Change Records; F2097;BillOfOperationsMaterial;BillOfOperationsPlant; BillOfOperationsVersion; BillOfOperationsProfile

Reason and Prerequisites

Program error

Notes 3322272 and 3584380 are implemented.

Solution

Use transaction SNOTE to implement the correction instructions. Then empty the browser cache.

You may also need to run the transaction /UI2/INVALIDATE_CACHES to clear the cache for SAP Fiori/UI2 components.
