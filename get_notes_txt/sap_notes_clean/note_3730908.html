Symptom

While finishing a production task, error message appears as,

"Identified stock DEF of label B not correct; identified stock XYZ correct".

(where DEF and XYZ represents Identified Stock IDs, B refers to Label ID)

Environment

SAP Business ByDesign

Reproducing the Issue

  1. Go to Production Control work center.
  2. Go to Task Control view.
  3. Search the Task ID GHI (where GHI represents the Production Task ID). 
  4. Choose Change Status.
  5. Select Finish.
  6. The Error occurs, "Identified stock DEF of label B not correct; identified stock XYZ correct".

Cause

A single line item within a production task is currently associated with two different labels, each labels referencing to two different identified stock IDs.

Resolution

  1. Go to Production Control work center.
  2. Go to Task Control view.
  3. Search and Open the Task ID GHI (where GHI is the Production Task ID). 
  4. Go to the Output Products tab and select the relevant line item.
  5. Open the Assigned Labels sub-tab for that line item.
  6. Verify the Identified Stock assigned to each label.
  7. Remove any label whose Identified Stock does not match the line item’s Identified Stock.
  8. Choose Save.
  9. Choose Change Status and select Finish.
