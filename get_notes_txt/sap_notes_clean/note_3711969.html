Symptom

  • Error message "AW003 - Asset has no settlement to be reversed" encountered when attempting to reverse a reversal posting.

Environment

  • Contract and Lease Management
  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA Finance
  • SAP S/4HANA

Reproducing the Issue

  1. Perform an AUC settlement reversal using transaction AB08;
  2. Attempt to reverse the reversal using transaction AB08 to correct the posting date;
  3. Observe the error message "AW003 - Asset has no settlement to be reversed."

Cause

Standard system behavior, it is not possible to reverse a reversal posting.

Resolution

  1. It is not technically possible to reverse a reversal in SAP.
  2. To correct the issue, perform the AUC settlement again using the correct posting date.
  3. For further guidance on monthly balance realignment, consult a financial consultant or accountant.

Keywords

AW003, AIBU reversal, asset settlement, posting date correction, reversal error, AB08.
