Symptom

  • Technical Support has requested to provide preview setting file
  • User want to preview and save different analytic model definitions, without creating a new analytic model

Environment

  • SAP Datasphere

Resolution

By choosing Export Preview Settings File in "Preview" mode, the definition of the preview of an analytic model will be exported as a local file. Please attach it to the support case that Technical Support can import this file to recreate the data preview for analysis purposes.

See Also
