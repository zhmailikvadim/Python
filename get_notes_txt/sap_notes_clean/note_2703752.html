Symptom

For a terminated time slice, user is still able to terminate it with same validity end date again through meter operator contract UI. The end of billing by meter operator process is triggered again as well during time slice termination. No error is raised for this case.

Other Terms

MOSB, MOS Billing, Terminate Time Slice, Validity End Date, TERMINATE_TIME_SLICE, /MOSB/CL_CONTRACT_ACTIONS

Reason and Prerequisites

Program Error

Solution

Please implement the correction instruction in the SAP Note or import the higher version of Support Package.
