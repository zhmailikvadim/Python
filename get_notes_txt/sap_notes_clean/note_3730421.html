Symptom

Error QV121 - 'Selected set code & does not exist, data entered is incomplete' is returned when choosing the selected set during usage decision - for example in transaction QA11.

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

  • SAP ERP Central Component
  • SAP ERP
  • SAP Enhancement package for SAP ERP
  • SAP S/4HANA, on-premise
  • SAP S/4HANA Cloud Private Edition

Cause

Missing codes for the selected set, that is table QPAC is empty for the affected selected set.

An example from transaction QS52:

Resolution

Maintain code(s) for the affected selected set in customizing. Relevant path:

IMG(transaction QCC0 or SPRO)
>Quality Management
>>Quality Inspection
>>>Inspection Lot Completion
>>>>Edit Code Groups and Codes for Usage Decisions
>>>>Edit Selected Sets for Usage Decisions
