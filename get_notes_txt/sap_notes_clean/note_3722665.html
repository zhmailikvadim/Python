Symptom

Create New and Import Static Group options are Missing under Actions for all Plans > Manage Compensation Groups > Manage Field Permission Group even if permissions are granted.

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Environment

  • SAP SuccessFactors Compensation
  • SAP SuccessFactors Variable Pay

Reproducing the Issue

  1. Navigate to Actions for All Plans
  2. Manage Compensation Group
  3. Manage Field Permission Group
  4. Notice that Create New and Import Static Group options are Missing

Cause

Edit Group access is not granted under Manage RBP and Super Admin Access.

Resolution

Navigate to Manage RBP and Super Admin Access locate the affected user's name, and grant Edit Group access.  

Keywords

Import Static Group, Create New Group, Permission, Manage RBP and Super Admin Access
