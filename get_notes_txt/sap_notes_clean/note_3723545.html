Symptom

When you run the report RPFIAT_AWFUV (Foreign Trade: Cross-Border Claims and Liabilities (Austria)) for general ledger accounts configured with open item management by ledger group, the system only returns line items from the fiscal year derived from the key date. Line items from prior fiscal years that are still open as of the key date are not included in the result.

Other Terms

 RPFIAT_AWFUV, FIAT_AWFUV, SKB1-XLGCLR

Reason and Prerequisites

 This SAP Note is relevant only for Austria.

Solution

After implementing this SAP Note, report RPFIAT_AWFUV correctly retrieves all open line items up to and including the fiscal year of the selected key date for general ledger accounts with ledger-specific open item management. This ensures that open items from prior fiscal years are included in the report output, consistent with the behavior of accounts using classic open item management.

Import the Support Package specified for your release or implement the attached correction instruction.
