Symptom

Work center was renamed in transaction CR02, however the old work center name remains unchanged in table AUFK-VAPLZ.

Environment

  • SAP R/3
  • SAP R/3 Enterprise
  • SAP ERP Central Component
  • SAP ERP
  • SAP Enhancement package for SAP ERP
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA, on-premise
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Open transaction CR02.
  2. Rename the work center using Shift + F7.
  3. Save the changes.
  4. Check table AUFK in SE16N table viewer and observe that field VAPLZ still displays the old work center name.

Cause

standard behavior

Table AUFK contains the Order Master Data and will not update until the relevant Order is also updated.

Resolution

  1. Access the affected work order in transaction IW32 or IW38.
  2. Save the work order to update table AUFK-VAPLZ with the new work center name.
