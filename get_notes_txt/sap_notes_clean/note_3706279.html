Symptom

This SAP Note provides you with corrections to the Peru Inventory and Balance Ledger (PE_INVENTORY_BALANCE_LEDGER) report in the Run Statutory Reports app for SAP Document and Reporting Compliance. It addresses the issue of ITAB_DUPLICATE_KEY runtime error in the electronic accounting book 3.6 Inventory and Balance Ledger (SUNAT code 030600).

When you run the PE_INVENTORY_BALANCE_LEDGER report for the electronic accounting book 3.6 Inventory and Balance Ledger (SUNAT code 030600), the report generation fails.

Other Terms

Peru, Document and Reporting Compliance, DRC, fka ACR, Inventory and Balance Ledger, Libro de Inventarios y Balances, PE_INVENTORY_BALANCE_LEDGER, PCGE, ITAB_DUPLICATE_KEY

Reason and Prerequisites

This SAP Note is relevant only for Peru.

You have implemented previous notes listed in SAP Note 3206609.

Solution

After implementing this SAP Note, the system successfully generates the report output.

SAP recommends that you install a solution by applying a Support Package. However, if you need to install a solution earlier, use the Note Assistant to implement the correction instruction. You can find more information about the SAP Note Assistant in the SAP Support Portal, under SAP Note Assistant.

Documentation for SAP S/4HANA for Peru is available in the system and in the SAP Help Portal. System documentation (such as field help, customizing, messages) is delivered via Support Packages. SAP Help Portal documentation is available at help.sap.com/viewer/s4h_op_peru.

The success message after saving a learning task was changed to "Task saved".

Other Terms

Incident Management, Manage Tasks, Learning task, Manage Incidents app (F4759)

Reason and Prerequisites

Prerequisites: For information about the validity of the corrections, see the correction instructions.

Solution

The support packages that contain this correction are listed in the "Support Packages" section.

Alternatively, you can implement the correction instructions of this SAP Note.
