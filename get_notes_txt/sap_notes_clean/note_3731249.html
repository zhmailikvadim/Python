Symptom

  • You have created an SRS order for estimation catalogue services (demands) that are available only in a specific workshop.
  • In the order details screen, you execute the 'Rescheduling' action.
  • In the action dialog, you observe that all workshops are available for selection in the 'Hierarchy ID' dropdown list.

Other Terms

rescheduling, workshop, estimation catalog, to do basket, tdo, srs, vss

Solution

  • Implement the attached note.
  • Refresh the metadata of /DBME/WMA_MAIN OData V2 service. Please follow the instructions described in note 3453755.
  • Implement the note 3724349 on the frontend server.
