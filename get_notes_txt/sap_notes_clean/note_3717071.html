Symptom

It is required support to run a full statistics gather on a tenant.

Environment

  • SAP Sales Performance Management
  • Incentive Management on Legacy Oracle

Resolution

  1. Create a support ticket with a Medium priority.
  2. Specify in the title that you wish for stats to be gathered and where you wish them to be gathered from. Example: "Run full stats gather on Production TCMP and EXT schema"
  3. In the description specify the following:
    1. Which schemas you want to have stats gathered. Ex: TCMP, EXT
    2. The level you want to have stats gathered. Ex: Table, Index
    3. The tenant and environment. Ex: XXXX - PRD
    4. What day and time would be best for this process to be ran, including timezone. Ex: 01/01/2024 12:00 PM EST

Please note: as a customer, you can set up custom processes in your environment so that you can perform a STATS gather for EXT schema. You can follow the documentation and KBAs below for more details:

  • KBA 3587513 - Gathering Table Statistics (Manual - Oracle) 
  • KBA 3222825 - How to explicitly gather stats on EXT (custom) tables
