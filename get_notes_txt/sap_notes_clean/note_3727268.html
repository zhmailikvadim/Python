Symptom

  • The Goods Movement Bin assigned to the Outbound Delivery is different from the Storage Bin configured in transaction /SCWM/PSASTAGE.

Environment

  • SAP Extended Warehouse Management (SCM-EWM);
  • SAP Extended Warehouse Management Delivery Processing (SCM-EWM-DLP);
  • SAP S/4HANA;

Reproducing the Issue

  1. Create a consumption delivery;
  2. Verify that the Goods Movement Bin assigned is different from the one maintained in transaction /SCWM/PSASTAGE.

Cause

  • The system takes the correct goods movement bin maintained in transaction /SCWM/PSASTAGE.
  • However, the ERP PSA/staging bin assignment is also maintained, which overwrites the correct goods movement bin with the ERP bin, resulting in the incorrect bin determination.

Resolution

  1. Access Table PKHD;
  2. Check the entries for the affected Plant and Supply Area;
  3. Verify the field "Destination Bin" and ensure it is correctly configured to prevent overwriting the data maintained in transaction /SCWM/PSASTAGE.
