Symptom

When using transaction "FINS_CFIN_SLT_TOUCH" there is error message:

No logging table defined in SLT
Message No.: FINS_CFIN_SLT_RO012

Environment

  • Financial Accounting (FI)
  • Central Finance (CF)
  • SAP S/4HANA Finance
  • SAP S/4HANA

Reproducing the Issue

  1. At Target/CFIN system, execute transaction `FINS_CFIN_SLT_TOUCH`.
  2. Provide the document data/parameters for it to be replicated again.
  3. Observe the error message "No logging table defined in SLT" (Message No. FINS_CFIN_SLT_RO012).

Cause

Missing configuration for the prerequisites of transaction "FINS_CFIN_SLT_TOUCH".

Resolution

Make sure that the prerequisite configurations for "FINS_CFIN_SLT_TOUCH" are defined, which are:

"Prerequisites

  • The user has authorization to run transaction FINS_CFIN_SLT_TOUCH in the Central Finance system.
  • The RFC usage SLT Logging with RFC destination has been added in Customizing for Central Finance under:
    Central Finance: Target System Settings-> Set Up Systems-> Maintain RFC Assignments and Settings for Source Systems.
  • The user has read access authorization for the relevant company codes in the source system.
  • The RFC user has been assigned to the role SAP_IUUC_REPL_REMOTE in the source system."
