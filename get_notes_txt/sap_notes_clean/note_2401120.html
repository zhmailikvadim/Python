Symptom

You have configured a sender IDoc_AAE communication channel with enabled acknowledgment settings (tab "Ack Settings"). When you execute the channel ping operation for this channel the ping fails with the following error:

"ALEAUD connection factory null is not available"

There is no error during the runtime processing of the channel.

Other Terms

channel ping, connection factory, acknowledgments

Reason and Prerequisites

Program error.

Solution

To correct this issue, apply the patch matching your support package version as listed in the patch table below. Follow the instructions described in SAP Note 952402.
