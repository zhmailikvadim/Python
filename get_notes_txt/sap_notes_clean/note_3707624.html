Symptom

Performance issues may occur during actual costing status determination when calling function modules FCML4H_STATUS_GET, FCML4H_STATUS_SET_UPDATE, and FCML4H_STATUS_CHECK_CLOSE.
This SAP Note corrects the issue and improves performance.

Other Terms

FCML4H_STATUS_GET, FCML4H_STATUS_SET_UPDATE, FCML4H_STATUS_CHECK_CLOSE

Reason and Prerequisites

Performance improvement

Solution

Implement attached correction instructions. Alternatively, install appropriate Support Package mentioned in this SAP Note.
