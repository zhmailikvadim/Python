Symptom

You have an OTC interest rate swap transaction having Effective From Date (calculation date) of interest flow prior to the start of the term. In the change mode if we try to edit or add more flows in the "Create Nominal change" and click the check button. System raises error message FTVV_Messages(915) "Calculation date before period start &".

This issue occurs only if in the transaction the indicator for End of Term Inclusive is set.

Other Terms

OTC swap, FTVV_MESSAGES(915), check_effective_from

Reason and Prerequisites

Program error

Solution

Implement the correction instructions or import the corresponding Support Package.
