Symptom

In Fiori app Clear G/L Accounts - Manual Clearing (App ID F1579), error message F5A397 occurs when the clearing is simulated.

Environment

  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition
  • SAP S/4HANA Finance
  • SAP Fiori

Reproducing the Issue

  1. Logon to Fiori launchpad.
  2. Open Fiori app Clear G/L Accounts - Manual Clearing.
  3. Enter filter selection parameters and click 'Go'.
  4. Select line items to clear.
  5. Click 'Simulate'.
  6. Error F5A397 occurs.

Cause

Program error.

Resolution

Implement SAP Note 3666529 - Post General Journal Entries: Error F5A 397 when simulating Journal Entry
