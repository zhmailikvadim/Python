Symptom

This KBA covers user based permission required to be granted to Employees who should be able to access AskHR application.

Note: this KBA only covers permission required for employees. For Administrator permission, refer to ECSC help document.

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

  • SAP SuccessFactors Employee Central Services Center
    • SAP SuccessFactors ECSC

Resolution

Below is the list of permissions which must be granted to all the employees to successfully access AskHR application:

  1. Employees should have an access to Employment Information.
  2. Employees should have an access to Job Information.
  3. Make sure "Company" field under Job Information has "View Current" permission granted. Also, "Job Information Action" in "View Current"  mode is must:
  4. Employees must be granted MDF Object Legal Entity permission in view mode to successfully launch all the catalog.
  5. Go to RBP of affected employees and search for "MDF Foundation Objects"> Legal Entity. View Current for employees.
  6. Employees must be granted access to Metadata Framework > Admin access to MDF OData API, as this allows employees to access the Ask HR mobile application. 
  7. Employees must have Employee Data > Employment Details MSS. Only View Current level, as this allows employees to view Ask HR tickets. 
  8. Employees must have Miscellaneous Permissions > Service Center Contact Information. View: Only for employees. Edit and Import/Export: Only for administrators. Allows employees to access Contact HR information, if configured in Ask HR.
  9. You can restrict the granted permission for user itself so that same data cannot be viewed for other employees:


NOTE: We have combined permissions required for AskHR together with other permissions suggested for you in a single Guide: List of Role-Based Permissions | SAP Help Porta

You may search for "Ask HR" and "Service Center" to find all Admin and User permissions required.
