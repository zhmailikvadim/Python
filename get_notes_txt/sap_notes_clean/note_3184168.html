Symptom

I need to update my Catalog Hierarchy.

Environment

SAP Ariba Catalog Management

Resolution

  1. Complete the Import System Commodity codes task. Ensure the commodity code file does not have any duplicate entries.
  2. Navigate to Catalog manager > Scheduled tasks.
  3. Look up the task named Generate Catalog Hierarchy from Commodity Codes and click Run.
  4. To implement the new version of the hierarchy for the catalog, click File > Open > Draft Version.
  5. To save the Draft Version as the new system version, click File > Save > System Version.
  6. Run the Index builder to apply the changes by navigating to Catalog manager > Index builder and clicking Rebuild. It is recommended to run this task outside of business hours.
