Symptom

• Users encounter the error APOC_OR_MESSAGES145 when attempting to print labels using output type "GOODS_RECEIPT_LABEL" with the standard template RM07ETIKETT of type 2.
• The standard program SAPM07DR and routine ENTRY_ETIE_PDF are utilized during this process.
• The error occurs when sending the output item to the printer.

Environment

  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Cause

• The standard NAST output control report "SAPM07DR" sub-form "ENTRY_ETIE_PDF" expects the NAST interface field "TNAPR-FONAM" to be set with the name of a SAPScript form.
• The S/4HANA output control framework does not support this field when populating values in TNAPR during rendering form template type 2. 
• The framework populates TNAPR attributes based on the type of form being processed:
a) Adobe PDF form (type 2): TNAPR-SFORM = .
b) SMARTFORM (type 3): TNAPR-SFORM = .
c) SAPScript (type 4): TNAPR-FONAM = .
• For Adobe PDF forms(form template type 2), the interface sets "TNAPR-FONAM" to "INITIAL". 
• The report "SAPM07DR" attempts to retrieve the SAPScript form name from the table "T159O Inv. Mngmt Print Function: Maintain Text Name for Labels".
• As "INITIAL" is set, the SAPScript form cannot be found, preventing the Adobe PDF form rendering framework from capturing the document in a spool request.

Resolution

1. Note that the standard report "SAPM07DR" has not been adapted to support the S/4HANA output control interface for issuing output using NAST output control.
2. Understand that the standard NAST output control form/report cannot be used with the S/4HANA output control framework.
3. Refer to SAP Note 2461075 for customizing output management in S/4HANA Cloud 1708 and On-Premise 1709 for goods movement transactions.
4. Option A: Customize printing labels using S/4HANA output control as the default document output process with forms using gateway service rendering technologies.
5. Option B: Customize NAST output control as the default document output process and utilize the standard Adobe PDF form "RM07ETIKETT" with the NAST output control report "SAPM07DR" and sub-form "ENTRY_ETIE_PDF".
