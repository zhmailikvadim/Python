Symptom

You send a simplified export declaration to the Belgian export system AES and receive an error message from the customs system with reference to rule C0592.

This indicates that the CurrencyExchange data group must not be submitted if no statistical value is submitted. Rule C0028 generally does not transfer a statistical value in simplified export declarations.

However, the system has created and submitted the CurrencyExchange data group.

Other Terms

 Belgium, export, C0592, C0028

Reason and Prerequisites

The error occurred after the end of the transition phase on December 14, 2025 because rule C0028 became active only at this time.

Solution

Implement the attached correction instructions or import the relevant support package.
