Symptom

You use bapi_ib_version_create to create an installment bond version with a variable interest rate condition. The field to set the interest rate of the first period is missing from bapi_ftbas_ib_condition.

Other Terms

interest_rate_first_period, tfm_pkond1stper, pkond1stper, percentage_rate_1st_period, percentage_rate_first_period

Reason and Prerequisites

 Program error.

Solution

 Import the corresponding support package. Alternatively, you can apply the attached correction instructions.
