Symptom

You have received a notification email advising of the switch in API Management Design-time Root Certificate from DigiCert Global Root G2 to DigiCert Global Root G3.

Environment

API Management (NEO)

Cause

Cause:
As part of our ongoing commitment to security and trust, SAP is updating the Root Certificate Authority (CA) used for certificates in Design-time environments from DigiCert Global Root G2 to DigiCert Global Root G3. For more information, please refer to the following SAP Note: 3666687.

Impact:
Client systems accessing API Portal or Developer Hub URLs that do not trust DigiCert Global Root G3 may encounter SSL handshake or certificate validation errors. As a result, users may be unable to view, create, or manage APIs, products, applications, or subscriptions.

Resolution

Action Required:
Customers accessing Design-time endpoints from any client systems must:

  • Check trust: Ensure your client system trusts the Root CA: “DigiCert Global Root G3”.
  • Update if necessary: If this certificate is not present, you must download and add it to your client system’s trust store before the migration date to avoid service disruptions.

FAQ:

1. Are web browsers affected?
Web browsers connecting to the SAP API Management Web UI (API Portal and Developer Hub) will generally remain unaffected, as the new Root CA is included in the latest versions of all major browsers and up-to-date operating systems. However, if your organization manages certificates manually, utilizes legacy browser versions, or maintains its own restricted trust store, you must manually add the new DigiCert Global Root G3 certificate to ensure continued access. 

2. Do I need to trust Intermediate or Leaf certificates?
No. SAP recommends trusting only the Root CA.

3. Does this affect Runtime API calls?
Runtime endpoints are migrated to a different root certificate, "DigiCert TLS RSA4096 Root G5". For more information, please refer to KBA 3705360.

4. Is there an impact on the Cloud Connector scenario? 
Impact is not anticipated. The DigiCert Global Root G3 root certificate (used for SAP BTP Neo endpoints for Cloud Connector connectivity) is widely included in standard JVM distributions. As long as the Cloud Connector is running on a supported JVM version, the certificate should already be present. For legacy or highly restricted environments, we recommend a manual verification of the trust store.
