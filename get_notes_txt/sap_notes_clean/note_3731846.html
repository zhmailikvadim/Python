Symptom

 You wonder for how long you can expect support for your applications built with the SAP NetWeaver RFC SDK (NW RFC SDK).

Other Terms

End-of-life, NW RFC library, sapnwrfc.dll, libsapnwrfc.so, lifecycle

Reason and Prerequisites

You found the end-of-maintenance information for a concrete NW RFC SDK version and ask yourself whether this means that your investments in enhancements built with the SDK are endangered.

Your application is built for an integration installed in an on-premises installation or in your virtual on-premises network in a hyperscaler environment.

Solution

SAP NetWeaver RFC SDK is made available to customers and partners according to the license agreement for SAP Connectors. SAP plans to continue supporting SAP NetWeaver RFC SDK in the future. If this decision changes, it will be communicated in advance through this SAP Note.

For details of the support strategy also with regard to supported operating systems for a concrete NW RFC SDK version, check the respective referenced SAP Notes.
