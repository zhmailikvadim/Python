Symptom

After switching to Full Profile mode, certain fields in employee profiles disappear a few seconds after being accessed through “View All.”

Environment

SAP SuccessFactors HCM Suite

Reproducing the Issue

1. Switch to Full Profile mode in the latest People Profile.
2. Open an employee profile and click “View All” on the relevant card.
3. Observe that all fields load initially, but some fields disappear after a few seconds.

Cause

This is a Known Issue.

Resolution

The fix for this issue is planned to be deployed on 1H 2026 release.

For release timelines, please review SAP SuccessFactors Product Release & Road Map Information.
