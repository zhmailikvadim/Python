Symptom

The user finalizes a SIMA activity. The system terminates the finalization with a timeout after a longer wait time.

Other Terms

HRP1000

/MSG/82_RL_R_BASIC_DATA

Reason and Prerequisites

Prerequisites:

  • The SIMA activity has many entities.
  • The table HRP1000 (organizational units) has a large number of entries.

Reason:  The system reads descriptions for each organizational unit determined.

Solution

History:
March 4, 2026: New

Troubleshooting description:
The system reads the organizational unit names valid on the current date only once when selecting the organizational units. The user can finalize the SIMA activity.

The changes contained in this SAP Note also have a positive effect on the runtimes when you use Web Dynpro loss management.
