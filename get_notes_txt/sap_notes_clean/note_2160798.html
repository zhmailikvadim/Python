Symptom

  • The application does not forces user to create an application password when using "Scan QR code" to add a connection in a newly installed SAP Mobile BI App
  • Blank view shows (App looks like hang) after click All reports to show all report the second time or later logs on the App connection

Other Terms

freeze, no response, BI4

Reason and Prerequisites

Environment:

  • SAP BusinessObjects Business Intelligence platform 4.1 (BI4.1)
  • SAP BusinessObjects Mobile for iOS 6.1.18
  • iOS 8.3

Reproduce Steps:

  1. Download and install a new SAP BI mobile app.
  2. Start it and create a new connection by scan QR code.
    =>Note that the application does not forces user to create an application password.
  3. Reports can be viewed successfully.
  4. Close the APP.
  5. Start the APP again and connect the same connection.
  6. Choose All Reports and try to refresh after connecting successfully.
    =>Note that the APP keeps loading and reports does not showed after a long time.

Cuase:

This is a product defect.

Solution

The issue was fixed in 6.2.10 patch.
