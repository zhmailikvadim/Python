Symptom

Users are not able to view Cancellation Policy of class before registration, unless there is a Special Request added to it

Environment

SAP SuccessFactors Learning

Reproducing the Issue

  1. Search for classes in Enhanced Search>
  2. Access Class with cancellation policy and without Special Requests >
  3. Click to "register" >
  4. User can register without checking the cancellation policy 

BUT

  1. Search for classes in Enhanced Search>
  2. If the Class has a Special Request and Cancellation Policy >
  3. Click on "Register >
  4. User is redirected to a page where they can view the Cancellation Policy without registering

Resolution

This scenario has been addressed by Engineering as a Product Enhancement and will be prioritized on Later Releases.
