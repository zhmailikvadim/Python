Symptom

Symptom

The system issues error message RW 033 ("Balance in Transaction Currency") during an asset retirement with special retirement type 1 or 4.

Other Terms

ABGTYP, RETIREMENT_TYPE, ABAVN, AB01L, writeup, RW033, special retirement

Reason and Prerequisites

Program error

Solution

Program corrections

Other Terms

Business Event, Counter, Payment Item, Bank Payment Transaction Counter, TRBK

Reason and Prerequisites

Special Development

Solution

It is recommended to import the relevant Support Package for your release. You can also use transaction SNOTE to implement the correction instructions.

1) Please complete the manual activity described in the note to enable the Bank Payment Transaction Counter business event.

2) The business event can be extended by enhancing the model of the abstract entity FS_AE_COUNTER_CT_CHNG with custom fields. Enhancement spot ES_FSTB_CNTR_EVENT is provided to enrich these custom fields.

3) Please use the customizing "img/Transactional Banking->Tools->Business Events->Assign Product and Transaction Type to BankAccountEntry" to set the filter criteria for Bank Payment Transaction Counter business event.
