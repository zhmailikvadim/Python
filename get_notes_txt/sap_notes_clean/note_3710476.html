Symptom

In Portfolio Management, when a user or user role is newly added or changed in the Authorizations tab, the data is not updated in the tables that store information for simplified authorization. This applies to creating or updating bucket, item, and initiative data for the portfolio.

Due to this missing information, users who have sufficient authorization to read the portfolio objects may not be able to access the data using the portfolio management APIs.

Other Terms

Portfolio Management, Simplified Authorization, API_PROJECTPORTFOLIO, SAP S/4HANA 2025, SAP S/4HANA Cloud Private Edition 2025

Reason and Prerequisites

 Program Error
