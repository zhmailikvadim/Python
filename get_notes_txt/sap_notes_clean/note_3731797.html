Symptom

Getting the below errors while setting the configuration parameters :

  • Error: (dberror) [7]: feature not supported: change not allowed for tenant database (filename=global.ini, section=persistence, key=savepoint_max_pre_critical_flush_duration).
  • Error: (dberror) [7]: feature not supported: change not allowed for tenant database (filename=indexserver.ini, section=transaction, key=idle_cursor_alert_timeout).
  • Error: (dberror) [7]: feature not supported: change not allowed for tenant database (filename=indexserver.ini, section=transaction, key=uncommitted_write_transaction_lifetime).

Environment

  • HANA Cloud Services HANA Cloud

Reproducing the Issue

  1. Attempt to configure the parameters in hana cloud using ALTER SYSTEM ALTER CONFIGURATION commands.
  2. Observe errors indicating certain parameters are flagged as invalid with dberror [7] “feature not supported: change not allowed for tenant database”.

Cause

The targeted parameters are not available/supported in hana cloud and changes of these parameters are not allowed at the tenant database level.

Resolution

  1. Refer to the hana cloud configuration parameter reference to verify which parameters are supported: Configuration Parameters for SAP HANA Cloud.
  2. Only set parameters that are listed as supported for hana cloud.
