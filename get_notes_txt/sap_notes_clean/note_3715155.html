Symptom

You use account clearing. The generated clearing flow has the same payment date as the cleared flow.

This can lead to undesired results, for example, during the calculation of interest on arrears.

This SAP Note provides a new parameter with which the posting date can be specified as the payment date.

Other Terms

FZKL, DDISPO, IOA

Reason and Prerequisites

Previously, this function was not provided.
