Symptom

NCo3 : Getting exception while opening Hierarchy Variables

Other Terms

Program error

Solution

  • SAP NetWeaver 7.0 BW Front End for GUI 740

    Import Front-End Patch (FEP) 800 (or higher) for SAP NetWeaver 7.x BW Front End (bi740sp8_800-10004472.exe) into your system. The FEP will be available as soon as SAP Note 2216329 with the short text "SAPBWNews NW 7.x BW Add-On Frontend Patch 800 - GUI 7.40", which describes this FEP in more detail, is released for customers.

    This SAP Note might already be available before the FEP is released.
    You can check the planned availability dates in the attached SAP Note 1085218.
