Symptom

In the DRC Cloud Edition "Resolve Errors" app, the following message is shown for the affected eDocument ID:

"Error occurred while the PUSH step, detailed information: Check the destination configurations for destination '<destination_name>'. Response returned HTTP status code: 404. Check the destination configurations maintained in your BTP subaccount. If the problem still persists, create an incident in the LOD-LH-DRC-FND application component."

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

  • SAP ERP
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition
  • SAP BTP
  • SAP Document and Reporting Compliance, Cloud Edition

Reproducing the Issue

  1. Launch transaction edoc_cockpit
  2. Submit an eDocument
  3. Check the logs in DRC Cloud Edition - Resolve Error application

Cause

  1. Electronic Communication Service GCO_INTERNAL is inactive
  2. BTP Destination URL is not configured correctly

Resolution

  1. Electronic Communication Service GCO_INTERNAL should be activate: 
    • SAP S/4HANA 2022 or higher version:  Activating HTTP Service for SAP S/4HANA 2022 or Higher or for SAP S/4HANA Cloud Private Edition | SAP Help Portal 
    • SAP S/4HANA 2021 or lower versions:  Activating HTTP Service for SAP S/4HANA 2021 or Lower | SAP Help Portal 
    • SAP ERP: Activating HTTP Service | SAP Help Portal
  2. Check the URL configured in BTP destination configuration:
    • Make sure you added a slash (/) at the end of of the URL in destination.
    • Review the destination configuration, as follows:
      • SAP S/4HANA: Check user authorization described in the Setting Up Connectivity.
      • SAP ERP: Check user authorization described in the Setting Up Connectivity.
