Symptom

A dictionary entry is automatically deleted after you merge it with another dictionary entry.

"Image/data in this KBA is from SAP internal systems, sample data, or 
demo systems. Any resemblance to real data is purely coincidental."

Environment

  • Product: SAP Signavio Process Manager
  • Dictionary for SAP Signavio Process Manager

Cause

Resolution

The deletion of one of the merging entries is expected behaviour.

When you merge 2 dictionary items, you can select which entry retains as the resulting entry. The other one will be deleted.
