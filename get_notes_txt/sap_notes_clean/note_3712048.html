Symptom

  • Errors occur with the commentary service in Analysis for Office.
  • Error message: "Comments cannot be synchronized with BI platform. Reason: Encode failure. The database for comments did not start correctly."
  • Comments working for Web Intelligence reports

Environment

  • SAP BusinessObjects Business Intelligence Platform 4.x
  • SAP BusinessObjects Business Intelligence Platform 2025
  • Analysis for Office
  • Storing comment in Audit Database SQL Server

Reproducing the Issue

  1. Open CMC> Applications> Commentary Service
  2. Select the option Use Audit database
  3. Open an Analysis for Office report from the BO portal.
  4. Observe the error message: "Comments cannot be synchronized with BI platform. Reason: Encode failure. The database for comments did not start correctly."
  5. Try to add and save comments
  6. See comments not being save

Cause

Driver lost during upgrade. 

Resolution

  1. Download the correct driver as per KBA: 2313289 - How to configure BI commentary to use a new database in Business Intelligence Platform ? - SAP for Me
  2. Paste the driver in <installdir>\SAP BusinessObjects Enterprise XI 4.0\java\pjs\services\BICommentaryService\lib.
  3. Restart APS with Commentary Service
  4. See the comments working in Analysis for Office
