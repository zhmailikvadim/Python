Symptom

This SAP Note contains a report to automate the creation/modification of objects that are used in the solution delivered via SAP Note 3707421.

Other Terms

SAP Note 3707421;

Reason and Prerequisites

This SAP Note delivers a tool report that creates or changes objects that are prerequisite for SAP Note 3707421. The implementation of this SAP Note has no impact on your existing functionality.

Good to know: when you install an SAP Note, some prerequisites might be required. The prerequisites are SAP Notes that were already released and that change the same context of the current SAP Note. So, even if these prerequisites are not necessarily related to the symptom of the current SAP Note, you should implement them due to technical reasons.

IMPORTANT

  • This report serves only as a tool to generate the required objects automatically. Therefore, it has no positive or negative effect to transport the report NOTE_3707420 to your production system.
  • The objects created by the execution of this report can only be used after you apply the main SAP Note that delivers the solution.
