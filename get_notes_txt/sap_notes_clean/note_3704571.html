Symptom

  • SAP Screen Personas Add-on was successfully uninstalled using SAINT transaction 
  • SAP Screen Personas entry is still visible in the Installed Product Version tab.

Environment

SAP Screen Personas

Reproducing the Issue

  1. Uninstall Screen Personas Add-on using SAINT transaction.
  2. Check the 'Installed Product' Version tab in the System Status.

Cause

The reason for the difference is that the Personas uninstallation was performed via transaction SAINT, instead of using SUM / Maintenance Planner.

The discrepancy cannot be avoided because SAINT and SPAM don't maintain the "Installed Product Versions" tab. 

Resolution

  1. To resolve the issue, follow the procedure described in the BC-UPG-ADDON Knowledge Base Article ##2452670 - Product versions are not shown as expected in System -> Status 
  2. See also the BC-UPG-TLS-TLA note ##2560351 - Installed Software Component Versions don't align with Installed Product Versions in system status
