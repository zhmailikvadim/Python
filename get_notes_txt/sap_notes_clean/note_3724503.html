Symptom

Unexpected activity unit is used in production order confirmation. Need to know system behavior of how activity unit is decided during confirm calculation. 

Environment

SAP S/4HANA Cloud Public Edition

Cause

The calculation for production confirmation is in accordance with the "Base Unit" maintained in manage cost rates. 
For example, if activity unit H is maintained for the relevant cost center/activity type in the manage cost rates, 'Hour' will be the base unit for calculation in work center. 

Resolution

Maintain the desired activity unit for the relevant cost center/activity type in the manage cost rates.
