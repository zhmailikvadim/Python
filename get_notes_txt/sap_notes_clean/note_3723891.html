Symptom

  • Wage type /649 not generated when there is an offcycle before the regular period
  • Gratuity provision calculation issue

Environment

  • SAP ERP Human Capital Management
  • SAP Payroll Control Center
  • SAP enhancement package for SAP ERP

Reproducing the Issue

  1. Run payroll via PC00_M24_CALC
  2. Display Log
  3. Check wage type /649 not generated

Cause

Wage Type /651 Gratuity Provision C/over

Processing Class  6  Enter wage type from old payroll account in LRT -->CYCL.X006   is set to 2 Wage type is taken from same year

Resolution

  1. Go to table V_512W_D
  2. Wage Type /651 Gratuity Provision C/over
  3. Processing Class  6  Enter wage type from old payroll account should be set to 1
