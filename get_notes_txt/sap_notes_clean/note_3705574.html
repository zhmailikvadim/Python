Symptom

You are using the Preview Events Fiori app in SAP Customer Activity Repository applications bundle 6.0 SPS01 through SPS02.

You notice that the offer content is not displayed when previewing events if the attribute type is Free Text and the content length exceeds 1333 characters.

Other Terms

Preview Events, EVENTPRV, Fiori App, Event, Attribute Value, CDS Views /PRM/EVENTPRV_OFFER_CONTENT

Reason and Prerequisites

Program Error

Prerequisite:

Ensure that backend SAP Note 3679622 – Preview Events Fiori App does not display Offer Content has been implemented.

Solution

Implement the attached correction instructions in the front end system.
Additionally, ensure that SAP Note 3679622 is implemented in the back end.
