Symptom

Customer would like more information about how the Account Merge will work in SAP Sales Cloud Version 2.

Environment

  • SAP Sales Cloud Version 2
  • SAP Service Cloud Version 2 

Resolution

In the initial release, it will not be possible to initiate a merge directly from the Account worklist. A new Account Merge tile will be available under Settings, where an administrator can select the relevant records and designate one as the surviving record.

The system will automatically determine which data is retained or copied during the merge process.

In future releases, users will have more flexibility to choose which specific data (for example, phone numbers) should be preserved, whether from the surviving record or from one of the identified duplicates.

Once development is finalized and the feature is officially released, the complete details and documentation will be made available.
