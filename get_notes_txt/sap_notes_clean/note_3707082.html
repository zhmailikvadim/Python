Symptom

You maintain breakdown categories (in transaction CX1I4). Upon saving there is the error popup G0 115 ("Items in category &2 need a required breakdown for characteristic &1"). The popup refers to breakdown categories for statistical equity holdings (EHA) adjustments (technical ID §EH*), which are automatically generated by the system

Other Terms

G0115, statistical equity holdings adjustments

Reason and Prerequisites

The system had generated the statistical EHA items with optional breakdown by subitem category and subitem. This is allowed, because it is needed for usage of multiple subitems per EHA scope. Therefore, the error popup G0 115 is not justified.

Solution

 Import of Support Package or advanced correction by implementation of correction instruction
