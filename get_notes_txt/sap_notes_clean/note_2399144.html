Symptom

In system monitoring application when you click on the Metric Documentation link it gives the following error:

Environment

SAP Solution Manager 7.2

Reproducing the Issue

  1. Execute the transaction SM_WORKCENTER
  2. Select System and Application workcenter --> System Monitoring
  3. Select the required system SID and select System Monitoring



  4. In the system hierarchy details, double click on a metric to display the metric details info screen



  5. Select Metric Documentation and error message is shown

Cause

The service "/default_host/sap/bc/webdynpro/sap/documentation" was not automatically activated

Resolution

  1. Execute transaction SICF
  2. In field "Service Name" enter DOCUMENTATION as shown below and Execute:



  3. In results screen right click the documentation node and choose Activate Service:

Keywords

SOLMAN, Metric Documentation, Service, SICF, Missing, Service cannot be reached, 403, error
