Symptom

How to check if auto merge is disabled for table. 

Environment

  • SAP HANA Database

Reproducing the Issue

  1. Access the SAP HANA Database.
  2. Run the SQL statement: `select table_name, auto_merge_on from TABLES where table_name = 'TABLE_NAME';`. 
  3. Check the value in the AUTO_MERGE_ON column for the specified table. 

Resolution

  • Auto merge can be activated or deactivated for an individual table in the system view TABLES (SYS). Update the value in the AUTO_MERGE_ON column to TRUE or FALSE as needed.
  • Use the SQL statement: `alter table disable automerge`.
  • To verify the auto merge status, run the SQL statement: `select table_name, auto_merge_on from TABLES where table_name = 'TABLE_NAME';`.
