Symptom

Encounter error "No authorization to access service group 'API_PROJECTBILLINGREQUEST'" when trying to call the project billing request API.

Environment

SAP S/4HANA Cloud Public Edition

Cause

Missing authorization for the communication user to access the service group "API_PROJECTBILLINGREQUEST".

Resolution

Ensure the communication arrangement used for communication scenario Project Billing Integration (SAP_COM_0909) is correctly assigned to the communication user configured for inbound communication.
