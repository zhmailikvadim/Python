Symptom

You are replicating an employee located in Slovakia from SuccessFactors to Employee Central Payroll and the following error is shown in SLG1:

  • Wrong work municipality number!
    Message No. 3A202

Environment

  • SAP SuccessFactors Employee Central Payroll
  • Employee Master Data replication
  • Point To Point (PTP)

Cause

The Address Information is getting replicated with a State ID that is not configured in table T005 and/or the combination of System Language, Country and State are not set in T005U. This check is made in class CL_HRPA_INFOTYPE_0006_SK, method CHCEK_STATE.

Resolution

  1. Change, in SuccessFactors, the value for field State from these addresses to one that is valid in T005U;
       
    OR
       
  2. Create an entry in T005U to make this combination available;
       
    OR
       
  3. Create a Code Value Mapping in T77SFEC_CVMAPC to convert the replicated State ID into a valid one in T005U.

Keywords

Wrong work municipality number, 3A202, Slovakia, PTP, Point To Point, Employee Replication, SY-LANGU, LAND1, BLAND, SPRAS, T005, T005U.
