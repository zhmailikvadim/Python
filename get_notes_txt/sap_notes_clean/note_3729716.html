Symptom

  • Unable to create product with the material type ERSA (Spare Parts) in the App "Create Material" or "Manage Product Master Data".
  • The standard material type ERSA (Spare Parts) is missing in the SSCUI 103539 'Define Attributes of Material Types.

Environment

SAP S/4HANA Cloud Public Edition

Cause

Material Type ERSA is delivered as part of Building Block BLK (XX). This building block is used across 30 scope items as listed below:

Resolution

  1. Review the system to check if any of the listed scope items in Cause section are currently activated.
  2. If none of the scope items are activated, activate any of the listed scope items to enable the material type ERSA.
