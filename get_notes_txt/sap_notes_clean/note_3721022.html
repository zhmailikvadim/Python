Symptom

How to build a Story report to fetch users' responses to surveys?

Resolution

In order to see the users' answers (similarly as can be seen in individual survey response reports within the Learning module), a Story can be build with the following relationship:

  • User > Assigned Survey Details > Survey Response > Question > Answer Choices, from where you can select Question Text, Rating Scale, Answer ID, and Answer Choice Labels to be able to identify the "User Response Value" from table Survey Responses belonged to which answer option.

IMPORTANT:

Scenarios where the "User Response Text" column shows only 'Open Ended' responses is the application's expected behavior. In order to fetch 'Rating Scale' responses, it's necessary to use the "User Response Value" column.

Please note that the "User Response Value" column will show the Rating number of the answer submitted by the user, not its Description. Just for illustration purposes, here's an example:

  • Let's consider a dummy Rating scale named ABC, which has 5 ratings >

  • Rating Scales are located at Learning Administration > References > System Administration > Rating Scales. Considering that the rating scale ABC has the ratings 1, 2, 3, and 4, and each rating has a description:
    • Rating: 1 - Description: One
    • Rating: 2 - Description: Two
    • Rating: 3 - Description: Three
    • Rating: 4 - Description: Four

  • Then the Rating scale ABC gets associated to a survey and the sample user answers "One" (which stands for the rating number 1) >

  • At the respective Story Report, pulling the "User Response Value" column, the data shown will be 1.

Please note that currently it's not possible to pull the descriptions of the Rating scale answers, only its numbers through the "User Response Value" column. To suggest a different system behavior, it would be possible to have an enhancement request created for it (as per KBA 2090228), where the Learning Product Management team can further review and assess the requested functionality to be considered in a future release.
