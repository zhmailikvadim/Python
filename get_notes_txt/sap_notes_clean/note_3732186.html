Symptom

In My Timesheet (S/4HANA/Version 4) application, the work schedule information like start/end time, breaks from the S/4 HCM system was not displayed to the user.

Other Terms

F3074A, My Timesheet V4, Work schedule, Start time, End time, Break, HCM.

Reason and Prerequisites

This is implemented based on a feature request.

To check the work schedule, click on the Recorded / Target link for the respective day in the application.

Prerequisite: The Note: 3725658 needs to be implemented in the corresponding Gateway/Frontend system.

Solution

Please implement the automatic correction instruction attached to this SAP Note or alternatively you can implement the corresponding support pack.
