Symptom

When Submitting Greece eBooks to MYDATA system throws following error:
Error when calling &EDOC_AIF_INIT_REQUEST_MAPPING& 


Image/data in this KBA is from SAP internal systems, sample data, or demo systems and are for reference purposes only.  


Environment

  • SAP ERP
  • SAP S/4HANA
  • SAP Document and Reporting Compliance service
  • SAP Document and Reporting Compliance, cloud edition

Reproducing the Issue

  1. Launch EDOC_COCKPIT transaction.
  2. Enter the Greece eBook source key
  3. Submit it
  4. Display XML
  5. Check Application Logs

Cause

Configuration issue in /N/AIF/CUST.

Resolution

  1. Access transaction /N/AIF/CUST.
  2. Navigate to Define structure Mappings.
  3. Locate entries for (/EDOGR --> INVOUT -->1 and /EDOGR --> INVOUTR -->1).
  4. In Assign Actions, update the namespace from /EDOGR to /EDO for both INVOUT and INVOUTR.
  5. Retest the scenario to verify the resolution.
