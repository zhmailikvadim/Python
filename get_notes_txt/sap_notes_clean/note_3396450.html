Symptom

Changing the zoom value in the settings tab of an SAP Business Client window causes flickering and freezes the Business Client shell window completely. This happens only if there are more than one Business Client process running in parallel.

Other Terms

SAP Business Client; SAP BC; NWBC; Zoom; Flickering; Freeze

Reason and Prerequisites

The problem only occurs if the settings are opened in a shell window as a new tab and another Business Client process is running in parallel. The zoom value is being synchronized between all open shell windows and all running processes.
The zoom slider changes constantly the value and this leads to an endless loop between the different processes.

A workaround without this correction is to use the Settings dialog from the System Selector, because the final values are only applied after the Apply or Ok button is pressed.

Solution

Please install the patch mentioned in section "Support Packages & Patches" or a newer one (SAP Business Client patches are always cumulative).
