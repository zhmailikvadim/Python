Symptom

  • Users got deleted from Role in SAC Tenant if the following Warning ignored:
  • Warning: "This action can't be completed because the number of selections is limited to 4000."

Environment

SAP ANALYTICS CLOUD 2026.10

Reproducing the Issue

  1. Try to Assign more than 4000 members per Role
  2. get a warning: "This action can't be completed because the number of selections is limited to 4000."
  3. Ignore the Warning and continue
  4. All previous members wiped-out

Cause

  • This is a Product Limitation - 4000 members per Role

Resolution

Workaround solution:

Create a Copy of the Role and add new members to the Copy
