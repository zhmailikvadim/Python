Symptom

  • BODS job not writing to AWS S3 bucket.
  • Data is not written to AWS.
  • No data is written.
  • Job runs successfully.

Environment

SAP Data Services

Cause

The Remote Directory setting on the Amazon S3 file location definition was not set correctly.

Resolution

In order to write the file to a directory path underneath the defined AWS S3 bucket, the Remote Directory parameter must be set to that path in the Amazon S3 file location definition.
