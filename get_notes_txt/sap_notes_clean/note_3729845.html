Symptom

Attempting to remove the SFTP public key in the XDL workspace shows the "Remove SFTP key for user" option disabled.

Environment

  • SAP Sales Performance Management
  • Incentive Management on Hyperscalers

Reproducing the Issue

  1. Open the XDL workspace.
  2. Go to File Transfer Settings.
  3. Observe that "Remove SFTP key for user" is disabled (greyed out).

Cause

The SFTP key was migrated during the move from XDL to GoAnywhere. Migrated keys cannot be deleted through the user interface.

Resolution

  1. Please create an SAP Support Ticket with Medium priority (P3);
  2. Add the following information:
    1. Tenant ID
    2. Environment
    3. Login ID
    4. Email
  3. A Support Engineer will be assigned to the case and will help you with your request.
