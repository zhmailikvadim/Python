Symptom

I do not see an expected rate when I enter or submit a time sheet.

Cause

The expected rate is not on the work order.
 
This can occur when a job posting is created before a rate is added to the corresponding job posting template. In this scenario, the work order inherits only the rates that existed on the template at the time the job posting was created, unless the rate is added manually during work order creation.
 
Later updates to job posting template rates do not retroactively update existing job postings or work orders, which is expected system behavior. Because the expected rate was never added to the work order, it does not appear when you enter or submit the time sheet.

Resolution

Revise the work order to add the missing rate.
 
The Creator can add and remove rate codes or rate schedules rule on the job posting must be set to Yes to add or remove rates during a work order revision. This rule is editable on existing job postings via Actions > Edit Job Posting.

Use the steps below to revise the work order and add the missing rate:

  1. Sign into your SAP Fieldglass account
  2. Click View > Work Order
  3. Update filters to locate the Work Order > Click ID to view the work order record
  4. Click Actions > Revise
  5. Ensure the revision's start date is on or before the impacted time sheet's start date
  6. Click + Add or remove rates under Accounting
  7. Update filters to locate desired rate > Tick the checkbox for the desired rate
  8. Click Add Selected > Click Update
  9. Complete the revision workflow
Once the revision is approved and accepted, the worker will see the expected rate when entering or submitting the time sheet.
