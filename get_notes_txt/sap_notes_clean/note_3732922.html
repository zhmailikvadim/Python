Symptom

When tried to open in BW cockpit the Process Chain Runs popup the following error message:

After this popup, the following page showing "Target was not found" page with the following error:

Failed to resolve dependencies of
sap/bw4/lib/components/backend/Component.js'
->'sap/bw4/lib/components/backend/js/Utilities.js'
-> 'sap/bw4/lib/components/backend/js/Dialogs.js'
-> 'sap//m/RadioButton.js': failed to load 'sap//m/RadioButton.js'
from
sap/public/bc/ui5_ui5/1/resources/~XXXXXXXXXXXXXX~/sap//m/Radi
oButton.js: 404 - NOT FOUND

Environment

SAP BW/4HANA 2023

Reproducing the Issue

  1. Go to Tcode BW4WEB.
  2. Go to Search and enter "Process Chain".
  3. Select - "Process Chains - View Active chains".
  4. Error message popup.

Cause

Missing correction from the system.

Resolution

Make sure all available UI correction applied to your BW/4HANA 2023 system and followed the manual step from the  2861491 - UI problem in BW/4HANA Cockpit persists after implementation of a note:

"Execute the following reports in the ABAP server:

  • /UI5/APP_INDEX_CALCULATE
  • /UI2/INVALIDATE_CLIENT_CACHES
  • /UI2/INVALIDATE_GLOBAL_CACHES
  • For SAP_UI 7.58 onward, run the report /UI2/PAGE_CACHE_SYNCHRONIZE (or transaction /UI2/SYNC_PBC)  See the documentation."
