Symptom

You are trying to create a SAP Business Technology Platform (SAP BTP. URL: https://cockpit.btp.cloud.sap) formation. However you notice performance problems and are not able to proceed further. 

Screenshot:

Environment

  • SAP Business Technology Platform (BTP)
  • SAP Cloud Foundry (CF)
  • Release Independent

Reproducing the Issue

  1. Access SAP BTP cockpit SAP Business Technology Platform (SAP BTP. URL: https://cockpit.btp.cloud.sap).
  2. Navigate to the Global Account.
  3. Choose System Landscape ->Formations.
  4. Choose Create Formation and the above stated error occurs.

Cause

The root issue is that there are large number of systems in this Global Account, combined with many formations in draft/Action Required/Synchronizing/Error state, which causes performance issues.

Resolution

This is a known problem and we are working on fixing this performance issue. 
Till then, cleanup and remove any systems in the Global Account that are no longer needed. Also fix erroneous formations in the Global Account.

Keywords

Subaccount Usage, formation, hanging
