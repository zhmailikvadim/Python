Symptom

On the Case screen the the SLA section is not being populated with the SLA timepoints.

Environment

SAP Service Cloud Version 2

Reproducing the Issue

  1. Navigate to Cases.
  2. Open a Case.
  3. Navigate to the SLA section.
  4. Observe that SLA timepoints are not displayed.

Cause

In the SLA configuration screen, the working calender that has been selected is not valid for the current year.

Resolution

  1. Navigate to Settings.
  2. Navigate to Working Calender.
  3. Ensure that the Working Calender that is being used by the SLA is valid for the current year.
