Symptom

  • In HKRCYEA0, Item 28 (The number of disabled) displays as 0 even though a deduction amount is present.
  • The disabled field in the personal deduction section appears blank.

Environment

  • Human Capital Management (HCM)
  • SAP S/4 HANA
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for HANA
  • SAP SuccessFactors Employee Central Payroll

Reproducing the Issue

  1. Execute SE38 for program HKRCYEA0.
  2. Item 28 displays as 0 for The number of disabled and The disabled field in the personal deduction section appears blank.

Cause

 Missing maintenance in IT0021 (Family/Related Person).

Resolution

  • For the count issue (Item 28 shows 0):

    • Check the handicaped person flag in IT0021 and ensure it is selected for the relevant dependent/person.

  • For the blank disabled field in personal deduction:

    • Maintain the appropriate value in the handicaped code field in IT0021.

Keywords

 HKRCYEA0, IT0021,  Handicapped person, Handicapped code.
