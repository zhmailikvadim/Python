Symptom

I would like to know if when I change the P User's email in Managed Gateway for Spend&Network (ISMG), if the P User's password is also changed

Resolution

No, the password remains the same. You can change the P User's e-mail without altering the P User's password.

See Also

To check how to change the P User's email or P User's password, review the following KBA KB0400752 - How do I change e-mail ID and password for P user on Managed Gateway for Spend&Network?
