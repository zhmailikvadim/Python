Symptom

Wrong translation/Missing translation

Other Terms

SE63

Reason and Prerequisites

English Translation of ABAP package PC18_JMHZ are not shipped to customer in support pacakges before:

SAP_HR 608 SP156
S4HCM 100 SP45
S4HCM 101 SP32

Solution

Translation of the objects will be deliveried with Support Package below

SAPKE608F7 for SAP_HR 608 SP157
SAPK-10046INS4HCM for S4HCM 100 SP46
SAPK-10133INS4HCM for S4HCM 101 SP33

Package: PC18_JMHZ
Software Component: SAP_HR/S4HCM
Application Component: PY-CZ


Please implement the support package mentioned above to get the missing English text.

Correction delivery schedule can be found on SAP SUPPORT PACKAGE STACK SCHEDULE:

https://support.sap.com/content/dam/support/en_us/library/ssp/release-upgrade-maintenance/maintenance-information/schedules-for-maintenance-deliveries/SAPschedule.pdf


Please refer to the following notes:

585116 Translation after installation of WebAS SP25
110910 Deleting the language load


Note:

Translations in a customer system may be overwritten in the following instances:

  • If the object happens to be part of a future support package
  • Upgrade to a new release
  • Import of new language CDs from Support Releases
