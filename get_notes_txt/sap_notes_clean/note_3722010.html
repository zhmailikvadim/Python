Symptom

Mass processing for Business Partner data is getting stuck in “In Process (Activation)” status and the dump is raised in ST22 as below:

Category               ABAP programming error
Runtime Errors         UNCAUGHT_EXCEPTION
Except.                CX_FSBP_PARTNER_INCONSISTENT
ABAP: Program          FSBP_API_ADAPTER_INTERN=======CP
Application Component  FS-BP

Environment

SAP Master Data Governance-Consolidation and Mass Processing

Resolution

  • The issue is due to missing mandatory certificates in the supplier's setup, as per the configuration. For example, Supplier X, with account group Y, does not have the required 'CR' certificate, which is mandatory according to the configuration. Review SAP Help documentation: Supplier Certificates | SAP Help Portal and You must specify one certificate as the main certificate. 

See Also

SAP Note  3147727 - BP_BCL: CX_FSBP_PARTNER_INCONSISTENT in FSBP_API_ADAPTER_INTERN
