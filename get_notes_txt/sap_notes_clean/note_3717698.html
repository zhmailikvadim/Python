Symptom

  • The DRC Colombia Income Withholding Tax Declaration report (CO_WHT_INCOME_DCL) is generated successfully but with no content.
  • The issue is observed for any report period, for example, December 2023.

Environment

  • SAP S/4HANA
  • DRC Colombia
  • Statutory Reporting
  • Withholding Taxes

Cause

The documents are not being reported because the Company Code was configured to use the Withholding Tax Reporting Date, as can be checked at view V_WTREPDATE.

With this configuration, the posting date from the documents will be ignored, and the report will consume the date defined by the app "Manage Withholding Tax Items" in the pre-activity for the DRC Colombia Income Withholding Tax Declaration report.

More information about this process can be found at KBA 3358696 - How Withholding Tax reporting date functionality in SPRO works.

Resolution

  1. Review the configuration of the Company Code at V_WTREPDATE or define the date from the documents at the activity "Manage Withholding Tax Items".
  2. Run the report again after making the necessary changes.
