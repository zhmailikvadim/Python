Symptom

You are using SAP GTS to declare you import declaration to Danish customs system. You are implementing the new DMS Import declaration.

This SAP Note contains customizing values for codelist CUCRS relevant for DMS Import and Export.

Other Terms

UCC, Union Customs Code, DMS, Import, Export Codelist, CUCRS Result of Customs Controls_19_03_2026.xml

Reason and Prerequisites

Legal Change

Solution

Import the code lists from the attached XML files.

Please upload the CUCRS code list into the SAP GTS system, which is necessary for integration with the Danish customs system DMS.

The CUCRS code list contains data from the Danish national registration system DK-CRS (Danish Customs Registration System).

Official values of all possible code lists can be found at: [Github] in the file "Codelists - Import.xlsx".


Procedure:

  1. Go to transaction SPRO.

  2. Navigate to SAP Global Trade Services, edition for SAP HANA.

  3. Select Customs Management.

  4. Select Customs Code Lists.

  5. Choose Maintain Customs Code Lists for Legal Regulation.

  6. Select the country: CUSDK.

  7. Click on Import.

  8. Choose the file path : "Result of Customs Controls_19_03_2026.xml".

  9. Make sure that Simulation Mode is unchecked.

  10. Then select either Add New Data or Replace and Add Data.
