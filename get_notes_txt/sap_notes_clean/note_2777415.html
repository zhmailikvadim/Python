Symptom

The “Activate Inactive nametabs” phase ends in error with a list of erroneous /AMR/* views

Other Terms

CARAB 4.0, CARAB 5.0; Activation; SAP Allocation Management, /AMR/* views

Reason and Prerequisites

Activation fails for /AMR/* views

Solution

Run the report RUTDDLSACT for these views. This should successfully activate them and you can proceed with your process.
