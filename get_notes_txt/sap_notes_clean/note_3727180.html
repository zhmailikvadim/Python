Symptom

You can use transaction /IBS/BSC_CAB_MAN_CHG ("VL Manual Correction/Assignment") to backdated assignment of CFS investors to a building loan contract only up to the contract start date.

Reason and Prerequisites

The check for the valid-from date of the assignment is implemented for the contract start date.

Solution

In the above mentioned The check in the method /IBS/CL_BSC_BO_BBS->/IBS/IF_BP_INTEGRATION_BUP~ADD_NEW_DEAL_RELATION integrates the date service.

This means that the date type to be used as the minimum value for the "valid-from" of the assignment can be set in the IMG activity "Assign Date Types to Date Usage Purposes per Business Transaction" (/IBS/CMO_DATRL).

After you implement this SAP Note, check the settings in the above mentioned IMG activity and enhance it as required. In the above The date service is used for the business transaction "Contract Change: Manual DL Processing" (CAB_MAN_CHANGE) with the purpose "Check Validity of BP Role" (ROLE_CHECK). Note the wildcard capability of the table.
