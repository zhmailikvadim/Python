Symptom

Data in transaction J1IDISPBLK is not displaying despite all required configurations being completed.

Environment

 SAP S/4HANA

Reproducing the Issue

  1. Run transaction J1IDISPBLK.
  2. Observe that no data is displayed in the report after execution.

Cause

SAP note 3317915 - Updates to Asset Accounting for India (IT Depreciation), in not implemented or if even its implemented, manual steps mentioned in the note is not completed.

Resolution

  1. Refer to SAP note 3317915 - Updates to Asset Accounting for India (IT Depreciation) .
  2. Check whether the note is implemented in the system.
  3. If the note is not implemented, please implement it. And complete the post implementation manual steps given in the note attachment i.e., India_Key_figure_group.docx 
  4. While implementing SAP Note shows 3317915 - Updates to Asset Accounting for India (IT Depreciation), it gives the message "Cannot be implemented", please check if manual step mentioned in the note are performed or not. If not please perform the manual steps.
  5. Re-run transaction J1IDISPBLK and verify that the data is now displayed.

Keywords

India, J1IDISPBLK,J1INBLKTRNS, IT depreciation, FI-LOC-AA-IN, Asset accounting, Block, Depreciation
