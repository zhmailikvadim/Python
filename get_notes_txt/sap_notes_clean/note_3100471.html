Symptom

When an external candidate creates a profile on the career site, they see the message with the format: 'Successfully created user (email address) for (CO ID)'
This message 

Can the external candidate profile creation acknowledgement message be edited or removed altogether?

Environment

SAP SuccessFactors Recruiting Management

Resolution

The message cannot be disabled but it can be amended using the Manage Languages feature. See KBA 2576546 - Using the Manage Language Feature to Change Labels in SuccessFactors.

The key to use to change the message is :RCM_CANDIDATE_PROFILE_SUCCESS_MSG:
