Symptom

As described in the Regulatory Change Manager under Changes: Meal allowance limits 2026, this SAP Note provides you with the relevant system changes to address the changes in the lunch allowance exempt limits.

For more information, see Portaria n.º 51-B/2026/1. The website that can be accessed using this hyperlink is an external website which is not part of SAP’s offering.

Other Terms

(for more information, see SAP Note 2197631)

Solution
