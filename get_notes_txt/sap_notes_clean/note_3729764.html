Symptom

The System Landscape Directory (B1ServerToolsSLD) installation fails during the post-install action referencing HTTP connect timed out exception.

Reproducing the issue

Scenario:

Install the System Landscape Directory.

Expected behavior:

System Landscape Directory installs correctly.

Actual behavior:

The System Landscape Directory installation fails with the following error in the installation log:
"Error on GenerateSecurityToken: java.net.http.HttpConnectTimeoutException: HTTP connect timed out".

Cause

Program Error

Solution

SAP intends to provide a feature package or patch to solve the symptom that is described in this SAP Note. The section References below lists the related feature package or patch once they become available. Be aware that References can only be confirmed at feature package or patch release date. SAP delivers feature packages or patches only for selected releases at its own discretion, based on the business impact and the complexity of implementation.
