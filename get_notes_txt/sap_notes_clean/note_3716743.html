Symptom

You create a BOM with components and substitutes and provide MUE parameters. During an update, you notice the following symptoms:

- When you delete one substitute assignment, the system deletes all substitutes.

- When you update substitutes with a new ECN and change MUE parameters, the system deletes all substitutes.

- When you change only the ECN and MUE parameters, the system deletes all substitutes.

Other Terms

SV, Structure Version, Components, Items, Substitutes, Alternative Items

Solution

Implement this SAP Note to correct substitute updates with MUE parameters.
