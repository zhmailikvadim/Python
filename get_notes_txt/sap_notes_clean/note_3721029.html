Symptom

Candidate receives the error message "Enter a valid email address" when filling out the I-9 Section 1 task in Onboarding and the error occurs specifically when the email address contains a "+" character.

Environment

SAP SuccessFactors Onboarding 2.0

Reproducing the Issue

  1. Candidate logs into SAP SuccessFactors Onboarding.
  2. Candidate fills out the I-9 Section 1 task using an email address containing a "+" character.
  3. The system displays the error message: "Enter a valid email address".

Cause

Expected Behavior

Resolution

  1. Ensure that candidates use an email address without the "+" character when completing the I-9 Section 1 task.
  2. Inform candidates that email addresses containing the "+" character will cause validation errors in the I-9 Section 1 form.
