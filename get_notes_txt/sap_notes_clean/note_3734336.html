Symptom

This SAP Note collects sapui5 corrections shipped to the Test Suite sapui5 application 'Test Plan Mass Update'.

Before implementing version 72.21.1 from March 30, you observe one of the following symptoms:

  1. Only the first 100 rows of the selection are displayed.

Solution

Apply the attached source code corrections or the corresponding support package.

Caution: After implementing a new version of a sapui5 application, you may have to clear your server cache.
Achieve this action by going to the transaction /nsmicm and follow the menu path: Goto -> HTTP Plugins -> Server Caches -> Invalidate globally.
Only then your final users will be able to use their local command to clear their browser cache.
