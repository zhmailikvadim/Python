Symptom

When you display an individual disbursement from the overview list of the disbursements, the program terminates due to a long runtime.

Other Terms

 MODULE auswahl430; RPNSUP

Reason and Prerequisites

SAP Note 2135117, which added a select to the table VDBEKI, runtime error with the where conditions BUKRS and RPNNR. If there is a large number of VDBEKI entries, the system generates a runtime error.

Solution

The performance of the selection is better when you use the index field RPNSUP (general daybook no.). Implement the attached correction instructions or import the relevant Support Package.
