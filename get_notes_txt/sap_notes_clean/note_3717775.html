Symptom

The system allows saving sales order without add items in manage sales order fiori app or VA01.

Environment

  • Sales And Distribution (SD)
  • SAP ERP Central Component
  • SAP ERP
  • SAP Enhancement package for SAP ERP
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Open the Manage Sales Order Fiori app or transaction VA01.
  2. Create a new sales order.
  3. Leave the material number field blank at the line-item level.
  4. Attempt to save the sales order.

Cause

Resolution

It is the standard behavior within the system to allow a sales order to be saved without items. This design supports business processes in which some customers initially create sales orders at the header level and add item details later.

However, based on customer specific business requirements, please consider that custom code to be added to the user exit USEREXIT_SAVE_DOCUMENT of program MV45AFZZ. This code should include a condition to prevent saving the document if it contains no items.

Keywords
