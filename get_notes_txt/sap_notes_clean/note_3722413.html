Symptom

In the report RCATS_CREATE_ADMIN_ASSIGNMENTS, search help for Activity Type is not displaying any values.

Other Terms

RCATS_CREATE_ADMIN_ASSIGNMENTS, Activity Type, LSTAR, KOKRS, Controlling area, CATS_ADM_ASSIGNMENT

Reason and Prerequisites

Search help values for Activity types require that the controlling area value be determined within the given validity period.

Solution

Implement the note or import the corresponding support package.
