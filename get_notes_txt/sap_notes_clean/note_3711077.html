Symptom

  • Concurrent employment profiles are not visible when proxying as the user, even if that user has been granted the permission role to view their own data.
  • Is it possible for a user (or a proxy acting as that user) to access a concurrent employment profile using a permission role where the target population is set to 'Granted user themselves'?

Environment

  • SAP SuccessFactors HXM Suite
  • SAP SuccessFactors Employee Central



Reproducing the Issue

  1. Proxy as the primary employment user.
  2. Navigate to the employee profile of the concurrent employment user.
  3. Find that the employee profile for the concurrent employment user are not displayed.

Resolution

  • Concurrent employment records are not directly visible when proxying as a user.
  • However, if the user has been granted permission to view their own data, they can access their concurrent employment profile using the Context Switcher. The context switcher supports switching between employment records, including concurrent employment and home/host assignments for global assignments. Please refer to the SAP Help Portal for detailed guidance on the context switcher feature Context Switcher and Employment Switcher Comparison | SAP Help Portal
