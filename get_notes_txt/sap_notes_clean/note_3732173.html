Symptom

Why do system is triggering an Error: Line item 1, Accounting, Accounting 1: Please enter a valid General Ledger on the Credit memo?

Why the user is able to see the Error: Please enter a valid General Ledger on the credit memo approval flow?

Cause

This error might be triggered by the below reasons:

  • When the values like GL account, Cost center, or any accounting fields are not being copied from Invoice to credit memo.

  • If there is no any accounting combination being maintained in CommodityExportMapEntry.csv file (CEME).

Resolution

  • Kindly access the system using an appropriate user role, such as Preparer, Requester, or a user belonging to the Invoice Manager / Invoice Editor group.
  • Navigate to the Credit Memo and proceed to the Accounting section.
  • Please ensure that the correct General Ledger account is selected, in accordance with the values maintained in the CEME file.

Keywords

General Ledger, Valid General ledger, Incorrect GL account, GL account.
