Symptom

You are working with Treasury and Risk Management. When processing FX deals with a BAdI implementation of TPM_TRL_MANIPULATE, error message CHECK-G is incorrectly triggered by method CHECK_FLOWS of class CL_BADI_MANIPULATE_TRL. The BAdI implementation does not change the currency nor the corresponding amounts.

This occurs when:
- A derived flow or valuation flow has an amount only in the valuation currency and 0 in the position currency
- The valuation currency is equal to the position currency

Without a BAdI implementation of TPM_TRL_MANIPULATE, no error is raised and processing completes successfully.

Other Terms

TPM_TRL_MANIPULATE, CL_BADI_MANIPULATE_TRL CHECK_FLOWS, CL_VALUATION_TRANSACTION_TLV EXECUTE_BADI, lcl_call_manipulate_badi check_flows.

Reason and Prerequisites

Programming error.

Solution

Please implement the SAP Note or the Support Package.
