Symptom

This article describes functionality introduced in the May 2026 SAP Fieldglass release

Job Posting or Job Posting Template descriptions may appear as plain text instead of showing bulleted lists, line breaks, or styled text as expected
You may notice that:

  • Bullet points or spacing do not appear after saving
  • Uploaded descriptions look different than expected
  • Formatting does not appear for older postings or templates
  • Text displays as a single block instead of structured content

Reproducing the Issue

This behavior may occur when:

  • You enter a description expecting formatting, but only plain text is shown
  • You upload a file and the description does not display bullets or spacing
  • Formatting symbols appear instead of rendered bullet points
  • Existing job postings or templates do not change after a system update

Cause

SAP Fieldglass supports rich text formatting for Job Posting and Job Posting Template descriptions, but formatting is applied only when it is explicitly added

This means:

  • Descriptions entered without formatting remain plain text
  • Uploaded descriptions must use a supported formatting structure
  • Existing descriptions are not automatically converted to formatted text

This behavior is expected and is part of how rich text formatting works in SAP Fieldglass.

Resolution

Use rich text formatting in the SAP Fieldglass user interface

You can add formatting directly when creating or editing a Job Posting or Job Posting Template:

  1. Open the Job Posting or Job Posting Template
  2. Enter the description using bullets, spacing, or styled text
  3. Save your changes and review the formatted display

Use rich text formatting when uploading descriptions

When updating descriptions through file uploads:

  • Formatting must be included in the description field using a supported structure
  • Descriptions uploaded as plain text will remain plain text
  • Existing descriptions must be re‑uploaded if you want formatting to appear
