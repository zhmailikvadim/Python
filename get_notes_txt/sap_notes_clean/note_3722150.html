Symptom

  • Mass change is performed in the Mass Changes to Purchasing Info Records app. When checking the job logs, the following error message appears: "An error occurred. Please create an incident on component MM-PUR-GF-MAS".
  • Additionally, the corresponding job has status "Canceled" in SM37 transaction. In the Job Log, the error "No authorization to execute the report." is shown.

Environment

  • Materials Management (MM)
  • SAP S/4HANA 2022
  • SAP S/4HANA 2023
  • SAP S/4HANA 2025
  • SAP S/4HANA Cloud Private Edition 2023
  • SAP S/4HANA Cloud Private Edition 2025

Reproducing the Issue

  1. Open the "Mass Changes to Purchasing Info Records" app
  2. Perform mass change (Online or Offline Mode)
  3. Click "Application Jobs"
  4. Observe the job logs. Error "An error occurred. Please create an incident on component MM-PUR-GF-MAS" is raised.

Cause

This issue occurs due to missing authorization.

Resolution

Assign the authorization objects 'S_PROGNAM' and 'S_APJ_TPL' to the affected user.

See Also

  • SAP Note 3190922 - New authorization concept for application jobs
