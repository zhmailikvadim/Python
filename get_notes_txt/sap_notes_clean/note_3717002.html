Symptom

Leverage the new “Configure Standard Order Type” feature in SAP Central Business Configuration.

Note:

"The creation of sales document type is not enabled during the Product Specific Configuration phase."

"The functionality of creating new sales document types is enabled for all customers in the main line and not in the parallel line."

Environment

  • SAP Central Business Configuration
  • SAP S/4HANA Cloud Public Edition

Cause

To use this feature scope BD9 has to be activated.

Sales document types cannot be created during the Product Specific Configuration phase.

Resolution

Configure sales document types allow customer to use a configuration template for order types by copying configuration values from a standard order type.

This feature can be used to create similar order types in SAP Central Business Configuration and deploy them to SAP S/4HANA Cloud Public Edition for productive use and further configuration. 

For more information follow the below:

  • SAP Help Portal - Configure Standard Order Type.
  • SAP Community - What’s New in SAP Central Business Configuration 2601.
  • Video tutorial - Create Custom Sales Order Type.
