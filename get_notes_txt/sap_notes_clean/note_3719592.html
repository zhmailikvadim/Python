Symptom

When processing contracts with are terminated, the Calculation Methodology "clear" (900) is derived togehter with the Calculation Method 'nn90000', where 'nn' is replaced with the calculator.

The resulting subledger documents should then contain this information in the fields

/BA1/C55CMETH - Calculation Method

/BA1/C55CMETHY - Calculation Methodology

Due to an error, those fields remain empty.

Reason and Prerequisites

 There is a program error.

Solution

Apply the correction instruction or the respective support package.
