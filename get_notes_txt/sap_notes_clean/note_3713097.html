Symptom

When attempting to open the Offers tab, the system displays an application error. The message includes an errorId similar to a7bba2ca-1488-4c23-b28a-2fd8f09eb43f with fingerprint 2e8eda67bc2844df5e7b93e9c9ac5cd4f212fd14.

Environment

SAP SuccessFactors Recruiting Management 

Cause

A Requisition Template is linked to a Status Set that has been deleted or is missing, causing the system to fail when resolving the required configuration for the Offers tab.

Resolution

Run the Check Tool for both the Job Application and Job Requisition templates to identify any missing or invalid Status Set references. The tool will display which templates and fields are pointing to a Status Set that no longer exists.

After reviewing the results, update the affected templates so they reference a valid Status Set. Once the necessary corrections are made, save the templates to apply the changes.

After the configuration is corrected and saved, the application error should no longer occur when accessing the Offers tab.
