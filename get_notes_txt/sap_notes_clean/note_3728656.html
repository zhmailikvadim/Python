Symptom

  • The responsible users are able to select all employees as a buddy in Assign a buddy task.

Environment

SAP SuccessFactors Onboarding

Reproducing the Issue

  1. Navigate to the onboarding module.
  2. Attempt to assign a buddy task to a new hire.
  3. Observe that users are able to select all employees as a buddy instead of being restricted to their direct reports.

Cause

  • Incorrect Role-Based Permission (RBP) settings, the "User Search" permission role assignments, which include "all employees".

Resolution

  1. Access the RBP troubleshooting page.
  2. Search for the relevant HR user.
  3. Navigate to "General User Permission > User Search" permissions.
  4. Review the user’s permission role assignments.
  5. Identify if any role assignments include "all employees".
  6. Modify the assignment to restrict the user search permissions to only include direct reports.
