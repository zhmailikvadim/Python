Symptom

When assessment is initiated, SuccessFactors sends the details to the vendor. What is the header type of the payload request sent? Can this header type be changed?

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

SAP SuccessFactors Recruiting Management

Reproducing the Issue

  1. Initiate an assessment
  2. Go to Event Notification Audit Log
  3. Click View under Event Details column
  4. The payload details will appear
  5. Here's how the Request looks like:

Resolution

The payload sent by SuccessFactors to the third-party vendor contains a Content-Type header set to XML.

This header type cannot be customized or changed to other format (e.g. json).
