Symptom

Importing a delivery unit fails, and a variation of the following error message is logged in the error log file:

"[Activation][8] Table import target table cannot be found. Table import target table cannot be found."

Environment

SAP NetWeaver

Reproducing the Issue

  1. Open SAP HANA Application Lifecycle Management
  2. Click the Products tab
  3. Click the Delivery Units tab
  4. Click Import
  5. Select the desired Delivery Unit for import

Cause

Delivery Units (DU) are transportable objects used for content delivery and the server receiving the import needs to have the pre-requisite components installed already.

DUs do not create tables, but rather populate a table with data. Since the table does not exist, the above error message is thrown and the DU is not imported successfully.

Resolution

Install the application the DU's content is for and then import the DU. The table will be created from the installation of the application, and the DU will then appropriately populate the table.
