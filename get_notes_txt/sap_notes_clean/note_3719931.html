Symptom

Error message AA632 - 'The line item violates the cutoff value of &2 in area &1' appear during manual depreciation posting.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Use the app "Post Depreciation Manually - Unplanned and Planned" to post manual depreciation.
  2. Error Message AA632 - 'The line item violates the cutoff value of &2 in area &1'.

Cause

Values in the depreciation areas are becoming negative. The corresponding other currencies (group currency, etc.) of the company code currency become negative after posting.

Resolution

Maintain the correct exchange rate in the Currency Exchange Rates app.

Keywords

error message AA632, cutoff value, depreciation area 15, manual depreciation, fixed asset, negative value allowed, Post Miscellaneous Transactions app, Manage Fixed Assets app, SAP S/4HANA Cloud
