Symptom

This SAP Note provides you with details about the Revaluation of Fixed Assets solution for Romania introduced via SAP Note 3212185.

This solution is not supported if the Universal Parallel Accounting (FINS_PARALLEL_ACCOUNTING_BF) business function is active in SAP S/4HANA.

Other Terms

Romania, Revaluation of Fixed Assets, Universal Parallel Accounting, SAP S/4HANA, FINS_PARALLEL_ACCOUNTING_BF, FAA_DOCLINES_COUNTRY, GLOFAA_RO_REVAL, AR29n

Reason and Prerequisites

The Revaluation of Fixed Assets solution for Romania enables the net revaluation posting via the Revaluation and New Valuation of Assets report (transaction AR29n), which is not supported in the Universal Parallel Accounting.

This SAP Note is relevant only for Romania.

Solution

This is a known restriction and there is no solution available for the time being.

For more information about UPA activation in SAP S/4HANA, see SAP Note 3191636.
