Symptom

It's missing and requested in transaction SDLP. that log results for a dynamic logpoint can be display in a new window.

This is very helpful to work with different logpoints at the same time, and keep displaying the list of logpoints also when analyzing logs.

Reason and Prerequisites

Issue in ABAP code. A correction is required.

Solution

Apply the provided correction or upgrade to the support package level, that is assigned to this note.

Please note that there are manual activities, since text elements appearing in the transaction UI have to be created.
