Symptom

Poland outbound invoice failed with the error in the history tab as Invalid certificate

Cause

Ariba Network do not support KSeF certificate based country configuration.

Resolution

Supplier must update the country configuration set up in their account with the type KSeF Token instead of KSeF certificate. To update the set-up follow the steps below:

  1. Login to SAP Ariba Network Supplier account
  2. Navigate to Initials->Company Profile
  3. Open the tab Country Configuration->Create
  4. Choose the radio button for KSeF token and create an entry
  5. Save

Keywords

KSeF, KSeF token, Poland, Poland outbound
