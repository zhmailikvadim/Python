Symptom

  • Plan amounts exist for an individual plan category in table ACDOCP and are displayed in P&L - Plan/Actual (F1710).
  • When selecting a virtual plan category, no plan amounts are displayed in F1710.

Environment

  • Controlling (CO)
  • SAP R/3
  • SAP R/3 Enterprise 
  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA Finance
  • SAP S/4HANA
  • SAP Fiori for SAP S/4HANA

Cause

Virtual plan categories are not supported in standard controlling reporting apps such as F1710. Only individual plan categories are evaluated.

Resolution

  • Use the individual plan categories in P&L - Plan/Actual (F1710).
  • If reporting on virtual plan categories is required, submit an enhancement request via SAP Customer Influence.

See Also

Refer to: KBA KB0731141 - Plan data not displayed in various BPC based Fiori apps

Keywords
