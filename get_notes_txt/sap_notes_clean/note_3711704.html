Symptom

With change request TTP 2474 (note 2987540), the field „No list Pr” (Trusted customer not valid for listed products) has been introduced. If the indicator is set for a customer, it is checked during the AWV determination if at least one sales order item contains a listed product. A product is considered as listed if it is not equal to N or 9X9999 or 9I999* or EAR99*.

But classification numbers 9N9999 and NA have not been included in the exceptions though it is not a listed product.

Other Terms

6000162009, INC44558223, C2SFIBU, AWV, 2474, TTP2474.

Reason and Prerequisites

Change Request: ticket 6000162009 / INC44558223.

Solution

Implement changes according to the correction instructions.
