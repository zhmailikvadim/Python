Symptom

When manipulating the budget of an order or WBS element, by supplementing the budget (t-codes KO24/CJ36) or performing a budget carryforward (t-codes KOCO/CJCO), the error message BP005 - "System error in : BPIF Table: O_JA Report: LKBPPU04 Routine: BPGE" is raised.

Environment

  • SAP ERP
  • SAP Enhancement package for SAP ERP
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA
  • SAP S/4HANA Finance

Reproducing the Issue

  1. Open t-code KOCO
  2. Insert the selection criteria
  3. Execute

Cause

There is an inconsistency in the budget profile of the order or WBS element.

Resolution

Reconstruct the CO availability control, as per SAP Note 922295 - Adjusting the availability control after Customizing changes.

See Also

922295 - Adjusting the availability control after Customizing changes
