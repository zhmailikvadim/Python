Symptom

************************************************************

WARNING: This is a CA-FL (Field Logistics) specific note. If you DON'T have

Field Logistics activated in your system, this note does not apply to you.

************************************************************

When using the 'Unstage' action in the Manage Kits application for a single-plant EWM-managed scenario, the resulting Inbound Delivery is incorrectly assigned to the staging storage location instead of the main inventory storage location.
