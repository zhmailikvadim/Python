Symptom

In the sheet "BOM Item", the field descriptions for "Item ID" and "Indicator: Internal Item Identification" contain contradictory information regarding the definition of external and internal Item ID.

The inconsistent explanations lead to ambiguity and may cause misunderstanding or incorrect maintenance of BOM item data. 

Environment

SAP S/4HANA Cloud Public Edition

Cause

The "Item ID" description is wrong.

Resolution

The tooltip description for the field “Indicator: Internal Item Identification” shall prevail.
The column header for the indicator is “Indicator: Internal Item Identification”, with the following text: "Please enter X if it’s internal, leave the field empty if it’s external."

The description for “Item ID” should be corrected as follows:
This field is used to identify a BOM item within a BOM group.
The item ID is technically mandatory for the system to establish internal links between records in subordinate sheets. It can be assigned as follows:
  • For an external item ID: Enter at least one non-numeric character.
  • For an internal item ID: The system automatically assigns the item ID.
