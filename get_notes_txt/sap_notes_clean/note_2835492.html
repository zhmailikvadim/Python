Symptom

Customer reports following error while running the Lease Posting Run ID XYZ (XYZ represents Lease Posting Run ID)

Error: Account determination missing for item type Historical Costs: Current Year.
Error: Other valuation items related to lease contract AA cannot be processed (AA represents contract ID).

Environment

SAP Business Bydesign

Reproducing the Issue

  1. Go to the Lease Contract Management work center.
  2. Select the view Lease Contract Posting Run.
  3. Select the Run ID XYZ.
  4. Select the Processing Status "Error" under the Execution Details section.
  5. Click View.
  6. Go to the tab Messages.

Error: Account determination missing for item type Historical Costs: Current Year.
Error: Other valuation items related to lease contract ZZ cannot be processed.

Cause

G/L account is missing in Historical Cost of the Account Determination Group of corresponding ROU Class ZZZ (ZZZ represents Right-of-Use Asset Class)

Resolution

You need to identify the Account Determination Group from the related Right Of Use Asset.

  1. Open the contract AA.
  2. Go to the Leased Objects tab and identify the Right of Use Asset ID.
  3. Open the Right of Use Asset.
  4. Identify the Account Determination associated.

Maintain the Historical Cost of the Account Determination Group:

  1. Go to the Business Configuration work center.
  2. Search and find the activity: Activity Charts of Accounts, Financial Reporting Structures, Account Determination.
  3. Select Fixed Assets under Account Determination.
  4. Select the correct Account Determination Profile.
  5. Go to the Historical Cost tab.
  6. Select Add Row button and Maintain a Historical Cost account for your Account Determination Group.

Keywords

IFRS16, Lease Posting Run, Historical Costs, Lease Type, Account Determination
