Symptom

In S/4HANA 2023, 'Work Approval' button is displayed when order is in CRTD status.
However, in S/4HANA 2020, Work Approval button is not displayed when order is in CRTD status and appears only after order is released.
The behavior is different for 'Work Approval' button in transaction IW32.

Environment

  • SAP ERP Plant Maintenance (PM)
  • SAP S/4HANA, On premise
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

2020 system

  1. Open order in transaction IW32.
  2. The status is CRTD NMAT PRC  RELR.
  3. 'Work Approval' button is not displayed.

2023 system

  1. Open order in transaction IW32.
  2. The status is WCM  CRTD NMAT NTUP PRC.
  3. 'Work Approval' button is displayed.

Cause

As per standard delivered WCM, activation of WCM (WCM status) does NOT depend on PM order release status.

Resolution

The 'Work Approval' button appears only when WCM status is set on PM order.
