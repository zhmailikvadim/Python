Symptom

You execute the analysis report RKKBCAL2 to compare two costing runs. You notice that the output list displays data of costing run 2 that does not match the material/plant of costing run 1.

Other Terms

CK40N, analysis step, RKKBCAL2, S_ALR_87099932

Reason and Prerequisites

Program error.

This issue can occur if the indicator "Intersection of Runs 1 and 2" is not selected.

Solution

Install the attached coding corrections.
