Symptom

The jobs triggered by Solution Manager to import/release/create transport requests remain in a pending state, waiting for a response when the system encounters a malfunction or experiences delays in its response. Consequently, this results in a buildup of queued jobs within Solution Manager.

Environment

SAP Solution Manager 7.2 SP12 - SP15

Reproducing the Issue

  1. Go to SM50
  2. Check the WP status for the jobs 

Cause

Missing code correction

Resolution

Implement and follow the instructions from SAP Note:

3229188 - Transport actions hanging when the system is not available – Change Control Management

See Also

1322696 - Work with Project Cycles where some systems are unavailable
