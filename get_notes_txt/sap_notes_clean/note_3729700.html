Symptom

This SAP Note provides XSDs and converter BAdI implementations, which are a technical prerequisite to use the new MX Rulebook 2026 (SWIFT CBPR+ Standards Release 2026) converters. They enable the possibility to perform the mapping inside of the MX converters based on the internally used ISO 20022 representation.

It contains converter BAdI implementations for the following ISO messages:

  • CAMT.105.001.03 ( Advice of Charges, Interest and Other Adjustments V3 )
  • CAMT.106.001.03  ( Request for Payment of Charges,Interests and other Expense ISO V3)

Other Terms

/PE1/BADI_FH_XML_CONVERTER_IPM, CAMT.105.001.03, CAMT.106.001.03, Charges Advice, Charges Request.

Reason and Prerequisites

Implementation of new ISO messages required new BAdI implementations.

Solution

Please implement provided correction instructions or corresponding Support Package for ISO Converter implementation.
