Symptom

"Unknown property” error while saving the Configure Public Sector Management Cost Object Assignments configuration.

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

SuccessFactors Employee Central: Higher Duties & Public Sector

Reproducing the Issue

  1. Navigate to Admin Center > Configure Public Sector Management Cost Object Assignments.
  2. Attempt to locate the "Allow Decimal Values for Percentage during Cost Assignments" option.
  3. If the option is visible, set it to "Yes" and click Save.
  4. Observe the error: "unknown property isDecimalValueAllowedForPercentageField for PSMCostObjectConfigurationGO".

     

Cause

Synchronization issue.

Resolution

  1. Go to Admin Center > OData API Metadata Refresh and Export.
  2. In the Refresh Metadata section, click Refresh to manually synchronize metadata.
  3. Return to Configure Public Sector Management Cost Object Assignments and save the setting again.
