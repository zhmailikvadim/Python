Symptom

Where can I find the URL for cXML Invoices?

  • HTTP POST requests to SAP Ariba for cXML invoice transfer require the correct endpoint URL.
  • Missing or incorrect endpoint URL prevents successful integration for invoice transfer.

Reproducing the Issue

  1. Log into SAP Business Network account. 
  2. Navigate to User initials > Settings > Electronic Invoice routing. 
  3. Go to URL to send cXML documents: 
  4. Attempt to transfer invoices via HTTP POST to SAP Ariba.
  5. Use an incorrect or missing endpoint URL for the transfer

Cause

  • The issue arises due to the absence or incorrect configuration of the endpoint URL required for HTTP POST integration with SAP Ariba.

Resolution

  1. Confirm the SAP Ariba endpoint URL format for invoice transfer:
  • Standalone environments: https://{service-base}/Buyer/cxmlchannel/{ANID}?ExternalCXMLRequest=true
  • Multi-ERP environments: https://{service-base}/Buyer/cxmlchannel/{SystemID}-{ANID}?ExternalCXMLRequest=true

      2. Ensure the following prerequisites are met:

  • Ariba Network ID (ANID).
  • Shared Secret configured by the SAP Business Network Administrator.
  • Correct service-base URL of the Ariba environment.
  1. Create a cXML InvoiceRequest and send it via HTTP POST with Content-Type set to XML.
  2. Verify that the invoice is processed successfully by SAP Ariba, ensuring network validations are completed.
  3. Refer to SAP documentation for detailed guidance on external cXML invoice integration.

See Also

  • For suppliers who want to integrate directly with the SAP Business Network (formerly Ariba Network) to send cXML invoices, the URL (often called "Post URL") depends on both the data center and the authentication method used (shared secret or certificate).

    For shared secret authentication:

    North America:
    https://service-2.ariba.com/service/transaction/cxml.asp
    European Union:
    https://service-2-eu.ariba.com/service/transaction/cxml.asp
    For certificate-based authentication:

    North America:
    https://certservice-2.ariba.com/service/transaction/cxml.asp
    European Union:
    https://certservice-2-eu.ariba.com/service/transaction/cxml.asp
     

    Refer to: SAP Help Documentation on importing external cXML invoices.
