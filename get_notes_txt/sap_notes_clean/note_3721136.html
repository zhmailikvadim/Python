Symptom

While launching the Fiori app "Consolidation Monitor" in Group Reporting or T-code CX20, the following error message is displayed: "Task 2011 in task group XXXX requires a document type".

Environment

SAP S/4HANA Finance for Group Reporting

Reproducing the Issue

  1. Launch the APP Consolidation Monitor or T-code CX20;
  2. Error message: "Task 2011 in task group XXXX requires a document type." appears.

Cause

Missing configuration

Resolution

Please use below way to solve this issue.

  1. Go to T-code CXGP and set the global parameters.
  2. Then go to T-code CXEA.
  3. For the Elimination task 2011, assign a document type.

Keywords

Consolidation Monitor, Group Reporting, Task 2011, Task Group, document type, elimination task, global parameters
