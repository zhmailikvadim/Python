Symptom

A custom field was created in a language other than English, such as German, but it is not displayed in the Field Editor UI.

Environment

SAP Emarsys Customer Engagement

Reproducing the Issue

  1. Create custom fields via Field Editor in German language.
  2. Observe that the fields do not appear in Field Editor.
  3. Check the contact's profile and notice the fields are visible but without names.

Cause

The fields were created in other language than English and the system user's "Form default language" setting is not selected for the launguage that the field was created, such as German, which prevents the fields from being visible in the Field Editor.

Resolution

Access the system using a user configured to use the German language. Select "German" in the "Form default language" settings for the user:

Also, maintain an English translation for the fields to ensure visibility for all users.
