Symptom

  • In the Maintain Business Users app, the "Display Changes" option does not work.
  • When clicking "Display Changes" in App "Maintain Business user", encounter the error message: Sorry, we can't find this page
    HTTP request was not processed because the previous request failed 

Environment

SAP S/4HANA Cloud Public Edition 2602

Reproducing the Issue

  1. Access the "Maintain Business Users" app.
  2. Select a user with ID CB9912345678.
  3. Attempt to use the "Display Changes" option.
  4. Encounter the error message: Sorry, we can't find this page
    HTTP request was not processed because the previous request failed 

Resolution

A fix will be delivered with HFC04 (14/15.February).

SAP S/4HANA Cloud Public Edition, 2-System Landscape – Upgrade & Maintenance Schedule

SAP S/4HANA Cloud Public Edition, 3-System Landscape – Upgrade & Maintenance Schedule

Please note the release date might be subject to change.

As a workaround, you can use the "Display Changes" on the Worklist of Maintain Business Users app and filter for the User ID to get the same content.
