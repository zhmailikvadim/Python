Symptom

PCC-Live payroll not updating IT0003 post payroll completion for employees

Resolution

 The change document (PCL2, RELID LA and SA) is only updated when there are data change to the infotype/fields. In case of repeat payroll run (during payroll control record correction phase), no changes is made to the PA0003 record. Since the payroll status is already being updated in the first productive payroll run. Thus there will be no change document entry created.
