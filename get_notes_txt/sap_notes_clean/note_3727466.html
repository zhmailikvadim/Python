Symptom

  • The user has many offboarding tasks that need to be triggered sequentially, where each task is triggered only after the completion of the previous one.

Environment

SAP SuccessFactors Onboarding

Cause

This is current system limitation.

Resolution

Please raise an enhancement request to include this in future release.
