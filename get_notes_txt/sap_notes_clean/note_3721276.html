Symptom

Apache SNI upgrade is scheduled for Live Production environments on 02 March at 20:00 UTC

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental"

Environment

  • SAP Commerce Cloud 2211
  • SAP Commerce Cloud 2211 for JDK21

Resolution

  1. The Apache SNI upgrade is scheduled for release to the Live Production environment on 02 March 2026.
    During the week of Feb 23, 2026 "production-sized" environments are still getting upgraded, if they are not live (for example P2 Prod-like environments).
    • Feb 23 -> Non-Live environments.
    • Mar 2 -> Live Production environments.
  2. After the upgrade is implemented in the production environment, the following actions are required to ensure the new Apache version(Server version: Apache/2.4.65 (Unix)) is reflected:
    1. A customer can re-deploy an existing build, or even just change a configuration and apply properties in the cloud portal.
    2. Customers can check their version of Apache by checking the Apache logs at startup or by validating the same from Dynatrace (Technology & Processes->Apache HTTP Server->Apache Web Server apache* apache*-igc-*).
    3. These actions can be performed after 02 March, but it is advisable not to delay them for an extended period to ensure optimal functionality.
