Symptom

UI is not responsive and displays the following message "We'll be back soon! Sorry for the inconvenience but we're performing some maintenance at the moment. We'll be back online shortly!"

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

SAP Sales Performance Management

 

Reproducing the Issue

  1. Access the environment.
  2. Attempt to log in.
  3. Observe that the page indicates the environment is under maintenance.

Resolution

If users are unable to access the Incentive Management, please open a Support ticket and provide the following details:

  • How many users are being impacted?
  • Provide HAR traces for Support to analyze
  • Which environments are facing this behavior?

For more information on how to capture the HAR traces, please review the following KBA:

  • KBA 3503700 - How to to collect HAR (HTTP Archive) files for support - SAP for Me
