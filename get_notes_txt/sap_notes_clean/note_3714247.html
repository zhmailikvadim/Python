Symptom

In the custom business objects (new version) app the value helps for determination and validation trigger condition fields and side effect targets don't display any entries.

Other Terms

Custom Business Objects (New Version), CBO2, Value Help

Reason and Prerequisites

Due to a coding error the affected value helps use hidden filters which are incorrectly defined to use fields from an entity which doesn't exist at runtime as the filter values.

Solution

Correct the coding error by correctly defining the hidden filters to use filter values from an entity which exists at runtime. Instead of trying to use the fields of the trigger condition field or side effect target for which the value help is defined, use the fields of the determination or validation for which the trigger condition field or side effect target will be maintained.
