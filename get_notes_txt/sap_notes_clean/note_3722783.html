Symptom

The rates and monetary amounts related to employees' PAYE from the previous tax year no longer apply. The payroll system must reflect the latest changes in the rates and monetary legislation, effective March 1, 2026. This SAP Note describes the relevant system changes to ensure legal compliance for the 2026/2027 tax year. You can find additional information on the regulatory change manager tool under 'Monetary Amounts changes effective March 01, 2026'.

Other Terms

T511P, BRSAL, BURTH, BURHE, BURDB, BURDH, HLONT, HLNAR, MXDLS, LSATH, AABAT

Reason and Prerequisites

Legal change.

This SAP Note is relevant only for South Africa localization.

Solution

After applying this SAP Note, the SAP system updates the value of the relevant payroll constants in the payroll constants T511P table, effective March 1, 2026.

SAP recommends that you implement the solution by using the Support Package specified for your release.

If you need to install this solution earlier, run the manual steps in 'Manual Instructions for SAP Note 3722783.pdf' attached to this SAP Note.
