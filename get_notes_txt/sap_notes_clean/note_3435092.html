Symptom

Remote Conversion Task List failed with following error message: 

  • There is no active DLM run with maintained target system.

Environment

  • SAP BW/4HANA

Reproducing the Issue

Execute Task List SAP_BW4_TRANSFER_CLOUD_REMOTE directly in TCode STC01. 

Resolution

Please do not execute task list directly for remote conversion. 

It is necessary to start the remote conversion from TCode RSB4HCONV -> Execute Scope Transfer. 

Create a new package and follow the procedure step by step.
