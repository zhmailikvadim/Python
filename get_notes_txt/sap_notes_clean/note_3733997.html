Symptom

During rehire, when clicking Continue before Job Information, the system displays an error: 

Please enter a valid Pink – NRIC. You should use the format ANNNNNNNA.

The issue occurs only during rehire.

Environment

SAP SuccessFactors Employee Central

Reproducing the Issue

  1. Start Add New Employee and choose the option to rehire an inactive employee.
  2. Enter personal and national ID information.
  3. Click Continue to proceed to Job Information.
  4. Observe the error message.

Cause

Dependents of the employee have an incorrectly formatted National ID.

Resolution

Delete/modify the dependents with the invalid National ID format, then it is able to proceed to the job information section by clicking Continue.
