Symptom

FMs CSAP_MAT_BOM_READ and CSAP_MAT_BOM_SELECT are used during SV as Classic BoM maintenance or read. Internal material number is passed to these FMs, however it is wrong and could lead to errors. External number has to be used instead.

Other Terms

PLMSI, SV, Classic BoM, Structure version, material number, PV ID

Solution

Implement this SAP Note. It provides a fix to use external material number for FMs CSAP_MAT_BOM_READ and CSAP_MAT_BOM_SELECT
