Symptom

Measurement concept management instance has multiple versions. The same point of delievery is used across version with different reference cloud location ID text (REF_POD_DESC). For such cases, incorect reference cloud location id text is referred to read configurations.

Other Terms

/APEU, MCM, DER, REF_POD_DESC, Reference Cloud Location ID Text , Point of Delievery,

Reason and Prerequisites

Program error.

Release 2602.

Solution

Please implement the attached the correction instruction.

During access to point of delievery, the Instance guid is not considered leading to read of previous version data. This results into different reference cloud location id text. With the corrections, now it will read with the instance guid leading to correct Reference Cloud Location ID Text.
