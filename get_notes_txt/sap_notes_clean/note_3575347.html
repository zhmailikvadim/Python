Symptom

For JAVA system, the products S/4HANA and ABAP PLATFORM are assigned for the source system, after choosing the option of "Update SAP NetWeaver" in Maintenance Planner, there is no Netweaver versions are listed. For example, S/4HANA 2020 and ABAP PLATFORM 2020.

Environment

SAP JAVA systems

Resolution

The current version of system is S/4HANA platform, which is a different product from SAP Netweaver, so the system can only be upgraded to higher version of S/4HANA platform, the Netweaver versions can not be chosen.

For more information about upgrade/update troubleshooting, you can also refer to our System Upgrade & Update Troubleshooting Wiki.
