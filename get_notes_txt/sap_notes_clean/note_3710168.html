Symptom

When the candidate is submitted via a data capture form the candidate is not searchable in candidate search when using the candidate profile extension 

Environment

SAP SuccessFactors Recruiting Management 

Reproducing the Issue

  1. Go to candidate search 
  2. Search using data capture form and candidate profile extension 
  3. Candidate not found 

Resolution

The engineering team will be delivering a code fix for this issue during the 1H 2026 Release

The Release Dates are as follows

  • Preview: 13th of April 
  • Production: 15th of May
