Symptom

  • A result object contains a comma in its name (for example, “Store, Name”), the column header is split during CSV export in Data Mode, resulting in unexpected additional columns.

Other Terms

CSV export, Webi

Reason and Prerequisites

Environment:

  • SAP BusinessObjects Business Intelligence platform 4.3 Service Pack SP05 Patch04
  • Windows
  • Web Intelligence (Webi)
  • CSV Export

Steps To Reproduce:

  1. Login to BI Lauchpad
  2. Navigate to Folders > Web Intelligence having result object with a comma in its name (e.g., “Store, Name”).
  3. Export a Webi report in CSV with Data option.
  4. Observe that the column header is split, resulting in unexpected additional columns, misaligning the header data.
