Symptom

Customers would like to receive notifications related to SAP Commerce Cloud, including:

  • Service disruptions

  • Planned or unplanned maintenance

  • System upgrades

  • Patch releases and updates

Additionally, customers ask:

  • How to subscribe to these notifications

  • Whether SAP Commerce Cloud patches must be applied by SAP

  • If customers can receive alerts when new patches are released

Environment

SAP Commerce Cloud 

Cause

By default, customers may not have Cloud System Notification Subscriptions (CSNS) configured for their S-user.
Without this configuration, notification emails for system updates, maintenance, or patch releases will not be received.

Additionally, SAP Commerce Cloud patching and infrastructure updates are managed by SAP, and customers do not directly apply patches themselves.

Resolution

1. Subscribe to SAP Commerce Cloud notifications

Customers can subscribe to notifications through the Cloud System Notification Subscriptions (CSNS) service in SAP for Me.

Steps:

  1. Log in to SAP for Me

  2. Navigate to the Cloud System Notification Subscriptions (CSNS) service

  3. Add or manage subscriptions

  4. Select the relevant SAP Commerce Cloud system

  5. Configure the notification categories you wish to receive

This service allows customers to receive alerts for:

  • Service disruptions

  • Planned maintenance

  • Upgrade activities

  • Other system-related events

2. Check planned and unplanned maintenance

Maintenance notifications are available in the:

Cloud Availability Center

This portal provides visibility into:

  • Planned maintenance windows

  • Unplanned outages

  • Service status information

3. Patch management in SAP Commerce Cloud

In SAP Commerce Cloud (Public Cloud):

  • SAP manages patching and platform updates

  • Customers do not manually apply patches to the infrastructure

If a patch or update is required for a specific issue, SAP Support will coordinate the appropriate action.

4. Receive notifications for new patches

To be informed when new patches or updates are released:

Ensure that your CSNS subscription includes patch-related notifications for your SAP Commerce Cloud system.
