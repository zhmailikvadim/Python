Symptom

When the rate table is accessed in edit mode and a check is executed without any changes to the screen values, the system triggers the following error message: "The maximum value must not be less than the minimum value," or vice versa.

Other Terms

/scmtms/tc_rates -321, 322, rate table, minimum, maximum, check,

Reason and Prerequisites

Program error

Solution

Implement the correction instructions or upgrade to the latest support package
