Symptom

The following error is happening while trying to upload file from SAP Note 1707321 - BW System Copy: Post Copy Automation (BW-PCA)

The 'FILE_TEMPLATE' given in the input file is not supported. Message no. SCWN_NTAN023

Environment

SAP Netweaver

Cause

System working as expected. BW PCA notes need to be installed based on the scope of the system copy, therefore, SNOTE Note analyzer cannot be used.

Resolution

  1. Copy the contents of text file "Z_SAP_NOTE_ANALYZER.TXT" available in the attachment of SAP Note 1707321 - BW System Copy: Post Copy Automation (BW-PCA)
  2. Create a custom program and paste the content copied in step 1
  3. Execute this report and load the XML file provided in the attachment of the same note

Keywords

SCWN_NTAN, 023, SNOTE
