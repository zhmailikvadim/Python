Symptom

  • Error occurs when starting t-code S_AC0_52000887 – Receivables: Profit Center.
  • The transaction terminates with a short dump.
  • Runtime error: MESSAGE_TYPE_X.
  • ABAP program: SAPLKYPW1 (include LKYPW1U03).
  • Message class KH, number 299 with variable RKD_CURRENCY_UNIT_GET.
  • Termination at statement MESSAGE x299(kh) WITH 'RKD_CURRENCY_UNIT_GET'.
  • Application component: EC-EIS-DCM.

Environment

• SAP R/3

• SAP R/3 Enterprise

• SAP ERP Central Component

• SAP ERP

• SAP enhancement package for SAP ERP

• SAP enhancement package for SAP ERP, version for SAP HANA

• SAP S/4HANA Finance

• SAP S/4HANA

Reproducing the Issue

  1. Start t-code S_AC0_52000887 – Receivables: Profit Center.
  2. Observe that the system terminates with a short dump (MESSAGE_TYPE_X).

Cause

  • Upgrade issue requiring regeneration of drilldown reports after the upgrade.

Resolution

  1. In each upgraded system, regenerate the FI drilldown reports in all affected clients using the respective drilldown reporting transaction.
  2. Perform the regeneration first in a non-production/test client to validate the outcome.
  3. Refer to: 2388578 - Runtime errors when executing drilldown reports after upgrade.

See Also

  • Refer to: 2388578 - Runtime errors when executing drilldown reports after upgrade
