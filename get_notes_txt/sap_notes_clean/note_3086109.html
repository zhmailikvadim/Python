Symptom

For some bank accounts, when you create flows from the payment request table to the FQM_FLOW table, the House Bank / House Bank Account / Tech ID are missing for the offsetting PAYRQ flow.

Other Terms

PAYRQ, Incoming payment

Reason and Prerequisites

This is a program error.

Solution

Implement the corrections of this SAP Note using the transaction code SNOTE in your back-end system.
