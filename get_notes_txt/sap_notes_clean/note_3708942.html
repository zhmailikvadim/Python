Symptom

Error occurs when adding a new value in characteristic change transaction CT04.

Environment

  • SAP ERP
  • SAP S/4HANA
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Call transaction CT04.
  2. Set characteristic, change to edit mode.
  3. Select tab "Values".
  4. Add a new value with a value X, which already exists as description of another existing value.
  5. Click Save.
  6. Observe the error message: "Value X entered more than once."

Cause

The system checks the combination of neutral values and language-dependent value descriptions across all languages.

If a combination is not unique, the system issues error C1034.

Resolution

Enter a value that does not yet exist.

Make sure that the neutral values and the language-dependent value descriptions are unique in all languages.
