Symptom

In the EarlyWatch Alert (EWA) report, under the Hardware Utilization Data chapter, one or more of the following issues are observed:

  • Operating System not detected for one or more hosts.
  • Message displayed: “OS not detected”
  • Performance Data missing for one or more hosts
  • Performance Data column shows “--” instead of OK.

Environment

Solution Manager 7.2

Resolution

1: OS Not Detected

  • In the Hardware Utilization Data section, the Operating System field is empty or shows “OS not detected” for certain hosts.
  • Even though Performance Data may show OK, the OS information is missing.

KBA: 2813029 – EarlyWatch Alert (EWA): “OS not detected” in Hardware Utilization Data chapter

2: Performance Data Missing

  • The Performance Data column shows “--” for one or more hosts.
  • Hardware capacity checks fail due to missing performance metrics.

KBA: 2813036 – Performance Data missing in EarlyWatch Alert (EWA) report.

See Also

  • 3007580 - OS data incorrect or missing in EarlyWatch Alert (EWA)
