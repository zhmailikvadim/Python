Symptom

This note performs interface changes for Large Document Handling(LDH) for RFx and introduces no additional features

Other Terms

Large Document Handling , LDH, RFX , BUS2200

Reason and Prerequisites

This SAP note alone does not bring any new functionality as a part of its implementation. It only adds the interfaces that would be needed to bring the system on a consistent code base so that support can be effectively provided to other OSS messages.

Solution

Import the corresponding support package or implement the attached advanced correction manually.
