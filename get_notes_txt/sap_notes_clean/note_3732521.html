Symptom

This SAP Note provides an overview and summary of the most important SAP Notes in the context of BW Data Transfer Process (DTP).

New notes will be always immediately included. New errors usually are only relevant for very special situations but however an error in the transformation has always the risk to get somehow inconsistent data therefore we highly recommend to update you system with the new notes on a regular basis. If a new error regarding transformation is found in your system always apply all new corrections that are provided with this Note.

Other Terms

Data Transfer Process (DTP), SAP BW Note Analyzer

Reason and Prerequisites

Any kind of error in the Data Transfer Process (DTP) metadata or runtime.

Solution

As alternative to the SP implementation you can also apply the list of referenced notes. To do so best is follow following steps.

Attached to that note you’ll find zip files following the naming conventions:
SAP_BW_Data_Transfer_Process_<date: yyyy-mm-dd>.zip
e.g. SAP_BW_Data_Transfer_Process_2024-10-02.zip

Use the most current zip file for the note analyzer. The older zip-files are just kept for versioning.

Within the zip you'll find a text file “Z_SAP_BW_NOTE_ANALYZER.txt”. Save the file on your computer and open it in an editor.
In the corresponding system, use transaction SE38 to create a report called “Z_SAP_BW_NOTE_ANALYZER”.
Copy and paste the content of the above-mentioned text file into the report and activate it.

Furthermore you'll find one XML file within the zip.  Save the attached XML file and execute the SAP Note Analyzer (Report Z_SAP_BW_NOTE_ANALYZER).

Click on the icon “Load XML file” and choose the XML file.

Select radio button “Check implementation state against information in XML file” and check checkbox “Download needed SAP Notes”.

Select radio button “In background”.

Run the report.

A background process will be started which downloads the required notes.

When the process finished successfully, execute the report again with a de-selected checkbox “Download needed SAP Notes”.

A list of notes with its implementation status will be shown. You can now install the missing notes by clicking on the corresponding icon in the list

You can find the User Guide for SAP BW Note Analyzer on the SAP Help Portal
https://help.sap.com/viewer/p/SAP_BW4HANA

Also you can read the the attached user guide on the note analyzer.

As alternative to the SP implementaion and the Note Analyzer the attached Excel spreadsheet can be used to filter the relevant notes for a given release and support package.

Please note: Due to technical restrictions, if you are starting from BW/4HANA 2023 SP0, the usage of the Note Analyzer is not recommended and may lead to implementations problems.
