Symptom

You executed a transaction for cost center accrual (option Detail List must be active for some transactions). You display the detail list. You can't export the detail list although the functionality was available in past.

Other Terms

KAZP KSA3 KSA8 excel

Reason and Prerequisites

Program error,  a side effect of SAP Note 3439020 fixing ALV logic. Fix in this  SAP Note is necessary only for SAP_BASIS 7.58 and higher., correction instructions for lower SAP_BASIS releases are not complete. 
Older SAP Note 3552615 solves the issue for overhead transactions, the issue still occurs in cost center accrual transactions.

Solution

Please install the SAP Note or corresponding Support Package.
