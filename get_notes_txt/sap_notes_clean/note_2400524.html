Symptom

Questions were created in the Questionnaire TAB (corresponding with the configured settings), but no text was shown in the application.

If the logon language is different than the language initially used to create the questions, the questionnaire entries are not translated (REQUIRED: The Questions are translated accordingly in the system, or no translation will happen)

Other Terms

Questionnaire, empty text for questions, no translations for questions

Reason and Prerequisites

The source code had a problem in the implementation.

Solution

Please implement the correction instructions or install the corresponding service pack.
