Symptom

When you run report RPUSADVF0, the system displays a misleading title on the selection screen: "Sélection des salariés pour l'acquisition du taux PAS".

Other Terms

RPUSADVF0, selection screen, report title

Reason and Prerequisites

The system uses an incorrect report title.

Solution

After implementing this SAP Note, the system displays the correct title on the selection screen of report RPUSADVF0: "Sélection de nouveaux matricules pour une DSN SADV".

The correction included within this SAP Note is part of the HR Support Packages referenced below.
If you need the correction before the HRSP is available, implement the attached correction instructions. Note that correction instructions are a short-term solution and not a substitute for the Support Package.
