Symptom

Report CFG_DOCUMENT_PUBLISHER fails to replicate documents related to a configurable product to the FTP server.

Other Terms

CFG_DOCUMENT_PUBLISHER,  COM_CFG_DB_PUBLISH_DOC_FTP, FTP_GET_FILE_DESCRIPTORS, Knowledgebase

Reason and Prerequisites

Program error.

In form routine FTP_GET_FILE_DESCRIPTORS information about the files that are to be replicated is processed incorrectly. This can lead to duplicate deletion requests for a given file which causes the replication to fail.

Solution

Correction in ABAP function module COM_CFG_DB_PUBLISH_DOC_FTP.
