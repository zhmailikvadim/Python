This SAP Note provides you with corrections to the following reports:

  • EFD ICMS IPI (/TMF/EFD) report:
    • Register 0450: The use of the Define Additional Information Code (/TMF/D_COD_INF_D) table is not working properly to get the text description from the Additional Information Code (COD_INF) field.
    • Register 1360: The EFD_1360 view is being filtered by run date. However, the view should not be date-filtered.
    • Register C591: The EFD_C591_014 view is considering the SORTER field from the EFD_C590_014 view. Consequently, the register is reported out of order.
  • General:
    • Table NF Item ICMS-ST indemnity (/TMF/D_NFI_RESS): You cannot create an entry on the /TMF/D_NFI_RESS table with different values for the NF Item Number (NUM_ITEM_ULT_E) field.

Other Terms

TDF, Brazil, SPED, EFD ICMS IPI, 0450, C591, /TMF/D_NFI_RESS.

Reason and Prerequisites

This SAP Note is relevant only for Brazil.

You have implemented SAP Note 3713733.

You have upgraded your system to SAP TDF Support Package 20.

Solution

After implementing this SAP Note, the updates listed below are made to your system:

  • /TMF/EFD report:
    • Register 0450: The J_COD_INF join is updated to consume the ctr/NF_INF_COMPLEMENTAR view and the SPED EFD: Store COD_INF during report execution (/TMF/D_COD_INFO) table. As a result, the register is generated with the correct values for the COD_INF field from the ctr/NF_INF_COMPLEMENTAR view.
    • Register 1360: The date filter from the P_LACRE projection is removed. As a result, the system now considers all data from the ctr/COMBUSTIVEL_LACRE_BOMBA view and reports the register correctly.
    • Register C591: The new EFD_C591_017 view was created consuming the EFD_C590_017 view. As a result, this corrects the SORTER field and ensures that the C590 view reports properly.
  • General:
    • The NF Item Number (NUM_ITEM_ULT_E) field is flagged as a key of the /TMF/D_NFI_RESS table. As a result, you can create entries with different values for the NUM_ITEM_ULT_E field.

As a general rule, SAP recommends that you implement the Support Package specified for your release. However, if you need to install a solution earlier, use the SAP Note Assistant to implement the correction instruction. You can find more information about the SAP Note Assistant in SAP Support Portal, under SAP Note Assistant.
