Symptom

When using the file.format.dsv or file.format.json component in a dataflow, the generated file includes both date and time in the name using the ${now} dynamic value placeholder.

Environment

  • SAP Customer Data Cloud
  • Dataflows (IdentitySync)

Reproducing the Issue

  1. Use the file.format.dsv/file.format.json component in a dataflow to generate a file.
  2. Specify the format using ${now} for the file name.
  3. Observe that the generated file contains both date and time, instead of only the date.

Cause

Resolution

To generate a file containing only the date in the filename, specify the format ${now:yyyyMMdd}.csv/${now:yyyyMMdd}.json in the file.format.dsv/file.format.json component. 

For example, if today is 2026/02/24, then the filename is "20260224.json".
