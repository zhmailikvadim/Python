Symptom

A business operation payoff should be executed.

After approval the process stops with an error message, the reprocessing is not possible.

Reason and Prerequisites

This problem is caused by a program error.

Solution

Implement the attached program corrections or import the
relevant Support Package.

After an error message, the steps already performed are not reset; the current processing status is saved in BO data.
Reprocessing can be started at the point where the error occurred.

The work item remains in status 'In process' and is accessible in the inbox of approver.
