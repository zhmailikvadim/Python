Symptom

The registration deadline for a course is ignored during enrollment via the program.

Environment

SAP SuccessFactors Learning

Reproducing the Issue

  1. Create a course with an expired registration deadline.
  2. Add the course to a program.
  3. Attempt to register for the course through the program using the "ENROLL" option.
  4. Observe that the registration is recorded despite the expired registration deadline.

Cause

Known Issue

Resolution

The Learning engineering team has addressed the issue by correcting the backend code.

  • The fix for this issue is planned to be deployed on 1H 2026 release.
  • For release timelines, please review SAP SuccessFactors Product Release & Road Map Information.
