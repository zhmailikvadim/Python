Symptom

Asset transfer processed at an exchange rate that does not align with the defined M or ME rates.

Environment

  • FIN Asset Accounting​
  • SAP S/4HANA ​
  • SAP S/4HANA Finance​
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Perform an Asset Transfer (transaction ABT1N) from one Company Code to another (e.g., from Company Code XXXX to YYYY).
  2. Check the exchange rate applied during the transfer.
  3. Notice that the exchange rate does not align with the defined M or ME rates for the specified period.

Cause

Exchange rate was manually informed by user.

Resolution

  • Check tCode EWCT informing dates, source and target currency to see if the value is aligned.
  • If it shows correctly, confirm field HWMET in table BSEG, if it contains 'E', it means that the translation were not determined by OB08/TCURR, but was instead provided manually by the user, either as the rate or the value itself.
