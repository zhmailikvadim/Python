Symptom

  • Error message: "Method INIT_MAPPING~SELECT_EDOMXCUSTPARAM does not exist" when implementing DRC Mexico SAP notes or when running a syntax check on method INIT_MAPPING.

Environment

  • SAP ERP
  • SAP S/4HANA
  • SAP Document and Reporting Compliance service

Reproducing the Issue

Error happens in transaction SNOTE when implementing DRC Mexico SAP notes or when running a syntax check on method INIT_MAPPING.

Cause

Missing relevant SAP notes from SAP DRC Mexico.

Resolution

  1. Verify the implementation status of SAP Note 3167391 - eDocument Mexico - Annex 20 V4.0: Full Solution or SAP Note 3167387 - eDocument Mexico - Annex 20 V4.0: Basic Solution.
  2. Ensure that all manual steps outlined in the SAP Note have been completed.
  3. Both the SAP notes contains the highlighted method, it depends on which version of DRC Mexico is being used.

See Also

Refer to KBA 2523263 - eDocument Mexico Troubleshooting Guide
