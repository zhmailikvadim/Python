Symptom

An error message is displayed when the user attempts to complete their data during the onboarding process - externalCode is a required field/association in Object X

Environment

SAP SuccessFactors Onboarding

Reproducing the Issue

  1. Attempt to complete the onboarding data collection process.
  2. Error message occurred upon submission: externalCode is a required field/association in Object X

Cause

The Subject User Field in the custom task from Configure Object Definition is set to "none".

Resolution

  1. Access the Configure Object Definition
  2. Locate the custom task
  3. Set the Subject User Field to "externalCode".
  4. For all other custom tasks with custom MDF objects, ensure the Subject User Field is set to "externalCode" as outlined in the guide.

See Also

Creating a Custom MDF Object for Custom Tasks.
