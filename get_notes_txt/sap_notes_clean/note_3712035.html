Symptom

This SAP Note provides technical corrections that are relevant for the Determinación de la duración de un contrato (HR_E_GET_CONTRACT_DURATION) function module performance.

Note:

The implementation of this SAP Note has no functional impact on the program's behavior. However, it might be used as a prerequisite for future developments or corrections.

Solution

The Datos Fiscales (IT0062) Datos fiscales infotype table is added to the HR_E_GET_CONTRACT_DURATION function module parameters to optimize the data retrieval when it is already available in its usage context.
