Symptom

The SAP Note 3711219 (Enablement of CBS and IBS Tax Situation in MASS Transaction) requires this SAP Note to be implemented as a prerequisite.

Other Terms

Legal Change, Brazil, Mass, CBS, IBS, Tax Situation, VBAP

Reason and Prerequisites

This SAP Note is relevant only for Brazil.

This SAP Note contains a report and automatic correction instructions to create/modify objects that are used in the solution delivered via SAP Note 3711219.

Solution

After you implement this SAP Note, the system includes the Data Dictionary (DDIC) objects required to enable the Tax Situation for CBS and IBS field in the Mass Maintenance transaction.

You can receive some object either via automatic correction instruction (aCI) or the new report NOTE_3711218. The delivery method depends on your SAP_BASIS level.

Use the Note Assistant (SNOTE) to implement the correction instructions. You can find more information about the Note Assistant in SAP Support Portal, under https://support.sap.com/en/my-support/knowledge-base/note-assistant.html.

IMPORTANT

  • This report serves only as a tool to generate the required objects automatically. Therefore, it has no positive or negative effect to transport the report NOTE_3711218 to your production system.

  • The objects created by the execution of this report can only be used after you apply the main SAP Note that delivers the solution.

List of Affected Objects:

  • Structures
    • BAPISDITM
    • BAPISDITMX
    • VBAPKOM
    • VBAPKOMX
