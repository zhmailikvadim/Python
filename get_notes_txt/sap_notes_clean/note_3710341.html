Symptom

When trying to perform actions in SAP API Management, it doesn't work even when the needed roles are assigned to the user.

An error similar to the one below might appear in the UI:

Possible error in role assignment. If you are assigned with Admin role, it is necessary that no other role is assigned to you; Check role assignment.

Environment

SAP API Management

Cause

APIPortal.Guest role collection conflict with other SAP API Management roles.

Resolution

If the affected user is meant to have other API Management role collections assigned to it, remove the APIPortal.Guest role collection from the affected user.
