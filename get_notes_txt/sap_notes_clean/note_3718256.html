Symptom

Hyperlink from token does not work in job description field in job requisition form .

***Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.***

Environment

SAP SuccessFactors Recruiting Management

Reproducing the Issue

  1. Navigate to a job requisition form
  2. Add a token in the job description field
  3. Add a hyperlink to this token
  4. Post the job requisition to the career site
  5. Observe that the hyperlink displays the HTML tag instead of the correct link

Resolution

This issue will be resolved in the 1H 2026 version.

The new version will be released on the following dates:
-Apr.13 for Preview instance
-May 15 for Production instance
