Symptom

This SAP Note contains information about eSocial event S-2190 Employee Hiring - Preliminary Registration (Admissão de Trabalhador - Registro Preliminar): how to implement it in your system and how to generate it.

Other Terms

eSocial; S-2190

Solution

To implement the S-2190 event in your system, follow these steps:

  1. Implement SAP Notes 2233746 and 2332106, or install the relevant HR Support Packages referenced in the SAP Notes.
  2. Follow the instructions described in document "Manual_Instructions_SAP_Note_2347618.pdf" that is attached to this SAP Note. Note that:
    • This step of the solution is not delivered in an HR Support Package, so you must follow the document instructions if you want to have event S-2190.
    • There are optional instructions in the document for the Personnel and Organization (PA-PAO) component (HR renewal).

After you have the event in your system, to hire an employee and generate the S-2190 event, execute the steps described in document "eSocial_Preliminary_Registration_and_Employee_Hiring.pdf" that is attached to this SAP Note.
