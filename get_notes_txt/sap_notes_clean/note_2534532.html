Symptom

You create a Sales Order with one or more Third Party Items (item category TAS) with Order Related Billing Relevance (FKREL = F) and Update Group = 12.

You invoice one of the Third Party Items.

The remaining items Invoice Receipt Status is "No Invoice Receipt".  The Billing Status for these items is currently not relevant for billing (VBAP-FKSAA = ' ').

You expect a Type 100 commitment from the remaining non-invoiced items but there is no Type 100 commitment for this order.

Other Terms

Third Party Items, TAS, FKREL=F, Billing Status of Third Party Item, Update Group=12

Reason and Prerequisites

This is a program error.

Solution

Implement the attached correction instructions or import the relevant Support Package to correct this error.
