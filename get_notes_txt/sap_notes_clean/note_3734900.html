Symptom

During Inbound Yard Order (IYO) completion there is no possibility to copy data to the Outbound Yard Order (OYO).

Solution

BAdI /SAPYL/YO_ADJ_DATA_ON_CMPLT provides ability to copy data from IYO to OYO.

You can use this BAdI to adjust data on Yard Order completion, specifically to copy data from an Inbound Yard Order (IYO) to its corresponding Outbound Yard Order (OYO). The Fallback implementation copies Driver's data as example.
