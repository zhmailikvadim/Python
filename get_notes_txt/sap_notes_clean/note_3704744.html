Symptom

This SAP Note provides you with corrections to the following issues regarding the Simplified PDF application:

    • When users upload the simplified file consecutively in the same application session and do not click the 'Update' button after the first upload, duplicate records are created in certain country-specific infotypes medical expense(0812), donation expense(0858) and expense information(0881). The error occurs when maintain the relevant infotypes' eligibility in V_T582ITVCLAS is either:
      • No table entry
      • Eligibility is set to Not Permitted (0) or Permitted in Certain Circumstances (2)

    Other Terms

    Simplified PDF.

    Reason and Prerequisites

    This is a program error. You must apply SAP note 3702607 together.

    Solution

    You can choose one of the following solutions:

    • Implement the relevant HR support package or Local Version Legal Change (CLC) package
    • Implement the relevant correction instructions in this SAP Note.
