Symptom

  • Requirement for a Custom Task with upload attachment field for Hiring Managers and have the attached file be send in an email notification to the candidates.
  • Is it possible for the uploaded file in Custom Tasks of Hiring Managers to be included in an email notification?
  • Configure a Hiring Manager custom task with upload attachment field following the steps in KBA 3460531 - Enhancements to Custom Tasks for Onboarding and Offboarding, then configure a custom email trigger and template to include a static document of the uploaded file from the custom task, is this feasible?

Environment

SAP SuccessFactors Onboarding

Resolution

In the current Onboarding feature, this isn't possible. The custom tasks do not support this as of now.

  • The uploaded attachment is a static document which will be stored in DMS. However for each candidate, they will upload his/her own document.
  • The static document must already exists in DMS, so that we can select it in the rule (see Configuring Business Rule for Static Documents as Email Attachments). However in this case, the rule for static document in the custom task email trigger wouldn't let us select the document that was uploaded by the candidate in the custom task.

If this is deemed necessary for the business, you may submit an enhancement request for Product Management review in the upcoming or future releases.
