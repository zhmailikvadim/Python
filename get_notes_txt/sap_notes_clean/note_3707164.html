Symptom

There is an error in the ABAP Dictionary Activation. If parallelization is disabled, the package building (for reduction of memory consumption) in activations ends without handling all necessary objects, leading to inconsistencies in dependent objects after the activation.

Other Terms

CBDA, In-Memory Activation, DDIC, Dictionary, Upgrade

Solution

Implement the correction instructions of this note and for following activations, the error won't occur anymore. Existing inconsistencies still need to be fixed in the affected system.
