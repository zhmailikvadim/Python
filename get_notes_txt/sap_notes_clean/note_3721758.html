Symptom

  • /416 gross income is projected wrongly when payroll is not run for a period
  • AFY: /416 gross income is projected wrongly when payroll is not run for a period
  • Symptom holds good for AFY as well i.e. Alternate Financial Year.
  • Wage types are cumulated in an incorrect cumulation interval.

Environment

  • Human Capital Management (HCM)
  • Payroll India
  • SAP R/3
  • SAP ERP Core Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA

Reproducing the Issue

  1. Employee belongs to AFY (Alternate Financial Year). 
  2. Use T-Code PC00_M40_CALC and run payroll in live till period 10 2025 (December 2025) as an example.
  3. Skip processing for period 11 2025 i.e. January 2026.
  4. Directly run payroll for period 12 2025 (February 2026).
  5. Observe the incorrect behaviour in the Gross Summary projection (/416) in the current month. 

Resolution

Implement SAP Note 3621259 - SETCU: Withdrawal of SAP Note 3563402

Keywords

/416, CRT /124, Gross total income incorrect, AFY, A.F.Y, Alternate Financial Year
