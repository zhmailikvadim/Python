Symptom

When trying to use a mathematical formula in a calculated column, for example MIN(), the following error message appears:

"The function "MIN" cannot be used in the formula as it is not supported by the system"

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

SAP SuccessFactors HCM Suite

Reproducing the Issue

  1. Create a Story Report
  2. Go to the Query Designer
  3. Create a calculated column and try to use a mathematical formula, error message will appear

Cause

This is an expected behaviour. Mathematical functions are only available in the Calculation Editor (tables and charts) and not in the query designer.

Resolution

To use mathematical formula, the calculated column have to be created in either a table or chart.
Note: Calculated columns are only available in Cross-Tab table and not in List table.
