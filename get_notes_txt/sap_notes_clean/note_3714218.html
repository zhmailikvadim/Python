Symptom

There are very rare cases found, where the page cache creation caused a dump due to improper system status. To avoid full dumps we introduced a mechanism to fetch these dumps and to put the information on the issue to the log. So the next run of the synchronization should be able to synchronize the remaining data

Other Terms

dump, st22, page cache, UI2/PAGE_CACHE_SYNCHRONIZE, /UI2/SYNC_PBC

Reason and Prerequisites

insensitive coding

Solution

apply attached fix
