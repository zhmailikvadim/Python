Symptom

In SAP Intelligent Clinical Supply Management, while releasing the PCN via Clinical Order Application, sometime system fails to release the PCN due to any locking issue, however system updated the Medication number with PCN UUID link. Later when system releases the PCN the UUID links changes however as the Medication number is already assigned with old PCN UUID, system could not able to find the PCN linkage.

This correction report will correct the PCN UUID link to Medication Numbers.

Other Terms

ISLS, Intelligent Clinical Supply Management, Manufacturing, Release PCN,Medication List, PCN UUID,

Reason and Prerequisites

You use SAP Intelligent Clinical Supply Management or this SAP Note is a prerequisite of another SAP Note.

Solution

Execute the correction report
