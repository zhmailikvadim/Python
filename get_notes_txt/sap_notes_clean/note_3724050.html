Symptom

After installed VSS 7.0 with product version S4HANA_2025
While invoicing a VSS order, there is a warning message: Define settlement parameters for order type 2000 in Customizing. Message No. /DBE/CO021
This means the settlement rules have not been generated automatically although the customizing has been configured correctly (Controlling > Internal Orders > Actual Postings > Settlement > Automatic Generation of Settlement Rules)

Other Terms

VSS, no Settlement rule is defined, automatic Settlement rule

Reason and Prerequisites

Handling of this situation is due to a program error.

Solution

If the settlement rule has not been generated automatically - as a workaround - it can be posted manually.
1. Go to transaction KO02
2. Go to 'settlement rule' and save the Internal Order
3. Settlement rule will be triggered

If the automatic settlement rule generation is required, please apply SAP note # 3716349 - Automatic Settlement rule generation error for Sales order and Internal Order
