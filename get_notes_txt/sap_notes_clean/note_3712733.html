Symptom

There is no data generated in the Thailand PP36 VAT Return, even though the status shows as generated successfully in the app Run Statutory Reports.

Environment

SAP S/4HANA Cloud Public Edition.

Reproducing the Issue

  1. Open the app Run Statutory Reports.
  2. Fill out the report entity field as Thailand VAT Reporting Entity.
  3. Choose a reporting period.
  4. Click on New Run.
  5. The run status shows generated successfully, but no data is availabile.

Cause

The tax code for the journal entry was not maintained within the tax group.

Resolution

Check the tax code corresponding to the journal entry.

In the configuration activity 'Define Enhanced Tax Box Structure,' maintain the tax code for the tax group, and ensure the correct maintenance of the relationship between the tax group and the tax box.

Keywords

PP36, Thailand VAT Reporting Entity, tax code, statutory report, no data
