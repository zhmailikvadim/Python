Symptom

The "Inspection setup" field in the Quality Management view of the material master is not ticked in display mode (t-code MM03), however upon opening the "Inspection setup" section, the field appears ticked automatically.

Environment

  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Access transaction MM03.
  2. Enter the material number.
  3. Select the Quality Management view.
  4. Enter Plant, observe that the "Inspection setup" field is unchecked.
  5. Select the "Inspection setup" section.
  6. Click OK/press Enter.
  7. Observe that the "Inspection setup" field is now flagged.

Cause

The issue is caused by a data inconsistency.

Resolution

  • Implement SAP Note 3639815 to prevent the issue from occurring in the future.
  • Correct existing records by opening the affected material/plant combinations in transaction MM02 via the Quality Management view and saving the changes.
