Symptom

In case of ECC initiated PO creation, Service entry sheet 'GR despite IR' logic (ERS) did not find correct PO number. PO for ERS ECC initiated should be taken from the IDOC data.

Other Terms

/ASADEV/CL_ACI_SES_MULTI_LINE, _CHECK_SES_GR_DESP_IR.

Reason and Prerequisites

Missing PO search logic

Solution

Install this OSS note to fix ERS logic for ECC initiated POs.
