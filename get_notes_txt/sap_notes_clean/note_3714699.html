Symptom

You declare your exports from SAP GTS to the German customs system AES 3.0.

You can only specify an authorization of the type C514 (local clearance in accounting) or C513 (central customs processing) by specifying a header document of the category "Submitted Document" (HDOC) with the number of the authorization.

Therefore, if the authorization is entered on the document tab page at header level, this is taken into account only in the case of an authorization for the simplified procedure of the type C512.
