Symptom

EWA report shows the following alert: "Enhanced hardware monitoring for virtualized environments is not working for some of the current hosts".

Environment

  • Windows operating system
  • Virtualization: Nutanix AHV 

Reproducing the Issue

  1. Generate an EWA report for a Windows-based landscape running on Nutanix AHV.
  2. Observe the alert "Enhanced Hardware Monitoring in Cloud Environments" for all relevant hosts.

Cause

  • Windows operating systems on Nutanix AHV are not supported.
  • AHV support in this context applies to Linux only.
  • For Windows, supported virtualization platforms are Microsoft Hyper‑V and VMware vSphere.

Resolution

Perform migration to the supported environment as described in SAP note 1492000.

See Also

  • SAP note 1492000 - General Support Statement for Virtual Environments
