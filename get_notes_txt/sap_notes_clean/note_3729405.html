Symptom

Off-cycle payments may be posted to different accounts than regular payroll. To keep posting accurate, configure additional wage types

Reproducing the Issue

- The posting is being performed for the off-cycle payments; 

- Different accounts are used for the posting runs of off-cycle payments and regular payroll payments. 

Resolution

1. Create wage types

  • Start transaction OH11.
    • Choose a method of maintenance: Copy                 
    • HCM Localization: 36 Ukraine
  • Enter the following entries:

O710 Alimony         Alimony    I710 Alimony         Alimony

O720 Fine            Fine       I720 Fine            Fine

O750 Payment to post Post       I750 Payment to post Post

OGIP Fine            Fine       IGIP Fine            Fine

OGIS Alimony         Alimony    IGIS Alimony         Alimony

  • Set Test Run is empty.
  • Select all entries.                                                                                                          
  • Press button Copy.

2. Start transaction SM30. 

  • View V_T51P6;
  • Press button Edit;
  • HCM Localization 36;
  • Select the following entries:

I710         Alimony 01.01.1900            31.12.9999            A               3               1

I720         Fine         01.01.1900            31.12.9999            B               3               1

I750         Payment to post                  01.01.1900            31.12.9999            A               1               4

IGIP         Fine         01.01.1900            31.12.9999            B               2               4

IGIS         Alimony 01.01.1900            31.12.9999            A               2               4

  • Press button Delete.

3. Start Transaction SM30

  • View V_T51P1;
  • Create the following entries:

OC           O710      Alimony 01.01.1900            31.12.9999            I710

OC           O720      Fine         01.01.1900            31.12.9999            I720

OC           O750      Payment to post                  01.01.1900            31.12.9999            I750

OC           OGIP      Fine         01.01.1900            31.12.9999            IGIP

OC           OGIS       Alimony 01.01.1900            31.12.9999            IGIS

4. Start transaction SM30

  • View V_512W_O;
  • HCM Localization 36;
    • For wage types O710, O720, O750, OGIP, OGIS: set value 5 of processing class 74

    • For the wage types I710, I720, I750, IGIP, IGIS: 
      • Set value 2 of processing class 74;
      • Delete Cumulation 10;
      • Clear Value in evaluation class 02.
