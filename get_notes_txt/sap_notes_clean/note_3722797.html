Symptom

  • In SAP GUI transaction VF11, users can specify a 'Reversal Reason' (field VBRK-STGRD) during cancellation.
  • In Fiori App F0797, the cancellation action works, but user is unable to provide 'Reversal Reason' for cancellation document.

Environment

•Sales And Distribution (SD)
•Logistics Execution (LE)
•SAP R/3 •SAP R/3 Enterprise 4.7
•SAP ERP Central Component
•SAP ERP
•SAP Enhancement package for SAP ERP
•SAP Enhancement package for SAP ERP, version for SAP HANA
•SAP S/4HANA
•S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Open GUI transaction VF11 and observe the cancellation screen with the 'Reversal Reason' field (VBRK-STGRD).
  2. Open Fiori App F0797 and attempt to cancel a billing document.
  3. Observe that there is no field or parameter to capture a 'Reversal reason'.

Cause

The standard Fiori App F0797 (Manage Billing Documents) does not include functionality to set a cancellation reason during billing document cancellation.

Resolution

  1. Verify that the standard Fiori App F0797 does not support setting a 'Reversal Reason'.
  2. Refer to SAP documentation for embedding SAP GUI HTML transactions as Fiori apps to enable cancellation with a cancellation reason.
  3. Use SAP GUI HTML transactions, such as VF11 or VF02, embedded as Fiori apps to perform cancellations with a specified cancellation reason.
