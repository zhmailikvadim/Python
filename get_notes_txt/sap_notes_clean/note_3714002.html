Symptom

If an erroneous asynchronous outbound proxy message is deleted only on the proxy side, it is not possible to cancel it from /AIF/ERR. However, with this note, it is allowed to cancel it from /AIF/ERR.

Other Terms

Asynchronous outbound proxy message, /AIF/ERR, cancel messages

Reason and Prerequisites

This is a code defect.

Solution

Implement attached correction instruction or corresponding support package.
