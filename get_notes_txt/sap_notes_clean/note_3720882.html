Symptom

Despite activating the New Version Management for Purchase Contracts, the Version History tab is still not displayed in the Manage Purchase Contracts app.

Environment

  • Materials Management (MM)
  • SAP S/4HANA 2023
  • SAP S/4HANA 2025
  • SAP S/4HANA Cloud Private Edition 2023
  • SAP S/4HANA Cloud Private Edition 2025

Reproducing the Issue

  1. Activate the New Version Management for Purchase Contracts
  2. Open the Manage Purchase Contracts
  3. Create a new purchase contract
  4. Observe that the Version History tab is not displayed.

Cause

  • The old Version Management is still activate. In Fiori, only the New Version Management is supported.
  • Missing customizing. 
  • Purchase contracts were not created in Fiori. 

Resolution

Ensure that the old Version Management is disabled for the relevant purchase contract document type in the OLME transaction > Version Management > Set Up Version Management for External Purchasing Documents.

The New Version Management is supported for the scenarios described in this SAP Help Page: Version Management in Purchase Contracts.
