Symptom

  • Error message: "Entry Pricing Person for RoleCode field not consistent as per code list restriction. Please contact your system administrator."
  • The error occurs when attempting to add a user to the Sales Team under an Opportunity with a specific RoleCode.
  • Selected value is within user's access restriction.

Environment

SAP Cloud for Customer

Reproducing the Issue

  1. Navigate to Sales -> Opportunities.
  2. Open any Opportunity.
  3. Navigate to the Sales Team facet and click on "Add".
  4. Select the RoleCode "Sales Contract Support".
  5. Select the same user and click "Save".
  6. Observe the error message: "Entry Pricing Person for RoleCode field not consistent as per code list restriction. Please contact your system administrator."

Cause

The error occurs due to the affected employee does not have an Organizational Assignment maintained in the master data, which prevents the system from determining the employee's context.

Resolution

  1. Maintain the Organizational Assignment for the affected employee in the master data to resolve the issue.
  2. Alternatively, consider the following options to address the inconsistency:
    a. Delete the related Code List Restriction.
    b. Update the "Suppress Message" flag for message ID 015 (CM_MSG_CLR) in the Business Configuration work center to prevent the inconsistency warning from appearing.
