Symptom

Symptom

Note created to revert the changes of note 2992029.

Other Terms

GMTUK, GDT time zone

Reason and Prerequisites

Program error.

Solution

Implement the attached correction instructions or import the specified Support Package.

The note carries the following corrections:

1. The unique generated ID for analytical applications contains only the key fields if all key fields are requested.

2. No data request is sent to the database if the client requests no entries with the option $top=0.

3. Localization for sorting, filtering or aggregation is only done if the application explicitly requests it, or if the sorting, filtering or aggregation elements are character-like.

Solution

Apply the correction instructions.
