Symptom

In the process "Receipt of 55126 from Responsible DSO to SUP" (DE_B_MDC_55126_DSO2SUP_RECV), the existing value of field /US4G/S_UBD_SETTLE_MALO-TIME_SERIES_TYPE is not filled correctly in the MDX inbound data container list in step 1600-0200. This occurs when handling duplicate cases with meter procedures E02 and E14.This issue is relevant for the market roles: SUP, DSO, MOS.

Other Terms

UCOM, MDX inbound data container, TIME_SERIES_TYPE, meter procedure E02, E14, /UCOM/MDX_UBDKEY, /UCOM/RP_MDX_DATA_CONT_V2, /UCOM/CL_MDX_DATA_CONT_API_V2->MERGE_EXISTING, 55126

Reason and Prerequisites

The error occurred because the system did not correctly populate the existing value of the TIME_SERIES_TYPE field in the MDX inbound data container list.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the system correctly fills the existing value of /US4G/S_UBD_SETTLE_MALO-TIME_SERIES_TYPE in the MDX inbound data container list for meter procedure E24(E02 + E14). You can verify the fix by processing a 55126 message with duplicate meter procedures E02 and E14, and checking that existing vlaue of TIME_SERIES_TYPE is populated correctly in the MDX inbound data container list.
