Symptom

An interface is required to send Inspection Lot changes from SAP S/4HANA to Third-Party system. The interface payload must be populated with additional fields that Third-Party system requires for further processing. If we are not able to send this information as part of the payload, then Third-Party system would not be able to receive the Inspection Lot changes without the correct information

Other Terms

QM, Inspection lot, Events, inspection Lot events, Extensiblity

Reason and Prerequisites

The events of R_InspectionLotTP are exposed to the released projection I_InspectionLotTP_2.

Below are the objects that has to be adjusted to allow events extensibility:

  • Behavior Definition -     R_INSPECTIONLOTTP (Inspection Lot)
  • Behavior Definition -     I_INSPECTIONLOTTP_2  (Inspection Lot)
