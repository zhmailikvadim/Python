Symptom

  • Granting the system privilege ADAPTER ADMIN from a dedicated admin user to another user fails.
  • Indexserver trace shows: "User is not allowed to grant system privilege(s) ADAPTER ADMIN".
  • Indexserver trace shows: "User tried to execute 'grant ADAPTER ADMIN to user'".
  • The granting user has ROLE ADMIN and USER ADMIN assigned.

Environment

  • SAP HANA Platform Edition

Cause

  • The granting user does not have the ADAPTER ADMIN system privilege with the grant option; therefore the user cannot grant ADAPTER ADMIN to others.

Resolution

Verify whether the granting user has ADAPTER ADMIN with grant option by running:

select * from effective_privileges 
where user_name = 'B_USERADMIN' 
and object_type = 'SYSTEMPRIVILEGE'
and privilege = 'ADAPTER ADMIN'; 

Check column IS_GRANTABLE.

For the user<xxx> to be able to assign ADAPTER ADMIN System Privilege to any other user, this user should have this system privilege with the Grant option. Else this would not be possible.

OR

The other way would be to grant the ADAPTER ADMIN Privilege from a user like SYSTEM who has the ADAPTER ADMIN System Privilege with Grant Option assigned.
