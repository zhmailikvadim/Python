Symptom

When searching for Transactions created under Product Type 02A (mutual funds) in the app 'Display Treasury Posting Journal', no data is displayed when Transactions are selected.

Environment

SAP S/4HANA Cloud Public Edition

Resolution

Instead of selecting "Transaction" to display postings, use the filters "Securities Account" and "Security Class" when searching for data related to securities, this is because "Transaction" cannot represent securities details directly, and thus the data appears empty when "Transaction" is selected as a filter.

Keywords

Display Treasury Posting Journal, Securities Account, Security Class, Product Type 02A, Transaction
