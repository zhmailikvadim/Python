Symptom

  • An error occurs when attempting to access "Development" from the home page.
  • Error message displayed: "The requested operation is not available."

Environment

SAP SuccessFactors Career Development Planning

Reproducing the Issue

  1. Login as the affected user.
  2. Navigate to the "Development" tab from the home page.
  3. Observe the error message: "The requested operation is not available."

Cause

It is a known error. The issue occurs specifically when accessing the legacy Career Worksheet sub-module under Development.

Resolution

Product Engineering is investigating a solution and planning to fix in 2605.

Click on star to bookmark this article to receive updates about this issue. For more information about updates from subscribed KBAs, you can refer to the KB Article 2171560 - How to be notified of new or updated SAP Notes or KBAs.

WORKAROUND

For other Development sub-modules, the recommend approach is to access them via the respective deep links e.g. https://performancemanager8.successfactors.com/sf/mentoring

At this time, there is no workaround available for the legacy Career Worksheet.  If Growth Portfolio is enabled in the system, it is strongly recommend using the latest Career Worksheet.
