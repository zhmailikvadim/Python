Symptom

Unable to launch the Transaction Code tile in Fiori within SAP Build Workzone .

Error : 504 Gateway Timeout 

Environment

  • Build Workzone
  • S/4 Hana

Reproducing the Issue

  1. Start the transaction tile in Build Workzone.
    E.g. : Fiori app : Launchpad Content Manager - Client Specific 

Cause

The configured value for the destination parameter "HTML5.timeout" is too short.   

Resolution

Extend the HTML5.timeout parameter in the destination parameter.
