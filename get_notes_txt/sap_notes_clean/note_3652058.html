Symptom

  • The "Upload" button in Trace Analysis does not perform any action after selecting and attempting to upload an XML file.
  • No warning or error message is displayed when the issue occurs.

Environment

  • FRUN: Trace Analysis (TA)

Reproducing the Issue

  1. Open the Trace Analysis application in FRUN.
  2. Select a Business Transaction XML file.
  3. Click on the "Upload" button.

Cause

The issue is caused by a known defect in FRUN 5.0, which will be addressed in the next Support Package (FRUN 500 FP02).

Resolution

  1. Upgrade to FRUN 500 FP02 once it is released, as the issue is resolved in this version.
  2. Note that the Trace Analysis application is deprecated, and it is recommended to use Real User Monitoring (RUM) for similar functionality.

See Also

  • Refer to: KB1263064 - Composite Note for Root Cause Analysis in SAP Focused Run 5.0 SP00.
