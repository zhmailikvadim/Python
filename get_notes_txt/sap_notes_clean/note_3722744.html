Symptom

Error message "Billing documents cannot be cancelled due to differing currencies" encounters when cancelling the billing document.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Access the "Change Billing Documents" app.
  2. Attempt to cancel billing document.
  3. Encounter the error message: "Billing documents cannot be cancelled due to differing currencies".

Cause

  • The cancellation of billing documents whose currency differs from the sales order is not supported when the "cash management"(table: T001-XFDIS) for company code is activated. 

  • The sales order associated with the billing document was created (or changed) with a currency that differed from the billing document's currency.

Resolution

  1. Change the sales order currency to match the billing document currency.
  2. Then attempt to cancel the billing document again.
