Symptom

You have maintained a over 40 characters long text in Inspection Characteristic Remark in Record Inspection Results (F1685A) app. However, when trying to fetch the text via API_INSPECTIONLOT_SRV/A_InspectionResult, the "InspectionResultText" property is limited to 40 characters.

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

SAP S/4HANA Cloud Public Edition

Cause

This is standard design.

Currently, the "InspectionResultText" property can only display short text. The long text is not stored in any released CDS View. Therefore, it's not possible to fetch the long text by Inspection Lot API. It's only possible to verify if it's long text via the "InspectionResultHasLongText" property.

Resolution

If long text needs to be fetched via Inspection Lot API, raise a feature request through the Customer Influence Portal  under the session SAP S/4HANA Cloud Public Edition, Manufacturing.
