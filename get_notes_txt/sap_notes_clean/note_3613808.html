Symptom

You execute transaction FPCVS ("Clarification of Confirmed Payments - Ext. Pymt Coll. CVS").

When you search for unconfirmed payments, you are unable to see all payments because the scroll bar does not work.

Other Terms

FPCVS, FP-CVS, DFKKCVS,

Reason and Prerequisites

Program error

Solution

Implement the source code changes. Implement a manual advance correction or import the Support Package containing the corrections.
