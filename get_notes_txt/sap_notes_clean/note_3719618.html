Symptom

  • RFC connection status metric is not visible in Health Monitoring application of SAP Cloud ALM 

Environment

SAP Cloud ALM

Reproducing the Issue

  1. Launch SAP Cloud ALM
  2. Navigate to the Operations → Health Monitoring application in SAP Cloud ALM
  3. Select the system
  4. Select the 'Communication' tab
  5. The RFC connection status metric is not displayed

Cause

  • This metric requires additional configuration before it will be displayed in the Health Monitoring application. 

  • This metric requires ST/PI 740 SP33 or Higher

Resolution

  1. Open the Health Monitoring Application

  2. Select the configuration icon

  3. Choose the RFC Connection Status metric

  4. In the section 'Metric Configuration', click the + icon to specify the details of the RFC destination(s) you wish to monitor
