Symptom

An issue was observed in S/4HANA system with report S_ALR_87012089, in which the standard icons were missing from the toolbar.

Environment

  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

1. Open report S_ALR_87012089

2. Observe the application tool bar is missing in the report.

Cause

Missing SAP Notes

Resolution

Implement the below SAP Notes:

3466325 - RFKABL00: RFKABL00: missing push buttons in output list
3468596 - ALV: Append Lists, Standard functions and Double Click 
3475188 - ALV list - 'Continue Search' button in PF status not active 

Keywords

3466325, 3468596, 3475188, S_ALR_87012089, Tool Bar, Icons Missing, ALV report.
