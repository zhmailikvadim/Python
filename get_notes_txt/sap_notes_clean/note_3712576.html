Symptom

When accessing IT0016 for an employee under the contract "printing" section you try to download the form with button "Download Form". The path to select where the file will be saved is shown but after pressing, no file is downloaded

Environment

  • Human Capital Management (HCM)
  • SAP R/3
  • SAP R/3 Enterprise 4.7
  • SAP ERP Core Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA 

Reproducing the Issue

  1. Go to PA30
  2. Go to IT0016 for a PERNR
  3. Go to "Printing" section
  4. Generate the contract and push "Download Contract"

Cause

This situation may occur as result of the expiration of the ADS credentials in the system. Note that in this situation no error message is displayed as the ADS service is an independent application.

Resolution

Please follow the indications on SAP note 736902 - Adobe Credentials to check the status of the ADS Adobe Interactive Forms credential in the system and request a new one if needed. Once the credential is active the contract will be generated.
