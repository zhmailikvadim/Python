Symptom

This SAP Note describes an issue that related to the validation of the MBOM input data in Production section of the Production Version.

It happens when the alternative bom entry is empty but the BOM Usage is filled in. So the system tries to validate the MBOM data just by the BOM Usage value.

Other Terms

PP-PEO; Production Version

Reason and Prerequisites

 Program error.

Solution

Apply the attached correction instructions

The system will not validate just partially entered MBOM data.
