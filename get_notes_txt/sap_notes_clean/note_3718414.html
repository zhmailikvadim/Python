Symptom

The text for message ARM_INSP_MESSAGES 046 was 'Inspection of delivery &1 must be created by SAP EWM'.
This might be misleading in certain situations.
In the case of embedded eWM, for example, the inspection gets created automatically by eWM when the Goods Receipt is posted.
So the inspection is already in place before the warehouse inspection is called.

Other Terms

OData service, eWM not supported, decentral WM not supported

Reason and Prerequisites

no prerequisites

Solution

Implement the correction instruction.
The text gets changed to 'Delivery &1: Distributed deliveries are not supported.'
