Symptom

Nominal amount increase/decrease in the swaps contract is not displayed correctly in transaction TI92.

Environment

  • Treasury and Risk Management (TRM)
  • SAP Financial Supply Chain Management
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP version for SAP HANA
  • SAP S/4HANA
  • SAP S/4HANA Finance

Reproducing the Issue

  1. Edit the Swap deal to add other principle decrease or increate and save.
  2. Open t-code TI92 to check the Nominal Incoming or Nominal Outgoing field, it only shows the original nominal amount.

Cause

Transaction TI92 is designed to display the contract data as it was initially conceived, which may differ from the current state of the contract.

Resolution

  1. Understand that transaction TI92 displays the nominal amount based on the initial contract data, while transaction FTR_00 displays the current state of the contract.
  2. Use transaction FTR_EDIT to view the initial contract data on the structure tab and review changes to the nominal amount or conditions display.
  3. Use transaction FTR_00 to view the current state of the contract.
  4. Select the appropriate transaction based on the specific requirement:
  • Use TI92 for initial contract data.
  • Use FTR_00 for current contract data.
