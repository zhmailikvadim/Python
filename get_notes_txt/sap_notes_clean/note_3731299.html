Symptom

When saving a sales order that contains a sales kit (BOM), the system raises an error if items are configured to use both advanced and classic intercompany processes.

  • Classic ICO process for item & and adv. ICO process can't be combined (Message no.: SD_SLS_INTERCOMPANY066)

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Create a sales order that includes a sales kit (BOM).
  2. Configure at least one BOM item to follow the advanced intercompany process and another item to follow the classic intercompany process (for example, via plant or item category settings that drive different intercompany scenarios).
  3. Attempt to save the sales order.
  4. Observe that the system issues message SD_SLS_INTERCOMPANY066 and prevents saving.

Cause

The error occurs because the current item structure contains a mix items for both the advanced intercompany process and the classic intercompany process. This combination of advanced intercompany process and the classic intercompany process is not supported.

Resolution

As outlined in the restriction note for the scope item 'Advanced Intercompany Sales' (5D2), this combination of processes is not supported for sales kits.

Review the item structure and ensure that it contains either items the advanced intercompany process with sell-from-stock or items for classic intercompany process with sell-from-stock. Adjust the item structure accordingly to comply with the supported configurations.
