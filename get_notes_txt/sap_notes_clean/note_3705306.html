Symptom

The Custom Code Analytics (CCA) App currently available via SAP for Me will be sunset by December 31, 2025. Customers using the CCA App for monitoring and managing custom developments across SAP landscapes should prepare for the transition to its successor.

Other Terms

CCA, Sunset, 2025 year end

Reason and Prerequisites

SAP is evolving its extensibility management strategy to align with the Clean Core principles and the RISE with SAP Methodology. The CCA App will be replaced by the RISE with SAP Methodology Dashboard (RwSMD) starting January 2026, accessible via SAP Cloud ALM.

Solution

SAP offers successors for Custom Code Analytics (CCA) App depending on customer needs.

 

  1. SAP Cloud ALM with RISE with SAP Methodology Dashboard

    The RISE with SAP Methodology Dashboard in SAP Cloud ALM gives you an overview of your systems' clean core compliance and provides - amongst others - an overview of different extensibility-related KPIs.
    See https://help.sap.com/docs/cloud-alm/applicationhelp/extensibility for more details.

    The "Business Modification" View of the CCA App is planned to be offered in the RISE with SAP Methodology Dashboard. Timelines will be published once available.

    For more information:

    https://www.sap.com/products/erp/rise.html

    https://help.sap.com/docs/cloud-alm/applicationhelp/rise-with-sap

    Demo:

    https://support.sap.com/en/alm/demo-systems/cloud-alm-demo-system.html

    Additional Information:

    To support this transition, SAP has updated the Service Catalog for Extensibility Management. The catalog includes expert-led services, learning journeys, and direct contact points for support.

    See https://www.sap.com/services-support.html for more details.

Built on SAP Cloud ALM, it brings Premium Engagement expertise into a scalable cloud content subscription, combining SAP Analytics Cloud dashboards with AI supported insights and guided actions to help customers improve SAP feature utilization, reduce operational effort, increase self-sufficiency, and accelerate secure innovation adoption.

Deactivation of Custom Code Analytics with SAP Solution Manager

 

It is suggested to stop the Custom Code collector "Custom Code Analytics".

To do so, in your Solution Manager, go to the "SAP Solution Manager Configuration", section "Custom Code Management".

Navigate to step 3.

Click on the link "Custom Code Analytics Setup" on the top right corner of the table "System Overview".

Select all systems and click the button "Deactivate Selected".

Close the popup.
