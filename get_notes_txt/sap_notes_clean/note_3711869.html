Symptom

This SAP Note delivers Business configuration set - /ACCGO/ACM_SIMPLSNP_CNFG_BC_SET for Simplified snapshots. This BC set must be activated to start using new simplified snapshots.

Other Terms

Agricultural Contract Management; ACM; Simplified snapshot; Contract snapshots; Snapshot BC set; /ACCGO/ACM_SIMPLSNP_CNFG_BC_SET

Reason and Prerequisites

Missing BC set

Solution

Import the relevant support pack or apply the correction instruction(s).
