Symptom

You get following error when you attempted to log on to HANA.

Connect to database failed, rc=1, rcSQL=10
SQLCODE    : 10
SQLERRTEXT : authentication failed: Detailed info for this error can be found with correlation ID '<correlation ID>'
sql error 10     performing CON
authentication failed: Detailed info for this error can be found with correlation ID '<correlation ID>'

Environment

  • HANA Platform Edition 2.0 SPS8 and higher.

Reproducing the Issue

Try to log on to HANA

Cause

Starting with HANA 2.0 SPS8, authentication failure would be recorded in system view AUTHENTICATION_ERROR_DETAILS.
You may query this view to get detailed authentication failure information.

Resolution

You need to log on with another user who has either the system privileges CATALOG READ or SELECT on system view AUTHENTICATION_ERROR_DETAILS.
Then run following SQL to check the results:

SELECT * FROM AUTHENTICATION_ERROR_DETAILS WHERE CORRELATION_ID = '<CORRELATION_ID>'

It could shows you detailed reasons, e.g. User <user name> does not exist, Wrong password, etc.
