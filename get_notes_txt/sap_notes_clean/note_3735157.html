Symptom

Why is the Delete option for repeatable sections is not visible when the supplier user revises the registration questionnaire after the supplier is registered?

Cause

The site configuration parameter Application.SM.DisableRepeatableSectionDeleteOnceRegistered is set to Yes, which hides the Delete option for repeatable section answers (such as bank/tax accounts) after a supplier reaches Registered status.

Resolution

Log a case with Ariba Support to set the site configuration parameter Application.SM.DisableRepeatableSectionDeleteOnceRegistered to No.
