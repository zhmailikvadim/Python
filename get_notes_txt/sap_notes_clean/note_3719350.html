Symptom

Error message: "File contains element <ClrSysMmbId> that is not defined in the format" encountered in the Manage Incoming Payment Files app.

Environment

SAP S/4HANA Cloud Public Edition

Cause

  • The format mapping used during import does not contain this field.
  • The format mapping used during import contains this field, but under an incorrect node.

Resolution

  1. Compare the structure of the bank statement file with the format mapping.
  2. Identify discrepancies between the bank statement file structure and the format mapping.
  3. Update the format mapping to include the element to align with the structure of the bank statement file.
