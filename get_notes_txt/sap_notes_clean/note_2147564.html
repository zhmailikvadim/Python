Symptom

Please apply this note when required during configuration of SAP hybris Marketing

This note is also required if you are experiencing one of the following errors.

- 2125159 Email Marketing: Extend XS-role for technical user

- 2151191 Campaign Content: Unicode encoded characters corrupted

Other Terms

SAP CEI, CEC, Marketing Engagement, SAP_MEM, SAP_CUAN, Kampagne

Reason and Prerequisites

Modeling error in SAP HANA delivery unit (DU) SAP_MEM or Errors in the XS coding relevant for Email sending.

Solution

Apply the attached correction instruction, which will install the latest version of DU SAP_CUAN. In list of symptoms fixed by this note (see above), you find the version number of this DU in which the fix is included.
