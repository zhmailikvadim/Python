Symptom

You download the DSN anomalies Excel file from the B2A Dashboard to the application server. The logical filename, HR_FR_DSN_DSN_ANO_XLS, is configured in transaction FILE for this purpose. However, the physical file name does not appear, and the corresponding field remains blank on the selection screen.

Other Terms

DSN, B2A

Reason and Prerequisites

The parameter that contains the physical file name on the selection screen is not refreshed.

Solution

After implementing this SAP Note, the system correctly refreshes the parameter that contains the physical file name.

These corrections are available with the HR Support Packages referenced below.
If you have an urgent need of these corrections, follow the correction instructions provided.
