Symptom

Error AM056 occurs when attempting to update the address of a Business Partner in transaction BP :

  • "Specify a valid address use (ADDRESS_REFERENCE)" - Message no. AM056

Environment

  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition
  • SAP NetWeaver

Reproducing the Issue

  1. Run transaction BP
  2. Open a Business Partner
  3. Change the address
  4. Save it

Cause

Error AM056 can be caused by a mismatch between ADDRESS_GUID in the table BUT020 and the ADDRESS_GUID in the corresponding ADRV entry in the APPL_KEY field. The APPL_KEY field in this case should be a combination of client number + BP number + ADDRESS_GUID.

Resolution

The error can be resolved by copying, creating and executing the Z report available in note 3591492.

The report resolves the mismatch between ADDRESS_GUID in the table BUT020 and the corresponding ADRV entry.
