Symptom

In the futures account master data (TPM4), you copy an existing futures account in company code A in order to create a new futures account in a different company code B. Even though the bank details are not valid for company code B, this is not revealed by a check and you are able to save the data. Moreover, the value help only returns the values that are available in company code A.

Other Terms

F4 help, TPM4, LCL_POS_ACCOUNT_OBJECT

Reason and Prerequisites

Programming error

Solution

Implement the attached correction instructions or the relevant Support Package.
