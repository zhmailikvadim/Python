Symptom

As the physician, you create and save a medication order in Smart Medication.

Later, you select the saved order in the "Drafts" list for further processing.

You change the start of the order and try to activate the order by choosing the "Continue to Active" pushbutton.

The system does not activate the order, but instead displays the following error message:

Order xxxxx: Start time of an order in "Active" status cannot be changed

Other Terms

NWBC, Smart UI

Reason and Prerequisites

Program error

Solution

As a solution to the described problem. install the SAP Note 2734265.
