Symptom

You are using ABAP Data Integration to replicate the data into SAP Business Data Cloud. You are experiencing errors in Data Transformation when installing the Data Product in BDC:

"Primary keys are required but missing in runtime data CSN for datasource XXX..."

Other Terms

S/4HANA, ABAP Data Integration, SAP Business Data Cloud, APRS, CSN

Reason and Prerequisites

The ABAP-based source system is an S/4HANA 2025 SP01 system.

Solution

Implement the corrections in the ABAP-based source system.
