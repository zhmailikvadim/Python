Symptom

Additional Quantity 1 is not available for selection as a Variable Portion Type in Receiver Basis during the configuration of Top-Down Distribution in the Manage Allocations app.

Environment

SAP S/4HANA Public Cloud Edition

Reproducing the Issue

  1. Open the Manage Allocations app in SAP S/4HANA Cloud Public Edition.
  2. Configure a Top-Down Distribution (TDD).
  3. Navigate to Receiver Basis → Variable Portion Type.
  4. Attempt to select the dimension Additional Quantity 1 as the Variable Portion Type.
  5. Observe that QUANT1 – Additional Quantity 1 is not available for selection.

Cause

Resolution

For QUANT1, QUANT2, QUANT3 please activate the usage for these 3 additional quantities, then select the additional quantity field as a variable portion type using the Additional Reference Key Figure in the TDD template configuration.

Refer to FAQs for Top-Down Distribution in Universal Allocation | SAP Help Portal for more information.
