Symptom

When you execute non-FI-CA report or transactions integrated with Cash Application, the execution time may very long and system job log displays the FI-CA error message "Client &1 does not contain an active application area" and the job is canceled. This issue occurs when FI-CA (contract account) is inactive in your system.

Other Terms

Cash Application, contract account, class CL_FAR_PA_ML_ENRICH, method IF_FAR_PA_ML_ENRICH~FICA_ENRICH_KEY, FKK_GET_APPLICATION, >0149

Reason and Prerequisites

The method IF_FAR_PA_ML_ENRICH~FICA_ENRICH_KEY does not catch exceptions from FKK_GET_APPLICATION.

Solution

To resolve this issue, implement this SAP Note or import the relevant Support Package.
