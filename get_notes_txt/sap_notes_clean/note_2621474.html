Symptom

A new language "Ukrainian" has been added in SAP Business One 9.3 PL03.

Reproducing the issue

  1. Open the General Settings form (Administration → System Initialization).
  2. On the Display tab, check the values in Language.

Expected behavior:

  • The UI displays Ukrainian as one of the values for Language. 
  • You can set Ukrainian as your Language option.

Actual behavior:

There is no value 'Ukrainian' displayed on the UI.

Cause

Legal requirement

Solution

SAP intends to provide a patch or patches in order to enable this requirement in SAP Business one. The section Reference to Related Notes below will list the specific patches once they become available. Any corresponding Info files of the patches in SAP Service Marketplace will also show the SAP Note number. Be aware that these references can only be set at patch release date. SAP will deliver patches only for selected releases at its own discretion, based on the business impact and the complexity of the implementation.
