Symptom

When opening an Interactive Adobe Form through SAP or Adobe Reader, the following message is displayed:

"This document enabled extended features in Adobe Acrobat Reader under an evaluation agreement. The evaluation period for extended features has ended. Please ask the document author for the latest version of the document."

The issue still happen even for newly generated PDFs after a new ReaderRights credential has been installed

Environment

  • Forms Service by Adobe
  • SAP NetWeaver all releases

Reproducing the Issue

  1. Install or renew the Adobe ReaderRights credential in NWA.
  2. Open an Interactive Adobe Form through SAP Portal.
  3. The form is displayed in Adobe Acrobat Reader.
  4. Try to edit the form fields.
  5. The fields are not editable and the error message is shown.
  6. "This document enabled extended features in Adobe Acrobat Reader under an evaluation agreement. The evaluation period for extended features has ended. Please ask the document author for the latest version of the document."

Resolution

  1. After installing or renewing the Adobe ReaderRights credential, ensure that the new credential is correctly applied in the system.

    1. Install the ReaderRights credential in NWA as described in:

      • KBA 2860179 – How to install a Reader Rights Credential in NWA

      • SAP Note 736902 – Adobe Credentials

    2. If PDFs were generated with expired ReaderRights credentials, run report FP_REAPPLY_READER_RIGHTS as mentioned in the SAP Note 3706798 and SAP Note 3703281

      • This report reapplies the ReaderRights to existing PDFs.

    3. If the issue persists after installing the new credential, restart the Java servers to ensure that the new credential is fully recognized.
