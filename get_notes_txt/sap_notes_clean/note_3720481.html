Symptom

  • The job fails every time with the following error message
  • BIW-230108: |Data flow xxx|Loader Query@BW Load data to BW failed: RequestID = <xxx>, Packet Number = <1>, Error = RFC_CONVERSION_FAILURE- Could not convert from 4103 codepage to 4110 codepage, rc = 512 xxx [RFC_BODI][4103].

Environment

  • SAP Data Services
  • BW target

Reproducing the Issue

  1. Design a dataflow extracting from MSSQL and loading to BW
  2. In the Query transform->Schema Out, set the data length of output fields to varchar(60)
  3. Include the dataflow into a job
  4. Run job, it fails with error

Resolution

  1. Click into the Query transform of issue dataflow
  2. Double click the fields to edit the property of fields which are varchar(60)
  3. Increase the length from 60 to 255 to accommodate the full string from source
  4. Save changes
