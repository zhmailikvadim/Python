Symptom

An inspection lot has already been generated for an equipment within a maintenance order.
The equipment is then reinstalled to a different functional location.

When the inspection lot is generated again for the same order, the system creates a new inspection lot for the equipment.

Other Terms

checklist, tplnr, equnr, create_lots_for_oper_objects

Reason and Prerequisites

Programming error. Equipment and functional location were always checked together during checklist generation.

Solution

Please implement the attached correction instructions or install the corresponding support package.
