Symptom

When returning budget more than the available budget via CJ38, no budget-related error message is displayed.

Environment

  • Controlling (CO)
  • SAP ERP 
  • SAP Enhancement package for SAP ERP
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA
  • SAP S/4HANA Finance

Reproducing the Issue

  1. Go to transaction CJ38 (Change return)
  2. Enter return budget for the corresponding WBS element more than the assigned amount
  3. Click on save icon
  4. No budget-related error message is displayed

Cause

Incorrect configuration of tolerance limit for the budget profile

Resolution

  1. Navigate to SPRO -> Project Systems -> Costs -> Budget -> Define Tolerance Limits

  2. Create a Controlling area/Budget profile entry for the transaction group "06 Budgeting" and specifying your business's tolerance limit

The above configuration would enable the availability control check on CJ38 transaction and generate the relevant budget-related error message.
