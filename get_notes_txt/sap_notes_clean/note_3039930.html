Symptom

You have activated Two Envelop bidding without Simultaneous Logon enablement.

In Responses and Awards screen, Response Comparison tab is not visible.

If simultaneous logon is not activated then RFx Response Open Status (PS_PRICE_OPND) flag will be initial and Response Comparison tab should be visible.

Other Terms

Two evevalop bidding, Simultaneous Logon sim_lgn_req, RJT_PRICE_BID.

Customizing table /SAPPSSRM/TD_PRC, SIM_LGN_REQ, TWO_EVN_REQ flags.

For RFx Process Type if RJT_PRICE_BID flag is set and Technical Bid not opened (flag PS_TECH_OPND in initial) then Response Comparison tab should be invisible.

Reason and Prerequisites

Program error.

Solution

Implement the note or import relevant support package.
