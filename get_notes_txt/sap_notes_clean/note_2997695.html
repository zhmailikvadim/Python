Symptom

  • Unable to add association to the Frequency object.
  • This also applies to the wrapper object.

Environment

Employee Central - Foundation Object

Reproducing the Issue

  1. Add association in corporate data model to frequency object and import
  2. The association doesn’t appear on the frequency object in manage org, pay and job structures

Cause

Not possible from the code perspective. The Frequency is an object referenced by multiple modules in SuccessFactors and association is EC feature only, hence adding associations would cause issues for the other modules, hence reason why can’t add an association.

Changing the wrapper object in configure object defintions will make no difference here either.

Resolution

No plans currently to change this due to the dependencies for this object in other SF modules

Keywords

Frequency object, Associations, Association for Frequency Object, ECT-27090, ECT-92661
