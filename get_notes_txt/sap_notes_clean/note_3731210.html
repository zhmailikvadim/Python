Symptom

SHIP (or You install) installs a new connector version that also contains the new, additional Connector API Version 3.3, which by default only returns replies in JSON format instead of XML format.

The SAP add-on for communication with the SHIP Connector expects responses in XML format by default, does not recognize the new format, and terminates with an error.
