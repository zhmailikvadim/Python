Symptom

When trying to create or update a benefit, the error "Please enter “Benefit Contact Details”." shows, even though the Benefit Contacts filed is not mandatory. 

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

SuccessFactors Employee Central - Global Benefits

Reproducing the Issue

Create or update a benefit without informing any Benefit Contact. 

Upon saving, the error is thrown:

Cause

The field "Email Notification For Enrollment" is set to "Yes".

Resolution

If the Email Notification for this benefit is set to yes, it is necessary to add at least one contact with a valid email ID in the Useful Contact section. This field will be required because the system needs to use this Contact's email ID to send notifications to the employee.

If the Email Notification is not required for the business, you can set to No and will be able to save the benefit without any Contact Informed.
