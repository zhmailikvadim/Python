Symptom

The column 'Object Type' is not visible on initial load in the Find Technical Objects app

Other Terms

DFS_TOMANS1_A1, Find Technical Objects, app variant

Reason and Prerequisites

Relevant code added for component change.

Solution

Now the column Object Type is visible on initial load of the application.

As a general rule, SAP recommends that you install a solution by applying a Support Package. However, if you need to install a solution earlier, use the Note Assistant to implement the correction instruction. You can find more information about the Note Assistant in SAP Service Marketplace, under service.sap.com/note-assistant.

2.The actual work for a work order operation is displayed in SAP Business Network Asset Collaboration (BNAC) only when final confirmations are available in ERP (S/4/ECC).

Other Terms

BNAC, work order operation

Reason and Prerequisites

 Advance development

Solution

Implement the note to resolve the above mentioned issues.

After implementing this note:

  1. Technical objects (functional locations/equipment) linked to a work order operation through a notification will be displayed in SAP Business Network Asset Collaboration (BNAC).
  2. The actual work for a work order operation will be displayed in SAP Business Network Asset Collaboration (BNAC) when partial or final confirmations are available in ERP (S/4/ECC).
