Symptom

You have set up the function described in SAP Note 3554925 - DE: Hybrid DRG remuneration agreement 2025 + §301 supplement from 12/12/2024 with effect from 1/1/2025 - Postoperative Posttreatment for Hybrid DRG in the system to be able to bill a hybrid DRG case with a value increased by EUR 30.00 if postoperative posttreatment took place.

Service rule R11 (for creating the billable hybrid DRG service) only takes into account the admission date of the case when checking the conditions, but not the surgery date.

Other Terms

Hybrid DRG, 115f, §115f, hybrid DRG regulation, hybrid DRG conversion agreement, hybrid DRG remuneration agreement, postoperative postprocessing, R11, ISH_CHECK_SERVICES_DRG

Reason and Prerequisites

There is a program error.

The error becomes noticeable if the surgery did not take place on the admission day and postoperative posttreatment on 21. day after the surgery.

Solution

Refer to the correction instructions for your release.

After you implement the correction instructions, service rule R11 also takes the surgery date into account when checking the conditions.
