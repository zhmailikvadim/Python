Symptom

When using app "Manage Mass Maintenance - Business Partner app" in S/4HANA Cloud environment, field customer/supplier account group is not updated

Environment

Master Data Governance in S/4HANA Cloud

Reproducing the Issue

1. Using "Export Master Data - Business Partner" app to export BP data.

2. Update the downloaded file and update field customer/supplier account group

3. Using "Manage Mass Maintenance - Business Partner" app and create process with modified file

4. After activation of the process, there is no change for field customer/supplier account group

Cause

The account group change is currently not supported in the S/4HANA Cloud environment using the Manage Mass Maintenance – Business Partner app

Resolution

A change of the Customer/Supplier Account Group cannot be done in current Manage Mass Maintenance – Business Partner app, which is not possible within the S/4HANA Cloud environment.

However, in future releases there will be a corresponding message shown in the activation step to inform the user this restriction as described in SAP Note 3133794 -  CVI_CIF: Warning CVI_EI 001 for KTOKD/KTOKK and warning CVI_EI 123 when account group change is disabled.

See Also

SAP Note 3133794 -  CVI_CIF: Warning CVI_EI 001 for KTOKD/KTOKK and warning CVI_EI 123 when account group change is disabled.

Keywords

Customer Account Group, Supplier Account Group, Manage Mass Maintenance, MDG Cloud
