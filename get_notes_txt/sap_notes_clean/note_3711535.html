Symptom

The ODN information is not valid in the Create Supplier Invoice (F0859) app.

Environment

SAP S/4HANA Cloud Public Edition

Resolution

    The ODN availability in the Create Supplier Invoice (F0859) app is tentatively planned for delivery in release 2608.

    Use the Create Supplier Invoice - Advanced (MIRO) app as a workaround.
