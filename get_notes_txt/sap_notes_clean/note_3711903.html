Symptom

In SAP S/4HANA Cloud Public Edition, you want to know whether it is possible to view the exact line of error of your journal entry API payload in the response message of API call.

Environment

SAP S/4HANA Cloud Public Edition

Resolution

As of release 2602, this is not possible, the system does not support displaying the erroneous line number in the API payload error message.

It can be raised as a feature request through Customer Influence.
