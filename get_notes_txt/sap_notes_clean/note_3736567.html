Symptom

  • How to configure SAPJCE for Websphere Liberty 25 when deploying BOE and AdminTools for BI 2025?
  • Http 404 error after installing and deploying 2025 P900 BOE webapp ear on Websphere liberty server 25  [In F12 browser debugger logs, logon.jsp url  shows a 404 missing page or 404 not found]. 
  • Websphere logs show errors like -
    Caused by: java.lang.ClassNotFoundException: com.sap.commoncryptolib.provider.SAPJCE cannot be found by com.businessobjects.bcm_1.0.0 
    ..
    javax.servlet.ServletException: java.lang.UnsatisfiedLinkError: Can't load sapcrypto 
  • BI TraceLog traces show -
    Request URI=/BOE/portal/26033123456/FioriBI/logon.jsp
    ....
    javax.servlet.ServletException: java.lang.UnsatisfiedLinkError: Can't load sapcrypto
     at org.apache.jasper.runtime.PageContextImpl.doHandlePageException(PageContextImpl.java:865)
     at org.apache.jasper.runtime.PageContextImpl.handlePageException(PageContextImpl.java:794)
     at org.apache.jsp.logon_jsp._jspService(logon_jsp.java:150)
  • Cannot login to Admin tools  / Querybuilder  which fails with the error -
    Error 403: SRVE0190E: File not found: /querybuilder/ie.jsp

     

Environment

BI 2025 Patch 900 with Tomcat(windows)
Windows 2022 OS
IBM Websphere Liberty server 25 (on linux)

Reproducing the Issue

  1. Upgrade BI 4.3 SP7 to 2025 p900 - Tomcat works fine
  2. Deploy BOE webapp on websphere liberty 25 per Web application deployment guides for Windows and Unix [see section: Websphere Liberty server deployment]
  3. Login to CMC or BI Launchpad or AdminTools

Cause

  • Websphere Liberty needs to be configured for SAPJCE for login

Resolution

refer to steps documented in the SAPJCE configuration

  • SAP note 3112068 - How to configure SAPJCE libraries for web application servers?  

See Also

  • SAP note 3112068 - How to configure SAPJCE libraries for web application servers?  
  • SAP 3153615 -  HTTP 500 error while accessing CMC or BI launchpad after deploying BI 4.3 SP02+ or 2025 [sapjce]
