Symptom

You've created a section in the statement and included some conditions for it, one of the conditions is based on a grayed out field, however the section is not being shown in the statement. 

Environment

  • SAP SuccessFactors Compensation
  • SAP SuccessFactors Variable Pay

Reproducing the Issue

  1. Go to Manage Statement Templates.
  2. Select a section and create a condition.
  3. In the conditions include a field that is using eligibility rule and is grayed out.
  4. Save the statement.
  5. Generate the statements.
  6. Check the statement for one of the users with the field grayed out.
  7. The section is not displayed.

Cause

This is a system limitation. 

Resolution

Grayed out fields are not supported in conditions for Compensation statements, as the system is not considering that field the user will not meet the conditions, therefore the section won't display in the statement. 

Please note that blank fields are supported, for those type of scenarios is possible to use "is null" in the criteria. 

If you'd like to have this functionality considered within future development cycles, kindly submit an enhancement idea, for further information on how to do so please refer to KBA 2090228 - How to submit enhancement ideas for SAP SuccessFactors Products - SAP for Me
