Symptom

I want to retrieve the detailed item information shown under the Sourcing Request Items tab in a Sourcing Request, but Operational Reporting for Sourcing only returns high‑level document information and not the item‑level details. I cannot find item‑level data.

Reproducing the Issue

  1. Navigate to the Documents tab in the Sourcing Request within the system.
  2. Click on Sourcing Request Items to move to another screen displaying item details.
  3. Attempt to retrieve detailed item information using the Operational Reporting for Sourcing API using document type "SourcingRequest"

Cause

Resolution

Use the Event Management API to retrieve both the event and the detailed item information.

  1. Retrieve the Sourcing Request event:
    • Use the following endpoint to retrieve the event structure and list of items:
      GET /events/{eventId}
    • Example:
      curl --request GET \
        --url "https://openapi.ariba.com/api/sourcing-event/v2/prod/events/[eventId]?realm=[realm]&user=[username]" \
        --header "Authorization: Bearer [access_token]" \
        --header "accept: application/json" \
        --header "apiKey: [apiKey]"
  2. Retrieve details for a specific item:
    • Each item has its own detailed structure, accessible through:
      GET /events/{eventId}/items/{itemId} 
    • Example:
      curl --request GET \
        --url "https://openapi.ariba.com/api/sourcing-event/v2/prod/events/[eventId]/items/[itemId]?realm=[realm]&user=[username]" \
        --header "Authorization: Bearer [access_token]" \
        --header "accept: application/json" \
        --header "apiKey: [apiKey]"
