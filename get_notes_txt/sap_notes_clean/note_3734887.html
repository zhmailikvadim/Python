Symptom

Certain scripts within Integration Flows or script collections reference deeply nested or internal application classes, which are unsupported. Such dependencies may increase the likelihood of incompatibilities during future runtime version upgrades. 

Environment

  • Cloud Integration
  • SAP Integration Suite

Resolution

  1. Avoid using unsupported internal classes in scripts within Integration Flows or script collections to prevent potential incompatibilities during future runtime version upgrades.  
  2. Review the list of unsupported classes provided in this KBA and ensure that scripts do not rely on them.  
  3. Refactor scripts to use supported classes and APIs to ensure compatibility with future runtime enhancements or upgrades.
