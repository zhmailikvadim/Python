Symptom

When starting a Work Order or Operation, then syncing a device, the Work Order/Operation becomes inoperable. No mobile status buttons are present.

Other Terms

  • No button for Mobile Status
  • Work Order
  • Operation
  • ICMTANGOAMF10-43446

Reason and Prerequisites

  • BAS/Visual Studio Code with SAM MDK Project
  • `patch` command (available on BAS and macOS by default) via terminal or other patching utility
  • Read the guidelines on how to apply the patch and read the appendix section if you receive any error

Solution

  • Find attached file 3731923.patch
  • Save patch file into known location
  • Run command in base metadata folder: `patch -p1 < /path/to/3731923.patch`. Validate 3 files changed and no hunk failures.
  • Republish application

The following files are affected by this patch:

  • /Actions/ClockInClockOut/WorkOrderClockInOut.action
  • /Rules/ClockInClockOut/ClockInClockOutLibrary.js
  • /Rules/WorkOrders/MobileStatus/WorkOrderMobileStatusLibrary.js
