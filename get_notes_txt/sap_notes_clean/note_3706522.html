Symptom

  • A subscription fails with the error: Document:Internal Server Error. An internal server error occurred while running connector or subscription. Please contact SAP Fieldglass support.
  • Server message indicate an SFTP exception such as: Caused by: java.io.IOException: inputstream is closed.
  • A subsequent run of the same subscription may complete successfully without any changes.

Cause

  • The failure is caused by a temporary SFTP connectivity interruption during execution.
  • The SFTP session is unexpectedly closed while accessing the target directory, resulting in the input stream being closed.
  • This is typically due to transient external factors such as network instability, SFTP server timeout or restart and firewall or proxy closing the SSH session.
  • The issue is not related to subscription configuration, credentials, or directory permissions.

Resolution

  1. No action is required if the subscription succeeds in subsequent runs.
  2. If the issue recurs, monitor for repeated failures, review SFTP server logs at the failure timestamp and check network stability and server timeout settings.
  3. This behavior represents a transient operational issue rather than a configuration or system defect.
