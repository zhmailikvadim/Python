Symptom

The SCS_EXT node under /sap/bc/srt/ is not available in SICF.

  • "Service cannot be reached" error occurs when attempting to call a standard service from */SCS_EXT/* node.

Environment

  • ABAP Web Services
  • SAP NetWeaver
  • SAP NetWeaver Application Server for SAP S/4HANA
  • ABAP PLATFORM - Application Server ABAP
  • SAP S/4HANA 2023

Cause

The path /sap/bc/srt/scs_ext is related to Cloud environments and is automatically generated by Cloud tenants.

Resolution

Configure a binding in SOAMANAGER using the Alternative Access URL. In the Transport Settings tab, provide the required URL formatted as follows:

  • /sap/bc/srt/scs_ext/sap/<service-name>?sap-client=<client-number>

Save the configuration. An External Alias is generated with the same configuration as the original SICF node, representing the "Calculated Access URL".
