Symptom

 Implement, upgrade or install SAP Commerce Cloud, composable storefront 221121.9.

Other Terms

General Information

Composable storefront is a lean, Angular-based JavaScript storefront for SAP Commerce Cloud. Composable storefront communicates with SAP Commerce Cloud exclusively through the Commerce REST API.

Summary of New Features in Composable Storefront 221121.9

Title

Notes

New minor version of Angular and Angular CLI required

The minimum required version for Angular is now 21.2.0, and the minimum required version for Angular CLI is now 21.2.0. Previously, the minimum required version for Angular and Angular CLI was 21.1.0.

If you are updating to composable storefront 221121.9, the update to a new minor version of Angular does not introduce any breaking changes, but is necessary to avoid errors about peer dependency conflicts during the update to version 221121.9 of composable storefront.

For more information, see Update Release 221121.9.

 

Reason and Prerequisites

Install and upgrade to the March 2026 release of SAP Commerce Cloud 2211-jdk21.9, or the March 2026 release of SAP Commerce Cloud 2211.50.
