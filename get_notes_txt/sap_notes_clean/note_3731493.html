Symptom

We are unable to import master data from the SAP S/4HANA system into the Ariba Invoicing solution after the 7YH scope was provisioned.

Resolution

To replicate the core data from SAP S/4HANA system Please follow the below steps:

  1.  Select the correct integration details link from below corresponding to the source system.

    Source System

    Integration Details

    SAP S/4HANA Cloud Public Edition

    This integration is enabled when the application with which core data management is integrated is provisioned and no separate set-up task is needed. However, you must perform some manual steps after the provisioning to enable core data replication. For more information, see Core Data Replication Configuration.

    SAP S/4HANA Cloud Private Edition

    SAP S/4HANA

    SAP ERP

    This integration involves manual configuration. For more information, see Integrating Core Data Management with SAP ERP, SAP S/4HANA, or SAP S/4HANA Cloud Private Edition.

      2. In case of SAP S/4HANA Cloud Public Edition: Follow the Core Data Replication Configuration guide and replicate the data using Replicate by Replication Model app. Once replicated you can run the Schedule Import of Master Data in Ariba Invoicing and view the master data.

      3. In case of SAP S/4HANA Cloud Private Edition: This integration involves manual configuration. For more information, see Integrating Core Data Management with SAP ERP, SAP S/4HANA, or SAP S/4HANA Cloud Private Edition.
