Symptom

  • Uploading a file larger than 1 MB in the Maintain PIRs application fails.
  • Error message: "File Size Error", "Uploaded file is xxxFile size exceeds the limit of 1MB"

Environment

  • SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Open the Maintain PIRs application.
  2. Attempt to upload a file larger than 1 MB.

Cause

  • The Maintain PIRs application enforces a 1 MB upload size limit due to performance considerations.

Resolution

  1. Split the source file into smaller parts (each ≤ 1 MB).
  2. Upload each part separately in the Maintain PIRs application.
  3. If larger upload capability is required in the future, submit an enhancement request via the Customer Influence Portal: refer to KBA 2712608 - How to Submit Feature Requests for Core Logistics and Manufacturing in S/4HANA Cloud Public Edition.

See Also

  • Refer to: KBA 2712608 - How to Submit Feature Requests for Core Logistics and Manufacturing in S/4HANA Cloud Public Edition
