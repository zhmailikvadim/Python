Symptom

Why do Purchase Orders (PO) fail with Invalid From Credentials in SAP Integration Suite Managed Gateway for Spend Management and SAP Business Network?

Cause

Supplier identifiers for the procurement application is missing the vendor and the system ID mapping

Resolution

To update the information under Unique Supplier ID Configuration under Buyer SAP Business Network account, please follow the steps below:

  1. Navigate to Manage Vendors > Search for Suppliers.

  2. Select the option Search only my trading partners

  3. Enter the Supplier ANID

    Ensure that no country is selected.

  4. Click Search.

  5. Once the result appears, click Edit Preferences or Edit

  6. Update and map the Vendor ID to the Application System ID  > Save the changes

Once this is completed, please reprocess the failed transaction in Managed Gateway for Spend&Network. This update will allow the PO to be processed successfully to SAP Business Network

See Also

Alternate way to update the Unique Supplier ID Configuration details is as follows

  1. Navigate to Supplier Enablement > Active Relationships

  2. Go to Classic View 

  3. Search Suppliers under Current Suppliers

  4. Select the Supplier > More Actions > Edit

  5. Follow 1st method steps from 2 to 6

You may refer the KBA How do I reprocess FAILED messages from Managed Gateway for Spend&Network Transaction Tracker? to reprocess the outbound transaction from Managed Gateway for Spend&Network formerly called CIG
