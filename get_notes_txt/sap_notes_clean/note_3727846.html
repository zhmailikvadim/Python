Symptom

I do not have a report to display user usage details as login information, areas of use and usage of SAP Spend Control Tower application.

Resolution

Currently there is no SAP Spend Control Tower report to display user usage details as login information, areas of use and usage of SAP Spend Control Tower application.

In case you feel this is essential for your business, please consider to submit an Improvement Request.

See Also

If you would like to request a report with this information, please raise an Enhancement Request.

Keywords

sct user usage, sct user login, sct activity
