Symptom

When copying and pasting an object or field in Crystal Report Designer for SAP Business One, the following error appears: Not implemented.

Reproducing the issue

Prerequisites:

You are using Crystal Report Designer for SAP Business One 2025 SP0 P1 and below.

Scenario:

1. Open or create a new Crystal report, and add a few fields or text objects.

2. Copy one of the fields or objects.

3. Paste it anywhere on the report.

Expected behavior:

The field can be copied correctly.

Actual behavior:

The following error appears: Not implemented and the field is not copied.

Solution

SAP intends to provide a feature package or patch to solve the symptom that is described in this SAP Note. The section References below lists the related feature package or patch once they become available. Be aware that References can only be confirmed at feature package or patch release date.
