Symptom

Symptom

INA:

  • Filtering for a hierarchical dimension input help for a dimension does not work if the internal key is used in the filter criteria.
  • For the filter, the capability SupportsHierarchySelectionAsFlatSelection controls whether the result is returned in a flat list or hierarchically.

To request the input help in a flat list, the capability SupportsHierarchySelectionAsFlatSelection AND the parameter ConvertToFlatSelection=true must be set.

Other Terms

Type = Dimension

Reason and Prerequisites

Program error/advance development

Solution

Implement the correction.

Other Terms

CX_SY_CONVERSION_OVERFLOW, MENGE, QUANTITY

Reason and Prerequisites

There's a program error. It can occur in case of large amounts or large quantities in the deferred revenues. When being aggregated into G/L items in the transfer documents, an overflow might occur.

Solution

Implement this SAP Note or the given Support Package. The system will prevent the overflow situation by posting single documents in case that the amounts or quantities are too high.
