Symptom

In Focused Run, monitoring of BW process Chains or ABAP Jobs have been configured in job & Automation Monitoring, but not all jobs or process chains are detected

Environment

SAP Focused Run

Reproducing the Issue

  1. Transaction FRUN

  2. Click on Job & Automation Monitoring

  3. In Scope Selector select the affected managed system

Resolution

  1. Make sure that the latest version of the corrections note has been implemented as described in the Prerequisites

    ST-PI with corrections as follows:

    ST-PI 700 SP31 (or higher)  

    ST-PI 740 SP16 - 20 and SAP note 3102288 

    ST-PI 740 SP21+ (optimally SP23+) and SAP note 3281776 

    ST-PI 740 SP26+ and SAP Note 3563104

  2. Turn off data collection in Job & Automation Monitoring and after 10 minutes turn on again. This will request fresh metadata in the log files
