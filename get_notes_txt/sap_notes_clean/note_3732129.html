Symptom

  • The replacement materials associated with the return order are currently grayed out, which means it cannot be modified within the return order because the return delivery has been successfully created for the return order.
  • When attempting to create a subsequent delivery free of charge (SDF) with an alternate replacement material, the system selects the original material from the return order instead of the replacement material in MSR_CRD.
  • Error message: "Item contains errors and cannot be copied."
  • In some cases, a message indicates the template item cannot be copied and must be reactivated, preventing the replacement delivery.

Environment

  • Sales And Distribution (SD)
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Create a return order and return delivery.
  2. Open MSR_CRD and attempt to create the SDF (subsequent delivery free of charge) with a manual inserted replacement material.
  3. Observe that the system considers the original returned material rather than the replacement and issues the error "Item contains errors and cannot be copied."

Cause

  • Missing correction code in system
  • Missing inspection in return process.

Resolution

  1. Implement Note 3608285 - Subsequent delivery free of charge is not created when giving alternate replacement material in the inspection order to ensure the necessary correction code is in system.
  2. Create an inspection for the return order so that replacement determination is controlled by the inspection.
    • In the inspection, maintain the desired alternate replacement material for the returned item.
    • In MSR_CRD, create the SDF again. The system will determine the replacement material from the inspection and the SDF should be created without the copy error.
