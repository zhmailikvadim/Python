UPDATE 2 February 2018: This note has been re-released with updated ‘Solution’, ‘Header data’ and ‘Validity’ information

SAP Sybase Cockpit, SAP Control Center (SCC), SQL Anywhere Monitor, and SQL Anywhere On Demand Edition Cloud Console administration tools could allow an attacker to trick an authenticated user to send unintended request to the web server. This vulnerability is due to insufficient CSRF protection. Refer to CVE-2011-2461 and Adobe Security Bulletin APSB11-25 for more information.

Some well-known impacts of CSRF vulnerability are -

  • Attacker could take actions on behalf of an authenticated user
  • Loss of non-repudiation

 

Other Terms

Open redirect, cross-site redirect, cross-domain redirect, XSRF, session riding

Reason and Prerequisites

SAP Sybase Cockpit, SAP Control Center (SCC), SQL Anywhere Monitor administration tools are Adobe Flex applications that are impacted by the vulnerability described in CVE-2011-2461 and Adobe Security Bulletin APSB11-25.

Solution

Follow instructions below to apply the fix most appropriate for your production environment.
