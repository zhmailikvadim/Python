Symptom

hdblcm.log

Installing SAP HANA Local Secure Store...
  Resuming broken installation at step 'Calling 'postInstall' Event Handler'.
  Preparing package 'LocalSecureStore'...
  Preparing package 'Installer'...
  Installing Local Secure Store to /lss/shared/HX4...
Update of SAP HANA Database components failed.
  Installation of SAP HANA Local Secure Store failed
    Installation failed
      error installing
        Cannot install
          Custom event postInstall failed
            Initializing persistence failed!

In some cases, also crash dump is generated and the following error can be discovered in this:

Unloading library /lss/shared/XXX/exe/libsapcrypto.so that is marked as permanent!; $condition$=!cur->m_IsPermanent 

Environment

SAP HANA, platform edition 2.0

Resolution

- Stop service oneagent 
- Rename system file /etc/ld.so.preload which is containing only one line : /lib64/liboneagentproc.so
- Rename library /lib64/liboneagentproc.so
Then, resume SAP HANA update. 
Revert system file and library to original names, restart service oneagent.
