Symptom

Harmonized Scenarios are enabled in the system. See: Introducing Harmonized Scenarios

The global configuration parameter DEFAULT_EXPIRY_DAYS_HSM is set to a value higher than 5.

From Planner Workspace (PWS), the expiration date can be set according to DEFAULT_EXPIRY_DAYS_HSM.

However, in the Manage Versions and Scenarios app, the maximum expiration time allowed for a new harmonized scenario is limited to 5 days.

Environment

  • SAP Supply Chain Management (SCM)
  • SAP Integrated Business Planning (SCM-IBP)

Cause

The Manage Versions and Scenarios app currently restricts the maximum expiration period for harmonized scenarios to 5 days.

The primary tool for managing harmonized scenarios remains the Planner Workspace (PWS) application.

Resolution

To create harmonized scenarios with longer validity (up to 70 days, or the value defined in DEFAULT_EXPIRY_DAYS_HSM), scenarios must be created from Planner Workspace (PWS).
