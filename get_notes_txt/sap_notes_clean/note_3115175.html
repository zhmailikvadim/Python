Symptom

In the past there were various checks missing in the validation process of the availability rule table of the Capacity Availability Settings.

 This SAP Note adds the missing checks.

 Before you implement this SAP Note make sure you have implemented all of these notes:

  • 0003090383
  • 0003096250
  • 0003096273
  • 0003096888
  • 0003101632
  • 0003108483
  • 0003110592
  • 0003111732

 The messages used by the checks are contained in SAP Note 3110648, thus you have to install SAP Note 3110648 before you install this one.

Reason and Prerequisites

Program Error

Solution

Implement the attached Correction Instruction or the mentioned Support Package.
