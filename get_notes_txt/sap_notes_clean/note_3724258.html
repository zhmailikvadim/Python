Symptom

When ERP documents (such as sales orders, purchase orders, or other document types) are created using external number ranges containing alphanumeric characters, the associated erp text elements fail to transfer to Transportation Management during integration processing.

Other Terms

TM, /SCMTMS/CL_LOGISTICS_DATA_PROV, Logistics Integration

Reason and Prerequisites

 There is a program error.

Solution

 Implement the corresponding correction instructions or install the relevant support package instead.
