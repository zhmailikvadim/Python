Symptom

In eDocument Cockpit, selecting Go to -> Display PDF for AP or AR direct FI credit notes shows the error: "No file found for the selected eDocument; PDF cannot be generated." When the UUID is entered manually, the PDF displays successfully. You want to know the reason.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Open the eDocument Cockpit APP.
  2. Search for the electronic document.
  3. Choose the button Go to - Display PDF. then observe the mentioned error message.

Cause

As per legal guidelines, in an electronic document (credit memo or debit memo), you must indicate both the IRBM Unique Identifier Number and internal document number of the original invoice. 

This UUID is displayed in the IRBM Unique Identifier Number column of the document list in the eDocument Cockpit or Manage Electronic Documents app. It's also displayed in the PDF file for the electronic invoice. 

Resolution

The mentioned error occurs as expected if no UUID is maintained.
