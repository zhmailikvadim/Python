Symptom

Maintenance for InfoObject Hierarchies

  • User can add invalid dates and the OK button is always enabled.
  • When you create a node and try to change the details, it indicates that there is a duplicate.

Master Data Management

  • When the input was a string on a NUMC field, a duplicate error was showing.
  • It was accepting a string error on a NUMC field.
  • The save button was always on.

RSPM Management for DataStore Objects/InfoObjects/Open Hub Destinations

  • The title was getting truncated. (Accessibility)
  • The labels were getting truncated when the screen was resized. (Accessibility)
  • Target Text was getting truncated. (Accessibility)
  • When the screen is tiny, the log button disappear. (Accessibility)

Other Terms

BW4 Cockpit, Data Preview, MDM, MD Hierarchy Maintenance, RSPM
