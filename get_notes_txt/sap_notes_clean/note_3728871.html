Symptom

You are processing the Maturity Structuring for a securities account which is based on expected cash flows. The method derived is based on a methodology using cash flows. The process fails and the error message "Error reading book value for ..." is written to the log.

Other Terms

Methodology '050'
Message Class '/BA1/BR_MATGRP_CALC'
Message Number '023'

Reason and Prerequisites

There is a program error.

Solution

Apply the correction instruction or the respective support package.
