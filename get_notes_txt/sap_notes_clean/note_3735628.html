Symptom

Not all fields configured in Adaptation Mode are displayed in the Case Header section [sap.crm.caseservice.uiview.caseDetailsHeader]. After ending adaptation, only the first two fields are displayed in the top-right of the Case Header.

Environment

SAP Service Cloud Version 2

Cause

This behavior is expected and is a part of the system’s design. The top-right area of the Case Header has a fixed layout that supports displaying only two fields. Additional fields may be visible during Adaptation Mode, but they will not be displayed after ending Adaptation Mode.

Resolution

The system is designed to display only two fields in the top-right area of the Case Header. This area has a fixed layout and does not support displaying additional fields, even though more fields may appear during Adaptation Mode.

To highlight more key fields in the header, you can use the “Highlights (Expanded Mode)” section. In this section, you can configure and display up to four additional fields, which will appear when the header is expanded.
