Symptom

  1. How is SAP Databricks consumption calculated?
  2. Are there any partner-specific pricing models, evaluation discounts, trial entitlements, or bundled credits available for SAP partners who are testing Databricks via SAP Business Data Cloud for enablement and prototyping purposes?

Environment

  • SAP Business Data Cloud
  • SAP Databricks

Resolution

  1. The pricing model for SAP Databricks is based on the subscription model for SAP BDC, which is measured in Capacity Units (CUs), and all billing is managed by SAP. SAP calculates usage in SAP Databricks based on compute types and storage options, then converts this into CUs for each Capacity Service. For more details, you can consult the "SAP Business Data Cloud Supplement" document which can be found from the SAP Databricks Metering online help. There are tools at our disposal to estimate costs: SAP Business Data Cloud Capacity Unit Estimator and/or SAP Databricks Sizing Calculator.
  2. Regarding partner-specific pricing, the cost for Databricks usage remains the same. Details about the Partner TDD (Test, Demo, and Development) offering can be found in the SAP Partner Portal.
