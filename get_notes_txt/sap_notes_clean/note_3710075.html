Symptom

When you view a virtual bucket, you expect the related source documents to be reflected in the bucket items. However, the source documents are not displayed in the virtual bucket. Even when accessing the calculation bucket directly from the calculation bucket workbench using transaction /IRM/IPBBCLBM, the items are not shown in the items tab. Since the claim data is available in the CDS view of the bucket, it is expected to be displayed consistently in the items tab as well.

This note corrects the behavior by ensuring that source documents are properly reflected in the items tab of the virtual bucket.

Other Terms

virtual bucket, source documents not displayed, calculation bucket workbench, /IRM/IPBBCLBM, bucket items tab, cds view, claim data

Reason and Prerequisites

This is caused by program error.

SAP Note # 3635489 is available for V4 Releases.

Solution

Please implement the correction instructions attached. These corrections are only valid if you have applied the Support Packs.

Release 6.4 Base Package, 6.3 Base Package, 6.2 Base Package, and 6.1 Base Package.

Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object.
