Symptom

In the Manage Process Orders Fiori app (F4587), the Scheduled Start Date is displayed incorrectly (one day earlier) in the main list view.
The value shown in the list does not match the date and time stored in the backend system.

Other Terms

F4587, Manage Process Orders, MfgOrderScheduledStartDateTime, UTC

Reason and Prerequisites

 Program error. The application is incorrectly taking into consideration a time zone set-up of user's local PC / device.

Solution

Implement the attached Correction Instruction.
After implementation, clear the frontend/UI cache as described in SAP Note 2319491 to activate the updated application behavior.
