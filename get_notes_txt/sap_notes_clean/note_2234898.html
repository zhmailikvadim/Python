Symptom

The result set of an SQL SELECT statement (without the addition ORDER BY or ORDER BY PRIMARY KEY) may not be sorted.

Other Terms

selectsort, read table binary search

Reason and Prerequisites

Some databases output the result set of a SELECT statement sorted if a database index is used to collect the result set. They do that even if no ORDER BY or ORDER BY PRIMARY KEY enhancement is used. However, this is an undocumented and unguaranteed function that depends on various database parameters and context parameters.

Solution

Implement the attached program corrections.
