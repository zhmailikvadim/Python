Symptom

An error occurs when attempting to create a personalization rule in SAP Emarsys. The system fails to save the rule after inserting the products and displays an error message.

Environment

SAP Emarsys

Cause

Even though it currently appears as selectable in the UI, the item_id field was used in the rule set but it is not supported in personalization rules.

Resolution

Use any custom field that has unique values and can serve as a unique identifier for the products instead.
