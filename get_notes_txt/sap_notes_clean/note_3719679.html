Symptom

  • Password reset emails are not received when using SAP ID as identity provider

Environment

  • SAP ID Service

Reproducing the Issue

  1. Attempt to sign in through SAP ID service with your email address
  2. Click "Forgot password" when being prompted for password
  3. Provide your email address for the password reset then click "Continue"
  4. Observe that no password reset email is received.

Cause

There is no S-user created with the provided email address. To be able to use SAP ID Services, you need a valid S-user.

Resolution

Reach out to your administrator to create an S-user as per KBA 1271482 - "How does an administrator create or delete S-user IDs? - SAP for Me".

You can find your administrators under SAP for Me --> My Important Contacts.
