Symptom

  • Transaction SM21 on the production SLT system shows repeated SQL insert failures into two logging tables.
    sql error -10512 performing INS on table /1LT/XXXX.
    Row not processed due to previous error.
  • Errors persist despite restarting the replication

Environment

SAP Landscape Transformation Replication Server (SLT)

Cause

Triggers on the target table cause failed inserts by not accounting for delete/insert scenarios.

Resolution

  1. Option 1: Drop the target triggers
    Option 2: Rework the triggers on the target table to make them account for delete/insert scenarios, and clear the problematic records from the target logging tables.
  2. Stop the affected table replication and start again
