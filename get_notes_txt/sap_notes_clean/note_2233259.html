Symptom

ECTR shows the error message "No original available for document..." during the URL generation / checkout process.

Other Terms

(PLM-16573)

Reason and Prerequisites

The CDESK_SRV_GET_FILES returns wrongly only not active originals.
ECTR drops this not active originals and returns a misleading error message.

Solution

The error handling in the ECTR FM has to be improved to return a better error message.

Please implement this note.
