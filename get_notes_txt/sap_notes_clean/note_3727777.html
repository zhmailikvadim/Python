Symptom

  • Web UI experiences issues with the latest 146 version of Chrome Browser.
  • When selecting a value in pop-up window, the wheel keeps spinning and ends with a 500 error.
  • Other performance and pop up issues can occur with this Version of Chrome
  • The issue occurs in Chrome Version 146.0.7680.72 (Official Build) (64-bit) or higher. 

Environment

  • SAP Customer Relationship Management
  • SAP Solution Manager
  • SAP S/4HANA
  • Web UI

Reproducing the Issue

1. Install the latest version of Chrome browser 146 (Version 146.0.7680.72 (Official Build) (64-bit)  or higher.
2. Access the Web UI, open a pop up, and select a value from that Pop up
3. After the pop up closes, spinning wheel appears on the Web UI and the screen freezes.

Cause

The issue occurs in chrome version Version 146.0.7680.72 (Official Build) (64-bit)  or higher.

Resolution

Check the the following notes are implemented in the affected system:

SAP Note 3376531 - Deprecate unload event
SAP Note 3388942 - Deprecate unload event
SAP Note 3421431 - HTTP header Permisions-Policy     >>>>>>>>>>    (This note has an attachment with some manual pre and post implementation steps to be executed)
SAP Note 3427088 - Removing HTTP header Permissions-Policy
SAP Note 3422431 - Close event is executed two times on popups
SAP Note 3456476 - Adding iframe allow attribute to allow unload event
SAP Note 3444103 - Enable unload event via HTTP 

Then remove cache as per SAP Note 2050322 - Clearing Caches
