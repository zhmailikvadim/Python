Symptom

The qualification process disappeared from the Qualification tab of the Supplier profile after Commodity/Region/Department values were updated via mass edit.

Reproducing the Issue

  1. Update the Commodity /Region /Department values through mass edit for the Qualification process project.
  2. Open the project Advanced view and perform a manual sync.
  3. Observe that the qualification project is deleted from the Qualification tab of the Supplier 360 profile.

Cause

When a process project is created, responses for Commodity, Region, and Department trigger the relevant modular questionnaires, the decision maker then make the decisions for those fields. Directly updating the  Commodity, Region, or Department via mass edit for these process projects is unsupported and may corrupt the project, causing it to be removed from the supplier profile. 

Resolution

Updating the Values for the Process project through mass edit is not supported.
