Symptom

When extracting with Datasource 0CRM_CATEGORIZATION_HIER, field FIELDNM is empty in Node segment. This can cause errors in BW system.

Environment

  • SAP Customer Relationship Management 7.0 and higher
  • SAP enhancement package for SAP Customer Relationship Management
  • SAP enhancement package for SAP CRM, version for SAP HANA

Reproducing the Issue

  1. Go to T-code RSA3
  2. Enter Datasource 0CRM_CATEGORIZATION_HIER and HIENM
  3. Click Extraction
  4. Go to Node Segment. Field FIELDNM is empty
     

Cause

FIELDNM gets cleared in the form routine CALL_HIER_CUSTOMER_FUNCTION_30

Resolution

Please implement Note 1997720 in your CRM system

Keywords

CALL_HIER_CUSTOMER_FUNCTION_30, CL_RSSH_HIERARCHY_BASE, MESSAGE_TYPE_X
