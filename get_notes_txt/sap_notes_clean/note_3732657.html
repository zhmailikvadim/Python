Symptom

While the SAP Best Practices content was being installed, below error was encountered during the activation.

Start  activation :         SIMG_CFMENUOLSDOVA3
Not activated - error
Check workaround documents / configuration guides for a solution
If you cannot resolve the error, create a ticket on component SD-BF-CNT
End of activation :         SIMG_CFMENUOLSDOVA3

Environment

  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Cause

The issue is because there are some client 000 entries exist in the system and caused the inconsistency.

Resolution

The error step can be skipped and no follow-up issues will be caused.

Keywords

SIMG_CFMENUOLSDOVA3, BD1(XX), best practices, activation, ecatt
