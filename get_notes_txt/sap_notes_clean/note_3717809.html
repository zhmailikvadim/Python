Symptom

In the standard Separation Notification (HUNCSEP0) report, the system submits partial contributions to the United Nations Joint Staff Pension Fund (UNJSPF) when you run it in batch mode.

Other Terms

  • Separation notification
  • HUNCSEP0
  • 0961 

Reason and Prerequisites

Program error.

This SAP Note is relevant only for non-profit organizations.

Solution

After applying this SAP Note, the system transmits all valid contributions to the United Nations Joint Staff Pension Fund (UNJSPF) when you run the Separation Notification (HUNCSEP0) report in batch mode.

To receive these changes, implement the support package specified for your release.

Alternatively, you can implement the attached correction instructions.

As a general rule, SAP recommends that you implement the Support Package specified for your release. However, if you need to install a solution earlier, use the SAP Note Assistant to implement the correction instruction. You can find more information about the SAP Note Assistant in SAP Support Portal, under SAP Note Assistant.
