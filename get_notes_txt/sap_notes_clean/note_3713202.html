Symptom

When you activate receipt management at the company code level for the IS-U industry, the system does not generate the receipt number.

Other Terms

FP-REPT,  FPREPT, TFK001B, TFK000C, XREPT,  3709109, 3655973

Reason and Prerequisites

The IS-U events was not adapted.

Solution

After implementing this SAP Note, the system generates receipt numbers correctly when you activate receipt management at the company code level for the IS-U industry.

To resolve this issue, implement this SAP Note in accordance with the attached correction instructions or import the relevant Support Package.
