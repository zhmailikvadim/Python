Symptom

When displaying an existing edition in the Edition Monitor, the UI incorrectly shows the Propagate field as true even though the corresponding edition has the Propagate flag set to false in the database.

Additionally, when switching to edit mode, the Propagate field is not editable, preventing the user from correcting or adjusting the flag value which should be possible.

Other Terms

Edition Propagation, Edition Monitor, Analysis of Edition, UI

Reason and Prerequisites

The Propagate flag was by default set to true without considering the database values.

Solution

Please implement the SAP note which solves this incosistency.
