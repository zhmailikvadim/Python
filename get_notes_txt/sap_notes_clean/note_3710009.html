Symptom

This SAP Note introduces functional enhancements for the Poland KSeF e-Invoicing solution:

  • Offline PDF Issuance: Enables the generation of PDF visualizations (including issuer-certified QR codes) to be issued to buyers during periods of Tax Authority service unavailability and other communication issues.

  • Technical Corrections: Enables creation, linking and submission of technical correction documents for invoices previously issued in offline mode.

Note:

To use Offline PDF functionality, the Electronic Documents should be submitted to DRC Cloud Edition using the Electronic Document Interface PL_KSEF_SEND_REQUEST Version 4.

The Electronic Documents created before the activation of the Interface PL_KSEF_SEND_REQUEST Version 4 do not support Offline PDF functionality.

To activate the Interface and maintain other mandatory customizations, please refer to the attachment SAP Note 3710009 Customization Steps PDF.

For more details about the functionalities, please refer to the attachment DRC Poland KSeF Offline Mode and Technical Corrections PDF.
