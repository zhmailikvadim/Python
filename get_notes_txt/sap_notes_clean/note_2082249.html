Symptom

The french view file (FEC) solution was implemented but some of the column names are missing or some column names ar displayed multiple times after each other in the created file.

Reproducing the Issue

Running the view 2FR_FI01 in transaction FTWY as french format. The output file does not contain some column names or some column names are displayed multiple times after each other.

Cause

Step 1 of the note 2188794 is not executed in the affected system.

Resolution

The report ZTXW_LANGU_TEXT_NV1 should be started which is attached to the note 2188794. Once this is done, the headers will be displayed with correct values.
