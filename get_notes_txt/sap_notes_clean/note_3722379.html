Symptom

You are using a WD ABAP UI element IFrame and want to set a permission policy according to SAP note 3683742. This is not possible.

Other Terms

LocalNetworkAccess; LNA; Local Network Access; iFrame

Reason and Prerequisites

Program error

Solution

Please implement the attached correction instruction or the corresponding support package.

You need to provide the permissions in the 'allowAccess' property of the iframe UI element.

----------------

- CAUTION: This note contains manual pre-implementation steps that need to be executed before implementing any correction instruction!

- After you have executed the manual pre-implementation steps, please implement the attached correction instruction. Another option would be to implement the corresponding support package (without manual pre-implementation steps).

Additionally, please upgrade the Unified Rendering version via note 2090746 to at least UR2602.

Caution: Before implementing this note please execute the mentioned manual pre-implementation steps.
