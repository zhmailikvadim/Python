Symptom

In the SAP Ariba Buying and Invoicing that is integrated with SAP ERP or SAP S/4HANA using SAP Integration Suite, managed gateway for spend management and SAP Business Network, after implementation of Note 3591444, quality certificates are generated for the second (or higher) ASN items associated with the same purchase order item. However, the certificate status is incorrect for these additional ASN items: the system displays status 2 "Certificate expected", whereas the correct status should be 1 "Certificate received", consistent with the first ASN item.

Cause

This issue occurred due to an erroneous code. As a result, the status of the quality certificate for the second or higher item of the ASN is getting as '2' which is 'Certificate expected'.

Resolution

Now, the code has been enhanced to resolve the problem and please implement this SAP Note. For more information, please see SAP NOTE 3680607.
