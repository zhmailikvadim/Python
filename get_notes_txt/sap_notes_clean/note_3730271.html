Environment

Product versions:

  • SAP Commerce Cloud 2211.46 jdk17
  • SAP Commerce Cloud 2211.1 jdk21

Cause

We intentionally removed the OOTB samlKeystore.jks by design.

Resolution

The samlKeystore.jks file that was previously provided with SAP Commerce Cloud for SAML SSO test purposes has been removed. For information on how to create your own samlKeystore.jks file, see Configuring SAP Commerce Cloud.

See Also
