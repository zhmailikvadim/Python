Symptom

  • You want to edit/delete the details of an Person in the system.
  • You cannot find the related details in any application transaction to do so (e.g. customer, vendor, plant, etc.)

Environment

  • SAP NetWeaver 
  • Release Independent

Resolution

  1. Transaction SADP (Contact person addr.maint. init.scr.) is available to carry out direct maintenance on an Person address
  2. First goto table ADRP and using the Person number or First/last Name (or other related details) of the entry you need to change, find the person group (field "PERS GROUP") of the entry and also take note of some of the person details. These details maybe needed to narrow the selection if multiple entries are found.



  3. In trasaction SADP enter the Person goup into the left box and hit enter:



  4. Then on the next screen extend the Selection options and enter the first/last name (and/or any other details that will find the Person) from ADRP to find the entry, if you know the related address number this can be entered at the bottom to restrict the selection. If there are multiple entries found then a pop-up will be displayed (as seen below) with the different Person options that can be selected:



  5. Once you have selected the entry you require, the entry can be edited using the Change button or the entry deleted using the Delete Selection (Dustbin) button.
  6. The following message "Address can only be maintained in primary object; indicator in table TSAD8 - Message no. AM121" maybe thrown when attempting to edit/delete the address.



  7. To resolve this message, goto transaction SM30, enter table TSAD8, and using then Maintain button select the flag "Direct maintenance" for Person group of the Person you are attemping to change, then save this change.
  8. If the entry you require to be edited cannot be found using transaction SADP, please attempt to use transaction SADQ (again you will have to first enter the person group) which is for private address maintenance. Here you can enter the Address and Person numbers aswell as details about the entry:
