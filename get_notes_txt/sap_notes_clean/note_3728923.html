Symptom

You notice that the Annual salary (wage type /433) is calculated incorrectly especially after the implementation of SAP Note 3609119 - NST00: Improvements in the Calculation of Annual Salary. This results in the incorrect determination of Special Rate for Wage Tax (wage type /432).

Other Terms

NST00, /43C, /43D, /433, /432, Annual Wage SP, Jaarloon BB, Bijzondere beloningen

Reason and Prerequisites

The wage types '/43C' (Table Wage Current Period Including Reservations) and '/43D' (Missed Table Wage Including Reservations for Previous Inactive Periods) are used as the basis for calculation of annual salary, although it is not available in the payroll results of the previous year.

Solution

Delivery

Import the HR Support Package relevant for your release. For more information, see SAP ERP Human Capital Management Support Package Schedules or SAP S4 HCM Support Package Schedules.

SAP recommends that you implement the Support Package. However, if you need to install a solution earlier, use the SAP Note Assistant.
