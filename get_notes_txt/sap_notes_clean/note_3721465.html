Symptom

During note implementation or system upgrade, there are errors below:
TABL  /MDG* was not activated.
Field &1 in table /MDG* is specified twice

Environment

Master Data Governance on S/4HANA

Reproducing the Issue

Implement SAP Note or perform system upgrade.
Mentioned error happens and activation is ended.

Cause

There are duplicated fields in certain table/structure.

Resolution

Please use transaction SE11 to check relevant table/structure and manually delete the duplicate field, especially .INCLUDE structure. Ensure that all the fields are appearing only once in the structure/table.

Please also refer to SAP Note 1624942

See Also

SAP Note 1624942
