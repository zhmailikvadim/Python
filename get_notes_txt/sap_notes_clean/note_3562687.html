Symptom

When trying to execute the report RSZDM_CC_TABLE_DELTA_RULES as per the third steps from the note "3309908 - ATC Compliance Checks for Zero Downtime Option of SUM - Initial Shipment 2023", the following dump occurred.

"

Category               ABAP programming error                                                      

Runtime Errors         UNCAUGHT_EXCEPTION                                                          

Except.                CX_ZDM_CC_TABLE_DELTA_RULES                                                 

ABAP: Program          CL_ZDM_CC_TABLE_DELTA_RULES===CP                                            

Application Component  BC-UPG-DTM-TLA                                                              

"

In "Error analysis" part of the dump, the following message occurred.

"

The reason for this exception was:                                                                                         

    Error calling SCWB_DOWNLOAD_EVENTS (sy-subrc:2;err-msg:Release data not updated)

"

Cause

HTTP connection SAP-SUPPORT_PORTAL is not configured correctly.

Resolution

Please follow the "Prerequisites" shown in note 3309908.

"Make sure that the HTTP connection SAP-SUPPORT_PORTAL exists and is working in ABAP transaction SM59 by choosing button "Connection Test"."

See Also

3309908 - ATC Compliance Checks for Zero Downtime Option of SUM - Initial Shipment 2023

3309908, CX_ZDM_CC_TABLE_DELTA_RULES, SAP-SUPPORT_PORTAL, SCWB_DOWNLOAD_EVENTS
