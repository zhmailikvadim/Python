Symptom

This SAP Note is a collective Note that provides guidelines and contains information for the implementation of DRC Netherlands EC Sales List, covering a list of frequently asked questions and SAP Notes that relate to Netherlands EC Sales List to support implementation.

Although every effort has been made to provide a thorough set of FAQs in this SAP Note, this document is not exhaustive. If additional FAQs arise, this SAP Note will be updated to reflect the most current and accurate information.

We therefore recommend that you regularly check whether new notes have been added which may be of interest to you. It contains the most important notes, but not necessarily all of them.

If your question is not answered in this SAP Note, log a customer incident in SAP Support Portal under FI-LOC-ECS component.

Other Terms

Netherlands, ECSL, Intracommunautaire Prestaties, NL_ECSL_DCL, FAQ, Implementation Leading Note, ACR, SAP S/4HANA for Advanced Compliance Reporting, Advanced Compliance Reporting, Run Advanced Compliance Reports

Reason and Prerequisites

Reason

You want to understand, install, configure and/or use Netherlands EC Sales List for SAP S/4HANA and have questions about it.

Prerequisites

Your DRC framework is up-to-date.

Please read SAP Note 3033455 to find out more information about SAP notes containing consolidated DRC changes.

Please read SAP note 3488367 and 3478591 to find out more information about Submission via SAP Document and Reporting Compliance, cloud edition.
