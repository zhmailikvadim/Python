Symptom

Message no. HRPAY99PM244 'Error during INSERT T52SPL &1 &2' occurs after running t-code PUOCBA. "Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental." 

Environment

  • Human Capital Management (HCM)
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP SuccessFactors Employee Central Payroll
  • SAP S/4 HANA

Reproducing the Issue

  1. Run transaction PUOCBA.
  2. Select payroll section.
  3. Input process model and HCM localization.
  4. Click ‘Execute’ button.
  5. Message no. HRPAY99PM244 'Error during INSERT T52SPL &1 &2' displays in the job log. (&1: process id and &2: step id).
    For example, ‘Error during INSERT T52SPL 0000000039 0000000001’ is displayed in the job log.

Cause

Incorrect configuration.

Resolution

Follow and verify the configurations in the SAP Note 1163794 - Process Model does not run correctly.

See Also

SAP Note 1163794 - Process Model does not run correctly

Keywords

HRPAY99PM244, INSERT, T52SPL, Process model, PUOCBA
