Symptom

You send a remote OData V4 request containing a bulk action/function and a filter. The filter does not get applied and the response contains the action/function results for all entities of the entity set. If the request is sent locally, the filter works as expected.

Other Terms

 SAP Gateway, OData V4, action, function, $each, bulk action, bulk function, filter

Reason and Prerequisites

For actions/functions with a return type other than "Entity", the filter condition got dropped when processing a remote request.
