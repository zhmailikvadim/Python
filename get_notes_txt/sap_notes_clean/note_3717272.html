Symptom

The performance of Fiori applications that return business objects with status‑related properties is severely impacted due to an anti‑pattern in the I_StatusObjectStatus CDS view. This issue becomes clearly visible in systems containing a large volume of Maintenance Notifications or Maintenance Orders, where the view’s design leads to significant performance degradation during data retrieval.

Other Terms

Performance, I_StatusObjectStatus, I_MAINTNOTIFICATIONTP, I_MAINTORDEROPERATIONTP, I_StatusObjectStatus_2, Status, F2175, F2071

Reason and Prerequisites

Design anti-pattern.  Performance anti-pattern in I_StatusObjectStatus

Solution

 Implement the advanced SAP Note

Other Terms

DE_B_GPKE_NIN_DSO, DSO, device data, process date, NIN, GPKE, device time slices

Reason and Prerequisites

The error occurs because of a program error that causes the system to fetch device data without considering the process date.

Solution

Implement the corrections of this SAP Note or import the relevant support package.
After implementing this SAP Note, the system fetches the correct device data based on the process date. You can verify the fix by running the process and checking that the device time slices are correct
