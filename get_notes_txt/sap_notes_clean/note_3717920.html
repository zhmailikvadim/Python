Symptom

In the EarlyWatch Alert (EWA) report, the following alert is displayed:

“Archiving is not set up for EWM Objects”

Environment

SAP Solution Manager 7.2

Reproducing the Issue

  1. Generate an EarlyWatch Alert (EWA) report.
  2. Navigate to the EWM section of the report.
  3. Observe the alert:
    Archiving is not set up for EWM Objects :“Missing archiving for recommended EWM objects”

Cause

  • The EarlyWatch Alert (EWA) check determines the archiving status by evaluating entries in table ADMI_RUN, which records the execution of archiving runs for EWM archiving objects.
  • If no entries exist in ADMI_RUN, it indicates that archiving has not been executed at least once for the recommended EWM archiving objects.

Resolution

  1. Set up archiving for the recommended EWM archiving objects using standard SAP archiving tools.
  2. To avoid the alert: Archiving must be executed for at least 15% of the recommended EWM objects
  3. Verify that entries are created in table ADMI_RUN after execution.
