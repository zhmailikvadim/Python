Symptom

When using Set Values or manual entry in the grid, the Action Code value should be updated as expected.

However, once the Action Code is set, it cannot be modified consistently through Set Values or manual grid entry.

This note will fix the issue so that the Action Code can be updated correctly using Set Values or manual entry.

Other Terms

/IRM/GCRM, Action Code, Set Values, manual grid entry, action code determination, inconsistent grid update

Reason and Prerequisites

This is caused by program error

For V4, SAP Note #3704983 is available

Solution

Please implement the correction instructions attached. These corrections are only valid if you have applied the following support packs.

Release 6.4 Base Package

Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object.
