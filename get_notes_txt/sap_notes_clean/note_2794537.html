Symptom

You use the SAP ERP client for E-Bilanz in the above version or you plan to perform an update to this version and require further information about the new functions implemented in this version.

Other Terms

SAP ERP, E-Bilanz, client SP03, ERiC 29, SAP ERP client for E-Bilanz, E-Bilanz with ERP, update, Taxonomy 6.2

Reason and Prerequisites

This SAP Note describes changes with regard to the specified version of the SAP ERP client for E-Bilanz.

Solution


New functions

Version 1.0 Support Package 03 Patch Level 9 (Build 1.0.3.52) of ﻿SAP ERP client for E-Bilanz provides the following changes:

  • Update of the ERiC component to version 29.5.4.0
  • Support of actual case transfer of electronic financial statements (E-Bilanz) with taxonomy version 6.2.

Corrections

  • Compatibility improvements for Office 365

 

Installation requirements

The new version of the ERiC component has resulted in the following changes to the installation requirements.

  • No change. As of Patch 6, the following still applies:
    • Microsoft Visual C++ Redistributable for Visual Studio 2017 (x86)
    • Microsoft.NET Framework 4.6.1

Known restrictions

  • For more information, see SAP Note 1754469.

Installation

This SAP Note describes how to update versions that have already been installed. For an initial installation, follow the instructions from SAP Note 1754621.

 

Update


Proceed as follows to update an earlier version of the product that has already been installed:

    1. Uninstall the previous version of the SAP ERP client for E-Bilanz using Control Panel in Microsoft Windows.
    2. Check whether the installation folder for the SAP ERP client for E-Bilanz has been completely removed. If the folder was not completely removed, remove it manually.
    3. Then, carry out the steps described in the installation guide:
    • http://help.sap.com/boebilanz10 -> "Administrator's Guide" / "Administratorhandbuch SAP ERP client for E-Bilanz" -> "SAP ERP client for E-Bilanz Installationshandbuch" -> "Ablauf der Installation"
