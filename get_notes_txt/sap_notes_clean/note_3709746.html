Symptom

Error on executing the query with parallel mode. The query involves compounded objects, local sids and exception aggregation:

Fatal Runtime Exception occured
An exception was raised.
System error in program CL_RSR_FIPT_TRANSFORMATOR and form GET_MAPPING-01-
Object Collapsed Position in code >> Row: 18 Inc: IF_RSR_FIPT_TRANSFORMATOR~GET_MAPPING Prog: CL_RSR_FIPT_TRANSFORMATOR
Cache in database cannot be used.

Other Terms

Parallel processing, local sids, compounding, CMP, CL_RSR_FIPT_TRANSFORMATOR, layer scenario, layer query.

Reason and Prerequisites

Program error.

Solution

To solve the issue please apply the Support Package as mentioned below. In urgent cases you can use the correction instructions.
