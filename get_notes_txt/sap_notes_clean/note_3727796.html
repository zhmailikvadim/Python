Symptom

User needs to report on Benefit reimbursement claim history.

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Environment

SuccessFactors HCM Suite

Resolution

On Canvas Advanced Report, use the following Tables to Report on Benefit reimbursements claims:

  1. BenefitEmployeeClaim
  2. Global Job Information (For user details such as Name and UserID)
  3. Benefit (For benefit name)
