Symptom

You may get pop-up errors when generating xml file in the following scenario:

Scenario 1: Employee's NS training is after 6 month of hiring , while the NRS claim submission date is within the first 6 months.

"Reason: Element 'item_amt' is unexpected according to content model of parent element 'variable'.
Expecting: mth6."

For example you may encounter this error for the case below:

  • Employee is hired in Jan. 
  • Employee has NRS training in July 
  • NRS claim submission date is in June.

Scenario 2: When executing the program for huge number of employees

"Reason: Element 'item_amt' is unexpected according to content model of parent element 'variable'.
Expecting: formula_string."

Other Terms

 RPCNRSR0_01, Pop-up Error, xml

Reason and Prerequisites

This is a correction on legal function.

Solution

As a general rule, SAP recommends that you install a solution by applying a Support Package. However, if you need to install a solution earlier, use the Note Assistant to implement the correction instruction.
