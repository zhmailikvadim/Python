Symptom

You use the Run Statutory Reports app to run the Annual VAT Declaration - Form 390 report for Spain (report category ES_VAT_ANNUAL_M390_NATIONAL). You notice that the system calculates an incorrect amount for box [108].

Other Terms

ES_VAT_ANNUAL_M390_NATIONAL, Document and Reporting Compliance, DRC, Modelo 390, Box 108, Box 125, Box 126, Box 127, Box 128.

Reason and Prerequisites

This SAP Note is relevant only for Spain.

Solution

After implementing this SAP Note, the system includes the values from boxes [125], [126], [127], and [128] in the calculation for box [108] in the Annual VAT Declaration - Form 390 report.

Version Date Reason
1 2026-02-13 Pilot Release
2 2026-03-02 Release to all customers.
