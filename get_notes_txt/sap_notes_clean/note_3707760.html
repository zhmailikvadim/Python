Symptom

When trying to use a Self-Signed Client Certificate, an error occurs, stating that the connection is Unauthorized.

*Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.*

Environment

  • SAP S/4HANA Public Cloud All Versions

Cause

This is not an error.

Self-Signed Certificates cannot be used for Client Authentication.

Resolution

Self-Signed Certificates cannot be used for Client Authentication.

Instead of this, use a client certificate signed by an appropriate certification authority (CA).

A list of all the root CAs approved by SAP Global Security is available in SAP Note #2801396 - "SAP Global Trust List "

See Also

SAP Note #2801396 - "SAP Global Trust List "
