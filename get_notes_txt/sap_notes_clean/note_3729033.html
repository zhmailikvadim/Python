Symptom

You use Transaction Manager of Treasury and Risk Management. When exercising an FX Option transaction using the new TTM framework, the Cash Settlement section does not provide any field to enter a settlement price.

Other Terms

CL_TTM_OPT_CS_SCREEN, GET_FMOD, TTMS_OPT_SETTLEMENT_UI-RATE

Reason and Prerequisites

Program Error.

SAP Note 3574330 has been implemented or included in your system release.

Solution

Preliminary correction via Correction Instruction or implementation of the corresponding Support Package.
