Symptom

You have submitted the SAF-T Financial version 1.2 file on the Norwegian Tax Authority portal and the following error message is shown:

"The file contains format errors according to the XSD definition. Unknown XSD version: 1.10 ".

Other Terms

SAFTN_G, SAFTN_E, SAFTN_D, SAFTN_M, SAF-T, SAF-T Financial, Skatteetaten, SAFTNI_GEN_TRANS, Run Compliance Reports, Version 1.2

Reason and Prerequisites

The validation of the XSD schema has been updated by the Norwegian Tax Authority.

This SAP Note is relevant only for Norway.

Solution

This note updates the SAP system table SAFTNI_GEN_TRANS with the schema location for the XSD version 1.20.

You can find more information about the Note Assistant in Support, under https://support.sap.com/en/my-support/knowledge-base/note-assistant.html.
