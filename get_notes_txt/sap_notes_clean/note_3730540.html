Symptom

The Simulation functionality is missing in the SAP Signavio Process Manager after the switch to the Rise Modeler license.

Resolution

The functionality is no longer available with the new Modeler license. The reason for the removal of the functionality for new license types is that a general offboarding is planned for the simulation functionality. It is not possible to activate the functionality separately for a workspace with the new license type.

In case more clarification is needed, please reach out to your Account Executive or the Customer Success Partner.
