Symptom

Header text for document posted by G/L balance valuation (transaction FAGL_FCV/FAGL_FC_VAL) is consists of character/number.

Environment

  • Financial Accounting (FI)
  • SAP R/3
  • SAP R/3 Enterprise 
  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA Finance
  • SAP S/4HANA

Cause

In case of G/L balance valuation the document header text consists of the following parts:
Character 1- 2: valuation area 
               3-10: valuation key date 
              11-12: run-ID (from table FAGL_BSBW_HST_BL, field RUNID)   
              13-22: G/L account number

Resolution

This is standard system design.

Keywords

FAGL_FCV, FAGL_FC_VAL, foreign currency valuation, balance valuation, header, text, BKTXT
