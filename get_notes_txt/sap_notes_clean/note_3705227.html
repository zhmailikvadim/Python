Symptom

- Error in job logs of /UI5/UPD_ODATA_METADATA_CACHE
- Job logs show: 5 succeeded, 0 not supported, 266 unknown, and 0 with errors
- Warning information (unknown) not changed to informational message after applying notes
   3533167 - Issues related to report /UI5/UPD_ODATA_METADATA_CACHE
   3087369 - OData Metadata Cache job log shows wrong warning messages
   2917307 - Backend Context Token - Disable logs for Unknown services/Annotation files

Environment

- SAP NetWeaver
- SAP Fiori

Cause

- Missing authorization for the batch user
- Batch user unable to read information about V2 OData Services

Resolution

1. Check the application log in transaction SLG1 with Object /UI2/BE
2. Provide the batch user the Authorization Object /IWFND/ADM with ACTVT and Value '03'

Keywords

/UI5/UPD_ODATA_METADATA_CACHE, Job logs, Missing cache token, Authorization, SLG1, /UI2/BE, /IWFND/ADM
