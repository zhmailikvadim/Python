Symptom

If the Access Control is connected to Employee Center Payroll application, the last updated on date in Access Control will be used instead of the last updated on in target system(Employee Center Payroll application) during the respository sync from Access Control to IAG.

Other Terms

AC to IAG repository role sync,  SAP Identity Access Governance, EC payroll

Reason and Prerequisites

Use AC last update on date for EC Payroll target application in the AC to IAG repository role sync if AC target system is EC Payroll system.

Solution

Please apply the correction included in the note
