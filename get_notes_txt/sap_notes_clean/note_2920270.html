Symptom

If regressions are known for kernel 773 patch level 251, they are listed in this SAP Note under the "References" section.

This list is updated as soon as any new regressions are discovered, and therefore should be consulted regularly.

Other Terms

Kernel, error list, disp+work, KRNL773PL251, regression

Reason and Prerequisites

Regressions have been discovered for kernel 773 patch level 251.

Regression is an error that was caused as an unwanted side effect of a previous kernel patch.

The known regressions are listed below in the "References" section.

Solution

Examine the individual SAP Notes listed in the "References" section, in order to determine:

  • What regression errors are currently known for kernel 773 patch level 251.
  • Whether a workaround exists that can be used to help the specific regression at the currently used kernel PL.
  • Whether a new kernel PL is available fixing the specific regression.

There is a regression note for each single Kernel PL released (further information in note 1802333).

The list of all changes (error corrections and improvements) is still available at the usual place (info file) in the SAP Support Portals.
