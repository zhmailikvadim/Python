Symptom

This SAP Note provides you with relevant system changes to implement the Fiori Application of Download & Upload, called as Meter Reading Tracking, for IS-Utilities Brazilian Localization.

The creation of this functionality allows IS-Utilities Brazilian Localization users to know the amount of orders selected and delivered to external systems.

Other Terms

IS-U, Utilities, Brazil, electricity, electric energy, water, gas, Fiori, App, Download, Upload

Reason and Prerequisites

By installing this SAP Note, you have the implementation and objects to the Download & Upload Fiori App for IS-Utilities Brazilian Localization, to meet the requirements mentioned in the Symptom section.

As a prerequisite, you have applied the notes referenced in SAP Note 3052803 - Central Note for IS-U Brazilian Localization Phase I, or have installed the version of S/4HANA mentioned in validity section.

This SAP Note is relevant only for IS-U Brazil.

Solution

Use transaction SNOTE to implement the attached correction instructions, which consists in implementation of Meter Reading Tracking App that will meet the requirement to track the root cause of orders not delivered, if it happens in the standard selection or in the interface with external systems.

As a general rule SAP recommends that you implement the Support Package specified for your release. However, if you need to install a solution earlier, use the SAP Note Assistant to implement the correction instruction. You can find more information about the SAP Note Assistant in SAP Support Portal, under SAP Note Assistant.
