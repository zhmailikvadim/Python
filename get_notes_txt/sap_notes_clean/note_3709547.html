Symptom

When creating a note and a user forgets to select a language and clicks 'Create', the dialog prompts the user to select a language. However, after selecting the language and clicking 'Create' again, the dialog does not allow the creation. The user is unable to proceed

Other Terms

 Dialog, Notes, Combobox

Reason and Prerequisites

 This issue is caused by a bug in the language selection logic

Solution

Import the relevant Support Package or perform the manual activities.

Other Terms

F1868, download, iFrame, timeout, cancel

Reason and Prerequisites

The issue occurs for large payment media files or on slow systems.
Download is performed by iFrame, which is removed from DOM before downloading finishes.
This cancels request and whole download process.

Solution

Implement the attached Correction Instruction or the mentioned Support Package.

Afterwards reindex the application on your frontend (gateway) system as described in SAP Note no. 2227577 and clean the caches according to SAP Note no. 2017117.
