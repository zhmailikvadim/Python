Symptom

When creating a note and a user forgets to select a language and clicks 'Create', the dialog prompts the user to select a language. However, after selecting the language and clicking 'Create' again, the dialog does not allow the creation. The user is unable to proceed

Other Terms

 Dialog, Notes, Combobox

Reason and Prerequisites

 This issue is caused by a bug in the language selection logic

Solution

Import the relevant Support Package or perform the manual activities.
