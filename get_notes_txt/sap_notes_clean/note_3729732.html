Symptom

After double-clicking the results from RHINTE10, no detail appears on the screen. "Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Environment

  • Human Capital Management (HCM)
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for HANA
  • SAP S/4HANA
  • SAP SuccessFactors Employee Central Payroll

Reproducing the Issue

  1. Run RHINTE10 via T-code SE38.
  2. Double-click on the result, there is no details displayed.

Cause

This is a program error.

Resolution

Implement SAP Note 3650795 - ALV: Custom buttons disappeared or have no effect when clicked.

Keywords

RHINTE10, detail screen, pop up, SE38.
