Symptom

  • IDoc for posting Goods Issue fails with error VU010: "Document is incomplete: You cannot post goods movement", even though the IDoc passes all the incompletion fields.
  • If the same data is input in VL02N, the Goods Issue is created.

Environment

  • Sales And Distribution (SD)
  • Logistics Execution (LE)
  • SAP R/3
  • SAP R/3 Enterprise 4.7
  • SAP ERP Central Component
  • SAP ERP
  • SAP Enhancement package for SAP ERP
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA

Reproducing the Issue

  1. Use IDoc for testing.
  2. Attempt to post Goods Issue via WHSCON IDoc.
  3. Observe the error message "Document is incomplete: You cannot post goods movement".

Cause

In standard, the function `IDOC_INPUT_DELVRY` does not process the segment `E1ADRM1`. Therefore, even the partner information is maintained in this segment, system will not handle it and update the data into the delivery.

Resolution

To address your concern, if the partner function (e.g. Forwarding Agent) is required for the delivery process, it is recommended to input the value directly into the delivery before proceeding with the GI posting via WHSCON IDOC. Alternatively, if you intend to update the Forwarding Agent number through the WHSCON IDOC process, this would necessitate a custom modification on your end.
