Symptom

In the IAM Information System app, when maintaining "Restriction" as the main entity, the "Restriction Value" column is not visible in the resulting table, however, it is available in the filter searching section as "Restriction Value". 

Environment

SAP S/4HANA Cloud Public Edition 2602

Reproducing the Issue

  1. Open IAM information system
  2. Maintain main entity as "Restriction"
  3. Resulting table does not display Restriction Value Column

Resolution

A fix will be delivered with HFC06(14/15 March).
