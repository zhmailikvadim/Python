Symptom

When running payroll (HARCALC0) with off-cycle reason AJUS for an employee that has been terminated, wage type Acum.impuesto s/cta.ctes. (/484) is being reported twice for the period of termination.

Other Terms

&TERMINACION; (further information see note 1152671)

Reason and Prerequisites


The following Support Package level is the minimum prerequisite:

ReleaseSupport Package
6.00 HR SP A1
6.04 HR SP 67

 (For more information, click on "R/3 HR Support Package Schedule" in www.service.sap.com/ocs-schedules.)

Solution

From now on, the system will only consider the value generated by the off-cycle for wage type Acum.impuesto s/cta.ctes. (/484).

As general rule, SAP recommends that you install a solution by applying a Support Package. However, if you need to install the solution earlier, proceed as follows:

    1. Use Note Assistant (SNOTE) to implement the correction instructions.


You can find more information about the Note Assistant in SAP Service Marketplace, under service.sap.com/note-assistant.
