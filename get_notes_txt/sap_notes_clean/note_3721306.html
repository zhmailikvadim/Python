Symptom

Error appears while importing the field "Ok to Rehire" to Yes, for temporary assignments. 

Environment

SAP SuccessFactors Employee Central

Reproducing the Issue

  1. Navigate to Import Employee Data.
  2. Select Termination Details and choose the file.
  3. Observe the error during validation/import: "XXX is an invalid external code for event reason."

Cause

Expected behaviour. 

Resolution

After a higher duty or temporary assignment is terminated, it cannot be reinstated (re-hired). Hence, the system will show an error and will not allow to edit the records. Additionally, Higher Duty does not support the import of Employment Termination Information; therefore, the import will fail. 

See Also

Ending or Deleting a Higher Duty or Temporary Assignment | SAP Help Portal
