Symptom

  • Error message: "The System Owner cannot be updated at the moment. Please try again later. If this condition persists, please submit an SAP support ticket using the component LOD-ANA-BI, or have your administrator submit one for you." happens when transferring it in a tenant.

Environment

  • SAP Datasphere

Reproducing the Issue

  1. Navigate to Security -> Users;
  2. Select an user;
  3. Click 'Assign as System Owner';
  4. Click 'Ok';

Cause

Resolution

Make sure to assign a role to the previous system owner in order to proceed with transfer.

The role need to include "System Information" Update privilege.
