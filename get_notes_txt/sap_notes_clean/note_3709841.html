Symptom

  • There is a need to troubleshoot connectivity issues between BTP and SAP systems during adaptation project creation or deployment via Cloud Connector (SCC).
  • There is a need to know how to test connectivity using curl commands in the Business Application Studio (BAS) Terminal.
  • There is a need to use traces from Cloud Connector and backend systems to diagnose connectivity problems for adaptation project.
  • There is a lack of understanding on how to start analyzing connectivity issues when creating or deploying adaptation projects to BAS.

Environment

  • SAP Business Application Studio (BAS)
  • SAP Business Technology Platform (BTP)
  • SAP Cloud Connector (SCC) 
 
 
 

Resolution

Refer to the following SAP wiki: Troubleshooting connectivity in an Adaptation project.
It goes through the different ways one can troubleshoot connectivity issues between BTP and SAP systems via Cloud Connector (SCC) during adaptation project operations, and contains three topics as below:
  • Troubleshooting in BAS  
  • Troubleshooting in Cloud connector 
  • Troubleshooting in the on-premise system
