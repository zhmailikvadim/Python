Symptom

  • I try to post a job via Recruiting Posting, but an error occurs when filling some fields.
  • I have completed all the required fields but cannot validate my posting.

Environment

SAP SuccessFactors Recruiting Posting

Resolution

In order to investigate further why the error occurs and before raising a ticket to the Support team, please collect and provide the following details:

  • User who is trying to access Recruiting Posting (name, e-mail address)
  • Job posting (job reference and job title)
  • Job boards/schools selected on the first step
  • Fields values (both common and specific fields)
  • Screenshots of the error message
  • Full support user access and company ID
  • Any additional relevant information
