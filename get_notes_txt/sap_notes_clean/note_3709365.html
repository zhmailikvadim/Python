Symptom

You are encountering error "Service cannot be used" after added an Calculation Measure in Custom Analytical Query.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Go to application Custom Analytical Queries.
  2. Select the required custom analytical query.
  3. Add-> Calculated Measure.
  4. Edit the Expression of the Calculated Measure.
  5. Save and Preview.
  6. Error "Service cannot be used".

Cause

You have input element which type is not a measure but a dimension in the expression of the calculated measure in custom analytical query.

Resolution

Elements which type is a Dimension cannot be used for Calculated Measure. This is system standard behavior.

You can add required calculation directly to the data source of the custom query which is a custom CDS view based on your business need, and then view it in your custom query.

Keywords

dimension, measure, calculated measure, custom analytical query, custom CDS view
