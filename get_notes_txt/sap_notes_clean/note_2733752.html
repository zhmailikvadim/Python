Symptom

While creating a DataStore Object (advanced) it may not be possible to find all Source System Types

In Detail:

1) Create a DataStore Object (advanced) by using the Create Wizard
2) Use DataSource as Template
3) Search for SourceSystem
Issue: Not all kinds of Source Systems are found

Other Terms

DataStore Object (advanced), DataSource as Template, Source System, Error when getting results: Variable 'searchtype' not found in the uri template for value help(discovery)

Reason and Prerequisites

Program error

Solution

Apply the support package related or implement the correction instruction attached.

To solve the issue, apply the following Support Packages:
