Symptom

1.Technical objects (functional locations/equipment) linked to a work order operation through a notification are not displayed in SAP Business Network Asset Collaboration (BNAC).

2.The actual work for a work order operation is displayed in SAP Business Network Asset Collaboration (BNAC) only when final confirmations are available in ERP (S/4/ECC).

Other Terms

BNAC, work order operation

Reason and Prerequisites

 Advance development

Solution

Implement the note to resolve the above mentioned issues.

After implementing this note:

  1. Technical objects (functional locations/equipment) linked to a work order operation through a notification will be displayed in SAP Business Network Asset Collaboration (BNAC).
  2. The actual work for a work order operation will be displayed in SAP Business Network Asset Collaboration (BNAC) when partial or final confirmations are available in ERP (S/4/ECC).
