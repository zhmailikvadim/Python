Symptom

In the result of the compare run with transaction SCU0 one of these message are raised:
No remote SM30 editing dialog
Remote view/table not activ


Environment

This KBA applies to all SAP Products based on NetWeaver and S4H.

Reproducing the Issue

  1. Perform a compare run with transaction SCU0
  2. In the result list, below columns Entries one of these messages are reported:
    Remote view/table not activ
    No remote SM30 editing dialog

Cause

Data comparison is unavailable as the objects do not exist in the target system of the compare.

Resolution

  1. Logon to the target system of the compare
  2. Execute transaction SE11
  3. Enter the name of the table in field Database table or the name of the Maintenance View in field View
  4. Press button Display
  5. Messages E2007 ([object name] does not exist. Check name) or E2503 ([object name] does not exist) indicate that the comparison objects are missing from the target system. For this reason, a comparison cannot be performed.

Keywords

SCU0, adjust, compare
