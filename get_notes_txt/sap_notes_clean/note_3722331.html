Symptom

When creating an Agreement Request with reference to an existing Agreement from the DO, the validity dates maintained in the DO version are not being copied to the newly created Agreement Request. Instead, the Agreement Request continues to display the older validity dates from the original agreement rather than the updated dates from the DO version.

Other Terms

DO Posting, /IRM/IPAGR_INFOCUS_SET, /IRM/IPAR_INFOCUS_SET, Agreement Request Validity, /IRM/IPAGRM

Reason and Prerequisites

This is caused by program error.

SAP Note # 3722332 is available for ECC Releases.

Solution

Please implement the correction instructions attached. These corrections are only valid if you have applied the Support Packs.

Release V4 - 2023 V3 and 2023 V3 Support Pack 1.

Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object.
