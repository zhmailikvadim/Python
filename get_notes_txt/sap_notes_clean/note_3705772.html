Symptom

The web channel campaign is displaying the form header in the block template, even though the header was left empty (blank) in the form configuration.

Environment

SAP Emarsys Customer Engagement 

Reproducing the Issue

  1. Create a new web channel campaigns;
  2. Add the form in the content block;
  3. Save the changes;
  4. Preview the web channel campaign;
  5. he form header is displayed as 'Registration' and not empty. 

Cause

This is fall-back mechnism of web channel in case the form as no header.

Resolution

To overcome this scenario, it's necessary to modify the web channel block template to add the following code in the "Styles" tab:

form .wpst-title {
  display: block;
  font-size: 0;
  height: 0;
  overflow: hidden;
}

Keywords

Form header, web channel
