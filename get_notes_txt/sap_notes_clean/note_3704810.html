Symptom

In the process "Start of Basic or Backup Supply for Electricity (SUP View)" (DE_B_GPKE_SBS_SUP), the process continues at step 0760 even when the GP clarification (BPEM) case hasn't been processed yet.

This issue is relevant for the market role: SUP.

Other Terms

DE_B_GPKE_SBS_SUP, step 0760, step 1100, BPEM, open clarification case, 2504, SBS, Start of Basic or Backup Supply for Electricity

Reason and Prerequisites

The error occurred because of a configuration error in the system.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the system correctly waits for the GP clarification (BPEM) case to be processed before continuing the process at step 0760.

If your S/4HANA Product Support Package is SAPK-10207INUS4G, activate the BC set /US4G/3_0_SUP_SP07_NOTE_3704811 as follows:

1. Run transaction SCPR20.
2. Enter the BC set /US4G/3_0_SUP_SP07_NOTE_3704811.
3. Select the **Activate** button or choose **BC Set** → **Activate** from the menu.

You can verify the fix by checking that the process waits at step 0760 until the GP clarification (BPEM) case is processed before continuing.
