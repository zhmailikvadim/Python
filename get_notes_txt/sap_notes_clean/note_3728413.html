Symptom

In an SAP system upgrade with SUM(Software Update Manager), the phase ADDON_LANGINC comes up and asks for language packages for Addon components.

Environment

SAP ABAP System.

Cause

SUM process finds a language is installed in the system,  but the certain language info is not found for the addon components listed in the current upgrade download directory.

Resolution

SUM process find a language is installed in the system,  but the certain language info is not found for the addon components listed in the current upgrade download directory.

So the phase ADDON_LANGINC comes up, in case some further language packages for those Addons can be included.

It's not mandatory to include language packages for those addon components. If individual language packages are released for those addon components, include them. If no individual language packages are released at all,  just continue without additional language packages.
