Symptom

Candidates with future Hire Date visible in Canvas/Story reporting

Environment

SAP SuccessFactors Onboarding

Reproducing the Issue

  1. Create canvas/Story report
  2. Candidate coming from Onboarding is visible in Manage Pending Hire with a future Hire Date

Cause

Since Onboarding data persisted in EC entities, candidate with future Hire Date is included in the reports.

Although this could be an unexpected behavior from the customer's perspective, it is what is happening. 

Resolution

If this is unexpected for customer, there are two possible workarounds:

  1. Remove the External Users Target Population
  2. Apply filters on the user status, as per described below:

Table report:

  1. Edit report created on Person and Employment Export domain (In Table reporting Future Hire dated records are visible only in Person and Employment Export)
  2. Filters > Filter Group 1 > Refine Criteria 
  3. Employee Information > find field "status" 
  4. In the popup - By My Selection
  5. Select all the applicable user statuses depending on the types of users that you would like to see in the report

Canvas report:

  1. Edit report
  2. Select Table > Edit Component > Edit Query
  3. Global Job Information
  4. find field "The Hiring Process Is Not Yet Complete"
  5. Add a filter criteria based on this
