Symptom

  • Error RECTM038 occurs when associating the same fixed asset with a new contract for continuity of accounting and depreciation.
  • The error message RECTM038 indicates that the asset is already used in other contract.

Environment

  • Contract and Lease Management
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA Finance
  • SAP S/4HANA

Reproducing the Issue

  1. Execute transaction code RECN/RECECN.
  2. Associate asset already used in another contract at the new contract.
  3. Observe the error message RECTM038.

Cause

  • The system prevents the reuse of an asset already assigned to a closed contract to ensure accounting integrity.
  • Transferring a lease asset to a new contract is not supported. 

Resolution

  1. Make sure that the previous contract is thoroughly closed and confirm that the associated asset has been closed.
  2. When creating a new contract, even if it functions as a renewal or extension, a new asset must be assigned. 
  3. For transition scenarios, refer to SAP Note 2544232 - Leasing: Handling transition scenarios.

See Also

  • SAP Note 2544232- Leasing: Handling transition scenarios.
  • SAP Note 2662137- SAP Contract and Lease Management (SAP RE-FX) - FAQ note.
