Symptom

  • The DNS configuration for the custom sender address in Career Site Builder (CSB) has been completed successfully, with all four DNS settings showing as active (green).
  • Despite the DNS configuration being active, the "Use Custom Sender Address" option remains unavailable.

Environment

SAP SuccessFactors Recruiting Marketing

Reproducing the Issue

  1. Verify that all four DNS configuration settings are active (green).
  2. Attempt to enable the "Use Custom Sender Address" option.

Cause

DNS Configuration for Use of Custom Sender Address is not containing the expected values.

Resolution

The slider is disabled until the email infrastructure is set up correctly. If you can't turn the setting on, use the Verify DNS Configuration for Custom Site Sender Address section to double check your DNS configurations. All entries must appear as active and the entries matching the expected values.
This a technical process that's best performed by your organization's IT personnel, or someone with technical knowledge of your email infrastructure and access to your organization's DNS provider account.
