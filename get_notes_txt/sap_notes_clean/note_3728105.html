Symptom

When displaying results in the Query Preview form, the time in Creatn Time - Incl. Secs column is displayed in a wrong format.

Reproducing the issue

Scenario: 

  1. In Administration → System Initialization → General Settings → Display tab, set Time Format as 12H.
  2. In Sales - A/R → Sales Order, create Sales Order No.: 1. The system time when creating this Sales Order is 07:39:59 AM, for example.
  3. Open Tools → Queries → Query Generator and execute the following detection query: SELECT CreateTS, * FROM ORDR WHERE DocNum=1 

Expected behavior:

The Creatn Time - Incl. Secs of the Sales Order No.: 1 is displayed as 7:39:59AM.

Actual behavior:

The Creatn Time - Incl. Secs of the Sales Order No.: 1 is displayed as 27:59PM.

Solution

SAP intends to provide a feature package or patch to solve the symptom that is described in this SAP Note.
