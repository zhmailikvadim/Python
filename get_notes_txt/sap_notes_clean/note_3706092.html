Symptom

In the processes "End of Supply for Electricity (DSO View)" (DE_B_GPKE_DSO_EOS) and "Sending of Deregistration Due to End of Supply for Electricity (DSO View)" (DE_B_GPKE_DEREG_DSO), the system incorrectly triggered the start of backup supply in step 2420 for the shutdown scenario "EndBackupSupWithoutFollowOnSup" with transaction reason Z41. In this deregistration scenario, no follow-on supplier should be assigned.

Other Terms

GPKE, DE_B_GPKE_DSO_EOS, DE_B_GPKE_DEREG_DSO, transaction reason Z41, backup supply, basic supply, deregistration, shutdown scenario, EndBackupSupWithoutFollowOnSup, step 2508, step 2508_00

Reason and Prerequisites

Program Error

Solution

This issue is relevant for the market role: DSO.

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the system no longer triggers the start of backup supply for transaction reason Z41 in the end of supply process. Import the BC set /US4G/3_0_DSO_SP07_NOTE_3706092 to apply the fix. You need to retest the scenario. You can verify the fix by checking that the start of backup supply is not triggered for any shutdown transaction reason.
