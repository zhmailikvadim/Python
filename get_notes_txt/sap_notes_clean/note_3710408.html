Symptom

The reminder email notifications are not getting triggered for the I9 reverification task using "(ONB) I9 Section 3 Initiated For Employee Trigger" trigger:

  • In case when you add other roles in recipient builder in addition to "New Hire"

Environment

SAP SuccessFactors Onboarding 2.0

Reproducing the Issue

  1. Initiate I9 reverification task for a candidate
  2. Email notification gets triggered
  3. The reminder notification does not trigger according to the reminder configurations set in the email template

Cause

This is a known issue.

Resolution

This issue has been identified for a Code Correction, the Engineering team is working on it, (Jira ID: OBX-42038), the fix will be deployed in the upcoming release: 1H202.

If this doesn't get resolved post the above mentioned timeline, please create a support case under the component: LOD-SF-OBX-NF to check for the latest updates on it.
