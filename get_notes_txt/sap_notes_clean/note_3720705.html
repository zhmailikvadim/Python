Symptom

In the process "Adjustment of Billing-Relevant Intelligent Metering System" (DE_B_MCM_IMS_BILL_ADJ_PROCESS), the parent process waited for the completion of the follow-up process in step 0050. If the follow-up process was stuck, the parent process was consequently blocked, preventing the overall process from being completed.

This issue is relevant for the market role: DSO.

Other Terms

MCM_DE, DE_B_MCM_IMS_BILL_ADJ_PROCESS, follow-up process, parent process, billing adjustment, intelligent metering system, IMS, BC set

Reason and Prerequisites

The error occurred because of a configuration error.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the parent process no longer depends on the completion of the follow-up process, allowing it to complete independently. Import the BC set US4G_MCM_3_0_COM_2602_3720705.

You need to retest the scenario. You can verify the fix by checking that the parent process completes successfully even when the follow-up process is stuck.
