Symptom

  1. Alternative language is not visible for last name on Latest People Profile.
  2. After click on view all, it is visible for both first name and last name
  3. It works correctly on PP3

Environment

SAP SuccessFactors Employee Central

Reproducing the Issue

  1. Go to personal information section
  2. "Alternative language" is visible only for first name field.
  3. Click on view all
  4. Alternative language is visible for both first name and last name field.

Resolution

The Engineering team has confirmed that this behavior is due to a regression issue.

The fix is scheduled to be delivered in an upcoming 1H2026 release 

Keywords

Alternative language, Latest People Profile, Legacy People Profile, First name, Last name, PPX, PP3
