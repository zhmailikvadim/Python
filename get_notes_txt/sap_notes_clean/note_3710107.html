Symptom

An error occurs when attempting to enable MS Teams integration via Administrator > General Settings > Microsoft Teams > Enable Microsoft Teams Integration > User delegation > login with Outlook Admin credentials.

Environment

SAP Cloud for Customer

Microsoft teams integration

Reproducing the Issue

  1. Navigate to Administrator > General Settings.
  2. Under Microsoft Teams, click on "Enable Microsoft Teams Integration".
  3. Select User delegation.
  4. Login with Outlook Admin credentials to approve the app
  5. Encounter the error.

Cause

The issue occurs due to the absence of a maintained email address for the the given admin user.

Resolution

  1. Maintain a valid Microsoft email address for the admin user.
  2. Retry the integration process.
