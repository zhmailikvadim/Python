Symptom

In S/4 Hana releases the function module READ_TEXT for certain format types like DOC, RTF after the formatting error V2 202 is displayed does not return a header line in the internal table LINES.

When the user either double clicks on the text area or selects Detail button, the text does not open up in word or customized editor and cannot be displayed.

Other Terms

VX202, Text is formatted -> Details, VA03 ,Text Edit Control, tlinetab, TDFORMAT, header line.

Reason and Prerequisites

Program error. There was a design change in the data returned from the function READ_TEXT for certain formats.

Solution

Implement the attached correction instruction.
