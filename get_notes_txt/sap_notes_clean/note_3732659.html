Symptom

While posting the revision, the system does not display the specific condition type authorization error when the authorization check fails. Instead, it only shows the message 'Revision cannot be posted', which prevents the user from identifying the exact reason for the failure.
This note ensures that the authorization error is also displayed during posting.

Other Terms

/IRM/IPPQM, revision_rules_post, Revision Post, V_KONH_VKS, /IRM/GPRM 027, /IRM/IPPQRVM

Reason and Prerequisites

This is caused by program error.

Note : SAP Note#3732693 is available for ECC release.

Solution

Please implement the correction instructions. These corrections are only valid if you have applied the following Support Packs.

Release V/4 – 2022 V2, 2023, 2023 V2, 2023 V3 and 2023 V3 Support Pack 1

Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object
