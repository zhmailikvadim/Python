Symptom

  • HMRC have issued an updated version of the Starter Checklist for use from the 6th of April 2026.

  • The following changes will be added to the onboarding version of the form to enable employees to select the correct options.

  • The changes to the attached form include:

    • New Student Loan plan type 5
    • Changes and guidance to make it clear that employees should only select one plan type - even if the employee has multiple plan types
    • Clearer instructions for the employer should their employee select multiple plan type.

Environment

SAP SuccessFactors Onboarding

Resolution

The new 2026 version Starter Checklist form has been deployed to all instances.

Onboarding Compliance Form Updates

To verify if the form has been deployed, please go to Manage Data >> Compliance Form and check if UK_STARTER_CHECKLIST_2026_V1 is there.
