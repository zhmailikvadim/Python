Symptom

While processing the AE2 with cancellation of MILSTRIP messages for subcontracting POs, the process ends with the error code ME866 -"Deletion of item with quantity 0 not possible in case of subcontracting".

Other Terms

PSDMP, MILSSTRIP, /PSFMS/RECEIVE, Subcontracting PO, AE2 Processing.

Reason and Prerequisites

This error occurs while deleting the scheduling lines along with the PO line item during the MILSTRIP cancellation process during the subcontracting PO item processing ( EKPO-PSTYP = 'L' ).

 

Solution

Implement the correction Instruction to fix the issue.
