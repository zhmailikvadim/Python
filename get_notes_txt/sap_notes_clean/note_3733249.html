Symptom

When creating a new contract from Fiori or modifying an existing contract by adding a new line item, the SDM version field in table /ACCGO/T_CAKITM is not populated correctly. Instead of being updated with the expected value '01', the field remains blank.

Replication Steps:

  1. Create a new contract of any type in Fiori.
  2. Save the contract.
  3. Open table /ACCGO/T_CAKITM and search for the newly created contract.
  4. Observe that the SDM version field is blank, whereas it should be updated as '01'.

Other Terms

Agricultural Contract Management; ACM; SDM version; /ACCGO/T_CAKITM

Reason and Prerequisites

 Program error

Solution

Import the relevant support pack or apply the correction instruction(s).
