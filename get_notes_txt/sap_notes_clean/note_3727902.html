Symptom

Teams has been maintained in the Manage Teams and Responsibilities App as described in SAP Documentation Manage Sales Plans, some users are assigned as this Team's member. A user who doesn't belong to the team cannot get the team returned in F4 help for field "Shared with Team".

Environment

  • Sales and Distribution (SD) 
  • SAP S/4HANA 

Reproducing the Issue

  1. Navigate to the "Manage Sales Plans" application (F2512).
  2. Create a value plan or a quantity plan.
  3. Click F4 for field 'Shared with Team', there is no result returned.

Cause

  • CDS View for the F4 help is C_SalesPlanTeamVH which has a condition that userID is current user.
  • The issue occurs because the user attempting to share the Sales Plan is not a member of the team. The F4 help for the "Shared with Team" field checks whether the current user belongs to the team.
  • This is described in SAP documentation Manage Sales Plans :
    =========================
    You can only assign a team that you are a member of.
    =========================

Resolution

  • Assign the user to the Team.
  • Alternatively, create sales plans using the userID that is a member of the Team.
