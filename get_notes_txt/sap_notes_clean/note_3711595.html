Symptom

The issue is customer find report CS_BOM_VALID_TO_CALC has different behavior in mode "validity for all BOMs" and "validity for selected BOM".

  • With validity for all BOMs, VALID_TO not updated for sharing STLKN items -> correct. 
  • If specific BOM is entered in mode "validity for selected BOM", VALID_TO got updated for sharing STLKN items -> incorrect. 

Other Terms

 CS_BOM_VALID_TO_CALC, STLKN, VALID_TO

Reason and Prerequisites

Program error.

Solution

Implement the attached correction instructions.
