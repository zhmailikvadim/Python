Symptom

How can a custom column (for example, Employee Information) be added as an element in the Budget Assignment page?

Environment

  • SAP SuccessFactors Compensation
  • SAP SuccessFactors HCM Core 2505

Reproducing the Issue

  1. Navigate to the Budget Assignment page in SAP SuccessFactors Compensation.
  2. Attempt to add a custom column to display additional Employee Information elements such as Unit Code/Legal Entity.

Cause

Expected behavior.

Resolution

Adding a custom column in the Budget Assignment page is not supported due to system design constraints.

Keywords

employee information, budget assignment page, custom column, SAP SuccessFactors Compensation, enhancement request, Unit Code, Legal Entity, legacy design, system behavior, SAP Influence Campaign.
