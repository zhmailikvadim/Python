Symptom

In the Fiori Launchpad, within the SNP document, an Action Code field is available for selection. However, after selecting an action code and clicking Save, the changes are not retained when navigating back to the document page. This note corrects the save behavior to ensure the selected action code is properly stored and displayed.

Other Terms

/VTA/BDOSNP ,Fiori Launchpad, Action Code Not Saved, Fiori UI Issue

Reason and Prerequisites

 This is caused by Program error.

Solution


Please implement the correction instructions. These corrections are only valid if you have applied the following Support Packs.

Release V/4 – 2023 V2, 2023 V3 and 2023 V3 Support Package 1

Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object
