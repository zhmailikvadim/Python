Symptom

You are using the new UI5 Scripting Editor in CRM IC. When creating a new question, if you add all the question and answer sections first and then fill them in afterward, instead of filling in each section as you add it, their order may be incorrect when you reopen the question after saving and closing it.

Other Terms

 CRM_IC_UI5_SCR, CRM_IC_SCRIPT_EDITOR, _bIsNewSection, _addQuestionArea

Reason and Prerequisites

 Program error

Solution

 Apply attached correction instructions
