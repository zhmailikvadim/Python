Symptom

Message "Too many bank statement items were selected" in FEB_BSPROC or FEBAN happens when selecting over 10000 items.

Environment

SAP S/4HANA

SAP ERP

Reproducing the Issue

1. Display bank statement items in FEBAN/FEB_BSPROC
2. Only 10000 ibank statement items get displayed or "Too many bank statement items were selected" displayed

Cause

After note 3372872, this limiation is imported in FEBAN

Resolution

If it needs to display more than 10000 items in FEB_BSPROC. You can set user parameter FEBAN_MAX_ITEMS in SU01 to the number of items to be displayed in FEBAN/FEB_BSPROC.
