Symptom

Line-oriented authorization check is activated for warehouse number in EWM. User try to maintain condition records. The authorization check is missing. The implementation for the warehouse number authority check (data element /SCWM/DE_LGNUM_COND, BadI /SAPCND/ROLLNAME) is not called.

Other Terms

/SAPCND/IF_EX_ROLLNAME~AUTHORITY_CHECK /SCWM/CL_IM_I_LGNUM_COND

Reason and Prerequisites

The value for  'Implementation Type for Maintenance' in customizing 'Create Field Catalog' for the field 'Warehouse number' is set to '2' (Default Attribute Conversion).

Implementation type = '2': Field processing is partly generic, that is, the field transport (= attribute conversion), an implementation is not required, but is required for the field check, input help and short texts. Therefore, an implementation for the BAdIs /SAPCND/ROLLNAME or /SAPCND/MNT_USG_ROLL for the named methods is necessary in order to avoid runtime errors. The method for the attribute conversion (ATTRIBUTE_CONVERSION) is not called during runtime.

Solution

Implement the attached Correction Instruction or apply the corresponding Support Package.
