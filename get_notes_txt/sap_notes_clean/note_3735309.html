Symptom

I have followed the instructions provided in SAP Integration Suite, Managed Gateway for Spend Management and SAP Business Network - SAP BTP NEO to CF Migration and all steps are completed. How to confirm if I am ready for the Managed Gateway for Spend&Network migration from SAP Neo to SAP Cloud Foundry environment? Should I disconnect the existing Neo subaccounts to test if transactions will go through the new CF subaccounts? 

Resolution

A successful Cloud Connector connection to the new CF subaccounts is sufficient for migration preparation. Do not disconnect the existing Neo subaccounts for testing. All transactions are still routed through the existing Neo subaccounts to SAP ERP systems in both test and production environments. Transactions will be routed to CF subaccounts only from the planned migration date. There is no option to control or route transactions to CF subaccounts for testing.
