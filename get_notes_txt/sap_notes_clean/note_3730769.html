Symptom

  • Filtered option is not visible under Organizational Updates in the New Home Page, despite having multiple folders.
  • The issue only occurs when there are only two folders under the Organizational Updates - with three or more folders, the filter is visible

Environment

  • SAP SuccessFactors HCM Suite
  • SAP SuccessFactors HCM Suite 2511

Reproducing the Issue

  1. Log in to SuccessFactors
  2. Navigate to the Home Page.
  3. Scroll to the Organizational Updates section and check for the filtered option.

Cause

It is currently a known issue that the filter option appears only when there are three or more folders configured in Manage Home Page -> Manage Organizational Updates, and each folder contains cards. With only two folders, the filter isn’t displayed.

Resolution

  • The expected behavior is that the filter option should be visible with only two folders as well.
  • The engineering team already developed a fix for the issue, which will be applied with the 2605 release (Preview: April 13th, Production: May 15-16th).
