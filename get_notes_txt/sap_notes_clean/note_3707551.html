Symptom

Error occurred in period consumption calculation when installing IMS device info record, specifically if the period consumption in base meter doesn't exist

Other Terms

Period Consumption, IMS, DIR,SET_AUTO_ON_INSTALL , CALC_PERIOD_CONSUMPTION

Reason and Prerequisites

 Program error

Solution

Please implement the correction instruction in the SAP Note, or import the higher version of Support Package.

please note that the initial period consumption will default to 0 if there is no previous period consumption available in the base meter. If this fallback logic does not align with your requirements, you may need to implement the BADI `/US4G/IF_BADI_DIR_MAIN->SET_AUTO_ON_INSTALL` to customize the behavior accordingly.
