Symptom

Report /CEECV/ROFI_BP that displays content /CEECV/ROFI_BPTY table issues DB749 error message "Invalid date; enter the date in the format DD.MM.YYYY" when a date is entered in field "Record last changed on" on its selection screen. Even though the format is correct.

Other Terms

Romania, Customer, Vendor, Master, /CEECV/ROFI_BP, DB749, Invalid Date, Date Format

Reason and Prerequisites

Program error

Solution

Apply appropriate support package or implement attached correction instruction by means of SNOTE transaction.
