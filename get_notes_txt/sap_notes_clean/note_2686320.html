Symptom

If you want to use various tank IDs in various items in a sales order, the following problem occurs when you change the tank ID in one of the OGSD views "OGSD TeleSales" or "OGSD: Item Detail":

  • The newly set tank ID is not only transferred to the currently changed item, but transferred to all items.
  • In addition, warnings involving the new tank ID in the actually changed item are issued twice.

As a result,

  • the tank information is updated incorrectly in the table OIK37
  • the forecast CPR is run with an incorrect data basis

Other Terms

ROIIO0100-TANK_ID, screens SAPMV45A 8940 and /ICO/SAPLMO_TV_OIIO 0100, /ICO/MO_R9 data collation update step AV01, AV02

Solution

Implement the attached correction instructions or import the specified support package.
