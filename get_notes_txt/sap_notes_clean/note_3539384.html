Symptom

Transaction GLPLUP does not display the file description which is maintained in planning profile

Environment

  • Financial Accounting (FI)
  • General Ledger
  • SAP ERP
  • SAP ERP Central Component
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA

Reproducing the Issue

  1. Execute transaction GLPLUP  
  2. Select the search help for 'File Descriptions'
  3. The file descriptions maintained in GLPLADM cannot be seen.

Cause

File description has not been generated correctly.

Resolution

  1. The file descriptions in FI planning can be created/or regenerated in transaction GLPLADM.
  2. Run transaction GLPLADM
  3. Select corresponding profile and double click on "Tables which can be planned"
  4. Select total table and double click on "Planning tasks for the table"
  5.  Flag checkbox "Int. Excel" for desired profile, select the desired profile and double click on "Default parameters"
  6. Fill the selection screen and press "Overview screen (F5)"
  7. The file description will be generated
  8. Press "Save file description"

Keywords

GLPLUP, File Description, GLPLADM, planning profile
