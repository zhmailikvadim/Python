Symptom

  • Accounting document line timestamp does not appear in document flow.
  • Accounting document line timestamp does not correctly display in document flow.

Environment

  • Sales And Distribution (SD)
  • SAP ERP Central Component
  • SAP ERP
  • SAP Enhancement package for SAP ERP
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Navigate to transaction VF03.
  2. Attempt to view the accounting document line timestamp in the document flow.
  3. Observe that the timestamp does not appear or incorrect.

Resolution

Confirm whether the following corrections notes have already been implemented.

If the required correction code is missing in system, implement it and retest the behavior to verify the resolution. 

a. [3393334 - SD/FICA: Time of FI-CA accounting document is displayed incorrectly in document flow display]

b. [3495756 - Date and Time of FI-CA accounting document is displayed incorrectly in document flow display](https://i7p.bss.net.sap/sap/support/notes/3495756)
