Symptom

When Approve Timesheets (S/4HANA/Version 4) application is integrated with My Inbox and the current user is delegating another user in the My Inbox settings, the selected subsitute user does not see the relevant records for approval.

Other Terms

F2585A, Approve Timesheet V4, Substitution, Workitem, Delegation, My Inbox.

Reason and Prerequisites

Work items were not retrieved for the substitute user.

Solution

Please implement the automatic correction instruction attached to this SAP Note or alternatively you can implement the corresponding support pack.
