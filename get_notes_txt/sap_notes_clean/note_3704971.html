Symptom

  • Tables are not visible in the optimized story experience.
  • Error message displayed: "We've encountered an unexpected issue when loading this object. Contact technical support."
  • Unable to access the builder when clicking on a table.
  • Unable to create a table; no action occurs when selecting the create table option or dragging and dropping it from the left panel.

Environment

SAP Analytics Cloud version 2026.1

Reproducing the Issue

  1. Update to version 2026.1.
  2. Attempt to view tables in the optimized story experience; tables are not visible.
  3. Click on a table to access the builder; no access is granted.
  4. Attempt to create a table by selecting the create table option or dragging and dropping it from the left panel; no action occurs.

Cause

This issue can occur due to an issue with page popup widgets.

Resolution

The development team is currently working on a fix for this issue.

As a workaround for urgent issues you can convert the page popup to a Story Popup
