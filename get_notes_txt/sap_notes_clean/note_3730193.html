Symptom

This KBA explains the configurations required for auto-filling IT0181 additional fund type when hiring a Singapore employee.

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

 

Environment

  • SAP ERP
  • SAP S/4HANA
  • Human Capital Management (HCM)
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP SuccessFactors Employee Central Payroll

Resolution

  1. Log on to the SAP system with the user who will perform the hire action for Singapore employees.
  2. Maintain the parameter 'UGR' value as 25 under the user profile by following the path: System -> User Profile -> Own Data.

  3. Define infogroups by running transaction code SPRO, then follow the path: Personnel Management -> Personnel Administration -> Customizing Procedures -> Actions -> Define infogroups.
    1. Choose 'User Group Dependency on Menus and Info Groups'.

      Tick 'User group-dependent' checkbox. For example, the infogroup 10 Hiring is configured as below:.
    2. Choose 'Infogroup modifier' and set the proper return value for feature IGMOD if a specific infogroup modifier is needed, or leave it blank.
    3. Choose 'Info Group' and fill in the infogroup as 10, which is the example shown in step 3.a.
      • Ensure entries are maintained for user group 25, which is aligned with the UGR value configured in step 2.
      • The infogroup modifier should be configured in alignment with step 3.b.
      • Operations for inserting IT0002, IT0077, and IT0186 should be in an earlier sequence before inserting IT0181.
  4. Configure the infogroup as 10 for Hiring action 01 in table T529A.
  5. Check the configuration of features '25FUN', '25ID1', and '25ID2' based on IT0002 Religions, IT0077 Ethnic origin and IT0186 CPF Fund type.
    The return value is the additional fund type that is auto-filled into IT0181.
    For example, the additional fund type 01 'CDAC' should be auto-filled for employees whose ethnic origin is Chinese and CPFID is Singapore citizens.



    Choose 20 Chinese for the Ethnic origin.

    Choose 01 for the CPF Fund type.



Note: Different additional fund types have varying auto-fill conditions. For further clarification, kindly contact the relevant authorities:
CDAC: https://www.cdac.org.sg/
SINDA: https://www.sinda.org.sg/
MBMF: https://www.muis.gov.sg/
