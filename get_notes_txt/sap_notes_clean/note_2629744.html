Symptom

This SAP Note delivers the following corrections and enhancements for the German Company Pension Scheme (CPS):

  1. Report RPCWPCD0 (Pension Calculation) and report RPCWPZD0 (Actuarial Interface)
    1. Termination with error "Start or end date initial"
      The program terminates and the system issues error message 5D 311 ("Start or end date initial"). There is no reference to the affected personnel number.
      To aid with troubleshooting, the personnel-number-related message 5D 334 ("Return code xx in RP_01C_TIME_COMPUTE") is issued by the report RPCWPCD0 or RPCWPZD0 when this error occurs.
       
    2. Function module RP_01C_BAV_ERROR_ROUTINE
      In customer enhancements, you can use the function module RP_01C_BAV_ERROR_ROUTINE to return the master program (parameter E54_EPROG) and the error routine (parameter E54_EFORM). Due to a program error, however, the system returns the error routine in E54_EPROG instead of the master program.
       
  2. Pension equalization payment in the company pension scheme (CPS): Report RPCAVVD0 ("Determination Distribution Proposal")
    1. When the system determines the distribution proposal for a current pension (pension payment), it does not fill the internal table LT_BVAZR with logging data due to a program error.

Other Terms

Report RPCWPCD0
Report RPCWPZD0

Reason and Prerequisites

Program error

Solution

This is delivered in an HR Support Package. As an advance correction, you can implement the attached correction instructions.
