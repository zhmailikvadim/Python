Symptom

In My Timesheet V3 application, when a recorded entry is changed, the unit is not reassessed accordingly.

Other Terms

My Timesheet V3, Assignment, F3074, Unit, Wagetype, Hours, Currency

Reason and Prerequisites

The stored unit value in the existing record prevents the system from determining the unit again based on the new set of values.

Solution

 Apply the corresponding support pack. Alternatively, implement the correction instructions attached to the SAP Note.
