Environment

  • ABAP Platform
  • SAP S/4HANA
  • ITSmobile

Reproducing the Issue

  1. Search for itsmobile03 service in t-code SICF (this is a test itsmobile service available in most systems).
  2. Right-click in itsmobile03, press "Test Service" and login.
  3. Click on + button to navigate a next page.
  4. Select a radio button “509 F4 Help” and then press button STRT. 
  5. Press F4 help in the first input field -> dump occurs.

Cause

  • Programming error in Kernel code.

Resolution

Update the Kernel in the affected system as outlined in the following SAP Note:

3726373 - Searchhelp in itsmobile service

See Also

  • KBA: 2221538 - ITS_UNEXPECTED_TOKEN or ITS_UNDEFINED_NAT_FUNCTION dumps in ITSmobile application
