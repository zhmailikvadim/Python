Symptom

  • The Statement of Work Status is 'Pending Buyer Review' 
  • The buyer user updating the SOW is different than the current Owner, and they did not change the owner field
  • Why is there a Warning: The new owner will replace the previous owner as the collaborator on the Clauses?

Cause

This is expected behavior when a SOW is in Pending Buyer status and a user clicks Edit

Resolution

  • No changes are made unless the Owner is actually updated
  • This is an informational message. It indicates that if the Owner is changed during the editing process, the new Owner will automatically replace the previous one as the Clause collaborator
  • This ensures the correct user has access to manage Clauses moving forward
