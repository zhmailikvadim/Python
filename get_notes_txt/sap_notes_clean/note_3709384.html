Symptom

Finished Product on PMR header is missing relevant leading zeros (e.g., "0100004415" is reported as "100004415").

Environment

  • SAP Extended Warehouse Management
  • SAP S/4HANA

Reproducing the Issue

  1. Execute transaction code /SCWM/PMR using document number(as PMR).
  2. Check the field "/SCWM/PRODUCTION_INFO - Finished Product or Order Text"; it shows the value without the leading zeros at the beginning, but the finished product should contain the leading zeros.

Cause

The field "/SCWM/PRODUCTION_INFO - Finished Product or Order Text" is a text field, and leading zeros are removed based on a past requirement to exclude them when the material name contains only numbers.

Resolution

  1. If leading zeros are required, modify the following code part: ‘/SCWM/CL_SP_PWR->COMPOSITION_HEAD()’.
  2. Refer to correction note 2473527, which introduced this behavior.

See Also

SAP Note 2473527 - /SCWM/PMR: Field /SCWM/PRODUCTION_INFO contains leading zeroes
