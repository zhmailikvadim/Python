Symptom

This SAP Note provides two minor technical improvements for translations of text elements of objects that were delivered in SAP Note 3692222.

Other Terms

Chernobyl, 3452, IT3452, P3452, HCMT_BSP_PA_UA_R3452, RP_3452, HRCUAI3452, CHEC, V_T591A

Reason and Prerequisites

You have implemented SAP Note 3692222 with the initial delivery of certificates of individuals affected by the Chornobyl disaster.

Solution

As a general rule, SAP recommends that you implement the Support Package specified for your release.

However, if you need to install the solution earlier, use the SAP Note Assistant to carry out automatic and manual correction instructions. For more information, see SAP Note Assistant.

This SAP Note installs the report NOTE_3731216 in your system.

Note that translations are only delivered via Support Package.

Key aspects of changes

  • The name of subtype CHEC for infotype Personal Records and Credentials (3452) has been changed to 'Chornobyl Certificate'.
  • Field label lengths of data element HRPADUA_EDOC_EXPDT have been extended to maximum possible values for each type of label (short, medium, long, heading).
