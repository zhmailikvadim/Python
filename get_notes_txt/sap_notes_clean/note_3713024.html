Symptom

  • The system does not provide an option to select prospect as an option for the individual customer during lead creation.
  • However, this is available for accounts (Existing/Prospect)

Environment

  • SAP Sales Cloud Version 2

Reproducing the Issue

  1. Go to Navigation Menu
  2. Go to Leads
  3. Click + icon
  4. Observe for Accounts, we have Existing or Prospect
  5. Click on Individual Customer
  6. Observe that Prospect is not available

Cause

This is the current system behaviour

Resolution

  1. Go to Navigation Menu
  2. Go to Individual Customers
  3. Select Role as Prospect
  4. Enter other details and save
  5. Go to Leads
  6. Click + icon
  7. Go to Individual Customer
  8. Add the created Individual Customer under Individual Customer Name field
