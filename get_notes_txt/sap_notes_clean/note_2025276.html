Symptom

  • Following error appears while creating a WebI report based on Excel file (.xls or .xlsx) with Range Name selection:
    "An error has occurred from the server. Unable to create or update the Excel personal data provider: an error occured while retrieving the data sampling. (IES 10881)"
  • It works fine if range name is not selected.

Environment

  • SAP BusinessObjects Business Intelligence BI 4.0
  • SAP BusinessObjects Business Intelligence BI 4.1
  • Web Intelligence Rich Client

Reproducing the Issue

  1. Open WebIntelligence Rich Client.
  2. Create new Web Intelligence document selecting Excel as data source.
  3. Select the .xls or .xlsx file you wish to use for the data source.
  4. On field Selection, select Range Name.
  5. Select a range and click ok. The following error appears- "An error has occurred from the server. Unable to create or update the Excel personal data provider: an error occured while retrieving the data sampling. (IES 10881)"

Cause

The Excel sheet name contains a space. Example: "My Sheet".

Resolution

  • Rename the excel sheet and remove the space or replace it with some other character like underscore.
  • Example: Rename "My Sheet" to "My_Sheet" or "MySheet".
