Symptom

You are executing distribution balance check report and see that the negative balance on default exception WBS element is shown as positive in the output. This issue inflates the balance on exception WBS element.

This situation can happen when a manual posting is done that moves the cost out of production order (credit entry) or a purchase order/Group WBS and distribution is run after the posting.

Other Terms

PMMO, Distribution Balance check, PMMO_DISTR_BALCHK, PMMO_PEG_EXC, Exception WBS Element, Default WBS element, KB11N, FB50

Reason and Prerequisites

This is due to a program error.

Solution

Implement the SAP Note or the service pack that contains it.
