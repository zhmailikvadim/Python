Symptom

Archiving session for archiving object workitem is incomplete, message "No authorization for operation 'Delete' of group 'DEVELOP'" (SWF_UTL001) found in corresponded archiving deletion job log.

Environment

  • SAP Business Workflow
  • SAP NetWeaver
  • SAP Web Application Server for SAP S/4 HANA
  • ABAP PLATFORM - Application Server ABAP

Cause

The execution user does not have the required authorization to perform the deletion operation for the archiving object WORKITEM.

Resolution

Assign the required authorization to the execution user then redo workitem archiving with T-code SARA:

  • Archiving object: S_WF_ADM
  • Field 'WF_GROUP' = DEVELOP
  • Field 'ACTVT' = 06

    Keywords

    WF_GDPR, SWF_WORKFLOW, WFMSTD, SARA, WORKITEM, SWF_UTL001, ARV_WORKITEM_DEL
