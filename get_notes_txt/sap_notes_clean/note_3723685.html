Symptom

It is mandatory for process template IDs to end with an underscore character _. However, when copying a template, no check is performed for the new copied template. This leads to issues during function execution.

Other Terms

LOD-PER-OP SP 27

Reason and Prerequisites

Program error

Solution

Please install the automatic correction request using SNOTE.

Already existing process templates that do not end in an underscore need to be deleted and recreated following the naming convention.
