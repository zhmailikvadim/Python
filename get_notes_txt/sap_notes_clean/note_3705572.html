Symptom

The SAP FC view only displays the last modified date for each user account. It does not provide a complete modification performed on user accounts.

Environment

SAP Financial Consolidation 10.1

Cause

This behavior is due to a system design limitation in SAP Financial Consolidation. SAP FC does not store or expose a full historical audit trail of user account changes.

Resolution

This limitation is by design and cannot be changed or enhanced within SAP FC. There is no technical workaround or configuration available to generate a complete historical change log for user accounts.
