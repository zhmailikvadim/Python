Symptom

In the French DSN (Déclaration Sociale Nominative) declaration, contribution code 076 in block 81 is allowed below contribution base code 22 in block 78 for MSA (Mutualité Sociale Agricole). For more information about authorized correspondences between gross and net amounts, see DSN Info 2498: "Correspondances autorisées entre les montants bruts et nets".

SAP Note Version Reason of the New Version
2 Typo corrected in the first step of the manual correction instructions

Other Terms

RPLDSNF0, DSN

Solution

After implementing this SAP Note, the system allows you to declare a block 81 with contribution code 076 below a block 78 with base code 22 for MSA.

These corrections are available with the HR Support Packages referenced below.
If you have an urgent need of these corrections, follow the correction instructions provided.
