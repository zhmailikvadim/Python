Symptom

To maintain the header information for Z4 report (App: Run Foreign Trade Regulations Z4(ID: S_ALR_87012405) and App: Run Foreign Trade Regulation Z4 – Basis Receivables/Paybles(ID: S_EB5_05000341)).

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Environment

  • SAP S/4HANA Cloud
  • SAP Central Business Configuration (CBC)

Resolution

The header information for Z4 report (App: Run Foreign Trade Regulations Z4(ID: S_ALR_87012405) and App: Run Foreign Trade Regulation Z4 – Basis Receivables/Paybles(ID: S_EB5_05000341)) can be maintained using the configuration activity 'Maintain Additional Parameters' (ID 102739). The header information for the Z4 report is filled from additional parameters SAPZ01 to SAPZ09 for the company code. The values of these parameters can be maintained in the configuration activity 'Maintain Additional Parameters' (ID 102739).

The Z4 report is specific to country/region Germany. Therefore the configuration must be performed for local version Germany (DE).
