Symptom

  • While attempting to change the payment method for supplier line items in bulk using the Manage Supplier Line Items app, the system restricts the selection to a maximum of 50 line items at a time.
  • This warning message is displayed - "Selection Limit : In this table you cannot add more than 50 items to your selection at a time"

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Open the Manage Supplier Line Items app.

  2. Apply the required filters to display a large number of supplier line items.

  3. Select multiple line items using the checkbox selection option.

  4. Attempt to select more than 50 line items at once.

  5. Observe that the system prevents selecting more than 50 items and displays the warning message regarding the selection limit.

Cause

  • This restriction is part of the standard system design in SAP S/4HANA Cloud and is implemented for performance and stability reasons.
  • The app is designed to handle smaller-scale updates, and mass changes involving thousands of line items are not supported within a single selection to prevent system timeouts and performance degradation.

Resolution

  • To update payment methods for a large number of supplier line items, the changes must be performed in multiple batches, selecting up to 50 items at a time.
  • Requests for additional functionality can be submitted through the SAP Customer Influence portal, as described in KBA 0002963059.
