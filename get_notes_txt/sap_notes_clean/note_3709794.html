Symptom

When you use the “Last Data Record” icon in the menu bar, the system displays the document with the highest document number, which may not be the most recently created document.

Environment

SAP Business One

Reproducing the Issue

  1. Navigate to the document you want to review (e.g., Sales Order, Purchase Order).

  2. Ensure the document form is in find mode so you can navigate records.  

  3. Click the “Last Data Record” icon in the menu bar (top of the application window).

  4. Observe that the system shows the record with the highest document number.

Resolution

This is the expected system behavior when using the “Last Data Record” icon in the menu bar.

Workarounds:

  • To view the latest document in a specific series, open the document, select the desired series, and press [Enter]. The system will display the last document for that series.
  • Alternatively, enter an asterisk [*] in the document number field - next to the series - to open the document list. From there, you can sort the list in ascending or descending order by any available column in the Form Settings.
