Symptom

A warning is shown in the transaction SU22 (Edit Authorization Default Values) for the Web Dynpro Application APPL_SOAP_MANAGEMENT regarding missing Authorization Default Values for S_SRT_SR_P (Services-Registry-Provider).

Other Terms

SU22, SOAMANAGER, S_SRT_SR_P

Reason and Prerequisites

The warning has been detected in the correction system for SAP_BASIS 7.50.
For releases higher than SAP_BASIS 7.50 the Authorization Default Values have been already maintained.

Solution

Implement the Support Package level referred in this SAP Note.
