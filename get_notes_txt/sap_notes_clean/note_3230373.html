Symptom

The deletion report GM_RECONSTRUCTION_FI_DELETE allows for deleting GMIA table entries for FI documents even if ILM is active.

Other Terms

GM_RECON_FI_REP

Reason and Prerequisites

You have assigned the ILM objects GM_DOC_AC or GM_DOC_TT to an audit area and have determined a retention time for GMIA table entries.

Solution

Implement the assigned correction instructions.

After implementation of this SAP Note, the deletion report GM_RECONSTRUCTION_FI_DELETE eventually cannot be used any more to delete GMIA table entries.

If you need to delete GMIA table entries before reposting FI documents with transaction GM_RECON_FI_REP, you can now mark the checkbox 'First Delete Ledger 90 Entries' in transaction GM_RECON_FI_REP.
