Symptom

  • System allows adding two distribution lines with the same attributes either both at item level, or separately on header and item level.
  • The expected error Cannot create duplicate distribution lines for central contract item is not raised.

Environment

  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition
  • SAP Ariba Central Procurement

Cause

System checks the duplication based on specific fields of distribution lines. If any of those are different, system considers the distribution lines different and it doesn't raise any error.

Resolution

To trigger the error all fields below need to match in the distribution lines:

  • Company Code (and Company Code Group in case of Hierarchy Contract)
  • Purchasing Organization
  • Purchasing Group
  • Currency
  • Payment terms
  • Outline Agreement Category
  • Outline Agreement Type
  • Plant

Keywords

F3144, duplicity check, duplication
