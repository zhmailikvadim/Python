Symptom

You want to implement the corrections described in SAP Note 3727607. These corrections require additional changes that are usually handled as manual tasks. This note is used to cover part of these changes automatically. Please, be aware about manual steps described in this SAP Note (Maintenance View extension and regeneration).

Other Terms

FSM, E4C, Connector, UDO, JOB

Reason and Prerequisites

This note implements an ABAP report that performs the required changes.

Solution

1. Implement the attached correction instructions.

2. Run report /PACG/ECM_PACG_NOTE_3727607 (ECC) or /PACG/ECM_S4PACG_NOTE_3727607 (S/4).

3. Extend and regenerated the '/PACG/ECM_VJOB' maintenance view, follow steps described in the manual post-installation steps.

3. Implement the note 3727607.
