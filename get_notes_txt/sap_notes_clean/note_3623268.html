Symptom

While maintaining tax classification in transaction BP the error Entry ZMWZ 7 does not exist in TSKD (check entry) Message No. 00058 appears

Environment

  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Transaction BP
  2. Role FLCU01
  3. Sales and Distribution tab
  4. Billing tab
  5. Fill in MWST in output tax
  6. Error appears

Cause

  • Tax category ZMWZ and Tax classification 7 is not maintained in table TSKD but this tax category exists in table KNVI 
  • This is an inconsistency

Resolution

Create a custom report to remove the entry from KNVI table

Keywords

Tax classification, FLCU01, KNVI, MWST, tax category, Entry ZMWZ 7  does not exist in TSKD (check entry), 00058, TSKD
