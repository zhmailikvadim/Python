Symptom

 Access restriction of site in stock overview is not working.

Environment

SAP Business ByDesign

Reproducing the Issue

  1. User has been assigned work center view: SCM_CUSTOMERDEMAND_STOCKOVERVI
  2. The read and write access is restricted to site ABC (ABC represents for the site ID).
  3. However if user go to Outbound Logistic Control Work Center and go to Stock Overview View.
  4. Filter by site, you find the restriction is not working, which means user still has access to other sites besides ABC.

Cause

There are other work centers as well where the user is having one or more unrestricted access.

Therefore, the user is able to see other sites as well in the stock overview.

Resolution

Please check the attached file for the work center view that require restrictions.

If you maintain restrictions for other work centers as well, this will help you allow the user to see stock from specific sites only.
