Symptom

  • Off-BizX User Preview shows that all users will be deactivated and 0 users will be migrated.
  • Conversion job fails with: "Validation failed: Following your updates of the rule condition, this user list will be identical to existing user lists - All users. Please modify the condition or use the existing user list instead."
  • After conversion, some users cannot see the home/toolbar navigation bar on the work zone landing page.

Environment

  • SAP SuccessFactors Work Zone
  • SAP Jam Collaboration

Reproducing the Issue

  1. Open the Off-BizX wizard and run User Preview: preview indicates 0 users to be migrated.
  2. Start the Off-BizX conversion: job fails with  "Validation failed: Following your updates of the rule condition, this user list will be identical to existing user lists - All users. Please modify the condition or use the existing user list instead." error

Cause

  • Dynamic User List condition duplicates an existing list (for example, “All users”), causing the conversion validation to fail.
  • Admin area (for example, “HR”) is linked to a Dynamic User List whose filter excludes many users, so those users do not get the homepage navigation tabs.

Resolution

In Off-BizX wizard, correct the Dynamic User List:

  1. Ensure the condition does not duplicate an existing list, or select the existing list instead.
  2. Refresh the User Preview to recalculate, then run the conversion again.

Fix admin area membership for the homepage navigation:

  1. In Admin Console, open the admin area used by the homepage tabs.
  2. Link it to a Dynamic User List that includes all intended users.
  3. Add that list to the area membership and save.
