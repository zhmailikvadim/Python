Symptom

According to installation guide 

External Components Prerequisites | SAP Help Portal

For SAP Financial Consolidation 10.1 only 

  • Microsoft Visual Studio C++ 2015-2022 Redistributable (x86) (or higher)

is required, but in fact, if component

  • Microsoft Visual Studio C++ 2015-2022 Redistributable (x64) (or higher)

is not installed, FC web applications such as WebAdmin displays an error:

"Could not load file or assembly 'Apcom.Framework.Connection.DLL' or one of its dependencies. The specified module could not be found."

Environment

SAP Financial Consolidation 10.1

Cause

Incorrect documentation.

FC web applications are 64 bit so the component

  • Microsoft Visual Studio C++ 2015-2022 Redistributable (x64) (or higher)

is actually required for SAP Financial Consolidation.

Resolution

Incorrect document will be fixed.

Keywords

EPM, BOFC, BFC, FC, x86, x64, 32 bit, 64 bit, assembly
