Symptom

When you run the Reporte Unidad de Gestión Personal y Parafiscales UGPP (RPC_PAYCO_UGPP) for an employee with cross-period absences, the following symptom may occur:

  • The begin and end dates of the absence are duplicated in the final report.

Other Terms

&UGPP (further information see SAP Note 2189265)

PC00_M38_UGPP;

Reason and Prerequisites

The following Support Package level is the minimum prerequisite:

Release Support Package
600 HR-CEECCO SP J0
604 HR-CEECCO SP F6
608 SAP_HRCCO SP 75

(For more information, click "HR Support Package Schedule" in support.sap.com/maintenance-schedule.)

Solution

Version Date Comments
2 December 10, 2021 Released for Customer.

From now on the system will no longer consider the duplicated dates in the report.

As general rule, SAP recommends that you install a solution by applying a Support Package. However, if you need to install the solution earlier, proceed as follows:

    1. Use Note Assistant (SNOTE) to implement the Correction Instructions.
