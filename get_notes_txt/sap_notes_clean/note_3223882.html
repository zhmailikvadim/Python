Symptom

You are setting up SAP BPC for S/4 HANA Finance and Embedded BW Reporting as per note 1972819.

You would like to know in which cases is the BPC license mandatory.

Environment

  • SAP BPC for S/4 HANA Finance and Embedded BW Reporting

Reproducing the Issue

  1. Follow note 1972819.

Resolution

If the parameters BPC_ACT: BPC-PAK integration active (or "BPC Embedded Model Active") in step 6.2 d) are active, the BPC license is required.

Otherwise, the BPC license is not required.

See Also

  • KBA 2432482 - BPC-NW - how to tell if BW-PAK or BPC embedded is activated
