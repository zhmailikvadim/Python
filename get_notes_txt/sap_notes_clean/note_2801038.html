Symptom

When buyers created a scheduling agreement release with attachments or comments and sent it to the suppliers, the ARBCIG_SARDETAIL segment was missing from the IDOC.

This missing segment is needed for the cXML that gets generated in SAP Ariba Cloud Integration Gateway.

Other Terms

ARBCI1, ARBCIG_SARELEASE, ARBCIG_DELINS, Scheduling Agreement Release, SAR.

Reason and Prerequisites

Each segment in the IDOC has an index. When attachments and comments were added to scheduling agreement release, the index number was changed for the ARBCIG_SARDETAIL segment. The segment was getting dropped because the IDOC was not identifying the segments with changed indices.

Solution

The issue has been fixed in the code to ensure that the existing indices don’t change.

Action Required

  • Apply this SAP Note.
