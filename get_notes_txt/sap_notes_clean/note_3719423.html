Symptom

After creation of manufacturing(planning) snapshot it is found that sometimes it is not added back to change notice due to workflow user lock if there is a workflow triggered in change notice.

Other Terms

Change Notice Planning Snapshot, Manufacturing(Planning) Snapshot not added Change Notice, Planning Snapshot

Solution

Implement the attached correction instruction.

By implementing this Note, issue related to scenarios wherein the manufacturing(planning) snapshot not being added back to change notice due to workflow user lock from change notice would be resolved. Now,  the change notice workflow is started only after the updates on change notice are completed to avoid conflicting locks.
