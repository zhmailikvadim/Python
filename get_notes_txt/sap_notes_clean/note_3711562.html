Symptom

I received error message "Cannot associate or remove Site from Linked Custom Field." why trying to associate or remove sites using the Custom Field Site Association Upload connector.

Cause

The Custom Field Site Association Upload connector does not support adding or removing sites for a linked custom field. Linked custom fields can only be updated from the main module.

Resolution

The Custom Field Site Association Upload connector allows you to add new sites to custom fields from the main module. Once a site is successfully associated with the main module, it will automatically be linked to all associated modules.
