Symptom

While creating Handling Unit via HU API, for for closed packaging material, the HU creation is failing with an error - "Handling unit exceeds maximum allowed volume"

Other Terms

S/4HANA, HU API, Handling Unit API, Total Volume, Closed, Open, Packaging material, HU_RAP/014

Reason and Prerequisites

Program Error

Solution

As a general rule, SAP recommends that you install a solution by applying a Support Package. However, if you need to install a solution earlier, use the Note Assistant to implement the correction instructions. You can find more information about the Note Assistant in service.sap.com/note-assistant
