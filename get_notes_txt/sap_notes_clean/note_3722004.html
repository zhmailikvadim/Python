Symptom

Swedish report to of SCB - Salary structure statistik (SLP) is reporting code 004 only for Employees with employment percent (IT0007-EMPCT) less than 100%. Authority requested to report code 004 always, including the full 100% of Employment.

Other Terms

RPSSBMS0, IT0007-EMPCT, Employment percentage

Solution

Requested change was implemented. Code 004 is now reported for all Employees, including cases of 100% percentage of Employment recorded in IT0007 (EMPCT).
