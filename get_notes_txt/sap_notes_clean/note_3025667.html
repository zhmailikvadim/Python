Symptom

Sometimes, Deliveries are not written in Fresh Goods Receipt File (ZFFI_PW_GOODS_RECEIPT.saf) which impacts Projected waste calculation.

Other Terms

 FFISCM, FRP Run, Projected Waste Calculation, ZFFI_PW_GOODS_RECEIPT.saf

Reason and Prerequisites

 Program Error

Solution

Please implement manual correction instructions before implementing the automatic one. You can also implement support package SAPK-70205INFFISCM, if available.
