Symptom

The texts in manual billing documents are printed incorrectly.

Other Terms

EA60, ERCHT, print form, ISU_DB_BILL_SELECT_MASS

Reason and Prerequisites

Program error

The print incorrectly implements database sorting instead of obtaining formatted, line item-based sorting from the billing document selection.

Solution

Implement the correction instructions attached to this SAP Note or import the relevant Support Package.

Other Terms

CL_RSR_OLAP_VAR

Reason and Prerequisites

This issue is caused by a program error.

Solution

Import the relevant Support Package or implement the attached corrections.
