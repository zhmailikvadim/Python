Symptom

This SAP Note provides you with updates for the Auxiliary Functions module to support alphanumeric CNPJ.

Other Terms

Brazil, SPED, S/4HANA, CTR, PRV_CARGA_INICIAL, /TMF/LD_SIGNATARF01

Reason and Prerequisites

This SAP Note is relevant only for Brazil.

You have implemented SAP Note 3720349 or SAP Note 3713732.

Solution

After implementing this SAP Note, the PRV_CARGA_INICIAL Hana view and the /TMF/LD_SIGNATARF01 ABAP form are updated. As a result, they both support alphanumeric CNPJ.

As a general rule, SAP recommends that you implement the Support Package specified for your release. However, if you need to install a solution earlier, use the SAP Note Assistant to implement the correction instruction. You can find more information about the SAP Note Assistant in the SAP Support Portal, under SAP Note Assistant. Note that the documentation of certain objects is delivered with the corresponding Support Package only.
