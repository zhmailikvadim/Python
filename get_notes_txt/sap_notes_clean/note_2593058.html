Symptom

In the Diagnostics Agent logs, the following Warning is present:

" Jan 23, 2018 8:05:06 AM [Thread[Reconnect,5,WILYHOST_SCHEDULER:Exec...] Warning com.sap.smd.wily.hostagent.sapcontrol.SapControlDestination - open(): Property "s
id" is not specified in SapAgentConfig.xml. Re-run managed system setup to update the latest destination configuration supporting authentication against SapStartSrv"

Environment

  • SAP Solution Manager 7.2

Reproducing the Issue

Check the Diagnostics Agent logs

Cause

Incorrect configuration of the Managed System.

Resolution

This error is most likely coming from an Agent on-the-fly and the existing configuration corresponds to an old instance maintained for the managed System.

To resolve this and make sure the configuration is correct, proceed as follows:

  1. Re-supply the data from the Managed System in the SLD as described here: 1884816 - Product information for technical system is missing or incorrect in the SLD
  2. Re-sync the SLD and LMDB
  3. Re-run the Managed System Configuration activities from the beginning, including the Introscope Host Adapter activity.

See Also
