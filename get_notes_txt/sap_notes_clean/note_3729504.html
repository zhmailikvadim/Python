Symptom

During the installation of SAP S/4HANA 2025 FPS01, activation errors occur because the CDS views /PLGA/C_TSO_EQUIPMENT and /PLGA/C_TSO_FUNCTIONALLOCATION now contain syntax errors.

Other Terms

Meter-to-Cash 2025 FPS01 Activation /PLGA/C_TSO_EQUIPMENT /PLGA/C_TSO_FUNCTIONALLOCATION

Reason and Prerequisites

In FPS01, in the CDS view I_LinearAssetManagementData, the association _LinearDataObjectType has been renamed to _ObjectType. The CDS views mentioned above select fields using the association path with the previous name. This results in the syntax errors.

Solution

Implement this note. The fields previously reached using this association path are then filled with constant values and the CDS views are syntax-error-free and can be activated. This adjustment is only temporary. After the installation of SAP S/4HANA 2025 FPS01, implement SAP Note 3733405. This note then adjusts the CDS views to use the new association paths.
