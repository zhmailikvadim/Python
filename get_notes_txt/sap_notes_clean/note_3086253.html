Symptom

  • Vertical axis texts in a web intelligence report are getting cut off while viewing the report in Ipad.
  • This is specifically seen with lower case letters.
  • However, same is working fine in BI launchpad.

Other Terms

SAP BI Mobile App, IOS, BI launchpad, Vertical axis texts, Ipad, Web Intelligence

Reason and Prerequisites

Environment

  • SAP BusinessObjects Business Intelligence Platform 4.x
  • SAP BI Mobile App 6.8.18 (iOS)
  • Web Intelligence
  • Bar Chart

Steps to Reproduce

  1. Create a Web Intelligence report with Bar Chart component having Vertical axis texts in lower case.
  2. Open the same report in SAP BI Mobile App.
  3. See the vertical axis texts in the Chart with lower case letters are getting cut off.

Solution

This issue has been fixed in SAP Business Objects IOS Mobile Application 6.8.21 version.
