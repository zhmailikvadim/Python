Symptom

The values for the transient fields of the accounts are missing in the exported Excel.

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

  • SAP Sales Cloud Version 2
  • SAP Service Cloud Version 2

Reproducing the Issue

  1. Go to Accounts List View 
  2. Use the advance filter/quick filter to filter out the target accounts 
  3. Click Export button in the list view
  4. Wait until it's completed 
  5. Download the file
  6. Open this file
  7. The fields "parentAccountId_transient", "parentAccountFormattedName_transient", "topAccountId_transient" and "topAccountFormattedName_transient" do not have any value although they have values in the list view

Resolution

This is standard system behavior. These four fields are not persisted in the account. Therefore they cannot be exported in the excel. For downloading Account Hierarchy related data, please go to Settings -> Import and Export Data -> Export and Select "Account Hierarchy" as the entity for the export.
