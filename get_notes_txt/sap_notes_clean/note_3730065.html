Symptom

  • DQM for SAP is not working.
  • Postal code format entered does not follow the proper format for the country.
  • When postal code format follows the correct format, DQM works as expected.

Environment

SAP Data Quality Management, version for SAP Solutions

Cause

The postal code format fails SAP's frontend check, so therefore the error is shown and DQM is not called.

Resolution

Enter the correct postal code format to get DQM for SAP to be called.
