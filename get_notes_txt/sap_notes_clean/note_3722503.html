Symptom

You have ST-PI 758 SP00 Installed.

Transaction code: /n/SDF/ALM_DIAGNOSTIC

or run the "check use case" for the CDM use case in /sdf/alm_setup,

You get the error message: The installed version of ST-PI is outdated: 0.

Other Terms

/SDF/CALM_CDM_CONN_DIAGNOSTIC, /SDF/ALM_SETUP, CDM, Cloud ALM.

Reason and Prerequisites

This is a program error.

Solution

Apply the attached correction instructions or install the corresponding support package.

Then the error in red will turn to warnings in yellow.

An older version of ST-PI is implemented: 0

Simply ignore this warning.  ST-PI 758 SP00 is the latest support package released to customer.

SP01 is on the way, not released yet.

This warning does not affect any functionality.
