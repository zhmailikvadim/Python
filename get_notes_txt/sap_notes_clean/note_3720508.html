Symptom

User would like to cancel a PO, Total Received quantity is already reversed, but user still could not submit cancel PR due to error Cannot cancel order because it is been invoiced. 

Environment

SAP Ariba Purchasing & Receiving

Cause

The PO is an ERS enabled, so there are 2 invoices auto-created and linked to this PO. User could not cancel a PO if there's any active invoice linked to it.

Resolution

Requester user will need to do the following:

  1. Make sure Total Received value for all Line items is 0
  2. Force Reject reconciled invoices in each IR header
  3. Access PR version 2 which is in composing status, make sure to delete all lines and click Submit 

PR status and PO status will change to Canceled after approvers approve this canceling PR.
