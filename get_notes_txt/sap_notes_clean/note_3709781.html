Environment

  • SAP ERP Logistics (PP)  
  • SAP S/4 HANA, on premise
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Run T-code: MD01N.
  2. Check the dump in T-CODE: ST22.

Cause

The field BSTMI is of type MENG13 which is length 13 with 3 decimals. This means maximum allowed value is: 9999999999.999.

Resolution

In order to avoid this dump you need to adjust UMREZ or UMREN value maintained for Info record.
