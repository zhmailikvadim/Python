Symptom

  • The classic Project Progress Report (transaction CNE5) available in SAP On-Premise / Private Cloud is not delivered in SAP S/4HANA Cloud Public Edition.
  • Traditional SAP GUI Project System reports (such as CNE5 or CJI3) are not part of the scope.
  • A standard S-curve chart is not delivered out of the box in SAP S/4HANA Cloud Public Edition.

Environment

SAP S/4HANA Cloud Public Edition

Resolution

  1. Use the following standard Fiori apps to monitor project progress:
  • Project Control – Professional Services Projects: This application analyzes project progress, including project status, milestones, planned vs. actual costs, estimates to complete, and billing relevance. Refer to SAP Help Portal - Project Control – Professional Services Projects.
  • Project Cost Report – Line Items: This application analyzes planned and actual project costs in detail and is recommended as an alternative to classic cost reports like CJI3 in Public Cloud. Refer to: SAP KBA 3524410 - Alternative Report for CJI3 in SAP S/4HANA Cloud Public Edition.
  • Review Customer Projects: This application supports project performance reviews, including percentage of completion and estimate-to-complete values. Refer to SAP Community - Project Tracking and Overview in SAP Public Cloud.

     2. Build S-curve visualizations using SAP Analytics Cloud (SAC) or external analytics tools based on project planned and actual data.

See Also
