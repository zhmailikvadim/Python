Symptom

  • Sort order in Web Intelligence document is not preserved when creating combined cube with flat hierarchical object

Other Terms

webi, webintelligence, WDM, data, mode, virtual, cubes, hierarchy

Reason and Prerequisites

Environment:

  • SAP BusinessObjects Business Intelligence platform 2025 Patch 7
  • Web Intelligence 

Steps to Reproduce:

  1. Create a Web Intelligence document containing a flat hierarchal object
  2. Add two queries based on this object 
  3. Run Queries
  4. Switch to Data Mode
  5. Insert a combined cube based on this object using left join operator for example
  6. Switch to Design mode
  7. Add a table based on the object from the newly created combined cube
    1. >> Notice that the sort order is different from that of parent objects

Cause:

  • This issue is caused by a defect in the product.

Solution

This issue is fixed in the patches listed in the "Support Packages & Patches" section below.
Due to different internal processes, the "Support Packages & Patches" section of the SAP Note will be populated as below:

  • for fixes delivered in "Support Packages": immediately (even if the "Support Package" is planned to be released several months later)
  • for fixes delivered in "Patches": ONLY once the "Patch" is released AND available on SMP (not before)
