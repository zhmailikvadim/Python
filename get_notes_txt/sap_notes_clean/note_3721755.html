Symptom

Unable to find a specific SAP user in the "Manage Substitute" list in my inbox

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Navigate to "Manage Substitute" within the My Inbox application.
  2. Attempt to add a substitute for workflow-related approvals.
  3. Search for the SAP user with an active SAP ID.
  4. Observe that the user is not displayed in the list despite having an active SAP ID.

Cause

The user ID is locked.

Resolution

Unlock the user and have a try again.

Keywords

manage substitute, my inbox, lock user
