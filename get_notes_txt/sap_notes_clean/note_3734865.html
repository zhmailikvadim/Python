Symptom

When executing report S_PL0_86000028 / S_PL0_86000029 - Fin. Statement: Plan/Act. Comparison, error message KY041 "No characteristic value XXX exists for characteristic Version due to compound" is raised and report  is not extracting any data after providing the correct input.

Environment

  • Financial Accounting
  • SAP ERP Central Component
  • SAP ERP
  • SAP S/4HANA Finance
  • SAP S/4HANA
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA

Reproducing the Issue

  1. Run S_PL0_86000029 - Fin. Statement: Plan/Act. Comparison.
  2. Provide the intended selection input (e.g Ledger, FIS Annual Rep.Struc, Plan Version, Fiscal Year, Period ...)
  3. Upon execution error message KY041 "No characteristic value XXX exists for characteristic Version due to compound" is triggered.

Cause

  • The entered Version value does not exist for the required compounded characteristic combination, resulting in KY041.
  • No plan totals exist for the selected combination in the General Ledger totals (e.g FAGLFLEXT) , so the report returns no records.

Resolution

  1. Use F4 value help in the selection to verify that the Version value exists for the required compounded characteristic combination (for example, Version and its superior characteristics).
  2. If the Version value does not exist, maintain it in master data using transaction GLPV for the relevant ledger.
  3. Verify that plan totals exist for the selected ledger/company/fiscal year/period; if they are missing, load or post the required plan data so the report has totals to read.
  4. Re-run S_PL0_86000029 with the corrected and valid selection values.
