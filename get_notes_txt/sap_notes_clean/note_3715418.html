Symptom

In JPK_KR_PD DRC solution CDS substring command is disallowing the use of functions in its parameters in system S4CORE 106 (ERN).

Other Terms

Poland, PL, Poland SAFT Report, PL SAFT Report, JPK_KR_PD, I_PL_SAFTJournalEntryCube, Decimals, Substring, DocumentItemText

Reason and Prerequisites

This note is relevant for Polish SAFT only.
You are using Polish SAF-T KR_PD (JPK_KR_PD) report.
You are getting syntax and activation errors related to missing field Decimals and/or substring parameters

Solution

The substring logic for field DocumentItemText has been adjusted for the cases when the field contains a star(*).
