Symptom

  • The implementation of this SAP Note has no impact on your existing functionality.
  • This SAP Note delivers changes that are usually treated as manual activities within SNOTE (for example, packages, ABAP dictionary objects, message classes).
  • Implement SAP Note 3721782 (DSN: Prerequisite Objects for SAP Note 3711767), before implementing primary SAP Note 3711767.

Other Terms

RPLDSNF0, block 53, working days, ACTI

Reason and Prerequisites

Delivery of prerequisite objects.
This SAP Note addresses a new requirement.

Solution

The correction enclosed within this SAP Note will be included in HR Support Packages referenced below. If you need the correction before the HRSP is available, you may apply the attached correction instructions.
It should be emphasized however, that correction instructions are only a short-term solution, not a substitute for the Support Package.

Use transaction SNOTE to implement the attached correction instructions before you implement the primary SAP Note 3711767. By doing so, report NOTE_3721782 will be installed in your system.
Implement the attached correction instructions or the corresponding HR-SP or CLC.
