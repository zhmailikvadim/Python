Symptom

This article describes how the net value is calculated in sales documents.

Environment

SAP S/4HANA Cloud Public Edition

Resolution

  • Net value in header level on the overview screen is a total value of net value at item level.
  • Net value at item level is a total value of the condition types that meet the conditions below:
    • The non-tax conditions.
    • The active and non-statistical conditions (KINAK not set to 'AKLMXZ' and not set to 'Y', as well as KSTAT not set to 'X').
    • The condition category is not 'g' for the discount condition type(Condition Class is A).
