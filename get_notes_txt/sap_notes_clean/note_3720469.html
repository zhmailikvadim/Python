Symptom

You want to run command "cf staging-environment-variable-group" or "cf ssevg"

It always fails with error "You are not authorized to perform the requested action" though you are already assigned with all possible Org and Space roles.

Environment

SAP Business Technology Platform, Cloud Foundry environment

Reproducing the Issue

Use the CF CLI command 'cf ssevg' to update environment variables, for example, "HTTPS_PROXY", etc

Cause

The CF CLI command 'cf ssevg' requires the admin role, which is restricted to platform operators.

Resolution

The CF CLI command 'cf ssevg' requires the admin role which is only available to platform operators. SAP cannot grant this role to customer accounts.
Also note that the staging environment variables are global parameters used by all users of the CF runtime. There is no concept in CF to set e.g. organization or space wide variable groups.
