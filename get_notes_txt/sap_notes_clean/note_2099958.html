Symptom

This Release and Information Note (RIN) contains information and reference to notes in the context of installing SAP Enterprise Project Connection 2.0 Support Pack 02 Patch 08 release. This note is subject to change. Please check on a regular basis

Other Terms

SAP EPC 2.0, SAP Enterprise Project Connection 2.0 SP02, EPC Patch 08

Reason and Prerequisites

This note is relevant for SAP Enterprise Project Connection 2.0 Support Pack 02 Patch 08 release.

Solution

Make sure you install new Support Packages regularly to prevent problems that have already been corrected from occurring in your SAP solution.(SAP's Release Strategy: http://service.sap.com/releasestrategy)

You will find general information and the Download Option under http://service.sap.com/patches

Follow patch installation instructions mentioned in SAP Note 1855364.

Read this note completely BEFORE applying SAP Enterprise Project Connection 2.0 Support Pack 02 Patch 08 to your system and follow the instructions mentioned in this note.

For information on changes, check all the fixes related SAP Notes attached to this note.

This patch requires SAP EPC ABAP Add-on component SAP Notes: 2059521 and 2059522.

If you have any questions or require more information on this patch then open a message with component CA-EPC.
