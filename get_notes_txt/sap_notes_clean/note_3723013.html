Symptom

The SDM version field in table /ACCGO/T_CAKITM is found blank for few records for SDM run during upgrade.

Other Terms

Agricultural Contract Management; ACM; Utility; SDM; SDM version; Contract item table; /ACCGO/T_CAKITM; Migration

Reason and Prerequisites

Program error.

Solution

Utility /ACCGO/ACM_UPD_CAKITM_UTIL has been provided to update SDM version in table /ACCGO/T_CAKITM.

Import the relevant support pack or apply the correction instruction(s).
