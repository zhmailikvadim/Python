Symptom

You maintain security master data for a stock through transaction FWZZ. When you enter data under the tab Security Swap, the system generates a right of category Security Swap which can be executed using transaction FWER. You set the New ID Number equal to the Old ID Number, and you set the denominator of the swap ratio to be smaller than the numerator, meaning that the number of shares decreases. In other words, you model a reverse stock split. When the corresponding right is executed, the system erroneously creates a translation flow, as well as a loss flow.

Other Terms

BUSTRANSCAT 2310, TRLTS_DERIV_REL, TPM_TRL_TRANSLATIONS

Solution

Implement the attached Correction Instruction (CI) or the relevant Support Package.
