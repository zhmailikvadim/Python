Symptom

An empty string response of a restful webservice call is mapped to a null value and can't be processed within an automation sequence as configured.

Other Terms

Universal Webservice Destination, JSON parser

Reason and Prerequisites

Program error.

Solution

Workaround

No workaround available.

Problem Solution by Patch

The problem will be corrected with patch 3 for Production Connector 3.0.

Installation of Patches

Download the current installation program for the patch from the SAP Software Center and install it in accordance with the instructions in the installation guide. For the current version of the installation guide, see the SAP Help Portal at https://help.sap.com/docs/PRODUCTION_CONNECTOR. You can use the installation program for the patch to update a version of your product that is already installed or to set up a new installation if the product is not yet installed on the target computer.

Before you start the installation, see also the relevant release installation note 3706769 for Production Connector 3.0.
