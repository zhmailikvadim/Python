Symptom

Multi-value code list field is not present for selection in Reference Field usage

Environment

Source: dsx
Type: Productive System
System ID/Name: CLOUD / my1002192.de1.crm.cloud.sap
Product version: SAP Sales Cloud and SAP Service Cloud Version 2 1.0
System managed by: SAP Cloud Tenant
Install base item: my1002192.de1.crm.cloud.sap
Sold product: CUSTOMER ONDEMAND

Reproducing the Issue

  1. Open extensibility Administration
  2. For Contact person see field of format type: Code
  3. Open Case -> Referenced Field Usage
  4. Field is not available

 

Cause

multi-value code list fields generates data type of type array which is not supported by reference field usages 

Resolution

To be addressed in a future release
