Symptom

The application dumps (runtime error CREATE_DATA_UNKNOWN_TYPE) occurs during custom field update operation in PPM project.

Steps Leading to Issue

  • Open PPM project
  • Navigate to a task within the project
  • Access Custom Tab (Custom Field Group)
  • Update at least one custom field value
  • Navigate to an element which does not have its own Custom Tab
  • Press Save button - runtime error occurs

Other Terms

DPR_TABLE_EXT, custom fields, custom tab, CL_DPR_API_APPLICATION_OBJECT

Reason and Prerequisites

Program error

Solution
