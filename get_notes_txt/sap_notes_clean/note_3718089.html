Symptom

  • The plugin in the action button of the POD fails to launch.
  • Clicking the button in the POD does not perform any action.

Environment

  • SAP Digital Manufacturing Cloud 2602;
  • SAP Digital Manufacturing Cloud 2605;
  • SAP Digital Manufacturing Cloud 2608;

Reproducing the Issue

Using Data Collection plugin as example:

  1. Open the POD.
  2. Select any SFC.
  3. Open the Data Collection List plugin.
  4. Click on the "Collect" button.
  5. Observe that no action is initiated upon clicking the button.

Cause

  • The Action Button in the POD is configured without "Multi-Instance" enabled.
  • Without this setting, the button attempts to locate the first instance of the plugin in the layout. If no instance exists, the button fails to launch any action.

Resolution

  1. Access the affected POD in POD Designer app.
  2. Go to the "Assign Actions" section under the Action button.
  3. Delete the existing configured action.
  4. Add again the plugin, ensuring the "Multi-Instance Configurable" option is enabled.
  5. After adding the plugin under "Actions", customize it further according to your business needs.
