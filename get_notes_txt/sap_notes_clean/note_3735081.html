Symptom

Virtual object creation fails with the following error message:

An execution error occurred: sql syntax error: incorrect syntax near "ROW": line 1 col 12984 (at pos 12984)

Other Terms

virtual object, virtual procedure, ROW, XSA, SAP HANA Cloud, ABAP remote source

Environment:

SAP HANA Cloud, SAP HANA Database Version 4.00.000.00.1762269801

Steps to Reproduce:

Prerequisites: You've created an ABAP remote source

  1. In the Database Explorer, navigate to Catalog > Remote Sources
  2. Open the ABAP remote source
  3. Expand BAPI Procedures
  4. Select "BAPI_ALM_OPERATION_GET_DETAIL" and click "Create Virtual Object"
  5. Observe the following error:
    An execution error occurred: sql syntax error: incorrect syntax near "ROW": line 1 col 12984 (at pos 12984)

Reason and Prerequisites

SQL generation for CREATE VIRTUAL TABLE/PROCEDURE/FUNCTION does not put quotes around column/parameter names. This can cause an issue in the SQL if the names are also keywords.

Solution

The development team is further investigating this known issue.
