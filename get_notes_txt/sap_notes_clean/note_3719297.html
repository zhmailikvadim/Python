Symptom

While executing temporary assignments you observe that Plant, Material, PIR, version of planned independent requirement and Base Unit of Measure are not stored in database table ARUN_SDO_ALLDOC.

Other Terms

Schedule Temporary Assignments Fiori App, ARUN_SDO_TEMPORARY_ASSIGNMENTS

Reason and Prerequisites

This issue is because of a program error.

Solution

The Program corrections are contained in the tab correction instruction of this note. You need to implement the corrections in your system.
