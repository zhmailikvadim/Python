Symptom

In a storage type, you use the "Stock on Resource" indicator to also create pick-warehouse tasks from HUs that are already located on a resource on route from the storage type. This is the case for a pick-point scenario, for example. For warehouse tasks created at the time at which the HU is located on a resource, however, the system no longer determines the activity area of the storage bin on which the HU was located. This may result in deviations during warehouse order creation and the determination of the profile for storage control.

Other Terms

AAREA, PRCES, PROCS, WOCR, LAER, source resource, source storage bin, P-point pick point

Reason and Prerequisites

There is a program error.

Solution

Implement the corrections.
