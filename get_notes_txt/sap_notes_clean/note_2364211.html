Symptom

When you use the search help H_T528B ("Help View for Positions") to select a position or a work station, a very long runtime is required in the application until the search help is visible.

Other Terms

H_T528B, T528T, PLANS, HR_T528B_F4_EXIT, SELECT

Reason and Prerequisites

Prerequisite:

  • The two tables T528B and T528T contain a very large number of entries (more than 1,000,000 entries).

Reason:

There is a program error.

Under certain circumstances, the system imports all position texts from the table T528T when processing the search help. If a very large number of entries exist in the table T528T, this causes a significant increase in the runtime.

Solution

The correction from this SAP Note ensures that only the required position texts are read from the database. In addition, the selection is restricted to a maximum of 5,000 entries to be displayed.

 

Import the relevant HR Support Package into your system.

To implement an advance correction, implement the attached correction instructions.
