Symptom

You are working with A2X Service Handling Unit. The Handling Unit Item shall be repacked with the action RepackHandlingUnitItem of the entity HandlingUnitItem. The parameter EWMWorkCenter is used to specify the Work Center. The following error message will be raised even the Handling Unit Item exist:

"Handling unit &1 with item &2 doesn't exist."

Other Terms

/SCWM/ODATA_API/124, Status Code 404, /SCWM/CL_BIL_API_HU_COMMON

Reason and Prerequisites

Program error.

Solution

Implement the correction instruction.
