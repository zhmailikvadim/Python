Symptom

The application "Process 194R document - India" is not accessible as the business catalog "SAP_FIN_BC_AP_OPER_IN_PC" is not available.

Environment

SAP S/4HANA Cloud Public Edition

Cause

The catalog is not available due to a missing scope item.

Resolution

  • The scope item "1J2" has to be activated to be able to use the "SAP_FIN_BC_AP_OPER_IN_PC" catalog.
  • The scope item can be activated as suggested in the KBA 3559243 - Activation of Scope Item - 1J2 (Compliance Formats - Support Preparation) in SAP Central Business Configuration - SAP for Me.
