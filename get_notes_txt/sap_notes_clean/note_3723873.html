Symptom

You want to implement Electronic Reporting for France via an Approved Platform (Invoice and Payment List).

This SAP Note delivers the BC sets that enable you to configure the following customizing:

  1. Fiscal Year Variant FR comprises a 10-day period for reporting Invoices under the normal monthly VAT scheme
  2. Report Category IDs: FR_INVOICE_LIST and FR_PAYMENT_LIST for Report Entity FR_RPG_ENT

Other Terms

France e-reporting, France eReporting, Invoice Reporting, Payment Reporting, B2BI, B2C, e-Reporting, eReporting, electronic report, electronic reporting, Statutory Reporting, Invoice List, Payment List, SRF, Document and Reporting Compliance, Document Compliance, Approved Platform, PA

Reason and Prerequisites

Reason: 

This SAP Note is only relevant for France.

Prerequisite: 

For a complete list of all corrections, please refer to the SAP Note:

3713986 - Electronic Reporting for France via Approved Platform: Implementation Overview.

Solution

Versioning

 Version

 Date

 Reason
  01  March 06, 2026

 Release for all Customers

Download the BC sets attached to this SAP Note (corresponding to your software component).

  1. Upload the BC sets to your development system using transaction SCPR3.
  2. Use the function Compare with Customizing Tables (Ctrl + F9) in transaction SCPR20 to ensure that no conflict exists. This is important especially for BC set BC_FIN_LOC_SRF_FYV_FR_EREP.bcs as it overwrites the Fiscal Year Variant FR, if it exists in your System.
  3. After uploading and validating the BC sets, activate them using transaction SCPR20 and set the Expert Mode flag during activation. If you encounter any errors, try activating the BC sets for a second time.

Activate the BC Sets in the sequence as shown below.

  • Software component S4_CORE
    • BCS_FIN_LOC_SRF_FYV_FR_EREP.bcs
    • HBCS_FIN_LOC_SRF_FR_RPG_ENT_EREP.bcs
