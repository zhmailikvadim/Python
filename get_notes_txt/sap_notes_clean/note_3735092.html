Symptom

You maintain customizing settings in Communication Framework. When current client is configured as 'changes without automatic recording', changes should be saved directly without recording in customizing transport.

In case of transactions:

  • /DBE/CF_TASK_TRX
  • /DBE/CF_NOTIF_TRG
  • /DBE/CF_VSS_EVT

system requires a transport entry, which consequently blocks the saving of modifications.

Other Terms

Communication Framework, CMF, Transport Request

Reason and Prerequisites

This is due to program error

Solution

Implement the attached correction
