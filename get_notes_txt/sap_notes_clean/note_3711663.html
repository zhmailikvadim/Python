Symptom

This article provides information on how to find the tables storing travel expense data e.g. the deductions entered in a business trip.

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Environment

SAP Travel Management - on premise

Cause

Resolution

Several trip data are stored in TE Cluster tables - accessible via transaction PRTE.

  1. launch transaction PRTE and enter personnel number and trip number
  2. 'Trip details' overview lists several tables storing data entered in the trip. e.g. deductions are stored in table ABZUG

 

Navigating to details provides information which deductions have been entered in particular; in case of the example shown below a  'Breakfast' deduction has been recorded.
