Symptom

During Synchronous execution, both RSPCENQUEUE  and RSPCSYNCLOCK are locked. This prevents jobs from running simultaneously.

Other Terms

RSPCSYNCLOCK , RSPCENQUEUE

Reason and Prerequisites

Program error

Solution

    Import Support Package 35 for SAP BW 7.00 (SAPKW70035) into your BW system. The Support Package will be available as soon as SAP Note 2009937 with the short text "SAPBWNews NW BW 7.0 ABAP SP35", which describes this Support Package in more detail, is released for customers.
