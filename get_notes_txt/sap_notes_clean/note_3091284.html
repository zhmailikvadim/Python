Symptom

When you log in the system with a different logon language, the run instances maintained on the custom Mass Data Run view are not displaying the description which was previously maintained.

Environment

  • SAP Cloud Applications Studio
  • SAP Cloud for Customer
  • SAP Business ByDesign

Reproducing the Issue

  1. Create a new MDR instance and fill the description.
  2. Log into the tenant on a different language.
  3. The description for the MDR instance is missing.

Cause

This is the expected system behavior. The description field is bound to a language code, and will only be available on the same language where it was initially maintained.

Resolution

The description will be displayed on the same logon language it was created.

Keywords

PDI, SDK
