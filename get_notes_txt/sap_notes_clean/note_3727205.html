Symptom

SAP Note 1051095 enabled the storage of pending data in transparent tables to avoid problems when using Unicode systems.

This SAP Note enables you to specify your own structures in the report RFVD_CHK_TRANSFER_VZFZE. If a table has changed (for example, due to new fields), the conversion may no longer work.

If you specify your own structure in the corresponding field (in the old format), this is reported for the conversion and the conversion is therefore still possible after the change.
