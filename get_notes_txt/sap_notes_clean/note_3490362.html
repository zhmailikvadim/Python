Symptom

The capabilities to develop Mobile Cards in SAP BAS and deploy them to SAP Mobile Services is removed.

Other Terms

SAP Mobile Cards.

Reason and Prerequisites

The strategic way to visualize information on a Card for SAP is to use SAP Integration Cards. SAP Integration Cards can be rendered on a mobile device in SAP Mobile Start.

Hence SAP is removing the feature of Mobile Cards from SAP Mobile Services.

Solution

Option 1: Transition to SAP Mobile Start

SAP Mobile Start can serve as an option for customers interested in a native launch experience for their users. It enables the end-users to immediately react to critical business situations by providing SAP S/4HANA notifications as well as approval workflows delivered by the SAP Task Center.

Please find introductory blogs as well as deep-dive how-to blogs on the SAP Mobile Start community page (https://community.sap.com/topics/mobile-experience/start).

Note: SAP Mobile Start is certified to be integrated into SAP S/4HANA and SAP S/4HANA Cloud but not for SAP ECC.

Pre-requisites: SAP Work Zone, standard edition

 

Option 2: Transition to Mobile Development Kit

In 2019 SAP launched the Mobile Development Kit, to enable customers and partners to build cross platform mobile solutions. Customer have the capability build the mobile clients in the cloud via Cloud Build Service inside SAP Business Application Studio.

The current scope of the MDK is documented in SAP Help https://help.sap.com/doc/f53c64b93e5140918d676b927a3cd65b/Cloud/en-US/docs-en/guides/getting-started/mdk/overview.html and the relevant.

Please check the official roadmap to understand which of the above capabilities are planned for upcoming releases: https://roadmaps.sap.com/ and search for “Cross-Platform Mobile App Development”.

 

Option 3: Transition to SAP BTP SDK for iOS / Android

SAP Cloud Platform SDK for iOS and Android enables fast development of native apps, with Swift or Kotlin. Along with SAP Fiori Design, which includes reusable design components, the SDKs also allows you to tap into the services and functions available in the SAP Cloud Platform so you can build beautiful intuitive native applications. (https://developers.sap.com/mobile)
