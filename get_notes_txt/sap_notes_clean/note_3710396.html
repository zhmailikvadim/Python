Symptom

To implement programmatic access to SAP API Management, you need to get the Client ID and Client Secret of a given APIM tenant.

Environment

SAP API Management

Resolution

To get the Client ID and the Client Secret for a given SAP APIM tenant, follow the steps below:

  1. Create a service instance in the BTP subaccount;
  2. Click on the instance row;
  3. In the right side panel, go to service keys;
  4. Create a new service key;
  5. If a service key is already available and there is no desire to create a new one, ignore step 4;
  6. Click on view service key;
  7. When accessing the service key, the client id, secret, token URL and request URL will be there.
