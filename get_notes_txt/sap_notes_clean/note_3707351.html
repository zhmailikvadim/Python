Symptom

In the Supplier Work Center, the address fields (City, Street, and Country) are not visible in Adaptation Mode for supplier objects.

Environment

SAP Business ByDesign

Reproducing the Issue

  1. Go to the Supplier Work Center.
  2. Choose Supplier Base.
  3. Select any supplier.
  4. Switch to Adaptation Mode.
  5. Observe the Main Address section.
  6. The fields City, Street, and Country are not displayed.

Cause

This behavior occurs due to standard system design.

Resolution

The missing details in Adaptation Mode are expected and align with standard system behavior in SAP Business ByDesign. Certain fields and placeholders are system-controlled and therefore cannot be modified using Adaptation Mode.
