Symptom

There is no focus feedback of User Selector tab.

Reproducible steps

1. Log on to the Web client.

2. Click Administration >> Process Template.

3. Click the "New" button.

4. Click "Value Helper" on the Process Owners row to open User Selector.

Other Terms

SAP Business Planning and Consolidation 10.1 for NetWeaver, WEB, User Selector, Accessibility, Keyboard Accessibility

Reason and Prerequisites

Program error

Solution

For SAP Business Planning and Consolidation 10.1, version for SAP NetWeaver on 740, upgrade to (Component CPMBPC 810) Support Package 11 and upgrade BW740(Component SAP_BW) to Support Package 16.
