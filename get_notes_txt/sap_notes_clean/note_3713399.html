Symptom

This Note corrects some errors in new Device Change Process in passive view.

This Note also provide the implementation to trigger Device Configuration Process (process id: DE_B_IM_DEV_CONFIG_CHG) to order rate use-case 7 or monthly meter readings in new Device Change Process in passive view.

Following processes are involved:

1. Setup / Replacement of Devices (DSO View): DE_B_IM_DEV_CHANGE_DSO_PASV

2. Setup / Replacement of Devices (SUP View): DE_B_IM_DEV_CHANGE_SUP_PASV

Other Terms

device change in passive view, DSO device change, SUP device change, DE_B_IM_DEV_CHANGE_DSO_PASV, DE_B_IM_DEV_CHANGE_SUP_PASV

Reason and Prerequisites

program errors and leftovers

Solution

Please implement the attached SAP Note or import higher version of Support Package.
