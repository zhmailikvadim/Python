Symptom

When running a pricing simulation, you expect the system to display only the conditions that apply to the material within the flexible group. Instead, the system displays all pricing rules, including those that are not applicable to the selected material.

This note corrects the behavior by ensuring that pricing simulation shows only the conditions relevant to the material within the flexible group.

Other Terms

/IRM/GPSM, /IRM/GPS_SRCDOC_COND_DATA_GET

Reason and Prerequisites

This is caused by program error

Please implement SAP Note#3697865 as a Prerequisite notes for 2021, 2022, 2022 V2, 2023, 2023 V2, 2023 V3 and 2023 V3 Support Pack 1

Solution

Please implement the correction instructions attached. These corrections are only valid if you have applied the Support Packs.

Release V/4 – 2021, 2022, 2022 V2, 2023, 2023 V2, 2023 V3, and 2023 V3 Support Pack 1

Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object.
