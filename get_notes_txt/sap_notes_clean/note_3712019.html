Symptom

SAP Customer Checkout provides a feature enabling a second printout of a gift card sale. The second printout will be titled with an additional header "Merchant Copy".

Environment

SAP Customer Checkout

Cause

Configuration / How-to

Resolution

To enable this feature

  1. Go to SAP Customer Checkout => Functions => Configuration => Sales => Article ID of Gift card
  2. Set the feature Print gift card for internal use to ON
