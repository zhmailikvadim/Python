Symptom

When making payments with app "Post Incoming Payments" or "Post Outgoing Payments", it has error "In the context of Data Services an unknown internal server error" after system upgrade.

Environment

SAP S/4HANA

Reproducing the Issue

1. Open app "Post Incoming Payments" or "Post Outgoing Payments" and make postings
2. Error "In the context of Data Services an unknown internal server error" happens

Cause

Usually, cache (either in server or browser) are not cleared after system upgrade. System still uses the old cached data which causes this error.

Resolution

Clear cache as shown in note 2319491

Keywords

Clear Cahe, Internal Server Error, 'post incoming payments', 'post outgoing payments'.
