Symptom

The Cost Optimizer Service dumps when trying to read sourcing information.
This happens when no valid T-Lane & MoT combination exists for a transortation lane relevant to the BoD a product is assigned to.

Other Terms

OTSL, eSPP, Service Parts Planning, Inventory Planning, Cost Optimizer PSM Service

Reason and Prerequisites

Program error.

Solution

Implement the correction in this Note.

The Cost Optimizer now recognizes the missing sources and will stop processing the sub-BoD of the affected location product.
An error is logged to the PSM application log accordingly.
