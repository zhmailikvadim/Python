Symptom

Improvement Request

This enhancement is in response to Customer Connect request 248435.

Inactive approvers with expired accounts were also getting the new work items in their inbox for approval

Request Reason

The solution enables to send the new work item for approval only to the active and valid approvers

So if a user validity is expired then the same will not receive the new work item for approval.

Other Terms

 Invalid, approver, expired, new work item

Reason and Prerequisites

Prerequisite: The user validity should be expired

Solution

Delivery

Upgrade to AC 12.0 SP14 or implement Note 3050472 - "Sending calls for approval only to valid approvers" to get the feature.
