Symptom

Consolidation BOM encounters issues when an existing change number is used for a BOM item.

Other Terms

 CL_MPE_VERSNBOM_API, Consolidation BOM

Solution

Please implement the Note.

In case of consolidation BOM, a new change number is generated for BOM item for every change and version creation when a planning BOM is created from maintain bill of material application.

A new method in the class CL_MPE_VERSNBOM_API is created to generate a new change number.
