Symptom

Indirect Assignment of roles is used on your system, and it is noticed that when the relationship (P-S, S-O etc) percentage is set to 0%, the authorization is not moved from the relevant user.

Environment

  • Human Capital Management (HCM)
  • SAP R/3
  • SAP R/3 Enterprise
  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA OnPremise

Reproducing the Issue

Change the relationship (P-S, S-O etc) percentage to 0% via PPOME, PP01 etc and then execute the indirect role assignment through RHAUTUPD_NEW or RHPROFL0.

Cause

This is the standard design. Percentage is not considered when checking the user via the evaluation path.

Resolution

Please make sure to delimit the relationship when you want to have the role removed.
