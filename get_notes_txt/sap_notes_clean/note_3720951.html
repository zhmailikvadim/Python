Symptom

  • While performing migration using the Migrate Your Data app for the Customer object, the same is to be extended with Credit Management data. 
  • The simulation and migration processes are successful, and the customer is extended to role UKM000. However, when accessing transaction BP and displaying role UKM000, no data is displayed.

Environment

  • SAP S/4HANA

Cause

Functions relevant to credit master data are missing in the customizing configuration.

Resolution

  1. Access transaction CRMC_BUT_CALL_FU.
  2. Add the following entries:
Event     Object   Item     Funct.Name          Call
CRMIN     BUPA     1050000  UKMBP_INBOUND_MAIN  X
CRMOU     BUPA     1050000  UKMBP_OUTBOUND_FILL X
