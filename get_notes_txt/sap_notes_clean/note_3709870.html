Symptom

Sometimes OData API can delete BOM item successfully, but some BOM items are deleted failed with 500 error 

Dump message:

  • Runtime error details:
  • Category: ABAP programming error
  • Runtime Errors: RAISE_SHORTDUMP
  • Exception: CX_RAP_REUSE_SRV_HOST_CALLBACK
  • Development Object: NONT BILLOFMATERIALITEM
  • Application Component: LO-MD-BOM

Environment

SAP S/4HANA Cloud Public Edition 2508

Cause

It is known issue.

Resolution

The issue will be fixed in 2602.1(12th March 2026).

See Also

  • Refer to: SAP Note 3072643 for additional information regarding support platforms.

Keywords

API_BILL_OF_MATERIAL_SRV, http 500 error, CX_RAP_REUSE_SRV_HOST_CALLBACK,NONT BILLOFMATERIALITEM
