Symptom

Which database tables store information about connector completion status, errors, exceptions and messages? 

Environment

SAP SuccessFactors Learning

Cause

Resolution

The tables "PA_CONN_LOG" and "PA_CONN_EXCEPTIONS" store information regarding connector completion status, errors, exceptions and messages. 

  • The error message displayed in the LMS UI under Connector -> View Errors, can be located in the database table "PA_CONN_LOG" within the MSG_DATA field. This table also contains information such as the connector's STATUS, STATUS_CODE, EXCEPTION count, and more.
  • The message and exception displayed in the LMS UI under Connector -> View Results -> View Details and View Logs can be located in the database table "PA_CONN_EXCEPTIONS" table under the ADDL_TEXT field. 

    For further insights, we recommend reviewing the Learning Data Dictionary. More table information can be found in the KBA 2318341.
