Symptom

When you use the Manage Basic Compliance Data - Unpackaged Products app or the Migration Cockpit, you can assign packaged products to an unpackaged product. The system then uses Product Compliance Event Processing (PCEP) to generate a compliance assessment for each packaged product. However, if you assign a large number of products at once, the system encounters an out-of-memory runtime error and stops processing.

Other Terms

PCEP, Migration Cockpit, Creation of Compliance Assessments

Reason and Prerequisites

Reason: The Product Compliance Event Processing (PCEP) is not optimized for the creation of a large number of assessments

Solution

After implementing this SAP Note, the system optimizes the creation of compliance assignments between PECP events and tasks. This optimization prevents the out-of-memory error.
