Symptom

  • Unable to create the import job using SQL database connection due to "An internal server error occurred in the Data Acquisition Service"
  • Existing import jobs would have been hang for more than one hour without success.

Environment

SAP Analytics Cloud

Reproducing the Issue

  1. Logon to the tenant
  2. Go to modeler -> Chose the target model -> Chose Data Management
  3. Click Import Data -> Select a data source -> Select SQL Databases as a Acquire Data

Cause

  • net::ERR_TUNNEL_CONNECTION_FAILED occurred in .har trace log.
  • Tomcat server which hosts SAC Cloud Agent is not running.

Resolution

Restart Tomcat server which hosts SAC Cloud Agent.
