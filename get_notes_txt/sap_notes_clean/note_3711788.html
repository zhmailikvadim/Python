Symptom

When testing an action in the action editor the following error appears :
**********************************************************
Failed to validate received response against error schema defined in action metadata for action 
**********************************************************

Environment

  • SAP Business Technology Platform (SAP BTP)
  • Cloud Foundry (CF)
  • SAP Build Process Automation

Reproducing the Issue

  1. Log on to SAP Build.
  2. Navigate to Action.
  3. Run the action test.

Cause

The error occurs due to a mismatch between the API's actual response and the expected response defined in the specification file.

Resolution

  1. View the actual API response:
    • After executing the test, click the Response button.
    • Click the View API button.
    • Select the Body tab.

  2. Update the output schema:
    • Navigate to the Output tab.
    • Select the relevant output code.
    • Edit the properties to correctly reflect the API's response.
