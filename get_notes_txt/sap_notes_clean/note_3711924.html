Symptom

In the Manage Compliance Scenario app, you go to the “Display by Location” tab. Then, you choose a location in the "Location Structure" section. This SAP Note provides performance optimizations for the following subsequent actions:

  1. Filtering by compliance requirement
  2. Filtering by location classifier
  3. Location Tree: Calculating the number of compliance scenarios belonging to locations
  4. “Scenario  Activities” tab:
    - Displaying concatenated compliance requirement names
    - Calculating validity periods
  5. “Manage Compliance Scenario” tab:
    - Displaying a list of compliance scenarios by selected location
    - Displaying concatenated requirement paragraph names
    - Displaying compliance requirement names
    - Calculating validity periods

Other Terms

CL_EHENV_SCEN_ENT_BYLOC_TR_FDR, CL_EHENV_SCEN_LOC_SRCH_RES_LST, CL_EHENV_SCEN_ACTIVITIES_FDR, CL_EHENV_UI_SCEN_HELPER, I_CMPLRQVERSSTRUCINSTANCEHN, I_CMPLRQVERSSTRUCTURE

Reason and Prerequisites

Reason: Program error.

Prerequisites: The validity of the correction is maintained in the correction instructions.

Solution

This SAP Note provides performance optimizations and resolves the issues related to poor performance in the "Display by Location" tab.

The Support Packages that contain this correction are listed in the "Support Packages" section.
Alternatively, you can implement the correction instructions of this SAP Note.
