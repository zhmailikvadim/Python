Symptom

  •  A Ocean freight booking is created with the container resource.
  •  In Item tab, if you select the container item then the Container Resource details are Displayed.
  •  When you check in the Freight Booking Worklist, In the Resource column the above Container Resource details are not Displayed or empty.

Environment

  • SAP Transportation Management Embedded in SAP S/4HANA 



Reproducing the Issue

1. Go to the Ocean freight booking, In Items tab click in container Item.
2. You can see the container details are displayed.
3. Go to the  freight booking worklist. In the Resource column, Container Resource details are not Displayed or empty

Resolution

Implement the below Note

3707827 - Resource information is missing in the FB worklist tile

Keywords

Resource, Container, Freight booking worklist, ULD, TUR, FB_AIR_DATA_BY_ATTR, FB_OCEAN_DATA_BY_ATTR, res_id
