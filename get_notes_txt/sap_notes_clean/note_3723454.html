Symptom

The connection type is Microsoft Azure Data Lake Storage Gen2. When validating connection, the error occurs:

-------------------------------------------------------------------------------
Connection "xxx" couldn’t be established.
- Data flows can’t be used because of errors in the connection.
- Replication flows can’t be used because of errors in the connection.
- Remote tables are not supported.

Data Flows: Cause: Possible causes of error: Detailed error message (failure to initialize configuration)
Replication Flows: Cause: Possible causes of error: Detailed error message (failure to initialize configuration)

-------------------------------------------------------------------------------

Environment

SAP Datasphere

Cause

It appears to be using an incorrect Shared Key.

Resolution

Verify the Shared Key on your side. Specifically, please check for any extra spaces in the entered Shared Key and test the Shared Key directly from your device to ensure its accuracy. 

Keywords

DSP, DS, DWC, ADLS, Gen2
