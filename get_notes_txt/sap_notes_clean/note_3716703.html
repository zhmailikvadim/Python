Symptom

Columns "source" and "lastModifiedBy" do not show value in Growth Portfolio Export via Integration Center even though the job executor has all the mandatory Growth Portfolio and Integration Center permissions enabled.

Environment

SAP SuccessFactors Talent Intelligence Hub

Reproducing the Issue

1. Download the ICD file for Growth Portfolio Export from SAP Business Accelerator Hub.
2. Import ICD file and create new integration.
3. Add the Destination setting in the Integration Job and Run the Job.
4. In the file exported in the SFTP, notice that the columns "source" and "lastModifiedBy" do not show any values.

Cause

This issue particularly happens when "Growth Portfolio Import and Export via API" is enabled in Provisioning > Company Settings.

This is a Known Issue.

Resolution

Product Engineering is investigating a solution.

Click on star to bookmark this article to receive updates about this issue. For more information about updates from subscribed KBAs, you can refer to the KB Article 2171560 - How to be notified of new or updated SAP Notes or KBAs.

Workaround

Disable "Growth Portfolio Import and Export via API"  in Provisioning > Company Settings.
