Symptom

You declare your import declaration in SAP GTS to customs system DMS.

You use procedure code 42 or 63 and receive an error DKW1099 stating that the Additional fiscal reference must be filled in and Role must be FR2

Other Terms

Denmark, DK, XML, import, Additional fiscal reference, Dometic Duty Tax Party, FR2

Reason and Prerequisites

Problem is caused by a program error.

The Additional fiscal reference is not implemented in the system.

To create the the Additional fiscal reference, you have to declare a partner with the type Customer (PATYP = 'CUSERW').

Solution

Implement the correction instructions.
