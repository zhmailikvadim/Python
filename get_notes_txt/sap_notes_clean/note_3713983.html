Symptom

Product item descriptions are missing in TM documents after an update from EWM. This issue occurrs in Adv. Shipping and Receiving, after setting the status to "Ready For Shipping", for example.

Other Terms

ASR; Adv. SR; EWM-TM integration;

Reason and Prerequisites

Because of a programming error the existing item description was overwritten with an empty entry.

Solution

Implement the corrections of the attached correction instructions or apply the corresponding support package. Afterwards, the product item description will not be deleted.
