Symptom

  • Clarification required on the stage of SUM tool execution for running the report /STTP/MIG40_CDB.
  • Inquiry about whether the report /STTP/MIG40_CDB can be executed before or during SUM tool execution.
  • Confirmation needed on the necessity of running the program /STTPEC/MIG_S4GTINSYNC.

Environment

SAP Advanced Track and Trace for Pharmaceuticals

Reproducing the Issue

  1. Begin the upgrade and conversion process from SAP ATTP 3.0 FP4 to SAP S/4HANA 2310 FP01.
  2. Attempt to determine the correct stage of SUM tool execution for running the report /STTP/MIG40_CDB.
  3. Verify whether the report /STTP/MIG40_CDB can be executed before or during SUM tool execution.
  4. Confirm the necessity of running the program /STTPEC/MIG_S4GTINSYNC.

Resolution

  1. Execute the report /STTP/MIG40_CDB in productive mode before launching the SUM tool.
  2. Run the report /STTP/MIG40_CDB during the pre-checks phase. Once the SUM tool is launched, the report cannot be executed anymore.
  3. The program (/report) /STTPEC/MIG_S4GTINSYNC must be executed as part of the upgrade process if the ECC system is upgraded. Follow the SAP Note 2716224 for guidelines on running /STTPEC/MIG_S4GTINSYNC, and reducing system downtime during migration. Please note, this repost is related to ECC migration.
  4. Follow the below product documentation for additional information:

a. STTPEC: Programs to Run Before Update  

b. Conversion Guide

c. SAP Note 2716224 - S/4HANA 1809+ Conversion/Upgrade Guidelines for Advanced Track & Trace - SAP for Me
