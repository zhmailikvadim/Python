Symptom

An application error is received when trying to open the position UI in 'Manage Configuration UI' page.

Environment

SAP SuccessFactors Employee Central: Position Management

Reproducing the Issue

  1. Login to instance.
  2. Navigate to 'Manage Configuration UI' page.
  3. Open Position UI.
  4. Application error is displayed.

Cause

There are some custom fields in the position object added with incorrect value for 'UI Renderer'.

Resolution

  1. Please correct the value of 'UI Renderer' field for the custom fields.
  2. String type field should not have any value maintained for UI renderer field. Picklist field can have value 'displayPickListWithoutExternalCode' and Generic object fields can have value 'displayPickListWithoutExternalCode'. More details can be found in Adding Fields | SAP Help Portal 

Keywords

Application error in UI, Manage configuration UI error, position error, UI field renderer, unable to open UI, displayPickListWithoutExternalCode, displayPickListWithoutExternalCode, position UI
