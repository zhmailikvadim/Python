Symptom

A new GINA release M26a has been released.

Other Terms

GIN, health information network, e-card release M26a, R26a, GINO

Reason and Prerequisites

Prerequisites:

  • Country Version Austria (AT)
  • Use of GINA interface version M25b
  • Implemented SAP Note 3678999 (interface version R25b/M25b) and any dependent SAP Notes or corresponding release level (patch level)

 Reason:

  • New GINA interface version M26a

Solution

The new services are available in IS-H for the relevant support packages of the specified releases. If you want to use interface version M26a in advance, perform the specified manual tasks and import the transport request that is suitable for your release from the relevant attachments. Then implement the correction instructions in your system.

Time sequence of rollout   

  • Go-live Release M26a (PROD rollout)
    • Wednesday, April 15, 2026

      • The e-card system will be opened on Wednesday, the 15th April 2026 to the new maintenance release. The transition is non-disruptive.

      • All changes (e.g. Removal and enhancement of interfaces, web interfaces) will take effect as of this day. For more information, see the e-card release note Release M26a Version 1.0.

You have the following options for maintaining the prerequisites for M26a from IS-H in your system:

a) Patch change:

  • 6.18 - SP46 (expected availability RTC xx.xx.2026) 
  • 6.17 - SP57 (detto)

b) Implement this SAP Note. Note the manual activities described under "Solution" (for example, Import changes to enterprise services due to the new WSDL files via R/K file in the system, as well as manual source adjustments if necessary). Before you implement this SAP Note, you must also take into account the required release levels.

With this SAP Note or as of the specified AOSPs, the new SVC interface version M26a is supported. If you want to use this version, you must set the parameter GIN - SVC Interface Version as follows:

o SVC when using M26a (current version)
o SVO for (further) use of M25b
