Symptom

  • In the Process Purchase Requisitions (V2) app, the arrow icon that previously appeared next to each purchase requisition line is no longer visible.
  • Users can no longer directly open purchase requisition details in this app.

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Environment

  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Open the app “Process Purchase Requisitions (V2)”.
  2. Observe that the navigation arrow is not shown next to list items, and direct navigation to item details via the arrow is not possible.

Cause

The navigation to object page is not part of the app's current design. Therefore, this functionality was removed by SAP Note 3425366.

In this app, navigation is expected via the document number (Smart Link).

Resolution

To display the purchase requisition, perform the steps below: 

  1. Click the document number in the first column to open the Smart Link popup
  2. Select the desired application from the Smart Link popup to open and view (or edit) the purchase requisition details (e.g., Manage Purchase Requisitions Professional).
