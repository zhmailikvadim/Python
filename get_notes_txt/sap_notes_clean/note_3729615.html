Symptom

  • In Manage Schedule Job Page, a new job is being created and its occurrence is planned to happen daily multiple times.
  • The "Job Occurrence" field displays the daily option with start date but time can be selected just once.
  • How can multiple times be selected in Manage Schedule Job Page?

Environment

SAP SuccessFactors HCM Suite

Cause

Currently, it is not possible to select multiple times in daily recurring jobs via the Manage Job Schedule front end.

Resolution

For jobs that require multiple attempts, reach out to the Partner/Consultant or Support to proceed with the necessary adjustments via provisioning. Below are suggestions on how Support/Partner can assist further:

  • Update exiting job: engage Partner/Consultant or Support to edit and add additional execution times.

    OR

  • Create new job: Engage Partner/Consultant or Support to create the job from scratch (depending of job type, it will require SFTP credentials).

In order to raise ideas and suggest changes to include it in a future release, create an enhancement request as per KBA 2090228 - How to Submit Enhancement Requests for SAP SuccessFactors Products.
