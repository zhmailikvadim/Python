Symptom

After implementing Note 3694898 for JPK FA, customers have encountered an error in some of the P tags.

Other Terms

JPK_FA, PL_SAFT_FA

Reason and Prerequisites

Before proceeding with the implementation of SAP Note 3715406, customers must first apply SAP Note 3694898. This step ensures that all necessary components are available and properly configured.

Solution

After implementing SAP Note 3715406, please verify that the report PL_SAFT_FA can be generated successfully.
