Symptom

Despite implementing note 3568853, the start date is incorrectly displayed, if a planned delivery time is maintained in the info record and no GR duration is maintained for the material master.

When you check the order details from liveCache, you notice, that two DUMMY activities are created. One activity has the same start and end date and one activity has the duration of the planned delivery time.

Other Terms

subcontracting, opening date, start date, dates, times, planned delivery time, delivery time, activity, dummy activity, pdt

Reason and Prerequisites

This note corrects the activity creation in case no goods receipt duration is maintained.

Solution

Implement the attached correction instruction or install the corresponding support package.
