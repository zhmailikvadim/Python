Symptom

  • What is SAP Audit Log Viewer Tool.
  • How to use the SAP Audit Log Viewer to retrieve audit data from an SAP SuccessFactors instance.
  • What are the data centers currently  supports the SAP Audit Log Viewer functionality .

Environment

SAP SuccessFactors HCM Suite 

Resolution

  • The "SAP Audit Log Viewer" tool enables secure retrieval and review of audit logs for SAP SuccessFactors instances through SAP for Me.

Access the audit log with either SAP Audit Log Viewer or the Search API:

    • To view and export audit data with SAP Audit Log Viewer, refer to Getting Started with SAP Audit Log Viewer Service in SAP for Me.
    • To retrieve audit logs with the ALS API, refer to Getting Started With SAP Audit Log Service Search API. The API returns the audit log as a JSON array.

Note:

1. SAP Audit Log Viewer is a UI-based tool in "SAP For Me”, similar to “Change Audit Reports” tool, customer can select audit data type, timeframe. Results are in Excel format, but data is more technical and raw.

2. Search API is API-Based, customer post API request to get audit data, customer input audit data type, timeframe in payload, results are in JSON format.

  • SAP SuccessFactors expose its audit data to the SAP Audit Log service and make it available to customers through the SAP Audit Log service UI and API. Learn how to access audit logs through the SAP Audit Log service  : Accessing Audit Logs Through SAP Audit Log Service | SAP Help Portal 
  • The "SAP Audit Log Viewer" service in SAP for Me is currently available only in specific compliance regions and corresponding SAP SuccessFactors Data Centers.The  tool  can be used only if the instance is hosted in one of the supported Data Centers listed below.

📌 Important Points :

  • Availability of the SAP Audit Log Viewer for SuccessFactors is being gradually expanded. Additional compliance regions and Data Centers will be supported in future releases. This KBA will be updated accordingly as new regions or Data Centers become supported.
  • To verify the current Data Center, refer to the "Show Version Information" in SuccessFactors instance or internal provisioning tools or KBA:2089448 - SuccessFactors Data Center Name, Location, URL, Domain Name, Mail Server details, SFTP details and Outbound IP addresses
