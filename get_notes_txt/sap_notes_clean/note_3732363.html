Symptom

When creating a goal from a performance form using a screen reader, the Save button is announced as Emphasized Button:

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

  • SAP SuccessFactors Goal Management
  • SAP SuccessFactors Performance Management

Reproducing the Issue

  1. Log into SuccessFactors
  2. In the module picker, select Performance > Reviews
  3. Select a performance form
  4. Navigate to the performance goals section and select the + sign
  5. Select Create from Scratch
  6. Populate details and navigate to the Save button
  7. Screen reader announces Emphasized Button

Resolution
