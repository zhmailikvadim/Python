Symptom

A runtime error occurs while processing the DCD:
Category               Installation error
Runtime Errors         CALL_FUNCTION_NOT_ACTIVE
Except.                CX_SY_DYN_CALL_ILLEGAL_FUNC 

Function Group /1BCDWB/SM* contains errors.

Environment

  • SAP S/4HANA
  • SAP S/4HANA Finance
  • SAP enhancement package for SAP ERP, version for SAP HANA

Cause

Function module /1BCDWB/SM* is automatically generated but sometimes this function module was corrupted or not active. 

Resolution

Run the program FP_GENERATE_FUNCTION_MODULE in transaction SE38 using the relevant form name as a parameter.

The relevant forms names can be found in SPRO:

Financial Supply Chain Management 
Credit Management 
Credit Risk Monitoring 
Documented Credit Decision 
PDF Form for Snapshot 
Assign Forms
