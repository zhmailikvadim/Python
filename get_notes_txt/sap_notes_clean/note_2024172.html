Symptom

Another SAP Note requires the corrections contained in this SAP Note as a prerequisite.

Other Terms

ISU_BUPA_BLOCKING

Reason and Prerequisites

This interface note is a preparatory SAP Note allowing the enhanced functions of locking and deleting person-related data in IS-Utilities. Implement this SAP Note if it is necessary as a prerequisite for other corrections.

Solution

Implement this SAP Note or import the specified Support Package. SAP Notes 1998712 and 2000004 must have previously been implemented as basic SAP Notes.
