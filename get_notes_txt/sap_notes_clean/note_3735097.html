Symptom

When executing KKBC_HOE, user encounters the error message: KN020 - "Summarized object does not exist: CO &".

Environment

  •  Controlling (CO).
  •  SAP R/3.
  •  SAP R/3 Enterprise.
  •  SAP ERP Central Component.
  •  SAP ERP.
  •  SAP enhancement package for SAP ERP.
  •  SAP enhancement package for SAP ERP, version for SAP HANA.
  •  SAP S/4 HANA Finance. 
  •  SAP S/4 HANA

Reproducing the Issue

1. Execute KKRC-object hierarchy  via background job for controlling area.

2. Verify logs to confirm hierarchy generation.

3. Run KKBC_HOE.

4. Observe the error message KN020: "Summarized object does not exist: CO &".

Cause

Data collection (summarization) step is a mandatory prerequisite before executing the summarized report KKBC_HOE. If this step is not correctly maintained, the system does not collect data in the defined summarization hierarchy, which results in the KN020 message.

Resolution

1.Define or create the summarization hierarchy using transaction KKR0.

2.Perform data collection for the relevant summarization hierarchy / sub‑hierarchy and required period using transaction KKRC

(Menu path: Controlling → Product Cost Controlling → Cost Object Controlling → Product Cost by Period → Information System → Tools → Data Collection → For Summarization Hierarchy).

3.Execute report KKBC_HOE after successful data collection.
