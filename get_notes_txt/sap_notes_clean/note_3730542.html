Symptom

You use the Fiori Master Request Search and expect the dependent search help for Sales Area fields to filter values based on the selected Sales Organization. When entering a Sales Organization, the Distribution Channel field is not filtered accordingly, and the system displays all possible combinations across all Sales Area fields. However, the search help should restrict values based on the selected Sales Organization.

This note corrects the behavior by ensuring that dependent search help fields are properly filtered based on the selected Sales Area values.

Other Terms

VKORG, VTWEG, SPART, /IRM/VC_GUOBJ, /IRM/CL_GAGR_UI_SERVICES, INPUT_HELP_PROCESS

Reason and Prerequisites

This is caused by program error.

Solution

Please implement the correction instructions attached. These corrections are only valid if you have applied the Support Packs.

Release V4 -  2023 V2.

1. Perform the manual pre-implementation steps that are described below.
2. Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object.
