This SAP Note provides you with corrections for the following EFD register:

  • EFD_C185: The fields VL_UNIT_ICMSST_CONV_COMPL and VL_UNIT_FCP_ST_CONV_COMPL are reported as 0 when they should be empty.

Other Terms

TDF, Brazil, SPED,EFD ICMS IPI,C185.

Reason and Prerequisites

This SAP Note is relevant only for Brazil.

You have implemented SAP Note 3238253.

You have upgraded your system to SAP TDF Support Package 16.

Solution

 After implementing this SAP Note, the following register is modified:

  • EFD_C185: When COD_MOT_REST_COMPL = '2', the fields VL_UNIT_ICMS_ST_CONV_COMPL,VL_UNIT_FCP_ST_CONV_COMPL and VL_UNIT_ICMS_NA_OPERACAO_CONV are empty, and the other fields are filled according to the calculation of the EFD_C185 view.
