Symptom

In the Manage Incident app (EHHSS_INC_REC_OVP), under the main filters tab, you can use the “Involved Person” search value help. The search value help of the “Involved Person” field now features an additional filter, “User ID.” The user ID is now also available as a column in the results table.

Additionally, the “Personnel Number” column in the results table has been renamed to “Work Agreement ID.”

In the Report Incident app (EHHSS_INC_REC_OVP), you can add new Incidents using the “Were there any witnesses?” radio button. With the “Yes” value selected, you can specify the person involved for the new incident creation. The search value help of the “Involved Person” field now features an additional filter, “User ID.”

The user ID is also available as a column in the results table.

Additionally, the “Personnel Number” column in the results table has been renamed to “Work Agreement ID.”
