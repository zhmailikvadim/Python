Symptom

When you want to upload an accrual object with a derived accrual item type in ACACTREE01, you take the example below in the uploaded file:

  • One line is an accrual item with the accrual method LINEAR
  • The other line is a derived accrual item with the accrual method DERIVED, the column Total Accrual Amount in Transaction Crcy is zero

After uploading the file successfully, you find in the result list, the column Total Accrual Amount in Transaction Currency of the derived accrual item type has a value, but it is expected to be zero.

Other Terms

Accrual Upload, ACACTREE01, Total Accrual Amount in Transaction Crcy, TOTAL_ACCR_AMNT_WSL, zero, Derived accrual item type

Solution

Apply this note or upgrade to the corresponding support package.
