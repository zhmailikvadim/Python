Symptom

  • User cannot view all sections of the KE30 Reports
  • The report consists of three parts: general information , Values (2nd) and result calculation (3rd part).
  • User can only view the values in 2nd part, while the 1st and 3rd parts are blank.
  • Other users can view all three parts.
    Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental. 

Environment


•Controlling (CO)
•SAP ERP
•SAP enhancement package for SAP ERP
•SAP enhancement package for SAP ERP, version for SAP HANA
•SAP S/4HANA Finance
•SAP S/4HANA

Reproducing the Issue

  1. Execute transaction code KE30 with Graphical output option
  2. Notice the not all sections are populated

Cause

Microsoft rolled out automatic updates of the WebView2 component to version 144 which introduced two issues in SAP GUI for Windows.

Resolution

Please follow the steps as per KBA 3704912 Latest Webview2 Version 144.xxx causes problems with SAP GUI HTML control when using edge as browser control in SAP GUI for Windows

Refer to KBA 3704912 Latest Webview2 Version 144.xxx causes problems with SAP GUI HTML control when using edge as browser control in SAP GUI for Windows
