Symptom

Error "No sufficient record found in control table" occurs when trying to post the billing document in SAP S/4HANA Cloud Public Edition.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Navigate to the "Create Billing Documents" app.
  2. Enter the billing document type.
  3. Enter the delivery number.
  4. Enter the billing date.
  5. Click on save.
  6. Observe the error.

Resolution

To resolve the error following steps need to be maintained:

  1. Go to SSCUI 106861 (Define Number Ranges for Outgoing Documents).
  2. Check for any entries with blank values in the fields Prefix Character, Number Group, or Number Range Interval.
  3. Maintain valid values in these fields or delete the duplicate/blank entry.
