Symptom

  • Inconsistencies are detected between Infotype 0016 (Contract Elements) and Infotype 0280 (Secondary Contract Elements).
  • Getting short dumps with UNCAUGHT_EXCEPTION CX_SY_PROVIDE_INTERVAL_OVERLAP

Environment

  • Human Capital Management (HCM)
  • SAP ERP Core Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP SuccessFactors Employee Central integration to SAP Business
  • SAP SuccessFactors Employee Central Payroll

Resolution

  • If PA0280 records (secondary infotype records of IT0016) are missing, execute report RPU28000 to automatically generate the missing entries.
  • If inconsistencies exist between PA0016 and PA0280 records, first delete the incorrect PA0280 entries. After the deletion, run report RPU28000 to regenerate the PA0280 records so that they align with the current data stored in PA0016. Deletion of PA0280 records can be performed using transaction SE16N, as no dedicated report is available for removing IT0280 entries.
It is recommended that this approach be tested in a non‑productive system first.
