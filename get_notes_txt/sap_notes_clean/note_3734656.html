Symptom

  • When opening a specific goal plan and selecting Goal Alignment Chart, the following error appears: "An application error occurred. Please try again later or notify support."
  • The following error can also appear: "An error occurred while attempting to load the goal Plan."
  • The issue can appear isolated to a single user.

Environment

SAP SuccessFactors Goal Management

Reproducing the Issue

  1. Open the target goal plan (for example, the current-year goal plan).
  2. Click Goal Alignment Chart.
  3. Observe the errors: "An application error occurred. Please try again later or notify support." and/or "An error occurred while attempting to load the goal Plan."

Cause

This is a Known Issue.

Resolution

The fix for this issue is planned to be deployed on 2H 2025 release (b2611).

For release timelines, please review SAP SuccessFactors Product Release & Road Map Information.

Workaround

No workaround is available.
