Symptom

When try to maintain characteristic master data from SAP BW/4HANA cockpit, got the information that means "can not find or access is not allowed".

Environment

SAP BW/4HANA

Cause

The authorization object S_RS_IOBJA is not added to the BW/4HANA logon user, or the field RSIOBJPART in the authorization object S_RS_IOBJA is not set to 'DATA' or '*', for example the field is set to 'DEFINITION'.

Resolution

Make sure the authorization object S_RS_IOBJA is added to the BW/4HANA logon user with the field RSIOBJPART value 'DATA' or '*'.

See Also

KBA 3436190 - BW Cockpit fails with error: Cannot read properties of undefined (reading 'link') 

Keywords

RSIOBJPART, S_RS_IOBJA, DATA, DEFINITION, infoobject, maintenance
