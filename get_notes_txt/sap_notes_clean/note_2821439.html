Symptom

  • In a Web Intelligence document, there is no option to select (--Show List of Values--) from User Prompt Input
  • Issue is observed only when the SAP BW variable is set to "Selection option".
  • When the SAP BW variable is changed to “Interval” then we are able to see (--Show List of Values--)  in the User Prompt Input.

Other Terms

webi, webintelligence, HTML, DHTML, BEx, BW, OLAP, BICS, variable, selection, mode, editor, viewer, interface

Reason and Prerequisites

Environment:

  • SAP BusinessObjects Business Intelligence Platform 4.2
  • Web Intelligence HTML interface
  • OLAP BICS
  • SAP BW

Steps to Reproduce:

  1. Logon to BI Launchpad
  2. Launch Web intelligence in HTML Mode
  3. Create a Web Intelligence report on SAP BW using OLAP BICS connection
  4. Make sure there are SAP BW variables set to "Selection option" at SAP BW level
  5. Click User Prompt Input
     >> There is no option to select (--Show List of Values--)

Cause:

  • The behavior described here is by design.
  • Indeed, in the User Prompt Input pane, user cannot directly modify the answers for BEx variables of "Selection option" type.

Solution

Workaround:

  • To modify the answers for BEx variables of "Selection option" type, use the 'advanced' option available in the User Prompt Input pane
