Symptom

In employee data replication, user defined a template to replicate IT0185 of a specific country. However, IT0185 is not replicated properly. 

Environment

  • SAP SuccessFactors Employee Central integration to SAP Business Suite
  • SAP SuccessFactors Employee Central integration with SAP Human Capital Management for SAP S/4HANA
  • Employee Master Data Replication
  • Business Integration Builder (BIB)

Reproducing the Issue

  1. Run employee master data replication.
  2. Check IT0185 record.

Cause

In employee data replication process, object_id PerNationalId is hardcoded in method ITERATE_COMPOUND_XML in class CL_ECPAO_IN_EC_DATA. 

The configuration Entity Name in EC in template group is different from PerNationalId.

Resolution

  1. Identify the entity name  in EC.
  2. Revert the entity name in EC to the SAP standard, "PerNationalId".
  3. Perform testing in the system to confirm that IT0185 is correctly generated.

Keywords

employee master data replication, IT0185, s4 hana, infotype record not created, entity name modified, PerNationalId, EC, ERP integration, replication issue
