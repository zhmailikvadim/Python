Symptom

This SAP Note brings a central fix for content-related issues with the SAP S/4HANA data migration content delivered with SAP S/4HANA 2023.

The Note is valid for:

  • SAP S/4HANA > Data Migration > Migrate Your Data - Migration Cockpit > "Migrate Data Using Staging Tables"

Other Terms

Data migration, Migrate your data, EHS Incident, EHS Task.

Reason and Prerequisites

Prerequisities:

  • You have installed SAP S/4HANA 2025 SP00.
  • You are using the SAP S/4HANA migration cockpit.
  • You are using the pre-delivered SAP S/4HANA data migration content without any modifications.

Solution

The Transport-based Correction Instruction (TCI) included with this SAP Note fixes the issues listed in the table below. For a detailed description of the issues, see the linked SAP Notes.

Technical Name Type Issue SAP Note
SIF_MGM_INCIDENT Migration Object SAP S/4HANA Migration Cockpit: EHS Incident - Involved Person cannot be migrated properly 3653173
SIF_EHS_TDEF Migration Object SAP S/4HANA Migration Cockpit: Task Definition - New Location Role Fields 3661582

Be aware that the TCI only corrects objects related to SAP-delivered content. As a result, the generated migration objects will be updated automatically.

Note: If you have modified/copied your object, the correction is not done within your modified/copied object.

You can refer to KBA 2543372 - "How to implement a Transport-based Correction Instruction."
