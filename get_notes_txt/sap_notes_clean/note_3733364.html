Symptom

When checking the year-end grid usage invoice with reference to budget billing in transaction INVMON, if a budget billing bill contains two items (one for the budget billing amount and one for the reduction), the deduction amount is calculated incorrectly and error EDEREG_INV548 is issued.

This issue is relevant for the market role: SUP.

Other Terms

IDEXGE, GUB, INVMON, JVR, /IDEXGE/ISU_DEREG_INV_DATA_020, budget billing, year-end grid usage invoice, deduction amount, EDEREG_INV548

Reason and Prerequisites

Program error.

Solution

Implement the corrections of this SAP Note or import the relevant support package.
After implementing this SAP Note, the system calculates the deduction amount correctly when checking the year-end grid usage invoice with reference to budget billing in transaction INVMON.
You can verify the fix by checking that the deduction amount is calculated correctly when processing year-end grid usage invoices with budget billing references in transaction INVMON.
