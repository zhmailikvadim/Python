Symptom

This SAP Note is a technical SAP Note with NO functional content.

This SAP Note corrects an ABAP Test Cockpit, also known as ATC, error by replacing a dynamic call of report Generic TemSe Viewer (RPUTMS9S) with static SUBMIT statement for the following objects:

  • Class CL_HRPAYBE_DIT 
  • Class CL_HRPAYBE_SBT 
  • Function module HR_BE_BOW_CREATE_TEMSE

Other Terms

 CL_HRPAYBE_DIT, CL_HRPAYBE_SBT, HR_BE_BOW_CREATE_TEMSE, ATC Check

Reason and Prerequisites

 This SAP Note is only relevant for Payroll Belgium.

Solution

  • Generally, SAP recommends that you implement the HR Support Package specified for your release.

  • However, if you need to install a solution earlier, use the SAP Note Assistant to implement the correction instruction.

  • You can find more information about the SAP Note Assistant in SAP Support Portal, under SAP Note Assistant.
