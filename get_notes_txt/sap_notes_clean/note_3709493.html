Symptom

You create a reporting view in Fiori and expect the Collapse option to work for group fields. However, when you try to use the Collapse option, it does not function as expected in the reporting view.

This note corrects the behavior by ensuring that the Collapse option works properly for group fields in Fiori reporting views.

Other Terms

/VIZI/RVWM, /VIZI/LRVWMF0V, VIEW_GRID_MJSON_PREPARE, Fields Goroup

Reason and Prerequisites

This is caused by program error.

Note : Please implement SAP Note#3382825 as a Pre requisite note for2022 V2 and 2023 Release

Solution

Please implement the correction instructions. These corrections are only valid if you have applied the following Support Packs.

Release V/4 – 2022 V2, 2023, 2023 V2, 2023 V3 Base Package or Support Pack 1.

Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object
