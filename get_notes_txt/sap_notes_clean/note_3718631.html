Symptom

You’re getting a validation error in the PO that prevents using asterisks (*) in the Manufacturer Part Number (MPN) field during processing.

Environment

  • S/4HANA Retail & Fashion Management (LO-RFM) 
  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Create a material with a MPN information for example: 12345ABCDEF5G-****-ZXCVBNM
  2. Create a PO with MPN 12345ABCDEF5G-****-ZXCVBNM, system throws an error "Stock segment is not valid..."

Cause

Wildcards are used to define rules for mapping stock to requirements and other nodes in segmentation. They cannot be used as actual segmentation values. If a segmentation value contains a wildcard character, an error is raised.

Resolution

It isn’t possible to add wildcard characters using the standard method.

Keywords

MPN, segmentation, wild card, wild character, asterisks, purchase order, PO, configuration
