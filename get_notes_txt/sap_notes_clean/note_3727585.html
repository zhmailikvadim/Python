Symptom

  • Account determination issue encountered during shipment cost creation.
  • Delivery tab is not visible for the failed shipment cost case.
  • General Ledger (GL) account is not reflected in the item account assignment for the failed shipment cost case.
  • Error message: "Detail account determination is not possible for shipment item xxx"  error  no. VY066.

Environment

  • SAP ERP Central Component all versions  
  • SAP ERP all versions
  • SAP R/3 Enterprise all versions
  • SAP R/3 all versions
  • SAP S/4HANA all versions
  • SAP enhancement package for SAP ERP all versions
  • SAP enhancement package for SAP ERP, version for SAP HANA all versions

Reproducing the Issue

  • Execute transaction "vi02".
  • enter SCD document 
  • status of account determination  (VFKK-STFRE)  is  A/B 
  • Observe that the delivery tab is missing and the GL account is not reflected in the item account assignment for the failed shipment cost case.

    Cause

    • Vy066 error is correct since any delivery is not assigned to the stage of the shipment. 
    • Based on T_56 settings for defined SCD item category
    • Item category cost distribution V_TVFT-KSTAU  is "C" delivery item
    • rec. indicator ( V_TVFTK-KOSTY) refers to Cost type (shipment cost item) and characterizes the reference to which a shipment costs item refers. This can be a shipment header, a transportation leg, a border crossing point, or a load transfer point.

    Resolution

    • ensure delivery are included in referenced shipment at STAGE level.
    • run VT02n -> include delivery in shipment stage at VT02n and save
    • run VI02 and redetermine account determination to be completed before settlement again.
