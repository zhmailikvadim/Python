Symptom

You try to edit additional data for relation A040 (A025) in PP01 transaction and getting info message 00 119 "Protected tab titles are not supported (tab TABSTR_025 title TAB1)". In display mode there is no ths message.

Other Terms

 MP100135, MODULE INIT_PAD25, Screen 4025,  A025, A040

Reason and Prerequisites

Reason for the information message 00 119  is explained in the note 334400 and described in the long text for the message.

Prerequisites:

You have changed the standard settings for the relationship in T777E to allow edit the relationship ( you removed 'X' value from the field MAINT for Indicator: Cannot be edited using standard transactions ).

In SAP standard system, 025 and 040 relationships can't be maintained directly in the basic transaction code PP01. It is defined by setting in the table view T777E.

It is strongly recommended to use only LSO specific transactions for the LSO application. Any changes done in generic transactions like PP01are not supported.

See SAP Note 3707299 - "PP01 is not a valid LSO transaction".

Solution

Apply attached correction instruction or import corresponding support package.
The corection allows you to suppress the message 00 119, but does not allow you to edit additional data.
