Symptom

When executing communication from B2A Manager for REGES, the response can sometime return error 'Error writing xstring to database'. This leads to end of processing and no data is retrieved.

Other Terms

 REGES, B2A Manager

Solution

Processing of responses is corrected to avoid the issue. The response causing the issue is labeled as text (as opposed to json/xml) to avoid the validation issue leading to the rejection of the record.
