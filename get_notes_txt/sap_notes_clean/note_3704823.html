Symptom

Automation Script might encounter failure for the above listed reference Business Scenarios.

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."   

Environment

SAP S/4HANA Cloud Public Edition 2602 HFC02

Reproducing the Issue

Automation Test execution might fail for business scenario J58 due to a new step "Journal Entry Analyzer - New Version".

Cause

The mentioned step is failing because of the change in the behavior of Fiscal Period. The test automation tool enters period value in 2 digits, however in the new app value is recognized with exact value.

Resolution

The standard script with adaption changes will be available in SAP S/4HANA Cloud Public Edition Release 2602.1

As a workaround, in the variant, change the value of Fiscal Period from $MONTH to current period.
