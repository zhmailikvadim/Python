Symptom

  • When launching a Fiori app, a generic error is shown with no further details. 
  • The app fails to load or displays an error popup without technical information in the browser.

Environment

  • SAP FIORI
  • SAP S/4HANA 

Reproducing the Issue

Resolution

  1. Switch to the SAP GUI of the Gateway front-end system used by the Fiori Launchpad.
  2. In the command field, enter /n/IWFND/ERROR_LOG and press Enter to open the SAP Gateway Error Log.
  3. Select the specific error line. The lower pane will displays the detailed information .
