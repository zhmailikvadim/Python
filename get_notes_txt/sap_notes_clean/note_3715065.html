Symptom

The unexpected translation occurs for S/4HANA Service Order in FSM Service Call Type.

Other Terms

FSM, E4C, Connector, PACG, Order, Type, Field, Service

Reason and Prerequisites

Multiple languages are maintained for a single company in table /PACG/ECM_LANG. The connector transfers all languages defined in the configuration to FSM. As a result, the system considers only the language with the last Sort Order indicator. The expected behavior is that only the first entry should be transferred, according to the maintained language sequence.

Solution

Please, implement this correction instruction.
