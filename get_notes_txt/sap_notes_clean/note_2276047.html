Symptom

The Manager can create notes for an employee in the Employee Profile application. They can enter information of an employee as notes for later reference.  As part of the functionality it is possible to add an attachment to the notes. Whenever a new attachment is added, the older attachments are not opening.

Other Terms

HRESS_EMP_PRO_G, CL_HRMSS_GW_RT_EMP_NOTES, Employee Profile , HR RENEWAL , EMPLOYEE NOTES

Reason and Prerequisites

Program Error

Solution

For the Solution, Please apply the corresponding support package. Alternatively, you can apply the correction instructions attached to the note.
