Symptom

  • An attempt to change price control from "S" (Standard Price) to "V" (Moving Average Price) in transaction MM02 under Accounting 1 view results in error message no. M3551.
  • It is not possible to change the value of negative stocks via material master change t-codes.

Environment

  • SAP ERP Logistics (LO)
  • SAP R/3
  • SAP R/3 Enterprise
  • SAP ERP Central Component
  • SAP ERP
  • SAP Enhancement package for SAP ERP
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Open transaction MM02.
  2. Enter Material number and plant.
  3. Navigate to Accounting 1 view.
  4. Attempt to change price control from "S" to "V".
  5. User encounters error message no. M3551 "You cannot change the price control indicator".

Cause

Changing the price control is not allowed in standard when any negative valuated stocks exist for the material. The negative value of stocks can exist either in the current period or in the previous period. 

Resolution

The field 'value based on moving average price (only with price ctrl S)' MBEW-SALKV/MBEWH-SALKV is not available on any material master data screens. Refer to F1 help of the field MBEW-SALKV in transaction SE11 for clarifications.

In transaction MM02 accounting 1 view, via the tab for the current or previous period, user can change the per. unit price to another value, press enter, change it back to the correct value (without saving in between), then the Value/MA price is recalculated to 0.
