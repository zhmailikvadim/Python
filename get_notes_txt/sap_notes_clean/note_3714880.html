Symptom

Payroll Driver is being ran for one or more payroll areas, and terminated employees are picked up in run, which is not expected.

Environment

  • SAP ERP
  • SAP S/4HANA
  • SAP SuccessFactors Employee Central Payroll
  • HR Renewal

Cause

This issue is a side effect of Note 3501215 - Payroll program does not run for a withdrawn employee although the employee has a retro date set.

Tis scenario might be seen after HRSP upgrade to SP level 145 or higher.

Resolution

This issue is fixed by Note 3612657 - Payroll program run for a withdrawn employee although the employee does not have a retro date set

Implementing this Note or the equivalent HRSP will resolve the issue.
