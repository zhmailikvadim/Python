Symptom

In the SAP Fiori application "Monitor EQS Activations and Deactivations," the target status and check status are displayed incorrectly for some special combinations of EQS PoD view and TS type that have no linked MaLo IDs.

Other Terms

US4G, EQS, Monitor EQS Activations and Deactivations, /US4G/EQSC_MCTRT, /US4G/V_EQSMCTRT, Configure Target Status of EQS PoD Activations, Target Status, PoD, TS Type, MaLo ID, Energy Quantity Settlement

Reason and Prerequisites

The error occurred because of a technical issue in the system.

Solution

Implement the corrections of this SAP Note.

After implementing this SAP Note, configure the target status in SPRO. You can do so in Customizing for SAP Utilities by choosing Intercompany Data Exchange > Utilities Solutions for Germany > Master Data Updates > Energy Quantity Settlement > Configure Target Status of EQS PoD Activations.

You can verify the fix by checking that the target status displays correctly for the EQS PoD view and its associated TS types where MaLo IDs are not linked.

Use
In this Customizing activity, you can configure the target status of EQS PoDs for a unique combination of 'EQS PoD Views' and 'Time Series Types.'
You can view the target status of an EQS PoD in the 'Monitor EQS Activations and Deactivations' SAP Fiori application. The target status is based on the combination of EQS PoD view and TS type, which are defined in Customizing.

Prerequisites
You have defined the EQS PoD and TS Type in the following Customizing activities:
- Define EQS PoD Views
- Define Time Series Types

Verwendung
In dieser Customizing-Aktivität können Sie den Zielstatus von MaBiS-Zählpunkten für eine eindeutige Kombination aus „MaBiS-Zählpunkt-Sichten“ und „Zeitreihenarten“ konfigurieren.
Sie können den Zielstatus eines MaBiS-Zählpunkts in der SAP-Fiori-App „MaBiS-Aktivierungen und -Deaktivierungen überwachen“ anzeigen. Der Zielstatus basiert auf der Kombination aus MaBiS-ZP-Sicht und ZR-Art, die im Customizing definiert werden.

Voraussetzungen
Sie haben den MaBiS-ZP und die ZR-Art in den folgenden Customizing-Aktivitäten definiert:
- MaBiS-ZP-Sichten definieren
- Zeitreihentypen definieren
