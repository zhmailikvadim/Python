Symptom

Approval History tab disappears on editing a job history record that previously had workflow approvals in the Latest People Profile (full profile) using the "Correct" option.

Environment

SAP SuccessFactors Employee Central: People Profile - Full Profile

Reproducing the Issue

  1. Open the Job Information card.
  2. Select a record that has Approval History details from a completed workflow.
  3. Select "Correct."
  4. Change any field (for example, a custom field that does not require workflow approvals).
  5. Save the changes.
  6. Observe that Approval History is no longer displayed as a tab.

Cause

This is known issue.

Resolution

  • The fix for this issue is planned to be deployed on April 13 with 1H 2026 release.
  • For release timelines, please review SAP SuccessFactors Product Release & Road Map Information.
