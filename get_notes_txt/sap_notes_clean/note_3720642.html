Symptom

In a S/4 HCM SAP Fiori application (for example, "My Team Calendar (Version 4)") it may happen that in the result list employee names are not filled during the search for employees.

Other Terms

HRFIO_EMPL_SEARCH, CL_HRFIO_EMPLOYEE_API, GET_NAME, GET_SORTABLE_NAME

Reason and Prerequisites

Reason:

Program error.

Solution

The corrections contained in this SAP Note ensure employee names are properly filled during the employee search.

 

Import the relevant HR Support Package into your system.

If you want to correct the problem in advance, implement the correction instructions attached to this SAP Note.
