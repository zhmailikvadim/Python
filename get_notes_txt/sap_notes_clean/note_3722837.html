Symptom

When downloading the file from the Fully Digitalized Electronic Invoices view, error occurs "Cannot download file; Mandatory field(s) Goods and Services Tax Code missing".

Environment

SAP Business ByDesign

Reproducing the Issue

  1. Go to the Customer Invoicing work center.
  2. Go to the Fully Digitalized Electronic Invoices view.
  3. Select the invoice ABC with Open status, click Prepare for Download.
  4. System opens Download File screen, click File Download button, the following messages raise:

error: Cannot download file; Mandatory field(s) Goods and Services Tax Code missing.

warning: Tax classification number is not maintained for the product XYZ.

  • ABC represents the customer invoice ID.
  • XYZ represents the product ID.

Cause

The Tax classification number is not maintained for the product in the invoice, hence Goods and Service Tax code is not determined and filled in the file. So the error raises.

Resolution

Maintain the tax classification number for the product in the system.

  1. Go to the Customer Invoicing work center.
  2. Select Maintain Golden Tax Master Data from the Common Tasks.
  3. In the section of Golden Tax Classification Numbers, click Add button.
  4. Maintain the Product ID or Product Category ID and the corresponding Tax Classification Number. 
  5. Save it.

Afterwards, perform the file download again in the Fully Digitalized Electronic Invoices view.

Keywords

Golden Tax; Fully Digitalized Electronic Invoice; file download; Golden Tax Classification Numbers; Mandatory field(s) Goods and Services Tax Code missing.
