Symptom

  • Overrides are defined for New York compliance form, but the form appears blank
  • Fields are visible in OData API but the form does not display any values.

Environment

SAP SuccessFactors Onboarding

Cause

  • Missing override configuration
  • If existing override is in place, check the fields if correct

Resolution

  1. Verify the configuration of the Global Filter Metadata Override field.
  2. Update the `metaOverrideType` field in the Global Filter Metadata Override as per the guide.
  3. Ensure the correct `entityPath` is configured by navigating to the OData API Data Dictionary and selecting the required navigation property from the table. Construct the `entityPath` accordingly.
  4. Ensure that the relevant fields are visible in the API Center under OData API Data Dictionary
  5. Perform an OData refresh after updating the configuration.
  6. Restart the candidate to capture the changes.
