Symptom

"SAP GRC PROCESS CONTROL" is installed on an SAP system, all of software component of "SAP GRC ACCESS CONTROL" are found in the system, or vise versa.

Environment

SAP ABAP System

Reproducing the Issue

Check the lists of product instance and software component in the system.

Cause

"SAP GRC PROCESS CONTROL" and "SAP GRC ACCESS CONTROL" share the same software component list.

Resolution

Since "SAP GRC PROCESS CONTROL" and "SAP GRC ACCESS CONTROL" share the same software component list, there is no need to install the other if one of them has been installed in the system.

Some detailed info:

The product instance "SAP GRC ACCESS CONTROL -- GRC ABAP Components" contains the same component list as "SAP GRC PROCESS CONTROL -- Process Control".

The product instance "SAP GRC ACCESS CONTROL -- Plug-in on EHP7, 8 and S/4HANA" contains the same component list as "SAP GRC PROCESS CONTROL -- Plug-in on EHP7, 8 and S/4HANA"
