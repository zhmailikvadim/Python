Symptom

  • Creating resubmission via Fiori App F0106A results in an error message: "Navigation to application is not possible".
  • Error occurs while accessing the Fiori app F4709.

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental." 

                  

Environment

  • Fiori-UI for Collections and Dispute Management
  • SAP S/4HANA 

Reproducing the Issue

  1. Open the app "Process receivables".
  2. Select customer.
  3. Select any invoice and click "Create resubmission".
  4. Error message is displayed: "Navigation to application is not possible".

Cause

Missing Configuration

Resolution

  1. Check that the affected app is configured according to the SAP Fiori Library:

      Manage Resubmissions for Accounts Receivable Accountant.

  2. Role SAP_BR_AR_ACCOUNTANT must be assigned to the user.

  3. Catalog SAP_SFIN_BC_AR_COLWL and SAP_SFIN_BC_AR_DOC_PROC must be assigned to the user /user group

  4. Verify that the Service UI_RESUBMISSIONS_MANAGE is active - transaction code SICF

  5. Check the Target Mappings settings for Manage Resubmissions , focusing on CollectionsResubmission-create
