Symptom

When settling a Condition Contract, the resulting credit memo's withholding tax could be wrong in two cases:

- Customer / Supplier line is created on header level (TMFK- ITEM_BASED = blank)
- Customer / Supplier line is created on item or condition level (TMFK- ITEM_BASED <> blank)

SAP Note 3375845 - Wrong WHT base for CCM credit memos, solved the first case.

This SAP Note solves the second case.

Other Terms

WHT Withholding tax CCS WZR WLF

Reason and Prerequisites

Program error

Solution

Apply correction instructions
