Symptom

Is there a recommendation for the use of IP address 0.0.0.0 for SAP ASE ?

Environment

SAP Adaptive Server Enterprise (ASE) - All versions

Resolution

  • The decision to use IP address 0.0.0.0 is a network consideration and not in SAP remit
  • There is no official recommendation on whether to use this address or not from ASE perspective
  • ASE Engineering do not test with this address
  • There are no known instances of ASE installations using this address

  • SAP Support notes that 0.0.0.0 is the most open network address for a host and considering server and data security are (increasingly) important, using that does not appear the most secure choice
