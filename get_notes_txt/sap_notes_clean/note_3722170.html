Environment

  • SAP Cloud Connector (SCC);
  • Infrastructure network;
  • Proxy server.

Cause

Network issues preventing SCC connectivity with SAP BTP platform.

Resolution

Proceed the investigation from the network/security layers of the infrastructure where SCC is running on, as in most cases such connection refused response raises due a network component blocking/refusing the same between SCC and BTP for "connectivitycertsigning.cf.sa31.hana.ondemand.com".
