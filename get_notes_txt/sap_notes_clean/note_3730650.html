Symptom

Some permission descriptions in Role-Based Permissions (RBP) duplicate the permission names.

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

SAP SuccessFactors HCM Suite

Resolution

In 1H 2026 release, the permission descriptions has been enhanced to provide a more detailed information about the permission. Please refer to this help guide for all the list of permissions that have been enhanced​ -> Enhanced RBP Permission Descriptions | SAP Help Portal
