Symptom

When copying a message with POL in the PLMSI Workbench, the Product Version Key from the Structure Version Key is not transferred to the request of the copied message, resulting in incorrect message data.

Other Terms

PLMSI, PLMSIFND, PLM System Integration, POL, Production Operation List, Workbench, request, SV, Structure Version, PV, Product Version, structureVersionKey, productVersionKey, Structure Version Key, Product Version Key

Solution

Implement this SAP Note to ensure that messages with POL are copied correctly in PLMSI Workbench.
