Symptom

When testing the connection in the Manage Withholding Tax Forms, the below error message is shown:

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Environment

  • BSI TaxProfileFactory (a.k.a. BSI eFormsFactory)

  • HR renewal 

  • SAP S/4HANA 

  • SAP ERP  

  • SAP SuccessFactors Employee Central Payroll 

Reproducing the Issue

  1. Navigate to Manage Withholding Tax Forms.
  2. Go to the Third Party configuration
  3. Test connection.

Cause

  • The credentials saved for the connection between ECP and BSI are not being accepted.
  • The BSI Admin user is locked.

Resolution

  1. Update the credentials in the SPRO path:
    Payroll > Payroll USA > Tax > SAP/BSI Interface > BSI SaaS > Define user credentials for BSI TaxFactory SaaS integration
  2. Regenerate the web service with the new information in the SPRO path:
    Payroll > Payroll USA > Tax > SAP/BSI Interface > BSI SaaS > Define web service for BSI TaxProfileFactory integration
  3. Confirm the lock status of the BSI Admin user in SU01 under the Logon Data tab:
    If the user is locked due to multiple authentication failed attempts, unlock the user and try again after completing steps 1 and 2.

Keywords

Test Connection Failed. Error: Please check your Username, Password and Client Number, The server sent HTTP status code 401, BSI TaxProfileFactory, TPF
