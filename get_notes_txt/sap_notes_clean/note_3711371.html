Symptom

  • After doing the picking, the flag "Pick Confirmation" is not updated at the delivery header in VL02N.

Environment

  • SAP ERP
  • SAP ERP Central Component
  • SAP R/3
  • SAP R/3 Enterprise
  • SAP enhancement package for SAP CRM, version for SAP HANA
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Missing setting in SPRO caused the issue.
