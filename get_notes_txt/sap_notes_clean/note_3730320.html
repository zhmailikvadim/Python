Symptom

You are running allocation cycle, and want to include the cycle total in the spool of the results.

Other Terms

SPOOL, ALLOCATIONS,CYCLE,TOTAL,SAPMKGA2

Reason and Prerequisites

Enhanced Development

Solution

Please implement the attached correction.

Add the following entry to the T811FLAGS table to enable total display:

TAB = 'CCSS'
FIELD = 'J_CYCL_SUM'
VALMIN = 'X'

If the total is no longer needed, you can change the value of VALMIN from 'X' to ' '.
