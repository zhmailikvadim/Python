Symptom

Incorrect Date format YYMMDD instead of DD.MM.YYYY is displayed in F1664 Supplier Evaluation by Time when displaying by document.

Environment

  • Materials Management (MM)
  • SAP S/4HANA
  • SAP Fiori

Cause

Incorrect configuration.

Resolution

Update the parameter OQ_DATEBASED_F_PROPERTIES = 'X' with report SAP_RSADMIN_MAINTAIN in SE38.

See Also

SAP Note 3383441 - OQ: datebased dimension $metadata - SAP for Me
