Symptom

  • When attempting to assign the deployment target for starter systems in CBC evaluation workspace, available starter tenants are greyed out and cannot be selected.
  • Below Warning message appears in Assign Deployment Target App. 
    "Demo data is not available for countries/regions configured with the Configuration Localization Tool. Demo data exists in the system for certain deployment targets, therefore these deployment targets are not selectable."

Environment

  • SAP Central Business Configuration
  • SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Add new customer local version country via Configuration Localization Tool (CLT) in Define Scope App.
  2. When attempting to assign the deployment target for starter systems in CBC evaluation workspace, available starter tenants are greyed out and cannot be selected. Below Warning message appears in Assign Deployment Target App. 
    "Demo data is not available for countries/regions configured with the Configuration Localization Tool. Demo data exists in the system for certain deployment targets, therefore these deployment targets are not selectable."

Cause

The starter system currently cannot be provisioned with CLT. KBA 3107866 - SAP Central Business Configuration : Configuration Localization Tool restrictions and limitations 

Resolution

  1. Remove all the customer local version countries in the Define scope app.
  2. Assign the starter system to the target CBC workspace again.
