Symptom

When generating billing documents for the MOS solution, the system consolidates billable items with discontinuous dates into a single billing document, even when there are gaps between the end date of one item and the start date of the next item.

This issue is relevant for the market role: MOS.

Other Terms

MOSB, /MOSB/FM_EVENT_8114, billing documents, billable items, discontinuous dates, billing unit

Reason and Prerequisites

Program error.

Solution

Implement the corrections of this SAP Note or import the relevant support package.
After implementing this SAP Note, the system splits billable items with discontinuous dates into different billing documents.
You can verify the fix by checking that the billing unit is different when billing items with discontinuous dates.
