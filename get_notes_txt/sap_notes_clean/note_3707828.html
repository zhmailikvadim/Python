Symptom

Prospect related fields are missing in the Leads data source.

Environment

  • SAP Sales Cloud Version 2
  • SAP Sales and Service Cloud Version 2

Reproducing the Issue

  1. Go to Settings ->Analytics -> Design Story 
  2. Click New
  3. Select Leads as Data Source
  4. The fields like Prospect Account name and email are missing 

Resolution

In an lead, it's possible to link an existing account/contact or use the prospect option. The fields with prospect option are not available on the Lead data source in SAC.

This is planned in the long-term roadmap, but the concrete timeline is not available.
