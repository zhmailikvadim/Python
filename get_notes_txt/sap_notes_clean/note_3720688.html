Symptom

The check of the enterprise number (UNRS) is enhanced due to requirements of the system check:

  • Company numbers that contain the same digit in the first twelve digits (for example, 444444444444000) are not permitted.
  • Company numbers with a 0 in the first position are regarded as test company numbers and are not permitted in notifications that are created from production systems.

Solution

In addition to the enhancements to the checks for the enterprise number, this SAP Note delivers the following change: The initial entry of the enterprise number did not previously trigger a new notification in the company data notification procedure if there were no other changes within the notification. For an event date as of 1/1/2026, the initial entry of the company number now also triggers a company data notification.

Import the relevant Support Package or implement the attached correction instructions.
