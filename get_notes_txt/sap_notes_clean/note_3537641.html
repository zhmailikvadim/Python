Symptom

A list report has been developed using CDS views. Filtering or searching does not work on fields that are based on virtual elements in the projection view.

Environment

  • ABAP programming model for SAP Fiori
  • SADL Service Adaptation Description Language
  • SAP NetWeaver
  • SAP NetWeaver Application Server for SAP S/4HANA
  • ABAP PLATFORM - Application Server ABAP

Resolution

Aggregations are not executed in ABAP but pushed down to the database. It is needed for better performance and for satisfying paging requirements (for example: OData options $top, $skip). However virtual fields do not exist on the database. Therefore virtual fields cannot be aggregated. It is a known restriction which cannot be circumvented.
