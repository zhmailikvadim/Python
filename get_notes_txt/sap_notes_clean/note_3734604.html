Symptom

This KBA explains how to use program RPUP2D10 to delete PCL2 data. "Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental." 

Environment

  • Human Capital Management (HCM)
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP SuccessFactors Employee Central Payroll
  • SAP S/4 HANA

Resolution

  1. Run program RPUP2D10 via t-code SE38.
  2. Input following fields. 
    Relation ID: Country specific PCL2 relid
    Sort field: PCLx key, which consists of PERNR number and sequence number
    Password: The username of the person who will run the report

    For example, when trying to delete U.S PERNR 9696969 with sequence number 00001, please input following information:

    Relation ID: RU

    Sort field: 0969696900001(09696969: PERNR number, 00001: sequence number)

    The password: *(* means username)

  3. Click ‘Delete’ button.
  4. The successful deletion message displays and shown below.
