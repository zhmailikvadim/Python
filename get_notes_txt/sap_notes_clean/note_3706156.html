Symptom

It is not clear how the depreciation is calculated for "depreciation to the day" (XDAILY) depreciation key in the first year of acquisition. 

Environment

  • FIN Asset Accounting
  • SAP ERP
  • SAP ERP Central Component
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA
  • SAP S/4HANA Finance
  • SAP S/4HANA Cloud Private Edition
  • SAP Fiori for SAP S/4HANA Finance,
  • SAP Fiori for SAP S/4HANA,
  • SAP Fiori for SAP S/4HANA Cloud Private Edition

Cause

There is a special depreciation logic for XDAILY depreciation key

Resolution

  1.  All acquisitions posted in the fiscal year where the depreciation start date lies are considered from the ordinary depreciation start date onwards for XDAILY depreciation key
  2. Period control has no effect on XDAILY
  3. If daily depreciation is activated and transfers need to be considered from the depreciation start date, it is recommended to post the transfers at the ordinary depreciation start date.
  4. If a legal requirement mandates a different period control based on daily depreciation, provide official documentation from the relevant government authority to SAP Support for further investigation.
