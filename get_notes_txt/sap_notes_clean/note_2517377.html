Symptom

  • When posting a goods receipt in transaction VL02N or VL32N, an 'Update Termination' occurs

Environment

  • SAP ERP Plant Maintenance (PM)
  • SAP ERP Logistics (LO)
  • SAP R/3
  • SAP R/3 Enterprise
  • SAP ERP Central Component
  • SAP ERP
  • SAP Enhancement package for SAP ERP
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA, on-premise
  • SAP S/4HANA, Cloud Private Edition

Reproducing the Issue

  1. Run transaction VL02N / VL32N and enter a delivery number:

  1. Select 'Post Goods Receipt' and Save. An 'Update was terminated pop-up' will appear

  1. Run SM13 to view where the termination occurred:

Cause

  • This issue is caused by a program error

Resolution

  • Run correction report RISTEQ07 from SAP Note 316868. This report will already exist on releases above 46C
  • Run the report, first with 'Test Mode' and 'Extended Check Mode' checked
  • The report can then be re-run with 'Test Mode' unchecked to correct any records which need to be repaired

Keywords

LIEP1U01, IS229, Material, Serial Number, MIGO, I/O error Error; function Update, RC= 4, key
