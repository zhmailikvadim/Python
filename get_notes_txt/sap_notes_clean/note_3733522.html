Symptom

X-API Key generated for the development tenant is being used successfully in the test tenant within the same landscape.

Environment

SAP Asset Performance Management

Resolution

  1. Use the same X-API key across tenants only if they are in the same landscape/region.  
  2. Obtain and use the JWT token specific to each tenant’s subaccount for API calls.  
  3. Avoid reusing a JWT token from one tenant in another tenant; for instance, a JWT token from the test tenant will not work for the development tenant.
