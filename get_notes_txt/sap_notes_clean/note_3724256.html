Symptom

Unable to upload the txt file for VAT Declaration (Model 303) for Spain, obtained by means of ES_VAT_DCL_M303_NATIONAL report for certain declarations: February 2026 or 2T 2026, and subsequent.

Environment

  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Log in to https://sede.agenciatributaria.gob.es/
  2. Attempt to import the txt file for model 303 (obtained in SAP system from Fiori App F1515 "Run Compliance Reports" for report category ES_VAT_DCL_M303_NATIONAL)
  3. The file is rejected by Tax Agency with this error log:
  • 30384     Debe cumplimentar la casilla 'Sujeto pasivo con derecho a deducir pago a cuenta de entregas de gasolinas, gasóleos y biocarburantes posteriores a la ultimación del régimen de depésito distinto del aduanero'.
  • E010252    La casilla 'Sujeto pasivo con derecho a deducir pago a cuenta de entregas de gasolinas, gasóleos y  biocarburantes posteriores a la ultimación del régimen de depósito distinto del aduanero' es obligatoria para periodos mensuales a partir de 02

Cause

  • Value "0" at parameter "ES_VAT_FUEL_DEDUCTION" is only valid for 2026 January and 2026 1T declarations.
  • For February 2026 and 2026 2T declarations (and subsequent), valid values for parameter "ES_VAT_FUEL_DEDUCTION" are:
    • “1” = Yes, taxpayer is eligible for deduction  
    • “2” = No, taxpayer is not eligible for deductions (as of February, 2T-4T 2026)

Resolution

  1. Enter the a valid value for parameter "ES_VAT_FUEL_DEDUCTION" under folder "Enter Parameters Specific to Reporting Entity"  at transaction SM34 for SRFVC_RPG_ENT for your relevant Reporting Entity, (or at transaction SPRO -> Document and Reporting Compliance -> Country/Region-Specific Settings -> Spain -> Statutory Reporting -> VAT Return-Form 303 -> General Settings -> Setting Up Your Statutory Reporting) 
  2. After the customizing for parameter "ES_VAT_FUEL_DEDUCTION" is properly maintained, execute a new run for the report
  3. Download a new txt file
  4. Attempt to import the new txt file for model 303 at Tax Agency site
