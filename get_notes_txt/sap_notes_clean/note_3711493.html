Symptom

  • 'Reversed' export declarations are still displayed in the "Manage Export Declarations" app (App ID F4439) even after implementing SAP Note 3690621.
  • Old cancelled customs declarations continue to appear in the app. 

Environment

  • Global Trade Services (GTS)
  • Global Trade Services / Logistics Services (SLL)
  • SAP Global Trade Services
  • SAP Global Trade Services, Edition for Hana

Reproducing the Issue

  1. Open the "Manage Export Declarations" app (App ID F4439).
  2. Verify the display of reversed or cancelled customs declarations.

Cause

Currently, whether an Export Declaration will be displayed in the app F4439 - 'Manage Export Declarations' is determined by following logic: 

If there is action processing proposal (not initial), declaration will always be displayed. 

If the action processing proposal is initial, at the same time, fallback procedure is active/declaration is not set to emergency/declaration is not in reversed status, then declaration will also be displayed. 

Other cases, declaration will not be displayed.

Resolution

Export declarations with status 'Reversed' can still be displayed in Manage Export Declarations dependent on the provide logic which should be checked.
