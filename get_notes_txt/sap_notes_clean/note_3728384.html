Symptom

  • The "Manage Service Entry Lean Services" app is not activated in the Central Procurement Public Cloud system.
  • Users are unable to create service entries in the CP (HUB) system using the Enhanced Limit functionality.

Environment

SAP S/4HANA Cloud Public Edition

Cause

This functionality is currently unavailable.

Resolution

SAP currently restricts SES processing in the HUB to maintain process integrity between the HUB and connected backend systems.
