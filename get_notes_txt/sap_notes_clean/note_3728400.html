Symptom

I am not receiving the confirmation message from the external system for the Quote Message Order Document. As a result, the quote is not generated in the S/4HANA system.

Environment

SAP S/4HANA

SAP Ariba Buyer 


SAP Ariba Sourcing

Reproducing the Issue

  1. A supplier sends a Quote Message Order Document from an external system.
  2. You check for the corresponding quote in your S/4HANA system.
  3. The quote document is missing.

Cause

The feature Confirmation Message from External System is not enabled for the Quote Message Order Document in your account configuration.

Resolution

This is an on-demand feature that must be enabled by SAP Support. This feature allows the system to process the confirmation and generate the quote in S/4HANA. To have this feature enabled, you must create a support case.

Keywords

quote message, s4, integration, confirmation failure, document creation, cxml
