Symptom

Is it possible to change the due date for already existing program in Learning?

Environment

SAP SuccessFactors Learning

Reproducing the Issue

  1. Assign a program to users in SuccessFactors Learning.
  2. Attempt to change the due date for the assigned program.
  3. Observe that there is no option to manually set or modify the required date for the program.

Cause

Resolution

  1. The due date of a program is determined by the program type -> Open-ended, Duration-based, or Scheduled; and the corresponding settings are  configured within the program record.
  2. The system automatically calculates the required date for Duration-based and Scheduled programs based on their defined settings. For Open-ended programs, a required date is not generated.
  3. Once the program type is set for a particular program, it cannot be changed later, and the required date cannot be modified for already existing programs. This is as per system design.
  4. As a workaround, create a new program and set the program type as per the business requirement.
  5. If functionality to change the program type is required in future releases, submit an enhancement request as explained in the KBA: 2090228 - How to Submit Enhancement Requests for SAP SuccessFactors Products
