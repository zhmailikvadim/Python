Symptom

  • The ABAP program "BADI_GCO_ECOM_GEN_RESP_IN=====CP" terminates due to an unexecutable statement.
  • Syntax errors occur in the include "CL_EDOC_GCO_ECOM_GEN_INC======CM007", line 33 of the program "CL_EDOC_GCO_ECOM_GEN_INC======CP".
  • The error message indicates that the identically named component "IS_PROCESS_INSTANCE_RESPONSE[]-GCOS_ECOM_AIF_GEN_RESPONSE-PROCESS_INSTANCE-FOLLOWUP_ACTIONS" cannot be converted to "LS_PROCESS_INSTANCE[]-GENERIC_PROCESS-PROCESS_INSTANCE-FOLLOWUP_ACTIONS".

Environment

  • SAP S/4HANA
  • SAP Document and Reporting Compliance

Resolution

  1. Implement SAP Note 3616236 - Correction for Follow-Up Actions - SAP for Me.
  2. Ensure that the data type "gcot_ecom_followup_actions" introduced by the note 3616236 is correctly applied.

Note: 

  • Before implementing this note 3616236, ensure that all prerequisite SAP Notes are implemented in the correct sequence as described in note 3672304 - What are the prerequisites for implementing Electronic Document Processing for a country/region (non-Peppol) which uses the integration with Document and Reporting Compliance, cloud edition? - SAP for Me 
  • The dump described in the Symptom section occurs only in the SAP S/4HANA system and the SAP Note 3616236 is applicable for only SAP S4HANA system.
