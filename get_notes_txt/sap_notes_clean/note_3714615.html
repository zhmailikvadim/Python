Symptom

 The Goods Receipt event for a Stock Transport Order (STO) in SAP S/4HANA does not send the actual delivery date (dateId=7). As a result, the system does not update the Actual Delivery Date field in the Shipment tab of SAP CGTO.

Other Terms

  • SAP S/4HANA, add-on for Industry Solution for Life Sciences (ISLS)
  • SAP Cell and Gene Therapy Orchestration (SAP CGTO)
  • Goods Receipt Event of Stock Transport Order
  • Actual Delivery Date (dateId=7)

Reason and Prerequisites

 An issue with the code.

Solution

 Please implement the correction instruction provided with the SAP Note.
