Symptom

When checking an effective date object on Manage Data the Search As Of filed shows yesterday date. 

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental." 

Environment

SAP SuccessFactors HCM Suite

Reproducing the Issue

  1. Go to the Manage Data page.
  2. Under Search, select an effective dated object

The Search As Of field shows yesterday's date.

Cause

This is an expected behavior. If you would like to select any other date, click on the calendar icon to change it. 

Resolution
