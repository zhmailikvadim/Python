Symptom

  • Certain fields in custom tasks for Onboarding and Offboarding are not visible in the UI.
  • The discrepancy occurs between the Configured Object Definition and the UI.

Environment

SAP SuccessFactors Onboarding.

Reproducing the Issue

  1. Go to 'Manage Onboarding and Offboarding Tasks'.
  2. Click on "Task Settings" and locate the task you are customizing.
  3. Click the pencil (edit) icon to edit the task.
  4. When the edit panel opens, click "Open Template Editor".
  5. Click the first gear icon, then select "Add Element".
  6. Choose a 'Type', and under 'Data Reference', you will notice that the expected fields are not available.

Cause

Certain fields in the Configure Object Definition tool are set to "Not Visible", preventing them from appearing in the Offboarding UI.

Resolution

  1. Go to 'Configure Object Definitions'.
  2. Search for Object Definition and find your configured template.
  3. Click on "Take Action > Make Correction".
  4. Locate the fields that are not appearing.
  5. Open "Details" for each field.
  6. Change the Visibility setting to: Editable, or Read Only.
  7. Save the changes.
