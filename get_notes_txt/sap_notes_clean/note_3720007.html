Symptom

During the transfer of non-preferential origin plants from SAP GTS to the feeder system via the “Distribute Settings to S/4HANA Systems” transaction (/SAPSLL/PRE_SET_DIST), a runtime error (ITAB_ILLEGAL_SORT_ORDER_BLK) is raised.

Short Text: Error when inserting a block of rows into a sorted table.

Other Terms

Feeder system plant update, non-preferential plant update, /SAPSLL/PRE_SET_DIST, /SAPSLL/PRE_SETTINGS_DISTR

Solution

Implement the attached Correction Instruction (CI) or the relevant Support Package.
