Symptom

Is it possible to hide the Local Time information shown in Spotlight?

Environment

  • SAP SuccessFactors Employee Profile
  • SAP SuccessFactors Employee Central

Reproducing the Issue

1. Search for any employee via Global Search and access their public profile (Spotlight)

2. You will see the information related to Local Time for the searched user.

Resolution

It is not supported to customize / hide items in Spotlight except for adding up to 3 custom fields.
