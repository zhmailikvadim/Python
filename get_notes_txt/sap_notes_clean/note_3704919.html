Symptom

The system has maintained custom components in QMIH_INCL_EEW_PS and IFLOT_INCL_EEW_PS structures.

 In this case the user sends a POST request to the API_MAINTNOTIFICATION with non-existing Functional Location the system terminates with a dump.

Other Terms

Notification, CREATE_NTF_HEADER, BALW_BAPIRETURN_GET2, CONVERSION_EXIT_TPLNR_INPUT, message class IL, error message IL100, IL/100

Reason and Prerequisites

The system should return with an accurate error message.

The returned message: Functional location & does not exist.

Solution

Apply this correction.
