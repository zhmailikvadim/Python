Symptom

Customers require clarification regarding:

  • Whether a third-party CTI license is mandatory for case creation and attachment upload.
  • Supported CTI providers.
  • Alternative methods to create cases and upload attachments without CTI integration.
  • Available APIs and integration mechanisms for external system integration.

Environment

SAP Service Cloud Version 2

Cause

  • Consulting
  • Confusion between:
    • CTI channel capabilities (real-time communication integration such as telephony and messaging) and
    • Case creation functionality, which can operate independently via APIs or email integration.
  • Most CTI Providers are designed for communication channel handling and is not inherently required for case management operations; however, this depends on the vendor's capabilities.

Resolution

  1. CTI Licensing Requirement
    • A third-party CTI license is required only if the CTI channel is being used.
    • The CTI channel in SAP Service Cloud Version 2:
      • Enables telephony integration within Agent Desktop
      • Supports real-time communication (calls, chat, SMS, messaging)
      • Requires integration with a supported external communication provider
    • If customers intend to:
      • Handle phone calls inside Agent Desktop
      • Use softphone capabilities
      • Integrate provider-based chat/SMS/WhatsApp channels
      • A valid license agreement with the respective CTI provider is mandatory. However:
        • Case creation itself does not require a CTI license.
        • Attachment upload does not require a CTI license.
        • CTI is only required when case creation is triggered through telephony or messaging interactions managed by a CTI provider
  2. Supported CTI Providers
    • Commonly integrated providers include:
      • Sinch
      • Genesys
      • ComBox
      • Amazon Web Services (e.g., Amazon SES, Amazon Connect), and more. Please refer to KBA 3156554 - CTI Integration Options with SAP Cloud for Customer (Service and Sales)
    • These providers support telephony and, depending on the vendor:
      • Chat
      • SMS
      • WhatsApp
      • Email
      • Social channels
    • Licensing and commercial agreements must be arranged directly with the provider.
    • These providers enable interaction handling and channel connectivity.
  3. Case Creation Without CTI Integration
    • Cases can be created without any CTI provider.
    • If the requirement includes (CTI integration is not required) :
      • Programmatic case creation
      • Uploading attachments
      • Triggering cases from external systems
      • Running test or simulation scenarios
  4.  Alternative Integration Options
    • Option A – Email Channel Integration
      • Inbound email configuration can automatically create cases.
      • Reference: SAP Service Cloud Version 2 – Email API
    • Option B – Case API Integration
      • Cases can be created directly using the Case OData APIs.
      • Reference: SAP Service Cloud Version 2 – Case API
  5. Attachment Handling
    • Attachments can be uploaded:
      • During case creation
      • After case creation
    • This requires usage of the Document Service API within SAP Service Cloud Version 2.
    • Attachments are stored using Document Service and linked to the case object.
