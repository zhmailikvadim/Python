Symptom

When starting run the Create Withholding Tax Certificates – Mexico app, the message “Source type is not activated for company code” is displayed.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Open the Create Withholding Tax Certificates – Mexico app (EDOC_MX_WTC).
  2. Enter a company code and a period range (for example, period 01 to 01) and a reporting year.
  3. Start the run; the system shows the message “Source type is not activated for company code”.

Cause

The required eDocument Source Type EDOC_LIST for Withholding Tax Certificates was not assigned and activated for the company code.

Resolution

Please assign the source type EDOC_LIST to your company code in the configuration activity Activate Source Type Documents for Company Code (SSCUI 102190).

See Also

Refer to: Configuration Options for Key Users: Electronic Withholding Tax Certificates
