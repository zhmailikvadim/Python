Symptom

When creating a BP with address data in a specific user time zone (time period), on save the BP standard address is not displayed. Even though in table level the data has been created without issue.

Environment

  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Tcode BP, create a BP with address data and save in a specific user time zone.
  2. When displaying the BP again, the standard address is not displayed (displayed as blank).

Cause

There is incorrect handling on time zone when fetching BP address data.

Resolution

Make sure the following notes are implemented:

  • 2234109 - Address information disappears after save.
  • 3128252 - Inconsistency in Business Partners address usages in creation of Business Partners

Keywords

BUT020, BUT021_FS, XXDEFAULT, timezone, BUPA_ADDRESS_GET_DETAIL, CONVERT_VALDT_TIMEZONE
