Symptom

Company Code and Site Code can be selected only during the creation of the SOW Bid. The values appear locked on the SOW created from SOW bid

Reproducing the Issue

  1. Create a new SOW Bid
  2. Submit bid response from the supplier
  3. Locate SOW Bids with Responses
  4. Choose one of the bids to Create an SOW > click Continue
  5. Values for Company Code and Site are not editable

Cause

The SOW inherits these values from the SOW Bid and they cannot be changed

Resolution

  1. Create a new SOW Bid with the required values
  2. Respond to the SOW Bid
  3. Create an SOW from the Response which will inherit values from SOW Bid
