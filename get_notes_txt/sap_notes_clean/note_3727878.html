Symptom

  • What is the sequence notes to be implemented for US Legal Changes 2025 using RFIDYYWT (System version must be lower than S4CORE10201).
  • For S4CORE10201 onwards RFIDYYWT is obsolete, DRC program must be used, see note 3067949.
  • Relevant for US  country only.

Environment

  • SAP ERP
  • SAP Enhancement Package for SAP ERP
  • SAP S/4 Hana
  • SAP R/3
  • Relevant for country US only

Reproducing the Issue

  1. All related issues in US 1099 reporting using Generic Withholding Reporting program RFIDYYWT.

Cause

Notes might be missing.

Resolution

Notes must be implemented in below order:

1. Note 3547966 - Prerequisite of SAP Note 3534174

During the post-implementation step of SAP note 3547966,  if a CI DDIC missing or not yet implemented error appears in the UDO report NOTE_3534174 execution logs, it can be safely ignored.

2. Note 3534174 - US File Legal Change 2024 for RFIDYYWT

3. Implement SAP Note 3680121 - US: Pre-Implementation for SAP Note 3635100

3.1 Error encountered during implementation of note 3534174:

- Error encountered while implementing SAP Note 3534174, which is a prerequisite for SAP Note 3635100.

- Error message: aCI DDIC missing or not yet implemented, follow KBA:

3553080 - Error: aCI DDIC missing or not yet implemented, when executing report NOTE_3534174

 

4. 3635100 - US Forms Legal Change 2025 & Improvements for RFIDYYWT report

5. 3694209 - US File Legal changes 2025 for 1099 & 1042S Reporting in RFIDYYWT
