Symptom

When you use the Document Viewer, there are situations in which the system stops responding to user activity. In this case, the system often displays the message "Document loading..." in the document viewer, or the window from which the document viewer was called shows the message "Waiting for confirmation...", but the system no longer does anything. The only way to resolve the situation is to terminate the SAP GUI session.

Other Terms

IMC, Inter-mode communication, dyInterModusCommPoss, FB03

Reason and Prerequisites

The described situation occurs if system functions are triggered in the Document Viewer window, for example, creating a new session with the menu function "System -> Create Session". As a result of a program error in the disp+work package of the SAP kernel, communication with the document viewer no longer works.

Solution

Use a disp+work package with at least the patch level specified in "Support Packages & Patches".
