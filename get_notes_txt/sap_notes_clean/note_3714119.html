Symptom

After the execution of Early Watch Alert - EWA Report, Table 'MMIM_PRED' has been identified as a relevant object in the Database growth and Prediction Analysis, showing significant volume and technical impact on the System. 

Environment

  • Materials Management (MM)
  • SAP ERP Central Component
  • SAP ERP
  • SAP Enhancement Package for SAP ERP
  • SAP Enhancement Package for SAP ERP, Version for SAP HANA
  • SAP S/4HANA

Resolution

Table 'MMIM_PRED' - MM-IM Temporary Data - Held Data in MIGO 

To re-organize the held data created by end users in MIGO, you have the following options:

a) request the end users to delete no more needed data in transaction MIGO directly
b) delete the data which you think the users do no more need centrally by the system administrator using transaction MBPM

Keywords

EWA, Early Watch Alert, Report, 'MMIM_PRED', MMIM_PRED, Temporary Data, Held Data, Hold, MIGO, MBPM, Database.
