Symptom

The option "OpenID Connect (SAP Cloud Identity)" is not available when creating a new API credential.

Environment

SAP Emarsys Customer Engagement

Reproducing the Issue

  1. Go to Management > Security Settings > API credentials.

  2. Click in the "Create API credentials" button.
  3. The option "OpenID Connect (SAP Cloud Identity)" is not available.

Cause

If you are not seeing all of these items that simply means that your contract type makes some of the options irrelevant to you, so we removed them for your convenience.

Resolution

Open a ticket to Emarsys support team via component "CEC-EMA-ACA" informing that you configured your IAS / Cloud Identity Services (CIS) according to the guide OpenID Connect (SAP Cloud Identity) and need the "OpenID Connect (SAP Cloud Identity)" option available.
