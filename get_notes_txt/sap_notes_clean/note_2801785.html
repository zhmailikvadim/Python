Symptom

In the transportation cockpit you open the dialog window to create a freight order / freight booking. On this popup you enter for example values in fields for source or destination location. Then you open the dialog window Select Schedule Departure. On this dialog window the search criteria for the source or destination location are not filled with the previously entered values on the first dialog window.

Other Terms

TC, /SCMTMS/PLN, Create, FO, FB, Schedule, Departure

Reason and Prerequisites

The problem is caused by a program error.

Solution

Implement the attached correction instructions or import the specified Support Package.
