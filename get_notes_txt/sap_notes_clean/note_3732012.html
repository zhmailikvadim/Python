Symptom

For BPC10.1 on BW7.50 releases, when user opened BPF activity reopen dialog, the "Add All" button is supposed to be enabled when there is any activity available for reopen in the left-side table. Sometimes, after reopening all available activities, the activities of previous steps became available for reopen, and user may find the "Add All" button is not enabled as expected.

Other Terms

reopen dialog, Add All button.

Reason and Prerequisites

Program error.

Solution

Upgrade BPC web client to equivalent support package level(BPC10.1 on BW7.50 SP36).
