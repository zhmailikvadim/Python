Symptom

There are differences in functionality within FSCM Credit Management between SAP Fiori apps and the corresponding SAP GUI transactions, such as features being missing, limited, or behaving differently.

Environment

  • SAP enhancement package for SAP ERP, version for SAP HANA All versions
  • SAP S/4HANA
  • SAP S/4HANA FINANCE
  • Financial Accounting (FI)

Reproducing the Issue

  1. Access the Display Credit Account Data App (F4825)
  2. Attempt to search for a Business Partner
  3. Field 'Higher Level Credit Account' is missing 
  4. Use the Manage Credit Accounts App (F4596) to access this information

This is just an example scenario.

Cause

SAP Fiori apps and SAP GUI transactions are not designed to be 1:1 functional equivalents. SAP Fiori apps are redesigned versions of SAP GUI transactions; therefore, the logic and coding may differ.

Resolution

If a required function is not available in the SAP Fiori app, users can:

  • Use the corresponding SAP GUI transaction to perform the required action.
  • Check whether an alternative SAP Fiori app exists that supports the required functionality.
  • Submit a development request via the Customer Influence and Adoption Program and refer to SAP Note 11 for further clarification.
