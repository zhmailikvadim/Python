Symptom

Is noted that in Latest People Profile Performance and Goals category is shown even with no Performance and Goals been enabled in the instance.

The category appears with error in the label: ???PPX_Performance_Goals???

Environment

  • SAP SuccessFactors Employee Profile
  • SAP SuccessFactors Performance and Goals

Cause

This is expected behavior, the Performance and Goals will appear despite performance and goals been disabled. This is allow usage of Trend cards as this can cards can be used with no Performance and Goals module.

Please note that there is already an enhancement been worked in order to remove unnecessary cards for Performance cards like  performanceGoals, performanceHistory and calibrationHistory  in case Performance and Goals module is disabled. Please track with your CSM: PMU-27627

Resolution

For now you can hide the cards and category in case this is not used. Please check documentation:

  • Showing or Hiding Full Profile Cards
  • Managing Access Control for Full Profile Categories
