Symptom

The Project Management Project is a related object to the Portfolio Item. In Portfolio Management a fiscal year variant is defined for the used portfolio-type. The fiscal year variant is influencing the Financial Planning/Capacity Planning of the item - but not influencing the resource planning of the Project.

Environment

SAP Portfolio and Project Management (PPM)
SAP Portfolio and Project Management for SAP S/4HANA (PPM)
SAP Portfolio and Project Management in SAP S/4HANA (PPM)

Reproducing the Issue

Prerequisition:
In customizing of the portfolio type a fiscal year variant is assigned: Spro > SAP Portfolio and Project Management > Portfolio Management > Global Customizing > Portfolio-Dependent Settings > Define Portfolio Types

In Frontend:

  1. Navigate to Portfolio Management > Portfolio Items
  2. Open an item in detail
  3. Call "Financial Planning/Capacity Planning"
  4. As a result the fiscal year variant is influencing the displayed periods of the Financial Planning/Capacity Planning.
  5. Call the assigned Project Management Project via button "Project > Roles/Tasks"
  6. Go to tab: Resources > Staffing Overview/Resource Overview
  7. The fiscal year variant of the Portfolio doesn´t influence the resource planning.

Resolution

In Project Management the functionality of the fiscal year isn´t used. So, the fiscal year variant which is maintained in the Portfolio doesn´t influence the assigned Project Management Project.
