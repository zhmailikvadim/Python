Symptom

When checking the Forms options, the Auto-Fill button is not visible.

Environment

SAP Enterprise Service Management

Reproducing the Issue

  1. Go to System Settings.
  2. Search for Forms.
  3. Open any Form.
  4. Click on the tree dots (...), the option Auto-Fill is not visible.

Cause

This feature is disabled by default.

Resolution

Please report a SAP Case so that our development team can enable the feature for you.
