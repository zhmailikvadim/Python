Symptom

You have blocked a specific asset class for your company code in the following configuration path:

T-code FAACFG → Asset Accounting - FAA_CONFIG_REDESIGN_1 → Asset Classes → Make Company Code–Dependent Settings for Asset Class

When you create a new asset in the Manage Fixed Assets app, a dialog appears asking you to enter the company code and asset class. However, in the value help for the Asset Class field, the blocked asset class still appears in the list.

If you select this asset class, the system then displays an error message stating that the asset class is blocked.

Other Terms

Universal Parallel Accounting, MFA

Reason and Prerequisites

It is a program error.

Solution

Install the note.
