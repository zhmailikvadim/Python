Symptom

There is a requirement to monitor whether projects have been assigned to a profitability segment

Environment

SAP S/4HANA Cloud Public Edition

Resolution

From Projects side, there is no way to display this information. As this information resides at WBS Element, there is no standard report that displays information regarding individual WBS data in a list format.

However, there is a field that is available in CDS view: I_EnterpriseProjectElement -> EntProjProfitabilitySegment. Using this one can build a report to fetch this information.

 

Also, the need to maintain the profitability segment in the project master is required only if 1:n for sales order assignments is maintained.

If just one sales order is assigned to the billing element, nothing needs to be maintained in the project master, the attribution will be read from the assigned sales order (no changes here).
