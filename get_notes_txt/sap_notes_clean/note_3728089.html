Symptom

  • Error message "FAGL_POST_SERVICE034" appears when attempting to reverse a document using transaction AB08.
  • The error occurs when trying to reverse a document for only one specific ledger.
  • The process redirects to the transaction FB08 interface from transaction AB08, where the error is encountered.

Environment

  • SAP S/4HANA Finance
  • SAP S/4HANA

Reproducing the Issue

  1. Attempt to reverse a document for a specific ledger using transaction AB08.
  2. Observe that the process redirects to the transaction FB08 interface from transaction AB08.
  3. Encounter the error message "FAGL_POST_SERVICE034".

Cause

  • The error occurs because reversing a document for only one specific ledger is not allowed. The system requires the reversal of the entire operational transaction across all ledgers of the company code.

Resolution

  1. Use transaction FB08 to reverse the entire operational transaction instead of attempting to reverse for a specific ledger.
  2. Ensure that the posting period related to the document is open before proceeding with the reversal.
  3. If the posting period is closed, open the relevant period for the account type and G/L account to complete the reversal.
  4. Update the system to the latest support package to avoid potential issues.
