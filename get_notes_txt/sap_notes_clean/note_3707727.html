Symptom

  • A time off request remains in a pending approval status despite being approved by the manager/approver.
    • WorkflowRequest ID exist and is marked as approved
    • However, in workbench and manage data it displays as pending.

Environment

  • SAP SuccessFactors  Employee Central
  • Time Off Workflow

Reproducing the Issue

  1. Submit a time off request in the system.
  2. Approve the time off request as the manager.
  3. Observe that the request remains in a pending approval status in the system.
    • WorkflowRequest ID exist and is marked as approved
    • In workbench and manage data it displays as pending.

Cause

The status of the employee's absence changed due to an edit made to the entry or by a follow up process.

Resolution

  1. Change the status of the affected employee time entry to "Approved".
  2. Perform this action using the Manage Data functionality.

Prerequisite:

  1. Go to Check Tool
  2. Filter the for Time Off feature
  3. Then for Employee Time Workflow
  4. Check the list of affected employee id
  5. Perform the steps suggested in the Check Tool

Keywords
