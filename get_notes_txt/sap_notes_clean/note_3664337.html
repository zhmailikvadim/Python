Symptom

What do suppliers see when receiving a virtual card order (VCO)?

Environment

  • SAP Ariba Buying
  • SAP Ariba Buying and Invoicing

Resolution

All virtual card suppliers, regardless of card provider (Visa or Mastercard), receive the VCO email notification from SAP Business Network with the title [Customer Organization] sent a new Purchase Order. This email provides virtual card order details and includes a Process order button to optionally create an account on SAP Business Network and view the full order details online.

Depending on the card provider that's tied to your virtual card program setup in SAP Ariba Buying solutions, the remaining email notifications and the supplier experience might differ slightly.

For example, if your issuing bank is tied to Visa, you'll receive up to three email notifications:

Email notificationDescription
Security Question Notification

For the first time using the supplier location or ad-hoc vendor contact, this email informs about the security question used when accessing virtual card details on the Visa Payables Automation Portal.

Security Answer Notification

For the first time using the supplier location or ad-hoc vendor contact, this email advises the security answer to provide on the Visa Payables Automation Portal.

Payment Advice Notification

This notification contains a randomized, masked virtual card number for the exact order amount, by secure email protocol. Suppliers click the link in the email body to access the Visa Payables Automation Portal to view the unmasked virtual card number, after entering the security answer and last four digits of the masked virtual card number.

When accessing the Visa Payables Automation Portal, suppliers can also download a copy of the payment remittance for their records.
