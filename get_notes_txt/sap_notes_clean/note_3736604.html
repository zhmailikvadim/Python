Symptom

Net Price in SA is not updated after pricing condition update which need to run report RM06ENP1.
But after running the report has updated the NETPR, it's not triggering the change Output. 

Environment

  • Materials Management (MM)
  • SAP ERP
  • Central Component SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA

Resolution

Price update using IDoc COND_A would not trigger change output for the Scheduling Agreement because this process does not trigger the Scheduling Agreement processing. Similarly purpose of Report RM06ENP1 is to just to determine price for current date and update net price. This report also does not trigger any other process for the Scheduling Agreement.

Keywords

RM06ENP1, output, change, net price
