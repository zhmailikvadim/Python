Symptom

The "Mass Activity - Aggregated Posting" (transaction ETRANSF) cannot post several cases due to the error EDEREG_INV-743: "No suitable G/L line was found for quantity &1 &2"

Other Terms

ETRANSF, EDEREG_INV743, 743(EDEREG_INV)

Reason and Prerequisites

Program error caused by SAP Note 2157445.

Solution

Implement the attached correction instructions or import the specified Support Package.
