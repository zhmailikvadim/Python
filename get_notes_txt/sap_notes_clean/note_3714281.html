Symptom

In some scenarios, customers may notice that the OnboardingInfo object is no longer available or cannot be retrieved via integrations, business rules, or reporting tools after certain onboarding or hiring actions are completed.

Environment

SAP SuccessFactors Onboarding.

Resolution

This behavior is expected.

The OnboardingInfo object is a temporary object maintained by the Onboarding module. Its lifecycle depends on the onboarding process status and it is automatically purged once the onboarding lifecycle ends.

The OnboardingInfo object is removed from the system when one of the following events occurs:

  • Onboarding process is completed
  • Onboarding process is canceled
  • A no-show is triggered
  • Candidate is converted to an employee
  • New hire data is purged
  • The ONB2Process close job is executed

Once purged, the OnboardingInfo object is no longer available for use in integrations, business rules, or reporting. For additional information about this object, refer to: Onboarding Info Object | SAP Help Portal.
