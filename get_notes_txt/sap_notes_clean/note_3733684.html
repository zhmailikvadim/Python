Symptom

  • Our company would like to use the date of birth field for Contingent and SOW Workers
  • When generating the Security ID in the SOW module, the date-of-birth portion is blank

Resolution

Companies can configure Security IDs to be Auto-Generated. Companies that have Auto-Generate Security IDs configured can add or remove certain elements from the generation of the ID.

To configure the Security ID format, follow the below steps:

  1. Sign into Fieldglass as an Admin user
  2. Navigate to Admin Menu > Security ID Formats
  3. Security ID Formats are configured by Country; select an existing configuration or create a new one 
  4. When creating a Security ID format users can add Custom Fields to be included (example: Date of Birth) 

It is important to consider the module to which the Security ID will be generated for:

  1. If adding a custom field like Date of Birth observe the module selected in the Module column
  2. Users can select which Module they want to use with the auto generate Security ID function by clicking +Add Module. Multiple modules can be selected
  3. If users want to use Auto-Generate Security ID for Contingent and Services modules both Job Seeker and SOW Worker must be selected

Only one configuration is allowed per Country. If a Country has been configured previously the configuration will need to be removed and recreated using the modules desired
