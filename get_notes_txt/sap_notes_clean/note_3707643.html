Symptom

In the quality info record of the material, functions such as creating purchase order are blocked. However, when carrying out these activities, system only propose a warning message 06883 "Procurement of material & blocked for quality reasons". The purchase order can still be created.

Environment

SAP S/4HANA Cloud Public Edition

Cause

The message mode in the QM control key which is assigned to the material in master data is set as "Information" in configuration activity 103900 - Define Control Keys for QM in Procurement.

Resolution

  1. Go to configuration activity 103900 - Define Control Keys for QM in Procurement.
  2. Create a custom control key.
  3. Set the message mode as "Error" in the control key.
  4. Assign the new control key to the material.

Keywords
