Symptom

Within the Business Process Monitoring workcenter with SAP Solution Manager 7.2, you wish to change the renderer used by the Metric Monitor tool for alerting data within the Business Process Monitoring or Alert Inbox applications.

Environment

SAP Solution Manager 7.2

Resolution

To change the chart renderer used for the Metric Monitor tool within the Business Process Monitoring application:

  1. Within the Business Process Monitoring workcenter, select the Business Process Monitoring application tile.
  2. Use the Scope Selection button to select a Solution - Branch and display configured alerting.
  3. Select a business process with active alerting to drill-down to the Metric Detail level.
  4. In the Trend and Value column for active alerting, select the chart/Trend icon.
  5. In the Metric Monitoring pop-up window, select the Show/Hide Chart Settings icon.
  6. In the menu bar that appears, the options Chart View and Table View are available to select from.
  7. The Chart View displays the alerting data using a line chart renderer.

To change the chart renderer used for the Metric Monitor tool within the Business Process Monitoring Alert Inbox application:

  1. Within the Business Process Monitoring workcenter, select the Business Process Monitoring application tile.
  2. Use the Scope Selection button to select a Solution - Branch and display configured alerting.
  3. Select an alerting object from the list and within the Alerts list below, select the Open hyperlink for an alert in column Status.
  4. In the Alert Details popup window, under section Alert Details, select the metric hyperlink displaying the chart icon.
  5. In the Metric Monitoring pop-up window, select the Show/Hide Chart Settings icon.
  6. In the menu bar that appears, the options Chart View and Table View are available to select from.
  7. The Chart View displays the alerting data using a column chart renderer.


NOTE:
The different chart renderers used in each application is not an error and is as per design.
