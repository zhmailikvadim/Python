Symptom

Data workbench error: internal error pop-up during file upload/upsert; error message "objectID header error".

Environment

  • SAP Cloud for Customer core applications
  • Data Workbench

Reproducing the Issue

  1. Log in to the application.
  2. Navigate to Data Workbench.
  3. Start an Upsert for the any object using a CSV file.
  4. Observe that an error occurs during upload.

Cause

Including the ObjectID header column in the upsert file causes the error.

Resolution

  1. Open the CSV file intended for the Account upsert.
  2. Remove the "ObjectID" column from the header and any associated values.
  3. Save the file and retry the Upsert in Data Workbench.
