Symptom

In the Data Services Management Console, if you use the object promotion management tool you are not able to see some datastore objects that are present in the central respository.

Other Terms

This SAP Note is a knowledge base to understand the error.

Environment:

  • SAP Data Services 4.2 SP14 Patch 21
  • SAP BusinessObjects Information platform services (IPS) 4.2 SP9 Patch 600
  • Linux environment (SUSE Linux Enterprise Server 11 SP4 / 3.0.101-108.35-default) operating system

Reason and Prerequisites

The local and central repository did not sync properly when adding objects from the local to the central repository.

Solution

There is no fix for this issue.

Steps to Resolve:

  1. Edit the datastore in the Data Services Designer. Save your changes.
  2. Check out the object and then check the object back into the central repository.
  3. Go to the Data Services Management Console and the datastore object is present.
