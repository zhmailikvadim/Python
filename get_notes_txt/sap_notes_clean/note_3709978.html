Symptom

SAP BTP ABAP ENVIRONMENT is used as the central check system to analyze the custom code remotely. And the analysis finishes with a lot of errors like:

  • Exc. CX_SY_RANGE_OUT_OF_BOUNDS Substring access (offset = 0, length = 1) to a data object of the size 0 exceeds valid boundaries.
  • Exc. CX_SY_NO_HANDLER An exception with the type CX_SY_RANGE_OUT_OF_BOUNDS was raised, but was not handled locally or declared in a RAISING clause

Environment

SAP Netweaver Release 700 - 758

Cause

Missing functionality.

Resolution

Implement SAP Note below on the checked system:

3677471 - ATC: Remote stub - Performance improvements and HTTP URIs

See Also

SAP Note 3677471 - ATC: Remote stub - Performance improvements and HTTP URIs

Keywords

CX_SY_RANGE_OUT_OF_BOUNDS, CX_SY_NO_HANDLER, remote, Custom Code Migration, CCM, custom code, BTP
