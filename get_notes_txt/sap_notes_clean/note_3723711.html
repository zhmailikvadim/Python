Symptom

  • Multi-language pack is activated in the SAP S/4HANA system.
  • In the Service Confirmation, the "Notes assignment block" only allows input text in EN language.
  • The dropdown list does not include other languages for example "AR" (Arabic).
  • The issue impacts Service Confirmation and other service / sales documents. 

Environment

SAP S/4HANA

Reproducing the Issue

  1. Create Service Order.
  2. Release the Service Order.
  3. Create Service Confirmation.
  4. Input text in the header "Notes" assignment block.

Cause

  • In SPRO IMG Path Cross-Application Components > General Application Functions > Notes > Establish Language for Notes, there is one entry for 'EN'.
  • This configuration restricts the available languages for notes to EN.

Resolution

  1. Navigate to the SPRO path: Cross-Application Components > General Application Functions > Notes > Establish Language for Notes.
  2. Remove the entry EN, to enable all languages for input text.

Keywords
