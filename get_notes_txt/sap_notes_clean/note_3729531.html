Symptom

While using S/4HANA for Group Reporting on release 1809 OP (S4CORE 103), in the Consolidation Profit Center hierarchy, the concatenation of NodeID and description is applied in node text (description) field.

The desired behavior for the Node Description in the Consolidation Profit Center hierarchy is to solely contain the text only, without any concatenation.

Other Terms

HRPP_NODE, HRPP_NODET, F2918, Manage Global Hierarchies, F7995, Manage Global Hierarchies

Solution

The fix is provided since release 1809 SP16 and from base release of 1909 OP (S4CORE 104).
