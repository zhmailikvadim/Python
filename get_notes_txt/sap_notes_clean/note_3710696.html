Symptom

You notice more dumps of a technical user in connection with the program "CL_FDC_TMP_DOC_ACCESS=========CP". Usually, the dump with the same content occurs even every 5 minutes. These specific dumps are not critical in principle, nor do they prevent the processes of the usual FI processes. However, they may occur in large quantities.

Other Terms

CX_FATAL_EXCEPTION, CX_GOS_API, UNCAUGHT_EXCEPTION, FOL, FI-GL-GL-A, ABAP programming error, CX_ODATA_CV_ATTACH_EXCEPTION, CL_GOS_API====================CP, CL_ODATA_CV_GOS_BO============CP, DELETE_AL_ITEM, SY 530, FDC_TIMER_DAEMON_CLEAN_UP

Reason and Prerequisites

The automatic background job for cleaning up the session status (see also SAP Note 2110680) encountered an error when you tried to remove an attachment. The corresponding attachment no longer exists.

Solution

Implement the attached corrections or import the relevant support package. Afterwards, the relevant exception should no longer occur during the cleanup of the session if the attachment no longer exists.
