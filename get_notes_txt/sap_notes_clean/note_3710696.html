Symptom

Transportation Management in S/4HANA: This SAP Note lays the technical foundation that allows messages from failed / missing charge calculation being passed through if insufficient data is entered into a TM-FRM document (e.g. Freight Order).

You create a Freight Order in a background job with active Automatic Charge Calculation. If the Freight Order has no Carrier assigned the charge calculation is not triggered during save. Transportation charges root node is not created, it causes issues during the Awarding process of Freight Tendering.

Other Terms

ADJ, TRANSPORTCHARGES, DET_ROOT_BS

Reason and Prerequisites

Program error

Solution

Implement the attached Correction Instruction (CI) or the relevant Support Package.
