Symptom

This note provides the reading program by index for the archive object ITMF_ARCH.

Other Terms

Brazil, SPED, S/4HANA, ITMF_ARCH.

Reason and Prerequisites

This SAP Note is relevant only for Brazil.

You have activated the infostructure ITMF_ARCH.

Solution

Follow the steps to activate the infostructures ITMF_ARCH:

  1. Go to transaction SARI.
  2. Choose Customizing.
  3. Choose the infostructure in the Archive Infostructure field.
  4. Choose the pushbutton Archive Infostructure and choose Activate.

Or follow the guidelines described in the SAP note 1780241 - How to create an Information Structure in the Archive Information System.

After implementing this SAP Note, the reading programs by index will be made available for the archive object ITMF_ARCH.
