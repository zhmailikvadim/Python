Symptom

Running the transaction TI10 or TJ05 in background mode can lead to a short dump, especially when the following conditions are given:

  • The System has Commodity Price Risk Hedge Accounting implemented
  • The System is using the Trader's Order Cockpit
  • The Commodity Financial Transaction you want to fixate a price for has a superseeded Hedge Specification assigned.

Other Terms

TI10, TJ05, Trader's Order Cockpit, Commodity Price Risk Hedge Accounting

Reason and Prerequisites

see above. In general, a short dump appears when running the transaction TJ05 or TI10 in background mode.

Solution

run the correction instructions provided within this oss note.
