Symptom

Repeated daily data entries is shown for all sub-tab Days/Weeks/Months in DB02/DBACOCKPIT -> Space -> Database -> Overview -> History

Environment

SAP systems with Oracle database

Reproducing the Issue

Navigate to DB02/DBACOCKPIT -> Space -> Database -> Overview -> History 
All sub-tab (Days/Weeks/Months) shows repeated daily data entries

Cause

Duplicate data collection

The space statistics for DB02/DBACOCKPIT is collected via the Background Job SAP_COLLECTOR_FOR_PERFMONITOR.
The Job forks several ABAP reports. One of them is the report RSORACOL which collects the space statistics for Oracle.

In addition to the above mentioned "standard" collection method other data collection jobs also scheduled like REFRESH SPACE STATISTIC:DEFAULT in the affected system.
This is not required, because the data is already collected by the SAP_COLLECTOR_FOR_PERFMONITOR/RSORACOL.

Resolution

De-schedule the not required duplicate data collection job like REFRESH SPACE STATISTIC:DEFAULT
