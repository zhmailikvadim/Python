Symptom

 After upgrading dpagent, the krb5.conf under krb5 folder would be overwritten.

Other Terms

SDI, Kerberos, krb5.conf

Environment:

  • SAP HANA SDI 2.0 SP08 Patch 20 and earlier

Steps to reproduce:

  • install dpagent 2.7.3
  • upgrade dpagent 2.8.1
  • check the krb5.conf file under krb5 folder

Reason and Prerequisites

krb5.conf file is not backup and restored.

Solution

This correction is delivered in the release(s) listed in the Support Packages and Patches section of this SAP Note.
