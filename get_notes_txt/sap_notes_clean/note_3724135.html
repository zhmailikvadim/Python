Symptom

During change of MCM instance, post acknowledgment to cloud fails with Error "Unable to load the metering location".

Other Terms

/APEU, /UCOM, post acknowledgment to cloud, /APEU/MCM000,  /APEU/MCM274, /APEU/CL_PRST_POST_ACK->DO_ACTION

Reason and Prerequisites

APEU

2602

Solution

Please implement the attached correction instruction.

Due to incorrect data, one of the earlier version of Market Location has created metering task in /APEU/INST_MEAS table. While sending acknowledgment to cloud, incorrect version has been referred leading to error. Reading of metering task table has been corrected to refer the current processing version.
