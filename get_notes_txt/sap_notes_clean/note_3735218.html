Symptom

ABAP short dump DBSQL_SQL_INTERNAL_DB_ERROR similar to the one below:

Category               External Error
Runtime Errors         DBSQL_SQL_INTERNAL_DB_ERROR
Except.                CX_SY_OPEN_SQL_DB
Date and Time          

Short Text

SQL error "SQL code: 129" occurred while accessing table "XXX".
...
...

Database error number: SQL code: 129 

Database error text: "SQL message: transaction rolled back by an internal error: the transaction is already closed externally"

Environment

SAP HANA Platform Edition

Reproducing the Issue

  1. Start transaction ST22 in the ABAP system.
  2. Open the relevant short dump showing DBSQL_SQL_INTERNAL_DB_ERROR with SQL code: 129 and the message “the transaction is already closed externally”.

Cause

Your system has an old SQLDBC version

Resolution

Please upgrade hdbclient as per KBA 2393013 - FAQ: SAP HANA Clients:

  • Point 7. How can a client be installed or upgraded?

See Also

  • SAP NOTE 3165810 - SAP HANA Client Supported Platforms
  • SAP NOTE 2925229 - Query Execution Fails With Error "the transaction is already closed externally"
