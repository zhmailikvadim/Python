Symptom

You are not able to view PDF files in the Manage Electronic Documents app.

Other Terms

Poland, electronic document, electronic invoicing, Poland electronic invoice, SAP Document and Reporting Compliance, KSeF, file type, PDF, incoming

Reason and Prerequisites

Reason

This SAP Note is relevant only for Poland.

Prerequisites

For a complete list of prerequisites, please refer to SAP Note 3279206 - Electronic Document Processing for Poland - KSeF: Implementation Overview.

Solution

After implementing this SAP Note, you will be able to visualize your PDF files in the Manage Electronic Documents app.
