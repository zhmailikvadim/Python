Symptom

During PDS transfer, you experience performance issues (side-by-side only). High runtime occurs during work center selection in CURTO_OPR_WORKCENTER_SEL.

Other Terms

SE38, RIMODINI, PDS, CURTO_OPR_WORKCENTER_SEL, FUGR PGS2, REPS LPGS2F05, IBP, work center selection, selection logic, integration check, g_ARBID_TAB

Reason and Prerequisites

A missing condition in the work center selection logic causes the system to perform an integration-related check even when the internal work center table is empty.

This leads to unnecessary processing and may negatively affect performance. No prerequisites are required.

Solution

Please implement this SAP Note.

The correction adjusts the condition so that the integration check is executed only when the target system is not an IBP system and the internal work center table contains relevant entries.

No manual post‑implementation steps are required.
