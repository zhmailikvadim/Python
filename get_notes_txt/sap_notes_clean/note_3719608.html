Symptom

Component: Risk Assessment
Module: Business Object

When you resolve the association between Risk Assessment (EHHSS_RISK_ASSESSMENT) and Signature (EHHSS_PROC_CTRL_RAS_SIGNATURE), an error occurs.

In transaction ST22 (ABAP Runtime Errors) you get the information:

  • Category            ABAP programming error
  • Runtime Errors   Message Type X
  • ABAP Program    /BOBF/CL_CONFRT_COMPLETE======CP
  • Application Component    AP-RC-BOF-CFN
  • Error analysis: Configuration error of BO EHHSS_PROC_CTRL_RAS_SIGNATURE of component EHS-MGM-RAS-WF; see application information.

Other Terms

EHS-MGM-RAS-WF, Association, cross business object relationship, ref. node, BO Business Object

Reason and Prerequisites

Reason:
The configuration of Business Object EHHSS_RISK_ASSESSMENT contains an error.

Prerequisites:
For information about the validity of the corrections, see the correction instructions.

Solution

After implementing this SAP Note, the system resolves the association between Risk Assessment and Signature without errors. The configuration of Business Object EHHSS_RISK_ASSESSMENT is corrected.

This correction changes the value of constant EHHSS_PROC_CTRL_RAS_SIGNATURE in interface IF_EHHSS_PRS_C:

  • Old value    '8390624DE67DCA48E10000000A42889F’,
  • New value   'E16C4801623C1FE184973100EBFD754D'.

If you use the constant of the UUID in your customer-specific source code, you must replace the value.

The "Support Packages" section lists the Support Packages that contain this correction.
Alternatively, you can implement the attached correction instruction.
