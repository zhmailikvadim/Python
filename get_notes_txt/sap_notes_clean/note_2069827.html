Symptom

When application's height is longer than the containing window, no scrollbar is added when the user selects the scale mode to be "Auto-fit width".

Other Terms

Auto fit, auto fit width, VC5, scrollbar

Reason and Prerequisites

Scrollbars should be automatically added when necessary in every scale mode. That is because in VC5 the minimun size of the objects is their dimensions in the design board. The patch will add scrollbars in auto-fit to width scale mode when application height is greater than the window's height (the browser), and when application widthis greater than the window's width.

Solution

Install the patch for relevant versions.

Workaround: use the scale mode "fix" instead.
