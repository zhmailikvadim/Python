Symptom

The GRID_VOLTG_LVL value is not available in the step data for the POD, causing the customizing logic to be evaluated with an empty parameter.

Other Terms

/APEU/, /UCOM/, MCM, Step Data, /APEU/CL_PRST_PREP_TMD_DATA, /APEU/CL_PRST_PREP_MIN_TMDDATA, /APEU/CL_PRST_PREP_DATA

Reason and Prerequisites

APEU 2602

Solution

This solution provides an enhancement to pass the value of Grid Voltage Level (GRID_VOLTG_LVL) into the step data.
