Symptom

When you try to delete a manual accrual object, the dialog of "Parameters for Accrual Posting" pops up.

You manually adjust default values of document date, posting date and posting period, the system still generates the journal entry on the default prompted date.

Other Terms

ACACTREE02, manual accruals, delete, posting period, posting date, document date

Reason and Prerequisites

This is a program error.

Solution

Apply the support package including this SAP Note or implement the corrections provided by this SAP Note.
