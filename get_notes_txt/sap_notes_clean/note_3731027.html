Symptom

When using the Individual Talent Data Transfer from Inactive to Active Employment, the users Trend portlet data from performance forms is not visible to the new employment.

Environment

SAP SuccessFactors Performance Management

SAP SuccessFactors Employee Profile

Reproducing the Issue

  • Go to Admin Center  Manage Individual Talent Data Transfer.

  • Select the person for whom you need to transfer data.

  • Select the source employment (inactive) and the target employment (active).

  • Choose the Transfer button to start the process.

  • After the transfer, note that the Trend block does not display the ratings from forms (if configured) that were present for the Inactive employment.

Cause

Known issue.

Resolution

A fix for this issue will be live on the 12th May 2026

Please tag this KBA to be notified of future updates:

  • 2171560- How to mark a KBA or SAP Note as a favorite and get notified when it is updated - SAP for Me https://me.sap.com/notes/2171560 

Workaround:

  • Routing the forms back and forward for the new employment allows the Trend portlet to reflect the forms.
