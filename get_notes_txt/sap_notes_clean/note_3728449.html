Symptom

Component: Incident Management
Module: Report Incident

You report incidents in the Report Incident application (ID: F1992). If you have insufficient permissions, the system does not correctly filter objects with failed authorization checks from the result list.

Other Terms

CL_EHHSS_INC_D_AUTH_CHECK

Reason and Prerequisites

A program error in the system causes the symptom.

The correction instructions contain the prerequisites.

Solution

After implementing this SAP Note, the system correctly filters objects with failed authorization checks from the result list.

The Support Packages section lists the Support Packages that contain these corrections.

Alternatively, you can implement the attached correction instructions.
