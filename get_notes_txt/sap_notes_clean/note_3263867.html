Symptom

There is an issue where an SAP Transaction code is missing for the for the deal usage report.  Additionally, some selection criteria are missing.  This note corrects those issues.

Other Terms

deal usage report, expected accruals, selection criteria

Reason and Prerequisites

PFG - Missing report on program /IRM/GDL_USAGE_DISPLAY

Solution

Manual instructions for creating the transaction

These corrections are valid for the system if you have one of the following releases of V4IA installed in your system:

Release V/4 1910 Base Package, 2020, 2020 V2


1. Correction is delivered in this note as manual pre-implementation step
2. Use transaction SNOTE to implement the manual correction instructions and activate the transaction code
