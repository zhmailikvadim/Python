Symptom

No employee data is included in the export file when exporting data from a custom MDF object.

Environment

SAP SuccessFactors HCM Core

Reproducing the Issue

  1. Go to Import and Export Data.
  2. Select the custom MDF object
  3. Click on Export.
  4. Go to Scheduled Job Manager.
  5. Click on View Details.
  6. Click on Download Status.
  7. Save Zip File.
  8. Open Zip File.
  9. There is no CSV file created that contains the user data.

Cause

The object has special characters in its label/name.

Resolution

  1. Open the object on the Configure Object Definitions page
  2. Remove any special characters like "()", "/", ">" that should exist on the object label/name.
  3. Export the data again as required.
