Symptom

The user creates a Document Version with a file attached as dynamic URL and corresponding customizing in SPRO 'Attachment Service - Storage Repository Activation' is missing.

As a result, the message 010 of message /PLMF/COMMON 'Exception called from &1 in &2 line &3' is raised instead of proper message about missing attachment service customizing.

Other Terms

PLMSI, Document Version, File, Dynamic URL, Attachment, Exception, Message

Solution

Please implement the note 3441639.
