Symptom

When the user uses a variant class as knowledge-base profile to collect attached configurable materials, the knowledge-base check shows contradicting messages for each material about configuration profile selection. The knowledge-base generation fails with error message "No config. profile exists for material ...".

Other Terms

KBGEN411, KBGEN042, KBGEN043, KBMTX_COLLECT_CONF_PROFILE, COLLECT_OO_CLASS, FIND_OR_COLLECT_MATERIAL, KMAT

Reason and Prerequisites

Program Errors.

Solution

Please implement the attached correction instructions.
