Symptom

  • HRA wage type derivation in IT0008 is incorrect as per the entered master data.
  • While maintaining IT0008, the system derives incorrect HRA wage types
  • When creating Infotype 0581 (Housing Rent Allowance) and maintaining HRA, the system automatically navigates to Infotype 0008 (Basic Pay)

Environment

  • SAP ERP Human Capital Management
  • SAP ERP Payroll India
  • SAP R/3
  • SAP R/3 Enterprise
  • SAP ERP
  • SAP ERP Central Component
  • SAP enhancement package for SAP ERP

Reproducing the Issue

  1. Use transaction PA30.
  2. Create Infotype 0008.

Resolution

  1. HRA wage types in IT0008 are derived via Dynamic Action linked to IT0581.
  2. This behaviour is standard, and a default Dynamic Action is linked to IT0581 and shipped with the standard solution.
  3. This dynamic action can be seen in the view V_T588Z.
