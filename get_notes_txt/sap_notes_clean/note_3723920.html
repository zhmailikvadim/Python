Symptom

When a order is edited, previously entered text is not protected and can be edited or deleted

Environment

  • SAP ERP Plant Maintenance (PM)
  • SAP ERP
  • SAP S/4HANA, on-premise
  • SAP S/4HANA, Cloud Private Edition

Reproducing the Issue

  1. Run transaction IW32
  2. Edit a order
  3. Adjust the existing long text

Cause

The order transactions or apps do not provide any option to lock long text. The text is not protected and can be edited by another user

Resolution


This feature is available for notifications but not for orders. If your orders have a linked notification then the text could be maintained there which provides log line and protected text

Keywords

IW32, Change, Process Maintenance Order, TEXT_CREATE_HISTORY_LINE,  SUSR_USER_ADDRESS_READ, tline-tdline
