Symptom

  • ATC exemption status is not updated in ADT, allowing multiple exemptions for the same ATC finding and preventing visibility of rejected statuses in ADT.
  • Approved ATC findings are not excluded from the ATC list in ADT.
  • In ADT, double-clicking certain ATC findings (e.g., "Usage of Internal API", "Usage of transactional-consistent classic API") results in "No Documentation available", while details are visible in SAP GUI.
  • ATC execution is only available at the package level, with no option for object-level execution.
  • Raising ATC exemptions from SAP GUI is not currently possible, which impacts usability for team members relying on SAP GUI.

Environment

ABAP Check Frameworks - ABAP Test Cockpit, Code Inspector

Reproducing the Issue

  1. Enable ATC on BTP ABAP 2511.
  2. Create a Clean Core variant and test it on a custom program.
  3. Execute ATC checks using the variant on a specific object or package.
  4. Raise ATC exemption requests via ADT for findings such as "Usage of Internal API" or "Usage of transactional-consistent classic API".
  5. Observe that approved ATC findings still appear in ADT and SAP GUI, and exemption statuses are blank, allowing multiple exemptions for the same finding.
  6. Double-click on certain findings in ADT and observe "No Documentation available" error.

Cause

  • The issue is caused by a bug in the Cloud system where the checksum version is not correctly propagated during the creation of ATC exemption requests.

Resolution

  1. Implement SAP Note 3693719 in the affected system to address the bug related to checksum versions for specific checks like "Usage of APIs".
  2. After implementing the note, recreate the exemptions for the affected findings.
  3. Verify that the exemption status is updated correctly and approved findings are excluded from the ATC list in ADT and SAP GUI.
  4. If the issue persists, check the configuration settings in the ATC results view and disable the "Include Exempted Findings" option to ensure only unresolved findings are displayed.
  5. Refer to the maintenance schedule for Hotfix Collection 4 for Release 2602 (February 13th - 15th, 2026) for additional corrections.
