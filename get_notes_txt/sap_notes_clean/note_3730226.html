Symptom

  • An error occurs when attempting to enroll a benefit in the benefits portlet.
  • Error message: "The URI is malformed. Invalid $filter expression: id eq".

Environment

Global Benefits

Reproducing the Issue

  1. Go to Benefits > Benefit Overview.
  2. In Benefits Available to Enrol, select the relevant benefit and choose Enrol.
  3. Observe the error: "The URI is malformed. Invalid $filter expression: id eq".

Cause

A product defect in the UI where the benefit id is not passed when loading the deductible enrollment, resulting in a malformed filter.

Resolution

A Jira bug ticket BEN-22070 was created to make code fix for B2605, which will be released on 13-Apr-2026 for Preview and on 15-May-2026 for Production.
