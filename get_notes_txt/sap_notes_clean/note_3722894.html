2. Enhancements and Legal Changes

E-Bilanz: Templates for taxonomy version 6.9
New templates for E-Bilanz taxonomy version 6.9 are available. See also SAP Note 3725080.

ERiC software component has been updated to version 43.3.2.0
For E-Bilanz reporting, the ERiC software component has been updated to version 43.3.2.0 which supports test filings with taxonomy version 6.9.

IXBRL CREATION PLUGIN 10.1: Support V5 of Transformation Registry
The new Transformation Registry v5 is supported.

XBRL TEMPL CREATION PLG 10.1: xBRL-CSV DORA creation
It's now possible to report all templates with xBRL-CSV, even if the templates are empty. The filing indicators will be set to true for all templates and all table files will be created. This behavior can be disabled via the new report parameter "xBRL-CSV: Report all templates".

 
 

3. Fixes

All corrections included in previous DM Stacks are also available in this DM Stack.

Various additional fixes have been implemented for this DM Stack, including the following:

DM EXCEL TO PDF PLUGIN 10.1: SAP Analysis for Office
As a user, you rendered a Microsoft Excel workbook to PDF, but cells showed an error because SAP Analysis for Office formulas were used in the cells.

IXBRL CREATION PLUGIN 10.1: Package creation
As a user, you checked the iXBRL Package and read the version number written as "Final Draft" inside the taxonomyPackage.xml. As this might indicate that the package is not final, "Draft" has been removed from the version number.
