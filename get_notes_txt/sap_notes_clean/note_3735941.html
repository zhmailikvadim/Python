Symptom

You post a clearing document. It contains at least one realized exchange rate item generated for a freely-defined currency. Its balance transaction currency (ACDOCA-RTCUR) is the same as the transaction currency (ACDOCA-RWCUR), which is incorrect, as for the account on which the exchange rate item is posted, the indicator "Manage Balances in Local Currency Only" (SKB1-XSALH) is set.

This inconsistency can cause realignment run GRPL to generate unnecessary postings.

Other Terms

ACDOCA, RTCUR, PSWSL, KDF, SKB1, XSALH, GRPL

Reason and Prerequisites

Program error

Solution

Please implement the attached correction instruction in your system.
