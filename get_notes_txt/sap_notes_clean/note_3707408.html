Symptom

This Note resolves the runtime dump that occurs during the background queue processing job triggered via the CCMS node.

Other Terms

Defense Disconnected S4, CCMS,Queue Background Process,CALL_FUNCTION_REMOTE_ERROR.

Solution

Implement the attached correction instructions provided with the NOTE and Manual steps are required to restart the terminated node

1. Go to Transaction code DFS_CCMS_CREATE or Go to Transaction code SE38 and Execute the report DFS_CREATE_CCMS_NODES

2. Enter the logical system

3. Select the specific node( Queue Process) within the report

4. Execute.
