Symptom

Reference Opportunity created in Sales Cloud is missing in SD Document flow (VBFA) in S/4HANA.

Environment

  • Sales and Distribution (SD) 
  • SAP S/4HANA 
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

Create Quotation in SAP S/4HANA from Opportunity in SAP Sales Cloud via iFlow.

Cause

The table SDRO is containing the data, but the data is not used for displaying the document flow.

Resolution

There is no standard solution developed to integrate the Sales Cloud V2 application with the SD document flow in S/4HANA.

Keywords

opportunity, quotation, sales cloud, crm, iflow, document flow, vbfa, sdro, replication, cpi, API_SALES_QUOTATION_SRV, sales cloud v2, SDDocRelatedObjectReference1
