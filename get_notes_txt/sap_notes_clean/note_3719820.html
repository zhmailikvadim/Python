Symptom

When you create a new EQS PoD in transaction /US4G/EQS_POD_MON, the result log is not cleared and continues to append old results without displaying an error message.

Other Terms

US4G, EQS, /US4G/EQS_POD_MON, result log, PoD creation

Reason and Prerequisites

The error occurred due to a technical issue that prevented the system from clearing the result log when creating a new EQS PoD.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the system refreshes the log each time you create a new EQS PoD. To verify the fix, create a new EQS PoD and check that the log displays only the current results without appending previous entries.
