Symptom

  • The created outbound delivery cannot be found in the GTS system.
  • The preceding outbound delivery document is flagged as Advanced Shipping and Receiving (ASR) relevant.
  • Proforma invoices created in S/4HANA are not visible in the Global Trade Services (GTS) system.
  • The document type has been configured for GTS transfer.

Environment

  • SAP S/4HANA OP

Reproducing the Issue

  1. Use VF01 to create a proforma invoice in S/4HANA with a preceding outbound delivery flagged as ASR relevant.
  2. Attempt to save the document and transfer the proforma invoice to GTS.
  3. Observe that the proforma invoice is not transferred to GTS.

Cause

  • The outbound delivery document is flagged as ASR relevant, which triggers item clearance during the proforma invoice creation process.
  • The system logic prevents the transfer of documents to GTS for ASR-relevant outbound deliveries unless processed via TM in standard.

Resolution

  1. Verify the outbound delivery document type and ensure it is flagged as ASR relevant (field: tm_adv_ship_recv) in the LIKP table.
  2. If TM is not planned for use, please disable the ASR functionality for the outbound delivery.
