Symptom

Able to create a rescission order via the MAPI BCA_MAP_OR_RESC_CREATE_ACTIV without maintaining the payment advice, however this does not work via the UI. A warning message "Provide a Payment Advice for DR payment direction" appears via the UI not allowing the order to get executed.

Other Terms

Rescission, MAPI

Reason and Prerequisites

Program Error

Solution

Use transaction SNOTE to implement the correction instruction in advance. Implement the relevant support package.
