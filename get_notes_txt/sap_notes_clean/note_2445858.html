Symptom

  • You run Property Tax return or Advance Calculation with PDF output. In case OKVED code is less then 8 digits the corresponding field in the print form has empty cells. These cells should be filled with dashes instead.
  • You run Property/Transport Tax return or Property Advance Calculation with PDF output. In case OKVED code is maintained with dots in Company Code Global Parameters, in the print form it is output with additional dots.

Other Terms

Property Tax Declaration, J3RALFPTAXDECL, J_3R_PTAX_DECL, Transport Tax Declaration, J3RALFTTAXDECL, J_3R_TTAX_DECL, OKVED2

Reason and Prerequisites

 Program Error

Solution

Implement manual correction instructions attached to the note or install the appropriate support package.
