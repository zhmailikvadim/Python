Symptom

You want to read the master data of a production order via the Manage Production Orders App. The production version used in this order has a different storage location the order header.
Reading the master data does not take over the storage location from the production version.

Other Terms

F2336, AFPOD-LGORT, SLOC, V_T399X-FLG_READ, T399X-FLG_READ, V_T399X-PVSEL, T399X-PVSEL, COR7, CO40, MKAL_EXPAND-ALORT, MKAL-ALORT, C223

Reason and Prerequisites

Program error.

Solution

After implementing this SAP Note, reading the master data in the Manage Production Orders App should take over the storage location maintained in the production version.
