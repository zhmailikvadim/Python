Symptom

 
 
Scope delivered on 28.07.2015:
  • Improved creation of Waybill print forms – “GU-27”, “GU-27u-VC”;
  • Requirement  to have an additional Waybill print forms – “GU-27e”, “GU-29O”;

Scope delivered with Manual instructions on 28.07.2015:

  • Requirement to have a print forms for Waybill;
  • Requirement to have default customizing of status scheme;

Other Terms

Transportation Management, TM Russian Localization, Status Scheme, GU-12, GU-27, GU-27e, GU-27u-VC, GU-29O, Waybill, ETRAN

Reason and Prerequisites

The above mentioned symptoms appear in your system.

Solution

  • TMRU 100 SP01 must be installed.
  • Read and implement SAP Note 2197641.
  • Implement correction instructions using transaction SNOTE.
  • Implement Manual Post-Implementation Steps.

Alternatively you can install Support Package indicated when it will be available.

  • This SAP Note is relevant only for Russian Federation.

Product documentation

System documentation

The following documentation-relevant objects are delivered with this SP:

  • Customizing (IMG):
    In Customizing for SAP Transportation Management under the Transportation Management -> Localization Functions for Transportation Management -> Russia
  • Transactions/SAP Easy Access entry:
    Run ta: NWBC, select role /TMRU/RURAIL.
    The NWBC path:
    1. Planning -> TM Russian Localization
    2. Master Data -> TM Russian Localization

Note that the documentation is delivered with the corresponding Support Package only.
