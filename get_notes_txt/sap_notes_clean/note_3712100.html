Symptom

In OKP2, there's no change documents. 

Environment

  • Controlling (CO)
  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
    • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA
  • SAP S/4HANA Finance
  • SAP S/4HANA Cloud Private Edition
  • SAP Fiori for SAP S/4HANA

Reproducing the Issue

In OKP2, input CO area, and other parameters, there's no change documents. 

Cause

Code = EHDC (Change Documents) is NOT active

Resolution

Please activate Code = EHDC, by following steps: 

Change menu in GUI Status
1 Call transaction SE80.
2 Enter program 'SAPMKCSP'.
3 Open GUI Status 'LISA' and change the menu Bar -> Goto -> place cursor on the Code = EHDC by choosing 'Function Code' button on toolbar to activate the menu.
5 Activate above changes.

Keywords

OKP1, OKP2, change document, change log, greyed out, grey, menu
