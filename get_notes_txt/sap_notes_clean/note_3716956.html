Symptom

During the calculation of secondary balances, the runtime error CALL_METHOD_NOT_IMPLEMENTED occurs with regard to the method /IBS/IF_BSC_SD_ADD_BAL~CALCULATE_MASS.

Reason and Prerequisites

SAP Note 3664911 added the method CALCULATE_MASS to the interface /IBS/IF_BSC_SD_ADD_BAL for use in the secondary balance initialization class /IBS/CL_BSC_ADD_BAL_ABSUM. An implementation for the secondary balance calculation classes is missing for the method.

  • /IBS/CL_BSC_ADD_BAL_ABSUM_OBS
  • /IBS/CL_BSC_ADD_BAL_ABSUM_SIMU
  • /IBS/CL_BSC_ADD_BAL_FIBZ
  • /IBS/CL_BSC_ADD_BAL_PAY_ACQFEE
  • /IBS/CL_BSC_ADD_BAL_VGA

Solution

For the above mentioned Classes create an (empty) implementation of the interface method, where the method is declared obsolete. Their use in /IBS/CL_BSC_AB_ABSUM_BATCH->/IBS/IF_BSC_INIT_ADD_BAL_MASS~CALCULATE_MASS) is replaced by /IBS/IF_BSC_SD_ADD_BAL~CALCULATE.
