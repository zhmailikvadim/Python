Symptom

You are using "Functional Location Hierarchy – Read" API with pagination ("skip" parameter). You notice that the pagination is not working.

Other Terms

API, API_FUNCNLLOC_STRUCLIST, sap-vdm-sot-FunctionalLocation-v1, skip, OFFSET, paging, CL_EAM_FUNCNLLOC_HIER_QUERY, query, select

Reason and Prerequisites

The handler class of the query did not handle the skip parameter.

Solution

 Implement the attached correction instruction or the corresponding Support Package.
