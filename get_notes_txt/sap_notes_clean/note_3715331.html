Symptom

This SAP Note provides you with the Inflation Accounting objects required to implement the SAP Note 3241752.

Other Terms

 Asset Accounting, J_1AA001, J1AI, J_1AAINFL_REVERSE, J_1AAINFO_01

Reason and Prerequisites

This SAP Note is relevant only for Inflation Accounting.

Solution

After implementing this SAP Note, the system provides the required objects to implement the SAP Note 3241752.

SAP recommends that you implement a solution by using a Support Package. If you need to implement a solution earlier, use the Note Assistant to implement the correction instruction. You can find more information about the Note Assistant in the SAP Support Portal, under SAP Note Assistant.

To implement the solution, you need to perform manual activities and implement the correction instructions that are valid for your release.

Documentation for Inflation Management is available in the system and on the SAP Help Portal. System documentation (such as field help, customizing, and messages) is delivered in Support Packages. SAP Help Portal documentation is available at the following Links:

  • SAP S/4HANA: Inflation Accounting (AA)
  • SAP ERP: Inflation Accounting (AA)
