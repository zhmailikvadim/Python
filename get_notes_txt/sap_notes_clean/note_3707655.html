Symptom

    You would like to know, how to implement all the latest corrections that have been released for the S/4HANA Migration cockpit into your S/4HANA system for all the migration cockpit scenarios. 

    Environment

    SAP S/4HANA 2020 - SAP S/4HANA 2025 Private Cloud, onPremise

    Cause

    Required configuration.

    Resolution

    Step 1

    • Implement note 3016862 – Note Analyzers with separated scenarios for ABAP-based Migration and Replication Technology 
    • (Implementing the note involves Manual activity)

    Step 2

    1. Call Transaction SE38
    2. Enter: CNV_NOTE_ANALYZER_MC_EXT 
    3. Execute (F8)

    Step 3

    To implement every corrections for the Staging Table migration approach:

    1. Select only the check boxes related to Migration Approach -“Migrate using Data using staging tables”
    2. Click on Execute (F8)

    After the execution:

    • Click on the Refresh text.
    • Implement all of the listed corrections

    To Implement the corrections related to Migration approach – Direct Transfer Approach (SAP Note: 2973957):

    1. Select  the check boxes related to Migration Approach “Migrate Data Directly from SAP System”
    2. Select the appropriate Migration Scenario.
    3. Click on Execute (F8)

    After the execution:

    • Click on the Refresh text.
    • Implement all of the listed corrections

    To Implement the corrections related to Migration Cockpit Fiori Frontend

    1. Select only the check boxes related to Migration Cockpit Frontend execute.
    2. Click on Execute (F8)

    After the execution:

    • Click on the Refresh text.
    • Implement all of the listed corrections
