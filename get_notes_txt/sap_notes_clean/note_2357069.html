Symptom

This SAP Note delivers cumulation wage types for the basis of the welfare ("prévoyance") and the supplementary retirement (“retraite supplémentaire”) contributions.

Solution

This SAP Note delivers the changes to the standard customizing. These changes have to be adapted to the customer-specific customizing.

The enclosed correction instructions will be included in the HR Support Packages referenced below. If you need the correction before the HRSP is available, you may want to apply the correction instructions. It should be emphasized however, that correction instructions are only a short-term solution, not a substitute for the Support Package.
