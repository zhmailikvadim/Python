Symptom

The specific Currency （e.g. INR） is not displayed when you search the Currency from B&I side.
Additionally, when you submit the Requisition from  B&I side, that was created in this Currency from GB side,  it results the following error message on the currency field:

 "Currency INR  is not supported."

Environment

SAP Ariba Buying and Invoicing

Cause

The associated currency mappings data was deleted, which affects the search results of Currency data and the submission of PRs in that currency.

Resolution

To enable the Currency on the B&I side, you can import the associated Currency Mappings data using the Import Currency Mappings task. 
Once the specific Currency Mappings data is imported successfully, the Currency is expected to be available on the B&I side.

See Also
