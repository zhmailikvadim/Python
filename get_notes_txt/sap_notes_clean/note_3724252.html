Symptom

This SAP Note generates the required DDIC elements and/or other supplementary repository objects in your system before implementing SAP Note 3645474.

Other Terms

DDIC, UDO, Message of SAP Note 3645474, NOTE_3724252

Reason and Prerequisites

Non-Source Code Elements of SAP Note 3645474

Solution

This SAP Note generates the required Non-Source Code Elements in your system before implementing SAP Note 3645474

1) Implement the correction instructions of this SAP Note, via SNOTE.

2) Run the newly created report (NOTE_3724252) in update mode, so that all the Non-Source Code Elements of SAP Note 3645474 are created and activated.
