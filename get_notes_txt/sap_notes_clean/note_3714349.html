Symptom

The Supplier Response Report option is missing from the Download Report menu for multi-envelope RFP sourcing events after the first envelope is opened

Reproducing the Issue

  1. Log in as a user > Create a sourcing event with multiple envelopes (add questions and line items)
  2. Add a supplier > Publish the event
  3. Log in as supplier > Submit a bid
  4. Stop the event > Open the first envelope
  5. Go to More Actions (···) > Download Report > Observe that the Supplier Response Report is missing

    Cause

    Resolution

    This is expected system behavior, the Supplier Response Report cannot be downloaded until all envelopes have been opened.
