Symptom

Position Org Chart icon is not displayed against the Position field in Job Information when changes are made via employee profile under All Actions > Change Job and Compensation Information.

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental. 

Environment

SAP SuccessFactors Employee Central - Job Information

Reproducing the Issue

  1. Navigate to the employee profile.
  2. Select Actions > Change Job and Compensation Information.
  3. Make changes to the Position field in Job Information.
  4. Observe that the Position Org Chart icon is not displayed against the Position field.
  5. Please refer to the images below for the difference when displayed and now not displayed via All Actions.

Without Org Chart Icon

With Org Chart Icon

Resolution

  1. Product Engineering is targeting 2H 2026 release. In case fix date is changed this KBA will be updated.
  2. Click on the star to bookmark this article to receive updates about this issue. For more information about updates from subscribed KBAs, you can refer to the KB Article 2171560 - How to be notified of new or updated SAP Notes or KBAs.

Workaround:-

  • The Position Org Chart Icon is available via Edit UI (pencil icon) & History UI (Clock icon) beside the position field.
  • Also, when we use/enable the setting of Latest Editing and History UI under Configure Latest People Profile, the position org chart ison is available.
