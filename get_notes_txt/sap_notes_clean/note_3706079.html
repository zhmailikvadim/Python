Symptom

  • Users or administrators are unable to locate or download the IAS SAML Metadata file from the Identity Authentication Service (IAS) user interface.
  • There is uncertainty about where in the IAS UI the SAML metadata can be accessed or how to export it for configuring SAML trust with a service provider or identity provider.

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.  

Environment

SAP Sales Performance Management

  • Incentive Management
  • Incentive Management on Hyperscalers
  • Incentive Management on Legacy HANA 

Resolution

Following are the steps to find the IAS SAML metadata file in the IAS UI.

  1. Login to IAS as an admin user 
  2. Then click on "Applications & Resources" and then "Tenant settings".
        
  3. In "SAML 2.0 Configuration" the option to download the IAS Metadata is found on top right corner of this section.
      
      


  4.  "Download Metadata File" is the option to fetch the IAS metadata from the UI.
