Symptom

When product data is replicated using SOA inbound messaging, the MDG, Consolidation application is failing to create source data records if product contains material ledger data.

Other Terms

CMP-MM, Material Ledger, cost estimate number, KALN1, KALNR, MBEW, CKMLCR, CKMLPR, CKMLHD

Reason and Prerequisites

Support for temporary KALN1 generation was missing.

Solution

Implement correction instruction or apply relevant Support Package
