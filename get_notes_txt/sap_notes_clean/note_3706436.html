Symptom

In the statutory reporting application, the Channel field always defaults to Print for correspondence documents with the Unique Taxpayer Reference (UT) role.

Other Terms

SAP Document and Reporting Compliance, statutory reporting, S/4HANA, correspondence document

Reason and Prerequisites

Program error.

Solution

After implementing this SAP Note, the system correctly determines the channel for correspondence documents with the Unique Taxpayer Reference role. As a general rule, SAP recommends that you implement a solution by implementing a Support Package. However, if you need to implement a solution earlier, use the Note Assistant to run the correction instruction. You can find more information about the Note Assistant at service.sap.com/note-assistant.
