Symptom

When a new Workspace is created the background image is vertically repeating itself.

Environment

SAP SuccessFactors Work Zone

Reproducing the Issue

  1. Create a workspace in Work Zone Advanced edition.
  2. Edit the workspace and add a background image to a section.
  3. Observe that the background image repeats vertically when viewed in edit mode or publish mode.

Cause

Issue was related to a code error.

Resolution

The fix for this issue is planned to be deployed 19th of March.

For more details regarding the Work zone release schedule please check the next KBA  3303878.
