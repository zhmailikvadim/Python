Symptom

  • When using Mass Action to update a field in executive review, values containing special characters are not updated on the worksheet.
  • Mass Action fails to update values for entries that contain "&", while direct updates in the form work correctly.

Environment

  • SAP SuccessFactors Compensation
  • Executive Review
  • Variable Pay

Reproducing the Issue

  1. Open the template in Executive Review.
  2. Start Mass Action on a string field whose valid values include labels containing "&".
  3. Choose a value that contains "&" and attempt to apply the update.
  4. Observe that the selected value is not saved via Mass Action, although it saves when edited directly in the form.

Cause

Mass Action handling of special characters is limited; values containing special characters such as "&" are not supported for update and this is expected behavior.

Resolution

  1. Avoid using special characters (for example, "&") in value labels intended to be updated via Mass Action.
  2. If values with special characters are required, use the offline workaround:
  3. Export the Executive Review worksheet.
  4. Manually update the affected values in the exported file.
  5. Reimport the updated file back into Executive Review.
  6. For a long-term improvement, submit an enhancement request for future support of special characters in Mass Action via the customer influence channel.
