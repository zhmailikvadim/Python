Symptom

When trying to Import projects into Information Steward, the Import Options tab is blank and hitting "Next" to continue the process is also unresponsive.

This issue has been already identified in Note 3344916, and has been reportedly fixed in SAP Information Steward 4.3 SP02 Patch 3, however can be still observed and reproduced in SAP Information Steward SP03 Patch 3

Reason and Prerequisites

SAP Data Services 4.3

SAP Information Steward SP03 Patch 4

Solution

 This issue has been forwarded to Development for further investigation.
