Symptom

Error Code: 400 in Latest People Profile: "Could not find acceptable representation. Supported Media Types are: [application/json]"

Environment

SAP SuccessFactors Employee Central

Reproducing the Issue

  1. Proxy as user.
  2. Click on "Finish Your Profile" on Home Page.
  3. Error appears on Personal info card and other cards: "Error Code: 400 Could not find acceptable representation. Supported Media Types are: [application/json]".
  4. If navigating directly to People Profile, no error occurs.

Cause

"Finish Your Profile" to-do task is not supported in PPX. These to-do tasks were generated before enabling PPX.

Resolution

  1. Navigate to Admin Center > Manage Home Page > Manage Card Settings.
  2. Disable the "Finish Your Profile" card.
  3. Go to Scheduled Job Manager and create a job with the type "Delete Cards from Database" to clean existing card data from the database.
    (Please note The "Finish Your Profile" option will appear after a short wait, once you have completed the first step.)

    Note: In 1H2026 release, there will be an item in Check Tool to help admins to detect the invalid or redundant data of Finish Your Profile to be fixed/removed.
