Symptom

You are migrating your data into SAP S/4HANA 2023 using the SAP S/4HANA migration cockpit with the "Service Contract OP 2023" migration object and getting duplicate service contracts.

This SAP Note is valid for:

  • SAP S/4HANA - Migrate Your Data Migration Cockpit - "Transferring Data Using Staging Tables"

Other Terms

Service Contract Migration, SIF_SRVC_CONTR_X, Duplicate Contract, LTMC, Migration Cockpit, Service Contract

Solution

The Transport-based Correction Instruction (TCI) included with this SAP Note will introduce a fix for service contract migration object listed in the table below.For detailed description on the issues see the linked SAP Notes.

Technical Name Type Issue SAP Note 

SIF_SRVC_CONTR_X

Migration Object

Service Contract Migration Cockpit - Duplicate Service Contracts

3728368

Be aware that the TCI only corrects the related objects of SAP delivered content. As a result, the generated migration objects will be updated automatically.

Note: If you have modified/copied your object, the correction is not done within your modified/copied object.

You can refer to the following SAP Knowledge Base Article (KBA): 2543372 - "How to Implement a Transport-Based Correction Instruction.
