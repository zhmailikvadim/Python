Symptom

What options exist for enhancing My Inbox - Expert Mode?

Environment

  • ABAP Platform
  • My Inbox

Resolution

  • There are currently no extensions available for My Inbox - Expert Mode.
  • The following extension points are available for regular My Inbox:
    • cross.fnd.fiori.inbox.view.Forward~extHookChangeListSizeLimit = Allows you to change the number of recipients displayed on the Forward screen
    • cross.fnd.fiori.inbox.view.S2~extHookChangeSortConfig = Allows you to change those properties that can be used for sorting the workflow tasks
    • cross.fnd.fiori.inbox.view.S2~extHookChangeGroupConfig = Allows you to change those properties that can be used for grouping the workflow tasks
    • cross.fnd.fiori.inbox.view.S2~extHookChangeFilterItems = Allows you to replace the standard filters with custom filters based on the filter key
    • cross.fnd.fiori.inbox.view.S2~extHookChangeMassApprovalButtons = Allows you to add, remove, and change the action buttons on the list screen in mass action mode
    • cross.fnd.fiori.inbox.view.S2~extHookGetCustomFilter = Allows you to create custom filters that can be used for filtering the workflow tasks
    • cross.fnd.fiori.inbox.view.S3~extHookOnDataLoaded = Allows you to perform additional requests and process the retrieved data
    • cross.fnd.fiori.inbox.view.S3~extHookGetEntitySetsToExpand = Allows you to retrieve additional entities from the back-end system
    • cross.fnd.fiori.inbox.view.S3~extHookChangeFooterButtons = Allows you to add, remove, and change the action buttons (for example, claim, release, and forward) on the detail screen
    • cross.fnd.fiori.inbox.view.ViewSubstitution~extHookAddFooterButtonsForSubs = Allows you to add buttons for the substitution screen
    •  S2.view.CustomerExtensionForObjectListItem = Allows you to replace the list item template with your custom list item template on the list screen. (\n\)Make sure that the ID of your custom list template is the same as the ID of the standard list item template.
    • S3.view.CustomerExtensionForObjectHeader = Allows you to replace the header of the detail screen with your custom header
    • S3.view.CustomerExtensionForInfoTabContent = Allows you to define the content you want to display on the Info tab
    • S3.view.CustomerExtensionForNoteTabContent = Allows you to define the content you want to display on the Note tab
    • S3.view.CustomerExtensionForAttachmentTabContent = Allows you to define the content you want to display on the Attachment tab
    • S3.view.CustomerExtensionForObjectLinkTabContent = Allows you to define the content you want to display on the Object Link tab
    • S3.view.CustomerExtensionForHistoryTabContent = Allows you to display the content you want to display on the History tab
    • S3.view.CustomerExtensionForAdditionalTabs = Allows you to create additional tabs
    • S3.view.CustomerExtensionForAdditionalDetails = Allows you to display additional information on the detail screen
    • TaskUI_S3.view.CustomerExtensionForNoteTabContent = Allows you to define the content you want to display on the Note tab
    • ViewSubstitution.view.CustomerExtensionForSubstitutionRowHeader = Allows you to add columns in the substitution table
    • ViewSubstitution.view.CustomerExtensionForSubstitutionRowData = Allows you to add column list items in the substitution table
    • ViewSubstitution.view.CustomerExtensionForSubstitutionAdditionalDetails = Allows you to display additional information in the substitution table
  • These are all list view, detail and substitution view extensions and hooks only
  • Detail view extensions and hooks will affect the detail view in My Inbox - Expert Mode as well
