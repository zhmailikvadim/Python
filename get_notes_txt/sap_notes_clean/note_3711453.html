Symptom

It is needed to know how to retrieve service child line accounting data from table(s). 

Environment

  • Materials Management (MM)
  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA

Resolution

Sl. No.

Table

Description

Key Fields

Comments

1

EKPO

Purchasing Document Item

EBELN, EBELP

Get the PACKNO

2

ESLL

Lines of Service Package

PACKNO, INTROW

Input the EKPO-PACKNO to ESLL-PACKNO to get all the SUB_PACKNO. Then input same in ESLL-PACKNO and get the outputs.

3

ESKL

Account Assignment Specification: Service Line

PACKNO, INTROW, NUMKN
  1. Input the ESLL-PACKNO to ESKL-PACKNO
  2. ESKL will hold only the quantity, percentage, but not the GL account details. ESKL-ZEKKN should be compared with EKKN-ZEKKN

4

EKKN

Account Assignment in Purchasing Document

EBELN, EBELP, ZEKKN

Get the outputs
