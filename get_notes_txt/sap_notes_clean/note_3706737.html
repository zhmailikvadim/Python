Symptom

  • When clicking on a post in the SAP Jam mobile application, the cell phone navigation bar prevents the option to add comments and likes to the post.
  • The issue occurs specifically with Android devices using a 3-button navigation system.
  • The issue does not persist when gesture navigation is used.

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental." 



Environment

SAP Jam Mobile

Reproducing the Issue

  1. Login to SAP Jam mobile application.
  2. Click on a post containing a video or image.
  3. Observe that the Like and Comment section is not accessible as it is hidden behind the navigation bar. 

Resolution

WORKAROUND: Use a Gesture navigation system instead of the 3-button navigation system. 

This was discussed with the Product Management team for JAM Mobile and for now, this behavior would not be addressed as there is a current workaround and considering that JAM (and JAM Mobile) is already on the process of being sunset by January 2027, there would be no changes or improvement on this anymore to accommodate devices that are using the 3-button navigation bar.
