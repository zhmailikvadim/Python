Symptom

While trying to archive parked or recurring documents using the object FI_DOCUMNT via transaction code SARA, the error "Recurring entry, reference or parked document" occurs.

Environment

  • Financial Accounting
  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA Finance
  • SAP S/4HANA

Reproducing the Issue

Follow the below steps:

  1. Go to transaction code SARA.
  2. Enter the archiving object FI_DOCUMNT.
  3. Execute with relevant information.

Cause

This is a standard SAP system limitation related to parked and recurring documents.

Resolution

As per standard SAP functionality, parked and recurring documents are not supported for archiving through the FI_DOCUMNT archiving object.
