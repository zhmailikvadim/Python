Symptom

After the Employee reports a work/life event, the exception is not being created as expected, even though the configuration is correct. 

Environment

SAP SuccessFactors Employee Central

Reproducing the Issue

  1. Log in as the employee;
  2. Report the event';
  3. Proxy as the approver;
  4. Approve the workflow;
  5. See the rule trace setting the exception as expected;
  6. No exception is created on the system.  

Cause

Incomplete configuration for Benefit Work/Life Event Configuration or missing permission to the "Benefits Email Configuration" object. 

Resolution

Make sure the "Benefits Email Configuration" object is created for the relevant Legal Entity and that the workflow approver has permission to the object.

For automatic exceptions, set Enable Email Notification to Yes in the Benefit Work/Life Event Configuration.
