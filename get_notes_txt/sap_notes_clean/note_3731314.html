Symptom

The CDS view I_BillingDocumentPrcgElmnt returns no data from a Preliminary Billing Document for all users when executed in the view browser.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Open the View Browser in CDS.
  2. Search for and open the CDS view I_BillingDocumentPrcgElmnt.
  3. Run the view without additional filters.
  4. Observe that no records are returned.

Cause

Standard system design.

Resolution

The CDS view I_BillingDocumentPrcgElmnt does not return data from PBDs, only for the SD document categories:

Description

SD Document Category

Invoice

M

Invoice Canceled

N

Credit Memo

O

Debit Memo

P

Credit Memo Canceled

S

Pro Forma Invoice

U

Intercompany Invoice

5

Intercompany Credit Memo

6

Down Payment Request (for advanced down payment processing)

DPRQ

Down Payment Request Cancellation (for advanced down payment processing)

DPCC

If you want to retrieve data from Preliminary Billing Documents, Billing Document Requests or Invoice lists you might use the Billing Process Document: I_BillingDocumentRequest
