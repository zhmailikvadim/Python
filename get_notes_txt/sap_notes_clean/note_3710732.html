Symptom

In the EWM system, the location field within PMR documents is not editable, and the option appears greyed out.

Environment

  • SAP Extended Warehouse Management
  • SAP S/4HANA

Reproducing the Issue

1. Navigate to the transaction /SCWM/PMR.
2. Enter the Warehouse Number. 
3. Enter PMR Number. 
4. Switch to edit mode and input the Goods Movement (GM) bin for the specific item associated with the particular order. 
5. It is greyed out. 

Cause

Manual entry of GM bin values into PMR items is not permitted. The GM bin value is automatically determined by /SCWM/PSASTAGE configuration settings.

Resolution

  1. It is not permitted to manually enter any GM bin value into a PMR item.
  2. The GM bin value is automatically determined based on the /SCWM/PSASTAGE configuration settings.
    1. If the GM bin does not show the correct value, review the configuration settings.
    2. Adjust the configuration to ensure the GM bin reflects the correct value.
  3. Close any incorrectly created PMRs using the TECO process.
  4. After updating the configuration, create a new PMR using the same manufacturing order.
  5. For detailed guidance on closing PMR documents, refer to the KBA: 3493909 - How to close PMR documents

See Also
