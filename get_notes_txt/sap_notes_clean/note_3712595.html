Symptom

You create the electronic wage statement using the report Notifications Submission of Electronic Create Wage Statement (RPCUVBD0_OUT). The AI contribution accounting (PDF) is created using the split form creation (see SAP Note 2511676). During the transfer to the spool, several print parameters (for example, recipient and department) are not transferred.

Solution

Import the relevant Support Package or implement the attached correction instructions. In addition to implementing the automatic correction instructions, you do not have to perform any manual tasks.
