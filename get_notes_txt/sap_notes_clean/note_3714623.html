Symptom

Component: Product Safety
Module: Global Label Management

You run transaction CBGL_CD03 to display the label data change documents from the material master. You notice that some change documents show an empty material number.

Other Terms

GLM, GLM+, GLMplus, CBGL_LB20

Reason and Prerequisites

Reason
A program error causes the system to create change documents without a material number.

Prerequisites
For information about the validity of the corrections, see the section Correction Instructions.

Solution

After implementing this SAP Note, the system records and displays the material number correctly in new change documents. This change affects only change documents created after you implement the SAP Note.

The section Support Package specifies the Support Packages that contain the correction.
Alternatively, you can implement the attached correction instructions.
