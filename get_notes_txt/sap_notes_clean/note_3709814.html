Symptom

Reason and Prerequisites

Issue number 348911

Reason:

Due to a programming error in the ABAP ODBC driver, an inconsistent synchronization (mutual exclusion) state may occur.

Affected Releases:

  • SAP HANA 2:
    • Revisions of SPS08
    • or higher

Prerequisites:

  • Smart Data Access to ABAP using ABAP ODBC adapter versions prior to 100.23.23

Solution

For SAP HANA 2 apply one of the following ABAP ODBC driver version:

  • ABAP ODBC adapter 100.23.23
  • or higher
