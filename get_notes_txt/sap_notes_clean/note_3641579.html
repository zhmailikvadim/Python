Symptom

A user group should have access to exporting the dictionary, but not the right to import a dictionary.

Resolution

Create a user group (if not already created):

    • Navigate to SAP Signavio Process Manager.
    • Click on 'Setup' in the top-right corner.
    • Select 'Manage users & access rights' from the drop-down menu.
    • In the pop-up window, go to the 'User groups' tab and click the green plus icon to create a new user group.
    • Switch to the 'Users' tab to add users to the newly created group.

Configure Access for the Group:

    • Return to the 'User groups' tab.
    • Select the newly created group.
    • On the right side, check the option 'General - Dictionary Import/Export' to enable export and import functionalities for the group.

Limit Import Functionality:

    • Navigate to the 'Access rights' tab.
    • Select 'Dictionary' in the panel on the right side.
    • Choose the user group and assign it 'View' access only.

With these configurations, the users can execute dictionary exports but will be unable to proceed with imports.
