Symptom

Error message "M8140 : Acct assgt 99 for order item & & does not exist" is generated while reversing the invoice using MR8M for an Enhanced limit item with Multiple account assignments. This happens when an unplanned account assignment is added in the invoice.

Other Terms

MR8M, M8 140, ME_READ_ITEM_INVOICE, xek08r_aufbauen_lean_services, Lean Service, LEINRFS1

Reason and Prerequisites

 This is due to program error.

Solution

Implement the attached correction instruction.

In addition to this correction, implement the correction instructions for note 3717791.
