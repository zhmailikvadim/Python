Symptom

Field Fixed Amount In Global Currency (KFSL) is not updated if plan data records only contain Amount In Global Currency (KSL).

Other Terms

 Program R_FINS_PLAN_TRANS_CO_ERP_2_S4H

Reason and Prerequisites

Field Fixed Amount In Global Currency (KFSL) is not in the aggregation level of the write API. Correction instruction 0020751258 0000853443 of note 0002968212 has to be implemented before implementing attached correction instruction

Solution

Update to this support package or implement attached correction instruction. Note 2968212 is prerequisite for correction instruction.
