Symptom

When an end-user attempts to approve a Catalog Approvable document, the approval action fails with the following error message:

  • "On Behalf Of has been deleted"

The document remains stuck in a pending approval state and cannot be progressed further through the workflow.

Cause

The error occurs when the system attempts to resolve the "On Behalf Of" or "Requester" user reference associated with the SDU or Catalog Approvable document during the approval process. If the referenced user has been deleted or deactivated from the system, the approval logic fails to validate, resulting in this error.

Resolution

This issue can be resolved by enabling the site-level parameter Application.Catalog.EnableSDUFixCM11317. This parameter applies a targeted correction to the SDU/Catalog approval process so that the system handles this scenario properly. When enabled, it retrieves the Third-Party User Password Adapter based on the UserID, ensuring that the correct Requester is identified and not the deactivated User object.

Please note that CMS must be enabled for this parameter to work, and the change must be implemented by SAP Support.
