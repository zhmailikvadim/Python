Symptom

This SAP Note generates required DDIC elements and/or other supplementary repository
objects in your system before implementing SAP Note 3709988.

Version 1: Original version

Other Terms

SAP Note 3709988, RPUUDOH0_3713537

Reason and Prerequisites

DDIC elements and/or supplementary repository objects of SAP Note 3709988

Solution

Implementation:

Implement the Correction Instructions of this SAP Note.

Log on to the system and run the newly created report in Update mode: RPUUDOH0_3713537
Depending on the SAP_BASIS release and SP level, SNOTE may handle the objects which the report tries to update.
If so, running the report will display such objects as "Covered by SNOTE" and skip the object. In this case, SNOTE
updates the relevant object. Make sure that the created objects are all active.
