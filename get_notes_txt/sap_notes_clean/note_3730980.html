Symptom

You are using Fiori app Import Foreign Exchange Rates (F2092). The system is configured that you do not have the authorization to upload exchange rate for certain currency pair or exchange rate type. When you tried to import the exchange rate for this currency pair and exchange rate type, instead of error message of no authorization, a warning message of "Entry with the same value already exists." or "Entry with a different value &1 already exists." is wrongly raised.

Other Terms

F_CUREXRAT, CHECK_AUTH_CREATE_EX_RATE, CL_MDIF_MD_PROCESSOR

Reason and Prerequisites

Programming error.

Solution

Please implement the SAP Note or the Support Package.
