Symptom

Incorrect information in SAP S/4HANA Cloud Public Edition 2602 product assistance for Collections and Dispute Automation:

Location:

SAP S/4HANA Cloud Public Edition -> Receivables Management -> Collections and Dispute Automation -> Business Transactions -> Collection of Receivables -> Collections and Dispute Automation

For SAP S/4HANA Cloud Public Edition 2602:

Contrary to the information provided in the online documentation for SAP S/4HANA Cloud Public Edition on the SAP Help Portal, the following chapters are not valid and have been corrected in the attached .pdf file:

  • Collections and Dispute Automation -> Business Transactions -> Collection of Receivables -> Process Worklist in Collections and Dispute Automation:
    The information on key features and app variants is not valid and has been corrected in the attached .pdf file.
  • Collections and Dispute Automation -> Business Transactions -> Collection of Receivables -> Process Worklist in Collections and Dispute Automation -> Changing Collections Agent Groups on the Worklist:
    The chapter is invalid.

For SAP S/4HANA Cloud Public Edition 2602.1 release, the online documentation was corrected:

  • Process Worklist in Collections and Dispute Automation

Incorrect system documentation of IAM App:

  • Manage Worklist in Collections and Dispute Automation (F8832_02_TRAN)
    This app authorization variant is currently inactive. The system documentation of this app authorization variant is invalid.

Solution

The above mentioned SAP Help Portal documentation has been corrected in the attached .pdf file and will be published on the SAP Help Portal as of release SAP S/4HANA Cloud Public Edition 2602.1.
