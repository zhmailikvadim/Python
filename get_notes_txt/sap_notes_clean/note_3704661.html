Symptom

When trying to execute transaction TPM18 error message T0605 is raised for example "Cross rate IDR/CNH for rate type M or M is too great or too small". 

Environment

  • Treasury and Risk Management (TRM)
  • SAP ERP
  • SAP ERP Central Component
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP Financial Supply Chain Management
  • SAP S/4HANA
  • SAP S/4HANA Finance
  • SAP S/4HANA Cloud Private Edition

Cause

Programming error

Resolution

Implement SAP Note: 3665390 - Incorrect FX Spot rate is used in NPV calculations - II

Keywords

T0605, TPM18
