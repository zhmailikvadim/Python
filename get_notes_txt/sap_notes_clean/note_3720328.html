Symptom

Any situation where user wants to create negative points Loyalty points.

E.g.

  • Loyalty points are assigned to users upon purchasing products.
  • Users redeem loyalty points into vouchers.
  • Users return purchased products after redeeming loyalty points into vouchers.
  • SAP CRM Loyalty Management does not allow assigning negative points after voucher redemption.

Environment

Customer Relationship Management 7.0

Resolution

This is the system expected behavior.

System does not allow the creation of negative Loyalty points. If a there is a Negative Loyalty point, the system will raise an error message.
