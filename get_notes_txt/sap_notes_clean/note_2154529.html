Symptom

- Text alignment from a Crystal Report document is not properly displayed when viewed in SAP BI LaunchPad,
- No problem when viewing the document directly within Crystal Designer 2013 or Crystal Report for Enterprise.

Other Terms

CR4E, CR

Reason and Prerequisites

Steps to duplicate

  1. Log in to SAP Crystal Report 2013 or SAP Crystal Report for Enterprise,
  2. View a document containing many words spread on several lines,
  3. Make sure to select "Justify" for the text alignment of the content,
  4. Notice that the report is properly displayed.
  5. Log into SAP BI LaunchPad,
  6. Go to Preferences > Crystal Reports > and test the behavior with the Web view format,
  7. Notice that the text is not justified along the right edge.

Cause

When using the value "justify" for the "text-align" property, a "Display:Block" is added in the span html tags. Then, the first word on the line is properly along the left edge but the last word is not along the right edge.

Solution

The issue has been fixed in the patches listed in the "Support Packages & Patches" section below.
The "Support Packages & Patches" section will be populated with the relevant patch levels once they are released.
