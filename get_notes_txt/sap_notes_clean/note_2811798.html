Symptom

When trying to create a new content in a group, the Link option is not available in the dropdown.

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

SAP Jam Collaboration

Reproducing the Issue

  1. Access the group
  2. Go to Content section
  3. Click in the Create button
  4. Link option is not listed

Cause

Enable File Sharing feature is not enabled in Jam Admin console.

Resolution

To be able to create link contents in the groups of Jam, you need to enable the file sharing. The feature is located at:

  1. Jam Admin Console
  2. Product Setup > Features
  3. Feature Management > Enable File Sharing
