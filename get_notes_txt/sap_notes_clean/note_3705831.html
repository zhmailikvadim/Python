Symptom

  • We have inactivated a Supplier, but we notice the Supplier is still able to create Profile Workers (PWs)
  • Are there any additional steps that need to be taken to ensure inactive suppliers cannot create a Profile Worker in Fieldglass?

Cause

When a Supplier is inactivated with a date range, the system will still allow them to create PWs

Resolution

You need to inactivate the Supplier without setting a date range. Despite the create function being available, the Supplier will see 'No items found' when selecting a Buyer even if they try to create a PW.

For Suppliers that have been inactivated with a date range, please reactivate and inactivate them again without setting a date range.

You can use the Activate or Inactivate Supplier Upload | SAP Help Portal connector if you need to mass update a large number of Suppliers.
