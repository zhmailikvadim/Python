Symptom

Following the implementation of the SAP Note 3434901, you experience a significant performance degradation when retrieving item sales texts via the Sales APIs.

Other Terms

CL_SD_S4H_STXL_UTILS, STXH, RV_BELEG, NUMBER_GET_NEXT, API_SALES_ORDER_SRV, API_CUSTOMER_RETURN_SRV, P_SALESDOCUMENTITEMTEXT, A_CreditmemoreqItemText, A_CustomerreturnItemText, A_CustProjSlsOrdItemText, A_DebitMemoReqItemText, A_SalesContractItemText, A_SalesOrderItemText, A_SalesQuotationItemText, A_SalesSchedgAgrmtItemText, A_SlsOrdWthoutChrgItmText, External Document Range.

Reason and Prerequisites

Due to the massive volume of records (billions) in STXH, using RTRIM to filter sales document numbers causes performance issues.

Solution

 Implement the attached correction instructions or import the specified support package.
