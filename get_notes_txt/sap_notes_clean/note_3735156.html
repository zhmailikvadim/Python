Symptom

Is it possible to include additional attachments to an existing invoice via enrichment API?

Resolution

The enrichment API does not allow including additional attachments to an existing invoice. We recommend to submit an enhancement request about this functionality.

As a workaround, you can follow the steps below to add more attachments to the invoice:

  1. Go to Manage Supplier Invoices Centrally app
  2. Edit the invoice
  3. You can add, delete, or edit the attachments that have the document type Legal Document or Other attachments
  4. Refer Handling of Multiple Attachments on the Supplier Invoice Detail Page and in the My Inbox App | SAP Help Portal
