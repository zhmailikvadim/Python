Symptom

  • In a CDS provider view (CDS view defined with @Analytics.dataCategory: #CUBE or #DIMENSION), field A is annotated with @Semantics.*. It is generated with technical InfoObject as base characteristic (0* InfoObject).
  • In transaction RSRTS_ODP_DIS, this information is not available

Other Terms

technical content, 0currency, 0unit, standard InfoObject, SAP delivered InfoObject

Reason and Prerequisites

After this note, the reference technical InfoObject name will be displayed for a field in the CDS provider view. If the InfoObject is not installed or activated correctly, a red light will be shown. Please correct it (See Note 2636754).

Solution

To solve the issue please apply the Support Package as mentioned below. In urgent cases you can use the correction instructions.
