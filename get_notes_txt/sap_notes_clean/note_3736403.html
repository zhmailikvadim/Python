Symptom

How to specify Receiving Types for Imported Receipts?

Resolution

The <urn:IsReceivingByCount> tag from the ExternalReceiptImportRequest payload specifies if an imported receipt created in Ariba is classified as a quantity based or an amount based receipt. 

  1. If the tag <urn:IsReceivingByCount>false</urn:IsReceivingByCount> then the receipt is classified as an amount based receipt. 
  2. If the tag <urn:IsReceivingByCount>true</urn:IsReceivingByCount> then the receipt is classified as a quantity based receipt.
