Symptom

1. Incorrect or Missing Settled Value and Net Posting Amount in Display Service Settlement Hierarchy Fiori Application.
In the Display Service Settlement Hierarchy application, settlement-related financial values may not be displayed correctly. Specifically, the Settled Value and Net Posting Amount fields may appear as blank, incomplete, or incorrect for individual service documents or hierarchy nodes, even though settlement has been successfully executed and corresponding accounting postings exist in the backend.

2. In Joule, While triggering the settlement run for service documents which are not relevant or settled value is zero, the response structure still displays the settled value as zero. However, no corresponding informational or warning message is returned in response.

3. In Joule, When the settlement is successfully executed for a specific Period and Fiscal Year, all relevant settlement details are displayed correctly. However, the Period and Fiscal Year fields in the response card remain blank.

4. In Joule, Settlement Execution or Service Hierarchy Retrieval fails in Joule for Maintenance Document IDs ( Length 12 Characters ) with Malformed Syntax Error

5. In Joule, after successful execution of the settlement, the system displays the transactional amount and currency instead of the controlling area amount and currency.

Other Terms

Display Service Settlement Hierarchy, Joule for Service Settlement Capability, Settlement Run

Reason and Prerequisites

 Program Error

Solution

 Implement the note.
