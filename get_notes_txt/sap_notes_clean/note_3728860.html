Symptom

  • After performing an upgrade to S/4HANA no dividends are created for stocks in case of a short position.

Environment

  • SAP S/4HANA
  • SAP S/4HANA Finance

Cause

Configuration

Resolution

  • Before the upgrade to S/4HANA this functionality was only possible by activating the modification as described in SAP Note: 1633648 - Securities: negative interest and repayments. 
  • If the cashflow generation within the conditions tab in the master data in FWZZ is set as "Parallel Conditions" no modification is necessary anymore.
  • Kindly note in case the cashflow calculation is set as "Single Condition" even in S/4HANA the modification in SAP Note: 1633648 - Securities: negative interest and repayments is still required. 

See Also

1633648 - Securities: negative interest and repayments

Keywords

short position, stock, dividends
