Symptom

Planning material fields MPGD-PRGRP, MPGD-PRWRK and MPGD-UMREF are hidden on the MRP3 material master view

Environment

  • SAP ERP Logistics (LO)
  • SAP R/3
  • SAP R/3 Enterprise
  • SAP ERP Central Component
  • SAP ERP
  • SAP Enhancement package for SAP ERP
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA, on-premise
  • SAP S/4HANA, cloud edition

Resolution

You need to switch on "Planning material" in customizing transaction OMT0 to display the Planning material fields

If this indicator is not set, the system hides the fields

Keywords

KZPRG, V_130S, SAPLMGD1, 2492, MRP
