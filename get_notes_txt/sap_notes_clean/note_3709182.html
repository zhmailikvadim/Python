Symptom

The ABAP Short Dumps of a Managed System Instance are being monitored in the System Monitoring application using the "Short Dumps" metric group.
There is a need to understand how to monitor specific Short Dumps.

Environment

SAP Focused Run

Cause

Resolution

  1. First, it is necessary to ensure that a custom monitoring template exists, as the default template cannot be changed.
    • If a custom template does not yet exist, please create one by following the steps outlined in KBA 3414261.
  2. To define custom filters, please add a new variant to the metric group "Short Dumps".
    • Then, it is possible to change the values of the Managed System Client and the ERROR_ID of the Short Dump.
      • The ERROR_ID field corresponds to the name of the runtime error in the ST22 transaction.
  3. After creating the required variants, save the modifications and reconfigure the managed system.
  4. The metric variants should now be visible in System Monitoring.
