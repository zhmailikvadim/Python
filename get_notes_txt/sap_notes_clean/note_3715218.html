Symptom

You want to install version 01.00.0027 of the ODBC driver for ABAP.

The new version fixes the following program errors:

  • A locking issue in the RFC library
 

Other Terms

ODBC, SQL Service, ABAP SQL Service, ABAP SQL External, HANA SDA, DBC2ABAP

Solution

This version is initially delivered with the patch level listed in the "Support Package Patches" section of this SAP Note. All subsequent versions also contain the corrections and improvements. Keep in mind that only the latest version may be available for download. Please see SAP Note 3076454 for information regarding to the driver configuration.
