Symptom

Error message FCLM_BT_MSG071 is displayed in Fiori App Make Bank Transfer:

'The function is supported only with the full cash management scope'.

But you have already selected "FULL SCOPE" in SPRO-->Financial Supply Chain Management-->Cash and Liquidity Management-->General Settings-->Define Basic Settings--->Cash Scope.

Environment

S/4 HANA On Premise or Private Cloud

Reproducing the Issue

Run Make Bank Transfers app.

Cause

Business Function FIN_FSCM_CLM is not activated.

If Activation Status is off, then system will consider it as Basic Cash Scope.

Resolution

Activated Business Function FIN_FSCM_CLM with transaction SFW5.
