Symptom

How are documents filtered under the Your Requests tab in Guided Buying (GB)? 

Reproducing the Issue

Login as a user who has created a few documents as the preparer. Go to the Your Requests tab in Guided Buying. The default filter is set to show the requests updated in the last 3 months. 

Resolution

Guided Buying displays the documents updated (not created) in the last three months. For example, if a document is created 6 months ago but updated a week back, then the same will show up on the Your Requests tab. 

See Also

The Network tab can be used under the inspect element option in the browser to view the query GB uses to display documents. Please follow the below steps to view the results. 

  1. Right click on the browser and click on Inspect element on the GB home page. 
  2. Switch to the Network tab.
  3. Now click on the Your Requests tab. 
  4. You can see the API requests being printed on the network tab of the browser. 
  5. Select the relevant result (FetchRequests?ApprovableSearchType). Switch to the Payload tab and it will show you the query string parameter. 
  6. Go to the Preview tab which shows the queries that are run.
  7. Go to the Response tab and scroll down to see the search parameters. It will print out the PR numbers showing in the results. You will also see the TimeUpdated as one of the filter elements.
