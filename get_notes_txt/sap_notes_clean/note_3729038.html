Symptom

We can authenticate into Signavio over IAS, but the groups are not assigned automatically. How to make it work?

Resolution

  1. To provide the groups into Signavio you need to use the parameter: signavio_groups_v1, which you can configure in the IAS -> Applications -> Attributes.
  2. The group in Signavio needs to have the same name as in IAS. The value "Name" from IAS will be used in the integration (not "display name"). This group must already exist in Signavio.
  3. The group needs to be assigned to an application. If you create or edit the IAS group, in the second step you need to select Signavio application. Otherwise, it will be not sent with the login.
