Symptom

When using IOS device to access Latest Candidate Experience and using Chinese stroke & Chinese handwriting to tab the words in text fields, the cursor “|” flashing is sometimes missing and input is interrupted.

Environment

SAP SuccessFactors Recruiting Management

Reproducing the Issue

  1. Use IOS device to access Latest Candidate Experience.
  2. In any text field, use Chinese Stroke or Chinese Handwriting input methods to type characters.
  3. Observe that the cursor flashing "|" disappears easily when typing multiple strokes quickly, causing the input to become unresponsive.

Resolution

The fix for this issue is planned to be deployed on 1H 2026 release.

For release timelines, please review SAP SuccessFactors Product Release & Road Map Information.
