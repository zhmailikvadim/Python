Symptom

In the My Permits (F5192) or Manage Compliance Requirements (F5038) apps, the system doesn't allow you to change the status of a permit revision. When you try to change it, the system produces an error saying that it's an "invalid status" for the permit lifecycle.

This error affects releases starting from SAP S/4HANA 2021, for compliance requirements that were created before that release. This is because the "Permit Lifecycle" field was introduced in SAP S/4HANA 2021, but the related migration object doesn't have a set default value for that field.

This SAP Note contains manual instructions for creating a report that fills this missing "Permit Lifecycle" value, eliminating the error.

Other Terms

Missing permit lifecycle value

Reason and Prerequisites

Reason: Migration Error. There is no default value set for the "Permit Lifecycle" field after upgrading to a newer release.

Solution

Implement the correction instructions of this SAP Note.
