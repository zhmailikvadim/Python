Symptom

When using functionality for searching and downloading invoice PDFs from SAP ERP while using SAP Commerce Cloud, the impex defining the Integration Object can't be found after Commerce version upgrading.

Environment

SAP Commerce Cloud

Cause

The core-customize/bin/modules/sap-billing/deprecated/sapinvoiceaddon/resources/impex/essentialdata_sapinvoicedocument.impex is not available as the sapinvoiceaddon is deprecated since 2205.

Details in Deprecation Status.

Resolution

The sapinvoiceaddon AddOn provides user interface components for B2B online stores to allow customers to view SAP ERP invoices in the online store.

And the impex script used for InboundSapB2BDocument could be found in below guide:

sapinvoiceaddon AddOn
