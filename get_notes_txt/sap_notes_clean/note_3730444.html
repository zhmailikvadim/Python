Symptom

There is a requirement to prevent or block the creation of the eDocument for Zero Value Invoices in SAP S/4HANA Cloud, Public Edition. The appropriate steps and prerequisites for preventing the creation of such eDocuments need to be identified. 

Environment

  • SAP S/4HANA Cloud Public Edition
  • SAP Document and Reporting Compliance, Cloud Edition

Reproducing the Issue

  1. Go to eDocument Cockpit application. 
  2. Observe that eDocument for the Zero Value Invoice are created. 

Cause

Creation of eDocuments for zero-value invoices is standard system behavior in SAP S/4HANA Cloud Public Edition.

Resolution

To prevent the creation of the eDocument for Zero Value Invoices in SAP S/4HANA Cloud, Public Edition, you can follow the detailed procedure outlined in the following SAP Community blog: How to prevent creation of Electronic Documents for zero value invoices.
