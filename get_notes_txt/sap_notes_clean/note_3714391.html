Symptom

In CAT2 transaction and My Timesheet application, when a timesheet record is released for approval using workflow, the approver is not determined correctly in some cases if the approver has Concurrent Employment.

Other Terms

CAT2, My Timesheet, F3074, F3074A, Approver, Workflow, Concurrent Employment, CE, BADI_CATS_APPROVAL, DETERMINE_APPROVER.

Reason and Prerequisites

For workflow based approver determination, when the BAdI Method: BADI_CATS_APPROVAL~DETERMINE_APPROVER returns the personnel number of the approver which is not the default assignment in Concurrent Employment, the determination of user ID of the approver fails.

Solution

Apply corresponding support pack. Alternatively, implement the correction instructions attached to the SAP Note.
