Symptom

In order to perform remote ATC checks, the Checked System has to be configured.

However menu path Code Inspector → Management of → Reference Check System is not available on the initial screen of SCI.

Environment

SAP NetWeaver

Reproducing the Issue

  1. Run transaction SCI.
  2. The Code Inspector initial screen is opened.
  3. Try to open menu path  Code Inspector → Management of → Reference Check System, the last option is not listed.

Cause

The manual post-implementation steps from SAP Note 2381403 were not performed or the SAP Note was not implemented.

Resolution

Implement SAP Note 2381403 and perform the manual post-implementation steps.

  • SAP Note 2364916 - Recommended SAP Notes for using ATC to perform remote analysis
