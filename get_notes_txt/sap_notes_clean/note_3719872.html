Solution

What’s New for SAP GRC UIDP Logging for SAP HANA (Support Pack - 00)

 

Centralized Configurations and Log Analysis

  • UI Data Protection Logging follows a centralized configuration and Log analysis model. Logging configurations are defined in a Central System and distributed to connected Plug-in Systems.
  • Logs are generated in the Plug-in Systems and sent to the Central system, where the Data Protection Expert performs Log Analysis.

Switch to the latest Streamlined Logging

  •  Classic Logging is removed.

Fiori-Based Log Analyzer

  • New fiori app to analyze logged data with visual filters and charts for fast and easy analysis.

User List App

  •  To maintain a suspect list from log analysis to quickly address potential security breaches.

Alerting

  • You can now set up and trigger alerting notifications when specific critical data is accessed.

Pseudonymization

  • You can pseudonymize user IDs to protect personal identities and ensure compliance.

Rest API for Logs

  • Logs can be consumed by external applications using this API.

Migration of Logs and Customizing

  • You can use these reports to migrate logs and configuration data to Streamlined Logging.
  •  Facilitate faster and smoother migration processes for existing customers.

For more information on SAP GRC UIDP Logging for SAP HANA, please refer to https://help.sap.com/docs/SAP_GRC_FOR_SAP_HANA
