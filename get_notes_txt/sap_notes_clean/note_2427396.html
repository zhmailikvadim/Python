Symptom

With this change request texts and forms for vendor declarations are changed according to the changes of the preferential agreement with Bosnia and Herzegovina.

Other Terms

C2S1000028722, C2S 1000028722, TTP1839, TTP 1839, vendor declaration, request, dunning, dun, form, YY_LE_AUD_MAT, YY_LE_AUD_MAT_E, YAUDP, YAUDP_E, SAPscript, SO10, SE71

Reason and Prerequisites

Change request.

Solution

The enhanced SAPscript objects are attached to this note. There are four files, two of them for German language and two for English language output.
Download the files and execute report RSTXSCRP to import them into the target system. They are activated automatically.
See the report documentation for more details.
Consider the illustrated implementation instructions that are attached to this note.
