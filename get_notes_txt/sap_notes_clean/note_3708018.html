Symptom

job SAP_COLLECTOR_PERFMON_RSTUNE80 cancels with error message "'SAPTUNE_HISTORY_SAVE' failed, sy-subrc: 2", all the configurations mentioned in KBA 2774952 are properly implemented.

Environment

  • Workload Monitoring Tool
  • SAP NetWeaver
  • SAP NetWeaver Application Server for SAP S/4HANA
  • ABAP PLATFORM - Application Server ABAP

Cause

This is because of retention period in ST03 is set too high, RELID table has many old and obsolete records.

Resolution

  1. go to
    ST03>Collector and Performance DB> Performance DB > Monitoring Database > Reorganization
    set here period for "Other retention period" the same like in "Standard Retention periods"
    2D,2W,2M
  2. go to
    ST03>Collector and Performance DB> Performance DB > Monitoring Database > Contents
    and delete RELID
