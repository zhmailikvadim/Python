Symptom

When replacing an SMGW which is connected to different MMDs (1:N relationship) by the process 3070, the process engine incorrectly triggers only 1 step 1020, followed by 1 step 1030. The creation of step 1030 blocks / delays the creation of the other 1020 steps.

To fix this issue, the following adjustments of the process configuration need to be done.

1. Changing the step link cardinality of the step conditions of step 1020 from '1:1' to '1:N' except for step 0530.

2. Changing the execution type of step 1030 from 'single processing' to 'multiple processing'.

Other Terms

Process 3070, 1:N SMGW replacement, Step 1020, Step 1030

Reason and Prerequisites

 This is a program error.

Solution

Please implement the correction instruction in the SAP Note or install further Support Package.
