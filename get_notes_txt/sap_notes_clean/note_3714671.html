Symptom

The prototyping parameter {LGL} has been newly introduced with this note:

  • {LGL:X} – The ledger group is derived directly from the ledger.

  • {LGL: } (default) – The ledger group is derived via the accounting principle.

This note enables testing of the functionality using EBRR prototyping parameters.

Please note that this functionality is intended for testing purposes only and has not yet been released for productive use.

Customers and implementers are responsible for thoroughly testing and validating the resulting system behavior before considering any productive usage.
