Symptom

Transportation Management in S/4HANA: When receiving an ASN update and using internal HU numbering, it can happen, that the external Package ID is taken over incorrectly as internal Package ID. This can lead to failing Delivery Update and leave inconsistent documents in the system.

Other Terms

Update Handler; /scmtms/cl_om_uh_fac_srv_map; Package IDs;

Reason and Prerequisites

Program Error

Solution

Implement the attached Correction Instruction (CI) or the relevant Support Package.
