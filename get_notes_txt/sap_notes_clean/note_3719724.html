Symptom

Currently, the survey notification email sent to managers using the template FollowupEvaluationAssignmentNotification includes multiple survey completion entries, but only a single direct link is provided.

Environment

SAP SuccessFactors Learning

Reproducing the Issue

  1. Assign an item to multiple users >
  2. Users complete the course >
  3. The survey notification is triggered to their manager >
  4. The manager clicks the notification link to launch the survey >
  5. The link redirects the manager to launch the survey for only one of the users >
  6. The manager is not able to answer the survey for all users using the provided link.

Resolution

The email template containing the direct link to surveys was enhanced with the 2505 product version, which is working as expected for users. However, in this use case, the product enhancement is insufficient, as the email contains multiple survey information for managers but only provides one direct link by default. A change in this behavior will be addressed as an enhancement request, but there is no timeline for implementation yet.

Please follow the Product Release and Road Map Information for product updates.

In the meantime, managers should use their 'Requests For Me' section to access all pending surveys.
