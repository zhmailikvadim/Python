Symptom

The Attachment XML Download connector fails, returning error: <ErrorMessage>[SOW ID] is invalid.</ErrorMessage>

Cause

  • This error occurs when attempting to download attachment data for a Statement of Work (SOW) that is in Closed status. 
  • The Attachment XML Download connector does not support SOWs in Closed status, which results in the invalid SOW ID error. 

Resolution

This behavior is working as designed.
