Symptom

  • When opening a Circle Map report, nothing is displayed.
  • Interfaces have Provider AND Consumer Applications linked, but the report is not generated.

Environment

LeanIX > Reports > Circle Map (Standard report)

Reproducing the Issue

  1. Open a saved Circle Map report.
  2. The canvas remains empty and a "No data" message appears

Cause

The Application-dimension filter excludes all provider/consumer applications (for example, Quality Seal: Broken OR Approved while Applications were in Draft state), resulting in no data displayed in the report.

Resolution

  1. Open the affected Circle Map report.
  2. Open the Filters panel.
  3. In the Application filter dimension, broaden the criteria to include the relevant Quality Seal states (for example, include Draft) or clear the Quality Seal restriction.
  4. Verify both Interface and Application filter dimensions are aligned so provider/consumer applications are included by the criteria.
  5. Apply the updated filters and reload the report; confirm that data appears.
  6. Save the report configuration for future use.
