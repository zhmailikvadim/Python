Symptom

OData API services API_PURCHASEORDER_PROCESS_SRV (V2) or API_PURCHASEORDER_2 (V4) are not creating Stock Transport Orders (STO).

Environment

  • SAP S/4HANA
  • OData APIs - Purchase Order

Reproducing the Issue

Use ODATA API to create the Stock Transport Order. Document will not be created, or will be missing critical STO related data.

Cause

OData APIs for purchase order creation (both V2 and V4) do not support the creation of intercompany Stock Transport Orders (STOs).

This limitation is documented in the SAP Help page for the OData services: 

  • Purchase Order (OData V2)
  • Purchase Order (OData V4)

Resolution

This is a design limitation of the current API versions. Creation of intercompany Stock Transport Orders via OData API is not supported.

If the business process requires to create STOs, the purchase order must be created using other channels, like transaction ME21N.
