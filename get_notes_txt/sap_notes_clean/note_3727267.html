Symptom

  • How many E-Signature incorrect PIN attempts are allowed?
  • What happens if a user enters an incorrect E-Signature PIN several times?

Environment

SAP SuccessFactors Learning

Reproducing the Issue

  1. Access SuccessFactors Learning >
  2. Access a training that requires E-Signature >
  3. Complete the content and open the E-Signature prompt >
  4. Enter the username and an incorrect PIN.

Resolution

  • The system allows three incorrect E-Signature PIN attempts, after that the user should reset their PIN.
  • For the first attempts, the system will display the message “You entered an incorrect username and/or the electronic signature”.
  • After the third attempt, the following message will be displayed: “The maximum allowable tries to enter a valid electronic signature PIN has been exceeded”.

Keywords
