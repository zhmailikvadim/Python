Symptom

Use case Complete onboarding tasks, if new hire activities are marked as optional Joule will not be triggered on those tasks when prompted.
 

Environment

SAP SuccessFactors Onboarding

Reproducing the Issue

  1. Assign Buddy Task or any other NHA is Optional Task
  2. Open Joule
  3. Ask if you have to assign buddy
  4. Response informs you do not have any task to perform

Cause

The assign buddy task is an optional task, optional tasks do not have a to do created therefore in case If new hire activities are marked as optional Joule will not be triggered on those tasks when prompted.

 

Resolution

Expected behavior due to new hire activities are marked as optional Joule will not be triggered on those tasks when prompted.
 

Keywords
