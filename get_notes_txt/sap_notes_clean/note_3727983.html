Symptom

Error FR791 occurred in Tcode: OBJVAL & OBJREG.

Obj. RECL/XXXXXXXXX/XXXX acct XXXXXXXXX: Account assignment not unique. Reconciliation required
Message No. FR791

Environment

  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA Finance
  • SAP S/4HANA

Reproducing the Issue

Execute Tcode: OBJVAL or OBJREG.

Cause

Program error

Resolution

Make sure below note is implemented in the system:

3044302 - OBJRECON / OBJVAL / OBJREG: Check assignment level of contract balances / Consideration of closed contracts in reconciliation

The message FR791 is a customizable message, it may technically can be switched to a type 'W' message via the setting in view V_T100C. However please note, in case of setting the message to 'W' (warning) the contract balances will be processed by OBJVAL/OBJREG which may lead to unwanted or inappropriate results.
