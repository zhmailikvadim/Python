Symptom

TM Document Flow is not displayed on "TM Status" screen of ECC douments, which are integrated with SAP TM. Message "Internal Service Error, no document flow data could be retrieved" (INT_TM_ERP352) is displayed.

Error message "Logical system for business system could not be determined in SLD" (/SCMTMS/BDFLW_QR000) is issued during processing in TM system.

Reason and Prerequisites

Program error

Solution

Implement the attached Correction Instruction (CI) or the relevant Support Package.
