Symptom

In a multi-level approval setup, when a delegation (managing parent) user attempts to submit an approval request to the next approver on behalf of the original approver, the system throws an error and blocks the workflow.

Environment

SAP CPQ

Reproducing the Issue

  1. Submit a quote for approval to User A.
  2. Log in as User B (managing parent of User A).
  3. Open the same quote and click Request Approval.
  4. Attempt to submit the approval to User C (next approver).
  5. System throws the message: “This approval request is above your authorized approval level. You can either reject a quote or submit it upstream for approval.”
  6. Developer Console error shows: “Action not available in the status.”
  7. The quote remains in Awaiting Internal Approval status with User A.

Cause

Validation issue is preventing delegated users from submitting approvals.

Resolution

This issue will be tentatively fixed in 2605 release.
