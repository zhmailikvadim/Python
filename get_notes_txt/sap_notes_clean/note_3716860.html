Symptom

  • During activation of Scope-Item DE_5D2_OP, the following errors occur:
  • In Building Block 5XM (XX): Error: Item category is not allowed according to item category determination.
  • Object Name: V_SDSLS_ITM_VCMC
  • Table Key: 300CBICICSL

Environment

  • Sales and Distribution (SD)
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Attempt to activate Scope-Item 5D2 for your release and location.
  2. Encounter errors in Building Block 5XM (XX) related to item category determination.

Cause

Resolution

  1. Navigate to transaction /SMB/BBI and open Implementation Assistant
  2. Find view V_SDSLS_ITEM_VCMC
  3. Reactivate the view
