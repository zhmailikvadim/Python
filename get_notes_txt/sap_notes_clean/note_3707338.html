Symptom

  • After BI 2025 upgrade, promotion management CLI fails with : Cannot invoke "java.util.Date.toString()" error
  • When using LCM CLI, you are getting the below error:
    Cannot invoke "java.util.Date.toString()" because the return value of "com.crystaldecisions.sdk.properties.IProperties.getDate(Object)" is null
  • Query used in CLI properties is similar to below:
    SELECT top 1000 si_cuid FROM CI_SYSTEMOBJECTS WHERE SI_KIND ='USER' and si_cuid in  ('cuid1','cuid2') 

Environment

SAP BusinessObjects BI Platform 2025 & above
SAP BusinessObjects BI Platform 4.3 & above
All supported OS according to PAM

Reproducing the Issue

Run LCM CLI with a query similar to above according to KBA  1969259

Cause

The Query requires to return more metadata info than just the si_cuid to promote content.

Resolution

To resolve the issue

  • The query had to be changed to return more than si_cuid for Promotion CLI to promote entire user metadata
  • so instead of "select si_cuid" , the query was changed to "select *" as the Promotion CLI job needs more metadata than just the si_cuid to pull the user content. 
  •  Query was changed to look similar to:
  • SELECT top 1000 * FROM CI_SYSTEMOBJECTS WHERE SI_KIND ='USER' and si_cuid in  ('cuid1','cuid2')
