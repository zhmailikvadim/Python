Symptom

When replicating an employee the following error message is displayed: 

  • Fill mandatory field : PXXXX-XXXXX for time segment xx.xx.xxxx to xx.xx.xxxx
    Message No. ECPAO_IN_PROCESSING039

These KBA aims to explain which fields are mandatory for the replication.

Environment

  • SAP SuccessFactors Employee Central integration to SAP Business Suite
  • Employee Master data replication 
  • Business Integration Builder (BIB)

Cause

Some fields are mandatory for the replication of determined Infotypes, and this mandatory fields can differ based on the country of the Company.

The system uses one of these two tables to determine the mandatory fields:

  •  V_T588MFPROPC (Custom) or
  • V_T588MFPROPS (SAP Internal) 

When a field have the flag "Required Entry" marked, that field will be set as mandatory for that Infotype during the replication.

Resolution

To fix this issue, follow these steps below:

  1. To identify which fields are mandatory, at first it is necessary to identify the Country/Molga of the Company in the T500P table.
  2. After that with the Molga and the Infotype values, it is possible to check inside the V_T588MFPROPC or V_T588MFPROPS table to identify which fields are mandatory.
  3. To make a field optional or mandatory please refer to SAP Note 2489963 - How to make a field Optional or Mandatory in Master Data Replication? - SAP for Me
    Please note that some fields are mandatory due to Country requirement, and cannot be set to optional.
