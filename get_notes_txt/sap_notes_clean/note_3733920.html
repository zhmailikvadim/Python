Symptom

Error T5002 'Telephone/fax code for country code & is not maintained' occurs when maintain telephone number.

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

  • Sales and Distribution (SD)
  • Logistics Execution (LE)
  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA
  • SAP Netweaver 

Reproducing the Issue

  1. Use t-code XK* XD* or BP trying to maintain telephone number.
  2. Error T5002 'Telephone/fax code for country code & is not maintained' occurs.

Cause

Missing customizing for the country settings.

Resolution

Maintain the country specific phone numbers using t-code OYC4
