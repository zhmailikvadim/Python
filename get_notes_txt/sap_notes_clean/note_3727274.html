Symptom

  • An error occurs while executing F.27.
  • Error message FB461 : "Correspondence type SAP01 is not permitted since no date required."

Environment

  • SAP R/3
  • SAP S/4HANA

Reproducing the Issue

  1. Execute F.27 in the system.
  2. Encounter the error message FB461 : "Correspondence type SAP01 is not permitted since no date required."

Cause

The standard correspondence type SAP01 is typically configured without a date requirement. However, F.27 is designed to run with date logic, which causes the system to throw the error when SAP01 is processed without a date.

Resolution

  1. Verify the program assigned to the correspondence type SAP01.
  2. Replace the program F.27 with RFKORD10 for processing correspondence type SAP01.
  3. Ensure the date requirement logic aligns with the configuration of correspondence type SAP01.
