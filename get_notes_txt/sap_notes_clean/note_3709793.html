Symptom

  • External OData bound action call associated with a a collection entity does not work as expected
  • While debugging the code line the call will show up as "undefined"

Environment

SAP Cloud Application Programming Model

Reproducing the Issue

  • External OData bound action call associated with a a collection entity does not work as expected. For example, trying to call a bound action like below does not work
    const svc = await cds.connect.to(''<service>");
    const payload = {
         <some payload>
        };
    let result = await svc.PostInvoice(payload);
  • Debugging this part of the code gives message "undefined"
 
 

Resolution

  • This is not supported yet in the CAP libraries. Customers can create enhancement request following KBA 0003673324 - How to submit a Feature Requests / Enhancement requests/ ideas 
  • As a workaround customers can write their code as follow
    const entry = {
          <some payload>
        };   
    const actionPath = '/PostInvoice/<Name space>/';
    result = await svc.send({ method: 'POST', path: actionPath, data: entry });
