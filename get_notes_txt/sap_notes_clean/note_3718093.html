Symptom

How to add a custom card to the home screen in SAC?

Environment

SAP Analytics Cloud 2026.2.5

Resolution

  1. Login to the SAC tenant
  2. Navigate to system --> Administration --> Default appearance tab
  3. Navigate to the Home Screen Cards
  4. Click on "Add a Custom Card"
  5. Add the description, upload images, set the tag type and the tag text. You can also choose to lock the card on the home screen for all users. 
  6. Click on save.
