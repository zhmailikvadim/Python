Symptom

You run HR-HK: Periodic MPF contribution report (format: eMPF) (HHKCPFE0) report to generate eMPF data for enrollment weekly. System displayed Payroll Contribution Period Start/End Date by selection period. However, according to the reply from eMPF, Payroll Contribution Period Start/End Date should be the first/last day of month.

The SAP Note adjusts report. When generating eMPF data for enrollment, system will display the first/last day of the enroll month as the Payroll Contribution Period Start/End Date.

Other Terms

eMPF, HHKCPFE0, begda, endda, Payroll Contribution Period Start Date, Payroll Contribution Period End Date

Reason and Prerequisites

This is a correction of legal function.

Solution

Apply the Support Packages or implement the corresponding Correction Instructions.
