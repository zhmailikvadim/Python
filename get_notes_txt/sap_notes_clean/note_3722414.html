Symptom

You are using the Fiori app 'Migrate Your Data' to migrate data to SAP S/4HANA.

You have copied a migration object/project and can't view the information for the copied objects in the 'Data Migration Status' app.

Other Terms

Migration Cockpit, staging table, staging approach, data migration, Migrate your Data App, LTMOM, Migration Object Modeler, Copy Project, Copy Migration Object

Reason and Prerequisites

You are using:

SAP S/4HANA

• Migrate Your Data - Migration Cockpit, Migrate Data Using Staging Tables

• Migration Object Modeler

Solution

Implement the attached correction using transaction SNOTE.
