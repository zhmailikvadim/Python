Symptom

Error message "More thresholds are required to build a Bullet chart" is displayed on the bullet chart in a story in SAP Analytics Cloud(SAC). The data source of the chart is based on a BW or S4 HANA Live connection.

Environment

SAP Analytics Cloud

Reproducing the Issue

Run the story.

Cause

Program error.

Resolution

Implement SAP Note 3678179 - INA: Thresholds are not correctly applied in the case of a Restricted Measure on the BW or S4 HANA ssytem.
