Symptom

There is SAC Mobile IOS SDK, is there SAC Mobile Android SDK?

Environment

SAP Analytics Cloud Mobile SDK

Resolution

There is no SAC Mobile Android SDK. There is only SAC Mobile iOS SDK, details can be found in the help guide SAP Analytics Cloud iOS SDK Developer Guide
