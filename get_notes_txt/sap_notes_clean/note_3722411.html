Symptom

The production order is not visible in the Return Components List when attempting to return components via the Receipt from Production window.

Environment

SAP Business One

Reproducing the Issue

Prerequisites:

You are working with SAP Business One, version for SAP HANA. 

Scenario:

  1. Go to Production → Production Order and create a new document. The components are set to Issue Method: Manual.
  2. Right click on the production order and select: Issue for Production. Add the document. 
  3. Go to Production → Receipt from Production. Select Return Components. List of Production Orders opens.
  4. Search the production order created at point 1. 

Expected behavior:

The production order is listed.

Actual behavior:

The production order is not listed. 

NOTE: The issue is reproduce in other parts of the software where Choose from List is used. For example: Outgoing Payments. 

Resolution

This is the correct system behavior. The number of available documents is higher than the number set for Choose From List in SAP Business One. 

  1. Go to Administration → System Initialization → General Settings → Display. Go to Choose from List Preferences.
  2. Check the value for settings: Display. No. of Rows in Choose From List. It is set to a limited number (for example: 10,000). 
  3. Change the setting from a limited number (e.g., 10,000) to Unlimited.
