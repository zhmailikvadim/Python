Symptom

The date picker in the My Leave Requests (Fiori 2.0 / Version 3) application does not mark the non-working last day of the month correctly.

Environment

  • My Leave Requests (Fiori 2.0 / Version 3)
  • SAP ERP
  • SAP S/4HANA

Reproducing the Issue

  1. Open the "My Leave Request" application.
  2. Click Create New Request
  3. Navigate to a month where the last day of the month falls on a non working day (weekend)
  4. Observe that this day is not grey like the other non working days

Cause

Programming error

Resolution

  1. Implement SAP Note 3705502
  2. Clean up the cache using the steps provided in SAP Note 2319491 - How to clean up the cache after applying changes that affect SAP Fiori apps

Keywords

date picker, my leave request, non-working last day, SAPUI5 1.71.75, hcm.fab.myleaverequest, F1311A, SAP Note 3705502
