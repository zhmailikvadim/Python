Symptom

You encounter problems with the area of Analytics --> Users --> License Consumption in the SAM4U tool in case you are licensed for an FUE metric. Instead of calculating the number of FUEs based on the Target Classification, it is based on the Current Classifications. Also the breakdown of use types contributing to the FUE count is based on the current classification instead of the target classification.

Other Terms

Software Asset Management, SAM4U Tool, SAM4U, PCE, License Consumption

Reason and Prerequisites

You have installed SAP TCI Note 3646933 - SAM4U Solution | Fiori Application | V2 and other prerequisite corrections which are referenced in SAP Note 3646933.

Solution

Implement this SAP Note to fix the above-mentioned issue.
