Symptom

User is not able to set up value of authorization object S_SERVICE for OData Service RSBPC_ODATA_EXPORT_SRV.

Other Terms

SAC, Export, write back to BW, Gateway

Reason and Prerequisites

Gateway service need to be executed to generate the HASH.

Solution

Execute Gateway service by either of two methods below:

1. Execute the Gateway service from Gateway Client,

  • Access TCODE /n/IWFND/GW_CLIENT to enter SAP Gateway Client
  • Input Request URI: /sap/opu/odata/sap/RSBPC_ODATA_EXPORT_SRV/$metadata

2. Access URL in the browser: http(s)://<hostname>:<port>/sap/opu/odata/sap/RSBPC_ODATA_EXPORT_SRV/$metadata

After that, verify by checking table USOBHASH, filter by OBJ_NAME = RSBPC*, the corresponding records of SRV_NAME are generated.
