Symptom

Prepayment Request posting fails when the supplier number maintained in the document is different from the corresponding Business Partner number.

Steps to Reproduce
1.Create a Trading Contract Price using transaction WB21.
2.Ensure that the Business Partner number is different from the corresponding Customer or Vendor number.
3.Add Prepayment Request (PPR) lines to the contract.
4.Execute transaction PPMGCO_PMT_REQPOST to post the Prepayment Request.
5.The system throws an error and PPR posting fails due to the validation check.
6.Error message Invalid Distribution Beneficiary &1.

Other Terms

PPMG; Prepayment Agreement; ACM; Agricultural Contract Management; Prepayment Request Post; PPMGCO_PMT_REQ_POST; PPMG_AGR 061

Reason and Prerequisites

Program error

Solution

Import the support package or implement the attached correction instructions.
