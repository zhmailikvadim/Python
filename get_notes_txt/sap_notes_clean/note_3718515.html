Symptom

  • After entering data in a field, the page reloads automatically.
  • The page becomes blank after editing a field.
  • Changes entered in the document are not saved.
  • Fields become unresponsive during Work Item creation or submission.
  • The system redirects unexpectedly or does not load the next step.

Reproducing the Issue

  1. Sign in to SAP Fieldglass.
  2. Navigate to a Work Item document (e.g., Job Seeker, Job Posting, or Statement of Work).
  3. Click Edit.
  4. Enter a value in any editable field.
  5. The page reloads, becomes blank, or does not load properly.

Cause

Certain application features may not function as expected due to browser session-related issues.

When a user logs in, the system creates a session that is maintained through cookies stored in the browser. If the session expires, cookies are blocked or corrupted, or cached browser files become outdated, the application may not process requests correctly. As a result, pages may reload unexpectedly, display as blank, or prevent data entry.

Resolution

If the issue occurs while editing Work Items, try one of the following troubleshooting steps:

  • Close all browser windows.
    1. Ensure all active browser sessions are fully closed (including additional tabs).
    2. Open a new browser window and sign in to SAP Fieldglass again.

      OR

  • Clear browser cache and cookies.
    1. Open the browser settings.
    2. Locate the option to clear browsing data.
    3. Select Cached images and files and Cookies and other site data.
    4. Confirm the action.
    5. Open the browser again and sign in to SAP Fieldglass. 

      OR

  • Log in again in a different supported browser
    • E.g., Google Chrome, Mozilla Firefox, Microsoft Edge

      OR

  • Access the application using an Incognito/Private browsing window.
    • This ensures a fresh session without previously stored cache, cookies, or extensions.
    • Common shortcuts:
      • Chrome / Edge: Ctrl + Shift + N
      • Firefox: Ctrl + Shift + P
    • Then sign in to SAP Fieldglass and reproduce the scenario.
