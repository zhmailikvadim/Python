Symptom

When syncing user data from SuccessFactors (BizX) to SAP Jam/Work Zone, what happens if a user’s email address is empty?

Environment

  • SAP SuccessFactors Work Zone
  • SAP Jam Collaboration

Resolution

SAP Jam always requires an Identity Provider (IdP), such as SuccessFactors or IAS, to manage user identities.

  • When SuccessFactors is directly integrated with SAP Jam without IAS: If a user’s email address is empty, the synchronization will still proceed successfully. Jam will automatically generate a dummy email address for that user, and all other user attributes will be updated normally.
  • When you use IAS to manage users, user provisioning is handled via IPS. Email is a required attribute for the users to be created in IAS.
