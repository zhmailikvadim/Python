Symptom

The borrower change was made for a loan contract. In transaction FMN9 the partner number is used in the select options for contract selection. In this case the documents posted with the old main partner are not displayed. It is expected that all documents of the selected contracts are displayed. The select option for the partner should not be considered for the selection of the documents.

Other Terms

FMN9 RFVDSO04

Reason and Prerequisites

Program error

Solution

Implement the attached Correction Instruction (CI) or the relevant Support Package.
