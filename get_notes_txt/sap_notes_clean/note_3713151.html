Symptom

  • You are planning to migrate from SAP JAM to Work Zone.
  • Can you create an backup version of the SAP JAM or Work Zone before proceed with major changes?
  • What methods is applied or recommended to ensure content migration accuracy?

Environment

  • SAP JAM Collaboration

  • SAP Build Work Zone, Advanced Edition

  • SAP SuccessFactors Work Zone

Cause

There is no data migration or backup functionality available for SAP JAM/Work Zone.

Resolution

During migration/conversion activities from SAP JAM to SAP SuccessFactors Work Zone, changes are applied in the system tenant level only, and data remains unaffected during task activities.

There are different export tools you can use to achieve it, kindly check KBA 2706983 - SAP Jam - How to backup or transfer content from one instance to another - SAP for Me.
