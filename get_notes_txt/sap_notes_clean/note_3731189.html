Symptom

When opening one of the bellow apps, it displays "No data":

  • Order-to-Cash Performance – Time Series (F2006);
  • Order-to-Cash Performance – Overview (F2242);
  • Order-to-Cash Performance – last 28 Days (F2005).

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

Cause

The scope item BKN, which is the responsible for the mentioned apps, is deprecated in SAP S/4HANA Cloud Public Edition for new installations from release 1805.

For installations before 1805, the apps may still appear in the launchpad because of the business catalog SAP_SD_BC_SALES_PROC_PERF_MC, but they do not return data.

Resolution

Do not use these apps in SAP S/4HANA Cloud Public Edition as the functionality is deprecated.
