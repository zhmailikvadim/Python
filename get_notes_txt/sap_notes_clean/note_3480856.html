Symptom

A proof is required, document that the SAC and the URL used for your tenant is compliant with all necessary security standards - i.e. configured SSL 1.2, DDoS tested, penetration tested and etc.

Environment

SAP Analytics Cloud

Resolution

Official documentation is listed as below:

Security - Help Portal
Platform Security - Help Portal
SAP Trust Center

SAC runs on the SAP Business Technology Platform, so some security measures are also implemented on platform level.

Also, some of the mentioned security details are self-evident: The SSL/TLS protocol is visible when inspecting and analyzing the certificate of the URL.

Checking the SAP Trust center where they can request certifications such as SOC2, ISO27001.  SAC is part of the BTP compliance.
