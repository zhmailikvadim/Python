Symptom

You schedule a job via the Work Manager app and expect the job execution planned start time to match the selected scheduling time. However, when the job is scheduled, the planned start time does not match the selected time.

This note corrects the behavior by ensuring that jobs scheduled via the Work Manager app execute at the selected time.

Other Terms

/VTA/LBCKMF0S, /VTA/CL_BWM_FRAMEWORK

Reason and Prerequisites

This is caused by program error.

Solution

Please implement the correction instructions attached. These corrections are only valid if you have applied the Support Packs.

Release V4 - 2023 V3, and 2023 V3 Support Pack 1.

Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object.
