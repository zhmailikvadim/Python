Symptom

  • The Standard KPI tile "Manage Customer Master Data" (F0850A) displays an error message despite all OData and ICF Services being activated and no visible authorization errors.
  • Error occurs when accessing the KPI tile via the Fiori Launchpad.

Environment

• SAP S/4HANA

• SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Open the Fiori Launchpad.
  2. Locate the App "Manage Customer Master Data" (F0850A).
  3. Add the tile "Customer Master - Create/Change/Display" to any of the Groups on the home page using the "Edit Home Page" option.
  4. Observe the KPI tile displaying an error message on the Finance Launchpad home screen.

Cause

Resolution

  1. Implement SAP Note 3259054 - Display count for Customers in Customer Master tile.
  2. Retest the issue after implementing the note.

See Also

  • Note 3259054 - Display count for Customers in Customer Master tile
