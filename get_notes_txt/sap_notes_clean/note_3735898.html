Symptom

When attempting to create sales document types in the CBC Configure Standard Order Type app, the error appears: "You are not authorized to create Document Types. Please create a support case to SAP support."

Environment

  • SAP Central Business Configuration
  • SAP S/4HANA Cloud Public Edition

Cause

The CBC user lacks role "SAP_CBC_CONSUMPTION_KEY_USER" and "SAP_CBC_CONSUMPTION_DISPLAY_USER".

Resolution

  1. Assign CBC role "SAP_CBC_CONSUMPTION_KEY_USER" and "SAP_CBC_CONSUMPTION_DISPLAY_USER" to your users in SAP Cloud Identity Services which connects your CBC system. More information: SAP Help Portal- Creating Business Users and Providing Access Rights
  2. Running Jobs for User Provisioning

See Also
