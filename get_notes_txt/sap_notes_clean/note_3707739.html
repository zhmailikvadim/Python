Symptom

Alternative-Based Confirmation (ABC) does not perform the expected substitution according to the substitution master data for plants, products, or storage locations.

Other Terms

aATP, Alternative-Based Confirmation, ABC, Third-Party Order Processing, TPOP, Substitution Control, Substitution Master Data

Reason and Prerequisites

The symptom can occur if all the following conditions are met:

  • The substitution is triggered from a follow-up substitution strategy for Third-Party Order Processing.
  • The substitution is set up with groups, controls, and access sequences for substitution.

The reason is that the substitution control is not found due to a programming error which leads to an overwrite of the scheduleline buffer entry which would be required in its original form.

Solution

Apply the correction instructions for this note or upgrade to a SAP S/4HANA support package that contains this correction.
