Symptom

You want to export product catalogs data through data source or data extraction process.

Environment

SAP Business ByDesign

Reproducing the Issue

  1. Go to Business Analytics work center 
  2. Go to Data Sources view 
  3. Select Design Data Sources sub view
  4. Search for Catalogs 
  5. You do not see any data source to export Catalog data

 Or

  1. Go to Application and User Management work center 
  2. Go to Data Extraction view
  3. You do not see the Catalog under Extraction objects

Cause

[Enter your Cause here...]

Currently there is no data source or data extraction process to export catalogs data. You can post this as a new requirement in influence forum - https://influence.sap.com/SAPByDesign

 

Resolution

You can export the catalogs from the below work center view.

  1. Go to Product Portfolio work center
  2. Go to Catalogs view 
  3. Select all Catalogs 
  4. Select Export button -> To Microsoft Excel

 This way you can export catalogs data to MS excel.
