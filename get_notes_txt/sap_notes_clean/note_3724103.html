Symptom

  • Directory fields "Advanced search" in a different language version are not displayed as expected.
  • Fields appear correctly when the language is set to English but disappear when set to something else.
  • No error messages or error codes are explicitly show. 

Environment

SAP SuccessFactors Employee Central

Reproducing the Issue

  1. Set language to English and navigate to Directory Search > Advanced search dropdown > Fields are visible.
  2. Set language to something else and navigate to Directory Search > Advanced search dropdown > Fields are not visible.

Cause

The issue arises due to a custom translation configured in the Manage Languages pack. The translation was overridden from `{0}:` to `{job_code}:Función`, which causes an error in the UI5 bundle.

Resolution

  1. Modify the string in the language pack from `{job_code}:Función` to `{0}:Función`.
  2. Alternatively, disable the language pack temporarily to resolve the issue.
