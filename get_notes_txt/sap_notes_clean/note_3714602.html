Symptom

You use the gCTS app and have valid credentials maintained for GitLab. However, when you try to push objects to a GitLab repository, the process fails with a permission error. A push can happen in the following situations:

- You manually push objects on the Objects tab in the gCTS app.

- You release change requests with ABAP objects in ABAP Development Tools.

- You manually commit and files in the gCTS app, for example after a conflict resolution. Then you try to push these local commits using the Push button on the Objects tab.

Other Terms

Git-Enabled Change and Transport System

Reason and Prerequisites

There were some changes in Gitlab authentication.

Solution

To solve the issue, implement the SAP Note or import the corresponding Support Package.
