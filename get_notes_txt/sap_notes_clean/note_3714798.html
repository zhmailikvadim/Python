Symptom

  • The "Create Task List App" activates the Long Text indicator at the Header Level, even when no long text is assigned.
  • Task Lists are not recognized and errors occur when assigned to a Maintenance Order.
  • Error message displayed: "Text &1 ID PLKO language EN not found."

Environment

  • SAP ERP Plant Maintenance (PM)
  • SAP enhancement package 8 for SAP ERP 6.0
  • SAP S/4HANA 1809
  • SAP S/4HANA 1909
  • SAP S/4HANA 2020
  • SAP S/4HANA 2021
  • SAP S/4HANA 2022
  • SAP S/4HANA 2023
  • SAP S/4HANA 2025
  • SAP S/4HANA Cloud Private Edition 2023
  • SAP S/4HANA Cloud Private Edition 2025

Reproducing the Issue

  1. Open the "Create Task List App."
  2. Create Task List 
  3. Save
  4. Add Task List Maintenance Order
  5. Observe the error message TD600 is raised

Cause

Program error 

Resolution

Please implement SAP Note 3568712

Keywords
