Symptom

Case1:
1. An OW is paid via an offcycle payroll,
2. Regular payroll run w/o OW
In this case the OW from offcycle is missing from current month's /3OW

Case2:
1. An OW1 is paid via an offcycle payroll
2. Regular payroll run also has OW2
In this case the OW2 from regular is missing from current month's /3OW

Case3:
1. An OW1 is paid via an offcycle payroll
2. Regular payroll run also has OW2
3. Correction payroll has additional OW3
In this case the OW3 from correction payroll run is missing from current month's /3OW

Version2 Updates:

Added for case2 & case3.

Other Terms

/3OW

Reason and Prerequisites

This is a program error.

Solution

Apply Support Package.

Alternatively, apply correction instructions attached to the note.
