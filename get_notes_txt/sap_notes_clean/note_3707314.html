Symptom

The issue occurs in the "My Team Calendar" App (F1309B) but it can be problem by other HCM Fiori application also, which use the same CDS views.

Employees assigned to non-active plan versions are appearing in the team calendar view, which should only display employees assigned to the active plan version. 
The view I_HCMORGHIERMANAGER used as the basis for the team calendar view, includes entries from non-active plan versions. 

Environment

H4S4 Fiori Apps

Reproducing the Issue

1. Open the "My Team Calendar" App (F1309B).
2. Observe that employees assigned to non-active plan versions appear in the team calendar view.
3. Verify the entries in the view I_HCMORGHIERMANAGER and table HRP1001 to confirm that non-active plan versions are included.

Cause

It's a bug.

Resolution

Please be sure, that the SAP Note 3688377 - "Fiori Anwendungen (S/4HANA): Falsches Auswerten der Planvariante" from PA-PA-XX component is there in the backend system.

The note will prevent an issue in many HCM Fiori apps that the active plan version can accidently swap over into other plan versions, if they share the same number ranges.
