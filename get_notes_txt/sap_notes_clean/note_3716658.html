Symptom

The user cannot access Joule.

Environment

SAP Ariba Procurement

SAP Ariba Sourcing

Reproducing the Issue

1. Attempt to open the Joule icon in SAP Ariba Procurement or SAP Ariba Sourcing.

2. If you encounter an error, it will state: "Refused to connect"

Cause

The user is attempting to log in with a third-party enterprise account instead of a enterprise user account.

Resolution

To enable user login, Joule requires the integration of the SAP Ariba solution with Identity Authentication, utilizing the IAS setup of the SAP solution.
