Symptom

In transaction SXMB_APPL, you notice that the new report SXMS_PARTNER_WIZARD is not displayed.

Other Terms

SXMB_APPL, report SXMS_PARTNER_WIZARD

Reason and Prerequisites

You use SAP Process Integration as a local Integration Engine in an application system. SAP Note 3721330 provides the new report SXMS_PARTNER_WIZARD ("Wizard: Interface-Specific Queue Determination"). You have the problem described above.

Solution

Implement the correction instructions or import the relevant Support Package.
After you implement the corrections, in transaction SXMB_APPL in the navigation tree in the path
"Integration Engine" -> "Queue Configuration" -> "Interface-Specific Queue Determination"
the new node "Wizard: Interface-Specific Queue Determination".
