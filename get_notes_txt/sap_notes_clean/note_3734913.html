Symptom

Text format in the visit summary is not the same as in the visit notes field. Numbers and bullets mentioned in notes of visit are missing in the generated visit summary.

Environment

SAP Cloud for Customer

Reproducing the Issue

  1. Go to the Visit Work Center.
  2. In Visit View, search for Visit “XXXX” (where XXXX represents Visit ID).
  3. Open Visit XXXX.
  4. Go Notes facet of Visit XXXX. Text in notes holds Numbering 1,2,3 and Bullets. For example as follows :-
              1. Test
              2. Test
              3. Test
        CCCCCC.
              • aaaa
              • bbbb
              • cccc
  5. Click on Action.
  6. Select Generate Summary Option from the Action.
  7. Go to the Attachment Facet of Visit XXXX.
  8. Select and open the Visit Summary PDF.
  9. Text reflected in the summary is without Numbers and Bullets
              TestTestTest
               CCCCCC. aaaabbbbcccc

Cause

Cause

The template for the Visit Summary is created using the Adobe LiveCycle Designer version 9.0 or lower.

The bullets/numbering is supported if the template is modified using Adobe Livecycle Designer Version 10.0 or higher

Resolution

Use Adobe Livecycle Designer Version 10.0 or higher to create/modify Template for Generating Visit Summary.

This guarantees the inclusion of numbers and bullets from the notes of Visit XXXX in the Visit Summary.

Please follow KBA ID 2308038 to enable the feature.
