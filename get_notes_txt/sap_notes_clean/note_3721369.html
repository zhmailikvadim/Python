Symptom

While running the SAF-T Norway version 1.30 in SAP Document and Reporting Compliance (DRC), "Report generation aborted." error message occurs in the Logs.

Related runtime error (ST22):

Category                       ABAP programming error
Runtime Errors              COMPUTE_INT_ZERODIVIDE
Except.                         CX_SY_ZERODIVIDE
ABAP Program               CL_SAFT_NO_DRC_ACCOUNT_GLE====CP
Application Component   FI-LOC-SAF-NO

 Short Text
     Division by 0 (type I or INT8)

Environment

  • SAP S/4HANA
  • Norway

Reproducing the Issue

  1. Execute the Norway SAF-T Report Version 1.30 in DRC.
  2. Observe error with the message: Report generation aborted.

Cause

Missing parameter: NO_SAFT_MAX_DOCS_IN_FILE 

Resolution

  1. Ensure the latest versions of the SAF-T Norway SAP Notes are implemented for the DRC solution.
  2. Verify the parameter NO_SAFT_MAX_DOCS_IN_FILE is correctly added under Transaction SM34 > SRFVC_RPG_ENT > Select NO_RPG_ENT > NO_SAFT_ACCOUNTING > Specify values for parameters specific to the report category.
