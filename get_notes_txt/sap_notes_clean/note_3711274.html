Symptom

When using the Mass Change function in Find Maintenance Orders to technically complete multiple orders that contain permits requiring issuance prior to completion, only the first order is successfully completed. The remaining orders fail with an error message indicating that the necessary permits must be issued before the order can be completed.

Other Terms

F2175, Permit, PERMIT_INIT_AFTER_POST, lif_mass_action_strategy~execute, BELGNTAB, INITFLG

Reason and Prerequisites

Program error

Solution

Implement the advanced note
