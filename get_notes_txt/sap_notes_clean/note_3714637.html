Symptom

  • An error occurs when uploading Intraday Bank statements, resulting in a short dump CX_AMDP_EXECUTION_FAILED.
  • The issue prevents the loading of Intraday Bank statements into the system, impacting business processes.

Environment

SAP S/4HANA 2023

Reproducing the Issue

  1. Attempt to upload an Intraday Bank statement into the system.
  2. Observe that the process fails, resulting in a short dump.
  3. Error CX_AMDP_EXECUTION_FAILED raised

Cause

The issue is related to the missing number range for object FCLM_CR.

Resolution

  1. Maintain the number range for object FCLM_CR with number range number 01 using transaction SNRO.
  2. Ensure the number range is set between 0000000001 and 9999999999.
  3. Re-test the scenario to confirm resolution.
