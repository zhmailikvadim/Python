Symptom

You are getting the following error while fetching advices using RFDP_ADVICE_V2: 'An exception has occurred in class "CX_SY_OPEN_SQL_DATA_ERROR". This exception was not caught in procedure "GET_ORIG_TRANS_BYPSP_ID" "(METHOD)".

Other Terms

Chargeback, transaction reference, RFDP_ADVICE_V2, SAP_FIN_AR_DP_ADV_JOB_TMPL_V2

Reason and Prerequisites

The transaction reference is longer than 40 characters.

Solution

If the transaction reference is longer than 40 characters, a warning is triggered indicating that the chargeback cannot be processed automatically and must be handled manually.
