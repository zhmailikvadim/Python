  • RFC server does not start.

Environment

SAP Data Quality Management, version for SAP Solutions

Cause

The Data Services Access Server has SSL setup and the RFC configuration parameter <AccessServerSSL>false</AccessServerSSL>  is set to false, not allowing connection.

Resolution

Set the <AccessServerSSL>false</AccessServerSSL> parameter to true in the flrfcsrv.xml configuration file.
