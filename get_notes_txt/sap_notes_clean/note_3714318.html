Symptom

The C4C webservice interface MaterialReplicationInitiatedByExternalIn fails in C4C with this error: "Description already exists in language ABC".

Environment

  • SAP Cloud for Customer
  • SAP S/4 HANA 

Reproducing the Issue

  1. Go to the Administrator work center.
  2. Go to the General Settings.
  3. Open the Web Service Message Monitoring under System Administration.
  4. Select query All
  5. Select the Material Replication message.
  6. Click in "Error log".
  7. See the erroneous message "Description already exists in language ABC".

Cause

The issue occurs because the payload contains multiple descriptions for the same language (ABC) for a single material ID. The system processes the first description and fails on the subsequent ones due to duplication.

Resolution

Fix the text in the S4 material master data to ensure that there are no duplicate entries for the same language.
