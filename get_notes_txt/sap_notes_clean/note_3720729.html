Symptom

When creating a contract with duplicate partner functions at the header level, upon saving the contract system throws an error 'Partner function is not defined at item level' in the Maintain ACM Trading Contract Fiori application.

Replication Steps:
1. Launch the Maintain ACM Trading Contract Fiori application.
2. Input an existing contract number and click the Copy Contract button to create a new record.
3. Navigate to the Partner tab at the Header level on the new screen.
4. Click the Add icon. Enter a duplicate partner function with the required details and click the Add button.
5. Complete all mandatory fields at the Item level.
6. Trigger the Save action. The system throws an error "Partner function is not defined at item level".

Other Terms

Agricultural Contract Management; ACM; Maintain ACM Trading Contracts fiori application; Partner Function; Partner

Reason and Prerequisites

Program error.

Solution

Import the relevant support pack or apply the correction instruction(s).
