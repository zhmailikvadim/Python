Symptom

When generating the "Canada GST and RITC Reporting" (CA_GST_HST_RITC_RETURN) report in SAP S/4HANA Document and Reporting Compliance (DRC), you notice that export transactions from Canada, often associated with tax code C1, are missing from the report output. Upon inspecting the financial documents, you find that a US tax jurisdiction code has been assigned to these transactions, even though the destination country is outside of the US.

Environment

SAP S/4HANA

Reproducing the Issue

  1. Open Document and Reporting Compliance (Statutory Reports).
  2. Select the report CA_GST_HST_RITC_RETURN for Canada GST/HST and RITC reporting.
  3. Choose the relevant year and period.
  4. Run the report where transactions include tax code C1.
  5. Observe that export items with tax code C1 are not listed in the report output.

Cause

The "Canada GST and RITC Reporting" report is designed to select financial documents based on a predefined list of relevant Canadian federal tax jurisdiction codes. If a transaction, such as an export from Canada, is incorrectly assigned a tax jurisdiction code from another country (e.g., the US), the report’s standard data selection filter will exclude it, causing it to be omitted from the output.

Resolution

To ensure the report includes these transactions, you must add the non-Canadian tax jurisdiction code to the list of codes that the report considers relevant.

  1. Navigate to the following path in the SAP Customizing Implementation Guide (IMG):
    Cross-Application Components > General Application Functions > Document and Reporting Compliance > Country/Region-Specific Settings > Canada > Statutory Reporting > Generating Goods and Services Tax/Harmonized Sales Text (GST/HST) Report > Specify Federal Level Tax Jurisdiction Code
  2. In this table, add a new entry for the US tax jurisdiction code that is appearing on your export documents.

By adding the code to this configuration table, you instruct the DRC report to include documents with this specific jurisdiction code in its data selection process for Canadian tax reporting.
