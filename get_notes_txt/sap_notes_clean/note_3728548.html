Symptom

Service order can be released via the order API, but retrieving the order details fails with the message: HTTP 400: Order xxxx is a SERVICE_ORDER that is not supported by this API

Environment

SAP Digital Manufacturing 2602. 

Reproducing the Issue

Attempt to GET service order details using Order API fails with the error message: "Order XXX is SERVICE_ORDER that is not supported by this API."

Cause

Yes, currently only release API supports Service Order.

Resolution

Currently, order API does not support retrieving service order

Keywords

service order, API error, GET endpoint failure, /v1/orders
