Symptom

There is a requirement to assign multiple test users within the same Test Plan/Test Process in the Test Automation Tool.

Environment

SAP S/4HANA Cloud Public Edition

Resolution

  1. Open the Manage Your Test Processes app.
  2. Navigate to the required test process where additional test users need to be assigned.
  3. Go to the Process Steps tab and click the Edit button available at the bottom right corner.
  4. Navigate to the required process steps.
  5. Under the Role column, select a new role or change the existing role that is associated with the test user in the Test Your Processes app.
  6. By assigning different roles to the process steps, multiple test users can be maintained within the same test plan/test process.
