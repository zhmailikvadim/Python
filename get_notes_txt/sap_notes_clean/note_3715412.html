Symptom

  • Clarification is sought on whether SAP Commerce Cloud's connector has an out-of-the-box (OOTB) capability to call the OR08 API for tax confirmation during the refund process.
  • The feature "Pause the confirmation of refunds for tax recalculation" is requested to be enabled in SAP Commerce Cloud connector.
  • Connector version v7.0.3 and mmp-sdk-front-9.4.0-jar-with-dependencies.jar are currently in use.
  • The provided link for SDK versions results in "Page not found".

Environment

SAP Marketplace Platform by Mirakl.

Reproducing the Issue

  1. Attempt to use SAP Commerce Cloud connector version v7.0.3 to call the OR08 API for tax confirmation during the refund process.
  2. Verify whether the feature "Pause the confirmation of refunds for tax recalculation" is enabled in the connector.
  3. Access the provided SDK versions link and observe the "Page not found" error.

Cause

Resolution

Please try the following steps (Mirakl Platform configuration):

  1. In Mirakl back office, go to Settings > Orders > Orders.
  2. Open the Workflow setting tab.
  3. In Tax recalculation triggers > Refunds, enable Pause the confirmation of refunds for tax recalculation.
  4. Choose whether it applies to:
    • Only partial refunds of an order line, or
    • All refunds of an order line
  5. Set the time limit after which refunds are automatically accepted even if taxes weren’t recalculated.
  6. Click Save.
