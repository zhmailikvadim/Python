Symptom

  • During patching ABAP SAPUI5 hitting error like below:

    The transport based correction (TCI) '' of SAP note '' is not available

Environment

ABAP SAPUI5

Reproducing the Issue

For example like this KBA, apply note 3710262.

Cause

transport based correction (TCI) missing.

Resolution

  1. Take this KBA as example, SAPK758015CPSAPUI is not avaible for note 3710262.
  2. go to https://me.sap.com/softwarecenter
  3. switch to tab "Support Packages & Patches"
  4. put SAPK758015CPSAPUI  in "Search in Software Downloads" area.
    note there are two search area, it is NOT the search in the top
  5. download, upload package to system using SPAM--load packages and you can initiate note implementation in SNOTE for note 3710262.

Keywords

The transport based correction, not available
