Symptom

When you try to create a time event in the time sheet UI or through Manage Clock in Clock out, you receive an internal error

Environment

  • Employee Central
  • Time Tracking

Reproducing the Issue

  1. Go to timesheet for a CICO employee
  2. Create a time event
  3. Internal error is received

Cause

Desynch of time events

Resolution

  1. Check tool
  2. Application Clock in Clock out
  3. Run all checks
  4. Events with issues are highlighted in "TimeEventTypeSynchronized"
  5. Run quick fix
  6. If the quick fix does not process events highlighted, go to manage data
  7. Verify if objects are present for time events highlighted and recreate them with the title and code from Configure Clock in Clock out

Keywords

Internal, CPI, error
