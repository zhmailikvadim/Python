Symptom

No dedicated error message is provided in case a user executes program CRM_EOP_INDEX_REBUILD without authorization. Message 022 of message class R111 is displayed in case the user is neither authorized via authorization object B_BUP_PCPT nor via transaction code CRMC_EOP_IDX_REBILD2.

Other Terms

CRM_DPP, 103

Solution

Apply both correction instructions. A new dedicated message is used in case the user has none of the above mentioned authorizations.
