Symptom

In transaction IE03/IH08, the system shows the Master Equipment when you search for Equipment via the tab ‘Find Site Equipment’.

Other Terms

S/4HANA Defense and Security, IE03, Find Site Equipment,IH08

Reason and Prerequisites

Coding Error

Solution

This note will remove the tab ‘Find site equipment’ as to search for Site/Critical equipment the tab ‘Find critical/site equipment’ should be used.

Implement the attached correction instruction and perform the manual activity to apply the changes.

Please perform the manual activity for text translations.
