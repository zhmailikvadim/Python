Symptom

  • Delayed value selection in Input Control, with the dropdown collapsing immediately after choosing a searched item in Web Intelligence.
  • It often requires 6–7 attempts for a successful selection.

Other Terms

Webi, Input Control, launchpad

Reason and Prerequisites

 Environment:

  • SAP BusinessObjects Business Intelligence Platform 4.3 Service Pack 3 Patch 12
  • Web Intelligence (Webi)

Step to Reproduce:

  1. Login to BI launchpad.
  2. Navigate to Applications > Select Web Intelligence.
  3. Select any data Source (Universe).
  4. Create a table with some objects in query panel and run the query.
  5. On Web Intelligence document, create Input Controls.
  6. From View mode, select the input control.
  7. Search for a value and try to select it by clicking on it.
  8. Input control dropdown list collapse.
  9. After 6-7 attemps of selection, the value gets selected.
