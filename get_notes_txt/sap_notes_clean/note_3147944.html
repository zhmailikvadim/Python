Symptom

 This coding change is needed to activate a performance optimization for Create Deep calls via OData service.

Other Terms

API_SALES_ORDER_SRV

API_SALES_CONTRACT_SRV

Reason and Prerequisites

When processing the payload from the incoming OData call, internally the so-called GOAL framework is called for each entity line-by-line.

Solution

Apply the given code changes. Please regard also note 3147478.

By this, the GOAL framework will be called in a bundled way for each entity which has no sub-entities (e.g. for pricing entity or partner entity this bundling is effective, but NOT for header or item entity).

All lines of a given entity (which has no sub-entities) will be processed in a bulk way all at once.
