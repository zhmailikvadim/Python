Symptom

You use the database management system SAP HANA and SYS.M_FEATURE_USAGE reports usage of deprecated component SYSTEMVIEW feature M_TENANTS.

Solution

Do not select from this view.

This view had always been empty. It will be removed without replacement.

It was initially planed to be used for multi tenancy, but when the HANA multi tenancy solution MDC (multi database containers) was implemented another view (M_DATABASES) was implemented instead.
