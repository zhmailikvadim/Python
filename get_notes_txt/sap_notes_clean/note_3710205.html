Symptom

Enable Business Partner is not showing in EDOC_EU_PART_LOOKUP where as it shows in SM30 on table EDOEUBUPAV

Environment

SAP S/4HANA

Reproducing the Issue

  1. Go to transaction code EDOC_EU_PART_LOOKUP.
  2. Enter the details
  3. Click "Run Report".
  4. Click "Look up for suppliers".

Resolution

  • This is expected system behavior. The report updates the table using direct ABAP logic and does not trigger the SAP standard change document mechanism.
  • In contrast, the SM30 transaction uses the Table Maintenance Generator with change logging enabled, which is why changes made via SM30 are tracked.
