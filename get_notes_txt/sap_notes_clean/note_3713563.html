Symptom

Report HFIUTAX_CARD_API does not contain the option to skip certain employees manually after receiveng tax card information from the authority.

Other Terms

 Tax Card; API; Finland; HFIUTAX_CARD_API

Reason and Prerequisites

This SAP Note is only relevant for Finland(HRFI)

Solution

A new function has been added to the ALV screen “Changes to Infotype 0205” in report HFIUTAX_CARD_API.

You can now select one or more rows and use the “Skip PERNR” button to manually exclude the selected personnel numbers (PERNR) from changes to Infotype 0205 (IT0205).

In addition, if a PERNR has a value maintained in IT0205 in the field Former Earnings (EARFE), the system displays a warning message for that specific PERNR.

Implementation

As a general rule SAP recommends that you implement the Support Package specified for your release. However, if you need to install a solution earlier, use the SAP Note Assistant to implement the correction instruction. You can find more information about the SAP Note Assistant in SAP Support Portal, under SAP Note Assistant.
