Symptom

In the process "Metering Point Takeover for Electricity (New MOS View)" (DE_B_WIM_MPTKO_MOSN), when you select the Cancel Process option in the user decision BPEM case at step 0450, the linked parent process "Termination of Metering Point Operation (MOS View)" (DE_B_WIM_TERM_MOS_DSO) is not canceled.

This issue is relevant for the market role: MOS.

Other Terms

WIM, DE_B_WIM_MPTKO_MOSN, DE_B_WIM_TERM_MOS_DSO, dMOS, BM_CLOSE_CLOUD_PROC_FROM_BPEM, /US4G/CL_EXCP_PROC_SOLVE_WIM, /US4G/CL_EXCEPTION_PROC_SOLVE

Reason and Prerequisites

Program error

Solution

Please implement the corrections of this SAP Note or import the relevant Support Package (SP).

After implementing this SAP Note, when you select the Cancel Process option in the user decision BPEM case created at step 0450 in process DE_B_WIM_MPTKO_MOSN, the system cancels the linked parent process as well. You can verify the fix by creating a process, selecting the Cancel Process option in the BPEM case at step 0450, and checking that the linked parent process status changes to Completed.
