Symptom

  • Lists steps to enable the search feature from custom columns of an external table in the transaction research widget of a dashboard.
  • Provides mandatory fields/columns necessary in the custom table to utilize the feature in the widget.
  • Enabling the feature requires custom setup and assistance from support.

"Image / Data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental" 

Environment

  • SAP Sales Performance Management
  • Incentive Management
  • Incentive Management on Hyperscalers

Reproducing the Issue

  1. From the Application dropdown, select 'My Incentive Dashboard Setup'.
  2. Create a new dashboard for title or plan.
  3. Create a Configure Transaction Research widget.

Resolution

  1. Ensure that the schema named EXT is present and custom tables are created (e.g., EXT.SAMPLE_WIDGET_TABLE) within the EXT schema.
  2. Ensure the tables contain at least three necessary columns: POSIITONSEQ, PAYEESEQ, and PERIODSEQ.
  3. Add any additional columns in the table to serve as displayable, configurable, or searchable fields in the widget.
  4. Insert rows into the tables to display fields in the Transaction Research widget configuration.
  5. Add the names of the tables created in the EXT schema to the CSI_EXTTABLESALLOWED_WIDGET table within the TCMP schema.
    1. Raise a Support Case to have the reference added in the above TCMP table - SQL to be provided in the case.
    2. Sample SQL - INSERT INTO TCMP.CSI_EXTTABLESALLOWED_WIDGET values (<Tenantid>,<Table_name>,current_timestamp);
    3. Support will enable the feature flag SalesPortal.transactionWidgetDisplayExternalSchemaTables.enabled for the environment.
  6. Verify the functionality of the External Schema Tables dropdown in the Transaction Research widget after completing the above steps.
    1. Note: Three mandatory columns mentioned in step 2 cannot be searched. Only the columns mentioned in step 3 can be used as displayable columns, searchable columns, or filters.
