Symptom

When users try to simulate a wave release that is two-step picking relevant the error below is raised:

"Wave &1 &2 has a 2-step release process, simulation not possible.
Message no. /SCWM/WAVE309"

Environment

  • SAP Extended Warehouse Management (SCM-EWM);
  • SAP Extended Warehouse Management Wave Management (SCM-EWM-WAV);
  • SAP S/4HANA.

Reproducing the Issue

  1. Navigate to transaction /SCWM/WAVE.
  2. Attempt to simulate a wave relevant for two-step picking.
  3. Error

Cause

Standard behavior.

Resolution

Simulation of two-step Waves is not possible in S/4 HANA versions older than 2022, as outlined on SAP Note 2783735 - EWM Wave Release Simulation.

It is not possible to downport the wave simulation functionality for older S/4 HANA versions due to system limitations.

See Also

SAP Note 2783735 - EWM Wave Release Simulation
