Symptom

In the process "End of Supply for Gas (DSO View)" (DE_B_GELIGAS_EOS_DSO), step 8940 triggers the outdated version 2310 of the linked Start of Backup Supply (SBS) process instead of the latest version 2604.

This issue is relevant for the market role: DSO.

Other Terms

End of Supply, Gas, GPKE, DE_B_GELIGAS_EOS_DSO, DE_B_GELIGAS_DSO_SBS, Backup Supply, linked process version, 2310, 2604, step 8940

Reason and Prerequisites

The error occurred because version 2310 was hardcoded in the linked process version field at step 8940.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the system triggers the latest version (2604) of the process "Start of Backup Supply for Gas (DSO View)" (DE_B_GELIGAS_DSO_SBS) from the End of Supply process. You need to retrigger the Start of Backup Supply step. You can verify the fix by checking that the latest version of the SBS process (2604) is triggered.
