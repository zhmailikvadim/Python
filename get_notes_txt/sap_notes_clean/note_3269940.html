Symptom

 During UI Logs transfer to ETD you see that some records were transfered with duplicates.

Other Terms

UIDP, UI Logging, Transfer to ETD

Reason and Prerequisites

Program error

Solution

To avoid duplicates we provide new timestamp handling by rounding Up Timestamp FROM and rounding Down Timestamp TO.

Implement the automatic correction instruction or implement the Support Package SP04 of SWC UIDP if available.
