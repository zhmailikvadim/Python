Symptom

DTP source aDSO is "Compress Data" type aDSO, there are two records with same key values in the aDSO inbound table and activation table, and DTP target is an infoobject master attribute.

After a full DTP request, can see the record in the activation table is loaded to target infoobject master data.

Environment

  • SAP BW/4HANA 2.0
  • SAP BW/4HANA 2021
  • SAP BW/4HANA 2023 

Cause

SAP program error

Resolution

Implement SAP Note 3452694 – Packetizer shall use extraction view of source ADSO.

Keywords

New table, Packetizer, compressed, uncompressed data, semantic group, package, Data Mart
