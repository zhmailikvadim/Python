Symptom

Under the list of fields available in Manage Recruiting Groups, Position field is not visible.

Can this be configured?

Environment

SAP SuccessFactors Recruiting

Reproducing the Issue

Goto Manage Recruiting Groups >> under people pool category you will see a list of fields displayed in which Position field is not displayed.

Resolution

The list of fields available from the Succession data model which are visible under Manage Recruiting groups is given in this KBA .

You can submit an enhancement request to include this standard field in the list which can be considered in the future releases.
