Symptom

Invoice attachments are not viewable in the Ariba UI and certain file types appear as unsupported or require downloading to open.

Reproducing the Issue

  1. Create or open an invoice that contains an attachment.
  2. Ensure the attachment is uploaded using a file type not included in the parameter value. 
  3. Observe that the attachment does not display inline and instead shows as unsupported or requires downloading.

Cause

The file type of the invoice attachment is not included in the parameter Application.Invoicing.ViewableInvoiceAttachmentExtension, causing the system to treat the attachment as non‑viewable and preventing it from opening inline in the UI.

Resolution

  • Add the required file extensions to the parameter Application.Invoicing.ViewableInvoiceAttachmentExtension so the system can recognize and display those attachment types as viewable.
  • After updating the parameter, Save the changes and allow the system to refresh; newly uploaded attachments with the configured extensions will then display inline in the Ariba UI.

Keywords
