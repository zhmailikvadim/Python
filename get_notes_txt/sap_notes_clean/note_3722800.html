Symptom

When trying to build with a version in preview release it throws an version unknown error, as the following:

Exception in thread "main" de.hybris.ccv2.corecommerce.version.supported.VersionNotSupportedException: Version [xxxx] is unknown. Supported versions are: 2211.43, 2211.44, 2211.45, 2211.46, 2211.47, 2211.48, 2211.49, 2211-jdk21.1, 2211-jdk21.2, 2211-jdk21.4, 2211-jdk21.5, 2211-jdk21.6, 2211-jdk21.7 

NOTE: In the example, ‘xxxx’ is just a placeholder. In the actual log, it will display the preview release version. 

Environment

SAP Commerce Cloud

Reproducing the Issue

  1. Download the Commerce Cloud distro regarding the preview release version from the Software Center.
  2. Update the manifest.json file with "commerceSuiteVersion": "{version}".
  3. Execute the standard build process.
  4. Observe the validation error indicating that the version is unknown.

Cause

The 'commerceSuiteVersion' property is intended for public released versions. The unknown exception is thrown due to a misconfiguration in the manifest.json file, because the incorrect property was used.

Resolution

Refer to Preview Releases - Implementation.
