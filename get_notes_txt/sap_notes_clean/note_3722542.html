Symptom

  • An alert is found in SAP HANA Cloud instance: "Configuration parameter [trace]/sql set to unsupported value."
  • The alert indicates the configuration parameter trace/sql is set to an incorrect value.

Environment

HANA Cloud Services HANA Cloud

Cause

The issue might have occurred due to the execution of the following incorrect statement in the past:
ALTER SYSTEM ALTER CONFIGURATION ('indexserver.ini', 'DATABASE') SET ('trace', 'sql') = 'on';

instead of the correct one:
ALTER SYSTEM ALTER CONFIGURATION ('indexserver.ini', 'SYSTEM') SET ('sql', 'trace') = 'on';

Resolution

Unset the incorrect parameter configuration using the following statement:
ALTER SYSTEM ALTER CONFIGURATION ('indexserver.ini', 'DATABASE') unset ('trace', 'sql');
