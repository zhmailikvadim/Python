Symptom

When using Git / GitHub for Version Management the CMS and VMS version of the objects do not update.

Environment

  • SAP BusinessObjects Business Intelligence (BI) 4.x
  • Git / GitHub

Reproducing the Issue

  1. Configure Git / GitHub for Version Management using the KBA 3335429 - Configure Git / GitHub for version management in BI Platform 4.x? [How To]
  2. Login to CMC, navigate to Version Management, select any infoobject,  click on the 'Add to VM' button.
  3. The CMS version and VMS version does not update and stays 0.

Cause

Workspace Directory path and Repository name both contain _ or - or space.

Resolution

This is a limitation: 

When configuring Git / GitHub for Version Management, we need to use simplified paths without space _ or - for "Workspace Directory" and "Repository name" fields.
