Symptom

After implementing the SAP Note 3704238, you manually update the ICSC file path URL when the year changes. In case when URL path is incorrect, system do not display any error message to the user.

Other Terms

  • CMT
  • ICSC
  • Salary scale
  • Pensionable remuneration
  • T7UNCMT_ICSC_URL 

Reason and Prerequisites

Program error.

This SAP Note is relevant only for non-profit organizations.

Solution

After applying this SAP Note, the system reads the URL path based on the processing year from table T7UNCMT_ICSC_URL. The system displays error message in case URL path maintained is incorrect.

To receive the changes, import the Support Package specified for your release.

Alternatively, implement the attached correction instructions.

As a general rule, SAP recommends that you install a solution by implementing the Support Package specified for your release. However, if you need to install a solution earlier, use the SAP Note Assistant to implement the correction instruction. You can find more information about the SAP Note Assistant in SAP Support Portal, under SAP Note Assistant.
