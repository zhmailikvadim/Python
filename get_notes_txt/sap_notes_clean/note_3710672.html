Symptom

The option to use HDM is not available in transaction ACACTREE02.

Other Terms

ACAC, ACE, ACACTREE02, HDM, GOS, DMS, AL, document, attachment

Reason and Prerequisites

Reason:
New functionality.

Prerequisite:
Ensure that the HDM framework is activated in the SAP Fiori apps for Accruals Management.

Solution

This SAP Note introduces HDM possibility for transaction ACACTREE02.

To apply this change and enable HDM:

1) Implement the attached Correction Instruction or the specified Support Package.

2) As a prerequisite, you must configure HDM for manual Accrual Objects according to the instructions in the following SAP Notes:

#3521117 - F5423 Manage Manual Accruals - HDM enablement [BE part]
#3521118 - F5423 Manage Manual Accruals - HDM enablement [FE part]

3) After the prerequisites are met, enable the HDM functionality in the GUI environment by ensuring
the BAdI implementation ACE_HDM_ATT_APP of BADI_HDM_STANDALONE_APP is active.
Alternatively, you can use your own implementation with the filter SAPOBJECTTYPE = 'AccrualObject'
and set the CV_ENABLE flag to 'X' (true) within the ENABLE_STANDALONE_SERVICE method.

The Manage Attachments option will then be available under the Generic Object Services menu in the transaction.
