Symptom

Value help for company code shows different result for Cash position and cash position details app respectively.

Environment

  • S/4HANA
  • S/4HANA Finance
  • S/4HANA Cloud

Reproducing the Issue

Open Cash position details app -> Click the F4 help for company code field.

Open the cash position app -> Click on Company code field.

Resolution

In Cash position app only valid data and its corresponding master data is displayed. So company code appear in the cash position is company code with valid flow data.

And in Cash position details app  all valid master data is displayed. It is normal that one will find more company code in CPD than in CP.

Keywords

Cash position details, Cash position, Company code, Value help, F4, S/4 Hana, Cash Management
