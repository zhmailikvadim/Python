Symptom

When open a project, the WBS element or other objects are missing, there is no authrization check fails error, no ACL or transaction variant used.

When try to create like WBS in Cj20n, the option is greyed out.

The issue may occur only for specific user.

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.  

Environment

  • SAP ERP Quality Management (QM)
  • SAP R/3
  • SAP R/3 Enterprise
  • SAP ERP Central Component
  • SAP ERP
  • SAP Enhancement package for SAP ERP
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA, on-premise
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Start transaction CJ20N (Project Builder).
  2. Open one project.
  3. WBS element is missing in project builder which does exist for the project.
  4. Attempt to create a new WBS and observe that the Create WBS option is greyed out as well.

Cause

A user-specific setting in CJ20N Options, the relevant missing objects are not checked.

Resolution

  1. In CJ20N, go to the menu bar and choose Settings > Options.
  2. In the Options dialog, tick the relevant options which are missing in Cj20n.
  3. Save the settings.
