Symptom

In the SAP Fiori 2.0 app "My Overtime Requests", the button for selecting the employee in whose name you do not want to act ("On Behalf of" function) appears.

Other Terms

Version 1.0.142, F2456, HCMFAB_COMMON, My Overtime Requests, F4939, HCMFAB_OTQ_MAN, UIHR002

Reason and Prerequisites

SAP Note 3212767 introduced the new parameter EV_USE_ONBEHALF_BACKEND_SEARCH in the BAdI HCMFAB_B_COMMON. The symptom occurs as soon as it is set to 'X'.

Solution

Import the relevant Support Package or implement the corrections using the correction instructions.
