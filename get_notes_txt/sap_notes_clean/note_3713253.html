Symptom

Service call priority cannot be deleted from "Settings and Configuration".

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

SAP Field Service Management

Reproducing the Issue

  1. Open the FSM Shell (https://de.fsm.cloud.sap/shell/)
  2. Navigate to 'Settings and Configuration'
  3. Go to 'Planning and Dispatching'
  4. Enter the menu 'Service Call'
  5. Check 'Service Call Priorities'

Actual Behavior:
Not able to remove the Service Call Priorities via the UI.

Expected Behavior:
Able to remove the Service Call Priorities via the UI.

Cause

On the UI there is no option to delete Service Call Priorities.

Resolution

There is a possibility to delete the 'Service Call Priorities' via API request.

  1. Open FSM 'Admin'

  2. Navigate to Company

  3. Go to 'Query API'

  4. Perform the following query to list all 'Service Call Priorities': 
    SELECT e FROM Enumeration e WHERE e.enumType = 'SERVICE_CALL_PRIORITY';  

  5. Identify the 'Service Call Priorities' that need to be removed

    Sample:
      SELECT e.code, e.enumType, e.id FROM Enumeration e WHERE e.enumType = 'SERVICE_CALL_PRIORITY' and e.code = 'Test Prio';  


    Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

  6. Copy the record ID

  7. Open the a API-Client like Postman, Insomnia, Bruno ect.

  8. Get data using GET Method.

    GET: https://{cluster}.fsm.cloud.sap/api/data/v4/{resourceName}/{resourceId}?dtos={resourceName.dtoVersion}

    Sample  https://{{cluster}}.{{baseUrl}}/api/data/v4/Enumeration/BA3B709BEF9146C2A83D30354462B4A2?dtos=Enumeration.13


    Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

    CURL:

        curl --request GET \
      --url 'https://de.fsm.cloud.sap/api/data/v4/Enumeration/BA3B709BEF9146C2A83D30354462B4A2?dtos=Enumeration.13' \
      --header 'x-account-id: 98765' \
      --header 'x-client-id: Bruno' \
      --header 'x-client-version: 1' \
      --header 'x-company-id: 123456'



  9. DELETE the data
    Use the Method DELETE.

    Sample:  https://{cluster}.fsm.cloud.sap/api/data/v4/{resourceName}/{resourceId}?lastChanged=1234567890



***Please note the default Service Call Priorities are hardcoded on mobile app.***
