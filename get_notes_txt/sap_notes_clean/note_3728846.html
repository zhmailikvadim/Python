Symptom

From OP2023 onward, it is no longer necessary to load pre-delivered template migration objects from client 000 into the target client. The report CNV_PE_MC_LOAD_OBJECTS has been adjusted to reflect this change.

Other Terms

Migration Cockpit, Load Objects, CNV_PE_MC_LOAD_OBJECTS

Reason and Prerequisites

The SAP delivery logic of migration objects has been adjusted. Migration objects delivered by SAP are now directly available in the target client, so manual updates of template migration objects are no longer required.

Solution

Install the attached correction. If you attempt to execute CNV_PE_MC_LOAD_OBJECTS, the program will show the informational message and exit without processing.
