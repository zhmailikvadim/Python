Symptom

  • I accidentally marked an Event as complete, how can I undo it?
  • Is it possible to reverse Mark as Complete for an Event if Supplier has done it by mistake?

Resolution

Once an Event is marked as complete, it will be pending approval.

You can request the approver to reject the Event for resubmission.
