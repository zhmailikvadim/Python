Symptom

Options in the maintenance planner are all grayed out, preventing the creation of a maintenance plan.

Environment

  • Product: SAP NetWeaver
  • Maintenance Planning Tools

Reproducing the Issue

Log in to the maintenance planner.

Cause

The user has no authorizations.

Resolution

Accoridng to Maintenance Planner user guide, the user needs to be checked.

=====

The following authorizations must be fulfilled before you can use Maintenance Planner:

  1. Ensure that you have an s-user for the SAP Support Portal.
  2. Your user in SAP Solution Manager must be assigned to your s-user for the SAP Service Marketplace. For more information, see SAP Note 1822202.
  3. For the View Recommended Notes section, you need the following roles, depending on your requirements:

Roles

Authorization

INSTDISP (Display System Data)
    • Read-only access to the View Recommended Notes section.
    • Cannot view security notes.
    • Cannot change the processing status of notes.

INSTPROD (Edit System Data)
    • Access to the View Recommended Notes section.
    • Cannot view security notes.
    • Can change the processing status of all the notes (except security notes).

S4M_MP_SEC (View Security Notes)
    • Access to the View Recommended Notes section.
    • Can view security notes.
    • Can change the processing status of all the notes.

Note

  • You can access security notes if you already have the SECCONTACT role assigned.
  • To request authorization, search and contact your administrator from this page.
