Symptom

In BD97, after selecting the entry to be deleted and clicking the delete button, the system displays the error "select an RFC destination" and the corresponding entry cannot be deleted.

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental. 

Environment

  • ABAP Platform
  • SAP NetWeaver
  • SAP Web Application Server for SAP S/4 HANA

Cause

This occurs because the corresponding logical system was not deleted in BD54.

Resolution

First, delete the corresponding logical system entry in BD54. Only then can the entry be deleted in BD97.

KBA 3454115 - Unable to delete grey filter entry in BD64

Keywords

select an RFC destination, BD97, delete, BD54
