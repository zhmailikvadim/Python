Symptom

In case of SAP Business Data Cloud - Test, Demo, and Development (TDD) Partner, the Intelligent Applications and Data Packages shows the following message:

No available applications and packages
There are no applications and packages that can be installed or activated right now.

and/or

The Data Product's lifecycle is Inactive in SAP Business Data Cloud Cockpit / Catalog & Marketplace. 
The following warning message is displayed - This data product is not available for consumption and is disabled.

Environment

SAP Business Data Cloud

Reproducing the Issue

  1. Open the SAP Business Data Cloud Cockpit
  2. Navigate to Intelligent Applications and Data Package

Resolution

Under this channel we cannot handle commercial inquiries directly, as access to sensitive customer data is restricted to the account teams responsible for your organization.  

Latest information:
02.2026: There are no LoB-specific entitlements for partner TDD BDC tenants at this time. 
03.2026: We identified a tooling glitch that incorrectly showed all Data Packages and Intelligent Applications as available in the BDC TDD system, even though the TDD entitlements for those items have not yet been released. We apologize for any confusion this may have caused. A hotfix will be released as soon as possible to correct the displayed availability. If a partner has tentatively installed any of the affected Intelligent Applications, they will receive a notification explaining that they are not entitled to use those applications until the PI TDD license materials are available.

___________________________________________________________________________________________________________________________________________________

To make SAP Intelligent Applications and data packages available, a customer needs to integrate a supported SAP application (such as SAP S/4HANA Cloud, Private Edition, or SAP SuccessFactors) and license the corresponding Intelligence Package - SAP Help Portal Documentation: SAP Business Data Cloud Entitlements.

At the moment, the development team are actively working on extending this capability to Test, Demo, and Development (TDD) scenarios. As an initial step, People Intelligence will be enabled to support the integration of a BDC TDD account with a partner’s SAP SuccessFactors test and demo tenant.

In parallel, they are also preparing a demo data service that will allow SAP data products to be generated using demo data, without requiring a connection to a live SAP system.

Until these enhancements are available, SAP Intelligent Applications and data packages cannot yet be used in a BDC TDD account. We understand that this may be limiting, and we truly appreciate your patience while we work on these improvements.

If you have any further questions or need additional clarification, please review the available resources:

  • BDC TDD Licensing Guide

You can also contact tdd@sap.com and/or visit SAP Partner Portal.
