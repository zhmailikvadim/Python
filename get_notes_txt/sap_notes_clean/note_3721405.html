Symptom

  • Emarsys pop-up windows (e.g. Field Editor) open as separate browser windows.
  • Third-party tools, when used via browser extensions, are only attached to the original window and cannot display content or continue guided flows inside these pop-ups.

Environment

SAP Emarsys Customer Engagement

Reproducing the Issue

  1. Open SAP Emarsys in a browser integrated with a third-party tool via an extension.
  2. Perform actions that trigger pop-ups, such as editing fields in the Field Editor.
  3. Observe that the third-party tool remains attached to the original window and does not interact with the pop-up window.

Cause

  • Third-party tools integrated via browser extensions are designed to attach exclusively to the original browser window.
  • Pop-up windows in SAP Emarsys, such as the Field Editor, open as separate browser windows, which prevents third-party tools from displaying content or continuing guided flows within these pop-ups. 

Resolution

  1. Adjust the workflows of the third-party tool to avoid steps involving SAP Emarsys pop-up windows.
  2. Consult the third-party tool's support team regarding options for multi-window support.

Keywords

pop-up windows, field editor, browser extension, third-party tools, multi-window support, compatibility issue, separate browser windows.
