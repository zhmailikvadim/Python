Symptom

You execute a task of task category “Period Initialization”. Execution of this task generates short dump in case there is an increase of tax rate from zero to a non-zero value in the period of current task execution.

Other Terms

Division by 0 (type P) in program "CL_UC_TASK_EXECUTION_INIT_PER=CP",

Adjustment of deferred taxes, period initialization, zero tax rate,

BCD_ZERODIVIDE, CX_SY_ZERODIVIDE,  CL_UC_TASK_EXECUTION_INIT_PER=CP, CL_UC_TASK_EXECUTION_INIT_PER=CM00M, TAX_ADJ_EXECUTE

Reason and Prerequisites

Program error.

  • You activated the function "Adjustment of Deferred Taxes" in the settings of the consolidation area
  • There is a change of tax rate for the affected consolidation units:
    • Tax rate in previous period was zero
    • Tax rate in current period is non-zero

Solution

Implement the advanced corrections as described in the correction instructions or import the Support Package.
