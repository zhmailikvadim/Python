Symptom

In the DRC report SK_VAT_DCL for additional correction of VAT Return the values in the fields 36 and 37 are incorrectly calculated for the minority scenario.

Other Terms

SK_VAT_DCL, SK_VAT_DCL_V1, SK_VAT_DCL_V1, additional correction, field 36, field 37

Reason and Prerequisites

Program error

Solution

 As a general rule, SAP recommends that you implement the Support Package specified for your release. However, if you need to install a solution earlier, use the SAP Note Assistant to implement the correction instruction. For more information see SAP Note Assistant.
