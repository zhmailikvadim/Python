Symptom

If you use SAP GUI 7.40 for a system with an SAP Basis release lower than 702, the ABAP Editor terminates or the program code is displayed incorrectly in the ABAP Editor.

Other Terms

Syntax highlighting in the ABAP Editor

Reason and Prerequisites

Due to an enhancement of the ABAP Compiler, the ABAP Editor is no longer downward compatible with old releases in the GUI in which code completion is not supported.

Solution

Apply the specified patch for SAP GUI 7.40.
