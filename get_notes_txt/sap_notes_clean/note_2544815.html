Symptom

How to create Questionnaire Survey with or without User details in the survey (Anonymous Survey)

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

SAP SuccessFactors Learning

Resolution

  1. Go to Admin -> Learning -> Questionnaire Surveys -> Click Add New
  2. In the "Evaluation Level:" drop-down, choose "Item Evaluation: User Satisfaction"
  3. Enter all the required details and click save
  4. Under Options select Anonymous Survey:
  • Yes : Admin will not be able to view who submitted the survey
  • No :  Admin can see who has submitted the survey
