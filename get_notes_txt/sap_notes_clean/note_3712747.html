Symptom

In Change Quality Notification app, when trying to fill the material document number via F4 Help button, get below error message :
You do not have authorization for the transaction MB51. 

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Open Change Quality Notification app.
  2. Maintain Notification Type Q2 and press Enter.
  3. Open F4 Help in the Material Document field.
  4. Observe that the error occurs.

Cause

MB51 (Display Material Document List app) is deprecated in S/4ANA Cloud Public Edition, but the navigation target in the quality notification has not been adjusted accordingly yet.

Resolution

SAP is aware of this issue and working on the solution, but we cannot provide a fix on short notice. It will last some time to change the navigation in the value help for material document in quality notification from MB51 to the successor Material Documents Overview app (F1077). 
Currently, the only workaround in this case is to find a certain material document in Material Documents Overview app and copy the specific material document ID manually into the "Material Doc" field in the quality notification.

Keywords
