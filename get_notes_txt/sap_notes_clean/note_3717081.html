Symptom

While executing the report /IBPP/R3_TRANSFER_ORDER_R in the S/4HANA or ERP system to transfer new production or process orders, a runtime error occurs in the connected SAP GTS system.

The runtime error MESSAGE_TYPE_X is raised in program /SAPSLL/SAPLAPI_BOM.

Other Terms

/IBPP/R3_ORD_TRANSF

Reason and Prerequisites

The error is raised if the application log for the transfer is full and no new messages can be added. This occurs if the number of orders to be transferred is very high.

Solution

Implement the attached correction instructions or the corresponding service pack.

After the implementation of the note, the system will no longer transfer all orders at once but will instead create packages of 1,000 orders.
