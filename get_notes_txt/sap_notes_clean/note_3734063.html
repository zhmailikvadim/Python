Symptom

When opening the Data Monitor, you get an error, that "Dimension Y1 does not have a task group (for the Data monitor) in version ZZZ".

Environment

SAP S/4HANA Finance for Group Reporting

Cause

Configuration error.

Resolution

Usually, when the above mentioned error appears, it indicates an inconsistency related to the Period Category and/or Consolidation Frequency.

1. First, check in transaction CXP1 (IMG activity: "Assign Task Group to Version") if the assignment is correct.
2. Then, check Transactions: CXB2 and CXC9. The same Period Category, that is assigned to the relevant version have to be maintained in transaction CXC9 as well and a Consolidation Frequency with the same ID needs to be maintained in transaction CXB2 as well.

Keywords

Dimension Y1, Task Group, Data Monitor, Cons Frequency, Period Category, CXB2, CXP1, CXC9,
