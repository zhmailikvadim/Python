Symptom

You are using MTM versioned logistics and have a Material with Moving Average Price (VPRSV = V) . Since there is no standard implementation to support moving average as contract value in Stock MTM calculation , so you have implemented BADI_CMM_STOCK_GMDA_UPDATE to update the prices to GMDA . Here you notice that CMM_VLOGP-QUOTGROUP is still remaning blank which is unexpected .

Other Terms

QUOTGROUP, CMM_VLOGP, BADI_CMM_STOCK_GMDA_UPDATE , VPRSV = V , Moving Average Price

Reason and Prerequisites

This is a program error .

Prerequisites :

  • You are using material with MBEW-VPRSV = V (Moving Average Price).
  • There is a custom implementation of BADI_CMM_STOCK_GMDA_UPDATE to get the instrument key and load data to CMM_GMDA  table via MATREX.

 

Solution

Implement the attached corrections .
