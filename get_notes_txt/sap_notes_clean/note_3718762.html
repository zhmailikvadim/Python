Symptom

When I attempt to upload Cost Centers, the transaction fails with one or more of the following errors:

  • Error:

    Invalid values for System ID

    Invalid values for Account Category

    Invalid values for Owner

Cause

These errors typically indicate a mismatch between the data in your upload file and the system's configuration.

  1. System ID: The value provided in the upload file (or set as a default in the connector) does not exist in the System ID Pick List. This often happens after an environment refresh where custom pick list values are scrubbed.
  2. Account Category: There is a mismatch between the spelling in your file and the system's localized default. For example, if your company's default language is English (United Kingdom), the system expects "Cost Centre" rather than "Cost Center".
  3. Owner: The username provided in the Owner column is either invalid or does not belong to an active user in the current environment.

Resolution

You must align your environment configuration and upload file headers with the system's requirements.

To Resolve System ID Errors:

  1. Navigate to Admin Configuration > Pick Lists.
  2. Search for the System ID pick list and click View Values.
  3. Ensure the value from your upload file (e.g., EMQCLNT010) is present. If missing, add the value manually or via upload.

To Resolve Account Category Errors:

  1. Verify your Company Details to identify the default language.
  2. Update your upload file to match the expected spelling (e.g., use "Cost Centre" for UK localized tenants).

Note: If using Upload Column Transformation, ensure the "File Column Name" is not identical to the "Native Field Name" if no transformation is actually needed, as this can trigger a "Native column name already exists" error.

To Resolve Owner Errors:

  1. Ensure all usernames in the Owner column are active users in Fieldglass.
  2. If you want the system to use a default owner, you can set a default value in the Upload Column Transform section of the connector configuration.
