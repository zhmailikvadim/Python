Symptom

  • A specific job enters a hung state, and no error messages or error codes are displayed.
  • No updates can be seen in the trace or monitor logs.
  • The job contains a script with an exec function.

Environment

SAP Data Services 4.x

Cause

The job is stuck in an exec command. With flag 8, the exec function waits for the external program to complete, if the operating system command prompts for input or does not complete, SAP Data Services also waits indefinitely. Example:

  • exec('foo.bat', ' ', 8);

Resolution

Consider one of the following actions:

  1. Review the script section where exec is called (for example, commands like running a batch file or moving files) and identify the exact command invoked when the hang occurs.
  2. If appropriate for the job design, revise how exec is used (for example, avoid waiting for completion when not required) and consider using flag 256. Example:
    • exec('foo.bat', ' ', 256);

Review exec | SAP Help Portal documentation for additional information about execution flags.
