Symptom

  • How to import/use BW4HANA object type: Query in Data Services?
  • Is BW4HANA object type: Query supported in DS?

Environment

  • SAP Data Services
  • BW4HANA

Resolution

Currently, BW4HANA object type: Query is NOT supported by DS. Cannot import or use it inside DS datastore or job.

See details in online document: Connect to SAP BW/4HANA about what is supported by DS with SAP BW4HANA.
