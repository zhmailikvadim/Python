Symptom

When I run the ARBCIG_MASTER_DATA_EXPORT program to export supplier master data from the ERP system, the PaymentMethod column in the RemittanceLocationConsolidated.csv file is empty. I expect this column to be populated with the payment method data. This happens for both full and incremental master data loads.

Environment

  • Managed Gateway for Spend&Network
  • SAP Ariba Buying and Invoicing (B&I)
  • SAP ERP Central Component (ECC)
  • SAP S/4HANA

Cause

The standard functionality of the ARBCIG_MASTER_DATA_EXPORT program does not include the export of payment method information to SAP Ariba Procurement solutions. As a result, the RemittanceLocationConsolidated.csv file does not contain information for payment method.

This is explained in Import Supplier Data | SAP Help Portal.

Resolution

This behavior is part of the standard product design and there is no standard fix. If you need payment method data included in the exported file, consider the following options:

  • Engage your internal consulting or development team to implement a custom Business Add-In (BAdI) that populates the payment method information according to your business requirements.
  • Submit an enhancement request to have this capability considered as a standard feature in a future release.
