Symptom

  • During the upgrade to S/4HANA 2025, the Software Update Manager (SUM) process gets stuck in the IS_SELECT phase.
  • The addons UIAPE 102 and UIUCOM 102 are displayed with the status "Undecided," and no selection is possible for these addons.
  • The release notes for the addons (2708496 and 2708497) indicate that the "Keep version" option must be selected, but only "Upgrade" or "Deletion" options are available.

Environment

Downtime Minimization for ABAP

Reproducing the Issue

Repeat the IS_SELECT phase in SUM.

Cause

The stack XML file did not calculate the addons UIAPE 102 and UIUCOM 102, resulting in the inability to select the "Keep version" option during the IS_SELECT phase of the SUM process.

Resolution

  • Download the corresponding ACP files for the addons UIAPE 102 and UIUCOM 102 from the download service portal.
  • Place the downloaded ACP files in the SUM download directory.
  • Backtrack several steps in the SUM process.
  • Retry the SUM process to verify if the "Keep version" attribute is recognized.

For next stack xml file generation, add  "UIs on S/4HANA 2025 (outdated) " under "S/4HANA MASTER DATA GER 2602 " by choosing 'install/maintain the Addons ' after the S/4HANA 2025 is selected in MP.
