Symptom

You've run HANA SQL Stattement HANA_Configuration_MiniChecks" (available on SAP Note 1969700) and following checkID was reported:

M0451    MEMORY    Size of fulltext indexes 

Environment

  • SAP HANA PLATFORM EDITION 1.0
  • SAP HANA PLATFORM EDITION 2.0

Resolution

The Fulltext or fuzzy search indexes can consume significant amounts of memory and due to tokenization they can be significantly larger than a normal database index. 

Thus, you should check with the application teams to what extent these special index types are really required and deactivate large fulltext and fuzzy search indexes in case there is limited benefit.
