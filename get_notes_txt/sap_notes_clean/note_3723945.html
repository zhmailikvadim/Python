Symptom

The Cloud Foundry environment cannot be selected as the runtime while creating a HANA instance.

The system prompts for login to the Cloud Foundry environment to enable selection, but the button remains greyed-out even after logging in.

"Images/data in this KBA are from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Environment

SAP HANA Cloud

Reproducing the Issue

  1. Attempt to create a HANA instance in HANA Cloud Central.
  2. Try to select the cloud foundry environment as the runtime.
  3. Log in to the cloud foundry environment when prompted.
  4. Observe that the button remains disabled even after logging in.

Resolution

1. Sign in to the Cloud Foundry environment when prompted.

2. Refresh the HANA Cloud Central browser tab(not the refresh button within HCC page on the top right corner) after signing in to ensure the latest configurations are updated.

3. Attempt to select the CF runtime environment again.
