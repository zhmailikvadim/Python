Symptom

  • After CE2602 upgrade, result of value 0 cannot be recorded in Record Inspection Results app (F1685A) for quantitative characteristics.
  • Error message QI112 "Incomplete record for confirmation no. &00000000& : no measured value indicated" is raised when saving.

Environment

SAP S/4HANA Cloud Public Edition 2602

Reproducing the Issue

  1. Access the Record Inspection Results app.
  2. Attempt to record a result with a value of 0 for quantitative characteristic.
  3. When saving, system throws error message QI112 "Incomplete record for confirmation no. &00000000& : no measured value indicated".

Cause

This is a regression issue due to the upgrade.

Resolution

SAP is investigating the issue and working on a correction.

Before the correction is finished, as a workaround, use one of the following apps to do the result recording:

  • Record Inspection Results - Hierarchical Worklist (QE51N)
  • Record Inspection Result - For Operation (QE01)
  • Process Inspection Lots - Worklist (QA32)

Keywords
