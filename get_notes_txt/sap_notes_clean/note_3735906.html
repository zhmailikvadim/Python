Symptom

This document explain ways to use ESL token for external events.

Environment

SAP Emarsys Customer Engagement

Reproducing the Issue

  1. Insert the Product ID merge tag in an email or content block to render dynamic content from an external event.
  2. Try different tag variants such as {{ event.sku }}, {{ event.item }}, {{ event.item.item }}, {{ item.item }} and observe that none render the expected Product ID.

Resolution

  • Get a single payload attribute value: You can use token {{event.first.second.third.name}} to get the value “Harvey” from following event payload.
    • payload:

{ 
  “first”: { 
    “second”: { 
       “third”: { 
         “name”: “Harvey”
       }
    }
  }
}

  • Loop payload attribute with multiple values:
    • payload:

{
  "key_id": "3",
  "external_id": "test@example.com"
  "data": {
    "orderId": 1234,
    "orderItems": [
      {
        "productCode": 12,
        "imageUrl": "http://domain.com/12.jpg"
      },
      {
        "productCode": 13,
        "imageUrl": "http://domain.com/13.jpg"
      }
    ]
  }
}

    • html to loop event.orderItems:

<h1>Order ID: {{event.orderId}}</h1>
<ul>
{% foreach order in event.orderItems %}
  <li>
    {{order.productCode}}
    <img src="{{order.imageURL}}" />
  </li>
{% endforeach %}
</ul>

  • You can find Event payload samples here: Event payload samples - SAP Help
