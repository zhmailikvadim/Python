Symptom

  • When users open the Job Profile Acknowledgement task, in a system with Talent Intelligence Hub enabled, all skills display with proficiency "level 0" instead of the defined levels.

Environment

SAP SuccessFactors Recruitment
SAP SuccessFactors Talent Intelligence Hub

Reproducing the Issue

  1. Log in to the instance.
  2. Open the Job Profile Acknowledgement task.
  3. Observe that all skills show proficiency "level 0" even though those levels are not correct, even if there is no Level 0 in the Proficiency scale as it goes from 1-5.

Cause

Known issue in Talent Intelligence Hub-enabled instances.

Is not present in "Job Profile Builder" instances with "Manager Center of Capabilities".

Resolution

The solution of this issue is currently under investigation. Please follow this KBA for updates. 

Workaround

Change the Job Profile approval workflow to only include acknowledging the Job Profile changes.
