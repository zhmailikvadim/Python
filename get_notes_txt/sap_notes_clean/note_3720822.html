Symptom

  • Request for obtaining a list of Classic Data Analyzer Insights that will be deprecated in the SAC system.
  • The standard "Overview" story found under the "System" section of the SAC toolbar does not distinguish between classic and new data analyzer insights.

Environment

SAP Analytics Cloud

Cause

Resolution

It is possible to create a data analyzer insight that distinguishes between classic and new data analyzer using the SAC model SAC_PERFORMANCE_E2E

The dimension "Story Type" can be used to distinguish between classic insights (DA1) and the new DA insights (DA2).

Adding the dimensions "Story" and "Parent Folder" will provide the name of the insight and in what folder it can be found.

Adding the "Session Date" dimension as well can be used to filter only those insights opened since a certain date and thus filter out old insights that are no longer being used and the dimension "User" can be used to determine the owner of the insight that needs to be re-created.
