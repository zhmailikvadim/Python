Symptom

You've observed that the RGRTE payroll function produces an additional split when processing wage types with different currencies.

Environment

  • HR renewal
  • SAP S/4HANA
  • SAP ERP
  • SAP SuccessFactors Employee Central Payroll 

Cause

The RGRTE function does not currently support currency conversion.

Resolution

Currently, RGRTE does not perform currencies conversion. As a result, the system is designed to create distinct splits for each currency used, ensuring that wage values in different currencies are not aggregated.

The use of mixed currencies in wage types that require processing through the RGRTE function is not recommended, as this can affect calculations related to the Fair Labor Standards Act (FLSA). To maintain accuracy in payroll calculations, it is advisable to standardize the currency used across these wage types.
