Environment

  • SAP HANA Smart Data Integration
  • Office365’s SharePoint online

Cause

Office 365 SharePoint is different from SharePoint on premise, because Office 365 SharePoint uses Microsoft's OAuth authentication. When you visit that file link, it will redirect to Microsoft, and FileAdapter did not originally support this authentication feature.

Resolution

  1. If you are using Office365 SharePoint, upgrade the DP Agent to the latest version compatible with your environment (must be at least version 2.4.2).
  2. Then REFRESH the FileAdapter from the upgraded DP Agent.
  3. Now the File Adapter remote source should have credential options (application ID and tenant ID) relevant to Office365 SharePoint.
