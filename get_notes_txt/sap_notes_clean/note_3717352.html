Symptom

You have a data source based on a SAP Query infoset. If you try to navigate to the infoset out of transaction RSO2 you receive a transport popup for object R3TR AQQU SYSTBWGENER0SY<ID>.

If you run an extraction job for this data source, the following error appears in the job log and the job is canceled:

"The object R3TR AQQU SYSTBWGENER0SY<ID> doesn't exist
Assign a package
Specify the package or flag it as a local object"

Reason and Prerequisites

The SAP Query InfoSource catalog (AQBWCAT) refers to a query, which doesn't exist any longer

Solution

Implement the corrections using Note Assistant or import the relevant support package.
