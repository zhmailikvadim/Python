Symptom

As described in Regulatory Change Manager: Changes: Calculation of insurance premiums form for Q1 2026 (version 5.08), the electronic format of the Calculation Form for Social Insurance Contributions (RPCPAYRU_SICC) report is updated to the version 5.08.

This SAP Note provides you with the relevant system changes to address this change.

Other Terms

RSV, HRPAYRU_SICC, RPCPAYRU_SICC, SICC, SICC_2017, XML, 508.

Reason and Prerequisites

Letter of the Federal Tax Service of Russia No. BS-4-11/11504@ published on December 19, 2025.

Solution

As a general rule, SAP recommends that you implement the Support Package specified for your release.

However, if you need to install the solution earlier, use the SAP Note Assistant to carry out automatic and manual post-implementation correction instructions. For more information, see SAP Note Assistant.
