Symptom

Transaction Code: /PRA/PP_WORKPLACE

Following changes are required for BADI: /PRA/BADI_GET_COST_CENTER.

1. Adding optional importing parameter (GS_RUN_HDR).

2. Global reference instead of local reference for calling the BADI: /PRA/BADI_GET_COST_CENTER.

Other Terms

•General keywords: Add-on, IS-Oil, Industry Solution, Oil and Gas Upstream, PRA, Production and Revenue Accounting.

•Problem specific keywords: IS-OIL-PRA-REV-FI, /PRA/PP_WORKPLACE, /PRA/CL_PROCESS_PAYMENT_PROC->GET_COST_CENTER, Payment processing, Journalize, Well completion, PPA, clean core, sales date, well completion date, /PRA/BADI_GET_COST_CENTER, cost center, controlling area, global reference.

Reason and Prerequisites

Change request.

Solution

Optional parameter GS_RUN_HDR( this included the PP Run header information) has been included in the importing parameters of the interface of the BADI.

Apply the corrective instructions attached with the SAP Note.
