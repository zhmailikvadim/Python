Symptom

When you send a INVOIC message (business message 31002) for Grid Usage Billing to a receiver in Great Britain (GB), a non-EU country, the system raises error message /US4G/PROC_BILLING 030 "VAT Code of Receiver () is invalid in process" in application log.

This issue is relevant for the market role: DSO.

Other Terms

GUB, INVOIC, 31002, /US4G/PROC_BILLING 030, REDISND1, /US4G/CL_UBD_INVDOC_UTIL, FILL_RECEIVER_VAT_NUMBER, /US4G/IF_CONSTANTS_BILLING, Great Britain, non-EU Country, VAT Code

Reason and Prerequisites

Program error. Great Britain was not considered as a non-EU country.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the system doesn't raise error message /US4G/PROC_BILLING 030 when you send a INVOIC message for Grid Usage Billing to a receiver in Great Britain.
You can verify the fix by sending a GUB INVOIC message to a receiver in Great Britain and checking that the VAT Code of Receiver is filled in the INVOIC UBD data.
