Symptom

When scheduling a CR report using the new BI Launchpad UI, paginated Text format options are not in effect. (lines per page and text size).

Reason and Prerequisites

Steps to reproduce the issue:

1.  Log in using the new BI Launchpad UI (http:// ... /BOE/BILaunchpad)

2.  Navigate to a CR report and from the ellipses drop down menu select Schedule -> Formats -> Paginated Text

3.  Un-toggle "Use the export options defined in the report." and leave the lines per page as default (60)

4.  Schedule the report.

5.  Redo step 1-4 and change “Number of Lines per Page” to 20

6.  Compare the two scheduled reports and note that the lines per page are both 60, the 20 lines per page setting was not applied.

7.  Preform steps 1-6 in the old BI Launchpad UI (http:// ... /BOE/BILaunchpad), and observe that the second scheduled report has 20 lines per page.

Solution

This issue is fixed in the patches listed in the "Support Packages & Patches" section below. The "Support Packages & Patches" section will be populated with the relevant patch levels once they are released. For Business Intelligence Platform maintenance schedule and strategy see the Knowledge Base Article 2144559 in References section.
