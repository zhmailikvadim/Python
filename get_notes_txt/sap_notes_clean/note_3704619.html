Symptom

How to create contracts that have a service hierarchy and add them to the cart as any other item in guided buying?

Resolution

Please see the feature Support for Service Hierarchy Requisitions in Guided Buying (ACSS-683) that supports functionality to search for service items containing hierarchical items, and adding them to requisitions as any other item in Guided Buying. Please make sure to make note of the Prerequisites.
The procedure to import a contract with service hierarchies has been mentioned in the documentation Support for SAP ERP Service Specification Hierarchies. The commodity code chosen in the line item should be a service commodity code(ItemCategory = D) as defined in the ERP Commodity to Commodity mapping entry file.
