Symptom

When performing an RFC destination connectivity test to a SAP Cloud Integration endpoint, a HTTP 500 (Internal Server Error) is returned.

Environment

SAP Cloud Integration

Cause

This HTTP 500 response is returned due to the connectivity test not sending any payload with the request.

Resolution

An HTTP 500 response on the connectivity test can be considered a success, as the error is returned due to no payload being sent with the request.
With no payload, the adapter's parser will fail and the error is returned. Try to test the destination using it normally, sending a dummy payload and check the results.

Keywords

RFC destination connection test
