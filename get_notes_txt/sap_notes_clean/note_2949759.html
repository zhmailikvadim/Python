Symptom

Component: Risk Management
Module: My Risk Assessment Projects

You create a new Risk Assessment in My Risk Assessment Projects app (Fiori ID F1999). You select the type "JHA" (Job Hazard Analysis). In the search help for Jobs you search for available Jobs. The search result does not show the ID of the Jobs.

Other Terms

﻿﻿EHHSS_ELM_JOB_ID, EHFND_SHLP_EXIT_F4_BOBF_QUERY, EHHSS_JOB_ID, Health and Safety﻿﻿﻿

Reason and Prerequisites

Reason:
The column is not visible in the search result.

Prerequisites:
For information about the validity of the corrections, see the correction instructions.

Solution

The "Support Packages" section lists the Support Packages that contain these corrections.

Alternatively, you can apply the manual correction instruction.

If the column "ID" is not needed in the search result then this column could be hidden in the personalization settings.
