Symptom

Error message: "Resource not found for the segment 'ReviewBooklet'" appears in applications F6133 and F6401 after upgrading to S/4HANA 2025, while in Gateway, the error is "Metadata could not be loaded."

Environment

  • Group Reporting
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Access applications F6133 - Group Financial Statements Review Booklet and F6401 - Group View on Accounting .
  2. Observe the error "Resource not found for the segment 'ReviewBooklet'".
  3. In Gateway, the error is "Metadata could not be loaded."

Cause

Booklet ID is not activated in the system.

Resolution

  1. Run the transaction `ACTIVATE_REVIEW_BOOK`.
  2. Activate the following Booklet IDs:
  • SAP_FIN_CONS_RBOOKLET_AV1
  • SAP_FIN_CONS_GRB_A1
