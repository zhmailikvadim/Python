Symptom

During processing payment advise note for grid usage billing by transaction code INVMON, if aggregation documents are split, the logging is incomplete.

Other Terms

US4G, Grid Usage Billing, Grid Usage, INVMON, PAN, Payment Advice Note, Logging, Incomplete, Aggregation Document, Posting Document

Reason and Prerequisites

Function Missing

Solution

Implement following two actions for the solution:

1. Implement the correction instruction in the SAP Note or import the higher version of Support Package.

2. Open following configuration at IMG path: 'SAP Utilities' -> 'Intercompany Data Exchange' -> 'Bill and Payment Advice Note Processing' -> 'Make Basic Settings', click structure 'Customer Action in Monitoring' and check if following entry exists:

  • 'Descr. Cust. Action': Display Complete Application Logs
  • 'FM Customer Action': /US4G/DISP_COMPLETE_APP_LOG

If there is no such configuration entry, add one entry with 'Descr. Cust. Action' as 'Display Complete Application Logs' and 'FM Customer Action' as '/US4G/DISP_COMPLETE_APP_LOG'.

After you apply above solution, you can run transaction INVMON, perform searching and you will arrive the searching result view. Then there will be the option 'Display Complete Application Logs' after you click the button 'Execute Action' in tab 'Bill Documents for Selected Bill Header'. The complete application log will be shown if you select this open.
