Symptom

  1. Based on recent interpretation by ZUS , when force majeure absence occurs in a period from which allowance base is calculated, half of the daily work from counting class P02 should be considered an unpaid leave to which complementation does not apply.
     
  2. SAP Note 3233406 filters out some not relevant periods from the monthly log table, nevertheless they are still present in the complementation log table.

Other Terms

PLZLA, 3233406

Reason and Prerequisites

Program error

Solution

Payroll operation PLZLA is corrected.

For point 1: to activate the change, please create a switch called PY-3338672 into the table T7PL13 and set its value to '1' from 26.04.2023. Changes have impact on how wage types with U8 and UG options used in PLZLA configuration are processed during allowance base calculation.

For point 2: no activation switch is introduced.
