Symptom

You see there's a spike for the Capacity Unit consumption in BTP. It was nearly doubled.

Environment

SAP HANA Cloud

Cause

The activation of synchronous /asynchronous replicas  contributed to increased Capacity Unit consumption.

Resolution

  1. Review the SAP HANA Cloud Capacity Unit Estimator tool to understand the impact of memory allocation changes on Capacity Unit consumption.
  2. If necessary, consider deactivating replicas to reduce Capacity Unit consumption.
