Symptom

  • When payments are processed through the Fiori App F1345 Post Incoming Payments, the payments are not updated.
  • Consequently, the case status is not changed to Closed.

Environment

  • Dispute Management
  • S4CORE 106
  • S4CORE 107
  • S4CORE 108 

Reproducing the Issue

  1. Access the Fiori App F1345 - Post Incoming Payments.
  2. Process a payment for a dispute case.
  3. Observe that the payment is not updated and the case status is not changed to Closed.
  4. Checking the table FDM_DCPROC in backend system, this table has not been updated with the clearing information. 

Cause

  • Program error. 

Resolution

  1. Refer to SAP Note 3530607 - [SAP Fiori] Dispute case is not closed automatically.
  2. Apply the corrections provided in the SAP Note to resolve the issue.
