Symptom

  • During token rotation, existing API tokens have unclear usage status.
  • Exported logs do not show whether tokens are actively used or when they were last accessed.
  • It is unclear if token usage tracking (last used date/time or activity) exists in product logs.
  • There is a need to identify usage for a specific token or find an alternative way to validate token activity.

Environment

  • Product: SAP LeanIX solutions
  • LeanIX administration Technical users

Reproducing the Issue

No issue to reproduce

Cause

The product does not maintain separate access logs for technical users; only actions related to updating user permissions are tracked, so direct token last-used/activity information is not available.

Resolution

  1. Acknowledge that direct token usage (last-used date/time or dedicated access log) for technical users is not available in product logs.
  2. Verify that the user has the admin role, as only admins can create and manage technical users.
  3. Use the permission update log (audit trail) to identify actions performed by the technical user as evidence of activity.
  4. Review Fact Sheet audit logs to find changes attributed to the technical user.
  5. Review LeanIX integration sync logs to check for API activity associated with the technical user or the related integration.
