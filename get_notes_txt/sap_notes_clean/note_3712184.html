Symptom

Employer’s Return of Remuneration and Pensions for Tax Years (HHKCTXB0) report can generate IR56B forms for employees whose total income is 0. However, Inland Revenue Department (IRD) does not recommend submitting forms for these employees.

The SAP Note enhances the report and supports the configuration of not generating forms for employees whose total income is 0：

  • If you configure 'UT' in the feature HR-HK: End of tax year form (IR56B) (27IRB), report dose not generate forms for employees with zero total income and pops up error message 'The employee's total income is 0.'.
  • If you do not configure 'UT' in 27IRB, report generates forms for these employees by default. 

Other Terms

HHKCTXB0, 27IRB, IR56B, Total income, ir56-total

Reason and Prerequisites

This is an enhancement.

Solution

Do either of the following:

  • Import the relevant Support Package into your system.
  • Implement the correction instructions relevant to your release, then carry out the following manual activities:
    1. In transaction SE91, change message class 3Q, message 245.
    2. Enter the following data:
      Message Short text Self-explanatory
      245 The employee's total income is 0. X
    3. Save your entries.
