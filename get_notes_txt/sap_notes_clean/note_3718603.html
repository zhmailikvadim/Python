Symptom

A discrepancy is observed in the date field (for example, "Created On") between the Custom CDS View and the Custom Business Object; for example, in the Custom CDS View preview, the date is displayed as 31.01.2026, whereas the Custom Business Object shows it as 30.01.2026. 

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Navigate to the Custom Business object. 
  2. Created on value for system administrative value is different from the CDS view value for created On. 

Cause

The discrepancy is due to default system behavior where certain applications use "System Date and Time (UTC)" instead of "Local Date and Time" of the User.

Resolution

  • When accessing the views section through standard consumption channels (such as OData and Custom Business Objects), an output conversion is applied to ensure the correct date is displayed, however, within the views section itself, the raw server date (without conversion) is shown, which leads to the observed discrepancy.
  • This behavior is expected, as the system operates in UTC for cloud customers.
