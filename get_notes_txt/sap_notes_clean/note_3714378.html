Symptom

Duplicate warehouse code displayed in the List of Warehouses window.

Reproducing the issue

Prerequisites:

  1. You are working with SAP Business One, version for SAP HANA.
  2. Go to Administration → System Initialization → Company Details → Basic Initialization tab, select the checkbox Enable Multiple Branches.
  3. Go to Administration → Setup → Financials → Branches, create a new branch B01.
  4. Go to Administration → Setup → General → Users, assign this branch B01 to more than one user, for example, user manager and A001.
  5. Go to Administration → Setup → Inventory → Warehouses, create two new warehouses, code CD01 and CD02. Assign both warehouses to branch B01.
  6. Go to Administration → System Initialization → General Settings → BP tab, select the checkbox Enable Filtering Mechanism by Branch.

Scenario:

  1. Log in with user manager.
  2. Go to Sales - A/R → Sales Order, create an item type sales order. Input the customer code and choose branch B01.
  3. Add one row and choose one item. Enter 'CD' in the Whse column and press the Tab key on the keyboard.

Expected behavior:

In the opened List of Warehouses window, warehouse CD01 is listed in one line.

Actual behavior:

In the opened List of Warehouses window, warehouse CD01 is listed in multiple lines. This issue also occurs on other documents.
