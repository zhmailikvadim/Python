Symptom

You are using the Integration & Exception Monitoring app in SAP Focused Run (FRUN) to monitor scenarios and you observe the following issues:

  • Scenario filtering is not working as expected.
  • Upon loading the scenario, we are receiving a dump in UI with text: An exception with the type CX_AMDP_EXECUTION_FAILED was raised, but was not handled locally or declared in a RAISING clause.
  • The Web Service Messages card is not loading in the Communication Monitor page.

This occurs in scenarios when we configure the Payload Field Name filter parameter for Web Service Messages(ABAP) category.

Other Terms

SAP, FRUN, Integration Monitoring, AIM, Web Service Message(ABAP), Payload Field Name, WS_ABAP, Scenarios, Filters for component, Communication Monitor

Reason and Prerequisites

This issue occured due to a program error while handling the Payload Field Name filter parameter.

Solution

Implement the attached correction instruction(s).
