Symptom

In Perform Maintenance Jobs (F5104A), the Record Time dialog shows only a limited list in the Reason for Variance dropdown; newly added SPRO codes are missing because the SAPUI5 JSON model loads only the first 100 records by default.

Other Terms

F5104A, Reason for Variance, Perform Maintenance Jobs

Reason and Prerequisites

 The issue occurs because the Fiori app’s SAPUI5 JSON model loads only the first 100 records by default, and the number of SPRO‑maintained Reason for Variance codes exceeds this limit.

Solution

 Please implement the automatic correction instructions.
