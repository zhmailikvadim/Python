Symptom

During the evaluation of file-based audit log data, an explicit authorization check is performed on the object S_ADMI_FCD with the value AUDD when determining the files to be read.
However, with the introduction of object S_SAL, this should only be required as a fallback.

Other Terms

 RSAU_LIST_AUDIT_FILES, SM20, RSAU_READ_LOG

Reason and Prerequisites

 Program error

Solution

 Implement the attached corrections using SNOTE or by importing the referenced support package.

Solution Explanation

With this correction, the system checks for the object S_SAL with the activity SHOW_LOG.
The check S_ADMI_FCD with the value AUDD is used as a fallback only if there is no suitable authorization for this.
