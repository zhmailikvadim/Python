Symptom

 You are using real-time integration (RTI) between the SAP ERP system (SAP ECC or SAP S/4HANA) and SAP IBP, and you run the job "Real-Time Integration: Reconciliation" to compare the transactional data between the two systems.

The data collection in the ERP system is done by a job which is started by the IBP system and its status is monitored from IBP side. It may happen that the Job details are not yet available at the time of the first status check which could cause the IBP-side job to be failed "Data selection in external system failed" error message.

Other Terms

SAP IBP Real-Time Integration, Reconciliation

Solution

Implement the note or the corresponding Support Package.
