Symptom

  • For any reason, the Wage Tax returns were not correctly processed in the tax authorities' system.
  • The status in B2A for those returns are OK/closed
  • Tax authorities request to send the file again

Environment

Netherlands

Cause

Issues at Tax Authorities when processing the files

Resolution

  1. Set the system switch in T50BK to FTP.
  2. Resend the wage tax return files via FTP.
  3. Edit the XML to be sent via Notepad for ex, and change the time stamp (for ex 13:10:00 to 13:15:00) sothat a small change is detected
  4. Switch the system back to Webservice after having sent it via FTP
  5. Refer to SAP Knowledge Base Article 3615261 for detailed instructions on performing corrections in wage tax returns after a system transfer.
