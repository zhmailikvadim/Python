Symptom

  • An error occurs when attempting to extend the duration of a work package in the Work Packages tab of the Plan Customer Projects app.
  • Error message displayed: "Resource user is not active in October 2021."
  • The issue occurs in the "Work Packages" tab of the "Plan Customer Projects" app.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Plan Customer Projects app.
  2. Click on Work Package tab.
  3. Extend the duration of Work Package.
  4. Save.

Cause

The Retain Total Efforts functionality redistributes the total planned effort of roles/resources equally across the entire work package duration. When the work package starts before the user’s employment period, some planned hours fall outside of the employment period, leading to the error.

Resolution

  1. When changing the dates of the work package, select the "Retain Total Efforts for Common Periods" option.
  2. Save the changes.
  3. Confirm that the work package duration change is saved without the error.
