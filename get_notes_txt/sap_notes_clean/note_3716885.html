Symptom

When selecting a posting run that already has status 91 (deleted), the run history is updated again to "deleted". Even though it is not necessary as the run is already deleted.

Other Terms

RPCIPDEL, PEVSH, Run history, Status 91, Deleted

Reason and Prerequisites

Runs that are already in status "deleted" are not skipped.

Solution

Please implement assigned correction instruction or the corresponding HR Support Package.
