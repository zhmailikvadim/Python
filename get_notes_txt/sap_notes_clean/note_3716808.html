Symptom

Unified Rendering for Web Dynpro ABAP must be updated in line with error corrections.
These corrections solve the errors described with SAP note 3712914 Cumulative patch for UR-c2 2603

Other Terms

Unified Rendering (UR), WD ABAP, Web Dynpro, Rendering.

Reason and Prerequisites

This problem is caused by a program error.

Solution

The related TCI Package contains corrections for Unified Rendering from SAPK-75801INSAPUI to SAPK-75805INSAPUI with the following change list:

urversion 2.37.1
Code line/UR Patch c2/2603
Spam level needed check note 2187425
Deinstall allowed
TCI Created 2026-03-05

Also refer to the comments in SAP Note 1411980.

ATTENTION:
In case your system is SAP_UI 758 With Support Package 0 then this note can not be applied. You must apply Support Package 1 first.
Please acknowledge that the support for SAP_UI Innovation SPs has come to an end because Support Package 1 was released.
Lern more about  "Innovation Phase" in note 2217489 - Maintenance and Update Strategy for SAP Fiori Front-End Server
