Symptom

This SAP Note delivers the following enhancement for the German Company Pension Scheme (CPS):

Contribution reimbursement using batch input (report RPIAVAD0)
The batch input session generated by report RPIAVAD0 contains the call of transaction SM30. To run the session, the relevant user must therefore have the authorization for this transaction.
To be able to work without the assignment of the authorization for transaction SM30 in the production system, this SAP Note provides the new parameter transaction P01A_P01CBT. To be able to maintain the view V_P01CBT in the batch input session generated by report RPIAVAD0 using the parameter transaction P01A_P01CBT, you must set the new system parameter of the CPS BTRK (Contribution reimbursement: Maintenance V_P01CBT using Parameter Transaction in the view V_5DCY_A to "X" in Customizing of the CPS.

Other Terms

Report RPIAVAD0 Contribution reimbursement

Reason and Prerequisites

New function

Solution

This is delivered in an HR Support Package. As an advance correction, implement the attached correction instructions and execute the manual post-implementation steps.

See SAP Note 1232082 for information about the relationship between the HR Support Package and the technical name specified in the "Support Packages" section of this SAP Note.
