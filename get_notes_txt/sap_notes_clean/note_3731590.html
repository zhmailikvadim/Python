Symptom

This SAP Note corrects a problem during parallel processing if the treatment authorization is activated. If you call the programs RNWATSCO80 and RNWATADOK_CHECK with this setting, either the input file for the scoring is empty (RNWATSCO80) or the date of birth (RNWATADOK_CHECK) is missing.

This has already been corrected for the inpatient call (RNWATSCO80) using SAP Note 3674267, but not for the outpatient call.

Other Terms

LKF scoring, parallel processing, RNWATSCO80, RNWATADOK_CHECK, treatment authorization

Reason and Prerequisites

Prerequisites:

  • Country Version AT (Austria)
  • Latest Notes for LKF Scoring
  • Treatment Authorization
  • Parallel Processing of Programs

Reason:

In the case of an active treatment order, the scoring is not executed correctly in parallel processing mode.

Solution

The adjustments are available for the Support Packages of the specified releases. If you require the changes in advance or if the SAP Note is specified as a prerequisite, implement the correction instructions in your system.
