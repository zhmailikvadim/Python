Symptom

Failover does not work when more than one adapter is registered in the Agent group. When using the failover function of SDI, the failover will fail for part of the adapters when multiple adapters are used.

User friendly error message is not displayed for such issue.

Other Terms

This issue is found on these versions of products:

DPAgent 2.0 SP03 Patch52

HANA 1.0 SP12 Patch23

HANA 2.0 SP04 Patch0

Solution

To use the failover function, please register only ONE adapter in the agent which is added into the agent group.

Multiple adapters support will be available in future.
