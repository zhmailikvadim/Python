Symptom

  • When attempting to print an invoice using transaction J1BNFE, the printing process gets stuck.
  • The error message VN036 appears during the process.
  • Printing only resumes after clearing the user's queue.

Environment

  • SAP ERP
  • SAP S/4HANA
  • SAP Document and Reporting Compliance service
  • SAP Document and Reporting Compliance, Outbound Nota Fiscal

Reproducing the Issue

  1. Attempt to print an invoice using transaction J1BNFE.
  2. Observe that the printing process gets stuck.
  3. Wait for a prolonged period, error VN036.

Cause

The error happens because NAST is being used instead of the new Output Management in SAP DRC Outbound Nota Fiscal.

Resolution

Use the new Output Management instead of NAST.

NAST was used in SAP GRC. Note that GRC (SAP NF-e) mainstream maintenance has ended on 31.12.2025

See the following SAP Note for details: 0003595635 - End of Mainstream Maintenance for Nota Fiscal Electronica
