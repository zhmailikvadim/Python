Symptom

You have a process that updates partner address of a delivery document using a BAPI. The partner address to be changed was previously changed manually. Error message AM 020 - "Version '&' of the address doesn't exist' is issued".

Other Terms

International Address Version, ADRC, partner address, manual address, address number range, BAPI_OUTB_DELIVERY_CONFIRM_DEC, SHP_DELIVERY_PARTNER_UPDATE, SD_PARTNER_ADDRESS_SET

Reason and Prerequisites

The issue is caused by program error.

The issue only occurs if International Address Version is active in SD.

Solution

 Implement the attached correction or upgrade to the corresponding support packages.
