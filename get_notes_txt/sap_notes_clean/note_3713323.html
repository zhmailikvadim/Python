Symptom

How to change the label from "Resume" to "CV/Resume" in Application

Environment

SuccessFactors Recruiting Management

Resolution

The labels cannot be changed via XML templates in the Application from recruiter view. However, it is possible using Manage Languages feature for candidate's view.

Please refer to the See Also section to access the document related to Manage Languages feature.

See Also

  • Refer to: SAP Knowledge Base Article 2576546 - Using the Manage Language Feature to Change Labels in SuccessFactors
