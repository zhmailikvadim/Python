Symptom

The Customer is tracking data from the aggregation tables, which use the period profile monthly as per the bucket type configuration.
While deriving the periods during mapping, we are using the tracking period profile, which is causing the error.

Other Terms

Tracking, Collective Tracking, Period Profile, Aggregation, Period determination, /IRM/IPXXCIP, /IRM/IPXXCCP

Reason and Prerequisites

This is caused by program error

For V4, SAP Note #3732870 is available

Solution

Please implement the correction instructions attached. These corrections are only valid if you have applied the following support packs.

Release 6.1 – Base Package, 6.2 Base package, 6.3 Base Package, 6.4 Base Package and Support Pack 1

Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object.
