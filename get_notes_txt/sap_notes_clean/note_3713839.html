Symptom

  • The setting for the field "bank key" for the corresponding country has been set in Configuration Activity "Set Country Specific Check" for more than length 11.
  • When trying to add the values manually through maintain business partner. The length of field "bank key" can be set as more than length 11.
  • Error message: "Value is longer than the maximum permitted length &" is still generated when replicating the field "bank key" using SOAP API.

Environment

SAP S/4HANA Cloud Public Edition

Resolution

This is a standard behavior which has been documented in SAP Help Portal - BankDetails.

-> 

BankStandardID

The bank key (under which the bank data is stored in the appropriate country) is specified in this field.

 Note

The BankStandardID is restricted to 11 characters and was replaced by the BankInternalID. Nevertheless, if the BankStandardID is sufficient, it can still be used.
