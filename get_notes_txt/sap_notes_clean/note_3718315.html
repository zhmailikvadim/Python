Symptom

When creating the Agreement Request through IDOC processing, the external number is not being saved in the Agreement Request. Additionally, a runtime error is occurring  when the auto post flag is enabled during the IDOC processing, which interrupts the creation process. Due to this, the Agreement Request is either not created successfully or is created without the required external number.

Other Terms

KNUMA_EX, PSDOC_EX, External Number, /IRM/IP_AGR_REQUEST_CREATE

Reason and Prerequisites

This is caused by program error.

If you are in 2023 V3 Base Package,Please implement the latest version of SAP Note #3630804 as pre requisite note.

Solution

Please implement the correction instructions. These corrections are only valid if you have applied the following Support Packs.

Release V4 - 2023 V3 Base Package or Support Pack 1.

Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object.
