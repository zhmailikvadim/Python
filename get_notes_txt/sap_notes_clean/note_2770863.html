Symptom

The HTML5 web client does not support the SAML2 authentication method.

Other Terms

BFC, BOFC, architecture, communication

Solution

Update your SAP Financial Consolidation server installation to a support package, patch or release where the issue is fixed.

See the Support Packages & Patches section for details and available patches.

Notes:

  • For the implementation information and steps of SAML2, please refer to SAP Financial Consolidation Security Guide, section 3.6.
  • Note that as mentioned in the SAP Financial Consolidation 10.1 Support Package Install (IX. Web.Config settings modifications between SP08 and SP08 Patch 01), a new tag must be added in the Web.config file of the HTML5 application.
  • Warning: the SAML2 authentication concerns only the HTML5 web client and not the other clients (legacy web and heavy client).
