Symptom

 When attempting to add a new contract to an already open application, the system triggers an ABAP short dump. The operation fails with the runtime error CX_ABAP_BEHV_RUNTIME_ERROR in program CL_ABAP_BEHV_HANDLER_ASSOC, preventing the contract from being added successfully.

Other Terms

CX_ABAP_BEHV_RUNTIME_ERROR , RAP API , CL_ABAP_BEHV_HANDLER_ASSOC , INSURPLCYAPPLUUIDFOREDIT

Reason and Prerequisites

The code to copy the UUID was missing

Solution

As a general rule, SAP recommends that you install a solution by applying a Support Package. However, if you need to install a solution earlier, use the Note Assistant to implement the correction instruction. You can find more information about the Note Assistant in Support, under https://support.sap.com/en/my-support/knowledge-base/note-assistant.html.
