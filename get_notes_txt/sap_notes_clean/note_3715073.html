Symptom

You have the following configuration in the system:

  • Return value "N0507" is set in feature 27IRE.
  • The following name format is configured in the view V_T522N:
Format Prefix Name Format Ind. Sequence Number Field name
1 0002 5 1 NACHN
1 0002 7 1 NACHN
1 0002 7 2 ',
1 0002 7 3 NAME2
1 0002 7 4 VORNA

When you run the Notification by Employer of Employee Who Commences to Be Employed (HHKCTXE0) report, the system displays only the surname in item 2, “Name of Employee,” in the PDF file. The full name does not appear as expected.

After implementing this SAP Note, the system displays the full name of the employee in item 2, “Name of Employee,” in the PDF file.

Other Terms

Feature 27IRE, View V_T522N, Name Format, HHKCTXE0

Reason and Prerequisites

This is a correction of legal function.

Solution

Implement the attached Correction Instruction (CI) or the relevant Support Package.
