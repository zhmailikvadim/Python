Symptom

I maintained the Cross References in the Managed Gateway for Spend&Network for a document type, however the values are not appearing in the respective fields in the outbound payload.

Environment

Managed Gateway for Spend&Network

Cause

A bug can happen that can make that the values maintained in the interface to not be saved in the backend.

Resolution

Erase the Cross References for the document type and System ID with the issue and save the Cross References. Then maintain the Cross References again and save it again, this should save the changes in the backend.

For more details on how to maintain the Cross References check the Configuration Guide - Cross References Setup on SAP Integration Suite, Managed Gateway for Spend Management and SAP Business Network.
