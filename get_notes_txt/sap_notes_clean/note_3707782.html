Symptom

Error message CL103 issued while creating a class in t-code CL01. The error message states: "Error Info...   Update: error creating KLAH for "example_class".

Environment

  • SAP ERP Classification (CA)
  • SAP R/3
  • SAP R/3 Enterprise
  • SAP ERP Central Component
  • SAP ERP
  • SAP Enhancement package for SAP ERP
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA, on-premise
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Open transaction CL01
  2. Give all necessary data for creating a class (For example: description, etc.)
  3. Save
  4. "Update was cancelled" error appears

Cause

In t-code SNUM, the number status for object KLASSE is incorrect.

Resolution

  1. Open t-code SE16
  2. Open table KLAH
  3. Show all results for all class types
  4. Set "CLINT" field in descending order
  5. Check which is highest number (For example 500)
  6. Open t-code SNUM
  7. Open object KLASSE
  8. Click on "change" button
  9. Click on "interval editing" (F7 hotkey)
  10. Click on "change intervals" button
  11. Click on "change number range status" button 
  12. Change NR status field to a value, which is higher than the highest CLINT number (For example 501)
  13. Save
