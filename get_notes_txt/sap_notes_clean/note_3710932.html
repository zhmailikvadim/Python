Symptom

In SAP Ariba Invoicing, if an invoice that has tax only at line item level is uploaded, system does not determine tax codes in alignment with configured business rules. Tax Code gets determined per expectation if tax in the invoice pdf is at the header level.

Cause

Taxes at line level are not recognized by SAP Document AI(formerly known as Document Information Extraction). Link to documentation Extracted Line Items (Base Edition and Premium Edition). As a result system is unable to check the Business Rules if taxes are present at the item level in the invoice pdf.

Resolution

Please create an improvement request to implement functionality to determine tax code if tax is maintained at the item level alone.
