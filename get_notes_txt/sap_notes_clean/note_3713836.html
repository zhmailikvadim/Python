Symptom

  • Attempting to create a substitution rule for Group journal entries with cost center.
  • Error message FIN_RE_PARSER/031 - "Value XXX is not allowed for field CostCenter." is raised.

Environment

  • Group Reporting
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

in APP "Manage substitution and validation rule - Group journal entries"

Cause

Missing setting.

Resolution

  1. Open transaction code SM34.
  2. Enter 'FIN_RE_CONTEXT' and click 'Edit'.
  3. Select 'FINS_GR_JEI_1' and click 'Structures/CDS Views'.
  4. Select 'FINS_GR_JEI_1' and click 'Fields'.
  5. Navigate to the field CostCenter.
  6. Change the value in the column 'Message Type' from 'Error' to 'Warning'.
  7. Save the changes and transport them to other systems.
  8. Attempt to activate the substitution rule again.
