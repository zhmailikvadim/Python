Symptom

  • The App ID search functionality does not work in the App Finder.
  • Searching by App ID does not display the target application, while searching by App name does.

Environment

  • SAP Credit Management (FSCM)
  • Sales and Distribution (SD)
  • Fiori-UI for Credit Management
  • SAP S/4HANA 2023 and higher
  • SAP Fiori for SAP ERP

Reproducing the Issue

  1. Open the App Finder in the Fiori LaunchPad.
  2. Attempt to search for the application using App ID.
  3. Observe that the application does not appear in the search results.
  4. Search for the application using its App name instead.
  5. Observe that the application appears in the search results when using the App name.

Resolution

  1. Navigate to the transaction code "MM_APP" and search for the relevant Fiori application.
  2. Access the application and switch to the "Tiles" tab.
  3. Locate the "Tile Keywords" field and input the Fiori ID (App ID).
  4. Save the changes to enable the App ID search functionality for the specified application.
