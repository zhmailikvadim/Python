Symptom

The "Status" tab in Modeler UI takes very long time to load or even fails to load due to the following error:

Cannot handle request: failed to fetch graphs with query from HANA catalog: error while fetching graphs execution from store: error is encountered during iteration through row set: context deadline exceeded.

Environment

SAP Data Intelligence Cloud

Cause

There are too many paused graphs caused the slowdown and unresponsiveness.

Resolution

If the paused graphs are not intentionally retained, consider enabling 'Garbage collection for paused graphs' to automatically clean them up.
