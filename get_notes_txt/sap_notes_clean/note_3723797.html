Symptom

You can use the Billing: Case Selection program (RNWATASEL00) to create a case selection based on numerous selection criteria. If you want to exclude a cost object, the system issues the error message "Cost objects cannot be excluded".

This SAP Note is relevant only for the country version Austria (LV AT).

Other Terms

RNWATASEL00, case selection, insurance provider exclusion

Reason and Prerequisites

Reason:

Enhancement

 Prerequisite:

Country Version Austria (AT)

Solution

Implement the source code corrections.

After maintenance, you can exclude cost objects. Note that this exclusion only ignores a case if it is the only insurance provider valid at the time of the case. If there is at least one other, the case is still taken into account.

At least in this relationship, the program behaves in the same way as the selection for outpatient cases (RNWATASEL01).
