Symptom

To increase the user experience, the saved credentials in the Fiori Client is needed.

Environment

  • SAP Fiori Client
  • SAP Mobile Platform (SMP) SDK 3.0

Resolution

  1. When standard Fiori Client is used (downloaded from the Google Play, App Store, Microsoft Store)

Change the Fiori Launchpad URL in the Fiori Client and add ?allowSavingFormCredentials=true to the end of it.

Note: when other parameters are used the instead of the question mark ("?") add the ampersand/ and sign ("&").

  1. When customized Fiori Client is used (customized with SMP SDK 3.0)

Upgrade SMP SDK 3.0 to SP15 where this is possible.

Search in this blog for allowSavingFormCredentials and see how to do it exactly.

SAP Fiori Client (SP13+)

*Application Passcode is required to encrypt credentials in Datavault
