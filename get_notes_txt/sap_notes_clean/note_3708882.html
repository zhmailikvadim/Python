Symptom

The system does not save an attached file with name that contains comma or semicolon in the Web Client.

Reproducing the issue

Scenario:

Create a Sales Order in the Web Client and attach a file to the document with a name that contains comma or semicolon.

 

Expected behavior:

The file is attached and saved correctly.

 

Actual behavior:

The attachment is not saved in the document.

Cause

System Limitation

Solution

This is a system limitation.

Workaround:

1. Do not use comma or semicolon in file names that are attached in the Web Client.

2. In case the file name must contain comma or semicolon, attach the files to the document using the standard client.
