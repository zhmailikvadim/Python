Symptom

  • Testers are unable to apply a filter on the "Responsible" field for defects in SAP Cloud ALM 
  • When testers attempt to filter defects by their own name in the "Responsible" field, their name does not appear as an option, although other testers' names are selectable.

Environment

SAP Cloud ALM Implementation

Reproducing the Issue

  1. Launch SAP Cloud ALM > Implementation > Defects.
  2. Filter on the "Responsible" field.
  3. Observe that the user's own name is not available as a selectable option.

Cause

Resolution

To see only the defects that are assigned to you, select Me in the Responsible field
