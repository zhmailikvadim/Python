Symptom

Error message "No demands found" is raised when clocking in SAP GUI and Web Dynpro DBM Time Recording Terminal.

Due to this error no corresponding time event data and time sheet data are stored.

Other Terms

MRS, BAdI, /DBM/SCHED_TM_MRSS, Time Recording, TM 051

Reason and Prerequisites

Program error, side effect of the SAP Note 2574298.

Solution

Implement correction instructions of this SAP Note only when MRSS component is activated in the system.
