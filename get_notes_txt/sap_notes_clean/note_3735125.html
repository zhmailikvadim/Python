Symptom

FRUNCONT50009 has been released on 26 February 2026 but job SAP_FRN_RCD_CHECK_UPDATES is not downloading this new version.

Job SAP_FRN_RCD_CHECK_UPDATES 11454000 started
Step 001 started (program RCSU_AUTO_NOTIFY_AND_DWLD, variant , user ID FRN_BTC_RCD)
Job SAP_FRN_RCD_CHECK_UPDATES started
Step 1: Attempting to read the Notification XML from Post Box
Call to the postbox system. Method name: cl_csu_notification=>READ_FROM_POSTBOX
Step 1.1: Reading of the Notification XML is Successful
Reading of notification XML is successful. No of content available:
No new contents available for download
No new contents available for download
Job finished

Rapid Content Delivery tile is mentioning that the version 9 is available for download.

Environment

SAP Focused Run 5.0

Reproducing the Issue

  1. Execute transaction FRUN
  2. Check Rapid Content Delivery tile

Cause

There is an inconsistency in the database tables related with delivery version 9

Resolution

  1. Use RCSU_MANUAL_UPLOAD report to fix the RCD database tables (Note 3475561 Manual content update of FRUN-CONT 500 in SAP Focused Run).
  2. Apply SAP Note - 3633438, for the UI as the full contents is not visible.

    Keywords

    RCD, delivery, CSU, no new contents, content
