Symptom

In the "Define Derivation Scenarios for Business Partners" app, when attempting to create a field rule or table rule using DFKKBPTAXNUM as either the base or result table, you encounter an issue where the value help does not display the table DFKKBPTAXNUM.

Other Terms

DFKKBPTAXNUM, Derivation Scenarios, BP, Tax number, rules, DQM

Solution

Implementing this note via transaction SNOTE or applying the associated support package makes table DFKKBPTAXNUM selectable in the base table value help and the result table value help in the scope of Derivation Scenario rules.
