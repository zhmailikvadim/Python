Symptom

A user not authorized to post at a given sales organization is still able to post a material list.  The system can post the list even if the user is not authorized to maintain the details.

Other Terms

Customer List, Vendor List, Material List, /IRM/EPKULM, /IRM/EPPDLM, /IRM/EPLILM, LIST_ITEM_POST

Reason and Prerequisites

This is caused by program error.

SAP Note # 3511292 is available for ECC Releases.

Solution

Please implement the correction instructions attached. These corrections are only valid if you have applied the Support Pack(s):

Release V/4 - 2021, 2022, 2022 V2, 2023 and 2023 V2.

Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object.
