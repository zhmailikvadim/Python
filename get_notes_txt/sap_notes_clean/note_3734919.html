Symptom

  • Trying to upload a .woff file under Theming & Branding → Default Font Setting results in an error.
  • Error "Update default font setting failed" occurs when uploading a custom .woff file under Default Font Setting. 

 "Image / Data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental" 

Environment

  • SAP SuccessFactors Work Zone 
  • SAP Build Work Zone, Advanced Edition

Reproducing the Issue

  1. Go to Admin console → Theming & Branding → Default Font Setting
  2. Select "Custom Font" option
  3. Upload a .woff file
  4. Click Save
  5. Error dialog "Update default font setting failed" appears 

Resolution

  • The solution of this issue is currently under investigation.
  • Our Engineering team is working on a code fix to resolve the issue.
  • Tentative release is 2604A (Release Date: 29 Apr 2026).
  • Please follow this KBA for updates or changes on the release date.
