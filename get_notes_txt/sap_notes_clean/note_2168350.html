Symptom

F4 help for search criteria Category 1, Category 2 or Category 3 is not displayed - no combo box with values is displayed.

Other Terms

Utilities, ICWC, IUICOVW/InrDynamicFilter, CL_IUICOVW_INRDYNAMICFILT_IMPL, F4 help

Reason and Prerequisites

You are using account overview of business partner. You are inspecting tab page Interactions. You would like to filter records by search criteria Category 1, Category 2 or Category 3.

Solution

Implement correction instructions or import the relevant Support Package.

If there are too many F4 help values (more than a threshold), combo box is not displayed. This is one reason why F4 help is not displayed. Old threshold is 10 records. New threshold is set to 250 records.
