Symptom

You already created a checklist document from a maintenance order or via transaction IW91.
When opening the created document, the following issues can be observed:

  1. The Inspection Description field is missing in the document.
    It should be displayed in the Inspection Result section.

  2. If the checklist document was created for a maintenance order that contains multiple inspection lots, reopening the document shows only the last inspection lot.
    Previously created inspection lots are no longer displayed.

  3. The Inspection Result section is not printed if the Characteristic control indicators are not set, even though a valuation result exists.                                                                                                                                    
  4. The Long Text of the characteristic is always displayed in the original language.

Other Terms

checklist, QPAM, print, EAM_CL_PRINT

Reason and Prerequisites

Programming error.

Solution

Please implement the attached correction instructions or install the corresponding support package.
