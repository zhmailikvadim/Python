Symptom

  • I want to move a specific item, such as a Custom Field, to the target environment using the Configuration Mover tool.
  • I cannot find an option to choose which item to move, only the object type itself.

Cause

The Configuration Mover transfers all items for the selected object type in a single operation. This means that if you choose, for example, “Custom Fields,” all custom fields from the source environment will be moved to the target. It does not allow selecting individual items.

Resolution

To move only the desired items, you will need to use the object’s Download and Upload connectors. This connector-based approach provides the selectivity you are looking for.

Example: Moving a specific Custom Field from the source to the target environment:

  1. Use the Custom Field Configuration Download option in the source environment to extract the custom field data.
  2. Open the file and remove any custom fields you do not want to transfer.
  3. Use the corresponding Custom Field Configuration Upload option in the target environment to import only the fields you have chosen.
