Symptom

The email is sent by workflow send mail step, the email content contains unwanted spaces and unwanted line breaks.

Environment

  • SAP Business Workflow
  • SAP NetWeaver
  • SAP Web Application Server for SAP S/4 HANA
  • ABAP PLATFORM - Application Server ABAP

Cause

Program error

Resolution

Install this correction instruction of SAP Note 3549660 - Business Workflow: Send mail step sends malformed mails (unwanted spaces and random line breaks) or upgrade to the corresponding SP.

See Also

  • SAP Note 3549660 - Business Workflow: Send mail step sends malformed mails (unwanted spaces and random line breaks)

Keywords

WF_GDPR, SWF_WORKFLOW, WFMSTD, Send Mail Step, Task Description, Classic Workflow, SWDD, SOST
