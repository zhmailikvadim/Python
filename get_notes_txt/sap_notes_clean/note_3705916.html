Symptom

If several lock entries (for example, in different clients) exist or are maintained for a transaction in SM01_CUS, the application terminates with the runtime error DBSQL_SQL_ERROR.

Other Terms

SAIS_RSAUDITC_BCE, SAIS_RSAUDITC_BCE_CORE, TAPPL_LOCK, RFTBPROT_BCE

Reason and Prerequisites

 Program error

Solution

Implement the attached corrections using SNOTE or by importing the referenced support package.

Solution Explanation

The error is based on a dependency on the behavior of the SAP HANA database and has been corrected by adjusting the CDS views used.
If the correction cannot be implemented, only one lock entry should be set for each application. Note: Locks for the entire system can only be set in the system client (000).

The error can occur for all applications that are based on the CDS view SAIS_RSAUDITC_BCE (report RSAUDITC_BCE, SM01_DEV, SM01_CUS and the corresponding API).
