Symptom

This SAP Note provides you with adjustments to the Peru Journal Ledger (PE_JOURNAL_LEDGER), Peru Inventory and Balance Ledger (PE_INVENTORY_BALANCE_LEDGER), Peru Sales Ledger (PE_SALES_LEDGER), and Peru Purchase Ledger (PE_PURCHASE_LEDGER) reports in the Run Statutory Reports app for SAP Document and Reporting Compliance. It addresses the limited size of the reference field in the document header.

When you run the PE_JOURNAL_LEDGER, PE_INVENTORY_BALANCE_LEDGER, PE_SALES_LEDGER, and PE_PURCHASE_LEDGER reports, the limited size of the reference field in the document header prevents you from entering all required information. As a result, the Document Type, Series Number, and Document Number fields in the report output do not display complete information.

Other Terms

Peru, Document and Reporting Compliance, DRC, fka ACR, ODN, Journal Ledger, Inventory and Balance Ledger, Sales Ledger, Purchase Ledger, Libro Diario, Libro de Inventarios y Balances, Libro de Ventas, Libro de Compras PE_JOURNAL_LEDGER, PE_INVENTORY_BALANCE_LEDGER, PE_SALES_LEDGER, PE_PURCHASE_LEDGER

Reason and Prerequisites

This SAP Note is relevant only for Peru.

You have implemented previous notes listed in SAP Note 3206609.

Solution

After implementing this SAP Note, the system provides, in the document header, the new country/region-specific field Official Reference Number,which supports values up to 80 characters, providing all necessary information that is required for the Document Type, Series Number, and Document Number fields in the PE_JOURNAL_LEDGER, PE_INVENTORY_BALANCE_LEDGER, PE_SALES_LEDGER, and PE_PURCHASE_LEDGER report outputs.

For more information about the development delivered with this SAP Note, see the What's New topic on SAP Help Portal: Enablement of Field Official Reference Number at Document Header Level.

SAP recommends that you install a solution by applying a Support Package. However, if you need to install a solution earlier, use the Note Assistant to implement the correction instruction. You can find more information about the SAP Note Assistant in the SAP Support Portal, under SAP Note Assistant.

To implement the solution, you must carry out manual activities and implement the correction instructions that are valid for your release.

Documentation for SAP S/4HANA for Peru is available in the system and in the SAP Help Portal. System documentation (such as field help, customizing, messages) is delivered via Support Packages. SAP Help Portal documentation is available at help.sap.com/viewer/s4h_op_peru.
