Symptom

You are monitoring SAP Integration Suite (Cloud Integration) category in Integration & Exception Monitoring app of SAP Focused Run (FRUN) 5.0 SP00, FP01, FP02.

This is a mandatory fix to ensure that the data collection works seamlessly for category SAP Integration Suite (Cloud Integration). An additional field 'LastChangeTime' is added to the SAP Integration Suite (Cloud Integration) message payload.

If the note is not implemented, it will impact the data collection of the services configured with this category.

Other Terms

AIM, FRUN, Focused Run, Integration & Exception Monitoring, Data Collection, SAP Integration Suite(Cloud Integration), CPI, HCI

Reason and Prerequisites

The payload that the Integration & Exception Monitoring app receives from SAP Integration Suite (Cloud Integration) is altered with an additional field 'LastChangeTime'. Although this field is added for internal purpose of querying data, data processing logic needs to be adapted to prevent data loss.

Solution

Apply the attached correction instruction(s).
