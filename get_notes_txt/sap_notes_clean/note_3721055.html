Symptom

When phase control codes are configured in the backend system, the initial sync in SSAM fails and returns the error E-000 CX_SY_OPEN_SQL_DB.

Other Terms

SAP Service and Asset Manager, PhasedControl, Timeout, CX_SY_OPEN_SQL_DB, S4MOBADDON-8265

Reason and Prerequisites

This issue occurs because the number of work orders/operations is too large to be processed during the Phase Control execution producing a timeout exception.

Please ensure that the parameters maintained in transaction SU01 / SU3 are properly configured to limit the number of work orders/operations accordingly. This will help minimize the impact on device storage and overall performance before implementing this SAP Note.
