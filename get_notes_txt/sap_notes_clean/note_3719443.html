Symptom

In table T521B (Payee Keys), field ETEXT (Payee Name) has a length of only 25 characters.

Other Terms

V_T521B, ETEXT, VoP

Reason and Prerequisites

The field length limitation prevents you from entering longer payee names, which can cause data truncation issues.

Solution

After implementing this SAP Note, the system extends field ETEXT in table T521B to accommodate longer payee names. You can now enter payee names up to 140 characters. Note that SAP Note 3726620 delivers the changes of the DME report to write the extended field ETEXT to the table REGUH and thus into the transfer file.

To implement the correction, implement the synchronization package HRSP available in May 2026 or follow the correction instruction provided in this SAP Note.
