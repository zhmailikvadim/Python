Symptom

You're using CM for SAP S/4HANA or SAP CRM, as well as Google Chrome 142 or newer, respectively Microsoft Edge 143 or newer. You're starting a launch transaction. An error message appears, stating "The connection is blocked because it was initiated by a public page to connect to devices or servers on your local network."

Local Network Acces" feature has recently disabled by default by chromium browsers, With this note we set it enabled by default, but you have the option to disable the feature using the transaction WCF_PERM_POL.

Other Terms

Permission-policy, WCF_PERM_POL

Reason and Prerequisites

Google Chrome, Microsoft Edge, Local Network Access, LNA

Solution

Implement the assigned correction instructions.
