Symptom

Which fields can be mapped from LinkedIn to SAP SuccessFactors?

Environment

SAP Success Factors Recruiting Management 

Resolution

The following fields can only be mapped from LinkedIn to SAP SuccessFactors.

Standard fields:

  • FIRST NAME
  • LAST NAME
  • ADDRESS
  • HOME PHONE
  • CURRENT TITLE
  • CURRENT COMPANY

Background Elements:

WORK EXPERIENCE

  • START DATE
  • END DATE
  • JOB TITLE
  • JOB DESCRIPTION
  • EMPLOYER
  • PRESENT EMPLOYER (Boolean)

EDUCATION

  • START DATE OF ATTENDANCE
  • END DATE OF ATTENDANCE
  • NAME OF SCHOOL
  • DEGREE OBTAINED
  • FIELD OF STUDY
