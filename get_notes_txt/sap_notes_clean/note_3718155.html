Symptom

When executing the run "Activity-Based Costing - Production", the runtime error TSV_TNEW_PAGE_ALLOC_FAILED is raised during the save of the data to the database and the run is terminated.

Other Terms

Activity-Based Costing - Production, TSV_TNEW_PAGE_ALLOC_FAILED

Reason and Prerequisites

Program error

Solution

Version 1 of 19 February, 2025: New creation

Install the code correction.

Description of Correction:

When executing the run "Activity-Based Costing - Production", the save of the data to the database is executed successfully.

Functional Dependencies:

There are no functional dependencies.

Manual Changes:

There are no manual activities.
