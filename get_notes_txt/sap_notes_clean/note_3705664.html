Symptom

When releasing a blocked invoice via MRBR or other invoice release functionality and the Prompt Payment Act (PPA) customizing is active:

  1. An error message from message class F5A number 194 is raised: "You cannot change the content of field &" e.g. ZLSPR, ZBD1P.

    This occurs after the application of SAP Note 2134634 or the corresponding support pack.

  2. Alternatively, for the same transaction set you receive a short dump/runtime error in program LFACGFAP, subroutine DOCUMENT_CHANGE_VEND.

    This occurs for documents with multiple vendor lines after applying SAP Note 2596048 or the corresponding support pack. 

Previously the changes to address these scenarios were contained within Pilot Notes 2563600 and 3234969 respectively. Because the Notes are only available to Pilot customers, they were not contained within the standard solution. This required US Federal agencies to re-apply these Notes after applying support packs or upgrading. 

The functionality is now released in the standard solution.

Other Terms

MRBR, RM08RELEASE, FI_DOCUMENT_CHANGE, FM_PAYMENT_BLOCK_UPDATE, F5A 194, F5A194, PPA, ZLSPR, ZBD1P, RAISE_EXCEPTION, WRONG_INPUT

Reason and Prerequisites

Reason: Coding update required

Prerequisites: You have completed the necessary customizing required to activate US Federal Government functionality, and the necessary customizing that is required to have the Prompt Payment Act functionality active in your system for your company code and vendor.

Solution

Apply the correction instruction.
