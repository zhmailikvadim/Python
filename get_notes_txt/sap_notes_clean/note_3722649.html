Symptom

  • CDS view F0MMIM_MATSTVAL is not available in S/4HANA 2023.
  • The ABAP catalog equivalent of CDS view F0MMIM_MATSTVAL is not accessible in upgraded S/4HANA 2023 systems.

Environment

  • Materials Management (MM)
  • Inventory Management
  • SAP S/4HANA
  • Fiori UI for Stock and Goods Movements

Reproducing the Issue

  1. Open transaction SE11.
  2. Attempt to display CDS view F0MMIM_MATSTVAL.
  3. Observe that the view does not exist.

Resolution

  1. Use the CDS view F0_Mmim_Matstockquanvalbydate instead of F0MMIM_MATSTVAL.
  2. Access the CDS view F0_Mmim_Matstockquanvalbydate via transaction SE11.
  3. Note that F0MMIM_MATSTVAL was the ABAP catalog equivalent of CDS view F0_Mmim_Matstockquanvalbydate in lower releases.
  4. Ensure the full CDS view name is used.
