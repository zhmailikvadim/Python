Symptom

How to close/complete Legacy Data transfer?

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Environment

  • FIN Asset Accounting​
  • SAP S/4HANA ​
  • SAP S/4HANA Finance​
  • SAP S/4HANA Cloud Private Edition

Resolution

The Legacy Data transfer can be closed/completed in tCode FAA_CMP

Follow the steps:

1) Enter the tCode FAA_CMP:

2) Select the appropriate company code and click 'Legacy Data Transfer' tab:

3) Enter Edit mode by click 'Change/Display' button:

4) Close the Legacy transfer segment:

FAA_CMP, Legacy Data Transfer, LDT, Close, Complete legacy, segment.
