Symptom

  • A new feature in the commerce cloud Portal: "Transfer Properties -> Database and media storage" functionality is disabled and not accessible to S-Users.

Environment

SAP Commerce Cloud

Reproducing the Issue

  1. Attempt to access the new feature in the commerce cloud portal.
  2. Observe that the feature is disabled and not accessible to all users.

Cause

As mentioned in below statement in help documentation below:

"Due to the sensitive nature of the files in the database and media storage, you can't transfer this information from a Production environment. If you select a production environment as the source, the Data and Media Storage option becomes unavailable."

Still for non-production target environment like: Development or Stage, the issue would be from the un-materialized state of that environment.

Resolution

  1. Ensure that the target environment is materialized for data and media transfer to be enabled.
  2. Refer to the updated help documentation for further guidance.
