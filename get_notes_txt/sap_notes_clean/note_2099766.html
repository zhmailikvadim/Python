Symptom

If you want to define PM order operations with duration zero and release such operations to MRS planning board, you have to define a MRS demand default duration, because the MRS demand requires a duration which is not zero. In the MRS planning board you can work with such a MRS demand and for example shift the MRS demand or create MRS assignments.

Due to program error, the MRS demand default duration was written back to the PM order operation. But this is not correct, because the following PM order scheduling cannot be reflected properly in the MRS demand.

The system will only use the default duration in the initial creation of the MRS demand. If you change the MRS demand (for example shift the demand or change the duration) and save the MRS planning board, then the PM order operation will be updated with the recent change. Also the following PM order scheduling will be reflected properly in the MRS demand.

Other Terms

work unit

Reason and Prerequisites

This is due to a program error

Solution

Please implement the automatic correction.
