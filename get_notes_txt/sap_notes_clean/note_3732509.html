Symptom

In transaction QA08, it is intended to activate stock-relevant inspection type for a material-plant combination, in which quality inspection stock already exists.

Although indicator 'Activate despite insp.stock" is activated, below message is raised:

"No material processed" (message nr: QA256)

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Environment

  • SAP ERP Quality Management (QM)
  • SAP ERP Central Component
  • SAP ERP
  • SAP Enhancement package for SAP ERP
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA cloud private edition
  • SAP S/4HANA

Cause

Inspection type for available stock in quality inspection is not defined for the corresponding plant at plant-level QM customizing, which can be reached via following path:

Transaction QCC0 ---> Basic settings ---> Maintain settings at plant level (tab Insp.lot creation)

As result, the system cannot create a new inspection lot for the already existing quality inspection stock, which would be necessary in order to activate the stock-relevant inspection type.

Resolution

Maintain an inspection type 0800 in field 'InspTypeForInspStock' at above mentioned plant-dependent QM customizing.

For further information about the significance of inspection type 0800, read below SAP Note:

1036872 - Significance of inspection type 0800 unclear

And also make sure that indicator 'Post to insp.stock' is activated and quality score is maintained for this inspection type at inspection type default value customizing (transaction QCC0 ---> Quality inspection ---> Inspection lot creation ---> Define default values for inspection types)
