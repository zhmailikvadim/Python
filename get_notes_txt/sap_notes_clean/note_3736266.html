Symptom

  • When opening a new target group and attempting to add a lead, the leads cannot be sorted by account.
  • Sorting by account does not work in the "Add Leads" dialog within the target group.
  • Sorting by account works in the traditional Lead Overview.

Environment

SAP Sales Cloud version 2

Reproducing the Issue

  1. Go to Navigation Menu
  2. Go to Target Groups
  3. Open a new target group.
  4. Select Add Leads.
  5. In the "Add Leads" dialog, try sorting the leads by the Account column.

Cause

The fields Account ID and Account are added by adaptation. While the Account ID is part of the lead, the account name is merged "on the fly" and cannot be used for sorting. 

Resolution

This is the current system behaviour.
