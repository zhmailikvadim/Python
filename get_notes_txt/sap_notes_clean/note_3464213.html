Symptom

While executing SWPM in dialog mode receive "Running an installation on this distribution of Linux is not supported by SAP in any way!" 

Environment

UNIX/LINUX

Any DB

Product based on AS Netweaver

Reproducing the Issue

Executing sapinst (SWPM), it stops in few moments with the error message

Cause

The string "for SAP Applications" should not appear there.

Resolution

  1. Go to /etc/os-release 
  2. Modify PRETTY_NAME
    Example : PRETTY_NAME="SUSE Linux Enterprise Server for SAP Applications 15 SP1"
    replace PRETTY_NAME="SUSE Linux Enterprise Server 15 SP1" 
  3. Retry swpm

Keywords

PRETTY_NAME, /etc/os-release, SUSE Linux Enterprise Server
