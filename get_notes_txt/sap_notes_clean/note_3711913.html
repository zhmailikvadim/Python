Symptom

You need to send electronic documents for outgoing invoices in Convergent Invoicing using the PEPPOL network.

This SAP Note provides the following improvements:

  • Mapping of the Tax Currency Code (Company Code Currency)
  • If the Tax Currency Code (Company Code Currency) is different from the Document Currency Code, then the system adds a new line "TaxTotal" with the local currency in the XML.
  • Payment Means Due Date correction

  • When the Tax exempt code is ‘O’, then the eDocument XML must exclude the following fields:

    • Seller VAT ID
    • Buyer VAT ID
    • Seller tax representative VAT ID
    • VAT rate

   

Other Terms

 eDoc, e-document, e-Invoicing, electronic invoice, electronic invoicing, Document Compliance, Digital Compliance

Reason and Prerequisites

This SAP Note is only relevant for countries that participate in the Peppol network.

Prerequisites:

  • You use SAP S/4HANA with the eDocument Framework for Convergent Invoicing.
  • Convergent Invoicing is your source for billing documents.

Solution

Import the Support Package specified for your release or implement the attached correction instruction.

As a general rule, SAP recommends that you implement the Support Package specified for your release. However, if you need to install a solution earlier, use the SAP Note Assistant to implement the correction instruction. For more information, see SAP Note Assistant.
