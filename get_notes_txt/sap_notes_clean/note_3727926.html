Symptom

You try to send to carrier with B2B messages TransportationOrderrequest_Out XML for service order, but is not valid.

Environment

SAP Transportation Management embedded in SAP S/4HANA 2020+

Reproducing the Issue

  1. Attempt to use the PPF Action: TOR_TPNOR_REQ to trigger TransportationOrderrequest_Out XML.
  2. Observe that the action is not valid for Service Order and results in an error.

Cause

TransportationOrderrequest_Out is only valid for Freight Order (FO) or Freight Booking (FB). It is not supported for Service Order.

Resolution
