Symptom

In-app updates pop-up appears every time the SAP Cloud for Customer app is opened on mobile or tablet for the first time each day.

Environment

SAP Cloud for Customer

Reproducing the Issue

  1. Log in C4C app on a mobile or tablet.
  2. Observe the in-app updates pop-up prompting for download.

Cause

App Version Check Configuration
The app version check popup is enabled by default. While this feature can be disabled, it is primarily recommended for environments utilizing Mobile Device Management (MDM) for centralized deployment.

Environments Without MDM/MAM
In the absence of centralized version control, the popup is essential. Since app maintenance occurs at the individual device level, this behavior ensures all users remain compatible with the current major release and prevents fragmented versioning.

Environments With MDM/MAM
When MDM/MAM systems regulate the installed version across the fleet, the App Version Check can be safely disabled. In these controlled scenarios, the popup becomes redundant as version uniformity is guaranteed by the organization’s internal distribution policy.

Resolution

  1. Log in to the SAP C4C Administrator system
  2. Navigate to Administrator
  3. Select General Settings
  4. Go to Mobile Settings
  5. Select General Data
  6. Locate the option App Version Check and set it to No
  7. Locate the option Updates by Administrator and set it to Yes
