Symptom

You do not see descriptions of tax boxes in the generated reporting run for the PDV form of the VAT Return report for Croatia.

Other Terms

Croatia VAT, DRC, HR_VAT_DCL, PDV

Reason and Prerequisites

This SAP Note is relevant only for Croatia.

This SAP Note is applicable only for SAP Document and Reporting Compliance and report VAT Return - Forms PDV.

Solution

SAP recommends that you implement the Support Package specified for your release. If you need to implement the correction earlier, use the SAP Note Assistant to implement the correction instruction. You can find more information about the SAP Note Assistant in the SAP Support Portal under SAP Note Assistant.
