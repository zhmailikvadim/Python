Symptom

SAP Note 1947385 converted the DTP filter generation to the framework of the inverse transformation. The previous solution ignored start routines and end routines in relation to DTP filter generation and was therefore technically incomplete. Often, the partition criteria of a semantically partitioned object (SPO) is fulfilled using 1: 1 mapping. When a start routine is used in such scenarios, the implementation of a corresponding inverse routine is now required, as described in consulting note 1486659. When you implement this SAP Note, you can restore the previous, simplfied determination of the DTP filter criteria from the SPO partition criteria.

Other Terms

DTP filter, SPO partition criteria

Reason and Prerequisites

There is a program error.

Solution

See the attached correction instructions in the "Manual Activity" section.
