Symptom

Using the query enhancement customizing for BOPF Query Enhancements for Transportation Management, additional filter criteria are not taken into account during runtime.

Other Terms

/SCMTMS/CL_Q_SUPERCLASS; /SCMTMS/FRW_QUERY; /SCMTMS/CL_ENHNC_QUERY_CUST

Reason and Prerequisites

The customizing maintenance was extended with additional fields. This extension was not taken into account by the default implementation for the query enhancement BAdI which is using the query enhancement customizing.

Solution

Implement the attached Correction Instruction or the mentioned Support Package.
