Symptom

This is a Maintenance Patch release of SAP 3D Visual Enterprise Viewer 9.0 (FP09 MP1).

Some of the Fixes Include:

  • ECTR toolbar entries missing issue
  • Delta measurements visibility

Security Related Improvements:

  • FBX plugin has been sandboxed. Sandboxing is an industry-standard mechanism for ensuring processes are securely managed at the operating system level.

NOTE: Microsoft has now reached end of life for support of Windows 7; Jan 2020
https://www.microsoft.com/en-us/windows/windows-7-end-of-life-support-information
As of FP09 (the previous Feature Pack) VE Viewer is no longe certified against Windows 7 (will not be tested against this environment). We will support customers that maintain Windows 7, however cannot guarantee fixes if it is operating system dependent.

Other Terms

VE Viewer | VEV | 9.0 FP09 MP1 | 9.0FP09MP1 | 90FP09MP1 | 90FP9MP1

Reason and Prerequisites

SAP 3D Visual Enterprise Viewer 9.0 FP09 MP1 is valid as a standalone installation, as well as an upgrade to any version of SAP 3D Visual Enterprise Viewer 9.0.

Solution

SAP 3D Visual Enterprise Viewer:
This Maintenance Patch is delivered in a full product installer. It can be used to upgrade an existing installation of Viewer, or used to install the full product on it's own.

Download from the SAP Support Portal (Logon required):
Support Packages & Patches -> By Alphabetical Index (A-Z) -> V -> SAP Visual Enterprise

Steps to install:

  1. Double-click the executable to launch SAPSetup.
  2. Follow the steps presented by the SAPSetup wizard.
  • For an upgrade, FP09 MP1 will install into the location of the currently installed version of SAP 3D Visual Enterprise Viewer 9.0.
  • As a new install, follow the steps presented by the SAPSetup wizard to install the product.
