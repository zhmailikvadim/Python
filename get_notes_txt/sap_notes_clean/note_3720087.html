Symptom

The error message "Error 153 player configuration error" is displayed in the Youtube Video in Career Site.

Environment

SAP SUccessfactors Recruiting Marketing

Reproducing the Issue

  1. Access the Youtube Video in Career Site
  2. Attempt to play the video.
  3. Observe the error message: "Error 153 player configuration error".

Cause

The issue is caused by a custom component added in the CSB instance.

Resolution

  1. Navigate to CSB > Home Page > Open the Page > Custom Plugin Component.
  2. Remove the Custom Plugin Component from the page.
