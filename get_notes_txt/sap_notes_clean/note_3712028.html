Symptom

You are working in an EU localization. After making an update to the Federal Tax ID 1 you the following System Message:

Enter valid federal tax ID

Environment

  • SAP Business One 
  • SAP Business One, version for HANA

Reproducing the Issue

  1. Go to Administration > System Initialization > Company Details > Accounting data tab.
  2. In the Federal Tax ID1 specify invalid Id (should be 2 chars of country + number of digits) = enter wrong country code (e.g "LLL1234"). 
  3. Press Update 

Observe display of System Message - Enter valid federal tax ID.

Cause

The value entered is not in the correct format. 

Resolution

The tax ID should be a 2 character country code plus a number of digits depending on the country. 

Keywords

ID 10000298, error message
