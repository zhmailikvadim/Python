Symptom

You encounter one of the following problems:

  • The consideration of listing information for transportation lanes with source of supply type 4 "Supplying Site" does not work properly. Listing conditions outside of lane validity are nevertheless used.
  • Transportation lanes of the generic product are not used for a variant if there is a variant lane for the same supplier. It should be a valid lane, only with a lower priority.

Reason and Prerequisites

Program error.

Solution

Implement this note or the corresponding Support Package.
