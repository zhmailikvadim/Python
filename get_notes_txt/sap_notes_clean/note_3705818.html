Symptom

Error occurs while running the Termination Organizer or Payroll Driver in Fiori, resulting in the following message: "A back-end error has caused the application to terminate. Error Messaged returns: HTTPS 500." "Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Environment

  • Human Capital Management (HCM)
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4 HANA

Reproducing the Issue

  1. Running the Australia Payroll Driver or the Termination Organizer in a custom Fiori app.
  2. The process takes a long time and eventually displays the following message: "A back-end error has caused the application to terminate. Error Messaged returns: HTTPS 500.".

Cause

The processing time in Fiori exceeds the timeout setting. Please refer to the document "Timeout Options for ICM and Web Dispatcher" for more information about the timeout setting.

Resolution

To avoid the error, please use the Australia Payroll Driver with T-CODE PC00_M13_CALC_SIMU or Termination Organizer with T-CODE PC00_M13_CTO3 in SAP GUI.
If continuing with the Fiori app, please create a case with component BC-CST-IC to see whether the timeout setting can be extended.

See Also

SAP Note 824554 - ICM and SAP Web Dispatcher timeout parameters
