Symptom

You have posted a GI/GR or Posting Change for a CatchWeight relevant product which has a 14-digit decimal quantity in Base UoM. The system generates a WMTG* queue to post the remaining quantity to the Difference Analyzer, but this queue gets stuck with error message '/SCWM/GM067 - You must enter a catch weight quantity for item &1'.

Solution

Implement the Correction Instruction.
