Symptom

When we try to export/download the DO items data in operational report through Fiori in CSV, XLS and XLSX file formats, then in the downloaded file, the DO Item data is not showing appropriately. It is showing incorrect DO Item data.
When we try to open the downloaded file, we will get a popup screen, if we click on ‘Convert’ button in the popup window, then the DO Item data is shown incorrectly.

Other Terms

DO Item data, Operational Report, CSV Format, XLS Format, XLSX Format, File Downloaded/Export, /VTA/CL_BFTP_SERVICES, VALUE_INT_TO_EXT_CONVERT

Reason and Prerequisites

This is caused by program error.

Solution

Please implement the correction instructions attached. These corrections are only valid if you have applied the Support Packs.

Release V4 - 2021, 2022, 2022 V2, 2023, 2023V2, 2023 V3 and 2023 V3 Support Pack 1
