Symptom

During mass processing of purchase rebates to calculation buckets, the system is taking an excessive amount of time while sorting data in table EKBE. This sorting operation is causing performance degradation and delays in processing.


The implementation of the relevant SAP Note is expected to resolve the performance issue related to EKBE sorting.

Reason and Prerequisites

This is caused by program error.

SAP Note #3722544 for V4 Releases.

Solution

Please implement the correction instructions attached. These corrections are only valid if you have applied the Support Packs.

Release 6.4 Base Package or Support pack 1, 6.3 Base Package, 6.2 Base Package, and 6.1 Base Package.

Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object.
