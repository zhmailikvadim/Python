Symptom

  • Manager goes on leave, they assign an employee from the same organizational unit as a substitute.
  • When the substitute submits a leave request for themself, the request is incorrectly routed for approval to them as substitute of the first‑level approver.

Environment

  • Fiori HCM My Leave request/Approve leave request
  • WDA Leave request/Approve leave request

Reproducing the Issue

  1. Manager set an substitute from the same organization
  2. Substitute create a leave request

Cause

The business function HRMSS_SFWS_WDA_UI_02 is not activated.

Resolution

Activate the business function HRMSS_SFWS_WDA_UI_02 to prevent the leave request created by the substitute from being routed back to themselves.  

Note: The switch is applicable to both Fiori and WDA apps, as it is essentially an MSS switch.  
          Refer to the code parts also in Class CL_PT_ARQ_REQS_LIST method SEL_REQS_ATTSABS_FOR_NEXT_PROC for additional details.

Keywords

HCMFAB_LEAV_MAN, hcm.fab.myleaverequest, HRESS_AC_PTARQ_LEAVREQ_1, HRESS_A_PTARQ_LEAVREQ_APPL
