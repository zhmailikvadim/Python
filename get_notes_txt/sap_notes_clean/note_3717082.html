Symptom

A new value of T511K constant SIREV must be set valid from April 1, 2026.

The constant name is:
  PL 'Wskaznik waloryzacji podstawy wymiaru zasilku chorobowego'
  EN 'Valuation rate of sick pay base'

The new value is 105.10.

Other Terms

T511K, SIREV.

Reason and Prerequisites

The new rate is announced by ZUS on www.zus.pl and monitorpolski.gov.pl, based on average pay announcements by GUS on www.stat.gov.pl.

Solution

The new value for the constant has been set.

Please import the Support Package that includes this SAP Note or please follow the instructions under Manual Activity.
