Symptom

When trying to do any change in the sales order at item level system throws error message: V1348 - Schedule line category & is not defined in transaction VA02.  

Environment

  • Sales and Distribution (SD)
  • SAP ERP Central Component
  • SAP ERP
  • SAP Enhancement package for SAP ERP
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Go to VA02.
  2. Open the Sales order.
  3. Do a change at item level (For example change the quantity).
  4. System shows error message: V1348 - Schedule line category & is not defined. 

Cause

Missing customizing in transaction VOV5 as the system is trying to find the combination Item Category + MRP Type + Schedule Line Category. As it doesn't exist, the error message will be displayed.

Resolution

Maintain the correct combination for ITEM Category, MRP type and schedule line category in transaction VOV5.

See Also

SAP Note 20772 - V1348 during rescheduling/backorder processing

Keywords
