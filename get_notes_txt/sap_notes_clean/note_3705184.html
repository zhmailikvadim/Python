Symptom

  • An error occurs when refreshing a Web Intelligence (Webi) report that uses a HANA Input parameter:
    >> "sapsqldbcclient::getMembers() cube is NULL"

Environment

  • SAP BusinessObjects Business Intelligence Platform 4.3 Support Package 05 Patch 02
  • Web Intelligence (WebI)
  • Information Design Tool
  • SAP HANA

Reproducing the Issue

  1. Create a HANA calculation view with Input parameter in it.
  2. Create a universe in Information Design Tool based on the Calculation view.
  3. Consume the IDT universe to create a Web Intelligence document.
  4. Run the Web Intelligence report. In the HANA Input Parameter, list of values, the error is observed:
    >> "sapsqldbcclient::getMembers() cube is NULL".

Cause

Unused or out‑of‑context objects in the Web Intelligence document query triggered a getMember() lookup on a non‑existent HANA cube, and hence, enabling Query Stripping removed those unused objects, preventing the NULL cube call.

Resolution

  1. Apply query stripping to the affected Web Intelligence report.
  2. If the issue still persists, apply the same query stripping settings to the universe to prevent the error from recurring when creating Web Intelligence reports.

Refer to: 2061239 - BI 4.1: How to enable query stripping for Web Intelligence Document with universes designed with Information Design Tool
