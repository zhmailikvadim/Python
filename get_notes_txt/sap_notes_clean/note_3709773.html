Symptom

When you search for a tool in the successfactors search bar, it doesn't appear for you, despite having permissions to visualize it.

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Environment

SAP SuccessFactors HCM Suite

Reproducing the Issue

  1. Log in to the affected SAP SuccessFactors instance.
  2. Attempt to access the tool, example: "Proxy Import" or "Proxy Management" tools.
  3. Observe that the tools are not visible despite the required permissions being granted.

Cause

Key-words on Manage Action Search were inactivated.

Resolution

  1. Navigate to the "Manage Action Search" tool within the SAP SuccessFactors Admin Center.
  2. Locate the module where the tool is not appearing, by example, proxy-related key words, including "Proxy Management" and "Proxy Now."
  3. Go to "Actions" and activate the identified key words.
  4. Save the changes and verify that the "Proxy Import" and "Proxy Management" tools are now accessible, or any other tool that had this option disabled as per screenshot attached.
