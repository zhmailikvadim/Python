Symptom

You use transaction FMOPER to configure open budgeting periods. In the menu you select "Utilities, Change Logs" to display past configuration changes. However the underlying tables FMOPENPER and FMOPENPER_FUND are not enabled for change logging.

You have activated table logging in accordance with your audit requirements and checked the set of tables to log as described in SAP Note 112388. When you do this, you notice that tables FMOPENPER and FMOPENPER_FUND are not enabled for data logging.

Other Terms

transaction FMOPER - report RDDPRCHK - DD09L

Reason and Prerequisites

Configuration error

Solution

Import the respective Support Package or implement the attached manual correction instructions.
