Symptom

You are posting valuation cashflow items via RECEEP and reversing it with RECEEPRV. The original and reversal docs are posted in different ledgers in FI.

Other Terms

Ledger LDGRP ACCPR UPA VICECFRULE CERULE

Reason and Prerequisites

Program Error

The error occurs when you have defined a specific ledger in the valuation rule IMG settings in addition to the accounting principle.

Solution

Implement the attached Correction Instruction (CI) or the relevant Support Package.
