Symptom

After upgrading from Job Description Manager (JDM) to Job Profile Builder (JPB/JDM v2) the 'Job Description' field is still appearing in the Career Worksheet, when browsing Job Roles, and in Succession when nominating employee to a position

Environment

  • Career Development Planning
  • Succession

Reproducing the Issue

Go to the Development Plan

> Career Worksheet

> Browse Roles

> Add Role

> Expand a role and you can see the 'Job Description' field:

Cause

This field is from the Job Description Manager and hardcoded to the system

Resolution

It is not currently possible to remove this section. If having this removed is important for your company, you could open an enhancement request to have this put on the roadmap for a future release.
