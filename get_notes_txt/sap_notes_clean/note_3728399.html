Symptom

  • Users are unable to find the "Smart Discovery" option in the "Plus" (+) menu or the "Insert" toolbar of a Story.
  • The Smart Discovery feature appears is missing entirely from stories.
  • A message may appear suggesting that the feature is deprecated or not supported in the current design mode.

Environment

SAP Analytics Cloud 2026.2.10

Reproducing the Issue

  1. Access the SAC tenant.
  2. Go to the Stories section.
  3. Notice that the option to create a story "From a Smart Predict" is not visible.

Cause

The Smart Discovery feature was built on the Classic Story infrastructure. As part of SAP’s product roadmap, the Classic Story experience is being phased out and replaced by the Optimized Story Experience.

Because the Optimized Experience uses a completely redesigned architectural stack to provide better performance and stability, the legacy Smart Discovery feature has been decommissioned and is not compatible with this newer standard.

Resolution

To align with SAP's vision for Augmented Analytics, the functionalities previously provided by Smart Discovery have been transitioned into more focused, high-performance tools within the Optimized Experience.

Recommended Alternatives:

  • For Identifying Data Drivers: Use Smart Insights 
    • Instead of generating a separate dashboard, trigger Smart Insights directly on any data point or chart.
    • This provides a real-time side panel showing the top "Key Influencers" and contributors to that specific value.
  • For Natural Language Exploration: Use "Just Ask"
    • Use the new conversational AI interface to ask questions in plain English. The system will automatically generate the most appropriate visualization, which can then be pinned to your story.
  • For Ad-hoc Data Discovery: Use the Data Analyzer
    • For deep-dive, pivot-style exploration without the need for a formatted story, use the Data Analyzer tool.
