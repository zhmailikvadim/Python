Symptom

Account from the Manual Adjust process step is allocated via the error account.

Other Terms

/BA1/CL_AL_BR_COMMON_OBJ_CUST, /BA1/IF_AL_BR_CUST_ALLOC_BAL, GET_POSITION_OFFSET_ACC

Reason and Prerequisites

Program error

Solution

Implement attached correction instruction and execute steps described in manual activity.
