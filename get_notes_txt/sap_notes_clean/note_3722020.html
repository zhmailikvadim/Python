Symptom

  • The KSEF field is not visible in the RFUMSV00 report after following the guide from note 3469491 - Including the KSeF Field in the Advance Return for Tax on Sales/Purchases Report (RFUMSV00)

Environment

Relevant for ECC releases only

Reproducing the Issue

  1. Access program RFUMSV00 or transaction S_ALR_87012357.
  2. Run the report 
  3. Notice that there is no KSEF field

Resolution

Field KSEF is filled under field "User field 4" in RFUMSV00, it is not mentioned into the note, but it is visible in the .txt files

Example 
    CHECK i_handle = 'HAN1' OR i_handle = 'HAN3'.
    LOOP AT ch_tab_fieldcat ASSIGNING FIELD-SYMBOL(<fs_field>).
      CHECK <fs_field>-fieldname = 'USER_FIELD_4'.
      CLEAR ls_dfies.
      CALL FUNCTION 'DDIF_FIELDINFO_GET'
        EXPORTING
          tabname   = 'ZPL_KSEF'
