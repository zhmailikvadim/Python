Symptom

In the Fiori application F2795 (Task Logs), when viewing the details of the Currency Translation log, you may observe two columns, each labeled "Exchange Rate Indicator." This naming does not accurately reflect the distinct data each column holds.

Other Terms

F2795, FIN_CS_LOG

Solution

Implement the automatic correction instruction from this SAP Note or implement the corresponding Support Package.

Clear backend and frontend cache:

  • /n/IWBEP/CACHE_CLEANUP - Clear backend cache
  • /n/IWFND/CACHE_CLEANUP - Clear frontend cache
