Symptom

In the Edition Monitor application, the Change Request list is displayed with multi-row selection enabled. However, selecting rows beyond the lead selection has no effect — the UI does not react to additional row selections. Only the lead-selected row is processed. As a result Reschedule button on toolbar is not updated properly.

Other Terms

Edition Monitor, Display Change Requests, CL_USMD_EDITION_CREQ_GUI_WD

Reason and Prerequisites

ALV table does not react to other than lead selection and that's why toolbar is not updated when other selection is done.

Solution

Implement this SAP Note or update to corresponding Support Package.
