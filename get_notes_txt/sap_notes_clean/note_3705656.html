Symptom

After running the report /SCWM/ERP_STOCKCHECK in order to move existing rounding residuals to the difference analyzer you get the message "Rounding residuals posted to Difference Analyzer with warehouse tas &1".
When searching in the Warehouse Monitor (/SCWM/MON) for the Warehouse Task (WT) created you cannot find the WT that was created.

Environment

  • SAP Extended Warehouse Management (SCM-EWM);
  • SAP Extended Warehouse Management Warehouse Order Processing (SCM-EWM-WOP);
  • SAP Extended Warehouse Management Physical Inventory (SCM-EWM-PI);
  • SAP S/4HANA;

Reproducing the Issue

  1. Activate processing of rounding residual warehouse tasks.
  2. Post residual quantities in the report /SCWM/ERP_STOCKCHECK.
  3. Observe the popup message indicating warehouse task creation.
     "Rounding residuals posted to Difference Analyzer with warehouse tas &1".
  4. Search for the warehouse task in the warehouse monitor and note that it is not displayed.

Cause

The Warehouse Tasks created are Rounding residual WTs.
By default the Rounding residual warehouse tasks are not displayed in the warehouse monitor.

Resolution

  1. Access the Warehouse Monitor (/SCWM/MON).
  2. Select the Warehouse Task node.
  3. In the selection screen, enter the WT number and make sure the field "Rounding Residual WT" is set to X - Yes (Include RR warehouse tasks).
