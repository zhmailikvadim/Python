Symptom

When maintaining Tax Number Registration Status information using the BP Tax API, the maintained data is not retained after the API execution. Although the API call appears to complete successfully, the registration status information is lost or not updated correctly in the Business Partner tax data. As a result, the expected Tax Number Registration Status is not visible in the system after processing.

Other Terms

BUS_TAX, BUS_EI_STRUC_BAPITAX, TAXSTATUS, BUPA_TAX_ADD, BUPA_TAX_CHANGE

Reason and Prerequisites

Program Error

Solution

In general, SAP recommends that you install a solution by applying a Support Package. However, if you need to install a solution earlier, use the Note Assistant to implement the correction instruction. You can find more information about the Note Assistant in SAP Service Marketplace, under service.sap.com/note-assistant.
