Symptom

The error message "Application log information for this run is not available" is displayed when attempting to access tax trace details.

Environment

SAP Business ByDesign

Reproducing the Issue

  1. Go to Supplier Invoicing Work Center.
  2. Go to Invoice and Credit Memos View.
  3. Search for the relevant supplier invoice.
  4. Click Edit.
  5. Select View All.
  6. Navigate to Items tab.
  7. Go to Taxes subtab under Items tab.
  8. Click to display the tax trace details.
  9. The error message: "Application log information for this run is not available." appears.

Cause

The supplier invoice is with status "Posted". The tax trace details was not enabled before posting the supplier invoice. Hence the system does not show any details in the application log.

Resolution

Before posting the supplier invoice, the tax trace details needs to be enabled. Follow the below mentioned steps to enable it.

  1. Go to Supplier Invoicing Work Center.
  2. Go to Invoice and Credit Memos View.
  3. Search for the relevant supplier invoice document.
  4. Click Edit.
  5. Select View All.
  6. Navigate Taxes subtab under Items tab.
  7. Enable the "Active Tax Trace" option from the Actions menu.
  8. Save the changes before posting the Invoice document.
  9. Click to display the tax trace details.
  10. The tax trace details will be available in the application log.
