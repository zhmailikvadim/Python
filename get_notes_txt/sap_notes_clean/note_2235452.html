Symptom

During replication of employee data from Employee Central to SAP HCM an error message 054 PAOCF_EC_SERVICES "No secondary infotype &2 found for infotype &1 in period (&3 - &4)" might occur if dependents data is processed for infotype 0021 with secondary infotype records.

Other Terms

Package PA_SE_IN
Namespace http://sap.com/xi/PASEIN
Service interface EmployeeMasterDataReplicationRequest_In
Message type EmployeeMasterDataReplicationRequest

Reason and Prerequisites

Program error

Solution

Implement correction via SNOTE
