Environment

HANA Cockpit 2.0

Cause

You used a lesser-privileged DB user for the Cockpit upgrade according to the document below. However, XS Advanced runtime is updated as well during HANA Cockpit upgrade, so SYSTEM user has to be used. Note: Non-SYSTEM user does not work for update of XS Advanced runtime even if all system privileges are granted.

Create a Lesser-Privileged Database User for Update | SAP Help Portal
A lesser-privileged database user cannot be used to update an XS advanced runtime installation. The SYSTEM user is required and needs to be temporarily reactivated for the duration of the update.

Resolution

User SYSTEM user instead for the upgrade. Reactivate it in case it has been deactivated.

Create a Lesser-Privileged Database User for Update | SAP Help Portal
