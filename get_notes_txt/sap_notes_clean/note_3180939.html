Symptom

Resolution

When buyers on SAP Ariba Procurement solutions integrated with SAP ERP using SAP Business Network Adaptor for SAP NetWeaver created an SES for some of the child line items in a purchase order, the accounting allocation in SAP Ariba Procurement solutions was not posted correctly in SAP ERP.

The problem occurred because the LINO_NO in accounting distribution was not mapped appropriately with the LINE_NO in the service lines and unplanned services in SES.

The issue has been resolved by appropriately mapping the line numbers in accounting distribution with the line numbers in the service lines and unplanned services in SES.

Technical changes:

Following mapping changes are made:

Mapping: MM_WSServiceEntrySheetPush_RequestMessage_2_ARBA_CREATEENTRYSHEET_CREATE

SCV: ARIBA_ONDEMAND_SRVENTRYSHEET CI9 of ariba.com

 

Changes:

Target field:

/ns1:_-ARBA_-ENTRYSHEET_CREATE/ENTRYSHEETSRVACCASSVALUES/item/LINE_NO

Source field:

1./ns0:ServiceSheetExportRequest/ns0:ServiceSheet_ServiceSheetLineItemExportWS_Item/ns0:item/ns0:ReceiptItems/ns0:item/ns0:ServiceLineNumber

2./ns0:ServiceSheetExportRequest/ns0:ServiceSheet_ServiceSheetAccountingExportWS_Item/ns0:item/ns0:ReceiptItems/ns0:item/ns0:Accountings/ns0:SplitAccountings/ns0:item

 

ACTION REQUIRED FOR NEW AND EXISTING CUSTOMERS

  1. Log in to SAP Ariba Connect (https://support.ariba.com/Connect/Log_In). If you do not have a User ID and Password for Ariba Connect, contact your Ariba account executive.
  2. On the Home tab, click Product Information and Documentation.
  3. Click Cloud Integration.
  4. Click Resources. The Cloud Integration Resources page appears.
  5. In the Integration Tools section, click Integration tools for Ariba Procure-to-Pay.
  6. In the Integration Tools for SAP section, click SAP NetWeaver PI Mapping. If the list does not contain this entry, contact your Ariba account executive. The SAP NetWeaver Mapping (SAP Web Service Only) page appears.
  7. Click and download the SAP NetWeaver Mapping (SAP Web Service Only).zip file to a specified location on your hard drive.
  8. Extract the SAP NetWeaver Mapping (SAP Web Service Only).zip file and navigate to the following folder based on the version that you are using:
    Cloud Integration version 9.0 SP2: Ariba_P2P_SAP_Netweaver_Repository_CI9_SP2 > HF45
  9. Navigate to the ximappings folder and import the XI7_1_ARIBA_ONDEMAND_SRVENTRYSHEET_CI9_of_ariba.com.tpz file into your SAP Process Integration system.
