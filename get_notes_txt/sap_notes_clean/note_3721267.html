Symptom

Although you implemented SAP Note 3454705 in your system, you still encounter runtime error COLLECT_OVERFLOW_TYPE_P in method _INTERNAL_TO_GLU1 of class CL_FINS_ACDOC_GUSL_ADAPTER.

Other Terms

CX_SY_ARITHMETIC_OVERFLOW,  CL_FINS_ACDOC_GUSL_ADAPTER====CP, CL_FINS_ACDOC_GUSL_ADAPTER====CM00C

Reason and Prerequisites

The system summarizes values of technical field TIMESTAMP or LAST_CHANGE_DATETIME together with amounts, which causes the overflow error.

Solution

After implementing this SAP Note, the system clears the technical fields TIMESTAMP and LAST_CHANGE_DATETIME during summarization to prevent runtime error COLLECT_OVERFLOW_TYPE_P in method _INTERNAL_TO_GLU1 of class CL_FINS_ACDOC_GUSL_ADAPTER.
