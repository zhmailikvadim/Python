Symptom

You use Change Request Management to control your software distribution.

In CRM_UI you trigger action "Transport into Sandbox". After you select transport source and target system, then press the button "Execute", the following error message appears:

"Transport to sandbox failed"

Other Terms

ChaRM, ToC, Transport of Copies, Sandbox, System, Import, CHM1_ACTION_LOG, 076, CHM1_ACTION_LOG076

Reason and Prerequisites

Your transport landscape consists the same system on several transport tracks. The check of transport track for sandbox system is unnecessary.

Solution

Implement attached corrections to your solution manager system by using the Note Assistant (transaction SNOTE), or import the relevant Support Package.
