Symptom

A record in the worklist for application Postprocess Event-Based Postings - Overhead Calculation cannot be processed. Simulate or Post is canceled with a dump in backend (in class CL_FCO_RTOHC_PERSISTANCE_CALC).

Other Terms

KZPR F5763

Reason and Prerequisites

Application log assigned to a worklist item does not exist any more. The fix avoids the dump, but can't fix the root cause (missing application log).

Solution

Apply this SAP Note or corresponding Support Package.
