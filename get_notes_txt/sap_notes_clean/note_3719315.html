Symptom

The customer reported that the "EDIT" option, previously available for the PTR DRC report (AU_PTR_REP), is no longer appearing in the system.

Environment

  • SAP S/4HANA
  • SAP S/4HANA Finance

Reproducing the Issue

  1. Navigate to the statutory report functionality in SAP S/4HANA.
  2. Attempt to access the data preview screen.
  3. Observe that the 'edit' button is not visible.

Cause

It's a summary amount and calculated amount, editing is not an option and misleading.

Resolution

In the new AU-PTR amendment version, the editable feature is disabled.

Keywords

missing edit button, data preview screen, run statutory reports, Australia Payment Times Report (PTR), AU_PTR_REP, Payment Times Report (PTR), SAP S/4HANA
