Symptom

  • An error occurs during the Form I-9 Audit Trail Data Migration report from Onboarding 1.0 to Onboarding 2.0.
    • Error message: "Unhandled exception occurred during API request, message: Could not deserialize the response body stream as KMS.XpressHR.Core.DataMigration.RestClient.ErrorResponse."
    • Status code: 401.

Environment

SAP SuccessFactors Onboarding 1.0

Cause

Missing permissions for the user initiating the migration, or for the API Connection user

Resolution

  • Ensure the user initiating the migration has the following permissions: Administrator > Compliance Object Permissions > I-9 Audit Trail with View, Edit, and Import/export permissions with a role that has External user target population.
  • Ensure that the API Connection User has the following permissions: Administrator > Compliance Object Permissions > I-9 Audit Trail with View, Edit, and Import/export permissions with a role that has External user target population.

To check the API Connection User:

  1. Go to Super Admin
  2. Go to Features
  3. Go to SF_BizX_Connection
  4. The user is in the field API Authentication User
