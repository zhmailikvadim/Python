Symptom

  • A report connecting to a MySQL database hosted on another server encounters connectivity issues when accessed via Crystal Reports and Crystal Server.
  • Error messages displayed:
    • "Failed to retrieve data from database."
    • "Failed to retrieve data from database. Details: 08501: Lost connection to MySQL Server."
  • The connection test in ODBC Data Source Administrator (64-bit) is successful.

Environment

  • Crystal Reports 2020 Designer
  • Business View Manager

Reproducing the Issue

  1. Connect a report to a MySQL database hosted on another server using Crystal Reports or Crystal Server.
  2. Observe notifications indicating that tables from the database cannot be found.
  3. Encounter error messages:
    • "Failed to retrieve data from database."
    • "Failed to retrieve data from database. Details: 08501: Lost connection to MySQL Server."

    Cause

    The driver version is outdated.

    Resolution

    1. Verify the driver version on a machine where the connection works as expected and compare it to the problematic machine.
    2. Update the driver version to a supported one.
    3. Ensure the DSN is 64-bit and not 32-bit.
