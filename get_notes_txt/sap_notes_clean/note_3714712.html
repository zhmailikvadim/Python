Symptom

When a Purchase Order is created, the Output Management log displays the warnings "Form template name can't be initial." and "Email template name can't be initial."

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Create a Purchase Order in the system.
  2. Check the Output Management log for warnings.
  3. See the warnings "Form template name can't be initial." and "Email template name can't be initial."

Cause

The Form Template rule is only recognizing the 'Print' channel. The system requires a Form Template specifically assigned to the 'Email' channel. 

Resolution

Create a new rule for the 'EMAIL' channel:

  1. Open the "Output Parameter Determination" app.
  2. Select "Purchase Order" under the "Show Rules" section.
  3. Navigate to the "Form Template" determination step and ensure there is a row that covers the EMAIL channel.
  4. Activate the changes.
  5. Create a new purchase order to test the updated determination rules, as the system requires a fresh document to apply the revised rules.

Optionally, leave the 'Channel' column blank to apply the same Form Template to all channels.

Refer to the document Setting Up Output Management (1LQ) to ensure all setup steps are properly configured.
