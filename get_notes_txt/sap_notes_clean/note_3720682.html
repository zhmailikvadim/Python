Symptom

Pre-requisites check message is seen while connecting to BI Platform from Lumira Designer 2025 client

Other Terms

BI Platform

Reason and Prerequisites

1) Launch Lumira Designer 2025 client

2) In Connection tab , provide the BI Platform details

3) Click connect

>> Observe that Pre-requisitied check message is seen

"The following prerequisites have not been met. This can prevent the design tool from functioning correctly.

The design tool is newer than the deployed runtime version on the server. You might experience different runtime visualization or behavior between design tool and server execution."

Solution

workaround:

Click ok button and proceed with workflow

NOTE : Fix will be available in Lumira 2025 Patch1
