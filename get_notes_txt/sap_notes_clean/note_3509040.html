Symptom

You are using within your FPM Application a Search UIBB. For this search configuration you have configured a LIST ATS Configuration as result list. This list will not be shown on the UI at runtime.

Other Terms

FPM_SEARCH_UIBB

Reason and Prerequisites

Prerequisites: The Search configuration does not have any search attribute statically configured but a result list is configured.

Solution

Please implement the attached correction into your system.
