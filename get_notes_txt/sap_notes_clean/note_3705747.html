Symptom

  • Provisioning of the BDC cockpit failed with the error message: "The host name XXX is already being used. Please create a new solution with another host name."
  • The error occurred despite no existing BDC cockpit being provisioned.

  • Make sure to expand the Solution type entry. Without expanding the hierarchy, the following error is displayed:
    Solution provisioning has failed. Please contact customer support.
    This solution also has 1 message on one of its tenants. Please expand the solution and review the tenant notification.


Environment

SAP Business Data Cloud Cockpit

Reproducing the Issue

  1. Attempt to provision a BDC cockpit using the host name "XXX".
  2. Observe the error message indicating that the host name is already in use.

Cause

The host name "XXX" is already registered by another user within the same region, causing a conflict during provisioning.

The host name must be unique within the region/hyperscaler. Another customer in this same region/hyperscaler has already created a BDC using this name. Please choose a different name.

Resolution

  1. Select a unique host name for the BDC cockpit that has not been used within the region.
  2. Retry the provisioning process with the updated host name.
