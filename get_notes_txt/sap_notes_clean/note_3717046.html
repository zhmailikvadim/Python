Symptom

You use the FB03L to try to display an FI Document.
If you use the 'Reference No' as a selection and additionally have the Ledger filled, documents cannot be correctly determined.

Other Terms

XBLNR, Reference Number, Ledger, Ledger Group, RLDNR, LDGRP,

Reason and Prerequisites

Ledger Group not correctly determined

Solution

Import the support package or implement the manual advance correction.
