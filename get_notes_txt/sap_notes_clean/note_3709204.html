Symptom

In the TW Wage Type Reporter HTWCWTR0, the system fails to display decimal values for wage types with numeric fields, and all values are rounded up to integers. 

Environment

  • Human Capital Management (HCM)
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP SuccessFactors Employee Central Payroll
  • SAP S/4 HANA

Reproducing the Issue

  1. Run the program HTWCWTR0 via transaction code SE38, or go to transaction code PC00_M42_CWTR.
  2. Execute HTWCWTR0 for a Taiwan employee, then check numeric fields in the output. For example, wage type 1000 is 3000.5 in the RT table of payroll results but displays as 3001 in the HTWCWTR0 report. 

Cause

This is known issue. 

Resolution

Apply SAP Note 3679374 - HTWCWTR0: Incorrect Amount for Non-TWD Currency.

Keywords
