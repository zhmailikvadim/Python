Symptom

  • Following error occurs when refreshing Web Intelligence report based on SAP Datasphere Analytic Model with restricted key hidden calculation measures : "Database System returned state 'DATA_ACCESS_PROBLEMS'. Message = The used variable is not input enabled"

  • The message indicates that certain variables (e.g., date-related or input-specific variables) are not input-enabled, causing their provided values to be overwritten by default values. Additionally, some base measures or dimension attributes (e.g., inventory-related metrics) cannot be found, and certain custom members (e.g., formula-based calculations) reference non-existent members in dimensions.

  • This error specifically arises when the "Is Auxiliary" option is enabled at the backend level in the data platform for restricted key hidden calculation measures.

  • Conversely, if the "Is Auxiliary" option is disabled, the report refreshes successfully without issues.

Other Terms

webi, webintelligence, BICS, DWC, calculated, RKF, refresh, prompt, input, variable,

Reason and Prerequisites

Environment : 

  • SAP BusinessObjects Business Intelligence platform 4.3 Support Package 04 Patch 15
  • Web Intelligence

 Steps To Reproduce :

  1. Logon to BI Launchpad
  2. Open Web Intelligence report based on SAP Datasphere Analytic Model containing restricted key hidden calculation measures with "Is Auxiliary" setting enabled
  3. Refresh the report
  4. In Prompts window, enter the required values and click Run
    1. >> Error occurs

Cause :

  • This issue is caused by a defect in the product.

Solution

This issue is fixed in the patches listed in the "Support Packages & Patches" section below.
Due to different internal processes, the "Support Packages & Patches" section of the SAP Note will be populated as below:

  • for fixes delivered in "Support Packages": immediately (even if the "Support Package" is planned to be released several months later)
  • for fixes delivered in "Patches": ONLY once the "Patch" is released AND available on SMP (not before)
