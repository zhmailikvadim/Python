Symptom

When trying to open Account Changes in XD03 for a Customer, the following error is raised:

System error: Error in routine POSITIONEN_LESEN RC = 1
Message No. F4308

Environment

  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Start transaction XD03 (or XD02).
  2. Navigate to Environment → Account Changes → All Fields.
  3. Observe that the system shows error F4308: "System error: Error in routine POSITIONEN_LESEN RC = 1".

Cause

An inconsistency exists in the change document data for the customer master: a CDHDR entry is present without a corresponding CDPOS entry. This inconsistency causes the error during reading of change document items.

Resolution

The report RSSCD_CHECK_CDHDR_ALL from Note 1245574 can be used to identify and correct the inconsistencies.

If too many entries exists, the report may run into Out of Memory errors, in this case, it is possible to find the exact date for which consistencies exist by following these steps:

  1. Query table CDHDR with OBJECTID = <Customer Number>.
  2. Copy all document numbers in CDHDR (CHANGENR) and query table CDPOS with the same field.
  3. Compare these tables and find which documents are missing from table CDPOS.
  4. Execute RSSCD_CHECK_CDHDR_ALL with the date of missing documents (CDHDR-UDATE) and Change doc. object = DEBI.

The report can be executed in test mode first to identify how many entries will be corrected.
