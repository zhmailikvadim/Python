Symptom

In the app "Manage Launchpad Pages", you are experiencing a poor performance when opening a page that is assigned to numerous roles.

Other Terms

F4512, spaces design-time, /sap/opu/odata/UI2/FDM_PAGE_REPOSITORY_CUST_SRV/pageSet('...')?$expand=roles%2Fcatalogs%2FvizReferences, navigation to page detail, SAP_UI 7.58, ABAP Platform 2023, S/4HANA 2023

Reason and Prerequisites

 Due to a programming error, catalog content is redundantly processed for every role.

Solution

Install the corresponding support package (recommended) or import the corrections of this SAP Note.
