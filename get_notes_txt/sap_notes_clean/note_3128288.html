Symptom

An unnecessary variance rounding of 0.01 is calculated and posted on production order closing if you are working with high amounts.

Reproducing the issue

Scenario:

  1. Create two new items: a parent (moving average valuation method) and a component.
  2. Add a goods receipt: component, Quantity 1, Unit Price = 5,358,383.99.
  3. Add a production order: Special production order, parent Planned Quantity = 28,814.40; component Planned Quantity = 1, Manual issue method. Set the production order to Released.
  4. Add an issue for production: component Quantity = 1 (full quantity).
  5. Add a receipt from production: parent Quantity = 28,814.40 (full quantity).
  6. Set the production order to Closed and click Journal Entry Preview in the toolbar.

Expected behavior:

No variance is calculated and posted.

Actual behavior:

A variance of 0.01 is calculated on the journal entry between the inventory account and WIP inventory variance account.

Cause

System Limitation

Solution

This behviour is a system limitation.
