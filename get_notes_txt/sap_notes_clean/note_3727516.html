Symptom

Functional area created from SSCUI 102363 is not visible in app Manage Functional Area.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Create a functional area in SSCUI 102363.
  2. Navigate to the app Manage Functional Area.
  3. Observe that the functional area created in SSCUI 102363 does not appear in the app Manage Functional Area.

Cause

Only Functional Areas created within the app Manage Functional Area are visible there. Functional areas created in SSCUI 102363 lack required fields that are used within the Fund Management component, which the app Manage Functional Area is a part of.

Resolution

For Functional Area usage in Fund Management, create and maintain them directly within the app Manage Functional Area.
