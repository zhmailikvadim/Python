Symptom

You open a recipe in Display Multiple Recipes application and go to the Calculation Results tab, where you select the Standard Composition subtab. If you select the Display as Hierarchy button you'll see the ingredients hierarchy of the recipe with their compositions. Input items appear duplicated and duplicate entries have no description.

Other Terms

MFE, GET_COMPOSITION_LIST_HIERARCHY, GET_COMP_FOR_LEAF_ITEMS

Reason and Prerequisites

Program error

Solution

Implement the correction instruction or implement the support package of the SAP Note.
