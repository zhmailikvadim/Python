Symptom

When you try to display the Alv result from FIGLARTAXREP you receive a short dump as below:

Category               ABAP programming error
Runtime Errors         OBJECTS_NOT_CHARLIKE
ABAP Program           SAPLKKBL

 Short Text
     The current statement requires a character-type data object.

 What happened?
     Error in the ABAP application program.

     The current ABAP program "SAPLKKBL" had to be terminated because it found a
     statement that could not be executed.

 Error analysis
     In statement "STRLEN( obj )...", only character-type data objects are
      supported at
     argument position "obj".

     In this case, operand "obj" has the non character-type type "P LENGTH 6
      DECIMALS 6".

 Trigger Location of Runtime Error
     Program                                 SAPLKKBL
     Include                                 LKKBLF99
     Row                                     3,972
     Module Type                             (FORM)
     Module Name                             GEN_FIELD_OUT2

Environment

SAP ERP

Reproducing the Issue

  1. Execute the FIGLARTAXREP report
  2. Try to open the ALV data from result
  3. Observe the short dump with runtime error OBJECTS_NOT_CHARLIKE.

Cause

Missing logic from SAP Note 2879369 - Argentina Tax Reporting (RPFIGLAR_TAXREPORTING): Error in Download of Tax Details ALV for Daily VAT Option.

Resolution

Implement SAP Note 2879369 - Argentina Tax Reporting (RPFIGLAR_TAXREPORTING): Error in Download of Tax Details ALV for Daily VAT Option.

See Also

KBA 2651853 - Argentina Redesigned VAT Reports Troubleshooting Guide.
