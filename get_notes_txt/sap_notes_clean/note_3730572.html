Symptom

How to create a new plant in SAP Ariba Buying?

Environment

SAP Ariba Buying & Invoicing

Resolution

Kindly follow the next steps:

  1. Import Plant.csv: import the Ship To Address
  2. Import Address.csv: Purchasing Unit field is named as such, but looks for the Company Code. 
    1. Import the Bill To Address in this same file
    2. Determine which address is Ship To and Bill To (yes/no fields).
  3. Import Plant to Purchase Org Mapping (only for Plant Address): if not mapped, the Ship To address will not show up.

See Also

If the incorrect Bill To and Ship To address, kindly refer to this article: KBA 3190131 - Incorrect Bill To Address at header level of PO print.
