Symptom

The below error occurred in Resolve error app in BTP subaccount application 

Error occurred during the PUSH step: "The push to the destination returned the 403 HTTP status code. Check the destination configuration in your SAP BTP subaccount."

Environment

  • SAP ERP
  • SAP S/4HANA
  • SAP Document and Reporting Compliance, Cloud Edition
  • SAP S/4HANA cloud, private edition

Reproducing the Issue

  1. Open the DRC Cloud edition app.
  2. Navigate to Resolve error app

Cause

The HTTP service GCO_INTERNAL was not active.

Resolution

  1. Verify whether the HTTP service GCO_INTERNAL is active in the relevant system.
  2. If inactive, activate the HTTP service GCO_INTERNAL by following below help portal documentation

    SAP S/4HANA 2022 or Higher - Activating HTTP Service for SAP S/4HANA 2022 or Higher or for SAP S/4HANA Cloud Private Edition | SAP Help Portal 

    SAP S/4HANA 2021 or Lower - Creating HTTP Service | SAP Help Portal 

    SAP ERP - Activating HTTP Service | SAP Help Portal 

  3. Retry the eDocument processing or trigger the status update from the edocument cockpit.
  4. Confirm that the status updates automatically from "Acknowledge" to the correct "Accepted" or "Rejected" status.

See Also

SAP Note - 3346323 - eDocument (Poland) Troubleshooting Guide
