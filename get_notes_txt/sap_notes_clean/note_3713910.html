Symptom

In the process "Start of Supply MPES for Electricity (New SUP View)" (DE_B_MPES_SOS_SUP), the system sent an incorrect external code list ID from the S/4HANA system to the cloud in step 310. This affected the field SP_EXT_CODELIST_ID.

This issue is relevant for the market role: SUP.

Other Terms

MPES, Start of Supply, DE_B_MPES_SOS_SUP, DE_C_GPKE_CAN_SUPN, SP_EXT_CODELIST_ID, 55016, cancellation process, external code list ID

Reason and Prerequisites

The error occurred because of a logic error in the system.

Solution

Implement the corrections of this SAP Note or import the relevant support package.
After implementing this SAP Note, when a cancellation is triggered from the start of supply in the MPES process, the system passes the correct external code list IDs to the cancellation process. You need to retrigger the process. You can verify the fix by checking that no error occurs in the cancellation process while sending the 55016 message.
