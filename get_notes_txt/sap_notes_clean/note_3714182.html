Symptom

In the Fixed Asset Acquisition Form for Poland (PL_FXDAST_FORM_ACQ), the field Asset National Classification (KŚT) is empty in the generated PDF form, although the classification value is maintained in the Asset Master (for example in Evaluation Group fields).

Environment

  • SAP S/4HANA with Document and Reporting Compliance (DRC / ACR)

Reproducing the Issue

 

  1. Maintain the Asset National Classification in the Asset Master (for example in Evaluation Group fields such as ANLA-ORD41 or ANLA-ORD42).

  2. Execute report PL_FXDAST_FORM_ACQ via DRC.

  3. Generate the Fixed Asset Acquisition PDF form.

  4. The field KŚT / Asset National Classification is empty in the output.

Cause

The source field for the Asset National Classification is controlled by a report-specific configuration parameter in the DRC framework.

If this parameter is missing or not maintained, the report cannot determine which Asset Master field should be used as the source. As a result, the KŚT field remains empty in the generated form.

Resolution

Maintain the report-specific parameter for report ID PL_FXDAST_FORM_ACQ:

1. Open view cluster maintenance:

Transaction SM34
View cluster: SRFVC_RPG_ENT

2. Select the relevant Reporting Entity (e.g. Poland reporting entity).

3. Navigate to the report category:

PL_FXDAST_FORM_ACQ

4. Maintain the parameter:

G_FXDAST_NATL_CLFN_SRCE_FLD

5. Set the parameter value according to the desired source field for the Asset National Classification.

Supported values:

1 — National Classification Code (New Asset Master)
2 — Asset Super Number (ANLA-ANLUE) (Old Asset Master)
3 — Evaluation Group 1 (ANLA-ORD41) (Old Asset Master)
4 — Evaluation Group 2 (ANLA-ORD42) (Old Asset Master)
5 — Evaluation Group 3 (ANLA-ORD43) (Old Asset Master)
6 — Evaluation Group 4 (ANLA-ORD44) (Old Asset Master)
7 — Evaluation Group 5 (ANLA-GDLGRP) (Old Asset Master)

Note:
Value 1 can be used only for customers with the New Asset Master (UPA business function enabled).
For configurations with UPA disabled, values 2–7 can be used.

6. Save the settings.

7. Execute report PL_FXDAST_FORM_ACQ again.

The Asset National Classification (KŚT) field is now populated in the generated PDF form.

Note:
Only one source field can be used at a time.
