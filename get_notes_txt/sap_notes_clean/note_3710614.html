Symptom

When a data is selected from the incompletion log, the system navigates to an unexpected screen.

Environment

  • Sales And Distribution (SD)
  • SAP ERP Central Component
  • SAP ERP
  • SAP Enhancement package for SAP ERP
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. VA01/VA02 -> Edit -> Incompletion log.
  2. Selecting a data from the incompletion log and the system navigates to an unexpected screen.

Cause

The configuration for the incompletion procedure is incorrect.

Resolution

  1. In OVA2, open the corresponding incompletion procedure and check the incompletion fields.
  2. Verified whether the table and field names match those expected to be used for the incompletion check. 
  3. Ensured that the "Screen for creating missing data" (V_TVUVF-FCODE) is configured correctly to display the relevant screen for entering the missing data. 

Keywords

incompletion log, screen navigation
