Symptom

It takes a long time when trying to close a Pick List.

Reproducing the issue

Scenario:

1. Bin location is enabled and there are a large number of pick lists in your company.

2. Open SAP Business One Client, and go to an open pick list that isn't partially delivered.

3. Right-click and choose Close to close the pick list.

 

Expected Behavior:

The pick list closes quickly.

 

Actual Behavior:

The pick list takes much longer to close. For example, in one company with 2,725,690 PKL1 records, closing one pick list takes 40 seconds.

Cause

Program error

Solution

SAP intends to provide a feature package or patch to solve the symptom that is described in this SAP Note. The section References below lists the related feature package or patch once they become available. Be aware that References can only be confirmed at feature package or patch release date. SAP delivers feature packages or patches only for selected releases at its own discretion, based on the business impact and the complexity of implementation.
