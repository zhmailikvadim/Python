Symptom

  • An error occurs while attempting to update the logical file path "FI_DME_CREATE_PATH" by setting some word (Ex: /tmp/payment/<COMPANYCODE>/out/<FILENAME>) as a dynamic field in the FILE transaction.
  • An error occurs while attempting to update the file name by setting some word (Ex: /tmp/payment/<COMPANYCODE>/out/test) as a dynamic field file name in the FBPM (SAPFPAYM) transaction.

Environment

  • Financial Accounting
  • SAP R/3
  • SAP R/3 Enterprise
  • SAP ERP
  • SAP ERP Central Component
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA
  • SAP S/4HANA Finance

Reproducing the Issue

  1. Attempt to update the logical file path "FI_DME_CREATE_PATH" in the FILE transaction or file name in FBPM (SAPFPAYM) by setting some random word (Ex: /tmp/payment/<COMPANYCODE>/out/test)as a dynamic field.
  2. Observe that the error SG004 is triggered, stating "Company code cannot be used in the directory path, as it is not a reserved word."

Cause

  • The error occurs because that random word (Ex: Company Code) cannot be used in the directory path as it is not a reserved word.

Resolution

  • Specify logical path/ file name. You can include the following reserved words in pointed brackets. These are replaced with current values at runtime.
    Reserved Word       Replacement Text
    <OPSYS>             Operating system in call
    <INSTANCE>        R/3 application instance
    <SYSID>              R/3 application name in SY-SYSID
    <DBSYS>             Database system in SY-DBSYS
    <SAPRL>              R/3 release in SY-SAPRL
    <HOST>               Host name in SY-HOST
    <CLIENT>            Client in SY-MANDT
    <LANGUAGE>       Log on language in SY-LANGU
    <DATE>               Date in SY-DATUM
    <YEAR>               Year in SY-DATUM, 4-character
    <SYEAR>             Year in SY-DATUM, 2-character
    <MONTH>           Month in SY-DATUM
    <DAY>                Day in SY-DATUM
    <WEEKDAY>       Day of the week in SY-FDAYW
    <TIME>              Time in SY-UZEIT
    <STIME>            Hour and minute in SY-UZEIT
    <HOUR>             Hour in SY-UZEIT
    <MINUTE>          Minute in SY-UZEIT
    <SECOND>         Seconds in SY-UZEIT
    <PARAM_1>        External parameter 1
    <PARAM_2>        External parameter 2
    <PARAM_3>        External parameter 3

Examples:

  • Local directory for temporary files under Windows NT:
    D:\GLOBAL\ARCHIVE\<PARAM_1>\<FILENAME>
  • Local directory for temporary files under DOS:
    C:\TMP\<FILENAME>
  • Local directory for temporary files under UNIX:
    /usr/<SYSID>/local/tmp/<CLIENT>/<FILENAME>
