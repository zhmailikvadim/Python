Symptom

How to display existing tabs (for example, Tickets, Registered Products, Sales Data, Contacts) within the Overview pane of a business object. 

Environment

SAP Cloud For Customer

Reproducing the Issue

  1. Navigate to a business object and open the Overview pane.
  2. Turn on Adaptation.
  3. Select the Overview pane.
  4. Click Add under Adaptation.
  5. Observe that there is no option to add standard business objects inside the Overview pane.

Cause

By design, the Overview pane supports adding sections only; tabs are separate navigation areas and cannot be added inside the Overview pane.

Resolution

  1. If required, use the MOVE pane option to move specific business objects (for example, Tickets, Registered Products, Contacts) into the Overview; note that the moved content will no longer remain in its original section.
  2. If the needed content is not available via standard sections, implement a customization via PDI/SDK to expose the relevant business object data within the Overview sections.
