Symptom

  • The navigation bar in the WebUI is no longer available for the SAP-standard business role S4C_UT_IC.
  • The standard role, used to copy and enhance other business roles, is no longer functional for that purpose.
  • No specific error message or error code is mentioned.

Disclaimer: Image data in this KBA is from SAP internal systems. Any resemblance to real data is pure coincidence.

Environment

  • SAP S/4HANA

Reproducing the Issue

  1. Open the Web UI with the business role S4C_UT_IC.
  2. Observe that the navigation bar is not displayed.

Cause

The business role S4C_UT_IC has all work centers set to inactive, so the navigation bar does not show any links.



Resolution

  1. Start transaction WCF_CUST.
  2. Go to Business Role → Define Business Role.
  3. Select the business role S4C_UT_IC.
  4. Click Adjust Work Centers.
  5. Uncheck the Inactive checkbox for the relevant work centers.
