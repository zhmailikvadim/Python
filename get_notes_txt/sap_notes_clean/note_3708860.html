Symptom

 backoffice is unavailable 

 need to run a solr index cronjob via hac

Environment

  • SAP Commerce Cloud

Resolution

  1. Find out solrfacetsearchconfigration, in this example we use "solr config for backoffice"
  2. Use the following script in the HAC to initiate full indexing for Solr:

       import de.hybris.platform.solrfacetsearch.config.FacetSearchConfig;

     println("start backoffice full indexing");

     FacetSearchConfig facetSearchConfig=facetSearchConfigService.getConfiguration("solr config for backoffice"); // change it based on your facetsearchconfiguration name

     indexerService.performFullIndex(facetSearchConfig);

     println("finish backoffice full indexing");

Keywords

 Solr indexing, groovy, hac, full solr index cronjob.
