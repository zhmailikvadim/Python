Symptom

  • Systems overview page does not display after granting systems permissions on a custom role.
  • Error message: "File not found."
  • Error message: "The resource doesn't exist, or you don't have permission to read it."
  • Network response shows status 500 with messageId "NO_AUTH_READ"; sometimes a 404 is returned.

Environment

SAC Administration

Reproducing the Issue

  1. Log in with the affected user.
  2. Navigate to System > Overview.
  3. Observe that the page fails to load with "File not found" or "The resource doesn't exist, or you don't have permission to read it."
  4. In Security > Roles, note that an assigned custom support-related role has Full Data Access enabled, while another assigned general user role does not.

Cause

SAC Content folder is not shared with the affected users.

Resolution

  1. Navigate to System > Common in SAP Analytics Cloud.
  2. Locate the SAC Content folder.
  3. Share the SAC Content folder with the affected users, ensuring that existing subfolders and files are also shared.
  4. Confirm that the affected users can access the Systems Overview page after sharing the folder.
