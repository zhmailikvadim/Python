Symptom

Unable to manually update a client secret to add more unique characters to it after it has been generated within a cloud integration scenario

Environment

  • SAP Business Technology Platform
  • SAP Integration Suite
  • Cloud Integration

Cause

Client Secrets are automatically generated based on the information provided during set up.
Changing this value dynamically would cause authorisation issue.

Resolution

This is not possible as the secret generation is automatic and cloud integration does not allow for dynamic modification of this.
