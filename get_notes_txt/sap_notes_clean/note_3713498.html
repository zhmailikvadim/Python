Symptom

Upon displaying the filter of the 'Maintenance Activity Type' field in Fiori app Manage Maintenance Notifications and Orders, the corresponding label shows only 'Activity Type'. This short version of the Maintenance Activity Type can be misleading.

Upon using the value help of the 'Maintenance Activity Type' field filter, it can happen that only a dropdown is shown.
Be aware that the same value help is used for the Maintenance Activity Type fields in some Fiori apps like F2173 and F2175.

Other Terms

F4604, MMNO, C_MaintWrkReqAndOrdForPlanning, MaintenanceActivityType, DTEL EAM_MAINT_ACTIVITY_TYPE, DDLS C_MAINTWRKREQANDORDFORPLANNING, I_MaintActyTypeStdVH, resultSet, sizeCategory, MAT Description, F2173, Find Maintenance Order and Operation, Find Maintenance Orders and Operations, F2175, Find Maintenance Order, Find Maintenance Orders

Reason and Prerequisites

Instead of the 'Maintenance Activity Type' field label a shorter version was used, which has been modified for consistency and clarity.
The value help didn't display the Activity type itself, only the description was available which isn't necessarily unique.

Solution

Implement the attached Correction Instruction or the corresponding Support Package.
