Symptom

There is an issue with the balance sheet valuation procedures in actual costing/material ledger:

  • The system calculates Periodic Receipt Values (transaction/program CKMLBB_AGGREGATE).
  • The system displays Periodic Receipt Values (transaction/program CKMLBB_PERIODS_LIST).
  • The system calculates FIFO Prices (transaction CKMLBB_FIFO_CALCULATE/program CKMLBB_FIFO_CALCULATE).
  • The system displays Alternative Prices (transaction/program CKMLBB_PRICES_LIST).
  • The system processes Price Change with Alternative Prices (transaction/program CKMLBB_PRICES_CHANGE).

Other Terms

CKMLBB, CKMLBB0, CKMLBB_AGGREGATE, CKMLBB_PERIODS_LIST, CKMLBB_FIFO_CALCULAT, CKMLBB_FIFO_CALCULATE, CKMLBB_PRICES_LIST, CKMLBB_PRICES_CHANGE

Solution

SAP Support cases regarding the balance sheet valuation procedures in actual costing/material ledger are now processed in component MM-IM-VP (Balance Sheet Valuation Procedures). Do not use component CO-PC-ACT for these cases.

The component assignment change from CO-PC-ACT to MM-IM-VP is delivered via Support Package.
