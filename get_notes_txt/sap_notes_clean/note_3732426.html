Symptom

In the source system you have defined a unit of measurement, which does not exist in the target system. This unit of measurement is used as "Unit of measure for queue time" (WRTZE) in the operations of a shop floor order. After replicating the order to the target system, you receive information message "No language-specific unit defined in language &2 for internal unit &1" (BM302) and error message "Unit &1 is not created in language &2." (BM305) when navigating to the operation overview of transaction CO02 in the target system.

Other Terms

MPE, BM 302, BM 305, 302(BM), 305(BM), CUNI, AFVC-WRTZE, LOIPRO05, E1AFVOL

Reason and Prerequisites

Program error.

Solution

Apply correction instruction.
