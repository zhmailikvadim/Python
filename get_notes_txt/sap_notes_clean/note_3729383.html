Symptom

PR05/ WDA application FITE_EXPENSES/ Fiori travel app MTE: editing a trip in status 'transferred to FI' raises error message: 'Posting Run #### not yet posted. trip cannot be changed'  (error message ID 56 821).

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Environment

SAP Travel Management - on premise/ private cloud

Reproducing the Issue

Edit a trip in status 'transferred to FI'

Cause

Trip included in Posting run mentioned in the error message has not been posted correctly

Resolution

The Posting Run including the trip you are trying to edit has not been posted correctly (posting run ID is mentioned in the error message!)

Check KBA 1971166 - PRRW - Trip Posting Run Management: Posting run statuses for productive posting runs for more detailed information on posting run statuses in particular and how to resolve the inconsistency in trip posting.
