Symptom

After having applied SAP Note 3667901 the Holiday Pay calculation faces issues in case the calculation is performed in EUR4 currency.

The current SAP Note brings correction to the Payroll Function BHOLP to solve that issue.

Other Terms

Holiday Pay, BHOLP, EUR4, BH39

Reason and Prerequisites

You are using Payroll Localization Belgium (PY-BE)

Solution

As a general rule SAP recommends that you implement the Support Package specified for your release.

However, if you need to install a solution earlier, use the SAP Note Assistant to implement the correction instruction.

You can find more information about the SAP Note Assistant in SAP Support Portal, under SAP Note Assistant.
