Symptom

When exporting contracts using ARBCIG_CONTRACT_EXPORT, incorrect unit prices are sent to Ariba for currencies with decimal places other than two. The issue occurs because the price value is taken directly from the database and sent without conversion to the original amount based on the currency decimal configuration.

Other Terms

ARBCIG_CONTRACT_EXPORT, LARBCIG_CONTRACTU01 and ARBCIG_CONTRACT_GETDETAIL.

Reason and Prerequisites

The problem occurred due to an erroneous code.

Solution

The unit price is now converted using the appropriate currency conversion function module, ensuring that the original value is correctly populated according to the currency decimal configuration before being sent to Ariba.

Required Action

Apply the SAP Note
