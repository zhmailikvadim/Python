Symptom

You execute an action so that a pop-up is sent on which you can enter a reason code e.g., you create an "Order-Specific Routing" from the start screen of transaction CO02 "Production Order Change".

If you enter the reason code manually and do not select it via value help, then an error message is sent "Code of group in catalog is not relevant for these actions."

Other Terms

CO02, Change Production Order

Reason Code, OSR

GV_REASON_CODE_GROUP_CATALOG, GV_REASON_CODE_GROUP, GV_REASON_CODE

MPE_CORE001, MPE_CORE 001

Reason and Prerequisites

Program error: When manually entering the Reason Code, the corresponding Reason Code Group and Reason Code Group Catalog are not filled.

Solution

Implement the correction instruction
