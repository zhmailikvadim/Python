Symptom

In Hybris Administration Console under Monitoring -> Cache, “clear cache” button cannot clean the cache of webservice, e.g. productSearchCache cache in wsCacheManager cache manager which is defined in commercewebservices extension.

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental. 

Environment

SAP Commerce 2211 JDK21

Resolution

In commercewebservices extension, the cache manager wsCacheManager is defined in

/commercewebservices/web/webroot/WEB-INF/config/common/cache-config-spring.xml as:

<!-- Default cache manager for OCC: -->
<alias name="defaultWSEhcache" alias="wsEhcache"/>
<bean id="defaultWSEhcache" class="org.springframework.cache.jcache.JCacheManagerFactoryBean">
    <property name="cacheManagerUri" value="classpath:${commercewebservices.ehcache.location:commercewebservices/cache/ehcache.xml}" />
</bean>

<!-- instantiates a JCacheCacheManager instance using above created factory bean -->
<alias name="defaultWSCacheManager" alias="wsCacheManager"/>
<bean id="defaultWSCacheManager" class="org.springframework.cache.jcache.JCacheCacheManager">
    <property name="cacheManager" ref="wsEhcache"/>
</bean>

And all its relevant caches are defined in

/commercewebservices/resources/commercewebservices/cache/ehcache.xml, e.g. productSearchCache:

<cache alias="productSearchCache" uses-template="default">
    <expiry>
       <ttl unit="seconds">150</ttl>
    </expiry>
    <resources>
       <heap unit="entries">1000</heap>
    </resources>
</cache>

Here take productSearchCache cache as an example to clean, and it can be changed to other cache (i.e. any other caches defined in /commercewebservices/resources/commercewebservices/cache/ehcache.xml) in actual usage.

In the Hybris Administration Console under Console -> Scripting Languages, run the following groovy script on web service node:

import org.springframework.web.context.ContextLoader
import de.hybris.platform.spring.HybrisContextLoaderListener

def getBean(targetContext, targetBean) {
    def f = ContextLoader.getDeclaredField("currentContextPerThread")
    f.setAccessible(true)
    def contexts = f.get(HybrisContextLoaderListener)
    def appContext = contexts.find { targetContext.equals(it.key.getContextName()) }

    if (appContext == null) {
        return
    }

    def bean = appContext.value.getBean(targetBean)
    println bean

    return bean
}

def cacheManager = getBean("occ", "wsCacheManager")
def cache = cacheManager.cacheManager.getCache("productSearchCache")
println "before: " + cache.size()
cache.removeAll()
println "after: " + cache.size()
