Symptom

After uploading a new version of an application to BTP Mobile Services, the Agentry server not able to start. There are no special error message displayed. 

Environment

  • BTP Mobile Services
  • Agentry Cloud Edition (ACE) 

Reproducing the Issue

  1. Deploy a new application version on BTP Mobile Services cockpit.
  2. Attempt to start the Agentry server.
  3. Observe that the server fails to start without any specific error messages.

Resolution

Follow the troubleshooting steps provided in the Troubleshooting: Common Issues | SAP Help Portal. 

See Also

Refer to: Troubleshooting: Common Issues | SAP Help Portal
