Symptom

Supplier requires guidance on ow to set up catalog items with Minimum Ordering Quantities (MOQ)

Environment

SAP Ariba Catalogs

Resolution

In order to set up MOQ's in a catalog, buyer's realm must have the CMS enabled.

Supplier can add the MinimumQuantity field to their catalog and add the MOQ's for the desired items.

See Also

SAP Help Portal: Order Quantities for Customer Catalog Items
