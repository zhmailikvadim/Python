Symptom

  • Unable to add SP0014 for S4HCM 101 to the SPAM/SAINT queue.
  • SPAM/SAINT only generates a queue with SP0012 and SP0013, regardless of the selected target SP.
  • The prerequisite ST-PI 7.40 is missing, while ST-PI 7.58 is installed.

Environment

SAP ABAP system.

Reproducing the Issue

Attempt to define the SPAM/SAINT queue with support package SAPK-10114INS4HCM.

Cause

ACP package for SAPK-10114INS4HCM is missing.

Resolution

  1. Download the Attribute Change Package (ACP) for S4HCM=====101 from the SAP Support Portal.
  2. Load the ACP to the system.
  3. Attempt to include SAPK-10114INS4HCM again.

Keywords
