Symptom

When delete an attachment in app Manage Attachments, error triggered: Attachment ABC is being used for output. Deletion not allowed. 

*ABC represents attachment name

Environment

SAP S/4 HANA Cloud Public Edition

Reproducing the Issue

  1. Access App vf02
  2. Search a not completed billing document and open it.
  3. Click Service for Object, select Manage Attachments.
  4. In App Manage Attachments, select attachment ABC, click delete icon.
  5. A pop-up window shows information 'Are you sure you want to delete file "ABC"?', click Delete, error triggered: Attachment ABC is being used for output. Deletion not allowed.

 

Cause

The attachment is being used for some output of this billing document, hence it is not allowed to delete this attachment.

Resolution

In App vf02, open output request, select any output request which contains attachment ABC, click Display Attachments, untick checkbox Attach and checkbox Merge. Repeat this for all output requests which contain attachment ABC. Go back to billing document and Save.

This KBA also applies to preliminary billing document.

See Also

Help document: Controlling the Output Relevance and Merging of Attachments

Keywords
