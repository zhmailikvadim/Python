Symptom

  • Clarification requested on whether attribute 10319 (ROCNI_ZUCTOVANI_ZADOST) must be filled for activity type 13 in Czech Payroll.
  • In include HCZCJHM0_DM_IMP_RZD, the only reference to the related field appears commented out.

Environment

  • Product: SAP enhancement package for SAP ERP
  • Czech Republic
  • JMHZ

Reproducing the Issue

  1. Review include HCZCJHM0_DM_IMP_RZD and locate references to attribute 10319; observe that the only reference is commented out.
  2. Attempt to determine whether attribute 10319 must be populated for activity type 13 in RZD/MH processing.

Cause

Resolution

  1. For the 2025 annual tax reconciliation (RZD), configure outbound/mapping to send only the following attributes: 10036, 10037, 10320, 10321, 10322, 10323.
  2. Do not populate attribute 10319 (ROCNI_ZUCTOVANI_ZADOST) for activity type 13 for the 2025 processing, as it is not among the required attributes listed above.
  3. If the only reference to attribute 10319 in include HCZCJHM0_DM_IMP_RZD is commented out, keep it inactive for 2025 to align with the required attribute set.
  4. Validate end-to-end processing to ensure only the specified attributes are transmitted and that no errors occur due to unused attributes.

refer to: Czech Financial Administration statement on reporting annual tax reconciliation results
