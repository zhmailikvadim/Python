Symptom

Symptom

You are using Exposure Version 2 and some businessapps are not exposed where in the log for those applications you see the error message "No default target mapping maintained" as well as the warning message "No default visualization defined".

Other Terms

FDM API, replication context

Reason and Prerequisites

For locally replicated applications that have multiple tiles, for one tile the replication context to the LADI could not be determined in the FDM API. This leads to the error in the CDM Content Exposure.

Solution

Implement the attached Correction Instruction (CI) or the relevant Support Package.

Solution

 Install the relevant support package or implement this SAP note directly.
