Symptom

  • Error message: "The entered profile ID does not exist" appears after selecting a particular Inventory (IO) Profile when attempting to create a new application job to run an Inventory Profile through the Application Jobs fiori app.
  • The IO Profile name contains only numbers.
  • The same IO Profile can be run from Excel without any errors.

Environment

SAP Integrated Business Planning for Supply Chain (SCM-IBP)

Reproducing the Issue

  1. Create an Inventory (IO) Profile in the system where the name is all numeric characters (i.e 1234).
  2. Attempt to run the IO profile through the Application Jobs fiori app.
  3. Observe the error message: "The entered profile ID does not exist." after selecting the particular IO Profile.

Cause

  • A validation in the Application Jobs fiori app for Inventory Profiles does not expect the IO Profile Name to be entirely numeric.  
  • Excel does not have this validation, and therefore does not generate any error.

Resolution

  • A workaround exists to either make a copy of the profile, changing the Name of the IO Profile by adding just one alphabetic character, or creating a new name entirely that contains only alphanumeric characters.
  • The IO Profile in question can also continue to be run from Excel as is instead.
