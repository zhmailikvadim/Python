Symptom

SAP Category Management user receives email notifications requesting for approval, but when logging into SAP Category Management, the user's Inbox section is empty

Cause

User is not correctly assigned to the necessary roles

Resolution

  1. Go to your BTP Cockpit
  2. Go to Security > Trust Configuration 
  3. Select the value under Custom Identity Provider for Application and go to User Groups tab
  4. Search for Category Strategy Approver and copy the value under Name (it will be an alphanumerical string)

Now that you have the Name value, you can follow the below steps:

  1. Still on your BTP Cockpit
  2. Go to Security > Role Collections
  3. Search for the role ProcessAutomationEnableGlobalUserIdFallback
  4. Click on Edit and select the plus sign (+)
  5. Under Name paste the alphanumerical string and save
  6. Search for the role ProcessAutomationParticipant and do the same as described on steps 4 and 5
