Symptom

You use the Run Statutory Reports application to create the Russia Financial Statement.

In reporting entity configuration you maintained financial statement versions and cash flow statement versions with different time validity.

You create a new report run for the period. You expect the system to take the value of the financial statement version and cash flow statement version valid on the start of the reporting period, however, the system takes the value valid on the current date.

Other Terms

RU, Russia, SAP Document and Reporting Compliance – statutory reporting, formerly known as Advanced Compliance Reports, Russia Financial Statement, RU_FIN_STMNT, Run Statutory Reports, Run Advanced Compliance Reports, FINANCIALSTMNTHIERARCHY, FINANCIALSTMNTHIERARCHY3, FINANCIALSTMNTHIERARCHY4

Reason and Prerequisites

This SAP Note is relevant only for Russia.

Solution

Import the Support Package specified for your release or implement the attached correction instruction, followed by manual post-implementation steps.

As a general rule SAP recommends that you implement the Support Package specified for your release. However, if you need to install a solution earlier, use the SAP Note Assistant to implement the correction instruction. You can find more information about the SAP Note Assistant in SAP Support Portal, under SAP Note Assistant.
