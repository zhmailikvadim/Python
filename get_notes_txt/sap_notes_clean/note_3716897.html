Symptom

If multiple MeLo's are assigned to the same construct, the function module, which provides the MeterProfileCreate Webservice, can determine the wrong MeLo.

To fix this issue, the logic should be adjusted in the function module /IDEIM/WS_EXIT_METER_PROF_OUT. Since the device change is initiated from the MeLo, this MeLo from the meter details will be used. The existing logic is used as a fallback.

Other Terms

 Process 3060, Process 3070, MeterProfileCreate, /IDEIM/WS_EXIT_METER_PROF_OUT

Reason and Prerequisites

This is a program error.

Solution

 Please implement the correction instruction in the SAP Note or install further Support Package.
