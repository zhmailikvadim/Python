Symptom

Unexpected document types are included in the Philippines CAS Sales/Purchase Journal

Environment

  • SAP S/4HANA 
  • SAP S/4HANA Finance

Reproducing the Issue

  1. Generate the Philippines Purchase Journal (PH_CAS_PJ) report.
  2. Generate the Philippines Sales Journal (PH_CAS_SJ) report.
  3. Observe the inclusion of unexpected document types in the Purchase Journal and Sales Journal.

Cause

Resolution

  1. Confirm that the user has the ability to select the document type prior to running the report.
  2. Ensure that unexpected document types are not selected if they are not expected to appear in the reports.
