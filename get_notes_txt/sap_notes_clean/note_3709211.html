Symptom

This note includes the following corrections in the S/4 Defense Disconnected system:

Issue: Goods Receipt against Outbound Delivery for packing material during Delta Load (DL) was not possible.

Resolution: Enabled Goods Receipt against Outbound Delivery through MIGO where packing material is added in the Outbound Delivery which is not a part of Stock Transfer Order(STO) and serial number details will also be added for the finished goods in the outbound delivery.

Other Terms

Goods Receipt , GR, Packing material, Delta Load, (DL), MIGO, Stock Transfer Order, (STO), S/4 Defense Disconnected.

Solution

  • Implement the corresponding CI as per the note. Also follow the Manual Post-Implementation Steps.
  • Add new entry with field DFPS_KEY value as 'PACK_MAT_ACT' and field DFPS_VALUE as 'X' to activate packing material PGR against Outbound delivery funcationality. If the field DFPS_VALUE is blank then the complete functionality is disabled.
  • A custom configuration table 'DFS_MVT_TYP_CONF' is introduced to control custom movement type substitution. this table is checked for the movement type to get the substitute movement type and the reason for it.
    • Reason for Mvt type: If the entries are maintained in these fields, then only it will be used to map in the material document. Reason for movement for original is associated with Original Mvt. Type and Reason for movement for substitute is associated with substitute Mvt. Type.
