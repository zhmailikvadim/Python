Symptom

As described in SAP ONE Support Launchpad Notification of Legal Change Unified monthly employer report 2026 (JMHZ)

The functionality delivered in this SAP Note is currently under development. It is intended exclusively for customers registered as JMHZ pilot testers, or for the preview of features that may become available for productive use in the future.

These updates are provided:

  • REGZEC (DM - data mining, CS - compose submission, PF - process feedback, SO - special operations, INIT - import of IDs to IT3457)

    • INIT: Report HCZUJHR0_INIT has been extended to support the new XML input file type. XML file will be available to download from authority site ("Služba 2.12") starting from 1.3.2026.

  • MH

    • If you have entered the expected date of birth of the child into infotype 2001, it was previously necessary to run the payroll calculation again. Now it is no longer necessary to run the payroll calculation.

    • Till now the report HCZCJHM0_DM identified situations with full SI and Tax recalculation by existence of wage type M990. Now - additionally - it identifies the situation by existence of wage type assigned to processing class 27, value 1.

    • Enhancement of locks

    • Addition of cancellation support

    • Improvements to _WB and options for manual modifications
