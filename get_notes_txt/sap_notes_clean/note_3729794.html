Symptom

  • EBW_DQ_SS is running and because of various reason it was stoped (cacceld) manually.
  • Next date when mass activity should run it stops with error EH 049 "Mass activity is already running"

Environment

  • SAP S/4HANA
  • SAP ERP

Reproducing the Issue

  1. Run EBW_DQ_SS 
  2. Cancel this run
  3. Try to run a new run of EBW_DQ_SS
  4. Second EBW_DQ_SS is not starting
  5. EH 049 is raised

Cause

  • EBW_DQ_SS has a integrated locking mechanism.
  • This is to secure data integrity

Resolution

  • It is needed to unlock the mass activity as shown in the following picture
