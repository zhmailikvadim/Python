Symptom

There are multiple identical selects triggered on table CRMD_MKTTG_TG_T during the Billing Job run leading to performance issue.

These calls are triggered from the Function Module /1BEA/CRMB_BD_PRC_O_IT_MAPIN.

Other Terms

/1BEA/CRMB_BD_PRC_O_IT_MAPIN, CRMD_MKTTG_TG_T, Performance issue, Spring Billing, Billing Run, Print Job, CRM_MKTTG_TG_GETLIST, initial check

Reason and Prerequisites

An initial check was missing for the importing parameter IS_BDI_WRK-SOLD_TO_GUID before calling the FM 'CRM_MKTTG_TG_GETLIST' which leads to selection of all records in the table CRMD_MKTTG_TG_T.

Solution

Apply the correction instructions contained in the note.
