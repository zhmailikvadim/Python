Symptom

During an InPlace conversion to BCS4HANA, you may run into an issue with generated structures in name range /1SEM/UCX????_??_ITEM which become inconsistent. Typically, you will experience a program termination MESSAGE_TYPE_X in class CL_FINB_GN_BBI during first generation of your BCS data models under BCS4HANA.

Other Terms

UC_XI_ACTION, XI, SOA inbound plug, MDG-F, CL_UC_SEM_BW_UNINSTALL, GET_CREATED_OBJECTS

Reason and Prerequisites

There is a program error. These structures become obsolete in BCS4HANA. The system should automatically remove them.

Solution

Apply the attached source code correction or apply the corresponding support package in your system before starting the software conversion phase. If this has not been done, contact SAP support to remove those structures manually on demand.
