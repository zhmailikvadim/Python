Symptom

 When a pricing area is configured with multiple price sheets, navigating back and forth between the selection screen and the pricing grid causes the application to hang. This issue is accompanied by a JavaScript error: "Uncaught Syntax Error: Unexpected end of JSON input". This note addresses and resolves the issue.

Other Terms

/IRM/GPRPWM, Pricing Area, GS_METADATA ,Navigation issue, JSON INPUT, Uncaught Syntax Error
