Symptom

With SAP Financial Information Management, it is not possible to export as CSV file the rejected lines of the job run.

In Job Run History, clicking on "Export as CSV File" from Rejected Lines tab has no effect.

Reason and Prerequisites

In Job Run History, clicking on "Export as CSV File" from Rejected Lines tab has no effect.

If you open the Development Tools console of the browser, you see the following error : Uncaught TypeError: value.charAt is not a function.

As a workaround, you can still use the "Copy content to clipboard" button.

Solution

Update your SAP Financial Information Management installation to a Support Package, Patch or Release where the issue is fixed.

See the Support Packages & Patches section for details and available patches
