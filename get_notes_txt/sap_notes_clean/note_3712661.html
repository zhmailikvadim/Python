Symptom

In the process "Sending of 55616 from Responsible DSO to SUP" (DE_C_MDC_55616_DSO2SUP_SEND), the payload of the outbound UTILMD 55616 message incorrectly contained the SEQ+Z44 segment in step 1300.

This issue is relevant for the market role: DSO.

Other Terms

 MDC_55616_DSO2SUP, Location Bundle Structure, UTILMD, data provider method, condition 526, segment elements

Reason and Prerequisites

The error occurred because the data provider method for SEQ+Z44 didn't utilize condition [526].

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the system excludes the unexpected segment SEQ+Z44 and subsequent segment elements from the UTILMD 55616 message payload. You can import the BC set /US4G/102_SP08_DSO_NOTE_3712661 to apply the fix. You can verify the fix by checking that in the cloud process document at step 1300, the UTILMD 55616 message is sent successfully without error.
