Symptom

  • An error occurs when attempting to edit the IMG Define Priority Type in SAP S/4HANA Cloud Public Edition.
  • The error message displayed is "/CFG/WEBGUI_MSG025: Cust. Object V_T356A is not mapped in WEBGUI registry to the IMG activity SIMG_CFMENUOLIAOIO1."
  • The view/table switches to display-only mode.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

1.Attempt to edit the IMG Define Priority Types.
2.Observe the error message "/CFG/WEBGUI_MSG025: Cust. Object V_T356A is not mapped in WEBGUI registry to the IMG activity SIMG_CFMENUOLIAOIO1."
3.The view/table switches to display-only mode.

Cause

Configuration 'Define Priority Types' is not customizable i.e. cannot define new priority types in SAP S/4HANA Cloud Public Edition.

Resolution

All available configuration activities for Asset Management are listed in help documentation - Configuration Activities. Use 'Define Priorities for Each Priority Type'(SSCUI:102583) to customize priorities for pre-delivered priority types.
