Symptom

While Clicking on Status Button for Romania eInvoice system throws following error:

"Target structure of AIF interface /EDORO STATUSREQ 1 could not be determined - Message No. EDOCUMENT720"



Image/data in this KBA is from SAP internal systems, sample data, or demo systems and are for reference purposes only. 

Environment

  • SAP ERP
  • SAP S/4HANA
  • SAP Document and Reporting Compliance service
  • SAP Document and Reporting Compliance, cloud edition

Reproducing the Issue

1. Launch EDOC_COCKPIT
2. Enter Source Key
3. Click on Status Button

Cause

Missing Interfaces Name STATUSREQ & STATUSRES for Version 1 in /AIF/CUST->Interface Development->Define Interfaces.

Resolution

To Fix the issue, Follow below steps:

1.For S/4HANA Release:  
   Reactivate the BC set ZAIF_S4_EDOC_RO_V01 in the SCPR3 transaction. This will deliver the missing interfaces STATUSREQ and STATUSRES for Version 1, which are provided through 3233530 - eDocument Romania: eInvoice - Mapping changes.  

2.For SAP_APPL Release: 
   Reactivate the BC set ZAIF_APPL_EDOC_RO_V01 in the SCPR3 transaction. This will deliver the missing interfaces STATUSREQ and STATUSRES for Version 1, which are provided through 3233530 - eDocument Romania: eInvoice - Mapping changes
