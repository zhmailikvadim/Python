Symptom

It is not clear on how to how to delete custom layout in APP 'Process Inspection Lots -Worklist(' App ID QA32). 

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Environment

  • SAP S/4HANA, on-premise
  • SAP S/4HANA Cloud Private Edition

Resolution

  1. In APP 'Process Inspection Lots -Worklist', go to 'Menu'>'Goto'>'Variants'>'Delete'
  2. Search the expected custom layout, and click on choose.
  3. Click on 'Yes' in the deletion confirmation pop up. 

Keywords

delete custom layout, Process Inspection Lots -Worklist, App ID QA32
