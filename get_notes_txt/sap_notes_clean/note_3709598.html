Symptom

When attempting to maintain data with transaction OPS6 or with transactions S_ALR_87003580, S_ALR_87005375 or S_ALR_87005271, in a client with another client role as Production. Information appears: Client xxx has status 'not modifiable'. Data are not modifiable.

Environment

This KBA applies to all SAP Products based on NetWeaver and S4H.

Reproducing the Issue

  1. Execute transaction OPS6
  2. Information appears: Client xxx has status 'not modifiable'.

or

  1. Execute one of these transactions:
    S_ALR_87003580
    S_ALR_87005375
    S_ALR_87005271
  2. Information appears: Client xxx has status 'not modifiable'.

or 

  1. Execute transaction SPRO
  2. Press button SAP Reference IMG
  3. Execute one of these IMG activities
    Define Person Responsible
    Specify Persons Responsible for WBS Elements
    Specify People Responsible for Standard WBS Elements
  4. Information appears: Client xxx has status 'not modifiable'.

Resolution

Follow manual instruction of KBA 3138512 - SM30 | SPRO | Client XXX has status 'not modifiable' | Client role Customizing or Test
The object names is V_TCJ04

Keywords

Specify Persons Responsible for WBS Elements, Specify People Responsible for Standard WBS Elements, Person Responsible for Project/ Capital Investment Program,
