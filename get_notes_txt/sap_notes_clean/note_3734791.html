Symptom

In SAP S/4HANA Cloud Public Edition, when trying to reverse a journal entry with journal entry type RR, error message "Cancel secondary journal entry &1 via its reference document entry &2" is shown.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

Resolution

For EBRR real time revenue recognition document which has journal entry type RR, it can not be reversed and it does not need to be reversed. It can only be revaluated.

Either revalue associated document in EBRR monitor (Event-Based Revenue Recognition – Projects (Fiori ID F4767)) or start Run Revenue Recognition job. The revenue recognition will be recalculated to reflect current state.
