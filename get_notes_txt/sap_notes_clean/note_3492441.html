Symptom

Gender field is not successfully passing from Onboarding to Manage Pending Hires. Field is coming empty, although the same is mapped in Onboarding.

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Environment

SAP SuccessFactors Onboarding 1.0

Cause

The gender mapping is a Hardcoded mapping, in order to have this working you would need either to use full standard mapping or full custom mapping.

Resolution

  1. Check in Manage Business Configuration > personal information > look for field "gender", we can see the standard Gender field is enabled:



  2. Now, go to 'Field Mapping tool for integration with Employee Central' > we can see the affected field as a custom one:



  • The way that you have your system now, is custom on Onboarding side, and standard on EC Side, that will not work.
  • You need either to change the Onboarding side to standard, using the Gender Field, or create a custom picklist field on EC side.

If you wish to use a custom field in Employee Central, we recommend you to following: 2891069 - How to add a picklist to a field under Manage Business Configuration. On this KBA we have steps to create a custom field and relate it to a Picklist.

See Also

2891069 - How to add a picklist to a field under Manage Business Configuration
