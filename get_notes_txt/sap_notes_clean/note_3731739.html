Symptom

Using the CMP to update for example MARC-MMSTA. However, while processing the request observe that the fields MARD-PSTAT and MARC-STRA1 are also getting updated. These two fields are not within scope.

Notice that for some other materials, all plant-related MARD status fields are being updated as well.

Environment

SAP Master Data Governance - Material Management

Reproducing the Issue

  1. Upload file in the Mass Process and complete the process.
  2. Open the material updates in Change Documents
  3. Observe the changes for MARD-PSTAT too.

Cause

Expected behaviour

Resolution

With MDG Consolidation & Mass Processing, the basic principle of the function is about the current state of object.

It means that even you just involve specific and limited fields change in the mass processing process of an object, with the mentioned principle, the system will load the current state of the whole object and involve it in the process.

This clarifies why other fields changed even though it is not included in the load file or scope.
