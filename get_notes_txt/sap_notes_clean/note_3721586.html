Symptom

A order is generated from a maintenance plan but not all operations have been transferred from the task list

Environment

  • SAP ERP Plant Maintenance (PM)
  • SAP ERP
  • SAP S/4HANA, on-premise
  • SAP S/4HANA, Cloud Private Edition

Reproducing the Issue

  1. Run transaction IP10
  2. Release a scheduled call
  3. Check the operation list on the new order

Cause

Some operations in the task list are not valid on the plan date. These are not transferred to the order

Resolution

Ensure the required operations have validity period to include any possible plan date. Check the validity period as follows:
From task list (IA07) select an operation and choose menu 'Operation - Administrative data'

Keywords

IP30, IP30H, IP10, RISTRA20, RISTRA20H, Key date, Valid from
