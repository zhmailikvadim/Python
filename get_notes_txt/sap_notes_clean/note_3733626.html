Symptom

Query processing in /STTP/Cockpit > Serialized Trade Items takes an unusually long time post ATTP migration from ATTP 300 FP00 to ATTP 2310 FP01.

Environment

  • SAP Advanced Track and Trace for Pharmaceuticals
  • SAP S/4HANA

Reproducing the Issue

  1. Navigate to /STTP/Cockpit.
  2. Open Serialized Trade Items and search; observe that the response time is very high.

Resolution

  1. Implement the following SAP Note - 3567340 - Serialized Items Query performance improvement - SAP for Me
  2. After implementation, retest the Serialized Trade Items query in /STTP/Cockpit to confirm improved performance.
