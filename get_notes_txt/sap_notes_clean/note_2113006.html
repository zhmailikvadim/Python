Symptom

Transaction: O3U_VCR6

System allows a VCR to be deleted though an open VL Doc exists for that VCR.

Other Terms

General keywords: add-on, IS-Oil, industry solution, IS-OIL-PRA, Upstream Oil and Gas, Production and Revenue Accounting

Problem-specific key words: IS-OIL-PRA-REV-VAL,  valuation cross reference, VCR delete, open VL documents

Reason and Prerequisites

Program error.

Solution

If you need to implement this SAP Note urgently, refer to SAP Note 47531.
