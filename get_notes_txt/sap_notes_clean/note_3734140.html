Symptom

  • In the Latest People Profile, the Work Order portlet displays the pencil/edit icon to users who only have view permissions.
  • Clicking the pencil/edit icon shows the error message: "No data available or you do not have the necessary permissions".
  • The same error appears when selecting View All on the Work Order portlet and then clicking the Edit button.
  • Users with only “View Current” permission for Work Order also see the edit controls and encounter the error.
  • The behavior was not observed in the Legacy People Profile (All Actions behaves as expected).

Environment

SAP SuccessFactors Employee Central

Reproducing the Issue

  1. Log in as a user who has view-only permissions (for example, View Current or View Current and History/correct) for the Work Order object.
  2. Open a worker’s profile using the Latest People Profile.
  3. Navigate to the Work Order portlet.
  4. In the Latest People Profile, the Work Order portlet displays the pencil/edit icon to users who only have view permissions.
  5. When we Click the pencil/edit icon and observe the error: “No data available or you do not have the necessary permissions.”
  6. Additionally, we even noticed that users that only have "View Current" permissions for the Work Order object also see the Pencil-icon on the portlet and the Edit-button in the details page of the Work Order portlet.
