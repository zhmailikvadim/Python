Symptom

The SAP Note addresses the issue with incorrect setting the last version flag via using the RMPE_MIGRATE_BOM program. This happend during Migration process of the classic Date-Effective BOM with more than one change state. System does not set the last version flag (versnlastind) in STKO table.

Other Terms

Migration of Classic BOMs to Version-Controlled Unitized and Non-Unitized BOMs; RMPE_MIGRATE_BOM

Reason and Prerequisites

Program error

Should be used classic Date-Effective BOM with more than one change state.

Solution

Apply the attached correction instructions
