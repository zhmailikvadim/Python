Symptom

  • Unable to add a BOM item with material type FERT to an engineering/design BOM (usage 2) whose header material type is HALB.
  • Error message: "material type FERT not supported for BOM item".

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Open or create an engineering/design BOM (usage 2) for a header material with material type HALB.
  2. Attempt to add a component with material type FERT as a BOM item.
  3. Observe the error: "material type FERT not supported for BOM item".

Cause

In configuration activity 101163 “Allowed Material Types for BOM Items”, for header material type HALB only item material types HALB, ROH, and SWNV are allowed.

Resolution

It is system behavior.

See Also

Refer to: KBA 3072643.
