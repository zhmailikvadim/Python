Symptom

You are using the BDO app for maintaining distribution models. One of the following errors occurs:

  • You activate a push model. A pull model for the related object type does not exist. Thus, an initial load (synchronize with MDI) should be executed automatically. However, the initial load does not take place.
  • You trigger a data distribution from S/4 to MDI or vice versa. However, the data distribution is not executed.  

Other Terms

Business Data Orchestration
Master Data Orchestration
Initial Replication
Initial Load

Reason and Prerequisites

Code error

Solution

Implement the attached correction instruction or apply the most up to date support package
