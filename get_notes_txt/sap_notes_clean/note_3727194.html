Symptom

  • Unified Data Model is enabled in your instance
  • The JobLocation object is populated with values
  • Some of these values may no longer be in use, however there is no status to make them inactive

Environment

  • SAP SuccessFactors Recruiting Management
  • SAP SuccessFactors Recruiting Marketing

Reproducing the Issue

  1. Log in to the system.
  2. Navigate to Manage Data > Job Location.
  3. Select any JobLocation and Take Action
  4. Note there is no option to make the value inactive

Cause

The status field for the Job Location object is not visible by default

Resolution

  1. Navigate to Configure Object Definition for the Job Location object.
  2. Locate the field "mdfSystemStatus".
  3. Update the visibility of the "mdfSystemStatus" field to "Editable".
  4. Perform an OData Metadata Refresh.
  5. Once the refresh is completed, the status for Job Location objects can be updated to inactive.
