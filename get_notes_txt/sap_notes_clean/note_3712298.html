Symptom

  • Is SAP ASE officially supported in a Kubernetes-based environment ?

Environment

  • SAP Adaptive Server Enterprise (ASE)
  • Kubernetes

Cause

  • The SAP (Sybase) range of products is certified only on OS level and supported OS versions are listed here:
    • 3576582 - SAP ASE 16.1 Supported Operating Systems and Versions
    • 2987324 - SAP ASE 16.0 SP04 Supported Operating Systems and Versions
    • 2489781 - SAP ASE 16.0 SP03 Supported Operating Systems and Versions

Resolution

  • SAP (Sybase) Engineering does not test using Kubernetes
  • There are no features aimed at the use of ASE with Kubernetes
  • Support does not cover any specifics related to running of ASE in a Kubernetes environment
    • Support only extends to the general interoperability with a supported OS and when relevant to the running of the SAP product
