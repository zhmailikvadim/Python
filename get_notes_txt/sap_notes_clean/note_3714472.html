Symptom

Storage location fields are empty when executing the SAP query. The solution described in the KBA 1918331 is implemented, but the storage location text is still empty.

Cause

The fields required to determine the text is not filled.

Resolution

Execute the report RS_TEST_IDENTIFY_TEXT and click on the value in the 'Key (Key fields for text table)' column. The fields required to determine the text will be displayed. These fields should be filled.

See Also

KBA 1918331 - Text fields are blank in the SAP Query result
