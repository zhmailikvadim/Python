Symptom

In  LBBP_PD_SCF63, PERFORM or CALL FUNCTION "SPOOL_RQ_DELETE", the actual parameter " IV_BE_OBJECT_ID" is incompatible with the formal parameter "REQ_NUMBER".  Runtime error  CX_SY_DYN_CALL_ILLEGAL_TYPE is observed.

Other Terms

LBBP_PD_SCF63, IV_BE_OBJECT_ID,  po_number,  SPOOL_PO_DELETE.

Reason and Prerequisites

Code error.

Solution

Implement correction instruction or upgrade to relevant SP.
