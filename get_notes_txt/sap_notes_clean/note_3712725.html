Symptom

You use the Security Audit Log with file-based logging.
However, from earlier configurations, there are still log entries in the database that you now want to delete after the retention period has expired.
The option to delete table contents in transaction RSAU_ADMIN exists, but has no effect.

Other Terms

CL_SAL_ADMIN, RUN_DB_REORG, RSAU_FILE_ADMIN

Reason and Prerequisites

 Program error

Solution

Implement the corrections contained in this SAP Note using SNOTE or by importing the referenced support package.

Solution Explanation

During the deletion of log data, the table to be handled is not uniquely defined in the configuration "Recording in File System".
This SAP Note specifies that during this configuration, the system first deletes in the table RSAU_BUF_DATA. The table RSAU_LOG is taken into account only when the table RSAU_BUF_DATA no longer contains any data records.
