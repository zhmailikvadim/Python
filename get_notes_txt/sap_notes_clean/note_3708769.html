Symptom

This SAP note delivers the HR_IE_AE Standard Text object (transaction SO10), which contains the email text's HTML formatted template version of the Automatic Enrollment notification letter. The notification letter is attached to this email.

Other Terms

HR-IE, Automatic Enrolment, email, HR_IE_AE, HIECAEN0

Solution

To apply the text object in your system, apply the manual corrections located in the Correction Instructions section of this SAP Note. Alternatively, apply the HR Support Package(s) indicated for your release under Reference to Support Packages.
