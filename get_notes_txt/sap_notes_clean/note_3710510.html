Symptom

As you might have noticed. Currently, the system allows multiple password reset requests to be triggered within a short period of time, which may result in several new password emails being sent if the user repeatedly requests a password reset.

Other Terms

E-Recruiting, Forgot Password, PA-ER, New Password Requested

Reason and Prerequisites

This behavior occurs because there is no restriction on the number of reset requests that can be made within a specified timeframe. Implementing rate limiting would help restrict the frequency of password reset requests and prevent multiple password emails from being generated in a short duration.

Solution
