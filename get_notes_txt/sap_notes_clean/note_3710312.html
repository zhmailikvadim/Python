Symptom

During the Initial Load Extraction there is error message:

Delta load is not necessary in an IDR scenario
Message no. FIN_CFIN_MIGRATION026

Environment

  • Financial Accounting (FI)
  • Central Finance (CF)
  • SAP S/4HANA
  • SAP S/4HANA Finance
  • SAP ERP

Reproducing the Issue

Execute the Initial Load Extraction in the Target System.

Cause

The Initial Load Delta is part of the Classic Initial Load, and it is necessary there, however when using the IDR Initial Load it is not necessary to execute.

Resolution

During the IDR Initial Load there is no need for the delta execution. This is explained in documentation "IDR Initial Load – Extraction of Balances and Open Items":

"There is no delta load. This means that the periods before the document load period must be blocked while the extraction is running. You block and unblock previous periods in the step Manage Intermediate Data Retention."
