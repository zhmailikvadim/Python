Symptom

While creating settlement from "Settlement Workcenter" application or creating contract using "Manage ACM Contracts" application encountered an error "Destination not found".

Other Terms

Commodity Expense Management, CL_HTTP_CLIENT, CL_BGRFC_DESTINATION_INBOUND, Destination not found, LOG_CMM_EM, LOG_CMM_EM_BF, LOG_CMM_EM_OIL_BF, Manage ACM Contracts, Create Settlement.

Reason and Prerequisites

This error happens when the business functions 'LOG_CMM_EM_BF' and/or 'LOG_CMM_EM_OIL_BF' has activation status as switched "On", but the required system configurations are not setup in the system.

Solution

All HTTP Client and BGRFC class exceptions are handled.
