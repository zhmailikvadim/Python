Symptom

In transaction code GPA_ADMIN, you choose a guided procedure to execute and then in the Complete Step, you click 'Send by Email' or 'Export'->'Export to HTML' to generate a HTML report. After that, you observe a Custom Activity with initial status is displayed in the end of the HTML report.

Other Terms

Solution Manager, SolMan, GP, PDF report, Comment

Reason and Prerequisites

Program error.

Solution

Implement the attached Correction Instructions or install corresponding Support Package. After that, in the generated HTML report you will not see the initial Custom Activity.
