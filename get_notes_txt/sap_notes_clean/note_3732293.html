Symptom

When publishing an event, the system shows an error prompting to select a default exchange rate set (system-generated or manual) and does not auto-default a currency conversion rate set.

Cause

When the multi-currency rule Allow participants to select bidding currency is set to Yes, the exchange rate set dropdown becomes mandatory and is not auto-defaulted, this is working as designed.

Resolution

Follow below steps when the error prompt appears:

  1. Go to the event rules.
  2. In Select exchange rate set, choose a currency conversion rates set (for example, System Currency Conversion Rates).
  3. Proceed to publish the event after the selection is made.

Note: There is no parameter, template, or configuration available to bypass or default this selection when the rule Allow participants to select bidding currency is enabled.
