Symptom

You want to delete the generated programs of already deleted queries or InfoSets in your system. However, there is no option for doing this in the standard.

Solution

Use the Note Assistant to implement the correction instructions, or import the relevant Support Package.

With this correction, the report RSAQDEL0 now also cleans up programs in the standard area of the current client if the option 'Clean Up Obsolete Programs' is used.
For technical reasons, it is unfortunately not possible to clean up programs that belong to query objects of the standard area of another client.
