Symptom

This SAP Note generates required DDIC elements and/or other supplementary repository objects in your system for implementing SAP Note 3705745.

Version 1: General availability

Other Terms

SAP Note 3705745, RPUUDOH0_MANDT_3706049

Reason and Prerequisites

DDIC elements and/or supplementary repository objects of SAP Note 3705745

Solution

Implementation:

Implement the Correction Instructions of this SAP Note.

Log on to the requested clients in HU language and run the newly created report in Update mode (in all requested clients): RPUUDOH0_MANDT_3706049
(Depending on the SAP_BASIS release and SP level, SNOTE may handle the objects which the report tries to update. If so, running the report will display such objects as "Covered by SNOTE" and skip the object. In this case, SNOTE updates the relevant object.)

Make sure that the created objects are all active.
