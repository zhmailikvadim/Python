Symptom

Once the purchase order is approved or rejected user needs to get email notification with PDF file.

Environment

Materials Management (MM)
SAP ERP Central Component
SAP ERP
SAP enhancement package for SAP ERP
SAP enhancement package for SAP ERP, version for SAP HANA
SAP S/4HANA

Reproducing the Issue

1.Create PO 
2.Workflow will be triggered, and the approver will either approve or reject the PO.
3.User should get email Notification with PDF file 

Cause

Standard design.

Resolution

When the purchase order is approved or rejected users will receive email notifications after approval/rejection of the PO.  But as per standard system design SAP, it is not possible to add PDF file with email notification.

Keywords

PDF file, email notification, PO, approval or rejection of the PO.
