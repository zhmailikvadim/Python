Symptom

  • Attempting to perform mass edit in the Find Maintenance Items app to update fields (for example, main work center or priority) results in an error when starting the application job.
  • Error message: "User is not authorized to schedule job."

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Open the Find Maintenance Items app.
  2. Select multiple maintenance items.
  3. Choose Mass Edit and change a field such as priority or main work center.
  4. Start the job.
  5. Observe that the error “User is not authorized to schedule job.” is raised.

Cause

The Find Maintenance Items app is deprecated since CE2602, using the mass edit function can lead to authorization check failures when scheduling the corresponding application job.

Resolution

  • Use the successor app Manage Maintenance Items instead of the deprecated Find Maintenance Items app.
  • In app Manage Maintenance Items, use the Mass Change functionality to perform the required updates.
