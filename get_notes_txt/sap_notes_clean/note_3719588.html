Symptom

This note provides a fix for the inconsistency in the retreival of Restricted Savings subscription details between program FSTB_RSM_DISP_ANNUAL_SUBS and BAPI_FSTB_RSM_SUBS_DETAIL, where for Business Partners without ISA records the report displays an informational message while the BAPI returns an error message. The correction ensures the BAPI also returns an informational message to maintain consistent system behavior.

Other Terms

Restricted Savings Monitoring, RSM,Business Partner, BP

Reason and Prerequisites

 Program error

Solution

 Import the relevant support package for your release. Use the transaction SNOTE to implement the attached correction instructions in advance.
