Symptom

Sync job requisition data from SAP SuccessFactors Recruiting to SmartRecruiters, so that Winston Chat can use it.

Environment

SmartRecruiters Winston for SAP SuccessFactors

Resolution

Prerequisites

  • Your organization has a license for SmartRecruiters Winston for SAP SuccessFactors and it's enabled in Provisioning.

  • You have the following permission: Administrator Permissions > Manage Recruiting > Set Up Integration with SmartRecruiters Winston Chat

Procedure

  1. In SAP SuccessFactors HCM, go to Admin Center  Set Up Integration with SmartRecruiters Winston Chat.
  2. On the Field Mapping tab, for each SmartRecruiters field listed, select a corresponding SuccessFactors field to map it to.

    You need to complete a mapping for each field that's marked as required.

    The following SmartRecruiters fields are available for mapping:

    • Job Title

    • Job Description

    • Experience Level

    • Function

    • Industry

    • Type of Employment

    • Country Code

    • City

  3. Choose Next to continue.
    Your mapping is validated before you can proceed to the next step. If any errors are shown, correct them and try again.
  4. On the Value Mapping tab, for each field listed, select 1 or more corresponding allowable values in SuccessFactors.
    The following fields are available for mapping:

    • Function

    • Industry

    • Experience Level

    • Employment Type

  5. Choose Next to continue.
    Your mapping is again validated before you can proceed to the next step. If any errors are shown, correct them and try again.
  6. On the Data Sync tab, choose Initiate Data Sync.
    The sync status In Progress is displayed.
  7. The data sync process may take some time. Choose Refresh Status to check progress, or leave the page and check back later.
  8. When the sync process completes, confirm that all job data was synced successfully. You can see the following information.

    OptionDescription

    Sync Status

    Status only indicates if the process is running. It doesn't indicate if there were any errors during the last sync.

    • In Progress means the process is running.

    • Completed means the process was completed, with or without errors.

    • Failed means that the process was stopped unexpectedly, before it could complete. Try running it again.

    Total Synced

    Shows the number of job requisitions that were synced successfully.

    Total Errors

    Shows the number of job requisitions that were synced with errors. Correct the errors and try running it again.

    Last Sync

    Shows the date and time of the last sync.

9. If the Total Errors count is more than zero, choose Download Error Logs to download a full report with details.

10. Use error logs to identify and correct errors, as needed. They try running the data sync again.

Results

When you've run a data sync without any errors, job data is now synced successfully. The data in Recruiting job requisitions is now synced with the corresponding fields in SmartRecruiters. Winston Chat can now use that job data to interact with candidates on your career site.

Going forward, the data sync is maintained automatically. Whenever you create or edit job requisition data in a synced field, it's updated in SmartRecruiters automatically.

Next Steps

You can now enable Winston Chat on your career site.
