Symptom

Encountered an error retrieving database details from the System Landscape Directory while accessing the company database.

Reproducing the issue

Prerequisites:

Two SQL Server instances are registered in the System Landscape Directory (SLD), each containing a database with identical names.

Scenario:

  1. Restore the company database with the same schema name on both SQL Server instances.
  2. Log on to the SAP Business One Client.

Expected behavior:

Successful login to the SAP Business One client.

Actual behavior:

The following error occurs in the SAP Business One client:

"Failed to get database information from System Landscape Directory"

The login process for the web client is also failing with an error message:

Error while doing SAML SSO, check SLD log for detail; please open a new browser window, try to access the target SP resource once again.

The sldService.log contains the error notification:

"jakarta.persistence.NonUniqueResultException: More than one result was returned from Query.getSingleResult()"

Solution

SAP intends to provide a feature package or patch to solve the symptom that is described in this SAP Note.
