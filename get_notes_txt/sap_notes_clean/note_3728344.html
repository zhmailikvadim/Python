Symptom

This SAP Note introduces the updated version of the VAT Return (Version 5.12), in compliance with Government Regulation Order № ЕД-7-3/1227@ from February 02, 2026, and effective from Q1 2026. The updates apply to Sections 10-12 (Invoice Journal).

In this update, the following new elements have been introduced:
- New hierarchies: RU105, RU115, RU125

Other Terms

Incoming Invoice Journal, Outgoing Invoice Journal, J_3RF_REGINV, NO_NDS.10_1_003_05_05_12_01, NO_NDS.11_1_003_06_05_12_01, NO_NDS.12_1_003_07_05_12_01

Reason and Prerequisites

Legal Requirement according to to Government Regulation Order № ЕД-7-3/1227@ from February 02, 2026, valid from Q1 2026.

This SAP Note provides you with the relevant system changes to address the following legal changes:

  • Changes: VAT Return and electronic formats 2026

Solution

Apply the appropriate Support Package or install the attached correction instructions with manual pre- and post-implementation steps.

After installing this SAP Note, perform the following steps:

  1. SM30, view J_3RVATDSECHIER, Change. Add records for company code as follows:
    Section of VAT Return    Valid From      Hierarchy Code    
     10 01.01.2026 RU105
     11 01.01.2026 RU115
     12 01.01.2026 RU125
