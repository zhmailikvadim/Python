Symptom

There is a requirement to execute the available batch (.bat) or shell (.sh) script tools delivered with the SAP Cloud Connector installation.

Environment

  • SAP Cloud Connector

  • Microsoft Windows (for .bat scripts)

  • Linux/Unix-based operating systems (for .sh scripts)

  • Supported JDK version used by SAP Cloud Connector

Resolution

To use the script tools provided in the SAP Cloud Connector installation directory, follow the steps below:

1. Open Command Line with Required Privileges

On Windows: Open a Command Prompt with Administrator privileges or with sufficient read and write permissions to the SAP Cloud Connector installation directory.

On Linux/Unix: Open a terminal session with sufficient permissions to access and modify files in the installation directory.

2.  Verify JAVA_HOME Environment Variable

Ensure that the JAVA_HOME environment variable is set to the installation directory of the JDK currently used by SAP Cloud Connector.

To verify:

On Windows: echo %JAVA_HOME%

On Linux/Unix: echo $JAVA_HOME

If not set, configure JAVA_HOME accordingly before proceeding.

Note: When verifying the JAVA_HOME environment variable, system behavior differs between Windows and Linux/Unix platforms. On Windows, if JAVA_HOME is not set, running echo %JAVA_HOME% will return %JAVA_HOME%, indicating that the variable is undefined. On Linux/Unix systems, if JAVA_HOME is not set, executing echo $JAVA_HOME will return an empty line or no output. In both cases, this confirms that the environment variable is not configured.

For the script tools delivered with SAP Cloud Connector to function correctly, JAVA_HOME must be set to the root directory of a supported JDK installation (not the `bin` directory), and a new terminal session must be opened after setting the variable to ensure the changes take effect.

3. Navigate to the Installation Directory

Switch to the SAP Cloud Connector installation directory.

On Windows: cd <CloudConnector_Installation_Directory>\SAP\scc20

On Linux/Unix: cd <CloudConnector_Installation_Directory>/SAP/scc20

4. Execute the Required Script Tool

Run the respective tool depending on the operating system:

On Windows (.bat files): <toolname>.bat

On Linux/Unix (.sh files): ./<toolname>.sh

5. Provide Required Parameters (If Applicable)

If the script requires additional input parameters, the usage syntax will be displayed in the console output.

Follow the syntax instructions shown and rerun the command including the required parameters.

Additional Information

  • Always ensure that SAP Cloud Connector and its underlying JDK are running on supported versions to avoid compatibility issues.
  • Script execution may fail if insufficient permissions are granted or if JAVA_HOME is incorrectly configured.
  • For supported JDK versions and compatibility details, refer to the official SAP documentation for SAP Cloud Connector.
