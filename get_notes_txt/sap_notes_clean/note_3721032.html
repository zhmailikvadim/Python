Symptom

When you run the Tax Certificate Report & Media Form (HTWCTXM0) report for an employee with a large annual gross amount, the system displays a runtime error with the message "Overflow during an arithmetic operation (type p) in the program."

The relevant data type used for the gross value is INT4. If the value exceeds the upper limit of this data type, the system generates a runtime error.

To handle large gross amounts, this SAP Note changes the data type and extends the upper limit.

Other Terms

HTWCTXM0, Gross, Dump, Large Value

Reason and Prerequisites

This is a program error.

Solution

Do either of the following:

  • Apply the relevant Support Package.
  • Carry out the attached ManualSteps 3721032.pdf and then implement the corresponding correction instructions.
