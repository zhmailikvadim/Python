Symptom

  • An error occurs when you set a password of less than or equal to 7 characters when you install ASE.

    Configure New SAP ASE
    ---------------------

    SAP ASE Name (Default: ASESERVER): ASESERVER
    System Administrator's Password: <Enter a password not exceeding 7 characters>
    Confirm System Administrator's Password:
    The 'System Administrator's Password' must have at least 8 characters.

  • Every place you set a password during installation requires at least 8 characters.

Environment

  • SAP Adaptive Server Enterprise (ASE) 16.1 SP00 PL01 or higher

Cause

To comply with security standards, the default for 'minimum password length' is changed to 8.

Resolution

All passwords must be at least 8 characters long.


If you want to set a password less than 8 characters, reconfigure it after installation.

To change the System Administrator's password:

1> sp_configure 'minimum password length',6
2> go

1> alter login sa
2> with password '<old_password>' modify password '<new_password>'
3> go
