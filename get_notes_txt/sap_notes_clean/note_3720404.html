Symptom

  • The text box element in the search interface is not displayed.
  • The text box disappears after being clicked.
  • The issue occurs only for specific user accounts.

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Environment

  • SAP GUI for HTML
  • SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Open the APP: Display materials;
  2. Navigate to the interface where the search functionality is located;
  3. Observe that the text box element is not displayed;
  4. Click on the initially displayed text box and observe that it disappears.

Cause

The behavior occurs because the user activated the GUI Debugging option in SAP GUI for HTML setting, while GUI debugging is a development and support feature and should never be used in normal operations.

Resolution

Disable GUI Debugging option or reset it to the default state with following steps:

"Menu" button > Navigate to "Tools" > Select "Debugging" > Disable "GUI debugging" setting (button color is shown as gray)
