Symptom

When navigating to the parameter screen of selection step in display code with transaction CK40N, a pop-up is shown with message DB613 "Variant XXXX of report SAPRCK60 does not exist"

Environment

  • Controlling (CO)
  • SAP R/3
  • SAP R/3 Enterprise 4.7
  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA Finance
  • SAP S/4HANA

Reproducing the Issue

  1. Access transaction CK40N to create  a new costing run.
  2. Goto display mode and click parameter button for selection step

Cause

This is correct system behavior. Without clicking the parameter button in Edit mode, the system is not able to create the default variants for the step.

Resolution

Please goto Edit mode and Click the parameters button for the selection step. By this, the system will be able to create the variant and the pop-up will disappear.
