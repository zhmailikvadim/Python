Symptom

In SAP S/4HANA 2021, when you process staging for production materials, posting production consumption, or posting consumption reversal, the only supported feature is staging into production supply area (PSA) or consumption/consumption reversal posting from the PSA that's determined in the relevant production material request (PMR) document item that is used by the MES-driven staging process.

Now, you can apply this note to use an additional feature for MES-driven staging, that is, the dynamic staging feature. This enhancement can support the staging, consumption, consumption reversal postings of the requested stock for the PSA that differs from the determined PMR.

 These enhanced objects include DDIC objects, data elements, structures, table types and ABAP classes, function modules etc.

To find the help documents of this feature, see Dynamic Staging and MES-Driven Staging.

Other Terms

MES-Driven Staging, Dynamic Staging, Staging, Consumption, Reverse Consumption, Advanced Production Integration, Production Material Request.

Reason and Prerequisites

Dynamic staging is only available in SAP S/4HANA 2022. But you want to use it in SAP S/4HANA 2021.

Solution

 Implement this note.
