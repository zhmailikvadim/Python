Symptom

When using the Fiori app F6579 – Manage Work Permits, the operation count is not automatically refreshed after adding operation(s) to a selected order.
The count field does not update to reflect the newly added operations.

Other Terms

 F6579 – Manage Work Permits, Add operation, this.oModel.setChangeGroups

Reason and Prerequisites

This error is a program error.

Solution

Implement automatic correction provided in this SAP Note.

After you implement the corrections, please refresh cache by following SAP Note 2319491 - How to clean up the cache after applying changes that affect SAP Fiori apps
