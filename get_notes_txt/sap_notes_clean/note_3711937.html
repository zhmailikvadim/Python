Symptom

You start the foreign currency valuation as a reset run ("Reset Valuation" button clicked) for the balance valuation.

In the list of postings/posting proposals, fields that are filled in the valuation posting to be reset (for example, the profit center) are initial.

This is solely a display error; the reset postings are performed correctly.

This note is addition to note 3241861

Other Terms

Valuation, FAGL_FCV

Reason and Prerequisites

Program error

Solution

Implement the correction instructions.
