Symptom

When logging on in Finnish, the sales document type OR (standard order), the Finnish translation is VT, is missing from the Sales Document Type value help.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Log on in English and open VA01.
  2. Enter a valid sales organization, distribution channel, and division.
  3. Open the Sales Document Type value help and observe that 7the sales document type OR is available.
  4. Change the system language to Finnish.
  5. Open VA01 and enter the same organizational data.
  6. Open the Sales Document Type value help and observe that VT (Finnish translation for OR) is missing.

Cause

The root cause of the issue is the Language-dependent Conversion of Sales Document Types in VA01. The Language-dependent conversion for the document type AG - Quotation in Finish is TA. TA is however the code for a standard order in the system. Due to conversion the standard order code in Finish ‘VT’ is converted to the original code ‘TA’ and then mistakenly converted again from ‘TA’ to ‘AG’.

Resolution

This is a known issue in VA01. It is currently planned to be fixed in 2608 main release of SAP S/4HANA Cloud Public Edition. Please note that SAP may decide to alter this timeframe without warning.
