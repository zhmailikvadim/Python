Symptom

In the profile data for the compatible unit design, the system does not display a short text for the plant condition and the maintenance activity type.

Other Terms

CUM, /CUM/CUDESIGN, /CUM/CUORDER_N, USER_COMMAND_2000, ILA_TEXT_GET, ANLZU_CHECK, ANLZU_CHECK_O, /CUM/CUDESIGN_TXT, INGRP, INNAM, ILART, ILATX

Reason and Prerequisites

Reason:

There is a program error.

Prerequisite:
You exit the compatible unit design and then call it again.

Solution

Implement the attached correction instructions.
