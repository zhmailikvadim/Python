Symptom

Error message AU271 during execution of FAGLGVTR.

Environment

  • FIN Asset Accounting​
  • SAP S/4HANA ​
  • SAP S/4HANA Finance​
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. TCode FAGLGVTR;
  2. Company Code XXXX;
  3. Fiscal Year 202Y;
  4. Remove test flag.

Cause

The asset has no value in the 'First Acquisition On' field, even though it has a 'Capitalized On' date and an acquisition has been posted.

Resolution

  1. Access T-Code AS02.
  2. Manually input the date in the 'First Acquisition on' field for asset.
  3. Save the changes.
  4. Execute T-Code FAGLGVTR again to carry forward to Fiscal Year 202Y.
