Symptom

The display behavior of comment indicator icon is inconsistent. 

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental. 

Environment

SAP SuccessFactors Performance Management

Reproducing the Issue

  1. The latest form experience is enabled.
  2. The section is using rating option 3. 
  3. When the form is at signoff/complete stage, and the user viewing the form has not inputted any item comment before, the empty comment box indicator icon does not display for the user.
  4. However, for section comment, for the user viewing the form has not inputted any item comment before, the empty comment box indicator icon displays for the user.

Cause

This is a Known Issue.

Resolution

The fix for this issue is planned to be deployed on 1H 2026 release.

For release timelines, please review SAP SuccessFactors Product Release & Road Map Information.

Note

After the fix, the expected display behaviors are:

  1. if there's the standard rating/section comment field for user to fill regardless of being required or optional, and user has not filled, then empty comment icon is displayed.
  2.  if there's the standard rating/section comment field for user to fill regardless of being required or optional, and user has already filled, then greyed comment icon is displayed.
  3. if there's no standard rating/section comment field available for user to fill, then no comment icon is displayed.
