Symptom

When you add a formatted search (FMS) into a user-defined field (UDF) and it is triggered by a field in Bill of Materials form, the FMS is not executed.

Reproducing the issue

Prerequisites:

The option Enable SAP Business One Suggest is selected under the Choose from List Preferences section in the Administration → System Initialization → General Settings → Display tab.

Scenario:

  1. Under Tools → Customization Tools → User-Defined Fields - Management create a UDF for BoM - Rows under the category Production.
  2. Go to Production, open Bill of Materials (BoM) window and in the above UDF create an FMS triggered by field No. with query SELECT 1.
  3. Create a new, or open an existing BoM and start to type an item number into the No. field.
  4. A drop-down list with suggested item numbers is displayed. Select one of the items.

Expected behavior:

In the UDF value 1 is triggered by the FMS.

Actual behavior:

The FMS is not triggered in the UDF.

Solution

SAP intends to provide a feature package or patch to solve the symptom that is described in this SAP Note. The section References below lists the related feature package or patch once they become available. Be aware that References can only be confirmed at feature package or patch release date. SAP delivers feature packages or patches only for selected releases at its own discretion, based on the business impact and the complexity of implementation.

Workaround:

  1. Select the No. from the List of Items.
  2. Type in the item number and press TAB key.
