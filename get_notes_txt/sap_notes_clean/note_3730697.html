Symptom

When trying to access the Spot Buy Tab in Guided Buying I receive an error as "Sorry. It looks like this might take longer to fix" and the Spot Buy page doesn't not load.

Cause

This is happening because the improved catalog search page in Guided Buying isn't enabled, the error happens only when trying to access the Spot Buy page with the legacy search results page in Guided Buying.

Resolution

To fix the issue, enable the following parameters in Guided Buying Admin page:

  1. PARAM_ENABLE_IMPROVED_SEARCH_EXPERIENCE
  2. PARAM_ENABLE_IMPROVED_SEARCH_EXPERIENCE_2

See Also

For more information on the Improved search parameters, check the Improved Search in Guided Buying documentation.
