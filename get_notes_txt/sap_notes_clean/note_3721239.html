Symptom

When additional line items are added to an existing Sales Order, the system creates separate planned orders for each item. However, during subsequent JIS processing, no JIS call-off is generated for the newly added items since the system finds that the Sales Order is already used; and a message is issued that no demands were found.

As a result, JIS call-offs are created only for the original Sales Order item, while the subsequently added line items are ignored.

Other Terms

Next generation Just-In-Time processing, JIS call creation, S2P, JIS processing, planned order, Sales Order item.

Reason and Prerequisites

Program error.

Solution

Implement the correction instructions of this SAP Note or import the relevant Support Package.
