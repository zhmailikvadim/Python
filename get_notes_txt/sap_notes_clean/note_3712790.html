Symptom

Requirement: It should be possible to view only the necessary buttons for the selected row/Item in the tab 'All Objects' for Change Record.

Other Terms

PLM-CR, All Objects tab, display, items, filter, sort, object type, 3655773, 3680118.

Reason and Prerequisites

Advance Development.

Prerequisite SAP Note:3680118.

The correction from the current SAP Note should be implemented after implementing the changes from the SAP Note 3680118.

Solution

Use transaction SNOTE to implement the corrections delivered with this SAP Note in the affected Change Record UI system or import the relevant Support Package.
