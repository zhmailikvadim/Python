Symptom

Error "No Authorization to Start the UI for Transaction S_ER9_52004031" occurs when attempting to access Configuration Activity "Map Outbound Work Assignment Status" (SSCUI ID: 106183).

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental. 

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Navigate to SSCUI 106183 Map Outbound Work Assignment Status.
  2. Attempt to access the Configuration.
  3. Encounter the error "No Authorization to Start the UI for Transaction S_ER9_52004031".

Cause

Business Catalog ID SAP_CA_BC_IC_LND_WFD_OUT_PC is not assigned to the related Business Role. 

Resolution

Add the Business Catalog ID SAP_CA_BC_IC_LND_WFD_OUT_PC to the related Business Role.
