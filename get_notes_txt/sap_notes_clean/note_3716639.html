Symptom

After performing an Initial Load for Templates, you notice that some information is missing in the target tenant. Specifically, documents associated with the templates cannot be found.

Environment

  • SAP Sales Cloud Version 2
  • SAP Service Cloud Version 2

Reproducing the Issue

1. Log in to the SOURCE tenant.
2. Navigate to the User Icon > System Settings.
3. Go to Emails > Templates.
4. Filter by "ABC" (where "ABC" is the name of the template).
5. Navigate to the Documents tab.
6. Observe that there is at least one document present.

7. Log in to the TARGET tenant.
8. Navigate to the User Icon > System Settings.
9. Go to Emails > Templates.
10. Filter by ABC (where "ABC" is the name of the template).
11. Navigate to the Documents tab
12. Observe that no documents are present.

Cause

Documents are not automatically transferred during the Initial Load for Templates. This process requires manual intervention, as file transfers are not supported as part of the change project for Templates.

Resolution

To ensure that your documents are available in the Target tenant, download the documents from the source tenant and then upload them directly into the target tenant.
