Symptom

Although product hierarchy has been configured, there is no value for the field "Product Hierarchy" in the Sales Analytics apps such as "Incoming Sales Orders", "Sales Volume - Flexible Analysis", etc.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

Taking the "Incoming Sales Orders" app as an example:

  1. Open the "Incoming Sales Orders" app.
  2. Input the "Exchange Rate Type" and "Display Currency" fields and then choose "Go".
  3. In the filter results, there is no value for the field "Product Hierarchy".

Cause

  • The property "ProductHierarchyNode" is valid only for the On-Premise system.
  • For the Sales Analytics apps in SAP S/4HANA Cloud Public Edition, instead of using this field directly, there are specific steps as shown in the below "Resolution" section.

Resolution

  1. Add "Product" to columns or rows at the left-side navigation panel of the app.
  2. Right click on "Product".
  3. Select "Hierarchy".
  4. Click "Select Hierarchy" and in the next screen the hierarchies will be available.
