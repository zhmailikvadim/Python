Symptom

You want to implement the Electronic Document Consistency Framework to detect and resolve inconsistencies in your business documents.

This SAP Note provides the necessary technical objects to run the framework.

The framework is available starting from S4CORE 108 (SAP S/4HANA On-Premise 2023) and later versions.

Other Terms

SAP Document and Reporting Compliance, Electronic Document Consistency, Consistency Framework, EDC, DRC, Data Dictionary Objects, DDIC, Technical Objects

Reason and Prerequisites

Reason

You want to implement the necessary ABAP objects to use the apps and features of the Electronic Document Consistency Framework.
