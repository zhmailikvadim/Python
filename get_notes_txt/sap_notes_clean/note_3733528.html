Symptom

After upgrade of the Solution Manager 7.2 to SP19 and higher, you get a syntax error in program  CL_SMT_WC_TPM_TPCK_EXIT=======CP 
Include CL_SMT_WC_TPM_TPCK_EXIT=======CM011: Syntax error in line 000001
Method 'ADAPT_EVENT_DELETE' does not exist. Thereis, however, a method with the similar name 'ADAP 

Environment

  • SAP Solution Manager 7.2
  • Test Management

Reproducing the Issue

Syntax error after upgrade to SP >=19

Cause

Method ADAPT_EVENT_DELETE was removed and no longer needed in higher SP's

Resolution

The Method is obsolete in higher SP's and clean-up is required.

  1. Execute SE24
  2. Select Class CL_SMT_WC_TPM_TPCK_EXIT
  3. Click on Change
  4. From the menu, select Utilities > Clean Up > Method includes
  5. confirm the pop-up that you want to delete - click yes
  6. Activate
