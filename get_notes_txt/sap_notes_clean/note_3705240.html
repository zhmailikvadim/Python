Symptom

The message type /SAPSLL/BNKMAS_SLL does not exist in transaction code BD52.

Environment

  • SAP S/4HANA

Reproducing the Issue

  • Attempt to locate the message type /SAPSLL/BNKMAS_SLL via BD52 and observe that the message type is not found

Cause

Resolution

Follow the steps from SAP Note 1755925 and add message type /SAPSLL/BNKMAS_SLL manually via BD52.

See Also

  • Refer to SAP Note 0001755925 - SLL PI: Change Pointer for Bank.
