Symptom

This SAP Note delivers new versions of Payment Item BAPIs with an increased field length for the External Clearing Reference. These BAPI's do not have the Restricted Savings Enhancement integrated to them.

Other Terms

Payment Item BAPIs,

BAPI_FSTB_PAYMITEM_POST_CLRING,
BAPI_FSTB_PAYMITEM_POST_ITM_V1,
BAPI_FSTB_PAYMITEM_POST_RCV_V1,
BAPI_FSTB_PAYMITEM_POST_SND_V1,
BAPI_FSTB_PAYMITEM_GET_DET_V1,

External Clearing Reference, Payment Item Posting, BankClearingPaymentGroupID

Reason and Prerequisites

Advance Development

Solution

The following are the new version BAPI names for Payment Items:
BAPI_FSTB_PAYMITEM_POST_CLRING,
BAPI_FSTB_PAYMITEM_POST_ITM_V1,
BAPI_FSTB_PAYMITEM_POST_RCV_V1,
BAPI_FSTB_PAYMITEM_POST_SND_V1,
BAPI_FSTB_PAYMITEM_GET_DET_V1.

It is recommended to import the relevant Support Package for your release. You can also use transaction SNOTE to implement the correction instructions.
