Symptom

This SAP Note will delegate the message identifiers into R2026.JUN release envelope via derivation class.

Other Terms

T2, TARGET2, /T2_ENV_MSG_SR2026, /PE1/CL_IPM_MSGID_T2, /PE1/CL_IPM_T2_MSG_DISPATCHER

Reason and Prerequisites

New T2 Envelope for R2026.JUN release

Solution

Implement the correction instruction attached to this SAP Note. Alternatively, you can import the referenced Support Package.

The only difference between the envelope of R2026 and R2025 is the new Business Application Header (BAH) version 4. The underlying message identifiers remain the same and will be updated with the R2026.NOV release.
