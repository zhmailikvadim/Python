Symptom

You observed following error information:

e LJIT ljit_types_EAPI.cpp(00380) : types: failed to retrieve current connection
e ceExecute cePopCustomLjit.cpp(00580) : Llang Runtime Error: Exception::SQLError: No connection found

Other Terms

You're using planning engine command PleCheck

Reason and Prerequisites

Issue number 349085

Reason:

  • Due to a programming error in SAP HANA planning engine.
  • You are using planning engine of BW Integrated Planning in combination with PAK on HANA level. 

Solution

Apply one of the following SAP HANA Revisions:

  • SAP HANA 2:
    • Revisions >= 079.08 (SPS07)
    • Revisions >= 089.01 (SPS08)
    • or higher

Workaround:

  • Include the involved planning function in the table RSPLS_HDB_ACT_PF  via transaction SM30, so this specific PF is forced to be executed in ABAP runtime. Make sure the column "Active" of the table entry remains unchecked.
