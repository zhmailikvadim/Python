Symptom

How to record the activation of Switch BC Sets in a customizing transport, with is classified with attribute Special Customizing.
Example: (See the output of report SCPR_ACTIVATE_SBCSETS)

Environment

This KBA applies to all SAP Products based on NetWeaver and S4H.

Cause

The method is unknown.

Resolution

  1. Execute transaction SFW5
  2. In screen Security Information read the information and press button Continue
  3. Navigate over the menu Systems Settings -> Special Functions -> Special Customizing
  4. Tick only the Switch BC Set you plan to activate
  5. Press icon Activate Selected BC Sets  (F6)
  6. In screen Prompt for Customizing request or Prompt for Workbench Request press icon Create Request  (F8) or choose an existing request
  7. Press icon Continue
  8. Import the transport request to subsequent system
