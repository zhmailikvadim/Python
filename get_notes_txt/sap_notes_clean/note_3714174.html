Symptom

Using the frontend services methods in SAP GUI for HTML can give problems when filenames contain non-breaking spaces (UTF8: C2A0, UTF16: A000).
In such situations the non-breaking space is converted to a space (UTF8: 20, UTF16: 2000) and later accesses with this changed file names to the filenames on client site will fail.

Other Terms

SAP GUI for HTML, ITS, CL_GUI_FRONTEND_SERVICES, FILE_OPEN_DIALOG, FILE_EXISTS, GUI_UPLOAD.

Reason and Prerequisites

Because of technical reasons filenames with non-breaking spaces after spaces will be converted to spaces.

Solution

Do not use filenames containing non-breaking spaces after spaces.
