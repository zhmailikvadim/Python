Symptom

After enabling the new approach for time task generation, time tasks are not created despite all prerequisites being fulfilled as described in the migration guide. The issue occurs specifically with Connector version 400.

Other Terms

FSM, E4C, Connector, Time Task, Cloud Connector

Reason and Prerequisites

A minor issue in the implementation of the new time task creation logic in Connector version 400 prevents the correct generation of time tasks. This occurs even when the system is fully configured according to the migration documentation and all required prerequisites are met.

Solution

The issue has been resolved through a code correction. Please implement this correction instruction to enable proper time task generation with the new approach.
