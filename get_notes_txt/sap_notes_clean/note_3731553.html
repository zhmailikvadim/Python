Symptom

Error message "reports are not completed for earlier period(s)" is observed while trying to access GSTR3B reporting activities as the previous reporting task has not been completed.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Open the Run Statutory Reports application.
  2. Attempt to access a GSTR-3B reporting activity.
  3. Observe the error: "reports are not completed for earlier period(s)".

Resolution

Ensure the previous period reporting task is marked as completed and the "Use as of" Date for the report is maintained as suggested in Setting Up the GSTR3B Report | SAP Help Portal.

Keywords
