Symptom

In the process "Receipt of Supplier Clearing List from DSO (SUP View)" (DE_C_EQS_CLSUP_DSO_RECEIVE_SUP), ENDSETTLDATE was updated incorrectly when receiving the supplier clearing list for time slices greater than one.

This issue is relevant for the market roles: PRS, DSO.

Other Terms

/US4G/CL_PRST_EQS_CLSUP_IN, /US4G/EQS_CLSUPI, clearing list, ENDSETTLDATE, Receipt of Supplier Clearing List from DSO (SUP View), DE_C_EQS_CLSUP_DSO_RECEIVE_SUP

Reason and Prerequisites

Program error

Solution

Please implement the corrections of this SAP Note or import the relevant Support Package (SP).

After the fix is installed, ENDSETTLDATE will be updated correctly for more than one time slice.

You need to retrigger the process to apply the fix.

You can verify the fix by checking that ENDSETTLDATE is updated correctly for more than one time slice.
