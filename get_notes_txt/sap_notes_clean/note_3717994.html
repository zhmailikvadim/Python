Symptom

When creating a new parameter in the CPF Parameter Catalog via SPRO, the system does not allow data type TIME (HH:MM:SS) and raises an error.

Environment

SAP S/4HANA

Reproducing the Issue

  1. Go to SPRO > Sales and Distribution -> Sales -> Configurable Parameters and Formulas in Sales -> Requested Delivery Date in Sales Documents -> Define Parameter Catalog for Requested Delivery Date

  2. Enter values and press ENTER



  3. Error appears

Cause

The Manage Characteristic Catalog (CPF Parameter Catalog) supports only a limited set of data types based on the underlying VDM model.

Data type TIME (TIMS / HH:MM:SS) is not supported in standard.

Resolution

This is standard SAP system behavior.

It is not possible to create a CPF parameter with TIME data type in standard SAP.
