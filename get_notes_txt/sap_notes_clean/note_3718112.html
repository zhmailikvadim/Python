Symptom

How to filter user and group data in SAP Ariba SCIM API?

Resolution

    To filter user and group data using the SCIM API, you must use the filter query parameter, following the syntax below:

    • For Users:
      • GET https://{{baseUrl}}/api/mds/v1/prod/scim/Users?&filter=userName eq "userNameToFilter"
    • For Groups:
      • GET https://{{baseUrl}}/api/mds/v1/prod/scim/Groups?&filter=displayName eq "groupDisplayNameToFilter"

    Please note: as of February 2026, the SAP Ariba SCIM API documentation incorrectly specifies the query parameter for filtering as $filter. The correct parameter is filter, without the dollar sign. An internal ticket has been raised to address the issue with the API documentation.
