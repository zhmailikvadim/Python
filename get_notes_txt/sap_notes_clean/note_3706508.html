Symptom

In the application log for the availability check with advanced Available-to-Promise (external identification ATP_CHECK_ONLINE), the message ATP_SBC_SOURCING 001 appears, even though Supply Creation-based Confirmation (SBC) is not used for the corresponding material. The text of the message is "Sources are not configured for line item <x>/<y> product <z>".

Other Terms

CL_ATP_SBC_CONTROLLER_ADAPTER  CL_ATP_SBC_SOURCING_CONTROLLER  GET_SOURCING_METHOD_INSTANCE

Reason and Prerequisites

Program error: If an ATP check implicitly involves the deletion of a sales order item (for instance, when repeatedly checking an item that uses Alternative-based Confirmation), also SBC is involved about the deletion. There is no check if the item to be deleted is relevant for SBC. It is recognized that data to process the deletion in SBC are missing and the log entry is written.

With the correction, it is correctly recognized that SBC is not used for the item.

Solution

Upgrade to S/4HANA 2023 SP06. If this is not possible, apply the attached correction instruction.

Make sure that also the correction instructions from the prerequisite notes are applied. Usually this should be done automatically by the note assistant.
