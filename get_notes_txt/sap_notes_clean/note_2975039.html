Symptom

Up to now, a maximum of 9 additional documents can be created (FKK_CREATE_DOC_ADDITIONAL, event 0031). However, you may require more than nine additional documents.

Other Terms

FP-EVENT-0031 FKK_SAMPLE_0031

Reason and Prerequisites

*

Solution

Implement the correction instructions. The number of possible additional documents is increased from 9 to 9999.
To create more than nine additional documents, you have to set the new parameter I_XXL_ACTIVE and accept the new parameter E_HANDLE_XXL when calling FKK_CREATE_DOC_ADDITIONAL. In the module for the event 0031, the system then evaluates the new parameter I_HANDLE_XXL.
