Symptom

You are using outbound synchronous interfaces to integrate C4C with external system and you would like to know if the outbound synchronous interfaces create/update ID mappings

Environment

SAP Cloud for Customer

Resolution

As per the standard behavior, the outbound synchronous service interfaces will not update or create ID Mappings.
