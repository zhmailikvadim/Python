Symptom

When making changes in existing workflows in the Manage Workflows for Journal Entry Verification - In General Ledger app, an error message shows:

"An error occurred while editing the workflows.

The operation has been canceled."

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Open the "Manage workflows for General Journal Entry Verification" app.
  2. Deactivate an active workflow.
  3. Edit the desired value in the deactivated workflow.
  4. Save the changes and reactivate the workflow.
  5. Observe the error message displayed: "An error occurred while editing the workflows. The operation has been canceled."

Cause

Editing active workflow is not supported.

Resolution

  1. Open the existing workflow that requires modification.
  2. Select the "Copy" option to create a duplicate of the workflow.
  3. Modify the desired value in the newly created workflow.
  4. Save the changes made to the new workflow.
  5. Activate the new workflow.

Note: Deactivating or deleting the original workflow will not impact existing journal entries that were processed through the old workflow. Creating the new workflow will only affect future journal entries created after the activation of the new workflow.
