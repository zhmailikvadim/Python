Symptom

When trying to access SSCUI/Configuration Activity "Define Batch Input Session Parameters" ID 104808, the following error messages are raised: 

  • No authorization to start the UI for transaction S_ER9_11001862

Environment

  • SAP Central Business Configuration (CBC)
  • SAP S/4HANA Cloud public edition

Reproducing the Issue

  1. In the CBC workspace, click the tab Configuration Activities.
  2. Search for the Configuration Activity 104808.
  3. Click the entry.
  4. Click the button Go to Activity.
  5. Observe the error message.

Cause

CAUSE 1 - Missing Business Catalog.

OR

CAUSE 2 - SSCUI is specialized for certain countries.

Resolution

CAUSE 1 - Missing Business Catalog.

  1. The SAP_CA_BC_IC_LND_FIN_GL_ES_PC business catalog should be added to the BPC_EXPERT business role (or to the similar role in use).
  2. Additionally, to have editing rights, please ensure to grant "Write Access" to "Unrestricted" for the BPC_EXPERT business role (or for the similar role in use).

CAUSE 2 - The 104808 SSCUI is available only for Spain. The SSCUI will not be available for other countries.

If neither Cause 1 nor Cause 2 resolves the issue, please create a support case under component FI-LOC-FI-ES and provide the affected CBC user, as described in KBA 2838282.
