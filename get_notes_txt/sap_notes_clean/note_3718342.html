Symptom

Technical name of the table and the columns in the Local table contains other characters rather than alphanumeric characters and underscores in SAP Datasphere. It caused an error when loading the data to the table. The table was created by importing from a CSN/CSV file.

Environment

SAP Datasphere

Reproducing the Issue

Check the technical name of the table and the columns in the Local table.

Cause

Technical names can contain only alphanumeric characters and underscores. The validation on technical names is not invoked when importing the local table.

Resolution

The development team is checking whether the validation on technical names will be introduced when importing the local table. Before that, remove the spaces in the technical names in the CSN/CSV file.
