Symptom

  • Error occurs during installation at the repository configuration step.
  • Error message: "An error has occurred. Invalid Data Services repository connection information."

Environment

  • SAP Data Services 2025
  • Supported Linux OS

Reproducing the Issue

  1. Start the installer for Data Services on Linux.
  2. Proceed to the Local Repository configuration step and enter repository connection details.
  3. Click Next. The installer shows: "An error has occurred. Invalid Data Services repository connection information."

Cause

In this case, the $ODBCINI environment variable was not set.

Resolution

  1. Identify the absolute path to the odbc.ini file on the host.
  2. Set and export the ODBCINI environment variable to point to this file:
    ODBCINI=/path/to/odbc.ini; export ODBCINI
  3. Retry the repository configuration step in the installer.
