Symptom

  • EWA Report is generated but only PDF Version is available.
  • The Fiori and .DOCX version are not available. 

Environment

SAP Solution Manager 7.2

Reproducing the Issue

  1. Open the system.
  2. Go to SDCCN.
  3. Search for the version wanted.

Cause

Missing EWA Configuration related to Fiori and .DOCX version.

Resolution

To ensure full EWA generation (PDF, Fiori, Word) and email notification, follow the steps below:

  1.  Do not create EWAs manually for regular weekly reporting.
  2.  Ensure that SM:EXEC SERVICES job is: Scheduled, Running successfully, Not canceled or delayed.
  3. Allow the EWA to be generated automatically via the scheduled job.
  4. Wait for the next regular EWA cycle.
  5. Verify that: PDF, Fiori, and Word versions are available, Email notification is sent as expected.

Keywords

EarlyWatch Alert, EWA, SM:EXEC SERVICES, PDF only, Fiori missing, Word missing, manual EWA, Solution Manager, EWA email not sent.
