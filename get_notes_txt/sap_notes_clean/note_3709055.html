Symptom

  • An error occurs when attempting to update a project created using the "Project Control - Enterprise Projects" app.
  • The error message displayed is: "Project Element <project-id> exists.".

Environment

  • SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Create a new project using the "Project Control - Enterprise Projects" app.
  2. Save the project.
  3. Attempt to edit the project.
  4. Observe the error message: "Project Element <project-id> exists."

Cause

The issue occurs due to a draft entry with the same project external ID being created and not discarded.

Resolution

  1. Identify the draft entry associated with the project external ID via app Project Control - Enterprise Projects.
  2. Request the user to discard the draft entry.
  3. Once the draft entry is discarded, proceed to update the project.
