Symptom

  • When executing BAPI_FTR_SECURITY_DEALCREATE only an amount with 6 decimal places is allowed as the field for the price is defined with DEC(15,6)
  • When comparing it to TS01 the system allows to enter 14 decimal places. 

Environment

Treasury and Risk Management (TRM)

Cause

Configuration

Resolution

Use the field PRICE_LONG at the end of the import structure BAPI_FTR_CREATE_SECURITY within BAPI_FTR_SECURITY_DEALCREATE which supports the same amount of the decimals as in TS01.

Keywords

BAPI_FTR_SECURITY_DEALCREATE, price, PRICE_LONG
