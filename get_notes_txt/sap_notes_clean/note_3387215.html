Symptom

There is an exception exists for Inbound Delivery Notification for product in planning view however no error in respective document's document flow.

Environment

SAP Business ByDesign

Reproducing the Issue

  1. Go to Supply Planning work center.
  2. Go to Products view.
  3. Find required ID with planning area.
  4. Here select and choose open.
  5. In planning view, there is a row with Inbound Delivery Notification with an exception (red mark).

Cause

This is an exception for Inbound Delivery Notification as per the Availability Date of the document in the past and not for the document flow.

Resolution

  1. Go to Supply Planning work center.
  2. Go to Products view.
  3. Find required ID with planning area.
  4. Here select and choose open.
  5. Under Supply and Demand view, navigate to the Exceptions sub tab.
  6. Here exception detail is displayed.
  7. As per an exception, user can select and choose Acknowledge option to accept it.

Note: It is valid for other documents also.

Keywords

planning, exception, inbound delivery notification, red mark, supply and demand, planning view.
