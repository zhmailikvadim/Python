Symptom

PR is no longer visible or available in Ariba Buying / Ariba Guided Buying.

Reproducing the Issue

When user searches for the PR, it is no longer available/visible in the system.

Cause

PR was deactivated via Delete action on the document and currently it's not active in the database. With delete action on the document, database record was changed and inactivated. This causes the PR to be unavailable for further use.

Resolution

PR was deleted and is no longer active in the system (can be confirmed in Database by SAP Support).
Such PR cannot be reactivated and new PR needs to be created.

Keywords

PR inactive, PR not visible, PR delete
