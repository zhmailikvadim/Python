Symptom

1. If a document has been deleted in the feeder system and is no longer available in the GTS system, adding a new row to the "Document Number" attribute of the license will trigger an error message and block the license data from being saved.

Error message: Enter a valid document reference

2. If a user adds a new row with only the logical system filled in the "Document Number" attribute, then presses the Enter key and clicks the "Save" button, the new row is saved successfully.

3. If a user adds a new row for the "Document Number" attribute and presses the Enter key, and then adds a second new row and presses Enter again, a short dump occurs.

Runtime Errors: TABLE_INVALID_INDEX

Other Terms

/SAPSLL/LCLIC_MNG, /SAPSLL/LCLIC01, /SAPSLL/LCLIC02,  /SAPSLL/CON_LMGM222, TABLE_INVALID_INDEX

Reason and Prerequisites

Program error.

Solution

Apply the correction instruction in this note or upgrade to relevant support package.
