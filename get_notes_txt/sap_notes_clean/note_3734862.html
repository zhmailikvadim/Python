Symptom

When trying to display the payslip the following error message appears:

Error Details:

  • Technical Exception: Technical Exception not available
  • Error description: User *** does not exist in this period
  • Additional Information: The System encountered an error while opening the application. The operation Will be terminated

Environment

  • SAP R/3
  • SAP ERP Core Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP SuccessFactors Employee Central integration to SAP Business Suite
  • SAP SuccessFactors Employee Central Integration with SAP Human Capital Management for SAP S/4HANA
  • SAP SuccessFactors Employee Central Payroll

Reproducing the Issue

  1. Log in to SuccessFactors.
  2. Navigate to "View My Pay Statement" and click on "Pay Statement."
  3. Observe that the Payslip cannot be displayed and an error message appears: User *** does not exist in this period

Cause

This issue can have multiple root causes depending on the scenario:

  • User does not exist in SU01 transaction
  • User is locked
  • User validity incorrect (not in validity period)
  • Incorrect entry in IT0105 / subtype 0001 

Resolution

  1. Check the user in SU01 transaction if it exist at all.
  2. Check in SU01 transaction if the user is locked
  3. Check the validity period of the user
  4. Verify that the user has a valid entry in IT0105 / subtype 0001
