Symptom

The Report RSBKCLEANUPBUFFER dumps with SAPSQL_PARSE_ERROR and CX_SY_DYNAMIC_OSQL_SEMANTICS while accessing a not existing DB-Table

Other Terms

CX_SY_DYNAMIC_OSQL_SEMANTICS, SAPSQL_PARSE_ERROR, RSBKCLEANUPBUFFER

Reason and Prerequisites

Program Error

Solution

To solve the issue please apply the Support Package as mentioned below. In urgent cases you can use the correction instructions.
