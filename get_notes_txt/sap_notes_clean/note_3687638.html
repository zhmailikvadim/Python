Symptom

 You use the USTAT field in the Price Review search screen in Fiori and expect it to filter the results according to the configured Status Profile. The USTAT field does not apply the filtering as per the Status Profile. As a result, irrelevant or incorrect statuses appear in the search results. This note corrects the behavior by ensuring that the USTAT field in the Fiori Price Review search filters results accurately based on the assigned Status Profile.

Other Terms

/IRM/GDLRS, Status profile

Reason and Prerequisites

This is caused by program error.

SAP Note#3630241 is available for V4 release.

Solution


Please implement the correction instructions. These corrections are only valid if you have applied the following Support Packs.

Release – 6.4 Base Package & Support Pack 1
Release – 6.3 Base Package

Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object
