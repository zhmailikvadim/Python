Symptom

The move-in reversal for an IS-U contract cannot be performed due to the error EMV 116 "Documents &2 &3 &4 have not been reversed". The aforementioned documents are payments of cash security deposits.

Other Terms

EC53; EC53E; cash security; payment of cash security deposit; EMV-116; 116(EMV); EMV 116; EMV116

Reason and Prerequisites

Cause:

The data constellation as described below was not taken into account in the function module 'ISU_MOVE_IN_ITEMS_CHECK’.

Prerequisite:

The payments of cash security deposits have the origin "RT" (security payment transfer) and are already cleared (paid). The related cash security deposit has the status 98 (returned).

Solution

You can implement the corrections as an advance correction. The correction is usually delivered with the relevant Support Package for the release specified.

This correction changes the system behavior. The°error message is now issued°as warning message EMV 116. Check using the information provided by the message EMV 116°whether a move-in reversal can be carried out nevertheless.
