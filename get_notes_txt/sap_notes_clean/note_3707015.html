Symptom

You may get CVE note 3363690 from the Vulnerability note implementation list and you may face different issues when implementing the note including its manual steps.

Resolution

Before implementing this note, please verify if you do have MDG relevant function in use on the corresponding system. The easiest way is to check if you can run Tcode MDGIMG on your system. If you get error "Switch FIN_MDM_CORE_SFWS_EHP5 is inactive", it means that there is no MDG function run on yoru system. Then you do not need to implement note 3363690 as this only resolves potential vulnerability issue when use file upload function in MDG UI.
