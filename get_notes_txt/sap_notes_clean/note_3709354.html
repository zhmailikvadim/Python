Symptom

  • Unable to update the "sort string" field using the API operation /UpdateBOMItemWithECN.
  • Other fields on the BOM item are also not updated with blank values.

Environment

SAP S/4HANA Cloud public edition

Reproducing the Issue

  1. Use the API operation /UpdateBOMItemWithECN to update the "sort string" field.
  2. Provide the payload with blank values for the "sort string" field and other fields.
  3. Execute the API call.
  4. Observe that the "sort string" field and other fields are not updated, and no error messages are displayed.

Cause

The issue is caused by a bug in the system.

Resolution

The issue has been identified as a bug. A fix is planned to be deployed in CFD 2602.02, with the RTH date scheduled for 16.04.2026.
