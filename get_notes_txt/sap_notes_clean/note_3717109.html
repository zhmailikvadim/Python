Environment

  • SAP S/4HANA, on-premise Release Independent
  • SAP S/4HANA Cloud Private Edition Release Independent 

Reproducing the Issue

  1. Run MRP Live with transaction MD01N
  2. Execute the planning run for materials
  3. Review the job log or short dump in transaction ST22
  4. Observe termination with runtime error AMDP_EXECUTION_FAILED referencing method SELECT_SOURCE_FROM_CDS on ABAP Program CL_PPS4_MRP_PURCHASING========CP

Cause

The dump occurs because Source List data is incomplete. The Field Control Indicator (EORD‑EORTP) is not maintained for one or more materials included in the MRP Live run.

When the system attempts to evaluate sourcing options with database procedure SELECT_SOURCE_FROM_CDS, the missing field Control Indicator (EORD‑EORTP) leads to an invalid number error.

Resolution

Maintain the Source List with a valid Field Control Indicator (EORTP) for the materials involved in the dump.

  1. Identify the materials that caused the dump using the Material Identification option as explained in SAP Note 2668805.
  2. Open table EORD for each identified material and plant combination.
  3. Check the field EORTP (Field Control Indicator).
  4. If the field is not filled, recreate the Source List entry using transaction ME01.
  5. Save the updated entries and rerun MRP Live.
