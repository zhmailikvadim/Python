Symptom

When saving an e-mail as Activity, the follow-up Activity status is set to Completed.

Reproducing the issue

Prerequisites:

You are using the Outlook Integration add-on.  

Scenario:

  1. Open the Save E-mail as Activity option in Outlook Integration.
  2. Choose to Create a Follow-Up Activity and select the Activity to be a Task.
  3. Choose a (future) due date and then Save E-mail.

Expected Behavior:

The Follow-Up Activity in SAP Business One has the Status Not Started. 

Actual Behavior:

﻿The Follow-Up Activity in SAP Business One has the Status Completed. 

 

Solution

Correction of system behavior is not feasible in the currently supported versions of SAP Business One. The error has been recorded and is a candidate to be fixed in a future version.

Workaround:

Manually change the status of the Activity.
