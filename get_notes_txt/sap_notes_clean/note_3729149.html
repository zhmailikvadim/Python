Symptom

The following error occurs when opening control monitor

An exception with the type CX_SY_CONVERSION_NO_NUMBER was raised, but was not handled locally or declared in a RAISING clause.
The argument 'A' cannot be interpreted as a number

Environment

  • SAP Business Planning and Consolidation 10.1, version for SAP NetWeaver
  • SAP Real-Time Consolidation (RTC) in S/4HANA (BPC Embedded Model)

Reproducing the Issue

Open control monitor via BPC web client

Cause

Invalid control level property values (e.g. "A") are maintained in /ERP/ENTITY dimension. Specifically, for the control level property, the expected value should be a positive integer no greater than 4.

Resolution

Update the affected property values to valid positive integers within the specified range or leaving them empty.
