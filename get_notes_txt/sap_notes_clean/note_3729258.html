Symptom

  • An error occurs when posting interest flows in the transaction FWSO for a variable interest bond.
  • The error message Message No. TRS0063 is displayed: "Interest rate adjustment for &1 must take place before posting".

Environment

  • Treasury and Risk Management (TRM)
  • EA-FINSERV
  • S/4CORE
  • SAP ERP
  • SAP ERP Central Component
  • SAP enhancement package for SAP ERP
  • SAP S/4HANA
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA Finance

Reproducing the Issue

Execute transaction FWSO for the bond position, error message TRS0063 is raised in pop-up or message log by clicking button "Display message log" .

Cause

Missing interest rates for the relevant Reference interest rate for specific interest fixing dates.

Generally speaking, there has to be a valid entry in T056P for the interest FIXING DATE from the interest fixing condition. If there is no such entry on this special fixing date, an error message is triggered in transaction FWZE or FWSO (error message TRS0063). FWSO will always expect an entry in the reference table at the EXACT interest fixing dates - this is how this transaction works.

This is how this functionality is designed. If the system took an older entry in case that there is no entry at the exact date it would be possible that you fixed the interest with a wrong, non-up-to-date rate easily by mistake.

Resolution

  1. Maintain interest rate for the Reference interest rate in transaction `S_ALR_87004555` or view `V_T056P` via Tcode`SM30`.
  2. Update planned records using transaction `FWUP` as described in KBA 3377466 - How to fix variable interest rates for Securities.
