Symptom

The following symptom was identified in the Relação Contributiva para CGA (RPC_PAYPT_CGA) report:

  • When the employee becomes retired in the beginning of the month, the Desligado do serviço – Aposentação (47) Situation code is not reported. 

Other Terms

Transaction: PC00_M19_CGA_PS

&CGA; (for more information, see SAP Note 2197631)

Reason and Prerequisites

The following Support Package level is the minimum prerequisite:

ReleaseSupport Package
6.00 HR SP F3
6.04 HR SP B9
6.08 HR SP 47

(For more information, click on "HR Support Packages Schedule" in support.sap.com/maintenance-schedule)

Occurs because the RPC_PAYPT_CGA report was not able to process Social Security PT (0332) Infotype records when the employee changes the Regime de Segurança social (SSRGM) field from Regime Único (C1) or Regime Misto (C2) to a different one when the month changes.

Good to know: when you install an SAP Note, some prerequisites might be required. The prerequisites are SAP Notes that were already released and that change the same context of the current SAP Note. So, even if these prerequisites are not necessarily related to the symptom of the current SAP Note, you should implement them due to technical reasons.

Solution

The RPC_PAYPT_CGA report has been changed to process not only the situation code 47, but also other situation codes based on actions such as Cessação de funções (43), Falecimento (45), Desligado do serviço – Comissão ou requisição(48) and Licença sem vencimento ou licença especial ( > 90 dias) (50), to be processed when the employee stop contributing with CGA in the beginning of the month.

Example:

Proceed with the following steps to reproduce the symptom mentioned in this SAP Note.

Description

CGA – Employee becomes retired in the beginning of the month 

Preparation/Background

1. Make sure that action 14 is mapped in V_T5PPBSRM to generate situation code 47 with Modificador de data para tipo de ação (DAMOD) field equals Data de início menos de um dia (10) value.

2. Hire an employee on 01/01/2017 with EUR 1000 and contributes with CGA.

3. Execute payroll until 01/2017.

4. Retire the employee by:

Adding an action 14 starting from 01/02/2017.

Adding an 0332 Infotype with SS Regime "Reformado" starting from 01/02/2017.

5. Execute payroll until 02/2017.

Execution

1. Run the RPC_PAYPT_DRI for 02/2017.

Result

After the execution of step 1:

1. A situation code 47 must be reported with effective date 31.01.2017.

Note: These are the expected results once the SAP Note is installed.
