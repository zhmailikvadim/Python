Symptom

Short dump occurs when processing DESADV IDoc for changing an inbound delivery that is distributed to a receiving EWM system. The inbound delivery has batch split items.

Other Terms

MESSAGE_TYPE_X, ITAB_DUPLICATE_KEY_IDX_OP, /SPE/IDOC_INPUT_DESADV1

Reason and Prerequisites

You have installed note 3501484 or corresponding support packages.

When processing DESADV IDoc for changing an inbound delivery that is distributed to a receiving EWM system, a temporary delivery is created. With the note 3501484 the temporary delivery carries also the information on original delivery items in buffer table AT_LIPS_ORG. This original delivery items in buffer table AT_LIPS_ORG should be sorted out during creation of temporary delivery.

Solution

Please implement following correction instructions.
