Symptom

This KBA explains how to display rate and currency column after running wage type reporter. "Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental." 

Environment

  • Human Capital Management (HCM)
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP SuccessFactors Employee Central Payroll
  • SAP S/4 HANA

Resolution

Pre-requisite:

  • For Employee central Payroll system, SAP Note 3321150 has been implemented in the system. 
  • For Non-Employee central Payroll (e.g. SAP ERP, S/4HANA), following SAP Notes have been implemented in the system. 
    2085137 - H99CWTR0 - enhancement for Rate field
    2660399 - Improvement Request: Wage Type Reporter - Enhanced Rate Field (RTE) functionality

Steps to Display Rate and Currency Columns:

  1. Run t-code PC00_M99_CWTR.
  2. Tick the checkbox ‘show rate’ in other selections section.
  3. Execute it and rate plus currency column will be displayed successfully.

Keywords

PC00_M99_CWTR, H99CWTR0, Rate, Currency, Column
