Symptom

In the Timesheet Analytics - Activities application:

  • The Manager Personnel Number field was visible, though it was not supposed to be shown.

  • The chart was not interacting with the table on the detail page (no synchronization or filtering behavior).

  • The default filter on calendar date was not applied during application load.

Other Terms

F7189, Timesheet Analytics, Manager PERNR, Detail Page chart, Default Filter.

Reason and Prerequisites

  • The visibility of Manager Personnel Number field was not handled correctly.

  • The chart interaction was not explicitly handled.

  • A new feature has been introduced to set the default filter for calendar date. Refer SAP Note: 3719912 for implementation details.

Prerequisite: The SAP Note: 3719912 has to be implemented in the corresponding back-end system.

Solution

 Apply corresponding support pack. Alternatively, implement the correction instructions attached to the SAP Note.
