Symptom

  • The import of transport request fails with Return Code = 8 after implementing SAP Note 3562689.
  • Errors during import include a syntax error in program CL_I_D_BPWD_FIELD_CONTROL=====CP Include CL_I_D_BPWD_FIELD_CONTROL=====CM001.
  • The data object 'SC_NODE_ATTRIBUTE-I_BUSINESSPARTNERTP' does not have a component called 'INTERNATIO...'

Environment

  • SAP ERP
  • SAP S/4HANA
  • SAP ERP Central Component
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA Cloud Private Edition
     

Reproducing the Issue

  1. Implement SAP Note 3562689.
  2. Later import the transport request.
  3. Observe the syntax error and Return Code = 8 during the import process. 

Cause

There is inconsistencies between the ABAP code delivered by SAP Note 3562689 and the DDIC structure SC_NODE_ATTRIBUTE, particularly the field I_BUSINESSPARTNERTP. 

Resolution

1. Implement SAP Note 3293713 - Resolution for International Location Number Fields 

2. The manual corrections outlined in the above SAP Note 3293713 must be performed. 

Keywords

International Location Number, CR=8, CL_I_D_BPWD_FIELD_CONTROL=====CP, I_BUSINESSPARTNERTP
