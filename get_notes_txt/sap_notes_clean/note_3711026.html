Symptom

The system raises the error message: "FINS_ML_MIG061", and error message: "FINS_FI_MASS_DATA152".

The reset activity for migration step M10 – Migrate Material Ledger Master Data fails consistently, preventing the restart of the Finance migration monitor (FINS_MIG_STATUS).

Environment

SAP S/4HANA

Reproducing the Issue

  1. Open transaction FINS_MIG_STATUS (Start and Monitor Data Migration / migration monitor).
  2. Attempt to restart the migration sequence where step M10 is required.
  3. Execute Reset for M10 – Migrate Material Ledger Master Data.
  4. Observe the system errors:
  • FINS_ML_MIG061 ("Not all data has been deleted in package reset steps. Contact SAP support").
  • FINS_FI_MASS_DATA152 ("ML Master Data step ML_MBEW failed for work package Final RESET_ALL; try again").

Resolution

  1. Set the user parameter FML_MIG_RESET_ALL to "X" (ABAP True) for the user executing step M10 in transaction SU3 under the "Parameter" tab.
  2. Navigate to transaction FINS_MIG_STATUS, select the "Control" tab, and click the "Next Activity" button to reset and re-execute step M10.
  3. After successfully completing the migration step M10, remove the user parameter FML_MIG_RESET_ALL from the user who executed step M10 in FINS_MIG_STATUS.
