Symptom

Users want to complete the Offboarding "Employee Data Review" task automatically without any employee involvement.

Environment

SAP SuccessFactors Onboarding

Reproducing the Issue

  1. Configure an Offboarding 2.0 process that includes the "Employee Data Review" task assigned to the employee.
  2. Start an Offboarding process for a user.
  3. Observe that the "Employee Data Review" task requires employee action and does not auto complete.

Cause

Automatic completion of the Offboarding "Employee Data Review" task is not supported by the current product design.

Resolution

  1. Note that Auto completing the "Employee Data Review" task without Employee involvement is currently not supported.
  2. If this task is not required, you can configure the process variant such that the Employee Data Review step is not included, so it will not be initiated during the process.
  3. If you feel that this functionality could be enhanced to better meet your business requirements, we kindly recommend submitting your request through the Influence Portal.
    Our Product Management team will review your request, and based on feasibility, it may be considered for future releases.
  4. For more information on how to submit an idea, please refer to the following KBA:
    2090228 - How to Submit Ideas for SAP SuccessFactors Products
