Symptom

If you are not using the Croatia Fiscalization solution and it is not required for your business, you may experience unexpected behavior in billing transactions (for example, inconsistencies in the Interest Tab screen in VF03).

This issue can occur when Croatia Fiscalization–related enhancements implementations are active in the system.

Other Terms

Croatia Fiscalization, ES_SAPMV60A, /CCEE/FISC_BILLING_TAB, MV60AF0P_PBO_TABSTRIP, VF03, VF02, Billing Documents, Croatia, HR.

Reason and Prerequisites

Prerequisite note: 2600557

Multiple enhancement implementations are active for enhancement spot ES_SAPMV60A. One of these implementations belongs to the Croatia Fiscalization solution, which is delivered by SAP to support country-specific legal requirements for Croatia.

If this solution is not required in your system, the active enhancement implementation /CCEE/FISC_BILLING_TAB may interfere with standard billing processing.

Solution

This is a Consulting Note.

Implement Correction Instruction for your release.

After implemening this note enhancement implementation /CCEE/FISC_BILLING_TAB associated with enhancement spot ES_SAPMV60A will be deactivated.

Note: All changes should be validated in a test system before being applied to production.
