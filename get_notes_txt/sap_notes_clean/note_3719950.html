Symptom

  • You want general information about "HANA_License_Overview" report.
  • You want more detailed information and some deeper understanding about "HANA_License_Overview" report.
  • You want to understand how to use and when to use the report.
  • You want a centralized, single source of the available (and most important) information regarding "HANA_License_Overview" report.

Environment

SAP HANA, platform edition

Resolution

What is "HANA_License_Overview" report?

  • The report collects and displays information about the license associated with the SAP HANA database.
  • It provides additional useful information about the actual license.

When to use the report?

  • When you want to monitor a specific date and time range regarding system load (such as CPU, memory, disk, network, etc.).

Which is the master documentation of the SQL Statements Collection?

  • 1969700 - SQL Statement Collection for SAP HANA

Where to download the file(s)?

  • 1969700 - SQL Statement Collection for SAP HANA - - > Attachments

What is the structure of the "HANA_License_Overview" report file?

  • You can find the following sections in the file (comment out): [NAME], [DESCRIPTION], [SOURCE], [DETAILS AND RESTRICTIONS], [VALID FOR], [SQL COMMAND VERSION], [INVOLVED TABLES], [INPUT PARAMETERS], [OUTPUT PARAMETERS], [EXAMPLE OUTPUT].
  • You can find the SQL statement itself.

Can the report executed directly without any modification?

  • Yes, the report can copy-paste and run directly.
  • There is no /* Modification section */ available for this report.

How the report look like?

The following example shows the report itself.

-----------------------------------------------------------------------------------------------------------------------------------------------------------------
|HARDWARE_KEY|SID|INSTALL_NO|GLAS_APPLICATION_ID|PRODUCT_NAME|PRODUCT_DESCRIPTION|PRODUCT_LIMIT_GB|PRODUCT_USAGE_GB|EXPIRATION_DATE    |EXP_DAYS|PERMANENT|VALID|
-----------------------------------------------------------------------------------------------------------------------------------------------------------------
|A1234567890 |ABC|SAP-XXXXXX|000673             |SAP-HANA    |SAP HANA database  |             516|             119|2026/11/05 23:59:59|     254|TRUE     |TRUE |
-----------------------------------------------------------------------------------------------------------------------------------------------------------------

The following columns will be presented in the report.

  • HARDWARE_KEY: Hardware key
  • SID: System ID
  • INSTALL_NO: Installation number
  • PRODUCT_LIMIT_GB: License limit (GB)
  • PRODUCT_USAGE_GB: Used memory peak (GB)
  • EXPIRATION_DATE: License expiration date
  • EXP_DAYS: Number of days until license expiration
  • PERMANENT: TRUE for permanent license, otherwise FALSE
  • VALID: TRUE/FALSE
