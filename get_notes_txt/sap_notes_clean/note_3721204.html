Symptom

In the treaty, incorrect entries are displayed in the history for the "Additional Treaty Data" tab page.
When you change an entry on the "Additional Treaty Data" tab page, the system generates new history entries for all entries.

Other Terms

WoCo
Workers Compensation

Reason and Prerequisites

Prerequisite:
- "Additional Treaty Data" is defined for a treaty

Causes:
- Incorrect saving of creation and change data 
- If changed, all entries are marked as changed

Solution

History:

February 26, 2026: New

Troubleshooting Description:
The creation and change times are stored in UTC format.
Only one entry is created in the history table if a change has been made to the data record.
