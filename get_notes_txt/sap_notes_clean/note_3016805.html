Symptom

During execution of method CL_ABAP_STRUCTDESCR=>GET or CL_ABAP_STRUCTDESCR=>CREATE or at implicit dynamic structure type creation at certain ABAP statements, a runtime error RUNT_INTERNAL_ERROR occurs. The termination is triggered in function "ab_RollLoadRudiToAddress" of the SAP kernel.

Other Terms

RTTC, structure type creation

Reason and Prerequisites

The error is related to the ABAP type garbage collection. It can only occur under rare circumstances after several executions of the type garbage collection, where many structure types are created and ALL of these structure types are garbage collected. The error cannot occur, when at least one valid reference to a dynamically created instance of CL_ABAP_STRUCTDESCR always remains.

Solution

Apply a kernel patch for disp+work. For the minimum patch level for the relevant release, see the "Support Packages & Patches" section.
