Symptom

You execute RF transaction "Repack HU Item Manually" and by mistake scan source HU as destination HU. The system does not display any warning and WT for repack within the same HU is created and confirmed.

Other Terms

manual repack, repack, Repack HU Item Manually, RF

Reason and Prerequisites

Due to code error

Solution

New message has been created to warn the user the source and destination HUs are the same. The action can be confirmed or the user can go back and change destination HU. Implement the note and execute manual instructions
