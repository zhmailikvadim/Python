Symptom

  • When running transaction MC46, the export to Excel option/button is not available in the list menu or application function bar.
  • No error message is displayed.

Environment

  • Sales And Distribution (SD)
  • Logistics Execution (LE)
  • SAP S/4HANA
  • SAP S/4HANA Cloud 
  • SAP ERP Central Component
  • SAP ERP
  • SAP Enhancement package for SAP ERP
  • SAP Enhancement package for SAP ERP, version for SAP HANA

Reproducing the Issue

  1. Run transaction MC46.
  2. Attempt to export the report to Excel via the list menu.
  3. Observe that the export to Excel option is not available.

Cause

The export to Excel function is not provided by the application design for this report; therefore, the button/option is not displayed.

Resolution

  1. In the MC46 result screen, open the top menu and choose System -> List -> Save -> Save.
  2. In the save dialog, select the option “Text with Tabs.”
  3. Save the list to a file with the extension “.xls”.
  4. Open the saved file in Excel.
  5. Note: The saved file is a tab-delimited text file (not a native Excel workbook), but it can be opened and analyzed in Excel.
