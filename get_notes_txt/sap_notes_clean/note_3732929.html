Environment

  • SAP Process Control 12.0
  • SAP Process Control 12.0 for S/4 Hana

Cause

Dump is occurring due to inconsistent data of a risk 

Resolution

  1. There is an inconsistency in DB - it is concerning RISK object ID's
  2. Follow the below fix it by report GRFN_HR_DATA_CONSIST (select the entity RISK).
  3. Execute T-Code: SE38
  4. Enter report : GRFN_HR_DATA_CONSIST 
  5. Select Risk ID's
  6. Click on "fix inconsistency's"

Keywords

Risk, Object ID, SE38, GRFN_HR_DATA_CONSIST, CL_GRFN_UTIL_GENERAL==========CP, GRFN_REPLACEMENT_MASS_ACTIVATE.
