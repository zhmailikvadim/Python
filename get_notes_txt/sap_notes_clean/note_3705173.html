Symptom

Fiori app 'Record results for inspection points' (app ID: F2689) is used for results recording.

When results are attempted to be saved, below error is raised:

'Wrong value for field Valid values'

(Message nr: Q5042)

Environment

  • SAP ERP Quality Management (QM)
  • SAP S/4HANA
  • SAP S/4HANA Cloud, private edition

Cause

A 5 digit number (greater than 9999) had been maintained in field Inspected for one inspection characteristic.

In fiori app 'Record results for inspection points', BAPI_INSPOPER_RECORDRESULTS is used. And inside this BAPI, for field Inspected, value is truncated when it exceeds 4 digits (9999) due to a technical limitation. For further information, read below KBA:

2299245 - Digits of field SAMPLE_RESULTS-VALID_VALS have been cut using BAPI BAPI_INSPOPER_RECORDRESULTS

Resolution

When recording value greater than 9999 in field Inspected during results recording, use visually harmonized app 'Record Inspection Results - For Inspection point' (app ID: QE11) or 'Record Inspection Results - Hierarchical Worklist' (app ID: QE51N). 

Record Inspection Results - For Inspection Point

Record Inspection Results - Hierarchical Worklist
