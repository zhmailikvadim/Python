Symptom

 The syntax addition INDICATORS SET may lead to long execution times.

Other Terms

KERNELCOR 

Reason and Prerequisites

The problem can occur if the table to be updated has the column _DATAAGING.

Solution

If the affected UPDATE command has a static FROM clause, the affected program must be regenerated so that the change is used.


This correction was originally delivered with the kernel patch level specified in the "Support Package Patches" section of this SAP Note. All later kernel patch levels also contain this correction. Note that only the latest kernel patch is available for download. It contains all previously delivered corrections. This means that you can always download and apply the latest kernel patch in order to implement the corrections.

If no kernel patch level is entered in the "Support Package Patches" section, the patch is not yet available. The entry appears with the delivery of the corrections. Check this SAP Note to find out whether new kernel patch levels have been entered.

These corrections are delivered with the following SAP kernel archives:

  • Hotfix - file dw.sar
  • SP Stack Kernel - files SAPEXE.SAR and SAPEXEDB.SAR

Follow our recommendations for selecting and applying the kernel patch:

  1. Import the latest SP Stack Kernel if it contains the corrections. For a list of current SP stack kernels, see SAP Note ﻿2083594 (SAP Kernel Versions and SAP Kernel Patch Levels).
  2. Otherwise, the corrections will be provided with the next SP Stack Kernel. Apply the hotfix only if you need the corrections quickly.
  3. The correction may not be available for kernel releases that are no longer maintained. Perform an upgrade to the relevant downward-compatible kernel (DCK); see SAP Note 2083594 (SAP Kernel Versions and SAP Kernel SAP Patch Levels).
  4. Check the regression note for the target patch level before applying the patch. For detailed information, see SAP Note 1802333 (Finding information about regressions in the SAP kernel).﻿﻿
  5. For instructions on how to download and apply kernel patches, see SAP Note 19466 (Downloading SAP Kernel Patches).
