Symptom

You have implemented SAP Note 3633191 - YE: BAdI Methods to fill box 14 in T4A-NR form and block the generation of the T4A form and the documentation for the following Business Add-Ins (BAdI) is missing or incomplete:

  • BAdI Definition for the Canada Year End Report (HRPAYCA_B_YE_BOX_ADJUSTMENT)
  • BAdI for Rejecting Employees on Year End (HRPAYCA_B_YE_REJECT_EMPLOYEE)

Note

The implementation of this SAP Note is optional and won't cause any changes on the feature itself.

Reason and Prerequisites

Reason

Documentation update missing.

Prerequisites

You have applied SAP Note 3633191 - YE: BAdI Methods to fill box 14 in T4A-NR form and block the generation of the T4A form.

The following Support Package level is the minimum prerequisite:

Release Support Package
S4HCM 102 HR SP 00
S4HCM 101 HR SP 06
S4HCM 100 HR SP 19
SAP_HR 6.08 HR SP D0
SAP_HR 6.04 (EhP6 and EhP7) HR SP K2

For more information, access the SAP ERP Human Capital Management - Support Package Schedules (SAP_HR) or SAP S/4HANA On Premise Human Capital Management (S4HCM).

When you install an SAP Note, some prerequisites might be required. The prerequisites are SAP Notes that have already been released and that change the same context of the current one. Then, even if these prerequisites are not necessarily related to the symptom of this SAP Note, you must implement them due to technical reasons.

Solution

From now on, you can refer to the documentation of the following BAdIs:

  • HRPAYCA_B_YE_BOX_ADJUSTMENT
  • HRPAYCA_B_YE_REJECT_EMPLOYEE

You can also access them in the Implementation Guide for Payroll Canada, under Reporting and Statistics -> Year End Reporting.

Note

The solution delivered by this SAP Note will be available once you install the corresponding Support Package.
