Symptom

Unable to set the Minimum Value and Maximum Value on the X-axis when using a Scatter plot chart in a Web Intelligence document. These axis-scaling options are available in Business Intelligence 4.2 but are missing in Business Intelligence 4.3 and BI 2025.

Other Terms

webi, webinteligence, max, min, scale, properties

Reason and Prerequisites

Environment:

  • SAP BusinessObjects Business Intelligence Platform 4.3 Support Package 04
  • SAP BusinessObjects Business Intelligence platform 2025 Patch 08
  • Web Intelligence

Step to Reproduce:

  1. Log in to BI Launchpad.
  2. Create a Web Intelligence document based on the eFashion universe.
  3. Build and run a query that includes two dimensions and two measures.
  4. Insert a Scatter Plot chart into the report.
  5. In the Data Assignment panel, assign one object to each required field of the scatter plot (eg: Axis 1:Quantity Sold; Axis 2: Sales Revenue; Shape: Year)
  6. Open the chart formatting panel via Show Report Element Format.
  7. Select Value Axis → Style Settings → Scaling.
    1. >> Minimum and Maximum Value settings are missing.

Cause: 

  • This issue is caused by a defect in the product. 

Solution

This issue is fixed in the patches listed in the "Support Packages & Patches" section below.
Due to different internal processes, the "Support Packages & Patches" section of the SAP Note will be populated as below:

  • for fixes delivered in "Support Packages": immediately (even if the "Support Package" is planned to be released several months later)
  • for fixes delivered in "Patches": ONLY once the "Patch" is released AND available on SMP (not before)
