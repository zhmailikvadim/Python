Symptom

After upgrading to SAP S/4HANA Cloud Edition 2602, the TAX/VAT number duplication check is not working correctly in the "Manage Business Partner" app.

The system allows the creation of multiple Business Partners with the same TAX/VAT number, although duplicate TAX/VAT numbers should be prevented by system validation.

This behavior is observed only in the Manage Business Partner Fiori App after the upgrade.

Workaround

Until the fix is delivered, customers can use the "Maintain Business Partner" app to create Business Partners.
The VAT duplication check functions correctly in this app and prevents duplicate VAT number creation.

The issue is not production-critical due to the availability of this workaround.

Cause

Program Error.

Resolution

In general, SAP recommends that you install a solution by applying a Support Package. However, if you need to install a solution earlier, use the Note Assistant to implement the correction instruction from Note 3714829 - Duplicate BP Tax Number check is not working for BP Fiori App'.
