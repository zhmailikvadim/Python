Symptom

Transformation transport request ended with below errors:

  • Request <TR> (time stamp: 00000000000001)
    Message no. RSO886
  • Error R7 105 during after import handling of objects with type(s) TRFN
    Message no. RSO857
  • The object name is not allowed to be empty
    Message no. R7105

Also, The report RSTRAN_ROUT_RSFO_CHECK has ended with below errors:

TRANID =  OBJVERS = A
package of transformation =<Package_name>
Global with ID = <> is missing
Global2 with ID = <> is missing
No auto correction available, refer to note 1419291

Environment

SAP NetWeaver all versions

Reproducing the Issue

Transport TRFN

Cause

Missing InfoObjects in the source or target ADSOs of the transported transformation.

Resolution

When major changes are done to the TRFNs, it is recommended to transport the source and target objects with the transformation to avoid inconsistencies. 

See Also

1419291 - Cleaning up technical inconsistencies of transformations
