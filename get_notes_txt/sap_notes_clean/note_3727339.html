Symptom

When you add a document item for an import declaration prior to goods receipt, you cannot restrict by purchasing document. Therefore, too much data is read and there is a risk that processing is terminated with runtime error TIME_OUT.

Other Terms

/SAPSLL/SAPLCUHD_LOAD_IFACE, Add Item, /SAPSLL/CUS_SGL02I, /SAPSLL/CIBD_GET_DATA

Reason and Prerequisites

This problem is caused by a program error.

Solution

Implement the attached correction instructions.

After you have implemented the correction instructions, you can filter for specific purchasing documents or set a time restriction when adding a document item in an intermediate dialog. By default, the system restricts to purchasing documents that were created or changed in the last 30 days.
