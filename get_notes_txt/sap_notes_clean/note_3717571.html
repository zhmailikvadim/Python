Symptom

- An issue occurs when attempting to extract custom field values from the "Ariba Network Purchase Orders Buyer API."
- The GET queries provided in the description do not retrieve the custom field values.
- There is currently no parameter or field available in the API to retrieve custom fields.

Cause

The API does not support retrieving custom fields.

Resolution

At present, the Ariba Network Purchase Orders Buyer API does not support any parameters or custom fields. This request is an improvement request.
