Symptom

When clicking on the File Name tab of Attachments in Requisition, attachments are displayed based on the upload order (ascending/descending) rather than alphabetical order of the file name, is it possible to sort attachments by alphabetical order of file name in requisition?

Cause

The File Name in the Attachments is displayed in the order the files were uploaded. This is an expected behavior.

Resolution

For design change, raise an Improvement Request with our product team. For further information refer to: How do I create an improvement or enhancement request?
