Symptom

1. No bank transfer (wage types M840/M841) is generated for the Employer pension contribution in cases where the main personnel number is inactive and the associated reference number is relevant to the Employer pension contribution.
2. There is the typo in ABAP. Wage type is filled in the field ABART.
3. Additional symptom (WT /288, /289 is not created on the main personal number) belonging to the first situation described in the symptom 2.

Other Terms

HRCZ, RPCALCT0, RPCALCTM

Reason and Prerequisites

 This SAP Note is only relevant for the Czech Republic.
