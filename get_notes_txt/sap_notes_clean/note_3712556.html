Symptom

  • The "Download Preview Users" report generated from the Assignment Profile Rules tab does not include all fields that are included in the user profile

Environment

  • SAP SucessFactors Learning

Reproducing the Issue

  1. Navigate to Learning Administration → Assignment Profiles → Rules tab.
  2. Select the "Download Preview Users" option.
  3. Generate the CSV file.
  4. Compare the generated file with the User Details page and observe that not all fields are included

Cause

Resolution

  • The 'Download Preview users' only pulls from specific fields and does not include all user related fields.
  • The fields included in the download are:
    User ID    
    User First Name    
    User Middle Name    
    User Last Name    
    User Display Name    
    Organization ID    
    Organization    
    Employment Status    
    Employee Type    
    Job Location    
    Job Classification    
    Security Domain ID    
    Security Domain Description    
    Active    
    Address    
    City    
    State    
    Postal Code    
    Country    Manager 
    First Name    
    Manager Middle Name    
    Manager Last Name    
    Manager Display Name    
    Hire Date    
    Termination Date    
    Email Address    
    Has Access    
    Locked    
    Region ID
    Role ID
