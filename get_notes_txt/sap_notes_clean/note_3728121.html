Symptom

The (FAIR-based) process "DL Automatic Assignment" (/IBS/CAB_AUTO_ASSIGN) does not generate a work item for a manual check for corrected CFS investor accounts for which the "Rule Exclusion" indicator is set and there is a balance change.

Reason and Prerequisites

With the change introduced with SAP Note 3724281, the parameter type intended for controlling the work item creation for incorrect CFS investor accounts is used twice in the business function "Select CF Investor Accounts" (/IBS/CAB_SEL_INVES_ACCOUNT).

Solution

The implementation /IBS/CL_CAB_SEL_INVES_ACCOUNT of the above-mentioned BF is corrected.
