Symptom

  • The date columns in the Web Intelligence report are displaying as single digits instead of the expected yyyymm format.
  • This issue persists both when creating the report directly from HANA and when creating new reports from the universe.

Other Terms

webi, date, format, universe

Reason and Prerequisites

Environment:

  • SAP BusinessObjects Business Intelligence platform 4.3 Service Pack SP04 Patch04
  • Windows
  • Web Intelligence (Webi)
  • SAP HANA Cloud JDBC

Steps To Reproduce:

  1. Open the BI Launch Pad.
  2. Create a new Web Intelligence report using the relevant universe.
  3. Add the object to the report and click run and observe that they display as single digits instead of the expected yyyymm format.
  4. Alternatively, access the data directly from HANA and attempt to create a new report, noting the same single-digit display for the date columns.
