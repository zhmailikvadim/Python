Symptom

In the SAP Fiori app Manage Preference Properties – Customers/Suppliers, you see several data records for the same customer and the same administrative unit. On the object page, you see the note “Created by Migration Report”.

Other Terms

Migration, /sapsll/precup, /sapsll/prespp, /sapsll/pre_mig_partner_prop

Reason and Prerequisites

The preference properties for customers/suppliers were created using the migration program.
Customers/vendors exist several times in the table /SAPSLL/PNTBP with different "Logical System Group".
