Symptom

  • You use CG02 Web GUI for HTML or Manage Substances Fiori App (App ID: CG02);
  • When attempting to execute an Expert Rule for Transfer of Content using EH&S OCC the following error occurs:

    Error when calling Secondary Data Determination Using Expert
    Error message: "Error when opening an RFC connection (FUNCTION: 'exec_sapgui' cannot open SAPGUI)".
  • The error does not occur when performing the same steps on SAP GUI.

Environment

  • SAP ERP
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Open Manage Substances Fiori App (App ID: CG02) or CG02 in Web GUI for HTML;
  2. Click on Specification search;
  3. Input the specification number and click on Hit list;
  4. Click on user to change to Edit mode;
  5. Choose the specification -> Menu -> Utilities -> Secondary data -> Determination. A pop-up will appear;
  6. Choose ERC_PS or ERC_DG;
  7. A pop-up with the error appears:
  • Error when calling secondary data determination using EHS Expert.
  • Error when opening an RFC connection (FUNCTION: 'exec_sapgui' cannot open SAPGUI).

Cause

For technical Reasons, is not possible to launch EH&S OCC through SAP GUI for HTML or Manage Substances Fiori App (App ID: CG02), as explained in SAP Note 3282375.

Resolution

There are two alternatives:

  1. Start the Open Content Connector (OCC) from the specification workbench (CG02) in the SAP GUI (as mentioned in SAP Note 3282375);
  2. Load data via the OCC directly, as explained in SAP Note 3501690.

See Also

  • SAP Note 3282375 - Open Content Connector (OCC) cannot be called using SAP GUI for HTML.
  • SAP Note 3501690 - EH&S OCC: Open Content Connector without SAP GUI installation.
