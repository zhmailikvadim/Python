Symptom

User is unable to see an option to add Adjustment Type on PCard Charge Reconciliation. The user needs to add additional Shipping/Taxes during the charge reconciliation.

Cause

The Adjustment Type option on PCard Charge Reconciliation (PCCR) document will be appearing only when there is a POAmountVariance exception on the PCCR. If there is no POAmountVariance exception on PCCR, the button to add Adjustment Type will be hidden.

Resolution

While loading the PCard Charges, the Charge amount should be greater than the PCO total amount. Then only the POAmountVariance exception will be triggered and the user can add the Adjustment Type. 

The POAmountVariance exception for PCCR can be configured through the import task Import Charge Exception Type Validation Data.
