Symptom

  • Input controls based on Variables throw warning if we click on Input Control reset or save, close and open back the Document
    “The list of values has changed. The selected values could be obsolete”
  • Issue is only observed from HTML Mode and works fine in Java Applet and Rich Client

Other Terms

webi,webintelligence, DHTML

Reason and Prerequisites

Environment:

  • SAP BusinessObjects Business Intelligence Platform 4.2 Support Package 08
  • Server OS: Windows server 2016
  • Web Intelligence (HTML) 

Steps To Reproduce:

  1. Logon to BI Launchpad and launch Web Intelligence in HTML Mode
  2. Create a Web Intelligence Document on E-Fashion universe using Year State and Sales Revenue
  3. Create a new Dimension variable and declare State object in it
  4. Add a New Input Control using above created variable
  5. Select a value in it and save the Document
  6. Click on Input Control Reset or close and open back the Document and observe below warning on top of Input Control
  7. The list of values has changed. The selected values could be obsolete

Cause:

  • This issue is caused by a defect in the product.   

Solution

  • This issue is fixed in the patches listed in the "Support Packages & Patches" section below.
    Due to different internal processes, the "Support Packages & Patches" section of the SAP Note will be populated as below:
                    - for fixes delivered in "Support Packages": immediately (even if the "Support Package" is planned to be released several months later)
                    - for fixes delivered in "Patches":  ONLY once the "Patch" is released AND available on SMP (not before)
    For Business Intelligence Platform maintenance schedule and strategy see the Knowledge Base Article 2144559 in References section.
