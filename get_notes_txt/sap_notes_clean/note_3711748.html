Symptom

The long text of the IMG node 'General Settings for Subscription Transactions' was missing the description of the following flags:

  • Create CI Billing Plan Item for Standalone Selling Price
  • Set RAR Transfer Record for Provider Contract Item Based on CI Billing Plan

Other Terms

CI Bill Plan Item, Standalone Selling Price, Provider Contract, Subscription Order Management

Reason and Prerequisites

Missing long text in IMG node

Solution

The long text in the IMG node has been updated with the text highlighted in the attached pdf. The updated documentation will be available in SAP S/4HANA release 2023 SPS06 and 2025 FPS02.

The 'General Settings for Subscription Transactions' IMG node is available in the Customizing path 'SAP Customizing Implementation Guide > Service > Transactions > Settings for Subscription Transactions > General Settings for Subscription Transactions'.
