Symptom

It is required to create a processing rules with same fields /attributes in app F3555.
In this app the conditions are defined in which the processing rules should be applied and the actions that the processing rules should perform with the bank statement items.

Environment

Financial Accounting (FI)
SAP Fiori for SAP ERP
SAP S/4HANA
SAP Fiori for SAP S/4HANA Finance
SAP S/4HANA Cloud Private Edition

Cause

The logic in the processing rules conditions is as follows: conditions with different fields/attributes are treated as ’AND’ conditions with same fields/attributes are treated as ‘OR’.

Resolution

Examples of a bank statement item with different values in its memo line

Example 1 – a bank statement item with two values VALUE1 and VALUE2

Example 2 – a bank statement item with two values VALUE1 and VALUE3

Example 3 – a bank statement item with only one value VALUE1

 
A rule condition that uses different values with “OR”

This processing rule will process any bank statement item with VLAUE1 or VALUE3 in its memo line. A processing rule with these conditions will process the bank statement items from example 1, example 2, and example 3.

A rule condition that uses different values with “AND”

This processing rule will process only a bank statement item that has both values VALUE1 and VALUE3 in its memo line. A processing rule with this condition will process only a bank statement item from example 2.

This processing rule will process only a bank statement item that has both values VALUE1 and VALUE2 in its memo line. A processing rule with this condition will process only a bank statement from example 1.
