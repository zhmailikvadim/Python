Symptom

 You want to verify whether vulnerability CVE-2020-26836 will affect SAP S/4HANA system or not.

Environment

  • SAP S/4HANA system
  • All SAP Products except SAP Solution Manager 

Cause

The vulnerability CVE-2020-26836 is an Open Redirect issue that affects only SAP Solution Manager (Trace Analysis), version 7.2

The corresponding SAP Security Note 2938650 applies exclusively to systems where the ST (Solution Tools / Solution Manager) component is installed.

Resolution

CVE-2020-26836 does not apply as vulnerable to other productsm as it only affects SAP Solution Manager (Trace Analysis).

See Also

2938650 - [CVE-2020-26836] Open Redirect in SAP Solution Manager (Trace Analysis)

Keywords

CVE-2020-26836, open redirect, note 2938650, SNOTE not applicable, ST component missing, Solution Manager only, trace analysis, S/4HANA not affected, security vulnerability.
