Symptom

  • A Buyer user requests to change their assigned Supervisor.
  • A Supervisor needs to be updated due to organizational changes.

Cause

The user profile still lists the previous Supervisor as the Primary Supervisor. Until this assignment is updated, the system will continue routing notifications and approvals accordingly.

Resolution

Users with the Administrator role can update the Primary Supervisor by following these steps:

  1. Sign in to SAP Fieldglass
  2. Go to User > Locate and select the user profile
  3. Click Edit
  4. Update the Primary Supervisor field
  5. Click Save

Verify that the user performing the change has the Administrator role and the required permissions.

If the user does not have Administrator permissions or cannot edit user profiles, they should contact their Program Management Office (PMO) or system administrator for assistance.

Note: Changes to the Primary Supervisor may affect approval workflows and system notifications.
