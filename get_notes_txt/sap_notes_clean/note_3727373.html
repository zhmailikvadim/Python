Symptom

System is not updating the ERP Push error received from the ERP system for Advance Payments in the SAP Ariba Buying UI.

Cause

The Id field is incorrectly sent from the ERP system to Ariba, since Ariba uses this field to uniquely identify the correct Advance Payment record, providing an incorrect value prevents the system from applying or displaying the ERP Push error. The Id field must contain the Advance Payment Number.

Resolution

Ensure the actual Advance Payment Number is sent in the Id field. Once the correct ID and error fields are provided, Ariba will process the update successfully.

Keywords

Advance Payment Number, ID, ERP Push error, Ariba buying, ErrorMessage , ErrorSAPId
