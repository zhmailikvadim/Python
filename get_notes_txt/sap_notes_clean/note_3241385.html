Symptom

We would like to have all elements sorted as they are in the process. But the elements are scattered all over the process documentation.

The order on reports currently seems to be dictated by the order they are created on the flow, as opposed to the order they correctly follow in the process.

Resolution

You should check what option has been chosen in your process documentation template. Open the template and go to elements. Change there 'Sorting' to 'controlflow', if the order should correspond to the process flow:
