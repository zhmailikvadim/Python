Symptom

After the recent system upgrade, the report FINS_PLAN_SQL_LOG_CLEAN is no longer available in the system.

A replacement Report is required.

Environment

  • Controlling (CO)
  • SAP S/4HANA
  • SAP S/4HANA Finance
  • SAP S/4HANA Cloud Private Edition
  • SAP Fiori for SAP S/4HANA

Cause

This report was removed since OP2021.

Resolution

Please check the report R_FINS_PLAN_CLEAN_TEMP_DATA. This report uses the same class for cache cleaning as report R_FINS_PLAN_SQL_LOG_CLEAN.

Keywords

FINS_PLAN_SQL_LOG_CLEAN, not available, replacement, upgrade
