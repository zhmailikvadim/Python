Symptom

This SAP Note provides you with the relevant system changes to address to the new form 220 (also knows as Certificado de Ingresos y Retenciones) for 2025.

Other Terms

When you install an SAP Note, some prerequisites might be required. The prerequisites are SAP Notes that have already been released and that change the same context of the current one. Then, even if these prerequisites are not necessarily related to the symptom of this SAP Note, you must implement them due to technical reasons.

Solution

From now on, the Certificado de Ingresos y Retenciones (HCOCERT0) report will be able to generate the new "CIR" form for the fiscal year 2025. For that pupose, this SAP Note delivers the new SmartForm HRCO_CIR_2025.
