Symptom

• The system progress hangs when the profitability segment button is clicked via the Manage Journal Entries app.  

Environment

• S/4 HANA On Premise or Private Cloud  
• Transaction Apps  

Reproducing the Issue

1. Open the Manage Journal Entries app.  
2. Click on the profitability segment button.  
3. Observe that the system progress hangs.  

Cause

OData service FCO_PROF_SEGMENT_SRV is not activated  

Resolution

1. Activate OData service FCO_PROF_SEGMENT_SRV using transaction /n/IWFND/MAINT_SERVICE 

Keywords

system progress hangs, profitability segment, Manage Journal Entries app, OData service, FCO_PROF_SEGMENT_SRV, transaction, activation, S/4 HANA, On Premise, Private Cloud, journal entries, app issue, system freeze, troubleshooting, performance issue
