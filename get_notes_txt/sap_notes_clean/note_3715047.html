Symptom

This SAP Note provides you with essential information about SAP Ariba Sourcing and Procurement for Public Sector in S/4HANA Release OP2025 FPS1.

Other Terms

SAP Ariba Sourcing and Procurement for Public Sector in S/4HANA, S4PPS, PPS, S/4HANA 2025 FPS1

Reason and Prerequisites

 You want to get additional information about SAP Ariba Sourcing and Procurement for Public Sector in S/4HANA as part of product version SAP S/4HANA 2025.

Solution

  1.  As a first step kindly refer to the central release notes 3493301 and 3658070 to get information on SAP S/4HANA 2025
  2. Please refer to the product documentation for details on the features offered by SAP Ariba Sourcing and Procurement for Public Sector
  3. Please refer to this link to get the list of all Front-End Business Roles which are relevant for SAP Ariba Sourcing and Procurement for Public Sector
  4. The authorization objects relevant for SAP Ariba Sourcing and Procurement for Public Sector are described here
  5. To start using the solution, the Business Functions described here need to be activated in the system
  6. Please refer to this section for details on the various Fiori Apps for SAP Ariba Sourcing and Procurement for Public Sector
  7. Please follow the steps in this section to publish the required ODATA v4 Service Groups
  8. A guide on the configuration aspects of the solution are provided in an attachment of this note [S4HANA_OP2025_FPS1_Configuration_Guide_PPS.zip]

SICF SERVICE ACTIVATION

Internet Communication Framework (ICF) services must be activated to enable proper functionality of SAP Fiori applications, OData services, and web-based interfaces. All ICF services are initially inactive for security reasons and require manual activation based on your implementation requirements.

Functionality SICF Path Comments
Core Services /default_host/sap/bc/ui2/flp SAP Fiori Launchpad core functionality
  /default_host/sap/bc/ui2/start_up Launchpad startup services
  /default_host/sap/public/bc/ui2 Public UI2 resources
  /default_host/sap/public/bc/ui5_ui5 Public UI5 resources
Integration Services /default_host/sap/opu/odata Standard OData service endpoint with sub-nodes
  /default_host/sap/public/opu Public OData resources including MIME repository access
