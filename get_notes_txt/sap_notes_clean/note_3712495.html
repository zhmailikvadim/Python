Symptom

  • The basic search bar in the IAM Information System returns "No results found" message.
  • "No results found" message is returned unless the exact, case-sensitive application name is entered during application search in IAM app search bar.

Environment

  • SAP S/4HANA Cloud Public Edition
  • SAP S/4HANA Cloud Public Edition 2602

Reproducing the Issue

  1. Access the "IAM Information System" app
  2. Maintain IAM App as main entity
  3. Search for application in basic search bar or in IAM App search bar in lower case
  4. No results found message

Resolution

The search functionality for the IAM App tabs currently supports searching with specific fields using the corresponding column filters.

The filter field for the specific columns works only with the exact name. However, using wildcard character "*" returns a list of results that match the prefix of the entered value.

For example:

To search for a specific application such as Manage Sales Orders, it is recommended to enter the exact full application name (like Manage Sales Orders) in the IAM App search field or " Manage* " will display a list of applications whose names starts with Manage.
