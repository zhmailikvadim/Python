Symptom

This note delivers fix to the following issues

1. End of Service not being paid if the employee is terminated on the 1st day of the payroll period

2. End of service is not being paid if the employee is terminated in the past payroll month.

3. EOS Unused Provision amount is not generated if the employee is terminated in the past payroll month

Other Terms

End of service, EOS, SA91, SATER, HSACALC0, SAEON

Reason and Prerequisites

This note is relevant only for Kingdom of Saudi Arabia.

Version 1: Pilot Release - End of Service not being paid if the employee is terminated on the 1st day of the payroll period.

Version 2: Pilot Release - End of service is not being paid if the employee is terminated in the past payroll month.

Version 3: Removed the correction instruction 694236 and merged the changes with correction instruction 733863.

Version 4: Added the corrections for EOS Unused Provision amount is not generated if the employee is terminated in the past payroll month

Version 5: Normal release to the customers

Solution

Code changes have been performed to correct the behaviour. As a pre requisite follow the manual corrections SAP_Note_2885672_-_Manual_Implementation_Steps.pdf attached to the note and apply the attached correction instructions.
