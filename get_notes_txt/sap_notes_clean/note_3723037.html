Symptom

************************************************************

WARNING: This is a CA-FL (Field Logistics) specific note. If you DON'T have

Field Logistics activated in your system, this note does not apply to you.

************************************************************

In the Manage Logistics application, the Search function for the "Purchase Order/Item" field does not filter correctly for rental items and third-party items. Additionally, the filter appears twice in the filters section.

Other Terms

CA-FL, CA-FL-GR, CA-FL-SRV, Field Logistics, Manage Logistics (F5082)

SAP S/4HANA  OP 2023 FPS02, OP 2023 FPS03, OP 2023 SPS04, OP 2023 SPS05

Reason and Prerequisites

Program Error

Solution

As part of the solution, now the search function works as expected for rental/third-party items for the Purchase Order/Item field. It also appears once in the filters section.

Kindly implement the attached correction instructions or install the respective support package.
