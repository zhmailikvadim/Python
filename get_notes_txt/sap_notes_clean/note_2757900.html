Symptom

Transaction: O3UH_OUT_CDEX

While generating the CDEX outbound file, error is encountered due to missing SAP PRA product code 40G with CDEX product code 10 (Oil of undetermined origin).

Other Terms

Related Keywords:

  • General keywords: Add-on, IS-Oil, Industry Solution, Oil and Gas Upstream, PRA, Production and Revenue Accounting
  • Problem-specific key words: IS-OIL-PRA-REV-CW, Check Write, CDEX Outbound file, Switch BC SET, OI_PRA_OIUH_CDEX_PROD_T, OIUH_CDEX_PROD, PRA Product Code 40G, CDEX Product Code 10

Reason and Prerequisites

Mapping between PRA Product code and CDEX product code is missing in table OIUH_CDEX_PROD.

Solution

Apply the manual activity steps to create the table entry.
