Symptom

 You are using the Run Statutory Reports app to generate the SAF-T file for Luxembourg and face the issue that the <SupplierID> section of the file has incorrect values for foreign suppliers.

Other Terms

Luxembourg, LU, SAF-T, Tax Reporting, Audit, Standard Audit File, FAIA, LU_SAFT, Run Compliance Reports, Run Statutory Reports app, SRF_AUTHORING, P_LU_SAFTJournalItemBP

Reason and Prerequisites

This SAP Note is relevant only for Luxembourg.

Solution

Import the Support Package specified for your release or implement the attached correction instruction.

As a general rule, SAP recommends that you implement the Support Package specified for your release. However, if you need to install a solution earlier, use the SAP Note Assistant to implement the correction instruction. You can find more information about the SAP Note Assistant in SAP Support Portal, under SAP Note Assistant.

To implement the solution, you must carry out manual activities and implement the correction instructions that are valid for your release.

You can try to activate the objects contained in this SAP Note by executing the reports below:

  • For DDLS objects, execute the RUTDDLSACTSNOTE report according to the steps available in SAP Note 2607137.
  • For DCLS objects, execute the R_ACMTOOL_DELETE_INACTIVE_V report according to the steps available in SAP Note 2691812.
