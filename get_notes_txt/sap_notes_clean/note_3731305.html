Symptom

  • Custom MDF card fails to display the field values properly on the "View All" interface.
  • View All Button Not Displaying Content in Latest People Profile

Environment

  • SAP SuccessFactors HCM Suite
  • Metadata Framework

Reproducing the Issue

  1. Access instance with support access
  2. Navigate to an affected employee's profile
  3. Select "Miscellaneous" category
  4. Locate custom MDF card
  5. Click "View All" button
  6. No data displayed despite existing records

Cause

  • This is a Known Issue. 
  • The child object is nested inside a group and UI rule associated is applied inside grouping. 

Resolution

  • The fix for this issue is planned to be deployed on 2H 2026 release.
  • For release timelines, please review SAP SuccessFactors Product Release & Road Map Information.
