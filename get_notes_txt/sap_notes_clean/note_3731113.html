Symptom

The field VJPEI in table MBEW contains incorrect data. 

Same is true for other previous period VJ or VM fields in table MBEW. 

Environment

  • SAP ERP Logistics (LO)
  • SAP R/3
  • SAP R/3 Enterprise
  • SAP ERP Central Component
  • SAP ERP
  • SAP Enhancement package for SAP ERP
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Go to T-code SE16N.
  2. Table - MBEW.
  3. Enter material, plant.
  4. Value of VJPEI(Price Unit) field is incorrect.
  5. It can be observed in transaction code MM04 that the field value was changed.

Cause

Table MBEW will never have the correct values for previous periods. Due to a design decision decades ago, this data is stored in the corresponding history table MBEWH.

SAP notes 415349 and 193554 contain more details about how this data is stored and how it can be read with the corresponding XXXX_EXTEND function modules.

Resolution

Table MBEWH should be checked for previous period data. If an issue/inconsistency lies for previous periods data it must exist in table MBEWH.
