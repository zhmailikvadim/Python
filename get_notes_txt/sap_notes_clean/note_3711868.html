Symptom

In transaction FAGL_FCV, there is a requirement to restrict specific or all input parameters in a variant so that they cannot be edited by the user.

Environment

  • Financial Accounting
  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA
  • SAP S/4HANA Finance
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. FAGL_FCV
  2. Select variant
  3. Click enter
  4. Variant is unrestricted and can be edited after selection

Resolution

When creating a variant in FAGL_FCV, there is an option to protect the field when saving the variant. If the 'Protect field' option is checked, this prevents the field from being edited. To make fields protected:

  1. Open transaction FAGL_FCV.
  2. Enter required selection parameters for the variant.
  3. Click on 'Go to' in Menu bar.
  4. Click on 'Variants'.
  5. Click on 'Save as Variant' - here it is possible to make any fields protected and any fields unchangeable. 

Additionally, it is important to note that by selecting option 'Protect Variant' here, it means that the variant can only be changed by the user who created it or the last user to change it.
