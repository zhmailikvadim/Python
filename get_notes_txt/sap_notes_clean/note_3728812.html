Symptom

While submitting MM invoice with variance, the freight cost distribution document created for the variance amount stays in the life-cycle status of "In-process" instead of "Posted" status. As a result, the agency billing document for the variance amount is not created.

Other Terms

Condition Type, Agency Billing Document, Freight Cost Allocation document, Invoice Variance, Lean service procurement

Reason and Prerequisites

Program Error

Solution

Implement the attached source code correction, or the support package including this correction.
