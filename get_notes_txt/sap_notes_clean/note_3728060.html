Symptom

In the sales order processing, the association between BOM sub-items and their parent item is incorrect. This can cause a runtime error of type TIME_OUT in include FV45PF0P_PREISFINDUNG_VORBEREI (form PREISFINDUNG_VORBEREITEN of program SAPFV45P).

Other Terms

BOM, UEPOS, GRPOS, sorting, runtime, performance, endless loop

Reason and Prerequisites

The problem is caused by a program error (change of the sort order of internal table XVBAP) in program MV45AF0B_STATUS_TRANSFER.

Solution

Implement the correction instructions.
