Symptom

You run report RPCPAYUS_EIWO_FILE_CHECKER and it returns the error:

SXPG_CALL_SYSTEM: Command Name Is Not Defined (ZHRPAYUS_LIST_DIR).
Message No. HRPAYUS_MESSAGES471

Environment

  • HR Renewal
  • SAP S/4 Hana
  • SAP ERP
  • SAP SuccessFactors Employee Central Payroll

Cause

Command ZHRPAYUS_LIST_DIR hasn't been created in your system.

Resolution

The external operation system command ZHRPAYUS_LIST_DIR should be created on your system.
This can be done by following the manual step number 2 of SAP Note 2940542 (File Manager Does Not Read Files With Name Length Greater Than 32 Characters).

After that, the scenario can be retested and it should be fine.

See Also

2940542 (File Manager Does Not Read Files With Name Length Greater Than 32 Characters).
