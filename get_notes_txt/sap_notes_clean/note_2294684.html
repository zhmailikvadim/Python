Symptom

  • Application error when navigating to a user's employment information page
  • Application error while hiring via Manage Pending Hires

Note: Support Engineers please find the Splunk logs to compare this issue on the Internal Memo section.

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental." 

Environment

  • SAP SuccessFactors Employee Central
  • Employment information

Cause

Employee's Manager do not have a first name and last name filled in Employee Profile(EP)

Resolution

  1. Find the affected user Manager
  2. Add "First" and "Last Name" for this Manager.
