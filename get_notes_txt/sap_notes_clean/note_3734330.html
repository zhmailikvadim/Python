Symptom

You reserve a material number from the Hub system during the product version key reservation process, or you get a material number from the Hub system during the product version maintain request. The MDG process flag in the customizing under Cross-Application Components -> DM Product Version to SAP Material Master -> Map DM Product Version to SAP Material Master -> Define Mapping of Product Type is not checked. The system takes the material number from the Hub system based only on the RFC destination in the customizing under Cross-Application Components -> Define Runtime Settings. The system should take the material number from the Hub system only when the MDG process flag is set.

Other Terms

PLMSI, PLM system integration, PV, PV KR, Product Version, Product Version Key Reservation

Solution

Implement this SAP Note to ensure the system takes the material number from the Hub system only when the MDG process flag is set.
