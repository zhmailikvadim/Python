Symptom

User not able to generate landscape report because they do not have write access to where the BI support tool is installed

Environment

  • SAP BusinessObjects Business Intelligence Platform 4.x / BI 2025
  • SAP BI Platform Support Tool

Reproducing the Issue

  1. Remove write access for a specific user on the directory where the BI Support Tool is installed.
  2. Log in as that user.
  3. Open BI support tool 
  4. Attempt to generate a Landscape Report
  5. Observe the error: "Error encountered generating report output. Data may be incomplete"

Cause

This behavior is by design, the XML file is generated in the installation directory. Since the user does not have write permissions on this folder, the file cannot be created.
