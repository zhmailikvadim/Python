Symptom

When uploading master data using the Import Consolidation Master Data app in SAP S/4HANA Cloud Public Edition (Group Reporting), the upload fails with the following error:

"Xlsx sheet ABC contains errors"

(ABC represents the master data type)

Examples of master data types:

Financial Statement Item
Consolidation Unit
Consolidation Group
Other consolidation master data types

Additionally:

No detailed error message is displayed
No application log is generated
The same data can be created manually without issues

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Set the required Global Parameters in "Set the Global Parameters" app.
  2. Go to Import Consolidation Master Data app.
  3. Choose the desired master data type.
  4. Click "Upload" button.
  5. Select the prepared template file..
  6. Observe the error: "Xlsx sheet ABC contains errors".

Cause

The issue occurs when mandatory header (filter) fields in the Excel template are not maintained.

The import framework requires these fields to define the processing context before validating the data.

Mandatory fields include:

Consolidation Chart of Accounts
Consolidation Version
Effective Fiscal Year/Period (YYYY/PPP)
Language

If these fields are empty:

The system cannot determine the target context
The file fails during the pre-processing stage
No detailed log is generated because validation is not reached

Resolution

Maintain all mandatory header fields in the Excel template before uploading:

  1. Open the Excel template downloaded from the system
  2. Fill the Mandatory Filter Values in the header section:
    Consolidation Chart of Accounts
    Consolidation Version
    Fiscal Year/Period (YYYY/PPP)
    Language
  3. Save the file
  4. Re-upload using the Import Consolidation Master Data app

The upload should then be processed successfully.

Please ensure that all required fields in the file are completed.
