Symptom

  • Error message: "The maximum limit of 200 Explorations has been reached."
  • Users are unable to create additional Explorations in the SAP Customer Data Platform.

Environment

  • SAP Customer Data Platform
  • Explorations

Reproducing the Issue

  1. Attempt to create a new Exploration in SAP Customer Data Platform.
  2. Observe the error message stating that the maximum limit of 200 Explorations has been reached.

Cause

This is a product limitation.

Resolution

  1. Open a feature request to address the limitation.
  2. Refer to KBA 2700117 - How do I submit a product enhancement or feature request for SAP Customer Data Cloud?
