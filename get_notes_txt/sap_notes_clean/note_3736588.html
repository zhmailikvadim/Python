Symptom

  • The tile cannot be found in the App Finder or when searching in the Site Manager.

  • The standard Task Center (End-User) app is visible and working, but the Admin app is missing.

  • Channel Manager in Work Zone Site Manager shows the status: "Partial content was updated" after a refresh.

  • User has the TaskCenterAdmin role assigned but the Task Center Administration tile is not visible in the SAP Build Work Zone (Standard or Advanced) Launchpad.

Environment

SAP Build Work Zone, Standard Edition. 

Cause

Cause 1: Insufficient Role Collections The user has the administrative role but lacks the functional "Operator" role required for the BTP metadata service to expose the HTML5 application to the Launchpad.

Cause 2: Metadata Naming Conflict (The "{{flpTitle}}" Bug) Due to a known UI rendering limitation in the "New Experience" view mode, the app is indexed under its technical placeholder {{flpTitle}} instead of its friendly name.

Cause 3: Stale Content Mapping / Metadata Desync The internal link between the Work Zone "My Content" and the BTP HTML5 App Repository is broken or cached with "Partial" data, preventing the tile from rendering.

Resolution

Solution to Cause 1: Authorization Audit

  1. Log into BTP Cockpit > Security > Users.

  2. Ensure the following two Role Collections are assigned to the user:

    • TaskCenterAdmin

    • TaskCenterTenantOperator

  3. Log out and log back into the Work Zone site to refresh the session.

Note:  For Roles requirements on Task Center, the component is: LOD-BPM-INB.

Solution to Cause 2: Technical Naming Search

  1. Open Work Zone Site Manager > Content Manager > Content Explorer.

  2. Select the HTML5 Apps tab.

  3. Search for "Task Center Administration".

  4. If no results appear, search for {{flpTitle}}.

  5. If the app appears under either name, select it and click + Add to My Content.

For more information about Search in Work Zone: Check SAP Note/KBA: 3410270 - Missing apps when selecting content for tiles widget in Work Zone Spaces and Pages - New Experience View Mode - SAP for Me  (Component: EP-WZ-FS)

Solution to Cause 3: The "Hard Reset" (Recommended Last Resort)

If the app is visible in "My Content" but not on the Site:

  1. In Content Manager, locate the "Task Center Administration" (or {{flpTitle}}) item.

  2. Delete the item from your "My Content" list.

  3. Go back to Content Explorer > HTML5 Apps and re-add the app fresh.

  4. Navigate to Channel Manager and click the Update Content (circular arrows) for the HTML5 Apps provider.

  5. Re-assign the app to the appropriate Role, Group, and Space.

Note: For any steps or details about Work Zone use Component:  EP-WZ-PRV
