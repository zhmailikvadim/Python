Symptom

Information on how to migrate from SAP Intelligent Robotic Process Automation to SAP Build Process Automation.

Environment

  • SAP Intelligent Robotic Process Automation
  • SAP Build Process Automation

Resolution

Feature updates and Support priority is focused on SAP Build Process Automation and Desktop Agent 3.x.

For  better support, we are strongly advising you to :

  • Migrate your Desktop Agent from version 2.x to the latest Agent 3.x
    • Minimal supported SDK 1.26
    • Regularly updating your SDK to the latest available version is a best practice as you will benefit from fixes and new features.
    • Desktop Agent 3 offers better tools for troubleshooting of such issues, such as Support Mode.
    • See the article about Best Practices when Migrating from Desktop Agent 2 to Desktop Agent 3 
  • Migrate from Intelligent Robotic Process Automation to SAP Build Process Automation
    • End of Mainstream Maintenance]: 31.12.2026
      • Please considerate migration time
    • See the articles about:
      • [SBPA migration| SAP Intelligent Robotic Process Automation – the Path Forward]
      • [project export| Export a Project and project import| Project Import]
  • Migrate from Desktop Studio project to Cloud Studio Projects
    • End of Mainstream Maintenance]: 30.06.2025
      • Please considerate migration time
    • See the article about [Cloud Studio migration]
  • In any above cases, a Non-Regression test phase is recommended.
