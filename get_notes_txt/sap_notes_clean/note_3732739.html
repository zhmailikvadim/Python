Symptom

Cannot promote top-level security on BI4.3 SP04 Patch600

Reason and Prerequisites

Environment:

  • SAP BusinessObjects Business Intelligence Platform 4.3 SP04 Patch 600 

Steps to Reproduce:

  1. Go to Calendar->Manage->Top-Level security and set the security “MIS/View_test”.
  2. Promote user group “MIS” first from source to destination.
  3. Export calendar with top-level security “MIS/View_test” with promotion management.
  4. Import calendar including the user group and security with promotion management.
  5. After the import, user group and security “MIS/View_test” are promoted, but when go to Calendar->Manage->Top-Level security, top level security is not promoted.

Cause:

The cause is currently under investigation by Development.
