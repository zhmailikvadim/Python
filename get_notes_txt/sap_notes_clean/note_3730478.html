Symptom

When launching the Manage Contract Fiori app for agricultural contracts, the app displays the error advanced CPE is inactive press ok to navigate to the Maintain ACM Trading Contract Fiori app.

Environment

SAP S/4HANA

Reproducing the Issue

  1. Open the SAP Fiori launchpad.
  2. Start the Manage Contract app for agricultural contracts.
  3. Observe that the app shows an error and does not open.

Cause

The CPE feature toggle is inactive and the Agricultural Contract Management (ACM) business function for Advanced CPE is not activated, which prevents the new Fiori apps from running.

Resolution

  1. Activate the CPE feature toggle S4OP_FTGL_CMM_CPE_REDESIGN and activate ACM business function for Advanced CPE.
  2. Activate the ACM business function for Advanced CPE.
  3. Refer to: SAP Note 3631220 - Data conversion guide (see section 3.2.5) for detailed activation and data conversion guidance.

See Also

SAP Note 3631220 - Data conversion guide (section 3.2.5)
