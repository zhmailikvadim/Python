Symptom

  • An error occurs when the journal entry link generated in Fiori app 'Post General Journal Entries' and itis shared with the approver for posting.
  • Upon clicking the link, the approver encounters the error message: "Your session has ended. Please go back and try again."

Environment

  • SAP S/4HANA 

Reproducing the Issue

  1. Upload a test file in the 'Upload General Journal Entries' Fiori application.
  2. Click on the uploaded file name to open the 'Post General Journal Entries' Fiori app.
  3. Use the 'Send E-Mail' option to automatically generate a link in Outlook.
  4. Share the generated link with the approver.
  5. When the approver clicks the link, the error message "Your session has ended. Please go back and try again." appears.

Cause

In SAP standard the uploaded documents are only visible to the user who created them. 

Resolution

The "Send E-mail" feature is cannot be used as a substitute for workflow functions, as the functionality is designed for use by the same user within the same session and does not support concurrent access by multiple users.
