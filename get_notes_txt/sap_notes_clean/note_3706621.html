Other Terms

Romania, XNEGP, FIRO_ONESBAL, FIRO_SET_XNEGP_RWIN, Asset Management, F-92, ABAON

Reason and Prerequisites

This SAP Note is relevant only for Romania.

Design

Solution

Apply the Support Package specified for your release or implement the attached correction instruction.
