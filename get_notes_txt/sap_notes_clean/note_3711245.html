Symptom

  • F2601 Fiori App displays data only for the last four months.
  • The filter does not provide an option for date range
  • The system defaults to the latest four months.

Environment

  • Sales and Distribution (SD)
  • Fiori
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Open the Sales Management Overview Fiori App.
  2. Observe that the displayed data is limited to the last four months.
  3. Attempt to use the filter to extend the date range.
  4. Note that no option is available to modify the default four-month period.

Cause

According to standard design the F2601 Fiori App displays the last four months data only.

Resolution

Refer to the SAP Help documentation for Fiori app F2601 to understand its standard design limitations.

Use other Fiori Apps such as F2235 or F1249, that provide filtering options to select a desired time period.

Consider creating a custom version of app F2601 to extend the date range functionality.
