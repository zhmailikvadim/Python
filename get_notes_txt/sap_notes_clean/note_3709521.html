Symptom

Advanced wait options such as "Send at a particular time" or "Next Sunday" are not available in the restock program.

Environment

SAP Emarsys Customer Engagement

Reproducing the Issue

  1. Attempt to configure advanced wait options such as "Send at a particular time" or "Next Sunday" in an event-based program within the restock program.
  2. Observe that these options are not available due to the program's ineligibility for calendar-based scheduling.

Cause

Event-based automations are triggered individually per contact, and the exact entry time varies per contact. As a result, SAP Emarsys cannot safely calculate a fixed calendar time for sending, which is required for advanced wait options.

Resolution

1. Set a standard Wait Node with a relative wait time for the minimum duration required (e.g., 1 day).  
2. Immediately following the Wait Node, add a Decision Node using the condition: "Is the current day Sunday?"  
   • If Yes: Proceed to the "Send Email" node.  
   • If No: Route the "No" path to a 1-day Wait Node, which then loops back into the same Decision Node.
