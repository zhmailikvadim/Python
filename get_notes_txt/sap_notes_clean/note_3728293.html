Symptom

Re-processing an item that was rejected (release status 03) and corrected while it is in post-processing (status '70') causes the PI status to hang at '10'

Other Terms

/PE1/CL_PO_STANDARD_PROCESS~PROCESS_ITEM, Release status '03' rejected

Reason and Prerequisites

If a user attempts to resubmit an item that was rejected, corrected, and is currently in post-processing, then item would get stuck in status '10'.

There was a code logic to delete the item that was in release status-'03 rejected'. Hence the manual resubmission was not working and item was stuck in status '10'.

Solution

Implement the correction instruction attached to this SAP Note. Alternatively, you can import the referenced Support Package.
