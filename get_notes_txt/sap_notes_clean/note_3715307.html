Symptom

When you clear various time events for a user, you notice that some working times remain visible after refreshing the browser

Environment

  • Employee Central
  • Time Tracking

Reproducing the Issue

  1. View time sheet for a user with multiple time events
  2. In quick succession, delete various time events while leaving some in place
  3. Incorrect working times remain visible even though the associated events are removed

Cause

Technical limitation

Resolution

  1. Go to Admin Alerts > Processing Paired time events for Time Sheet
  2. Locate the user for which the problem happens, select the line item and Action > Reprocess
