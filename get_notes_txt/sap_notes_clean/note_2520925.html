Symptom

No spacing between template info (e.g. no spacing between Template and audit level) in the BPF report detail page.

Step to reproduce,

1) Logon bpc web client, standard environment

2) Go to bpf operation system report, choose template:BPFScenario, choose template version:BPFscenario (deployed)

3) Click display report, it will go to system report detail page

Other Terms

BPF, Report, no space

Reason and Prerequisites

UX issue

Solution

Please upgrade component BPC4HANA100 to Support Package 02
