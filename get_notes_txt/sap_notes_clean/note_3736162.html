Symptom

 Operation long text gets truncated when it is migrated to work instruction during routing migration using the report 'MPE_MIGRATE_CLSRTNG_TO_SFRTGS'

Other Terms

Migration, MPE_MIGRATE_CLSRTNG_TO_SFRTGS, Operation Long Text, Shop Floor Routing, Work Instruction

Reason and Prerequisites

 Program error

Solution

After implementing this SAP Note, operation long text will be properly migrated to work instruction
