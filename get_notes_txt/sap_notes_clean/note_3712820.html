Symptom

Custom formula of condition base value has been maintained for a tax condition in a pricing procedure, however, in an intercompany billing document which uses this procedure, this tax condition has AltCndnBaseVal as 2.

Environment

SAP S/4 HANA Cloud Public Edition

Reproducing the Issue

  1. Open the app Display Billing Documents (vf03).
  2. Search for billing document ABC and open it.
  3. Open item 10 and select the Conditions tab.
  4. Observe that tax condition XYZ has AltCndnBaseVal as 2, however in corresponding pricing procedure a custom formula has been maintained for this condition.

*ABC represents billing document ID

*XYZ represent tax condition ID

Cause

In intercompany billing, for tax conditions, the system sets condition basis formula 002 by default. Customizing formulas cannot override this default formula.

Resolution

System standard behavior

Keywords

unexpected condition base value, pricing procedure, SAP S/4HANA Cloud Public Edition, intercompany billing, tax condition, condition basis formula, billing document, condition base value.
