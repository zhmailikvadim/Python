Symptom

Deployment of subscriptions to the Production Connector fail with errors like: There is no such provider type OPCUA implemented right now!

Other Terms

Digital Manufacturing, Subscription, Deployment

Reason and Prerequisites

Program error in the migration of configuration

Solution

Workaround

No workaround available.

Problem Solution by Patch

First step is to install the patch 6 for Production Connector 3.0. After the patch is installed it is necessary to run the Migration Utility. The program is found in the installation folder of the Production Connector and it is called ProdConMigrationUtility.exe. You can get it by entering Migration Utility in the Windows Search. In the next step the program has to be executed as Administrator. In the pop-up of the program you should deselect the option Reinstall Windows Services for Agent Instances and then run the migration.

It is important that all agent instances are stopped before the migration program is executed!

Installation of Patches

Download the current installation program for the patch from the SAP Software Center and install it in accordance with the instructions in the installation guide. For the current version of the installation guide, see the SAP Help Portal at https://help.sap.com/docs/PRODUCTION_CONNECTOR. You can use the installation program for the patch to update a version of your product that is already installed or to set up a new installation if the product is not yet installed on the target computer.

Before you start the installation, see also the relevant release installation note 3706769 for Production Connector 3.0.
