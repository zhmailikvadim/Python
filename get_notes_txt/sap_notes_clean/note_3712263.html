Symptom

When you choose "Use Data" in the report ELStAM PHI: Administrator List  (RP_PAYDE_E2_PKV_LIST), a runtime error occurs when you save.

Reason and Prerequisites

You use transaction PA30 to lock the personnel number for which you want to transfer the data to the infotype SI Supplementary Insurance  (0079) in parallel.

Solution

Implement the attached correction instructions or import the specified support package.

After you implement the corrections, the system no longer issues a runtime error in this case, but instead issues a corresponding error message.

If you first locked the personnel number using transaction PA30 after choosing the pushbutton and before saving, the notification was set to the status Processed  even though the data was not transferred to IT79. In this case, transfer the data manually to IT79.
