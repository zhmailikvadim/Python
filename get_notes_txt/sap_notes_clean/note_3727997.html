Symptom

When I attempt to switch to Classic sourcing view, it fails with the error message: Switching to classic view is not supported for multi-round events.

Reproducing the Issue

  1. Open a Guided Sourcing RFQ, RFP, or RFI.
  2. Click View Options > Switch to Classic View
  3. Observe the error message: Switching to classic view is not supported for multi-round events.

Cause

As part of the 2603 release, multi-round events features are only available in Guided Sourcing. You cannot switch to the Classic Sourcing view for multi-round events.

Resolution

  • If you need to perform actions on a multi-round event, please continue to use the Guided Sourcing interface.
  • There is no way to bypass this restriction; you cannot switch to the classic view for multi-round events. If you require classic view for future events, you must create them as single-round events.
  • If you inadvertently configured the event for multi-round bidding and need classic view, you may need to recreate the event with only a single bidding round.
