Symptom

In SAP Multichannel Foundation for Public Sector when using the Co-Browsing functionalities sometimes an null-pointer exception may occur.

Other Terms

Multichannel Foundation for Public Sector ( MCF , FMCA ) ; UMCUI501 ;  FMCAUI5_MOBILE ; WebContent/Agent_oanel/model/OnlineDataProvider.js ;

Reason and Prerequisites

This issue occurs only when using the MCF PS solution of SAP with the Co-Browsing functionalities and the standard UI template.

In the template UI of Multichannel Foundation for Public Sector (Software Component FMCAUI5_MOBILE) in JavaScript Document WebContent/Agent_oanel/model/OnlineDataProvider.js an error could be thrown because not all by backend returned data has the necessary properties. So the UI is trying to read a not defined object and throws a null-pointer exception.

Solution

Implement the attached correction instructions or import the specified Support Package.

Due to this is an template UI change you can also compare your file WebContent/Agent_oanel/model/OnlineDataProvider.js with the attachment of this SAP Note and update your UI manually.
