Symptom

You download the Vizi Operational Report for Agreement Details in text format from Fiori and expect the file to contain agreement details. However, the downloaded text file is blank, while the same data is displayed correctly when the report is downloaded in CSV or Excel format.

This note corrects the behavior so that agreement details are included when downloading the report in text format from Fiori.

Other Terms

/VIZI/LOPERF0D, /VIZI/OPER, Excel, Exported, Blank File

Reason and Prerequisites

This is caused by program error

Please implement SAP Note#3549611 as a Pre-requisite note

Solution

Please implement the correction instructions attached. These corrections are only valid if you have applied the following support pack(s).

Release V4 - 2023 V3 and 2023 V3

Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object.
