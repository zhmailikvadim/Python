Symptom

When importing a Benefit Employee Claim record for an employee, an error is received: "java.lang.IllegalArgumentException: Can't pass empty values for 'workerId', using 'in' criteria"

Environment

SAP SuccessFactors Employee Central

Reproducing the Issue

  1. Log into environment
  2. Go to 'Import And Export Data' tool
  3. Import 'Benefit Employee Claim' record
  4. Receive error

Cause

The employee's Legal Entity is not configured as a valid 'Legal Entity' value in the Benefit's configuration

Resolution

When configuring a Benefit, each Legal Entity it is applicable to is defined in the 'Legal Entity' section. If an employee's Legal Entity / Company is not assigned under a Benefit, they will not even be checked to be eligible for the benefit.

If an employee needs to be enrolled in / claim a Benefit, their Legal Entity needs to be defined in the Benefit's configuration in 'Benefits Admin Overview'.
