Symptom

In transaction VL31N, attempting to delete a subcontracting component from the “SC Components” pop-up during inbound delivery creation raises error VLSC203 “Material must remain in the list of components.”

Environment

  • Sales and Distribution (SD)
  • Logistics Execution (LE)
  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Run transaction code VL31N.
  2. Enter a subcontracting purchase order.
  3. Select the delivery item and choose “SC Components”.
  4. Try to delete one of the component lines.
  5. Observe error VLSC203 “Material must remain in the list of components.”

Cause

Technical restriction: at least one component must remain for each material in the subcontracting component list; deleting the only (or last remaining) component for a material triggers VLSC203.

Resolution

  1. Instead of deleting the component, set the quantity to 0 for the component to bypass the restriction. 
  2. Save the changes to proceed with the inbound delivery creation.
