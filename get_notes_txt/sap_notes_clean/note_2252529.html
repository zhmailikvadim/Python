Symptom

In the standard system as currently delivered, on the selection screen of the Year End Reporting, Canada (RPCYERK0 or RPCYERK0_CE) report, you choose the Run History pushbutton. The system displays a History Log screen, and there is a row of pushbuttons at the top. The text on the View on ESS pushbutton is missing.

Other Terms

PC00_M07_CCYR, Employee Self-Service, PCERH, 70, SAPMPYK0.

Reason and Prerequisites

Program error.

Solution

As general rule, SAP recommends that you install a solution by applying a Support Package. However, if you need to install a solution earlier, use the Note Assistant (SNOTE) to implement the correction instructions.

You can find more information about the Note Assistant in SAP Service Marketplace, under service.sap.com/note-assistant.
