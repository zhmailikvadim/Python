Symptom

W_PRICAT_MAINTAIN: The designed behavior is, when the articles are processed using this transaction and the system returns an error - the default status is set as '04' (processed) after checking if article exists in database. This is as designed.

Note 1413969 needs to be removed, which changes the status to '05' in this case. There is a modification note which can be used for changing this behavior.

Other Terms

W_PRICAT_MAINTAIN, status 04, modification, notes 1852164, 2107877, note 2119770.

Reason and Prerequisites

Program Error.

Solution

Implement the attached correction instruction.
