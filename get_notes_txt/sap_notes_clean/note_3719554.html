Symptom

  • A newly created status schema is not appearing on cases.
  • Custom statuses are not visible in the status field when creating a new case.

Environment

SAP Service Cloud Version 2

Reproducing the Issue

  1. Go to Cases.
  2. Create a Case.
  3. Select the Status field.
  4. Observe that custom statuses are not visible.

Cause

  • The custom status schema has not been assigned to the case types.

Resolution

  1. The custom status schema should be added in case type settings.
  2. Go to Settings.
  3. Navigate to Case Types.
  4. Select the required Case Type.
  5. Select Create New Version.
  6. In the new version, add the custom status schema.
