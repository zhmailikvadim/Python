Symptom

Upon creating or updating a SD-, MM-, or LE document with active TM integration, you realize that certain data is missing in the TM documents or that freight units have been constructed unexpectedly. An authorization trace reveals insufficient permissions for TM activities, specifically regarding authorization objects such as T_TOR_EXE, T_TOR_PUR, or T_ADMIN.

Other Terms

MM-TM Integration; SD-TM; Integration; LE-TM Integration; TM Integration; Privileged Mode; Authorization Issue; SU22; SU54; Open Quantity Calculation; PFCG

Reason and Prerequisites

Missing TM authorizations causes incomplete data retrievals via BOPF resulting into incorrect integration results. The relevant authorization objects are not set as defaults and therefore are missing from customer-maintained roles done in transaction code SU23.

Solution

Implement the corrections of the attached correction instructions or apply the corresponding support package. Then the logistics integration part is executed in privileged mode so no TM-authorization checks will fail.

After you've installed this SAP Note, you run the CP21 - Notification of Departure from the Country of an Employee (RPCT21L0) report for an terminated employee. When you select the Display More Details checkbox and set the CP21 Tax Assessment Period - to field on the selection screen after 01.01.2026, a new column Resident is added at the end of the ALV output table. If the employee is resident in the year of termination, the value under this column is "X".

Other Terms

CP21 - Notification of Departure from the Country of an Employee, RPCT21L0, resident, /39B, TAXE-RESIS

Reason and Prerequisites

This is a program enhancement. By default the data source of the Resident column is the Resident Employee (/39B) wage type.

You have installed SAP Note 3686488 and 3691257 and implemented all the manual steps. After that, you have run Payroll driver for Malaysia (RPCALCL0) report at least once in the year of the employee's termination.

Solution

You can apply HR Support Package or carry out the manual activities attached and implement the correction instructions in this SAP Note.
