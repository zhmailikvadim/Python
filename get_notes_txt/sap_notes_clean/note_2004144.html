Symptom

You use ABAP Development Tools for SAP NetWeaver. You want to navigate to the declaration of a type in a local class include.

No navigation takes place or the system takes you to the ABAP Editor in the SAP GUI.

Other Terms

ADT, navigation, OCL

Reason and Prerequisites

  • You use ABAP Development Tools for SAP NetWeaver.
  • The affected class has not been migrated to the new class structure. This means that only three tabs (for the global class, class-relevant local types, and test classes) are displayed in the editor.

Solution

Import the specified Support Package or implement the attached correction instructions.
