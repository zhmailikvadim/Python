Symptom

Error occurs while trying to Delete the Outbound delivery - "Successor documents exist. Deletion is not allowed for document type LF."

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. In Manage Outbound Delivery Order app, reverse the Goods Issue for the Outbound delivery.
  2. In the Work Packing Center for outbound delivery app, release stock in order to cancel Picking task.
  3. In Manage Outbound Delivery Order, attempt to delete the Outbound delivery.
  4. Observe the errors: "Successor documents exist. Deletion is not allowed for document type LF." and "Activity 'Delete File' not allowed; Reason: Status 'Warehouse Activity' has value 'Partially Completed'."

Cause

Expected system behavior: an outbound delivery of the given document type cannot be deleted if picking tasks were created earlier, even if those tasks were later canceled.

Resolution

In the current system behavior, an Outbound delivery cannot be deleted when there are any actions taken on it.
If the requirement is to create a new Outbound delivery with different quantity, as a workaround below steps can be followed : 

  1. In app Manage Outbound Delivery Order, zero out the delivery quantity.
  2. Post Goods issue for the Outbound delivery in Manage Outbound Delivery Order.
  3. If required, create a new Outbound delivery with the required changes.
