Solution

Log of Changes

Version Date Description
1 October 5, 2023

SAP Note released for all customers.

For Phase I, the following changes have been addressed by this SAP Note.

Paper Form for Form 1096

The layout of Form 1096 was updated for the year of 2023. This SAP Note delivers the new layout of the Form 1096 (HR_F_1096_23) as specified by the Tax Authority. Only minor text and layout changes have been added to the form.

Paper Form for Form W-2/W-2c Puerto Rico

The layout of Form W-2 PR was updated for the year of 2023. This SAP Note delivers the new layout of the Form W-2 PR (HR_F_W2_PR_LP_23) as specified by the Tax Authority. Only minor text changes in the instructions pages of the form have been added to the form.

The layout of Form W-2c PR was updated for the year of 2023. This SAP Note delivers the new layout of the Form W-2c PR (HR_F_W2C_PR_LP23) as specified by the Tax Authority. Only minor text changes in the instructions pages of the form have been added to the form.

Form W-2/W-2C Box 14 for Paid Family and Medical Leave

For the State of Colorado, starting with tax year 2023, employers must report Paid Family and Medical Leave (PFML) in Box 14 of the Form W-2. This SAP Note delivers the required configuration to enable the reporting of the Tax Type 117 (EE PFML TAX STATE PLAN) for the State of Colorado and with the label "FAMLI", in the Box 14 of all form W-2 PDF layouts supported by SAP.

Magnetic/Electronic Filing for Form W-2/W-2C Box 14 for Paid Family and Medical Leave

The Third-Party W-2 (HR_F_W2_TP_TEMPS) and W-2c (HR_F_W2_TP_TEMPC) electronic file formats were updated to report the Colorado Paid Family Medical Leave (COPL):

  • Form Third-Party W-2 Layout (HR_F_W2_TP_TEMPS):
    • In Record H:
      • Position 610 (COPL Text) with length 5 created. The field Or directly is reporting "FAMLI". 
      • Position 615 (COPL Amount) with length 12 created. Taking from field "COPL" with field conversion 1.
  • Form Third-Party W-2c Layout (HR_F_W2_TP_TEMPC):
    • In Record I:
      • Position 202 (Corrected COPL Text) with length 5 created. The field Or directly is reporting "FAMLI".
      • Position 207 (Original COPL Text) with length 5 created. The field Or directly is reporting "FAMLI".
      • Position 212 (Corrected COPL Amount) with length 12 created. Taking from field COPL with field conversion 1.
      • Position 224 (Original COPL Amount) with length 12 created. Taking from field OCOPL with field conversion 1.
      • Position 236 (Blank) with length 498 was created.

Post-Application Checks

  • Since changes to magnetic media formats are included, you should review the magnetic media configuration of the formats that you file to ensure that the information (for example: contact name, contact phone blocking factor, registration number, and so on) remains consistent. To this end, in Customizing for Payroll USA (PY-US), choose Tax Reporter -> Tax Form Definition -> Define original tax forms, select your form by double-clicking the form name in the Tax form column, and choose Magnetic tape structure in the Dialog Structure. For more information, see the Customizing activity documentation attached to this activity.
  • Check to see whether changes, such as disk for tape or vice versa (see SAP Notes 306543), additional carriage returns, and/or EBCDIC to ASCII conversions (see SAP Note 133613) need to be reapplied.
  • Finally, we suggest that you run the configuration checker utility from within Tax Reporter (transaction PU19). To this end, choose Utilities -> Check configuration, select the help icon to the right of the message header for an explanation of each warning and/or error message.
