Symptom

You are using a Unified Rendering based framework (like Web Dynpro ABAP, SAP GUI for HTML) and are experiencing the following issue: The screen reader announces "Mehrwertige Änderung" for the German translated ARIA roledescription of the Multi-Value Input (Tokenizer).

Other Terms

Web Dynpro Java, Web Dynpro ABAP, WDJ, WDA, SAP GUI for HTML, UnifiedRendering, Unified Rendering, Lightspeed,

Reason and Prerequisites

Proprram Error

Solution

As part of regular intense quality- and patch creation tasks, all Unified Rendering fixes will be collected and shipped about every four weeks.

The fix will be shipped with the next "Cumulative patch for Unified Rendering 2602".

Once available it will be delivered for WD ABAP starting from 7.02, WD JAVA starting from 7.30 and SAP GUI for HTML all kernel releases.

Be aware this SAP note can’t be implemented.

Check SAP note 2500800 for the availability of the corresponding framework patch (delivery dates) and information about the patch process.

See also section "References".
