Symptom

When using SOAP 1.x sender adapter with message exchange pattern One Way and processing settings WS Standard (asynchronous mode), request processing might fail with exception WebIllegalStateException: The operation is not allowed. The HttpServletRequest is not associated with a client request.

Other Terms

Cloud Integration Content, SAP Netweaver Process Integration, SAP Netweaver Process Orchestration

Reason and Prerequisites

Cloud Integration Content in Process Orchestration

Solution

This is fixed with the Support Packages and Patches referenced by this SAP Note.
