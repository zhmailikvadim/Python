Symptom

Component: Foundation for EHS Management
Module: Positions

The system does not provide an option to select positions from HR Organizational Management with a range for the selection fields Valid From and Valid To.

In addition, the search with "is earlier than or on" and "is later than or on" does not work when you select a date.

Other Terms

BEGDA, ENDDA, is_later_equal, is_earlier_equal

Reason and Prerequisites

Program error.

You find the prerequisites in the relevant correction instructions.

Solution

After implementing this SAP Note, the system provides the ability to search with date ranges through the proxy class CL_EHFND_ORG_MGMT_PROXY.

Alternatively, you import the corresponding Support Package for the component extension of SAP Environment, Health, and Safety Management or the Feature Package Stack for the SAP S/4HANA system.
