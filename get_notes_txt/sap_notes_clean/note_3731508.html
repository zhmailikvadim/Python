Symptom

You post a payment lot to pay a payment specification for the payment lot (DFKKIP_GRP-PDTYP = '3'). The system terminates the posting and issues error message SG 105 "Enter rate &/& rate type & for 00/00/0000 in the system settings".

Other Terms

FPAY1B, AUSFD, payment specification, FPAY2, FP05, SG 105

Reason and Prerequisites

Reason:
Program error

Prerequisite:
You have created a payment specification for the payment lot and filled the execution date (DFKKIP_GRP-AUSFD). This is incorrect. This should only be filled for payment specifications for payment run.

Solution

Implement the attached correction instructions or import the specified support package. After you import the Support Package or implement the correction instructions, the payment specification for payment lots is processed correctly, even if the execution date is filled.
