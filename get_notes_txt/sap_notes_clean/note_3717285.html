Symptom

In the mass activity "Forward Documents to FI-AR/AP" (transaction FPSL), the system issues error KE 109:

  •  "Characteristic value 'X' for characteristic 'Product' is not allowed"

or

  • "Characteristic value 'Y' for characteristic 'Contract' is not allowed".

Other Terms

FPSL, Accounts Receivable (FI-AR), Accounts Payable (FI-AP), AKTYP = SLTR, PAOBJNR, MATNR, FKK_SAMPLE_5210, bapi_map_copa, BAPIACKEC9, CHARACTER, PROD_NO_LONG, CUST_CHAR_VALUE_LONG, KE109, KE-109, 109(KE), BAPI_ACC_DOCUMENT_CHECK, BAPI_ACC_DOCUMENT_POST

Reason and Prerequisites

Cause
The field length when transferring the characteristic values of a profitability segment to the document check or posting is limited to 18 digits. The transfer of a characteristic with more than 18 characters therefore leads to the error message.

Prerequisites
- The document to be forwarded contains a profitability segment as an assignment to a CO object.
- The profitability segment contains an article or contract characteristic with more than 18 characters.

Solution

Implement the attached correction instructions or import the specified support package.

In addition, implement SAP Note 3715703 if it is not already contained in the Support Package.
