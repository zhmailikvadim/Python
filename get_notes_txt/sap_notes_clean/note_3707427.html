Symptom

You add the field Text SGTXT (Item Text) from BSEG‑SGTXT to the Workbench Deduction Items session so the customer can use it to store information for matching deduction items to claim items and forming a Group. The field is added to structure /IRM/S_IPDCR_DEDUCTIONS and appears correctly in the Deduction Items screen. However, when the same field is added to the group‑related structures /IRM/S_IPDCR_GROUPS and /IRM/S_IPDCRGRPITM, the Item Text data does not appear in the Groups overview screen. This note resolves the issue by ensuring that the Item Text field is properly populated and displayed in the Groups overview screen.

Other Terms

/IRM/LIPDCRMF0B, /IRM/LIPDCRMF0D, /IRM/LIPDCRMF0S

Reason and Prerequisites

This is caused by program error

Solution

Please implement the correction instructions attached. These corrections are only valid if you have applied the Support Packs.

Release V/4 - 2021, 2022

Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object.
