Symptom

You have submitted the SAF-T Financial version 1.2 file on the Norwegian Tax Authority portal and the following error message is shown:

"The file contains format errors according to the XSD definition. Unknown XSD version: 1.10 ".

Other Terms

SAF-T, Audit File, ACR, DRC, SAP S/4HANA for compliance reporting, Statutory Reporting, SAP Document and Reporting Compliance, SRF_NAMESPACE, NO_SAFT_ACCOUNTING

Reason and Prerequisites

This issue is due to programming error.

This SAP Note is relevant only for Norway.

Solution

As a general rule, SAP recommends that you install a solution by applying a Support Package. However, if you need to install a solution earlier, use the Note Assistant to implement the correction instruction. You can find more information about the SAP Note Assistant in the SAP Support Portal, under SAP Note Assistant.

To implement the solution, you must carry out manual activities and implement the correction instructions that are valid for your release.
