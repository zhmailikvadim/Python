Symptom

Logs for /SDF/CALM BUILD_CDM are not allowed to be deleted by report SBAL_DELETE or archived via the application log archiving reports, before the expiration date is reached.

Other Terms

log, /SDF/CALM BUILD_CDM

Reason and Prerequisites

This is a program error.

Solution

Apply the attached correction instructions or install the corresponding support package.

For newer  /SDF/CALM BUILD_CDM log which is created after the note  3723922 implementation. It can be deleted right away.

For previous logs which were created before the note.  Still there is 28 days limitation.  They can be deleted after the expiration date is reached.
