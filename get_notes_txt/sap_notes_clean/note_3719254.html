Symptom

When trying to extract the data the following Error occurs:
RFC exception SYSTEM_FAILURE raised; Syntax error in program DMC_EXT_SERVICE_PROD_A2X.

Environment

SAP Business ByDesign

Reproducing the Issue

  1. Go to the Application and User Management Work Center.
  2. Go to the Data Extraction View.
  3. Select the relevant Run ID.
  4. Select the Application Log.
  5. Go to the Results tab.
  6. Select Expand all to observe the SYSTEM_FAILURE Error:
  • Action GenerateExtractObject processed in 1 package.
  • Action CreateExecutionPackage processed in 1 package.
  • Action Extract processed in 1 package.
  • Work package 1: Finished; RFC exception SYSTEM_FAILURE raised.
  • RFC exception SYSTEM_FAILURE raised; Syntax error in program DMC_EXT_SERVICE_PROD_A2X.

Cause

Due to inconsistency in Extraction Object  SERVICE_PROD_A2X001.

Resolution

1.Go to Application and User Management work center.
2.Open Data extraction view.
3.select New.
4.In the field extraction object select SERVICE_PROD_A2X_ENH instead of SERVICE_PROD_A2X.

While extracting the data, it is recommended to use the new enhanced extraction object SERVICE_PROD_A2X_ENH instead of SERVICE_PROD_A2X.The new enhanced extraction object SERVICE_PROD_A2X_ENH includes additional filters along with the existing filters.

Note: Extraction object SERVICE_PROD_A2X will be deprecated in the future.
