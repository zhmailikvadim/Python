Symptom

Optimizer engine execution time will follow the Optimizer profile 'Maximum Runtime in Minutes' setting.

Environment

Supply Network Planning

Optimizer

Reproducing the Issue

  1. Execute the SNP Optimizer.
  2. Observe that Step 2 execution time is long.

Cause

SNP Optimizer Step 2 runtime is Optimizer engine execution time. 

Resolution

Customizing /SPRO:  Display IMG > Advanced Planning and Optimization > Supply Chain Planning > Supply Network Planning (SNP) > Profiles > Define SNP Optimizer Profiles.

On 'Solution Methods' tab, Maximum Runtime (in Minutes) can be set here.

Checking Optimizer trace file parameter nDLAUF, it is engine execution time in second.

See Also

Refer to: SAP Note 2418301 - Long runtime until optimal solution

Keywords

optimizer job, step 2 runtime, supply network planning, Maximum runtime in minutes, trace profile, runtime limit, nDLAUF.
