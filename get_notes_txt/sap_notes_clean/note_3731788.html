Symptom

Error File has an embedded file type that is not supported. 'xl/embeddings/oleObject1.bin' when trying to upload a .xlsx file in the Reference Library.

Environment

Production and Non-Production.

Reproducing the Issue

  1. Log in with a user who can manage the Reference Library and navigate to Reference Library.
  2. Locate the item that should be updated > Edit > Browse.
  3. Choose Files > select an .xlsx file, and select Attach.
  4. Observe the error message related to xl/embeddings/oleObject1.bin.

Cause

The .xlsx file contains an embedded Outlook template (.oft) or any other non supported file format, which is not supported for upload in the Reference Library.

Resolution

  1. Open the affected .xlsx file in Microsoft Excel.
  2. Review each worksheet for embedded objects (for example, Outlook .oft or other non supported embedded items).
  3. Remove the embedded object(s) from the workbook.
  4. Save the workbook as a clean .xlsx file (without embedded files).
  5. Upload the cleaned .xlsx file to the Reference Library again and confirm the error no longer appears.
