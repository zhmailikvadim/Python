Symptom

  • When accessing BI Launch Pad (BILP) / Central Management Console (CMC) the following error is seen:

    HTTP Status 404 – Not Found
    Type Status Report
    Message The requested resource [/BOE] is not available
    Description The origin server did not find a current representation for the target resource or is not willing to disclose that one exists.

    Apache Tomcat/9.0.91

Environment

  • SAP BusinessObjects Business Intelligence Platform 4.x (BI 4.0 / 4.1 / 4.2/ 4.3)
  • Tomcat
  • Windows
  • Linux / Unix 

Reproducing the Issue

  1. Access CMC / BILP through the browser
  2. Observe the 404 is thrown

Cause

The web applications have not been deployed on Tomcat

Resolution

  1. Stop Tomcat
  2. In a command line, navigate to <BOBJINSTALLDIR>\SAP BusinessObjects\SAP BusinessObjects Enterprise XI 4.0\wdeploy
  3. Run the command (where X is the version of tomcat, eg 7 or 8 or 9, case-sensitive):
    wdeploy tomcatX -DAPP=BOE deploy
  4. Start tomcat

Keywords

Tomcat, 404 error, CMC, BI Launchpad, http 404
