Symptom

Detect an issue with the screen display when rotating the screen with the inventory clerk persona for Android device. The issue is that when viewing the screen vertically and rotate it horizontally, the top part covers the entire screen and does not allow to scroll down and view

Environment

  • SSAM Version 2410
  • MDK SDK 24.7.7
  • MDK Plugin-SDK 2410.0.4
  • Android version 12,13,14

Reproducing the Issue

  1. Access the inventory clerk persona page for Android device
  2. Rotate the screen
  3. When viewing the screen vertically and rotate it horizontally, the top part covers the entire screen and does not allow to scroll down and view

Cause

Product issue

Resolution

The issue will be fixed in the next version of SSAM targeting for 2605 as it requires changes at MDK level and not only metadata

Keywords

Asset Manager, Rotate, Screen, Horizontally, Inventory clerk persona, Allow, Scroll down, View
