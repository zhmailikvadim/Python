Symptom

While using the filters available in the B2B monitoring section, the receiver filter option does not work.
Clicking on this filter shows no action or response as if the filter is not present.

Environment

SAP Business Technology Platform
SAP Integration Suite

Cause

Since the recent UI update this filter is not working as expected

Resolution

There is no resolution that can be used on the user side for this issue.
Instead a patch fix is being worked on with an estimated release date around April 12th 2026.
