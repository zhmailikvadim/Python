Symptom

Probability of default is not correct in FINS_IMPAIRMENT for overdue items.

Environment

  • SAP S/4HANA Finance
  • SAP S/4HANA

Reproducing the Issue

Run transaction FINS_IMPAIRMENT and check field Probability of default.
The percentage is not as expected as defined in "Assign probability of default to aging increment".

Cause

Aging increment is not customized correctly.

Resolution

To run the impairment for overdue items, ensure a negative value is maintained while configuring the aging under General Settings. Refer to below screenshot as an example.
