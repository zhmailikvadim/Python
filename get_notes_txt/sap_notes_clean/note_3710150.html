Symptom

The Application Interface Framework (AIF) is configured in SAP S/4HANA Private Cloud Edition or On-Premise Edition. After the configuration, issues can be experienced in Transportation Management scenarios, such as certain BAdI methods not being triggered or executed as expected.

Environment

  • SAP Transportation Management Embedded in S/4HANA 1709+

Cause

AIF is not supported for Transportation Management in SAP S/4HANA Private Cloud and On-Premise editions. The AIF content is not delivered by SAP for TM on these environment models.

Resolution

  • For Transportation Management in both SAP S/4HANA Private Cloud and On-Premise editions, SAP does not provide any standard AIF content. AIF is only supported and delivered for TM in the Public Cloud edition.
  • In Private Cloud and On-Premise environments, Forward Error Handling (FEH) is the standard and supported mechanism for monitoring and handling interface errors in TM scenarios.
  • Although AIF can technically be configured and may appear to function in these systems, its usage is not supported due to the absence of SAP TM-specific AIF content. Therefore, AIF should not be used for TM in these deployment models.
