Symptom

Every action in the Audit Journal app causes that error pop-up occurs with message "A technical error has occurred." although the app is returning data correctly.

There is error in browser console: Deprecated API call of 'sap.fe.navigation.NavigationHandler.storeInnerAppState'.

Other Terms

Audit Journal, F0997, Batch error, UI5 error, storeInnerAppStateAsync, deprecated call, Compact Journal

Reason and Prerequisites

Incompatible source code with the latest UI5 version.

Solution

Implement the attached Correction Instruction or the mentioned Support Package.

After implementation of the note please recalculate the index of the app (like in the attached note 2449635) for project FIN_GLAUDITJRNL (tab 'Single SAPUI5 Repository Only'). It deletes the front-end caches.

Please implement also the backend note 3506967 which is required by another note that is part of the dependency chain of this front-end note.
