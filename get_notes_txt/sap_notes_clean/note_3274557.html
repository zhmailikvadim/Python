Symptom

Running a background job for programs such as UDM_GENWL - Creation of Worklists , the job is cancelled with the following error:

Triggering a runtime error RAISE_EXCEPTION dump.

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

  • SAP ECC
  • S4CORE

Reproducing the Issue

Goto transaction UDM_GENWL - Creation of Worklists 

Select Variant -> Then Program -> Execute in Background

Cause

Parameters for Parallel Proc. [x] Start Online is selected on the variant.
It is the reason for the error - Control Framework: Fatal error - GUI cannot be reached and consequently the dump in ST22.

Resolution

Don´t select the option [x] Start Online in the job which are executed via background.

311440 - Batch input and controls
