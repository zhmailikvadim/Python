Symptom

When a trade request is processed in transaction TPITRO (Process Trade Requests) and the corresponding deal is successfully created in the backend by the Trading Platform Integration (TPI), the Financial Transaction tab in the trade request detail view remains empty.

The tab is only populated after a full restart of transaction TPITRO. Within an active TPITRO session, re-opening the same trade request does not display the linked financial transactions, even though the fulfillment deal is already created successfully in the backend.

Other Terms

TPITRO, CL_TPI_TRADE_REQUEST method RETRIEVE_FULFILLMENT_AMOUNTS

Reason and Prerequisites

Programming error.

Solution

Please implement the SAP Note or the Support Package.
