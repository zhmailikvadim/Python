Symptom

If a MP Change process run into error, device info. record is not changed and PI time slice is not updated to "Producive" status, when triggering MOS Device Change process at this time, not all MALO PI time slices are selected, but process could continue without error raised, it caused an incorrect result.

Other Terms

MOS device change, DE_B_IM_DEV_CHANGE, MP change, DE_B_IM_PARAM_CHANGE

Reason and Prerequisites

Enhance API due to the special scenario

Solution

Please implement the attached SAP Note or import higher version of Support Package.
