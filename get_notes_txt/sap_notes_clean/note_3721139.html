Symptom

  • Only Shortest Text is shown in dimension display attributes for text and key objects when creating Web Intelligence document based on SAP BW data source
  • Issue is observed with both direct access mode to BEx query and BW OLAP universe generated from Information Design Tool
  • However, when accessing the same data source via OLAP universe generated by Universe Design Tool, all available texts (Short/Medium/Long) are displayed and available for reporting

Other Terms

webi, webintelligence, BICS, query, panel, QP, object, key, InfoObject, UNX, IDT, UNV, UDT, 

Reason and Prerequisites

Environment:

  • SAP BusinessObjects Business Intelligence platform 4.3 Support Package 04 Patch 7
  • Web Intelligence

Steps To Reproduce:

  1. Create a Web Intelligence document based on SAP BW data source (BEx query or BW OLAP universe)
  2. Drag any dimension that has text properties enabled (Short Text, Medium Text, Long Text) from BW InfoObject properties
    1. >> Notice that only Short Text is available for that dimension in Query Panel

Cause:

  • This issue is caused by a defect in the product

Solution

This issue is fixed in the patches listed in the "Support Packages & Patches" section below.
Due to different internal processes, the "Support Packages & Patches" section of the SAP Note will be populated as below:

  • for fixes delivered in "Support Packages": immediately (even if the "Support Package" is planned to be released several months later)
  • for fixes delivered in "Patches": ONLY once the "Patch" is released AND available on SMP (not before)
