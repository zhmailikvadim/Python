Symptom

You try to add a new contact to a partner and receive the error message ‘Employee inconsistent; employee ID in Identification missing’

Environment

SAP Hybris Cloud for Customer

Reproducing the Issue

  1. Go to the Partners work center
  2. Go to Partners view.
  3. Open a partner
  4. Go to the Contacts facet.
  5. Click on New and fill in the required details.
  6. Click on Save.
  7. You receive the error message: 'Employee inconsistent; employee ID in Identification missing’

Cause

You need to adjust the activity 'Number Ranges for Employees and Service Agents' to automatic.

Resolution

  1. Go to the Business Configuration Work Center.
  2. Go to the Implementation Projects view.
  3. Click on Open Activity List.
  4. Search and open activity 'Number Ranges for Employees and Service Agents'.
  5. Change the type to automatic.
  6. Save the changes.

Observation: The use of manual number range is intended for the first data entry or migration process. After that, please maintain the number range (begin number / end number) and allow the system to automatically increment the numbering for employees / service agents.

Keywords

Partner, Contact, Number Range for Employees and Service Agents
