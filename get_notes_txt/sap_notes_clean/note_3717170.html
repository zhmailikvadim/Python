Symptom

  • There is a requirement to configure Multi-Factor Authentication (MFA) in SAP HANA 2.0
  • Does SAP HANA provide native Multi-Factor Authentication (MFA) for username-password authentication?

Environment

SAP HANA Security & User Management

Resolution

SAP HANA 2.0 does NOT provide native MFA for username-password authentication. There is no built-in MFA method for HANA database users.

SAP HANA does not provide native Multi-Factor Authentication (MFA) for username-password authentication. However, MFA can be implemented indirectly through supported authentication mechanisms that enable integration with Single Sign-On (SSO) environments.

User authentication methods are defined as part of the individual user configuration and can leverage external identity providers or services that support MFA, such as SAP Identity Authentication Service (IAS), SSO via SAML or OAuth, or external identity providers like Active Directory and Microsoft Entra ID (formerly Azure AD).

Refer to the following resources for guidance on implementing MFA via supported authentication mechanisms:

  • User Authentication Mechanisms | SAP Help Portal
  • Single Sign-On Integration | SAP Help Portal
