Symptom

In a model with HANA MDX active (ENABLE_HANA_MDX) an EVDRE report contains the following aspects:

  • row expansion
  • suppression on rows
  • fixed columns

When selecting 'Expand and refresh' the data from the first column is copied across all columns. Additionally, data from first row is supressed and all other rows are shifted up by one row.

Environment

  • SAP Business Planning and Consolidation 10.0 version for NetWeaver SP18 or earlier
  • SAP Business Planning and Consolidation 10.1 version for NetWeaver SP07 or earlier

Cause

Program error

Resolution

Note 2118315 contains a code fix for this issue.

Please install this note or upgrade to the corresponding BPC-NW version.

 

Keywords

EVDRE, Expand, duplicate data, duplicate column, Duplicate tuple, ENABLE_HANA_MDX.
