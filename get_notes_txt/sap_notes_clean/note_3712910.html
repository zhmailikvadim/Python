Symptom

  • An administrator is unable to update roles in user profiles in SAP Enable Now Manager.
  • Error message: "A server error occurred. You have no permission to edit user roles".

Environment

SAP Enable Now - Manager

Reproducing the Issue

  1. Log in to SAP Enable Now Manager with an administrator account.
  2. Navigate to Administration > Users.
  3. Attempt to update roles for a user profile. Note: The issue does not occur for SEN default system roles.
  4. Observe the error message: "A server error occurred. You have no permission to edit user roles".

Cause

When a User Role is created in Manager, the Administrator who creates it is automatically set as the sole person able to assign that Role to other users. 

Resolution

  1. Log in to Manager with the admin that is received in the initial SEN credentials with the Welcome Email. If the credentials are no longer found a new submit can be requested via a case under KM-SEN-MGR.
  2. Navigate to Administration > User Roles.
  3. Select the User Role that needs to be modified.
  4. Click the Manage Access button.
  5. In the Selection field, ensure that the user role (administrators is advised) has the access to assign the custom role.
  6. Click OK to save the changes.
