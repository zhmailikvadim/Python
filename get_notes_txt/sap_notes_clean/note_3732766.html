Solution

Needs to be checked whether the trace files under the trace folder of this instance (on Linux OS) are normal? It appears that HANA is hanging when trying to retrieve the list of trace files from the trace directory.

The trace folders are:

/usr/sap/<sid>/HDB<instance_no>/<host>/trace
/usr/sap/<sid>/HDB<instance_no>/<host>/trace/DB_<tenant>

Trace files should be checked and removed (or at least reduced) if possible to speed up performance.
