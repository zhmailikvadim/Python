Symptom

You use the monitoring applications Monitor Messages. When you navigate to the object page and open the Linked Document tab, the table view management button is missing. You cannot edit or save your personalization view and table settings.

Other Terms

 Monitor Messages, personalization, table settings

Reason and Prerequisites

The manifest settings for the monitoring applications are missing the configuration for table personalization in the Linked Document tab on the object page.

Solution

Implement the correction instruction or the respective Support Package.

The correction updates the manifest settings of the monitoring applications (Monitor Messages) to enable personalization features in the Linked Document tab. After applying this SAP Note, the table view management button is available, and you can edit and save your personalization settings.
