Symptom

If an input control's list of values contains only a single member, the "All" choice will not be displayed.

Environment

SAP Analytics Cloud

Reproducing the Issue

  1. Open a new ODE story
  2. Create an input control with only one valid choice.
    >> Observe that the "All" option does not appear in the input control.

Cause

  • By design behavior.
  • When a filter’s list of values contains only a single member, the “All” option is intentionally not displayed. This is because selecting “All” would return the same result as selecting the only available member, so the option is omitted to avoid redundancy.

Resolution

  • Possible Workaround: Enable the option “Show Inactive Values”, after which the “All” option will become available for selection.
