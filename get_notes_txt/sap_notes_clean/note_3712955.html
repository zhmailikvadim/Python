Symptom

You submit items to the collection agency (transaction FP03D). On the selection screen, you restrict the selection to one or more company codes. The system displays the error message "No authorization to submit collection agency &1 company code &2" (ML_CAI_MESSAGE 090) for a company code that you have not selected on the selection screen.

Other Terms

FP-COLLAG, RFKKCOL2, FP03D, 090(ml_cai_message)

Reason and Prerequisites

Cause:
SAP Note 3424156

Prerequisite:
The problem may occur if both of the following conditions are met:

  • Items from several company codes have been released for collection. However, in the selection options, a restriction was made to a subset of these company codes.
  • You only have authorization for the subset of these company codes to be selected.

Solution

Implement the attached program corrections or import the relevant support package.
