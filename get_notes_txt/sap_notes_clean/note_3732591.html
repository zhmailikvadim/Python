Symptom

  • How to check and Where to find the phase of your CBC workspace?
  • How to change your CBC workspace phase between "Scope and Organizational Structure" phase and "Product-Specific Configuration" phase?

Environment

  • SAP Central Business Configuration
  • SAP S/4HANA Cloud Public Edition

Reproducing the Issue

Cause

Resolution

  • In the navigation pane, choose Settings. The Settings view appears. You can check the phase of your CBC workspace.

  • In order to change CBC workspace phase from "Product-Specific Configuration" to  "Scope and Organizational Structure" phase, click edit button in Define Scope App or Set Up Organizational Structure App.
    1. SAP Help Portal- Editing Scope
    2. SAP Help Portal- Editing Organizational Structure
  • In order to change CBC workspace phase from "Scope and Organizational Structure" phase to "Product-Specific Configuration" phase, the following Scope and Organizational Structure phase activities must be completed.
    1. Assign Deployment Target
    2. Define Scope
    3. Specify Primary Finance Settings
    4. Set Up Organizational Structure
    5. Deploy Scope and Organizational Structure. KBA 3522280 - CBC: Deploy button of "Deploy Scope and Organizational Structure" activity is greyed out
