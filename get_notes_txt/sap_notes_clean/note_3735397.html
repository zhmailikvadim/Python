Symptom

On the Learning Admin page of SAP SuccessFactors Learning, "End Date" filter for classes is not visible. 

Environment

SAP SuccessFactors Learning

Reproducing the Issue

  • Navigate to Learning Administration >
  • Navigate to Learning Activities>Classes
  • Notice that only "Start Date After" and "Start Date Before" filters are present
  • "End Date Before" or "End Date After" filters are not there

Cause

Working as expected.

Resolution

This is working as expected. This configuration is hard-coded and cannot be changed. As of now, we do not have the End date filter for this page.

If you would like to see the option of End Date in Class entity- Add/Remove Criteria in future LMS releases, you have the option to raise an enhancement request to highlight this to product management -


2090228 - How to submit enhancement requests for SAP Success Factors products 

Enhancements are customer-facing, and only the customer can create and vote for enhancements.
