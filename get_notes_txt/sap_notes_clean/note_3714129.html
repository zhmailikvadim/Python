Symptom

  1. Creation of ERP Sales Orders from Sales Quotes in status Won is no longer possible; the system incorrectly blocks the quote as "Completed".
  2. Error message: "You cannot edit the sales quote. It is completed."

Environment

  • SAP Sales Cloud and SAP Service Cloud Version 2
 



Reproducing the Issue

  1. Login to test tenant
  2. Navigate to Sales Quote from the Main menu
  3. Select the Sales Quote for which the status is "Won".
  4. Click Create ERP Sales Order.
  5. Observe the system message: "You cannot edit the sales quote. It is completed."

Cause

The system blocks ERP Sales Order creation because quotes marked as 'Won' are locked by the lifecycle, preventing edits.

Resolution

This is the expected behavior.

  1. Change the quote's progress status to "In Process".
  2. Try to create the ERP Sales Order again.
  3. Alternatively, as a workaround, you can use a pre-hook when setting the quote status to 'Won' with any custom reason, and have the lifecycle revert to 'In Process.' This may help achieve the desired outcome.
