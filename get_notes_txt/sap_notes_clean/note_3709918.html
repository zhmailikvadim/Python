Symptom

  • Error message "Fin. Statement Version * does not exist for chart of accnts *" (FINS_FI_CLS052) encountered while assigning Accounting Principles to a Financial Statement Version (FSV).
  • The referenced Financial Statement Version does not exist for the chart of accounts.
  • Error message "FINS_FI_HRRP001" appears when attempting to update the Financial Statement Version in transaction OB58.

Environment

  • Financial Accounting (FI)
  • SAP enhancement package for SAP ERP
  • SAP ERP Central Component
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA
  • SAP S/4HANA Finance

Reproducing the Issue

  1. Assign Accounting Principles to a Financial Statement Version (FSV).
  2. Observe the error message: "Fin. Statement Version * does not exist for chart of accnts *" (Message No. FINS_FI_CLS052).
  3. Attempt to update the Financial Statement Version in transaction OB58.
  4. Error message "FINS_FI_HRRP001" is displayed.

Cause

The Financial Statement Version does not exist for the specified chart of accounts.

Resolution

  1. Implement the following SAP Notes:
    SAP Note 3700186 - HRRP: Improve exception handling during replication.
    SAP Note 3700059 - HRRP: Improve the program FINS_HRRP_CLEANUP.
  2. Execute the program FINS_HRRP_CLEANUP in all systems where the Financial Statement Version cannot be activated or transported.

See Also

  • Refer to: SAP Note 3700186 - HRRP: Improve exception handling during replication.
  • Refer to: SAP Note 3700059 - HRRP: Improve the program FINS_HRRP_CLEANUP.
