Symptom

************************************************************

WARNING: This is a CA-FL (Field Logistics) specific note. If you DON'T have

Field Logistics activated in your system, this note does not apply to you.

************************************************************

In a Project Stock (Category Q) scenario, the WBS element fails to populate in the Initiate Returns by Product application when the return is initiated through the Initiate Returns by Order application. As a result, the Return STO draft is missing the WBS element and the Account Assignment Category 'Q'.

Other Terms

CA-FL, CA-FL-RET, Initiate Returns by Order(F4682), Initaite Returns by Product app(F4252), Project Stock, WBS element, SAP S/4HANA OP 2025 FPS0, SAP S/4HANA OP 2025 FPS01

Reason and Prerequisites

Program Error

Solution

 Kindly implement the correction instructions attached with this note.
