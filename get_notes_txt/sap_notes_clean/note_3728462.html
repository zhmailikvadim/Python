Symptom

You use the budgeting workbench (transaction code FMBB). You observe that when selecting a field in the screen header or in the ALV Tab Strip The focus is lost.

You may select the field either by entering an input value or selecting the field value from the dropdown list and press enter, the focus on the current field, is lost and jumps to the top of the field in the header screen.

Other Terms

Usability improvement, set focus on field selection

Reason and Prerequisites

Missing Set and Get on the cursor position on selected field, code enhancement

Solution

After applying the following correction, the focus on the selected field is kept after pressing enter.
