Symptom

Error message "An error occurred in the connection with the service. Try again later." (Message nr.: ID_FI_OVS002) while using App Online Validation for Multiple Partners (App ID: OVF_MASS_CHECK)

Environment

SAP S/4HANA Cloud Public Edition 

Cause

Resolution

1. First of all, it is recommended to clear browser history and cache.

2. It is need to check whether the RFC destination XXX is correct.

-> If not, a new RFC destination should be created and RFC code have to updated.

3. Try again later.

Keywords

validation error, ID_FI_OVS002, RFC, ovf, Online Validation for Multiple Partners,
