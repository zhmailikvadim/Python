Symptom

ERP triggers outbound delivery confirmations for deliveries that are not relevant for Advanced Shipping and Receiving (AdvS&R) processing in embedded Extended Warehouse Management (EWM) scenarios.

This creates unnecessary update tasks and RFC calls, which increases system load and causes redundant processing in EWM.

Other Terms

Embedded EWM, Decentralized EWM, ASR, /SCWM/OUTB_DLV_CONF_CENTRAL

Solution

Implement this SAP Note or the relevant feature package.

This SAP Note improves the ERP outbound delivery confirmation logic in the following ways:

  1. Detects whether the target system is the same as the local system in embedded EWM scenarios
  2. Triggers confirmations only for ASR-relevant deliveries in embedded scenarios
  3. Keeps the confirmation behavior unchanged for decentralized scenarios
  4. The sender logical system is determined explicitly within the method and is no longer taken from the TECHN_CONTROL structure
