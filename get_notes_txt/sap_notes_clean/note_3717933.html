Symptom

The (WAVE-based) process "Enter First Pension Payment" (BSC_STL_PR_R_FIRST_PAYMENT) cannot be executed for a construction savings contract in the status "Pension Phase" (090). The system issues the error message "Status 090 is not allowed for this operation in the building savings contract".

Reason and Prerequisites

The preliminary check "Val.: Check for Status "Active" (010, or 020)" (BSC_TK_CHECK_BSC_ACTIVE) has the effect that only construction savings contracts with the status "Active, ER Open" (010) or "Active, ER Paid" (020) are processed by the process.

Solution

The preliminary check mentioned above is replaced by the new preliminary check "Check for Status "Active" or "Annuity Phase" (BSC_TK_CHECK_BSC_ACTIVE_OR_PENSION with implementation /IBS/CLBSC_TK_CHECK_BSC_ACTPEN), which allows the construction savings contract status 090 in addition to the construction savings contract statuses 010 and 020.
