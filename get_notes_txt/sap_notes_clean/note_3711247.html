Symptom

This SAP Note contains an enhancement of the RAP business object /MSG/R_TM_TREATY ("Read Only"), which is required as a technical pre-implementation step for further development.

The RAP business object /MSG/R_TM_TREATY contains application data that is relevant for the assignment of treaties to a loss. The main application is to enable the consumer of the business object to read contract data and generate input helps for contracts.

Other Terms

Web Service

Remote API

Reason and Prerequisites

Retrieval of the application data that is relevant for the assignment of treaties to a loss.

Solution

Troubleshooting description:
The RAP business object /MSG/R_TM_TREATY (Read Only) has been enhanced with the following entities:

  • TreatyPeriod
