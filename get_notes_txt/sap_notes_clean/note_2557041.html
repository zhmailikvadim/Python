Symptom

Transaction MB5B and transaction MB51 show different stock value from transaction J3RFLVMOBVED/J3RFLVMOBVEDH for the same material.

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Environment

  • SAP ERP 6.0 and higher
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4 HANA

Reproducing the Issue

1, Run transaction MB51.

and result will be:

2, Run transaction MB5B.

as result:

3, Run transaction J3RFLVMOBVED with the same data:

and the result will be empty:

Cause

Transaction J3RFLVMOBVED displays only valuated stock and this report can work only with transaction MB5B.

Resolution

  1. Do not use transaction MB51 to check stock for transaction J3RFLVMOBVED.
  2. Use transaction MB5B only with flag "Valuated stock" to get the correct result. The same recommendation is valid for transaction J3RFLVMOBVEDH.
