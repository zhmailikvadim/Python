Symptom

When importing manual postings, the Subledger Journal Entries are persisted with out a release step, if no workflow is derived. If an import contains documents for multiple legal entities, the persistence of the Subledger Journal Entries can fail, while the Preliminary S/L Journal Entry are persisted.

Other Terms

Preliminary S/L Journal Entries use Result Type _S_PSLPD

Subledger Journal Entries use Result Type _S_SLPD

Reason and Prerequisites

There is a program error.

Solution

Apply the correction instruction or the respective support package.
