Symptom

  • In SAP Data Services Management Console, only "View WSDL" option is visible at the bottom of the Web Service Status page.
  • The "View WADL" option is not displayed.

Environment

SAP Data Services 4.3 SP02

Reproducing the Issue

  1. Log in to Data Services Management Console.
  2. Navigate to Administrator tab.
  3. Select Web Services option.
  4. Open Web Services Status.
  5. Only the "View WSDL" option is available.

Cause

  • The "View WADL" button was introduced in SAP Data Services 4.3 SP03. 
  • In version 4.3 SP02:
    • It is not configurable.
    • It cannot be enabled manually. 
  • This behavior is version-specific and is not caused by a misconfiguration or a product defect. 

Resolution

To use the "View WADL" functionality, upgrade SAP Data Services to version 4.3 SP03 or higher.
