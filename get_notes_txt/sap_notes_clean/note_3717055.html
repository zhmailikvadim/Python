Symptom

You use SAP ACM and the document category N3KVMST10 in the document category version 8 for the creation of lab referral certificates.

When you create a document for an ASV case, the system does not preset the BSNR but the ASV team number despite the assignment of a BSNR to the determined physician.

Other Terms

N3KVMST10

Reason and Prerequisites

Program error

Solution

The following applies to ASV cases:

  • If a BSNR is assigned to the determined physician, this should be preset.
  • If no BSNR is assigned to the determined physician, the ASV team number is preset.

The "ASV Team" function button is only active if the indicator for ASV case is set.

Import the specified Support Packages to solve the problem. You can implement collective correction 3716853 in advance.
