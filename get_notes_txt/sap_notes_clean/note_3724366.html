Symptom

Generate credentials for the procurement data warehouse tenant from the Identity Authentication service. These credentials are used to enable communication between SAP Ariba Category Management and the procurement data warehouse API.

NOTE: Perform these steps only when explicitly requested by SAP, as they are not required for the standard configuration flow.

Resolution

Prerequisites:
You must have provisioned SAP Spend Control Tower from SAP for Me.

Procedure:

  1. Access the SAP Cloud Identity Services UI.
  2. Click the Applications & Resources dropdown menu and select Applications.
  3. Locate and select the Procurement Data Warehouse DAC in the Applications panel.
  4. Click Provided APIs in the Application APIs section.
  5. Click Create in the Dependent Applications section.
  6. The Create Dependency dialog box will appear, please proceed as follows:
  • Dependency Name: Enter a dependency system name. For example, catman-pdw-api.
  • API: Select sap.catman.
  • Consuming Application: Leave the default New Application (default) option.
  • Application Name: Enter the application name. For example, catman-pdw.
  • Generate Secret: Enable this checkbox.
  • Click Create.

The Summary dialog box appears. Note down the corresponding values of the clientsecret, clientid, token endpoint, and resources fields and share them with the SAP support representative.
