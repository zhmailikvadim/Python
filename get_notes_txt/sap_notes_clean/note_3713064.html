Symptom

During Posting, a short dump / Runtime error CONVT_OVERFLOW is raised with exception CX_SY_CONVERSION_OVERFLOW.
This happens for Concur Expense Report with more than 256 tax line items.

Other Terms

Concur Integration, Concur Financial Posting, Concur Expense Report, Multiple Tax line items

Reason and Prerequisites

 Coding error

Solution

Apply the correction below or install next available support package for S/4Hana OnPremise:
2022 SP8

2023 SP6

2025 SP2
