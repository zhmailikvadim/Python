Symptom

The error message displayed is: FVG003  "SYST: Period & is not valid in FY variant &." when change the depreciation key or create assets.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Attempt to choose/change the depreciation key in asset master data.
  2. Observe the error message: "SYST: Period 000 is not valid in FY variant &."

Cause

The period control being used has not been defined in configuration 102712 - Maintain Fiscal Year Variants. 

Resolution

Check whether the period controls included in the depreciation key have been correctly maintained in configuration 102712 — Maintain Fiscal Year Variants.

If not, maintain all period controls in configuration 102712 — Maintain Fiscal Year Variants.

Keywords

asset accounting, depreciation key, sul6, period control, maintain fiscal year variants, syst error, period 000, fy variant  sap s/4hana cloud。
