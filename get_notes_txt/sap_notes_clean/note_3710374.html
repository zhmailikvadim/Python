Symptom

A runtime error UNCAUGHT_EXCEPTION with CX_PAK_INVALID_SUPER_PACKAGE or CX_PAK_INVALID_DLVUNIT occurs in the following cases:

  • When opening an ABAP development package that references a non-existing super package
  • When working with an ABAP development package that has an invalid transport layer or an invalid software component

Example dump:

Category               ABAP programming error
Runtime Errors         UNCAUGHT_EXCEPTION
Except.                CX_PAK_INVALID_DLVUNIT
ABAP: Program          SAPLPB_PACKAGE
Application Component  BC-DWB-TOO-PAK

Environment

  • SAP_BASIS 816
  • SAP_BASIS 916

Cause

Resolution

Implement the following SAP Note:

3635170 - PAK: Package can not be opened if a invalid super package is defined

See Also

SAP Note 3635170 - PAK: Package can not be opened if a invalid super package is defined
