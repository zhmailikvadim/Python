Symptom

In case of EAM Simplicity, you try to create a new Technical Object with another Technical Object as reference. Error message RECABC 050 ‘Subfunction &1 (&2) not active’ occurs. Although it is possible to create the Technical Object, even when the error message occurs. Nevertheless, the error message is not necessary, because Subfunction SIRE (RE-FX) is not activated in the system.

Other Terms

CL_EAMS_UI_EI_TECOBJ_FRW_APPCC, SET_VISIBILITY_REAL_ESTATE_TAB, SIRE, Technical Object, Equipment, Functional Location, EAMS, RECABC, 050, 050(RECABC), WDA, RECABC050.

Reason and Prerequisites

Program error.

Solution

Implement the program corrections.
