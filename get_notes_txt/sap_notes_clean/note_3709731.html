Symptom

In Customer Data Platform (CDP), enabled actions from cloud storage destination applications such as AWS S3, Azure Blob, Google Cloud are not available in CX flow setup. 

Environment

  • SAP Customer Data Platform
  • CX Flows
  • Actions

Reproducing the Issue

  1. Create an action in a destination application of type AWS-S3.
  2. Enable the created action in the destination application.
  3. Navigate to CX Flow setup and attempt to add the enabled action from the AWS-S3 destination application in the Actions Tab.
  4. Observe that the enabled action from the AWS-S3 destination application does not appear in the list of available actions.

Cause

Cloud storage actions (like AWS S3, Google Cloud, Google Cloud) are batch actions and are not supported in the CX Flow.

Resolution

The behavior is correct in the current design.
