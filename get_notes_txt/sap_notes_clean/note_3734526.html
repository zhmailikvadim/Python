Symptom

When creating an ODP connection connection the following alert is shown:

The chosen source system name is identical to the logical system name of the source system. If
you use this name, you cannot connect the system
using BW Service API technology, which must
use the logical system name.

Environment

SAP NetWeaver

Resolution

  • You can confirm and use the same name then the BW Service API will no longer work.
    Or
  • Use a different name for the ODP connection to maintain both.

IMPORTANT: When data is distributed between different systems, each system within a network has to be clearly identifiable. The “logical system” deals with this issue. A logical system is an application system in which the applications work together on a common data basis. In SAP terms, the logical system is a client. Since the logical system name is used to identify a system uniquely within the network, two systems cannot have the same name if they are connected to each other as BW, BW/4HANA systems or as source systems, or if there are plans to connect them in any way.

Keywords

ODP, connection, name, same, equal, SAPI
