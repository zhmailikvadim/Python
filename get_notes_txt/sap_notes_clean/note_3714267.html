Symptom

There's a requirement to use CDS Views related to In-House Repair using ADT (ABAP Development Tools) for different business needs, e.g. linking data from customer returns to in-house repair orders.

Environment

SAP S/4HANA Cloud Public Edition

Cause

CDS Views related to In-House Repair, e.g. I_InHouseRepair or I_InHouseRepairItem, are not released for Developer Extensibility, so it's not possible to create a data definition in developer extensibility using those standard CDS Views.

Resolution

It's completely understandable how the possibility of having those CDS Views available for developer extensibility would be of great use for specific business needs, so please consider creating a feature request for this business requirement. This will inform the responsible developers as to what additional functionality should be considered for future releases.
