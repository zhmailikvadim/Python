Symptom

Error occurs when attempting to activate "CWB_T_NOTE_DWLD_HTML_RESP" or "CWB_T_NOTE_DWLD_DS_HTML_RESP" in SE11:

TTYP CWB_T_NOTE_DWLD_HTML_RESP was not activated
Associated type for components LOW and HIGH not specified
Data type ' ' does not exist
Table type CWB_T_NOTE_DWLD_HTML_RESP is inconsistent

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental" 

Environment

  • Note Assistant tool (SNOTE)
  • SAP NetWeaver AS ABAP

Reproducing the Issue

  1. Enter transaction "SE11"
  2. Create new table types "CWB_T_NOTE_DWLD_HTML_RESP" or "CWB_T_NOTE_DWLD_DS_HTML_RESP"
  3. Save and activate it

Cause

The data type of object "CWB_T_NOTE_DWLD_HTML_RESP" or "CWB_T_NOTE_DWLD_DS_HTML_RESP" was incorrectly created as "Ranges Table type" instead of "Table Type".

Resolution

  1. Perform the manual Pre-Implement steps of SAP Note 3296107 - SNOTE: Improved handling of Unidentified IP Address error during download.
  2. Create the data type "CWB_T_NOTE_DWLD_HTML_RESP" or "CWB_T_NOTE_DWLD_DS_HTML_RESP" in SE11 correctly as "Table Type".
