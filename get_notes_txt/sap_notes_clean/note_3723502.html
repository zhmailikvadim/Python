Symptom

The "End Date" is correctly passed to CRAP_WORKCENTER_CREATE. But the FM still raises error "Enter End Date", message No. CR283, which can be checked in SLG1.

Environment

  • SAP S/4HANA, on-premise
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

SE37 single test for CRAP_WORKCENTER_CREATE or call the FM in custom programs.

Cause

Checkbox controls are set as Required in OPFA. For example, field "Reference indicator" (PAD011-LARN1_REF) is configured as Required.

Resolution

The current CRAP_WORKCENTER_CREATE FM can only handle the "Required" flag for standard input fields. Checkbox controls cannot be enforced as required due to technical limitations of this function. Please revert the checkbox field configuration from "Required" to "Input" to solve the issue.

Keywords

CRAP_WORKCENTER_CREATE, Required, CR283, PAD011-LARN1_REF, Reference indicator
