Symptom

  • You require the ENVID (envelope identifier) to be set when sending messages from the SAP system
  • The ENVID esmtp-keyword of the SMTP MAIL command is used to specify an "envelope identifier" to be transmitted along with the message and included in any DSNs issued for any of the recipients named in this SMTP transaction. The purpose of the envelope identifier is to allow the sender of a message to identify the transaction for which the DSN was issued.

Environment

  • SAP NetWeaver
  • Release Independent

Resolution

  1. The ENVID attribute is already present when a message is sent from the SAP system.
  2. Please note that the ENVID attribute is not present in the message header (MIME) when sending from SAP system, as per RFC standard 3461 ENVID should be set in MAIL command. 
  3. If you capture the SAPconnect trace (1639935) and/or SMICM trace (2351619) you will see that the ENVID attribute is present
