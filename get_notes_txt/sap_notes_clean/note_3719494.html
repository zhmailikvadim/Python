Symptom

Application job template Order-Based Planning: Delete Planned Receipts is used to delete planning elements. What is function of Receipts in Freeze Horizon parameter?

Environment

SAP Integrated Business Planning

Reproducing the Issue

Run Order-Based Planning: Delete Planned Receipts

Resolution

Receipts in Freeze Horizon: Planned elements that are within a freeze horizon for distribution or production cannot be changed by a planning run, but they can be changed by an application job in interactive planning. Select the checkbox if you want to apply the application job to planned receipts within a freeze horizon.

For example, there is a deployment stock transfer requisition created inside the freeze horizon, with the parameter unticked, this deployment stock transfer requisition will not be deleted upon the job execution. If the parameter ticked, the deployment stock transfer requisition within the freeze horizon will also be deleted.
