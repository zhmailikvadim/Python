Symptom

 An error "Navigation to this application is not supported" occurs when navigating from the "Manage Debit Memo Requests" app.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Open the "Manage Debit Memo Requests" app.
  2. Select any debit memo request in the list.
  3. Error appears.

Cause

The user doesn't have authorization to access to the "Manage Debit Memo Requests" app.

Resolution

  1. Check the required Business Catalog in the Fiori App Library.
  2. Check the IMPLEMENTATION INFORMATION tab and review the Business Catalogs section.
  3. Verify the user's existing Business Catalogs using the "IAM Information System" app.
  4. For the "Manage Debit Memo Requests" app, assign the Business Catalog SAP_SD_BC_DMR_PROC_MC/SAP_SD_BC_SO_ADV_PROC/SAP_SD_BC_SO_PROC_INTCO  to the user.
