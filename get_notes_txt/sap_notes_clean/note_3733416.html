Symptom

During workload generation for the HTML fragment search in report RCMX_TOOLS_SXS_MIGR_HTML_FRAG, the system ends with a short dump when SXS building blocks reference control recipe destinations that no longer exist. The dump occurs while the system determines migration status for deleted XStep roots (method SET_MIGR_STAT_FOR_DEL_XS_ROOT). Workload generation does not finish.

Other Terms

RCMX_TOOLS_SXS_HTML_MIGR_P01, SET_MIGR_STAT_FOR_DEL_XS_ROOT, CX_CMX_XS_EXCEPTION, IF_CMX_XS_STEP, IF_CMX_XS_PERSISTENT, IF_CMX_TOOLS_MIGR_WLIST_UTIL, SXS, XStep

Reason and Prerequisites

The reason is a program error. Method SET_MIGR_STAT_FOR_DEL_XS_ROOT in include RCMX_TOOLS_SXS_HTML_MIGR_P01 does not catch CX_CMX_XS_EXCEPTION when referenced control recipe destinations are missing.

Solution

Install this SAP Note. The correction adds exception handling in RCMX_TOOLS_SXS_HTML_MIGR_P01 so workload generation continues and marks the affected entry as migration failed.
