Symptom

When copying Easy Cost Plan version 3 to version 2, a runtime error occurs.

ABAP programming error
MESSAGE_TYPE_X
SAPLECP_EHP3
PS-COS-PLN-CAL

Message class …… CN_ECP_PS
Number.... 021
Variable 1 INTERNAL ERROR

The termination occurred in the ABAP program or include"SAPLECP_EHP3", in "ECP_COPY_ECP_CO_OBJECT". The main program was "SAPLCNPB_M"

Environment

  • SAP ERP Project System (PS)
  • SAP R/3
  • SAP R/3 Enterprise
  • SAP ERP Central Component
  • SAP ERP
  • SAP Enhancement package for SAP ERP
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA, on-premise
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Copy Easy Cost Plan version 3 to version 2.
  2. Observe the runtime error.

Cause

Program error

This error occurs only if:

  • the switch PS_SFWS_SC1 is activated and
  • the structure of the project was not expanded before copying

Resolution

Refer to SAP Note  3562420 - MECP: A short dump occurs when copying a plan version
