Symptom

While adding a competency (that has behaviours mapped to it) to a form's competency section with the Capability Picker the competency is not added to the section when the Latest Form Experience is enabled.

Environment

SAP SuccessFactors Performance Management

SAP SuccessFactors Job Profile Builder

Reproducing the Issue

  1. Go to a form's competency section
  2. Open the Capability Picker (Add New Competency)
  3. Select a competency from the list with a behaviour mapped
  4. Click on Add
  5. Competency is not added to the section

Cause

Known issue.

Resolution

Engineering is working towards the resolution of this issue. The fix is currently planned for the next 1H 2026 release. Please follow this KBA for updates and confirmation of the resolution timeline.

Workaround:

Disable the Latest Form Experience while adding the competencies to the form. Once they are successfully added, the Latest Form Experience can be enabled again.
