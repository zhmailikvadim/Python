Symptom

Local secure store is not detected in resident hdblcm when you try to uninstall LSS or display the installed components.





However, it is up and running.



Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

SAP HANA Platform 2.0

Cause

The manifest file in directory /lss/shared/<SID> is corrupted or deleted inadvertently.

Resolution

Copy the manifest file from the HANA installation media of same version to /lss/shared/<SID>. The manifest file is located in ../SAP_HANA_DATABASE/lss/packages after extraction of the HANA installation media.

Keywords

LSS is not detected, LSS is not displayed, LSS missing in hdblcm, LSS manifest file, manifest file of local secure store
