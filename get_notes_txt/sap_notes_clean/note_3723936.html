Symptom

  • When a user self-assigns a program in SAP SuccessFactors Learning, the program appears in the "Invest in Myself" section on the home page.
  • If an admin subsequently assigns the same program as required to the user, the program does not move to the "Required Learning" section on the home page.

Environment

SAP SuccessFactors Learning

Cause

The system does not allow a program to be assigned to a user multiple times. When a program is self-assigned, any subsequent assignment by an admin is disregarded, and the original self-assigned status is maintained.

Resolution

  1. Remove the self-assigned program from the user's Learning plan >
  2. Assign the program as required through the admin interface >
  3. Verify that the program appears in the "Required Learning" section on the user's home page >

Keywords
