Symptom

When attempting to load accounting view data of a material master using the data migration cockpit via product master object , an error message appears stating: "Specify Price Control."(M3834)

Environment

  • SAP ERP Logistics (LO)
  • SAP R/3
  • SAP R/3 Enterprise
  • SAP ERP Central Component
  • SAP ERP
  • SAP Enhancement package for SAP ERP
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA Cloud Private Edition 
  • SAP S/4HANA

Resolution

Implement SAP Note: 3372347 - Reference Handling For Products 

Keywords

Specify Price Control, M3834, Migration Cockpit, object
