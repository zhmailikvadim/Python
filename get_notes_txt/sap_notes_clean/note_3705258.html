Symptom

During the SPAU activation phase, syntax errors occur in the Vistex table /IRM/TIPXXATCC (BB, CB, CR, PR, SI, BR). The issue is caused by the fields BOART_SO and BOART_TA, which are defined with domains that differ from the domains of the corresponding fields in the assigned foreign key table.

This inconsistency leads to DDIC activation errors during SPAU processing.
This note corrects the domain definitions of the affected fields, thereby resolving the syntax and activation issues.

Other Terms

DDIC activation issue, SPAU, Vistex table, syntax error

Reason and Prerequisites

This is caused by program error

SAP Note#3705293 is available for V/4 Release
