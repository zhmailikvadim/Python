Symptom

This KBA provides step-by-step guidance on how to identify workflow job details such as: 

  • Name of the workflow job, 
  • Program name that is run by the job,
  • Used variant,
  • Used UserID,
  • Possible error messages in the job log.

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

  • SAP Business Workflow
  • SAP NetWeaver
  • SAP Web Application Server for SAP S/4 HANA
  • ABAP PLATFORM - Application Server ABAP

Resolution

  1. Start transaction SM37,
  2. Enter the job name in the "Job Name" field. Starting from with S/4HANA On-Premise 1709, workflow jobs start with "SAP_WORKFLOW_*",
  3. Specify the status and further search criteria if needed,



  4. Execute,
  5. Double-click the needed job,



  6. Click Job Log,



  7. The corresponding variant, job log, userID and additional details are shown.
