Symptom

Termination page displays 0 balance for all leave types when no future-year time account exists.

Environment

SAP SuccessFactors Employee Central

Reproducing the Issue

  1. Ensure an employee has available leave balances.
  2. Set the termination/last working day to fall in the following year.
  3. Confirm that no future-year time account exists.
  4. Check the Termination page for leave balance, which displays 0 days available for all leave types.

Cause

  • The issue occurs due to the absence of a future-year time account or the bookable period of the current time account not covering the termination date in the following year.

Resolution

  1. Create a future-year time account if the termination date falls in the next calendar year.
  2. Adjust the bookable period of the current time account to cover the termination date.

Keywords

leave balance, termination page, next calendar year, time off workbench, future-year time account, validity period, leave types, SAP SuccessFactors Employee Central
