Symptom

You want to remove the company code assignment for a profit center. The profit center is assigned to a material. Message KM588 is sent only as warning and the company code change can be saved by confirming the warning.

Other Terms

LRKPMHZ3, verify_no_assignment

Reason and Prerequisites

Program error

Solution

Implement the attached coding correction or the respective support package in your system.

Afterwards, the removal of the company code assignment is not possible if the profit center is still assigned to a company code. This will prevent inconsistencies between the profit center and material master data.
