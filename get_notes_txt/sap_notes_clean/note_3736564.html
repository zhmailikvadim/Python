Symptom

BusinessPartnerERPReplicationIn Web Service message has error 'Solve Configuration Issue in Code List Mapping for field CustomerPriceListTypeCode with value XXX, where XXX refers to a CustomerPriceListType External Code Value.

Environment

  • SAP Cloud for Customer core applications
  • Integration of C4C with ERP

Reproducing the Issue

  1. Go to Administrator > General Settings > Web Service Message Monitoring.
  2. Locate the incoming BusinessPartnerERPReplicationIn message from ERP that is in error.
  3. Open the message details and review the error text: "Solve Configuration Issue in Code List Mapping for field CustomerPriceListTypeCode with value XXX".

Cause

  • The external code "CC" for CustomerPriceListTypeCode is not maintained in the Business Configuration Overview.

Resolution

  1. Open Business Configuration and navigate to the Overview.
  2. Search and open Code List Mapping for Integration with External Applications and Solutions fine-tuning activity. 
  3. Find and open Code List Mapping for the mapping group defined in Code List Mapping of the 'Business Partner Replication from SAP ERP' Communication Arragement.
  4. Locate the CustomerPriceListTypeCode mapping.
  5. Add a mapping entry for external code XXX to the appropriate internal code.
  6. Save and activate the code list mapping changes.
  7. Reprocess the failed web service message or re-trigger the customer replication from ERP.
