Symptom

You are using Production Planning and Detailed Scheduling (PP/DS) for SAP S/4HANA or SAP ERP Add-on for Embedded Production Planning and Detailed Scheduling (PP/DS). This note involves performance enhancement of the delta order transfer report and provides user an option to restrict the type of order to be transferred.

Other Terms

PP/DS

PPDS

Delta order transfer report

SAP ERP Add-on for Embedded PP/DS

/SAPAPO/PPDS_DELTA_ORD_TRANS

Reason and Prerequisites

This involves performance enhancement of the delta order transfer report /SAPAPO/PPDS_DELTA_ORD_TRANS. And user is given a new feature where if it types ORD_TYP in ok_code,it will get an option to restrict the type orders to be transferred

Solution

As general rule, SAP recommends that you install a solution by applying a Support Package. However, if you need to install a solution earlier, use the Note Assistant to implement the correction instruction.
