Symptom

You use KIM shipping for the corresponding TI applications in SAP ACM. The KIM access key that can be displayed in amasys as of Version 3.31 or higher cannot be defined in Customizing.

Other Terms

KIM, Digital Samples

Reason and Prerequisites

Technical enhancement

Note the following prerequisite for using the KIM access key: amasys 3.31 or higher is installed in your system.

Solution

The reference IMG entry for "KIM: Configure Shipping" has been enhanced with the following two columns:

KIM access key: If you choose "Enter", you can enter the KIM access key (from amasys).

Status of KIM Access Key: In this column, the system uses icons to indicate whether you have entered a valid value.

Import the specified Support Packages to solve the problem. Refer to SAP Note 3711709.
