Symptom

On the Analytics Portal, users cannot download or view PDF reports. The following error message is displayed:

Unexpected server response (500) while retrieving PDF "https://xxxx:40000/Portal/api/histories/16/preview/pdf".

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

SAP Business One, version for SAP HANA

Reproducing the Issue

Prerequisites:

The company has been initialized in the Analytics Administration Console.

Steps to reproduce:

Access the Analytical Portal through https://ServerName:Port/Portal. 

Expected result:

PDF reports are downloaded or viewed successfully without any issues.

Actual result:

Fail to download or view PDF reports, and the following error message is displayed:

Unexpected server response (500) while retrieving PDF "https://xxxx:40000/Portal/api/histories/16/preview/pdf".

The following error is found in the Portal.log file located under \var\log\SAPBusinessOne\AnalyticsPlatform\Portal:

"message":"Tried to read data but the maximum length for this record type is 100,000,000.If the file is not corrupt and not large, please open an issue on bugzilla to request increasing the maximum allowable size for this record type.You can set a higher override value with IOUtils.setByteArrayMaxOverride()","stack_trace":"o.a.p.u.RecordFormatException: Tried to read data but the maximum length for this record type is 100,000,000.\nIf the file is not corrupt and not large, please open an issue on bugzilla to request \nincreasing the maximum allowable size for this record type.

Cause

This is a system limitation.

When the report data exceeds a certain limit, both the PDF preview and download fail with the error message 'RecordFormatException'.

Resolution

Workaround:

Download the report in an alternative format to view it.

Keywords

PDF report, download, run, 500, limit
