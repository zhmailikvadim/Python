Symptom

Forward navigation from any source code to an ABAP Dictionary object does not work in ADT. On using forward navigation (default shortcut: Ctrl + left mouse-button) nothing happens. 

Environment

ABAP Development Tools in Eclipse with a backend system:

  • SAP S/4HANA
  • SAP S4/HANA Cloud Public Edition
  • SAP NetWeaver 7.4 or newer

Reproducing the Issue

Try to forward navigate to any ABAP Dictionary object using ADT.

Cause

The target of the forward navigation is inactive. Forward navigation to DDIC objects (tables, CDS entities, etc.) only works if the object is active.

Resolution

  1. Open the target object (By default with shortcut Ctrl + Shift +  A)
  2. Activate the object (By default with shortcut Ctrl + F3)
  3. The forward navigation will now work as expected.
