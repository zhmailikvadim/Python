Symptom

Business requires further information about a SAP System Database

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

A Content Repository of SAP System Database

Resolution

  1. A SAP System Database repository will store data in a table instead of a content server as we can see in the screenshot below:

    In this example, the table used is "EXAMPLE_TABLE".

  2. The fields for HTTP and HTTPS ports are not available, this is because SAP System Database repositories use the ports set in transaction code "SMICM"
  3. The services in transaction SMICM are the same services the SAP System Database repository will use, if some port is set to "0" this can cause issues, refer to the KBA below for further details:
    3217247 - Content Repository XX Cannot be Accessed by HTTP

  4. For user authentication, the user is set in transaction code "SICF" in a service "contentserver", if set incorrectly, errors as such "401 Unauthorized" can happen, for further details to set properly, refer to the KBA:
    2827260 - Configuration steps for /sap/bc/contentserver service
