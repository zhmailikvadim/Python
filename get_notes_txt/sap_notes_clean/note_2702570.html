Symptom

How do I edit the CDC Accounts schema using the REST API?

Environment

  • SAP Customer Data Cloud
  • REST API 

Resolution

The Accounts Schema can be changed by invoking the REST API endpoint of accounts.setSchema.

For more information on reading the schema, please see How do I view the Gigya Accounts schema using the REST API?

Keywords

  • REST API
  • Registration-as-a-Service (RaaS)
