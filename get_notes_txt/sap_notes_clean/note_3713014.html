Symptom

  • Sales Support Block is enabled under Sales Data for a specific combination of sales organization and distribution channel; however, it does not prevent the creation of sales orders and sales quotes.
  • No error message is displayed when creating a sales quote, while an error message is triggered when creating an opportunity.

Environment

  • SAP Sales Cloud Version 2
  • SAP Sales and Service Cloud Version 2

Reproducing the Issue

  1. Open the Navigation menu.
  2. Navigate to the Accounts view.
  3. Open an account where Sales Support Block is enabled.
  4. Attempt to create a sales quote – no error message is displayed.
  5. Attempt to create an opportunity – an error message is displayed.

Cause

This is expected system behavior.

Resolution

Sales Support Block does not apply to Sales Quotes and Sales Orders. To restrict these documents, use Order Block to prevent specific combinations of sales data.
