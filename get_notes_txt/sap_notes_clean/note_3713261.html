Symptom

A  "SAP NetWeaver" pop-up appears before displaying the payslip in SuccessFactors, requesting for selection of a trusted SAML2.0 identity provider from dropdown.

Environment

  • SAP SuccessFactors Employee Central integration to SAP Business Suite
  • SAP SuccessFactors Employee Central Integration with SAP Human Capital Management for SAP S/4HANA
  • Mashup / Pay Statement

Reproducing the Issue

  1. Logon SuccessFactors
  2. Access the payslip for an employee in SuccessFactors Employee Central 

Cause

The selection mode of Identity Provider Discovery: Common Domain Cookie (CDC) is set as 'Manual' in tcode SAML2.

Resolution

  1. Go to transaction SAML2.
  2. Navigate to LocalProvider -> Service Provider settings.
  3. Set the "Selection Mode" as "Automatic" for the Identity Provider Discovery: Common Domain Cookie (CDC).

Keywords

sap netweaver,  pop-up issue, identity provider discovery, common domain cookie, selection mode manual, selection mode automatic, IDP URL
