Symptom

The buyer is unable to add attachment to a Schedule

Resolution

  • The Buyer can add attachment on Fee Response, Fee Revision Response and Fee Holdback responses in approved or pending Approval status(es) using new option 'Add Attachments' from Actions menu available on Fee Response details page
  • This feature is not currently supported on Schedule, Event and Management Event

Keywords

SOW line item, attachment, Event, Schedule, Fee, Statement of Work
