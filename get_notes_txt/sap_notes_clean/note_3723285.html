Symptom

I am unable to find a document on Ariba UI or SAP Ariba Analytical Reports.

Resolution

One possible reason for not finding the document in Ariba is that it may be located in a different realm than the one you are searching in.

  • Make sure you are checking in the correct system (test or production)
  • Please note that the Buyer site could have more than one child realm.


To verify this information, you can add the Source System field to an analytical report, from the Edit tab.
The results will show you which system the document belongs to.
