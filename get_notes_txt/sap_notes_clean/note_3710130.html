Symptom

GST tax amount shall be calculated based on the Retail Sale Price (RSP) for notified goods, with the provision coming into effect from 1 February.
Tax amount = (Retail sale price X tax rate in % of applicable taxes) / (100+ sum of applicable tax rate) .

Step 4: Create routine to calculate below value

Tax amount = (Retail sale price X tax rate in % of applicable taxes) / (100+ sum of applicable tax rate)

Note: Here Condition types ZOIG, ZOCG, ZOSG , ZOUG are created with these names and same have been used in logic as well. Please adjust the condition types with whichever names you create.

Use App: Custom Logic  (Authorization: SAP_BR_EXTENSIBILITY_SPEC)

 

 

 

Sample code :

*----------------------------------------------------------------------*

* Custom Routine for Tobacco GST Calculation

* Formula: Tax Amount = (Retail Price × Tax Rate) / (100 + Tax Rate)

*----------------------------------------------------------------------*

 

* Mandatory step - Move importing parameters to changing parameters

MOVE-CORRESPONDING item_amounts TO item_result_amounts.

MOVE-CORRESPONDING item_attributes TO item_result_attributes.

MOVE-CORRESPONDING item_quantities TO item_result_quantities.

MOVE-CORRESPONDING prcg_element_attributes TO prcg_element_result_amounts.

MOVE-CORRESPONDING active_price_amounts TO active_price_result_amounts.

 

* Declare local variables

DATA: lv_tax_rate       TYPE p DECIMALS 2,

      lv_base_amount    TYPE p DECIMALS 2,

      lv_total_tax_rate TYPE p DECIMALS 2,

      lv_tax_amount     TYPE p DECIMALS 2.

 

TRY.

 

    lv_tax_rate = prcg_element_attributes-conditionratevalue.

 

    lv_base_amount = prcg_element_attributes-conditionbasevalue.

 

    CASE prcg_element_attributes-conditiontype.

 

      WHEN 'ZOIG'.

        lv_tax_amount = ( lv_base_amount * lv_tax_rate ) / ( 100 + lv_tax_rate ).

        prcg_element_result_amounts-conditionbasevalue = lv_base_amount - lv_tax_amount.

 

      WHEN 'ZOCG' OR 'ZOSG' OR 'ZOUG'.

        lv_total_tax_rate = lv_tax_rate * 2.

        lv_tax_amount = ( lv_base_amount * lv_tax_rate ) / ( 100 + lv_total_tax_rate ).

        prcg_element_result_amounts-conditionbasevalue = lv_base_amount - ( lv_tax_amount * 2 ) .

 

    ENDCASE.

 

    prcg_element_result_amounts-conditionamount = lv_tax_amount.

 

  CATCH cx_sy_conversion_error cx_sy_arithmetic_error INTO DATA(lx_error).

 

    prcg_element_result_amounts-conditioninactivereason = 'X'.

    prcg_element_result_amounts-conditionamount = 0.

 

    RAISE EXCEPTION TYPE cx_ble_runtime_error

      EXPORTING

        previous = lx_error.

 

ENDTRY.

 

Note: Here Condition types ZOIG, ZOCG, ZOSG , ZOUG are created with these names and same have been used in logic as well. Please adjust the condition types with whichever names you create in the WHEN statements.
