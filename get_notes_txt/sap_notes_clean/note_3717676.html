Symptom

During the replication process of employee data, the system triggers an error message with the "work schedule plan rule XXXX is not valid for this employee on XX.XX.XXXX"   (PAOCF_EC_SERVICES153).

Environment

  • SAP SuccessFactors Employee Central integration to SAP Business Suite
  • Employee Master Data Replication
  • Non-BIB Replication

Reproducing the Issue

  1. Run the replication from Employee Central
  2. Check the logs in SLG1
  3. Message error is displayed

Cause

  • The error occurs due to missing entries in table T508A for the employee's work schedule data.  
  • Feature SCHKZ return the Work Schedule Rule not found in table T508A 

Resolution

  1. Verify that the necessary entry for the employee's work schedule data is present in table T508A.
  2.  Verify the customization of the Feature SCHKZ and ensure that the return value from the feature exists in table T508A. If it does not, it is necessary to adjust the feature configuration.
