Symptom

The "with memory use" option is selected under "Duration/Type" tab in the variant, but memory consumption is not displayed in the Runtime Analysis Evaluation Overview.

Reproducing the Issue

Execute ABAP runtime analysis in tcode SE30.

Resolution

1. Access tcode SE30 -> Variant -> Duration/Type tab -> Aggregation: None -> Select "With memory use (if aggregation not used)".

 

2. After collecting trace, click "Evaluate" and "Call hierarchy (F8)" in the Runtime Analysis Evaluation Overview.

3. Memory consumption is on the far right.
