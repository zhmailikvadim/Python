Symptom

  • Some system models are not functioning on the new SAC tenant.
  • When opening the models, the page appears blank, and errors occur during usage.
  • Identified models affected: SAC_PERFORMANCE_E2E, SAC_STATISTICS_MDS_QUERY_PERF, SAC_USER_FRIENDLY_PERF_ACTION, VERSION_STATISTICS_VIEW, DM_SCHEDULE_JOB, DM_SCHEDULE_LOG.
  • Error message observed: Error [Server]: (#42208) Can't read the view : _SYS_BIC:t.VQ5B2Q.sap.fpa.services.userFriendlyPerfLog/SAC_PERFORMANCE_E2E::[], reason: Can't read the view : _SYS_BIC:t.VQ5B2Q.sap.fpa.services.userFriendlyPerfLog/SAC_PERFORMANCE_E2E, clientStatusCode: 99999, serverStatusCode: 200.

Environment

  • SAP Analytics Cloud 2025.21.13 (Client)
  • SAP Analytics Cloud 2025.21.11 (Server)

Reproducing the Issue

  1. Navigate to the system overview story to observe missing data.
  2. Attempt to open the identified models:
  • SAC_PERFORMANCE_E2E
  • SAC_STATISTICS_MDS_QUERY_PERF
  • SAC_USER_FRIENDLY_PERF_ACTION
  • VERSION_STATISTICS_VIEW
  • DM_SCHEDULE_JOB
  • DM_SCHEDULE_LOG

Cause

The issue is suspected to be related to a backend side problem.

Resolution

The solution of this issue is currently under investigation. Please follow this KBA for updates.
