Symptom

When trying to use Transfer Stock App (App ID: F0753) to move stock, in the pop-up "Select Reason for Transfer," there is a header text for the reason. It is required to change or remove the header text.

Environment

  • SAP ERP
  • SAP S/4HANA
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • Retail

Resolution

The configuration for the text of the transfer reason can be found in the following path:

Materials Management -> Inventory Management and Physical Inventory -> Movement Types -> Record Reason for Goods Movements -> field 'Reason for Movement'

For example, for stock transfers between storage locations, the movement type used is 311. In the configuration mentioned above, the Reason for Movement for this movement type can be adjusted.
