Symptom

You're getting error "Template TESTING uses the same sequence number multiple times(Message No. MDC_PROCESS_MSG164)" when activating Best Practice 2025.

Environment

  • Master Data Governance on S/4 HANA on-premise 2025

Cause

The error is due to an improper standard delivered dummy process template for MDG consolidation&mass processing. 

Resolution

Please carry out the following steps:

  1. Run Tcode MDCIMG
  2. Execute activity: Configure Process Template ->Specify Process Template
  3. Select template “TESTING” and delete.
  4. Save the changes.
