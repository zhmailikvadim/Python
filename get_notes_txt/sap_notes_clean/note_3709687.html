Symptom

While creating a Case, the Category dropdown displays only 30 entries, even though more categories are configured in the system.

Environment

SAP Sales Cloud and SAP Service Cloud Version 2 

Reproducing the Issue

  1. Log in to the system.
  2. Navigate to Menu → Cases.
  3. Create a new Case.
  4. Select the Case Type as “ABC”.
  5. Open the selection list for the Category field.
  6. Observe that only 30 categories are displayed in the dropdown list. 

Cause

This behavior is by design. To ensure better performance and usability, the system limits the initial display of dropdown values to the top 30 entries.

Resolution

  1. By default, only the top 30 categories are displayed in the Category dropdown to enable faster loading and improved user experience. There is no limitation on the total number of categories that can be created or used in the system.
  2. If more than 30 categories exist, users can start typing the required category name directly into the field. The dropdown list will dynamically filter and display the matching categories based on the entered text.

Keywords

Case, Category, Dropdown, Display Limit, Selection List
