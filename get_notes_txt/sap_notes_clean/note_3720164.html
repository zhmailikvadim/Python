Symptom

How to change the text color of the Greeting message (“Good morning / Good afternoon”) displayed on the Latest Home Page banner.

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

SAP SuccessFactors HCM Suite

Resolution

To change the greeting text color on the Latest Home Page, please follow the steps below:

  1. Go to Admin Center.

  2. Navigate to Theme Manager.

  3. Select the active theme and click Fine Tune.

  4. Go to the Home Page section.

  5. Locate Greeting Text Color.

  6. Adjust the color as desired.

  7. Save the changes and apply the theme.

After saving, refresh the homepage to verify that the new greeting text color is reflected.
