Symptom

  • When logging on to SAC  from the SAC PowerPoint Add‑In the Login hangs on /sso/login/callback
  • The SAC Excel Add-In is able to logon along with the web version of Power Point

Environment

SAC 2025.21.11 

Reproducing the Issue

  1. Open Power Point
  2. Open SAC Add-In
  3. Add URL https://<SAC Tenant URL> to a tenant with SSO or Customer IDP enable and try to logon
  4. External browser will open and gets stuck at this URL https://<SAC Tenant URL>/sap/fpa/ui/office/office.html?landingPageUrl=https%3A%2F%2Fhcs.cloud.sap%2Fmspptaddin&landingPageVersion=1&addinVersion=1.1.0.0&addinName=SAP-Analytics-Cloud

Cause

The SAC Plug-In for Power Point desktop version does not support SSO as per help guide https://help.sap.com/docs/SAP_ANALYTICS_CLOUD_PPT/1f0ea2f4ed6145c0baf5da141034df31/6445b44d53c349d6b761cb67808dddb3.html under Client software  Requirements -> Microsoft Office : "SSO based connections are not supported on Windows and Mac."

Resolution

Use the web version of Power Point instead.
