Symptom

  • Incorrect HRA exemption calculation observed during payroll processing.
  • The value of wage type /4E4 in the RT table is not calculated correctly.
  • System is not deducting 10% of Annual Basic from the total Rent Paid during calculating HRA exemption.

Environment

  • Human Capital Management (HCM)
  • SAP R/3
  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • Payroll India

Reproducing the Issue

  1. Run the payroll using the payroll driver for the mentioned variant.
  2. Check the value of /4E4 in the final RT table.

Cause

On the V_T7INR3 table, the V3CON constant value for HRA-code HR04 has not been maintained correctly.

Resolution

  1. Execute transaction SM30.
  2. Enter view name as V_T7INR3.
  3. Click on the Position button.
  4. Pass the value of HRA-code as HR04 and constant as V3CON.
  5. If the constant is not present on the table , kindly create the entry with value '10' and save it.
