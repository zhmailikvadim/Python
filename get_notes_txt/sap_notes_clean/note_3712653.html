Symptom

When filtered transactions are exported via the CSV file option, some records are missing their column details such as Shipping Customer ID and Shipping Company.

Environment

  • SAP Sales Performance Management
  • Incentive Management on Legacy HANA 

Reproducing the Issue

  1. Access Incentive Management Admin.
  2. Access the Transaction Workspace.
  3. In the advanced filter option, filter by any Generic Attributes;
  4. Select the export csv option and export all records.
  5. Open the transaction csv in an excel sheet.
  6. Filter by Blanks in the affected column.
  7. Blank records may appear.

Cause

This is a Known Issue.

Resolution

The fix for this issue is planned to be deployed on 2605 2026 release.

For release timelines, please review SAP SPM 2026 Release Schedule - SAP Community 

Workaround

At this moment, the available workaround is to export the transaction files so that the number of records with missing fields lowers until there are no records with missing fields
