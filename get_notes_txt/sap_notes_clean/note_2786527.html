Symptom

You are archiving posting control orders with the archiving object FS_DISPORD and transaction SARA. For your own business-related checks, you have implemented the BAdI BADI_FS_TB_ILM_BUS_CHK. You notice that the field CONTRACT_INT and the field PRODEXT (BAPC_ORDER-ATTRI1) are initial for the term data that is transferred in the parameter IT_INPUT.

Other Terms

FS_DISPORD, ATTRI1, CONTRACT_INT, BADI_FS_TB_ILM_BUS_CHK_ARCH, BAPC_ORDER, BCA_TAB_DISPORDER_RT, BAPC_DTE_ORDER_ATTRI1

Reason and Prerequisites

Program error

Solution

Implement the attached correction instructions or import the specified Support Package.
