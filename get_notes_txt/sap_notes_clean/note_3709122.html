Symptom

User is not able to find GRC Reports below on Fiori:

  • Role Relationship
  • Roles by Owners and Approvers

Although same reports are available in NWBC

Environment

  • SAP Access Control 10.0
  • SAP Access Control 10.1
  • SAP Access Control 12.0
  • SAP Access Control 12.0 for SAP S/4HANA

Reproducing the Issue

  1. Log in NWBC and check the report desired
  2. Log in Fiori but could not find the same report

Cause

Due to a design limitation, the specified reports were not migrated to Fiori until the current GRC 12.0 Version.

Resolution

Use the NWBC links for those reports instead of attempting to access them on Fiori.

For more information on how to activate those reports in NWBC, refer to the 'See Also' tab below.
