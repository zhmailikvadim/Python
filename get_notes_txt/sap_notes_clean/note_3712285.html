Symptom

  • In the app "Post General Journal Entries" only SAP standard Reason Codes can be selected. 
  • It is not clear where Reason Codes need to be maintained.
  • A method is required to change the Reason Code differently from the standard ones provided by SAP.

Environment

SAP S4 Hana Cloud Public Edition

Reproducing the Issue

  1. Open the app "Post General Journal Entries".
  2. Attempt to use or modify the Reason Code.
  3. Observe that the Reason Code cannot be changed differently from the standard ones provided by SAP.

Cause

The configuration for maintaining Reason Codes is performed in Central Business Configuration, not within the app "Post General Journal Entries".

Resolution

  1. Log in to Central Business Configuration using the provided link (where my000001 needs to be replaced by the particular system URL):
    https://my000001-adm.s4hana.sapcloud.cn/ui#IMGActivity-execute?IMGActivity=SIMG_CFORFBOBBEZMAN&CustomizingObject=V_T053R&CustomizingObjectType=V&CustomizingTransaction=S_ALR_87003271&Type=SSCUI 
  2. Enter the Company Code as "ABC" (where ABC is the Company Code). 
  3. Press the green tick to proceed.
  4. Add or customize Reason Codes in the available configuration section.
