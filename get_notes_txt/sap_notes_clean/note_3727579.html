Symptom

Data fields for hierarchy information are missing from the generic API FARR_INBOUND_ORDER_API with import structure FARR_S_INB_ORDER_MI_API

Other Terms

BOM structures, SOAP

Reason and Prerequisites

Missing functionality

Solution

Implement the attached correction instruction.

The following fields for hierarchy data will be added to the import structure of type FARR_S_INB_ORDER_MI_API with group ID GRP_HILDO

  • RaHigherDocItemSenderComp   CHAR 3   Higher-Level Item Sender Component
  • RaHigherDocItemLogicalSystem CHAR 10 Logical System of the Higher-Level Item
  • RaHigherDocItemType                CHAR 4  Higher-Level Item Type
  • RaHigherDocItemID                    CHAR 35 Higher-Level Item ID
