Symptom

New version of ZUS Z-3 report (HPLCZ301) does not handle the maternity and paid parental leaves during unpaid parental.

Other Terms

ZUS Z-3, HPLCZ301, ABSTY=24, ABSTY=25.

Reason and Prerequisites

Gap in the implementation. SAP note 3534180 has been applied.

Solution

Report HPLCZ301 has been corrected.

Please apply the correction by importing the Support Package specified for your release, or by using the Note Assistant.
