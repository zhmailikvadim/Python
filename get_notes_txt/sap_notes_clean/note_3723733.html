Symptom

This SAP Note generates the required DDIC elements (and other bits) in your system before implementing SAP Note 3723731.

Other Terms

DDIC, UDO, Message of SAP Note 3723731, NOTE_3723733

Reason and Prerequisites

Non-Source Code Elements of SAP Note 3723731

Solution

This SAP Note generates the required Non-Source Code Elements in your system before implementing SAP Note 3723731

1) Implement the correction instructions of this SAP Note, via SNOTE.

2) Run the newly created report (NOTE_3723733) in update mode, so that all the Non-Source Code Elements of SAP Note 3723731 are created and activated.
