Symptom

  • In the DRC Cloud Edition 'Resolve Errors' app, the following message is shown for the affected eDocument ID:
    "Error occurred during the PUSH step, detailed information: The Cloud Edition received the following message when sending information that comes from the external communication party to your business system with destination <destination_name>: wrong tag The process instance ID is XXXXXXXXXXX, HTTP status code is 500".
  • OR, Inside the 'Schedule Jobs' application, following error is displayed:
    Problem occurred while transmitting the document to the business system. See details and try to reprocess in the Resolve Errors - Documents and Statutory Reports app. The Cloud Edition received the following message when sending information that comes from the external communication party to your business system with destination <destination_name>: Request failed with status code 500 The process instance ID is XXXXXXXXXXX, HTTP status code is 500

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

  • SAP ERP
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition
  • SAP BTP
  • SAP Document and Reporting Compliance, Cloud Edition

Reproducing the Issue

  1. Launch transaction edoc_cockpit
  2. Submit an eDocument
  3. Check the logs in DRC Cloud Edition - Resolve Error application

OR,

  1. Launch DRC Cloud Edition and schedule a job to pull supplier invoices in 'Schedule Job' app
  2. Check the job logs

Cause

This issue will usually occur due to some communication issue in the SAP BTP or Cloud Connector configuration and the push service requests may not reach the business system.

Resolution

  • Collect and check the traces for the Cloud Connector as described in SAP Note 2746754: Log and Trace files to troubleshoot scenarios involving SAP BTP > SCC > AS ABAP. Ensure the log levels are as indicated in each point.
  • Also review the the 'Resolution' steps described in KBA 3733392.
