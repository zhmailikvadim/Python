Symptom

A company sending an ACH payment is required to use the Company Entry Description field (which is in positions 54-63 in the Company/Batch Header Record) to provide the Receiver with a description of the purpose of each payment.

Starting from March 20, 2026, originators of DME files of format ACH will be required to use the new description “PAYROLL” for all PPD credits for the payment of wages, salaries, or other similar types of compensation.

Other Terms

DTADUSBH, FILL_BATCH_HEADER

Reason and Prerequisites

Legal change

Solution

Implement the attached correction instruction.

If you create your DME files with the help of the DMEE format tree US_ACH_DD, implement note no. 3729064.

If you create your DME files with the help of report RFFOUS_T, please implement note no. 3729130.
