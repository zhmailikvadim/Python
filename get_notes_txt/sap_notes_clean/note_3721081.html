Symptom

  • Assistance required to update users in the "XXXX" role to ALL cost centers.
  • Inquiry regarding the appropriate upload connector file to use for the update.

Environment

Integration - Uploads

Reproducing the Issue

  1. Attempt to update users in the "XXXX" role to ALL cost centers.
  2. Seek guidance on the upload connector file to use for the update.

Cause

Resolution

  1. Use the "User Cost Center Association Upload" integration to upload multiple user/cost center associations.
  2. Refer to the integration documentation for "User Cost Center Association Upload".
  3. To associate all cost centers for specific users, provide the column "Cost Centers" value as [ALL].
  4. Refer to the documentation link: SAP Fieldglass Integration Documentation.

See Also

Refer to: SAP Fieldglass Integration Documentation
