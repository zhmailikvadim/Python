Symptom

How long a token created by CPQ APIs endpoints lasts?

Environment

SAP CPQ

Resolution

  • Token generated through basic/api/token API lasts 5 minutes, with a refresh token which can be used once lasting for 20 minutes
  • Token generated through /oauth2/token lasts 5 minutes with a refresh token which can be used once lasting for 20 minutes
  • Token generated through core/GenerateJWT lasts 24h
