Symptom

When you log on to SAP Business One and click Enterprise search, refresh Fiori-style cockpit or open Pervasive Analytics Designer after a period of inactivity, the following error messages appear:

  • HTTP Status 500 - Internal Server Error
  • {"errorCode":401,"errorMessage":"Your session is invalid, you are quit. You can refresh page and login again."}

Environment

SAP Business One version for HANA

Reproducing the Issue

Prerequisite:

      The Fiori-style cockpit is enabled.

Scenario:

  1. Log on to the SAP Business One client.
  2. Leave the system inactive for a certain period.
  3. Attempt to perform an Enterprise search, refresh the Fiori-style cockpit or open Pervasive Analytics Designer.

Expected behavior:

The corresponding operation works correctly.

Actual behavior:

The following error messages appear:

  • HTTP Status 500 - Internal Server Error
  • {"errorCode":401,"errorMessage":"Your session is invalid, you are quit. You can refresh page and login again."}

Cause

The token expires due to extended user inactivity.

Resolution

  1. Access https://<server>:40020/auth/admin/sapb1/console
  2. On the left column, navigate to Configure > Realm settings.
  3. Select Session tab. 
  4. Adjust the value for SSO Session Idle and SSO Session Max to a larger duration.

Keywords

HTTP Status 500, enterprise search, Pervasive Analytics, refresh
