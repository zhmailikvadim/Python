Symptom

Because the root cause is set in the last part of the error message, the message text may be truncated when the error message is too long. We need to enhance the message truncation logic to make it configurable.

Other Terms

Replication SDI target

Environment

  • SAP HANA SDI 2.0 SP04 Patch 20

Solution

This correction is delivered in the release(s) listed in the Support Packages and Patches section of this SAP Note.

In dpagentconfig.ini, this can be configured through the following parameter:

framework.log.stringSize

You must restart the agent after you change the parameter in dpagentconfig.ini
