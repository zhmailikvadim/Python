Symptom

Error "Status of the Alert Calculation Job: The Job is not scheduled" is displaying when performing "Check Data Collection" activity for gray metrics

Environment

  • SAP Solution Manager 7.2

Reproducing the Issue

  1. Access System Monitoring workcenter
  2. Define Scope with affected system SID
  3. Run checking data collection for a gray metric under affected system:


 

Cause

 There is an issue related to Job SAP_ALERT_CALCULATION_ENGINE.

Resolution

  1. Check in transaction SM37 if job is running correctly (should run every 1 minute)
  2. If there is no job scheduled, go to SOLMAN_SETUP / Basic Configuration / Step 2. Schedule Jobs / search for SAP_ALERT_CALCULATION_ENGINE and schedule jobs as planned

Keywords

SOLMAN, Solution Manager, System Monitoring, SAP_ALERT_CALCULATION_ENGINE, "Status of the Alert Calculation Job: The Job is not scheduled", scheduled jobs as planned
