Symptom

You have created follow-up Purchase Orders against the Purchase Request, you are expecting the status of Purchase Request turns into "Purchase Order Created". However, it is still in status "No Follow-up".

Environment

SAP Business ByDesign

Reproducing the Issue

  1. Go to Purchase Requests and Orders work center.
  2. Go to Purchase Requests view.
  3. Open the affected Purchase Request, you will find the status of Purchase Request is "No Follow-up".
  4. However, if you navigate to document flow tab, you can see the follow-up Purchase Order has already been created and sent.

Cause

The status of Purchase Request item is set to  'Purchase Order Created' after the complete quantity in Purchase Request is consumed into Purchase Order(s).

If the Purchase Request is partially ordered, it will show the status "No Follow-up".

Resolution

This is a designed behavior. If the status is set to 'Purchase Order Created' on partially ordered Purchase Request, then the Purchase Request item which still has quantity yet to be consumed will no more appear in 'Open Purchase Requests' query and the corresponding 'Create Purchase Order' task will also be not active. Hence, system designed in this way.

Keywords

Purchase Request status, Purchase Order created, No Follow-up
