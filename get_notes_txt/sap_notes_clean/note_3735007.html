Symptom

Error "com.sap.gateway.core.ip.component.exceptions.OData4Exception: java.util.concurrent.TimeoutException, cause: java.util.concurrent.TimeoutException" is thrown by OData V4 adapter where batch processing is enabled.

Environment

  • SAP Integration Suite
  • Cloud Integration

Cause

The batch payload sent to the OData V4 receiver adapter exceeded the supported 2 MB maximum size for batch processing. Exceeding this limit leads to a timeout error.

Resolution

  • Verify the size of the OData V4 batch XML payload being sent by the iFlow. Ensure that each batch request payload does not exceed 2 MB (adapter limit for batch processing). Refer Configure the OData V4 Receiver Adapter.
  • If larger, split the data into multiple smaller batch requests or reduce the number of change sets/items so each payload stays under 2 MB.
