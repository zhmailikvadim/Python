Symptom

User cannot delete a data source, getting error "This data source is in use. You must remove the following items that use this data source before deleting it. Story level - Assigned Colours."

Environment

  • SAP SuccessFactors HCM Core 2505
  • Platform - Story reporting

Reproducing the Issue

  1. Login the instance
  2. Select a story report
  3. Select data source > Delete data source
  4. an error occur :   "This data source is in use. You must remove the following items that use this data source before deleting it. Story level - Assigned Colours."

Cause

The story has a conditional formatting with assigned colours 

Resolution

User must delete first Conditional Formatting before deleting the data source.

  1. Login
  2. Report Centre
  3. Open the affected story
  4. From the Tools menu > select Conditional Formatting.
  5. In the Conditional Formatting panel, expand the Story Defined section.
  6. Select Assign Colors.
  7. The Assign Colors panel appears, displaying a model, a dimension, and the dimension members.
  8. Choose your model.
  9. Choose a dimension.
  10. In the Dimension Members section, select a member's color option list and change the color.
  11. If you do not want to assign a color to a member, clear the check box. That member will now use the color palette from the chart builder.

When you have finished assigning or removing colors, click Apply.
To close the Conditional Formatting panel, click Done.
