Symptom

While trying to execute Business Rules from Consolidation Monitor,  get status Failed without message error nor error details. I checked ST22 and SLG1 and there are no logs. Consider that last week the rules were executed without error.

During support testing this was noted:

The following dump occurred when running account trans and CT.
Category ABAP programming error
Runtime Errors CONVT_OVERFLOW
Except. CX_SY_CONVERSION_OVERFLOW
ABAP Program CL_UJP_CONS_ENGINE============CP
Application Component EPM-BPC-NW-TDL
Short Text
Overflow when converting 32768

Environment

RTC1709, resolution may apply to other BPC products also.

Cause

Hierarchy configuration

Resolution

It was discovered that the heirarchy contained a child with the same name as the parent. This is not allowed.
