Symptom

  • After the enablement of Latest Homepage Experience, users are getting a pop-up window to tour the changes.
  • Customer would like to disable the tour so that users will not be shown this pop-up window.

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Environment

SAP SuccessFactors HCM

Cause

The "Explore Updated Homepage" pop-up tour window is hard-coded into the system and is designed to appear as a one-time notification for users to highlight the changes in the Latest Homepage. It is triggered automatically when the feature is enabled.

Resolution

This is working as designed. There are no means to disable the tour or prevent it from appearing to users when the Latest Homepage experience is enabled in the instance. This is a one-time pop-up window that will not appear anymore once the requirement to show it to the user (atleast once) is satisfied.
