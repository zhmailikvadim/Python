Environment

  • SAP Cloud Connector(SCC) release independent;
  • SAP Business Technology Platform(BTP).

Cause

There is a network component preventing this communication from fully establishing with BTP.

Resolution

Involve the network and security teams responsible for the infrastructure where the SCC is running, and troubleshoot same to confirm there is no issues raising from these layers, as most of time out issues arise outside SCC itself.
