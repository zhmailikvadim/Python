Symptom

Error message displayed: AY259 when using transaction ABT1N to transfer assets between companies.

Environment

  • FIN Asset Accounting​
  • SAP S/4HANA ​
  • SAP S/4HANA Finance​
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Attempt to use transaction ABT1N to transfer assets from one company to another.
  2. Observe the error message "AY259 Insurable values are no longer supported".

Cause

  • The issue occurs due to the transfer variant in table T082I, which is configured to transfer insurance-related fields (Group Fields 60 to 69) between assets.
  • These fields are no longer supported S/4HANA environment, resulting in the error message AY259 Insurable values are no longer supported. 

Resolution

  1. Create a new transfer variant that do not have fields 60 to 69 assigned in table T082I;
  2. Clear XCOPY checkbox for fields 60 to 69 in table T082I via debug or request SAP for a repair program;
