Symptom

  • An error occurs during the SAP BusinessObjects upgrade process: "Failed to update cache for execution. Program will exit."
  • The installer attempts to locate directories and cache entries but fails due to missing folders or files.
  • Setupengine logs contain following error lines:

    Error: Cache: CacheDUs - source directory not found: e:\Program Files (x86)\SAP BusinessObjects\InstallData\InstallCache\biac.webpath_bundle-4.0-da-64\14.2.6.2839\
    Error: CacheDUs - failed to cache DUs.
    Error: Failed to update cache for execution. Program will exit.

Environment

  • SAP BusinessObjects Business Intelligence Platform 4.x / BI 2025
  • All Supported OS.

Reproducing the Issue

  1. Attempt to upgrade SAP BusinessObjects Business Intelligence platform.
  2. Encounter error: "Failed to update cache for execution. Program will exit."
  3. Review logs indicating missing directories or cache entries.
    Eg., Error: Cache: CacheDUs - source directory not found: e:\Program Files (x86)\SAP BusinessObjects\InstallData\InstallCache\biac.webpath_bundle-4.0-da-64\14.2.6.2839\

Cause

  • Modifying or deleting InstallCache entries present in C:\Program Files (x86)\SAP BusinessObjects\InstallData\InstallCache\ may cause this error. 

Resolution

Temporarily disable antivirus during installation. Antivirus blocks access to required folders that installer wants to access. Never modify or delete any folders in InstallCache folder as that leads to many irrecoverable issues in BI Platform upgradation process.

The installer looks for the InstallCache entries according to the InstallManifest.sqlite i.e., previous installs and updates. So if there are any changes made to InstallCache folder, then the installer could not find the folders it is expecting. Then it throws this error.

If the setupengine logs show the error "source directory not found" for some particular directory, then follow the below steps:

  1. Create missing directories and subdirectories as indicated in setupgengine logs ( source directory not found: e:\Program Files (x86)\SAP BusinessObjects\InstallData\InstallCache\biac.webpath_bundle-4.0-da-64\14.2.6.2839\ ). 
  2. Run the installer again.
  3. Again, if you encounter the same error with different folder, then repeat from step 1 for that folder.
  4. Perform the steps until the installer throws no error of this type.
