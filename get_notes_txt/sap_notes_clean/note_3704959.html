Symptom

You run the Accrual Migration job and expect it to complete within a reasonable time. However, the job runs for more than 23 hours without finishing and must be cancelled manually.

This note corrects the behavior by improving the performance of the Accrual Migration job to prevent excessively long runtimes.

Other Terms

IPMGM_Accrual, Migration Job, /IRM/IPMGM, BUCKET_CALC_RUN_STATUS_FILL, BCKITM_GUID_PREPARE, /IRM/LIPMGMTOP, /IRM/IPMG_MASS_PROCESS, /IRM/IPMG_POSTINGS_PROCESS, /IRM/IPMG_PROCESS, SUBSEQUENT_FUNCTIONS_EXECUTE

Reason and Prerequisites

This is caused by Program error.

Please implement the latest version of SAP Note # 3702710 as pre requisite note.

Solution

Please implement the correction instructions attached. These corrections are only valid if you have applied the Support Packs.
