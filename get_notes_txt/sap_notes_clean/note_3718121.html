Symptom

You require information about shipment of SAP LUMIRA DESIGNER 2025.

Other Terms

SAP LUMIRA DESIGNER CLIENT, SAP LUMIRA, SAP LUMIRA SERVER

Reason and Prerequisites

This is the Release Note for SAP Lumira Designer 2025 delivery.

✔ Introduction/Scope

No New Features has been released for SAP Lumira Designer 2025.

In SAP Lumira Designer 2025, some products and features have been removed.

Please check this KBA for details : 3582179 - Which products and features are deprecated or removed in SAP Lumira Designer 2025.

✔ Supported Environment:

SAP Lumira Designer 2025 compatibility with SAP BusinessObjects Business Intelligence Platform 2025 Patch 800+

Please check this KBA for details : 3017689 - SAP Lumira and SAP Business Intelligence Platform BI Platform compatibility Matrix.

You can find more information on Supported Environment in Product Availability Matrix (PAM).

Solution

Download Information:

SAP Lumira Designer 2025 only supports one installation scenario: Support Packages & Patches.

  • If you have SAP Lumira Server 2.4 and SAP Lumira Designer 2.4 versions installed and want to update to SAP Lumira Designer Client 2025 and SAP Lumira Server 2025.
  • If you want a fresh install of SAP Lumira Server 2025 and SAP Lumira Designer Client 2025.
  • Shipment of SAP Lumira Designer 2025 is available on SAP Service Marketplace at:
    • Go to the SAP Software Download Center on SAP Support Portal at https://launchpad.support.sap.com/#/softwarecenter.
    • Only in SUPPORT PACKAGES AND PATCHES search for "SAP LUMIRA" term or go to the letter 'L'  Select "SAP LUMIRA" folder from the search results. *Please note that going forward both SAP Lumira Server 2025 and SAP Lumira Designer Client 2025 are under the same folder on SMP:  'SAP LUMIRA'.
    • Navigate to SAP LUMIRA DESIGNER 2025 folder to download SAP LUMIRA DESIGNER CLIENT 2025 for Windows and SAP LUMIRA SERVER 2025 for Windows and Linux.
      • For SAP LUMIRA DESIGNER 2025, click on .exe file to start the download on your machine.
      • For SAP LUMIRA SERVER 2025, select the operating system required (Windows or Linux), click on .zip file to start the download on your machine. Extract the .zip file to your target folder or machine.
  • To know more about Fixed Issues, refer to the SAP Notes in References section.

*Note: Both Full package for fresh installation and Patch package for update installation are available under downloads in [Support Packages and Patches]. For example, the title for Full package is "SP00 for SAP LUMIRA SERVER 2025" and title for Patch package is "PatchXX for SAP LUMIRA SERVER 2025 SP00".

Note: Please upgrade both Server and Client to the same patch/SP level.
