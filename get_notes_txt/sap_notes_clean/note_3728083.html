Symptom

When starting the MDS_COMPARE_TOOL, the selection fields for any of the directions between the source object and target object are completely missing.

Environment

  • Sales and Distribution (SD)
  • Logistics Execution (LE)
  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA

Reproducing the Issue

  1. Start the MDS_COMPARE_TOOL transaction.
  2. Choose any source/target object combination.
  3. Observe that the selection field is missing.

Cause

Resolution

  1. Execute the report MDS_COMPARE_GENERATE in the affected system.
  2. Verify that the selection fields are now visible in the MDS_COMPARE_TOOL transaction.

Keywords

mds_compare_tool, selection fields missing, s/4hana, master data synchronization, mds_generate, missing selection screen, source object, target object, dataset not displayed, synchronization tool, mds_compare_generate
