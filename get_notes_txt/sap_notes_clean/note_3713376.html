Symptom

When you run payroll to calculate carved-out benefits for a terminated employee, the system uses the start date of infotype Contribution Plan HK (0346) to calculate the years of service with ORS benefits. This may not reflect the actual ORSO effective date.

After implementing this SAP Note, you can specify the actual ORSO effective date in infotype Date Specifications (0041) to calculate the years of service with ORS benefits. The system provides a new feature, 27DYT, that allows you to define a date type for this calculation. To specify a date type, enter a new line that starts with 'C' in the return value of the feature.

The system enhances payroll function HKCOB to use the date from infotype 0041, based on the specified date type, as the start date for the years of service calculation. For example, if you maintain a line 'C01' in the return value of feature 27DYT, the system uses date type 01 from infotype 0041 as the start date for the calculation.

If you do not maintain the date in infotype 0041, the system continues to use the original calculation logic.

Other Terms

Infotype 0346, Infotype 0041, ORSO, Contribution Plan HK, Date Specifications, Feature 27DYT, Payroll function HKCOB

Reason and Prerequisites

You need to calculate the years of service with ORS benefits based on the actual ORSO effective date, not the start date of infotype 0346.

Solution

Do either of the following:

  • Implement the attached Correction Instruction (CI), then carry out the attached ManualSteps3713376.pdf.
  • Implement the relevant Support Package.
