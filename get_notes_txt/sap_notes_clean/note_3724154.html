Symptom

When you update a sales order in the SAP Business One Web Client, the following error message occurs: "RDR1.Description" القيمة طويلة جدًا في الحقل.

Reproducing the issue

Scenario:

  1. In the SAP Business One desktop client, add Sales Order No. 1 with:
    Line 1: add an item with Item type.
    Line 2: add text that contains more than 200 characters with Text type.
  2. In the SAP Business One Web Client, open Sales Order No. 1, and update it, then click Update.

Expected behavior:

The system should successfully update the sales order.

Actual behavior:

The following error message occurs: "RDR1.Description" القيمة طويلة جدًا في الحقل.

 

Solution

SAP intends to provide a feature package or patch to solve the symptom that is described in this SAP Note. The section References below lists the related feature package or patch once they become available. Be aware that References can only be confirmed at feature package or patch release date. SAP delivers feature packages or patches only for selected releases at its own discretion, based on the business impact and the complexity of implementation.
