Symptom

Import jobs fail after modifying connection sources in SAC

Environment

SAP Analytics Cloud 2025.21.11

Reproducing the Issue

  1. Login to SAC
  2. Open the Planning model in SAP Analytics Cloud.
  3. Run the import job from data management.
  4. Edit the host and port of that connection and save it.
  5. Attempt to refresh the import job under data management.

Notice : the import job fails

Cause

Changing the host and port settings modifies the metadata, which disrupts the original configuration of the import job.

Resolution

Set up separate connections, each with its own host and port.
