Symptom

When adding an Incoming or Outgoing Payment you receive the following System Message:

There is a document of this type with a later posting date

Click Continue and the document is added. 

Environment

  • SAP Business One
  • SAP Business One, version for SAP HANA

Reproducing the Issue

  1. In Administration → System Initialization → Document Settings the uncheck box "Block documents with earlier Posting Date". 
  2. You are adding an Incoming or Outgoing Payment document with a date that is earlier than the date of an existing Payment document. 

Resolution

This is a System Message making you aware that a document of this type with a later date exists. If you wish to add the document, click Continue. 

If you wish to block this kind of transaction, got to Administration → System Initialization → Document Settings and check box "Block documents with earlier Posting Date".
