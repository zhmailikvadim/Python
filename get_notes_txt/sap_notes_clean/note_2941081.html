Symptom

When running KSU5 / KSV5, there's message GA 733 (The segment contains no receivers).

Environment

  • Financial Accounting (FI)
  • Controlling (CO)
  • SAP R/3
  • SAP R/3 Enterprise 4.7
  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA

Reproducing the Issue

In cycle setting, you set Var.portion type as Actual Statistical Key Figures.

Then, when running KSU5 or KSV5, there's message GA733.

Cause

If statistical key figures are used as receiver tracing factor, “space” to “ZZ” always has to be maintained in the RTF (Receiver Tracing Factor) tab.
The reason is that the recovery indicator is not populated in statistical postings.

Resolution

 Please maintain “space” to “ZZ” in the recovery indicator in Receiver Tracing Factor tab.

See Also

SAP Note 514254 - ALLOCATION: Problems with recovery indicator
