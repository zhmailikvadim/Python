Symptom

In CRM WebUI, unexpected dots appear in some input fields when using skins like 'Signature Design'(NOVA).

Environment

  • SAP enhancement package 1 for SAP CRM 7.0
  • SAP enhancement package 3 for SAP CRM 7.0
  • SAP enhancement package 4 for SAP CRM 7.0

Reproducing the Issue

This issue is seen in input fields in Edit mode. For example:

  1. Access a business role,
  2. Search and display an account,
  3. Edit an assignment block for example 'Tax Categories'.

or:

  1. Access a business role,
  2. Create a sales order.

Resolution

Implement SAP Note 2359132 - Introducing "box-sizing: border-box" in inputField and dropdownlistbox tags.

See Also

After implementing the SAP Note, please read and follow the 'WARNING' part in this SAP Note 2359132 to make sure cache is cleared and new CSS content can be used.

Also, if you have developed your own custom skin based upon the files this Note modifies, please propagate this Note modifications to your own skin, for that skin to be fixed as well.
