Symptom

When executing a footprint calculation for a specific period, the process fails with a "Missing value for opening stock" error.

This behavior is observed even when:

  1. The product was successfully calculated in the immediately preceding period.

  2. The master data and business transactions (Data Readiness) are correctly maintained.

  3. Upon inspection in the Manage Footprint Results app, the previous period's result shows the status "Calculated with Warnings".

Environment

Sustainability Footprint Management

Reproducing the Issue

  1. Open the Calculate Footprints app.

  2. Create or select a footprint inventory for the subsequent period (e.g., April 2023).

  3. Ensure that the product from the previous period (which resulted in a "Warning") is part of the Opening Stock (beginning inventory) for the current month.

  4. Execute the calculation by choosing Calculate.

  5. Monitor the calculation status in the Organizational Footprint Inventories or Product Footprint Inventories tab.

Cause

Previously, the SFM calculation engine was designed with a strict validation rule: it would only "inherit" and carry over the footprint result from the previous period if its status was "Calculated Successfully".

If a Plausibility Check (configured in the Manage Plausibility Checks app) identified a deviation (e.g., a variance in CO2​e higher than the defined threshold), the item was marked with a "Warning". Under the old logic, this status was treated as "unreliable" for the opening balance of the next period, causing the calculation to break due to a missing reference value for the beginning of the month.

Resolution

A standard product enhancement has been released in SFM.

The calculation logic has been updated to allow the propagation of footprints with the status "Calculated with Warnings". This ensures that plausibility alerts—which are often based on business-justified variances—do not stop the automated roll-up process for subsequent periods.

Action Required:

  1. Ensure the previous period's calculation is finished (even if with Warnings).

  2. Restart the calculation for the current period in the Calculate Footprints app.

  3. The system will now correctly identify the "Warning" result as a valid Opening Stock footprint and complete the calculation.
