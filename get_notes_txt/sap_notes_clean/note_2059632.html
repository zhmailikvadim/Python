Symptom

When using ALA functionality to process precapitalized assets, one of the following error messages may occur:

  • /FAAM/MSG_CLASS623 "Item detail exists without header entry for event ..."
  • /FAAM/MSG_CLASS256 "PIS asset is not a mass asset; it must be supplied for event ..."
  • /FAAM/MSG_CLASS317 "Define reserve asset for event ... attributes in /FAAM/CU_RES_ASG"
  • /FAAM/MSG_CLASS261 "Enter a reversal reason for event ..."

When trying to reverse unitization of a precap install, there is no opportunity to enter the reversal reason.

Program error
