Symptom

You would like to know if there is a way to get data from the Content Manager in SAP Build Work Zone via API.
e.g. you would like to get roles, groups via API

Environment

  • SAP Business Technology Platform
  • SAP Cloud Foundry Portal Service
  • SAP Build Work Zone, standard edition
  • SAP Build Work Zone, advanced edition

Resolution

We don't expose any public API to get cdm content (data from Content Manager) from SAP Build Work Zone at the moment.

This capability is not supported yet in the product. You are welcome to open a feature request using this link: https://influence.sap.com/sap/ino/#/campaign/2279
Please set the category to Digital Experience -> Fiori Launchpad

See SAP Note 11 - Would you like to shape & influence future releases of SAP Products?

See Also

Currently supported APIs in SAP Build Work Zone:
https://api.sap.com/package/SAPBuildWorkZone/odata
