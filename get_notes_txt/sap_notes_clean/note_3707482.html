Symptom

You are trying to uninstall MDG_FND 749 and SAP_BS_FND 748 (as per SAP Note 2654906). An error appears early in the uninstallation:

  • " The deletion of objects of other software components is not allowed
    The following objects belong also to a foreign software component:
    LIMU REPS /BOBF/NOTE_2105577"

Resolution

  1. Go to t-code SE03 > 'Find Objects in Requests/Tasks'.
  2. Add this object REPS /BOBF/NOTE_2105577
  3. Execute.
  4. You should see 2 or more packages listed. Double click on the SAPK-748A*INSAPBSFND package (* being a wildcard).
  5. Look for the '/BOBF/NOTE_2105577' object and confirm there is a trash can icon associated with the entry.
  6. Open a case under BC-UPG-OCS-SPA component, and attach a screenshot of SE03 in step 5 above.

See Also

2654906 - Uninstallation of component versions SAP_BS_FND and MDG_FND
