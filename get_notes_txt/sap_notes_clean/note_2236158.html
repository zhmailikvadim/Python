Symptom

You use Unified Demand Forecast (UDF) as your modeling and forecasting engine and execute a model or forecast service.

It returns a message due to some data issues containing location information. The value of the location is a GUID (hexadecimal value) instead of a human readable location Id.

Other Terms

message, /DMF/MSG_HL, BAPIRETTAB, /DMF/UDF_BUSINESS, demand modeling, demand forecast, Unified Demand Forecast, UDF, Demand Data Foundation, DDF

Reason and Prerequisites

Program error.

Solution

Implement according to attached correction instructions.
