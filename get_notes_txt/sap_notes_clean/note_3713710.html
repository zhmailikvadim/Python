Symptom

  • The user is automatically assigned PDO to wave through PPF but it failed.

Environment

  • SAP Extended Warehouse Management (SCM-EWM)
  • SAP Extended Warehouse Management Wave management (SCM-EWM-WAV)

Cause

  • The item has a temporary item number assigned, causing the assignment and creation to fail.
  • The PPF Action /SCWM/WHR_WAVE_ASSIGN_NEW is set to "immediate processing", which leads to the issue.

Resolution

  1. Change the processing settings to "processing when saving document" for both "Processing At" and "Processing Times".
  2. Ensure that the document is completely created before assigning the item to the button because we need to wait document complete created then assign PDO to wave. 

Keywords

ppf action, immediate processing, assignment failure, obd, processing when saving document, , whr_wave_assign_new, temporary item number
