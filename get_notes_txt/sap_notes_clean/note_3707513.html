Symptom

When you ask to show the sales order Net amount on Joule, it returns "Wrong Net Values".

When you ask to show the sales order Customer Reference, it returns "Customer Name".

When you ask to show the sales order Document Type, it returns "No Order Type Present".

When you ask to show the sales order Material Codes, it returns "Only Material Description".

When you ask to show the sales order Plant Codes, it returns "Only Plant Description".

Other Terms

API_SalesDocuments_AI_SRV,TotalNetAmount,_TransactionCurrency,_CurrencyText,_SalesDocumentType,_SalesDocumentTypeText,PurchaseOrderByCustomer,Material,Plant.

Reason and Prerequisites

When you Ask for the Sales order information on Joule, it provides wrong or misleading data to the user.

When you ask for Net amount it displays wrong net value.

For Customer Reference it displays Cutomer Name.

For Order Type it displays No Order type present.

For Material Code it displays only Material Description.

For Plant Code it displays only Plant Description.

Solution

Implement the Specified Correction.

Post this correction, When you Ask for the Sales order information on Joule, it displays the data as below.

When you ask for Net amount it displays the Total net value.

For Customer Reference it displays Cutomer Reference.

For Order Type or Document type, it displays the Order Type description.

For Material Code it displays Material Code.

For Plant Code it displays Plant Code.
