Symptom

It is necessary to change the data collection frequency for the respective extractor of Configuration & Security Analysis.

Environment

SAP Focused Run 4.0 and above.

Cause

Resolution

  1. Go to "Configuration & Security Analysis - Administration".
  2. Search for the respective system.
  3. Select the respective extractor.
  4. Click on the "Configuration" button.
  5. Click on "Edit" option.
  6. Change the Start Condition frequency.
  7. Save the changes.

P.S.: It may take up to one hour until the changes are used by the system.
