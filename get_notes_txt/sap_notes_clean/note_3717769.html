Symptom

When running an LSMW for the mass upload of outline agreements with multiple items using BAPI, the system throws a short dump of type RAISE_EXCEPTION with the condition ALREADY_REGISTERED:
Category               ABAP programming error
Runtime Errors         RAISE_EXCEPTION
ABAP Program           SAPLMEOUT
Application Component  MM-PUR-OA
Condition ALREADY_REGISTERED 

Environment

  • Materials Management (MM)
  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA

Reproducing the Issue

  1. Access to LSMW.
  2. upload the file to call BAPI_CONTRACT_CREATE (it’s a contract with more than 10,000 lines).
  3. Then execute the remaining steps and wait for it to fail.

Cause

Standard system design has a limitation of 9999 lines per contract at one time. 

Resolution

Split the items into 2 files and make sure that a file contains max of 10000 lines at a time. This is how the standard design for using BAPI_CONTRACT_CREATE. 

Keywords

LSMW, BAPI_CONTRACT_CREATE, MAX, items, 9999, ALREADY_REGISTERED
