Symptom

Error: “500 HTTP: Internal Server Error” Occurs When Sending a Message from Network to an External System



Cause

  • The carrier's external system returned the error message "500 Internal Server Error".
  • This error indicates that the carrier's application is unable to process part of the data due to a field restriction or internal server issue.

Resolution

  • The network message to the external system fails with a "500 Internal Server Error" when BNL SAP CPI attempts to send the payload to the carrier's endpoint. This indicates an issue in the carrier's external system, which requires investigation by the carrier's internal IT team.
  • A "500 Internal Server Error" can occur for multiple reasons. To resolve the issue, refer to the log details in the Manage Integration Logs application.
