Symptom

  • During the import of DBA cockpit notes into the QA system, the process ended with RC8 during the "Generation of Programs and Screens" phase.
  • The error observed was: "Program CL_DO_UI_RUNTIME_ENGINE_SAPGUICP, Include CL_DO_UI_RUNTIME_ENGINE_SAPGUICM007: Syntax error in line 000065. Formal parameter 'CALLED_BY_DBOSC' does not exist."
  • Syntax errors were also reported in the development system. 

Environment

  • SAP Adaptive Server Enterprise 16.0
  • SAP NetWeaver

Cause

The symptom is caused by a program error.

Resolution

To resolve the issue, SAP Note 3587261 - Syntax Error due to Missing or Unknown Parameter "CALLED_BY_DBOSC" in ABAP Class and Include must be implemented.

See Also

  • SAP Note 3587261 - Syntax Error due to Missing or Unknown Parameter "CALLED_BY_DBOSC" in ABAP Class and Include
