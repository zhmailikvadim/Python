Symptom

Following symptoms are addressed in this SAP Note. The symptoms are:

1. In MPC app, when changing the condition amount in item tab, the system terminates with a runtime error and raises exception /BOBF/CX_FRW_FATAL if the validation fail for this change in fm MR_CHECK_TOLERANCE.

2. In MPC app, when creating a new contract and maintaining a currency for which there is no exchange rate maintained, system terminates with a runtime error and raises exception /BOBF/CX_FRW_FATAL if the validation fail for this change in fm CONVERT_TO_LOCAL_CURRENCY.

Other Terms

F1600A, LMEOUTP5B, LMEOUTP40, LMEOUTP02, MR_CHECK_TOLERANCE, CONVERT_TO_LOCAL_CURRENCY, RAISE_SHORTDUMP, runtime error, /BOBF/CX_FRW_FATAL, /BOBF/CL_TOOL_APPLICATION_ERR=CP, M8, 215, PE, M8215, Maintain tolerance limits for tolerance key PE (CoCode), SG, 105.

Reason and Prerequisites

Program error.

Solution

Implement the attached correction instruction.
