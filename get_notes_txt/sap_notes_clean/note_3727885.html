Symptom

  • The following error is seen multiple times in the transport log:
    • Type '/UI5/RT_REPO_CACHEBUSTER_TOKEN' is unknown.
  • The Fiori Launchpad fails to load with a code 500 Internal Server Error

Environment

SAP S/4HANA 2021

Cause

The new SAPUI5 runtime repository Note has not been implemented into the system where the transport is being imported

Resolution

  1. Implement SAP Note 3599156 - New SAPUI5 Runtime Repository
  2. Re-import the transports which failed with the error in the symptom of this KBA

Keywords

RT_REPO UI5 error syntax import transport request tr fiori launchpad flp RT_REPO_CACHEBUSTER_TOKEN
