Symptom

Once invoices are approved by KSeF, only a standard PDF form is generated, which does not meet business requirements, leading to a request for configuring or assigning a custom PDF form.

Environment

  • SAP ERP
  • SAP S/4HANA
  • SAP Document and Reporting Compliance Service
  • SAP S/4HANA cloud, Private edition

Reproducing the Issue

  1. Launch Tx code Edoc_cockpit
  2. Select any invoice with Status "Accepted by KSEF"
  3. Click on Display PDF

Resolution

Custom PDF forms are not supported within the standard solution. The generated PDF should be considered solely as a preview or reference document, whereas the XML approved by KSeF serves as the legally binding version of the document. Additionally, no backend form configuration is necessary, as the PDF is automatically generated within the DRCce framework.
