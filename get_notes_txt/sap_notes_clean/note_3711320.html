Symptom

During the replication of a serialized batch from SAP S/4HANA to SAP Batch Release Hub for Life Sciences, ISLS replicates the field MCH1-/STTPEC/SYNCTIME as serializationStartDate instead of MARC-/STTPEC/SERVALID in the batch API /dso_batches/BatchesStage.
This is incorrect, as the field refers to the date and time when the batch was synchronized to the SAP Advanced Track and Trace for Pharmaceuticals system. It doesn't indicate when the batch material became relevant for serialization.

Other Terms

ISLS, BRH, Batch Serialization, ATTP

Reason and Prerequisites

Program error

Solution

Please implement this note.
