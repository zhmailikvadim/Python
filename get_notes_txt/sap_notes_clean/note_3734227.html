Symptom

The Comments added for an Item in a Change Record are not reflected in a corresponding line of the 'All Objects' table, and vice versa. This issue occurs after adding a new line to a reference item table and immediately adding a comment.

Other Terms

S/4, PLM, CR, Change Record, All Objects, comments, BOBF, BOPF, DPC_EXT, OData, Fiori, F2097, F2097A

Reason and Prerequisites

Program Error.

Prerequisite SAP Notes: 3655773, 3710072.

Solution

Use transaction SNOTE to implement the corrections delivered with this SAP Note in the affected Change Record back-end system or import the relevant Support Package.

Benefit:

The business users now can see comments and replies for a newly added Change Record item line immeadietaly after its creation.
