Symptom

If you ran a branch after the 2602 test system upgrade, the parallel test tenant (T.120) is impaired and not fully usable.

Additionally, branch-related service requests in SAP for Me are currently unavailable.

Other Terms

Parallel Line, Parallel Test Tenant, 2602 Release, Branch

Reason and Prerequisites

With the SAP S/4HANA Cloud Public Edition 2602 upgrade, reference content containing errors was introduced.

Solution

A content fix reaches all systems on February 16, 2026. Until then, don't branch your main line or delete your parallel line. After February 16, 2026, you can initiate a branch again without restrictions.

Important: All main line tentants remain fully operational and stable. This includes the development tenant (D.080), customizing tenant (D.100), test tenant (T.100), and production tenant (P.100).

We are resolving the issue in the existing parallel line tenants. No action is required on your part.
