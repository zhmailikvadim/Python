Symptom

  • Labels are generated during inbound processing; however, they are not printed automatically.
  • Manual intervention is required to trigger the “Print all labels” function.

Environment

SAP Business ByDesign

Reproducing the Issue

  1. Go to Inbound Logistics Work center
  2. Select Purchase order view
  3. Search for the relevant Purchase Order
  4. Select and choose New supplier Delivery Notification.
  5. Go to the New Labels under the common tasks from the Inbound Logistics Work center
  6. We observe that the label is created but does not print automatically.
  7. Use manual intervention to select the "Print all labels" button to print the label.

Cause

The current system design does not include functionality for automatically printing labels without manual intervention.

Resolution

  1. The current system design does not support automatic label printing immediately after label creation.
  2. Labels need to be printed manually by selecting the "Print All Labels" option.
  3. To request a functional enhancement for automation, submit the requirement on the SAP Continuous Customer Influence Portal for review and potential future development.
