Symptom

This article describes functionality introduced in the May 2026 SAP Fieldglass release

Some Company Configuration settings that were previously view‑only are now editable in Configuration Manager. 
You may have questions such as:

  • Why configuration settings that were previously locked can now be changed
  • Whether this change is expected or requires action
  • Why warning messages or guidance text appear when adjusting certain settings
  • Why some configuration settings cannot be reversed once enabled
  • Whether changes to configuration settings could affect business workflows

Reproducing the Issue

You may see this behavior when:

  • Opening Company Configuration in Configuration Manager
  • Viewing or changing settings related to time, expense, invoicing, login, security, or integrations
  • Enabling a configuration and seeing warning messages or informational guidance
  • Attempting to disable a configuration that remains enabled after activation
  • Observing changes in system behavior after updating a configuration

Cause

  • This behavior is the result of a product enhancement in SAP Fieldglass that expands the set of Company Configuration settings you can manage directly in Configuration Manager
  • Previously, many of these settings were view‑only and required assistance to update.
  • With this enhancement, you have more visibility and direct control over your company configuration settings
  • The functionality of the configuration settings themselves has not changed
  • Only the ability to manage them directly in Configuration Manager has been enhanced

Resolution

This behavior is expected and does not require corrective action

What to Expect

  • Some Company Configuration settings that were previously view‑only are now editable
  • Warning messages or guidance text may appear to explain the impact of certain settings
  • Some configuration settings are intentionally irreversible once enabled
  • Configuration changes take effect based on the intended system behavior

Recommended Actions

  • Review warning messages and guidance carefully before enabling or changing a configuration
  • Make changes in a test or non‑production environment when possible
  • Use caution when adjusting login or security‑related settings to avoid access issues
  • Review related business processes after making configuration changes

If you are unsure about the impact of a configuration change, refer to SAP Fieldglass documentation or contact SAP Support for assistance.
