Symptom

Internal Server error while accessing Integration suite webui with irrespective of having all the required roles assigned to the user in BTP cockpit. 

Environment

  • SAP Cloud Integration
  • SAP Integration suite

Cause

  • Browsers store static files from websites, like images and scripts, in a cache to speed up loading times. Occasionally, these cached files can become outdated or corrupted, causing unexpected behavior.
  • Cookies store session information, user preferences, and other data specific to websites visited. If there are conflicts or issues with these cookies, it can lead to problems with logging in or using web services correctly.

Resolution

  • Ensure that the VPN or VDI isn't blocking the connection. Attempt to browse without VPN and VDI from a different browser.
  • If the issue persists, clear cache, cookies & history of all time and restart the browser. 

Keywords

Internal server error, Integration suite, Cloud integration webui, unable to access tenant, CPI, webui not accessible, tenant not accessible
