Symptom

 During navigation to ISU POD transaction from Display Measurement Concept Management application, the navigation fails due to date format issues.

Other Terms

MCM, /APEU, UCOM, UIUCOM, Display POD

Reason and Prerequisites

 Our application is using locale based date which is incompatible with the transaction.

Solution

We are now using fixed pattern date of dd.MM.YYYY for the navigation. This ensures that the navigation always completes.
