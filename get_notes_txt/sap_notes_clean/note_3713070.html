Symptom

  • The required use cases for SAP Integration Suite (Cloud Integration) are not visible under Cloud ALM Landscape Management.

  • All the other use cases like Integration Monitoring, Exception Monitoring, Health Monitoring are missing.

  • In the "Supported Use Cases" section, there is only "External API Management" displayed.

Environment

Landscape Management Service

Reproducing the Issue

  1. Access Cloud ALM Landscape Management.
  2. Navigate to the section where use cases for SAP Integration Suite should be displayed.
  3. Observe that only the External API Management option is visible, and other relevant use cases are missing.

Cause

  • The issue is a known problem that recently occurred and is acknowledged by the developers.

Resolution

  1. The developers have created a fix for the issue, which is scheduled to be available on the upcoming Monday.
  2. For further updates, refer to this Knowledge Base Article.

See Also

Refer to: SAP Knowledge Base Article CS20260011636618
