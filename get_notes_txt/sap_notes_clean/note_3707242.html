Symptom

  • While submitting DRC Belgium self-billing invoice, the error "Error in method READ_CBE. Please fill one of the input parameters, Endpoint is empty" is encountered.

Environment

  • SAP ERP
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition
  • SAP Document and Reporting Compliance, cloud edition

Reproducing the Issue

  1. Navigate to the Transaction code edoc_cockpit.
  2. Attempt to submit a self-billing invoice for DRC Belgium.
  3. Observe the error message: "Error in method READ_CBE. Please fill one of the input parameters, Endpoint is empty".

Cause

The issue is caused by missing code corrections required for self-billing functionality.

Resolution

  1. Implement SAP Note 3684621 - Electronic Invoice for Peppol Countries/Regions - Corrections.
  2. Ensure that all prerequisite notes are implemented in the correct sequence before applying SAP Note 3684621.
  3. Retest the submission of self-billing invoices for DRC Belgium to confirm resolution.
