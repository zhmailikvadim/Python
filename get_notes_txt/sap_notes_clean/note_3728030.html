Symptom

While reversing application document, system throws error as "Application reversal not allowed as non-standard settlement exist " Although, Non-standard settlement has been performed for underfill cancellation associated with other application document.

Other Terms

ACM, Agriculture Contract Management, Underfill, Per delivery, Non-standard settlement

Reason and Prerequisites

Program error

Solution

Replication Steps -

1- Create one TC contract Sales/Purchase side with tolerance type as per delivery and price it fully.
2- Create Nomination using above Trading contract.
3- Create LDC using Nomination and application is created with some underfill quantity.
4- Create final settlement and perform process till Revrec.
5- Create Non-Standard settlement using above contract for underfill cancelled quantity.
6- Perform invoice cancellation/settlement reversal for the above application document.
7- Perform application reversal and system throws error as "Application reversal not allowed as Non-standard settlement exist"
8- Application reversal should be allowed if Non-standard settlement group associated with application document underfill Id is cancelled/reversed.

Import the relevant support pack or apply the correction instruction(s).
