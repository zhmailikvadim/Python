Symptom

In My Timesheet v2 app, Labels were truncated in confirmation dialog box when the label text is long. This was due to UI5 padding issue. Timesheet v2 application followed a workaround changing the alignment of the text as no padding fix was provided from UI5 side.

This workaround is no more needed, the text alignments are reverted to previous state and the dialog width is increased to avoid truncation.

Other Terms

Truncate, alignment, 1.52, Fiori 2.0.

Reason and Prerequisites

This issue is a side effect of corrections from SAP Note 2720085 and 2689603.

Solution

Apply corresponding support pack. Alternatively, implement the correction instructions attached to the SAP Note.
