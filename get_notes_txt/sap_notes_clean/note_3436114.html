******************************************************************************************************************************************************************************************************************
WARNING: This is an SAP Tax Declaration Framework Note. If you do not have SAP TDF installed in your system, this SAP Note does not apply to you. SAP TDF requires a specific license.
******************************************************************************************************************************************************************************************************************

This SAP Note is relevant for Tax Declaration Framework for Brazil application installation or upgrade in the Simple Finance solution using the SAP Landscape Transformation Replication Server ﻿(SLT). For this action, you have to replicate the tables listed below in your SLT:

  • ACDOCA

  • FAGLFLEXT_BCK

  • FINSC_LD_CMP

  • FINSC_LEDGER

  • FINSC_LEDGER_REP

  • FINS_CURTP_MAP

  • GLT0_BCK

  • GLT0_DIF

  • T001A

  • T881

This SAP Note is updated regularly. Check the latest version of this SAP Note before proceeding with your installation or upgrade.

Other Terms

TDF, Brazil, SP, SLT, Replication, sFIN, SAP_FIN, Simple Finance

Reason and Prerequisites

This SAP Note is relevant only for Brazil.

Solution

After implementing this SAP Note, the Tax Declaration Framework for Brazil application is installed in your Simple Finance solution.

To proceed with the installation of your TDF with Simple Finance and SLT scenario, check for tables in SAP Note 2190963. Those tables are needed for every scenario regarding replication and SLT.
