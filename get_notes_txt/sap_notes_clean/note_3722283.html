Symptom

Below error occurs during the simulation step using the migration object "Routing" in the Migrate Your Data app. 

  • Enter a lower specification limit

          Message No. QP  671

Environment

  • SAP S/4HANA Cloud Public Edition
  • SAP S/4HANA Cloud Private Edition
  • SAP S/4HANA

Cause

The "Ind.: 0 Is a Lower Specification Limit" field in the "Inspection Plan Characteristic" sheet is empty.

Resolution

Locate the "Ind.: 0 Is a Lower Specification Limit" field in the "Inspection Plan Characteristic" sheet.

  1. If zero is a lower specification limit, enter X in the field.
  2. If zero is not a lower specification limit, leave the field empty.

Keywords
