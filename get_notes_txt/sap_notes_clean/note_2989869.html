Symptom

You run the Informes de I.R.P.F. (RPC190E0) program for an employee that has both positive and negative entries for a previous year and it matches the following criteria:

  1. The sum of these entries will produce a negative amount.
  2. There are different values of Tipo de relación (TIREL).

The output file generated by the program is reporting only one entry with a negative base and positive tax withholding (retención) for this previous year.

Example:

In 2020 an employee has the following amounts to report in Form 190 related to year 2019:

Year Base Tax withholding Relation type
2019                 500,00 EUR                     75,00 EUR 1
2019             -1.000,00 EUR                       0,00 EUR 2

The RPC190E0 program identifies that these entries have a different relation type and when changing the relation type of the negative entry to '1' the lines are merged:

Year Base Tax withholding Relation type
2019                 -500,00 EUR                     75,00 EUR 1

It is expected that these entries would not be merged as their sum generates a negative base. Only the relation type of the second entry should be changed:

Year Base Tax withholding Relation type
2019                 500,00 EUR                     75,00 EUR 1
2019             -1.000,00 EUR                       0,00 EUR 1

Other Terms

&190 - PC00_M04_C11X, PC00_M04_C190, PC00_M04_CCRI

For more information, see SAP Note 1071282 - CONSULT: List of tags for Spain notes.

Reason and Prerequisites

This issue is caused by wrongly merging previous years entries with different relation types instead of only changing the relation type to '1'.

When you install an SAP Note, some prerequisites might be required. The prerequisites are SAP Notes that have already been released and that change the same context of the current one. Then, even if these prerequisites are not necessarily related to the symptom of this SAP Note, you should implement them due to technical reasons.

Solution

From now on, the RPC190E0 program will not merge positive and negative entries that have a different relation type.
