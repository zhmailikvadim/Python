Symptom

  • Error message "SQL error, server error code: 686. start task error: '***' cannot insert NULL or update to NULL:***".
  • Error message in index server trace file: "Error executing "UPSERT  *** FROM "***"."_SYS_CE__popid_4_***"": cannot insert NULL or update to NULL: *** at ptime/query/checker/check_replace.cc:586; popid=4; nodename='$REQUEST$'

Environment

SAP HANA smart data integration (SDI)

Reproducing the Issue

  1. Execute the hdbflowgraph.
  2. Observe that the task fails with the error message mentioned above.

Cause

  • The source table and target table have different column sequences, which may cause the flowgraph to fail during the upsert operation.
  • The flowgraph might have been corrupted or altered, leading to the failure.

Resolution

  1. Verify the column sequence in the source and target tables to ensure they match.
  2. Update the flowgraph to align the column sequence between the source and target tables.
  3. Alternatively, create a new flowgraph using the template table in the target system.
