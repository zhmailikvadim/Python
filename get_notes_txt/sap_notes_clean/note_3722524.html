Symptom

  • Offboarding custom task does not display the text message/ field .

Environment

SAP Successfactor Onboarding 

Reproducing the Issue

  1. Navigate to the Offboarding task.
  2. Observe that the text message / field is not populating

Cause

  • The field cust_is set to "Not Visible".

Resolution

  1. Go to the "Configure Object Definitions".
  2. Search for the object definition "cust_  object ".
  3. Click on "Take Action" > "Make Correction".
  4. Open the field "cust_  field ".
  5. Change the field visibility to "Editable".
  6. Save the changes.
  7. Initiate a new Offboarding activity to verify that the field populates correctly.

See Also
