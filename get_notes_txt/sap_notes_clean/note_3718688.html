Symptom

I am getting an error message when trying to access a report in SAP Ariba Analytical Reports:

Error: Document Access Denied - You don't have access to this report. If you have any questions, please contact your system administrator.

Cause

It is possible that the report was built with a Source Data from 'ariba.analytics.fact.InvoiceLineItem' as main fact, which means this is data from Spend Visibility in SAP Ariba Spend Analysis.
If Spend Visibility is not enabled in the realm, this error message appears on the screen when trying to access the report that have Spend Visibility facts.

Resolution

As mentioned in the SAP Article KB0475743 - How to active the Ariba Spend Analysis Feature (SV-1110), it's a paid feature, and if you'd like to enable it again, you should contact your CSP.
