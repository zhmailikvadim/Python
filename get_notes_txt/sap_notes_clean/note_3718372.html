Symptom

  • Users would like to know which Event Type of Linkages (BOR Object Type, ABAP Class, XML Object Type) are active in the system.
  • Administrator would like to track how or why the event type of linkage is activated and deactivated in the past.

Environment

  • SAP Business Workflow
  • SAP NetWeaver
  • SAP Web Application Server for SAP S/4 HANA
  • ABAP PLATFORM - Application Server ABAP

Resolution

I. Current Status:

    1. Call transaction SWE2,
    2. Select Event Type Linkage details (Object Category, Obj. Type, Event Receiver Type,...),
    3. Check "Type linkage active" column. If this indicator is set, the type linkage between the event creator and the event receiver is activated.

II. Historical Changes:

  1. Call transaction SCU3,
  2. Click Evaluate Logs,
  3. Select data (Customizing Object SWFDVEVTY1, Evaluation for Customizing Object, Analysis Period, Evaluation for Tables and output option,...),
  4. Execute (In case of longer time period selection is needed, users have the possibility to execute ),
  5. Select table SWFDEVENA - Activations for Event Linkages on pop-up,
  6. Check "Type linkage active" column changes for the selected Event Type Linkage  (Data, Time , User, Transaction Code, Program Name column will show how and when the status changed).
