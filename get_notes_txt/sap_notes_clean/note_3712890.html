Symptom

This Note resolves an issue in measurement document synchronization during Delta Load. Due to a mismatch, an incorrect external object key value was populated in the object link table. As a result, the queue was generated with the object key value set to 'MEASUREMENTDOC' instead of the actual measurement document number, leading to queue cancellation.

Other Terms

Defense Disconnected S4,Measurement Document.

Solution

Implement the attached correction instructions provided with the NOTE.
