Symptom

Access to Readiness Check site (me.sap.com/readinesscheck) is failing with error below.

Secure Connection failed

An error occurred during a connection to pr.alm.me.sap.com PR_CONNECT_RESET_ERROR

Error code: PR_CONNECT_RESET_ERROR

Environment

SAP S/4HANA

SAP ERP 6.0

Reproducing the Issue

Open Readiness Check URL me.sap.com/readinesscheck 

Cause

Non-supported browser is being used to access Readiness Check page

Resolution

  • Error 'PR_CONNECT_RESET_ERROR' only occurs in Mozilla Firefox, use other browsers like Google Chrome or Microsoft Edge to access Readiness Check tool.
  • In Chrome: Select Settings > Privacy and security > Clear browsing data.
  • Disable Extensions: Temporarily turn off ad-blockers, VPNs or security extensions (e.g., uBlock Origin, HTTPS Everywhere).
  • Try Incognito/Private Mode: Opens without extensions/cached data.
  • Disable VPN/Proxy: Turn off any active VPNs or proxies.
  • Temporarily Disable Firewall/Antivirus: Check if tools like Norton, McAfee or Windows Defender are blocking the connection. Re-enable after testing.
  • Add Exception: Add pr.alm.me.sap.com to your firewall/antivirus allow list.
