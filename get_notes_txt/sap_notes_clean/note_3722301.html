Symptom

Getting "Invalid Values for Default Cost Center" error performing a User Upload.

Cause

Invalid cost center value was provided in the "Default Cost Centers" field column.  

Resolution

  • Ensure the value provided for "Default Cost Centers" is valid.
  • For Bulk Uploads, no value is required and will default to company configuration 
  • Alternatively, use the value '[BLANK]' to indicate no value for "Default Cost Centers".
