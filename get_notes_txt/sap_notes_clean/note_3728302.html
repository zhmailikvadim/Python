Symptom

  • Login to Promotion Management Wizard (PMW) fails
  • The login fails with error:

        Operation failed!

        class com.sap.commoncryptolib.provider.AlgorithmParametersDH (in unnamed module @0x38065cad) cannot access class sun.security.util.DerValue (in module java.base) because module java.base does not export sun.security.util to unnamed module @0x38065cad. 

Environment

  • SAP BusinessObjects Business Intelligence Platform 4.2 SP03 or older versions.
  • Promotion Management Wizard (PMW)
  • All supported OS

Reproducing the Issue

  1. Create a live to live promotion using the 4.3 SP2 PMW
  2. Logging into the source 4.2 Sp3 or older CMS the tool will throw the error.
  3. Failed with error: Operation failed!class com.sap.commoncryptolib.provider.AlgorithmParametersDH (in unnamed module @0x38065cad) cannot access class sun.security.util.DerValue (in module java.base) because module java.base does not export sun.security.util to unnamed module @0x38065cad. 

Cause

  • This behavior is expected. A stronger cryptographic hash mechanism has been introduced in SAP BusinessObjects Business Intelligence platform 4.2 SP05, which is not compatible with older versions.

Resolution

  1. This is a known limitation as per the Product Design/Implementation.
  2. Upgrade older version to BI 4.2 SP05 at least to fix this.
  3. Workaround: Export a job to an LCMBIAR file in the source system and then import the job from the LCMBIAR file to the destination system.
