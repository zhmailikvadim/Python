Symptom

Unable to download transactions, the system displays a message indicating that there are files in the queue being downloaded.



Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

Incentive Management on Hyperscalers

Reproducing the Issue

  1. Open the Incentive Management Admin page.
  2. Navigate to the Transactions section.
  3. Select any transaction.
  4. Click on the Export Data option.
  5. Observe the system displaying a message stating that there are files in the queue being downloaded.

Cause

The system was overwhelmed by the number of calls and required clearing the queue to proceed.

Resolution

  1. Change the browser settings to allow downloading multiple files from Incentive Management simultaneously.
  2. Refresh the page in Incentive Management.
  3. Allow the system to download all files in the queue.
  4. Once the queue is empty, verify that transactions can now be downloaded.
