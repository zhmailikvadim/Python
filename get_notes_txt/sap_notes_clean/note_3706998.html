Symptom

Unable to Search for User in User Manager UI Despite Active Status

Resolution

Users are unable to search for user in user manager option in UI , even though user is active  

To make active users available in the User Manager search, Application.Approvable.OrganizationalVisibilityPartitioning  must be enabled for approvable applications. This ensures that users are visible across the appropriate organizational units, allowing them to appear correctly in search results and approval-related workflows. 

The application uses organizational visibility rules to control which users appear in search results. If Organizational Visibility Partitioning is not enabled for approvable applications, active users may be hidden from search due to visibility restrictions. 

Keywords

User Manager, User not available in UI, User not able to search
