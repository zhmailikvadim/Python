Symptom

When processing a payment advice note, the incorrect MeMi document status 50 is kept in method /US4G/IF_PROCESS_ADVICE~PROCESS_ADVICE_CHECKB of class /US4G/CL_PROC_PAN_MEMI, if status 50 was incorrectly set before the payment advice note was processed.

This issue is relevant for the market role: DSO.

Other Terms

US4G, MeMi, /US4G/PROCESS_ADVICE_CHECKB, /US4G/CL_PROC_PAN_MEMI, /US4G/IF_PROCESS_ADVICE~PROCESS_ADVICE_CHECKB, Payment Advice Note

Reason and Prerequisites

Program error

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the system sets the MeMi status correctly in method /US4G/IF_PROCESS_ADVICE~PROCESS_ADVICE_CHECKB during processing.

You can verify the fix by checking that the MeMi status is set correctly during payment advice note processing.
