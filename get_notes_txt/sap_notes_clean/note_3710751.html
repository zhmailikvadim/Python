Symptom

This SAP Note provides you with updates to the Data Replication for US Tax Forms app.

Currently, the column headers in the Run Details view do not clearly represent the specific tax-related information being displayed. The Document Number (SRF_RPL_DOCUMENT_NUMBER) field and Item (SRF_RPL_DOCUMENT_ITEM) field use generic terminology that may cause confusion for users reviewing tax document data.

Other Terms

 Document and Reporting Compliance, DRC US, Fiori App ID F7760

Reason and Prerequisites

You have applied these SAP Notes:

  • 3710194 - [DRC U.S. - Sales and Use Tax] Updated column headers in Data Replication for US Tax Forms app

Solution

After implementing this SAP Note, the column headers in the Run Details view of the Data Replication for US Tax Forms app are updated to provide clearer and more specific terminology.

The system implements the following updates:

  1. The Document Number (SRF_RPL_DOCUMENT_NUMBER) field is renamed to Tax Document Number. As a result, the field label clearly indicates that the document number refers to tax documents.

  2. The Item (SRF_RPL_DOCUMENT_ITEM) field is renamed to Tax Item Number. As a result, the field label clearly indicates that the item refers to tax item numbers within tax documents.

Import the Support Package specified for your release or implement the attached (manual) correction instruction. As a general rule, SAP recommends that you implement the Support Package specified for your release. However, if you need to install a solution earlier, use the SAP Note Assistant to implement the correction instruction. You can find more information about the SAP Note Assistant in the SAP Support Portal, under SAP Note Assistant. Note that the documentation of certain objects is delivered with the corresponding Support Package only.
