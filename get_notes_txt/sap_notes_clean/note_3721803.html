Symptom

The switch Respect Rule-Based Generation of User ID During Manual Onboarding is not visible in Onboarding General Settings UI.

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

SAP SuccessFactors Onboarding

Reproducing the Issue

  1. Go to Onboarding General Settings UI
  2. There is no Manual Onboarding Settings

Resolution

As per internal, in 2511, it is behind toggle and not handled to enable/ disable via UI.

Additional things to consider per below in having this toggle enabled:

  • Please note:  To verify if toggle is successfully enabled, you will see that “Add new employee ID” field will be disabled. See attached screenshot
  • With this toggle enabled, on initiate onboarding using this page “Add new hire to Onboarding”, it will create userId using rule set at manage data—> Onboarding configuration—> DEFAULT_ONB2_CONFIG—> Rule for Onboarding User ID Generation. See next attached screenshot

On details for how to set this rule please take reference of implementation guide below
