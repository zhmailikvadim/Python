Symptom

You notice that within the email editor, when composing a new email, the email preview option (eye icon) is no longer displayed.

Environment

SAP Sales and Service Cloud Version 2.

Reproducing the Issue

  1. Logon to your SAP Sales and Service Cloud Version 2 environment.
  2. Navigate to any entity work list which supports emails e.g. Case.
  3. Open any case.
  4. Click on Send Mail.
  5. See the Preview option is no longer displayed within the email editor options.

Cause

This is a designed behavior, as the Preview option will only be available in case there are Branding Templates available in the system or assigned to the agent that is composing the email.

Resolution

This was a change introduced with an SAP Sales and Service Cloud release, so it would be part of the designed behaviors for the system.
