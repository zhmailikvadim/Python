Symptom

  • Error message: "URL does not pass security policy; upload failed" occurs when attempting to add a URL as an attachment in a freight order.
  • Error message no# /BOBF/ATF 170 is raised due to an internal exception CX_HTTP_WHITELIST in method CL_HTTP_UTILITY=>CHECK_HTTP_WHITELIST().

Environment

SAP S/4HANA 1709 and higher

Reproducing the Issue

  1. Open any Freight Order in Fiori (Tiles Edit Road Freight Order).
  2. Select Attachements Tab.
  3. Select Insert.
  4. Select URL.
  5. Add any URL.
  6. Getting Error "URL does not pass security policy; upload failed".

Cause

The error is caused by an exception CX_HTTP_WHITELIST in method CL_HTTP_UTILITY=>CHECK_HTTP_WHITELIST().

Resolution

Follow the instructions provided in the note to check and adjust the related whitelist.

  1. 2667053 - CX_HTTP_WHITELIST was raised
  2. For further details please check 3290787 - How to maintain entries in transaction UCON_CHW 

See Also

  • 2667053 - CX_HTTP_WHITELIST was raised
  • 3290787 - How to maintain entries in transaction UCON_CHW
