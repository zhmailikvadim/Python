Symptom

Application Id value shows blank when application is viewed.

Environment

SAP SuccessFactors Recruiting Mangement 

Reproducing the Issue

  1. Navigate to Recruiting Management and open a candidate’s Job Application.
  2. Locate the Application ID field on the application details; the value displays blank.
  3. Run a report on the same application and observe that an Application ID value exists in reporting while the UI shows blank.

Cause

  • Incorrect configuration of the Application ID field in the Job Application template.
  • The field was configured as a custom field with field ID "applicationId" and Field Type "textarea" instead of using the standard field with Field ID "id" and Field Type "text".

Resolution

  1. Open Admin Center and go to Manage Templates (Job Application template).
  2. Review the configuration of the Application ID field and identify any custom field defined as "applicationId" with Field Type "textarea".
  3. Remove or deactivate the incorrect custom field configuration for Application ID.
  4. Add the standard Application ID field:
  • Field ID: "id"
  • Field Type: "text"
  1. Save and publish the Job Application template.
  2. Verify by opening a Job Application in the UI to confirm the Application ID now displays correctly.
