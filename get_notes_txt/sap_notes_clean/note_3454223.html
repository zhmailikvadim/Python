Symptom

You would like to know if DEEP INSERT AND DEEP UPDATE functionalities are compatible with ODATA adapter. 

Environment

  • Cloud Integration
  • SAP Integration Suite

Resolution

The OData Adapter in CPI is based on OData specification. As per OData specification, Deep Update (PUT, PATCH, MERGE) is undefined. Deep Insert (POST) is supported.
Hence the CPI OData adapter supports PATCH of single entity only.

Keywords

Oadapter specification, Deep Update (PUT, PATCH, MERGE), Deep Insert (POST)
