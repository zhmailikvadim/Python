Symptom

  • FCO_R_COST_CENTER 039 error raised while uploading cost center.
  • Error message: "Area 1 cannot be used here".

Environment

SAP S/4HANA Coud Public Edition

Reproducing the Issue

  1. Attempt to upload cost center data into the SAP S/4HANA Cloud Public Edition system.
  2. Enter cost center groups in the KHINR/Hierarchy area field.
  3. Observe the error message "KS 070 - Area cannot be used here".

Cause

Resolution

Ensure that cost center groups entered in the KHINR/Hierarchy area field are assigned to the standard hierarchy. Refer to the SAP Help Documentation for detailed guidance:

  • Cost Center Standard Hierarchy
  • CO - Cost center 
  1. Verify the respective migration object documentation for additional details on hierarchy area.
  2. Try using standard hierarchy "1~" instead of "1" and check if the issue is resolved.
