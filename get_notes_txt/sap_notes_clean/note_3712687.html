Symptom

Restriction Between Assignment Purchasing Group to Material Group is Not Working

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Define the mapping in SSCUI 102479.
  2. Define selection operator NE (Not Equal).
  3. Create purchase requisition using Apps Manage Purchase Requisitions.
  4. Select the Purchasing Group and Material Group maintained in SSCUI 102479.
  5. Warning or error not appear as expected.

Cause

Although the NE (Not Equal) operator is visible in the configuration, it is not supported for this specific setup. Only positive operators such as EQ (Equal), CN (Contains), and BT (Between) are supported.

This means it is only supported to maintain the Purchase Organization in combination with the Material Group to determine what should be considered for validation and displayed in the F4 Help for My Purchase Requisition.

Resolution

If require the configuration to also support the NE operator, this would fall under a feature request.

  • To get started, see SAP Customer Influence for SAP S/4HANA Cloud Sales
  • For more information, see SAP KBA 2963059 - How to Submit Feature Requests for SAP S/4HANA Cloud Product
