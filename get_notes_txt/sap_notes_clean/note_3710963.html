Symptom

  • A Job Posting has routed to a buyer user for approval, but the user cannot view the Job Posting
  • The user is getting the error message when clicking on a Job Posting: You are not authorized to access this page. Please contact your System Administrator if you believe you should have access to it.

Cause

The buyer user does not have the correct Business Unit associated to their account

Resolution

Even if a buyer user is listed as the next approver, they may not be able to view the Job Posting if they do not have the correct Site, Cost Center, or Business Unit associated

An administrator can add these to the user by following these steps:

  1. Open the administrator menu > User > Locate the user in question
  2. On the left side, you will see boxes for Site, Cost Center, and Business Unit respectively
  3. Click into any of those boxes > Add > add the item from the Job Posting
  4. Click Add again

If you do not have these permissions, please contact the Program Management Office
