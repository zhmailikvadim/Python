Symptom

You use the Electronic Payment Integration or Bill of Exchange Management to run payment. The system allows the payment to exceed the budgeted amount slightly when you set a usage rate in the payment control rules.

The system shows a warning message when the payment usage rate is equal to the usage rate in the payment control rules.

Other Terms

Payment Control, EPIC, BOE, Zero, Budget Amount

Reason and Prerequisites

This is a program error.

The system uses a number type with insufficient precision to calculate the actual usage rate. The usage rate is not calculated correctly due to rounding.

Solution

Apply the corresponding support package or implement the correction instructions in this note.

The system will not show a warning message when the payment usage rate is equal to that in the payment control rules.
