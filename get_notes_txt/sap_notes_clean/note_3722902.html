Symptom

  • Profile pictures of users with multiple employments are not visible in the Succession Org Chart.
  • Upon viewing the user in the Succession Org chart, the profile picture is blank.
  • When clicking on the position to view the incumbent, the user's profile picture becomes visible.

Environment

Succession Org Chart

Reproducing the Issue

  1. Navigate to the Succession Org Chart. 
  2. Search for the example user
  3. Observe that the user's profile picture is not visible unless clicking on the position to view the incumbent.

Cause

Known error - code correction has been added

Resolution

    Workaround:

    1. Go to edit the profile picture
    2. Press save
    3. Go to Succession Org Chart
    4. Picture is working again

    A code correction has been added and will be evaluated for deployment in the upcoming release (2605).
