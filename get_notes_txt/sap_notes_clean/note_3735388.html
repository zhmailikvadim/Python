Symptom

You are working in the transportation cockpit with multiple windows. In an additional window, you have placed a detail UIBB like the TOR overview or the load planning UIBB. After you display the detail data for an object like a freight order or container unit, you close the additional window where the details are placed, without removing the detail data. Afterwards, you open this additional window again. When displaying the details for the same object as before, the detail UIBBs will not be updated with the current data.

Other Terms

TC, /SCMTMS/PLN, MW, AW, multiple windows, main window, additional window, details

Reason and Prerequisites

The problem is caused by a program error.

Solution

Implement the attached correction instructions or import the specified Support Package.
