Symptom

There is a requirement to change the value of properties from AS Java Managers without a Graphical User Interface.

Environment

SAP NetWeaver Application Server for Java

Resolution

  1. Open the Console Config (/usr/sap/<SID>/<InstID>/j2ee/configtool/consoleconfig.bat|sh) and navigate in its menu through the options below.
  2. Instance
  3. Managers
  4. Edit Manager Info
  5. Choose the Manager you want to edit
  6. It will cycle through the properties
    1. Type N to go to the next property
    2. Type D to reset to the template value
    3. Type E to edit.
