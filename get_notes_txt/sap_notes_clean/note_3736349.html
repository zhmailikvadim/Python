Symptom

When accessing the configuration activity "Define Common Distribution Channels", the "edit" button does not appear.

Environment

  • SAP S/4HANA Cloud Public Edition
  • SAP Central Business Configuration

Reproducing the Issue

  1. Access the configuration activity "Define Common Distribution Channels" in the Central Business Configuration environment.
  2. The button "Edit" is not appearing.

Resolution

Standard system design, for preventing activation errors, SAP Development team has disabled the "edit" option for SAP delivery entry in the activity "Define Common Distribution channels" in demonstration systems.

In all other environments, it is possible to edit SAP delivery entries.
