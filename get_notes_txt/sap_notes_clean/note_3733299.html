Symptom

In the My Time Events Fiori 2.0 Application, the Edit and Delete buttons are not visible in the Recorded Event List on the Time Event List tab on small screens in portrait mode.

Other Terms

F1506A, Edit, Delete, Buttons, Mobile Devices, Portrait Mode, Time Event List Tab, Recorded Event List

Reason and Prerequisites

Due to limited screen width on small screens in portrait mode in the My Time Events Fiori 2.0 Application, the Edit and Delete buttons are not visible in the Recorded Event List of Time Event list tab.

Solution

Kindly implement the correction instruction attached to this SAP Note or alternatively you can implement the corresponding support pack.
