Symptom

Error message while deploying the new API: "Unable to Register API - Missing configurations; Please contact support team to configure virtual host."

Environment

  • SAP BTP Integration Suite
  • SAP BTP APIM

Reproducing the Issue

  1. Attempt to create a new API on the SAP instance.
  2. Save the new API proxy.
  3. Observe the error message: "Unable to Register API - Missing configurations; Please contact support team to configure virtual host."

Cause

In the Virtual host associated to particular landscape, one Virtual Host needs to be set to default i.e, Isdefault = true.

Resolution

  1. Configure any virtual host as the default setting.
  2. Refer to the following guide for detailed steps: Configuring a Default Domain for a Virtual Host
  3. Retry the API creation process after completing the configuration.
