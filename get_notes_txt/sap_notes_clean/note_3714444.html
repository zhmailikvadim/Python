Symptom

  • Data is not pulling correctly from C4C to SAC, with certain columns appearing blank despite containing data in C4C.

Environment

  • SAP Analytics Cloud 2026.2.2

Reproducing the Issue

  1. Establish a connection between SAP Analytics Cloud (SAC) and SAP Cloud for Customer (C4C).
  2. Attempt to import data from C4C to SAC using the connection.
  3. Observe that certain columns appear blank in SAC, even though data exists in C4C.

Cause

This issue is currently under investigation of development team.
