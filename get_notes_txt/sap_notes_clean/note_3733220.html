Symptom

The ceiling for Medical Cash Allowance De Minimis (V_T511P: MEDCS) should be 4000 instead of 8000.

Other Terms

De Minimis, MEDCS

Reason and Prerequisites

This is a correction for legal function.

Solution

Apply the corresponding HR Support package;

Manual Update with reference to the steps below:

1, Use T-code SM30 to access the transaction,

2, Input 'V_T511P'

3, Find the entry for MEDCS with enddate 9999.12.21

4, Change the ceiling from 8000 to 4000

5, Save the changes
