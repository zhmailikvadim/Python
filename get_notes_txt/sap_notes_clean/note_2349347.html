Symptom

During item creation using item template, the field 'Project management template' is filled with a GUID.

This GUID corresponds to the item template's item type's defaulted Project Management Template.

Other Terms

Item template, Item type, Create item, Project management Template, GUID, Default, /RPM/ITEM_DETAILS, VI_GEN_INFO,  VALIDATE_PROJECT_TEMPLATE.

Reason and Prerequisites

Reason: Program error.

Prerequisite:

1. Item template doesnot have any Project template with default flag checked.

2. Item type of the Item template should have a Project template with default flag checked.

Solution

Implement the corrections provided in this SAP Note.
