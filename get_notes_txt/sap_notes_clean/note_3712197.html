Symptom

Unable to migrate data from S/4HANA System to SAP S/4HANA Cloud Public Edition using "Migrate Data Directly from SAP System" approach in Migrate Your Data app.

  • Source system: SAP S/4HANA or SAP S/4HANA Cloud Private Edition or SAP S/4HANA Cloud Public Edition
  • Target system: SAP S/4HANA Cloud Public Edition

Environment

SAP S/4HANA Cloud Public Edition

Resolution

It is not possible to use an SAP S/4HANA system, an SAP S/4HANA Cloud Private Edition system, or an SAP S/4HANA Cloud Public Edition system as a source system.

SAP Help Portal - Migrate Data Directly from SAP System - Before You Start
