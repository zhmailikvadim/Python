Symptom

You face an issue while interacting with Joule in S/4HANA Private Cloud Edition and you need to create a Case so that SAP Support can help you.

Environment

  • S/4HANA Private Cloud Edition 
  • SAP S/4HANA Private Cloud Joule Capabilities 

Resolution

When opening a Case with SAP Support, please make sure to provide us with the following. In addition, follow KBA 1922545 for general guidelines on Cases.

 

  • Reproduce the issue in Joule and collect the Conversation Traces (KBA 3666014 or this link)
  • In the Case description, provide the Capability and Prompt used. Sometimes trace files contain multiple traces. 
    • Make sure that the prompt is supported as shown in the SAP Help page below:
      https://help.sap.com/docs/joule/capabilities-guide/transactional-use-cases-4cdb2d4b06324f81ade019c6f8c4517a
  • Describe the issue and what's the expect outcome.
  • Ensure that the user is set with the correct roles and that the roles are exposed
    • To check which roles are necessary, check this
    • To set them up and expose them, check this (Step 2 onwards)
  • If the issue involves the Joule icon being missing or the Joule popup prompting for login, also provide:
    • The URL to your system
    • The logon data maintained in Logon Depot
    • More details here.

 

This Guided Answer can help you with several of the steps above.
