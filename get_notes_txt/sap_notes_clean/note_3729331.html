Symptom

Meter reading units are read using regional structure. Regional structure allows to maintain Meter reading unit across divisions. During Installation process, such cross division meter reading units are not identified causing error in installation process.

Other Terms

/APEU, MCM, /APEU/CL_MCM_CONFIG_ACCESS=>GET_REGIONAL_STRUCTURE, Meter reading Unit, MRU, Create Installation, ISU_DB_ADRSTRTMRU_SINGLE, Division

Reason and Prerequisites

Program Error

Release 2602

Solution

Please implement the attached the correction instruction.

If meter reading identification fails with division, fallback logical option to read cross division meter reading unit is introduced.
