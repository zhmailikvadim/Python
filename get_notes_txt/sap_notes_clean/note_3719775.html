Symptom

  • Error message: "Item YY1_i2d.mpe.prodoper.confirms1 of type UI Flexibility: Views and Changes has to remain in collection. Item has been exported."
  • Two separate entries in the Extensibility inventory have the same ID (YY1_i2d.mpe.prodoper.confirms1). One is of type 'UI Flexibility: Views and Changes', and the other is of type 'UI Flexibility: App Variants'.
  • The error occurs when attempting to add a newly created entry for 'Confirm Production Operation' to a software collection.

Environment

SAP S/4HANA Cloud Public Edition 2602

Reproducing the Issue

  1. Open the Software Collections app.
  2. Select the collection.
  3. Add an entry for 'Confirm Production Operation'.
  4. Observe the error message: "Item YY1_i2d.mpe.prodoper.confirms1 of type UI Flexibility: Views and Changes has to remain in collection. Item has been exported."

Resolution

It's a known issue which will be fixed on Hotfix 06(Mar 14th&15th).
