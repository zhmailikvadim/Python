Symptom

  • There is no direct option to select or configure an ArcGIS basemap within SAP Analytics Cloud (SAC) Geo Maps.
  • It is not possible to integrate or connect an ArcGIS map (e.g., web map, layers, or services) directly as a base layer in SAC Geo Maps.
  • Users want to embed their own ArcGIS web maps/services within SAC.
  • Functional limitations are observed when using custom widgets to embed ArcGIS maps, including fixed dimensions/measures, lack of in-chart filter functionality, and inability to export the custom widget.

Environment

  • SAP Analytics Cloud (Enterprise)

Cause

  • This type of integration is not natively supported by SAP Analytics Cloud (SAC).
  • Functional limitations are inherent to the use of custom widgets for embedding ArcGIS maps.
  • Please refer to the following blog to use custom widget: Deep integration of SAP Analytics Cloud and ArcGIS - SAP Community.

Resolution

  • An enhancement request to add the ability to embed ArcGIS web maps and services directly into SAP Analytics Cloud (SAC) Geo Maps has been submitted by the customer.
  • Please cast your vote here: Improvement Request Details - Customer Influence.
