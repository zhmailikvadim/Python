Symptom

In app Automatic Account Determination, when trying to transport the changes, the expected Transport Request is not shown in the option list.

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. In app Automatic Account Determination, make changes and click save.
  2. Pop-up “Assign Transport Request” may appear.
  3. The expected Transport Request is not shown in the option list in “Customizing Request”.

Resolution

There is a restriction that only the first 100 transport requests are shown in the option list.

The workaround is to use another user with less customizing transport requests or to release or cancel some of the customizing transport requests.

Keywords

Automatic Account Determination, transport request, restriction, list
