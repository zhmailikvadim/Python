Symptom

You are using monitor transaction /SCWM/MON. You enter no value as input criteria for the Product Number field and fill only one other field from the Basic Attributes part of the search criteria pop-up. This leads to very long processing times based on number of product records and this is not a valid search criteria. Sometimes this could also lead to a dump.

Other Terms

Warehouse Monitor, /SCWM/PROD_DATA_MON, /SCWM/SAPLPROD_OVERVIEW, s_matnr[]

Reason and Prerequisites

Program error.

Solution

Implement the correction instructions attached in this note.
