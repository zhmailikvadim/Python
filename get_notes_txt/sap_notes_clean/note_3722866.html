Symptom

In Renovated Learner Portal, when infotype 1029 is set to "Individual booking allowed" for a Course Type, 'Book' button is displayed instead of 'Book on Waitlist' when course capacity has been reached to maximum.

Other Terms

learning portal, waitlist booking, course, course type, book on waitlist, book, available seats

Reason and Prerequisites

The issue was occurring due to the changes introduced by SAP Note 2882999.

Solution

Apply the corresponding Support Package to incorporate the changes. Alternatively, apply the attached correction instructions from this SAP Note.
