Symptom

Unable to sort teams by user column in SAP Analytics Cloud

Environment

SAP Analytic Cloud 2026.2.6

Reproducing the Issue

  1. Login to SAC
  2. Navigate to Security --> Teams
  3. Select Sort 

Notice: The option to sort by users is unavailable.  

Cause

By design behavior of the product

Resolution
