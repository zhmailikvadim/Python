Symptom

  • An error occurs when attempting to send a document over DocuSign after tenant refresh.
  • Error message displayed: "Sending document over DocuSign failed".
  • Developer Console error message: "Invalid URI: The format of the URI could not be determined".

Environment

SAP CPQ

Reproducing the Issue

  1. Open quote.
  2. Go to Generate document for e-sign.
  3. Choose one template.
  4. Click on generate.
  5. Add signer.
  6. Send to DocuSign.
  7. Observe the error message: "Sending document over DocuSign failed".

Cause

The issue occurs when the DocuSign account integrated with the CPQ tenant is not set as the default account.

This can happen in both scenarios:

  • Multiple DocuSign accounts exist, and the correct account is not set as default
  • Only one account exists, but it has not been explicitly set as default

CPQ integrations rely on the default DocuSign account configuration.

Resolution

  1. Log in to the DocuSign account.
  2. Click on the user icon and navigate to Manage Profile.
  3. Locate the account settings section.
  4. Click "Set as default" for the appropriate account.
    • Note: Even if only one account is present, it must still be explicitly set as default.
  5. Save/confirm the changes.
  6. Retry sending the document from CPQ.
