Symptom

You are operating SAP Digital Manufacturing, release 2602, and you want to integrate cloud applications with machinery and equipment on the shop floor with the help of SAP Plant Connectivity (PCo) or the Production Connector for SAP Digital Manufacturing.

You require information about the versions of PCo and the Production Connector that are compatible with SAP Digital Manufacturing, release 2602.

Other Terms

Machine Model, Production Connectivity Model, PCo, Cloud, Edge, Cloud Connector, compatibility

Reason and Prerequisites

.

Solution

Integration with SAP Plant Connectivity (PCo)

The Production Connector has taken over cloud integration from PCo starting in November 2022. Since then, SAP has delivered new developments for cloud integration for the Production Connector only. If you are still using cloud integration with PCo, you must migrate to the Production Connector now because the support for cloud integration in Plant Connectivity has ended. This means that the 2602 release of SAP Digital Manufacturing will continue to work with the latest 15.5.4 release of SAP Plant Connectivity (PCo). However, SAP no longer provides any updates and corrections to the cloud integration part of SAP Plant Connectivity. You need to migrate to the Production Connector to fix any problems. The latest PCo release 15.5.5 no longer supports cloud integration. Therefore, an upgrade to PCo 15.5.5 is not possible if a cloud integration exists (see note 3489731). At some point in the future, it will no longer be possible to create new Plant Connectivity systems in the Production Connectivity app. In a further step, Plant Connectivity systems will stop working overall in SAP Digital Manufacturing.

If your connections to the Internet are routed through a proxy server, you might need to consider these proxy settings during outgoing Web service calls and in the communication with the UAA service. This requires installation of the Production Connector.

Integration with the Production Connector for SAP Digital Manufacturing

Release 2602 of SAP Digital Manufacturing is compatible with the following releases of the Production Connector:

  • Production Connector 2.6, published in August 2025.
  • Production Connector 2.7, published in November 2025.
  • Production Connector 3.0, published in February 2026.

Attention: Maintenance for release 1.0 of the Production Connector ended in November 2024, therefore, Production Connector 1.0 is no longer compatible with the release 2602 of SAP Digital Manufacturing. Perform the upgrade to the lastest available Production Connector release now to keep cloud integration supported.

If you want to use structured data types in the context of OPC UA source systems, refer to the consulting note 3461876 for more information about the scope of this function in Production Connector 3.0.

If you want to make use of SAP Digital Manufacturing for edge computing or of the new SAP MQTT Broker, you must install Production Connector 2.3, at least. However, SAP recommends the installation of Production Connector 3.0 to profit from new functions.

In general, SAP recommends that you always install the highest available support package and patch level of the Production Connector to profit from the latest corrections and improvements. SAP delivers new patches only for the latest available support package of a product. For the Production Connector this is support package 00 for release 3.0 (version 3.0).

Further Information

The SAP Digital Manufacturing Integration Guide describes how to set up the integration between the cloud and PCo or the Production Connector. You can access this guide through the Help Portal page for Digital Manufacturing under https://help.sap.com/docs/SAP_DIGITAL_MANUFACTURING. On this home page, switch to the Integrate tab.

In the Cloud Integration Guide 2602 of the Production Connector, you will find instructions on setting up the integration between the Cloud Connector and the Production Connector, which is the prerequisite for the integration. Note that the Control Center of the Production Connector can support you with this task through a guided configuration. Refer to the Help Portal page https://help.sap.com/viewer/product/PRODUCTION_CONNECTOR, Integrate tab, to access the Cloud Integration Guide 2602.
