Symptom

Importing the registration questionnaire response in SM administration results in the error: Supplier has to be invited, before being able to accept questionnaire import. Supplier VDRXXXX is skipped.

Cause

The error occurs because the supplier was manually set to Registered status without being invited, and there is no associated project with a completed Registration questionnaire for the supplier. 

Resolution

1. Ensure the supplier is invited before attempting to import the questionnaire.

2. Verify that there is an associated project with a completed Registration questionnaire for the supplier. 

3. Use either external registration or internal registration to import the questionnaire successfully.
