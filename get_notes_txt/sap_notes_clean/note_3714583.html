Symptom

You run a story report for employee central timesheet and you notice that some values are several factors higher than expected or have duplicates

Environment

  • Employee Central
  • Time Tracking

Reproducing the Issue

  1. Create a story report for time sheet in employee central
  2. Run report
  3. Some reported values are several factors higher than expected

Cause

Cancelled timesheet data is also being reported

Resolution

  1. Go to Reports Centre
  2. Add the 'Latest Time Sheet' field
  3. Ensuring this is filtered for 'TRUE' will allow you to only report on the most recent active data.
