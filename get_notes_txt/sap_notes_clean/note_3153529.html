Symptom

Transaction EL35 with restriction of the selection subset (in accordance with SAP Note 2807977) or transaction EL16 with the parallel processing object ANLAGE (in accordance with SAP Note 3096306) reads an unnecessarily large number of meter reading documents from the database.

Reason and Prerequisites

Gap in SAP Note 2807977 or 3096306.

Solution

Implement the attached correction instructions or import the specified support package.
