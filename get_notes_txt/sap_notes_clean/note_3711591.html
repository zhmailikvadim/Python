Symptom

IRAS has officially released the new PDF layout template for IR21 (Version IRIN 112/1/2025). This SAP Note delivers the corresponding new SAP IR21 PDF form: HR_SGTAXIR21_2026.

Compared with the previous version, the new PDF form includes the following key changes:

  • On Page 2, the Deductions section adds the item “Life Insurance Premium Deducted Through Salaries” (Evaluation Class 11, Value 46). The configuration for this amount has been documented in SAP Note 3610023.
  • The section “Employee’s Income Tax Borne by Employer” has been moved to the Income section on Page 2
  • The Appendix 1 and Appendix 2 pages now only display EEBR share records.

Other Terms

IR21, New PDF

Reason and Prerequisites

This is a legal change.

Solution

Apply the corresponding Support Package or implement the attached Manual Activity instructions in this SAP Note.

For more information about the development delivered with this SAP Note, see the entry for this SAP Note in the What's New topic on SAP Help Portal.
