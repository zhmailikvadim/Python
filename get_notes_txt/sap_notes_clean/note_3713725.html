Symptom

In the Fiori app Manage Workflows for Group Journal Entries (App ID F6164), the following error message occurs upon opening the app:

Modification is not allowed 

Workflows cannot be modified

Environment

SAP S/4HANA Finance for Group Reporting

Reproducing the Issue

  1. Logon to Fiori;
  2. Open the F6164 app;
  3. Error "Modification is not allowed. Workflows cannot be modified" occurs.

Cause

The scenario WS78500095 is not active in the system.

Resolution

Activate the scenario WS78500095 by using the following navigation path:

SPRO -> SAP NetWeaver (or ABAP Platform) -> Application Server -> Business Management -> SAP Business Workflow -> Flexible Workflow -> Scenario Activation -> Activate scenario WS78500095.

Keywords

error modification is not allowed, workflows cannot be modified, manage workflow, group journal entries, WS78500095, scenario activation, SAP Business Workflow, Flexible Workflow
