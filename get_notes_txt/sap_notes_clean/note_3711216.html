Symptom

Customers using SSAM 2511 observe that the structure /MERP/PM_CREWITEM_ENTITY_STR is configured with the enhancement category “Cannot Be Enhanced”.

Because of this configuration, the structure cannot be extended through append structures or custom enhancements. 

Environment

SAP Service and Asset Manager 2511

Cause

 Programming error

Resolution

This issue is addressed by the following SAP Note: 3710851 – Update Enhancement Category for /MERP/PM_CREWITEM_ENTITY_STR

See Also

Refer to: SAP Note 3710851 - Update Enhancement Category for /MERP/PM_CREWITEM_ENTITY_STR
