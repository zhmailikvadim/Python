Symptom

When taking multiple subsequent readings for measurement points, valuations codes are carried over when they are not explicitly selected.

This SAP Note will fix this issue.

Other Terms

SAP Asset Manager

SAM

EDT

Measurement Points

Valuation

Readings

Reason and Prerequisites

SAP Asset Manager

WEB IDE

BAS

Solution

Download the attached 3709802.patch file and save it in a known location, preferably adjacent to your SAM.mdkproject

  1. Open a terminal in the root of your SAM metadata (i.e. SAM.mdkproject/metadata or SAM.mdkproject/metadata/base.definitions)
  2. Run the command `patch --no-backup-if-mismatch -p4 < path/to/3709802.patch` in the terminal. 
  3. Verify in the terminal window that the following files are listed as patched:
  • patching file 'Rules/Measurements/Points/EDT/MeasuringPointsEDTOnCommit.js'
