Symptom

  • Two queries with identical prompt texts fail to merge when running Web Intelligence reports.
  • There is HANA variable in query1, while no HANA variable in query2.
  • The problem occurs in all reports that use multiple queries.
  • No specific error message is displayed, prompts remain separate instead of merging.

Environment

  • SAP BusinessObjects Business Intelligence platform
  • Web Intelligence
  • HANA View

Reproducing the Issue

  1. Create a Web Intelligence document with two queries.
  2. In Query 1, there is variable and define a prompt in the query panel.
  3. In Query 2, define a prompt in the query panel without using HANA variable parameters.
  4. Refresh the document and observe that the prompts with identical text are not merged.

Cause

  • The query prompts created in queries 1 and 2 depend on different HANA variables.
  • In WebI, it is not possible to merge two prompts having the same question if their dependencies are different.

Resolution

This is by design.
