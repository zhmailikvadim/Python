Symptom

You are creating a Communication Arrangement, but during setup you cannot find the relevant Communication Scenario in the proposed list of scenarios.

Reproducing the Issue

  1. Navigate to the Application and User Management workcenter.
  2. Go to the Communication Arrangements view.
  3. Select New.
  4. The required Communication Scenario does not appear in the list. 

Cause

As the necessary Communication Scenario for the desired Webservice has not yet been created, this is expected system behaviour. Therefore a Communication Scenario must be setup to allow for the creation of a Communication Arrangement.

Resolution

  1. Navigate to the Administration workcenter.
  2. Switch to the Integration workcenter.
  3. Go to Communication Scenario sub-view.
  4. Click  New - Add Operations.
  5. Under Service name, search for the Web Service you wish to use.
  6. Select Add Operation,  then press OK.
  7. Name the Communication Scenario and click Save and Close.

Following this, the Communication Scenario you just created will appear in the list of Scenarios when creating a Communication Arrangement.
