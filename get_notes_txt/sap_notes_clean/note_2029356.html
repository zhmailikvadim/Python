Symptom

In pension administration, the entered data is not saved in the subprocess "Alternative Basic Payments", "Up. Limit Income from Wages/Pension Imp.", or "Up. Limit Further Pension Payment" despite the output of the message "Simulation data saved successfully".

Reason and Prerequisites

You use a Public Services regulation in which the pay seniority (BDA) is no longer used.

Solution

Import the Support Package that is assigned to this SAP Note or implement the advance corrections.
