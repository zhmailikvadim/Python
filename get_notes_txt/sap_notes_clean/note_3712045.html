Symptom

An error occurs when cancelling the onboarding process:

Error: [COE0018]Failed to save object objectType=ONB2Process, externalCode=XXXXXXXXXXXXXXXXXXXXXXXXX

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

SAP SuccessFactors Onboarding

Reproducing the Issue

  1. Navigate to the Onboarding Dashboard
  2. Search for the affected new hire
  3. Actions > Cancel Onboarding
  4. Error: [COE0018]Failed to save object objectType=ONB2Process, externalCode=XXXXXXXXXXXXXXXXXXXXXXXXX

Cause

In Admin Center > Manage Organization, Pay and Job Structures, the effective date ("Effective as of") of the event reason selected when cancelling the onboarding process is set after the new hire’s start date.

Resolution

The effective date ("Effective as of") must be set to a date prior to the new hire's start date.

Please follow the steps below to resolve the issue:

  1. Go to Admin Center > Manage Organization, Pay and Job Structures
  2. Search for the Event Reason
  3. Select Take Action > Make Correction.
  4. Update the "Effective as of" date to a past date (e.g. 01/01/1900)
  5. Save
  6. Attempt to cancel the onboarding process again.
