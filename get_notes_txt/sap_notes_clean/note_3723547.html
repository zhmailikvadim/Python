Symptom

You are unable to access 'Manage process' or 'Manage configuration' even though the corresponding services are active in SICF.

Environment

  • Human Capital Management (HCM)
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP HANA
  • SAP SuccessFactors HCM Core 

Reproducing the Issue

  1. Access Employee Central (EC).
  2. Navigate to Payroll Control Center (PCC).
  3. Attempt to open 'Manage process' or 'Manage configuration'.
  4. Observe the error message:
    "Service cannot be reached. Note the termination occurred in system with error code 403 and for the reason forbidden."

Cause

  • If you test the OData service in transaction /N/IWFND/MAINT_SERVICE, an authorization error occurs.
  • This issue is caused because the user does not have authorization for the object 'S_SERVICE'.

Resolution

Please assign the authorization object 'S_SERVICE' to the logon user. 

See Also

SAP Note 3165869 Odata : No authorization to access Service

Keywords
