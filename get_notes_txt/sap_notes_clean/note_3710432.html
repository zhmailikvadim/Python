Symptom

Which are the event rules we can't edit Ariba Guided Sourcing template?

Resolution

When an RFI or RFP template from the Classic Sourcing UI is converted to a Guided Sourcing template (by setting Guided Sourcing Template = Yes), the system automatically enforces predefined values for specific event rules. These settings are not configurable in Guided Sourcing templates.

The following rules are locked, hidden, and automatically set as shown below for Guided Sourcing RFI/RFP templates:

Type of RuleRuleValue
Auction FormatBidding format for the eventEnglish
Capacity TypeCapacity type for the eventStandard
Bidding RulesCan participants create bundlesNo (visible only when Can participants create alternative responses = Yes)
Bidding RulesCan participants create tiersNo (visible only when Can participants create alternative responses = Yes)
