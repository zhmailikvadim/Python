Symptom

This KBA introduces on how to modify screen control of IT3386 per subtype. "Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Environment

  • Human Capital Management (HCM)
  • SAP ERP 6.0
  • SAP ERP Core Component 6.0
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for HANA

Resolution

  1. Configure feature P3386 via t-code PE03 and add the target subtype according to your business requirement, like the following example.
    For subtype 1, set the return value as '1'.
  2. Go to view V_T588M via t-code SM30 and input module pool MP338600.
    Please make sure the variable key maintained in V_T588M matches the return value in feature P3386. In this example, maintain variable key as '1' in V_T588M. 
  3. You can modify screen control of IT3386 per variable key which corresponds to the return value of subtype in feature P3386. For example, you can hide field Q3386-GBDAT of IT3386 subtype 1 like the following example.
  4. After saving the configuration in V_T588M, Q3386-GBDAT of IT3386 subtype 1 is hidden.

    You can modify feature P3386 and V_T588M MP338600 to meet your business requirement.

Keywords

IT3386, Subtype, Screen control, P3386, MP338600
