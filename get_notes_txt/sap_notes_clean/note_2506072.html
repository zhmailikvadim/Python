Symptom

You care unable to find where to change the client number for the Desktop Connection for SAP CRM add-on.

Environment

  • Desktop Connection for SAP CRM.
  • SAP CRM 7.0, 7.01, 7.02, 7.03.

Reproducing the Issue

  1. Install and Configure Desktop Connection for SAP CRM.
  2. After some time, its required to change the client the add-on is pointing to.
  3. It’s not clear where this configuration is so you are unable to update the client from the client side.

Cause

System Design

Resolution

  1. On the Desktop Connection client side, it’s not possible to update the client dynamically.
  2. This is only possible on the CRM server side.
  3. You can find detailed information about how to configure this in the Desktop Connection for SAP CRM Technical Configuration Guides.
