This issue is tracked in:  Internal Incident ID# 1970287345 for Known Issue

Environment

SAP Data Services 4.2

Reproducing the Issue

With the Capitalization option set to Mixed, process an address in Ada, OK or Ama, LA.

Keywords

DS, DQ, URAC, US, ADA, AMA

Reason and Prerequisites

Root Cause

ADA and AMA are also acronyms for the American Dental Association and the American Medical Association.  Current functionality does not distinguish if ADA and AMA are ogranization name acronyms or localities and will UPPER-CASE the words when matched to Casing Dictionaries - no matter what type of data they are.

Solution

There is no workaround for ADA and AMA to be output as Mixed-Case at this time.
