Symptom

No standard Professional Services form template is available for Austria. When configuring billing output, the system does not deliver a country-specific Professional Services form template for AT.

Environment

SAP S/4HANA Cloud Public Edition 

Reproducing the Issue

  1. Go to Maintain Form templates 
  2. Check search SDBIL_CI_PROF_SRV_AT
  3. No form template pre-delivery 

Cause

This is a missing pre-delivery form template.

Resolution

Currently, there is no predelivered standard Professional Services form template for Austria. As a workaround, create a custom form template by copying the standard template SDBIL_CI_PROF_SRV_DE and apply the required adjustments to meet Austrian localization requirements.

If a standard Professional Services form template for Austria is required, a feature request can be submitted via the SAP Influence Portal. For guidance, refer to:

  • 2963059 - How to Submit Feature Requests for SAP S/4HANA Cloud Product
