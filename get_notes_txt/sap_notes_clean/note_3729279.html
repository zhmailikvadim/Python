Symptom

1. The Italy Withholding Tax Report did not allow multiple Supplier selection in the run application, limiting user flexibility during report execution.
2. The signature section ( FIRMA DEL SOSTITUTO DI IMPOSTA ) in the report correspondence form was blank, resulting in incomplete or non‑compliant report submissions.

Other Terms

 IT_WHT_DCL_CU, IT, FIRMA DEL SOSTITUTO DI IMPOSTA, FIWTIT_CU_BP_CERTIFICATE, Italy, Signature, Supplier

Reason and Prerequisites

This Note is only valid for country/region Italy.

Solution

The run application has been enhanced to allow multiple vendor selections, enabling users to process several vendors in a single execution.
The blank signature issue has been corrected by updating the report to populate the appropriate signature details in the output.
