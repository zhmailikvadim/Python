Symptom

  • Hide Sub Report in Canvas Report.
  • Hide specific sub-reports from the final output while keeping others visible.

Environment

  • SAP SuccessFactors HCM Core

  • Report Center

  • Canvas Report

Reproducing the Issue

  1. Log into SuccessFactors.
  2. Navigate to reporting.
  3. Access the affected report canvas.
  4. Identify the sub-report in Preview Canvas report.
  5. Hide it.

Cause

  • Expected result in Canvas Report. 

Resolution

  • It's not possible to hide sub report in canvas report.
  • Recommend submitting an enhancement request to allow the Product Management Team to evaluate the suggestion for future updates.
  • Refer to the following resource for guidance on submitting enhancement ideas: 2090228 - How to submit enhancement ideas for SAP SuccessFactors Products 

See Also
