Symptom

Depending on the setting, you can create scoring data for a patient with outpatient cases for each movement/visit day (for all cases or for each outpatient case (day/case view)).

If a visit day falls on an admission or discharge day for an inpatient case  of the patient, the system does not create any scoring data for this outpatient movement day.

If necessary, no outpatient ELDA/P321 admission notification is created for this.

Other Terms

LKF model 2026, SCOX, scoring, scoring relevance

Reason and Prerequisites

Prerequisites:

  • Country Version AT (Austria)
  • LKF Scoring Model 2026

Causes:

  • Legal change
  • Further development

Solution

The adjustments are available for the Support Packages of the specified releases. If you require the changes in advance or if the SAP Note is specified as a prerequisite, implement the correction instructions in your system and, if necessary, perform manual tasks.

The new Scoring Relevance checkbox is available for outpatient movements. Select this if you want this movement of an outpatient case to be reported for scoring on an admission or discharge day of the patient outside the inpatient stay.

As before, you do not need to select this checkbox for scoring for other visiting days outside of an inpatient stay. Outpatient movements within the stay are not taken into account despite the selection as before.

In parallel, this movement would also be reported as an outpatient admission as part of ELDA/P321 if there was previously no notification for this outpatient case in this quarter.

Caution: The respective messages are not yet issued. This SAP Note initially enables only labeling.
