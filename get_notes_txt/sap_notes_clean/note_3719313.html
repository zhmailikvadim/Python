Symptom

When configuring Rate 2.0 custom lookup, I do not see custom field from work order module while choosing custom field as Key.

Resolution

  1. Navigate to Admin → Custom Fields.
  2. Create a new custom field and set the Parent Object to Job Posting.
  3. Link the custom field to the following modules:
    • Work Order
    • Worker
    • Work Order Revision
  4. Create a new Rate 2.0 Custom Lookup.
  5. Verify that the newly created custom field is now available for selection.
