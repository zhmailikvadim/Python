Symptom

You are using Transaction Manager within Treasury and Risk Management. You transaction TOENE to process the Hedge Management Cockpit for a specific hedging area using a given snapshot ID. The corresponding hedging area version contains multiple filters for hedges. When you drill down to the key figure "Net Hedges", the system may terminate with a runtime error. The runtime error appears with information similar to the following:

Runtime Errors SAPSQL_PARSE_ERROR
Except. CX_SY_DYNAMIC_OSQL_SYNTAX
ABAP: Program CL_TOE_QUERY_FACTORY==========CP
Application Component FIN-FSCM-TRM

Other Terms

TOENE, TOE_QUERY_HEDGES_IMP, DETERMINE_RELEVANT_DEALS, LT_CHECKED_WHERE

Reason and Prerequisites

Program Error

Solution

Preliminary correction via Correction Instruction or implementation of the corresponding Support Package.
