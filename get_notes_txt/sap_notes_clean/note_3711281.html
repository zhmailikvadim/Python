Symptom

After setting up AI agents with your CX AI Toolkit and Service Cloud v2 system you can not see the "ai case insights" tab on the right side in the side pane of the case detail view.

Environment

  • SAP CX AI Toolkit
  • SAP Service Cloud V2

Reproducing the Issue

  1. Navigate to the case detail view in the SAP Service Cloud V2.
  2. Observe that the AI Insights Panel is not visible in the side panel.

Cause

Principal Propagation is not enabled.

Resolution

  1. Set up Principal propagation to enable the visibility of the AI Insights Panel.
  2. Refer to the guide for enabling principal propagation for SAP Sales and Service Cloud Version 2.

    Please note that principal propagation isn't possible if you're using a corporate identity provider that hasn't been added to SAP Cloud Identity Services. You're required to use SAP Cloud Identity Services as a proxy when you sign in to SAP Sales and Service Cloud Version 2 using single sign-on.
