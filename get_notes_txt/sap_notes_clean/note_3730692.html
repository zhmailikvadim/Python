Symptom

  • The option to create an application does not appear in the View Browser application.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Log into the system.
  2. Open the View Browser application.
  3. Observe that the "Create Application" button is greyed out.

Cause

The Key User Extensibility feature is not activated in the Extensibility Settings.

Resolution

  1. Open the "Extensibility Settings" app.
  2. Activate "Key User Extensibility".
  3. Open the "View Browser" app and verify that the "Create Application" button is now available.
