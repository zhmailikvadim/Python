Symptom

You are using the Fiori app "Migrate Your Data " to migrate data to SAP S/4HANA using the migration approach "Migrate Data Directly from SAP System". However, a communication error occurs.

In the app, the following error could be displayed:

  • "RFC Error: Access using a 'NULL' object reference is not possible.".

In transaction ST22, you can view the runtime error:

  • "OBJECTS_OBJREF_NOT_ASSIGNED_NO"

Other Terms

Migrate Your Data, Migrate Data Directly from SAP System

Reason and Prerequisites

Deployment: SAP S/4HANA 2025 FPS01

Solution

Implement the corrections outlined in this SAP Note in the SAP S/4HANA system in accordance with the correction instructions.
