Symptom

  • When revising an SOW, the SOW workers' end dates were not extended to match the SOW
  • An SOW was revised without changing the end dates and the SOW Workers were not extended

Resolution

When revising an SOW with the Automatically Extend SOW Worker End Dates to Match SOW End Date configuration enabled users have an option to extend SOW Workers to match the End Date of the SOW

Workers are only extended to match the End Date of the SOW when the SOW End Date has been changed during the revision process

If the Automatically Extend SOW Worker End Dates to Match SOW End Date option is selected but the SOW's end date is not changed then the SOW Worker End Dates will not be updated
