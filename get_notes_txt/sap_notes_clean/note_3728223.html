Symptom

You are using PL/SK/RO Fixed Asset Form Document Reporting Compliance (DRC) reports introduced in the implementation overview SAP Note 3224734.

You have the following problem: Document reporting compliance (DRC) run for PL, SK, RO Fixed Asset Acquisition form DRC report generated acquisition form numbers for certain types of accounting documents for which they should not have been generated. You would like to prevent this from happening in the future. You would also like to cancel the acquisition form numbers that have already been generated in this way and stored in the FIAAGEND_FRMNUM table with a reference to this DRC run and accounting documents.

Other Terms

PL_FXDAST_FORM_ACQ, PL_FXDAST_FORM_LTR, PL_FXDAST_FORM_RET,  SK_FXDAST_FORM_ACQ, SK_FXDAST_FORM_RET, RO_FXDAST_FORM_ACQ, RO_FXDAST_FORM_LTR, RO_FXDAST_FORM_RET,

G_FXDAST_DOC_TYPES, SRF_REPORT_RUN_ON_DELETE, FIAAGEN_FRM_RUN_ON_DELETE

Solution

Implement this SAP Note, which solves the described problem, as follows:

1) A new parameter G_FXDAST_DOC_TYPES Document Types is delivered for the Fixed Asset Acquisition form DRC report. This parameter is defined in the configuration activity Setting Up Your Statutory Reporting (transaction SM34, view SRFVC_RPG_ENT). You can use this parameter to exclude irrelevant accounting document types. If the parameter is empty, all document types are selected.

2) The solution for all Fixed Asset Forms (Acquisition, Location Transfer, Retirement) has been extended with new functionality. If a specific DRC run is deleted, the records from the FIAAGEND_FRMNUM table that refer to this DRC run will also be deleted. If all DRC runs referring to a specific accounting document are deleted in this way, a new form number (acquisition or retirement form) will be generated for this accounting document during the next DRC run.

As a general rule, SAP recommends that you install a solution by applying a Support Package. However, if you need to install a solution earlier, use the SAP Note Assistant to implement the correction instruction.
