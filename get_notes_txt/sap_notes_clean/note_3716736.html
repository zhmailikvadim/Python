Symptom

The self service description field in the pay component object is not visible in the compensation information section of the employee profile

Environment

SAP SuccessFactors Employee Central - Foundation Object

Reproducing the Issue

  1. Navigated to Manage Organization, Pay and Job Structure.
  2. Selected the relevant Pay Component.
  3. Verified that:
    1. Display on Self Service is set to Yes.
    2. A value is maintained in the Self Service Description field.
  4. Navigated to the respective User Profile.
  5. The maintained Self Service Description is not displayed in the user profile under the Compensation Information portlet.

Resolution

  1. Engineering team plans to remove this field in a future release, as it currently has no functional impact.
  2. You may contact your Customer Success Manager or call our hotline number to know status of this enhancement request ECT-267827.
