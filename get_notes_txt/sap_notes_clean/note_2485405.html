Symptom

At least one of the following symptoms occurs:

  1. You run transaction Off-Cycle Workbench. You select an employee assigned to an organization included in Report Management System. You run a bonus payment payroll and save data for this employee. The system shows a runtime error OBJECTS_OBJREF_NOT_ASSIGNED in ABAP program CL_HRPA_INFOTYPE_CONTAINER====CP.
  2. You run payroll for an employee. The system sets the date in Infotype 3400 RMS Status of your employee to 31.12.9999, while it should set the date to the start date of the earliest calculated period for that employee.
  3. You run payroll for an employee with more than one personnel assignment in a system, where Concurrent Employment is switched on. The system sets the date in Infotype 3400 RMS Status only for the first personnel assignment of this employee.
  4. You change a shared infotype for an employee with more than one personnel assignment in a system, where Concurrent Employment is switched on. The system sets the date in Infotype 3400 RMS Status only for the first personnel assignment of this employee.
  5. You want to implement SAP Note 2486809. In this case, you need to install this SAP Note also to prevent a possible runtime error. The runtime error can occur during processing of a reporting area in Report Management System after implementing SAP Note 2486809 in the system. It occurs only if you have customized your organization to be simultaneously linked to two different reporting areas for the same report type.

Other Terms

RMS, PUOC_33, HR99RU_REPSE_MAINT, PA30, PC00_M33_CALC, IT3400, RRDAT, RRDAF, RETDT

Reason and Prerequisites

This SAP Note is relevant only for Russia.
You have activated Report Management System functionality.

Solution

As a general rule SAP recommends that you implement the Support Package specified for your release.
However, if you need to install a solution earlier, use the SAP Note Assistant to implement the correction instruction. You can find more information about the SAP Note Assistant in SAP Support Portal, under SAP Note Assistant.
