Symptom

  • Users assigned the "MEMBER" role are able to change factsheets and attributes for which they are not subscribed.

Environment

LeanIX Inventory subscriptions

Reproducing the Issue

  1. Assign a user the "MEMBER" role in LeanIX Inventory subscriptions.
  2. Verify the user's ability to edit factsheets and attributes not subscribed to.

Cause

The "MEMBER" role has edit permissions for the Fact Sheet details.

Resolution

  1. Ensure that the "MEMBER" role is set to read-only (no edit permissions).
  2. Assign edit permissions based on subscription roles in Advanced Permissions (e.g., Observer, Responsible).
  3. Refer to the following documentation for guidance:
  • Subscription Type Check: Subscription Type Check
  • Advanced Permissions: Advanced Permissions
  1. Test the changes with two users assigned the "MEMBER" role: one subscribed to a factsheet and one not subscribed, to confirm only the subscribed user can edit.
