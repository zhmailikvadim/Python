Symptom

IL Withholding tax report - form 856 - 

Israeli tax authority published new changes for reporting year 2025 for form 856.

You can see the changes that were published in the attached file "IL tax authority new instructions for 2025.pdf".

The changes are in record 60 structure as follows:

  1. New Field 17a was added - Additional Tax From All  Income Sources.
  2. New field 17b was added - Additional Tax From Capital Sources.
  3. The length of field 18, FILLER, was changed from 52 to 32 characters.

As the new fields that were added are not relevant for SAP suppliers and customers, by default, they will be filled with zeroes.

If you wish to add such amounts for some of your suppliers, you can use the BAdI /ATL/BADI_WHT_REPORT, method GENERAL_USER_EXIT (if you don't have this BAdI in your system you should use the function module /ATL/WT_CUSTOMER_EXIT2 user exit).

In the BAdI method or function module, you should add the relevant amounts to the relevant suppliers in the new fields that were added to tables VENDORS, VENDORS_MONTHLY, VENDORS_MONTHLY_SUG accordingly.

Pay attention, according to tax authority's instructions, the amounts of the new fields should be added also to the amount of the withholding tax that was deducted (field 12 in record 60). So, if you updated the amounts in the user exit BAdI or function module, you should also need to update the amount of the withholding tax that was deducted in the 3 mentioned tables accordingly.

In addition, if you are using the method GET_FICA_DATA in BAdI /ATL/BADI_WHT_REPORT or function module /ATL/WT_CAFI_CUSTOMER_EXIT1, you can update the new fields in the tables VENDORS_CA, VENDORS_MONTHLY_CA and VENDORS_MONTHLY_SUG_CA accordingly, and as mentioned, you should also update the amount of the field withholding tax that was deducted.

Other Terms

IL localization, Israeli localization, WT report, withholding tax report, /ATL/VK01_46, IL_WHT_856

Reason and Prerequisites

This SAP Note is relevant only for The Israeli Localization.

Solution

Import the Support Package specified for your release or implement the attached correction instruction.

As a general rule SAP recommends that you implement the Support Package specified for your release. However, if you need to install a solution earlier, use the SAP Note Assistant to implement the correction instruction. You can find more information about the SAP Note Assistant in SAP Support Portal, under SAP Note Assistant.
