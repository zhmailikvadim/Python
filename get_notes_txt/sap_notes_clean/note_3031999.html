Symptom

You are using the SAP Secure Login Server Command Line Interface and see the following issues:

SAP Secure Login Server Command Line Interface:

  1. The sapslscli command enroll or renew shall be used as Certificate Life Cycle Management but no output is shown after executing enroll or renew command.
  2. The sapslscli command does not show the version

Other Terms

sapslscli, CLM, Certificate Lifecycle Management, ICM, Trusted CAs, SLL

Reason and Prerequisites

SAP Secure Login Server Command Line Interface:

  1. You are using the sapslscli in conjunction with the CommonCryptoLib 8.5.36. Due to a programming error, the sapslscli crashes if enroll or renew are executed.
  2. The version output was not implemented

Solution

SAP Secure Login Server Command Line Interface:

Download and install the SAP Secure Login Server Command Line Interface (SECURE LOGIN LIBRARY) 3.0 SP02 PL11 (or higher). The following fixes comes with this patch:

  1. The programming error was fixed.
  2. Using the command sapslscli -h shows the version
