Symptom

After implementing SAP note 3593210 - [Src] Central Payment in S/4HANA 2025 - Enhancement in Source System, the content of view V_FIN_CFIN_HIBCA can be transported. However, due to the configuration in this maintenance view being system/client specific, it is not appropriate to allow transport.

Other Terms

Central Finance, CFin, Central Payment

Reason and Prerequisites

You have implement SAP note 3593210.

Solution

  1. Apply the corresponding support package or implement this SAP Note in your source system.
  2. Run UDO report FIN_CFIN_NOTE_3723589.
