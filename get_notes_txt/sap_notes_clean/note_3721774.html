Symptom

The right to return flag is not visible when an employee is assigned to a higher duty assignment.

Environment

SAP SuccessFactors Employee Central - Higher Duty or Temporary Assignment

Reproducing the Issue

  1. Assign an employee to a higher duty assignment with full responsibility.
  2. Observe that the right to return flag does not appear on the position org chart.

Cause

The position field in job information is set as mandatory, preventing the creation of the right to return object during the position follow-up process.

Resolution

  1. Modify the position field in job information to be non-mandatory.
  2. Assign a higher duty to a new user and verify if the right to return flag appears correctly.
