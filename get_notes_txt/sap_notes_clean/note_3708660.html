Symptom

You are trying to use file upload option to create new address data for business partner, but there is no address number generated.

Environment

Master Data Governance on S/4HANA

Reproducing the Issue

Use File Upload option to add new address data for business partner

Cause

The creation of new addresses is not supported. Addresses require a technical identifier (entity type ADDRNO) which cannot be created via the file upload.

Resolution

Please consider using SAP MDG, Consolidation and Mass Processing.

For more details, please refer to session "Scenario: Mass Change" in SAP Note 2847807 - MDG-BP/C/S/CA: Usage of MDG Tools and Processes.

See Also

2847807 - MDG-BP/C/S/CA: Usage of MDG Tools and Processes

Keywords

Address creation, File upload, Address number, entity type ADDRNO
