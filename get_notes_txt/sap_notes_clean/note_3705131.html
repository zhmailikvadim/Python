Symptom

Guidance on estimating the SAPS required for the  Edge Integration Cell (EIC) environment.

To use the provided sizing tool effectively, the following prerequisites must be met:

  1. Determine the throughput for your scenarios.
  2. Based on the complexity of the flows, assign the values in the Excel utility.

These two parameters are essential for accurate estimation using the sizing tool.

Environment

Edge Integration Cell (EIC)

Cause

EIC Sizing Calculation

Resolution

It is recommended to first refer to the SAP-provided Sizing Guide available on the Help Portal.

For further assistance, use the following tool for sizing: Attached EIC_Sizing_Calculator_Tool_v3.xlsx

This tool will help you calculate the required SAPS based on your inputs.

Components Covered

  1. Worker

  2. Policy Engine
