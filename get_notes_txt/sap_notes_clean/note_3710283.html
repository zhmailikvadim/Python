Symptom

  • Outbound interface from SAP Cloud Integration to Salesforce fails.
  • CPI message processing fails with HTTP 403 – Forbidden.
  • CPI internal traces show the response body error:
    • "message": "This session could not be mutually authenticated for use with the REST API",
      "errorCode": "MUTUAL_AUTHENTICATION_FAILED"

Environment

  • Cloud Integration
  • SAP Integration Suite

Cause

The error originates from Salesforce authentication and authorization validation. Salesforce rejects the inbound REST API call due to mutual authentication failure.

Resolution

  1. Validate Salesforce-side configuration, including:
    • Authentication mechanism (for example, OAuth or certificate-based authentication).
    • Session validity and API access permissions.
    • Connected App configuration and user authorization.
  2. Engage the Salesforce support or Salesforce administration team to investigate the authentication failure.
  3. Refer to public Salesforce documentation that explains this error and its causes:
    • MUTUAL_AUTHENTICATION_FAILED (explains scenarios where Salesforce rejects REST API calls due to certificate, session, or connected app configuration issues).
