Symptom

In MDG UI, you have predefined some search criteria and save the search as either private or global variant. You expect to set such saved search to be opened automatically as default search onwards.

Reproducing the Issue

  1. In MDG Search UI(Material/Business Partner/FI Objects), you predefined search criteria and save the search criteria either as Global or Private Saved Search.
  2. Load the search page again and you expect the saved search to be loaded by default.

Resolution

The saved search either can be saved as Global(Saved search can be used by all users) or Private(Can only be used by the user who creates it). There might be multiple global or private saved search created by users. If user needs to run one of the saved search as default, they need to go to the settings of the Search criteria section and select the corresponding saved search from the dropdown list of "Default Search". This will load the corresponding saved search when the search page is opened again. And if you expect to execute the saved search on opening the search UI, please also check "Run default Search on Open" indicator.

Note, there is a difference betwen MDG materail saved search as user needs to set the default search themselves. For BP and FI Objects search view, the default saved search can be set globally.
