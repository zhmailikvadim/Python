Symptom

Adding a user to a Partner ID

Resolution

Any Administrator of the account is able to add other users with the following steps:

  1. Log into the SAP Customer Data Cloud Admin Console 
  2. Click on the "Admin" link
  3. Click on the "Invite Administrator" button
  4. Enter the email address of the user
  5. Select the group the user should belong to

In circumstances where this is not possible, either due to lack of access or requiring elevated assistance from SAP Support Team, the SAP Support Team will ask to perform security verification to ensure that the request is authorized.  Please refer to the following article for more information on How do I make a support request that requires security verification?.
