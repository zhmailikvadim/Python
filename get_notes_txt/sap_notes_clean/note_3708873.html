Symptom

Search for open documents in the Service module returns documents in all statuses, even when searching for active documents only.

This behavior can be observed for the service module documents: Service Call, Service Contract and Equipment Card.

Reproducing the issue

Scenario:

  1. Go to Service → Service Call and go to Find mode using the shortcut Ctrl+F. 
  2. Notice that the field Call Status shows the value 'Open'.
  3. Type '*' within the Call No. field and press Enter or press the Find button.

Expected behavior:

Service calls in open status are shown in the List of Service Calls.

Actual behavior:

Service calls in all statuses are shown in the List of Service Calls.

Cause

System Limitation

Solution

This is a system limitation.

Workaround:

Before searching, change the Call Status to a value other than "Open" and press Find.
Then change the Call Status back to "Open" and press Find again.
