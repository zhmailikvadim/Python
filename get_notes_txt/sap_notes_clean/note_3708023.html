Symptom

In transaction EEDMSETTLDISP, the settlement run fails when the system uses the Settlement Document (SETTLDOC) field to retrieve profile values instead of the Selection Document (SETTLSELDOC) field when it is provided.

Other Terms

EQS, US4G, EEDMSETTLDISP, Settlement Run, Settlement Document, Settlement Selection Document, SETTLDOC, SETTLSELDOC, profile values

Reason and Prerequisites

The error occurred because of a technical issue where the system incorrectly used SETTLDOC to retrieve profile values instead of SETTLSELDOC.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the system uses SETTLSELDOC correctly to retrieve profile values, and the settlement run executes successfully.

You need to run the settlement steps again. You can verify the fix by checking that the settlement run executes successfully without errors.
