Symptom

Purchase requisition (PR) created by another user is visible in the Classic View but not visible in the Guided Buying View

Reproducing the Issue

  1. Enable the Purchasing Manager role for a user.
  2. Check the visibility of a purchase requisition (PR) created by another user in the Classic View.
  3. Observe that the PR is visible in the Classic View but not in the Guided Buying View.

Resolution

The Guided Buying (GB) functionality does not support viewing purchase requisitions (PRs) or purchase orders (POs) created by other users unless the user is designated as an approver.

If the requirement cannot be met, raise an enhancement request via the Influence Portal for the Product Management team to evaluate the feasibility of adding new functionalities.

See Also

For more information on how to submit an enhancement request via the Influence Portal, please refer to: How do I create an improvement or enhancement request for Ariba Procurement?
