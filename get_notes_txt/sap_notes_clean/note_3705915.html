Symptom

Cannot delete approvers from approval flow in Charge Reconciliation file

Resolution

Users with Customer Admin access are the only ones who can delete approvers from the approval flow. Users should verify that they have the appropriate access rights before attempting to delete any approvers.

Similar to the Invoice Reconciliation process, if a Customer Admin is added to the approval flow, the Customer Admin must first take action and then proceed to delete the respective node.

Please note that an exception will be raised for this document. These exceptions must be cleared before the user can make any changes to the approval flow.

Keywords
