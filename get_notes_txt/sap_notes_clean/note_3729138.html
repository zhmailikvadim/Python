Symptom

  • A shared view displays missing data when specific filters are applied during data preview.
  • Analytic model or fact view displays missing data during data preview.
  • Data mismatch when previewing a Analytic Model or View

Environment

  • SAP Datasphere

Cause

  • The issue is caused by new data in one of the source views, which was not yet reflected in the (shared) view. The (shared) view uses virtual access to the source views, and the data in the source views needs to be persisted for the shared view to reflect the updated data or

Resolution

  1. Use Data Pipeline Analyzer to troubleshoot the issue.
  2. The tool will show notification bubble which symbolizes the current status of the model.
  3. In our Help Portal you will find step by step exercises.
