Symptom

  • Error messages encountered during the Initiate Onboarding Process:
    Validate HRIS data for user.payComponentRecurring: xxx for Object payComponent is not a consistent value with the associated Object Country/Region with value XXX.

Environment

SAP SuccessFactors Onboarding

Reproducing the Issue

  1. Attempt to Initiate Onboarding for Internal hires through Recruiting
  2. Observe the candidates lands in BPE to-do error

Cause

  • The pay component ID "xxx" is not maintained for the Country/Region value "XXX" under Manage Data.

Resolution

  1. Verify that the particular Pay Component is Active in Manage Organization, Pay and Job Structures
  2. Navigate to Manage Data and ensure the Pay Component ID "xxx" is correctly maintained for the Country/Region value "XXX".
