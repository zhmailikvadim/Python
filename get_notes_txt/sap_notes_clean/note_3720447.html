Environment

S/4 HANA system

Kernel 793 PL345

Reproducing the Issue

Performing of some operations like double-click on a row or collapse/expand of rows in ALV GridView control in SAP GUI for HTML

Cause

program error

Resolution

Follow SAP Note 3715324 - ALV GridView: CORE DUMP after performing some operations like double-click or collapse/expand rows.

If the system has kernel 793 PL300 or higher, SAPWEBGUI.SAR file could be downloaded. 

If the system has kernel 793 lower than PL300, DW.SAR file could be used.

See Also

 Note 3715324 - ALV GridView: CORE DUMP after performing some operations like double-click or collapse/expand rows.
