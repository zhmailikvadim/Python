Symptom

In the SAP Fiori app Manage PCNs, the Print Location value help is empty in the Initiate Printing Dialog.

Other Terms

 ISLS, Manufacturing, Intelligent Clinical Supply Management, Manage PCNs, F6414, Initiate Printing, Print Location

Reason and Prerequisites

You use SAP Intelligent Clinical Supply Management or this SAP Note is a prerequisite of another SAP Note.

Solution

Implement the attached correction.

Execute calculation of SAPUI5 Application Index:

    - Run report /UI5/APP_INDEX_CALCULATE via transaction SE38.

    - Enter the SAPUI5 Repository '/CTUI/MANAGE_PCN' on the Single SAPUI5 Repository Only tab, and execute.
