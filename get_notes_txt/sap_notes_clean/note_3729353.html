Symptom

This Release Information Note (RIN) contains information and references to notes for implementing Support Package Stack 35 of SAP NetWeaver 7.40

Note: This note is subject to change.

Other Terms

SAP Netweaver 7.40 SP Stack  35 implementation.

Reason and Prerequisites

You are planning to implement Netweaver 7.40 SPS35. Before initiating the update/upgrade scenario, SAP recommends you to check the SP equivalence for this SP Stack - 1850327

Also, please see 2179450 - Activation errors for CDS views if feature is missing from database.

Please implement 2725610 - NR: TIME_OUT dump during automatic interval creation before update.

Please refer 2309320 for PI CTC: ABAP Dumps WF-BATCH MESSAGE_TYPE_X after installation, update or upgrade

Solution

This SP stack can be downloaded via SAP Service Marketplace

It contains corrections for several known issues. For more details please see below.

Java:

NW7.40 SPS35 shares the same Java stack as that of NW7.31 SPS38.

Therefore SP23 Java components will be downloaded when NW 740 SPS35 is selected. Please refer 2361845 for why there is a mismatch in the Java SP numbering.

Please refer to SAP Note 3078654 for Netweaver releases with redelivered JAVA stacks.
