Symptom

In class CL_DD_EXT_TABL_INCLUD_PROVIDER, method DETERMINE_FLAT_INCLUDES selects include_fields~* with an alias name. Setting an alias for a ~* selection has no effect.

This issue has no adverse effects, but the code as it is may be misleading.

Solution

Implement this SAP note or install the corresponding support package.

This is a technical correction to comply with coding guidelines. The change does not fix a functional issue.
