Symptom

  • The Email sent successfully, but the report was not delivered to the recipient.
  • The following error messages were recorded in the logs 
    CSISMTPClient::WaitForReply SMTPreply: 550 5.4.1 Recipient address rejected: Access denied
    CSISMTPClient::WaitForReply SMTPreply: 452 4.5.3 Requested action not taken: insufficient system storage 

Environment

  • SAP BusinessObjects Business Intelligence (BI) Platform 4.x/2025
  • All Operating system

Reproducing the Issue

1.In BI Launchpad, open any report.
2.Click Schedule.
3.Under Destinations, choose Email.
4.Enter the recipient email ID.
5.Click Schedule to run the job.
The system shows “Email sent successfully”, but the email does not reach the SMTP server, so it cannot be delivered to the recipient.

Cause

The recipient address was rejected by the mail server due to invalid email or access restrictions.
The mail server could not process the request because there is not enough storage space available.

Resolution

  • SMTP 550 indicates that a specific email address is invalid for mail delivery
  • SMTP 452 indicates that the mailbox for a specific email address is full, resulting in the rejection of new messages.
  • The mail system continues to send emails normally to all other users; only the email addresses with these issues fail to receive the messages.
