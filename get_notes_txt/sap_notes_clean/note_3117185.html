Symptom

In the detail display of a meter reading document in transaction EL31, a portion that does not belong to the meter reading unit in the same line appears on page 2.

Reason and Prerequisites

The meter reading unit in the meter reading document is (for whatever reason) not the same as the meter reading unit in the corresponding time slice of the installation.

(The portion displayed corresponds to the meter reading unit of the installation.)

Solution

Implement the attached correction instructions or import the specified support package.
