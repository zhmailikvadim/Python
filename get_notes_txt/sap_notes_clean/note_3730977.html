Symptom

  • Opening the Bank Integration launchpad page results in the error: "Failed to execute HTTP request: 404 (404)" with Page ID SAP_FIN_PGT_BSNAGT_PC and space ID SAP_BR_BANK_INT_SPECIALIST.
  • The business role SAP_BR_BANK_INT_SPECIALIST was copied from the SAP delivered template and assigned to the user.
  • The page template SAP_FIN_PGT_BSNAGT_PC is not available in the Manage Launchpad Pages app.
  • The issue exists in release 2602 or higher.

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental 

Environment

SAP S/4HANA Cloud Public Edition 2602 and higher

Reproducing the Issue

  • Navigate to the Bank Integration launchpad page.
  • The 404 error appears immediately upon opening the page: "Failed to execute HTTP request: 404 (404) — Page ID: SAP_FIN_PGT_BSNAGT_PC | Space ID: SAP_BR_BANK_INT_SPECIALIST" 

Cause

The standard Bank Integration pages were removed in release 2602, and the associated page template SAP_FIN_PGT_BSNAGT_PC is no longer available in the system.

Resolution

To create the page, follow the below steps:

  1. In the Manage Launchpad Spaces app, create a custom space using the delivered template SAP_BR_BANK_INT_SPECIALIST as the template, this ensures the space is aligned with the Bank Integration role structure.
  2. Assign the newly created custom space to the relevant business role in the Maintain Business Roles app to make it visible to the intended users.
  3. In the Manage Launchpad Pages app, create a new custom page that will serve as the replacement for the deleted standard Bank Integration page.
  4. Search for the required business catalog associated with the Bank Integration scenario that is SAP_FIN_BC_BSNAGT_PC to identify the available applications.       

  5. Assign the identified applications to the custom page.
