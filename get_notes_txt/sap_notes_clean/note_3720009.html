Symptom

You receive a reversal of the health insurance fund and assign it using the report Assignment of Compensation Benefits Inbound (RPCEEHD0_IN). When you process the notification using the report Process Confirmations of Compensation Benefits (RPCEEVD0_IN), the system issues error message HRPAYDESVEE 130 ("The reversed inbound notification was not found"). This error occurs only in distributed reporting.

In addition, reversals of inbound notifications are not displayed in the administrator list for EEL inbound notifications (RPCEELD0_IN) if no assignment to an inbound notification has been made.

Other Terms

Social Insurance

Solution

The following corrections are made:

  • In future, reversals of inbound notifications that could not be assigned to an inbound notification are displayed in the administrator list for EEL inbound notifications (RPCEELD0_IN).
  • During the assignment of EEL inbound notifications, the notifications will be sorted in ascending order by creation date (DSLW-ED). If the creation date is the same, reversals are first assigned.

To clean up reversals that have not been completely assigned, the assignment must be repeated. To do this, you must first undo the incomplete assignment. The source code of the report ZRP_PAYDE_EE_UNDO_ASSIGN is attached to this SAP Note. This report must be created and executed in the affected decentralized clients.

As a result, all inbound reversals for which no assignment to an inbound notification has taken place are returned to the status Received. The assignment to the personnel number is also undone. The status history (P01EE_HIST) is deliberately not cleaned up so that the procedure can be traced.

You can then start the assignment report RPCEEHD0_IN in the decentralized systems (RFC connections = NONE).

After the cleanup, the report ZRP_PAYDE_EE_UNDO_ASSIGN can be deleted.

Implement the attached correction instructions or import the specified support package. After you import the HR Support Package or implement the correction instructions, you do not have to perform any further follow-on activities.
