Symptom

  • Supplier contacts exist in Sourcing but not in Supplier Management.
  • How do I sync Supplier contacts from Sourcing to Supplier Management?

Cause

Resolution

To sync Supplier Contacts from Sourcing to Supplier Management:

  1. Export Supplier Users from Sourcing:
    1. Click Manage > Administration > Choose Site Manager > Data Import/Export.
    2. Click on the Export tab.
    3. Search for the data export task Export Supplier Users (CSV).
    4. Click Export.
    5. On the Specify an adapter source dropdown menu, choose All.
    6. Click OK.
  2. Save the exported file NewSupplierUserExportCSV.csv and don't change its name.
  3. Open the file and perform the following actions:
    1. Delete the first line, which specifies UTF-8 encoding.
    2. After the final TimeZone column, add the additional columns: firstName, middleName, lastName, countryCode, telephone, isPrimary
    3. These columns are mandatory for the import but can be empty.
    4. Keep only the supplier user that you need to sync with SM and delete the rest of the supplier users.
    5. Optional steps
      1. For any FullName values you don't want to break into first, last, and middle names using the automatic system, enter values in the firstName, middleName, and lastName columns and remove the name from the FullName column on that row.
      2. To designate a supplier contact as the primary contact for the supplier, enter TRUE in the isPrimary column.
  4. Import in Supplier Management:
    1. Click Manage > SM Administration > Click Data import and export.
    2. On the File type dropdown menu, choose Supplier contacts from sourcing.
    3. Click Choose File.
    4. Navigate to the supplier contact CSV file and select it.
    5. Click Import.
    6. The Import Summary tab shows the status of your import.
