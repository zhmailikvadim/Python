Symptom

When editing configuration parameters in the SAP HANA .ini files, the change does not take effect.

In the service trace files, the following error is observed

<parameter 1> and <parameters 2> can not be enabled in parallel, use one or the other;

For example

data_backup_max_chunk_size 204800 and parallel_data_backup_file_channels 4 can not be enabled in parallel, use one or the other; $condition$=m_DataBackupFileChannels == DataBackup::ParallelDataFileChannels::min || m_DataBackupMaxChunkSize == 0

Environment

SAP HANA Platform Edition

Reproducing the Issue

Try to edit configuration files with conflicting, mutual exclusive parameters set

Cause

The parameters in the error message are mutual exclusive and only one can be set.

Resolution

Consult the

SAP HANA Configuration Parameter Reference

for more information on the parameters involved and decide which to set.
