Symptom

When a candidate's application is disqualified it is possible for the candidate can see the following message on their application on the review step:

Thanks for applying! While we won’t be proceeding with your application, we invite you to check out other roles that might interest you.

Environment

SAP SuccessFactors Recruiting Management 

Reproducing the Issue

  1. Log in to the external career site.
  2. Navigate to My Applications and selected a job application which candidate has applied for but has been disqualified. 
  3. Navigate to end of the application and arrive on the Review step of the application
  4. See the message: Thanks for applying! While we won’t be proceeding with your application, we invite you to check out other roles that might interest you.

Cause

The message is appearing from the label key RECRUITING_APPLICATION_SUBMITTED_CANDIDATE_NOT_SELECTED_MESSAGESTRIP 

Resolution

The message is being removed in the 2026 release in April/May.

SAP SuccessFactors Product Release & Road Map

Release Dates:

1H 2026
Preview: April 13 
Production: May 15 - 16

If in the interim you require the message to be removed please use the Manage Languages tool to update the label.

2576546 - Using the Manage Language Feature to Change Labels in SuccessFactors

Label Key to update: RECRUITING_APPLICATION_SUBMITTED_CANDIDATE_NOT_SELECTED_MESSAGESTRIP
