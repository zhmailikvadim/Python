Symptom

The user finalizes a SIMA activity. The system takes a long time to finalize. The maximum Web Dynpro application duration defined in the system settings is exceeded.

Other Terms

/MSG/RABRIDXZUO

Reason and Prerequisites

Prerequisite: The SIMA activity is a loss document and has many entities.

Reason: the system performs data determinations record by record. This unnecessarily increases the time during finalization.

Solution

History:
March 17, 2026: New

Troubleshooting Description:
The system determines the data once. The user can finalize the SIMA activity.
