Symptom

When you create a shared absence in transaction PA30 for an employee with concurrent employment, the system displays an error message stating that no absence quota exists, although there is enough quota available.

This issue occurs when:
- You have concurrent employment active in the system
- You want to create a shared absence
- You have shared quotas in the system
- Several quotas exist from previous years that can no longer be deducted
- An organizational change occurred between different quota periods, causing the quota grouping to change

Other Terms

PA30, CE, concurrent employment, shared absences, shared quotas, T556A

Reason and Prerequisites

This is a program error.

Solution

Please implement the correction instruction or install the related support package.
