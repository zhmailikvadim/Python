Symptom

When deleting an Outbound Delivery Item linked to a hierarchical In-House Service Object (main item with sub items), the system must update both the sub item attributes and the main item attributes (for example, availability status and logistics attributes). Previously, only the sub-item attributes were updated, which could lead to inconsistencies in the main item.

Reason and Prerequisites

Programming error: hierarchical items were not considered.

Solution

This note enables correct deletion handling for Outbound Delivery Items related to hierarchical In-House Service Objects. When a sub-service object is updated, the corresponding main service object is also updated as required.

Import the Support Package specified for your release or implement the attached correction instructions. As a general rule, SAP recommends implementing the Support Package specified for your release.
