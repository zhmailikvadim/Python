Symptom

You use SAP ACM and PPA samples 6 and SPS 11 (document categories N3KVMST06 and N3KVPTV11 in the current version) to request appointment arrangement codes.

If the patient data is transferred without information about the patient's sex when requesting a placement code, this triggers an error message.

Other Terms

eTS, eTSS, appointment service point, eAppointment service

Reason and Prerequisites

Program error

Solution

If no information on the patient's sex has been made and consent to the transmission of patient data has been granted, no placement code can be requested. The system issues an error message asking you to enter the gender data in the admission.

Import the specified Support Packages to solve the problem. You can implement collective correction 3716853 in advance.
