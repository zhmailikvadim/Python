Symptom

  • The Outbound/Request_IN payload is experiencing a delay in triggering after the vendor map sync is completed.
  • Delay in payload generation when the External Approval task is configured within Supplier Request.

Resolution

This is expected behavior when the external approval task is enabled. The delay occurs because a scheduled job runs at regular intervals to handle external approval processes.

Keywords

Payload, Payload generation delay, Request_In, External approval, Supplier Request
