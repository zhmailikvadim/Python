Symptom

  • After implementing the SAP Note 3644473 you receive an error:
    Error message: "1 errors found. Activate anyway? Method GET_DQUERY_VALUEHELPS Line 194: The formal parameter 'IV_DO_NOT_DELETE' does not exist."

Environment

SAP S/4HANA 

Reproducing the Issue

  1. Attempt to implement SAP Note 3644473.
  2. Encounter the error message: "1 errors found. Activate anyway? Method GET_DQUERY_VALUEHELPS Line 194: The formal parameter 'IV_DO_NOT_DELETE' does not exist."

Cause

The SAP Note 3644473 requires the implementation of SAP Note 3551430, which introduces the formal parameter "IV_DO_NOT_DELETE". This note was added as a prerequisite in version 11 of SAP Note 3644473.

Resolution

  1. Implement SAP Note 3551430 - "Added functionality to reference storing utility class".
  2. After successful implementation of SAP Note 3551430, proceed with the implementation of SAP Note 3644473.
  3. Ensure that all prerequisite notes for SAP Note 3644473 are successfully implemented before attempting to re-implement the note.
  4. Test the implementation to confirm resolution of the issue.
