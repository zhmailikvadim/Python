Symptom

The service node approvetimesheets1 is not available in the system when accessing the service path /sap/bc/ui5_ui5/sap/approvetimesheets1 in transaction SICF

Environment

  • SAP S/4HANA 
  • Approve Timesheet (S/4HANA / Version 4) 

Reproducing the Issue

  1. Attempt to access the service path /sap/bc/ui5_ui5/sap/approvetimesheets1 in transaction SICF
  2. Observe that the service node approvetimesheets1 is not available in the system

Resolution

When using SICF, the technical name must be used when searching for services. For Approve Timesheets (S/4HANA / Version 4) please use the technical name TSAPPROVETIMES1 in the 'Service Name' parameter
