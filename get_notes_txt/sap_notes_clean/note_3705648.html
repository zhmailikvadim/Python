Symptom

  • Create crystal report using HTML text interpretation
  • Reports exported as PDF from the Central Management Console (CMC)/BI launchpad(BI) are cut off/are not aligned properly.
  • The same reports exported from Crystal Reports designer display all text correctly.

Other Terms

crystal, text allignement, pdf

Reason and Prerequisites

Environment:

  • SAP BusinessObjects Business Intelligence Platform 4.3 Support Package 05
  • Crystal Reports Designer

Steps to reproduce:

  1. Login to CMC/BI.
  2. Navigate to the report and export the report in PDF format.
  3. Observe that the generated PDF has missing text/misaligned text.
  4. Compare by executing the same report using Crystal Reports designer to generate a PDF.
  5. Observe that the generated PDF contains all the text aligned correctly.

Cause:

Solution
