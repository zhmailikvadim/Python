Symptom

You're running report /IBS/MCY_CORR_ACCNT_GUID to fix CYT table entries with wrong ACCNT_GUID. But the run is terminated by the error E105(/IBS/CY_GEN) 'Error when writing in table /IBS/TCY_ITD_l'. No data is fixed. That happens if the following conditions are fulfilled:

  • The package size is smaller than the number of accounts selected so that multiple packages are processed.
  • The report finds account to fix in at least two packages.

Other Terms

E/IBS/CY_GEN105, /IBS/CY_GEN105, /IBS/CY_GEN 105, /IBS/CY_GEN, 105

Reason and Prerequisites

 Program error.
