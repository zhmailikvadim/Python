Symptom

You run transaction code PUIT_UI to test the decoupling class of infotype Receipts & Misc. Information  HK (0440). The field Receipt Type (RETYP) is filled by inputting text and does not have a dropdown menu to select the subtype, as it does in transaction PA30.

The SAP Note enhances the field RETYP of screen structure HCMT_BSP_PA_HK_R0440 and provides a dropdown list for selection.

Other Terms

IT0440, RETYP, HCMT_BSP_PA_HK_R0440, PUIT_UI

Reason and Prerequisites

This is a correction of legal function.

Solution

Do either of the following:

  • Apply the Support Package;
  • Carry out the manual activities in ManualSteps3728067.pdf and implement the corresponding correction instructions.
