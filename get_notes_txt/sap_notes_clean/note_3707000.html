Symptom

This document explains how to download SAP HANA Data Lake Client.

Environment

SAP HANA Data Lake Client 1.0

Resolution

Please follow the below steps:

  1. Go to Software Center - SAP for Me
  2. Select "Support Packages and Patches"
  3. Enter "DATA LAKE CLIENT" in "Search in Software Downloads" field
  4. Download client package to fit your OS (For Windows x64 and For Linux x86_64 are available)
