Symptom

You use the HR Renewal interfaces and want to create or change a record of the infotype Challenge (0004). The system issues the error message "Fill in all required entry fields".

Other Terms

DSITZ, place of work, required entry field, severe challenge

Reason and Prerequisites

In the Web Dynpro interface (HR Renewal) for the infotype Severe Challenge (0004), the field City of  Administrative Office is a required entry. This is incorrect.

The problem does not exist in the SAP user interface (SAP GUI).

There is an error in SAP standard Customizing.

Solution

Import the relevant Support Package or implement the manual correction instructions.

Information about the manual activity: Check whether you have made customer-specific settings for the field DSITZ in the table T588MFPROPC and correct them if necessary. The customer-specific settings in the table T588MFPROPC override the settings delivered by SAP in the table T588MFPROPS. To maintain the two tables, there is a maintenance view (V_T588MFPROPS or V_T588MFPROPC). In particular, if you have not made any customer-specific settings, delete the entry for the field DSITZ from the table T588MFPROPS. For this, refer to the manual correction instructions.
