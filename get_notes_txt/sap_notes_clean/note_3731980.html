Symptom

This SAP Note provides you with corrections to the Colombia DIAN - Format 1001 (CO_DIAN_1001) report in the Run Statutory Reports app for SAP Document and Reporting Compliace. It addresses the issue of duplicate withholding tax amounts in the report output.

Currently, when you run the CO_DIAN_1001 report, the system duplicates the withholding tax amounts if the invoice receipt contains multiple expense line items. The system duplicates the withholding tax amount based on the number of expense lines in the invoice receipt.

Other Terms

DRC, Colombia, DIAN, Magnetic Media, 1001 Report Format, RETA, RETP, COMUN

Reason and Prerequisites

To implement the Magnetic Media compliance reports solution, you need a separate license for Document and Reporting Compliance (DRC). For more information, contact your SAP key account manager.

You have installed the Magnetic Media compliance reports for Colombia. For more information, check SAP Note 2717885.

This SAP Note is relevant only for Colombia.

You have applied SAP Note 3704612.

Solution

After implementing this SAP Note, the system correctly aggregates the withholding taxes and their respective values as posted in the invoice receipt.

As a result, when you run the CO_DIAN_1001 report, the system accurately reports the withholding tax amounts accurately, regardless of the number of line items in the invoice receipt.

As a general rule, SAP recommends that you install a solution by applying a Support Package. However, if you need to install a solution earlier, use the Note Assistant to implement the correction instruction. You can find more information about the SAP Note Assistant in the SAP Support Portal, under SAP Note Assistant.

To implement the solution, you must carry out manual activities and implement the correction instructions that are valid for your release.

You can activate the objects contained in this SAP Note by executing these reports:

  • For DDLS objects, execute the RUTDDLSACTSNOTE report according to the steps available in SAP Note 2607137.
  • For DCLS objects, execute the R_ACMTOOL_DELETE_INACTIVE_V report according to the steps available in SAP Note 2691812.
