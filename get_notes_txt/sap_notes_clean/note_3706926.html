Symptom

  • The system displays the line "Layout LTR < - > RTL" when pressing the SAP Logon icon button near the menu at the top of the screen.

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Environment

  • SAP GUI for Windows
  • ABAP Platform

Reproducing the Issue

  1. Open the SAP GUI for Windows.
  2. Press the button near the menu at the top of the screen.
  3. Observe the line "Layout LTR < - > RTL" displayed.

Cause

The options appear for users who have selected left-justified or right-justified character sets. This configuration causes the "Layout LTR < - > RTL" option to appear in the SAP GUI menu options.

Resolution

1. Select a character set (as highlighted in the blue square below) that is not left-justified or right-justified.

2. Restart the SAP GUI.

3. Verify that the "Layout LTR < - > RTL" option no longer appears.

An administrator can also select the character set for end users by defining it via Windows Registry Editor. The Character set-specific registry values can be found in the SAP GUI Administration Guide.

***WARNING***: This resolution involves editing the registry. Using the Registry Editor incorrectly can cause serious problems. Use the Registry Editor at your own risk. Refer to 1323322 for more information.
