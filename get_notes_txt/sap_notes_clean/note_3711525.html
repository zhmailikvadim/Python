Symptom

  • Web Intelligence report hangs when refreshing specific SAP BW Query
  • Issue is observed with one specific technical query

Other Terms

webi, webintelligence, BEx, BICS, refresh

Reason and Prerequisites

Environment:

  • SAP BusinessObjects Business Intelligence platform 4.3 Support Package 04 Patch 16
  • Web Intelligence

Steps To Reproduce:

  1. Login to BI Launchpad
  2. Navigate to Folders
  3. Open the problematic Web Intelligence report based on a specific SAP BW query
  4. Refresh the report
    1. >> Notice that it is stuck on refresh state

            Cause:

            • This issue is caused by a defect in the product.

            Solution

            This issue has been fixed by SAP Note 3705898
