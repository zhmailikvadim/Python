Symptom

'Services for object' button is not available in the Fiori app "Manage Credit Memo Requests".

Environment

  • Sales And Distribution (SD)
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Access the Fiori app "Manage credit memo requests".
  2. Attempt to locate the 'Services for object' button to add attachments.
  3. Observe that the option to add attachments is not available in the Fiori app.

Cause

The parameter 'SD_SWU_ACTIVE' has not been assigned to the user, which is a prerequisite for or using the GOS (Generic Object Services) functionality. 

Resolution

  1. Assign the parameter 'SD_SWU_ACTIVE' to the logon user using Tcode SU3.
  2. Make sure that the logon user is a dialog user who can view the 'Services for Object' button. 
  3. After assigning the required parameter, the 'Services for object' button should also become available in the Web GUI for the VA03 app.

Keywords
