Symptom

A hybrid Kapsel application's barcode scanner does not work on Windows 10 version 1703. It does not show the camera preview.

Environment

SAP Mobile Platform (SMP) SDK 3.0

Cause

This is because of a Windows 10 issue with the 15063 build.

Resolution

  • Microsoft fixed this issue in build 16281 that is in RS3.
  • Upgrade to SMP SDK 3.0 SP15 PL11 that contains the fix.
