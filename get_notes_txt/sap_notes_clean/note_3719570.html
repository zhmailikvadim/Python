Symptom

With the new 2602CE release, the Advanced Indicator column includes a specific icon that, when clicked, provides a list of unsupported features. Selecting this icon navigates you to the SAP GUI for HTML app, where these features can be supported. This icon is displayed only for info records that are best managed within the SAP GUI for HTML app, specifically in the Display Purchasing Info Record section.

The removal of the "Advanced Indicator" column from the Manage Purchasing Info Record list page impacts the visibility of validity dates and prices, causing these fields to appear empty.

Environment

SAP S/4HANA 2602 Public Edition 

Resolution

To ensure accurate data visibility, keep the Advanced Indicator column visible in the list view while working with the application. This column is essential for determining values in other fields.

With the Hotfix Collection 06, a correction is planned to eliminate the mandatory selection of the Advanced Indicator. In the meantime, please keep the icon indicator selected to retrieve the content in the other columns.

Please note that the Hotfix Collection date is a planned date and may be subject to changes.
