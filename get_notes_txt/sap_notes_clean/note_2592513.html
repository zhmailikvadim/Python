Symptom

In transaction PPOME, on the planned working time tab page, if you enter an approval percentage value that is too high, the system may generate a dump since the number of characters is not sufficient for a field. In this case, the system will now issue an error message.

Other Terms

Upd_p1011-dyavg, new_worktime, IT1011, planned working time

Reason and Prerequisites

Program error

Solution

Import the relevant HR Support Package, or implement an advance correction in accordance with the attached correction instructions.
