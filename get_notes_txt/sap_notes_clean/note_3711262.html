Symptom

The Dutch tax office has issued a revised specification for the Netherlands Taxonomy 20 (NT20), effective from January 1, 2026. This update impacts how call-off stock operations are reported in the XBRL file.

Other Terms

NL, ECSL, ICP aangifte, NL_ECSL_DCL, NL_ECSL_DCL_v20.0, NT20, Netherlande Taxonomie, SRF, ACR, DRC, Call-off stock

Solution

Report definition NL_ECSL_DCL_v20.0 has been adjusted to reflect the latest legal changes.

As a general rule, SAP recommends that you install this solution by applying a Support Package. However, if you need to install it earlier, it is recommended to utilize the most current version of Note Assistant to apply the corrective instructions. To ensure that Note Assistant is updated, please refer to SAP Note 875986 for NetWeaver version 7.0x or SAP Note 1668882 for NetWeaver versions 7.3x and higher.
