Symptom

In Fiori launchpad, the tile Perform Maintenance Jobs displays number "1" instead of reflecting the actual number of jobs.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Open launchpad space Maintenance Execution in Fiori launchpad.
  2. In the Confirmation section, find the "Perform Maintenance Jobs" tile (F5104A).
  3. Observe that the tile counter shows "1" instead of the expected number of jobs.

Cause

The tile counter reflects the number of records for the 'mine and my teams' filter. If the logged-in user is not assigned to a team or relevant work center, the counter will not reflect overall records and can show an unexpected value such as "1".

Resolution

Check the logged-in user's assignments and ensure the user is assigned to the appropriate team and relevant work center(s). After updating assignments, refresh the Fiori launchpad or the tile and recheck the counter.
