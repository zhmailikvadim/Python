Symptom

 How to find out which role contains a specific Fiori catalog

Environment

SAP_Netweaver 7.4 or higher

Resolution

  1. Display the table AGR_BUFFI in transaction SE16 
  2. Input the catalog ID in the field "internet link". For example, if the catalog is: SAP_BASIS_TCR_T, input *SAP_BASIS_TCR_T*
  3. Press "Execute" button, the roles will dispaly

Keywords

Fiori, catalog, dispaly, role, remote catalog, PFCG
