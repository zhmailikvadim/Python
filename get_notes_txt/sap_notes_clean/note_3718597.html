Symptom

When you implement SAP Note 3591743 "Fiori Intercompany Balance Reconciliation: Email Template for Ukraine, PDF and Email Language Selection", you face the following issues during the implementation:

  • Object DOCU HYSIMGFIRUV_RECON_BP_E does not exist
  • Object DOCU HYSIMGFIUAV_RECON_KIND does not exist

Other Terms

Reconciliation report, RECONCILIATION_REPORT, Russia, Ukraine, F1819, Documentation Hypertext module

Reason and Prerequisites

Missing objects

Solution

Implement automatic correction instructions of this SAP Note and perform post-implementtaion steps, or implement the mentioned Support Package.
