Symptom

In DK pain.001, service level code is not mapped correctly which leads to incorrect determination of transaction type based on service level code.

Other Terms

/PE6/CL_FH_XML_INPUT_RTNS, /pe6/t_xml_trtyp.

Reason and Prerequisites

Program error. When service level code is present at payment info level or transaction level, in both cases it is not mapped correctly.

Solution

Implement the correction instruction attached to this SAP Note. Alternatively, you can import the referenced Support Package.
