Symptom

The 'Time-dependent condition' checkbox is not displayed in the contract document type configuration node.

Environment

  • Materials Management (MM)
  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA

Reproducing the Issue

  1. Navigate to OLME -> Contract-> Define document type.
  2. The 'Time-dependent condition' checkbox is not displayed.

Cause

Missing configuration.

Resolution

  1. Navigate to OLME -> Contract-> Define document type.
  2. Select the relevant document type.
  3. Click on 'Edit' at the top, then choose 'Change field contents'.
  4. Locate the time-dependent field, replace its value with 'X', and save the entry.
  5. Verify the changes in the T161 table to ensure the updated value is reflected.

Keywords

Change field contents, T161 table, OLME configuration, define document type, MK, VK
