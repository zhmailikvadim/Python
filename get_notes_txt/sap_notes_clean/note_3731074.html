Symptom

Clicking edit in a custom page in  Manage Launchpad Pages does not change the screen or do anything. The browser console can show "Uncaught (in promise) TypeError: Cannot read properties of undefined (reading 'attachPatternMatched')" error.

Environment

  • SAP Fiori Launchpad ABAP Services
  • SAP S/4HANA, private edition

Reproducing the Issue

  1. Open the SAP Fiori launchpad.
  2. Launch the app "Manage Launchpad Pages".
  3. Select any custom page.
  4. Choose Edit. The screen does not change and the editor does not open.

Cause

After a SAPUI5 patch update, page caches and the UI5 app index were not refreshed, causing the app to fail when navigating to the edit view.

Resolution

  1. Ensure the system is on the latest applicable SAPUI5 patch for release in the system. If needed, apply the relevant SAP Note via SNOTE.
  2. Run the report /UI2/PAGE_CACHE_SYNCHRONIZE to clean and synchronize the Fiori Launchpad page caches.
  3. Recalculate the UI5 app index by running /UI5/APP_INDEX_CALCULATE and restrict it to the SAPUI5 repository SUI_PAGE_MAN.
  4. Log off and log on to the launchpad, then open "Manage Launchpad Pages" and try Edit again to confirm the fix.
