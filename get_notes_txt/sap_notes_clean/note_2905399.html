Symptom

Some exceptions are not handled, it will lead to dumps in system whenever there is a config issue.

Other Terms

 /PF1/CL_BC_ACCTPRX_IHB, /PF1/CL_IHB_CLASSIFICATION

Reason and Prerequisites

After implementation of SAP Note 3406156, In-house Bank End of Day objects cannot work properly because the exceptions are not included in the method signature.

Solution

Code corrections.

Please apply SAP Note or assigned Support Package.
