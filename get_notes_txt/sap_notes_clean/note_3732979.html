  • Version management of APS is running with errors after upgrading from 4.3.3 to 2025.
  • The IFRS and OFRS servers are in disabled mode.
  • APS Log Entries show below errors:

Environment

  • SAP BusinessObjects Business Intelligence Platform 4.x
  • All Supported Operating System

Reproducing the Issue

  • Upgrade from BI 4.3 to 2025
  • The APS is running with errors.
  • In APS, the version management APS is running with errors.
  • Additionally, the IFRS and OFRS servers are in a disabled state.

Cause

  • The APS for version management service requires the File Repository Servers (IFRS and OFRS) to be in an enabled state.

Resolution

  • Ensure that after migrating from SAP BusinessObjects version 4.3 to 2025, it is crucial to use the correct license key for the 2025 environment
