Symptom

If multiple company cars are registered for an employee in the same period, only one of them is correctly processed in Payroll function for company car (P0442) during payroll run (RPCALCV0). Company cars are registered in Company car infotype (IT 0442).

Version

Date

Reason

1

2017-04-26

Pilot release.

2

2017-06-21

Released to customers

Solution

Payroll Function for Company Car has been amended. Payroll will process all relevant cars in payroll period.

Installation

However, if you need to install a solution earlier, use the Note Assistant to implement the correction instruction.
