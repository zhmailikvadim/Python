Symptom

The note is only a requirement for other notes.

Solution

  • SAP BW 7.50
    Implement Support Package 7 for SAP BW 7.50 (SAPK-75007INSAPBW) into your BW system.
  • SAP BW 7.51
    Implement Support Package 2 for SAP BW 7.51 (SAPK-75102INSAPBW) into your BW system. 
  • SAP BW/4HANA 1.0
    Implement Support Package 2 for SAP BW/4HANA 1.0 (SAPK-10002INDW4CORE) into your SAP BW/4HANA system. 

In urgent cases you can use the correction instructions.

Before you use the correction instructions, make sure that you check SAP Note 1668882 and SAP Note 2248091 for transaction SNOTE.

This SAP Note might already be available before the Support Package is released. In this case, however, the short text still contains the term "preliminary version".
