Resolution

The card provides an at-a-glance summary of the number of services that have been requested for the user's cloud solutions, grouped by their status.

In order to access this card, an S-user must have the below authorizations assigned:  

  • Create Billable Service Requests (SRV_BIL_AD)
  • Create Service Requests (SRV_CREA)
  • Display Service Requests (SRV_REQ)

To request the necessary authorization, contact the Super Administrators (listed here) or use the self-service authorization request according to KBA 2891528.

For detailed information about this authorization, refer to About the User, Authorization and Administrator Concept.
