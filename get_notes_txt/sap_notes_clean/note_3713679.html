Symptom

Vendor is fully onboarded but is not visible when creating a Purchase Requisition (PR). 

Environment

SAP Ariba Buying & Invoicing

Reproducing the Issue

  1. Attempt to create a Purchase Requisition (PR) for Vendor.
  2. Observe that the vendor profile is not visible.

Cause

The Supplier is only extended to "X" Purchasing Unit, while the User belongs to Purchasing Unit "Y".

Resolution

Two approaches are possible to resolve this behavior:

  1. Extend Supplier to Purchasing Unit "Y" using the Supplier Data (Consolidated File) task.
  2. Assign the User to Purchasing Unit "X" using the Responsible User Assignments task.

To use either task, please go to Core Administration > Data Import/Export.

It is important to note that the Responsible User Assignments task should be first tested in the TEST environment since it is a highly sensitive file.

See Also
