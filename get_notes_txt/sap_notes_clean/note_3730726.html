Symptom

Follow the guide to set the parameters in the system. There is no guided tour from new hire's login page.

  • Plugin Server: https://webassistant.enable-now.cloud.sap/
  • Parameters: autoStartTour=ONB2_JOURNEY

Environment

SAP SuccessFactors Onboarding 2505

Cause

The Onboarding Guided Tour is open on the Onboarding Tab.

Resolution

You need switch from the HOME page to the Onboarding Tab. Then the Onboarding Guided Tour starts automatically.
