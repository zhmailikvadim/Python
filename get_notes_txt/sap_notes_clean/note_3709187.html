Symptom

For BPC2021, user opened work status report page, and may find the time criteria is set with expected start date. Normally the start date is set to the first day of current month. This issue happened under some specific date-time format. It also occurred to other system report types with date time criteria, like BPF operation report, comment report, audit data change report and administration reports.

Other Terms

user preference, date time format, system reports

Reason and Prerequisites

Program error.

Solution

Upgrade BPC web client to equivalent SP level(BPC2021 SP14).
