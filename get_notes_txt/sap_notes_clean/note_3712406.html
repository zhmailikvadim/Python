Symptom

  • The Flexible Upload fails when attempting to include "AdHoc Item" in the upload file.
  • The Error message G00743 'Technical name for field AdHocItem in upload file is invalid' is raised when the fieldname AdHocItem is used in the upload file in the data definition row (row starting with D)
  • It is unclear which fieldname to be used in the flexible upload file

Environment

SAP S/4HANA Cloud Private Edition, SAP S/4HANA Finance for Group Reporting

Reproducing the Issue

  1. Attempt to add the AdHoc Item field in the flexible upload file
  2. Enter the fieldname 'AdHoc Item' in the upload file
  3. Observe the error message G00743 'Technical name for field AdHocItem in upload file is invalid' during the upload

Cause

Wrong fieldname used

Resolution

    For Flexible Upload: Ensure that in the data definition row starting with 'D' in the upload file, the field name 'ConsolidationAdhocItem' is specified.
