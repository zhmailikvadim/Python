Symptom

You use the User Information System (SUIM) to analyze change documents for users. In the result list, you see values such as E or J in the column for the type of change document.

Other Terms

S_BCE_68002311
RSUSR100N

Reason and Prerequisites

No changes were made to the change document object IDENTITY (transaction SCDO). The single field logging of the change documents has been changed. Since then, new entries with the indicators E and J have been created.

Solution

Use transaction SNOTE to implement the attached correction instructions or import the relevant Support Package.
