Symptom

  • The system does not allow the selection of new accounting principles.
  • Only Local accounting principle appears as an option in configuration "Assign Accounting Principles to Ledgers and Company Codes" - SSCUI 103556.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

Cause

The available accounting principles are determined by the countries/regions activated in the Central Business Configuration (CBC) project and the finance localization scope.

Resolution

  1. Do not attempt to create new accounting principles; only SAP-delivered principles are available based on country/region scope.
  2. In Central Business Configuration, review the scoped countries/regions for Finance.
  3. Add and activate the required countries/regions during initial scoping to import their local accounting principles.
  4. Ensure the chosen parallel accounting scenario supports any required global principles (for example, IFRS or US-GAAP), if applicable.
  5. Before changing scope, confirm no postings exist; if postings are present, reset transaction data using the “Reset Transaction Data” job according to standard procedures.
