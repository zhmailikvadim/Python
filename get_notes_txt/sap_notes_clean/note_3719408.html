Symptom

  • Remove or Modify Time Periods in the Time Profile
  • Customer would like to delete Unintended time periods from the Time Profile
  • The customer reported a massive, unexpected increase in data records (data volume inconsistency) extracted by a CIDS task
  • Customer identified that they had created extra time periods by running a copy operator

Environment

  • SAP Integrated Business Planning (SCM-IBP)
  • SAP Cloud Integration for data services  (CI-DS)

Resolution

If your goal is to change the structure of the time profile itself (e.g., remove weeks, move the horizon):

You can adjust time profile entries by:

  1. Exporting the time profile CSV
  2. Removing or editing rows for unwanted periods
  3. Re‑uploading the time profile using Data Integration Jobs → Replace
  4. When doing this, SAP advises enabling Allow Extension Only to prevent unintended data loss
