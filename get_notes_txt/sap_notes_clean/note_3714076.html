Symptom

  • Aggregation settings “Aggregation per Calendar Week” and “Aggregation for Time Statement Period” are applied across all Key Facts and Allowance Hours even when set to "Yes" for only one Placeholder.
  • This behavior is inconsistent with the expected functionality, where settings should apply individually to each Placeholder.

Environment

  • SAP SuccessFactors Time Tracking

Cause

This is a Known Error.

Resolution

The fix for this issue is planned to be deployed with the release 1H2026.
For release timelines, please review SAP SuccessFactors Product Release & Road Map Information.

Workaround
No workaround is available.
