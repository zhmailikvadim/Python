Symptom

Under certain conditions, fiscal year derivation may result in inconsistencies that affect number range assignments, leading to incorrect document numbering sequences.

Other Terms

Angola, RE-FX, SAF-T, SAFTN, RSDV, Digital Signature, Flexible Real Estate Management, Real Estate Invoice

Reason and Prerequisites

You have installed the solution for SAF-T & DS Angola RE-FX according to SAP Note 2946198.

Solution

The fiscal year derivation logic has been corrected to ensure consistent and accurate calculation of fiscal periods. This fix resolves the number range assignment issues and prevents future inconsistencies in document numbering based on fiscal year determination.

As a general rule, SAP recommends that you implement the Support Package specified for your release. However, if you need to install a solution earlier, use the SAP Note Assistant to implement the correction instruction. You can find more information about the SAP Note Assistant in SAP Support Portal, under SAP Note Assistant. Note that documentation, transactions, and customizing activities are only delivered via Support Package.
