Symptom

The transaction type (BSEG-BEWAR) is not populated from open items into clearing items.

Environment

    • SAP R/3
    • SAP ERP Central Component
    • SAP ERP
    • SAP enhancement package for SAP ERP
    • SAP enhancement package for SAP ERP, version for SAP HANA
    • SAP S/4HANA Finance
    • SAP S/4HANA

Reproducing the Issue

  1. Execute transaction like F-04 for clearing open items.
  2. Observe that the transaction type (BSEG-BEWAR) in the clearing document is blank even has value in cleared document

Cause

The transaction type (BSEG-BEWAR) is not a field that will be filled in the clearing document from the cleared items.

Resolution

  1. Confirm that this is standard system behavior.
  2. As an alternative solution, consider implementing a customer enhancement to substitute the transaction type from BSEG to the clearing item but this will not be supported by SAP standard.

Keywords

bseg-bewar, bewar, transaction type, clearing items, open items, posting, clearing.
