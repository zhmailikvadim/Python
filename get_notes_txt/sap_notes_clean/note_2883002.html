Symptom

  1. Report - Canvas option is missing for creating an Advance Report
  2. Configuration related to Advance Reporting is set correctly in Provisioning and Analytics.
  3. Canvas Reports permissions are controlled from RBP and user has permission to create a canvas report (or)
  4. Canvas Reports permissions are controlled from user maintenance and user has Report Admin / Implementation partner role to create a canvas report

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

 

Environment

SAP SuccessFactors HCM Suite

  • Analytics & Reporting(Ad Hoc, YouCalc, ORD)

                      Advance Reporting (ODS)

Reproducing the Issue

  1. Log in into instance from any of the Browsers
  2. Click Reporting
  3. Click New
  4. Report Canvas option will be missing

Cause

Default role is missing in Analytics.

Resolution

Raise a support ticket to check if the Default role is set for the instance in Analytics backend.
