Symptom

You are using SAP Solution Manager 7.2 with Focused Build ST-OST.
After upgrade to ST-OST SP 13 and higher, you have a syntax error in program /SALM/CL_CUSTOMER_EXT_UTIL
Include /SALM/CL_CUSTOMER_EXT_UTIL====CM01U: Syntax error in line 000001
Method 'DEFINE_VALUE_HELP_ENTITYSET' does not exist. The following similarly-named methods do exist: 

Environment

  • SAP Solution Manager 7.2
  • Focused Build ST-OST

Reproducing the Issue

  • Syntax error in program /SALM/CL_CUSTOMER_EXT_UTIL
  • Focused Build Requirements Management

Cause

The method DEFINE_VALUE_HELP_ENTITYSET was removed and no longer needed from ST-OST SP >=13

Resolution

The Method is obsolete in higher SP's and clean-up is required.

  1. Execute SE24
  2. Select Class /SALM/CL_CUSTOMER_EXT_UTIL
  3. Click on Change
  4. From the menu, select Utilities > Clean Up > Method includes
  5. confirm the pop-up that you want to delete - click yes
  6. Activate
