Symptom

This SAP Note addresses the following functionalaties for the Electronic Customer Invoices B2B process.

  • e-Reporting an unsent message.
  • Fiscalization update for outgoing e-invoice.

Other Terms

Reason and Prerequisites

Reason

According to legal requirements in Croatia, companies are generally obliged to transmit data via electronic transmission.

Prerequisites:

  • First implement all the prerequisite requirements listed in the SAP Note 3499841 - Electronic Document Processing for Croatia: Implementation Overview.
  • Follow the instructions as per the attched file "3712966_Manual_Pre-Implementation_Steps.pdf"
