Symptom

This SAP Note delivers objects that are used in the solution delivered via SAP Note 3725117.

Other Terms

DDIC; OBRA_CNPJ; WRKPLC_INSC_N; COMPANY_ID; SAP Note 3725117.

Reason and Prerequisites

Reason

This SAP Note delivers objects that are prerequisite for SAP Note 3725117.

Prerequisites

The following Support Package level is the minimum prerequisite:

Release Support Package
SAP_HRCBR 6.04 (EhP6 and EhP7) HR SP J1
SAP_HRCBR 6.08 HR SP B9
S4HCMCBR 1.00 HR SP 08
S4HCMCBR 1.01 HR SP 00
S4HCMCBR 1.02 HR SP 00

For more information, access the SAP ERP Human Capital Management - Support Package Schedules.

When you install an SAP Note, some prerequisites might be required. The prerequisites are SAP Notes that have already been released and that change the same context of the current one. Then, even if these prerequisites are not necessarily related to the symptom of this SAP Note, you must implement them due to technical reasons.

Solution

This SAP Note delivers to you the following dictionary objects that are used in the SAP Note 3725117.

  • New HRPADBR_CNPJ_NO_FORMAT data element

  • Changes to T7BRC0 table

  • Changes to T7BRTAXWRKPLC table

  • Changes to T7BR_UNINHE table

Log of changes

Version Date Description of the change
2 March 11, 2026  Released for Customer

Delivery

  • For releases S4HCMCBR 100, 101, 102 and SAP_HRCBR 608: the previously described objects are created or modified by SNOTE.
  • For release 604: the previously described objects are created or modified by execution of the NOTE_3709740 report.
