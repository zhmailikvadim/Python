Symptom

If you select a Test Configuration or you start the Solution Documentation (SOLDOC) with default selection a Test Configuration, a performance issue can happen.

Other Terms

Test Configuration, TESTCONFIGVARLOGCOMP, SOLDOC, Performance

Reason and Prerequisites

Performance issue with method CL_AGSTCE_ECATT_ABSTRACT->GET_TEST_CONF_VARIANTS

Solution

Implement the current correction instructions of the SAP Note with transaction SNOTE and the issue should be solved.
