Symptom

This is a side effect note for the SAP Note 2114162. The settlement strategy is switching back to Copy fixed and redetermine for settlement creation even if there are charges in the order. Hence, there are difference in charges between order and settlement document.

Other Terms

SAP TM, CC, Charge Calculation, Settlement Dump, /SCMTMS/CL_TRNSPCHRG_A_CALCHRG

Reason and Prerequisites

Program Error

Solution

Implement the SAP Note or upgrade to latest SP release.
