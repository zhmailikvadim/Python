Symptom

Token label displays Unicode characters instead of correct Chinese, Japanese, or Korean text.

***Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.***

Environment

SAP SuccessFactors Recruiting Management

Reproducing the Issue

  1. Change the system locale to Japanese, or Korean, or Simplified/Traditional Chinese
  2. Navigate to the Recruiting Email Template page
  3. Select a template 
  4. Add tokens
  5. Observe that some labels display Unicode characters instead of the correct text  

Resolution

This issue will be resolved in the 1H 2026 version.

The new version will be released on the following dates:
-Apr.13 for Preview instance
-May 15 for Production instance
