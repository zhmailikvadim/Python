Symptom

It is being requested to change the IP address to a brand new IP Address for specific environment on SAP Incentive Management.

Environment

  • SAP Sales Performance Management
  • Incentive Management

Resolution

Requests to change or update the IP Address for a specific environment are not eligible. The reason for this request being not eligible is the following: Changes to this structure, create items that need to be uniquely tracked and managed, putting this configuration, and access into the system at risk due to non-standard setup. Additionally, branching changes into secondary systems, would have further impact, and also need to be uniquely tracked and managed

Keywords

ip address change, network security policy, ip, outbound ip
