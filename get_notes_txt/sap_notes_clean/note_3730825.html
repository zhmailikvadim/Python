Symptom

A user wants to do GR correction for a EWM inbound delivery by entering quantity differences for HUs with exception code DIFD in tcode /scwm/pack, however, the error "Config of delivery adjustment for item type &1 missing (except. handling)" happens, the message no. is /SCWM/EXCEPTION075.
"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental." 

Environment

SAP Extended Warehouse Management (EWM)

Reproducing the Issue

Enter quantity differences for a HU with exception code DIFD via tcode /scwm/pack tab "Differences".

Cause

missing customizing.

Resolution

Go the the customizing node "Define Exception Codes" under
-> SCM Extended Warehouse Management
-> Extended Warehouse Management
-> Cross-Process Settings
-> Exception Handling

Select the entry for the EWM warehouse and the exception code DIFD, double click on "Define Exception Code" on the left side

Select the entry for "PPT Packing/Unpacking Products (Putaway)" and step "16", double click "Maintain Process Parameters", next, double click "Delivery Adjustment in Case of Differences":

If there is no entry for Item type mentioned by the error message and Process Code "I001", create an entry for the item type and process code I001, save the change.
