Symptom

  • Attempting to launch the Object Wizard from the contextual menu of the Data Source Explorer
  • Using the Agentry Connector Studio Eclipse plug-in
  • The Object Wizard silently fails to launch
  • No error message appears, but the Object Wizard does not appear

Environment

  • SAP Mobile Platform SDK 2.3 (Agentry)
  • SAP Mobile Platform SDK 3.0 (Agentry)
  • Agentry 6.x.x.x (Editor Plugin)
  • Eclipse SDK IDE (Version 3.7.x - Indigo)

Reproducing the Issue

  1. Create a new system connection
    • From the Project Explorer, right-click "System Connections" and select Add System Connection > SQL Database
    • No specific values need to be entered
  2. Delete the newly created system connection
  3. Open the Data Source Explorer in Eclipse (Window > View > Data Source Explorer)
  4. Expand the Database Connections directory
  5. Connect to an existing data source (Right-click > Connect)
  6. Navigate to the "Tables" directory
  7. Right-click one of the database tables
  8. Select "Agentry: Connector Studio > Object Wizard"
  9. The contextual menu closes, and the Object Wizard does not appear

Cause

  • A defect in the editor prevents the wizard from launching under specific circumstances
  • If a System Connection is located in the Trash Bin, the wizard will fail to launch

Resolution

From the Agentry Editor Plugin in Eclipse:

  1. Under the "Window" drop-down menu, select "Show View" > "Trash Bin"
  2. Locate any object with the type "SQL System Connection"
  3. Right-click each object and select "Remove Selected Definition"

 

Keywords
