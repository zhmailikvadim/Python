Symptom

Error message: "Content data for object '***.xsl' not found". As an example, fdq_aa.xsl.

Reproducing the Issue

Implement SAP Note 2758146.

Resolution

  1. In transaction SE38 and execute report MR_CHECK_MIMES.
  2. Select the option “Display MIME Object Attributes” and enter the URL of the problematic file (e.g., /sap/public/bc/abap/fdq_aa.xsl). You can find the file name from SAP Note 2758146.
  3. Copy the technical GUID from the last Technical Information tab in the popup.
  4. Execute report MR_DELETE_MIME and paste the technical GUID. Press F8 to delete it.
  5. Repeat steps 2–4 for all MIME files associated with the note, including files in other paths and any “.BAK” files.
  6. Re-implement SAP Note 2758146.
