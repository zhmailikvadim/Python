Symptom

This article outlines the information on how to submit enhancement request via SAP Continuous Influence Program.

Environment

  • SAP Sales Performance Management
    • Advanced Workflow

Cause

Enhancement requests are no longer handled through traditional support cases but via the SAP Continuous Influence Program for Advanced Workflow.

Resolution

To submit enhancement requests or suggest new features for SAP Sales Cloud products, you should utilize the SAP Continuous Influence Program. This program allows you to directly interact with the SPM Product Team by submitting suggestions, voting on existing suggestions, and collaborating with other customers.

Steps to Access SAP Continuous Influence:

  1. Visit the SAP Continuous Influence portal; https://influence.sap.com/sap/ino/#campaign/2460 
  2. Log in using your community user ID.
  3. Enter your community password on the subsequent page.
  4. Complete the "Upgrade your user account for Customer Influence" page by filling in the mandatory fields and accepting the Terms and Conditions. This step is necessary to enable your community credentials for use with SAP Continuous Influence.
  5. Once completed, you will be redirected to the SAP Continuous Influence SAP homepage.
  6. Please proceed with creating the enhancement request in the influence portal.
