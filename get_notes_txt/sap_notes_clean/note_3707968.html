Symptom

The user is unable to select the FS hierarchy "X2- Cash Flow Statement" in the detailed value list/dropdown in the page filters of the Cash Flow tab of the GR booklet app.

The dropdown list appears empty for the user despite having the same security, roles, and catalogs as other users.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Open the GR booklet app.
  2. Navigate to the Cash Flow tab.
  3. Attempt to select the FS hierarchy "X2- Cash Flow Statement" in the page filter hierarchy dropdown.
  4. Observe that the dropdown list appears empty for the user.

Cause

For any end user, Set global parameters app is the first step. 

Resolution

  1. Set the global parameters in the "Set Global Parameters" app.
  2. Open the GR booklet app and navigate to the Cash Flow tab.
  3. Verify that the FS hierarchy "X2- Cash Flow Statement" is now selectable in the page filter hierarchy dropdown.
