Symptom

You run the Generación de notificaciones al IMSS (HMXCAFL0) report for an employee that has a Datos IMSS (IT0369) infotype record with a start date less than the first Asignación organizativa (IT0001) infotype record start date.

In the scenario described above, the system does not process the employee.

Other Terms

&SEGSOCIAL - For further information, see SAP Note 1179364 - CONSULT: List of tags for Mexico notes

PC00_M32_CAFL0; IT0369; IT0000; IT0001;

Reason and Prerequisites

Reason

This issue happens because the system wrongly skips the employee when trying to find a valid 0001 infotype record in the start date from the 0369 infotype.

When you install an SAP Note, some prerequisites might be required. The prerequisites are SAP Notes that have already been released and that change the same context of the current one. Solution

From now on, the system processes the employee if the start date of Infotype 0369 is earlier than the start date of the employee’s first Infotype 0001.
