Symptom

  • In transaction HU02, error HUSELECT011 appears.
  • Message: "Storage type xxx does not allow repacking - HUs cannot be changed".
  • Repacking or changing the handling unit in HU02 is blocked. 

Environment

  • Sales And Distribution (SD)
  • Logistics Execution (LE)
  • SAP R/3
  • SAP R/3 Enterprise 4.7
  • SAP ERP Central Component
  • SAP ERP
  • SAP Enhancement package for SAP ERP
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Start transaction HU02.
  2. Select a handling unit that is currently in the storage type xxx.
  3. Attempt to repack or change the handling unit; observe error HUSELECT011: "Storage type xxx does not allow repacking - HUs cannot be changed".

Cause

  • The handling unit is located in a storage type that does not allow repacking; handling units cannot be changed in that storage type. 

Resolution

  1. Use transaction LT09 to move the handling unit to the repacking area (for example, storage type 923).
  2. Retry the repacking or change in transaction HU02.
