Symptom

The HCZLECASB report does not account for the following two scenarios:

  • Employees whose primary PERNR in IT0153 changes within the processing period (the report assumes a single, static main PERNR throughout), and
  • Employees who, despite having terminated employment contracts, continue to receive income subject to insurance contribution deductions (the report disregards inactive employees in this context).

Other Terms

HCZLECASB, HRCZ

Reason and Prerequisites

This SAP Note is relevant for the Czech Republic only.

Solution

The HCZLECASB report has been updated to correctly handle the scenarios described in the Symptom section.

Important: Apply current version of SAP Note 875986 for NW 7.0x or 1668882 for NW 7.3x and higher. Otherwise an old version of SNOTE can lead to missing coding changes when applying SAP Notes.
