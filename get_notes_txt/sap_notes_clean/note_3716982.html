Symptom

In the Approve timesheet FIORI app. The manger Inbox has one workitem ID for more than one employee and many entries.

Environment

  • ERP2025
  • SAP S/4HANA
  • SAP S/4HANA 2023
  • SAP Fiori for SAP S/4HANA 2023

Reproducing the Issue

1.Login as Manager in the app Approve timesheet 
2. Click on My Inbox
3. Check the Time sheet notification entries for approval
4. Click on Open tasks
5. Check the entries for approval

Cause

In the Standard workflow TS31000007 that is released from ERP2005 version, It is current scenario.
Before this release, workitems were created for each employee. Approver had to click on each workitem in order to approve records , which was time consuming.
With ERP2005, the approver gets at the maximum only one workitem which contains CATS records for all the employees sorted according to personnel numbers. So the approver has only one workitem to work with which makes Approval more easier and saves him lot of time.

Resolution

This is the standard behavior. 
The customer can always create his own workflow, taking into account that this is a customer modification not supported by SAP. For help to create the workflow can contact with the support component BC-BMT-WFM

Keywords
