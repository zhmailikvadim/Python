Symptom

In transaction COOIS, when switching to a specific variant, system issue error message "A variant from another transaction (COHV) was selected." as a termination message. 

Environment

  • SAP ERP Production Planning (PP)
  • SAP R/3
  • SAP R/3 Enterprise
  • SAP ERP Central Component
  • SAP ERP
  • SAP Enhancement package for SAP ERP
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA, on-premise
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Access transaction code COOIS.
  2. Select a variant which was created from transaction code COHV. 

Cause

  • The issue occurs because the user does not have authorization to access transaction code COHV, which is required for the variant.
  • The system performs an authorization check for the transaction associated with the variant. If the user lacks the required authorization, the system issues an error message and terminates the transaction.

Resolution

Ensure that the user has the necessary authorization for transaction code COHV to use variants created from COHV.
