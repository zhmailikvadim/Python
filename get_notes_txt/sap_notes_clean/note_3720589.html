Symptom

  • The selection field "Plant" in transaction /SAPSLL/CUS_INV_R3 displays as "S_WERKS" and contains no description.

Environment

  • SAP Global Trade Services, edition for SAP HANA

Reproducing the Issue

  1. Execute transaction /SAPSLL/CUS_INV_R3.
  2. Observe that the selection field "Plant" displays as "S_WERKS" and contains no description.

Resolution

Call transaction code 'SE38' with program '/SAPSLL/CUS_INV_CREATE_R3'

  • Choose radio button 'Text elements' and click button 'Change'.
  • Click tab 'Selection texts' and choose checkbox 'Dictionary ref.' for the entry 'S_WERKS'.
  • Save and activate the changes.

See Also

Refer to: SAP Note 0003390912 - Plant is missing as selection criteria for intercompany proforma invoice types.
