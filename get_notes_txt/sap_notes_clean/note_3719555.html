Symptom

Requirement to compare supplier pricing between two different sourcing events within the same sourcing project (for example, an initial auction and a subsequent RFP), in order to determine for each supplier the price difference between the first and second event.

Cause

SAP Ariba Sourcing does not provide a native functionality to compare supplier pricing across multiple events within the same project.

Resolution

Currently, SAP Ariba Sourcing does not provide an out-of-the-box report or user interface feature that directly compares supplier pricing between two separate sourcing events — even if both events are part of the same project.

The standard bid comparison and reporting tools, such as Bid Summary and Configurable Bid Comparison report are designed to analyze supplier responses within a single event only.

Alternatively, you can achieve the required comparison using the following method:

1. Export bid data from each event separately

  • Run reports such as Bid Summary report or Configurable Bid Comparison Report
  • Export the reports to Excel.

2. Align supplier and item data externally

  • Match suppliers from Event 1 and Event 2.
  • Align line items, lots, or pricing terms as applicable.
