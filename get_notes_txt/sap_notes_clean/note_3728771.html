Symptom

  • The system performs the valuation based on document currency amounts instead of local currency (company code currency).
  • The issue occurs during the use of the FAGL_FC_TRANS transaction code.

Environment

  • SAP S/4HANA
  • SAP S/4HANA Finance

Reproducing the Issue

  1. Execute the FAGL_FC_TRANS transaction code.
  2. Check the source currency type in the proposal screen.

Cause

The source currency type is retrieved from the company code settings (T-code FINSC_LEDGER).

If the source currency of the valuated currency type is set to 00 (Document Currency) in the company code settings, the system performs valuation based on document currency amounts instead of local currency balances.

Resolution

  1. Review the company code settings in T-code FINSC_LEDGER.
  2. Verify the source currency type for the relevant valuated currency type.
