Symptom

An application error occurs when creating a job requisition from position with the fingerprint 91d0b3df1ce37b045c4a27f7e2113d4d2ed2976e.

Environment

SAP SuccessFactors Recruiting Management

Reproducing the Issue

  1. Attempt to create a job requisition from position management.
  2. An application error occurs.

Cause

There is a custom field with the ID that has the same ID as a standard field, for example isDraft used on the requisition template.

Resolution

Custom fields on the Job requisition template should use unique field IDs. If you have a custom field using a standard field ID it can cause a conflict in the system resulting in errors. To avoid this issue please change the field ID. Please note this will unfortunately lead to data loss.

See Also

List of Standard Job Requisition field IDs.
