Symptom

  • We want to use the schedule publication feature in SAP Analytics Cloud
  • Unable to find the option to schedule publication in our SAP Analytics Cloud system
  • How to enabled the publication feature in SAP Analytics Cloud?

Environment

  • SAP Analytics Cloud hosted on SAP Data Center i.e. NEO environment
    • A single-digit number in your SAC Tenant URL, for example us1 or eu1 or jp1, indicates an SAP data center i.e. NEO environment

Cause

  • Schedule publication feature is not supported on SAP Data centers i.e. NEO environment

Resolution

  1. The scheduling publication is available only on Cloud Foundry environments listed below
    1. AWS Europe Central (EU10)
    2. AWS US East (US10)
    3. AWS Brazil (BR10))
    4. AWS Japan (JP10)
    5. AWS Sydney (AP10)
    6. AWS Singapore (AP11)
    7. Ali Cloud China (CN40)
  2. If you want to use this feature, please contact your SAP Account Executive or SAP Customer Engagement Executive to evaluate your options to migrate from NEO to Cloud Foundry.
