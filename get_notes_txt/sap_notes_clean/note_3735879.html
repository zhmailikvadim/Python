Symptom

Certain fields (House Bank/Tax Code, ...) is not available as Dimension in New Journal Entry Analyzer - F0956B app, while available in Journal Entry Analyzer - F0956.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Access Journal Entry Analyzer (New Version) - F0956B.
  2. In Layout, search for House Bank or Tax Code in Available Objects, observe that they are not available under Dimensions
  3. Access the classic Journal Entry Analyzer - F0956 app and verify that House Bank and Tax Code are available there.

Cause

Journal Entry Analyzer (New Version) - F0956B is based on CDS views C_GLAccountLineItemQuery and I_JournalEntryAnalyzerCube, which do not expose House Bank or Tax Code by default.

Resolution

In order to add new dimensions from Journal Entry or Association views to the Journal Entry Analyzer (New Version), utilize the Data Source Extension function in app Custom Fields following below steps:

  1. Open the app Custom Fields
  2. Navigate to tab Data Source Extension
  3. Create new Data Source Extension for Data Source I_JournalEntryAnalyzerCube
  4. In tab Field Selection, add the necessary fields (House Bank, Tax Code, ...).
  5. Save and Publish the I_JournalEntryAnalyzerCube extension.
  6. Create a second Data Source Extension for the CDS view C_GLAccountLineItemQuery and include the newly extended fields from I_JournalEntryAnalyzerCube.
  7. Save and Publish the C_GLAccountLineItemQuery extension.
  8. Open the New Journal Entry Analyzer (F0956B) and add House Bank and Tax Code as dimensions in the report.
