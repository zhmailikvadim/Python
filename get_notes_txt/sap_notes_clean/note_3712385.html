Symptom

When opening a Price Review document, the derived attributes are not automatically loaded during document initialization. As a result, the derived attributes are missing when the document is first displayed. This SAP Note addresses the issue by ensuring that the derived attributes are correctly determined and loaded when the Price Review document is opened.

Other Terms

/IRM/IPPDM, Derived attribute, Price review Display

Reason and Prerequisites

This is caused by program error.
Note : SAP Note#3712394 is available for ECC release.

Solution

Please implement the correction instructions. These corrections are only valid if you have applied the following Support Packs.

Release V/4 – 2022 V2, 2023, 2023 V2, 2023 V3 and Support Pack 1

Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object
