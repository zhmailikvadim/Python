Symptom

In class /SCMTMS/CL_DG_CHECK_SERVICES, there are buffer tables (static variables) that are used for non-DG-relevant document keys. However, these buffer tables are never cleared, so that in some scenarios the DG check is not executed for documents, although it became relevant in the meanwhile (e.g. by loading a DG-relevant item).

Other Terms

Dangerous Goods, Buffering

Reason and Prerequisites

There is a program error.

Solution

Implement the corrections of the attached correction instructions or apply the corresponding support package.
