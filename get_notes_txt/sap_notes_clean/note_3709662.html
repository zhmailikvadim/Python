Symptom

In the "Reporting for General Licenses" application, when generating report files, the "Goods Description 1–4" fields are missed from the XML file if the product number is not found in the license master data materials.

Other Terms

/SAPSLL/CONLCLIC_GEN, PRTXT, GTXT1, GTXT2, GTXT3, GTXT4, BAFA

Reason and Prerequisites

Program error.

Solution

Apply the correction instruction in this note or upgrade to relevant support package.
