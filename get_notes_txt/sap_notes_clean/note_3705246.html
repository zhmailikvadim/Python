Symptom

Rehire coordinator with all necessary permissions found an empty employment profile selection to-do task for a certain candidate

Environment

SAP SuccessFactors Onboarding

Cause

The issue occurs because the candidate associated with the "Employment Profile Selection" to-do task has already been hired, and the Recruitment Management (RCM) data cannot be fetched.

Resolution

In this case, check tool cannot remove the to do task. Need to run script to remove it.

Keywords

stale to-do tasks, employment profile selection, onboarding 2.0, home page cards, RCM data, process trigger ID, SuccessFactors onboarding, task removal, onboarding tasks
