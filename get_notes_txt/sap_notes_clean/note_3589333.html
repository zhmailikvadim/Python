Symptom

 When attempting to install HANA Tools, you encounter the following error:

Unable to read repository at [URL]
Unable to read repository at [URL]
PKIX path building failed: sun.security.provider.certpath.SunCertPathBuilderException: unable to find valid certification path to requested target

The issue still persists after following the resolution instructions provided in 3131747, where you need to Adapt the eclipse.ini (found in <drive>:\Users\<user>\eclipse\<java_version>\eclipse\eclipse.ini.

Environment

  • Eclipse
  • ABAP Development Tools

Resolution

Since Eclipse 2024-12 the parameters below are set as default:

-Djavax.net.ssl.trustStore=NONE
-Djavax.net.ssl.trustStoreType=Windows-ROOT

Delete these lines in the eclipse.ini.
