Symptom

The configuration model in AVC application Simulate Configuration Models delivers results, which cannot be explained.

Environment

  • SAP S/4HANA Cloud Public Edition
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

Perform a configuration with AVC application Simulate Configuration Models in a context of a business object, e.g. configure sales order item.

Cause

The AVC configuration model is not properly defined. 

Resolution

  1. Open the application Simulate Configuration Models in FIORI launchpad.
  2. Set the Business Object e.g. sales order.
  3. Open the side panel.
  4. Switch from Inspector to Trace tab and activate the trace.
  5. Make changes in the configuration.
  6. Reload the configuration model.
  7. Analyze the trace result to investigate unexpected configuration results.
  8. Change the configuration model if needed.
