Symptom

Posting a downpayment or downpayment request results in error "Down payments to asset are not supported" FINS_FAA_POST004.

Environment

SAP Public Cloud Edition

Reproducing the Issue

When trying to post a downpayment or downpayment request eg with a PO with an asset assigned, the error is triggered. 

Cause

Downpayments or downpayment requests posted directly to an asset were never supported, the posting is correctly rejected. 

Resolution

Post to an investment project, ensuring downpayments and downpayment requests are cleared before settlement. 

See Help Guide for full process.
