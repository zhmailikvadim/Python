Symptom

  • The job template "Reprocess Workflow Events" for reprocessing workflow items does not pick up certain DCD-related workflow events.
  • Workflow events that are stuck in the event queue with status “Error” are not reprocessed as expected.

Environment

SAP S/4HANA Cloud Public Edition

Cause

The issue occurs due to program error. 

Resolution

The issue is fixed with release CE2602.2.

Keywords

DCD, Dispute Case Document, workflow reprocessing job, workflow event queue, event queue error status, workflow event not triggered, workflow event not reprocessed, event status Error, workflow items stuck, function module incorrect call, workflow event ignored, background job not picking events, CE2602.2, workflow correction, reprocessing failed, DCD workflow issue, event delivery failure
