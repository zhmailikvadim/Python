Update of SAP Note 3606625

26/03/2026 The solution part of this SAP Note has been extended by information, relevant for customers, who did a modification of any of the wage types listed below.

-------------------------------------

Within the Belgian Payroll it is possible for several taxes to manually overrule the base taken into account for the calculation of the corresponding taxes, or to over rule the calculated taxes themselves.

This is done on employee level by creating records of infotype Additional Payments (0015) with the corresponding model wage type or a customer specific wage type which contains similar settings for the processing classes, evaluated by the corresponding personnel calculation rules for the tax processing.

With this SAP Note 3606625, the naming of the wage types and the processing class values has been reworked. In addition, all wage types and processing class values have been documented, to explain their usage.

Reason and Prerequisites

This SAP Note is only relevant for Payroll Processing Belgium.

Solution

As a general rule, SAP recommends that you install a solution by applying a Support Package. 

You can find more information about the Note Assistant in SAP Service Marketplace, under service.sap.com/note-assistant.

Note that documentation, transactions, and Customizing activities are only delivered via Support Package.

Note

The improvements delivered via this SAP Note will be delivered via HR Support Packages only, as only wage type description, processing class value descriptions and HR documentation objects are changed, the processing of the wage types is not impacted by this SAP Note.

Although only the description and the documentation of the above mentioned wage types and processing class values has been changed, the complete customizing of the wage types in table T512W had to be included in the delivery. Customers, who modified in the past any of these wage types, which are all in the SAP name range, should take action to ensure that they, after the implementation of the HR Support Package, again apply their modification to these wage types.

Several KBAs describe how to proceed with table T512W and HR Support Package implementations.
