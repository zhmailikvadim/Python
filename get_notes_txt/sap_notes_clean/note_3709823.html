Symptom

Message ME039 is configured as error message and the delivery date is in the past in the service PO. Then, even ME039 is triggered as error, the PO can be saved. 

Environment

  • Materials Management (MM)
  • SAP S/4HANA 1809
  • SAP S/4HANA 1909
  • SAP S/4HANA 2020
  • SAP S/4HANA 2021

Reproducing the Issue

  1. Set ME039 as error message.
  2. Create a service PO with delivery date in the past.
  3. Save the PO.

Cause

The following note is missing:
3131368 - service PO's can be saved despite error messages on item

Resolution

Implement note 3131368 - service PO's can be saved despite error messages on item.

See Also

Keywords

ME039, delivery date, past, service
