Symptom

  • Missing Tabs :  future costing run and current costing run Tabs  are  not avaiable  in accounting view of T- code MM03.
  • Missing Mat. Price Analysis botton is missing in  in accounting view of T- code MM03 .

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental." 

Environment

  • SAP ERP
  • SAP enhancement package  for SAP ERP
  • SAP S/4 HANA

Reproducing the Issue

  1. Open transaction MM03 .
  2. Select the material 
  3. Navigate to the accounting view 1.
  4. Observe that specific tabs such as  future costing run  and current costing run  are missing.
  5. Observe that Mat. Price Analysis botton is missing.
  6. Observe that tabs such as "Future costing run", "Current costing run", and "Prev. costing run" are displayed for normal material

Cause

The material with the missing tab is a split-valuated material and the missing  tab view  is for the mother segment of the material.

Resolution

  1. Open transaction MM03 for the split-valuated material.
  2. Click on the "Org. Level" icon.
  3. Select the valuation type (e.g., 1, 2, 3, 4, or 5).
  4. Verify that the complete view, including all tabs, is displayed as seen in the accounting view of a normal material.
