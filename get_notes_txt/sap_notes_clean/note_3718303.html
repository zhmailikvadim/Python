Symptom

User consolidates products with the MDG, Consolidation Fiori application. Product data includes assigned engineering change numbers (ECNs). The ECN data of the active records added to the consolidation process is not modified in the active area. However, during the activation step, the MDC_MODEL 065 error is raised, indicating concurrent ECN changes in the active area tables.

Other Terms

MARA_AEOI, MARA_AEOI_PRC, PPEND_MARA_AEOI, RESOLVE_CONCURRENT_CHANGES, CL_MDC_DATA_MAT

Reason and Prerequisites

Program error.

Solution

Implement correction instructions or apply relevant Support package.
