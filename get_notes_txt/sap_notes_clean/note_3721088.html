Symptom

  • The field "Allow Workers to manage their Cost Centers" is visible on the SOW worker edit page.
  • The visibility of this field needs to be disabled.

Environment

BNS-FG-SWK-EDT

Reproducing the Issue

  1. Navigate to the SOW worker edit page.
  2. Locate the setting "Allow Workers to manage their Cost Centers."

Resolution

  1. This can be done by disabling a company config "Workers have access to all Cost Centers and Task Codes from applicable associations."
  2. Disable the configuration and verify that the field "Allow Workers to manage their Cost Centers" is no longer visible on the SOW worker edit page.
  3. This config is available to Configuration Manager to edit.
