Symptom

The spool request identifier containing the rendered document is now being captured in the output request item assigned to channel "PRINT" as it is being issued.  When an output request item assigned with a spool request identifier is either deleted or the status is reset from 'Error', the corresponding spool request identifier will now be deleted from the spool.

Other Terms

S/4HANA output control, spool request identifier, APOC_D_OR_ITEM, SPOOL_REQUEST_ID, SP01

Reason and Prerequisites

This is an enhancement for SAP S/4HANA Output Control.

Solution

Apply this SAP Note or upgrade to the listed Support Package.
