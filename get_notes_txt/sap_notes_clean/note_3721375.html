Symptom

I need to enable SAP Central Invoice Management integration with SAP Integration Suite, managed gateway for spend management and SAP Business Network for master data. 

Do I need to create a case with SAP Technical support? What information do I need to provide?

Environment

  • Managed Gateway for Spend&Network
  • SAP ERP Central Component (ECC)
  • SAP S/4HANA
  • SAP Central Invoice Management integration

Resolution

To enable SAP Central Invoice Management integration with SAP Integration Suite, managed gateway for spend management and SAP Business Network for master data, you perform the following steps.

  1. Review the setup requirements in the guide Setting Up Invoice Processing with SAP Ariba Central Invoice Management (4N6), specifically pages 22 and 23, which describe the prerequisites for integration enablement and user requirements.

  2. Create a case in the SAP Support Portal under the component BNS-ARI-CI-PLT.

  3. Use the following subject line for the incident:
    SAP Integration Suite, managed gateway for spend management and SAP Business Network user ID (P-User ID) for Central Invoice Management master data

  4. In the incident description, request the enablement of SAP Central Invoice Management integration with SAP Integration Suite, managed gateway for spend management and SAP Business Network for master data.

  5. Provide the required details for P-User creation in the ticket:

    • Zone ID: 

    • System ID: 

    • E-Mail: 

After the incident is processed and the P-User is created, SAP enables the required integration for master data between SAP Central Invoice Management and SAP Integration Suite, managed gateway for spend management and SAP Business Network.
