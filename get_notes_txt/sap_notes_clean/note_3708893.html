Symptom

In a BoM you defined an alternative item group. If the BoM is subject to parameter effectivity, it is possible to define different validities of the components on the alternative item group. It is possible that the primary component of the alternative item group is the only valid component of this group for certain parameter values. If now a shop floor order is created with such parameter values the primary component will be the only component in the alternative item group. In the manage order app for this primary component the exchange dialogf can be opened, even no alternative component is available.

Other Terms

Manage Production Orders, AIG, Exchange Alternative Components

Reason and Prerequisites

Program Error

Solution

After applying the correction instruction the alternative item information from the primary component will be removed at order creation in case no alternative components are available.
