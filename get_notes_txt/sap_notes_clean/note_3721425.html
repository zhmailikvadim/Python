Symptom

You declare your import declaration in SAP GTS to customs system DMS.

If you enter more than one container on the "Packaging Data" tab at header level, the incompleteness check prevents you from sending the message with the following error message: "Assign container ... to an item"

The error message remains even if you assign the container to an item.

Other Terms

Denmark, XML, UCC, /SAPSLL/LLEG_ECC_DATAF56, dk_dms_im_container_reference, container

Reason and Prerequisites

The problem is caused by a program error.

Prerequisities:

  • Note 3643910 - UCC: Cross-country logic for data retrieval - containers and item references in the import customs declaration

Solution

Implement the correction instructions.
