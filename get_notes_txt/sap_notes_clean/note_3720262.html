Symptom

You are working in an A/R or A/P Invoice, when changing the row quantity to zero you receive the system message: 

Existing freight charges for the row will be deleted; continue?

Environment

  • SAP Business One
  • SAP Business One, version for SAP HANA

Reproducing the Issue

  1. Go to Sales - A/R → A/R Invoice: select a Business Partner (BP) and enter some lines with items and freight. 
  2. Go back to one of the lines, change the quantity to zero and tab out of the field. 

Cause

  • This behavior is aligned with system functionality. 
  • The system does not permit a freight amount to remain when the quantity is set to zero. 

Resolution

  • The system message informs you that the line freight will be removed if you select "OK" to proceed.
  • If you select "Cancel," the quantity will remain unchanged.
