Symptom

  • In transaction AW01N, the 'document number' field is empty for certain transactions.
  • When trying to open affected line item error message is displayed: "It is not possible to display a document from Financial Accounting!" (Message No. A3004).

Environment

  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA Finance
  • SAP S/4HANA

Reproducing the Issue

  1. Transaction AW01N.
  2. Following parameters filled: Company Code / Asset / Fiscal Year
  3. The 'document number' field in the transactions section is empty.
  4. Verify the same in Table ANEP for the specified asset and fiscal year.

Cause

  • The issue is caused by the use of transaction AS91 for external asset acquisition during the legacy data transfer for assets. This transaction creates Asset Accounting (AA) documents without corresponding Financial Accounting (FI) documents.
  • As per SAP S/4HANA 1809, data structure changes in Asset Accounting have moved actual data from tables ANEP, ANEK, and others to ACDOCA, with statistical data stored in FAAT_DOC_IT.

Resolution

It is considered as standard system design.
