Symptom

Allowed fields for Dynamic Packaging are restricted in transaction FAGLGVTR_PJO via parameters 'Fixed BS Packaging Fields' and/or 'Fixed P&L Packaging Fields'. If the entered fields are not allowed for Dynamic Packaging, e.g. because they are derived within CDS view FGL_BCF_BS or via BAdI BADI_GVTR_DERIVE_FIELDS, the program ends with runtime error UNCAUGHT_EXCEPTION. The program should validate if there are still allowed fields for Dynamic Packaging after applying the field restrictions entered via 'Fixed BS Packaging Fields' and/or 'Fixed P&L Packaging Fields'.

Other Terms

UNCAUGHT_EXCEPTION CX_SY_STRUCT_ATTRIBUTES CL_FGL_BCF_SP_UTIL============CP FAGLGVTR_PJO

Reason and Prerequisites

Missing validation of parameters 'Fixed BS Packaging Fields' and 'Fixed P&L Packaging Fields'.

Solution

Implement correction instructions or corresponding support package.
