Symptom

  • The metric "SAPconnect Send Errors (Today)" is visible in Health Monitoring, but no corresponding event is available to configure events/notifications.
  • When an event will be available so notifications can be sent.

Environment

  • Product: SAP Cloud ALM
  • SAP Cloud ALM - Health Monitoring

Reproducing the Issue

  1. Open Health Monitoring in the application.
  2. Locate the metric "SAPconnect Send Errors (Today)" and attempt to configure an event for notifications.
  3. Observe that no standard event is available for this metric.

Cause

There is currently no standard event available for the "SAPconnect Send Errors (Today)" (SOS) metric in Health Monitoring; it is planned for a future release.

Resolution

  1. Be aware that a standard event for the "SAPconnect Send Errors (Today)" metric is not yet available.
  2. Refer to the product’s "What’s New" page to track the availability of this event when it is released: What’s New for the application.
  3. Once the event becomes available, configure it by following the documented procedure: Configure Events in Health Monitoring.
