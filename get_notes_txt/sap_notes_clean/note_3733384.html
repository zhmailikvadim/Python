Symptom

Warning message WN131 ("Central deletion flag set for site") does not appear when calling transactions WB02 (Change Site) or WB03 (Display Site), even though the archiving flag (XT001W-ACHVM = 'X') is set for the plant.

Other Terms

WB01,WB02, WB03, WN131

Reason and Prerequisites

The archiving check in Include MWBE3FA0_ACHVM_PRUEFUNGEN, FORM ACHVM_PRUEFUNGEN, is only executed when OK_CODE = SPACE. However, in transactions WB02 and WB03, the OK_CODE is always set to 'ENTR' upon initial screen entry, causing the CHECK statement to fail and the archiving check to be skipped.
