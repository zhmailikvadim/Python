Symptom

The following error occurs during implementation of SAP Note 3707710:

Error for Class CL_FIISVAT_HR_PDV_SEQUENCE, Method GET_DATA
"P_HR_CUSTOMERVENDORLINEITEM" is not declared as a table, projection
view, or database view in ABAP Dictionary or does not exist in an
active version.

Environment

  • SAP S/4HANA
  • S4CORE 107
  • S4CORE 108
  • S4CORE 109
  • Croatia

Resolution

Implement SAP notes 3657467 - HR_VAT_DCL new URA customizing for Non-deduct.Input Tax and 3659895 - HR_VAT_DCL - support long invoice number.
