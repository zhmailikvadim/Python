Symptom

SAP Note 3125708 - IT0699: Specify Base Wage Type in HR Renewal adds the specification of the base wage type in the infotype to the function of the infotype German Pension Reform D (0699). If you use module Customizing (view cluster VC_T5DR2) to specify a base wage type, the system checks whether this base wage type is permitted for the module. This check fails in the HR Renewal interface when you create, update, or copy an infotype record. This error does not occur in the SAP GUI.

Other Terms

AVMG, 0699, HR Renewal

Reason and Prerequisites

For the error to occur, the specification of a base wage type must be activated using module Customizing and the permitted wage types must be maintained.

Solution

Implement the attached correction instructions or import the specified support package.

After you import the HR Support Package or implement the correction instructions, you do not have to perform any further follow-on activities.
