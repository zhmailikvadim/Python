Symptom

Error while configuring Employee History field's mapping through: "People Profile" EC Category + any Employee Central Field Mapping

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Environment

SAP SuccessFactors Compensation
SAP SuccessFactors Variable Pay

Reproducing the Issue

  1. Navigate to Compensation Home > Select a Variable Pay template
  2. Go to Plan Setup > Settings > Employee Central Employee History Field Mapping
  3. Select "People Profile" for EC Category > Select any field for Employee Central Field Mapping
  4. Click Save > Error pop's up: 4c2013d53f6118554e6c9ddf9cbcf8eea58e5944 

Cause

In current system, mapping an employee history record field to any of People Profile's field is NOT supported. This is a legacy behavior.

Resolution

WORKAROUND:

A possible workaround, instead of mapping to People Profile fields, please, map it to a field in one of the EC Portlet such as: "Job Information", "Compensation Information", "Personal Information", etc.
