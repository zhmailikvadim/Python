Symptom

  • Story designers are now able to quickly review data they are about to upload to a story before creating an embedded dataset with local files since SAC 2026 QRC1 update

Environment

  • SAP Analytics Cloud 2026.2

Resolution

  • Since SAC 2026 QRC1 update, a new data preview dialog allows story designers to quickly review data they are about to upload to a story before creating an embedded dataset with local files.  
  • This preview helps visualizing both the raw data from the file, but also the dimensions and measures the application will create. 
  • It is also possible to perform data preparation tasks before committing the data to the story.
  • This preview also streamlines the data upload process by eliminating unnecessary steps and reducing potential errors. User can now easily abandon incorrect uploads or proceed directly to the story canvas without entering the full dataset editing screen.
  • For detailed information, please refer to help document Add Data to Stories | SAP Help Portal.
