Symptom

Employee can insert and save a record in Personal Information without making any changes to the existing record in Latest People Profile. 
A warning message appears: "Saving this data will duplicate an existing record. Are you sure you want to proceed?"


Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental. 




Environment

SAP SuccessFactors Employee Central

Reproducing the Issue

1. Navigate to Personal Information.
2. Click edit.
3. Provide an effective date, and save the record.

Cause

This is expected behavior.

Resolution

This is expected behavior.
The system allows saving records even if no changes are made, but issues a warning about duplication.
