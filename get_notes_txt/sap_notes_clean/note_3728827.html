Symptom

You try to open an equipment in the Change Technical Object application in Fiori Launchpad, while the same equipment is already opened in edit mode in a parallel session. The error message "Equipment X is already locked by user Y" occurs, the application is opened in display mode, and extra tabs are visible, although they are hidden in the parallel session.
