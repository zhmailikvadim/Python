Symptom

The provided SAP note 3560125 - Support transferring IDs and bank details to GTS is only applicable to S4 systems, and an equivalent SAP note for ECC system is requested.

Environment

  • Global Trade Services (GTS)
  • Global Trade Services / Logistics Services (SLL)
  • SAP Global Trade Services 

Cause

The swift code screening functionality is specific to the E4H system.

Resolution

The swift code screening functionality is specific to the E4H system, and there is no such equivalent SAP note for the ECC system. For more details, please refer to online help portal.
