Symptom

For any customer account, the field formattedName only includes the values from Name and AdditionalName, and does not incorporate AdditionalName2 or AdditionalName3.

Environment

  • SAP Sales Cloud Version 2
  • SAP Sales and Service Cloud Version 2

Reproducing the Issue

  1. Open any customer account.
  2. Fill in the fields Name, AdditionalName, AdditionalName2, and AdditionalName3.
  3. Retrieve the account using the API endpoint: https://{SystemURL}/sap/c4c/api/v1/account-service/accounts/{id}.
  4. Note that the field formattedName only contains the values from Name and AdditionalName, and does not include AdditionalName2 or AdditionalName3.

Resolution

This is the current system behavior in Sales Cloud Version 2. The formattedName is derived only from the fields Name and AdditionalName. The fields AdditionalName2 and AdditionalName3 are not considered in the formatted name derivation logic and are intended for extended name information only.
