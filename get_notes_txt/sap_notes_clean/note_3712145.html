Symptom

Error "In the context of Data Services an unknown internal server error occurred" occurs when selecting a bank statement item in Reprocess Bank Statement Items App.

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental." 

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Open the Reprocess Bank Statement Items App.
  2. Select one bank statement item.

Cause

The posting date is blank for the selected bank statement item.

Resolution

Since these bank statements have posted, the posting date cannot be modified through Manage Bank Statements App. Hence, reverse the statements with blank posting date, then re-upload/ create these items and make sure that the posting date is fully filled.
