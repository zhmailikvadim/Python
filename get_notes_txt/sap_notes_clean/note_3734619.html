Symptom

In SAP S/4HANA Cloud Public Edition, in app Clear G/L Accounts - Manual Clearing, field "segment" for certain line items are shown as empty. 

However, in app Display Line items in General Ledger, it can be seen that these items have value for field segment.

Environment

SAP S/4HANA Cloud Public Edition

Resolution

App Clear G/L Accounts displays data on entry level, while app Display Line items in General Ledger displays data on ledger level.

For the line items with issue, it can be seen from corresponding journal entries that they have segment assigned only on ledger view, but not on entry view.

Therefore, these line items will have empty segment in app Clear G/L accounts.
