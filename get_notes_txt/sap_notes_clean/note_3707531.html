Symptom

  • Dragging a nested node to outside table columns should result in that node being changed to a root node. However, this behavior is not occurring as expected.

Environment

SAP CAP – node.js runtime

Reproducing the Issue

Trying to drag nested node to root.

Cause

Program limitation.

Resolution

  1. The feature to change a nested node to a root node by dragging it outside table columns is implemented and will be available in the next cds release >=9.6.4.
  2. Track the cds versions at the following link: cds versions.
  3. Refer to the release notes for updates: release notes.
