Symptom

You want to replace the SAP Common Crypto Library (CCL) of an SAP HANA Client with a newer version.

Other Terms

CCL, Cryptographic Service Provider, CommonCryptoLib, SAPCRYPTOLIB, libsapcrypto.so

Reason and Prerequisites

All SAP HANA clients support newer SAP Common Crypto Library (CCL) versions than that delivered with the client installation.

Find the version history of the SAP Common Crypto Library in SAP Note 1848999.

Solution

The SAP HANA client allows to replace the installed CommonCryptoLib SAR package (SAPCRYPTOLIBP) with a newer version.

Download and install the SAP Common Crypto Library from the SAP Software Center.

To avoid any disruptions, please perform the replacement while the client is not running.

The SAP Common Crypto Library binary can be found and replaced in the root directory of the SAP HANA client installation:

Linux / AIX:
$HDB_CLIENT_HOME/libsapcrypto.so

macOS:
$HDB_CLIENT_HOME/libsapcrypto.dylib

MS Windows:
%HDB_CLIENT_HOME%/sapcrypto.dll
