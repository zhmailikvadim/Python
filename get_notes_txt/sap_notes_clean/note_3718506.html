Symptom

  • Custom branding of SuccessFactors cannot be enabled in SAP Jam.
  • The option for SuccessFactors themes is not selectable in SAP Jam Admin Branding settings.
  • The option "SuccessFactors (V12 or higher required)" is missing

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental." 

Environment

  • SAP Jam Collaboration
  • SAP SuccessFactors HCM Suite

Reproducing the Issue

  1. Navigate to SAP Jam Admin.
  2. Go to Branding settings.
  3. Attempt to select the SuccessFactors themes option in the Web sub-tab.
  4. Observe that the option is missing.

Cause

The unavailability of SuccessFactors themes is expected behavior after SAP Jam was decoupled from IAS integration.

Resolution

  • Note that the unavailability of SuccessFactors themes is expected behavior after decoupling.
  • If Jam is not integrated with BizX, then custom theming from SF will not be available.
  • This is expected behaviour from the system.
