Symptom

This SAP Note introduces the updated version of the VAT Return (Version 5.12), in compliance with Government Regulation Order № ЕД-7-3/1227@ from February 02, 2026, and effective from Q1 2026. The updates apply to Sections 1-7 (J3RVATDECL).

In this update, the following new elements have been introduced:
- New hierarchies: RUV26
- New VAT Return version: RUVAT2026

Other Terms

J_3RVATDECL,  J3RVATDECL, RUVAT2026, RUV26, 5.12, VAT Return, NO_NDS_1_003_00_05_12_01, XML

Reason and Prerequisites

Legal Requirement according to Government Regulation Order № ЕД-7-3/1227@ from February 02, 2026, valid from Q1 2026.

This SAP Note provides you with the relevant system changes to address the following legal changes:

  • Changes: VAT Return and electronic formats 2026

Solution

Apply the appropriate Support Package or install the attached correction instructions with manual implementation steps.
