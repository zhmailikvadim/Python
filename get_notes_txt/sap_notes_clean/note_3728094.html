Symptom

  • During an upgrade to S/4HANA 2025, the SUM tool displayed an error, and dispatcher services were observed to be GRAY.
  • Error message in SUM trace: "SYSTEM MANAGER: SUM instance with status GRAY not allowed."

Environment

  • Upgrade tools for ABAP (SUM 2.0)
  • SAP target release 816 (S/4HANA 2025)
  • SAP kernel release 916

Reproducing the Issue

  1. Initiate an upgrade to S/4HANA 2025 using the SUM tool.
  2. Observe that dispatcher process is GRAY during the upgrade process in the mentioned phase.

Cause

  • Kernel version mismatch was identified, with old kernel executables present in the directories DIR_CT_RUN and DIR_EXECUTABLES, instead of the target kernel version (916).
  • The presence of old kernel executables in directories DIR_CT_RUN and DIR_EXECUTABLES instead of the target kernel version (916) caused the dispatcher services to remain in GRAY status.

Resolution

  1. Move out the old kernel executables from directories DIR_CT_RUN and DIR_EXECUTABLES.
  2. Extract and replace the kernel of the target version (916) in the aforementioned directories.
  3. Restart the application server.
  4. Verify if all processes are in GREEN state.
  5. Repeat the SUM phase.
