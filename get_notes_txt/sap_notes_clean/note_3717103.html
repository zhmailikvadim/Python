Symptom

The G5 certificate rollout for Cloud Foundry landscapes is in progress and has already been completed in some regions.  Others regions were paused and are expected to resume in the coming weeks.  To see the current rollout status please refer to SAP Note 3566727 - Root Certificate Replacement in the SAP BTP, Cloud Foundry Environment.

After the switch to the G5 certificate, MDK based applications may fail to initially connect with the error "The server URL you are trying to access is not reachable.  Use different connection settings or try again later".

Solution

iOS\iPadOS Devices

  • If the device is running iOS 18 or higher, the G5 certificates should already be supported and no changes to MDK clients should be necessary.
  • If the device running an older iOS/iPadOS version can be updated, then updating to iOS/iPadOS 18 or higher is the best option.
  • Older devices that cannot be updated to iOS 18 or higher can manually install the G5 certificate and trust it to add support for the G5 certificate.
  • No MDK client updates should be required after adding the certificate to the device.

Android Devices

  • If the device is running Android 14 or higher, the G5 certificates should already be supported and no changes to MDK/SSAM clients should be necessary.
  • If the device running an older version of Android can be updated to Android 14 or higher, then updating is the best option as no new MDK client version is needed after the device OS update.
  • Older devices that cannot be updated to Android 14 or higher will need to use a newer MDK client version.
    • MDK 25.6.x and above have support for G5 already built into the MDK client.
    • Upgrade your MDK based application to use MDK 25.6 or higher and create a new branded client and rollout to your devices.
    • Newer MDK client versions work with both the existing G2 Certificate landscapes as well as the new G5 landscapes

Customers using an SAP application based on MDK, like SAP Service and Asset Manager, should contact the appropriate application support team for any specific upgrade questions.
