Symptom

  • In the Download Center, the Cloud Printer Manager is visible, but click  download it shows "NO DATA"
  • Why can't I download Cloud Print Manager?

Environment

S/4HANA Output Management Print Manager

Reproducing the Issue

  1. Navigate to the Download Center
  2. Locate the Cloud Printer Manager
  3. Attempt to download the Cloud Printer Manager
  4. Observe that the Download Center displays "NO DATA"

Cause

There are below possibilities for this issue:

  • Trying to access old Patch of the CPM which may already archived and not possible to access.
  • The S-user missing authorization for downloading.

Resolution

Possible solutions:

  • Confirm and download the SAP CLOUD PRNT MGR 800 FOR PI latest patch, as old ones are archived 
  • Follow the tips in KBA 1568179 - How to use and find the software downloads in SAP Software Center - SAP for Me (e.g. S-user ID missing Software Download authorization)
