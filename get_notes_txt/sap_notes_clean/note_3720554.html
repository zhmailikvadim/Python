Symptom

In NZDt AddOn, DEVCLASS SM&P is added to the exception table for table classification /NZT/S4_KB_EXCEP with PROCESS_OPTION=6.
In Netweaver 7.5, package SM&P is renamed to S_MODELLING_AND_PROTOTYPING, therefore the /NZT/S4_KB_EXCEP entry for DEVCLASS SM&P becomes insufficient.

The following entry is added to the /NZT/S4_KB_EXCEP table:
MT_ID: SAP
OBJECT_TYPE: 1
OBJECT_NAME: S_MODELLING_AND_PROTOTYPING
PROCESS_OPTION: 6

Other Terms

downtime minimization tool for SAP S/4HANA, NZDT

Reason and Prerequisites

Any of the following add-on support packages is installed in your system landscape:

  • NZDT 1.0 SP03

Solution

Required Action

In order for this note correction to be effective, the note must be implemented before:

  • Starting a new analysis run

Implement the correction instructions in transaction SNOTE on the following systems of your system landscape:

  • Central system
