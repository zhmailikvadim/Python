Symptom

During the pre-upgrade checks, no GUI window appears for any warnings as the system checked customer-provided demo DB that had the stored procedure.

Reproducing the issue

Scenario:

Use the setup wizard to upgrade company DB.

Expected behavior:

The upgrade logs have this warning for ID 22:

Check: Stored procedures consistency check

Desc: (22) Checks consistency of created stored procedures between upgraded company DB and SBO-COMMON. For more information, see SAP Note 2191992.

Test result: Warning

Actual behavior:

Upgrade wizard does not show any warning message for any pre-checks of stored procedures.

Solution

SAP intends to provide a feature package or patch to solve the symptom that is described in this SAP Note.
