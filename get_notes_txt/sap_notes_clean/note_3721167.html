Symptom

While submitting Belgium eDocuments , system throws following error:
"Attribute authority of field identifier is empty in Sender" Message no. EDOCUMENT_EU079
"Attribute authority of field identifier is empty in Receiver" Message no. EDOCUMENT_EU080

Image/data in this KBA is from SAP internal systems, sample data, or demo systems and are for reference purposes only.

Environment

  • SAP ERP
  • SAP S/4HANA
  • SAP Document and Reporting Compliance service
  • SAP Document and Reporting Compliance, cloud edition

Reproducing the Issue

  1. Launch Edoc_cockpit transaction
  2. Navigate to Belgium EU eInvoice
  3. Click on Submit Button

Cause

Missing configurations of fix values 

Resolution

Ensure the value for `Fix value SBDH AUTHORITY` is maintained in Tx code AIF/CUST . This will help ensure consistency and accuracy in the process.
