Symptom

You edit a business partner in transaction BP in a customer role, such as FLCU00 or FLCU01. You delete industry code 2, for example, and expect that industry code 3 (and the following) will move up, as is the case in transaction XD02. This does not happen.

Other Terms

Customer, CVI, customer vendor integratio

Reason and Prerequisites

A function is missing in transaction BP

Solution

Implement this SAP Note or import the SP.
