Environment

  • SAP Analytics Cloud(Enterprise)
  • Import Data Connections (OData service)
  • SAP Datasphere

Reproducing the Issue

  1. Schedule a calendar event for a multi-action to refresh the import for a dimension.
  2. Execute the scheduled import job for the dimension model "M_CCF_NIC_PLANNING_01".
  3. Observe the job failure with the error message: "The job failed due to an error in plugin service. Please check your source for more information."

Cause

This issue is being investigated by development.

Resolution

The solution of this issue is currently under investigation. Please follow this KBA for updates.
