Symptom

No entries are displayed in the "Characteristics as Key Figures" folder in DataStore object maintenance for the time characteristics contained in the data part of a DataStore object of the type "Direct Update" with activated planning mode. You can therefore not select any SID key figures for these time characteristics.

Other Terms

SID key figures, time characteristics

Reason and Prerequisites

Time characteristics are ignored.

Solution

Corresponding entries are displayed in the "Characteristics as Key Figures" folder in DataStore object maintenance for time characteristics from the data part of a DataStore object of the type "Direct Update" with activated planning mode.

You can import the following support package instead of implementing this SAP Note:

  • SAP BW 7.40

    Import Support Package 11 for SAP BW 7.40 (SAPKW74011) into your BW system. The support package is available once SAP Note 2100895 ("SAPBWNews NW BW 7.4 ABAP SP11"), which describes this support package in more detail, has been released for customers.

           
In urgent cases, you can implement the correction instructions as an advance correction.

You must first read SAP Note 1668882, which provides information about transaction SNOTE.

To provide information in advance, the SAP Note(s) mentioned above might be available before the corresponding support package has been released. In this case, the short text of the SAP Note will still contain the words "Preliminary version".
