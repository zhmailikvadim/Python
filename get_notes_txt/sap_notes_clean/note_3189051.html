Symptom

Resolution

Previously, for a buyer on SAP Ariba Buying and Invoicing integrated with SAP ERP or SAP S/4HANA and SAP Fieldglass using file channel integration, the MASTER_DATA_EXPORT program did not run when scheduled on SM37.

This issue occurred due to a coding error.

Now, the code has been enhanced to fix the problem.

Action required for new and existing customers

  1. Go to https://connectsupport.ariba.com and log in. If you do not have a User ID and Password for Ariba Connect, contact your Ariba account executive.
  2. Click Answers → Documentation & Learning.
  3. Navigate to the Cloud Integration tab and click Resources. The Ariba Cloud Integration Resources page appears.
  4. Under Integration Tools, click Integration tools for Ariba Procure-to-Pay. The Procurement and Invoicing Resources page appears.
  5. On the Procurement and Invoicing Resources page, click ABAP Transports for SAP ERP.
  6. Download and extract the SAP Transports (SAP only).zip.
  7. Navigate to Ariba_P2P_SAP_Transports → Ariba_P2P_SAP_Transports_CI-9 SP2 HF51_Sep2021.
  8. Navigate the transports folder and import the required transports to the appropriate location on your SAP ERP or SAP S/4HANA system.
