Symptom

Biographical information custom fields are not appearing dependent's biographical info when personInfo_dependent is disabled.

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental

Environment

SAP SuccessFactors Employee Central

Reproducing the Issue

1.You can configure a custom field under personInfo and you have disabled personInfo_dependent.

2. When you try to edit dependent's personInfo from UI, you can see custom field is not there.

Cause

This is an expected behavior.

Resolution

Please raise an enhancement request if you would like show the custom fields under dependent's personInfo when custom fields are under personInfo hris element.
