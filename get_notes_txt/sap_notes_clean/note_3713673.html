Symptom

You want to change the work center view name Opportunitys inside Sales (Verkauf) work center to Opportunities.

Environment

SAP Cloud for Customer core applications

Reproducing the Issue

  1. Logon in German / Dutch language.
  2. Go to Verkauf (Sales) work center.
  3. See the view name Opportunitys.
  4. You would like to change to Opportunities.

Cause

This is the current correct translation.

Resolution

  1. Where anglicised terms are used in German, they are treated like German words grammatically.
  2. Therefore, the English plural term "opportunities" becomes "Opportunitys" (with capital O and plural s) in German.

Keywords

Sales work center, work center view name, Opportunities translation, German language, anglicised terms, plural terms, correct translation, Opportunity Management, Verkauf work center, change view name, language settings
