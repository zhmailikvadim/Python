Symptom

The following steps are executed in sequence in order for the inconsistency to be observed:

1. A TPO exists in which VAS items have been determined from the FSH_VAS_SD for the key figures. The same VAS is transferred to the linked purchase order.

2. FSH_VAS_RD is executed for the VAS redetermination on the linked purchase order and no VAS items are found from the condition tables. A deletion indicator is set for the VAS items in the linked purchase order. In the sales order, the VAS items are removed.

3. FSH_VAS_RD is again executed now for the VAS redetermination on the sales order and valid VAS items are found from the condition tables. The same VAS items are then transferred to the purchase order, however instead of removing the deletion indicator, the same VAS items are inserted in to the purchase order VAS table.

Other Terms

Third party order, TPO, FSH_VASS_TL, FSH_VASM_TL, FSH_VAS_SD, FSH_VAS_MM, FSH_VAS_RD, FSH_VAS_SER, FSH_VAS_SUB_SER, FSH_VAS_DEL_IND.

Reason and Prerequisites

Reason:

Program error.

Prerequisites:

1. FSH_VAS_SD contains valid VAS items for the key figures and is determined correctly in the sales order.

2. FSH_VAS_MM does not contain any valid VAS items for the same key figures of the purchase order.

Solution

Implement the attached corrections.
