Symptom

When executing application Display Line Items in General Ledger, depending on what filters are filled, below views are applied automatically:
- SAP - Grouped 
- SAP - Grouped by Company Code Only 
- SAP - Grouped by G/L Account Only 
- SAP - Ungrouped 

They are applied also in case the "Standard view" is set as default. 

Environment

As of SAP S/4HANA 2023

Cause

Program design.

Resolution

This feature is available as of SAP S/4HANA 2023:
Dynamic Grouping in the Display Line Items in General Ledger App

The "Standard" view is overwritten only in case it has not been modified and there are no modified standard or custom views set to default. 
In case modified standard views or a custom views are set to default, the dynamic grouping is not applied.
