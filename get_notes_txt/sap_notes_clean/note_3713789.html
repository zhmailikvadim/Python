Symptom

  • Key Position Field is displayed as blank in the table report when using the Domain Succession MDF Position with EC Employee Central option.
  • Key position is shown as blank for some positions because key positions data is not maintained in UI (SOC).
  • Key position is shown as blank for all positions when logged in as the affected user.

Environment

SAP SuccessFactors Succession Management

Reproducing the Issue

  1. Login and proxy as affected user
  2. Navigate to Report Center and open the report
  3. Observe that key positions data is shown 
  4. Login as affected user
  5. Navigate to Report Center and open the report
  6. Observe that key position data is shown as blank for all positions

Cause

The issue is caused since the language translations are not maintained for the values in Position criticality picklist.Specially, if the language translations for the picklist values are not maintained in users default locale.

Resolution

  1. Identify the affected user default locale in UDF File
  2. Verify that the Key Position field is a picklist field and its value source is correctly configured
  3. Check the labels for the picklist field and confirm labels for position criticality picklist values are maintained and not left blank in any language
  4. Maintain the labels in all the languages  and specially labels in users default locale for all options in the picklist
  5. To modify the labels, navigate to Admin Center → Picklist Center → Search for the picklist field → Edit labels for the required locales
  6. Ensure the labels for all required locales are properly configured to avoid blank values in the reports.

Note: Note that key position data is shown as blank for some positions because key positions data is not maintained in UI (SOC)
