Symptom

There is a performance issue in the MSAD app when the Unassign function is used with SO/Line Item in the selection.

Environment

  • S/4HANA Retail & Fashion Management (LO-RFM) 
  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Enter values for SO and its line items.
  2. Click Go.
  3. Select all displayed records.
  4. Click Unassign.
  5. Expected result: A job is scheduled successfully.
  6. Open Job Overview and check the log report for messages.

Cause

Data sent incorrectly.

Resolution

The recommend strategy is sending both document numbers and document/items to improve performance. 

Keywords

performance, SO, arun, assign, unassign, job, document, number, allocation, sales order
