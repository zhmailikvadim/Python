Symptom

When pricing with CDOTE from GUI, the 'Broker Account' dropdown in the order popup incorrectly truncates leading zeros from the account values.

Replication Steps:
1. Launch transaction WB22.
2. Input the contract number and navigate to the Conditions tab at Item.
3. Click the 'Price a Quantity' button, select a future condition, and then click 'Price with CDOTE'.
4. Once the CDOTE order popup appears, select order type FPEP.
5. Select a Counterparty Broker from the dropdown menu.
6. Observe that the 'Broker Account' dropdown value incorrectly truncates leading zeros.

Other Terms

Agricultural Contract Management; ACM; CDOTE; Broker Account

Reason and Prerequisites

Program error.

Solution

Import the relevant support pack or apply the correction instruction(s).
