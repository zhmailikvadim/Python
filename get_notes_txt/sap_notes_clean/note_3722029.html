Symptom

Error message "Solve Configuration Issue in Code List Mapping for field CashDiscountTermsCode with value ABC" (where ABC means a generic Ext. Code) when replicating data from S/4HANA or ERP to C4C

Environment

  • SAP Cloud for Customer integration with SAP ERP.
  • SAP Cloud for Customer integration with SAP S/4HANA.

Reproducing the Issue

  1. Go to the Administrator work center.
  2. Go to the General Setting work center view.
  3. Open the Web Service Message Monitoring under System Administration.
  4. Select query All.
  5. Search for erroneous message with Detailed Error Text "Solve Configuration Issue in Code List Mapping for field CashDiscountTermsCode with value ABC"

Cause

The external code is not available on the Code List Mapping for Integration with External Applications and Solutions fine-tuning activity under Local Data Type Name CashDiscountTermsCode.

Resolution

to resolve the issue, please follow the steps below:

  1. Go to the Business Configuration work center.
  2. Go to the Overview work center view.
  3. Search and open the fine-tuning activity Code List Mapping for Integration with External Applications and Solutions.
  4. Open the Maintain Code List Mapping.
  5. Filter for Local Data Type Name CashDiscountTermsCode.
  6. Create the mapping for the External Code ABC.
