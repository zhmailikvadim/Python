Symptom

How to customize the font color for field of custom MDF object and display it in the Employee Profile page?

Environment

SAP SuccessFactors HCM Suite

Reproducing the Issue

  1. Go to Manage Configuration UI and click on the Edit Properties icon of the desired field.
  2. Select "Custom 1" as the font color under the Style section and save the changes.
  3. Navigate to the Employee Profile page, where the text color of the field should match the "Custom 1" color.

Resolution

To customize the "Custom 1" font color, follow these steps:

  1. Go to Theme Manager.
  2. Choose the desired theme.
  3. Select Fine Tune > Portlets.
  4. Modify the "Custom 1 Text Color" and "Custom 2 Text Color" settings.

Alternatively, it is also possible to customize the color values by exporting the theme file and using the following parameters:

  • portlet.custom1TextColor=#10C7F5
  • portlet.custom2TextColor=#D91AEE
