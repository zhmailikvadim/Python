Symptom

When attempting to load catalog items for an multi supplier subscription with the IsPartial field set to "False", the following validation error is encountered: "Unspecified Supplier items require IsPartial to be true".

Environment

SAP Ariba Catalog Management

Cause

The configuration setting "Allow Multi-Supplier CIF" is set to "No", which triggered the validation error during catalog import for multi suppliers catalog. 

Resolution

  1. Update the configuration setting "Allow Multi-Supplier CIF" to "Yes".
  2. Re-import the subscription to resolve the validation error.

See Also

Refer to: How do we create a catalog for multiple suppliers?

Keywords

catalog import, validation error, partial catalog items, allow multi-supplier CIF, catalog activation
