Symptom

When replicating an Organizational Object and processing its replication request, the following error is raised in the Application Logs (SLG1):

&1 &2: Error when creating new infotype 1001 record for object &4 -- Message No. SFIOM_PROCESSING206

Environment

  • SAP SuccessFactors Employee Central integration to SAP Business Suite
  • SAP SuccessFactors Employee Central integration to SAP S/4HANA
  • Organizational Objects Replication Employee Master Data Replication
  • Business Integration Builder (BIB)

Reproducing the Issue

  1. Replicate an Organizational Object from SuccessFactors Employee to SAP ERP HCM or SAP S/4HANA OR Replicating an Employee's Organizational Assignment.
  2. Check Application Logs (SLG1).

Cause

In order to maintain consistency the replicated Organizational Data, determines the Relationship Subtype Time Constraint in table T77ZR using the following specific parameter values: Client ID (MANDT), Org. Object Type (SOBID), Infotype (INFTY), Subtype (SUBTY) and Type of Related Org. Object (SOBID)

If there's no valid entry based on these values in table T77ZR, the time constraint for this relationship subtype is not determined, the insert/update procedure is aborted and the error is raised.

Resolution

To fix this issue, the following must be done:

  • Ensure the Primary Field Mapping Subtype has been correctly maintained in the relevant Transformation Template mapped to Infotype 1001 - Field SOBID, as per the implementation guide: Configuring Primary Field Mapping for Organizational Object Replication.
    NOTE: here is a mapping example: Mapping IDs of Parent Business Units and Parent Divisions and the Sample Content delivered by SAP can guide with the correct mapping configuration: Sample Content Available for Mapping Organizational Data Fields.

  • If the Relationship Subtype has been correctly mapped to Infotype 1001 - Field SOBID, ensure there is a valid entry for the relevant Relationship Subtype Time Constraint in table T77ZR.
    NOTE: the Time Constraint behavior in Organizational Management (OM) and its configuration is fully explained in the following KBA: 3477691 - Time Constraints in Organizational Management (OM).
