Symptom

You run the Cálculo de nómina española (RPCALCE0) program for a employee with a fixed quota contract in 2026. However, the system generates the MEI quotas with obsolete values.

Example:

You reproduce the symptom by following these steps:

Description

Payroll run of an apprentice contract employee in 2026.

Preparation/Background

  1. Hire an apprentice contract employee in 2025.
  2. Set the Tipo de Relación Laboral (IT0061-EMPL_RELATION) to 087.

Execution

Run the RPCALCE0 program for this employee in January, 2026.

Result

The system generates the Trab.equida (/3N0) and Empr.equida (/3N1) wage types with values EUR 1.80 and EUR 9.25, respectively.

Other Terms

&CALC - PC00_M04_CALC;

Mecanismo de Equidad Intergeneracional (MEI)

For more information, see SAP Note 1071282 - CONSULT: List of tags for Spain notes.

Reason and Prerequisites

Reason

The amounts for MEI contributions have been updated for the 2026 year.

Prerequisites

The following Support Package level is the minimum prerequisite:

Release Support Package
S4HCM 102 HR SP 00
S4HCM 101 HR SP 05
S4HCM 100 HR SP 18
SAP_HR 6.08 HR SP C9
SAP_HR 6.04 (EhP6 and EhP7)  HR SP K1

For more information, click on SAP ERP Human Capital Management - Support Package Schedules.

Note:

When you install an SAP Note, some prerequisites might be required. The prerequisites are SAP Notes that have already been released and that change the same context of the current one. Then, even if these prerequisites are not necessarily related to the symptom of this SAP Note, you should implement them due to technical reasons.

Solution

From now on, the system presents the following values for MEI contributions, represented by new entries in the Constantes de nómina (V_T511P) view:

Constant Description Amount Currency Start date
E85ME C.Formac. MEI Empresa 10,36 EUR 01.01.2026
E85MT C.Formac. MEI Trabajador 2,07 EUR 01.01.2026
E87ME C.Aprend. MEI Empresa 10,36 EUR 01.01.2026
E87MT C.Aprend. MEI Trabajador 2,07 EUR 01.01.2026
E97ME C.Aprend. MEI Empresa 10,36 EUR 01.01.2026
E97MT C.Aprend. MEI Trabajador 2,07 EUR 01.01.2026

Example:

Following the steps from the example of the Symptom section.

Result

The system generates EUR 2.07 for the Trab.equida (/3N0) and EUR 10.36 for the Empr.equida (/3N1) wage types.
