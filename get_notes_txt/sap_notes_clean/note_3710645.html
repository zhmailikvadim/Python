Symptom

Planning run ends with warning "There is already a scheduled execution not yet finished" and without any other result.

Environment

SAP Business ByDesign

Reproducing the Issue

  1. Navigate to work center Supply Planning, access view Planning Runs.
  2. Select planning run ABC.
  3. Open application log XYZ.
  4. Select the tab Results.
  5. Observe that there is only one warning message: "There is already a scheduled execution not yet finished".

*ABC represents planning run ID

*XYZ represents application log ID

Cause

When multiple jobs are executed simultaneously for a specific planning run, none of them can take effect. In the corresponding application logs, under the tab Results, there will be only one warning message: "There is already a scheduled execution not yet finished".

To validate this, select this planning run, click button View Jobs, select "Scheduled Jobs" in drop down list, usually there are multiple jobs scheduled at same date and time.

Resolution

Avoid schedule multiple jobs at totally same date and time for planning run.
