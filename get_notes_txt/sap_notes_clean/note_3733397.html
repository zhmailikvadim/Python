Symptom

The Shared with customer sites section shows a list of customer sites with whom you've shared this configuration bundle. You can update the customer information or remove customer sites if you don't want to share the bundle with that customer site.

Resolution

Use this procedure to edit or remove customer sites with whom you've shared configuration bundles in the Intelligent Configuration Manager workspace.

Prerequisites
1.You must be a member of the Partner Administrator group and Customer Administrator group.

2.You must use the Sourcing and Contracts site for this functionality.

You can follow the below steps to edit or remove customer sites with whom you have shared configuration bundles 

1.Go to the Intelligent Configuration Manager workspace. For detailed steps, see Accessing Intelligent Configuration Manager Workspace.

2.From the menu bar, choose Bundles > Manage bundles.

3.From the Manage bundles page, the Bundles created section lists all preconfigured bundles created in this site. These can be shared with multiple customer sites.

Optionally, use the full-text search criteria in the search  field to search for the bundle.

1.Choose a bundle row with the Created status that you want to share by selecting > to open and view the bundle details page.

2.From the Shared with customer sites section, choose a row by selecting the radio button.

  • To update the customer site details, select Edit. In the Edit customer label box, update the Customer label field, and select OK.
  • To remove the bundle from a customer site, select Remove. In the conformation box, select Remove.
