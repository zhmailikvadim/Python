Symptom

You have a revenue accounting contract contains several POBs which are all having start date type 4. One of them is a leading-linked structure. When the corresponding events are processed to the POBs not belonging to the leading-linked structure, the exception CX_SY_ITAB_LINE_NOT_FOUND is raised from the method CL_FARR_POB_LEADING_LINKED->IF_FARR_POB_STRUCTURE~FULFILL().

Other Terms

FARR, Revenue Accounting And Reporting, Start Date Type 4, CX_SY_ITAB_LINE_NOT_FOUND

Reason and Prerequisites

The moving statement doesn't find the corresponding POB ID.

Solution

Apply this note.
