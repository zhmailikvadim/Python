Symptom

Some ownership functionalities get improved in BPC10.1 with this note.

Other Terms

Ownership manager, member selector.

Reason and Prerequisites

Improve some ownership functionalities in BPC10.1.

Solution

For SAP Business Planning and Consolidation 10.1, version for SAP NetWeaver on BW740, upgrade to(Component CPMBPC 810) Support Package 12 and upgrade BW740(Component SAP_BW) to Support Package 17.

For SAP Business Planning and Consolidation 10.1, version for SAP NetWeaver on BW750, upgrade to BW750(Component SAP_BW) Support Package 06.

Following ownership functionalities get improved in BPC10.1 version:

1. Group/Entity hierarchy tree is initially collapsed.

2. Add two button icons "Expand All" and "Collapse All" after checkbox "show description".

    "Expand All" is used to expand whole Group/Entity tree.

    "Collapse All" is used to collapse whole Group/Entity tree.

3. After expand the whole Group/Entity hierarchy tree, user can search the Group member or Entity member with Brower "find" functionality (Ctrl + F).

4. When adding Entity members for a Group, user can select checkbox "Only Show Unused Members" in Member Selector to hide entities that have been used in the left hierarchy tree.

5. If user has checked option "Only Show Unused Members" in Member Selector, when a new Member is checked, it will not disappear in the left panel of Member Selector.
