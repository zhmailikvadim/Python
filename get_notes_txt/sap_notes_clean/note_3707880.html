Symptom

Inactive Distribution Rule is available for selection in SAP Business One, Web client.

Reproducing the issue

Prerequisites:

  • The Use Multidimensions checkbox is selected in the SAP Business One client (Administration → System Initialization → General Settings → Cost Accounting tab) and In Separate Columns is selected for Display Distribution Rules.
  • There are several active Dimensions (Financials → Cost Accounting → Dimensions).
  • There are several active Cost Centers (Financials → Cost Accounting → Cost Centers).

Scenario:

  1. Log in SAP Business One client, go to Financials → Cost Accounting → Distribution Rules, define several Distribution Rules for Dimension X.
    • Distribution Rules "A", "B", "C", with Active checkbox ticked and the Effect From... To date range covers the system date.
    • Distribution Rules "D", with Active checkbox not ticked.
    • Distribution Rules "E", with Active checkbox ticked however the Effect From... To date range does not cover the system date. For example, Effective From 01.01.2024 To 31.12.2024，while the system date is 10.01.2026.
  1. Go to Purchasing - A/P → Purchase Order, select a Business Partner in the header and an item in the row.
  2. Make the Dimension X visible in the row, press tab in the Dimension X column, and the Distribution Rules "A", "B", "C" are in the List of Distribution Rules window.
  3. Log in SAP Business One web client and try to create the same Purchase Order and select Distribution Rules for Dimension X.
Expected behavior:

Only the active Distribution Rules ("A", "B", "C") are in the List of Distribution Rules window.

Actual behavior:

All the Distribution Rules ("A", "B", "C","D", "E") are in the List of Distribution Rules window.

Solution
