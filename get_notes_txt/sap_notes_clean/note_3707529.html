Symptom

When using the CL_BCS_MAIL_MESSAGE class to test email functionality, the following error is encountered:

"No technical configuration could be determined for this API."

Environment

SAP S/4HANA 2023

Reproducing the Issue

Use standard transaction SBCS_MAIL_TEST to send one test email via CL_BCS_MAIL_MESSAGE

Cause

"Default SMTP Configuration" is not enabled for the SMTP Configuration ID

Resolution

Ensure the "Default SMTP Configuration" checkbox is selected for the SMTP Configuration ID

Keywords

CL_BCS_MAIL_MESSAGE, SMTP Configuration, SBCS_MAIL_CONFIGSMTP, Default SMTP Configuration, Email Error
