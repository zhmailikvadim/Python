Symptom

While updating file though SAP GUI, it is observed that files exceeding a certain size are scanned in several chunks as opposed to being scanned as a whole. 

Following error message could be found in virus scan trace:

[XXX.XXX.XX] XXXXXX t22703 LIB:VsaScanBuffer(): Scan Byte Buffer xxxxxxxx bytes  at xxxxxxxx. Top level Scan (not in recursion)
[XXX.XXX.XX] XXXXXX t22703 LIB:VsaScanBuffer(): SafeFileName: xxxxx
.....
[XXX.XXX.XX] XXXXX LIB:VsaScanBuffer() returns with -6, "Extension ".pdf" does not match content "application/octet-stream" 

Environment

Netweaver AS ABAP 740
Netweaver AS ABAP 750
Netweaver AS ABAP 751
Netweaver AS ABAP 752

Cause

This is due to program bug.

Resolution

Implement SAP Note 2621219 - GUI_DOWNLOAD: Filetype BIN not downloaded correctly.

Keywords

 GUI_DOWNLOAD, Virus Scan, PDF, Zip, Could not be scanned
