Environment

SAP ERP 6.0

Reproducing the Issue

  1. Execute the program RC_HCM_COLLECT_ANALYSIS_DATA via transaction SA38 in the productive client of the production system to schedule and download the data archive by performing the following three steps:
    1. Select the checks to be included in the analysis.
    2. Choose Schedule Analysis to schedule the master job to collect data.

Cause

Program error

Resolution

  • Apply version 31 or higher of note 3193560 - SAP Readiness Check for SAP SuccessFactors HCM
  • If this dump persists, check the following source code extract
