Symptom

- Unable to locate RFC function modules in SAP Business Application Studio (BAS) / Service Center module
- RFC function modules not visible in the list when creating external connection using Build Code
- Issue occurs while connecting to ABAP RFCs on on-prem system from SAP BTP CAPM with NodeJS

Environment

- SAP Business Technology Platform (BTP)
- SAP Business Application Studio (BAS)
- NodeJS
- On-premise ABAP system

Cause

- Incorrect configuration in BTP Destination and Connector setup
- API filter selection in BAS set to "Classic APIs" instead of "All"

Resolution

1. **BTP Destination and Connector setup:**
   - Verify that the RFC function module(s) are correctly maintained in your BTP Destination and in the connector.
   - Refer to the official SAP documentation for detailed steps: [SAP System Service Provider](https://help.sap.com/docs/bas/sap-business-application-studio/sap-system-service-provider#explore-sap-system-functions)

2. **API filter selection in BAS:**
   - Ensure you select the **“All”** option instead of **“Classic APIs”** to locate your own RFC function module(s).
