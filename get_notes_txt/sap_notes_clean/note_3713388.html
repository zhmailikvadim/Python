Symptom

You have requested a Tenant Copy or Tenant Refresh and received the new tenant's details. External IDs for all Master Data will be pointing to the communication system from source/old tenant. You want to integrate the new tenant with another S4HANA tenant.

Environment

  • SAP Sales & Service Cloud V2 Integration with SAP S4HANA
  • SAP Sales & Service Cloud V2 Integration with SAP ERP
  • SAP Sales & Service Cloud V2 Integration with SAP S4HANA Cloud

Resolution

  1. Unlike Sales & Service Cloud Version 1, all External IDs in Sales & Service Cloud V2 are stored in the respective entity rather than a separate structure. Every External ID is linked to Communication System Technical ID(technical id).
  2. The technical ID is present in the communication system copied from the source system
  3. In the new tenant received, create a new communication system with the System ID in a specified format (SIDCLNT111) and choose the same technical id from old communication system.
  4. Delete the old communication system.
  5. Create communication configuration with new communication system.
  6. Do data load from S4HANA system and all the ID mappings will be auto updated with new communication system.
