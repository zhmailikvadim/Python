Symptom

When running Risk Analysis in the Analyze Access Risks stage for a Business Role, the Impact Analysis Criteria section appears greyed out.

Environment

SAP Access Control 10.x
SAP Access Control 12.0

Reproducing the Issue

1. Launch NWBC
2. Open the Business Role
3. Navigate to the Analyze Access Risks phase
4. The Impact Analysis Criteria section appears greyed out

Cause

An Analysis Type must be selected before proceeding.

Resolution

Please set the Analysis Type to Impact Analysis; the Impact Analysis Criteria section will then become available.
