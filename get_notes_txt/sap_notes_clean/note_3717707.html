Symptom

When performing searches via the Global Search Bar in the Pre-Production (Test) environment, results are inconsistently displayed—sometimes appearing as expected, while at other times, no results are returned.

Environment

Non-Production Environments

Reproducing the Issue

  1. Sign in to your SAP Fieldglass account on a Pre-Production (Test) Environment
  2. Type and ID for an existing record on the Global Search Bar.

Error: 'Your search -  record_id - did not match any documents. (record_id being a Work Order, Worker or any other object ID).

Cause

Due to performance constraints, starting December 2025, indexing is unavailable in Pre-Production Environments.

Resolution

Items will need to be searched using the Navigation Menu.
