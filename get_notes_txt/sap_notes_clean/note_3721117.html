Symptom

For foreign currency postings in FI, there is a difference in the tax total in a local currency of two currency units. The tax code is a reverse charge tax code with a total of 0 in the transaction currency. You would expect this in the local currencies as well. The tax code contains conditions with 100% and -100% tax.

Other Terms

BR, HWSTE, H2STE, H3STE

Reason and Prerequisites

Special tax codes with 100% conditions if there are conditions with +100% and -100% that balance to zero.

This can happen especially in Brazil.
