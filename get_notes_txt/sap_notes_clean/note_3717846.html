Symptom

This SAP Note provides instructions for enabling a new version of the Romania VAT 300 declaration tax box reporting structure in SAP S/4HANA Cloud Public Edition. The system supports Romania’s VAT changes effective from January 2026.

Other Terms

Romania, VAT 300, legal change, RO_VAT_300, RO_VAT_300_V5, DRC, Document and Reporting Compliance

Reason and Prerequisites

This update follows a legal change that takes effect from January 2026. This SAP Note is valid only for Romania.

Solution

Instructions:

A. Copy already existing tax box structure (in case you already configured it in SAP Note 3670097):

  1. Run Configuration activity: Define Enhanced Tax Box Structure (ID: 103011).
    a. Select Configure
    b. Enter RO (Romania) in the Country/Region field.
    c. Click Apply to proceed.

  2. Modify the validity of the existing tax box structure.
    a. For the row with Tax Box Structure Type = V300N, change Valid To = 31.12.2025.
    b. Select the checkbox for the row with Tax Box Structure Type = V300N and click on Expand <-> Collapse. For Tax Box Structure Type = V300N there will be two rows: 01.07.2025 - 31.12.2025 and 01.01.2026 - 31.12.9999.
    c. Select the checkbox for the row with Tax Box Structure Type = V300N, Valid From: 01.01.2026 and click on Delete. Confirm and save.

  3. Copy the existing tax box structure with new validity.
    a. Select the checkbox for the row with Tax Box Structure Type = V300N and click on Copy As.... For the newly created record, maintain Valid From: 01.01.2026 and Valid To: 31.12.9999.
    b. When prompted, select the option to copy all dependent entries.

  4. Delete the nodes that are no longer used from the new tax box structure.

  • List of obsolete nodes: R12_3_1, R12_3_2, R24_1, R24_2, R25_3_1, R25_3_2, R69_1, R69_2, R70_1, R70_2, R71_1, R71_2, R73_1, R73_2, R74_1, R74_2, R75_1, R75_2, R76_1, R76_2, R77_1, R77_2

      Note: Start from the Assignment of Tax Group to Tax Box view. Continue with Define Tax Boxes (External) and Define Tax Groups (internal) views. Save after maintaining each view.

B. Create new tax box structure from scratch (in case you did NOT configure it in SAP Note 3670097 as a part of the previous legal change):

  1. Run Configuration activity: Define Enhanced Tax Box Structure (ID: 103011).
    a. Select Configure
    b. Enter RO (Romania) in the Country/Region field.
    c. Click Apply to proceed.

  2. On each configuration activity tab, select (Menu) > Table View > Import Spreadsheet > Contents and import the respective excel sheets attached to this note in each tab.

      Note: For Define Tax Groups (Internal) tab (excel sheet no. 3), import variant 3a in case RITA is not active in the system, and variant 3b (with Reporting Country field maintained) in case RITA is active.

  3. Maintain the tax groups.

  • On the Define Tax Groups (Internal) tab, assign the correct tax codes and transaction types to the corresponding tax groups.
