Symptom

  • The Event Management API returns error code 10078 with an error message when using the endpoint GET /events/{eventId}/items/{itemId}/supplierInvitations/{invitationId}.
  • Error: Given supplier invitationId not found for the event 
  • The API fails to retrieve supplier invitation details for the specified item.

Environment

Strategic Sourcing - REST API

Reproducing the Issue

  1. Execute a GET request against the Event Management API for the endpoint /events/{eventId}/items/{itemId}/supplierInvitations/{invitationId}.
  2. The API returns the following error response:
      {
        "error": {
          "errorCode": 10078,
          "description": "",
          "message": "Given supplier invitationId not found for the event."
        }
      }

Cause

  • The error occurs due to a mismatch between the API endpoint being used and the configuration of the sourcing event.
  • In Ariba Sourcing, when all suppliers are invited to all items by default, this information is stored as a null value at the item level. Consequently, no specific invitation records exist to be queried for that item.

Resolution

  1. Use the event-level API endpoint /events/{eventId}/supplierInvitations/{invitationId} instead of the item-level API endpoint.
  2. This event-level API correctly retrieves the supplier invitation details, as invitations are managed at the overall event level when all suppliers are invited by default.
