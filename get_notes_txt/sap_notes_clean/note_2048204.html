Symptom

You execute the BAPI /SMERP/CS_DOIBASE_UPD() to update IBase information. As a result, there is an error message added to the Syclo log with the message "\CLASS=/SYCLO/CL_CORE_RFC_OO_DATA - /SMERP/CS_DOIBASE_UPD - IV_TIME Function module parameter has invalid structure".

Other Terms

ERPADDON-297, IBase,  /SMERP/CS_DOIBASE_UPD

Reason and Prerequisites

Programming error

Solution

Apply attached correction instructions.
