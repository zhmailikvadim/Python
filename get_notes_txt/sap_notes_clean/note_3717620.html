Symptom

You use transaction GM_RRB_IND and observe two symptoms.

  • The billing is happening for GM value types that are not relevant for the billing rule. The issue occurs because you have posted expenses on G/L account XXXXXX for GM Value Type YY for certain Grant. But this G/L account XXXXXX is set as billable only for another GM Value Type ZZ. There is no entry for GM Value TYPE YY in customizing view V_GMFIUPD.
  • The billing is not happening even though the billing rule is correct and the GL Account is set as 'Real Posting'.

Other Terms

View V_GMFIUPD (Maintenance View for FI Update in GM)

Reason and Prerequisites

Program error

Solution

After applying the following correction instructions, the GL Account defined as 'Real Posting' will always be billable even if you unset the indicator 'Billable'. This indicator is only valid for GL Account defined as statistical posting.
