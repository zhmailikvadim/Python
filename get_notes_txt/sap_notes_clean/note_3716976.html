Symptom

  • An error occurs when using BAPI_ASSET_ACQUISITION_POST to post asset acquisitions. The document is created for the leading depreciation area, but no posting is created for the parallel depreciation area xx.
  • The BAPI returns the error message: "AAPO Type: E Number: 704 - Depreciation area xx is a parallel area; it cannot be selected."

Environment

  • FIN Asset Accounting
  • SAP S/4HANA
  • SAP S/4HANA Finance
  • SAP S/4HANA Cloud Private Edition

Resolution

  1. Verify the customizing settings for depreciation area 22 in transaction OABT (or OABD).
  2. Ensure that the identical indicator for value takeover and depreciation terms takeover is set for parallel currency areas.
  3. Use the standard acquisition BAPI BAPI_ASSET_ACQUISITION_POST for updating the values in parallel area only. Enter the values in the PROPREAREAVALUES table of the BAPI, specifying 0 for the base depreciation area and the desired amount for the parallel currency area.
