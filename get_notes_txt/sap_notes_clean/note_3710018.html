Symptom

The Manage Payment Advices (F2550A) Fiori app does not display images for Excel files. This issue occurs when users attempt to view Excel files (.xls and .xlsx) within the app's viewer. 

Environment

  • SAP S/4HANA
  • SAP Fiori

Reproducing the Issue

  1. Go to Manage Payment Advices (F2550A) app
  2. Import the Excel file
  3. Open the payment advice details from awaiting confirmation tab.
  4. The details screen opens with the viewer on the right side of the window, but no image is displayed in the viewer.

Cause

The app is designed to display images for PDF files within the viewer, while Excel files such as ".xls" and ".xlsx" are not supported for in-app viewing. This design ensures technical functionality and security compliance. 

Resolution

  1. Convert the Excel file to PDF format.
  2. Open the PDF file in the Manage Payment Advices (F2550A) app to view the images.
