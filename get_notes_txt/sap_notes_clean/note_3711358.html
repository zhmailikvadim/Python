Symptom

This SAP Note provides you with corrections to the Manage Tax Items reporting activity for U.S. Sales and Use Tax, addressing the issues listed below:

  • When displaying reversal documents in the Tax Item View, the system shows positive amounts instead of negative amounts.

  • When you filter tax items by Customer (Customernumber), Supplier (Suppliernumber), Document Number (Accountingdocument) or Sender Document Number (Senderaccountingdocument), the filter works correctly in the Document Header View but does not reflects in the Tax Item View.

Other Terms

Document and Reporting Compliance, DRC US, Manage Tax Items for U.S. Sales and Use Tax, MTI, tax item view, reversal, customer, supplier, document number, sender

Solution

After implementing this SAP Note, the updates listed below are made to your system:

  • The Tax Item View displays negative amounts for reversal documents. As a result, you can correctly identify and reconcile reversal transactions.

  • Filter by fields Customer, Supplier, Document Number and Sender Document Number functions correctly in the Tax Item View. As a result, your filters are applied consistently to both Document Header View and Tax Item View.

Import the Support Package specified for your release or implement the attached (manual) correction instruction. As a general rule, SAP recommends that you implement the Support Package specified for your release. However, if you need to install a solution earlier, use the SAP Note Assistant to implement the correction instruction. You can find more information about the SAP Note Assistant in the SAP Support Portal, under SAP Note Assistant. Note that the documentation of certain objects is delivered with the corresponding Support Package only.
