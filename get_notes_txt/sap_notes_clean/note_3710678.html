Symptom

Invoice Time Withholding Tax Reporting is not supported for Bulgaria Withholding Tax Report, BG_WHT_194_203.

Other Terms

Bulgaria, BG_WHT_194_203, DRC, Document and Reporting Compliance, Bulgaria Withholding Tax, Invoice Time Withholding Tax Calculation

Reason and Prerequisites

Support for Invoice Time Withholding Tax Reporting is added in Bulgaria Withholding Tax Report, BG_WHT_194_203. Open Invoices, Open Credit memos, Cleared invoices and Cleared Credit memos having Invoice Time Withholding Tax will be included in reporting after applying this SAP Note.

Solution

This SAP Note is valid only for BG_WHT_194_203 DRC Report and country/region Bulgaria.

As a general rule, SAP recommends that you install a solution by applying a Support Package. However, if you need to install a solution earlier, use the Note Assistant to implement the correction instruction. You can find more information about the Note Assistant in SAP Service Marketplace, under service.sap.com/note-assistant.
