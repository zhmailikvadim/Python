Symptom

  • HTML notation is displayed in the program description when searching by program title.
  • When searching by program ID, the program description is displayed correctly without HTML notation.
  • When searching by word in item description, HTML tags are also displayed as text instead of expected effect.

Environment

SAP SuccessFactors Learning

Reproducing the Issue

  1. Access SAP SuccessFactors Learning >
  2. Search for the program by its program ID >
  3. Observe that the program description is displayed correctly without HTML notation >
  4. Return to the Learning home page and search for the same program by its program title >
  5. Observe that HTML notation is displayed in the program description.

Cause

Resolution

Our Learning Engineering team has finalized their work on the situation and the fix is targeted to be delivered with the 1H2026 Learning system version. Additional details are available in the following link: Product Release & Road Map Information.
