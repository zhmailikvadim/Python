Symptom

The eDocument remains stuck in the status "Sending to Interface Requested", and the error message "Error triggering message, check interface log for details" is displayed. As a result, the customer is unable to submit the eDocument.

Environment

  • SAP S/4HANA 
  • SAP Document and Reporting Compliance, Cloud edition

Reproducing the Issue

  1. Create and submit eDocument for Portugal Transport Registration in EDOC_COCKPIT.
  2. Observe that the status remains “Sending to Interface Requested”.
  3. Check logs and observe interface triggering error.

Cause

This issue occurs due to missing AIF (Application Interface Framework) values in the IMG Activity "/AIF/V_IFACT" (accessible via AIF Customizing: Interface Development > Define Structure Mappings). 

The following required entries were missing in the BC Set ZAIF_S4_EDOC_PT_TRREG_V03:

/EDOPTTRREG    11/EDOGENERIC_ACTIONX
/EDOPTTRREGR    11/EDOGENERIC_ACTIONX

Resolution

To resolve this issue, implement the updated BC Set ZAIF_S4_EDOC_PT_TRREG_V03 as described in SAP Note 3282752.

This ensures that all necessary AIF structure mapping entries are available.
