Symptom

  • User list does not appear when logging in with an email address. 
  • Email login fails to initiate the user selection list. 
  • User list popup fails to load during email login attempt. 
  • Unable to select certain accounts when logging in with an email address. 

Environment

SAP for Me

Reproducing the Issue

1. Navigate to SAP for Me

2. Attempt to log in using an email address and password

3. Observe that the user list does not appear for account selection.

Cause

The system displays only users that match the email address's password used during login. Accounts with different email addresses or passwords are excluded from the user selection list.

Resolution

  1. Verify that the linked accounts have the same email address and password as the SAP ID Service.
  2. Make sure all the users listed in Your Accounts list in SAP ID Service using the same password.

  3. If not, change the password as per KBA 1808560.

See Also

KBA 3708029 - Linked account(s) missing from the UID selection list - SAP Universal ID (UID)
