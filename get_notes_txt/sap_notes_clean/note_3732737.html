Symptom

KPI Cards are not available to be selected under mobile administration settings. 

Environment

SAP Sales and Service Cloud V2

Reproducing the Issue

  1. Login to the system
  2. Click on profile icon in the top right corner
  3. Click on system settings
  4. Open All Settings > Mobile Administration.
  5. Navigate to the home page.
  6. Observe that KPI cards are missing.

Cause

The KPI cards were not enabled as Mobile Relevant in the system settings.

Resolution

  1. Log in as Administrator.
  2. Go to System Settings.
  3. Search for "Home Page".
  4. Select System Preferences -> Home Page.
  5. Open the Mobile Relevant Cards tab.
  6. Under the KPI Cards section, set the KPI cards to be used in the mobile apps as Enable ON.
  7. Press Save.
  8. Go to System Settings.
  9. Search for "Home Page".
  10. Select Mobile Administration -> Home Page.
  11. Open the Manage Home Page Cards tab.
  12. Press the + (Add) button.
  13. Select KPI Card.
  14. Mark the KPI cards to add.
  15. Press Add.
