Symptom

This note generates the supplementary objects of SAP Note 3713526.

Version information

Version 1: Released for Customer

Other Terms

SAP Note 3713526, RPUUDOH0_3714640

Reason and Prerequisites

Supplementary objects of SAP Note 3713526

Solution

Implementation:

1. Apply the Correction Instructions of the SAP Note 3713526 with the Note Assistant (transaction SNOTE; more information on service.sap.com/note-assistant)

2. Apply the Correction Instructions of the SAP Note 3714640 with the Note Assistant (transaction SNOTE; more information on service.sap.com/note-assistant)

  • Run the newly created RPUUDOH0_3714640 report in "Update & Activate" mode. 

The modifications will also be delivered with the HR-CEE/SAP_HR/S4HCM SP03/2026 package.
