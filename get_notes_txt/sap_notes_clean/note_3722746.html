Symptom

Formula calculation for Inspection Characteristics, Checklists, etc. may not work as expected. Per arithmetic rules, addition/subtraction and multiplication/division are to be evaluated in left-to-right order. This note corrects the computation so it behaves in this way.

Other Terms

  • Formula Calculation
  • Inspection Characteristics
  • Checklists

Reason and Prerequisites

  •  BAS/Visual Studio Code with SAM MDK Project
  • `patch` command (available on BAS and macOS by default) via terminal or other patching utility
  • Read the guidelines on how to apply the patch and read the appendix section if you receive any error

Solution

Note: if implementing SAP Note 3625668, there may be merge conflicts. These will have to be manually resolved.
