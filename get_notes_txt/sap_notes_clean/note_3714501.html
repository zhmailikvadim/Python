Symptom

PR created by MRP live on held status.
The issue only occur if I run T-code: SHDB then MD01N, if run T-code: MD01N directly then PR created successfully. 

Environment

  • SAP ERP Logistics (PP)  
  • SAP S/4 HANA, on premise
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Execute T-code: SHDB, record T-code: MD01N with the specified variant settings.
  2. Complete the recording and save the results.
  3. Check PR in T-code:MD04.

Cause

Wrong variant.

Resolution

Please ensure the following settings are selected when creating the MD01N variant in T-code: SHDB.

  • Default Size
  • Cont. after Commit
