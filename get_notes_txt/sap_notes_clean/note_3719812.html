Symptom

When currencies such as Indonesian Rupiah (IDR) are used, the amount fields are not correctly linked to their currency reference fields. As a result, the displayed amounts appear 100 times lower than expected in the Report Workbench.


This note resolves the issue by ensuring that the currency and amount fields are displayed correctly.

Other Terms

 /IRM/IPRPM, SEARCH_GRP_FIELDS_GET, SEARCH_RESULT_DATA_PREPARE, Report Workbench, Amount Fields, Currency Reference Fields.

Reason and Prerequisites

This is caused by program error.

Solution

Please implement the correction instructions attached. These corrections are only valid if you have applied the Support Packs.


Release 6.4 Base Package or Support Pack 1, 6.3 Base Package, 6.2 Base Package, and 6.1 Base Package.

Release 60E Support Pack 7.

Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object.
