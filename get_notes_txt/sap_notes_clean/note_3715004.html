Symptom

After upgrading the SAP S/4HANA Cloud Public Edition systems, a new blank column without any column header is visible in the following apps:

  • Manage Customer Master Data
  • Manage Supplier Master Data
  • Manage Business Partner Master Data

The newly introduced column has no heading or label, which causes confusion for business users and impacts usability of the list view.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

1. Access the SAP S/4HANA Cloud Public Edition system after the upgrade to version 2602.

2. Open the following standard BP Fiori apps:

  • Manage Customer Master Data
  • Manage Supplier Master Data
  • Manage Business Partner Master Data

3. Observe the presence of a new blank column without any column header in the list view of the apps.

Cause

Resolution

With the 2602 release, the table type was changed to "Grid table". In the Grid table, an extra column is introduced to display the editing status with an empty column header.

For example:

  • If a draft is created, a pencil/edit icon will be displayed in this column.
  • If no draft is created, the column will appear blank.

This behavior is by design and is not a regression issue. For more details, refer to the SAP UI5 documentation SAP UI5 documentation
