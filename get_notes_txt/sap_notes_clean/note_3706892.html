Symptom

Message fails with an error at Odata receiver adapter flow step "com.sap.gw.core.ip.exception.ODataProcessingException: Error in xml payload. Property "_xyz is invalid".  This entity may slightly differ from case to case.

However, this error log refers to an edmx file, and the error is not visible in the ODATA error log in SAP S/4HANA.

Environment

  • SAP Cloud Integration
  • SAP Integration suite

Cause

The issue could occur when the entity displayed in the error has a cardinality of 1:1 (One to One). This can differ from case to case. 

Resolution

Workaround:

  • The current logic is unable to handle empty nodes for the target entity with 1:1 cardinality.
  • If this empty XML node is removed, the parsing logic works.

Our development team is working on this gap in their development cycle.
