Symptom

When changing the periodicity in the Extended Settlement tab, the system incorrectly treats the account as an ICL account and overwrites the Payment Details for Payment Details Type 400001, resulting in cleared fields and a changed record sequence.

Other Terms

Intercompany loans, icl, payment details, extended settlements.

Reason and Prerequisites

Program Error

Solution

 Import the relevant Support Package that is specified for your release. As a manual advance correction, you can implement the attached correction instructions using transaction SNOTE.
