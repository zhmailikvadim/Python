Symptom

This SAP Note is relevant only for the country version Switzerland (CH).

It contains the following enhancements and adjustments:

  • In the case of a multiple selection of individual case numbers on the selection screen, only the first case number with the additional information "and more" will be output in the header data of the reports mentioned below in the future, instead of the confusing list of each individual case number. This adjustment is made in the programs RNWCHAMBP_TSI, RNWCHAMBP_KON, RNWCHAMBP_AMB, and RNWCHAMBP_GRP.
  • In the report RNWCHAMBP_GRP for the grouping, the tax point value returned was previously output several times (that is, for each service of the affected outpatient treatment) when the outpatient flat rate was output. In the future, this output will only take place in the line with the outpatient flat rate.
  • During the generation of patient encounters using the report RNWCHAMBP_KON, the system currently does not process cases that contain only services to be assigned. The program now also executes contact generation for such cases.
  • In future, the service rules C1 and C2 (flat rate/billability) will also take into account the new period "Per Outpatient Treatment" for controlling the billability of services within outpatient treatment. This makes it possible, for example, to charge implants in addition if they are not covered or included in the outpatient flat rate.
