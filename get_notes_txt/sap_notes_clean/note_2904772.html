Symptom

Other Terms

US4G, TSO, ORDERS, Z11, MSCONS, 13018, /US4G/CL_UBD_DATA_SUSC, GET_DATA_FOR_QUERY, DE_B_MET_MRV_COMPLAINT_MALO

Reason and Prerequisites

Program error.

Solution

Please implement the correction instructions mentioned in this Note or import higher Support Packages.
