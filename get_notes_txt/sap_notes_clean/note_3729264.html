Symptom

While generating a payment order with the Create Payments app, if a new financial institution is added to the bank chain list in the recipient item window, the system throws a short dump/error and the entry is not added.

Other Terms

GETWA_NOT_ASSIGNED, F3648, payment item

Reason and Prerequisites

During the addition of the new bank chain entry, incorrect API calls were made to the backend accessing the new key even before it was saved to the database.

Solution

Please apply the SAP note or assigned support package.
