Symptom

Joule has been integrated with a supported product.
You would like to understand whether this integration enables that product to function as a web client, allowing it to retrieve information and execute tasks from another supported product.

For example:
Joule is integrated with Microsoft 365 Copilot. Can Microsoft 365 Copilot now access and retrieve Compensation use cases from SAP SuccessFactors?

Environment

  • SAP Business Technology Platform (SAP BTP. URL: https://cockpit.btp.cloud.sap) 
  • SAP Cloud Foundry (CF)
  • Joule AI copilot

Reproducing the Issue

In this example scenario you have integrated Joule with Microsoft 365 Copilot. One of the pre-requisite is that you have atleast one SAP business tenant (Such as SAP S/4HANA Private Cloud, SAP SuccessFactors) integrated with Joule as well. More details: 
Integrating Joule with Microsoft 365 Copilot

Cause

Resolution

Yes this is supported. 
In this example scenario, you can image Microsoft 365 Copilot acting as the alternative client for the Joule Web Client. Do note that regular limitations still apply. For example SF Proxy access is not supported (this means that you cannot access another person's account and perform actions on their behalf). You can see all supported use cases here:
Joule in SAP SuccessFactors: Supported use cases
