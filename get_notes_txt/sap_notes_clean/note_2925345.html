Symptom

The default display variant for the remuneration statement (feature return value 01) for Switzerland in the feature PBCHO does not allow you to select the HRFORMS remuneration statement provided for Switzerland with SAP Note 2250119 on the selection screen for the payroll driver.

Other Terms

Payroll Switzerland

Reason and Prerequisites

Update of default settings (selection parameters for the remuneration statement)

Solution

Import the HR Support Package specified for your release.

Content of the change:

After you import the HR Support Package, the feature return value for Switzerland is changed from 01 to 11. If the feature PBCHO is not modified, the selection screens for the payroll driver for the country version Switzerland (programs RPCALCC0 and RPCALCC0_CE) also show the "Name of the HR Form" parameter in the "Remuneration Statement Parameters" frame.
