Symptom

You observe Available  Sources of supply does not show up any suppliers data while creating a New Sales Order although the product DEF(DEF refers to Product id) contain valid list price.

Environment

SAP Business ByDesign

Reproducing the Issue

  1. Go to Sales Order Work Center
  2. In Common task,Select New Sales Order
  3. Fill in all entries like Account, Product id DEF, Quantity etc.
  4. Select Assign Source of supply by selecting the line item DEF

You observe that available sources of supply does not pull up any suppliers data although there is a valid list price.

  1. Go to Product Portfolio Work Center
  2. Navigate to List Price View
  3. Search for DEF

You observe that there is a valid list price.

Cause

Requested date in Sales Order is beyond the List price Valid to Date, System is not able to find any  sources of supply for the given requested date, Hence system is not able to determine any sources of supply.

Resolution

We can resolve the issue in 2 different ways

  1. Maintain the requested date in Sales Order within the timeframe of List Price Valid to or
  2. Extend the Valid to date of List Price to a future date for the Product DEF 

Keywords

Available Sources of Supply is Missing for Sales Order, Assign source of supply does not show up any suppliers data
