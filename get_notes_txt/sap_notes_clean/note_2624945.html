Symptom

  • Analysis edition for OLAP does not display information and warning messages generated at database and middleware levels.
  • Such messages are returned when validating or submitting variable answers.

Other Terms

AOLAP, A-OLAP, prompt

Reason and Prerequisites

Message handling did not check for messages apart from variables related scenario.

Solution

A parameter in the "mdaclient.properties" file allows to configure if messages should be displayed or not: enable.query.processing.messages
This file is located in folder %WEB_APP_SERVER%\webapps\BOE\WEB-INF\config\custom\, if it doesn't exist then duplicate and customize file from folder ...\config\default\
Its potential values are true or false. Its default value is set to true.

The issue has been fixed in the patches listed in the "Support Packages & Patches" section below.
The "Support Packages & Patches" section will be populated with the relevant patch levels once they are released.
