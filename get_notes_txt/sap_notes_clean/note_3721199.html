Symptom

When performing a Solr index job, Backoffice shows the following error and the index job fails:

Environment

  • SAP Commerce Cloud
  • Solr 9.8

Reproducing the Issue

  • Navigate to Backoffice → Administration Cockpit → Facet Search Configurations.
  • Trigger an index job and the error appears.

Cause

This error is caused by a missing luceneMatchVersion configuration in the solrconfig.xml file.

Resolution

  1. Open solrconfig.xml file, and add or uncomment <luceneMatchVersion>9.11</luceneMatchVersion>.
  2. Commit the code and restart the Zookeeper and Solr pods in rolling mode.
