Symptom

You are using the DBA Cockpit for SAP Adaptive Server Enterprise (SAP ASE) database server. One of the features is the Automatic Table Maintenance (ATM). That allows to provide groups of tables to be automatically checked for the neccessity of index statistic update or table reorganisation defined by criteria provided. The evaluation and execution can be assigned to time windows set to a range of weekdays (from/to Weekday) and therein given a start- and end-time. This allows the administrator to minimize performance impact by chosing times of low system usage.

Caused by an error, the 'to' time definition is propagated to the database so that the time window will only expire at midnight.

Other Terms

Sybase, SAP Adaptive Server Enterprise, SAP ASE, Automatic Table Maintenance, ATM, Time Window, DBACOCKPIT

Reason and Prerequisites

Program Error

Solution

Import the appropriate Support Package. Alternatively, implement the correction instructions that are attached to this SAP Note.
