Symptom

After executing ATC in central check scenario, there is a failure in job log. 

Failure: Check 'XXX' is not remote RFC-based
Check 'XXX' is not remote RFC-based
Include YYY; Line 11
Project could not be scheduled
Job canceled after system exception ERROR_MESSAG

Environment

SAP S/4HANA

Resolution

* Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.    

  1. Confirm the check variant name by ATC>Basic setting
  2. Go to SCI, input variant name and display. 
  3. Review all selected check class. 
  4. There might be some class that do not support RFC-based without green arrow.

         

      5. Consider to use the correct variant for different ATC check scenario and different purpose.
