Symptom

  • When posting fixed asset investment support via BAPI_ASSET_INV_SUPPORT_POST, the system raises error AAPO167: “Data inconsistency: There is no remaining depreciation area to be posted”.

Environment

  • FIN Asset Accounting
  • SAP R/3
  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA Finance
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Open transaction SE37.
  2. Run function module BAPI_ASSET_INV_SUPPORT_POST to post investment support

Cause

This error is introduced by SAP Note 2581502, attempting to post transactions (for example, acquisition or retirement) without an amount via BAPIs leads to error AAPO167 (“Data inconsistency: There is no remaining depreciation area to be posted”).

Resolution

When calling BAPI_ASSET_INV_SUPPORT_POST to post investment support, under table INVESTMENTSUPPORTDATA, do not leave the amount empty or zero.

See Also

SAP Note: 2581502 - BAPIs allow posting with zero amount
