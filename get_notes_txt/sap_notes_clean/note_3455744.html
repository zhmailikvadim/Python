Symptom

When a user tries to create Story report on Employee Profile Background element can see below error.
 
Get story query error: We found some issues in the query validation. For details, use the validation information in the toolbar.
Correlation ID: ......

40117    :    'xxx':40117:Repository: Encountered an error in repository runtime extension;Model inconsistency. Unsupported or missing view for data source 'xxx'

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Environment

SAP SuccessFactors People Analytics

  • Story report

Reproducing the Issue

1. Navigate to Report center

2. Create a new story, select the table 'A-bcd' in query. Column id is like '......_sac#......_A-bcd#......'

3. Click query preview then will get below error

We found some issues in the query validation. For details, use the validation information in the toolbar.
Correlation ID: ......

40117    :    '......_A-bcd':40117:Repository: Encountered an error in repository runtime extension;Model inconsistency. Unsupported or missing view for data source '......_SAC_......_A-bcd'

Cause

SAP HANA doesn't allow special characters (eg: '-') in the table/view names. The table/view name is calculated based on the original object Id, if it has '-', calc view creation will fail in the background.    

Resolution

With this HANA limitation please do not use Special characters (eg: '-') in the object Id.
