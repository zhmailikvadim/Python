Symptom

  • Would like to know which SLT tables are mandatory for standard CAR POS DTA functionality.
  • Only utilizing the POS DTA scenario within CARAB.

Environment

SAP Customer Activity Repository

Cause

Would only like to use POS DTA within CAR.

Resolution

In order to use CAR POS DTA functionality and none of the other CARAB scenarios, there is a list of tables outlined in the CARAB User Guide for this:

https://help.sap.com/docs/CARAB/e95c8443f589486bbfec99331049704a/6047b252c89a4d27e10000000a441470.html?version=LATEST&locale=en-US

Master Data Check/Enrichment

Main SAP ERP Tables

Article (Material) Number and Description

MARA, MAKT

Article EAN/GTIN Numbers

MEAN

Article (Material) Store (Plant)

MARC

Units of Measure for Article

MARM, T006, T006A, T006D, T006T

Store (Plant) Number and Description

T001W

Distribution Channel

TVTW, TVTWT

Merchandise Categories (Material Groups)

T023, T023T

Sales Organizations

TVKO, TVKOT

Company Codes

T001

 

Furthermore, a number of PIPE tasks require additional SAP ERP master data:

Tasks

Main SAP ERP Tables

Supply - DMF-Based Applications

KONBBYH, KONBBYPRQ, KONBBYT, KONDN, WAKH, WAKP, WAKR, WRF_MATGRP_HIER, WRF_MATGRP_HIERT

Supply WFM

KLAH, KSSK, SWOR, TCLA

Distribution

WLK2

If plan to analyze Multichannel Sales data or Inventory data, then additional tables will need to be replicated as well, also outline in the link provided above.

The data for other tables that are not mentioned in this list will not be required, only the table and table structure will need to be present to ensure that the HANA views are able to be activated successfully. 

Make note - The table/table structure from the following SLT table list still needs to be replicated:

https://help.sap.com/doc/33074f869a0644579c817d4fe99680a4/6.0.0/en-US

These tables are replicated to either SAP_ECC or SAP_S4H schemas, or the physical schema mapped to either of these.
