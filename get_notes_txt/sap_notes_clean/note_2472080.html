Symptom

  • You have integrated BEx query within your Fiori Launchpad.
  • While clicking the tile, user receives an Error: "Service cannot be reached, HTTP 404 - Not found"

Environment

  • SAP Fiori
  • SAP BW

Cause

A 404 GET error suggests that the request has been sent with an invalid data key, but does not exist.

Resolution

To fix the issue;

  1. Identify and confirm the exact OData service caused the 404 error. This can be done by either checking in the browser developer tool, and going to the Network Tab. Or from the onscreen error message.
  2. You will then have to check in your system, if the service is pointing to the correct backend system where the OData is located, (using tcode /IWFND/MAINT_SERVICE) then you need to check if the user has the authorization assigned to access this service in both Frontend and Backend System.
