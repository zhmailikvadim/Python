Symptom

Following fields are missing for configuration "Sensitive Fields for Activated Bank Accounts" in SSCUI Define Settings for Bank Account Master Data:

  • CASH_POOL_ID
  • CASH_POOL_TARGET_AMT_MIN
  • REVISION
  • DELETED_FLAG
  • REVISION

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Navigate to SSCUI Define Settings for Bank Account Master Data (ID: 102336).
  2. Open the configuration for “Sensitive Fields for Activated Bank Accounts”.
  3. Observe that several previously available fields are not listed.
  4. Attempting to add these fields manually in SSCUI 102336 but result in an error "Field XXX does not belong to object XXX. Select one using input help".

Cause

The affected fields were intentionally removed as part of the CE2402 update. Older tenants may still display these fields without functional impact, while newer tenants do not include them by design.

Resolution

Do not attempt to reintroduce the removed fields for configuration "Sensitive Fields for Activated Bank Accounts" in SSCUI Define Settings for Bank Account Master (ID: 102336), the absence is by design and they have no functional impact.
