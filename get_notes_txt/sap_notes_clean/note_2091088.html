Symptom

If the default value of interval variable is defined as EQUAL in query designer, the variable cannot be shown in new report wizard when user creates report or input form.

Other Terms

new input form wizard, inputform, new inputform wizard, range variable

Reason and Prerequisites

Program error

Solution

For SAP Business Planning and Consolidation 10.1, version for SAP Netweaver, upgrade to BW740(Component SAP_BW) support package 10.
