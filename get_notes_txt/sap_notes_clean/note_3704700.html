Symptom

For the cost originating from material ledger settlement , the material number is missing in the FI document after settlement.

Environment

  • SAP R/3
  • SAP R/3 Enterprise 
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA Finance
  • SAP S/4HANA

Reproducing the Issue

  1. Execute transaction KO88.
  2. Check Material document from FI document.

Cause

  • Material ledger does not send the CO origin (field HRKFT) data to CO.
  • The absence of CO origin in the original postings prevents the material number from being displayed in the settlement follow-up documents.
  • This is the current standard system behavior.

Resolution

In general, material number needs to be encoded in CO Origin (field HRKFT) in the original postings, so that it can be used by settlement and saved in the settlement follow-up documents. If CO origin (field HRKFT) is blank, no material will be displayed in the settlement follow-up document. This is the current standard behavior, and there is no exit available to change it at this time.
