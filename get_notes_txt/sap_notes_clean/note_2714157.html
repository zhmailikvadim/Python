Symptom

The data collection run with transaction DTINF_START_COLL takes a very long time for delivery documents (ILM object RV_LIKP).

Solution

The correction is delivered via support package. Parts of it can also be applied via correction instruction.

The existing purposes for ILM object RV_LIKP need to updated with transaction DTINF_ADJUST_MODEL before executing a new data collection run.

Other Terms

SRM Shopping Cart, Catalog Extractor, Product Relationships, MDM Catalog, Language

 Program Error

Solution

Implement Correction Instruction
