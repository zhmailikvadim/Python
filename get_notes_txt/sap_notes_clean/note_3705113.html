Symptom

For logistics requests without any assigned countries the "Business Process / Area" column of the "Requests" table within the "Process Requests from Logistics" app (Fiori ID F3342) erroneously displays the countries of the preceding row.

Reason and Prerequisites

Reason:

Program error.

Prerequisites:

For information about the validity of the corrections, see the correction instructions.

Solution

The "Support Packages" section lists the Support Packages that contain these corrections.

Alternatively, you can implement the attached correction instructions.
