Reproducing the Issue

N/A

Cause

This is caused by a virus-scanner moving a file on the Task Engine server, during the creation process.

The Disclosure Management Task Engine process, creates a number of temporary folders and file in location: C:\ProgramData\SAP\Input

If these files are scanned / moved by a anti-virus program, you will see this error message

Resolution

You need to exclude this folder from the virus scanner

E.g. in Window Defender, https://www.thewindowsclub.com/exclude-a-folder-from-windows-security-scan

NOTE: this can vary for different tools that are used
