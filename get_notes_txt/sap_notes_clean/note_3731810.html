Symptom

Component: Foundation for EHS Management
Module: Tasks

When you set the status to Central Block for Business Partners in transaction BP, the system saves centrally blocked Business Partners as task assignees, task owners, or task approvers. This occurs when you enter the Business Partner ID in the input field for task assignees, task owners, or task approvers.

Other Terms

IF_EHFND_BUSINESSUSER_PROXY, CL_EHFND_BUSINESSUSER_PROXY, CL_EHFND_TDEF_INST_A_UPDATE, CL_EHFND_TDEF_V_BASE

Reason and Prerequisites

An error in the system causes the symptom.

The relevant correction instructions contain the prerequisites.

Solution

After implementing this SAP Note, the system no longer allows you to enter the ID of centrally blocked Business Partners as task assignees, task owners, or task approvers.

Alternatively, implement the corresponding Support Package for the component extension of SAP Environment, Health, and Safety Management or the Feature Package Stack for the SAP S/4HANA system.
