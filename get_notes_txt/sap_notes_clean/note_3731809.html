Symptom

Processing of BAPIs (i.e. BAPI_SALESORDER_CHANGE, BAPI_SALESORDER_CREATEFROMDAT2, BAPI_QUOTATION_CREATEFROMDATA2) fails with error CFG_IS_INCONSISTENT_OR_INCOMPLETE, message no. CU013. 

Environment

Variant configuration

Resolution

The following Knowledge Base Articles can help explain the cause of CFG_IS_INCONSISTENT_OR_INCOMPLETE and suggest steps or workarounds for a different system response:

  • 1453002 CU013 in IDOC or BAPI process - CFG_IS_INCONSISTENT_OR_INCOMPLETE: instructions for correctly supplying configuration data to BAPI structures.

  • 3729964 Tool for tracing errors in Variant Configuration: transaction CUTRACE records the configuration logic at runtime and may clarify the system behavior and can serve as a starting point for troubleshooting. 

  • 1931917 'Specify a valid configurable material' (M3544) issued when trying to configure a Material Variant: in some cases, object dependencies cannot complete successfully because the characteristic values are not available when needed. 

  • 399120 Order processing, ALE/IDOC with incomplete configuration: in certain situations it can make sense to permit incomplete configurations.

Keywords

CU 013, Idoc, configuration, quotation, sales order
