Symptom

In CV02N transaction a user requires to make changes on a Document Info Record (DIR) but an error is raised: "Document & is locked by & - Message no. 26045 

Environment

  • SAP Document Management System (CA-DMS)
  • SAP Enhancement package for SAP ERP
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA on-premise
  • SAP S/4HANA Cloud Private Edition 

Reproducing the Issue

  1. Transaction CV02N 
  2. Enter Document Info Record (DIR)
  3. Error is raised: "Document & is locked by &" - Message no. 26045

Cause

Resolution

You may consider 2 options to resolve the error:

  1. Contact the user stated in the error and request the user to exit the Document Info Record (DIR)
  2. Alternatively access the SM12 transaction and perform the following steps:

a. Enter the following information:

  • Client: Should be automatically displayed
  • User Name: 'Enter user id displayed in the error'

b. Click 'Search'

c. Click on 'List', select all the entries, and click the 'delete' button icon (Delete Selected Locks)

Keywords

CV02N, CV04N, Document locked, Message 26045, SM12, Delete locks, DIR locked, DMS
