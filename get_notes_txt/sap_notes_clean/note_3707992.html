Symptom

In transaction WB2R_SC, Activity Reason is not displaying anything in the dropdown when language = IT but correctly displayed when language = EN.

Environment

Condition Contract Settlement in SAP S/4HANA

Reproducing the Issue

Aside from confirming in transaction WB2R_SC, this can also be checked in transaction SE11 with difference logon language -> WBRK -> double click on LFGRU -> Search Help Name: WLF_SH_ACTIVITY_REASON.

Cause

Missing configuration.

Resolution

Maintain the Activity Reason entries in Italian using the following IMG path:
WALF: Gestione della liquidazione >> Impostazioni di base >> Definire motivi dell’attività

Keywords

WB2R_SC, WBRK, LFGRU, IT, Italian, Activity Reason, language, WLF_SH_ACTIVITY_REASON
