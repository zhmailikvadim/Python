Symptom

In the process "Start of Basic or Backup Supply for Electricity (DSO View)" (DE_B_GPKE_SBS_SUP), some step reads UBD data from step 200 without considering the status of the step before reading. This results in incorrect step data being processed.

This issue is relevant for the market role: DSO.

Other Terms

SBS, DE_B_GPKE_SBS_SUP, UBD, POD, step 200, Basic Supply, Backup Supply, DSO View, process cluster, supply scenario

Reason and Prerequisites

 The error occurred because some steps reads data from step 200 without checking the status of the step, leading to incorrect data being processed in the system.

Solution

 Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the system reads POD data correctly from step 200 in the process. You need to retrigger the process. You can verify the fix by running the SBS scenario and checking that all steps process the correct data.
