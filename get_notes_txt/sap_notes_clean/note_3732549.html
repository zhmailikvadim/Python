Symptom

Short Dump raised in the system: Function module "CON_FIN_ACC_INTERFACE_PROJECT" not found.

Environment

IS-CWM

Reproducing the Issue

Category               ABAP programming error
Runtime Errors         CALL_FUNCTION_NOT_FOUND
Except.                CX_SY_DYN_CALL_ILLEGAL_FUNC
ABAP: Program          SAPLRWCL

Cause

the function module CON_FIN_ACC_INTERFACE_PROJECT is part of the old architecture of the industry solution Catch Weight Management. It is not relevant for your implementation.

The module is registered in Table TRWPR and the check on it can be switched off in table TRWCA.

Resolution

  1. Please start transaction SM30 for Table/View TRWCA and click ‘Maintain’.
  2. You will get the overview about FI/CO Interfaces and the information which of them are active. 
  3. Search for the entry ‘INV’.
  4. Deselect the sign ‘Active’ for the component ‘INV’ and safe the change.

Afterwards modules from component ‘INV’ won’t be considered during the billing process. The program dump won’t occur any longer.
