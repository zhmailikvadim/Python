Symptom

This SAP Note provides guidelines for implementing the reconciliation of India GST reports using SAP Document and Reporting Compliance. The following requirements are necessary when using the Reconcile Draft Reports (DRR) application:

  • The user is unable to download the generated Authority File and Source File.

Other Terms

Document Compliance, DC, Document and Reporting Compliance, Reconcile Draft Reports, DRR, India GST, Source File, Authority File, Souce File Download, Authority File Download

Reason and Prerequisites

You want to implement, configure, and use the Reconcile Draft Reports for India GST.

Prerequisites

  • SAP Note 3482143 - DRC India: Reconcile Draft Reports: The details of the application for India GST Reports named Reconcile Draft Reports
  • SAP Note 3645575 - DRC India: Reconcile Draft Reports: Updates in the Reconcile Draft Reports application
  • SAP Note 3668971 - DRC India: Reconcile Draft Reports: Updates in the Reconcile Draft Reports application

Solution

After implementing this SAP Note, you can download and view the generated Authority File and Source File.

As a general rule, SAP recommends that you implement the support package specified for your release. However, if you need to install a solution earlier, use the SAP Note Assistant to implement the correction instruction. You can find more information about the SAP Note Assistant in the SAP Support Portal, under SAP Note Assistant.
