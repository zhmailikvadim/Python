Symptom

Translatable fields are displayed in the Change History section of position object even if there were no changes done to the fields.

Environment

SAP SuccessFactors Employee Central: Position Management

Reproducing the Issue

  1. Login to instance.
  2. Navigate to Manage Positions.
  3. Open any position.
  4. Check the Change History section on the left.
  5. There are some translatable fields displayed even if there are no specific changes done to the fields.

Cause

Translations are not maintained for the affected field in the logged in user's locale. 

Resolution

Maintain the translation for the field in the logged in user's locale. For example, If logged in user's locale is en_US, then please ensure a translation is maintained in en_US for the affected field.
