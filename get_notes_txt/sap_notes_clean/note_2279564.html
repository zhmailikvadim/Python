Symptom

Member selector doesn't have static ID when the user changes work status.

Reproducible steps:
1. Log on to the BPC Web Client ->  Go to Embedded environment -> Go to "Library" -> Go to "Public" in tree -> Open report for changing work status ->
2. Click "Change Work Status" button in toolbar-> Click some dimension for changing status -> Check ID of the member selector by browser debugging tool -> Check Symptom.

Actual result: Member Selector doesn't have an ID value when the user changes the work status. So, it needs a fix for automation testing.

Expected result: Member Selector should have static ID for automation test.

Other Terms

Member Selector, Work Status, Embedded environment

Reason and Prerequisites

Program error

Solution

For SAP Business Planning and Consolidation 10.1, version for SAP NetWeaver on 740, upgrade to BW740(Component SAP_BW) Support Package 15.

For SAP Business Planning and Consolidation 10.1, version for SAP NetWeaver on 750, upgrade to BW750(Component SAP_BW) Support Package 04.
