Symptom

In the standard system as currently delivered, when you execute the Tax Reporter (PU19) report for the Form W-2 PR, the Date of Birth for the employee is not being printed in the electronic file.

Other Terms

Date of Birth, Configuration Check, PU19, SM34, SFP, W-2, W2, PR, Puerto Rico, DYEA, DMON, DDAY, Magmedia, TEMSE, Phase II, HR_F_MMREF_S_PR, HR_F_MMREF_1_PR.

Reason and Prerequisites

You have applied SAP Note 2360638 - Year End 2016 Phase I for U.S. Tax Reporter

Solution

As a general rule, SAP recommends that you install a solution by applying a Support Package. However, if you need to install the solution earlier, use Note Assistant (SNOTE) to implement the manual correction instructions.

You can find more information about the Note Assistant in SAP Service Marketplace, under service.sap.com/note-assistant.
