Symptom

Security deal is created, and payments details are populated automatically. The direction depends on the sign of flows existing in the deal. If only one flow direction is existing ("+" or "-"), then only 1 payment line is inserted with the corresponding direction. It is required that both directions are populated.

Other Terms

FTR_CREATE VTBZV

Solution

Preliminary correction via Correction Instruction or implementation of the corresponding Support Package.

This SAP Note is a soft modification. After implementing the note, you have to activate it.

How to activate the soft modification:

Note the usage instructions in SAP Note 732499.
Activate this SAP Note (3714786) in the method IF_JBR_SOFT_MODIFICATIONS~IS_ACTIVE of the class ZCL_SOFT_MODIFICATIONS_CUST using the CASE menu bar (see the section entitled 'Activate a new note' in delivery Note 732499). For example

CLEAR r_x_active.
CASE i_note_id.
WHEN 3714786.
r_x_active = 'X'.
ENDCASE.

You can use transaction TEST_SOFT_MOD to test whether the modification is active for this SAP Note (enter the note number and choose ENTER).
