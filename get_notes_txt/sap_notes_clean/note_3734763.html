Symptom

The issue only affetcs PTC Creo family tables.

A possible crash of PTC Creo occurs in the Import process with PTC Creo Version 11.0.0.0 - 11.0.5.0 and 12.4.0.0.

The issue was observed in PTC Creo during the Import process with SAP ECTR interface to PTC Creo.

Please note that the issue does not occur in general, it depends on how the family table of a Generic Assembly was created with internal control flags and dependencies.

Unfortunately we have no deeper detailed information about that.

Environment

FromToSolved in PTC Creo

Cause

The issue occurs in PTC Creo during the SAP ECTR Import process with SAP ECTR interface to PTC Creo.

Resolution

To solve the issue use the following PTC Creo versions

>= PTC Creo 11.0.6.0 
>= PTC Creo 12.4.1.0

Please note that this is no error of the SAP ECTR interface to PTC Creo.
