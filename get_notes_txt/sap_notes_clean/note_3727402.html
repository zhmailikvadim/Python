Symptom

 When posting Price Review using the program /IRM/IPPD_POST in background mode, the system generates processing messages during execution. However, these messages are not written to the SM37 job log, This note resolves the issue by updating the message handling logic so that all relevant processing messages generated during the background execution of the program.
