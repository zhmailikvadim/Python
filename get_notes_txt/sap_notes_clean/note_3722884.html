Symptom

In phase PREP_EXTRACT/SUMASK, the Software Update Manager (SUM) does not accept the user passwords of HANA, and asks for "Database Security User"

=========================================================

User "DDIC" in Client 000
Password: ****************************************

Database Security User


The database "SYSTEM" user is deactivated.
Enter a valid database user which has the "USER ADMIN"-privilege.


DB Security User Name: xxxxx
Password: ***************

Database User "SYSTEM"
Password: ***************

Database User "SAPABAP1"
Password: ***************
=========================================================

in log HDB_VALIDATE_PASSWORD.LOG, below error message is recorded

EXECUTING /<path>/SUM/abap/../jvm/jre/bin/java -cp /<path>/SUM/abap/bin/com.sap.hdb.sl.lib.jar:/<path>/SUM/abap/bin/com.sap.hdb.core.jar com.sap.hdb.core.main.cmd.HdbCmdMain HDB_CMD_CLAZZ_NAME=IS_CONNECT_POSSIBLE DATABASE_SID=SID DATABASE_HOSTNAME=<Host> INSTANCE_NUMBER=00 DATABASE_SQL_USER_NAME=SAPHANADB READ_FROM_STDIN=TRUE JDBC_PATH=/<path>/SUM/abap/bin DATABASE_MODE=MULTIPLE SYSTEMDB_PORT=30013 DATABASE_USER_ADMIN_NAME=HANAADMIN DATABASE_USER_ADMIN_PASSWORD=
Thu Feb 26 09:00:10 GMT+01:00 2026 | com.sap.hdb.core.main.cmd.HdbCmdMain | DATABASE_USER_ADMIN_PASSWORD=, Wrong format. Arguments must have the format key=value!

Environment

SAP S/4HANA, SAP ERP, SAP Netweaver

Reproducing the Issue

  1. Upgrade SAP S/4HANA system using SUM 2.0
  2. Enter passwords for HANA DB during SUM phase PREP_EXTRACT/SUMASK.

Cause

SYSTEM user is locked or deactivated in HANA DB. Another valid database user which has the "USER ADMIN" privilege is required.

Resolution

  • Activate the database user SYSTEM, then restart SUM upgrade
  • In case the database user SYSTEM cannot be activated at moment, it is necessary to prepare another valid database user which has the "USER ADMIN" privilege.

Refer to the instruction in SUM guide section Password of User DDIC and Database Users (Phase SUMASK)

-------------------------------------------------------------------------------------------------------------------------------------------

Enter in addition the SAP HANA database-specific passwords for the database user and for the database user SYSTEM.

Moreover, the Software Update Manager asks for the name and the password of the SAP HANA database security user. However, this dialog appears only when the database user SYSTEM is deactivated. In this case, the Software Update Manager asks for a security user that has the system privilege USER ADMIN. It uses this security user to activate the user SYSTEM if necessary. Then, the user SYSTEM is used for a short time and deactivated again afterwards.
