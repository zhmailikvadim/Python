Symptom

When I try to create a new user, the system displays a message indicating that the username already exists, even though I know there is no user with the same username in my company.

Cause

SAP Fieldglass validates usernames globally. When a username is entered during user creation, the system checks the entire SAP Fieldglass database — across all buyers, suppliers, and workers — to ensure the username is unique.

If the username already exists in any other Fieldglass account, even if it is not related to your company, the system will prevent the creation.

Resolution

Use a different username that is not already in use in the SAP Fieldglass global database.
