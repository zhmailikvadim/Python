Symptom

  • Gross sales have been maintained for the reporting rule
  • The gross sales are not used for calculation of the sales by the sales-based settlement

Environment

  • Flexible Real Estate Management (RE-FX)
  • Contract and Lease Management
  • SAP ERP 
  • SAP ERP Central Component
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA Finance
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Maintain sales for the reporting rule using "Gross Sales"
  2. Execute the sales-based settlement
  3. Observe that the sales-based settlement report in transaction RESRIS displays sales as 0 

Cause

  • The reporting rule is configured to use gross sales, but the sales rule is incorrectly defined with the "Gross/Net" indicator set to "Net", causing the system to use net sales instead.

Resolution

  1. Update the settings for the sales rule to use "Gross" instead of "Net".
  2. Reverse any settlements that have already been processed.
  3. Resettle the contracts after adjusting the sales rule 

Keywords

RESRIS, gross sales, net sales, sales-based rent, reporting rule, sales rule, gross/net indicator,  sales-based settlement, RESRSE
