Symptom

When opening the ownership manager via BPC web client, the ownership data fails to load with the error message:
"Failed to get ownership data
  503 service not available"

Environment

  • SAP BusinessObjects Planning and Consolidation, version for SAP NetWeaver
  • SAP BusinessObjects Planning and Consolidation, version for SAP BW/4HANA

Reproducing the Issue

  1. Logon to BPC web client
  2. Open the ownership manager, it fails with error message:
    "Failed to get ownership data
      503 service not available"

Cause

Connected to the wrong server that isn't listed in SM51.

Resolution

Involve your basis team to connect to the right server.

Keywords

SAP BPC , Ownership data , Failed to get ownership data , 503 service not available , BPC web , wrong server
