Symptom

In FINS_MIG_STATUS, M20 step, you can see many errors FINS_ML_MIG 013. 
These errors are false positive as MBEW against MBVMBEW check for certain material is equal.

Other Terms

M20

Reason and Prerequisites

The issue is side effect caused by implementing of the SAP Note 3677772
(3677772 can be already part of S4CORE service package, so it will not be visible in SNOTE)

Solution

- Implement this SAP Note and rerun M20 step.
- If rerun is not possible, check for correctness using report FINS_MASS_DATA_TEST.
