Symptom

If no device information can be determined when you call transaction /ENBI4/BILL_ANALYSE, the system issues a corresponding message. If you then execute a refresh of the displayed tree structure using an action, the system issues the same error message again, but this time as a dialog box, which leads to confusion. Since the message has already been issued once, the new output of the message should be suppressed.

Reason and Prerequisites

Program error

Solution

You can implement this SAP Note in your system either manually or by importing the relevant Support Package.
