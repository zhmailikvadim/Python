Symptom

In the processes "Start of Supply for Electricity (DSO View)" (DE_B_GPKE_DSO_SOS_REG) and "Start of Supply for Electricity (DSO View)" (DE_C_GPKE_DSO_SOS_REG), the dormant market location scenario was not handled. Steps 1670, 1680, 0910, 0920, 0930, and 0940 has been added to cater the Dormant Market Location functionality.

This issue is relevant for the market role: DSO.

Other Terms

GPKE, DE_B_GPKE_DSO_SOS_REG, DE_C_GPKE_DSO_SOS_REG, BM_GPKE_SOS_DORMANT_REQ, BM_GPKE_SOS_DORMANT_RESP, AGGR_POD_ID, UBD POD structure, Work Package 1, Dormant Market Location

Reason and Prerequisites

 The error occurred because the handling of dormant market location data was missing in the start of supply for electricity process in both Cloud and S/4 systems.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the system handles dormant market location data as part of Work Package 1 in the start of supply for electricity process in both Cloud and S/4 systems. The field AGGR_POD_ID is added as part of the UBD POD structure. You need to retrigger the start of supply for electricity process for the dormant market location scenario. You can verify the fix by checking that the start of supply for electricity process is handled successfully for the dormant market location scenario.
