Environment

  • SAP HANA Cloud Database

Cause

  • Instance mapping is only available for instances hosted in the same BTP region.

Resolution

  • Please check Cloud Foundry API endpoints for both source and target subaccounts
    i.e
    source API Endpoint:https://api.cf.eu10.hana.ondemand.com

    Target API Endpoint:https://api.cf.eu20.hana.ondemand.com

    eu10 != eu20 (different BTP region)
