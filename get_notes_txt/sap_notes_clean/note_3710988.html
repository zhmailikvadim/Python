Symptom

The following options are not visible in the Users - Setup window for non-superusers:

  • ... button to change the password
  • Password Never Expires
  • Change Password at Next Logon
  • Locked

Environment

SAP Business One

Reproducing the Issue

Scenario:

  1. Create a new non-superuser and grant full authorization for all modules.
  2. Log in as this newly created user.
  3. Navigate to the Users - Setup window and locate the user.

Actual behavior: 

  1. The Password section on the General tab is not visible.
  2. The following fields are not displayed: Password Never Expires, Change Password at Next Logon, Locked.
  3. The ... button to open and change the password is also not available.

Resolution

  •  This is the correct system behavior.
  • Only superusers are permitted to manage password-related settings for other users.
  • Non-superusers can change their own passwords by navigating to Administration > Setup > General > Security > Change Password.
