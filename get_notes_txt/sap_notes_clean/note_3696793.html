Symptom

Procurement documents created in S4/HANA are successfully posted in SAP Business Network and subsequent postings for follow-on documents is also successful in S/4HANA based on incoming messages from SAP Business Network to S/4HANA via Managed Gateway.

However, status notification of these postings for Order Confirmation and Ship Notice from S4/HANA to Managed Gateway is failing with status in Transaction Tracker as ‘DUMPED’.

Environment

SAP S/4HANA Cloud Public Edition, SAP Integration Suite Managed Gateway

Cause

The technical status outbound triggered from SAP S/4HANA to Managed Gateway, for internal status check and two-way match within Managed Gateway, is failing at Managed Gateway due to missing message body for the API BusinessNetworkStatusUpdateNotification_Out.

Resolution

This record with DUMPED status in Managed Gateway Transaction Tracker is internal to Managed Gateway and does not impact any transactional documents or their status whatsoever. Hence the record with ‘DUMPED’ status can be ignored.

SAP will include the message body in the affected API in one of the future Public Cloud Releases, so that records with ‘DUMPED’ status no longer appear in Managed Gateway Transaction Tracker.

Keywords

CIG, EDI, Transaction Tracker, status notification
