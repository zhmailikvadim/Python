Symptom

In the Fiori application Manage Time Deposit, when adding a Reference Account for an account, the value help (F4 search help) is not displayed for the Routing ID field.

Other Terms

BSP Apploication - FS_TD_MANS1,Time Deposit Management, Package - FS_CB_TIMEDEPOSIT_MANAGE, Component CA

Reason and Prerequisites

The issue was caused by a program error. It has been corrected so that the F4 search help for the Routing ID field is now available when adding a Reference Account.

Solution

Import the relevant Support Package for your release. You can use transaction SNOTE to implement the correction instructions relevant for your release in advance.
