Symptom

Payment item has validation error during OPO processing in advanced statement forwarding scenario. Status tag is missing in camt format.

Error message in the log "Schema Validation Error: Missing Tag Sts (XML Validation ID sequence / Path  Document/BkToCstmrStmt/Stmt/Ntry)".

Other Terms

Advanced bank statement forwarding,
CAMT.053.001.08, CAMT.053.001.02, CAMT.052.001.08, CAMT.052.001.02,MT940
Document/BkToCstmrStmt/Stmt/Ntry/Sts, 
/PF1/FH_CONV_XSD005

Reason and Prerequisites

If Funds code is missing in incoming MT940 file, then Status tag is not mapped during MT940 to CAMT file format conversion. It caused validation errors.

Solution

Status tag is always populated in MT940 inbound converter.
Please apply SAP Note or assigned Support Package.
