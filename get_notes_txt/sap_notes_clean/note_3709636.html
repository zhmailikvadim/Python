Symptom

  • In transaction O4NM, within Nomination Overview, the Quick Search drop-down does not contain one or more of the standard fields:
    • Reference Document
    • Schedule Date
    • Nomination No.
    • Nomination Key
  • As a result, you cannot search for/open nominations based on the missing criterion

Environment

SAP S/4HANA

Reproducing the Issue

  1. Run transaction O4NM.
  2. Open Nomination Overview.
  3. Open the Quick Search drop-down and observe that the expected field (for example, Nomination Key) is not available.

Cause

The field is not maintained in SPRO transaction for Quick Search fields for nominations.

Resolution

1. Run transaction SPRO.

2. Choose SAP Reference IMG.

3. Navigate to:

    • Logistics - General
    • Trader’s and Scheduler’s Workbench
    • Basic Functions
    • Graphical User Interface Settings
    • Define Quick Search Fields for Nominations

4. Execute Define Quick Search fields for nomination

5. If the desired field (for example, Nomination Key) is missing, choose New Entries and add it.

6. click on save 

7. If we recheck the table the field will be added 

8. Return to O4NM > Nomination Overview and open the Quick Search drop-down. The field should now be available.
