Symptom

Via SAP Note 3127916 the field Number of Dependents over 66 with Care Needs (CRN65) has been introduced on infotype Fiscal Data (0101) and in payroll table Tax (ST). This field however can't be displayed on the remuneration statement using either transaction PE51 or HRFORMS.

Other Terms

IT0101, P0101, PA0101, PS0101, MP010100, BST00, BF01, CEDT, RPCEDTB0, SAP_PAYSLIP_BE

Reason and Prerequisites

You are using Payroll localization Belgium.

Solution

  • As a general rule SAP recommends that you implement the Support Package specified for your release.
  • You can find more information about the SAP Note Assistant in SAP Support Portal, under SAP Note Assistant.
  • For Remuneration Statements, using the technology of transaction HRFORMS, field ST-CRN65 has been added to MetaNet SAP_DEFAULT for HCM Localization 12. In particular to MetaDimension TAX_BE and MetaStar PAY_TAX_BE=>TAX_BE. 
  • The delivery of the changes of the MetaNet SAP_DEFAULT is NOT possible via correction instructions, only via HR Support Packages.
  • Via this SAP Note 3728289 the delivery of the changes to MetaNet SAP_DEFAULT for the Enhancement Packages EHP7 & EHP8 is done.
