Symptom

Transaction: O3URV_DOC

Valuation document is rejecting with reason code 28B -'CA VOLUMES NOT FOUND FOR VL TRANSACTION KEY' in following scenario:

  • Delivery network producing extracted carbon dioxide (CO2).
  • Volumes are booked at CA level with allocation method 'Sales Point Originated'.
  • Contract level allocation in Valuation.

Other Terms

  • General keywords: IS-Oil, Industry Solution, IS-OIL-PRA, Upstream Oil and Gas, Production and Revenue Accounting
  • Problem specific key words: EP207, Product code 207, extracted carbon dioxide, CO2, Reject reason 28B, Gaseous plant product, No Wet gas, Contract level scenario, Sales Point Originated

Reason and Prerequisites

Program error

CO2 volumes are updated in the table OIU_SB_RNEWC while processing the Contractual Allocation, and this changes were implemented through SAP Note 2732999.

In the current scenario, for the DN with only CO2 volumes (EP207), VL is unable to fetch the volumes from the table OIU_SB_RNEWC which leads to Zero Volume rejects.

Solution

Implement the attached correction instructions to resolve the issue.
