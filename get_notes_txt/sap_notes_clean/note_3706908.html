Symptom

Time Sheet Upload posts error 'Invalid values for Location Code', even though the field's value and its formatting have been vetted and appear to be accurate.

Cause

The error occurred due to incorrect format of the field header name. The file was using the header name 'Location_Code', but the correct name is 'Location Code'. (The correct header name does not have an underscore). This is noteworthy, because most of the fields in this upload do require the underscore.

Resolution

Once the header name was updated to 'Location Code', the upload was successful.
