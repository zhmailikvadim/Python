Symptom

  • Can an S3 Amazon file location connect to a remote directory with subfolders
  • Error: "AWS S3 file </../../../.csv> download error: <Not Found: The specified key does not exist.>. 

Environment

SAP Data Services 14.3.3.x

Reproducing the Issue

Create a file location connecting to an S3 protocol and configure remote directory with subfolders. 

Cause

sub folders definition was not correct

Resolution

  • the remote directory cannot have a "/" in the beginning and requires a "/" at the end.
  • working configuration created as follows: Remote Directory sub1/sub2/sub3/

Keywords

sub folders, file location, S3 Amazon
