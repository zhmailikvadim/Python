Symptom

Cumulative patch for UR-Lightspeed 2603 includes several Unified Rendering Lightspeed bug fixes that were reported since the last cumulative patch 2602. Detailed information about the process can be found in SAP note 2500800 - General information about cumulative patches for Unified Rendering.

This is the complete list of issues fixed by this SAP note (see also References Section):

3714143 TextEdit: Does not open in Chrome and Edge browser
3727016 DatePickerSlider: Time stamp past 12 pm in the 12 hour format cannot be conveniently entered

Other Terms

Web Dynpro ABAP, WDA, Web Dynpro JAVA, WDJ, SAP GUI for HTML, Unified Rendering, UnifiedRendering, UR, Lightspeed,

Reason and Prerequisites

Using WD ABAP starting from 7.02, WD JAVA starting from 7.30 and SAP GUI for HTML all kernel releases. For WD ABAP 7.00 and 7.01 or WD JAVA < 7.30 a different process is in place.

IMPORTANT:

- Read always the English version of this SAP note. Other languages might be outdated due to a delay in the translation process.

Solution

The cumulative Unified Rendering patch 2503.1 is available since 2025-03-18.

The additional quality- and production process of the different frameworks takes usually around two weeks without guarantee. Therefore the specific framework patch/note will be usually released two weeks after the Unified Rendering release date of the patch.

Detailed information about the process can be found in SAP note 2500800 - General information about cumulative patches for Unified Rendering.

To fix this issue, please see the description for your basis framework:

A) Web Dynpro ABAP:

Please check the SAP note

2090746 - WD ABAP: Unified Rendering Update - Instructions And Related Notes

if this cumulative Unified Rendering patch is already available for your specific release. If not please re-check SAP note 2090746 at a later point of time.

Important: Please install always the newest available Unified Rendering patch found in SAP note 2090746 (Patches are always cumulative).

B) Web Dynpro JAVA and SAP Enterprise Portal (Theming):

Please check the SAP note

2503680 - WD JAVA: Unified Rendering Update - Instructions And Related Notes

if this cumulative Unified Rendering patch is already available for your specific release. If not please re-check SAP note 2503680 at a later point of time.

Important: Please install always the newest available Unified Rendering patch found in SAP note 2503680 (Patches are always cumulative).

C) SAP GUI for HTML

Please check the SAP note

2504011 - SAP GUI for HTML: Unified Rendering Update - Instructions and Forecasts

if this cumulative Unified Rendering patch is already available for your specific release. If not please re-check SAP note 2504011 at a later point of time.

 

Important Remark:

In case of using a custom theme in Enterprise Portal or ABAP backend it might be necessary to update the theme meta data.

For more information and details see

  • SAP note 2500800 - General information about cumulative patches for Unified Rendering --> Solution area
  • SAP note 1446099 - Application (Web Dynpro or ITS) is not displayed in customer theme

If these steps are not done it can happen that the custom theme is not used by the backend or looks distorted.
