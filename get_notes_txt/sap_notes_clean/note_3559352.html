Symptom

Enhancements for NCTS Phase 5 are required for communication with the customs system in the transit procedure area.

The Belgian customs authorities have published the national message Closing Data for Transit Operation (BE IENCTS043). To be able to receive this, you must implement the following SAP Notes:

  • 3553052 - BE NCTS Phase 5: Message structures for message IENCTS043
  • 3553054 - BE NCTS Phase 5: Message constants for message IENCTS043
  • 3553055 - BE NCTS Phase 5: Class for message mapping for message IENCTS043
  • 3553883 - BE NCTS Phase 5: Changes in document determination for message IENCTS043
  • 3553053 - BE NCTS Phase 5: Processing of message IENCTS043 as unloading permission

This SAP Note contains the required Customizing.

A further prerequisite is that you have implemented Version V1-48 from SAP Note 3083392 (Cross-country conversions for SEEBURGER BIS 6.7).

Other Terms

NCTS, transit, IENCTS043, Customizing

Solution

Adjust the Customizing as described in the manual correction instructions or import the specified support package.
