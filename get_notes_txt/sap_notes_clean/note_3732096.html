Symptom

Why deleted project's task appear in the Procurement Operations Desk (POD) tab?

Resolution

Deleting a Sourcing Project or Sourcing Request is not fully supported in the Procurement Operations Desk (POD) model. As a result, tasks from deleted projects may still appear in the POD task tab.

Workaround: To remove tasks from POD, delete the tasks individually within Sourcing before deleting.

See Also

If you require this functionality, submit a feature request for deletion support in POD.

Follow KBA How do I create an improvement or enhancement request for Ariba Procurement? for feature request.
