Symptom

You want to use the new disbursement for loan contracts in transaction FN8X.

Up to now, the new disbursement was released only for the product types 340 (consumer loan). 360 (installment loan), and 370 (investor contract). SAP Note 1936376 enabled you to also activate this business operation for mortgage loans and general loans by setting the export parameter E_DISB_ACTIVE in the method ACTIVATE_DISBURSEMENT (BAdI FVD_BUS_OP).

You want to be able to activate the new disbursement for mortgage loans and general loans, as well as the other new business operations, using an indicator in the product type, and therefore no longer require a BAdI implementation.

This is enabled by this SAP Note.

As described in SAP Note 1936376, you should check whether the provided function meets your requirements before you use the new disbursement in the productive system.

Other Terms

FN8X; new disbursement

Reason and Prerequisites

This was not previously provided.

Solution

Implement the attached correction instructions or import the specified Support Package.
