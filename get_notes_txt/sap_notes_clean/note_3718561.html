Symptom

In transaction QM01/QM02/QM03, only the description (like 'Customer complaint') of the notification type is displayed, and the technical ID (like Q1) is not.

Actual:

Desired:

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Environment

  • SAP ERP Quality Management (QM)
  • SAP ERP Central Component
  • SAP ERP
  • SAP Enhancement package for SAP ERP
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA cloud private edition
  • SAP S/4HANA

Cause

Indicator 'Show keys within dropdown list' is deactivated in SAP GUI settings.

Resolution

Activate indicator 'Show keys within dropdown list' at SAP GUI settings.

It is possible to reach this indicator via following steps:

  1. In SAP Logon, click button at the top left corner.
  2. Click Options.
  3. Select folder Interaction Design.
  4. Click on Visualization 1.
