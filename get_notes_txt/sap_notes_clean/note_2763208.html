Symptom

You are migrating price lists using the migration tool, however in the simulation step you receive the error:

"The combination of context elements is not supported"

Environment

SAP Business ByDesign

Reproducing the Issue

  1. Go to the Business Configuration work center.
  2. Go to the Implementation Projects view.
  3. Click on Open Activity List.
  4. Search and open the activity Migration of Price Lists.
  5. Open the link Migrate Price Lists Using the Migration Tool.
  6. On the new screen, select the migration file and click Execute Migration.
  7. Select Execute Migration Step by Step.
  8. In the simulation step, the error "The combination of context elements is not supported" is shown.

Cause

For the migration of price lists, certain fields should not be maintained in the template depending on the type of price list being imported (in some cases even if the field is marked as mandatory in the template)

As example:

  • For Price Lists for specific accounts, the Account ID field in the Items sheet should not be populated as it is filled in the General sheet.
  • For Price Lists where specific product IDs are defined in the Items sheet of the template, the Product Category ID field in the Items sheet should not be populated.

Resolution

Please adjust the template according to the type of price list being imported.
