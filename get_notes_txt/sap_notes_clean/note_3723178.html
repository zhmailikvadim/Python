Symptom

Rejection reason 01 - Reference Document Item Not Available observed in Intrastat transaction /ecrs/rp_edit when settlement documents are sent via report WLF_INTR_TRANSFER_4

Other Terms

Rejected, declaration, reference, POI

Reason and Prerequisites

Missing functionality

Solution

This note provides these new transfer events to adjust the intrastat date for transfer:

4A         Intrastat Map Header

4B         Intrastat Map Item

 

In standard, credit memos are transferred as negative corrections.

If the settlement document item has a billing document or other settlement document as direct reference via fields LFTYPV/WBELNV/POSNRV or REF_TYPE/REF_DOC_NR_1/REF_ITEM_NUMBER, this is transferred as reference document. The intrastat declaration then tries to get the correction period from this.

If there is no reference, or the reference period cannot be determined, the items are shown with a rejection reason in the intrastat report and have to be declared manually if needed. This is correct behavior from standard perspective.

Implement correction instructions.
