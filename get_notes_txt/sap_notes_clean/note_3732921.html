Symptom

You cannot edit a real estate contract or rental object on the Conditions screen if the real estate object to be edited only contains hidden conditions and more condition types are defined as hidden in the condition group.

Other Terms

RECN, REBDRO, RECD, VICDCOND, GUI_RECD_CONDITIONS, required entry field, message 00 055, 00055, condition group

Reason and Prerequisites

This is caused by a program error.

Solution

Import the relevant Support Package for your release or Use transaction SNOTE to implement the correction instructions in advance.
