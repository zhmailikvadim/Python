Symptom

The My Timesheet app (F3074A) use the "Shared List" regarding assignments. But the functionality is not clear.

Environment

SAP S/4HANA

SAP S/4HANA 2023

SAP Fiori for SAP S/4HANA 2023

Resolution

In My Timesheet V4 app there are new categorizations for the Assignment: personal and shared. 

Personal List includes assignments created by the user and Shared List includes the ones created and provided by the administrator.

Shared are not editable by the user, but it could be copied to personal and to modify it.

The report RCATS_CREATE_ADMIN_ASSIGNMENTS has been provided for administrator users to push Assignments into employee assignment list in My Timesheet V4 application. 
Administrator users could choose the set of employees for whom the assignments are to be created via selection parameters in the report.

Relevant tables:
HRFIO_ADM_ASSGMT
HRFIO_ASSIGNMENT

In the HRFIO_ASSIGNMENT, admin assignments are flagged in field ADMIN_ASSGMT.

In the report RCATS_CREATE_ADMIN_ASSIGNMENTS there is no possibility to upload the assignments with a prepared excel file.
