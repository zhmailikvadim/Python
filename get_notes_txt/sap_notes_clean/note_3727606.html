Symptom

User creates a Shopping Cart or other document with catalog item. While opening the catalog the following error occurs: "404 Table not maintained. The requested resource is not available" 

Environment

SAP Supplier Relationship Management

Reproducing the Issue

  1. Log in to the SRM system.
  2. Start to create a document.
  3. Open a catalog.

Cause

Manual steps were not completed after implementing the security note 3638716 [CVE-2026-0513] Open Redirect Vulnerability in SAP Supplier Relationship Management (SICF Handler in SRM Catalog). 

Resolution

  1. Go to the transaction SM30
  2. Open the table  SAPSRM_CTLG_LIST
  3. Maintain HOOK_URL and TARGET_URL in the table (refer to the SAP Note 3638716 or KBA 3706762)
