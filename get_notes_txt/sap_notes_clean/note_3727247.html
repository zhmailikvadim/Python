Symptom

When attempting resizing an attachment, SAP Asset Manager does not check whether the attachment is an image. This note prevents attempting to resize non-image formats, such as PDFs, Word Documents, etc.

Other Terms

  • Attachments 
  • Resize
  • PDF
  • Image

Reason and Prerequisites

  • BAS/Visual Studio Code with SAM MDK Project
  • `patch` command (available on BAS and macOS by default) via terminal or other patching utility
  • Read the guidelines on how to apply the patch and read the appendix section if you receive any error

Solution

  • Find attached file 3727247.patch
  • Save patch file into known location
  • Run command in base metadata folder: `patch -p1 < /path/to/3727247.patch`. Validate 2 files changed and no hunk failures.
  • Republish application
