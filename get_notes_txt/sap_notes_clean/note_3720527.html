Symptom

In 'Manage Billing Documents' app (F0797), an attachment has been added via Attachments tab. When 'Select Attachments for Output' popup is opened from Output Items tab, the attachment already added to billing document is not displayed.

Environment

  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Go to 'Manage Billing Documents' app (F0797).
  2. Open invoice in Edit mode.
  3. Add an attachment via Attachments tab.
  4. In Output Items section, double click on the Attachments button, the attachment already added to billing document is not showing.

Cause

When the printer determined for output items is not a PDF printer, the standard framework prevents attachments.

Resolution

In SPAD, adjust the printer to a PDF printer. If the change of existing output device will break the other business processes, either create a new PDF printer or use an existing PDF printer, then adjust OPD "Printer Settings" to determine output request item with the newly created or existing print queue.
