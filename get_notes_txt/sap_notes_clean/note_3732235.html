Symptom

During execution of some reports or queries using function module REOM_OM_READ_COSP_DATA (or REOM_OM_READ_COSS_DATA) the selected data are doubled. The selection for some of the data include data posted to different clients.

Other Terms

REOM_OM_READ_COSP_DATA, REOM_OM_READ_COSS_DATA, client, MANDT, S_ALR_87013019

Reason and Prerequisites

Program error, data from different clients are selected/aggregated

Solution

Implement the correction instruction
