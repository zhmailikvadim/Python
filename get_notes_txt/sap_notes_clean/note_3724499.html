Symptom

How do I customize the validation message for Bank Key/ABA Routing Number in Supplier Management Questionnaires?

Cause

Resolution

  1. The current system design does not allow customization or modification of the default Bank Key/ABA Routing Number validation message in the event of errors.
  2. You can submit an Enhancement Request to evaluate the possibility of improving the functionality in future updates.
