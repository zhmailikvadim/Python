Symptom

In the process "Sending of EQS PoD Deactivation for Supplier Time Series Total to SUP (DSO View)" (DE_B_EQS_POD_DACTV_SUP_DSO), the process in the system stops at step 8950 and does not proceed to step 8999.

This issue is relevant for market role DSO.

Other Terms

EQS, PoD deactivation, supplier time series total, settlement, MaBiS, DE_B_EQS_POD_DACTV_SUP_DSO, PDoc, precondition

Reason and Prerequisites

The error occurs because an incorrect precondition is configured for step 8999, preventing the process from continuing.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the system completes the process successfully. Import the BC set /US4G/3_0_DSO_SP01_NOTE_3735298. Run the stuck process document (PDoc) again. Verify the fix by checking that the PDoc completes successfully.
