Symptom

Symptom

When you execute the new input help with a hierarchy, the attributes of the characteristic are incomplete.

Other Terms

INA, BW, F4

Reason and Prerequisites

BW SP08, INA request with "Query/ValueHelp"

Solution

  • SAP BW 7.40

    Import Support Package 11 for SAP BW 7.40 (SAPKW74011) into your BW system. The support package is available once SAP Note 2100895 ("SAPBWNews NW BW 7.4 ABAP SP11"), which describes this support package in more detail, has been released for customers.

           
In urgent cases, you can implement the correction instructions as an advance correction. In this case, you must implement the following SAP Notes:

  • 2115138 - INA: Attributes missing from new input help
  • 2117561- ABAP BICS: Additional selection
  • 2118209 - INA: New input help with empty result
  • 2120177 - BICS: Hierarchy attributes



You must first read SAP Note 1668882, which provides information about transaction SNOTE.

To provide information in advance, the SAP Note(s) mentioned above might be available before the corresponding support package has been released. In this case, the short text of the SAP Note will still contain the words "Preliminary version".

 

Other Terms

CRM_UPDATE_BUFFER_DATA, checkman , Interactive reporting, pipeline performance management, issue.

Reason and Prerequisites

Checkman issues in CRM_UPDATE_BUFFER_DATA .

Solution

Apply the note.
