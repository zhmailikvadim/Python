Symptom

In the Pennsylvania Residency Certification Form within the Onboarding module, the help text includes a “Click Here” hyperlink intended to assist users in identifying their County, Tax Collection District, and Municipality for Local Earned Income Taxes (EIT).

When users select the hyperlink, the system redirects to an invalid webpage displaying the error: “This site can't be reached.”

Environment

SAP SuccessFactors Onboarding

Reproducing the Issue

  1. Navigate to the Pennsylvania Residency Certification Form in the Onboarding module.
  2. Locate the help text under the header “Pennsylvania Work Certification Form.”
  3. Click the “Click Here” hyperlink provided in the help text.
  4. The link redirects to an invalid webpage displaying the message: “This site can't be reached.”

Cause

This is a known issue.

Resolution

The Engineering team is currently investigating and working on a fix under Jira reference OBX-42516.

This Knowledge Base Article will be updated once the issue is resolved and a correction is deployed.
