Symptom

Error 403 occurs when attempting to access a SAP Help Portal document.

Environment

SAP Help Portal

Reproducing the Issue

Access any Help Portal document.

Cause

The link to the documentation is outdated. The version may be deprecated or publicly unavailable.

Resolution

In the image below, the general structure of a link to the SAP Help Portal can be seen: 

Important note: recently, the SAP Help Portal URL has been reworked. It is now much easier to find the "version" field, right at the end of the link:

  • https://help.sap.com/docs/SAP_MARKETING_CLOUD/ac1eab4c66bc490da7ac2c378c46b0e7/7de38c63de2644138ca2582b39fcb3c6.html?version=2105.500
    This link will not be accessible as this version is out of maintenance.

By changing the "version" field to latest, the SAP Help Portal will automatically redirect the user to the latest available version of the document.

The modified and correct link will be:

  • https://help.sap.com/docs/SAP_MARKETING_CLOUD/ac1eab4c66bc490da7ac2c378c46b0e7/7de38c63de2644138ca2582b39fcb3c6.html?version=latest
