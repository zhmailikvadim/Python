Symptom

This KBA clarifies the criteria for Server Housekeeping Object Versions on SAP Enable Now.

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

SAP Enable Now - Manager

Cause

Resolution

When performing Object Versions housekeeping, the system will only delete versions that:

  • Were created before the selected threshold date
  • Are not active/published (published/tagged and most recent versions are always retained)
  • If there is only one version of an object available, this version will generally be considered the most recent and will not be deleted by housekeeping. The latest version is always maintained, regardless of whether it is the only version or a higher-numbered one.
