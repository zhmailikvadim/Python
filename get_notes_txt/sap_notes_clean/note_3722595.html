Symptom

You want to use the Fiori app Perform Maintenance Jobs (F5104A) to record time confirmation for an operation. After you opened one of the operations from the list report, you click at the 'Record Time' button and see that the Activity Type dropdown is empty. However, when using the transaction IW41, the Activity Types are available.

Other Terms

F5104A, F2174, Perform Maintenance Jobs, Find Maintenance Order Confirmation, Record Time, C_ObjPgMaintOrderConfirmation, I_MaintOrderConfActivityTypeVH, Time Confirmation, Activity Type, Work Center, MaintOrdConfActyTypHasConf, Plant

Reason and Prerequisites

In the CDS View, Activity types were restricted to those belonging to Work Centers that have existing Order Confirmations.

Solution

Implement this SAP Note.

This restriction was deleted, therefore the dropdown now shows all available Activity Types.
In the Find Maintenance Order Confirmation (F2174) the existing behavior was kept. Therefore, only Activity Types that are associated with a Confirmation are shown.
