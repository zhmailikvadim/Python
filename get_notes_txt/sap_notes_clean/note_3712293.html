Symptom

'Display posting rules - Event Based' has no receivers during production order creation with co-products (Joint Production scenario)

Environment

SAP S/4HANA Cloud Public Edition

Cause

This is the current behavior for joint-production rules.

Resolution

After the rule and order is saved, next time the receiver would display the materials of co-products.

See Also

Settlement Rules of Manufacturing Orders (Event-Based Posting Rules)
