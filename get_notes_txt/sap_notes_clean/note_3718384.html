Symptom

New infotype framework CLASS CL_HR_T7FIU0 method READ has different logic than Module Pool (PA30) FORM re7fiu0.

This difference can lead to replication issues.

Other Terms

IT0206

Reason and Prerequisites

SQL condition for BEGDA was wrong and the last valid record was not found.

Solution

This SAP Note corrects issues described in the Symptom section.

As a general rule, SAP recommends that you install a solution by applying a Support Package.
