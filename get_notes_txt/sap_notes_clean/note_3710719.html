Symptom

Due to requirements from authority, some customers need to perform initial mass upload of infotype 0009 via report HSKCSCP0. This report enables to create registry sheet "change", even if there was no change at that specific date. This can be used to perform this initial mass upload. But it might be useful if this functionality is used, not to show other types of registry sheet outputs.

Other Terms

Report HSKCSCP0, IT0009, mass announcement.

Reason and Prerequisites

This SAP Note is relevant only for Slovakia (HRSK).

Solution

This SAP Note delivers modification of report HSKCSCP0 not to show any other output than for infotype 0009 in case field "Vynútiť zmenu 0009 ku dňu" is used.
