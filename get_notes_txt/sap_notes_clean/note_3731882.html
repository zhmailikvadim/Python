Symptom

  • Stock-Requirements List MD04 with Material Grouping:

 When for a subcontracting process for a component to be provided the material master data of the subcontracting mrp area are deleted,

 then the tabs of the material grouping for Cross-Plant and Intra-Plant are disappearing.

  • Stock Transfer Special Procurement Key for generic subcontracting MRP areas:

  For generic subcontracting MRP areas, master data are set similar to the plant master data and an existing special procurement key of type 

  stock transfer is taken. In this context also stock transfer procurement keys with an issuing storage location for source mrp areas different from 

  the plant MRP area can be wrongly chosen.

Other Terms

MD04, Material Grouping, Cross Plant, Intra Plant, Tabs, MDMA-LOEKZ, CL_PPH_MRPRECORD, SELECT_MATERIAL_OVERVIEW_NEW.

Reason and Prerequisites

Program error.

Solution

The issue is solved with the assigned correction in the next release 2608.

The true subocntracting mrp area is converted into a generic subcontracting mrp area without material master data and

the general situation in MD04 must remain unchanged.

For the second symptom the correction takes care that only a stock transfer procurement key is taken which belongs to a plant mrp area and

has got no issuing storage location assigned.
