Symptom

  • Client process unexpectedly disconnected
  • ASE errorlog contains an Infected with Signal 11 (SIGSEGV) with Address line function locklogin(), followed by a stack trace
  • Stack trace contains the following functions:
locklogin
lobj_droploginmain
lobj_droplogin
s_execute.cold

Environment

  • SAP Adaptive Server Enterprise (ASE) 16.1 SP0
  • Configuration "cloudservices" is on

Cause

This issue has been identified as CR 831699. 

Resolution

This issue has been fixed. 
The fix will be available starting in SAP Adaptive Server Enterprise (ASE) 16.1 SP0 PL02

Keywords

signal, SIGSEGV, locklogin, drop login, cloudservices
