Symptom

  • An error occurs when attempting to create a Business Partner in S/4, displaying the message: "Internal error in SLD API."

Environment

  • SAP Global Trade Services Edition for Hana

Reproducing the Issue

  1. Create business partner via transaction code BP in the S4 system.
  2. Access transaction SRT_MONI in the GTS system.
  3. Use the Message ID with Receiver Interface Name BusinessPartnerSUITEBulkReplicateRequest_In.
  4. Observe the following error details:
  • SEVERITY: error
  • TEXT: Internal error in SLD API
  • URL: http://vhruoad1ci.sap.ssab.com:8000/sap/xi/docu_apperror?xxx
  • ID: 010(LCR)

Cause

The logical system for the business system SAP_CI is missing in the customizing settings located at DRFIMG -> Define Custom Settings for Data Replication -> Define Technical Settings for Business Systems.

Resolution

Please implement the following Note and then retest the process.

3226945 - Error 'UCON RFC Rejected' during RFC call to S/4HANA Cloud

Keywords

internal error, sld api, business partner creation, s/4, mdg_idm_get_lcl_system, sxms_bs_cache, srt_admin, logical system missing, business system id, replication error, inbound processing, sld connection
