Symptom

In processes DE_B_MET_MR_INFO_MOS/DE_B_MET_MR_INFO_DSO, step 0200 is not completed due to step 0200-0899 not being triggered after actions of uploading MR completed.

Other Terms

US4G, DE_B_MET_MR_INFO_MOS, DE_B_MET_MR_INFO_DSO, DE_B_EMBED_UPD_MR_VALUE

Reason and Prerequisites

Process configuration is incomplete for missing step preconditions.

Solution

Please kindly implement the attached Correction Instruction, and take following step to complete the Manual Activity.

 1. Download the BC-Set attached to this SAP Note.
 2. Run transaction SCP3, upload the BC-Set.
 3. Run transaction SCPR20, input the BC-Set name and then activate it.
