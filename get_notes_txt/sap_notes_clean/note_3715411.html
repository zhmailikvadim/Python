Symptom

Labelling Category field is getting changed under label data view for material but the change log is not captured in MM04. Other fields on the same view are also not recorded in t-code MM04.

Environment

  • SAP ERP Logistics (LO)
  • SAP R/3
  • SAP R/3 Enterprise
  • SAP ERP Central Component
  • SAP ERP
  • SAP Enhancement package for SAP ERP
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Go to t-code MM02.
  2. Enter Material.
  3. Select Label Data view.
  4. remove/change a field value for such as brand name/labelling category.
  5. Save.
  6. enter t-code MM04.
  7. No change log captured for the change made.

Cause

This is standard system behavior. Label data change log is stored in table CDHDR/CDPOS with the object LABEL instead of MATERIAL and transaction MM04 only shows the change log for change object MATERIAL. 

Resolution

Execute transaction code CBGL_CD03 to display label data change documents from the material master.
