Symptom

  • Implementation Partner or Consultant needs to reset 'hradmin' credentials
  • Customer would like request reset of 'hradmin' credentials
  • No available admin user to reset password of hradmin

Environment

SAP SuccessFactors Onboarding 1.0

Reproducing the Issue

1. Navigate to SAP SuccessFactors HCM and look for the "Incorrect Password" error. 

Cause

  • New Request
  • Forgotten Password
  • Incorrect Password
  • Lock Credentials

Resolution

For Live Customer(s)

The hradmin credentials are only used by Implementation Partner(s), Consultant(s), or Professional Services(PS) to initially configure Onboarding application via a standalone link. If you are currently engage with an SAP Certified Partner or Consultant, please advise them to follow the instruction below.

 

For Implementation Partner(s) and Consultant(s)

  1. Login and navigate to HCM Cloud Operations Portal using your SAP Credentials
  2. Go to Operation Requests tab and select Request Forms
  3. Under ONB Requests section, click ONB Hradmin Password Reset
  4. Provide the needed information and ask written approval from the client by using below template. 
  • From: Customer@customeremail.com
  • To: Partner@partneremail.com
  • Subject: Onboarding Reset 'hradmin' Credentials

Note

  • Approval should be provided in written English
  • Approval emails must be attached in a format that preserves the sender (customer) headers, such as .msg. Copy and pasted text does not meet this requirement
