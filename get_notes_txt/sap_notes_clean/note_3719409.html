Symptom

Unable to display Custom Field in App Create Supplier Invoice - Advanced. 

Environment

SAP S/4HANA Cloud Public Edition

Cause

This is standard system design.

Resolution

Custom fields of the Business Context Procurement: Supplier Invoice Item with Purchasing Document Ref. / Consignment (MM_SI_ITEM) can be currently added to the Create Supplier Invoice App only. In the Create Supplier Invoice Advanced App the available business context is Procurement: Supplier Invoice Header.

Keywords

MIRO, Create Supplier Invoice - Advanced, Create Supplier Invoice, Custom field, MM_SI_ITEM
