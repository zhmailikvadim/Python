Symptom

You are using non-numlike objects with MWB Write Behavior 6. When executing the simulation/migration a dump occurs.

Other Terms

MWB, BTC, Business Transformation Center

Reason and Prerequisites

You are using MWB or any of its consumers in combination with write behavior 6.

SAP Note 3693194 and 3707796 (if applicable) is implemented.

MWB functionality is used on one of the following releases:

  • SAP S/4HANA 2025 FPS00 - FPS01
  • SAP S/4HANA 2023 FPS00 - SPS05
  • SAP S/4HANA 2022 FPS00 - SPS07

Solution

Implement the attached correction using transaction SNOTE.
