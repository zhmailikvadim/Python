Symptom

  • Discrepancies are observed between the Gross Price at the line item level and the Condition level.
  • No error message is explicitly mentioned, but the issue involves pricing inconsistencies.

Environment

  • Materials Management (MM)
  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA

Reproducing the Issue

  1. Access transaction ME33K.
  2. Navigate to contract and choose the line item.
  3. Review the Gross Price at the line item level and compare it with the Condition level.
  4. Observe discrepancies.

Cause

Inconsistency issue.

Resolution

  1. Open the contract in change mode (ME32K) and navigate to the service screen.
  2. From the menu option, select Service specification → Prices → Find New.
  3. Save the contract.
  4. Perform the test to verify if the issue persists.
  5. If the issue remains unresolved, contact support for further assistance.
