Symptom

System allows user to copy a promotion which has been changed using One Click Action in the Promotion overlay of Customer Business Plan or the Promotion Planning screen without saving it first.

Other Terms

Promotion Copy, OCA, CBP, conditions timeshift

Reason and Prerequisites

If save is not done before a promotion is copied then changes to some data like BW planning, or changes to generated conditions are not copied over to the new promotion and are lost.

Solution

Implement the attached correction instruction
