Symptom

It is not possible to select a collection contact person id during customer contact creation from Process Receivables Fiori app.

 "Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Environment

  • Fiori-UI for Collections and Dispute Management
  • S/4HANA

Reproducing the Issue

  1. Access the 'Process Receivables' app 
  2. Navigating to the Fiori F4707 - Manage Customer Contact application.
  3. Attempting to select a contact person using the Contact Person ID field.
  4. Observing that no contact persons are displayed.  

Cause

 The contact person has not been created as a "Collection Contact Person".

Resolution

Ensure the relationship category 'Has Contact Person' has been created for the contact person:
