Symptom

You are using SFTP receiver channel with Message Protocol: File Content Conversion and you have configured fieldFixedLengthType=byte. When field width in the actual document is less than the defined in fieldFixedLengths, spaces are not getting appended to the file correctly.

Other Terms

 SFTP, receiver, FCC, fieldFixedLengthType, byte

Reason and Prerequisites

Bug fix. Byte length not calculated properly while appending spaces.

Solution

Code changes have been made to resolve this issue.

To get this correction available in your Support package you need to apply the patch matching to your support package version as per the instructions in the NetWeaver Support Package Stack guide which can be found on the SAP Service Marketplace as described in SAP Note 952402.
