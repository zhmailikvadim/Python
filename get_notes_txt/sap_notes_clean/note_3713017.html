Symptom

At the account level, a field labeled “Natural Person” is available. Clarification is required regarding the purpose of this field and how it is used within the system.

Environment

  • SAP Sales Cloud Version 2
  • SAP Sales and Service Cloud Version 2

Reproducing the Issue

  1. Open an Account.
  2. Expand the account panel.
  3. Observe the “Natural Person” toggle.

Cause

The Natural Person field originates from legacy SAP ERP data models. Although the field is available in SAP Sales Cloud Version 2, it is not currently tied to any functional business logic within the application.

Resolution

The Natural Person field is currently not used by any business process in SAP Sales Cloud Version 2. It is present for compatibility and data model alignment purposes only.

Keywords

Natural Person, Account, SAP ERP, toggle, Sales Cloud Version 2, V2
