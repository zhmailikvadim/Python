Symptom

When trying to add new tab to asset master, error ATAB022 "Tab 09 is not allowed. " happens.

Environment

  • SAP R/3
  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA Finance
  • SAP S/4HANA

Reproducing the Issue

Creating new tab in asset master

Cause

Standard design.

Double-click on the error message, check the following description:
*****************
Tab 09 is not allowed.
Message no. ATAB022

Diagnosis
A maximum of 08 tabs are allowed in tab layouts of application FI-AA.

Procedure
Enter a value between 01 and 08.
****************

If there has already been 8 tabs there. 09 cannot be created, for the max number of tabs is limited.
This is the standard limitation from the system to restrict to only 8 tabs.
There is no way to bypass it using the standard setup.

Resolution

Consider to add the customer fields to an existing tab. 

Keywords

user layout, AS01, AS02, AS03
