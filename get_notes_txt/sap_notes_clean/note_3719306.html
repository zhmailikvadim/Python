Symptom

Error "No valid business transaction type: RFBU" generated in Manage SES app (F2027). 

Environment

  • SAP S/HANA
  • Fiori 
  • SAP S/4HANA Private Cloud 

Reproducing the Issue

  1. Access the Manage Service Entry Sheets app (F2027)
  2. Attempt to process the application.
  3. Error message "No valid business transaction type: RFBU" is generated. 

Cause

Missing customizing content. 

Resolution

Run the report FINS_BUSTRNSCTN_CNTNT_RETRIEVE from SAP Note 3269873

See Also

SAP Note  3269873 - Various symptoms caused by non-existent business transaction types
