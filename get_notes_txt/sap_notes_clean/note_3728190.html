Symptom

In the semiretirement forms (request for semiretirement or change form), the employer name or address of the employer from central address management is printed incorrectly.

Other Terms

Employer, T536C, V_T536C

Reason and Prerequisites

You use Payroll Austria and a current form for semiretirement.

Solution

The correction is provided by HR Support Package (via CLC if necessary).
You can implement an advance correction by implementing the automatic correction instructions using SNOTE.

Technical Details

After you implement this SAP Note, the system first checks whether the enhanced address information (NAME1, STRAS, ORT01, PSTLZ) is available from address management and - as technically named in T536C - transferred. If these are not available, the unstructured data is still transferred from the lines "LINE".

For the request for semiretirement (using the example for 2026), this means:

  • Field Employer in = NAME1
  • Company Address field = STRAS, PSTLZ + ORT01. Note that the house number is added automatically by Address Management.

For the change form (using the example for 2024, but also for 2026 in the future), this means:

  • Field Employer in = NAME1

If the enhanced address information is not available, the following applies across forms:

  • Field Employer in = LINE1
  • Company Address field = LINE2 + LINE3
