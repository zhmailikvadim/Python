Symptom

Attempting to assign class type 399 to a material results in an error:
"Class type 399: the status of class does not allow assignment. Message no. CL531"

Environment

  • SAP ERP Logistics (LO)
  • SAP ERP Classification (CA)
  • SAP ERP
  • SAP Enhancement package for SAP ERP
  • SAP S/4HANA
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Start transaction MM02 for a material.
  2. Open the Classification view.
  3. Attempt to assign a class of class type 399 to the material.
  4. Alternatively, try the assignment in transaction CL20N.
  5. Observe error CL531 stating the class status does not allow assignment.

Cause

Class type 399 is intended for Variant Configuration HANA view generation and is not designed for direct assignment to objects in transactions like MM02 or CL20N.
The system prevents assignment by design, leading to message CL531.

Resolution

To generate such views use report BSCL_CLASS_VIEW_GENERATION in your SAP HANA system. For details, see note 0001892589 .
