Symptom

The field that needs to be used to filter ServiceCalls or Activities by is not an option in the filter list

Environment

SAP Field Service Management

Reproducing the Issue

  1. Open the Dispatcher Board and navigate to the Service Call/Activity list.
  2. Open the basic filter panel.
  3. Try to add or use a filter for the desired field; the field is not available among the basic filter fields.

Cause

The field is not included as a basic filter field in the current system configuration.

Resolution

  1. Create a User Defined Field (UDF) on the object to store the field value (avoid Date/Time/Datetime types so it appears in basic filters).
  2. Configure a Business Rule to populate the UDF when the object is created or updated:
    1. Set the rule trigger on object create/update.
    2. Map the UDF to the field value needed to filter.
    3. Save and activate the rule.
  3. Confirm the UDF appears in the basic filter panel for Service Calls/Activities.
  4. Use the basic filter to filter Service Calls/Activities by the new UDF.
  5. Optional: If native support is required, submit a product enhancement request to have the field added as a basic filter field in the Dispatcher Board.
