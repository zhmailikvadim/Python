Symptom

  • In SAP Cloud connector, subaccount shows "Cannot load configuration: Short read of DER length";
  • Subaccount Certificate: Unexpected situation encountered;
  • Tunnel Information -> Status: Wrong configuration

Environment

SAP Cloud Connector

Cause

The subaccount certificate created under SCC installation is corrupt. Hence the subaccount configuration cannot be read,

Resolution

  1. Delete the subaccount from SCC -> Connector -> Subaccount Dashboard -> Click on "Delete" button for this specific subaccount
    *All configurations will be removed from the installation for this subaccount
  2. Add the subaccount again following steps in Adding and Managing Subaccounts | SAP Help Portal
