Symptom

How can a Budget Assignment History Audit Report be extracted to determine who made changes in the Budget Assignment and when those changes occurred?

Environment

  • Product: SAP SuccessFactors Compensation
  • Forms & Templates

Reproducing the Issue

Cause

System Limitation.

Resolution

It is currently not possible to report on or extract an audit history for Budget Assignment changes in SAP SuccessFactors Compensation.

We recommend submitting an Influence / Enhancement Request using https://influence.sap.com/sap/ino/#/idea/354014 so Product Engineering can consider adding Budget Assignment auditing in a future release.
