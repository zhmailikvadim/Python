Symptom

The Plant has been set in Default Values settings for users.
However, the plant is not proposed when opening app MD04 (Monitor Stock / Requirements List)

Environment

SAP S/4HANA Cloud Public Edition 

Reproducing the Issue

  1. Go to user settings.
  2. Go to 'Deafult Values' to set default value e.g. Plant under 'Materials Management'.
  3. Open app MD04,  the set plant is not displayed by default. 

Cause

The value was not selected from the Help.

Resolution

Select the value from the Value Help rather than entering manually. 

Keywords

Default Values, Plant, Material, MD04, CO01, Monitor Stock / Requirements List
