Symptom

You declare your exports from SAP GTS to the French customs system DELTA Export.

Even after you implement SAP Note 3707088, the incompleteness check issues an error with reference to the rule R0201.

Other Terms

FR, France, DELTA-E, DELTA, export, CUSANM, CUSVER, R0201, NAT_502, HANDLE_CONSIGNOR, message class /SAPSLL/CUS_ECC_UCC, 040

Reason and Prerequisites

The reason for this is the rule NAT_502, which has since been overridden by customs, which is taken into account programmatically in the data retrieval for the outbound messages.

Solution

Implement the attached correction instructions or import the specified support package.
