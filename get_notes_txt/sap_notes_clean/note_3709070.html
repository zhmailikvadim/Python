Symptom

  • Qualifications are defined on the Job Posting Template
  • The Qualifications section is blank during Job Posting creation

Cause

  • Use Qualification lookup rule is set to Yes on the Job Posting Template
  • When this rule is enabled, Qualifications from the Job Posting Template will not be displayed during Job Posting creation
  • When this rule is enabled and a Qualification custom lookup exists, Qualifications will be auto-populated from the custom lookup pick list

Resolution

Disable this rule to have Qualifications pulled from the Job Posting Template instead of the custom lookup:

  1. Navigate to the Job Posting Template > Click Edit
  2. Continue to the second page > Disable Use Qualification lookup
  3. Click Update
