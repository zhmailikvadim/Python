Symptom

  • Displaying job classification instead of position title on the dashboard 

Environment

Sap Successfactor Onboarding

Reproducing the Issue

  1. Navigate to the Offboarding dashboard.
  2. Review the displayed fields for offboardees.
  3. Observe that the Location and Job Classification are displayed, but the Position Title is not populated.

Cause

  • This behavior is hardcoded in the system.

Resolution

  1. This behavior is expected and hardcoded in the system. The system selects the Job Classification instead of the Position Title.
  2. If this behavior is required to be changed in future updates, it is recommended to raise an enhancement request via the following link: https://me.sap.com/notes/2090228

See Also
