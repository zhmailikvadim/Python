Symptom

You edit the data of an infotype 0001 record for an employee with the master data or the Roadmap application. When you do this, several positions are assigned to the employee that are listed in the lower table (group "Organizational Assignment") of the screen.

If there is then a check of the infotype 0001 data (roundtrip), the system may incorrectly delete all positions from the table, except for the position that is saved in the main record of infotype 0001.

Other Terms

Personnel & Organization

Reason and Prerequisites

Program error

Solution

Import the relevant HR Support Package. In urgent cases, you can manually implement the changes in accordance with the correction instructions.
