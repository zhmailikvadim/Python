Symptom

  • Unable to create a Cloud Foundry space in the SAP BTP subaccount.
  • Error message shows:
    "Unable to process request because an error occured. Error details: Invalid organization. Ensure the organization exists and you have access to it."

Environment

BTP Cloud Foundry

Reproducing the Issue

  1. Attempt to create a Cloud Foundry space in the SAP BTP subaccount.
  2. Observe that the space cannot be created.

Cause

Missing roles in Cloud Foundry.

Resolution

  1. Verify if the "Cloud Foundry" dropdown is available in the left-hand sidebar of the subaccount.
  2. Check if the user experiencing the issue is listed as an "Org Member" in the Cloud Foundry organization.
  3. If the user is not listed, add the user as an Org Member by following the steps outlined in the SAP documentation: Managing Org Members
