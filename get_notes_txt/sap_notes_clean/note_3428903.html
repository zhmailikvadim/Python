Tasklist RS_B4H_TO_READY4CONVERSION fails with error:

DEST XXXXX not RSPM-migrated      
IOBJ XXXXX not RSPM-migrated
Not all objects are RSPM-migrated

Environment

BW 7.5

Cause

Requests of Infoobject (IOBJ) and Open Hub Destinations (OHS) need to converted to RSPM

Resolution

Click on the hotspot in line "Transfer Request Info. for InfoObjects and OpenHub Destination" to create automatically a new tasklist run SAP_BW4_TRANSFER_READY4CONV. This tasklist run transfer the request information of the BW objects IOBJ and OHS. After the tasklist SAP_BW4_TRANSFER_READY4CONV has been finished with success, continue with RS_B4H_TO_READY4CONVERSION.


It is recommended to start the tasklist RS_B4H_TO_READY4CONVERSION short before upgrading the system to BW/4HANA as it is then no longer possible to load master data, see note SAP Note 2930058
