Symptom

In BPC Embedded Model DAP detail page, user can open the Member Selector, but there is no response when clicking the "OK" button. The issue occurs if the Dimension doesn't have any Text field. In other words, there is no Description field of any Master Data.

Reason and Prerequisites

This is a program error.

Solution

For SAP Business Planning and Consolidation 10.1, version for SAP NetWeaver on BW750, upgrade to BW750(Component SAP_BW) Support Package 22.

For SAP Business Planning and Consolidation 10.1, version for SAP NetWeaver on BW751, upgrade to BW751(Component SAP_BW) Support Package 13.

For SAP Business Planning and Consolidation 10.1, version for SAP NetWeaver on BW752, upgrade to BW752(Component SAP_BW) Support Package 09.

For SAP Business Planning and Consolidation 10.1, version for SAP NetWeaver on BW753, upgrade to BW753(Component SAP_BW) Support Package 07.

For SAP Business Planning and Consolidation 10.1, version for SAP NetWeaver on BW754, upgrade to BW754(Component SAP_BW) Support Package 05.

For SAP Business Planning and Consolidation 10.1, version for SAP NetWeaver on BW755, upgrade to BW755(Component SAP_BW) Support Package 03.

For SAP Business Planning and Consolidation 11.0, version for SAP BW/4HANA , upgrade to(component BPC4HANA100) Support Package 16.

For SAP Business Planning and Consolidation 11.1, version for SAP BW/4HANA , upgrade to(component BPC4HANA200) Support Package 08.
