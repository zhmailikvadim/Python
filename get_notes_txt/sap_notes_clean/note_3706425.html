Symptom

Specific languages, such as Catalan (CA) and Welsh (CY), cannot be selected as logon languages in the public cloud edition environment.

Environment

SAP S/4HANA Cloud Public Edition 2508.

Reproducing the Issue

  1. Attempt to select a specific language, such as Catalan (CA) or Welsh (CY), as the logon language in the SAP S/4HANA Cloud Public Edition environment.
  2. Observe that these languages are not available for selection. 

Cause

The Language Configuration app allows configuration of active languages for application use.

Active languages are the set of languages available for user interaction within applications. By default, a fixed list of 41 languages is available via the value help for languages, which can be customized using the Language Configuration app. 

Logon languages are the set of languages available for logging into the system. This set is fixed and cannot be modified using the Language Configuration app. 

Resolution

Currently, there are 34 languages available as logon languages. These can be found in the following PAM link:
https://userapps.support.sap.com/sap/support/pam?hash=s%3DSAP%2520S%252F4HANA%2520CLOUD%25202508%26filter%3D%26o%3Dmost_viewed%257Cdesc%26st%3Dl%26rpp%3D20%26page%3D1%26pvnr%3D73555000100900008302%26pt%3Dt%257CLANG%26ainstnr%3D73555000104900016333

This component handles only the technical aspects of supporting languages in the SAP landscape, but not translation and related services.
