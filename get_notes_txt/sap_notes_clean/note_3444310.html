Symptom

This SAP note provides customers additional functionality for the CCM to OCM migration scenario.

In case of finalization the original data from CCM contract might not be up to date. See also note 2731374. As a result, you will most likely get an allocation error,

because there are steps missing in order processing. That itself is not an issue of migration.

Note 2731374 describes what to do in this case. This note provides a better tailored error message to detect this finalization scenario.

Other Terms

Optimized Contract Management,

Migration,

Finalization

Reason and Prerequisites

This is a special development for migration from classic contract management to optimized contract management in a distributed scenario.

This note shall be considered as a part of preliminary delivery. It provides a foundation for additional migration functionality, which will be provided later.

Solution

This note shall only be implemented if this is the prerequisite for another error correcting note.

This SAP note provides customers a basis for CCM to OCM migration. There will be further deliveries regarding the migration functionality available in the future.
