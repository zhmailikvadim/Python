Symptom

This SAP Note provides the following revisions for the special deduction of common life insurance since 2026 for employees with dependents under age 23.

  • new field CLIDP in structure PS0146
  • new configuration for common life insurance deduction with deduction indicator 'J' in table T5J40
  • new methods to read the configuration from table T5J40

Other Terms

CL_HRPAYJP_T5J40, GET_MAX_DEDUCT, GET_Y14_UPDED, GET_Y70_UPLMT, READ, CL_HRPAYJP_P0146_DED, GET_YEA_DED_LIFE

Reason and Prerequisites

This is a part of legal changes.

Solution

You can apply the relevant HR Support Package or implement the manual changes together with the correction instructions in this SAP Note.
