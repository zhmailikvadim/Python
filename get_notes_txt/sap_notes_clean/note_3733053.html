Symptom

Execution of BAPI_LOCSRVAPS_GETLIST2 stops because of runtime error CREATE_DATA_UNKNOWN_TYPE. This runtime error might also occur when you are executing other BAPIs.

Other Terms

CREATE_DATA_UNKNOWN_TYPE, SAPL10002, L10002F03, LOC_CHECK_AUTHORIZATION

Reason and Prerequisites

  • SAP Note 3608233 or relevant Support Package is implemented at your S/4 system.
  • The BAPI is called by RFC. 

Solution

Implement the attached correction instruction or implement the relevant Support Package to adjust the code. Afterwards the BAPI execution will finish successfully without runtime error.
