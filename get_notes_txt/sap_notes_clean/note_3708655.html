Symptom

The QCT check “HANA Configuration Infrastructure” reports RED/NG. The check details may show messages such as “Database not found”, “Operation not defined” (for example related to HECQC HANA operations), or SQL connection errors like “connection refused” on one or more hosts.

Environment

SAP HANA DB Version 2.0

Cause

Based on the reported check output, the RED status can occur when the QCT execution scope includes a host for which the required SQL connection to the SAP HANA database cannot be established, or when the target host cannot be processed correctly by the executed HANA check.

Resolution

Verify that the QCT execution scope contains only intended SAP HANA database hosts and that SQL connectivity to the affected SAP HANA database is working for those hosts.

If the reported host is not intended for the executed HANA check, correct the scope definition and run the QCT report again.

If SQL connectivity cannot be established, resolve the underlying connectivity issue first and then re-run the QCT report.

Keywords

QCT, HECQC, HANA Configuration Infrastructure, connection refused, database not found
