Symptom

In Search Courses Fiori App, when a text which is more than 8 characters is selected from the dropdown, an error stating Value violates facet information maxlength=8 is displayed.

Other Terms

Search Courses, Filters, Location, maxlength, 8 characters

Reason and Prerequisites

The application had not handled the length of the filter correctly.

Solution

Apply the corresponding Support Package to incorporate the changes. Alternatively, apply the attached correction instructions.
