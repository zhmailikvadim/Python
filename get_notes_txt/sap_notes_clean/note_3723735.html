Symptom

You have stocks with and without SLED in your warehouse. Selecting specifically stocks without SLED is possible with note 3310048, However, the same problem occurs for the Latest Delivery Date selection.

Other Terms

LDD VFDAT S_LDD

Reason and Prerequisites

Program Error

Solution

Implement the code correction

Please note that the correction might require your selection variants, as it does not automatically exclude the empty SLED. For instance if your selection is:

Latest Delivery Date <= Today

Then, the empty VFDAT will fall into this selection. You could for instance change this selection to:

Latest Delivery Date between 01.01.1900 and Today

Alternatively, you may exclude the initial SLED (00.00.0000) in the select-options, if you want to keep the stocks with empty SLED unconsidered.
