Symptom

ENSA2, Enqueue Server 2 shows yellow status when checked with sapcontrol command:
              sapcontrol -nr <instance_number> -function GetProcessList
 
enq_server, Enqueue Server 2, YELLOW, Running but status info unavailable

Environment

  • SAP NetWeaver                                           
  • SAP NetWeaver Application Server for SAP S/4HANA    
  • ABAP PLATFORM - Application Server ABAP                  
  • Operating System independent
  • Database independent

Reproducing the Issue

sapcontrol -nr $$ -function GetProcessList

Cause

There are configuration errors in the enqueue parameters during the kernel downgrade process.

Resolution

Please review the enqueue server parameters and compare them with the configuration before the issue started. Switching to the Standalone Enqueue Server 2 | SAP Help Portal 

Keywords

upgrade, status, SAP ASCS instance, Running but status info available
