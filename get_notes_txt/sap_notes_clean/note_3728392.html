Symptom

In My Timesheet V4 the users want to see for example the activity types and the connected text by the time entries without any other action. 

Environment

HCM Fiori My Timesheet V4

Reproducing the Issue

Open My Timesheet V4 app

Resolution

In order to permanently show the details of the Assignment as a text under each entry, the BADI: HRFIO_B_TSH_FUNCTIONAL with the Method: HRFIO_IF_TSH_FUNCTIONAL~GET_ATTRIBUTES_LIST can be used.
This enhancement displays the description of the Attribute for each entry permanently below each of the Assignment Entry Fields.

As an example code for the Activity type:
method HRFIO_IF_TSH_FUNCTIONAL~GET_ATTRIBUTES_LIST.

   DATA: ls_attribute_list TYPE HRFIO_S_TIMESHEET_ATTRIBUTES.

    ls_attribute_list-gtnc_fieldname = 'ControllingArea'.
    ls_attribute_list-field_label = 'Controlling Area'.
    ls_attribute_list-field_name = ls_attribute_list-gtnc_fieldname.
    ls_attribute_list-text_display_format = '#TEXT_FIRST'.
    ls_attribute_list-field_rank = 3.
    APPEND ls_attribute_list TO et_attributes_list.
    CLEAR ls_attribute_list.

    ls_attribute_list-gtnc_fieldname = 'HCMActivityType'.
    ls_attribute_list-field_label = 'Activity Type'.
    ls_attribute_list-field_name = ls_attribute_list-gtnc_fieldname.
    ls_attribute_list-text_display_format = '#TEXT_LAST'.
    ls_attribute_list-field_rank = 10.
    APPEND ls_attribute_list TO et_attributes_list.
    CLEAR ls_attribute_list.
...
...

  endmethod.

The Text display format can be as '#NO_TEXT', '#TEXT_LAST', '#TEXT_FIRST' also.
