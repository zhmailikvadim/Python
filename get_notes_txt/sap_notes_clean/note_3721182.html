Symptom

  • Error message "The document is too large to be downloaded. Please filter by material"  while exporting cost component split data in app 'Display Actual Costing Result'
  • Unable to download cost component information in 'Display Actual Costing Result' app (f6073) even after filtering by single material.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Navigate to the app Display Actual Costing Result (F6073).
  2. Enter the values for filtering 
  3. Click "Go".
  4. Change the view to "Cost Component Split".
  5. Click "Custom Export to Excel".
  6. Observe the error message: "The document is too large to be downloaded. Please filter by material."

Resolution

  1. Ensure that the filter fields for Company Code and Plant are left empty while filtering.
  2. Click on the export option to generate the required output.
