Symptom

  • The system by default displays dates in the MM/DD/YYYY format.
  • A request was made to change the date format to DD/MM/YYYY throughout the system.

Environment

SAP Sales Performance Management
- Incentive Management

Reproducing the Issue

  1. Login to any workspace.
  2. Observe the date format displayed as MM/DD/YYYY.

Cause

Customer was using incorrect locale settings in the browser, leading to the issue.

Resolution

  1. Verify the locale settings configured in the browser.
  2. Adjust the locale settings as per local requirement to ensure the date format displays as DD/MM/YYYY.
  3. Conduct validation across workspaces to confirm the updated date format aligns with the requirements.

Refer to: SAP Help Documentation - Locale Settings
