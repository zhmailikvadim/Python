Symptom

With the final due date validation, it was not possible to create an order from a notification if a Priority is assigned but the Final Due Date is missing — even when the BAdI EAMS_LACD_CALCULATION is implemented to clear the message or adjust its severity.

Other Terms

FDD (Final Due Date), Maintenance Notification, Maintenance Order, BAdI (Business Add-In), Message Type Configuration, Message IW881, IBAPI_H_CHANGE_INTERNAL, SAP GUI Transactions (IW21, IW22, IW31, IW32, IW34), EAMS_LACD_CALCULATION.

Reason and Prerequisites

Program error.

Solution

To resolve this issue, apply the correction from this note and implement the BAdI EAMS_LACD_CALCULATION to clear the messages or adjust its severity.
