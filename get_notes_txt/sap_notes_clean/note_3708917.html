Symptom

Version History

Version 1 (28.01.2026): Pilot Version

Version 2 (06.02.2026): General Availability (No changes from Pilot Version)

With the introduction of SAP Note 3676025 (PY-IE: SSP: Used Entitlement (checked incorrectly)), the payroll function IESXP rejects any employee that is sick on New Year's Day, but not sick the day before New Year's Day, and New Year's Day has been changed to a working day for that employee - via a Substitution infotype (2003).

HRIE_GEN(021)                 Internal Error; Contact SAP. (Class CL_HRIE_SSP Method ADJUST_REM_ENT_DATES)

Other Terms

IESXP, HRIE_GEN(021), Class CL_HRIE_SSP, Method ADJUST_REM_ENT_DATES

Solution

EITHER

Apply the relevant HRSP

OR

Apply the correction instructions attached to this SAP Note, via SNOTE.
