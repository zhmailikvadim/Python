Symptom

When posting GR to a Purchase Order, inspection lot is created, but QM order is not automatically created. It is possible to manually create the QM order with transaction QM02.

Environment

  • SAP ERP Quality Management (QM)
  • SAP R/3
  • SAP R/3 Enterprise
  • SAP ERP Central Component
  • SAP ERP
  • SAP Enhancement package for SAP ERP
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA, on-premise
  • SAP S/4HANA Cloud Private Edition

Cause

Missing configuration or missing account assignment information.

Resolution

Prerequisites to enable the automatic creation of a QM Order:

  1. Set flag "Individual QM order" for the inspection type in the material master
  2. Maintain "Order Type" in customizing path: transaction QCC0 -> Quality Inspection -> Inspection Lot Creation -> Maintain Inspection Type
  3. Maintain account assignment object in the Purchase order, so it can be copied to the settlement rule of the individual QM Order. For example, specify a cost center in the purchase order.
