Symptom

  • The location list upload in the MaCo-Cloud remains stuck in "processing" status and does not progress.
  • The issue occurs in the productive system (IMP / iMSB Production S/4HANA).
  • Supported file formats for upload include CSV, XLS, and XLSX.

Environment

SAP S/4HANA Utilities Master Data Updates Option for Germany

Reproducing the Issue

  1. Attempt to upload a location list file in the MaCo-Cloud.
  2. Observe that the upload remains stuck in "processing" status and does not progress.

Cause

The backend job responsible for processing the uploaded files was stopped, preventing the files from being processed.

Resolution

  1. Verify the backend job status to ensure it is active.
  2. Re-enable the backend job if it is found to be stopped.
  3. Process the uploaded file.
  4. If the upload fails due to re-enabling your test system, contact SAP customer services.
  5. Once re-enablement is completed from SAP end, re-upload the file(s)

Keywords
