Symptom

  • Embedded (inline) images are not visible in the standard timeline view.
  • Notes appear empty in the Interaction Log or Note Card.
  • Behavior occurs in:
    • Case Notes
    • Agent Desktop Timeline Notes

Environment

  • SAP Service Cloud Version 2
  • SAP Sales Cloud Version 2 

Reproducing the Issue

  1. Open the Case List.
  2. Open any case.
  3. Navigate to the relevant notes in the timeline.
  4. Observe that the notes appear without any format and inline images are missing.

Cause

  • This behavior is expected by design.
  • The Timeline Card / Interaction Log only supports plain text.
  • Inline images and rich-text formatting are not rendered in the standard timeline view.
  • The timeline view is intentionally limited to simple text for performance and UI consistency reasons.

Resolution

  • Users must use the Preview mode (Eye icon) to view : 
    • Inline images
    • Rich-text formatting
    • Embedded content
  • This applies to:
    • Notes in Cases
    • Timeline notes in Agent Desktop
