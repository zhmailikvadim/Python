Symptom

You run the app Cash Flow Statements or the report Cash Flow Statement (transcation IDCNCASH) for China. Some document items aren't displayed when you use the Profit Center filter on the selection screen.

Other Terms

Cash Flow Statement, IDCNCASH, J_3RFFORM4, China, CN

Reason and Prerequisites

This is a program error.

The system removes the document items incorrectly from the results.

Solution

Apply the corresponding support package or implement the correction instructions in this note.
