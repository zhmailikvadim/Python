Symptom

For legacy documents persisted with an initial profitability segment number '0000000000', the aggregation of the posting items does not work.

Other Terms

ADJINV PAOBJNR POSTDOC_AGGR TFK2705

Reason and Prerequisites

As of SAP Note 3320010, the initial value for the profitability segment  is changed from NUMC (initial value = '000000000') to CHAR (initial value = ' '). 

Program error

Solution

Implement the attached correction instructions or import the relevant support package.
