Symptom

When you perform an intercompany transfer for service objects, item to item relationships between the in-house service objects in the document flow can be generated but not persisted. This leads to an incomplete or inconsistent document flow and can lead to incorrect system behavior and UI inconsistencies in intercompany in-house services.

Other Terms

Manage In-House Services

Reason and Prerequisites

Method map_ihr_doc_flow_ico in class CL_CRMS4_1O_API_MAP_ADAPTER contained an error.

Solution

Implement this SAP Note to correct the behavior of method map_ihr_doc_flow_ico in class CL_CRMS4_1O_API_MAP_ADAPTER.
