Symptom

You use SAP GTS to send an electronic message from a customs export declaration to the Dutch customs system, DMS.

You attach a document to the item in the declaration.

You receive a rejection (IE556) from customs with the error: "INVALID VALUE USED."

Other Terms

Customs Management, CuMa, customs declarations, DMS, belastingdienst, customs, export, item, documents, it_do

Reason and Prerequisites

Program error.

Rule BG0642 was updated and is now not valid for documents on item level.

Documents now stay on the item level and are not moved to the header level.

Solution

Implement the correction instructions.
