Symptom

There is a requirement to extract business partner website URL using a single or a combination of CDS views

Environment

S/4HANA Cloud Public Edition

Reproducing the Issue

Not Applicable

Cause

System design

Resolution

There is currently no single CDS view for this purpose at the moment.

However by using the combination of the CDS views below there is a solution:

  1.  Retrieve the standard address ID using CDS View I_BPCurrentDefaultAddress.
  2.  Create an association to I_AddressURI by passing the above Address ID, with the filter URLIsCurrentDefault = 'X’ and include the field UniformResourceIdentifier in the field list.
