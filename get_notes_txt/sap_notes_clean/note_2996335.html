Symptom

The program FINS_MIG_PRECHECK_CUST_SETTNGS does not handle the severity 'Error skippable' correctly.

Other Terms

Pre-check, SIC, Simplification Item Check

Reason and Prerequisites

The severity of some prechecks was changed to 'Error skippable'. This severity was not regarded in report FINS_MIG_PRECHECK_CUST_SETTNGS yet.

Solution

Implement the corrections of this SAP Note or apply the corresponding support package.
