Symptom

  • The Growth Portfolio export displays blank values in the Proficiency Level field for certain records.
  • The system shows the message: “Proficiency level hasn’t been assigned for this attribute yet.”

Environment

SAP SuccessFactors Talent Intelligence Hub

Reproducing the Issue

  1. Login to the instance.
  2. Proxy as the affected user.
  3. Navigate to Growth Portfolio.
  4. Search for the affected attribute.
  5. Observe the message: “Proficiency level hasn’t been assigned for this attribute yet.”
  6. Perform a Growth Portfolio export and observe blank values in the Proficiency Level field for the affected records.

Cause

The issue occurs because the affected user has not been assigned a proficiency level for the specific attribute in the Growth Portfolio.

Resolution

  1. Assign a proficiency level for the affected user in the Growth Portfolio to ensure the rating displays correctly.
  2. Verify the proficiency level assignment by performing a Growth Portfolio export and confirming the values appear correctly.
