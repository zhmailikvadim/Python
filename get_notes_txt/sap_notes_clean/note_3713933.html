Symptom

Additional Application Server isolation is required in some certain situation during upgrade procedure.

Environment

SAP ABAP System

Cause

Additional Application Server isolation is required.

Resolution

Refer to the following web page and isolate certain Application Servers through logon group configuration.

https://help.sap.com/docs/SLTOOLSET/a182800ba0f943c8a466c10c28175fff/22a0ea62f13f4181b4fc9acb550d4317.html

For specific users who connect to Application Servers directly, they should change the direct server or use logon group.

If any RFC users are connected without logon group configuration, check the column "client host"  in transaction AL08 to identify the client programs.

Keywords

AAS, Additional Application Server, isolation, isolate, SUM, software update manager, application instance
