Symptom

  • Linked user missing when login using UID
  • Linked account missing using UID login
  • Not all linked accounts appear when login with UID
  • Only some of the linked accounts can be selected with UID logon
  • Cannot select some of the accounts with UID login

Environment

SAP Universal ID (UID)

Reproducing the Issue

  1. There are multiple users linked with the UID email account
  2. When logging in with UID, only some of the linked accounts (not all) can be seen and selected from the account selection page

Resolution

Universal ID is deprecated as authentication method.

When user tries to login with email address and password, SAP ID will show the list of users/accounts with the same email and password.

Linked accounts with different email address or password will not be shown in the account selection page.
