Symptom

Is the SQL Anywhere OEM edition a separate download?
The SQL Anywhere OEM Edition is not in the SAP download portal.

Environment

SAP SQL Anywhere (any supported versions)

Resolution

SQL Anywhere package can be downloaded from SAP download portal. This package is common to all editions of the product. In fact, it is the license key that, entered during the installation, enables the components / add-ons specific to each and every edition.

Keywords

Authenticated, Sybase, authentication
