Symptom

Within Last Mile Distribution, you are working with the Route Overview app. You retrieve a high volume of routes which leads the system to generate dumps that are shown in transaction ST22. The system identifies the outer exception of the dump as type CX_SADL_DUMP_DATABASE_FAILURE for development object SRVD UI_LE_LMD_SETTLMT_OVW. The exception CX_SADL_DUMP_DATABASE_FAILURE is caused by the inner exception CX_SY_OPEN_SQL_DB that raises an error when converting a database field to the target field.
As a result, in the Route Overview app, tiles do not display any data. The issue affects in particular the tiles that show the settlement status overview and the route life cycle status overview.

Other Terms

LMD, Route Overview, Dump, F4743

Reason and Prerequisites

An integer overflow occurred during a SQL SUM() aggregation.

Solution

Implement the attached correction instruction to resolve the described issue.
