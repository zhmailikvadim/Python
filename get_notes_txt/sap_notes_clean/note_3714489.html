Symptom

When an account maintenance document is generated via BAPI_GRIRDOC_CREATEMULTIPLE (BAPI is called externally—for example, through the Fiori application Intelligent Goods and Invoice Receipt Account Reconciliation (F3302))—the corresponding FI document header is created without a transaction code (TCODE); the BKPF-TCODE field remains blank.

Other Terms

Account maintenance, F3302, Intelligent Goods and Invoice Receipt Account Reconciliation, TCODE, MR11, BAPI_GRIRDOC_CREATEMULTIPLE.

Reason and Prerequisites

Reason

Program Error.

Prerequisites

Account maintenance document created by BAPI_GRIRDOC_CREATEMULTIPLE.

Solution

The corrections shall be delivered with the specified support packages. For an advance correction, kindly implement the attached correction instructions.
