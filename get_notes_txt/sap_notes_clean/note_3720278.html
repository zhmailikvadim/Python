Symptom

  • While executing the Edit/pencil action in Job Info beside the position field, when clicking on the Position Org Chart icon, it pops up behind the current screen and is not accessible. After closing the "Change Job and Comp" dialog box, it reveals the Position Org Chart.
  • The issue is happening via All Actions > "Change Job and Comp" action, also after performing the above action.
  • Users are unable to view/access the Position Org Chart while executing actions.

Environment

SAP SuccessFactors Employee Central - Job Information

Reproducing the Issue

  1. Navigate to the employee profile.
  2. Execute the Job Information Edit/Pencil or Take Action ->"Change Job and Comp Info" action.
  3. Enter the Event Date to navigate to the screen requiring Position updates.  
  4. Click on the Position Org Chart icon beside the Position Field.  
  5. Observe that the Position Org Chart dialog box opens behind the current screen and is inaccessible.

Resolution
