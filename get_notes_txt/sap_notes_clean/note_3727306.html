Symptom

When accessing a Custom MDF Object in Manage Data, loading times are excessively long.

Environment

SAP SuccessFactors HCM Core

Reproducing the Issue

  1. Navigate to Manage Data.
  2. Search for a Custom MDF object and select it.
  3. Observe long loading times.

Cause

This is expected behaviour.

Resolution

  • If the number of child records exceeds the expected limits, it will result in slower loading times.
  • For more information regarding guidelines and limits, please refer to this Help Guide: Important Configuration Guidelines and Usage Limits | SAP Help Portal.
