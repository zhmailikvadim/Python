Symptom

  • Schedule Publication fails with error "Scheduling the publication failed because the export task could not be completed in time. See Details (Error Code: eSE112)." for the story based on live connection to SuccessFactors.
  • Generation Status: The publication failed because the export task could not be completed in time. See Details (Error Code: eSE112). 
  • Delivery Status: The publication could not be delivered because the publication generation was unsuccessful. 

Environment

  • SAP Analytics Cloud 2025.21.11
  • SuccessFactors

Reproducing the Issue

  1. Log in to SAP Analytics Cloud.
  2. Open a story based on live connection to SuccessFactors.
  3. Run Schedule Publication
  4. The publication failed with error: The publication failed because the export task could not be completed in time. See Details (Error Code: eSE112). 

Cause

  • This is by design.
  • Scheduling Publication is not supported for live SuccessFactors data sources.

Resolution

  • Refer to Help Guide for data sources that supported in schedule publication.
