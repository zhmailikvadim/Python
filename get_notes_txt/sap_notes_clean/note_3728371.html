Symptom

When checking the CIG log, I see a 503: Max Transacted Session Exceeded error.

Environment

SAP Ariba Sourcing

SAP Ariba Buyer

SAP Integration Suite, managed gateway for spend management and SAP Business Network ISMG

Reproducing the Issue

The issue occurs intermittently during periods of high system load when many transactions are being processed at the same time.

Cause

The 503: Max Transacted Session Exceeded error occurs when the maximum number of allowed concurrent transacted sessions is exceeded on the system. This happens when too many requests are processed simultaneously, temporarily preventing new sessions from being created.

Resolution

This is a temporary issue caused by high transaction volume. Wait for the system to process the current high volume of transactions.The error will resolve itself, and pending documents will be processed automatically once system load decreases.

Keywords

Integration, failure, stuck, document, transaction, performance, load, throttling, busy, gateway
