Symptom

You plan to perform an Support Package Stack or Enhancement Package update with the SUM tool and plan to include the uninstallation of Add-On products in this update process.

Other Terms

Add-On Uninstallation, SUM, SAINT

Reason and Prerequisites

The inclusion of Add-On uninstallations into a SUM process is currently only possible when performing an system upgrade. The support of the inclusion of Add-On uninstallations into other SUM processes like Support Package Stack or Enhancement Package update requires changes and extensions in the uninstallation preparation function.

Solution

This functional gap is closed with SPAM/SAINT Version 0071, so please apply a SPAM/SAINT Update with version 0071 or higher.

As long as this SPAM/SAINT Version is not available yet, import the previous SPAM/SAINT Update - Version 0070 and apply the attached correction instruction with the Note Assistant.

Note the following: When you have implemented the correction instruction, you can reset the note to original in transaction SPAU or remove it from the modification browser (transaction SE95) after you have imported SPAM/SAINT Version 0071. Ensure that no objects are changed when you remove the note. To do this, deselect all objects in the traffic light dialog box.
