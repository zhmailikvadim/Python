Symptom

Runtime error occurs when report /PF1/R_IDOC_RESTART is executed to restart IDOC process.

Other Terms

 /PF1/CL_IPM_P_IDOC, IDoc, /PF1/R_IDOC_RESTART, REGEX_TOO_COMPLEX

Reason and Prerequisites

Converting a string to an xstring for large data volumes during XML object preparation leads to an error occurs due to codepage conversion.

Solution

IDoc data is passed in raw format to avoid code-page conversion.

Please apply SAP Note or assigned Support Package.
