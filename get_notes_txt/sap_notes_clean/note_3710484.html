Symptom

Error message VL559-"Delivery &1 does not exist in the database or in the archive" is issued when attempting to open deliveries using transaction VL02N.

Environment

  • Sales And Distribution (SD)
  • Logistics Execution (LE)
  • SAP S/4HANA
  • SAP S/4HANA Cloud 
  • SAP ERP Central Component
  • SAP ERP
  • SAP Enhancement package for SAP ERP
  • SAP Enhancement package for SAP ERP, version for SAP HANA

Reproducing the Issue

  1. Attempt to open deliveries using transaction VL02N.
  2. Encounter error message VL559.

Resolution

In standard, already archived deliveries cannot be deleted from system.

See Also

If the delivery has been linked to sales order and you have confirmed that the delivery has indeed been archived, the message indicates that the business process for these deliveries is complete. The correct business process is to archive the corresponding sales orders instead of attempting to reuse them, as the deliveries have already been completed and fully delivered.
