Symptom

If a project is created with a project definition ID longer than 20 characters, its background task's processing status remains ‘Initiated’ due to the failure in creating the WBS element. The similar issue is observed when project coding mask is maintained for project definition IDs.

Other Terms

MMA, Project Definition, WBS Length, Coding Mask

Reason and Prerequisites

Program Error

Solution

Implement this SAP Note to fix the above mentioned issues.
