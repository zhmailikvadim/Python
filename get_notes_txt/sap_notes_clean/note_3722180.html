Symptom

  • User needs to create the Work zone, Standard Edition as a target system in IPS following the document
    SAP Build Work Zone, standard edition | SAP Help Portal
  • User is unable to view the Workzone service instance or unable to download the service key

Environment

SAP Build Work Zone, Standard edition

Cause

The user may not be assigned as the org manager for the Cloud Foundry (CF) org, or the user lacks the required Space Developer role.

Resolution

  • If the user is unable to see the Work zone service instance, check the KBA 3722188 - Workzone Service Instance Not Visible Despite Administrator Access
  • If the instance is visible but unable to view or download the service key, follow the steps in the KBA 3316378 - Users are Unable to View Contents of an Existing Service Key

Keywords

Work zone instance, service key, org member, space, space developer, administrator, access, IPS, target system
