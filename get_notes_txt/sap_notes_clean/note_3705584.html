Symptom

In the SAP Financial Consolidation (SAP FC) Client, the columns “Date Created” and “Last Changed” display timestamps based on the local desktop (client machine) time zone.

Environment

SAP Financial consolidation 10.1 

Reproducing the Issue

  1. Login to the SAP FC Desktop Client.
  2. Open the Security view, add the columns  “Date Created” and “Last Changed”

Cause

This behavior is by system design in SAP Financial Consolidation. The SAP FC Client renders date and time fields according to the local operating system time zone of the user’s desktop.

Resolution

This behavior is by design and cannot be changed through standard configuration in SAP FC. There is no supported option to force the display of “Date Created” or “Last Changed” columns to a unified server or UTC time zone in the SAP FC Client.
