Symptom

 During navigation from app "Display Supplier Balances" on values of comparison year "Purchases" column on "Compare" tab to app "Manage Supplier Line Items" the "Sales-Related Item" Indicator is not set to "Yes"(true) in the filter bar.

Solution

Implement the attached correction instructions or import the specified support package.
