Symptom

  • Incorrect data has been retrieved in SAP BO from SAP Datasphere.  

Other Terms

web intelligence, datasphere, refresh

Reason and Prerequisites

Environment:

  • SAP BusinessObjects Business Intelligence Platform SP4 Patch 17. 
  • Datasphere
  • Web Intelligence
  • Windows 

Steps to Reproduce:

  1. Log in to BI LaunchPad.

  2. Open the affected Web Intelligence report.

  3. Refresh the report using the required prompt values.

  4. Observe the data returned in the Web Intelligence report.

  5. Next, open the corresponding Analytical Model in SAP DataSphere.

  6. Compare the data displayed in BI LaunchPad with the data in SAP DataSphere.

  7. When comparing results, the data shown in BI LaunchPad differs from the data in SAP DataSphere.
