Symptom

  • “Apply changes in all queries sharing the same data source ” Option is Disabled
  • Issue is observed only when Web Intelligence Document is based on OLAP connection (SAP HANA and OLAP BICS)

Other Terms

webi, webintelligence, HANA, Change source, empty, BICS, bex, universe, Queries, blank, predefined

Reason and Prerequisites

Environment:

  • SAP BusinessObjects Business Intelligence Platform 4.2 Support Package 07
  • Web Intelligence
  • OLAP connection (OLAP BICS)
  • SAP HANA
  • Bex

Steps To Reproduce:

  1. Open a Web Intelligence report with Multiple Queries in it coming from same Data Source 
  2. Make sure that data source is using an OLAP connection
  3. Switch to Design mode
  4. Select Data access tab, Tools then Change Data Source
  5. Select any query, we will observe: 

>> Choose existing data source is blank
>> Apply changes in all Queries sharing the same data source” option is grayed out.

Cause:

  • This issue is caused by a defect in the product.

Solution

This issue has been fixed using SAP Note 2878713
