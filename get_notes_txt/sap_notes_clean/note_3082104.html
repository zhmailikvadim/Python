Symptom

You adjust the column width in the My Activities report. System does not keep the change when you re-open the report.

Reproducing the issue

Prerequisites:

You are working with SAP Business One 10.0.

Scenario:

  1. Go to Reports → Business Partners → My Activities.
  2. Drag the column and make the column width wider.
  3. Re-open the My Activities report.

Expected behavior:

The modification of the column width remains.

Actual behavior:

System resets the column width to the original.

Solution

SAP intends to provide a feature package or patch to solve the symptom that is described in this SAP Note. The section References below lists the related feature package or patch once they become available. Be aware that References can only be confirmed at feature package or patch release date. SAP delivers feature packages or patches only for selected releases at its own discretion, based on the business impact and the complexity of implementation.

Workaround:

To adjust the column width, use UI Configuration Template in the My Activities window.
