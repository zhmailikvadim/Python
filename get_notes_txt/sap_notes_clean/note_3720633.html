Symptom

When executing transaction FBL1N for multiple vendors, the header shows an asterisk (*) instead of full information.

The images and data in this KBA are taken from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental. 

Environment

  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA

Reproducing the Issue

  1. Execute transaction FBL1N.
  2. Select multiple vendors during execution.
  3. Observe that the header fields display an asterisk (*) instead of full information.

Cause

Result is output in ALV Grid display:

Resolution

Switch from ALV Grid display to ALV List display using the menu option "Settings - Switch list".
