Symptom

The Copy Financial Plan Data Fiori app does not provide an option to scope the copy action to a specific project or WBS element.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Open the Copy Financial Plan Data Fiori app in SAP S/4HANA Cloud Public Edition.
  2. Attempt to copy financial plan data using the app.
  3. Observe that there is no option to limit the copy action to a specific project or WBS element.

Cause

Resolution

In SAP S/4HANA Cloud Public Edition, the Copy Financial Plan Data Fiori app copies plan data based on the selected source and target plan categories. While you can use filter fields like ledger, fiscal year, or company code to narrow the dataset, the app does not expose a Project ID or WBS element field to limit the copy to a specific project or WBS object. Therefore, the copy action operates over all plan records in the chosen plan category matching your filter selections, not just a single project/WBS.
