Symptom

After implementing the SAP Note 3206411 "BASIS - Hierarchy Adaptations", differences in the hierarchy table are not displayed anymore in the  “Collected Payments” app (Fiori ID: F4747).

Other Terms

LMDRTCOLPAYS1, F4747, Collected Payments, Differences, Hierarchy Table, SAP Note 3206411

Reason and Prerequisites

You have implemented the SAP Note 3206411- "BASIS - Hierarchy Adaptations". Otherwise, differences in the hierarchy table are displayed correctly.

Solution

To display differences in the hierarchy table, implement the attached corrections.
