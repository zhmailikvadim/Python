Symptom

When you run the integration for SV as PEO Version BOM and send an empty "transformationMatrix" for the component, the system stores an instance with initial geometric coordinates.

Other Terms

PLMSI, PLM System Integration, SV, SVasPEOVersionBOM, SAP_SVasPEOVersionBOM, Version BOM, SVV, transformationMatrix, DMUTMX

Solution

Implement this SAP Note to prevent the system from storing data with empty "transformationMatrix" by returning an error.
