Symptom

The customizing configuration for complaint code list G_0081 is overwrited by SAP BC set, so that customer need to update customizing configuration manually.

Other Terms

SUP, Gas Grid Usage Billing, Code List G_0081, Reason Code, Automatic Invoice Document Complaint, Suppress, /US4G/V_COMPL_R

Reason and Prerequisites

Configuration errors

Solution

Implement the corrections of this SAP Note or import the relevant support package.

This issue is relevant for the market role: SUP.

After updating customizing configuration manually, you can complain invoice document for gas grid usage billing, and active the automatic invoice document complaint and suppress functionality.
