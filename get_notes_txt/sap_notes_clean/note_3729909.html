Symptom

While enabling Assess Job Seeker Resumes Using AI to any Child Site error is thrown:

 'Assess Job Seeker Resumes Using AI' Site Configuration cannot be enabled as it is currently disabled on Parent Site

Resolution

If 'Assess Job Seeker Resumes Using AI' is enabled on company config level at a later stage, Parent Sites have this option flagged to "No" and it is not editable from UI

How to enable option 'Assess Job Seeker Resumes Using AI' on Parent Site:

This can be done with connector Site Configuration Upload

Please find a link for the connector below:

Site Configuration Upload

Once the Parent Site has option 'Assess Job Seeker Resumes Using AI' enabled, a user with permission "Manage Site Configurations" ticked on User level can map this option on Child Sites as needed 

  1. Navigate to the Child Site
  2. Click the 3 dots on top right corner
  3. Select "Edit Configuration"
  4. Scroll to the option > Tick to Enable or Disable
  5. Enter Reason > Update
