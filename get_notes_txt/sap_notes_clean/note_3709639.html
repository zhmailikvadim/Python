Symptom

Rehire with New Employment inactive profile is showing in global search

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental. 

Environment

SAP SuccessFactors Onboarding 2.0

Reproducing the Issue

  1. Search for the rehire user in Global Search
  2. Rehire with New Employment Inactive Profile is available in search.

Resolution

Rehire with Old Employment Inactive Profile is available in global search. It is expected standard behavior.

Rehire with New Employment Inactive Profile is also available in global search. To avoid getting the inactive profile in global search follow the steps as mentioned.

1) Search for Company System and Logo Settings.

2) Disable the checkbox "Show inactive user in people search (requires Role-based Permissions)  ".
