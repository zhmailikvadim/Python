Symptom

When using a mobile browser with the Latest Home Page Experience enabled, tapping "All" in Organizational Updates section opens the second page as expected. However, tapping "All" again to return to the Home Page causes the Organizational Updates section to disappear.

“Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Environment

SAP SuccessFactors HCM Suite

Reproducing the Issue

  1. Log in to SuccessFactors using a mobile browser. 
  2. From the Home Page, go to the Organizational Updates section.
  3. Tap the "All" button to navigate to the second page.
  4. On the second page, tap the "All" button again to return to the Home Page.
  5. Observe that the Organizational Updates section is no longer visible.

Resolution

The fix for this issue is planned to be deployed on 1H 2026 release.

For release timelines, please review SAP SuccessFactors Product Release & Road Map Information.

Workaround:

  • On the second page, tap the "X" (close) button instead of "All". You will be returned to the Home Page and the Organizational Updates section will remain visible.
  • Alternatively, refresh the Home Page after returning. The Organizational Updates section should reappear.
