Symptom

You open the integration framework 2.0 version and want to display the documentation.

Solution

The documentation is available with the delivery, but must be imported into the BizStore.

  • To open the integration framework 2.0, use the following access address: <server>:<port>/B1iXcellerator/exec/dummy/com.sap.b1ip.system.cc/bfd/AdminConsole.bfd?!defdoc=/com.sap.b1i.common/menu/opdev.xml
  • To import the documentation, choose Maintenance → ZIP Import and select the following in the Choose Archive field:
    • For the documentation
      ...\Program Files\SAP\SAP Business One Integration\IntegrationServer\Tomcat\webapps\B1iXcellerator\B1iFDocs\DEVGenDocs.zip
    • For adapter information:
      ...\Program Files\SAP\SAP Business One Integration\IntegrationServer\Tomcat\webapps\B1iXcellerator\B1iFDocs\DEVAdapterDocs.zip

  • Documentation about how to build an integration framework Docker image and install it on Docker is attached to the note.
