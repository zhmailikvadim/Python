Symptom

There is no option to retrieve the statement once the price review document is posted. The client wants to view statements for price review documents after posting. However, statements are only available for price review documents before posting.

This SAP Note resolves the issue by enabling the retrieval and display of statements for price review documents even after they are posted.

Other Terms

/IRM/IPPDM, /IRM/LIPPDMF0D, DOCUMENT_METADATA_PREPARE

Reason and Prerequisites

This is caused by program error.
Note : SAP Note#3708982 is available for ECC release.

Solution

Please implement the correction instructions. These corrections are only valid if you have applied the following Support Packs.

Release V/4 – 2021, 2022, 2022 V2, 2023, 2023 V2, 2023 V3 Base Package or Support Pack 1.

Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object
