Symptom

An error occurred in step 1100 of the process "Receipt of Meter Reading Request (MOS MaLo View)" (DE_C_MET_MRR_MALO) on the Cloud side and an exception document has been generated .

Error Message: Exception EX_CP_RECEIVER_AGENCY_NUMBER_NOT_FILL raised

This issue is relevant for the market roles: MOS.

Other Terms

Meter Reading Request, Supply scenario, MeLo, MaLo, Class /US4G/CL_PRST_MRR_GET_MRV, DE_B_MET_MRR_RCV_MOSMELO, DE_C_MET_MRR_MALO

Reason and Prerequisites

The error occurred because the system was unable to obtain complete supply scenario data in step 0500 of the process "Meter Reading Request (MOS MeLo) (DE_B_MET_MRR_RCV_MOSMELO)" on the Back end side, which caused an error in step 1100 of the process "Receipt of Meter Reading Request (MOS MaLo View)" (DE_C_MET_MRR_MALO) on the Cloud side.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the system obtains complete supply scenario data and sends it to the Cloud side. The process "Receipt of Meter Reading Request (MOS MaLo View)" (DE_C_MET_MRR_MALO) now works correctly on the Cloud side.

Note: Existing process documents with this error can't be reprocessed as completed. Select the "Cancel Process" solution for the exception document.

You can verify the fix by checking the following:
- The process "Meter Reading Request (MOS MeLo)" (DE_B_MET_MRR_RCV_MOSMELO) is completed in the back end, and the UBD of step 0500 (Read Meter Reading Result) is correct.
- The process "Receipt of Meter Reading Request (MOS MaLo View)" (DE_C_MET_MRR_MALO) is completed on the Cloud side, and the status of step 1100 is completed.
