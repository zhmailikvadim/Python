Symptom

When updating info records with offline mode in Mass Changes to Info Records app, warning message AM483 shows but there is no further information about this message.

Environment

  • Materials Management (MM)
  • SAP FIORI
  • SAP S/4HANA

Reproducing the Issue

  1. Upload Excel file for any PIR in Mass Changes to Purchasing Info Records app.
  2. Execute the process and check the log of the background job.

Cause

Warning message AM483 occurs due to the absence of comment maintained for the supplier in transaction code BP under the "Address Overview" tab.

Resolution

To avoid message AM483, go to t-code BP -> Search the supplier -> Select the entry under "Address Overview" tab -> Click "Choose" button to check the details -> Fill anything in the comments field -> Save. 

See Also

This message is also triggered in SAPGUI transaction code, for example ME11, ME12 and ME13, in the background. However, it does not show in the screen. 

Keywords
