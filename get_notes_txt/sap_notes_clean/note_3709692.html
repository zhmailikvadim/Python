Symptom

Trying to restore Hana DB by using Hana Cockpit but not able to see the option for restoring it.

Environment

SAP HANA, platform edition

Reproducing the Issue

  1. Open Hana Cockpit
  2. Choose the Database under Database Directory
  3. Select the Database Backup tile

Cause

The restore should be launched from SYSTEMDB, instead of tenant DB.

Resolution

From the system database, choose the Database Management, then select the Tenant database and then choose Tenant Actions > Recover Tenant.

Refer to the below site, for how to recover the SYSTEMDB and the Tenant:

Recovering an SAP HANA Database
