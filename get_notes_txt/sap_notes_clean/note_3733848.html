Symptom

  • Change source of Web Intelligence document from UNV to UNX fails with Error: “An internal error occurred while calling 'processDPCommandsEx' API. (Error: ERR_WIS_30270)." 

Other Terms

processDPCommandsEx, ERR_WIS_3027, UNV, UNX, custom SQL, Change source, internal error

Reason and Prerequisites

Environment:

  • SAP BusinessObjects Business Intelligence 4.3 Support Package 03 Patch 300
  • Web Intelligence
  • Universe (UNX)

Reproducing the Issue:

  1. Log in to BI laucnhpad.
  2. Create/open a Web Intelligence report built on Universe (UNV) and define a Custom SQL for this report.
  3. In Information Design Tool, convert the UNV universe to UNX universe.
  4. In BI launchpad, change the source of the Web Intelligence report from UNV to UNX Universe.
    >> Noticed it fails with the following error is encountered:
    "An internal error occurred while calling 'processDPCommandsEx' API. (Error: ERR_WIS_30270)"
