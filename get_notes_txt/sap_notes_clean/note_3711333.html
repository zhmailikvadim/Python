Symptom

3/16/2026: The manual correction instructions for the HR Forms remuneration statement have been enhanced so that a '-' must be specified as the arithmetic operator for the assignment of wage types to cumulation wage types (point 1. b).

3/6/2026: The manual correction instructions for uploading PE51 forms have been enhanced so that certain displayed error messages can be ignored.

This SAP Note provides the display of the active pension on the remuneration statement.

The function for settling the capitalized annuity was provided with SAP Note 3669507 - Asset-side annuity: Delivery of the function.

Other Terms

SAP_PAYSLIP_DE_O, DF01, DFB1, DFKA

Reason and Prerequisites

Law for the Tax Promotion of Employees at Pension Age (Active Pension Act)

Solution

Implement the attached correction instructions or import the specified support package.

  • In the remuneration statement, in the section "Remuneration Elements" or 'Basic Pay', specify the amount of the tax-exempt active pension for non-recurring payments and regular payments.

After you import the HR Support Package or implement the correction instructions, you must perform the following activities.

  • Implement the changes described in the manual correction instructions in your own forms.
