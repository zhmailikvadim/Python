Symptom

In the Fiori application F2795 - Task Logs, when a value using a range of special periods from 013 to 016 is manually typed into the "Fiscal Year Period" value help, an error message appears, preventing the user from using this value to filter records.

Other Terms

FINCS_FISCAL_YEAR_PEROID, I_FiscalYearPeriod

Solution

Import of Support Package or advanced correction by implementation of correction instruction.
