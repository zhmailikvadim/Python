Symptom

When executing FBL*N reports, below dump happens
Category               ABAP Programming Error
Runtime Errors         SYNTAX_ERROR
ABAP Program           SAPDBKDF
Application Component  Not assigned

In PERFORM or CALL FUNCTION "WRITE_INTO_BSIK", the actual parameter "YBSAK" is incompatible with the formal parameter "I_BSIK".

Environment

SAP ERP

Reproducing the Issue

Execute FBL*N and dump happens

Cause

Table BSIK and BSAK have different fields.
The appended fields between those 2 tables are different

Resolution

Correct the appended fields between those 2 tables and make sure the fields contained in those 2 tables are the same

Keywords

SYNTAX_ERROR, SAPDBKDF, BSIK, BSAK, different fields, FBL1N, FBL3N, FBL5N.
