Symptom

You want to update the SAP HANA XS Advanced Cockpit

Other Terms

XSA Cockpit, SAP HANA Extended Application Services, advanced model, administration

Reason and Prerequisites

Download the installation archive from the SAP Support Portal.

Further reading: Maintaining the XS Advanced Run-time Environment with a Graphical User Interface

Solution

Changes

This release of the XS Advanced Cockpit contains the following changes:

  • Added a new page Global Environment to manage global staging and running environment variables.

Installation Instructions

  • Log-in at XS Advanced as XSA_ADMIN user with the xs cli and target the SAP space
  • Call the xs cli command

    xs install <installation-archive> 

    to start the installation.
