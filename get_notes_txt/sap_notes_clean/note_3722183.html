Symptom

When I create goods receipt in SAP ERP via transaction code MIGO, no Ariba output message is generated. Thus no goods receipt is sent to SAP Ariba solutions.

Cause

The Collective Slip checkbox is not checked during goods receipt creation.

Resolution

Make sure the Collective Slip checkbox is checked by verifying the following parameter is defined in SAP ERP:
1. Go to transaction code SU01. 
2. Enter value into User field and display the details.
3. Go to Parameters tab.
4. Make sure parameter NDR is defined with X value.
