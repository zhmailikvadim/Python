Reason and Prerequisites

This is a program error.

Solution

Once the corrections are applied, you must reactivate and regenerate the logical Index of the HCPR.

  • SAP BW 7.50
    Implement Support Package 14 for SAP BW 7.50 (SAPK-75014INSAPBW) into your BW system. The Support Package will be available as soon as SAP Note 2668631 with the short text "SAPBWNews 7.50 BW ABAP SP14", which describes this Support Package in more detail, is released for customers

  • SAP BW 7.51
    Implement Support Package 8 for SAP BW 7.51 (SAPK-75108INSAPBW) into your BW system. The Support Package will be available as soon as SAP Note 2671143 with the short text "SAPBWNews 7.51 BW ABAP SP8", which describes this Support Package in more detail, is released for customers. 

  • SAP BW 7.52
    Implement Support Package 3 for SAP BW 7.52 (SAPK-75203INSAPBW) into your BW system. The Support Package will be available as soon as SAP Note 2613219 with the short text "SAPBWNews 7.52 BW ABAP SP3", which describes this Support Package in more detail, is released for customers.

  • SAP BW 7.53
    Implement Support Package 1 for SAP BW 7.53 (SAPK-75201INSAPBW) into your BW system. The Support Package will be available as soon as SAP Note 2653494 with the short text "SAPBWNews 7.53 BW ABAP SP1", which describes this Support Package in more detail, is released for customers.

  • SAP BW/4HANA 1.0
    Implement Support Package 10 for SAP BW/4HANA 1.0 (SAPK-10010INDW4CORE) into your SAP BW/4HANA system. The Support Package will be available as soon as SAP Note 2646765 with the short text "SAPBWNews SAP BW/4HANA 1.0 SP10", which describes this Support Package in more detail, is released for customers. 

 

In urgent cases you can use the correction instructions.

Before you use the correction instructions, make sure that you check SAP Note 1668882 and SAP Note 2248091 for transaction SNOTE.

This SAP Note might already be available before the Support Package is released. In this case, however, the short text still contains the term "preliminary version".
