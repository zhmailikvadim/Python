Symptom

You notice the announcement about the integration of Notification Management integration with User Management.



"This app will soon be deprecated. The notification recipients will now be available in the User Management app.
You must have the User Administrator role in the User Management app to add and modify users and distribution lists. For more information, refer Application help - User Management ."

**Notification Management migration to User Management has been delayed and will take some time due to some architecture challenges,  It is recommended for customers to continue using Notification Management application and when the migration is ready and done it will be communicated explicitly**

Disclaimer: "Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Environment

  • SAP Cloud ALM 1.0

Reproducing the Issue

  1. Access SAP Cloud ALM launchpad
  2. Select Operation group
  3. Click on Notification Management tile

Resolution

The integration of Notification management with User Management and subsequent phase out of Notification management is planned for Year 2026.
Since it is a critical topic for future and it's important to inform customers early, the information details are attached to Notification management tile.
If there is any progress, this KBA will be updated accordingly.
