Symptom

The following symptoms occur on standalone Web Dynpro ABAP / FPM calls or when the applications are embedded in NWBC, Enterprise Portal or Fiori Launchpad:

  • A floorplan page is rendered in only 50% of the available space of the browser window,
  • There is an unnecessary vertical scroll bar added as well:

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Environment

  • Floorplan Manager for Web Dynpro ABAP
  • Fiori Framework Page
  • Web Dynpro for ABAP
  • SAP NetWeaver 7.50 - SAP_UI SAPK-75005INSAPUI or below

Reproducing the Issue

Start any FPM application:

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Cause

  • Program error

Resolution

Apply SAP Note 2361675 - OVP, GAF, OIF: Page is cutoff in the middle.
