Symptom

  • Icons in a list report remain static during scrolling, even when the associated cell values and colors change dynamically.
  • The issue occurs in the application using Fiori elements v4.
  • The problem persists regardless of the number of rows scrolled.

Environment

UI5 ABAP delivery tools

Reproducing the Issue

  1. Access the application using the Workzone Launchpad.
  2. Open the list report displaying order information.
  3. Scroll down the report.
  4. Observe that the icons remain static and do not update, while the associated cell values and colors change dynamically.

Cause

The issue is caused by a bug in SAPUI5 version 1.143 (a monthly Cloud only version), which affects the dynamic update of icons in tables during scrolling.

Resolution

The issue has been resolved in SAPUI5 version 1.144.0.

See Also
