Symptom

You run the Cálculo de nómina española (RPCALCE0) program for an employee with the following conditions:

  • Daily contribution group configured in the Seguridad Social (ES) (IT0061) infotype.
  • A Vacaciones no disfrutadas (VND) situation with an amount below the minimum monthly contribution limit that is generated for all February month.
  • The Topes AT/EP mens. para diarios (TOCPD) payroll constant set to '1' in the Constantes de nómina (V_T511K) view.

The system incorrectly calculates the Base Cotizada VND CP (/3VD) wage type below the minimum monthly contribution limit.

Example:   

Proceed with the following steps to reproduce the symptom mentioned in this SAP Note.

Disclaimer: this test case example aims to provide a better understanding of this SAP Note purpose. Please, take into account that some additional cases might need to be tested, considered, or impacted.

Description

Daily employee with 31 days of VND.

Preparation/Background

  1. Hire a daily employee.
  2. Set the TOCPD payroll constant to '1' in the V_T511K view.
  3. Terminate the employee in January 2025.
  4. Create a 40 days VND payment below the minimum limit starting on January 30, 2025.

Execution

Run the payroll for this employee for February 2025.

Result

The system generates the /3VD wage type below the minimum monthly contribution limit.

Other Terms

&CALC - PC00_M04_CALC

For more information, see SAP Note 1071282 - CONSULT: List of tags for Spain notes.

Reason and Prerequisites

Reason

This issue is a side effect caused by the solution delivered by the SAP Note 3475893 - CALC: VND amount below minimum for daily employees in 31-days months.

The solution delivered by SAP Note 3475893 does not consider the 28/29 days of February.

Prerequisites

The following Support Package level is the minimum prerequisite:

Release Support Package
S4HCM 102 HR SP 00
S4HCM 101 HR SP 06
S4HCM 100 HR SP 19
SAP_HR 6.08 HR SP D0
SAP_HR 6.04 (EhP6 and EhP7)  HR SP K2

For more information, access SAP ERP Human Capital Management - Support Package Schedules.

Note:

When you implement an SAP Note, some prerequisites might be required. The prerequisites are SAP Notes that have already been released and that change the same context as the current one. Then, even if these prerequisites are not necessarily related to the symptom of this SAP Note, you must implement them for technical reasons.

Solution

From now on, the system generates the CP VND wage types above or equal to the monthly minimum in February.

Example:

Following the steps from the example of the Symptom section.

Result

The system generates the wage type /3VD with the minimum monthly value.

Retroactivity:

To avoid problems with retroactivity, the changes were sheltered with the entry 'NOTE' '03721646' in the V_T5F99VD table view.
For further information, see SAP Note 515323 - CONSULT.: Protección de correcciones.
