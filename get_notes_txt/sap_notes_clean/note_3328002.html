Symptom

This note will change the navigation mode from OBN to Launchpad for some user interfaces (UI).

The related UIs and operations are as follows:

  • UI: Reconciliation Between Revenue Postings and General Ledger

        Operations: This displays the job information when you press the Run button.

  • UI: Transfer Revenue

        Operations: You can navigate to the Revenue Posting Jobs Monitor UI by pressing the Job Monitor button.

  • UI: Calculate Contract Liabilities and Assets

        Operations: You can navigate to the Revenue Posting Jobs Monitor UI by pressing the Job Monitor button.

  • UI: Revenue Posting Run - Posting Mode

        Operations: You can navigate to the Revenue Posting Jobs Monitor UI by pressing the Job Monitor button.

  • UI: Revenue Posting Reversal and Repost

        Operations:

            1) You can navigate to the Revenue Posting Jobs Monitor UI by pressing the Job Monitor button.

            2) You can navigate to the Revenue Posting Jobs Monitor UI by selecting the link type value Run ID in the results list of the Revenue Posting Reversal tab.

            3) You can navigate to the transaction code FB03 to display the document by selecting the link type value G/L Documents in the results list of the Revenue Posting Reversal tab.

            4) You can navigate to the Revenue Posting Jobs Monitor UI by selecting the link type value Original Run ID in the results list of the Revenue Posting Repost tab.

            5)You can navigate to the Revenue Posting Jobs Monitor UI by selecting the link type value Reversed Run ID in the results list of the Revenue Posting Repost tab.

  • UI: Revenue Posting Jobs Monitor

        Operations:

           1)You can navigate to the transaction code FB03 to display the document by selecting the link type value G/L Documents in the results list of the G/L Document tab.

           2)You can navigate to the transaction code FARR_ALL_DOC to display the list of all documents generated by one run ID, by selecting the link type value Run ID in the results list of the All Documents tab.

           3)You can navigate to the Revenue Posting Reversal and Repost UI by selecting the Reverse button in the result list of the All Documents tab.

After you have applied this note, you need to test these operations to ensure that all the Launchpad configurations are correct.

Also, if you have configured the System Alias for your system, you need to adjust it according to the operational configurations above to ensure that the System Alias value in your system configuration is the same as the one in the Launchpad configuration.

Other Terms

Revenue Accounting, Launchpad, Job Monitor, Transfer Revenue, Calculate Contract Liabilities and Assets, Revenue Posting, Revenue Posting Reversal and Repost, Reconciliation Between Revenue Postings and General Ledger, FB03, FARR_ALL_DOC, FARR_SHOW_JOB

Reason and Prerequisites

ADVANCE DEVELOPMENT

Solution

Apply this note and follow the steps in manual activity to complete the configurations of Launchpad
