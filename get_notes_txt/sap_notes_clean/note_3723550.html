Symptom

  • System doesn't perform any validation in date format when creating or changing an agreement using BAPI.
  • When an incorrect validity date format is provided, the system accepts it without raising an error.

Environment

  • Materials Management (MM)
  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERPSAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Cause

The BAPI does not include basic value validations like date checks, as these checks are expected to be performed by the caller side.

Resolution

Make sure that the basic validations like date checks are performed before calling the BAPI. For date checks Function Modules like DATE_CHECK_PLAUSIBILITY can be used.
