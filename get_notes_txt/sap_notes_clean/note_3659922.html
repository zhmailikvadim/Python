Symptom

When attempting to upload the XML file generated for VAT declaration to the Slovenian tax authority portal, an error occurs.

  • The error message states: "Name Cannot begin with '.' character, hexadecimal value 0x00. Line 1 Position 40"

Environment

  • SAP ERP
  • SAP S/4HANA

Reproducing the Issue

  1. Generate the DME file with the use of program RFUMSV00
  2. Once the DME file is generated, export it in an XML format using program RFASLDPC
  3. Attempt to submit the generated XML file to the Slovenian tax authorities

Cause

The issue arises because the generated XML file is not using the required UTF-8 encoding.

Resolution

  1. Access program RFASLDPC to download the XML file from the application server to the local PC.
  2. On the selection screen of the program, check the flag "Manual Entry of Code Page".
  3. Enter "4110" as the Target Code Page. Note that "4110" is the SAP equivalent of UTF-8.
  4. Download the file again and verify that the encoding is UTF-8.
  5. Upload the corrected XML file to the Slovenian tax authority portal.
