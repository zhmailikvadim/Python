Symptom

When accessing the People Profile tab in an ESM case, you notice the SAP SuccessFactors profile displayed belongs to the Agent instead of the Requester of the case.

Environment

SAP Enterprise Service Management

Reproducing the Issue

  1. Logon to the SAP Enterprise Service Management tenant.
  2. Navigate to Self Service work list.
  3. Open any case.
  4. Navigate to People Profile tab.
  5. See the SAP SuccessFactors profile displays the agent profile (logged in user in ESM) rather than the profile of the requester.

Cause

This might happen in case the employee is mapped to the mashup parameter in the Mashup settings itself, rather than in the Case Details UI via the Adaptation Mode.

Resolution

Follow the steps below to adjust the mapping:

  1. Navigate to Self Service work list.
  2. Open any case.
  3. Navigate to People Profile tab.
  4. Click on the User Profile.
  5. Click Start Adaptation.
  6. Edit the mashup properties in the UI.
  7. Select the employee/employeeDisplayId in the mashup parameter.
  8. Click Apply.
