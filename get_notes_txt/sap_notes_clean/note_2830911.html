Symptom

Admin is not able to change or select a different currency for costs while creating record learning - financial.

Environment

SAP SuccessFactors Learning

Cause

If enableCappingFactor is set to true, the learning event recorder to display system default currency instead of the drop-down.

By default, this property is set to false. Set it to true for French government reporting. 

Resolution

  1. navigate to System Admin > Configuration > System configuration;
  2. search for enableCappingFactor;
  3. change it to false;
  4. save changes.

See Also

SAP SuccessFactors Learning System Properties in LMS_ADMIN
