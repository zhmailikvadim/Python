Symptom

Starting / Stopping TREX instances from the AdminTool fails doesn't work. An error message is displayed in the MC tab of Lanscape -> Services: HTTP 415: Unsupported Media Type: HTTP Error: 'Unsupported Media Type'

Reason and Prerequisites

This is a bug in TREX/BWA and it is resolved in Rev.61 of TREX 7.10, Rev.32 of BWA 7.20, Rev.72 of BWA 7.00 and Rev.30 of TREX 7.25

Solution

You can either upgrade to TREX 7.10 Rev.61 / BWA 7.20 Rev.32 / BWA 7.00 Rev.72 / TREX 7.25 Rev.30 or use the attached python script and replace it into <SID>/SYS/exe/nuc/<platform>/trex/python_support (<SID>/SYS/exe/nuc/<platform>/python_support for BWA 7.00)
