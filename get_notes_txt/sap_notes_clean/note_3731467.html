Symptom

Material description field is not available in the 'Determine Inventory Balance Sheet Value' app.

Environment

SAP S/4HANA Public Cloud Edition

Reproducing the Issue

  1. Open the “Determine Inventory Balance Sheet Value” app and run the valuation with the desired selection criteria.
  2. Review the result list and observe that material description is not shown alongside the material number.

Resolution

  1. To execute with the 'Update Inventory Balance Sheet' flag
  2. Input the required details, such as materials and quantities ensuring all relevant data .
  3. Afterward, review the results in the 'Inventory Balance Sheet Values' app, where updated material descriptions, balances, and other relevant fields can be checked.
