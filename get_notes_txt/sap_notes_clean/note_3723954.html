Symptom

When you try to implement SAP Notes from SPAU or SNOTE, get below error message :

"Lock for R3TR applies to a request that does not exist or is empty

Message no. TK322".

Environment

SAP_BASIS release dependent systems.

Cause

The cause is unknown.

Resolution

Check table TLOCK, search the note number and TR number mentioned in Diagnosis part inside the TK322 error message.

You will see those TR and Note object exist in table TLOCK but there's no entry in table E070 which means the TR does (no longer) exist and the entries in TLOCK are orphaned data which need to be wiped. 

Follow the SAP KBA 2537981 - SE13 | TK117 | Request lock on nonexistent task/request , execute the attached program in the KBA to cleanup the data.
