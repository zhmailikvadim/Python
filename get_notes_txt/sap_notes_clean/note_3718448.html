Symptom

The system calculates bracket B of the "Contribution d'équilibre technique" (CET) up to nine times the Social Security ceiling instead of up to eight times the Social Security ceiling.

Other Terms

CET, RPCALCF0, contribution, Agirc-Arrco, FCCO

Reason and Prerequisites

This issue occurs due to an error in the calculation logic for the CET contribution brackets.

Prerequisite: Part-time employee contributing to Agirc-Arrco as a full-time equivalent

Solution

After implementing this SAP Note, the system calculates bracket B of the CET up to eight times the Social Security ceiling.

These corrections are available with the HR Support Packages referenced below.
If you have an urgent need of these corrections, see the correction instructions provided.
