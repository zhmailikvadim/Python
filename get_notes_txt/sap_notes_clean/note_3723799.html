Symptom

In transaction FEB_BSPROC you select a bank statement item and go to the "Assignment" tab. You use the "Read More Items" section to select additional items from more than one company code and click the "Start Search" button. For one of the selected company codes an error is raised. Possible are failed period checks, for example, error F5 201 or number range issues like F5 150. This error is justified. However, when you remove the affected company code from the selection and start a new search, the same error will be raised again. However, the affected company code is not relevant anymore and should not be checked.

Other Terms

FEB_BSPROC, BUKRS, AUSGLEICHSKONTO_LESEN, SAPMF05A, F5 201, F5201, F5150, F5 150

Reason and Prerequisites

Program error

Solution

Implement the attached Correction Instruction (CI) or the relevant Support Package.
