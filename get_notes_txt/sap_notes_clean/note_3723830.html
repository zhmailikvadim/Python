Symptom

Spot Buy catalog showing incorrect items or items that should be restricted.
Spot Buy catalog label (name) is showing incorrect value based on the user's country/location.
Spot Buy catalog results are not showing expected items

Reproducing the Issue

Trigger search in Guided Buying and review results page.
Guided Buying > Search > on improved search results page > "Spot Buy catalog" tab

Cause

Incorrect spot buy catalog results or/and label, can be caused by incorrect location filter on the search query.
This is caused by missing or incorrect Ship To assigned to affected user. When no ship-to address is assigned to user, system defaults to US location. This causes Spot Buy results and label returned for US country/location.
If incorrect Ship To is assigned, results and label is returned for assigned Ship To country/location.

Resolution

User's Ship To assignment needs to be reviewed and corrected.
This can be managed via:
Core administration > User Manager > Other User info > Ship To

NOTE: No Ship To assigned means that GB Spot Buy search will be set to default location (e.g. US)
