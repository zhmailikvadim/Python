Symptom

The following errors occur when processing infotype 0657 via the renewal screen:

  • ABAP dump with the error message "CX_HRPA_VIOLATED_POSTCONDITION" during maintenance operations (Create/Change).
  • Incorrect error handling, which also results in a short dump with "INTERNAL SERVER ERROR 500" message (e.g. during new contract or payment sequence number generation).

Other Terms

Renewal IT0657, CL_HRPA_INFOTYPE_0657, ABAP dump, CX_HRPA_VIOLATED_POSTCONDITION, INTERNAL SERVER ERROR 500.

Reason and Prerequisites

Program error.

Solution

Errors have been corrected.

Please apply the correction by importing the Support Package specified for your release, or by using the Note Assistant.
