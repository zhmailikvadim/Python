Symptom

  • How can we disable People Search in Employee Profile so that no user can be searched in the search box?
  • How can we disable search box within BizX Platform?

Environment

SAP SuccessFactors HCM Suite

Resolution

We need to disable 2 permissions in RBP under General User Permission:

1. Organization Chart Navigation Permission

Organization Chart Navigation Permission controls user’s hierarchy. If this permission is disabled, user’s hierarchy (manager/peer/direct report) will not appear in Employee Profile.

2. Company info access > User Search

If we disable 'user search' here, then any user outside user’s hierarchy will not be searched out in the search box.

 

No user can be searched if both 2 permission are disabled at the same time and no search box will be visible within the system.
