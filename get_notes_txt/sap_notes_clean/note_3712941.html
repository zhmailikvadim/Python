Symptom

You encounter an error while creating an opportunity via API:
"Opportunity ID XXXXXXXXXX already exists. Either provide a unique ID or increase the number range status in the configuration."

Environment

  • SAP Sales Cloud and SAP Service Cloud Version 2
 



Reproducing the Issue

  1. Attempt to create an Opportunity using API and add the Display ID of a Deleted Opportunity in the Payload.
  2. Opportunity ID XXXXXXXXXX already exists. Either provide a unique ID or increase the number range status in the configuration.

Cause

It is not feasible to send display ID of a deleted opportunity in the payload to be created as a new opportunity.

Resolution

This is expected system behaviour.
