Symptom

This KBA explains the usage of checkbox 'Voluntary CPF as output of the report' in Singapore CPF monthly report program RPCCPFR0 selection screen.

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."  

Environment

  • Human Capital Management (HCM)
  • SAP ERP Core Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA

Resolution

Example employee is hired at 01.01.2025 with monthly OW 20000SGD, and the EE has excess voluntary CPF contribution maintained in IT0186 as below:

  1. Run original payroll for period 01.2025.
    /305 in RT: The CPF EE mandatory + voluntary contribution = 7400 * 20% + 600SGD = 1480SGD + 600SGD = 2080SGD
    /306 in RT: The CPF EE+ER mandatory + voluntary contribution = 7400 * 37% + 1000SGD = 2738SGD + 1000SGD = 3738SGD
     
    Meanwhile, this employee has CDAC contribution and SDF generated as below:
  2. Generate CPF monthpy report for 01.2025 by running program RPCCPFR0.
    • Without ticking the checkbox 'Voluntary CPF as output of the report', generated file includes mandatory CPF, additional fund and SDF contribution:

    • With ticking the checkbox 'Voluntary CPF as output of the report', generated file only includes voluntary contribution:

  3. Employee received 50000SGD AW applied in February, and the payroll run for period 02 is finished. Unpaid AW is generated as 36800SGD and saved in /310.
  4. Employee is terminated effective 01.04.2025, run original payroll for period 03:
    /305 in RT: The CPF EE mandatory + Voluntary contribution = 7400 * 20% + 600SGD = 1480SGD + 600SGD = 2080SGD
    /306 in RT: The CPF EE+ER mandatory + Voluntary contribution = 7400 * 37% + 1000SGD = 2738SGD + 1000SGD = 3738SGD

    The CPF adjustment has occurred, shortfall is generated as below:
  5. Generate CPF monthpy report for 03.2025 by running program RPCCPFR0.
    • Without ticking the checkbox 'Voluntary CPF as output of the report', generated file includes mandatory CPF, additional fund and SDF contribution:
    • With ticking the checkbox 'Voluntary CPF as output of the report', generated file includes voluntary contribution and shortfall:
