Symptom

You are using the DBA Cockpit to perform a runtime explain on a remote ASE database. An error occurred that the user cannot be found.

Other Terms

Sybase ASE, Runtime Explain, Performance, Process Monitor, remote database, non-SAP Database, DBA Cockpit

Reason and Prerequisites

SAP ASE supports case sensitive database names. As the <SID> database of an SAP ABAP application server is always in upper case, the DBA Cockpit was not fully supporting case sensitive user database names. With this SAP Note support for case sensititve database names in secondary database connections has been added for SAP ASE.

Solution

Implement the attached correction instructions using transaction SNOTE or implement the SAP support package stack relevant to this SAP Note.
