Symptom

When navigating within the succession org chart with a screen reader in Virtual PC Cursor mode, card content is not announced when using the arrow keys. Instead, the arrow keys move you to the next card in the hierarchy:

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

SAP SuccessFactors Succession Planning

Reproducing the Issue

  1. Log into SuccessFactors using a screen reader
  2. In the module picker, select Succession > Succession Org Chart
  3. Select either the Positions or People dial and enter search criteria
  4. Turn Virtual PC Cursor on
  5. Navigate through the org chart using arrow keys but the screen reader does not announce the card content. Instead, the arrow keys move you to the next card in the hierarchy.

Cause

This is a known issue.

Resolution

The fix for this issue is planned to be deployed with the 2H 2026 release.

For release timelines, please review SAP SuccessFactors Product Release & Road Map Information.

Workaround

Turn Virtual PC Cursor off and navigate using the Tab key.
