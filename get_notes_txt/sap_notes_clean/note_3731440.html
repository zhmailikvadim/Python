Symptom

  • The customer would like to switch the IPv4 network configuration on the SAP Windows Server from a static IP address to a DHCP-based IP assignment.

Environment

  • SAP NetWeaver
  • Windows
  • Windows Failover Cluster (high-availability configuration)

Cause

  • DHCP-based IP configurations are not supported for high‑availability SAP systems on Windows Failover Cluster. During failover, the virtual IP address can change, and DNS caching can prevent clients from reaching the system.

Resolution

  1. Maintain a static IPv4 configuration for the cluster’s virtual IP and related resources as required for Windows Failover Cluster deployments of SAP NetWeaver.
  2. For more information about IP Addresses in a Microsoft Failover Cluster Configuration, refer to IP address requirements for SAP on Windows Failover Cluster
