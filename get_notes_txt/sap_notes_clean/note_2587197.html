Symptom

Customer using an SAP Solution with integration to SAP Signature Management by DocuSign has an issue.

Problems or questions concerning support for a DocuSign product resold by SAP.

Environment

SAP Signature Management by DocuSign

Reproducing the Issue

Cause

Customer has a question about how support works between SAP and DocuSign or what information to provide to support.

Resolution

The following support procedure applies for any SAP customer who has a valid support contract with SAP and uses any DocuSign product resold by SAP:

  • For issues regarding DocuSign products, customers should ALWAYS assign support component XX-PART-DSN
  • DocuSign works via SAP's support infrastructure to help our customers.
  • SAP support channels will be the entry point for customers in addressing any support issue.
  • SAP customers can open cases via SAP Solution Manager or SAP for Me
  • For C4C and ByD, customers should not create a support case in the C4C solution itself but instead should create a support case via the SAP Launchpad.
  • SAP will provide the initial response and must perform an initial investigation before involving DocuSign.
  • As of Q2 2022, SAP Ariba customers should not use AribaConnect to report a DocuSign issue - Instead the customer should create a support case on SAP for Me in component XX-PART-DSN.

The Following SAP KBAs have useful information about opening and closing support cases:
1296527 - How to create a support case (contact SAP Product Support) - SAP for Me
1239839 - How to close a support case - SAP for Me
2331471 - How your feedback counts in completing the SAP Support Interaction Survey
2623991 - Reporting an case in Hybris Cloud for Customer

For any submitted support case, the following information should be provided to SAP/DocuSign (in addition to the customer’s contact information):

  • What is your DocuSign account name or what is your API Account ID?  https://support.docusign.com/articles/Where-do-I-find-my-DocuSign-account-number
  • Admin User name:
  • Admin User email:
  • Admin User phone#:
  • Admin API User ID (if known): 
  • What product area is DocuSign integrated with? e.g. SuccessFactors Recruiting module or SuccessFactors Performance Management
  • If only specific users are affected, please provide their email address.
  • What browser are you using? Have you checked other browsers? What OS, Windows version, mobile device?
  • What date did the issue occur? Is the issue reproducible?
  • What is the envelope ID or what is the template ID?   https://support.docusign.com/en/articles/Where-do-I-find-my-envelope-ID
  • Please provide a document with screenshot showing the issue being reproduced? What are the expected results? What is the actual result?
