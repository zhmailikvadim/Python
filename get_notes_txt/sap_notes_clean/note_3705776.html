Symptom

There is a need to confirm the JVM version running along with SCC.

Environment

  • SAP Cloud Connector(SCC);
  • JVM version.

Resolution

There are two possibilities to check the JVM version running along with SCC:

  1. Launch to SCC Web UI, from the upper corner right hand click on the <user ame> dropdown , and click on "About";

           





  2.  The alternative way is to download the SCC trace (ljs_trace.log / scc_core.trc), extract the zipped file that is downloaded from SCC web UI, open up file "scc_about.txt" and similar details above are returned.
