Symptom

SAP Netweaver introduced new functionality for blocking and deletion of user master records also for those Business Suite Solutions that uses SAP Basis 7.50.

For further information have a look into SAP note 2918197.

Other Terms

Data Protection and Privacy, GDPR, User Data, SU06, IDENTITY, USER_DEL_LOCK_ENTRY, ILM, Deletion, Blocking, Personal Data

Reason and Prerequisites

  • You are using SAP BASIS 7.50 / SP17 or higher.
  • You have enabled the new user data blocking and deletion functionality provided by SAP note 2826256. 

Solution

With this note, Project Systems deliveres the below listed entries in system table USRVETOTABLES for supporting the standard veto checks for the deletion of user data from the blocking area for master records as described in SAP note 2918197.
