Symptom

When accessing the /N/SCWM/MON transaction and opening the VAS Orders under the Inbound and Outbound nodes, data is not found.

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Environment

  • SAP S/HANA
  • SAP NetWeaver
  • SAP Extended Warehouse Management (SCM-EWM)
  • SAP Extended Warehouse Value-added Services (SCM-EWM-VAS)

Reproducing the Issue

  1. Open the transaction /N/SCWM/MON.
  2. Navigate to the node Inbound -->Document-->Vas order or Outbound-->Document-->Vas order
  3. Execute.
  4. No data is found.

Cause

  • Standard behavior.
  • In the node Outbound-->Document-->Vas order, only VAS orders with category B are displayed.
  • In the node Inbound -->Document-->Vas order  VAS, only VAS orders with category A are displayed. 

Resolution

To see all VAS orders with different order categories, it is necessary to do the following:

  1. Open the transaction /N/SCWM/MON.
  2. Navigate to the node Inbound -->Document-->Vas order. 
  3. Execute.
