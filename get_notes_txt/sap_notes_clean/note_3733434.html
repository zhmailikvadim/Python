Symptom

You work with the process framework within the Last Mile Distribution (LMD) solution.

When you run a process of the process framework, the system records the corresponding ABAP transaction ID in the process maintenance table LMD_PRCS_PRCMNT.

 When the process of the LMD process framework times out, the Process Framework Process Supervisor Report (P_LMD_PRCSS_FRWK_SUPERVISOR) identifies it after a defined period. The system then calls the implementation of the interface method LMD_PRCSS_FRWK_PRCSS_STEP_ERR->timeout_occured( ) if it exists. However, the system does not provide the current transaction ID of the timed-out process to that method, although this information is helpful for handling the timeout situation.

Other Terms

LMD, Process Framework, timeout, LMD_PRCSS_FRWK_PRCSS_STEP_ERR, timeout_occured, transaction id

Reason and Prerequisites

LMD_PRCSS_FRWK_PRCSS_STEP_ERR->timeout_occured( ) offers no importing parameter for the transaction_id. By implementing this SAP Note, the transaction ID is added and set accordingly before the system calls any implementation of the method.

Solution

Implement the attached correction instruction.
