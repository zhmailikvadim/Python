Symptom

  • The error FINS_CFIN_CO_MESSAGE301 occurs during the replication via CO_DOC interface.
  • FINS_CFIN_CO_MESSAGE 301: Business transaction &1 for management accounting document &2 is invalid.

Environment

Replication of CO Internal Postings

Reproducing the Issue

  1. Replicate a CO document via CFIN interface /FINCF/CO_DOC.
  2. Observe error FINS_CFIN_CO_MESSAGE301.

Cause

Missing functionality.

Resolution

  • Only the Business Transactions listed in the note 2103482 are currently supported.
  • Avoid replication of not supported business transactions by setting a filter in the SLT side. Refer to SAP Note 2154420 for available filters in SLT for replication.
  • Use a Central Finance BAdI to map the business transaction to a supported type. This can be accessed via customizing CFINIMG --> Central Finance: Target System Settings --> BAdIs: Central Finance. For this, consult with the business team to implement the mapping, considering the specific business scenario.

See Also

  • Refer to SAP Note 2103482 - Central Finance: Features of CO Secondary Posting Interface: SAP Note 2103482.
  • Refer to SAP Note 2154420 - SAP LT Replication Server for SAP Central Finance: SAP Note 2154420.
