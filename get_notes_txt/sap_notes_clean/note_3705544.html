Symptom

You are using PAOM Master Data application. On the overview page, when you click the View All button of Addresses Biz Card to display all the address related data of an employee, a runtime error occurs.

Other Terms

Personnel & Organization, PAOM, Infotype 0006, Address, HR Renewal, IT0006

Reason and Prerequisites

The reason was due to program error.

Solution

Import the relevant HR Support Package. In urgent cases, you can manually implement the changes in accordance with the correction instructions.
