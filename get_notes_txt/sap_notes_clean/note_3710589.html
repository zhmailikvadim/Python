Symptom

In GEF's Spatial Data Editor application, the search for Object Type - "WBS Element in Execution" does not filter the WBS Elements on their system status and the search result lists all the WBS Elements that matches the search value entered by the user.

Other Terms

SAP S/4HANA 2025, SAP S/4HANA Cloud Private Edition 2025, Geographical Enablement Framework, GEF, Spatial Data Editor, WBS Element in Execution

Reason and Prerequisites

Programming error. No pre-requisites required.

Solution

With the implementation of this note, the search result for Object Type "WBS Element in Execution" will show the WBS Elements that are in execution phase, i.e., WBS Elements with system status Released (REL) or Partially Released (PREL).
