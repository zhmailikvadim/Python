Symptom

In SE16, after sorting two or more fields, and use total (SUM) function, the following dump is triggered:

Category               ABAP programming error
Runtime Errors         ITAB_LINE_NOT_FOUND
Except.                CX_SY_ITAB_LINE_NOT_FOUND

     RAISING clause.
    Since the caller of the procedure could not have anticipated this
    exception, the current program was terminated.
    The reason for the exception occurring was:
    The reason for the exception is:
    Access failed for table "T_ADDED_LVC_DATA".
    Access method: "INDEX".
    Line index (for "INDEX" access) / key name (for "KEY" access): 32.
    Key values: " " " " " " " "

Environment

SAP_BASIS 758 SP02-SP04
SAP_BASIS 816.

Reproducing the Issue

1) Execute SE16 -> or open table 'Contents' from SE11.

2) Apply sorting (ascending or descending) for two or more fields.

3) Apply Total (SUM) on one of the fields.

4) The dump is raised.

Resolution

Implement the following SAP Notes:

1) 3633209. 

2) 3662062.

See Also

  • SAP Note 3633209 - ALV: Merge of cells when sorting not working properly
  • SAP Note 3662062 - ALV: Total function is not applied when selecting multiple columns
