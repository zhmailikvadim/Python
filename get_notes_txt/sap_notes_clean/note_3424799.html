Symptom

Getting below error while adding new location ID in SAP Analytics Cloud (SAC):

"Failed to call SAP Analytics Cloud Agent. Please contact your administrator to ensure that SAP Analytics Cloud Agent is running, configured correctly, and is reachable from the SAP Cloud Platform Cloud Connector."

Environment

SAP Analytics Cloud

Reproducing the Issue

1. Log in to SAC.

2. Go to system > Administration > Datasource Configuration.

3. While adding a New Location Id, getting below error:

"Failed to call SAP Analytics Cloud Agent. Please contact your administrator to ensure that SAP Analytics Cloud Agent is running, configured correctly, and is reachable from the SAP Cloud Platform Cloud Connector."

Cause

The internal host name and port has been used while adding New Location ID in SAC.

Resolution

Use virtual host and port instead of internal host and port while adding a New Location ID in SAC.
