Symptom

I would like to understand about the currency conversion in Fieldglass.

Cause

Resolution

SAP Fieldglass supports the use of multiple currencies and allows for currency conversion. Conversion factors are used to convert currency amounts to their equivalent values in the other currencies. This allows for consistency when viewing financial data.

Currency conversion factors can be established in the following ways:

  • Manual Conversion: Manually entered using the Currency Conversion admin menu. Admin > Financial Data > Currency Conversion or upload using the SAP Fieldglass standard Currency Conversion Upload.
  • Automatic Conversion: Using an integration with OANDA to maintain the currency conversion table. Fieldglass currency rates are updated by OANDA each night at 8:15 PM CST.
