Symptom

EA13 dump  MESSAGE_TYPE_X   55-502

Environment

Product version: SAP S/4HANA Cloud Private Edition 2023

Reproducing the Issue

EA13 MESSAGE_TYPE_X   55-502

Cause

Utilities not activated:

The application log table BALOBJ doesn't contain any data. That means all ISU entries are missing. This entries are generated by switch BC set ISU_APPL_LOG_C. As the name says, there is a switch that triggers this BC set. The switch is ISU_MD_CCS and it is not activated. I checked all switches for Utilities (IS-U) and found out that IS-U is not activated. Additionally, ISU invoicing tables eg ERDK  don't contain any data.

Resolution

using ISU transaction EA13 while ISU not activated doesn't make sense
