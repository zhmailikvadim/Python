Symptom

You execute the Plan Price Calculation for a version other than version 000. You can see the  Cost Component Split via the result screen but it will not be updated on the database.

Other Terms

KSPI, Plan Price Calculation, Version, Profit Center valuation, Group valuation

Reason and Prerequisites

Program error.
The error can only occur if version 000 has assigned the group or the profit center valuation.

Solution

Implement the attached correction instruction
