Symptom

Integration & Exception Monitoring is not displaying the data for ABAP WebService messages.

Environment

SAP Focused Run

Reproducing the Issue

Check the agent log

Cause

UCON mechanism is active but the function module '/SDF/IMA_DC_WS_ABAP' which is essential to collect the data was not added to the Communication Assembly (CA).

Resolution

As mentioned in the KBA 2616341, add this function module to Communication Assembly(CA) as the function module '/SDF/IMA_DC_WS_ABAP' is essential to collect the ABAP WebService data for Integration & Exception Monitoring.

Additionally, check whether the other function modules listed in the below Security Guide for SAP Focused Run were added to CA.

  • Focused Run | SAP Help Portal
    -> Select the corresponding FRUN version
    -> Security Guide
    -> Network and Communication Security
    -> Unified Connectivity Usage with SAP Focused Run
    -> Integration & Exception Monitoring
    Check whether the listed function modules are included in CA.
