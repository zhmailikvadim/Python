Symptom

In the IMG activity Define Value Types, you have defined a calculation for a value type at participation level. The calculation is not performed.

Other Terms

/MSG/H_RISK1

/MSG/HPRT_VL,

/MSG/HPRT

/MSG/HP_WERTART

/MSG/SFWS_SC_660_402

Reason and Prerequisites

Prerequisite:

  • The value type of the participation to be calculated uses a value type of the previous participation as the source.
  • The switch for the value type container in the participation period is active.

Reason:

During processing, the value type container is not taken into account. The value types are read only from the “Value Types for Participation” table. Since the source value type used is contained in the period entity when the value type container is activated, no calculation takes place.

Solution

History:
March 20, 2026: New

Description of the troubleshooting/technical effect:

The system includes the active value type container when it calculates the value types. The value types from the container are determined correctly. The value type is calculated.
