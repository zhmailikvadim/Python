Symptom

Settlement document is missing in the flow tree in Condition Contract in 'WCOCO'

Environment

  • Condition Contract Management
  • Condition Contract Settlement
  • SAP S/4 HANA

Reproducing the Issue

1.Create a Condition Contract via WCOCO.
2.Create a delta accrual for the contract using transaction WB2R_SC/WB2R_SV.
3.Create a manual settlement document for the condition contract using transaction WB2R_DOC_ENTRY.
4.Create an automatic partial settlement and final settlement using transaction WB2R_SC/WB2R_SV.
5.Check the document flow using the transaction code WCOCO , the documents are missing in flow tree.

Cause

Program Error

Resolution

Implement the following notes:

  • 3527927 - Document flow not adjusted for LFTYPN
  • 3528498 - Incorrect Handling of Collapsed Nodes in Hierarchical Document Flow
  • 3374049 - Not all settlement documents are read during document flow display
  • 3402194 - Display of document flow for settlement documents for predecessor and successor contract
  • 3386597 - Document flow does not display all follow-on documents

Keywords

WCOCO , CONDITION CONTRACT , CL_WLF_DOC_FLOW_TREE , Document Flow Tree , CL_WLF_DOC_FLOW
