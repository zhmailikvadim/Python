Symptom

  1. The log of program FINS_SWITCH_TO_OPEN_ITEM shows warning message FINS_SWITCH_OP 095. The message short text is not correct. You can activate open item management in spite of the warning message.
  2. The translation for several message short texts of message class FINS_SWITCH_OP was not delivered correctly.

Other Terms

FINS_ACTIVATE_OIM; FAGL_SWITCH_OP095; FAGL_SWITCH_OP127; FAGL_SWITCH_OP305; FAGL_SWITCH_OP306; FAGL_SWITCH_OP319

Reason and Prerequisites

Program error. Translation error

Manual Post-Implementation Steps

The correction instructions deliver changes to the German texts of several messages of message class FAGL_SWITCH_OP. The translations are only delivered via Support Package. Please note that these instructions are written in English. You may need to abbreviate some of the texts in order for them to fit the maximum length defined for message short texts.
The automatic translations into other languages will have the same issues regarding the maximum length.
In order to maintain translations for the messages, follow these steps:

  • Start transaction SE91.
  • Enter message class FAGL_SWITCH_OP and message number 121.
  • Choose function "Display".
  • Choose menu path "Goto >> Translation".
  • Enter original language DE, and your required target language.
  • Maintain the translations for the following messages:
    • 095: Items managed on open item basis already exist
    • 127: Bal. carryfwd for ld. &1 missing in curr. fiscal year, reconc. unreliable
    • 305: Clearing by ledger group is active. XSALH must remain active
    • 306: XSALH mandatory (GR/IR, shipment cost, or SFP account)
    • 319: Archived postings found. Activation only possible after &1.
  • Save the changes, and repeat the above steps for any additional required languages.
