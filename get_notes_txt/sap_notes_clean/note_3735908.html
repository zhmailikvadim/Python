Symptom

A released CDS view or views combination is required to extract the enterprise project Parent WBS ID, and Hierarchy Level, for use in an external API or custom ABAP Cloud development.

Environment

SAP S/4HANA Cloud Public Edition

Cause

There is currently no direct CDS view or combination of views for WBS hierarchy that is released for 'Use in Cloud Development' or 'API State'.

Resolution

  1. Review whether the views I_EntWBSElementHierarchy and I_EntWBSElementHierNode (released for 'Use in Key User Apps', not for 'Use in Cloud Development') fulfill your requirements.
  2. If Cloud Development or API consumption is required, submit a feature request or vote on the existing request in the Customer Influence Portal:
    3206294 - How to Submit Feature Requests for SAP S/4HANA Cloud Enterprise Portfolio and Project Management.
