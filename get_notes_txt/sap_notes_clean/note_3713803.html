Symptom

After posting a Goods Receipt against a Purchase Order (PO) with Multiple Account Assignment, the delivery cost (Freight) does not have a separate material document generated in the PO History tab. Additionally, no new entry is generated in the EKBZ table.

Environment

  • Materials Management (MM)
  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA

Reproducing the Issue

  1. In transaction code ME21N, create a PO with multiple account assignments, ensuring the PO includes a freight condition.
  2. Post the goods receipt in transaction code MIGO.
  3. Check the PO history in ME23N. There is no material document generated for this freight cost. And there is no entry in the EKBZ table.

Cause

This is a system error.

Resolution

Implement the note

3680927 - MAA2: Mehrfachkontierung mit unbewertetem Wareneingang und geplanten Bezugsnebenkosten

See Also

note 3680927 - MAA2: Mehrfachkontierung mit unbewertetem Wareneingang und geplanten Bezugsnebenkosten
