Symptom

How can I differentiate between test orders and production orders?

Environment

SAP Business Network for Procurement & Supply Chain  

Reproducing the Issue

  1. Receive an order in your system. 
  2. Check the account ID associated with the order. 
  3. Verify if the account ID ends with T or D to determine if it is a test order.

    Cause

    • Test orders are linked to accounts with IDs ending in T or D. 
    • Production orders are linked to accounts without these suffixes.

    Resolution

    If you want to identify the order you have received, check the account ANID associated with the order.

    • If the account ID ends with T or D, the order is a test order and does not need to be fulfilled.
      • If the account ID does not end with T or D, the order is a production order and should be fulfilled.
