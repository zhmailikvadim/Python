Symptom

How to capture BRN Reference for country Philippines while clearing open items in EBS?

Environment

  • SAP ERP
  • SAP R/3
  • SAP S/4HANA
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA

Cause

  •  Country specific Requirement

Resolution

  • The KIDNO field can be used to store the BRN number, similar to the UTR number, and line items can be posted and cleared based on the KIDNO field.

Keywords

BRN Reference, KIDNO,901 algorithm
