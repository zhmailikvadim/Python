Symptom

Would like the system to show error message if the postal code is not five numerical digits or 10 characters (5 digits + 1 hyphen + 4 digits) according to U.S. ZIP code format.

Environment

  • SAP Sales Cloud Version 2
  • SAP Sales Cloud and SAP Service Cloud Version 2

Resolution

  1. Click on the User icon.
  2. Click Settings.
  3. Click All Settings.
  4. Click Language & Region under General.
  5. Select Country/Region.
  6. Find United States of America.
  7. Update Postal Code Check Rule: Check Against Country-Specific Edit Format.
