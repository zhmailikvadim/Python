Symptom

This note implements technical checks for the Business Process Chain UUID, ensuring that only valid UUIDs are accepted and preventing invalid formats from being processed.

For a complete overview of the new functionality Direct Debit Mandates, please check consulting note 3445810.

Other Terms

Direct Debit Mandate, DDM, Business Process Chain UUID, Package FS_TB_DIRECT_DEBIT_MAND_AL

Reason and Prerequisites

Missing checks of the Business Process Chain UUID

Solution

Import the relevant Support Package that is specified for your release. As a manual advance correction, you can implement the attached correction instructions using transaction SNOTE.

Please consider the manual instructions for the long text message.
