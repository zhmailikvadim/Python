Symptom

When attempting to modify the SpecialGLAccountAssignment (HZUON) field in APARItems using the Manage Journal Entries app or APIs, the following error message appears:  “”Change of field Sp.G/L Assgt is not possible. Field is not open for input." 

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Attempt to change the SpecialGLAccountAssignment (HZUON) in the BO interface i_journalentrytp or 
  2. Encounter the error messages: "Change of field Sp.G/L Assgt is not possible. Field is not open for input." and "Change of Journal Entry Item CoCd/ Journal entry number/ fiscal year/ line item rejected."

Cause

SpecialGLAccountAssignment can be changed only in below conditions:

  • Account Type D + Sp. G/L Trans.Type A or W
  • Account Type K + Sp. G/L Trans.Type W

Resolution

Ensure that the following conditions are met for field SpecialGLAccountAssignment (HZUON) to be editable:

  • Account Type D + Sp. G/L Trans.Type A or W
  • Account Type K + Sp. G/L Trans.Type W

Otherwise, it's standard behavior to get the errors.
