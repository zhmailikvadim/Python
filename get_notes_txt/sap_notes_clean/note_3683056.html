Symptom

A program needs to process a large database table (like for example table ACDOCA) and the performance of the program is low.

The internal table mt_package_results does not contain the BAPIRET2_TAB table to return error messages in this format.

Other Terms

CL_FINS_PARALLELIZATION , EXECUTE_PARALLELIZATION , get_mt_package_results( )

Solution

After implementing this note, the table mt_package_results (which can be accessed with the method get_mt_package_results( )) contains the BAPIRET2_TAB table.
