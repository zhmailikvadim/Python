Symptom

Admin user see "Data Model Compatibility Issue Detected" message in Cloud Foundry integration page as shown below:

Environment

SAP LeanIX

Cause

There are two potential root causes:

1. All required fact sheet types, relations, and subtypes/categories are missing from the existing meta model configuration in the workspace even the workspace got updated to Meta Model Version 4. 
2. The existing key names in Meta Model Configuration do not match exactly the requirements. For example: Platform fact sheet type - Key name: 'TechPlatform' instead of 'Platform'

Resolution

Please verify your Meta Model Configuration by going to Administration > Meta Model Configuration

All required fact sheet types, relations, and subtypes/categories must be fulfilled as the followings, and the key names will need to match exactly as listed below:
