Symptom

This SAP note delivers report MPE_PLN_ROUTING_STXH_FIX which can be used to find and delete orphaned operation long texts in planning routings. Due to a bug in old coding these long text without parent operation could appear, which prevents creation of same operations in the next run of PiP based routing handover

Other Terms

MSFR14; Planning routing

Reason and Prerequisites

Program error

Solution

Apply the attached correction instruction. Afterwards execute report MPE_PLN_ROUTING_STXH_FIX, first in test mode, then if there are findings not in test mode to fix the data in the system. Flag 'Delete ALL' shall be used cautiously as it will process the long texts of all the planning routings in the system
