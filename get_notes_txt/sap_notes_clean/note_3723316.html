Symptom

  • RE-FX contracts are managed with subject to valuation using asset integration
  • When processing the valuation posting of the asset cash flow, error message RERAPP 053 "Transaction may only post to G/L accounts." is raised in the log for the asset posting
  • However, the other postings for interest and liability are successful

Environment

  • Flexible Real Estate Management (RE-FX)
  • Contract and Lease Management
  • SAP ERP 
  • SAP ERP Central Component
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA Finance
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Access transaction RECEEP
  2. Execute the transaction in Simulation mode.
  3. Access the Error Log to view the error message RERAPP 053 "Transaction may only post to G/L accounts."

Cause

  • Configuration issue of the account assignment of the flow type used for the transfer posting from asset clearing account to the liability. This posting must use G/L accounts only.

Resolution

    Review the configuration settings for the asset postings of the valuation.

    1. Verify which flow type is  configured for the posting of "Clearing of Asset Transaction"

     

    Customizing path:

    - Flexible Real Estate Management (RE-FX)
    - Conditions and Flows
    - Flow Types
    - Assign Reference Flow Types

    Example: 

    Flow type I110 is used for the asset posting (the flow type can be reviewed in the asset cash flow)

    Check the entry for reference CEK:

    CEK Clearing of Asset Transaction    I110    Right of Use (AA)    I150    Present Value

    Flow type I150 is used for the posting from the asset clearing account to the liability account.

     

    2. Review the account determination for the assigned flow type (I150 in this example)

    Customizing path:

    - Flexible Real Estate Management (RE-FX)
    - Accounting
    - Automatically Generated Accounting Documents
    - Account Determination
    - Assign Account Symbol to Flow Type

    The account determination settings for flow type I150 have to be configured with account type "S" (G/L account).

    Debit Account Type: S  (G/L Accounts)

    Credit Account Type: S  (GUL Accounts)

    Example: 

    I150   Present Value    S    ASSET_CLEARING    S    LEASLIABILITY

    See Also

    Refer to: SAP Note 2580277 - Flow types and account determination during the valuation of leasing contracts
