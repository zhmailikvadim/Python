Symptom

 When we edit the amount of an already activated disbursement order in the future without withheld items and items to fund, it triggers an error.

Other Terms

 #FUNDED_LOANS; Disbursement

Reason and Prerequisites

 You have installed Support Package 5 of localizations for banking services from SAP 8.0.

Solution

After installing this note the system will properly handle the disbursement.

As general rule, SAP recommends that you install a solution by applying a Support Package. However, if you need to install the solution earlier use Note Assistant (SNOTE) to implement the correction instructions.

List of new/changed objects delivered in this note:

Classes/Methods:

    • /FSGLO/CL_FINA_DISB DELETE_MESSAGE_FROM_LOG

    • /FSGLO/CL_FINA_DISB IF_BCA_OR_DISB_ORDER_EVENTS~ACTIVATE

    • /FSGLO/CL_FINA_DISB IF_BCA_OR_DISB_ORDER_EVENTS~SAVE

Note that documentation, transactions and Customizing activities are only delivered via Support Package.
