Symptom

  • Overlays are not functioning as intended; overlays are being added when no attribute value have been set.
  • The custom attribute is of Type Dictionary List

Environment

Workspace Setup for SAP Signavio Process Manager

Reproducing the Issue

  1. Assign a custom attribute dictionary attribute to a model, task, or sub-process.
  2. Save the changes.
  3. Preview the changes in the Collaboration Hub.
  4. Overlay is displayed even when the custom attribute value is not set.

Cause

Resolution

  1. Navigate to Process Manager > Setup > Attribute Visualization > Overlay for the custom attribute > Rules.
  2. Modify the rule for Relation to "is not empty".
  3. Save the changes.
  4. Verify that the overlay is hidden when the attribute has no value.
