Symptom

  • I want the worker to be able to enter “1” to record time for a day rate.
  • Submitting time in hours should generate an error.

Cause

Resolution

Enable the Use Prorated Day setting on the Rate to allow workers to enter a “1” for a day rate. When this setting is Yes, you can configure Allowed Values that define which decimal portions of a day a worker can enter on a timesheet.

If you want workers to enter only a full day, set the Allowed Values list to 1.
If you want to support partial days (e.g., a half day), add values such as 0.5. For example, if the day rate is 684.00 and the worker enters 0.5, the system automatically bills 342.00.

When a worker enters a value that is not included in the Allowed Values list, the system displays an error such as:

Invalid entry in Time Worked. Total on 16/01/2026 must be one of the following values: 1.00.

Note that Use Prorated Day cannot be edited on an existing Rate, so you must create a new Rate if you need to enable this functionality.
