Symptom

You activated new Offsetting Account Determination in S/4HANA (SAP Note 2621199 - Offsetting Account Determination in S/4HANA).

You want to use Memorial Order Creation function based on new offsetting accounts.

Other Terms

ERP for Banking, Memorial order, Memo order, FIBKRU_MO_CR_OFFLN, RPFIBKRU_MO_CR_OFFLN

Reason and Prerequisites

New functionality.

Solution

Implement automatic correction instructions of this SAP Note or implement the mentioned Support Package.

Note: If document for the report isn't processed based on new solution then offsetting accounts from old solution are used.
