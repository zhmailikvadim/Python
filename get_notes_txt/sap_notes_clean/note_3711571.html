Symptom

There is no bookmark button in Journal Entry Analyzer (New Version) app.

Environment

SAP S/4HANA Cloud Public Edition

SAP S/4HANA Cloud 2602 and higher version

Cause

Standard behavior.

Resolution

As of SAP S/4HANA Cloud 2602, the new app consolidates bookmarks and classic variants into a single new variant object. The new variant automatically saves both variable settings and layout changes whenever they are modified.

Layout can be saved by doing below steps.

  1. Change the layout based on requirements. For example, field such as "Amount in Freely Defined Currency 1" can be added in layout.
  2. Click “Standard” and click “save as” to save the layout.
  3. The layout is saved.

Keywords

layout, bookmark, Journal Entry Analyzer (New Version)
