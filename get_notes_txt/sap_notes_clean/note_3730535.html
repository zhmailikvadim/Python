Symptom

Error KI013 "Internal program error. Contact SAP" is raised while posting a purchase requisition.

Environment

  • Controlling (CO)
  • SAP S/4HANA
  • SAP S/4HANA Finance
  • SAP S/4HANA Cloud Private Edition
  • SAP Fiori for SAP S/4HANA

Reproducing the Issue

  1. The new commitments management solution for cost centers is active
  2. The user posts a purchase requisition
  3. The error is raised

Cause

A statistical account assignment is being used along with the cost center. This is not supported for the new commitments management solution according to SAP Note 2778793.  

Resolution

Adjust the account assignment provided to use only the cost center as a real account assignment. 

See Also

SAP Note 2778793 - Constraints with New Commitments Management Solution (S/4HANA 1809 and higher releases)

Keywords
