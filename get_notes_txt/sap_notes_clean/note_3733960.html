Symptom

A User is unable to see "Add" button for the all of the color palettes in CSB

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental

Environment

SAP SuccessFactors Recruiting Management

SAP SuccessFactors Recruiting Marketing

Reproducing the Issue

Goto CSB >> Appearance >> Global >> choose a color component to add more colors, there is no add button visible as below:

Cause

The user role in the CSB does not have the color editing permission.

Resolution

To provide the permission under CSB: 

Goto User >> Roles >> Edit the role associated with user >> Make sure enable the color editing permission for the user role as shown in the above image.
