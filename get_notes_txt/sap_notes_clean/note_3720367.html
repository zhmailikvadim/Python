Symptom

  • Error message L3A037 - "Putaway via inbound delivery. No item specifications allowed."
  • Transfer orders created using function module L_TO_CREATE_DN are not moving stock correctly.
  • The system terminates transfer order creation when delivery item-dependent entries are transferred to table IT_DELIT for inbound deliveries in Handling Unit-managed storage locations. 

Environment

  • Logistics Execution (LE)
  • SAP R/3
  • SAP R/3 Enterprise 4.7
  • SAP ERP Central Component
  • SAP ERP
  • SAP Enhancement package for SAP ERP
  • SAP Enhancement package for SAP ERP, version for SAP HANA 
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Attempt to create a transfer order with reference to an inbound delivery using function module L_TO_CREATE_DN.
  2. Specify the desired input for the stock movement.
  3. Error message L3A037 is raised and the transfer order creation is terminated. 

Cause

  • The function module L_TO_CREATE_DN does not support delivery item-dependent entries transferred to table IT_DELIT for transfer order creation when the inbound delivery is based on an HU-managed storage location. 

Resolution

  1. If the requirement is to make specifications for putaway of handling units in a HU-managed location, then the putaway transfer order would have to be created using the function module L_TO_CREATE_MOVE_SU.
  2. Use function module L_TO_CREATE_MOVE_SU to create the putaway transfer order for Handling Units with specifications for the input bin.
  3. Refer to the long text of error L3A037 in transaction code SE91 for further details. 

See Also

  • Refer to SAP Note 0000109533 for additional guidance on using SAP function modules.
