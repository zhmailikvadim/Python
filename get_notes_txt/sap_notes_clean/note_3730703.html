Symptom

Performance Improvement

Solution

Implement the correction instructions in this SAP Note by selecting all the objects in the SAP Note and overwriting the existing objects in your system.

After implementing the SAP Note, kindly calculate the SAPUI5 application index by referring to SAP Note (2449635 - How to calculate SAPUI5 application index).

And clear the browser cache after implementing the SAP Note.
