Symptom

In the process "Sending of Quotation (MOS View)" (DE_B_MOSB_QUOT_MOS), after the "Request for Quotation" process is started and you select the "Trigger Quotation" button on the Time Slice UI page, a second "Sending of Quotation" process is triggered by the job in step 0010.

This issue is relevant for the market role: MOS.

Other Terms

US4G, Trigger Quotation, /US4G/CL_CTR_API, TRIG_QUOTES_PROC, Time Slice UI, Request for Quotation, duplicate process, MOS

Reason and Prerequisites

The error occurred because there was no filter for either the 'Sending of Quotation (MOS View)' or 'Request for Quotation (MOS View)' process.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, if there is an active "Sending of Quotation (MOS View)" process (not terminated) or an active "Request for Quotation (MOS View)" process, the system stops the triggering of a new "Sending of Quotation (MOS View)" process.

You can verify the fix by selecting the "Trigger Quotation" button on the Time Slice UI page. The system displays error message /US4G/PROC_BILLING082 (Quotation sending process was not triggered for time slice &1), and a new "Sending of Quotation (MOS View)" process is not created.
