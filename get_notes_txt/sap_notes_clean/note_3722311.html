Symptom

  • An pop up occurs when attempting to create a production order due to a "No routing found" message.
  • The error prevents the successful creation of the production order.
  • The user is seeking a solution to bypass the "No routing found" message or configure the production order type to skip routing checks.

Environment

  • SAP ERP Production Planning (PP)
  • SAP ERP All versions
  • SAP ERP Central Component All versions
  • SAP R/3 All versions
  • SAP R/3 Enterprise All versions
  • SAP S/4HANA All versions
  • SAP S/4HANA Cloud Private Edition All versions
  • SAP enhancement package for SAP ERP All versions
  • SAP enhancement package for SAP ERP, version for SAP HANA All versions

Reproducing the Issue

  1. Use T-code CO01 to create a production order.
  2. Encounter "No routing found" pop up during the process.
  3. Production order creation fails due to the error.

Cause

In the current system design, production orders cannot be created without routing, as routing is an essential component of the production version. It seems that the business design may conflict with this standard design.

Resolution

To proceed, you might consider creating a default routing, as this will allow you to successfully create production orders.
