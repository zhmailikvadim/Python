Symptom

  • The job service is blocked and cannot be stopped or started.
  • The service status is showing as "starting" instead of "running."
  • When trying to update any configuration you can not save with error "update failed!".

Environment

SAP Business One
Job Service

Reproducing the Issue

  1. Perform an update in the Job Service Configuration and/or.
  2. Attempt to stop or start the job service.

Cause

Resolution

To address this, please try the following steps:

1. Click on the "Stop" button to halt the service and then restart it.
2. Alternatively, you can restart the SLD service by executing the command `systemctl restart sapb1servertools.service` for Hana systems, or, restart Server Tools from Windows Services on SQL systems.
