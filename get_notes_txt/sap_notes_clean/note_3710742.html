Symptom

As of SAP_BASIS release 757, when opening the Layout Administration screen in the settings dialog, an HTML End-of-List section is displayed.

This section may appear empty or contain technical information text such as:

  • Report:BCALV_TEST_FULLSCREEN,Log_Group:,Handle:

This text was intended to provide technical details about the layout loaded from table LTDX, allowing analysis without debugging.

The information is displayed only if the user has the required authorizations and if at least one layout is available in the administration screen.

In addition, a new test report SALV_GUI_LAYOUT_EXPLORER (transaction SALV_LAYOUT_EXPLORER) was introduced.

It uses this technical data to display layout details from LTDX, primarily to support troubleshooting, especially in cloud environments.

Reason and Prerequisites

Reason

The End-of-List information in the Layout Administration screen was originally intended to support SAP analysis together with the new test report.

However, this information is technical in nature and does not need to appear on the screen in all scenarios.

Prerequisites

The following Support Package level is the minimum prerequisite:

Release Support Package
SAP_BASIS 816 SP 00
SAP_BASIS 758 SP 02
SAP_BASIS 757 SP 00

When you install an SAP Note, some prerequisites might be required. The prerequisites are SAP Notes that have already been released and that change the same context of the current one. Then, even if these prerequisites are not necessarily related to the symptom of this SAP Note, you must implement them due to technical reasons.

Solution

This note changes how technical layout information is displayed when opening the Layout Administration settings.

The HTML End-of-List section will no longer be shown.

Instead, the technical information is now available directly within the layout list as hidden columns.

The following new columns can be displayed if the user has the required authorization:

  • Report 
  • Handle 
  • Log Group 

These fields can be added to the list using the Change Layout dialog in the Layout Administration screen.
