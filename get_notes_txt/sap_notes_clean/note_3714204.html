Symptom

When you execute the report ELStAM Procedure: Transfer XML Data to ELStAM Table - Distributed Reporting  (RPCE2SD0_IN_VR), the system issues the following error message:

"New entry could not be inserted in table P01E2_PKV_HIST for GUID <GUID>." (HRPAYDE_E2_PKV 021).

Reason and Prerequisites

You have implemented one of the following two SAP Notes:

  • 3703296 - ELStAM PHI: New pushbutton in the administrator list for updating the infotype SI Additional Insurance (0079) without check
  • 3704926 - ELStAM PHI: Incorrect sorting of status history in administrator list

Solution

Implement the attached correction instructions or import the specified support package. After you import the HR Support Package or implement the correction instructions, you do not have to perform any further follow-on activities.

After you implement the corrections, the system no longer issues the error message. The affected notifications remain in the central system with the status To Be Processed. 
but have no effect since they are no longer selected.
