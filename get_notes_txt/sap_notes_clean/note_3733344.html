Symptom

With transaction FB08 you want to reverse a FI document posted via BAPI. As a rule you should reverse BAPI postings also with the reversal BAPI!
The reversal with FB08 results in error F5 842: "Balance in 1 currency".

The document to be reversed was posted via BAPI with a ledger group and has document status 'L'. In the general ledger there are additional line items which were only posted in non-FI integrated currencies.

Other Terms

F5 842, F5842, BAPI, BSTAT = L, BSTAT = 'L'

Reason and Prerequisites

Program error.

Solution

Please implement the correction instructions.
