Symptom

When performing multiple device change using process 3070, it is possible that after receiving multiple web services SmartMeterGatewayMeterProfileCreateConfirmation_In at the same time. The installation of the new device is terminated because the material is locked by IS-U function.

Other Terms

IM4G, 3070, Device Replacement, M3, 897

Reason and Prerequisites

 This is a program error.

Solution

Please implement the correction instruction in the SAP Note or install further Support Package.
