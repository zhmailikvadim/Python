Symptom

In the standard system as currently delivered, you run the Year End Reporting, Canada (RPCYERK0) in background mode after the application of SAP Note 3037569 with the Display output log parameter active and the program takes much longer to finish.

Other Terms

Year End Reporting, RPCYERK0, log, background.

Reason and Prerequisites

You have applied SAP Note 3037569.

To display the complete information, according to SAP Note 3037569, the system needs to read and print much more information than before, thus degrading performance in this scenario.

Solution

From now on, it is possible to configure the behavior of SAP Note 3037569.

In the Customer values for configuration options (V_T5F99K2) view you can now use the following configuration option:

Configuration option Description
YEWTL YEAR END - WAGE TYPE CALCULATION ON LOG 

This configuration option has the following values:

  • ON - the system executes the behavior as described in SAP Note 3037569. In this case please be aware that the performance for mass execution is much lower.
  • OFF - the system generates the output log as before the application of SAP Note 3037569. This is the default configuration. In case no configuration exists in the V_T5F99K2 view the system behaves as OFF.
