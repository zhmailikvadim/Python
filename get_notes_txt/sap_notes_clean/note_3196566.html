Symptom

When creating a Technical Object ( Equipment ) and inputting a Main Asset Number - the error 'Function is not available' Message no. SFIN_FI004 appears and not known why

Environment

  • SAP S/4HANA Cloud Public Edition
  • Asset Management

Reproducing the Issue

  1. Access the app Create Technical Object.
  2. Select Technical Object Type 'Equipment'.
  3. Select Technical Object Category 'P Production Resource Tool'.
  4. On the Organization Data tab input a Company Code.
  5. In the field 'Main Asset Number/subnumber' input an Asset Number.
  6. Press Enter.
  7. Error appears with 'Function is not available ' Message no. SFIN_FI004.

Cause

We do not currently support Equipment and Asset integration in SAP S/4HANA Cloud Public Edition and it is a mistake that the field Main Asset Number/subnumber is visible there.

We will remove the field from the App.

Resolution

Please follow the KBA 2840635 and submit a feature request if you would like the functionality added in a future Release.
