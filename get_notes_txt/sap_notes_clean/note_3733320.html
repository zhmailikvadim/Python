Symptom

You are unable to regenerate the activity list recurrences using the transaction PYT_ADMIN_TRANSACTION.

Cause

When activities are added to or removed from the activity list, the corresponding recurrences are not automatically updated.
To reflect these changes, the recurrences must be manually maintained in the Plan Activity List.
Therefore, PYT_ADMIN_TRANSACTION cannot regenerate or refresh the recurrences.

Resolution

If an activity was removed from the activity list → Delete the related recurrence in the plan activity list.
