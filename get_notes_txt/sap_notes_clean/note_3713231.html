Symptom

  • When executing transaction ICAFU as a background job, the job fails with the error message: "Cannot perform frontend function in batch input mode".
  • The job runs successfully in the frontend mode but fails in the background mode.

Environment

S/4HANA Intercompany Matching and Reconciliation

Reproducing the Issue

  1. Call transaction ICAFU .
  2. Navigate to Menu > Program > Execute in background.
  3. Set Job start to "Immediate".
  4. Check the job log in transaction SM37, which displays the error message: "Cannot perform frontend function in batch input mode".

Cause

Resolution

The transaction ICAFU calls GUI frontend services to upload or download files, which cannot run in background mode.

Keywords

ICAFU, ICA_UPLOAD_MATCHING_ITEMS, batch input mode, frontend function error, background job failure, SM37, intercompany reconciliation, S/4HANA, automation, GUI frontend services, ICMR
