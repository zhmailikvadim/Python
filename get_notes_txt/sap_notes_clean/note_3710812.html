Symptom

If execution status of a change record is In Process (05), then it shall be possible to carry out following item process status action:

  1. From Initial (00) to Change Unnecessary (02)
  2. From Change Possible (01) to Change Unnecessary (02)
  3. From Change Unnecessary (02) to Change Possible (01)

Other Terms

 /PLMI/CR_PRO_VAL

Reason and Prerequisites

Missing functionality

Solution

Apply the related hotpackage or the manual correction instruction
