Symptom

The address data of the equipment, functional location or service order are changed on the FSM side. Based on these changes, a message with the address data is generated and processed in ECC/S4. In addition to the changes made in the fields mapped from FSM, some address data fields are cleared. The error is caused by incorrect propagation of the values that are to be changed.

Other Terms

FSM, E4C, Connector, Address, /PACG/ECM_ADDRESS_IN, Equipment, Functional location, Service order

Reason and Prerequisites

The address update process has been improved with an additional address selection step, ensuring that updates apply only to the correct fields while leaving unchanged values unaffected.

Solution

Please, implement this correction instruction.
