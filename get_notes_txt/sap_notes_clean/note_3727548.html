Symptom

The timestamp functionality used in the CDS view FS_IAF_I_PAYM_LABEL does not correspond to the timestamp format used in the used data definitions.

This correction introduces a new parameter in the correct timestamp format.

Other Terms

IAF, Intelligent Account Finder, CDS FS_IAF_I_PAYM_LABEL, +AMDP_HDB 007, +AMDP_HDB007 TSTMP_CURRENT_UTCTIMESTAMP

Reason and Prerequisites

Program error

Solution

Import the relevant Support Package that is specified for your release.
As a manual advance correction, you can implement the attached correction instructions using transaction SNOTE.
