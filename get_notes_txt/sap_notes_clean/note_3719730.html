Symptom

The Delete button is grayed out in the Outbound Delivery application. When accessing the Change Outbound Delivery app (VL02N), the Delete button is disabled and cannot be selected.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Open the Change Outbound Delivery app.
  2. Enter the outbound delivery number and open the document.
  3. From the top menu, select Outbound Delivery.
  4. Check the available options.

Result: The Delete button is grayed out and cannot be selected.

Cause

The outbound delivery is relevant for Extended Warehouse Management (EWM). Once the document is distributed to Cloud WM, the Delete button is no longer available.

Resolution

Since the outbound delivery is EWM-relevant and has already been distributed to Cloud WM, deletion is no longer possible due to standard system design. In this case, the recommended approach is to technically reverse and close the document by adjusting the quantities to zero and completing the process flow.

Please follow the steps below:

  1. Reverse the Goods Issue for the outbound delivery (if it has already been posted).
  2. Cancel any open Warehouse Tasks related to the delivery.
  3. Reverse or cancel the picking for the delivery (or for the relevant delivery items).
  4. Adjust the delivery item quantities to zero.
  5. Post the Goods Issue again to complete and close the delivery on the Logistics Execution (LE) side.

This final step triggers an update message to the LE delivery, setting it to Completed status with zero quantity, in line with the standard system behavior.
