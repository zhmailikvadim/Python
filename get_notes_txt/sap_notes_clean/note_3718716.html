Symptom

When an issue occurs on the SAP Analytics Cloud Apps on Android how can users retrieve the logs of it? 

Environment

  • SAP Analytics Cloud (Enterprise) 2026.02 & up
  • SAC Mobile App 1.54.0 & up
  • Android 15 & up

Reproducing the Issue

NA

Cause

NA

Resolution

  1. Log in to SAC via the Android App.
  2. Tap on the top right Profile icon.
  3. Tap on Settings.
  4. Tap on Enable Log File.
  5. Exit the settings and reproduce the issue.

Now in the same Settings page, the logs.txt file can be sent to a valid email address via the Android device built in email.
