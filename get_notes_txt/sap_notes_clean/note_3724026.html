Symptom

You define a mapping method on the BO Data Model Mapping Rules tab of the Configure Business Object Mappings Fiori app. You create the mapping method as a copy of sample method MAPPING_METHOD of class /APE/CL_INTG_PARSER. When you run the business object mapping, the system displays a runtime error because the mapping method contains incorrect parameters.

Other Terms

Configure Business Object Mappings, Mapping Method, /APE/CL_INTG_PARSER, MAPPING_METHOD, CT_DATA

Reason and Prerequisites

Program Error.

The sample method MAPPING_METHOD in class /APE/CL_INTG_PARSER contains outdated parameter definitions that are incompatible with the current business object mapping framework.

Solution

Implement the SAP Note or install further Support Package.
