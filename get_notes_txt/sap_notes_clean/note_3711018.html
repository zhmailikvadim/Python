Symptom

In the Migration Cockpit for the direct transfer approach, you are using the scenario SAP AFS to SAP S/4HANA to migrate the Sales Orders. After data selection, you observe that order of long text is not aligned with Source AFS Sales orders.

Other Terms

DMIS, Direct Transfer, Migration Cockpit, Transfer Data Directly from SAP System, Migrate Data Directly from SAP System, Data Migration, Migrate Your Data, SAP AFS to S/4HANA, Longtext,  Migration Cockpit, New Implementation, Greenfield implementation, DMIS 2011 SP25, DMIS 2011 SP26, DMIS 2011 SP27

Reason and Prerequisites

Program error.

Solution

Implement the note in the source system.
