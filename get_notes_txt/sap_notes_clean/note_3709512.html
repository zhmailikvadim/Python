Symptom

In a DMLT+NZDT project where XPRAS and/or conversion reports are run from the execution tree during downtime, tree activity NZTR1_RUN_CNV_JBS_AF_DT_XPRA_R parses and shows the error log lines that were written by the XPRA and/or conversion report in question, both during uptime and downtime.

Other Terms

downtime minimization tool for SAP S/4HANA, NZDT

Reason and Prerequisites

Any of the following add-on support packages is installed in your system landscape:

  • NZDT 1.0 SP03

Solution

Error log lines of the uptime execution are now discarded, such that only error log lines of the downtime XPRAs and conversion reports will be parsed and shown.

Required Action

In order for this note correction to be effective, the note must be implemented before:

  • Executing tree activity "Run conversion jobs after XPRAs and AIMs" NZTR1_RUN_CNV_JBS_AF_DT_XPRA_R

Implement the correction instructions in transaction SNOTE on the following systems of your system landscape:

  • Receiver system
