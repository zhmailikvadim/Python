Symptom

CVE-2025-68161 detected for Adobe LiveCycle Designer 11 as below:

Path: C:\Program Files (x86)\Adobe\Designer 11.0\Java\Libs\log4j-core-2.17.2.jar
Installed version: 2.17.2
Fixed version: 2.25.3

Environment

  • Interactive Forms by Adobe
  • Adobe LiveCycle Designer 11 Patch 30 or earlier

Reproducing the Issue

  1. Install Adobe LiveCycle Designer 11 (SP30 or earlier).
  2. Run a vulnerability scan against the installation directory.
  3. Observe CVE-2025-68161 detection pointing to log4j-core-2.17.2.jar under C:\Program Files (x86)\Adobe\Designer 11.0\Java\Libs.

Resolution

Install the latest version and patch from Adobe LiveCycle Designer as the issue is solved form SP31.

If the old jar files still present in the system after the installation, attempt to rename the old files from .jar to .bak or delete the old files manually
