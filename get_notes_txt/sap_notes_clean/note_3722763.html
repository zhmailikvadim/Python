Symptom

You are migrating data in Migrate Your Data app with Direct Transfer approach. 

The migration object being used is PM - Equipment.

You notice that the field TXASP of table EQKT is empty in target system even though is filled in source

Environment

All S4/HANA versions

Reproducing the Issue

  1. Migrate the data
  2. Go to SE16 in target system
  3. Search for table EQKT
  4. The field TXASP remains empty even though the migration was successful

Cause

Misfunction fixed by a note.

Resolution

Please implement this SAP note below and migrate the data again.
3707567 - Equipment Migration Object: Issue with Primary Language Update in EQKT Table

See Also

SAP Help: Data Migration

Keywords

PM - Equipment, direct transfer, TXASP, EQKT
