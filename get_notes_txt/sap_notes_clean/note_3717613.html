Symptom

  • When creating a standard cost estimate, the selected valuation strategy is "Valuation Price According to Price Control in the Material Master."
  • However, in table KEKO, the system updated the record with this valuation strategy and additionally assigned the sub-strategy "Net Purchase Order Price."
  • Ideally, sub-strategies should not be considered when the material valuation is based on "Valuation Price According to Price Control in the Material Master."
  • The presence of a sub-strategy in this case suggests a potential data inconsistency in the KEKO database table.

Environment

  • Controlling (CO)
  • SAP R/3
  • SAP R/3 Enterprise 
  • SAP ERP
  • SAP ERP Central Component
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA

Reproducing the Issue

  1. Start transaction code CK11N.
  2. Fill-in the required parameters.
  3. Create the standard cost estimate.
  4. The valuation strategy selected is "Valuation Price According to Price Control in the Material Master".
  5. There is no sub-strategy selected in this scenario.
  6. Save the standard cost estimate.
  7. Check the entry generated in the database table KEKO. The sub-strategy "Net Purchase Order Price" is updated in the KEKO entry.

Cause

Program error.

Resolution

To solve the issue, please implement the SAP Note:

3682793 - Sub-strategy filled incorrectly in table KEKO

Keywords

Standard cost estimate; Valuation strategy; Valuation Price According to Price Control in the Material Master; KEKO; Sub-strategy.
