Symptom

When clearing accounts receivable in the Post Incoming Payments app, the system displays messages "Posting takes place in previous fiscal year" and "You can’t make the posting while there are open warning messages".

Environment

SAP S/4HANA Cloud Public Edition

Cause

Wrong application is used.

Resolution

For clearing purpose, use the Clear Incoming Payments app.

See Clear Incoming Payments for more details.
