Symptom

You are working with a PMR document. All the component items have PSASTAGE configuration entry, so the Goods Movement bin and the Staging method has been determined well during the document creation. After the document creation you change the PSASTAGE entry so you change either the GM bin or the staging method or both values in that. Now, when you perform the consumption, you will see that these values are going to be redetermined/changed during the consumption post, however these values mustn't be redetermined anymore.

Other Terms

PMR, PSASTAGE, consumption, the staging method is changing during GI, GI, /SCWM/CL_PWR_GMBIN

Reason and Prerequisites

Program error

Solution

Implement the correction instruction or the equivalent support package
