Symptom

In rare cases (see Reason and Prerequisites section) not all currency positions of a contract are identified. As a consequence corresponding previous target values are thus not cleared.

Other Terms

/BA1/C55CURPOS

Reason and Prerequisites

This note is relevant for

  • Enhanced AFI Logic, formerly Accounting for Financial Instruments Based on Expected Cash Flows (ECF)

This symptom only occurs if

  • there are several currency positions for the same contract/position,
  • there are no CFs delivered,
  • there are no target values delivered (for any calculator) on the key date

Solution

Implement the attached correction instructions or the corresponding support package
