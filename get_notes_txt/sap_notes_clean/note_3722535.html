Symptom

The carrier is replaced after creating settlement documents and credit memo.

User attempts to create FSD from UI. New window opens for FSD creation with blank screen without any messages.

Other Terms

Invoicing Status Not Updated, Manual FSD Creation Opens Blank Screen

Reason and Prerequisites

Carriert does no match with the existing active FSDs and CMs.

Solution

Cancel the FSDs and CMs of previous carrier to settle the new carrier.

Implement the attached correction instructions to have messages displayed on the blank screen.
