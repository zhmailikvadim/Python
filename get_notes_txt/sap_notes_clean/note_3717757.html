Symptom

In the Manage Force Elements app, when copying a force element in the background, change documents are not generated correctly if the source force element is not assigned to any measure.

Additionally, the Valid From date is not converted to the correct date format when navigating from the Manage Force Elements app to the Copy Force Element in Background report.

Other Terms

S/4HANA Defense and Security, Manage Force Elements, Background copy, DFS_FE_MASS_COPY

Reason and Prerequisites

Program error

Solution

Implement the attached correction instruction.
Please also implement the frontend Note 3722638 for the fix to work correctly.
