Symptom

  • Error: You cannot request a quantity greater than the quantity available.
  • Quantity exceeds the licenses purchased
  • Creation of license key results in an error
  • Cannot request a new license key due to error
  • Unable to create BusinessObjects license keys

Environment

SAP for Me > License Keys

Reproducing the Issue

While trying to request a BusinessObjects license key, receive an error indicating you cannot request a quantity greater than the quantity available.

Cause

The License Keys application keeps track of the number of license keys (quantities) already requested for each license material, version, and production type. If the number of licenses being requested is greater than the remaining number of licenses available for that material/version, an error is produced.

Resolution

When creating license keys, the quantity of licenses you are licensed for in your contract is displayed in the material drop-down (e.g. Licensed = #). The quantity currently available is displayed in the version drop-down (e.g. Remaining number = #). If the "Number of Licenes" you enter is greater than the quantity remaining, you will receive an error.

In the example below, license material 7018569 has a total licensed quantity of "3". However, it appears license keys for the full quantity (3) were already created for that particular version (e.g. 4.2) and license type (e.g. Production). The version drop-down will indicate the number of licenses that you can create (based on other license keys already generated). In this case, there are zero (0) remaining licenses available.

Tips:

  • To view already-generated license keys for your account, see KBA 2033055 
  • If license keys are deleted (see KBA 1845621), the "Remaining number" of licenses available will increase
  • If you need a quantity greater than the "Licensed" amount, plesae contact your SAP Account Manager to have your licenses updated
  • For Concurrent Session, CPU, and CAL based license materials, this quantity will deplete as keys are requested for production deployments. For Named user license materials, this quantity does not deplete because a named user can have access to multiple systems. More detail information please refer to KBA 2947504

See Also

  • If you have issues generating license keys create a case for component XX-SER-LIKEY-BOJ and include the installation number, product name, product version, and license material. It's also helpful to include screenshots of any errors you receive.
  • License Key How-to docs can be found at https://support.sap.com/keys-systems-installations/keys.html (expand License Keys Help).
