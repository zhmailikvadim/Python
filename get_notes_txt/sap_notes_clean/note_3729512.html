Symptom

The system displays warning message 083(HRPAYFR_PY) "Le split WPBP est requis pour la rubrique &1." without sufficient detail to identify the root cause during payroll processing, especially in retroactive scenarios.

The message does not provide key contextual information such as payroll period, wage type group type, and legal code.

This lack of information makes analysis difficult for you, particularly when you work with blocks such as 51 or 52.

Other Terms

WPBP split, wage type, payroll period, legal code, wage type group type, FREM, retroactivity

Solution

After implementing this SAP Note, the system displays updated message 083(HRPAYFR_PY) "La rubrique &1 requiert un split WPBP (pér. &2, type grp rub &3, code &4)" with three additional parameters: payroll period, wage type group type, and legal code.

These corrections are available with the HR Support Packages referenced below.
If you have an urgent need of these corrections, see the correction instructions provided.
