Symptom

In agent.log (note 1292836), below similar error can be seen:
com.sap.solman.agent.mai.GenericMetricCollector collectForMetricGroup failed
com.sap.solman.agent.mai.api.CollectorException: com.sap.solman.agent.hostagent.HostAgentNotAvailableException: HostAgent stub com.sap.solman.agent.jax.ws.hostagent.saphostcontrol.SAPHostControlInterfacegetCIMObject timed out.

HANA indexserver trace shows below similar error:
w Statistics       StatisticsMonitors.cpp(00304) : StatisticsMonitor::handleRemoteRequest[M_HOST_AGENT_METRICS] failed

Query view M_HOST_AGENT_INFORMATION_ and M_HOST_AGENT_METRICS also failed with error "could not query from ...".

Cause

Resolution

Maintain etc/hosts for host agent as per below guide:

Configuration of the Hosts File | SAP Help Portal
