Symptom

You want to change or maintain a new number range for document types.

Environment

SAP S/4HANA Cloud

Reproducing the Issue

  1. Go to the Implementation Activity Define Document Types.
  2. Select any SAP delivered document type or your own created document type. For example document type '00'.
  3. Under Dialog Structure, double click 'Number Ranges/Auto Reversal'.
  4. For version XYZ, select for example Number Range '01' and select 'Display Number Range'. (XYZ stands for the consolidation version ID)

Here on the page 'Number Range Object EC-MC: Documents', all the fields are greyed out and no action can be performed.

Cause

For document types for automatic posting, you can only change the description text. You cannot add or change the number range.

Resolution

The number range of document type for automatic posting cannot be changed. This is the designed system behaviour.

Keywords

Number Range, Document Types, Number Ranges/Auto Reversal, Define Document Types.
