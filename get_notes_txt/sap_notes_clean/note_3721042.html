Symptom

In SAP S/4HANA on-premise versions 2022 and 2023, you cannot find SGC5* related e-document types in table EDOTYPE for electronic invoice processing in Singapore - Submission to Tax Authority.

Other Terms

e-invoice, eInvoicing, e-Invoicing, electronic invoice, electronic invoicing, Singapore eInvoice, SG eInvoice, e-document, eDoc, e-doc, document compliance, digital compliance, DC, document and reporting compliance, electronic document processing, C5

Reason and Prerequisites

The SGC5* e-document types for Singapore electronic invoice submission to the tax authority are missing from table EDOTYPE in OP 2022 and OP 2023.

**Prerequisites:**

- You use SAP S/4HANA on-premise version 2022 or 2023.
- You require electronic invoice processing for Singapore.

Solution

Version Date Changes
1 February 26, 2026 Released for customers

Import the Support Package specified for your release. Alternatively, implement the attached correction instruction and then carry out the attached manual activities.

As a general rule, SAP recommends that you implement the Support Package specified for your release. However, if you need to install a solution earlier, use the SAP Note Assistant to implement the correction instruction. For more information, see SAP Note Assistant.
