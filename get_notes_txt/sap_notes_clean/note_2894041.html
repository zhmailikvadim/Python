Symptom

When user tries to navigate from Defects app (i2d.qm.defect.records1) to object page of Manage Production Operations app (i2d.mpe.operations.manages1), it navigates to worklist view of operations app.

Ideally, it should take user to details screen of operations app (details screen for order number, which user navigated from defects app).

Other Terms

i2d.mpe.operations.manages1, i2d.qm.defect.records1, Supervisor apps

Reason and Prerequisites

Navigation was not properly handled in component.js file of Manage Production Operations app.

Solution

Implement this note to navigate from Defects app, to details page of Manage Production Operations app.
