Change 3/20/2026:  Missing correction instructions added for message HRPAYDE_QG 017.

Symptom

This SAP Note contains corrections for the functions for the settlement of qualification money. These are currently being piloted and have not yet been generally released.

List of corrections:

  • When you enter the lost work for a development action using RHC substitutions in the infotype Substitutions  (2003), lost hours were also generated before the program correction if the substitution was entered for a public holiday. 
  • If the qualification allowance is not paid for the entire month, a proportional planned remuneration and actual remuneration must first be formed to calculate the notional remuneration for SI. To calculate the proportional planned and actual remuneration, a daily amount of 1/30 of the monthly amount is previously formed in the SAP system. This value, rounded to 2 decimal places, is then multiplied by the reference days of the qualification money. However, according to the specifications of the ITSG, the calculation is to be performed in one step without rounding the daily amount. After you implement the program corrections, the calculation is adjusted to the specifications of the ITSG. 
  • The documentation is currently incomplete. This SAP Note adds an overview documentation with general information about qualification money in the SAP system. 

Solution

Implement the attached correction instructions or import the specified support package.
After you import the HR Support Package or implement the correction instructions, you do not have to perform any further follow-on activities.
