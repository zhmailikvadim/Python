Symptom

    Page Schema View shows that the SCIM API SuccessFactorsExtendedAttributes is set as "To be deprecated". What does that mean exactly?

    Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

    Environment

    • SAP SuccessFactors HCM Suite
      • SCIM API

    Resolution

    The deprecation of these attributes do not mean that they will be removed.

    The current plan is to actually move them to another section in SCIM API, not remove them. So, if you use them in your integrations, you should not be affected by this deprecation.

    There are no planned dates for when this will happen. But when it happens, communications will be sent with the details so there can be proper timing for testing and adjusting.
