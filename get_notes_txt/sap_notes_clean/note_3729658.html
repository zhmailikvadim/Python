Symptom

The "Last Modified Source" field is visible in the Compensation portlet in the Latest People Profile.

Environment

SAP SuccessFactors Employee Central: Latest People Profile

Reproducing the Issue

  1. Search for a user with compensation details.
  2. Navigate to the Compensation Information category.
  3. Click on "View All" on the Compensation Information card.
  4. Observe that the "Last Modified Source" field is visible.

Cause

  • This is known error.

Resolution

The fix for this issue is planned to be deployed on 1H 2026 release.

For release timelines, please review SAP SuccessFactors Product Release & Road Map Information.
