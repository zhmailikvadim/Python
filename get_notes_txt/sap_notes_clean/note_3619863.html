Symptom

When opening Database Explorer the application opens but nothing is displayed.
Message in the bottom right displays the message:
Tasks in Progress 
Loading SQL Consoles...

Environment

XSA

Reproducing the Issue

  1. At OS level, confirm the URL for webide with the command: xs apps
  2. Confirm that the xsa-cockpit is working without issues
  3. Connect and can login to the webide URL without issues
  4. Database Explorer loads but does not display any of the SQL console tabs
  5. WEB IDE tab loads forever
  6. Issue persists when using a different browser or clearing the cache

Cause

If XSA is accessible from OS level, and the login connection to XSA Cockpit and other XSA applications is possible, then this can indicate a performance issue on the database level.

Resolution

Confirm the health of the database by verifying performance and availability of disk and memory.
If the database recently experienced a disk full or other resource issue, and the issue persists when the database is healthy, perform a restart of XSA from OS level.

Should the issue persist, perform a diagnostics of XSA on OS level with the command:
XSA diagnose
