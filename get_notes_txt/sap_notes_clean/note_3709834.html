Symptom

In the process "Preparation and Waiting for Sending of Meter Reading (MOS MaLo View)" (DE_B_MET_PREPARE_SEND_MR_MOSMALO), existing meter readings are incorrectly determined as missing in step 0300. This causes an error in the next step 0350.

Other Terms

US4G, /US4G/CL_PRST_DET_MR_ORDER, GET_MISSING_MR_ORDERS, DE_B_MET_PREPARE_SEND_MR_MOSMALO

Reason and Prerequisites

 The error occurred because of a program error in the device comparison logic.

Solution

Implement the corrections of this SAP Note or import the relevant support package.
