Symptom

  • Error message No valid template found
    • This error appears when importing a ZIP file template from SAP Ariba Test to Production realm

Environment

SPM Projects, test, production, template, migrate

Reproducing the Issue

  1. Export the template from the SAP Ariba Test realm.
  2. Import the ZIP file template to the SAP Ariba Production realm.
  3. Observe the error message No valid template found

Cause

  • The SIPM feature SM-5060 was not enabled, which is required for successful template import.

Resolution

  1. Enable the SIPM feature SM-5060 in the SAP Ariba Production realm.
  2. Import the template again to the Production realm.
  3. Manually update the supplier profile questionnaire with the necessary questions.
