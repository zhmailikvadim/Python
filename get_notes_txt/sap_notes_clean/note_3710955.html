Symptom

When making a request from an apikey/site using a specific SDK, 400001 "Invalid request format" is returned.

Environment

  • SAP Customer Data Cloud
  • SDK Validation

Cause

The 'Enable SDK Validation' setting is turned on for the apikey.

Resolution

  1. Check the 'Enable SDK Validation' setting for the apikey under Identity > Security > SDK management.
  2. Either turn off the 'Enable SDK Validation' setting or allow the required SDK/REST API to accept requests sent from specific SDKs.

Keywords

Gigya, Error Code 400001
