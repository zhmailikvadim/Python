Symptom

Slowness is observed when uploading files using the Excel Data Loader to update positions.

Environment

  • SAP Sales Performance Management
  • Incentive Management on Hyperscalers

Reproducing the Issue

  1. Upload an Excel file to update positions using the Excel Data Loader.
  2. The system prompts an alert stating the job has started.
  3. No progress updates or execution status are displayed.
  4. The process appears stuck due to the lack of feedback or alerts.

Cause

This is a Known Issue.

Resolution

The fix for this issue is planned to be deployed with the patch 2605. For patch timelines, please review SAP SPM 2026 Release Schedule - SAP Community 

Workaround

No workaround is available.
