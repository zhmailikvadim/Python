Symptom

When you implement correction instructions with complex attributes (TYPTYPE = 4) that have the READ-ONLY indicator set, the keyword READ-ONLY is generated in duplicate in the source code for the section. This results in a syntax error and you are unable to activate the classes or interfaces subsequently.

Other Terms

Class Builder, READ-ONLY, SCWB, Note Assistant, LSEMCLIFF01

Reason and Prerequisites

You have used the Note Assistant or Correction Workbench to change a class or interface.

This correction is relevant as of SAP_BASIS 740.

Solution

Import the support package or implement this SAP Note.
