Symptom

This SAP Note includes multiple corrections for the "Usage of APIs" ATC check, which is part of the clean core category.

  • Fix compatibility with other ATC checks that are checking object type DEVC
  • Correctly report nested database operations as Updating or Selecting
  • Report the usage of DDIC views and DDIC database tables in DDIC views
  • Function modules where the function group changes between releases are now correctly reported
  • Prevent an ASSERTION_FAILED short dump that may occur in CL_SATC_CI_CHECK_MODULE=>EVALUATE_FINDINGS or CL_SATC_CI_CHECK_ENH_MODULE=>EVALUATE_FINDINGS when the "Usage of APIs" check tries to report a check failure for an obsolete configuration

Other Terms

Clean Core, CL_YCM_CC_CHECK_API_USAGE, Classic APIs, ABAP Extensibility Guide

Reason and Prerequisites

You're performing ATC checks "Usage of APIs" belonging to category "Clean Core".

Solution

Import the support package or implement this SAP Note.
