Symptom

When you try to change the status of a Planned or Released Production Order to Canceled from the Open Items List, the system displays the message: Status of Production Order X cannot be changed.

If you go to an individual document and try to change its status to Canceled, the system displays the message: Action is valid for planned or released production orders with no transactions [Production Order - Product No.]  [Message 3702-18].

Environment

SAP Business One 

Cause

System Behaviour

Resolution

This is the correct system behavior: once a document links to a Production Order, we cannot change its status to Canceled; we can only change its status to Closed.
