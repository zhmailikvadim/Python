Symptom

When running reports like S_ALR_87013558 or using form in CJE0 the option to export to excel is not available 

Environment

  • SAP ERP Project Systems (PS)
  • SAP R/3
  • SAP R/3 Enterprise
  • SAP ERP Central Component
  • SAP ERP
  • SAP Enhancement package for SAP ERP
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition 2023

Reproducing the Issue

  1. Run S_ALR_87013558  or CJE0
  2. Select form (if needed )
  3. Enter project
  4. Execute .

Cause

The option is not available using Graphical version of the report.

Resolution

You need to change the report to Classical 
Run CJE0  and select Form  eg 

In the OutPutType Tab 
Select either Classical or Graphic 

Save the changes
