Symptom

 In the process "Cancellation of Supply for Gas Division (SUPO View)" (DE_B_GELIGAS_CAN_SUPO), the system threw a classic exception in /US4G/IF_BADI_GPKE_SUPO_CANC instead of a class-based exception.
This issue is relevant for the market role: SUP.

Other Terms

/US4G/IF_BADI_GPKE_SUPO_CANC, classic exception, class-based exception, GPKE_SUP, gas division, SUPO

Reason and Prerequisites

The error occurred because of a coding issue that caused the system to throw a classic exception instead of a class-based exception.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the system throws a class-based exception instead of a classic exception in /US4G/IF_BADI_GPKE_SUPO_CANC. You can verify the fix by triggering the process DE_B_GELIGAS_CAN_SUPO and confirming that the system handles exceptions correctly.
