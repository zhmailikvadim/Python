Symptom

Error message AA651 encountered during execution of transaction AJRW.

Environment

  • FIN Asset Accounting​
  • SAP ERP​
  • SAP ERP Central Component
  • ​SAP enhancement package for SAP ERP​
  • SAP enhancement package for SAP ERP, version for SAP HANA​
  • SAP S/4HANA ​
  • SAP S/4HANA Finance​
  • SAP S/4HANA Cloud Private Edition​

Reproducing the Issue

  1. Execute transaction AJRW for fiscal year XXXX;
  2. Observe error message AA651 during the process.

Cause

Multi-level method incomplete, Validity Period in Years field is 0. By standard it is 999.

Resolution

  1. Access the customizing settings for multi-level method in tCode AFAMS.
  2. Complete the missing year information in the method.
  3. Re-execute transaction AJRW for fiscal year XXXX.
