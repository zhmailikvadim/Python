Symptom

After clicking Excel Export from Collections Browser in Backoffice, you get a 502 proxy timeout error in the UI.

Environment

Cloud Commerce Backoffice.

Reproducing the Issue

  1. Go to Backoffice.
  2. Open Report Definition. 
  3. Open Report.
  4. Select from date to date.
  5. Click Search for result.
  6. Click On Export Button.

Set backoffice.excel.export.max.rows= to a higher number, default is 2000, safety threshold is 500 items. Also having a large number of attributes for each item in the report can trigger a proxy timeout.

Cause

Too many items and/or attributes to be returned before proxy server's timeout expires.

Resolution

Even though the UI times out in the foreground, the report may still be generated in the Background, check your download location for the report.

To fix UI issue and speed up report generation try:-

1. Check the timeout value for Backoffice:

3455527 - Error: 502 - Bad Gateway or Proxy Timeout 

2. Check how many attributes are necessary to export with each item. It is recommend to only export the attributes you need, as if you have 100 attributes exported for 200 items thats 20,000 attributes in total, and localised attributes may require more processing. See:

Exporting an excel file

3. The report will work by generating SQL's which should be visible in Dynatrace monitoring, check if these need additional indexes to function faster and check that they are executed on the READ ONLY REPLICA DB:

Check if the report queries are using read-only DB (can be that all Backoffice queries are going to READ ONLY REPLICA DB by default see):

Querying the Read-only Replica

or Hint the report queries to use Read-only DB:

Query Categorization

Querying the Read-only DB is better for reporting functions.

Note: The Excel export tool is a report on Product Configuration meant for bulk product updating. By default, export and import actions are available in Collection Browser, for a Product type. If used to create other types of reports (such as customer or promotions reports) it may need customisation.
