Symptom

As part of data dump (DB Extract) process, customer was looking for SQL table definitions in the form of CREATE TABLE scripts for extracted tables (to import the extractions into their database).

An option mentioned in the Microsoft documentation: https://learn.microsoft.com/en-us/sql/relational-databases/tables/view-the-table-definition?view=sql-server-ver17

Environment

  • SAP Sales Performance Management
  • Agent Performance Management 

Cause

Due to decommissioning the environment, customer requested data to be exported from various tables of database, that is needed for audit purposes.

For these tables customer was looking for SQL table definitions in the form of CREATE TABLE scripts, that would help them expedite the process to import the extractions into their database.

Resolution

Engineering confirmed that CREATE TABLE scripts cannot be provided, therefore the customer was advised to use the Data Dictionary (A list of fields available on each form, and provides specific details regarding each field and form. This tool allows the customer to find information on any type of entity in the database) for SQL table definitions.

On the UI of APME (There are two ways to access the Data Dictionary: from the Help menu, or by double-clicking a field or other label on a page.) an option is available to export the information from Data Dictionary to PDF.
