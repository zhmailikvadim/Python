Symptom

  • Multimedia Widget with video shows the preview image broken after saving it
  • A broken image appears for uploaded videos in Widget

Environment

  • SAP Build Work Zone Advanced Edition
  • SAP SuccessFactors Work Zone

Reproducing the Issue

  1. Navigate to the affected page and scroll down
  2. Enter edit mode and access the settings of the Multimedia Widget
  3. Paste the URL of the locally stored video into the input field
  4. Observe that the video is loaded and a preview image is visible
  5. Save the widget and observe that the preview image disappears, and a broken image appears in the upper right corner of the container

Cause

The "Videos/Audios" feature in the Admin Console → Feature Enablement → Features is disabled. 

Resolution

To resolve this scenario, follow below steps:

  1. Enable the "Videos/Audios" feature in:
    • Admin Console → Feature Enablement → Features
  2. Re-upload any videos that were uploaded before enabling this feature

When this feature is disabled, uploaded videos are treated as regular documents, preventing subsequent processing such as transcoding and thumbnail generation.
