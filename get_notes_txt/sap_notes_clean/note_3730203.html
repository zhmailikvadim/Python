Symptom

When displaying agreement postings for multiple agreements that belong to two different agreement types, the actual accrual amounts are shown as doubled in the periods view. As a result, the values displayed do not match the expected actual accrual amounts.

Other Terms

/IRM/IPAGP, calc_perd_amt_general_prepare, ACAMT, Actual Accruals

Reason and Prerequisites

This is caused by program error.
Note : SAP Note#3730244 is available for V/4 release.

Solution

Please implement the correction instructions attached. These corrections are only valid if you have applied the Support Packs

Release 6.4 – Base Package & Support Pack 1
Release 6.3 – Base Package

Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object
