Symptom

From 2022 dismissal compensation for hourly/daily paid employees needs to be calculated in same way as for salaried employees.

Other Terms

dismissal compensation, GR13, GRD2, T511K

Reason and Prerequisites

This SAP Note is relevant only for Greece.

Solution

Payroll rules GR13 (Division of valuation bases (GR)) and GRD2 (Dismissal Compensation) have been changed to calculate the dismissal compensation for hourly/daily paid employees in a same way as salaried employees. New calculation is used from 2022 on.

New constant has been added to table T511K:

Constant Text From To Amount
DCDPM Dismissal Compens. Days/Month 01.01.2022 31.12.9999 22

This constant is used to calculate base wage type /003 (Salary /monthly rate/) for hourly/daily paid employees. Dismissal calculation then uses wage type /003 in the same way as it is used for salaried employees. Before 2022 wage type /003 was not calculated for hourly/daily paid employees.
