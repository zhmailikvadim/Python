Symptom

During the reversal of a move-in, errors occur, for example: EMV-116 "There are non-reversed documents: &2 &3 &4", >1-720 "Reversal can only be executed with the reversal service", or >0-245 "&1 is reversal document or clearing reset. Resetting not permitted"

Other Terms

EC60 EMV116 116(EMV) >1720 720(>1) >0245 245(>0) EMV250 250(EMV) TFK001

Reason and Prerequisites

Payment documents still exist for the contract in question. These were processed, but a document was assigned to the contract to be reversed again.
However, in this case, the system no longer issues the relevant error message EMV 250 "Move-in reversal not possible; payment documents &2 &3 &4 exist".

The origin of this document was not taken into account correctly in the program checks. This affects the following values:

45   Payment Order Lot
1A   External Cash Payment
1B   External Card Payment
1C   External Check Payment
1D   External Postal Order
1E   External Payment
7A   Real-Time Payment (Outgoing Payment Bank)
7B   Real-Time Payment (Card)
7E   Real-Time Payment (Incoming Payment Bank)
7V   Real-Time Payment (Virtual Account)

Solution

Implement the program corrections. After you implement these corrections, the system issues error message EMV 250 again in this case.
The document reassigned by the reset must be handled differently, for example, by transfer posting to the contract account.
For more information about the move-in reversal, see FAQ Note 619930.
