Symptom

 In the processes "Deregistration (SUP View)" (DE_B_GPKE_DEREG_SUP) and "Deregistration (SUP View)" (DE_B_GELIGAS_DEREG_SUP), the importing parameter /APE/S_PROC_STEP_DATA_ALL was not available as optional in BAdI /US4G/IF_BADI_GPKE_SUPO_DEREG.

Other Terms


Deregistration, DE_B_GPKE_DEREG_SUP, DE_B_GELIGAS_DEREG_SUP, BAdI, /US4G/IF_BADI_GPKE_SUPO_DEREG, /APE/S_PROC_STEP_DATA_ALL, importing parameter, GPKE_SUP

Reason and Prerequisites

 This SAP Note addresses a customer implementation request to enhance the BAdI interface by adding an optional importing parameter.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the importing parameter /APE/S_PROC_STEP_DATA_ALL is available as optional in BAdI /US4G/IF_BADI_GPKE_SUPO_DEREG. This enhancement doesn't affect existing implementations. You can verify the fix by checking that the importing parameter /APE/S_PROC_STEP_DATA_ALL appears as optional in the parameters of BAdI /US4G/IF_BADI_GPKE_SUPO_DEREG.
