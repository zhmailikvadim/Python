Symptom

In SE16N, if don't input any variant in field "Displ. Variant", in "save as", there's no  checkbox “default”.

Environment

  • Controlling (CO)
  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
    • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA
  • SAP S/4HANA Finance
  • SAP S/4HANA Cloud Private Edition
  • SAP Fiori for SAP S/4HANA

Reproducing the Issue

In SE16N, if don't input any variant in field "Displ. Variant", in "save as", there's no  checkbox “default”.

Cause

This change in behavior after upgrade is introduced by Note 3232958, since there is a potential issue wherein the default layout in ALV can overwrite the field catalog that SE16N/SE16H creates according to your selections. Therefore, we have decided not to allow default layout saving and have created Note 3232958 to prevent such overwrites. So the Default checkbox is not visible after upgrade.

Prerequisites: Corrections from Note 3232958 and Note 3566577 are implemented before applying this Note.

Resolution

Solution, please refer to note 3642760.
