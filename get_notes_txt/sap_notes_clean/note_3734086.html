Symptom

How to hide Pie Chart in Compensation Information in Latest People Profile?

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental." 

Environment

  • SAP SuccessFactors Employee Central

Reproducing the Issue

  1. Login to the instance.
  2. Go to an user's Latest People Profile.
  3. Go to Compensation Information.
  4. View there is a Pie Chart.

Resolution

This feature will be live in customer environment from 1H2026 release. 

  1. Go to Configure Latest People Profile.
  2. Edit Compensation Information card.
  3. There is an option 'Hide Pie Chart'. 

 

Keywords

SAP Employee Central, Compensation Information, Latest people profile, Pie Chart, 1H2026 release.
