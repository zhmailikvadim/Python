Symptom

Users are unable to create a Job Requisition when selecting “Browse Families and Roles” during the requisition creation process.

Upon clicking the option, the system does not proceed to the expected page and instead remains on the Select Job Requisition Template screen. No error message is displayed.

Environment

SAP SuccessFactors Recruiting Management

Reproducing the Issue

  1. Go to Recruiting > Job Requisition.
  2. Select "Create Requisition".
  3. Choose a Job Requisition Template.
  4. Click “Browse Families and Roles”.
  5. Observe that nothing happens and no error is shown.

Cause

The translations for “Family” and “Role” entities are not configured in Manage Data for the user’s default login language.

As a result, when users attempt to select “Browse Families and Roles”, the system cannot properly load the required data, causing the page to remain on the Select Job Requisition Template screen without displaying any error message

Resolution

Ensure that translations for “Family” and “Role” entities are maintained for the user’s default login language.

  1. Navigate to Admin Center.
  2. Go to Manage Data.
  3. From the first dropdown, select Family or Role.
  4. Select the specific Family or Role record.
  5. Click on the Globe icon beside the Family Name or Role Name.



  6. Maintain the required translation for the user’s default login language.
  7. Save the changes.
