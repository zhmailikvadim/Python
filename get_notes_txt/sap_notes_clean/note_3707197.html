Symptom

Other Terms

S/4HANA, SAP LT Replication Server, SAP HANA, Trigger, Database, Migration Cockpit

Reason and Prerequisites

SAP LT Replication Server is used on one of the following releases:

S4/HANA OP2023 SP00 - SP05
S4/HANA OP2025 SP00 - SP01

Solution

Implement the corrections on the SLT system.
