Symptom

When running the employee replication program there are no requests reaching the staging area (ECPAO_EE_VIEWER) during employee replication using BIB

The SLG1 logs for the employee show the following error message:

You don't have the authorization to change table ECPAO_QRY_ADM

Message No. ECPAO_IN_QUERY002

Reproducing the Issue

  1. Replicate employee from EC to ERP using report ECPAO_EE_ORG_REPL_QUERY
  2. Check the staging area ECPAO_EE_VIEWER and you see there are no entries
  3. When checking the SLG1 logs for the employee the following error message is shown:

You don't have the authorization to change table ECPAO_QRY_ADM

Message No. ECPAO_IN_QUERY002

Cause

The replication user maintained does not have the necessary authorizations to make changes to the ECPAO_QRY_ADM table so the replication fails.

Resolution

Ensure that the necessary authorizations are provided to the replication user maintained so that changes can be made to the ECPAO_QRY_ADM table.

Keywords
