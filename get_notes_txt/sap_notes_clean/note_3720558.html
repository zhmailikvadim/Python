Symptom

While creating a new implementation in the Custom Logic application, a specific BADI is not available to select.

Environment

  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Access to the Custom Logic application.
  2. Create a new implementation.
  3. Search for the BADI in "Extension Point" search value, it shows no relevant result.

Cause

The BADI may not be available with the current system.

Resolution

Check the compatibility of the BADI with the system by following below steps:

  1. On SAP GUI, kindly access transaction /SCFD_REGISTRY.
  2. Go to tab BADI Registration
  3. Find and click on the BADI
  4. You will see which system this BADI is available.
    • OnPremise: It will be available on On premise and Private cloud system.
    • Cloud: It will be available on Public cloud system.

Keywords

BADI, extension point, custom logic
