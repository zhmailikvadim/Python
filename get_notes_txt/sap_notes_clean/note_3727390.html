Symptom

You would like to perform a goods receipt reversal in /SCWM/PRDI on HU tab, by selecting the HUs that were posted goods receipt earlier. The GRR will be terminated with a dump if: the HUs contains the original item only partially and an automatically generated packaging item is already added to the hierarchy (IPHU item) and if the HU number contains also alphabets, not just numeric characters. This note corrects the code not to raise dump, but you should know that the GRR won't be possible for partial amounts on HU tab. After implementing this note, you will see the proper error message about that issue.

Other Terms

IPHU, GRR, CHECK_GM_DOCUMENTS

Reason and Prerequisites

 Program error

Solution

 Implement the correction instruction or the equivalent support package
