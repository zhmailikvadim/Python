Symptom

 Validation errors on payment item level during MT940 file processing.

Other Terms

MT940, Field 61, Reference of the Account Servicing Institution,/PF1/FH_CONV_XSD005, SWIFT/MT940/TextBL/SeqB/f-61/Detail

Reason and Prerequisites

According to SWIFT documentation MT940 field 61 subfield 8 "Reference of the Account Servicing Institution" is an optional field. But if it is not provided in incoming MT940 file, then field 61 can not be correctly parsed and getting fully mapped into SWIFT/MT940/TextBL/SeqB/f-61/Detail tag. It causes validation errors during PO processing.

Solution

Regular expression pattern is adjusted in text adapter. If "Reference of the Account Servicing Institution" field does not exist in incoming file, then other fields from field 61 will be mapped accordingly.

Please apply SAP Note or assigned Support Package.
