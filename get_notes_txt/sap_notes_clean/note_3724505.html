Symptom

A valid inspection plan exists. But inspection lot is created in CRTD status and inspection plan can't be assigned in QA02. Error "No inspection plan could be found" (QA278) occurs when selecting inspection plan in QA02.

Environment

  • SAP ERP Quality Management (QM)
  • SAP R/3
  • SAP R/3 Enterprise
  • SAP ERP Central Component
  • SAP ERP
  • SAP Enhancement package for SAP ERP
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA, on-premise
  • SAP S/4HANA Cloud Private Edition

Cause

Source inspection lot is created against a Purchase Order, and the order unit used in the purchase order differs from the material base unit. The unit conversion is not maintained in the material master.

Resolution

Maintain unit conversion between the order unit and base unit in the material master.
