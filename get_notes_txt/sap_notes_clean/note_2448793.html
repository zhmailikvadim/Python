Symptom

This document will show you how save data in a SAP PowerBuilder DataWindow with data that has been retrieved from the database.

Environment

SAP PowerBuilder 12.6

Resolution

  1. To Save data in the DataWindow, open the DataWindow in the SAP PB IDE DataWindow painter.
  2. Right click in the data view and choose Retrieve to retrieve the data. 



  3. After the data has been retrieved in the DataWindow, click File ->Save As or  File->Save, depending on whether you need to retain the original DataWindow without the data.



  4. If you later open the saved DataWindow, it will have data saved in it. 
  5. If desired you can edit the data in the DataWindow in the data view.
