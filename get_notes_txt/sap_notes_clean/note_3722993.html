Symptom

  • Distribution channel and division are not determined when creating a follow-up sales order from a visit generated via Dynamic Visit Planning (DVP) add-on.
  • The Sales Organization Determination Method for visits created via DVP is displayed as "Provided by an Unspecified External Source".

Environment

SAP Cloud for Customer

Reproducing the Issue

  1. Create a Route Group with no Distribution Channel and Division maintained.
  2. Activate the Route Group and generate a visit list.
  3. Generate a visit from the list menu.
  4. Start and finish the visit.
  5. Create a new sales order from the visit.
  6. Observe that the distribution channel and division are not determined in the sales order.

Cause

For visits created via the Dynamic Visit Planning (DVP) add-on, the visits are treated as being created from an external source.

This classification prevents the follow-up determination of the distribution channel and division.

Resolution

Ensure that the Division and Distribution Channel are maintained in the Route Group. This configuration allows the system to correctly redetermine the Division and Distribution Channel when creating a follow-up sales order from a visit generated via DVP.
