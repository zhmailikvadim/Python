Symptom

You are running Interaction Center functionality in S/4 with email integration via ERMS and SAP Connect. While opening certain emails from the Inbox list the inline images are not getting displayed correctly. Only placeholders are visible. This happens most likely when emails have been sent by Apples Mail program and server.

Other Terms

inline images, Apple Mail program

Reason and Prerequisites

When you inspect the MIME contents of those emails, there are references to embedded images similar to this one:

cid=<mf_177A18F2-2AB5-462E-9D1B-4F05A0ABC28F/L0/001>

This format interferes with our own KPRO format which also contains 2 slashes ("/"). Therefore KPRO logic gets applied instead of reading the data correctly from MIME repository.

Solution

Apply the attached correction instruction according your S4CRM or S4CEXT release or upgrade to the corresponding support package version mentioned in the correction instructions.
