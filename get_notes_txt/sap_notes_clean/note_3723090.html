Symptom

Wrong variable position in Arabic translation of message M3 351

Other Terms

SE63

Solution

Translation of objects using transaction SE63 (see further below) / Delivery with Support Package

SAPKH60637 for SAP_APPL 606 SP 0037
SAPKH61732 for SAP_APPL 617 SP 0032
SAPK-61826INSAPAPPL for SAP_APPL 618 SP 0026



Transport Object:

PgID Object Name
LIMU MESS M3351

 

Package: MG
Software Component: SAP_APPL
Application Component: LO-MD-MM


English Text: Material & not maintained in plant &
Wrong Target Text: لم تتم صيانة المادة & في المصنع &
Correct Target Text: لم تُعالَج المادة &1 في الوحدة &2


The error can be corrected in translation object:

Object type (shortcut): MESS
Object name: M3                  351


(Please check if the texts fit to your Support Package level!)

Please use transaction SE63 to translate the text(s).

  1. Enter /oSE63 in the command field (transaction field) and press Enter
  2. Enter Object type (see above) in the command field and press Enter
  3. Enter Object name (see above) in the field Object Name
  4. If required change Source and/or Target language
  5. Press Edit
  6. Enter or correct the translation in the text field below the source text
  7. Choose Save to save your translation(s)
  8. Choose Back (F3) to leave the translation editor

Correction delivery schedule can be found on SAP SUPPORT PACKAGE STACK SCHEDULE:

https://support.sap.com/content/dam/support/en_us/library/ssp/release-upgrade-maintenance/maintenance-information/schedules-for-maintenance-deliveries/SAPschedule.pdf

Note:

Translations performed in a customer system may be overwritten in the following instances:

  • If the object happens to be part of a future support package
  • Upgrade to a new release
  • Import of new language CDs from Support Releases
