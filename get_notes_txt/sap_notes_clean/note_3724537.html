Symptom

- It is needed to issue DC-e model 99 and the monitor J1BNFE is not prepared to handle this new model.
- The system does not generate the XML for DC-e, including the mandatory TAG identifier for Model 99.

Environment

SAP ERP (all versions)
SAP DRC (all versions)

Reproducing the Issue

1. Attempt to process Modelo 99 (DC-e) using Monitor J1BNFE.
2. Attempt to generate XML for DC-e, including the mandatory TAG identifier for Model 99.

Cause

The system is not prepared to handle this new model 

Resolution

- It is necessary to develop an own solution to deal with DC-e Model 99.
- This is current not supported. 

Keywords

Model 99, Modelo 99, J1BNFE, DCE, DCe, DC-e, Tag identifier
