Symptom

  • The implementation of this SAP Note has no impact on your existing functionality.
  • This SAP Note delivers changes that are usually treated as manual activities within SNOTE (for example, packages, ABAP dictionary objects, message classes, table content).
  • Before implementing the primary SAP Note 3729512, implement this SAP Note.

Solution

Implement the attached correction instructions by using transaction SNOTE or the corresponding HR-SP or CLC before you implement the primary SAP Note 3729512.

By doing so, report NOTE_3731028 is installed in your system.
