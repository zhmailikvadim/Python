Symptom

Recruiters can add AI-extracted skills and skills from Talent Intelligence Hub to the job requisition, job profile, and in interview central using a unified skills picker.

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

  • SAP SuccessFactors Recruiting Management
  • SAP SuccessFactors Talent Intelligence Hub

Resolution

Prerequisites

  • Ensure that the Skills Attribute field is defined in the job requisition template.

  • Make sure you've enabled Talent Intelligence Hub.

  • Run the TIH Skills Standardization job in AI Services Administration > Assisted Skills Standardization  > Scheduled Job Manager to enable the AI-Assisted Skills Standardization Setting.

  • Enable either or both of the following settings in Admin Center:

    • Assisted Applicant Screening in AI Services Administration > Feature Settings  > Recruiting.

    • Assisted Candidate Skills Recommendations in AI Services Administration > Feature Settings > Recruiting.

Procedure

  1. Choose the Add option under the skills field in the Job Requisition Details > Job Profile page.
  2. Accept the acknowledgment message before viewing the AI-extracted skills.
  3. Review the listed skills in the Add Skills pop up. The skills are divided into two sections:

    -Organizational Skills – Skills defined in the central attributes library of Talent Intelligence Hub.
    -External Skills – AI- extracted skills identified from a job description or job role using a universal taxonomy provided by a third-party vendor.
  4. Search for relevant skills using the search bar or filter using Tags. You can also enable the AI recommended Skills toggle to have AI filter relevant skills based on the job role, job description or other similar skills that have already been added.
    You can click on each skill to view more details about the skill. This opens the Skill Details pop up that will include a description of the skill and an explanation on why the skill is recommended.
  5. Select the required skills and choose Add.

Results

The added skills will be displayed in tabular form under the Skills field. Here, you can choose the Mark as Primary toggle to define the primary skills for the requisition.

These added skills will also be displayed in Interview Central.
