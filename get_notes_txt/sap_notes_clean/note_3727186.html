Symptom

The 'Delete' button is grayed out, preventing the deletion of a document type created by copying the wrong existing document type.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Open Implementation Activities.
  2. Select 'Define Document Types.'
  3. Click 'Edit' and select the document type intended for deletion.
  4. Click 'Menu' > 'Edit' > 'Delete.'

Cause

Deletion of customer records and SAP records is not supported in SSCUI 101522 - Define Document Types.

Resolution

  1. Confirm that deletion of document types created in SSCUI 101522 is not supported as per the SSCUI list in the roadmap view and related documentation.
  2. If deletion of the document type is critical, request Expert Configuration service through the appropriate representative (e.g., CSM/CSP, D-CSM, Account Executive).
  3. If there is no dedicated Customer facing role assigned then submit the feature request via Customer Influence Portal.
