Symptom

Transaction DB01 is missing an authorization default status setting for authorization object S_RFC_ADM.

You see this when checking transaction DB01with transaction SU22.
Alternatively, when using the ABAP Test Cockpit (ATC) you may see the following result:

   Check Title: Authorization Default Data Check (SU22 Data)
   Check Message: Undefined default statuses exist, authorization object ...
   Object Name: DB01
   Object Type: TRAN
   Package: STUN_DBA

Reason and Prerequisites

The required authorization default values were not previously delivered by SAP.

Solution

SAP delivers the missing data with the specified support package.
