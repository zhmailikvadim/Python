Symptom

You use the function module FOLDER_MAINTAIN in customer-specific programs to maintain folder data (table VDPOPO). As of SAP Note 3548049, change documents are written for table changes in the table VDPOPO and the changes are called in an update task.

This SAP Note provides an optional parameter I_FLG_UPDATE_TASK in the module FOLDER_MAINTAIN that enables you to control whether table changes are to be made in an update task or not. The default value is ABAP_TRUE (= Execution in Update Task).
