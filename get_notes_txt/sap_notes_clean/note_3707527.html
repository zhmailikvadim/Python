Symptom

You have used the keys VMENGE08 for additional quantity postings in configuration activity 500268 - Define Inventory Postings. However, the "Other quantity posting" field configured for VMENGE08 doesn't appear in the Inspection Lot Stock tab of Record Usage Decision app.

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental." 

 

Environment

SAP S/4HANA Cloud Public Edition

Cause

This is a known coding issue.

The posting proposal for UD VMENGE08 and VMENGE09 are artifacts of the Private Cloud solution. These fields are used in combination with an extension possibility (customer exit) that is not usable in S/4HANA Cloud Public Edition system.

Resolution

The values of Posting UD VMENGE08 and VMENGE09 will be removed in SAP future releases to avoid further configuration of an unsupported feature.
Please undo these entries and configure the other posting proposals according to business needs.

If it's needed to have an own, configurable posting line in the usage decision, raise a feature request through the Customer Influence Portal  under the session SAP S/4HANA Cloud Public Edition, Manufacturing.
