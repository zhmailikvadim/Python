Symptom

  • The web intelligence document is based on UNX which is using Java connector (JCO) -> SAP ERP connection. 
  • The connection is not using Save Language, so the PVL language is used.
  • The issue happens when Preferred Viewing Locale (PVL) is set as Japan (Japanese), refresh the webi with a special prompt value, error occurs as 
    The following database error occurred: [Data Federator Driver] [Server] [Connector 'connection name'] ConnectionServer has reported an Exception: String index out of range: 3. For information about this error, please refer to SAP Knowledge Base Article 2054721 on the SAP Support Portal. (IES 10901)
  • No issue if PVL is set as English (United States) even with the same prompt value
  • Even for PVL is Japan (Japanese), the issue is only happening for some special prompt value only. Using other prompt value it can also work.

Other Terms

webi, webintelligence, refresh, ERP, JCo

Reason and Prerequisites

Environment:

  • SAP BusinessObjects Business Intelligence Platform 4.2 Support Package 04
  • Web Intelligence

Steps to reproduce:

  1. Log on to BI launch pad.
  2. Open the problematic WebI document.
  3. Refresh and input the prompt value to run query.
  4. It fails with error : [Data Federator Driver] [Server] [Connector 'connection name'] ConnectionServer has reported an Exception: String index out of range: 3

Cause:

  • This issue is caused by a defect in the product.
  • It turned out that the issue is not related to PVL (Preferred Viewing Locale).
  • The issue is caused by the position of '/' character which means EOF (End Of File) in the data of SAP system.

Solution

  • This issue is fixed in the patches listed in the "Support Packages & Patches" section below.
    Due to different internal processes, the "Support Packages & Patches" section of the SAP Note will be populated as below:
                    - for fixes delivered in "Support Packages": immediately (even if the "Support Package" is planned to be released several months later)
                    - for fixes delivered in "Patches":  ONLY once the "Patch" is released AND available on SMP (not before)
    For Business Intelligence Platform maintenance schedule and strategy see the Knowledge Base Article 2144559 in References section.
