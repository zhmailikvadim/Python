Symptom

When you create a purchase order during the tax calculation the system raises a short dump.

************************************************************************************************************************************************

WARNING: This SAP Note is specific to IS-OIL/IS-MINE.

If you DO NOT have IS-OIL/IS-MINE installed on your system, this SAP Note DOES NOT apply to you.

Implementing this SAP Note without IS-OIL/IS-MINE installed can cause serious damage to your system.

Other Terms

IS-OIL, Brazil, DUMP, J_1BCALCULATE_TAXES

Reason and Prerequisites

This SAP Note is relevant only for Brazil.

You are using SAP Oil and Gas solution.

This issue is a side effect caused by the solution delivered by SAP Note 3730612.

Solution

After you implement this SAP Note, the tax calculation is performed as expected.

As a general rule SAP recommends that you implement the Support Package specified for your release. However, if you need to install a solution earlier, use the SAP Note Assistant to implement the correction instruction. You can find more information about the SAP Note Assistant in SAP Support Portal, under SAP Note Assistant.

Note that the documentation of certain objects is delivered with the corresponding Support Package only.
