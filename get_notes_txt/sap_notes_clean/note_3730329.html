Symptom

The process "Metering Location Change (MOS View)" returns an error when the device change installation date is the same as the POD start date.

The system attempts to retrieve meter reading orders for two days: the device change date and the day before the device change date. However, the POD isn't valid on the previous date because the POD is only valid from the device change date onward.

Other Terms

MACO, US4G, Metering, MDC, DE_B_WIM_MPM_MOS,Metering Location Change (MOS View)

Reason and Prerequisites

 Code issue

Solution

Please download and implement the SAP note in your target system.
