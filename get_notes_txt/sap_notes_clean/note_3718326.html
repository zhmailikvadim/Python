Symptom

When using 4AZ scenario to replicate contracts from Ariba to S4 and try to map the item texts from CIG, but on S4 side the interface input structure is missing an XML tag to handle text ID.
The corresponding dictionary type have the field, there is also a proper mapping logic to handle it, but it is missing from the input structure.

The missing field: input - purchasecontract - textcollection- texttypecode - content

Environment

  • Materials Management (MM)
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA

Resolution

Send the TextTypecode in Item text Collection node in the below format and test the behavior.

 <TextCollection actionCode="01">
    <TextTypeCode> </TextTypeCode>
    <CreationDateTime> </CreationDateTime>
    <ContextText>
     <TextElementLanguage>EN</TextElementLanguage>
     <TextElementText>Test</TextElementText>
    </ContextText>
   </TextCollection>
