Symptom

In Design Time Tool, HRASR_DT transaction, when a PDF form is generated, fields of the type STRING appear in the interface with a data type that is restricted to 255 characters. As a result, longer entries are truncated and cannot be output in the PDF.

Other Terms

HRASR_DT, Design Time Tool, String, Generate PDF, truncated text

Reason and Prerequisites

The issue was occurring due to programming error in class CL_HRASR00_DT_FORMS_API method GENERATE_FORM_INTERFACE. The datatype was incorrectly checked with keyword STRING instead of STRG due to which the field length was defaulted to length of 255 characters.

Solution

Apply the relevant Support Package or implement the corrections in accordance with the correction instructions.
