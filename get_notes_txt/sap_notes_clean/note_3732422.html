Symptom

Document upload in SSAM app fails with errors below: 

  • "HTTP status from the back end is 412."
  • "State of the resource (entity) was already changed (If-Match)." 

Environment

  • SAP Service and Asset Manager

Resolution

  1. In Configpanel (/SYCLO/CONFIGPANEL), navigate to OData Model Configuration -> Document Entity -> Property List.
  2. Remove the eTag from the FileSize property.
  3. Reset the client.
