Symptom

When you generate the S-1200 Remuneração de trabalhador vinculado ao Regime Geral de Previd. Social (Employee remuneration from Social Security General Regime) event for a terminated employee who have more than one personnel number (PERNR) and payment of PLR, the information of the payroll runs are reported wrongly.

Other Terms

eSocial; PLR; complementary payments; AJUS;

Reason and Prerequisites

You have implemented SAP Note 2674899.

Solution

Event S-1200 generation updated to report the dmDev nodes according to the personnel number.

The changes described in this SAP Note will be included in an HR Support Package, as indicated in section Support Packages.

The Support Package includes the following objects:

  • Changes in method PROCESS of class CL_HRPAYBR_EFDP_AFTER_TERM_PLR

As a general rule, SAP recommends that you implement the Support Package specified for your release. However, if you need to install a solution earlier, use the SAP Note Assistant to implement the correction instruction. You can find more information about the SAP Note Assistant in SAP Support Portal, under SAP Note Assistant.
