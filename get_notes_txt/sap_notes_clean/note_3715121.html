Symptom

Change of a commitment document results in unexpected reassignment records (0600 - 0650 pair) which changes the statistical indicator from 'Y' to '' thus creating a new budget consumption. You may experience unexpected availability control error.

Other Terms

 FMAVC

Reason and Prerequisites

Program error.

This issue only occurs when all of the following applies:

 - Commitment updating is set to PBET

- You reversed carry forward of a commitment document

- You set up the carry forward in a way that the sender records of the carry forward (amount type 0300) are updated with the statisticaly indicator 'Y'.

Solution

Program correction
