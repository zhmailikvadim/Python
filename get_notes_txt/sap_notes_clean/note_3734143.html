Symptom

  • While importing a transport request using the 'Import Collection' app, the error 'The worklist is updating. Please try again soon' is displayed, preventing the import from proceeding.
  • The error appears when starting the import in the 'Import Collections' app and, in some cases, occurs on the first attempt, after which the import proceeds upon refreshing. 

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Open 'Import Collection' App.
  2. Select the transport request 
  3. Click on 'Check and Import'.
  4. Observe the error: 'The worklist is updating. Please try again soon'.

Cause

Resolution

This behavior indicates that a background job is running to check for new transports available in the 'Import Collection' app. This is not an issue and ensures that imports are not triggered before they are ready. It is recommended to wait for the job to complete and then refresh the page after some time before proceeding with the transport request import.
