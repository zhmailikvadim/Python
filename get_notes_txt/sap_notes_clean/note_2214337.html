Symptom

Finance Localization for Egypt Private Sector.

Other Terms

Egypt, Private sector, Finance localization, Withholding Tax, Stamp Tax, Company Certificates, Remittance Statement for WHT & Stamp, Witholding/Stamp tax clearing(Egypt), FIMENA_CRMAINTAIN, FIMENA_COMPCERT, GLO_FIN_FI_EG, FIN_LOC_CI_47, FIAPEG_TXCLR, FIAPEG_TXREM

Reason and Prerequisites

New Legal requirement for Egypt Private Sector.

Solution

Apply appropriate support package.

Solution documentation for Egypt Localization can be found at the link given below-

http://help.sap.com/saphelp_globext607_10/helpdata/en/e2/ee21565e059f21e10000000a44538d/content.htm?frameset=/en/20/2a2f56c7cef81ae10000000a441470/frameset.htm&current_toc=/en/23/21e953f46eee2ee10000000a44176d/plain.htm&node_id=4

To activate Egypt Localization functionalities, you should activate Business Function FIN_LOC_CI_47 via transcation SFW5. Please go through the documenattion before you activate the Business Function.

If needed following BC sets with template customization can be activated via transaction SCPR20.

GLO_FI_EG_CNTRL

Egypt FI Localization Template- Controlling Area

GLO_FI_EG_COA

Egypt FI Localization Template- Chart of Accounts

GLO_FI_EG_COMP

Egypt FI Localization Template- Company Code

GLO_FI_EG_COMP_CERT

Egypt FI Localization Template- Comp. Certificates

GLO_FI_EG_GL

Egypt FI Localization Template- G/L Accounts

GLO_FI_EG_KV_ACC

Egypt FI Localization Template- Cust./Vend. Account Grps

GLO_FI_EG_TXP

Egypt FI Localization Template - Tax Procedure

GLO_FI_EG_WHT

Egypt FI Localization Template- Withholding Tax Codes

GLO_FI_EG_WT_CONFIG

Egypt FI Localization Template- Withholding Tax Codes

Note- Post Dated Cheques functionality is available with SAP note 2254658. Please implement this note to activate the function for Egypt.
