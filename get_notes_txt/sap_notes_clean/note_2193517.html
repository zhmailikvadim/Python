Symptom

In BPF template workspace, add content, input form or report, then deploy template and create instance.

Go to MY ACTIVITIES workspace, if user changes a member on context bar under Overview, the member on context bar under content,inputForm or report will not be inherited.

Other Terms

Standard model, My activities, Overview, Context bar

Reason and Prerequisites

Program error

Solution

For SAP Business Planning and Consolidation 10.1, version for SAP NetWeaver on 740, upgrade to(Component CPMBPC 810) support package 08 and upgrade BW740(Component SAP_BW) to support package 13.

For SAP Business Planning and Consolidation 10.1, version for SAP NetWeaver on 750, upgrade to BW750(Component SAP_BW) to support package 01.
