Symptom

You call either one of the methods CL_MD_BP_MAINTAIN->MAINTAIN or CVI_EI_ADAPTER->DO_START_INBOUND_MAIN or function module CVI_EI_INBOUND_MAIN in order to change a business partner that is assigned to a customer. With the change you would like to add a new company code to the customer and - at the same time - create a new customer sepa mandate for which the paying company code is the company code to be created. The process is aborted with error message FIN_SEPA 006 "0001 is not a paying company code for payer ... or linked payee ...".

Other Terms

Business Partner, Customer, Vendor, FIN_SEPA 006, FIN_SEPA, 006, FIN_SEPA006, sepa mandate, paying, company code, CIF, CVI_EI_ADAPTER, CL_MD_BP_MAINTAIN

Reason and Prerequisites

Side-effect of note 3706180, which was supposed to solve the issue, but did not.

Solution

Implement the attached correction instruction or the corresponding support package for your release level.
