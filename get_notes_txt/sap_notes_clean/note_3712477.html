Symptom

Is it possible to use a cleansing rule to delete items over a certain number if the catalog file has more items than desired?

Cause

The supplier does not want to remove items from their catalog to create a smaller catalog. 

Resolution

At this time there is not a way to use a cleansing rule to achieve the desired results. The team would need to submit an Improvement Request via the Customer Influence program.
