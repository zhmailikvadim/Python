Symptom

In the process "Sending of Invoice" (DE_B_CP_INVC_SEND), when an error occurs when triggering the invoice for one billing document, the system does not trigger invoice sending of invoices for the remaining billing documents.

This issue is relevant for the market roles: DSO, MOS.

Other Terms

US4G, Billing Enhancement, INVOIC, FICA Event, 2649, /US4G/CL_PROC_CI_FICA_ENH_MOSB, /US4G/IF_PROCESS_CI_FICA_ENH~PROCESS_CI_FICA_EVENT_2649, /US4G/RP_RESEND_INVOIC, MOS, MOSB

Reason and Prerequisites

Program error.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, when an error occurs while triggering the sending of an invoice for one billing document, the system continues to trigger the sending of invoices for the remaining billing documents.
You need to use report /US4G/RP_RESEND_INVOIC to resend failed invoices.
You can verify the fix by testing the invoice sending process with multiple billing documents and confirming that all invoices are processed even when individual errors occur.

Currently, resending by invoicing documents for the billing solution MO is supported.
