Symptom

This article summarizes the scenarios of configuring the visibility field in latest goal.

Environment

SAP SuccessFactors Goal Management

Resolution

When using the latest Goal Management, configure as follows to suit your usage scenarios.

 
use-text-for-privacynew-obj-share-status-publicInclude permissions for share and private-access in the goal plan?Result
truetrueYesConfigure the Visibility field to be available on the Goal Plan page and be editable, and set the goals to be public by default
truetrueNoConfigure the Visibility field to be available on the Goal Plan page but as a read-only field, and set the goals to be public by default
truefalseYesConfigure the Visibility field to be available on the Goal Plan page and be editable, and set the goals to be private by default
truefalseNoConfigure the Visibility field to be available on the Goal Plan page but as a read-only field, and set the goals to be private by default
falsefalseN/AHide the Visibility field and set the goals to be private by default
falsetrueN/AHide the Visibility field and set the goals to be public by default

Note:

In reporting.

  • Visibility Code for Public is 0; 
  • Visibility Code for Private is 1.
