Symptom

Favorites are not being updated with each Knowledge Base sync for models marked as favorites. 

Environment

SAP CPQ

Reproducing the Issue

  1. Go to quotes.
  2. Open quote.
  3. Add product from favorites.
  4. Latest KBs not updating favorites with each sync.

Cause

Expected behavior

Resolution

  • Favorites keep the exact configuration done in the product considering its version. 
  • Upgrade favorites to the latest product version is only available on Quote 2.0
