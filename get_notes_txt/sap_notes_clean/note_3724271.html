Symptom

When using Calculation Bucket as the source document for ADM processing, the target records are not filtered based on the defined target mapping. As a result, the system incorrectly retrieves all line items from the impacted bucket, including deleted lines.

This SAP Note corrects the behavior to ensure that target records are properly filtered according to the target mapping.

Other Terms

/IRM/CL_IP_AUTO_DATA_MAINTAIN, Calculation Bucket, Source document, ADM, Target, Mapping

Reason and Prerequisites

This is caused by program error.

Solution

Please implement the correction instructions. These corrections are only valid if you have applied the following Support Packs.

Release V/4 –  2022 V2, 2023, 2023 V2, 2023 V3, and Support Pack 1

Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object
