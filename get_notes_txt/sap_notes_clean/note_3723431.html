Symptom

Backorder processing (BOP) result is incorrect when the requirement segment value (SGT_RCAT) is initial.

Other Terms

"Segmentation Strategy", "Requirement Segment", "Segmentation Value"

Reason and Prerequisites

Prerequisties:

  • The requirement segment value (SGT_RCAT) is initial.
  • A valid stock segment (SGT_CATV) is available for a requirement segment (SGT_RCAT) value of initial.

Reason:

  • The stock segment (SGT_CATV) should be determined even when the requirement segment value (SGT_RCAT) is initial. 

Solution

Implement the attached correction instructions.
