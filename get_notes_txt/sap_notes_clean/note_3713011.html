Symptom

When running program SAPFGLBCF_V2 from Advanced Financial Closing the run fails with this error messasge:
Internal session terminated with a runtime error DYNP_TOO_MANY_RADIOBUTTONS_ON

Other Terms

 AFC SAPFGLBCF_V2 Tripple assigned parameter type TESTR

Solution

Apply SP or follow those steps:
1. In your ABAP system open transaction SPRO
2. Click on SAP Reference IMG
3. Open Cross-Application Components -> Advanced Financial Closing - Local Settings
4. Start "Register Programs for Parameter Mapping"
5. Search for the entry of program SAPFGLBCF_V2 variant S4OP000001
6. Select the row
7. Navigate into "Parameter Maintenance"
8. Clear out the value "TESTR" in column Par. Type for the rows P_TEST_A (Aggregated) and P_TEST_D (Detailed)
9. Save and exit.
10. In your AFC, open the App Monitor Communication System
11. Open the system which represents your ABAP system
12. Synchronize the system two times.
