Symptom

In SSAM, when a user adds a part using Online Material Search, the Edit Part screen does not load for the newly added Part. This Note resolves the issue with validating the Material details.

Other Terms

  • SAP Service and Asset Manager
  • Operations
  • Part
  • Material

Reason and Prerequisites

  •  BAS or Visual Studio Code with SAM MDK project
  • `patch` command (available on BAS and macOS by default) via terminal or other patching utility
  • Read the guidelines on how to apply the patch and read the appendix section if you receive any error

Solution

To apply the patch, download the attached 3706106.patch file and save it in a known location, preferably adjacent to your SAM.mdkproject.

If you are on Windows, see SAP Note 3533879 for how to install a .patch file in Windows. If you are on a Mac, follow the steps below:

  1. Open a terminal in the root of your SAM metadata (i.e. SAM.mdkproject/metadata or SAM.mdkproject/metadata/base.definitions)
  2. Run the terminal command `patch --no-backup-if-mismatch -p4 < path/to/3706106.patch`. 
  3. Verify in the terminal window that there are no errors.
