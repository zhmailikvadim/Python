Symptom

After staging the specific materials in CO02, PMR is not created due to the following error message "WM staging of materials not completed for entire quantity".

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Open CO02 and stage the specific materials.
  2. Observe the error message "WM staging of materials not completed for entire quantity".
  3. Verify that PMR is not created.

Cause

Control Cycle for the specific materials has not been created in APP Create Control Cycle for WM.

Resolution

Maintain Control Cycle for the specific materials in APP Create Control Cycle for WM with the best practice of Scope Item 6NI.

See Also

Scope Item 6NI

Keywords

WM staging, materials, entire quantity, CO02, PMR, SAP S/4HANA Cloud Public Edition, Control Cycle, Create Control Cycle for WM, Scope Item, 6NI
