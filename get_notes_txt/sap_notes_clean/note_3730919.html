Symptom

In SAP HANA Studio modeler, you notice that some columns display a different name from the source column.
For example, the name in the source node is Join_3.OPERATION_ID, while it displays as _BIC_OPERATION_ID. You want to know how to control this display. 

 "Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Environment

  • SAP HANA, platform edition 1.0
  • SAP HANA, platform edition 2.0

Reproducing the Issue

  1. Open a calculation view in SAP HANA Studio modeler.
  2. Check the column name in the Output.

Cause

Resolution

Follow these steps:

  1. Select the column.
  2. Input the desired name in the "Name" textbox under "Properties".
