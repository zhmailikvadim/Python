Symptom

Experiencing failures when attempting to run the Bulk Business Partner Import/Export process. The system consistently returns an error indicating Invalid Excel file for importing.

Environment

SAP CPQ

Reproducing the Issue

  1. Navigate to setup
  2. Import/export
  3. Bulk business partners import/export
  4. Export all business partners.

Cause

The issue related to missing column names in the Excel file, which led to the error during the import process.

Resolution

To ensure proper functionality, the columns in the file must include a column name.
