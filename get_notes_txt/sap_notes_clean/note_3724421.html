Symptom

The Career Site Logo is updating or changing whenever you zoom past 150% above.

Environment

SuccessFactors Recruiting Marketing

Reproducing the Issue

  1. Access the Career Site
  2. Notice the logo (upper left corner of page)
  3. Zoom the browser up to 150% or above
  4. Notice how the logo is changing

Cause

Header desktop and mobile layout has different configurations. 

Resolution

This is due to the configuration applied in the Mobile Layout. The zoom level is enforcing to apply the header for Mobile Layout including the logo.

To change this, you can upload the same logo from the Desktop Layout.

  1. Access CSB.
  2. Under Appearance, select Styles.
  3. On the left panel, you should see the Header configuration for each locale.
  4. Select the affected brand.
  5. Choose the Mobile Layout Tab. 
  6. Click on the Logo to change.
  7. Save.
