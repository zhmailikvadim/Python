Symptom

In the process "Processing of Disconnection Order (SUP View)" (DE_B_GPKE_DISC_DC_ORD_SUP), the clarification case remains open after the disconnection is resumed. The last step "End Process: Processing of Disconnection Order" (8999) is not reached, which prevents the system from closing the clarification cases automatically.

This issue is relevant for the market role: SUP.

Other Terms

GPKE_SUP, DE_B_GPKE_DISC_DC_ORD_SUP, Processing of Disconnection Order (SUP View), /US4G/RP_MDR_PROCESS_TRIG, clarification case, disconnection order, step 8999

Reason and Prerequisites

The error occurred because the process did not reach the last step 8999, which is responsible for automatically closing open clarification cases.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the system processes step 8999 and automatically closes open clarification cases. You can verify the fix by triggering the process using the report /US4G/RP_MDR_PROCESS_TRIG and checking that step 8999 runs successfully and that clarification cases are closed.
