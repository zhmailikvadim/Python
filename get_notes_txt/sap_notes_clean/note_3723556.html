Symptom

In the Agreement / Agreement Request application, when the Sales Office (VKBUR) value is changed, the corresponding Sales Group (VKGRP) dropdown values are refreshed. However, the field still retains the previously selected values instead of updating with the correct value based on the new Sales Office selection.

This SAP Note provides the necessary corrections to ensure that the Sales Group field refreshes properly and loads the latest valid values.

Other Terms

VKBUR, VKGRP, Dropdown, Agreement, /IRM/IPAGRM

Reason and Prerequisites

This is caused by program error.

SAP Note #3723520 is available for ECC Releases.

Solution

Please implement the correction instructions attached. These corrections are only valid if you have applied the Support Packs.

Release V4 - 2022 V2 Base Package, 2023 V2 Base Package, 2023 V3 Base Package or Support Pack 1.

Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object.
