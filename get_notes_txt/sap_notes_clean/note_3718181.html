Symptom

  • Custom navigation links created are not appearing in the Top Navigation menu (module picker) after activating the "Latest Shell Bar" in a preview environment.
  • The custom navigation links appears in other location but the Top Navigation

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

  • SAP SuccessFactors HCM Suite

Reproducing the Issue

  1. Activate "Latest Shell Bar" in Company System and Logo Settings.
  2. Create any custom navigation link and make sure the location is selected as "Top Navigation"
  3. Observe that custom navigation links created are not showing in the module picker

Cause

  • Identified as defect by SF engineering team with activation of "Latest Shell Bar" in 2H 2025

Resolution

  • Engineering Team have fixed the issue under reference WEF-167450(internal).
  • The fix has been delivered and is available with the 2605 release.
  • Customers are advised to verify the behavior after 2605 release
  • For release timelines, refer to: 1H 2026 SAP SuccessFactors Release Upgrade Windows
