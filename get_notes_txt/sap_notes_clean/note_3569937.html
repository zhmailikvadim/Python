Symptom

  •  When publishing a Publication with Web Intelligence document with prompt interval values, prompt interval values are not displayed within history page in BI LaunchPad.

Other Terms

Webi, Parameter, LP, scheduling

Reason and Prerequisites

Environment:

  • SAP BusinessObjects Business Intelligence Platform 4.3 Support Package 04
  • SAP BusinessObjects Business Intelligence platform 2025
  • Web Intelligence
  • Launch Pad

Steps To Reproduce:

  1. Create a publication with Web Intelligence document with interval prompt.
  2. Go to "Report Features" tab and "Prompts" section.
  3. Tick the option "Use prompt values from source document".
  4. Save the Publication.
  5. Schedule the Publication.
  6. Once the scheduling is successful click on the Publication instance within Publication history.
    1. >> Note that publication parameters column is incorrect: "=." or  ";" is displayed instead of interval bounds

Cause:

  • This issue is caused by a defect in the product. 

Solution

  • The solution is currently under development investigation.
