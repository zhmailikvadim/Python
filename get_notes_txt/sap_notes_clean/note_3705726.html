Symptom

This note contains the list of notes to be installed for using Field Logistcs persona in SSAM 2511 release for S4CORE 108(2023) and the configuation file.
Field Logistcs persona is currently supported only above 2023 FPS01.

Other Terms

SSAM, Field Logistics Persona, 2511

Reason and Prerequisites

Missing Field Logistics Objects in 2511 S4CORE 108 TCI (2023)

Implement pre-requisite notes before installing the current notes:
TCI: MAIF 2505 Integration Application support for S/4HANA 2023 for software component S4CORE
TCI: MAIF 2511 Integration Application support for S/4HANA 2023 for software component S4CORE

Solution

Please implement the below notes in sequence.

1. 3635412 : SSAM Field Logistics objects in New Package
2. 3642217 : SAP Service and Asset Manager - Field Logistics - Exchange table for Shipment Items (Ready to pack)
3. 3659747 : FL Implementation of Packed Packages exchange
4. 3651234 : SSAM Field Logistics Persona in 2511
5. 3704353 : SSAM Field Logistics Persona in 2511 (2)

Once the above notes are implemented, please proceed with below steps to upload the configuration file.

Please download the attached configuration file:
FL_2511_Config.bin

Then, follow the manual activities below to import the new configuration.

1. Execute the program /MFND/CORE_CONFIG_IMPORT_PROG in transaction SE38

2. Provide the file path in the 'Filename' parameter of the Selection Screen (f4 help for file browser)

3. You may optionally provide a 'Customizing Request' to capture the changes

4. Check the 'Test Run' checkbox and execute the program to ensure no errors occur

5. Successful output should contain the rows modified for each table. All changes are rolled back when 'Test Run' is checked

6. Return to the Selection Screen

7. Uncheck 'Test Run' and then execute

8. Successful output should contain the rows modified for each table and the transport in which the changes were saved to, if provided
