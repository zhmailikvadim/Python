Symptom

  • Error message: "An error occurred. Please create an incident on component 'FI-FIO-AR'."
  • Unable to reverse the bank statement in the "Manage Bank Statements" application.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Open the "Manage Bank Statements" application.
  2. Attempt to reverse the bank statement for the specified company code and house bank accounts.
  3. Observe the error message: "An error occurred. Please create an incident on component 'FI-FIO-AR'."

Cause

  • The ABAP runtime error during the reversal process is caused by missing corresponding journal entries in the system.

Resolution

  1. Confirm if there has been any recent system reset or activity that may have removed the journal entries.
  2. If testing the reversal of a bank statement, use statements posted after the reset.
  3. Alternatively, re-upload the bank statement file and then proceed with the reversal.
