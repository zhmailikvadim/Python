Symptom

Reservation created from “Business Object Interfaces :  Reservation Document" cannot be posted in the app “Stock Transfer for Reservation” - due to error message “No goods movements can be selected”.

Environment

  • SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Go to app 'Stock Transfer for Reservation'.
  2. Enter Reservation Number.
  3. Execute.
  4. Error:  “GOODSMOVEMENT008  No goods movements can be selected”

Cause

Refer to the SAP help documentation for the app Stock Transfer for Reservation

With this app, you can select the stock transfer reservations you created in the pull list via a pick list and process the reservations further. 

Resolution

Process the goods movement for the reservation via the app Manage Reservation Item, enter the Reservation and select the Create Goods Movement.

Keywords

Business Object Interfaces, Reservation, Stock Transfer for Reservation, No goods movements can be selected, GOODSMOVEMENT008, Manage Reservation Item, Create Goods Movement.
