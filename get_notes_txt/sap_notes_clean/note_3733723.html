Symptom

  • Error message: “Error in GUI input data: invalid list dimension” (Message No. 00123) in transaction /OPT/VIM_WP when launched via webgui or Fiori tile.
  • Error occurs in VIM when interacting with an ABAP list (for example, selecting a Delivery Note).
  • Issue may occur intermittently, often after the UI area has been minimized by a docking container and/or the browser window has been resized during the session.

Environment

  • SAP S/4HANA
  • ABAP Platform
  • SAP GUI for HTML

Reproducing the Issue

  1. Open t-code /OPT/VIM_WP via WebGUI or Fiori tile.
  2. Navigate to a document and open a search help (for example, Delivery Note), then execute to display the ABAP list.
  3. Select an entry in the list and choose Copy/Confirm to return to the calling screen.
  4. Error “Error in GUI input data: invalid list dimension” may occur. 

Cause

Programming error.

Resolution

Upgrade the Kernel in the affected system as per the following SAP Note:

3698313 - WebGUI connection terminates with error "invalid list dimension"
