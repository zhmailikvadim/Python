Symptom

The error "El XML no contiene el tag ono existe informacion de ResponseCode" is displayed for the eInvoices from Peru.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Go to the eDocument Cockpit App.
  2. In the Source Document field, enter ABC. (where ABC represents the Source Document field)
  3. Select Go to... -> Preview/Display XML.

The following error messages are displayed:

- Message: 2128
- Message Details: 2128 El XML no contiene el tag ono existe informacion de ResponseCode

Cause

The error "El XML no contiene el tag ono existe informacion de ResponseCode" is displayed when the tag <ResponseCode> is empty in the DiscrepancyResponse section of the XML of the eDocuments from Peru. The tag <ResponseCode> is filled based on the information maintained in the Value Mapping DISCREPANCY_RESP_FI.

Resolution

Since the <ResponseCode> tag is displayed empty in the eDocument XML, it is necessary to review the information in the Value Mapping DISCREPANCY_RESP_FI. This review will help confirm whether the correct and valid data has been maintained in the Value Mapping DISCREPANCY_RESP_FI to ensure the <ResponseCode> tag is properly filled in the XML.
