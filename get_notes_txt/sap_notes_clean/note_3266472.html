Symptom

You run the Function Module /DMF/OPIF_VENDOR_FUND_INBOUND to update the vendor fund that was previously created. You  process the content of the staging tables. For the funded products, the quantity field on the term level will be wiped out.

Other Terms

Vendor Fund, Quantity, Total Amount, Terms

Reason and Prerequisites

The field on the UI is bound to upper_limit. The update was changing the lower_limit instead of upper_limit.

Program error.

Solution

Implement the correction instructions.
