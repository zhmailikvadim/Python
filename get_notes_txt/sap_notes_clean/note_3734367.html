Symptom

 This helps with identifying the potential issues in customized SOI BAdI Implementation, which needs to be analyzed by SAP support. A prerequisite is to activate the user parameter FINOC_RULE_HIST_ACTV.

Other Terms

Manage Organizational Changes, org change, BAdI, Rule

Reason and Prerequisites

The history table will record the result of customized BAdI and help with identifying the invalid results.
