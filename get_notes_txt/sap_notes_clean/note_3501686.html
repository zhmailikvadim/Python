Symptom

"Host" System Type missing from report RLMDB_REPORTING_SYSTEMS.

When executing the report the following options are seen:

  • SAP HANA Database
  • Application Server ABAP
  • Application Server Java

Environment

SAP Focused Run 4.0

Reproducing the Issue

  1. Execute SE38 transaction
  2. Execute RLMDB_REPORTING_SYSTEMS report
  3. There is no Host option.

Cause

The report RLMDB_REPORTING_SYSTEMS is obsolete in FRUN.

Resolution

Now all the System Types can be checked using LMDB Tools UI.

To check all currently existing hosts:

  1. Open LMDB Tools
  2. Select "LMDB Reporting for Hosts"
  3. Enter the filters to restrict the reported hosts
  4. Click the "Go" button

Furthermore, you can perform searches, customize columns, sort, and export the table content to Microsoft Excel.

See Also
