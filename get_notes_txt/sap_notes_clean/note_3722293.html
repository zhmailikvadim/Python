Symptom

  • Tasks fail intermittently when writing data to IBP.
  • Error message: Error creating RFC function </IBP/INTCPI_WRITE_DATA_001>: <RFC_COMMUNICATION_FAILURE- SSSLRC_CONN_CLOSED : Remote Peer has closed the network connection. unable to send data to myXXXXX-api.scmibp.cn:443>. Notify Customer Support.

Environment

  • SAP Cloud Integration for data services

Cause

The connection established with IBP is unstable due to suboptimal configuration parameters in the IBP datastore.

Resolution

Update the IBP datastore configuration to the following parameters:

  • Number of connection retries: 20
  • Interval between retries (ms): 2000
