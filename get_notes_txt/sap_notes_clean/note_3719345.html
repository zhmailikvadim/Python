Symptom

In MCM, the reprocessing of step 20 (Update local measurement instance status to Shutdown) in finalization event always remained stuck if it got in error state for once.

Other Terms

APEU

Reason and Prerequisites

Program error, process step ID - MCM_FINAL_DB_UPD

Solution

This NOTE allows to set the instance data again during the re execution of step 20 in the shutdown finalization process.
