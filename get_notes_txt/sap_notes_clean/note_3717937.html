Symptom

• Unable to update recommendation status to "Implemented" from "In Process".  
• Error message displayed: "The recommendation status could not be updated to Implemented. Currently, the Rule service is not supported for EIoT subscriptions."  


"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental"  

Environment

  • SAP Asset Performance Management

Reproducing the Issue

1. Open Manage Recommendations (GraphQL version).
 
2. Select a Recommendation with the status set to "In Process".  
3. Attempt to change the status to "Implemented".  
4. Observe the error: "The recommendation status could not be updated to Implemented. Currently, the Rule service is not supported for EIoT subscriptions."  

Cause

  • EIOT rules are not supported in the Recommendation application (GraphQL version).

Resolution

Switch to the Recommendation New application (CAP application version) where this functionality is supported.
