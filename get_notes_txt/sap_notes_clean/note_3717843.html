Symptom

The 'Sort Field' field is missing in the Change Maintenance Order app (WDA).

Environment

  • SAP ERP Plant Maintenance (PM)
  • SAP S/4HANA, on-premise
  • SAP S/4HANA, Cloud Private Edition

Reproducing the Issue

  1. Access the Change Maintenance Order (WDA) app.
  2. Enter Reference order number
  3. Navigate to the Location data tab.
  4. Observe that the "Sort Field" field is missing.

Cause

Customizing not in place

Resolution

It is possible to add Sort Field using form (UIBB  EAMS_WDC_ORD_LOCATION_CFG_V2)

  1. Open the Application in customizing mode 
  2. Navigate to the Location Tab 
  3. Click "Show customizable areas"
  4. Click on Wrench icon 
  5. To access form UIBB  EAMS_WDC_ORD_LOCATION_CFG_V2
  6. Drag and drop Sort Field to the desired position on the screen
  7. Save setting
