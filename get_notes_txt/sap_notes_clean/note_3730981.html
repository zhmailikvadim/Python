Symptom

Component: Product Safety
Module: Report definition

You want to create a report or generate a report for template. Within the report template you have specified symbols for instance control, e.g. <01EHS_ICBCB>. The generation of the report fails and you get the error message "Symbol value determination returns empty symbol type or empty symbol ID" (CM_REPORT_GEN106).

Other Terms

CM_REPORT_GEN
106
MSDS
Instance control
Symbol type
Symbol id

Reason and Prerequisites

Reason
Program error. The values are not filled by the corresponding subroutines.

Prerequisites
For information about the validity of the corrections, see the correction instructions.

Solution

The section "Support Packages" specifies the support packages that contain the corrections.

Alternatively, you can implement the attached correction instructions.
