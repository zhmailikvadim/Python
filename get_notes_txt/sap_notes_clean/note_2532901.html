Symptom

When you use transaction /SAPSLL/MARC_UL_R3 ('Copy R/3 MARC Entries to GTS'') to transfer the product classification from a feeder system, the validity start date of the new classification entries is transferred from the validity start date of the tariff number.

Other Terms

SAP R/3, MARC, upload, /SAPSLL/CTSNUM, /SAPSLL/CTSNUMC, /SAPSLL/PRCTS, /SAPSLL/PRCTSC, DATAB, /SAPSLL/MARC_UPLOAD_R3

Reason and Prerequisites

Program error

Solution

Implement the attached source code corrections in accordance with the release level so that, in the future, the valid from date is transferred from the current date.
