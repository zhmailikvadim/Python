Symptom

While you try to create a Visit, you receive an error:
RBAM: No authorization for 'BuiltIn <default> WRITE' on instance 'DEF' of 'BONodeProxy <default> BUSINESS_ACTIVITY <default> ROOT' in context 'UIApplication <default>".

Environment

SAP Cloud for Customer

Reproducing the Issue

  1. Go to Visits view
  2. Click to create a new Visit
  3. Fill the information and click Save
  4. Error occurs:
    RBAM: No authorization for 'BuiltIn <default> WRITE' on instance 'DEF' of 'BONodeProxy <default> BUSINESS_ACTIVITY <default> ROOT' in context 'UIApplication <default>" 

Resolution

Please check the following:

1 - If the user is permitted to visits with its sales data
2 - Check if when creating a visit, no sales data is generated.

Please try to set one of its sales data on its employee data as main.
