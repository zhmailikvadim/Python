Symptom

When adding or editing a task under Learning Administration > Content > Tasks, there is no option to add translations to the task description.

Environment

SAP SuccessFactors Learning

Cause

Resolution

This is a current system limitation. Two enhancement requests have been created to address this limitation in future releases: Allow for Translation Description for Task Checklist Content Object and Task Checklist Description - Localization Option Needed.

Keywords

task checklist, translations, locale, learning administration, task description, checklist descriptions, unavailable, localization, globe, description, add
