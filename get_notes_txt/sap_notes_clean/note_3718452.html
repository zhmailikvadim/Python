Symptom

The main include of the Belgium payroll program contains unnecessary source code lines and comments.

Other Terms

RPCALCB0, include, clean-up

Reason and Prerequisites

You want to improve the clarity of the payroll program.

This SAP Note is only relevant for Payroll Belgium.

Solution

The nested and useless includes are deleted. These changes do not modify any functionality of program RPCALCB0. It is a clean-up action. It has also neither an impact on the execution time nor on the memory consumption.

However, if you are using a customer-specific program instead of RPCALCB0, you might need to adapt it because some includes have been deleted or moved. If you have syntax errors in your specific program after implementing this SAP Note, refer to SAP Note 3338143 to try to adapt it. If you don't manage to get rid of all the syntax errors, you might create a support incident in component PY-XX.

These corrections will be available with the next HR Support Packages referenced below.

If you have an urgent need of these corrections, apply the following correction instructions.
