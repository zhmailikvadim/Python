Symptom

Titles in the audit page do not display a remove date.

Environment

  • SAP Sales Performance Management
  • Incentive Management

Reproducing the Issue

  1. Navigate to the audit page for titles.
  2. Check the remove date for the rate table assignments.

Cause

In this case, seeing records without a remove date is expected, since there are active versions of the item. As long as the versions are not overlapping in the database and all related records are consistent, this behavior is normal and does not indicate an issue.

Resolution

Validate that the total number of audit records corresponds to the total current active records.

Keywords

rate table assignment, audit page, remove date missing, active versions, incentive management, rate table discrepancy,  audit records consistency, overlapping database records.
