Symptom

  • Getting below error message while refreshing Web Intelligence report based on CMS Universe delivered with SAP BusinessObject.
    • The following database error occurred: (CS) "Unexpected behavior: GC overhead limit exceeded (max heap: 2048 MB) ". For information about this error, please refer to SAP Knowledge Base Article 2054721 on the SAP Support Portal. (IES 10901)

Other Terms

idt, webi, cms universe, gc overhead limit, ies 10901, max heap, cs.cfg

Reason and Prerequisites

Environment:

  • SAP BusinessObjects Business Intelligence 4.3 Support Pack 05 Patch 03
  • Web Intelligence (Webi)
  • Information Design Tool (IDT)
  • Oracle 19c

Reproducing the Issue:

  1. Log in to BI Launchpad.
  2. Create a new Web Intelligence document using BI on BI CMS Universe given with product [BI platform CMS system database.unx].
  3. Add following objects in the Results Objects panel: ID, Name, Kind, PropertyValue, PropertyValueLongText, PropertyName, PropertyLevel, PropertyIsContainer, PropertyPath.
  4. Add following objects in query filter panel: Kind, Scheduling Instance, Ancestor.
  5. Try to refresh the document.
  6. Notice refresh fails with below error message:
    >> The following database error occurred: (CS) "Unexpected behavior: GC overhead limit exceeded (max heap: 2048 MB) ". For information about this error, please refer to SAP Knowledge Base Article 2054721 on the SAP Support Portal. (IES 10901)
  7. Now remove the following objects from result panel: PropertyValue, PropertyValueLongText.
  8. Try to refresh the document, notice now the report is getting refreshed.
