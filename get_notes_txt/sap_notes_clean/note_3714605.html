Symptom

When filtering by artifact in the Message Monitor, you may notice that some Integration Flows (IFlows) are listed by their descriptive name, while others are displayed only by their technical ID names.

Environment

  • SAP Integration Suite
  • SAP Cloud Integration

Cause

This issue can occur in specific scenarios, such as a migration from SAP PI/PO or the Neo environment to Integration Suite. It is caused when a deployment notification for the artifact is not successfully processed by the system, which prevents the descriptive name from being displayed.

Resolution

To resolve this issue, redeploy the affected IFlows. Specifically, you only need to redeploy the artifacts that are currently displayed by their technical ID.

Once redeployed, the artifact's descriptive name will appear correctly in the Message Monitor.

Keywords
