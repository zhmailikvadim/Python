Symptom

You're using SAP CRM. You've set up a BOR-based launch transaction to be opened via SAP GUI. You have downloaded the LaunchBORinSAPGUI.sap shortcut by clicking on the launch transaction in the web UI. Upon opening it, the transaction does not start, or is missing the intended context.

Reason and Prerequisites

SAP Note 3697487 introduced logic for BOR-based launch transactions opened in SAP GUI to be able to deal with the case of the container holding the information about the transaction to be called being populated without an index. The check for whether the index is present was done at a processing point where the respective workarea was not yet filled, and hence did not work correctly.

This Note changes the logic, so that the check happens appropriately. Consequently, the container is correctly received and processing continues as expected.

Solution

Implement the assigned correction instructions.
