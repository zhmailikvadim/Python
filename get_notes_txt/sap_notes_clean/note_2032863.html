Symptom

If you set the parameter SAVE_OTF_SPOOL using the report RSTXPDF3, the PDF conversion process terminates with an error.

The error tells you that the formatting used does not support a specific page format. (TD465)

Other Terms

SCREEN, INCH12, TD 465, RABAX_CALLING_RABAX, MESSAGE_TYPE_X

Reason and Prerequisites

This problem occurs if the PDF converter tries to process OTF files that are not actually valid. The OTF data contains a page format and a device type that are not compatible (the page format does not exist in the device type).

The problem occurs, for example, if the OTF data was generated from a SAPscript form and the parameter DEVICE = OTF_MEM was set for the call of the function module OPEN_FORM. In this case, the device type SCREEN is entered into the OTF data, and this does not contain any page formats.

Solution

Implement the attached correction instructions or import the relevant Support Package.

Activate the option SAVE_OTF_SPOOL only for troubleshooting. Do not set it permanently. See also SAP Note 924752.

Reason and Prerequisites

Installation of the Note is only required if it serves as a prerequisite. Correction Instructions do not affect any existing functionality.

Solution

Please implement the attached Correction Instructions or the mentioned Support Package.
