Symptom

You use SAP ACM and the universal prescription in document category version 2 or 3 to issue electronic orders.

If you do not select all ePrescription-enabled orders of a universal prescription in the medication order overview from the clinical work station, all ePrescription-enabled orders belonging to the universal prescription are then available for the signature in the signature overview and the system does not display any information about this.

Other Terms

AMVO overview, N3KVUNIRP

Reason and Prerequisites

Improve user guidance

Solution

If not all ePrescription-enabled orders of a universal prescription are selected in the medication order overview from the clinical work station, the system informs the user in the signature overview that ePrescription-enabled orders of a universal prescription can only be fully signed and set and therefore all corresponding ePrescription-enabled orders exist for the signature.

The function is available with the specified Support Packages. You can implement collective correction 3728447 in advance.
