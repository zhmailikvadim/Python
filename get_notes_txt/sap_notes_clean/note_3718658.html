Symptom

This SAP Note delivers the package GLO_FIN_IS_CO_DIAN_1006 for the Colombia DIAN 1006 Report Format (CO_DIAN_1006) report in the Run Statutory Reports app for SAP Document and Reporting Compliance. The package is used to store development objects for the CO_DIAN_1006 report.

Other Terms

DRC, Colombia, DIAN, Magnetic Media, 1006 Report Format

Reason and Prerequisites

To implement the Magnetic Media compliance reports solution, you need a separate license for SAP Document and Reporting Compliance. For more information, contact your SAP key account manager.

You have installed the Magnetic Media compliance reports for Colombia. For more information, check SAP Note 2717885.

This SAP Note is relevant only for Colombia.

Solution

After implementing this SAP Note, the package GLO_FIN_IS_CO_DIAN_1006 is available in your system for the CO_DIAN_1006 report.

As a general rule, SAP recommends that you install a solution by applying a Support Package. However, if you need to install a solution earlier, use the Note Assistant to implement the correction instruction. You can find more information about the Note Assistant in SAP Service Marketplace, under service.sap.com/note-assistant.

To implement the solution, you must carry out manual activities and implement the correction instructions that are valid for your release.

Documentation for Magnetic Media compliance reports is available in the system and in the SAP Help Portal. System documentation (such as field help, customizing, messages) is delivered via Support Packages. SAP Help Portal documentation is available at help.sap.com/viewer/s4h_op_colombia.

We also recommend you to check our FAQ SAP Note (2717885), in which you find an extensive list of frequently asked questions and recommendations that relate to Magnetic Media compliance reports.
