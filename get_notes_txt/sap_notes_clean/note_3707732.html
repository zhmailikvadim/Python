Symptom

While running transaction OMIZ, the system displays the TK430: Client XXX has status 'not modifiable' error message. 

Environment

This KBA applies for all SAP Products based on NetWeaver and S4H

Reproducing the Issue

Start transaction OMIZ

Cause

Current Settings flag is not set.

Resolution

Please set the flag 'Current Settings' for the following objects with transaction SOBJ within your development system, and afterwards import the transport request into your production system.


VC_MDL
V_MDLV
V_MDLW
V_MDLG
V_MDLL


For more information, please refer to SAP note 2442887 - SOBJ | How to assign object attribute Current Settings to a maintenance object.
