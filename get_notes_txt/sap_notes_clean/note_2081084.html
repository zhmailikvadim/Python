Symptom

The performance of Archive POWLs can be improved by avoiding multiple access to the same file in case of having more than one archive infostructure active for the same archive object.

Other Terms

/BOFU/CL_ARCH_GENER1; RFA_QUERY_INT1;  mt_rfa_root_key_file_index

Reason and Prerequisites

The archiving query is determining all archiving files from archive, which fit to the selection criteria with which the query / POWL was executed. In case that many info structures exist for one archive file, it can happen that one file is determined multiple times what can lead to conflicts in the later processing of the query request.

Solution

Implement the correction from the correction instruction or implement the support package of the note.
