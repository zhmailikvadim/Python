Symptom

In the Fiori app F1511 (Request Maintenance), when Alternative Labeling is active and a user profile is set for the technical object, entering the user‑profile‑defined label of a Functional Location returns no result.
Other search inputs continue to work as expected.

Other Terms

PM-FIO-WOC-MN; F1511; C_TECHNICALOBJECT; C_FUNCTIONALLOCATION; CFUNCTLLOCATION; I_FUNCTIONALLOCATIONLABEL; I_FUNCLOCATIONLABELINTERNAL; _FUNCLOCATIONLABELSEARCH; TECHNICALOBJECTLABEL; Functional Location; Alternative labeling; User profile

Reason and Prerequisites

The reason is a program error. When Alternative Labeling is active and a user profile is assigned, the search scope in C_TECHNICALOBJECT did not include the user‑profile‑dependent label for Functional Locations, so it could not be found by that label.

Solution

Apply attached Correction Instruction. The correction enables C_TECHNICALOBJECT to source the Functional Location search label from the user‑profile‑dependent label association and propagates the required label association from C_FUNCTIONALLOCATION, making the user‑profile label searchable in F1511.
No manual activities are required.
