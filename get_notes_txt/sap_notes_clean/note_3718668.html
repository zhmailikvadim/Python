Symptom

List of systems is not available when the AO parameter PreferredPlatform is changed and you are using the Analysis Office version 2.8SP18

Environment

  • SAP Analysis for Microsoft Office
  • SAP Analysis for Microsoft Office, edition for SAP Analytics Cloud

Reproducing the Issue

  1. Open SAP Analysis for Office.
  2. Attempt to insert a new query.
  3. Observe that the list of systems is not displayed.

Cause

Program error.

Resolution

This issue is fixed in the Analysis Office version 2.8SP21 as per SAP note 3424915.

As a workaround, set the AO parameter PreferredPlatform to option 'Selectable' (Value 0).

See Also

  • SAP note 3424915- SAP HANA connections are not displayed in open dialog
  • SAP Help-PreferredPlatform
