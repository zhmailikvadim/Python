Symptom

When attempting to create or generate an interface format in the Interface Toolbox (PU12) for PCL2 clusters, the generation process fails with technical errors, most notably TFW_API029 ("Table info not found").

Other Terms

PU12, PCL2, RPCIFI25, LOAD_TABLES, TFW_API029, HRPAY structures, Swiss Payroll, TFW_GET_TABLETYPE_INFO, Interface Toolbox.

Reason and Prerequisites

A program error: SAP Note 3314951 is a prerequisite.
