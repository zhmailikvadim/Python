Symptom

When navigating from the Fiori App for Budget and Commitment/Actual line Item Analyzer to the FM Line Item Browser, the parameters corresponding to Universal Hierarchy are not recognized and in the end not applied in the report output.

You observe that, although this is intended for FMBROWSER, this is applicable for any Funds Management report using Universal Hierarchy.

Other Terms

FMBROWSER, RFFMRPF_LIB, P_UH_VIS, Universal Hierarchy, CL_GUI_OBJECT=>WWW_ACTIVE

Reason and Prerequisites

Reason: Program Error

Prerequisite: Funds Management report using Universal Hierarchy

Solution

After applying the correction instructions, the Universal Hierarchy panel will be defaulted in an "open" status (where the fields are all shown on screen) as opposed to hidden when a program using Universal Hierarchy is called via web or fiori.
