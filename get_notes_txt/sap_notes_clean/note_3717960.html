Symptom

Processing option 'BAM account posting by clearing item' is visible on statement agreement tab in Route and Clearing Agreement program ,but this option is not available.

Other Terms

Route and Clearing Agreement, Statement Agreement, Processing Option

Reason and Prerequisites

Statement agreement does not support "BAM account posting by clearing item" processing option.

Solution

Processing option 'BAM account posting by clearing item' is excluded from the list of possible values for statement agreement.
Please apply SAP Note or assigned Support Package
