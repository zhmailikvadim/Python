Symptom

You use the fiori app 'Manage Budget Request', Fiori ID F8332.

You have defined in fiori app, Manage Budget Template, different authorization groups in order that only certain users are allowed to create, copy or delete budget requests.

You have defined a role with authorization object 'F_FMBU_TEMP' with activity '03' (Display) and with the relevant authorization group for the users who are allowed to use the budget template in the budget request app.

You have defined a role with authorization object 'F_FMBU_REQ' for the activity '01'(Create), '03' (Display) and '06'(Delete) for those users.

You observe the following symptoms:

  • When a non-restricted user tries to delete a budget request, he receives the unexpected error FMBS222.
  • When a restricted user tries to copy a budget request, he does not receive any error.
