Symptom

The System > Monitor page is missing after Feature Delivery (version 2026.2.1) in SAC

Environment

SAP ANALYTICS CLOUD 2026.2.1

Reproducing the Issue

  1. Log in to SAP Analytics Cloud.
  2. In the side bar, attempt to locate the Monitor page under the System Option.

Notice : The Monitor page is no longer accessible.

Cause

The System > Monitor page was retired as part of the Feature Delivery (version 2026.2.1) in SAP Analytics Cloud.  

Resolution

1. The System Monitor page has been deprecated, and its features are now integrated into the System Overview page.  
2. For historical license usage, navigate to System Overview > Trends > Licenses Trend.  
3. To view tenant errors and warnings, access the Alerts page within the System Overview section.
