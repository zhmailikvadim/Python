Symptom

  • The navigation bar in the People Profile is not visible to users with basic permission roles.
  • Admin users can see the navigation bar without issues.
  • Users with basic permissions cannot view the navigation bar, even though they have access to certain People Profile sections.

Environment

  • SAP SuccessFactors HCM Suite
  • SAP SuccessFactors Employee Central

Reproducing the Issue

  1. Log in to the SAP SuccessFactors Employee Central system with a user account that has basic permission roles.
  2. Access the legacy People Profile of another user.
  3. Observe that the navigation bar under the header is not displayed.
  4. Log in with an admin account and access the same People Profile.
  5. Observe that the navigation bar is visible.

Cause

Lacking of permission.

Resolution

  1. Review the Role-Based Permissions (RBP) assigned to the affected users.
  2. Ensure that the users have the permission:
  • User Permissions > Employee Views > Profile
