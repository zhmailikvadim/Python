Symptom

  • The system shows the next step for a work order as Need to Activate and I cannot see the Activate button.
  • I can see the Activate button but I cannot press it.

Reproducing the Issue

  1. Log in to your SAP Fieldglass account.
  2. Navigate to the Work Order with the next step shown as Need to Activate.
  3. Notice that the Activate button is missing or grayed out and cannot be clicked.

Cause

  • You do not have permission to Activate this work order.
  • The Activate button cannot be pressed if there are required Activity Items that still need to be completed.
  • Even Program Management Office (PMO) will not have the Activate button available if there are pending Activity Items.

Resolution

  1. Log into your SAP Fieldglass account
  2. Open the Work Order and go to the Activity Items tab
  3. If there are any pending Activity Items, you must complete them before activating the Work Order
  4. Once all Activity Items are completed, the Activate button will become available
    • If there are no pending Activity Items and the Activate button is still not visible, you do not have permission to activate this Work Order. Contact your company’s Program Management Office (PMO) team to have the Work Order activated
