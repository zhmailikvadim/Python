Symptom

Message FAA_POST075 occurred in CJ88 settlement

Environment

  • Simple Finance
  • S/4 HANA Finance
  • S/4 HANA

Reproducing the Issue

  1. Create a project with investment measure Asset Under Constructure (AUC) using parallel currency areas
  2. Post down payment and clear down payment in different periods which have different exchange rate
  3. In the AUC, lead area 01 has no down payment but down payment remains in parallel currency area 
  4. When executing CJ88 with process type 'Full settlement', message FAA_POST 075 issues saying 'Settlement not possible: Down payments on 000090000127 0000 not cleared'

Cause

The reason for message FAA_POST 075 is described in SAP Note 35309 .

Resolution

Please follow below steps to fix the issue. afterwards, you can execute CJ88 successfully:

1) Ensure SAP Note 2710743 implemented in your system
2) Refer to SAP Note 2773431 and post a manual adjustment to write off the down payment in the parallel currency areas where down payment remain
3) Execute CJ88 with process type 'Full settlement'
