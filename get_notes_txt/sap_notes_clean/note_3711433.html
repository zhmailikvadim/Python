Symptom

The field IV_MR_PERIOD_LENGTH is not set when the method /US4G/IF_BADI_MP_DSO~DETERMINE_MP of BAdI /US4G/BADI_MP_DSO is called in /US4G/CL_IM_DSO_PROCESS_API so that the field cannot be evaluated in custom BAdI implementations.

This can happen on distributer (DSO) and on supplier (SUP) side.

Other Terms

UTILMD 55653, 55643, DE_B_MDC_TRG_DEV_CHANGE_SUP, DE_B_IM_DEV_CHANGE_SUP,DE_B_MDC_TRG_DEV_CHANGE_DSO, DE_B_IM_DEV_CHANGE_DSO

Reason and Prerequisites

Missing adjustment of the US4G code for a recent change in the MaCo cloud

Solution

Implement this SAP Note
