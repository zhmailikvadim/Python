Symptom

As of January 1st, 2026 the count of days with obligation to pay income compensation has changed from 10 to 14. This change has been already delivered as a part of Consolidation package 2026. But report for HI announcements needs to be updated as well to be aligned with mentioned Legal change.

Other Terms

HRSK, HSKCZPL2, T511K, VKZ21

Reason and Prerequisites

This SAP Note is relevant only for Slovakia (HRSK)

Solution

Source code of report HSKCZPL2 was updated to avoid hardcoded 10 days and all occurrences were replaced by the reading of constant VKZ21 to ensure that all possible future changes of constant value will be safely supported by this report.

Implementation
