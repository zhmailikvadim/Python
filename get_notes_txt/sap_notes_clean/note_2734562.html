Symptom

  • Currently, in Reports - Recruiting V2 domain, both 'Offer Detail' and 'Last Offer Detail' can only fetch the last approved date.
  • This KB article is a quick reference on how to report on Offer Detail Approved Date of each approver using Integration Center Report.

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Environment

SAP SuccessFactors Recruiting Management

Resolution

Necessary Recruiting Permissions:

(Please check See Also section for more information)

OData API Job Offer ExportRequired
OData API Application Export
OData API Candidate Export
OData API Job Requisition Export		Required if related fields are added to the report

Creating an Integration Center Report:

[Example offer approval]

  1. Go to Integration Center → Create Scheduled Simple File Output Integration

 

  1. Search for ‘JobOfferApprover’ entity and select necessary data fields
  2. Select 'approverActionDate' to pull approved date by each approver
  3. Select ‘approverAction’ to pull approver action (sign/reject)
