Symptom

A Production Process (PPD) is called from a button in a POD. A POD variable Process Lot is not passed to the PPD as an input parameter.

Environment

SAP Digital Manufacturing 2511

SAP Digital Manufacturing 2602

Reproducing the Issue

Configure POD, so that the Default Main Input Type is configured as SFC.

Cause

SAP DM 2511 and 2602 are designed in a way that if the Input Type is SFC, only the SFC is sent to the Production Process. If the Input Type is Process Lot, then only the Process Lot is sent. 

Resolution

Change the Default Main Input Type to Process Lot.
