Symptom

You use Transaction Manager of Treasury and Risk Management. You want to create a TRM document with big figure in the field Number of Units (e.g. Investment Fund with 1000000000 units).

You are not able to create such deal due to a program error (too small data element).

Other Terms

TRM module, Payment Reason Derivation, Security Account

Reason and Prerequisites

The issue is caused by program error.

Solution

As a general rule SAP recommends that you implement the Support Package specified for your release. However, if you need to install a solution earlier, use the SAP Note Assistant to implement the correction instruction. You can find more information about the SAP Note Assistant in SAP Support Portal, under SAP Note.
