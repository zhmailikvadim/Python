Symptom

Deleting an Inspection Plan using transaction QSR6 is giving a log message saying:

CV 537: "Task list &2 &1 (type &2) is still used in inspection lots"

Environment

  • SAP ERP Quality Management (QM)
  • SAP R/3
  • SAP R/3 Enterprise
  • SAP ERP Central Component
  • SAP ERP
  • SAP Enhancement package for SAP ERP
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA, on-premise

Cause

If an inspection plan has been used in an inspection lot; even though the UD (usage decision) was made, the inspection plan will not be deleted.
Only after all inspection lots that used the inspection plan are archived, the inspection plan can be deleted.

Resolution

To be able to delete the task list make sure of the following:

  1. There are no open inspection lots related to the task list.
  2. All related inspection lots should be archived before deleting the task list.
