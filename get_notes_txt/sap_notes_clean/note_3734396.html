Symptom

When creating an Engagement Request and within the Select Supplier page I search for a supplier but other suppliers not relevant are appearing.

Resolution

As you type, each individual character is processed as its own search string and appended to the overall query. As a workaround, you can copy and paste the full supplier name into the search field. This ensures the system performs a single search on the complete name, rather than running incremental searches for each typed letter.
