Symptom

Electronic cancellation documents processed with the mapping file eDoc_Xrechnung_3.0_PEPPOL_Marketing_Documents_(SAP)v1.07.SPP generated missing values for <InvoiceDocumentReference> (ID / IssueDate).

Reproducing the issue

Prerequisites:

Working in the localization for Germany with the electronic documents functionality set up using the mapping file "eDoc_Xrechnung_3.0_Peppol_Marketing_Documents_(SAP)v1.07.SPP" which is attached to SAP Note 2957995.

Scenario:

  1. Go to Sales → A/R Invoice and create an electronic A/R Invoice. The electronic document is created successfully.
  2. Create an electronic cancellation document based on the A/R Invoice created in previous step. Export the XML file for this document. 

Actual behavior:

The <InvoiceDocumentReference> (ID / IssueDate) fields are missing in the generated electronic cancellation document. INVOICE DOCUMENT REFERENCE is defined as preceding invoice number with linked value PrevINV.DocNum.

Expected behavior:

The <InvoiceDocumentReference> (ID / IssueDate) fields are populated correctly in the generated electronic cancellation document.
