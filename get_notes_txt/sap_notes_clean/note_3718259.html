Symptom

  • Replicate a table in LTRC and get a dump:
  • Category               ABAP programming error

    Runtime Errors         ASSERTION_FAILED
    ABAP Program           IUUC_CREATE_COBJS_IF_TBL_ENTRY
    Application Component  CA-LT-SLT
    Date and Time          03.11.2025 12:25:20

     

     Short Text
         The ASSERT condition was violated.

Environment

SAP Landscape Transformation Replication Server (SLT)

Cause

  • The entry in IUUC_MT_HEADER was deleted, which led to the corruption of the MT-ID configuration.

Resolution

  1. The configuration is corrupted and creating a new MT-ID configuration to replace it is mandatory
  2. Add the tables to the new MT-ID configuration to replicate it.
