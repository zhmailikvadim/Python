Symptom

In Fiori app Query Browser, select one query and click "Open for Analysis" button, error message "The navigation target #AnalyticQuery-preview?=Your Query Name of the app could not be resolved" raised.

Other Terms

 F1068,Query Browser,Open for Analysis,AnalyticQuery-preview, F8344

Reason and Prerequisites

From S/4HANA 2025 FPS01 release, the business user preview with multidimensional analysis needed authorization is missing in SAP pre-delivered standard businss role SAP_BR_EMPLOYEE

Solution

It is recommended to implement the fix with the next service package FPS02 containing this correction, alternatively, the below attached manual correction instruction can be implemented.

One customizing PFCG role in your system such as ZSAP_BR_EMPLOYEE,  has to be modified in below way to include transaction F8344 for business user preview with MDA for analytical query from Fiori app F1068 Query Browser.

1: Launch transaction PFCG, input your customizing PFCG role ID, such as "ZSAP_BR_EMPLOYEE"  and change it, select your transport request.

2: under the Menu tab, click add transaction, select transaction, input transaction code F8344, continue with button "Assign Transactions", save it.

3: go to Authorizations tab, click Change Authorization Data button, under object class AAAB, autorization object S_TCODE, transaction code F8344 should be in TCD field value list, click save button.

4: assign the customizing PFCG role to the users.
