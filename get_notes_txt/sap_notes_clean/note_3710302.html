Symptom

  • On Embedded Analytics, attempting to open the affected story results in the error: "An error occurred while fetching the list of members. Internal Error Occurred While processing."
  • Trying to open the same EA Story using incognito mode, clearing cache, and switching browsers does not solve the issue
  • All users are affected by this issue
  • While opening the report the  network `getResponse` endpoint is returning the following response:
    • "Text": "Internal error occurred while processing.",
                  "DetailedError": "Caught exception : exception 70000258: insufficient privilege: Detailed info for this error can be found with guid '<example_guid>'\n"

Environment

  • SAP Sales Performance Management
  • Embedded Analytics

Reproducing the Issue

  1. Open Embedded Analytics;
  2. Navigate and open the affected story;
  3. The fields of the story shows the following error: "An error occurred while fetching the list of members. Internal Error Occurred While processing."
  4. Opening the browser developer networking tab, the `getResponse` endpoints are returning the following response:
    1. "Text": "Internal error occurred while processing.",
                  "DetailedError": "Caught exception : exception 70000258: insufficient privilege: Detailed info for this error can be found with guid '<example_guid>'\n" 

Cause

The database view used for this Embedded Analytics Story does not have structured privileges applied to it.

Resolution

  1. Apply structured privileges to the database view used in the affected story
  2. Refer to the following resources for guidance:
  • Help Portal Creating SQL Views in EXT Schema with Structured Privilege Check | SAP Help Portal
  • KBA 3477443 - How to create and configure a view with Structured Privilege Check for custom data models for Embedded Analytics - SAP for Me
