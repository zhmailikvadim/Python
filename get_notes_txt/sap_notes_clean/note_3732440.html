Delivery classes for database tables for Schedule S do not correspond to the specifications of the SAP IC standard "Business Configuration"

Reason and Prerequisites

The delivery class of the database tables for Schedule S applications for defining the function control of the data extraction currently corresponds to the value "C".

According to the requirement "BC-1: Configuration Data Design" of the SAP IC standard "Business Configuration", these definitions fall under the point "3.1: Current Settings" of the standard.
As a result, the delivery class of these definitions must correspond to the value "A".

Solution

History:

March 25, 2026: New

Implement the source code corrections.

Troubleshooting Description:

This SAP Note changes the delivery classes of the relevant database tables from "C" to "A" in accordance with the specifications of the SAP IC standard "Business Configuration".

No manual changes are required.
