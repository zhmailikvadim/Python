Symptom

  • IAS tenant list is empty when provisioning SAP Databricks on SAP Business Data Cloud

Environment

  • SAP Business Data Cloud
  • SAP Databricks

Reproducing the Issue

  1. Access the SAP Databricks provisioning interface.
  2. Attempt to select an Identity Authentication System (IAS) tenant from the dropdown menu.
  3. Observe that the IAS tenant list is empty, preventing further provisioning.

Cause

  • IAS tenant is not created or available for the respective Customer CRM ID
  • The SAP Databricks login user is not present on the IAS tenant

Resolution

  1. Check the IAS tenant assignment for the respective Customer CRM ID on https://iamtenants.accounts.cloud.sap 

  2. If no IAS tenants exist, proceed to creating an IAS tenant per Get Your Tenant | SAP Help Portal 

    NOTE: newly-created IAS tenants will take 24-48hrs to appear on https://iamtenants.accounts.cloud.sap 

  3. Confirm the availability of the IAS tenants on the BDC Landscape on SAP4ME

    SAP4ME -> Portfolio & Products -> Business Data Cloud -> Customer Landscape -> Systems



  4. Once the required IAS tenant is available for the respective Customer CRM ID, make sure that the appropriate Admin users and the SAP Databricks login user exist on the IAS tenant per View Assigned Tenants and Administrators | SAP Help Portal.

  5. For clarifications on the BTP Global Account/sub-account under which the IAS tenants need to be created, contact the SAP Account Manager (see KBA 2626344 - Who is the SAP Account Manager for my company?)
