Symptom

The term QM notification is used in various texts for EAM notification workflow items.
These texts have been generalized by this note so that now only the term message is used. This is also the case for QM notifications.

Other Terms

standard task, workflow template, TG20000014, TG20000015, TG20000016, PFWS, PFTS, SWIA, Task Name, Task, Work Item text, QM notification, QMSM, RELEASED.

Reason and Prerequisites

Prerequisites

You use one of following workflows :
- WS00200074    PM: Complete task
- WS00200097    SM: Complete task
- WS00200063    QM: Complete task
- WS20000314    QM: Complete released task

 

Solution

Log on in the language EN and carry out the steps specified in the manual correction instructions. If you need the adjusted text also in other languages log on in this languages and adjust the text accordingly or do the translation via transaction SE63.
SAP can deliver this translations only via support package listed in this note.
