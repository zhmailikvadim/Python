Symptom

  • An error occurs when attempting to process UTILMD 55629 changes to technical resource data in the MaCo Cloud test system.
  • The error message displayed is: "Die erneute Verarbeitung für MKN20261102TR2 ist aufgrund von fehlenden Konfigurationseinstellungen fehlgeschlagen. Melden Sie einen Vorfall." or “The reprocessing for MKN20261102TR2 failed due to missing configuration settings. Please report an incident.”
  • The issue affects all MaCo Cloud test systems.

Environment

Market Communication Cloud Market Message Transfer

Reproducing the Issue

  1. Upload UTILMD 55629 via the "display market messages" app.
  2. Use the example edifact file provided in the appendix.

Resolution

The issue was temporary. Once the system maintenance completed, the issue was resolved.
