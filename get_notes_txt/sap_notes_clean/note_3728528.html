Symptom

While editing the connector property file, the following connector setting "actionOnInvalid" is available and it accept some specific values. See them below at the Resolution section.

Environment

SAP SuccessFactors Learning

Resolution

According to Engineering, the possible values for this feature are:

"reject": The entire record is rejected
"default": The @DEFAULT@ transformation value is used
"nullify": Leaves value as null

Keywords

connector, user, sf, transformation, actionOnInvalid, rule, default, reject, nullify
