Symptom

When installing the SDK, the following error occurs:

Cannot complete the install because one or more required items could not be
found.
Software being installed: AdapterFramework 2.4.2
(com.sap.hana.dp.feature.feature.group 2.4.2)
Missing requirement: com.sap.hana.dp.adapter 2.4.2
(com.sap.hana.dp.adapterframework 2.4.2) requires 'package
com.google.common.collect 27.1.0' but it could not be found
Cannot satisfy dependency:
From: AdapterFramework 2.4.2 (com.sap.hana.dp.feature.feature.group 2.4.2)
To: com.sap.hana.dp.adapterframework [2.4.2]

Other Terms

Replication, SDI, target, log file

Environment:  

SAP HANA SDI 2.0 SP04 Patch 23 (2.4.2.3)

Reason and Prerequisites

We occasionally invoke com.google.common.collect 27.1.0 in the SDK

Solution

This correction is delivered in the release(s) listed in the Support Packages and Patches section of this SAP Note.

com.google.common.collect was changed to 15.0.0
