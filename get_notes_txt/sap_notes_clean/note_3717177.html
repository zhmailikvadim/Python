Symptom

When an employee’s service anniversary coincides with the pay period end date, payroll treats the scenario as if the employee were terminating on that date. As a result, the system considers an incorrect accrual portion for leave liability, leading to an incorrect leave liability amount.

Other Terms

  • Programs : HNZCALC0, HNZLLVL0, HNZCLV00
  • Transactions : PC00_M43_CALC, PC00_M43_LLVL
  • Function Modules : HR_CALC_QUOTA_PAYMENTS_NZ, HR_CALC_LEAVE_PAYMENTS_NZ
  • Other terms : Entitlement, Accrual, Leave liability, New Zealand leave processing

Type of Object Change

Workbench Objects - No
Code - Yes
Schema/Rules - No
Wage Types - No
Customizing Tables – No

Reason and Prerequisites

The system determines the gross amount using an incorrect LVAWE record, which leads to an incorrect accrual calculation.

Solution

The code in function modules HR_CALC_QUOTA_PAYMENTS_NZ and HR_CALC_LEAVE_PAYMENTS_NZ has been enhanced to ensure the correct LVAWE record is selected for deriving the gross amount.

The required code changes have been done to resolve the issue mentioned in 'Symptom' above.
