Symptom

1. You are using the Manage Assortment application in SAP Customer Activity Repository applications bundle 5.0 FPS03, 5.0 FPS04, 5.0 SPS05, 5.0 SPS06, 5.0 SPS07, 5.0 SPS08

2. You use extension hooks in "retail.rap.genericassortment.Util.ODataServices", e.g., extHookGetProductsForProductModuleTable, however, you custom code for the extension hooks are not called.

Other Terms

SAP Customer Activity Repository applications bundle, CARAB, Manage Assortments, extension hook.

Reason and Prerequisites

This is a program error.

Solution

Implement this note.

** Changes to the content of the SAPUI5 ABAP repository require the index to be updated using the calculation report '/UI5/APP_INDEX_CALCULATE'. When the correction instruction is applied via a note, please ensure this report is executed either manually or by an existing scheduled periodic background task after the note is applied. See note: 2227577 - Recalculation of application index after implementing a note.
