Symptom

It is not possible to send an e-mail via Outlook from SAP Business One because New Outlook does not support opening from a desktop client application.

Reproducing the issue

Prerequisites: 

You are using new Outlook.

Scenario:

  1. Go to Administration → System Initialization → General Settings → Services tab.
  2. Select Outlook e-mail as your default e-mail method.
  3. Use the e-mail Icon to send an e-mail.

Expected behavior:

SAP Business One Client will open the Outlook Client to send an e-mail.

Actual behavior:

SAP Business One Client cannot open the Outlook Client because New Outlook does not support opening from a desktop client application.

Cause

System limitation

Solution

This is a system limitation
