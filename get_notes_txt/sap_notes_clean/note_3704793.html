Symptom

When calling API_MAINTENANCEORDER to search for Maintenance Orders by a Technical Object Label value that does not exist, the API raises a short dump (runtime error) instead of returning an empty result set. As a result, the search call fails and any process relying on this query cannot handle non‑existent labels.

Other Terms

API_MAINTENANCEORDER, API_MAINTENANCEORDER_2, A_MaintenanceOrder, FunctionalLocationLabelName, CL_EAM_TECHOBJ_FILTER_EXIT, CX_SADL_DUMP_APPL_MODEL_ERROR

Reason and Prerequisites

Program error.

Solution

Apply this SAP Note.
