Symptom

Client needed to know if there was any config available for auto closing of profile workers.

Environment

Profile Worker - Edit

Cause

Resolution

  1. We have a company configuration setting: Auto close Profile Worker when end date exceeds (# of days)?
  2. When enabled, Profile Workers will be automatically Closed after their End Date exceeds the number of
    days specified in the company config.
  3. All statuses of PW will be supported, and allow auto close.
