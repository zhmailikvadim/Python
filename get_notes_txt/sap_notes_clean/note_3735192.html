Symptom

  • When opening a story, the browser developer console displays: Unstable calculations with the same guid or name detected. GUIDs and names should be unique across all calculations
  • The console also logs a calculation detail object such as: {guid:'MeasureValues', name:'xxx', type:'restrictedMeasure', subType:'undefined', isDeleted:'false'}
  • “Unstable calculations with the same guid or name detected…”
  • Users ask whether this console message affects story functionality

Environment

  • SAP Analytics Cloud (SAC) Enterprise 2026
  • SAC Story Calculations

Reproducing the Issue

  1. Open a story in the browser.
  2. Open the browser’s developer tools and go to the Console tab.
  3. Observe messages similar to: “Unstable calculations with the same guid or name detected…” and an object like {guid:'MeasureValues', name:'xxx', type:'restrictedMeasure', subType:'undefined', isDeleted:'false'}.

Cause

  • This is a warning message which can be ignored. This message is for the developer and can be ignored. 
  • It has no functional impact on the story.

Resolution

  1. Treat the message as informational only; it has no functional impact on the story.
  2. No content changes in the story are required unless you experience actual functional problems.
