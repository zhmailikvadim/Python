Symptom

In the process "DTD Checks for Invoice of Grid Usage Billing" (DE_INV_GUB_DTD), valid as of 01.10.2025, step 0400 requires an error correction due to legal changes issued by the BDEW that were published on 16.01.2026.

This SAP Note is relevant for the following role: SUP (Supplier).

Other Terms

IDEXGE, Error Correction, 16.01.2026, Grid Usage Billing, DTD checks for Invoice of Grid Usage Billing, DE_INV_GUB_DTD, 491, 492, 494, GE_INVC_DTD_GUB_2510_CHK_494, E_0406, E_0407

Reason and Prerequisites

Error correction due to legal change.

Solution

Implement the corrections in this SAP Note or import the relevant support package.

After implementing this SAP Note, when the process value of the item level DTD check step 491 is ZERO_AMT, the system continues to the next item.

You need to reset the invoice and then trigger the DTD check process again.

You can verify the fix by checking the DTD check process.
