Symptom

Why the Reviewer getting Replace Document option under Review Task and not able to submit the changes in it?

Resolution

Under the Contract Workspace (CW) Template check for the Review Task has the option Reviewer Can Edit is set to Yes or No.

Because of this option set to Yes the Reviewer will have the option Replace Document and cannot submit or attach the reviewed document to the task and that impacts the option or Review Changes under the CW Review Task

To check the option 

  1. Navigate to Review Task > Click on dropdown
  2. Edit Task > Advance Task Details
  3. Under Advance Task details check option Reviewer Can Edit is set to Yes or No.
  4. If set to Yes the change it to No and Save.

So the Reviewer will not get the Replace Document option and can submit the Reviewed documents.
