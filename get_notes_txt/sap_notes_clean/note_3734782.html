Symptom

In the Console tab of the browser dev tools errors "Failed to load resource: the server responded with a status of 404 (NOT FOUND)" are observed when a SAPUI5 application tries to load /sap/suite/ui/smartbusiness/ resources.

Other Terms

/sap/suite/ui/smartbusiness/, smartbusiness, 404 not found

Reason and Prerequisites

Program error

Solution

Implement the automatic correction instructions of this SAP note using SNOTE.
