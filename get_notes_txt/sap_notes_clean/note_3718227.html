Symptom

IL-includes are missing in the migration object definition and generated runtime objects for tables which have an entry in table /NZT/ASS_RUL_MAP on the central system for a given mass transfer. As a result, the transfer of these table may yield inconsistent results.

Other Terms

downtime minimization tool for SAP S/4HANA, NZDT

Reason and Prerequisites

Any of the following add-on support packages is installed in your system landscape:

  • NZDT 1.0 SP02
  • NZDT 1.0 SP03

Solution

In order for this note correction to be effective, the note must be implemented before:

  • migration objects are defined / created. If the migration objects are already created, they must be re-created, and the runtime objects must be re-generated as well

Implement the correction instructions in transaction SNOTE on the following systems of your system landscape:

  • central system
