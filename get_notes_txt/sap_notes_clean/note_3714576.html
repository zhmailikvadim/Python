Symptom

In Agent Desktop there is no Payment option available under the Create Button

Environment

SAP Service Cloud Version 2

Reproducing the Issue

  1. Go to Agent Desktop
  2. Search for a Contract Account value and accept
  3. Click on the Create button under "What would you like to do?"
  4. Payment option is not available.

Cause

Option is not enabled and needs to be enabled via Adaptation Mode

Resolution

  1. Go to Agent Desktop
  2. Search for a Contract Account value and accept
  3. Start Adaptation Mode
  4. Click on the + button under "What would you like to do?" and then select the Pencil Icon
  5. Click on the Arrow beside Create
  6. Make Payment Visible
