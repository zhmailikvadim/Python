Symptom

A dump is encountered either after upgrading to SAP S/4HANA Cloud Private Edition 2025 or after implementing Note 3589823 (Customizing Table for KE396):

Category               ABAP programming error
Runtime Errors         PERFORM_PARAMETER_MISSING
Except.                CX_SY_DYN_CALL_PARAM_MISSIN
ABAP: Program          SAPLKEAK
Application Component  CO-PA

It is observable that this occurs in COPA_PROFITABILITY_SEGMENT Function Module under perform MOVE_QUANTITY_UOM.

Environment

  • Controlling and Profitability Analysis
  • SAP ERP Central Component
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA Finance
  • SAP S/4HANA

Reproducing the Issue

  1. Execute the COPA_PROFITABILITY_SEGMENT Function Module.
  2. Attempt to call the MOVE_QUANTITY_UOM and DERIVE_CRITERIA FORMs from the RK2L* Report.
  3. Observe the dump in ST22 with the error message "PERFORM_PARAMETER_MISSING".

Cause

Incorrect implementation of SAP Note 3589823.

Resolution

  1. De-implement SAP Note 3589823.
  2. Review the manual steps required by SAP Note 3589823.
  3. Re-implement SAP Note 3589823, ensuring that all manual steps are executed correctly.
    • Specifically, execute the function module RKE_GENERATE_INTERFACE_ACT as part of the manual steps.
  4. Verify the resolution by testing the COPA_PROFITABILITY_SEGMENT Function Module.

See Also

SAP Note 3589823 - KE396 Customizing table
