Symptom

In the technical settings for the database tables, you can define that changes to the table are logged. This is only required in very specific situations, for example, to analyze errors that cannot be reproduced.

This function is activated for the table CABNT in the standard settings. This is unnecessary.

Other Terms

SE11

Reason and Prerequisites

This problem is caused by an incorrect entry for the table CABNT

Solution

Implement the correction instructions.
