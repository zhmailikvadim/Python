Symptom

  • Fiori app F2031 "Monitor Project Progress" is not available.

Environment

 SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Unable to find app "Monitor Project Progress" via Enterprise Search or App Finder.

Cause

The behavior is standard: Fiori app F2031 "Monitor Project Progress" is not a standalone app and is accessed via related apps (for example, "Project Control – Enterprise Projects" or "Project Briefs").

Resolution

  1. Ensure Scope Item 1YF is active.
  2. Do not search for "Monitor Project Progress" as a standalone tile in the launchpad.
  3. Access the app via other Project apps, for example, "Project Control – Enterprise Projects" or "Project Briefs".
