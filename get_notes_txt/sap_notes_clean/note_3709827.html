Symptom

A custom email template was configured using the Onboarding Activity Recipient Builder for the CC Recipient Builder.

Environment

SAP SuccessFactors Onboarding

Reproducing the Issue

  1. Configure a custom email template
  2. Use the Onboarding Activity Recipient Builder for the CC Recipient Builder.
  3. In production, assign users to primary and secondary tasks (e.g., IT Checklist and AP Checklist).
  4. Complete an onboarding task
  5. Observe that the user who completed the IT Checklist receives the email as expected, but users assigned to the AP Checklist also receive a blank copy of the IT Checklist email.

Cause

Configuration issue related to the use of the Onboarding Activity Recipient Builder as the CC Recipient Builder.

Resolution

  1. Verify the configuration of the custom email template
  2. Confirm that the Onboarding Activity Recipient Builder is set as the CC Recipient Builder, which sends emails to all responsible users of an activity/program.
  3. To ensure emails are sent only to specific recipients (e.g., Manager or a specific user), use the Explicit Recipient Builder to specify users by their email or name.
