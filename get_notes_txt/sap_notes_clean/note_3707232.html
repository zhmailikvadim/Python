Symptom

The Accrual Object field is not available in journal entry postings.

Environment

  • SAP S/4 HANA
  • SAP S/4 HANA FINANCE
  • Financial accounting (FI)

Reproducing the Issue

Try to post accrual and notice that accrual object is not updated in ACDOCA.

Cause

The Accrual Object field can be made available in posting screens by configuring it as optional or mandatory in the field status of the corresponding G/L account. 

Resolution

When entering a journal entry, the field status of each field (including the Accrual Object) is determined by the combination of: 

  • The Field Status Group maintained in the G/L Account Master Record (Transaction FS00) 

  • The Field Status defined in the Posting Key Configuration (Transaction OB41) 

Hence in the settings 'Accrual object' shall be made available:

This configuration is performed in Transaction OB14 and OB41, under the section “Additional Account Assignments.” 

Transaction OB14 requires specifying the Field Status Variant, which is defined in the company code master data (Customizing view V_001_B – “Enter Global Parameters”). 

 
 
 

Keywords

OB14, OB41, Field status, field status variant, accrual object, ACDOCA, accrual
