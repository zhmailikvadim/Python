Symptom

The new (Feature Set B) global user list in the BTP Cockpit is missing the first and last name for some users.

The affected users are the Global admins from before the cockpit was upgraded to Feature Set B.

Environment

SAP Business Technology Platform, Cloud Foundry environment

Cause

Limitation. Currently the first name and last name are not considered during the Feature Set A -> Feature Set B migration.

Resolution

As a workaround refer to Create Users documentation.

In this documentation there is a note which describes this scenario and explains how to solve it.

See Also

User and Member Management

CF, FSB, FSA
