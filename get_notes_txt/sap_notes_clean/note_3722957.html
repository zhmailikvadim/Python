Symptom

If you choose a different category than PLN when using C_ACTYTYPEPLANCOSTRATE_SRV, you receive an error saying that only PLN is allowed.

Other Terms

C_ACTYTYPEPLANCOSTRATE_SRV, e025, FINS_ACDOC_API, Universall Parallel Accounting, F3162, Manage Cost Rates - Plan

Reason and Prerequisites

API Restrictions

Solution

If you have activated the business functions Universal Parallel Accounting (FINS_PARALLEL_ACCOUNTING_BF, short: UPA) to use Manage Cost Rates - Plan (F3162), you still need to use category PLN, as any other is prohibited.

With this correction, you can export to any category.

Please implement the correction instructions or the corresponding Support Package.
