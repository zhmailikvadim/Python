Symptom

At the bottom of the Choose Layout pop-up, print, sort and filter buttons are not visible.

Environment

SAP S/4HANA Cloud Private Edition 2023

Cause

Standard expected behavior.

Resolution

These buttons have been removed from the bottom of the Choose Layout pop-up. However, these functions are still available through the right-click context menu.

The print function has been intentionally removed.

Keywords

Choose Layout, Button, Print, Sort, Filter, Not Visible
