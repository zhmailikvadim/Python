Symptom

When a column is selected using the context menu action (mouse button or the respective keyboard shortcut) and the subsequent action causes a new data table to be sent from the backend, the column may not get selected on a immediately following context menu action.

Example: In any Gridview open the context menu on column index N (which selects column N and opens the menu), select Hide from the context menu and then open the context menu on column index N (which is now another column, because former column N got hidden) again: the context menu opens, but yet no selection is performed.

Other Terms

ITS, SAP GUI for HTML, WebGUI

Reason and Prerequisites

Program error

Solution

Important: If there is no entry in the section Support Package Patches with the patch number for the desired kernel release yet, please re-check this SAP note at a later point in time. In this case the patch is not yet available for download and still in production or validation. The patch number will be added automatically once the corresponding patch is available on SAP Support Portal. You can subscribe to a SAP note to get informed when it is updated.

To resolve this issue please do the following:

Determine the patch number of the SAPEXE SAR package with the highest patch number for your kernel version on the SAP Support Portal. If the SAPEXE patch number is higher than or equal to the highest patch level number you see in the section Support Packages Patches of this SAP note for your SAP Kernel version, you should download and install the SAPEXE SAR package.

Otherwise you have the following options:

  1. If the system runs with the latest 916 or 793 stack kernel the recommended method is to apply the SAPWEBGUI.SAR package without updating the full SAP Kernel. For details about SAPWEBGUI.SAR please read SAP note 2412840 - Kernel Patch SAPWEBGUI.SAR

  2. The system does not run with the latest 916 or 793 download and install the most recent DW.SAR archive.

Once the section Support Package Patches below in this note contains the SAP Kernel patchlevel of the release you are interested in, you can find up to date information about the status of the patch delivery in the following SAP notes:

SAP Kernel 916 (S/4HANA 2025, 2023, 2022, 2021 and 2020): 3605070 

SAP Kernel 793 (S/4HANA 2023, 2022, 2021 and 2020): 3338320

FAQ:

Expected delivery dates of SAP Kernel patches

  • SAP Kernel patches are usually delivered weekly or bi-weekly on Friday depending on the release. From the patch level of latest delivered patch you can roughly conclude when the patch you need will become available.

Downloading SAP Kernel patches

  • All patches are available in the Software Downloads section on SAP Support Portal (http://support.sap.com/patches). Detailed information on the exact location of the patches can be taken from note 563161. SAP Note 330793 explains how to download patches from SAP Support Portal.

Installation of SAP Kernel patches

  • SAP note 19466 explains how install SAP Kernel patches.
