Symptom

Error message AA633 - "Manual depreciation not allowed with depreciation key" occurs when attempting to create a new interval with an automatic forecast  depreciation key in the Manage Fixed Assets

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Open manage fixed assets app
  2. Add interval
  3. Change depreciation key from MANU to an automatic depreciation key;
  4. Save.

Cause

If the depreciation key used is a manual depreciation key and there have been some manual depreciation posted on the asset it is not longer possible to change it to a depreciation key with automatic calculation.

Resolution

  1. Reverse the manually posted ordinary depreciation
  2. Add the interval with an automatic forecast  depreciation key 
  3. Save
