Symptom

How to exclude declined suppliers from next round for multi-round events?

Cause

Resolution

In order to exclude declined suppliers from participating in the next round in a multi-round event, the parameter  Application.AQS.ExcludeDeclinedSuppliersFromNextRound needs to be enabled.

Please note that events that had already been created before this parameter was enabled may not be affected by this change.

The default value for this parameter is No.

See Also

This parameter is not self-service, if you need to enable it please contact SAP Support to have it enabled: How to create a Service Request on SAP for Me
