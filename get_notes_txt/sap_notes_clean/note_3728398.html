Symptom

You are using the Business Transformation Center and want to apply the latest corrections and features which are delivered as part of S/4HANA 2025 FPS1.

Other Terms

 BTC

Reason and Prerequisites

S/4HANA 2023 FPS1 or higher.

Solution

Please implement the Transport-based Correction Instruction (TCI) attached. This update includes bug fixes and ensures the application runs smoothly at all times.

Improvements

  • Improvements in Access plan calculation and static delimitations
  • Improvements in Cross-System Viewer : Addition of Busy/Idle Chart, Load Curve and Box Plot and statistics summary.
  • TransformationExecution.Start - Parallelization of command processing
  • Migration calculation/portioning status adjustment for reading type 5 tables
  • Drive mode improvement for migration tuning of calculation/portioning parallelization
  • Post-Processing improvement for automatic handling of GOS attachments
  • Enhanced renumbering logic to prevent duplicate renumbering mapping for the same old number

Note: To implement a TCI Note, the SNOTE transaction of the ABAP system must be enabled for the same. To enable SNOTE for TCI refer to the below resources.

2187425 - Information about SAP Note Transport based Correction Instructions (TCI)
