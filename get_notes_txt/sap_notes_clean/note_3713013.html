Symptom

You want to restore the lead that was accidentally deleted by a user

Environment

  • SAP Sales Cloud and SAP Service Cloud Version 2

Cause

This is expected system behaviour.

Resolution

You can fetch the details of the deleted lead as long as:

1. You have the UUID:
GET:
https://myxxxxxx.test.crm.cloud.sap/sap/c4c/api/v1/lead-service/leads/18desbf7-0000-11f1-a9b4-a3ea97f4fxxx

2. It is not more than 183 days since the time of deletion. The logs are stored for 183 days.

Keywords

restore, leads, lead, delete, deleted, deletion, user
