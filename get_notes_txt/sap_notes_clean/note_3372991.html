Symptom

SAP BusinessObjects Web Intelligence has a URL with parameter that could be vulnerable to XSS attack. The attacker could send a malicious link to a user that would possibly allow an attacker to retrieve the sensitive information.

Other Terms

﻿XSS, reflected XSS,CVE-2023-42474

Reason and Prerequisites

The application does not sufficiently encode user-controlled inputs.

Solution

The vulnerable parameter of the URL is not used anymore and has been removed.

This issue is fixed in the patches listed in the "Support Packages & Patches" section below.

For Business Intelligence Platform maintenance schedule and strategy see the Knowledge Base Article 2144559 in References section.
