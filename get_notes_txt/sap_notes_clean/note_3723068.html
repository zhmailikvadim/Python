Environment

  • SAP S/4HANA, on-premise
  • SAP S/4HANA Cloud Private Edition

Resolution

Implement following SAP Note to resolve the DUMP:

3497952 - ALV Layout - Short dump when choosing additional fields in layout selection.

***

Remark:

This SAP Note has an intended effect of hiding the options like 'Show...' and 'Hide' from the context menu while adjusting the layout. 

This way also the option for 'Technical name' display - which is connected to the 'Show...' option - becomes unavailable. Nevertheless, with following workaround it still can be made visible:

'Ctrl + Shift + F6' key combination can make the 'Technical name ' column visible, while 'Ctrl + Shift + F7' key combination can hide the 'Technical name ' column.

****

Read following SAP Notes for more details:

3613519 - ALV: Missing functions in the column header context menu of Choose Layout popup
"...
From now on, the system adds back the functions for Filter and for Sorting.
Other functions that were available in previous releases, such as Show and Hide will still be removed from the context menu, as they can manipulate the internal Layout screen causing errors to the application.
..."

1009650 - Consulting: Standard behavior in the ALV
"...
[39] The dialog boxes for selecting and saving a layout variant no longer offer functions such as "Hide" and "Show" for layout changes and for printing.
...
[39] In the same way as for the changes to the display of the context menu of an ALV Grid as of Release 758 (see SAP Note 3468780), some functions of the ALV Grids that are hidden in the PF status or in the toolbar are no longer displayed in the context menu of a column header of the "Select Layout" and "Save Layout" dialog boxes (SAP Note 3497952).
The display of layouts can be restricted by selecting the 'Layout Settings' dropdown menu in the 'Select Layout' dialog box.
In both dialog boxes, you can use the sort functions on the heading of the 'Layout' column.
The "Find" and "Set Filter" functions are still available in the context menu.
..."
