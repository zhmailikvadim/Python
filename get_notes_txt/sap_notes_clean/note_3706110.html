Symptom

  • A custom field is created in SAP CDP with a Boolean data type
  • The field is automatically created in Emarsys as a Text field instead of Boolean

Environment

SAP Customer Data Platform

Cause

  • Emarsys does not support Boolean field type. The field created in SAP CDP will show up as Text field in Emarsys.

Resolution

This is an expected behaviour, please refer to the Emarsys product documentation: Field types

Keywords

SAP CDP, Emarsys, boolean field, string field, field mapping, automatic activation, custom field type, sap cdp-emarsys integration, field transformation, supported field types, customer data platform
