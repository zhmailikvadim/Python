Symptom

  • The "Taxes" button does not respond when clicked within the Purchase Order – Advanced app in SAP S/4HANA Cloud Public Edition.
  • The issue persists across multiple users and purchase orders.
  • No error messages or codes are displayed when the button is clicked.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Open the Create Purchase Order – Advanced app.
  2. Create or access an existing purchase order.
  3. Attempt to click on the "Taxes" button.
  4. Observe that the button does not respond or display any information.

Cause

The Scope Item 1J5 for Brazil has not been activated, which prevents the system from displaying Condition Types associated with the related Tax Code.

Resolution

  1. Activate Scope Item 1J5 for Brazil.
  2. Verify that the Condition Types associated with the related Tax Code are correctly configured.
  3. Test the functionality of the "Taxes" button after activation to ensure it displays the expected tax information.
