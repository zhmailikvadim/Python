Symptom

If the ENABLEJRN property is maintained in one dimension, and the value of it is not maintained as 'Y', these members will not display in the context bar of the consolidation monitor.

This property should only affect the journal component.

Other Terms

ENABLEJRN, context bar, consolidation monitor

Reason and Prerequisites

This is a programming error.

Solution

For SAP Business Planning and Consolidation 10.1, version for SAP NetWeaver, upgrade to (Component CPMBPC 810) support package 06 and upgrade BW740 (Component SAP_BW) to support package 11.
