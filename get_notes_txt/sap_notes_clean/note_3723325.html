Symptom

 When performing a mass change in the Agreement Workbench to update the Sales Office and Sales Group fields, the system clears the existing field values instead of updating them with the new entries. As a result, the fields remain blank after execution of the mass change.

Other Terms

/irm/cl_ipagr_process, GENERIC_DATA_GET, VKBUR, VKGRP, Mass Change

Reason and Prerequisites

This is caused by program error

SAP Note#3723447 is available for V/4 Release

Solution

Please implement the correction instructions attached. These corrections are only valid if you have applied the Support Packs.

Release 6.4 - Base Package and Support Pack 1

 Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object.
