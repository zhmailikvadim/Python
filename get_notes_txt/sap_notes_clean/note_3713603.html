Symptom

Error message "Invalid filename..." when uploading a section document with parentheses "()" in the filename.

Environment

SAP SALES CLOUD CPQ

Reproducing the Issue

  1. Login in CPQ
  2. Create a quote
  3. Click on Generate document
  4. Go to customize
  5. Try to Upload new file that contains a parenthesis in file name.

Cause

Undocumented limitation of the application regarding the usage of parentheses in the filename when uploading files in document generation.

Resolution

Parentheses cannot be used in filename to upload files in document generation.

Keywords

parenthesis, file, filename, document, generation, quote, CPQ, invalid
