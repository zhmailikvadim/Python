Symptom

  • The Amount field (IDX_AMOUNT) and the DSO field (IDX_FIGURE) in the UKMBP_VECTOR_IT table is not populated.
  • The Amount field (IDX_AMOUNT) is not updated by the UKM_TRANSFER_VECTOR program.

Environment

  • SAP S/4HANA
  • SAP S/4HANA Finance
  • SAP enhancement package for SAP ERP, version for SAP HANA

Cause

  • The UKM_TRANSFER_VECTOR wasn't executed.
  • The IDX_AMOUNT field is not populated by SAP Standard, it has been added to the relevant structure and tables for optional customization.

Resolution

1. To update the DSO field (IDX_FIGURE), execute the UKM_TRANSFER_VECTOR report. For more details refer to KBA 3479741.

2. The Amount field (IDX_AMOUNT) can be populated by creating an implementation for the UKM_FILL BAdI functionality. Ensure both methods, FILL_VECTOR and FILL_FIELDS, are implemented during customization. For more details refer to KBA 3499166.

Note that the IDX_AMOUNT field is not directly related to the DSO field (IDX_FIGURE).
