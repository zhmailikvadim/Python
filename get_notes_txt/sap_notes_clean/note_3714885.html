Symptom

In the process "New Installation Process (SUP View)" (DE_B_GPKE_NIN_SUP), the customer extension in the Checkgroup is not triggered in steps 8950 and 0850.  
This issue is relevant for the market role: SUP.

Other Terms

GPKE, Checkgroup, DE_B_GPKE_NIN_SUP, CHKGRP_SUP_NIN_END_EXT, BC set, new installation process, SUP, 8950, 0850

Reason and Prerequisites

The error occurred because of an incorrect precondition in the Checkgroup, which prevented the customer extension from being triggered.

Solution

Implement the corrections of this SAP Note or import the relevant support package.  
After implementing this SAP Note, the system correctly triggers the customer extension in the Checkgroup. Activate the BC set /US4G/1_0_SUP_SP08_3714885.  
You can verify the fix by checking that the customer extension is triggered in the Checkgroup during the process.
