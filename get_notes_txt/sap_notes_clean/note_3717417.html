Symptom

While running wdeploy getting error(DEP00302)

Environment

  • SAP BusinessObjects Business Intelligence (BI) Platform 4.x
  • Supported all operating system

Reproducing the Issue

  1. Run the command wdeploy tomcat9 -DAPP=BOE deploy
  2. Getting below error :
  3. Error: Application server supplied instance name is not correct. (DEP00302)

Cause

The paramter as_instance was incorrect.

Resolution

  1. Navigate to C:\Program Files (x86)\SAP BusinessObjects\SAP BusinessObjects Enterprise XI 4.0\wdeploy\conf\config.tomcat9 file.
  2. The paramter for as_instance was incorrect.
  3. After making the changes, we were able to run the wdeploy.
