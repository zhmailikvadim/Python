Symptom

You run the Production Operation List integration with Phantom Components and notice the following symptoms:

1. You send a Production Operation List with Phantom Components DM message, but the BOM Application setting is missing in the POL L1 mapping customizing. You receive the error message No suitable object found, which does not provide enough information.

2. You send a Production Operation List with Phantom Components DM message that uses a Structure Version which is not yet integrated. The system raises an internal runtime error with status 500.

Other Terms

PLMSI, POL, Phantoms, /PLMF/POL_CORE108, /PLMF/POL_CORE109

Solution

Implement this SAP Note to improve error messages for Production Operation List integration with Phantom Components.
