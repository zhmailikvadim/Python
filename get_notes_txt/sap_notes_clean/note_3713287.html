Symptom

When doing remote client copy via SCC9N(New client copy tool), you want to specify RFC groups same as old client copy tool SCC9.

Environment

SAP_BASIS release 754 and above using new client copy tool.

Cause

The new client copy tool does not support load balancing across RFC server group.

Resolution

The new client copy tool does not support load balancing across RFC server group. The new client copy tool uses current application server for load distribution. 

You can provide the number of parallel processes on the SCC9N launch screen under 'Processing Settings'.

See Also

3577605 - How to specify RFC groups when doing remote client copy in SCC9

Keywords

SCC9N RFC server, RFC server client copy, client copy parallel process
