Symptom

Symptom

You are opening a Bank Account from Manage Bank Accounts app. You wanted the system to trigger a correspondence letter upon triggering the performing the action using the action button 'Start Opening'.

Other Terms

F1366A, Open, e-mail, letter

Reason and Prerequisites

The Bank Correspondence was not available earlier for Bank Account opening process, now it is enabled.

Solution

Implement the corrections of this SAP Note and follow the below steps.

  1. Create the e-mail and the form templates as per your requirement.
  2. IMG Settings: Assign the newly created templates in the configuration
    • Assign Email Templates: SAP Customizing Implementation Guide -> Cross-Application Components -> Output Control -> Assign Email Templates
    • Assign Form Templates: SAP Customizing Implementation Guide -> Cross-Application Components -> Output Control -> Assign Form Templates
  3. The Account action will be '01 - Open Bank Account' for opening the bank account. In Transaction Code OPD (Output Parameter Determination) use this action '01' for configuring the Bank Correspondence object for correspondence letter in opening process.

To get the standard e-mail template FCLM_BAM_OM_ACCOUNT_OPEN and the form template FCLM_BAM_ACCOUNT_OPEN upgrade to the relevant support package or higher.

Other Terms

FTW1A, FEC, Control Totals

Reason and Prerequisites

The Control Totals is not to be calculated for period 0 for FEC. Due to program error, the totals is calculated and this results in errors.

Solution

Implement the corrections via SNOTE.
