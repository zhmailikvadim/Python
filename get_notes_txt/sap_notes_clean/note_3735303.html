Symptom

About the layout setting for cost component split columns in app 'Material Price Analysis'.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Open the Material Price Analysis (CKM3) application.
  2. Enter a material and plant, and set currency/evaluation as needed.
  3. Choose 'Change Layout' and select cost component split (CCS) columns (for example, direct material) to remove the column.
  4. Save the layout, observe that the selected CCS columns are not removed and may appear at the front of the table.

Cause

The Change Layout function works only for non–cost component split (CCS) columns in Material Price Analysis; CCS columns cannot be removed through Change Layout by design.
The main reason is that the following ALV tree will display the columns based on the CCS selected in the second view, for example view 'H Main Cost Component Split' and it will override the layout change settings.

Resolution

It's suggested you to right click the CCS column and choose 'Hide column' instead(see screenshot 'Hide column.jpg').
