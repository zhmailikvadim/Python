Symptom

The tickbox of permission Worker can submit Time Sheet revisions is greyed out and cannot be deactivated by design once it is enabled.

I want to to be disabled from the back end, if it is possible.

Resolution

Workers can submit Time Sheet revisions" was previously named as 'Revise Time Sheet for Non Rate Schedule Templates'.

Company configuration: Workers can submit Time Sheet revisions is applicable only for the Non Rate schedule Time sheets, as we have 2 options - CRDB, REVISE. 

This means when this configuration is ON, Company can create only Time Sheet Revisions and not Credit Debit Memo (CRDB) for correction, when Non Rate Schedule Time sheets are used.

 

So if we turn off this config, Buyer company will lose the ability to create Time Sheet revision for Non Rate Schedule TS. 

We suggest not to disable this flag.

Keywords
