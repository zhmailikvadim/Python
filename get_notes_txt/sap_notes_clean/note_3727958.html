Symptom

When PCW (Transaction FSH_PCW) is first opened, it correctly starts in Change Mode. After pressing the Back button to return to the selection screen and opening PCW again, it unexpectedly switches to Display Mode.

Other Terms

PCW, Production Control Workbench, Combined Master Production Order, Master Production Order, Production Order, Combined Master Planned Order, Master Planned Order, Planned Order

Reason and Prerequisites

Reason 

This problem is caused by a program error.

Prerequisite - User parameter FSH_PCW_DISPLAY is not set or is blank (‘ ’), which defaults PCW to Change Mode.

Solution

Implement the correction instruction provided with this SAP Note.
