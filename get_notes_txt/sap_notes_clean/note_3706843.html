Symptom

  • SAP RAR Pending Review Worklist functionality is not available for contracts created using Optimized Contract Management (OCM).
  • A Performance Obligation (POB) reflects status Pending Review.
  • Suspend Posting checkbox is selected.
  • No error message is displayed.

Environment

  • Revenue Accounting (RA).
  • Financial Accounting (FI).
  • Fiori UI for Revenue Accounting and Reporting
  • SAP ERP Central Component.
  • SAP ERP
  • SAP enhancement package for SAP ERP.
  • SAP enhancement package for SAP ERP, version for SAP HANA.
  • SAP S/4HANA.
  • SAP BRF +

Reproducing the Issue

  1. Log in to the Fiori Launchpad (/UI2/FLP) to access the Conflicted Revenue Contracts Worklist user interface (UI) (App ID: F4271).
  2. Locate the specific contract containing the POB in question.
  3. Check POB Status Details.

Cause

Optimized Contract Management (OCM) utilizes a different architecture based on real-time processing and immediate validation. This real-time processing model eliminates the intermediate review stages that the Pending Review Worklist relies upon.

Resolution

As Pending Review Worklist functionality is only available for Classic Contract Management, SAP Fiori apps for managing revenue contracts serve as a workaround for the issue. 

Conflicted revenue contracts can be reviewed and conflicts for attribute changes can be resolved using the Conflicted Revenue Contracts Worklist user interface (UI) (App ID: F4271).
