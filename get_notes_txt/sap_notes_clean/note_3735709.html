Symptom

You want to map fields in Recruiting so that data flows correctly to Onboarding.

Environment

  • SAP SuccessFactors Onboarding
  • SAP SuccessFactors Recruiting Management

Resolution

To ensure correct data flow from Recruiting to Onboarding, field mapping must be properly configured.

For detailed instructions, step-by-step guidance, and additional resources, refer to the following SAP Help Portal documentation: Help with Mapping Fields from Recruiting to Onboarding | SAP Help Portal.
