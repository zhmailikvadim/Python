Symptom

  • Veteran Fields does not show in Manage Pending Hires.
  • Global Info field 'Veteran Status' shows empty during Rehire Verification Step even though a value is available in the Rehire's Employee Profile.

Environment

SAP SuccessFactors Onboarding

Reproducing the Issue

  1. Login to the instance as the HR Coordinator or Hiring Manager
  2. In To Do, open Employment Profile Selection 
  3. Select the affected candidate
  4. Accept as Rehire On Old Employment
  5. Process Rehire Data Review and continue through Personal Information
  6. See that Veteran Status field shows empty, but it has a value in Employee Profile

Cause

Configuration Issue:

  • In Manage business configuration GUI, the field "Veteran Status" is mapped to a picklist with missing option ID passed from Recruiting.
  • The option id is found in a different picklist, but not in the picklist assigned in Manage Business Configuration.

Resolution

There are two possible options:

  • OPTION 1: Add the Option ID to the picklist assigned in Manage Business Configuration UI for the 'Veteran Status' field.
  • OPTION 2: In Recruiting Templates from where the Veteran Status field is mapped, update the XML with the same picklist used in Manage Business Configuration. Ensure that the picklist used IN EC and in the recruiting templates are the same.
