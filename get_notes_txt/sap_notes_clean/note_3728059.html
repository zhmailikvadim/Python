Symptom

You edit an interest rate instrument with variable interest. You replace a first interest fixing condition with a second interest fixing condition. The interest fixing conditions use different calendars. The public holidays of the second interest fixing condition are also transferred to the first interest fixing condition. The first interest fixing condition therefore includes all public holidays of the subsequent interest fixing conditions. This is incorrect because each interest fixing condition must only have the public holidays for its own specified calendar.

Environment

Treasury and Risk Management (TRM)

Reproducing the Issue

Interest movement flows added for example on December 17, 2025 for transactions created in TM_51. Posted until the last September 17, 2025 interest movement flow.
Run TBB1 for processing on December 17, 2025, but the interest movement flows have not been updated to posted on December 17, 2025.

Cause

The public holidays of the second interest fixing condition are also transferred to the first interest fixing condition. The first interest fixing condition therefore includes all public holidays of the subsequent interest fixing conditions. This is incorrect because each interest fixing condition must only have the public holidays for its own specified calendar.

Resolution

Implement below note:

3358020 - Interest fixing conditions with different calendars accumulate public holidays
