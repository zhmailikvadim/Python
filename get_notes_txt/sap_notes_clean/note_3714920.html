Symptom

You would like to know whether SAP ASE is impacted by the CVE-2026-21945.

Environment

  • SAP Adaptive Server Enterprise (ASE) 16.x
  • SAP Adaptive Server Enterprise (ASE) 16.x for Business Suite

Resolution

This vulnerability is in the Security component of Java. The impact on SAP ASE also depends on customer's application.
If customer is not loading and running untrusted code (e.g. code that comes from the Internet), then SAP ASE is not impacted by this CVE-2026-21945. 

In case customers have any concern about this, please follow KBA below to update the JRE version. 
KBA 2529144 - How to update or remove a version of JRE provided with SAP ASE - SAP for Me
