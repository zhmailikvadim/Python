Symptom

  • Expected ratings are displayed as numeric values (e.g., 1, 2) in the employee assessment step but as words (e.g., beginner, expert) in the manager assessment step.

Environment

SAP SuccessFactors Performance Management

Reproducing the Issue

  1. Login and Proxy as effected employee.
  2. Navigate to the Performance tab and access form.
  3. Go to any rating section.
  4. Observe that the expected rating is displayed as numeric values for employee.
  5. Route the form to managerial step the rating is displayed as words.

Cause

This is a Known Issue.

Resolution

The fix for this issue is planned to be deployed on 1H 2026 release (b2605).

For release timelines, please review SAP SuccessFactors Product Release & Road Map Information

Workaround

No workaround is available.
