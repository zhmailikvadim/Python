Symptom

In certain integration scenarios, it is required to determine all documents that belong to an export set.

Other Terms

PLM-41757 /DSCSAG/DOC_GET_EXPORT_SET, /DSCSAG/CL_DOC_EXPORT_SERVICE, /DSCSAG/CL_EXPORT_CONTEXT, /DSCSAG/CL_DIR_VERSION_UTIL

Reason and Prerequisites

The function is intended for scenarios where a complete export set of documents must be determined based on one or more head documents.

Solution

A new RFC-enabled function module /DSCSAG/DOC_GET_EXPORT_SET has been implemented.
