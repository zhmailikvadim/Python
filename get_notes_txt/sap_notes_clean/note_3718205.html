Symptom

The advanced pricing details are not visible on catalog items in production but they are visible in test.

Reproducing the Issue

  1. Edit requisition with catalog item
  2. Select Line Item > Actions > Edit Details
  3. Observe the additional pricing details fields are not visible

Cause

The catalog item was loaded without any of the advanced pricing detail fields. There is an out-of-the-box Visibility condition: HasPriceBasisQuantity.

Resolution

Ensure the catalog item is loaded with a PriceBasisQuantity value.

See Also

There is also an out-of-the-box Editability condition: CatalogItemEditable.

In order for these fields to be edited, the user must belong to the Purchasing Agent group.
