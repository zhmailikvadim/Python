Symptom

Is it possible to send termination emails to a user’s personal email based on the selected Event Reason?

Environment

SAP SuccessFactors Onboarding.

Resolution

The system does not support sending termination or signature-related emails conditionally based on the Reason for Termination or to an alternative (personal) email address.

This is an expected system behavior and is considered system design, not a configuration or data issue.

If a personal email address must receive the termination emails, it must be set as the primary email address before the document or signature process is initiated. Please note that this change will cause all onboarding communications for that user to be sent to the personal email address.
