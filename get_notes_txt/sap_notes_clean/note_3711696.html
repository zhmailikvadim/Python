Symptom

In the process "Setup / Replacement of Devices (MOS View)" (DE_B_IM_DEV_CHANGE), Scenario 24 with additional devices, the system does not correctly identify the meter device in step 0660 "Create device info record for RLM device".

This issue is relevant for the market roles: MOS, SUP, DSO.

Other Terms

IM4G, DE_B_IM_DEV_CHANGE, Scenario 24, device info record, RLM device, equipment, device type, device category, Class /US4G/CL_BF_DIR_MAIN, GET_DEVICE_CAT_DIR_4_RLM, GET_REGISTER_GROUP_DIR_4_RLM

Reason and Prerequisites

The error occurred because the system did not correctly identify the meter device during the device change process.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the system correctly identifies the meter device in step 0660. You need to retrigger the process. You can verify the fix by creating a device change process and checking that the system correctly identifies the meter device. Retest the process from the back end to confirm successful execution.
