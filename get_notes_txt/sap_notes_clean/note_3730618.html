Symptom

An invoice is created with a standalone special handling line item and the Tax Calculation Failed exception is firing on that line item.

Cause

The commodity code on the special handling line item does not match the commodity code listed in parameter Application.Invoicing.SpecialHandlingCommodityCode which results in the tax code lookup not triggering for that line item. 

Resolution

Ensure that the commodity code listed in parameter Application.Invoicing.SpecialHandlingCommodityCode is correctly configured so that the line item defaults that commodity code resulting in a successful tax code lookup.

Keywords

special handling line item tax calculation failed
