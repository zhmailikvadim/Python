Symptom

Can we connect a new parallel S/4 system to existing SAP Ariba Invoicing system?

Resolution

Yes, it is possible to connect multiple different S/4 HANA system to a single SAP Ariba Invoicing Tenant.

See Also

Note: The SAP Ariba Invoicing tenant created with Test, Demo and Development license can connect with SAP S/4HANA test system, SAP S/4HANA demo system or SAP S/4HANA development system, but it can only connect with one system. Therefore, only one of these system tenants should be connected. This tenant is not allowed to connect with an SAP S/4HANA production system.
