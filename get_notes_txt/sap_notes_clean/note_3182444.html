Symptom

User is not being able to create/delete tiles using the user interface(UI) in Guided Buying (GB). The get the following error message while doing so:

Cannot create or edit the tile. Make sure the tile links to a published form using the uniqueName value of the form.

Cause

 The gbdata catalogs subscriptions cannot be generated if the supplier is not activated.

Resolution

To reactivate previously Supplier Organizations:
  1. Go to Supplier Manager section > Search for the [unspecified] supplier
  2. If the supplier is not activated click on Actions > Activate

See Also

The [unspecified] supplier is responsible for the gbdata subscriptions in the realm. It needs to stay activated in order to use Guided Buying.
