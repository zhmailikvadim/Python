Symptom

  • During of after CA-GTF-CSC-EDO-FR SAP Notes implementation there are the following syntax errors in the system:
    • "EDOFRPAYMT" is not defined in the ABAP Dictionary as a table, projection view, or database view.
    • Type "EDOFRTAXATTR" is unknown.

Environment

  • SAP ERP
  • SAP S/4HANA
  • SAP Document Reporting and Compliance Service

Cause

  • Missing implementation and manual steps from SAP Note 3413099.

Resolution

  1. Refer to and implement SAP Note 3413099, and perform all required manual steps described in the note.
  2. Ensure the ABAP Dictionary objects delivered by the note are present and activated through SE11, in particular types EDOFRTAXATTR and EDOFRPAYMT.

See Also

  • Refer to: SAP Note 3413099 - Electronic Document Processing for France via Approved Platform
