Symptom

How to hide the Source Tracker tag from Recruiting page?

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

SAP SuccessFactors Recruiting Marketing

Resolution

  1. Navigate to Manage Permission Roles page
  2. Select the role that you want to hide the Source Tracker tag
  3. Remove the following three permissions:
    -Recruiting Permissions > Recruiter RMK SSO Permission
    -Recruiting Permissions > Job Marketing
    -Manage Recruiting > Manage Career Site Builder
  4. In Career Site Builder, make sure the Recruiter Sync job is scheduled. 
    If the job is not scheduled, run the job once manually
