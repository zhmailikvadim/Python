Symptom

  • User is unable to select WBS Element while creating a PR.
  • The WBS Element is not available in the dropdown when creating a PR.

Environment

SAP Ariba Buying & Invoicing

Reproducing the Issue

  1. Create PR
  2. Select Company Code
  3. Edit details of the line item
  4. Select the WBS dropdown
  5. The desired WBS Element is not available

Cause

The WBS element is mapped to a Company Code different than the user's, causing the issue.

Resolution

To resolve the issue, the following steps should be taken:
1. Navigate to Core Admin > Site Manager > Data Import/Export.
2. Import Company Code WBS Mapping.
3. Add a new entry to map the WBS Element to the User's Company Code.
