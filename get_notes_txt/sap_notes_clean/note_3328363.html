Symptom

You wonder how to automatically update the Exchange Rates in your system via web service.

Environment

SAP Business ByDesign

Resolution

The system offers the following Web Service API that enables you to connect external applications to your system to create or invalidate exchange rate information in your system

Web Service Name: ManageExchangeRateIn
Description: Manage Exchange Rates

Refer to the documentation below for further information:

Web Service APIs in SAP Business ByDesign
