Symptom

You create three Version BOMs from one request, and these BOMs have components with the same IDs. When you send an update for all of them in one request, you see the error "Enter BOM item keys to be updated" (MPE_VERSBOM/023).

Other Terms

PLMSI, Version BoM, Grouping

Solution

Implement this SAP Note to fix the error when updating grouped Version BOMs.
