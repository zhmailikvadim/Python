Symptom

  • Delete data from a table for a specific condition.
  • Delete data for a specific date from the table
  • Delete a specific data from table

Environment

SAP HANA Database

Reproducing the Issue

Identify the tables from which data needs to be removed.

Resolution

  • DELETE FROM <table_name> WHERE <condition>
  • If the WHERE clause is omitted, all records from the table will be deleted.
  • To understand how to delete records from a table using the WHERE clause  Refer to SAP HANA SQL Reference Guide for SAP HANA Platform DELETE Statement (Data Manipulation)
