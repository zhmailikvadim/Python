Symptom

Currently annotation alternativeNames fails but would be required to allow natural language search in CDS views.

Other Terms

Enterprise Search, NLS, @EnterpriseSearch.nls.variantElements.alternativeNames, @EnterpriseSearch.nls.compoundElements.alternativeNames,

@EnterpriseSearch.nls.alternativeNames,

Solution

Please install the attached correction.

In case of errors like "Annotation 'LanguageDependency.maxLength' is not allowed for array types" please follow the following steps:

1) install the Note
2) go to SE06 - System Change option and change entry for Software Component "SAP_BASIS" to modifiable
3) go to SE24, enter CL_DDLA_SYNTAX_CHECK and activate it
4) re-install the Note
5) go to SE06 - System Change option and change back the entry for Software Component "SAP_BASIS" to not modifiable
