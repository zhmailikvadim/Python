Symptom

The fields ERDAT, ERZET and ERNAM in the /PACG/ECM_ACTI table store information about the date, time, and user used during the creation of an activity. During the processing of any incoming /PACG/ECM_ACTIVITYIN IDoc, these values are overwritten with the values received in the IDoc that represents the activity change. This incorrect updating leads to inconsistencies in analyses and reports based on this table.

Other Terms

FSM, E4C, Connector, /PACG/ECM_ACTI, /PACG/ECM_ACTIVITYIN

Reason and Prerequisites

Incorrect logic in /PACG/ECM_CL_S_I_ACTSVO_E → /PACG/ECM_IF_I_ACTIVITY ~ PROCESS_SALES_ACTIVITY.

Solution

Please, implement this correction instruction.
