Symptom

 On Inspection lot result recording screen, when there are more MICs/ readings to records, there is lag while scrolling up or scrolling right to enter data.

Reason and Prerequisites

Reason:

 Conflict with the interception of the event by the scroll controllers

Prerequisites:

  • Visual Studio Code with SAM MDK project

Solution

To use the solution, please follow the steps:

  1. Download the Plugin-EditableDataTable-Release-xcframework.zip file from the attachments.
  2. Unzipping the content of .zip file will give the folder extension-EditableDataTable
  3. Replace the existing extension-EditableDataTable folder from SAM.mdkproject/extensions/ with the one retrieved from the zip.
  4. Rebuild the client using create-client.command.

The attached extension changes are only valid with SAM MDK Plugin 2410.0.2 (which uses MDK 24.7.6)
