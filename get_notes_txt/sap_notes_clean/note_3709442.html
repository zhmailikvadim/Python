Symptom

In Display Line Items in General Ledger app, only the Cost Center code is visible for line items associated with an expired Cost Center.

 

Environment

SAP S/4HANA Cloud Public edition. 

Cause

This is system standard behavior. 

Resolution

It's not possible to make an expired Cost Center Name to be displayed in Display Line Items in General Ledger app. Alternatively, extend the validity periods of required cost centers follow this SAP Note 3706959

The system uses a time-dependent approach to manage master data, applying a validity check that filters out descriptive information for expired master data as of the report's run date. This design ensures data consistency and integrity.
