Symptom

When a document is rejected by KSeF, it is not possible to resubmit (retry) the same document after making corrections. Error code: SAP_DRC_PM_INVALID_ACTION.

Environment

  • SAP ERP
  • SAP S/4HANA
  • SAP Document and Reporting Compliance service

Reproducing the Issue

  1. Access transaction code EDOC_COCKPIT.
  2. Try to resubmit a rejected eDocument.
  3. Error SAP_DRC_PM_INVALID_ACTION.

Cause

Resubmission was not yet available for SAP DRC Poland

Resolution

Implement the SAP note 3710009 - Electronic Document Processing for Poland - KSeF: Offline PDF Generation and Technical Corrections. This contains the functionality to resubmit rejected eDocuments in DRC Poland.
