Symptom

  • After upgrading from Windows XP to Windows 7, error 7008 "Advantage server file open failure" is received when trying to access data
  • Data can be accessed normally from other machines that are still running Windows XP

Environment

  • SAP Advantage Server Database 9.1
  • Windows 7

Reproducing the Issue

  1. Upgrade the machine's OS from Windows XP to Windows 7.
  2. Attempt to access data as per usual.

Cause

Advantage Server Database 9.x is not fully supported by Windows 7 which may cause this and/or other errors.

Resolution

  1. Upgrade to a newer version of Advantage Server Database - version 10.x and later are supported by Windows 7, or
  2. Use Windows XP to run Advantage Server Database

Keywords

Sybase, ADS, 11816755
