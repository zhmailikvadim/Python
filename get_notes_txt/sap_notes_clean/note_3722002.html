Symptom

There is no option to create entity as appointment from an activity plan.

Environment

  • SAP Sales Cloud and SAP Service Cloud Version 2.
  • SAP Sales Cloud Version 2.
  • SAP Service Cloud Version 2. 

Reproducing the Issue

  1. Go to the Activity Plan view.
  2. Click on + button to create New Activity Plan.
  3. In the New Activity Plan screen, under Define Activity Plan click on the drop-down of field Entity.
  4. There is only option to create entity as Visit but not an Appointment.

Cause

Currently only Visit as entity is supported using Activity Plan, not an Appointment.

Resolution

This is current system behavior . 

For an enhancement request please raise it via Influence Forum - https://influence.sap.com/sap/ino/#/campaign/3766 

Keywords

Activity Plan, Appointment, Visit, Entity, Define Activity Plan
