Symptom

Error message: "Failed to resolve navigation target '#Shell-startWDA? 

Environment

SAP Access Control 12.0

Reproducing the Issue

  1. Access Fiori
  2. Open Work Inbox Tile
  3. Save it.

Cause

Missing configuration

Resolution

Follow below steps to resolve this issue:

  1. Run transaction SM30 with view V_GRPCCUST1
  2. Search for indicator "FLP_NAVIGATION"
  3. Modify Activate flag and set it to unchecked
  4. Save Changes.
  5. Clear system and browser cache.

Issue should be resolved. This flag is for in place navigation of WD applications which is not supported for Access Controls currently. 

Keywords

Work Inbox, FLP_NAVIGATION, Activate flag, V_GRPCCUST1.
