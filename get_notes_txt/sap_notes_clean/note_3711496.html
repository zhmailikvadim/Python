Symptom

Data Integration job which uploads key figure data shows rejected values.

Rejection Description is: Not all key figures have a disaggregation mode defined

Environment

SAP Integrated Business Planning

Cause

One of the key figures mapped in the upload is missing configuration for "Disaggregation Mode"

Resolution

Review the key figure configuration of the key figures mapped in the data integration job. Check attribute as key figures.

Keywords

disaggregation error, cids, data integration jobs app, missing disaggregation mode, key figure, zmandateattribute3, rejection message
