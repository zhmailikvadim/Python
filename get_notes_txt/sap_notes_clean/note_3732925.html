Symptom

What are the supported custom fields in SAP Ariba SCIM API?

Resolution

The SAP Ariba SCIM API supports custom fields only for primitive data types String, Integer, Boolean and Date. Custom fields based on different data types, such as Ariba's Flex Master Data (FMD) classes, are not supported.

The official API documentation currently doesn't mention this limitation. This is being addressed internally.
