Symptom

Message FINS_ACDOC_POST098 raised during posting of payment program.  This error message is raised after SP is imported containing note 3325350.

The bank clearing account used in the payment program is of type 'Balance Sheet Accounts' but the house bank account used in the run requires a G/L account of type 'Cash Account' with G/L Account Subtype  'S' ( Bank SubAcccount)

Other Terms

FINS_ACDOC_POST098   bank reconciliation account bank sub account F110

Reason and Prerequisites

Missing check within payment program

As mentioned in the SAP Help Cash and Liquidity Management - Define G/L Accounts for Payment Processes - Use bank reconciliation account  all bank clearing accounts should be of type bank subaccount.

If this preferred configuration is not used the error message could be switched off to allow postings to G/L Account with account type Balance Sheet Accounts

Solution

Implement the correction instruction.

Afterwards maintain T100S and V_T100C as described in the manual instruction.
