Symptom

  • Recruiter is getting error when forwarding a candidate as applicant (Forward as Applicant):
    Error: Failed to add candidate to requisition due to system error. or The individual couldn't be added to the requisition because of a system error.
  • Employee is getting error when referring a friend to a Job Requisition:
    Error: User has already been submitted to this job.
  • Agency is getting error when submitting a candidate to a Job Requisition:
    Error: User has already been submitted to this job.

Environment

SAP SuccessFactors Recruiting Management

Cause

The error occurs because candidate already created a draft application in the target job requisition.

Resolution

The Forward as Applicant, Refer a Friend, or Agency referral will not work for candidate with existing draft application in the target job requisition.

Recruiters can use the option "Forward to Requisition" instead, and then invite the candidate to apply so they can continue their existing draft application and submit it.

Note: In case the draft application is deleted, it is still considered as draft application and will work the same as withdraw application.
