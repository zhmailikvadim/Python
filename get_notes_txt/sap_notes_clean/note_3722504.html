Symptom

You upgraded your SAP ECC solution to SAP Enhancement Package 8 (EHP8) for SAP ERP 6.0 and get the following syntax error introduced by the upgrade:

The data object "GS_ANWDG" does not have a component called "MWB_ACCESS".

Other Terms

MWB_ACCESS, DMC_ANWDG, DMIS, EHP Upgrade, ECC

Reason and Prerequisites

You upgraded your SAP ECC solution to EHP8 but get a syntax error introduced by the EHP8 upgrade.

Solution

Implement the manual correction instruction:

  1. Add a field with name MWB_ACCESS to the end of the table DMC_ANWDG. The data element should be BOOLEAN.
  2. Activate the table. 

The syntax errors should no longer occur.
