Symptom

An error was encountered when the system is trying to create a PPO while closing an account using asynchronous payment processing.

Other Terms

PPO, Account Closure, TRBK, Postprocessing, Async PAM, Payment Advice

Reason and Prerequisites

Program Error

Solution

A new PPO business process has been introduced and code has been adjusted. Please follow the steps mentioned in the manual activity to get the entries in the required tables.

You can use transaction SNOTE to implement the correction instructions in the relevant support pack.
