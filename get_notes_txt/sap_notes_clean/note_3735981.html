Symptom

When processing electronic bank statements (format MT940), the Payee Name is not fully visible in the Item Text field of the journal entry.

  • The system maps the content of :86: (remittance information) into Item Text.
  • The Payee Name is truncated.
  • Only part of the information is displayed in the journal entry line item.

Environment

SAP S/4HANA Cloud Public Edition

Cause

The item text is stored in field FEBEP-SGTXT(Item Text), which has a maximum length of 50 characters. This is standard system behavior.

Resolution

  1. Journal entry item text (FEBEP-SGTXT) is limited to 50 characters and cannot be extended via application settings. This is the designed system behavior.
  2. If additional information from the EBS file is needed for automation rather than display in the item text, consider using the Manage Processing Rules for Bank Statements app to parse, derive, and use that information in clearing logic.
