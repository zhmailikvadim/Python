Symptom

When user tries to create or edit a Template in the system, the following error is displayed - 'Changes in configurations are currently not allowed'.

Environment

SAP Sales Cloud and SAP Service Cloud Version 2

Reproducing the Issue

  1. Go to User Profile and click on Settings.
  2. Navigate to Emails and click on Templates.
  3. Create a New Template with necessary details.
  4. Click on Save.
  5. Notice that error occurs - 'Changes in configurations are currently not allowed.'

Cause

The system is marked as "Target" system in the Change Project.

Resolution

Changes cannot be made directly in the "Target" system. Changes must be made in the "Source" system and then transported to the "Target" system.

See Also


3594775 - Cannot Save Changes Due to Unchecked Change Project in SAP Sales/Service Cloud
