Symptom

Missing example implementation for customer BADI /PF1/BADI_FH_XML_ORDER_OPM.

Other Terms

Advanced bank statement forwarding,
CAMT.053.001.08, CAMT.053.001.02, CAMT.052.001.08, CAMT.052.001.02.

Solution

Example implementation for customer BADI /PF1/BADI_FH_XML_ORDER_OPM is created. It shows how to access order, ORP and entry items and enrich outgoing XML . Example provided for CAMT.053.001.02 format, but the same principle can be applied to other formats.

Please apply SAP Note or assigned Support Package.
