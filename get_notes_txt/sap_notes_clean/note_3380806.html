Symptom

The service groups in local integration scenario are not configured successfully.

There is no entry triggered in the pending task for the related local integration scenario.

Environment

ABAP Platform

S/4HANA 

Reproducing the Issue

Run Pending task in SOAMANAGER

Cause

  • The profile maintained in Provider System is not in the same version in the Profile.
  • The profile assigned into Service Definition in Local Integration Scenario is not the same one maintained in the Provider System.

Resolution

  1. Go to the Provider System in SOAMANAGER
  2. Deactivate the Provider System
  3. Click button to update to current version and then activate it again. 
  4. Check the Profile configuration.
  5. Check the profile assigned in Service Definition in Local Integration Scenario.
  6. Run pending task in SOAMANAGER again.

Keywords

Pending task, SOAMANAGER, Profile, Provider system, Web Service Configuration, Local Integration Scenario
