Symptom

During the implementation of Group Reporting ICMR, customer can only find ICMR Elimination task 2042, but for task S022 (IC elimination of Other Income and Expenses) is missing in there system.

Environment

 SAP S/4HANA Finance for Group Reporting

Reproducing the Issue

  1. Open transaction code CX8TT16.
  2. Find ICMR Elimination task S022 is missing.

Cause

Resolution

Please check the version of your S/4HANA. Only after version 2023 FPS02, the ICMR Elimination Task S022 is present. 

For further details, you can refer to SAP Help: Assign Elimination Method to Task | SAP Help Portal .
