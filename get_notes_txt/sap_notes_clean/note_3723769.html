Symptom

  • Employee cannot display the payslip in SuccessFactors for a particular period.
  • Error message: We’re having trouble finding the requested pay statement. Please contact your Payroll Admin.

Environment

  • Human Capital Management (HCM)
  • SAP R/3
  • SAP ERP Core Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA

Reproducing the Issue

  1. Login to SuccessFactors
  2. Try to display the payslip
  3. Not all payslips are displayed
  4. Error message: "We’re having trouble finding the requested pay statement. Please contact your Payroll Admin."

Cause

Employee has missing authorization for info type (e.g. IT0128)

Resolution

  1. Debugging METHOD READ_1_PERAS_1_INFOTYPE / CL_HRPAY99_IR_4_PNPCEREPORTINGCP
  2. Variable IM_INFTY and LX_MISSING_AUTH
  3. Check info type entries for that particular period for the employee

Keywords
