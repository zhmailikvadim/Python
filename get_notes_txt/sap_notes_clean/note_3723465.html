Symptom

When add an new data source reference for a user, the error message appears "Maximum number of entries for the available data sources is reached".

Environment

  • SAP BusinessObjects Business Intelligence Platform (BI) 4.3+
  • All supported operating systems

Reproducing the Issue

  1. Navigate to the Database Credentials section.
  2. Attempt to add more than two Data Source References.
  3. Observe the error preventing the addition of more than two entries.

Resolution

Before adding the data source reference, it is necessary to create it in CMC->Data Source References.

  1. Create a new Data source in CMC->Data Source References.
  2. Add this Data source to the new, the error message disappears.
