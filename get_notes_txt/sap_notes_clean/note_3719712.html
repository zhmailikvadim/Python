Symptom

After implementing SAP Note 3708125, you run report RPFIES_M347 to generate the file for transactions with third parties (Form 347) for the year 2025 in background processing. The system displays runtime error ASSIGN_TYPE_CONFLICT or GETWA_NOT_ASSIGNED depending on your release.

Other Terms

RPFIES_M347, ASSIGN_TYPE_CONFLICT, GETWA_NOT_ASSIGNED, M347, ES_M347, Modelo 347

Reason and Prerequisites

This SAP Note is relevant only for Spain.

Note 3708125 misses the logic for 2025 format in Batch Mode.

Solution

Version Date Reason
1 February 26, 2026 Pilot release.
2 March 20, 2026 Release for all customers.

Import the Support Package specified for your release or implement the attached correction instruction.

As a general rule SAP recommends that you implement the Support Package specified for your release. However, if you need to install a solution earlier, use the SAP Note Assistant to implement the correction instruction. You can find more information about the SAP Note Assistant in SAP Support Portal, under SAP Note Assistant.

To implement the solution, you must implement the correction instructions that are valid for your release.
