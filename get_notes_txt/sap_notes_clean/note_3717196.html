Symptom

When editing and saving Material Memo via MM01 and MD04, the Commit timing are different. 

Environment

SAP S/4HANA Cloud public edition. 

Cause

The timing of Commit when updating material memo depends on each application when the text is finally saved to the database.

Resolution

In the editor of material memo from MD04,  click on 'Save' button first, then click on 'Back' or 'Exit' or 'Cancel' button to permanently save the material memo. 
In Material Master, to save the material memo permanently, the Save button for the whole material master needs to be clicked on as well. 

Keywords

Text has been saved temporarily in the main memory, see long text, TD733, Material Memo, MD04, MM01, MM02, Monitor Stock / Requirements List, Create Material, MRP view 4.
