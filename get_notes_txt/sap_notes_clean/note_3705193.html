Symptom

When using an iFlow that should be a standard SAP to SAP interface, the iFlow is showing as billable when it shouldn't.

Environment

  • SAP Business Technology Platform
  • SAP Integration Suite
  • Cloud Integration

Cause

Any edit action on a standard SAP to SAP iFlow will cause this content to be marked as chargeable

Resolution

Completely redeploy the affected content. This will involve getting the content directly from the Discover section of your cloud integration / integration suite tenant.
It is vital that the content not be edited once it has been redeployed.

If the content is configurable meaning it is marked as "Configure Only", configuration changes are still possible.

it is recommended to keep backups of all deployed content.
