Symptom

You run transaction FPINTM1 (Interest Run) and the system cancels the job due to runtime error SAPSQL_PARSE_ERROR.

Other Terms

FPINTM1, performance, HANA, CX_SY_DYNAMIC_OSQL_SEMANTICS, FKK_BP_LINE_ITEMS_SEL_LOGICAL

Reason and Prerequisites

Programm error.

The problem occurs only if the performance optimization for interest run is active and you select the option 'Only Cleared Items' for interest mass run.

In S/4HANA, all optimizations for FI-CA on SAP HANA are active by default. You can deactivate individual optimizations in the Implementation Guide for Contract Accounts Receivable and Payable under Technical Settings -> Deactivate Optimizations for Database Access.

Solution

Implement the attached program corrections or import the relevant Support Package.
