Symptom

Using Transaction CKMATCON to add a large number of records to a Selection List creates a performance issue when inserting and saving the records.

Environment

  • Controlling (CO)
  • SAP R/3
  • SAP R/3 Enterprise 4.7
  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA.

Reproducing the Issue

  1. Input the transaction code CKMATCON.
  2. Create List (F5).
  3. Add selection criteria - Material and plant.

Cause

The problem is caused by the fact when using transaction code CKMATCON, certain consistency checks are being performed for every line before the selection list can be saved. Transaction CKMATCON is not designed to handle large amounts of materials.

Resolution

  1. Use transaction CKMATSEL instead to create a selection list.
  2. Transaction CKMATSEL can be run in the background. 
  3. It also allows you to upload a large list of materials.
  4. Transaction CKMATSEL uses a logical database Select which allows for a large number of materials.
