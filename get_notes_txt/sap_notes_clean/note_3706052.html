Symptom

You have defined single selections for certain characteristics. You want to use these single selections as the source selection or in the mapping condition of a load from data stream or flexible upload method. The system does not allow that.

Other Terms

SISEL

Reason and Prerequisites

If there is an identical field (infoobject) defined in the context of source data basis and also in the context of normal data basis, the Business Consolidation (BCS) considers them as different fields.

Consequently, when you are creating the selection on the source field (source selection or condition selection in mapping), it was not possible to use the single selections which are defined for the same field in BCS.

Missing functionality.

Solution

With this enhancement, the single selections are also available in the load from data stream or flexible upload method customizing. This is now possible also for source fields, which are also defined in the current BCS data basis. There are two locations where you can use these single selections:

  • On the Selection tab, when defining the Selection of Source;
  • On the Mapping tab, when creating selection for the condition operator.
