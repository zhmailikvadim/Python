Symptom

In the process "Start of Supply for Electricity (DSO View)" (DE_B_GPKE_DSO_SOS_REG), step 0700-0030 always sent the MoS type as 'CompetitiveMos', even when the assigned MoS was a default MoS.

Other Terms

GPKE, Start of Supply, Assigned MoS, MoS Type, PoD Data, Meter Operator, Default MoS, Fallback MoS, Competitive MoS, DE_B_GPKE_DSO_SOS_REG

Reason and Prerequisites

The error occurred because the logic to identify the correct MoS type was missing.

Solution

This issue is relevant for the market role: DSO.

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the system identifies the MoS type based on the assigned MoS and correctly determines whether it's default, fallback, or competitive. You need to retrigger the process. You can verify the fix by checking that the MoS type is determined correctly based on the assigned MoS.
