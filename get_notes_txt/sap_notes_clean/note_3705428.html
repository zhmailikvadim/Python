Symptom

The Maintain Business Users app does not provide an option to set centralized default values for company codes for business users.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Open Maintain Business User app 
  2. Attempt to locate an option to set centralized default values for company codes for business users.
  3. Observe that no such option exists.

Resolution

SAP Fiori does not currently support mass configuration of user default values via the Maintain Business Users app.

Individual users can maintain their own preferences under Settings → Default Values; however, these settings are part of the Fiori Launchpad personalization layer. As a result, they cannot be assigned globally or conditionally (for example, based on Country Code) at the application or view level across all users.

The Maintain Business Users app supports only user restrictions and does not provide functionality to define or control default values.
