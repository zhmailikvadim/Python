Symptom

How to create a simple output Integration Center Job.

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

    • SAP SuccessFactors HXM Suite
        ○ Integration Center

Resolution

1. Please navigate to 'Integration Center' > My Integrations

2. Click '+Create' > Scheduled Simple File Output Integration

3. Search entity that you want to export, e.g. Job Information (EmpJob), Biographical Information (PerPerson)

4. Select the fields > Click 'Select' button

5. We can check the data under 'Configure Fields':

6. We can select the Preview record number each page under the glass icon tab (Default 25 records each page).

7. We can add filter under 'Filter and Sort', time base filter document please help to check: Time-Based Filters | SAP Help Portal 


8. Please add the SFTP credential, folder and file name under 'Destination Settings'. Help document please help to check: SFTP Settings | SAP Help Portal 

9. If you want to schedule this Integration Center Job, please select the schedule type under 'Scheduling'

10. We can check the configuration details under 'Review and Run':

Note: If you have changed and saved multiple times, please confirm the if the 'Scheduled Version' is same as the 'Current Version', if not, please refer to this KBA: 2430723 - Changes not Applied to Scheduled Integrations in Integration Center - SAP for Me
