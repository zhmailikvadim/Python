Symptom

  • Time Sheet Rate Group functionality does not appear to me in the Administration Menu
  • I want to use the Time Sheet Rate Group functionality 

Cause

Time Sheet Rate Group functionality is not enabled by default in SAP Fieldglass. It is a tool that becomes available once the Rate Schedule company configuration is enabled.

Resolution

To have access to the Time Sheet Rate Group functionality, the Rate Schedule company configuration will have to be enabled. The enablement of Rate Schedule requires assistance from SAP Fieldglass Product Support.

Please create a case with SAP Fieldglass Product Support to request the enablement of Rate Schedule for your company. Please note that the request must be created by a Buyer Administrator User or PMO User.
