Symptom

  • Comments can't be added into Web Intelligence or Lumira reports, the following errors may be seen in the UI:
    Comments Service: Database initialization failed
    The database used for comments did not start correctly
  • In the APS hosting the commentary service logs, the below error will be seen:
    Failed to initailze commentary:csEX

Environment

  • SAP BusinessObjects Business Intelligence (BI) Platform 4.x / 2025
  • Windows
  • Linux / Unix 

Reproducing the Issue

  1. Try to insert / add a comment into a report
  2. Observe it fails with one of the errors mentioned in the symptoms

Cause

The commentary service can't initialize due to missing command line parameters

Resolution

  1. Navigate to CMC (Central Management Console) -> Servers -> APS hosting commentary service -> Properties
  2. In the command line, ensure the following parameter exists (replace <INSTALLDIR> with the path to this directory) :
    Windows: "-Dbusinessobjects.connectivity.directory=<INSTALLDIR>/SAP BusinessObjects Enterprise XI 4.0//dataAccess/connectionServer"
    Linux: -Dbusinessobjects.connectivity.directory=<INSTALLDIR>/sap_bobj/enterprise_xi40//dataAccess/connectionServer
  3. Restart the APS

Note: If using New BI commentary database (not auditing), ensure the JDBC driver for the database has been placed in the right location, see the documentation:
