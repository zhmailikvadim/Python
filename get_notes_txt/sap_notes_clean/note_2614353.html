Symptom

When you try to edit a Costing Sheet and the system displayed the message TK455 "Enter numeric values only" unjustifiably when you add the change to a transport.

Environment

  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA Finance
  • SAP S/4HANA

Reproducing the Issue

  1. Execute transaction KZS2.
  2. Make a change.
  3. Save in Transport request
  4. Error TK455 "Enter numeric values only" is displayed.

Cause

Customizing maintenance dialog is not up to date.

Resolution

Please execute the provided steps at the resolution of KBA 2482224. Please use as Table/View 'CO_CALC' and as function group '0COE'.

Keywords

KZS2, overheads, SPRO, transport, customising, customizing, Gemeinkosten
