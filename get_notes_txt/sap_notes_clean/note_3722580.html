Symptom

  • When opening and refreshing a Web Intelligence report in the BI Launchpad, only partial results are displayed instead of the complete dataset expected from the query. 

Environment

  • SAP BusinessObjects BI Platform 4.3
  • Web Intelligence (Webi)

Reproducing the Issue

  1. Log in to BI Launchpad
  2. Open a Web Intelligence report
  3. Refresh the report
    >> Observe that partial results are displayed instead of the full dataset

Cause

This issue typically occurs due to restrictive settings configured at multiple levels:

  • Web Intelligence Query Properties: "Max rows retrieved" or "Max retrieval time" parameters
  • Universe Parameters: "Limit size of result set to" or "Limit execution time to" settings
  • Access Restrictions: Connection limits set in security controls

Resolution

To resolve the issue, please verify/perform the following:

  1. Verify Web Intelligence Query Settings:
    1. Open the Web Intelligence Report
    2. Access the Query Panel
    3. Navigate to Query Properties
    4. Uncheck the following options:
      • Max rows Retrieved
      • Max Retrieval Time
  2. Verify Universe Settings (Information Design Tool):
    1. Open Information Design Tool (IDT)
    2. Open the Business Layer of your Universe
    3. Navigate to Query Options
    4. Uncheck the following options:
      • Limit size of Result Set to
      • Limit execution time to
  3. Verify Access Restrictions (if issue persists):
    1. Open Information Design Tool (IDT)
    2. Navigate to Security Editor
    3. Verify if any Data Security Profile(s) exist
    4. In Data Security Profile, under Control, ensure that the following properties have been unchecked:
      • Limit size of result set to
      • Limit execution time to
