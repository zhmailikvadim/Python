Symptom

In case, a split is present for child record of Family infotype (IT0021), the amount of child dependency allowance is doubled in that particular payroll period. This is incorrect.

In these cases, the system should generate only one amount corresponding to the eligibility of the child.

Other Terms

  • IT0021
  • Dependency allowances.
  • Class CL_HRPADUN_DP - Method GET_DP_STAT

Reason and Prerequisites

Correction to legal function

Solution

Relevant code changes have been made to resolve this issue. Apply the correction instructions attached with the SAP Note to obtain the changes.

Alternatively, apply the relevant Support Package.
