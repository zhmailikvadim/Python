Symptom

  • Need to check and implement the latest corrections for Migration cockpit - Stating Tables Scenario.
  • SAP Support requested to run the report CNV_NOTE_ANALYZER_MC_EXT and implement the latest corrections.

Environment

  • SAP S/4HANA

Resolution

Follow the Steps below.

  1. Implement note 3016862 – Note Analyzers with separated scenarios for ABAP-based Migration and Replication Technology.
    (Implementing the note involves Manual activity)

  2. Go to Transaction SE38
    1. Enter CNV_NOTE_ANALYZER_MC_EXT & execute. (Transaction Code: CNV_NA_MC can also be used to implement the corrections)

  3. To Implement the corrections related to Migration approach – Staging Tables

    1. Select only the check boxes related to "Migration Approach: Migrate Data Using Staging Tables” and execute. 

      (Version of the Central Note may vary depending on the versions of S/4HANA system) 

    2. Press on the "Refresh" text.
    3. Notes which needs to be implemented will be listed. Please implement all the notes. (Some of the notes involve manual activity to implement)
