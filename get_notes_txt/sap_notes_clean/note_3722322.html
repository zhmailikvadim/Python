Symptom

 In the process "Receipt of 55640 from Responsible MOS to DSO" (DE_B_MDC_55640_MOS2DSO_RECV), step 1350 encounters an error in check ID CHK_MDC_CHECK_ACT_REG_MOS_EXIST.

This issue is relevant for the market role: DSO.

Other Terms

MDC, DE_B_MDC_55640_MOS2DSO_RECV,CHK_MDC_CHECK_ACT_REG_MOS_EXIST, /US4G/CL_CHK_MDC_PROC, CHECK_MDC_ACT_REG_MOS_EXIST, MOS registration

Reason and Prerequisites

 The error occurred because of a code issue in the check logic.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the system ensures that irrelevant MOS registration processes are not linked to the MDC process. You need to reverse step 1350 and then re-run the process. You can verify the fix by checking that no active MOS registration process is linked to the MDC process when no meter location (Melo) is determined.
