Symptom

CCR iteration functionality for campaign 'Differences in Content' inconsistencies still deliverers incorrect results with error code 434 although SAP Note 3443992 is already implemented at your system.

Other Terms

/SAPAPO/CIF434, campaign, production campaign, delta report, Deltareport3, /SAPAPO/CCR, Comparison/Reconcillation of transactional data, process order, production order

Reason and Prerequisites

  • Solution delivered with SAP Note 3443992 is not complete.
  • This inconsistency is detected for process orders and also for production orders.  

Solution

Implement the correction instruction attached to this SAP Note or implement the relevant Support Package to get correct CCR results for campaigns after executing the iteration functionality.
