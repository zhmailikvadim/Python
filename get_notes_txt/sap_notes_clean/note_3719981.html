Symptom

You open an application that contains an ALV control which has been instantiated and placed into a container, but has not yet been fully initialized (method SET_TABLE_FOR_FIRST_DISPLAY has not been called).
In this state, the ALV contains no output table and no fieldcatalog, resulting in an empty gray area on the screen.

When you attempt to open the context menu (right‑click) on this incomplete ALV, the following runtime error is raised:

Category ABAP programming error
Runtime Errors RAISE_EXCEPTION
ABAP Program CL_SALV_GUI_CORE_STRUCTDESCR==CP
Application Component BC-SRV-ALV
Program error Exception condition "REFERENCE_IS_INITIAL" triggered

There might be other ocasions where this DUMP is raised, whenever the inner state of the ALV does not contain the necessary data to execute the field descriptor.

Other Terms

ALV; ABAP List Viewer; SAP List Viewer; Grid; List; Context Menu; Right-Click;

Reason and Prerequisites

Reason

The ALV framework dynamically builds the context menu based on the current table data and field catalog (for example, actions such as Show/Hide Columns, Copy, Sort, Export to Excel, etc.).

Also, if the ALV has been instantiated but not yet fully loaded, no table data or fieldcatalog is available.

In this situation, the ALV attempts to evaluate menu functions based on missing references.

This results in the exception REFERENCE_IS_INITIAL and causes the DUMP.

Prerequisites

The following Support Package level is the minimum prerequisiti:

Release Support Package
SAP_BASIS 816 SP 00
SAP_BASIS 758 SP 02

When you install an SAP Note, some prerequisites might be required. The prerequisites are SAP Notes that have already been released and that change the same context of the current one. Then, even if these prerequisites are not necessarily related to the symptom of this SAP Note, you must implement them due to technical reasons.

Solution

The ALV framework has been corrected to properly handle the scenario where the ALV control exists but has not yet been initialized with data.

From now on:

  • No runtime error will be raised when the context menu is requested for an unloaded ALV.
  • If no functions can be offered (because no data and no fieldcatalog exist), the context menu will simply not be displayed.

Delivery

As a general rule, SAP recommends that you install a solution by applying a Support Package.

However, if you need to install the solution earlier: Use the Note Assistant (SNOTE) to implement the correction instructions.
