Symptom

You have set up all access requirements for Process Data Management in SAP Signavio Process Intelligence correctly. However, when trying to access the Repository or Data Integration tab, the error message "You don't have permission to view this page" appears.

Environment

SAP Signavio Process Intelligence - Process Data Management Content

Reproducing the Issue

Set up Access Requirements for Process Data Management as outlined in the Documentation. See that despite the successful configuration of feature sets, users with an active Process Intelligence license can't access the tabs mentioned above.

Cause

Users who have been created need to be provided with access rights to the Repository folder in Collaboration Hub > Settings > Access Management first. 

Resolution

In Collaboration Hub > Settings > Access Management, select the folder "Repository". Add the users or user groups manually using the blue "+ Add Users" button. Afterwards, they will have access to the Process Data Management Content in Process Intelligence.
