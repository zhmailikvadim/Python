Symptom

  • In Stock Transfer Order (STO), stocks are marked as blocked status for inspection by customs authority.
  • Once inspected, stocks are moved to unrestricted status.
  • The blocked status in STO results in goods not being considered in Material Requirements Planning (MRP).

Environment

  • Materials Management (MM)
  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA

Reproducing the Issue

  1. Mark stocks as blocked status in a Stock Transfer Order (STO) for inspection by customs authority.
  2. Verify that the blocked stocks are not considered in Material Requirements Planning (MRP).

Cause

Resolution

  1. Change the setting via transaction code: OPPQ.
  2. Specify if the stock is relevant for planning using Customizing. Navigate to: Production > Material Requirements Planning > Planning > MRP Calculation > Stocks > Define Availability of Stock in Transfer/Blocked Stock/Restricted Stock.
  3. Refer to the following link for detailed guidance - Integration of Stocks
