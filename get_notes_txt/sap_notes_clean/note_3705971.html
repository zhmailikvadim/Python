Symptom

Build is displayed as running with "Building 100%" state for many days under Builds screen in Cloud Portal.

Environment

SAP Commerce Cloud

Cause

In some circumstances the build process may be terminated without change of the status. As status stays unchanged the build is wrongly displayed in Cloud Portal as still running although there isn't any ongoing process for it.

Resolution

  1. The problematic build will be automatically marked as failed by a dedicated task after one month.
  2. Ignore problematic build on the list and trigger a new build, as subsequent builds are not impacted by this issue.
