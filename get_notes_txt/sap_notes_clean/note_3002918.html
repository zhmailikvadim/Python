Symptom

You have multiple SIRET and then multiple B2A and to get the PAS rates for all the SIRET it's not possible to pass through each B2A/SIRET (due to time consuming and too much clicks).

Solution

One more action has been provided to the customer from the B2A Dashboard, this action will display all the rate known for the declaration found by the B2A Dashboard.

The advantage of this action is to be able for the end user to get the PAS rate (known by the SAP system) for multiple SIRET/declarations in one shot. In the result list, you will have enough details to identify the main month, SIRET, B2A ID, PAS rate, status of the update in IT3403, employee number....

This action can be run in background too.

Remarks: Don't forget to allow this action to the end user in doing the customizing in table T5FDSNB2AACT.
