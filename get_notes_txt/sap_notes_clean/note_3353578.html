Symptom

When trying to open Solution Administration (SOLADM) or Solution Documentation (SOLDOC), then a ASSERTION_FAILED dump in class CL_LMDB_DB_LC_SYSGRP is raised.

Other Terms

Logical Component Group, LCG, System Group

Reason and Prerequisites

It should exist only one system group for a LCG. The reason how to create the inconsistency that more than one exist is unkwon.

Solution

Implement the current correction instruction of this note and execute the report R_LMDB_SYSG_UNIQUE. If there are findings, send the ticket to SAP Support that a further investigation and a deletion of the inconsistency could be done.
