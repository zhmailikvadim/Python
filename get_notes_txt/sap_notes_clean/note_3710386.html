Symptom

  • In the Export Software Collection app of the parallel customizing system, the option to create a new software collection is not available.
  • '+' icon is missing in the export software collection app of the parallel customizing system.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Access the 'Export Software Collection' app in the parallel customizing system.
  2. Observe that the '+' icon is missing in the top left corner of the interface.

Cause

The Export Software Collection app will show a default software collection that can be used to transport all items that are created in the project line client of the development system.

Resolution

The key user extensibility items can be added to this software collection only. This brings the following restrictions:

  • There will be no option to create further software collections.
  • The default software collection can't be deleted.
  • It's not possible to move items to another software collection, or to merge software collections.
  • It's not possible to do a hotfix export. You have the option to export changed and deleted items, or all items altogether.
  • Only client-dependent items belonging to the same client will be visible and can be added to the software collection.
