Symptom

The note enables Serial Read of data during execution of Analytical CDS Query. The Analytical CDS Query has to be annotated wih annotation

'@analytics.settings.Enforceserialexecution: true'

Other Terms

'@analytics.settings.Enforceserialexecution: true'

Analytical Query

Reason and Prerequisites

Program Error

Solution

To solve the issue please apply the Support Package as mentioned below. In urgent cases you can use the correction instructions.

Delivery Information

  • SAP BW 8.16
    Implement Support Package 02 for SAP BW 8.16 (SAPK-81602INSAPBW) into your BW or AS ABAP system. The Support Package will be available as soon as SAP Note 3692519 with the short text "SAPBWNews NW BW 8.16 ABAP SP02", which describes this Support Package in more detail, is released for customers. 

In urgent cases you can use the correction instructions.

Before you use the correction instructions, make sure that you check SAP Note 1668882 and SAP Note 2248091 for transaction SNOTE.

This SAP Note might already be available before the Support Package is released. In this case, however, the short text still contains the term "preliminary version".
