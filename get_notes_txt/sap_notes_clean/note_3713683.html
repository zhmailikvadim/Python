Symptom

"An error occurred. Please create an incident on component CA-GTF-CSC-EDO-ES." error is triggered after running Job Spain SII: Create and Submit eDocument List in Schedule Jobs for eDocument Processing - Spain app.

Environment

SAP S/4HANA Cloud Public Edition.

Reproducing the Issue

  1. Access Schedule Jobs for eDocument Processing - Spain app
  2. Filter by Spain SII: Create and Submit eDocument List 
  3. Run
  4. error is triggered

Cause

At least one company code and a creation date must be selected. The maximum number of eDocuments in the list must also be entered respecting the allowed value for parameter "Max. Number eDocuments in List" in the list creation Job template is 10,000. 

Resolution

Review Parameters maintained accordingly to SAP Help Schedule Jobs for eDocument Spain SII.
