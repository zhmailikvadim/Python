Symptom

When creating production order with external numbering and custom fields via API_PRODUCTION_ORDER_2_SRV/A_ProductionOrder_2 POST method, error message CO/080 “External number assignment (enter valid order number)” occurs. However, creating production order with external numbering but without custom fields, or updating the external numbering production order with custom fields works fine.

Environment

SAP S/4HANA Cloud Public Edition

Cause

This is a known coding issue.

Resolution

SAP will provide a hotfix correction.
The correction will be delivered with Hotfix Collection 8. The RTH date is 16.04.2026.
Note that SAP may change the plan without any notice.

Keywords

API_PRODUCTION_ORDER_2_SRV, A_ProductionOrder_2, production order, custom fields, external numbering, CO/080
