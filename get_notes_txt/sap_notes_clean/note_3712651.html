Symptom

When renaming a segment in a DDA view, the Universe UI displays an error: "Name is not valid. Please change according to the naming guidelines." despite satisfying the naming convention guideline.

Environment

  • SAP Customer Data Platform
  • Direct Data Access

Reproducing the Issue

  1. Go to the DDA View
  2. Rename a Segment 
    • Error "Name is not valid. Please change according to the naming guidelines." is displayed.

Resolution

The error may displayed even if the name meets all the requirements stated in the naming convention guideline if the name is already in used by the segment.
