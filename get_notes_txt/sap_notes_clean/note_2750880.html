Symptom

  • Security scanning tool may report that a vulnerability warning about the DELETE method found via OPTIONS banner.
  • Not a proof of actual abuse, just an observation.
  • Description is:

"The Web server contains a flaw that may allow a remote attacker to delete arbitrary files using the HTTP method 'DELETE', resulting in a loss of integrity."

Environment

SAP PowerDesigner Portal (PD) all versions

Cause

The issue has been identified and logged under CR 818286. Its description reads, "Documentation should indicate indicate that the HTTP method is required by PD, and will cause no vulnerability."

Resolution

There is currently no target SAP PD 16.x Service Pack scheduled for this issue SAP CR 818286. It is being assigned to our Product Group.
