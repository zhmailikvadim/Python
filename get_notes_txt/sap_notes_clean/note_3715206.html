Symptom

  • The material codes are included in the parameters for the Marking step in CK40N.
  • However, when the Marking step is executed, none of the materials are processed.

Environment

  • Controlling (CO)
  • SAP R/3
  • SAP R/3 Enterprise 
  • SAP ERP
  • SAP ERP Central Component
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA

Reproducing the Issue

  1. Start the transaction code CK40N.
  2. Open the costing run.
  3. Execute the steps Selection, Costing and Analysis.
  4. In the Marking step, include the material codes in the parameters section.
  5. Execute the marking step. 
  6. None of the materials selected are processed.

Cause

  • The company code was maintained in the parameters for the Marking step.
  • However, the materials selected for the Marking step belong to a plant that is not assigned to the company code specified in the parameters.

Resolution

  • When the Marking step is executed, the system selects only the materials assigned to the company code specified in the parameters.
  • To resolve the issue, ensure that the correct company code is maintained in the parameters, or remove the company code from the parameters so that the system does not filter out materials as per to company codes.

Keywords

CK40N, Marking Step, Cost Estimate, Materials, Not Processed, Not Selected, Marking Allowance.
