Symptom

In the RECEEP transaction, the error message RECERA005 "No account determination value defined for object .......;" is raised.

  • Flexible Real Estate Management (RE-FX)

  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

In the RECEEP transaction, the error message RECERA005 is raised when the valuation is posted. 

Cause

The account determination for the valuation object is incorrectly configured.

Resolution

  1. Make sure that the account determination is assigned to the object type of the valuation object in the following path:
  2. Transaction: RECACUST -> Balance-Sheet Valuation of Contracts -> Valuation Object-Dependent Control Parameters

       3. Make sure that the account determination is assigned to the flow types in the following path:

       4. Transaction: RECACUST -> Accounting -> Automatically Generated Accounting Documents -> Accounting Determination -> Assign Account Symbol to Flow Type

Keywords

RECERA005, Valuation Object, Account Determination
