Symptom

When a Customer JIS Call request from OEMs is cancelled after being forwarded, the system does not automatically cancel the corresponding forwarded JIS calls. Additionally, the standard system does not transmit the cancellation information to Tier 2 suppliers. As a result, Tier 2 suppliers may continue to process calls that should have been cancelled, leading to inconsistencies in the supply chain.

Other Terms

S2P, Cancel forwarded JIS call, OEM, JIS, JIT, Tier 2 Supplier, Cancel

Reason and Prerequisites

Advanced Development

Solution

  1. Implement the S2C SAP Note (Pre-requisite: SAP Note 3711528).
    Ensure that the correction instructions of SAP Note 3711528 are implemented or the relevant Support Package is imported.

  2. Perform the required manual activities to create the JIT action in the system.

  3. Implement the current SAP Note: Implement the correction instructions contained in this SAP Note or import the relevant Support Package to adjust the system behavior accordingly.

After the following the above steps, the system will automatically cancel the corresponding forwarded JIS calls when a Customer JIS Call request from OEMs is cancelled. Furthermore, the cancellation information will be automatically transmitted to Tier-2 suppliers, ensuring consistency and synchronization across the supply chain.
