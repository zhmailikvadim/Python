Symptom

Message 06076 appears as an informational message rather than being triggered as an error.

Environment

  • Materials Management (MM)
  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA

Reproducing the Issue

  1. Open transaction ME38 .
  2. Enter a scheduling agreement number.
  3. Select a line item and enter a schedule quantity that exceeds the target quantity of the agreement.
  4. Observe that message 06067 “Scheduled quantity greater than target quantity” is displayed as information message .

 

 

Cause

Standard System design 
 

Resolution

In transaction ME38, message 06 067 (“Scheduled quantity greater than target quantity”) is issued as a status/information message because the message type is hard‑coded in the standard program.
 
This can be observed in the below code

SAPMM06E / MM06EFET_ETT_MENGE
FORM / ETT_MENGE

   WRITE ekpo-menge TO cmng1 UNIT ekpo-meins.
   WRITE ekpo-ktmng TO cmng2 UNIT ekpo-meins.
   MESSAGE s067 WITH cmng1 ekpo-meins
             cmng2 ekpo-meins.

The use of message 06 067  forces type “S” (Status/Information), which is displayed in green.
