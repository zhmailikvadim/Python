Symptom

  • I am creating a Job Posting
  • I cannot define the Rate Schedule on the Augment page
  • I receive an error when I try to continue to the Review and Submit page: Please select Rate Schedule

Cause

One of the following conditions is true:

  • User role permission Job Posting - View Rates During Creation is disabled
  • Job Posting Template rule Default Rate Schedule on Job Requisition Creation is set to No

Resolution

Follow the corresponding steps depending on the cause

Option 1: Enable the user role permission for the Job Posting Creator

  1. Navigate to the Admin Configuration menu (gear icon) > Click User Role
  2. Select the User Role associated to the affected user > Click Edit
  3. Scroll down to the Job Posting module > Check the box for View Rates During Creation
  4. Click Update

Result: Job Posting Creator can manually select from any available Rate Schedule

Option 2: Update the rule on the Job Posting Template

  1. Navigate to the Admin Configuration menu (gear icon) > Click Job Posting Template
  2. Click Edit > Set Default Rate Schedule on Job Requisition Creation to Yes
  3. Continue through the remaining pages > Click Update

Result: The Rate Schedule will automatically default to the Selected Rate Schedule on the Job Posting Template, and cannot be changed

Option 3: Follow both Options 1 and 2
