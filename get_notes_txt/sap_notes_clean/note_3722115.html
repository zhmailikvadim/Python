Symptom

A system error is issued when publishing a supplier level contract.

Cause

While this may not be the only reason but please check if the supplier level contract has multiple line items(pricing terms). If it has then the contract request has been corrupted. User could have imported multiple line items into a supplier level contract either via CLID(Contract Line Item Document) in the Contract Workspace or via excel sheet. Either of this should not be done for a supplier level contract and the publish action could error out with a system error. 

Resolution

Please delete the contract request(aka contract terms) and retry creating the terms so that only one line item exists in the supplier level contract request. If deleting the contract request isn't possible please create a case with SAP Support. SAP Support can deactivate the contract request and repair the Contract Workspace. Once done customers are expected to recreate the contract terms but this time kindly ensure there is only one item(pricing term) in the supplier level contract request.
