Symptom

Application error occurs during rehire in new hire data review; error displayed: "An application error occurred. Please try again later, or notify support (with the error information: errorId=XXXX timestamp=XXXX, and the version information from the current screen) if the issue keeps occurring."

Environment

SAP SuccessFactors Onboarding

Reproducing the Issue

  1. Go to the Onboarding dashboard 
  2. process the candidate activity and getting an error in NHDR 

Cause

  • Pay component details were not maintained for the rehire prior to starting the onboarding process, causing a data sync issue that led to the application error during submission in New Hire Data Review.

Resolution

  1. Verify that the required pay component details for the rehire are maintained .
  2. Restart the candidate activity in Onboarding to trigger data synchronization.
  3. Navigate to New Hire Data Review and submit the data again.
  4. Confirm that the task completes without error.
