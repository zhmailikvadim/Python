Symptom

The Event Management API endpoint is timing out for retrieving data from SAP Ariba Sourcing Events with a large number of items and/or suppliers. An error message is returned.

HTTP status: 504 Gateway Time-out

Error: The upstream server is timing out

Reproducing the Issue

  1. Run the Event Management API to get details for a Sourcing Event with large number of items and/or suppliers.
    • E.g.: GET /api/sourcing-event/v2/prod/events/{eventId}/scenarios
  2. Observe that the request fails with a timeout response stating The upstream server is timing out.

Cause

The issue you are experiencing with the API timing out is related to the large number of associated items and suppliers included in it. Although the API has a retry mechanism in place, the time required for SAP Ariba Sourcing to retrieve the data exceeds the limit, resulting in the connection closing with a timeout error message.

Resolution

Add the query parameter itemTermDataFetchMode set to BASIC to the API request to limit the payload to essential scenario data (excluding item/term metadata).

If detailed item and term information is needed, retrieve it separately using GET /api/sourcing-event/v2/prod/events/{eventId}/items.
