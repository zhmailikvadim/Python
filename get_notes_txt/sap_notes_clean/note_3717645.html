Symptom

When hiring manager tries to access BPE: To-do to perform data correction via notification link triggers "You don’t have the required permission or task assignment to access this page".

Environment

SAP SuccessFactors Onboarding

Reproducing the Issue

  1. BPE errors are generated and hiring manager receives notification that BPE: To-Do is available with processes to be corrected
  2. Hiring manager clicks on the link to access those tasks
  3. Triggers error "You don’t have the required permission or task assignment to access this page".

Cause

Missing permissions.

Resolution

The error message "You don’t have the required permission or task assignment to access this page" is caused by missing permissions in the SAP_ONB2_ErrorFlowAdmins group under Manage Onboarding Dynamic Groups.

To resolve this issue and allow users to view the data errors through email services, please ensure the relevant users are added to this group.

Important Note: After users are added to this group, they will not be able to access any previous Business Process (Data Errors) records. This is because the necessary permissions were not granted at the time those records were generated. For example, if a Business Process error occurred yesterday, and a new user is added to the group today, they will only be able to view future Business Process errors, not those from the past.
