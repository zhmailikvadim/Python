Symptom

Some columns are missing after 'change layout' in RFZALI20.

Environment

SAP_BASIS 758 SP00 or higher

Reproducing the Issue

1.Execute SE38 RFZALI20 with checkbox 'Display variance maintenance'

2.Click 'Change layout' button. On the popup, change layout, add or remove fields. Push Apply button.

3.Push button 'Save layout'. On the popup enter the name of Layout,.

4.Go back to initial screen of RFZALI20, you can see some columns are missing in the layout.

Resolution

Please apply all the missing notes list in KBA 3616874 Column Set in Change layout is not working in S/4HANA 2023

See Also

 3616874 Column Set in Change layout is not working in S/4HANA 2023

Keywords

RFZALI20, layout, change layout, Column, Columns, field, hidden fields.
