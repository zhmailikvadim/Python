Symptom

  • New functional locations and equipment are not replicating to FSM.
  • Error message: "OID is initial" observed in Replication Status.
  • Error: "ODM Mapping error" observed in Message Monitoring.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Access the app "Replicate by Object Selection".
  2. Select Object Type 'Installation Point'.
  3. Fill in the functional location.
  4. Select MDI as the target system.
  5. Click "Replicate" and then "Display Logs".
  6. Observe the error message: "OID is initial".

Cause

The replication failure occurred due to an issue with the OID retrieval from the underlying CDS views.

Resolution

  1. A fix for the underlying CDS views was developed to address the OID generation issue.
  2. The fix was delivered with 2602 HFC04 on 02/14/2026~02/15/2026.

Keywords
