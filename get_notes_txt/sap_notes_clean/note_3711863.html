Symptom

You use the Maintain HR Master Data (PA30) transaction code to create or change a record of the Employees Provident Fund (0196) infotype with begin date 01.07.2013 and end date 31.08.2013. You select Employee Statutory rate or Employer Statutory rate checkbox and select the Personal Retirement Age checkbox, set Age as 59, then save the entry. The system displays a runtime error:

CONVT_TO_NUMBER

After you've installed the SAP Note, above error disappears.

Other Terms

Employees Provident Fund, Infotype 0196, MP019640, Q0196-RETPA, P0196-RETIN, P0196-RETAG, RP_GET_EPF_RATES_MY, RPCPDFL0

Reason and Prerequisites

This is a program error. You have installed SAP Note 3661150.

The Personal Retirement Age field and Age field in the infotype 0196 are obsolete and hidden in the SAP Note 1909155 from September, 2013.

Solution

Apply the HR Support Package or implement instructions in this SAP Note.
