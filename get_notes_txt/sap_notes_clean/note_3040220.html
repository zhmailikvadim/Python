Symptom

 You are using Subcontracting with Source Location in PP/DS and the output material is the same as an input material. The subcontractor location is MRP area. You observe that in the Subcontracting PDS the location for the input component is the plant instead of subcontractor.

Other Terms

PDS, Production Data Structure, Subcontracting, MRP area

Reason and Prerequisites

Program error

Solution

Implement this note or the relevant Support Package. Afterwards retransfer affected PDS.
