Symptom

There is a requirement to assign a survey to playbook cards in opportunity applicable to all phases in the opportunity, however, the error: "Survey is not unique to the playbook" is raised when trying to add the same survey to a different phase.

Environment

  • SAP Sales Cloud Version 2

Reproducing the Issue

  1. Navigate to playbooks.
  2. Open a Playbook.
  3. Select a Sales Phase.
  4. Click "+".
  5. Select Create Activity and then select Survey.
  6. Try choosing a survey that is already selected in a previous phase. 
  7. Error: "Survey is not unique to the playbook" is thrown.

Cause

Resolution

The same survey cannot be added multiple times to the same opportunity. This is not supported from Surveys.
