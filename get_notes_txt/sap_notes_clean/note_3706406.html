Symptom

When the driver component is used, and the Enable Driver flag is not marked, any new values entered into editable fields in Fiori simulate mode in the tracking are not saving and reverting to their original values.

This note resolves the issue by ensuring that editable fields retain the modified values as expected.

Other Terms

/IRM/IPCIM, DOCUMENT_DATA_DISPLAY, Fiori Simulate, Simulate, Editable Fields, Driver Enable, Fiori.

Reason and Prerequisites

This is caused by Program error.

Solution

Please implement the correction instructions. These corrections are only valid if you have applied the following Support Packs.

Release V/4 – 2022 V2, 2023, 2023 V2 ,2023 V3 Base Package or Support Pack 1.

Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object
