Symptom

When changing default client to HTML5 the error: Layer Strategy Must be Customer appears

Reproducing the Issue

  1. Go Company Settings
  2. Select the Client tab
  3. Change the Preferred Client to HTML5
  4. The error Layer Strategy must be Customer appers

Cause

The current Business User is assigned to PDI_PARTNER_DEVELOPMENT Work Center

Resolution

  1. Go to Application and User Management work center.
  2. Select the Business Users view.
  3. Select the respective user.
  4. Click on the Edit button and select the Access Rights option.
  5. In the Work Center and View Assignment tab select the Identity Role PDI_PARTNER_DEVELOPMENT.
  6. Deactivate this assignment and then press the Save and Close button.
  7. Once this user has logged of and logged on to the system again the error will not occur anymore.
