Symptom

A skill created in SAP Build Joule Studio is working correctly, but not for all users.

Environment

Joule Studio.

Reproducing the Issue

  1. Create a skill in SAP Build Joule Studio.
  2. Test the skill with different users:
    1. Some receive the expected response
    2. Others receive incorrect responses

Cause

This takes place due to a lack of authorization from the end user to execute the Joule skill.

Resolution

Ensure all users who should access the skill have the "Execute" privilege in the settings of the environment in which the skill was deployed. Refer to step 6 of Share an Environment | SAP Help Portal.
