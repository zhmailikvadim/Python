Symptom

French translation for the button 'Not Started' in the Fiori app Manage RFQs is missing.

Environment

  • Materials Management (MM)
  • SAP Fiori 
  • SAP S/4HANA

Reproducing the Issue

  1. Log on the Fiori system in French.
  2. Open the Manage RFQs app.
  3. Input an RFQ, then observe the text 'Not Started' is not translated into French.

Cause

Resolution

  1. Log on to the system in French and go to transaction SE80.
  2. Select Class → CL_MM_PUR_RFQ_APPROVER_EXIT and open the method IF_SADL_EXIT_CALC_ELEMENT_READ~CALCULATE.
  3. Navigate to Text Elements and update the value maintained for text element 001 ('Not Started').

Alternatively, use transaction SE63:

  1. Enter Object Type: REPT.
  2. Select Class: CL_MM_PUR_RFQ_APPROVER_EXIT.
  3. Specify Source Language: English.
  4. Specify Target Language: French.
  5. Edit and update the translation maintained for 'Not Started'.

Keywords

se80, se63, text elements, translation issue, cl_mm_pur_rfq_approver_exit, IF_SADL_EXIT_CALC_ELEMENT_READ~CALCULATE
