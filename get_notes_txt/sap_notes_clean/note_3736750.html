Symptom

  • When I log in a test/ production environment, buyer names and user names are set to FN, LN.
  • I can't see names and usernames of other buyer users in the environments I am logged in. 

Cause

This is happening because the company configuration Suppress buyer user names is enabled for the environment you are logging in.

Resolution

To resolve this you would first need a Configuration Manager account:

  1. Log into your configuration manager account.
  2. Click Company Configurations module.
  3. Click Edit > Find Suppress buyer user names.
  4. Disable the configuration, provide mandatory Reason
  5. Save changes.

Once you have disabled this configuration, the names of buyer users will show up as expected.
