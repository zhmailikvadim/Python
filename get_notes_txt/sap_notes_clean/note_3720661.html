Symptom

  • You would like to know the impact of SAP Java Buildpack－Breaking changes in cf-java-logging-support announced in What's New for SAP Business Technology Platform | SAP Help Portal 
  • Whether this update has any impact on Node.js applications deployed with nodes_buldpack.

Environment

  • SAP Business Technology Platform - BTP

  • Cloud Foundry (CF)

Resolution

The Breaking changes in cf-java-logging-support is only relevant for Java applications that cover both of the following requirements:

  • The application is packaged as a WAR (check the pom.xml file) and runs on either a TomEE or Tomcat runtime (check the manifest.yml file).
    The application is using dependency cf-java-logging-support-servlet-jakarta (check the pom.xml file).

Therefore, the update has no impact on Node.js applications deployed with nodes_buldpack.
