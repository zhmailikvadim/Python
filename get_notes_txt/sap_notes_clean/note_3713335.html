Symptom

You use My Travel Requests (Version 2) application and on the detail page at the entity level, the Add button tooltip is currently displaying "Create Travel Request." Instead, it should display "Add" since the action is related to adding an entity item, not creating a travel request.

Other Terms

UITRV001, 300, PAOC_TRV_V2, REUSE, FIN_TRV_REUSE, Release 2.0

F0409A

My Travel Requests V2, MTR V2

Reason and Prerequisites

Program errors.

Solution

Apply the attached correction instructions or implement the corresponding Support Package.
