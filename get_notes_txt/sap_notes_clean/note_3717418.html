Symptom

When creating the impact list, some changes are classified incorrectly. Some minor changes do appear as critical and vice versa. The SWIFT/BIC change is basically classified as a minor change, except this SWIFT/BIC is used for a house bank account.

Other Terms

RFBVBIC2M, RFBVBIC2M_BTCH

Reason and Prerequisites

Error in program.

Solution

Apply the correction using the attached correction instructions or via the corresponding Support Package.
