Symptom

Users are unable to synchronize the mobile application and encounter the following error:

“This application cannot continue, because the ERP user is missing.”

Environment

SAP Field Service Management

Reproducing the Issue

1. Log into the mobile app.  
2. Press the synchronization button.  
3. Observe that synchronization fails with the error message: Users cannot synchronize mobile app - This application cannot continue, because the ERP user is missing.  

Cause

The issue is related to the Policy Group and Person setup:

  • Person READ permission is set to OBJECTS_IN_SAME_ORG_LEVEL (Correct).
  • User (X) is allocated to Org Level (X) (Correct).
  • Person records (via Query API) are not assigned to the Org Level (Incorrect).

Resolution

Ensure that Person records are assigned to the correct Org Level via an API request.

Steps to Resolve

  1. Identify the missing Org Level assignment for the affected Person record.
  2. Update the Person record via API request to assign the Org Level ID.

Keywords
