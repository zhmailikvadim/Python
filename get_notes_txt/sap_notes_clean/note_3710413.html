Symptom

  • Users can see and use all 'Application Job Templates' in the 'Application Jobs' app.
  • There is no option to restrict specific 'Job Templates' for individual users in App 'Application Jobs'.

Environment

SAP S/4HANA Cloud Public Edition

Resolution

To restrict Application Job Templates for specific users, the business catalog SAP_CORE_BC_APJ_TPL must be assigned to the user role to manage authorizations for the Application Jobs app.

As mentioned in Maintaining Authorizations | SAP Help Portal

You can define restrictions using the 'APPLICATION JOB TEMPLATE LAYER' restriction type, which allows you to control read or write access to templates of other users. The possible values for the layer field are Global, Shared, and Private. These settings enable you to manage who can view or edit the job templates, ensuring that only authorized users have access to specific templates.
