Symptom

You use the request printout for clearing requests and the SAP standard form VERRECHANORDNUNG. Commitment items that are longer than 13 characters are displayed incorrectly.

Other Terms

Print, request, F824, FIPOS, FIPEX

Reason and Prerequisites

There is a program error. The error occurs only in the relevant SAPscript form.

Solution

The form is attached to this SAP Note.

1. Call transaction SE38  and enter the report RSTXSCRP  ("Upload/Download SAPscript Objects").

The form required for this is contained in the attachment (VERRECHANORDNUNG.txt).

2. Download the form first and start the report.

3. In the Object Selection and Module Control  group box, select the Form radio button.

4. Enter VERRECHANORDNUNG as the object name.

5. Set the mode to IMPORT

6. In the Control Parameters for File Operation  group box, in the File Name  field, enter the path on which you saved the form.

7. All other fields and checkboxes remain empty.

8. Execute the report and select a transport request.
