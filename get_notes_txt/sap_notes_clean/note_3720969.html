Symptom

Is it possible to add another term like the Document URL?

Cause

Resolution

Currently, we can only add one Document URL term to each line item in the event and we aren't able to add more than one term of that type.

See Also

The Document URL term uses a link that participants can click to obtain documents from your site about the item with a description.

For more information about the document URL term, please refer to the following documentation: DocumentURLOutBound
