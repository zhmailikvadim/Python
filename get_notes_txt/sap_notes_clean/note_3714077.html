Symptom

You use Intrastat solution in SAP S/4HANA or SAP GTS, edition for HANA. Currently, weights below 1 gram are not supported in Intrastat.

Current correction allows support in the Intrastat solution for weights lower than 1 gram.

Other Terms

Intrastat, GTS, Tare mass, weight, unit of measure, KG, VE01, MEIS, /SAPSLL/IS_SD, /SAPSLL/IS_MM, NTWEI, /ECRS/RP_EDIT, UoM

Reason and Prerequisites

This is missing functionality.

Prerequisite note "3718398 - Support for low weights". If using SAP GTS, edition for HANA, note 3718398 must also be implemented in the GTS system.

Solution

Implement the correction instructions provided with this note or upgrade to the relevant service pack.
