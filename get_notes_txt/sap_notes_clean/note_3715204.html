Symptom

You enter a document, for example, in transaction FB01. Then use the option Document > Simulate General Ledger. There is a mandatory field in an automatically generated line item that needs to be entered manually. Accordingly, the general ledger simulation is not executed initially. The system navigates to the overview screen and highlights the affected line. In addition, the system issues the warning message "Correct the selected items". After you have corrected this line, use the option Document > Post directly. The system issues error message FINS_ACDOC_POST 000.

Note that the error message FINS_ACDOC_POST 000 can occur only in the S4CORE releases. In older releases, you should still implement the corrections to avoid problems with the trace mode of the document splitting of General Ledger Accounting (new).

Other Terms

FB01, FB60, FINS_ACDOC_POST000, FINS_ACDOC_POST 000, F5 413, F5413, SAPMF05A 0750, SAPMF05A 0700, PBO_ANZEIGE_MODIFIKATION, FAGL_SET_SIM_MODE.

Reason and Prerequisites

Program error

Solution
