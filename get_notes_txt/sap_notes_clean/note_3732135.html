Symptom

You open NWBC of MDG and try to do some navigation like, create new change request, display object or display change request. The navigation will result in a 500 Internal Server Error with following message:

Web Dynpro application does not exist (Message E SWDP_RUNTIME 403)

Environment

  • Master Data Governance on S/4 HANA on-premise 2023
  • Master Data Governance on S/4 HANA on-premise 2025

Cause

The issue happens due to error with UR patch 2603 and 2604

Resolution

Please apply note 3726992 - JSON parse issues occuring since UR c2 2603 / lightspeed 2604.

Keywords

SWDP_RUNTIME, 403, 500 Internal Server Error
