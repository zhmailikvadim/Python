Symptom

  • Unable to remove actual finish date in project plan.
  • Error message: "You cannot cancel. Superior WBS is confirmed."

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Go to app "Project Control - Enterprise Project".
  2. Select Project.
  3. Navigate to Project Planning.
  4. Press "Edit".
  5. Open all lines.
  6. Remove actual finish date on WBS element.
  7. Save.
  8. Error occurs. "You cannot cancel. Superior WBS is confirmed."

Cause

  • The issue occurs due to a logic where removing the actual finish date for a parent WBS element requires all its child WBS elements to have an actual finish date.

Resolution

  1. Maintain the actual finish date for all child WBS elements related to the parent WBS element.
  2. Once the actual finish dates for all child WBS elements are maintained, proceed to remove the actual finish date for the parent WBS element.
