Symptom

You want to disable the synchronization between SAP Cloud for Customer and Microsoft Outlook.

Environment

SAP Cloud for Customer

Resolution

  1. Go to the Administrator work center.
  2. Select the General Settings view.
  3. Select the Business Roles hyperlink.
  4. Open the Business Role ABC (ABC represents the Business Role ID).
  5. Select the Fields & Actions facet.
  6. Select the Add Row button in the Business Actions Restrictions section.
  7. Select the value selection in the field Name.
  8. Select the value GW_TASK_DOWNLOAD and set the Access Restriction to Disabled.
  9. Select the Add Row button in the Business Actions Restrictions section.
  10. Select the value selection in the field Name.
  11. Select the value GW_TASK_UPLOAD and set the Access Restriction to Disabled.
  12. Save the changes.
  13. Update the Access Rights of the assigned Business Users.

Keywords

Synchronization, Tasks, SAP Cloud For Customer, Microsoft Outlook, Disable, Sync
