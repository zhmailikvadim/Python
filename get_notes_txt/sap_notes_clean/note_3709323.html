Symptom

This SAP Note corrects following two problems:

  • The system aborts after the deletion of packages within a LE Delivery. The problem happens during the update of the Pick-Up Freight Order.
  • A Freight Unit is directly assigned to a Pick-Up Freight Order. After any stop update the location propagation is changing the location of the Pick-Up Freight Order stop. This update should not happen. In this scenario it is normal that the destination location of the Freight Unit and the Pick-Up Freight Order differs.

Other Terms

Multi Modal, ASR, Pre-Pickup Freight Order, Multi Assignment, Drayage

Solution

Implement the attached correction instructions or import the specified Support Package.
