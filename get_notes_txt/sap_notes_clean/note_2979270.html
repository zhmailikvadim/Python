Symptom

You create an article using the product master app. If you enter a reference store again and replenishment data is maintained in the global reference store, the system may issue error messages. Despite the error message, the replenishment data is already copied. If you then enter a different store, the replenishment data may be deleted.

Other Terms

F1602, Manage Product Master Data, CL_PROD_PLANT_DET_CRT_UPD, /BOBF/IF_FRW_DETERMINATION~EXECUTE

Reason and Prerequisites

Program error

Solution

Implement the attached correction instructions or import the specified support package.
