Symptom

  • In quote UI unable to download excel template for a quote table. 
  • Error popup 'Error downloading quote table excel template'.

  • Page redirects to quote list.

Environment

SAP CPQ

Reproducing the Issue

  1. Create quote table.
  2. Check allow excel upload.
  3. Define columns and set permissions for 'Column will be EDITABLE for selected groups'.
  4. Any user not having edit permission will receive error when attempts to download excel file from quote table on quote UI.
  5. Observe the error "Error downloading quote table excel template" and the page redirects to the quote list.

Cause

No editable columns for the quote table.

Resolution

  1. Go to setup.
  2. Under quotes, go to quote tables.
  3. Edit a quote table.
  4. Click on edit permissions for a column.
  5. Column is in read-only status for all Permission Groups by default. For editable and hidden status please select Permission Groups.
