Symptom

When accessing the Onboarding page within Manage Pending Hires, the following error appears: ServerErrorException class com.successfactors.coapi.converter.proxy.ResultMapProxy cannot be cast to class java.lang.String (com.successfactors.coapi.converter.proxy.ResultMapProxy is in unnamed module of loader java.net.URLClassLoader @774698ab; java.lang.String is in module java.base of loader 'bootstrap').

Environment

SAP SuccessFactors Onboarding

Reproducing the Issue

  1. Navigate to Manage Pending recruits.
  2. Select onboarding.
  3. Encounter the error.

Cause

Conflict in the the columns "location" and "pay grade" fields in the "Configure Columns for the Manage Pending recruits".

Resolution

Remove the columns "location" and "pay grade" in "Configure Columns for the Manage Pending recruits".

2541970 - Permission for Draft Records under Manage Pending Hire/Recruit - SAP for Me.
