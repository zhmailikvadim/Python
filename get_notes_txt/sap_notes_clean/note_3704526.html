Symptom

During server-side synchronization of contacts from SAP Cloud for Customer (C4C) to Microsoft Outlook, the Web Page field in Outlook is automatically populated with the SAP C4C tenant or instance URL.

Environment

SAP Cloud For Customer

Reproducing the Issue

  1. Create a new contact or open an existing contact in SAP Cloud for Customer (C4C).

  2. Synchronize the contact with Microsoft Outlook using server-side synchronization.

  3. Open the synchronized contact in Outlook.

  4. Verify that the Web Page field in Outlook is automatically populated with the SAP C4C tenant or instance URL.

Cause

This is the system's intended behavior and has been in effect since the product's initial release.

Resolution

If a change to this behavior is desired, submit a request in the Influence Portal, including the current behavior and the expected behavior, for consideration in future implementations.

Refer to the Influence Portal at the following link: Influence Portal Link.
