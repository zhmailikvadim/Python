Symptom

In the provider contract display screen, the system displays an incorrect total transaction price at the revenue account item level. This issue occurs when the system determines the total transaction price from the billing plan amounts and event 8230 adjusts the revenue account item during creation (DFKKRA_RAI_MI-CRMET = 7).

Other Terms

SAP Revenue Accounting, SAP CI, FP_VT3, TRPRC_TOTAL , RA_OI_ORIG '1', event 8230.

Reason and Prerequisites

When event 8230 adjusts the revenue account item during creation, the system must call the same event when displaying the total transaction price.

Solution

Implement the correction instructions or import the relevant Support Package.
