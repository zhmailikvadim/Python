Symptom

You run data archiving or data destruction and the system performs legal hold checks even when the ILM Business Function switch is inactive, which can reduce performance unnecessarily.

Other Terms

SARA, Legal Hold, PA3246

Reason and Prerequisites

Reason : The system does not verify that the ILM switch is active before performing the legal hold check.

Solution

After implementing this SAP Note, the system verifies that the ILM Business Function switch is active before performing legal hold checks. This improves performance during data archiving and data destruction processes.

Implement the attached correction instructions or import the specified Support Package.
