Symptom

You post a Real-Time Payment with a value for the grouping key. On the user interface, on the Overview tab of the monitor FKKEXC_MONI, the system displays the grouping key with status Open.

Other Terms

FP-RTP, PAYKY

Reason and Prerequisites

Reason: Program error.

Solution

After implementing this SAP Note, the monitor shows no status for the Real-Time Payments (RTP) grouping key because no business process exists for it.

Implement the corrections according to the attached correction instructions or import the relevant Support Package.
