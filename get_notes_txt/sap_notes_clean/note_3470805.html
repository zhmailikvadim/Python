Symptom

Unable to access "Give a Badge to.." functionality in action/global search. It shows the option; however, upon selecting it doesn’t load. 

Environment

SAP SuccessFactors Employee Central: Employee Profile

Reproducing the Issue

  1. Login to the instance and search for the option “Give a badge to..."
  2. Notice that the user details don’t load upon selecting.

Cause

Expected behavior. 

Resolution

Please input a target user after selecting "Give a badge to ... " action, then give badge dialog will popup.

Eg: Admin Center- Give a badge to ... User ABC.
