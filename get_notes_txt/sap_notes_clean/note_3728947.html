Symptom

The standard LTMC master recipe migration template does not include an option to upload Production Resource/Tool (PRT) information.

During manual recipe creation using transaction codes (C201 or C202), it is possible to add PRTs to a recipe phase.

The absence of PRT upload functionality in the LTMC template prevents migration of recipes with associated PRT data.

Environment

  • SAP ERP Production Planning (PP)
  • SAP R/3
  • SAP R/3 Enterprise
  • SAP ERP Central Component
  • SAP ERP
  • SAP Enhancement package for SAP ERP
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA, on-premise

Reproducing the Issue

  • Attempt to migrate PRT data using the recipe data migration template in LTMC (SAP S/4HANA Migration Cockpit).
  • Observe that there is no option to upload Production Resource/Tool (PRT) information.

Cause

The Production Resource/Tool (PRT) is not included in the Master Recipe template. Although routings and master recipes are technically similar, process orders do not support PRTs.
While it is technically possible to maintain PRTs in a master recipe, process orders cannot handle them. Therefore, using this combination is not recommended.

Resolution

The process order does not support PRTs. Therefore, the LTMC recipe template also does not include PRT data because it would not be handled properly. 
This is considered an unsupported combination. See SAP Note 350614 - Production resource/tool and process manufacturing
