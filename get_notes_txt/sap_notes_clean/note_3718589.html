Symptom

How to to manual delete a lot of objects in Business Objects?

Environment

  • SAP BusinessObjects Business Intelligence platform 4.x
  • SAP BusinessObjects Business Intelligence platform 2025

Resolution

Examples bellow are with users, but the process is applicable for all BI objects. 

If you want to delete all users: 

  1. Go to CMC> Users and Groups> User list
  2. Select the first user in the page
  3. Press SHIFT and select the last user in the page 
  4. All users of this page will be selected
  5. Right click in the page
  6. Click in delete and confirm 
  7. Repeat it the next page in case exists

If you want to delete just specific users: 

  1. Go to CMC> Users and Groups> User list
  2. Select the first user you want to delete
  3. Keep CTRL pressed and select all the other users you want to delete
  4. Right click in one of the selected users
  5. Click in delete and confirm 
  6. Repeat it the next page in case exists
