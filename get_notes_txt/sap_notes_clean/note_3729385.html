Symptom

When initiating onboarding for any hire type through the API with dependent information, the visibility of dependent information to external has no value (the PerPerson entity record's visibility is not set to 'y').

This leads to an issue during the personal data collection (PDC) step, where the existing dependent data fails to load.

The error message displayed is: "An application error occurred. Please try again later, or notify support (with the error information of errorId=123456789123456789132, timestamp=2026-01-01T01:12:43.193+0000, and the version information from the current screen) if the issue keeps occurring."

Environment

SAP SuccessFactors Onboarding

Reproducing the Issue

  1. Attempt to submit the Personal Data Collection (PDC) step from the candidate's perspective.
  2. Observe the application error with the provided error ID and timestamp.

Cause

This is a known issue.

Resolution

The solution to this issue is currently under investigation.
