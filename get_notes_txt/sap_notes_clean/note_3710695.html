Symptom

Localization fields for pre-commitment documents are activated in PO transactions on the account assignment screens of services items (PSTYP equal to 9), however, they shall not be active.

Other Terms

Portugal localization, Public Sector Management, Funds Management, SAPLKACB, ME21N, ME22N, ME23N, ME29N

Reason and Prerequisites

You have implemented the Public Sector for Portugal functionality.

Pre-commitment document fields on account assignment screens are incorrectly initialized.

Solution

Pre-commitment document and item fields initialization on the detected account assignment screens has been corrected.

Import the Support Package specified for your release or implement the attached correction instructions.

As a general rule, SAP recommends that you implement the Support Package specified for your release. However, if you need to install a solution earlier, use the SAP Note Assistant to implement the correction instruction. You can find more information about the SAP Note Assistant in SAP Support Portal, under SAP Note Assistant.
