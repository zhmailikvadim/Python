Symptom

When attempting to access Fiori Launchpad, user receives error net::ERR_INSECURE_RESPONSE on Android Fiori Client.

Environment

  • Fiori Client 
  • Android
  • Public CA Client certificate

Reproducing the Issue

  1. Install SAP Fiori Client on Android
  2. Attempt to connect to user's Fiori Launchpad that uses certificate from a Public CA
  3. Observe error net::ERR_INSECURE_RESPONSE
  4. You can access via mobile Chrome on same device, but receive a certificate warning

Cause

Error is caused by an untrusted certificate or incomplete certificate chain on the Android device.  Chrome will allow you to trust, but Fiori Client would not as the chain needs to be complete for Fiori Client (by design for enterprise browser).

Resolution

Error was caused by missing CA Root and/or Intermediate certificate in the Android Certificate Store.

  1. Installed required CA Root and/or Intermediate Certificates from CA Web site.  The same issue could occur for different CA certificates missing from the device trust store such as Digicert, Entrust, Verisign, or a internal certificate signing server root and intermediate certificates.
  2. Uninstalled and reinstalled Fiori Client
  3. Retested  Fiori Client and was able to access the user's Launchpad since the certificate chain is now complete
