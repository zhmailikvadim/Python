Symptom

  • You have sucessfully installed Diagnostics Agent and need to connect it to Solution Manager 7.2
  • You want to get the exact SMD Script for your Solution Manager with minumum effort

Environment

  • Diagnostics Agent
  • SAP Solution Manager 7.2

Resolution

  1. Login to Solution Manager
  2. Go to transaction sm_workcenter
  3. Go to root cause analysis
  4. Select Agent Adminstration
