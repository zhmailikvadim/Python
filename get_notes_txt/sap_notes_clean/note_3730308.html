Symptom

Other Terms

ESH_N_EXPORTDECLARATION, ESH_S_EXPORTDECLARATION, /SAPSLL/BTDTC, 3021, Proformarechnung

Reason and Prerequisites

This problem is caused by a program error.

Solution

Implement the attached correction instructions.
