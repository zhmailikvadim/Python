Symptom

You have created a contract and nomination with reference to the contract. Using nomination, LDC is created, saved and released which created an application document in final status. After this, new anlaysis version is enterd in LDC and settlement is perfomed for application document. During this time, few characteristics are not evaluated.

Other Terms

Agricultural Contract Management, Settlement, DPQS, Latest analysis

Reason and Prerequisites

Issue is happening because DPQS characteristic was getting rejected due to few conditions before evaluation of derived characteristic which was derived from the same characteristic.

Solution

Import the support pack or implement the correction instructions
