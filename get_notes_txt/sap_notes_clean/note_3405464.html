Symptom

 "Failed to update setup engine executables.  Program will exit" when installing Web Intelligence Rich Client.

Environment

SAP BusinessObjects Business Intelligence Platform 4.x.

Reproducing the Issue

1. Install Crowdstrike in Windows machine.

2. Run the WebSetup.exe installation 

3. The following error is thrown :"Failed to update setup engine executables.  Program will exit"

Cause

Crowdstrike is blocking user to write in registry files. 

Resolution

The issue is a bug and should be resolved in Falcon Sensor for Windows 6.34.14806.

Keywords

Crowdstrike, Crowdstrike, WebSetup.exe, Failed, engine, BI.
