Symptom

The plugin class for uninstalling component CYT4HANA is not complete. Some objects are missing. The plugin class for uninstallling component CYT is also affected.

Other Terms

S4, CYT4HANA,  SAPK-100AGINCYT4HANA, Capital Yield Tax for Banking for S/4HANA, UNINSTALL_PACKAGES_TEST, -99

Reason and Prerequisites

Program error

Solution

Import the relevant Support Package for your release. You can use transaction SNOTE to implement the correction instructions relevant for your release in advance.

There are no manual activities.
