Symptom

- An error occurs in the system post S/4 HANA 2023 SPS 05 Upgrade.
- Error message: "Syntax error in program 'CL_SDM_ACTION_CROSS_CLT=======CP' Dumps."
- The ABAP program "CL_SDM_MIGR_FINISHED_CROSS_CLTCP" terminates due to an inexecutable statement.
- Specific syntax error: "The method 'SHALL_UPDATE_SHOULD_EXEC' is not declared or inherited in class 'CL_SDM_ACTION_CROSS_CLT'."

Environment

SAP S/4HANA

Resolution

Please implement SAP note 3687202  .
