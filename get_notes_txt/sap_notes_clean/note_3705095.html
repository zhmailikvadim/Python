Symptom

You wish to configure the system so the first and last clocks are paired

Environment

  • Successfactors employee central
  • Time tracking

Cause

Not possible

Resolution

If a day has more than 2 time events, it is currently not possible to pair only the first and last event of the day. Each time event will be paired with the next one until a stop event is reached for the day. If a new start event is created on the same day, it will create a separate entry in the timesheet UI from that point onwards. Recorded working time within the timesheet UI will take the total time entered.

Keywords

Pairing, CICO, ignore event
