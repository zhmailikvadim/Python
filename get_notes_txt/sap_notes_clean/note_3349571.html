Symptom

For message FINS_BTTYPE_CHECKS 001 the entries of the message variables is not correct

Other Terms

Period lock, open posting period, close posting period

Fiori App Manage Posting Periods, DB table openpostgperiod

Message FINS_BTTYPE_CHECKS 001, FINS_BTTYPE_CHECKS001

Reason and Prerequisites

Program error

You are using SAP S/4HANA On Premise, the Business Function FINS_PERIOD_CTRL_BY_BTTYPE is active and the flag "Open and Close Posting Periods for CO Transactions by Company Code" has been activated in customizing.

Solution

  1. Implement the correction instructions

  2. If the long text in German is requested: Execute report NOTE_3349571 to get the long text of message FINS_BTTYPE_CHECKS 001 in the German translation
