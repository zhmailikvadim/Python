Symptom

The report RPDABWD0 issues error messages for the payroll modifiers

  • 09 (sick pay bonus calculation)
  • 39, 49 (sick pay bonus calculation public sector)

for several absences because an entry is missing for combination of processing category and the valuation rule 21 in the table T5D0A .

Solution

The missing entries are delivered.

The corrections are delivered by HR Support Package.
See SAP Note 1232082 for the connection between the HR Support Package and the technical name specified in the note under "Support Packages".

You can implement an advance correction (see the manual correction instructions).

Required actions after importing the Support Package or implementing the advance corrections

The following entries were delivered in the table/view T5D0A (Absence Distribution):
