Symptom

The technical field AccountingDocument used in Domestic Sales and Purchasing List DRC Report is not reusable with contract accounting and there is no identifier to differentiate if the document is sourced from Financial Accounting or

Contract Accounting

Other Terms

Romania, RO, VIES, 394, DRC, Statutory reports, Contract accounting, FICA, FI, BELNR_D, FIRO_DSPL_DOCUMENT_NR, FIRO_DSPL_DOCUMENT_ORIGIN, AccountingDocument, P_RO_STRPTAXITEMLOG, C_RO_STRPDOMESTICSLSPURLISTC

Reason and Prerequisites

This SAP Note is relevant only for Romania.
This SAP Note is applicable only for SAP Document and Reporting Compliance.

Prerequisite: 3713060 - RO DSPL 394: DDIC Technical change to AccountingDocument (DRC)

Solution

As a general rule, SAP recommends that you implement the Support Package specified for your release. However, if you need to install a solution earlier, use the SAP Note Assistant to implement the correction instruction. You can find more information about the SAP Note Assistant in SAP Support Portal, under SAP Note Assistant.
