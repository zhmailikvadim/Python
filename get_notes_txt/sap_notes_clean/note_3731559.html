Symptom

  • In the off-neo migration scheduling (step 4), the calendar displays a date with “48 available” but no individual time slots are shown.
  • Clicking the date or the “available” indicator does not reveal any specific hours.
  • No error message is displayed.

Environment

 SAP Cloud Integration for data services

Reproducing the Issue

  1. Open the tenant’s Administration and go to "Off‑Neo" Migrations.
  2. On the Get Started tab, do not click Continue.
  3. Navigate to the scheduling view (step 4) and select a date.
  4. Observe that only the “available” count is shown and individual migration time slots are not displayed.

Cause

Resolution

  1. In Off‑Neo Migrations, open the Get Started tab and click Continue to initiate the migration wizard.
  2. Proceed to step 4 (scheduling) and select a day in the calendar.
  3. Review the Migration Slots (hours/time) that are now displayed for the selected day.
  4. Choose an appropriate slot and proceed with the migration planning.
