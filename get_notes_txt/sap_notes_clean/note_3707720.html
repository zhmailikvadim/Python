Symptom

The following error message occurs when executing OBJVAL:

Obj. RECL/'&&&&&&&&&&&&&&&'/'&&&&' acct '&&&&&&': Account assignment not unique. Reconciliation required

Message No. FR791

Environment

  • Financial Accounting (FI)
  • SAP ERP
  • SAP ERP Central Component
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA Finance
  • SAP S/4HANA

Cause

SAP Note 3044302 is missing from the system

Resolution

Follow the recommendations in the following SAP Note and implement the corrections:

SAP Note 3044302 - OBJRECON / OBJVAL / OBJREG: Check assignment level of contract balances / Consideration of closed contracts in reconciliation 

See Also

https://me.sap.com/notes/3044302
