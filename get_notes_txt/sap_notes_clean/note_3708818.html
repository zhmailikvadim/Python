Symptom

  • Error encountered while previewing a query that includes the "Compensation Structure" field from Country specific Compensation Information object
  • Error message displayed: "We found an error while creating the query in the database. For assistance, please contact your administrator or the SAP Product Support team, if needed."

Environment

Data Modelling - Module & Entity Data Issues

Reproducing the Issue

  1. Access the instance with appropriate support access.
  2. Navigate to the Story Reports section.
  3. Create a new Story Report.
  4. Select "Compensation Information" -> Join with any Country Specific Compensation Information object (where compensation-structure identifier field is defined)
  5. Select field under the sub-object "Compensation Structure"
  6. Attempt to preview the query.
  7. Observe the error message: "We found an error while creating the query in the database."-

Cause

The Compensation Structure field is currently available only in the Parent object (Compensation Information) as part of the 2505 release.

Resolution

The Compensation Structure object has been supported only from the Parent object up to now

Reportability of this field from the Country-Specific Compensation Information object is planned as a future enhancement. However, no tentative release date has been defined yet.
