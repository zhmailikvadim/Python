Symptom

You want to implement Electronic Reporting for France via an Approved Platform and have access to the Invoice and Payment Lists on your Fiori launchpad.

This SAP Note contains the relevant objects for making the new Run Statutory Reports - France e-Reporting ( App ID: F1515_FR_4 ) available on your launchpad.

Other Terms

France e-reporting, France eReporting, Invoice Reporting, Payment Reporting, B2BI, B2C, e-Reporting, eReporting, electronic report, electronic reporting, Statutory Reporting, Invoice List, Payment List, SRF, Document and Reporting Compliance, Document Compliance, Approved Platform, PA

Reason and Prerequisites

Reason: 

This SAP Note is only relevant for France.

Prerequisite: 

You have implemented the latest version of Electronic Document Processing Framework in your system. The complete overview of the instructions for the implementation can be found in the Help Portal, under SAP ERP and SAP S/4HANA.

The prerequisites listed in the SAP Notes 3481251, 3481252 and 3616319 have to be implemented before implementing this SAP Note.

For a complete list of all corrections, please refer to the SAP Note 3713986 - Electronic Reporting for France via Approved Platform: Implementation Overview.

Solution

After implementing this SAP Note, your system will be updated with the following objects:
