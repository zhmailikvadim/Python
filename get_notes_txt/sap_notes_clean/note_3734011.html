Symptom

An error "Please provide 1 background element in the payload" occurs when adding new information in People Profile.

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

  • SAP SuccessFactors Employee Profile: Latest People Profile
  • SAP SuccessFactors Employee Profile: Background Elements

Reproducing the Issue

  1. Open People Profile for a user.
  2. Navigate to the relevant section under Background Elements.
  3. Click to add a new record and attempt to save.
  4. Observe the error: "Please provide 1 background element in the payload".

         

Cause

A field within the background element was configured as both required and hidden, creating a conflicting configuration.

Resolution

  1. Go to Manage Business Configuration.
  2. Navigate to Employee Profile > Background Elements -> relevant element.
  3. Identify the field configured as both Required and Hidden.
  4. Set either Required to "No" or Hidden to "No" (both not enabled at same time).
  5. Save the configuration.
