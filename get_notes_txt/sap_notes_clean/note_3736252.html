Symptom

Delete "Integration with SAP Business Data Cloud" Formation fails; synchronization shows errors and the operation returns :

Errors occurred while synchronizing. Choose Resynchronize to restart the synchronization process. If the problem persists, report a case in the  component and provide the formation ID "" together with the following technical details:

  • • Synchronization between SAP Datasphere System xxxx and SAP Business Data Cloud System xxxx has failed.
    – SAP Business Data Cloud System xxxx returned a technical error for assignment with ID xxxx :
    "Operation: unassign has failed. Receiver Tenant State: DELETING, Formation Tenant State: DELETING, Application Namespace: sap.bdc-cockpit. Error: {\"code\":\"99999\",\"message\":\"An error has occurred\",\"guid\":\"xxxx \",\"module\":\"catalog\",\"causes\":[{\"code\":\"99999\",\"module\":\"Error\",\"message\":\"DSP tenant xxxx  is in migration process with status: Migration Failed\"}],\"statusCode\":500}"

  • • Synchronization between SAP Business Data Cloud System xxxx and SAP Datasphere System xxxx  has failed.
    – SAP Datasphere System xxxx  returned a technical error for assignment with ID xxxx :
    "Operation: unassign has failed. Receiver Tenant State: DELETING, Formation Tenant State: DELETING, Application Namespace: sap.datasphere. Error: {\"code\":\"99999\",\"message\":\"An error has occurred\",\"guid\":\"xxxx\",\"module\":\"catalog\",\"causes\":[{\"code\":\"99999\",\"module\":\"Error\",\"message\":\"DSP tenant xxxx  is in migration process with status: Migration Failed\"}],\"statusCode\":500}"

Environment

SAP Business Data Cloud
SAP Datasphere

Reproducing the Issue

  1. SAP4ME: Open the Customer Landscape and go to the Formations area.
  2. Attempt to exclude systems from a formation of type “Integration with SAP Business Data Cloud”.
  3. Attempt to delete the formation.
  4. Observe that the actions fail and synchronization errors

Cause

A catalog migration for the SAP Datasphere catalog was initiated when the Datasphere system was added to the formation and remained pending or failed without being confirmed. The tenant stayed in a migration state (“Migration Failed”), which blocks unassign operations and prevents deleting the formation.

Resolution

  1. Log in to the relevant SAP Datasphere tenant with a Data Catalog Administrator role.
  2. Confirm and complete the catalog migration as described in the Migrating Catalog Content | SAP Help Portal .
  3. After migration completion, return to the Customer Landscape > Formations.
  4. Exclude the systems from the formation.
  5. Delete the formation.
