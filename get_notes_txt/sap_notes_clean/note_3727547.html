Symptom

An application requests an hierarchical sequential (HS) display with grid view specified in HS settings and print output specified in print settings. The grid view should be ignored and the list view output as defined in the print settings. However this does not happen; instead the print options are ignored and the data is displayed online in grid view.

Other Terms

ALV; ABAP List Viewer; SAP List Viewer; Grid; List; HS-LIST; Hierseqlist; hierseq; HSLIST; HS; printing;

Reason and Prerequisites

Reason

The preparation of the hierarchical sequential grid view does not take print settings into account.

Prerequisites

The following Support Package level is the minimum prerequisite:

Release Support Package
SAP_BASIS 816 SP 00
SAP_BASIS 758 SP 02

When you install an SAP Note, some prerequisites might be required. The prerequisites are SAP Notes that have already been released and that change the same context of the current one. Then, even if these prerequisites are not necessarily related to the symptom of this SAP Note, you must implement them due to technical reasons.

Solution

From now on, the grid view is overridden by print settings and the list view sent for output.

Delivery
