Version 2 (01.04.2026): General Availability (No changes from Pilot Version)

SAP Note 2812932 (PY-IE: SMART PAYE: PAY: 'Nil Returns') delivered a solution for customers to not send Payroll Submissions to ROS if those submissions didn't have any meaningful data (i.e. a 'Nil Return'). This SAP Note changes that solution to ignore 6 fields that could still have data, but the data in those fields is not meaningful (as the data has come from ROS in the first place).

CT1N             Income Tax Cut Off
CT1W            Income Tax Cut Off
CT2N             Income Tax Cut Off
CT2W            Income Tax Cut Off
TXCN             Income Tax Credit
TXCW            Income Tax Credit

Other Terms

SMART PAYE, HIECPAY0, Nil Return

Solution

EITHER

Apply the relevant HRSP

OR

Apply the correction instructions attached to this SAP Note, via SNOTE.
