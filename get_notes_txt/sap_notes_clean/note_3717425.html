Symptom

CIN details change log history is not displayed in report S_ALR_87012089 after upgrading the system from ECC to S/4HANA. 

Environment

  • SAP S4HANA

Reproducing the Issue

  1. Migrate vendor/customer data from ECC to S/4HANA system 
  2. Access the report S_ALR_87012089
  3. Observe that all changes related to CIN details like PAN details change log history were missing.

Cause

It is standard behavior. 

Resolution

  1. Confirm that the issue is due to standard system behavior.
  2. If new changes related to the CIN are made in BP after migration, the change log will be displayed in the report S_ALR_87012089

Keywords

PAN, J_1IMOVEND, S_ALR_87012089, CIN details, INDIA, change log, Change log history, vendor data, customer data
