Symptom

You edit a label set. On the ingredient label tab page, you navigate between the destination locations in the statements screen area. You then copy the statements from a destination location. When you save your changes, the system terminates processing with the runtime error DBSQL_DUPLICATE_KEY_ERROR, exception CX_SY_OPEN_SQL_DB.

Other Terms

Recipe Development, LBL, labeling,  LRMSL_WUI_LABEL_SAVEU06, ACTION_COPY_STATEMENTS, CL_RMSL_HDL_STATEMENTS

Reason and Prerequisites

Program error

Solution

Implement the correction instructions.
