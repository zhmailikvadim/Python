Symptom

You want to use the program RSUSR_LOAD_FROM_ARCHIVE to load archived change documents into the shadow database USARC_CDRED.

In the User Information System (transaction SUIM), you execute the function 'Change Documents' -> 'For Users' (report RSUSR100N) to evaluate changes made by users. The archive data is to be taken into account.

During the selection, the archive data is not read for the selection criteria for address data or person responsible, even though this was reloaded into the corresponding shadow table using the program RSUSR_LOAD_FROM_ARCHIVE.
