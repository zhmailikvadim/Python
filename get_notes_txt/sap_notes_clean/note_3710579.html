Symptom

You have assigned WBS element to the maintainence order. When you now display this maintainence order in CN47N, the WBS element column is empty.

But it only happens, when your maintainence order operations have different WBSelements. Like operation 10 has WBS 'Proj.1.1' and operation 20 has WBS 'Proj.2.1'. And you have run report for one project then it shows WBS field empty for other project's WBSelement.

It shows incorrect wbs element if the wbs is initial for the operation selected in the report.

Other Terms

CN47, CN48, CN47N, CN48N

Reason and Prerequisites

Progam error.

Solution

Implement the correction.
