Missing third party balance in case vendor has batch control with balance open in more than one.

Incorrect entry taken from valuation history (e.g.: MBEWH) when the target period has no movement. Up to this note they take the previous entry, with this fix it will take the subquent period which is correct according to SAP Note 193554

If you face performance issues on SPED EFD registers K200 and H005, H010, H020.

Other Terms

TDF, Brazil, CTR, EFD, K200, H005, H010, Inventory, performance, INVENTARIO, PRV_INVENTARIO

Reason and Prerequisites

  • This note is relevant only for Brazil.
  • You have applied SAP Note 2409887

Solution

  • As a general rule, SAP recommends that you install a solution by applying a Support Package. However, if you need to install a solution earlier, use the Note Assistant to implement the correction instruction. You can find more information about the Note Assistant in SAP Service Marketplace, under service.sap.com/note-assistant.
  • Note that documentation, transactions, and customizing activities are only delivered via Support Package.

Inventory views were adjusted in order to take into account all the batches of third party stock, and filtering of correct period was adjusted.

The period selection ifself helped on performance topic but also some enhancements on joins were introduced resulting in memory save speeding up processing.

After importing the correction instruction of this SAP Note, perform manual actions as described on the attached file "Manual Instructions for SAP Note 2409736.pdf"
