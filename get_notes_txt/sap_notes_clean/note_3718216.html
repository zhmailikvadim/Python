Symptom

You are executing breakpoints upload program via excel and encounter 'Invalid Material Number' error. This issue generally happens when the material number contains all numbers without any characters.

Other Terms

PMMO_PEG_BRKPT_MAINT002, PMMO_PEG_BRKPT_MAINT, RPMMO_UPLOAD_BREAKPOINTS, Breakpoints, PMMO_BREAKPOINT_UPL, Material Number, MATERIAL, NHA_MATERIAL

Reason and Prerequisites

This error is due to a program error, where conversion exits are not implemented for material fields.

Solution

 Implement the SAP Note or the service pack that contains it.
