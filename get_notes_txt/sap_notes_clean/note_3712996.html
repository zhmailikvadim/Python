Symptom

  • When attempting to edit the "Knowledge Transfer Plan Task Complete By" date, the system does not allow selecting today's date as the complete due date.
  • Error message displayed: "Date must be in the future."

Environment

SAP Successfactors Onboarding

Reproducing the Issue

  1. Navigate to the offboarding section.
  2. Attempt to update the "Complete By" date for the Knowledge Transfer Plan task to today's date.
  3. Observe that the "Add Task" button becomes inactive.
  4. Select today's date again from the calendar.
  5. Notice that the error message "Date must be in the future" appears.

Cause

  • The system configuration enforces that the "Complete By" date for the Knowledge Transfer Plan task must be set to a future date.

Resolution

  1. Confirm that the system behavior is functioning as designed, where the "Complete By" date must be a future date.
  2. If the business requirement necessitates allowing today's date as the "Complete By" date, raise an enhancement request for the product team to review and consider this change in future system updates.
  3. Refer to the following link to submit an enhancement request:

Enhancement Request Submission Portal
