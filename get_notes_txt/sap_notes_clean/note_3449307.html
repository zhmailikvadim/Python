Symptom

Unified Rendering for Web Dynpro ABAP must be updated in line with error corrections.
These corrections solve the errors described with SAP note 3511493 - Cumulative patch for UR-Lightspeed 2410

Other Terms

Unified Rendering (UR), WD ABAP, Web Dynpro, Rendering

Reason and Prerequisites

This problem is caused by a program error.

SAP Note 2187425  describes how to prepare your system for the use of TCI(Transport based Correction Instructions) and how this SAP Note can be used in the SAP Notes Assistant (transaction SNOTE).

The TCI  technology is mandatory in release SAP_BASIS 702 for updating Unified Rendering.

For more information about the UR Update Process please read SAP Note 2090746.

*Important*

If your SP level is lower than SAPKB70217, please upgrade your SP version first.

*Attention please*

Check report wdg_maintain_ur_mimes: In case the Unified Rendering version of your system is 10.30.0.228259.0 or lower ( that means your systems UR version is from 2015 ! )  we strongly recommend to implement UR 2111 via TCI note 3112302 first in order to avoid Activation Errors during the implementation of prerequisites.

Solution

The related TCI package  contains corrections for unified rendering from NW702SP17 to NW702SP26 until the following change list:

urchangelist 400007
jschangelist 352793
urversion 10.30.7.400007.0
Code line/UR Patch UnifiedR_03_REL/2410
Spam level needed check note 2187425
Deinstall Allowed
TCI Created 2024-09-18 (YYYY-MM-DD)

Also refer to the comments in SAP Note 1411980.
