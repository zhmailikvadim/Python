Symptom

Internal SAP applications require adjustments to the internal SSF API function SSFR_GET_CERTIFICATE_CHAIN.

  1. It should additionally return the certificate chain of the corresponding Replacement PSE
  2. It should issue a more informative error message when attempting to retrieve the Certificate Chain from a Verification PSE

Reason and Prerequisites

Missing functionality

Solution

Implement the assigned SNOTE Correction Instruction or SAP Support Package.
