Symptom

It was expected another value in report "Price History per Product and Supplier" (ID: SRMIV_B03_MQ0004) such as from a List Price (e.g.). 

Environment

SAP Business ByDesign

Reproducing the Issue

  1. Go to Business Analytics work center.
  2. Go to Design Reports view. 
  3. Search for SRMIV_B03_MQ0004. 
  4. Open it. 
  5. Insert the necessary info. 

For specific product, the Net Price is different from expected. 

Cause

By clicking on Add Fields button, the user can select any key figures which can be added it to the Report from the underlying Data Source. In this way, the key figure "Net Price" is taken from Data Source "Supplier Invoice Item Details", which ideally has a lot of reports on top of the same. Thereby for the report Price History per Product and Supplier it has only four standard fields namely Quantity, Net Amount, Average Price, Net value. There are underlying specific logic for each report to calculate the Key Figure values. Hence adding from the Data Source would not have the correct values.

In this example, the product's Net Price comes from a Supplier Invoice document, and not a List Price. 

Resolution

There are other reports, such as "Monitoring Invoices", which shows the detailed item information where Net Price would be useful to show.
