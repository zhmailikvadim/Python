Symptom

Create or copy a new project, insert the 'Planned Start' and 'Planned Finish', note that 'Actual Start' and 'Actual Finish' dates are greyed out and not editable. 

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Access Project Control - Enterprise Projects app.
  2. Choose one existing project.
  3. Click the "Copy" button.
  4. Enter parameters in relevant fields.
  5. Observe that 'Actual Start' and 'Actual Finish' dates are greyed out and not editable. 

Cause

This is a standard system behavior by design in SAP S/4HANA Cloud Public Edition.

Resolution

Actual project dates are derived automatically by the system when the project status changes (e.g., to In Execution or Completed).

The Actual Start Date is set automatically when you change the project status from 'Preparation' to 'In Execution' status
The Actual Finish Date is set automatically when you the project to Completed, or you technically complete all WBS elements (depending on configuration).

For data consistency and audit reasons, these fields are not open for manual editing.
