Symptom

When moving the proficiency level slider and attempting to save, screen readers announce Update Proficiency level to [value] for undefined.

Environment

SAP SuccessFactors Talent Intelligence Hub

Reproducing the Issue

  1. Log into SuccessFactors with a screen reader
  2. In the module picker, select Growth Portfolio
  3. Navigate to a skill card > Edit Proficiency
  4. Move the slider to a different proficiency level
  5. Select Save
  6. Screen reader announces Update Proficiency level to [value] for undefined

Cause

This is a known issue.

Resolution

The fix for this issue is planned to be deployed with the 1H 2026 release.

For release timelines, please review SAP SuccessFactors Product Release & Road Map Information.

Workaround

No workaround is available.
