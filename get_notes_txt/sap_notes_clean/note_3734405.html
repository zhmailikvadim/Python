Symptom

I configured a custom routing to send a document to a different SAP ERP System ID but I'm receiving the following error in the Managed Gateway for Spend&Network Transaction Tracker:

Customer XXXXXX associated connection is not configured or saved.. Suggested Action : Configure connection information in the Integration Portal »>Project -> Basic Information -> Connections or My Configurations ->Connections.

Environment

Managed Gateway for Spend&Network

Cause

When using custom routing it is necessary that the System ID is used in a Managed Gateway for Spend&Network Project for the Ariba Solution you are trying to integrate with.

Resolution

  1. Verify if the System ID you are pointing is present in a Project.
  2. If not, follow the KBA KB0400676 - How do I create a project and connection on SAP Integration Suite, managed gateway for spend management and SAP Business Network? to associate the System ID to a Project.

See Also

KBA KB0400676 - How do I create a project and connection on SAP Integration Suite, managed gateway for spend management and SAP Business Network?
