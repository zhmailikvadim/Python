Symptom

During a transport import process, the import terminates, because a dependent transport have not been imported yet

Reproducing the Issue

Try to import the transport request again

Cause

Dependent transport wasn't imported yet

Resolution

This can happen with and without Change Request Management (ChaRM).

  1. If the transport import is done via STMS, then please read the online help Defining Dependency Relationships Between Requests in the Change and Transport System. This topic belongs to the application component BC-CTS-TMS.
  2. If the transport import is triggered from ChaRM, then please read the online help Predecessor check. This is a subject of the application component SV-SMG-CM.
