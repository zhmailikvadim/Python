Symptom

This KBA describes the steps to resolve the income type code in SAP system not matching the one in RD Prep program.

Environment

  • Human Capital Management (HCM)
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP SuccessFactors Employee Central Payroll
  • SAP S/4 HANA

Resolution

401N, 4012 or 402I is the income type value which is default on RD prep program. SAP text file uses code "1" for income type 1 or 401N.

When importing the ITF1A text file to RD Prep program for the first time, change the income type value from 401N to 1. This is a one time change on the first time using RD prep program.
