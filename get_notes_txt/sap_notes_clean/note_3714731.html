Symptom

When executing the task list SAP_BW4_AFTER_UPDATE or SAP_BW4_AFTER_UPGRADE, the task CL_RSTCO_200_CONF_SW_UPDATE might fail with a message like:

BW/4HANA: Scope mismatch. Installed Sciope: 1. Needed Scope: 16



Other Terms

SAP_BW4_AFTER_UPGRADE, SAP_BW4_AFTER_UPDATE, Object Scope, Operating Scope, RSADMINA

Reason and Prerequisites

The reason for this is that the scope is reduced, if a content object from a higher scope cannot be activated successfully.

Solution

To solve the issue please apply the Support Package as mentioned below. In urgent cases you can use the correction instructions.

To repair this problem, you need to identify the object from the application logs (log object RSTCO_ACT) and correct the activation error manually.

Afterwards you need to run the Report RSTCO_200_REPAIR_RSADMINASCOPE contained in this note to repair the operation scope - it will be set to 16 again.
