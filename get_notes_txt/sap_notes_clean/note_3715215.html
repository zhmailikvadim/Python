If you use ping to test the Web service connections to the SHI (HR_DE_GKV_<company number employer>) and to the DSRV (HR_DE_DSRV_<company number employer>) in SOAMANAGER, the Web service ping fails with the following HTTP return codes:
  • Web service ping failed (RC=405). Service Ping Error: Method Not Allowed



  • Web service ping failed (RC=500). Service Ping Error: Internal Server Error

Environment

  • Human Capital Management (HCM)
  • SAP R/3
  • SAP ERP Core Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA

Cause

These HTTP responses occur because GKV and DSRV expect a complete Web service call and not a simple ping.

Solution

Only test these connections using the report Test Report for Communication with the GKV and DSRV (RPUSVHD1), since this tests the connection as if it were an actual data transfer.
