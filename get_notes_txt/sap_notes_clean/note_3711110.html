Symptom

Function module RHXPROVIDE_PERIODS creates unnecessary split with high dates if input table has record with end date equal high date.

Example Table with one entry:

01.01.2015-31.12.9999

Result with unnecessary split:

01.01.2015-30.12.9999

31.12.9999-31.12.9999

Other Terms

 RHXPROVIDE_PERIODS

Reason and Prerequisites

Side effect of note 3703906.

Solution

Apply attached correction instruction or import corresponding support package.
