Symptom

In S/4HANA Group Reporting, the Display Group Journal Entries (DGJE) Fiori application does not render and waits indefinitely for a backend response.

Other Terms

CL_FINCS_JRNLENTR_L_01_DPC, CL_FINCS_JRNLENTR_L_01_MPC, metadata, Display Group Journal Entry, F2573.

Reason and Prerequisites

Program error.

Solution

Please follow correction instructions to implement this SAP Note.

Caution: The issue may not be resolved immediately. If it persists, perform manual cache cleanup as follows:

  • after implementation, clear the gateway cache via /n/IWFND/CACHE_CLEANUP;
  • clear the backend cache via /n/IWBEP/CACHE_CLEANUP;
  • clear your web browser cache.
