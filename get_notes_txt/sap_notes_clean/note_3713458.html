Symptom

You use Smart Chart 2.0 to manage physician orders.

For a patient with two cases that overlap in time, create an old physician order, i.e. in the old format, within this overlap period.
If you now open Smart Chart and try to display these physician orders, you notice that
that the system only displays the old physician order of the opened case.
In "Total", on the other hand, the system displays both physician orders.

Other Terms

SC2.0, physician order, PO, case

Reason and Prerequisites

Program error

Solution

Smart Chart now displays all physician orders of the selected period, regardless of the case assignment.

Import the specified Support Packages to solve the problem.
You can implement the corrections in advance.
