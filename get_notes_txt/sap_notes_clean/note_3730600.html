Symptom

In the process "Creation of a 100% Supplier Assignment to a Feed-In Installation (DSO View)" (DE_B_MPES_OBS_FOR_DS_OBLIGATION_DSO), the future supplier was only saved in the process data but not in the process step data in step 0300.


This issue is relevant for the market role: DSO.

Other Terms

MPES, DE_B_MPES_OBS_FOR_DS_OBLIGATION_DSO, feed-in installation, supplier assignment, process data, process step data, user decision

Reason and Prerequisites

The error occurred because the solution of the user decision was only saved in the process data.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the system saves the new supplier in both the process data and the process step data. You can verify the fix by retesting the scenario and checking that the future supplier is correctly saved in both the process data and the process step data.
