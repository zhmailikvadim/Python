Symptom

Try to create a legacy asset using AS91 and system promptly raises error AC661.

Environment

  • FIN Asset Accounting​
  • SAP S/4HANA ​
  • SAP S/4HANA Finance​
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Open transaction AS91.
  2. Enter Asset Class and Company Code.
  3. Press enter.

Cause

It is not allowed to have two depreciation areas with same currency type + same value takeover area (OABT).

Resolution

For depreciation areas with same currency type + same value takeover area, please either delete the duplicated area or change value takeover area accordingly to ensure there is no such depreciation areas with same currency type + same value takeover area.

You can deactivate one of them in OAYZ as well.
