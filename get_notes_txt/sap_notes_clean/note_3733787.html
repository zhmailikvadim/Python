Symptom

When trying to add the embedded report views in any tab in the Account details page, the adaptation mode is exited successfully but nothing is shown.

Environment

SAP Cloud for Customer

Reproducing the Issue

  1. Go to Customer work center.
  2. Select Account work center view.
  3. Select any Account and start adaptation mode.
  4. Go to any tab (for example: Sales Insights).
  5. Add the embedded report views.

The adaptation mode is exited successfully but nothing is shown there.

Cause

The user hasn't maintained access restrictions for the report views.

Resolution

Please ensure that restrictions are maintained in both reports and the user can view the embedded report.

Keywords

Embedded report, Report view, Account, Not shown, No data
