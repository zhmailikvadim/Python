Symptom

  • Even after employees complete the required service period, the system calculates gratuity based on 21 days instead of applying the Qatar-specific gratuity rule.
  • During payroll processing for Qatar, an issue was observed in the End of Service (EOS) calculation.

Environment

  • Human Capital Management (HCM)
  • SAP R/3
  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • Qatar Payroll

Reproducing the Issue

  1. Go to pc00_mqa_calc t-code.
  2. Run the payroll for the issue employee.
  3. Check the value of the /649 , /650 wage type from the RT table.

Cause

QAEPD feature value has not been maintained correctly.

Resolution

  1. Go to PE03 t-code.
  2. Pass the feature name as : QAEPD.
  3. Click on the change button.
  4. Maintain the value correctly for the applicable company code.
