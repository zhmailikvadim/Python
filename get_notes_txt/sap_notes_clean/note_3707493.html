Symptom

  • When a universe description is changed in Central Management Console , the Last Published date in Information Design Tool also updates, even though the universe wasn’t actually published.

Other Terms

Universe, Last Published Date, IDT, CMC, description, update

Reason and Prerequisites

Environment:

  • SAP BusinessObjects BI Platform BI 4.3 Support Package 04 Patch 1100
  • Information Design Tool

Steps to Reproduce:

  1. Log in to Information Design Tool (IDT) and connect to the repository resources.
  2. Under Repository Resources, navigate to any UNX universe (for example, eFashion.unx). A popup displays details such as Name, Path, Revision, Last Publisher, and Description.
  3. Log in to the Central Management Console (CMC) using another user account that has access to the universe.
  4. Navigate to Universe, update the description of the universe (eFashion.unx), then save and close.
  5. Return to Information Design Tool and hover over the universe.
     >> The Last Publisher date has changed even though the universe was not published.
