Symptom

Deleted Higher Duty/Temporary Internal Assignment profiles are appearing as inactive profiles in the inactive search after being completely deleted.

Environment

  • SAP SuccessFactors Employee Central
  • Higher Duty Assignment

Reproducing the Issue

  1. Search for a user on Higher Duty Assignment
  2. Delete one of the past terminated HDA profiles clicking on the Trash icon from Take Action > Manage Higher Duty or Temporary Assignment 
  3. Now Go To People Search, check "Include inactive users in search" > Type the Emp ID in the search bar
    3. Observe multiple inactive empty profiles are appearing in the drop-down and some are redundant leading to confusion and ambiguous user experience.

Cause

This is a defect in 2511 release and only specific to latest people profile - PPX, the deleted Higher Duty Assignment profile does now show up in PP3 which is expected. 

Resolution

  • Engineering has created a code correction - ECT-268222 to fix the behavior and the fix is tentatively targeted for 1H 2026 release however the dates are yet to be confirmed. The KBA will be updated timely with the relevant information.
  • To receive updates on this issue, please bookmark the related KBA by clicking the star icon.
  • For more information about receiving notifications for new or updated SAP Notes/KBAs, refer to KBA 2171560 - How to be notified of new or updated SAP Notes or KBAs.
