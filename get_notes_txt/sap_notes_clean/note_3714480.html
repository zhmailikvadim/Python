Symptom

  • Error occurs when generating B2A entries for HI Union in report RPLSDGJ0.
  • The issue is observed when selecting "Health Insurance Only" under Submission Authority: HI Union.
  • No error occurs when "Health Insurance or Pension" is selected under Submission Authority: Pension Office.

Environment

  • Human Capital Management (HCM)
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP SuccessFactors Employee Central Payroll
  • SAP S/4HANA

Reproducing the Issue

  1. Execute program RPLSDGJ0 with the following parameters:
    Submission Authority: HI Union
    Insurance type: Health Insurance Only
  2. Execute the report and click 'Generate XML' button
  3. Observe the result: Duplicated B2A entry 22HUKPFD000000000000000XXXX.

Cause

  • Different area type configuration leads to discrepancies in B2A entry generation.

Resolution

  1. When generating XML files for Submission Authority "HI Union", the following entry will be created in B2A manager.
    Area: HU (Health Insurance Union)
    Document class: KPFD (CSV Summary)
  2. When generating entries for Submission Authority "Pension Office", the following entry will be created in B2A manager.
    Area: SI (Social Insurance)
    Document class: GEPE (Geppen)
  3. Please select correct "Area" and "Document class" to check if there is duplicated entry.

    See Also

    Go to SE16->table PB2AMGR to check if there is the existing B2A entry.
