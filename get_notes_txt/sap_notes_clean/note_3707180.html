Symptom

  • The compliance forms for Spain, ES_MOD_145 / Spain Modelo 145 appear twice in the onboarding documents section.
  • Duplicate form presented for signature

Environment

SAP SuccessFactors Onboarding

Cause

  • The duplication is expected behavior introduced in the 2H 2025 release. The enhancement supports the generation of two versions of the Spain modelo 145 compliance form: one for the employee and one for the employer. The footer text differentiates the two copies.
    • Page 1: “Ejemplar para el Perceptor”
    • Page 2: “Ejemplar para la Empresa”

Resolution

  1. This change was introduced in 2H 2025 release as part of OBX-41360 to enhance the Spain Modelo 145 form generation logic to include and support the second page (“Ejemplar para la Empresa”) in addition to the existing recipient copy.

  2. Both pages display identical content aside from header/footer designation
