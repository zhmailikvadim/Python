Symptom

The Import of performance parameters from another Data Provider does not work properly
when entering a format like hh:mm:ss

Environment

SAP Sports One

Cause

In SAP Sports One you can only upload and save purely numerical values for performance parameter (example 5,67)
Therefore a format like hh:mm:ss is not possible.
Excel is saving the values as Fraction of a Day (example 0,0XXX values)

Resolution

If you want to upload data from another data provider (example Firstbeat) then you have to
convert the data in the excel file to minutes or seconds.

Alternatively there is also the option to use the Upload of Tracking Data functionality in SAP Sports One.
SAP offers here the possibility to format the values. It is then not necessary to do this in the Excel File.
