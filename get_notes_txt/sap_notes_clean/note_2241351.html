msg.RDF workplace

Reason and Prerequisites

Program error

Solution

History:

Version 1 from 11/9/2015: New　

Implement the source code corrections. The problem is solved with Support Package SAPK-70010INMSG.　

Solution:

The visibility of the method CHECK_ACCU_CURR() and the class attribute MR_PARENT_MESSAGE1 of the class /MSG/CL_VRP_LIB_RETROCESSION is changed from "Private" to "Protected".

Technical effects:

There are none.

Manual changes:

You do not have to perform any manual tasks.
