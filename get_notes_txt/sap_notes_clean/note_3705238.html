Symptom

The service call code was added to the assignment screen configuration. However, the service call code is not displayed in the dispatching board assignment.

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental." 

Environment

  • SAP Field Service Management (FSM)

Reproducing the Issue

  1. Log in to the Shell
  2. Navigate to the dispatching board.
  3. Actual Behavior:
    Service Call code is not visible in the dispatching board assignment.
  4. Expected Behavior:
    Service Call code should be visible in the dispatching board assignment.

Cause

  • 'Planner Assignment Web' Configuration for 'serviceAssignment.serviceCall.code' was set to `visibility = hidden`, resulting in the Service Call code being invisible in the dispatching board assignment.

Resolution

  1. Access the 'Planner Assignment Web' Configuration section.
  2. Locate the configuration for `serviceAssignment.serviceCall.code`.
  3. Update the visibility setting from `hidden` to `visible`.
  4. Verify that the Service Call code is now displayed in the dispatching board assignment.
