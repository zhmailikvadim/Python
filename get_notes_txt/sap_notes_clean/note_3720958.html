Symptom

This SAP Note contains extension of functionality of identification documents processing. Further SAP Notes might be required for the consistency of identification documents processing after implementation of this SAP Note.

 

Key aspects of the changed system behavior:

  • View Document Catalog Types was created to maintenance of different groups of identification documents
  • View Codes of Identification Documents (IT0290) was created for maintaining the characteristics of identification documents by catalogs.

Other Terms

HRPAD99RU_DOCS, V_T7RU0290IDCODE, V_T77PAD99RU_DC, S_HRI_55003976, T7RU0290CODE, T77PAD99RU_DC_T, T77PAD99RU_DC, PRUDOCDSC, HRPAD99RU_DOC_CATALOGUE, HRPAD99RU_DOC_CAT_DESCR, HRPAD99RU_ID_CODE

Reason and Prerequisites

  • This SAP Note is relevant only for CIS countries/regions.
  • Some objects that required for future localization development must be shipped in advance.
  • You have implemented SAP Note 3720959 with prerequisite objects (UDO).

Solution

The implementation of this SAP Note can impact your existing functionality. Implement this SAP Note only if other SAP Note required it.

As a general rule, SAP recommends that you implement the Support Package specified for your release. However, if you need to install a solution earlier, use the SAP Note Assistant to implement manual pre-implementation and automatic correction instructions. You can find more information about the SAP Note Assistant on the SAP Support Portal, under SAP Note Assistant.
