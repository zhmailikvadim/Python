Symptom

When  execute "Actual assessment (T-cd:FAGLGA15)", an event has occurred in which not all of the object data in the profit center group specified in the sender condition is relevant for assessment.

Environment

  • SAP S/4HANA Finance
  • SAP S/4HANA

Reproducing the Issue

Execute FAGLGA15, specify "profit center group" in sender group.

Cause

Program error.

Resolution

Implement below 2 notes:

2886473 - GL Allocations: Preliminary Profit Center check I.

3286807 - Allocations - Issues with invalid/locked profit centers

See Also

2886473 - GL Allocations: Preliminary Profit Center check I.

3286807 - Allocations - Issues with invalid/locked profit centers

Keywords

FAGLGA15, profit center group
