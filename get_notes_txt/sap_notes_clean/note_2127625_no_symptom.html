What is Fixed in 6.1.17?

The following issues found earlier in the application are now fixed:

  1. In Web Intelligence documents with OpenDocument URLs, if the OpenDoc URL points to the latest instance of a Web Intelligence document, the target document does not display the latest instance when seen on device.
  2. In cross tables, if a background image is set for cells in a vertical column, the image appears curtailed in the cells when seen on the device
  3. In BW date and date-interval prompts, the date field collapses in the prompt pop up when users make value selections. Hence, it is impossible to specify values for date
  4. When refreshing Web Intelligence reports, even after specifying prompt values in the pop up, the pop up does not disappear
