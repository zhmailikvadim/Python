Symptom

When trying to access SSCUI/Configuration Activity "Activate Omnichannel Sales Promotions for Sales Orders" ID 107198, the following error messages are raised: 

  • No authorization to start the UI for transaction S_ER1_70001729

Environment

  • SAP Central Business Configuration (CBC)
  • SAP S/4HANA Cloud Public edition

Reproducing the Issue

  1. In the CBC workspace, click the tab "Configuration Activities".
  2. Search for the Configuration Activity ID 107198.
  3. Click on the entry.
  4. Click the button "Go to Activity".
  5. The error message is raised.

Cause

  • Missing Business Catalog.

Resolution

  1. Add the SAP_CA_BC_IC_LND_RFM_OPPS_PC business catalog to the BPC_EXPERT business role (or to the similar role in use).
  2. Grant "Write Access" to "Unrestricted" for the BPC_EXPERT business role (or for the similar role in use) to ensure editing rights.
  3. If issue persists, please create a support case under component LO-RFM-OPP-2CL and provide the affected CBC user, as described in KBA 2838282.
