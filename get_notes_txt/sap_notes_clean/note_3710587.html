Symptom

You have enabled the connection to Cloud ALM and during the use case check, you received a warning message recommending: CN=DigiCert RSA4096 Root G5, O="DigiCert, Inc.".

Other Terms

/SDF/ALM_SETUP, Check use-cases.

Reason and Prerequisites

SAP is transitioning to the "DigiCert TLS RSA4096 Root G5" certificate. During the transitional period, the note will support verification of both the old and new certificates.

Solution

To create the text objects, you need to run report /SDF/NOTE_3710503 after implementing the correction instruction attached to this Note.
