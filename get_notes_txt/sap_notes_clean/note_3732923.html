Symptom

Creation or modification of a text data source is necessary for the ad hoc query.

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental." 

Environment

  • SAP R/3
  • SAP ERP Core Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA

Resolution

With the help of report RS_TEST_IDENTIFY_TEXT, it is possible to identify what is being linked as a text data source to any table and it's contents.

If the text for a field cannot be determined automatically or it has to be changed, changing the DDIC settings is possible so that the algorithm can be used successfully, or defining an exception for this field explicitly. The exception can be defined without further modification.
SAP defines an exception by maintaining the TEXC_FLD, TEXC_DE and TEXC_DOM tables. Using a Customizing function, which contains a detailed description located in the SAP Implementation Guide under 'Basis' -> 'SAP Query' -> 'Exceptions for text procurement'. As of Release 6.20, use the following path: 'SAP Web Application Server' -> 'SAP Query' -> 'Exceptions for text procurement'.
After defining an exception, it can be tested with report RS_TEST_IDENTIFY_TEXT.
