Symptom

When doing web-service and process step field mapping using /IDXPF/MAPPING, if the web-service component type is table type not structure, program dumped if press 'F4' help for a web-service field name.

Other Terms

/IDXPF/MAPPING, set_f4_help, /IDXPF/LFG_MAPPING_CONFIGF00,

Reason and Prerequisites

Refer to symptom.

Solution

Implement this SAP Note or install further Support Packages.

Note:

This SAP Note does not directly address a customer incident and, therefore, does not have to be implemented.

Changes made to the associated code and DDIC may impact your existing implementations. You can defer installing this SAP Note if it is not required for your implementation, and instead wait until the next service pack is available that will implicitly include the associated corrections.

If you have installed an associated add-on in your system, such as IDEX-AT or IDEX-UK, and the add-on may also be impacted by this SAP Note, we strongly recommend that you contact your SAP IDEX add-on maintenance team to find out whether installing this SAP Note will cause any disruptions.
