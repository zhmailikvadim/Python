Symptom

  • Unable to list business units (BU) in the Users workspace.
  • Unbale to create a new user and assign read only BU nor full access BU 
  • Without selecting BU/PU for admin users in the Users workspace, the data access cannot be limited.
  • Unable to run processing logic (PL) for any processing units (PU) as the PU's are not listed.
  • Unable to set BU for organizational data nor to rule elements.

Environment

  • SAP Sales Performance Management
  • Incentive Management on Hyperscalers

Reproducing the Issue

  1. Open Users workspace in Incentive Management (IM).
  2. Edit a selected user.
  3. Attempt to list the BU under either read-only business units or full access business units.
  4. Note that no BU's are listed.

Cause

The Admin user was initially created with no BU under the Read only BU nor Full access BU.

Resolution

  1. When creating a new user, ensure to assign both read-only business unit (BU) access and full access BU.
  2. Without assigning these BU access levels, the newly created user will not see the BU in the dropdown list.
  3. An admin who already have the read-only business unit (BU) access and full access BU can assign the users with BU
  4. If there is a need to find who among the admin's have necessary access to assign BU - Reach out to support via support case.
