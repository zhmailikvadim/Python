Symptom

When you remove the reference controllable resource from the technical resource master data, the system doesn't trigger the process "Sending of 55617 from Responsible DSO to SUP" (DE_C_MDC_55617_DSO2SUP_SEND).

This issue is relevant for the market roles: DSO, SUP.

Other Terms

MDC, RESRCTR0100, RESOURCE, technical resource master data, reference controllable resource, TECH_REF_TO_CTRL, Ref to Contrllable Res, 55617, MDX outbound processing framework

Reason and Prerequisites

The error occurred because the configuration was missing.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, activate the BC set /US4G/3_0_COMM_SP07_NOTE_3709203 using transaction SCPR20. Once you remove the reference controllable resource from the technical resource master data, the MDX outbound processing framework triggers the process "Sending of 55617 from Responsible DSO to SUP" (DE_C_MDC_55617_DSO2SUP_SEND). You can verify the fix by removing a reference controllable resource from the technical resource master data and checking that the system triggers the process.
