Symptom

You are using the Run Statutory Reports app to prepare VAT reporting for Russia.

In Russia, the standard VAT rate has been increased from 20% to 22%, effective January 1, 2026. As part of this change, adjustments are delivered to analytics to reflect the new tax rate for VAT Return, Sales Ledger, and Purchase Ledger reporting.

Other Terms

RU, Russia, SAP Document and Reporting Compliance – statutory reporting, formerly known as Advanced Compliance Reports, Russia VAT Return, RU_VAT_DCL, RU_SALES_LEDGER, RU_PURCHASE_LEDGER, Run Statutory Reports, Run Advanced Compliance Reports, Tax grouping version 335, DP19

Reason and Prerequisites

Legal Change.

This SAP Note is relevant only for Russia.

This SAP Note provides you with the relevant system changes to address the legal change: Changes: Sales and purchase ledgers January 1, 2026

Solution

As a general rule SAP recommends that you implement the Support Package specified for your release. However, if you need to install a solution earlier, use the SAP Note Assistant to implement the correction instruction. You can find more information about the SAP Note Assistant in SAP Support Portal, under SAP Note Assistant.

For more information about the development delivered with this SAP Note, see the What's New topic on SAP Help Portal: Updates to Sales and Purchase Ledgers and VAT Return Analytics (2026).
