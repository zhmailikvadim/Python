Symptom

Requirement: It should be possible to configure and control the properties of the items of the Changes Record dynamically also in the 'All Objects' tab instances.

Other Terms

PLM-CR, All Objects tab, display, items, filter, sort, object type, 3655773, 3680118.

Reason and Prerequisites

Advance Development.

Prerequisite SAP Note:3655773 & 3680118.

Solution

Use transaction SNOTE to implement the corrections delivered with this SAP Note in the affected Change Record system or import the relevant Support Package.

Benefit:

After implementing the corrections from both the SAP Notes, the business users can get the below mentioned features:

  • When a Change Record Item (say MAT-Material) inherits 'Change Number' from header, the corresponding item instance is disabled for deletion from the tab 'All Objects'.
  • When a CR Item requires 'Digital Signature' for change in the value for 'Processing Status', this CR item cannot be deleted from the tab 'All Objects'.
  • When Process Route Task's are created for a CR Item, the CR Item cannot be deleted from the tab 'All Objects' for the below mentioned Items-types
    • MAT
    • RCP
    • BOM
    • INP
  • The field 'Item Relevance' will be read only for the below mentioned Item-types from the tab 'All Objects' having the values mentioned under the section - Configuration for 'Define Status Checks and Actions'
    • MAT
    • BOM
    • MR
    • INP
    • SPC
    • RCP
    • PRG
    • ETL

Configuration for 'Define Status Checks and Actions':

  • Execute transaction SPRO 
  • Navigate to "SAP Customizing Implementation Guide -> Logistics - General -> Product Lifecycle Management (PLM) -> Change Records -> Settings for Change Records".
  • Choose the node "Define Status Checks and Actions".
  • Say for a particular row for having the below values:
    • Field 'User Status Action' - SE10
    • Field 'Status Profile' - S02
    • Field 'Status Act' is having one of the values mentioned below:
      • LOCKITMS - Lock All Items of Change Record
      • PRTLLOCK -  Partial Lock All Items of Change Record
