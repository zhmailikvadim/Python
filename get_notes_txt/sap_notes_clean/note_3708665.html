Symptom

In the Agreement Request auto-post scenario, when multiple Agreement Requests are selected and approved using the mass processing option, the system approves and posts only the first document. The remaining Agreement Requests in the selected list are not processed and are ignored by the system.

This note ensures that all documents are approved and posted successfully.

Other Terms

/IRM/IPAGRM, mass_header_update, fcode_auto_post, Mass Process Approvals, Auto Post

Reason and Prerequisites

This is caused by program error.

Note : SAP Note#3708626 is available for ECC Release

Solution

Please implement the correction instructions. These corrections are only valid if you have applied the following Support Packs.

Release V/4 – 2022 V2, 2023, 2023 V2 ,2023 V3 Base Package or Support Pack 1.

Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object
