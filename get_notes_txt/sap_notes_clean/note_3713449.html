Symptom

There are no fiori applications that are similar with the transaction for the upload of preference rules (transaction /SAPSLL/PREF_RLS_UPL) and the display of upload logs of preference rules (/SAPSLL/PREF_RLS_UPM).

Environment

  • SAP Global Trade Services
  • GTS edition for SAP HANA 

Resolution

For the transactions /SAPSLL/PREF_RLS_UPL and /SAPSLL/PREF_RLS_UPM,
only for GTS edition for SAP HANA version 2025, will there be a Fiori application available.

For example.

  • App : Upload Preference Rules - Preference Management
    With this app, you can upload Preference Rules from an XML file.

  • App : Display Upload Logs for Preference Rules - Preference Management
    With this app, you can display the logs that were written during the upload of Preference Rules.

  • App : Delete Uploaded Preference Rules - Preference Management.
    With this app, you can delete the uploaded Preference Rules.
