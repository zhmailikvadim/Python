Symptom

SAP GUI ALV Grid dumps when huge data is edited and this happens only with non-unicode systems. This happens if we press ENTER or save or refresh grid after entering new data.

Other Terms

SAP GUI for Windows; sapgui; ALV Grid; gridview; GET_MODIFIED_CELLS; edit;

Reason and Prerequisites

Program error

Solution

Please apply the relevant corrections instructions based on the SAP_BASIS release of the system.
