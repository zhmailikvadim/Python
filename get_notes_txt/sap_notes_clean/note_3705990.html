Symptom

  • When changing values on a Parent Grouping Input Control, the restrictions applied to dependent input control are being set incorrectly due to which incomplete values are intermittently observed in the dependent input control.

Other Terms

webi, grouped input control, ic, grouped ic, restriction, incorrect values

Reason and Prerequisites

Environment:

  • SAP BusinessObjects BI Platform BI 2025 Patch 6 
  • Web Intelligence

Steps to Reproduce:

  1. Log in to BI Launchpad.
  2. Open the Web Intelligence Report containg a grouped input control with multiple dependent input control filters. 
  3. Change the values for parent input control & check values in the dependent input controls.
    >> On multiple attempts at performing the Steps (3), you will notice that some values are not seen in the list in the dependent input control, intermittently.
