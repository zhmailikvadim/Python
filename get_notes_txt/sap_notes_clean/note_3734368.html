Symptom

In the iContent Management tool, active courses aren’t displayed depending on the filter (Course Affiliation, Environment, and Deletion Status).

Environment

SAP SuccessFactors Learning

Reproducing the Issue

  1. iContent Management tool >
  2. Review the table data > Ex.: A content file has two course affiliations, one active and one inactive >
  3. Now, select "Inactive courses" in the "Course Affiliation" >
  4. The same content file will only show one inactive course, the active course link is no longer visible.

Resolution

Our Learning Engineering team is currently working on this situation. Please, set this KBA as a favorite (star icon on the top right corner) to receive updates on it. Check the KBA 2171560 to understand how to get notified when a KBA is created/updated.

The observed behavior can lead to deletions of contents related to active courses, considering that the Learning Administrator can schedule deletion without any warning about it.

Once the work is concluded, the data in the table rows should remain static regardless of if a filter is applied, meaning that the linked active courses should remain visible.

Additionally, when the Learning Administrator selects content with active course affiliations for deletion, a confirmation dialog should appear, notifying them of the associated risks and that deleted content cannot be recovered. The Learning Administrator may proceed to schedule the deletion or add it to an existing job only after acknowledging the warning.
