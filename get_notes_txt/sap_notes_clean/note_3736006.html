Symptom

You use My Inbox and you have one of the symptoms described in the following SAP Notes:

3725442 - My Inbox Suspend Button Is Not Emphasized

3725390 - My Inbox Displays Two Error Messages When Loading Failed

Other Terms

F0862

Reason and Prerequisites

Fiori My Inbox, F0862

Solution

Apply this SAP Note, use SMICM transaction to clear ICM cache as described in SAP Note SAP Note 2319491 - How to clean up the cache after applying changes that affect SAP Fiori apps, and clear the end user browser cache.
