Symptom

In the process "Start of Supply for Gas (New SUP View)" (DE_B_GELIGAS_SOS_SUP), when you restart the process at step 0330, the popup doesn't prefill all the information that was passed in the trigger report initially.

This issue is relevant for the market role: SUP.

Other Terms

 GELIGAS, Start of Supply for Gas, DE_B_GELIGAS_SOS_SUP, BPEM, restart, prefill, popup, trigger report

Reason and Prerequisites

The error occurred because of a logic error in the system.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the BPEM popup displays all the required information that was passed in the trigger report. You need to rerun step 0330. You can verify the fix by checking that all values are prefilled in the popup when you rerun step 0330.
