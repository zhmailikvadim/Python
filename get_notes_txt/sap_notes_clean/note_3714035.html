Symptom

When running CDI you get the error message: 'Alias or namespace Org.OData.Core.V1 already defined'

Other Terms

SAP Gateway CDI

Reason and Prerequisites

There was a change in the gateway layer which is used by CDI.

Solution

To solve the issue please apply the Support Package as mentioned below. In urgent cases you can use the correction instructions.
