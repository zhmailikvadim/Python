Symptom

This SAP Note addresses the symptoms below:

You are receiving an error "Attributes of interface PL_KSEF_SEND_REQUEST are not maintained" on selecting the Push button Submit in eDocument Cockpit or in Manage Electronic Documents application.

(or)

You are implementing the Offline PDF feature delivered in the SAP Note 3710009 - Electronic Document Processing for Poland - KSeF: Offline PDF Issuance and Technical Corrections

Note:

This SAP Note is relevant only for S/4HANA releases 102 - 106 and ECC releases 606, 617 and 618.

Other Terms

Poland, eDocument, eDoc, eInvoicing, electronic invoice, electronic invoicing, Poland eInvoice, Document Compliance, Digital Compliance, DC, B2B, National System of e-Invoices, KSeF, Krajowy System e-Faktur, e-Faktura, faktura ustrukturyzowana, faktura elektroniczna, Electronic Document, Document and Reporting Compliance, electronic document processing, FA(2) Schema Version Code, FA(3) Schema Version Code, Retry, Resubmit, Delete, Offline PDF

Reason and Prerequisites

Reason: 

Missing EDOINTDRCATTRDET Table Entries for Offline PDF feature.

Solution

Date Version Reason
01.02.2026 1 Released for Customers

Import the Support Package specified for your release or implement the attached correction instruction.

SAP recommends that you implement the Support Package specified for your release. However, if you need to install a solution earlier, use the SAP Note Assistant to implement the correction instruction. You can find more information about the SAP Note Assistant in SAP Support Portal, under SAP Note Assistant.
