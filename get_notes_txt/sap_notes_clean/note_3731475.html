Symptom

Internal Error when attempting to delete effective dated records from the history in the Latest People Profile for Personal Information and Dependent Portlet with "Enable the latest Editing and History UI for Employee Central HRIS entities on Full Profile".  


Environment

SAP SuccessFactors Employee Profile
SAP SuccessFactors Employee Central - Personal Information and Dependent Portlet

Reproducing the Issue

  1. Navigate to the Latest People Profile with "Enable the latest Editing and History UI for Employee Central HRIS entities on Full Profile".
  2. Attempt to delete an effective dated record of one time slice for Personal Information/Dependent Portlet from the history in the Latest People Profile of any random employee. 
  3. Observe the "Internal Error" message and the inability to delete slices from the history. 

Cause

Known issue

Resolution

  1. Engineering team has identified the root cause and created code corrections.
  2. The fix for the issue will be provided as part of release 2605.
