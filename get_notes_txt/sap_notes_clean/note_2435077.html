Symptom

Run report RFUMSV00 to issue VAT report for company located in Thailand, according to the Thailand legal requirement, the branches of business partners are supposed to be listed, however by default it cannot display the branch information.

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Environment

•    SAP R/3

•    SAP R/3 Enterprise

•    SAP ERP Central Component

•    SAP ERP

•    SAP enhancement package for SAP ERP

•    SAP enhancement package for SAP ERP, version for SAP HANA

•    SAP S/4HANA Finance

•    SAP S/4HANA

Resolution

To display Branch information in report RFUMSV00 please refer below steps:


Run report RFUMSV00 via T-code SE38 and give a company code whose country is configured to Thailand, and in “Output lists” section, click “Configure” button in row Output tax line items or input tax line items:

 

Click the “change layout” tool in toolbox:

 

Select hidden fields “Branch Office Number” and “Head Office Number” from right table to left table, then click “Copy” button:

 

Click “Save Layout” tool box:

 

Give a name and description, save it:

 

Back to selection screen and give the name you just maintained:

Then the two fields will show in ALV list.
