Symptom

You receive ATC findings for key user objects of type VIEW or STOB in transportable packages, e.g. indicating that they are not cloud-ready, although you've implemented the according SAP note 3706047 already.

Other Terms

Usage of not released APIs, Allowed Object Types in Cloud Development, Cloud-Readiness Checks, Key User Items, ATC false positives

Reason and Prerequisites

The ATC checks do not identify objects of type VIEW or STOB as being part of the ABAP Cloud programming model, although they belong to CDS entites created by key user apps.
As a prerequisite, implement the SAP notes 3605376, 3593439 and 3706047.

Solution

This note is relevant for all the systems where ATC checks are being performed. It does not need to be implemented in the central ATC system.

To resolve the ATC issues, please do the following:

  • Make sure the SAP notes 3605376, 3593439 and 3706047 are implemented already.
  • Implement this correction instruction or install support package 2.
