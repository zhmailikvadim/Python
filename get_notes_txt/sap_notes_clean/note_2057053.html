Symptom

The line created in a manual billing document through price grouping has an incorrect line item type.

Other Terms

CSNO 99999 PreisZus PrZu BELZART

Reason and Prerequisites

There is a program error.

Line items with at least two different item line types are transferred from a previous document. The first available line item type is used, but this of course does not correspond with the line item type actually created through price grouping.

Solution

Implement the correction instructions attached to this SAP Note or import the relevant Support Package.
