Symptom

In Start Process application, you are using Custom table and have hidden the 'Add' button using BAdI.

When you edit a row and select any dropdown which has User Event configured, it is seen that the 'Add' button appears on the form when table dialog is closed.

Other Terms

Fiori form, Start Process, Custom table, Add button, Dropdown, User Event

Reason and Prerequisites

The issue was occurring due to an issue with table buttons formatter. When custom table is used, the 'Add' button visibility should be considered from the BAdI.

Solution

Apply the corresponding Support Package to solve the error. Alternatively apply the attached correction instructions of this SAP Note.
