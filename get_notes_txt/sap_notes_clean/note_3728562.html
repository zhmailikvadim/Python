Symptom

  • SAP Cloud ERP private, pre-packaged system is not visible in the list of systems to be included in a Business Data Cloud (BDC) formation.

Environment

  • SAP Business Data Cloud

Reproducing the Issue

  1. In SAP for Me, navigate to BDC;
  2. Click on Formation and select a formation type "Integartion with SAP Business Data Cloud";
  3. Click on "Include Systems" and attempt to locate the PCE system with its system number;
  4. Observe that the PCE system is not visible in the list of systems to be added.

Cause

The formation type "Integration with SAP Business Data Cloud" does not currently support systems of type "SAP Cloud ERP private, pre-packaged".

Resolution

Note that the integration of SAP S/4HANA Private Cloud Edition (PCE), pre-packaged, with SAP Business Data Cloud formations is currently under development.
Refer to the SAP Business Data Cloud and SAP S/4HANA PCE road maps for updates on the availability of this functionality, expected during 2026 Q2.
