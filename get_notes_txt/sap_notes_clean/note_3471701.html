Symptom

  • Rescheduling an instance in BILP (BILP) results in the error:
    Error: Resource not found: ID =<ID> (RWS 00009)
  • The same error does not occur in Central Management Console (CMC)

Reason and Prerequisites

Environment

  • SAP BusinessObjects Business Intelligence Platform 4.x (BI 4.3)
  • Windows
  • Linux / Unix 

Steps to Reproduce

  1. Log in to BILP
  2. Navigate to a report
  3. Access the report history
  4. Reschedule an instance
  5. Observe the error

Solution

This issue is fixed in the patches listed in the "Support Packages & Patches" section below.

The "Support Packages & Patches" section will be populated with the relevant patch levels once they are released.

For Business Intelligence Platform maintenance schedule and strategy see the Knowledge Base Article 2144559 in References section.

Workaround

Reschedule the report via CMC
