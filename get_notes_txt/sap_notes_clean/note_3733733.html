Symptom

Please note that this SAP note is intended for customers how have upgraded their S/4HANA system using the S4FND TCI note.

Current flow of /SMFND/SDF_BUFFER_REFRESH_PROG always deletes the existing records in /SMFND/D_4MT_BUF. This means that if the API calls to retrieve the data fails, the caching table is left empty, which will block processes like form assignment.

Additionally, this note will add more detailed logging for connectivity errors for better troubleshooting

Other Terms

S4MOBADDON-8432, MAIF, SAP Dynamic Forms, Form Definitions, Form Categories, Caching, log, TCI
