Symptom

Customers would like to know if it is possible to edit or populate a workflow field directly when approving a workflow using Joule.

Environment

  • SAP SuccessFactors HCM Suite
  • Joule

Reproducing the Issue

  1. Open a workflow approval in Joule.
  2. Attempt to edit an workflow field directly before approving.
  3. Observe that only the “approve with comment” option is available and the field cannot be edited.

Cause

Direct editing of workflow fields from within Joule during approval is not a supported capability according to current Joule use cases.

Resolution

It is not possible to edit or update workflow fields during approval via Joule.

According to Use Cases Supported in Joule | SAP Help Portal, Joule provides a set of predefined use cases, including workflow approvals.

However, these use cases are limited to executing actions and do not include the ability to modify workflow data.
