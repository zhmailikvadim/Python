Symptom

There is a requirement to identify the changes made in SPRO.

Environment

  • SAP TM 9.0+
  • SAP S/4HANA 1709+

Resolution

  1. Open transaction SPRO
  2. Search for the relevant customization option for example 'Define Freight Order Types' (Select it with a double click, but do not execute!)
  3. Once it is selected click on the 'Change Log' button at the menu bar.
  4. Specify the start and end dates for the change log.
  5. OPTIONAL: Selecting 'ALV Grid Display' from the Output Options will provide a summary of the changed tables as a result, while not selecting it will provide a more extensive result containing the changed fields as well. 
  6. Execute the query and it will display the result.
