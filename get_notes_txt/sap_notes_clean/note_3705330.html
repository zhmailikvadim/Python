Symptom

  • Org assignment replication error: Employee XXXXXXXX: Manager XXXXXXXX not yet replicated from EC - Message No. SFIOM_PROCESSING013
  • Error occurs even though Manager XXXXXXXX already exist in ERP/S4 system

Environment

  • SAP SuccessFactors Employee Central Integration to SAP Business Suite
  • BIB 
  • Org Assignment Replication

Reproducing the Issue

  1. Enable S(position)-S(manager position) relationship in Employee Org Assignment replication.
  2. Run Org Assignment replication for a specific employee from SF to ERP/S4 system.

Cause

  • In the class CL_PAOCF_EC_KEY_MAP_EMPLOYEE and method CHECK_PERNR_BUKRS_MOLGA, a check is performed for table T500P for the manager's company ID(bukrs) and manager's country( mogla).  If no entry found, the error SFIOM_PROCESSING013 will be raised. 

Resolution

  1. Go to transaction SE16, open and edit table T500P.
  2. Add an entry for the manager's company ID(bukrs) and manager's country( mogla).
  3. Save.
