Symptom

A disconnection / reconnection or a move-in / move-out is processed and the meter reading result shall be taken. The Meter reader is changed from the default value to the new desired value. After a round trip, the old value re-appears unchanged. The correct Meter reader is stored.

Other Terms

EC86 EC87 Meter reader EC60 EC55E EC50E

Reason and Prerequisites

Program error

Solution

Implement the attached Correction Instruction or the mentioned Support Package.
