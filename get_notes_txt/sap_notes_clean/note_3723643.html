Symptom

In the process "Sending of Delivery Note (DSO View)" (DE_B_GUB_DELVNOT_REQ_SEND_DSO), the system picks up an incorrect OBIS code in steps 0010 and 0015.

This issue is relevant for the market role: DSO.

Other Terms

US4G, GUB, Delivery Note, OBIS code, /US4G/CL_UBD_ENERGY_VALUE_A1, FILL_UBD_ENERGY_VALUES, DE_B_GUB_DELVNOT_REQ_SEND_DSO

Reason and Prerequisites

Program error

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the system picks up the correct OBIS code in the delivery note process. You need to retrigger the process.
You can verify the fix by checking that the OBIS code is picked up correctly in the additional data of step 0010 in the process "Sending of Delivery Note (DSO View)" (DE_B_GUB_DELVNOT_REQ_SEND_DSO).
