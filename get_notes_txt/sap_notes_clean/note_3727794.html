Symptom

When attempting to upload the file in the 'Manage Small Business ABN List' activity within the Australia Payment Time (PTR) Report in the app Run Statutory Reports.The following error message is displayed: "Supplier not available for uploaded VAT registration number".

Environment

SAP S/4HANA Cloud Public Edition.

Reproducing the Issue

  1. Open the app Run Statutory Reports.
  2. Open an Australia Payment Time Reporting (PTR) report.
  3. Run and complete the 'Get Supplier ABN List' activity.
  4. Upload the list of ABNs in the SBI Tool.
  5. Save the SBI Tool output on a local drive.
  6. Open the 'Manage Small Business ABN List' activity.
  7. Select the company code.
  8. Select the 'Upload file' option and choose the SBI Tool output file.
  9. Click 'Execute' and the following error message is displayed:"Supplier not available for uploaded VAT registration number".

Cause

Spacing issue in the system configuration preventing recognition of the updated format output from the SBI Tool.

Resolution

Remove the spaces so that the system can recognize it.

Keywords
