Symptom

Files imported into LMS using the "Import content" tool are not displayed under the "iContent Management" page in LMS Administration.

Environment

SAP SuccessFactors Learning

Reproducing the Issue

  1. Import content files into LMS using the "Import Content" tool.
  2. Navigate to the "iContent Management" page to check for the uploaded files.
  3. Confirm that the files do not appear under this page.

Cause

There is a job which should synchronize the data every 2-3 hours, but it is not working as expected.

Resolution

Engineering team is working on the fix, currently 1H2026 release is targeted. Please follow this KBA for any new information.
