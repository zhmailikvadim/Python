Symptom

This SAP Note provides you with an update to enable the Manual Adjustment feature for the Ireland Return Trading Details report (report category IE_RTD_DCL).

Other Terms

Ireland, IE_RTD_DCL, IE_RTD_DCL_2, Ireland Return Trading Details, Manual Adjustment, Edit, DRC, SAP Document and Reporting Compliance

Reason and Prerequisites

This SAP Note is relevant only for Ireland.

Solution

After implementing this SAP Note, the Ireland Return Trading Details report will be updated with the Manual Adjustment feature. The Manual Adjustment feature for Statutory Reports enables you to edit data if there are unavoidable last-minute adjustments needed to ensure correct legal reporting.

As a general rule, SAP recommends that you implement the Support Package specified for your release. However, if you need to install a solution earlier, use the SAP Note Assistant to implement the correction instruction. For more information see SAP Note Assistant.
