Symptom

  • I set up a Field Data Populate Rule to copy the value of a custom field from a source module to a target module
  • After editing the document and update the value in the source module, the corresponding field in the target module did not update automatically

Resolution

The Field Data Populate rule is only triggered during the creation and initial submission of the source document. If the target field is already populated, changes made to the source field after submission will not trigger the rule again, and the target field will not update automatically
