Symptom

When the user tries to display archived 'change documents' for an archived payment distribution item, an incorrect error message "Change documents not found" is thrown inspite of archived change documents being available.

Other Terms

/FSPD/ITEM_BVW_DISP, ILM, PD Item

Reason and Prerequisites

Program Error

Solution

Import the relevant Support Package for your release.

Use transaction SNOTE to implement the attached correction instructions in advance.
