Symptom

You want to implement SAP Convergent Mediation by DigitalRoute 9.0 support package 1.

Other Terms

SAP Convergent Mediation, SAP Convergent Policy, DigitalRoute, SAP CC, SAP Convergent Charging, SAP Hybris Billing mediation, SAP Hybris Billing service control, MediationZone.

Reason and Prerequisites

You have implemented SAP Convergent Mediation by DigitalRoute 9.0.

Solution

SAP Convergent Mediation 9.0 by DigitalRoute Support Package 1 equals partner release number MediationZone 9.0.1.1, s. also SAP note 2925138.

The bug fix appendix can be found as PDF attached to the note. The most current version is available in the documentation portal InfoZone of DigitalRoute, use this link Bug_Fix_Appendix-9.0.1.1.pdf, s. also SAP note 2924977. Please be aware that the bug fix appendix document included in the delivery file might meanwhile updated by DigitalRoute and is outdated.

The package is available on SAP download center, check attachment 'How to find Support Package 1 for SAP CM BY DIGITALROUTE 9.0.pdf' on how to find it.

Be aware support packages for SAP Convergent Mediation by DigitalRoute always contains the full package scope (not only the corrections). Only those components are replaced during installation which are related to the bugs listed in bug fix appendix.

If you do not have a permanent license key check SAP note 2751421.
In case you encounter problems when installing the software, report an incident using component XX-PART-DRT.
