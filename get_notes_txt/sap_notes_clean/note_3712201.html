Symptom

When perform Data Preview in Analytic Model, dimension description is not displayed correctly. For example:

  • Compound description of compound dimensions are not displayed, same description for one dimension is displayed regardless of the other dimension' value.
  • When Data Access Language in Datasphere is changed, dimension description becomes empty.
  • Key is displayed for dimension description.

Environment

SAP Datasphere

Reproducing the Issue

  1. In Analytic Model, click the Data Preview.
  2. Add a dimension to Rows/Columns.
    -> The description for dimension is not displayed correctly. For example, wrong/empty/duplicate description is displayed.

Cause

Incorrect setting of dimension description.

Resolution

Please refer to following Support Content to maintain the description of dimension correctly, and check the 'Common incorrect usage' and 'Limitations' section.

How to maintain description for a dimension in Analytic Model
