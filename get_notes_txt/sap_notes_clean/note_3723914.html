Symptom

This KBA explains how profile works for rate routings.

Environment

  • SAP S/4HANA, on-premise
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Go to transaction CA21 – Create Rate Routing.
  2. Select Profile.
  3. Enter
  4. Enter required fields such as Plant.
  5. Save the rate routing.
  6. Check table PLKO. Fields like Base quantity BMSCH/ Numerator UMREZ/ Denominator UMREN are defaulted with values maintained from the selected Profile.

Resolution

  • The profile for rate routing is maintained in transaction OP8H or OP58 > Profile/Rate Routing.
  • On the initial screen of CA21, enter a profile. The values from the profile will be taken as default values and saved at the routing header level (PLKO-BMSCH/UMREZ/UMREN).
  • Later when adding operations, the values will be defaulted to operations, but they can be maintained/changed in each operations.
