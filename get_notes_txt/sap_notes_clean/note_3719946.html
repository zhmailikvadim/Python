Symptom

The SAP application datastore fails to be crated with error "Could not open the ICU common library" .

Other Terms

DS, sap, datastore, ICU, icuuc57, icudt57,icuin57, dll, path, environment library

Reason and Prerequisites

Environment

  • Windows 64-bit
  • SAP Data Services XI4.3 and higher version

Cause

 SAP GUI is not installed at the Designer's Windows OS.

Solution

Upgrade the DS to the latest version.
