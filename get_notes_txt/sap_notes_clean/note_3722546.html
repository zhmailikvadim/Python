Symptom

When a user attempts to Submit a Receipt, the document does not get submitted.
No explicit error message is displayed.

Upon review of the Receipt approval flow, the user attempting to submit the Receipt is not included as an approver in the first approval node.
The approval rule uses the predefined action Add Requester and All Team Members, and the expectation is that Guided Buying team members are included in the approval flow. However, they are not added to the first approval node.

Cause

There are two separate team functionalities :

  • Team Requisitioning and Team Receiving are defined in SAP Ariba Procurement Solutions.

  • Team Buying and Receiving in SAP Ariba Guided Buying

Guided buying teams are completely separate from teams in SAP Ariba Procurement solutions. Guided buying team requisitions aren’t supported in SAP Ariba Procurement solutions; similarly, SAP Ariba Procurement solutions team requisitions aren't supported in guided buying.

The predefined action Add Requester and All Team Members applies only to the Procurement Solutions-defined team structure.
Guided Buying team members are not considered by this predefined action. As a result, only Procurement Solutions-defined team members are added to the approval node.

If the submitting user is defined only as a Guided Buying team member, not as part of the Buyer team, the user is not included in the first approval node and cannot submit the Receipt.

Resolution

  1. Verify that the users expected to perform receiving are included in the first approval node.

  2. Confirm whether the required team members are defined in the Buyer team structure or only in the Guided Buying team structure.
    Note: there is no Guided Buying equivalent of the predefined action Add Requester and All Team Members; this action applies only to Buyer-defined teams.

  3. (a) If Guided Buying team members must participate in the approval flow, update the approval rule to explicitly include the relevant users, groups, or roles supported by Buyer approval logic. It is recommended to use user groups to add Guided Buying team members as approvers.
    (b) Alternatively, maintain the required users within the Buyer team structure if the predefined action Add Requester and All Team Members is to be used.

  4. After making the changes, test the Receipt submission and confirm that the expected users are included in the first approval node and that the document can be submitted successfully.
