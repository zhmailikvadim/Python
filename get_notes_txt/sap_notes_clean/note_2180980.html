Symptom

This SAP Note corrects SAP Note 2131640 - Special data types in SAP HANA ODP

If these corrections are not implemented, the following symptoms may occur during the extraction of SAP HANA views via ODP:

- Poor performance

- Memory overflow in the methods Wrap_Data or get_memory_size_of_object during data packing

- Selection fields of subscriber (for example, SAP BW InfoPackage or DTP) are enhanced for DB selection and return unexpected results

Other Terms

HANA view ODP DATATP cl_abap_memory_utilities get_memory_size_of_object performance selection

Reason and Prerequisites

There is a program error.

Solution

  • SAP BW 7.30
    Import Support Package 14 for SAP BW 7.30 (SAPKW73014) into your BW system. The Support Package is available once SAP Note 2132939 ("SAPBWNews NW BW 7.30 ABAP SP14"), which describes this SP in more detail, has been released for customers.
  • SAP BW 7.31 (SAP NW BW 7.3 EHP 1)
    Import Support Package 17 for SAP BW 7.31 (SAPKW73117) into your BW system. The Support Package is available once SAP Note 2139356 ("SAPBWNews NW BW 7.31/7.03 ABAP SP17"), which describes this SP in more detail, has been released for customers.
  • SAP BW 7.40
    Import Support Package 13 for SAP BW 7.40 (SAPKW74013) into your BW system. The Support Package is available once SAP Note 2176288 ("SAPBWNews NW BW 7.40 ABAP SP13"), which describes this SP in more detail, has been released for customers.

In urgent cases, you can implement the correction instructions as an advance correction.

You must first read SAP Note 1668882, which provides information about transaction SNOTE.

To provide information in advance, the SAP Notes mentioned above may already be available before the Support Package is released. In this case, the short text of the SAP Note contains the words "Preliminary version".
