What’s New for SAP GRC Assurance and Compliance for SAP HANA (Support Pack - 00)

Release Highlights for SAP Audit Management

Master Data

  • You can now import master data and raise issues for a finding without an RFC connector if SAP Audit Management runs on the same system as Process Control and Risk Management.
  • You can now upload analysis for multiple risks using a spreadsheet template

Audit Planning

The following charts appear in the header section of audit plan groups and audit plans.

  • Cost
  • Effort
  • Risks Covered by Audit
  • Audits by Phase

Resource Management

You can now restrict the roles that can be assigned as substitutes.

Audit Preparation

You can now assign controls to audit procedures directly in the spreadsheet. The value help is configured such that you can only select controls that are already assigned to the parent work package

Audit Execution

The following apps have been added to help you manage audit recommendations:

  • My Audit Recommendations
  • Track Open Audit Recommendations
  • Display Historical Audit Recommendations

Audit Procedures

You can now create questions that has multiple answer types in a question procedure. The answers may be in the form of ratings, single choice, or multiple choice.

Working Papers

The following enhancements have been made for working papers.

  • Access your audit working papers directly from a WebDAV client (for example, Windows Explorer) using the WebDAV interface. A folder is created for each audit. You can open and edit a working paper using desktop applications.
  • Use the Microsoft SharePoint and Knowledge Provider (Kpro) repositories along with the HANA database to store your working papers and other attachments. You can select a default repository based on the object type. You can also move files between repositories.
  • Search for working papers both in the working paper section and in specific working paper category folders.
  • When uploading working papers, view the upload status in the progress bar..

Audit Analytics

The Audit Management Coverage app can be used to analyze audit coverage using multiple cards. Data related to the following areas are covered:

  • Risks
  • Controls
  • Organizations
  • Dimensions
  • Processes

AI Extensions

SAP Audit Management now provides AI-assisted capabilities with the help of generative AI. The AI extensions support the following functions:

  • Update Finding fields
  • Generate executive summary for audits
  • Suggest work programs

Other Enhancements

The following enhancements have been made across different audit areas

  • Consolidate multiple email notifications such that users receive all notifications in a single email.
  • Configure notifcations so that specific people will receive email alerts when an comment is posted for action plans.
  • Filter activities for an audit by selecting a time range or specific action.
  • Enable a CDS-based value help for a custom field filter. The CDF will then display a value help input field along with the search functionality.
  • Integrate the SAP workflow with the Audit Management workflow. The My Audit Tasks app has been added, which helps you track all tasks that require an action from you.
  • Create custom cards for the Audit Management Overview page. When creating the cards, you can either use an Odata service entity or a custom CDS view.
  • Search the LDAP directory using a speciic server instead of the default LDAP server. 
  • Save variants for worklists in the following apps:
    • Risk Register
    • Controls
    • Audit Universe
    • Manage Audit Plans
    • Consolidate Audit Plans
    • Audits
    • Work Packages
    • Findings
    • Recommendations
    • Action Plans
    • Load object page data from OData services faster due to improved performance of Fiori apps.

 

Release Highlights for SAP Business Integrity Screening

Enhanced features:

  • The Detection Runs and Calibrtion Detection Strategies apps have been rebuilt using Fiori.
  • The Process Address Screening Hits app can be extended to enhance the user interface.
  • The Edit Multiple option can be used to add attributes to multiple user groups.
  • The SAP Note for Excluded Terms and Terms Mapping has been updated to include details on using non-whitespace languages.

The following feature has been deprecated:

  • Business Rule Methods in Manage Detection Methods has been deprecated because of the deprecation of the HANA Rule Framework. See SAP Note 2455694.

Release Highlights for SAP Business Partner Screening

The integration scenario for Master Data Screening: Decision in SAP Master Data Governance has been deprecated.

Release Highlights for SAP Tax Compliance

Monitoring Objects for Compliance Check

When monitoring objects generated for compliance checks, you can now regenerate objects with HANA database hints. You can use multiple hints for an object.
