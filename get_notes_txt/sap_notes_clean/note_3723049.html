Symptom

You create withholding tax accounting for Switzerland using the HR form SAP_QSTC0_CH01  (transaction:  PC00_M02_HRF_QSTC0).

For this form, SAP Note 3711365 enabled the creation of settlements for several payroll units with one call. However, it is not possible to create different header lines for each payroll unit. The WHT payroll runs of all payroll units are also output in the same form.

Other Terms

Withholding tax, WHT accounting, payroll unit, preliminary report, SAP_QSTC0_CH01, HRFORMS, SP01, spool

Reason and Prerequisites

The print program of the HR form SAP_QSTC0_CH01 creates exactly one print job for each call. If several payroll units are processed in one call, all settlements end up in a single form and spool request. It is not possible to separate the header lines for each payroll unit within the HRFORMS framework because the form is opened and closed only once. To obtain separate forms with separate header lines, the print program must be called once for each payroll unit.

Solution

Import the HR Support Package relevant for your release. If you implement the solution in advance, use the Note Assistant to implement the correction instructions relevant for your release. You can find more information about the Note Assistant on SAP Support Portal at https://support.sap.com/notes.

The implementation of this SAP Note provides a new program (program: RPUQSTC0_HRF, transaction: PC00_M02_HRF_RPQSTC0) that enables the creation of WHT payroll runs for several payroll units at the same time, whereby a separate form is generated for each payroll unit.

The program executes the HR form SAP_QSTC0_CH01 internally once for each payroll unit. The spool requests created are displayed in a list after processing.
