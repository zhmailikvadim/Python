Symptom

You call the "Edit Day's Events" function for a free-text order, and the system triggers a dump.

Other Terms

Edit Day's Events, Free-Text Order

Reason and Prerequisites

Program error

Solution

This SAP Note changes the system behavior as follows:​
You should note that the "Edit Day's Events" function can still not be used for free-text orders because these orders do not have day's events. If you still try to call this function, the system issues the error message "This function is not available for free-text orders" (750/N1ME1).

Import the specified support packages to solve this problem.
You can implement collective note 3003429 in advance.
