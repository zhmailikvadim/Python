Symptom

The Change number status was not updated when using MODIFY action while using the C1 API "I_ChangeRecordTP_3". Also errors raised during modification action were not propagated back.

When change number is not allowed to be updated due to configurations, relevant message is not returned.

Other Terms

PLM-CR, PLM-WUI-OBJ-ECN, I_ChangeRecordTP_3, MODIFY, Change number status, error message, BO action, API, C1 release, ECN Change Number.

Reason and Prerequisites

Program error

Solution

Use transaction SNOTE to implement the corrections delivered with this SAP Note in the affected Change Record system or import the relevant Support Package.
