Symptom

A blank screen comes up and an error is displayed:

Error when printing: SYSTEM ERROR
Message no. HRHAP00_DOCUMENT160

Environment

  • SAP ERP 6.0, Enhancement Pack 4 (EHP 4)
  • SAP ERP 6.0, EHP 5
  • SAP ERP 6.0, EHP 6
  • SAP ERP 6.0, EHP 7 (and higher)

Resolution

  1. Verify the correct Print Layout is selected for the template.
    • Run transaction PHAP_TEMPLATE.
    • Select the template with the issue.
    • Select the Layout tab.
    • Under Print Layout, verify the layout is correct for the appraisal type:
      • PMP PDF - PMP appraisals
      • Standard - Flexible appraisals
      • Print Appraisal Documents: Old (with Smart Forms) - PMP appraisal Start Forms
  2. Verify the printer has been setup properly.
    • Test with printer LOCL.
    • For the printer with the error, go through the following steps:
      • Run Transaction SE16.
      • View table TSP03D.
      • Enter the printer name.
      • Make a note of the entry in the PATYPE field.  This is the Device Type Name.
      • View table TSP0B.
      • Enter the PATYPE.  The PATYPE needs to be setup for the printer to work proprly.
      • If the PATYPE is not in the system, @SWIN will be used by default.  If this is also not setup, this will cause further problems.
