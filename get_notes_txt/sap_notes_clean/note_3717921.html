Symptom

Facing below error while accessing Manage Workflow for Supplier Invoice Approval app in SAP Ariba Invoicing:

Unable to load Cloud Studio. DEP_1031: Environment does not exist. Correlation ID: null

Resolution

You must enable the feature Approval for Supplier Invoices in Processing Services. For more information, see Approval for Supplier Invoices.
