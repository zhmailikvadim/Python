Symptom

When replicate document to target system, get error "Store a cost element for credit down payments" (message number KI165) in AIF.

Environment

  • Financial Accounting (FI)
  • SAP R/3
  • SAP R/3 Enterprise 
  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA Finance
  • SAP S/4HANA

Reproducing the Issue

Check error message via AIF after replication.

Cause

Missing customizing for controlling area.

Table TKA01, the fields which are empty and related are:

  • CElem vend.dwnpaymnt (KSTAR_FIN)
  • RevTyp cus.downpaymt (KSTAR_FID)

Resolution

Maintain missing customizing via T-code OKEP.
