Symptom

When you create an import declaration referring to a purchase order for which the company code has changed, the currency for the goods value is incorrect in the declaration.

Other Terms

Customs document, Goods value currency, /SAPSLL/6800_TO_CD_MAP, /SAPSLL/LEGCULC, /SAPSLL/CIBD_01, /SAPSLL/CIBD_CSD_CREATE

Reason and Prerequisites

The problem is caused by a program error.

Solution

Implement the correction instructions for your release.
