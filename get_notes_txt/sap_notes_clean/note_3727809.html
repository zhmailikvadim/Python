Symptom

When two filter fields are configured with the same picklist id in Internal and External Career Search Settings, only one of the filter fields is visible in Career Site. This behavior occurs even if the filter fields are defined in separate Job Requisition Templates.

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

SAP SuccessFactors Recruiting Management

Reproducing the Issue

  1. In Internal and External Career Search Settings, map two(2) filter fields to the same picklist id


  2. In the Job Requisition Templates, define the filter fields accordingly and map them to the same picklist id

  3. Go to Career Site, notice that only one of the two filter fields are displaying:


  4. Modify the configuration so that filter fields are mapped to unique picklist id


  5. Both filter fields are now displaying in Career Site:

Resolution

This is an expected behavior. The system does not allow multiple filter fields to use the same picklist id in Career Site. Multiple filter fields with identical picklist values is redundant for search functionality.

For the filter field to display in Career Site, ensure that each filter field is configured with unique picklist id.
