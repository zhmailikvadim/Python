Symptom

A job of the line item migration to SAP Simple Finance 1503 is cancelled with short dump CX_AMDP_EXECUTION_FAILED in ZCL_FINS_MIG_UJ_HDB_GENERATED=>IF_FINS_MIG_UJ_HDB~COPY. The SQL code of the database is 257 "sql syntax error".

Reason and Prerequisites

The system time zone is west of Greenwich, i.e. SYST-TZONE < 0

Solution

Implement the correction instructions.

Execute report FINS_MIG_UJ_MAPPING_GENERATE to re-generate ZCL_FINS_MIG_UJ_HDB_GENERATED.

Restart the line item migration in transaction FINS_MIG_UJ with option "complete current run".
