Symptom

After running app Import Tax Codes after Transport in P system, error “Docs posted with Tax Code &1, Jurcode &2 are after ValidFrom &3” occurred.

Environment

SAP S/4HANA Cloud Public Edition

Cause

The tax procedure was changed for US from external to internal in previous years. There are some data inconsistencies in the system.

Resolution

To import the tax rates, below steps could be considered.

  1. In Test system SSCUI 103887 - Message Control in Tax Area, enter "FOT_US_INT_TAX_CALC" in "Application Area".
  2. Add a new entry with the message number "092". Enter "-" in both fields "Online" and "Batch Input". Save in a transport.
  3. Release and import this transport in P system.
  4. Run app "Import Tax Codes after Transport" again in PROD system.
  5. In Test system change the type of message FOT_US_INT_TAX_CALC/092 from "-" to "E" in the field "Online" and "Batch Input". Save and Release the transport.
