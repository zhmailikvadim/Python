Symptom

The system is not triggering the automatic IDOC for Sales Quotes replication from the source system to Sales Cloud as expected.

Environment

Integration for SAP Sales/Service Cloud.

Sales Cloud Version 2

S/4Hana

Reproducing the Issue

Configure S/4 for automatic confirmation and edit a Quote or Order to trigger the Idoc

Observe that the automatic idoc is not triggered in the output type, and the bi-directional process does not function as expected.

Cause

Activation of the New Output Management (BRF+) prevents the traditional NAST-based processing for the affected document categories.

Resolution

  1. Verify the setup in transaction VV11 for the relevant location and ensure that sales document types are added to enable output type determination.
  2. Confirm whether the new output management (BRF+) is active in the S/4 system. If BRF+ is active, note that NAST-based output type will not work.
  3. If the new output management (BRF+) is active, implement the user exit logic described in SAP Note 3634576: replication of sales orders or quotes from source system to Sales Cloud without NAST.
  4. Test the implemented configuration by creating or editing a quote to ensure proper functionality.
