Symptom

Return allowance prompting after charge tender.

Environment

SAP Point-of-Sale 2.3

Reproducing the Issue

  1. Ring a return w/ receipt over $50 as a level 2 cashier.
  2. Select 'charge' as the tender and swipe/key a card as prompted.
  3. The register will prompt that the maximum dollar amount is $50.00 andasks for a manager sign on.
  4. Wait a few minutes to sign on as manager.
  5. The register will not accept the manager log on - the register freezes with a pop up saying "minimum digit 1".

Resolution

 

For details about the resolution of this issue, see SAP Note 2013674.

Keywords

Point of Sale, Total Time, User Exit
