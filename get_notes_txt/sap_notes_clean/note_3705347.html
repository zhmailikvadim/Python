Symptom

Error messages "data source of xxx does not allow upload(see customizing)" and "upload of file xxx terminated" occur during commodity code upload.

Environment

  • SAP Global Trade Services Edition for Hana

Reproducing the Issue

  1. Attempt to upload commodity codes in SAP Global Trade Services E4H system.
  2. Observe the error messages: "data source of xxx does not allow upload" and "upload of file xxx terminated" in the logs.

Cause

  • The data source configuration for the numbering scheme content is incorrect.
  • If the data source for the numbering scheme content is defined as 'local', uploading the content is not permitted.

Resolution

  1. Review the configuration settings for the data source of the numbering scheme content in SAP Global Trade Services E4H system.
  2. Ensure that the data source is not set to 'local', as this setting prevents content upload.
  3. Modify the configuration to define the data source correctly, ensuring it allows uploads.
