Symptom

 In transaction code 7KE1, users observe that the sequence of input fields for Excel differs in the planning layouts across systems.

Environment

• Profit center account (EC-PCA)

• SAP R/3

• SAP R/3 Enterprise

• SAP ERP Central Component

• SAP ERP

• SAP enhancement package for SAP ERP

Reproducing the Issue

  1. Open transaction code 7KE1.
  2. Observe that the sequence of input fields for Excel differs in the planning layouts across systems. 

Cause

A change was made in one system but was not transported to other systems.

 

 

Resolution

  1.  Synchronize the planning layout across different systems.
  2. Re-transport the layout to synchronize between different system (menu option "Planning layout/ Transport form")

Keywords

PCA, planning, 7KEX, 7KE2, 7KEC
