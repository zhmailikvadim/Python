Symptom

You encounter issues with the import of UCM Datasets into the SAM4U tool, either via RFC or file upload. Target classifications are not calculated and it is not possible to drill down into the users' roles & profiles.

Other Terms

Software Asset Management, SAM4U Tool, SAM4U, PCE, UCM Data Import

Reason and Prerequisites

You have installed SAP TCI Note 3646933 - SAM4U Solution | Fiori Application | V2 and other prerequisite corrections which are referenced in SAP Note 3646933.

Solution

Implement this SAP Note to fix the above-mentioned issue.
