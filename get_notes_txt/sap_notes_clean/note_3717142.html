Symptom

In the DO Workspace, users belonging to one Sales Organization are able to display and maintain DOs that belong to other Sales Organizations.

This note corrects the authorization handling to ensure users can only view and maintain DOs for their assigned Sales Organization.

Other Terms

DO Workspace Authorization, Sales Org Restriction, Workspace Security

Reason and Prerequisites

 This is caused by program error

Solution

Please implement the correction instructions. These corrections are only valid if you have applied the following Support Pack(s):

Release V/4 – 2022 V2 and 2023

1. Perform the manual pre-implementation steps that are described below.

2. Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object.
