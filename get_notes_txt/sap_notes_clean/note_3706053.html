Symptom

The tool provided with SAP Note 2560592 for the conversion of authorization roles terminates with an error if the authorization object S_RS_DTP is to be converted. The problem is only visible in the last phase of the conversion (during the generation of the target roles), but the cause of the error is already in the initial run. The problem was solved with SAP Note 3572789, but occurs again as a side effect of the correction of SAP Note 3564220.

Other Terms

S_RS_DTP, ODSO, CUBE, RSTLDTPSRC, RSTLDTPTGT

Reason and Prerequisites

This problem is caused by a program error.
