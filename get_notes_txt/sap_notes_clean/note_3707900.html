Symptom

You try to update SV represented as Classic BOM with Model Unit Effectivity by deleting Substitutes that have a Reference Designator from a BOM component. Or you update SV component with Reference Designator by changing MUE parameter and quantity. Despite a successful response, you may find incorrect entries in sub-item Key Mapping, that may subsequently cause errors when updating sub-items.

Other Terms

PLMSI, PLM System Integration, SV, Structure Version, SVasSAPClassisBOM, Subitem, MUE, Reference Designator, Key Mapping, KM

Solution

Implement this SAP Note to fix an incorrect Sub-item Key Mapping handling.
