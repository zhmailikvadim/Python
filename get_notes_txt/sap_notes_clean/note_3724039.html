Symptom

A SMGW is assigned to multiple meters, if change SMGW via MOS Device Change process, MeLo installations at meters can't be filled in Installation Reply, it leads a wrong result that not all parameterization instances are selected, SMGW are not changed in all MeLos and MaLos.

Other Terms

MOS device change, DE_B_IM_DEV_CHANGE, installation reply

Reason and Prerequisites

Program error

Solution

Please implement the attached SAP Note or import higher version of Support Package.
