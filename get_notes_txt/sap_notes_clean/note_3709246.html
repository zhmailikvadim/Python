Symptom

This KBA explains how to display the Tax Type (Obsolete) field for Chinese employees in Infotype 0531 within HR Renewal. "Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Environment

  • Human Capital Management (HCM)
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP SuccessFactors Employee Central Payroll
  • SAP S/4 HANA

Resolution

By default, the Tax Type (Obsolete) field is not displayed in HR Renewal. To enable the display of this field in HR Renewal, customers can maintain and define field properties in V_T588MFPROPC. Refer to the following steps.

  1. Run transaction code SM30 and edit view V_T588MFPROPC.
  2. Enter infotype 0531.
  3. Maintain the field EETYP as below and then click 'save'. 
    • Version:28
    • Structure:P0531
    • Field Name: EETYP


Then the "Tax Type (Obsolete)" field is displayed in HR Renewal.
