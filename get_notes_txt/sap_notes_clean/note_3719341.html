Symptom

SDM /ACCGO/CL_SDM_ACM_T_PR_ASP_01 got executed to update the new fields of table /ACCGO/T_PR_ASP.
Updated values are found to be inconsistent.

Other Terms

Agricultural Contract Management; ACM; Pricing Aspect ; SDM /ACCGO/CL_SDM_ACM_T_PR_ASP_01

Reason and Prerequisites

Program error.

Solution

Import the relevant support pack or apply the correction instruction(s).

Utility required to update fields used in SDM /ACCGO/CL_SDM_ACM_T_PR_ASP_01, so that required records will be picked up and corrected in re-execution of same SDM .

Following updates are made in table /ACCGO/T_PR_ASP using this utility.
1. Reset sdm_version field to 01.

2. Reset /ACCGO/T_PR_ASP fields which were introduced in 2022 release.( for ex.  accgo_prc_fix_status , future_amount, basis_amount)
