Symptom

You want to create or change a Vehicle Scheduling Optimization Profile (transaction /SAPAPO/VS_OPT2).

When you want to add or change select options for Vehicles on the 'Resources' tab and enter a pattern (i.e. a selection containing the character '*') for the resource name, the system incorrectly shows error message 'Resource &1 does not exist' (message /SAPAPO/VS 014).

Other Terms

/SAPAPO/VS_OPT2, /SAPAPO/VS 014

Reason and Prerequisites

You have implemented SAP note 2251070 or the corresponding support package.

You have not activated Simplified Blocking and Deletion of Customer / Vendor Master Data in SCM (see SAP note 2007926 for further details).

Solution

Implement the attached correction instructions or import the specified Support Package.
