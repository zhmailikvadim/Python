Symptom

For Standing Orders items the Product Description is duplicated.

Environment

SAP Ariba Guided Buying

Reproducing the Issue

  1. In Guided Buying, open an item configured as a Standing Order from the catalog.
  2. Observe that the Product Description appears twice on the item details.

Cause

Resolution

A code correction is expected to be delivered by Engineering in April.
If the behavior persists after the code correction, please contact Support and reference this KBA so we can continue the investigation.
