Symptom

Transaction code MM06 (Flag material for deletion) is not supported in the SAP S/4HANA Cloud Public Edition due to its cloud-first architecture.

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Environment

SAP S/4HANA Cloud Public Edition

Cause

Traditional SAP GUI transactions like MM06 are not supported in the cloud-first architecture of S/4HANA Public Cloud. The public cloud follows a Fiori-based user experience model, eliminating direct access to classic SAP GUI transactions.

Resolution

Refer to the following steps of Solution for the "Flag material for deletion": 

  1. Access Manage Product Master Data app;
  2. Search for the material to be flagged for deletion;
  3. Edit the material record;
  4. Navigate to the General Data tab;
  5. Set the deletion flag indicator;
  6. Save the changes.
