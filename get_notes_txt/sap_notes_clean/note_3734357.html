Symptom

Why I am unable to search a supplier in Supplier Management though it's available in the CSV file exported from SM Administration.

Environment

  • Product: SAP Ariba Supplier Lifecycle and Performance
  • Supplier Lifecycle & Performance Import/Export

Cause

If the Supplier Request is not fully approved, SM Vendor will be created in the SM database and will be present in CSV export file but not be searchable in the SM.

Resolution

  1. Verify the approval status of the supplier request project in SLP.
  2. If the supplier request project is not approved, complete the necessary steps and approve the project so the supplier becomes searchable.
  3. If the supplier is intentionally left unapproved, note that the supplier will remain not searchable by design.
