Symptom

You want to implement SAP Note 3725290 - and this Note provides all prerequisite objects - the objects must be installed before the Note 3725290 is applied.

Other Terms

report NOTE_3731129

Reason and Prerequisites

This Note is prerequisite for Note 3725290.

Solution

As a general rule SAP recommends that you implement the Support Package specified for your release. However, if you need to install a solution earlier, use the SAP Note Assistant to implement the correction instruction. You can find more information about the SAP Note Assistant in SAP Support Portal, under SAP Note Assistant.

Implement automatic correction instructions and follow manual instructions. After you apply this Note, you can apply Note 3725290.
