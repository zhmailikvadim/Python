Symptom

An API is required that lets you create Clinical Sales Orders that can be consumed by various interace frameworks (not limited to OData API).

Other Terms

ICSM, Intelligent Clinical Supply Management, Distribution, Sales Order, Interface

Reason and Prerequisites

You use SAP Intelligent Clinical Supply Management.

The only available interface is the SO OData API, which can be called externally to create clinical sales orders.

Solution

Implement the corrections in this note and call the public method CREATE_SALES_ORDER_EXTERNAL of class /CTCO/CL_API_SAL_ORD_DPC_EXT in the custom application to create clinical sales orders.
