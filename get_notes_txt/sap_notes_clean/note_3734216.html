Symptom

The content of the Bank Account Contract ID in the Balance Key Figures business event data was incorrectly mapped to the IBAN of the contract

Other Terms

S4TRBK 2.0, BACID, IBAN, Balance Key Figures, Business Events, BKF

Reason and Prerequisites

Program Error

The SAP Note 3728490 is a pre-requisite for this Note.

Solution

Import the relevant support package or use transaction SNOTE to implement the corrections in advance.
