Symptom

In Payroll Control Center - Manage configuration applications, when configure conditions in Data section, the value help can't display part of values for the fields.

Other Terms

PCC, dimension, search help, dataset

Reason and Prerequisites

The value help has a default max records limitation which is 10000 due to performance considerations. When the number of values of the fields is larger than 10000, part of values can't be listed in value help of fields.

Solution

Implement the corrections in your system or import the relevant Support Package.

After applying this SAP Note, the max records limitation is set to 100000.
