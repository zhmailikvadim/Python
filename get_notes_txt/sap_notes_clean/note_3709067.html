Symptom

  • Program RPM_DX_FIN_CAP_UPLOAD fails to upload data.
  • The Data Transfer Application Log is empty (no messages are displayed)

Environment

  • SAP Portfolio and Project Management
  • SAP Portfolio and Project Management for SAP S/4HANA
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition 

Reproducing the Issue

  1. Use program RPM_DX_FIN_CAP_UPLOAD to upload financial planning data.
  2. The program does not display any success or error messages (the Data Transfer Application Log is empty)
  3. The planning data is not uploaded.

Cause

The cell format of one or more of the columns in the excel sheet has been modified (e.g. the property "decimal places" of a "Number" field has been changed).

Resolution

  1. Utilize the functionality "Display DX Layout" to create a template Excel file.
  2. Do not change the format of any cells in the Excel file. All cell properties must be exactly the same as the template Excel file.
