Symptom

When calling a web-based application (ICF service), browser displays the following error message:

500 Internal Server Error
Message E S_PROFGEN_START_AUTH 000 cannot be processed in plugin mode HTTPS

Environment

  • ABAP platform
  • SAP NetWeaver

Cause

Missing authorization for the affected ICF service.

Resolution

Assign the authorization for that service to the user (e.g. assign a role that has this service to the user, or add the missing service to their current role as per KBA 3638313 - How to add an web dynpro service to a role).
