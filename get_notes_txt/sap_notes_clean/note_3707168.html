Symptom

  • An error occurs when creating or updating a contract in Flexible Real Estate Management (RE-FX).
  • The system displays the error message: "There are no objects assigned to contract XXXXXXXXXXXXX."
  • Technical error message: REXCIN208

Environment

  • Flexible Real Estate Management (RE-FX)
  • Contract and Lease Management
  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA Finance
  • SAP S/4HANA

Reproducing the Issue

  1. Create a contract using transaction RECN
  2. Attempt to save the contract.
  3. Observe the error message: "There are no objects assigned to contract XXXXXXXXXXXXX."

Cause

The error message "There are no objects assigned to contract XXXXXXXXXXXXX (Message no. REXCIN208)" is raised because not all required data had been provided when creating the contract, which is necessary for the GST functionality in Flexible Real Estate Management.

Resolution

  1. Ensure that a suitable RE-FX object, such as a business entity, building, or property, is assigned to the contract. These objects are necessary for deriving the GST data.
  2. Note that other objects like contract objects or cost centers are not supported for this functionality.
  3. Refer to SAP Note 2440811 for detailed explanations regarding the requirement for assigning a RE-FX object to derive GST information.

See Also

  • Refer to: SAP Note 2440811 - Goods and Services Tax (GST) - 2017 changes.
  • Refer to: SAP Note 2496879 - Adjusting or suppressing error messages.
