Symptom

This SAP Note addresses two issues related to separation pay:

  • The system calculates extra separation pay incorrectly when you run off-cycle payroll with Separation (0009) or Mid-separation (0011) reason for senior executives.
  • The (21)Payment Date field doesn't populate correctly when multiple Separation payment (0538) infotype records exist and you run The Separation Payment Tax Receipt (HKRSEPR0) report.

Other Terms

KRSEP, Extra Separation Pay

Reason and Prerequisites

This is a program error.

Solution

Apply the relevant HR support package or correction instructions in this SAP Note.
