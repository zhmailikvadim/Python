Symptom

How do I export customizations in SAP Ariba Buying through Intelligent Configuration Manager (ICM)?

Resolution

Please perform the below steps to download the customizations.

  1. Navigate to Manage > Core Administration (for Buyer).
  2. Click Intelligent Configuration Manager > Manage Configuration and then click Continue.
  3. Select Buying and Invoicing (for Buyer), then choose Field Configurations from the drop-down list.
  4. Identify and select the class for which the customizations need to be downloaded.
  5. Click on "Download" to retrieve the customization details.

Finally, extract the downloaded zip file.
This will have folders containing one Javascript Object Notation (JSON) files, which you can view in any text editor
