Symptom

You run payroll for a suspended or withdrawn employee. The system displays DSN-Val error message 51.011/CCH-16 in the monthly DSN related to this payroll:
"Vous avez déclaré une rémunération brute non plafonnée (type 001) différente de « 0.00 » pour une période incluse dans le mois principal déclaré sans avoir renseigné, sous le même bloc « Versement individu - S21.G00.50 » parent, une base assujettie de type « 03 - Assiette brute déplafonnée », « 11 - Base forfaitaire soumise aux cotisations de Sécurité Sociale », « 22 - Base brute spécifique » ou « 23 - Base exceptionnelle (Agirc Arrco) » rattachée au mois principal déclaré (S20.G00.05.005). Ceci n'est pas admis."

Other Terms

DSN, DSN-Val, CCH-16, RPLDSNF0

Reason and Prerequisites

The error occurs because block 78 code 02 or code 03 is absent. The specific missing code depends on how the system triggered retroactivity and which blocks (02 or 03) the system generated automatically.

SAP Note Version

Reason for the new Version

6

Following a synchronization error in SAP internal development system, the correction instructions for the release  S4HCMCFR 102 must have been reworked.
Important: if you are using this release, please install as stated in the new "post-implementation step" the SAP Note 3719520 that is fixing this synchronization error.

Solution

After implementing this SAP Note, the system generates the correct block 78 code 02 or 03 for suspended or withdrawn employees during payroll processing.

The correction in this SAP Note is part of the HR Support Packages referenced below.
If you need the correction before the HRSP is available, implement the attached correction instructions. Correction instructions are a short-term solution and not a substitute for the Support Package.
