Symptom

In the Manage Journal Entries app, the “Item Text” field appears populated in display mode, but disappears/clears when the user clicks Edit for a journal entry line item.

Environment

SAP S/4HANA Cloud Public Edition

Cause

When Edit is selected, the app switches from Ledger View to Entry View. The Item Text shown in Ledger View may not be displayed the same way in Entry View, so it appears as if the text was removed.

Resolution

In Edit mode, switch the journal entry back to Ledger View to display the same information as before:

  1. Open Manage Journal Entries

  2. Select the journal entry line item

  3. Click Edit

  4. In Edit mode, change the view selection back to Ledger View

Outcome
Item Text and other line item details will display as expected once Ledger View is selected again.
