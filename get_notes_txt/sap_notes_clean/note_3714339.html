Symptom

During execution of a CKMLCP step one of the following issues occurs:

  • The error message MLCCS-101 'Field overflow when determining the cost component split' is raised in function module FCMLA_REP_ONL_UPD_PREPARE.
  • When the CKMLCP step 'Revaluation of Consumption' is repeated, it updates wrong CCS data in table FCML_CCS_REP.
  • One of the CKMLCP steps is repeated. Some cost components to which values were allocated during the first execution do no longer receive values during the second execution. 
    In this situation the records for these obsolete cost components are not deleted from table FCML_CCS_REP.

Other Terms

FCML_REP; FCML_CCS_REP, Online Update

Reason and Prerequisites

Online update for reporting tables FCML_REP, FCML_CCS_REP as of note 1947357 is active.

Solution

Apply the correction.
