Symptom

Performance form titles are not getting translated for some forms and the 'for' word (PERFORMANCE_NEWDOC_FOR key) shows mixed languages in the Performance Inbox (My Forms page) and in table report.

Environment

SAP SuccessFactors Performance & Goals 

Reproducing the Issue

The issue can be reproduced by switching the system language to another enabled language and observing the form titles of performance forms.

The issue might not be present in all form templates.

Cause

1. The issue occurs due to the enabled option "Enable form title in Form Info page" in the form template settings. This option has a limitation that prevents the form titles from being translated.

2. Cache issue with the localization language files

Resolution

1. Disable the setting "Enable form title editability in Form Info page" in Form Template Settings.

2. Cache issue: 

  1. Select and save form in the affected language
  2. Log in again, or switch to a user whose default locale is set to the affected locale.

If there are multiple forms affected by this issue there is an opportunity to Open&Save templates in mass. 2076641 - Mass Update Live Performance Forms using Open&Save Documents - Performance Management

If none of the above solution solve the issue, please open a support case to disable the deprecated setting "Enable form title editability in the form" by the Engineering team.
