Symptom

Encounter the error FCO_COST_RATE 032 "Employment &1 must match the company code of the cost rate sender." when trying to update the rate in the app "Manage Cost Rates- Services". 

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Attempt to update rates in the "Manage Cost Rates - Services" application.
  2. Encounter error message: "Employment &1 must match the company code of the cost rate sender." 

Cause

The cost rate must start within the employment’s validity, i.e., in the same fiscal period. 
For example, if The employment is valid on Feb-01-2026, which falls in Fiscal Year Period 002/2026, but the current cost rate starts in period 001/2026, this is causing the issue. 

Resolution

Review the employment validity period associated with the service cost rate and ensure the cost rate starts within the employment’s validity period, specifically within the same fiscal period.
