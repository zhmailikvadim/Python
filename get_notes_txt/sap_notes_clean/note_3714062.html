Symptom

For a routing, you have made changes to the inspection characteristics for an operation using a change number. You have not released the change number.

When you create and release a new production order for the material of the routing from scratch, the characteristics for the inspection lot are created in sync with the release key of the change number.

When you create a new production order with an existing production order as reference, a different version of the characteristic is selected during order release.

Other Terms

Change number, routing, master recipe, inspection lot, order release, CO01, production order, COR1, process order, ECM_FREIGABE_SETZEN

Reason and Prerequisites

In case you create a new production order with reference and release it directly in the same transaction CO01, the function module ECM_FREIGABE_SETZEN is not called and a different version of the characteristic is selected during order release.

Solution

Implement the attached correction instructions.
