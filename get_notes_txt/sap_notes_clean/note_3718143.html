Symptom

In SAP Fiori Launchpad Content Manager, you navigate to a catalog in the Catalog Overview. When right-clicking on a tile entry to access 'Tile Parameters' or 'Target Mapping Parameters', the popup displays parameters from the previously selected tile instead of the tile that was right-clicked. This error only occurs when right-clicking on a non-selected line without first left-clicking it.

Other Terms

/UI2/FLPCM_CUST, /UI2/FLPCM_CONF

Reason and Prerequisites

Program Error

Solution

Implement the attached Correction Instruction (CI) or the relevant Support Package.
