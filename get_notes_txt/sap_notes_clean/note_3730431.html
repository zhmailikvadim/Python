Symptom

When you use the 'bxml-flattening' interface, the scaling factor defined in the BW query is used unexpectedly for the cell property 'Value'.

Other Terms

Scaling Factor Sign Reversal bxml VALUE FORMATTED_VALUE

Reason and Prerequisites

There is a program error.

The problem occurs only if a plus/minus sign reversal is active for the affected BW key figure. This is a side effect of SAP Note 3515144.
