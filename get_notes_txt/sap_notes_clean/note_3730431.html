Symptom

Component: Risk Management
Module: Risk Assessment

You edit risk assessments on the Manage Risk Assessment Projects app (App ID EHHSS_UI_RAS_ENTRY). You select the step Determine Initial Risk or Determine Residual Risk for a risk. The system encounters a runtime error. The ABAP Runtime Errors transaction displays the following error details:

Category ABAP programming error
Runtime Errors OBJECTS_OBJREF_NOT_ASSIGNED_NO
Exception CX_SY_REF_IS_INITIAL
Web-Dynpro-Component EHHSSWDC_RAS_OIF_RMATRIX

 

Other Terms

EHHSSWDC_RAS_OIF_RMATRIX

Reason and Prerequisites

Reason:
Program error.

Prerequisites:
For information about the validity of the corrections, see the correction instructions.

Solution

After implementing this SAP Note, the system no longer encounters the runtime error when you determine initial risk or residual risk in the Manage Risk Assessment Projects app.

The Support Packages section lists the Support Packages that contain these corrections.

Alternatively, you can implement the attached correction instructions.
