Symptom

Errors occur during the implementation of SAP Note 3697345.
Error messages indicate that changes to certain objects are forbidden, resulting in the termination of the note implementation.

Environment

SAP ERP

Cause

Resolution

Follow manual steps below:

1. Go to T-Code SE16, enter table name PROGDIR.

2. Enter object name (of the locked object) and execute.

3. Select change mode and change the field EDTX value from 'X' to space.

4. Save the changes.

5. Once the 'Edit Lock' is unchecked, continue applying the SAP Note.
