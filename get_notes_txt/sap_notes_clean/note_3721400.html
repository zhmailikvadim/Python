Symptom

You are working with Balance Sheet FX Risk hedge requests. It is not possible to provide a Fixing Date or settlement currency when you create the hedge requests for FX Non-Deliverable Forward instrument in T-code FXM_REQUEST, nor in the Fiori App Process Hedge Requests - Balance Sheet FX Risk. The created hedge request is later wrongly taken as a FX Forward request in the trading platform.

Other Terms

FXM_REQUEST, FXM_RP_SEL, CL_FXM_BSE_PARAM, CONVERT_BSR_PARAM, CL_FXM_BSR, CREATE_BSR, CREATE_TR, MODIFY_BSR, _CREATE_BSR_BY_SINGLE_PARAM, FXM_BSR_OUTPUT, FXM_REQUEST_SEL, I_BALSHTEXPSRHDGREQ, I_BALSHTEXPSRHDGREQTP, C_BALSHTEXPSRHDGREQTP, I_BALSHTEXPOSURESNAPSHOTTP, F4764, FXNonDeliverableFwdFixingDate, CashSettlementCurrency,

Reason and Prerequisites

Programming error.

Solution

Please implement the SAP Note or the Support Package.
