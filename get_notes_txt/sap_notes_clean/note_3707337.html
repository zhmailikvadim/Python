Symptom

You have to pay an employer payment to your employee for which no WW duty exists. Aof-contribution and WKO-contributions are however due. You receive error message Basis wage for WW, WAO/WIA, and SI must be equal; check wage type &1 (HRPAYNLLA_EN189). The error message should not be shown.

Other Terms

NLLA0

Reason and Prerequisites

The issue is caused by a too strict check.

Solution

Import the HR Support Package relevant for your release. For more information, see SAP ERP Human Capital Management Support Package Schedules or SAP S4 HCM Support Package Schedules.

As a general rule SAP recommends you to implement the Support Package. However, if you need to install a solution earlier, use the SAP Note Assistant.
