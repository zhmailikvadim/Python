Symptom

In some cases, the virtual register in RUC1 lacks initial meter readings even though a meter reading was provided for the base register.

This note provides additional log entries in the MOS device change process in case of missing meter readings for the virtual registers in an IMS for a newly installed DIR. This should allow a better tracing of the source of the problem.

Reason and Prerequisites

The initial meter readings for RUC1 come from the base registers' meter reading in the installation reply.

Solution

Implement the correction with /nSNOTE.
