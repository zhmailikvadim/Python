Symptom

During a client copy using SCCLN/SCC9N some of the tables are getting failed because of the error 'Dereferencing of the NULL reference'

Environment

SAP_BASIS 757 > 757 SP 08

Reproducing the Issue

  1. Perform a client copy operation using the SCCLN/SCC9N tool 

Cause

Note 3497667 version 7 

Resolution

  1. Implement the latest version 8 of SAP Note 3497667 - "New Client Copy: Missing data in tables if the system has 'Table Replicas'"

Keywords

SCCLN, SCC9N tool, dereferencing NULL reference, SAP Note 0003652777
