Symptom

Feature Request for ISQL when connecting to SAP IQ, to prompt to enter old and new passwords when a password is expired.

Please note:

  • ISQL is not an official client tool for SAP IQ
  • DBISQL is the official client tool for SAP IQ
  • Behaviors using ISQL and DBISQL to connect to SAP IQ might differ.
  • In case ISQL doesn't behave as expected, DBISQL should be used instead.

Environment

  • SAP IQ 16.1
  • SAP Adaptive Server Enterprise SDK

Reproducing the Issue

  1. Connect to SAP IQ server, either with ISQL or DBISQL as user DBA and run:
    CREATE USER user1 IDENTIFIED BY welcome;
    CREATE LOGIN POLICY LP1 password_expiry_on_next_login=ON;
    ALTER USER user1 LOGIN POLICY LP1;
  2. Connect with ISQL as user1 without =nogui
    isql -Uuser1 -P 'welcome' -SPRO21_FR_PRD_SIQ

    Msg 13361, Level 16, State 0:
    SQL Anywhere Error -1152: Password has expired
    CT-LIBRARY error:
    ct_connect(): protocol specific layer: external error: The attempt to connect to the server failed.

Cause

SAP CR 825953 has been logged to address this new feature request. Its description reads, "Feature Request for dbisql -nogui to prompt to enter old and new passwords when password is expired."

Resolution

  • There is currently no target SAP IQ Service Pack scheduled for the new feature request CR 825953.
  • It is being assigned to our Product Group.  
  • Please mark this KBA as a Favorite to receive an update when this enhancement is implemented.
