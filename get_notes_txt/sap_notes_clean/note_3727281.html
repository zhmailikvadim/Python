Symptom

  • Why the Approver does not have the Message Type Edit on SOW Response Approval Group?
  • User needs to be able to Edit the SOW when submitting the Response

Cause

Editor Message Type is not selectable at the Approval level for the SOW Response and SOW Revision Response Modules as it should only act on the Supplier or Integration Response. 

Resolution

If the Buyer User needs to Edit the Statement of Work it needs to be done under the Statement of Work or SOW Revision Approval Group.
