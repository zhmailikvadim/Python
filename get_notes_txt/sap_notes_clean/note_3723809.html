Symptom

  • An error occurs when attempting to change the cost center in the Asset Master Data in transaction AS02, displaying the message: “Fiscal Year XXXX is already closed in Financial Accounting.”
  • The system triggers recalculation of asset values for Fiscal Year XXXX, which is open in Asset Accounting (FI-AA) but closed in Financial Accounting (FI-GL).

Environment

  • FIN Asset Accounting
  • SAP ERP
  • SAP ERP Central Component
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA
  • SAP S/4HANA Finance
  • SAP S/4HANA Cloud Private Edition
  • SAP Fiori for SAP S/4HANA Finance,
  • SAP Fiori for SAP S/4HANA,
  • SAP Fiori for SAP S/4HANA Cloud Private Edition

Cause

The inconsistency between Asset Accounting (FI-AA) and Financial Accounting (FI-GL) for Fiscal Year XXXX causes the system to attempt recalculation of asset values for XXXX. Since FY2025 is closed in Financial Accounting, the system cannot process adjustments or validations for depreciation values, resulting in the error message “Fiscal Year 2025 is already closed in Financial Accounting.”

Resolution

  1. Open the Financial Accounting periods for Fiscal Year XXXX in transaction OB52.
  2. Alternatively, create a new time-dependent interval in AS02 with the new cost center effective as of fiscal year start in which asset accounting and FI periods is open (like for example 01/01/2026)
