Symptom

When you display the infotype SI Additional Insurance (0079) in HR Renewal, the checkbox Change by ELStAM  is always initial. In addition, it is not a display field, but can be edited.

Reason and Prerequisites

The value of the checkbox is not transferred to HR Renewal and is therefore always displayed as initial there.

Solution

Implement the attached correction instructions or import the specified support package. After you import the HR Support Package or implement the correction instructions, you do not have to perform any further follow-on activities.

After you implement the corrections, the value of the checkbox is transferred to HR Renewal and displayed correctly. In addition, the field can no longer be edited, but is only a display field as required.
