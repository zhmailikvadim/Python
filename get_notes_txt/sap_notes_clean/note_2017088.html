Symptom

The checkbox "Enter Registration Number" can not be set in the activity sequences for Processing Trade and Scrapping. Therefore the registration number can not be entered in the customs declaration in these target procedures, although necessary.

Other Terms

Processing Trade, Scrapping, Enter Register Number, Process, Activity Sequence, /SAPSLL/LLEG_PROF02, RNOIN

Reason and Prerequisites

Program error

Solution

Apply the correction instruction attached to this note.
