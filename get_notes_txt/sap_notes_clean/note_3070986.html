Symptom

In the monitor for revenue accounting items, the two fields PL_ACCOUNT and GL_ACCOUNT are not differentiated; only the text 'G/L Account' is displayed. As a result, the column headers are not unique.

In addition, fields are missing in the structures for the revenue accounting items.

Other Terms

FARR_RAI_MON, FARR_RAI_CONF, FARR_PL_ACCOUNT

Reason and Prerequisites

The fields PL_ACCOUNT and GL_ACCOUNT use the same data element.

Solution

Implement the corrections or import the relevant Support Packages.

The following fields have been added:

XPREDOC: Indicator that describes that a revenue accounting item is the predecessor of another revenue accounting item (for all statuses if the RAI class category is Order Item).

PROC_TMSTP_UTC: Processing time of the processed revenue accounting item (for the main items of all revenue posting items with the status "Processed").

Important: Before you make these changes, you should delete all revenue accounting item classes, including the generated objects.

Also implement SAP Note 2015926.
