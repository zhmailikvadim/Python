Symptom

When using F3888 to create an 'Operational Maintenance' Planning Bucket with 'Recurrence: Once', Message No. EAM_MTPLNGBKT031 is raised

Environment

  • SAP ERP Plant Maintenance (PM)
  • SAP S/4HANA, on-premise
  • SAP S/4HANA, Cloud Private Edition

Reproducing the Issue

  1. Open the 'Manage Maintenance Planning Buckets' App
  2. Create
  3. Type: Operational Maintenance
  4. Recurrence: Once

Cause

Improper User Handling

Resolution

The information within Message No. EAM_MTPLNGBKT031 explains this:

  • You have defined this maintenance planning bucket as a one-time planning bucket by setting the recurrence to Once.
  • However, you have selected a planning bucket type that only allows recurring planning buckets.
  • Planning buckets of the type Operational Maintenance, for example, support you in planning your day-to-day maintenance operations.
  • Therefore, maintenance planning buckets of this type are always considered as recurring planning buckets.

Review the documentation HERE for a better understanding in regards to creating Operational Maintenance Planning Buckets

Keywords

EAM_BKT_MANS1, Bucket type '&1' (&2) must recur more than once. Change the recurrence.
