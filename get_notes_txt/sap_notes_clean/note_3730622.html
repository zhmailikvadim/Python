Symptom

  • Need to postpone a scheduled release for a specific instance

  • Need to change or select a different release date for a specific instance

Environment

SAP SuccessFactors HCM Suite

Resolution

Releases are scheduled at the data center level and occur on the same dates for all customers. Postponing or rescheduling for individual instances is not supported.
