Symptom

  • Unable to reassign the Final Review task to another user in the Onboarding module.
  • Changing the person from the Process tab in Manage Data reverted back to the previous assignee upon saving.

Environment

SAP Successfactors Onboarding

Reproducing the Issue

  1. Navigate to Manage Data.
  2. Select Process and input the Process ID.
  3. Attempt to change the assignee of the Final Review task.
  4. Save the changes.
  5. Observe that the changes are reverted to the previous assignee.

Cause

Resolution

Whoever are the participants of the NHDR , they are the ones who will get this task assigned. If you would like this task to be reassigned, then you would need to restart the Onboarding Process, after updating the NHDR assignment rule.
