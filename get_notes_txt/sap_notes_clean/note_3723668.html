Symptom

After you implemented SAP Note 3498700, the content of table FIPTC_STAMP_TAX (maintenance view FIPTV_STAMP_TAX) has incorrect content for stamp tax 1.2 and 3.

Other Terms

SAFT, SAF-T, RPFIEU_SAFT, Portugal, SAF-T(PT), DDIC, Technical Note, Interface Note

Reason and Prerequisites

This SAP Note is relevant only for Portugal.

This SAP Note is a preparation SAP Note that provides DDIC objects for further developments.

If this SAP Note is required as a prerequisite for later fixes, you must implement it.

Solution

Import the Support Package specified for your release or implement the attached correction instructions.

As a general rule, SAP recommends that you implement the Support Package specified for your release. However, if you need to install a solution earlier, use the SAP Note Assistant to implement the correction instruction. You can find more information about the SAP Note Assistant in SAP Support Portal, under SAP Note Assistant.
