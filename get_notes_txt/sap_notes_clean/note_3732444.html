Symptom

  • The Planning Group field (MARC-SCM_RRP_SEL_GROUP) in MM02 (Advanced Planning tab) shows the message: "No values for this selection."
  • Manually entering a value (for example, 'SS') in the Planning Group field returns the error: "Enter a valid planning group."

Environment

  • SAP S/4HANA
  • Production Planning and Detailed Scheduling

Reproducing the Issue

  1. Open transaction MM02 and go to the Advanced Planning tab.
  2. Use F4 help for the Planning Group field (MARC-SCM_RRP_SEL_GROUP) and observe the message "No values for this selection."
  3. Manually type a value (for example, 'SS') in the Planning Group field.
  4. Observe the error "Enter a valid planning group."

Cause

The required planning group was not configured in the system, so the field has no selectable values and manual entries are rejected.

Resolution

  1. Start transaction SM30 and enter view /SAPAPO/RRP_GRP.
  2. Choose the Customizing option to switch to customizing maintenance.
  3. In the activity selection dialog, select the activity to maintain the view.
  4. Choose New Entries.
  5. Enter the planning group key and description (and any fields marked as mandatory), then save.
  6. Assign or create a customizing transport request and save.
  7. Verify that the planning group is now available in MM02 (Advanced Planning tab) via F4 help and accepts manual entry.
