Symptom

You run the Currency Translation task from the Data Monitor (CXCD) and you get an ABAP Dump like below.

ABAP Dump:
Category                      ABAP programming error                                                       
Runtime Errors             ITAB_DUPLICATE_KEY                                                           
ABAP: Program             CL_FINCS_CTR_CONFIG_PROVIDER==CP                                
Application Component  FIN-CS-COR

A row already exists with this key 

The current ABAP program "CL_FINCS_CTR_CONFIG_PROVIDER==CP" had to be terminated because it found a statement that could not be executed.

Environment

SAP S/4HANA Finance for Group Reporting

Cause

Configuration error.

Resolution

Check the Sequences, that you defined for the Currency Translation Method, that you use. When at least two Sequences are defined with the same Sequence Number, then the above mentioned ABAP Dump occurs.
The sequence numbers should be different/unique for all sequences in a Currency Translation Method.

Keywords

Data Monitor, ABAP Dump, ITAB_DUPLICATE_KEY, Currency Translation, Sequence, Sequence Number, CXCD
