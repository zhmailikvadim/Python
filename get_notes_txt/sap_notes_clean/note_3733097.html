Symptom

If several target values are delivered for the same key date and point in time (/BA1/C55BEFCH), but different buckets, some of the target values might not get the correct time effect assigned when applying the target value read sequence.

Reason and Prerequisites

This note is relevant for

  • Enhanced AFI Logic, formerly Accounting for Financial Instruments Based on Expected Cash Flows (ECF)
  • Methodology 950

Solution

Implement the attached correction instructions or the corresponding support package
