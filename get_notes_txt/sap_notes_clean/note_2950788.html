Symptom

The report RSBDELTAREPAIR cannot reduce the active field in RSBODSLOGSTATE for ODSOs because Processed_one and Processed_all have not yet been corrected.

Other Terms

RSBDELTAREPAIR, ACTIVE, RSBODSLOGSTATE, PROCESSED_ONE, PROCESSED_ALL.

Reason and Prerequisites

Program error

Solution

  • SAP BW 7.30
    Import Support Package 21 for SAP BW 7.30 (SAPKW73021) into your BW system. The support package is available once SAP Note 2849350 ("SAPBWNews BW 7.30 ABAP SP21"), which describes this SP in more detail, has been released for customers.

  • SAP BW 7.31 (SAP NW BW 7.3 EHP 1)
    Import Support Package 28 for SAP BW 7.31 (SAPKW73128) into your BW system. The support package is available once SAP Note 2925889 ("SAPBWNews NW BW 7.31/7.03 ABAP SP28"), which describes this SP in more detail, has been released for customers.

  • SAP BW 7.40
    Import Support Package 25 for SAP BW 7.40 (SAPKW74025) into your BW system. The support package is available once SAP Note 2927720 ("SAPBWNews NW BW 7.40 ABAP SP25"), which describes this SP in more detail, has been released for customers.

  • SAP BW 7.50
    Import Support Package 20 for SAP BW 7.50 (SAPK-75020INSAPBW) into your BW system. The support package is available once SAP Note 2942764 ("SAPBWNews NW BW 7.50 ABAP SP20"), which describes this SP in more detail, has been released for customers.

  • SAP BW 7.51
    Import Support Package 12 for SAP BW 7.51 (SAPK-75112INSAPBW) into your BW system. The support package is available once SAP Note 2887846 ("SAPBWNews NW BW 7.51 ABAP SP12"), which describes this SP in more detail, has been released for customers.

  • SAP BW 7.52
    Import Support Package 8 for SAP BW 7.52 (SAPK-75208INSAPBW) into your BW or AS ABAP system. The support package is available once SAP Note 2892448 ("SAPBWNews NW BW 7.52 ABAP SP8"), which describes this SP in more detail, has been released for customers.

    Import Support Package 05 for SAP BW 7.53 (SAPK-75305INSAPBW) into your BW or AS ABAP system. The support package is available once SAP Note 2896020 ("SAPBWNews NW BW 7.53 ABAP SP05"), which describes this SP in more detail, has been released for customers.

In urgent cases, you can implement the correction instructions as an advance correction.

To provide information in advance, the SAP Note(s) mentioned above might already be available before the support package is released. In this case, however, the short text of the SAP Note still contains the term "preliminary version".
