Symptom

Discount from Business Network copy order is not applied in SAP Ariba Buying

Reproducing the Issue

  1. Send a PO to the network using cXML that includes a line-level discount (for example, AdditionalDeduction of type Discount within UnitPrice/Modifications).
  2. Confirm the discount is visible on the network.
  3. Synchronize the PO to Ariba Buying (copy order scenario).
  4. Observe that the line discount information is not present in Ariba Buying.

Cause

The requested functionality is not available as per the product design. Discount information cannot be updated in Ariba Buying for copy orders.

Resolution

This behavior is expected by design. If there is a valid business requirement, an improvement request needs to be submitted.

Please refer How do I create an improvement or enhancement request for Ariba Procurement? for the steps to raise an improvement request.

See Also

This is documented in Taxes and Charges on Requisitions and Orders | SAP Help Portal.
