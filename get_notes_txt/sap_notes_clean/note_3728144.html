Symptom

  • A new reference category was defined and assigned to an external number range like 02.
  • When executing TBR6 and entering an external number an error is raised that the number range 01 is missing. 

Environment

  • SAP Financial Supply Chain Management
  • SAP S/4HANA
  • SAP S/4HANA Finance

Cause

Configuration

Resolution

  • Kindly note that the reference regarding the number range is hardcoded to 01. 
  • Therefore, it is not possible to use another number range like 02 within TBR6.
  • If an external number range needs to be used kindly make sure to use number range 01 and set the flag for external number range. 

Keywords

TBR6
