Symptom

When processing skip inspection lots, the automatic posting change from Quality Inspection Stock to Unrestricted-Use Stock in SAP Extended Warehouse Management (EWM) is not triggered.

This behavior occurs when an inspection lot is modified after its initial creation (for example, through a manual inspection plan assignment and inspection sample calculation). If these changes result in the lot status changing to a Skip, the expected communication to EWM for the stock posting change is not initiated.

Other Terms

 IOT, QIE, IV_EVENT_SKIP, decentral

Reason and Prerequisites

 Program error.

Solution

 Implement correction instruction.
