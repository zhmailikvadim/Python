Symptom

When attempting to create a Smart Insight in a chart after drilling down into the account dimension, the feature is greyed out with the message: "Smart Insights are not available after drilling down on account dimension."

Note: Smart Insights are not supported for table cells in Optimized stories 

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Environment

SAP Analytics Cloud 2026.2.6 

Reproducing the Issue

  1. Go to SAC;
  2. Open the Story;
  3. Drill down on the Account dimension in the chart;
  4. Try to use the Smart Insights;
  5. Observe that it is greyed out and showing the error message above:

Cause

Current product limitation: Smart Insights is not available after drilling down on the Account dimension in stories.

Resolution

A fix for getting insights on accounts in drill state is currently under validation to be delivered in the version 2026.09, which would be the after QRC2.
