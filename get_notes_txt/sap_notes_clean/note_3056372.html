Symptom

After applying the March 2021 Microsoft security update you experience issues when printing within SAP Financial Consolidation.

Restarting the instance(s) resolves the issue temporarily.

Environment

SAP Financial Consolidation (FC) 10.1

Reproducing the Issue

Apply the March 2021 Microsoft security update and try to print from the Web or Thick clients.

Cause

Microsoft identified an issue that affects Windows 10 devices which applied the March 2021 security update released March 9, 2021 and the out-of-band updates released March 15, 2021. Affected devices might receive unexpected results when printing from some apps or to some printers.

Resolution

Microsoft has released an out-of-band optional update and is now available on the Microsoft Update Catalog for all versions of Windows and on Windows Update.

For more information on this and a list of MS updates that will resolve the issue please visit, https://docs.microsoft.com/en-us/windows/release-health/windows-message-center#1575
