Symptom

  • I logged into my account and all of the workers are showing up as FN, LN.
  • Workers name are suppressed and their names aren't showing up, instead they're FN, LN.

Cause

Site configuration Suppress Worker Personal Details (does not include Profile Workers) is enabled. 

Resolution

To disable this configuration, you would need to run the Company Site Configuration Upload.

The upload will allow you to disable the configuration Suppress Worker Personal Details (does not include Profile Workers). 

If this connector is not enabled for your environment you can follow the How to Enable Connector via Configuration Manager document to enable it. 

See Also

  • How to Enable Connector via Configuration Manager
