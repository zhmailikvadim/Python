Symptom

  • The child-related wage types /607, /608, and /609 are not generated in the monthly payroll.
  • The child-related wage types /617, /618, and /619 are not generated in periodical Shoyo payroll.
  • The child-related wage types /627, /628, and /629 are not generated in non-periodical Shoyo payroll.

“Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.“ 

Environment

  • Human Capital Management (HCM)
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP SuccessFactors Employee Central Payroll
  • SAP S/4HANA

Reproducing the Issue

  1. Implement note 3709289.
  2. Execute monthly payroll/periodical Shoyo/non-periodical Shoyo calculation using RPCALCJ0.
  3. Observe that the wage types /607, /608, etc are not generated in the payroll result.

Cause

 The custom schema (Zxxx) does not contain rule JPS1 and JK12. As a result, the relevant wage types are not generated.

Resolution

  • Please review your custom schema (Zxxx) using transaction PE01.
  • Ensure that rule JPS1 and JK12 are assigned to your schema.
  • If JPS1/JK12 are missing, please add the rule at the appropriate position in your schema to enable generation of the required child-related wage types.
