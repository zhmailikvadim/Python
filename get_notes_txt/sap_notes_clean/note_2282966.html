Symptom

The SAP Business One 9.2 PL01 installation does not copy files for XcelsiusPortal, RFQ, CustomerCheckout and Mobile dashboard in Scenario_Supplements and the scenario features either do not work correctly, or do not work at all.

Reproducing the issue

Scenario:
With SAP Business One  9.2 PL01, you install the integration component (integration framework).

Expected Behavior:
The intstallation copies the information of XcelsiusPortal, RFQ, CustomerCheckout and Mobile dashboard file scenario packages to the target system.

Actual Result:
Files are missing for the XcelsiusPortal, RFQ, CustomerCheckout and Mobile dashboard scenario packages.

 

Solution

Temporary workaround:

  1. Create the Program folder in ..\Packages\B1 Integration Component\Technology.
  2. In the Program folder, create the ExternalContent folder.
  3. Before installation, copy the complete Scenario_Supplements folder from ..\Packages\B1 Integration Component\Technology\Content" to "..\Packages\B1 Integration Component\Technology\Program\ExternalContent.
  4. Install the Integration Component.

 The issue does not occur when installing SAP Business One 9.2 PL00 and PL02 and higher PLs.
