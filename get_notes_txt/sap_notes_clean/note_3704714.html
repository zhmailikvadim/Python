Symptom

  • The Custom Analytical Queries app in SAP S/4HANA is not functioning properly.
  • When attempting to preview queries, there is a 403 Forbidden message on the screen.
  • While checking in IWFND/ERROR_LOG, observe that the error message "The request URL or the metadata definition is incorrect" is displayed.
  • Then, A 404 error message, "Resource not found for the segment 'exception'," is observed when replaying the request via SAP Gateway Client.
  • However, the issue is not caused by missing authorizations, as confirmed by the user's security team.

Environment

  • SAP S/4HANA 2023 (onPrem system).
  • Custom Analytical Queries app.

Reproducing the Issue

  1. Open the Custom Analytical Queries app.
  2. Attempt to preview any query via clicking "Preview" button.
  3. In SAP GUI, use t-code IWFND/ERROR_LOG
  4. Encounter the error message: "The request URL or the metadata definition is incorrect."
  5. Replay the request via SAP Gateway Client to observe the 404 message: "Resource not found for the segment 'exception'."

Cause

The issue occurs because the required ICF service used by the Preview function in the Custom Analytical Queries app is not active. When the Preview button is executed, the system attempts to access this inactive ICF service, resulting in a 403 Forbidden error, followed by backend gateway errors (404 “Resource not found for the segment 'exception'”).

These errors are not authorization‑related but are caused by the missing activation of the relevant ICF node.

Resolution

Follow the steps below to activate the ICF Node / ICF Service:

  1. Log in to the SAP system and enter transaction code SICF.
  2. In the "Service Name" field, enter "FPM_BICS_OVP" and click "Execute".
  3. In the Virtual Hosts/Services tree, locate the service.
  4. If the service name appears in grey/darker text, it is inactive.
  5. Right-click on the service and select "Activate Service".
  6. Once the service is activated, retest the system. The app should now be accessible.

To know which ICF services is required for an app:

  1. Navigate to SAP Fiori Apps Reference Library.
  2. Search for the application name, then click on it.
  3. Navigate to "IMPLEMENTATION INFORMATION" tab.
  4. Expand the "Configuration".
  5. Find the "Technical Name" column, in the table below "SAPUI5 Application" section.

Note that: These technical names are the ICF nodes for the SAPUI5 application that must be activated on the front-end server.
