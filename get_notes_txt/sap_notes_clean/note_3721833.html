Symptom

The following Infotype 3267 relevant corrections are delivered in this SAP Note:

  • Display field (MVMEL) on the renewal screen for subtype A100
  • Corrections in related validations

Version information

Version 1: General availability

Other Terms

R/3 HR CEE, Hungary, IT3267, renewal

Reason and Prerequisites

The prerequisite for the correction:
HR-CEE/SAP_HR/S4HCM SP11/2025 - HR-CEE/SAP_HR/S4HCM SP03/2026 or the equivalent CLC package for Hungary.

Solution

Apply the Correction Instructions of the current SAP Note (3721833) with the Note Assistant (transaction SNOTE; more information on service.sap.com/note-assistant).

The modifications will be delivered with the HR-CEE/SAP_HR/S4HCM SP04/2026 package and the equivalent CLC package for Hungary.
