Symptom

In View All “There’s nothing to show yet” is appearing despite data having been entered. 

The data can be seen in the Summary Card and View History. 

Environment

SAP SuccessFactors HCM 

Reproducing the Issue

  1. Login to SuccessFactors. 
  2. Navigate to an employee profile on the Latest People Profile. 
  3. Enter data into a custom MDF card. 
  4. While some fields/values are visible in the summary card, when trying to review in "View All" mode the message “There’s nothing to show yet” appears. 
  5. When checking in "View History", all of the data can be seen without issue. 

Cause

This is a known issue. 

Resolution

This fix for this issue is planned to be deployed on  1H2026 release.

    For release timelines, please review SAP SuccessFactors Product Release & Road Map Information.
