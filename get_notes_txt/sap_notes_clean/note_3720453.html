Symptom

  • I would like to have the Site to be populated with a default value during Profile Worker (PW) creation
  • I prefer not to use the Site code field on the PW module, so this would help mitigate confusion

Resolution

Due to the current system design, the Site field cannot be locked on the PW creation form page.

If there is only Site associated to the Legal Entity (LE), for example LE 1 is associated to Site 1, then the Site 1 will be defaulted in the Site field.
