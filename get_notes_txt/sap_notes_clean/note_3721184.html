Symptom

Dynamic User List, an issue has been identified where modifying the Dynamic User List by removing the rules and republishing it results in the status remaining stuck on "Pending".

Environment

  • SAP SuccessFactors Work Zone

Reproducing the Issue

  1. Create and publish a new Dynamic User List.
  2. Observe that the status correctly changes to "Completed" as expected.
  3. Modify the Dynamic User List by removing the rules and republishing it.
  4. Observe that the status remains stuck on "Pending".

Cause

  • Engineering replicated the issue and confirmed it is a bug requiring a code correction.

Resolution

  1. Engineering has created a code correction to address this issue.
  2. When the condition is cleared, after the user list synchronization completes successfully, the member list will be emptied, and the status will be updated to "Completed".
  3. The fix will be available in version 2603B, with a release date of 16 April 2026.
