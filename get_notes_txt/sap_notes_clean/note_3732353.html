Symptom

  • Inbound ORDERS message is rejected with a negative APERAK.
  • The system does not detect an existing QUOTES reference.
  • Error message: QUOTES_REF_NOT_FOUND.

Environment

  • Product: SAP S/4HANA Utilities master data updates option for Germany
  • SAP S/4HANA Utilities Master Data Updates Option for Germany

Reproducing the Issue

  1. Open the relevant market communication cloud tenant.
  2. Open the affected cloud process for the inbound ORDERS message.
  3. Observe that a negative APERAK is sent with value QUOTES_REF_NOT_FOUND.

Cause

The UBD data does not provide /mp4g/ubd_data-supply_scenario[1]-request-pod_id, preventing derivation of the QUOTES reference.

Resolution

  1. Review the UBD data for the affected subscription process and locate the supply_scenario section.
  2. Add or maintain the required request-pod_id within supply_scenario.
  3. Reprocess or retest the ORDERS message and confirm that the QUOTES reference is found and no negative APERAK is generated.
  4. Refer to the process documentation for guidance on required UBD fields.
