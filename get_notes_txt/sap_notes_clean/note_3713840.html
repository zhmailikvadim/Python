Symptom

The Change Mode of Bank Account Master Data Distribution in outbound does not work in the SAP S/4HANA Cloud Public Edition 2602 parallel line.

Environment

SAP S/4HANA Cloud Public Edition

SAP S/4HANA Cloud Public Edition 2602

Cause

The Change Point Fields Configuration for bank account, house bank account, and house bank within Bank Account Distribution does not support for the parallel line in the SAP S/4HANA Cloud Public Edition 2602 release.

Resolution

If this feature is required, the Expert Configuration Guide must be requested and implemented in the parallel line.

See Also

Scope Item 34P - Treasury Workstation Cash Integration. 

Keywords

Change Mode, Bank Account Master Data Distribution, Parallel Line, Bank Account, House Bank Account, House Bank, Replicate by Replication Model
