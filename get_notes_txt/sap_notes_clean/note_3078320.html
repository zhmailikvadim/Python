Symptom

You use Change Control Management (ChaRM) to control your system software distribution.

You want to manage your Employee Central Payroll ECP landscape changes using ChaRM.

Environment

SAP Solution Manager 7.2

Resolution

These are the required steps to configure this scenario:

1. Connect the ECP landscape with the SLD of the SAP Solution Manager, call transaction RZ70 Registration in the System Landscape Directory, in the domain controller system of your ECP landscape and fill the RFC Connection to SLD section with this data:

Gateway Host: enter this communication string:  /H/saprouter@SAP/S/3299/H/saprouter@customer/S/3299/H/customer-host

  • saprouter@SAP -> like sr-amer.payroll.gtm.ondemand.com, please look for the saprouter@SAP information in attachment of SAP KBA 3281332 - Setting Up an SNC-Based SAProuter Connection for Employee Central Payroll Systems
  • saprouter@customer -> public IP of the customer saprouter
  • customer-host -> internal IP of customer host where the SolMan system is installed

Gateway service: sapgwXX, where XX is the instance number of the server where the SolMan system is running

Run the registration by clicking in the Start SLD Data Collection Now. You will see in the registration exit this information:

Used RFC Destination: SLD_UC

RFC call successful

2. Request via incident in component XX-INT-CLD-PAY-OPS these changes to the TMS settings of the ECP systems

  • The transport routes between the ECP systems need to be client dependent
  • TMS parameters for the ECP systems landscape:

Parameter NameValueGlobal
CTC1Yes
IMPORT_SINGLE_ONLY1No
IMPORT_SINGLE_STRATEGY1No
NO_IMPORT_ALL1No
WORKFLOW_STRATEGY0No

3. Logon to the SAP Solution Manager system and check in LMDB under Transport Domains tab, that the transport domain for the ECP landscape DOMAIN_XXX exists and the ECP systems are under the ABAP Systems tab.

You will see the systems under System in TMS column.

In Edit mode Create the technical systems for these TMS systems.

You will get the Technical systems of system type Application Server ABAP, that you will need to use when defining the Logical component group for ECP landscape in your Solution.

Then create the requested RFCs from the SAP Solution Manager system to the managed systems: TMW, LOGIN and TRUSTED connections.

 

Your ECP landscape can be managed via Change Control Management scenario like any other of your landscapes.
