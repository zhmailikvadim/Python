Symptom

The synchronization from Solution Manager to Signavio runs on error. The message is: 'Sync errored for ObjectType: PROCSTEP with error: Data property does not exist in the sourceObject for the Process Step'

Resolution

This message appears, when in the synchronization a scope is used, in which process step originals are missing. Please check the scope setup in Solution Manager: 

  1. Choose the respective scope in Solution Manager.
  2. In the Scope menu choose the option 'More'.
  3. Check the option 'Include Originals'
  4. Optionally, add to the scope 'Process Step Library' via 'Select Structures'.
