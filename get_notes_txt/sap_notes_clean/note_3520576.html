Symptom

When creating partner function using FM BAPI_BUPA_PFCT_ADDFUNCTION in S/4HANA, the dump error SYSTEM_ABAP_ACCESS_DENIED is raised.

Environment

  • SAP S/4HANA

Reproducing the Issue

  1. Create partner function data using FM BAPI_BUPA_PFCT_ADDFUNCTION.
  2. Dump error SYSTEM_ABAP_ACCESS_DENIED is raised.

Cause

The FM BAPI_BUPA_PFCT_ADDFUNCTION is specific to pure CRM scenario and not supported not in S/4 release.

Resolution

To create/update partner function in S/4HANA, please use the API CL_MD_BP_MAINTAIN instead.
