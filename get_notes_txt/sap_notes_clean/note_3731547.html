Symptom

You create wage notifications for employees. To do this, you execute the program Data Extractor for Uniform Wage Notification Procedure (ELM) (report: RPLELMC0, transaction: PC00_M02_LELM0) in version ELM 5.0 (document type: 0005 - "ELM 5.0 - PIV/EIV") and have selected the domain LAW/Tax on the selection screen.

You must create pension statements and have selected the value "3 - Internal XML Schema Validation" for Additional Checks on the selection screen. The system issues the following error messages:

  • Error during validation of XML at position 34.780 (The element <NumberOfSalaryCertificate> is not provided at this point. (Message no. HRPAYCH_ELM50, 060)
  • Error during additional check (003 "Internal XML Schema Validation")
  • Data extractor finished with errors

You must create wage notifications for employees and have selected the value "3 - Internal XML Schema Validation" for Additional Checks on the selection screen. The system issues the following error messages:

  • Error during validation of XML at position 3.572 (the element <Remark> is not visible here. Expected: <person>) (message no. HRPAYCH_ELM50, 060)
  • Error during additional check (003 "Internal XML Schema Validation")
  • Data extractor finished with errors
