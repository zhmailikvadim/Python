Symptom

A user-friendly error message is required when an exception occurs during execution, instead of displaying a technical or unclear error message to the end user.

Environment

  • Cloud Integration
  • BTP Integration Suite

Cause

This allows the system to present clearer and more meaningful error information during execution failures.

 
 
 

Resolution

The error message can be customized by using the placeholder ${exception.message} to dynamically display the exception details in a user-friendly format.
