Symptom

The transactions should be supporting sending of Business System ID, instead of Logical System ID, when the core data management setup is set to send Business System ID. To give this functionality for Quote Message Purchase Order, the following SAP Note is prepared.

Other Terms

Quote Message Purchase Order, core data management parameter, ARBCIG_MD_USE_BUS_SYSID

Reason and Prerequisites

Bug fixes

Solution

Please implement this SAP Note for successful transactional processing.
