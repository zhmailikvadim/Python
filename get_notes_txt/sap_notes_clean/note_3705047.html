Symptom

Symptom 1 - Authorizations

The API classes to start the following processes execute an authorization check on authorization object /US4G/IM with hard-coded division 01:

  • DE_B_IDEV_PARAM_PROFILE_CREATE
  • DE_B_IDEV_PARAM_PROFILE_CLOSE
  • DE_B_IDEV_SEND_WS_PARAM_PROF_CREATE
  • DE_B_IDEV_SEND_WS_PARAM_PROF_CHANGE
  • DE_B_IDEV_RESOURCE_CREATE
  • DE_B_IDEV_RESOURCE_CLOSE
  • DE_B_IDEV_DEVICE_PROFILE_CREATE
  • DE_B_IDEV_DEVICE_PROFILE_CLOSE
  • DE_B_IDEV_SEND_WS_DEVICE_CREATE

Symptom 2 - Check Web Service Result

If an error occurs in process DE_B_IDEV_CTB_INSTALL in the step to call a web service sub process and the step has to be repeated, the process may not continue properly as it may be checking the result in the cancelled start step.
