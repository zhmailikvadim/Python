Symptom

  • The “?” help icon for the field Effective Start Date is not visible on the latest People Profile, despite being displayed correctly in the Configuration UI.
  • The issue is also observed in the legacy People Profile 
     

Environment

SAP SuccessFactors HCM Suite

Reproducing the Issue

  1. Add help text on effective Start Date in the Object Definition
  2. Check Configuration UI and verify that "?“ Symbol and text exist.
  3. Navigate to people profile (latest or legacy)
  4. Edit the portlet and observe that the Effective Start Date is missing the help icon

Cause

The object definition only has the field name mdfSystemEffectiveStartDate, which means it was originally created with effective dating set to None.

Resolution

This is the expected behavior when the object was created with effective dating set to None initially.

To make the help text icon appear, you will need to follow below steps:

  1. Go To Configure Object Definition and edit the Object definition
  2. edit the field mdfSystemEffectiveStartDate
  3. change the name to "effectiveStartDate"
  4. On the Configuration UI select the Value Type resolver to be equal to "effectiveStartDate"
  5. Clear MDF cache as explained in KBA 2534674
