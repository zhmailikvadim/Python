Symptom

In MM-SRV Scenario, a PR is transferred to SRM and it is available in Sourcing Cockpit. Later, the PR has been changed and an additional service item is added to the PR and Transferred again to SRM. When the Purchaser searches for the PR in SOCO Step-1, it only shows the newly added service item in the Result list. The service item that was present before the PR change is lost now.

Other Terms

SOCO, Result List, Sourcing Cockpit, PR Change, PS_HL_ITEM, PS_HANDLE_ITM, MAP_TO_PDO

Reason and Prerequisites

Reason : Program Error

Solution

Kindly apply the attached correction instructions or import to the corresponding support package.
