Symptom

  • Dump received when running the report RFWT0010.

    Runtime Errors: UNCAUGHT_EXCEPTION
    Exception Class: CX_RITA_APP_LOGGER_ERR
    ABAP Program: CF_TAX_PROPOSAL_FACTORY=======CP
    Application Component: FI-TXL-TGM
    Except.: CX_RITA_APP_LOGGER_ERR

Environment

  • Product: SAP S/4HANA Cloud Private Edition
  • Withholding Tax (Calculation)

Reproducing the Issue

  1. Run the standard report RFWT0010.
  2. Observe that a runtime dump occurs with UNCAUGHT_EXCEPTION in program CF_TAX_PROPOSAL_FACTORY=======CP.

Cause

Resolution

  1. Open transaction SLG0 (Application Log: Object/Subobject).
  2. Verify that Object FIGL exists.
  3. Verify that Subobject FIGL_WHT_0010 exists with description “Adjustment of Withholding Tax Information to Relevant Types”.
  4. If Object FIGL or Subobject FIGL_WHT_0010 are missing, create them accordingly in SLG0.
  5. Re-run RFWT0010 and confirm that the dump no longer occurs.
