Symptom

When using SSAM, application may not allow a user to re-onboard after resetting. App will be stuck in a completely white screen or a white screen with a "Loading" message. This generally occurs if there is a sync running during the reset.

Other Terms

  • White Screen
  • Loading Screen
  • App Unresponsive
  • App Reset

Reason and Prerequisites

  • BAS/Visual Studio Code with SAM MDK Project
  • `patch` command (available on BAS and macOS by default) via terminal or other patching utility
  • Read the guidelines on how to apply the patch and read the appendix section if you receive any error

Solution

  • Find attached file 3720331.patch
  • Save patch file into known location
  • Run command in base metadata folder: `patch -p1 < /path/to/3720331.patch`. Validate 1 file changed and no hunk failures.
  • Republish application
