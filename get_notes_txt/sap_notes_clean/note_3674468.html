Symptom

It is not possible to define global settings for 'Create Billing Documents' Fiori Application (F0798) in the 'Billing settings' screen (accessed via the Billing Settings button).

Environment

  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Cause

There is no central configuration to set a default value for billing setting as this is an in-app setting.

Resolution

This is the standard design.
However, if a user has set a billing setting, the same billing setting will be cached, and the user does not need to set it again next time.

See Also

SAP KBA 3388930 - Fiori app Create Billing Documents (app ID F0798) doesn't create the accounting document for billing document automatically
