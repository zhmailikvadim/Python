or

Error in EDOC_COCKPIT while submitting invoice to KSeF:

The exchange of electronic documents or report submission failed while executing the IFLOW_HTTP_COMMAND action because of an unexpected error in the process orchestration in SAP Document and Reporting

Environment

  • SAP ERP
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition
  • SAP Document and Reporting Compliance Cloud Edition

Reproducing the Issue

  1. Execute EDOC_COCKPIT
  2. Select an eDocument from Poland KSeF Invoice
  3. Submit

or

  1. Schedule a job to pull invoices from KSeF
  2. Open Resolve Errors and check error for SUPPLIER_INVOICE (B2B_RETRIVAL)

Cause

Certificate Issue

Resolution

This issue occurs when the Certificate has insufficient authorizations to access the API endpoints of KSeF.

Check with KSeF app and re-upload the certificates, if it still not working, then contact KSeF as this issue is with the Certificate and not SAP iFlows or DRC.
