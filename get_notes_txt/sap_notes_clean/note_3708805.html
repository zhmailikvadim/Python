Symptom

You open a BC Set in ADT Editor and the BC Set contains Customizing Objects which has forbidden fields (not supported data types). The Field Attribute value of such fields are shown as 'Default Field Value' instead of 'Forbidden Field'.

Other Terms

SCP1, AFF, IF_AFF_SCP1_V1, LOC, SCPR

Reason and Prerequisites

The constant for forbidden field was not included in the ABAP File Format.

Solution

Implement this Note.
