Symptom

The navigation from a source FPM/ WD application to a target application by means of the ABAP Report Launchpad fails due to a malformed URL that has been created as the result of an erroneous parameter mapping.

Other Terms

ABAP Report Launchpad, CL_APB_LPD_START_APPL_BROWSER, CL_APB_LPD_START_APPL_NWBC_3, CL_APB_LPD_START_APPLICATIONS, escaped URL

Reason and Prerequisites

Program errror.

Solution

Implement the attached corrections or import the relevant Support Package.
