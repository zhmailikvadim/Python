Symptom

  • Users are unable to add the Exasol JDBC driver under Manage JDBC Material → JDBC Drivers.
  • Attempts to upload different versions of the Exasol JDBC driver do not result in a successful deployment. Additionally, no specific error message or error code is displayed during the upload process.

Environment

  • SAP Integration Suite
  • Cloud Integration

Resolution

The Exasol database is not part of the list of databases supported by the JDBC adapter in SAP Cloud Integration.

As a result, installing or deploying JDBC drivers for unsupported databases is not supported.

For the complete list of supported databases, refer to the SAP Help Portal documentation on Configure JDBC Drivers for Cloud Integration.
