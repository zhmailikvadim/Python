Symptom

You run the System Measurement using transaction USMM to determine the number of users and chargeable objects for the Engines in the SAP software. The measurement result for Engine "S4H Human Resources" is incorrect. Application ID 0001 (Core HR: Employee) is not measured correctly.

Other Terms

USMM, USMM_OLD, RSUVM007, RSUVM008, RSUVM077, MANDT

Reason and Prerequisites

Reason: The system performs the measurement only for the current client. In the subquery, the SELECT statement does not include the USING CLIENT addition.

Solution

After implementing this SAP Note, the system measures Application ID 0001 (Core HR: Employee) correctly for Engine "S4H Human Resources". Implement the next Synchronization HRSP available in May 2026 or implement the correction instruction provided in this SAP Note.

A new version of the correction instruction has been created: customers who implemented already the previous version of this SAP Note have to implement the new version of the SAP Note (new correction instruction).
