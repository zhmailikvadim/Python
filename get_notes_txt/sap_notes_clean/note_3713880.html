Symptom

  • "Launch Note Analyzer" option is disabled/greyed out in SNOTE.

           

  • KBA 3370112 - Note Analyzer GUI is missing after Note 3200109 implementation is followed.
  • SAP Note 3007273 - Note Analyzer integration into SNOTE is implemented and post manual activities are performed in the system.    

         

 "Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental" 

Environment

  • Note Assistant tool (SNOTE)
  • SAP NetWeaver AS ABAP

Reproducing the Issue

  1. Enter transaction "SNOTE"
  2. Click on "Goto" in Menu bar and select "Other Tools"
  3. Check for "Launch Note Analyzer" option

Resolution

1. Perform the below pre manual steps and implement SAP Note 3359322 - Note Analyzer: Improvements in the UI and Integration with Background Implementation of SAP Notes. 

1. Go to transaction SE41
2. Program: "R_SCWN_NA_NT_BRWSR"
   Status: "PF_1000"
3. Click on Change button
4. Expand "Application Toolbar" section and add the following at last
5. Function Code: "NT_ANL"
   Functional Type: "Normal Application Function"
   Function Text: "Note Analyzer"
   Icon Name: "ICON_ARROW_RIGHT"
   Icon Text: "Launch Note Analyzer"
   Info. Text: "Launch Note Analyzer"
6. Function Key: Ctrl-Shift-F1
7. Save and Activate the changes

2. Relaunch SNOTE transaction to reflect the changes.
