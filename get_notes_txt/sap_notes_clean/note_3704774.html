Symptom

  • How to Disable Commenting in Hub

Environment

Commenting for SAP Signavio

Reproducing the Issue

  1. Share a process diagram with a new user by entering their email address in the share function within SAP Signavio Process Manager.
  2. The user receives a login link to access the diagram.
  3. The user receives a reset password link, even though SSO is enforced and a password is not required for login.

Cause

Resolution

To disable commenting globally,

  • go to Setup of Explorer in the SAP Signavio Process Manager

  • open Edit general Configuration.

  • Under Hub, users will no longer see the comments icon on top right corner
