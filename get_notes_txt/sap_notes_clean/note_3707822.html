Symptom

Using Calculated Columns for Date Format Conversion

Environment

SAP SuccessFactors HCM Suite

Resolution

Using Calculated Columns for Date Format Conversion

Step 1: Create a Calculated Column

  1. Navigate to your Story Report in SAP Analytics Cloud
  2. Go to Query Designer and select Data → Create Calculated Column 
  3. Select Calculated Column → Add

Step 2: Apply Date Conversion Formula

For converting dates to numeric format (DDMMYYYY), use this formula structure:

1CONCAT(
2  RIGHT(CONCAT("0", TOTEXT(DAY([FIELDNAME]))), 2),
3  CONCAT(
4    RIGHT(CONCAT("0", TOTEXT(MONTH([FIELDNAME]))), 2),
5    TOTEXT(YEAR([FIELDNAME]))
6  )
7)

For YYYYMMDD format, use:

1CONCAT(
2  TOTEXT(YEAR([FIELDNAME])),
3  CONCAT(
4    RIGHT(CONCAT("0", TOTEXT(MONTH([FIELDNAME]))), 2),
5    RIGHT(CONCAT("0", TOTEXT(DAY([FIELDNAME]))), 2)
6  )
7)
