Symptom

  • SAP HANA Cloud released Multi-Region DisasterRecovery feature on 2025.28.8 (QRC 3/2025)
  • You would like to know the supported scope.

Environment

SAP HANA Cloud QRC 3/2025 and QRC 4/2025

Resolution

Multi-Region DisasterRecovery feature (cross-region replication) for SAP HANA Cloud instances is currently supported only between AWS US10 (US East/Northern Virginia) and AWS US11 (US West/Oregon).

*There is plan to support AWS EU10(Europe/Frankfurt) & AWS EU13(Europe/Milan) in 2026.Q1. Plans are in place to gradually add support for other IaaS services and regions in the future.

For SAP HANA Cloud instances in other regions not get supported yet, you may consider enable High Availability (HA) and Disaster Recovery (DR) options within the same region to ensure resilience.

  • High Availability: Single-zone synchronous replication.
  • Disaster Recovery: Multi-zone asynchronous replication.
