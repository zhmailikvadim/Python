Reason and Prerequisites

The function for changing the packaging material can be enabled; however, its usage is strongly restricted to exceptional or urgent cases only.

When the packaging material is changed for a TU, all dependent parameters (for example, dimensions, capacity, and related attributes) are updated automatically based on the newly assigned packaging material. Incorrect usage of this function may lead to inconsistencies in logistical processes.

After the change is executed, the system sets the indicator "Packaging Material Changed" for the affected TU. This indicator is irreversible and cannot be reset.

The updated packaging material is subsequently considered in all Shipping and Receiving processes and is displayed in the TU overview.

The system provides full traceability of changes. Information about the user and timestamp of the modification is available in the "Change Documents" tab, which contains the complete change history.

Customizing

Path in customizing: Yard Logistics -> Master Data -> Setting up the Yard -> Define Yard Numbers

By default, the Customizing setting "Changing Packaging Material Allowed" is set to "No". Activation of this setting should be carefully evaluated according to business requirements.

Solution

Implement the note.

In case you need to allow changing Packaging Material, in customizing for Yard Logistics choose: Master Data -> Setting up the Yard -> Define Yard Numbers
