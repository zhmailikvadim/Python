Symptom

  • An error occurs when attempting to concatenate Measure based dimensions in Story designer Builder 
  • Formula could not be computed: Choose a property for dimension is the error encountered when performing any action

Environment

  • Story Designer - Filter, Calculated Column

Reproducing the Issue

  1. Create measure-based calculations in the Story Designer page
  2. Convert these measures into dimensions using Measure-Based Dimension, and
  3. Then attempt to concatenate the resulting dimensions with the CONCAT function.

Cause

Calculation design issue: dimensions represent objects rather than text or numeric values, requiring explicit reference to their properties (e.g., ID or Description).

Resolution

    Ensure that each dimension explicitly references one of its properties (e.g., [p/ID]) to retrieve valid values and avoid validation errors, like below 

    CONCAT([d/"X Dimension"].[p/ID], CONCAT(" " ,CONCAT([d/"Y Dimension"].[p/ID] , CONCAT(" " ,[d/"Z Dimension"].[p/ID] ) ) ) )
