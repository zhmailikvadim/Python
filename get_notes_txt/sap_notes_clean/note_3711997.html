Symptom

RE-FX contracts are managed with valuation and integration to Asset Accounting.

Error message RECABC010 is raised for processes that involve lease assets connected with RE-FX contracts, specifically:

  • Simulation or display of a valuation step (transaction RECN/RECECN)
  • Opening a lease asset with the Asset Explorer (transaction AW01N)
  • Depreciation run  in Asset Accounting (transaction AFAR)
  • Open a new fiscal year (transaction FAGLVGTR)
  • 0FI_AA_20 (FI-AA: Transactions and Depreciation)

Environment

  • Flexible Real Estate Management (RE-FX)
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA Finance
  • SAP S/4HANA

Reproducing the Issue

  1. Execute transaction RECN/RECECN valuation simulation, AW01N or AFAR.
  2. Observe error RECABC010 
    1. Entry  (Currency Translation Rule) is not in table TIVCACTRULE.
    2. Entry INDXCN (Additional Data for Adjustment Rule TASK) is not in table TIVAJSPECTASK.
    3. Valuation Rule and Company Code does not exists 

Cause

  • This message is related to a program issue and requires the implementation of a note to correct the code.

Resolution

  1. Implement SAP Note 3644330- RECEEP: Runtime error SYSTEM_SHM_CHANGE_LOCK_ACTIVE.
  2. Although the note 3644330 does not explicitly mention this error, It was verified that this correction should solve the issue. 
  3. Implement and test the implementation in a test environment to ensure the resolution addresses the issue.
