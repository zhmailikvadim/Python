Symptom

In a Price Simulation new proforma, it is expected that the mapped value for a material is displayed as the correct value based on the new proforma's effective date using price review date. However, the system is using a different value from a prior effective date of price review document.

Other Terms

/IRM/IPMXMPM, /IRM/IPPDM, /IRM/IPMXM, /IRM/IPMXDM, Price review, Price Simulation, Proforma, Pricing Date, Effective Date.

Reason and Prerequisites

This is caused by program error

SAP Note#3710414 is available for ECC Releas

Please implement SAP Note#3573911 as a Prerequisite note for 2020V2, 2021, 2022, 2023, 2023 V2 Release

Solution

Please implement the correction instructions attached. These corrections are only valid if you have applied the Support Packs.

Release V/4 - 2020V2, 2021, 2022, 2023, 2023 V2, 2023 V3 and 2023 V3 Support Pack 1

Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object.
