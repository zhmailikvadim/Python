Symptom

You download the Consolidation Unit master data from the Import Consolidation Master Data app (App ID: F3924) and you see the "Company ID" field as empty, even if the "Company" field is filled in the Define Consolidation Units app (App ID: F4685).

Environment

SAP S/4HANA Finance for Group Reporting

Reproducing the Issue

1. Set the global parameters using the Set Global Parameters app or transaction CXGP.
2. Open the Import Consolidation Master Data app (App ID: F3924).
3. Download the Consolidation Unit master data from the Import Consolidation Master Data app (App ID: F3924)
4. In the downloaded file you see the "Company ID" field as empty, even if the "Company" field is filled in the Define Consolidation Units app (App ID: F4685).

Cause

Known program error.

Resolution

Please check, if the below SAP Notes are applicable to your release and support package level. If they are applicable, but not implemented yet, then please implement them to your system.

3420475 - XLSX download of FS Items, Consolidation Units and Groupstructure omits fields of form 'VALUE - TEXT', that contain consecutive spaces 

3328732 - XLSX - Export (download) condeneses consecutive spaces

After the note implementation, please also refresh the caches both on your backend and frontend system according to note 2319491.
