Symptom

For the eTransfer desktop application, SP03 patch 1 has been released. The technical name and release is "PLVAT_DESKTOP 100 SP03 Patch 1".

The single reason for the current update is:

  • Error in decryption of XML file

Other Terms

SAF-T Poland, PLVAT_DESKTOP 100, eTransfer, Standard Audit File, SAFT, SAF-T, Poland, Polska, Audit, JPK, Jednolity Plik Kontrolny, Legal Change, OECD, Tax Reporting, VAT, STANDARD AUDIT FILE F.TAX 1.0, SAP ERP, add-on for Polish SAF-T regulatory requirements 1.0, Release Note

Reason and Prerequisites

This SAP Note only applies to users who want to upgrade their existing installation of the eTransfer desktop application to the the latest version.

If you have not installed and configured the eTransfer desktop application yet, this SAP note is not relevant for you. Instead, you can start with a clean installation of SP03. Please refer to SAP Note 3357329.

Solution

Attention: The software and its documentation are only visible to customers who purchased the license for "SAP ERP, add-on for Polish SAF-T regulatory requirements 1.0".

1. Download the Software

Follow the steps below for downloading the software from the SAP Service Market Place:

  • Go to https://launchpad.support.sap.com/#/softwarecenter
  • Search for "PLVAT_DESKTOP 100"
  • Select maintenance software component "PLVAT_DESKTOP 100"
  • Select the "DOWNLOADS" menu item
  • Download the ZIP file

 

2. Upgrade the Software

  • Install application (extracting the downloaded ZIP) in new location.
  • Set required settings needed to connection (proxy and port if needed)
  • Application folders needs permission to create/read/execute.

It is possible to upgrade application by the following steps only if version is at least 3.0.0 .

  • Follow the steps below for upgrading the software:
  • Go to the folder where you installed the eTransfer desktop application
  • Copy the file "configuration.properties" to a safe location
  • Install the upgrade by extracting the downloaded ZIP file to the existing installation folder. When asked, confirm that you want to replace the existing content.
  • Copy configuration data from the file "configuration.properties" in your safe location to the new file in the installation folder.

 

3. Documentation

Follow the steps below for downloading the updated documentation from the SAP Help Portal:

  • Go to https://help.sap.com/plvat
  • Log on
  • Download the documentation
