Symptom

This KBA lists characters that can be used to Start, Split, Serialize, SetQuantity of SFCs and references another KBAs, to indicate further restrictions against SFCs with some of these characters in another SAP DM functionalities.

Environment

SAP Digital Manufacturing 2511

SAP Digital Manufacturing 2602

Cause

 SFCs can be started, split, serialized and quantity can be set if they have the following characters:

  • Numerals 0 to 9
  • Alphabets A to Z (uppercase only)
  • Following allowed special characters:
    • _
    • $
    • !
    • )
    • (
    • +
    • ~
    • @
    • ^
    • =
    • -
    • *
    • .

Resolution

Lowercase characters, leading and trailing spaces and any other special characters should not be used for creating SFCs.

Excluding Scrap, Relabel and Invalidate SFC APIs, already affected SFCs with not listed characters will not be allowed in the other APIs related to SFC execution.

If you already have SFCs with the characters that are not listed above, you can use Relabel API to continue with them but with the listed characters.
