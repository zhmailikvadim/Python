Symptom

After upload journal entries with Document Header Text, the document header text appears as "0000000 when viewed in Manage Journal Entries app.

Environment

SAP S/4HANA Cloud Public Edition.

Reproducing the Issue

  1. Open the Upload General Journal Entries app.
  2. Use the journal entry Excel upload template and upload the file with Document Header Text entries.
  3. Open the Manage Journal Entries app to review the posted items.
  4. Observe that the Document Header Text is not updated and displays as "00000".

Cause

The Document Header Text field in the Excel upload template was formatted as Date instead of General, causing the value to be converted (for example, to "00000") and not posted as intended.

Resolution

In the file upload to the Upload General Journal Entries app, change the format for the value entered in Document Header Text from Date to General.
