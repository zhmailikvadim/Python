Symptom

The SAP system does not include the modifications outlined in the most recent SARS BRS PAYE for the August 2026 release. To comply with the latest SARS BRS PAYE version V 25 0 0, this SAP Note provides the relevant system changes. Further details are described in regulatory change manager tool under BRS(PY): PAYE Employer Reconciliation Version 25.0.0, March 01, 2026.

Other Terms

RPCTCGW0, RPCTCGW1, RPCTCGW3, RPCTCDW3, MP014940, SARS Code 4150, SARS Code 4588, SARS Code 3100, Reason Code

Reason and Prerequisites

Legal Change.

This SAP Note is relevant only for South Africa localization.

Solution

After implementing this SAP Note, the system updates the RPCTCGW1, RPCTCGW3 and RPCTCDW3 includes and the Module pool program MP014940.

As a general rule, SAP recommends that you install the solution by implementing the Support Package specified for your release.

However, if you need to install the solution earlier, please implement the correction instructions using the SAP Note Assistant. You can find more information about the SAP Note Assistant in SAP Support Portal, under SAP Note Assistant.
