Symptom

You use Focused Build Test Steps together with Test Steps Parameters to design and execute manual test cases with SAP Solution Manager.

In the Test Steps Parameters, you can only view the first 100 Test Plans under Test Plan Assignment, even if there are more than 100 Test Plans.

Other Terms

Test Steps Parameters,Test Plan,Test Plan Assignment

Reason and Prerequisites

This issue occurs due to a program error.

SAP Note Validity: ST-OST SP16

Solution

Please implement this SAP Note to solve the described issue.

After the note implementation, please make sure to reset the application index: execute report /UI5/APP_INDEX_CALCULATE via transaction SE38, go to tab "Single SAPUI5 Repository only", enter "/SALM/TM_TS_PAR" as repository and execute the report.
