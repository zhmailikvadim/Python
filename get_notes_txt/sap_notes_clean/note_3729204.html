Symptom

This KBA mentions several points to note when implementing ABR JAR file following Note 2705107 - Recommendation on generation of KeyStore (JKS) file for SCPI."Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Environment

  • Human Capital Management (HCM)
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP SuccessFactors Employee Central Payroll
  • SAP S/4HANA

Resolution

  • When the Java SE version is higher than JavaSE-9, make sure check box "Create module-info.java file" is not selected. Refer below screenshot.
    Or use JavaSE-1.8.
  • There are 3 Jar files in the note. Make sure Keystore Java Project.jar is imported first into src folder.
    After importing, the src folder should contain below documents:
  • Make sure both abrakm.jar and AUSkey-dep.jar are imported by Add External JARS.
    After importing, the 2 Jar files should appear in Referenced Libraries.
