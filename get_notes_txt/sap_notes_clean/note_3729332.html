Symptom

If a default value is set for an attribute in the file template configuration while other attribute values are provided through the upload file, the system incorrectly overwrites all attributes with the default value during upload.

This SAP Note ensures that the attribute values are correctly updated in the list document.

Other Terms

List Upload, Customer list, File template, Default values, /IRM/EPKULFTPM, /IRM/EPKULM,  /VTA/CL_BFTP_SERVICES, SECTION_INIT_LINE_APPEND

Reason and Prerequisites

This is caused by program error

Solution

Please implement the correction instructions. These corrections are only valid if you have applied the following Support Packs.

Release V/4 – 2021, 2022, 2022 V2, 2023, 2023V2, 2023 V3, and 2023 V3 Support Package 1.

Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object
