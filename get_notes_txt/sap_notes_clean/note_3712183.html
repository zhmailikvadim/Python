Symptom

How to have Flexible workflow only available for PO based invoice

Environment

  • SAP FIORI
  • SAP S/4HANA

Cause

Flexible workflow could be controlled by app: Manage workflows for Supplier Invoice

Resolution

  1. Access app: Manage workflows for Supplier invoice
  2. Create/edit workitem for Supplier invoice
  3. in Start Conditions select the preconditions : "Invoice is based on Purchase Order"

Keywords

Manage workflows for Supplier Invoice; Po-based invoice; F2873 ; Start Condition
