Symptom

In some cases, the "Why-Found" determination of the ESH search terminates with the search error "Search engine error: Error in federated search".

This affects Releases SAP_BASIS 740 and SAP_BASIS 750.

Other Terms

CALCULATE_WHY_FOUND / ENHANCE_WHY_FOUND_MATCHES

Reason and Prerequisites

Program error

The error affects model-based search connectors.

This affects value help search queries and search queries with "Why-Found" for attributes that have the namespace indicator "/<namespace>/..." in their names.

Valid for SAP_BASIS 740 (as of Support Package SP11) and for SAP_BASIS 750.
Higher releases are not affected by this error.

Solution

Implement the correction contained in this SAP Note.
