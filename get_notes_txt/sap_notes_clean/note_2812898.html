Symptom

In the menu for Retail Fuel Network Operations (RFNO), there is an entry for the display of the external ID, which the incorrect description text "Define External ID".

The definition of the location IDs is not maintained here but in transaction O402 (Change Location).

Other Terms

RFNO, /ICO/RN_EXTLOC, external location ID, Retail Fuel Network Operations

Solution

You cannot implement this SAP Note. It is delivered with service pack 5 for S4SCSD 1.0.
