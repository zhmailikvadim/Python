Symptom

  • Unable to upload document in BILaunchpad.
  • Get the error message : Not a valid logon token (FWB 00003).

Other Terms

FWB 0003, logon, token,fioriBI ,document , upload,BI ,

Reason and Prerequisites

Environment:

  • SAP BusinessObject Business Intelligence Platform 4.3 Support Pack 04 Patch 500.
  • All Operating System

Steps to reproduce:

  1. Login to BILaunchpad
  2. Click on Folders
  3. Click on + and Upload a Document

We have introduced one property upload.file.allowed.formats for doing a server-side check. The same value should be present in both biprws.properties and global.properties. The biprws.properties file is located under the path: <INSTALLED_APPLICATION-SERVER_DIR>\<APPLICATION_SERVER>\webapps\BOE\WEB-INF\config\default\biprws.properties.

Global.properties

  • Purpose: This configuration file is used for client-side checks.
  • Scope: It operates at the application level in the browser.
  • Function: It ensures that only the specified file formats are allowed to be uploaded from the client side, providing an initial level of validation.

Biprws.properties

  • Purpose: This configuration file is used for server-side checks.
  • Scope: It operates at the server level, providing a more robust layer of security.
  • Function: It prevents the uploading of junk or redundant files and ensures that it matches the allowed formats, even if an attempt is made to bypass client-side checks using tools like Burp.
