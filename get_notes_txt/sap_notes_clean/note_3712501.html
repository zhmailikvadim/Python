Symptom

  • An issue occurs during the conversion of text '4102' to Codepage '1100' in the transaction SUSG, where either characters are detected that cannot be represented in the Codepage or the conversion fails for other reasons.
  • The ABAP program 'CL_ABAP_CONV_IN_CE============CP' terminates due to incorrect data being generated during conversion.
  • The issue started after applying Support Packages.

Environment

  • Internationalization (I18N)
  • SAP Systems based on SAP NetWeaver 7.4
  • non-Unicode

Reproducing the Issue

  1. Run transaction SUSG
  2. Observe the dump CONVT_CODEPAGE with the error where characters cannot be converted from '4102' to Codepage '1100'
  3. Note the termination of the ABAP program 'CL_ABAP_CONV_IN_CE============CP'.

Cause

An issue occurs during the conversion of text '4102' to Codepage '1100', where either characters are detected that cannot be represented in the Codepage or the conversion fails for other reasons. The ABAP program 'CL_ABAP_CONV_IN_CE============CP' terminates due to incorrect data being generated during conversion. 

Resolution

Implement the latest version of SAP Note 3594059 in the system.
