Symptom

In the process "Sending of Non-Settlement-Relevant Request Without Dependencies from Auth SUP to DSO as SUP (44162, 55162)" (DE_B_MDC_SUP_N_STLREL_N_DEP_AUTH_SUP_DSO), the description text incorrectly contained the obsolete text "55162".

This issue is relevant for the market role: SUP.

Other Terms

DE_B_MDC_SUP_N_STLREL_N_DEP_AUTH_SUP_DSO, UTILMD 44162, process description, non-settlement-relevant request, authorized supplier

Reason and Prerequisites

The error occurred because the process description in the configuration was incorrect.

Solution

Please implement the corrections of this SAP Note by importing BC set /US4G/3_0_SUP_SP08_NOTE_3729000(102-SP08), /US4G/3_0_SUP_SP07_NOTE_3729000(102-SP07), /US4G/3_0_SUP_SP00_NOTE_3729000(103-SP00) or the relevant Support Package (SP).

After implementing this SAP Note, the system updates the process description to "MDC: Sending of Non-Settlement-Relevant Request Without Dependencies from Auth SUP to DSO as SUP (44162)" for process DE_B_MDC_SUP_N_STLREL_N_DEP_AUTH_SUP_DSO, removing the obsolete text "55162". You can verify the fix by checking that the process description no longer contains the obsolete text "55162".
