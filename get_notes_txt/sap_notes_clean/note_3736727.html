Symptom

After order-related sales order item is fully billed, it is no longer possible to change its sales quantity.

Environment

  • Sales and Distribution (SD)
  • SAP ERP Central Component
  • SAP ERP
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Create order-related sales order via VA01.
  2. Create billing for full quantity with reference to this order via VF01.
  3. Try to change sales quantity via VA02.

Cause

It is standard behavior.

Resolution

  • Add a new item or create new sales order with the product.
  • If it is required to edit sales order quantity in existing fully billed item, then user-exit USEREXIT_FIELD_MODIFICATION(MV45AFZZ) for dialog transaction or USEREXIT_FIELD_MODIF_BAPI(MV45AFZZ) for SD BAPI can be utilized to re-open the screen input of sales order quantity field.

See Also

SAP Note 512307.
