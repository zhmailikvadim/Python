Symptom

  • Attachments (.PDF, etc.) are not opening in My Inbox Fiori application.
  • Attachments are downloaded without any extension from My Inbox Fiori application.
  • DMS attachments are not displayed in My Inbox Fiori application.

Environment

  • Task Gateway
  • Gateway Business Workflow Enablement
  • SAP Business Workflow
  • SAP NetWeaver
  • SAP Web Application Server for SAP S/4HANA
  • SAP Gateway
  • My Inbox Fiori Application
  • DMS attachment

Reproducing the Issue

  1. Open My Inbox Fiori app,
  2. Select workitem with attachment,
  3. Click attachment tab,
  4. Download attachment.

Cause

DMS attachment, SO_FILENAME is missing from the object header.

Resolution

System works as per SAP standard design. Task Gateway and My Inbox application does not support DMS attachments for SAP_BASIS 7.52 and lower releases.

See Also

For further details about file name extension read SAP Note 1459896 -BCS: Support for four-digit file name extensions.
