Symptom

Set the rule condition for selecting message locale 

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

SAP SuccessFactors Onboarding

Resolution

  1. Go to Configure Business Rule
  2. Email Services
  3. Select Email Message Locale
  4. Select the necessary email category. 
  5. Set the if condition based on the Users Company Country and then set the email message locale to based on the country. Attaching Sample rule for reference.
