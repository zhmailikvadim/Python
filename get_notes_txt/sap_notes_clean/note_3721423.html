Symptom

The full name of the user is displayed in the Manage Job Profile Last Modified By field instead of the user’s display name. 

Environment

  • SAP SuccessFactors Job Profile Builder
  • SAP SuccessFactors Talent Intelligence Hub

Cause

Expected Behavior / Current System Limitation

Resolution

Our internal review in the SAP demo confirms that this is expected behavior.

However, should you wish to see this behavior changed, please provide feedback to the Product Management team via the Influence Page where other customers will also have the opportunity to support your functionality request. Please raise an Enhancement request as per the KBA below: 

https://me.sap.com/notes/2090228 

 

Keywords

sap, success factors, SF, JPB, job profile builder, Manage Job Profile content, roles
