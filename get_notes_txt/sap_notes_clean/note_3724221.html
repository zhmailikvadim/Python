Symptom

  • SmartEdit login page displays blank screen.
  • Error message: "I/O error on GET request for 'https://127.0.0.1:8088/smarteditwebservices/v1/configurations': Connection refused: /127.0.0.1:8088".
  • Exception message: "org.springframework.web.client.ResourceAccessException".
  • Exception message: "org.springframework.messaging.MessageHandlingException".

Environment

SmartEdit

Reproducing the Issue

  1. Open the SmartEdit login page.
  2. Observe the blank page displayed.
  3. Check the browser console logs for errors.

Cause

The issue was caused by essential environment variable ${ENV::KEYSTORE_ALIAS} removed in Commerce Cloud jdk21.

Resolution

  1. Identify the missing environment variable required for the Tomcat SSL connector configuration.
  2. Update the custom server.xml file to remove the ${ENV::KEYSTORE_ALIAS} 
  3. Perform new deployment without ${ENV::KEYSTORE_ALIAS} in customized server.xml files. 
  4. Verify that the SmartEdit login page loads successfully without errors.

Keywords

smartedit, blank login page, tomcat ssl connector, connection refused, I/O error, resource access exception, message handling exception, server.xml, environment variable, ssl configuration.
