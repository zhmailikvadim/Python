Symptom

  • Encountering an issue when assigning a dynamic group to assign a responsible group to a dynamic group.
  • Need to increase the responsibility group limit from 200 to 500 in Onboarding General Settings using Dynamic Group.

Environment

SAP SuccessFactors Onboarding

Reproducing the Issue

  1. Admin Center -> Onboarding Dynamic Groups -> Make sure there are more than 200 groups (700-800 groups)
  2. Admin Center -> Onboarding General Settings -> Click on existing responsible group or create new
  3. Click on Dynamic groups dropdown
  4. Notice that only 200 are displayed, remaining are removed
  5. Requirement is to populate up to 1000 groups on this dropdown

Cause

When high number of dynamic groups are present, only 200 are listed in the responsible group dropdown due to a UI limit of 200 for the component.

Resolution

PERMANENT FIX: OBX-41996

  • The Engineering Team has introduced a fix to accommodate increasing the number of listed dynamic group in the responsibility group being added in Onboarding General Settings, and is scheduled to be deployed on 1H 2026.
  • NOTE: Roadmap disclaimer applies
  • Please monitor this KBA if there's any changes with the dates.
