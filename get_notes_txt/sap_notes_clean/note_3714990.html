Symptom

In certain cases due to incorrect/corrupt data during Payoff execution, the system is not able to handle contract save errors and this leads to issues in Payment Distribution run when there are multiple Payment distribution Items processed within the same run containing other Payoff's that has no issues/errors during execution.

Other Terms

Payoff Quote Execution, Payment Distribution Execution, /FSPD/IPD_MR1, BCA_OR_PAYF

Reason and Prerequisites

Program Error

Solution

Implement this correction via SNOTE or upgrade to relevant support package.
