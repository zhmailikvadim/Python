Symptom

  • This article provides detailed steps to extract custom EXT procedures from the database using the Touchless Deployment process via command prompt on Windows operating systems.
  • It aims to assist users in performing the procedure efficiently while ensuring proper setup and execution.

Environment

  • SAP Sales Performance Management
    • Incentive Management on Legacy HANA 

Resolution

Pre-requisites:

  1. Install java (rel >= 8).
  2. Download the TDClient script file.
  3. Unzip the file to a desired path on your machine.

Steps to extract procedure:

  1. Open Command prompt.
  2. Navigate to bin folder from the unzipped folder from above steps.
  3. Declare environment variable with the following commands:

    SET TD_BASEURL=https://1234.callidusondemand.com
    SET TD_USERNAME=<user> (a user who has access to IM)
    SET TD_PASSWORD=<password>
    SET TD_LOGDIRECTORY=<logdirectory> (optional: Points to a directory which is used by TDClient to extract and store log files for customer auditing. If the directory contains any spaces or special symbols, make sure to surround the directory with double quotes “ .)

  4. Verify connectivity by running the command : TDClient.cmd -o testConnection
  5. Confirm the values of the environment variables using: echo %TD_BASEURL%
  6. Perform the procedure export using the command: TDClient.cmd -o codeDownload -type PROCEDURE -object <Object name>  -ld <path where you want to download>
  7. After successful completion, locate the stored procedure at the specified download path.
  8. Exit the Command Prompt application to conclude the session.
