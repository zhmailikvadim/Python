Symptom

  • A requirement exists to add a Document Flow step next to Offboarding tasks in the Process Variant Manager, but the system displays the error message: “Process steps are not available.”
  • The issue occurs when attempting to add the Document Flow step after completing Offboarding tasks.

Environment

SAP SuccessFactors Onboarding

Reproducing the Issue

  1. Add a Document Flow step next to Offboarding tasks in the Process Variant Manager.
  2. Observe the system displaying the error message: “Process steps are not available.”

Cause

The system does not support managing the document flow based on the task itself in Offboarding scenarios.

Resolution

  1. Configure a business rule using the Select Process Variant rule scenario.
  2. Refer to the Setting a Business Rule for a Process Flow topic for detailed instructions.
  3. Review the following resources for additional guidance:
  • 3534935 - Offboarding Process Variant Not Available in Business Rule for Selection Onboarding - SAP for Me
  • Offboarding Process Flow Now Available in Process Variant Manager | SAP Help Portal
  • Setting a Business Rule for a Process Flow | SAP Help Portal

See Also
