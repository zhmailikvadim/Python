Reason and Prerequisites

 This SAP Note contains a report to automate the creation or modification of objects used in the solution provided by the SAP Note

Solution

Use transaction SNOTE to implement the attached correction instructions before you implement the primary SAP Note 3692222. The transaction SNOTE installs the reports NOTE_3692222 in your system. After that, implement the primary SAP Note 3692222.

As a general rule, SAP recommends that you implement the Support Package specified for your release. However, if you need to install the solution earlier, use the SAP Note Assistant to implement automatic correction instructions. For more information, see SAP Note Assistant.

Note: Automatic correction instructions for releases S4HCMCUA contain corrections for dictionary structure HRPADUA_S_IT3452_SCREEN which may not be activated correctly. Then it should be activated after implementation of the primary SAP Note 3692222.
