Symptom

When you navigate to a process run log from the list of business transactions in the construction savings contract display (tab page "General Data", subarea "Status" of transaction /IBS/BSC_SHOW), the system issues the error message "An error occurred during the select in the table /WAV/T_PRUN_INFO".

Reason and Prerequisites

The jump supports the display only for logs of interrupted WAVE-based processes.

Solution

The jump is enhanced with the option to display logs of completed, WAVE-based, and FAIR-based processes. As a result, the following is displayed:

  • WAVE-Based, Interrupted: Exception Processing
  • WAVE-Based, Completed: WAVE Monitoring
  • FAIR-Based: Application Log (of Logical Run)

as far as display is still possible after the logs are available.
