Symptom

The supplier says they loaded a new catalog subscription to the Ariba Network (AN) but it does not download to our Buyer site, even though the Ariba Network Full Subscription Synchronization task ran afterwards and did not show any errors.
We also use Catalog Content Manager (CMS) and it also did not show any errors.

Environment

Ariba Network Full Subscription Synchronization task, Ariba Buying and Invoicing, Ariba Network

Reproducing the Issue

In Ariba Buying and Invoicing:

  1. Go to Manage > Core Administration > Catalog Manager > Scheduled Tasks
  2. Search for Ariba Network Full Subscription Synchronization task
  3. Click Run
  4. Click Refresh Status after a few minutes, and once the task status changes to Completed, continue to the next step:
  5. Go to Manage > Core Administration >  Catalog Manager > Catalogs
  6. Search for the supplier name or the catalog subscription name
  7. See the catalog is not there.

If your site uses Catalog Content Manager as well, follow these additional steps:

  1. Go to Manage > Core Administration >  Catalog Content Manager > Content Documents
  2. Search for the subscription name in question
  3. See it is not there.

If at any point you encounter an error, or see the subscription on your Buyer site but still have issues seeing it properly in the UI, contact SAP Support with this article and describe what steps you took so far, as your issue likely has a different cause.
If the subscription is in neither of the areas mentioned and you found no errors:

  1. Log in to the Ariba Network site.
  2. Go to the Catalogs tab.
  3. See the subscription is not listed.

Cause

The catalog subscription is not subscribed to on the Buyer's Ariba Network site and the AN Sync task is not informed to add it to the list of catalogs to be imported into Buyer.

Resolution

To resolve this make sure the subscription is subscribed to on the Ariba Network site:

  1. Log in to the Ariba Network site.
  2. Go to Catalogs > Search (tab)
  3. Search for the subscription and:
    3. a) if you find the subscription, subscribe to it.
    3. b) if you do not find the subscription, contact the supplier and ask them to make sure it is published correctly Once that is done, repeat steps 1-3. a) on your Ariba Network site.

Once you subscribe to the catalog either wait for the Ariba Network Full Subscription Synchronization (AN Sync) task to run or, if you have that option, run it manually.
If at this point the subscription still does not appear in your Ariba Buyer site, reach out to SAP Support and mention which steps from this article you took.
