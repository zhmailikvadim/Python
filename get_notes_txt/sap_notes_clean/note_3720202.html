Symptom

What is the difference between Main WD Config Key and Property WD Config Key in the documentation: "Translating Tiles, Catalogs, and Groups in the Customizing Layer"

Environment

SAP S/4HANA

Resolution

Property WD Config Key is for Tile translation.

Main WD Config Key is for Catalog translation.
