Symptom

You attempt to post the final settlement for an Agreement from the calculation run and expect the accrual adjustment amount to be apportioned correctly to the accounting lines. However, the system throws the error “Accrual amount is zero, accrual is not possible,” and the accrual adjustment amount is not properly apportioned to the accounting lines during posting.

This note corrects the behavior by ensuring that accrual adjustment amounts are correctly distributed to the accounting lines, allowing successful posting of the final settlement.

Other Terms

/IRM/IPCIM, /IRM/IPCI_ACCRUAL_ADJUST, Calculation Run, Settlement, Postings, Accrual Adjustment, Accounting

Reason and Prerequisites

This is caused by program error

SAP Note#3733554 is available for  V/4 Release

Solution

Please implement the correction instructions attached. These corrections are only valid if you have applied the Support Packs.

Release 6.4 - Base Package and Support Pack 1

Release 6.3 - Base Package

 Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object.
