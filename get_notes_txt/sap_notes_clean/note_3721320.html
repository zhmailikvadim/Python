Symptom

Zero amount lines missing in G/L view (table ACDOCA) despite the lines are existing in the entry view (table BSEG).

Other Terms

WZR2

Reason and Prerequisites

Zero amount lines are deleted in the G/L view (ACDOCA) although BSEG-BUZEI is not initial because the BUZEI is not checked correctly due to a program error. Zero amount lines already removed during processing are still evaluated as if they were not removed while checking for remaining lines. As a result, all zero amount lines are removed for one BUZEI and no line is remaining in the end for the affected BSEG-BUZEI.

Solution

Implement the attached correction instructions or upgrade to the latest support package.
