Symptom

  • Missing integration of Forward Error Handling (FEH) for asynchronous XML messages related to ExportDeclarationSUITERequest_In.
  • Failed XMLs in GTS systems are not passed to FEH framework.
  • No integration scenario exists for Error and Conflict Handling (ECH/FEH) or its successor, Application Interface Framework (AIF).

Environment

  • Global Trade Services (GTS)
  • Global Trade Services / Logistics Services (SLL)
  • SAP Global Trade Services 10.0
  • SAP Global Trade Services 10.1
  • SAP Global Trade Services 11.0
  • GTS Edition for Hana

Cause

Currently in GTS there is no connection to the Error and Conflict Handler (ECH/FEH) and there are no future plans for this integration. There is also no integration with its successor, the AIF.

Resolution

Standard behavior as there is no current integration with GTS provided. 

Keywords

FEH, forward error handling, exportdeclarationsuiterequest_in, GTS, AIF, XNL messages, missing GTS integration, asynchronous processing, failed xmls, gts documentation, sap notes.
