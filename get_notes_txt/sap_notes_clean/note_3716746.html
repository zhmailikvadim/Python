Symptom

When executing the 'Employment Tax (IRPF) reports' program (RPC190E0), the TemSe file is generated but cannot be displayed and the error message "The TemSe object could not be identified" is shown in the log.

Environment

  • Human Capital Management (HCM)
  • SAP R/3
  • SAP ERP Core Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA

Reproducing the Issue

  1. Execute the 'Employment Tax (IRPF) reports' program (RPC190E0).
  2. Click to view the Temse file but nothing is shown.
  3. The program's log displays error message "The TemSe object could not be identified".
  4. The TemSe file generated may be opened and viewed with the 'TemSe files display and download' program (RPUTMSE0).

Cause

The required entries of 'Valid Attribute Values' (T52B4) and 'Assignment of Values to Objects' (T52B5) tables do not exist or ar incorrect, and so the following check fails:

Program: RPC190E0
Include: RPC19ME0

* ---------------------------------------------------------------------*
*       Form  GET_TEMSE_OBJECT_NAME
* ---------------------------------------------------------------------*
*       Obtener el nombre del objeto TemSe
*----------------------------------------------------------------------*
*      <--P_TEMSE_OBJECT    Nombre del objeto TemSe            *
*----------------------------------------------------------------------*
FORM get_temse_object_name CHANGING p_temse_object TYPE
                                                   p4tms_temse_object.

*   Obener el Nombre del Receptor
  SELECT * FROM t52b5 UP TO 1 ROWS
           WHERE potyp = 'TMS2'                   AND
                 ponam = p_temse_object-short_name AND
                 pattr = 'TOE'.
  ENDSELECT.

  IF sy-subrc = 0.
    p_temse_object-receiver_id = t52b5-pwert.
  ELSE.
    PERFORM append_error USING space  text-e11.
  ENDIF.

ENDFORM.                               " GET_TEMSE_OBJECT_NAME

Resolution

Create or correct the corresponding entries in T52B4 and T52B5 tables according to the last SAP Note(s) delivering the technical changes for the generation of Form 190.

Note: for year 2025, the last technical changes were introduced with SAP Notes 3698146 - IRPF: Objects delivery for SAP Note 3698145 and 3698145 - IRPF: Form 190 generation for fiscal year 2025.

3698145 - IRPF: Form 190 generation for fiscal year 2025.
