Symptom

  • When using the web dynpro application "Change Maintenance Order," an error occurs.
  • The issue arises when attempting to update a technical object (e.g., equipment) in the Maintenance Order.
  • Error message IW053: "Work center &1 plant &2 not permitted for Plant Maintenance" is displayed.

Environment

  • SAP ERP Plant Maintenance (PM)
  • SAP S/4HANA 1809
  • SAP S/4HANA 1909
  • SAP S/4HANA 2020
  • SAP S/4HANA 2021
  • SAP S/4HANA 2022
  • SAP S/4HANA 2023
  • SAP S/4HANA Cloud Private Edition 2023

Reproducing the Issue

  1. Open the web dynpro application "Change Maintenance Order."
  2. Update the equipment in the technical object field.
  3. Save the Maintenance Order.
  4. Error message "Work center &1 plant &2 not permitted for Plant Maintenance" appears.

Cause

Program error 

Resolution

  1. Review the details outlined in SAP Note 3552617.
  2. Implement this Note or update to the relevant Support Package for your release.
