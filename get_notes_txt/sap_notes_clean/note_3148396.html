Symptom

You are running an US Foreign-Trade Zone. You cannot admit a semi-finished product into your zone that is procured externally.

Changes are needed to be able to create a Local Clearance Admission after posting GR containing a SFP.

Other Terms

Foreign-Trade Zone, FTZ, USA, US, SFP, semi-finished product inventory layer

Reason and Prerequisites

New functionality

Solution

Please implement correction instruction.

Please read Composite SAP Note 3133769 to enable full scope of handling semi-finished product within its new inventory layer.

Other Terms

Authorization, authorization check, SAP Records Management

Reason and Prerequisites

SAP Records Management does not contain authorization checks for checking an authenticated user's authorization to access some of its functions. This may result in undesired system behavior.

Solution

Please apply the Support Package mentioned in this SAP Note or the respective correction instruction.

By the correction instruction authorization check of authorization object S_SRMSY_CL will be implemented.
