Symptom

  • When using transaction code /SAPSLL/PR_CLWB_CUS, an error message "You do not have authorization for this function" is displayed.
  • The system is checking for change authorization (ACTVT 02) in the authorization object GTS_PR_NS.

Environment

  • SAP Global Trade Services, edition for SAP HANA

Reproducing the Issue

  1. Execute transaction code /SAPSLL/PR_CLWB_CUS.
  2. Observe the error message: "You do not have authorization for this function."

Cause

  • The transaction code /SAPSLL/PR_CLWB_CUS is hardcoded to check 'ACTVT 02' for change authorization. By design /SAPSLL/PR_CLWB_CUS is intended for mass classification rather than display purposes.

Resolution

  1. Use Fiori apps like 'Manage Products' for displaying products.
  2. Refer to the help document available at the following link: Manage Products | SAP Help Portal.

See Also

Refer to: Manage Products | SAP Help Portal
