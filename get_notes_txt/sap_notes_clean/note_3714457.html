Symptom

A field named "Job Description" was created in the Job Information HRIS under Manage Business Configuration to appear in Recruit-to-Hire data mapping. However, it is not showing in the Recruit-to-Hire data mapping tool.

Environment

SAP SuccessFactors Onboarding

Reproducing the Issue

  1. Login to the SAP successFactors system
  2. Go to the recruit to hire data mapping> job info> the Job Description field is not there.

Cause

The corresponding custom field for the Job Description was not created in the RCM templates (Job Requisition/Application templates), preventing it from appearing in the Recruit-to-Hire data mapping tool.

Resolution

  1. Create the custom field in the RCM templates (Job Requisition/Application templates).
  2. Once the custom field is created, it will automatically appear in the data mapping tool and can be selected without any issue.
  3. Refer to the following guide for field mapping between Recruiting and Onboarding: Field Mapping Between Recruiting and Onboarding.(https://help.sap.com/docs/successfactors-recruiting/setting-up-and-maintaining-sap-successfactors-recruiting/field-mapping-between-recruiting-and-onboarding).

Keywords

Recruit-to-Hire data mapping field, Job Description, custom filed in recruit to hire data mapping.
