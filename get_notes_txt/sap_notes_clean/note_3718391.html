Symptom

When you create or change a customs inbound delivery, data is transferred from other inbound deliveries to the customs delivery. The symptom is caused by SAP Note 3334632.
The symptom occurs if you process several inbound deliveries using transaction /SAPSLL/DS_D_MM0B_R3 or if the relevant integration scenario uses the API /SAPSLL/API_6800_CIBD_SYNCH_MS.
