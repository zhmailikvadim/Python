Symptom

With enabling the 'Separate Fix FX Amounts and Remeasure and Translate FX' flag on the Legal Entity you noticed, that the previously incorrect CTA PL accounts have been fixed and are now exclusively posting against CTA OCI when translating the functional currency amount into a presentation currency amount.

Solution

As an interim solution, the CTA OCI accounts (36*) can be re-allocated to CTA PL accounts (6*).

Please be aware, that this is not a recommended way of adjusting the now correct postings. Adjustments created using this functionality interfere with IAS21.

 

If this still is something you are considering, please implement the attached correction to enable the primary allocation step to use the CTA SLA Groups.
