Symptom

Dump RAISE_EXCEPTION for exception condition "T681Z_MISSING" triggered during creating invoice via VF01:

Environment

•Sales And Distribution (SD)
•Logistics Execution (LE)
•SAP R/3
•SAP R/3 Enterprise 4.7
•SAP ERP Central Component
•SAP ERP
•SAP Enhancement package for SAP ERP
•SAP Enhancement package for SAP ERP, version for SAP HANA
•SAP S/4HANA

Reproducing the Issue

Create invoice via VF01 and save, dump will be raised

Cause

FM EXIT_SAPLV60B_011 was activated without any coding, resulting in the clearing of the KOMPCV structure. This causes the exception condition "T681Z_MISSING" to be triggered.

Resolution

  1. Use SMOD to activate the EXIT_SAPLV60B_011 coding.
  2. Ensure that coding exists to fulfill E_KOMKCV and E_KOMPCV structures, for example:
  • E_KOMKCV = I_KOMKCV
  • E_KOMPCV = I_KOMPCV

    3. If the EXIT_SAPLV60B_011 is not intended for use, deactivate it as per the solution provided in SAP Note 48431.
