Symptom

When the unloading date is used as an entry date for BPS scheduling (e.g. for STOs in consolidation scenarios), it should automatically be validated backwards.

Other Terms

 aATP; scheduling; BPS; entry date validation

Reason and Prerequisites

Scheduling with unloading date as entry point is currently only relevant in consolidation.

Solution

Implement the attached correction instruction or upgrade to the support package that includes the correction.
