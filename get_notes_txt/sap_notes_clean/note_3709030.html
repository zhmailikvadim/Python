Symptom

Module: EH&S WWI

You print labels using the SAP EH&S HVP printer driver. You notice that some texts have the wrong background color.

Other Terms

EHSWWISP70

Reason and Prerequisites

Reason
Program error in hvpdisp.dll

Prerequisites
EH&S WWI, all versions

Solution

After implementing this SAP Note, the system prints labels with the correct background color.

A correction is available with EH&S WWI 3.2 SP70. Apply this patch or a more recent patch. SAP Note 568302 provides information about the availability of patches, as well as information about storing, downloading, and applying them.

The following files were updated:
hvpdisp.dll, version 1.3.62 from 01/23/2026 or newer.
