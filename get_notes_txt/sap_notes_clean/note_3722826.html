Symptom

Adding Scope in the import settings of a job, after saving the change find it's not taking effect.

Environment

SAP Analytics Cloud (Enterprise) 2026.2.6

Reproducing the Issue

  1. Logon to SAC and open an imported model, select one job within it.
  2. Click Import Settings > Import Method set to Clean and replace subset of data.
  3. Click Add Scope > add any dimension as the new scope.
  4. Click save
  5. Refresh the page and check the import settings again.
  6. See the new-added object is now missing in the page.

Cause

This case is currently under investigation by development group.

Resolution

This case is currently under investigation by development group.
