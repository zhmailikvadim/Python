Symptom

In FBCJ tcode, as per the note 2655076, it is possible to retrieve sales order number from the source system(for central payment), when posting customer down payment in 'Cash Receipts' tab.
However, even if central payment is not enabled, the F4 search help always uses the result of cFIN search help (FINS_CFIN_SD_ORDER_SRCH) to populate the screen field. Thus, the screen field is blank even after the selection of value from F4 help.

Other Terms

FBCJ, Cash Receipts, Central Payment, F4 help on KDAUF, Sales order

Reason and Prerequisites

This is because of a program error.

Solution

Implement the automatic correction instructions provided in the note.
