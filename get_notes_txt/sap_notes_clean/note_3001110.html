Symptom

The default values of the technical settings for In-Force Data Extraction are not set optimally.

Other Terms

LRM

Life

/MSG/X_INF_DAT_EXT_CEDED_PP

Reason and Prerequisites

Program error

Solution

Version 1.0 from 12/9/2020: New

Implement the source code corrections.

The changed function is contained in Support Package SAPK-80404INMSG.

Description of the new or changed function:

The default values for the fields "Number of Work Instances" and "Package Size" have been adjusted.
A comment has been added for the affected parameters in the F1 help of the report.

Technical dependencies:

There are no technical dependencies.

Manual changes:

You must perform manual tasks.
