Symptom

  • Using transaction code WEL1, the table can not be edited
  • Error message: "Client 000 has status 'not modifiable' (Message No. TK430)".

Environment

  • SAP S/4HANA

Reproducing the Issue

  1. Access transaction code WEL1 in any system.
  2. Observe that the table can not be edited.
  3. Error message appears: "Client 000 has status 'not modifiable' (Message No. TK430)".

Cause

  • Obsolete transactions

Resolution

The transactions WEL0 and WE45 are maintenance transactions for the table EDILOGADR. This table will currently still be used by different applications, but they are all moving away from this table, as it should not be used any more. In case the table is still needed, it is still free to use it. This is the reason, why the transactions are deprecated. The transactions still can be used if necessary, but there is no successor available.
