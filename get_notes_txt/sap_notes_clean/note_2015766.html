Symptom

You have created a system landscape where all your destinations are not in the namespace MBT_ . In the system landscape all destinations have a green status. But the status of the system landscape is still initial.

Other Terms

System Landscape, CNV_MBT_RFC_700_CHECK_LS, initial state

Reason and Prerequisites

If no destination with prefix MBT_ is used the logic for checking the status assumes that no destinaton is distributed. But in the case that no destination with prefix MBT_ is used the distribution is not required and obsolete.

Solution

If all destinations in the system landscape are not in the namespace MBT_ then the distributed check is not required.

This note has to be taken over into the Control System. With the next check in the System Landscape the status should change to a value not equel initial.
