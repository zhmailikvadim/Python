Symptom

You open a project template and navigate to the checklist reference. When you do this, you notice that the fields on the "Basic Data" tab page are aligned to the right, which is incorrect.

Other Terms

Alignment, hAlign, sap-ie=Edge, standards mode

Reason and Prerequisites

The problem is caused by a program error.

The error occurs only if the application is opened with the URL parameter "sap-ie=Edge".

Solution

Implement the program corrections.
