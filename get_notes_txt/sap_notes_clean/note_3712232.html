Symptom

During the execution of RDD* jobs (for example job RDDDIS0L), you find error message

BY2 Database error 100 at EXE

in the kernel developer trace.

Other Terms

HDB_DBSL_CHECK_SCALEOUT

Reason and Prerequisites

With SAP Note 3486167, SAP HANA scale-out systems are better supported in the ABAP dictionary. Thus the ABAP Dictionary code has to determine, whether the system is a scale-out system. This action is done by calling a pseudo-procedure HDB_DBSL_CHECK_SCALEOUT, which in turn does a select on SYS.M_LANDSCAPE_HOST_CONFIGURATION. In case the system is not a scale-out system, no rows are returned from that select, which causes the BY2 trace error message. See also SAP KBA 3604933.

Solution

You can safely ignore these error messages in trace.
