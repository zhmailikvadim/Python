Symptom

In the agent inbox, you run a lean search for e-mails. However, the results list is empty, and the system issues the following information message:

"Inbox Customizing of item categories wrong or component set incomplete" (message CRM_IC_AUI 509).

Other Terms

CL_CRM_AUI_QUERY_SERVICE, AUIWorkitemLeanQuery

Reason and Prerequisites

The problem is caused by a programming error that is caused by SAP Note 1873634.

Solution

Implement the correction instructions that correspond to your release.
