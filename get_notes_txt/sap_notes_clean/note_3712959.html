Symptom

Spanish EC Sales List within statutory reporting framework is generated with multiplied values for documents with more than one partner line.

Other Terms

Spain, ES, S/4HANA, EC Sales List, ECSL, ES_ECSL_DCL_V1, Modelo 349, Form 349, Corrections, DRC

Solution

As a general rule, SAP recommends that you install this solution by applying a Support Package. However, if you need to install it earlier, it is recommended to utilize the most current version of Note Assistant to apply the corrective instructions. To ensure that Note Assistant is updated, please refer to SAP Note 875986 for NetWeaver version 7.0x or SAP Note 1668882 for NetWeaver versions 7.3x and higher.
