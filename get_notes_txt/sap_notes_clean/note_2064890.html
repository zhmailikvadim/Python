Symptom

When inspecting the application log for the capital calculation mass-run (transaction BCA_CAPITAL_CALC), you notice that accounts for which error messages are shown are not flagged for restart. Furthermore, in the sub-object 'Statistical Data', the program shows return code 0 ("Successfully Completed"), even though you would except return code 4 ("Completed with Individual Error").

Other Terms

capital invalidation, error handling, BCA_OBJ_GET_CAPITAL_LIST, exception

Reason and Prerequisites

Program error.

Solution

Install the latest support package or implement the attached correction instructions in advance.
