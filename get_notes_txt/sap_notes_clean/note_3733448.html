Symptom

A new goal plan is added in the instance and permissions are properly configured, still the new goal plan is not showing for users in list of goal plans within Goal module.  

Environment

SAP SuccessFactors Goal Management

Reproducing the Issue

  1. Upload goal plan in the instance
  2. Go to Manage Permission Roles tool and provide permission to goal plan 
  3. Navigate to Goals page 
  4. See that impacted goal plan is not listed 

Cause

RBP permissions take approximately 10 minutes to take effect. 

In case after 10 minutes the goal plan is still not visible, an RBP refresh can be run either in the instance via Scheduled Job Manager or via Provisioning to sync all permissions in the system. 

Resolution

Via Instance: 

  1. Go to Scheduled Job Manager
  2. Run an RBP Refresh Model job 

Via Provisioning: 

  1. Login to Provisioning 
  2. Go to Company Settings 
  3. Search Refresh RBP Permission Configuration and click on Refresh 
  4. Enter companyID and click on confirm 
  5. Log out they system and login again
  6. The Goal Plan should now be visible for the user in Goal Plan list.
