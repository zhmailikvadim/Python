Symptom

In transaction SPAU_ENH, the customer enhancement implementation shows a conflict in adjustment. For BAdI definition /UCOM/BADI_MDX, the interface method /UCOM/IF_BADI_MDX~ADJUST_AFTER_COMPOSE_DATA_CONT is not implemented.

This issue is relevant for all market roles.

Other Terms

MaCo, /US4G/CL_BADI_MDX

Reason and Prerequisites

Program error

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the system resolves the conflict in the enhancement implementation. You can verify the fix by checking in transaction SPAU_ENH that no conflict is shown for the enhancement implementation.
