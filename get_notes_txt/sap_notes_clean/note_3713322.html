Symptom

Observe the following error in the statistics of the remote source.

  • Error: Sub Component: ADAPTER_ERROR
  • Error: There are no active remote subscriptions on the remote source.

Environment

SAP HANA smart data integration (SDI)

Reproducing the Issue

  1. In SAP HANA Database Explorer, navigate to the remote source.
  2. Double click the remote source to open it.
  3. Click Statistics and observe the error ADAPTER_ERROR and There are no active remote subscriptions on the remote source.

Resolution

This message indicates that there are no active remote subscriptions configured in the system, and can be ignored if no real-time replication is configured.

Keywords

adapter_error, sub component error, remote subscriptions, exception
