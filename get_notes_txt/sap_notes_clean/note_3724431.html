Symptom

After upgrading to SAP S/4HANA 2023 SP03, all inbound journal entry postings to SAP via the JournalEntryBulkCreationRequest_In interface fail with a technical runtime error UNCAUGHT_EXCEPTION (exception class CX_SY_DYN_CALL_ILLEGAL_TYPE).

Environment

  • Financial Accounting
  • SAP ERP
  • SAP S/4HANA Finance
  • SAP S/4HANA
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA

Cause

  • The error indicates a mismatch in internal ABAP type/structure handling during a dynamic call, such as parameters or structures no longer matching the expected signature after the upgrade (e.g., DDIC structure changes, proxy class regeneration required, etc.).

Resolution

  1. Please implement below notes with all required manual steps:
  • 3325557 - /AIF/ERR: 'Restart' button behavior not correct for interface /FINAC/JRNLENTRIN&JELDGR_IN
  • 3438627 - AIF Content for SOAP Journal Entry API (Scenario ID: SAP_COM_0002)
  • 3477010 - API of Journal Entry – Post (Asynchronous): No Journal Entry Confirmation and Message (Scenario ID: SAP_COM_0002)

    2. Please re-test after implementing the notes to confirm resolution of the issue.
