Symptom

Component: Foundation for EHS Management
Module: Limit Checks

You configure limit checks for amounts. You collect amounts, or the system calculates amounts that cause a limit check to fail. In the Monitor Data application, on the user interface, the system displays a warning or error message for the amount.

You adjust the amount value by using the Replace function to meet the configured limit checks. In the Monitor Data application, the warning or error message disappears because the limit check passes.

However, the record for the failed limit check remains in the database table EHENVD_AMNS_ENVL in the database. This causes incorrect reporting of failed limit checks.

Other Terms

CL_EHFND_AMOUNTS_HELPER

Reason and Prerequisites

A program error in the system causes the symptom.

You find the prerequisites in the relevant correction instructions.

Solution

After implementing this SAP Note, the system deletes the record from the database table EHENVD_AMNS_ENVL when the amount passes the limit check successfully.

Alternatively, you import the corresponding Support Package for the component extension of SAP Environment, Health, and Safety Management or the Feature Package Stack for the SAP S/4HANA system.
