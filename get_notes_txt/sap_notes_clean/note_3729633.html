Symptom

This KBA provides instructions for how to delete HR employee master data. 

Environment

  • Human Capital Management (HCM)
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for HANA
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition
  • SAP SuccessFactors Employee Central Payroll

Resolution

Deleting Personnel Numbers via SAP Standard Report

In Non-Production Systems:
Use the standard program RPUDELPN to delete employee master data, allowing for mass data deletion.

In Production Systems:
Use the standard program RPUDELPP to delete personnel numbers. RPUDELPP does not support mass deletion due to the sensitive nature of personnel data. For more information, refer to the Destruction of Personnel Numbers | SAP Help Portal.

Deleting Personnel Numbers via Destruction Object

Use the Destruction Object HRPA_PERNR to delete personnel numbers of employees who have left after the required retention period. To use the Destruction Object, business function ILM must be activated as per SAP Note 1559133. For further details, see the Destroying Personnel Numbers Using HRPA_PERNR | SAP Help Portal.
