Environment

SAP Adaptive Server Enterprise (ASE) 16.1 PL01

Cause

The issue has been logged under SAP CR 831912. Its description reads, "ASE 16.1 Remote Backup getting interrupted with signal 11 error, while same same configuration works fine on 16.0.".

Resolution

    There is currently no target SAP ASE Service Pack scheduled for this issue.

    • It has been assigned to our Product Group.
