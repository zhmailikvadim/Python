Symptom

  • Issue happens only in IOS safari browsers.
  • Users with MacBooks using Safari browser experience issues with the user interface on the Termination screen.
  • Calendar fields do not display the calendar properly.
  • Dropdown lists appear reduced in size.
  • Error message pop-up screens are significantly reduced in size.

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

Sap SuccessFactors Employee central

Reproducing the Issue

  1. Choose any user.
  2. Take action>Termination.
  3. Click on the Calendar field/dropdowns or click submit without filling mandatory fields.
  4. Observed issues include a reduced calendar display, a shrunken dropdown size, and a minimized error message pop-up.

       

Cause

Resolution

As per the confirmation from engineering team this issue will be fixed in the 1H 2026 release.
