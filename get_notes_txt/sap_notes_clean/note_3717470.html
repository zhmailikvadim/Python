Symptom

With note 1327372, report RSPO_MOVE_ADS_FILES was delivered. The purpose of this report is to move ADS-generated spool files move from the global directory to client-specific subdirectories. However, the report should not do anything if the ADS spool requests are already stored in the database table TSPADS.

Other Terms

DIR_GLOBAL

Reason and Prerequisites

Program error

Solution

Apply the solution from this note or the support package mentioned herein.
