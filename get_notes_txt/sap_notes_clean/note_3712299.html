Symptom

To a shop floor routing you assigned multiple BoMs for the same material with different alternatives or usages. During creation of a shop floor order or when re-reading master data the wrong BoM is selected.

Other Terms

 Production Version, MKAL, STLAL, STLAN, EXE_STLAL, EXE_SLAN

Reason and Prerequisites

SAP Notes 3712299 and 3653046 are prerequisites for the implementation of this SAP Note.

Solution

Apply correction instruction.
