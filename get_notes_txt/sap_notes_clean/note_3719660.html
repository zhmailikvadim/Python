Symptom

Is there an API available for sending purchase orders to the Buyer and Supplier Ariba Network from non-SAP systems?

Resolution

Currently there is no API available that supports the import of purchase orders from non-SAP systems into the Buyer Ariba Network. This limitation implies that purchase orders generated from systems outside of SAP cannot be directly imported or integrated into the Ariba Network through an API. 

You can file an Enhancement Request to propose this feature to the engineering team for consideration in future product releases.
