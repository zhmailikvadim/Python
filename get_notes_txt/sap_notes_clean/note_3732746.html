Symptom

Field "IsProductionRelevant" could not set correctly via API API_BILL_OF_MATERIAL_SRV_0002.

Environment

SAP S/4HANA Cloud Public Edition

Cause

It is known issue. 

Resolution

  1. Apply workaround option A: When posting the BOM item, set IsProductionRelevant = true and remove BomItemIsCostingRelevant from the payload.
  2. Alternatively, apply workaround option B: Remove both IsProductionRelevant and BomItemIsCostingRelevant from the payload to rely on default values.
  3. A permanent fix is planned for 2026-04-18 to 2026-04-19. After the fix is available, retest the POST to confirm that IsProductionRelevant follows the payload value.
