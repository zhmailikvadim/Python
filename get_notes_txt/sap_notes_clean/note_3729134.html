Symptom

You run HR-HK: Periodic MPF contribution report (format: eMPF) (HHKCPFE0) report to generate eMPF data for new hires. However, system retrieves member class only from the current payroll period and does not retrieve member class from each payroll period for new hires.

The SAP Note adjusts report to retrieve member class from each payroll period instead of current payroll period. It ensures that the member class is correctly retrieved for all new hire records.

Other Terms

eMPF, HHKCPFE0, Member class, gracd

Reason and Prerequisites

This is a correction of legal function.

Solution

Apply the Support Packages or implement the corresponding Correction Instructions.
