Symptom

  • Saving changes in transaction OQN6 (Catalog profile - maintain Q notifications) leads to error message:
    TK430: Client &1 has status 'not modifiable'.
  • SAP Note "390167 - Error TK430 in Transaction OQN6 in the productive client" seems to be obsolete for S/4HANA.

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

  • SAP S/4HANA, on-premise
  • SAP S/4HANA Cloud Private Edition

Cause

The maintenance of the catalog profile belongs to Customizing.

Resolution

There is the need to edit the catalog profile as 'current setting' also in the production system and avoid error TK430.

  1. Call transaction SOBJ and click 'Edit'.
    Position to object V_T352B_CL, and in the detail screen for its header data, set indicator 'Current Settings'.
    Save the entry.
  2. Object V_T352B_CL is a view cluster and contains views V_T352B_B and V_T352C_B. Therefore, repeat the previous step within transaction SOBJ also for these two objects:
    a) Edit object V_T352B_B: In the detail screen for its header data, set indicator 'Current settings'. Save the entry.
    b) Edit object V_T352C_B: In the detail screen for its header data, set indicator 'Current settings'. Save the entry.
  3. Call transaction QCC2 and navigate to following path:
    >Quality Notifications
    >>Notification Creation
    >>>Notification Content
    >>>>Define Catalog Profile
    Do not open it, only set cursor onto node 'Define Catalog Profile'.
    Navigate to menu 'Edit > Display Technical Info' (or Display IMG Activity).
    Goto tab 'Maint.objects' and switch to change/edit mode.
    For Customizing Object V_T352B_CL, replace 'Transaction' value by OQN6.
    Save the entry.
  4. Call transaction SE16 for Table CUSAMEN.
    Execute without any selection parameter.
    Click icon for 'Create' to add a new entry:
    - DOC_CLS = SIMG (Documentation: Class)
    - DOC_OBJ = OLQN-OQN6 (Documentation object)
    - MENU = QM00 (Menu name)
    Save the entry.
  5. Test the catalog profile editing.
