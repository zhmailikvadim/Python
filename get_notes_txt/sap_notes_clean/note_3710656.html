Symptom

The CDS view "I_HCMPersonWorkAgreement" for the output of all contracts assigned to a personnel number may return several identical entries.

Other Terms

I_HCMPERSONWORKAGREEMENT, I_HCMEMPLPERSNMBRSFORUSR, DISTINCT, IT0105, P0105.

Reason and Prerequisites

Prerequisite:

  • The relevant user is incorrectly assigned to several personnel numbers at the same time in infotype 0105 ("Communication").

 

Reason:

There is a program error.

Solution

The corrections contained in this SAP Note ensure that the CDS view does not return identical entries.

 

Import the relevant HR Support Package into your system.

If you want to correct the problem in advance, implement the correction instructions attached to this SAP Note.
