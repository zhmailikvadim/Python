Symptom

  • The 'incoterms location' field is visible on quotes but cannot be found in the Quote Standard Fields page under Setup to restrict access permissions.
  • Only the incoterms field is visible.

Environment

SAP CPQ

Reproducing the Issue

  1. Navigate to Setup.
  2. Select Quotes.
  3. Choose Quote Fields.
  4. Open Quote Standard Fields.
  5. Observe that the incoterms location field is not visible on this page.

Cause

Bug in application.

Resolution

  1. The issue has been identified as a bug related to the visibility of the location field under the Quote Standard Fields tab.
  2. The functionality was inadvertently missed during implementation.
  3. The fix for this issue is scheduled to be included in the SAP CPQ-2605 release, planned for May.

See Also

  • Refer to: Known Issues in SAP CPQ
