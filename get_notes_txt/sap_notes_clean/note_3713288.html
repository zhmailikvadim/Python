Symptom

In the performance form, an audit trail indicates that a user proxied to themselves.

Environment

SAP Successfactors Performance Management

Reproducing the Issue

  1. Open the performance form.
  2. Send the form to the previous step using the form view.
  3. Observe the audit trail showing the user proxied to themselves.

Cause

This is a Known Issue.

Resolution

The fix for this issue is planned to be deployed on 1H 2026 release.

For release timelines, please review SAP SuccessFactors Product Release & Road Map Information.

Workaround

No workaround is available.
