Symptom

Some SAP Ariba Analytical Reports are coming with the warning message WARNING: UNSCANNABLE EXTRACTION FAILED in the Background report email title.

Cause

This warning message indicates that an email security system (firewall/anti-virus) could not scan an attachment because it failed to unpack or decompress it. This is often due to corrupt files, password-protected archives, or unsupported file formats, requiring the sender to resend the file or check the compression. 

Resolution

This is a warning from some scan software, the SAP code does not give this warning.
