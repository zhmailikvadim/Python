Symptom

1. There is no column for displaying comment in BOE and transaction query result, which is inconvenient for checking detailed information of BOE and transaction.

2. There are no columns for displaying transaction type and transaction number in BoE query result, which make it hard to trace related transaction from BoE.

Other Terms

POWL, Current Transaction Type, Comment

Reason and Prerequisites

This is a program error.

Solution

Apply the correction instructions in the SAP Note or the corresponding Package.
