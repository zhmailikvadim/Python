Symptom

When adding a Relationship to an Account, is is automatically marked as Main despite this not being checked when the user is adding the Relationship.

Environment

SAP Sales Cloud Version 2

Reproducing the Issue

  1. Create an Account
  2. Navigate to General -> Relationship.
  3. Add a new Relationship with type ABC and leave the Main relationship indicator unchecked

Result: the Relationship is automatically marked as Main

Cause

By design, the first Relationship of each type (for example, "Has Ship-to party") is automatically set as the main/default.

Resolution

This is expected behavior of the system.
