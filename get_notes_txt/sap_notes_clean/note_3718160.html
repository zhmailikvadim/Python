Symptom

Getting the following message while trying to create a query using the Custom Analytical Queries app (F1572):

"Query creation is no longer supported in this app. To create a new query, use the Custom Analytical Queries and Services app (F1572A)."

Environment

SAP S/4HANA Cloud Public Edition 2602

Reproducing the Issue

  1. Accessing Custom Analytical Queries app (F1572).
  2. Click on Create
  3. Getting message "Query creation is no longer supported in this app. To create a new query, use the Custom Analytical Queries and Services app (F1572A)."

Resolution

As per Custom Analytical Queries | SAP Help Portal 

Note: You cannot create a new query in this app. To create a new query, go to Custom Analytical Queries and Services app.
