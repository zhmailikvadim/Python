Symptom

Short dump encountered when executing T-code /PF1/IHB_NOTIFI. Dump occurs due to zero object reference.

Other Terms

/PF1/IHB_NOTIFI, /PF1/CL_OPM_CAMT_53_001_02, ADD_TXDTLS, CX_SY_REF_IS_INITIAL, LO_TXINF

Reason and Prerequisites

Program error. Access using a 'ZERO' object reference is not possible.

Solution

Implement the attached correction or upgrade to a latest relevant Support package.
