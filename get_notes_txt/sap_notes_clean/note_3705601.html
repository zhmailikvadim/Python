Symptom

The following error is raised in the Product Master after selecting  Numbering Scheme.

"Classification start is outside number validity" (Message No. /SAPSLL/CORE_PRODUCT118).

Environment

  • Global Trade Services (GTS)
  • SAP Global Trade Services, edition for SAP HANA

Reproducing the Issue

  1. Go to Manage Products (Transaction /n/SAPSLL/PRODUCT_02)
  2. Enter product number
  3. Click Change/Display
  4. Go to tab Classification
  5. Select a Numbering Scheme --> "Classification start is outside number validity" error is raised

Cause

The Classification Period field values are not within or equal to the TARIC Validity Period.

If Classification Period assigned to the Product exceeds the TARIC Validity Period, it will result in the error. This inconsistency needs to be fixed for this functionality to work properly.

Resolution

  1. Run report /SAPSLL/PROD_CLS_CHECK to identify inconsistent classification entries.
  2. Use transaction /SAPSLL/PRCL_CUS_DEL to delete the inconsistent classification entries.
  3. Recreate the classification data.
