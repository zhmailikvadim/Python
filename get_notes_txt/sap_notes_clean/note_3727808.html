After updating the ATC server to release 816 the run fails with the error: "Inspection was not executed due to an error while calling RFC RS_ABAP_GET_DDL_DEPENDENCIES_E ()".

Environment

ABAP platform

SAP_BASIS 816 SP00

Cause

The error is caused by some Data Definition (DDLS) objects where the TADIR object name of the DDLS and the view inside the object have different names.

Resolution

This is a known issue reported to the development team. The cause is known and does not constitute a problem. The error can be ignored for now.

Monitor this KBA for updates regarding the resolution of this issue.
