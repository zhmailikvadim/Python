Symptom

A request to add a new "On-Hold" button in the workflow to complement the existing "Approve" and "Reject" options.

Environment

  • SAP Sales Performance Management
    • Advanced Workflow for Incentive Management

Reproducing the Issue

To access the Custom actions workspace, navigate to Setup > Fields & Forms > Custom actions.

Cause

  • You can create Custom actions to use in all workflows, regardless of its status.
  • The Custom actions feature depends on Form Event scripts which you configure on the Events action settings when you create or edit a custom action.

Resolution

  • Refer to the documentation for creating custom actions in SAP Advanced Workflow.
  • Access the relevant guide using the following link: SAP Advanced Workflow - Custom Actions Documentation.

See Also

SAP Advanced Workflow - Custom Actions Documentation.

Keywords

on-hold button, custom actions, approve, reject, workflow customisation, button implementation.
