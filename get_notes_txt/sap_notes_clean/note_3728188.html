Symptom

In adaptation mode, when editing sections in entities, a dropdown menu appears upon opening a second edit panel, showing sections that are inconsistent with the "Manage Sections" view.

Environment

SAP Sales Cloud and SAP Service Cloud Version 2

Cause

Expected behavior: At a given point in time, the flex framework can only take a single context. When a section is opened in edit mode, its flex context remains active until explicitly closed. Opening a second edit panel without closing the first causes the dropdown menu to display fieldsets from the overall UI view.

Resolution

Close the first edit panel before opening another section or fieldset in adaptation mode.
