Symptom

If a trade promotion with generated fund usage is changed in a way that an update of the fund usage is required (e.g. change of plan dates) it might happen that on saving a short dump of type RAISE_EXCEPTION - Exception condition "EX_ERR_UPDATE" triggered occurs. This is only the case if the user is logged on in a different language than the trade promotion was originally created with.

Other Terms

Trade Promotion, Fund Usage, Description

Reason and Prerequisites

Program error

Solution

Implement the correction
