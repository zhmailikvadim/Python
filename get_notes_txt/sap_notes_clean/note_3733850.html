Symptom

  • Measurements exported from the measurements workspace are incomplete compared to what the UI displays.
  • With a business unit filter and a period applied, the UI shows a total but the exported file contains fewer records.
  • The discrepancy occurs during export from the UI; executing the equivalent query directly on the database returns the full set.
  • No error message is displayed.

Environment

  • SAP Sales Performance Management
  • Incentive Management on Hyperscalers

Reproducing the Issue

  1. Navigate to Incentive Management Admin and open the measurements workspace.
  2. Select a Business Unit filter and choose a Period.
  3. Confirm the displayed total measurements.
  4. Select Export and proceed without changing any export options.
  5. Open the exported file and observe fewer records than the UI total.

Cause

The export logic misses records when a Business Unit filter is applied and the measurement count exceeds approximately 900, leading to a mismatch between the UI count and the exported file.

Resolution

The Engineering team is conducting a thorough investigation to propose a permanent fix for this behavior, which is currently targeted to be implemented in the 2605 release. Please find below the expected 2026 default release dates for all versions:
