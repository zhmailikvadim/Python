Symptom

When creating a subscription contract from a solution quotation via API, a BL 208 dump occurs if a custom implementation of the BAdI CRM_PROVIDER_COPY (method COPY_ORDER_TO_CONTRACT) raises an exception.
If the same process is executed via the UI, a BSP error occurs instead.

Other Terms

solution quotation, subscription contract, copy, CRM_PROVIDER_COPY, error_occurred, dump, BSP error

Reason and Prerequisites

Program error.

Solution

Implement the attached correction instruction.
