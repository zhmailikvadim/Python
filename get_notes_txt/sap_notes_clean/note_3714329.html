Symptom

In the latest people profile, will see error 400 when clicking "Correction" for Job Information/Personal Information for future-dated onboarding candidate.

Environment

SAP SuccessFactors Onboarding 2H 2025

Reproducing the Issue

  1. Navigate to the latest employee profile of the external user.
  2. Go to Job data.
  3. View history for job information.
  4. Click on "Correct."
  5. Observe the error message: "Looks like something went wrong. Please try again. If the issue persists, contact your system administrator. Error Code: 400."

Cause

This is a Known Issue.

Resolution

The fix for this issue is planned to be deployed with the patch 2H 2025 P23(March 24 for preview and 26 for production). For patch timelines, please review SAP SuccessFactors Patches Knowledge Base.

No workaround is available at the moment.
