Symptom

  • Error 'You no longer need to reject' encountered when attempting to reject events via upload
  • Reject option is not visible for manual rejection of events

Cause

  • The user role does not have the permission 'Reject Approved SOW Line Item' enabled under the SOW Line Item module

Resolution

  1. Navigate to the Admin page in the production environment
  2. Select User Role > Search for Role 
  3. Enable the permission Reject Approved SOW Line Item

Once the permission is enabled, users can utilize the connector 'Approve Reject Item Upload' to reject the line item

Keywords

Event reject, sow line item, role permission, error message 'you no longer need to reject'
