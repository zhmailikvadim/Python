Symptom

Restriction of company code in the "Manage Billing Documents" app is not currently supported.

Environment

SAP S/4HANA Cloud Public Edition

Cause

Currently the "Manage Billing Documents" app only performs authorization checks on the "billing type" and "sales organization"; it does not check permissions for the "company code". The main reason is that VF01/VF02 only performs checks in LV60AA08 on 'V_VBRK_VKO' and 'V_VBRK_FKA', with the check existing solely at the sales organization level rather than the company code level (F_BKPF_BUK). 

Resolution

This is not supported currently.
