Symptom

When you use an archiving object to delete data records for one or more employees, the preprocessing phase is slow.

Other Terms

HCMDP, IT0001, SARA

Reason and Prerequisites

Prerequisites:

  • SAP Note 3421419 is implemented in your system.
  • Each personnel number in your system has several records of infotype 0001.

Solution

After implementing this SAP Note, the system processes the preprocessing phase faster.

A new version of the correction instruction is available. If you already implemented the previous version of this SAP Note, implement the new version.

Implement the Synchronization HRSP available in May 2026 or implement the correction instruction.
