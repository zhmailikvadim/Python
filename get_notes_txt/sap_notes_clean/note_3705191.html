Symptom

When creating a new Tenant from a backup, not all backup files are displayed in the Tenant Creation Wizard.

Reproducing the issue

Scenario:

  1. Logon to the Cloud Control Center (CCC) with an operator and go to Customer Management → Tenants.
  2. Choose a tenant from a HANA Service Unit.
  3. Export the tenant more than once.
  4. Attempt to create a new tenant from the backup.

Expected behavior:

Show all backup files for this tenant.

Actual behavior:

No backup files are found for this tenant.

Solution
