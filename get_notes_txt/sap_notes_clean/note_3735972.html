Symptom

 In the process "Receipt of Load Shape Values (DSO)" (DE_B_MET_PROFILE_VALUE_RCV_DSO), the system dumps in step 0110-0050 when updating load shape values with more than one profile.

Other Terms

MET, Receipt of Load Shape Values, stortdump, DE_B_MET_PROFILE_VALUE_RCV_DSO,/US4G/CL_UBD_PROF_VALUE_03,/US4G/CL_UBD_PROF_VALUE_03 => DET_POD_PROFILES_FOR_IMS

Reason and Prerequisites

The error occurred because the code used APPEND on a sorted table, causing a dump when processing multiple profiles.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the system updates load shape values without a dump when processing more than one
