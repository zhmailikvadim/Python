Symptom

Prerequisite Objects for SAP Note 3718415

Other Terms

Sales Ledger, Purchase Ledger, J_3RF_PURCHASE_LEDGER_X, J_3RF_SALES_LEDGER_X, FIARRU_SALES_LEDGER, FIAPRU_PURCHASE_LEDG

Reason and Prerequisites

This SAP Note contains the report NOTE_3718415 to automate the creation/modification of objects that are used in the solution delivered via SAP Note 3718415.

The implementation of this SAP Note alone has no impact on your existing functionality until the solution SAP Note is applied.

SAP Note 3718415 requires this SAP Note to be implemented as a prerequisite.

Solution

This SAP Note creates the Prerequisite Objects of SAP Note 3718415.

Implement the correction instructions from this SAP Note.

Perform manual post-implementation steps from this SAP Note.
