Symptom

 When we try to set the outcome for the SAP Status flow/Workflow which is configured by using the BS02 Tcode, then we are getting a short dump.

Other Terms

Short Dump, Status profile, Status flow, BS02, Outcome setting, /IRM/GSTAT

Reason and Prerequisites

This is caused by Program error.

Solution

Please implement the correction instructions. These corrections are only valid if you have applied the following Support Packs.

Release V/4 – 2023 V2, 2023 V3 and 2023 V3 Support Package 1

Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object
