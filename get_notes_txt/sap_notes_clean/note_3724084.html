Symptom

The Addon component UIHR002 100 is listed in the upgrade phase IS_SELECT in SUM(Software Update Manager).

Environment

SAP ABAP System

Reproducing the Issue

Retry the upgrade phase.

Cause

The upgrade target stack is incomplete -- the product instance for UIHR002 100 is not planned in the upgrade target stack.

Resolution

To resolve the issue, perform the actions below.

1. Reset/cancel current upgrade.

2. Plan the upgrade again in MP and ensure the Addon instance "SAP Fiori for SAP S/4HANA <version> - UI for HCM" is included for target release besides the configuration in the previous plan.

3. Restart the upgrade from scratch with a new extracted SUM folder and the new stack file.

Keywords

SUM, upgrade, IS_SELECT, SAP Fiori for SAP S/4HANA, UIHR002
