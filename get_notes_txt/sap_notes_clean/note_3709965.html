Symptom

Open a recipe in Display Recipe application. Select Additional Functions -> Mass Change. Choose "Recipe – Formula Items" as the Mass Change function and click Continue. In the Recipe Formula Items table, change the stage of the primary output to a stage used by one of the input items.

When streams are not allowed, the first input formula item with the same stage is deleted.

Only the output stream item (if present) with the same stage should be deleted, and input items should remain unchanged.

This issue also occurs in the Multi Formula Editor (Change Multiple Recipes) when you change the stage of the Primary Output.

Other Terms

MFE, /plmb/if_rcp_attr=>gc_item_type_os, process item

Reason and Prerequisites

Programming error

Solution

Implement the correction instruction or implement the Support Package of the SAP Note.
