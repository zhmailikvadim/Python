Symptom

  • After the HRSP upgrade for Payroll Australia, Function SETCU behaves incorrectly even when cumulation-related views (for example, V_T54C3 and V_T54C1) have not been changed. 
  • Function ADDCU also produces incorrect CRT table output. 

Environment

  • Human Capital Management (HCM)
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP SuccessFactors Employee Central Payroll
  • SAP S/4HANA

Cause

Note 3563402 - SETCU generates incorrect cumulation for leaving is present after the upgrade, but the required follow-up notes have not been implemented.

The issue occurs after upgrading to following support package level:

Resolution

Choose one of the following options:

  • Upgrade support package level to SAPK-608F1INSAPHRRXX and above.
  • Implement Note 3653632 - Payroll function SETCU - incorrect behaviour.
