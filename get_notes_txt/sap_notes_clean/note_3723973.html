Symptom

Archived data need to be reloaded for archiving objects EC_PCA_SUM or EC_PCA_ITM.

Environment

  • Profit Center Accounting
  • SAP S/4HANA Cloud Private Edition
  • SAP S/4HANA
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP enhancement package for SAP ERP
  • SAP ERP SAP ERP Central Component

Reproducing the Issue

  1. Attempt to reload the archived data for archiving object EC_PCA_ITM.
  2. Observe that reload programs are unavailable for the archiving object.

Cause

Reloading is not recommended because it will have a negative effect on other data and create data inconsistency. 

Resolution

Reload programs for archiving objects EC_PCA_SUM or EC_PCA_ITM are not provided.

See Also

SAP Note 611701 - Archiving EC_PCA_ITM, EC_PCA_SUM: Standardization

SAP Note 422836 - Archive. Profit Center archiving: How to read archived data
