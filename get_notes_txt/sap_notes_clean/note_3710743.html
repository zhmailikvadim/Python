Symptom

You use the radio frequency (RF) transaction 'Loading by HU' in SAP S/4HANA for Manufacturing Logistics.
After completing the loading process system creates new unloading warehouse tasks (WT).
However, instead of creating a new unloading warehouse order (WO), these tasks are added to an existing unloading WO that is assigned to another Tour.

Other Terms

WO with status 'In Process'

Manufacturing logistics, RF, Loading by HU, Unloading by HU, WO, WT, Tour

Reason and Prerequisites

Program error.

Solution

Implement the attached correction instruction or install the corresponding support package.
