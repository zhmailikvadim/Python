Symptom

There are two symptoms when starting program FJEP_IHB_REVERSE_GL_TRANSFER:

  1. The program does not update the IHB payment transactions: It only reverses the journal entry in FI, but resetting the field POSTING_STATUS in DB table /PF1/DB_ITEM_FI is by mistake not done.
  2. A dump occurs in the program because of duplicate entries in an internal table.

Reason and Prerequisites

The described first symtpom is a side-effect of note 3534349.

Solution

Apply the attached program correction.

The correction also creates the new transaction FJEP_IHB_REV_TRANSFR that starts program FJEP_IHB_REVERSE_GL_TRANSFER.
