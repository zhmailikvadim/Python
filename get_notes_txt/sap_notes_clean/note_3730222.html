Symptom

  • An additional icon for 'Up and Down arrow' appears under the Survey tab within items. However, this is not clickable or operable.
  • The issue occurs when using a custom theme in the Theme Manager. 

Environment

  • SAP SuccessFactors Learning

Reproducing the Issue

  1. Access Learning Administration.
  2. Navigate to Learning Activities.
  3. Search for any item and open the record.
  4. Select the “Surveys” tab.
  5. Observe an additional “up and down” arrow icon displayed on the page.

Cause

This is identified as a product bug related to custom theme rendering that causes an additional, non-functional “up and down” arrow icon. 

Resolution

The backend code has been fixed by engineering. The fix version is scheduled to be delivered as part of the 1H 2026 release.

For product-related information, please review the SuccessFactors Product Release & Road Map

Current scheduled dates:

  • For Production: May 15-16, 2026
