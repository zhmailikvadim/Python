Symptom

When converting several planned orders at a time, the system only converts the first planned order of the selection and it shows the popup dialog “Error calculating costs” for the rest.

Environment

  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition
  • SAP S/4HANA Finance

Reproducing the Issue

  1. Run tcode COR8.
  2. Select 3 planned orders or more.
  3. Click on “Convert Planned Order” button.
  4. It shows “Error calculating costs” message.
  5. Click Cancel option.
  6. Only one of the planned orders is converted.

Cause

Program error.

Resolution

Please implement below notes: 

SAP Note 3032420 - New Cost Object Controlling: Saving plan costs in ACDOCP for manufacture order fails

SAP Note 3045986 - New Cost Object Controlling: Improve performance when calculate or save multiple manufacturing orders

SAP Note 3080888 - New product cost controlling: Multiple plan orders convert to production orders, some order's plan cost is missing in ACDOCP
