Symptom

You use claim settlement in CRM side and transfer it to ECC by Accounting Integration. And you want to know whether it can create a parking accounting document in ECC, instead of creating a posting accounting document directly.

Environment

  • SAP Customer Relationship Management 7.0 and higher
  • SAP enhancement package for SAP Customer Relationship Management
  • SAP enhancement package for SAP CRM, version for SAP HANA

Reproducing the Issue

  1.  Approve the claim and execute the settlement in CRM side.
  2.  Check the new created accounting document in ECC side.

Cause

Technically, the indicator to create a parking document is determined by the field DOC_STATUS to the structure DOCUMENTHEADER of BAPI_ACC_DOCUMENT_POST on ECC side. Currently there is no field DOC_STATUS and related logic to decide how to fill this field on CRM side. So this function is not supported in standard.

Resolution

The standard integration from CRM to ECC FI doesn't support creating a parking document.

See Also

Please check note 2092366 - Parking with BAPI_ACC_DOCUMENT_POST to get more details regarding creating a parking document function.

Keywords

TPM, Claim, Settlement, Parking document, ECC.
