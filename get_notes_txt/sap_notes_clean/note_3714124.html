Symptom

  • Requirement to upload all licenses during the data load activity.
  • Current use of 'Manage License' Fiori app to create licenses individually.
  • Request for an alternative app or tool to enable bulk loading of licenses.

Environment

  • Global Trade Services (GTS)
  • Global Trade Services / Logistics Services (SLL)
  • SAP Global Trade Services
  • SAP ERP
  • SAP S/4HANA

Reproducing the Issue

  1. Attempt to load all CTA licenses during the data load activity in the cutover phase.
  2. Use the 'Manage License' Fiori app to create licenses individually.
  3. Observe the inability to perform bulk license creation using the current app.

Cause

This is a standard system design to ensure compliance with different legal frameworks across various jurisdictions.  

Resolution

  • No standard functionality provided by SAP to upload and create mass licenses in one operation due to the inherent complexity and compliance requirements of license management.
  • Each license must be created and defined by specific attributes that are unique to individual legal regulations and business requirements.

Keywords

bulk license creation, Manage License, license upload, license creation, mass license, license management
