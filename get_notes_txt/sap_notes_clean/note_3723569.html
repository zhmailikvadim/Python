Symptom

You use SAP ACM and use electronic health professional cards (HBA) to sign electronic orders.

If there is only one ECC certificate on the HBA and no valid RSA certificate, you cannot sign.

Other Terms

Universal prescription, N3KVMST01, N3KVUNIRP, ePrescription, eVDGA, eAU, RSA, ECC, crypt

Reason and Prerequisites

Program error

Prerequisites:

  • You use a connector with a firmware version lower than PTV6.
  • There is no (valid) RSA certificate on the HBA, only an ECC certificate.

Solution

Import the specified Support Packages to solve the problem. You can implement the corrections in advance.

Note that this SAP Note has SAP Note 3722671 as a prerequisite.
