Symptom

When using a tool such as the NetWeaver Web Services Navigator to test the ProductionLogService.findProductionLogByDetails web service, the following error message occurs and no result is displayed for the service call:

'Error during deserialization of SOAP re-response on the client: XML Deserialization Error. XML is not valid. Element [http://www.sap.com/me/production]<productionLogList> is required in <ns4:Response> but can not be found.'

Other Terms

WSNavigator, production log, ws

Reason and Prerequisites

This error is occurring because the SAP ME response object for the findProductionLogByDetails web service does not define a minOccurs value for the productionLogList element in the response.  When minOccurs is not defined, its default value is 1 which means that element is required in the response.  As a result, if no Production Log records are found for the request, an error is occurring during deserialization of the response because the findProductionLogByDetails service is not returning the productionLogList element as expected.

Solution

The solution is provided in the form of a code change so that the productionLogList element is defined with a minOccurs=0 so that the element is optional in the response object.

IMPACT

The code change done to address the symptom above is isolated to the ProductionLogService.findProductionLogByDetails web service
