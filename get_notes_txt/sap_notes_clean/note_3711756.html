Symptom

You have added either the "Move-In" or "Move-Out" buttons to a specific Page Layout and then assigned this page layout to specific users. When these users open the Customer Hub, they cannot see these buttons.

Environment

  • SAP Sales Cloud Version 2
  • SAP Service Cloud Version 2

Reproducing the Issue

Log in as Administrator:

  1. Go to the Agent Desktop.
  2. Search for a Customer.
  3. Click on Accept. This action opens the Customer Hub.
  4. Click on the User Icon, then select Start Adaptation.
  5. Change from Main Layout to any Custom Page Layout.
  6. Click on the "Create" button. You will see the Move-In and Move-Out buttons added.

Log in as an end user:

  1. Go to the Agent Desktop.
  2. Search for a Customer.
  3. Click on Accept. This action opens the Customer Hub.
  4. Click on the "Create" button. The Move-In and Move-Out buttons are NOT visible.

Cause

The user is assigned to a Role which does not have access to the "moveIntegrationService" business service.

Resolution

You should add the "moveIntegrationService" business service to the user Role by following these steps:

  1. Click on the User Icon.
  2. Go to System Settings > All Settings. 
  3. Go to Users and Control > Roles.
  4. Open Role assigned to the User.
  5. Click on "Search and Add Business Service".
  6. Select the "moveIntegrationService" business service.
  7. Set proper Read and Write accesses.
  8. Save it.

                                                                                         

 Once this service is added, users assigned to these page layouts should be able to see the Move-In and Move-Out buttons.
