  • Transaction FAGLGVTR_PJO run in foreground
  • Same variant in background finishes immediately with “Errors occurred. Check parameter!” (Message No. FINS_BCF_PJO052).

Environment

  • SAP S/4HANA Finance
  • SAP S/4HANA
  • Financial Accounting (FI)

Reproducing the Issue

  1. Start FAGLGVTR_PJO run in foreground; it completes,
  2. but scheduling the same parameters in background ends with job log “Errors occurred. Check parameter!” (FINS_BCF_PJO052).

Cause

  • Background execution prerequisites not fulfilled: error occurred in the job.

Resolution

  1. Before background scheduling, use “Check Parameters” on the selection screen and correct any reported issues.
  2. Ensure background prerequisites.
  3. Use Parallel Job Organization as guided; refer to: 3016749 - BCF: Parallel Job Organization - FAQ.
