Symptom

In the Change of Control Box process (DE_B_IDEV_CTB_CHANGE), the device location field (DEVLOC) in the additional process data isn't populated when the Installation of Control Box process (DE_B_IDEV_CTB_INSTALL) is called in step 0800.

This issue is relevant for the market role: MOS.

Other Terms

IM4G, DE_B_IDEV_CTB_CHANGE, DE_B_IDEV_CTB_INSTALL, DEVLOC, device location, installation reply, device master data, /US4G/CL_IDEV_PROC_CTB_INST, /US4G/CL_PRST_TRIG_CTB_INST, control box

Reason and Prerequisites

The error occurs because the device location isn't populated from the installation reply if it can't be derived from the device master data.

Solution

Implement the corrections from this SAP Note or import the relevant Support Package.

After you have implemented this SAP Note, the system populates the device location from the installation reply if it can't be derived from the device master data. To verify the fix, check that the DEVLOC field in the additional data of process DE_B_IDEV_CTB_INSTALL is populated.
