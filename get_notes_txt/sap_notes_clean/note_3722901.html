Symptom

Error "The attribute SIGN of advanced parameter UMSKZ has an invalid value" in the Schedule Accounts Receivable Jobs app.

Environment

SAP S/4HANA Cloud Public Edition

Cause

Incorrect configuration maintained in the SSCUI 101401 - Define Variants for Correspondence Types.

Resolution

Update the configuration in SSCUI 101401 - Define Variants for Correspondence Types. Verify that the special GL indicator value ranges are correctly defined.

Keywords

schedule accounts receivable jobs, special GL indicator, umskz, correspondence variant SAP 13, error message, invalid value, SSCUI 101401, advanced parameter, customer statement generation, public cloud
