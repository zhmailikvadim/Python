Symptom

When using the Market Segments - Plan/Actual Fiori app (F0925), selecting a bookmarked layout modifies the header selection parameters unexpectedly.

Environment

  • CO Controlling
  • SAP Fiori
  • S/4HANA
  • S/4HANA Finance

Reproducing the Issue

  1. Open the Market Segments - Plan/Actual Fiori app.
  2. Select a bookmarked layout.
  3. Observe that the selection parameters are modified.

Cause

  • The bookmarks capture the full app state, including selection parameters, ALV layouts, and views, overriding separate variants when selected.
  • Selection parameters are stored at the header level, while ALV (lower level) favorites should not affect them.
  • This behavior differs from the classic SAP GUI, where header and ALV variants are isolated.

Resolution

Please follow the steps below to resolve the issue:

  • Delete the sap-iapp-state parameter from the bookmark URL.
  • Launch the Fiori application directly from the tile instead of using the bookmark.
