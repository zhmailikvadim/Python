Symptom

The following error happens while opening a story on HANA CLoud live connection with SAP Analytics Cloud mobile APP:

  • 404 Not Found - We could not find the resource you're trying to access.

Environment

  • SAP Analytics Cloud (Enterprise) 2026.2.6
  • SAC IOS mobile application

Reproducing the Issue

  1. Logon to SAC APP via IOS mobile device
  2. Open any story on HANA Cloud live connection.
  3. Briefly see a 404 error after opening the story, then the story data will be loaded successfully without doing anything else.

Cause

This issue is currently under investigation by SAC development group.

Resolution

This issue is currently under investigation by SAC development group.

Keywords
