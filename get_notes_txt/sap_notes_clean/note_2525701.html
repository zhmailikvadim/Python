Symptom

You are using Production Planning and Detailed Scheduling (PP/DS) for SAP S/4HANA.

Error messages are not found in Delta order transfer report

Other Terms

PPDS

PP/DS

PP/DS for SAP S/4HANA

Delta order transfer report

Reason and Prerequisites

Program error

Solution

As general rule, SAP recommends that you install a solution by applying a Support Package. However, if you need to install a solution earlier, use the Note Assistant to implement the correction instruction.
