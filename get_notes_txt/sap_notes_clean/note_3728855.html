Symptom

HPLUZLA1 program requires e-ZLA files in CSV format.

XML format seems to be recommended by ZUS recently.

Other Terms

e-ZLA, HPLUZLA1, XML, IT3421.

Solution

HPLUZLA1 program has been extended. Now it can also read XML files.

The file format is recognized automatically. There is no need to indicate the format on the selection screen.

Please apply the correction by importing the Support Package specified for your release, or by using the Note Assistant.
