Symptom

Due to business requirements, the posting option "Return delivery" from QA11 needs to be disabled. Guidance is required on how to configure this functionality.

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

  • SAP ERP Quality Management (QM)
  • SAP R/3
  • SAP R/3 Enterprise
  • SAP ERP Central Component
  • SAP ERP
  • SAP Enhancement package for SAP ERP
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA, on-premise
  • SAP S/4HANA Cloud Private Edition

Resolution

The relevant customizing for Stock posting options in usage decision can be reached at following path:
Call transaction QCC0 ----> Quality Inspection ----> Inspection Lot Completion ----> Define Inventory Postings.

To disable "Return Delivery", simply delete this line in the customizing, then retest in QA11.

Follow the same steps to modify other posting options or add new ones.
