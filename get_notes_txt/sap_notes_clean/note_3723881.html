Symptom

When trying to paste the data to fill the fields in the Manage Payment Advices app, the system throws an error related to UTF encoding.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Open app Manage Payment Advices
  2. Go through the line to insert the data
  3. Paste
  4. Error displayed

Cause

Different technology used in the development architecture of SAP Fiori application.

Resolution

Functionality is available only in SAP GUI applications. It's necessary to manually insert the field.

See Also

KBA 2704632 - Unable to copy & paste values from excel file to the SAP Fiori

Keywords

Copy, Paste, Manage Payment Advices, UTF-8, Encoding
