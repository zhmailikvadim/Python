Symptom

The report Create eAU Requests (RP_PAYDE_EAU_CREATE_NOTIFS) displays the information message There are still confirmations with the status "To Be Checked" (HRPAYDE_EAU028) even though the checkbox Create Log is deactivated on the selection screen.

Solution

Implement the attached correction instructions or import the specified support package. After you import the HR Support Package or implement the correction instructions, you do not have to perform any further follow-on activities.

The information message is displayed only if the Create Log checkbox is selected. Note that this function is intended for test purposes only. The execution during the mass processing of personnel cases leads to long runtimes and terminations.
