Symptom

When using Assign Deployment Target app in CBC evaluation workspace to connect starter development tenant, starter development tenant is missing. 

Environment

  • SAP Central Business Configuration
  • SAP S/4HANA Cloud Public Edition

Resolution

Starting from 2602 version, SAP will be provisioning and providing only starter system customizing tenant to the customer. Development tenant will no more be provisioned for the starter system.
