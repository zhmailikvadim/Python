Symptom

  • How Document size works in Mobile App?
  • Is there a default attachment size for Mobile App?

Environment

SAP SuccessFactors Mobile

Resolution

Mobile App will follow the configurations you have done in SuccessFactors system around Document Attachment / Manage Document feature.

In order to know the configurations, you can follow the KBA 2518360 - The Manage Documents Feature in SuccessFactors.

2518360 - The Manage Documents Feature in SuccessFactors
