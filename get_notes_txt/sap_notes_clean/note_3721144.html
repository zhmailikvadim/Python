Symptom

When creating Purchase Requisition(PR) via MRP in a S/4 HANA system, the GR processing time maintained in material master data is not considered in some cases.

Environment

  • SAP ERP Logistics (PP)  
  • SAP S/4 HANA, on premise
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Run T-code: MD03/MD01N
  2. Check the PR in T-code: MD04.

Cause

 If only an info record exists and there is no SA or contract available, the GR time maintained in the material master will always be applied.  Otherwise the GR time maintained in SA/Contract will be considered even it's 0.

Resolution

This is standard behavior.
