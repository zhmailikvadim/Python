Symptom

When you use the Configure Processes app, the system doesn't display the step data of a process. The system raises exception CX_SADL_DUMP_APPL_MODEL_ERROR with the reason "Page size overrun: Query implementation of '/APE/C_PROCESSSTEPTREE_C' returned too many records".

Other Terms

Configure Processes, Step, F5834A, /APE/C_PROCESSSTEPTREE_C

Reason and Prerequisites

The system has a limitation that prevents it from displaying process step data when the query returns too many records.

Solution

Implement the attached corrections or import the relevant Support Package
