Symptom

OAuth/OIDC-based authentication requires to send notifications from SAP BTP Cloud Logging Service to the SAP Alert Notification Service (ANS).
However, the SAP Help documentation Integrating with SAP Cloud Logging Service | SAP Help Portal for Cloud Logging integration indicates that only Basic Authentication is supported.

Environment

SAP Business Technology Platform

Reproducing the Issue

  1. Attempt to configure the application logging service to send notifications to the alert notification service using OAuth authentication.
  2. Refer to the documentation Integrating with SAP Cloud Logging Service | SAP Help Portal , which suggests basic authentication for connectivity.
  3. Observe that OAuth authentication is not supported for direct integration.

Cause

The application logging service does not support OIDC authentication.

Resolution

Currently, SAP Cloud Logging does not support OIDC-based authentication. The OIDC support is planned with an official announcement expected by the end of Q1 2026.
Once released, this will allow full OIDC-based integration with SAP Alert Notification, which already supports OIDC. Until then, only Basic Authentication is available.
