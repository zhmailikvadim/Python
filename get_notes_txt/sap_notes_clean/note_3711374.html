Symptom

Performance History block behaviour when Individual Talent Data are transferred from inactive employment to active per this guide: Transferring Individual Talent Data from Inactive to Active Employment.

Environment

SAP SuccessFactors Performance & Goals

Reproducing the Issue

Per the guide  Transferring Individual Talent Data from Inactive to Active Employment People profile Performance History block is not listed among the supported Data Categories for Transferring Individual Talent Data, will it be transferred to the new active employment?

Cause

Resolution

Despite People profile data are not supported for Talent Data Transfer, when Performance forms are transferred to the new employment, Performance History block will pull the data from the forms.

Please note that the new employment user should have access to the Performance History block set per the guides:

Configuring a Scorecard Block in the Legacy People Profile 

Configuring Performance History Card for Full Profile
