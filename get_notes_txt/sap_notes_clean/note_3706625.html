Symptom

Not able to delete ledger group in SSCUI 106930 - Define Ledger Group

Environment

SAP S/4HANA Cloud Public Edition 

Reproducing the Issue

  1. Access SSCUI 106930 - Define Ledger Group
  2. Attempt to delete an entry. 
  3. Delete option is not available

Cause

The deletion functionality is not supported in SSCUI 106930 - Define Ledger Group. 

Resolution

The deletion functionality is not supported in SSCUI 106930 - Define Ledger Group. 

See the Roadmap Viewer.  In roadmap viewer, check the excel Configuration Activity.xlsm (SAP Customer) where it is mentioned that for SSCUI 106930 - Define Ledger Group, the column "Delete Customer Records" states "Deletion not supported".
