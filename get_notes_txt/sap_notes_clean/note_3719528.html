Symptom

There is a value display issue even though the data source contains values for the specific fields. The values are not being fetched correctly from the JSON preparation process for display, despite being assigned.
This note resolves the above issue, and the values will now be displayed correctly.

Other Terms

/VIZI/RVWM, Value display(Description) , Data source, /VIZI/RVW_GRID_JSON_PREPARE

Please implement the correction instructions. These corrections are only valid if you have applied the following Support Packs.

Release V/4 – 2022 V2, 2023 and 2023 V2
