Symptom

With the SAP Build Work Zone Advanced mobile app, all of the workspaces can be accessed that are assigned to. Different type of content can be displayed, as well as perform workspace actions from the mobile device. 

Environment

SAP Build Work Zone , Advanced edition

Resolution

To start using the SAP Build Work Zone Advanced mobile app, do the following:

  1. Under your avatar, select Settings.

  2. Go to SAP Mobile Application in the side panel.

  3. Scan the QR code to install the mobile application.

  4. Save your changes.

Keywords
