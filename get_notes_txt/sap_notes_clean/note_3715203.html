Symptom

In the process "Device Installation/Replacement and Finalization of Technical Master Data" (APEU_B_MCM_TMD_CHANGE_PROCESS) within the MCM process cluster, the system does not create a clarification case when an exception occurs in step 0215-0060-0030 while triggering the IM4G process for market partner (MP) change.

This issue is relevant for the market role: DSO.

Other Terms

DE_B_MCM_MP_CHG_PROCESS, MCM_DE, MCM_DE_TRIGGER_IM4G_MP_CHANGE_PROCESS, /US4G/CL_PRST_TRIGG_MP_CHG, CREATE_EXCEPTION, APEU_B_MCM_TMD_CHANGE_PROCESS, MCM process cluster, exception document, clarification case

Reason and Prerequisites

The issue occurs because the configurations of the process and exception are in different process clusters.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the system creates a clarification case when an exception occurs while triggering the IM4G process for MP change in the specified process.

You need to retest the scenario. You can verify the fix by checking that a clarification case is created when an exception occurs during the IM4G process for MP change.
