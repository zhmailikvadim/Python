Symptom

User is not able to copy a template even though it has Template Creator role. 

Resolution

The Template Creator group alone is not enough for you to be able to create copies of templates. You also need to have a group that allows you to create the project/workspace-type that you are trying to copy.

For example, if you want to copy a Sourcing Project template you will need a group that allows you to create it like Sourcing Agent (or for Contract Workspace template, you need Contract Agent).

Keywords
