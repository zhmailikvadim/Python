Symptom

For any eDocument in Taiwan, unable to locate the source from which the system retrieves the seller information for the XML file or the PDF form.

Environment

SAP S/4HANA Public Cloud Edition

Reproducing the Issue

  1. Open the eDocument Cockpit APP.
  2. Select one of any eDocument under any processes of Taiwan.
  3. Choose the button Export File or the button Display.
  4. Want to locate the value under node 'Seller' in the XML or the source of the seller information in the PDF.

Cause

Resolution

The Identifier Number of the Seller is determined by the setting of SSCUI 106596, Define Business Places for Taiwan.

  1. In your configuration environment, search for Define Business Places for Taiwan.
  2. Select Configure and search for the company code that corresponds the eDocument.
  3. Select the Business Place on the Header Tab in the Billing Document.
  4. The corresponding values would be found in the field 'tax number 1'.

The Address of the Seller is determined by the setting of SSCUI 105651, Edit Addresses for Company Codes.

  1. In your configuration environment, search for Edit Addresses for Company Code.
  2. Select Configure and search for the company code that corresponds the eDocument.
  3. Click on Address, the corresponding values would be found.
