Symptom

  • User without authorization M_MIPA_ORG can view stock overview of a plant through ME23N.
  • But Error CO 004 " You have no authorization for plant XXXX " is raised only when trying to access via CO09 for users without M_MIPA_ORG.

Environment

  • Materials Management (MM)
  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA

Reproducing the Issue

  1. ME23N
  2. Display PO
  3. Go to Environment -> Item -> Availability 
  4. No error raised 
  5. Stock overview is displayed.

Cause

  • ME23N and CO09 follow different call stacks to show the ATP result screen, and the authorization objects checked in these transactions are different.
  • When the 'availability' is clicked in PO, the function module 'ATP_EXPLANATION' is called directly to display the ATP result. It does not call CO09 transaction, the SY-TCODE is ME22N / ME23N. The authorization object M_MIPA_ORG is not checked.
  • When CO09 is executed, authorization object M_MIPA_ORG is checked before 'ATP_EXPLANATION' is called.

Resolution

This is the standard system behaviour.
