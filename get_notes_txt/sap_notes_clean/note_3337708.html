Symptom

While entering the date manually during the change of remediation due date in Asessment form, incorrect date format gets accepted

Other Terms

PC 12.0, Assessment, OWP, GRPC_AIF_ASSESSMENT

Reason and Prerequisites

 Code is missing

Solution

· Run transaction SFP
· Open form GRPC_AIF_ASSESSMENT in change mode
· Go to menu Utilities => uploading/Downloading => Uploading Form and upload the attached form with file 'SFPF_GRPC_AIF_ASSESSMENT.XML ' from ZIP file
· Save and activate the changes
