Symptom

You are using the transaction /DBE/VSEARCH.

You have a need to perform customer specific operations just before the transaction /DBE/VSEARCH triggers the search for the vehicles. You would like the search criteria to be passed to the BAdI. You would like to have a possibility to alter the search criteria.

Other Terms

/DBE/VSEARCH, BAdI, search.

Solution

Implement the attached note.

Notice, that once you have implemented the note, the new BAdI method /DBE/BADI_V_SEARCH_UI=>ON_VEHICLE_SEARCH_01 has appeared. You can implement it.
