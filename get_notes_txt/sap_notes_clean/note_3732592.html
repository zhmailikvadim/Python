Symptom

  • Global header search returns no users in people search.
  • Global header search returns no actions in action search.
  • No error message is displayed.

Environment

  • SAP SuccessFactors Platform
  • SAP SuccessFactors HCM Suite

Reproducing the Issue

  1. Open the global header search.
  2. Enter a valid user name or feature/action keyword.
  3. Observe that no people or actions are returned.

Cause

People (directory) search was not enabled.

Resolution

Please check whether the relevant permissions and functions mentioned in the Prerequisites and permissions for enabling People Search are enabled. 

See Also

Prerequisites and permissions for enabling People Search
