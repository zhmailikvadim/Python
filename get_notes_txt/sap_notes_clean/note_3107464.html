Category                           ABAP programming error
Runtime Errors                  MOVE_TO_LIT_NOTALLOWED_NODATA
ABAP Program                  /SAPAPO/SAPLCDPS_FUNC_DISPATCH
Application Component      SCM-APO-PPS

Other Terms

MOVE_TO_LIT_NOTALLOWED_NODATA, /SAPAPO/FDIS_DISPATCH_FUNCTION, /SAPAPO/FDIS_BEFORE, CL_PPDS_RES_SCH_BO=>DISPATCH_FUNCTION.

Reason and Prerequisites

The dump occurs only when the BAdI "/SAPAPO/CDPS_FDISHEU" is active due to programming error.

Solution

Implement the attached correction instructions or import the support package relevant for your release.
