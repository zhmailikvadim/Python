Symptom

In SAP S/4HANA Cloud Public Edition, when executing the following tasks in the Data Monitor for Group Reporting:

  • 1080 – Validation of Reported Data
  • 1180 – Validation of Standardized Data

The system returns the following message:

"Task is obsolete. New Balance Validation is activated."

The tasks cannot be executed.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Open the Data Monitor app.
  2. Fill in the required Global Parameters.
  3. Run the Validation for Reported Data task (1080) and Validation for Standardized Data task (1180).
  4. Observe the following error messages:

"Task 1080 is obsolete. New Balance Validation is activated."

"Task 1180 is obsolete. New Balance Validation is activated."

Cause

The New Balance Validation framework is activated in the system.

In recent releases of SAP S/4HANA Cloud Public Edition, was introduced a new validation framework called Balance Validation for financial data validation.

When this framework is active:

  • The legacy validation tasks in the Data Monitor are no longer used.
  • This includes:

1080 – Validation of Reported Data
1180 – Validation of Standardized Data

As a result, the system automatically marks these tasks as obsolete and displays the error message.

Resolution

When the Balance Validation framework is active, validations must be executed using the Balance Validation apps instead of the legacy Data Monitor tasks.

Use the following applications:

Maintain Validation Rules and Rule Groups

Use:

"Manage Balance Validation Rules and Rule Groups" app

This app allows you to:

  • Define balance validation rules
  • Create and maintain rule groups
  • Assign rules to rule groups.

Execute Validation and Analyze Results

Use:

"Run Balance Validation and View Results" app

This app allows you to:

  • Execute balance validation runs
  • Select the relevant rule group
  • Review validation messages and results.

For further details and guidance, please refer to the following blog post: SAP S/4HANA Group Reporting - New Balance Validation Process and the KBA 3577657 - How to finish Group Reporting New Balance Validation Setup.
