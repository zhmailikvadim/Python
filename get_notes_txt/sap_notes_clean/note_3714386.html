Symptom

You want to integrate Joule with SAP Signavio solutions, and you require credentials (Client ID, service key, etc.) to configure the destinations for using the navigation service.

Cause

Resolution

  1. Create a new instance for integration between the SAP BTP subaccount created for Joule and SAP Signavio solutions.
  2. Navigate to the JSON tab after creating the instance.
  3. Locate the service key details and other information in the JSON tab.

See Also

Refer to: Configure Destination Service and Destinations
