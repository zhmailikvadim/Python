Symptom

An MPD file of a snapshot contains BOM component assignments for instances that belong to an Alternative Item Group. When transferring the items in the PiP Workspace, an error message MPE_RTGV_API 181 is raised (Instances in an Alternative Item Group must be assigned to same OA.). However, all instances of the component are assigned to the same Operation Activity.

Other Terms

MSFR1, MSFR4, Snapshot,

Reason and Prerequisites

Coding error.

Solution

Implement the correction instruction.
