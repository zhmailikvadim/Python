Symptom

You work with My Inbox Master-Detail layout using a screen reader. When you select a task in the Task List, task details are opened in the Details Area, however the screen reader does not provide this information. In addition you do not know how to navigate to Details Area to read task details with the screen reader.

Other Terms

Fiori My Inbox, F0862

Reason and Prerequisites

Missing functionality. 

Solution

Apply the referenced SAP Note corresponding to your My Inbox Support Package, use SMICM transaction to clear ICM cache as described in SAP Note SAP Note 2319491 - How to clean up the cache after applying changes that affect SAP Fiori apps, and clear the end user browser cache.
