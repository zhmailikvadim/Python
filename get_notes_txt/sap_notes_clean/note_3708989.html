Symptom

Error while performing system rename of AAS using Software Provisioning Manager Tool (SWPM SAP Note: 1680045) 

Error message –
Cannot use instance 'D00' for this option. The instance must start with 'DVEBMGS'.

Environment

 SAP ERP 6.0

Resolution

As per the system rename guide, it is not possible to rename Additional application server instance.

So, uninstall AAS and install it again.

 

For more information, please check the system rename guide:

https://help.sap.com/docs/SOFTWARE_PROVISIONING_MANAGER/30839dda13b2485889466316ce5b39e9/c35a204cff6c4e15bc78810cb65fb494.html
