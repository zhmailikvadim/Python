Symptom

Purchase order standard screen Item Overview manage layout button not appear in ME21N, ME22N, ME23N in S/4 HANA and getting the same option in the ECC system.

Environment

Materials Management (MM)
Version for SAP HANA
SAP S/4HANA

Reproducing the Issue

1.Create PO through ME21N.
2.Check manage layout button to add the tax code field in Item Overview.

Cause

Standard limitation in SAP.

Resolution

The item “overview” grid in ME21N/ME22N/ME23N is driven by a fixed data/ALV structure. Only fields that are part of that underlying structure (and exposed to the ALV/CDS/OData used by the UI) can be added to the item overview via the normal screen-layout/change-layout options. Many fields that live in the “Item Details” dialog are stored or exposed differently, so they won’t appear as selectable columns for the overview.

So as per standard system design of SAP, in purchase order Item Overview grid, manage layout button not appear in ME21N, ME22N, ME23N in S/4 HANA and you cannot add fields like tax code which are available in item details to item Overview Tab through screen layout option.

Keywords

ME21N, ME22N, ME23N, PO, Item overview, Item details, screen layout, fields, tax code.
