Symptom

Clarification is requested on whether the SAP Start product can be used by subscribing to the SAP Build Work Zone – Build-Default Plan.

Environment

  • SAP Cloud Foundry 
  • SAP Business Technology Platform
  • SAP Start

Cause

The SAP Build Work Zone – Build Default Plan does not support SAP Start. 

Resolution

  • SAP Start cannot be used with the SAP Build Work Zone – Build-Default Plan.

  • SAP Start can be set up with different SAP Build Work Zone, standard edition service plans: foundation or standard. Please refer Service Plans and Metering | SAP Help Portal 

Keywords

SAP Start, Service plan metering, build-default plan, Work zone, Foundation plan, build-default
