Symptom

  • The Review Availability Screen (RACR) is not functioning as expected.  
  • The screen appears frozen.  
  • Three dots are displayed, but no data is loaded.  
  • An error message regarding missing ODATA service is shown. 

DISCLAIMER: Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

• S/4 Hana  
• Advanced ATP  
• Availability check  
• Available to Promise (ATP) 

Reproducing the Issue

1. Perform ATP check in transaction VA01.  
2. Perform ATP check in transaction VA02.  

Cause

  • Missing or wrong ODATA customizing.

Resolution

Verify the system settings via these steps:

1. Access transaction "/UI2/FLPCM_CUST".

2. Navigate to the "Tiles/Target Mappings" tab.

3. Search for "ATPCHECK".

4. Select the second entry and navigate to "Services". Choose "Check and Show Services".  

5. Verify that all traffic lights are green.

 

6. Navigate to the third tab "ICF services" and confirm the traffic light status.

7. Ensure the following user roles are assigned for proper RACR screen display:

  • SAP_BR_INTERNAL_SALES_REP
  • SAP_BR_ORD_FLFMT_MNGR_SCHDG

8. Further common customizing issues are covered by FAQ KBA 2981977 - FAQ: RACR - Review Availability Check Result - Screen.
