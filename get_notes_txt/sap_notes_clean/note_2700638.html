Symptom

While creating a new Private or Corporate Account from the Account Management work center, the Country and Language fields are automatically filled.

Environment

SAP Business ByDesign

Reproducing the Issue

  1. Go to the Account Management work center
  2. Go to the Common Tasks
  3. Select New Corporate Account or New Private Account
  4. You can see the Country and Language fields are automatically already filled

Cause

When creating an Account, the defaulting of Country and Language fields is user dependent. This means the field will be filled with the last Country / last Language entered by the user.

If this does not occur in other tenants, probably all users have maintained Accounts and entered Country / Language fields, and therefore get the defaults suggested from the last entries. You may check the same for a certain user. That is, create an Account with setting the correct Country and Language and Save. Now create the next Account and you see that Country and Language will be defaulted.

Resolution

It is the system behavior. There is no configuration/setting to set this country auto-filled during Account creation.
