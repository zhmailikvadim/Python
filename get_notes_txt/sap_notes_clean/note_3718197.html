Symptom

  • When accessing the Carrier Selection button → Available Carriers from the Transportation Cockpit FOR Quadrant, carriers maintained on the lane are displayed correctly.
  • There is no Save or Confirmation button available to assign the selected carrier to the Freight Order.
  • The system displays available carriers but does not provide functionality to complete the assignment process from this view.

Environment

  • SAP S/4HANA Cloud Private Edition;
  • Transportation Cockpit.

Reproducing the Issue

  1. Open the Transportation Cockpit and navigate to the Freight Order section.
  2. Select a Freight Order and click on the Carrier Selection button.
  3. Select Available Carriers.
  4. No 'Save' button is displayed.

Cause

  • Standard behavior;
  • The "Available carrier" button in the Transportation Cockpit screen evaluates carriers for each business document based on specified carrier selection settings but does not provide functionality to perform carrier assignment or save ranking results;
  • The screen is designed as a display-only interface with no edit mode enabled;

Resolution

Refer to SAP Note 3227861 - Lead/Line Selection in Carrier Selection Result Screen, for verification.

This note was created to disable the selection mode (line selection/lead selection) in the Carrier Selection Result Screen as it is currently non-functional.
