Symptom

You send an OData V4 request and expect a PDF as response of the SAP Gateway framework. The timestamp fields within the response appear in the PDF formatted like e.g., "19.770.724.000.000,0000000" instead of the expected "24.07.1977 00:00:00,0000000".

Other Terms

SAP Gateway, OData V4, PDF, timestamps, TIMESTAMP, TIMESTAMPL

Reason and Prerequisites

The type information for fields based on TIMESTAMP and TIMESTAMPL was not correctly forwarded to the point where the PDF is rendered and therefore those timestamps appear in their packed number representation.

Solution

Kindly implement this Note.
