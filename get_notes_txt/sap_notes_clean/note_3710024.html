Symptom

While creating defects for components assigned to the operation activity, the defective quantity is validated against the available quantity of the operation activity. This issue is observed only for production order of non-serialized material.

Other Terms

Defect, Non Serialized, Component, Defective Quantity, Planned Quantity, Required Quantity, Record Defect, Perform Work

Reason and Prerequisites

Program Error

Solution

After implementation of this SAP Note, the system will not validate the defective quantity while creating defects for components assigned to the operation activity.
