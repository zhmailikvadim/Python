Symptom

Received the below error while trying to perform testConnection after setting up variables as defined in Setting Up Environment Variables for touchless deployment.

User authentication failed!
Could not establish http connection!

Environment

  • SAP Sales Performance Management
    • Incentive Management on Legacy HANA 

Reproducing the Issue

  1. Open Command prompt.
  2. Navigate to bin folder from the unzipped folder of TDClient sctipt.
  3. Declare environment variable with the following commands:

    SET TD_BASEURL=https://1234.callidusondemand.com/
    SET TD_USERNAME=<user> (a user who has access to IM)
    SET TD_PASSWORD=<password>

  4. Verify connectivity by running the command : TDClient.cmd -o testConnection
  5. Receive the mentioned error.

Cause

The system fails to establish a proper connection when the BASEURL is declared with a trailing forward slash (/). 

Resolution

  1. Update BASEURL to the following format: https://1234.callidusondemand.com.
  2. Retest the connection using the command: TDClient.cmd -o testConnection.
