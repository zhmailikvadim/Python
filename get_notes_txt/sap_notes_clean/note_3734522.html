Symptom

  • Clicking "Edit Prompt Values" has no responase when scheduling Web Intelligence in BI Launch Pad
  • Unable to change values by "Edit Prompt Values" when scheduling a Web Intelligence document from the Launch Pad.

Other Terms

WebI, schedule, BI 2025, prompt, edit, no response

Reason and Prerequisites

Environment:

  • SAP BusinessObjects Business Intelligence Platform (BI) 2025 Patch 9
  • Web Intelligence (WebI)
  • Scheduling in BI Launch Pad

Steps To Reproduce:

  1. Logon BI Launch Pad.
  2. Location one WebI report which base on BW OLAP connection.
  3. Schedule the WebI Report.
  4. Navigate to "Report Feature" tab and "Prompts" section.
  5. Click Edit Prompts Values.
  6. It can click and show up the Prompt dialog.
  7. Try to change one parameter for prompt.
  8. Click Apply and page will go back.
    Note that the select value did not reflect in the prompt values.
  9. Click the Edit Prompt Values again.
    Note that there is no responase for edit prompt, all the buttons in this page has no response.
    Scheduling works fine in CMC.
