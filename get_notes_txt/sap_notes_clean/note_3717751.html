Symptom


After upgrading the system, the "Back" button is missing from the updated version of the T-code CO11N web page.

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental. 

Environment

  • SAP Production Planning (PP)
  • SAP S/4HANA (All versions) 
  • SAP S/4HANA Cloud Private Edition (All versions) 
  • SAP enhancement package for SAP ERP (All versions) 
  • SAP enhancement package for SAP ERP, version for SAP HANA (All versions) 

Cause



Resolution

Use the newly introduced buttons as a replacement for the missing "Back" button.
