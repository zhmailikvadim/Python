Symptom

We've identified an issue in report RPCPEFT0 involving the calculation of mandatory old-age savings contributions for hazardous work. The recently added wage types M840 and M841 are mistakenly allocating payments to participant "U" (employee contribution) rather than the correct participant "Z" (employer contribution).

Other Terms

RPCPEFT0, HRCZ

Reason and Prerequisites

This SAP Note is relevant for the Czech Republic only.

Solution

Report RPCPEFT0 has been modified to properly allocate employer payments for wage types M840 and M841 as described in the issue details.

Important: Apply current version of SAP Note 875986 for NW 7.0x or 1668882 for NW 7.3x and higher. Otherwise an old version of SNOTE can lead to missing coding changes when applying SAP Notes.
