Symptom

You executed a query over the SEM-BCS virtual provider for totals records. Filtering a characteristic of the role Consolidation Group, the runtime error GETWA_NOT_ASSIGNED can occur.

Other Terms

SAPLUCR0, LUCR0U07

Reason and Prerequisites

Prerequisites

You restricted the characteristic of the role Consolidation Group per query element.

Reason

There is a program error.

Solution

Implement the correction instructions or import the Support Package.
