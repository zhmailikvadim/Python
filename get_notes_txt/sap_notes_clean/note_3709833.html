Symptom

  1. In Fiori launchpad, all the tiles in error state: cannot load tile
  2. App can be opened with direct link
  3. Chekc the start_up request, the target mapping are returned
  4. Adding the SAP_ALL, the tile will display correctly

Environment

SAP S/4HANA 1909 or higher

Reproducing the Issue

  1. Access the Fiori Launchpad, there is error: cannot load tile
  2. Adding the SAP_ALL, tile will display correctly

Cause

Missing FLP end user role

Resolution

  1. refer to the following docuemnt to create a flp user role: https://help.sap.com/docs/ABAP_PLATFORM_NEW/a7b390faab1140c087b8926571e942b7/85be3fff35604fa09a1668dd97ef4407.html?locale=en-US&version=LATEST
  2. assign the role to the user

Keywords

fiori, sap_all, cannot load tile
