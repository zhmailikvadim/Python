Symptom

In transaction /PACG/ECM_SC_GR, the source object for the region determined for the service order can be defined. The object region of the service order is a combination of the source object (e.g. the planner group or the work centre) and the plant. However, in the service order these values can be taken from the different plants. Due to the incorrect plant selection the value of the region is created with the wrong plant.

Other Terms

FSM, E4C, Connector, Region, Planner group, Work Centre, /PACG/ECM_SC_GR, /PACG/ECM_IS_GROUPDATA

Reason and Prerequisites

Incorrect logic in /PACG/ECM_CL_S_IDOC_O_SCALL→IDOC_OUTPUT.

Solution

Please, implement this correction instruction.
