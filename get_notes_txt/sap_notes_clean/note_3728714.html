Symptom

When you click Goto → Accounting Documents, the system issues the message:"No subsequent document found in Accounting." in transaction PCP0.

Environment

  • Human Capital Management (HCM)
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP SuccessFactors Employee Central Payroll
  • SAP S/4HANA

Reproducing the Issue

  1. You are running HR and FI in different SAP systems and posting results via ALE connection.
  2. You have successfully posted the payroll result to FI.
  3. You attempted to access the FI document by navigating to Goto → Accounting Documents.
  4. The message "No subsequent document found in Accounting." is displayed.

Cause

The method DocumentDisplay (display accounting documents) of the object type AcctngServices is missing in BD64 configuration.

Resolution

Assign an RFC destination for the special method "DocumentDisplay" (display of the accounting documents) of the object type "AcctngServices" in the HR system.

  • You can configure this in ALE Customizing under:
    • Sending and Receiving Systems → Systems in Network → Synchronous Processing → Determine RFC Destinations for Method Calls
