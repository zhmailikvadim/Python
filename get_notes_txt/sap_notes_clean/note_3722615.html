Symptom

In the SAP Fiori app Mass Maintenance of Bills of Material Items (F6578),the user selects the "Production Relevant" field within the Select Attributes dialog to add it to the mass change scope.

Upon clicking the Simulate button to validate the changes, the application fails to process the request.The system returns a technical error message: "XML Error while parsing an XML stream".

The simulation results are not displayed, and the mass maintenance process cannot proceed.
