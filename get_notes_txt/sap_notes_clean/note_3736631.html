Symptom

How to change the "Card Title" of a custom MDF Object in the Configure Latest People Profile?

Disclaimer: Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

SAP SuccessFactors HCM

Reproducing the Issue

  1. Open Configure Latest People Profile.
  2. Locate a custom MDF Object card added to the profile.
  3. Observe the Card Title displays in the upper section of the page and another one below the General Information.

Cause

Resolution

To rename the Card Title of a custom MDF object displayed in Configure Latest People Profile, please update the “Label” field in the Configure Object Definition for the specific custom MDF object.

Kindly follow the steps below to do so.

  1. Go to Configure Object Definitions.
  2. Select the relevant custom MDF object.
  3. In the object’s properties, update the "Label" field to the desired Card Title.
  4. Save the changes.
  5. Return to Configure Latest People Profile and verify the card now shows the updated title.
