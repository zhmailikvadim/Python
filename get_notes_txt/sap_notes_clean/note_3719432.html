Symptom

This note addresses an issue related to the creation of measurement documents at the selected time in the Central/Remote system when the change protection for the measuring point was or is available in the respective system at that time.

The correction introduces under the disconnected switch in the 'Save Cycle' of Measurement Document. Previously, the system behavior followed the logic implemented in the ELSE condition of the Disconnected switch.

With this change, the system now properly handles measurement document creation when the change protection for the measuring point exists in the corresponding system.

Other Terms

S4 Defense Disconnected, Measuring Document, IK21, IK22

Solution

Implement the attached correction instructions provided with the NOTE.

Note : Translation for Message DFS_DISCONNECTED_EXT  137, will be shared shortly.
