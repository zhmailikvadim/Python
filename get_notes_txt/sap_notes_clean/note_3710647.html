Symptom

  • SAP CI-DS agent is down with the error present in DSOD logs: "Stopping Ghost Agent since the org is deactivated/deleted".
  • After KYMA migration agent is not able to connect to the cloud tenant.

Environment

SAP Cloud Integration for data services

Cause

"As mentioned in Prerequisites and Restrictions for the Tenant Migration, the minimum Agent version on the Kyma environment is the 2409 (1.0.11.6065) release. This Agent version has a script that can be used to easily change which server the Agent points to. " -  Configure Agents In The New Sandbox Environment | SAP Help Portal 

Resolution

  1. Update the agent to a version higher than 2409 (minimum required version: 1.0.11.6065).
  2. Reconfigure the agent to connect to the KYMA environment.
