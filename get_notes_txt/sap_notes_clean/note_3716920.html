Symptom

This note delivers a new version of the print form 033076 - Receipt/export notification for the transport of excise goods under suspension of duty/in free circulation using the default procedure for the fallback procedure of the Excise Movement and Control System (EMCS) Germany.

The form /SAPSLL/PF_DE_EMCS_ROR2 replaces the form /SAPSLL/PF_DE_EMCS_ROR.

The interface /SAPSLL/PF_DE_EMCS_ROR2 replaces the interface /SAPSLL/PF_DE_EMCS_ROR.

The following corrections were made with the versions of the form:

    • Initial version
