Symptom

When running a report in the app Balance Sheet/Income Statement - Multidimensional (App ID W0161), selecting many company codes with different company codes currencies, error messages were displayed: 

No shared currency type for the selected company codes

Currency type 10 is not unique with regard to the currency as company code currency

Error while reading data; navigation is possible

Result set for query 2CCFIFINSTMTCMP is empty

Environment

SAP S/4HANA Cloud Public Edition.

Reproducing the Issue

  1. Open to the Balance Sheet/Income Statement - Multidimensional app.
  2. Fill in all mandatory fields.
  3. Select multiple company codes with different company code currencies.
  4. Execute
  5. Observe the error message: "Result set for query 2CCFIFINSTMTCMP is empty."

Cause

The error occurs when multiple company codes are selected that have different currency codes assigned to currency type 10.

The Balance Sheet/Income Statement - Multidimensional app uses query 2CCFIFINSTMTCMP, which requires all selected company codes to have the same currency for the specified currency type.

Resolution

Ensure that all selected company codes have the same company code currency for currency type 10.
