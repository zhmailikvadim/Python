Symptom

  • When one user displays a PDF for an EDocument in the EDOC_COCKPIT another user cannot open the same EDocument in PDF.
  • Error Message no. EDOCUMENT005 "Source document comp.code &1/ type &2/ key &3/ log.system &4 is locked" is reported when another user tries to display the same EDocument in PDF.
  • Multiple users are not able to display an EDocument at the same time in PDF.

Environment

  • SAP S/4HANA
  • SAP ERP
  • SAP Document and Reporting Compliance Service

Reproducing the Issue

  1. Access EDOC_COCKPIT transaction.
  2. Navigate to the KSeF edocuments.
  3. Select an eDocument.
  4. Attempt to display the PDF using the Display PDF button.
  5. Observe that the Edocument displays error Message no. EDOCUMENT005 "Source document comp.code &1/ type &2/ key &3/ log.system &4 is locked".

Cause

Only 1 user at a time can open an EDocument in the EDOC_COCKPIT and display it in PDF. 

Resolution

It is standard system behavior that when a user opens an EDocument from the EDOC_COCKPIT that this document is then locked so that no other user can access it. Therefore multiple users are not able to open and display in PDF the same EDocument.
