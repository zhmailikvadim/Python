Symptom

How to create an case for Agentry Editor issue?

Environment

  • SAP Mobile Platform SDK 3.x

Resolution

Please provide the content below in the case and send to SAP  for further investigation.

  • Problem Description
  • Test Environment (For example: Agentry application version, Agentry Editor version, SMP Runtime Server version, Agentry Client version and Agentry AddOn version)
  • How to reproduce this issue with detailed information or use some screenshots to illustrate the issue in details
  • Events.log (Path: %SMP_HOME%MobilePlatform3\Server\log\agentry)
  • ServerName-smp-server.log with debug level (Path: %SMP_HOME%MobilePlatform3\Server\log) How to adjust the log level on SMP Cockpit for Agentry? Please see KBA 2137792 for reference
  • Analysis result with detailed information or use some screenshots to explain your analysis

Keywords

How to, Create, case, Ticket, Agentry Editor, SMP
