Symptom

  • Replication stopped working 
  • Observer job is not running
  • Replication is not working between source and target
  • User BATCH_USER is not authorized to run LTB Job Dispatcher
  • /LTB/JOB_DISPATCHER, SAP_DHCDC_DISPATCHER and SAP_DHCDC_HOUSEKEEPER are stuck post security patch on April 2nd
  • /1DH/OBSERVE_LOGTAB is not running post security patch on April 2nd

Environment

ABAP CDC Engine

Cause

On April 2nd there was a security patch that restricted authorizations on the BATCH_USER in Client 000

Resolution

  1. Run the select statement - Select * from "/LTB/JOB_QUEUE" in DBACOCKPIT or the DB level
  2. If there is an entry in /LTB/JOB_QUEUE for client 000 where JOB_NAME = /1DH/OBSERVE_LOGTAB or /1DH/PUSH_CDS_DELTA they should be deleted, to delete the entries open a case in component PCO-OPS
  3. After the entries are removed from /LTB/JOB_QUEUE go to the correct client (not client 000), run transaction DHCDCMON and select Dispatcher Job
