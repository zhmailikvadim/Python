Symptom

  • A custom CDS view created in the Custom CDS Views app is not visible in that app.
  • The custom CDS view is visible in the View Browser app with status "Draft".
  • The custom CDS view cannot be edited or deleted because it does not appear in the Custom CDS Views app.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Create a custom CDS view using the Custom CDS Views app.
  2. Open the Custom CDS Views app and try to find the created view.
  3. Observe that the view is not listed in the Custom CDS Views app but appears in the View Browser app with status "Draft".
  4. Attempt to delete the custom CDS view; deletion is not possible because the view does not appear in the Custom CDS Views app.

Cause

The custom CDS view is in draft status and owned/locked by another user. By design, draft versions are visible and manageable only to the owner in the Custom CDS Views app.

Resolution

  1. Identify the user who owns or has locked the draft custom CDS view via app Extensibility Inventory. 
  2. Ask the owner to open the Custom CDS Views app and either publish the draft or delete/discard it, as required.
