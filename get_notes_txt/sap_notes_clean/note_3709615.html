Symptom

Services for Groupware Integration - groupwareSsiService is missing for business roles.

Environment

SAP Sales Cloud and SAP Service Cloud version 2

SAP Sales Cloud version 2

SAP Service Cloud version 2 

Reproducing the Issue

  1. Go to settings.
  2. Search for Groupware.

Result: Observe that groupware is not available.

Cause

Groupware is available only for Sales tenants and is not available for Service tenants.

Resolution

  1. Confirm that the tenant is categorized as a service tenant.
  2. Note that groupware integration is not available for service tenants as it is a sales-related service.
  3. No further action is required as this is the system's expected behavior.

Keywords
