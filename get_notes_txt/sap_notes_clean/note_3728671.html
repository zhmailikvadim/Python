Symptom

How to edit List of Choices for a Global and Custom terms

Resolution

1. All terms are stored in the database with a uniquename:

  • Terms prefixed with GITA indicate that the term is Global.
  • Terms prefixed with RITA are classified as Local.

2. When a new term is created and published from a Sourcing Event Template, the term is available across the site for use in all other templates, events and contract workspaces. 

3. If any properties related to Global Term need to be modified, then user need to edit it at Sourcing Event Template level. Example : List Of Choices, Answer Type, etc. 

4. If any properties related to Local term need to modified, then user can edit at any template level where that term was created.
