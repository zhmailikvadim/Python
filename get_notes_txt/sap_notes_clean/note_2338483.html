Symptom

Under certain circumstances, an endless loop occurs in the function ANAB or AWPBP if a day in NCALE is processed with the T5A4Q identifier '..'.

Other Terms

RPCALCA0
Absence valuation
ABSV
ABSTY
Attendance/absence category

Reason and Prerequisites

There is a program error.

Solution

Import the corresponding HR Support Package into your SAP system.

For an advance correction, proceed as follows:

  • Upload the attached correction instructions into your SAP system using transaction SNOTE.
  • Perform retroactive accounting for the affected persons.
