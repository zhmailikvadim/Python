Symptom

'Apply Completes Only' column contains negative values within the Applicant Conversion Report in Advanced Analytics.

Environment

SAP SuccessFactors Recruiting Marketing 

Reproducing the Issue

  1. Access the System
  2. Navigate to the Recruiting Module
  3. Navigate to the Applicant Conversion Report within Advanced Analytics.
  4. Locate the 'Apply Completes Only' column.
  5. Observe negative values displayed in the column.

Cause

With the upcoming transition to SAC from Advanced Analytics, Product Management has made the decision to focus their efforts on the new platform, so they will not be addressing this issue in the current system.

Resolution

Do not use the data contained within the 'Apply Completes Only' column in the Applicant Conversion Report within Advanced Analytics.
