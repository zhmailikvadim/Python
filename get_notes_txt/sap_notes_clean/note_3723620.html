Symptom

  • Users find the special instructions field in the content import tool confusing.

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

SAP SuccessFactors Learning

Resolution

  1. The special instructions field in the content import tool is included only in the email notification ContentDeploymentNotification.
  2. This notification is typically utilized by Premium Content Management service (PCM).
  3. The PCM team receives the email notification to test the uploaded content.
