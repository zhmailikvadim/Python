Symptom

  • The quality seal field is ignored during the Excel import process for application fact sheets.
  • The issue occurs when attempting to import 2,300 items using Excel.
  • The inability to include the quality seal during import leads to manual updates being required for each fact sheet.

Environment

LeanIX Inventory Excel Import

Reproducing the Issue

  1. Prepare an Excel file containing application fact sheets, including the quality seal field.
  2. Import the file into the LeanIX Inventory system.
  3. Observe that the quality seal field is ignored during the import process.

Cause

The quality seal is intentionally ignored during Excel import to ensure individual review of each fact sheet.

Resolution

1.Utilize Automations within LeanIX to address the issue:

a. Add a tag or a field change to act as a trigger for the required updates.

b. Use the Excel import to set the trigger across the relevant fact sheets.

2. Refer to the LeanIX documentation for Automations to implement the solution:
