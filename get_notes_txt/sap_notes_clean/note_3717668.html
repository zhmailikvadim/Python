Symptom

  • The DRC Colombia Income Withholding Tax Declaration report (CO_WHT_INCOME_DCL) generates with errors.
  • The error message states: "Length of the value XXXXXXXXXX exceeds the max. permitted length 60." at Element Name BUSINESSPARTNERNAME.
  • The issue is related to the name of Business Partners (BPs), where Name 1 and Name 2 are being shown.

Environment

  • SAP S/4HANA
  • DRC Colombia
  • Statutory Reporting
  • Withholding Taxes

Reproducing the Issue

  1. Run statutory report by selecting report name CO_WHT_INCOME_DCL
  2. Generate a new run and observe the error message "Generated with Errors."

Cause

The issue is caused by incorrect data at the source document (for One Time Business Partner) or at the Master Data from the Business Partner.

Element BUSINESSPARTNERNAME is defined at CDS view P_StRpWhldgTaxItem, field Name. For more information on source of the field, check the CDS view.

Resolution

  • The partner's master data should be corrected.
  • It is expected that at field Name 1 the first name is informed and at Name 2 the surname.
  • Try to use a total of 60 chars at fields Name 1 and Name 2 from the Business Partner Master Data. Avoid repeating names to save chars.

Keywords
