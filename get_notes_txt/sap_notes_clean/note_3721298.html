Symptom

In Asset History Sheet Fiori reporting App (F1615) there are some Key Figures missing in the output.

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental"

* Key Figure 10 is not reported.

Environment

  • FIN Asset Accounting
  • SAP S/4HANA
  • SAP S/4HANA Finance
  • SAP S/4HANA Cloud Private Edition
  • SAP Fiori for SAP S/4HANA Finance,
  • SAP Fiori for SAP S/4HANA,
  • SAP Fiori for SAP S/4HANA Cloud Private Edition

Cause

Reported missing KFs are marked as 'Hidden' for the Key Figure Group (Indicator: Key Figure is hidden when Balance Amount is zero) in SM34 for View Cluster VC_FAAC_KFCSET_AHS for affected Key Figure Group:

Resolution

Unhide these KFs if you need to always see these reported via SM34 -> View Cluster VC_FAAC_KFCSET_AHS selecting affected Key Figure Group.
