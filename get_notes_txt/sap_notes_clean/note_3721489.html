Symptom

  • The activity id "Prepare and Save GSTR1" is not available; instead "Save GSTR1" is displayed.
  • Clicking "Save GSTR1" does not open the "Manage Outgoing Documents" activity.

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Run the statutory report India GSTR1.
  2. Observe that the option "Prepare and Save GSTR1" is not available; only "Save GSTR1" is displayed.
  3. Click on "Save GSTR1" and observe that the "Manage Outgoing Documents" application does not open.

Cause

Missing configuration

Resolution

  1. Make sure to have activated 5XU.
  2. Ensure that the "Document and Reporting Compliance Features" (SSCUI ID - 105518) are configured to select the countries/regions where document and reporting compliance features are required.
  3. Open the "Run Statutory Reports" application and follow these steps:

a. Click on "Reporting Tasks".

b. Select "Update Reporting Tasks".

c. Provide the Reporting Entity (e.g., IN_1200_BM), Report Name (e.g., IN_GSTR1_DCL), and Reporting Year (e.g., 2025).

d. Click on "Update".

e. Once updated successfully, provide the Report Name, Reporting Entity, and Reporting Year, then click on "Go".
