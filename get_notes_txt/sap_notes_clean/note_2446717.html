Symptom

The note resolves the following issues

  • When extracting the quality codes the quality codes are not restricted to the ones assigned to a specific quality notification type for which the quality codes are being extracted.
  • When processing inbound QualityNotificationRequest cXML message there is an error because of missing priority type AN.

Other Terms

Ariba Network Integration, QualityNotificationRequest, cXML, SAP Add-On

Reason and Prerequisites

Program Error

Solution

Implement the code changes in the attached note or apply the relevant support pack.
