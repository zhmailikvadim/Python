Symptom

You declare your transit procedures to the Dutch NCTS from SAP GTS.

The customs authority responds to an incorrect transit declaration with a negative acknowledgment (IE906). The inbound IDoc for this message cannot be updated in SAP GTS and obtains the IDoc status Application document not posted (51) with the error message The message (IE906) does not exist in the current activity (NLVVA) (/SAPSLL/CUS_ECC 593).

In the IDoc segment /SAPSLL/E2CCEHD_CNTRL, the field CUMES is filled with the value IE906. In the segment /SAPSLL/E2CCEHD, the field EDINR is filled with the value IE015000001.

Other Terms

/SAPSLL/CUS_ECC593

Reason and Prerequisites

Program error

Solution

Make sure that your converter uses message conversions from at least Version GTSNL_2300489_20170914_V1-11 in SAP Note 2300489.

Implement the Customizing changes as described in the correction instructions.
