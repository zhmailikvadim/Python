Symptom

WARNING: This is an SAP Tax Declaration Framework Note. If you do not have SAP TDF installed in your system, this SAP Note does not apply to you. SAP TDF requires a specific license.
******************************************************************************************************************************************************************************************************************

This SAP Note provides you with corrections to the following reports:

  • EFD ICMS IPI (/TMF/EFD) report:
    • Register 0450: The use of /TMF/D_COD_INF_D table is not working properly to get the text description from Additional Information Code (COD_INF) field.
  • Customer Influence 291932:
    • Include the Company Code and the Branch in the naming standard of the files generated by the SPED reports.

Other Terms

TDF, Brazil, SPED, EFD ICMS IPI, 0450.

Reason and Prerequisites

This SAP Note is relevant only for Brazil.

You have implemented SAP Note 3711978.

You have upgraded your system to SAP TDF Support Package 20.

Solution

After implementing this SAP Note, the updates listed below are made to your system:

  • /TMF/EFD report:
    • Register 0450: The CA_TXT_COMPL calculated field on the J_P_COD_INF node is updated to get the text description from /TMF/D_COD_INF_D table as priority. As a result, the register 0450 is generated correctly.
  • Customer Influence 291932:
    • The Company Code and Branch have been added to the beginning of the file name generated by all SPED programs in the GET_FILE_NAME method of class /TMF/CL_SPED. As a result, all generated SPED files will now include the Company Code and Branch in their file names.

As a general rule, SAP recommends that you implement the Support Package specified for your release. However, if you need to install a solution earlier, use the SAP Note Assistant to implement the correction instruction. You can find more information about the SAP Note Assistant in SAP Support Portal, under SAP Note Assistant.
