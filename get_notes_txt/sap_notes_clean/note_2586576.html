Symptom

Can we configure the Completion Status fields in Record Learning Tool?

Environment

SAP Successfactors Learning

Reproducing the Issue

  1. Under the Record Learning Tool, notice that you will be prompted to fill out fields relevant to the completion status / item details to be recorded
  2. Customer wants to change these fields

Resolution

The fields that you need to fill out when using the Record Learning Tool (Completion Status details like Completion Date/Time/Time Zone/Grade/Total Hrs/Credit Hrs/Contact Hrs/CPE) is standard. We can't add or remove other fields.

See Also

2413745 - How to record learning in LMS
