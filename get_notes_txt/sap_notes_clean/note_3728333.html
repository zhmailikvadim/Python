Symptom

Component: Incident Management
Module: Safety Observation

When you open a Safety Observation in the Manage Incident app and click "Classification", additional or nonexistent groups can be selected in certain languages. There's also no fallback mechanism when translations aren't available in a language.

Other Terms

CL_EHHSS_SO_GROUP, GET_TABLE, EHHSSC_SO_GRP, EHHSSC_SO_GRP_T

Reason and Prerequisites

Reason:
Program error.

Prerequisites:
For information about the validity of the corrections, see the correction instructions.

Solution

After you implement this SAP Note, only the groups that have been set up in customizing are shown on the Safety Observation classification tab. If the selected language has no translation, the system uses the default language.

The "Support Packages" section lists the support packages that contain these corrections.

Alternatively, you can implement the attached correction instructions.
