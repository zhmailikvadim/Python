Symptom

This SAP Note provides you with corrections in the Argentina Journal Accounting Book (J_1AF004) report. It addresses an error in reporting documents spanning multiple fiscal years.

When you run the J_1AF004 report for a period spanning multiple fiscal years, the system does not display all documents in the report output.

Other Terms

Argentina, AFIP, Compact Document Journal, Journal Accounting Book, Libro Diario, J_1AF004, Fiscal Year, S_ALR_87012294.

Reason and Prerequisites

This SAP Note is relevant only for Argentina.

Solution

After implementing this SAP Note, the system correctly displays all relevant documents in the J_1AF004 report output when the reporting period spans multiple fiscal years.

As a general rule, SAP recommends that you implement a solution by implementing a Support Package. However, if you need to implement a solution earlier, use the Note Assistant to implement the correction instruction. You can find more information about the SAP Note Assistant in the SAP Support Portal, under SAP Note Assistant.
