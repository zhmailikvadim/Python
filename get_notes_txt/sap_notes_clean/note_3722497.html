Symptom

You want to configure the SAF-T Financial version 1.3 using the new Grouping Category and Grouping Codes, as specified by the Norwegian Tax Authorities, but some Grouping Categories and Grouping Codes are missing for the Public Sector.

Other Terms

SAFTN_G, SAFTN_E, SAFTN_D, SAFTN_M, SAF-T, SAF-T Financial, Skatteetaten, SNI_NO_REL_GRP, SNI_NO_REL_GRPT, SNI_NO_STD_GRCT, SNI_NO_STD_GRCTT, Run Compliance Reports, Version 1.3

Reason and Prerequisites

New Grouping Categories and Grouping Codes specified by the Norwegian tax authority.

This SAP Note is relevant only for Norway.

Solution

This note updates the SAP system tables with Grouping Codes for the following Grouping Categories:

Grouping Category Description
KA_K KOSTRA Arts Kommune
KA_KS KOSTRA Arts KS
KS Kontoplan Stat

As a general rule, SAP recommends that you install a solution by applying a Support Package. However, if you need to install a solution earlier, use the Note Assistant to implement the correction instruction.

You can find more information about the Note Assistant in Support, under https://support.sap.com/en/my-support/knowledge-base/note-assistant.html.
