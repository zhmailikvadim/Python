Symptom

I need to add custom fields to a rate schedule and I would like to know how to do it

Resolution

  1. Sign into your SAP Fieldglass account
  2. Admin Configuration (Gear Icon) > Rate Schedule
  3. Locate and open the desired Rate Schedule > Edit
  4. Click Continue > Under Custom Fields section, click + Add Custom Fields
  5. Located the desired custom field, chick the respective check box > Add Selected (You may select multiple custom fields to add all at the same time)
  6. Chick the Mandatory check box and set a Default Value if applicable
  7. Click Update 

Note: If you want the above changes to affect existing workers, make sure the Rate Schedule field Use Admin Structure is set as Yes and complete a Work Order Revision for the respective workers
