Symptom

In SAP S/4HANA Cloud Public Edition, error message: SV167 - Enter a value within the allowed work area is raised when selecting application area in SSCUI 103887 Message control in tax area. For example, the application area "FS" is selected.

The user has required authorization.

Environment

SAP S/4HANA Cloud Public Edition

Resolution

Use SSCUI 103887 (Message Control in Tax Area) only with permitted application areas defined by the system for tax configuration.
Enter an allowed application area related to tax (for example, "FOT_COMMON" or "FOT_TDT"). Unsupported application areas will raise error.
