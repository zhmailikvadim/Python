Symptom

When attempting to reverse the journal entry in the app Manage Journal Entries, the following error message is displayed: "BOE_MESSAGE 201 Posting status of Bill of Exchange is not ready for posting.".

Environment

SAP S/4HANA Cloud Public Edition.

Reproducing the Issue

  1. Open the app Manage Journal Entries.
  2. Choose one journal entry.
  3. Click "Reverse" and the following error message is displayed: "BOE_MESSAGE 201 Posting status of Bill of Exchange is not ready for posting.".

Cause

The transaction status is Claim, Created.

Resolution

Open this BoE transaction and then click the "Back To Register Ready".

Keywords

reverse, BoE transaction, BOE_MESSAGE 201, Posting status of Bill of Exchange is not ready for posting, Back To Register Ready
