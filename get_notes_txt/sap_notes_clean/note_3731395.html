Symptom

Transaction code SADT_PROTECTED_MODE is missing in the system.

Reproducing the Issue

SAP Netweaver Release 740 support package level lower than 08

Resolution

Transaction code SADT_PROTECTED_MODE is available starting from release 740 SP08.

Be note that SADT_PROTECTED_MODE is primarily used by SAP support in rare support scenarios.
