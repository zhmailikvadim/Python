Symptom

In production cost accounting, an incorrect further allocation of the "Costs" value field may occur if quantities are written to a value field with a lower number.

In the following example, the VALUE4 and VALUE5 costs are not allocated correctly.

Example:

VALUE1 Proportional costs Cost element group 1

VALUE2 Fixed costs cost element group 1

VALUE3 Quantities Cost element group 2

VALUE4 Proportional costs cost element group 3

VALUE5 Fixed costs cost element group 3

Other Terms

Production Cost Accounting

Reason and Prerequisites

Program error

Solution
