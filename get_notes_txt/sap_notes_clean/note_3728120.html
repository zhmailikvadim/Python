Symptom

You are receiving the following error when terminating a master contract containing a beneficiary: 'No value transfer permitted in field Business Partner No.'.

The reason for this is the missing clearance of the exporting parameters at the beginning of the function module BCA_OBJ_CN_BENEF_GET_DB. This means that the master contract beneficiary data from an earlier call of the same function module is still being used for product checks at the account level in the subsequent processing. Since this does not match the product configuration for the given account, the error message is triggered.

Other Terms

Function Module BCA_OBJ_CN_BENEF_GET_DB, Contract Element BCBF

Reason and Prerequisites

Program Error

Solution

Import the relevant Support Package that is specified for your release. As a manual advance correction, you can implement the attached correction instructions using the transaction SNOTE. There are no manual activities.
