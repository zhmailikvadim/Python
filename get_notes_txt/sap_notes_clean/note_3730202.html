Symptom

Budget data for project is uploaded via the app Import Financial Plan Data. Uploaded data is visible in app Display Financial Plan Data but is not reflected in app Project Budget Report

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Import budget using the "Import Financial Plan Data" app with only the Project Definition field filled and the WBS Element field left blank.
  2. Open "Display Financial Plan Data" and verify the imported plan/budget is visible.
  3. Open "Project Budget Report" and observe that no budget is displayed.

Cause

The import file does not maintain the WBS Element (PS_POSID), resulting in financial planning entries without a WBS element in the system. The Project Budget Report CDS logic selects only entries assigned to a WBS element, so entries without field PS_POSID are excluded.

Resolution

  1. Update the budget import file to include the WBS Element (PS_POSID) for every line. If the intent is to budget the overall project, use the top-level WBS that is automatically created with the same ID as the project (enter the same ID in both PS_PSPID and PS_POSID).
  2. Re-import the corrected file with the "Import Financial Plan Data" app.
  3. Verify that the budget now appears in "Project Budget Report".
