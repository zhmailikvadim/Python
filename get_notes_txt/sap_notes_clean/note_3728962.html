Symptom

The view icon (eye icon) in the relational data UI is grayed out for some database.

Environment

SAP Engagement Cloud

Reproducing the Issue

  1. Go to Add-ons > Relational Data > Connections tab.
  2. The eye icon (view connection details) is disabled:

Cause

The Eye icon (to view the DB) it is only available for Emarsys Hosted ones or MySQL connections.

Resolution

This is the expected system behaviour.
We are constantly thriving to improve our system and hearing your suggestions is the best way forward. Our product managers regularly check the Influence page to hear what changes users would like to see in place and check those for feasibility. Idea requests can also be voted for by other customers.
If you would like to suggest a change to this behavior, please submit your idea at the SAP Emarsys Influence page.
Please note that it is not possible for SAP Support Team to provide you additional information regarding the availability of this feature in a future release.

Keywords
