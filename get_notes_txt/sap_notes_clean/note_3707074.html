Symptom

With the release 2602 CE, DMS (Document Management Service) can now be added as external (SL1) via the Attachment facet at the header level of a Purchase Requisition in the Manage Purchase Requisitions - Professional app or via Documents tab in the Create Purchase Requisition - Advanced.

This can enable the transfer of header attachments from Purchase Requisition to Purchase Order.

Environment

SAP S/4HANA Cloud Public Edition 2602

Resolution

To be able to transfer attachments between Purchase Requisitions (PR) to Purchase Orders (PO) the configuration on the HDM needs to be done. For this the customizing activity Activate HDM for Attachment Object Type can be used by the configuration expert. Please check the path shown in the screenshots below.

In the CBC or Manage Your Solution find the SSCUI 105346:

Open the activity Activate HDM for Attachment Object Type and activate the PURCHASEREQUISITION, PURCHASEREQUISITIONITEM and PURCHASEORDER:

Select PURCHASEREQUISITION, PURCHASEREQUISITIONITEM or PURCHASEORDER and navigate to Activate Document Frameworks, then select Document Management System and navigate to Maintain Default and Restricted Document Types. In this view is possible to maintain which will be the default Document Type for attachment:

SL1 is for External Use
SL9 is for Internal Use

After the configuration a new tab will be displayed in the Purchase Requisition and Purchase Order Header in the Advanced app, the Documents tab: 

All the attachments added in this tab will be transferred from the Purchase Requisition to the Purchase Order. Meaning that if an attachment is desired to be transfer it need to be added in the Documents tab on header and not in the Service for Object - Attachment List.
In the Item line there is also a button called Documents:

Also in the Manage Purchase Requisitions - Professional app the SL1 document type will be displayed:

Attachments added through here will be sent to the Purchase Order after the conversion between PR and PO.
