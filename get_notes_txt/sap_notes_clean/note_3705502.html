Symptom

S4IDM patch 4.0.55 brings following adjustments and fixes:

Bug fixes for:

  • Vehicle area not being updated when delivery zone is changed in the "Start Variant"
  • Gantt gets escaping on loading/unloading dragging.
  • Entering large numbers fails for Quantity popups.

New and/or enhanced features:

  • Configure custom user actions and call them on the S4IDM main toolbar.
  • Modify the quantity already planned to a compartment with a new context menu action.

Other Terms

 IDM, Integrated Dispatch Management

Reason and Prerequisites

This SAP Note ships S4IDM UI version 4.0.55.

This SAP Note also requires backend SAP Note 3713008 as a prerequisite only for

  • Vehicle area not being updated when delivery zone is changed in the "Start Variant"

This SAP Note also requires backend SAP Note 3706780 as a prerequisite only for

  • Configure custom user actions and call them on the S4IDM main toolbar.

This SAP Note also requires backend SAP Note 3688567 as a prerequisite only for

  • Modify the quantity already planned to a compartment with a new context menu action.

Solution

Implement this SAP Note.

Frontend SAP Notes build on each other and must be installed in sequence.
