Symptom

  • Excel file is corrupt and can not be opened.
  • When exporting data from the scorecard, the excel file that is generated cannot be opened in Microsoft 360.
  • Error:  "We found a problem with some content in export data file. Do you want to recover as much as we can?"
  • When we click on yes, we get the message as the file is corrupted and cannot be opened.

Reason and Prerequisites

Environment:

SAP Information Steward 4.3

Steps to Reproduce:

  1. In the Scorecard Data Quality Detail view, select the Export failed data from select rules in a scorecard arrow in the top right hand icon list.
  2. Select all rule bindings listed in the next window.
  3. Select Export.
  4. Select to Export to Excel.
  5. Attempt to open the excel file after export.

Solution

Workaround:

Export the data to a zip file instead of to excel.

This has been forwarded to development for further investigation.
