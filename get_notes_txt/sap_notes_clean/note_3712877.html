Symptom

GJCBA: Shortdump when running cutback in a system where time dependent tax calculation is active

Other Terms

JVA, Joint Venture Accounting, FOT_TDT-020, FOT_TDT020, FOT_TDT 020, Tax Calculation Date Determination Type 'UNDEFINED' is not allowed, GJ-845, GJ845, GJ 845, Warning - no partners found for equity group, T8JQ

Reason and Prerequisites

This note fixes 2 issues:

(1) When cutback is run in a system where time dependent tax calculation is active, a shortdump can occur to due missing "registration" of the cutback program for this functionality.

(2) When cutback is run in parallelization mode, warning message GJ-845 is displayed whereas this does not happen when the cutback is run without parallelization.

Solution

Implement the attached code corrections with transaction SNOTE.
