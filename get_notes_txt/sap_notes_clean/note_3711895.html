Symptom

You create an request for quotation (RFQ) with an awarded quotation. Then you create a Purchase Order (PO) out of the quotation. You don't change the proposed supplier address. Nevertheless a new address number is generated for the PO, if the supplier has an E-Mail address.

Other Terms

S/4 HANA, I_PurchaseorderTP,  SAP Fiori App 'Manage Supplier Quotations', SMTP_ADDR, email, ADRNR, CL_I_D_PURCHASEORDERTP_SSUPPA, CL_I_A_ACTVATE_PURCHASEORDERTP, CHECK_HDR_ADDRESS_WAS_CHANGED, emailaddress, Supplier Quotations, F1991, ADDR_NUMBER_GET, EKKO-ADRNR, i_addressemailaddress

Reason and Prerequisites

Program error.

Solution

Implement the attached correction instructions.
