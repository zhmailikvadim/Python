Symptom

During the configuration of IDCNCASH, the system allows the selection of either debit or credit for a single node, but not both. The requirement is to include both debit and credit transactions in one node.

Environment

China

Cause

Incorrect configuration

Resolution

Note that for a single child node, the system is designed to allow either debit or credit transactions, but not both. To address this requirement, create a parent node first and then add two child nodes under the parent node: one for debit transactions and another for credit transactions.

Keywords

IDCNCASH, cashflow report, debit transactions, credit transactions, single node
