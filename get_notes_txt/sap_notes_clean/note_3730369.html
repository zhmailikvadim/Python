Symptom

In SAP Note 3676825 new legislation regarding excluded absences was delivered. Further modifications are needed in report HSKCELD0 to properly report maternity leave (and other excluded absences) since 1.1.2026.

Other Terms

HRSK, report HSKCELD0, maternity leave, excluded absences.

Reason and Prerequisites

This SAP Note is relevant only for Slovakia (HRSK).

Solution

This SAP Note delivers modification of report HSKCELD0 to properly report excluded absences since 1.1.2026.

 

Implementation

  • As a general rule, SAP recommends that you install a solution by applying a Support Package. However, if you need to install a solution earlier, use the Note Assistant to implement the correction instruction. You can find more information about the Note Assistant in SAP Service Marketplace, under service.sap.com/note-assistant.
