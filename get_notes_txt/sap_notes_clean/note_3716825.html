Symptom

  • Users were unable to select declarations for a specific month in transaction /ECRS/RP_EDIT (/ECRG/RP_EDIT). 
  • Users could not specify the maximum number of items that the system selects by default. 
  • Users were unable to hide totals of key figures such as Invoice Value, Net Weight, etc., on the Intrastat declaration overview screen.

Environment

  • SAP S/4HANA 
  • SAP S/4HANA Cloud Private Edition
  • SAP S/4HANA Cloud Public Edition
  • GTS E4H

Reproducing the Issue

  1. Navigate to transaction /ECRS/RP_EDIT. 
  2. Observe that filter settings are not available. 
  3. Note that personal settings cannot be configured.

Cause

Before implementing note 3713402:

  • Filtering for specific months and years was not available in transaction /ECRS/RP_EDIT or app 'Manage Intrastat declaration'. 
  • Users were unable to specify the maximum number of items selected by default in the system. 
  • The system did not allow users to hide totals of key figures such as Invoice Value, Net Weight, etc., on the Intrastat declaration overview screen.

Resolution

***Image / Data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.***


After implementing note 3713402 the following filter settings and personal settings are provided in transaction /ECRS/RP_EDIT or app 'Manage Intrastat declaration': 


  • Filter for current month and previous month. 




  • Filter a specific month by manual input for declaration month and year. 



  • To get the complete overview, radio button 'Manual input' without specify declaration year and month must be selected. 



By clicking button 'Personal setting, it is possible to change user specific settings for a certain POI (provider of information): 

  • Maximum number of items
    • Lower values for the maximum number of items reduce the system response time.
    • By default, the maximum number of items is set to one thousand.
    • Unless the maximum number of items is specified, all items are selected.
      The selection of items is sorted by correct indicator. This means that incorrect items are selected first, followed by correct items.



  • Hide totals on Intrastat declaration overview screen
    • Specifies if system should hide the totals of key figures like Invoice value, Net Weight etc. on Intrastat declaration overview screen.
    • By default, all totals are calculated and displayed on overview screen.
    • By hiding the totals, the response time of system is reduced.
