Symptom

You are using the app "Display Correspondences" (F3693) and try to print a correspondence.

During the printing process, there is an issue caused by the output device.

Other Terms

Display Correspondences, F3693, CONVERSION_EXIT_SPDEV_INPUT, TDDEST, TSP03L, TSP03

Reason and Prerequisites

The app takes over the long name of the output device instead of the short name (4 characters) as required by the process.

Error in code base.

Solution

Please implement this SAP Note or import the latest support package.
