Symptom

  • The requirement is to restrict standard values and display only intended values in the F4 help for the Consolidation Version field in the Manage Assignment app (App ID: F3870).
  • The authorization object RVERS/E_CS_RVERS restricts values in the SAP GUI but does not restrict the F4 value help in Fiori.

Environment

  • Group Reporting
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Open the Manage Assignment app (F3870) in Intercompany Matching and Reconciliation.
  2. Navigate to a screen where the Consolidation Version field is available and open the F4/value help.
  3. Observe that standard Consolidation Version values are listed, and the authorization object RVERS/E_CS_RVERS does not limit the list in Fiori, although it works in the SAP GUI.

Cause

  • The F4 value help for Consolidation Version is provided by the standard search help FC_RVERS and, in Fiori, does not evaluate RVERS/E_CS_RVERS to restrict the value list.

Resolution

Search help FC_RVERS is used for Consolidation Version filters in ICMR product, enhance the exit of this search help to display the intended values.
