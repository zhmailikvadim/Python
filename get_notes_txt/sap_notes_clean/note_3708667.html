Symptom

During file upload of company code assignment for profit center, errors below may occur:
"&1 &2 Company code &3 must also be assigned to the controlling area"
"Company code & does not exist"

You find that there are inconsistencies for the issued company codes assignment in MDG staging area.

Environment

Master Data Governance on S/4HANA

Reproducing the Issue

Use file upload functionality to upload for Profit Center company code assignment

Cause

There are inconsistent company code assignments to profit centers.

Resolution

To correct the inconsistency, manual steps are necessary. You can either use single object maintenance to correct the inconsistencies one by one for each profit center, or you can use File Upload to correct multiple profit centers at once.

If change/update is needed, please refer to SAP Note 2988310 - How to fix the inconsistency of Profit Center to Company Code Assignments for the correction.

If the company codes are not used at more, please refer to SAP Note 3702162 - Obsolete company code postprocessing for cleaning up of company code assignments.

See Also
