Symptom

The report RFM_PP_BOM_CONSISTENCY_CHECK detects issues with Fashion BoM links (in table FSH_BOM_VD). But the report fails to fix the issues. The report also shows the error message RFM_PP_BOM_CONST_CHK020 for the dependent BoM Components.

Other Terms

Deviation Quantities, FSH_BOM_VD, Fashion BoM, Fashion BoM link, RFM_PP_BOM_CONSISTENCY_CHECK, RFM_PP_BOM_CONST_CHK013, RFM_PP_BOM_CONST_CHK019, RFM_PP_BOM_CONST_CHK020

Reason and Prerequisites

Reason 

This problem is caused by a program error.

Solution

Implement the correction instruction provided with this SAP note.
