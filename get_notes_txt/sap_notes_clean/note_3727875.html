Symptom

As described in the regulatory change manager under:

Complementary Law No. 224  published on December 26, 2025 (LC 224/2025).

This SAP Note provides relevant system changes related to the IPI tax rate (<pIPI>) field which contains four decimal places in the RATE4DEC field.

Other Terms

Brazil, Outbound, Nota Fiscal Eletronica, Electronic Nota Fiscal, Brazil eInvoice, Cloud, NF-e, LC 224/2025, SAP Document and Reporting Compliance, cloud edition, SAP Document and Reporting Compliance, outbound invoicing option for Brazil

Reason and Prerequisites

You have SAP Note 3727874 applied in your system.

This SAP Note is relevant only for Brazil.

Solution

After you implement this SAP Note, your system maps the <pIPI> tag using the value from the RATE4DEC field of the J_1BNFSTX table. If the RATE4DEC field is not provided in the electronic document, the system will use the value from the RATE field of the J_1BNFSTX table.
