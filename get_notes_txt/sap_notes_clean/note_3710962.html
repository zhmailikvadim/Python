Symptom

When the system language is different than English, certain tiles fail to load correctly.

Environment

SAP Cloud Identity Access Governance 2.0

Reproducing the Issue

  1. Logon to IAG;
  2. Check any Tile.

Cause

Missing configuration to enable the language used during access.

Resolution

Enable the desired Language under Configuration > Language.

Keywords

Language, IAG, English, Tile, Not Working, Configuration
