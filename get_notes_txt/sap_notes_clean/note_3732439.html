Symptom

You are using the Fiori application Manage Promotional offers and you work with offers containing attributes of type "Text". You see that there is only a simple input field not displaying or allowing you to add line breaks.

Other Terms

Line Breaks, Attribute Value, Attribute Table, NWBC

Reason and Prerequisites

The used control of type sap.m.Input was not sufficient for the requirements of this field. Therefore it will be replaced by a control of type sap.m.TextArea.

Solution

Implement this note or the respective support package.
