Symptom

  • Payment requests released from the free form payments application do not appear in transaction F111 for automatic processing.

Environment

  • Financial Accounting (FI)
  • SAP R/3​
  • ​SAP ERP Central Component​
  • SAP ERP​
  • SAP enhancement package for SAP ERP​
  • SAP enhancement package for SAP ERP, version for SAP HANA​
  • SAP S/4HANA
  • SAP S/4HANA Finance

Reproducing the Issue

  1. Create a payment request in the free form payments application using transaction FIBLFFP.
  2. Verify if the payment request appears in F111.

Cause

The issue arises due to improper configuration in transaction FIBLFFP.

Resolution

1. Navigate to Environment -> Settings -> Processing Steps to customize the steps for automatic initiation when creating a payment request via FIBLFFP.  
2. Configure the following steps: 

  • Online Payment  
  • Dual Control 
  • Release
  • Start F111
  • Start PMW  

3. If "Start PMW" is selected, ensure the PMW format maintained in FBZP is created with the relevant variant from transaction OBPM4.  
4. If customizing is set to trigger F111 automatically, selecting 'Start F111' or "Start PMW" will initiate F111 for each payment request.  
5. To run one F111 for multiple payment requests, first perform FIBLFFP, then initiate F111.  
6. Use program RFPRQZLP to schedule the payment program F111 automatically in a periodic job.
