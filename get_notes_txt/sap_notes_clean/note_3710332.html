Symptom

  • An attempt to load the product catalog resulted in an "Import catalog timed out" error.

Environment

SAP Customer Data Platform

Reproducing the Issue

  1. Attempt to load the product catalog for the specified URL.
  2. Observe timeout error on UI

Cause

  • The mapping between catalog schema and product schema yielded an empty set of mapped values, except for the specific field.
  • This field in the input file is not unique.
  • The input file CSV headers did not match the expected schema.

Resolution

  1. Ensure the mapping between catalog schema and product schema yields non-empty sets of mapped values, in addition to specific field.
  2. Verify that the specific field in the input file is unique.
  3. Confirm that the input file CSV headers match the expected schema.
  4. Refer to the official documentation for uploading product catalogs:

Uploading Product Catalog - SAP Customer Data Platform.

See Also

Refer to: Uploading Product Catalog - SAP Customer Data Platform.
