Symptom

  • SAP IBP failed to generate adequate supply for Product XXX at Target Location TL1, despite correct master data and sufficient capacity.
  • Target Location TL1 experienced a shortfall, while Source Location SL1, with stock available and a maximum stock value of zero, did not supply to Target Location TL1 as intended.

Environment

  • Order-Based Planning Finite Heuristic
  • SAP Integrated Business Planning for Supply Chain (SCM-IBP)

Reproducing the Issue

  1. View data at Product XXX, Target Location TL1 and Source Location SL1, for the following Key Figures
    1. Total Demand, 
    2. Projected stock,
    3. Total Receipt,
    4. Maximum Receipt
    5. Production Receipts for Interactive Planning

Cause

  • If Production Receipts for Interactive Planning = 0 

Resolution

  • If Production Receipts for Interactive Planning = 0, yes, this could indeed be a root cause for why no supply is being generated.

Keywords

SAP IBP, supply generation issue, location shortfall, master data, free capacity, Order-based planning, Finite heuristic, Projected stock, Hana trace logs, Supply planning runs, Production receipts, Production Receipts for Interactive Planning, Maximum Receipt
