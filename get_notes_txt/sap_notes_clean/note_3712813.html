Symptom

While running payroll calculation in year 2026 with retrocalculation back to year 2025 various warning/error messages may occur in payroll function SKCTX. For example message, that some fields in P0252 are not identical for all connected PERNRs or message that total non-taxable income of PERSON is not equal to sum of amounts from all PERNRs. Payroll then ends without any reasonable error. This SAP Note brings correction of mentioned issue.

Other Terms

HRSK, HSKCALCM, HSKCALC0, SKCTX, P0252-STRSO, Phase CALC

Reason and Prerequisites

This SAP Note is relevant only for Slovakia (HRSK)

Solution

Source code of payroll function SKCTX was updated to solve the issue with improper messages.
