Symptom

The graph components do not render correctly in SAPUI5 version 1.24.x

Other Terms

SAP Simple Finance add-on, SAP Fiori, SAP HANA, SAP Business Suite, Analytics, UIHSFIN1, FIN_CO_MARGINDASHBOARD, FIN_NETMARGIN

Reason and Prerequisites

Check release information note 2009516

Solution

The following issues were solved and the corrections will be available in SP02:

  • Corrections in graph rendering for compatibility with SAPUI5 version 1.24.x
  • Revenue amout format changed to display full figures

Enhancements

  • New Settings screen: user parameters are now configured in the app; backend configuration for user-specific settings is not needed anymore.
