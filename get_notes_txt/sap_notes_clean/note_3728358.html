Symptom

Navigating to Manage Pending Hire Page while changing "Select Pending Hires" to Onboarding, page fails to load

Environment

SAP SuccessFactors Employee Central 

Reproducing the Issue

  • Navigate to Manage Pending Hires
  • Select Pending Hires to Onboarding
  • Notice page fails to load/ loading indefinitely 

Cause

Stale or corrupted role based permissions

Resolution

Via provisioning, run the  Refresh RBP Permission Configuration and Run the RBP Self Healing Tool jobs in order to resolve the issue

See Also

Refreshing RBP after Changes in Provisioning Configuration | SAP Help Portal
