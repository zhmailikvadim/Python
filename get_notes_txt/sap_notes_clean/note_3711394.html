Symptom

  • Goal Start Date is displayed incorrectly when accessed from the performance form.
  • The configured start and due dates on the goals are 1/1/26 - 12/31/26.
  • When viewed from the performance form, the start date appears as 12/31/25 and the end date as 12/30/26.

Environment

SAP SuccessFactors Performance Management

Reproducing the Issue

  1. Open the performance form.
  2. Access Goal Section > select Goal > Goal Details
  3. Observe that the start date and end date is one day late compared to the configured date
    • Configured: 1/1/2026 and 12/31/2026
    • Showing: 12/31/2025 and 12/30/2026

Resolution

As per Engineering Team, the planned fix for this issue is set to be deployed is on the b2605 (1H 2026) release. Release date would be:

  • Preview: April 13, 2026
  • Production: May 15 - 16, 2026
