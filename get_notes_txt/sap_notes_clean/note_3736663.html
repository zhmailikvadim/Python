Symptom

Databricks data product installation shows no target space in SAP Datasphere

Environment

SAP Datasphere

Reproducing the Issue

  1. Open the data product shared from the external producer in the Catalog.
  2. Start the installation for the shared API/data product.
  3. In the “select Target space” step, the target space list is empty.

Cause

The external data product producer system was not configured under Administration > Business Data Product, and no target spaces were added for consumption; therefore, no eligible target spaces were available during installation.

Resolution

Follow SAP Help Portal Documentation: Authorize Spaces to Install SAP Business Data Cloud Data Products.

See Also

SAP Help Portal Documentation: Installing Marketplace Data Products
