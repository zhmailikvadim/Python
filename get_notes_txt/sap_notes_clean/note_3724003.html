Symptom

While the note 3564247  is applied in the Snote , there is an error pops out as below:

Package S_ETD_UDO does not exist.

Environment

Snote tool

Reproducing the Issue

1. Start Snote Tcode

2. Implement the note  3564247

Cause

Missing prerequisites for the note 3564247.

Resolution

The package S_ETD_UDO is introduced by the note 3199893 . In order to implement note 3564247, it is necessary to first implement note 3199893. And ensure that all manual pre-implementation steps outlined in the note are followed before proceeding.
