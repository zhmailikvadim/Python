This SAP Note corrects the following error in the incompleteness check:

  • You want to adjust an item that is specified as a supplementary item in the unloading permission in such a way that it is no longer a supplementary item and correctly specify the gross weight for this. Afterwards, the system issues the error message of the incompleteness check "Item &1: Remove gross weight" (/SAPSLL/CUS_NCTS_EU 569).
