Symptom

After configuring ALM alerts as described in KBA 3345862 - "How to setup ALM alerts", the alerts are not functioning for successful cases.

Environment

Release Independent 

Cause

ALM alerts is not supported for successful cases. 

Resolution

As per described in ALM documentation here, ALM alerts is only for NON successful cases.

Alert Management never create an alert. ONLY if the application calls the alert management.

So if in success case the application doesn’t call the alert management than no alert will be send.
