Symptom

  • Category               ABAP programming error
    Runtime Errors         EXPORT_TABLE_UPDATE_CONFLICT
    ABAP Program           CL_DMC_ERROR_LOG==============CP
    Application Component  CA-LT-MIG
    Date and Time          30.01.2026 04:09:16

     

     Short Text
         Problems with parallel EXPORT to the same table entry.

     

     What happened?
         Error in the ABAP application program.

         The current ABAP program "CL_DMC_ERROR_LOG==============CP" had to be
          terminated because it found a
         statement that could not be executed.

  • Target is an ABAP system

Environment

SAP Landscape Transformation Replication Server (SLT)

Cause

  • DMC_C_WL_TABL_OP table in the source or target system has other tables maintained, which makes tables not maintained in DMC_C_WL_TABL_OP can't be replicated successfully.

Resolution

  1. Stop the replication of in the LTRC transaction.
  2. If it's OK, delete all the tables that's maintained DMC_C_WL_TABL_OP in the source or target system to allow all the tables to be replicated. If it's not OK to do so, maintain the failed source table in the DMC_C_WL_TABL_OP table in the source or target system.
  3. Start the replication again
