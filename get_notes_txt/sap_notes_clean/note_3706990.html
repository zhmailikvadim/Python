Symptom

When launching the Review Booklet application, the Generative AI "Analytical Business Insight" side panel fails to open. The browser console (F12) shows the authorization error "No Start UI_ANALYTIC_BUSINESSINSIGHT," indicating the user lacks the required Start authorization for the UI_ANALYTIC_BUSINESSINSIGHT OData service and preventing the Analytical Business Insight UI from loading.

Other Terms

 Analytical Business Insight, UI_ANALYTIC_BUSINESSINSIGHT, AI400

Reason and Prerequisites

In S/4HANA 2025 FPS00, FPS01 release, UI_ANALYTIC_BUSINESSINSIGHT OData service is missing in SAP pre-delivered standard businss role SAP_BR_EMPLOYEE.

Solution

It is recommended to implement the fix with the next service package FPS02 containing this correction, alternatively, the below attached manual correction instruction can be implemented.

One customizing PFCG role in your system such as ZBR_EMPLOYEE,  has to be modified in below way to include transaction F8405.

1: launch transaction /UI2/FLPCM_CONF, in the Catalogs tab, click create to create Fiori business catalog, input your customized ID, such as "ZBC_RVB_ABI_Test" and Title "Analytics - Review Booklet Business Insights", and save it.

2: select the created catalog ZBC_RVB_ABI_Test, click Add Tiles/Target Mappings, choose "Add Tiles/TMs to selected Catalog".

3: input F8405 , click go, then select the found record, click Add TM Reference.

4: launch transaction PFCG, input ZBR_EMPLOYEE and change it, select your transport request.

5: under the Menu tab, click add transaction, select Launchpad Catalog under SAP Fiori Launchpad, input Catalog ID ZBC_RVB_ABI_Test and continue.

6: go to Authorizations tab, click Change Authorization Data button to ensure F8405 transaction is under object class AAAB for authorization object S_TCODE, click save button.

7: assign the customizing PFCG role to the users for testing.
