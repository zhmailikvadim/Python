Symptom

  • PDF generated in EDOC_COCKPIT is displayed in Polish language.
  • Requirement to display PDF in English

Environment

  • SAP ERP
  • SAP S/4HANA
  • SAP Document and Reporting Compliance, Cloud edition

Reproducing the Issue

  1. Access the edoc_cockpit in the system.
  2. Generate a PDF for an invoice.
  3. Observe that the PDF content is displayed in Polish language.

Cause

The PDF visualization for Poland KSeF is currently delivered in Polish language as part of standard functionality.

Resolution

This is standard system behavior.

Currently, SAP standard delivers the PDF visualization in Polish language only.
There is no configuration available to change the PDF language to English.

Additional Information

  • English-language PDF visualization (including QR code) is not yet officially delivered.

  • Any future enhancements will be communicated via SAP Notes, WorkZone, or Regulatory Change Manager (RCM).
