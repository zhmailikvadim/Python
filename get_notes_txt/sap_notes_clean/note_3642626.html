Symptom

In case negative APERAK/CONTRL is received for settlement message, the billable item for MeMi document still leads to the creation of a billing document.

Other Terms

Overtake and Undertake Quantities Billing, MeMi Document, BM_MM_STL_QTY_REQ_UPD_APERAK, DE_C_CP_RECV_DELAY_APERAK, APERAKCheckError, DelayAPERAKCheckError, CONTRLCheckError, Settlement Message Reference, Settlement Quantity, Status of the Latest MSCONS, Billable Item, Billing, 8112, 8114, /US4G/CL_PROC_CI_FICA_ENH_MEMI, /US4G/IF_PROCESS_CI_FICA_ENH~PROCESS_CI_FICA_EVENT_8112, /US4G/IF_PROCESS_CI_FICA_ENH~PROCESS_CI_FICA_EVENT_8114, DSO

Reason and Prerequisites

The reason was an incomplete status check.

Solution

Please implement the corrections of this SAP Note or import relevant Support Package.

This SAP Note is relevant for the following roles: DSO

After implementing the correction in this SAP Note, a billable item with an incorrect final status of settlement message reference cannot lead to the creation of a billing any longer.
