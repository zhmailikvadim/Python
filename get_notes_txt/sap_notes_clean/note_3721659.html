Symptom

There is a requirement to check what user, what time and what value was changed in the HTTP Provider Application Alias.

Environment

SAP NetWeaver Application Server for Java

Resolution

  • Logon to the AS Java NWA (http://<host>:<port>)
  • Go to Log Configuration
  • In Tracing Locations find com.sap.engine.services.httpserver.server.hosts.impl.HostPropertiesRuntimeInterfaceImpl
  • Set it to severity Info
  • Save Configuration

From now on any change in the Virtual Hosts will be logged into the defaultTrace (/usr/sap/SID/InstID/j2ee/cluster/serverX/log/defaultTrace*.trc). Note that past occurrences in which the tracing was still not enabled won't be found in the traces.
