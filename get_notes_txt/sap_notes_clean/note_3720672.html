Symptom

The old Web Monitor App does not show texts in the popup where one can choose the sort fields.

Reason and Prerequisites

program error

Solution

To solve the issue please apply the Support Package as mentioned below. In urgent cases you can use the correction instructions.

Delivery Information

  • SAP BW 7.40
    Implement Support Package 36 for SAP BW 7.40 (SAPKW74036) into your BW or AS ABAP system. The Support Package will be available as soon as SAP Note 3716127 with the short text "SAPBWNews NW BW 7.40 ABAP SP36", which describes this Support Package in more detail, is released for customers.
  • SAP BW 7.50
    Implement Support Package 35 for SAP BW 7.50 (SAPK-75035INSAPBW) into your BW or AS ABAP system. The Support Package will be available as soon as SAP Note 3649452 with the short text "SAPBWNews NW BW 7.50 ABAP SP35", which describes this Support Package in more detail, is released for customers.

In urgent cases you can use the correction instructions.

Before you use the correction instructions, make sure that you check SAP Note 1668882 and SAP Note 2248091 for transaction SNOTE.

This SAP Note might already be available before the Support Package is released. In this case, however, the short text still contains the term "preliminary version".
