Symptom

You are using the Focused Build standalone extension for Cutover Checks and Post-Cutover Activities.

When executing Synchronize Deployment, you encounter the following error messages in the UI and the application log (SLG1):

  • "Error in syntax for transport target, use the F4 help."
  • "Unable to create synchronization ToC from [xxxxx] for [yyyyy]." (Note: [xxxxx] and [yyyyy] are system-specific valuess.)
  • "No transport found"

Other Terms

CutOver, Synchronize Deployment,  TOC, target , SID, TR061, /SALM/CHARM_PCA003

Reason and Prerequisites

The Extended SID is used for the target system. However, due to a programming error, this long SID was not converted correctly.

Solution

Please implement the attached correction instructions.
