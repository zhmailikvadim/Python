Symptom

Attribute/element BDCFSI is not available in the SOA service ChartOfAccountsReplicationRequest_V1.

Environment

  • SAP S/4HANA
  • SAP Master Data Governance

Cause

BDCFSI (breakdown category) is not supported by the SOA service ChartOfAccountsReplicationRequest_V1 for Data Model 0G.

Resolution

  1. Please refer to: 0002021246 - MDG-F: (Un-) Supported Fields in Data Model 0G and verify that BDCFSI is not supported in ChartOfAccountsReplicationRequest_V1.
  2. For consolidation use cases, use Consolidation Governance to replicate FSI data to SEM-BCS; ChartOfAccountsReplicationRequest_V1 is not designed for BDCFSI.

See Also

0002021246 - MDG-F: (Un-) Supported Fields in Data Model 0G
