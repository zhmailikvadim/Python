Symptom

This SAP Note provides you with updates for the EFD-Reinf report.  Currently, when you run EFD-Reinf reports for the R-4000 series, the system limits output to 9,999 lines. The system does not support reports that exceed this limit.

Other Terms

Brazil, Statutory Reporting, EFD-Reinf, R-4000

Reason and Prerequisites

This SAP Note is relevant only for Brazil.

You have implemented SAP Note 3707803.

Solution

After implementing this SAP Note, the system supports output generation of up to 99,999,999 lines for EFD-Reinf reports in the R-4000 series. The update applies to the following views:

  • R4010 - Main Fields (REINF_R4010_MAIN)
  • R4020 - Main Fields (REINF_R4020_MAIN)
  • R4040 - Main Fields (REINF_R4040_MAIN)
  • R4080 - Main Fields (REINF_R4080_MAIN)
  • R4099 - Main Fields (REINF_R4099_MAIN)
