Symptom

This Knowledge Article provides additional information and answers to questions which are raising while implementing Security Note 3697099.

Environment

All Business Suite Applications including S4/HANA and CRM comprising the Software Components listed in the Security Note 3697099.

Resolution

  • Which customers are affected? -> All customer running SAP products comprising software components WEBCUIF 7.00 and higher or S4FND 102 and higher or SAP_ABA 7.00 (exactly 7.00 no higher versions affected) and having activated CRM_IC_ISE node of the ICF tree (transaction SICF).  
  • Is there any workaround in lower SP levels, where the fix is not implementable? -> Workaround as described in security note 3697099 can be applied too.
  • Will the upgrade require down time? ->No downtime required.
  • Are any other components, other than the component (mentioned in the security note), impacted by applying the fix? -> No.
  • Is applying the patch enough or do we need to perform any manual configuration steps after deploying it? -> Patch is good enough, deactivating the node is a safety measure recommended for any unused ICF node.
  • Is there any workaround to fix this vulnerability temporarily? -> Workaround as described in security note 3697099.
  • I have implemented the workaround. am I still affected? -> No. But, note that this workaround is a temporary fix and is not a permanent solution. SAP strongly recommends you apply the corrections outlined in the security note, which can be done in lieu of the workaround or after the workaround is implemented. 
  • What would the impact be if I applied the workaround in SAP Note 3697099 and stopped the affected service completely? -> It is only impacting the so called Interactive Scripting Editor (Design Time Tool to create Call Scripts for Interaction Center agents)
  • Is applying the patch enough or we ne need to perform the workaround after patching, as well? -> Patch is enough, workaround is a general recommendation to not keep open service not required productively. This service CRM_IC_ISE should only be activated if the Interactive Scripting Editor is getting used.
