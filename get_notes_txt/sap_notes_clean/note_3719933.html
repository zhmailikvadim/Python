Symptom

  • An error occurs when attempting to delete the CT attribute, displaying the message: "You cannot delete Attribute Value &1 from SAP delivered Content(FINCS_MD_FSITEM 152)".
  • The CT attribute value is confirmed not to be used for any FS item's master data.

Environment

  • SAP Fiori for SAP S/4HANA Finance
  • SAP S/4HANA Finance for Group Reporting

Reproducing the Issue

  1. Attempt to delete the CT attribute.
  2. Observe the error message: "You cannot delete Attribute Value &1 from SAP delivered Content".

Cause

The issue occurs because the naming convention of the attribute value starting with "S" is recognized as standard built-in configuration, which cannot be deleted.

Resolution

  1. Change the FS Item attribute value names starting with "S" to another name and save.
  2. Delete the entry after changing the name.
  3. Note that try to avoid defining attributes starting with "S" in future.
