Symptom

In the Overage Correction report, TR code /STTP/CORR_OVERAGE, step Post Shipping Event for SSCC the disposition value from Business Transaction is not captured in shipping event. Due to this value missing in Shipping event, EPCIS reporting is not generated.

Other Terms

US DSCSA, Correction Report, Overage Correction, Virtual Shipments, Post Manual Event, Shipping Event, Disposition.

Reason and Prerequisites

Programming error. The SAP Note is only available for Advanced Track and Trace for Pharmaceuticals.

Solution

Please install the SAP Note.
