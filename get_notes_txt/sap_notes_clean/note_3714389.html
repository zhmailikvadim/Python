Symptom

When a medication number set has been marked as obsolete in the SAP Fiori app Manage Medication Lists, some of the medication numbers in the set which are being used in a different study are given status 'Packed' at goods receipt, which is incorrect. No further status changes are allowed for medication numbers once the medication number set they belong to has been marked as obsolete.

Other Terms

ISLS, Manufacturing, Intelligent Clinical Supply Management, medication number status

Reason and Prerequisites

You use SAP Intelligent Clinical Supply Management or this SAP Note is a prerequisite of another SAP Note.

Solution

Implement the attached corrections.
