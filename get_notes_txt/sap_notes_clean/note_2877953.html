Symptom

You are trying to upload the Custom Price List with extension field using Excel template and the upload is not succesful.

Environment

SAP Business Bydesign

Reproducing the Issue

  1. Go to Product and service portfolio.
  2. Navigate to Price Lists View.
  3. From the Drop down , select your Custom Price List and click on Go.
  4. Click on Download button to download the excel template for your Custom Price List.
  5. Connect to SAP Business ByDesign with the relevant credentials.
  6. Fill all the details in the excel template and click on Save data to 
  7. Message : "0 Record(s) are saved successfully to SAP Business Bydesign" appears even though you have entered all the details correctly. 

Cause

When you have used the extension fields which has drop down values in the Price List , you have to use the Code Value instead of Descriptions for the extension fields while entering the data in excel template because the system does not accept the descriptions as keys.

Resolution

Kindly use the Code Value, while entering the value of extension field in the excel template.
