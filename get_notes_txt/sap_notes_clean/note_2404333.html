Symptom

You use SAP NetWeaver Process Integration (Exchange Infrastructure) and use the component monitor to monitor ABAP components. The monitor overview displays an error in the "Self Test" column.

Other Terms

﻿RSXMB_ARCHIVE_PLAN, ABAP proxy, Integration Server, self-test

Reason and Prerequisites

In the component monitor, you monitor ABAP components of the type "ABAP Proxy" or "Integration Server". An error is displayed in the "Self-Test" column of the component overview. The self-test details contain an error message.

The error message tells you that the program RSXMB_ARCHIVE_PLAN has not been released as a periodic background job and that the recommended period is 1 day.

The cause is the following entry which was delivered by mistake in the list of interfaces to be archived (transaction SXMB_ITFACTION):

Interface: AI_Person
Namespace: http://test.com/user_decision
Type: O

Solution

Call transaction SXMB_ITFACTION and delete the entry from the list of interfaces to be archived.

Alternatively, you can schedule the program RSXMB_ARCHIVE_PLAN using transaction SXMS_ADMI_ARCH.
