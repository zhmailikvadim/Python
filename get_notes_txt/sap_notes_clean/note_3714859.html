Symptom

I am a Supplier User and I need to add a specific Diversity option not currently available in SAP Fieldglass.

Resolution

The Diversity options available in SAP Fieldglass are limited to the following six types.

  • MBE
  • DBE
  • WBE
  • VBE
  • SBE
  • LGBTBE

It is not possible to add more Diversity options.
