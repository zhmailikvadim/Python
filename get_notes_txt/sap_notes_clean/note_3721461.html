Symptom

The selection of multiple sales order items in app "Manage Production Operations" is not possible when entering the values manually. You need to use the value help.

Steps to reproduce:
1) Open app "Manage Production Operations" F2335
2) Via "Adapt Filters" add the fields sales order and sales order item
3) Insert a sales order number and then try to insert more than one value in field "Sales order item"

In the "Manage Production Orders"-App (F2336), the same issue was already fixed with SAP Note 3400350.

Other Terms

Fiori Elements, C_MANAGEOPERATIONS, valueHelpDefinition,

Reason and Prerequisites

Program error

Solution

After implementing the SAP Note, you can insert a value, confirm with enter and insert the next value. You didn't need to use the value help.
