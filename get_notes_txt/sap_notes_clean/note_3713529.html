Symptom

In IW91 transaction, you are trying to set usage decision for an inspection lot, where the corresponding inspection type does not have selected set defined in plant level. You notice that there is no popup open with the selected set and inspection lot plant.

Other Terms

checklist, TQ30, QPAC, UD, TQSS1_AVE, Selected Set, Code Group

Reason and Prerequisites

Programming error. TQ30 table was not checked to read the usage decision selected set for the inspection type.

Solution

Please implement the attached correction instructions or install the corresponding support package.
