Symptom

  • When configuring the market datafeed, specific data sources are not available in the selection list.
  • In the app Assign Currency Notations – Datafeed, the dropdown for the "Data Source" field does not display the required data source option.
  • The required data source option is provided by the Refinitiv data option according to the RIC Excel.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Open the app Assign Currency Notations – Datafeed (App ID: F6413).
  2. Select the Data Provider (e.g., Y001 for Refinitiv).
  3. Click on a currency combination.
  4. Check the Data Sources field options.
  5. Observe that the required data sources such as ST-PNB (Poland) and ST-BDMDOF (Mexico) are missing.

Cause

The data sources are supported by the market data provider (e.g., Refinitiv) but have not been manually defined in the system's master data configuration. Only defined data sources are available for selection in the currency notation assignment.

Resolution

To make the missing data sources available, follow these steps:

  1. Access to SSCUI Define Data Sources for Datafeed (ID: 103070).
  2. Click on New Entries.
  3. Add the missing data source codes as supported by the Refinitiv Instrument Codes.
  4. Save the configuration, then the added data sources will be available in the dropdown list for selection.
