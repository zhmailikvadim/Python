Symptom

This SAP note explains the boundary conditions of support for SAP HANA 2 in SAP CRM 7.0 Enhancement Package 3.

Note: this SAP note is subject to change. Please check this note for changes on a regular basis

Other Terms

CRM 7.0 Enhancement Package 3; CRM 7.13; SAP HANA 2

Reason and Prerequisites

You plan to implement SAP CRM 7.0 EhP 3 with version 2 of SAP HANA; or you plan to upgrade the SAP HANA database in your SAP CRM 7.0 EhP 3 system to SAP HANA 2.

Solution

SAP HANA database 2.0 SPS02 Revision 22 or higher is released for enhancement package 3 for SAP CRM 7.0. This applies to all support package stack levels of SAP CRM 7.0 EhP 3.

Please note that you will need to be using an SAP Kernel version that supports SAP HANA 2 - see the Product Availability Matrix for more details. Information on support for SAP HANA 2 with SAP Kernel 7.49 64-Bit Unicode can be found in SAP Note 2420699.

General Information regarding support for SAP HANA 2

Support for SAP HANA 2 is optional, SAP HANA 1 continues to be supported in SAP CRM 7.0 EhP 3.

If operating SAP CRM 7.0 EhP 3 on SAP HANA 2, the minimal required revision of SAP HANA 2 is documented in the release information note of SAP CRM 7.0 EhP 3 (SAP Note 1737725).

In the following, we only refer to the optional usage of SAP HANA 2 in SAP CRM 7.0 EhP 3. The scope of support for SAP HANA 1 is not affected by any details related to support of SAP HANA 2.

Support for specific features of SAP HANA 2 through the application

In SAP CRM 7.0 EhP 3, there is no specific support for new functionalities introduces with SAP HANA 2. That is, the application's usage of the SAP HANA database remains unchanged, whether your are operating the application on SAP HANA 1 or on SAP HANA 2.

Next to functional enhancements, SAP HANA 2 introduces enhanced capabilities in areas of system administration and system operations whose usage does not require application adoption. Customers, hosting partners or other involved parties may choose to utilize these capabilities based on their own assessment of relevance and impact on system operations.
