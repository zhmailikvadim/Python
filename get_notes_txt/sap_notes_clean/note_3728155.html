Symptom

I am synchronizing from Solution Manager to Signavio using BPM connector. In my sync protocol I can see the following warning message: Sync Skipped for ObjectType: DIAGRAM is empty. Import of the Diagram was skipped with name XXXXX and ID ZZZZZ 

Resolution

This warning means that in the synchronized solution there is an empty collaboration diagram or process diagram. In the warning message you can see the name and the ID of the diagram. An empty diagram won't be transferred to Signavio. There are the following options for further proceeding:

  • Leave it like it is - the warning doesn't influence the synchronization.
  • Check if the content of the diagram needs to be created.
  • Check if the diagram is still needed.
