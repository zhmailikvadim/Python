Symptom

If delta accruals are created during the intercompany settlement of condition contracts, the accruals index (table WCOCOF) is updated for both sides (vendor and customer). This is incorrect. This SAP Note corrects this so that the accruals index is updated only for the leading side. The leading side is the side for which pricing is carried out first. This is set in the settlement process type.

Reason and Prerequisites

This is due to a program error.

Solution

Implement the correction or the corresponding support package.
