Symptom

When trying to access SSCUI/Configuration Activity "Define Service Team Profile and Assign Service Team Types" ID 103247, the following error message is raised: 

  • No authorization to start the UI for transaction S_ER1_83000045

Environment

  • SAP Central Business Configuration (CBC)
  • SAP S/4HANA Cloud Public edition

Reproducing the Issue

  1. Navigate to the CBC workspace and click on the "Configuration Activities" tab.
  2. Search for the Configuration Activity ID: 103247.
  3. Click on the entry for "Define Service Team Profile and Assign Service Team Types".
  4. Click the "Go to Activity" button.
  5. The error message "No authorization to start the UI for transaction S_ER1_83000045" is raised.

Cause

Missing Business Catalog.

Resolution

  1. Add the SAP_CA_BC_IC_LND_S4CRM_SO0_PC business catalog to the BPC_EXPERT business role (or to the similar role in use).
  2. Grant "Write Access" permissions to "Unrestricted" for the BPC_EXPERT business role (or for the similar role in use).

If causes adding the business catalog did not solve the issue, please create a case for the CRM-S4-SRV-SVO-2CL component and inform the affected CB user (KBA 2838282).
