Symptom

  • When viewing a case in detail view, the case flow section remains visible even after being hidden via adaptation mode.
  • The title of the collapsed section remains visible, which does not fully align with the definition of hiding.

Environment

Case Management for SAP Sales/Service Cloud

Reproducing the Issue

  1. Go to Case.
  2. Enable adaptation mode.
  3. Select "Enable Start Adaptation".
  4. Select "Collapse by default" for the Case flow section.

Cause

The section is collapsed and not hidden; the label for the section remains visible to indicate its presence and allow users to expand it.

Resolution

This is expected behavior. No changes are required as the system is functioning as designed.
