Symptom

When changing Earmarked funds generated for Recurring requests and for multiple Fiscal years the corresponding indicator within the Document header is being deleted.

This happens in case the change is done by using function module FMFR_CHANGE_FROM_DATA.

The error has implications in the Fiscal year change. The Earmarked funds is treated as ordinary case and not as a multi-years one.

Other Terms

PSOEF_MLTYR

Reason and Prerequisites

Program error

Solution

Please apply the correction instruction of this note.
