Symptom

When reversed application line item is split into two different vendors and one same vendor then applied to contract having lesser quantity, the remaining unapplied quantity is added to RTA line item created during split instead of creating new RTA line item, resulting in quantity loss.

Other Terms

ACM, Agriculture Contract Management, Partial application, Vendor split

Reason and Prerequisites

Program error

Solution

Implement the Correction Instruction to fix the issue.
