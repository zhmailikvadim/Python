Symptom

You want to check your electronic documents in an easy-to-read format instead of in XML format, for example, to review your invoice before submitting it to your receiver.

Environment

SAP S/4HANA

SAP ERP

Reproducing the Issue

N/A

Cause

N/A

Resolution

Please refer to following steps:

  1. Go to the eDocument Cockpit (EDOC_COCKPIT) transaction or Manage Electronic Documents app. Select the Japan eInvoice process.
  2. Select one or more eDocuments and choose Display->Display HTML.

The system displays the electronic document in HTML.

Note

If the electronic document hasn't been submitted yet, it shows the current version of the document in the system. Once you submit the electronic document, this action displays the final version of the document.
