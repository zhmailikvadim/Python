Symptom

Search for “Not Imported” Open Content Network (OCN) courses without selecting any of the “Content Provider” fields and import one or more. After importing, all “Content Provider” fields are selected.

Environment

SAP SuccessFactors Learning

Reproducing the Issue

  1. Learning Administration > Content > Open Content Network > Search for "Not Imported" courses > Don’t select any “Content Provider” field >
  2. Select one or more courses to import > Import the courses >
  3. The page reloads > After it, all “Content Provider” fields are selected.

Resolution

This unexpected behavior doesn’t impact the functionalities in question. Searching for “Not Imported” OCN courses without any “Content Provider” selected or with all of them selected leads to the same search results. However, our Learning Engineering team is currently planning to address it.

Please, set this KBA as a favorite (star icon on the top right corner) to receive updates on it. Check the KBA 2171560 to understand how to get notified when a KBA is created/updated.
