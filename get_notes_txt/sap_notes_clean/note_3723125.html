Symptom

In the Manage Material Coverage (F0251A) app, users are presented with two actions — Delete and Cancel Process — which appear different but effectively behave like two versions of the same Delete function.

Other Terms

F0251A, Manage Material Coverage, MRP, Purchase Order Proposal, Delete Action, Cancel Action

Reason and Prerequisites

The reason is a program error.

Two separate buttons are defined on the view for deleting the entry, but both are wired to the same logic.

Solution

Apply the attached Correction Instruction.
The solution removes the duplicated delete handling.
