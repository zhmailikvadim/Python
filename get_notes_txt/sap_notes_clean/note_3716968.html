Symptom

It has requested to obtain the latest execution date for database objects (Package, Function, Stored Procedure, View) located in the EXT schema.

Environment

  • SAP Sales Performance Management
  • Incentive Management

Cause

Information has been requested to review the last time a database object has been used inside the Database.

Resolution

The information requested is not retrievable from Database.

Keywords

database objects, execution date, EXT schema, audit log, database performance, triggers, monitoring mechanisms, database analysis, Incentive Management, last time triggered, procedure, package, DB object
