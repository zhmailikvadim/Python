Symptom

Cannot open Goal Details in Completed Performance Forms, when Latest Form Experience is enabled. Error appears: "Unable to load data"

Environment

SAP SuccessFactors Performance Management 2H 2025

Reproducing the Issue

  1. Enable Latest Form Experience
  2. Open a completed Performance form and scroll to Goal section
  3. Click on a goal
  4. In goal details tab error appears: "Unable to load data"

Resolution

The fix for this issue is planned to be deployed with the patch 2H 2025 P19.

Fix dates: Preview: 24-Feb-2026 | Production: 26-Feb-2026

For patch timelines, please review SAP SuccessFactors Patches Knowledge Base

Workaround

  1. Workaround :
    Disable Latest Form Experience
  2. Workaround:
    For the completed form, use the Send Copy function to send the form to the employee themself, and then reopen the form again. The Goal details then will be displayed.
  3. Workaround:
    Alternatively, enable “Everyone can access the revision history of the document” on the Company System and Logo Settings page. Be aware that once this option is enabled, any employee who can open the form info page can view snapshots from every routing step. If this is not desired behavior, the admin should disable the Form Info button in the form template settings.
