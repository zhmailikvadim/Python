Symptom

After asking for support team to provide the SFTP server mentioned at this SAP Help article, it is noticed that only SSH-key authorization is provided, but not user/password.

Environment

SAP Emarsys

Cause

Only SSH-key authorization is available for the SAP Engagement Cloud SFTP server that can be requested by the customers.

Resolution

This is the expected system behaviour.

SAP is constantly thriving to improve the product and hearing customer's suggestions is the best way forward. The product managers regularly check the Influence page to hear what changes users would like to see in place and check those for feasibility. Idea requests can also be voted for by other customers.

To suggest a change to this behaviour, please submit the improvement/feature idea at the SAP Emarsys Influence page.

Please note that it is not possible for SAP Support Team to provide additional information regarding the availability of this feature in a future release.
