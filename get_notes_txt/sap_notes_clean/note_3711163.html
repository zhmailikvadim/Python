Symptom

We want to use the SAP Ariba integration with SAP Signavio Process Intelligence by using the CI package.

Resolution

Please follow the documentation on the SAP Business Accelerator Hub: https://api.sap.com/package/SAPAribaAPIExtractor/overview. In this documentation, you can find the required CI package and an explanation of what is needed in the CI configuration files regarding secrets from SAP Ariba & SAP Signavio Process Intelligence to use the integration.
