Symptom

When deleting an operation in the Manage Maintenance Order application (F5241) and then creating a new operation with the same operation number, an error occurs upon saving. The system incorrectly states "Enter Another Operation Number", even though only one operation with that number is present.

Other Terms

F5241, R_MAINTENANCEORDERTP, 404(cp), Maintenance Order

Reason and Prerequisites

Coding issue.

Solution

Apply this SAP Note.
