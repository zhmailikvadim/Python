Symptom

In the Management Console of Plant Connectivity, you have configured a multiple call destination system and you use the relevant pushbuttons to navigate to other destination systems. In this case, an unhandled exception may appear and the Management Console may have to be closed.

Other Terms

Management Console, Plant Connectivity, unhandled exception

Reason and Prerequisites

Program error

Solution

The error is corrected with Patch 3 for Plant Connectivity 15.4 with Support Package 02 (PCo 15.4.2).

Download the current installation program for Plant Connectivity 15.4 from SAP Service Marketplace and install it in accordance with the instructions in the installation guide. The current version of the installation guide is available on the PCo documentation page on the SAP Help Portal under https://help.sap.com/pco. You can use the installation program for the PCo patch to update a PCo version that is already installed or to set up a new installation if PCo is not yet installed on the target host.

Prior to the installation, also refer to the installation note 2926478 for PCo 15.4.2.
