Symptom

User wants to create a label print of Supplier Invoice Number.

Environment

SAP S/4HANA Cloud Public Edition.

Reproducing the Issue

  1. Create a Supplier Invoice.
  2. Record the Supplier Invoice.
  3. Attempt to automatically print the Supplier Invoice Number.

Cause

The current release doesn't support this functionality.

Resolution

In current release version, there is no general printing function for supplier invoices, only for dedicated cases (e.g. ERS, invoice reduction).
