Symptom

  • ASE with full database encrytion report the following messages in errorlog:

    Got Null in buf__free_side_mass mass_ptr:0x145bcd9e10 bmass_stat:0x4201000
    pcstkwalk.fdpr.chunk.1
    ucstkgentrace
    ucbacktrace.fdpr.chunk.1
    bwfinish.fdpr.chunk.2
    sddone_normal
    blkioComplete
    kdcProcessCompletions
    udcPoll
    iocRunController
    krtctskRun
    ktskRun
    ktRTCThreadMain
    thread_create_handler
    end of stack trace, kernel service process: kpid 0
  • There is no ill impact on the ASE operation.

Environment

Adaptive Server Enterprise (ASE) 16.0

Cause

The issue has been logged under SAP CR 824909. Its description reads, " The message "Got Null in buf__free_side_mass mass_ptr:<address> bmass_stat:<hex value>" along with a stack trace is reported in the SAP ASE Errorlog when using full database encryption.".

Resolution

  • There is currently no target SAP ASE Service Pack scheduled for this issue. It has been assigned to our Product Group. Please mark this KBA as a Favorite to be automatically notified when the CR status is changed.
  • These messages are informational can be safely ignored.
  • CR 824909 will address on reporting an information message instead of stacktraces.
