Symptom

  • The Story report is fetching data from a Foundation object that is not maintained under Manage Data.
  • Data is being fetched from cancelled workflows with status "PH" in the story report output.

Environment

  • Product: SAP SuccessFactors People Analytics
  • Data Modelling - Module & Entity Data Issues

Reproducing the Issue

  1. Build a Story report to extract data from a Custom MDF object.
  2. Observe that the report displays information under the affected object that is not available in Manage Data or any of the portlets.

Cause

The Story report includes records from cancelled workflows; these appear with status "PH" and are included in the report output.

Resolution

  1. Open the Story report.
  2. Add a filter on the object's record status field.
  3. Set the record status filter to "N" to return only approved, normal records.
  4. Validate the output to ensure records with status "PH" (cancelled workflow data) are excluded.

Data related to the cancelled workflows can be reviewed from the Manage Workflow Requests page.
