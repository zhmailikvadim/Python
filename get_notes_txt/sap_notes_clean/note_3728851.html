Symptom

  • How to set a User's Status to Blocked ?

Environment

SAP Sales and Service Cloud Version 2

Reproducing the Issue

  1. Go to User Menu.
  2. Go to Settings.
  3. Go to All Settings.
  4. Go to Data Import and Export.
  5. Go to Import and Export Data.
  6. Click Export.
  7. Select Entity as User.
  8. Select Excel or CSV.
  9. Click Export.
  10. Open the exported Excel file.
  11. Observe that some users have the status "Blocked". You want to set some other users to Status - Blocked.

Resolution

  1. Go to User Menu.
  2. Go to Settings.
  3. Go to All Settings.
  4. Go to Users and Control.
  5. Go to Users.
  6. Open the respective User.
  7. Go to the User Details section.
  8. Scroll down and change the "Valid Until" date to a past date.
