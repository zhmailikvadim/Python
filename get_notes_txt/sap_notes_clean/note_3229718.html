Symptom

The supplier is using SAP Ariba Supply Chain Collaboration integrated with SAP ERP or SAP S/4HANA using SAP Ariba Cloud Integration Gateway. Previously, when the supplier enabled the USE_PLANT_TIMEZONE parameter and created an Advance ship notice (ASN) based on the scheduling agreement and provided the manufacturing and best before dates (Date of Manuf. and SLED/BBD), those dates did not appear in the inbound delivery created on the ERP system.

Other Terms

ARBCI1, CL_ARBCIG_DISP_DELV_NOTIF_IMPL

Reason and Prerequisites

The issue occurred due to an erroneous code.

Solution

Now, the code has been enhanced to address the issue. The manufacturing and best before dates are displayed in the inbound delivery created on the ERP system correctly.

 

Action Required

Apply the SAP Note.
