Symptom

  • Error message displayed: "Transforming external to internal tracking data failed."
  • Technical details provided:
  • Correlation ID: xyz
  • Request ID: xyz
  • Timestamp: xyz

Environment

SAP Sports One

Reproducing the Issue

  1. Open the relevant training session.
  2. In the Training Info screen area, choose Import Tracking Data.
  3. Choose a predefined tracking configuration from the list.
  4. Select the relevant file for import.
  5. Review the table entries and save.
  6. Observe the error message: "Transforming external to internal tracking data failed."

Cause

  • There might be a duplicate name entry in the imported file caused the file conversion to fail, resulting in the error.

Resolution

Check the file if it contains duplicate drill names
