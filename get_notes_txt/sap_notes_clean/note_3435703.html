Symptom

There is a formula with a constant number value, this value will not correct into an ALPHA confomr value in the run time.

An advanced correction only availbale for SAP BW/4HANA 2021 and SAP BW/4HANA 2023. FOr other releaes pleas install the named Support Package.

Other Terms

ALPHA, compar; IF statement

Reason and Prerequisites

program error

Solution

To solve the issue please apply the Support Package as mentioned below. In urgent cases you can use the correction instructions.

  • SAP BW/4HANA 2.0

  • SAP BW/4HANA 2021
    Implement Support Package 09 for SAP BW/4HANA 2021 (SAPK-30009INDW4CORE) into your SAP BW/4HANA system. The Support Package will be available as soon as SAP Note 3381477 with the short text "SAPBWNews SAP BW/4HANA 2021 SP09", which describes this Support Package in more detail, is released for customers.
  • SAP BW/4HANA 2023
    Implement Support Package 02 for SAP BW/4HANA 2023 (SAPK-40002INDW4CORE) into your SAP BW/4HANA system. The Support Package will be available as soon as SAP Note 3381441 with the short text "SAPBWNews SAP BW/4HANA 2023 SP02", which describes this Support Package in more detail, is released for customers.
