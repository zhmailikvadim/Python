Symptom

Even email address has been maintained in customer master data, it is not displayed in XML file of JP eDocument.

Environment

  • SAP ECC
  • SAP S/4HANA

Reproducing the Issue

  1. Run T-code EDOC_COCKPIT
  2. Choose JP eDocument
  3. Display XML file of JP eDocument

Cause

"Flag: This Communication Number is Not Used" is ticked for the email address in the customer master data.

Resolution

Please un-ticked "Flag: This Communication Number is Not Used" for the email address in the customer master data.
