Symptom

When running DRCs (previously called ACRs) report for GB VAT return, the following error is displayed in the log: Report submission failed. Internal error occured. Please contact SAP support.

Environment

  • SAP S/4HANA 
  • SAP S/4HANA Cloud Private Edition

Cause

Authentication to the HMRC portal might be expired and a token renewal is required.

Resolution

1. HMRC authentication credentials need to be renewed every 18 months. Follow the steps given in SAP Help Portal Configuring Token-Based Authentication to renew the tokens.
2. Afterwards, try generating and submitting the report again.

See Also

SAP Help Portal Configuring Token-Based Authentication
