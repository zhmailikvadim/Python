Symptom

  • The Connection was terminated abnormally when running the command alter user "user_name" identified by encrypted '***', then SAP IQ Abort.
  • The error occurs as:
    **************************************************
***   SAP IQ Abort:
***      From:  stcxtlib/st_server.cxx:2501

***      Thread: 140687305398016  (TIX: 23)
**************************************************
  • The typical stack trace looks like:
    Unparser::put_string

Environment

SAP IQ 16.1 

Cause

The issue has been addressed under SAP CR 831873. Its description reads, "In rare cases, the server crashes if an ALTER USER statement is interrupted. This has been fixed.".

Resolution

SAP CR 831873 is scheduled to be addressed in the next PL of IQ 16.1 SP05. Please mark this KBA as a Favorite to be automatically notified when the CR status is changed.
