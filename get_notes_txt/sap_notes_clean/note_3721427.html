Symptom

Running the Poland SAF-T Fixed Assets (JPK_ST_KR) report in SAP Document and Reporting Compliance (DRC) results in error message:

Message Text:
Value is not of the min. required length 1

Element Name:
tns:E_1

State/Province:
E_1 Asset ID

Environment

  • SAP S/4HANA
  • Poland

Reproducing the Issue

  1. Configure Poland SAF-T Fixed Assets reports in SAP S/4HANA.
  2. Attempt to extract files through the DRC Report.
  3. Observe the error message: "Value is not of the min. required length 1" for field E_1

Cause

In cluster FIPL_SA_CONF the Definition Key INV_NR_KFL is incorrectly configured for the company which the report was run for.

Resolution

In the Cluster FIPL_SA_CONF check the Asset Definition Keys for the company which the report was run for.

The error is related to Definition Key INV_NR_KFL.

The definition key field can be chosen from the two choices:

Def. Key Field    Tab Nam         Fld Nam           KeyLength

Inventory            ANLA            INVNR                  25
MasterFixedAsset ANLA            ANLN1                  12

Make sure the appropriate logic is chosen in order to display the Asset data correctly.
