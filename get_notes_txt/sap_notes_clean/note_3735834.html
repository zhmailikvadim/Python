Symptom

How to make parameter CONDITIONTAXCODE changeable in BAdI PRCG_DOC_ITEM_CONDITION_AMOUN.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Use the standard BAdI PRCG_DOC_ITEM_CONDITION_AMOUNT in purchase pricing to determine tax conditions.
  2. Attempt to modify parameter CONDITIONTAXCODE within the BAdI implementation.
  3. Observe that CONDITIONTAXCODE cannot be changed and the expected tax code is not derived via this BAdI.

Cause

Missing functionality.

Resolution

The parameter CONDITIONTAXCODE is defined as an importing parameter rather than a changing one, which means it is not possible to modify it in the standard system behavior.

To address this requirement, it's recommended submitting a feature request via the SAP Influence Portal. This channel allows the product team to evaluate customer needs and consider potential enhancements in future releases: 

SAP KBA: 2963059 - How to Submit Feature Requests for SAP S/4HANA Cloud Product
