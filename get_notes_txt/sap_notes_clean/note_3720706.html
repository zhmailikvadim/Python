Symptom

When attempting to assign an object distribution formula to a contract condition where J7 generic rental object type (GRO) is used, the system shows error REGOBD028 - 'Use valuation rule for distribution postings' and prevents assignment.

Environment

SAP S/4HANA Cloud Public Edition

Cause

  • Direct object distribution via the contract condition is not allowed when J7 rental objects are assigned to the lease contract.
  • Posting logic requires a valuation rule for distribution postings.

Resolution

  1. Create one or more ledger-specific valuation rule with the "Transfer posting" valuation type.
  2. When executing valuation postings, this rule will distribute revenue/cost entries to the WBS element or Cost Center assigned to the Usable Object linked to the Rental Object.

Keywords

cash flow distribution, wbs element, lease contract, j7 rental object, valuation rule, transfer posting, distribution rule YA, usable object, revenue distribution, cost distribution
