Symptom

The ACR report Asset Fiscal Maps (Portugal) (PT_AA_FISCAL_MAPS) is used to generate XML files for maps 31 and 32.

As intended, the XML files contain no rows for completely depreciated assets.

However the total XML amounts include also amounts relevant to completely depreciated assets.

In case that only completely depreciated assets exist for a particular map/asset category, the XML file generation error is reported (the line with totals exists, but no asset lines).

Other Terms

PT_AA_FISCAL_MAPS, ACR, XML, XML Totals

Reason and Prerequisites

Incorrect handling of XML totals

Solution

The XML totals no longer include amounts of completely depreciated assets. Hence, also the above described generation error is avoided.

As a general rule, SAP recommends that you install a solution by applying a Support Package. However, if you need to install a solution earlier, use the Note Assistant to implement the correction instruction.

You can find more information about the Note Assistant in SAP Service Marketplace, under service.sap.com/note-assistant.
