Symptom

You attempt to delete an existing Close Out Claim for an Agreement and expect the system to remove the claim successfully. When trying to delete the Close Out Claim, no action is taken, and the system does not delete the record as expected. However, other actions within the Agreement behave normally.

This note corrects the behavior by ensuring that existing Close Out Claims can be deleted successfully.

Other Terms

Post Close Out Claim, Mass Process Close Out Claim, Transactional Calculation Run, Accruals, Delete Close Out Claim

Reason and Prerequisites

 This is caused by program error.

Solution

Please implement the correction instructions attached. These corrections are only valid if you have applied the following support packs.

Release V4 - 2022 V2, 2023, 2023 V2 ,2023 V3 Base Package, or Support Pack 1.

Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object.
