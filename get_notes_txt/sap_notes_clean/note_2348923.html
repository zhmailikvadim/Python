Symptom

This SAP Note contains information about restrictions and recommendations for the usage of

  • SAP GUI for HTML and Browser Usage
  • Fiori Launchpad
  • SAP Themes (additionally see SAP Notes 710719 and 2540597)
    • SAP Belize
    • SAP Quartz
    • SAP Horizon
  • SAP Business Client

in the context of Extended Warehouse Management functionality. Moreover, it contains some recommendations to ensure a better usability during usage.

In addition to that it provides some Additional UI Related Information

  • Setup information for the usage of the Fiori Launchpad
  • Support of Enterprise Search in the context of warehouse management

Other Terms

SAP S/4HANA, SAP Belize, SAP Quartz, SAP Horizon, Fiori3.0,  Fiori 3, "SAP Fiori: the user experience of SAP S/4HANA"

Reason and Prerequisites

Availability of SAP themes in SAP S/4HANA. The Fiori Launchpad is the intended entry point for business users.

Solution

SAP GUI for HTML and Browser Usage

If you are using the browser to run Extended Warehouse Management transactions, the transactions are rendered in SAP GUI for HTML. Therefore, all limitations for SAP GUI for HTML apply. For additional information, see SAP Note 314568. This is especially the case, if you use the Fiori Launchpad with the browser, but the same applies to the usage of the HTML version of the Business Client.

Those general limitations have consequences during the usage of Extended Warehouse Management functionality, especially the ones mentioned below in more detail:

Use of SET/GET Parameters

The usage of SET/GET parameters is limited. Therefore, it is strongly recommended to set the following user parameters in your user master data:

  • parameter /SCWM/LGN Warehouse Number with your favorite warehouse number
  • parameter /SCWM/MON Monitor with the monitor you want to use, for example 'SAP'

to improve the user experience during usage. If you have not maintained the user parameters, it can happen that you are forced to enter the warehouse number and/or the monitor again and again upon transaction launch, or you might encounter issues upon navigation from one to another transaction.

Additionally check SAP Note 3381077 that improves the behavior of SET/GET.

Use of Drag&Drop

The usage of Drag&Drop has some restrictions, for example the drag from a list cell. Therefore, the following transactions have restricted usability. However, all functions can be executed without the necessity to make use of Drag&Drop.

  • /SCWM/DCONS Deconsolidation in Goods Receipt
  • /SCWM/PACK Packing - General

This restriction only applies to releases <= S/4HANA OP 2020.

Copy Content to the Clipboard

In some cases, it is not possible to copy content to the clipboard as in SAP GUI for Windows. See SAP Note 314568 for more details for your software version.

Use of STARTING NEW TASK

The ABAP statement STARTING NEW TASK to open a User Interface in a new session is not supported using the browser. Consequently, the application in a browser behaves differently from running the application in Business Client or SAP GUI. For example:

  • The navigation from transaction /SCWM/MON Warehouse Management Monitor to other User Interfaces is always done in-place and does not open a new session
  • The navigation from Extended Warehouse Management User Interfaces to transaction /SCWM/MON Warehouse Management Monitor, for example from transaction /SCWM/TODLV_I Create Warehouse Tasks for Warehouse Requests - Putaway for Inbound Delivery just to name one, is depending on the release either not possible or done in-place.

Use of Business Graphics

The use of business graphics in SAP GUI for HTML is not supported. This especially affects the following transactions:

  • /SCWM/EGF Warehouse Cockpit cannot be used.
  • /SCWM/EGF_CHART Process Chart Template cannot be used.
  • /SCWM/GWL Display Graphical Warehouse Layout cannot be used.
  • /SCWM/MEDI_AQTY Merchandise Distribution - Maintain Quantity Adjustment (Flow-Through) does not offer the statistics display. As the statistics display is only a different view on data which is already on the screen, functionality is not restricted.
  • /SCWM/PL Labor Management Planning and Simulation does not offer the statistics display. As the statistics display is only a different view on data which is already on the screen, functionality is not restricted.
  • /SCWM/MFS_ROUTES Display Routes (Static) cannot be used.

Use of Local File Selection Dialog

The file selection dialog is typically required in the file up- and download transactions, such as for example

  • /SCWM/ISU Stock Data Transfer, Initial Stock Upload
  • /SCWM/PI_UPLOAD Upload Storage Bins and Count Data
  • /SCWM/IPU Initial Data Transfer of Packaging Specification
  • etc.

If running in the browser, it is required to first load the files to the web server. This requires some additional clicks for a local file upload in comparison to the usage with a local installed client, such as the SAP GUI or the Business Client Desktop Version. For additional information, see SAP Note 300645.

Use of Graphical Screen Painter

The graphical screen painter is not supported in SAPGUI for HTML. Consequently, the transaction /SCWM/RFSCR RF Screen Manager has a reduced usability.

Automatic Scrolling

SAP GUI for HTML automatically scrolls back to the focused field upon Enter. Therefore you might encounter reduced usability in some cases. For example, if you scroll down to a lower section of a UI and then switch to a different tab on a tab control, this may cause the UI to scroll back up to the top. Note that this is a general SAP GUI for HTML behavior but might have more impact with the new themes SAP Belize and SAP Quartz due to the new metrics.

Use of Crystal Reports Viewer

The use of Crystal Reports Viewer is not supported in SAP GUI for HTML. Therefore, it is not possible to make use of the examples provided for the warehouse management monitor or to create own crystal report ALV variants (see SAP Note 1353044 for more details).

Fiori Launchpad

If you are using the Fiori Launchpad in a browser to run Extended Warehouse Management transactions all restrictions mentioned in SAP GUI for HTML and Browser Usage applies. See SAP Note 2540597 for additional information about the restrictions for the usage of SAP GUI for HTML in the Fiori Launchpad.

UI navigation, technically realized via launchpad configuration using transaction LPD_CUST, will display the called transaction outside the Fiori shell in a new session. This includes navigation in the following applications:

  • Shipping Cockpit
  • User Interfaces for Dock Appointment Scheduling
  • User Interfaces for Transit Warehousing 
  • User Interfaces for Staging and Consumption in Production

Moreover, navigation from side panels is not supported in the browser.

If you are using one of the above applications, check if SAP Note 3357308 applies to your release.

SAP Belize

Theme SAP Belize has been the first theme with some major deviations to all existing former themes. See SAP Notes 2348661 and 2328598 for more information. This has consequences for the use of the new design in the context of Extended Warehouse Management application such as:

  • Scroll Bars

For some transactions, you might recognize more scroll bars in comparison to former themes. This might reduce the usability of some transactions such as /SCWM/GRPE and /SCWM/GRPI for Goods Receipt Preparation. This and the restriction as per SAP Note 803582 might as well lead to a reduced usability in transactions such as

    • /SCWM/TO_DISP Display Warehouse Task
    • /SCWM/SGI Pickup of Goods

with large tables and as well

    • /SCWM/CORR Maintain Correction Delivery
    • /SCWM/GRWORK Goods Receipt Workload

with single tables which are larger than the screen size.

To reduce this issue to a minimum, it is recommended to ensure that there is sufficient space for the UI to be displayed. This can be achieved by different means, for example by choosing an appropriate screen resolution, or by working in full screen mode, or by using the zoom function of the browser to adapt the displayed content to the available space.

  • Form views in the warehouse monitor (transaction /SCWM/MON) are displayed as HTML-based form views and are not adjusted to the new theme SAP Belize. As a result, neither colors and font sizes are displayed according to the theme, nor are the labels right aligned. Starting with SAP S/4HANA 2022 On-Premise however, you can decide to display the form views as popups which enables a theme-specific display of the form view data. To use this option set indicator Popup View for your Monitor in Customizing for SCM Extended Warehouse Management under Extended Warehouse Management->Monitoring->Warehouse Management Monitor->Define Monitors.

You can find further information on the theme SAP Belize, including access to cookbook, via SAP Note 2348661.

Transactions and Web Dynpro applications contained in the following technical catalogs are supporting the new visual design with the usability restrictions explicitly mentioned in this SAP Note:

  • SAP_TC_SCM_EWM_BE_APPS - available with SAP S/4HANA 1610
  • SAP_TC_SCM_EWM_AD_BE_APPS and SAP_TC_SCM_EWM_S4_BE_APPS - available with SAP S/4HANA 1709

SAP Quartz

Same restrictions apply as for theme SAP Belize. For more information on the availability and support of theme SAP Quartz and corresponding restrictions see SAP Note 2773530 and SAP Note 2814266. See note 2796898 for further information about the support of SAP Quartz using SAP GUI for Windows.

SAP Horizon

Same restrictions apply as already described for SAP Belize and SAP Quartz apply.

SAP Business Client

For prerequisites and restrictions for the use of SAP Business Client see one of the following SAP Notes depending on your release:

  • SAP Note 2227396 SAP Business Client 6.0
  • SAP Note 2928874 SAP Business Client 7.70
  • SAP Note 3220574 SAP Business Client 8.0
