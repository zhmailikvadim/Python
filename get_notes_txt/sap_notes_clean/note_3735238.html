Symptom

It is required to collect the logged on user number information for last period.

Environment

  • SAP NetWeaver ABAP                                        
  • SAP NetWeaver Application Server for SAP S/4HANA    
  • ABAP PLATFORM - Application Server ABAP

Resolution

RZ20 -> SAP CCMS Monitor Templates -> System Configuration -> [SID] -> System Configuration -> Concurrent Users -> Select Concurrent Users (all Clients) MTE -> Display Details (Shift+F6)
