Symptom

In Fiori App Create Supplier Invoice, error 'G/L account (local) is not unique, please use value help for input' is thrown during Supplier Invoice creation when the posting is performed with reference to G/L Account which has alternate G/L account enablement. The error does not occur in MIRO transaction with the same data.

Other Terms

G/L ACCOUNT, ALTKT, LOKKT, FDC_POSTING_001, FI_DOC_ITM_LOKKT_HKONT .

Reason and Prerequisites

Reason

Program Error.

Prerequisites

1. Alternative G/L Account functionality is configured in the system.

2. Alternative G/L Account is enabled for the G/L account.

Solution

The corrections shall be delivered via the specified support packages.
