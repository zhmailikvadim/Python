Symptom

  • Newly updated field labels in Story Designer are not appearing in story visualizations
  • Field labels display old values despite successful label changes in designer interface
  • Label updates visible in edit mode but not reflected in story view mode
  • Inconsistent label display across different widgets or story components

Environment

SAP SuccessFactors HCM Suite

Cause

Possible Root Causes

1. Story Designer Cache Issues
  • Story Designer interface caching previous label configurations
  • Browser cache retaining old field label definitions
  • Story session cache not refreshing after label changes
  • Widget-level cache preventing label update propagation
2. Label Hierarchy Conflicts
  • Model-level labels overriding Story Designer changes
  • Data source field labels taking precedence over custom labels
  • Widget-specific label settings conflicting with designer updates
  • Query Designer labels not synchronized with Story Designer changes 
3. Field Mapping and Configuration
  • Field mappings not updated after label changes
  • Calculated field labels not refreshing properly
  • Dimension attribute labels not propagating to visualizations
  • Measure label formatting preventing display updates
4. Translation and Localization Issues
  • Missing translations for custom field labels causing display problems
  • Language-specific label configurations not properly maintained 
  • Extension field labels requiring translation setup even for original language
  • Locale settings affecting label visibility in different languages

Resolution

Resolution Steps

Option 1: Story Designer Label Refresh

  1. Force Label Update

    • Re-edit field labels in Story Designer field properties
    • Save story explicitly after making label changes
    • Refresh Story Designer interface and verify label persistence
    • Test label display in story view mode after refresh

  2. Field Remapping

    • Remove affected fields from widgets and re-add with updated labels 
    • Update field mappings in Story Designer data panel
    • Refresh field list in Story Designer to pull latest configurations
    • Verify field binding reflects updated label information
Option 2: Translation and Language Configuration

  1. Extension Field Translation Setup

    • Navigate to Extensibility → Language Adaptation for extension fields 
    • Maintain translations for all extension fields, even for original language
    • Make dummy changes to extension fields to trigger model updates
    • Verify translation setup for custom field descriptions 

  2. Language-Specific Label Management

    • Check user locale settings and adjust if necessary 
    • Verify label translations exist for user's language preference 
    • Test story with different language settings to isolate issues
    • Update field descriptions and labels in Learning Administration if applicable 
Option 3: Widget-Level Label Override

  1. Manual Widget Label Update

    • Access individual chart/table styling and label options
    • Override field labels directly at widget level
    • Apply custom formatting to display updated labels
    • Update axis labels, legends, and column headers manually

  2. Widget Recreation

    • Delete problematic widgets showing incorrect labels
    • Recreate charts/tables with updated field configurations
    • Use copy/paste functionality to reset widget label settings
    • Apply consistent labeling across all recreated components
Option 4: Cache and Session Resolution

  1. Clear Story Cache

    • Perform hard browser refresh (Ctrl+F5 or Cmd+Shift+R)
    • Clear all browser data including cache and local storage
    • Reset SAC application cache through developer tools
    • Clear any saved story states or bookmarks with old labels

  2. Story Recreation for Persistent Issues

    • Create new story instance to test label visibility 
    • Recreate problematic reports if existing stories don't reflect changes
    • Import updated story templates from Content Store if available
    • Apply template-specific customizations with proper label configuration
