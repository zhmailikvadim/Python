Symptom

  • Material confirmation posting for planned stock material for maintenance order and the error 'Message_Type_X' occurs when attempting to post a goods movement from SAP Field Service Management to SAP S/4HANA Private Cloud.



    "Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."     

Environment

  • SAP S/4 HANA Private Cloud Edition
  • SAP S/4 HANA OP
  • SAP Field Service Management (FSM)
  • SAP Cloud Platform Integration (CPI)

Reproducing the Issue

  1. Perform a material confirmation from SAP Field Service Management (FSM) to SAP S/4HANA
  2. Actual Behavior:
  3. Observe that the message fails in SAP with a Message_Type_X error
  4. Expected Behavior:
  5. confirmation should be successful without errors

Cause

There is a program error when creating material document using Function Module 'MB_CREATE_GOODS_MOVEMENT'.

Resolution

Please implement SAP Note 3675553: 3675553 - Message_Type_X error when posting goods movement for Plant Maintenance Order from SAP Field Service Management - SAP for Me

See Also

SAP Note: SAP Note 1797736 - SAP Gateway Troubleshooting Guide - SAP for Me

Keywords

material confirmation failure, st22 dump, Message_Type_X error, Plant Maintenance Order, Replicate Confirmed Maintenance from SAP Field Service Management to SAP S4HANA
