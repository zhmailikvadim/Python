Symptom

You want to use the new IFKL interface function to automatically replicate HR data relevant for the Swissdec service standard KLE from your SAP system to an external C-LTVD-certified system. To do this, you need the documentation for the relevant programs.

Other Terms

IFKL,  service standard KLE, RPUIFKC0, RPLIFKC0

Reason and Prerequisites

See SAP Note 3680417

Solution

Import the specified Support Package. After you import the Support Package, the documentation for the following programs is available:

  • RPLIFKC0: Interface to External C-LTVD Systems

  • RPUIFKC0: IFKL: Auxiliary Program for Registration
