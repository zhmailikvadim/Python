Symptom

You are using ABAP Data Integration to replicate the data into SAP Business Data Cloud. You see a BDC subscription in transaction DHADM in the Subscriber Overview. Using the Button "Unsubscribe" stops the subscription in the CDC engine but leaves entries in table DHAFR_OBJSUBSREG.

Other Terms

S/4HANA, ABAP Data Integration, SAP Business Data Cloud, APRS, CDC, Delta Reconstruction, DHADM

Reason and Prerequisites

The ABAP-based source system is an S/4HANA 2025 FPS00-SP01 system.

Solution

Implement the corrections in the ABAP-based source system.
