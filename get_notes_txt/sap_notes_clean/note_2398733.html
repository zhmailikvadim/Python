Symptom

If the SFC uses relaxed routing, and more than one complete request are tracked at the same time, the inventory's QTY would be updated wrongly without check.

Other Terms

PAPI, relaxed routing, complete

Reason and Prerequisites

Incorrect quantity update when two user work with same SFC same time.

Solution

The fix for the issue has been done in a form of code correction and will be delivered in core component.

Impact:

Complete and quick complete.
