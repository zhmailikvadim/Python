Symptom

This note provides support for Linear Asset Management data integration so that SAP Dynamic Forms can be pre-processed with linear data. Measurement readings that include linear data can also be post-processed to include linear data into the resulting measurement document

The following entities have been enhanced to support linear data:

  • Work Order Header
  • Work Order Operation
  • Work Order Suboperation
  • Notification Header
  • Notification Item
  • Equipment
  • Functional Location
  • Measuring Point
  • Measurement Document

Other Terms

S4MOBADDON-8307, MAIF, SAP Dynamic Forms, EAM, LAM
