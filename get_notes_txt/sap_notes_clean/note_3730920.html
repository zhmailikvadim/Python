Symptom

When you maintain resource attribute values using transaction /US4G/RESRC_MAINT, the system doesn't allow you to enter lowercase values. The system converts all entries to uppercase.

Other Terms

MaCo, /US4G/RESRC_ATTR, /US4G/D_RESRC_ATTR_VAL

Reason and Prerequisites

Advance development

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the system allows you to maintain lowercase values for resource attribute values. You can verify the fix by maintaining a lowercase value for a resource attribute value using transaction /US4G/RESRC_MAINT and checking that the lowercase value is stored and displayed correctly.
