Symptom

Stuck queues for inspection lot with error messages:

  • "data for storage type search is not defined (&1)" "/SCWM/L3 007".
  • "Could not fully determine the destination data" "/SCWM/PUT_BIN_DET" "/SCWM/L4 054".

Environment

SAP S/4HANA Cloud Public Edition

Cause

Resolution

  1. Navigate to the Warehouse Monitor application → Tools → Application log.
  2. Set the object as "/SCWM/WME", subobject as "QINSP", and specify the relevant date and time.
  3. Locate the failed warehouse task (WT) log for putaway task creation.
  4. Review the WT log to identify the issue.
