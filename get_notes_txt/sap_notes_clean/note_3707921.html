Environment

SAP Analytics Cloud

Resolution

Important Notice

  • Classic stories will be deprecated with the Q2 2026 quarterly release of SAP Analytics Cloud version 2026.8.x 
       
  • The planned deployment date of the quarterly release of Q2 2026 of SAP Analytics Cloud is: May 16-18, 2026 For reference, see the SAP Knowledge Base Article:
      
    2888562 - Intelligent Enterprise Suite: Harmonized release calendar for SAP Cloud products
       
  • In case your system is not on quarterly schedule, this release will be deployed roughly one month earlier.
         
  • Classic Account Models are not scheduled for deprecation in Q2 2026 of SAP Analytics Cloud.

  • Note that as of SAP Analytics Cloud wave 2025.14, it is no longer possible to create new Classic Stories, new Applications and new Points of Interest models, but it is still possible to view and edit it.
           
  • For more details, see the following SAP Technology Blogs:
    • Important Notice: Changes in SAP Analytics Cloud Classic Stories & Analytic Applications
    • The Next Phase in the Deprecation of the Classic Experience in SAP Analytics Cloud
