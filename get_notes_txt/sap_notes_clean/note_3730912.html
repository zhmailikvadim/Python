Symptom

 Users encounter an error message stating "User UUID is missing or empty" when attempting to open the task center from SAP Build Work Zone, Standard Edition application.

Environment

  • Identity Authentication Service
  • SAP Build Work Zone, Standard Edition
  • Task Center

Reproducing the Issue

  1. Sign in to the work zone site.
  2. Open the task center app from the launchpad.
  3. Observe the error message: "User UUID is missing or empty".

Cause

The error occurs because the tokens issued to the application do not include the required user_uuid claim.

Resolution

  1. Access the IAS (Identity Authentication Service) Administration Console.
  2. Navigate to the application that corresponds to the affected Work Zone subscription used by Task Center.
  3. Go to the Attributes settings.
  4. Ensure that the user_uuid attribute is activated.
  5. If a Corporate Identity Provider (IdP) is configured, verify that identity federation is enabled according to the relevant KBA: 3699812 - How to Include IAS Global User ID as user_uuid Attribute While Using Corporate IdP.

See Also

3699812 - How to Include IAS Global User ID as user_uuid Attribute While Using Corporate IdP.
