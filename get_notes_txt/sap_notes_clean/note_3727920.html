Symptom

In the MDC cluster, there was no extension point available to enable customers to update the load profile after updating settle_malo.

This issue is relevant for the market roles: DSO, MOS, SUP

Other Terms

/us4g/badi_prof_value_access, /US4G/CL_UBD_DATA_POD, /US4G/CL_UBD_DATA_SETTLEMENT, UPDATE_STL_MALO, update_load_profile, extension point, BAdI, load profile, settle_malo

Reason and Prerequisites

This enhancement was implemented based on a customer request to provide extensibility for updating load profiles after settle_malo updates.

Solution

Please implement the corrections of this SAP Note or import the relevant Support Package (SP).

After implementing this SAP Note, the system provides an extension point in MDC methods, allowing you to add custom logic for updating the load profile after updating settle_malo. No additional action is required unless you choose to implement the BAdI extension. You can verify the fix by implementing the BAdI extension and confirming that your custom logic for updating the load profile runs successfully after the settle_malo update.
