Symptom

When overfill lot is deleted then relevant snapshot text was not present for later tracking for e.g. In case of settlement group generation failure(after overfill lot generation) when  overfill reversal API is called then there is no snapshot of deleted overfill lot.

Other Terms

ACM; LO-AGR-APP; Overfill reversal; snapshot; snapshot text

Reason and Prerequisites

Prerequisite: ACPE should be "Active"

Reason: When overfill lot is deleted then relevant snapshot text was not present for later tracking.

Solution

Implement the attached correction instructions to introduce snapshot text parameter in new overfill reversal API.
