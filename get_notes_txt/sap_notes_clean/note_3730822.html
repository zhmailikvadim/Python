Symptom

When searching for any application in Fiori Launchpad, the following message appears:
"AppSearchError - App search had an error which prevents showing app search results. Please try to reload the page. Technical details of this error may be found in the browser console."

Environment

  • Product: SAP S/4HANA foundation
  • NetWeaver Enterprise Search

Reproducing the Issue

  1. Open Fiori Launchpad.
  2. Use the search to look for any app.
  3. Observe the error message “AppSearchError … prevents showing app search results.”

Resolution

  1. If the system has SAP_UI 7.58 or higher, schedule the report /UI2/PAGE_CACHE_SYNCHRONIZE as a background job (/UI2/PAGE_CACHE_SYNC) on a regular basis to keep the user-independent cache synchronized.
  2. Reload the Fiori Launchpad and test your application search again.

See Also

  • Refer to the online help Scheduling the Synchronization of the User-Independent Cache
