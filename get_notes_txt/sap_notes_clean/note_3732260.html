Symptom

Challan validity needs to be maintained for subcontracting challans.

Environment

SAP S/4HANA Cloud Public Edition

Resolution

To set up challan validity, implement BAdI "J_1IG_SUBCON" and enhance the method "GET_CHLN_VALIDITY" to define the required validity. If the challan should not expire, set the field EX_EOU_FLAG to "X".
