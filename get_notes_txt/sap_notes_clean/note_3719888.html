Symptom

While attempting to release the agreement, the system displays the error message: “You are not authorized to release agreement type in the sales area.” Although the user has been assigned the relevant roles and authorizations.

This note ensures that the user is able to post the agreement request successfully.

Other Terms

 /IRM/IPAGRM, doc_release_authority_check, V_IP_PAGMT, AUTHORITY CHECK, /IRM/IPAGR 088, /IRM/IPPRARM, Agreement Request Post

Reason and Prerequisites

This is caused by program error

SAP Note#3719896 is available for ECC Release

Solution

Please implement the correction instructions attached. These corrections are only valid if you have applied the Support Packs.

Release V/4 - 2023 V2, 2023 V3 and 2023 V3 Support Pack 1

 Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object.
