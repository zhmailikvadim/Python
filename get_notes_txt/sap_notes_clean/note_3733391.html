Symptom

The Troubleshooting Guide link redirects to unavailable page
"Service Unavailable

No server is available to handle this request, or the application is temporarily down for maintenance. Excuse us for the inconvenience."

Environment

  • SAP Integration Suite
  • Cloud Integration (CPI)

Reproducing the Issue

  1. Sign in to your tenant.
  2. Navigate to the upper-right corner of the dashboard.
  3. Select the Help/Profile icon.
  4. Choose “Troubleshooting Guide” from the dropdown.
  5. Observe that the system redirects to a documentation viewer page that fails to load or shows as unavailable.

Cause

The issue is related to the redirection behavior from the SAP Integration Suite home page, where the Troubleshooting Guide link points to an incorrect or unavailable destination.

Resolution

  1. Access SAP Cloud Integration directly using the CPI URL: https://<tenant>.it-cpiXXX.cfapps.<region>.hana.ondemand.com/itspaces
  2. Open “Troubleshooting Guide” from the Help/Profile menu.
  3. Use this entry point as a workaround until the integration suite redirection is corrected.
