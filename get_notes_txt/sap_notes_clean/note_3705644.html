Symptom

The rule created and assigned in userAccountInfo HRIS Element doesn't trigger as per checking rule trace 

Environment

SAP SuccessFactors Employee Central

Cause

Common Problems for Business Rules in Employee Central | SAP Help Portal

Resolution

Configuration issue. Unsupported rule configuration.

Even if the system allows you to create the rule, it won't trigger.
