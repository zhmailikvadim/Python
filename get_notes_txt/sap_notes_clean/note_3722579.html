Symptom

While attempting to replicate employee data using transaction HRSFEC_PTP_EE_REPL for employee an error occurs. This issue is manifested through an error message displayed in transaction SLG1:

  •  Employee class is missing in the job information of employee (HRSFEC_SERVICES165)

Environment

  • SAP SuccessFactors Employee Central Payroll
  • Employee Master Data Replication



Reproducing the Issue

  1.  Go to SE38
  2.  Select report RP_HRSFEC_PTP_EE_REPLICATION
  3.  Fill the mandatory fields
  4.  Execute
  5.  Check logs in SLG1

Cause

Employee class is missing in the job information of employee.

  • The following field is affected p0001-persg
  • The following field is affected p0001-persK 

Resolution

There are missing mapping for the fields employee_class/employment_type on table T77SFEC_CVMAPC. 

These fields must be mapped into a ERP value entry.
