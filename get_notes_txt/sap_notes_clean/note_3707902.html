Symptom

In an Import Scenario when you submit the Nota Fiscal for the Tax Authorities (SEFAZ), the system returns the rejection error 1094: "Rejection: The total of the NF-e differs from the sum of the total of the items on the invoice".

Other Terms

Brazil, SEFAZ, Nota Fiscal, Import, NF, Taxes, NF Totals, vItem, vNFTot, vNF, XML, Rejection 1094

Reason and Prerequisites

This SAP Note is relevant only for Brazil.

You have SAP Note 3707904 applied in your system.

You have SAP Note 3675209 applied in your system.

Solution

After you implement this SAP Note, the system calculates the sum of item values for importing notas fiscais in a new field called Total Value of all items in the Nota Fiscal (NF_ITEM_TOT) in structure Nota Fiscal Header Additional Segment (J_1BINDOCD).

As a general rule SAP recommends that you implement the Support Package specified for your release. However, if you need to install a solution earlier, use the SAP Note Assistant to implement the correction instruction. You can find more information about the SAP Note Assistant in SAP Support Portal, under SAP Note Assistant.

Note that the documentation of certain objects is delivered with the corresponding Support Package only.
