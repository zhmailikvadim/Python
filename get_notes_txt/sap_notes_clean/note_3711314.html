Symptom

This SAP Note provides you with corrections to the Peru Inventory and Balance Ledger (PE_INVENTORY_BALANCE_LEDGER) report in the Run Statutory Reports app for SAP Document and Reporting Compliance. It addresses duplicate journal entry lines in the report output for the electronic accounting book 3.9 (SUNAT code 030900) in the Inventory and Balance Ledger.

When you run the PE_INVENTORY_BALANCE_LEDGER report, the system duplicates journal entry lines in the report output when multiple items of a journal entry are related to the same main asset number. As a result, the report displays repeated rows for the same journal entry.

Other Terms

Peru, Document and Reporting Compliance, DRC, fka ACR, Inventory and Balance Ledger, Libro de Inventarios y Balances, PE_INVENTORY_BALANCE_LEDGER, 030900, PCGE

Reason and Prerequisites

This SAP Note is relevant only for Peru.

You have implemented previous notes listed in SAP Note 3206609.

Solution

After implementing this SAP Note, the system no longer duplicates journal entry lines in the report output of the electronic accounting book 3.9 (SUNAT code 030900) in the PE_INVENTORY_BALANCE_LEDGER report.

SAP recommends that you install a solution by applying a Support Package. However, if you need to install a solution earlier, use the Note Assistant to implement the correction instruction. You can find more information about the SAP Note Assistant in the SAP Support Portal, under SAP Note Assistant.

To implement the solution, you must carry out manual activities and implement the correction instructions that are valid for your release.

Documentation for SAP S/4HANA for Peru is available in the system and in the SAP Help Portal. System documentation (such as field help, customizing, messages) is delivered via Support Packages. SAP Help Portal documentation is available at help.sap.com/viewer/s4h_op_peru.
