Symptom

There is a translation error for key COMMON_Home for es_MX and es_ES locale codes.

It is translating to "Teléfono del domicilio". The correct text should be "Inicio".

A correction is being prepared to be deployed and update the default translation, planned to be delivered on February 24th, 2026. This is subject to change and this article will be updated accordingly.

As a workaround, you may use the Manage Languages tool to update the text for key COMMON_Home:
