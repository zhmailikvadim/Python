Symptom

  • Error message: "invalid hook url" appears when accessing an OCI catalog from SAP GUI transaction ME51N/ME21N
  • The issue occurs when using SAP GUI 800 with "Edge" browser mode (catalog can be reached in lower SAP GUI version or when using "Internet Explorer" mode)

Environment

  • Materials Management (MM)
  • SAP ERP Central Component
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA
  • SAP GUI for Windows 800

Reproducing the Issue

  1. Open transaction ME51N.
  2. Click on the "Catalog" button.
  3. Select the catalog from the catalog list, if multiple catalog exist
  4. Observe the "invalid hook url" error message

Cause

Case sensitivity issue with SAPEVENT URL

Resolution

The web developer responsible for the catalog should change SAPEVENT to lower case to adopt to the changes applied with the patches mentioned in SAP Note 3355910 - SAP GUI for Windows Edge HTML Control: Fixed issues caused due to introduction of new custom URI support APIs - sapevent handling by ABAP applications requires lower case

See Also
