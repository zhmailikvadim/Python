Symptom

There is a requirement to implement an SAP Note in a managed system, and clarification is needed regarding the correct procedure using transaction SNOTE.

Environment

  • Managed system
  • S/4 HANA system

Reproducing the Issue

Not applicable.
This KBA provides procedural clarification on implementing an SAP Note in a managed system via transaction SNOTE.

Resolution

To implement an SAP Note in the managed system via transaction SNOTE, follow the steps below:

  1. Log on to the managed system.

  2. Execute transaction SNOTE.

  3. Choose Download SAP Note.

  4. Enter the SAP Note number to be implemented and choose Execute.

  5. Confirm that the SAP Note has been reviewed.

  6. After the download is completed, expand the New node.

  7. Select the relevant SAP Note (latest version).

  8. Choose Implement SAP Note.

  9. Follow the on-screen instructions to complete the implementation.
