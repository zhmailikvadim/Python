Symptom

  • Intermittent failures occur during scheduled and on-demand refresh jobs in SAP Analytics Cloud.
  • Error message: "An unexpected error occurred while trying to update the dimension data. Events exception: Options is null calling listener [object Object]."
  • The issue occurs during the Customer dimension update phase inside a Refresh Group.
  • Fact data import succeeds, but the dimension update fails, causing the refresh group to end with errors.
  • Manual retry immediately after failure always succeeds without changes.
  • The issue occurs randomly  at different timestamps, without any data changes.

Environment

  • SAP Analytics Cloud 2025.21.13 (Client)
  • SAP Analytics Cloud 2025.21.11 (Server)

Reproducing the Issue

  1. Execute a Refresh Group containing:
  • Fact data import (Ultimate Query).
  • Customer dimension update.
  1. Observe:
  • Fact import succeeds.
  • Customer dimension update fails intermittently with the error.
  1. Re-run the same job immediately:
  • Everything succeeds without any changes.

Cause

This issue is currently under investigation of development team.

Resolution

The solution of this issue is currently under investigation. Please follow this KBA for updates.
