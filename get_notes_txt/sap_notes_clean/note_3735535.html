Symptom

After upgrading to 2023 V2, when opening the Additional Data tab in the Agreement Request, the column widths are automatically reduced, making it difficult to view the full data without manual adjustment.

This issue is resolved by implementing the relevant SAP Note, which provides the necessary corrections.

Other Terms

 Additional Data, Grid, /IRM/IPAGRM, Agreement Request, Column Width

Reason and Prerequisites

This is caused by program error

SAP Note#3735510 is available for ECC release.

Solution

Please implement the correction instructions attached. These corrections are only valid if you have applied the Support Packs.

Release V/4 – 2023V2, 2023 V3 and 2023 V3 Support Package 1.

Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object
