Symptom

When clicking "Save" or "Check and Simulate" in ACACTREE02, in some cases there is a popup "Parameters for Accrual Posting" displayed, in some other cases it is not.

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Environment

  • SAP S/4HANA Finance
  • SAP S/4HANA

Reproducing the Issue

  1. Go to transaction ACACTREE02
  2. Create/change an Accrual object
  3. Click "Save" or "Check and Simulate"

Cause

This popup is only displayed for accrual item type that has opening posting.
This can be checked in below path:
ACEIMG->Accrual Postings->Assign Accrual Item Types to Journal Entry Types and Posting Schemas

Resolution

This is decided by the customizing in below path:
ACEIMG->Accrual Postings->Assign Accrual Item Types to Journal Entry Types and Posting Schemas
