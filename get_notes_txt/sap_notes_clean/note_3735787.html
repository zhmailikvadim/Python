Symptom

This SAP Note delivers technical changes to the following objects, with no functional impact:

  • CL_HRPAYBR_EFDE_JOB_TAB 
  • CL_HRPAYBR_EFDE_POSITION_TAB 
  • CL_HRPAYBR_EFDE_REMUNERATION 
  • IF_HRPAYBR_EFD_MIGRATION_INFO 

Other Terms

PC00_M37_EFD_PAY_GEN; Remuneration; eSocial; S-1200; Migration; PBR99;

Reason and Prerequisites

Reason

This issue is a side effect caused by the solution delivered by the SAP Note 3700999 - eSocial - Alphanumeric CNPJ - Changes in standard events.

When using a type group in a class, some systems require the explicit adition of the type group in the class definition.

This issue may appear only in release 6.04.

Prerequisites

The following Support Package level is the minimum prerequisite:

Release Support Package
SAP_HRCBR 6.04 (EhP6 and EhP7) HR SP J1
SAP_HRCBR 6.08 HR SP B9
S4HCMCBR 1.00 HR SP 08
S4HCMCBR 1.01 HR SP 00
S4HCMCBR 1.02 HR SP 00

For more information, access the SAP ERP Human Capital Management - Support Package Schedules.

When you install an SAP Note, some prerequisites might be required. The prerequisites are SAP Notes that have already been released and that change the same context of the current one. Then, even if these prerequisites are not necessarily related to the symptom of this SAP Note, you must implement them due to technical reasons.

Solution

This SAP Note adds the type group declaration in the following classes:

  • CL_HRPAYBR_EFDE_JOB_TAB 
  • CL_HRPAYBR_EFDE_POSITION_TAB 
  • CL_HRPAYBR_EFDE_REMUNERATION 
  • IF_HRPAYBR_EFD_MIGRATION_INFO
