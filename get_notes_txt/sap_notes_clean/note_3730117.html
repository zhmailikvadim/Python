Symptom

When attempting to nominate a user for a talent pool via the talent pool nominations portlet in the latest people profile, the system displays the error: "looks like something went wrong".

Environment

  • Product: SAP SuccessFactors Succession & Development
  • Talent Pools

Reproducing the Issue

  1. Log in with a user who has permission to manage talent pool nominations.
  2. Open a user’s people profile and navigate to the Succession section.
  3. In the Talent Pool Nominations portlet, select Add.
  4. Open the Talent Pool dropdown to choose a pool.
  5. Observe that the error "looks like something went wrong" appears.

Cause

The MDF object definition for “Talent Pool” has the standard field “effectiveStatus” set to “Not Visible”, removing it from the entity metadata and causing the OData error when the portlet queries TalentPool/effectiveStatus.

Resolution

  1. Go to Admin Center > Configure Object Definitions.
  2. Select Object Definition and search for “Talent Pool”.
  3. If needed, choose Take Action > Make Correction.
  4. Locate the standard field “effectiveStatus”.
  5. Set Visibility to “Editable”.
  6. Save the changes.
  7. Go to Admin Center > OData API Metadata Refresh and Export, and run a metadata refresh.
  8. Log out and back in, then retry adding a talent pool nomination in the people profile portlet.
