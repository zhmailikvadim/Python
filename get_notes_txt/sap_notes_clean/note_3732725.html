Symptom

In Fiori App F1602, if a configurable material is added to the variant and the material is saved, error "Cannot save. There is an inconsistency in the draft" occurs.

Environment

  • SAP ERP Logistics (LO)
  • SAP S/4HANA
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Access the Fiori app F1602.
  2. In General Data, set configurable material for variant.
  3. Attempt to save.
  4. The system shows the error that the draft is not consistent.

Cause

Only classic variant configuration profile is assigned to the configurable material.
Using a configurable material in Fiori app F1602 supports only Advanced Variant Configuration (AVC). This causes the draft validation to fail and blocks saving.

Resolution

For configurable material with classic variant configuration, please perform the maintenance using transaction MM02 instead of Fiori app F1602.
