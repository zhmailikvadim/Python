Symptom

  • The implementation of this SAP Note has no impact on your existing functionality.
  • This SAP Note delivers changes that are usually treated as manual activities within SNOTE (for example, packages, ABAP dictionary objects, message classes).
  • After you have implemented the primary SAP Note 2737095, implement this SAP Note. 

Other Terms

CPRO, MSA, block 81.

Solution

 Implement the attached correction instructions or the corresponding HR-SP or CLC.

Use transaction SNOTE to implement the attached correction instructions after you implement the primary SAP Note 2737095 By doing so, report NOTE_2737101 will be installed in your system.
