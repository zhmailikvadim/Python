Symptom

You are using transaction "RSRT" for example for queries 2CCSRPT30Q or 2CCSRPT40Q. When defining the report parameters, you use the value help and you get an ABAP dump like below.

Category                        ABAP programming error
Runtime Errors               OBJECTS_TABLES_NOT_COMPATIBLE
ABAP: Program               CL_RSRT_DATA_CONTAINER========CP
Application Component    BW-BEX-OT

Environment

Embedded Analytics for S/4HANA Finance for Group Reporting.

Reproducing the Issue

1. Set the global parameters in transaction CXGP.
2. Use transaction RSRT and execute queries 2CCSRPT30Q or 2CCSRPT40Q.
3. You try to use the value help, when defining the report parameters.
4. You get the above mentioned ABAP dump.

Cause

Known program error.

Resolution

Please implement SAP Note 3575710 - "RSRT with dump OBJECTS_TABLES_NOT_COMPATIBLE calling value help" to your system, then perform the test again.

See Also

https://me.sap.com/notes/3575710

Keywords

RSRT, embedded analytics, OBJECTS_TABLES_NOT_COMPATIBLE, CL_RSRT_DATA_CONTAINER========CP, value help, 2CCSRPT30Q, 2CCSRPT40Q, F4 help
