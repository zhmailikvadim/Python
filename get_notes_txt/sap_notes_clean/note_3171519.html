Issue

A purchase order fails to send to SAP Business Network because of the following error message: Unable to create new private supplier.

Cause

The supplier's privateid values within the Ariba procurement solution does not match with the values maintained within the SAP Business Network account.

Solution

To resolve the issue:

  1. Go to Supplier Enablement > Active Relationships.
  2. Search for the supplier, and then choose More Actions > Edit.
  3. Check the value entered for the supplier's Private ID field under the Enter the supplier's private IDs for your procurement applications option.
  4. Compare this value to the privateid sent in the cXML of the purchase order.
  5. Make sure that the privateid matches in both these locations.
