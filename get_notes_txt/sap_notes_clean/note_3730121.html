Symptom

This Note Intends to Clarify regarding the preservation of created on and by data when proceeding with a migration from SAP Cloud for customer V1 to V2.

Environment

SAP Cloud for Customer

Reproducing the Issue

Cause

Resolution

Using the Data Transfer Tool (DTT) preserves the original administrative data, including adminData.createdOn and adminData.createdBy, during V1 to V2 migration.

Keywords

adminData, createdOn, createdBy, preserve creation date, audit fields, historical values, dtt, data transfer tool, impex, data import export, v2 rest api, sales and service cloud v2, initial data load, migration v1 to v2, administrative data
