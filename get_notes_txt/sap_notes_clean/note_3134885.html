Environment

  • SAP HANA 1.0 SPS12 or SAP HANA 2.0 SPS03
  • SLES 15

Reproducing the Issue

Install SAP HANA 1.0 SPS12 or SAP HANA 2.0 SPS03 on a SLES 15 system.

Cause

Some OS settings not configured well or files not installed in SLES 15.

Resolution

  • The error refers to 1984787 - SUSE LINUX Enterprise Server 12: Installation notes - SAP for Me which is a generic note. This indicates some of the OS pre-requisites may not be in place.

  • Please check 2578899 - SUSE Linux Enterprise Server 15: Installation Note - SAP for Me, especially the packages highlighted below;
    • In order to install SAP HANA 1.0 SPS12 or SAP HANA 2.0 SPS03 on a SLES 15 system, make sure that the SLE-Module-Legacy15 (Legacy Module) is enabled, so you can install additional packages that are needed for SAP HANA:
      • libopenssl1_0_0
      • libssh2-1 (only required for a scale-out (distributed) environment)
        2788495 - Using HANA hdblcm fails with libssl.so.1.0.0 / libssl.so.0.9.8 cannot open shared object file - SAP for Me
        
    • missing insserv-compat package
      • E.g. a HANA installation fails with the following error message:
      • INFO: Starting external program /hana/shared/HDB/global/hdb/saphostagent_setup/installsapinit.sh
      • INFO: Command line is: /hana/shared/HDB/global/hdb/saphostagent_setup/installsapinit.sh
      • INFO: Output line 1: /sbin/insserv: No such file or directory
      • The "insserv-compat" package is not installed. Please install the recommended pattern list as mention above. The "insserv-compat" package is part of the "sap_server" pattern.
        
    • In case the csh shell is used for the <sid>adm it could happen that the environment is not correct initialized. Please check which package version of the tcsh is installed."rpm -q tcsh"
      • If the version is: SLES 15 / 15 SP1 --> tcsh-6.20.00-4.6.1
      • Please update to >= tcsh-6.20.00-4.9.1
        
      
  • Please also make sure HANA settings are configured as per 2684254 - SAP HANA DB: Recommended OS settings for SLES 15 / SLES for SAP Applications 15 - SAP for Me
