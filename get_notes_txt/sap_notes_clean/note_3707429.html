Symptom

After an upgrade is performed, issues occur in the Universal Worklist (UWL). Guidance is required on the steps and basic configuration checks that should be followed to analyze and resolve these issues.

Environment

  • SAP NetWeaver Enterprise Portal - release Independent
  • Universal Work List

Resolution

  1. Frist step is to clear UWL cache as per KBA 2141647 - How to clear the UWL cache [VIDEO]
  2. Run a connection test for the system object to the back-end system (System Administration → System Landscape → Test Connection) and ensure the test completes successfully.

    If the issue still persists, then below information should be given to SAP for further analysis:
  3. Complete Portal component Version as per KBA 1757810 - How to get the complete list of software components on the NetWeaver Application Server Java
  4. Step by Step  document with FULL PAGE screenshots, describing how to reproduce the issue.
  5. Reproduce the issue and collect traces as per KBA 2261632 - Taking UWL traces [VIDEO]
