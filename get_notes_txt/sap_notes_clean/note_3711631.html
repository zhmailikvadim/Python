Symptom

Some documents with withholding tax (WHT) information are not retrieved in the statutory report for Spain withholding modelo 216.

Environment

SAP S/4HANA Cloud Public Edition.

Reproducing the Issue

  1. Access the statutory report for Spain Withholding Modelo-216 via Run Statutory Report App.
  2. Review the generated report and observe that certain documents are missing from the statutory report despite being present in the generic report even though the WHT type is used in the original documents.

Cause

The WHT types being used in the missing documents have not been maintained in Configuration Activity 104311 - Assign Country/Rgn Specific Withholding Tax Keys to Withholding Tax Codes.

Resolution

  1. Add the corresponding WHT types to Configuration Activity 104311 - Assign Country/Rgn Specific Withholding Tax Keys to Withholding Tax Codes.
  2. Re-generate the statutory report for Spain Withholding Modelo-216.

Keywords

withholding tax, statutory report, modelo-216, missing documents, WHT type YC, configuration activity 104311, assign country-specific withholding tax keys, Spain
