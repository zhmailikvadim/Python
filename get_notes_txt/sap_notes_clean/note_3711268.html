Symptom

Voided remuneration statement not displayed from off-cycle workbench

Environment

  • Human Capital Management (HCM)
  • SAP R/3
  • SAP ERP Core Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA

Reproducing the Issue

When accessing the transaction PUOC_10, PUOC_99 or PUOC_10 for Off Cycle Workbench, the remuneration statement cannot be displayed

Cause

Void means a logical deletion or override.

The program is working as designed.

The program is not designed to show the remuneration statement for voided runs.

Resolution

It is the intended behavior not to display a remuneration statement, that corresponds to a voided payroll result.

Keywords

Payslip, Paystub, Remuneration statement, Off-Cycle workbench, PUOC_10, PUOC_07, PUOC_99, Void, Payroll result
