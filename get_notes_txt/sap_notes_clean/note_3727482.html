Symptom

  • Error "you need to define the result for process plksef step drcc_resp first (edoprocsetflag)" EDOCUMENT064 arises when submitting PL KSEF e-document.

Environment

  • SAP ERP
  • SAP S/4HANA
  • SAP Document and Reporting Compliance Service

Reproducing the Issue

  1. Navigate to transaction code `edoc_cockpit.
  2. Attempt to submit the eDocument.
  3. Observe the error message: "You need to define the result for process PLKSEF step DRCC_RESP first (EDOPROCSETFLAG)" EDOCUMENT064.

Cause

  • The class responsible for handling the PL KSEF eDocument process (CL_EDOCUMENT_PL_KSEF) does not handle the "REJECTED" status, leading to the issue.

Resolution

  1. Implement the latest version of the SAP Note: 3702882 - Electronic Document Processing for Poland - KSeF: Re-submission Enablement for Customer Invoice (PLKSEF Version 2) and Delete Enablement for Supplier Invoice (PLKSEFIN Version 2).
  2. Ensure all prerequisites mentioned in the overview note 3279206 Electronic Document Processing for Poland KSeF: Implementation Overview SAP for Me are implemented before proceeding.

Note: If the note 3702882 is already implemented, skip this solution and submit an incident to the component CA-GTF-CSC-EDO-PL.

See Also
