Symptom

For stock transfer orders (stock transport orders and stock transfer requisitions) where the corresponding transportation lane created, there is an error below raised at IBP Inbound queue. (bgRFC Monitor app)
- vendor location XXX not found

Environment

SAP Integrated Business Planning for Supply Chain

Reproducing the Issue

  1. Create an STR/STO.
  2. Create an integration model for the created STR/STO.
  3. Activate the integration model to integrate the TR/STO.
  4. Check the Message Monitoring Overview app (bgRFC Monitor app) at IBP , there is an error stating "Vendor location XXX not found."

Cause

The supply plant XXX has a vendor assigned to it.
If purchasing info records are used as transportation lanes between plants, you must integrate the vendor assigned to the supplying plant as well. 
This can be confirmed in table LFA1. (Supplier field and Plant field.)

Resolution

  1. Integrate vendor location XXX into IBP in advance to resolve the error.
  2. Verify that the vendor location XXX is correctly mapped and available in IBP.
  3. Reprocess the error queue.
