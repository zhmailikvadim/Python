Symptom

The message "The workflow &1 is not defined" (MDC_PROCESS_MSG 105) is raised when activating a MDG CMP BC set. 

Environment

  • SAP Master Data Governance
  • SAP S/4HANA

Cause

Missing structural authorization for the userID executing the BC set activation.

Resolution

  1. Verify if the workflow exists and is activated in transaction SWDD;
  2. If the workflow is not active, activate it using transaction SWDD. A new version will be created, and artifacts will be regenerated. To activate the workflow, follow KBA 3516595 - Activating a Specific Workflow Definition and Corresponding Version;
  3. Ensure the userID executing the BC set activation has the required structural authorization profile. To check and maintain the structural authorization, use transaction OOSB.
