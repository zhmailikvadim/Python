Symptom

For the Social Balance Reporting (RPCSBGB0) the structure PMSBA, used in the feature BSOBA for the decisions, is extended with following fields via this SAP Note:

  • Pay scale type (P0008-TRFAR)
  • Pay Scale Area (P0008-TRFGB)
  • Pay Scale Group (P0008-TRFGR)
  • Pay Scale Level (P0008-TRFST)
  • Worker Code (SI Employee Category) (P0100-RSZKN)

The feature BSOBA, as well as corresponding methods of classes CL_HRPAYBE_STRUCTURE_PMSBA, CL_HRPAYBE_FEATURE_BSOBA, is enhanced accordingly.

Other Terms

Feature BSOBA, Structure PMSBA, SoBa, Social Balance, CL_HRPAYBE_FEATURE_BSOBA, CL_HRPAYBE_STRUCTURE_PMSBA, READ_DATA_FOR_CONDITION, READ_FEATURE_VALUE, FILL_STRUCTURE_PMSBA, READ_TAB_PMSBA, RPCSBGB0

Reason and Prerequisites

SAP Note 3728429 is prerequisite.

 This SAP Note is only relevant for Payroll Belgium.

Solution

  • Generally, SAP recommends that you implement the HR Support Package specified for your release.
  • However, if you need to install a solution earlier, use the SAP Note Assistant to implement the correction instruction.
  • You can find more information about the SAP Note Assistant in SAP Support Portal, under SAP Note Assistant.
