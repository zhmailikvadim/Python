Symptom

In the Components of Batch outbound implementation (LSCH_CPNTB), the call for outbound BAdI is missing for specific scenarios.

The outbound BAdI call was provided for Process Orders but is missing for Purchase Order and Material document based replication.

As a result, BAdI implementation is not triggered for the above mentioned scenarios.

Other Terms

ISLS, DRF, LSCH_CPNTB, Components of Batch, Outbound BAdI

Reason and Prerequisites

Implement dependent Notes.

Solution

Implement this Note.

Outbound BAdI calls are introduced for all scenarios in Components of Batch outbound implementation.
