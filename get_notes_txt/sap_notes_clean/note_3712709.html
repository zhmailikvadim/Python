Symptom

Using WBS substitution you are adding investment profile for WBS Elements. While creating new WBS Elements the auto substitution triggers and adds investment profile for settlement with AuC. But while checking the created WBS Element the AuC is not created and the system status AUC is not updated on WBS Element.

Other Terms

AUC, Invetment Profile, WBS Substitution, CJ20N, Project Builder, Asset Under Construction.

Reason and Prerequisites

 Program Error.

Solution

 Implement the correction in the SAP Note .
