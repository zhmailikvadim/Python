Symptom

Need to change the download path while downloading a file from a Fiori App.

Environment

SAP S/4HANA

Cause

The download location is controlled by the web browser’s settings; there is no standard capability in SAP Fiori to set a per-app or per-tile download path.

Resolution

The download behavior is based on the browser settings, if the setting "Ask where to save each file before downloading" is enabled on browser, you can select which folder to save the file while downloading, otherwise it'll be downloaded to the default folder in the browser settings. This is browser behavior and not controlled by SAP products.
