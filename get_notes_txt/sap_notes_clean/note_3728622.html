Symptom

  • More than 100 queues are saved in SMQ3 and are older than 365 days.
  • Users need guidance on downloading saved queues in an Excel sheet.


Environment

  • SAP Extended Warehouse Management (SCM EWM )
  • SAP S/4HANA

Reproducing the Issue

  1. The issue is not reproducible.

Cause

  • SAP advises against saving or deleting queues to avoid inconsistencies between systems.
  • Users have already saved queues and now need to download them for audit or analysis purposes.

Resolution

  1. Navigate to T Code SE16N
  2. Run the table TRFCQINS.
  3. Apply a filter using the date.
  4. Download the data into an Excel sheet.
