Symptom

Absence history for multiple years is visible on the web UI but only current year absences are displayed on the SuccessFactors Mobile App.

Environment

SuccessFactors

EmployeeCentral

Time Off

Reproducing the Issue

  1. Log in to the web UI and observe that absence history for multiple years is visible.
  2. Log in to the SuccessFactors Mobile App with the same user and permissions.
  3. Observe that only current year absences are visible.

Cause

Resolution

The behavior is an actual system functionality and there is currently no enhancement planned to modify it. As a workaround, customers can utilize the `Absence Calendar` feature to navigate to the previous month and review the absence requests for the specific month in question.
