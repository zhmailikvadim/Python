Symptom

You are executing Site Master WB01, WB02 or WB03, navigating to "Listing/Materials planning" tab, but BOM Maintenance tab,  Control data for BOM (T001W-FSH_BOM_MAINTENANCE) field is not shown.

Environment

  • SAP ERP
  • SAP S/4HANA
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • Retail

Reproducing the Issue

Execute WB01, WB02 or WB03 and navigate to "Listing/Materials planning" tab

Cause

I. Required customizing is not set.

II. Data dictionary objects for BOM maintenance indicator in site master is not available in the system (only applicable for Release: 618, Support package: To SAPK-61802INEARETAIL)

Resolution

I. Set the correct customizing by following the steps provided below.

  1. Transaction SM30 and Enter 'V_T078W' in the Table/View input field
    1. click on customizing
    2. Continue w/o Specifying Project
        

     OR

  1. Select Transaction-Dependent Field Selection
  1. Open – Transaction – SPRO -> Press F5 (Click on SAP Reference IMG)
    1. Follow the Path:
      Logistics – General -> Plant Master -> Control Data for Plant Maintenance -> Transaction-Dependent Field Selection



        
  2. Select Activity (Create/Change/Display plant) and Click on Detail (Press CTR + SHF + F2).

  3. Click on 'FieldStat.plant'.

  4. Double click on  'Listing'.

  5. In row 'Fashion Indicators', click on the Radio Button under 'Opt. entry' column .

  6. Press the Save button (CTR + S).



II. If you did not manage to execute the aforementioned steps successfully and you are using a system, which is in Release: 618, Support package: below or equal to SAPK-61802INEARETAIL, please execute the steps provided in Note: 2311344 

See Also
