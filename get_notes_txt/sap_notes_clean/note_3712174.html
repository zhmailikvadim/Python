Symptom

When the case attribute description exceeds 30 characters, it is truncated in the search dialog. In the attribute dialog, the description is displayed in full.

Other Terms

search, attribute, attribute description, document, record, case, document template, record models

Reason and Prerequisites

The structure SCMG_ATTR_ALV_OUTPUT uses the component type CHAR30 for the label fields, limiting the display length to 30 characters.

Solution

Perform the following manual changes.

1. Go to transaction SE11.
2. Select the 'Data Type' option.
3. Enter 'SCMG_ATTR_ALV_OUTPUT' and select 'Change' button.
4. Go to the 'Components' tab.
5. For the component 'LABEL1', change the component type from 'CHAR30' to 'SRMALVL'.
6. Repeat step 5 for the component 'LABEL2'.
7. Save and activate the object.

After these changes, the description display length in the search dialog is increased to 64 characters.

Limitation
Descriptions longer than 64 characters will be truncated. The 64-character maximum is a limitation of the underlying KPRO (Knowledge Provider) framework.
