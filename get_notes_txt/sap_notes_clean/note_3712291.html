Symptom

When doing migration(transaction FARRIC_OL) from sender component to revennue accounting and reporting with direct posting configured on the RA contract header, the migration fails during invoice processing with error message FARR_FOUNDATION#086.

The stored call stack shows the error message is triggered by CX_FARR_ASSERT raised in method CL_FARR_DEFITEM_LIST->GET_UPD_DEFITEM().

Other Terms

FARR, Revenue Accounting and Reporting, Direct Posting, Migration, FARRIC_OL

Reason and Prerequisites

The first entry with migration recon key in deferral item buffer is marked to 'frozen' during direct posting handling in contract creation phase; which prevent the invoice being registered on the same deferral item in the following invoice processing.

The note also downported the correction from note 3441182 to S4CORE106.

Solution

 Apply this note.
