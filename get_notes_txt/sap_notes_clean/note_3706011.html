Symptom

When creating derivation queries for liquidity items via transaction FLQQA1, the error "Unable to Create Queries for Origin Symbol C (Message No. FLQ037)" is raised. 



“Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.”

Environment

S/4HANA 2023 onwards

Reproducing the Issue

Create derivation queries for liquidity items via FLQQA1(or use the following path: SPRO -> Financial Supply Chain Management -> Cash and Liquidity Management -> Cash Management -> Derivation Rules for Liquidity Items -> Define Queries for Liquidity Item Derivation)

Cause

In Basic scope, liquidity item derivation queries cannot be created.

Resolution

  1. Verify that the FIN_FSCM_CLM business function is activated in SFW5.
  2. Configure the system to utilize the Full Scope of Cash Management by navigating to the following path:
    SPRO -> Financial Supply Chain Management -> Cash and Liquidity Management -> General Settings -> Basic Settings -> Set the configuration as 'Full Scope'.
