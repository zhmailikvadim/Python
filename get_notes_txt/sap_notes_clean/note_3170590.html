Symptom

You are using "DSC edition for PEO" and are integrating goods issues for production order components from a S/4HANA system with active PEO into an ECC system.

Consumption deliveries for components that should be withdrawn from an EWM-managed storage location are not created.

Other Terms

asynchronous EWM,

Reason and Prerequisites

Program error:

If all goods movements that are posted together should be withdrawn from an EWM-managed storage location via a consumption delivery no goods movement document is posted. This by mistake was intepreted as an error situation so that no data was posted.

Solution

Implement the correction instruction

Other Terms

F_KKINK, authorization check, AUTHORITY-CHECK

Reason and Prerequisites

There is no check for the authorization object F_KKINK (FI-CA authorization object for the collection agency).

Solution

Implement the attached correction instructions as an advance correction. Otherwise, the corrections are provided with the next Support Package.
