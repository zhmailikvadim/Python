Symptom

  • When using a custom icon on an Action Button, the UI appearance of the button is inconsistent and not displayed as expected in certain instances.

Environment

  • SAP Digital Manufacturing 2602 and higher.

Reproducing the Issue

Configure your Action button Icon with an icon that is not from the Icons input help list, example: 

When opening the POD, the appearance of the button may be different then the expected, example:

Cause

When using icons outside the sap-icon library, like the SAP-icons-TNT library, for example, some POD Designer style options may not work as expected.

Resolution

The behavior reported in this KBA is working as designed. 

The documentation recommends using the Icon input help to choose the icon, see link below:

https://help.sap.com/docs/sap-digital-manufacturing/pod-designer/configuring-pod-action-buttons?q=Action+Button

Although you can pick an icon outside of this, it may not work as expected.
