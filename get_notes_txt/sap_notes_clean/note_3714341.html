Symptom

I have a Project for one Ariba Solution on my Managed Gateway for Spend&Network, but I'm using more than one Ariba Solution in my integration. Can I use the same Project for the other Ariba Solution?

Environment

Managed Gateway for Spend&Network

Resolution

You need to create a Project in Managed Gateway for Spend&Network for each Ariba Solution you are using, it is not possible to use it for more than one Ariba Solution.

For more information on how to create a project check the Installation Guide - Creating a Project.
