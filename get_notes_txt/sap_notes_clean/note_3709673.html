Symptom

When performing a Content Network transport, the CH20 location does not appear in the list of available destination systems.

Environment

SAP Analytics Cloud.

Reproducing the Issue

  1. Navigate to Export → Content Network Storage.
  2. Select the content to be transported.
  3. Set the required package properties.
  4. Proceed to the Destination selection step.
  5. Attempt to select the target location.
    >> Observe that CH20 is not listed among the available destination locations.

Cause

CH20 is a restricted landscape. Therefore, sharing or transporting private packages from CH20 to other landscapes, as well as from other landscapes to CH20, is not permitted.

Resolution

  • The scenario is by design as per Help Page Transporting Your Content Through Content Network Storage:
  • Transporting content is not allowed between SAP Analytics Cloud tenants that are based on:
  • SAP Neo and Cloud Foundry platform
  • EUDP (European Union Data Protection) and non-EUDP framework
  • Canary and Production release strategy

Keywords

Transport, Content network, CH20, Restriction, SAC.
