Symptom

An ABAP Web Service is called. The endpoint called was created using a Simplified Web Service Configuration.

In the response message and in the web service error log you only get an error message like:
Invalid simple client provider call.

Solution

Implement the attached Correction Instruction or the mentioned Support Package.

After apply the corerction you get a more detailed error message in the error log.

You will get an error message like:
Invalid simple client provider call Error when calling SOAP Runtime functions: Authentication Using MYSAPSSO2 Cookie is not allowed (Basic=X, Cert=X, Ticket=)

In addition you now get the reason (here: "Authentication Using MYSAPSSO2 Cookie is not allowed").
And you get the information which logon methods are allowed (here basic authentification and logon using certificate but not logon by ticket).
