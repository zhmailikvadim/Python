Symptom

After migrating documents from Onboarding 1.0, customers are unable to easily identify which documents belong to which users when using the standard migration and filtering options.

Environment

SAP SuccessFactors Onboarding 1.0

Reproducing the Issue

  1. Log in to SAP SuccessFactors.

  2. Navigate to Manage Documents.

  3. Click Apply Filter.

  4. Observe that the document names are associated with the administrator who performed the migration, rather than the original users, as shown in the “Uploaded by” field.

Cause

The “Uploaded by” field in Manage Documents reflects the migration job initiator (administrator) rather than the original Onboarding 1.0 user who generated or uploaded the document.

Resolution

To accurately identify all documents generated or uploaded by candidates from Onboarding 1.0, the Advanced Filter must be used.

  1. Log in to SAP SuccessFactors.

  2. Navigate to Manage Documents.

  3. Select Advanced Filter.

  4. Under Document Owner, choose User and search for the desired user.

  5. Set Entity Type to User.

  6. Click Apply Filter.
