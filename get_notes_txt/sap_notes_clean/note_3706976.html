Symptom

Brazil Localization features not available in Contracts FIORI app

Other Terms

Brazil localization, Royalties facet, FIORI app, Manage ACM Contracts, Maintain ACM Contracts, Approve ACM Contracts, Brazil facet, DPC_EXT class & MPC_EXT class

Reason and Prerequisites

Prerequisite: 3706129 - Brazil Localization: Royalties and Brazil facet in Contracts FIORI app is implemented in the system
Reason: Brazil localization related CDS and common class changes are available in the system

Solution

Brazil localization features are available in Manage ACM Contracts, Maintain ACM Contracts & Approve ACM Contracts applications.

The facet 'Royalties' is added in the Header section of Contract and 'Brazil' facet is added in the Item section of Contract.

Royalties facet in the header will have the information related to royalties technology, royalties percentage, royalties category.

Brazil facet in the item will have the information related to the material usage.

This note has all the code changes related to the CL_ACM_MNTNCONTR_DPC_EXT & CL_ACM_MNTNCONTR_MPC_EXT
