Symptom

When exposing a group or page some applications are not assigned to the page or group, although the referenced tiles are exposed as business apps.

The referenced tiles are not part of catalogs, that are scoped for the exposure. However, other tiles referencing the same original tile are exposed as part of scoped catalogs.

In the exposure log you see messages for the affected entities like: "Tile ABC: Lost reference to catalog XYZ".

Other Terms

cFLP, launchpad Service

Reason and Prerequisites

Programming Error

Solution

Implement this Note or the corresponding support package
