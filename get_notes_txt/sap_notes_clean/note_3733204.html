Symptom

In the process "Sending of Quotation (MOS View)" (DE_B_MOSB_QUOT_MOS), the system triggers a BPEM case EX_MOSB_MDC_PROC_NOT_COMPL in step 0034 when the MDC process isn't completed. However, the quotation sending process doesn't link the MDC process, making root cause analysis difficult without a deep technical investigation.

This issue is relevant for the market role: MOS.

Other Terms

MOS, MOSB, DE_B_MOSB_QUOT_MOS, BPEM case, EX_MOSB_MDC_PROC_NOT_COMPL, Linked Documents, process log, quotation, MDC process

Reason and Prerequisites

Program error.

Solution

Implement the corrections of this SAP Note or import the relevant support package.
After implementing this SAP Note, when a BPEM case raises EX_MOSB_MDC_PROC_NOT_COMPL because the MDC process isn't completed, you can check the MDC process status in the Linked Documents tab or review the process log.
You need to retest the process. You can verify the fix by checking that when a BPEM case raises EX_MOSB_MDC_PROC_NOT_COMPL, you can access the MDC process status through the Linked Documents tab or the process log.
