Symptom

Retrocalculating an employee, RPLDSNF0 (Déclaration Sociale Nominative) generates structure S21.G00.78 with amount 0.00 because S21.G00.79 below carries a retroactive amount. In the final file S21.G00.78.004 (amount) is not generated. This results in a blocking error in the DSN control tool (DSN VAL).

Reason and Prerequisites

This was a program error, which is corrected with this note.

Solution

The correction enclosed within this note will be included in HR Support Packages referenced below. If you need the correction before the HRSP is available, you may apply the attached correction instructions. It should be emphasized that correction instructions are only a short-term solution, not a substitute for the Support Package.
