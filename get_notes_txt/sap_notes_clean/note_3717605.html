Symptom

You create a purchase order or a purchase requisition with an enhanced limit item and get one of these error messages:

  • MMPUR_SPROC/072 - Flexible Real Estate Management is not supported for Lean Services.
  • MMPUR_SPROC/073 - Final account assignment is not supported for Lean Services.

In order to achieve consistent system behavior, implement this SAP Note together with Note 3645782.

Other Terms

Reason and Prerequisites

Error in the source code

Solution

Please implement the attached correction instruction.
