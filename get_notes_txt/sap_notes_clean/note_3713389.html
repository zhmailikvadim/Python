Symptom

The creation of production or process order confirmations or proposals for these confirmations are still provided by API even though they are restricted for the user using the Maintain Business Roles app.

Other Terms

oAuth, C_AFRU_AWK, F8950_PROCORDCONF_TRAN, F8950_PRODNORDCONF_TRAN, API_PROD_ORDER_CONFIRMATION_2_SRV, API_PROC_ORDER_CONFIRMATION_2_SRV

Reason and Prerequisites

No authority checks were implemented for authority restriction check on authorization object C_AFRU_AWK.

Solution

Program Restriction - 2608
