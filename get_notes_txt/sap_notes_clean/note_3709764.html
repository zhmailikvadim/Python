Symptom

  • List of SLT tables required for CARAB 5.0
  • List of SLT tables required for CARAB 6.0
  • Issue encountered during the activation of HANA views via /CAR/ACTIVATE_HTA report.
  • Error message: invalid table name:  Could not find table/view TSKM in schema
  • Missing tables TSKM and TSKMT in the replication schema.
  • Activation errors for containers /POSDW/SAP_CAR_ECC_HDI and /POSDW/SAP_CAR_S4H_HDI.

Environment

  • SAP Customer Activity Repository
  • CARAB 5.0
  • CARAB 6.0

Cause

Missing tables in the replication schema mapped to either SAP_S4H or SAP_ECC.

Resolution

Please use the below links to access the spreadsheet containing list of tables that need to be replicated via SLT for both ECC and S/4HANA source systems:

For CARAB 5.0:

https://help.sap.com/doc/1a6cd228d38b444295a8c6c2f6c3384d/5.0/en-US

For CARAB 6.0:

https://help.sap.com/doc/33074f869a0644579c817d4fe99680a4/6.0.0/en-US

Ensure that the list is restricted to your current source system version - i.e. S/4HANA 2023
