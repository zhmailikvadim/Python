Symptom

How to set a Prompt variant as default on a webI report instead of selecting it manually?

Environment

  • Web Intelligence (WebI).
  • SAP BusinessObjects Business Intelligence Platform 4.1 -Support pack 6 and later.
  • SAP BusinessObjects Business Intelligence Platform 4.2 -all Support packs and Patches.

Reproducing the Issue

The variant feature allows users to save frequently used prompts on a web intelligence report as variants.

1. Logon to BI Launchpad and open an existing webi report that has a prompt.
2. Go to Design mode and go to Query Panel>Run Query>Promot screen shows up.
3. Choose the values you would like to add to Prompt variant>Give the name of Prompt variant and click on Save button. 


4. A prompt variant has been created successfully and the list of prompt variants can be seen as shown below:

5. Click OK and run the report to see the results of selected Prompt Variant.
6. Now open the same webi report again and go to the prompt screen.
7. You will not find any default prompt varaint there as shown below:

Cause

This is by design behavior.

Resolution

Default Prompt variant cannot be set and have to chosen maually.
