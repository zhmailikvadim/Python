Symptom

  • It is not possible to replace a standard system message in the Opportunity entity using language adaptation.
  • The Message ID for the standard message (e.g. "Please enter a party with role Contact Person") cannot be identified.

Environment

  • SAP Sales Cloud Version 2
  • SAP Sales and Service Cloud Version 2

Cause

This is the expected system behavior. Party-related messages are standard and shared across all transactions.

Resolution

This scenario is currently not supported. Party messages cannot be modified via language adaptation, as they are common across all transactions.

See Also

Refer to: 3498271 - New Customer Request for Additional Feature or Function in SAP standard software

Keywords
