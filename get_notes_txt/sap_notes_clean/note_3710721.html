Symptom

Corrections for HANA service data or new HANA service Data are delivered in SAP Note. This is the SAP Note for support package ST-PI 740 SP 34. For ST-PI 2008_1_7xx there no Support Package will be delivered anymore.

Other Terms

 EarlyWatch Alert, SAP HANA Database, ST-PI, EWA

Reason and Prerequisites

Minimum for implementation of this SAP Note:
ST-PI 740 SP 24 or
ST-PI 2008_1_7xx SP 34

Solution

Implement this SAP Note or the Support Package ST-PI 740 SP 34. For ST-PI 2008_1_7xx please implement this SAP Note.

New: 9.2.2026 (Version 149)

- TABLE_OID: For all tables names extracted in the download the TABLE_OID is now also delivered.

Changed:

- M_MERGE_ERRORS: Also SCHEMA_NAME introduced.
