Symptom

The IHB G/L transfer program FJEP_IHB_REV_TRANSFR posts the same payment transaction twice in FI.

Other Terms

FI Document Reversal, FJEP_IHB_REVERSE_GL_TRANSFER, FJEP_IHB_REV_TRANSFR.

Reason and Prerequisites

The symptom only occurs if you have used the program FJEP_IHB_REVERSE_GL_TRANSFER to reverse journal entries in FI that were posted before by the IHB G/L transfer.
The symptom is caused by an error in the CDS view I_IHBLedgerTransferLineItem: This view by mistake did not exclude reversal and reversed line items from being selected.

Solution

Please apply the attached program correction. Afterwards, no duplicate postings will be created anymore in future.
