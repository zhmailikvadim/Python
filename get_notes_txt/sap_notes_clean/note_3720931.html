Symptom

When creating a new business partner with the supplier role in the "Manage Business Partner" application, the following error occurs:

  • "Tax Number 1 (LFA1-STCD1) is a required entry field."

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Use the "Manage Business Partner" application.
  2. Attempt to create a new business partner with supplier role.
  3. Observe that error message "Tax Number 1 (LFA1-STCD1) is a required entry field." occurs when saving.

Cause

The field LFA1-STCD1 is set to Required in customizing.

Resolution

  1. Open the SSUCI/Configuration Activity 103127 - Define Account Groups with Screen Layout (Vendors).
  2. Select the Account Group(s) of the Business Partner for which it is required to make a tax number(s) optional.
  3. Navigate to "General data -> Control" and select the tax number type, e.g., Tax Number 1.
  4. Change the field setting to "Opt. entry."
