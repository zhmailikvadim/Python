Symptom

Not all characters listed in KBA 3707514 can be used to create a Next Number pattern. An error like on a screenshot below is thrown.

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental. 

Environment

  • Digital Manufacturing 2511
  • Digital Manufacturing 2602

Cause

Manage Next Numbers app allows only characters, numbers, hyphen and underscore for SFC next number pattern. The system works as designed.

Resolution

If you need to use characters, which can not be defined in the Manage Next Numbers app but listed in 3707514, they may need to be used from a corresponding order release API method.
