Symptom

In the Analytical List Page created in the Manage KPI Fiori App, you want to use the Combination Column Line charts with multiple measures are columns and a single measure is line

Environment

SAP S/4HANA, Smart Business App

Cause

The Combined Column Line Chart is based on the standard template for all inputs and supports one column with multiple lines

Resolution

Not able to set multiple measures are columns and a single measure is line

See Also

Configuring KPI Details with Label-Value Pairs
