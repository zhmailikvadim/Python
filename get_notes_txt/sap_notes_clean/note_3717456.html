Symptom

After updating data in EC for one employee, the whole employee data is replicated in payload instead of only the affected portlet.

  • For example:  address data is updated in EC. The payload sent from EC shows the address portlet updated and other portlets with no changes. 

Environment

  • SAP SuccessFactors Employee Central Payroll
    Employee Master Data Replication
    PTP replication
  • SAP SuccessFactors Employee Central integration to SAP Business Suite
    Employee Master data replication
    Business Integration Builder (BIB)

Resolution

This is the expected behavior. The SELECT in that query returns all portlets included in the replication in EC each time it runs; it does not limit results to only those changed since the last run.
