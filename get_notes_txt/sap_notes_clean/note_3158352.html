Symptom

  • How to change language of CPI Tenant page.

Environment

  • Cloud Integration Suit

Reproducing the Issue

Steps:-

  1.  Login to CPI tenant.
  2.  Text of the page is in the language of region.
  3. Text expected in some other language.

Cause

  1.  Language of CPI tenant required to be change.
  2.  For Example : In the below image currently language of page is Chinese and user would like to see the text in German language.

Resolution

  1. Tenant's default language is the browser language, means tenant adopt to browser's settings. 
  2. Hence it is required to change the browser language in the expected language of page.

Keywords

Language, CPI Tenant Page, Browser Setting's, Cloud Integration Suit.
