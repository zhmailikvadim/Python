Symptom

Due to some unreleased changes in the Vehicle model, when you try to run task list SAP_ESH_INITIAL_SETUP_WRK_CLIENT, you run into import errors blocking further progress.

Other Terms

VMS, Vehicle management system, SAP_ESH_INITIAL_SETUP_WRK_CLIENT, ESH_IEF_IMP_OM_UPDATE_CONTENT, ESH_OM_OBJ_MODEL_REP-093, ESH_OM_OBJ_MODEL_RP2-225, ESH_OM_OBJ_MODEL_REP-044, ESH_IEF_MSG-081

Reason and Prerequisites

Program error

Solution

Implement the correction instructions attached in this note.
Once done, run the report ESH_IEF_IMP_OM_UPDATE_CONTENT with the checked box 'Try Import multiple. Times/all SWC and execute'.
