Symptom

You have changed the original language of a main object (for example, a data element). You then try to edit the corresponding documentation/docu object. You receive message SD 049: 'Maintain the document in language <xx>'.

The rest of the procedure for revising the documentation is unclear.

Other Terms

Master language, SD049, repository object, ABAP Dictionary object, SD 073, SD073, SD 109, SD109

Reason and Prerequisites

The document cannot be edited because the language used for maintenance does not correspond to the original language of the docu object.

The original language of the main object and the docu object can be different. The original language of the docu object remains unaffected by the conversion of the original language of the main object.

Solution

Read the long text of message SD 049 or SAP Note 61941 (Change master language of documents) with regard to the available options.
