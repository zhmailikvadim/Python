Symptom

CSA Collector ABAP UCON http white list scenario is active. This collector is showing wrong values in CSA compared with the configuration in transaction UCONCOCKPIT

e.g.: System <SID> HTTP-UCON configuration mode is “Active” in transaction UCONCOCKPIT, but the value reported in CSA is “logging”.

Environment

  • SAP Focused Run 4.0
  • SAP Focused Run 5.0

Reproducing the Issue

  1. Execute transaction FRUN
  2. Open tile for Configuration and Security Analysis
  3. Select the affected system and check the collector ABAP UCON http white list scenario

Cause

Program error

Resolution

This issue is fixed for the affected releases described in SAP Note 3701952. Please note that this note includes a prerequisite to SAP Note 3682637 for technical reasons.

Keywords

Logging, Simulated, Cross Origin Resource Sharing, CORS, UCON_CHW, READ_PHASE_OF_CONTEXT_TYPE, CL_HTTP_WHITELIST_API
