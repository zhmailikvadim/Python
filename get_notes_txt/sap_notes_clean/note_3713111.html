Symptom

Not able to create or update TimeAccountType object via OData API or Integration Center.

Cause

Expected behavior. POST (edit) operations are not available for TimeAccountType via OData API or Integration Center.

Resolution

Create or update TimeAccountType via Manage Data or Import and Export data features (refer to Time Account Type).

If you would like to suggest this as an enhancement, please submit it on the SAP Influence website by following the KBA 2090228.
