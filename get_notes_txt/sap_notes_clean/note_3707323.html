Symptom

In the output PDF form 'HR_CZ_ZUCTDAN_5460_29' of the report RPCJVST0M, the processing year is hardcoded in the form title.

Other Terms

RPCJVST0M, PDF form, HRCZ

Reason and Prerequisites

  • This SAP Note is relevant only for the Czech Republic

Solution

The report RPCJVST0M and its output PDF form 'HR_CZ_ZUCTDAN_5460_29' for ATS are updated so that the processing year in the title of the form is filled dynamically.

Important: Apply current version of SAP Note 875986 for NW 7.0x or 1668882 for NW 7.3x and higher. Otherwise an old version of SNOTE can lead to missing coding changes when applying SAP Notes.
