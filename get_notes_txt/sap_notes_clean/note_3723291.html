Symptom

How to export multiple SAP Analytics Cloud SAC stories into a single file?

Environment

  • SAP Analytics Cloud

Cause

Product limitation.

Resolution

It is not currently possible to export multiple SAC stories into a single file using native SAC features.

However a possible option is to use third party software. One such example is made by APOS. The following page contains more details Hybrid Bursting - Merging SAP BI & SAC Content.

"Collate Stories. Burst pages from multiple stories as a single document."

Note: As this is third party software, any support issues with it are outside the scope of SAP Support.

Feel free to submit this as an idea in our Customer Influence page as per KBA 2424376 - How do you create an enhancement request and provide feedback for SAP Analytics Cloud?
