Symptom

  • SCI Bundling has not started
  • SCI Bundling Wizard has not appeared

Reproducing the Issue

  1. Open IDPAdmin tool
  2. Select Add SAP Cloud Identity Services
  3. The SCI Wizard does not appear or it is closed. 

Resolution

Re-launch the SCI Wizard by select the In Progress or select ... and choose "Add SAP Cloud Identity Services"
