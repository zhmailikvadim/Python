Symptom

This SAP Note is relevant for you only if you use SAP EEG Billing for German Electricity Feed.

When you implement SAP Note 3724868, the system issues the following error message:

  • VIEWFRAME_/ENBIL/V_MLDBOG - Changes to /ENBIL/LUM_CONFIGU03 are forbidden by SAP
  • VIEWFRAME_/ENBIL/V_MLDTAB - Changes to /ENBIL/LUM_CONFIGU05 are forbidden by SAP
  • VIEWFRAME_/ENBIL/V_TABCAT - Changes to /ENBIL/LUM_CONFIGU07 are forbidden by SAP
  • VIEWPROC_/ENBIL/V_MLDBOG - Changes to /ENBIL/LUM_CONFIGU04 are forbidden by SAP
  • VIEWPROC_/ENBIL/V_MLDTAB - Changes to /ENBIL/LUM_CONFIGU06 are forbidden by SAP
  • VIEWPROC_/ENBIL/V_TABCAT - Changes to /ENBIL/LUM_CONFIGU08 are forbidden by SAP

Other Terms

ENBIL

Reason and Prerequisites

ENBIL 606 SP23 or lower

Solution

The affected modules must be regenerated.

Perform the following manual steps:

1.) Call transaction SE54 and enter the view /ENBIL/V_MLDBOG in the "Table/View" field.

2.) Select "Generated Objects" and then choose "Create/Change".

3.) On the following screen, choose "Change" (F7).

4.) In the dialog box (Change Generated Elements), choose Module Create and confirm the selection by choosing Continue (Enter). 

5.) The modules VIEWFRAME_/ENBIL/V_MLDBOG and VIEWPROC_/ENBIL/V_MLDBOG are regenerated. The editor lock should no longer be set.

Repeat steps 1.) to 5.) for the views /ENBIL/V_MLDTAB and /ENBIL/V_TABCAT.

Then implement SAP Note 3724868 again.
