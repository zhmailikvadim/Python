HR-CEE/SAP_HR/S4HCM SP11/2025 - HR-CEE/SAP_HR/S4HCM SP03/2026

Solution

---------------------------------------------------------
Installation steps
---------------------------------------------------------

The modifications consist of two parts:
1. Correction Instructions (attached to SAP Note 3729882)
2. Correction Instructions (attached to the current - 3729881 - SAP Note)

1. Apply the Correction Instructions of SAP Note 3729882 with the Note Assistant (transaction SNOTE; more information on service.sap.com/note-assistant) and execute the further implementation steps in the SAP note.

IMPORTANT!
- Log on to the requested clients in HU language and run the newly created report in Update mode (in all requested clients): RPUUDOH0_MANDT_3729882
(Depending on the SAP_BASIS release and SP level, SNOTE may handle the objects which the report tries to update. If so, running the report will display such objects as "Covered by SNOTE" and skip the object. In this case, SNOTE updates the relevant object.)
- Make sure that the created or modified objects are all active.

2. Apply the Correction Instructions of the current SAP Note (3729881) with the Note Assistant (transaction SNOTE; more information on service.sap.com/note-assistant).

The modifications will also be delivered with the HR-CEE/SAP_HR/S4HCM SP04/2026 package (and the equivalent CLC package for Hungary).
