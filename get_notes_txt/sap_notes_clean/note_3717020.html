Symptom

In ME21N transaction an error is raised while inserting a Document Info Record (DIR) via the 'Linked Documents' functionality: 
"Document type & does not allow you to assign to object EKKO_PO" - Message no. 26050

Environment

  • SAP Document Management System (CA-DMS)
  • SAP S/4HANA on-premise

Reproducing the Issue

  1. Run Transaction ME21N.
  2. Create a Purchase Order with mandatory information
  3. Click the 'Documents' icon
  4. Enter Document Info Record (DIR)
  5. Error is raised: "Document type & does not allow you to assign to object EKKO_PO" - Message no. 26050

Cause

Missing configuration

Resolution

  1. Transaction DC10
  2. Create a new document type or select an existing document type 
  3. Under  'Define object links' enter the following entries: 
    Object: EKKO_PO
    Dynpro No.: 500 
  4. Save

See Also

  • Refer to: SAP Note 3664086 - Using DMS for Purchase Order Header attachments in the SAP Fiori app. Manage Purchase Orders
