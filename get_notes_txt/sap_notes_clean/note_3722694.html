Symptom

  • In the "Manage Budget Documents – Project" app, users can only perform budget supplements, returns, and transfers in the global currency.
  • Users are required to calculate and enter amounts in global currency instead of project currency.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Access the "Manage Budget Documents – Project" app.
  2. Attempt to perform budget supplements, returns, or transfers using project currency.
  3. Observe that the app only allows these actions in global currency.

Cause

Feature is currently not available.

Resolution

  1. An Improvement Request has been submitted to address this matter.
  2. The improvement request is under evaluation, and the next update is expected in Q4/2026.
