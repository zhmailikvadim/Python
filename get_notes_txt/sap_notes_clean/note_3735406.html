Symptom

You run the Display of Court Deduction Records (HTWCTDC0) report with the Detail By Payroll Period checkbox selected.

The header text of LREMA in the output is incorrect. This SAP Note fixes this issue.

Other Terms

HTWCTDC0, DDIC text

Reason and Prerequisites

This is a Programe error.

Solution

Do either of the following:

  • Apply the relevant Support Package.
  • Implement the corresponding correction instructions.
