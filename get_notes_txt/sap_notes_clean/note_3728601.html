Symptom

System raises an error with message "At least one invoice line is missing an item name" while submitting e-invoice.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Open the eDocument Cockpit App.
  2. Search for the e-invoice.
  3. Choose the button submit, you will see an error.
  4. Choose the button Go to- Application Log, you will get the mentioned error.

Cause

In the Singapore C5 release, the field Item Text for the Line Item has been specifically designed to map to tag Item Name, which is mandatory requirement set forth by the GST InvoiceNow according to https://go.gov.sg/gstinvoicenowreq-resources.

Resolution

Please ensure that the field Line Text for each line item contains the required value. Once this has been verified, proceed to submit the eDocument again.
