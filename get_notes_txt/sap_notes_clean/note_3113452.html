Symptom

GJ19A: Changing type of message G5-236 from error to warning

Other Terms

JVA, Joint Venture Accounting, G5236, G5 236, Equity groups and are assigned to different equity types

Reason and Prerequisites

When the equity adjustment program is run for two equity groups that belong to different equity types, error G5-236 is displayed, and the process is stopped. The error message is based on the assumption that the same equity group ID can be used for different equity type. As this is not possible anyway, the error message can be changed to a warning message.

Solution

Implement the attached code correction with transaction SNOTE.
