Symptom

In a production order, operations have a control key that results in external processing of the operations. The subcontracting indicator is set; components exist as parts to be provided.

If subcontractor MRP areas are used, an availability check is not carried out there, but generally at the level of the storage location.

Other Terms

Production order

External processing, subcontracting, parts provided
MRP area
Availability check, ATP, Available-to-Promise

Reason and Prerequisites

Use of MRP areas in subcontracting.

A necessary precondition is the implementation of the SAP Note 3590365.

Solution

Implement the changes from the attached correction instructions.

The relevant changes are part of the SAP Standard from release S4CORE 109 onwards.
