Symptom

Keyword search functionality in Talent Search does not return results when keywords from picklists are entered, despite filter options working correctly. For example Risk of Loss field. 

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental. 

Environment

SAP SuccessFactors Succession Management
Talent Search

Reproducing the Issue

  1. Open Talent Search.
  2. Enter keywords such as 'risk of loss low' in the search field.
  3. Observe that no results are returned.

Cause

This is current expected behavior of the system. 

If you want to filter by a pickliest field, you should use the Option ID of that field. You can find it in Picklist Center.

Resolution

In keyword search, if you want to search by a picklist field use the option ID of the field value.

In case this is important to your business, we encourage you to submit an enhancement request to have this functionality considered within future development cycles.

For detailed steps on how to raise an enhancement request please follow the KBA 2090228 - How to Submit Ideas for SAP SuccessFactors Products - SAP ONE Support Launchpad
