Symptom

  • All applications appear as deprecated for certain scope items.
  • Uncertainty regarding whether the scope of test automation has been removed or if new applications are covered.
  • Inquiry about whether the issue could be due to activation of deprecated applications in the catalog or the new applications not being active in the design.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Open the app 'Test Your Process'.
  2. Navigate to the respective test script for scope item (Example:1AB).
  3. Observe that all related applications appear as deprecated.

Cause

The test plan/script content for scope item (Example:1AB) is deprecated.

Resolution

The test automates associated with scope item (Example:1AB) have been deprecated and are not recommended for creating new test plans, as these scripts will no longer be updated with the latest changes. SAP does not provide further updates for these automated scripts.
