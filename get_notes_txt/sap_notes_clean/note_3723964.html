Symptom

  • There are SKIP_CFIN_CLNT and SKIP_CFIN_CLNT2 filters which can be edited to filter client, but SKIP_CFIN_CLNT is only applicable for AUFK, CFIN_ACCHD, KEKO, RBKP, EKKO and COBK; SKIP_CFIN_CLNT2 is only applicable for FIRUD_VATREF, FIRUD_VAT_LTXT, T00F, T001, T001RWT; T001WT, T005, T007A, T007B, T030, T059C, T059FB, T059K, T059P, T059MINMAX, TTXD, and TVAT001 
  • How to filter client for other tables?

Environment

SAP Landscape Transformation Replication Server (SLT)

Resolution

  1. Filter SKIP_CFIN_CLNT can be used for CFIN_CMT_HT table.
  2. Filter SKP_CFIN_CLNT2 can be used for T000F, T003, and T059Z table.
  3. Filter SKIP_COST can be used for COST table. In the default code below, set ??? to the client value
    IF im_mandt = '???'.
      ELSE.
       SKIP_TRANSACTION.
    ENDIF.

As for how to filter the client for other tables not listed, please create a case or chat to SAP Technical Support team
