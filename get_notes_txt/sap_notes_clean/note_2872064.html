Symptom

You carry out the generic organizational object replication from Employee Central in your ERP system.

On the selection screen of the query program RH_SFIOM_ORG_OBJ_REPL_QUERY you specify more than 50 Organizational objects. In this case the generation of the query fails with the error message 'Query can't be created; last modified date goes back more than 3 months' .

Other Terms

 CL_SFIOM_ORG_OBJ_REPL_QUERY

Reason and Prerequisites

There is a program error.

Solution

Import the Support Package related to this SAP Note or implement the relevant correction instructions in your system.
