Symptom

When you run the student loan fund report(HTHCSTL0) for employee whoes loan payment cannot be deducted or fully deducted, a reason code should be generated.

Currently, the program can only generate the reason code 02. For other reason codes, the customers have to use the BADI which was delivered in the note 3500844 to define the logic.

There is no other configuration approach.

Other Terms

HTHCSTL0

Reason and Prerequisites

This is a correction of legal function.

Solution

This note delivers a new table T7THLR and its corresponding maintainence view V_T7THLR so that you can map the action/action reason in SAP system to the following SLF reason codes.

Reason Code Text
     01
 The borrower has resigned/no longer personnel of the org.
     03 The borrower is on leave without pay.
     04 The borrower is under disciplinary action and without pay.
     05 The borrower is deceased.

To be noted: This note only includes the related DDIC objects of the table. It is not a complete solution of the enhancement of STL reason code. The program change will be included in another note later.

Apply the HR support package to get this change.

Alternatively, you can first follow the steps in the attached manual instruction then apply the correction instruction.
