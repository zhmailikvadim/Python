Symptom

Pipeline Run is only generating balance.

Environment

  • SAP Sales Performance Management
  • Incentive Management

Reproducing the Issue

  • Go to Pipeline workspace
  • Run a pipeline for a specific period
  • Se only balance being generated.

Cause

The Period is finalized

Resolution

Only Balances are generated for finalized positions.

As per Finalize Stage: Processing Details | SAP Help Portal:

"Only balances are calculated for closed positions. Balances are carried forward to the next, non-finalized period where they are used in the calculation of new payment amounts." 

See Also

Help Portal: Finalize Stage: Processing Details | SAP Help Portal

Keywords

balance generation, finalized periods, compensate pipeline, pay pipeline, monthly balances, negative balances
