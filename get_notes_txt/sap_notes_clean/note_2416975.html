Symptom

In the standard system as currently delivered, you run the Tax Reporter (PU19) transaction to generate the Form 1099-R. In the paper form 1099 Mult (HR_F_1099R_MULT), box 13 (State/Payer's state no.) is not reporting its configurated values for forms generated for the reporting year 2016.

Other Terms

P100, P101, SFP, 1099R, PDF

Reason and Prerequisites

Reason

Correction of legal function

Prerequisites

As a prerequisite, you have applied SAP Note 2360637 - Year End 2016 Phase I for U.S. Tax Reporter

Solution

As general rule, SAP recommends that you install a solution by applying a Support Package. However, if you need to install a solution earlier, use the Note Assistant (SNOTE) to implement the Manual Instructions.

You can find more information about the Note Assistant in SAP Service Marketplace, under service.sap.com/note-assistant.
