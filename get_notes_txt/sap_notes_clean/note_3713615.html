Symptom

The finding appears as an ATC error (not a functional runtime error) and is reproducible when executing the same ATC check.

Other Terms

ATC finding, ATC violation, ABAP Test Cockpit error, static check issue, code quality finding, ATC message.

Reason and Prerequisites

The change required is purely corrective to satisfy the ATC rule and does not alter program behavior.
