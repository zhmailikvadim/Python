Symptom

Social Insurances Contributions Calculation report HROCERC1 causing dump when reading data that are duplicated in PY table RO_0004.  Data that are processed by PROVIDE command can't contain duplicates.

Other Terms

HROCERC1, RO_0004,

Solution

Processing of data related to Challenges which report read from PY table RO_0004 and IT0004 is extended with the check for duplcates and locked records. Such records are now removed before processed by function PROVIDE.
