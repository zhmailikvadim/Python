Symptom

  • ABAP dump with error "UNCAUGHT_EXCEPTION".
  • Exception class: CX_RITA_APP_LOGGER_ERR.
  • Call stack indicates program CF_TAX_PROPOSAL_FACTORY=======CP.
  • Termination occurs in CL_RITA_APPLICATION_LOGGER====CP.
  • Dump occurs during execution of program RFWT0010 or VF04.

Environment

• Financial Accounting (FI)

• SAP S/4HANA Finance

• SAP S/4HANA

Reproducing the Issue

  1. Run program RFWT0010 or VF04.
  2. Observe that an ABAP dump occurs with "UNCAUGHT_EXCEPTION" and exception class CX_RITA_APP_LOGGER_ERR in program CF_TAX_PROPOSAL_FACTORY=======CP

Cause

Missing Application Log configuration: sub-object FIGL_WHT_0010 under object FIGL is not maintained in transaction SLG0.

Resolution

  1. Open transaction SLG0 (Application Log: Object maintenance).
  2. Check whether object FIGL exists. If it does not, create object FIGL.
  3. Under object FIGL, verify that sub-object FIGL_WHT_0010 exists with the description "Adjustment of Withholding Tax Information to Relevant Types".
  4. If the sub-object is missing, create sub-object FIGL_WHT_0010 with the above description.
  5. If you cannot create or edit entries in SLG0, request the required authorizations or involve the user's role responsible for Basis/ABAP to perform the maintenance.
