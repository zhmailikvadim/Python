Symptom

It is noticed during partial decision when the same Handling Unit (HU) receives materials with multiple inspection lots the partial decision cannot be saved. The system triggers an error message indicating that the QIS step must be completed first. However, when you are attempting to close the QIS step, the system issues another error stating that all quality controls must be completed beforehand.

Other Terms

QA11, Partial decision, Close HU, /SCWM/QUI_WRKST102, /SCWM/QINSP315, Stock Based Inspection, POSC, QIS,

Reason and Prerequisites

Program error.

Solution

Implement the attached correction instructions.
