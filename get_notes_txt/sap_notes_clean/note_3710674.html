Symptom

You would like to install SAP Enterprise Threat Detection log collector 1.0 SP04 PL02 for SAP Enterprise Threat Detection, cloud edition to be able to protect on-premise systems from cyberattacks.

Solution

You can find the installation file(s) in the Support Packages and Patches area of SAP Software Download Center at https://support.sap.com/swdc.

Please note:

  • This is an add-on component for SAP Enterprise Threat Detection, cloud edition.

Documentation

All guides for SAP Enterprise Threat Detection, cloud edition are available at https://help.sap.com/sapetd_cloud.

All guides for SAP Enterprise Threat Detection log collector are available at http://help.sap.com/sapetd_logcollector

SAP Enterprise Threat Detection log collector 1.0 SP04 PL02 provides corrections for the following issues:

  • Bad performance of Log Collector's HTTPSender. It is improved in this patch by supporting multi-threading now. 

 Please note that SAP Enterprise Threat Detection log collector 1.0 SP04 PL02 supports and recommends to use Apache Kafka Version 3.9.1
