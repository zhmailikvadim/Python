Symptom

A position (number XXXX) in SAP SuccessFactors Employee Central cannot be deleted despite onboarding being canceled for the new hire (assignment ID XXXX).


Error ::The system indicates that an employee is already assigned to the position. 

Environment

SAP SuccessFactors Onboarding

Reproducing the Issue

  1. Navigate to >Manage Position> and search for the relevant position 
  2. Click Edit and try to delete the position 
  3. system throwing the error " The system indicates that an employee is already assigned to the position. "

Cause

The issue pertains to a position with matrix relationships and dependencies, including child positions, which prevent deletion.

Resolution

  1. Change the Effective Status of the position to "Inactive" to soft delete the position.
  2.  Verify that there are no child positions linked to this position.
  3. Remove any matrix relationships or related dependencies from the position.
  4.  Once these steps are completed, the position can be deleted successfully.

Keywords

Position, manage position, position delete

The system indicates that an employee is already assigned to the position
