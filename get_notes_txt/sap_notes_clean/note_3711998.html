Symptom

You run the SLD: fichero de bases (RPC_PAYES_CRETA_BASES) program for an employee in the following scenario:

  • The employee receives a salary greater than the maximum contribution base. 
  • A partial absence occurs on the first day of a month and this day is adjusted.
  • No relevant contribution concepts are present in this interval (consisting of only one day). 

Based on the adjusted day, the system places the solidarity contribution concepts to the interval from the second day to the end of the month, resulting in a denial from the Social Security with the following errors:

  • CE 499: SÓLO SE ADMITE EN EL PRIMER TRAMO
  • CE 498: SÓLO SE ADMITE EN EL PRIMER TRAMO
  • CE 497: SÓLO SE ADMITE EN EL PRIMER TRAMO

Example:

You reproduce the symptom by following these steps:

Description

SLD Run for an employee with solidarity contribution and the first day of the month adjusted

Preparation/Background

  1. Hire a monthly employee on January 01, 2025 with a salary of EUR 15,000.

  2. Create a partial parental leave from February 01, 2025, to March 01, 2025.

  3. In the Seguridad Social (ES) (IT0061) infotype, set the Horas de trabajo al mes (contratos tiempo parcial) (P0061-HOCON) field to 50 for March 01, 2025.

  4. Run payroll up to 03.2025.

Execution

Run the RPC_PAYES_CRETA_BASES program for this employee for the period 03.2025, selecting Regular settlement in regulatory period of revenue (L00) payment type.

Result

The system generates an interval from March 02, 2025, to March 31, 2025, with the following concepts:

  • Cotización Adicional Solidaridad. Exceso Primer Tramo (497)
  • Cotización Adicional Solidaridad. Exceso Segundo Tramo (498)
  • Cotización Adicional Solidaridad. Exceso Tercer Tramo (499)

Other Terms

&SLD - PC00_M04_CRETA_BASES

For more information, see SAP Note 1071282 - CONSULT: List of tags for Spain notes.

Reason and Prerequisites

Reason

The system considers that the first interval does not have any significant contribution and does not recognize the interval from day 01 to day 01 as a relevant interval.

Prerequisites

The following Support Package level is the minimum prerequisite:

Release Support Package
S4HCM 102 HR SP 00
S4HCM 101 HR SP 06
S4HCM 100 HR SP 19
SAP_HR 6.08 HR SP D0
SAP_HR 6.04 (EhP6 and EhP7)  HR SP K2

For more information, click on SAP ERP Human Capital Management - Support Package Schedules.

Note:

When you install an SAP Note, some prerequisites might be required. The prerequisites are SAP Notes that have already been released and that change the same context of the current one. Then, even if these prerequisites are not necessarily related to the symptom of this SAP Note, you should implement them due to technical reasons.

Solution

From now on, the system places the solidarity concepts in the first interval (day 01), even if the day is adjusted.

Example:

Following the steps from the example of the Symptom section:

Result

      The system generates an interval from March 1st, 2025, to March 1st, 2025, with the concepts 497, 498, and 499.
