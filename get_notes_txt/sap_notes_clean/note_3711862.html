Symptom

When creating an AI Agent, the desired Extension Field is not visible on the Update SAP Sales and Service Cloud Record Tool.

Environment

  • SAP CX AI Toolkit
  • SAP Cloud for Customer

Reproducing the Issue

  1. Go to AI Agents tab
  2. Click on Create AI Agent
  3. Follow the steps to create the agent using a C4C integration
  4. On Step 4 search for the desired Extension Field on the Update SAP Sales and Service Cloud Record Tool
  5. See that the field is not searchable

Cause

The extension field may not be enabled in the OData endpoint of the C4C tenant.

Resolution

To verify if the field is a returnable element, the following API can be called, substituting the C4C URL:
 
https://<your-tenant>/sap/c4c/odata/v1/c4codataapi/$metadata?sap-label=true&$filter=ServiceRequest

Afterwards, in case the Extension field is not returned, it is necessary to enable it in the settings.
