Symptom

The system displays only PDF export options instead of Excel export options in Access rule details using "Export".

Environment

SAP Access Control 12.0

Reproducing the Issue

  1. Navigate to NWBC > Setup > Generated Rules > Access Rule Detail.
  2. Attempt to export rule details to Excel.

Resolution

It is working as expected , it is showing correct options only (i.e PDF) while using "export" button.

However , if you use the button "export result sets" then it will be downloaded as excel file.

See Also

Refer to: SAP Note 592085 - Set up the HTTP Connect service
