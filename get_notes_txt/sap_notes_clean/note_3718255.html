Symptom

You would like to know how to control batch creation, batch classification or expiration date checks in inbound deliveries. 

Environment

SAP S/4HANA Cloud Public Edition

Resolution

To control how batch creation, batch classification or expiration date checks should be carried out in inbound deliveries, go to configuration activity 102338-Define Batch Creation Controls in Inbound Deliveries

Keywords

Batch Creation, Batch Classification, Expiration Date, Inbound Deliveries, 102338, Batch Creation Controls
