Symptom

When exposing Fiori Launchpad content to the launchpad on SAP BTP via report /UI2/CDM3_EXP_SCOPE, you notice that there are business applications in the log, that show the message "Application type URLT not supported". You did not select those applications for exposure.

Other Terms

Reason and Prerequisites

Program error. The message is displayed for content that is not in scope and has to be removed.

Solution

Apply the attached correction instructions or install the latest support package.
