Symptom

  • Error occurred while loading remote source information: SQL Error: Unable to connect remote source: [SAP AG][LIBODBCHDB SO][HDBODBC] Communication link failure;-10709 Connection failed (RTE:[200110] Invalid connect reply (server may not be SAP HANA) (<ip address>:<port> -> <SQL endpoint>.<region>.hanacloud.ondemand.com:443)).
  • All IP whitelisting are completed, and the server is reachable via ping.

Environment

SAP HANA Cloud

Reproducing the Issue

  1. Attempt to connect the remote source HANA Cloud Database.

Cause

this instance was deleted

Resolution

  1. Verify the instance status in the BTP cockpit.
  2. Ensure the connection is attempted with an existing HANA Cloud Database instance.
