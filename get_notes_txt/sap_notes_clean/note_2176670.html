Symptom

[Reproduce steps]

Logon to web client -> Go to library ->
Create new Folder A, B, C, D under Public. ->
Add new folder B-1 under Public.

Actual result: A, B, C, D, B-1
The left folder tree of library is not sorted automatically after adding a new folder.
The new folder is added to last child node of parent.

Expected result: A, B,  B-1, C, D

Other Terms

Library, Folder tree

Reason and Prerequisites

Program error

Solution

For SAP Business Planning and Consolidation 10.1, version for SAP NetWeaver on 740, upgrade to(Component CPMBPC 810) support package 08 and upgrade BW740(Component SAP_BW) to support package 13.

For SAP Business Planning and Consolidation 10.1, version for SAP NetWeaver on 750, upgrade to BW750(Component SAP_BW) support package 01.
