Symptom

  •  Parameters are showing up as 'Technical Name' instead of 'Description on the Prompt Summary window when refreshing a crystal report in Crystal Reports designer.

Other Terms

 CR2025 parameter description prompt summary

Reason and Prerequisites

Environment

  • SAP Crystal Reports 2025 Patch 8
  • SAP BW (DSL)

Steps to Reproduce

  1. Open a report containing parameters in Crystal Reports designer.
  2. Refresh the report.
  3. Observe that on the prompt summary window the parameters are showing up as 'Technical Name' instead of 'Description'.
