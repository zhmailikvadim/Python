Symptom

Encounters the error message “errorMessage=No handler found for GET /rest/recruiting/candidate/talent/v1/talentPools name=Error” when attempting to add a candidate to a Talent Pool using the Latest Workbench. This issue does not occur when performing the same action via the Legacy Workbench or from the Candidate’s tab, where adding the candidate to a Talent Pool works as expected.

Environment

SAP SuccessFactors Recruiting Management

Reproducing the Issue

  1. Navigate to the Recruiting module.
  2. Select a Job Requisition ID.
  3. Choose a candidate and click on the three dots menu.
  4. Select "Add to Talent Pool".
  5. Search for the talent pool name.
  6. Observe the error message: "errorMessage=No handler found for GET /rest/recruiting/candidate/talent/v1/talentPools name=Error".

Resolution

Our Engineering team has identified the root cause of the issue and has implemented a code correction which will be included in 1H2026 release. For more details on release schedules, please refer to Product Release and Road Map Information | SAP SuccessFactors | SAP Community.
