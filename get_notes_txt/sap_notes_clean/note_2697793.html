Symptom

Error 'No modules were booked; internal error during module selection' showing in module change with 'Check/Correct Conditional Booking' using the transaction code PIQST00

Other Terms

PIQST00, Module change, Check/Correct Conditional Booking

Reason and Prerequisites

Check/Correct Conditional Booking is generating an error message that is preventing module booking changes though the pre-requisite modules are completed.

Solution

Implement the correction instructions

Other Terms

S_TABU_DIS, STDDAT, TDDAT_SAP

Reason and Prerequisites

Missing function

Solution

Implement the attached SAP Note using SNOTE or by means of the referenced support package.
See SAP Note 927123.
