Symptom

  • Unable to restrict the list of hierarchies displaying for dimensions.
  • Unable to control the list of hierarchies to choose from for a dimension in SAP Analytics Cloud.
  • All hierarchies created for a dimension are displayed without an option to limit what is shown to the user.

Environment

SAP Analytics Cloud 2026.2.6

Reproducing the Issue

  1. Log in to SAP Analytics Cloud.
  2. In a story, select a dimension with multiple hierarchies.
  3. Change the hierarchy to use for the dimension.
  4. Observe that all hierarchies created for the dimension are displayed, and there is no option to limit what hierarchies are displayed for the users.

Cause

There is no option in SAP Analytics Cloud to restrict what hierarchies to display for a dimension for users.

Resolution

  • The suggestion is to propose a product enhancement on the SAP Customer Influence platform.
  • For information on how to submit a product enhancement, see the SAP Knowledge Base Article 2424376

Keywords

SAC, hierarchy, restrict
