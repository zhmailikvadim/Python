Symptom

When maintaining BOM items in t-code CS01/CS02/CS03, purchase tab is not being display for every BOM items.

Environment

  • SAP ERP Logistics (LO)  
  • SAP R/3
  • SAP R/3 Enterprise
  • SAP ERP Central Component
  • SAP ERP
  • SAP Enhancement package for SAP ERP
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4 HANA on premise
  • SAP S/4 HANA Cloud Private Edition

Reproducing the Issue

  1. Open BOM in t-code CS02/CS03. 
  2. Double click on BOM item. 
  3. Purchase tab is displayed for some BOM items. 
  4. For other BOM items, purchase tab is not displayed. 

Cause

Purchase information for BOM item is only applicable when the material is not placed in storage. Purchase tab for BOM items is displayed only if:  

  • The BOM item is a non-stock item (item category = 'N'). 
  • The BOM item has a special procurement key defined as "Direct Procurement" maintained. 

Resolution

Review if the BOM item should have purchase information maintained. If yes, create it as a non-stock item or set an appropriate special procurement key. 

How to configure a  "Direct Procurement" special procurement key in t-code OMD9: 

  • Procurement Type should be set to 'F'. 
  • Indicator "Direct Procurement" should be set to active.
