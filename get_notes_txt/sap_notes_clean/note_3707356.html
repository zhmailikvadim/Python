Symptom

"Something went wrong. Please try again or contact your system administrator" error returned when trying to view an item's classes via library search when acting as a Delegate.

Environment

SAP SuccessFactors Learning

Reproducing the Issue

  1. Log into HCM >
  2. Navigate to the Learning Homepage >
  3. Select "Act as" and choose a user to act as their delegate >
  4. Browse the library and select "Browse All" >
  5. Click on "Show Classes" for any item >
  6. Observe the error message: "Something went wrong. Please try again or contact your system administrator."

Resolution

Our Engineering team has investigated the scenario and the correction is planned to be delivered within the 1H2026 release version, as per the schedule in Product Release & Road Map Information for SaaS environments and Learning Validated SaaS Release Schedule | SAP Help Portal for VSaaS environments. 

In the meantime, the following workarounds are available:

  • On the search results page, click on the item to open the item details page where all the classes can be viewed;
  • On the search results page, click on "Show Classes", which will generate the error message, then click on "Show All Available Classes". This will navigate to another page where all the classes can be viewed.
