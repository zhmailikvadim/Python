Symptom

You are using Notes Management in PP/DS and you want to highlight mass planning objects (orders/perations) in Detailed Planning Scheduling Board, for which text-notes were created/changed for the specified period of time.

Other Terms

S/4HANA PP/DS, Detailed Planning Scheduling Board, Notes Management, /SAPAPO/CDPS0

Reason and Prerequisites

Please note the solution for S/4HANA 2023 FPS0 and S/4HANA 2023 FPS1 and higher releases will differ slightly.

Solution

The functionality of mass highlighting of planning objects (orders/operations) in /SAPAPO/CDPS0 will be available in the following path: Menu -> Edit -> Select -> User-Defined.

Please follow the below mentioned manual steps:

For S/4HANA 2023 FPS0 release

1.  first implement SAP Note 3445741 - Highlighting of objects in /SAPAPO/CDPS0 for which text-notes have been recently changed. Via this SAP Note CLAS /SAPAPO/CL_NM_DET_SCHD_PLNBRD will be delivered to the system.

2. In transaction CMOD:

  • create Enhancement Project
  • assign Enhancement: "APOCDPS8"
  • implement Enhancement Component: "EXIT_SAPLCDPSUSER_008" (below is the example of the ABAP code to highlight planning objects for which text-note were changed since last 24 hours)

"Collection of Orders and Operations whose Notes were changed during the last specified hours
NEW /sapapo/cl_nm_det_schd_plnbrd( )->get_ord_opr_with_changed_notes(
EXPORTING
is_period = VALUE #( start = i_start
end = i_end )
iv_hours_of_latest_changes = 24
IMPORTING
et_operation = e_oper_tab
et_order = e_orders_tab
et_bapi_return_info = DATA(lt_bapi_return_info) ).
IF line_exists( lt_bapi_return_info[ type = 'E' ] ).
"Error handling
ENDIF.

"Remove mark from all objects marked before calling current user action
e_first_deselect_all = abap_true.

  • activate Enhancement Component: "EXIT_SAPLCDPSUSER_008"
