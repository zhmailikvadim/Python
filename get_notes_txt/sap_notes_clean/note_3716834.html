Symptom

 When you started the debt change process with different assets, the process was not started.

Reason and Prerequisites

The reason for the error was the validation that checked the eligibility of the assets.

Solution

Adjust the validation so that the asset value types to be checked can be set. This means that the product is converted to a new Customizing Table in which customers can configure which VW categories are not to be checked.
