Symptom

When you click on a material in the list report view of F0251A Manage Material Coverage, the object page opens, but never loads any data.

Other Terms

F0251A, Manage Material Coverage, Material Requirements Planning, MRP, Production Planning, data not loading, no data, stuck, loading screen

Reason and Prerequisites

There was an unstable call to a component in the frontend code, which breaks the app depending on the UI5 version being used.

Solution

This note replaces the call with a safer method and fixes the issue. Note: For technical reasons, the fix for release 2023sp1/UIS4HOP19001 is contained in separate note 3732902.
