Symptom

While pre-screening questions answers are set in sequence, the sort order is messed up in Reimagined Candidate Experience.

Environment

SAP SuccessFactors Recruiting Management

Reproducing the Issue

  1. Set pre-screening questions and answers in job requisition.
  2. Proceed through the application process and respond to pre-screening questions in Reimagined Candidate Experience.
  3. Observe that the answers are not displayed in the intended sequence.

Resolution

The answer option should follow how it is entered in the job requisition Question field. The fix for this issue is planned to be deployed on 2H 2026 release (b2611).

For release timelines, please review SAP SuccessFactors Product Release & Road Map Information.
