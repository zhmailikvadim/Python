Symptom

When executing the run "Activity-Based Costing - Production", and the labor costs are calculated for Process/GL Account, the fixed labor costs are not reduced.

Other Terms

Activity-Based Costing - Production, Proportional Labor Costs, Fixed Labor Costs

Reason and Prerequisites

Program error

Solution

History:

Version 1 of 17 March, 2026: New creation

Install the code correction.

Description of Correction:

When executing the run "Activity-Based Costing - Production", and the labor costs are calculated for Process/GL Account, the fixed labor costs are reduced.

Functional Dependencies:

There are no functional dependencies.

Manual Changes:

There are no manual activities.
