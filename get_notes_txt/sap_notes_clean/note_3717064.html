Symptom

A DTP request failed with errors:

  • Metadata inconsistency in method 'ALTER_RANGE_PARTITION'. Location: '-1-'.
  • Exception CX_RS_FAILED logged

Environment

SAP Datasphere, SAP BW Bridge.

Cause

Metadata inconsistency for the ADSO.

Resolution

Reactivate the affected ADSO.

Keywords

inventory-enabled adso, bw bridge, data loading issue, partitioned adso, metadata inconsistency, alter_range_partition, data refresh, adso reactivation,
