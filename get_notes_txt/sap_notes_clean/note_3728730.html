Symptom

  • The message "One or more calculations are invalid or missing. Please review the calculations used in this table." is shown in the Table widget after converted story from classic to ODE.

Environment

  • SAP Analytics Cloud (Enterprise) 2026.2.10

Reproducing the Issue

  1. Log in to SAC
  2. Access a classic story with a Table widget working fine.
  3. Convert the classic story to Optimized Design Experience story.
  4. Check the Table widget in the converted ODE story.
  5. Observe the message "One or more calculations are invalid or missing. Please review the calculations used in this table." in the Table widget.

Cause

The cause of the issue is not explicitly mentioned in the case information.

Resolution

The solution of this issue is currently under investigation. Please follow this KBA for updates.
