This problem was originally solved with the note BA-BEA: DSEU termination with runtime error GETWA_NOT_ASSIGNED (3583335).

Since implementation of note BA-BEA: DSEU additional months for unwid. Release from work with continued payment (3684107), the error occurs again. During the modification of the method for creating the remuneration modules, a transfer parameter for creating the splits was not transferred correctly.
