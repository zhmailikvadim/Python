Symptom

You're replicating contract accounts from SAP ERP to Cloud for customer and observe the error message "Solve Configuration Issue in ID Mapping for Element PARTY_ID-CONTENT With ID "XXXX" on the service interface BusinessAgreementReplicationInERP.

Environment

  • Cloud for customer
  • SAP S/4HANA

Reproducing the Issue

  1. Open the C4C system.
  2. Navigate to the Administrator work center.
  3. Open the Web service message monitoring.
  4. Search for the message ID or error message.
  5. Observe  the error message "Solve Configuration Issue in ID Mapping for Element PARTY_ID-CONTENT with ID "XXXX"

Cause

  • There is missing ID mapping for integration in C4C for the Business Partner on the failing payload.
  • Check if the partyID in the payload is present in the business partner workcenter withiin C4C, if not present than you need to replicate this Business Partner from S4.
  • The ID mapping of "ERP accounts" is missing.

Resolution

Maintain the ID mapping in C4C manually or replicate the Business Partner from S4 and this will automatically create the ID mapping in C4C.

Keywords

C4C, ID mapping. ID mapping for integration, Business Partner, S4, PartyID, PARTY_ID-CONTENT, BusinessAgreementReplicationInERP
