Symptom

When using S/4HANA output control as the default document output process for sales documents, exception NO_SUPPORTED_ACCESS is raised in the program SD_SDOC_PRINT01, when attempting to determine the serial number of a maintenance item, and causes the printing program to crash.  The printing program should ignore this exception because it is not relevant for printing the document.

Other Terms

S/4HANA output control, serial number, SD_SDOC_PRINT01, NAST, GET_SERNOS_OF_DOCUMENT, NO_SUPPORTED_ACCESS, NAST_PROTOCOL_UPDATE

Reason and Prerequisites

This issue is caused by a program error.

Solution

Implement the correction instructions provided with this SAP Note or upgrade to the relevant Service Pack.
