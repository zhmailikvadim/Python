Symptom

  • An error occurs when attempting to upload a CSV file to /SCWM/ISU for stock upload.
  • The Handling Unit Type (HU Type) is specified in the CSV file but it is cleared during the upload process.
  • The uploaded data shows the HU Type as blank instead of the one from CSV file.

Environment

  • Goods Receipt
  • Goods Issue

Reproducing the Issue

  1. Prepare a CSV file containing stock data, including Handling Unit Type set to "E1" for example.
  2. Attempt to upload the CSV file to /SCWM/ISU for stock upload.
  3. Observe that the Handling Unit Type is cleared and appears blank in the uploaded data.

Cause

Resolution

It is the standard behavior that Handling Unit Type is cleared and appears blank in the uploaded data if the Handling Unit Type in the CSV file is the same as that maintained in the product master data of the packaging product.
