Symptom

  • Failed to delete migration object for table MDKP of mass transfer 00L
    Migration object <object_name> is locked by user <username>
  • SLT tables are in status 'Load /Replication blocked'.

Environment

SAP Landscape Transformation Replication Server (SLT)

Cause

Obsolete consumer registrations in the source system. Consumers link to configurations that are either deactivated, empty, or no longer exist

Resolution

  1. Identify obsolete consumer registrations in the source system LP1 by reviewing the IUUC_1N_CONS_REG table.
    Reference KBA 2467080  -  How to check if SLT logging tables are obsolete and OK to delete
  2. Delete obsolete consumer registrations manually from the IUUC_1N_CONS_REG table using SE16 or SE16N.
  3. If manual deletion is not possible, use function module IUUC_UNREGISTER_1N_PNT in SE37 on the source system to deregister invalid consumers. Fill the required fields as follows:
    • IV_LOGTAB_NAME: Name of the logging table (e.g., /1CADMC/00002282).
    • IV_PNT_GUID: GUID of the configuration found in SLT table IUUC_REPL_MAP
    • IV_APPL_TAB: Name of the application table.
    • IV_LOGICAL_POOL_TABLE: Leave blank.
    • IV_LOG_TYPE: S.
    • IV_DBCON_NAME: Leave blank (used only for non-SAP systems).
