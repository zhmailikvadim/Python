Symptom

  • User unable to log in to FSM Web App (Shell).
  • Error message displayed: "An unexpected error occurred during logon, please log on again."

Environment

SAP Field Service Management

Reproducing the Issue

  1. Open the FSM Web App (Shell)
  2. Attempt to log in using the affected user’s credentials

Actual Result:
The user is unable to log in to the Shell due to the missing Allowed Client: Shell permission
The following error message appears: "An unexpected error occurred during logon, please log on again."

Expected Result:
The user should be able to log in to the FSM Web App successfully without receiving an error message

Cause

  • The affected users were missing the Shell permission in the Allowed Clients configuration, which prevented them from authenticating and accessing the FSM Dispatching Board Web App.

Resolution

  1. Log in to the Admin Portal
  2. Navigate to the Policy Group of the affected user
  3. Locate the Allowed Clients configuration section
  4. Add the Shell permission for the user under Allowed Clients
  5. Save the changes
  6. Verify that the user can successfully log in to the FSM Web App

Keywords
