Symptom

Files are not reaching Express Data Loader after manually drop them using SFTP.

Environment

  • SAP Sales Performance Management
  • Incentive Management on Hyperscalers
  • Incentive Management on Legacy HANA 

Reproducing the Issue

  1. Drop files for transfer to the SFTP system.
  2. Observe that not all files are successfully received by the Express Data Loader.

Cause

Files are being renamed during SFTP file migration to Express Data Loader.

Resolution

Please avoid any processes that can potentially rename the files that are being dropped. In case of files are being renamed, the system will not fully process the file and it will not reach the Express Data Loader to be processed as expected.

Keywords

SFTP, file transfer failure, renamed files, XDL system, inbound directory, file upload timestamp, dependent file, batch files, file processing failure, Express data loader, GoAnywhere
