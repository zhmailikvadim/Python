Symptom

In the Process Production Versions(F6400) app, the Operation Overview in the Tasklist tab does not display Master Recipe data even though the data exists in the backend. This issue occurs only when any value is explicitly specified in the authorization object C_ROUT.

Other Terms

Process Production Versions, F6400, C_MasterRecipeActivity, I_RoutingHeader, I_MasterRecipeHeader, C_ROUT, Access Control, DCL

Reason and Prerequisites

Program Error

Solution

Implement the attached Correction Instructions.

Note: Please activate the DDL View C_MasterRecipeActivity first followed by DCL View C_MasterRecipeActivity to avoid error "Element _RecipeHeader does not exist in entity C_MasterRecipeActivity" in activation.
