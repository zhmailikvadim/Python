Symptom

You are looking to filter the 'One-Time Deduction' and 'Recurring Deduction' portlets so they only show particular pay components in their 'Pay Component' drop-downs

Environment

SAP SuccessFactors HCM Suite

SuccessFactors Employee Central

Reproducing the Issue

  1. Go to an employee's profile
  2. Go to the 'One-Time Deduction' or 'Recurring Deduction' portlet and add a record
  3. When selecting the 'Pay Component' drop-down, you wish for it to be filtered to only show certain pay components

Resolution

As of the current product, it is possible to filter the pay components which show in the 'One-Time Deduction' and 'Recurring Deduction' portlet drop-downs if the pay components are configured to be associated to Country/Region.

Our handbook documentation details how to configure the pay components based on Country/Region.

If your pay components are configured to be based off of Country/Region, then the 'One-Time' and 'Recurring' Deduction portlets will automatically filter the pay components based on the employee's country/region.

NOTE: Currently, associating pay components based on Legal Entity is not recommended or supported, and while it is possible to use a custom configuration for it, it will only work for 'Recurring Deduction' and not for 'One-Time Deduction' (more details in internal memo). As this configuration is not currently supported, Support cannot help with investigation of the same if the customer chooses to configure it and it stops working at any point.
