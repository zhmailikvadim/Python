Symptom

Field record-id  of national id records in dependent portlet is visible without permission in Latest People Profile.

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

SAP SuccessFactors Employee Central 2H 2025

Reproducing the Issue

  1. Navigate to the dependent > national id portlet.
  2. Field record id is visible even permission (Employee data > HR information > record-id) is not granted.
     
  3. In Legacy People profile, the field is invisible.
      

Cause

This is a Known Issue.

Resolution

The fix for this issue is planned to be deployed on 1H 2026 release.

For release timelines, please review SAP SuccessFactors Product Release & Road Map Information.

Workaround

No workaround is available.
