Symptom

There is error "CLS_S4_CONV 004: Invalid call; Characteristic short and long value must be equal" in BAPI_OBJCL_CHANGE.

Environment

  • SAP ERP Logistics (LO)
  • SAP ERP Classification (CA)
  • SAP S/4HANA
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA Cloud Private Edition

Cause

In BAPI_OBJCL_CHANGE, table ALLOCVALUESCHARNEW, fields VALUE(VALUE_CHAR & VALUE_NEUTRAL) and VALUE_LONG(VALUE_CHAR_LONG & VALUE_NEUTRAL_LONG) contain different values.

Resolution

If both fields VALUE(VALUE_CHAR & VALUE_NEUTRAL) and VALUE_LONG(VALUE_CHAR_LONG & VALUE_NEUTRAL_LONG) are provided, it needs to have same values in corresponding field VALUE and VALUE_LONG.

If system is already in target mode, the best approach is to only use field with suffix "_LONG".
For information on different modes in S/4HANA system, please check KBA 2782444.

Keywords

CLS_S4_CONV004, BAPI_OBJCL_CREATE, MFLE, Material Field Length Extension, compatibility mode, OBJECT_KEY, OBJECT_KEY_LONG
