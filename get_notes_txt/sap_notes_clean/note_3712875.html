Symptom

In the Fiori app "Display Meter Reading," action buttons on the detail page display in English even when you log in with German language.

Other Terms

 Display Meter Readings, Fiori app, action button, German language, language settings, localization, translation, detail page

Reason and Prerequisites

 The error occurred because of a program error that prevented the system from displaying the action buttons in the correct language.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the system displays action buttons in German on the detail page of the "Display Meter Reading" app when you log in with German language. You can verify the fix by logging in with German language and checking that the action buttons on the detail page display in German.
