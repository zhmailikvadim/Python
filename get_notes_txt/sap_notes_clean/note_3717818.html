Symptom

Although the logon language is Japanese, the F1 help document is not displaying in Japanese, but in English.

Environment

SAP Netweaver release 750 or higher.

Reproducing the Issue

  1. Go to SE38.
  2. Input any program and display.
  3. Press F1 key to show the help document.

Cause

Since SAP basis 7.50, the F1 help information is not offered in Japanese anymore. Only English and German documentations are available in SAP basis 7.50 and higher.

Resolution

Other languages are not displayed from release 7.50 on, because some years ago it was decided that the ABAP Keyword documentation is translated from German to English only. From release 7.55 on, the original language of the ABAP Keyword documentation is English and it is translated to German only.
Reference:2390554 - Translation of the ABAP Keyword Documentation
