Symptom

The following issues are resolved/fixed as part of this SAP Note:

1. While creating a new plan from the RA drop-down, the previous year is not shown for the fiscal year.

2. The date format in the Import Planning Data popup is not aligned with the user’s SU3 configuration.

3. The Product Picker shows an incorrect GTIN for BOM headers. This has been fixed by ensuring that when a structure product = ‘X’, the GTIN CU and CU unit remain blank.

4. In the Fast Promotion screen, some filter values are not displayed, and the filter criteria does not work when “All My Responsibility Areas” is selected in the RA drop-down.

5. When creating a TPM-only RA Promotion, the Flat View is empty on the Detail Promotion Summary screen after adding products.

6. Document URL loading issue in TPM new vs. old TPM_pro roles: If a document with the same name is attached multiple times in Plan and Promotion, the wrong attachment is referenced/downloaded.

Other Terms

Create Plan, Import Planning Data, GTIN for BOM headers, Fast Promotion filters, Attachments

Solution

Please implement the provided correction instructions.
