Symptom

Returned with dump when click preview from W0017 Change maintenance order from print out put tab.

Environment

SAP S/4HANA Cloud Private Edition

  3. Navigate to Additonal functions tab > Print

  5. Execute preview

Cause

Program Error

Resolution

  1. Implement note 3644576
  2. Also For resolving the error "URL for redirect does not exist in HTTP allow list" start the transaction UCON_CHW or UCONCOCKPIT and maintain the URL the Whitelist. It is possibly there already under 'Not covered by Whitelist'. In this case it only has to be added to the covered sites by clicking on 'To Allowlist'. The site must have a green light in front of it, when checked in UCON_CHW. Additionally if you find a corresponding dump in transaction ST22, you should find the URL there beneath LV_SOURCE_URL.
