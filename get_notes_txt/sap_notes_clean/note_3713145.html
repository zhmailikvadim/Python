Symptom

In the app Configure Substitution Strategy, it is not possible to save.

Other Terms

ATP; aATP; Third-Party Order Processing; TPOP; Alternative-Based Confirmation; ABC; Substitution Strategy; Sourcing Profile

Reason and Prerequisites

The symptom incorrectly happens if all the following conditions are met:

  • Remaining Open Quantity Assignment to the leading substitution is maintained;
  • substitution method Plant Substitution with substition source All Possible Plants is maintained as single substitution method;
  • sourcing profile is maintained.

Solution

Apply the correction instructions for this note or upgrade to a SAP S/4HANA support package that contains this correction.
