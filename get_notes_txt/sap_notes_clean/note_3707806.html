Symptom

Error: You cannot change a field name appears while maintaining the SPRO parameters for SAP Integration Suite, Managed Gateway for Spend Management and SAP Business Network.

Reproducing the Issue

  1. Log in to SAP ERP or SAP S4HANA
  2. Navigate to SPRO path Integration with Other SAP Components > Managed Gateway for Spend&Network for Buyer > Master Data  > General Settings  > Maintain Parameters > Sourcing  
  3. Click on the Modify button.
  4. Select all the parameters
  5. Delete all the entries and paste the entries copied from another resource. 
  6. Observe the error: Field Name cannot be updated.

Cause

Copying and pasting entries from other resource inadvertently results in a space value being pasted in column Field name , which is interpreted as an editable field, causing the error: Field Name cannot be updated.

Resolution

  1. Avoid copying and pasting Field Name column values when maintaining parameters for Managed Gateway for Spend and Business Network configuration.
  2. Copy and paste only the variant, partition name, and Low column values to prevent errors.
