Symptom

When LMS admin user tries to connect to PRD, the following error is generated:

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

SAP SuccessFactors Learning

Cause

LMS admin user does not have the REPORT_DEVELOPER role assigned

Resolution

  1. Navigate to System Admin > Admin Management > Search for user
  2. Once within the user details, select the Assigned Roles tab
  3. Search for the REPORT_DEVELOPER role and add it to the user

 

  1. Now when the user attempts to logon to PRD again, access is granted.

Keywords

Unknown User, PRD, VJDBC, REPORT_DEVELOPER, could not access PRD
