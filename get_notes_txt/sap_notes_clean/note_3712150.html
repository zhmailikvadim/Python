Symptom

Logos and text are not displaying correctly when using form template with fragments.

Environment

SAP S/4HANA

Resolution

Logos and texts are determined together with master form template determination. In order to have expected master form template determined, please specify the rules in path SPRO -> Cross-application components -> Output Control -> Define Rules for determination of master form template in a sequential way, including relevant business data for determination. Inside each entries, logos and texts can be maintained. For more details, please refer to online help documentation.
