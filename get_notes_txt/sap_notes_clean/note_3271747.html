Reason and Prerequisites

Issue number 282753

Reason:

Due to a programming error in SAP HANA, a race condition may occur when accessing files if the log segment is encrypted. This, in turn, causes an indexserver crash with signal 6.

Affected Releases:

  • SAP HANA 2:
    • Revisions of SPS00 - SPS04
    • Revisions <= 059.06 (SPS05)
    • Revisions <= 066.00 (SPS06)

Prerequisites:

  • System Replication is active

AND

  • Log encryption is active i.e. ﻿global.ini [database_initial_encryption] log_encryption is set to true

Solution

Apply one of the following SAP HANA Revisions:

    • SAP HANA 2:
      • Revisions >= 059.07 (SPS05)
      • Revisions >= 067.00 (SPS06)
      • or higher

    • or higher
