Symptom

When you release data in SOLDOC, the 'Element Order changed' could not be released.

Other Terms

Order changes, Sequence changes, Release

Reason and Prerequisites

The order change release is not possible if the element is not more existing.

Solution

Please implement the current correction instruction with transaction SNOTE.
After that, the release of order changes could be done although there are inconsistencies.
It is also delivers the report SMUD_DELETE_INCON_ORDER which could indentify and delete the inconsistent order / sequence changes.

Other Terms

C_BOMMASSMAINTHEADER, C_BOMMASSMAINTITEM

Reason and Prerequisites

Program error.

Solution

Please implement this SAP note.
