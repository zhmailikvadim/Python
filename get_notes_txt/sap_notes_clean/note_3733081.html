Symptom

Error FCO_SFIN018 when trying to delete an additional quantity in Controlling → General Controlling → Additional Quantities → Define Additional Quantity Fields.

No change allowed: Field &1 is already filled in controlling area &2

Environment

  • Controlling and Profitability Analysis
  • SAP S/4HANA Finance
  • SAP S/4HANA

Reproducing the Issue

  1. Open SPRO and navigate to Controlling → General Controlling → Additional Quantities → Define Additional Quantity Fields.
  2. Select an additional quantity.
  3. Choose Delete.
  4. Save.

Cause

Posting data already exists for the additional quantity. For consistency reasons, the system does not allow deletion of an additional quantity once documents have been posted with it.

Resolution

  1. Ensure that SAP Note 3682197 is implemented.
  2. For data consistency reasons, it is not possible to delete additional quantity if related postings already exist. To resolve this, before attempting deletion, first clear the posted additional quantity records from ACDOCA using the appropriate Z-ABAP program.
