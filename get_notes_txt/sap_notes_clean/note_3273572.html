Symptom

Software Update Manager (SUM) uses HTTPS (Port 1129) while launching the SUM URL.

Environment

  • SAP Host Agent 7.22 PL52 or higher.
  • Software Update Manager (SUM)

Reproducing the Issue

  • Launch SUM tool using the command "SUMSTART confighostagent".
  • Launch the SUM URL using HTTP (Port: 1128) and it will be redirected to HTTPS (Port: 1129).

Cause

As of SAP Host Agent 7.22 PL 52, the secure hypertext transmission Protocol (HTTPS) is the default. A request on HTTP will be automatically redirected to HTTPS while launching the SUM tool in the browser. If the SSL configuration for Host Agent has not been performed, the URL launch in browser will fail.

Resolution

If the system is not configured or misconfigured with SSL for Host Agent, the URL launch will fail. Configure SSL for Host Agent in order to proceed with SUM launch.

The Host Agent provides self-signed certificates that can be accepted in the browser. For the details, refer to SAP Note 2662416.

NOTE: Auto upgrade feature of SAP Host Agent has to be considered. In between the upgrade, if the SUM URL is not working with HTTP, it could be because the Host Agent was upgraded automatically.
