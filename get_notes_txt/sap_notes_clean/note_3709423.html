Symptom

This note delivers changes to /NZT/IF_RTP4MTOBJ_PARAMS which are needed for SAP Note 3677755.

Other Terms

downtime minimization tool for SAP S/4HANA, NZDT

Reason and Prerequisites

Any of the following add-on support packages is installed in your system landscape:

  • NZDT 1.0 SP03

As a prerequisite to this note, note 3677413 - Oracle: configure log table storage parameter - prerequisite note must be implemented before this note is implemented.

Solution

In order for this note correction to be effective, the note must be implemented before:

  • Log tables are created

Implement the correction instructions in transaction SNOTE on the following systems of your system landscape:

  • Central system
