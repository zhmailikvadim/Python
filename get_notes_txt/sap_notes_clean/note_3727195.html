Symptom

  • A sales document line item is only partially delivered but is already set to completed.
  • A sales order item status is set to completed after the first reference, even if the full quantity is not consumed.

 

Environment

SAP S/4HANA Cloud Public edition

Reproducing the Issue

  • Open App: Create Sales Order - VA01.
  • Create a sales order with line item of item category CBTN for a quantity of 2 or more.
  • Open App: Create Outbound Delivery - With Order Reference (VL01N).
  • Create a delivery for the sales order with a quantity of 1.
  • Open App: Display Sales Order - VA03.
  • Check that the line item status in the sales order shows Completed.

Cause

The Item Category 'Item Type' is configured as 'A' (Value Item). By design, value items are completed after the first reference, either through delivery or invoicing.

Resolution

Please use another item category where the Item Type does not equal 'A' - Value Item.
