Symptom

In the MOSB process cluster, you cannot change the status for quotation data when the common access reference is empty because the data fails the validation check.
This issue is relevant for the market role: SUP.

Other Terms

Manage MOS Billing Quotations, Fiori APP, F8274, Quotation, Change Status, /US4G/I_MSBQT_HEADER, validate_head_fields_mandatory, common access reference

Reason and Prerequisites

Program error.

Solution

Implement the corrections of this SAP Note or import the relevant support package.
After implementing this SAP Note, the system allows you to change the status for quotation data even when the common access reference is empty.
You need to change the status again for the affected quotation data.
You can verify the fix by checking that the status change for quotation data is completed successfully.
