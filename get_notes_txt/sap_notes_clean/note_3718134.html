Symptom

This SAP Note extends the functionality added by SAP Note 3605745 - Tax jurisdiction code for export US <-> CA (which is intented for class-based accounting) to non class-based accounting scenarios (such as supplier/vendor billing document / customer expense invoice).

Other Terms

WLF_INVOICE_ITEM_CREATE,CALCULATE_TAX_ITEM, TXJCD

Reason and Prerequisites

 Note 3605745 - Tax jurisdiction code for export US <-> CA

Solution

 Implement the attached correction instructions.
