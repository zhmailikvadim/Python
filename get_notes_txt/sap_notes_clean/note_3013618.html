Symptom

You execute the database list of journal entries. From the output list, for a single line item, you navigate into the display of posting entry (document). This action terminates in error message GK 825 ("Document &1 does not exist in fiscal year &2").

Other Terms

Database list of journal entries, display of posting entry, CX56, CX56A, FINCS_DOC_LIST, FINCS_DOC_LIST_RL

Reason and Prerequisites

The line item from which you intend to navigate has been posted on a value for record type which is different from 0 ("actuals").

Solution

Import of Support Package or advanced correction by implementation of correction instruction

Functional Limitation

The "Display of Posting Entry" which you reach by navigation from the "Database List of Journal Entries" does not display the following fields of the consolidation application "SAP S/4HANA Finance for Group Reporting":

  • Additional characteristics
  • Key user extensibility fields

You have to launch the Fiori app "Display Group Journal Entries" for viewing those fields in a consolidation document of "SAP S/4HANA Finance for Group Reporting".
