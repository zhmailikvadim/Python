Symptom

During the SI contribution posting after mBGM creation (RPCGMBA0_FICO), the system does not always transfer the "Activating Cost Distribution" (from T52SWCODIST) to FI/CO, for example, if you use procedure 1 (Posting liabilities according to Organization Assignment infotype (0001)).

Other Terms

RPCGMBA0_FICO, activating cost distribution, T52SWCODIST

Reason and Prerequisites

Program error

Solution

The corrections are provided via HR support package (via CLC, if necessary). You can perform an advance correction by implementing the correction instruction using transaction SNOTE.

This solution ensures that the "Activating Cost Distribution" (from T52SWCODIST) is transferred to FI/CO. It specifies how liabilities are to be distributed during posting.
