Symptom

Performance issues observed during query execution on the SEVT table in the SBO-Common database. Thousands of records can be found in the SEVT table.

Environment

SAP Business One integration framework

Resolution

The table "SEVT" contains the log of all steps or changes made within the SAP Business One, that is, each time you update a record either a document definitions or master data, in this table creates a log of that change, but just who did it, when he did as object code or registration was made 

Content of the SEVT table can be truncated if you are not requiring to have the generated B1 events handled by the Integration Framework. 
We always recommend to create a backup On SBOCOMMON database before executing the deletion task.

In case you want some list of events to use with an Integration Framework you set filters in the Event Sender service.
