Symptom

The Pipeline Flow view displays opportunities even if their current Close Date falls outside the selected fiscal year or date range.

For example:

  • Fiscal Year selected: FY2026
  • Opportunity Close Date: outside FY2026
  • The opportunity is still visible in the Pipeline Flow list and chart

This situation may give the impression that the Close Date filter is not working correctly.

Environment

  • SAP Sales Cloud And Service Cloud Version 2
  • SAP Sales Cloud Version 2

Reproducing the Issue

1. Navigate to Pipeline Flow.  
2. Select a Fiscal Year (e.g., FY2026).  
3. Observe the list of opportunities displayed.  
4. Identify opportunities where the current Close Date is outside the selected fiscal period.  
5. Confirm that these opportunities are still included in the results.  

Cause

This behavior is by design.

The Pipeline Flow is a historical analysis tool, not a real-time filter based on the current Close Date.

The system determines which opportunities to include based on:

  • The state of the opportunity at the Start Date of the selected period, OR
  • The initial (first) state if the opportunity was created within the selected timeframe

If the opportunity had a Close Date within the selected fiscal period at that time, it is included in the Pipeline Flow — even if the Close Date was later changed to a value outside the range.

Additionally:

  • The Opportunity list shows current values
  • The Pipeline Flow logic uses historical values

This difference leads to the perceived inconsistency.

Resolution

  1. This is the current and expected system behavior.  
  2. If you wish to view opportunities strictly based on their current close date, then:
    • Use Opportunity List views.  
    • Use Reports/ Analytics.
