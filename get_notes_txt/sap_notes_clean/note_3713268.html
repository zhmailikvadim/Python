Symptom

The function "automatic generation of packaging items for Goods Issue" is set up. However,  no packaging items are created when posting GI for an ODO, there is a warning message "No control parameter entries found for BSKEY &1 and document type &2, the message no. is /SCWM/ERPINTEGRATION111.

Environment

  • SAP Extended Warehouse Management
  • SAP S/4HANA

Reproducing the Issue

  1. Run tcode /SCWM/PRDO to display the ODO, the ODO is picked with an HU.
  2. Post GI for the ODO, no packaging item is created automatically, there is a warning message "No control parameter entries found for BSKEY &1 and document type &2".

Cause

missing of customizing.

Resolution

  1. Maintain an entry for the document type of the ODO in the following customizing path:
    => SCM Extended Warehouse Management
    => Extended Warehouse Management
    => Interfaces
    => ERP Integration
    => Delivery Processing
    => Control Message Processing Dependent on Recipient
  2. Create a new ODO to test the result.

Keywords

/SCWM/ERPINTEGRATION111, automatic generation of packaging items for Goods Issue.
