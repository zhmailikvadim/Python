Symptom

  • An inconsistency occurs in the "Start Date" and "Due Date" fields within the performance management form's goal section goal detail.
  • The dates displayed in the performance form are one day off compared to the dates set in the goal.

Environment

SAP SuccessFactors Performance Management

Reproducing the Issue

  1. Log in to the system and proxy as the user.
  2. Navigate to the Performance tab and access the performance form.
  3. Open the development section goal detail within the performance management form.
  4. Observe that the "Start Date" and "Goal Due Date" fields are displayed as one day off compared to the dates set in the goal.

Cause

This is a Known Issue and has been caused by Time Zone conversion.

Resolution

The fix for this issue is planned to be deployed on 1H 2026 release (b2605).

For release timelines, please review SAP SuccessFactors Product Release & Road Map Information

Workaround

No workaround is available.
