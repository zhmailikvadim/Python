Environment

  • SAP Integration Suite
  • SAP Business Technology Platform
  • Cloud Integration

Cause

The OData API responds with an error due to unacceptable content characteristics based on the Accept Header sent in the request. Specifically, the parameter "odata.metadata=full" is not supported by the OData V4 framework, resulting in the HTTP Status 406 (Not Acceptable) error.

Resolution

Replace "odata.metadata=full" with "odata.metadata=minimal" in the Accept Header of the OData V4 adapter request.
