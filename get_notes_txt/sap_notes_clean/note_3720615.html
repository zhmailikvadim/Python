Symptom

  • Require certificate text in the alert when it is generated 

Environment

SAP Cloud ALM - Health Monitoring

Reproducing the Issue

  1. Logon to SAP Cloud ALM
  2. Navigate to Operations => Health Monitoring
  3. Select Alerts on Left hand Tab
  4. Observe Alerts generated for Certificates do not display certifcates text name

Cause

The threshold type by default is "Numeric", whereas to display certificate name it must be type "Text"

Resolution

  1. Logon to SAP Cloud ALM
  2. Navigate to Operations => Health Monitoring app
  3. Ensure the On-Premise system is selected in the "Scope" area
  4. Ensure Metric threshold values are specified
    1. Click the "Configuration" icon in the top right of the page
    2. Select the relevant On-Premise system from the list
    3. Click the "Metrics" tab in the pop-up
    4. Select the Certificates metric from the list to open its configuration dialog
    5. Specify threshold type as "Text" and add the text of the certificate in the text field **  make sure default is not maintained as critical, if so, all certificates will rate RED ** 
    6. Save the metric threshold configuration.
