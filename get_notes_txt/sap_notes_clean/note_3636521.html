Symptom

What's the default format of the Offer Letter PDF footer (Page Numbers)?

Environment

SAP SuccessFactors Recruiting Management

Reproducing the Issue

  1. Create a PDF Offer Letter in SAP SuccessFactors Recruiting.
  2. Observe the font type and font size in the printed PDF.

Resolution

The default format for the Offer Letter PDF footer (Page Numbers) is as follows.

  • Font Size: 12 pt (13.8 pt after conversion)
  • Font: Arial Unicode MS

The PDF is generated at a 115% scale. Therefore, the actual font size is calculated as: {Actual Font Size} = {Original Font Size} × 1.15.

Note that both the font size and font type of the footer cannot be changed. To request an improvement for this feature, it is recommended to submit it to the Customer Influence page for SuccessFactors Recruiting. This platform enables users to vote for each other's ideas, post comments, and receive direct feedback from the Product Management team. Refer to the KBA 2090228 for more information.
