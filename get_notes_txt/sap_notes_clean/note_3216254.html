Symptom

When uploading zip file by Hot Folder, the following exception is thrown:

Failed to apply Zip transformation.; nested exception is org.zeroturnaround.zip.ZipException: Failed to process zip entry '__MACOSX/._electronics 2 with action de.hybris.platform.cloud.hotfolder.spring.integration.zip.CloudUnZipTransformer$$Lambda$1641/0x0000000801c23440@29abcb4f

Environment

SAP Commerce Cloud

Reproducing the Issue

Compressing folder on Mac OS system and upload the zip file by Hot Folder

Cause

When compressing folder, it will contain a folder named "__MACOSX"

Resolution

Running the command as below

zip -d filename.zip __MACOSX/\*

Keywords

hot folder, mac os, folder
