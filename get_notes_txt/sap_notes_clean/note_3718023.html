Symptom

  • Clarification on SAP’s policy regarding downgrading or SAP Bundle Patches (SBP).

Environment

Oracle database (All supported versions)

Cause

Non‑Reversible Oracle Patch Level

Resolution

  • Downgrading Oracle from a higher version or higher patch set to a lower “base release” provides no technical advantage and is not recommended
  • If a version change is necessary, upgrade to the latest certified Major version Release.
  • SAP does not support downgrading Oracle from a version higher to lower due to the cumulative nature of Release Updates and SAP Bundle Patches (SBPs).
  • Reverting to an earlier version is only possible through a fresh install
  • SAP Bundle Patch Deinstallation
    • SAP Bundle Patch does not support deinstallation. To revert to a previous, older SAP Bundle Patch, follow the installation instructions of that older SAP Bundle Patch. Do not, however, revert the OPatch and MOPatch utilities to older versions during that process. Leave them at the versions that were installed with this SAP Bundle Patch.
