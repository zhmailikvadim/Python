Symptom

This SAP Note provides you with the relevant system changes to support the calculation of the first-time use of the multi-use packaging for the Netherlands EPR Packaging Waste report.

Other Terms

SAP Responsible Design and Production, RDP, Netherlands, EPR Declaration, NL EPR, Netherlands EPR Packaging Waste

Reason and Prerequisites

This SAP Note is relevant only for the Netherlands EPR Packaging Waste report.

Solution

With this release, the Netherlands EPR Packaging Waste report supports the calculation of the first-time use of the multi-use packaging report fraction.
