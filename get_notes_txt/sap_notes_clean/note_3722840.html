Symptom

Activation of SAP Fiori Apps by Business Role with Task List SAP_FIORI_CONTENT_ACTIVATION fails during the activation process. 

Error message: Service: <xxxx> 1 failed.  For detailed analysis activate failed service manually with transaction /IWFND/MAINT_SERVICE 

Follow the message to activate OData Services by /IWFND/MAINT_SERVICE and received error message:

Request <TR name> does not have target
Message No. S_LMCFG_CORE_TASKS000

Environment

SAP S/4HANA

Reproducing the Issue

  1. Call STC01.
  2. Run Task List SAP_FIORI_CONTENT_ACTIVATION.
  3. Select Business Roles to run
  4. Activate OData Services by /IWFND/MAINT_SERVICE

    Cause

    The transport layer has not been correctly setup.

    Resolution

    Follow the attached guide "Activation of OData Services in Prod Systems with task lists.pdf" in SAP Note 2886433 to maintain the transport layer to resolve the issue.

    SAP Note 2886433 - Fiori Setup: Activation of OData Services in Prod Systems with task lists
