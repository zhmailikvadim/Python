Does CVE-2025-64775 affect SAP Information Steward?

Environment

  • SAP Information Steward 4.3 SP03 Patch 8

Resolution

The product is not exposed to this security issue, because the SAP Information Steward 4.3 SP03 Patch 8 and 2025 uses the struts version which is not in the affected version range in the CVE.
