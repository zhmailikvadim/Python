Symptom

The value for field “Personnel Number” is displayed in app Display Line Items in General Ledger, but the description for the field is not displayed.

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental."

Environment

SAP S/4HANA Cloud Public Edition

Resolution

Key user can enable access to the unmasked full name attribute using data source extension. To get a general overview about how to use data source extensions for this application, see App Extensibility: Display Line Items in General Ledger (Key User). For the data source extension, full name attribute from the Person Work Agreement Details (_PersonWorkAgreement_1) association can be selected. Also, a custom label can be defined, such as Full Name Unmasked, to distinguish it from the original attribute. After saving and publishing the data source extension, the full name information is available again in the application.
