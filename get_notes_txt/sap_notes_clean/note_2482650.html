Symptom

You enter your electronic customs import declaration and use a tariff number for which authority codes must be declared, too. If you change the specifications on the authority code without making any other changes in a customs declaration, no changes are saved.

Other Terms

Customs declaration, OGA, PGA, NCLO, authority codes, /SAPSLL/CORCTZ, /SAPSLL/CL_CUHD, PREPARE_OBJECT

Reason and Prerequisites

Program error

Solution

Implement the correction instructions for your release.
