Symptom

When checking the flag “Turn off Log Update” on the Configuration screens (on both Mass and individual maintenance), the Turn Off Log Update (fieldname TOFLG) cannot be maintained. Upon clicking the SAVE button, this field value is not being saved to DB table.

The workaround is to maintain this flag at DB level manually for Table /IRM/TIPART. Even after setting this flag value = X at DB level, this flag is shown as ‘unchecked’ on configuration screens.

Other Terms

TOFLG, Config - Turn Off, Agreement request

Reason and Prerequisites

This is caused by program error.

SAP Note# 3470997 available for ECC release.

Solution

Please implement the correction instructions. These corrections are only valid if you have applied the following Support Pack(s):

Release V/4 – 1910 Base Package, 2020 Base Package, 2020 V2, 2021, 2022, 2022 V2 and 2023.

Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object
