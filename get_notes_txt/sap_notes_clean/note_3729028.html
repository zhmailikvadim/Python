Symptom

  • In the Test Your Processes app of SAP S/4HANA Cloud Public Edition (Q/T tenant), the email ID is not available for selection in the User Name or Email ID field during test user maintenance.
  • Only user names appear in the value help, which restricts selection using email ID. 

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Open the Test Your Processes app.
  2. In the Maintain Roles pane, click + Add User.
  3. Click on the User Name or Email ID field.
  4. Open Value Help (Select: Business Users popup).
  5. Observe that only user names are available for selection.

Resolution

This behavior occurs because the setting to use email ID as the preferred credential is disabled by default.

During Test user selection:

  1. Open the Value Help (Select: Business Users popup)

  2. Locate the option “Set Email ID as preferred credential for user maintenance”

  3. Change the toggle button from NO to YES

  4. Perform the test user search and selection again

Once the toggle is set to YES, the email ID becomes available and can be selected for user maintenance.
