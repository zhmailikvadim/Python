Symptom

'Unknown' value listed as client session (in case of CR for Enterprise) in CMC

STEPS TO REPRODUCE

  • Login to CMC
  • Connect to the same CMS via CR for Enterprise
  • Navigate to CMC->Sessions
  • 'Unknown' is listed as value under client sessions.

Reason and Prerequisites

The process to determine the session invloves comparing the client action CUID against the SI_REFERNCE_CUID of a set of predefined audit applicaitons. The CR for Enterprise is not included in this set. Hence the unknown session value.

Solution

The fix compares the client session with the CR info Object to determine the sessoin value of CR.
