Symptom

You observe that most of the analysis result fields are blank when you uploaded the resulted zip file in Hana Cloud Measurement Analysis Tool.

Environment

  • SAP HANA, platform edition 1.0
  • SAP HANA, platform edition 2.0

Resolution

The most of the HCMT tests do not have a KPI, so there are no green or red traffic lights. This is as expected behavior. Only HCMT tests that are relevant for certified IaaS solutions are evaluated. All other tests are basically for information only, or might be interesting for special purposes - there is no KPI defined.
