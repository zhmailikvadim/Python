Symptom

When accessing the Payroll Control Center (PCC), you may encounter the following error message:
"Sorry, there is no data for you. Possible reasons are as follows: (1) No payroll systems have been assigned to you. (2) You have no permission for SAP System Configuration. Please contact your system administrator."

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

  • Human Capital Management (HCM)
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP SuccessFactors Employee Central Payroll
  • SAP S/4HANA

Reproducing the Issue

  • Navigate to Payroll from SuccessFactors Employee Central.
  • Observe that the system issues the above error message.

Cause

The error occurs due to the missing system name for certain users in the payroll system assignment.

Resolution

To resolve this issue, follow these steps:

  1. Access Admin Center->Manage Data->Payroll System Assignment.
  2. Press 'Take Action'->Make correction
  3. Add System Name
