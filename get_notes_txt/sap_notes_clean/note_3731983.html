Symptom

Symptom

The bgRFC monitor (transaction SBGRFCMON) does not display the transaction ID for bgRFC units. There is no way to correlate bgRFC units with their originating transaction.

Other Terms

SBGRFCMON, TRANSACTION_ID, bgRFC, unit monitoring, QRFC_I_UNIT, QRFC_O_UNIT, TRFC_I_UNIT, TRFC_O_UNIT

Reason and Prerequisites

The transaction ID is not stored in the bgRFC unit tables. This makes it difficult to trace and diagnose bgRFC processing issues.

Solution

Apply the note or upgrade to the relevant Support Package version.

If applying the note, ensure that the pre-implementation manual activity steps are performed.

When you run the MX_VAT_REP report, partial payments are not considered in the calculation.

Other Terms

DRC, Mexico, VAT Reporting, VAT Declaration, MX_VAT_REP

Reason and Prerequisites

This SAP Note is relevant only for Mexico.

To implement Mexico compliance reports, you need to install the Document and Reporting Compliance (DRC) framework. For more information, see Document and Reporting Compliance.

You have installed the Mexico compliance reports for Mexico. For more information, see SAP Notes 2548443 or 2598781.

You have applied SAP Note 3731346.

Solution

After implementing this SAP Note, the Mexico VAT Declaration (MX_VAT_REP) report considers partial payments in the calculation. As a result, the calculation is correctly reported in the report output.

As a general rule, SAP recommends that you install a solution by applying a Support Package. However, if you need to install a solution earlier, use the Note Assistant to implement the correction instruction. You can find more information about the SAP Note Assistant in the SAP Support Portal, under SAP Note Assistant.

You can try to activate the objects contained in this SAP Note executing the reports below:

  • For DDLS objects, execute the RUTDDLSACTSNOTE report according to the steps available in SAP Note 2607137.
  • For DCLS objects, execute the R_ACMTOOL_DELETE_INACTIVE_V report according to the steps available in SAP Note 2691812.
