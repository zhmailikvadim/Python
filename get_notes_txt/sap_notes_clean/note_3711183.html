Symptom

RFQ with attachment is failing with below error in Managed gateway for spend management and SAP Business Network.

Transimission failed with the error java.lang.Exception: java.lang.Exception: SOAP Fault :Exception occured when processing the request for event: S4HanaRequestForQuotation, with the following message: javax.xml.stream.XMLStreamException: Inbound S4HanaRequestForQuotation message: Operation: S4Hana Type: Service Message: Exception while reading the mime message @ line 31 in SoapResponseParse.groovy while posting to customer ANXXXXXXXXXXX-T - RealmName: target system with HTTP ResponseCode: .. Suggested Action : Please contact support with ErrorCode:CIG-PLT-04641. 

Environment

S4HANA Private Cloud.

Cause

Missing " Create Fallback Logical Port for Outbound Services " configuration in S4HANA Private Cloud.

Resolution

  As per the 4BL S4HANA Private Cloud Guide, Perform the below steps in order to resolve this issue:


  1. Navigate to section 3.7.6 in the guide: Create Fallback Logical Port for Outbound Services.


  2. Navigate to step 12: In the Transport Binding section, ensure MTOM is selected as the Optimized XML Transfer, Choose Next.

Keywords

CIG, RFQ, Attachment, 4BL, Managed Gateway
