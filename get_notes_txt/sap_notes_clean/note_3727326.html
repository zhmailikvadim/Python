Symptom

  • Not able to add a legal entity for multiple benefit enrollment processes.
  • Error message: "These legal entities xxx have already been assigned. Please select different legal entities."

Environment

  • Global Benefits

Reproducing the Issue

  1. Navigate to Manage Data > Configure Benefit Enrollment.
  2. Select ( Configure Benefit Enrollment Process Name )
  3. Click "Take Action" > "Make Correction".
  4. Attempt to add Legal Entity XXXX. Save.
  5. Observe the error message: "These legal entities (XXXX) have already been assigned. Please select different legal entities."

Cause

  • System limitation.

Resolution

Verify that the Legal Entity being added already exists in another Configure Benefit Enrollment Process.

Engineering is working on it and it is being tracked via Jira Ben-20866 target will be 1H2026.

For more information about updates from subscribed KBAs, you can refer to the KBA 2171560 - How to mark a KBA or SAP Note as a favorite and get notified when it is updated - SAP for me
