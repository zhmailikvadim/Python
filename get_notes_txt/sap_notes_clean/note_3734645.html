Symptom

There is no output item tab in Manage Outgoing Check.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Open the app Manage Outgoing Checks.
  2. Select an outgoing check document.
  3. Observe that the Output Items tab is not displayed.

Cause

The check was in void status, and the Output Items tab is not displayed for checks in void status.

Resolution

  1. In Manage Outgoing Checks, verify the status of the check. If the status is Void, note that the Output Items tab will not appear by design.
  2. To review the existing output document for a voided check, open the app Manage Output Items and search for the corresponding outgoing check; open the generated PDF/output there.
  3. If you need to access the Output Items tab within Manage Outgoing Checks, select or create a check that is in New or Cashed status (for example, a check created by a payment process) and open it again.
  4. If the check was created manually and shows status "Manually Written", printing from Manage Outgoing Checks is not supported; use the manual process or refer to the note in the See Also section.

See Also

Refer to: You cannot print the manually created check - Message no. FS675
