Symptom

  • A service contract is created with a yearly based billing plan with value 120 USD.
  • The following invoice correctly reflects with the yearly value 120 USD.
  • The credit memo request created with referenced to preceding invoice only displays 10 USD, which corresponds to a single month instead of full year amount in condition tab.

Environment

•Sales And Distribution (SD)
•Logistics Execution (LE)
•SAP R/3
•SAP R/3 Enterprise 4.7
•SAP ERP Central Component
•SAP ERP
•SAP Enhancement package for SAP ERP
•SAP Enhancement package for SAP ERP, version for SAP HANA
•SAP S/4HANA

Resolution

During credit memo request creation process, the conversion factor KOMP-ANZ_MONATE is set to 1 by default as system cannot know how which factor to be taken as no billing plan info available. If it is expected to have other conversion factor, please consider modifying the logic inside USEREXIT_PRICING_PREPARE_TKOMP (include MV45AFZZ) to adapt TKOMP-ANZ_WOCHEN, ensuring the correct factor is applied during credit memo request creation.
