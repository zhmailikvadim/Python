Symptom

New deduction-free amounts of allowances are valid from March 1, 2026.

These amounts are used for limitation of deductions from allowances in the payroll function PLDED.

There are 3 payroll constants in table T511K which require this update:

  • SILL3 - minimal pension,
  • SILL4 - allowance protection from alimony execution,
  • SILL5 - allowance protection from other executions.

These 3 amounts were announced by ZUS at https://monitorpolski.gov.pl/MP/2026/220.

The new values are:

  • SILL3 = 1978.49,
  • SILL4 =   849.42,
  • SILL5 = 1401.57.

The first payroll that requires the new values is the regular payroll for March 2026.

Other Terms

T511K, SILL3, SILL4, SILL5, PLDED, limitation of deductions from allowances.

Reason and Prerequisites

Legal change.

Solution

The new values for the constants have been set.

Please import the Support Package that includes this SAP Note or follow the instructions under Manual Activity.
