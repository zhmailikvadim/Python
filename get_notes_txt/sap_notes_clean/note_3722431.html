Symptom

An error occurs when attempting to unsubscribe from SAP Integration Suite. 

"Unsubscription Failed 
Couldn't unsubscribe from Integration Suite. Please try again. If the problem persists, open a support case using the component: LOD-HCI-PI."

“Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.“

Environment

  • SAP Integration Suite 

Reproducing the Issue

  1. Open SAP BTP Cockpit. 
  2. Navigate to Subaccount → Instances and Subscriptions. 
  3. Select Integration Suite. 
  4. Click "Unsubscribe". 
  5. System returns error: "Couldn't unsubscribe from Integration Suite. Please try again. If the problem persists, open a support case using the component: LOD-HCI-PI."

Cause

The cause for the error to occur could be that Formations exist in the Global account. 

Resolution

Before you unsubscribe from SAP Integration Suite service:

  1. Delete the Process Integration Runtime service instances that you created while activating Cloud Integration capability.
  2. Delete your tenant from Formations. From your global account, navigate to System Landscape → Formations →  <your-tenant> →  Actions → Delete.

If the issue continues after performing the above steps, please open a case with the LOD-HCI-PI-PRV team and provide the following details:

  • Latest timestamp of the unsubscription attempt
  • Integration Suite URL
  • Subaccount subdomain
  • Subaccount ID

This will help the team investigate the issue further.
