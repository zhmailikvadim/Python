Symptom

This SAP note does not fix any application issues. It is created to down port the extension of  CIF RFC/QRFC structures to lower S/4HANA, ECC and APO releases. This
structure consistency is a prerequisite when any other extension of the structures will happen in future.

Other Terms

CIF, RFC structures, QRFC structures, TECHS, Parameter Effectivity, DSC PP/DS

Reason and Prerequisites

The end-to-end enablement of Parameter Effectivity transfer via CIF was introduced with SAP S/4HANA 2023 FPS3 for the following object types:

  • Project Networks
  • Planned Independent Requirements 
  • Sales Order
  • PP/DS and Core Planned and Production Orders
  • Purchase Requisition and Purchase Order

However, the corresponding CIF structures used for RFC/QRFC communication in lower SAP S/4HANA, ECC and APO releases were not extended to include the Parameter Effectivity field (TECHS).

Solution

Install this SAP Note to extend the affected CIF RFC/QRFC structures in the lower S/4HANA, ECC and APO releases with the Parameter Effectivity field TECHS.
This correction ensures a consistency of the structures at the different S/4HANA releases.
