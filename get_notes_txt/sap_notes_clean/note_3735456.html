Symptom

In transaction SPROXY_START, we have set the constraint for a data element, e.g. 
Min. (incl.)     1      
Max. (incl.)    10

however when we input 0 for this data element, the value is accepted without any error reported.

Environment

SAP ECC 6.0

Cause

The constraint does not included in the transformation of the data element.

Open the data type via transaction SPROXY_START, choose Menu Utilities -> Settings -> mark option "Show Transformations in Proxy Editor" -> choose OK button, then we will find Transformation button shown in the External tab, click on it, we could find the corresponding transformation ID.
In transaction STRANS -> Display, open this transformation, the constraint does not exist.

Resolution

Open the data type transaction SPROXY_START, choose Test, mark "Enforce stylesheet generation" and choose Execute to regenerate the transformation.
