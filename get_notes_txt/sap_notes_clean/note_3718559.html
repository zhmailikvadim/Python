Symptom

  • When sending emails to multiple only one recipient receives emails, even when multiple recipients are added manually or via the "reply to all" button.

Environment

SAP Service Cloud Version 2

Reproducing the Issue

  1. Test the email functionality in the ticketing system.
  2. Add multiple recipients manually to an email or use the "reply to all" button.
  3. Observe that only one recipient receives the email.

Cause

  • The issue is caused by the enabled feature that redirects all emails to one address, preventing multiple recipients from receiving emails.

Resolution

  1. Go to Settings.
  2. MTA Configuration.
  3. Remove the address maintained in Outbound Email Address to Redirect.
