Symptom

Is the Validate and Enrich Receipt Data in an External System feature supported through the Cloud Integration Gateway (CIG)?

Resolution

The Validate and Enrich Receipt Data in an External System feature is not supported via Cloud Integration Gateway (CIG). To enable this functionality, the required web service integration task Process Receipt Externally must be activated using middleware as the endpoint. Middleware is required to establish the connection between the web service and the ERP system.

See Also

Please refer Validate and Enrich Receipt Data in an External System for more details
