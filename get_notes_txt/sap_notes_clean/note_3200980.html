Symptom

When we try to create a Web intelligence report on top of Bex then gets the below error.

“Unexpected exception while calling the SL Service"

Other Terms

 Webi, Bex, OLAP, BICS

Reason and Prerequisites

Environment 

  • BO Version : 4.3 Support Pack 1 Patch 5
  • Web Intelligence

Steps to Reproduce 

  1. Open BI launchpad->Webi.
  2. Select SAP BW HANA as data source.
  3. Select the BW connection related to it.
  4. Put the query in search box and select it.
  5. Click on OK.
  6. The error message mentioned above appears
