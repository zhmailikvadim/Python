Symptom

  • In the user interface, the Average Evaluation Score displays correctly on the worker, confirming that the evaluation exists. However, this value does not display in the report output.
  • The Average Evaluation Score appears as 0.00 for all workers on the report.
  • The report successfully pulls other related fields such as Evaluation Date and Evaluators, yet the Average Evaluation Score remains 0.00 for all workers.

Reproducing the Issue

  1. Open the worker report. Verify that Evaluation Date and Average Evaluation Score appear as fields on the report.
  2. Run the report.
  3. Observe the results. Note that Evaluation Date populates correctly, but Average Evaluation Score displays 0.00.
  4. Cross-check in the user interface. View the same worker’s record in the UI and confirm that the Average Evaluation Score is visible and shows a nonzero value.

Resolution

  1. Edit the worker report.
  2. Replace the Average Evaluation Score field with the field Average Score, located under Worker → Details.
  3. Run the report.
  4. Confirm that the correct score is displayed when using this field.

Keywords
