Symptom

When the method IF_HRPA_INFOTYPE_3672~GET_VARKY of the decoupling class CL_HRPA_INFOTYPE_3672_01 is run, a graphical notification is triggered if a notification of an occupational illness is to be created after December 31, 2025. This is factually correct according to DM Org and DB Org. However, technically, the last delivered implementation can lead to problems in certain scenarios, for example, if Processing and Forms is used.

This SAP Note solves the problem by refactoring the code appropriately.

Solution
