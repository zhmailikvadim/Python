Symptom

The logic for determining the FTZ ID is not harmonized between print data retrieval and data retrieval for electronic messages.

After you have implemented this SAP Note, the logic is harmonized for:

  • the form CBPF214
  • the form CBPF7501
  • the entry summary electronic message that can be sent from a weekly entry
  • the electronic export declaration that can be sent from a customs export declaration

Reason and Prerequisites

Program error

Solution

Implement the correction instructions.
