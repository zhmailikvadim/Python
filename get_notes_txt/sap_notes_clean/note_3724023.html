Symptom

In the process "Start of Supply for Gas (DSO View)" (DE_B_GELIGAS_SOS_DSO), step 8940 triggers an outdated version (2310) of the linked Start of Billing Settlement (SBS) process instead of the current version (2604).

This issue is relevant for the market role: DSO.

Other Terms

GPKE, Start of Supply for Gas, SoS, SBS, Start of Billing Settlement, linked process, process version, 2604, 2310, DE_B_GELIGAS_SOS_DSO

Reason and Prerequisites

The error occurred because the linked process trigger was configured with an incorrect version number.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the system triggers the current version (2604) of the SBS gas process when initiated from step 8940 of the Start of Supply for Gas process.

To implement the fix, please go to transaction SCPR20 and activate the BC set /US4G/3_0_DSO_SP08_NOTE_3724023.

You need to retest the scenario. You can verify the fix by checking that when the Start of Supply for Gas process (version 2604) triggers the linked process, the SBS process version 2604 is initiated.
