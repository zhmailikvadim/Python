Symptom

You have activated the attribute assignment of postings to WBS elements with CO-PA characteristics in transaction KEATTR, and notice that the system did not assign the attributes of the underlying sales order item.

Other Terms

KEATTR, WBS element, KDPOS

Reason and Prerequisites

  1. The WBS element to which postings have been made does not have a settlement rule. 
  2. The posted WBS element is used as an account assignment for exactly one sales order, but in several items

Solution

Please use SNOTE to implement the source code corrections delivered with this SAP Note.
