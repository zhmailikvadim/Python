Symptom

  • Web Intelligence schedules are stuck in Running state. These jobs stuck in running state cause other jobs to stuck in pending.
  • Restarting Web Intelligence Processing Server resolves the issue and to free-up the queue.
  • While refreshing Web Intelligence document there is no issue observed.  

Other Terms

webi jobs, webi stuck, jobs stuck, web intelligence processing server jobs.

Reason and Prerequisites

Environment

  • SAP BusinessObjects Business Intelligence 4.3 Support Package 04 Patch 600.
  • Web Intelligence
  • SAP BW BAPI 
  • SAP HANA

Reproducing the Issue

  1. Choose any WebI report conencted to SAP BW BAPI or SAP BW on HANA.
  2. Schedule the report (scheduling method, time, format and recurrence are all irrelevant).
  3. Observe that the scheduled job remains in Running/Pending status and never runs or fails.
