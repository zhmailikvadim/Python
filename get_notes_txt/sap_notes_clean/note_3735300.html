Symptom

The download integration fails with Error: 'SQL Error. Probably due to SQL time out or any SQL exception. Please Contact SAP Fieldglass for more information.'

Cause

Duplicate custom field names were included in a combined custom field across multiple modules, which generated malformed SQL during the connector download

Resolution

  1. Identify the combined custom field that aggregates custom fields from multiple modules and verify if any individual custom fields share the same name.
  2. Delete the combined custom field.
  3. Delete the individual custom fields involved that have non-unique (duplicate) names across modules.
  4. Recreate the individual custom fields, ensuring each field name is unique across modules (use module‑prefixed or otherwise distinct names).
  5. Recreate the combined custom field using the newly created, uniquely named individual fields.
  6. Run the download connector again to confirm the SQL error is resolved.

Keywords

sql error, connector download, combined custom field, duplicate field names, unique field names, download failure, sql exception
