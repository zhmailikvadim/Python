Symptom

When you run the Certificate for LI/NHI payment (HTWLCE00) report and select the Group by Company Code field, the system groups data at the company code level.

In this case, the fields Unified ID and Withholder, which are at the enterprise level, should be removed during grouping.

This SAP Note provides the following enhancements when you select the Group by Company Code field:

  • In the ALV output, the system removes the Unified ID field and replaces it with the Company Code field.
  • In the LI/NHI forms, the system removes the Unit ID and Withholder fields.

Other Terms

PLCE0TW1, PLCE0TW2, Unit ID, Withholder

Reason and Prerequisites

This is an enhancement.

Solution

Do either of the following:

  • Apply the relevant Support Package.
  • Carry out the attached ManualSteps3712226.pdf and then implement the corresponding correction instructions.
