Symptom

System monitor does not show the full disk size as configured in tenant configuration.

Environment

SAP HANA Cloud Services 

SAP Datasphere

Reproducing the Issue

  1. Open the system monitor and review the storage overview; observe that the displayed capacity is smaller than in tenant configuration
  2. Run the SQL statement: SELECT * FROM "SYS"."M_DISKS"; not the total size is shown either

Cause

This is SAP hana cloud issue

Resolution

The solution of this issue is currently under investigation. Please follow this KBA for updates.
