Symptom

  • Error occurs when attempting to add an HTML mashup to the mashup component in an EC file.
  • A dump is generated during the process of editing the HTML mashup.
  • Error message in UI Designer: The selected Mashup Component cannot be an Inscreen Mashup.

"Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental." 

Environment

SAP Cloud for Customer

Resolution

Check the mashup settings in UI Designer and verify whether the ViewStyle parameter is set to inScreen. If so, change it. For the mashup to be added to an EC component in UI Designer, it should not be Inscreen.

Keywords

html mashup, mashup component, viewstyle, ui designer, inScreen
