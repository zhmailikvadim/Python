Symptom

  • How to set data label inside chart bar in SAP Analytics Cloud

Environment

  • SAP Analytics Cloud(SAC) 2026

Reproducing the Issue

  1. Login SAC.
  2. Create a story.
  3. Add a chart with Bar/Column chart type.
  4. Notice that data labels are outside of bars.

Cause

  • The symptom is by design.

Resolution

There is no method to make data label show inside of bar.
But there is working around that change the Bar/Column chart type to Stacked Bar/Column chart type.
Then it is possible to show data label in bar.

on, in, embed
