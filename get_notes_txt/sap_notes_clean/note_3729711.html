Symptom

 In the process "Handling of MDX Inbound" (DE_B_CP_EMBED_MDX_IN), step 8999 runs into an error when the process is embedded inside another embedded process.

Other Terms

 US4G, MDX inbound handling, DE_B_CP_EMBED_MDX_IN, US4G/CL_PRST_EMB_MDX_IN_LAST, ENHANCE_STEP_DATA, embedded process

Reason and Prerequisites

 The error occurred because of a code issue in step 8999.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, You need to reverse the erroneous step 8999 and run the process again. You can verify the fix by checking that step 8999 completes successfully.
