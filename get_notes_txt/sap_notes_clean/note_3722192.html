Symptom

When users do not have a profile picture and have a special character in their first name, feedback request comments show the incorrect initials. For example, the card displays the correct initials, but the comments show the incorrect initials:

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

SAP SuccessFactors Continuous Performance Management

Reproducing the Issue

  1. Log into SuccessFactors
  2. In the module picker, select Continuous Feedback
  3. Select any tab
  4. The feedback comments display the incorrect initials

Cause

This is a known issue.

Resolution

The fix for this issue is planned to be deployed in the 1H 2026 release.

For release timelines, please review SAP SuccessFactors Product Release & Road Map Information.

Workaround

No workaround is available.
