Symptom

The ABAP runtime error RAISE_SHORTDUMP with the exception CX_SADL_DUMP_DATABASE_FAILURE occurs when displaying data in the Maintain Agreement-Specific Product Properties app.

Other Terms

F8598, Maintain Agreement-Specific Product Properties, CX_SADL_DUMP_DATABASE_FAILURE, TRIGGER_DATABASE_FAILURE_DUMP, CL_SADL_SHORTDUMP=============CM009, CL_SADL_SHORTDUMP=============CP

Reason and Prerequisites

The dump occurs because the data element /SAPSLL/BOM_COMP_COUNT is of data type INT1, which causes a numeric overflow and results in a system dump.

Solution

Implement the correction instructions included in this SAP Note via transaction SNOTE or apply the corresponding Support Package.

This will modify the data type of the data element /SAPSLL/BOM_COMP_COUNT to INT4.
