Symptom

  • When replicating employee master data, the default configuration assigns the reporting line unit (RLU) manager from the organisational structure as the expense approver. However, some customers need to use their own custom logic to determine the desired expense approver which may not necessarily be the RLU manager.

Image/data in this KBA is from SAP internal systems, sample data, or demo systems. Any resemblance to real data is purely coincidental.

Environment

Concur Expense, SAP ERP, SAP S/4HANA, SAP ERP integration with SAP Concur solutions, SAP S/4HANA integration with SAP Concur solutions

Resolution

The attached best practice provides a guideline for customers to implement their own logic for expense approver determination.
