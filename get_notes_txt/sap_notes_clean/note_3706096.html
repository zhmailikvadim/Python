Symptom

User is unable to find the list of placeholders that can be utilized in Email Templates.

Environment

SAP Sales Cloud and SAP Service Cloud Version 2

Reproducing the Issue

  1. Go to Settings.
  2. Search for Emails.
  3. Navigate to Templates.
  4. Click on Create New Template.
  5. Notice that the Placeholder Tab is not available.

Cause

The placeholder tab is not visible due to missing mandatory field selections, specifically Object Type and Template Type, in the Template creation screen.  

Resolution

Ensure that mandatory fields such as Object Type and Template Type are populated with values in the Template creation screen.  

  1. Go to Settings.
  2. Search for Emails.
  3. Navigate to Templates.
  4. Click on Create New Template.
  5. Select a value in Template Type and Object Type fields.

Keywords

Templates, Placeholder, Email, Service, Object Type, Template Type
