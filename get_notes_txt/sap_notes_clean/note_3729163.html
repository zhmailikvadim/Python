Symptom

In the XML file generated from the JP eDocument, the content of node "party_name" is incorrect.

Environment

  • SAP ECC
  • SAP S/4HANA

Reproducing the Issue

  1. Run T-code EDOC_COCKPIT
  2. Choose JP eDocument
  3. Display the XML file of the eDocument

Cause

"Party_name" is concatenated with "Name1"~"Name4" from the customer master data.

Resolution

If the value of "Party_name" not meet business requirement, following workarounds can be considered:

  • Changing customer master data
  • Changing value of "Party_name" in BADI EDOC_ADAPTOR~SET_OUTPUT_DATA    Fill or change eDocument data after mapping
