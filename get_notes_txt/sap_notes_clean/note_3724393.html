Environment

  • SAP IQ 16.1 SP05 PL16

Reproducing the Issue

  • No reproduction.

Cause

  • The issue has been identified and logged under SAP CR 831807. Its description reads, "Stack trace on hs_subohc::SmallDelete". 

Resolution

  • There is currently no target SAP IQ Service Pack scheduled for this issue (SAP CR 831807). It has been assigned to our Product Group. Please mark this KBA as a Favorite to be automatically notified when the CR status is changed
    Workaround:
    Since stack trace refers smalldelete, try changing HG_DELETE_METHOD option to 2 since 32,511 rows are getting deleted/updated. See whether that works.
    Run sp_iqcheckdb on these HG indexes to see whether there is any issue.
    Try dropping and recreating the corrupted indexes.
