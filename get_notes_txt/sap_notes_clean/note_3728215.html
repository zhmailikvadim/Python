Symptom

The DGFiP BAR file for neutral PAS rates uses only a year, not a specific date, for each PAS rate. Program RPCBAREMEF0 uploads the content of the BAR file to table T5FPASBAREME with the correct validity (start and end dates) for each year except for the year 2026.

Other Terms

"Prélèvement à la source", T5FPASBAREME

Reason and Prerequisites

The DGFiP BAR does not handle mid-year validity periods for neutral PAS rates.

Solution

After implementing this SAP Note, download the DGFiP BAR file at the net-entreprises website. Run program RPCBAREMEF0.
The system sets the start date for the year 2026 rate entries to May 1, 2026. The system delimits the end date of all entries for the year 2025 to April 30, 2026.

The HR Support Packages referenced below contain these corrections. For urgent requirements, implement the correction instructions that follow.
