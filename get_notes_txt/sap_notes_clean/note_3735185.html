Symptom

User is unsure which Joule edition they have or should use, leading to confusion about features, licensing, or installation. 

Environment

  • Joule.
  • Joule Studio.
  • Joule for Developers.
  • Joule for Consultants.

Resolution

Joule Base

Joule Base is the foundational SAP AI copilot. It provides a natural language interface for interacting with SAP applications, enabling users to perform standard tasks and retrieve information using conversational input. Its focus is on supporting core SAP interactions in a consistent and controlled manner.

Core focus: Basic system navigation, data search, and simple transactional support for daily productivity.

Core features include:

  • Intelligent navigation

  • Access to SAP Help documentation

  • Document Grounding

  • Simple transactions

  • https://www.sap.com/products/artificial-intelligence/ai-assistant.html

Joule Studio

Joule Studio is SAP's development platform for creating custom AI capabilities and business-ready agents. It enables organizations to build tailored AI solutions that integrate with their specific workflows and business processes through a comprehensive development environment.

Core focus: Building custom AI agents, skills, and workflow automation for specific business requirements.

Core features include:

  • Custom AI agent and skill creation capabilities,
  • Workflow automation design and implementation
  • Document grounding tools for knowledge-based agents
  • Integration with SAP Build for comprehensive development workflows

Additional information:

  • What is Joule Studio?
  • https://www.sap.com/products/artificial-intelligence/joule-studio.html

Joule for Consultants

Joule for Consultants is designed specifically for SAP functional experts, consultants, and implementation specialists . It enables consultants to quickly obtain precise and in-depth answers regarding the design, implementation, or update of SAP systems across the entire SAP product portfolio.

Core focus: System configuration guidance, project decision support, and access to SAP best practices for consulting professionals

Core features include:

  • Access to comprehensive SAP documentation and best practices
  • ABAP and CDS code interpretation and analysis capabilities
  • System configuration guidance and troubleshooting support
  • Real-time, context-aware guidance grounded in SAP's authoritative knowledge

Additional information:

  • SAP Joule for Consultants
  • https://www.sap.com/products/artificial-intelligence/ai-assistant/sap-consulting-capability.html

Joule for Developers

Joule for Developers targets ABAP developers, technical consultants, and development teams working with SAP technologies. It assists with day-to-day development tasks including coding, testing, and process development within SAP environments.

Core focus: Development assistance, code generation, and technical support for SAP development environments

Core features include:

  • ABAP code generation, testing, and remediation capabilities
  • Integration with SAP Build Code and ABAP Development Tools
  • Code explanation, refactoring, and documentation generation
  • CDS view creation and debugging assistance for data modeling
