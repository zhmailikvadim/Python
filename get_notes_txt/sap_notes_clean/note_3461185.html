Symptom

You create the statements of earnings for sick pay, maternity pay, or transitional pay using the Creation of Outbound Notifications for Compensation Benefits report (RPCEEVD0_OUT). However, the data fields for ending the employment relationship are not filled in the data module DBAL General or in the data module DBMU Remuneration statement for calculating maternity allowance.

Other Terms

Notification procedure for compensation benefits (EEL)
Report RPCEEVD0_OUT

Reason and Prerequisites

The information about ending the employment relationship must be entered in infotype 0651 SI Carrier Certificates. As a result of the changes delivered with this SAP Note, the notifications are now created with the status With Errors if a future leaving has already been entered in the master data but a record of the infotype 0651 does not yet exist for the relevant absence.

Solution

The corrections are delivered by HR Support Package. Alternatively, you can implement the correction instructions as an advance correction.
