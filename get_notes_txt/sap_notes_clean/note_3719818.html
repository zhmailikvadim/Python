Symptom

  • Error occurred during the Universal Journal (UJ) data release for a plan version.
  • Error messages encountered:
  • MSG_FINCS_DRT 201: Error occurred in Custom Field Mapping.
  • MSG_FINCS_DRT 204: Field &1 is not available for transfer from Accounting.

Environment

Release Universal Journals

Reproducing the Issue

  1. Navigate to the Data Monitor.
  2. Set the following global parameters:
  3. Execute the Data Release UJ task.

Cause

Resolution

  1. Verify if SAP Note 3517721 - "Custom specific fields requested for transfer for planning" has been implemented.
  2. If the note is not implemented, proceed to implement SAP Note 3517721.
  3. After implementation, test the Release UJ task for the specified planning version again.

See Also

  • Refer to: SAP Note 3517721 - Custom specific fields requested for transfer for planning
