Symptom

When rendering Portal Activity report of type "Number of Users who logged on", the chart that is rendered using an applet technology is no longer supported

Therefore, the chart is removed from the report. Nevertheless, the report still contains all relevant data in it's table.

Other Terms

htmlb chart, portal activity report chart, applet technology, Number of Users who logged on

Reason and Prerequisites

Reason: program error.

Prerequisites: You are using Enterprise Portal 7.3, 7.4 or 7.5.

Solution

If a chart is needed, you can view the chart using new SAPUI5 technology - for more information refer to SAP note 2405239.

Under the Support Packages & Patches tab in this note, you can check for the appropriate SP and Patch where this old chart was removed.
