Symptom

  • The typeahead is not working on custom object field on the JobApplication in new applicant workbench
  • The field does not display values in alphabetical order nor Typing the name of a field value narrow downs the results effectively.
  • User are required to scroll through all fields values to locate the desired one

Environment

SAP SuccessFactors Recruiting Management

Reproducing the Issue

  1. Go to requisition >> open any candidate's application
  2. Attempt to edit and update the custom object field.
  3. Observe that the values are not in  alphabetical order nor the field does filter results effectively when typing and only bolds visible selections, making it difficult to find the desired value for the user

Resolution
