Symptom

  • BadI Implementation for changing the Poland KSeF 2.0 FA3 schema Invoice.
  • The Display/Preview XML functionality is not working due to an implementation check of the BAdI EDOC_INTERFACE_CONNECTOR, method DISPLAY_EDOCUMENT.
  • Error message: "eDocument cannot be displayed; Check implementation of BADI EDOC_INTERFACE_CONNECTOR" (Message No. EDOCUMENT097).

Environment

  • SAP ERP 
  • SAP S/4HANA
  • SAP Document and Reporting Compliance Service

Reproducing the Issue

  1. Run the transaction code Edoc_cockpit. 
  2. Select the poland KSEF eInvoice document and display the XML.
  3. Observe the error message: Edocument097

Cause

  • Missing implementation of localized class CL_EDOC_INTF_CONN_PL_V2 for the interface connector for SAP S4 release. 
  • Missing implementation of localized class ZCL_EDOC_INTF_CONN_PL_V2 for the interface connector for SAP ERP release.

Resolution

  1. Ensure that communication through cloud service is active by configuring in SM30 view EDOCOMPPROCCOMV. For more information on this refer KBA 3723819 DRC Poland KSEF 2.0 eDocument Version Display Issue SAP for Me resolution section.
  2. Define interface type for eDocument, ECC interface type should be AIF_PROXY for SAP ECC release, and STANDARD for S4 release. Refer to the following resources for more information: 
    For SAP ECC: Defining Interface Type for eDocument | SAP Help Portal.
    For SAP S4 HANA: Defining Interface Type for eDocument | SAP Help Portal.
  3. Verify that the note 3676968 is implemented and manual activity UDO report NOTE_3676968 is run, which delivers class CL_EDOC_INTF_CONN_PL_V2 relevant for SAP S4HANA.
  4. For SAP ECC release, follow the instructions for manual activities from SAP Note 3682042 eDocument Poland KSEF eInvoice: FA (3) Application Interface Framework (AIF) BAdI Implementations (Relevant for SAP ERP Only SAP for Me) which delivers class ZCL_EDOC_INTF_CONN_PL_V2 relevant for SAP ECC.
    Review the instructions for implementing the interface connector for ECC in the SAP Help Portal: Implementation of BAdI EDOC_INTERFACE_CONNECTOR | SAP Help Portal.
  5. Ensure all prerequisites mentioned in the overview note 3279206 Electronic Document Processing for Poland KSeF: Implementation Overview SAP for Me are implemented before proceeding.
