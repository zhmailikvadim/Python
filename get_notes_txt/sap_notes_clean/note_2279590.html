Symptom

You are in the table view. You have selected elements that are to be changed using mass maintenance. During this, you notice that the field control for the fields "Responsible Role" and "Responsible Resource" is not taken into account and data is copied to fields that are not ready for input.

Other Terms

Mass change, mass maintenance, DPR_TABLE, DPR_TS_API_RESPONSIBLE, RESPONSIBLE_NAME, RESPONSIBLE_ROLE

Reason and Prerequisites

Program error

Solution

Use transaction SNOTE to implement this SAP Note in your system.
