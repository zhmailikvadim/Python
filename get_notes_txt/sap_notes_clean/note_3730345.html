Symptom

  • How to retrieve the data of recurring and successful instances in the history using Query Builder?

Environment

  • SAP Business objects Version 4.x / BI 2025
  • All Supported OS

Cause

  •  Retrieve the data of recurring and successful instances in the history using Query Builder

Resolution

1. Login Query Builder - Localhost:8080/AdminTools 

2. Run the following query:

SELECT top 1000 * FROM CI_INFOOBJECTS WHERE SI_KIND = '<NAME_OF_REPORT>' AND ( SI_RECURRING = 1 OR (SI_INSTANCE = 1 AND SI_SCHEDULE_STATUS = 1))

Keywords

success instance, recurring, query builder, retrieve data, Business objects,
