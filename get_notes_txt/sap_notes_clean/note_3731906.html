Symptom

Netherlands Audit file generated with error: "Value XXXXX is greater than the max. permitted value 9999" for element linescount.

Environment

SAP S/4HANA Cloud Public Edition.

Reproducing the Issue

  1. Open the Run Statutory Reports app.
  2. Search for the Report Name: Audit File for The Netherlands - NL_AUDIT_FILE.
  3. Open the report "Audit File for The Netherlands".
  4. Select the activity "Audit File for The Netherlands 3.2.1".
  5. Observe that the run is generated with error: "Value XXXXX is greater than the max. permitted value 9999" for element name linesCount.

Cause

The Audit File XSD XmlAuditfileFinancieel3.2.1.xsd defined by the tax authority restricts the linesCount element (type Nonnegativeinteger10) with maxInclusive value="9999". Runs exceeding this limit fail schema validation.

Resolution

As of 2025, file format version 4 is available in the system, removing the restriction previously imposed on the linesCount element.

  1. Go to Run Statutory Reports app.
  2. Search for the report "Audit File for The Netherlands - NL_AUDIT_FILE".
  3. Choose the activity "Audit File for The Netherlands 4.0".
  4. Start a new run for the desired reporting period.
