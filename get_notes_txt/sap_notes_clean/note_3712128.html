Symptom

  • The requirement to schedule jobs for the program `ICA_DATA_REMEDIATION` to update field values in Intercompany Matching and Reconciliation (ICMR) for changed documents.
  • Inquiry regarding the possibility of defining a filter string as a dynamic selection so that the `FISCYEARPER` field can be changed based on each period.

Environment

  • Group Reporting
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Schedule jobs for the program `ICA_DATA_REMEDIATION`.
  2. Attempt to define a filter string as a dynamic selection for the `FISCYEARPER` field to change it based on each period.

Cause

Standard system design.

Resolution

Please note that there is no standard solution available for this functionality within the program. This is due to the program's primary purpose, which is to address inconsistency in data. If you are aware of the root cause of the inconsistent data, it is recommended to focus on resolving the underlying issue to prevent its occurrence, rather than relying on this program to address the symptoms.
