Symptom

  • Inquiry regarding whether SAP Fieldglass mobile applications comply with specified mobile application security standards.
  • Request for confirmation on adherence to MDM/MAM policies for mobile access paths.
  • Inquiry about baseline security measures for mobile instances (SaaS or custom-developed).

Resolution

SAP Fieldglass provides two mobile apps that are available for download from the Apple App Store (iOS) and Google Play (Android) - listed below. In addition to mobile apps, the SAP Fieldglass web interface is accessible and compatible on browser interfaces on mobile devices. Pages have been written responsively for mobile browsers such as login screens, approvals, worker close, and work order revision screens.

SAP Fieldglass Manager Hub - Designed to provide convenient, role-based access to key capabilities from our desktop application, the SAP Fieldglass Manager Hub mobile app lets you:

  • Log in quickly and securely via PIN or biometric identification
  • Expedite worker engagement with options to create job postings, vet and shortlist candidates, schedule interviews, hire workers, review rates, and create and revise work orders
  • See worker status and alerts at a glance through the My Workers dashboard
  • Oversee and approve/reject worker activities, time sheets, and expense sheets
  • View and process pending approvals, interviews, work items, and more

SAP Fieldglass Time Entry - By making it simple to manage time sheets on a mobile device, this app helps external workers focus more on the job and less on logistics with the ability to:

  • Log in quickly and securely via PIN or biometric identification
  • Quickly record, review, and revise hours worked without logging into a
    desktop application
  • Input hours on a daily basis to enhance time reporting accuracy
  • Instantly respond to rejected timesheets and keep invoicing cycles on track
  • See up-to-the-minute status of previous and pending time sheets

Regarding the mobile apps provided by SAP Fieldglass, MDM integration is not necessarily required. All users must already be registered users of the SAP Fieldglass system. A QR Code, located in the profile section of the desktop application, is used when authorizing the mobile device with the user account. Devices may be de-authorized via the desktop application. No customer data is stored on the mobile device. SAP Fieldglass web interface is accessible and compatible with browser interfaces on mobile devices. Certain pages have been written responsively for mobile browsers including login, worker close, and work order revision screens.

SAP Fieldglass follows industries standards to protect the mobile app. Installation and use of mobile apps on rooted or jailbroken devices is prohibited. The app is an extension of SAP Fieldglass VMS (web application) and does not store any business/sensitive data in the device. For security reasons, user registration is not supported by mobile apps, mobile app users must first register on the web application to access the app. The app requires a two-legged authentication to pass before any data can be viewed or transacted. The app communicates with the server over HTTPS with valid and secure certificates to ensure confidentiality.

All SAP Fieldglass mobile apps are subject to a third-party penetration test that is conducted on an annual basis. The SAP mobile distribution team is responsible for ensuring all corporate and product security standards have been met prior to distributing to the online app stores.
