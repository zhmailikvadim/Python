Symptom

Reassignment of a warehouse task from one warehouse order to another via the API is not supported when ELS logging is enabled, if the warehouse task or either the source or destination warehouse order has planned workloads assigned.

Other Terms

Extended Warehouse Management (EWM), Planned Workload, Warehouse task, Warehouse Order, Labor Demand Planning (LDP), Engineering Labor Standards (ELS), Warehouse Order and Task A2X API

Reason and Prerequisites

Reassignment of warehouse tasks in warehouse orders with planned workloads using the API is not supported when ELS logging is enabled.

Remark: This note is only relevant if you have implemented release S/4HANA Private Cloud 2025 FPS1 or have already implemented notes 3673447 & 3684698 in release 2025 FPS0.

Solution

Implement this note.
