Symptom

Supplier Material Number field does not support the search/auto-fill function from the database. Users must enter the internal "Material Number" first to retrieve the corresponding "Supplier Material Number."

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Search Create the Purchase order App
  2. Input the supplier material number
  3. The system shows no results
  4. Attempt to use the "Filter," but there is no search bar for "Supplier Material Number".

Cause

This is currently a functionality which is unavailable.

Resolution

  1. The solution of this issue is currently under investigation. Please follow this KBA for updates.
  2. Users may consider raising a feature request via SAP Customer Influence to request the addition of this functionality.
  • Steps to submit a feature request:
  1. Open SAP Customer Influence (http://influence.sap.com/saps4hanacloud).
  2. Register and accept the Terms of Use.
  3. Choose a session of interest.
  4. Search for existing improvement requests. If available, vote and comment on them.
  5. If no existing improvement requests are found, create and submit a new improvement request.
  6. Refer to KBA 2963059 for detailed instructions on submitting feature requests.

See Also

  • Refer to: SAP Note 0003606803 for additional information.
  • Refer to: KBA 2963059 for details on submitting feature requests in SAP S/4HANA Cloud.
