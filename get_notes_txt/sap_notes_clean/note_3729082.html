Symptom

  • Error message: "User status COMP is active (WBS 12345)"
  • Time and expense confirmations fail to sync/post from Salesforce to SAP S/4HANA Cloud for Customer Project.

12345 Represents the WBS ID.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Go to Project Control - Professional Services projects.
  2. Search for project ID.
  3. Go to Affected Work package ID .
  4. Note that WBS has a work item assigned but no staff confirmed.
  5. Attempt to sync/post time and expense confirmations from Salesforce to SAP S/4HANA Cloud for project ID.
  6. Observe the error message: "User status COMP is active (WBS 12345)".

Cause

Affected WBS has a work item assigned but no staff confirmed, causing the error.

Resolution

  1. Identify the work item assigned to WBS that is not staff-confirmed.
  2. Remove the incomplete work item from WBS  directly in S/4HANA Cloud.
  3. Alternatively, confirm the staff for the work item in WBS and save the changes.
  4. Test the sync/posting operation from Salesforce to SAP S/4HANA Cloud for project ID.
  5. Verify that the error message no longer appears and the operation is successful.
