Symptom

While performing the Create and Generate process for Price Review, the system terminates with a runtime error.

This SAP Note addresses the issue and prevents the runtime dump during Price Review generation.

Other Terms

/IRM/IPPMM, /IRM/IPPD, /IRM/IPPD_POST, Price review creation , Price review Generate ,Price review runtime error

Reason and Prerequisites

This is caused by program error.
Note : SAP Note# 3709292 is available for V/4 release.

Solution

Please implement the correction instructions attached. These corrections are only valid if you have applied the Support Packs

Release 6.4 – Base Package & Support Pack 1
Release 6.3 – Base Package
Release 6.2 – Base Package

Use transaction SNOTE to implement the correction instructions specified below and activate the affected program object
