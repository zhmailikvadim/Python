Symptom

You are using BAI files to mark you checks as cashed and want to update check register table PAYR remotely in the sending HR system by using report RFEBCKHR. 
The checks are not marked but no error occured.

Other Terms

RFEBCKHR, HR_POSTING_UPDATE_PAYR_XBANC, PAYR, REGU, PMPAY

Reason and Prerequisites

A software error prevents the PAYR to updated

Solution

Implement the correction instruction or support package.
