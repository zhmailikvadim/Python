Symptom

There is a need to understand the calculation logic for field KZ83 in the output of DRC report DE_VAT_ADV_RTN

Environment

SAP S/4HANA

Resolution

Tax Box 83 (field KZ83) for the DE_ADV_VAT_RTN report is calculated and not directly mapped. No tax code should be directly mapped to this tax box.

The calculation logic is as follows:

  • Total Output Tax Amount - Total Input Tax Amount - Amount in KZ39 = Amount in KZ83

Keywords

DE_VAT_ADV_RTN, calculation logic, KZ83, DRC, Germany Advance VAT Return
