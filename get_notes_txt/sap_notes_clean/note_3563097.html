Symptom

This KBA explains the trigger of movement type 651 or 653 when executing Post Goods Received (PGR) during the sales returns process.

Environment

SAP S/4HANA Cloud Public Edition

Cause

Resolution

  1. Open the configuration activity "Assign Schedule Line Categories (ID: 105642)".
  2. For the item category, e.g., CBEN, verify whether the currently proposed schedule line category is WN (movement type 653) or DN (movement type 651).
  3. Adjust the schedule line category according to the business need.

See Also

  • Lean Customer Returns (BDD)
    "Goods is returned to unrestricted-use stock with schedule line WN (movement type 653) by default. You can use configuration activity Assign Schedule Line Categories (ID: 105642) to post return to blocked stock with schedule line DN (movement type 651)".
