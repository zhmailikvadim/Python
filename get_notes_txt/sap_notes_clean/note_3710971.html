Symptom

You use the Supply Assignment (ARUN) functionality as part of the Order Fulfillment Process.
You create Stock Transport demand documents with Supply Assignment (ARUN) relevant materials.
You then run the Supply Assignment Process for these demand documents and assign the Stock Transport Order (STO) item to multiple storage location physical stock.
When you create the Outbound Delivery Documents (OBD) for the corresponding Stock Transport Order, the system displays incorrect storage location stock in the Outbound Delivery.

Other Terms

Supply Assignment, ARUN_BDBS, OBD

Reason and Prerequisites

Prerequisites
The Stock Transport Order has an item that is supply assigned against multiple storage location stock.
Reason
This issue occurs due to a program error.

Solution

The program corrections are contained in the tab correction instruction of this note. You need to implement the corrections in your system.
