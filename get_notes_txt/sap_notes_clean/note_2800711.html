Symptom

When you are trying change the Processor of Service Ticket through Data Workbench, system throws below error

Create not allowed for properties: RoleCategoryCode. Check metadata for ServiceRequestOtherPartyCollection

Service: ServiceRequestOtherPartyCollection

Environment

SAP Service Cloud

Reproducing the Issue

  1. Go to Data Workbench work center
  2. Upload the file for ServiceRequestOtherParty
  3. Import the data

Cause

Cause for the issue is that you are trying to change the RoleCategoryCode which is Updatable False

Resolution

If you want to update the Processor, then Role Code should be 40 and RoleCategoryCode should remain as 100.

Keywords

Rolecategorycode, rolecode
