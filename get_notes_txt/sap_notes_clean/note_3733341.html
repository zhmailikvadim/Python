Symptom

The app Manage Substitution/Validation Rules (ID F4406) is not visible, even though all roles and business catalogs have been assigned to the user.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

Cause

The app is present but listed as “Manage Substitution / Validation Rules” with two blank spaces around the “/” character. Searching or scanning for the name without these spaces can cause it to appear as not found.

Resolution

  1. In the SAP Fiori launchpad, search for the app using the exact tile name “Manage Substitution / Validation Rules” (note the spaces around “/”).
  2. Alternatively, search using the app ID “F4406” to locate the tile independent of the display name.
  3. Once found, add the app to the user’s home page or a relevant group to ensure easy access going forward.
