Symptom

In the process "Installation, Replacement, or Shutdown of Devices" (DE_B_MCM_DEV_INSTLN_PROCESS) within the MCM_DE cluster, a runtime error (CX_SY_ITAB_LINE_NOT_FOUND) occurs in step 0080-0020.

The error occurs in class /US4G/CL_DEF_BADI_MCM_DATA_ACC, method /US4G/IF_BADI_MCM_DATA_ACCESS~DERIVE_BILLREL_DEVICES_ADJUST, when the system tries to read missing entries from the LT_POD internal table.

This issue is relevant for the market role: DSO.

Other Terms

/US4G/CL_DEF_BADI_MCM_DATA_ACC, MCM_DE, DE_B_MCM_DEV_INSTLN_PROCESS, CX_SY_ITAB_LINE_NOT_FOUND, device installation, device replacement, device shutdown, internal table, LT_POD, DERIVE_BILLREL_DEVICES_ADJUST

Reason and Prerequisites

The issue occurs because of a program error.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, the system checks whether an entry exists in the LT_POD internal table before reading it.

You need to retest the scenario. To verify the fix, run the process again and confirm that the runtime error no longer occurs.
