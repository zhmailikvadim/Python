Symptom

CT04 does not allow saving, when two characteristic values have the same description in the same language. Error message C1223 is issued.

Environment

  • SAP ERP Classification (CA)
  • SAP R/3
  • SAP R/3 Enterprise
  • SAP ERP Central Component
  • SAP ERP
  • SAP Enhancement package for SAP ERP
  • SAP Enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA, on-premise
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Open transaction CT04.
  2. Open any characteristic.
  3. Navigate to the "Values" tab.
  4. Add two values.
  5. Assign the same description to both values in the same language.
  6. Save

Cause

In standard SAP systems, the description field is language-dependent. Therefore, two characteristic values cannot have identical descriptions in the same language.

Resolution

Ensure that descriptions assigned to characteristic values are unique within the same language.

See Also

SAP Note 1292079 - Chars mgmt: C1 047 Language-dependent value already exists
