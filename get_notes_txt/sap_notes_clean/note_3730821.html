Symptom

During material consolidation, the rollback step does not complete and remains in progress. The screen then flickers and the "Continue" option is not available, preventing further processing. 

Environment

  • SAP Master Data Governance
  • Consolidation & Mass Processing

Reproducing the Issue

  1. Log in to SAP Fiori Launchpad.
  2. Navigate to Manage Consolidation Process.
  3. Create or open an existing consolidation process.
  4. Execute the process until rollback step is required.
  5. Attempt to perform Rollback

Result:

  • System hangs during rollback.
  • Screen flickers.
  • "Continue" option is not available.

Cause

The issue is caused by authorization inconsistencies for the SAP_WFRT user.

Resolution

Ensure SAP_WFRT user has correct S_RFCACL authorization & maintain trusted RFC configuration properly.

Keywords

MDG, Material Consolidation, Rollback Issue, Continue Button Missing, SAP_WFRT, S_RFCACL, RFC Authorization, CL_MDC_LOAD, Fiori Flickering, MDC_PROCESS_SRV
