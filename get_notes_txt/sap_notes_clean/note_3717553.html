Symptom

  • When using transaction /ECRS/RP_EDIT or the app 'Manage Intrastat declaration,' the quantity in the supplementary unit is missing or displayed as zero, despite the supplementary unit being available.  
  • An error message appears: 'Enter a valid quantity in the supplementary unit - /ECRS/BL_TD 86.'

Environment

  • International Trade Management
  • Intrastat Reporting
  • SAP S/4HANA
  • SAP S/4HANA private cloud edition
  • SAP S/4HANA public cloud edition

Reproducing the Issue

  • Access transaction /ECRS/RP_EDIT or the app 'Manage Intrastat declaration.'  
  • Open one of the Intrastat declarations.  
  • Select an item from the declaration.  
  • Observe the error message: 'Enter a valid quantity in the supplementary unit - /ECRS/BL_TD 86.'  

Cause

  • The material master lacks a conversion for units of measure.  
  • The table MARM is missing an entry for the relevant MATNR and MEINH. 

Resolution

  1. Open transaction MM02 using the relevant material number from the Intrastat declaration item.  
  2. Navigate to 'Additional Data'.  
  3. Click on the 'Units of Measure' button.  
  4. Add the conversion between the supplementary unit and the base unit of measure. 
  5. Save the changes.
  6. Rerun the Intrastat declaration using transaction VE01 (Sales) or MEIS (Purchasing) to update the report.
