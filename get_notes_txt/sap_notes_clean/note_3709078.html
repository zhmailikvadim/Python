Symptom

While attempting to post an Invoice using BAPI_INCOMINGINVOICE_CREATE1, the system does not allow posting via Freight Settlement Document (FSD), which is part of the TM solution.

Environment

  • Materials Management (MM)
  • SAP ERP Central Component
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP, version for SAP HANA
  • SAP S/4HANA (all Versions)

Reproducing the Issue

  1. Open transaction code SE37.
  2. Use BAPI: BAPI_INCOMINGINVOICE_CREATE1.
  3. Refer to Table: TM_ITEMDATA.
  4. It is not possible to enter Freight Settlement Document.  

Resolution

  1. It is not possible to post using the Freight Settlement Document (FSD).
  2. The user needs to use the TOR number ID to post the MIRO via BAPI_INCOMINGINVOICE_CREATE1:
