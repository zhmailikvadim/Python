Symptom

When a user acknowledges the changes shown in the change alert dialog in the Perform Operation Activity application, the acknowledgment is successfully processed. However, the change alert dialog becomes frozen or unresponsive, and a continuous busy indicator appears on the screen, making the user refresh the application to continue working.

Other Terms

PEO, Change Alert, Change Alert Acknowledgement, Perform Work, Busy Indicator, Screen Freeze

Reason and Prerequisites

Program Error.

Solution

After implementing this SAP Note, the change alert dialog will be closed once the change alert is acknowledged.
