Symptom

I need to close a Work Order Change Request that is in Created status and would like to know how to proceed. 

Resolution

The Work Order Change Request must be declined by the supplier before it can be closed.

Supplier action

  1. Go to Worker > Work Order

  2. Select Work Order Change Requests

  3. Search for the Work Order Change Request ID and click on its result

  4. Click Decline

  5. Select a Reason

  6. Click Decline

After the Change Request is declined, the status will be updated.

Buyer action

  1. Go to Worker > Work Order

  2. Search for the Work Order Change Request ID and click on its result

  3. Click Actions > Close Work Order Change Request

  4. Enter a Reason

  5. Click Accept

The Work Order Change Request will be closed.
