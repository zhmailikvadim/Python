Symptom

  • Not able to upload Classifiers using a template under Plan Data > Classifications
  • The import file hangs with the "Check Status" displaying "Waiting" indefinitely.
  • No error messages are generated, and the process does not complete.

Environment

  • SAP Sales Performance Management 
  • Incentive Management

Reproducing the Issue

  1. Navigate to Plan Data > Classifications > Classifiers.
  2. Attempt to upload the provided import file.

Cause

The incorrect classifier type was used during the upload process.

Resolution

  1. Ensure that the correct classifier type is selected during the upload process.
  2. Use the classifier created under Classifications as the "generic classifier type."
  3. Verify that the import file matches the required format for the classifier type.
