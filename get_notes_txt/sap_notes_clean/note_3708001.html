Symptom

In the process "Start of Basic or Backup Supply for Gas (New SUP View)" (DE_B_GELIGAS_SBS_SUP), you're unable to read process step data in the DETERMINE_EOG_TYPE method implementation of BAdI /us4g/badi_gpke_sup_sbs when the framework calls it in /US4G/CL_PRST_OUT_SBS_SUP_CONF->ENHANCE_STEP_DATA in step 1300.

This issue is relevant for the market role: SUP.

Other Terms

Geligas, BM_GPKE_SUP_SBS_RESP, /US4G/IF_BADI_GPKE_SUP_SBS, /us4g/badi_gpke_sup_sbs, DETERMINE_EOG_TYPE, ENHANCE_STEP_DATA, BAdI, Start of Basic or Backup Supply for Gas

Reason and Prerequisites

The error occurred because the importing parameter is_process_step_data wasn't passed in the call of CALL BADI lr_badi_sbs_sup->determine_eog_type in /US4G/CL_PRST_OUT_SBS_SUP_CONF->ENHANCE_STEP_DATA.

Solution

Implement the corrections of this SAP Note or import the relevant support package.

After implementing this SAP Note, you can implement custom code based on process step data in the DETERMINE_EOG_TYPE method implementation of BAdI /us4g/badi_gpke_sup_sbs when the framework calls it in /US4G/CL_PRST_OUT_SBS_SUP_CONF->ENHANCE_STEP_DATA
