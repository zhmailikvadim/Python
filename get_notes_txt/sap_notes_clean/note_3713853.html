Symptom

Error messages displayed in TBEX when importing the file: 
"To-factor contains invalid signs!"
"From-factor contains invalid signs!"

Environment

  • Treasury and Risk Management (TRM)
  • SAP Financial Supply Chain Management
  • SAP ERP
  • SAP enhancement package for SAP ERP
  • SAP enhancement package for SAP ERP version for SAP HANA
  • SAP S/4HANA
  • SAP S/4HANA Finance

Reproducing the Issue

  1. Go to T-code TBEX.
  2. Import file - for example Securities market data.

Cause

The validation logic for the market_data_table_rec-ffact field was updated to ensure that only numeric characters (0-9) and spaces are allowed.

Resolution

Ensure that the market_data_table_rec-ffact field in the imported file contain only numeric characters (0-9) and spaces.

Keywords

TBEX; Securities market data; Market data management
