Symptom

One can still create infocubes after the system has been set to B4H(aka strict) mode.

Other Terms

BW/4HANA

Reason and Prerequisites

The wizard for creating infocubes with namespace contains an error. This correction insurres that only infocubes in namespace /CPMB/ can still be created.

Solution


  • SAP BW 7.50
    Implement Support Package 8 for SAP BW 7.50 (SAPK-75008INSAPBW) into your BW system. The Support Package will be available as soon as SAP Note 2419099 with the short text "SAPBWNews 7.50 BW ABAP SP8", which describes this Support Package in more detail, is released for customers

  •  

In urgent cases you can use the correction instructions.

Before you use the correction instructions, make sure that you check SAP Note 1668882 and SAP Note 2248091 for transaction SNOTE.

This SAP Note might already be available before the Support Package is released. In this case, however, the short text still contains the term "preliminary version".
