Symptom

When using Report DMC_MT_RT_DELETE_REMOTE to delete the function groups in remote systems, the function groups aren't deleted.

Other Terms

MWB, Function Group deletion, Generated Function Modules

Reason and Prerequisites

You're using MWB or any of its consumers and would like to delete the generated function groups in the remote systems.

MWB / DMIS functionality is used on one of the following releases:

  • DMIS 2018 SP11 - SP14
  • DMIS 2020 SP07 - SP10
  • DMIS 2025 SP00
  • SAP S/4HANA 1709 FPS00 - SPS11
  • SAP S/4HANA 1809 FPS00 - SPS11
  • SAP S/4HANA 1909 FPS00 - SPS11
  • SAP S/4HANA 2020 FPS00 - SPS11
  • SAP S/4HANA 2021 FPS00 - SPS09
  • SAP S/4HANA 2022 FPS00 - SPS07
  • SAP S/4HANA 2023 FPS00 - FPS05
  • SAP S/4HANA 2025 FPS00 - FPS01

Solution

 Implement the attached correction using transaction SNOTE.
