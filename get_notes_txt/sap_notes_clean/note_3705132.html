Symptom

  • The Supplier Risk Engagement API does not support bulk extraction of all risk workspaces, as it requires a specific workspace ID for each request.
  • The API successfully extracts data but only for a single, specified workspace ID.
  • The requirement is to extract data for all workspaces updated with configured content without specifying each individual workspace ID.

Environment

Supplier Risk API

Cause

  • The Supplier Risk Engagement API is designed to retrieve information from a single project or questionnaire at a time and lacks functionality for bulk data extraction across multiple workspaces.

Resolution

  1. Explore the Analytical Reporting API as an alternative for bulk data extraction.
  2. Refer to the Supplier Risk fact documents available within the Analytical Reporting API prefixed by "SR", such as:
    • SRProjectFact
    • SRProjectSurveyResponseFact
  3. Follow the detailed guidance provided in SAP Note 3635403 to utilize the Analytical Reporting API effectively.
