Symptom

  • I want to add a custom meeting URL (such as Microsoft Teams or Zoom) to an interview
  • An error occurs when scheduling an interview prompting to add the URL to the Allowed URLs list in Admin Configuration

Cause

Resolution

To allow a URL plus any paths within that URL, enter the domain name only: 

  1. Click Admin Configuration (gear icon) > Allowed URL
  2. Click New
  3. Enter website domain only in the Address field
    • Example: if website is https://www.amazon.com/books, enter https://www.amazon.com 
  4. Set Category to Interview
  5. Enter Reference Name
  6. Click Add
