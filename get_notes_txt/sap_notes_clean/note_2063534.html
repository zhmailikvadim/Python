Symptom

You use the DataSource 0FC_BP_ITEMS in delta mode. You would like to write data to delta queue by the mass activity FPBW_EXTRACT. It could happen that the error e031(FK) "Enter a valid start date" occurs and nothing is written to the delta queue.

Other Terms

FP-BWEXTR, 0FC_BP_ITEMS, FPBW_EXTRACT, 031(FK), FK031, FK 031, BWET

Reason and Prerequisites

Program error.

Solution

Implement the attached correction instructions or import the relevant Support Package.
