Symptom

  • Charts does not display data.
  • Charts displays the message: "You do not have enough authorization"
  • When viewing a story in SAP Analytics Cloud that contains charts based on an SAP BW Query using an SAP BW Live connection, it displays data for some users, but for other users it displays the error message: "You do not have enough authorization"

Environment

SAP Analytics Cloud

Reproducing the Issue

  1. In SAP Analytics Cloud, create a story, and add charts based on SAP BW Query using SAP BW Live Connection.
  2. When viewing the story, the charts returns data.
  3. When other users view the same story, the charts returns the error message:  "You do not have enough authorization"

Cause

The SAP BW User does not have enough authorizations on the SAP BW System to see data.

Resolution

  • When using an SAP BW Live Connection, the authorizations are all handled on the SAP BW System.
  • Contact the SAP BW BASIS Administrator to provide authorizations for the SAP BW Users to see the SAP BW Cube the SAP BW Query uses, as well as the data on the SAP BW side.

Keywords

SAC, SAP BW Query, rights, authorizations
