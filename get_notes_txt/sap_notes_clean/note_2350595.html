Symptom

/SCMTMS/SFM_OPT: x appears as a message when you optimize with the
CS optimizer.

Other Terms

Carrier Selection, optimizer

Reason and Prerequisites

A CS optimizer of a newer release (for example, csopsvr 10.0) that raises additional messages was imported in downport mode under an older SAP TM release (for example, TM 8.0). In occasional cases of emergency, a new message may also have been added between the Support Packages of a release, provided that this was urgently required as part of the process and notifications.

Solution

SAP recommends to install at least Transportation Management Service Pack. If this is not possible for some reason, the follwing correction has to be done manually.

Call Transaction SE91.
Display the message classes /SCMTMS/SFM_OPT.
Select the corresponding message x.
Maintain the selected message (F8).
Enter the message short text (see below for the text).
The message is self-explanatory.
Save your changes.
