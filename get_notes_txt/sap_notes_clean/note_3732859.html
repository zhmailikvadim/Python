Symptom

The requirement is to restrict a certain user to DISPLAY ONLY mode in SAP Central Business Configuration Workspace.

Environment

SAP Central Business Configuration

Resolution

Please create the business user and assign the read only access role: SAP_CBC_CONSUMPTION_DISPLAY_USER 

Note: Same business user cannot be restricted to display only at one workspace level and write access to different workspace in CBC, for that purpose multiple business user(s) need to be created and the access limited user should be assigned to the workspace(s) where read only access is required.
