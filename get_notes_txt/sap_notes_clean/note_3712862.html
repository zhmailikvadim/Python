Symptom

Cannot navigate to purchase requisition from schedule lines in item detail screen of sales order.

Environment

  • Sales and Distribution (SD) 
  • SAP S/4HANA
  • SAP S/4HANA Cloud Private Edition

Reproducing the Issue

  1. Call up the sales order using VA03 (GUI).
  2. Navigate to Menu -> Item -> Schedule lines.
  3. Double-click on the purchase requisition number.
  4. In the Schedule Line Data screen, click on the 'Procurement' tab.
  5. In the 'Procurement' tab, double-click on the purchase requisition number under 'External procurement'.
  6. Observe that the system does not navigate to Screen 102 of purchase requisition information.

Resolution

Implement SAP Note 3626492- When a batch input map is recorded, which adds a component in a production order, which eventually causes the creation of a purchase requisition, an error "screen SAPLEBNE 102 was expected" occurs.
