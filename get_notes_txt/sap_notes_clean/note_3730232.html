Symptom

You are working with a PMR document which contains multiple single order staging relevant components with the same parameters like material, owner, CAT.. In this case the system must create saparate staging WTs for all component items instead of merging them into one WT. The merging is allowed by using cross order relevant components.

Other Terms

PMR, PWR, Single order staging, multiple components in BOM with same material, PCSO, BOM, stage WT

Reason and Prerequisites

Program error. Cause/Side effect note of 3719871

Solution

Implement ther corretion instruction or the equivalent support package.
