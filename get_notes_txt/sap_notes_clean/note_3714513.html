Symptom

  • The issue occurs when retrieving location data in a story report.
  • A custom-string field (Label MDF) is configured in the Foundation Object: Location, with its value type set to a custom MDF object.
  • After maintaining values in Manage Organization, Pay, and Job Structure<location, creating a story report from the Location object. 
  • The retrieved MDF field displays a number instead of the MDF label.
  • Investigation reveals that the number corresponds to the internal ID of the MDF data.  

Environment

SAP SuccessFactors Employee Central 

  • Foundation Object
  • Story report

Reproducing the Issue

    Cause

    This is a known limitation. This limitation occurs because SAC reports do not support custom field references from legacy foundation objects, such as location, frequency, and location group. 

    Resolution

    Use a canvas report as a workaround to retrieve this type of data.  

    Keywords

    location, frequency, location group, foundation object, number, internal ID, internalId, canvas report , MDF configuration, story report, data retrieval issue
