Symptom

  • An issue occurs when attempting to change the ship-to-party while creating a sales order in the "Manage Sales Orders - Version 2" (F3893) app.
  • The ship-to-party cannot be changed from within the selection window unless the BP number is entered directly into the field first.

Environment

  • Sales Orders (Public Cloud)
  • SAP S/4HANA Cloud Public Edition CE2602

Reproducing the Issue

  1. Open App "Manage Sales Orders - Version 2" (F3893).
  2. Create a new sales order and fill the following details with relevant data:
    • Sales Order Type: OR
    • Sales Organization.
    • Distribution Channel.
    • Division.
  3. Set the value for Sold-to-party.
  4. Attempt to change ship-to-party within the ship-to popup window.
  5. It is not possible.

Cause

Resolution

  • The fix for this issue has been developed and is planned for delivery in Hot Fix Collection 05, scheduled to be released during the weekend of CW9 (February 28 to March 1).
  • As a temporary workaround, remove duplicated entries from the table response or manually enter the value in the ship-to-party field.
