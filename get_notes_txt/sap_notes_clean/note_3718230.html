Symptom

  • The pipeline job remains in a "Pending" status for an extended period.
  • Logs do not provide detailed information regarding the issue.

Environment

  • SAP Sales Performance Management
  • Incentive Management on Hyperscalers

Reproducing the Issue

  1. Run a pipeline job in the SAP Incentive Management system.
  2. Observe that the job remains in a "Pending" status for an extended period.
  3. Check the logs for details and confirm that no relevant information is displayed.

Cause

Resolution

  1. Try to Cancel the pending "Pipeline" job.
  2. Resubmit the processes.
  3. If the process still in pending status please proceed with the next step.
  4. Create a support ticket sharing the pipeline run parameters details.
