Symptom

Freight units are assigned to a freight order using the TGI service and the FU items are identified using the BTD ID. If an event is posted within the same message it could happen that information on the referenced items in the FO is missing such as product details.

Other Terms

Item Mapping, Injected Items, Update Handler

Reason and Prerequisites

Program error

Solution

 Implement the attached correction instructions or import the specified Support Package.
