Symptom

  • Comparing planning calendar/factory calendar in IBP with ERP (source system), the calendars are different/mismatch/not synched.
  • No same calendars between IBP and ERP.
  • Differences for UoM values between IBP and ERP.
  • Differences for currency values between IBP and ERP.

Environment

  • SAP Integrated Business Planning (SCM-IBP)
  • SAP ERP
  • SAP S/4HANA

Cause

  • Real-Time Integration (Calendar, UoM, Currency) job was not run
  • Real-Time Integration (Calendar, UoM, Currency) ran without “Erase Prev. Integrated Data”

Resolution

  1. Job Real-Time Integration (Calendar, UoM, Currency) needs to run to integrate the calendars, UoM and currencies values from ERP to IBP.
  2. Important: If this job runs without the “Erase Prev. Integrated Data” option, previously integrated data is kept and merged with the new data that is being integrating.
  3. For calendars, be aware that the BAdIs mentioned in the SAP help page Integration of Configuration Data can modify the behavior of integration.
