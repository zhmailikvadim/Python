Symptom

Quotation marks ("") are inserted into the cycle description when a hyphen (-) is present in the description during upload.

Environment

  • Controlling (CO)
  • SAP S/4HANA
  • SAP S/4HANA Finance
  • SAP S/4HANA Cloud Private Edition
  • SAP Fiori for SAP S/4HANA

Reproducing the Issue

  1. Create a template for allocation cycles.
  2. Input a cycle description containing a hyphen (-), for example, "6-1 test".
  3. Upload the template to the system.
  4. Observe that quotation marks ("") are added to the the cycle description.

Cause

Standard system behavior designed to prevent CSV injection attacks.

Resolution

Special quotation marks are added because the system is designed to prevent CSV injection attacks when a description includes any of the following characters: =, +, -, @, or ". It's able to adjust the description text when editing the cycle, after the template has been uploaded.

This protection is not needed only when downloading the template. After implementing SAP Note 3562836, it's able to download the template without any additional characters. However, for cycle uploads the quotation marks will be always added if the template contains the mentioned characters.
