Symptom

When uploading the XML of a Signed eDocument in the FACE Portal, an error is displayed.

Environment

SAP S/4HANA Cloud Public Edition

Reproducing the Issue

  1. Go to the eDocument Cockpit App.
  2. In the Source Document field, enter ABC (where ABC represents the Source Document field).
  3. Select Go to -> History.
  4. Double Click the Signed XML file.
  5. Upload the Signed XML downloaded in the FACE Portal.

The error is displayed.

Cause

The validation tools and logic of the FACe Portal are external components and fall outside the scope of SAP Technical Support. 

Resolution

In cases where errors originate from FACe Portal validations, it is necessary to contact the corresponding Tax Authority to identify the root cause and obtain further guidance.

Keywords

FACe, validation, Spain, logic, Portal
