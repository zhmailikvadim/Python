Symptom

When one of the following symptoms occurs:

1) An incorrect item category is determined when a valid service contract is provided in the payload of an HTTP POST OData V4 request.
2) Providing the service order item planned service start date and planned service end date fields in the HTTP POST OData V4 payload prevents the service order item from being created.

Other Terms

Service Order OData V4 Planned Service Start Date Planned Service End Date Item Category Determination
