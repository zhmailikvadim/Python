Symptom

You have set up the function described in SAP Note 3554925 - DE: Hybrid DRG remuneration agreement 2025 + §301 supplement from 12/12/2024 with effect from 1/1/2025 - Postoperative Posttreatment for Hybrid DRG in the system to be able to bill a hybrid DRG case with a value increased by EUR 30.00 if postoperative posttreatment took place.

However, the system only bills the case with the normal value, even though the conditions for billing the value increased by EUR 30.00 have been met and service rule R11 has changed the §301 charge key of the service.

Other Terms

Hybrid DRG, 115f, §115f, hybrid DRG regulation, hybrid DRG implementation agreement, hybrid DRG remuneration agreement, postoperative postprocessing

Reason and Prerequisites

This problem is caused by a program error.

The error occurs if service rule R11 has changed the §301 charge key of the service only at the time of billing.

Solution

Refer to the correction instructions for your release.

After you implement the correction instructions, the value increased by EUR 30.00 is also billed if the service rule R11 does not change the §301 charge key until the time of billing.
